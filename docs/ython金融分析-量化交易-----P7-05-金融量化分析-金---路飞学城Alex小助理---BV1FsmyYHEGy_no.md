# Python金融分析+量化交易：P7：05 金融量化分析-金融分析 📈

在本节课中，我们将要学习金融分析的核心方法。上一节我们介绍了金融和股票的基础知识，本节中我们来看看如何通过分析手段来判断股票的买卖时机，而不是盲目投资。

金融分析主要分为两种方法：基本面分析和技术面分析。

## 基本面分析

![](img/b103979a9c473f59034faa1fe96ac5f9_1.png)

基本面分析的核心是评估公司的运营状况，即我们之前提到的公司自身因素对股价的影响。这包括判断当前经济状况、行业发展前景以及公司自身的经营好坏。以下是基本面分析的三个层面：

![](img/b103979a9c473f59034faa1fe96ac5f9_3.png)

以下是基本面分析的三个主要方面：
1.  **宏观经济面分析**：分析国家的财政政策、货币政策等宏观因素。例如，判断国家当前是鼓励资金流入股市还是鼓励储蓄。但需注意，宏观经济规律并非总是与股市表现一致。
2.  **行业分析**：判断整个行业的发展趋势。例如，评估教育、IT或传统工业（如钢铁、煤炭）等行业的前景。可以通过观察该行业中几只代表性股票的走势来辅助判断。
3.  **公司分析**：具体分析目标上市公司的财务状况。上市公司需定期公开经审计的财务报告（财报）。投资者可以通过分析财报数据（如公司盈利、每股收益等）以及新闻、实地考察等信息，来判断公司运营是否健康、是否具有投资价值。

## 技术面分析

上一节我们介绍了基本面分析，它侧重于公司内在价值。本节中我们来看看技术面分析，其核心观点是：**所有信息都已蕴含在市场交易数据中**。技术面分析通过研究历史市场走势和价格图表来预测未来价格动向。

技术面分析依赖一系列技术指标。以下是两个最基础且重要的技术指标：

以下是两个基础且重要的技术指标：
*   **K线**：又称蜡烛图，是反映单个交易周期（如一天）内开盘价、收盘价、最高价和最低价的图表。
    *   **阳线**（通常为红色或空心）：表示该周期**收盘价高于开盘价**，即价格上涨。公式可表示为：`收盘价 > 开盘价`。
    *   **阴线**（通常为绿色或实心）：表示该周期**收盘价低于开盘价**，即价格下跌。公式可表示为：`收盘价 < 开盘价`。
    *   K线的**实体部分**上下沿分别代表开盘价和收盘价，**影线**的顶端和底端则分别代表最高价和最低价。
*   **移动平均线（MA）**：又称均线，是计算过去若干个周期平均价格的曲线，用于平滑价格波动，显示趋势。
    *   例如，`MA5`表示5日均线，其每个点的数值是**前5个交易日收盘价的平均值**。公式为：`MA5 = (P1 + P2 + P3 + P4 + P5) / 5`，其中P代表每日收盘价。
    *   同理，`MA60`表示60日均线。将每个交易日的均线值连接起来，就形成了移动平均线。

![](img/b103979a9c473f59034faa1fe96ac5f9_9.png)

技术面分析就是通过研究K线形态、均线组合（如后续会讲到的双均线策略）以及其他指标（如KDJ, MACD）来寻找交易信号。

![](img/b103979a9c473f59034faa1fe96ac5f9_11.png)

本节课中我们一起学习了金融分析的两大支柱：基本面分析和技术面分析。基本面分析从宏观经济、行业和公司层面评估价值；技术面分析则从历史市场数据中寻找价格规律和交易时机。理解这两种方法是进行理性投资和后续量化交易策略开发的基础。
