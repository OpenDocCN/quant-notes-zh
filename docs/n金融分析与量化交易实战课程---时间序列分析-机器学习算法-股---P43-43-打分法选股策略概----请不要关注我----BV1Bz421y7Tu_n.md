# 量化交易实战课程：P43：打分法选股策略概述

![](img/af6048b58f02fb893281747b73f36745_0.png)

在本节课中，我们将要学习一种在量化选股中非常常用的策略——因子打分选股法。我们将了解其核心思想、实施前提，并梳理出清晰的实现步骤。

## 策略核心思想 🎯

上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看如何综合多个因子进行更全面的选股。

因子打分选股的核心思想，是模仿学生考试的总分排名。假设我们有300只股票（股票池），每只股票都有A、B、C、D四个因子指标（例如市盈率、净资产收益率等）。我们的目标不是只看某一个因子，而是为每只股票的每个因子打分，然后计算一个**总分**。

具体流程如下：
1.  对每只股票，分别根据其因子A、B、C、D的数值进行打分。
2.  将每只股票在所有因子上的得分相加，得到该股票的**综合总分**。
3.  对所有股票按总分从高到低进行排名。
4.  选取排名最高的前10或20只股票，作为下一期调仓的投资组合。

这样，我们选出的就是“各科成绩”综合最优的股票，而非仅仅“单科状元”。这个策略的关键在于如何科学地为每个因子打分。

## 策略实施前提 📋

在开始为因子打分之前，我们必须明确一个重要的先验知识：每个因子与股票预期收益率的相关性方向。

以下是理解这一前提的关键点：

*   **正相关因子**：因子值**越大**，通常预示着未来收益率可能**越高**。例如，净资产收益率（ROE）、每股收益（EPS）通常被认为是“越大越好”的因子。
*   **负相关因子**：因子值**越小**，通常预示着未来收益率可能**越高**。例如，市盈率（PE）在估值模型中常被认为是“越小越好”（估值越低）。

**为什么必须知道这个？**
因为打分需要标准。对于一个“越大越好”的因子，我们自然应该给数值大的股票打高分；对于一个“越小越好”的因子，则应该给数值小的股票打高分。如果方向判断错误，整个选股逻辑就会失效。

![](img/af6048b58f02fb893281747b73f36745_2.png)

**如何获知因子的相关性方向？**
1.  **查阅研究报告**：各大券商和金融机构会发布大量因子研究，其中包含了因子的经验性结论。
2.  **自行计算验证**：我们之前在因子分析章节中介绍的方法，可以通过历史数据计算因子与未来收益的IC值或Rank IC值，来实证检验其相关性方向。

在接下来的实战中，我们会选择几组已知方向的因子（一组“越大越好”，一组“越小越好”）来构建策略。

## 策略实现步骤概览 🛠️

基于以上思想，我们可以将因子打分选股策略的实现分解为以下几个关键步骤：

以下是实现打分法选股的核心步骤列表：

1.  **确定股票池与因子**：选定初始股票池（如沪深300），并确定用于打分的多个因子。
2.  **确定因子方向**：明确每个因子与预期收益的相关性是正（越大越好）还是负（越小越好）。
3.  **因子数据标准化**：由于不同因子的量纲和范围差异巨大（如市盈率是几十，而ROE是零点几），需要先进行标准化处理，使其具有可比性。常用方法包括**Z-Score标准化**：`因子标准化值 = (原始值 - 因子均值) / 因子标准差`
4.  **因子打分**：根据标准化后的因子值和因子方向，为每只股票的每个因子赋予分数。例如，对于“越大越好”的因子，可以直接使用标准化后的值作为分数；对于“越小越好”的因子，则使用标准化值的负值或进行反向排名。
5.  **计算综合得分**：将每只股票在所有因子上的得分相加（或加权平均），得到该股票的综合总分。公式可表示为：`股票总分 = 因子A得分 + 因子B得分 + ... + 因子N得分`
6.  **排序与选股**：根据综合总分对所有股票进行降序排列，选取排名靠前的若干只股票构成投资组合。
7.  **定期调仓**：策略通常按固定周期（如每月）重复执行以上步骤，卖出不在新组合中的股票，买入新入选的股票，完成调仓。
