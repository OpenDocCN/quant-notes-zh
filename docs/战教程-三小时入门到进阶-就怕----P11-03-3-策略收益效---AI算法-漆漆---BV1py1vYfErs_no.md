## 计算基础收益率

上一节我们介绍了如何生成交易信号。本节中，我们来看看如何基于这些信号计算策略的最终收益。

首先，我们需要计算股票每日的基础对数收益率。这是评估任何策略表现的基准。对数收益率的计算公式为：

`log_return = log(今日股价 / 昨日股价)`

![](img/f54cb4c0982de8d786d7519647761aee_5.png)

![](img/f54cb4c0982de8d786d7519647761aee_7.png)

在代码中，我们使用Pandas的`shift`函数来获取前一天的股价，并进行计算。

![](img/f54cb4c0982de8d786d7519647761aee_9.png)

```python
# 计算对数收益率
data['return'] = np.log(data['close'] / data['close'].shift(1))
```

执行以上代码后，我们得到了一个名为`return`的列，它代表了股价每日的自然增长率。

![](img/f54cb4c0982de8d786d7519647761aee_11.png)

![](img/f54cb4c0982de8d786d7519647761aee_13.png)

## 应用策略计算策略收益率

基础收益率反映了市场的自然波动。现在，我们将引入上一节生成的交易信号（`position`列），来计算策略调整后的收益率。

![](img/f54cb4c0982de8d786d7519647761aee_17.png)

我们的策略逻辑是：当`position`为1时，我们持有股票，收益与市场同步；当`position`为-1时，我们进行反向操作（例如做空），收益与市场相反。因此，策略收益率可以通过以下方式计算：

`strategy_return = position * return`

![](img/f54cb4c0982de8d786d7519647761aee_19.png)

![](img/f54cb4c0982de8d786d7519647761aee_21.png)

需要注意的是，为了与收益率的计算对齐，`position`列也需要进行`shift(1)`操作，代表使用前一天的信号来决定今天的操作。

```python
# 应用策略信号计算策略收益率
# 对position进行shift，使其与收益率对齐
data['strategy'] = data['position'].shift(1) * data['return']
```

![](img/f54cb4c0982de8d786d7519647761aee_23.png)

![](img/f54cb4c0982de8d786d7519647761aee_25.png)

以下是计算过程中的关键步骤列表：
*   **对齐信号**：使用`shift(1)`确保交易信号作用于正确的交易日。
*   **计算策略收益**：将调整后的信号与基础收益率相乘，得到策略每日的收益率。

![](img/f54cb4c0982de8d786d7519647761aee_27.png)

## 汇总并比较最终收益

计算出每日的策略收益率后，我们需要将其汇总，以评估从期初到期末的总收益，并与“买入并持有”策略进行比较。

![](img/f54cb4c0982de8d786d7519647761aee_31.png)

汇总过程分为两步：
1.  **累加对数收益率**：将对数收益率求和，得到整个期间的总对数收益。
2.  **还原为实际收益**：通过指数函数 `exp(总对数收益)`，将总对数收益还原为实际的资产倍数（即1元本金最终变为多少元）。

![](img/f54cb4c0982de8d786d7519647761aee_33.png)

```python
# 计算总收益（还原为实际倍数）
total_return = np.exp(data['return'].sum())
total_strategy_return = np.exp(data['strategy'].sum())
```

![](img/f54cb4c0982de8d786d7519647761aee_35.png)

执行计算后，我们可以得到两个结果：
*   `total_return`：代表“什么都不做”（买入并持有）策略下，1元本金最终的价值。
*   `total_strategy_return`：代表应用我们的双均线策略后，1元本金最终的价值。

![](img/f54cb4c0982de8d786d7519647761aee_37.png)

通过对比这两个数值，我们可以直观地判断策略是否有效。例如，如果`total_strategy_return`显著大于`total_return`，则说明我们的策略创造了超额收益。
