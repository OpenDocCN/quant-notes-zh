## 构建交易逻辑

![](img/5fd522761da05fdff249b56ab306058d_34.png)

![](img/5fd522761da05fdff249b56ab306058d_36.png)

在完成股票筛选后，接下来我们需要在`handle_bar`函数中实现具体的交易逻辑。这个函数会在每一个交易周期（如每一天）被调用。

![](img/5fd522761da05fdff249b56ab306058d_38.png)

首先，我们需要判断当前账户的持仓情况。这可以通过查询`context`对象中的`portfolio.positions`属性来实现，它是一个包含所有持仓信息的字典。

![](img/5fd522761da05fdff249b56ab306058d_40.png)

![](img/5fd522761da05fdff249b56ab306058d_13.png)

以下是查询持仓的代码：
```python
current_positions = context.portfolio.positions
```

![](img/5fd522761da05fdff249b56ab306058d_42.png)

核心的交易逻辑如下：
1.  **判断初始状态**：如果当前持仓为空，则按平均比例买入筛选出的股票池中的所有股票。
2.  **执行调仓操作**：如果当前持仓不为空，则进行调仓。
    *   **卖出**：遍历当前持仓，如果某只股票不在最新的筛选股票池中，则将其全部卖出。
    *   **买入**：遍历最新的筛选股票池，如果某只股票当前未持有或持有比例不足，则按平均比例买入。

![](img/5fd522761da05fdff249b56ab306058d_44.png)

![](img/5fd522761da05fdff249b56ab306058d_46.png)

以下是调仓逻辑的关键代码片段：
```python
# 卖出不在股票池中的股票
for stock in current_positions.keys():
    if stock not in context.filtered_stocks:
        order_target_percent(stock, 0)  # 卖出全部

![](img/5fd522761da05fdff249b56ab306058d_48.png)

![](img/5fd522761da05fdff249b56ab306058d_50.png)

# 买入股票池中的股票，平均分配资金
for stock in context.filtered_stocks:
    order_target_percent(stock, 1.0 / len(context.filtered_stocks))
```
其中，`order_target_percent(stock, target_percent)`是一个重要的交易API，它的作用是调整指定股票`stock`的仓位，使其占总资产的比例达到`target_percent`。`target_percent=0`代表清仓，`target_percent=0.1`代表使该股票仓位达到10%。

## 策略回测与结果分析

![](img/5fd522761da05fdff249b56ab306058d_56.png)

策略编写完成后，我们将其放入回测框架中运行，以评估其历史表现。

![](img/5fd522761da05fdff249b56ab306058d_66.png)

![](img/5fd522761da05fdff249b56ab306058d_58.png)

回测结果会生成一系列绩效指标和图表。初始结果可能不理想（例如出现亏损），这很常见，因为策略效果与回测时间段、参数设置紧密相关。

![](img/5fd522761da05fdff249b56ab306058d_60.png)

![](img/5fd522761da05fdff249b56ab306058d_62.png)

![](img/5fd522761da05fdff249b56ab308d_70.png)

![](img/5fd522761da05fdff249b56ab306058d_64.png)

我们需要重点关注以下几个核心指标：

![](img/5fd522761da05fdff249b56ab306058d_66.png)

![](img/5fd522761da05fdff249b56ab306058d_68.png)

1.  **累计收益率**：策略在整个回测期间的总收益。图中通常包含**策略收益率曲线**和**基准收益率曲线**（如沪深300指数）。对比两者，可以判断策略是否跑赢了市场。
2.  **年化收益率**：将累计收益率折算成年化的收益水平，便于与其他投资产品比较。
3.  **最大回撤**：策略在历史上从任一高点下跌到最低点的最大幅度。**公式**为：
    `最大回撤 = (波峰价值 - 波谷价值) / 波峰价值`
    这个指标衡量了策略可能面临的最大亏损风险，数值越小越好。
4.  **夏普比率**：衡量策略承担单位风险所获得的超额回报。**公式**为：
    `夏普比率 = (策略年化收益率 - 无风险利率) / 策略收益率的年化标准差`
    该比率越高，说明策略的风险调整后收益越好。如果收益为负，夏普比率通常也为负。

![](img/5fd522761da05fdff249b56ab306058d_70.png)

![](img/5fd522761da05fdff249b56ab306058d_74.png)

![](img/5fd522761da05fdff249b56ab306058d_72.png)

此外，回测结果还包含详细的交易记录，可以查看每一笔买卖操作的时间、价格和数量，用于进一步分析策略的细节。

## 总结

![](img/5fd522761da05fdff249b56ab306058d_74.png)

本节课中我们一起学习了量化策略从数据处理到回测分析的全过程。我们首先通过**转置操作**将财务数据格式化为便于分析的样式，然后构建了一个基于股票池的**轮动交易策略**，在`handle_bar`中实现了“汰弱留强”的调仓逻辑。最后，我们通过回测系统评估策略，并学习了如何解读**累计收益、年化收益、最大回撤和夏普比率**等关键绩效指标。记住，一个策略的回测结果受多种因素影响，初次表现不佳是优化迭代的开始。
