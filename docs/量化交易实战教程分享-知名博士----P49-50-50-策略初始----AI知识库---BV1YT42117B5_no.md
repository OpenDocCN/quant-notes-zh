## 构建数据获取函数

![](img/8ab15f4a8d80e3f0db7a9fa6c93377ab_67.png)

上一节我们设置了调仓的框架，本节中我们来看看如何具体获取用于打分的股票数据。我们将定义一个名为 `get_stocks` 的函数来完成数据查询与处理。

![](img/8ab15f4a8d80e3f0db7a9fa6c93377ab_69.png)

![](img/8ab15f4a8d80e3f0db7a9fa6c93377ab_71.png)

![](img/8ab15f4a8d80e3f0db7a9fa6c93377ab_73.png)

![](img/8ab15f4a8d80e3f0db7a9fa6c93377ab_75.png)

以下是 `get_stocks` 函数的核心步骤：
1.  **构建查询语句**：指定需要获取的财务指标。
2.  **应用筛选器**：将股票范围限定在沪深300池内。
3.  **获取数据**：执行查询并获取 `DataFrame` 格式的数据。
4.  **数据转置**：调整数据格式，便于后续按股票进行索引和计算。

```python
def get_stocks(context):
    # 1. 构建查询语句 (query)，获取“越高越好”的指标
    q = query(
        fundamentals.eod_derivative_indicator.eps, # 每股收益 (EPS)，越高越好
        fundamentals.financial_indicator.roe, # 净资产收益率 (ROE)，越高越好
        fundamentals.financial_indicator.roa # 总资产收益率 (ROA)，越高越好
    ).filter(
        fundamentals.stockcode.in_(context.hs300) # 2. 筛选股票池
    )
    # 3. & 4. 获取数据并转置
    df_up = get_fundamentals(q).T

    # 用同样的方法获取“越低越好”的指标
    q = query(
        fundamentals.financial_indicator.debt_to_asset_ratio, # 资产负债率，越低越好
        fundamentals.eod_derivative_indicator.pb_ratio, # 市净率 (PB)，越低越好
        fundamentals.eod_derivative_indicator.market_cap # 总市值，越低越好
    ).filter(
        fundamentals.stockcode.in_(context.hs300)
    )
    df_down = get_fundamentals(q).T

    return df_up, df_down
```

![](img/8ab15f4a8d80e3f0db7a9fa6c93377ab_77.png)

**代码关键点说明**：
*   `query(...)`：用于构建对财务数据库的查询语句。
*   `.filter(...)`：用于对查询结果进行筛选，这里限定股票代码在 `context.hs300` 池中。
*   `get_fundamentals(q)`：执行查询，返回原始数据。
*   `.T`：对返回的 `DataFrame` 进行转置操作，将行变为列，列变为行，使得每行数据对应一只股票的所有指标，方便后续处理。
*   `df_up` 和 `df_down`：分别存储“越高越好”和“越低越好”两类指标的 `DataFrame`。
