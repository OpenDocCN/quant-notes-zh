# 量化交易与Python金融分析实战：P70：凸函数定义与常见函数凸性分析（2）

在本节课中，我们将继续深入学习凸函数的定义，并运用定义来分析一系列常见函数的凸性。我们将重点讨论指数函数、幂函数、绝对值幂函数、对数函数、负熵、范数以及极大值函数等，并学习如何根据函数的不同特性选择合适的定义或条件来证明其凸性。

## 指数函数的凸性

上一节我们介绍了凸函数的基本定义。本节中，我们来看看指数函数的凸性。

![](img/116a00295fa12a7f15c58be26370f327_1.png)

对于函数 **f(x) = e^(ax)**，其中 **a** 是任意实数。这个函数对于任意实数 **a** 都是凸函数。

由于 **a** 是任意的，无法通过逐个画图验证，因此我们使用二阶条件（即二阶导数非负）来证明。

以下是证明步骤：
1.  计算一阶导数：**f'(x) = a * e^(ax)**
2.  计算二阶导数：**f''(x) = a^2 * e^(ax)**
3.  分析二阶导数：因为 **a^2 ≥ 0** 且 **e^(ax) > 0** 恒成立，所以 **f''(x) ≥ 0** 恒成立。

![](img/116a00295fa12a7f15c58be26370f327_3.png)

因此，函数 **f(x) = e^(ax)** 是凸函数。

![](img/116a00295fa12a7f15c58be26370f327_4.png)

## 幂函数的凸性

接下来，我们分析幂函数 **f(x) = x^a** 的凸性，其中 **x > 0**（为保证在实数域内定义良好），**a** 为实数。

我们同样使用二阶条件进行分析。

以下是分析过程：
1.  计算一阶导数：**f'(x) = a * x^(a-1)**
2.  计算二阶导数：**f''(x) = a(a-1) * x^(a-2)**
3.  讨论二阶导数的符号：
    *   当 **a ≤ 0** 或 **a ≥ 1** 时，**a(a-1) ≥ 0**，且 **x^(a-2) > 0**，因此 **f''(x) ≥ 0**，函数为凸函数。
    *   当 **0 < a < 1** 时，**a(a-1) < 0**，且 **x^(a-2) > 0**，因此 **f''(x) ≤ 0**，函数为凹函数。
    *   当 **a = 1** 时，函数为仿射函数 **f(x) = x**，既是凸函数也是凹函数。
    *   当 **a = 0** 时，函数为常数 **f(x) = 1**，既是凸函数也是凹函数。

所以，幂函数的凸性取决于指数 **a** 的取值。

## 绝对值幂函数的凸性

为了将定义域扩展到整个实数域，我们考虑绝对值幂函数 **f(x) = |x|^p**，其中 **p** 为实数（通常 **p > 0** 且 **p ≠ 1** 以避免在 **x=0** 处不可导的问题）。

这个函数在 **x=0** 处可能不可导，因此不能直接使用二阶条件。我们的策略是：先尝试求导，如果可导则用二阶条件；如果不可导，则回归凸函数的第一定义。

以下是分析思路：
1.  当 **x > 0** 时，**f'(x) = p * x^(p-1)**, **f''(x) = p(p-1) * x^(p-2)**。
2.  当 **x < 0** 时，**f(x) = (-x)^p**, **f'(x) = -p * (-x)^(p-1)**, **f''(x) = p(p-1) * (-x)^(p-2)**。
3.  分析：
    *   当 **p > 1** 时，在整个定义域上（除原点需单独考虑连续性），**f''(x) ≥ 0**，函数是凸的。实际上，当 **p ≥ 2** 时函数在原点也是光滑的；当 **1 < p < 2** 时，原点处一阶可导但二阶不可导，凸性由第一定义保证。
    *   当 **p = 1** 时，函数为 **f(x) = |x|**。它在原点不可导，但通过凸函数的第一定义（任意两点连线在函数图像上方）可以证明它是凸函数。
    *   当 **p < 1** 时，情况复杂，没有统一的凸性或凹性。例如：
        *   **p=0** 时（定义 **0^0=1**），函数为常数1，既是凸的也是凹的。
        *   **p=0.5** 时，函数图像在 **x≥0** 部分是凹的，在 **x≤0** 部分也是凹的，但整体既不是凸函数也不是凹函数。

结论：绝对值幂函数 **f(x) = |x|^p** 在 **p ≥ 1** 时是凸函数；当 **p < 1** 时，其凸性需要针对具体的 **p** 值单独讨论。

## 对数函数与负熵的凸性

![](img/116a00295fa12a7f15c58be26370f327_10.png)

![](img/116a00295fa12a7f15c58be26370f327_12.png)

现在，我们来看对数函数及其相关的一个重要函数——负熵。

首先是对数函数 **f(x) = log(x)**，定义域为 **x > 0**。其图像直观上是凹的。

![](img/116a00295fa12a7f15c58be26370f327_14.png)

我们用二阶条件证明：
1.  一阶导数：**f'(x) = 1/x**
2.  二阶导数：**f''(x) = -1/(x^2)**
3.  由于 **f''(x) < 0** 恒成立，所以对数函数是严格凹函数。

![](img/116a00295fa12a7f15c58be26370f327_16.png)

接下来是负熵函数 **f(x) = x * log(x)**，定义域为 **x > 0**。这个函数在信息论中非常重要。

![](img/116a00295fa12a7f15c58be26370f327_18.png)

我们用二阶条件分析其凸性：
1.  一阶导数：**f'(x) = log(x) + 1**
2.  二阶导数：**f''(x) = 1/x**
3.  由于 **x > 0**，所以 **f''(x) > 0** 恒成立。因此，负熵函数是严格凸函数。

![](img/116a00295fa12a7f15c58be26370f327_20.png)

有趣的是，对数函数（凹函数）乘以 **x** 后，变成了凸函数（负熵）。在信息论中，经常需要最大化熵（即最小化负熵），而最小化一个凸函数是相对容易的优化问题，这解释了为什么“最大熵原理”被广泛使用。

![](img/116a00295fa12a7f15c58be26370f327_22.png)

## 范数的凸性

![](img/116a00295fa12a7f15c58be26370f327_24.png)

范数是定义在向量空间上的函数，满足非负性、齐次性和三角不等式。例如，向量的2-范数（欧几里得范数）**||x||_2**。

对于一般的范数 **p(x)**，它可能无法写出具体的表达式或求导，因此我们使用凸函数的第一定义来证明其凸性。

证明如下：
任取向量 **x, y** 和标量 **θ ∈ [0, 1]**。
1.  构造凸组合：**θx + (1-θ)y**
2.  应用范数的三角不等式：**p(θx + (1-θ)y) ≤ p(θx) + p((1-θ)y)**
3.  应用范数的齐次性：**p(θx) = |θ| p(x) = θ p(x)**，同理 **p((1-θ)y) = (1-θ) p(y)**。因为 **θ** 和 **(1-θ)** 非负，绝对值可去掉。
4.  因此，**p(θx + (1-θ)y) ≤ θ p(x) + (1-θ) p(y)**。

这正好满足凸函数的第一定义。所以，任何范数都是凸函数。

需要特别注意的是“零范数”（**||x||_0**），它表示向量中非零元素的个数。虽然名字里有“范数”，但它不满足齐次性（例如，**||2*x||_0** 不一定等于 **2*||x||_0**），因此它不是真正的范数，也不是凸函数。这也是在压缩感知等领域中，常用1-范数代替0-范数进行优化求解的原因。

## 极大值函数及其光滑近似

最后，我们讨论极大值函数 **f(x) = max{x1, x2, ..., xn}**，其中 **x** 是n维向量。这个函数不可导。

我们使用凸函数的第一定义证明其凸性：
任取向量 **x, y** 和标量 **θ ∈ [0, 1]**。
1.  **f(θx + (1-θ)y) = max_i { θx_i + (1-θ)y_i }**
2.  对于每个分量 **i**，有 **θx_i + (1-θ)y_i ≤ θ * max_i{x_i} + (1-θ) * max_i{y_i}**。
3.  因此，**max_i { θx_i + (1-θ)y_i } ≤ θ * max_i{x_i} + (1-θ) * max_i{y_i} = θ f(x) + (1-θ) f(y)**。

所以，极大值函数是凸函数。优化问题中的“极小化极大问题”（minimax）之所以被研究，正是因为极大值函数是凸的，使得问题具有一些良好的性质。

![](img/116a00295fa12a7f15c58be26370f327_26.png)

由于极大值函数不可导，优化起来困难。实践中常用一个光滑的可导函数来近似它，称为 **Log-Sum-Exp** 函数：
**LSE(x) = log( sum_{i=1}^n exp(x_i) )**

![](img/116a00295fa12a7f15c58be26370f327_28.png)

这个函数具有以下性质：
1.  它是光滑可导的。
2.  它是极大值函数的一个上界近似：**max{x_i} ≤ LSE(x) ≤ max{x_i} + log(n)**。
3.  它本身也是凸函数。可以通过计算其Hessian矩阵（是半正定的）来证明，尽管过程稍复杂。

![](img/116a00295fa12a7f15c58be26370f327_30.png)

因此，**Log-Sum-Exp** 函数是处理涉及极大值操作时一个非常实用的凸光滑近似。

![](img/116a00295fa12a7f15c58be26370f327_32.png)
