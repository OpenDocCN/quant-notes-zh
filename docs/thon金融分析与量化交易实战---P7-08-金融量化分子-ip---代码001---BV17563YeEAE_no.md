## 魔术命令简介
上一节我们介绍了IPython的基础环境。本节中，我们来看看IPython提供的一系列以百分号（`%`）开头的特殊指令，它们被称为“魔术命令”。这些命令为交互式编程提供了强大的辅助功能。

魔术命令可以直接在IPython的交互式命令行中使用，无需退出当前环境，就能完成诸如运行外部脚本、管理代码片段、测试代码性能等操作。

## 运行外部脚本：`%run`
在标准的Python命令行中，如果想运行一个已保存的`.py`文件，通常需要退出命令行，然后使用`python filename.py`这样的命令。在IPython中，我们可以使用`%run`命令直接运行，非常方便。

以下是具体操作步骤：
1.  首先，我们创建一个简单的Python脚本文件，例如`hello.py`，内容如下：
    ```python
    # hello.py
    print("Hello, World!")
    ```
2.  在IPython交互环境中，使用`%run`命令加上文件名来执行它：
    ```python
    %run hello.py
    ```
    执行后，会输出`Hello, World!`。这意味着我们可以在不离开IPython环境的情况下，轻松运行和测试独立的脚本文件。

## 粘贴并执行代码：`%paste`
在编写代码时，我们经常需要从编辑器或其他地方复制一段代码到交互环境中进行测试。直接粘贴可能会因为缩进（如Tab键）等问题导致错误。`%paste`魔术命令可以很好地解决这个问题。

![](img/c1fc60422d52aa9e6026d13a8e09c761_11.png)

`%paste`命令会读取系统剪贴板中的内容，将其作为代码块执行，并自动处理常见的格式问题。

![](img/c1fc60422d52aa9e6026d13a8e09c761_13.png)

以下是使用`%paste`的例子：
1.  假设我们在一个文本编辑器中编写了以下函数，并复制到了剪贴板：
    ```python
    def say_hello(name):
        greeting = f"Hello, {name}!"
        return greeting

    print(say_hello("Quant"))
    ```
2.  在IPython中，我们不需要手动粘贴，直接输入：
    ```python
    %paste
    ```
    命令会先打印出剪贴板中的代码，然后执行它，输出结果为`Hello, Quant!`。这对于测试较长的代码片段非常有用。

## 测量代码执行时间：`%timeit`
在性能分析和优化时，我们经常需要测量某段代码或函数的执行时间。虽然可以使用Python的`time`模块，但对于执行速度极快的代码，单次测量可能不准确（结果可能为0秒）。`%timeit`命令通过多次运行代码并计算平均耗时，提供了更精确的测量。

`%timeit`命令适用于评估小段代码或简单操作的性能。

以下是使用`%timeit`的例子：
1.  我们首先生成一个包含1万个随机数的列表：
    ```python
    import random
    my_list = [random.randint(1, 10000) for _ in range(10000)]
    ```
2.  如果我们想测量Python内置的`sort()`方法对这个列表排序所需的时间，可以这样做：
    ```python
    %timeit my_list.sort()
    ```
    命令会输出类似`146 µs ± 742 ns per loop`的结果。这表示排序操作平均耗时146微秒，并给出了误差范围。`%timeit`会自动决定循环运行的次数（例如“7次运行，每次循环100万次”），以确保测量结果的统计意义，这对于优化高频调用的代码至关重要。

## 总结
本节课中我们一起学习了IPython中三个非常实用的魔术命令。
*   `%run`命令允许我们在交互式环境中直接运行外部的Python脚本文件。
*   `%paste`命令能够智能地执行剪贴板中的代码，避免手动粘贴可能带来的格式错误。
*   `%timeit`命令则为代码性能分析提供了高精度的测量工具，通过多次运行取平均值来获得可靠的结果。
