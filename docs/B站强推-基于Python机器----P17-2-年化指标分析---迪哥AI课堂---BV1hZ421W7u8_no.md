## 回测收益率 📊

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_18.png)

回测收益率是评估交易策略最直接的指标。它表示在整个回测周期结束后，你的初始资金最终变成了多少，即总收益与初始本金的比值。

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_20.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_22.png)

以下是计算回测收益率的核心概念：
*   **公式**：`总收益率 = (期末总资产 - 期初总资产) / 期初总资产`
*   **代码表示**：在Python中，如果我们有一个代表资产净值的序列 `net_value`，那么总收益率 `total_return` 可以计算为：
    ```python
    total_return = (net_value[-1] - net_value[0]) / net_value[0]
    ```

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_24.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_26.png)

这个指标直观地告诉你策略是盈利还是亏损，以及盈利或亏损的幅度。例如，一个结果为1.13的收益率意味着你的资产翻了一倍多（增长113%），而一个负值则代表亏损。

## 年化收益率 🔄

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_32.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_34.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_36.png)

理解了策略的总收益后，我们还需要考虑时间因素。年化收益率将不同时间长度的总收益标准化，转化为一个假设每年都能获得的平均收益率，这使得不同周期的策略之间具有可比性。

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_38.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_40.png)

以下是年化收益率的计算公式：
*   **公式**：`年化收益率 = (1 + 总收益率)^(250 / N) - 1`
    *   `总收益率`：即上面计算出的回测总收益率。
    *   `N`：回测周期的总交易日天数。
    *   `250`：通常假设一年有250个交易日（市场工作日）。
*   **代码表示**：假设我们已有总收益率 `total_return` 和交易日天数 `N`，计算年化收益率 `annual_return`：
    ```python
    trading_days_per_year = 250
    annual_return = (1 + total_return) ** (trading_days_per_year / N) - 1
    ```

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_42.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_44.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_46.png)

年化收益率是业界更常用来衡量和比较策略表现的指标。它能更公平地反映策略的盈利能力。例如，一个耗时3年获得113%总收益的策略，其年化收益率约为27%，这比简单地报告总收益更能说明其每年的平均表现。

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_48.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_50.png)

![](img/00e5a7275a00e6f4d3826e1f4d1d5fdb_52.png)

本节课中我们一起学习了量化策略收益分析的两个基石：**回测收益率**和**年化收益率**。回测收益率告诉我们策略的最终盈亏结果，而年化收益率则考虑了时间价值，为我们提供了标准化的年度盈利能力参考。它们是评估任何交易策略绩效时首要关注的核心指标。
