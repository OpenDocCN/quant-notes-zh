## 量化分析核心流程

上一节我们介绍了课程资料，本节中我们来看看进行量化交易分析需要遵循的核心步骤。一个完整的量化分析流程通常包含以下几个环节：

1.  **数据获取**：获取股票的历史行情、实时数据等原始信息。
2.  **数据准备与清洗**：处理原始数据中的缺失值、异常值，使其规范化。
3.  **指标建立**：基于原始数据计算各类技术分析指标，如均线、RSI等。
4.  **模型设计**：构建数学模型，用于实现“智能择股”（选择股票）或“智能择时”（选择买卖时机）。
5.  **策略制定**：明确具体的交易规则，例如在何种指标条件下买入或卖出。
6.  **回测测试**：使用历史数据模拟运行策略，检验其盈利能力和稳定性。
7.  **评估与改进**：分析回测结果，优化模型和策略参数。

Tushare是一个免费、开源的财经数据接口包，它为我们完成了数据的采集、清洗和存储工作，返回的数据格式多为Pandas的`DataFrame`，非常便于分析。


在Python代码中，我们这样导入并使用它：

```python
import tushare as ts
```

### 获取历史K线数据

我们可以使用 `get_k_data()` 函数获取股票的历史日K线数据。

```python
# 获取万科A（000002）从2023-01-01到2023-12-31的数据
df = ts.get_k_data('000002', start='2023-01-01', end='2023-12-31')
# 查看前5行数据
print(df.head())
```

这段代码会返回一个包含日期（`date`）、开盘价（`open`）、收盘价（`close`）、最高价（`high`）、最低价（`low`）、成交量（`volume`）等字段的表格。

### 获取实时行情数据

要获取股票的实时数据，可以使用 `get_realtime_quotes()` 函数。

```python
# 获取单只股票实时数据
real_time_df = ts.get_realtime_quotes('000002')
# 筛选我们关心的字段
print(real_time_df[['code', 'name', 'price', 'bid', 'ask', 'volume', 'amount', 'time']])

# 获取多只股票实时数据
codes = ['000002', '000001', '000858']
multi_real_time_df = ts.get_realtime_quotes(codes)
```

## 计算基础衍生变量

获取基础数据后，我们可以基于它们计算一些简单的衍生指标。这些指标反映了价格和成交量的基本变化。

以下是几个基础衍生变量的计算示例：

```python
# 假设df是我们获取的K线数据DataFrame
# 1. 涨跌幅（(收盘-开盘)/开盘）
df['close_open_pct'] = (df['close'] - df['open']) / df['open']

# 2. 振幅（(最高-最低)/最低）
df['high_low_pct'] = (df['high'] - df['low']) / df['low']

# 3. 昨日收盘价（将收盘价向上平移一行）
df['pre_close'] = df['close'].shift(1)

# 4. 价格变动（今日收盘 - 昨日收盘）
df['price_change'] = df['close'] - df['pre_close']

# 5. 价格变动百分比
df['p_change'] = (df['price_change'] / df['pre_close']) * 100
```

## 计算常用技术指标

除了基础变量，量化分析中更常用到一系列经典技术指标。手动计算它们非常复杂，我们可以借助专门的库，例如 **TA-Lib**。

> **注意**：TA-Lib是一个广泛使用的技术分析库，但其原始安装可能稍复杂。有时需要下载预编译的包或从特定源安装，例如：`pip install TA_Lib‑0.4.24‑cp39‑cp39‑win_amd64.whl` （文件名因系统和Python版本而异）。

### 相对强弱指标 (RSI)

RSI用于衡量近期涨势相对于跌势的强弱。值越大（通常>70为超买），涨势越强；值越小（通常<30为超卖），跌势越强。

**公式**：`RSI = 100 - 100 / (1 + RS)`，其中 `RS = N日内上涨幅度平均值 / N日内下跌幅度平均值`。

使用TA-Lib计算12日RSI非常简单：

```python
import talib
df['rsi_12'] = talib.RSI(df['close'], timeperiod=12)
```

### 动量指标 (MOM)

动量指标衡量一段时间内股价的变化速度。

**公式**：`MOM = 当日收盘价 - N日前收盘价`

计算5日动量：

```python
df['mom_5'] = talib.MOM(df['close'], timeperiod=5)
```

### 指数移动平均线 (EMA)

EMA是对收盘价进行加权平均的指标，对近期价格赋予更高权重，比简单移动平均线反应更快。

**公式**：`今日EMA = α * 今日收盘价 + (1 - α) * 昨日EMA`，其中 `α = 2 / (N+1)`。

计算12日和26日EMA：

```python
df['ema_12'] = talib.EMA(df['close'], timeperiod=12)
df['ema_26'] = talib.EMA(df['close'], timeperiod=26)
```

### 异同移动平均线 (MACD)

MACD由快线（DIF）、慢线（DEA）和柱状图（MACD）组成，用于判断趋势转折和强度。

**注意**：TA-Lib中`MACD`函数返回的三个值`macd`, `macdsignal`, `macdhist`，分别对应国内的**DIF**、**DEA**和**MACD柱（即(DIF-DEA)*2）**。

计算MACD（参数12，26，9为常用设置）：

```python
df['macd_dif'], df['macd_dea'], df['macd_hist'] = talib.MACD(df['close'],
                                                              fastperiod=12,
                                                              slowperiod=26,
                                                              signalperiod=9)
```

## 数据清洗与整理

在计算了众多指标后，数据开头部分会因为计算窗口（如5日均线需要前5天数据）而存在空值（NaN）。我们需要将其清洗掉。

```python
# 删除含有空值的行
df_clean = df.dropna()
# 查看清洗后的数据
print(df_clean.head())
```
