## 概述

上一节我们介绍了Prophet模型的基本构建与预测流程。本节中，我们来看看模型中的一个核心可调参数——`changepoint_prior_scale`。这个参数控制着模型对历史数据中突变趋势的敏感程度，对预测结果有显著影响。我们将通过代码实验，直观地理解其作用并掌握调参方法。

## 核心参数：突变点权重

`changepoint_prior_scale`参数指定了突变点的先验权重。其含义是：**赋予突变点较大的权重，模型会更重视训练数据中的突变趋势**。

公式表示为：模型对突变点的重视程度 ∝ `changepoint_prior_scale`

这个参数设置存在一个权衡：
*   **权重值较大**：模型会更努力地拟合训练数据中的波动，包括突变点。这可能导致**过拟合**，即模型过于贴合训练数据，而在未知数据上表现不佳。
*   **权重值较小**：模型对突变点不敏感，趋势预测会更为平缓和保守。这可能导致**欠拟合**，即模型未能充分学习数据中的有效模式。

Prophet框架中，此参数的默认值为 **0.05**，这是一个相对较小的值，表明模型默认对突变趋势持保守态度。

## 参数影响实验

为了直观展示不同`changepoint_prior_scale`值的影响，我们进行以下实验。以下是实验的核心代码逻辑：

```python
# 定义不同的参数值进行测试
cp_scale_list = [0.001, 0.05, 0.1, 0.2]
predictions = {} # 用于存储不同参数下的预测结果

for scale in cp_scale_list:
    # 1. 使用当前参数创建模型
    model = Prophet(changepoint_prior_scale=scale)
    model.fit(train_df)
    
    # 2. 构建未来时间帧并进行预测
    future = model.make_future_dataframe(periods=180)
    forecast = model.predict(future)
    
    # 3. 保存预测结果
    predictions[scale] = forecast[‘yhat’]
```

以下是不同参数值对应的预测曲线与真实数据（黑点）的对比分析：

*   **蓝色线 (0.001)**：参数值极小。模型几乎忽略了所有的突变点，预测出一条非常平缓的直线，未能捕捉训练集中的下降和上升趋势，呈现明显的**欠拟合**。
*   **红色线 (0.05)**：默认参数。模型捕捉到部分趋势，但依然比较保守，对某些突变（如峰值）的反应不够充分。
*   **灰色/黄色线 (0.1, 0.2)**：参数值较大。模型紧密地跟随了训练数据的波动，包括各个突变点。预测趋势更加激进（例如，对上升趋势预测得更高）。这虽然对历史数据拟合更好，但**过拟合**风险增加。

## 量化评估与参数选择

仅观察拟合曲线不够客观，我们需要量化评估不同参数在测试集上的表现。我们计算并比较不同参数下的训练误差（`train_error`）和测试误差（`test_error`）。

```python
# 计算误差示例 (此处为简化逻辑)
errors = {}
for scale in cp_scale_list:
    train_pred = predictions[scale][:len(train_df)]
    test_pred = predictions[scale][len(train_df):]
    
    train_error = mean_absolute_error(train_df[‘y’], train_pred)
    test_error = mean_absolute_error(test_df[‘y’], test_pred)
    
    errors[scale] = {‘train’: train_error, ‘test’: test_error}
```

观察误差结果，我们可以发现规律：
*   **训练误差**：随着`changepoint_prior_scale`增大，模型对训练数据拟合得更好，因此训练误差持续下降。
*   **测试误差**：测试误差会先随着参数增大而下降（模型能力提升），但超过某个最优值后可能回升（过拟合导致泛化能力下降）。

**参数选择策略**：我们应选择使**测试误差最小**的那个`changepoint_prior_scale`值作为最终模型参数。例如，在实验中，当参数从0.2继续增大到0.7时，测试误差从127.6下降至66左右，此时0.7即为更优的选择。

## 模型调优后验证

使用优化后的参数（例如0.7）重新训练模型并进行预测，与默认参数结果对比：
*   **预测值**：1263
*   **真实值**：1294
预测误差相比使用默认参数时有所减小，说明调参有效提升了模型的预测准确性。
