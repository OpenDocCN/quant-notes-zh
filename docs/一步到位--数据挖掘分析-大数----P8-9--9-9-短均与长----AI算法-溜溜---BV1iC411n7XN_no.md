## 计算短期移动平均线

![](img/acefe9542a5a88bf0733998ee5e0c187_8.png)

![](img/acefe9542a5a88bf0733998ee5e0c187_10.png)

首先，我们需要计算一个短期移动平均线。移动平均线是分析股票趋势的常用指标，它通过计算指定时间窗口内股价的平均值来平滑价格波动。

通常情况下，短期移动平均线使用5个交易日（即一周）的数据。但由于我们的示例数据时间跨度较长（2010年至2018年），为了更清晰地展示效果，我们将适当增大窗口。我们先以10天作为短期窗口进行计算。

以下是计算短期移动平均线的代码：

![](img/acefe9542a5a88bf0733998ee5e0c187_12.png)

![](img/acefe9542a5a88bf0733998ee5e0c187_14.png)

```python
# 假设 df 是包含苹果股票（‘AAPL’）价格的数据框
# 计算10日短期移动平均线，并命名为‘M1’
df[‘M1’] = df[‘AAPL’].rolling(window=10).mean()
```

![](img/acefe9542a5a88bf0733998ee5e0c187_16.png)

![](img/acefe9542a5a88bf0733998ee5e0c187_18.png)

这段代码对苹果股票的价格序列取了一个窗口大小为10的滚动平均值。

![](img/acefe9542a5a88bf0733998ee5e0c187_20.png)

## 计算长期移动平均线

![](img/acefe9542a5a88bf0733998ee5e0c187_22.png)

接下来，我们计算长期移动平均线。长期均线能反映更稳定的趋势，常见的窗口有20日或30日。这里我们使用30日作为长期窗口。

![](img/acefe9542a5a88bf0733998ee5e0c187_24.png)

![](img/acefe9542a5a88bf0733998ee5e0c187_26.png)

以下是计算长期移动平均线的代码：

![](img/acefe9542a5a88bf0733998ee5e0c187_28.png)

```python
# 计算30日长期移动平均线，并命名为‘M2’
df[‘M2’] = df[‘AAPL’].rolling(window=30).mean()
```

现在，我们的数据框中包含了原始股价、10日短期均线（M1）和30日长期均线（M2）三列数据。

![](img/acefe9542a5a88bf0733998ee5e0c187_30.png)

观察图表，我们可以识别两种重要的交叉信号：
1.  **死亡交叉**：当短期均线从上方向下穿越长期均线时，通常被视为下跌信号，提示可能不适合买入或应考虑卖出。
2.  **黄金交叉**：当短期均线从下方向上穿越长期均线时，通常被视为上涨信号，提示可能是一个买入机会。

## 生成交易信号

![](img/acefe9542a5a88bf0733998ee5e0c187_58.png)

![](img/acefe9542a5a88bf0733998ee5e0c187_60.png)

为了更自动化地识别交叉点，我们可以创建一个“头寸”信号列。该信号明确指示短期均线相对于长期均线的状态。

![](img/acefe9542a5a88bf0733998ee5e0c187_62.png)

以下是使用NumPy的 `where` 函数生成交易信号的代码：

```python
import numpy as np

![](img/acefe9542a5a88bf0733998ee5e0c187_64.png)

![](img/acefe9542a5a88bf0733998ee5e0c187_66.png)

# 生成交易信号：当短期均线大于长期均线时，标记为1；否则标记为-1
df[‘position’] = np.where(df[‘M1’] > df[‘M2’]， 1， -1)
```

![](img/acefe9542a5a88bf0733998ee5e0c187_68.png)

这个新列 `position` 的值为1或-1，分别代表“看涨”和“看跌”状态。

为了清晰展示，我们可以将信号绘制在独立的右侧坐标轴上：

![](img/acefe9542a5a88bf0733998ee5e0c187_70.png)

```python
# 创建主坐标轴和右侧坐标轴进行绘图
ax = df[[‘AAPL‘， ‘M1‘， ‘M2‘]].plot(figsize=(12， 6))
df[‘position’].plot(ax=ax， secondary_y=True， style=‘g‘) # 在右侧坐标轴绘制信号线
plt.show()
```

![](img/acefe9542a5a88bf0733998ee5e0c187_72.png)

![](img/acefe9542a5a88bf0733998ee5e0c187_74.png)

![](img/acefe9542a5a88bf0733998ee5e0c187_76.png)

在最终的图表中，右侧坐标轴的绿色信号线在1和-1之间切换。**从-1向上突破到1的转折点对应黄金交叉，从1向下跌破到-1的转折点则对应死亡交叉**，这使得买卖信号的识别变得一目了然。
