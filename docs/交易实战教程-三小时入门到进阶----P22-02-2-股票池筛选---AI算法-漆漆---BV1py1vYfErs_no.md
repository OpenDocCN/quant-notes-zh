## 构建筛选逻辑
我们的目标是：从沪深300成分股中，查询每只股票的营业收入，然后按收入降序排列，最终选出前十名。

以下是具体的代码实现步骤：

首先，导入必要的模块并构建查询。我们查询的指标是“营业收入”。
```python
# 导入查询模块
from jqdata import *
```

然后，在策略的 `before_trading` 函数中编写筛选逻辑。
```python
def before_trading(context):
    # 构建查询语句
    q = query(
        indicator.operating_revenue  # 查询营业收入指标
    ).filter(
        # 过滤条件：股票代码必须在沪深300成分股中
        indicator.code.in_(get_index_stocks(‘000300.XSHG‘))
    ).order_by(
        # 按营业收入降序排列
        indicator.operating_revenue.desc()
    ).limit(
        10  # 限制返回前10条结果
    )
```

查询完成后，我们将结果（一个DataFrame）保存到 `context` 中，方便后续使用。
```python
    # 执行查询
    df = get_fundamentals(q)
    # 将结果存入context
    context.selected_stocks = df
    # 打印结果进行验证
    print(df)
```

2.  确认指数代码。可以在帮助文档的“指数”部分查找“沪深300”对应的正确代码。
