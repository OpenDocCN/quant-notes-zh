## 图表操作回顾 📊

![](img/ff4c8db2844c2ff6849f5512deec7825_10.png)

上一节我们介绍了多种图表类型及其适用场景。图表是数据可视化的重要工具，能够帮助我们直观地理解数据。

以下是常见的图表类型及其核心用途：

![](img/ff4c8db2844c2ff6849f5512deec7825_12.png)

*   **柱状图**：在垂直方向对比不同类型的数据。
    *   **公式/代码示例**：`plt.bar(x, height)` (使用matplotlib)
*   **条形图**：在水平方向对比不同类型的数据。
*   **折线图**：反映一段时间内某一或多个数据序列的变化趋势。
*   **散点图**：反映两种数据类型之间的相关关系。
    *   **公式/代码示例**：`plt.scatter(x, y)`
*   **气泡图**：在散点图基础上增加气泡大小维度，可反映三种数据类型间的联系。
*   **饼图**：体现部分与整体的占比关系。
*   **雷达图**：反映不同类型在多个维度上的综合对比分析。
*   **面积图**：通过面积大小进行不同类型数据的对比。
*   **组合图**：结合两种及以上图表类型，用于展示量级差异巨大的多类数据。

## Excel数据分析实例回顾 📈

![](img/ff4c8db2844c2ff6849f5512deec7825_20.png)

在掌握了图表工具后，我们通过一个Excel实例学习了完整的数据分析流程。这个流程是数据分析项目的通用框架。

![](img/ff4c8db2844c2ff6849f5512deec7825_22.png)

![](img/ff4c8db2844c2ff6849f5512deec7825_24.png)

![](img/ff4c8db2844c2ff6849f5512deec7825_26.png)

数据分析的核心步骤如下：

![](img/ff4c8db2844c2ff6849f5512deec7825_28.png)

![](img/ff4c8db2844c2ff6849f5512deec7825_29.png)

![](img/ff4c8db2844c2ff6849f5512deec7825_30.png)

1.  **明确问题/需求**：理解业务方需要解决的具体问题。
2.  **确定分析思路**：规划从哪些维度、采用何种方法进行分析。
3.  **数据获取与处理**：获取原始数据，并进行数据清洗。
4.  **数据分析**：基于清洗后的数据，执行分析计算。
5.  **数据可视化**：将分析结果通过图表等形式呈现。
6.  **报告撰写与汇报**：形成分析报告并进行成果汇报。
7.  **提出建议与跟进**：给出业务建议，并跟踪落地效果。

其中，**数据清洗**是至关重要的一环，直接影响到分析结果的准确性。

数据清洗通常包含以下步骤，但并非必须按顺序严格执行，需根据实际情况调整：

![](img/ff4c8db2844c2ff6849f5512deec7825_32.png)

*   **选择子集**：选取与分析相关的数据列。
*   **列重命名**：将不合适的列名修改为清晰易懂的名称。
    *   **代码示例** (Pandas)：`df.rename(columns={'旧列名': '新列名'})`
*   **删除重复值**：识别并删除数据表中的重复行。
    *   **代码示例** (Pandas)：`df.drop_duplicates()`
*   **缺失值处理**：处理数据中的空值（NaN）。
    *   **直接删除**：若缺失数据量小且不影响整体，可直接删除。
    *   **人为修改**：对于文本型且数量不大的缺失，可根据其他列信息推断填充。
    *   **统计值填充**：对于数值型且缺失量大的情况，常用**均值**或**中位数**填充。当数据离散程度大时，优先使用**中位数**。
*   **一致化处理**：将数据格式、单位等统一规范。例如，将文本型数字区间转换为可计算的数值。
*   **异常值处理**：识别并处理明显偏离正常范围的数据点。
    *   **文本型**：根据业务逻辑进行人为判断与删除。
    *   **数值型**：通过统计描述（如最大值、最小值）或业务经验判断，剔除可能影响分析结果的极端值。
