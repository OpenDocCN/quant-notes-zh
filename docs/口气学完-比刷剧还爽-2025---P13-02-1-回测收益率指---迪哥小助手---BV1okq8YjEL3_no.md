## 策略评估指标简介
上一节我们介绍了策略回测的基本概念。当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以判断策略的盈亏情况。然而，仅仅知道“赚了”或“赔了”是不够的。

我们需要更精确的指标来回答以下问题：
*   策略具体盈利或亏损了多少？
*   与市场基准（如大盘指数）相比，策略表现如何？
*   策略承担了哪些风险？这些风险是否值得？
*   与其他策略相比，本策略有何优势？

这些问题的答案，就来自于一系列的策略评估指标。本节课，我们将聚焦于第一个核心指标。

## 回测收益率详解
回测收益率是最基本、最直观的策略评估指标。它衡量的是，在回测周期内，策略最终获得的收益相对于初始本金的比率。

例如，在某个策略平台的回测结果中，你可能会看到类似“回测收益：4.25%”的指标。这表示，从回测开始到结束，你的策略使总资产相对于初始本金增长了约4.25%。

![](img/e9318c0e4ef94c20f5a667820851b74f_11.png)

![](img/e9318c0e4ef94c20f5a667820851b74f_13.png)

### 计算公式
回测收益率的计算公式如下：

![](img/e9318c0e4ef94c20f5a667820851b74f_15.png)

![](img/e9318c0e4ef94c20f5a667820851b74f_17.png)

**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

![](img/e9318c0e4ef94c20f5a667820851b74f_19.png)

用公式表示为：
`R = (P_t / P_0) - 1`
其中：
*   `R` 代表回测收益率。
*   `P_t` 代表回测结束时的资产总值（卖出所有头寸后的总现金）。
*   `P_0` 代表回测开始时的初始本金。

![](img/e9318c0e4ef94c20f5a667820851b74f_21.png)

这个指标直接反映了策略在整个回测周期内的绝对盈利能力。

## Python代码实战：计算回测收益率
理论部分我们已经了解，现在让我们通过Python代码，实际计算一下回测收益率。我们将使用一份示例股票数据来模拟策略回测后的资产净值走势。

首先，进行环境准备，导入必要的库并加载数据。

![](img/e9318c0e4ef94c20f5a667820851b74f_25.png)

![](img/e9318c0e4ef94c20f5a667820851b74f_27.png)

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便后续图表显示中文
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设‘date’列是日期，将其设为索引
data = pd.read_csv('your_stock_data.csv', index_col=0)
data.index = pd.to_datetime(data.index) # 确保索引为日期时间类型
print(data.head())
```

假设我们的 `data` DataFrame 中的 `‘close’` 列代表策略在每一天的资产净值（可以理解为每天收盘后账户的总价值）。我们将基于此计算回测收益率。

计算过程分为两步：
1.  计算资产净值相对于期初的比值序列。
2.  根据比值序列计算最终的回测收益率。

以下是具体的计算步骤：

```python
# 第一步：计算每日资产净值相对于期初净值的比值
# 公式为：每日净值 / 期初净值
relative_value = data['close'] / data['close'].iloc[0]

![](img/e9318c0e4ef94c20f5a667820851b74f_29.png)

# 可视化资产净值变化曲线
plt.figure(figsize=(10, 6))
plt.plot(relative_value.index, relative_value.values)
plt.title('策略资产净值相对变化曲线')
plt.xlabel('日期')
plt.ylabel('相对净值 (期初=1)')
plt.grid(True)
plt.show()

![](img/e9318c0e4ef94c20f5a667820851b74f_31.png)

# 第二步：计算最终的回测收益率
# 期末相对净值 = relative_value.iloc[-1]
# 回测收益率 = 期末相对净值 - 1
backtest_return = relative_value.iloc[-1] - 1

![](img/e9318c0e4ef94c20f5a667820851b74f_33.png)

![](img/e9318c0e4ef94c20f5a667820851b74f_35.png)

# 将结果整理成表格形式展示
result_df = pd.DataFrame({
    ‘回测收益率’: [backtest_return]
}, index=[‘数值’])
print(result_df)
```

![](img/e9318c0e4ef94c20f5a667820851b74f_37.png)

![](img/e9318c0e4ef94c20f5a667820851b74f_39.png)

执行以上代码后，你将得到：
1.  一张图表，展示了策略资产净值随时间相对于期初的变化情况。当曲线高于1时，表示盈利；低于1时，表示亏损。
2.  一个表格，清晰地显示出计算得到的回测收益率数值。

**代码解读**：
*   `data[‘close’].iloc[0]` 获取了期初的资产净值。
*   `relative_value` 序列反映了每一天资产相对于期初的成长情况。
*   `relative_value.iloc[-1]` 获取了期末的相对净值。
*   最终的回测收益率 `backtest_return` 就是期末相对净值减去1。
