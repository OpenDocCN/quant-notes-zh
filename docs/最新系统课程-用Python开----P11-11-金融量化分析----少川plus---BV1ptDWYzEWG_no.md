在本节课中，我们将学习如何使用NumPy库创建各种类型的数组。这是进行金融数据分析和量化计算的基础，掌握不同的数组创建方法能帮助我们更高效地准备和处理数据。

上一节我们介绍了NumPy数组（ndarray）的一些常用属性，本节中我们来看看如何创建它们。我们已经知道可以使用`array`方法将列表转换为数组，除此之外，NumPy还提供了多种便捷的函数来创建具有特定内容的数组。

## 创建特定值数组

![](img/05c23e8dcc1079617f6655f24830ad23_1.png)

![](img/05c23e8dcc1079617f6655f24830ad23_3.png)

以下是几种创建填充了特定值的数组的方法。

### 创建全零数组
使用`np.zeros`函数可以创建一个所有元素都为0的数组。默认情况下，创建的是浮点数（`float64`）类型的数组。

```python
import numpy as np
a = np.zeros(10)
print(a.dtype)  # 输出：float64
```

![](img/05c23e8dcc1079617f6655f24830ad23_7.png)

![](img/05c23e8dcc1079617f6655f24830ad23_9.png)

如果想创建整数类型的全零数组，可以通过`dtype`参数指定。

```python
a_int = np.zeros(10, dtype=int)
print(a_int.dtype)  # 输出：int64
```

![](img/05c23e8dcc1079617f6655f24830ad23_11.png)

![](img/05c23e8dcc1079617f6655f24830ad23_13.png)

### 创建全一数组
与`zeros`类似，`np.ones`函数用于创建所有元素都为1的数组。

![](img/05c23e8dcc1079617f6655f24830ad23_15.png)

```python
b = np.ones(5)  # 创建包含5个1.0的数组
b_int = np.ones(5, dtype=int)  # 创建包含5个1的整数数组
```

![](img/05c23e8dcc1079617f6655f24830ad23_17.png)

![](img/05c23e8dcc1079617f6655f24830ad23_19.png)

### 创建“空”数组
`np.empty`函数用于创建一个“空”数组。这里的“空”并非指值为零或None，而是指分配内存后，不进行初始化，数组中的值是内存中残留的、未定义的随机数据。

![](img/05c23e8dcc1079617f6655f24830ad23_21.png)

```python
c = np.empty(100)
print(c)  # 输出内容为随机值
```

![](img/05c23e8dcc1079617f6655f24830ad23_23.png)

![](img/05c23e8dcc1079617f6655f24830ad23_25.png)

这种方法比`zeros`少了一步初始化操作，因此速度稍快。**仅在你确定后续操作会覆盖所有数组元素时使用**，否则残留的随机值可能导致错误。

![](img/05c23e8dcc1079617f6655f24830ad23_27.png)

## 创建序列数组

![](img/05c23e8dcc1079617f6655f24830ad23_29.png)

在数据处理中，经常需要生成有规律的数值序列。NumPy为此提供了强大的工具。

![](img/05c23e8dcc1079617f6655f24830ad23_31.png)

### 使用`arange`函数
`np.arange`函数类似于Python内置的`range`，但功能更强大，可以直接生成NumPy数组。它接受起始值、结束值和步长参数。

![](img/05c23e8dcc1079617f6655f24830ad23_33.png)

```python
# 生成0到99的整数序列
seq1 = np.arange(100)
# 生成10到20，步长为2的序列
seq2 = np.arange(10, 20, 2)
# 步长可以是小数，这是原版range做不到的
seq3 = np.arange(0, 1, 0.1)
```

![](img/05c23e8dcc1079617f6655f24830ad23_35.png)

![](img/05c23e8dcc1079617f6655f24830ad23_37.png)

### 使用`linspace`函数
`np.linspace`函数用于在指定的区间内生成**等间隔**的数值点。它与`arange`的关键区别在于：第三个参数指定的是要生成的**元素数量**，而不是步长。生成的数组**包含**结束值。

![](img/05c23e8dcc1079617f6655f24830ad23_39.png)

![](img/05c23e8dcc1079617f6655f24830ad23_41.png)

```python
# 在0到100之间均匀生成101个点（包含0和100）
linear_seq = np.linspace(0, 100, 101)
```

![](img/05c23e8dcc1079617f6655f24830ad23_43.png)

`linspace`在科学计算中非常有用，例如在绘制函数图像时，需要在定义域内取大量等间隔的点来计算函数值。

![](img/05c23e8dcc1079617f6655f24830ad23_45.png)

![](img/05c23e8dcc1079617f6655f24830ad23_47.png)

```python
# 在-10到10之间取10000个点，用于绘制y=x^2的图像
x = np.linspace(-10, 10, 10000)
y = x ** 2  # 计算y值
# （此处可配合matplotlib进行绘图）
```

![](img/05c23e8dcc1079617f6655f24830ad23_49.png)

![](img/05c23e8dcc1079617f6655f24830ad23_51.png)

## 创建特殊矩阵

最后，我们介绍一个在线性代数中常用的函数。

### 创建单位矩阵
`np.eye`函数用于生成单位矩阵（Identity Matrix），即主对角线元素为1，其余元素为0的方阵。

![](img/05c23e8dcc1079617f6655f24830ad23_55.png)

![](img/05c23e8dcc1079617f6655f24830ad23_57.png)

```python
# 创建一个3x3的单位矩阵
identity_matrix = np.eye(3)
```
输出结果为：
```
[[1. 0. 0.]
 [0. 1. 0.]
 [0. 0. 1.]]
```
如果不涉及线性代数运算，这个函数使用频率较低。

本节课中我们一起学习了多种创建NumPy数组的方法：
1.  **`np.zeros` / `np.ones`**：创建全零或全一数组，可通过`dtype`控制数据类型。
2.  **`np.empty`**：快速分配内存但不初始化，包含随机值，使用时需谨慎。
3.  **`np.arange`**：生成等差序列数组，功能比`range`更强（支持小数步长）。
4.  **`np.linspace`**：在指定区间生成指定数量的等间隔点，常用于数值计算和绘图。
5.  **`np.eye`**：生成单位矩阵，用于线性代数运算。
