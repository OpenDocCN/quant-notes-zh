## 模型参数详解

现在，让我们深入看一下建模的具体步骤。我们回到代码，查看`create_model`函数的内部实现。

第一步是重载绘图工具。然后实例化Prophet模型。在实例化时，可以指定一系列参数来控制模型行为。

![](img/9a005f306f32b671307ac3a231031bf7_23.png)

```python
model = Prophet(
    yearly_seasonality=True,
    weekly_seasonality=False,
    daily_seasonality=False,
    changepoint_prior_scale=0.05
)
```

以下是关键参数说明：
*   `yearly_seasonality`, `weekly_seasonality`, `daily_seasonality`: 布尔值，用于控制是否考虑年、周、日的季节性变化。
*   `changepoint_prior_scale`: 这是一个关键参数，默认值为**0.05**。它控制模型趋势变化的灵活性，值越大，模型对趋势变化的捕捉越敏感（更“奔放”）；值越小，趋势越平滑（更“保守”）。
*   你还可以传入`holidays`参数来指定节假日的影响。

如果想深入了解每个参数，可以按住`Ctrl`键点击`Prophet`进入源码查看。由于源码内容较多，我们在此不深入探讨，感兴趣的同学可以自行研究。



![](img/9a005f306f32b671307ac3a231031bf7_12.png)

在实例化模型时，我们传入了自定义参数，例如添加了一个以30.5天为周期的季节性变化。最后，将构建好的模型返回。

拿到时间序列模型后，接下来需要传入数据进行训练。

![](img/9a005f306f32b671307ac3a231031bf7_29.png)

![](img/9a005f306f32b671307ac3a231031bf7_31.png)

这里有一个关键操作：`self.training`。它的含义是，我们可以只用最近几年的历史数据来训练模型。例如，如果只想用近三年的数据预测，就可以在这里指定`training=3`，模型将只使用最近三年的数据进行训练。

![](img/9a005f306f32b671307ac3a231031bf7_32.png)

训练使用`model.fit()`方法完成。训练结束后，我们使用`model.make_future_dataframe()`函数。

```python
future = model.make_future_dataframe(periods=0)
```

![](img/9a005f306f32b671307ac3a231031bf7_34.png)

`periods`参数指定为**0**，意味着我们只对历史数据进行拟合和预测，不生成未来的时间点。稍后我们会演示将`periods`设为90或180，即预测未来90天或180天的情况。

![](img/9a005f306f32b671307ac3a231031bf7_36.png)

现在，我们使用`model.predict(future)`对历史数据进行预测。这个过程可能会稍慢。预测结果是一个DataFrame，其中会自动添加以下几列：
*   `yhat`: 预测值。
*   `yhat_lower`: 预测区间的下限。
*   `yhat_upper`: 预测区间的上限。

我们可以将`yhat`作为核心预测指标，`yhat_lower`和`yhat_upper`作为置信区间。

## 识别突变点

![](img/9a005f306f32b671307ac3a231031bf7_73.png)

接下来，我们学习如何识别时间序列中的“突变点”。

突变点是指趋势发生显著变化的点。例如，从下降趋势转为上升趋势，或增长速率突然加快/减慢的点。Prophet可以自动检测这些点。



![](img/9a005f306f32b671307ac3a231031bf7_75.png)

![](img/9a005f306f32b671307ac3a231031bf7_76.png)

![](img/9a005f306f32b671307ac3a231031bf7_58.png)

![](img/9a005f306f32b671307ac3a231031bf7_78.png)

![](img/9a005f306f32b671307ac3a231031bf7_79.png)

检测突变点的步骤如下：
1.  照常创建和训练模型。
2.  训练后，可以从`model.changepoints`属性中获取模型自动检测到的突变点日期。
3.  我们还可以计算这些突变点处趋势变化的方向（二阶导数的正负）。二阶导数大于0表示在该点趋势加速（正向突变），小于0表示趋势减速或转向（负向突变）。
4.  最后，在原始数据图上，用不同颜色的竖线标记出正向突变点（如绿色）和负向突变点（如红色）。

这样，我们就能直观地看到历史数据中趋势发生重大变化的关键时刻。

## 进行未来预测

最后，也是大家最感兴趣的部分：如何进行未来预测。

![](img/9a005f306f32b671307ac3a231031bf7_85.png)

![](img/9a005f306f32b671307ac3a231031bf7_86.png)

假设当前日期是2018年7月21日，我们想预测未来180天的股价走势。操作非常简单，只需在创建未来数据框时，将`periods`参数设置为**180**。

```python
future = model.make_future_dataframe(periods=180)
forecast = model.predict(future)
```

![](img/9a005f306f32b671307ac3a231031bf7_88.png)

模型会自动将历史数据的时间轴向后延伸180天，并对这些未来的日期做出预测。在画图时，未来部分只有绿色的预测线及置信区间，而没有黑色的真实观测点。



![](img/9a005f306f32b671307ac3a231031bf7_90.png)

![](img/9a005f306f32b671307ac3a231031bf7_85.png)

![](img/9a005f306f32b671307ac3a231031bf7_91.png)

![](img/9a005f306f32b671307ac3a231031bf7_92.png)

观察预测结果，模型基于历史趋势，推测未来股价可能会经历一段下降然后再上升的过程。你可能会觉得这个预测相对“保守”，没有给出更大幅度的上涨预期。这正是由前面提到的 `changepoint_prior_scale` 参数控制的。通过调整这个参数，我们可以让模型的预测趋势变得更“平滑保守”或更“灵活奔放”。



![](img/9a005f306f32b671307ac3a231031bf7_90.png)

本节课中，我们一起学习了使用Facebook Prophet库进行时间序列预测的完整流程：
1.  **模型构建**：实例化`Prophet`类，并理解关键参数（如季节性开关、`changepoint_prior_scale`）的作用。
2.  **训练与拟合**：使用`model.fit()`训练模型，并用`model.predict()`对历史数据进行拟合，得到预测值和置信区间。
3.  **趋势分解**：利用`model.plot_components()`可视化数据的整体趋势、年度和月度等季节性规律。
4.  **突变点检测**：通过`model.changepoints`识别趋势发生显著变化的关键时间点。
5.  **未来预测**：通过设置`make_future_dataframe(periods=N)`来预测未来N天的走势。
