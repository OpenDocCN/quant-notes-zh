首先，我们需要一个函数来获取当前账户的持仓股票列表。

![](img/06869946fb5b73f068d796693c29dffd_11.png)

以下是 `get_holdings` 函数的实现步骤：

![](img/06869946fb5b73f068d796693c29dffd_13.png)

1.  从 `context` 对象中获取持仓信息 `context.portfolio.positions`。
2.  遍历所有持仓，检查其数量 `position.amount` 是否大于0。
3.  将所有数量大于0的持仓股票代码添加到一个列表中。
4.  返回这个持仓股票代码列表。

```python
def get_holdings(context):
    # 初始化一个空列表，用于存放当前持有的股票代码
    holdings = []
    # 遍历账户中的所有持仓
    for position in context.portfolio.positions.values():
        # 检查该持仓的股票数量是否大于0
        if position.amount > 0:
            # 如果持有，则将股票代码加入列表
            holdings.append(position.security)
    # 返回当前持仓股票列表
    return holdings
```

现在，我们可以在 `rebalance` 函数中使用这个新函数。我们已经有了新的股票排名 `stocks`，也知道了当前持仓 `holdings`。接下来，我们需要计算两者之间的差异，以确定需要买入和卖出的股票。

以下是计算买卖列表的逻辑：

![](img/06869946fb5b73f068d796693c29dffd_19.png)

*   **需要买入的股票 (`to_buy`)**：在新排名 `stocks` 中，但不在当前持仓 `holdings` 中的股票。这代表了我们应该新增的标的。
*   **需要卖出的股票 (`to_sell`)**：在当前持仓 `holdings` 中，但不在新排名 `stocks` 中的股票。这代表了我们应该剔除的标的。

![](img/06869946fb5b73f068d796693c29dffd_21.png)

为了高效地进行集合运算（求差集），我们将列表转换为集合（`set`）。

![](img/06869946fb5b73f068d796693c29dffd_23.png)

![](img/06869946fb5b73f068d796693c29dffd_25.png)

```python
# 获取当前持仓
holdings = get_holdings(context)
# 计算需要买入的股票：新排名中有，但持仓中没有的
to_buy = set(stocks) - set(holdings)
# 计算需要卖出的股票：持仓中有，但新排名中没有的
to_sell = set(holdings) - set(stocks)
```

计算出买卖列表后，我们就可以执行交易指令了。通常的交易逻辑是先卖出不再需要的股票，释放资金，然后再买入新的目标股票。

![](img/06869946fb5b73f068d796693c29dffd_31.png)

以下是执行卖出操作的步骤：

1.  遍历 `to_sell` 列表中的每一只股票。
2.  使用 `order_target_percent` 函数，将该股票的持仓比例调整为0%，即全部卖出。

```python
# 执行卖出操作：先卖出不再需要的股票
for stock in to_sell:
    # 将该股票持仓比例调整为0%，即全部卖出
    order_target_percent(stock, 0)
```

卖出操作完成后，我们获得了现金。接下来，我们需要将这些现金平均分配到所有需要买入的股票上。

![](img/06869946fb5b73f068d796693c29dffd_37.png)

以下是执行买入操作的步骤：

![](img/06869946fb5b73f068d796693c29dffd_39.png)

1.  检查 `to_buy` 列表是否为空。如果为空，则直接返回，无需操作。
2.  计算可用于投资的现金总额：`context.portfolio.total_value`。
3.  计算每只待买入股票应分配的资金比例：`1.0 / len(to_buy)`。
4.  遍历 `to_buy` 列表，为每只股票下达买入指令，买入指定的资金比例。

![](img/06869946fb5b73f068d796693c29dffd_41.png)

![](img/06869946fb5b73f068d796693c29dffd_43.png)

```python
# 执行买入操作
# 如果没有需要买入的股票，则直接返回
if len(to_buy) == 0:
    return

![](img/06869946fb5b73f068d796693c29dffd_45.png)

# 计算每只股票应买入的平均资金比例
avg_weight = 1.0 / len(to_buy)

![](img/06869946fb5b73f068d796693c29dffd_47.png)

![](img/06869946fb5b73f068d796693c29dffd_49.png)

# 遍历所有需要买入的股票
for stock in to_buy:
    # 买入该股票，使其在投资组合中的权重达到 avg_weight
    order_target_percent(stock, avg_weight)
```

![](img/06869946fb5b73f068d796693c29dffd_55.png)

![](img/06869946fb5b73f068d796693c29dffd_57.png)

    *   获取当前持仓 `holdings`。
    *   计算买卖列表 `to_buy` 和 `to_sell`。

2.  **常见错误修正**：
    *   **标点符号**：确保代码中使用的是英文逗号 `,`，而不是中文逗号 `，`。
    *   **数据类型**：对列表进行集合差集运算时，需先将其转换为 `set` 类型。
    *   **未使用的变量**：清理代码中定义但未使用的变量（如 `last_rank`），或将其注释掉，保持代码整洁。
