# 金融量化分析：03：股票市场构成 🏛️

在本节课中，我们将要学习股票市场的构成。了解市场中有哪些参与者以及他们各自扮演的角色，是理解股票交易和进行量化分析的基础。

上一节我们介绍了股票的分类，本节中我们来看看股票市场的构成，即市场中有哪些参与者。

## 公司与投资者

公司和投资者是市场中最基础的双方。公司是需要融资的一方，投资者是提供资金的一方。公司通过上市向投资者募集资金，投资者则通过购买股票进行投资。

## 监管与服务机构

但是，股票交易不能让公司和投资者直接进行，否则可能存在暗箱操作。因此，市场需要监管和服务机构。

以下是主要的监管与服务机构：

*   **证监会**：这是证券行业的监管机构。公司想要上市，必须向证监会提交各种材料。证监会负责审查公司是否存在欺诈、洗钱等违法行为，并有权决定公司能否上市或是否应退市。
*   **证券业协会**：这是一个行业自律组织，作用相对较弱。例如，证券从业资格考试通常由其主办。
*   **交易所**：交易所提供股票集中交易的场所。在中国，主要有上海证券交易所和深圳证券交易所。早期投资者需要亲自到交易所交易，现在则通过网络连接到交易所的系统进行。

## 证券中介机构（券商）

个人投资者通常不能直接进入交易所买卖股票，这主要是由于历史原因和成本考量。过去，在交易所拥有交易“席位”的费用极其高昂。

为了解决这个问题，**证券中介机构**（俗称券商）应运而生。例如中信证券、中金公司等。这些券商在交易所拥有席位，并开发了交易软件（如各券商APP或同花顺等）。投资者通过券商的软件提交交易指令，券商再利用其在交易所的席位，代表投资者完成交易。

## 交易所与板块

中国有两个主要交易所：上海证券交易所和深圳证券交易所。每个交易所内部又划分为不同的板块。

以下是各交易所的板块划分：

*   **上海证券交易所**：主要包含**主板**。
*   **深圳证券交易所**：包含三个板块：
    *   **主板**
    *   **中小板**：为规模较小但成长性较好的公司提供融资平台。
    *   **创业板**：为创业型、高科技公司提供融资平台，上市门槛相对主板较低。

## 大盘指数

对于每个板块，我们常用一个**大盘指数**来概括其整体表现。指数反映了该板块内众多股票价格变动的综合趋势，是判断市场整体“向好”或“向坏”的重要指标。

以下是各板块对应的主要指数：

*   **上海主板**：**上证指数**（沪指）
*   **深圳主板**：**深证成指**（深成指）
*   **中小板**：**中小板指**
*   **创业板**：**创业板指**

指数可以理解为一个经过加权计算的平均值，其公式概念可表示为：
**指数 = f(各成分股价格, 权重)**
它描绘了整个市场“盘子”的综合走势图。

本节课中我们一起学习了股票市场的核心构成。我们认识了市场的核心参与者——公司与投资者，了解了维持市场秩序的监管机构（证监会），知道了提供交易场所的交易所及其不同板块（主板、中小板、创业板），也明白了个人投资者需要通过券商进行交易的原因。最后，我们引入了**大盘指数**的概念，它是衡量市场整体冷暖的关键指标。理解这些基础构成，是后续进行深入量化分析的重要前提。
