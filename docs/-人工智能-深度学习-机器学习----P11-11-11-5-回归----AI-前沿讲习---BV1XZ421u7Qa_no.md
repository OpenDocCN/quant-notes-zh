## 构建回归方程

上一节我们介绍了金融时间序列的基本处理方法，本节中我们来看看如何量化两个指标之间的关系。构建回归方程的方法有很多，我们可以使用多种工具包。这里我们使用NumPy库中的`polyfit`函数来计算回归系数。

![](img/ee0520b97df5ded379f46d56af196347_7.png)

![](img/ee0520b97df5ded379f46d56af196347_9.png)

`polyfit`函数可以帮助我们计算一个回归模型的系数。其基本形式是`Y = KX + B`，其中`K`是斜率，`B`是截距。该函数还可以拟合更高阶的多项式。

![](img/ee0520b97df5ded379f46d56af196347_11.png)

![](img/ee0520b97df5ded379f46d56af196347_13.png)

以下是使用`polyfit`函数构建一元线性回归方程的步骤：

![](img/ee0520b97df5ded379f46d56af196347_15.png)

![](img/ee0520b97df5ded379f46d56af196347_17.png)

1.  准备数据：确保用于计算的两个指标数据中没有空值。
2.  调用函数：传入自变量`X`、因变量`Y`以及指定的多项式阶数`deg`。
3.  获取系数：函数返回的系数数组，对于`deg=1`，返回`[K, B]`。

![](img/ee0520b97df5ded379f46d56af196347_19.png)

![](img/ee0520b97df5ded379f46d56af196347_21.png)

```python
import numpy as np
import pandas as pd

# 假设 df 是一个包含 ‘SPX’ 和 ‘VIX’ 列的DataFrame
# 首先处理可能存在的空值
df[['SPX', 'VIX']].dropna(inplace=True)

![](img/ee0520b97df5ded379f46d56af196347_23.png)

# 指定 X 和 Y，这里以 SPX 为自变量，VIX 为因变量
X = df['SPX'].values
Y = df['VIX'].values

![](img/ee0520b97df5ded379f46d56af196347_25.png)

# 使用 polyfit 计算一元线性回归系数 (deg=1)
coefficients = np.polyfit(X, Y, deg=1)
K, B = coefficients  # K 是斜率，B 是截距
print(f"回归方程为: Y = {K:.4f} * X + {B:.4f}")
```

![](img/ee0520b97df5ded379f46d56af196347_27.png)

![](img/ee0520b97df5ded379f46d56af196347_29.png)

## 可视化回归方程

![](img/ee0520b97df5ded379f46d56af196347_31.png)

得到回归方程后，我们可以将其与原始数据一起绘制在图表上，以便直观地观察关系。

![](img/ee0520b97df5ded379f46d56af196347_33.png)

以下是绘制散点图及回归线的步骤：

![](img/ee0520b97df5ded379f46d56af196347_35.png)

![](img/ee0520b97df5ded379f46d56af196347_37.png)

1.  绘制原始数据的散点图。
2.  根据回归方程计算对应的预测Y值。
3.  在同一坐标系中绘制回归线。

```python
import matplotlib.pyplot as plt

![](img/ee0520b97df5ded379f46d56af196347_39.png)

# 绘制散点图
plt.figure(figsize=(10, 6))
plt.scatter(X, Y, s=16, label='原始数据点')

![](img/ee0520b97df5ded379f46d56af196347_41.png)

![](img/ee0520b97df5ded379f46d56af196347_43.png)

![](img/ee0520b97df5ded379f46d56af196347_45.png)

# 根据回归方程计算预测值
Y_pred = np.polyval([K, B], X)  # 等同于 K * X + B

![](img/ee0520b97df5ded379f46d56af196347_47.png)

# 绘制回归线
plt.plot(X, Y_pred, color='red', linewidth=2, label='回归线')

![](img/ee0520b97df5ded379f46d56af196347_49.png)

![](img/ee0520b97df5ded379f46d56af196347_51.png)

plt.xlabel('SPX')
plt.ylabel('VIX')
plt.title('SPX 与 VIX 的线性回归关系')
plt.legend()
plt.grid(True)
plt.show()
```

## 计算相关系数

除了回归方程，相关系数是衡量两个变量之间线性关系强度和方向的另一个重要指标。其值介于-1到1之间。

![](img/ee0520b97df5ded379f46d56af196347_65.png)

![](img/ee0520b97df5ded379f46d56af196347_67.png)

在Pandas中，计算两个序列的相关系数非常简单。

![](img/ee0520b97df5ded379f46d56af196347_69.png)

以下是计算并展示相关系数矩阵的方法：

```python
# 计算 SPX 和 VIX 的相关系数矩阵
correlation_matrix = df[['SPX', 'VIX']].corr()
print(correlation_matrix)
```
该矩阵对角线上的值均为1（变量与自身的完全相关），而非对角线上的值则显示了`SPX`与`VIX`之间的相关系数。

## 附加任务：滑动窗口相关系数

![](img/ee0520b97df5ded379f46d56af196347_79.png)

在实际分析中，我们常常需要观察关系如何随时间变化。这时可以计算滑动窗口内的相关系数。

以下任务是：随着时间推移，计算`SPX`与`VIX`在一个固定时间窗口内的相关系数变化情况。

以下是实现滑动窗口相关系数计算的步骤：

![](img/ee0520b97df5ded379f46d56af196347_81.png)

![](img/ee0520b97df5ded379f46d56af196347_83.png)

1.  为其中一个指标（如`SPX`）定义一个滑动窗口（例如250天）。
2.  对于窗口内的每一个子序列，计算其与另一个指标（`VIX`）对应数据的相关系数。
3.  将每个窗口计算出的相关系数按时间顺序连接起来，形成一条时间序列。

![](img/ee0520b97df5ded379f46d56af196347_85.png)

![](img/ee0520b97df5ded379f46d56af196347_87.png)

```python
# 设置窗口大小
window_size = 250

![](img/ee0520b97df5ded379f46d56af196347_89.png)

# 计算 SPX 滚动窗口与 VIX 的相关系数
# 这里使用 .rolling() 创建窗口，并为每个窗口应用 .corr() 函数计算与 VIX 的相关系数
rolling_corr = df['SPX'].rolling(window=window_size).corr(df['VIX'])

![](img/ee0520b97df5ded379f46d56af196347_91.png)

![](img/ee0520b97df5ded379f46d56af196347_93.png)

# 绘制滚动相关系数变化图
plt.figure(figsize=(12, 6))
rolling_corr.plot()
plt.title(f'SPX 与 VIX 的 {window_size} 日滚动相关系数变化')
plt.xlabel('日期')
plt.ylabel('相关系数')
plt.grid(True)
plt.axhline(y=0, color='black', linestyle='--', linewidth=0.5) # 添加y=0的参考线
plt.show()
```
通过这个图表，我们可以清晰地看到`SPX`与`VIX`之间的相关性在不同市场时期是如何动态变化的。
