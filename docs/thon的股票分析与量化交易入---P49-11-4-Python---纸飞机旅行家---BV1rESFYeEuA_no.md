在本节课中，我们将要学习量化策略回测中三个核心的风险指标：Alpha、Beta和夏普比率。理解这些指标对于客观评价一个交易策略的优劣至关重要。

![](img/de3535984e59fef1eb270cb0cc862f53_3.png)

## 风险指标概述

上一节我们介绍了如何对MACD策略进行回测，本节中我们来看看如何解读回测结果。风险指标就是对策略表现的量化评价工具。

![](img/de3535984e59fef1eb270cb0cc862f53_5.png)

首先，我们需要理解什么是风险。在投资领域，**风险**指的是投资收益的不确定性。这种不确定性源于市场中未知或不可预测的因素（例如地缘政治事件），它们可能对股价产生正面或负面的影响。

风险指标具有以下特点：
*   它有助于投资者对策略进行客观评价。
*   它仅根据每日收盘后的收益进行计算，每日更新一次，不考虑盘中的价格波动。

## Alpha（阿尔法）指标

在理解了风险的基本概念后，我们首先来学习Alpha指标。**Alpha**衡量的是投资者获得的、与市场整体波动无关的回报。简单来说，无论大盘涨跌，策略都能获得的超额收益就是Alpha。

![](img/de3535984e59fef1eb270cb0cc862f53_7.png)

Alpha的计算基于“超额收益”的概念。以下是其核心思想与公式：

![](img/de3535984e59fef1eb270cb0cc862f53_9.png)

**核心概念**：Alpha = 策略的实际收益 - 基准（如沪深300指数）的收益 - 由Beta解释的收益部分。

![](img/de3535984e59fef1eb270cb0cc862f53_11.png)

**公式**：
`α = Rp - [Rf + β * (Rm - Rf)]`
其中：
*   `α`：阿尔法值
*   `Rp`：策略的年化收益率
*   `Rf`：无风险利率（通常参考国债收益率，当前约为3%）
*   `β`：贝塔系数（下一节介绍）
*   `Rm`：基准（如沪深300）的年化收益率

**举例说明**：如果您的策略年收益为15%，基准年收益为10%，无风险利率为3%，且Beta为1，那么Alpha约为 `15% - [3% + 1*(10%-3%)] = 5%`。这5%就是超越市场波动和系统风险带来的超额收益。

在回测平台（如聚宽）上，可以方便地查看Alpha值。通常它会展示不同时间周期（如1个月、3个月、1年）的Alpha，帮助您评估策略在不同阶段的超额收益能力。

![](img/de3535984e59fef1eb270cb0cc862f53_13.png)

## Beta（贝塔）指标

![](img/de3535984e59fef1eb270cb0cc862f53_15.png)

介绍完衡量超额收益的Alpha，本节我们来看看衡量系统风险的Beta指标。**Beta**反映了投资组合收益相对于市场基准收益的波动性，即策略对大盘变化的敏感程度。

**定义**：Beta是一个系数。它表示当市场基准变动1%时，策略预期会变动多少。

![](img/de3535984e59fef1eb270cb0cc862f53_17.png)

**公式**：
`β = Cov(Rp, Rm) / Var(Rm)`
其中：
*   `Cov(Rp, Rm)`：策略日收益率与基准日收益率的协方差。
*   `Var(Rm)`：基准日收益率的方差。

**举例说明**：
*   如果策略的Beta为**1.5**，意味着市场上涨1%时，策略可能上涨1.5%；市场下跌1%时，策略可能下跌1.5%。这表明策略波动比市场更大。
*   如果策略的Beta为**-1.5**，则意味着它与市场呈反向关系。市场上涨1%，策略可能下跌1.5%；市场下跌1%，策略可能上涨1.5%。

在回测平台中，点击“贝塔”选项即可查看该策略的Beta历史数据，从而判断其与市场的相关性及波动幅度。

![](img/de3535984e59fef1eb270cb0cc862f53_19.png)

## 夏普比率（Sharpe Ratio）

![](img/de3535984e59fef1eb270cb0cc862f53_21.png)

我们已经分别学习了衡量超额收益（Alpha）和系统风险（Beta）的指标。有没有一个指标能同时兼顾收益与风险呢？这就是非常著名的**夏普比率**。

夏普比率衡量的是**每承受一单位的总风险，能够获得多少超额报酬**。它将风险单位化，便于在不同策略之间进行“性价比”比较。

**公式**：
`Sharpe Ratio = (Rp - Rf) / σp`
其中：
*   `Rp`：策略的年化收益率
*   `Rf`：无风险利率
*   `σp`：策略年化收益率的波动率（标准差）

**如何理解**：
*   **夏普比率为正**：值越高，说明在承担相同风险的情况下，策略获得的超额回报越高，策略的“性价比”越好。这是投资者追求的目标。
*   **夏普比率为负**：通常没有比较意义，因为它意味着策略收益尚不及无风险利率。

在回测平台上，您可以查看策略的夏普比率历史走势。一个长期保持较高正夏普比率的策略，通常被认为是稳健且有效的。

## 本章总结

本节课我们一起学习了量化策略回测中三个最核心的风险指标。

1.  **Alpha（阿尔法）**：代表与市场波动无关的超额收益。其公式为 `α = Rp - [Rf + β*(Rm - Rf)]`。Alpha越高，说明策略获取超额收益的能力越强。
2.  **Beta（贝塔）**：代表策略的系统性风险，即相对于市场基准的波动敏感度。其公式为 `β = Cov(Rp, Rm) / Var(Rm)`。Beta的绝对值越大，策略波动相对于市场越大。
3.  **夏普比率**：综合了收益与风险的指标，衡量单位风险下的超额收益。其公式为 `Sharpe Ratio = (Rp - Rf) / σp`。夏普比率越高（为正时），策略的风险调整后收益越好。

![](img/de3535984e59fef1eb270cb0cc862f53_23.png)

这三个指标是评价一个量化策略性能的基石。在后续的实践中，您需要结合这些指标，而不仅仅是最终的总收益率，来全面、客观地评估和优化您的交易策略。
