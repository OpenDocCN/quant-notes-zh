## 概述

上一节我们介绍了风险与回报的基本概念。本节中，我们来看看如何量化评估“风险调整后的收益”，这正是夏普比率的核心作用。它帮助我们判断，为了获得一定的超额收益，我们所承担的风险是否值得。

## 夏普比率的核心概念

夏普比率描述的是“单位风险所带来的超额收益”。我们可以通过一个类比来理解：假设有两份工作，一份是稳定的文职，月薪1万元；另一份是战地记者，月薪5万元但风险极高。单纯比较收入，后者更高。但如果我们考虑“每单位风险对应的收入”，就需要一个指标来衡量，夏普比率就是这样的指标。

![](img/34d198ac778364a11e9f5e5bca2b2f5a_1.png)

在金融投资中，我们同样面临选择：是投资于低风险低收益的国债，还是投资于高风险高收益的股票组合？夏普比率帮助我们做出更科学的比较。

![](img/34d198ac778364a11e9f5e5bca2b2f5a_3.png)

**指标越高，意味着每承担一单位风险，所获得的超额收益就越高，该投资组合的“性价比”也就越好。**

## 夏普比率的计算公式

![](img/34d198ac778364a11e9f5e5bca2b2f5a_5.png)

理解了其含义后，我们来看如何计算夏普比率。其核心思想是比较投资组合的收益与无风险收益的差值，再除以投资组合的风险（波动性）。

公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**: 投资组合的平均收益率
*   **Rf**: 无风险收益率（例如国债利率）
*   **σp**: 投资组合收益率的标准差（代表风险）

这个公式计算的是**年化夏普比率**。在计算时，我们需要对收益率进行年化处理。

## 使用Python计算夏普比率

接下来，我们通过Python代码，实际计算几只股票的夏普比率。假设我们已经有了一组股票的历史日度收益率数据。

以下是计算步骤：

1.  **准备数据**：获取股票历史价格，并计算日收益率。需处理缺失值，通常用前一天的收益率填充。
2.  **设定参数**：设定年化无风险收益率（例如5%）。
3.  **执行计算**：根据公式，计算每只股票的年化夏普比率。

```python
import numpy as np
import pandas as pd

![](img/34d198ac778364a11e9f5e5bca2b2f5a_7.png)

# 假设 `returns` 是一个DataFrame，列是各股票名称，行是日度收益率
# 步骤1: 计算平均日收益率
mean_daily_returns = returns.mean()

# 步骤2: 设定年化无风险利率为5%，并计算日度无风险利率
risk_free_rate = 0.05
daily_risk_free_rate = risk_free_rate / 250  # 假设一年有250个交易日

![](img/34d198ac778364a11e9f5e5bca2b2f5a_9.png)

# 步骤3: 计算超额收益（日度）
excess_daily_returns = mean_daily_returns - daily_risk_free_rate

# 步骤4: 计算收益率的标准差（日度）
daily_std = returns.std()

# 步骤5: 计算年化夏普比率
# 年化超额收益 = 平均日超额收益 * 250
# 年化波动率 = 日波动率 * sqrt(250)
sharpe_ratios = (excess_daily_returns * 250) / (daily_std * np.sqrt(250))

print(sharpe_ratios)
```

运行以上代码后，我们将得到一个夏普比率序列。例如，输出可能如下：

```
股票A   -0.15
股票B    0.82
股票C    1.35
dtype: float64
```

## 如何解读结果

![](img/34d198ac778364a11e9f5e5bca2b2f5a_13.png)

根据计算结果，我们可以对投资标的进行筛选和排序：

*   **股票C（夏普比率1.35）**：这是三者中最优的选择。它意味着承担一单位风险，能获得最高的超额收益。
*   **股票B（夏普比率0.82）**：其“风险收益性价比”次于股票C。
*   **股票A（夏普比率为负）**：这意味着其收益甚至低于无风险利率（如国债），投资它既承担了风险，又没有获得足够的补偿，通常应避免。

![](img/34d198ac778364a11e9f5e5bca2b2f5a_15.png)

**因此，在构建投资组合时，我们应倾向于选择夏普比率更高的资产。**
