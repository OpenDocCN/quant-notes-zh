## 第一步：获取交易日列表

首先，我们需要确定要分析的日期范围。因为后续查询数据时需要传入具体的日期参数。

![](img/138882c63b4f263bd65d9002a516902a_14.png)

以下是获取指定时间段内所有交易日期的步骤：

![](img/138882c63b4f263bd65d9002a516902a_16.png)

![](img/138882c63b4f263bd65d9002a516902a_18.png)

1.  **导入必要模块**：从 `alphalearns` 工具包中导入数据处理和绘图模块。
2.  **定义函数**：创建一个函数 `get_trading_date`，接收开始日期和结束日期两个参数。
3.  **调用API**：使用平台提供的函数获取该时间段内的所有交易日列表。
4.  **返回结果**：函数返回一个包含所有交易日的列表，供后续步骤使用。

![](img/138882c63b4f263bd65d9002a516902a_20.png)

![](img/138882c63b4f263bd65d9002a516902a_22.png)

核心代码如下：

![](img/138882c63b4f263bd65d9002a516902a_24.png)

![](img/138882c63b4f263bd65d9002a516902a_26.png)

```python
def get_trading_date(start_date, end_date):
    """
    获取指定日期范围内的交易日列表。
    参数:
        start_date (str): 开始日期，格式如‘2019-01-01’
        end_date (str): 结束日期，格式如‘2020-01-01’
    返回:
        list: 交易日列表
    """
    trading_dates = get_trade_days(start_date, end_date) # 假设的API函数
    return trading_dates
```

## 第三步：构建并执行因子查询

![](img/138882c63b4f263bd65d9002a516902a_50.png)

有了日期和股票池，我们就可以构建查询语句来获取具体的因子数据了。这里我们以市盈率（P/E Ratio）因子为例。

![](img/138882c63b4f263bd65d9002a516902a_52.png)

![](img/138882c63b4f263bd65d9002a516902a_54.png)

以下是查询单日因子数据的步骤：

![](img/138882c63b4f263bd65d9002a516902a_56.png)

![](img/138882c63b4f263bd65d9002a516902a_58.png)

1.  **构建查询对象**：使用 `query()` 函数，指定要查询的因子字段，例如 `valuation.pe_ratio`。
2.  **添加过滤条件**：通过 `filter()` 方法，限定只查询股票池 `stock_pool` 中的股票。
3.  **执行查询**：使用 `get_fundamentals()` 函数，传入构建好的查询对象和具体的查询日期（例如交易日列表中的第一天），执行查询。
4.  **检查数据**：获取返回的数据，通常是一个三维数据结构，但有效数据集中在最后一个维度。我们可以查看其形状和头部几条数据以验证结果。

![](img/138882c63b4f263bd65d9002a516902a_60.png)

![](img/138882c63b4f263bd65d9002a516902a_62.png)

核心代码如下：

![](img/138882c63b4f263bd65d9002a516902a_64.png)

![](img/138882c63b4f263bd65d9002a516902a_66.png)

```python
# 1. 构建查询语句
q = query(valuation.pe_ratio).filter(valuation.code.in_(stock_pool))

![](img/138882c63b4f263bd65d9002a516902a_68.png)

![](img/138882c63b4f263bd65d9002a516902a_70.png)

# 2. 执行查询（查询第一个交易日的数据）
factor_data = get_fundamentals(q, date=trading_dates[0])

![](img/138882c63b4f263bd65d9002a516902a_72.png)

![](img/138882c63b4f263bd65d9002a516902a_74.png)

# 3. 查看数据形状和前几条记录
print(factor_data.shape)
print(factor_data.iloc[0, 0, :5]) # 查看前5只股票的市盈率数据
```

![](img/138882c63b4f263bd65d9002a516902a_76.png)

![](img/138882c63b4f263bd65d9002a516902a_78.png)

**代码解释**：
*   `valuation.pe_ratio`：指定查询市盈率因子。
*   `valuation.code.in_(stock_pool)`：过滤条件，确保只返回股票池中的股票。
*   `trading_dates[0]`：传入我们之前获取的交易日列表中的第一个日期。
