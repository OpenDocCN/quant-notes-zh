*   **自动选择高收益品种**：程序可以自动比较深市和沪市逆回购的实时收益率，并选择收益率更高的品种进行交易，而不是固定选择某一个。
*   **设置最低收益率门槛**：许多券商提供收益相对固定的隔夜理财产品（例如年化`2.2%`）。我们可以设定一个最低收益率（如`2.2%`），只有当逆回购收益率高于此门槛时才执行，否则自动将资金转入券商的理财产品，从而提高闲置资金的整体收益。
*   **优化交易时机**：通过历史数据统计，可以发现逆回购收益率在日内某些时段（如`15:00`整、`15:08`或`15:18`左右）可能出现相对高点。程序可以根据策略的实际情况，动态选择最优的借出时间点。

1.  **剔除高风险股票**：建立一个“黑名单”，排除财务造假、被立案调查、已被ST/*ST的股票。策略每次运行时读取这个名单并自动过滤。
2.  **剔除短期涨幅过高股票**：A股市场个股容易暴涨暴跌，短期涨幅过大的股票回调风险也较高。可以设定一个阈值（如近期涨幅超过`50%`），将其剔除。
3.  **剔除盈利下滑的股票**：选择基本面稳健的公司。例如，可以剔除最近一期财报中净利润同比下滑的股票。
4.  **筛选并买入高股息股票**：在经过上述过滤后剩余的股票池中，选择股息率最高的前N只股票，进行等权重买入。

*   **套利策略**：利用同一资产在不同市场、不同期限或关联资产之间的价格偏差进行无风险或低风险套利，如期货的跨期套利、跨品种套利。
*   **多因子模型**：这是目前主流量化策略的核心。通过挖掘对股价有预测能力的多个因子（如价值、成长、动量、质量等），构建综合评分模型来选股。持续挖掘新的有效因子是保持策略竞争力的关键。
*   **机器学习策略**：运用机器学习算法（如深度学习、强化学习）从海量数据中自动寻找复杂的非线性关系与交易模式。目前许多高频交易（T0）算法都应用了机器学习技术。
