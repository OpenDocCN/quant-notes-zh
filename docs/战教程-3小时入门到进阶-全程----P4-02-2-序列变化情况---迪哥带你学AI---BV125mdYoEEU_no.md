## 计算每日差异值

![](img/fa84d377cefd256d1926f6cd2231c65b_6.png)

首先，我们需要计算每一天指标相对于前一天的差异值。这个指标反映了数据的日度变化幅度。

![](img/fa84d377cefd256d1926f6cd2231c65b_8.png)

在Pandas中，我们可以使用 `diff()` 函数轻松实现这一计算。该函数默认计算当前行与前一行之间的差值。

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

![](img/fa84d377cefd256d1926f6cd2231c65b_10.png)

执行上述代码后，从第二天（例如1月5日）开始，每个值都等于当天数据减去前一天数据的结果。第一行由于没有前一天的数据，其结果为 `NaN`（非数字）。

![](img/fa84d377cefd256d1926f6cd2231c65b_12.png)

这个差异值指标在时间序列分析中非常常用。我们不仅可以计算一天的差异，还可以通过指定 `periods` 参数来计算间隔多天的差异（例如，两天、一周）。

## 计算每日增长率

差异值是一个绝对值，有时我们更关心相对变化，即增长率。例如，股价从1元涨到2元是100%的增长，而从10000元涨到10001元则增长甚微。

![](img/fa84d377cefd256d1926f6cd2231c65b_14.png)

![](img/fa84d377cefd256d1926f6cd2231c65b_16.png)

增长率的计算公式为：`(后一天的值 - 前一天的值) / 前一天的值`。

![](img/fa84d377cefd256d1926f6cd2231c65b_18.png)

Pandas提供了 `pct_change()` 函数来直接计算这个百分比变化。

```python
# 计算每日增长率
pct_data = data.pct_change()
print(pct_data.head())
```

![](img/fa84d377cefd256d1926f6cd2231c65b_20.png)

计算完成后，数据将显示为增长率（例如0.052表示增长了5.2%）。通常股价的日增长率数值较小。

## 统计分析变化情况

![](img/fa84d377cefd256d1926f6cd2231c65b_22.png)

计算出差异值或增长率后，我们还可以对其进行统计分析，以了解整体变化特征。

![](img/fa84d377cefd256d1926f6cd2231c65b_24.png)

例如，我们可以计算所有交易日增长率的平均值。

![](img/fa84d377cefd256d1926f6cd2231c65b_26.png)

![](img/fa84d377cefd256d1926f6cd2231c65b_28.png)

```python
# 计算增长率的平均值
mean_pct = pct_data.mean()
print(mean_pct)
```

![](img/fa84d377cefd256d1926f6cd2231c65b_30.png)

## 可视化分析结果

![](img/fa84d377cefd256d1926f6cd2231c65b_32.png)

为了更直观地展示各个指标的平均增长率，我们可以使用条形图进行可视化。

![](img/fa84d377cefd256d1926f6cd2231c65b_34.png)

以下是使用Matplotlib绘制条形图的代码：

```python
import matplotlib.pyplot as plt

![](img/fa84d377cefd256d1926f6cd2231c65b_36.png)
```
