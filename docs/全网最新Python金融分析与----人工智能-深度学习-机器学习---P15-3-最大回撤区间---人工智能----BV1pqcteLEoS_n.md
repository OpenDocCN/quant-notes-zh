在本节课中，我们将要学习一个重要的风险指标——最大回撤。它用于衡量投资策略在特定时间段内可能面临的最糟糕情况，即资产价值从峰值到谷底的最大跌幅。

## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何评估策略的风险。最大回撤描述的是在投资过程中，资产净值从历史最高点下跌到随后最低点的最大幅度。它衡量的是“最惨”的下跌经历，即投资者可能承受的最大账面亏损。

其核心计算公式如下：

**最大回撤 = max( (P_i - P_j) / P_i )**

其中：
*   **P_i** 代表第 i 天的资产净值（峰值）。
*   **P_j** 代表第 i 天之后某一天（j > i）的资产净值（谷底）。

![](img/00a184b712618de33fc2c75d6f96d036_5.png)

这个公式遍历所有可能的“峰值-谷底”组合，找出净值缩水比例最大的那个区间。它不是一个评估策略绝对好坏的指标，而是一个关键的风险指标。即使一个策略长期是盈利的，但如果其最大回撤过大，也可能因为投资者无法承受中途的巨大亏损而被迫止损退出。

![](img/00a184b712618de33fc2c75d6f96d036_7.png)

## 如何计算最大回撤？

![](img/00a184b712618de33fc2c75d6f96d036_9.png)

理解了最大回撤的概念后，我们来看看如何在Python中实现它的计算。核心思路是：我们需要找到历史上每一个点与其之前最高点相比的回撤，然后从中选出最大值。

![](img/00a184b712618de33fc2c75d6f96d036_11.png)

![](img/00a184b712618de33fc2c75d6f96d036_13.png)

以下是计算最大回撤的关键步骤：

![](img/00a184b712618de33fc2c75d6f96d036_15.png)

1.  **计算历史累计最大值序列**：对于数据中的每一天，计算截至到该天的历史最高净值。
2.  **计算每日回撤**：用当天的历史最高净值减去当天的实际净值，得到每日的“潜在亏损”金额。
3.  **计算每日回撤率**：将每日的“潜在亏损”金额除以对应的历史最高净值，得到每日的回撤率。
4.  **找出最大回撤率**：在所有每日回撤率中，找出最大值，即为最大回撤。

![](img/00a184b712618de33fc2c75d6f96d036_17.png)

![](img/00a184b712618de33fc2c75d6f96d036_19.png)

我们可以使用Pandas库的 `cummax()` 函数来高效地计算历史累计最大值。

![](img/00a184b712618de33fc2c75d6f96d036_21.png)

![](img/00a184b712618de33fc2c75d6f96d036_23.png)

![](img/00a184b712618de33fc2c75d6f96d036_25.png)

```python
import pandas as pd

![](img/00a184b712618de33fc2c75d6f96d036_27.png)

![](img/00a184b712618de33fc2c75d6f96d036_29.png)

# 假设 df[‘net_value’] 是代表每日资产净值的序列
# 1. 计算历史累计最大值
historical_max = df[‘net_value’].cummax()

![](img/00a184b712618de33fc2c75d6f96d036_31.png)

# 2. 计算每日回撤（金额）
drawdown_amount = historical_max - df[‘net_value’]

![](img/00a184b712618de33fc2c75d6f96d036_33.png)

![](img/00a184b712618de33fc2c75d6f96d036_35.png)

# 3. 计算每日回撤率
drawdown_rate = drawdown_amount / historical_max

![](img/00a184b712618de33fc2c75d6f96d036_37.png)

# 4. 找出最大回撤率
max_drawdown = drawdown_rate.max()
print(f“最大回撤率为：{max_drawdown:.2%}”)
```

## 最大回撤区间的可视化

计算出最大回撤率后，我们通常还希望知道这个最大亏损具体发生在哪一段时间。这被称为“最大回撤区间”。在专业的回测图表中，这个区间通常会用阴影高亮标出，直观地展示策略经历的最艰难时期。

![](img/00a184b712618de33fc2c75d6f96d036_43.png)

![](img/00a184b712618de33fc2c75d6f96d036_45.png)

虽然手动计算区间起止点稍复杂，但许多量化分析库（如`empyrical`, `backtrader`等）都提供了直接输出最大回撤及其区间的函数。理解其计算原理有助于我们更深刻地解读这些结果。
