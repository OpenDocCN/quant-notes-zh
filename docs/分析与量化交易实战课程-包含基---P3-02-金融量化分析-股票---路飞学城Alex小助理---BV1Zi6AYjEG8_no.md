# 量化交易教程：P3：02 金融量化分析-股票基本知识和股票分类 📈

在本节课中，我们将要学习股票的基础知识，包括股票的定义、作用以及分类。理解这些概念是进行金融量化分析的第一步。

## 股票的定义

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

我们可以用一个例子来形象地解释股票。假设有一位创业公司的老板需要资金，而一位投资者（亿万富翁）愿意提供资金。如果投资者只是借钱给老板，这属于借贷行为。但投资者更希望获得公司的所有权和未来收益的分享权。

于是，投资者选择出资换取公司的股票。例如，一个公司初始市值为5亿，由创始人自己出资1亿，另外四位投资者各出资1亿。那么，每位出资者就获得了公司20%的股票。这代表了他们对公司的出资证明和所有权份额。

## 股票的作用

上一节我们介绍了股票是股东身份的凭证，本节中我们来看看股票具体有哪些作用。

股票主要有两个核心作用：

1.  **证明股东身份与权利**：持有股票意味着你是公司的股东。作为出资人，你拥有在股东大会投票等权利。
2.  **获取收益**：股东可以通过两种主要方式获利：
    *   **公司分红**：当公司盈利时，可能会将部分利润按持股比例分配给股东。例如，公司年净利润5000万，持有20%股份的股东可获得1000万的分红。
    *   **交易获利（资本利得）**：股东可以在证券交易所将股票转让给他人。如果卖出价格高于买入价格，就能赚取差价。例如，初始投资1亿获得20%股份，当公司市值从5亿增长到50亿时，这20%股份的价值就变成了10亿。此时卖出部分股份即可实现收益。

对于资金量较小的普通股民而言，虽然持股比例低，分红的绝对金额可能较小，但通过买卖股票赚取价差仍然是主要的获利方式。不过，普通股民买卖股票必须通过**证券交易所**进行。

## 什么是上市

了解了股票的作用后，一个自然的问题是：普通股民如何买卖股票？这就引出了“上市”的概念。

所谓上市，就是企业通过**证券交易所**，首次公开向全社会投资者发行股票，以募集资金的过程。

公司不能随意向公众募集资金。它需要达到一定体量，并向证监会提交申请，披露财务报表等信息。经审核批准后，公司才能在证券交易所挂牌。上市后，公司的股票就可以被所有股民公开买卖。

公司寻求上市的主要目的是为了融资。相对于向少数富人融资，上市可以向千千万万的普通投资者募集资金，潜力巨大。为了保障广大投资者的资金安全，证监会对公司的上市资格审核非常严格。

公司第一次公开发行股票并上市的行为，就叫做**IPO**。

## 股票的分类

我们已经知道了股票是什么以及公司如何上市，接下来我们系统地看看股票有哪些分类方式。

股票主要有两种分类方式。

### 按业绩分类

按公司业绩表现，股票常见分为以下三类：

*   **蓝筹股**：指资本雄厚、信誉优良的大公司股票。例如**中石油、中石化**。这类公司体量巨大，经营稳定。
*   **绩优股**：指业绩优良公司的股票。它们可能规模不是最大，但盈利能力持续强劲。例如**贵州茅台**。
*   **ST股**：中文名为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记，以警示投资者该公司存在风险。

### 按上市地区分类

根据公司上市交易的地点和交易货币，股票可分为：

*   **A股**：在中国大陆（上海、深圳证券交易所）上市，以**人民币**认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以**外币**（如美元、港币）认购和交易的股票。
*   **H股**：在中国**香港**上市的公司股票。
*   **N股**：在美国**纽约**上市的公司股票。
*   **S股**：在新加坡上市的公司股票。

### 不同股市的交易规则

不同地区的股市有其特定的交易规则，这对于量化交易策略的设计至关重要。以下是几个关键规则：

*   **A股规则**：
    *   **涨跌幅限制**：为防止股价剧烈波动，保护投资者，A股设有每日**10%**的涨跌幅限制。公式表示为：
        `今日最高价 <= 昨日收盘价 * 1.1`
        `今日最低价 >= 昨日收盘价 * 0.9`
    *   **T+1交割制度**：当日（T日）买入的股票，需到下一个交易日（T+1日）才能卖出。这旨在减少市场投机行为。
*   **B股规则**：
    *   同样采用**T+1**交割制度。
    *   资金实行**T+3**交收，即卖出股票后，资金需3个交易日才能提现。
*   **港股（H股）/美股（N股）规则**：
    *   **T+0交易**：当日买入的股票可以当日卖出。
    *   **无涨跌幅限制**：股价在交易日内可以自由波动。
