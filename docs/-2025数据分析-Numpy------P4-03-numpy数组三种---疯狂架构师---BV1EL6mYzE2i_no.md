## NumPy模块简介 📊

![](img/999b39ff16cdf7a46fa5f328c0a2c929_7.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_9.png)

在开始学习具体操作前，先对这三个模块进行初步介绍。需要明确的是，**Pandas模块是学习的重点和核心操作对象**。在数据分析中，重点讲解和使用的都是Pandas模块。相比之下，NumPy和Matplotlib虽然也会用到，但其重要性不及Pandas。因此，本课程的中心是学习Pandas模块。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_11.png)

首先介绍NumPy模块的基本使用。NumPy模块在后期的学习中，主要是为Pandas模块提供数据源。等我们学完Pandas后，你就会明白什么是数据源，以及NumPy如何为Pandas提供数据源。

大致介绍一下NumPy模块。你可以将NumPy理解为一个**一维或多维的数组**。如果你有其他编程语言（如C、Java）的经验，一定知道数组的概念。如果只接触过Python，Python中没有传统意义上的数组，但我们学过列表。如果你不知道什么是数组，可以暂时把数组看作一种特殊的列表，或者把列表看作一种特殊的数组。无论是列表还是数组，它们都表示一种数据结构，即一种可以装载数据的容器。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_13.png)

因此，NumPy模块对应的容器叫做**数组**，这个数组可以装载相关数据。暂时，我们先把NumPy模块当作一个一维或多维的数组。

那么，什么是一维，什么是多维呢？例如，我们定义一个列表 `a_list = [1, 2, 3]`，可以暂且称这个列表为一维列表。何为二维列表？例如 `a_list = [[1, 2, 3], [4, 5, 6]]`，这是列表套列表，我们称之为二维列表。三维列表的含义类似，即列表套列表再套列表。这里举的是列表的例子，我们的数组也是一样的，有一维数组，也有多维数组。

数组与列表肯定有区别，至于区别是什么，稍后会介绍。所以，暂时你只需把NumPy模块当作一个一维或多维的数组即可。这个一维或多维的数组主要装载**数值型数据**。因此，NumPy模块的重点操作在于它对数值进行运算。也就是说，在后期的数据分析中，如果我们需要对一组数值进行某种形式的运算，可以先将待计算的数值装载到NumPy数组中，再基于NumPy数组的相关特性进行运算。

综上所述，NumPy模块的重点在于两点：
1.  它是一个容器，装载的数据是**数值型数据**，并主要用于对这些数值型数据进行运算。
2.  NumPy模块是Python**科学计算库的基础**。这意味着NumPy模块中已经为我们封装好了各种各样基于科学计算的操作工具类。例如，后期想对一组数据进行反三角函数的运算，我们不需要手动实现，直接调用NumPy中封装好的工具类即可。

这就是NumPy模块的基本介绍。

## 创建NumPy数组的三种方式 🛠️

介绍完NumPy后，接下来我们看看如何使用NumPy数组。首先，你需要创建NumPy数组。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_15.png)

### 方式一：使用 `np.array()` 从列表创建

![](img/999b39ff16cdf7a46fa5f328c0a2c929_17.png)

要使用NumPy数组，首先需要导入NumPy模块。

```python
import numpy as np
```

业界约定俗成地使用 `np` 作为NumPy的别名。导入模块后，接下来创建数组。NumPy模块不需要单独安装，只要我们安装好了Anaconda这样的集成环境，其中就已经集成了各种必要的模块。

首先，创建一个一维数组。

```python
arr = np.array([1, 2, 3])
arr
```

我们调用 `np.array()` 函数。如果不知道如何使用，可以按 `Shift+Tab` 打开帮助文档。`array()` 函数的主要参数是 `object`，表示数据源。例如，我们传入列表 `[1, 2, 3]`。另一个参数 `dtype` 表示数据类型，这里我们先使用默认值。默认情况下，`dtype` 会根据传入数据的原始类型来确定数组元素的类型。

执行后，返回一个一维NumPy数组。在单元格中，如果想打印某个变量的值，不需要写 `print`，直接输入变量名并按执行快捷键即可显示内容。

能创建一维数组，同样可以创建多维数组。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_19.png)

```python
arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
arr_2d
```

![](img/999b39ff16cdf7a46fa5f328c0a2c929_21.png)

现在，数据源 `[[1, 2, 3], [4, 5, 6]]` 是一个二维列表，意味着创建好的 `arr_2d` 是一个二维数组。执行后，返回一个由两行三列组成的二维数组。一维表示一条线，二维表示一个平面（如表格）。因此，返回的两行三列的平面就是一个二维NumPy数组。当然，你也可以创建三维、五维甚至十维的数组，但最常用的是一维、二维和三维数组，因为维度再高就不便于人类理解了。

至此，我们掌握了第一种创建NumPy数组的方式。接下来，我们探讨一个问题：数组和列表到底有什么区别？

![](img/999b39ff16cdf7a46fa5f328c0a2c929_23.png)

我们说过数组可以是特殊的列表，列表也可以是特殊的数组，但它们之间一定有区别。让我们测试一下。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_25.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_27.png)

```python
arr_mixed = np.array([1, 2.2, 'three'])
arr_mixed
```

这个数组中存放了三个类型不同的元素：整型、浮点型和字符串型。打印创建好的数组，会发现数组元素的类型统一被强制转换成了字符串类型。你会奇怪，明明原始数据有整型、浮点型和字符串型，为什么最终都转成了字符串？这种情况在列表中不会发生，因为列表中的元素类型可以是不同的。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_29.png)

在数组的演示中，我们发现数组中存储的数组元素类型必须是统一的。如果原始元素类型不同，最终一定会被强制转换成同一类型。为什么会转成字符串呢？如果没有字符串类型会转成什么？

```python
arr_mixed2 = np.array([1, 2.2, 3.3])
arr_mixed2
```

执行后，都转成了浮点型。这意味着数据类型是有优先级的：字符串的优先级大于浮点型，浮点型大于整型。如果原始数据中有字符串类型，最终都转成字符串；如果没有字符串但有浮点型，都转成浮点型；如果只有整型，就保持整型。这就是列表和数组的一个主要区别。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_31.png)

### 方式二：使用 `plt.imread()` 从图片创建

![](img/999b39ff16cdf7a46fa5f328c0a2c929_33.png)

接下来，我们看看NumPy数组的另一种创建形式。我们有一个需求：将外部的一张图片读取并加载到NumPy数组中，然后尝试改变数组元素的数值，观察对原始图片的影响。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_35.png)

NumPy数组是一个容器，容器里装载数据。不同的容器装载数据，对数据的组织形式和可进行的操作是不同的。例如，列表以列表元素形式组织数据，字典以键值对形式组织数据。数据装载到不同的容器中，我们就可以基于该容器的特性对数据进行不同形式的处理。我们重点学习的是NumPy数组这个容器，看它有什么特性，以及基于这些特性可以对数组元素进行哪些处理。

那么，如何将外部的一张图片装载到NumPy数组中呢？在当前目录下，有一张图片 `1.jpg`。我们将 `1.jpg` 的图片内容读取到NumPy数组中。图片本质上是由二进制数据构成的，二进制数据也是数值型数据，可以装载到NumPy数组中。

在进行这个操作之前，我们需要接触另一个模块——Matplotlib，它是三剑客中的最后一个模块。后面我们会系统讲解，现在先初步使用一下。

```python
import matplotlib.pyplot as plt
```

![](img/999b39ff16cdf7a46fa5f328c0a2c929_37.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_39.png)

导入 `matplotlib.pyplot` 模块，并起别名为 `plt`。然后，使用 `plt.imread()` 方法读取图片。

```python
image_array = plt.imread('1.jpg')
image_array
```

`imread()` 读取完成后，会将读取的数据放在一个NumPy数组中。`image_array` 就是返回的数组，这个数组中装载的就是图片内容。我们打印 `image_array` 查看。

现在返回的是一个NumPy数组。它是几维的？一维NumPy数组只有一个中括号，二维有两个中括号，三维有三个中括号。这里返回的是一个三维NumPy数组，里面存储的数据就是读取出的图片数据。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_41.png)

这个三维NumPy数组表示图片数据，我们可以将其显示出来。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_43.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_45.png)

```python
plt.imshow(image_array)
```

`imshow()` 表示将NumPy数组进行可视化展示。刚好这里的NumPy数组存储的是一张图片数据，所以展示出来就是原始图片。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_47.png)

现在我们知道 `image_array` 里存储的是原始图片数据。如果让 `image_array` 减去100，表示将数组中的每一个元素都减去100。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_49.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_51.png)

```python
image_array_modified = image_array - 100
plt.imshow(image_array_modified)
```

现在 `image_array_modified` 中存储的还是图片的原始数据吗？不是了，因为我们对图片原始数据进行了数值相减的操作。这会对图片数据产生什么影响？再次显示这张图片，你会发现它与原始图片有明显的差异。这个差异就源于我们将图片的原始数据都减去了100。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_53.png)

你会发现，现在我们似乎对图片做了某种处理。例如，之前用的美图秀秀等修图软件，为什么可以对图片进行处理？处理的本质原理不就是对图片的数据进行某种特定形式的处理和运算吗？比如瘦脸，一定是通过特定算法定位图片中脸部的数据，然后对其进行某种算术运算，从而达到瘦脸效果。所谓的瘦脸效果，也是对整个图片脸部的数据进行了某种形式的处理和运算。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_55.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_57.png)

由此我们知道，使用 `imread()` 可以返回一个NumPy数组，并且数组中的内容源于外部的一张图片。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_59.png)

### 方式三：使用NumPy内置函数创建

最后一种创建数组的方式，是使用NumPy提供的一些内置函数。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_65.png)

例如，`np.ones()` 函数。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_67.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_69.png)

```python
np.ones((3, 4))
```

![](img/999b39ff16cdf7a46fa5f328c0a2c929_71.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_73.png)

`ones()` 需要传入一个 `shape` 参数，表示形状。`(3, 4)` 表示三行四列。执行后，返回一个三行四列的二维数组，并且数组内部填充的元素都是1。显然，`np.zeros()` 填充的就都是0了。

`zeros()` 和 `ones()` 用得不多，我们重点从 `linspace` 开始。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_75.png)

```python
np.linspace(0, 100, 20)
```

`linspace()` 需要传入参数：`start`（起始值）、`stop`（结束值）和 `num`（数量）。这里从0开始，到100结束，数量为20。执行后，返回一个一维数组，数组元素有20个。这意味着 `num` 表示返回数组元素的个数。观察数组，你会发现每两个元素之间的差是固定的。因此，`linspace()` 返回的是一个一维的等差数列数组，它只能返回一维数组，不能返回多维数组。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_77.png)

除此之外，还有 `arange()` 函数。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_79.png)

```python
np.arange(10, 50, 2)
```

![](img/999b39ff16cdf7a46fa5f328c0a2c929_81.png)

`arange()` 的参数是：`start`（起始值）、`stop`（结束值）和 `step`（步长）。这里从10开始，到50结束，步长为2。执行后，返回从10到50、每隔2取一个值所组成的数列。它返回的也是一个一维的等差数列数组，同样只能是一维的。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_83.png)

你会发现，`linspace` 和 `arange` 返回的都只能是一维数组。而 `zeros` 和 `ones` 都有 `shape` 参数，可以指定维度，因此可以返回一维或多维数组。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_85.png)

最后一个函数是 `random`。

```python
np.random.randint(0, 100, size=(5, 3))
```

`random.randint()` 用于返回随机数组。参数 `low` 是最小值，`high` 是最大值，`size` 是形状（与 `shape` 相同）。这里从0开始，到100结束，形状为 `(5, 3)`，即五行三列。这意味着返回一个五行三列的二维数组，数组元素是0到100之间的随机整数。每次执行，数组元素都会变化。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_87.png)

![](img/999b39ff16cdf7a46fa5f328c0a2c929_89.png)

通过这几个函数的介绍，意味着我们也可以通过这些函数创建一维或多维的NumPy数组。

![](img/999b39ff16cdf7a46fa5f328c0a2c929_91.png)
