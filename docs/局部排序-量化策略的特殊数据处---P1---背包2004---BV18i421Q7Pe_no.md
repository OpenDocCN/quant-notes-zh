## 核心逻辑：为何使用排名？
上一节我们介绍了使用标准化来处理特征。本节中我们来看看另一种思路。

![](img/a17bac1222aa3d333c323521fe239c44_3.png)

在股票投资中，我们的决策通常基于**比较和选择**。例如，我们构建模型预测股票的未来收益，最终目的是为了**选出收益最高的几只股票**进行投资。我们并不需要精确知道每只股票的具体收益数值，只需要知道它们的**相对排名**。

![](img/a17bac1222aa3d333c323521fe239c44_5.png)

这个逻辑同样适用于**因子**（特征）。例如，我们关心一只股票的市盈率（PE），但更重要的是该股票的市盈率在**当天整个市场所有股票中处于什么位置**，而不是其具体的PE数值。因此，将因子值转换为局部排名，可能比标准化更能反映其在投资决策中的真实意义。

![](img/a17bac1222aa3d333c323521fe239c44_7.png)

![](img/a17bac1222aa3d333c323521fe239c44_9.png)

![](img/a17bac1222aa3d333c323521fe239c44_11.png)

![](img/a17bac1222aa3d333c323521fe239c44_13.png)

**核心公式/概念**：
*   **全局排名**：在整个历史数据集的所有样本中进行排序。`rank(all_data)`
*   **局部排名**：在**每个交易日（date）** 内，对所有股票进行排序。`rank(data.groupby('date'))`

![](img/a17bac1222aa3d333c323521fe239c44_15.png)

![](img/a17bac1222aa3d333c323521fe239c44_17.png)

## 实现步骤：对目标变量和因子进行局部排序
理解了局部排名的逻辑后，我们开始动手实现。我们将对目标变量（未来收益）以及选定的因子（如市盈率、市值）进行转换。

![](img/a17bac1222aa3d333c323521fe239c44_19.png)

![](img/a17bac1222aa3d333c323521fe239c44_21.png)

![](img/a17bac1222aa3d333c323521fe239c44_23.png)

以下是实现局部排名的关键步骤：

1.  **按日期分组**：因为我们需要的是每个交易日内所有股票的相对排名，所以分组键是日期（`date`），而不是股票代码。
2.  **计算排名百分比**：由于不同交易日的股票数量可能不同（例如，有股票停牌或新上市），直接使用排名序号（如第1名、第100名）不具有可比性。因此，我们计算**排名百分比**（`pct=True`），即该股票排名位置占总股票数的比例。这确保了结果始终在0到1之间，具有统一尺度。

![](img/a17bac1222aa3d333c323521fe239c44_25.png)

### 对目标变量（未来收益）进行局部排序
我们首先处理要预测的目标——未来30天的收益。

![](img/a17bac1222aa3d333c323521fe239c44_27.png)

![](img/a17bac1222aa3d333c323521fe239c44_29.png)

![](img/a17bac1222aa3d333c323521fe239c44_31.png)

```python
# 假设 df 是包含 ‘date', ‘stock_code', ‘return_30d' 等列的DataFrame
df[‘return_rank'] = df.groupby(‘date')[‘return_30d'].rank(pct=True)
```
这段代码为每一行数据（即某只股票在某一天）的 `return_30d` 值，在其所属的交易日分组内计算百分比排名，结果存入新列 `return_rank`。

### 对特征因子进行局部排序
接下来，我们以市盈率（PE）和市值（Market Cap）为例，对它们进行同样的处理。

![](img/a17bac1222aa3d333c323521fe239c44_42.png)

![](img/a17bac1222aa3d333c323521fe239c44_44.png)

以下是需要转换的因子列表：
*   **市盈率（PE）**：`df[‘pe_rank'] = df.groupby(‘date')[‘pe'].rank(pct=True)`
*   **市值（Market Cap）**：`df[‘market_cap_rank'] = df.groupby(‘date')[‘market_cap'].rank(pct=True)`

运行这些代码后，我们就得到了三个新的、经过局部排序处理的特征：`return_rank`, `pe_rank`, `market_cap_rank`。

## 使用排名特征构建模型
现在，我们已经准备好了新的特征。接下来，我们将使用这些排名特征来训练一个简单的线性回归模型。

![](img/a17bac1222aa3d333c323521fe239c44_62.png)

![](img/a17bac1222aa3d333c323521fe239c44_64.png)

![](img/a17bac1222aa3d333c323521fe239c44_66.png)

1.  **准备数据**：
    *   **特征矩阵（X）**：使用 `pe_rank` 和 `market_cap_rank`。
    *   **目标变量（y）**：使用 `return_rank`。
2.  **训练模型**：将数据分割为训练集和测试集，使用线性回归进行拟合。
3.  **模型评估**：观察模型的决定系数（R²），与使用原始值或标准化值训练的模型进行对比。在本例中，使用排名特征后，R² 值有轻微提升。
