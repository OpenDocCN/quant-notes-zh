![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_3.png)

## 什么是最大回撤？📊

上一节我们介绍了策略分析中的各种指标，本节中我们来看看最大回撤。它描述的是投资过程中“最惨”的阶段。具体来说，它衡量的是从任意一个历史高点开始，到后续任意一个低点为止，资产价值下跌的最大幅度。

其核心公式如下：

**最大回撤 = max( (Pi - Pj) / Pi )**

其中：
*   **Pi** 代表某个历史时间点 `i` 的资产价值（峰值）。
*   **Pj** 代表在时间点 `i` 之后的某个时间点 `j` 的资产价值（谷底）。

这个公式计算的是从峰值 `Pi` 到后续谷底 `Pj` 的相对跌幅。我们需要遍历所有可能的 `i` 和 `j`（`j > i`），找出其中最大的那个跌幅值。

## 最大回撤的意义 ⚠️

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_5.png)

最大回撤是一个**风险指标**，而非盈利指标。它回答的问题是：在执行这个策略时，投资者可能经历的最大、最痛苦的亏损幅度有多大。

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_7.png)

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_9.png)

即使一个策略长期来看是盈利的，但如果它的最大回撤非常大（例如资产价值腰斩），那么在执行过程中，投资者可能需要承受巨大的心理压力和市场风险。了解最大回撤有助于投资者评估自己能否承受策略带来的潜在风险。

## 如何计算最大回撤？🧮

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_11.png)

理解了概念后，我们来看看如何用Python进行计算。核心思路是：先找出每个时间点之前的最高峰值，然后用这个峰值减去当前值，再除以峰值，最后找出所有计算结果中的最大值。

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_13.png)

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_15.png)

以下是计算最大回撤的步骤：

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_17.png)

1.  **计算累积最大值**：对于时间序列中的每一个点，计算从开始到该点为止的最大值。
2.  **计算回撤**：用每个点的“累积最大值”减去该点的“实际值”，得到下跌的绝对幅度。
3.  **计算相对回撤**：将下跌的绝对幅度除以对应的“累积最大值”，得到相对跌幅。
4.  **找出最大值**：在所有相对跌幅中，找出最大的一个，即为最大回撤。

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_19.png)

我们可以使用Pandas库的 `cummax()` 函数来高效地完成第一步。

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_21.png)

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_23.png)

```python
import pandas as pd

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_25.png)

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_27.png)

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_29.png)

# 假设 df[‘close’] 是包含每日收盘价的时间序列数据
# 1. 计算累积最大值
cumulative_max = df[‘close’].cummax()

# 2. 计算每日的回撤（峰值 - 当前值）
drawdown = cumulative_max - df[‘close’]

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_31.png)

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_33.png)

# 3. 计算每日的相对回撤（回撤 / 峰值）
relative_drawdown = drawdown / cumulative_max

![](img/72da9f78ef2a91e10c6f8e3a5c092a8a_35.png)

# 4. 找出最大回撤
max_drawdown = relative_drawdown.max()
print(f“最大回撤为：{max_drawdown:.2%}”)
```
