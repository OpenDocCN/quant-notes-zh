## 计算基础收益率

上一节我们介绍了如何通过股价数据生成交易信号。本节中我们来看看如何计算策略的收益。首先，我们需要计算股票每日的基础收益率。

在量化分析中，我们通常使用对数收益率，因为它具有更好的数学性质（如可加性）。计算对数收益率的公式如下：

![](img/47b53f74372c78459e18c31df7d98674_5.png)

`log_return = log(price_t / price_{t-1})`

![](img/47b53f74372c78459e18c31df7d98674_7.png)

其中，`price_t` 代表当前价格，`price_{t-1}` 代表前一天的价格。在Python的Pandas库中，我们可以通过以下代码实现：

![](img/47b53f74372c78459e18c31df7d98674_9.png)

```python
# 假设 df[‘Close’] 是股价序列
returns = np.log(df[‘Close’] / df[‘Close’].shift(1))
```

![](img/47b53f74372c78459e18c31df7d98674_11.png)

执行这段代码后，我们得到了一个名为 `returns` 的序列，它代表了股票每日的增长率。

![](img/47b53f74372c78459e18c31df7d98674_13.png)

![](img/47b53f74372c78459e18c31df7d98674_15.png)

## 结合策略信号计算收益

接下来，我们需要将策略信号与基础收益率结合起来。我们之前构建了一个 `position` 序列，其中值为1代表我们跟随原始趋势（即做多），值为-1代表我们反转趋势（即做空）。

![](img/47b53f74372c78459e18c31df7d98674_17.png)

以下是结合策略的核心逻辑：
*   当 `position` 为 1 时，我们的策略收益等于基础收益率。
*   当 `position` 为 -1 时，我们的策略收益等于基础收益率的相反数（即 `-returns`）。

![](img/47b53f74372c78459e18c31df7d98674_19.png)

因此，策略收益可以通过将 `position` 序列与 `returns` 序列逐元素相乘来计算。同样，我们需要对 `position` 序列进行 `shift(1)` 操作，以确保信号作用于正确的交易日。

![](img/47b53f74372c78459e18c31df7d98674_21.png)

![](img/47b53f74372c78459e18c31df7d98674_23.png)

```python
# 计算策略收益
strategy_returns = position.shift(1) * returns
```

![](img/47b53f74372c78459e18c31df7d98674_25.png)

计算完成后，我们可以查看 `strategy_returns` 序列。在数据前期，由于信号多为1，策略收益可能与基础收益相似。

![](img/47b53f74372c78459e18c31df7d98674_27.png)

![](img/47b53f74372c78459e18c31df7d98674_29.png)

## 计算并比较最终累积收益

![](img/47b53f74372c78459e18c31df7d98674_31.png)

为了评估策略的最终效果，我们需要将每日的收益率累积起来，还原为总资产的变化。这需要两个步骤：
1.  对收益率序列进行累加。
2.  将对数收益率通过指数函数还原为实际的总资产倍数。

以下是计算最终累积收益的代码：

![](img/47b53f74372c78459e18c31df7d98674_33.png)

```python
# 计算原始“买入并持有”策略的最终价值
cumulative_returns = np.exp(returns.cumsum())
final_value_buy_hold = cumulative_returns.iloc[-1]

![](img/47b53f74372c78459e18c31df7d98674_35.png)

![](img/47b53f74372c78459e18c31df7d98674_37.png)

# 计算双均线策略的最终价值
cumulative_strategy_returns = np.exp(strategy_returns.cumsum())
final_value_strategy = cumulative_strategy_returns.iloc[-1]
```

![](img/47b53f74372c78459e18c31df7d98674_39.png)

执行代码后，我们可以比较两个结果。例如，结果显示：
*   原始“买入并持有”策略：1元本金最终变为约4元。
*   双均线策略：1元本金最终变为约5.8元。

![](img/47b53f74372c78459e18c31df7d98674_41.png)

这表明，在我们测试的这段时期内，应用双均线策略获得了比单纯持有股票更高的收益。

![](img/47b53f74372c78459e18c31df7d98674_43.png)
