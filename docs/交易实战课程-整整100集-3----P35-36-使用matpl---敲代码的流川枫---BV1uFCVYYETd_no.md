# 量化交易实战课程：P35：使用matplotlib绘制数学函数图像 📈

在本节课中，我们将学习如何使用Python的matplotlib库来绘制数学函数的图像。核心思想是：计算机通过绘制一系列密集的点来“模拟”出平滑的曲线。我们将通过绘制几个具体的函数来掌握这一方法。

## 绘制原理

上一节我们介绍了基本的绘图方法，本节中我们来看看如何绘制数学函数。计算机屏幕由像素点构成，它本身无法直接绘制完美的曲线或圆形。所谓的“线”或“曲线”，实际上是由一系列非常密集的点连接而成。例如，在画图软件中放大一个圆，会发现它是由许多小点组成的。

![](img/3b56030abbf55a034c03d82c73d487b2_1.png)

因此，要绘制函数 `y = f(x)` 的图像，我们需要：
1.  生成一个在X轴定义域内非常密集的点序列（X值列表）。
2.  根据函数公式，计算出每个X值对应的Y值。
3.  将这些（X, Y）点对传递给绘图函数，即可得到近似的函数曲线。

## 生成密集点序列

以下是生成密集点序列的关键步骤，我们将使用NumPy库的 `linspace` 函数。

![](img/3b56030abbf55a034c03d82c73d487b2_3.png)

`linspace` 函数用于在指定的区间内生成均匀间隔的数值。其基本语法为：
```python
numpy.linspace(start, stop, num)
```
*   `start`：序列的起始值。
*   `stop`：序列的结束值。
*   `num`：要生成的样本数量。

例如，要在区间 `[-100, 100]` 内生成10000个点，可以这样写：
```python
import numpy as np
x = np.linspace(-100, 100, 10000)
```
这样，我们就得到了一个包含10000个均匀分布在-100到100之间的数值数组。点数越多，最终绘制的曲线就越平滑。

## 实战演练：绘制三个函数

接下来，我们以三个具体函数为例进行绘制：
1.  `y1 = x`
2.  `y2 = x^2`
3.  `y3 = 3*x^3 + 5*x^2 + 2*x + 1`

![](img/3b56030abbf55a034c03d82c73d487b2_5.png)

我们将使用相同的X值数组来计算这三个函数的Y值。

![](img/3b56030abbf55a034c03d82c73d487b2_7.png)

### 1. 计算Y值数组

首先，我们根据上面生成的 `x` 数组，利用NumPy的数组运算能力，高效地计算出三个Y值数组。
```python
import numpy as np
import matplotlib.pyplot as plt

# 生成X值数组
x = np.linspace(-100, 100, 10000)

# 计算三个函数的Y值数组
y1 = x                     # y = x
y2 = x ** 2                # y = x^2
y3 = 3 * x**3 + 5 * x**2 + 2 * x + 1  # y = 3x^3 + 5x^2 + 2x + 1
```
NumPy允许直接对数组进行平方、立方等数学运算，这比使用循环效率高得多。

### 2. 绘制图像并添加样式

计算完数据后，我们使用 `plt.plot()` 函数分别绘制三条曲线，并为它们设置不同的颜色和标签以便区分。
```python
# 绘制第一条曲线：y = x
plt.plot(x, y1, color='red', label='y = x')

# 绘制第二条曲线：y = x^2
plt.plot(x, y2, color='green', label='y = x^2')

# 绘制第三条曲线：y = 3x^3 + 5x^2 + 2x + 1
plt.plot(x, y3, color='blue', label='y = 3x^3 + 5x^2 + 2x + 1')

# 添加图例
plt.legend()

# 显示图形
plt.show()
```

### 3. 处理图像显示问题

在绘制时，可能会因为不同函数的数值范围差异巨大而导致某些曲线在图中显示不明显（例如，立方函数增长极快，在图中可能呈现为一条贴近X轴的直线）。

为了解决这个问题，我们可以手动设置图形的Y轴显示范围，使用 `plt.ylim()` 函数。
```python
# 设置Y轴的显示范围为 -1000 到 1000
plt.ylim(-1000, 1000)
```
将此行代码添加到 `plt.legend()` 之前，可以强制图表在指定的Y值范围内显示，从而使所有曲线都能清晰可见。根据需要，你也可以使用 `plt.xlim()` 来调整X轴的显示范围。

## 总结

![](img/3b56030abbf55a034c03d82c73d487b2_9.png)

本节课中我们一起学习了使用matplotlib和NumPy绘制数学函数图像的方法。核心步骤可以总结为以下三步：
1.  使用 `np.linspace()` 生成定义域内密集的X值数组。
2.  利用NumPy的数组运算，根据函数公式 `y = f(x)` 计算出对应的Y值数组。
3.  使用 `plt.plot()` 绘制 `(x, y)` 点，并通过设置颜色、标签、坐标轴范围等让图像更清晰美观。
