# 从零开始量化：第3课：数据类型、运算符、变量与参数详解 🧮

在本节课中，我们将深入学习量化编程的基础核心概念。我们将详细探讨数据类型、各种运算符的用法、变量与参数的区别，并理解序列函数与简单函数的工作原理。这些知识是构建任何交易策略的基石。

## 数据类型详解 🔢

上一节我们介绍了语言的基本要素，本节中我们来看看构成代码基础的数据类型。在编程中，数据类型定义了数据的种类以及可以对其执行的操作。

在MC中，主要有三种基本数据类型：

1.  **数值**：用于存储数字。在计算机底层，数值以二进制形式（0和1）存储。为了高效地存储整数和小数，计算机科学引入了“浮点数”的概念。浮点数并非固定分配存储空间给整数和小数部分，而是可以“浮动”分配，从而在数值范围和精度之间取得平衡。其底层原理涉及复杂的科学计算，曾获得诺贝尔奖级别的认可。需要注意的是，由于二进制表示法的限制，计算机在处理某些小数（如0.1）时可能存在微小的精度误差。
    *公式示例：`数值变量 = 3.14`*

2.  **字符串**：用于存储文本信息。在MC中，字符串必须用英文半角双引号引起来。需要注意的是，MC平台目前不支持直接输出或使用中文字符串，注释也建议使用英文。
    *代码示例：`字符串变量 = “HelloMC”`*

3.  **布尔**：用于表示逻辑真或假。布尔值只有两个：`True`（真）和`False`（假）。它通常是一个条件判断的结果。
    *代码示例：`布尔变量 = 收盘价 > 开盘价` // 如果成立，则`布尔变量`的值为`True`。

为了加深理解，我们可以通过一个简单的例子来演示布尔类型：

```pascal
Variables: 条件判定(false);
条件判定 = 1 < 5; // 判断1是否小于5，结果为真
Print(条件判定); // 输出结果为 True
```

## 运算符介绍 ➕➖✖️➗

理解了数据类型后，我们需要知道如何操作它们。运算符就是用于执行各种运算的符号。MC中的运算符主要分为四类。

![](img/dbe628039b6cdb0f30ce0818b6685275_1.png)

![](img/dbe628039b6cdb0f30ce0818b6685275_3.png)

以下是各类运算符的说明：

*   **数学运算符**：用于基本的算术计算，包括加(`+`)、减(`-`)、乘(`*`)、除(`/`)和括号(`()`)。它们**只能**应用于数值类型。需要特别注意：除数不能为零。
*   **关系运算符**：用于比较两个值之间的关系，结果为布尔值(`True`或`False`)。包括小于(`<`)、大于(`>`)、小于等于(`<=`)、大于等于(`>=`)、等于(`=`)和不等于(`<>`)。
    *注意：在MC中，赋值和相等比较都使用单等号(`=`)，这与许多其他编程语言不同。*
*   **逻辑运算符**：用于连接多个布尔条件。`And`表示“且”（所有条件都需成立），`Or`表示“或”（至少一个条件成立即可）。
*   **字符串运算符**：目前主要是加号(`+`)，用于将两个字符串连接（拼接）成一个新的字符串。
    *代码示例：`“策略” + “信号”` 的结果是 `“策略信号”`。

## 变量与参数 📦

在程序中，我们需要容器来存储和操作数据。变量和参数就是这样的容器，但它们在性质上有着关键区别。

以下是变量与参数的核心对比：

*   **变量**：是一个存储工具，其值可以在程序运行过程中**动态变化**。例如，一条移动平均线的值会随着每根新K线的到来而不断更新。你可以把它想象成一个贴有名字标签的盒子，里面的内容可以随时更换。
    *代码示例：`Variables: 动态均价(0); 动态均价 = (最高价 + 最低价) / 2;`*
*   **参数**：在MC中称为`Input`，是一种**常量**。在单次策略运行过程中，其值**固定不变**。它通常用于定义策略中需要反复调整的固定值，如均线周期、止损比例等。使用参数的好处是，你可以在不修改核心代码的情况下，通过策略设置界面轻松调整这些值，便于回测和优化。
    *代码示例：`Input: 均线周期(20);` // 定义一个名为“均线周期”的参数，默认值为20，在代码运行中不可更改。

**为什么需要参数？** 假设你的策略中在多处使用了“20”这个均线周期。如果直接写数字20，当你想测试30周期时，就需要找到所有“20”并逐一修改，极易出错。而使用参数`均线周期`，你只需在参数设置框中将20改为30即可，代码会自动应用，安全又高效。

## 序列函数与简单函数 🔄

函数是可重复使用的代码块。在MC中，函数根据其是否依赖历史数据进行计算，被分为“序列函数”和“简单函数”，这是一个重要概念。

两者的区别如下：

*   **简单函数**：函数内部**不引用**自身的历史值或内部变量的历史值。它仅在每次被调用时，基于当前的输入值计算并返回一个结果。计算速度快，占用资源少。
    *示例：一个计算`(最高价 - 最低价)`的函数，它只用到当前K线的数据。*
*   **序列函数**：函数内部**需要引用**自身的历史返回值或内部变量/数组的历史值。这意味着它需要为每一根K线存储一系列历史值。计算相对复杂，但能实现如移动平均线这样的需要历史状态的功能。
    *示例：指数移动平均线(`XAverage`)的计算。当前K线的EMA值，依赖于前一根K线的EMA值，这就是一个典型的序列函数。*
    *公式示意：`当前EMA = (价格 - 前一期EMA) * 平滑系数 + 前一期EMA`*

MC编译器通常会**自动判断**一个函数是简单函数还是序列函数。如果你的函数代码中使用了中括号`[]`来引用历史数据（如`Close[1]`），或者函数返回值中包含了函数自身（递归），那么它就会被识别为序列函数。
