## 模型训练与历史数据拟合

构造出模型后，我们可以使用Facebook Prophet框架内置的函数进行训练和深入分析。

![](img/a762eb52745c074b4ef50929088eaf35_35.png)

以下是构建和训练模型的核心步骤：

1.  **实例化模型**：首先，我们创建一个Prophet模型实例。在实例化时，可以指定一系列参数，例如是否绘制季节性变化图。一个关键参数是`changepoint_prior_scale`，它控制模型对趋势变化的灵活度，默认值为`0.05`。
    ```python
    model = Prophet(changepoint_prior_scale=0.05, yearly_seasonality=True)
    ```
2.  **准备训练数据**：我们使用历史数据进行训练。可以通过`training_years`参数指定使用最近几年的数据。例如，使用最近3年的数据进行训练。
    ```python
    train_data = data[-3*365:] # 假设数据按天记录，取最近3年
    ```
3.  **拟合模型**：将训练数据传入模型进行拟合。
    ```python
    model.fit(train_data)
    ```
4.  **生成预测框架**：使用`make_future_dataframe`函数生成一个包含历史及未来日期的数据框架。参数`periods`指定需要预测的未来天数。当`periods=0`时，表示只对历史数据进行拟合和预测，不预测未来。
    ```python
    future = model.make_future_dataframe(periods=0)
    ```
5.  **进行预测**：对生成的数据框架进行预测。预测结果会包含`yhat`（预测值）、`yhat_lower`和`yhat_upper`（置信区间上下界）等列。
    ```python
    forecast = model.predict(future)
    ```

## 突变点检测

![](img/a762eb52745c074b4ef50929088eaf35_79.png)

![](img/a762eb52745c074b4ef50929088eaf35_81.png)

![](img/a762eb52745c074b4ef50929088eaf35_83.png)

在时间序列中，趋势发生显著变化的点被称为突变点。Prophet可以自动检测这些点。

![](img/a762eb52745c074b4ef50929088eaf35_85.png)

![](img/a762eb52745c074b4ef50929088eaf35_87.png)

突变点可能表现为：由下降转为上升、上升速度突然加快、或下降速度突然放缓等。Prophet通过观察序列趋势的二阶导数变化来识别这些点。

![](img/a762eb52745c074b4ef50929088eaf35_89.png)

![](img/a762eb52745c074b4ef50929088eaf35_91.png)

![](img/a762eb52745c074b4ef50929088eaf35_93.png)

以下是检测和可视化突变点的过程：

![](img/a762eb52745c074b4ef50929088eaf35_95.png)

![](img/a762eb52745c074b4ef50929088eaf35_97.png)

![](img/a762eb52745c074b4ef50929088eaf35_99.png)

1.  **获取突变点**：训练好的模型对象中包含了检测到的突变点信息。
    ```python
    changepoints = model.changepoints
    ```
2.  **分析变化方向**：对于每个突变点，可以计算其趋势变化率（相当于二阶导数的符号）。变化率为正表示在该点趋势转为向上或加速向上；为负则表示趋势转为向下或加速向下。
3.  **可视化**：在原始序列图上，可以用不同的颜色（如绿色代表正向变化，红色代表负向变化）的竖线标记出这些突变点。

## 未来趋势预测

![](img/a762eb52745c074b4ef50929088eaf35_113.png)

预测未来是时间序列分析的核心应用。使用Prophet进行未来预测非常简单。

关键在于在生成预测数据框架时，将`periods`参数设置为一个大于0的值，例如180天。

![](img/a762eb52745c074b4ef50929088eaf35_115.png)

![](img/a762eb52745c074b4ef50929088eaf35_117.png)

![](img/a762eb52745c074b4ef50929088eaf35_119.png)

```python
future = model.make_future_dataframe(periods=180)
forecast = model.predict(future)
```

![](img/a762eb52745c074b4ef50929088eaf35_121.png)

这样生成的`forecast`数据框将包含历史日期和未来180天的日期。对于未来的日期，只有预测值（`yhat`）和置信区间，没有真实的观测值（`y`）。

![](img/a762eb52745c074b4ef50929088eaf35_123.png)

![](img/a762eb52745c074b4ef50929088eaf35_125.png)

![](img/a762eb52745c074b4ef50929088eaf35_131.png)

![](img/a762eb52745c074b4ef50929088eaf35_127.png)

将预测结果画出来，可以看到绿色预测线向未来延伸。模型基于历史规律，预测未来价格可能会先经历一段下降，然后再上升。预测的保守或激进程度，可以通过调整`changepoint_prior_scale`参数来控制。该值越大，模型对历史数据中的波动越敏感，预测可能更“奔放”；该值越小，模型越倾向于平滑的趋势，预测更“保守”。
