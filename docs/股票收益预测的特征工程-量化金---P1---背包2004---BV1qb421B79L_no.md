```python
def create_ta_features(df):
    # 对单个股票的数据框df进行计算
    ma5 = talib.SMA(df['close'], timeperiod=5)
    ma10 = talib.SMA(df['close'], timeperiod=10)
    # ... 可以计算更多指标
    # 将结果组合成一个新的DataFrame
    new_features = pd.concat([ma5, ma10], axis=1)
    new_features.columns = ['MA5', 'MA10'] # 指定列名
    return new_features
```

我们可以对目标变量进行两种处理，便于不同目的的分析：
*   **二值化**：将未来收益率转化为0/1标签（例如，大于0为1，否则为0），用于分类问题。
    ```python
    data['target_binary'] = data['future_10d_return'].apply(lambda x: 1 if x > 0 else 0)
    ```
*   **横截面排名**：在每个交易日，对所有股票的收益率进行排名（或百分位排名），将绝对收益转化为相对强弱，常用于多因子模型。
    ```python
    data['target_rank'] = data.groupby('date')['future_10d_return'].rank(pct=True)
    ```

最后，构建模型所需的特征矩阵`X`和目标向量`y`：
