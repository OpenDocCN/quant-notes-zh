## 策略框架概述

上一节我们介绍了用于量化分析的几个核心工具。本节中，我们来看看如何将这些工具组合起来，编写一个完整的量化交易策略。一个典型的策略框架主要由两部分组成：**初始化**和**策略函数**。

## 策略框架的组成部分

以下是策略框架的两个核心部分：

1.  **初始化**：这是整个策略开始运行时设置的内容，主要用于设定全局变量和参数。
2.  **策略函数**：这是策略的核心逻辑，会根据市场数据重复执行，生成交易信号并执行买卖操作。

接下来，我们将分别详细介绍这两个部分。

### 1. 初始化函数

初始化函数在整个策略的生命周期中**只会被执行一次**。它的主要作用是设定基准、定义交易标的、设置运行频率以及初始化一些全局变量。

在常见的量化平台（如聚宽）中，初始化函数通常名为 `initialize`，并接收一个名为 `context` 的全局对象作为参数。`context` 对象用于存储账户、持仓等信息。

以下是一个初始化函数的例子：

```python
def initialize(context):
    # 设定要交易的股票（这里以平安银行为例）
    context.security = ‘000001.XSHE’
    # 设定策略的运行频率（例如：每天运行）
    run_daily(trade, time=‘every_bar’)
```

![](img/3581a732647ea172b3445f9901e0e437_1.png)

在这段代码中：
*   `context.security` 是一个**全局变量**，存储了证券代码 `‘000001.XSHE’`（代表深交所的平安银行）。
*   `run_daily` 是一个API，用于设定策略函数 `trade` 每天运行。

### 2. 策略函数

策略函数是量化交易策略的**核心**。它会根据最新的市场数据（如价格、成交量）重复执行，判断买卖时机并下达交易指令。

策略函数的名称可以由用户自定义（例如 `trade`），它通常也接收 `context` 作为参数。平台会提供下单API（如 `order`）来执行买卖操作。

以下是一个策略函数的简单例子：

```python
def trade(context):
    # 买入1000股 context.security 指定的股票
    order(context.security, 1000)
    # 卖出800股 context.security 指定的股票
    order(context.security, -800)
```

这段代码演示了基本的买入（正数）和卖出（负数）操作。

## 一个简单的实用策略框架

了解了基本组成后，我们来看一个结合了初始化和策略函数的简单但实用的策略框架。

**策略逻辑如下：**
*   **初始化部分**：
    *   投资标的：平安银行 (`000001.XSHE`)。
    *   运行频率：每天运行一次。
*   **策略函数部分**：
    *   如果**上一个时间点的收盘价**高于**过去5日均线价格的1%**，则**全仓买入**。
    *   如果**上一个时间点的收盘价**低于**过去5日均线价格**，则**空仓卖出**（即卖出所有持仓）。
    *   在图表上画出上一个时间点的价格。

这个策略试图捕捉价格短期偏离均线后的回归机会。

## 实战编码演示

现在，我们进入实战环节，在量化平台（以聚宽为例）上实现上述策略。量化平台通常提供集成的开发环境（IDE）来编写和回测策略。

以下是实现该策略的核心代码步骤：

首先，定义初始化函数，设定股票和运行频率。

```python
def initialize(context):
    # 设定交易标的为平安银行
    g.security = ‘000001.XSHE’
    # 设定每天运行策略函数 trade
    run_daily(trade, time=‘every_bar’)
```

接着，定义核心的策略函数 `trade`。

![](img/3581a732647ea172b3445f9901e0e437_3.png)

```python
def trade(context):
    # 获取股票过去5天的收盘价数据
    stock_data = attribute_history(g.security, 5, ‘1d’, [‘close’])
    # 计算5日均价
    ma5 = stock_data[‘close’].mean()
    # 获取当前时间点的收盘价（即上一时间点价格）
    current_price = stock_data[‘close’][-1]
    # 获取当前账户的可用现金
    cash = context.portfolio.available_cash

    # 生成交易信号：当前价高于5日均价1%则全仓买入
    if current_price > ma5 * 1.01:
        # 计算可买数量（全仓）
        order_value(g.security, cash)
        # 记录日志
        log.info(‘买入’, g.security)
    # 生成交易信号：当前价低于5日均价则空仓卖出
    elif current_price < ma5:
        # 卖出所有持仓
        order_target(g.security, 0)
        # 记录日志
        log.info(‘卖出’, g.security)
    # 在图表上记录当前价格
    record(price=current_price)
```

代码说明：
*   `attribute_history`：用于获取历史数据。
*   `mean()`：计算均值，得到5日均价 `ma5`。
*   `context.portfolio.available_cash`：获取账户可用现金。
*   `order_value`：按金额下单。
*   `order_target`：调整持仓至目标数量（0代表清仓）。
*   `log.info`：输出运行日志。
*   `record`：记录数据用于绘图。

运行回测后，可以查看策略在指定时间段（例如2019-01-01至2020-06-30）的表现，包括收益率、与基准（如沪深300）的对比等分析图表。初学者可以通过这个简单案例理解策略从编写到回测的完整流程。
