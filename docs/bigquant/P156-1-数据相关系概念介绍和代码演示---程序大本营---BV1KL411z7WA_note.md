# 机器学习降维算法：P156：1-数据相关概念介绍和代码演示 📊

在本节课中，我们将要学习数据降维中的一个核心基础概念——相关性。理解变量之间的相关性是进行有效降维的关键第一步，它能帮助我们识别并剔除冗余信息，从而简化数据集，提升模型效率。

在许多研究与应用领域中，我们通常需要观测包含多个变量的数据。收集大量数据后，我们进行分析研究以寻找规律。例如，天文学家开普勒从其导师第谷手中获得了海量的天文观测数据。这些数据包含多个变量，如星星的位置、亮度等。开普勒通过数学方法对这些庞杂的数据进行总结归纳，最终发现了著名的开普勒三大定律。这个故事说明，从多变量数据中提炼关键信息至关重要。

对于多变量大数据集，丰富的信息无疑为研究和应用提供了更多可能性。然而，数据并非越多越好，数据的质量更为关键。我们希望从数据中“取其精华，弃其糟粕”。那么，如何定义精华与糟粕呢？这需要我们具备识别关键信息的能力。在数据中，变量与变量之间并非孤立存在，它们存在着**相关性**。如果盲目地减少指标（变量），可能会损失有用的信息，甚至导致错误的结论。

我们可以将机器学习算法比作烹饪工具，有普通的锅，也有高级的全自动炒菜锅。而**数据就是食材**。食材（数据）的质量直接决定了最终菜肴（模型结果）的品质。即使拥有最先进的工具（算法），如果食材（数据）本身不佳，结果也不会理想。因此，对数据进行恰当的处理，提取关键特征，是机器学习中至关重要的一环。

## 什么是相关性？🔗

上一节我们提到了数据间存在相关性，本节中我们来看看相关性具体指什么。

**相关性**描述了两个变量之间线性关系的强度和方向。例如：
*   **正相关**：一个变量增加时，另一个变量也倾向于增加。例如，身高和体重通常呈正相关。
*   **负相关**：一个变量增加时，另一个变量倾向于减少。例如，近视眼镜的度数越大，能清晰看到的距离通常越短（视力越差）。

首先，我们创建一个模拟的学生成绩数据集。

```python
import numpy as np
import pandas as pd

# 创建一个包含50名学生三门课程成绩的DataFrame
# 成绩范围在0到150之间
df = pd.DataFrame(
    np.random.randint(0, 150, size=(50, 3)),
    columns=[‘Python‘, ‘English‘, ‘Chinese‘]
)
print(df.head())
```

`DataFrame`是Pandas库中的一种二维表格型数据结构，类似于Excel表格，它由行和列组成，是数据处理中非常常用的结构。

接下来，我们使用Pandas的`.corr()`方法计算数据集中所有数值列两两之间的**皮尔逊相关系数**。

```python
# 计算相关性系数矩阵
correlation_matrix = df.corr()
print(correlation_matrix)
```

执行代码后，你会看到一个相关性系数矩阵。这个矩阵揭示了以下信息：
*   矩阵的对角线（自己与自己比较）值均为**1**，表示**完全正相关**。
*   其他位置的值表示不同列之间的相关性，其**取值范围在-1到1之间**。
    *   **1**：表示完全正相关。
    *   **-1**：表示完全负相关。
    *   **0**：表示没有线性相关性。
*   在我们的随机数据中，`Python`、`English`、`Chinese`成绩之间的相关系数可能接近0，说明它们之间没有明显的线性关系，这是符合随机数据预期的。

为了进一步理解，我们进行两组对比实验。

**实验一：添加无关列**
我们添加一列完全由常数（如150）组成的`Physics`成绩。由于这列数据没有波动，与其他变量的协方差为0，在计算相关系数时会产生`NaN`（非数字）结果。这说明了相关性衡量的是**协同变化**的关系。

**实验二：创建高度相关列**
现在，我们基于`Python`成绩列，创建一门新的`Math`成绩列。`Math`成绩是在`Python`成绩的基础上，加上一个在[-10, 10]范围内的随机波动生成的。

```python
# 创建与Python高度相关的Math列
df[‘Math‘] = df[‘Python‘].map(lambda x: x + np.random.randint(-10, 10, size=1)[0])
print(df[[‘Python‘, ‘Math‘]].head())

# 重新计算相关性矩阵
correlation_matrix_new = df.corr()
print(correlation_matrix_new)
```

观察新的相关性矩阵，你会发现`Python`和`Math`两列之间的相关系数**非常接近1**（例如0.99），这表明它们之间存在**高度正相关**。

## 相关性在降维中的应用 🎯

通过上面的演示，我们可以得出一个重要的降维思路：

**如果两个特征（变量）高度线性相关，那么它们所提供的信息在很大程度上是冗余的。** 例如，`Python`和`Math`成绩走势几乎一致，知道其中一个，就能在很大程度上预测另一个。

因此，在构建机器学习模型时，为了简化模型、减少计算量并避免“多重共线性”等问题，我们可以考虑剔除其中一个高度相关的特征，而不会显著损失信息。这就是降维算法（如主成分分析PCA）背后的基本动机之一——找到数据中最重要的、互不相关的“精华”部分。

以下是基于相关性进行特征筛选的基本逻辑：
1.  **计算**：计算数据集中所有特征之间的相关性矩阵。
2.  **识别**：找出相关系数绝对值非常接近1的特征对（例如 > 0.95 或 < -0.95）。
3.  **决策**：对于每一对高度相关的特征，根据业务知识或模型需求（如特征重要性）保留其中一个，删除另一个。
4.  **迭代**：重复此过程，直到没有高度相关的特征对为止。
