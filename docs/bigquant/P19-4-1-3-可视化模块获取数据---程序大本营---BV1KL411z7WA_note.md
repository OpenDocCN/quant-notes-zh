在本节课中，我们将学习如何使用可视化模块来获取平台上的金融数据。我们将介绍两种主要的数据获取方式：通过数据源模块获取原始行情数据，以及通过基础特征抽取模块获取因子库数据。课程将包含详细的步骤说明和操作演示，确保初学者能够轻松上手。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_3.png)

## 通过数据源模块获取金融数据 📈

![](img/0e12614f2f616e42bfd00a8ae6377dcd_5.png)

上一节我们介绍了可视化策略的基本概念，本节中我们来看看如何获取具体的金融数据。首先，我们将学习如何使用数据源模块来获取平台的原始行情数据。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_7.png)

我们需要在一个空白的可视化策略模板中，拖入三个模块：数据源模块、证券代码列表模块和输入特征列表模块。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_9.png)

以下是这三个模块的具体作用：
*   **数据源模块**：用于设置所要抽取的数据表名称。
*   **证券代码列表模块**：用于设置要抽取的股票范围和起止时间范围。
*   **输入特征列表模块**：用于设置需要获取的数据表中的具体字段。

我们来看一下具体的操作。

进入编写策略界面，然后点击“新建”。

点击“可视化空白策略”，在弹出的窗口中填入新建的策略名称，这里可以采用默认的“可视化策略”。

点击“新建”按钮，构建一个空白的可视化策略。界面左侧为模块列表，中间是画布区域，右侧是模块的属性栏。

在左侧的“数据输入输出”列表中，拖入“数据源模块”、“证券代码列表模块”以及“输入特征列表模块”。可以通过点击“自适应”按钮，让模块在画布中自适应显示。

将鼠标移动到“数据源模块”的输入节点上，可以看到提示需要连接“证券代码列表模块”。将鼠标移动到“证券代码列表模块”的输出节点，单击左键并拖拽到“数据源模块”的对应输入节点上，实现两个模块的连接。

用同样的方法，连接“输入特征列表模块”的输出节点到“数据源模块”的另一个输入节点。这样就完成了工作流的搭建。

模块连接后，需要设置模块的参数。首先点击“证券代码列表模块”，在右侧的属性栏中填入所需获取金融数据的时间范围。例如，填入 `2019-01-01` 到 `2019-06-01`。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_11.png)

“股票代码列表”参数如果不填写，则默认为起止时间范围内的所有股票。如果填写，可以采用分行格式来指定要抽取的股票范围。例如，可以设置为：
```
000001.SZA
600004.SHA
```
这代表需要抽取这两只股票的历史金融数据。

紧接着设置“数据源模块”。在“数据源的ID或别名”输入框中，需要输入所要抽取的数据表名称。我们可以通过文档的“数据字典-数据表”查看到对应的表类型。

例如，在“行情数据”中可以看到“A股行情日线数据”的表名为 `8e_dcn_stock_a`。复制这个表名，在数据源模块的“ID或别名”参数中输入它。这样就完成了数据源模块的参数填写。“开始日期”和“结束日期”会自动继承M2模块（即证券代码列表模块）的设置。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_13.png)

最后，在M3模块（输入特征列表模块）中填入所需要的数据表字段。例如，`8e_dcn_stock_a`（A股日线行情数据）表中的字段包括了 `open`（开盘价）和 `close`（收盘价）。我们取出这两个数据字段，并填入到输入特征列表中。

这样我们就完成了通过数据源模块获取行情数据的工作流编制。

## 通过基础特征抽取模块获取因子库数据 🔍

上一节我们介绍了如何获取原始行情数据，本节中我们来看看如何获取更复杂的因子库数据。我们可以通过“证券代码列表模块”、“输入特征列表模块”和“基础特征抽取模块”来实现因子库数据的抽取。

在“输入特征列表模块”中填入所需要抽取的因子名称。通过“证券代码列表模块”指定需要抽取因子数据的股票范围和起止时间。最后通过“基础特征抽取模块”实现因子数据的抽取。在该模块的“向前取天数”参数中，可以设置向前抽取的因子数据天数，以获取更多的历史数据。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_15.png)

在实际操作中，可以用类似的方法，在可视化策略画布中拖入“证券代码列表模块”、“输入特征列表模块”以及“基础特征抽取模块”。通过连线将三个模块连接在一起。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_17.png)

在“证券代码列表模块”中填入所需的时间范围和股票代码范围。这里依然填写 `2019-01-01` 到 `2019-06-01`。如果默认抽取全市场的股票范围数据，则“股票代码列表”中不填入任何信息。

在“输入特征列表模块”中填入所需要抽取的因子数据字段名。可以在文档的“因子库”说明中查看可以获取的因子字段名。

例如，需要获取每只股票的上市天数 `list_day_0`，可以填入该字段名。如果需要抽取对应的复权因子，可以填入对应的因子名称，如 `adjust_factor_1`（代表前一日的复权因子）。输入时，系统会自动提示进行因子补全，并显示每个因子的真实含义。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_19.png)

![](img/0e12614f2f616e42bfd00a8ae6377dcd_20.png)

填入输入特征列表后，就完成了因子数据抽取的流程搭建。

## 运行模块与查看结果 ▶️

工作流搭建完成后，我们需要运行模块来获取数据。可以通过右键点击“数据源模块”或“基础特征抽取模块”，选择“运行选中模块”来执行单个模块或整个工作流。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_22.png)

![](img/0e12614f2f616e42bfd00a8ae6377dcd_24.png)

在画布中右键选中“数据源模块”，点击“运行选中模块”。可以看到策略左上角会显示一个旋转的圆圈，表示运行状态。下方的日志中会打印出每个模块运行的时间范围、结果完成与否等信息。

同样的操作，右键单击“基础特征抽取模块”，并选择“运行选中模块”，也可以看到运行过程和日志。当模块全部运行完毕后，会显示“运行完成”，左上角的圆圈停止旋转。

如果我们想查看抽取的数据（即模块运行的结果），有两种方式。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_26.png)

**第一种方式：通过界面可视化查看。**
右键选中一个模块，选择“查看结果”，会弹出一个窗口显示该模块的运行结果。例如，查看数据源模块的结果，会显示抽取的 `open` 和 `close` 两列数据。点击某一列，可以查看该列数据的统计值，包括唯一值数量、缺失值、平均值、中位数、最小值、最大值、标准差等统计信息，以及统计分布图。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_28.png)

**第二种方式：通过代码交互式获取。**
模块的运行结果存放在缓存 `data_source` 中。可以通过 `模块号.data` 方法获取数据存储的缓存地址，再通过 `.read()` 或 `.read_df()` 方法来实现 `DataFrame` 格式数据的读取。

![](img/0e12614f2f616e42bfd00a8ae6377dcd_30.png)

例如，要查看M1（数据源模块）的运算结果，可以在代码单元格中输入：
```python
m1.data.read_df()
```
或
```python
m1.data.read()
```
然后点击运行，即可看到模块返回的数据运行结果，这与前端展示的数据完全一致。如果只想查看数据的前几行，可以加上 `.head()`。

同样的道理，可以查看M6模块（基础特征抽取模块）的运行结果，输入：
```python
m6.data.read_df()
```
即可查看到所抽取的因子库数据。

## 总结 📝

![](img/0e12614f2f616e42bfd00a8ae6377dcd_32.png)

本节课中，我们一起学习了如何使用可视化模块获取金融数据。我们掌握了两种核心方法：一是通过**数据源模块**、证券代码列表和输入特征列表的配合，获取指定的原始行情数据；二是通过**基础特征抽取模块**，从因子库中抽取所需的因子数据。我们还学习了如何运行模块，并通过两种方式（可视化界面和代码交互）查看运行结果。这些是构建更复杂量化策略的数据基础。
