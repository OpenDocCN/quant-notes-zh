在本节课中，我们将继续学习线性回归的核心概念，包括真实值、预测值、误差、损失函数，以及简单线性回归与多元线性回归的区别和表示方法。理解这些概念是掌握线性回归模型的基础。

## 真实值与预测值

上一节我们介绍了线性回归的目标是找到最优解。本节中我们来看看与最优解直接相关的两个核心概念：真实值与预测值。

*   **真实值**：通常用符号 **`y`** 表示，代表我们试图预测的实际目标值。
*   **预测值**：通常用符号 **`ŷ`** （读作“y hat”）表示，代表我们将已知数据 `x` 代入模型公式后计算得到的结果。

我们构建模型的目的，是希望预测值 `ŷ` 能够尽可能地接近真实值 `y`。

## 误差与损失函数

由于模型并非完美，预测值与真实值之间总会存在差距，这个差距就是**误差**。

*   **误差**：即预测值 `ŷ` 与真实值 `y` 之间的差异。
*   **损失函数**：为了衡量模型在整个数据集上的整体误差，我们使用**损失函数**来计算。在线性回归中，最常用的损失函数是基于**最小二乘法**原理构建的。整体误差通常用 **`loss`** 表示。

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_8.png)

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_10.png)

我们的目标就是找到一个模型，使得这个 `loss` 值最小，这意味着模型的预测最准确。

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_12.png)

## 简单线性回归

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_14.png)

理解了基本概念后，我们来看线性回归的具体形式。首先是最简单的情况——简单线性回归。

简单线性回归研究**一个**自变量 `x` 与一个因变量 `y` 之间的关系。其目标是找到一条最能代表数据点趋势的直线。

例如，房屋面积（`x`）与房屋价格（`y`）通常存在正相关关系，但并非严格直线上。简单线性回归就是要找到那条最优的直线（红色线），其方程形式为：

**`ŷ = w * x + b`**

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_16.png)

其中：
*   **`ŷ`** 是预测值。
*   **`w`** 是直线的**斜率**，代表 `x` 对 `y` 的影响程度。
*   **`b`** 是直线的**截距**。
*   **`x`** 是输入的特征值。

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_18.png)

我们的任务就是根据数据求解出最佳的 `w` 和 `b`。

## 多元线性回归

在现实中，影响结果的因素通常不止一个。接下来我们看看更普遍的情况——多元线性回归。

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_20.png)

多元线性回归考虑**多个**自变量 `(x₁, x₂, ..., xₙ)` 对一个因变量 `y` 的影响。例如，银行评估信用额度时，会综合考虑你的收入、城市、还款记录等多个因素。

其方程形式扩展为：

**`ŷ = w₁*x₁ + w₂*x₂ + ... + wₙ*xₙ + b`**

其中：
*   **`w₁, w₂, ..., wₙ`** 是每个特征 `xᵢ` 对应的系数（权重）。
*   **`b`** 仍然是截距。

值得注意的是，截距 `b` 可以看作是特征值恒为1的一个特殊特征的权重 `w₀`。因此，上述公式可以统一改写为：

**`ŷ = w₀*1 + w₁*x₁ + w₂*x₂ + ... + wₙ*xₙ`**

这样，`w₀` 到 `wₙ` 都是需要求解的模型参数。

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_22.png)

![](img/0ee667c1caa1284dc36b8ab1b3f3bf8e_24.png)

## 多元线性回归的向量化表示

为了简化公式，便于计算和推导，我们通常使用向量和矩阵的形式来表示多元线性回归。

将系数 `w₀` 到 `wₙ` 写成一个列向量 **`W`**，将特征 `1`（对应截距）, `x₁` 到 `xₙ` 写成一个列向量 **`X`**，则预测值的计算可以表示为两个向量的点积（即内积）：

**`ŷ = Wᵀ · X`**

其中 **`Wᵀ`** 表示权重向量 `W` 的**转置**（即行向量）。这种表示与上述求和公式完全等价，但更加简洁、紧凑，也是后续进行矩阵运算和梯度下降法推导的基础。
