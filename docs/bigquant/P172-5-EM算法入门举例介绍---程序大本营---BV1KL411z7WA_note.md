## 问题引入：调查身高分布

假设我们需要调查学校中男生和女生的身高分布。我们抽取了100名男生和100名女生，并将他们按性别分成两组。我们统计了这100名男生和100名女生的身高数据。我们知道，身高数据通常服从正态分布。

正态分布（高斯分布）的概率密度函数由两个参数决定：均值（μ）和方差（σ²）。其公式如下：

**公式：**
`p(x | μ, σ²) = (1 / √(2πσ²)) * exp(-(x - μ)² / (2σ²))`

其中，`μ` 代表身高的平均值，`σ²` 代表身高的方差。我们的目标就是根据抽样得到的身高数据，来估计这两个参数（μ 和 σ²）。这两个参数就是我们所说的**隐变量**。

有同学可能会问：既然我们已经有了100个男生的身高数据，直接计算平均值和方差不就可以了吗？确实，这是一种直接的方法。但是，抽样数据可能存在偏差（例如，抽到的样本普遍偏高或偏低）。更重要的是，EM算法为我们提供了一种在更复杂、数据不完整或存在隐变量的情况下，进行参数估计的通用框架。接下来，我们就根据这个思想进行求解。

![](img/f56bddc762e4813f7df02fd72caad254_5.png)

## 将问题数学化

上一节我们引入了具体问题，本节中我们来看看如何将其转化为一个数学问题。

已知条件有两个：
1.  样本服从的分布模型：正态分布。
2.  随机抽样得到的样本数据（即每个人的身高）。

我们要求解的是模型的参数（μ 和 σ²）。根据已知条件，我们可以使用**极大似然估计**来求解未知参数。极大似然估计是一种用于估计模型参数的统计学方法。

现在，我们将问题数学化。假设我们有`n`个样本（例如，`n=100`个男生的身高数据），记为 `x₁, x₂, ..., xₙ`。样本的概率密度函数为 `p(xᵢ; θ)`。这里的 `θ` 代表我们要估计的参数集合，即 `θ = (μ, σ²)`。

这个表示法的含义是：在参数为 `θ` 的条件下，样本 `xᵢ` 出现的概率。对于正态分布，这个概率就是上面给出的高斯公式。

由于这100个样本是独立抽取的，因此同时抽到这组样本的概率等于它们各自概率的乘积（即连乘）。这个概率反映了在参数为 `θ` 时，得到当前这组样本的可能性。我们希望这个可能性越大越好。

**似然函数（Likelihood Function）** 可以表示为：
`L(θ) = ∏ (i=1 to n) p(xᵢ; θ)`

我们需要找到一个参数 `θ`，使得似然函数 `L(θ)` 的值最大。这个使得似然函数最大的 `θ` 值，就称为 `θ` 的**最大似然估计**，记作 `θ̂`。

## 极大似然估计的求解步骤

上一节我们定义了似然函数，本节中我们来看看如何求解最大似然估计。

以下是求解极大似然估计的标准步骤：

![](img/f56bddc762e4813f7df02fd72caad254_7.png)

1.  **写出似然函数**：即所有样本概率的连乘，`L(θ) = ∏ p(xᵢ; θ)`。
2.  **取对数似然函数**：由于连乘运算复杂，我们通常对其取自然对数，将连乘转化为连加。因为对数函数是单调递增函数，所以最大化 `L(θ)` 等价于最大化 `ln L(θ)`。
    **对数似然函数**：`ℓ(θ) = ln L(θ) = Σ ln p(xᵢ; θ)`
3.  **求导并令导数为零**：对对数似然函数 `ℓ(θ)` 关于参数 `θ` 求导，并令导数等于零。
4.  **解方程**：求解上一步得到的方程（或方程组），得到的解 `θ̂` 即为所求的参数估计值。

![](img/f56bddc762e4813f7df02fd72caad254_9.png)

对于我们的身高例子（正态分布），将高斯概率密度公式代入上述步骤，分别对 `μ` 和 `σ²` 求导并令其为零，最终可以解出：
-   `μ` 的最大似然估计就是样本均值：`μ̂ = (1/n) Σ xᵢ`
-   `σ²` 的最大似然估计是样本方差（注意分母是n）：`σ̂² = (1/n) Σ (xᵢ - μ̂)²`

## 理解极大似然估计的思想

我们再来深入理解一下极大似然估计的核心思想。

在多数情况下，我们是根据已知的条件（模型和参数）去推测可能出现的结果（数据）。例如，知道了平均身高和方差，我们可以预测身高的分布范围。

![](img/f56bddc762e4813f7df02fd72caad254_11.png)

而**极大似然估计恰恰相反**：它是已知了结果（观测到的数据），反过来寻找最有可能导致这个结果出现的条件（模型参数）。也就是说，它是一种“由果溯因”的方法。

![](img/f56bddc762e4813f7df02fd72caad254_13.png)

![](img/f56bddc762e4813f7df02fd72caad254_15.png)

我们可以用一个比喻来理解：想象两个人在地球的赤道上，一个人向东走，一个人向西走。虽然方向相反，但他们最终会相遇。同样，传统的“由因推果”和极大似然估计的“由果推因”，目的都是找到数据与模型之间最匹配的那个点（参数）。

在之前介绍极大似然估计的黑球白球例子中，我们计算发现当黑球比例 `p=70%` 时，观测到当前数据的概率最大。反过来，如果我们以“令观测数据的出现概率最大”为目标去求解 `p`，解出来的值也正是 `70%`。这完美诠释了极大似然估计的思想。

1.  我们首先将一个实际问题（估计身高分布的均值和方差）转化为数学问题。
2.  接着，我们引入了**极大似然估计**作为求解模型参数的核心方法，并详细讲解了其数学表示和求解步骤（写似然函数、取对数、求导、解方程）。
3.  最后，我们探讨了极大似然估计“已知结果，寻找最可能条件”的反向思维本质，并通过比喻加深了理解。

这个例子虽然简单，但它揭示了EM算法的基础：在存在隐变量或数据不完全时，通过迭代优化（E步和M步，本例中M步即极大似然估计）来逼近模型的最优参数。在后续更复杂的问题中，EM算法将发挥更大的作用。
