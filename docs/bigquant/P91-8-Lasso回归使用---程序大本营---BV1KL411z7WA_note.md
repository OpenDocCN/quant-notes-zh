## 公式解析与概念回顾

![](img/487041767ad77527d4b222125617a4cd_13.png)

上一节我们介绍了岭回归，本节我们来看看Lasso回归。Lasso回归同样是一种带有正则化项的线性回归模型，其核心目的是防止过拟合，并可能产生稀疏解。

![](img/487041767ad77527d4b222125617a4cd_15.png)

![](img/487041767ad77527d4b222125617a4cd_17.png)

以下是Scikit-learn官网上给出的Lasso回归损失函数公式：

![](img/487041767ad77527d4b222125617a4cd_19.png)

![](img/487041767ad77527d4b222125617a4cd_21.png)

**公式：** `(1 / (2 * n_samples)) * ||Xw - y||^2_2 + alpha * ||w||_1`

![](img/487041767ad77527d4b222125617a4cd_23.png)

![](img/487041767ad77527d4b222125617a4cd_25.png)

![](img/487041767ad77527d4b222125617a4cd_27.png)

在这个公式中，`||.||` 符号表示**范数**。范数是针对向量或矩阵的一种度量方式。

![](img/487041767ad77527d4b222125617a4cd_29.png)

![](img/487041767ad77527d4b222125617a4cd_31.png)

![](img/487041767ad77527d4b222125617a4cd_33.png)

*   **L1范数**：`||w||_1 = |w1| + |w2| + ... + |wn|`。这是Lasso回归使用的正则化项，它促使部分权重系数变为零。
*   **L2范数**：`||w||_2 = sqrt(w1^2 + w2^2 + ... + wn^2)`。这是岭回归使用的正则化项。

![](img/487041767ad77527d4b222125617a4cd_35.png)

公式中 `||Xw - y||^2_2` 部分就是普通线性回归（最小二乘法）的损失函数。`alpha` 是正则化强度系数，控制正则化项的影响大小。

![](img/487041767ad77527d4b222125617a4cd_37.png)

![](img/487041767ad77527d4b222125617a4cd_39.png)

![](img/487041767ad77527d4b222125617a4cd_41.png)

需要注意的是，公式开头的 `1/(2*n_samples)` 是一个常数项，其存在与否不影响优化目标（求最小值）时 `w` 的解。在Scikit-learn中，不同算法的公式表述可能略有不同，这属于正常现象。

以下是使用Lasso回归的核心步骤：

![](img/487041767ad77527d4b222125617a4cd_59.png)

![](img/487041767ad77527d4b222125617a4cd_61.png)

![](img/487041767ad77527d4b222125617a4cd_63.png)

1.  **导入必要的库并创建模拟数据**
2.  **初始化Lasso回归模型并训练**
3.  **观察系数变化，理解稀疏性**
4.  **与随机梯度下降（SGD）回归进行对比**

```python
# 1. 导入库
import numpy as np
from sklearn.linear_model import Lasso, SGDRegressor

# 创建模拟数据：100个样本，20个特征
np.random.seed(42) # 确保结果可复现
X = 2 * np.random.randn(100, 20)
w = np.random.randn(20, 1)
b = np.random.randint(1, 10, size=1)
# 生成目标值 y = Xw + b + 噪声
y = X.dot(w) + b + np.random.randn(100, 1)

# 注意：许多回归模型期望 y 是一维数组，这里将其展平
y = y.ravel()
print(f"X形状: {X.shape}, y形状: {y.shape}")

# 2. 使用Lasso回归
# alpha=1.0 是默认值，对于此数据可能过大，会导致所有系数归零
lasso = Lasso(alpha=0.1) # 尝试调整alpha值，如0.01, 0.5, 1.0
lasso.fit(X, y)

![](img/487041767ad77527d4b222125617a4cd_65.png)

![](img/487041767ad77527d4b222125617a4cd_67.png)

print("Lasso回归系数 (部分可能为0):")
print(lasso.coef_)
print("Lasso回归截距:")
print(lasso.intercept_)

![](img/487041767ad77527d4b222125617a4cd_69.png)

# 3. 使用SGD回归（L1正则化）进行对比
sgd = SGDRegressor(penalty='l1', alpha=0.01, max_iter=1000)
sgd.fit(X, y)

print("\nSGD回归系数 (L1正则):")
print(sgd.coef_)
print("SGD回归截距:")
print(sgd.intercept_)
```

运行上述代码，你会发现：
*   当 `alpha` 值较大（例如1.0）时，Lasso回归求出的所有权重系数 `w` 都可能变为0。
*   当 `alpha` 值适中（例如0.1）时，部分不重要的特征对应的系数会变为0，而重要特征的系数会保留一个较小的非零值。这就是**稀疏性**。
*   与SGD回归（同样使用L1正则化）的结果对比，可以看到Lasso回归的系数绝对值普遍更小，即向零收缩得更明显。

## 核心参数说明

为了更有效地使用Lasso回归，以下是一些关键参数的简要说明：

*   **`alpha`**：**最重要的参数**。正则化强度。必须为正浮点数。值越大，正则化惩罚越重，系数越趋向于零（稀疏性越强）。
*   **`fit_intercept`**：布尔值，默认为True。是否计算模型的截距项。
*   **`max_iter`**：整数，默认为1000。模型求解器的最大迭代次数。
*   **`tol`**：浮点数，默认为1e-4。优化的容忍度（精度）。如果两次迭代间损失函数的下降小于此值，则停止迭代。
*   **`selection`**：`{‘cyclic’, ‘random’}`，默认为‘cyclic’。系数更新循环方式。‘random’通常能使算法更快收敛到更精确的解，尤其当`tol`大于1e-4时。

## 总结

本节课中我们一起学习了Lasso回归的核心原理与应用。

1.  **原理回顾**：Lasso回归在普通线性回归的损失函数中增加了**L1范数**作为正则化项（`alpha * ||w||_1`），其作用是限制模型复杂度，防止过拟合，并倾向于产生稀疏模型（即部分特征系数为零）。
2.  **核心特性**：通过调整 `alpha` 参数，可以控制模型的稀疏程度。`alpha` 越大，越多的特征系数被压缩至零。这一特性使得Lasso回归天然具备**特征选择**的能力，可以用于识别并剔除不重要的特征。
3.  **代码实践**：我们使用 `sklearn.linear_model.Lasso` 类实现了Lasso回归，并通过调整 `alpha` 值直观观察了系数从全零到部分稀疏的变化过程。同时，与 `SGDRegressor`（使用L1惩罚）进行了简单对比。
4.  **应用场景**：Lasso回归特别适用于**特征数量较多，且怀疑其中许多特征不相关或冗余**的场景。它可以帮助我们构建一个更简单、可解释性更强的模型。
