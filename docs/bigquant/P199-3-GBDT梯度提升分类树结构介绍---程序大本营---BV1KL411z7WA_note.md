在本节课中，我们将要学习梯度提升决策树（GBDT）在分类任务中的核心结构和工作原理。我们将从回归树的基础出发，逐步理解如何将其应用于分类问题，并掌握其关键的计算步骤。



## 概述

GBDT是一种强大的集成学习算法，它通过组合多个弱学习器（通常是决策树）来构建一个强学习器。在分类任务中，GBDT的核心思想是使用回归树来拟合概率的“残差”或梯度，从而逐步逼近真实的分类边界。本节课将详细介绍这一过程的结构与实现逻辑。



## 从回归树到分类问题

![](img/d2b31899e28cdfb46ed7bc7ae2191a2b_3.png)

![](img/d2b31899e28cdfb46ed7bc7ae2191a2b_5.png)

上一节我们介绍了GBDT在回归问题中的应用，本节中我们来看看它如何解决分类问题。关键在于，GBDT处理分类时，其基学习器仍然是**回归树**，而不是分类树。

模型的目标是学习一个函数F(x)，将输入特征映射到一个可以用于分类的值。对于二分类问题，我们通常使用逻辑回归的框架，即用sigmoid函数将预测值转换为概率：

**P(y=1 | x) = σ(F(x)) = 1 / (1 + exp(-F(x)))**



## 损失函数与负梯度（伪残差）

在回归问题中，我们使用均方误差（MSE）作为损失函数，并拟合真实值与当前预测值之间的残差。在分类问题中，我们使用对数似然损失（Log Loss）。对于第i个样本，其损失函数为：

**L(y_i, F(x_i)) = -[y_i * log(p_i) + (1-y_i) * log(1-p_i)]**

其中，`p_i = σ(F(x_i))` 是模型预测样本为正类的概率。

为了最小化这个损失，我们计算损失函数关于当前模型预测值F(x)的负梯度。这个负梯度被称为“伪残差”，它是我们下一棵回归树需要拟合的目标。对于第m轮迭代的第i个样本，其伪残差 `r_{im}` 的计算公式为：

**r_{im} = y_i - p_i^{(m-1)}**

这里，`p_i^{(m-1)}` 是前m-1棵树集成后对样本i的预测概率。可以看到，伪残差就是**真实标签（0或1）与当前预测概率之间的差值**。



## 构建一棵新的回归树

获得当前所有样本的伪残差后，我们就有了新的训练目标。以下是构建下一棵回归树的关键步骤：

1.  **确定拟合目标**：将上一步计算出的伪残差 `r_{im}` 作为当前树需要拟合的“标签”。
2.  **构建回归树**：使用决策树算法（如CART），以特征x为输入，以伪残差r为输出目标，构建一棵回归树。这棵树会将输入空间划分为若干个互斥的区域（叶节点）。
3.  **计算叶节点输出值**：对于回归树划分出的每一个叶节点R_jm，我们需要计算一个最佳的输出值γ_jm。这个值不再是简单取该节点内样本伪残差的平均值（像回归问题那样），而是通过最小化损失函数来求解。其近似解公式为：

    **γ_{jm} = Σ_{x_i ∈ R_{jm}} r_{im} / Σ_{x_i ∈ R_{jm}} (p_i^{(m-1)} * (1 - p_i^{(m-1)}))**

    这个公式可以理解为：用叶节点内所有样本的**伪残差之和**，除以这些样本的**预测概率方差之和**。它给出了使损失函数下降最快的方向上的一个步长。



## 模型更新与集成

得到第m棵树及其各个叶节点的输出值γ后，我们就可以更新整个集成模型。更新规则如下：

**F_m(x) = F_{m-1}(x) + ν * Σ_{j} γ_{jm} * I(x ∈ R_{jm})**

这里：
*   `F_{m-1}(x)` 是上一轮的模型预测值。
*   `ν` 是学习率，一个重要的超参数，用于控制每棵树贡献的权重，防止过拟合。
*   `Σ_{j} γ_{jm} * I(x ∈ R_{jm})` 表示：对于输入x，找到它在第m棵树中所属的叶节点R_jm，并将该叶节点的输出值γ_jm作为本棵树的预测值。
*   `I(·)` 是指示函数。



## 完整的算法流程

现在，让我们将上述步骤整合起来，看看GBDT用于二分类的完整流程：

1.  **初始化模型**：初始化一个基础预测值F_0(x)。一个常见的做法是初始化所有样本的预测概率为数据集中正例的比例的对数几率，即 `F_0(x) = log(正例数 / 负例数)`。
2.  **循环迭代（对于 m=1 到 M）**：
    *   **步骤一：计算伪残差**。对每个样本i，计算 `r_{im} = y_i - p_i^{(m-1)}`，其中 `p_i^{(m-1)} = σ(F_{m-1}(x_i))`。
    *   **步骤二：拟合回归树**。使用数据 `{ (x_i, r_{im}) }` 训练一棵回归树，该树被划分为J个叶节点区域 R_{1m}, R_{2m}, ..., R_{Jm}。
    *   **步骤三：计算叶节点值**。对于每个叶节点区域R_{jm}，计算其输出值 `γ_{jm}`。
    *   **步骤四：更新模型**。更新整体模型：`F_m(x) = F_{m-1}(x) + ν * Σ γ_{jm} * I(x ∈ R_{jm})`。
3.  **得到最终模型**：经过M轮迭代后，得到最终的强学习器 `F_M(x)`。
4.  **进行预测**：对于新样本x，计算 `F_M(x)`，然后通过sigmoid函数得到预测概率 `p = σ(F_M(x))`。根据概率是否大于0.5（或自定义阈值）来判断类别。



## 总结

![](img/d2b31899e28cdfb46ed7bc7ae2191a2b_7.png)

本节课中我们一起学习了GBDT处理分类问题的核心结构。关键在于理解以下三点：首先，GBDT分类使用**回归树作为基学习器**；其次，每棵树拟合的目标是损失函数的**负梯度（伪残差）**，即真实标签与当前预测概率的差值；最后，每棵树叶节点的输出值需要通过一个**特定的公式**进行计算，以最大程度地降低损失。通过多轮这样的迭代和模型叠加，GBDT能够逐步修正错误，最终形成一个强大的分类模型。
