## 正规方程的应用

首先，我们启动代码编辑环境，并创建一个名为“多元线性回归”的文件夹用于本次课程。

接下来，我们导入必要的工具包。

```python
import numpy as np
from sklearn.linear_model import LinearRegression
```

然后，我们粘贴上一节课使用的八元一次方程数据。这是一个没有考虑截距的方程。

```python
# 定义特征数据 X 和目标值 y
X = np.array([[382, 764, 1146, 1528, 1910, 2292, 2674, 3056],
              [1, 2, 3, 4, 5, 6, 7, 8],
              [1, 1, 1, 1, 1, 1, 1, 1],
              [1, 2, 3, 4, 5, 6, 7, 8],
              [8, 7, 6, 5, 4, 3, 2, 1],
              [1, 3, 5, 7, 9, 11, 13, 15],
              [1, 2, 4, 8, 16, 32, 64, 128],
              [1, 1, 2, 3, 5, 8, 13, 21]]).T
y = np.array([1, 5, 5, 3, 8, 4, 7, 7])
```

### 求解原始八元一次方程

对于这个八元一次方程，如果它有唯一解，我们至少需要八个独立的方程。我们首先使用线性回归和正规方程来求解它。

使用 `sklearn` 的线性回归模型进行求解，并指定不拟合截距。

```python
model = LinearRegression(fit_intercept=False)
model.fit(X, y)
print("系数 (coef_):", model.coef_)
print("截距 (intercept_):", model.intercept_)
```

使用正规方程手动求解。正规方程的公式为：

**公式：** `θ = (X^T * X)^(-1) * X^T * y`

```python
theta = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)
print("正规方程求解结果:", theta)
```

执行以上代码，两种方法都会得到相同的标准答案 `[1, 5, 5, 3, 8, 4, 7, 7]`。这说明对于这个八元方程组，我们的求解是正确的。

## 为方程添加截距项

现在，我们尝试为这个方程添加一个截距项 `b`，假设其值为 `12`。在多元线性回归中，带截距的方程形式为：

**公式：** `y = w1*x1 + w2*x2 + ... + w8*x8 + b`

由于 `b` 也是一个需要求解的未知变量，我们可以将其重命名为 `w0`，其系数恒为 `1`。因此，方程变为：

**公式：** `y = w0*1 + w1*x1 + w2*x2 + ... + w8*x8`

这意味着我们需要在原始特征数据 `X` 的右侧增加一列，其值全部为 `1`，代表 `w0` 的系数。同时，因为方程整体向上平移了 `12`，目标值 `y` 也需要相应加上 `12`。

以下是操作代码：

```python
# 为 y 添加截距的影响
print("原始的 y:", y)
y_new = y + 12
print("增加截距后的 y:", y_new)

# 为 X 增加一列，值全为 1，代表 w0 的系数
X_new = np.concatenate([X, np.ones((8, 1))], axis=1)
print("增加截距列后的 X 形状:", X_new.shape)
```

此时，`X_new` 的形状变为 `(8, 9)`，代表我们有 **8个方程**，但要求解 **9个未知数** (`w0` 到 `w8`)。

### 尝试求解新方程

我们再次使用正规方程和线性回归模型来求解这个新的方程组。

```python
# 使用正规方程求解
theta_new = np.linalg.inv(X_new.T.dot(X_new)).dot(X_new.T).dot(y_new)
print("添加截距后，正规方程求解结果:", theta_new)

# 使用线性回归模型求解
model_new = LinearRegression(fit_intercept=False)
model_new.fit(X_new, y_new)
print("添加截距后，模型系数:", model_new.coef_)
```

执行代码后，你会发现求解结果与期望值 `[12, 1, 5, 5, 3, 8, 4, 7, 7]` 相去甚远。这是为什么呢？

## 问题分析与解决

问题的核心在于数据形状的改变。`X_new.shape` 的输出 `(8, 9)` 告诉我们：现在我们有 **8个方程**，却要求解 **9个未知数**。

从数学角度讲，这构成了一个“不定方程组”。当方程数量少于未知数数量时，方程组通常没有唯一解，而是有无数个解。算法（如正规方程或线性回归）在这种情况下会计算出一个“最优解”（例如最小二乘解），但这个解并不是我们预设的那个特定解。

用一个更简单的例子类比：方程 `x + y = 3` 是一个二元一次方程，但只有一个方程。它有无数个解（如`x=1,y=2`；`x=0,y=3`等），而没有唯一解。

### 解决方案：增加方程数量

要获得唯一解，我们需要使方程的数量至少等于未知数的数量。既然我们现在有9个未知数，我们就需要至少9个独立的方程。

因此，解决方法就是**为数据集再增加一个样本（即一个方程）**。这样，数据形状就会变成 `(9, 9)`，形成一个九元一次方程组，理论上就可以求解出唯一的系数值（包括我们预设的截距 `12`）。

## 总结

本节课我们一起学习了以下内容：
1.  **回顾了正规方程**在多元线性回归中的应用。
2.  **再现了问题**：当为八元一次方程添加截距项后，使用正规方程求解失败。
3.  **分析了问题根源**：添加截距列后，方程数量（8个）少于未知数数量（9个），导致方程组没有唯一解。
4.  **理解了核心概念**：线性代数中，要求解唯一解，**方程的数量必须大于或等于未知数的数量**。对于有截距的线性回归，确保你的样本数（方程数）至少比特征数多一个。
5.  **提出了解决方案**：通过增加样本数据，使方程数量与未知数数量一致，从而让正规方程能够计算出预期的唯一解。
