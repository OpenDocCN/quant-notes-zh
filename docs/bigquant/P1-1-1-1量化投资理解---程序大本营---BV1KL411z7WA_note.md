# AI量化导论：P1：什么是量化投资

![](img/2eb5c0c79b35a75e2919eb2cc0872348_1.png)

在本节课中，我们将要学习量化投资的基本概念，了解其与传统投资的区别，并认识几位量化投资领域的代表性人物。

![](img/2eb5c0c79b35a75e2919eb2cc0872348_3.png)

## 概述

量化投资是一种利用数学模型、统计分析和计算机技术进行投资决策的方法。它旨在通过系统性的规则来消除人为情绪的影响，实现更稳定、更科学的投资回报。本节将介绍量化投资的定义、关键人物以及它与传统投资的本质区别。

![](img/2eb5c0c79b35a75e2919eb2cc0872348_5.png)

## 量化投资的定义

量化投资是一种采用数量化方法进行定量研究，并以此指导投资交易的方式。随着计算机技术的发展，量化投资得到了迅速的普及和应用。这一定义表明，量化投资是一种区别于传统主观判断的新型投资范式。

![](img/2eb5c0c79b35a75e2919eb2cc0872348_7.png)

## 量化投资的风云人物

以下是量化投资领域几位具有代表性的人物及其贡献。

![](img/2eb5c0c79b35a75e2919eb2cc0872348_9.png)

**詹姆斯·西蒙斯**
詹姆斯·西蒙斯是世界级的数学家，曾与著名数学家陈省身合作进行理论研究。他后来进入金融界，创立了文艺复兴科技公司。这是一家纯粹采用量化策略的对冲基金，其核心团队成员主要由物理学家、数学家、统计学家和计算机科学家构成。

**大卫·肖**
大卫·肖在华尔街被称为“定量之王”。他创立的德劭投资管理公司，目前管理的资产规模已达500亿美元。德劭投资的交易量在高峰期曾占纽约证券交易所总交易量的5%。该公司于2019年在中国完成私募管理人登记，正式进入中国市场。

**伊曼纽尔·德曼**
“宽客”一词广为人知，主要源于物理学家伊曼纽尔·德曼博士。他撰写了畅销书《宽客人生》。如今，“宽客”主要指那些将统计学、数学和计算机知识应用于金融领域的人员。德曼博士曾任职于美国贝尔实验室，后加入高盛集团，并于2000年被评为年度金融工程师。

![](img/2eb5c0c79b35a75e2919eb2cc0872348_11.png)

**瑞·达利欧**
瑞·达利欧创立了全球资产管理规模最大的桥水基金。他本人在2019年《福布斯》全球亿万富豪榜上排名第57位。桥水基金主要采用“纯阿尔法”策略，累计为客户创造了358亿美元的收益。2019年，桥水基金在中国发行了首支私募产品——“桥水全天候中国私募投资基金一号”。

## 量化投资与传统投资的区别

![](img/2eb5c0c79b35a75e2919eb2cc0872348_13.png)

上一节我们介绍了量化投资的代表人物，本节中我们来看看量化投资与传统投资的核心区别。

传统投资主要依赖投资者的个人经验和主观判断进行决策。一个形象的比喻是：传统投资类似于**中医**，通过“望、闻、问、切”来诊断病情。

量化投资则是依据历史数据的统计规律，通过模型产生投资信号。它类似于**西医**，借助仪器检测获得客观数据，再基于数据给出诊断。

两者的关系可以概括为：当传统的投资经验和理念被**数量化的工具**实现时，就形成了量化投资。因此，传统投资并非量化投资的对立面，两者各有适用的场景。量化投资真正的对立面是**主观交易**。量化代表了投资从“艺术”走向“科学”的必然趋势。

![](img/2eb5c0c79b35a75e2919eb2cc0872348_15.png)

以下是两种投资方式代表人物的业绩对比：

*   **传统投资代表**：沃伦·巴菲特。他采用基本面价值投资方法，在1989年至2008年间取得了约**20%** 的平均年化回报率。在2008年金融危机中，其回报率为 **-15%**。
*   **量化投资代表**：詹姆斯·西蒙斯。在相同的1989年至2008年间，其平均年化回报率约为**35%**。在2008年，其回报率约为**80%**，并未受到金融危机的显著影响。

在过去的半个多世纪，众多投资大师中，如巴菲特、索罗斯、彼得·林奇（均为基本面投资代表）和詹姆斯·西蒙斯（量化投资代表），长期业绩统计显示，**詹姆斯·西蒙斯的投资表现最为突出**。这有力地证明了量化投资方法的有效性。

## 总结

![](img/2eb5c0c79b35a75e2919eb2cc0872348_17.png)

本节课中，我们一起学习了量化投资的基本概念。我们了解到，量化投资是一种基于数据、模型和计算机技术的科学投资方法。通过认识詹姆斯·西蒙斯、大卫·肖等代表性人物，我们看到了量化投资在实践中的巨大成功。最后，通过与传统投资的对比，我们明确了量化投资的核心优势在于其**系统性、纪律性和对情绪的规避**，这是投资走向科学化的重要路径。
