## 拉格朗日乘子法：从等式约束出发 ➕

拉格朗日乘子法的主要目的，是将**带有等式约束**的优化问题，转化为**无约束**的优化问题。

![](img/d2556e56f79f612ceef3b578dd661c09_11.png)

支持向量机的约束条件是大于等于某个值，属于**不等式约束**，这比等式约束更复杂。为了理解更复杂的情况，我们需要先从更简单的等式约束说起。

### 等式约束问题的标准形式

一个典型的等式约束优化问题可以写成如下形式：
```
最小化 f(x)
约束条件为 h(x) = 0
```
其中，`f(x)` 是目标函数，`h(x) = 0` 是等式约束。

### 拉格朗日的天才转化

拉格朗日提出，可以引入一个**新的自变量 λ（拉格朗日乘子）**，构造一个**拉格朗日函数 L(x, λ)**，使得原问题严格等价于新问题。

原问题：
```
最小化 f(x), 满足 h(x) = 0
```
等价于新问题（无约束）：
```
最小化 L(x, λ) = f(x) + λ * h(x)
```
这里，`λ` 是可以取任意值的系数。

![](img/d2556e56f79f612ceef3b578dd661c09_13.png)

### 求解方法

转化后，问题变成了对无约束函数 `L(x, λ)` 求极值。我们只需分别对 `x` 和 `λ` 求偏导数，并令其为零：
```
∂L/∂x = 0
∂L/∂λ = 0
```
求解这组方程得到的 `(x, λ)`，其中 `x` 就是原约束问题 `f(x)` 在条件 `h(x)=0` 下的极值点。

![](img/d2556e56f79f612ceef3b578dd661c09_15.png)

## 方法总结与理解 💡

![](img/d2556e56f79f612ceef3b578dd661c09_17.png)

通俗地理解，拉格朗日乘子法就是通过引入一个乘子 `λ`，将限制我们“自由行动”的等式约束 `h(x)=0`，吸收到一个新的无约束函数 `L(x, λ)` 中。

![](img/d2556e56f79f612ceef3b578dd661c09_19.png)

以下是该方法的优势：
*   **去约束化**：将有约束的大难题，转化为无约束的、可以直接求导的标准问题。
*   **求解便利**：通过求解简单的偏导数方程组，就能得到原问题的最优解。
