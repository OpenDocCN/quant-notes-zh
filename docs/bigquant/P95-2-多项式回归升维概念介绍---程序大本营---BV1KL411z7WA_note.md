## 什么是多项式回归？

![](img/e957925b0ad6f99f451a7b6af72a2141_7.png)

上一节我们介绍了基础的线性回归模型。线性回归的特点是，模型中的特征变量（如 `x`）的幂次都是一次幂。

那么什么是多项式呢？多项式意味着方程中可以包含更高次幂的项，例如二次幂（`x²`）、三次幂（`x³`）等。

## 理解维度与升维

为了更好地理解多项式回归，我们先介绍“维度”的概念。

*   **一维**：相当于一条直线，数据只有一个特征。
*   **二维**：相当于一个平面坐标系，数据有两个特征（如 `x` 和 `y`）。
*   **三维**：相当于我们生活的立体空间，数据有三个特征。

多项式回归本质上是线性回归的一种衍生。我们来看一个例子，这是一个一元二次方程：

**y = a * x² + b * x + c**

如果我们想用之前学过的线性回归方法求解这个方程中的系数 `a`, `b`, `c`，会发现很困难。因为之前的线性模型处理的都是“直线”关系，而这里的 `x²` 项引入了“曲线”关系。

如下图所示，蓝色数据点呈现明显的曲线规律，用一条红色的直线（线性模型）去拟合，效果会很差。而使用一个二次多项式（曲线）则可以完美地拟合数据的规律。

![](img/e957925b0ad6f99f451a7b6af72a2141_9.png)

那么，如何让线性回归模型也能处理这种曲线关系呢？答案是：**升维**。

升维，就是把低维度的数据，通过某种方式转换成更高维度的数据。这就像当我们身处低层次时，解决问题的手段有限；而提升到更高维度或层次后，解决问题的思路和方法就会变得更多、更有效。

在机器学习中，数据维度低往往意味着特征信息有限。升维就是增加数据的特征数量，让模型有更多的“信息”去发现数据中隐藏的复杂规律。

## 如何进行升维？

最常见的升维手段，就是对已有的特征进行组合运算，特别是乘法运算。

例如，对于一个原始特征 `x`，我们可以通过“自乘”来创建新的特征：
*   `x` 本身是一维特征。
*   `x²` 就是通过升维得到的二维特征（在多项式意义上）。

这不仅仅局限于平方。以下是常见的升维操作：
*   **平方**：`x²`
*   **立方**：`x³`
*   **不同特征相乘**：`x1 * x2`

## 升维实战：从线性到多项式

假设我们有一个简单的数据集，包含两个原始特征 `x1` 和 `x2`。使用多元线性回归，模型方程如下：

**y_pred = w0 + w1*x1 + w2*x2**

现在，我们使用**二阶多项式**对这个二维数据进行升维。这意味着我们考虑所有特征的一次项、二次项以及特征间的乘积项。

以下是升维后得到的新特征集合：
1.  `x1` （原始特征）
2.  `x2` （原始特征）
3.  `x1²` （`x1` 的平方）
4.  `x2²` （`x2` 的平方）
5.  `x1 * x2` （`x1` 和 `x2` 的交互项）

通过这个操作，我们将特征数量从原来的 2 个扩展到了 5 个。这个过程就是**二阶多项式升维**。现在，我们的线性回归模型方程就变成了：

**y_pred = w0 + w1*x1 + w2*x2 + w3*x1² + w4*x2² + w5*x1*x2**

虽然这个方程看起来更复杂了，但它本质上仍然是一个关于**新特征**（`x1`, `x2`, `x1²`, `x2²`, `x1*x2`）的**线性**模型。线性回归算法可以很好地处理它，并拟合出数据中的非线性关系。
