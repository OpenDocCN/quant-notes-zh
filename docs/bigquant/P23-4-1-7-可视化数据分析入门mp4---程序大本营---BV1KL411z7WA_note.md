# 可视化数据分析入门：P23：4.1.7：截面数据分布分析 📊

在本节课中，我们将学习可视化数据分析的入门知识，重点是如何对金融时间序列中的截面数据进行分布分析。我们将通过绘制各种图表，直观地观察和理解数据在不同维度上的分布特征。

金融数据通常以时间序列的方式进行存储。为了观察某一个时间截面上的数据，我们可以对其截面数据进行抽取，并分析截面数据的分布。这里我们以截面财务数据分析为例，简要介绍截面数据分布经常使用的模块功能。

![](img/cad1ac9f4fa4572264206ba534eaf117_1.png)

上一节我们介绍了分析截面数据分布的目的，本节中我们来看看具体的操作步骤。

以下是进行截面数据分布分析的主要流程：

1.  **数据抽取**：首先通过证券代码列表和输入特征列表，以及数据源模块，从财务数据表中抽取某一日的截面数据。
2.  **单变量分析**：然后我们会对某一个字段进行直方图的分布绘制和观察，以及概率密度曲线的绘制观察。
3.  **双变量分析**：接着我们针对双变量进行联合分布绘制和观察。
4.  **多变量分析**：对于多变量，我们可以进行两两双变量的联合分布组图绘制。
5.  **分组分析**：最后我们可以对数据进行分组分布图绘制。我们可以指定所需分组的字段，对数据进行分组展示，并分组绘制箱型图、小提琴图。

---

### 1. 数据抽取

分析的第一步是获取目标数据。我们需要从庞大的时间序列数据库中，提取出特定日期（时间截面）下所有证券的财务数据。这通常涉及以下核心元素：

*   **证券代码列表**：指定需要分析哪些股票或债券。
*   **输入特征列表**：指定需要分析哪些财务指标，例如市盈率（PE）、市净率（PB）、净资产收益率（ROE）等。
*   **数据源模块**：连接数据库或数据API，执行查询。

这个过程可以用一个简单的伪代码表示：
```python
# 伪代码示例
截面数据 = 数据源.查询(日期=‘2023-12-31’, 证券列表=[‘000001’, ‘600519’], 特征列表=[‘PE’, ‘PB’, ‘ROE’])
```

### 2. 单变量分布分析

获得数据后，我们首先可以观察单个财务指标在所有证券中的分布情况。最常用的工具是直方图和概率密度曲线。

*   **直方图**：将数据范围划分为若干个连续的区间（称为“箱子”），并统计落入每个区间的数据点数量。它能直观展示数据的集中趋势和离散程度。
*   **概率密度曲线**：是直方图的一种平滑估计，可以更清晰地展示数据分布的总体形状（如是否对称、是否存在多个峰值）。

通过观察单变量分布，我们可以快速了解该指标的市场整体水平、是否存在极端值（异常值）等。

### 3. 双变量联合分布分析

在金融分析中，研究两个变量之间的关系至关重要，例如研究市盈率（PE）与净资产收益率（ROE）是否存在关联。这时可以使用散点图或二维核密度图。

*   **散点图**：将两个变量的值作为坐标点绘制在二维平面上，可以直观看到变量间的相关性和分布模式。
*   **二维核密度图**：在散点图的基础上，用颜色深浅表示数据点的密集程度，能更清晰地揭示两个变量的联合分布热点区域。

### 4. 多变量关系组图

当我们需要同时考察多个变量（如PE、PB、ROE、营收增长率）两两之间的关系时，可以绘制**散点图矩阵**。它是一个网格状图表，对角线通常是每个变量的单变量分布（直方图或密度图），而非对角线位置则是每对变量之间的散点图。这能让我们一次性浏览所有变量对的潜在关系。

### 5. 分组分布分析

有时我们需要比较不同组别间数据的分布差异，例如比较不同行业上市公司的市盈率分布。这时可以进行分组可视化。

![](img/cad1ac9f4fa4572264206ba534eaf117_3.png)

*   **分组箱型图**：箱型图可以展示数据的中位数、四分位数以及异常值。分组绘制可以清晰对比不同组别的集中趋势和离散程度。
*   **分组小提琴图**：结合了箱型图和核密度图的特点，既能展示统计摘要，又能展示任意位置的分布密度，是更强大的分组分布比较工具。
