# 机器学习基础：P51-2：为八元一次方程增加截距的解决方案 🧮

在本节课中，我们将学习如何为一个八元一次方程组增加截距项，并理解这如何影响线性回归和正规方程的求解过程。我们将通过具体的代码示例，演示如何通过增加方程来获得唯一解，并解释截距在模型中的作用。

---

## 标准答案与系数定义

![](img/26524adf52802b03636a37bec0b29f39_1.png)

上一节我们介绍了八元一次方程的基本概念。本节中，我们来看看如何定义标准答案。

以下是标准答案 `w` 的定义：
```python
w = np.array([15, 384, 17, 12])
```
这个数组 `w` 代表了我们八元一次方程中各个未知数的系数标准答案。

---

## 增加第九个方程

有了标准答案，我们可以构造新的方程来扩展方程组。

以下是创建第九个随机方程 `x9` 的步骤：
```python
x9 = np.random.randint(-15, 15, size=8)
```
这行代码生成了一个包含8个随机整数（范围在-15到15之间）的数组，作为新方程的系数。

---

![](img/26524adf52802b03636a37bec0b29f39_3.png)

## 计算新方程的目标值

现在，我们利用新方程的系数和标准答案来计算其对应的目标值 `y`。

![](img/26524adf52802b03636a37bec0b29f39_5.png)

以下是计算目标值并加上截距的公式：
```python
y_value = x9.dot(w) + 12
```
计算结果是 `219`。因为原始八个方程的目标值都包含了截距 `12`，所以第九个方程也必须加上相同的截距，以保持方程组的一致性。

![](img/26524adf52802b03636a37bec0b29f39_7.png)

---

![](img/26524adf52802b03636a37bec0b29f39_9.png)

## 扩展目标值向量 `y`

![](img/26524adf52802b03636a37bec0b29f39_11.png)

我们需要将新计算出的目标值整合到原有的目标值向量中。

以下是扩展 `y` 向量的代码：
```python
y = np.concatenate([y, [219]])
```
执行后，`y` 向量的最后一个值变为 `219`。

---

## 扩展特征矩阵 `x`

![](img/26524adf52802b03636a37bec0b29f39_13.png)

接下来，我们需要将第九个方程也加入到特征矩阵 `x` 中，并为其添加代表截距的列。

![](img/26524adf52802b03636a37bec0b29f39_15.png)

![](img/26524adf52802b03636a37bec0b29f39_17.png)

以下是处理并扩展 `x` 矩阵的步骤：
1.  首先，为 `x9` 添加一个代表截距的常数项 `1`。
    ```python
    x9_with_intercept = np.concatenate([x9, [1]])
    ```
2.  然后，将这个一维数组转换为二维行向量，并拼接到原始矩阵 `x` 的末尾。
    ```python
    x_new = np.concatenate([x, [x9_with_intercept]], axis=0)
    ```
    执行后，`x_new` 矩阵变为9行9列，最后一行是 `x9` 的系数加上截距项 `1`。

---

![](img/26524adf52802b03636a37bec0b29f39_19.png)

## 使用正规方程求解

现在，我们拥有了扩展后的特征矩阵 `x_new` 和目标向量 `y`，可以应用正规方程来求解系数。

以下是正规方程的求解公式和代码：
```python
coefficients = np.linalg.inv(x_new.T.dot(x_new)).dot(x_new.T).dot(y)
```
求解出的前八个系数与最初的标准答案 `w` 完全一致，最后一个系数是截距 `12`。这证明通过增加一个有效方程，我们成功为扩展后的九元一次方程组找到了唯一解。

## 使用线性回归验证

![](img/26524adf52802b03636a37bec0b29f39_27.png)

![](img/26524adf52802b03636a37bec0b29f39_29.png)

正规方程有效，线性回归模型也应该得到相同的结果。

以下是使用线性回归模型进行拟合和验证的代码：
```python
from sklearn.linear_model import LinearRegression
model = LinearRegression(fit_intercept=False)
model.fit(x_new, y)
print(model.coef_)
```
模型求解出的系数与正规方程的结果一模一样，验证了两种方法在数学上的统一性。

![](img/26524adf52802b03636a37bec0b29f39_31.png)

## 理解 `fit_intercept` 参数

![](img/26524adf52802b03636a37bec0b29f39_33.png)

![](img/26524adf52802b03636a37bec0b29f39_35.png)

![](img/26524adf52802b03636a37bec0b29f39_37.png)

`fit_intercept` 参数控制模型是否单独计算截距。理解其影响至关重要。

![](img/26524adf52802b03636a37bec0b29f39_39.png)

![](img/26524adf52802b03636a37bec0b29f39_41.png)

以下是设置 `fit_intercept=True` 时的行为分析：
当设置 `fit_intercept=True` 时，线性回归模型会单独计算截距项 `model.intercept_`。此时，从 `model.coef_` 中得到的最后一个系数会变为 `0`，因为截距的作用已被分离出来。计算出的 `model.intercept_` 值约等于 `12`。

![](img/26524adf52802b03636a37bec0b29f39_43.png)

如果我们手动从特征矩阵中移除代表截距的最后一列（即全为1的列），再设置 `fit_intercept=True`，模型会计算出前八个系数和一个独立的截距（约等于12），这与我们的预期完全相符。

## 核心问题回顾与总结

![](img/26524adf52802b03636a37bec0b29f39_45.png)

![](img/26524adf52802b03636a37bec0b29f39_47.png)

本节课中我们一起学习了为线性方程组增加截距项的全过程。

![](img/26524adf52802b03636a37bec0b29f39_49.png)

最初，八元一次方程有唯一解。为其增加一个截距项 `b` 后，问题变为求解九个参数（八个系数 `w` 加一个截距 `b`），但方程数量仍为八个，导致方程数量少于未知数数量，无法求得唯一确定解。这就是最初使用正规方程或线性回归得不到准确系数的根本原因。

![](img/26524adf52802b03636a37bec0b29f39_51.png)

解决方案是**增加一个有效的方程**。我们利用已知的标准答案 `w`，随机生成第九个方程的系数 `x9`，计算出其对应的目标值 `y9`（需加上截距12），并将其与截距列一起添加到原方程组中。这样，我们就得到了一个拥有九个方程、九个未知数的完整方程组，从而可以求解出唯一的标准答案。

![](img/26524adf52802b03636a37bec0b29f39_53.png)

通过这个过程，我们深入理解了：
1.  **方程数量与未知数数量的关系**是线性方程组有唯一解的关键。
2.  **截距的本质**是一个需要被求解的模型参数。
3.  **正规方程与线性回归**在原理上是相通的，`fit_intercept` 参数决定了截距的处理方式。
4.  通过构造额外的数据，可以解决因信息不足（方程数不够）导致的无法求解问题。

![](img/26524adf52802b03636a37bec0b29f39_55.png)
