## 什么是逻辑回归？

上一节我们介绍了机器学习的基本任务类型。本节中我们来看看逻辑回归的具体定义。

逻辑回归虽然名字中带有“回归”二字，但它本质上是一个**分类算法**。这类似于“卡巴斯基不是司机”、“红烧狮子头没有狮子头”，我们不能仅凭名称来判断其功能。

逻辑回归算法隶属于 `sklearn.linear_model` 模块，这表明它是一种**线性模型**。其核心思想是基于线性回归的结果，通过一个特殊的函数将其映射到概率值，从而完成分类任务。

## 分类与回归的区别

在深入逻辑回归之前，我们需要明确分类任务与回归任务的根本区别。

以下是两者的核心对比：

*   **回归**：预测一个连续的数值。例如，根据房屋面积、地段预测房价，或根据历史数据预测未来趋势。其输出是任意实数。
*   **分类**：预测样本所属的类别或标签。例如，根据邮件内容判断是“垃圾邮件”还是“正常邮件”，根据花朵特征判断其品种。其输出是离散的类别。

逻辑回归解决的是典型的**分类问题**。

## 逻辑回归的核心：Sigmoid函数

逻辑回归之所以能将线性输出转化为类别概率，依赖于一个关键的数学函数——Sigmoid函数。

Sigmoid函数，也称为逻辑函数，其数学表达式为：

**公式：** `σ(z) = 1 / (1 + e^(-z))`

其中，`e` 是自然常数，`z` 是线性模型的输出（例如 `z = w1*x1 + w2*x2 + ... + b`）。

这个函数具有以下重要特性：
1.  其输出值被严格限制在 **(0, 1)** 区间内，可以完美地解释为概率。
2.  函数图像是一条平滑的 **S型曲线**。
3.  其导数可以用自身表示：`σ'(z) = σ(z) * (1 - σ(z))`，这个性质在模型训练（反向传播）中非常有用。

让我们通过代码来直观感受Sigmoid函数的形状。

```python
import numpy as np
import matplotlib.pyplot as plt

# 定义Sigmoid函数
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

# 生成数据点
x = np.linspace(-5, 5, 100)
y = sigmoid(x)

# 绘制图像
plt.plot(x, y, color='green')
plt.title('Sigmoid Function')
plt.xlabel('x')
plt.ylabel('sigmoid(x)')
plt.grid(True)
plt.show()
```

执行上述代码，你将看到一条从0平滑过渡到1的S型曲线。当线性输出 `z` 为很大的正数时，Sigmoid输出接近1；当 `z` 为很大的负数时，输出接近0；当 `z=0` 时，输出恰好为0.5。

## 如何进行分类决策？

理解了Sigmoid函数后，逻辑回归的分类过程就清晰了。

模型首先计算线性组合 `z`，然后通过Sigmoid函数得到概率值 `p = σ(z)`。这个 `p` 值表示样本属于“正类”（通常标记为1）的概率。

以下是分类的决策规则：

*   如果 `p >= 0.5`，模型预测该样本属于**正类 (1)**。
*   如果 `p < 0.5`，模型预测该样本属于**负类 (0)**。

这里的0.5被称为**决策边界**。本质上，当线性部分 `z >= 0` 时，`p >= 0.5`。因此，逻辑回归是在学习一个线性边界（一条直线或一个平面）来区分不同类别的数据。

## 为什么在工业界备受青睐？

逻辑回归模型结构简单、计算效率高，且输出具有概率意义。这使得它具备许多优点：

1.  **可解释性强**：模型的权重系数可以直接反映特征对结果的影响程度。
2.  **计算成本低**：训练和预测的速度都很快，适合处理大规模数据。
3.  **概率输出**：不仅能给出分类结果，还能给出属于各类别的置信度（概率），这在风控、医疗等领域至关重要。
4.  **不易过拟合**：尤其是当配合适当的正则化手段时。

正因为这些优点，逻辑回归常被称为“**一招逻辑斯蒂打天下**”，是许多机器学习应用的首选基线模型。

## 总结

本节课中我们一起学习了逻辑回归的基础知识。我们明确了逻辑回归是一个**线性分类模型**，其核心在于使用 **Sigmoid函数** 将线性回归的输出映射到(0,1)区间，解释为概率，并通过设定阈值（如0.5）来完成分类决策。它因其简单、高效和良好的可解释性，成为工业界解决分类问题的利器。
