# 机器学习：P171：极大似然思想

在本节课中，我们将要学习数理统计中的一个核心概念——**极大似然估计**。我们将通过生动的例子和严谨的推导，理解其基本思想，并了解它在EM算法等机器学习方法中的重要作用。

## 什么是极大似然估计？🤔

上一节我们介绍了EM算法的基本框架，本节中我们来看看支撑其第一步（E步）的关键思想：极大似然。

极大似然，也称为最大似然，其核心是寻找使“似然函数”取得最大值的参数。这里的“似然”可以理解为模型的可能性或概率。

为了理解这个概念，我们可以看两个生活中的例子。

以下是两个类比示例：
*   **猎人打猎**：一位同学和一位猎人一起打猎。一声枪响后，野兔应声倒下。由于猎人的命中率通常远高于同学，我们更倾向于推断这一枪是猎人射中的。这种基于“谁更有可能命中”的推断，就体现了极大似然的思想——我们选择那个使“观察到野兔被击中”这一事件**发生概率更大**的假设。
*   **世界杯比赛**：假设中国队与德国队进行世界杯决赛。根据两队当前的水平，德国队获胜的概率显然更大。博彩公司开出的赔率（例如赌德国赢赔率低，赌中国赢赔率高）也正是基于这种概率估计。我们预测德国队赢，就是选择了**可能性更大**的结果。

如果理解了上述例子，你就已经掌握了极大似然的核心直觉。

## 罐中取球：一个经典案例 🎱

现在，我们通过一个更具体的数学模型来深入理解。这个案例在之前的线性回归课程中曾介绍过，今天我们将结合极大似然思想重新回顾。

假设有一个罐子，里面装有黑白两种颜色的球。我们不知道球的总数，也不知道黑白球的比例。我们希望通过实验来估计白球的比例。实验方法是：每次从摇匀的罐子中随机取出一个球，记录颜色后**放回**，重复这个过程。

假设我们重复了100次实验，记录到70次是白球，30次是黑球。

**请问：罐子中白球的比例最有可能是多少？**

很多人的直觉答案是70%。这个感觉是正确的。下面，我们来看这个“感觉”背后的数学原理。

## 数学推导：为什么是70%？🧮

![](img/c1ba85e05b52ed89e904a5eaee125e44_5.png)

上一节我们通过直觉得到了答案，本节中我们来看看严谨的数学推导过程。

![](img/c1ba85e05b52ed89e904a5eaee125e44_7.png)

![](img/c1ba85e05b52ed89e904a5eaee125e44_9.png)

我们设罐子中白球的比例为 **p**，那么黑球的比例就是 **1-p**。
*   取一次球是白球的概率：**p**
*   取两次球都是白球的概率：**p²**
*   取五次球都是白球的概率：**p⁵**

那么，在我们“100次实验，70次白球，30次黑球”的特定观察结果下，这种情况发生的概率是多少呢？这需要用到排列组合和概率乘法。

![](img/c1ba85e05b52ed89e904a5eaee125e44_11.png)

这种情况发生的概率（即似然函数 **L(p)** ）可以表示为：
`L(p) = C(100, 70) * p^70 * (1-p)^30`
其中，`C(100, 70)` 是从100次试验中选出70次为白球的组合数，是一个常数。

**极大似然估计要解决的问题就是：当 p 等于多少时，这个概率 L(p) 最大？** 也就是说，白球比例 p 为多少时，最有可能产生我们观察到的数据（70白，30黑）。

我们通过求导来找到这个最大值点。为了计算方便，通常对似然函数取对数（称为对数似然函数），因为对数函数是单调的，最大化对数似然函数等价于最大化原似然函数。

1.  对数似然函数：`l(p) = ln L(p) = ln[C(100, 70)] + 70*ln(p) + 30*ln(1-p)`
2.  对 p 求导并令导数为零：
    `dl/dp = 70/p - 30/(1-p) = 0`
3.  解方程：
    `70/p = 30/(1-p)`
    `70(1-p) = 30p`
    `70 - 70p = 30p`
    `70 = 100p`
    `p = 0.7`

推导结果 **p = 0.7**，与我们的直觉完全吻合。这个严谨的数学过程揭示了我们大脑“快速直觉”背后的运算逻辑。我们的大脑如同一个高效的计算机，瞬间完成了类似的概率评估，给出了最可能的答案。

## 与EM算法的联系 🔄

现在，我们已经回顾了极大似然估计。这正是**EM算法**中“M步”（最大化步）的核心。

EM算法的流程可以概括为：
1.  **E步（期望步）**：基于当前参数，计算隐变量的期望。
2.  **M步（最大化步）**：将E步得到的期望视为已知数据，进行**极大似然估计**，更新模型参数。

然后，算法回到第一步，用新的参数重新计算期望。如此**循环迭代**，E步和M步交替进行。每次循环后，参数估计都会更接近真实值。当参数不再发生显著变化时，我们就得到了最终的优化结果。

**本节课中我们一起学习了：**
1.  **极大似然估计**的基本思想：在给定观测数据的情况下，选择那个使数据出现**概率最大**的模型参数。
2.  通过“罐中取球”的经典案例，从直觉和数学推导两个层面理解了极大似然估计的应用，并得出了关键公式 `p = 0.7`。
3.  明确了极大似然估计是**EM算法**中M步的理论基础，它驱动参数不断迭代优化，直至收敛。
