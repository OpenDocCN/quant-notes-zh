# 机器学习基础：P54：最小二乘法公式推导 📈

在本节课中，我们将学习如何从正态分布和最大似然估计出发，推导出机器学习中至关重要的最小二乘法公式。我们将一步步解析，确保初学者也能理解其背后的数学原理。

## 正态分布与概率密度函数

上一节我们介绍了最大似然估计的概念，本节中我们来看看如何将其应用于最常见的连续分布——正态分布（也称高斯分布）。正态分布的概率密度函数是我们推导的起点。

正态分布的概率密度函数公式如下：

**公式：**
```
p(x) = (1 / (σ * √(2π))) * e^(-(x - μ)² / (2σ²))
```

![](img/c986fb5acd8f997cdc52c59b503c1c7a_1.png)

在这个公式中：
*   **μ** 表示平均值。
*   **σ** 表示标准差。
*   **π** 和 **e** 是两个重要的数学常数。

![](img/c986fb5acd8f997cdc52c59b503c1c7a_3.png)

这个函数描述了在给定平均值和标准差的情况下，一个数据点出现的概率密度。在最大似然估计的语境下，这个 `p(x)` 就相当于之前例子中“从罐子中取出特定颜色球”的概率。

![](img/c986fb5acd8f997cdc52c59b503c1c7a_5.png)

## 从最大似然到误差分析

理解了概率密度函数后，我们需要将其与我们的预测问题联系起来。在回归问题中，我们关注的是预测值与真实值之间的误差。

我们定义误差 **ε** 为真实值 `y_i` 与预测值 `ŷ_i` 之间的差值。在线性回归中，预测值通常由模型参数 `w` 和输入特征 `x_i` 决定，即 `ŷ_i = w^T * x_i`。

因此，误差可以表示为：
**公式：**
```
ε_i = y_i - w^T * x_i
```

我们假设这些误差服从均值为0的正态分布。这意味着对于每一个数据点 `(x_i, y_i)`，其出现的概率（或更准确地说，概率密度）可以由上面的正态分布公式描述，其中 `x` 替换为我们的误差 `ε_i`。

## 似然函数的构建与对数变换

![](img/c986fb5acd8f997cdc52c59b503c1c7a_7.png)

![](img/c986fb5acd8f997cdc52c59b503c1c7a_9.png)

为了找到最有可能产生我们观测数据的模型参数 `w`，我们需要构建似然函数。

以下是构建和转换似然函数的关键步骤：

1.  **构建似然函数**：假设所有数据点的误差是独立同分布的，那么整个数据集的联合概率密度（似然函数 `L`）就是每个数据点概率密度的乘积（累乘）。
    **公式：**
    ```
    L(w) = ∏ p(ε_i)
          = ∏ (1 / (σ * √(2π))) * e^(-(y_i - w^T * x_i)² / (2σ²))
    ```

2.  **引入对数似然**：直接对累乘的似然函数 `L(w)` 求最大值非常困难。一个关键的技巧是取自然对数（以 `e` 为底），将累乘问题转化为累加问题。因为对数函数是单调递增函数，所以最大化 `L(w)` 等价于最大化 `ln(L(w))`。
    **公式：**
    ```
    ln L(w) = ∑ ln( p(ε_i) )
    ```

3.  **展开并简化**：将对数似然函数具体展开，并利用对数的运算法则进行简化。常数项（如 `1/(σ√(2π))`）和对数运算会使公式中的指数项 `e^...` 简化。
    **公式推导简化后核心项为：**
    ```
    ln L(w) = C - (1/(2σ²)) * ∑ (y_i - w^T * x_i)²
    ```
    其中 `C` 是与参数 `w` 无关的常数项。

## 推导最小二乘目标函数

经过对数变换和简化后，我们得到了一个清晰的表达式。

在简化后的对数似然函数 `ln L(w)` 中，第一项 `C` 是常数。因此，最大化 `ln L(w)` 就等价于**最小化**后面的求和项 `∑ (y_i - w^T * x_i)²`。

由此，我们得到了最小二乘法的目标函数（损失函数）：
**公式：**
```
J(w) = (1/2) * ∑ (y_i - w^T * x_i)²
```
（公式中的 `1/2` 系数是为了后续求导方便而添加的，不影响优化结果。）

这个公式的意义非常直观：它衡量了模型在所有数据点上预测误差的平方和。我们的目标就是找到一组参数 `w`，使得这个平方和误差 `J(w)` 最小。这也印证了“最小二乘法”名称的由来——最小化误差的平方（二乘）和。

## 总结与意义

![](img/c986fb5acd8f997cdc52c59b503c1c7a_11.png)

本节课中我们一起学习了最小二乘法公式的完整推导过程。

![](img/c986fb5acd8f997cdc52c59b503c1c7a_13.png)

![](img/c986fb5acd8f997cdc52c59b503c1c7a_15.png)

我们从**正态分布**的概率假设出发，通过**最大似然估计**的原理构建了似然函数。为了便于求解，我们使用了**对数变换**这一关键技巧，将复杂的累乘问题转化为累加问题。最终，我们发现最大化对数似然函数等价于最小化预测误差的平方和，从而自然而然地推导出了**最小二乘法**的目标函数 `J(w)`。

![](img/c986fb5acd8f997cdc52c59b503c1c7a_17.png)

![](img/c986fb5acd8f997cdc52c59b503c1c7a_19.png)

这个推导过程揭示了最小二乘法不仅是一个直观的“让误差平方和最小”的准则，它背后有着坚实的概率统计基础（即假设误差服从正态分布）。高斯正是利用这一方法，成功计算出了谷神星的轨道，展现了该公式的强大力量。在机器学习中，它成为了线性回归等众多模型的基石。
