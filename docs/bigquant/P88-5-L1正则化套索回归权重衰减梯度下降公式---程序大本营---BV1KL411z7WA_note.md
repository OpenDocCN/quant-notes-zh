## 梯度下降更新规则回顾 🔄

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_9.png)

上一节我们介绍了L1正则化的图形化理解，本节中我们来看看其权重更新的数学推导。

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_11.png)

梯度下降的核心是沿着损失函数的负梯度方向更新参数，其基本更新规则为：

**`w_new = w_old - η * ∇J(w)`**

其中：
*   **`w`** 代表模型权重。
*   **`η`** 代表学习率（步长），是一个大于0的常数。
*   **`∇J(w)`** 代表损失函数 `J(w)` 关于权重 `w` 的梯度。

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_13.png)

## L1正则化损失函数分解 📊

套索回归（Lasso Regression）的损失函数 `J(w)` 由两部分组成：原始损失（如均方误差）和L1正则项。

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_15.png)

**`J(w) = J₀(w) + α * Σ|w_i|`**

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_17.png)

其中：
*   **`J₀(w)`** 是原始损失函数（例如线性回归的均方误差）。
*   **`α`** 是L1正则化系数，控制正则化的强度，`α > 0`。
*   **`Σ|w_i|`** 是所有权重绝对值的和，即L1正则项。

我们的目标是对这个新的损失函数 `J(w)` 求梯度 `∇J(w)`。

## L1正则项的梯度求解 🧮

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_19.png)

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_21.png)

对 `J(w)` 求梯度，关键在于求解L1正则项 `α * Σ|w_i|` 的导数。由于绝对值函数在零点不可导，我们需要分段讨论。

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_23.png)

L1正则项对某个特定权重 `w_i` 的偏导数可以表示为：

**`∂(α * |w_i|) / ∂w_i = α * sgn(w_i)`**

这里的 **`sgn(w_i)`** 是符号函数，其定义如下：

*   当 `w_i > 0` 时，`sgn(w_i) = 1`
*   当 `w_i < 0` 时，`sgn(w_i) = -1`
*   当 `w_i = 0` 时，函数不可导，在实际计算中通常作为特例处理（例如直接置零或跳过更新）。

**推导理解**：
*   当 `w_i > 0` 时，`|w_i| = w_i`，对其求导结果为 `1`。
*   当 `w_i < 0` 时，`|w_i| = -w_i`，对其求导结果为 `-1`。

## L1正则化下的权重更新公式 ✍️

综合原始损失梯度 `∇J₀(w)` 和L1正则项的梯度，我们得到完整的权重更新公式。

对于第 `i` 个权重 `w_i`，其更新规则为：

**`w_i_new = w_i_old - η * [ ∂J₀/∂w_i + α * sgn(w_i_old) ]`**

这个公式清晰地展示了L1正则化如何影响权重更新。

## 公式行为分析：权重衰减与特征选择 🔍

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_25.png)

现在，我们来分析这个更新公式如何使权重“衰减”并可能变为零。

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_27.png)

以下是两种情况的分解：

1.  **当 `w_i_old > 0` 时**：
    *   `sgn(w_i_old) = 1`
    *   更新公式变为：`w_i_new = w_i_old - η * ∂J₀/∂w_i - η * α`
    *   由于 `η * α > 0`，相当于在原始更新基础上额外减去一个正数。因此，新的 `w_i` 会**变小**，向零靠近。

2.  **当 `w_i_old < 0` 时**：
    *   `sgn(w_i_old) = -1`
    *   更新公式变为：`w_i_new = w_i_old - η * ∂J₀/∂w_i + η * α`
    *   由于 `η * α > 0`，相当于在原始更新基础上额外**加上**一个正数。对于一个负数，加上一个正数，其值会**变大**（例如从-0.5变为-0.2）。这意味着它的**绝对值在变小**，同样是在向零靠近。

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_29.png)

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_31.png)

**核心结论**：
无论权重初始值为正还是为负，增加L1正则化后，其更新过程总会促使权重的**绝对值不断减小**，即发生“权重衰减”。当某个权重的绝对值被衰减到非常小甚至直接更新为零时，对应的特征就被模型忽略，这实现了**特征选择**的功能。这与上一节中图形化解释——最优解常出现在坐标轴（某些维度为零）上——是完全一致的。

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_33.png)

## 不同符号表示法 📝

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_35.png)

在一些文献中，正则化系数可能用 `λ`（lambda）表示，这与我们用 `α` 表示本质相同。公式形式完全一致：

![](img/ac0d69180eef5dcdcd8b3d32a764cf91_37.png)

**`w_i_new = w_i_old - η * [ ∂J₀/∂w_i + λ * sgn(w_i_old) ]`**
