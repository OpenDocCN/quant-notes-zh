## 本节作业任务

![](img/bcca482a01f99c576c1e01aed4002387_9.png)

本节我们学习了支持向量机（SVM）这一强大的分类算法。本次作业的核心任务是：**使用SVM模型完成同样的手写数字分类任务，并与逻辑回归模型的性能进行对比**。

![](img/bcca482a01f99c576c1e01aed4002387_11.png)

为了获得更可靠的对比结果，我们不能仅凭一次随机划分的数据来评判模型优劣。因此，本次作业要求进行多次实验。

![](img/bcca482a01f99c576c1e01aed4002387_13.png)

以下是具体的作业步骤：
1.  **数据准备**：同样从MNIST数据集中抽取5000个样本。
2.  **模型训练与评估**：使用SVM模型（例如，`sklearn.svm.SVC`）进行分类。
3.  **多次实验**：将数据随机拆分为训练集和测试集（比例仍为0.1），重复此过程100次。
4.  **计算平均准确率**：记录每次实验的测试准确率，最后计算100次实验的平均准确率。
5.  **性能对比**：将SVM的100次平均准确率与逻辑回归的100次平均准确率进行比较，分析哪个模型在此任务上表现更优。

![](img/bcca482a01f99c576c1e01aed4002387_15.png)

通过这种方式，我们可以更稳健地评估支持向量机和逻辑回归在处理手写数字分类问题上的性能差异。

![](img/bcca482a01f99c576c1e01aed4002387_17.png)
