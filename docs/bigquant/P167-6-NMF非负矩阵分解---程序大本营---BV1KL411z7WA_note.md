在本节课中，我们将学习一种特殊的降维算法——非负矩阵分解。与PCA和LDA不同，NMF要求所有数据均为非负数，这使得它在处理具有实际物理意义的数据（如身高、体重、像素强度）时，分解结果更易于解释。

## 核心思想 📖

非负矩阵分解的基本思想很简单。对于任意给定的一个**非负矩阵 V**，算法旨在找到两个**非负矩阵 W 和 H**，使得它们的乘积近似等于原矩阵。

![](img/d729489211926cf67d328807e4bb3ee9_6.png)

上一节我们介绍了矩阵分解的通用概念，本节中我们来看看NMF的具体形式。其核心公式如下：

**V ≈ W · H**

其中：
*   **V** 是原始的非负数据矩阵（形状为 `m × n`）。
*   **W** 称为基矩阵（形状为 `m × k`），可以理解为新的特征。
*   **H** 称为系数矩阵（形状为 `k × n`），可以理解为样本在新特征上的投影。

这里的 `k` 通常小于 `n`，从而达到降维的目的。关键在于，**W** 和 **H** 中的所有元素都必须是非负的，这保证了分解后的成分具有可加性和明确的物理意义。

## 算法原理图解 🖼️

![](img/d729489211926cf67d328807e4bb3ee9_12.png)

![](img/d729489211926cf67d328807e4bb3ee9_14.png)

为了直观理解，我们可以通过一个简化的图示来说明。假设原始矩阵 **V** 中的元素用不同形状（如圆圈、三角）表示有数据（非零）或空数据（零）。

![](img/d729489211926cf67d328807e4bb3ee9_16.png)

![](img/d729489211926cf67d328807e4bb3ee9_18.png)

在矩阵乘法 **W · H** 中，**W** 的每一行与 **H** 的每一列进行点积，生成 **V** 中对应位置的元素。如果 **W** 中某个位置是“圆圈”（代表零），那么无论与 **H** 中什么值相乘，结果都是零。反之，如果 **W** 中是“三角”（代表一个正数特征），它就会与 **H** 中对应行的多个值相乘并求和，从而在 **V** 的对应行中产生多个“三角”。

这个过程形象地展示了NMF如何通过少数几个“基特征”（**W**）的线性组合（系数由 **H** 提供）来近似表示所有原始数据。

## 代码实战：鸢尾花数据集 🌸

理解了原理后，我们通过一个实际案例来演示NMF的应用。我们将使用经典的鸢尾花数据集。

以下是代码实现步骤：

```python
# 1. 导入必要的库
import numpy as np
from sklearn import datasets
from sklearn.decomposition import NMF

# 2. 加载鸢尾花数据集
X, y = datasets.load_iris(return_X_y=True)
print(“原始数据 X 的前5行：”)
print(X[:5])
```

![](img/d729489211926cf67d328807e4bb3ee9_20.png)

鸢尾花数据集的四个特征（花萼长度、花萼宽度、花瓣长度、花瓣宽度）都是具有物理意义的正数，非常适合用NMF处理。如果使用PCA降维，结果可能出现负数，导致解释性变差。

![](img/d729489211926cf67d328807e4bb3ee9_22.png)

![](img/d729489211926cf67d328807e4bb3ee9_24.png)

```python
# 3. 创建NMF模型并进行降维
# n_components 指定降维后的特征数（k值）
# max_iter 指定最大迭代次数以逼近最优解
nmf = NMF(n_components=2, max_iter=1000, init=‘nndsvda’)

![](img/d729489211926cf67d328807e4bb3ee9_26.png)

# 拟合模型并转换数据，得到系数矩阵 W
W = nmf.fit_transform(X)
print(“\n降维后的数据 W (前5行)：”)
print(W[:5])

# 4. 查看降维前后的数据形状
print(f“\n原始数据形状：{X.shape}”) # 输出：(150, 4)
print(f“降维后数据形状：{W.shape}”) # 输出：(150, 2)
```

可以看到，数据从4维成功降到了2维。并且，**W** 中的所有值都是非负的。

## 获取基矩阵与关系验证 🔍

根据NMF公式 **V ≈ W · H**，我们得到了 **W**，那么对应的基矩阵 **H** 在哪里呢？

```python
# 5. 获取基矩阵 H
H = nmf.components_
print(“\n基矩阵 H：”)
print(H)

![](img/d729489211926cf67d328807e4bb3ee9_28.png)

![](img/d729489211926cf67d328807e4bb3ee9_30.png)

# 6. 验证 V ≈ W · H 的关系
# 计算 W 和 H 的乘积，近似还原原始数据 X
X_reconstructed = np.dot(W, H)
print(“\n使用 W 和 H 重构的数据 (前5行)：”)
print(X_reconstructed[:5])
print(“\n原始数据 X (前5行)：”)
print(X[:5])
```

通过对比 `X_reconstructed` 和原始 `X` 的前几行数据，可以发现它们的值非常接近，这验证了 **X ≈ W · H** 的近似关系。

## 算法特点与注意事项 ⚠️

NMF的核心特点是**非负性**。这一约束带来了两个主要影响：
1.  **可解释性强**：分解出的特征（**W**）和系数（**H**）都是非负的，符合许多现实场景中“部分构成整体”的直观理解（例如，一张图片由几种基本图案叠加而成）。
2.  **应用限制**：输入数据必须是非负的。如果数据中包含负数，NMF算法会报错。因此，在使用前需要确保数据满足非负条件，或进行适当的预处理（如归一化到正值范围）。

## 总结 📝

![](img/d729489211926cf67d328807e4bb3ee9_34.png)

![](img/d729489211926cf67d328807e4bb3ee9_36.png)

本节课中我们一起学习了非负矩阵分解。
*   **核心目标**：将非负矩阵 **V** 分解为两个非负矩阵 **W** 和 **H** 的乘积，即 **V ≈ W · H**，以实现降维和特征提取。
*   **关键特性**：所有输入和输出矩阵的元素都必须**非负**，这保证了结果具有良好的可解释性。
*   **工作流程**：通过迭代优化，寻找 **W** 和 **H**，使它们的乘积尽可能接近原始矩阵。
*   **应用场景**：特别适用于所有特征值均为正数的数据集，如图像处理、文本主题提取和生物信息学等领域。
