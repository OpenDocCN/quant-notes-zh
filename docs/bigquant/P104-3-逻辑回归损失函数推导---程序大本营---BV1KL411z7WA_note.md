在本节课中，我们将学习逻辑回归模型的核心组成部分——损失函数的推导过程。我们将从概率表示开始，逐步构建似然函数，并通过数学变换得到最终的损失函数表达式，为后续的参数优化打下基础。

## 概率表示与整合

上一节我们介绍了逻辑回归的预测函数，本节我们来看看如何用数学公式表示样本的预测概率。

我们的目标是预测一个二分类结果，其中标签 `y` 只能是 0 或 1。根据逻辑回归的定义，模型预测 `y=1` 的概率为 `h_θ(x)`，预测 `y=0` 的概率则为 `1 - h_θ(x)`。

我们可以将这两种情况的概率整合到一个统一的公式中：

**P(y | x; θ) = (h_θ(x))^y * (1 - h_θ(x))^(1-y)**

这个公式巧妙地利用了指数运算：当 `y=1` 时，公式简化为 `h_θ(x)`；当 `y=0` 时，公式简化为 `1 - h_θ(x)`。这样，我们就得到了单个样本的预测概率。

## 构建似然函数

![](img/ff623646b48ec569be40c320c4764183_4.png)

在得到单个样本的概率后，我们需要考虑整个数据集的拟合程度。在线性回归中，我们曾使用“最大似然估计”的思想，这里同样适用。

![](img/ff623646b48ec569be40c320c4764183_6.png)

最大似然估计的核心思想是：找到一组参数 `θ`，使得当前观测到的所有数据（样本）出现的**联合概率**最大。对于独立同分布的 `m` 个样本，其联合概率是每个样本概率的乘积。

![](img/ff623646b48ec569be40c320c4764183_8.png)

![](img/ff623646b48ec569be40c320c4764183_10.png)

因此，整个数据集的似然函数 `L(θ)` 定义如下：

**L(θ) = ∏_{i=1}^{m} P(y^{(i)} | x^{(i)}; θ) = ∏_{i=1}^{m} (h_θ(x^{(i)}))^{y^{(i)}} * (1 - h_θ(x^{(i)}))^{1-y^{(i)}}**

![](img/ff623646b48ec569be40c320c4764183_12.png)

我们的目标是最大化这个似然函数 `L(θ)`。

![](img/ff623646b48ec569be40c320c4764183_14.png)

## 对数似然与化简

直接对连乘形式的似然函数 `L(θ)` 进行最大化运算非常困难，无论是数学求导还是计算机计算都不方便。

为了解决这个问题，我们引入对数变换。由于自然对数函数 `ln(x)` 是单调递增函数，最大化 `L(θ)` 等价于最大化 `ln(L(θ))`。更重要的是，对数运算能将连乘转换为连加，极大简化计算。

我们对似然函数取自然对数，得到对数似然函数 `ℓ(θ)`：

**ℓ(θ) = ln(L(θ)) = ∑_{i=1}^{m} [ y^{(i)} * ln(h_θ(x^{(i)})) + (1 - y^{(i)}) * ln(1 - h_θ(x^{(i)})) ]**

这个推导过程应用了对数的运算法则 `ln(a*b) = ln(a) + ln(b)` 和 `ln(a^b) = b * ln(a)`。最终，我们将复杂的连乘问题转化为了一个相对简单的连加问题。

![](img/ff623646b48ec569be40c320c4764183_16.png)

## 从最大似然到损失函数

目前我们得到的是需要**最大化**的对数似然函数 `ℓ(θ)`。然而，在机器学习领域，我们通常习惯定义并**最小化**一个“损失函数”或“成本函数”。

为了实现这个转换，我们只需在对数似然函数前加上一个负号：

**J(θ) = -ℓ(θ) = -∑_{i=1}^{m} [ y^{(i)} * ln(h_θ(x^{(i)})) + (1 - y^{(i)}) * ln(1 - h_θ(x^{(i)})) ]**

这个函数 `J(θ)` 就是逻辑回归的损失函数，通常被称为**交叉熵损失函数**。我们的优化目标从“最大化 `ℓ(θ)`”转变为“最小化 `J(θ)`”。

## 与线性回归的对比

理解逻辑回归损失函数后，可以将其与线性回归进行对比，以加深理解：

以下是两者的核心区别：
*   **损失函数形式**：线性回归通常使用**均方误差损失**，而逻辑回归使用**交叉熵损失**。
*   **输出与假设**：线性回归输出连续值，假设服从高斯分布；逻辑回归输出概率，假设服从伯努利分布。
*   **优化目标**：虽然最终都转化为最小化问题，但它们的出发点不同（最小化误差 vs 最大化似然）。

尽管形式不同，但两者的求解思路高度相似：定义损失函数后，通常都采用**梯度下降**（或其变种）来迭代优化参数 `θ`，因为直接令导数为零求解方程往往非常困难。
