# 机器学习入门：P57：作业介绍

在本节课中，我们将介绍一个关于线性回归的课后作业。这个作业旨在帮助大家巩固对线性回归模型的理解，并学习如何将其应用于实际数据集进行预测。

## 作业概述

本次作业的核心任务是使用线性回归模型来预测火力发电量。我们将使用提供的数据集来完成模型的训练与评估。

![](img/91fe3de5f7eb9f5276804e8e8d52365c_3.png)

## 数据集说明

以下是关于本次作业所用数据集的详细信息。

数据集包含两个文件：
*   `train.txt`：用于训练模型的数据文件。
*   `test.txt`：用于测试模型性能的数据文件。

![](img/91fe3de5f7eb9f5276804e8e8d52365c_5.png)

数据集中包含从 `v0` 到 `v37` 共38个字段（特征）。
