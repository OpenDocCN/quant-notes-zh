# 机器学习：P129：SVM支持向量机对偶问题转化 🔄

![](img/c567d6a795fb5a5d77c5601132a4eff7_0.png)

![](img/c567d6a795fb5a5d77c5601132a4eff7_2.png)

在本节课中，我们将要学习支持向量机（SVM）中一个关键的数学步骤：将对偶问题从原问题中转化出来。我们将理解为什么需要这个转化，以及原问题与对偶问题之间的关系。

上一节我们介绍了如何通过KKT条件构建拉格朗日函数，并得到了一个包含`min`和`max`的优化问题。本节中我们来看看如何将这个“原问题”转化为更容易求解的“对偶问题”。

![](img/c567d6a795fb5a5d77c5601132a4eff7_4.png)

## 原问题的求解困难

我们之前得到的优化问题（公式九）形式如下：

![](img/c567d6a795fb5a5d77c5601132a4eff7_6.png)

**原问题：** `min_x max_λ L(x, λ)`

![](img/c567d6a795fb5a5d77c5601132a4eff7_8.png)

这个公式表示：先固定`x`，求关于拉格朗日乘子`λ`的最大值；然后，在所有`x`中，找到使这个最大值最小的那个`x`。

![](img/c567d6a795fb5a5d77c5601132a4eff7_10.png)

![](img/c567d6a795fb5a5d77c5601132a4eff7_12.png)

这个形式**不太好求**。因为我们需要先求解内部关于`λ`的**最大值**问题，这本身可能就很复杂，然后再在这个基础上求外部的**最小值**，使得整体求解难度很大。

![](img/c567d6a795fb5a5d77c5601132a4eff7_14.png)

## 对偶问题的定义与引入

![](img/c567d6a795fb5a5d77c5601132a4eff7_16.png)

![](img/c567d6a795fb5a5d77c5601132a4eff7_18.png)

为了解决这个困难，我们引入**对偶问题**。它的定义非常直接：将原问题中的`min`和`max`运算顺序对调。

![](img/c567d6a795fb5a5d77c5601132a4eff7_20.png)

**对偶问题：** `max_λ min_x L(x, λ)`

![](img/c567d6a795fb5a5d77c5601132a4eff7_22.png)

这个公式表示：先固定拉格朗日乘子`λ`，求关于`x`的最小值；然后，在所有`λ`中，找到使这个最小值最大的那个`λ`。

从直觉上看，对偶问题**求解起来更简单**。因为先求关于`x`的最小值（通常可以通过求导数为零来求解），然后再求最大值，这个计算路径通常更直接。

![](img/c567d6a795fb5a5d77c5601132a4eff7_24.png)

## 原问题与对偶问题的关系

![](img/c567d6a795fb5a5d77c5601132a4eff7_26.png)

一个自然的问题是：将`min`和`max`直接对调，这样得到的解和原问题的解一样吗？这两个问题等价吗？

答案是：**原问题与对偶问题并不完全等价**。它们之间存在一种“对偶性”关系。

以下是关于对偶性的核心概念：

*   **弱对偶性**：对于所有的优化问题，其**对偶问题的最优值总是小于或等于原问题的最优值**。用公式表示就是：
    `max_λ min_x L(x, λ) ≤ min_x max_λ L(x, λ)`
    这个性质总是成立。

*   **强对偶性**：在某些特定条件下（例如，原问题是凸优化问题且满足Slater条件），对偶问题的最优值**等于**原问题的最优值。即：
    `max_λ min_x L(x, λ) = min_x max_λ L(x, λ)`
    此时，通过对偶问题求得的解就是原问题的最优解。

### 为什么弱对偶性成立？

![](img/c567d6a795fb5a5d77c5601132a4eff7_28.png)

![](img/c567d6a795fb5a5d77c5601132a4eff7_30.png)

我们可以从直观上理解“小于等于”的关系：

![](img/c567d6a795fb5a5d77c5601132a4eff7_32.png)

![](img/c567d6a795fb5a5d77c5601132a4eff7_33.png)

*   **对偶问题** (`max_λ min_x L`)：可以理解为 **“从一堆小的里面找最大的”**。
    *   内部 `min_x`：对于每一个固定的`λ`，我们找到一个最小的函数值（一个“小”的值）。
    *   外部 `max_λ`：我们在所有`λ`对应的这些“小值”中，找出最大的那个。
*   **原问题** (`min_x max_λ L`)：可以理解为 **“从一堆大的里面找最小的”**。
    *   内部 `max_λ`：对于每一个固定的`x`，我们找到一个最大的函数值（一个“大”的值）。
    *   外部 `min_x`：我们在所有`x`对应的这些“大值”中，找出最小的那个。

![](img/c567d6a795fb5a5d77c5601132a4eff7_35.png)

显然，“从一堆小的里面挑出的冠军（最大值）”，其数值不会超过“从一堆大的里面挑出的落败者（最小值）”。因此，对偶问题的值 ≤ 原问题的值。

幸运的是，SVM的优化问题满足**强对偶性**的条件。这意味着我们可以安全地将复杂的原问题转化为相对容易求解的对偶问题，并且最终求得的解是等价的。

![](img/c567d6a795fb5a5d77c5601132a4eff7_37.png)

![](img/c567d6a795fb5a5d77c5601132a4eff7_39.png)

## 总结

![](img/c567d6a795fb5a5d77c5601132a4eff7_41.png)

![](img/c567d6a795fb5a5d77c5601132a4eff7_43.png)

本节课中我们一起学习了支持向量机中对偶问题的转化。

![](img/c567d6a795fb5a5d77c5601132a4eff7_45.png)

![](img/c567d6a795fb5a5d77c5601132a4eff7_47.png)

1.  我们首先指出了原问题 `min_x max_λ L` 直接求解的困难。
2.  接着，我们通过调换极小化和极大化的顺序，定义了**对偶问题** `max_λ min_x L`，它通常更容易求解。
3.  然后，我们探讨了原问题与对偶问题的关系，介绍了**弱对偶性**（对偶解 ≤ 原解）和**强对偶性**（对偶解 = 原解）。
4.  最后，我们了解到SVM模型满足强对偶条件，因此可以通过求解对偶问题来等价地获得原问题的最优解。这为下一节实际求解SVM的对偶形式奠定了重要的理论基础。
