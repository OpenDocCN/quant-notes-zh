上一节我们介绍了如何使用K-Means算法对数据进行聚类划分。本节中，我们来看看如何评价聚类结果的好坏，并解决一个关键问题：K-Means算法中的K值该如何选择。

K值代表我们希望将数据划分成的类别数量。那么，分成几类效果最好呢？这需要一个客观的度量指标。在分类任务中，我们使用准确率；在回归任务中，我们使用R²或均方误差。对于聚类任务，一个常用的评价指标就是**轮廓系数**。

## 轮廓系数的定义与公式

![](img/9963c523da65cd1d15d9ae9f405d5278_8.png)

轮廓系数用于衡量一个样本点与其自身簇内其他点的相似度，以及与其他簇的分离度。

针对单个样本点 `i`，其轮廓系数 `s(i)` 的计算公式如下：

**`s(i) = (b(i) - a(i)) / max(a(i), b(i))`**

其中：
*   **`a(i)`**：样本点 `i` 到其**所属簇内**所有其他样本点的平均距离。这被称为**簇内不相似度**。`a(i)` 越小，说明样本点 `i` 与同簇的其他点越相似，簇越紧密。
*   **`b(i)`**：样本点 `i` 到**其他某个簇**中所有样本点的平均距离的最小值。这被称为**簇间不相似度**。`b(i)` 越大，说明样本点 `i` 与其他簇的点越不相似，簇间分离度越高。

## 如何理解轮廓系数

一个好的聚类结果应该满足两个直观的标准：
1.  **簇内越紧密越好**：同一个簇内的数据点应该彼此相似，距离较近。
2.  **簇间越分离越好**：不同簇的数据点应该差异明显，距离较远。

以下是轮廓系数如何反映这两个标准：
*   当 `a(i)` 很小时，说明点 `i` 与同簇点非常接近。
*   当 `b(i)` 很大时，说明点 `i` 远离其他簇。
*   理想情况下，我们希望 `a(i)` 趋近于0，而 `b(i)` 远大于 `a(i)`。此时，根据公式 `s(i)` 会趋近于1。

因此，**轮廓系数的取值范围在 [-1, 1] 之间**：
*   **`s(i)` 接近 1**：说明样本点 `i` 的聚类结果合理，簇内紧密，簇间分离。
*   **`s(i)` 接近 0**：说明样本点 `i` 处于两个簇的边界上。
*   **`s(i)` 接近 -1**：说明样本点 `i` 可能被分配到了错误的簇。

对于整个数据集的聚类效果，我们计算所有样本点轮廓系数的平均值，作为**总体轮廓系数**。总体轮廓系数越大，说明聚类效果整体越好。

## 使用轮廓系数选择K值

轮廓系数为我们选择K-Means算法中的K值提供了一个量化依据。以下是其应用步骤：

1.  尝试不同的K值（例如，从2到10）。
2.  对每个K值运行K-Means算法进行聚类。
3.  计算每个K值对应的总体轮廓系数。
4.  选择**总体轮廓系数最大**的那个K值，作为最优的聚类数目。

这种方法可以帮助我们避免主观猜测，基于数据本身的结构来确定最合适的类别数量。
