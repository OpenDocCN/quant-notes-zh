在本节课中，我们将学习如何通过多项式升维技术，让线性回归模型能够拟合非线性数据。我们将从创建模拟数据开始，演示普通线性回归的局限性，然后通过升维操作，使模型成功捕捉数据的真实规律。

---

## 导入必要的库

首先，我们需要导入进行数据操作、模型训练和结果可视化所需的库。

![](img/5220e02b5eae456f80a88590452e1a09_4.png)

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
```

---

## 创建模拟数据

接下来，我们创建一组模拟数据。这组数据基于一个二次方程生成，并添加了一些随机噪声，以模拟真实世界中的非线性关系。

```python
# 创建特征数据 X
x = np.linspace(-1, 11, 100)
# 创建目标值 y，基于二次方程并添加噪声
y = (x - 5) ** 2 + 3 * x - 12 + np.random.randn(100)
```

数据创建完成后，我们需要将其转换为机器学习算法要求的二维形状。

```python
# 将一维数据转换为二维数据
x = x.reshape(-1, 1)
```

为了直观理解数据的分布，我们可以将其可视化。

```python
plt.scatter(x, y)
plt.show()
```

---

## 使用普通线性回归

上一节我们创建了数据，本节中我们来看看如果直接使用普通的线性回归模型来拟合这些数据，效果会如何。

我们首先训练一个线性回归模型。

```python
# 声明并训练线性回归模型
model = LinearRegression()
model.fit(x, y)
```

然后，我们生成一组测试数据，并用训练好的模型进行预测。

```python
# 创建测试数据
x_test = np.linspace(-2, 12, 300).reshape(-1, 1)
# 使用模型进行预测
y_predict = model.predict(x_test)
```

![](img/5220e02b5eae456f80a88590452e1a09_6.png)

最后，我们将原始数据点和模型的预测线绘制在同一张图上进行对比。

```python
plt.scatter(x, y)  # 原始数据点
plt.plot(x_test, y_predict, color='green')  # 模型预测线
plt.show()
```

![](img/5220e02b5eae456f80a88590452e1a09_8.png)

观察结果，我们会发现绿色的预测直线无法很好地拟合蓝色的数据点。这是因为我们的数据本质上是非线性的（含有 `x**2` 项），而普通线性回归只能拟合直线。

---

## 通过多项式升维改进模型

![](img/5220e02b5eae456f80a88590452e1a09_10.png)

既然普通线性回归无法解决问题，本节我们将通过对特征进行多项式升维来改进模型。核心思想是为模型提供 `x` 的平方项作为新特征，这样模型就能学习到非线性关系。

![](img/5220e02b5eae456f80a88590452e1a09_12.png)

以下是实现升维的关键步骤：

1.  **创建新特征**：计算原始特征 `x` 的平方，得到 `x**2`。
2.  **合并特征**：将原始特征 `x` 和新特征 `x**2` 在列方向上合并，形成一个新的二维特征数组。

![](img/5220e02b5eae456f80a88590452e1a09_14.png)

```python
# 1. 创建平方项特征
x_squared = x ** 2
# 2. 合并原始特征与平方项特征
x2 = np.concatenate([x, x_squared], axis=1)
```

![](img/5220e02b5eae456f80a88590452e1a09_16.png)

**请注意**：目标值 `y` 不需要任何处理。升维操作只作用于特征 `X`。

![](img/5220e02b5eae456f80a88590452e1a09_18.png)

---

## 使用升维后的数据训练模型

特征升维完成后，我们就可以使用新的二维特征数据 `x2` 来训练一个新的线性回归模型。

```python
# 使用升维后的数据训练新模型
model2 = LinearRegression()
model2.fit(x2, y)
```

同样地，在预测时，测试数据也必须进行相同的升维处理。

```python
# 对测试数据进行相同的升维处理
x_test_squared = x_test ** 2
x_test2 = np.concatenate([x_test, x_test_squared], axis=1)
# 使用新模型进行预测
y_predict2 = model2.predict(x_test2)
```

现在，我们再次可视化结果。

```python
plt.scatter(x, y)  # 原始数据点
plt.plot(x_test, y_predict2, color='red')  # 升维后模型的预测曲线
plt.show()
```

![](img/5220e02b5eae456f80a88590452e1a09_20.png)

这次，红色的预测曲线完美地拟合了蓝色的数据点。我们还可以查看学到的模型参数。

```python
# 输出模型系数和截距
print("模型系数 (coef_):", model2.coef_)
print("模型截距 (intercept_):", model2.intercept_)
```

![](img/5220e02b5eae456f80a88590452e1a09_22.png)

## 总结

![](img/5220e02b5eae456f80a88590452e1a09_26.png)

本节课中我们一起学习了多项式回归升维的核心实战。我们首先看到了普通线性回归在处理非线性数据时的局限性。然后，我们通过为特征添加多项式项（如 `x**2`）进行升维，将问题转化到更高维度的空间中，使得线性回归模型能够拟合出曲线，从而成功捕捉了数据的真实规律。这种方法强大而有效，是处理非线性关系的常用技巧。
