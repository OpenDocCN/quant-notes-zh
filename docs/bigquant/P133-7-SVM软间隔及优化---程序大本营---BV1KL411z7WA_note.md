# 机器学习：P133：SVM软间隔及优化 🧠

在本节课中，我们将要学习支持向量机（SVM）的一个重要概念——软间隔。我们将探讨当数据并非完美线性可分时，如何通过引入松弛变量和惩罚系数来优化模型，使其能够处理现实世界中带有噪声或异常值的数据。

## 从硬间隔到软间隔 🔄

上一节我们介绍了基于线性可分数据的硬间隔SVM。硬间隔要求所有样本点都必须被超平面正确划分，条件非常苛刻。

但在实际应用中，数据往往并非完美可分。例如，可能存在一些样本点“越界”，跑到其他类别的区域中。在这种情况下，无论我们如何寻找，都无法找到一个超平面能够完美地将所有数据点分开。

为了处理这种情况，我们需要引入“软间隔”的概念。相比于硬间隔，软间隔允许部分样本点不满足严格的约束条件，即允许个别样本点出现在间隔带内部，甚至被错误分类。

## 引入松弛变量 📏

为了度量这种“软”的程度，我们为每个样本引入一个**松弛变量**（Slack Variable），通常用希腊字母 **ξ**（读作xi）表示，且 **ξ ≥ 0**。

加入松弛变量后，原始的约束条件 `y_i(w·x_i + b) ≥ 1` 就变成了：
`y_i(w·x_i + b) ≥ 1 - ξ_i`

这意味着，样本点不一定非要满足大于等于1的严格边界，它可以有一定的“误差”。这个ξ_i的值大小，就代表了该样本点的“违规”程度。

以下是不同情况下松弛变量ξ的取值：

*   **完全分类正确且在边界外或边界上**：ξ = 0。样本点完全符合硬间隔的要求。
*   **分类正确但在间隔带内**：0 < ξ < 1。样本点虽然被正确分类，但进入了本不该进入的间隔区域。
*   **被错误分类**：ξ > 1。样本点被分到了错误的类别一侧。

## 优化目标函数：引入惩罚系数C ⚖️

引入了允许误差的松弛变量后，我们需要相应地修改SVM的优化目标函数。我们不能无限制地允许误差，否则模型将失去分类意义。

因此，新的目标函数在最小化 `1/2 * ||w||^2`（最大化间隔）的同时，还需要最小化所有松弛变量的总和，以控制总体误差。新的目标函数如下：

![](img/7f406a81b8a20cc1ed6808ae3e695fbb_1.png)

`min (1/2 * ||w||^2 + C * Σξ_i)`

其中，**C** 是一个大于0的常数，称为**惩罚系数**（Penalty Parameter）或正则化参数。

![](img/7f406a81b8a20cc1ed6808ae3e695fbb_3.png)

*   **C的作用**：它控制着我们对分类错误的容忍程度。
*   **C很大时**：意味着模型对误差的惩罚很重，会迫使所有ξ_i尽可能小，模型会倾向于将所有样本（包括可能的噪声点）都正确分类，这可能导致**过拟合**。
*   **C很小时**：意味着模型对误差的惩罚较轻，允许较大的ξ_i存在，模型的间隔会变“宽”，容忍更多的样本点出现在间隔带内或被误分，这可能导致**欠拟合**。

![](img/7f406a81b8a20cc1ed6808ae3e695fbb_5.png)

**公式总结**：
软间隔SVM的完整优化问题可以表述为：
```
min (1/2 * ||w||^2 + C * Σξ_i)
s.t. y_i(w·x_i + b) ≥ 1 - ξ_i, 且 ξ_i ≥ 0 (对于所有i)
```

## 在代码中理解参数C 💻

现在，我们回到代码层面来理解这个参数。在Scikit-learn的SVM实现中，参数`C`正是我们上面讨论的惩罚系数。

```python
# 在sklearn的SVC中
from sklearn.svm import SVC
model = SVC(C=1.0) # C的默认值通常是1.0
```

![](img/7f406a81b8a20cc1ed6808ae3e695fbb_7.png)

![](img/7f406a81b8a20cc1ed6808ae3e695fbb_9.png)

查看官方文档对`C`的描述：“正则化参数。正则化的强度与C成反比。” 这意味着：
*   `C`值**越大**，正则化强度越**弱**，模型越倾向于拟合训练数据（可能过拟合）。
*   `C`值**越小**，正则化强度越**强**，模型更倾向于找到一个更宽间隔的简单模型（可能欠拟合）。

![](img/7f406a81b8a20cc1ed6808ae3e695fbb_11.png)

这和我们理论上的分析完全一致：C越大，对误差（ξ）的惩罚越大，模型越不想犯错；C越小，对误差越宽容。

## 总结 📝

![](img/7f406a81b8a20cc1ed6808ae3e695fbb_13.png)

本节课中我们一起学习了SVM的软间隔优化。

![](img/7f406a81b8a20cc1ed6808ae3e695fbb_15.png)

1.  **核心问题**：现实数据常非线性可分，硬间隔SVM无法应用。
2.  **解决方案**：引入**软间隔**，允许部分样本点不满足严格约束。
3.  **关键工具**：
    *   **松弛变量 ξ**：量化每个样本点的“违规”程度。
    *   **惩罚系数 C**：在最大化间隔和最小化分类错误之间进行权衡的超参数。
4.  **优化目标**：最小化 `1/2 * ||w||^2 + C * Σξ_i`，这是一个权衡“间隔宽度”和“分类误差”的过程。
5.  **实践意义**：参数`C`是调整SVM模型性能的关键。需要通过交叉验证等技术来寻找最适合数据集的`C`值。
