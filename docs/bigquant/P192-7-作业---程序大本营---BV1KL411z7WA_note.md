## 🌲 决策树与归一化

![](img/be0533063fd9ca2cf8848da1699e019c_3.png)

决策树是一种基于树状结构的模型，它通过一系列**if-then**规则对数据进行分割。其核心在于根据特征值寻找最佳分割点，以最大化信息增益（分类）或最小化均方误差（回归）。

决策树算法在分裂时，**只关心特征值之间的顺序和阈值**，而不关心特征值的绝对大小或尺度。因此，特征的缩放（如归一化或标准化）不会改变数据的顺序关系，也就不会影响树的分裂点选择。

**结论**：对于决策树（包括分类树和回归树），数据**不需要**进行归一化处理。

以下是验证方法：
*   可以使用 `sklearn.tree.DecisionTreeClassifier` 和 `DecisionTreeRegressor` 分别对原始数据和归一化后的数据进行训练。
*   观察两种情况下生成的树结构、特征重要性以及模型性能指标（如准确率、MSE），它们应该保持一致。

代码示例（思路）：
```python
from sklearn.tree import DecisionTreeClassifier
from sklearn.preprocessing import StandardScaler

# 假设 X_train, y_train 是原始数据
model_raw = DecisionTreeClassifier()
model_raw.fit(X_train, y_train)

# 对数据进行归一化
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train)
model_scaled = DecisionTreeClassifier()
model_scaled.fit(X_train_scaled, y_train)

# 比较两者的性能或树结构
```

## 📊 逻辑斯蒂回归、岭回归与归一化

与决策树不同，逻辑斯蒂回归和岭回归都属于**基于梯度或距离的线性模型**。

*   **逻辑斯蒂回归**：核心是**sigmoid函数**和**极大似然估计**，其决策边界是线性的，形式为 `σ(w^T * x + b)`。
*   **岭回归**：在线性回归的基础上加入了**L2正则化项**，损失函数为 `||y - Xw||^2_2 + α * ||w||^2_2`。

这些模型的目标函数优化过程（如梯度下降）会受到特征尺度的影响。如果特征尺度差异巨大，模型会倾向于更关注数值大的特征，导致：
1.  **收敛速度慢**：梯度下降过程会沿着等高线曲折前进，需要更多迭代。
2.  **解的质量可能下降**：特别是对于正则化模型（如岭回归），正则化项是对所有权重进行同等惩罚。如果某个特征本身数值很大，其对应的权重即使很小，也会对预测产生巨大影响，这违背了正则化“公平惩罚”的初衷。

**结论**：对于逻辑斯蒂回归、岭回归等线性模型，数据**需要**进行归一化处理，以确保所有特征处于同一量纲，帮助模型更快、更稳定地找到最优解。

## 🎯 总结

![](img/be0533063fd9ca2cf8848da1699e019c_5.png)

本节课中我们一起学习了不同机器学习算法对数据归一化处理的需求差异。

*   **决策树（分类/回归）**：其分裂规则基于特征值的相对顺序，与尺度无关，因此**无需归一化**。
*   **逻辑斯蒂回归与岭回归**：作为基于梯度的线性模型，其优化过程和正则化效果受特征尺度影响，因此**必须进行归一化**。

理解这一区别有助于我们在实际项目中正确地进行数据预处理，从而构建更有效的模型。
