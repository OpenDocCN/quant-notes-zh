# 机器学习：P108：逻辑回归梯度下降更新公式推导

![](img/dd0ef85c4b95acffaa2acbc8488eccc7_0.png)

在本节课中，我们将深入探讨逻辑回归的核心——梯度下降更新公式的推导过程。我们将从回顾逻辑回归的基本概念开始，逐步推导出其权重更新规则，并解释其与线性回归的深刻联系。

## 回顾：逻辑回归与Sigmoid函数

上一节我们介绍了逻辑回归，其核心是Sigmoid函数。该函数能将任意实数映射到(0,1)区间，输出值可解释为概率。

Sigmoid函数的公式如下：
`σ(z) = 1 / (1 + e^{-z})`
其中，`z = θ^T x`，即线性方程的计算结果。

这个映射过程类似于将线性回归的预测值转化为一个概率，从而用于解决分类问题。

## 梯度下降与权重更新

与线性回归类似，逻辑回归也使用梯度下降法来寻找最优参数（权重），使损失函数最小化。梯度下降的核心是权重的迭代更新公式。

权重更新公式如下：
`θ_{j}^{(t+1)} = θ_{j}^{(t)} - α * ∂J(θ) / ∂θ_{j}`
其中：
*   `θ_{j}^{(t+1)}` 是第 `t+1` 次迭代后第 `j` 个参数的值。
*   `θ_{j}^{(t)}` 是第 `t` 次迭代时第 `j` 个参数的值。
*   `α` 是学习率，控制每次更新步长的大小。
*   `∂J(θ) / ∂θ_{j}` 是损失函数 `J(θ)` 对参数 `θ_j` 的偏导数，即梯度。

我们的目标是求出这个梯度 `∂J(θ) / ∂θ_{j}` 的具体表达式。

## 关键一步：Sigmoid函数的导数

在开始推导损失函数的梯度之前，我们需要了解Sigmoid函数一个优美的数学特性：其导数可以用自身表示。

对Sigmoid函数 `σ(z)` 求导：
`σ‘(z) = σ(z) * (1 - σ(z))`
这个特性将大大简化后续损失函数求导的过程。

## 损失函数梯度推导

上一节我们得到了逻辑回归的损失函数（对数似然损失）：
`J(θ) = - (1/m) * Σ [y^{(i)} * log(h_θ(x^{(i)})) + (1 - y^{(i)}) * log(1 - h_θ(x^{(i)}))]`
其中 `h_θ(x) = σ(θ^T x)` 是Sigmoid函数。

现在，我们对损失函数 `J(θ)` 关于某个参数 `θ_j` 求偏导。为清晰起见，我们先考虑单个样本 `(x, y)` 的贡献，最后再求和取平均。

1.  **展开损失函数**：对于单个样本，损失项为 `L(θ) = -[y * log(h_θ(x)) + (1-y) * log(1 - h_θ(x))]`。
2.  **对 `θ_j` 求偏导**：
    `∂L(θ)/∂θ_j = - [ y * (1/h_θ(x)) * ∂h_θ(x)/∂θ_j + (1-y) * (1/(1-h_θ(x))) * (-∂h_θ(x)/∂θ_j) ]`
    这里应用了对数函数 `log(x)` 的导数为 `1/x` 以及链式法则。
3.  **代入Sigmoid导数**：我们知道 `h_θ(x) = σ(θ^T x)`，所以 `∂h_θ(x)/∂θ_j = h_θ(x) * (1 - h_θ(x)) * x_j`。将其代入上式。
4.  **合并化简**：
    `∂L(θ)/∂θ_j = - [ y * (1/h_θ(x)) * (h_θ(x)(1-h_θ(x))x_j) - (1-y) * (1/(1-h_θ(x))) * (h_θ(x)(1-h_θ(x))x_j) ]`
    `= - [ y * (1 - h_θ(x)) * x_j - (1-y) * h_θ(x) * x_j ]`
    `= - [ y*x_j - y*h_θ(x)*x_j - h_θ(x)*x_j + y*h_θ(x)*x_j ]`
    `= - [ y*x_j - h_θ(x)*x_j ]`
    `= ( h_θ(x) - y ) * x_j`
5.  **考虑所有样本**：将m个样本的梯度求和并取平均，得到最终的梯度表达式。
    `∂J(θ)/∂θ_j = (1/m) * Σ ( h_θ(x^{(i)}) - y^{(i)} ) * x_j^{(i)}`

## 最终的更新公式

将求得的梯度代回最初的权重更新公式，我们得到逻辑回归参数 `θ_j` 的更新规则：

`θ_{j}^{(t+1)} = θ_{j}^{(t)} - α * (1/m) * Σ ( h_θ(x^{(i)}) - y^{(i)} ) * x_j^{(i)}`

如果用向量形式表示所有参数 `θ` 的更新，则为：
`θ^{(t+1)} = θ^{(t)} - α * (1/m) * X^T ( σ(Xθ^{(t)}) - y )`

## 与线性回归的联系

观察最终的梯度形式 `( h_θ(x) - y ) * x_j`，你会发现它与线性回归最小二乘法损失函数（均方误差）的梯度形式 `( (θ^T x) - y ) * x_j` 惊人地相似。

唯一的区别在于，逻辑回归的预测值是Sigmoid变换后的概率 `h_θ(x)`，而线性回归是直接的线性预测值 `θ^T x`。这揭示了二者在优化思想上的同源性。逻辑回归可以看作是在线性回归的基础上，增加了一个Sigmoid“激活函数”，使其适用于分类任务。

## 总结

本节课中我们一起学习了逻辑回归梯度下降更新公式的完整推导。
1.  我们首先回顾了逻辑回归使用Sigmoid函数将线性输出映射为概率。
2.  然后引入了梯度下降法的基本权重更新框架。
3.  接着，我们利用了Sigmoid函数导数可用自身表示的特性 `σ‘(z) = σ(z)(1-σ(z))`。
4.  通过对数似然损失函数进行求导，并经过一系列化简，最终得到了简洁的梯度表达式：`(1/m) * Σ ( h_θ(x^{(i)}) - y^{(i)} ) * x_j^{(i)}`。
5.  最后，我们指出了该更新公式与线性回归梯度公式在形式上的高度一致性，理解了逻辑回归实质上是线性模型在分类问题上的一个扩展。
