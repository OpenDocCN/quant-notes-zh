# 金融因子预处理：4.2.3.1：金融因子预处理的常用流程 📊

![](img/a1630b1254e8cf5cce77f3b6e053b05b_0.png)

在本节课程中，我们将学习金融因子数据预处理的标准化流程。这一流程旨在处理原始因子数据中的常见问题，如异常值、量纲差异和缺失值，为后续的量化分析和模型构建打下坚实基础。

上一节我们介绍了金融因子的基本概念，本节中我们来看看如何对它们进行系统性的预处理。

## 数据预处理流程概述

金融因子数据通常是时间序列数据。在处理这类数据时，我们通常会对每个时间截面（即某一特定交易日）上的横截面数据进行一系列标准化操作。标准流程包括：**中位数去极值处理**、**标准化处理**和**缺失值处理**。

## 详细处理步骤

以下是金融因子预处理的三个核心步骤。

### 1. 中位数去极值处理

此步骤旨在识别并处理横截面数据中的异常值（极值），以避免它们对整体分析产生过大影响。

*   **获取截面数据**：首先，选取某一特定时间点（如某交易日）所有股票的因子值，构成一个横截面数据集。
*   **计算中位数与绝对偏差**：计算该截面数据的中位数（Median）。然后，计算每个数据点与中位数的绝对偏差（Absolute Deviation）。
*   **计算绝对偏差的中位数（MAD）**：计算所有绝对偏差的中位数，得到一个稳健的离散度估计。
*   **定义并处理异常值**：将异常值定义为距离中位数超过 `n` 倍 MAD 的数据点。例如，常设定 `n=5`。公式表示为：
    **异常值条件：|x_i - Median| > n * MAD**
    对于识别出的异常值，通常有两种处理方式：
    *   **缩尾处理（Winsorization）**：将超过 `n * MAD` 的极大值替换为 `Median + n * MAD`，极小值替换为 `Median - n * MAD`。
    *   **直接删除**：在特定分析中，也可能直接删除这些异常样本。

### 2. 标准化（归一化）处理

此步骤旨在消除不同因子之间的量纲差异，使它们具有可比性，并转化为适合机器学习模型输入的格式。

*   **计算截面统计量**：对经过去极值处理的截面数据，计算其平均值（Mean）和标准差（Std）。
*   **Z-Score 标准化**：将每个原始因子值减去该截面的平均值，再除以其标准差。公式表示为：
    **x_i‘ = (x_i - Mean) / Std**
*   **处理效果**：经过此处理，新的截面数据 `x_i‘` 近似服从均值为0、标准差为1的标准正态分布。这使得不同因子处于同一尺度，便于比较和加权，同时也加速了后续机器学习模型的收敛。

### 3. 缺失值处理

最后一步是处理标准化后数据中可能存在的缺失值（NaN）。

*   **处理原因**：即使原始数据经过处理，标准化过程或数据源本身仍可能产生缺失值。大多数数学模型无法直接处理缺失值。
*   **常用方法**：
    *   **填充法**：用特定值填充缺失值，例如用0（假设标准化后均值为0）、该因子的截面均值或中位数进行填充。
    *   **删除法**：直接删除含有缺失值的样本（行）或特征（列）。在横截面分析中，需谨慎使用删除法，以免损失过多数据。

---

本节课中我们一起学习了金融因子预处理的完整流程。我们首先通过**中位数去极值法**稳健地处理了异常值；然后通过**Z-Score标准化**消除了量纲，使数据服从标准正态分布；最后对**缺失值**进行了填充或删除。这套流程是量化金融中准备因子数据的通用方法，能有效提升数据质量与后续模型的性能。
