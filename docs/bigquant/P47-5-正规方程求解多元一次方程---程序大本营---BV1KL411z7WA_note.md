在本节课中，我们将学习如何使用正规方程（Normal Equation）来求解多元一次方程组。这是一种基于线性代数的解析方法，可以直接计算出线性回归问题的最优解，而无需使用梯度下降等迭代算法。我们将从简单的二元一次方程入手，逐步扩展到三元乃至八元一次方程，并理解其背后的数学原理。

## 线性回归举例 🔍

上一节我们介绍了线性回归的基本概念。本节中，我们来看一个具体的例子。

我们有一个简单的二元一次方程组：
*   x + y = 14
*   2x - y = 10

我们可以通过口算得出解：x = 8, y = 6。接下来，我们将用代码来验证这个结果。

![](img/dd7bad330f6accd722d4d03f1981eb72_5.png)

![](img/dd7bad330f6accd722d4d03f1981eb72_7.png)

## 正规方程的使用 ⚙️

现在，我们来看看如何使用正规方程来求解上述方程组。首先，我们需要将方程组转化为矩阵形式。

以下是构建系数矩阵 `X` 和目标值向量 `y` 的步骤：
```python
import numpy as np
# 系数矩阵 X，每一行代表一个方程中 x 和 y 的系数
X = np.array([[1, 1],
              [2, -1]])
# 目标值向量 y
y = np.array([14, 10])
```

NumPy 的线性代数模块提供了直接求解线性方程组的方法。
```python
# 使用 np.linalg.solve 直接求解
solution = np.linalg.solve(X, y)
print(solution) # 输出: [8. 6.]
```
求解结果与我们口算的一致。我们可以将这个解代入原方程进行验证：
*   8 + 6 = 14
*   2*8 - 6 = 10

## 正规方程的数学原理 🧮

刚才我们使用了 `np.linalg.solve` 这个便捷函数。现在，让我们深入了解一下它背后的数学原理——正规方程。

![](img/dd7bad330f6accd722d4d03f1981eb72_9.png)

正规方程的公式如下：
**w = (X^T X)^{-1} X^T y**

![](img/dd7bad330f6accd722d4d03f1981eb72_11.png)

![](img/dd7bad330f6accd722d4d03f1981eb72_13.png)

![](img/dd7bad330f6accd722d4d03f1981eb72_15.png)

其中：
*   **w** 是我们要求解的权重向量（即方程的解，如 [x, y]）。
*   **X** 是特征矩阵（系数矩阵）。
*   **X^T** 是 X 的转置矩阵。
*   **^{-1}** 表示矩阵的逆。
*   **y** 是目标值向量。

![](img/dd7bad330f6accd722d4d03f1981eb72_17.png)

让我们根据这个公式，一步步手动计算之前二元一次方程的解：
```python
# 1. 计算 X 的转置 X^T
X_T = X.T
# 2. 计算 X^T 点乘 X
step1 = X_T.dot(X)
# 3. 计算 (X^T X) 的逆矩阵
step2 = np.linalg.inv(step1)
# 4. 计算 (X^T X)^{-1} 点乘 X^T
step3 = step2.dot(X_T)
# 5. 最后点乘 y，得到解 w
w = step3.dot(y)
print(w) # 输出: [8. 6.]
```
可以看到，通过分步执行正规方程公式，我们得到了与直接调用 `solve` 函数相同的结果。`np.linalg.inv` 用于计算矩阵的逆，你可以将其理解为矩阵的“倒数”。

## 求解三元一次方程 🔢

理解了二元方程后，我们进一步挑战一个三元一次方程组：
*   x - y + z = 180
*   2x + y - z = 83
*   3x - 2y + 6z = 256

![](img/dd7bad330f6accd722d4d03f1981eb72_19.png)

对于更复杂的方程组，手动计算变得困难，但使用正规方程则非常简单。

![](img/dd7bad330f6accd722d4d03f1981eb72_21.png)

以下是求解三元一次方程的完整代码：
```python
# 构建系数矩阵 X 和目标值向量 y
X = np.array([[1, -1, 1],
              [2, 1, -1],
              [3, -2, 6]])
y = np.array([180, 83, 256])

![](img/dd7bad330f6accd722d4d03f1981eb72_23.png)

# 使用正规方程公式求解
w = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)
print(w) # 输出解向量 [x, y, z]
```
运行代码，我们可以快速得到方程组的解。这展示了正规方程在处理多维线性问题时的强大能力。

![](img/dd7bad330f6accd722d4d03f1981eb72_25.png)

## 挑战：八元一次方程 🚀

![](img/dd7bad330f6accd722d4d03f1981eb72_27.png)

最后，我们来看一个更具挑战性的问题——八元一次方程组。当方程数量和数据维度变得非常大时，手动提取系数和计算几乎不可能，但正规方程依然可以轻松应对。

假设我们有一个包含8个未知数的方程组，其系数矩阵 `X` 和目标向量 `y` 已经给出（数据较长，此处用省略号表示）。
```python
# 假设 X 和 y 是已经定义好的大型矩阵和向量
# X.shape 可能为 (m, 8)， y.shape 为 (m,)，其中 m 是方程数量
w = np.linalg.inv(X.T.dot(X)).dot(X.T).dot(y)
print(w) # 输出包含8个值的解向量
```
无论问题多么复杂，只要它是线性可分的（即存在唯一解），我们都可以通过这一行代码快速得到解析解。`X.T` 操作代表矩阵转置，即行与列互换。

## 总结 📝

![](img/dd7bad330f6accd722d4d03f1981eb72_29.png)

本节课中我们一起学习了正规方程。
1.  **核心概念**：我们学习了正规方程 **w = (X^T X)^{-1} X^T y**，这是一个用于求解线性回归问题最优解的解析方法。
2.  **应用方法**：我们使用 NumPy 库，通过 `np.linalg.solve` 或分步实现正规方程公式，成功求解了从二元到八元的线性方程组。
3.  **优势**：与梯度下降等迭代法相比，正规方程不需要选择学习率，也不需要多次迭代，可以直接计算出精确解，特别适用于特征数量不是特别巨大的情况。
4.  **注意点**：当特征矩阵 `X` 的维度非常高（例如上万维）时，计算逆矩阵 `(X^T X)^{-1}` 的代价会非常大，此时可能更适合使用梯度下降法。
