## 理解数据维度与升维操作 🧮

![](img/149d8d6010f41302ba28ad6b1f51e11e_27.png)

![](img/149d8d6010f41302ba28ad6b1f51e11e_29.png)

我们可以再重新解释一下。你能够看到，这个 `x` 我们原来给的数据是一维的。之所以说它是一维，是因为虽然数据的形状是二维的（例如一个数组），但你会发现它只有一列。只有一列，我们称之为一维的特征。

![](img/149d8d6010f41302ba28ad6b1f51e11e_31.png)

![](img/149d8d6010f41302ba28ad6b1f51e11e_33.png)

我们给的数据，你发现它是拐弯的。根据中学知识，你知道既然它拐弯了，就像一个抛物线，那这个抛物线就是二次幂。

![](img/149d8d6010f41302ba28ad6b1f51e11e_35.png)

![](img/149d8d6010f41302ba28ad6b1f51e11e_37.png)

我们只给了一维的数据，它拟合出来的效果就非常不好。因为我们的规律是拐弯的，但是使用线性回归训练一维数据时，线性模型找不到拐弯的规律。线性回归就是直来直去，它不会拐弯。所以它只能感觉到数据整体是增加的，但具体的拐弯规律没找到。

## 升维的具体操作与效果对比 ⬆️

![](img/149d8d6010f41302ba28ad6b1f51e11e_43.png)

那怎么操作呢？我们说升维。什么叫升维？我们来对比一下。

![](img/149d8d6010f41302ba28ad6b1f51e11e_45.png)

以下是升维前后的数据对比：

![](img/149d8d6010f41302ba28ad6b1f51e11e_47.png)

![](img/149d8d6010f41302ba28ad6b1f51e11e_49.png)

```python
# 升维前的数据 (前5条)
display(x.head())
# 升维后的数据 (前5条)
display(x2.head())
```

通过对比，你就能发现不同。升维前，数据是一维特征。升维后，数据是二维特征。这第二维和第一维有关系：第二维就是第一维的平方。

我们升到二维，其实就是在原有数据的基础上，给它加了一个平方项。你的数据不能凭空产生，第二列就是第一列的平方。

这就是“衍生”：从原来的一维，衍生到二维、三维，甚至更多。我们发现，数据经过这样的变化之后，再使用同样的线性回归模型（模型没有变，只是数据变了），规律就找到了。

很多时候，不是模型不行，而是数据不行。那么修改数据、改变方向，让自己多学一点知识，就可以取得更好的效果。

## 总结 📝

本节课中我们一起学习了多项式回归中的升维概念。我们了解到：

![](img/149d8d6010f41302ba28ad6b1f51e11e_57.png)

1.  面对新数据，合适的升维维度需要通过尝试来确定。
2.  升维的本质是通过对原始特征进行数学运算（如平方），衍生出新的特征，将数据从低维空间映射到高维空间。
3.  即使使用简单的线性模型，只要数据经过合适的升维处理，也能拟合出复杂的非线性规律。
4.  核心操作可以概括为：`新特征 = 原始特征 ** n`，其中 `n` 是我们要尝试的维度。

![](img/149d8d6010f41302ba28ad6b1f51e11e_59.png)

![](img/149d8d6010f41302ba28ad6b1f51e11e_60.png)

关键点在于：**通过改变数据的表示方式（升维），可以让简单的模型解决复杂的问题**。
