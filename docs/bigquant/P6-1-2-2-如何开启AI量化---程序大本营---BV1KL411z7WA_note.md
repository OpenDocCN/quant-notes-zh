# AI量化投资入门：1.2.2：如何开启AI量化之旅 🚀

在本节课中，我们将要学习如何开启AI量化投资。我们将了解一个实用的AI量化平台，并介绍学习AI量化所需的知识体系与核心工具。



上一节我们介绍了AI量化是投资的必然选择，本节中我们来看看如何具体开启AI量化。



开启AI量化最快的途径，是在优矿（UQER）这个人工智能量化平台上进行策略研究。优矿是国内唯一一家以AI为基因和特色的量化平台，成立至今已有三年多的历史。



优矿成立的初衷，是希望将AI的能力赋予广大投资者。我们可以从影响AI发展的三个核心要素来了解这个平台。

以下是优矿平台在三个要素上的优势：

*   **数据**：优矿拥有股票、期货、期权、基金、美股、港股等金融数据。
*   **算法**：优矿支持主流的机器学习和深度学习框架，例如 `scikit-learn`、`TensorFlow`、`PyTorch` 等。它还提供专门针对股票市场的专有选股排序模型，例如 `AlphaRank`。
*   **算力**：优矿提供比个人计算机更强大的计算资源，并且配置灵活。



![](img/40aaae40c465f876e3a7cced052d0fe7_3.png)

了解了平台之后，我们来看看学习AI量化的知识体系。AI量化课程体系主要分为以下三个部分。

以下是课程体系的三个阶段：

1.  **初级阶段**：介绍编程知识和AI量化策略的基本研发流程。目标是帮助学员能够按照模板策略，快速复现并运行策略。
2.  **中级阶段**：在模板策略或他人分享的策略基础上，进行创新和调整。例如，修改参数、算法或交易逻辑。
3.  **高级阶段**：介绍因子研究体系以及机器学习、深度学习的最佳实践。目标是帮助学员深入理解策略逻辑，并评估策略的收益与风险。



接下来，我们讨论本次培训的编程语言选择。我们选择的是目前在量化投资领域非常热门的 **Python** 语言。

Python语言的最大特点是灵活，并且拥有大量的开源软件包。在所有编程语言中，Python也属于上手比较容易的语言。

虽然Python的纯计算性能不如C++，但这就像我们要去市中心，不会选择速度最快的飞机，而是选择更灵活的汽车。这就是我们选择Python的原因。

因为在策略研究阶段，研究人员需要将时间和精力集中在业务逻辑的灵活调整与修改上，使用Python是最高效的。



在金融数据处理方面，Python有两个使用特别广泛且强大的开源包。

以下是两个核心的金融数据处理库：

*   **NumPy**：提供高性能的多维数组对象及计算工具。
    ```python
    import numpy as np
    # 创建一个数组
    price_array = np.array([100, 101, 102, 103])
    ```
*   **Pandas**：提供了快速、灵活、明确的数据结构，旨在使数据清洗和分析工作变得简单快捷。
    ```python
    import pandas as pd
    # 创建一个DataFrame
    df = pd.DataFrame({'price': [100, 101, 102, 103]})
    ```

这两个在金融领域非常热门的包，在设计之初就考虑到了金融研究的需求。



一旦踏上AI量化的学习之路，你就成为了一名“宽客”（Quant）。宽客是处于金融领域金字塔塔尖的一群人，他们运用数学和计算机模型来管理基金资产。
