## 数据加载与查看

![](img/594376e487898a354bab9348909595f7_1.png)

上一节我们介绍了线性回归的基本原理，本节中我们来看看如何将理论应用于实际数据。

首先，我们需要加载数据。作业提供的数据文件是CSV格式，但其列与列之间使用制表符（Tab）进行分隔，而非标准的逗号。因此，在使用Pandas读取时，需要指定分隔符参数。

![](img/594376e487898a354bab9348909595f7_3.png)

以下是使用Pandas加载数据的核心代码：

```python
import pandas as pd
# 读取训练数据，指定分隔符为制表符 '\t'
train_data = pd.read_csv('train.csv', sep='\t')
```

执行上述代码后，数据将被成功加载到`train_data`这个DataFrame中。

## 理解数据结构

数据加载完成后，我们需要理解其结构。训练数据（`train.csv`）的最后一列名为`target`，这就是我们的预测目标值（y）。而`target`列之前的所有列，则是特征数据（X）。

我们可以使用Pandas的切片操作将特征（X）和目标（y）分开：

```python
# 假设所有列直到倒数第二列都是特征
X_train = train_data.iloc[:, :-1]
# 最后一列是目标值
y_train = train_data.iloc[:, -1]
```

分开后，我们就可以使用线性回归模型对`X_train`和`y_train`进行建模。

## 测试数据与预测

除了训练数据，我们还拥有一份测试数据（`test.csv`）。测试数据中只包含特征（X），不包含目标值（y）。

以下是加载测试数据的代码：

```python
test_data = pd.read_csv('test.csv', sep='\t')
```

我们的任务就是使用在训练集上训练好的线性回归模型，对测试集的特征进行预测，得到其对应的`target`值。这通过模型的`.predict()`方法实现：

```python
# 假设 lr_model 是已训练好的线性回归模型
y_test_pred = lr_model.predict(test_data)
```
