在本节课中，我们将学习聚类算法的核心划分标准，即如何衡量数据点之间的相似度。理解这些度量标准是掌握聚类算法的关键。

上一节我们介绍了聚类是无监督学习，其目标是让同一类别内的数据点尽可能相似，不同类别间的数据点尽可能不同。本节中，我们来看看如何具体量化这种“相似”与“不同”。

## 相似度的度量

对于计算机而言，数据点之间的相似度需要通过数学方法进行量化。以下是几种常用的度量方式。

### 欧式距离

欧式距离是最直观的距离度量方式，它计算的是多维空间中两点之间的直线距离。

![](img/0ac0103663d5db169d804534888e971a_5.png)

![](img/0ac0103663d5db169d804534888e971a_7.png)

*   **二维空间公式**：`distance = sqrt((x1 - x2)^2 + (y1 - y2)^2)`
*   **三维空间公式**：`distance = sqrt((x1 - x2)^2 + (y1 - y2)^2 + (z1 - z2)^2)`

![](img/0ac0103663d5db169d804534888e971a_9.png)

![](img/0ac0103663d5db169d804534888e971a_11.png)

距离越近，表示两个数据点越相似。

![](img/0ac0103663d5db169d804534888e971a_13.png)

![](img/0ac0103663d5db169d804534888e971a_15.png)

### 闵可夫斯基距离

![](img/0ac0103663d5db169d804534888e971a_17.png)

闵可夫斯基距离是一个更通用的距离公式，通过调整参数 `p`，可以衍生出多种距离度量。

![](img/0ac0103663d5db169d804534888e971a_19.png)

![](img/0ac0103663d5db169d804534888e971a_21.png)

其通用公式为：
`distance = (Σ|xi - yi|^p)^(1/p)`

根据 `p` 取值的不同，它代表不同的距离：

*   **曼哈顿距离 (p=1)**：当 `p=1` 时，公式变为 `distance = Σ|xi - yi|`。这模拟了在网格状道路（如城市街区）中行走的距离，必须沿着坐标轴方向移动，不能走对角线。
*   **欧式距离 (p=2)**：当 `p=2` 时，就是标准的欧式距离。
*   **切比雪夫距离 (p=∞)**：当 `p` 趋向于无穷大时，距离由各维度差值中最大的那个决定，即 `distance = max(|xi - yi|)`。这类似于国际象棋中“王”的走法，横、竖、斜均可一步到达相邻格，两点间的最少步数等于其行差和列差中的最大值。

### 余弦相似度

![](img/0ac0103663d5db169d804534888e971a_23.png)

在处理某些类型的数据（如文本）时，直接使用距离度量可能不合适。此时可以使用余弦相似度，它关注的是两个向量在方向上的差异，而非绝对距离。

余弦相似度通过计算两个向量夹角的余弦值来评估其相似性，公式如下：
`similarity = cos(θ) = (A·B) / (||A|| * ||B||)`
其中 `A·B` 是向量的点积，`||A||` 和 `||B||` 是向量的模（长度）。

展开后的计算公式为：
`similarity = (x1*x2 + y1*y2) / (sqrt(x1^2 + y1^2) * sqrt(x2^2 + y2^2))`

余弦相似度的取值范围为 **[-1, 1]**。
*   **值越接近 1**：表示两个向量方向越一致，越相似。
*   **值越接近 -1**：表示两个向量方向完全相反。
*   **值等于 0**：表示两个向量正交（夹角为90度），不相关。

余弦相似度的一个典型应用是**文本分类**。通过将文章转化为特征向量（如词频向量），计算其余弦相似度，可以有效地对文章进行自动归类。
