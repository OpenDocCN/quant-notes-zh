## 概述

![](img/e1b66d39812ce5170ba3347bf03ac945_2.png)

上一节我们介绍了梯度下降的基本概念，本节中我们来看看如何将其应用到具体的线性回归问题中。线性回归的目标是找到一个线性方程，使其预测值尽可能接近真实值。无论方程简单（如一元一次）还是复杂（如多元一次），其核心思想都基于最小二乘法。

## 最小二乘法与损失函数

线性回归模型的拟合优劣，需要一个标准来衡量。这个标准就是损失函数，我们通常使用最小二乘法来定义它。

最小二乘法的核心思想是：**希望模型的预测值与真实值之间的差距尽可能小**。这种差距通过“残差”（预测值减真实值）的平方和来衡量。

其公式如下：

**公式：**
`J(θ) = 1/2 * Σ (h_θ(x^(i)) - y^(i))^2`

**代码描述（伪代码）：**
```python
def compute_cost(X, y, theta):
    m = len(y) # 样本数量
    predictions = X.dot(theta) # 计算预测值 h_θ(x)
    cost = (1/(2*m)) * np.sum(np.square(predictions - y))
    return cost
```

![](img/e1b66d39812ce5170ba3347bf03ac945_4.png)

**公式解读：**
*   `J(θ)`：损失函数，值越小表示模型拟合越好。
*   `h_θ(x^(i))`：第 `i` 个样本的预测值。
*   `y^(i)`：第 `i` 个样本的真实值。
*   `Σ`：对所有训练样本求和。

![](img/e1b66d39812ce5170ba3347bf03ac945_6.png)

![](img/e1b66d39812ce5170ba3347bf03ac945_8.png)

**为什么用平方而不是绝对值？**
使用平方项（而非绝对值）主要是为了数学上的便利。平方函数处处可导，这使得我们可以使用求导（梯度下降）的方法来寻找最小值。而绝对值函数在零点不可导，处理起来更为复杂。

**为什么乘以 1/2？**
这里的 `1/2` 是一个常数，其作用是为了后续求导时简化计算。对平方项求导会产生一个系数 `2`，`1/2` 与之相乘正好抵消，使得推导出的梯度公式更加简洁。

![](img/e1b66d39812ce5170ba3347bf03ac945_10.png)

## 梯度下降更新公式推导

![](img/e1b66d39812ce5170ba3347bf03ac945_12.png)

理解了损失函数后，我们回顾梯度下降的通用更新公式，并将其应用到线性回归的特定损失函数上。

梯度下降的通用公式是：
`θ_j^(t+1) = θ_j^(t) - α * (∂J(θ) / ∂θ_j)`

![](img/e1b66d39812ce5170ba3347bf03ac945_14.png)

**公式解读：**
*   `θ_j^(t)`：第 `t` 次迭代时，第 `j` 个参数的值。
*   `α`：学习率，控制每次更新的步长。
*   `∂J(θ) / ∂θ_j`：损失函数 `J(θ)` 对参数 `θ_j` 的偏导数（梯度）。

我们的目标就是求出线性回归损失函数 `J(θ)` 对某个参数 `θ_j` 的偏导数。

**推导过程：**

![](img/e1b66d39812ce5170ba3347bf03ac945_16.png)

1.  写出线性回归的预测函数（假设有 n 个特征）：
    `h_θ(x) = θ_0 * x_0 + θ_1 * x_1 + ... + θ_n * x_n`
    （通常令 `x_0 = 1`，以便将截距 `θ_0` 统一处理）

![](img/e1b66d39812ce5170ba3347bf03ac945_18.png)

2.  对损失函数 `J(θ)` 中的 `θ_j` 求偏导。这里运用链式求导法则：
    `∂J(θ)/∂θ_j = ∂ [1/2 * Σ (h_θ(x) - y)^2] / ∂θ_j`

![](img/e1b66d39812ce5170ba3347bf03ac945_20.png)

3.  将常数 `1/2` 和求和符号 `Σ` 提到求导运算外（求导是线性运算），先对平方项求导：
    `= 1/2 * Σ [2 * (h_θ(x) - y) * ∂(h_θ(x) - y)/∂θ_j]`

4.  简化系数 `2` 和 `1/2`：
    `= Σ [(h_θ(x) - y) * ∂(h_θ(x) - y)/∂θ_j]`

![](img/e1b66d39812ce5170ba3347bf03ac945_22.png)

5.  注意，`y` 是真实值，与参数 `θ_j` 无关，其导数为 0。`h_θ(x)` 对 `θ_j` 的偏导数就是 `x_j`（因为 `h_θ(x)` 是 `θ_j` 的线性函数）。因此：
    `∂(h_θ(x) - y)/∂θ_j = ∂(θ_j * x_j)/∂θ_j = x_j`

6.  最终得到梯度表达式：
    `∂J(θ)/∂θ_j = Σ [(h_θ(x^(i)) - y^(i)) * x_j^(i)]`

将求得的梯度代入通用更新公式，就得到了**线性回归的梯度下降更新公式**：

**核心更新公式：**
`θ_j^(t+1) = θ_j^(t) - α * Σ [(h_θ(x^(i)) - y^(i)) * x_j^(i)]`

**向量化表示（更高效的实现）：**
`θ^(t+1) = θ^(t) - α * (1/m) * X^T · (X · θ^(t) - y)`
其中 `X` 是特征矩阵，`y` 是标签向量，`m` 是样本数量。

## 公式符号总结与理解

为了避免混淆，我们来明确一下公式中关键符号的含义：

以下是关键符号的说明：
*   **`θ_j` / `w_j`**：模型第 `j` 个参数（权重）。`θ` 和 `w` 是同一概念的不同符号表示。
*   **`t` / `n`**：梯度下降的迭代次数（第 `t` 轮或第 `n` 轮更新）。
*   **`α` / `η`**：学习率，控制参数更新的步幅。
*   **`h_θ(x^(i))`**：用当前参数 `θ` 对第 `i` 个样本 `x^(i)` 做出的预测值。
*   **`x_j^(i)`**：第 `i` 个样本的第 `j` 个特征值。
*   **`Σ`**：对所有训练样本（`i` 从 1 到 `m`）求和。

**公式的直观理解：**
更新公式 `θ_j = θ_j - α * Σ(误差 * 特征值)` 非常直观。
*   **误差 (`h_θ(x) - y`)**：预测值与真实值的差距。误差越大，说明当前参数 `θ_j` 的“过错”越大，需要调整的幅度也越大。
*   **特征值 (`x_j`)**：该特征值的大小决定了参数 `θ_j` 对这次误差应承担多少“责任”。特征值越大，其对应的参数对误差的影响通常也越大。
*   **求和 (`Σ`)**：考虑所有样本的综合影响，而不是单个样本，使更新方向更稳定。
*   **学习率 (`α`)**：控制整体调整的幅度，避免步子太大（震荡）或太小（收敛慢）。

## 总结

本节课中我们一起学习了线性回归梯度下降更新公式的完整推导。
1.  我们从**最小二乘法**定义的损失函数出发，理解了衡量模型好坏的标准。
2.  然后，我们回顾了**梯度下降**的通用参数更新规则。
3.  接着，通过**求偏导**，我们将通用规则应用到线性回归的特定损失函数上，推导出了关键的更新公式：`θ_j = θ_j - α * Σ[(预测值-真实值) * 特征值]`。
4.  最后，我们明确了公式中每个符号的含义，并从直观上理解了该公式的工作逻辑：参数沿着减少所有样本上总误差的方向，按一定步长（学习率）进行迭代更新。
