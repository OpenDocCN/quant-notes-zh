# 机器学习：P106：损失函数立体化呈现（二）

## 概述
在本节课中，我们将学习如何将逻辑回归的损失函数进行可视化，包括二维等高线图和三维立体图。我们将看到损失函数如何随着模型参数（权重 w1 和 w2）的变化而变化，并理解为什么它是一个“凸函数”，从而可以通过梯度下降法找到最优解。

---

## 避免数值计算错误
上一节我们介绍了损失函数的基本计算。本节中我们来看看如何避免计算中出现的数值问题。

在计算概率时，如果概率值过小，可能导致对数运算报错（如出现 `-inf`）。为了解决这个问题，我们对计算出的概率值 `p` 进行裁剪，确保其不小于一个极小值（如 `1e-6`）。

```python
p = np.clip(p, 1e-6, 1 - 1e-6)
```

经过这样的裁剪处理，计算出的值就不会无限小，从而避免了数值错误。

---

## 数据准备与单参数可视化
数据准备好后，我们开始进行可视化。首先，我们可视化损失函数随单个权重 `w1` 变化的情况。

以下是绘制 `w1` 对应损失函数的步骤：
1.  创建一个图形窗口，并设置其尺寸。
2.  在图形中添加一个子视图。
3.  使用 `plt.plot` 绘制 `w1` 与对应损失值的折线图。

```python
import matplotlib.pyplot as plt
plt.figure(figsize=(12, 9))
plt.subplot(2, 2, 1)
plt.plot(w1_space, loss1, color='green')
plt.title('w1')
```

执行代码后，我们可以看到绘制出的损失函数曲线。这条曲线是一个“下凸函数”（或凸函数），这意味着它存在一个最低点（最优解），我们可以通过梯度下降法逐步找到这个解。

---

## 可视化第二个参数
同样，我们可以可视化损失函数随第二个权重 `w2` 变化的情况。

绘制 `w2` 对应损失函数的步骤与 `w1` 类似：
1.  在图形中添加第二个子视图。
2.  绘制 `w2` 与对应损失值的折线图。

```python
plt.subplot(2, 2, 2)
plt.plot(w2_space, loss2, color='red')
plt.title('w2')
```

注意，`w1` 和 `w2` 的取值范围可能不同。有时曲线中会出现一段近乎水平的区域，这是因为在计算过程中，概率值可能趋近于零，导致对数函数变化平缓。为了使图形更美观，我们可以调整 `w1_space` 和 `w2_space` 的生成范围。

---

## 双参数联合可视化：等高线图
上一节我们分别看了单个参数的影响，本节中我们来看看如何将两个参数 `w1` 和 `w2` 联合起来可视化。

我们想要绘制一个图形，其横坐标为 `w1`，纵坐标为 `w2`，颜色深浅表示损失值的大小，这种图形称为“等高线图”。

首先，我们需要创建 `w1` 和 `w2` 的网格数据。这可以使用 `np.meshgrid` 函数完成。

```python
import numpy as np
w1_grid, w2_grid = np.meshgrid(w1_space, w2_space)
```

`np.meshgrid` 函数接收两个一维数组，并返回两个二维数组，它们代表了所有 `(w1, w2)` 的组合点。例如，输入 `[1,2,3]` 和 `[2,4,6,8]`，它会生成一个网格，覆盖所有 `(1,2), (1,4), ... (3,8)` 的点。

接着，我们计算网格上每个点对应的损失值，这会得到一个损失值矩阵。

```python
loss_grid = loss_function(x, y, w1_grid, w2_grid, b)
```

最后，使用 `plt.contour` 函数绘制等高线。

```python
plt.subplot(2, 2, 3)
plt.contour(w1_grid, w2_grid, loss_grid, colors='purple')
```

为了使图形更规整（呈现标准的同心圆状），我们通常需要对输入特征 `x` 进行标准化（如 Z-score 归一化），并暂时忽略截距 `b` 的影响，专注于研究权重 `w1` 和 `w2` 的规律。

```python
from sklearn.preprocessing import StandardScaler
x = StandardScaler().fit_transform(x) # Z-score 归一化
b = 0 # 忽略截距，专注于 w1, w2
```

调整参数范围并重新计算后，我们就能得到清晰的、一圈圈的等高线图，越靠近中心损失值越低。

## 三维立体可视化
最后，我们将损失函数以三维形式呈现，这能让我们更直观地看到其“山谷”形状。

以下是创建三维损失函数曲面图的步骤：
1.  创建一个新的图形窗口。
2.  指定子图为 3D 投影。
3.  使用 `plot_surface` 函数绘制曲面。
4.  设置坐标轴标签和视角。

```python
fig = plt.figure(figsize=(12, 9))
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(w1_grid, w2_grid, loss_grid, cmap='rainbow')
ax.set_xlabel('w1')
ax.set_ylabel('w2')
ax.view_init(30, -30) # 调整视角
plt.show()
```

从三维图中可以清晰地看到损失函数像一个“碗”或“山谷”，中心最低点即为最优解。这直观地证明了梯度下降法可以沿着最陡峭的路径“下山”，最终找到损失最小的参数组合。

---

## 总结
本节课中我们一起学习了逻辑回归损失函数的多种可视化方法：
1.  **单参数曲线**：展示了损失函数随单个权重变化的情况，确认其为凸函数。
2.  **双参数等高线/面图**：展示了损失值在 `w1`-`w2` 平面上的分布，中心为最优解。
3.  **三维立体图**：直观呈现了损失函数的“山谷”形状。

![](img/da007b3632e63516b1dd7f7f110d51ff_9.png)

通过这些可视化，我们不仅理解了损失函数的形态，也直观地看到了梯度下降法工作的原理——从高处（高损失）沿着梯度方向逐步走向谷底（低损失，最优解）。同时，我们也认识到数据标准化（归一化）对于模型训练和可视化效果的重要性。
