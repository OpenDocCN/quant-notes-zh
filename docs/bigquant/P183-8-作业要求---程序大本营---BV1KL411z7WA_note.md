## 代码实现步骤

以下是实现基于基尼系数的决策树模型的关键步骤：

![](img/29d25fab179f1a27cdc81b3dd462951c_9.png)

1.  **导入模型**：从 `sklearn.tree` 导入 `DecisionTreeClassifier`。
2.  **实例化模型**：创建模型对象时，指定参数 `criterion='gini'`。这是将分裂条件设置为基尼系数的关键。
3.  **训练模型**：使用特征数据 `X` 和标签数据 `y` 来训练模型。注意，如果数据集中包含目标列，需要进行切片操作将其分离。

![](img/29d25fab179f1a27cdc81b3dd462951c_11.png)

示例代码框架如下：
```python
from sklearn.tree import DecisionTreeClassifier

# 假设 X 是特征数据，y 是标签数据
# 如果数据集中 y 是最后一列，可能需要切片：X = data[:, :-1]

![](img/29d25fab179f1a27cdc81b3dd462951c_13.png)

model = DecisionTreeClassifier(criterion='gini') # 指定使用基尼系数
model.fit(X, y)
```
执行上述代码后，模型将默认使用基尼系数进行节点分裂。

## 理解基尼系数计算

![](img/29d25fab179f1a27cdc81b3dd462951c_17.png)

模型运行后，你可能会看到类似 `0.42` 或 `0.375` 的基尼系数值。为了深入理解决策树的构建过程，我们需要知道这些数值是如何计算出来的。

![](img/29d25fab179f1a27cdc81b3dd462951c_19.png)

课堂上我们已经学习了信息熵的计算公式。现在，我们来看看基尼系数的计算公式。

基尼系数的计算公式为：
**Gini = 1 - Σ(pi)²**
其中，**pi** 代表数据集中第 **i** 个类别出现的概率。

![](img/29d25fab179f1a27cdc81b3dd462951c_21.png)

这个公式的含义是：从数据集中随机抽取两个样本，其类别标签不一致的概率。基尼系数越小，表示数据集的纯度越高。

![](img/29d25fab179f1a27cdc81b3dd462951c_23.png)

为了彻底理解决策树的构造逻辑，你需要能够手动计算出类似 `0.42` 和 `0.375` 这样的基尼系数值。这是本次作业的核心挑战。
