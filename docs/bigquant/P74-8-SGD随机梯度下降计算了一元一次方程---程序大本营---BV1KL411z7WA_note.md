## 数据准备与问题回顾

![](img/cd6cffd0215f78d4b0cabd3ea439d9a6_5.png)

上一节我们介绍了批量梯度下降，本节中我们来看看随机梯度下降（SGD）的实现。首先，我们需要创建用于训练的数据。我们以一个简单的一元一次方程 `y = w*x + b` 为例，其中 `w` 是斜率，`b` 是截距。

以下是数据创建和导包的代码：

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置随机种子以保证结果可复现
np.random.seed(42)
# 生成100个在[0, 2]区间内的随机数作为特征x
x = 2 * np.random.rand(100, 1)
# 设定真实斜率为4，真实截距为6，并加入一些随机噪声
y = 4 + 6 * x + np.random.randn(100, 1)
```

为了在方程中引入偏置项（截距b），我们需要对特征数据x进行扩展，增加一列全为1的数据。

```python
# 为x增加一列全为1的偏置项
x = np.concatenate([x, np.full_like(x, 1)], axis=1)
```
执行此操作后，`x` 的形状变为 `(100, 2)`，其中第一列是原始特征，第二列全是1，用于与截距b相乘。

## 随机梯度下降（SGD）算法实现

与批量梯度下降使用全部数据计算梯度不同，随机梯度下降在每次迭代中**随机选取一个样本**来计算梯度并更新参数。这种方法计算速度快，但更新路径较为曲折。

以下是实现SGD所需的核心步骤和参数初始化：

```python
# 设置训练轮次（epoch）
n_epochs = 100
# 初始化参数theta，包含斜率和截距，形状为(2, 1)
theta = np.random.randn(2, 1)
# 定义一个计数器，用于调整学习率
count = 0

# 定义一个学习率调度函数，随着训练进行逐渐减小学习率
def learning_rate_schedule(t):
    t0, t1 = 5, 50
    return t0 / (t + t1)
```

接下来是SGD的核心训练循环。在每一轮（epoch）训练中，我们首先打乱数据的顺序，然后逐个样本进行参数更新。

```python
for epoch in range(n_epochs):
    # 生成一个索引数组并打乱顺序，实现数据的随机化
    index = np.arange(100)
    np.random.shuffle(index)
    # 根据打乱的索引重新排列数据
    x_shuffled = x[index]
    y_shuffled = y[index]

    # 遍历打乱后的每一个样本
    for i in range(100):
        # 随机抽取一个样本
        xi = x_shuffled[i:i+1]  # 保持xi为二维数组，形状为(1, 2)
        yi = y_shuffled[i:i+1]  # 保持yi为二维数组，形状为(1, 1)

        # 计算当前样本的梯度
        gradients = xi.T.dot(xi.dot(theta) - yi)
        # 获取当前迭代的学习率
        eta = learning_rate_schedule(count)
        count += 1
        # 更新参数theta
        theta -= eta * gradients
```

在代码中，`xi` 和 `yi` 被特意保持为二维数组（例如形状 `(1, 2)` 和 `(1, 1)`），这是为了确保后续的矩阵点乘运算 `xi.T.dot(...)` 能够正确执行。如果直接取一维数组，会破坏矩阵运算的维度要求，导致错误。

## 核心概念总结与对比

本节课中我们一起学习了随机梯度下降（SGD）的实现。我们来总结几个关键点：

1.  **核心思想**：SGD在每次参数更新时，只使用**一个随机样本**来计算梯度。其参数更新公式为：
    `theta = theta - eta * gradient_i`
    其中 `gradient_i` 是单个样本计算出的梯度。

2.  **数据随机化**：在每轮训练开始时，使用 `np.random.shuffle()` 打乱数据顺序至关重要，这确保了样本被随机抽取，避免了循环可能带来的固有顺序偏差。

3.  **维度一致性**：在实现时，要特别注意数据的维度。单个样本 `xi` 应保持为二维数组（如 `shape=(1, n_features)`），以兼容矩阵运算，这是实现中一个常见的易错点。

4.  **与批量梯度下降的对比**：
    *   **批量梯度下降（BGD）**：使用全部数据计算梯度，更新稳定，方向准确，但每次计算开销大。
    *   **随机梯度下降（SGD）**：使用单个样本，计算速度快，能跳出局部极小值，但更新波动大，收敛路径曲折。

![](img/cd6cffd0215f78d4b0cabd3ea439d9a6_11.png)

随机梯度下降因其高效性，特别适用于大规模数据集。虽然本例用于简单的一元方程，但其原理是理解现代深度学习优化器的基础。通过调整学习率调度和迭代轮次，可以使其更好地收敛到最优解。
