## 构建SVM的数学问题

上一节我们介绍了SVM寻找最大间隔分类器的直观思想。本节中，我们来看看如何将这一思想转化为一个具体的数学优化问题。

与线性回归使用最小二乘法、逻辑斯蒂回归使用Sigmoid函数类似，解决任何机器学习问题都需要一个明确的目标函数。支持向量机的目标，就是找到一个最优的分类超平面。

## 从分类线到数学方程

假设我们最终要找的分类线（决策边界）是 **l₂**，其两侧的边界线分别是 **l₁** 和 **l₃**。在二维空间中，一条线可以用方程表示。我们要求解的就是这个方程的系数。

我们可以将超平面（即分类线）的方程表示为：
`wᵀx + b = 0`
其中，**w** 是法向量（决定了直线的方向），**b** 是截距。我们的目标就是确定 **w** 和 **b**。

由于 **l₁** 和 **l₃** 是 **l₂** 的决策边界，它们必须与 **l₂** 平行。平行意味着斜率（即 **w**）相同，只有截距 **b** 不同。因此，我们可以假设：
*   **l₁** 的方程：`wᵀx + b = -c`
*   **l₂** 的方程：`wᵀx + b = 0`
*   **l₃** 的方程：`wᵀx + b = c`

这里 **c** 是一个正常数。**l₁** 和 **l₃** 相对于 **l₂** 进行等距离偏移，以确保算法对两边的样本“一视同仁”，间隔最大。

## 距离公式回顾

![](img/87d58186d865bd4eb0b958b6580274d9_7.png)

在推导目标函数前，我们需要回顾两个基础的距离公式。

![](img/87d58186d865bd4eb0b958b6580274d9_9.png)

**点到直线的距离公式：**
在二维空间中，点 `(x₁, y₁)` 到直线 `Ax + By + C = 0` 的距离 **d** 为：
`d = |Ax₁ + By₁ + C| / √(A² + B²)`

**两条平行线间的距离公式：**
对于两条平行线 `Ax + By + C₁ = 0` 和 `Ax + By + C₂ = 0`，它们之间的距离 **d** 为：
`d = |C₁ - C₂| / √(A² + B²)`

![](img/87d58186d865bd4eb0b958b6580274d9_11.png)

当我们将问题扩展到 **n** 维空间时，点变为 `x = (x₁, x₂, ..., xₙ)`，平面方程变为 `wᵀx + b = 0`。距离公式在形式上保持一致，分母中的 `√(A² + B²)` 变为向量 **w** 的模（或称L2范数）`||w||`。

![](img/87d58186d865bd4eb0b958b6580274d9_13.png)

![](img/87d58186d865bd4eb0b958b6580274d9_15.png)

## 间隔最大化与目标函数构建

我们希望决策边界上的样本点（即“支持向量”）到分类超平面 **l₂** 的距离 **d** 越大越好。这个距离 **d** 恰好等于两条平行边界线 **l₁** 和 **l₃** 之间的距离。

根据平行线距离公式，**l₁** 与 **l₂** 或 **l₃** 与 **l₂** 之间的距离 **d** 为：
`d = |c - 0| / ||w|| = c / ||w||`

为了简化计算，我们可以对边界线方程进行缩放。令方程两边同时除以 **c**，我们得到：
*   **l₃**（上方边界）变为：`(wᵀ/c)x + (b/c) = 1`，我们重新记 `wᵀ/c` 为新的 **wᵀ**，`b/c` 为新的 **b**，则方程为 `wᵀx + b = 1`
*   **l₁**（下方边界）同理变为：`wᵀx + b = -1`

经过缩放后，两条边界线间的距离 **d** 简化为：
`d = |1 - (-1)| / ||w|| = 2 / ||w||`

我们希望最大化这个间隔 **d**，即最大化 `2 / ||w||`。这等价于最小化 `||w||`。为了后续求导方便，我们通常最小化 `(1/2)||w||²`，其单调性与最小化 `||w||` 一致。

## 引入分类约束条件

我们还需要确保所有样本点被正确分类。我们将 **l₃** 上方的点标记为类别 **+1**，**l₁** 下方的点标记为类别 **-1**。那么，对于任意样本点 `(xᵢ, yᵢ)`，应满足：
*   若 `yᵢ = +1`，则 `wᵀxᵢ + b ≥ 1`
*   若 `yᵢ = -1`，则 `wᵀxᵢ + b ≤ -1`

我们可以将这两个约束合并为一个简洁的不等式：
`yᵢ (wᵀxᵢ + b) ≥ 1`，对于所有样本 `i = 1, 2, ..., n`

## SVM的最终目标函数

综合以上分析，支持向量机的目标可以表述为：**在满足所有样本点分类正确的约束下，找到使得分类间隔最大的超平面参数**。

![](img/87d58186d865bd4eb0b958b6580274d9_17.png)

![](img/87d58186d865bd4eb0b958b6580274d9_19.png)

这转化成了一个数学优化问题——**带不等式约束的二次规划问题**。其标准形式如下：

**最小化：** `(1/2) ||w||²`
**约束条件（subject to）：** `yᵢ (wᵀxᵢ + b) ≥ 1`， 对于 `i = 1, 2, ..., n`

其中，`s.t.` 是“subject to”的缩写，意为“满足以下条件”。

## 总结

![](img/87d58186d865bd4eb0b958b6580274d9_21.png)

本节课中，我们一起学习了支持向量机核心目标函数的推导过程。我们从寻找最大间隔的几何直觉出发，通过定义超平面方程、回顾距离公式、引入缩放技巧和分类约束，最终得到了一个清晰的最优化问题：**最小化 `(1/2)||w||²`，并满足 `yᵢ(wᵀxᵢ + b) ≥ 1` 的约束**。这个目标函数完美地封装了SVM“最大间隔”和“正确分类”的核心思想。在接下来的课程中，我们将探讨如何求解这个带约束的优化问题。
