## 问题引入：寻找曲线上的最近点

上一节我们介绍了优化问题的基本概念，本节中我们来看看一个具体的几何问题。

![](img/eb6c61ae90704a619cbb914724478483_7.png)

我们有一个方程：`x² + y = 3`。这个方程对应的图像是一条曲线。

我们的目标是：找到这条曲线上距离原点 `(0, 0)` 最近的点。换句话说，就是求函数 `f(x, y) = x² + y²`（原点到点 `(x, y)` 距离的平方）在约束条件 `g(x, y) = x² + y - 3 = 0` 下的最小值。

## 几何直观：等高线与相切

为了解决这个问题，我们引入一种几何思路。

想象我们以原点为圆心，画一个半径为 `r` 的圆。圆上所有的点到原点的距离都是 `r`。如果我们逐渐增大半径 `r`，这个圆会逐渐扩大。

关键观察是：当这个不断扩大的圆**第一次**与我们的曲线 `x² + y = 3` 接触（即相切）时，这个切点就是曲线上离原点最近的点。此时圆的半径 `r` 就是最短距离。

至此，我们得到了第一个重要结论：**在极值点（最近点），目标函数 `f` 的等高线（圆）与约束曲线 `g` 相切。**

## 梯度的性质：等高线的法线

为了将相切这个几何条件转化为代数方程，我们需要用到梯度的概念。

对于函数 `f(x, y) = x² + y²`，其梯度为：
`∇f = (∂f/∂x, ∂f/∂y) = (2x, 2y)`

梯度有一个重要性质：**函数在某一点的梯度向量，垂直于该点函数等高线的切线方向**。也就是说，梯度是等高线的法线方向。

同样，对于约束函数 `g(x, y) = x² + y - 3`，其梯度为：
`∇g = (∂g/∂x, ∂g/∂y) = (2x, 1)`

在曲线 `g=0` 上任意一点，`∇g` 也垂直于该曲线的切线。

## 核心推导：梯度向量平行

现在回到我们的相切点。在这一点，圆 `f` 的等高线与曲线 `g` 相切，意味着它们拥有**同一条切线**。

由于梯度方向垂直于切线，那么在相切点，圆 `f` 的梯度向量 `∇f` 和曲线 `g` 的梯度向量 `∇g` 必然都垂直于同一条切线。因此，这两个梯度向量在这一点是**平行的**。

两个向量平行，意味着其中一个向量是另一个的标量倍。我们可以用数学公式表达为：
`∇f = λ * ∇g`
其中 `λ` 是一个标量系数，称为拉格朗日乘子。

同时，点必须满足约束条件：
`g(x, y) = x² + y - 3 = 0`

## 构建拉格朗日函数

![](img/eb6c61ae90704a619cbb914724478483_9.png)

我们将上面的条件结合起来：

![](img/eb6c61ae90704a619cbb914724478483_11.png)

1.  平行条件：`∇f = λ * ∇g`
2.  约束条件：`g(x, y) = 0`

对于我们的例子，即：
`(2x, 2y) = λ * (2x, 1)` 和 `x² + y = 3`

这构成了一个方程组。我们可以通过求解这个方程组找到极值点 `(x, y)` 和对应的乘子 `λ`。

拉格朗日乘子法的巧妙之处在于，它将上述方程组等价地转化为对一个新函数求无约束极值的问题。

我们构造一个**拉格朗日函数**：
`L(x, y, λ) = f(x, y) + λ * g(x, y)`
代入具体函数：
`L(x, y, λ) = (x² + y²) + λ * (x² + y - 3)`

求解原约束优化问题，等价于对拉格朗日函数 `L` 求所有变量（包括 `λ`）的偏导数，并令其为零：

![](img/eb6c61ae90704a619cbb914724478483_13.png)

以下是需要求解的方程组：
`∂L/∂x = 2x + λ * 2x = 0`
`∂L/∂y = 2y + λ * 1 = 0`
`∂L/∂λ = x² + y - 3 = 0`

解这个方程组，即可得到候选极值点。可以验证，这个方程组展开后，与前文根据“梯度平行”列出的方程组完全一致。

![](img/eb6c61ae90704a619cbb914724478483_15.png)

## 方法总结：拉格朗日乘子法

现在，我们可以给出拉格朗日乘子法的一般形式。

对于一个等式约束优化问题：
最小化 `f(x)`
约束条件为 `h(x) = 0`

其求解步骤如下：

1.  构造拉格朗日函数：
    `L(x, λ) = f(x) + λ * h(x)`
    这里引入了一个新变量 `λ`（拉格朗日乘子）。

![](img/eb6c61ae90704a619cbb914724478483_17.png)

2.  对拉格朗日函数的所有变量（原变量 `x` 和乘子 `λ`）求偏导数，并令其为零：
    `∇_x L(x, λ) = 0`
    `∇_λ L(x, λ) = h(x) = 0`

![](img/eb6c61ae90704a619cbb914724478483_19.png)

3.  求解上述方程组，得到的解 `(x*, λ*)` 即为原约束优化问题的候选极值点。

**为什么可以这样做？**
因为 `∇_x L = 0` 等价于 `∇f + λ∇h = 0`，即 `∇f = -λ∇h`，这正是“目标函数梯度与约束函数梯度平行”的条件。而 `∇_λ L = 0` 等价于 `h(x) = 0`，即约束条件本身。因此，求解拉格朗日函数的驻点，完美地封装了极值点的两个必要条件。

本节课中我们一起学习了拉格朗日乘子法的原理推导。

![](img/eb6c61ae90704a619cbb914724478483_21.png)

![](img/eb6c61ae90704a619cbb914724478483_23.png)

我们从“求曲线上距离原点最近的点”这个几何问题出发，通过分析等高线相切的条件，得出了“在极值点，目标函数与约束函数的梯度向量平行”这一关键结论。随后，我们展示了如何通过引入拉格朗日乘子 `λ`，并构造拉格朗日函数 `L = f + λg`，将这一几何条件转化为一个易于求解的无约束优化问题（求 `L` 的驻点）。

![](img/eb6c61ae90704a619cbb914724478483_25.png)

拉格朗日乘子法的核心思想是：**通过引入额外的变量（乘子），将约束条件吸收到目标函数中，从而把复杂的约束优化问题转化为相对简单的无约束问题**。理解这一推导过程，有助于我们更深刻地掌握这一在机器学习中至关重要的数学工具。
