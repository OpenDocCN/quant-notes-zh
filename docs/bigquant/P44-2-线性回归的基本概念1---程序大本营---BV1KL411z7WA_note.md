## 有监督学习与回归问题

上一节我们介绍了机器学习的基本范畴，本节中我们来看看什么是有监督学习。

有监督学习是指使用带有标签的历史数据进行训练，让模型从数据中学习规律。这就像学生在考试后，老师会提供标准答案，学生通过对比答案来纠正自己的错误，从而在后续的考试中提高分数。有监督学习就是“以史为鉴”，从已知结果的数据中学习预测未来。

回归问题是机器学习中的一类核心问题。它主要关注**因变量**（需要预测的值）与一个或多个**自变量**（影响因素）之间的关系。万事万物之间都存在联系，我们的目标就是使用数学方程来量化这种关系。

以下是关于回归问题中关键术语的解释：
*   **因变量**：需要预测的值，通常用 `y` 或 `target` 表示。例如，明天的股票价格。
*   **自变量**：影响因变量的因素，通常用 `x1, x2, ..., xn` 表示。例如，影响股票价格的政策、经济数据、国际形势等。

线性回归算法就是研究这些变量之间的关系，并将其数字化、公式化。

## 目标变量与特征

在机器学习中，我们将需要预测的值称为**目标变量**。这个值通常是连续的。

**连续值**是指数值可以无限细分，在某个区间内可以取任意值。例如：
*   身高：可以是 175.0 cm，也可以是 175.23 cm。
*   体重：可以是 65.0 kg，也可以是 65.158 kg。
*   智商、情商等指标，理论上也可以精确到小数。

与连续值相对的是**离散值**。

**离散值**是泾渭分明、类别明确的数值。例如：
*   省份：新疆、北京、河北等，没有“0.5个新疆”。
*   性别：男、女。
*   兵种：海、陆、空。

理解了数据的类型后，我们就可以开始构建模型来描述它们之间的关系。

## 简单线性回归模型

现在，我们来看最简单的回归模型——简单线性回归。

简单线性回归对应的公式是一个一元一次方程：
`y = w * x + b`

在这个公式中：
*   `y` 是**目标变量**，即未来要预测的值。
*   `x` 是影响 `y` 的**自变量**（特征）。
*   `w` 和 `b` 是公式的**参数**，即我们需要通过数据求解的模型。`w` 是斜率，`b` 是截距。

例如，海拔 (`x`) 和温度 (`y`) 在一定范围内就近似存在线性关系（如海拔每升高100米，气温下降约0.5℃）。如果我们能求出 `w` 和 `b`，就能用这个方程预测任何海拔下的温度。

因为这个模型只涉及一个自变量 `x`，且 `x` 的幂次为1（线性），所以我们称之为**简单线性回归**。

## 为什么叫“线性”？

为了直观理解“线性”的含义，我们可以通过代码绘图来观察。

```python
plt.plot(x_linear, y_linear, color='red')
plt.title('Linear Function: y = 0.8x - 5')
plt.show()

![](img/aef17a858b016fc7b3ca8e54794d5817_11.png)

# 创建二次函数（非线性）的图像
x_curve = np.linspace(0, 10, 50)
y_curve = x_curve**2 + 4*x_curve - 15  # x**2 表示x的平方
plt.plot(x_curve, y_curve)
plt.title('Non-linear Function: y = x^2 + 4x - 15')
plt.show()
```

执行代码后，你会发现一次函数的图像是一条**直线**，而二次函数的图像是一条**曲线**（抛物线）。线性回归研究的就是这种变量间呈一次方（直线）关系的问题。

## 方程、算法与最优解

你可能会有疑问：这看起来不就是初中学过的一元一次方程吗？为什么现在叫“机器学习算法”？

两者的核心确实都是方程 `y = w*x + b`。关键区别在于求解的目标：
*   **数学方程**：追求**唯一解**。给定确定的 `x` 和 `y`，解出的 `w` 和 `b` 是精确、唯一的。
*   **机器学习算法**：追求**最优解**。我们拥有大量包含噪声的真实数据，目标是找到一组 `w` 和 `b`，使得方程的计算结果最接近所有真实数据点，即“合情合理”、“堪用”的解。

这就引出了线性回归的核心数学原理——**最小二乘法**。历史上，高斯正是用这个方法计算出了天体的运行轨道。线性回归的求解就是基于最小二乘法推导而来的。

3.  区分了**连续值**与**离散值**这两种数据类型。
4.  掌握了**简单线性回归**的核心公式：`y = w*x + b`，并了解了其中参数的含义。

6.  认识到机器学习中的“算法”求解的是基于真实数据的**最优解**，而非数学上的唯一解。
