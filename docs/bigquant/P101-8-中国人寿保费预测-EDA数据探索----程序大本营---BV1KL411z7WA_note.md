## 数据标准化的重要性

![](img/be33e16a7924548f37e7cb0f98a5162c_5.png)

![](img/be33e16a7924548f37e7cb0f98a5162c_7.png)

上一节我们介绍了模型训练的基本流程，本节中我们来看看数据预处理中的一个关键步骤：标准化。

![](img/be33e16a7924548f37e7cb0f98a5162c_9.png)

![](img/be33e16a7924548f37e7cb0f98a5162c_11.png)

在机器学习中，特别是使用随机梯度下降等优化算法时，对数据进行标准化或归一化处理是必要的。这是因为不同特征的数据量纲和数量级可能差异巨大。

![](img/be33e16a7924548f37e7cb0f98a5162c_13.png)

![](img/be33e16a7924548f37e7cb0f98a5162c_15.png)

![](img/be33e16a7924548f37e7cb0f98a5162c_17.png)

例如，在我们的数据中，某些特征可能是两位数，而另一些特征（如保费）可能是四位数。如果不进行标准化，数量级大的特征会主导模型的训练过程，导致求解结果不准确甚至完全错误。

![](img/be33e16a7924548f37e7cb0f98a5162c_19.png)

**核心公式**：标准化通常使用 `(x - mean) / std` 进行计算，将数据转换为均值为0、标准差为1的分布。

因此，在使用随机梯度下降等算法前，必须进行相应的规划（标准化）操作。

## 代码实战：加载与观察数据

现在，我们开始进行中国人寿保费预测的EDA部分。首先需要加载数据并观察其结构。

以下是加载数据所需的代码：

```python
import pandas as pd
import seaborn as sns

# 加载数据
data = pd.read_excel(‘中国人寿保费数据.xlsx’)
# 查看前几条数据
print(data.head())
```

执行代码后，我们可以看到数据集包含以下列：年龄、性别、体重指数、子女数量、是否吸烟、所在地区以及最后一列“charges”（保费）。我们的目标是分析前面这些特征如何影响最终的保费。

## 通过可视化探索特征影响力

为了判断一个特征是否对目标变量（保费）有重要影响，我们可以绘制其概率密度分布图进行观察。一个好的特征应该在不同类别间展现出明显的分布差异。

以下是绘制概率密度图的通用代码模板：

```python
sns.kdeplot(data=数据, x=‘目标列’, hue=‘特征列’, shade=True)
```

### 分析性别对保费的影响

我们首先分析“性别”特征。通过绘制男女保费的分布曲线，可以发现两条曲线高度重合，吻合度约95%。这表明在保险定价中，性别差异带来的影响微乎其微，可以认为该特征区分度不大。

### 分析吸烟对保费的影响

接下来，我们分析“是否吸烟”这个特征。图像显示，吸烟者与不吸烟者的保费分布曲线差异非常显著。吸烟者的保费普遍更高。这是一个“好特征”，因为它具有很高的区分度，能明确影响保费定价。

### 分析地区对保费的影响

然后，我们查看“地区”特征。按照东南、西南、西北、东北划分后，不同地区的保费分布曲线同样高度相似。这说明在当前粗略的区域划分下，地区对保费的影响也不明显，该特征的重要性较低。

### 分析子女数量对保费的影响

![](img/be33e16a7924548f37e7cb0f98a5162c_21.png)

![](img/be33e16a7924548f37e7cb0f98a5162c_23.png)

最后，我们观察“子女数量”特征。图像显示，拥有不同数量子女的家庭，其保费分布存在一定差异。这个特征具有一定的区分度，建议在后续建模中保留。

![](img/be33e16a7924548f37e7cb0f98a5162c_25.png)

![](img/be33e16a7924548f37e7cb0f98a5162c_27.png)

通过以上可视化分析，我们可以初步判断：
*   **区分度低、可考虑删除的特征**：性别、地区。
*   **区分度高、应保留的特征**：是否吸烟、子女数量。
*   **需进一步分析的特征**：年龄、体重指数等。
