# 量化因子构建教程：P26：4.2.1.3：利用表达式引擎构建因子举例2 🧮

在本节课中，我们将学习如何通过微矿平台的金融数据表来构建一个具体的量化因子。我们将以构建“过去五日收益率”因子为例，详细介绍从数据范围指定、表达式编写到最终结果计算的全过程。

## 概述

我们将通过四个核心模块的协作来完成因子构建：首先指定股票和时间范围，然后定义因子计算表达式，接着从数据源提取所需的基础数据列，最后通过表达式引擎计算出因子值。整个过程清晰展示了平台化工具在因子研究中的高效应用。

## 构建流程详解

以下是构建因子的四个核心步骤。

### 1. 指定数据与股票范围

首先，我们需要确定计算因子所使用的数据范围。这包括时间范围和股票池。

*   **中心代码列表模块**：此模块用于指定股票代码范围和时间范围。如果在股票代码列表中留空，系统将默认获取指定时间范围内的所有股票数据。

### 2. 定义因子表达式

![](img/6450a14fc44d63bb256f2592157e60e8_1.png)

接下来，我们需要明确要计算的因子是什么，并用表达式将其描述出来。

*   **输入特征列表模块**：在此模块中，我们输入因子计算表达式。例如，要计算过去五日的收益率，其表达式为 `close / shift(close, 5)`。我们可以使用等号（`=`）为这个较长的表达式设置一个简短的别名，例如 `right_5 = close / shift(close, 5)`。这样，在后续调用时，我们只需使用 `right_5` 即可。

### 3. 提取基础数据

定义了表达式后，我们需要从数据库中提取表达式中所需要的基础数据列。

*   **数据源模块**：此模块用于连接并指定具体的数据表。我们需要将其设置为包含股票日线行情的数据表（例如 `bd_cn_stock_a`）。运行此模块后，它会根据前序模块设定的范围，提取出所有股票的收盘价（`close`）数据列。

### 4. 计算衍生因子

![](img/6450a14fc44d63bb256f2592157e60e8_3.png)

最后，我们利用提取到的基础数据和定义好的表达式，计算出最终的因子值。

*   **衍生特征抽取模块**：此模块是表达式引擎的核心。它接收从数据源模块提取的 `close` 数据，并根据我们在“输入特征列表”中定义的表达式（`right_5 = close / shift(close, 5)`）进行计算。运行此模块后，将生成名为 `right_5` 的新列，这就是我们构建的“过去五日收益率”因子。

## 操作案例演示

![](img/6450a14fc44d63bb256f2592157e60e8_5.png)

让我们通过一个实际案例来直观感受整个操作流程。

首先，在画布中依次拖入并连接上述四个模块：`中心代码列表` -> `输入特征列表` -> `数据源` -> `衍生特征抽取`。

![](img/6450a14fc44d63bb256f2592157e60e8_7.png)

**第一步：设置参数**
在`中心代码列表`模块中，设置好时间范围，股票代码列表留空以选择全市场股票。在`输入特征列表`模块中，输入表达式：`right_5 = close / shift(close, 5)`。

**第二步：提取数据**
在`数据源`模块中，选择股票日线行情数据表（如 `bd_cn_stock_a`）。右键点击该模块并选择“运行”，运行成功后，可以查看结果，确认已正确提取出 `close` 价格序列。

**第三步：计算因子**
运行`衍生特征抽取`模块。计算完成后，查看结果表，会发现多出一个名为 `right_5` 的列。点击该列，可以查看其详细的统计信息，包括缺失值数量、唯一值数量、平均值、最大值等，这有助于我们初步评估因子的质量。
