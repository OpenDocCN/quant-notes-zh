## 平台自动标注模块介绍

上一节我们介绍了数据获取的基础知识，本节中我们来看看如何为数据添加标签。平台提供了“自动标注股票”模块，可以快速为股票数据生成标注。

通过连接“证券代码列表”模块和“自动标注股票”模块，可以实现股票数据的快速标注。在“自动标注股票”模块中，平台提供了如`close`、`open`等基础字段用于构建表达式。

例如，要构建股票的未来五日收益率作为标注，可以使用以下表达式：
```
(shift(close, -5) / shift(open, -1)) - 1
```
计算后的结果将作为`label`列返回。

## 标注数据的后处理

得到原始的`label`列后，我们通常需要对其进行一系列处理，例如去极值、映射分类以及过滤。

以下是处理`label`列的三个典型步骤：

![](img/73f66e7a63c1f487f2e986aa141333ab_2.png)

1.  **去极值处理**：首先，我们使用`clip`函数对`label`列进行极值处理。具体方法是先计算`label`数据列的1%分位数和99%分位数，然后利用`clip`函数去除小于1%分位数和大于99%分位数的数据。
    ```
    label_clipped = clip(label, quantile(label, 0.01), quantile(label, 0.99))
    ```

2.  **映射分类**：接着，我们使用`qcut`函数将去极值处理后的`label`数据进行映射分类。例如，这里我们将其分成20个类别。
    ```
    label_binned = qcut(label_clipped, 20, labels=False)
    ```

3.  **条件过滤**：最后，我们通过`where`函数指定过滤条件。例如，通过条件`shift(high, -1) == shift(low, -1)`，将一字涨停或一字跌停的股票的`label`改写为`NaN`值，其他股票数据则保持`label`不变。
    ```
    label_final = where(shift(high, -1) != shift(low, -1), label_binned, NaN)
    ```
