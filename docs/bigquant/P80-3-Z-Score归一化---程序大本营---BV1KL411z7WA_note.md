## 概述

在机器学习中，我们处理的数据往往来自不同的来源，具有不同的单位和量纲。例如，年龄以“岁”为单位，而薪水以“元”为单位。这种差异会影响算法的性能。为了解决这个问题，我们需要对数据进行归一化或标准化处理。上一节我们介绍了最大值最小值归一化，本节中我们来看看另一种更常用的方法：Z-Score归一化。

![](img/67dcafb3498c3eb506d1735f5cc95170_4.png)

## 什么是Z-Score归一化？💡

Z-Score归一化，也称为零均值标准化或标准化。它的核心思想是将原始数据转换为符合标准正态分布（均值为0，标准差为1）的数据。

我们可以用一个类比来理解：不同国家使用不同的货币（如人民币、美元、欧元）。为了在国际市场上比较购买力，我们需要通过汇率进行换算。同样，在数据中，为了消除不同特征之间单位和量纲的影响，使它们具有可比性，我们需要进行标准化处理。

![](img/67dcafb3498c3eb506d1735f5cc95170_6.png)

经过Z-Score标准化后，数据将呈现以下特点：
*   数据符合正态分布（高斯分布）。
*   数据的**均值（μ）为0**。
*   数据的**标准差（σ）为1**。

## 核心公式 📐

Z-Score归一化的计算公式如下：

![](img/67dcafb3498c3eb506d1735f5cc95170_8.png)

**`z = (x - μ) / σ`**

其中：
*   **`x`** 是原始数据中的一个值。
*   **`μ`** 是该特征所有样本数据的**平均值（均值）**。
*   **`σ`** 是该特征所有样本数据的**标准差**。

标准差的计算公式为：
**`σ = sqrt( (1/n) * Σ (xi - μ)^2 )`**

**公式解读：**
*   **分子 `(x - μ)`**：表示当前值偏离均值的程度。如果 `x` 大于均值，结果为正值；如果 `x` 小于均值，结果为负值。因此，标准化后的数据可以包含负数。
*   **分母 `σ`**：表示数据的离散程度。除以标准差可以将不同特征的数据“缩放”到同一个尺度上。

## 代码实现：手动计算 🖥️

![](img/67dcafb3498c3eb506d1735f5cc95170_10.png)

以下是使用NumPy库手动实现Z-Score归一化的步骤。

![](img/67dcafb3498c3eb506d1735f5cc95170_12.png)

首先，我们创建两组量纲不同的示例数据。

![](img/67dcafb3498c3eb506d1735f5cc95170_14.png)

```python
import numpy as np

# 创建第一组数据：年龄（范围0-10）
x1 = np.random.randint(0, 10, size=(10, 1))

# 创建第二组数据：薪水（范围1000-10000）
x2 = np.random.randint(1000, 10000, size=(10, 1))

# 合并两组数据
x = np.concatenate((x1, x2), axis=1)
print(“原始数据 x:\n”, x)
```

接下来，我们根据公式进行计算。

![](img/67dcafb3498c3eb506d1735f5cc95170_16.png)

```python
# 计算每一列（每个特征）的均值，axis=0表示按列计算
mean_val = np.mean(x, axis=0)
print(“均值 mean_val:\n”, mean_val)

![](img/67dcafb3498c3eb506d1735f5cc95170_18.png)

# 计算每一列（每个特征）的标准差
std_val = np.std(x, axis=0)
print(“标准差 std_val:\n”, std_val)

# 应用Z-Score公式进行归一化
norm = (x - mean_val) / std_val
print(“Z-Score归一化结果 norm:\n”, norm)
```

最后，我们验证标准化后的数据是否均值为0，标准差为1。

```python
# 验证归一化后数据的均值（应接近0）
print(“归一化后数据的均值:\n”, np.mean(norm, axis=0))

![](img/67dcafb3498c3eb506d1735f5cc95170_20.png)

# 验证归一化后数据的标准差（应接近1）
print(“归一化后数据的标准差:\n”, np.std(norm, axis=0))
```

![](img/67dcafb3498c3eb506d1735f5cc95170_22.png)

运行上述代码，你会发现 `norm` 数据中既包含正数也包含负数，并且其各列的均值接近0，标准差接近1。

## 代码实现：使用Scikit-learn库 🧰

在实际项目中，我们通常使用成熟的机器学习库来完成预处理。Scikit-learn提供了非常便捷的标准化工具。

以下是使用Scikit-learn进行Z-Score标准化的两种方式。

**方式一：分步进行（拟合和转换）**

```python
from sklearn.preprocessing import StandardScaler

![](img/67dcafb3498c3eb506d1735f5cc95170_24.png)

# 创建StandardScaler对象
scaler = StandardScaler()

![](img/67dcafb3498c3eb506d1735f5cc95170_26.png)

# 第一步：拟合数据，计算均值和标准差
scaler.fit(x)

# 第二步：使用拟合好的参数转换数据
norm2 = scaler.transform(x)
print(“使用Scikit-learn归一化结果 norm2:\n”, norm2)
```

**方式二：一步完成（拟合并转换）**

```python
# 创建StandardScaler对象并一步完成拟合和转换
norm3 = StandardScaler().fit_transform(x)
print(“一步完成归一化结果 norm3:\n”, norm3)
```

使用Scikit-learn得到的结果与手动计算的结果是完全一致的。`StandardScaler` 的 `fit` 方法用于从数据中学习参数（μ和σ），而 `transform` 方法则应用这些参数进行实际的转换操作。

## 总结

本节课中我们一起学习了Z-Score归一化。
*   **目的**：消除数据特征之间的量纲影响，使其转换为均值为0、标准差为1的标准正态分布数据，提升机器学习模型的性能。
*   **核心公式**：**`z = (x - μ) / σ`**。
*   **结果特点**：转换后的数据可正可负，且符合标准正态分布。
*   **实现方法**：可以手动使用NumPy根据公式计算，也可以直接调用Scikit-learn库中的 `StandardScaler` 类，后者更为简便和高效。
