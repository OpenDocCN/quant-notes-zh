## 概述

上一节我们介绍了轮廓系数，本节中我们来看看另一个常用的聚类评价指标：调整兰德系数。与轮廓系数不同，调整兰德系数是一种需要知道数据真实标签的外部评价指标，它通过比较聚类结果与真实标签的一致性来评估聚类质量。

## 调整兰德系数原理

![](img/62921b8df7b04b637583e4e75fb09cc9_4.png)

调整兰德系数是在兰德系数基础上改进的指标，旨在衡量两个数据划分（聚类结果与真实类别）之间的相似度。其核心思想是计算“划分正确”的样本对在所有样本对中所占的比例，并进行调整以消除随机划分的影响。

![](img/62921b8df7b04b637583e4e75fb09cc9_6.png)

![](img/62921b8df7b04b637583e4e75fb09cc9_8.png)

以下是其核心概念的定义：
*   **C**：表示数据真实的类别划分。
*   **K**：表示聚类算法得到的结果划分。
*   **a**：在C中被划分为同一类，同时在K中也被划分为同一簇的样本对数。
*   **b**：在C中被划分为不同类，同时在K中也被划分为不同簇的样本对数。

当聚类结果与真实标签完全一致时，效果最好。兰德系数（RI）的原始公式为：
`RI = (a + b) / C(n_samples, 2)`
其中，分母 `C(n_samples, 2)` 表示从所有样本中任意选取两个样本的组合数，即所有可能的样本对总数。

调整兰德系数（ARI）对兰德系数进行了修正，使其期望值在随机划分下为0，完美匹配时为1，并且可能为负值。这使其更具说服力和可比性。

## 调整兰德系数的使用

![](img/62921b8df7b04b637583e4e75fb09cc9_10.png)

理解了原理后，我们来看看如何在代码中使用调整兰德系数来评估K-Means聚类效果，并选择最佳的K值。

以下是使用调整兰德系数的步骤：

![](img/62921b8df7b04b637583e4e75fb09cc9_12.png)

1.  导入必要的库。
2.  准备数据和真实标签。
3.  使用不同的K值进行K-Means聚类。
4.  计算每个K值对应的调整兰德系数。
5.  可视化结果，选择得分最高的K值。

![](img/62921b8df7b04b637583e4e75fb09cc9_14.png)

```python
# 导入必要的库
from sklearn.metrics import adjusted_rand_score
from sklearn.cluster import KMeans
import matplotlib.pyplot as plt
import numpy as np

![](img/62921b8df7b04b637583e4e75fb09cc9_16.png)

# 假设 X 是特征数据，y_true 是真实的类别标签
# X, y_true = load_your_data()

![](img/62921b8df7b04b637583e4e75fb09cc9_18.png)

scores = [] # 用于存储不同K值下的调整兰德系数

# 尝试不同的K值（从2到6）
for k in range(2, 7):
    # 创建KMeans模型
    kmeans = KMeans(n_clusters=k)
    # 训练模型
    kmeans.fit(X)
    # 预测聚类标签
    y_pred = kmeans.predict(X)
    # 计算调整兰德系数
    # 参数：真实标签 (y_true), 预测标签 (y_pred)
    ari_score = adjusted_rand_score(y_true, y_pred)
    # 将得分存入列表
    scores.append(ari_score)

# 可视化不同K值对应的调整兰德系数
plt.plot(range(2, 7), scores, color='red')
plt.xlabel('K值选择')
plt.ylabel('调整兰德系数')
plt.title('不同K值下的调整兰德系数')
plt.show()
```

运行上述代码后，会生成一个折线图。通常，调整兰德系数最高的K值对应的聚类结果与真实标签最吻合。例如，当K=3时得分接近1，说明聚类结果与数据的3个真实类别高度一致。

## 与轮廓系数的对比

![](img/62921b8df7b04b637583e4e75fb09cc9_20.png)

![](img/62921b8df7b04b637583e4e75fb09cc9_22.png)

上一节我们使用了轮廓系数，本节使用了调整兰德系数。两者主要有以下区别：
*   **是否需要真实标签**：轮廓系数是内部评价指标，无需真实标签；调整兰德系数是外部评价指标，需要真实标签。
*   **得分范围**：轮廓系数范围在[-1, 1]之间；调整兰德系数范围约为[-0.5, 1]，完美匹配时为1。
*   **应用场景**：当没有真实标签时，使用轮廓系数；当有真实标签并想验证聚类算法还原真实分组的能力时，使用调整兰德系数。
