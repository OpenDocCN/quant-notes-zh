# 机器学习数学基础：P175：Jensen不等式介绍 📊

在本节课中，我们将要学习一个在机器学习（特别是EM算法推导中）非常重要的数学工具——Jensen不等式。我们将从凸函数的定义开始，逐步理解Jensen不等式的含义、几何解释及其应用场景。

## 凸函数的定义

上一节我们提到了Jensen不等式与凸函数密切相关，本节中我们首先来看看什么是凸函数。

![](img/8d973717ab67f5087dc542baa4356f38_1.png)

设函数 `f(x)` 的定义域为实数。如果对于所有实数 `x`，其二阶导数 `f''(x)` 都大于等于零，即：
`f''(x) >= 0`
那么函数 `f` 就是一个凸函数。

![](img/8d973717ab67f5087dc542baa4356f38_3.png)

![](img/8d973717ab67f5087dc542baa4356f38_4.png)

在数学和机器学习语境中，这种形状（开口向上，像碗一样）的函数被称为“凸函数”。为了更直观且避免与中文的“凹”字混淆，我们可以将其理解为“下凸函数”，即图形向下凸起。

与凸函数相对的是凹函数（或称“上凸函数”），其图形向上凸起，二阶导数 `f''(x) <= 0`。

## Jensen不等式公式

理解了凸函数后，我们现在可以正式介绍Jensen不等式。

如果 `f` 是一个凸函数，`X` 是一个随机变量，那么Jensen不等式指出，函数的期望值大于等于期望的函数值。其公式表述如下：
`E[f(X)] >= f(E[X])`
其中，`E` 表示数学期望（即平均值）。

当且仅当随机变量 `X` 是常数（即取值固定）时，上述不等式取等号。

对于凹函数，不等式方向相反：
`E[f(X)] <= f(E[X])`

## Jensen不等式的几何解释

公式可能有些抽象，接下来我们通过一个几何例子来直观理解它。

假设我们有一个下凸函数（即凸函数），其图形如下图所示。设随机变量 `X` 以各0.5的概率取值为 `a` 和 `b`。

*   `E[X]`（X的期望）是 `a` 和 `b` 的中点。
*   `f(E[X])` 对应的是函数在该中点处的值（图中红点）。
*   `E[f(X)]` 是 `f(a)` 和 `f(b)` 这两个函数值的平均值（图中蓝点）。

由于函数是下凸的，连接 `(a, f(a))` 和 `(b, f(b))` 两点的线段位于函数图像的上方。因此，该线段的中点（即 `E[f(X)]`，蓝点）必然位于函数图像上对应点（即 `f(E[X])`，红点）的上方。这直观地证明了 `E[f(X)] >= f(E[X])`。

## 核心要点总结

![](img/8d973717ab67f5087dc542baa4356f38_6.png)

本节课中我们一起学习了Jensen不等式。

![](img/8d973717ab67f5087dc542baa4356f38_8.png)

1.  **凸函数**：二阶导数大于等于零的函数，图形呈“下凸”形状。
2.  **Jensen不等式**：对于凸函数 `f` 和随机变量 `X`，有 `E[f(X)] >= f(E[X])`。它意味着“先求平均再计算函数值”不会超过“先计算函数值再求平均”。
3.  **几何意义**：不等式反映了凸函数图像在其任意两点连线的下方这一几何性质。
4.  **应用**：该不等式是推导EM算法等机器学习核心理论的关键工具。
