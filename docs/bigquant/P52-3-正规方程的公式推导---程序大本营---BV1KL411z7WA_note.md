## 矩阵运算基础 📐

在开始推导之前，我们需要回顾并掌握几个关键的矩阵运算公式。这些公式是后续推导的基石。

![](img/b2aa28580c0d80fb78660ddde59c0285_5.png)

以下是矩阵转置与求导的核心公式：

![](img/b2aa28580c0d80fb78660ddde59c0285_7.png)

*   **矩阵转置公式**：
    *   `(A + B)^T = A^T + B^T`
    *   `(AB)^T = B^T A^T`
    *   `(A^T)^T = A`
*   **矩阵求导公式**：
    *   `∂(x^T) / ∂x = I` （单位矩阵）
    *   `∂(x^T A) / ∂x = A`
    *   `∂(A x) / ∂x = A^T`

这些公式是矩阵运算中的基本定理，记住并理解它们即可。

## 最小二乘法的历史背景 📜

上一节我们提到了最小二乘法，本节中我们简单了解一下它的起源。正规方程的核心思想源于最小二乘法。

高斯在24岁时，利用最小二乘法成功预测了谷神星的轨道，其核心是最小化误差平方和，即求解以下目标函数的最小值：
`J(θ) = Σ (y_i - θ^T x_i)^2`
这个简单的公式背后蕴含着深刻的数学原理。我们现在就站在前人的肩膀上，来推导其解析解——正规方程。

## 正规方程的逐步推导 🔍

![](img/b2aa28580c0d80fb78660ddde59c0285_9.png)

接下来，我们将损失函数用矩阵形式表示，并一步步进行推导。我们的目标是找到令损失函数`J(θ)`最小的参数`θ`。

损失函数的矩阵形式为：
`J(θ) = (y - Xθ)^T (y - Xθ)`
其中，`X`是特征矩阵，`y`是目标值向量，`θ`是待求的参数向量。

![](img/b2aa28580c0d80fb78660ddde59c0285_11.png)

![](img/b2aa28580c0d80fb78660ddde59c0285_13.png)

### 第一步：展开矩阵乘法

首先，我们将矩阵乘法展开：
`J(θ) = (y^T - θ^T X^T)(y - Xθ)`
`= y^T y - y^T Xθ - θ^T X^T y + θ^T X^T Xθ`

由于`y^T Xθ`是一个标量，其转置等于自身，即`(y^T Xθ)^T = θ^T X^T y`，且两者相等。因此，中间两项可以合并：
`J(θ) = y^T y - 2θ^T X^T y + θ^T X^T Xθ`

### 第二步：对参数θ求导

我们对`J(θ)`关于`θ`求导，以找到其最小值点。应用前面提到的矩阵求导公式：
`∂J(θ)/∂θ = ∂(y^T y)/∂θ - 2 * ∂(θ^T X^T y)/∂θ + ∂(θ^T X^T Xθ)/∂θ`
`= 0 - 2 X^T y + 2 X^T Xθ`
`= -2 X^T y + 2 X^T Xθ`

### 第三步：令导数为零求解

损失函数`J(θ)`是一个关于`θ`的凸函数（因为包含`θ^T X^T Xθ`项），其最小值点出现在梯度（导数）为零处。因此，我们令导数等于零：
`-2 X^T y + 2 X^T Xθ = 0`
等式两边同时除以2，并移项：
`X^T Xθ = X^T y`

### 第四步：求解参数θ

现在，我们需要从等式`X^T Xθ = X^T y`中解出`θ`。矩阵没有直接的除法运算，我们通过左乘逆矩阵来求解。

假设`X^T X`是可逆的，我们在等式两边同时左乘其逆矩阵`(X^T X)^{-1}`：
`(X^T X)^{-1} (X^T X) θ = (X^T X)^{-1} X^T y`
根据逆矩阵的定义，`(X^T X)^{-1} (X^T X) = I`（单位矩阵），而单位矩阵乘以任何向量等于该向量本身：
`I θ = (X^T X)^{-1} X^T y`
因此，我们得到了正规方程的解：
`θ = (X^T X)^{-1} X^T y`

本节课中我们一起学习了正规方程的完整推导过程。我们从矩阵形式的损失函数出发，通过矩阵展开、求导、令导数为零并利用逆矩阵求解，最终得到了求解线性回归模型最优参数的公式 **`θ = (X^T X)^{-1} X^T y`**。理解这个推导，不仅让我们知道了公式的由来，也让我们更深刻地理解了最小二乘法的本质——寻找使预测误差平方和最小的模型参数。
