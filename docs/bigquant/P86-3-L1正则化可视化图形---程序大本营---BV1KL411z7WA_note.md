## 环境与代码初始化

首先，我们需要创建一个新的Python文件并导入必要的库。

```python
import numpy as np
import matplotlib.pyplot as plt
```

## L1正则化方程

上一节我们介绍了L1正则化的概念。本节中我们来看看其对应的数学方程。L1正则化项是模型参数绝对值的和。

**公式**：`L1 = |w1| + |w2|`

在二维空间中，我们可以用变量`x`和`y`分别代表`w1`和`w2`。因此，我们的目标函数可以表示为：

**公式**：`f(x, y) = |x| + |y|`

为了在二维平面上绘制其等高线（即约束边界），我们令`f(x, y)`等于一个常数`C`。教程中首先令`C=0`进行探索，但很快会发现`|x|+|y|=0`只在原点成立，没有线。因此，更合理的做法是令`C=1`，即绘制方程`|x| + |y| = 1`的图形。

## 分情况讨论并绘制图形

由于方程中包含绝对值，我们需要分情况讨论以去掉绝对值符号。以下是四种情况：

**情况一**：假设 `x >= 0` 且 `y >= 0`。
此时，绝对值可以直接去掉，方程变为 `x + y = 1`，即 `y = 1 - x`。

**情况二**：假设 `x < 0` 且 `y >= 0`。
此时，`|x| = -x`，方程变为 `-x + y = 1`，即 `y = 1 + x`。

**情况三**：假设 `x < 0` 且 `y < 0`。
此时，`|x| = -x`，`|y| = -y`，方程变为 `-x - y = 1`，即 `y = -x - 1`。

**情况四**：假设 `x >= 0` 且 `y < 0`。
此时，`|y| = -y`，方程变为 `x - y = 1`，即 `y = x - 1`。

现在，我们将这四种情况对应的线段绘制出来。

```python
# 设置图形尺寸，保证坐标轴比例一致
plt.figure(figsize=(6, 6))

# 情况一: x>=0, y>=0, y = 1 - x
x1 = np.linspace(0, 1, 50)
y1 = 1 - x1
plt.plot(x1, y1, color='k')  # ‘k’代表黑色

# 情况二: x<0, y>=0, y = 1 + x
x2 = np.linspace(-1, 0, 50)
y2 = 1 + x2
plt.plot(x2, y2, color='k')

# 情况三: x<0, y<0, y = -x - 1
x3 = np.linspace(-1, 0, 50)
y3 = -x3 - 1
plt.plot(x3, y3, color='k')

# 情况四: x>=0, y<0, y = x - 1
x4 = np.linspace(0, 1, 50)
y4 = x4 - 1
plt.plot(x4, y4, color='k')

# 设置坐标轴范围
plt.xlim(-2, 2)
plt.ylim(-2, 2)

# 移动坐标轴到图形中心（美化）
ax = plt.gca()
ax.spines['right'].set_color('none')
ax.spines['top'].set_color('none')
ax.spines['bottom'].set_position(('data', 0))
ax.spines['left'].set_position(('data', 0))

plt.show()
```

## 图形解读与核心概念

运行上述代码后，我们将得到一个以原点为中心的黑色菱形。

这个菱形就是方程 **`|x| + |y| = 1`** 的图形，它代表了L1正则化项的约束边界。在机器学习中，当我们试图在L1正则化约束下最小化原始损失函数`J0`时，最优解必须落在这个菱形边界上或内部。L1正则化的一个关键特性是其约束区域（菱形）的顶点在坐标轴上，这容易导致最优解的参数值恰好为0，从而实现特征选择（稀疏性）。
