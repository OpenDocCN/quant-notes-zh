## 什么是聚类？

上一节我们通过代码实现了聚类，现在我们来探讨其背后的原理。聚类算法是如何将数据划分成不同类别的呢？

首先，我们回到课件中，对理论进行说明和介绍。什么是聚类？让我们看一下scikit-learn官网的解释。

![](img/c6e8f4b729c1c7812ac3438946a044ba_5.png)

在scikit-learn官网中，我们可以看到机器学习任务主要分为几大类。`Classification`（分类）和`Regression`（回归）是我们之前学习过的。分类的目标是将数据点分开，例如区分红色和蓝色的点。回归则是寻找数据点的规律和趋势。

而`Clustering`（聚类）是另一个分支。与分类和回归不同，聚类所对应的数据只有特征`X`，没有目标值`y`。聚类的中文意思是“聚集成类”，其作用是将相似的对象自动分组到集合中。如下图所示，它将数据点划分到红色、绿色、蓝色等不同区域。

## 聚类的作用

了解了聚类的基本定义后，本节我们来看看聚类具体有哪些作用。

聚类主要有以下几个作用：
1.  **知识发现**：发现事物之间的潜在关系。如果数据间的关系类似或一致，就可以将其划归到同一个类别。
2.  **异常值检测**：在对数据进行类别划分时，某些数据点无法归入任何现有类别，这些点可以被视为异常值或离群点。离群点通常距离其他数据群较远，通过聚类可以实现异常值的检测和删除。
3.  **特征提取**：当原始数据特别多时，可以通过聚类提取关键数据，用更少的数据表示原始信息，这称为特征提取或数据压缩。

## 无监督学习

上一节我们介绍了聚类的作用，本节中我们来看看它所属的范畴——无监督学习。

我们之前学习的算法大多属于**有监督学习**。其过程是：给定训练集`X`和标签`y`，选择合适的模型进行学习，生成模型后进行预测。这就像学生时代的学习，每次考试后都有标准答案（`y`）来校正自己的理解。

而**无监督学习**则不同。它只有数据`X`，没有标签`y`。算法只能根据`X`本身的相似程度来做一些事情。如果两个数据的数值大小相似，就可以认为它们是相似的。聚类就是无监督学习的一种主要形式。

对于大量未标注的数据集，聚类的目标是：按照数据特征的相似性进行分组，使得**类别内相似度大**，**类别间相似度小**。相似度大意味着数据越近似；相似度小意味着不同类别的数据应该离得远。

![](img/c6e8f4b729c1c7812ac3438946a044ba_7.png)

![](img/c6e8f4b729c1c7812ac3438946a044ba_9.png)

此外，聚类还可以用来改变数据的维度。例如，在之前线性回归的学习中，我们接触过数据升维。通过聚类，我们可以将聚类的结果作为一个新的维度（特征）添加到原始训练集中。

在代码中，这可以如下操作：
```python
# 假设 X 是原始数据，y_pred 是聚类算法预测出的类别标签
X[‘类别’] = y_pred
print(X)
```
此时，原始数据`X`中就增加了一个名为“类别”的新属性。

![](img/c6e8f4b729c1c7812ac3438946a044ba_11.png)

## 聚类算法种类

![](img/c6e8f4b729c1c7812ac3438946a044ba_17.png)

了解了无监督学习的概念后，我们来看看聚类算法有哪些不同的种类。

![](img/c6e8f4b729c1c7812ac3438946a044ba_19.png)

![](img/c6e8f4b729c1c7812ac3438946a044ba_21.png)

聚类算法多种多样。在scikit-learn官网上，有一张名为“Overview of clustering methods”的图片，展示了不同聚类算法在相同数据集上的效果。

![](img/c6e8f4b729c1c7812ac3438946a044ba_23.png)

如下图所示，每一行是不同的算法，每一列是相同的数据。可以看到，对于相同的数据，不同算法的划分效果截然不同。

例如，使用`K-Means`算法时，它可能将上半部分数据和凸出的角归为一类（黄色），其余为另一类（蓝色）。而使用`DBSCAN`算法时，则能更好地区分出数据中不同的簇。

因此，不同算法的原理不同，对数据的划分效果也会不同。在课程中，我们会讲解一些常用和经典的算法，如`K-Means`、`DBSCAN`、`Gaussian Mixture`等。
