## 理论介绍：从线性回归到Softmax

上一节我们介绍了线性回归的基本概念。本节中我们来看看它的一个推广形式——广义线性回归。广义线性回归是线性回归的一种推广模式，就像广义相对论是狭义相对论的升华。

这个推广过程涉及多种概率分布，例如正态分布、二项分布和多项分布。数学家研究发现，这些分布都可以从线性回归模型衍生出来。了解这个背景知识即可。

*   **二项分布**：例如投硬币，结果只有两种（正面或反面）。
*   **多项分布**：例如掷骰子，结果有六种（1到6点）。

Softmax回归的理论证明基于广义线性回归，其公式较为复杂。我们暂时跳过证明部分，直接关注其核心的概率计算公式。

---

## Softmax回归的概率方程

![](img/083c442aaf93ed60c949ce875a57ac65_2.png)

Softmax回归用于多分类问题。假设我们有K个类别，其核心思想是为每个类别计算一个概率值。

![](img/083c442aaf93ed60c949ce875a57ac65_4.png)

对于一个给定的输入，其属于第k个类别的概率计算公式如下：

**P(y=k) = e^(z_k) / Σ(e^(z_j))**

其中，`z_k` 是对应于第k个类别的原始分数（例如线性模型的输出），分母是对所有类别（j从1到K）的 `e^(z_j)` 求和。这个公式确保了所有类别的概率之和为1。

---

## 公式解析与示例

为了更清晰地理解这个公式，我们通过一个具体例子来说明。

假设我们有三个原始数值：`[3, 1, -3]`。我们的目标是使用Softmax函数将它们转换为概率。

以下是计算过程：

1.  **计算分子**：对每个数值进行以自然常数e为底的指数运算。
    *   `e^3 ≈ 20.0855`
    *   `e^1 ≈ 2.7183`
    *   `e^(-3) ≈ 0.0498`

![](img/083c442aaf93ed60c949ce875a57ac65_6.png)

2.  **计算分母**：将所有指数运算结果相加。
    *   分母 = `20.0855 + 2.7183 + 0.0498 ≈ 22.8536`

![](img/083c442aaf93ed60c949ce875a57ac65_8.png)

3.  **计算每个类别的概率**：用各自的分子除以分母。
    *   P(数值3) = `20.0855 / 22.8536 ≈ 0.88`
    *   P(数值1) = `2.7183 / 22.8536 ≈ 0.12`
    *   P(数值-3) = `0.0498 / 22.8536 ≈ 0.00`

![](img/083c442aaf93ed60c949ce875a57ac65_10.png)

可以看到，转换后的概率和为1（`0.88 + 0.12 + 0.00 = 1`），且原始值越大，对应的概率也越大。这个过程本质上也是一种**归一化**。

## 与OvR方法的对比

![](img/083c442aaf93ed60c949ce875a57ac65_14.png)

在介绍多分类时，我们曾学习过“一对多”（One-vs-Rest， OvR）方法。现在我们来对比一下Softmax和OvR。

![](img/083c442aaf93ed60c949ce875a57ac65_16.png)

以下是OvR方法的主要特点：
*   **优点**：普适性广，实现简单。有多少个类别就训练多少个二分类器。
*   **缺点**：容易导致样本类别不平衡，因为每个分类器训练时，正例样本远少于负例样本。

![](img/083c442aaf93ed60c949ce875a57ac65_18.png)

Softmax回归的出现，正是为了更直接、更均衡地处理多分类问题，它一次性为所有类别输出概率，避免了OvR的样本不平衡问题。

## 代码实战：手写Softmax函数

理解了原理之后，我们现在回到代码中，亲手实现一个Softmax函数。

```python
import numpy as np

def softmax(z):
    # 对输入z进行指数运算
    exp_z = np.exp(z)
    # 计算所有指数值的和
    sum_exp_z = exp_z.sum()
    # 每个类别的概率 = 其指数值 / 指数值总和
    return exp_z / sum_exp_z

![](img/083c442aaf93ed60c949ce875a57ac65_20.png)

# 定义原始数据
z = np.array([3, 1, -3])
# 调用softmax函数
probabilities = softmax(z)
print("Softmax概率:", probabilities)
print("四舍五入保留两位小数:", np.round(probabilities, 2))
```
执行以上代码，将得到与理论示例一致的结果：`[0.88, 0.12, 0.00]`。
