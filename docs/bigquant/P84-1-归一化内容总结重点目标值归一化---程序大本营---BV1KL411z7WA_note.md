## 回顾：归一化核心概念

上一节我们介绍了归一化的强大功能及其基本数学原理。归一化主要有两种常用方法。

以下是两种核心的归一化公式：

![](img/1c07d132de860456bc2eb602c46cfbcf_3.png)

1.  **最大值最小值归一化（Min-Max Normalization）**
    该方法将数据线性缩放至[0, 1]区间。
    **公式**：`x_normalized = (x - min(x)) / (max(x) - min(x))`

2.  **零均值标准化（Z-Score Standardization）**
    该方法将数据转换为均值为0、标准差为1的分布。
    **公式**：`x_standardized = (x - μ) / σ`
    其中，均值μ和标准差σ的计算公式如下：
    *   均值公式：`μ = (1/n) * Σ(x_i)`
    *   标准差公式：`σ = sqrt((1/n) * Σ(x_i - μ)^2)`

我们可以根据这些公式手动实现，也可以便捷地使用`sklearn`库中提供的工具包来完成。

## 实战回顾：归一化效果验证

在上一节的实战中，我们使用了天池工业蒸汽量预测数据集。通过对比实验，我们直观地验证了归一化的效果。

以下是关键的实验发现：
*   **未归一化**：模型预测的评分（MSE）约为3点多。
*   **应用最大值最小值归一化后**：模型评分提升至1点多，实现了显著的性能飞跃。

在这个实战中，我们对训练数据和测试数据都进行了归一化处理。这里需要特别注意一个操作：我们对目标值（即要预测的工业蒸汽量）也进行了归一化。

## 重点探讨：目标值归一化

一般情况下，模型要学习的目标值不需要进行归一化。因为目标值是我们希望模型直接输出的真实尺度。

然而，在天池工业蒸汽量这个“开卷考试”式的竞赛中，评估标准是均方误差（MSE）。只要提交的结果能使MSE分数降低，任何合理的操作都是被允许的。由于我们不知道目标值的标准答案，可以进行各种尝试来寻找最优解。

在本次实战中，我们对目标值进行归一化后，模型性能得到了意外提升。这可以看作是一个针对特定竞赛数据集的优化技巧。你可以这样理解：对目标值进行转换，相当于对问题的答案空间进行了一次线性变换，而模型学习这个变换后的关系可能更为容易。

因此，这里得出一个重要结论：

以下是关于目标值归一化的关键说明：
*   **普遍原则**：在常规的机器学习项目中，**不应**对目标值进行归一化或转换。
*   **特殊情况**：在某些竞赛或探索性场景中，如果对目标值进行特定处理（如本例中的归一化）能显著提升评估指标，则可以谨慎使用。但这属于针对性的优化，并非通用流程。
*   **核心提醒**：在未来的项目实践中，请默认不要对目标值进行操作。本次操作是一个基于特定数据集的特殊发现。
