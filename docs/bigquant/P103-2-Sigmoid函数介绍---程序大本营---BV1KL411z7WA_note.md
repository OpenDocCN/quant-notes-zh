## 逻辑斯蒂回归与Sigmoid函数

上一节我们介绍了逻辑斯蒂回归的基本概念。本节中，我们来看看其核心的Sigmoid函数。

逻辑斯蒂回归是在多元线性回归的基础上，将结果缩放到0到1之间。在方程中，`h_θ(x)` 是对这个模型的描述。Sigmoid函数的作用正是将任意数据转换到0到1之间。

既然数据被转换到了0到1之间，就可以通过概率来进行类别的划分。这就是其巧妙连贯的地方。我们根据概率来划分类别：`h_θ(x)` 越接近1，就属于正例（一种类别）；`h_θ(x)` 越接近0，就属于负例（另一种类别）。通常以0.5作为划分的临界点。其中，`h_θ(x)` 就是我们的概率函数。

## Sigmoid函数的构成

![](img/7ffaefee656cb089679c9216c1a3d3db_7.png)

在描述中，我们首先看到 `h_θ(x)`，然后又看到 `g(z)`。这里 `θ^T x` 代表什么？它正是线性回归的方程。既然是线性回归，其值可能很大，也可能很小，无法直接用于分类。

为了解决这个问题，我们将 `θ^T x` 视为一个整体，即多元线性回归方程的输出。然后，我们将这个整体作为输入，放入Sigmoid函数中。具体来说，我们令 `z = θ^T x`，并将其代入Sigmoid函数 `g(z)`。

经过这样的转换，无论原来的线性回归值是1万还是-200，最终都会被映射到0到1之间的概率值。这样，我们就可以进行类别的划分了。逻辑斯蒂回归概率函数的构建非常巧妙。

## 分类器的本质与决策边界

我们知道，分类器的本质就是找到决策边界。就像人类大脑根据某个边界划分类别一样。

![](img/7ffaefee656cb089679c9216c1a3d3db_9.png)

在逻辑斯蒂回归中，我们通常把0.5作为分类边界。我们要找的就是预测值 `ŷ`（即 `h_θ(x)`，也就是概率）等于0.5时的条件。

当 `ŷ = h_θ(x) = 1 / (1 + e^{-θ^T x}) = 0.5` 时，我们求解 `θ^T x`。为什么此时 `θ^T x` 等于0？因为如果 `θ^T x = 0`，代入方程：`e^0 = 1`，那么 `1 / (1+1) = 0.5`。

所以，决策规则如下：
*   如果 `z = θ^T x > 0`，则预测为类别1。
*   如果 `z = θ^T x < 0`，则预测为类别0。
*   如果 `z = θ^T x = 0`，则概率刚好为0.5。

这个函数在数学中被称为**单位阶跃函数**的对数几率形式。经过这样的转变，只要概率大于0.5，统统归为类别1；小于0.5，统统归为类别0。这就好比以某个财富值（如100万）为界划分“富人”和“穷人”。这个函数主要用于二分类，但也可以通过扩展实现多分类。

## Sigmoid函数的求解过程图示

![](img/7ffaefee656cb089679c9216c1a3d3db_11.png)

接下来我们看一下它的求解过程图示。

在图示中，左边部分 `Σ (x_i * w_i) + b` 对应一个线性方程。我们使用符号 `z` 来表示这个线性方程：`z = b + Σ (x_i * w_i)`。其中，`b` 是截距，`Σ` 是求和符号（西格玛）。

这个 `z` 就对应Sigmoid函数的输入。我们对 `z` 进行Sigmoid变换：`g(z) = 1 / (1 + e^{-z})`。它所对应的方程，就是代码中画出来的S型曲线。

![](img/7ffaefee656cb089679c9216c1a3d3db_13.png)

这就是逻辑回归的完整过程。我们学习算法，既要知其然，也要知其所以然。

## 二分类的伯努利分布基础

二分类问题有一个特点：正类的概率加上负类的概率等于1。你只能属于类别0或类别1，两者概率之和为1。

![](img/7ffaefee656cb089679c9216c1a3d3db_15.png)

一个简单的试验只有两种可能结果，例如：正面和反面、成功或失败、有缺陷或无缺陷、病人康复或未康复。这就像抛硬币，要么正面，要么反面。

为了方便起见，在代码中我们一般不用“正例”、“负例”这样的字符表示，而是使用 **0** 和 **1**。通常，0表示负例，1表示正例。在很多场景下，我们可能更关注正例（或更关注负例，如医生更关注未康复的病人）。

下面的定义就建立在这类实验基础上：如果随机变量 `X` 只能取0和1两个值，并且相应的概率为 `P(X=1) = p`，那么另一类的概率自然就是 `P(X=0) = 1 - p`。

这就好比一个罐子里有红球和白球，摸到红球的概率是 `p`，那么摸到白球的概率就是 `1-p`。我们称变量 `X` 服从参数为 `p` 的**伯努利分布**。

![](img/7ffaefee656cb089679c9216c1a3d3db_17.png)

## 伯努利分布的公式表达

在进行公式书写时，可以写成以下形式，即概率质量函数：
`f(x; p) = p^x * (1-p)^{(1-x)}`, 其中 `x ∈ {0, 1}`

![](img/7ffaefee656cb089679c9216c1a3d3db_19.png)

这里的 `x` 代表类别（0或1），不要把它理解为连续坐标轴上的数值。

这个公式如何理解？你只要把 `x` 的值（0或1）代入方程即可：
*   当 `x = 1` 时，`f(1; p) = p^1 * (1-p)^{0} = p * 1 = p`
*   当 `x = 0` 时，`f(0; p) = p^0 * (1-p)^{1} = 1 * (1-p) = 1-p`

对于 `x` 既不是0也不是1的情况，其概率为0。数学家喜欢将不同的公式合并，这就是合并后的简洁公式。
