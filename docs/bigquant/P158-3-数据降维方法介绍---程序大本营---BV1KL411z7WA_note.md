# 机器学习入门：P158：数据降维方法介绍

![](img/12c9545e29d460282ffbfe448fcfe156_0.png)

在本节课中，我们将要学习数据降维的核心概念及其主要方法。数据降维是机器学习中一项重要的预处理技术，它可以帮助我们在保留数据关键信息的同时，减少数据的特征数量，从而提升模型效率并避免“维度灾难”。

## 数据降维方法分类

![](img/12c9545e29d460282ffbfe448fcfe156_2.png)

上一节我们介绍了数据降维的基本概念，本节中我们来看看具体有哪些方法。数据降维方法主要分为**线性**和**非线性**两大类。

*   **线性方法**：指降维变换是线性的，可以理解为通过一条直线或一个平面来映射数据。我们之前学过的线性回归就是典型的线性模型。
*   **非线性方法**：指降维变换是非线性的，例如通过抛物线等曲线来映射数据。在数学上，非线性关系可以用一元二次方程等来表示。

以下是两类方法中的具体代表算法：

*   **线性降维方法**：
    *   **PCA**：全称为 **Principal Component Analysis**（主成分分析）。
    *   **LDA**：全称为 **Linear Discriminant Analysis**（线性判别分析）。

*   **非线性降维方法**：
    *   **流行学习**：例如等度量映射、局部线性嵌入等算法。
    *   **NMF**：全称为 **Non-negative Matrix Factorization**（非负矩阵分解）。

## 方法详解

### 线性降维：PCA与LDA

PCA是一种无监督学习方法，旨在找到数据方差最大的方向（主成分），并将数据投影到这些方向上。其核心思想是**坐标轴旋转**，使得在新坐标系下，数据的第一个维度具有最大方差，第二个维度具有次大方差，依此类推。

LDA是一种有监督学习方法，它不仅考虑数据的方差，更关注如何**最大化类间距离并最小化类内距离**，从而找到能最好地区分不同类别的投影方向。

### 非线性降维与NMF

非线性方法（如流行学习）适用于数据存在于一个非线性流形上的情况。简单理解，“流形”就像一张被揉皱的纸，虽然在高维空间中看起来复杂，但其本质是二维的。这些方法试图“展开”这张纸，恢复其低维结构。

NMF是一种特殊的矩阵分解方法。它要求分解后的所有矩阵元素均为非负数。这类似于算术中的因数分解，例如 `6 = 2 × 3`。在NMF中，一个数据矩阵 `V` 被近似分解为两个非负矩阵 `W` 和 `H` 的乘积：

`V ≈ W × H`

通过这种分解，我们可以从原始数据中提取出具有实际意义的特征（例如，在文本分析中，`W` 可能代表“主题”，`H` 代表文档与主题的关联程度）。

## 总结

本节课中我们一起学习了数据降维的两大方法类别。线性方法（如PCA和LDA）通过直线或平面进行投影，而非线性方法（如流行学习和NMF）则能处理更复杂的、弯曲的数据结构。理解这些方法的区别是选择合适降维技术的第一步，在后续课程中我们将深入探讨其中具体算法的原理与应用。
