# 机器学习数据预处理：2：最大值最小值归一化 📊

在本节课中，我们将学习一种重要的数据预处理技术——最大值最小值归一化。我们将了解其原理、公式、代码实现，并探讨其优缺点。

## 概述

上一节我们讨论了数据未归一化对梯度下降等算法的影响。本节中，我们来看看如何通过**最大值最小值归一化**（也称为离差标准化）来解决数据量纲不统一的问题。其核心思想是将所有特征数据线性映射到[0, 1]的区间内。

## 归一化的本质与目的

规划的本质是把数据映射到0~1之间，或者说把数据映射到一定的范围之内。

梯度下降优化时步调不一致的根本原因，是不同特征（例如x1和x2）的数量级不同。就像个人财富与企业家的财富，计算单位（万 vs 亿）不同，导致不在一个量级上。

机器学习算法中也会遇到属性之间量级不同的问题。这时进行归一化，其实就是将不同属性的数量级进行统一。更扩展一点来说，如果有更多特征，就是把各个维度的特征进行数量级的统一，做到无量纲化。

## 第一种归一化方式：离差标准化

接下来我们看一下第一种规划的方式。这第一种规划的方式叫做离差标准化，我们最后得到的结果范围都是在0~1之间。

以下是其转换函数：

![](img/f4e61938b8bc6a29e3fd9f12feef4827_1.png)

![](img/f4e61938b8bc6a29e3fd9f12feef4827_3.png)

**公式：**
`X_norm = (X - X_min) / (X_max - X_min)`

![](img/f4e61938b8bc6a29e3fd9f12feef4827_5.png)

现在你就能够看到，分子是数值减去它的最小值，分母是最大值减最小值。分母是数据最宽的一个范围。

![](img/f4e61938b8bc6a29e3fd9f12feef4827_7.png)

你可以想一下，X什么时候最大？X最大就是X_max。如果X是X_max，计算出来的归一化数值应该等于1。什么时候它最小？最小就是X_min。如果X是X_min，分子（最小值减去X_min）等于零，求解出来的值就是零。

上面是最大值的情况，得到的数值是一；下面是最小值的情况，得到的值是零。其他数值范围在X_max和X_min之间，所以结果就被映射到0~1之间。无论你的数值多大或多小，都可以进行这样的转换。

我们很容易就发现，使用最大值和最小值归一化的优点，是可以把数值规划到0~1之间。但其缺点也很明显：如果有一个离群点，规划就会受到影响。例如，在一个数据集中加入“马云”或“雷军”的财富值，这个特别大的值算出来是1，而其他数据几乎都是0。所以说它受离群值的影响比较大。离群点会把整个数据区间拉长，导致其他有效数据被压缩。

## 代码实践：手动实现归一化

现在咱们回到代码当中，为各位演示一下。

```python
import numpy as np

# 创建模拟数据，模拟两个量纲不同的特征
x1 = np.random.randint(0, 10, size=10)  # 特征1，范围0-10
x2 = np.random.randint(1000, 5000, size=10)  # 特征2，范围1000-5000

# 将两个一维数组合并成一个二维数组，每列代表一个特征
x = np.c_[x1, x2]
print("原始数据 x:\n", x)

# 手动实现最大值最小值归一化
# axis=0 表示沿着列的方向计算（即对每个特征单独计算）
x_min = x.min(axis=0)
x_max = x.max(axis=0)

# 应用归一化公式
x_norm = (x - x_min) / (x_max - x_min)
print("\n归一化后的数据 (保留两位小数):\n", np.round(x_norm, 2))
```

运行以上代码，你会发现归一化前两列数据差距巨大，而归一化后，所有数据的范围都被压缩到了0~1之间。例如，第一列中的最大值9，经过计算后恰好等于1。

根据每一列进行缩放，就实现了特征的归一化，统一了量纲。

## 代码实践：离群点的影响

简单不简单？咱们刚才在课件中说到了离群值的问题，比如“马云”的财富特别大，就会导致一个数值是1，其他的数值都是0。来在代码当中演示一下。

```python
# 演示离群点的影响
# 在第二列（索引1）的某个位置（例如索引6）插入一个极大的异常值
x_with_outlier = x.copy()
x_with_outlier[6, 1] = 1000000  # 插入一个离群点
print("包含离群点的数据:\n", x_with_outlier)

# 对包含离群点的数据进行归一化
x_min_outlier = x_with_outlier.min(axis=0)
x_max_outlier = x_with_outlier.max(axis=0)
x_norm_outlier = (x_with_outlier - x_min_outlier) / (x_max_outlier - x_min_outlier)
print("\n包含离群点数据的归一化结果:\n", np.round(x_norm_outlier, 2))
```

运行代码后，你会发现由于异常值的存在，它被归一化为1，而其他原本有差异的数据几乎全部被压缩成了0。这就像在一个班级里，如果和“比尔盖茨”的财富比起来，大家的财富都近似为零。因此，在使用最大值最小值归一化时，通常需要先清洗或处理掉离群点。

## 使用 Scikit-learn 库实现

上面我们使用自己写的公式完成了归一化。接下来我们使用 `scikit-learn` 库中的方法为大家演示一下。

以下是使用 `scikit-learn` 的方法：

```python
from sklearn.preprocessing import MinMaxScaler
import numpy as np

# 创建缩放器对象
mm_scaler = MinMaxScaler()

# 使用同样的数据 x (或 x_with_outlier) 进行拟合并转换
# 注意：这里使用原始数据 x 进行演示
x_norm_sklearn = mm_scaler.fit_transform(x)

# 设置numpy打印选项，禁用科学计数法，便于观察
np.set_printoptions(suppress=True)
print("Scikit-learn 归一化结果 (保留两位小数):\n", np.round(x_norm_sklearn, 2))
```

调用 `scikit-learn` 中的 `MinMaxScaler` 方法，可以方便地实现最大值最小值的归一化，其结果与我们手动计算的结果完全一致。

## 总结

![](img/f4e61938b8bc6a29e3fd9f12feef4827_9.png)

本节课中我们一起学习了**最大值最小值归一化**。
*   我们理解了其核心目的是通过公式 **`(X - X_min) / (X_max - X_min)`** 将不同量纲的特征统一映射到[0, 1]区间，以优化梯度下降等算法的性能。
*   我们通过手动编写代码和调用 `scikit-learn` 的 `MinMaxScaler` 两种方式实现了该归一化。
*   同时，我们也认识到这种方法的**主要缺点是对离群点非常敏感**，一个极大的异常值会“拉长”整个数据范围，导致其他有效信息被压缩。因此，在实际应用中，往往需要先进行数据清洗，处理或剔除离群点。
