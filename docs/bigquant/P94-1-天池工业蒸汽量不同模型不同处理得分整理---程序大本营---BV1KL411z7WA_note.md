以下是基于得分对模型特性的进一步解读：

![](img/b47ac085e9e7c297d90a43f5ecc74461_7.png)

*   **Lasso回归（L1正则化）表现出色的原因**：Lasso回归的优化目标是在普通最小二乘法的损失函数上增加一个L1正则化项。
    **公式**：`损失函数 = 均方误差(MSE) + α * Σ|wi|`
    其中，`wi` 是模型系数，`α` 是控制正则化强度的参数。L1正则化倾向于将一部分不重要的特征系数**压缩至零**，从而实现特征选择。其出色的效果暗示了原始数据中可能存在一部分不重要的特征。
*   **岭回归（L2正则化）效果有限的原因**：岭回归增加的是L2正则化项。
    **公式**：`损失函数 = 均方误差(MSE) + α * Σ(wi²)`
    L2正则化会缩减所有系数，但**很少将系数恰好变为零**。它更倾向于让所有系数共同变小来处理共线性问题。在本数据上，其改善幅度（3.0896 -> 2.89）不如Lasso显著。
*   **弹性网络的综合优势**：弹性网络结合了L1和L2正则化。
    **公式**：`损失函数 = MSE + α * [ρ * Σ|wi| + (1-ρ)/2 * Σ(wi²)]`
    其中，`α` 控制整体正则化强度，`l1_ratio`（即公式中的 `ρ`）控制L1正则化的比例。它综合了两种正则化的优点，在本例中取得了最佳成绩（0.1277）。**注意**，通过调整 `alpha` 和 `l1_ratio` 参数，此分数有可能进一步优化。
*   **归一化与衍生算法结合的效果**：将Z-Score归一化与岭回归、Lasso回归等结合后，得分反而比单独使用这些衍生算法要差（例如，Lasso: 0.1318 vs Lasso+Z-Score: 0.2759）。这可能说明对于这些已经内置了正则化来应对不同尺度系数问题的模型，额外的归一化预处理在本数据集上并未带来协同增益，或需要不同的参数调优策略。
