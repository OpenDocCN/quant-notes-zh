# 量化交易基础：P8：1.3.2：交易基础知识 📚

![](img/118bbd2a392aab6a76480addbb66a6ad_1.png)

在本节课中，我们将学习股票交易的基础知识。了解这些核心机制，不仅能指导实盘交易，更能帮助我们在策略回测中建立更贴近现实的模型。

## 交易机制概述

![](img/118bbd2a392aab6a76480addbb66a6ad_3.png)

上一节我们介绍了市场的基本概念，本节中我们来看看股票是如何成交的。股票交易遵循**价格优先**和**时间优先**的原则。这意味着，在报价相同的情况下，先提交的订单优先成交；在时间相同的情况下，价格更优的订单优先成交。

为了争取成交，可以以涨停价挂单；为了获取更优价格，则需要在提交时间上抢占先机。

## 竞价方式

![](img/118bbd2a392aab6a76480addbb66a6ad_5.png)

交易所在不同时段采用不同的竞价方式，这直接影响了订单的匹配过程。

以下是两种主要的竞价方式：
*   **集合竞价**：交易所在特定时间点（如开盘、收盘）对所有接收到的买单和卖单进行一次性的集中撮合，产生一个统一的成交价格。
*   **连续竞价**：在交易日的正常时段，交易所对不断新进入的买单和卖单进行连续的、逐笔的撮合。

![](img/118bbd2a392aab6a76480addbb66a6ad_7.png)

目前，A股市场在开盘和收盘阶段采用集合竞价制度，目的是防止个别大额订单对这两个关键价格造成过度影响。需要注意的是，在开盘集合竞价的前5分钟，订单可以撤销，因此不能简单地用挂单情况来判断股价走势。

## 特殊规则与制度

除了基本的竞价原则，A股市场还有一些特有的规则，这些规则深刻影响着交易策略的设计。

以下是几个关键制度：
*   **ST制度**：ST是“特别处理”的缩写。交易所会对出现财务状况或其他异常状况的上市公司股票进行特殊标记，以提醒投资者注意风险。在量化策略中，`ST`常作为一个过滤因子使用，例如在构建股票池时排除ST股票。
*   **T+1制度**：这是A股与美股等市场的一个重要区别。它指的是**当日买入的股票，需等到下一个交易日才能卖出**。这一制度极大地限制了日内高频交易策略在股票市场的应用。
*   **涨跌停制度**：规定股票在一个交易日内的价格波动幅度限制（通常为±10%）。此举旨在抑制股价的日内极端波动。在策略回测中需注意：涨停的股票当天无法买入，若忽略此点会高估策略收益；超出涨跌幅限制的订单将被视为废单。
*   **交易单位**：股票买卖按“手”进行，**1手 = 100股**。订单数量必须是100股的整数倍。

## 交易成本

交易费用是评估策略盈利能力时不可忽视的一环。低估手续费会导致回测结果虚高，无法反映策略的真实表现。

![](img/118bbd2a392aab6a76480addbb66a6ad_9.png)

通常，手续费按成交金额的一定比例计算，例如常见的设置为：
*   买入费率：0.03%（万三）
*   卖出费率：0.13%（千分之1.3）
*   最低收费：单笔手续费不足5元时，按5元收取

在策略评估中，经常需要进行**手续费敏感性测试**，即逐步提高手续费率，观察策略收益率的衰减情况。如果收益率下降平缓，则说明策略对交易成本的鲁棒性较强。
