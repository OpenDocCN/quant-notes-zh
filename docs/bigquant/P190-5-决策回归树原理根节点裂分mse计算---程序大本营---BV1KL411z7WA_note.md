上一节我们介绍了决策回归树的基本概念，本节中我们来看看根节点裂分的具体计算。

## 根节点裂分条件

根据图示，根节点的裂分条件是 **x0 <= 3.142**。这个条件将数据集分成了左右两个子集。

## 计算左右分支的MSE

以下是计算左右分支MSE的步骤：

1.  **根据裂分条件筛选数据**：使用条件 `x_train[:, 0] <= 3.142` 将训练数据 `x_train` 和对应的标签 `y_train` 分为左右两部分。
2.  **计算MSE**：对于每个子集，计算其MSE。MSE的计算公式是预测值与真实值之差的平方的均值。在决策树中，一个节点的“预测值”通常取该节点内所有样本标签的**均值**。

因此，对于左侧分支，其MSE计算公式为：
**MSE_left = mean( (y_left - mean(y_left))^2 )**
右侧分支同理。

## 代码实现与演示

![](img/6792066ce4d2e331049c0a3d37ff1978_16.png)

![](img/6792066ce4d2e331049c0a3d37ff1978_18.png)

现在，我们通过代码来具体实现上述计算过程。

![](img/6792066ce4d2e331049c0a3d37ff1978_20.png)

```python
# 定义裂分条件
cond = x_train[:, 0] <= 3.142

![](img/6792066ce4d2e331049c0a3d37ff1978_22.png)

# 获取左侧分支的数据
y_left = y_train[cond].ravel()  # 使用.ravel()或.reshape(-1)将数据展平为一维数组
# 获取右侧分支的数据 (取反操作 ~cond)
y_right = y_train[~cond].ravel()

![](img/6792066ce4d2e331049c0a3d37ff1978_24.png)

![](img/6792066ce4d2e331049c0a3d37ff1978_26.png)

# 计算左侧分支的MSE
mse_left = ((y_left - y_left.mean()) ** 2).mean()
print(“左侧分支MSE:”, mse_left)

# 计算右侧分支的MSE
mse_right = ((y_right - y_right.mean()) ** 2).mean()
print(“右侧分支MSE:”, mse_right)
```

执行上述代码后，输出结果应为：
```
左侧分支MSE: 0.307
右侧分支MSE: 0.307
```
这与图示中左右子节点显示的MSE值（0.307）完全一致。

## 关键点与常见错误

在实现过程中，需要注意以下关键点：

![](img/6792066ce4d2e331049c0a3d37ff1978_28.png)

![](img/6792066ce4d2e331049c0a3d37ff1978_30.png)

*   **数据维度**：`x_train[:, 0] <= 3.142` 产生的布尔索引是二维的，直接用于索引 `y_train` 可能导致形状不匹配错误。使用 `.ravel()` 或 `.reshape(-1)` 方法可以将其转换为一维数组，这是解决此问题的常用方法。
*   **取反操作**：使用波浪线 `~` 对布尔条件进行取反，以获取不满足条件的样本（即右侧分支）。
*   **公式实现**：确保MSE的计算公式正确实现，即先求差值、平方，最后再求均值。注意括号的使用，以确保运算顺序正确。
