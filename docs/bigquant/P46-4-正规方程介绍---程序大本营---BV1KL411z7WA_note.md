# 机器学习：P46：正规方程介绍 🧮

在本节课中，我们将要学习线性回归中一个重要的数学工具——正规方程。我们将了解它的来源、公式含义以及如何通过矩阵运算来理解它。

## 什么是正规方程？

上一节我们介绍了线性回归的基本概念，本节中我们来看看求解模型参数的一种解析方法。

最小二乘法可以将误差方程转换为有确定解的代数方程，从而求解出未知参数。这个有确定解的代数方程，称为最小二乘法估计的正规方程。

## 最小二乘法回顾

为了理解正规方程，我们首先需要回顾最小二乘法。这个方法由高斯提出，其核心公式如下：

**公式：**
`J(θ) = Σ (hθ(x_i) - y_i)²`

以下是公式中各个部分的解释：
*   **Σ**：代表求和符号，表示对所有数据点进行计算。
*   **平方项**：公式中的“二”次方，正是“最小二乘法”名称中“二”的由来。
*   **y_i**：表示第 `i` 个数据的真实值。
*   **hθ(x_i)**：表示我们的线性回归模型对第 `i` 个数据的预测值。其中 `θ` 代表模型的参数（系数），例如斜率和截距。

因此，`hθ(x_i) - y_i` 计算的是单个数据点的预测误差。最小二乘法的目标就是找到一组参数 `θ`，使得所有数据点的预测误差的平方和 `J(θ)` 最小。

## 从最小二乘法到正规方程

根据最小二乘法的公式，我们可以推导出正规方程。正规方程的解可以用矩阵形式简洁地表示。

**公式：**
`θ = (XᵀX)⁻¹ Xᵀy`

在这个公式中：
*   **θ**：就是我们要求解的模型参数向量（等价于之前提过的 `w`）。
*   **X**：是所有样本特征数据组成的矩阵。
*   **y**：是所有样本真实值组成的向量。
*   **Xᵀ**：是矩阵 `X` 的转置。
*   **(XᵀX)⁻¹**：是矩阵 `(XᵀX)` 的逆矩阵。

## 矩阵形式推导的直观理解

你可能会问，为什么求和符号 `Σ` 在矩阵形式中消失了？这需要从矩阵乘法的规则说起。

原始的最小二乘误差公式是标量计算（数字的加减乘除）。当我们使用矩阵来表示所有数据时，为了得到所有误差的平方和，我们需要进行一种特定的矩阵运算。

**关键点：** 矩阵乘法中的 **一行乘以一列** 操作，其本质就是对应元素相乘后再求和。

![](img/1d1c3f9ba61047bfa32001a58e1144fe_1.png)

例如，假设有两个向量 `a = [1, 2]` 和 `b = [3, 4]`。我们想计算 `1*3 + 2*4`。这可以通过矩阵运算 `aᵀ * b` 来实现：
1.  将行向量 `a` 转置 (`aᵀ`) 变成列向量 `[1; 2]`。
2.  执行矩阵乘法 `[1; 2]ᵀ * [3; 4]`，这正是一行 `[1, 2]` 乘以一列 `[3; 4]`，结果就是 `1*3 + 2*4 = 11`。

![](img/1d1c3f9ba61047bfa32001a58e1144fe_3.png)

回到我们的误差公式。预测误差向量是 `(Xθ - y)`。要计算这个向量中所有元素的平方和，就等价于计算 `(Xθ - y)ᵀ * (Xθ - y)`。这个矩阵乘法自动完成了“对应元素相乘并求和”的操作，因此不再需要显式地写出求和符号 `Σ`。

所以，最小二乘法的目标函数用矩阵形式可以优雅地写为：
**公式：**
`J(θ) = (Xθ - y)ᵀ (Xθ - y)`

通过对这个关于 `θ` 的矩阵函数求导，并令导数为零，就可以推导出上面提到的正规方程 `θ = (XᵀX)⁻¹ Xᵀy`。

本节课中我们一起学习了正规方程。我们首先回顾了最小二乘法的原理，然后了解了正规方程是求解最小二乘问题的一个解析解公式 `θ = (XᵀX)⁻¹ Xᵀy`。最后，我们解释了为什么最小二乘公式的矩阵形式中求和符号会消失，其关键在于矩阵转置乘法实现了一行乘以一列的求和操作。正规方程提供了一种直接计算最优参数的方法，无需使用梯度下降之类的迭代算法。
