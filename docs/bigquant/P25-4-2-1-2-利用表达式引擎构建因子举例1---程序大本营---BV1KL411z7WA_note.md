# 量化因子构建教程：P25：4.2.1.2-利用表达式引擎构建因子举例1 🧮

在本节课中，我们将学习如何利用表达式引擎，基于基础因子构建一个复杂的衍生因子。具体来说，我们将构建一个名为“申万一级行业市盈率平均值”的因子。

上一节我们介绍了因子构建的基本概念，本节中我们来看看如何通过实际操作来构建一个具体的因子。

## 因子定义与构建思路

我们首先介绍如何通过因子库中的基础因子来构建新因子。这里我们构建的因子是“申万一级行业的市盈率平均值”。

该因子的构建依赖于两个基础因子：
1.  **申万一级行业类别**，其代码为 `industry_sw_level1`。
2.  **个股的市盈率因子**，其代码为 `pe_ttm0`。

我们的构建思路是：利用 `industry_sw_level1` 作为分组依据，计算每个行业内所有股票的 `pe_ttm0` 的平均值。这可以通过表达式引擎中的 `group_mean` 函数实现。

以下是完整的因子构建流程。

## 构建流程详解

整个因子构建流程分为三个主要步骤：设定数据范围、准备基础因子、计算衍生因子。

### 步骤一：设定数据范围

首先，我们需要指定股票数据的抽取范围。在本例中，我们抽取从 **2019年1月1日** 到 **2019年5月1日** 的全市场股票数据。

![](img/8a770b7f5533fd8ba21ae02fbfa08b0d_1.png)

### 步骤二：准备基础因子

接着，我们在“输入特征列表”中构建对应的因子表达式。我们通过“基础特征抽取模块”来获取两个基础因子的数据：
*   市盈率 (`pe_ttm0`)
*   申万一级行业 (`industry_sw_level1`)

![](img/8a770b7f5533fd8ba21ae02fbfa08b0d_3.png)

### 步骤三：计算衍生因子

最后，我们通过“衍生特征抽取模块”来计算复杂表达式的值。我们需要计算的表达式是：
```
group_mean(industry_sw_level1, pe_ttm0)
```
该模块将接收基础因子的数据，并根据上述表达式，计算出每个申万一级行业内的平均市盈率，并将结果作为新的因子列添加到最终的 `DataFrame` 中。

## 实际操作演示

现在，我们来实际操作这个例子。



我们进入策略编写界面，点击“新建可视化空白策略”。



点击“新建”后，我们从左侧的模块列表中按顺序拖入所需模块。

以下是需要拖入画布的四个模块：
1.  **证券代码列表模块** (位于“数据输入输出”列表)
2.  **输入特征列表模块** (位于“数据输入输出”列表)
3.  **基础特征抽取模块** (位于“特征抽取”列表)
4.  **衍生特征抽取模块** (位于“特征抽取”列表)

将四个模块拖拽至画布后，我们进行模块连接。



模块连接步骤如下：
1.  连接“基础特征抽取模块”的输出节点至“衍生特征抽取模块”的输入节点。这表示我们将抽取到的基础因子数据 (`DataFrame`) 传递给衍生特征抽取模块作为计算的基础。
2.  连接“输入特征列表”模块的输出节点至“衍生特征抽取模块”的另一个输入节点。这表示我们将定义好的因子表达式 `group_mean(industry_sw_level1, pe_ttm0)` 传递给衍生特征抽取模块。

完成连接后，衍生特征抽取模块将根据传入的基础因子数据列和因子表达式，执行计算。它会计算出每个行业分组内 `pe_ttm0` 的平均值，并将计算结果作为新的因子列保存到输出的 `DataFrame` 中。

本节课中我们一起学习了如何利用表达式引擎构建复杂因子。我们以“申万一级行业市盈率平均值”为例，完整演示了从定义因子、设定数据、准备基础数据到通过 `group_mean` 函数计算衍生因子的全过程。通过这个例子，你可以掌握使用基础模块组合和表达式引擎来创建自定义因子的基本方法。
