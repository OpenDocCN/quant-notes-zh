# 量化标注教程：P28：4.2.2.2：常见的标注表达式定义 📝

在本节课中，我们将学习几种常见的标注表达式定义方法。这些方法可以帮助我们构建更复杂、更有信息量的标签，用于量化模型的训练和评估。

上一节我们介绍了基础的未来收益率标注方法，本节中我们来看看如何定义更复杂的标注表达式，例如经波动率调整的收益率和相对收益率。

## 核心标注表达式定义

以下是几种常见的标注表达式及其定义方法：

1.  **未来N日收益率**
    这是最基础的标注，计算公式为：
    `(shift(close, -n) / shift(open, -1)) - 1`
    其中，`shift(close, -n)` 表示未来第n日的收盘价，`shift(open, -1)` 表示下一个交易日的开盘价。

2.  **未来N日经波动率调整后的收益率**
    此标注在收益率的基础上，除以了未来收益率的波动性（标准差），以衡量风险调整后的收益。实现方法是先计算未来N日收益率，再除以该收益率序列的标准差。可以使用平台函数 `std()` 计算标准差。

3.  **未来N日经平均真实波幅调整后的收益率**
    真实波幅（ATR）是衡量价格波动性的另一个指标。我们可以利用平台提供的 `ta_atr()` 函数计算真实波幅，然后用未来N日收益率除以该值，得到经ATR调整后的收益率。

4.  **未来N日的相对收益率**
    相对收益率是指标的收益率相对于某个基准指数（如沪深300、中证500）的收益率。在自定义标注模块中，我们可以指定 `benchmark` 参数来设定基准。然后，通过 `benchmark_close` 和 `benchmark_open` 字段获取基准指数的收盘价和开盘价，并构建类似的因子表达式来计算相对表现。
