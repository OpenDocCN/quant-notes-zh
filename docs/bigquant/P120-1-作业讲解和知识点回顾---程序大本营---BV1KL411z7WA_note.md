### 主成分分析（PCA）简介

PCA（Principal Component Analysis）是一种降维技术。我们的手写数字图像有784个像素点，即784个特征。其中许多特征是冗余或不重要的。

PCA的作用是“取其精华，去其糟粕”，保留数据中最主要的信息（方差最大的方向），舍弃次要信息。这类似于点菜时，主菜（硬菜）价值最大，而甜点、汤品相对次要。

![](img/719cb9ab56b9ab4aafd5645c457b17be_5.png)

在代码中，我们通过 `PCA(n_components=0.95)` 声明了一个PCA对象，参数 `0.95` 表示保留原始数据 **95%** 的重要信息。执行转换后，数据维度从784维降到了 **148维**，显著减少了计算量。

![](img/719cb9ab56b9ab4aafd5645c457b17be_7.png)

```python
from sklearn.decomposition import PCA
pca = PCA(n_components=0.95)
X_pca = pca.fit_transform(X)
```

转换后的 `X_pca` 数据依然能有效代表原始数据。

4.  **循环计算平均分**：我们使用降维后的数据（`X_pca`）进行100次循环。在每次循环中：
    *   使用 `train_test_split` 方法随机拆分数据。
    *   声明模型、进行训练和预测。
    *   累计每次的准确率分数。
    *   循环结束后，将累计总分除以100，得到平均准确率。

经过优化（数据抽样和PCA降维），逻辑回归运行100次平均耗时 **28.7秒**，平均准确率为 **0.85**；支持向量机运行100次平均耗时 **1分19秒**，平均准确率为 **0.957**。

**结论**：对于这个手写数字分类案例，支持向量机的表现优于逻辑回归。但需要注意的是，这只是一个特定案例的结果。在工业界，逻辑回归因其简单高效，在解决二分类问题时被广泛应用，甚至有“一招逻辑回归打天下”的说法。两种算法各有侧重，适用于不同场景。

## 支持向量机（SVM）核心知识点回顾

上一节我们重点介绍了用于分类的支持向量机。SVM的核心思想是寻找一个最优的超平面，以**最大间隔**将不同类别的数据分开。我们称这个最优超平面为决策边界。

### SVM的目标函数

我们一步步推导了SVM的目标函数。最终，我们希望求解的优化问题是：

**最大化间隔**，同时满足所有数据点被正确分类的约束。这可以形式化为以下带约束的优化问题：

> 找到参数 **w** 和 **b**，使得间隔 \( \frac{2}{\|\|w\|\|} \) 最大，同时对于所有训练样本 \( (x_i, y_i) \) 满足 \( y_i(w \cdot x_i + b) \geq 1 \)。

经过数学变换，这个最大化间隔问题等价于以下**最小化**问题，也就是我们最终得到的目标函数：

\[
\min_{w, b} \frac{1}{2} \|w\|^2
\]
\[
\text{subject to } y_i(w \cdot x_i + b) \geq 1, \quad \forall i
\]

![](img/719cb9ab56b9ab4aafd5645c457b17be_9.png)

![](img/719cb9ab56b9ab4aafd5645c457b17be_11.png)

### 拉格朗日乘子法

为了求解上述带约束的优化问题，我们引入了**拉格朗日乘子法**（Lagrange Multiplier Method）。该方法的核心是将**有约束**的优化问题转化为**无约束**的优化问题，从而方便求解（例如通过求导）。

对于原问题：
\[
\min f(x) \quad \text{s.t.} \quad h(x) = 0
\]

我们可以构造拉格朗日函数：
\[
\mathcal{L}(x, \lambda) = f(x) + \lambda h(x)
\]

其中，\( \lambda \) 称为拉格朗日乘子。这样，原问题就转化为对拉格朗日函数 \( \mathcal{L}(x, \lambda) \) 求极值的问题。通过对 \( x \) 和 \( \lambda \) 分别求偏导数并令其为零，即可求解出最优解。

在SVM的推导中，我们正是应用了拉格朗日乘子法来处理分类约束条件，从而得到对偶问题并最终求解。

### SVM的决策与支持向量

在代码演示中，我们展示了SVM如何对线性可分的数据进行划分。求解得到的超平面（决策边界）由斜率 **w** 和截距 **b** 决定。

![](img/719cb9ab56b9ab4aafd5645c457b17be_13.png)

![](img/719cb9ab56b9ab4aafd5645c457b17be_11.png)

那些**距离决策边界最近**的数据点被称为**支持向量**。正是这些支持向量决定了最优超平面的位置。间隔边界（图中虚线）会穿过这些支持向量。整个SVM模型的核心就是由这些关键的支持向量所支撑的。
