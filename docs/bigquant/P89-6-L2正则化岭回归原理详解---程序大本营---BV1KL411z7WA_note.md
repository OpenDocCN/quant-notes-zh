在本节课中，我们将学习线性回归的另一种正则化方法——岭回归。我们将重点探讨其核心原理，即L2正则化，并通过公式和代码演示其几何意义，帮助你理解它如何防止模型过拟合。

## 上一节回顾：从L1到L2正则化

上一节我们介绍了使用L1范数（绝对值之和）进行正则化的套索回归。本节中，我们来看看另一种正则化方式：L2范数，它对应着岭回归算法。

![](img/d83d16e28e742187a9ea13001498e080_9.png)

岭回归的损失函数是在普通线性回归损失函数的基础上，增加了一个L2正则化项。其核心公式如下：

![](img/d83d16e28e742187a9ea13001498e080_11.png)

![](img/d83d16e28e742187a9ea13001498e080_13.png)

**损失函数 = 原始损失 (J₀) + α × Σ(wᵢ²)**

其中，`α` 是控制正则化强度的超参数，`wᵢ` 是模型的各个系数（权重）。L2范数本应是各系数平方和再开方，但在这个公式中，我们直接使用了平方和，因此称为L2正则化。

## L2正则化的几何意义 ⭕

为了直观理解L2正则化，我们可以将其在二维空间（仅有两个系数 w₁ 和 w₂）中可视化。以下是其对应的方程和图形绘制过程。

L2正则化项的方程（当 α=1 时）为：
**1 = w₁² + w₂²**

这个方程描述了一个圆形。我们可以通过Python代码来绘制它：

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置图形为正方形，确保圆不被拉伸
plt.figure(figsize=(4, 4))

![](img/d83d16e28e742187a9ea13001498e080_15.png)

# 定义x的取值范围
x = np.linspace(-1, 1, 100)

![](img/d83d16e28e742187a9ea13001498e080_17.png)

![](img/d83d16e28e742187a9ea13001498e080_19.png)

# 根据方程 y = ±√(1 - x²) 计算y值
def f(x):
    return (1 - x**2)**0.5

![](img/d83d16e28e742187a9ea13001498e080_21.png)

![](img/d83d16e28e742187a9ea13001498e080_23.png)

y1 = f(x)  # 上半圆
y2 = -f(x) # 下半圆

![](img/d83d16e28e742187a9ea13001498e080_25.png)

![](img/d83d16e28e742187a9ea13001498e080_27.png)

# 绘制圆形
plt.plot(x, y1, color=‘k’)
plt.plot(x, y2, color=‘k’)
plt.show()
```

执行上述代码，你将看到一个标准的圆形。这与我们理论中的图形一致。

![](img/d83d16e28e742187a9ea13001498e080_29.png)

与上一节L1正则化形成的菱形相比，L2正则化形成的圆形被磨去了棱角。这个几何形状的差异导致了关键的特性：当原始损失函数（J₀）的等高线与这个圆形约束区域相交时，交点落在坐标轴（即系数为0）上的概率大大降低。

![](img/d83d16e28e742187a9ea13001498e080_31.png)

![](img/d83d16e28e742187a9ea13001498e080_33.png)

**这正是岭回归不具备“稀疏性”（即不会将大量系数恰好压缩为0）的直观原因。** 然而，通过L2正则化的限制，求解出的系数值会被整体向零点压缩，从而减小模型的复杂度，达到防止过拟合的效果。

![](img/d83d16e28e742187a9ea13001498e080_35.png)

![](img/d83d16e28e742187a9ea13001498e080_37.png)

## 权重更新规则 🔄

![](img/d83d16e28e742187a9ea13001498e080_39.png)

理解了L2的几何意义后，我们来看看在梯度下降算法中，权重是如何更新的。这能进一步揭示系数“缩小”的机制。

![](img/d83d16e28e742187a9ea13001498e080_41.png)

![](img/d83d16e28e742187a9ea13001498e080_43.png)

我们对包含L2正则项的损失函数求导。对于第 `j` 个系数 `θⱼ`（即 `wⱼ`），L2正则项 `α * Σ(θᵢ²)` 的偏导数为：
**∂L2/∂θⱼ = 2αθⱼ**

在梯度下降中，权重的更新公式为：新权重 = 旧权重 - 学习率 × 损失函数的梯度。

将原始损失梯度与L2正则项梯度合并，我们得到岭回归的权重更新公式：

**θⱼ := θⱼ - η[ ∂J₀/∂θⱼ ] - η(2α)θⱼ**

为了公式简洁，常数 `2` 可以被吸收到超参数 `α` 中，因此常写作：

![](img/d83d16e28e742187a9ea13001498e080_45.png)

**θⱼ := θⱼ - η[ ∂J₀/∂θⱼ ] - ηαθⱼ**

![](img/d83d16e28e742187a9ea13001498e080_47.png)

进一步整理，可以得到一个更清晰的形式：

![](img/d83d16e28e742187a9ea13001498e080_49.png)

![](img/d83d16e28e742187a9ea13001498e080_51.png)

**θⱼ := (1 - ηα)θⱼ - η[ ∂J₀/∂θⱼ ]**

从这个公式可以看出，**在每次更新时，权重 `θⱼ` 会先乘以一个小于1的因子 `(1 - ηα)`，从而被主动缩小**，然后才减去原始损失的梯度项。这使得最终训练出的模型系数相比普通线性回归会更小。

![](img/d83d16e28e742187a9ea13001498e080_53.png)

## 核心要点总结 📝

本节课中我们一起学习了岭回归及其背后的L2正则化原理：

![](img/d83d16e28e742187a9ea13001498e080_55.png)

![](img/d83d16e28e742187a9ea13001498e080_57.png)

1.  **核心公式**：岭回归通过在损失函数中增加 **α × Σ(wᵢ²)** 项来惩罚大的模型系数。
2.  **几何图形**：L2正则化在二维空间中约束区域是一个 **圆形**，这与L1的菱形不同。
3.  **关键特性**：由于圆形约束，岭回归的解通常不会恰好为0，因此**不具备特征选择的稀疏性**，但能**有效压缩所有系数的大小**。
4.  **更新机制**：通过梯度下降公式 **θⱼ := (1 - ηα)θⱼ - η[ ∂J₀/∂θⱼ ]** 可以清晰地看到，权重在每次迭代中都会被系统性地缩小。
5.  **根本目的**：通过限制系数的大小，降低模型复杂度，从而提高模型的泛化能力，**防止过拟合**。

![](img/d83d16e28e742187a9ea13001498e080_59.png)

![](img/d83d16e28e742187a9ea13001498e080_60.png)

简单来说，岭回归就像给模型的权重施加了一个向原点拉动的力，让模型变得更“保守”和“稳健”，特别适用于特征众多且存在共线性的场景。
