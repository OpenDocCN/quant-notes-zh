## 决策回归树原理

上一节我们介绍了决策回归树的基本概念，本节中我们来看看其分裂所依据的核心指标——均方误差的计算原理。

![](img/ee20561b6ddf9b6820bb8c233943052b_13.png)

MSE的公式如下：

![](img/ee20561b6ddf9b6820bb8c233943052b_15.png)

$$
MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y_i})^2
$$

![](img/ee20561b6ddf9b6820bb8c233943052b_17.png)

在这个公式中：
*   `y_i` 代表第 `i` 个样本的真实目标值。
*   `\hat{y_i}` 代表第 `i` 个样本的预测值。
*   `n` 代表样本的总数。

为了演示原理，我们使用深度为3的一棵回归树进行说明。

```python
model = DecisionTreeRegressor(max_depth=3)
model.fit(X_train, y_train)
```

我们将这棵树可视化，并重点关注其根节点的MSE值。

```python
dot_data = tree.export_graphviz(model, filled=True, rounded=True)
graph = graphviz.Source(dot_data)
graph
```

生成的决策树图形显示，根节点的MSE值为 **0.5**。这个值代表在未进行任何分裂时，当前节点数据的均方误差。

## 未分裂时MSE值的计算

理解了MSE的公式后，我们现在手动计算一下根节点的0.5是如何得出的。这对应于模型在首次分裂前的状态。

计算基于我们的训练数据 `y_train`。在未分裂时，模型没有其他信息，因此对于该节点内所有样本的预测值 `\hat{y}`，通常使用该节点内所有样本真实值的**平均值**来表示。

以下是计算步骤：

1.  **计算目标值的平均值**：首先求出 `y_train` 所有样本的平均值作为初始预测值。
2.  **计算每个样本的误差平方**：用每个样本的真实值减去这个平均值，然后对结果进行平方。
3.  **计算均方误差**：将所有样本的误差平方值求和，然后除以样本总数 `n`。

让我们用代码实现这个过程。注意，我们的 `y_train` 是一个二维数组（尽管只有一列数据），在计算时需要注意轴的方向。

```python
# 计算y_train的平均值，作为初始预测值
y_mean = y_train.mean(axis=0)

![](img/ee20561b6ddf9b6820bb8c233943052b_23.png)

![](img/ee20561b6ddf9b6820bb8c233943052b_25.png)

# 计算每个样本的预测误差并平方
squared_errors = (y_train - y_mean) ** 2

# 计算MSE：求所有误差平方的平均值
mse_manual = squared_errors.mean()
print(f"手动计算的MSE: {mse_manual}")
print(f"四舍五入后: {np.round(mse_manual, 1)}")
```

执行这段代码，得到的结果将是 `0.4996875`，经过四舍五入后恰好是 **0.5**。这与决策树图形中根节点显示的MSE值完全一致。

**关键点回顾**：公式中的求和再除以样本数 `(1/n) * Σ(...)`，其本质就是求算术平均值。因此，在代码中我们直接使用 `.mean()` 方法即可得到相同结果。
