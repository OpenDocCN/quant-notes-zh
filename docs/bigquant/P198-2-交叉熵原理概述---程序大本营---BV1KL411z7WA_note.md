在本节课中，我们将要学习机器学习中一个非常重要的概念——**交叉熵**。交叉熵是衡量两个概率分布之间差异的指标，在分类任务中常被用作损失函数，以评估模型预测结果与真实标签之间的差距。

## 什么是交叉熵？ 🤔

交叉熵源于信息论，用于度量两个概率分布之间的“距离”。在机器学习中，我们通常有一个真实的概率分布 **P**（例如，样本的真实标签）和一个模型预测的概率分布 **Q**。交叉熵越小，说明预测分布 **Q** 越接近真实分布 **P**。

它的核心公式如下：

![](img/970025996658ae5740b17f1f62e1af6d_3.png)

**H(P, Q) = - Σ P(x) * log(Q(x))**

![](img/970025996658ae5740b17f1f62e1af6d_5.png)

其中，`P(x)` 是真实分布中事件 `x` 发生的概率，`Q(x)` 是模型预测该事件发生的概率。

## 为什么使用交叉熵？ 💡

![](img/970025996658ae5740b17f1f62e1af6d_7.png)

上一节我们介绍了交叉熵的定义，本节中我们来看看为什么它在分类问题中如此有效。

![](img/970025996658ae5740b17f1f62e1af6d_9.png)

在分类任务中，特别是多分类任务，模型的输出通常是一个概率分布（例如，通过Softmax函数得到）。我们的目标是让模型输出的概率分布尽可能接近真实的“one-hot”编码分布（真实类别概率为1，其余为0）。

交叉熵损失函数能很好地衡量这种差异。当预测完全正确时，交叉熵损失为0；预测错误时，损失会增大，从而有效地指导模型参数的更新。

## 交叉熵的计算示例 📝

为了更好地理解，我们通过一个简单的例子来计算交叉熵。

假设我们有一个三分类问题，某个样本的真实标签是第2类。那么：
*   真实分布 **P** 为 `[0, 1, 0]`
*   假设模型预测的分布 **Q** 为 `[0.1, 0.7, 0.2]`

![](img/970025996658ae5740b17f1f62e1af6d_11.png)

以下是计算该样本交叉熵损失的步骤：

1.  根据公式，我们只关心真实概率为1的那个类别（即第2类）。
2.  计算该项：`-1 * log(0.7)`
3.  使用自然对数（或以2为底的对数，通常机器学习中使用自然对数），得到损失值。

![](img/970025996658ae5740b17f1f62e1af6d_13.png)

用代码可以这样表示：

```python
import numpy as np

# 真实标签（one-hot编码）
P = np.array([0, 1, 0])
# 模型预测的概率
Q = np.array([0.1, 0.7, 0.2])

# 计算交叉熵
# 因为P中只有第2个位置是1，所以只需计算 -1 * log(Q[1])
loss = -np.sum(P * np.log(Q))
print(f"交叉熵损失为：{loss:.4f}")
```

![](img/970025996658ae5740b17f1f62e1af6d_15.png)

运行这段代码，我们可以得到具体的损失数值。通过改变预测概率 `Q`，你可以观察到预测越准确（`Q[1]`越接近1），损失值越小。

![](img/970025996658ae5740b17f1f62e1af6d_17.png)

## 交叉熵与逻辑回归 🧩

在二分类问题中，交叉熵有更具体的形式，通常称为**二元交叉熵**或对数损失。

假设真实标签 `y` 为 0 或 1，模型预测为正类的概率为 `ŷ`。则二元交叉熵损失公式为：

**Loss = - [ y * log(ŷ) + (1 - y) * log(1 - ŷ) ]**

这个公式是通用交叉熵公式在二分类情况下的特例，它同样衡量了预测概率与真实标签的差距。
