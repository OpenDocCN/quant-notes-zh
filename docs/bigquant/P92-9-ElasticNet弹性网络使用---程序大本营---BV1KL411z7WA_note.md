在本节课中，我们将要学习弹性网络回归。弹性网络是一种结合了L1正则化（套索回归）和L2正则化（岭回归）的线性回归模型，旨在综合两者的优势，以应对特征相关性和模型复杂度控制等问题。

![](img/28d6ff78b159dea9b298e6a3a1484304_4.png)

## 弹性网络的概念与公式

上一节我们介绍了套索回归和岭回归，本节中我们来看看它们的结合体——弹性网络。

弹性网络是岭回归和套索回归技术的混合。它是一种使用L1和L2范数作为先验正则项来训练的线性回归模型。这种组合允许我们学习到一个只有少量参数是非零系数的稀疏模型，同时保留了岭回归的稳定性。

其核心目标函数公式如下：

![](img/28d6ff78b159dea9b298e6a3a1484304_6.png)

`min(||y - Xw||^2) + α * ρ * ||w||_1 + 0.5 * α * (1 - ρ) * ||w||_2^2`

其中：
*   `||y - Xw||^2` 是普通线性回归的损失项。
*   `α * ρ * ||w||_1` 是L1正则化项（套索回归部分）。
*   `0.5 * α * (1 - ρ) * ||w||_2^2` 是L2正则化项（岭回归部分）。
*   `α` 控制整体正则化的强度。
*   `ρ`（或 `l1_ratio`）控制L1和L2正则化的混合比例。当 `ρ=1` 时，模型退化为套索回归；当 `ρ=0` 时，模型退化为岭回归。

## 弹性网络的特点与应用场景

![](img/28d6ff78b159dea9b298e6a3a1484304_8.png)

弹性网络在实践中有其独特的优势，特别是在处理具有相关性的特征时。

![](img/28d6ff78b159dea9b298e6a3a1484304_10.png)

以下是弹性网络的主要特点：

1.  **处理特征相关性**：当数据集中许多特征彼此相关时（例如，身高与体重、数学成绩与物理成绩），弹性网络通常比单独的套索或岭回归表现更好。
2.  **兼具稀疏性与稳定性**：它继承了套索回归产生稀疏解（部分系数为零）的能力，同时也具备了岭回归处理共线性问题的稳定性。
3.  **参数权衡**：通过调整 `l1_ratio` 参数，可以在稀疏模型（套索）和稳定模型（岭回归）之间进行灵活的权衡。

## 代码实践：使用Scikit-learn实现弹性网络

现在，我们将通过代码来具体操作弹性网络回归。

以下是使用Scikit-learn库实现弹性网络回归的示例代码：

```python
# 导入必要的库
from sklearn.linear_model import ElasticNet
from sklearn.datasets import make_regression

![](img/28d6ff78b159dea9b298e6a3a1484304_12.png)

# 生成模拟回归数据集
X, y = make_regression(n_features=10, random_state=0)

![](img/28d6ff78b159dea9b298e6a3a1484304_14.png)

![](img/28d6ff78b159dea9b298e6a3a1484304_16.png)

# 创建弹性网络回归模型实例
# alpha：正则化强度；l1_ratio：L1正则化的混合比例（即公式中的ρ）
model = ElasticNet(alpha=0.5, l1_ratio=0.4, random_state=0)

![](img/28d6ff78b159dea9b298e6a3a1484304_18.png)

# 训练模型
model.fit(X, y)

# 输出模型学到的系数
print(“弹性网络回归系数：”)
print(model.coef_)
```

![](img/28d6ff78b159dea9b298e6a3a1484304_20.png)

运行这段代码，你将看到模型求解出的系数。与普通线性回归相比，这些系数值会被“压缩”，并且其中一部分系数会变为零。例如，你可能会观察到类似以下的输出：

```
弹性网络回归系数：
[ 0.         0.        -0.         0.36      -0.62      0.
 -0.         0.        -0.44      0.       ]
```

这展示了弹性网络的稀疏化效果：部分特征系数被缩减为零，而保留的非零系数值也普遍变小，从而实现了特征选择和防止过拟合的双重目的。你可以通过调整 `alpha` 和 `l1_ratio` 参数来观察模型系数的变化。
