# 量化交易入门：P13：BigStudio及Notebook实操指南 🚀

在本节课中，我们将学习如何在BigStudio平台上使用Notebook进行策略开发。我们将详细介绍策略开发环境的界面布局、文件管理操作，以及Notebook的基本使用方法，帮助你快速上手编写和运行量化策略代码。

---

## 策略开发环境导航 🧭

上一节我们介绍了如何进入BigStudio平台。本节中，我们来看看策略开发环境的具体布局和功能。

进入策略开发环境后，左侧有三个导航栏。点击“策略”导航栏，可以看到个人的策略文件夹列表。

以下是文件管理的主要操作：

*   **上传文件**：点击“上传文件”按钮，可以上传本地的Python脚本（.py）或CSV数据文件。
*   **下载文件**：右键点击任一文件，选择“下载”，可将策略或数据文件保存到本地。
*   **文件操作**：可以对文件进行复制、粘贴、剪切、删除、重命名等操作。
*   **克隆**：“克隆”功能等同于“复制”加“粘贴”，用于快速创建文件副本。
*   **刷新**：点击“刷新文件夹列表”按钮，可以更新当前文件夹的显示内容。

## 创建新策略或文件夹 📁

了解了文件管理后，我们来看看如何创建新的内容。右侧的“新建”按钮用于创建新的策略或文件夹。

点击“新建”按钮，会弹出创建选项。

以下是可创建的项目类型：

*   **股票策略**：专门用于股票交易的策略模板。
*   **期货策略**：专门用于期货交易的策略模板。
*   **通用策略**：不限定市场的通用代码环境。
*   **文件夹**：用于分类管理策略文件。

## 使用Notebook编写策略 📓

现在，我们开始实际操作。点击“通用策略”来创建一个空白的Notebook交互式编程环境。

![](img/d60544ff4d0736b901bb974963b43d9f_5.png)

在创建时，我们将这个Notebook的名称保持为默认的“notebook”，然后点击“新建”。

![](img/d60544ff4d0736b901bb974963b43d9f_5.png)

新建完成后，我们就进入了Notebook编辑界面。Notebook以“Cell”（代码单元）为最小单位来组织和管理代码。

每个Cell上方提供了一系列操作按钮。

以下是Cell单元的主要操作功能：

*   **向上/向下插入单元**：在当前Cell的上面或下面新增一个空Cell。
*   **删除单元**：删除当前Cell。
*   **剪切/复制/粘贴单元**：对Cell进行移动或复制。
*   **上移/下移单元**：调整Cell的排列顺序。

在Cell中输入需要执行的代码，例如输入：
```python
1 + 1
```
然后按下 `Control + Enter` 组合键，或点击右侧的“运行”按钮。服务器会在后台运行这段代码并返回结果。可以看到，返回的运行结果是 `2`。

你可以在不同的Cell单元中输入不同的代码段。点击“运行全部”按钮，平台会按照从上到下的顺序依次执行所有Cell，并分别返回对应的结果。

## Notebook的高级功能与分享 🔗

掌握了基础编写和运行后，我们来看看Notebook的其他实用功能。

你可以通过点击“保存”按钮，来保存对当前Notebook文件的修改。

在代码运行过程中，如果某个Cell执行时间过长，可以点击“终止”按钮来停止该单元的运行。点击“重启内核”按钮，可以中断所有正在运行的任务并重新启动策略的执行环境。

“克隆策略”功能可以实现对整个Notebook文件的复制。点击右侧的“策略链接”，可以查看该策略的唯一访问链接。你可以复制此链接，在社区发帖时将你的策略分享到平台上。

此外，点击“分享”按钮可以设置更详细的分享权限。
