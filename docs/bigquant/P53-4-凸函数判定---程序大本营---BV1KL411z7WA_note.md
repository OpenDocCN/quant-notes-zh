# 机器学习：P53：4-凸函数判定

在本节课中，我们将要学习凸函数的定义、判定方法及其在机器学习中的重要性。理解凸函数能帮助我们确认损失函数的最优解是全局最优解。

## 凸函数的定义与重要性

上一节我们介绍了损失函数的概念，本节中我们来看看如何判定一个函数是否为凸函数。在讨论区有小伙伴提到了凸函数，并画出了类似下图的曲线。这个曲线就是凸函数。

![](img/7826409e88fc01ecb9c06a5a075d7bef_1.png)

判定损失函数是凸函数的好处在于，我们可以肯定地知道，通过求解得到的极值点就是最优解，并且一定是全局最优解。很多公式不一定要全部记住，但需要自己动手写一写，加深理解。

## 凸函数与凹函数的定义差异

以下是关于凸函数和凹函数定义的重要说明。请注意，不同地区（如中国和美国）对“凸”和“凹”的定义可能不同。

*   在一些教材或网络资料中，上图所示的形状可能被称为“凹函数”，因为它看起来是“凹下去”的。
*   但在机器学习领域，尤其是基于英文原版教材的原理中，通常采用美国的定义方式：将上图所示的函数形状定义为**凸函数**，而将相反形状（像拱形）的函数定义为凹函数。
*   这种定义差异可能源于语言和文化背景。算法的原理多由国外学者发明，因此我们沿用了他们的定义标准。

在机器学习领域，凸函数有明确的数学判定方法。

## 凸函数的数学判定方法

那么，如何用数学方法判定一个函数是否为凸函数呢？我们一起来看一下。

![](img/7826409e88fc01ecb9c06a5a075d7bef_1.png)

![](img/7826409e88fc01ecb9c06a5a075d7bef_3.png)

![](img/7826409e88fc01ecb9c06a5a075d7bef_3.png)

核心方法是求函数的二阶导数。

**判定准则**：对于一个函数 `f(x)`，如果它的二阶导数 `f''(x)` **大于等于零**，那么这个函数就是凸函数。

![](img/7826409e88fc01ecb9c06a5a075d7bef_5.png)

用公式表示为：
`f''(x) >= 0`

![](img/7826409e88fc01ecb9c06a5a075d7bef_7.png)

当确认损失函数是凸函数后，我们就可以放心地对损失函数求一阶导数，并令导数等于零。这样求解出来的解一定是全局最优解。

## 凸函数判定的意义

![](img/7826409e88fc01ecb9c06a5a075d7bef_9.png)

为什么凸函数的判定如此重要？请看下面的对比。

![](img/7826409e88fc01ecb9c06a5a075d7bef_5.png)

*   **左侧的凸函数**：其曲线光滑且只有一个最低点（谷底）。我们之前学过的最小二乘法公式对应的就是这种凸函数，因此我们可以使用正规方程求解，并确信得到的是最优解。
*   **右侧的非凸函数**：其曲线“曲溜拐弯”，存在多个低谷。其中可能只有一个是全局最优解，其他都是局部最优解。在数据复杂、无法可视化的情况下，优化算法很容易陷入局部最优而找不到最好的解。

![](img/7826409e88fc01ecb9c06a5a075d7bef_7.png)

对于高维空间（例如三维，本质是二维图像的视角变换），道理是相同的。判断函数是否为凸函数，依然是求其二阶导数（或海森矩阵在高维情况下是否半正定）。

## 实例：最小二乘法的凸性

让我们以最小二乘法为例，具体应用凸函数的判定准则。

![](img/7826409e88fc01ecb9c06a5a075d7bef_9.png)

最小二乘法的损失函数经过推导，其二阶导数形式为 `X^T X`。这相当于矩阵 `X` 与其自身的点乘（内积），其结果一定**大于等于零**。

![](img/7826409e88fc01ecb9c06a5a075d7bef_11.png)

用公式表示其凸性判定：
`f''(θ) = X^T X >= 0`

因此，最小二乘法的损失函数一定是凸函数，这保证了我们通过求导找到的解是全局最优解。

## 机器学习的实用主义思想

最后，我们探讨一下机器学习中的一个重要思想。

![](img/7826409e88fc01ecb9c06a5a075d7bef_13.png)

![](img/7826409e88fc01ecb9c06a5a075d7bef_11.png)

机器学习不追求模型百分之百正确或完美，而是强调**只要有价值、堪用即可**。这就像解决问题不需要总是使用最强大的工具，合适、够用就好。

![](img/7826409e88fc01ecb9c06a5a075d7bef_15.png)

![](img/7826409e88fc01ecb9c06a5a075d7bef_13.png)

我们的目标是找到**堪用**的解决方案，而非理论上绝对完美的解。这种务实的思想贯穿于机器学习的模型选择与优化过程中。

**本节课总结**

![](img/7826409e88fc01ecb9c06a5a075d7bef_17.png)

本节课中我们一起学习了：
1.  **凸函数的定义**：在机器学习语境下，形状如“碗状”的函数被称为凸函数。
2.  **凸函数的判定方法**：核心是检查其二阶导数是否 `f''(x) >= 0`。对于最小二乘法，其二阶导数 `X^T X >= 0` 恒成立，因此它是凸函数。
3.  **凸函数的重要性**：凸函数能保证找到的极值点是全局最优解，避免了陷入局部最优的问题。
4.  **机器学习的核心思想**：追求**有价值且堪用**的解决方案，而非绝对的完美。

![](img/7826409e88fc01ecb9c06a5a075d7bef_18.png)

理解凸函数有助于我们更深刻地认识许多优化算法（如正规方程）为何有效，并为后续学习更复杂的非凸优化问题打下基础。
