# 量化交易基础：P2：1.1.2：量化交易策略 📈

![](img/42b6909d19e62ac643f5d20a8e947d03_0.png)

在本节课中，我们将要学习量化交易策略的核心概念、基本流程、常见陷阱以及评估方法。量化投资是一种基于数学模型和计算机技术的科学投资方式，旨在通过系统性的方法捕捉市场机会。

## 概述

量化投资作为一种科学的投资方式，其优势主要体现在纪律性、及时性、系统性、准确性和分散化这几个方面。

## 量化投资的五大优势

以下是量化投资相较于传统投资方法的五个主要优势。

1.  **纪律性**：严格按照策略信号进行交易，可以克服人性中贪婪与恐惧等弱点。
2.  **及时性**：利用计算机的高性能运算，及时跟踪市场变化，第一时间发现投资机会。高频交易是这一特点的典型体现。
3.  **系统性**：通过多层次的量化模型，从多角度观察并处理海量数据，以捕捉大量投资机会。量化策略通常拥有完整的交易系统。
4.  **准确性**：量化策略采用数量化模型刻画交易信号，所有信号均由模型准确预测得出，不带主观情绪偏差。
5.  **分散化**：量化投资可以选择多市场、多标的、多种交易模式，天然具备风险分散的特点。正如一句名言所说：“金融市场没有免费的午餐，如果有，那就是分散化。”

## 量化投资的基本流程

上一节我们介绍了量化投资的优势，本节中我们来看看一个典型的量化策略是如何从想法变为实践的。量化投资一般遵循以下流程。

1.  **数据获取**：数据是量化研究的原料和基础。数据质量直接影响研究成果。
2.  **策略回测**：在历史数据上测试交易思想。例如，“每年年初买入行业龙头股票并持有”就是一个交易思想。需要通过回测判断其是否能成为盈利策略。
3.  **模拟交易**：回测表现优秀的策略，需进行一段时间的模拟交易，观察其表现是否符合预期，以检验策略在近期市场的适应性。
4.  **实盘交易**：通过模拟交易检验的策略，可投入实盘。但需注意，即使回测和模拟表现良好，实盘仍可能因市场变化而失效。

## 策略回测的重要性与陷阱

![](img/42b6909d19e62ac643f5d20a8e947d03_4.png)

回测是量化投资中至关重要的一环，甚至有人认为未经回测的策略不能算作量化策略。然而，大部分策略无法通过回测检验，且回测过程中存在诸多陷阱。

以下是回测中常见的几个陷阱。

*   **未来函数**：指在回测中，策略使用了当时无法获取的未来信息。例如，一个策略在2010年“知道”了2015年某公司的财报数据。
*   **过拟合**：指模型过度学习和记忆了历史数据中的噪音，导致在未来表现不佳。其公式可简单理解为：模型复杂度 > 数据信息量。
*   **幸存者偏差**：在构建股票池时，只包含了目前存活的股票，而排除了历史上已退市的股票，从而粉饰了回测结果。
*   **交易费用低估**：在回测中使用的交易手续费或税费低于实际水平，虚高了策略收益。
*   **冲击成本忽略**：大规模交易行为本身会影响市场价格，产生额外的买入成本或卖出损失，这在回测中常被忽略。
*   **小样本偏差**：使用的历史数据样本量太少，导致分析结论的普适性不强。
*   **偷价漏价**：在回测中假设以不可能实现的“最高价卖出”或“最低价买入”进行撮合，这在实盘中是不切实际的。

## 策略评估指标

当策略完成回测后，我们需要一套客观统一的标准来评估其优劣。评估指标主要分为两类：衡量盈利能力的指标和衡量抗风险能力的指标。

![](img/42b6909d19e62ac643f5d20a8e947d03_6.png)

以下是一些常用的策略评估指标。

*   **收益率 / 年化收益率**：策略的绝对收益表现。
*   **夏普比率**：最通用的策略评价指标之一，代表经风险调整后的收益。公式为：`(策略收益率 - 无风险利率) / 策略收益波动率`。
*   **最大回撤**：策略在历史上从任一峰值下跌的最大幅度，是衡量下行风险的关键指标。
*   **阿尔法与贝塔**：衡量策略相对于市场基准的超额收益（阿尔法）和市场风险暴露（贝塔）。
*   **收益波动率**：策略收益的波动程度，是风险的基本度量。
*   **信息比率**：衡量策略超额收益的稳定性。
