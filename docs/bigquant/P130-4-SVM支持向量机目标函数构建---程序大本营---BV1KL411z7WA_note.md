# 机器学习：P130：SVM支持向量机目标函数构建

在本节课中，我们将学习如何构建支持向量机的目标函数，并通过拉格朗日乘子法、KKT条件和对偶问题，将其转化为一个更易求解的优化问题。我们将一步步推导，最终将求解原始参数的问题转化为求解拉格朗日乘子的问题。

## 概述

支持向量机的核心目标是找到一个最优超平面，以最大化分类间隔。这可以表述为一个带约束的最优化问题。本节课，我们将详细讲解如何从原始目标函数出发，通过引入拉格朗日乘子，构建拉格朗日函数，并利用对偶转换，将问题转化为求解拉格朗日乘子的形式。

## 目标函数与约束

上一节我们介绍了SVM的基本思想是最大化间隔。本节中我们来看看如何形式化这个目标。

SVM的原始优化问题是最小化权重向量的范数，同时满足所有样本点的分类约束。其目标函数和约束条件如下：

最小化：$\frac{1}{2}||w||^2$
约束条件：$y_i(w^T x_i + b) \geq 1, \quad \forall i$

## 构建拉格朗日函数

为了求解这个带约束的优化问题，我们引入拉格朗日乘子法。以下是构建拉格朗日函数的步骤：

首先，我们为每一个约束条件引入一个非负的拉格朗日乘子 $\lambda_i$。

接着，我们构建拉格朗日函数 $L(w, b, \lambda)$，它将原始目标函数与所有约束条件结合起来：

$$L(w, b, \lambda) = \frac{1}{2}||w||^2 + \sum_{i=1}^{n} \lambda_i [1 - y_i(w^T x_i + b)]$$

其中，要求 $\lambda_i \geq 0$。

## 转化为对偶问题

直接求解上述拉格朗日函数的最小值比较困难。因此，我们将其转化为对偶问题，这通常更容易求解。

根据优化理论，原始问题可以等价于如下形式：

$$\min_{w, b} \max_{\lambda \geq 0} L(w, b, \lambda)$$

我们可以交换极小和极大的顺序，得到其对偶问题：

$$\max_{\lambda \geq 0} \min_{w, b} L(w, b, \lambda)$$

## 求解内部极小化问题

现在，我们固定拉格朗日乘子 $\lambda$，先求解内部关于 $w$ 和 $b$ 的极小化问题。

我们对拉格朗日函数 $L$ 分别求关于 $w$ 和 $b$ 的偏导数，并令其等于零。

以下是求解过程：

1.  对 $w$ 求偏导：
    $$\frac{\partial L}{\partial w} = w - \sum_{i=1}^{n} \lambda_i y_i x_i = 0$$
    由此得到：
    $$w = \sum_{i=1}^{n} \lambda_i y_i x_i$$

2.  对 $b$ 求偏导：
    $$\frac{\partial L}{\partial b} = -\sum_{i=1}^{n} \lambda_i y_i = 0$$
    由此得到：
    $$\sum_{i=1}^{n} \lambda_i y_i = 0$$

## 代入得到对偶形式

我们将上面求得的 $w$ 和关于 $b$ 的约束条件，代回拉格朗日函数 $L$ 中。

经过代入和化简，原拉格朗日函数中关于 $w$ 和 $b$ 的项被消去，得到了一个只关于拉格朗日乘子 $\lambda$ 的函数。

最终的对偶优化问题如下：

$$\max_{\lambda} \sum_{i=1}^{n} \lambda_i - \frac{1}{2} \sum_{i=1}^{n} \sum_{j=1}^{n} \lambda_i \lambda_j y_i y_j (x_i^T x_j)$$

约束条件为：
$$\sum_{i=1}^{n} \lambda_i y_i = 0$$
$$\lambda_i \geq 0, \quad \forall i$$

## 问题转化与求解

至此，我们完成了一个关键的转化。

我们最初的问题是求解最优的权重 $w$ 和偏置 $b$。现在，问题变成了求解一组最优的拉格朗日乘子 $\lambda_i$。

一旦我们通过求解上述对偶问题得到了 $\lambda_i$，就可以利用公式 $w = \sum_{i=1}^{n} \lambda_i y_i x_i$ 轻松地计算出最优的权重向量 $w$。偏置 $b$ 则可以通过支持向量（对应 $\lambda_i > 0$ 的样本）的约束条件求出。

求解对偶问题（即求解 $\lambda$）的常用高效算法是序列最小优化算法。

## 核心概念：向量内积

![](img/58f3c36020811397d1e51da5ca949d06_3.png)

在对偶问题的目标函数中，出现了 $x_i^T x_j$ 这一项，它表示两个样本特征向量的内积。

向量内积的代码如下：
```python
# 假设 xi 和 xj 是两个向量
xi = [xi1, xi2, ..., xin]
xj = [xj1, xj2, ..., xjn]
# 内积计算
dot_product = sum(xi[k] * xj[k] for k in range(n))
```
在支持向量机中，内积运算非常重要，它是核函数方法的基础。
