在本节课中，我们将学习如何使用数学中的导数概念，通过编程来求解一个函数的最小值。我们将从一个具体的抛物线方程入手，演示如何通过求导并令导数为零来找到最小值点，并理解这种方法的应用场景与局限性。

## 构建函数和它的导函数

首先，我们定义一个具体的函数，并可视化它的图像。这是一个抛物线方程。

我们导入必要的库，并定义函数 `f(x)`。

```python
import matplotlib.pyplot as plt
import numpy as np

# 定义函数 f(x) = (x - 3.5)**2 - 4.5*x + 10
f = lambda x: (x - 3.5)**2 - 4.5*x + 10
```

接下来，我们生成一组x值，并计算对应的y值，以便绘制函数图像。

```python
# 生成从0到11.5的100个等间距点
x = np.linspace(0, 11.5, 100)
y = f(x)

![](img/6fab85f46e33c98171581644823f6782_3.png)

# 绘制函数图像
plt.plot(x, y)
plt.show()
```

执行代码后，我们可以看到一条抛物线的图像。现在的问题是：如何求这个方程的最小值？

## 通过导数求解最小值

![](img/6fab85f46e33c98171581644823f6782_5.png)

根据数学知识，求解函数最小值的一个方法是求其导函数，并令导函数为零。

![](img/6fab85f46e33c98171581644823f6782_7.png)

![](img/6fab85f46e33c98171581644823f6782_9.png)

上一节我们构建了函数，本节中我们来看看如何求它的导数。对于函数 `f(x) = (x - 3.5)**2 - 4.5*x + 10`，其导函数 `f'(x)` 可以通过求导规则计算得出。

**导函数公式推导如下：**
1.  对 `(x - 3.5)**2` 求导：`2 * (x - 3.5)`
2.  对 `-4.5*x` 求导：`-4.5`
3.  常数 `10` 的导数为 `0`

因此，导函数为：
**`f'(x) = 2*(x - 3.5) - 4.5`**

为了找到最小值点，我们令导函数为零并求解 `x`：
**`2*(x - 3.5) - 4.5 = 0`**
**`2x - 7 - 4.5 = 0`**
**`2x = 11.5`**
**`x = 5.75`**

![](img/6fab85f46e33c98171581644823f6782_11.png)

![](img/6fab85f46e33c98171581644823f6782_13.png)

计算得出，当 `x = 5.75` 时，函数取得最小值。我们可以在图中将这个点标记出来。

```python
# 计算最小值点的坐标
min_x = 5.75
min_y = f(min_x)

# 在图像上标出最小值点
plt.plot(x, y)
plt.scatter(min_x, min_y, color='red', s=30) # s代表点的大小
plt.show()
```

![](img/6fab85f46e33c98171581644823f6782_15.png)

在生成的图像中，红色的点即为计算得到的最小值点，它确实位于抛物线的最低点。

## 方法总结与思考

![](img/6fab85f46e33c98171581644823f6782_17.png)

本节课中我们一起学习了通过求导来解析地求解函数最小值的方法。这种方法在拥有明确函数表达式时非常直接和高效。

然而，在机器学习的很多实际场景中，我们面对的往往不是现成的函数公式，而是一堆数据。我们的目标是找到能最好地拟合这些数据的规律（即函数）。这时，我们就无法直接使用上述“求导令其为零”的方法，因为没有明确的公式可供求导。这就需要引入新的优化算法，例如我们将在后续课程中详细讲解的**梯度下降法**。梯度下降法的核心思想正是借鉴了“沿着导数（梯度）方向寻找最小值”这一原理，但它适用于更普遍、没有解析解的情况。

总结来说：
*   **有明确公式时**：求导并解方程是最高效的方法。
*   **只有数据时**：需要借助像梯度下降这样的迭代优化算法来寻找最优解。

![](img/6fab85f46e33c98171581644823f6782_19.png)

理解本节的基础数学原理，是掌握后续复杂优化算法的关键。
