## 数据加载与查看

上一节我们介绍了本节课的目标，本节中我们来看看如何加载和查看波士顿房价数据集。

首先，我们需要导入必要的库。

```python
import numpy as np
from sklearn import datasets
from sklearn.linear_model import LinearRegression
```

接下来，我们加载波士顿房价数据集。

```python
boston = datasets.load_boston()
```

加载的数据集是一个类似字典的对象，包含以下核心部分：
*   **`data`**: 特征数据，即影响房价的各项指标。
*   **`target`**: 目标值，即每个样本对应的房价。
*   **`feature_names`**: 特征名称，描述了`data`中每一列数据所代表的含义。

现在，我们分别提取这些数据。

```python
x = boston[‘data’]   # 特征数据
y = boston[‘target’] # 目标值（房价）
feature_names = boston[‘feature_names’] # 特征名称
```

为了理解数据，我们先查看特征名称。

```python
print(feature_names)
```

以下是特征名称的部分解释：
*   `CRIM`: 城镇人均犯罪率。
*   `NOX`: 氮氧化物浓度（空气污染指标）。
*   `TAX`: 每万美元的全额财产税税率。
这些指标都与房价存在潜在关系。

接着，我们查看特征数据`x`和目标值`y`的形状。

```python
print(x.shape)
print(y.shape)
```

输出结果`(506, 13)`表示共有506个房屋样本，每个样本有13个特征。输出结果`(506,)`表示有506个对应的房价目标值。这证实了特征数据与目标值是一一对应的关系。
