```python
# 假设我们创建包含噪声的多元一次方程数据
# X 是特征矩阵，y 是目标值
import numpy as np
np.random.seed(42)
num_samples = 100
num_features = 18

![](img/202154136ad71ac7bca403571354579b_13.png)

# 生成真实的系数和截距
true_theta = np.random.randn(num_features + 1)  # +1 是截距项
# 生成特征数据 X
X = np.random.randn(num_samples, num_features)
# 计算目标值 y，并添加噪声
y = X @ true_theta[1:] + true_theta[0] + np.random.randn(num_samples) * 0.1
```

上面这段代码创建了多元一次方程的数据。我们也可以创建八元或更复杂的十八元一次方程。

## 实现随机梯度下降 ⚙️

![](img/202154136ad71ac7bca403571354579b_15.png)

数据创建好后，我们来实现随机梯度下降算法。

![](img/202154136ad71ac7bca403571354579b_17.png)

以下是随机梯度下降的核心代码框架：

```python
def stochastic_gradient_descent(X, y, learning_rate=0.01, epochs=100):
    m, n = X.shape
    # 初始化参数，包含截距项
    theta = np.random.randn(n + 1)
    # 为X添加一列1，用于计算截距
    X_b = np.c_[np.ones((m, 1)), X]

    for epoch in range(epochs):
        for i in range(m):
            # 随机选取一个样本
            random_index = np.random.randint(m)
            xi = X_b[random_index:random_index+1]  # 保持二维结构
            yi = y[random_index:random_index+1]
            # 计算梯度
            gradients = 2 * xi.T.dot(xi.dot(theta) - yi)
            # 更新参数，学习率可随时间衰减
            eta = learning_rate / (epoch + 1)  # 逆时衰减
            theta = theta - eta * gradients.flatten()
    return theta
```

大家看这段代码，这就是随机梯度下降的代码。我们将它应用到我们的数据上。
