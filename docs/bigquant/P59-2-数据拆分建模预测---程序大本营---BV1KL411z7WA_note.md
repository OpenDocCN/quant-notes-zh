## 数据拆分

![](img/34e8b83fe9331215f248b104d2e8eda2_9.png)

上一节我们介绍了如何加载波士顿房价数据集。本节中，我们来看看如何将数据拆分为训练集和测试集。

![](img/34e8b83fe9331215f248b104d2e8eda2_11.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_13.png)

我们总共有506个样本数据。拆分的作用是将数据分为两部分：一部分用于训练模型，另一部分用于验证模型效果。通常，我们会将80%的数据作为训练数据，20%的数据作为验证数据。

训练数据用于让算法（线性回归）学习数据特征与房价之间的关系。验证数据则用于测试训练好的模型是否准确，模型预测越准确，说明算法效果越好。

![](img/34e8b83fe9331215f248b104d2e8eda2_15.png)

以下是数据拆分的具体步骤：

![](img/34e8b83fe9331215f248b104d2e8eda2_17.png)

1.  **创建并打乱索引**：首先，我们为所有数据样本创建索引（0到505），然后使用 `np.random.shuffle()` 方法打乱索引顺序，以确保数据随机分布。

    ```python
    import numpy as np
    # 创建数据索引
    index = np.arange(506)
    # 打乱索引顺序
    np.random.shuffle(index)
    ```

![](img/34e8b83fe9331215f248b104d2e8eda2_19.png)

2.  **划分训练集与测试集**：根据80%和20%的比例，从打乱后的索引中切分出训练索引和测试索引。

    ```python
    # 计算训练数据数量（80%）
    train_size = int(506 * 0.8)  # 约为405
    # 获取训练集索引
    train_index = index[:train_size]
    # 获取测试集索引（剩余的20%）
    test_index = index[train_size:]
    ```

![](img/34e8b83fe9331215f248b104d2e8eda2_21.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_23.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_25.png)

3.  **根据索引提取数据**：使用划分好的索引，从原始特征数据 `x` 和目标值 `y` 中提取出对应的训练集和测试集。

    ```python
    # 提取训练数据
    x_train = x[train_index]
    y_train = y[train_index]
    # 提取测试数据
    x_test = x[test_index]
    y_test = y[test_index]
    ```

![](img/34e8b83fe9331215f248b104d2e8eda2_27.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_29.png)

4.  **检查数据形状**：打印输出数据的形状，以确认拆分是否正确。

    ```python
    print(f"训练数据形状: {x_train.shape}, {y_train.shape}")
    print(f"测试数据形状: {x_test.shape}, {y_test.shape}")
    ```
    输出结果应类似于：训练数据 (405, 13), (405,)；测试数据 (101, 13), (101,)。

数据拆分完成后，我们就得到了用于模型训练和验证的两部分数据。

## 数据建模 🧮

数据拆分完成后，接下来我们进行数据建模。建模的本质就是使用算法（线性回归）从训练数据中学习，得到一个能描述特征与目标值之间关系的方程（模型）。

我们使用 `LinearRegression` 算法进行建模。该算法默认会计算截距项。

以下是建模的具体步骤：

![](img/34e8b83fe9331215f248b104d2e8eda2_39.png)

1.  **导入并创建模型**：从 `sklearn.linear_model` 中导入线性回归类，并创建一个模型实例。

    ```python
    from sklearn.linear_model import LinearRegression
    # 创建线性回归模型，允许计算截距
    model = LinearRegression(fit_intercept=True)
    ```

![](img/34e8b83fe9331215f248b104d2e8eda2_41.png)

2.  **训练模型**：调用模型的 `fit` 方法，将训练特征 `x_train` 和训练目标值 `y_train` 传入，让算法进行学习。

    ```python
    # 使用训练数据拟合模型
    model.fit(x_train, y_train)
    ```

![](img/34e8b83fe9331215f248b104d2e8eda2_43.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_45.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_47.png)

3.  **查看模型参数**：训练完成后，模型会得到一组系数（斜率）和一个截距，它们共同构成了预测方程 `y = w1*x1 + w2*x2 + ... + wn*xn + b`。

    ```python
    # 获取模型的系数（斜率，对应每个特征）
    coefficients = model.coef_
    # 获取模型的截距
    intercept = model.intercept_
    print("模型系数:", coefficients)
    print("模型截距:", intercept)
    ```

    系数有正有负，其含义如下：
    *   **正系数**：表示正相关。例如，房屋面积（`RM`）越大，房价通常越高。
    *   **负系数**：表示负相关。例如，犯罪率（`CRIM`）或氮氧化物浓度（`NOX`）越高，房价通常越低。

    我们可以将系数与数据集的特征名称对应起来，以验证其合理性。

    ```python
    # 假设 features 是特征名称列表
    for feature, coef in zip(features, coefficients):
        print(f"{feature}: {coef:.4f}")
    ```

## 模型验证与应用 🔍

![](img/34e8b83fe9331215f248b104d2e8eda2_61.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_63.png)

我们已经训练好了一个模型（方程）。本节中，我们使用预留的20%测试数据来验证这个模型的效果，即看它的预测是否准确。

![](img/34e8b83fe9331215f248b104d2e8eda2_65.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_67.png)

以下是模型验证与应用的步骤：

![](img/34e8b83fe9331215f248b104d2e8eda2_69.png)

1.  **进行预测**：调用训练好的模型的 `predict` 方法，对测试集特征 `x_test` 进行预测，得到预测房价 `y_pred`。

    ```python
    # 使用模型对测试数据进行预测
    y_pred = model.predict(x_test)
    # 将预测结果保留两位小数，便于查看
    y_pred_rounded = np.round(y_pred, 2)
    ```

    在机器学习中，预测值通常用 `y_pred` 或 `y_hat` 表示。

![](img/34e8b83fe9331215f248b104d2e8eda2_71.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_73.png)

2.  **对比预测值与真实值**：将模型预测的前若干个结果与测试集中的真实房价进行对比，直观感受预测准确性。

    ```python
    # 查看前30个预测结果和真实值
    print("预测值 (前30个):", y_pred_rounded[:30])
    print("真实值 (前30个):", y_test[:30])
    ```

    通过对比可以发现，预测值与真实值存在一定波动，但大体趋势相近。这是因为房价受多种复杂因素影响，模型无法做到100%精确，但能提供一个可靠的估值参考。

![](img/34e8b83fe9331215f248b104d2e8eda2_75.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_77.png)

![](img/34e8b83fe9331215f248b104d2e8eda2_79.png)

3.  **理解预测误差**：预测出现偏差是正常现象。误差可能来源于数据本身（如房屋的特殊情况、标价策略）、模型局限性以及未考虑到的隐藏因素。专业的房地产平台正是利用此类算法为房屋进行估价评估。

## 总结

本节课中我们一起学习了机器学习项目中的关键三步：

1.  **数据拆分**：将数据集按比例（如8:2）随机划分为训练集和测试集，为模型训练和评估做好准备。
2.  **数据建模**：使用线性回归算法对训练集进行学习，得到一个包含系数和截距的预测模型，并理解了系数正负的意义。
3.  **模型验证**：使用测试集对训练好的模型进行预测，并通过对比预测值与真实值来评估模型的实用性和准确性。

![](img/34e8b83fe9331215f248b104d2e8eda2_83.png)
