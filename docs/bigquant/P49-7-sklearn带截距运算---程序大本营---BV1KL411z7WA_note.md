在本节课中，我们将学习如何使用 `sklearn` 库中的 `LinearRegression` 模型来计算带截距的线性方程。我们将通过一个具体的八元一次方程例子，演示如何从无截距模型过渡到有截距模型，并理解其背后的数学原理和代码实现。

---

## 带截距的线性方程计算

上一节我们介绍了无截距线性方程的求解。本节中，我们来看看如何为方程添加一个截距项。

在约定俗成的表示中，我们通常将特征变量写作大写的 **X**，将目标值写作小写的 **y**。这类似于初中方程 `y = kx + b` 的写法，其中 `x` 是变量，`y` 是目标值。代码中的变量名大小写不影响计算结果，但遵循惯例有助于提高代码的可读性。

### 构建带截距的方程

我们以上一节的八元一次方程为基础，为其添加一个截距项。原方程求解出的系数为 `[1, 5, 15, 3, 8, 4, 17, 12]`。

原方程可以表示为：
**`f(x) = x₁ + 5*x₂ + 15*x₃ + 3*x₄ + 8*x₅ + 4*x₆ + 17*x₇ + 12*x₈`**

现在，我们为这个方程统一加上一个截距 `12`。这意味着每个预测值都会增加 `12`，因此目标值 `y` 也需要相应增加 `12`。新的目标值 `y` 变为 `[351, -100, 24, 12, 138, -383]`（具体数值根据原 `y` 计算）。

### 使用 sklearn 拟合带截距模型

在 `sklearn` 的 `LinearRegression` 中，有一个参数 `fit_intercept` 专门用于控制是否计算截距。对于这个新的带截距方程，我们将其设置为 `True`。

以下是核心代码步骤：

```python
# 导入库并创建模型，指定计算截距
from sklearn.linear_model import LinearRegression
model = LinearRegression(fit_intercept=True)

# 使用新的 X 和 y 进行拟合（训练）
model.fit(X, y_new)

# 查看结果
print("截距 (intercept_):", model.intercept_)
print("系数 (coef_):", model.coef_)
```

运行上述代码后，模型会计算出新的系数和截距。你可能会发现，此时计算出的截距并不是我们添加的 `12`，系数也与原系数不同。这是正常的，因为当 `fit_intercept=True` 时，`sklearn` 并未使用我们之前提到的正规方程直接求解，而是采用了其他数值计算方法。

### 验证模型有效性

尽管系数和截距变了，但新模型仍然是原方程的一个有效解。我们可以通过以下计算来验证：

```python
# 使用模型预测的结果
y_pred = model.predict(X)

# 手动计算：X * coef_ + intercept_
y_manual = X.dot(model.coef_) + model.intercept_

# 比较 y_pred 和 y_manual，它们应该与新的 y_new 完全相等
print("预测值 y_pred 是否等于 y_new:", np.allclose(y_pred, y_new))
print("手动计算值是否等于 y_new:", np.allclose(y_manual, y_new))
```

运行代码后，你会发现 `y_pred` 或 `y_manual` 与新的目标值 `y_new` 完全一致。这证明模型求解出的系数和截距是有效的，它们共同构成了原方程的一个等价表示。

## 通过修改数据 X 来包含截距

上一节我们通过设置 `fit_intercept=True` 让模型计算截距。本节中，我们换一种思路：通过修改输入数据 `X` 来“手动”包含截距项，并强制模型使用正规方程求解。

### 理解截距的另一种表示

在线性方程 `y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b` 中，截距 `b` 可以看作是特征 `x₀` 的系数，其中 `x₀` 恒等于 `1`。因此，方程可以重写为：
**`y = w₀*x₀ + w₁x₁ + w₂x₂ + ... + wₙxₙ`**，其中 **`x₀ = 1`**，**`w₀ = b`**。

### 修改数据矩阵 X

根据上述原理，我们可以在原始特征矩阵 `X` 的右侧添加一列值全为 `1` 的列，来代表 `x₀`。

以下是修改 `X` 的代码：

```python
import numpy as np

# 假设原始 X 是 (m, n) 的矩阵
# 创建一个形状为 (m, 1) 且值全为1的列
ones_column = np.ones((X.shape[0], 1))

![](img/4ec4a4d5c90903744b97e783869f2ab8_1.png)

# 将这一列拼接到原始 X 的右侧（沿 axis=1 方向连接）
X_modified = np.concatenate([X, ones_column], axis=1)

![](img/4ec4a4d5c90903744b97e783869f2ab8_3.png)

# 显示修改后的 X 和新的 y
print("修改后的 X (最后一列为1):\n", X_modified)
print("新的目标值 y:\n", y_new)
```

![](img/4ec4a4d5c90903744b97e783869f2ab8_5.png)

执行后，`X_modified` 的最后一列将全部变为 `1`。

### 使用正规方程求解

现在，我们使用修改后的 `X_modified` 和新的 `y_new`，并将 `LinearRegression` 的 `fit_intercept` 参数设置为 `False`（因为截距已包含在数据中）。理论上，这应该促使模型使用类似正规方程的方法直接求解。

```python
# 创建模型，指定不单独计算截距
model2 = LinearRegression(fit_intercept=False)
model2.fit(X_modified, y_new)

# 查看求解出的系数，最后一个系数就是截距 b
print("模型系数 (包含截距):", model2.coef_)
# 注意：此时 model2.intercept_ 将为 0，因为截距已包含在 coef_ 中
```

同时，我们可以直接使用正规方程公式来验证：
**`W = (X^T * X)^(-1) * X^T * y`**

```python
# 使用正规方程手动计算系数 W (包含截距项)
W = np.linalg.inv(X_modified.T.dot(X_modified)).dot(X_modified.T).dot(y_new)
print("正规方程求解的系数 W:", W)
```

理想情况下，`model2.coef_` 和手动计算的 `W` 应该一致，并且最后一个值接近我们最初添加的截距 `12`。在实际操作中，由于数值计算精度等原因，结果可能略有差异，但原理是相通的。

## 课程总结与核心公式

本节课中我们一起学习了带截距的线性回归计算。

我们掌握了两种处理截距的方法：
1.  **使用 `fit_intercept=True`**：让 `sklearn` 的 `LinearRegression` 自动计算截距，这是最简便的方法。
2.  **修改输入数据**：在特征矩阵中增加一列 `1`，将截距作为其中一个特征系数来求解，并通过设置 `fit_intercept=False` 来配合此操作。

![](img/4ec4a4d5c90903744b97e783869f2ab8_7.png)

![](img/4ec4a4d5c90903744b97e783869f2ab8_9.png)

本节课的核心是理解正规方程，它是线性回归的基石之一。其公式如下：

**`W = (X^T X)^{-1} X^T y`**

![](img/4ec4a4d5c90903744b97e783869f2ab8_11.png)

其中：
*   **`W`** 是包含所有权重系数（`w₁, w₂, ..., wₙ`）和截距（`b`，如果数据中包含 `x₀=1` 的列）的向量。
*   **`X`** 是特征矩阵（可能已包含一列 `1`）。
*   **`y`** 是目标值向量。
*   **`X^T`** 表示 `X` 的转置矩阵。
*   **`( )^{-1}`** 表示矩阵的逆。

`sklearn` 的 `LinearRegression` 在内部封装并优化了这个求解过程。理解这个公式有助于你深入理解模型是如何从数据中“学习”到规律的。
