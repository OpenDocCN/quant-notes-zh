## 回归问题与分类问题

机器学习任务主要分为两类：分类问题和回归问题。
*   **分类问题**：预测结果是离散的类别。例如，判断一个人是“男”还是“女”。
*   **回归问题**：预测结果是连续的趋势或数值。例如，预测一个孩子在不同年龄时的身高或体重。

决策回归树与决策分类树结构相似，但其分裂指标不同。

## 回归树的分裂指标

决策回归树使用以下指标来衡量分裂的好坏：

以下是两种常用的回归树分裂指标：

1.  **均方误差**：这是最常用的指标。它计算所有样本的真实值与预测值之差的平方的平均值。
    *   **公式**：`MSE = (1/n) * Σ(y_i - ŷ_i)^2`
    *   其中，`y_i` 是真实值，`ŷ_i` 是预测值，`n` 是样本数量。

![](img/f7ccc4084b4cdc7a90cf20eeee0ba7a0_3.png)

2.  **平均绝对误差**：它计算所有样本的真实值与预测值之差的绝对值的平均值。
    *   **公式**：`MAE = (1/n) * Σ|y_i - ŷ_i|`

相比之下，决策分类树的分裂指标是**信息熵**、**信息增益**或**信息增益率**。

## 回归树的本质：细粒度的分类

我们可以将决策回归树理解为一种**细粒度的分类问题**。分类问题（如男女）类别明确且有限。而回归问题（如身高）的预测值是连续的，但决策树会通过多次分裂，将连续的取值区间划分为多个细小的子区间。树的深度决定了划分的精细程度。

以下是决策树结构的核心概念：
*   **根节点**：树的起始点。
*   **子节点**：由父节点分裂产生的中间节点。
*   **叶节点**：树的末端节点，代表最终的预测结果（分类的类别或回归的取值区间）。

## 决策树与线性回归的对比

与线性回归模型不同，决策树（无论是分类还是回归）**没有“方程系数”** 的概念。决策树算法本身就是一个由节点和分支构成的**树形结构**，这个结构就是模型本身。而线性回归模型则是一个带有系数的数学方程（例如 `y = w*x + b`）。
