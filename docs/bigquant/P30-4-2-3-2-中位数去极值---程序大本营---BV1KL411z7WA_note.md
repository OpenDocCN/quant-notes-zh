# 量化数据处理教程：4.2.3.2：中位数去极值 📊

![](img/319958d64e801f063fcf3c64a5c5011b_0.png)

在本节课中，我们将学习如何使用中位数去极值法处理量化投资中的因子数据。这种方法可以有效减少极端值对数据分析的影响，使数据分布更稳健。

## 概述
中位数去极值是一种常见的数据预处理方法。它通过计算每个截面上数据的中位数和绝对偏差，将超出特定范围的极端值替换为边界值。本节课将介绍如何在量化平台中应用此方法。

## 模块介绍与连接
上一节我们介绍了因子抽取的基本概念，本节中我们来看看如何对抽取的因子进行去极值处理。

量化平台提供了丰富的数据处理模块。我们可以使用“去极值处理模块”来实现中位数去极值法。操作时，需要将抽取的因子数据连接到该模块。

具体有两种方式指定需要处理的列：
*   通过连接“输入特征列表模块”来指定。
*   直接在去极值模块的“指定列”参数中手动填入列名。

## 操作步骤演示
以下是构建一个简单去极值策略的具体步骤。

1.  在策略画布中，依次拖入“证券代码列表模块”、“输入特征列表模块”和“基础特征抽取模块”。
2.  在左侧导航栏搜索“去极值”，找到并拖入“去极值处理模块”。
3.  配置“证券代码列表模块”，设置时间范围为2019年1月1日至2019年5月1日，股票范围选择“全市场”。
4.  在“输入特征列表模块”中输入 `return_20`，这代表我们要抽取过去20日的收益率因子。
5.  将“基础特征抽取模块”连接到“去极值处理模块”，作为其输入数据源。
6.  将“输入特征列表模块”连接到“去极值处理模块”的另一个输入端，以指定需要处理的列名 `return_20`。
    *   **替代方案**：也可以断开此连接，直接在去极值模块的“指定列”参数中手动填入 `return_20`。
7.  在去极值模块中，设置“标准差的倍数”来定义极值的边界。例如，设置为3表示将超出中位数±3倍绝对偏差的值视为极值。
8.  运行所有模块。

## 效果对比与总结
运行完成后，我们可以对比处理前后的数据。

查看“基础特征抽取模块”的输出数据，`return_20` 因子的原始中位数为1.02，最大值为5.81。
再查看“去极值处理模块”处理后的数据，`return_20` 因子的最大值变为1.76。

从数据分布图可以看出，经过中位数去极值处理后的因子数据，其长尾效应得到了明显减弱，数据变得更加集中和稳健。
