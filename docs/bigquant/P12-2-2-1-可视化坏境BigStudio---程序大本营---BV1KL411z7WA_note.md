# 量化交易入门：P12：2.2.1：可视化环境BigStudio 🖥️

在本节课中，我们将要学习BigStudio，这是被框平台提供的可视化编程界面。我们将了解如何访问、注册并开始使用这个环境来构建量化交易策略。

![](img/dde7257796ae6cfe96a19bbc1be460d6_1.png)

## 访问与注册

上一节我们介绍了BigStudio的概念，本节中我们来看看如何进入这个环境。

首先，通过登录网站 `www.beikuang.com`，注册账号并登录。点击右侧的“免费注册”按钮，可以进行账号的申请。申请账号后可以进行登录。随后，点击“编写策略”按钮，进入策略开发界面。

![](img/dde7257796ae6cfe96a19bbc1be460d6_3.png)

## 界面概览

成功进入策略开发界面后，我们可以点击“新建”按钮。在弹窗中选择“可视化AI策略”，即可进入BigStudio的策略开发环境。

观察网页页面，可以发现页面从左至右分为四个区域。

以下是四个主要区域的介绍：
*   **策略列表区**：管理你创建的所有策略。
*   **BigStudio的模块区**：提供各种可拖拽的功能模块。
*   **Fix Studio的画布区**：通过拖拽模块并连接它们来构建策略逻辑的核心区域。
*   **模块参数和帮助文档区**：用于配置所选模块的具体参数，并查看相关说明。

我们通过点击模块参数，可以实现参数的修改。

## 界面操作技巧

在页面中，可以发现图中绿色的方块内有一些箭头。我们可以通过点击这些按钮和箭头，来实现这四个面板区域的展开或者收缩。

在画布的下方，可以看到这里有一个导航栏。

以下是导航栏的主要功能：
*   通过 **加号（+）** 和 **减号（-）**，可以进行画布的缩放。
*   通过 **一比一（1:1）** 按钮，实现实际尺寸的显示。
*   此外，还可以通过拖拽实现画布的移动。
