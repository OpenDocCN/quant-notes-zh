## 信息熵：衡量数据的“混乱程度” 🔀

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_8.png)

上一节我们介绍了决策树的基本结构，本节中我们来看看决策树构建所依据的标准。这个标准就是**信息熵**。

信息熵源自热力学第二定律中的“熵”概念，在计算机科学中，它用于描述一个系统或数据集的“混乱”或“不确定性”程度。想象一个容器中最初完全分离的红球和绿球，此时系统非常有序。当我们摇晃容器，红球和绿球逐渐混合，系统就变得越来越混乱。信息熵就是用来量化这种混乱程度的指标。

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_10.png)

香农为信息熵给出了明确的定义和计算公式：

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_12.png)

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_14.png)

**公式：**
`H(X) = - Σ [p(x) * log₂(p(x))]`

我们对这个公式进行说明：
*   `Σ` 是求和符号。
*   `p(x)` 表示某个类别或事件出现的**概率**，其值在0到1之间。
*   `log₂(p(x))` 是以2为底的对数。
*   公式前有一个负号，这是因为当 `p(x)` 在0到1之间时，`log₂(p(x))` 的结果为负数或零。加上负号是为了让最终的信息熵值保持为正数，便于理解和比较。

简单来说，一个数据集越“纯”（例如，所有样本都属于同一类别），其信息熵就越低（接近0）。反之，数据集越“混乱”（各类别样本均匀混合），其信息熵就越高。

## 信息增益：衡量划分带来的“有序性提升” 📈

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_18.png)

理解了信息熵后，我们引入**信息增益**这个概念。它是决策树选择划分特征的关键依据。

信息增益的定义是：在知道了某个特征（条件）的信息之后，原数据集的不确定性（熵）**下降的程度**。信息增益越大，说明使用这个特征进行划分，能让数据变得越有序。

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_20.png)

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_22.png)

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_24.png)

其计算公式为：

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_26.png)

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_28.png)

**公式：**
`Gain(Y, X) = H(Y) - H(Y|X)`

其中：
*   `Gain(Y, X)` 是特征X对于数据集Y的信息增益。
*   `H(Y)` 是划分前数据集Y的原始信息熵。
*   `H(Y|X)` 是在已知特征X的条件下，数据集Y的条件熵。它等于根据特征X的每个取值划分出的子集熵的加权平均。

![](img/1d461d72a2b27de4ddaab4a2f91ed5ef_30.png)

这个公式的含义很直观：用划分前的“混乱度”减去划分后的平均“混乱度”，得到的就是这次划分带来的“有序度提升”，即信息增益。决策树算法在每一步都会计算所有可用特征的信息增益，然后选择**信息增益最大**的特征作为当前节点的划分标准。
