# 量化分析：P21：4.1.5：时间序列可视化实操 📊

在本节课中，我们将学习如何使用可视化工具对财务时间序列数据进行探索性分析。我们将通过一系列步骤，从数据加载到多种图形绘制，来观察和理解数据的分布特征与变量间的关系。

![](img/08d1575ad4a589a1973db88c2381afb3_1.png)

## 数据加载与预处理

首先，我们需要加载数据并进行基本的预处理。在可视化界面中，拖入“数据模块”、“正确代码列表模块”和“输入特征列表模块”。

以下是具体操作步骤：
1.  设置“数据模块”的数据源，表名为 `financial_semf_f_c_stop_a`，用于抽取经过向后填充处理的财务数据。
2.  设置“正确代码列表模块”的开始和结束时间均为 `2017-04-05`，以抽取该日全股票市场的财务数据。
3.  在“输入特征列表模块”中，填入需要抽取的财务字段。这里我们抽取 `roe`、`le`、`eps` 和 `cash_ratio` 四个字段。
4.  对抽取的数据进行缺失值处理，即删除所有包含缺失数据的行。

![](img/08d1575ad4a589a1973db88c2381afb3_3.png)

完成数据加载和清洗后，我们就可以开始进行可视化分析了。

![](img/08d1575ad4a589a1973db88c2381afb3_5.png)

![](img/08d1575ad4a589a1973db88c2381afb3_6.png)

## 单变量分布分析

![](img/08d1575ad4a589a1973db88c2381afb3_8.png)

上一节我们完成了数据准备，本节中我们来看看如何分析单个变量的分布情况。

我们通过模块搜索功能，找到“绘制单变量直方图模块”并将其连接到数据处理流程之后。

![](img/08d1575ad4a589a1973db88c2381afb3_10.png)

以下是该模块的参数设置：
*   **绘制的列名**：设置为 `le`。
*   **绘制概率密度分布图**：勾选此选项。
*   **分组的数量**：设置为 `1000`。

![](img/08d1575ad4a589a1973db88c2381afb3_12.png)

右键选中该模块并点击“运行”。运行后，该模块会绘制出 `le` 字段对应的直方分布图以及叠加的概率密度曲线图，这有助于我们直观地了解该财务指标的数值分布范围和集中趋势。

![](img/08d1575ad4a589a1973db88c2381afb3_14.png)

## 双变量关系分析

了解了单个变量的分布后，我们进一步探索两个变量之间的关系。

![](img/08d1575ad4a589a1973db88c2381afb3_16.png)

我们拖入“绘制双变量联合分布模块”并进行连接。在该模块中，我们设置第一个变量和第二个变量均为 `roe`，并指定绘图类型为“线性回归”。

![](img/08d1575ad4a589a1973db88c2381afb3_18.png)

![](img/08d1575ad4a589a1973db88c2381afb3_19.png)

右键点击该模块并运行。模块将绘制出 `roe` 与自身（或可设置为其他变量）的联合分布图，并以线性回归线展示其相关性趋势。

## 多变量联合分布分析

![](img/08d1575ad4a589a1973db88c2381afb3_21.png)

为了同时观察多个变量之间的复杂关系，我们可以使用更高级的多变量可视化工具。

我们拖入“绘制多变量联合分布组图”模块，将其连接到缺失值处理模块和输入特征列表模块，并设置对应的绘图类型。

以下是绘图类型的设置：
*   **对角线图类型**：设置为“直方图分布”。
*   **上三角图类型**：设置为“概率密度分布”。
*   **下三角图类型**：设置为“散点图分布”。

运行该模块后，可以看到一个矩阵式的图形结果。对角线是各个变量自身的直方图与密度曲线；左下三角区域展示了任意两个变量之间的散点图关系；右上三角区域则展示了变量对的概率密度分布。

## 分组数据可视化

![](img/08d1575ad4a589a1973db88c2381afb3_23.png)

在实际分析中，我们经常需要按照某个维度（如时间）对数据进行分组观察。

![](img/08d1575ad4a589a1973db88c2381afb3_25.png)

我们新增一个“输入特征列表模块”，并填入分组所需的字段：`fs_quarter_year`（财报年份）和 `fs_quarter_index`（财报期号，如季报编号）。然后拖入“绘制分组分布图模块”并进行连接。

以下是该模块的参数设置：
*   **X轴数据列名**：指定为 `fs_quarter_year`，即按年度分组。
*   **Y轴数据列名**：指定为 `fs_le`，即绘制 `le` 指标的分布。
*   **图展示类型**：可以选择 `strip`（条状图）、`box`（箱型图）、`violin`（小提琴图）或 `scatter`（散点图）。
*   **数据分组的列名**：指定为 `fs_quarter_index_0`。这表示在绘制每个财务年度的数据时，内部会再按照财报期号进行分组并以不同颜色展示。

![](img/08d1575ad4a589a1973db88c2381afb3_27.png)

运行模块后，可以看到在2017年4月5日，不同股票的财务数据状态：部分股票是2016年年报数据，部分则是2016年三季报数据。它们通过不同的颜色在图中得到了区分。

![](img/08d1575ad4a589a1973db88c2381afb3_29.png)

![](img/08d1575ad4a589a1973db88c2381afb3_30.png)

我们可以重新勾选“箱型图”类型并运行，此时数据会以箱型图的形式重新展示，能够更清晰地看到不同期号下 `le` 指标的中位数、四分位数及离散程度。

![](img/08d1575ad4a589a1973db88c2381afb3_31.png)
