# 量化数据处理：P31：4.2.3.3-标准化处理 📊

![](img/c90ff5406e8f4545bf54a4f1f292c222_0.png)

在本节课中，我们将学习如何使用标准化处理模块，对经过去极值处理后的因子数据进行截面标准化。标准化是量化数据处理中的关键步骤，旨在消除不同因子间的量纲差异，使其具有可比性。

## 模块连接与配置 🔌

上一节我们介绍了去极值处理，本节中我们来看看如何对处理后的数据进行标准化。首先，我们需要将去极值处理模块的输出数据连接到标准化处理模块。

![](img/c90ff5406e8f4545bf54a4f1f292c222_2.png)

具体操作步骤如下：
1.  在左侧模块搜索列表中，输入“标准化”并找到“标准化处理”模块。
2.  将该模块拖入画布。
3.  将“去极值处理”模块的输出节点，连接到“标准化处理”模块的输入节点。

## 指定标准化列 📝

连接模块后，我们需要指定哪些因子列需要进行标准化处理。标准化处理模块提供了两种方式来指定目标列。

以下是两种指定方式：
*   **方式一：手动指定列名**。在模块的参数配置面板中，直接输入需要标准化的列名。
*   **方式二：传入特征列表**。通过连接一个包含特征列名的列表输入节点，动态地指定需要标准化的因子。

配置完成后，运行标准化处理模块。

## 结果验证 ✅

![](img/c90ff5406e8f4545bf54a4f1f292c222_4.png)

模块运行完毕后，我们可以点击“查看结果”来检查标准化处理的效果。标准化旨在将数据转换为均值为0、标准差为1的分布。

例如，我们查看名为 `return_20` 的列，可以发现：
*   该列数据的平均值非常接近 **0**。
*   该列数据的标准差非常接近 **1**。

这表明标准化处理已成功执行，该因子数据已被转换到标准正态分布尺度上。

![](img/c90ff5406e8f4545bf54a4f1f292c222_6.png)

## 总结 📋

本节课中我们一起学习了标准化处理的操作流程。我们首先将去极值处理后的数据连接到标准化模块，然后通过指定列名或传入特征列表的方式选择需要处理的因子，最后运行模块并验证结果，确认数据已被转换为均值为0、标准差为1的标准形式。这一步为后续的因子合成与模型训练奠定了数据基础。
