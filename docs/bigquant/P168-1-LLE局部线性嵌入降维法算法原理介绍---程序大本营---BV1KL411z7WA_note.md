上一节我们介绍了线性判别分析，本节中我们来看看另一种降维方法——局部线性嵌入。LLE算法特别擅长处理像“瑞士卷”这样具有非线性流形结构的数据，其核心思想是保持数据在降维前后的局部线性关系。

## 算法核心思想

LLE算法关注于降维时保持样本局部的线性特征，这与PCA和LDA关注全局方差或类间区分度不同。由于LLE在降维时保持了样本的局部特性，它被广泛应用于图像识别和高维数据可视化。LLE是一种非线性的降维技术，而PCA和LDA属于线性的降维技术，可以说LLE是流形学习领域最经典的算法之一。

## 什么是流形？

在传统机器学习中，数据点之间的距离和映射通常定义在欧式空间。然而在实际中，数据可能分布在更复杂的空间，例如球面（黎曼空间）。在非欧式空间中，传统的距离度量可能失效。LLE算法在处理这类非欧式空间中的数据时，有可能得到更好的结果。

## LLE算法主要步骤

LLE算法的实现主要分为三步，以下是具体步骤：

1.  **寻找近邻**：为数据集中的每一个样本点，寻找其k个最近的邻居点。
2.  **计算重建权重**：对于每个样本点，利用它的k个近邻点，计算出一个局部重建权重矩阵。这个权重矩阵表达了该样本点如何由其邻居线性表示。其数学目标是**最小化局部重建误差**。
    *   **公式**：对于样本点 `x_i`，其重建误差为 `|x_i - Σ_j (W_ij * x_j)|^2`，其中 `x_j` 是 `x_i` 的邻居，`W_ij` 是待求的权重，且满足 `Σ_j W_ij = 1`。
3.  **计算低维嵌入**：在低维空间中，保持步骤2中求得的权重关系不变，寻找所有样本点的低维坐标。其数学目标是**在低维空间中保持这些权重关系**。
    *   **公式**：最小化 `|y_i - Σ_j (W_ij * y_j)|^2`，其中 `y_i` 是 `x_i` 对应的低维坐标。

## 算法特点与应用

LLE算法广泛用于图像识别和高维数据可视化。它是非线性的降维技术，而PCA和LDA是线性的降维技术。LLE是流形学习方面最经典的算法之一。
