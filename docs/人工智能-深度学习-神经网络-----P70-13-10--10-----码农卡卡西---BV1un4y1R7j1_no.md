## 分类任务与特征提取

上一节我们介绍了时间序列的回归预测，本节中我们来看看分类任务。分类任务的核心在于特征提取。对于时间序列数据，我们需要先将其转化为一组可量化的特征，例如**最小值**、**最大值**、**均值**、**变化频率**等统计特征，然后才能使用分类模型进行处理。

## 引入tsfresh库

![](img/0c004b57da5463111d351b0a40ceac4a_5.png)

## 实战：时间序列分类

我们将通过一个工业设备监控的示例来演示整个过程。假设我们有多个传感器（A, B, C, D, E, F）采集的时间序列数据，目标是判断设备的工作状态是“正常”还是“异常”。

### 1. 加载与观察数据

![](img/0c004b57da5463111d351b0a40ceac4a_19.png)

首先，我们使用 `tsfresh` 库提供的示例数据集，并观察数据的基本形态。

```python
from tsfresh.examples import robot_execution_failures
df, y = robot_execution_failures.load_robot_execution_failures()
print(df.head())
```

![](img/0c004b57da5463111d351b0a40ceac4a_21.png)

数据通常包含以下列：
*   `id`: 每条时间序列的唯一标识。
*   `time`: 时间戳。
*   `A-F`: 六个传感器采集的数值。

![](img/0c004b57da5463111d351b0a40ceac4a_23.png)

![](img/0c004b57da5463111d351b0a40ceac4a_25.png)

我们可以通过绘图直观地对比正常与异常序列的差异。

![](img/0c004b57da5463111d351b0a40ceac4a_27.png)

### 2. 提取时间序列特征

![](img/0c004b57da5463111d351b0a40ceac4a_29.png)

![](img/0c004b57da5463111d351b0a40ceac4a_31.png)

接下来，我们使用 `tsfresh` 的核心功能进行特征提取。

以下是提取特征的核心代码：
```python
from tsfresh import extract_features
extracted_features = extract_features(df, column_id="id", column_sort="time")
print(extracted_features.head())
```

这段代码会以 `id` 为单位对时间序列进行分组，并按 `time` 排序，然后自动计算并生成一个包含大量特征（如均值、方差、峰值等）的新数据表。

### 3. 特征筛选与建模

![](img/0c004b57da5463111d351b0a40ceac4a_33.png)

![](img/0c004b57da5463111d351b0a40ceac4a_35.png)

提取出的特征数量可能非常庞大，其中包含一些与分类目标无关的特征。`tsfresh` 提供了特征筛选功能，可以帮助我们选择最相关的特征。

以下是进行特征筛选和决策树分类的示例代码：
```python
from tsfresh import select_features
from tsfresh.utilities.dataframe_functions import impute
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import classification_report

![](img/0c004b57da5463111d351b0a40ceac4a_37.png)

# 1. 处理缺失值
impute(extracted_features)

![](img/0c004b57da5463111d351b0a40ceac4a_39.png)

# 2. 筛选与标签y相关的特征
features_filtered = select_features(extracted_features, y)

![](img/0c004b57da5463111d351b0a40ceac4a_41.png)

![](img/0c004b57da5463111d351b0a40ceac4a_43.png)

# 3. 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(features_filtered, y, test_size=0.25)

![](img/0c004b57da5463111d351b0a40ceac4a_45.png)

![](img/0c004b57da5463111d351b0a40ceac4a_47.png)

# 4. 训练决策树分类器
clf = DecisionTreeClassifier()
clf.fit(X_train, y_train)

![](img/0c004b57da5463111d351b0a40ceac4a_49.png)

![](img/0c004b57da5463111d351b0a40ceac4a_51.png)

# 5. 评估模型
y_pred = clf.predict(X_test)
print(classification_report(y_test, y_pred))
```

通过以上步骤，我们就能完成一个完整的时间序列分类任务。
