在本节课中，我们将要学习夏普比率这一核心概念。夏普比率是衡量投资绩效的重要指标，它帮助我们理解在承担单位风险时，能获得多少超额收益。

## 夏普比率的含义

上一节我们介绍了风险的概念，本节中我们来看看如何量化风险与收益的关系。夏普比率描述的就是这样一个指标：对于每承担一单位的风险，我们能获得多少收益。

我们可以用一个生活中的例子来理解。例如，一份在战乱地区日薪极高的工作，其高薪对应的是极高的生命风险。夏普比率所做的，就是评估这种“风险”与“回报”是否“值当”。指标越高，意味着承担单位风险所获得的收益越高。

在选股或构建投资组合时，如果单从夏普比率这个指标来看，我们应选择比率更高的选项，因为它代表了更高的风险调整后收益。

## 夏普比率的计算逻辑

理解了夏普比率的含义后，我们来看看它是如何计算的。其核心思想是比较“有风险的投资”与“无风险的投资”之间的差异。

![](img/c4b38d40cd78c4fc403c8181651dfb9a_5.png)

例如，银行的无风险理财产品（如国债）可能提供5%的年化收益。而一个有一定风险的投资组合，其预期年化收益可能是15%。这两者之间存在10%的收益差。

夏普比率就是计算这个收益差（即超额收益），再除以投资组合收益的波动率（标准差）。波动率代表了风险的大小。

**夏普比率公式**如下：
`Sharpe Ratio = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差`

公式中的收益率和标准差通常基于相同的时间周期（如年化）。计算时，我们使用投资组合的历史收益率数据。

## 实战计算夏普比率

接下来，我们通过代码演示如何计算多只股票的夏普比率。在计算前，我们需要准备好股票的历史收益率数据，并对数据中的缺失值进行处理（例如，用前一天的收益率填充）。

以下是计算夏普比率的关键步骤：

1.  **获取收益率数据**：使用处理好的股票日收益率序列。
2.  **设定无风险利率**：例如，年化无风险利率设为5%。
3.  **计算年化超额收益率**：用股票的平均日收益率减去日化的无风险利率，再年化。
4.  **计算年化波动率**：计算股票日收益率的标准差，并年化。
5.  **计算夏普比率**：将年化超额收益率除以年化波动率。

```python
import numpy as np
import pandas as pd

![](img/c4b38d40cd78c4fc403c8181651dfb9a_7.png)

# 假设 returns 是一个包含多只股票日收益率的 DataFrame
# 例如：returns['stock_A'], returns['stock_B']...

# 设置年化无风险利率，例如 5%
risk_free_rate = 0.05
# 一年通常按250个交易日计算
trading_days_per_year = 250

# 计算每只股票的平均日收益率
mean_daily_returns = returns.mean()
# 计算每只股票日收益率的标准差
std_daily_returns = returns.std()

# 计算年化超额收益率
# 日化无风险利率 = 年化无风险利率 / 年交易天数
excess_return_annualized = (mean_daily_returns - risk_free_rate/trading_days_per_year) * trading_days_per_year

# 计算年化波动率（标准差）
volatility_annualized = std_daily_returns * np.sqrt(trading_days_per_year)

# 计算夏普比率
sharpe_ratios = excess_return_annualized / volatility_annualized

![](img/c4b38d40cd78c4fc403c8181651dfb9a_9.png)

print(sharpe_ratios)
```
