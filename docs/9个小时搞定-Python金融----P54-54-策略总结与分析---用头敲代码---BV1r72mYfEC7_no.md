在这个较长的时间段内，策略收益与基准收益相差不大，没有显示出明显的优势。这可能与选择的时间区间有关。

## 优化测试条件与对比分析

![](img/b933ad4c5325a8333902b9276bc0bce9_40.png)

![](img/b933ad4c5325a8333902b9276bc0bce9_42.png)

于是，我们选择一个市场环境可能更好的时间段进行测试：`2016-01-04`到`2017-01-04`。在这一年里，策略收益明显高于基准收益，结果更为理想。

![](img/b933ad4c5325a8333902b9276bc0bce9_44.png)

这个结果引出了一个思考：我们之前一直选择综合得分**排名前十**的股票。如果选择排名**靠后**的股票，即指标表现最差的股票，结果会怎样？

![](img/b933ad4c5325a8333902b9276bc0bce9_46.png)

![](img/b933ad4c5325a8333902b9276bc0bce9_48.png)

以下是调整股票选择范围的代码：

![](img/b933ad4c5325a8333902b9276bc0bce9_50.png)

![](img/b933ad4c5325a8333902b9276bc0bce9_52.png)

```python
# 原代码：选择排名前10的股票（得分最高）
selected_stocks = ranked_stocks.iloc[:10]

![](img/b933ad4c5325a8333902b9276bc0bce9_54.png)

![](img/b933ad4c5325a8333902b9276bc0bce9_56.png)

# 修改后代码：选择排名末尾的股票（得分最低）
selected_stocks = ranked_stocks.iloc[10:]  # 此处仅为示例，实际应选择排名末N位
```

![](img/b933ad4c5325a8333902b9276bc0bce9_58.png)

![](img/b933ad4c5325a8333902b9276bc0bce9_60.png)

我们执行修改后的策略。结果发现，基准收益已经是负值（约-10%），而选择“差生”股票组合的收益比基准更差。这从反面验证了我们的打分标准是有效的：得分高的股票组合确实比得分低的组合表现更好。

![](img/b933ad4c5325a8333902b9276bc0bce9_62.png)

![](img/b933ad4c5325a8333902b9276bc0bce9_64.png)

## 课程总结

![](img/b933ad4c5325a8333902b9276bc0bce9_80.png)

本节课中我们一起学习了如何对一个量化策略进行回测分析与总结。

![](img/b933ad4c5325a8333902b9276bc0bce9_82.png)

1.  **策略验证**：我们通过回测发现，基于特定财务指标的打分策略，在多数情况下能够产生一定的超额收益。
2.  **稳定性测试**：通过调整回测时间区间，我们认识到策略表现受市场整体环境影响很大，在熊市中可能失效。
3.  **逻辑反证**：通过选择低分股票进行对比回测，从反面验证了打分指标的有效性。
4.  **报告分析**：我们学会了查看回测报告中的关键信息，如收益曲线、最大回撤、交易详情和账户净值变化。

![](img/b933ad4c5325a8333902b9276bc0bce9_84.png)

![](img/b933ad4c5325a8333902b9276bc0bce9_86.png)

打分法是一种直观且常用的多因子选股方法，能够快速构建出一个具备逻辑基础的投资策略。但需要注意的是，没有任何策略能保证在所有市场环境下都盈利，实际表现会受到宏观经济、市场情绪等多种因素影响。大家可以将此策略作为一个起点，尝试调整打分指标、权重或调仓频率，以探索更适合自己的量化模型。
