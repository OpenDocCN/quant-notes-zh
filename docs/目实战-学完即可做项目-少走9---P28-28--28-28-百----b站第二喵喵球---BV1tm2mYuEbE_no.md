## 因子数据概述

上一节我们介绍了量化交易的基本概念，本节中我们来看看因子数据是什么。因子可以理解为选股的标准或指标。例如，在购买股票时，投资者可能依据市盈率、市净率或营收增长率等标准进行筛选。这些对最终投资决策和收益有影响的指标，就是量化策略中的“因子”。

当我们获得因子数据后，不能直接用于建模。就像数据挖掘任务一样，我们需要对原始数据进行预处理，以消除噪声、统一量纲，并减少无关变量的影响。

## 因子数据预处理三步走

![](img/e979a7e6f57d28c09af2dd071d05c2ef_3.png)

以下是因子数据预处理的三个核心步骤：

1.  **去极值**：处理数据中的离群点或异常值。
2.  **标准化**：将不同量纲和取值范围的因子数据转换到统一的尺度上。
3.  **中性化**：消除因子数据中与某些系统性风险（如行业、市值）的相关性。

前两步是数据挖掘中的常见操作，而第三步“中性化”在因子策略中尤为重要。接下来，我们将按照这个顺序，首先详细讲解“去极值”中的“百分位法”。

## 百分位去极值方法

![](img/e979a7e6f57d28c09af2dd071d05c2ef_9.png)

在数据中，绝大多数数据点通常集中在某个范围内，但少数数据点可能远离主体，这些点被称为极值或离群点。直接删除这些数据点可能造成信息损失。一种更优的方法是设定一个合理的边界，将超出边界的数据“拉回”到边界上，而不是丢弃它们。这就是“去极值”的核心思想。

![](img/e979a7e6f57d28c09af2dd071d05c2ef_11.png)

![](img/e979a7e6f57d28c09af2dd071d05c2ef_13.png)

以下是几种常见的去极值方法，本节课重点介绍“百分位法”：

![](img/e979a7e6f57d28c09af2dd071d05c2ef_15.png)

![](img/e979a7e6f57d28c09af2dd071d05c2ef_17.png)

*   **分位数与中位数**：中位数是数据排序后处于中间位置的值。与均值相比，中位数对极值不敏感，能更好地代表数据的中心趋势。分位数则扩展了这个概念。
*   **四分位数**：将排序后的数据四等分，可以得到三个关键分位点：
    *   **Q1（第一四分位数）**：25%的数据小于等于此值。
    *   **Q2（第二四分位数）**：即中位数，50%的数据小于等于此值。
    *   **Q3（第三四分位数）**：75%的数据小于等于此值。
*   **计算上下限**：利用四分位数计算数据的上下边界。一个常用的方法是计算四分位距（IQR），然后设定边界。
    *   **公式**：`IQR = Q3 - Q1`
    *   **下限**：`Lower Bound = Q1 - k * IQR`
    *   **上限**：`Upper Bound = Q3 + k * IQR`
    *   其中，`k` 是一个常数（通常取 1.5）。任何小于下限或大于上限的值都被视为极值。
*   **缩尾处理**：将识别出的极值调整到边界值上。例如，如果一个值大于上限，则将其设置为上限值；如果一个值小于下限，则将其设置为下限值。

```python
# 示例代码：使用pandas进行百分位去极值（缩尾处理）
import pandas as pd
import numpy as np

![](img/e979a7e6f57d28c09af2dd071d05c2ef_19.png)

# 假设 factor_data 是一个包含因子值的Pandas Series
def winsorize_by_percentile(series, lower_percentile=0.05, upper_percentile=0.95):
    """
    使用百分位法进行去极值（缩尾）处理。
    :param series: 待处理的数据序列
    :param lower_percentile: 下百分位，默认5%
    :param upper_percentile: 上百分位，默认95%
    :return: 处理后的数据序列
    """
    lower_bound = series.quantile(lower_percentile)
    upper_bound = series.quantile(upper_percentile)
    # 将小于下限的值替换为下限，大于上限的值替换为上限
    winsorized_series = series.clip(lower=lower_bound, upper=upper_bound)
    return winsorized_series

# 示例：对苹果股价数据进行处理（此处仅作演示，实际因子数据来自专业平台）
# 生成模拟股价数据
np.random.seed(42)
apple_prices = pd.Series(np.random.normal(150, 20, 1000)) # 均值150，标准差20
# 人为添加一些极值
apple_prices.iloc[0] = 50   # 极低值
apple_prices.iloc[1] = 300  # 极高值

print("原始数据描述（包含极值）:")
print(apple_prices.describe())

![](img/e979a7e6f57d28c09af2dd071d05c2ef_21.png)

# 应用百分位去极值函数
winsorized_prices = winsorize_by_percentile(apple_prices, 0.05, 0.95)

print("\n去极值后数据描述:")
print(winsorized_prices.describe())
```

## 方法对比与选择

![](img/e979a7e6f57d28c09af2dd071d05c2ef_23.png)

除了百分位法，常见的去极值方法还有**标准差法（MAD）**和**中位数绝对偏差法**。百分位法的优点在于不依赖于数据服从正态分布的假设，且对极值鲁棒，计算简单直观，是实践中非常常用且有效的方法。
