描述性统计分析旨在通过数学方法总结和描述数据集的主要特征，而可视化则将这些特征以图形方式直观呈现。两者结合，能帮助我们快速把握金融数据的分布、中心趋势和离散程度。

## 描述性统计分析

描述性统计提供了一系列指标来概括数据。对于金融时间序列数据（如股价），我们通常关注以下几类核心指标。

以下是常用的描述性统计指标：

*   **集中趋势指标**：反映数据的中心位置。
    *   **均值**：所有数据点的平均值。公式为：`均值 = 数据总和 / 数据个数`
    *   **中位数**：将数据排序后位于中间位置的值。
    *   **众数**：数据中出现次数最多的值。

*   **离散程度指标**：反映数据的波动或分散情况。
    *   **方差**：各数据点与均值之差的平方的平均值。公式为：`方差 = Σ(每个数据 - 均值)² / 数据个数`
    *   **标准差**：方差的平方根，其单位与原始数据一致。公式为：`标准差 = 方差的平方根`
    *   **极差**：最大值与最小值之差。

*   **分布形态指标**：
    *   **偏度**：描述数据分布不对称性的程度。正偏表示右尾较长，负偏表示左尾较长。
    *   **峰度**：描述数据分布陡峭或平坦的程度。高峰度意味着数据集中在均值附近，尾部较厚。

在Python中，我们可以使用Pandas库快速计算这些指标。假设我们有一个名为`df`的DataFrame，其中包含一列股价数据`‘Close’`。

```python
import pandas as pd

![](img/481cc044937b506deb5bce37480ab8ff_0.png)

# 计算基本描述性统计
print(df[‘Close’].describe())

# 计算特定指标
mean_price = df[‘Close’].mean()

```

了解了如何用数字描述数据后，我们将通过图形更直观地探索数据。

## Python可视化分析

可视化能将复杂的统计结果转化为易于理解的图表。Matplotlib和Seaborn是Python中最常用的绘图库。

以下是几种在金融数据分析中至关重要的图表类型及其作用：

![](img/481cc044937b506deb5bce37480ab8ff_1.png)

*   **折线图**：用于展示数据随时间变化的趋势，是观察股价走势最直接的图表。
    ```python
    import matplotlib.pyplot as plt
    df[‘Close’].plot(title=‘股价走势图’)
    plt.show()
    ```

*   **直方图与密度图**：用于观察数据的分布情况。直方图显示数据在不同区间的频数，密度图则平滑地显示概率分布。
    ```python
    import seaborn as sns
    # 直方图
    df[‘Close’].hist(bins=50)
    # 密度图
    sns.kdeplot(df[‘Close’])
    plt.show()
    ```

*   **箱型图**：用于识别数据中的异常值，并直观显示数据的中位数、四分位数和极值。
    ```python
    df[[‘Close’]].boxplot()
    plt.show()
    ```

*   **移动平均线图**：在折线图上叠加计算出的移动平均线，有助于平滑短期波动，识别长期趋势。
    ```python
    df[‘Close’].plot(label=‘收盘价’)
    df[‘Close’].rolling(window=20).mean().plot(label=‘20日移动平均线’)
    plt.legend()
    plt.show()
    ```
