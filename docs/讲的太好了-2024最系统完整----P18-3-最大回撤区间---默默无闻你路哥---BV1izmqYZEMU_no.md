## 最大回撤的定义

上一节我们介绍了策略分析中的一些基本概念，本节中我们来看看**最大回撤**。

在策略分析中，最大回撤表示的是资产净值从峰值到谷底的最大跌幅。它描述的是投资过程中可能经历的最糟糕的时期。

具体来说，在投资过程中，资产价格会有下跌的时候。其中，从某个高点开始，到后续某个低点结束，这段**跌幅最大且持续时间最长**的区间，就称为最大回撤区间。

最大回撤不能仅从字面理解。它本质上描述的是投资过程中的“惨烈程度”，即你可能会承受的最大损失风险。它评估的不是策略的盈利能力，而是策略的**风险水平**。

---

## 最大回撤的计算公式

为了更精确地理解这个“惨烈程度”，我们通过公式来看。

最大回撤的计算公式如下：

![](img/29bd53eef8aa75c678a08ed3ab54c709_4.png)

**最大回撤 = max( (P_i - P_j) / P_i )**

![](img/29bd53eef8aa75c678a08ed3ab54c709_6.png)

其中：
*   **P_i** 代表第 i 天的资产价格（或净值）。
*   **P_j** 代表第 i 天之后（j > i）的某一天资产价格。
*   我们需要遍历所有可能的 i 和 j（j > i），找出使得 `(P_i - P_j) / P_i` 这个比值最大的那个组合。

![](img/29bd53eef8aa75c678a08ed3ab54c709_7.png)

这个公式计算的是从高点 **P_i** 到低点 **P_j** 的**相对缩水比例**。比值越大，说明回撤越严重。

![](img/29bd53eef8aa75c678a08ed3ab54c709_9.png)

## 最大回撤的意义与重要性

![](img/29bd53eef8aa75c678a08ed3ab54c709_13.png)

最大回撤是一个关键的风险指标。它告诉你，即使一个策略长期来看是盈利的，但在过程中你可能会经历一段非常难熬的时期，资产会大幅缩水。

例如，在2008年金融危机中，股市从高点大幅下跌，许多投资者承受了巨大的损失。最大回撤区间描述的就是这样一段“最惨”的时期，它能让你对策略的潜在风险有清醒的认识。

## 使用Python计算最大回撤

接下来，我们看看如何用Python代码计算最大回撤。核心思路是使用**累积最大值**来辅助计算。

![](img/29bd53eef8aa75c678a08ed3ab54c709_23.png)

![](img/29bd53eef8aa75c678a08ed3ab54c709_24.png)

以下是计算最大回撤的步骤：

![](img/29bd53eef8aa75c678a08ed3ab54c709_26.png)

1.  **计算累积最大值序列**：对于每个时间点，计算截至到该点的历史最高价。
2.  **计算每日回撤**：用该点的累积最大值减去当前的实际价格，得到潜在的下跌幅度。
3.  **计算每日回撤率**：将每日回撤除以对应的累积最大值，得到回撤比例。
4.  **找出最大值**：在所有每日回撤率中，找出最大值，即为最大回撤率。

![](img/29bd53eef8aa75c678a08ed3ab54c709_27.png)

以下是使用Pandas库进行计算的示例：

![](img/29bd53eef8aa75c678a08ed3ab54c709_29.png)

```python
import pandas as pd

![](img/29bd53eef8aa75c678a08ed3ab54c709_30.png)

# 假设 df[‘close’] 是资产的每日收盘价序列
# 1. 计算累积最大值
cumulative_max = df[‘close’].cummax()

![](img/29bd53eef8aa75c678a08ed3ab54c709_32.png)

# 2. 计算每日回撤（累积最大值 - 当前值）
drawdown = cumulative_max - df[‘close’]

# 3. 计算每日回撤率
drawdown_pct = drawdown / cumulative_max

![](img/29bd53eef8aa75c678a08ed3ab54c709_34.png)

![](img/29bd53eef8aa75c678a08ed3ab54c709_35.png)

# 4. 找出最大回撤率
max_drawdown = drawdown_pct.max()
print(f“最大回撤率为：{max_drawdown:.2%}”)
```

**代码解释**：
*   `.cummax()` 方法用于计算序列的累积最大值。
*   `drawdown_pct` 序列中的每个值，都代表了从历史最高点回落到当前点的比例。
*   `.max()` 方法找出这个比例序列中的最大值，即最大回撤。
