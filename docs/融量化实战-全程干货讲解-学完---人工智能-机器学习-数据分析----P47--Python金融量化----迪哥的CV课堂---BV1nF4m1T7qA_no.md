## 因子收益率的概念

上一节我们介绍了IC值，它衡量的是因子与收益率之间的相关性。本节中我们来看看因子收益率。

因子收益率是一个具体的数值，它描述了因子对资产收益的贡献权重。其核心思想可以通过一个线性回归模型来理解：我们将股票的收益率（因变量）与因子值（自变量）进行回归，得到的回归系数即为因子收益率。

![](img/7d0db716721f2c26c0b4270472fcb0b2_7.png)

**公式**：
`收益率 = 因子 × 因子收益率 + 偏置项`

在这个公式中：
*   **收益率**：通常指股票在某一期的收益。
*   **因子**：我们选定的某个因子（例如市盈率、市值等）在当期的值。
*   **因子收益率**：就是我们要求解的权重参数 `W`，它表示该因子对收益的贡献程度。
*   **偏置项**：回归模型中的常数项。

因此，因子收益率 `W` 是一个具体的数值，我们可以为每一个因子计算其因子收益率。

![](img/7d0db716721f2c26c0b4270472fcb0b2_9.png)

## 因子收益率的用途

![](img/7d0db716721f2c26c0b4270472fcb0b2_11.png)

因子收益率主要有两个核心用途。

![](img/7d0db716721f2c26c0b4270472fcb0b2_13.png)

**1. 可视化分析因子有效性**
我们可以通过分组回测来观察因子在不同取值区间带来的平均收益。以下是分析步骤：
*   将因子值从小到大分为若干组（例如5组）。
*   计算每组股票在下一期的平均收益率。
*   通过柱状图等形式可视化展示。

![](img/7d0db716721f2c26c0b4270472fcb0b2_15.png)

例如，在下图中，横轴代表因子分组，纵轴代表该组股票的平均收益。我们可以直观地看到，第2组（因子值在某个中等区间）带来的平均收益最高，这说明该因子在这个取值范围内可能最有效。

![](img/7d0db716721f2c26c0b4270472fcb0b2_17.png)

![](img/7d0db716721f2c26c0b4270472fcb0b2_19.png)

**2. 作为因子筛选的量化指标**
当我们有成百上千个候选因子时，需要一套量化标准进行筛选。因子收益率本身就可以作为一个筛选指标。通常，我们会综合多个指标进行评估。

![](img/7d0db716721f2c26c0b4270472fcb0b2_21.png)

![](img/7d0db716721f2c26c0b4270472fcb0b2_23.png)

以下是常用的因子评估指标：
*   **因子收益率**：衡量因子对收益的直接贡献。
*   **IC值（信息系数）**：衡量因子与未来收益的相关性。
*   **IR值（信息比率）**：由IC均值除以IC标准差得到，衡量因子预测能力的稳定性。

![](img/7d0db716721f2c26c0b4270472fcb0b2_25.png)

![](img/7d0db716721f2c26c0b4270472fcb0b2_27.png)

在实际筛选中，我们会综合考虑这些指标，评选出综合表现更优的因子。

![](img/7d0db716721f2c26c0b4270472fcb0b2_29.png)

![](img/7d0db716721f2c26c0b4270472fcb0b2_31.png)
