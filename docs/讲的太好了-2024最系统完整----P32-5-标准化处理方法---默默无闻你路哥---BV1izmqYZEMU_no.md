在本节课中，我们将要学习数据预处理中一个至关重要的步骤——标准化。标准化能够消除不同数据特征在量纲和取值范围上的差异，为后续的金融分析与建模打下坚实基础。

## 标准化的目的与原理

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_1.png)

上一节我们介绍了数据预处理的重要性，本节中我们来看看如何通过标准化处理数据。

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_2.png)

标准化操作的核心目标是：**使不同特征（或指标）的数值处于同一量级，从而避免某些特征因其数值范围较大而对模型结果产生不成比例的影响**。

为了直观理解，假设我们有两个特征X1和X2。X1的取值范围较大，而X2的取值范围较小。如果直接使用原始数据，机器学习算法可能会更“重视”取值范围大的X1，而“忽视”X2，这并非我们期望的结果。标准化的目的就是让所有特征都能被“平等”地看待。

## 标准化的数学公式

标准化的具体操作通过一个数学公式完成，该公式分为两个步骤：

**标准化公式**：
`X_standardized = (X - μ) / σ`

其中：
*   `X` 是原始数据。
*   `μ` 是数据`X`的均值。
*   `σ` 是数据`X`的标准差。

## 公式分步详解

这个公式看似简单，但每一步都有其明确的作用。

### 第一步：去均值（中心化）

`X - μ` 这一步称为“去均值”或“中心化”。
它的作用是**将数据分布的中心移动到零点**。处理后的数据会以坐标原点（0点）为中心大致对称分布。这解决了数据原点不一致的问题。

### 第二步：除以标准差（缩放）

`/ σ` 这一步是“缩放”。
它的作用是**消除不同特征取值范围（方差）的差异**。标准差σ衡量了数据的离散程度。对于取值范围大、离散程度高的特征，其σ值也大，相除后数值会被压缩。对于取值范围小、离散程度低的特征，其σ值也小，相除后数值变化相对不大。最终，所有特征的数据都会被缩放至一个相近的数值范围内（通常围绕0分布，大部分值在[-1, 1]或[-3, 3]之间）。

通过这两步，数据既完成了中心化，也完成了缩放，达到了标准化的目的。

## 标准化的代码实现

理解了原理后，我们可以自己动手实现标准化函数。以下是使用Python和NumPy库的实现方法：

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_4.png)

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_5.png)

```python
import numpy as np

def standardize(data):
    """
    对输入的数据进行标准化处理。
    参数:
        data: 一个NumPy数组，形状为 (n_samples, n_features) 或 (n_samples,)
    返回:
        标准化后的数据。
    """
    # 计算数据的均值和标准差
    # axis=0 表示沿列（特征）方向计算，确保每个特征独立计算其均值和标准差
    mean = np.mean(data, axis=0)
    std = np.std(data, axis=0)
    
    # 应用标准化公式： (数据 - 均值) / 标准差
    # 为避免除零错误，通常会给标准差加上一个极小的数（如1e-8）
    standardized_data = (data - mean) / (std + 1e-8)
    
    return standardized_data
```

## 使用Scikit-learn进行标准化

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_7.png)

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_8.png)

在实际项目中，我们通常使用成熟的机器学习库。Scikit-learn提供了高效且功能丰富的标准化工具。

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_9.png)

以下是使用Scikit-learn中的`StandardScaler`进行标准化的方法：

```python
from sklearn.preprocessing import StandardScaler
import numpy as np

# 示例数据
data = np.array([[30, 200], [20, 150], [100, 500], [50, 350]])

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_11.png)

# 1. 创建StandardScaler对象
scaler = StandardScaler()

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_12.png)

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_13.png)

# 2. 拟合数据并计算均值和标准差
scaler.fit(data)

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_14.png)

# 3. 转换数据
standardized_data = scaler.transform(data)

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_16.png)

print("原始数据：")
print(data)
print("\n标准化后的数据：")
print(standardized_data)
```

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_17.png)

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_18.png)

运行上述代码，你会发现原本数值范围差异很大的数据（如30和500），经过标准化后，都转换到了相近的数值区间内。

## 标准化操作总结

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_20.png)

本节课中我们一起学习了数据标准化处理。

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_21.png)

![](img/4e0a7d5b8a28c87f9a562c6cdd328a75_22.png)

*   **目的**：消除不同特征间量纲和取值范围的差异，使所有特征对模型的影响权重相当。
*   **核心公式**：`X_new = (X - μ) / σ`。
*   **步骤**：第一步“去均值”实现中心化，使数据以0为中心；第二步“除以标准差”实现缩放，使数据范围一致。
*   **实现**：可以手动使用NumPy实现，也可以直接调用Scikit-learn中的`StandardScaler`，后者更为方便和稳健。

标准化是数据预处理中最常用、最有效的方法之一。在开始任何机器学习或量化分析模型前，对数据进行标准化处理通常是一个很好的起点。
