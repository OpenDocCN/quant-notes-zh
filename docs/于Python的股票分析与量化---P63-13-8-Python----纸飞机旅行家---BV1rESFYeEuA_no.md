## 策略背景与算法介绍

上一节我们介绍了新能源股票轮动量化交易策略，本节中我们来看看另一种基于价值投资理念的策略：低估值量化交易策略。

首先介绍策略背景。在介绍选股策略时，我们曾提到股票净值是决定股票市场走向的主要依据。通常，市净率越低的股票，其投资价值越高；相反，其投资价值就越小。简单来说，越被市场低估的股票，未来出现大幅上涨的可能性就越大。

然而，凡事都有例外。在A股市场，除了考虑低估值（即股票净值），还必须综合考虑当时的市场环境、公司的经营状况和盈利能力。例如，一只股票如果持续下跌甚至跌破发行价，很可能意味着公司本身存在问题。投资这类股票风险极高，可能导致投资者“买着买着变股东”。

因此，我们策略的核心是：**市净率要尽量低**，同时要**综合市场环境、公司经营状况和盈利能力**进行筛选。

接下来分享低估值量化交易策略的具体算法。该算法主要包含以下五个要点：

以下是策略算法的五个核心条件：

1.  **市净率（PB）小于1**：这是筛选低估值的核心指标。
2.  **负债比例低于市场平均值**：用于评估公司的财务健康状况。
3.  **流动资产至少是流动负债的1.2倍**：用于评估公司的短期偿债能力和流动性管理水平。
4.  **每月调仓一次**：定期调整投资组合，保持策略的时效性。
5.  **设置止损风控**：这是一个经验系数。如果十天内沪深300指数跌幅超过10%，则可能意味着整个市场环境不景气，此时应清仓止损，暂停买入。

总结一下整个策略算法：我们通过**市净率小于1**来寻找低估值股票；通过**负债比例低于市场平均值**和**流动资产是流动负债的1.2倍**来评估公司的经营与盈利能力；通过**十天内沪深300指数跌幅不超过10%** 来监控大盘环境，作为风控止损条件。每月调仓一次以保证组合健康。其中的止损参数（如10%）是一个经验值，感兴趣的同学可以自行调整优化，甚至引入机器学习方法来寻找最优参数。

def get_stocks(context):
    # 获取沪深300指数的成分股
    stocks = get_index_stocks(g.index)

    # 获取股票的基本面数据：资产负债表和估值数据
    # ‘balance’ 相关字段包含负债信息，‘valuation’ 相关字段包含估值信息
    q = query(
        valuation.code,
        valuation.pb_ratio,                # 市净率
        balance.total_current_assets,      # 流动资产合计
        balance.total_current_liability    # 流动负债合计
    ).filter(
        valuation.code.in_(stocks)         # 仅筛选指数成分股
    )

    # 获取查询结果
    df = get_fundamentals(q)

    # 进行条件过滤
    # 条件1：市净率小于1
    df = df[df['pb_ratio'] < 1]
    # 条件2：流动资产至少是流动负债的1.2倍
    df = df[df['total_current_assets'] / df['total_current_liability'] > 1.2]

    # 计算负债比例（这里用流动负债/流动资产作为一个简化示例，实际可能需用总负债/总资产）
    # 注意：此为示例逻辑，真实策略中应使用更准确的财务指标
    df['liability_ratio'] = df['total_current_liability'] / df['total_current_assets']

    # 计算市场平均负债比例
    market_mean_liability_ratio = df['liability_ratio'].mean()

    # 条件3：负债比例低于市场平均值
    df = df[df['liability_ratio'] < market_mean_liability_ratio]

    # 返回最终筛选出的股票代码列表
    return list(df['code'])
