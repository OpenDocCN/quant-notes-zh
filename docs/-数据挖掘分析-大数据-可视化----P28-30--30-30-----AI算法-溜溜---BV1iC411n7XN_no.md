## MAD法原理

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_8.png)

MAD法的公式看起来稍显复杂，但计算过程其实很简单。它的核心思想是：**利用数据的中位数和绝对偏差的中位数来定义一个合理的数值范围，超出此范围的值被视为极值**。

以下是MAD法的计算步骤：

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_10.png)

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_12.png)

1.  **计算原始数据的中位数**：首先，对于一个给定的数据序列，我们计算它的中位数。假设中位数为 `median`。
2.  **计算绝对偏差**：然后，计算序列中**每一个数值**与中位数 `median` 的**绝对差值**。这会形成一个新的序列。
    *   公式为：`|x_i - median|`，其中 `x_i` 是原始数据中的每一个值。
3.  **计算MAD值**：接着，对这个由绝对差值构成的新序列，再次计算它的**中位数**。这个结果就是 **MAD值**。
4.  **确定上下边界**：最后，利用中位数和MAD值来确定数据的正常范围。
    *   **上界** = `median + N * MAD`
    *   **下界** = `median - N * MAD`
    *   其中，`N` 是一个常数，通常取值为 **1.4826**。这个值是为了使MAD法在正态分布数据下与标准差法具有一致性。

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_14.png)

任何落在 `[下界, 上界]` 区间之外的数据点，都可以被认为是极值，并进行相应的处理（如截断或替换）。

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_16.png)

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_18.png)

```python
import pandas as pd
import numpy as np

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_30.png)

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_32.png)

def filter_extreme_mad(series, n=1.4826):
    """
    使用MAD法对序列进行去极值处理。
    
    参数:
    series: pandas Series，待处理的数据序列。
    n: float，MAD的倍数，默认为1.4826。
    
    返回:
    Series: 处理后的数据序列。
    """
    # 1. 计算原始序列的中位数
    median = series.quantile(0.5)
    
    # 2. 计算每个数据点与中位数的绝对偏差
    abs_deviation = (series - median).abs()
    
    # 3. 计算绝对偏差的中位数，即MAD值
    mad = abs_deviation.quantile(0.5)
    
    # 4. 计算上下边界
    upper_bound = median + n * mad
    lower_bound = median - n * mad
    
    # 5. 将超出边界的数据截断到边界值
    filtered_series = series.clip(lower=lower_bound, upper=upper_bound)
    
    return filtered_series
```

![](img/aef7d14194cd2f3e29b4cfe75fd22bf2_34.png)

代码解释：
*   `series.quantile(0.5)`：用于计算序列的中位数。
*   `(series - median).abs()`：计算每个值与中位数的绝对偏差。
*   `.clip(lower=lower_bound, upper=upper_bound)`：Pandas的内置方法，将所有小于下界的值设置为下界，所有大于上界的值设置为上界，中间的值保持不变。
