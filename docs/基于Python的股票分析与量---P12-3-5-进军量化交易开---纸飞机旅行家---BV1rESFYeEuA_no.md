## 股票交易基础知识

上一节我们介绍了股价及其影响因素，本节中我们来看看股票交易本身及其相关的基础知识。



### 交易时间

中国A股的交易时间通常为周一至周五，法定节假日除外。具体时段如下：
*   上午：09:30 至 11:30
*   下午：13:00 至 15:00

每个交易日仅有4个小时可以进行交易，一年大约有250个交易日。



### 竞价成交规则

A股市场的成交遵循“价格优先、时间优先”的原则。以下是竞价的具体时段划分：

*   **价格优先**：对于买方，报价越高越优先成交；对于卖方，报价越低越优先成交。
*   **时间优先**：在报价相同的情况下，委托时间早的订单优先成交。

以下是竞价时段的具体划分：
*   **开盘集合竞价 (09:15 - 09:25)**：此阶段以成交量最大的价格作为开盘价，通常由机构主导。
*   **连续竞价 (09:30 - 11:30, 13:00 - 14:57)**：这是散户主要参与的交易时段，确保每一笔委托都能即时匹配成交。
*   **收盘集合竞价 (14:57 - 15:00)**：最后3分钟，以成交量最大的价格作为收盘价。



### 交易单位

了解股票的交易单位是下单的基础。以下是相关概念：

*   **报价单位**：`股`。股价以每股多少元来标示。
*   **交易单位**：`手`。1手等于100股，是最小的买卖单位。
*   **最小变动单位**：`0.01元`。即股价变动的最小单位是1分钱。

### 其他核心概念

除了交易规则，还有几个重要的基础概念需要了解。

以下是三个关键概念：
1.  **庄家与散户**：
    *   **庄家**：指资金雄厚，能显著影响证券市场行情的大额投资者，通常是机构。
    *   **散户**：指投入股市资金量较小的个人投资者。
2.  **换手率**：某段时间内的成交量与流通总股数的比率。公式为：`换手率 = 成交量 / 流通总股数`。该指标反映股票活跃度：
    *   `10% - 50%`：非常活跃。
    *   `< 1%`：非常不活跃，建议初学者谨慎关注。
3.  **市盈率 (P/E)**：每股市场价格与每股税后利润的比率。公式为：`市盈率 = 股价 / 每股收益`。它衡量投资回报年限：
    *   通常，市盈率在`16~20`之间被视为合理区间。
    *   过高（如`>50`）可能表示股价被高估，存在泡沫风险；过低可能表示公司不被看好。



## 认识K线图 📊

在了解了交易的基本规则后，我们来看看技术分析中最重要的工具之一——K线图。

![](img/e6fda4855b5f3b1e2cc4d9b1a37f28e1_5.png)

K线图，又称蜡烛图，记录了股票在每个周期（如日、周）内的开盘价、收盘价、最高价和最低价。其基本形态如下图所示：



（此处原有一张K线图示意图，描述其包含上影线、下影线、实体部分，红色代表上涨，黑色/绿色代表下跌。）



关于K线图，有两点重要提示：
1.  **需结合成交量分析**：单独看K线形态意义有限，必须与对应周期的成交量结合观察，才能更有效地判断市场情绪。
2.  **不是盈利的保证**：精通K线分析不代表能在股市中稳定盈利。股市是零和博弈，真正能持续盈利的投资者极少。



## 关键财务指标 💰

技术分析关注图表，而基本面分析则关注公司的财务状况。每一家上市公司都会定期发布财务报告。



（此处原有一张上市公司财报截图，展示其中包含的ROE、每股收益、净利润等数据。）



对于初学者，可以优先关注以下三个核心财务指标：
1.  **净资产收益率 (ROE)**：公式为 `ROE = 净利润 / 平均净资产`。它衡量公司运用自有资本的效率，**越高越好**。
2.  **净利润**：公司经营的最终成果，直接反映盈利能力，**越高越好**。
3.  **成长性**：主要通过**营业利润增长率**和**净利润增长率**来体现。这反映了公司未来的发展潜力，增长率**越高越好**。
