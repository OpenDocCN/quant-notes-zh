## 概述
在本节课中，我们将要学习一个在金融投资领域至关重要的风险调整后收益指标——**夏普比率**。我们将理解它的定义、计算方式以及在量化选股中的实际应用。

## 夏普比率的定义与意义
上一节我们介绍了投资回报率，本节中我们来看看如何衡量风险与收益的关系。

![](img/8555c0eeeb81ca10f0aa850430d1d9d7_2.png)

夏普比率描述的是，在承担单位风险的情况下，所能获得的超额收益是多少。指标越高，意味着对于单位风险，投资者获得的补偿（收益）越高。在选股时，如果仅从夏普比率这一指标出发，我们倾向于选择该值更高的股票或投资组合，因为它代表了更高的风险调整后收益。

![](img/8555c0eeeb81ca10f0aa850430d1d9d7_4.png)

![](img/8555c0eeeb81ca10f0aa850430d1d9d7_6.png)

其核心思想可以类比为：一份在战乱地区日薪数万的工作，虽然收益高，但风险也极高。夏普比率就是用来量化“为了这份高收益，承担如此高的风险是否值得”。

## 夏普比率的计算公式
理解了夏普比率的意义后，我们来看看如何具体计算它。

计算夏普比率需要比较投资组合的收益与无风险收益（如国债、银行存款利率），并考虑投资组合的波动性（风险）。其标准公式如下：

**夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差**

为了更直观，以下是使用Python进行计算的代码框架描述：

![](img/8555c0eeeb81ca10f0aa850430d1d9d7_8.png)

```python
# 假设 returns 是投资组合的日收益率序列
# risk_free_rate 是年化无风险收益率，例如 0.05 代表5%
# 计算年化夏普比率
excess_returns = returns - risk_free_rate/250  # 将年化无风险利率转为日利率
sharpe_ratio = np.sqrt(250) * (excess_returns.mean() / excess_returns.std())
```

![](img/8555c0eeeb81ca10f0aa850430d1d9d7_10.png)

**公式解释**：
1.  `投资组合收益率 - 无风险收益率`：得到超额收益，即承担风险所获得的额外补偿。
2.  `/ 投资组合收益率的标准差`：将超额收益除以波动性（风险），得到单位风险对应的超额收益。
3.  `np.sqrt(250)`：假设一年有250个交易日，将日度计算的比率年化，以便于在不同周期产品间进行比较。

## 实战计算与结果解读
接下来，我们将在Python环境中实际计算几只股票的夏普比率。

以下是计算步骤的关键点说明：
1.  首先，需要获取股票的历史收益率数据。
2.  在计算前，需处理数据中的缺失值（例如，用前一天的收盘价填充停牌日的数据）。
3.  应用上述公式进行计算。

![](img/8555c0eeeb81ca10f0aa850430d1d9d7_12.png)

假设我们对A、B、C三只股票进行了计算，得到如下结果：
*   股票A夏普比率：0.85
*   股票B夏普比率：1.32
*   股票C夏普比率：-0.15

![](img/8555c0eeeb81ca10f0aa850430d1d9d7_14.png)

![](img/8555c0eeeb81ca10f0aa850430d1d9d7_16.png)

**结果分析**：
*   股票B的夏普比率最高，意味着在承担相同单位风险的情况下，其历史表现提供的收益补偿最优。
*   股票A次之。
*   股票C的夏普比率为负值，表明其历史收益甚至未能覆盖无风险收益，通常不作为优先考虑对象。
因此，仅从夏普比率的角度，**股票B**是更优的选择。
