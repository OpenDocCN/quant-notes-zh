在本节课中，我们将要学习机器学习模型优化的核心步骤——参数更新。具体来说，我们将深入探讨梯度下降算法中，如何分别对模型的不同参数进行更新，以找到使损失函数最小的最优参数组合。

上一节我们介绍了损失函数的概念，本节中我们来看看如何通过优化算法来最小化这个损失函数。

## 参数更新的基本思路

当我们需要优化一个方程（例如线性回归模型 `y = θ₀ + θ₁x`）以找到最优的 `θ₀` 和 `θ₁` 时，一个关键问题是：应该同时更新这两个参数，还是分别更新它们？

答案是**分别更新**。原因在于，输入特征 `x₀`（通常为常数1，对应 `θ₀`）和 `x₁`（对应 `θ₁`）在数据样本之间是相互独立的。既然特征本身没有关系，那么它们所对应的参数 `θ₀` 和 `θ₁` 在优化过程中也应当独立处理。因此，在优化过程中，我们需要分别计算 `θ₀` 和 `θ₁` 的更新方向。

这个过程可以想象在一个三维空间中寻找山谷的最低点（即损失函数的最小值）。`θ₀` 和 `θ₁` 各代表一个轴。我们首先为 `θ₀` 找到一个合适的下降方向，同时也为 `θ₁` 找到另一个合适的下降方向。两者沿着各自的方向共同前进，最终达到综合最优的结果。

以下是参数更新的核心步骤：
1.  **求偏导**：分别计算损失函数 `J` 对每个参数 `θⱼ` 的偏导数，以确定各自的“下山”方向。
2.  **走一小步**：沿着偏导确定的**反方向**（梯度下降方向）前进一小步。步长必须小，步长过大可能导致“跌倒”，即算法不收敛或收敛效果差。
3.  **更新参数**：根据计算出的步长更新参数值。
4.  **迭代**：以更新后的参数作为新的起点，重复上述步骤，直至收敛。

这个流程是一个反复迭代的过程。

## 数学描述与推导

那么，如何用数学公式来描述这个“下山”任务呢？我们从一个具体的损失函数开始。

我们之前定义的损失函数（均方误差，MSE）是针对所有训练样本的平均误差，其目的是评估模型在整个数据集上的表现，而不仅仅是单个样本。损失函数 `J(θ)` 的公式如下：

**`J(θ) = (1/(2m)) * Σ (y⁽ⁱ⁾ - h_θ(x⁽ⁱ⁾))²`**
*   `m`：样本总数。
*   `h_θ(x⁽ⁱ⁾)`：模型对第 `i` 个样本的预测值，例如 `h_θ(x) = θ₀ + θ₁x₁ + θ₂x₂ + ...`。
*   `y⁽ⁱ⁾`：第 `i` 个样本的真实值。
*   平方项 `(...)²` 是为了放大预测值与真实值之间的较大差异。
*   `1/(2m)` 中的 `1/2` 是为了后续求导时简化计算，`1/m` 用于计算平均损失。

现在，假设我们要更新其中一个参数 `θⱼ`。我们需要计算损失函数 `J(θ)` 对 `θⱼ` 的偏导数。

以下是求偏导的关键推导过程：
*   在求和项 `Σ (y⁽ⁱ⁾ - h_θ(x⁽ⁱ⁾))²` 中，`h_θ(x⁽ⁱ⁾)` 是包含 `θⱼ` 的线性组合。
*   当对 `θⱼ` 求偏导时，`h_θ(x⁽ⁱ⁾)` 中与 `θⱼ` 无关的项（如 `θ₀x₀`, `θ₁x₁` 等）均视为常数，其导数为0。
*   只有包含 `θⱼ` 的项 `θⱼ * xⱼ⁽ⁱ⁾` 在求导后保留，其导数为 `xⱼ⁽ⁱ⁾`。
*   根据链式法则，平方项 `(...)²` 求导会产生一个系数 `2`。
*   同时，括号内 `(y⁽ⁱ⁾ - h_θ(x⁽ⁱ⁾))` 对 `h_θ(x⁽ⁱ⁾)` 求导会得到一个 `-1` 的系数（因为 `h_θ(x⁽ⁱ⁾)` 前面是负号）。

综合以上，损失函数 `J(θ)` 对参数 `θⱼ` 的偏导数 `∂J/∂θⱼ` 为：

**`∂J/∂θⱼ = (1/m) * Σ (h_θ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾`**

**公式解释**：
*   `(h_θ(x⁽ⁱ⁾) - y⁽ⁱ⁾)`：第 `i` 个样本的预测误差。
*   `xⱼ⁽ⁱ⁾`：第 `i` 个样本在第 `j` 个特征上的值。
*   `Σ`：对所有 `m` 个样本的乘积 `误差 * 特征值` 求和。
*   `1/m`：计算平均梯度。

这个偏导数 `∂J/∂θⱼ` 指明了 `θⱼ` 的梯度（上升）方向。为了最小化损失函数，我们需要朝梯度的**反方向**移动。因此，参数 `θⱼ` 的更新规则为：

**`θⱼ := θⱼ - α * ∂J/∂θⱼ`**

**代码描述**：
```python
# 假设 theta 是参数向量，X 是特征矩阵，y 是标签向量，alpha 是学习率
def update_parameters(theta, X, y, alpha):
    m = len(y)
    predictions = X.dot(theta)  # 计算预测值 h_theta(X)
    errors = predictions - y    # 计算预测误差
    # 计算所有参数的梯度 (∂J/∂θ)
    gradient = (1/m) * X.T.dot(errors)
    # 沿梯度反方向更新所有参数
    theta_new = theta - alpha * gradient
    return theta_new
```

**更新规则解释**：
*   `θⱼ`：参数当前值。
*   `α`：学习率，控制每次更新的步长，即“走一小步”中的步长。
*   `∂J/∂θⱼ`：计算出的梯度。
*   `θⱼ - α * ∂J/∂θⱼ`：从当前位置 `θⱼ` 出发，沿着梯度反方向移动 `α * ∂J/∂θⱼ` 的距离，得到参数更新后的新值 `θⱼ'`。

这个“负负得正”的过程（梯度 `∂J/∂θⱼ` 本身包含 `(预测值-真实值)`，而更新时又减去梯度）确保了参数始终朝着减少损失函数值的方向调整。
