## 量化交易入门：P2：1-fbprophet股价预测任务概述 🔮

上一节我们了解了课程全貌，本节中我们来看看如何利用工具进行股价预测。

股价预测是量化交易中常见的任务，其目标是基于历史价格数据，预测未来的价格走势。`fbprophet` 是由Facebook开源的一个强大且易用的时间序列预测工具。

`fbprophet` 将时间序列分解为趋势、季节性和节假日效应三个主要成分，其核心模型是一个可加性回归模型，公式如下：

`y(t) = g(t) + s(t) + h(t) + ε_t`

其中：
*   `g(t)` 表示趋势项，模拟非周期性的变化。
*   `s(t)` 表示季节性项，模拟周期性的变化（如周、年）。
*   `h(t)` 表示节假日项，模拟在不定期的假期中发生的影响。
*   `ε_t` 表示误差项，代表模型未捕捉的随机波动。

使用 `fbprophet` 进行预测通常只需几行代码，它自动处理了缺失值、异常值，并提供了直观的参数调整接口，非常适合金融时间序列的初步分析与预测。

---

## 量化交易入门：P3：2-时间序列分析 📈

在了解了预测工具后，我们需要理解其处理的对象——时间序列数据。

时间序列分析是研究按时间顺序排列的数据点序列的统计方法。在金融领域，股票价格、交易量等都是以时间序列的形式存在。

时间序列数据通常包含以下成分：
1.  **趋势**：数据在长期内呈现的上升、下降或平稳的走向。
2.  **季节性**：数据在固定周期（如一天、一周、一年）内重复出现的规律性波动。
3.  **周期性**：非固定周期的波动，通常与经济周期相关。
4.  **随机性**：无法用趋势、季节性或周期性解释的、无规律的波动。

分析时间序列有助于我们理解市场的内在规律，并为建立预测模型（如上一节提到的`fbprophet`）奠定基础。常见的分析方法包括绘制时序图、计算移动平均、分解序列等。

## 量化交易入门：P4：3-fbprophet时间序列预测实例 💻

理论需要结合实践，本节我们将动手完成一个完整的股价预测实例。

我们将使用 `fbprophet` 库对某只股票的历史价格进行建模和预测。以下是核心步骤的代码框架：

```python
# 步骤1：导入库并准备数据
import pandas as pd
from prophet import Prophet

# 假设df是一个包含‘ds’（日期）和‘y’（价格）两列的DataFrame
# df = pd.read_csv('stock_price.csv')

# 步骤2：创建并拟合模型
model = Prophet(
    yearly_seasonality=True,  # 启用年季节性
    weekly_seasonality=True,   # 启用周季节性
    daily_seasonality=False    # 股价通常不分析日季节性
)
model.fit(df)

# 步骤3：构建未来时间框架并进行预测
future = model.make_future_dataframe(periods=30)  # 预测未来30天
forecast = model.predict(future)

# 步骤4：可视化结果
fig1 = model.plot(forecast)
fig2 = model.plot_components(forecast)
```

通过这个实例，你可以看到 `fbprophet` 如何自动拟合历史数据中的趋势和季节性，并生成包含不确定性区间的未来预测。`plot_components` 函数能直观展示趋势、周季节性、年季节性等成分，帮助理解模型认为的股价驱动因素。

## 量化交易入门：P6：5-突变点调参 ⚙️

在预测模型中，趋势并非一成不变。本节我们学习如何调整模型以捕捉趋势的变化。

现实中的时间序列，其趋势可能会发生改变，例如增长速率加快或放缓，甚至方向反转。`fbprophet` 将这些趋势变化点称为“突变点”。

模型默认会自动检测突变点，但有时我们需要根据先验知识进行调整。以下是相关的关键参数：
*   `changepoint_prior_scale`: 控制趋势灵活性的参数。值越大，模型对趋势变化的拟合越强，但也可能过度拟合噪声。默认值为0.05。
*   `n_changepoints`: 设置潜在突变点的最大数量。默认值为25。

例如，如果我们认为亚马逊股价在某个时期（如云计算业务爆发）增长模式发生了根本性变化，可以适当增大 `changepoint_prior_scale`，让模型更注重那个时间段附近的趋势变化。

```python
model = Prophet(changepoint_prior_scale=0.1)  # 增加趋势灵活性
model.fit(df)
```

通过调整这些参数，我们可以使模型更好地适应历史数据中重要的趋势转折，从而可能提高预测的准确性。

## 量化交易入门：P7：1-连续指标变化情况分析 🔄

从本节开始，我们将视角从价格预测转向技术指标分析，这是量化策略的另一基石。

技术指标是通过对股价、成交量等原始数据进行数学计算得出的值，用以辅助判断市场状态和未来走势。首先，我们分析指标值本身随时间连续变化的情况。

以最常用的**简单移动平均线（SMA）**为例，其计算公式为：
`SMA(n) = (P1 + P2 + ... + Pn) / n`
其中 `P` 代表收盘价，`n` 代表计算周期（例如20天）。

通过绘制SMA线，我们可以观察：
1.  指标曲线的**方向**：向上、向下或走平，代表趋势。
2.  指标曲线的**斜率**：变化的速度。
3.  指标值与股价的**相对位置**：股价在均线上方还是下方。

连续观察指标的变化，比只看单点数值更有意义，它能帮助我们感受市场的“动量”和“力度”。

## 量化交易入门：P8：2-时间序列重采样操作 🔁

在分析技术指标时，我们经常需要转换数据的时间频率，这就需要用到重采样。

重采样是指将时间序列数据从一个频率转换到另一个频率的过程。例如，将日数据转换为周数据或月数据，或者将分钟数据转换为小时数据。

在Pandas中，使用 `resample` 方法可以方便地进行重采样：

```python
# 将日频收盘价数据重采样为周频（计算每周最后一个交易日的收盘价）
weekly_data = daily_data['close'].resample('W').last()

# 将日频数据重采样为月频（计算每月平均收盘价）
monthly_data = daily_data['close'].resample('M').mean()

# 将分钟数据重采样为小时数据（计算每小时内的成交量之和）
hourly_volume = minute_data['volume'].resample('H').sum()
```

重采样的主要目的有：
1.  **降低噪声**：高频数据波动大，转为低频可以平滑曲线，更容易观察长期趋势。
2.  **匹配策略周期**：如果交易策略基于周线或月线，就需要相应频率的数据。
3.  **数据对齐**：便于不同频率或不同来源的数据进行合并分析。

## 量化交易入门：P9：3-短均与长均计算实例 📊

均线是技术分析的核心工具。本节我们通过计算短期和长期均线来具体演示。

移动平均线主要有两种：**简单移动平均（SMA）** 和 **指数移动平均（EMA）**。EMA对近期价格赋予更高权重，反应更敏捷。

以下是使用Pandas计算SMA和EMA的示例：

```python
import pandas as pd

# 假设df包含‘close’列
close_prices = df['close']

# 计算20日简单移动平均线
short_sma = close_prices.rolling(window=20).mean()

# 计算60日简单移动平均线
long_sma = close_prices.rolling(window=60).mean()

# 计算12日指数移动平均线
short_ema = close_prices.ewm(span=12, adjust=False).mean()

# 计算26日指数移动平均线
long_ema = close_prices.ewm(span=26, adjust=False).mean()
```

**短期均线**（如20日）对价格变化更敏感，能更快反映近期趋势。
**长期均线**（如60日）更平滑，代表长期趋势方向。

将短均线和长均线绘制在同一张图上，是分析趋势强弱和潜在转折点的常用方法。两者之间的相互关系（如交叉）能产生重要的交易信号，这将在后续章节详细探讨。

## 量化交易入门：P10：4-指标相关情况分析 🤝

单独看一个指标往往不够，分析多个指标之间的关系能提供更深入的洞察。

指标相关性分析旨在研究两个或多个技术指标之间是否存在统计上的关联关系。例如，股价上涨时，成交量指标是否通常也同步放大？

最常用的度量是**皮尔逊相关系数**，它衡量两个变量之间的线性相关程度，取值范围在-1到1之间。
*   **+1** 表示完全正相关。
*   **0** 表示无线性相关。
*   **-1** 表示完全负相关。

在Python中，可以轻松计算相关系数矩阵：

```python
# 假设 DataFrame `indicators` 包含了多个技术指标列
correlation_matrix = indicators.corr()
print(correlation_matrix)
```

通过分析相关系数矩阵，我们可以：
1.  **避免冗余**：如果两个指标高度相关（如相关系数>0.8），它们可能提供相似的信息，在构建多因子模型时可考虑只保留一个。
2.  **发现协同效应**：确认某些指标确实存在预期的同步或反向关系。
3.  **验证逻辑**：检查指标表现是否符合市场常识或策略假设。

## 量化交易入门：P11：5-回归方程与相关系数实例 📉

让我们通过一个具体的回归分析实例，来量化指标与股价回报之间的关系。

我们不仅想知道两个指标是否相关，还想知道它们之间具体的数量关系。这可以通过线性回归来实现。

假设我们想研究“过去20日的动量”（一个技术指标）与“未来5日的收益率”之间的关系。我们可以建立如下一元线性回归模型：

`未来收益率 = α + β * 动量指标 + ε`

其中：
*   `α` 是截距项。
*   `β` 是回归系数，表示动量指标每变动一个单位，未来收益率平均变动多少。
*   `ε` 是随机误差。

使用`statsmodels`或`scikit-learn`库可以进行回归分析：

```python
import statsmodels.api as sm

# X是动量指标，y是未来收益率
X = sm.add_constant(momentum_indicator)  # 添加常数项（截距）
model = sm.OLS(y, X).fit()
print(model.summary())  # 输出详细的回归结果
```

回归结果中的 `R-squared` 表示模型解释力，`β` 系数的 `P-value` 用于判断其是否显著不为零。如果 `β` 显著为正，说明该动量指标对预测未来收益有正向作用。这个实例将相关性分析推进到了更具预测性和可操作性的阶段。

## 量化交易入门：P12：1-金叉与死叉介绍 ✖️➜✔️

技术指标的重要应用之一是生成交易信号。金叉和死叉就是由均线系统产生的经典信号。

**金叉**：指短期移动平均线由下向上穿越长期移动平均线。这通常被视为**买入信号**，意味着短期趋势开始转强，可能开启一波上涨行情。

**死叉**：指短期移动平均线由上向下穿越长期移动平均线。这通常被视为**卖出信号**，意味着短期趋势开始走弱，可能面临一波下跌。

以下是判断金叉和死叉的逻辑代码示意：

```python
# 计算短期均线（如5日）和长期均线（如20日）
df['short_ma'] = df['close'].rolling(window=5).mean()
df['long_ma'] = df['close'].rolling(window=20).mean()

# 判断金叉：昨日短均 <= 长均，且今日短均 > 长均
df['golden_cross'] = (df['short_ma'].shift(1) <= df['long_ma'].shift(1)) & (df['short_ma'] > df['long_ma'])

# 判断死叉：昨日短均 >= 长均，且今日短均 < 长均
df['death_cross'] = (df['short_ma'].shift(1) >= df['long_ma'].shift(1)) & (df['short_ma'] < df['long_ma'])
```

金叉和死叉是趋势跟踪策略的基础。然而，它们具有滞后性，在震荡市中可能产生频繁的假信号，因此在实际使用中需要结合其他指标或进行参数优化。

## 量化交易入门：P13：2-买点与卖点可视化分析 📍

信号需要被直观地验证。本节我们学习如何在价格图上可视化标记买卖点。

将计算出的金叉（买点）和死叉（卖点）在股价K线图或走势图上标记出来，可以直观地评估信号出现的时机和质量。

使用 `Matplotlib` 库可以方便地实现：

```python
import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(14, 7))
ax.plot(df.index, df['close'], label='Close Price', alpha=0.5)
ax.plot(df.index, df['short_ma'], label='5-day MA', alpha=0.8)
ax.plot(df.index, df['long_ma'], label='20-day MA', alpha=0.8)

# 用绿色向上箭头标记金叉（买点）
buy_signals = df[df['golden_cross']]
ax.scatter(buy_signals.index, buy_signals['close'], color='green', marker='^', s=100, label='Buy Signal')

# 用红色向下箭头标记死叉（卖点）
sell_signals = df[df['death_cross']]
ax.scatter(sell_signals.index, sell_signals['close'], color='red', marker='v', s=100, label='Sell Signal')

ax.set_title('Stock Price with MA Cross Signals')
ax.legend()
plt.show()
```

通过可视化，我们可以轻松观察：
1.  信号是否出现在趋势启动的早期。
2.  在震荡区间是否产生了大量无效（亏损）信号。
3.  买卖点之间的持仓周期大致多长。
这种直观分析是优化策略参数和过滤假信号的第一步。

## 量化交易入门：P14：3-策略收益效果分析 💰

生成信号并可视化后，最关键的一步是量化评估策略的盈利潜力。

策略收益分析旨在通过历史数据回测，计算如果严格按照策略信号进行买卖所能获得的收益。我们通常将策略收益与简单持有股票（买入并持有）的收益进行对比。

核心计算步骤如下：
1.  **生成持仓信号**：根据买卖点，生成每天的持仓状态（1代表持有，0代表空仓）。
2.  **计算每日收益率**：`当日收益率 = (当日收盘价 / 前日收盘价) - 1`
3.  **计算策略每日收益率**：`策略日收益率 = 持仓状态 * 股票日收益率`。当空仓时，收益率为0（假设无风险收益为0）。
4.  **计算累计收益**：`累计收益 = (1 + 策略日收益率).cumprod() - 1`

```python
# 简化的收益计算
df['strategy_returns'] = df['position'].shift(1) * df['daily_returns']  # 使用前一天的持仓决定今天的收益
df['strategy_cumulative'] = (1 + df['strategy_returns']).cumprod()
df['buy_hold_cumulative'] = (1 + df['daily_returns']).cumprod()
```

通过绘制策略累计收益曲线和买入持有收益曲线的对比图，可以一目了然地看出策略是否跑赢了市场基准，以及在哪些时间段表现突出或欠佳。

## 量化交易入门：P15：4-均线调参实例 🎯

没有一个参数放之四海而皆准。本节我们通过实例学习如何优化均线策略的参数。

在双均线交叉策略中，短期均线周期和长期均线周期（如5日和20日）是可调参数。不同的参数组合会产生不同的信号频率和收益效果。

调参过程通常是一个网格搜索：
1.  定义参数范围：例如，短均线周期 `short_period` 从3到10，长均线周期 `long_period` 从20到50。
2.  遍历所有参数组合。
3.  对每一组参数，在历史数据上运行策略，计算最终的总收益率、夏普比率等评价指标。
4.  选择表现最优的参数组合。

```python
best_return = -float('inf')
best_params = (0, 0)

for short in range(3, 11):
    for long in range(20, 51):
        if short >= long:
            continue  # 短周期必须小于长周期
        # 使用当前(short, long)参数计算策略收益
        cumulative_return = calculate_strategy_return(df, short, long)
        if cumulative_return > best_return:
            best_return = cumulative_return
            best_params = (short, long)

print(f"最优参数：短周期={best_params[0]}, 长周期={best_params[1]}, 累计收益={best_return:.2%}")
```

需要注意的是，在历史数据上表现最优的参数，在未来不一定有效，可能存在过度拟合。因此，调参后需要在样本外数据（回测时未使用的数据）上进行验证。

## 量化交易入门：P16：1-回测收益率指标解读 📊

一个策略的好坏需要多维度评估。从本节开始，我们系统学习量化策略的核心评价指标。

回测收益率是最直观的指标，它表示在回测期间，策略带来的总回报。通常有以下几种计算方式：
1.  **累计收益率**：策略从开始到结束的总收益。
    `累计收益率 = (最终净值 / 初始净值) - 1`
2.  **年化收益率**：将累计收益率折算到每年，便于比较不同时间长度的策略。
    `年化收益率 = (1 + 累计收益率)^(252 / 回测交易日数) - 1` （假设年交易日为252）
3.  **超额收益率**：策略收益率减去基准（如沪深300指数）收益率的差值。

单纯看收益率高低是片面的。一个收益率很高的策略，可能伴随着巨大的风险和回撤。因此，收益率必须与风险指标结合分析。

---

## 量化交易入门：P17：2-年化指标分析 📅

为了公平比较不同运行周期的策略，我们需要将收益和风险指标年化。

年化处理使得不同时间跨度的策略具有可比性。除了年化收益率，还有以下重要年化指标：

*   **年化波动率**：衡量策略收益的波动程度，是风险的基本度量。计算方式为策略日收益率的标准差乘以交易日数的平方根（通常取√252）。
    `年化波动率 = 日收益率标准差 * √252`

*   **年化下行波动率**：只考虑负收益率的波动率，更能反映投资者实际感受到的“亏损风险”。

在Python中，可以这样计算：

```python
import numpy as np

# 假设 daily_returns 是策略的日收益率序列
annual_return = (1 + daily_returns).prod() ** (252 / len(daily_returns)) - 1
annual_volatility = daily_returns.std() * np.sqrt(252)

# 计算下行波动率
negative_returns = daily_returns[daily_returns < 0]
downside_volatility = negative_returns.std() * np.sqrt(252) if len(negative_returns) > 0 else 0
```

年化指标为我们提供了一个统一的尺度，是进行策略筛选和组合管理的基石。

## 量化交易入门：P18：3-最大回撤区间 😰

最大回撤是衡量策略风险和控制能力的关键指标，它描述了投资者可能面临的最坏情况。

**最大回撤** 是指在选定周期内，资产净值从任一峰值点下跌到随后最低谷点的最大幅度。它衡量的是策略的**下行风险**。

计算公式为：
`最大回撤 = (峰值 - 谷值) / 峰值`

更重要的是**最大回撤区间**，即从峰值日期到谷值日期的这段时间。分析回撤区间内的市场环境、策略行为，对于改进策略至关重要。

计算最大回撤的示例：

```python
# 计算累计净值曲线
cumulative_returns = (1 + daily_returns).cumprod()

# 计算历史最高净值曲线（到当前时点的峰值）
running_max = cumulative_returns.expanding().max()

# 计算每日的回撤
drawdown = (cumulative_returns - running_max) / running_max

# 找到最大回撤及其区间
max_drawdown = drawdown.min()
max_drawdown_end = drawdown.idxmin()  # 回撤结束（谷底）日期
# 找到回撤开始的日期（谷底之前的最高点日期）
max_drawdown_start = running_max.loc[:max_drawdown_end].idxmax()
```

一个优秀的策略不仅要有高收益，还必须将最大回撤控制在可接受的范围内。过大的回撤可能导致投资者在黎明前因心理压力而止损退出。

## 量化交易入门：P19：4-夏普比率的作用 ⚖️

夏普比率是综合衡量策略“风险调整后收益”的最著名指标。

**夏普比率** 表示每承受一单位总风险（波动率），能获得多少超额回报（超过无风险利率的收益）。公式为：
`夏普比率 = (年化收益率 - 无风险利率) / 年化波动率`

*   **分子**：超额收益。吸引力所在。
*   **分母**：总风险（波动率）。成本所在。

夏普比率越高，说明策略在承担相同风险的情况下获得了更高的超额收益，即**性价比**更高。通常，夏普比率大于1被认为不错，大于2则非常优秀。

```python
# 假设 annual_return, risk_free_rate, annual_volatility 已定义
sharpe_ratio = (annual_return - risk_free_rate) / annual_volatility
```

**注意事项**：
1.  夏普比率假设收益服从正态分布，且投资者同等厌恶上行和下行波动，这与现实可能不符。
2.  它衡量的是总波动，对于主要担心亏损的投资者，**索提诺比率**（使用下行波动率作为分母）可能更合适。
尽管如此，夏普比率因其简洁有效，仍是业界最通用的绩效衡量标准之一。

## 量化交易入门：P20：5-阿尔法与贝塔概述 αβ

阿尔法和贝塔将策略收益分解为与市场相关的部分和与策略自身能力相关的部分。

这是从资产定价模型（如CAPM）中引申出的重要概念：
*   **贝塔**：衡量策略收益相对于市场基准收益的**敏感性**或**弹性**。它是一个风险系数。
    *   贝塔 = 1：策略波动与市场同步。
    *   贝塔 > 1：策略波动比市场更剧烈（进攻型）。
    *   贝塔 < 1：策略波动比市场平缓（防御型）。
*   **阿尔法**：衡量策略超越市场基准的**超额收益**部分，这部分收益通常归因于策略的选股或择时能力。阿尔法是所有主动管理者追求的目标。

它们可以通过以下回归方程得到：
`策略收益率 - 无风险利率 = α + β * (市场收益率 - 无风险利率) + ε`

一个理想的主动管理策略应该追求**高阿尔法、低贝塔**，这意味着它能独立于市场涨跌创造稳定收益。而高贝塔带来的高收益可能只是承担了更高的市场系统风险，并非真正的管理技能。

## 量化交易入门：P21：1-量化交易概述 🤖

在学习了基础分析和评价指标后，我们现在正式对量化交易进行一个全局概述。

量化交易是一种利用数学模型、统计分析和计算机算法来识别和执行交易机会的方法。其核心思想是将投资逻辑转化为可验证、可重复的规则系统，以克服人类情绪和认知偏差的影响。

量化交易的主要特点包括：
*   **系统性**：基于明确的规则，而非主观感觉。
*   **纪律性**：严格执行策略信号，避免情绪化交易。
*   **回溯检验**：在历史数据上验证策略的有效性。
*   **自动化**：由计算机程序自动执行交易。

其主要应用领域涵盖股票、期货、期权、外汇、加密货币等几乎所有流动性高的金融市场。从简单的均线交叉到复杂的机器学习模型，都属于量化交易的范畴。

## 量化交易入门：P24：1-策略任务分析 🎯

从本节开始，我们将进入多因子选股策略的实战环节。首先明确策略任务。

多因子选股是量化选股的主流方法，其核心思想是：找到一系列对股票未来收益有预测能力的指标（因子），通过综合这些因子的信息来筛选出预期表现更好的股票。

构建一个多因子选股策略，通常需要完成以下任务：
1.  **因子选取**：基于金融逻辑或数据挖掘，选择候选因子（如估值因子、成长因子、动量因子、质量因子等）。
2.  **数据处理**：对原始因子数据进行清洗、去极值、标准化、中性化等预处理。
3.  **因子合成**（可选）：将多个单因子合成为一个综合因子。
4.  **股票筛选**：根据因子值对股票进行排序或打分，选择排名靠前的股票构成投资组合。
5.  **组合构建与调仓**：确定投资权重，并按照固定周期（如每月）进行调仓。
6.  **回测评估**：对策略历史表现进行回测，计算各项绩效指标。

我们将按照这个流程，一步步实现一个简单的多因子策略。
