## 计算原始增长率

上一节我们介绍了如何获取和处理股价数据，本节中我们来看看如何计算股价的每日增长率。

在量化分析中，我们通常使用对数收益率来计算增长率，因为它具有更好的数学性质（如可加性）。计算对数收益率的公式如下：

**公式：** `log_return = log(price_t / price_{t-1})`

其中，`price_t` 代表当前交易日的股价，`price_{t-1}` 代表前一个交易日的股价。

![](img/332cc8108814706e68a07cfb5e455278_5.png)

以下是使用Python代码计算对数收益率的步骤：

![](img/332cc8108814706e68a07cfb5e455278_7.png)

```python
# 假设 df 是包含股价的DataFrame，列名为 ‘Close’
df[‘returns’] = np.log(df[‘Close’] / df[‘Close’].shift(1))
```

这段代码的作用是：
1.  使用 `.shift(1)` 将股价序列向下移动一位，得到前一天的股价。
2.  用当天的股价除以前一天的股价。
3.  对结果取自然对数 `np.log`，得到每日的对数收益率。

![](img/332cc8108814706e68a07cfb5e455278_9.png)

执行后，我们会在数据框中得到一列名为 `returns` 的增长率数据。

## 应用策略计算策略收益

![](img/332cc8108814706e68a07cfb5e455278_11.png)

![](img/332cc8108814706e68a07cfb5e455278_13.png)

我们已经有了股价的原始增长率，现在需要将我们的交易策略应用进去。回顾一下，我们的“双均线策略”会生成一个交易信号 `position`，这个信号指示我们应该如何操作。

`position` 的值代表我们的持仓状态：
*   当 `position = 1` 时，表示我们预测股价会上涨，因此**跟随**原始走势（做多）。
*   当 `position = -1` 时，表示我们预测股价会下跌，因此**反向**操作原始走势（例如做空或空仓，在简化模型中我们视为收益取反）。

![](img/332cc8108814706e68a07cfb5e455278_15.png)

因此，策略的每日收益可以简单地用信号值乘以原始收益率来计算。

**公式：** `strategy_return = position * return`

![](img/332cc8108814706e68a07cfb5e455278_17.png)

同样，在计算时，我们需要确保 `position` 信号与收益率在时间上对齐，因此也需要对 `position` 序列进行移位操作。

以下是计算策略收益的代码：

![](img/332cc8108814706e68a07cfb5e455278_19.png)

```python
# 假设 df 中已有 ‘returns’（原始收益率）和 ‘position’（策略信号）列
df[‘strategy’] = df[‘position’].shift(1) * df[‘returns’]
```

![](img/332cc8108814706e68a07cfb5e455278_21.png)

这段代码将前一天的交易信号 `position` 应用到当天的收益率上，计算出策略当天的收益。

![](img/332cc8108814706e68a07cfb5e455278_23.png)

## 比较策略与原始收益

![](img/332cc8108814706e68a07cfb5e455278_25.png)

计算完每日的策略收益后，我们需要评估整个时间段内策略的总体表现，并与什么都不做的“买入并持有”策略进行对比。

![](img/332cc8108814706e68a07cfb5e455278_27.png)

由于我们使用的是对数收益率，要得到最终的总收益，需要将所有每日的收益率相加，然后通过指数函数 `exp()` 将其还原为实际的资金倍数。

![](img/332cc8108814706e68a07cfb5e455278_29.png)

**公式：**
1.  总对数收益率 = `sum(每日收益率)`
2.  最终资金倍数 = `exp(总对数收益率)`

以下是计算并比较两种方式最终收益的代码：

```python
# 计算总对数收益率
total_log_return_buy_hold = df[‘returns’].sum()
total_log_return_strategy = df[‘strategy’].sum()

![](img/332cc8108814706e68a07cfb5e455278_31.png)

# 还原为最终资金倍数
final_multiple_buy_hold = np.exp(total_log_return_buy_hold)
final_multiple_strategy = np.exp(total_log_return_strategy)

![](img/332cc8108814706e68a07cfb5e455278_33.png)

print(f“买入并持有策略：1 元最终变为 {final_multiple_buy_hold:.2f} 元”)
print(f“双均线策略：1 元最终变为 {final_multiple_strategy:.2f} 元”)
```

![](img/332cc8108814706e68a07cfb5e455278_35.png)

通过对比这两个结果，我们可以直观地看到应用交易策略是否带来了超额收益。例如，在示例中：
*   买入并持有：1元变成了约4元。
*   双均线策略：1元变成了约5.8元。

![](img/332cc8108814706e68a07cfb5e455278_37.png)

这表明，在该历史数据上，我们的双均线策略比简单的买入并持有策略获得了更高的收益。

## 总结

本节课中我们一起学习了量化策略收益分析的核心步骤：
1.  **计算原始增长率**：使用对数收益率公式计算股价的每日涨跌。
2.  **计算策略收益**：将交易策略生成的信号（`position`）与原始收益率结合，得到策略的每日收益。关键公式为 `strategy_return = position * return`。
3.  **评估策略效果**：通过累加对数收益率并还原为资金倍数，对比策略收益与基准收益（买入并持有），从而量化策略的有效性。

![](img/332cc8108814706e68a07cfb5e455278_39.png)

这个过程是回测任何交易策略的基础，帮助你用数据客观地评估一个策略的潜在盈利能力。
