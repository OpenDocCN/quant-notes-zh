## 图表的意义

![](img/3874f799b19994e298e16303053cad5e_9.png)

![](img/3874f799b19994e298e16303053cad5e_10.png)

上一节我们介绍了数据分析的基本概念，本节中我们来看看为什么需要图表。

![](img/3874f799b19994e298e16303053cad5e_12.png)

在数据分析中，我们不仅要对数据进行处理，还需要将分析结果清晰地呈现给他人。文字描述有时难以传达数据的核心信息，而图表能提供强大的视觉冲击力，使信息一目了然。

例如，描述一只猫的可爱之处，用语言可能效果有限。但展示一张猫的照片，其可爱程度便不言而喻。同样，面对一列营收数据，人们可能难以快速理解。若将其转化为图表，例如柱状图，便能立刻看出“广州地区营收情况较好”这一结论。图表使汇报内容更易被接受和理解，是传达分析思想的有效方式。

## 创建第一个图表：柱状图

理解了图表的意义后，本节我们来动手创建最简单的图表——柱状图。

以下是创建柱状图的步骤：
1.  准备数据：例如“超一线城市2018年营收”数据，包含城市名称和对应营收数值。
2.  选择数据区域。
3.  点击Excel的 **`插入`** 选项卡。
4.  点击 **`推荐的图表`** 或直接选择柱状图图标。
5.  在弹出的窗口中确认选择柱状图，点击 **`确定`**。

![](img/3874f799b19994e298e16303053cad5e_14.png)

![](img/3874f799b19994e298e16303053cad5e_15.png)

一个基本的柱状图便创建完成。图表直观地展示了各城市营收数据的对比。

![](img/3874f799b19994e298e16303053cad5e_17.png)

## 图表的组成部分

![](img/3874f799b19994e298e16303053cad5e_19.png)

成功创建图表后，我们需要认识图表的各个部分，以便后续进行自定义设置。

![](img/3874f799b19994e298e16303053cad5e_21.png)

一个典型的图表由以下核心部分构成：
*   **图表标题**：表达图表的中心思想，例如“2018年超一线城市营收”。
*   **绘图区**：图表的核心区域，包含数据图形（如柱体）。
*   **数据系列**：绘图区中代表同一类数据的一组图形，例如“北京”系列柱体。
*   **数据标签**：显示在图形上方的具体数值。
*   **坐标轴**：
    *   **水平轴（X轴）**：通常显示分类信息，如城市名称。
    *   **垂直轴（Y轴）**：通常显示数值大小，如营收金额。
*   **图例**：说明不同数据系列代表的含义（例如用不同颜色区分2018年和2019年数据）。
*   **网格线**：辅助查看数据大小的背景线条。

![](img/3874f799b19994e298e16303053cad5e_23.png)

认识这些部分非常重要，因为当我们需要修改图表样式时，必须准确选中对应的部分。选中某个部分（如绘图区、标题或坐标轴）后右键单击，选择 **`设置...格式`**，即可进入对应的格式设置面板，进行颜色、边框、字体等个性化调整。

![](img/3874f799b19994e298e16303053cad5e_25.png)

## 更多图表类型与应用场景

![](img/3874f799b19994e298e16303053cad5e_26.png)

![](img/3874f799b19994e298e16303053cad5e_28.png)

掌握了柱状图的基础后，本节我们来看看其他几种常用的图表类型及其适用场景。

![](img/3874f799b19994e298e16303053cad5e_30.png)

### 条形图

条形图与柱状图类似，区别在于其数据条是水平方向的。它适用于分类名称较长或需要水平对比数据的场景。

![](img/3874f799b19994e298e16303053cad5e_32.png)

![](img/3874f799b19994e298e16303053cad5e_34.png)

![](img/3874f799b19994e298e16303053cad5e_35.png)

以下是创建条形图的步骤：
1.  选择数据（例如，不同歌曲在5月份的播放量）。
2.  点击 **`插入`** -> **`推荐的图表`**。
3.  选择条形图并确认。

![](img/3874f799b19994e298e16303053cad5e_37.png)

创建后，可以点击图表旁的 **`+`** 号添加或删除元素，如数据标签。通常，为了图表美观，我们会删除网格线。

### 折线图

![](img/3874f799b19994e298e16303053cad5e_39.png)

![](img/3874f799b19994e298e16303053cad5e_41.png)

折线图用于显示数据随时间或其他连续变量变化的趋势。

![](img/3874f799b19994e298e16303053cad5e_43.png)

以下是创建折线图的步骤：
1.  选择数据（例如，一首歌在12个月内的播放量）。
2.  点击 **`插入`** -> **`推荐的图表`** -> 选择折线图。
3.  点击确定，即可看到播放量随时间波动的趋势线。

![](img/3874f799b19994e298e16303053cad5e_45.png)

折线图也支持多系列对比，可以同时展示多首歌的趋势，便于分析不同类别的走势差异。

### 散点图

![](img/3874f799b19994e298e16303053cad5e_47.png)

散点图用于展示两个变量之间的相关关系，由一系列点构成。

![](img/3874f799b19994e298e16303053cad5e_49.png)

![](img/3874f799b19994e298e16303053cad5e_50.png)

![](img/3874f799b19994e298e16303053cad5e_51.png)

![](img/3874f799b19994e298e16303053cad5e_53.png)

以下是创建和分析散点图的步骤：
1.  选择两列数值数据（例如，身高和体重）。
2.  点击 **`插入`** -> **`推荐的图表`** -> 选择 **`XY散点图`**。
3.  点击确定后，图表会显示身高与体重的分布情况。
4.  为了更清晰地观察趋势，可以右键点击数据点，选择 **`添加趋势线`**。

要量化相关程度，可以使用Excel的“数据分析”工具计算相关系数。首先需要在 **`文件`** -> **`选项`** -> **`加载项`** -> **`Excel加载项`** 中勾选 **`分析工具库`**。启用后，在 **`数据`** 选项卡下点击 **`数据分析`**，选择 **`相关系数`**，指定数据区域即可计算。相关系数越接近1或-1，表示相关性越强。

![](img/3874f799b19994e298e16303053cad5e_55.png)

## 图表的进阶美化与标注

![](img/3874f799b19994e298e16303053cad5e_57.png)

![](img/3874f799b19994e298e16303053cad5e_59.png)

学习了各种图表的基本创建后，本节我们来看看如何对图表进行美化和重点标注，以增强其表达力。

![](img/3874f799b19994e298e16303053cad5e_61.png)

我们可以对图表的特定部分进行突出显示。以散点图为例，若想强调某个偏离主体的数据点：
1.  单击该数据点一次选中所有点，再单击一次即可单独选中它。
2.  右键选择 **`设置数据点格式`**。
3.  在 **`标记`** 选项中，可以调整点的 **`大小`**、**`填充颜色`** 和 **`边框`**，使其与众不同。

此外，我们还可以在图表中添加说明文字或形状：
*   **插入文本框**：在 **`插入`** 选项卡中选择 **`文本框`**，在图表上绘制并输入文字（如“异常点”），用于解释特殊数据。
*   **插入形状**：在 **`插入`** 选项卡中选择 **`形状`**，如箭头，可以引导观众视线，强调关键信息。

通过这些操作，我们可以让图表不仅展示数据，更能清晰地讲述数据背后的故事。
