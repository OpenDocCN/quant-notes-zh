# 金融量化AI实战：P31：5-标准化处理方法 📊

在本节课中，我们将要学习数据预处理中一个至关重要的步骤——标准化。标准化是机器学习与数据分析中的基础操作，其目的是消除不同特征（或指标）之间由于量纲和取值范围不同所带来的影响，使模型能够更公平地“看待”每一个输入特征。

## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看如何通过标准化来统一数据的尺度。

![](img/70958f3a34ca5a3802486c1bde692318_1.png)

![](img/70958f3a34ca5a3802486c1bde692318_2.png)

假设我们有一组数据，包含两个特征X1和X2。从下图可以直观地看到，X1和X2的取值范围存在明显差异。X1的取值范围较大，而X2的取值范围相对较小。

![](img/70958f3a34ca5a3802486c1bde692318_1.png)

这种取值范围的不同会带来一个问题：在后续的模型计算中，取值范围大的特征可能会对结果产生不成比例的巨大影响，从而掩盖了其他特征的作用。标准化的核心目的，就是为了让所有特征在数值层面上处于**相近的尺度**，以便模型能够平等地学习每个特征的信息。

## 标准化的公式分解

标准化的操作遵循一个明确的数学公式。其核心思想分为两步：去中心化和缩放。

标准化的计算公式如下：
**`z = (x - μ) / σ`**
其中：
*   `x` 是原始数据。
*   `μ` 是该特征所有数据的**均值**。
*   `σ` 是该特征所有数据的**标准差**。

### 第一步：去均值（中心化）

首先，我们来看公式中的 `(x - μ)` 部分。这个操作被称为“去均值”或“中心化”。

它的作用是**将数据的分布中心移动到零点**。如下图所示，原始数据可能并不以坐标原点为中心。减去均值μ后，新的数据会以零为中心对称分布。这解决了数据偏移的问题。

![](img/70958f3a34ca5a3802486c1bde692318_4.png)

### 第二步：除以标准差（缩放）

然而，仅仅去中心化还不够。如下图所示，即使数据以零为中心对称了，不同特征（X1和X2）的**取值范围（离散程度）** 依然不同。X1的数据点更分散，X2的数据点更集中。

![](img/70958f3a34ca5a3802486c1bde692318_5.png)

这时就需要公式中的 `/ σ` 部分，即“除以标准差”。标准差（σ）是衡量数据离散程度的指标。数据越分散，标准差越大；数据越集中，标准差越小。

通过这个除法操作：
*   对于原本取值范围大、离散程度高的特征（标准差大），除以一个大数，会使其数值范围缩小。
*   对于原本取值范围小、离散程度低的特征（标准差小），除以一个小数，其数值范围变化相对不大。

最终结果是，所有特征都被缩放至一个以0为中心、标准差为1的**标准正态分布**的尺度上，它们的取值范围变得近似相同。

## 标准化的代码实现

理解了原理后，我们可以动手实现标准化。以下是使用Python和NumPy库手动实现标准化的示例代码：

![](img/70958f3a34ca5a3802486c1bde692318_4.png)

![](img/70958f3a34ca5a3802486c1bde692318_5.png)

```python
import numpy as np

def z_score_normalization(data):
    """
    执行Z-score标准化
    参数:
        data: 输入数据，假设为二维NumPy数组，形状为 (n_samples, n_features)
    返回:
        标准化后的数据
    """
    # 计算均值和标准差
    # axis=0 表示沿着列（特征）方向计算
    mean_val = np.mean(data, axis=0)
    std_val = np.std(data, axis=0)

    # 应用标准化公式: (x - μ) / σ
    normalized_data = (data - mean_val) / std_val

    return normalized_data
```

![](img/70958f3a34ca5a3802486c1bde692318_7.png)

代码说明：
1.  `np.mean(data, axis=0)`：计算每个特征列的均值。
2.  `np.std(data, axis=0)`：计算每个特征列的标准差。
3.  `(data - mean_val) / std_val`：对每个数据点应用标准化公式。

## 使用工具包快速标准化

在实际项目中，我们通常使用成熟的机器学习库来快速完成此类操作，例如Scikit-learn。

以下是使用Scikit-learn的`StandardScaler`进行标准化的方法：

```python
from sklearn.preprocessing import StandardScaler

![](img/70958f3a34ca5a3802486c1bde692318_11.png)

# 初始化标准化器
scaler = StandardScaler()
# 拟合数据并计算均值和标准差
scaler.fit(original_data)
# 执行转换
normalized_data = scaler.transform(original_data)
# 或者使用 fit_transform 一步完成拟合和转换
# normalized_data = scaler.fit_transform(original_data)
```

![](img/70958f3a34ca5a3802486c1bde692318_12.png)

![](img/70958f3a34ca5a3802486c1bde692318_13.png)

使用`StandardScaler`的优势在于，它能够方便地保存计算出的均值和标准差，用于对后续的新数据（如测试集）进行**相同尺度**的转换。

![](img/70958f3a34ca5a3802486c1bde692318_14.png)

## 总结

![](img/70958f3a34ca5a3802486c1bde692318_22.png)

![](img/70958f3a34ca5a3802486c1bde692318_23.png)

本节课中我们一起学习了数据标准化处理。
*   **目的**：消除不同特征间量纲和取值范围的差异，使模型训练更稳定、高效。
*   **原理**：通过 **`z = (x - μ) / σ`** 公式，先进行去中心化，再进行缩放。
*   **实现**：可以手动使用NumPy计算，也可以便捷地调用Scikit-learn等库的`StandardScaler`工具。
*   **效果**：处理后的数据以0为中心，各特征尺度一致，更适合大多数机器学习算法。
