# 量化交易完全可自学教程：P47：因子打分法选股策略概述 📊

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分法。我们将了解其核心思想、实施步骤以及背后的逻辑。

## 概述

之前我们介绍了如何通过单一因子（如市盈率）来判断股票的好坏。然而，在实际投资中，一个股票的表现往往由多个因素共同决定。本节我们将探讨一种更综合的方法：因子打分选股策略。该策略的核心思想是，为每只股票的多个因子分别打分，然后计算总分，最后根据总分排名来筛选出表现最佳的股票组合。

## 策略核心思想

上一节我们介绍了单一因子分析，本节中我们来看看如何综合多个因子进行决策。

因子打分选股策略，顾名思义，就是为影响股票表现的各个“因子”进行评分。这里的“因子”可以理解为衡量股票不同维度的指标，例如估值、成长性、质量、市场预期等。在每个大类因子下，又有具体的指标，例如市盈率（估值因子）、净资产收益率（质量因子）等。

我们之前讲解的方法，例如计算因子的Z值，更像是基于单一因子来评判股票。而今天的策略将其放大，进行综合评估。好比学生的总成绩由各科成绩加总而成，股票的综合得分也由其各个因子的得分加总决定。我们通过计算每只股票在所有选定因子上的总分，并进行排名，从而选出总分最高的股票构建投资组合。

## 策略实施步骤

以下是实施因子打分法选股策略的关键步骤。

![](img/edc5c7b099bae2370f4a6f2842f74a50_1.png)

![](img/edc5c7b099bae2370f4a6f2842f74a50_3.png)

1.  **选择因子**：首先，需要确定使用哪些因子来评价股票。例如，可以选择四个因子：A（市盈率）、B（市净率）、C（净资产收益率）、D（营收增长率）。
2.  **确定因子方向**：对于每个选定的因子，必须事先知道其与预期收益率的相关性是正向还是负向。例如，净资产收益率（ROE）通常是“越高越好”，而市盈率（PE）在某些策略中可能是“越低越好”。这个知识可以来源于金融研究报告或历史数据的统计分析。
3.  **为因子打分**：根据每只股票在每个因子上的具体数值，按照其方向性（越高越好或越低越好）进行打分。数值越有利，得分越高。
4.  **计算总分**：将每只股票在所有因子上的得分相加，得到该股票的综合总分。
    **公式**：`股票总分 = 因子A得分 + 因子B得分 + 因子C得分 + 因子D得分`
5.  **排名与筛选**：对所有股票的总分进行排名，选取排名靠前的股票（如前10名或前20名）作为当期投资组合。
6.  **定期调仓**：该策略通常定期执行（例如每月一次），根据最新的数据重新计算分数、排名并调整投资组合。

## 关键前提与已知条件

在具体操作之前，我们必须明确一些关键的前提条件。

首先，我们必须知道每个因子与收益率的相关方向（正或负），因为这是打分的基础。例如，对于“每股收益”和“净资产收益率”这类因子，通常认为数值越大越好。这些信息可以作为先验知识，从公开的研究报告中获得，也可以通过我们之前学习的因子分析方法自行计算验证。

其次，在后续的实际编程演示中，我们会预先选择两组因子：一组是“越高越好”的因子，另一组是“越低越好”的因子。这确保了我们的打分标准是清晰且一致的。
