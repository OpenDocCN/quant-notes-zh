1.  **中心化函数参数缺失**：在调用中性化函数时，需要传入正确的参数，包括指定的因子和返回DataFrame中的列名。
    ```python
    # 错误示例：缺少列名参数
    result = sm.OLS(y, x).fit()
    # 修正后：传入正确的列名
    result = sm.OLS(y, x).fit()
    df['resid'] = result.resid  # 假设列名为‘resid’
    ```

2.  **缺少模型训练步骤**：在进行中性化等操作前，必须确保已经对模型（如线性回归）进行了拟合（`.fit()`）。
    ```python
    # 错误示例：未进行拟合就使用结果
    model = sm.OLS(y, x)
    # 修正后：先进行拟合
    model = sm.OLS(y, x).fit()
    resid = model.resid
    ```

3.  **函数或变量名拼写错误**：仔细检查代码中的拼写，例如 `quantile` 误写为 `qant`，`list` 误写为 `lis` 等。
    ```python
    # 错误示例
    stock_list = get_stock_lis()
    # 修正后
    stock_list = get_stock_list()
    ```

以下是评估策略效果时需关注的核心指标：

![](img/a377c8b7c555e847688af3319710c417_80.png)

![](img/a377c8b7c555e847688af3319710c417_82.png)

![](img/a377c8b7c555e847688af3319710c417_83.png)

*   **累计收益率**：策略从开始到结束的总收益。对比策略收益与基准（如沪深300指数）收益，判断策略是否跑赢大盘。
*   **年化收益率**：将累计收益率折算成年化收益，便于与其他策略或投资产品比较。
*   **最大回撤**：策略在运行期间，资产净值从最高点到最低点的最大跌幅。**最大回撤越小，说明策略控制风险的能力越强**。
*   **夏普比率**：衡量每承受一单位风险，能产生多少超额回报。**夏普比率越高，说明风险调整后的收益越好**。
    `夏普比率 = (策略年化收益率 - 无风险利率) / 策略年化波动率`
*   **交易详情**：观察具体的调仓记录，验证策略逻辑是否按预期执行（例如，每月初是否进行了调仓）。

本节课中我们一起学习了量化策略的效果评估流程。我们首先运行策略并调试代码错误，然后通过分析累计收益、年化收益、最大回撤和夏普比率等关键指标，来综合评价策略的盈利能力和风险控制水平。这个过程是策略迭代优化中至关重要的一环，帮助我们判断策略是否有效，并为后续改进提供方向。
