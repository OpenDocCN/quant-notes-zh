## 📚 概述

在本节课中，我们将要学习机器学习在量化交易中的核心应用之一：特征选择。我们将探讨如何从原始数据中筛选出高质量的特征，并深入介绍两种在实践中极其有效但理论背景相对“玄学”的算法——遗传算法和神经网络（特别是其核心训练算法：反向传播）。掌握这些方法将帮助你构建更稳健、更强大的量化交易模型。

## 🎯 第1节：特征选择的意义与方法

上一节我们学习了如何从原始数据中构建训练集（即每个时间点对应的特征X和目标Y）。本节中我们来看看如何优化这些特征。

特征选择（Feature Selection）的核心在于：我们最初选择的特征（X）往往是凭经验或直觉选取的，可能包含冗余、无关甚至全为零的列。低质量的特征会严重影响任何分类器或回归器的性能。因此，我们需要通过优化方法来筛选出一组“开了挂”的X。

特征选择主要有三种思路：

1.  **子集选择（Subset Selection）**：从原始特征（例如300个）中直接选出一个子集（例如30个）用于建模。选出的特征值本身不变。
2.  **系数收缩（Shrinkage）**：不直接剔除特征，而是通过让模型系数有规律地变化（例如某些系数自动变为零）来实现选择。我们之前学过的Lasso回归干的就是这件事。
3.  **降维（Dimensionality Reduction）**：将原始的高维特征（例如300维）通过某种函数变换为低维的新特征（例如20维）。新特征的每一列都是原始特征的某种线性或非线性组合，其物理意义已发生改变。主成分分析（PCA）、自编码器（Autoencoder）等都是降维方法。

## 🔍 第2节：子集选择的具体策略

本节我们聚焦于第一种方法：如何选出一个最优的特征子集。

评价一个特征子集好坏的标准，通常使用回归任务中的R方值（R-squared）。R方值越接近1，说明模型拟合越好。我们的目标是找到一个能使R方值最大的特征子集。

如果暴力枚举所有可能的特征组合，其计算复杂度是指数级的，不可行。因此，我们需要一些逼近最优解的启发式方法。

以下是几种常见的策略：

*   **前向选择（Forward Selection）**：这是一种贪心算法。开始时模型不包含任何特征。然后，依次尝试加入每一个特征，并保留能使模型性能（如R方）提升最大的那个特征。接着，在已选特征的基础上，再次尝试加入剩余特征中能带来最大提升的那个。如此循环，直到性能不再显著提升为止。
*   **后向选择（Backward Selection）**：与前向选择相反。开始时模型包含所有特征。然后，尝试剔除每一个特征，并保留对模型性能损害最小的那个（即剔除后R方下降最少）。如此循环，直到不能再剔除为止。
*   **混合方法**：结合前向和后向选择，每一步可能考虑增加或删除特征。

这些方法虽然简单，但在实际中经常被使用。它们为后续更复杂的优化算法（如遗传算法）奠定了基础。

## 🧬 第3节：遗传算法（Genetic Algorithm）原理

现在，我们进入本节课第一个重点内容：遗传算法。这是一种受生物进化论启发的优化算法，广泛应用于各种复杂优化问题，包括特征选择和投资组合优化。

遗传算法的理论依据很简单：
1.  **种群与适者生存**：一个包含许多个体（解）的种群，适应环境（目标函数）的个体存活并繁衍，不适应的被淘汰。
2.  **基因决定性状**：个体的好坏由其“基因”（编码）决定。

在计算机中，我们这样建模：
*   **基因型（Genotype）**：用一串编码（通常是二进制串）表示一个解。例如，在特征选择中，`10011001`可以表示选择第1、4、5、8个特征，不选其他特征。
*   **表现型（Phenotype）**：基因型所对应的实际解。在特征选择中，就是被选中的那组特征。
*   **适应度函数（Fitness Function）**：评价一个解好坏的函数。在特征选择中，就是使用该特征子集建模后得到的R方值。我们的目标是找到适应度最高的个体。

遗传算法通过模拟生物进化过程来寻找最优解，主要包括以下操作：

1.  **初始化**：随机生成一个包含许多个体（基因型）的初始种群。
2.  **选择（Selection）**：根据个体的适应度，以一定概率选择优秀的个体进入“交配池”。适应度越高，被选中的概率越大。
3.  **交叉（Crossover）**：随机从“交配池”中选取两个个体作为父母，交换它们部分基因片段，产生新的后代个体。这模拟了基因重组。
4.  **变异（Mutation）**：以很小的概率，随机改变某个个体基因串上的某些位（如0变1，1变0）。这引入了新的基因可能性。
5.  **迭代**：用新产生的后代种群替代旧种群，重复进行选择、交叉、变异的过程。经过多代演化后，种群中的优秀个体其适应度会越来越高，从而逼近最优解。

遗传算法的强大之处在于其巨大的随机搜索能力和对非凸优化问题的良好表现。它不依赖于问题的具体数学形式，只需要能定义适应度函数即可。

## 🧠 第4节：神经网络与反向传播（Back Propagation）算法

本节我们学习第二个重点：神经网络，特别是其核心训练算法——反向传播（BP）。理解BP是掌握神经网络本质的关键。

### 4.1 从生物神经元到数学模型

生物神经元接收来自其他神经元的信号，进行加权求和。如果总刺激超过某个阈值，它就“激活”并向下游神经元发射信号。

最简单的数学模型是**感知器（Perceptron）**：
*   输入：特征向量 `X = [x1, x2, ..., xn]`
*   操作：计算加权和 `S = w1*x1 + w2*x2 + ... + wn*xn`
*   输出：通过一个阈值函数，`输出 = 1 if S > T else 0`

为了便于优化，我们进行两点改进：
1.  **引入偏置项（Bias）**：增加一个恒为1的输入 `x0`，其权重 `w0` 等效于阈值 `T`。这样，判断条件变为 `S + w0 > 0`，我们只需学习权重 `w`，无需单独学习阈值。
2.  **软化激活函数**：将不可导的阶跃函数替换为光滑可导的Sigmoid函数：`σ(x) = 1 / (1 + e^{-x})`。其导数有很好的性质：`σ‘(x) = σ(x) * (1 - σ(x))`。

### 4.2 多层神经网络与优化目标

将多个神经元分层连接，就构成了多层神经网络。我们的目标是学习网络中的所有权重 `W`，使得网络的预测输出 `Z` 尽可能接近真实值 `D`。

为此，我们定义一个损失函数（Loss Function），例如均方误差：`P = -1/2 * (Z - D)^2`。我们的任务转化为：**找到一组权重 `W`，使得损失函数 `P` 最大（即负的误差最小）**。这是一个优化问题。

### 4.3 反向传播（BP）算法的精髓

求解上述优化问题的常用方法是梯度下降法：`W_new = W_old - α * ∂P/∂W`。其中 `α` 是学习率。核心难点在于如何高效计算损失函数 `P` 对网络中每一个权重 `W` 的偏导数 `∂P/∂W`。

反向传播算法巧妙地解决了这个问题，它包含两个核心思想：

1.  **链式法则（Chain Rule）**：对于网络中的任何一个权重，`∂P/∂W` 可以分解为从损失 `P` 到该权重路径上各节点偏导数的连乘。例如，对于一个简单两层网络，`∂P/∂W2 = (∂P/∂Z) * (∂Z/∂P2) * (∂P2/∂W2)`，其中每一项都是可计算的。
2.  **动态规划/反向计算**：这是BP算法效率的关键。如果对每个权重都独立地用链式法则从头计算，计算量会随着网络层数指数级增长。BP算法从输出层开始，**反向**逐层计算并缓存每个节点的梯度 `∂P/∂节点`。当计算更前面层的权重梯度时，可以直接复用后面层已计算好的结果，从而将计算复杂度降低到与网络层数呈线性关系。

正是这种“反向传播误差信号并复用计算结果”的机制，使得训练深层神经网络成为可能。理解这一点，就掌握了神经网络学习的核心。

## 🔄 第5节：循环神经网络（RNN）简介

最后，我们简要介绍一种处理序列数据的神经网络——循环神经网络（RNN）。

之前学的网络（如前馈神经网络、CNN）在处理当前输入时，不会“记住”之前的输入。但在很多任务中（如股票价格预测、自然语言处理），当前输出与历史信息密切相关。

RNN的核心思想是让网络具有“记忆”能力。它在每个时间步 `t` 接收当前输入 `X_t`，并结合上一个时间步的隐藏状态 `H_{t-1}`，共同计算当前输出 `Y_t` 和新的隐藏状态 `H_t`。其公式可以简化为：`H_t = σ(W * H_{t-1} + U * X_t)`，`Y_t = V * H_t`。

通过这种结构，`H_t` 理论上包含了从开始到 `t` 时刻的所有历史信息，从而使得当前预测 `Y_t` 能够利用过去的信息。

在实践中，基础的RNN存在梯度消失/爆炸问题，难以学习长期依赖。因此，更常用的是其变体——**长短期记忆网络（LSTM）**。LSTM通过引入“门”机制（输入门、遗忘门、输出门等）来更精细地控制信息的记忆和遗忘，有效地缓解了长期依赖问题。对于应用者而言，可以将其视为一个更复杂的、但输入输出形式与RNN相同的黑盒模块。
