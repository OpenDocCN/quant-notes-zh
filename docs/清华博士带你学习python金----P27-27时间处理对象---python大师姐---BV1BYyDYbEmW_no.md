## 时间对象基础

上一节我们介绍了课程背景，本节中我们来看看Python中处理时间对象的基础知识。在Python标准库中，处理时间对象的类和方法位于`datetime`和`time`模块中。

`datetime`模块中有一个方法，可以将字符串转换为时间对象，即`strptime`方法。其函数名可以这样理解：“P”代表解析（Parse），即将字符串解析为对象；“F”代表格式化（Format），用于格式化打印。因此，将字符串转换为对象的方法是`strptime`。

其基本用法如下：
```python
datetime.datetime.strptime('2001-01-01', '%Y-%m-%d')
```
其中，第一个参数是字符串，第二个参数是格式字符串，例如`%Y`代表四位年份，`%m`代表月份，`%d`代表日期。

![](img/61f786ee70e2b652a7cbe46fca8e7d6e_3.png)

## 自动解析时间字符串

然而，使用`strptime`方法转换比较麻烦，每次都需要输入格式字符串。在实际数据中，时间字符串的格式可能多种多样，例如有人用横杠分隔，有人用斜杠分隔，格式并不统一。

![](img/61f786ee70e2b652a7cbe46fca8e7d6e_5.png)

能否自动检测格式呢？有一个名为`dateutil`的库可以实现。如果你安装了Pandas，这个库通常已经自动安装好了，因为它是Pandas的依赖库之一。

![](img/61f786ee70e2b652a7cbe46fca8e7d6e_7.png)

`dateutil`库中有一个`parser`子模块，其`parse`函数可以自动解析多种格式的时间字符串，无需手动指定格式。

![](img/61f786ee70e2b652a7cbe46fca8e7d6e_9.png)

以下是使用示例：
```python
from dateutil import parser
parser.parse('2001-01-01')
parser.parse('2001/01/01')
parser.parse('Jan 1, 2001')
```
该函数支持各种各样的格式，例如横杠、斜杠，甚至包含月份英文缩写的格式。不过，它主要支持国际通用格式，对于纯中文格式（如“2001年1月1日”）可能无法解析。

![](img/61f786ee70e2b652a7cbe46fca8e7d6e_11.png)

## Pandas批量时间转换

我们之所以要将字符串转换为时间对象，是因为时间对象在后续分析中具有更多、更有用的功能。

![](img/61f786ee70e2b652a7cbe46fca8e7d6e_13.png)

如果我们有大量数据需要转换，Pandas提供了一个非常便捷的函数`to_datetime`。这个函数可以传入一个列表或数组，批量将时间字符串转换为时间对象。

以下是使用示例：
```python
import pandas as pd
pd.to_datetime(['2001-01-01', '2001/01/02'])
```
即使列表中字符串的格式不一致，该函数也能自动识别并转换。转换后的结果是一个`DatetimeIndex`对象。这种索引对象非常适合用作数据框（DataFrame）的索引，其优势我们将在后续课程中详细说明。

![](img/61f786ee70e2b652a7cbe46fca8e7d6e_15.png)

本节课中我们一起学习了Python中时间对象的基础知识、使用`dateutil.parser`自动解析时间字符串，以及利用Pandas的`to_datetime`函数进行批量转换。掌握这些方法是处理金融时间序列数据的重要第一步。
