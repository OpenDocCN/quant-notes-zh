## 财务因子概述 📊

上一节我们介绍了量化选股的概况和白马股选股模型。本节中，我们来看看基本面选股中一个更系统化的工具——财务因子。

财务因子是用于评价企业基本面情况的一系列量化指标。在量化交易中，我们通常使用财务因子来评估公司的健康状况和投资价值。财务因子主要可以分为以下四类：

以下是财务因子的四种常见类别：

1.  **成长类因子**：衡量公司增长能力的指标，是基本面选股中最常用的因子类别。主要包括营收因子和利润因子。
2.  **规模类因子**：反映公司规模大小的指标。主要包括总市值、流通市值、总股本和流通股本。
3.  **价值类因子**：基于价值投资策略，用于衡量股票价格相对于其内在价值是否被低估的指标。主要包括市净率（PB）、市销率（PS）和市盈率（PE）。
4.  **质量类因子**：评估公司盈利质量和资本结构的指标。主要包括净资产收益率（ROE）和总资产净利率。

以上是财务因子的基本介绍。接下来，我们将深入探讨成长类因子下的具体内容。

## 成长类因子详解：营收与利润 📈

在了解了财务因子的整体框架后，本节我们将详细拆解成长类因子，它主要包含营收因子和利润因子两大类。

**营收因子**主要用于评估公司的收入增长情况。以下是营收因子包含的三个核心指标：

*   **营业收入同比增长率**：衡量当期营业收入相对于上年同期营业收入的增长幅度。
*   **营业收入环比增长率**：衡量本期营业收入相对于上一期（如上月或上季度）营业收入的增长幅度。
*   **营业总收入**：公司在一定时期内通过所有经营活动产生的收入总额。

**利润因子**则侧重于评估公司的盈利能力。以下是利润因子包含的五个常见指标：

*   净利润同比增长率
*   净利润环比增长率
*   营业利润率
*   销售净利率
*   销售毛利率

本章后续内容会对这些因子逐一进行详细介绍。本节我们主要聚焦于营收因子。

## 营收因子指标解析 🔍

我们已经知道营收因子包含三个核心指标。现在，我们来逐一解析它们的计算方法和含义。

### 营业收入同比增长率

营业收入同比增长率用于比较公司当期与上年同期的收入增长情况。其计算公式为：

`营业收入同比增长率 = (当期营业收入 - 上期营业收入) / 上期营业收入 * 100%`

**注意**：公式中的“上期营业收入”特指上一个年度同期的营业收入（例如，今年第一季度对比去年第一季度）。

在量化平台（如聚宽）中，通常使用特定的查询字段来获取该数据。其代码中的关键查询字段为：
`query(fundamentals.inc_revenue_year_on_year)`

### 营业收入环比增长率

营业收入环比增长率用于比较公司本期与上一期（如本月与上月，或本季度与上季度）的收入增长情况。其计算公式与同比增长率类似，但比较的基准期不同。

在量化平台中，其关键查询字段通常为：
`query(fundamentals.inc_revenue_annual)`

**核心区别**：同比增长是**跨年度**的同期比较（如今年4月 vs 去年4月），而环比增长是**相邻周期**的比较（如今年4月 vs 今年3月）。

![](img/a198ccc0723fc4855459d3a9901f3e6c_1.png)

### 营业总收入

营业总收入是指公司在一定会计期间内，通过所有主营业务和其他业务活动所产生的收入总和。其概念相对直接，公式为：

`营业总收入 = 主营业务收入 + 其他业务收入`

在量化分析中，我们常关注其绝对值或它与其他指标（如净利润）的比率。一个相关的常用指标是“净利润/营业总收入”，用于衡量收入的盈利质量。其查询字段可能类似于：
`query(fundamentals.net_profit_to_total_revenue)`

以上便是营收因子各指标的理论部分。接下来，我们进入代码实战环节，看看如何用Python获取和运用这些数据。

## 代码实战：Python实现营收因子选股 💻

理论需要实践来巩固。本节我们将通过Python代码，演示如何查询并运用营收因子进行初步选股。

我们将使用一个假设的量化分析框架（例如聚宽`JQData`的风格）来演示。首先，我们需要导入必要的库并设置运行环境。

```python
# 示例：查询营收同比增长率大于300%的股票
# 假设使用类似聚宽（JoinQuant）的API
stock_list = get_fundamentals(
    query(
        fundamentals.inc_revenue_year_on_year  # 营收同比增长率指标
    ).filter(
        fundamentals.inc_revenue_year_on_year > 300  # 筛选增长率>300%
    ).order_by(
        fundamentals.inc_revenue_year_on_year.desc()  # 按增长率降序排列
    ).limit(
        100  # 限制返回数量
    ),
    date='2023-09-01'  # 指定查询日期
)
print(stock_list[['code', 'inc_revenue_year_on_year']])
```

这段代码会筛选出在指定日期营收同比增长率超过300%的股票，并按增长率从高到低排列。

接下来，我们可以查询营收环比增长率。

![](img/a198ccc0723fc4855459d3a9901f3e6c_3.png)

```python
# 示例：查询营收环比增长率
stock_list_qoq = get_fundamentals(
    query(
        fundamentals.inc_revenue_annual  # 营收环比增长率指标
    ).filter(
        fundamentals.inc_revenue_annual > 50  # 筛选条件示例
    ),
    date='2023-09-01'
)
print(stock_list_qoq[['code', 'inc_revenue_annual']])
```

最后，我们来看如何查询与总营收相关的指标。

```python
# 示例：查询净利润与总营收的比率
stock_list_profit_ratio = get_fundamentals(
    query(
        fundamentals.net_profit_to_total_revenue  # 净利润/总营收指标
    ).order_by(
        fundamentals.net_profit_to_total_revenue.desc()
    ),
    date='2023-09-01'
)
print(stock_list_profit_ratio[['code', 'net_profit_to_total_revenue']])
```

执行以上代码后，你可以得到相应的股票列表。请注意，实际数据中可能会出现总营收或比率为负值的情况，这通常需要结合具体公司的财务报告进行深入分析。
