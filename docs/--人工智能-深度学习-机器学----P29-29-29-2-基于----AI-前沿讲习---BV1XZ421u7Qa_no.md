## 核心思路与函数定义

![](img/acc2e102d029784fd45874744e2229f8_11.png)

我们的目标是：给定一个数据序列，计算其第一四分位数（Q1，25%）和第三四分位数（Q3，75%）。然后，将所有小于Q1的值设置为Q1，将所有大于Q3的值设置为Q3。这样，数据就被“修剪”到了Q1和Q3之间的范围。

![](img/acc2e102d029784fd45874744e2229f8_13.png)

![](img/acc2e102d029784fd45874744e2229f8_15.png)

![](img/acc2e102d029784fd45874744e2229f8_17.png)

以下是实现此功能的核心函数：

![](img/acc2e102d029784fd45874744e2229f8_19.png)

```python
import pandas as pd
import numpy as np

![](img/acc2e102d029784fd45874744e2229f8_21.png)

![](img/acc2e102d029784fd45874744e2229f8_23.png)

def winsorize_by_percentile(series, low_percentile=0.25, high_percentile=0.75):
    """
    基于百分位对序列进行去极值处理（缩尾处理）。
    
    参数:
    series (pd.Series): 待处理的数据序列。
    low_percentile (float): 下百分位，默认0.25（Q1）。
    high_percentile (float): 上百分位，默认0.75（Q3）。
    
    返回:
    pd.Series: 处理后的数据序列。
    """
    # 1. 对数据进行排序
    sorted_series = series.sort_values()
    
    # 2. 计算指定百分位对应的值
    # quantile函数可以一次性计算多个分位数
    q_values = sorted_series.quantile([low_percentile, high_percentile])
    
    # 3. 获取最小和最大边界值
    min_val = q_values.iloc[0]  # 对应low_percentile的值
    max_val = q_values.iloc[1]  # 对应high_percentile的值
    
    # 4. 使用np.clip将数据限制在[min_val, max_val]范围内
    # 小于min_val的值变为min_val，大于max_val的值变为max_val
    clipped_series = np.clip(series, min_val, max_val)
    
    return clipped_series
```

## 关键参数与实际应用

在函数`winsorize_by_percentile`中，`low_percentile`和`high_percentile`是两个关键参数。

![](img/acc2e102d029784fd45874744e2229f8_55.png)

![](img/acc2e102d029784fd45874744e2229f8_57.png)

以下是关于参数选择的说明：

![](img/acc2e102d029784fd45874744e2229f8_61.png)

![](img/acc2e102d029784fd45874744e2229f8_63.png)

*   **示例参数**：本例中使用0.25和0.75是为了让处理效果在图表中更明显。这意味着我们将25%的最小值和25%的最大值都视为极值并进行调整。
*   **实际参数**：在真实的量化交易或数据分析中，通常会使用更严格的范围，例如`low_percentile=0.025`和`high_percentile=0.975`。这意味着只将最低的2.5%和最高的2.5%的数据视为极值。这样可以在消除极端异常值的同时，保留更多的原始数据信息。
*   **灵活性**：你可以根据具体数据和模型的需要，灵活调整这两个参数的值。
