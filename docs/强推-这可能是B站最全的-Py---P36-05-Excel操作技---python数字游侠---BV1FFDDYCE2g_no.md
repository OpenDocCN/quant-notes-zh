## 定位功能 🎯

![](img/f86a795ebc772fe2e1c46663d5f6da9e_11.png)

上一节我们介绍了Excel的基础，本节中我们来看看如何高效地定位和处理特定单元格。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_13.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_15.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_17.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_19.png)

定位功能（快捷键 `Ctrl + G`）能帮助我们快速选中符合特定条件的单元格，例如空值或可见单元格。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_21.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_23.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_25.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_27.png)

以下是定位功能的三个典型应用场景：

![](img/f86a795ebc772fe2e1c46663d5f6da9e_29.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_31.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_33.png)

**场景一：批量填充空值为0**
当数据区域中存在大量空单元格需要统一填充为0时，手动操作效率低下。
1.  在一个空白单元格中输入 `0` 并复制（`Ctrl + C`）。
2.  选中目标数据区域。
3.  按下 `Ctrl + G` 打开定位对话框，点击“定位条件”。
4.  选择“空值”并点击“确定”。此时所有空单元格会被选中。
5.  **注意**：鼠标不要点击其他位置，直接按 `Ctrl + V` 粘贴，即可将所有空值填充为0。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_35.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_37.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_39.png)

**场景二：对分组数据快速求和**
当需要为每个分组（如不同类别）计算小计时，无需为每个空单元格单独写公式。
1.  选中包含分组数据和空白总计行的区域。
2.  按下 `Ctrl + G`，选择“定位条件”中的“空值”。
3.  点击“确定”后，所有待求和的空白单元格被选中。
4.  直接点击工具栏的“自动求和”（或按快捷键 `Alt + =`），Excel会自动为每个分组计算总和。

**场景三：安全删除筛选后的可见行**
直接删除筛选后的行可能导致隐藏的数据被误删。使用定位功能可以避免此问题。
1.  对数据进行筛选。
2.  选中筛选后需要删除的可见行。
3.  按下 `Ctrl + G`，选择“定位条件”中的“可见单元格”。
4.  点击“确定”后，仅选中的可见行被高亮。
5.  此时再进行删除操作，就不会影响到隐藏的数据。

## 选择性粘贴 🔄

学会了精准定位，接下来我们看看如何更智能地进行粘贴操作。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_43.png)

选择性粘贴允许我们在粘贴时选择特定的格式或运算方式，而不仅仅是复制原始内容。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_45.png)

以下是选择性粘贴的两个关键用途：

![](img/f86a795ebc772fe2e1c46663d5f6da9e_47.png)

**用途一：对批量数据统一进行运算**
例如，需要为所有员工的工资增加2000元。
1.  在一个单元格中输入 `2000` 并复制（`Ctrl + C`）。
2.  选中所有需要增加工资的数值区域。
3.  右键点击，选择“选择性粘贴”。
4.  在对话框的“运算”区域选择“加”，然后点击“确定”。所有选中单元格的值都会自动加上2000。

**用途二：将公式结果转换为静态值**
当表格中的计算结果由公式产生，需要移除公式只保留数值时（例如，发给领导的数据）。
1.  选中包含公式的单元格区域并复制（`Ctrl + C`）。
2.  右键点击目标位置，在“粘贴选项”中选择“值”（或通过“选择性粘贴”对话框选择“数值”）。
3.  粘贴后的数据将不再包含公式，仅为静态数值，可以安全地进行后续删除或移动操作。
    *   **额外技巧**：选择性粘贴中的“转置”功能可以将行数据转换为列，或反之。

## 快速填充 ⚡

![](img/f86a795ebc772fe2e1c46663d5f6da9e_49.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_51.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_53.png)

处理数据时，快速录入序列是常见需求。Excel的快速填充功能非常智能。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_55.png)

快速填充能根据已有数据的模式，自动生成序列或拆分、合并数据。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_57.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_59.png)

**基础填充：序列与自定义列表**
1.  **数字/日期序列**：在起始单元格输入起始值（如 `1` 或 `2023/1/1`），拖动单元格右下角的填充柄（实心十字），即可生成连续序列。输入两个值（如 `1`, `3`）再拖动，可生成等差数列（`1,3,5,7...`）。
2.  **自定义文本序列**：Excel可以填充如“部门一、部门二...”这样的自定义序列。
    *   设置方法：点击“文件”->“选项”->“高级”，找到“编辑自定义列表”。
    *   在“输入序列”框中按行输入你的序列（如：`张,王,李,赵`），点击“添加”->“确定”。
    *   之后，在单元格输入“张”，向下拖动填充柄，即可自动填充“王、李、赵...”。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_61.png)

**智能填充：Ctrl + E**
这是Excel 2013及以上版本的神器，能识别模式并自动填充。
*   **合并数据**：如果A列是姓名，B列是城市，想在C列得到“姓名-城市”的格式。只需在C列第一个单元格手动输入示例（如“张三-北京”），然后选中C列区域，按下 `Ctrl + E`，下方单元格会自动按此模式填充。
*   **拆分数据**：反之亦然，如果有一个“张三-北京”的单元格，在旁边单元格输入“张三”，然后按 `Ctrl + E`，也能自动拆分出所有姓名。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_63.png)

## 查找与替换 🔍

![](img/f86a795ebc772fe2e1c46663d5f6da9e_65.png)

数据清理中，查找和替换是使用频率最高的功能之一。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_67.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_69.png)

查找（`Ctrl + F`）和替换（`Ctrl + H`）功能可以帮助我们快速定位和修改数据。

**基本查找与替换**
1.  **模糊与精确查找**：默认查找是模糊匹配。例如，查找“李”会找到所有包含“李”的单元格（如“李四”、“张李”）。若需精确查找“李四”，可在查找时勾选“选项”中的“单元格匹配”。
2.  **格式查找**：可以查找具有特定格式（如黄色填充、红色字体）的单元格。在查找对话框中点击“格式”按钮来设定或选取格式。
3.  **安全替换**：进行批量替换前，**最好先选中目标数据区域**，再打开替换对话框。这样可以避免替换到不需要修改的区域。替换前使用“查找全部”预览结果也是一个好习惯。

**使用通配符进行高级查找**
通配符可以在模糊查找中增加规则。
*   `?` （问号）：代表任意单个字符。例如，查找“李??”可以找到所有姓李的三字姓名（如“李建国”）。
*   `*` （星号）：代表任意多个字符。例如，查找“*师”可以找到所有以“师”结尾的文本（如“工程师”、“设计师”）。

## 分列与合并 ✂️

![](img/f86a795ebc772fe2e1c46663d5f6da9e_71.png)

当数据被不规范地记录在一个单元格时，分列与合并功能就派上用场了。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_73.png)

分列功能可以将一个单元格内的文本按规则分割到多个列；合并则是其反向操作。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_75.png)

**分列操作**
例如，将“甲，喜欢，Excel”按逗号分隔成三列。
1.  选中该列数据。
2.  点击“数据”选项卡下的“分列”按钮。
3.  选择“分隔符号”，点击“下一步”。
4.  勾选“逗号”作为分隔符（可在“数据预览”窗口查看效果），点击“下一步”。
5.  设置每列的数据格式（如常规、文本、日期），并选择目标输出位置，点击“完成”。

**合并操作**
将多列数据合并到一列，最简单的方法是使用连字符 `&`。
*   公式示例：`=A1 & "-" & B1 & "-" & C1` 可以将A1、B1、C1单元格的内容用“-”连接起来。
*   **快捷方法**：同样可以使用前面提到的 `Ctrl + E` 智能填充来实现合并。

## 去除重复项 🗑️

![](img/f86a795ebc772fe2e1c46663d5f6da9e_77.png)

数据分析中，经常需要清理重复的数据记录。

“删除重复项”功能可以基于一列或多列的组合来识别并删除重复行。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_79.png)

**操作步骤**
例如，数据表中有“商品ID”和“更新日期”两列，需要删除同一天同一商品的重复记录。
1.  确保数据格式正确（如日期列为日期格式，ID为文本格式，可使用分列功能调整）。
2.  选中数据区域（或点击数据区域内任一单元格）。
3.  点击“数据”选项卡下的“删除重复项”按钮。
4.  在弹出的对话框中，勾选作为判断依据的列（本例中勾选“商品ID”和“更新日期”）。这表示只有当这两列的值都完全相同时，才被视为重复。
5.  点击“确定”，Excel会提示删除了多少重复值，保留了多少唯一值。

## 日期时间规范 📅

![](img/f86a795ebc772fe2e1c46663d5f6da9e_89.png)

来自不同渠道的数据，日期格式往往五花八门，统一格式至关重要。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_91.png)

规范日期时间主要是将各种文本形式的日期转换为Excel可识别的标准日期格式。

**方法一：查找替换**
对于使用“.”作为分隔符的日期（如 `2023.01.01`）。
1.  选中日期数据列。
2.  按 `Ctrl + H` 打开替换对话框。
3.  在“查找内容”中输入 `.`，在“替换为”中输入 `/` 或 `-`。
4.  点击“全部替换”。完成后，将单元格格式设置为“日期”即可。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_93.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_95.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_97.png)

**方法二：分列功能万能法**
对于任何混乱的文本日期，分列功能是最可靠的规范方法。
1.  选中日期数据列。
2.  点击“数据”->“分列”。
3.  前两步直接点击“下一步”跳过（或根据实际情况选择分隔符）。
4.  在第三步的“列数据格式”中，选择“日期”，并指定原数据的日期顺序（如YMD）。
5.  点击“完成”。Excel会自动将文本转换为标准日期值。

![](img/f86a795ebc772fe2e1c46663d5f6da9e_99.png)

![](img/f86a795ebc772fe2e1c46663d5f6da9e_101.png)

**复杂情况处理**
对于缺少前导字符的日期（如 `98.1.2` 代表1998年1月2日），可能需要结合分列和公式（`=“19”&A1`）或连接符先补全年份，再用分列转为日期格式。
