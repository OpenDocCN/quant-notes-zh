    # 设定股票池为沪深300指数成分股

### 查询财务数据

![](img/cef7206a1fd35ea4564110d73f193815_17.png)

量化交易的核心是数据挖掘，我们需要从海量数据中提取有效指标。所有可用的数据指标都在平台的官方文档中。

以下是查询财务数据的关键步骤：

1.  **确定查询指标**：首先，需要明确要查询的财务指标。例如，我们想根据“营业总收入”来筛选股票。
2.  **使用查询函数**：平台提供了 `query` 函数来获取数据。我们需要仔细阅读文档，了解该函数的参数和用法。
3.  **构建查询语句**：查询语句 `query` 用于指定要获取的指标字段。

### 应用筛选与排序规则

仅仅查询到数据还不够，我们需要对结果进行过滤、排序和数量限制，以得到最终想要的股票列表。

以下是构建完整筛选逻辑的步骤列表：

![](img/cef7206a1fd35ea4564110d73f193815_19.png)

*   **过滤条件 (`filter`)**：确保我们只处理股票池内的股票。例如，判断股票代码是否在沪深300成分股列表中。
*   **排序规则 (`order_by`)**：根据选定的指标（如营业总收入）对股票进行排序。我们可以选择升序或降序，这里我们按营收降序排列，以选取营收最高的股票。
*   **数量限制 (`limit`)**：我们不需要所有股票，只选取排名最靠前的一部分。例如，限制只输出前10只股票。

![](img/cef7206a1fd35ea4564110d73f193815_21.png)

![](img/cef7206a1fd35ea4564110d73f193815_23.png)

将以上步骤组合起来，就构成了完整的筛选代码。我们将筛选结果存入 `context` 对象中，以便在后续的交易函数中使用。为了验证筛选结果，可以先将其打印出来。

![](img/cef7206a1fd35ea4564110d73f193815_25.png)

![](img/cef7206a1fd35ea4564110d73f193815_27.png)

```python
def before_trading(context):
    # 构建查询，获取营业总收入指标
    q = query(
        fundamentals.financial_indicator.operating_revenue
    ).filter(
        fundamentals.financial_indicator.code.in_(context.stock_pool)
    ).order_by(
        fundamentals.financial_indicator.operating_revenue.desc()
    ).limit(10)

    # 执行查询，获取DataFrame格式的结果
    context.filtered_stocks = get_fundamentals(q)

    # 打印查看筛选结果
    print(context.filtered_stocks)
```
