# 量化交易教程：P5：05 金融量化分析-金融分析

在本节课中，我们将要学习金融分析的核心方法，了解如何通过基本面和技术面分析来判断股票的投资价值，避免盲目买卖。

上一节我们介绍了金融和股票的基础知识，本节中我们来看看如何进行具体的金融分析。买卖股票不能仅凭感觉，需要借助科学的分析手段来判断其未来走势。

## 基本面分析

基本面分析的核心是评估公司的内在价值，主要依据影响股价的公司自身因素，如经济环境、行业前景和公司运营状况。以下是基本面分析的三个层面：

![](img/03626d2dacde3e8fde1f20bdcace327f_1.png)

**宏观经济分析**：分析国家整体的财政政策、货币政策等宏观经济状况。这有助于判断资金是倾向于流入股市还是被储存起来。但需注意，宏观经济规律并非总是与股市表现一致。

![](img/03626d2dacde3e8fde1f20bdcace327f_3.png)

**行业分析**：评估特定行业（如教育、IT、能源等）的整体发展前景。可以通过观察该行业内几只代表性股票的走势来辅助判断。

![](img/03626d2dacde3e8fde1f20bdcace327f_5.png)

**公司分析**：这是最具体的层面。例如，在考虑购买某公司（如贵州茅台）的股票时，需要深入研究该公司的公开财务数据。上市公司会定期发布经过审计的财务报表（如年度报告、季度报告），这些数据是公开且相对可靠的。投资者可以通过分析财报（如盈利情况、每股收益）以及结合新闻、实地考察等信息，来判断公司运营是否健康、是否具有投资价值。

## 技术面分析

上一节我们介绍了基本面分析，本节中我们来看看技术面分析。技术面分析的核心观点是：所有影响市场的因素都已反映在历史交易数据中。它通过研究股价的历史走势和一系列技术指标来预测未来动向。

以下是两种基础且重要的技术指标：

![](img/03626d2dacde3e8fde1f20bdcace327f_7.png)

![](img/03626d2dacde3e8fde1f20bdcace327f_9.png)

**K线图**：K线是记录单个交易日（或特定周期）内四个关键价格（开盘价、收盘价、最高价、最低价）的图形化工具。
*   **阳线与阴线**：当收盘价高于开盘价时，称为“阳线”，通常用红色或空心实体表示，代表当日股价上涨。当收盘价低于开盘价时，称为“阴线”，通常用绿色或实心实体表示，代表当日股价下跌。
*   **K线构成**：K线由“实体”和“影线”组成。实体的上下边缘分别代表开盘价和收盘价。上影线的顶端代表最高价，下影线的底端代表最低价。
*   **特殊形态**：存在一些特殊K线形态，例如“十字星”（开盘价等于收盘价）或“光头光脚线”（无影线，开盘价/收盘价即是最低价/最高价）。不同的K线形态组合可用于后续分析。

**移动平均线（MA）**：均线是通过计算过去一段时间内股价的平均值，并将每日的点连接起来形成的平滑曲线。其英文为 `Moving Average`。
*   **计算公式（以5日均线MA5为例）**：`MA5 = (P1 + P2 + P3 + P4 + P5) / 5`，其中 P1 至 P5 分别代表过去5个交易日的收盘价（或其他选定价格）。每日计算并连线，即得到5日均线。
*   **常见均线**：例如 `MA5`（5日均线）反映短期趋势，`MA60`（60日均线）反映中长期趋势。通过观察不同周期均线的相对位置和交叉情况（如后文将提到的“双均线策略”），可以辅助判断买卖时机。

![](img/03626d2dacde3e8fde1f20bdcace327f_11.png)

本节课中我们一起学习了金融分析的两大支柱：基本面分析和技术面分析。基本面分析侧重于公司的内在价值与宏观环境，而技术面分析则专注于市场价格行为本身。理解K线和移动平均线是开启技术分析大门的第一步。
