## 第一步：读取股票数据

首先，我们需要获取并读取股票数据。这里我们选择微软（MSFT）的股票历史数据作为分析对象。

```python
import pandas as pd
import yfinance as yf
import matplotlib.pyplot as plt

# 读取微软股票数据
stock_symbol = 'MSFT'
data = yf.download(stock_symbol)
```

执行上述代码后，系统会从网络下载数据。首次运行可能会稍慢，因为它需要加载必要的库并获取数据。请确保已安装 `pandas`, `yfinance`, `matplotlib` 这三个库。

## 第二步：数据预处理与探索

在开始分析前，我们需要对数据进行一些基本的预处理和探索。

以下是数据预处理的关键步骤：

1.  **重置索引**：将日期列设置为索引，便于时间序列分析。
2.  **提取关键列**：创建 `ds`（日期）和 `y`（目标变量，如收盘价）列。
3.  **计算衍生指标**：例如，计算每日价格变化（`daily_change`）。
4.  **确定时间范围**：找出数据中的最早和最晚日期。

```python
# 重置索引，使‘Date’成为一列
df = data.reset_index()

# 创建时间序列分析所需的列
df['ds'] = df['Date']  # 日期列
df['y'] = df['Close']  # 目标变量（收盘价）
df['daily_change'] = df['Close'] - df['Open']  # 每日价格变化

# 确定数据的时间范围
start_date = df['ds'].min()
end_date = df['ds'].max()
print(f"数据时间范围：从 {start_date} 到 {end_date}")
```

通过以上步骤，我们得到了一个结构清晰、便于分析的数据集。

## 第三步：数据可视化

拿到时间序列数据后，第一步通常是将其可视化，以直观了解其趋势和模式。

上一节我们准备好了数据，本节中我们来看看如何绘制股票价格走势图。

```python
def plot_stock_data(df, start_date=None, end_date=None, columns=['Close']):
    """
    绘制股票数据图
    df: 数据框
    start_date: 起始日期
    end_date: 结束日期
    columns: 要绘制的列名列表
    """
    # 如果没有指定日期范围，则使用全部数据
    if start_date is None:
        start_date = df['ds'].min()
    if end_date is None:
        end_date = df['ds'].max()

    # 筛选指定日期范围内的数据
    mask = (df['ds'] >= start_date) & (df['ds'] <= end_date)
    plot_df = df.loc[mask]

    plt.figure(figsize=(14, 7))
    for column in columns:
        plt.plot(plot_df['ds'], plot_df[column], label=column, linewidth=2)
    plt.xlabel('Date')
    plt.ylabel('Price (USD)')
    plt.title(f'Stock Price Trend ({start_date.date()} to {end_date.date()})')
    plt.legend()
    plt.grid(True)
    plt.show()

# 绘制微软股票收盘价走势图
plot_stock_data(df, columns=['Close'])
```

运行代码后，你会看到微软股票从1986年至今的整体上涨趋势。图中可能包含一些明显的波动，例如2000年左右的下跌，这反映了历史上的市场事件。

我们可以编写一个函数来计算指定时间段内的投资回报。

```python
def calculate_return(df, start_date, end_date, shares=100):
    """
    计算在指定时间段内买入并持有股票的收益
    df: 包含价格的数据框
    start_date: 买入日期
    end_date: 卖出日期
    shares: 购买的股数
    """
    # 获取买入和卖出日的收盘价
    buy_price = df.loc[df['ds'] == start_date, 'Close'].values[0]
    sell_price = df.loc[df['ds'] == end_date, 'Close'].values[0]

    # 计算总收益
    total_return = (sell_price - buy_price) * shares
    return_pct = ((sell_price / buy_price) - 1) * 100

    print(f"买入日期: {start_date.date()}, 价格: ${buy_price:.2f}")
    print(f"卖出日期: {end_date.date()}, 价格: ${sell_price:.2f}")
    print(f"持有 {shares} 股的总收益: ${total_return:.2f}")
    print(f"收益率: {return_pct:.2f}%")
    return total_return

# 示例：计算从1999年到2002年持有100股的收益（可能为负）
# 注意：需要确保日期在数据范围内
try:
    calculate_return(df, pd.Timestamp('1999-01-01'), pd.Timestamp('2002-01-01'), 100)
except IndexError:
    print("指定的日期不在数据范围内，请调整日期。")
```
