## 超平面：分类的基础

![](img/0e79afa625457cb76a29c9396072622a_6.png)

上一节我们提到了支持向量机是一种分类方法。本节中，我们来看看其核心概念——超平面。

![](img/0e79afa625457cb76a29c9396072622a_8.png)

在p维空间中，一个p维超平面是一个维度为p-1的平坦子空间。例如：
*   在二维空间中，超平面是一条直线。
*   在三维空间中，超平面是一个平面。

![](img/0e79afa625457cb76a29c9396072622a_10.png)

![](img/0e79afa625457cb76a29c9396072622a_12.png)

超平面由以下线性方程定义：
`β₀ + β₁X₁ + β₂X₂ + ... + βₚXₚ = 0`

![](img/0e79afa625457cb76a29c9396072622a_14.png)

![](img/0e79afa625457cb76a29c9396072622a_15.png)

超平面可以将p维空间分割成两个部分。在分类问题中，我们可以利用这一点：位于超平面一侧的观测点被归为一类，另一侧的则归为另一类。

![](img/0e79afa625457cb76a29c9396072622a_17.png)

## 使用超平面进行分类

理解了超平面的定义后，我们来看看如何用它进行分类。

![](img/0e79afa625457cb76a29c9396072622a_19.png)

我们有一个包含n个观测值的数据集，每个观测值有p个特征。每个观测值都属于两类之一，我们用定性响应变量Y来表示，其取值为-1或+1。当我们获得一个新的测试观测点`X*`时，目标是利用一个分离超平面来预测它的类别。

![](img/0e79afa625457cb76a29c9396072622a_21.png)

以下是使用超平面进行分类的基本思路：
*   找到一个能将两类观测点分开的超平面。
*   根据新观测点位于超平面的哪一侧来决定其类别。

![](img/0e79afa625457cb76a29c9396072622a_23.png)

## 分离超平面的问题与选择

![](img/0e79afa625457cb76a29c9396072622a_25.png)

![](img/0e79afa625457cb76a29c9396072622a_27.png)

虽然使用超平面进行分类的思路很直观，但在实践中会遇到两个主要问题。

![](img/0e79afa625457cb76a29c9396072622a_29.png)

![](img/0e79afa625457cb76a29c9396072622a_31.png)

首先，分离超平面可能根本不存在。当两类数据点严重混杂、无法用一条直线（或平面）完全分开时，就会出现这种情况。

![](img/0e79afa625457cb76a29c9396072622a_33.png)

![](img/0e79afa625457cb76a29c9396072622a_35.png)

其次，即使存在分离超平面，通常也会有无限多个。这就引出了一个关键问题：我们应该选择哪一个超平面？一个自然的选择是**最大保证金超平面**，即距离所有训练观测点都最远的那个分离超平面。

![](img/0e79afa625457cb76a29c9396072622a_37.png)

![](img/0e79afa625457cb76a29c9396072622a_39.png)

## 构建最大保证金分类器

![](img/0e79afa625457cb76a29c9396072622a_41.png)

为了解决选择哪个超平面的问题，我们引入了最大保证金分类器。

![](img/0e79afa625457cb76a29c9396072622a_43.png)

![](img/0e79afa625457cb76a29c9396072622a_45.png)

我们首先计算每个训练观测点到给定分离超平面的垂直距离。所有这些距离中的最小值被称为该超平面的**保证金**。最大保证金分类器的目标就是找到那个能使这个保证金值最大的分离超平面。

![](img/0e79afa625457cb76a29c9396072622a_47.png)

![](img/0e79afa625457cb76a29c9396072622a_49.png)

这个最优超平面仅由距离它最近的那些点决定，这些点被称为**支持向量**。正是这些支持向量“支撑”着超平面。有趣的是，在远离超平面的区域添加新的数据点，通常不会改变最大保证金超平面的位置，因为决定它的是那些最近的支持向量。

## 最大保证金超平面的数学构建

最大保证金超平面是一个特定优化问题的解。

![](img/0e79afa625457cb76a29c9396072622a_51.png)

我们需要找到参数`β₀, β₁, ..., βₚ`，以最大化保证金`M`。同时，需要满足以下约束：对于每一个观测点`i`，有 `y_i (β₀ + β₁x_i1 + ... + β_p x_ip) ≥ M`。这个约束确保了每个观测点都位于超平面的正确一侧，并且距离超平面至少为`M`。此外，通常会对系数施加一个归一化约束（如`∑β_j² = 1`），以保证解的唯一性。

![](img/0e79afa625457cb76a29c9396072622a_53.png)

## 最大保证金分类器的局限性

![](img/0e79afa625457cb76a29c9396072622a_55.png)

![](img/0e79afa625457cb76a29c9396072622a_57.png)

尽管最大保证金分类器在理论上很优雅，但它有两个主要局限性。

![](img/0e79afa625457cb76a29c9396072622a_59.png)

第一，它要求数据是**线性可分**的。如果两类数据无法用一个超平面完全分开（即非可分情况），那么这个分类器就无法直接应用。

![](img/0e79afa625457cb76a29c9396072622a_61.png)

第二，即使数据是线性可分的，最大保证金分类器也可能对训练数据中的个别点（特别是支持向量）**过于敏感**。添加或移动一个靠近边界的点，可能会导致超平面发生显著变化，从而产生较高的模型方差，影响其在新数据上的泛化能力。

正因为这些局限性，我们需要对最大保证金分类器进行扩展。在下一讲中，我们将介绍**支持向量分类器**，它通过允许一些观测点落在边界内甚至错误的一侧，来增强模型的鲁棒性，以处理不可分数据并控制模型的复杂度。

![](img/0e79afa625457cb76a29c9396072622a_63.png)

## 总结

![](img/0e79afa625457cb76a29c9396072622a_65.png)

![](img/0e79afa625457cb76a29c9396072622a_66.png)

本节课中我们一起学习了：
1.  **超平面**是支持向量机用于分类的基本几何工具。
2.  **最大保证金分类器**在线性可分数据中，寻找距离所有观测点最远的分离超平面。
3.  决定这个最优超平面的关键点被称为**支持向量**。
4.  最大保证金分类器可以通过求解一个带约束的优化问题来构建。
5.  该方法的局限性在于要求数据线性可分，并且可能对支持向量过于敏感，这引出了对其扩展版本的需求。
