在本节课中，我们将学习一种在数据预处理中常用的技术——基于百分位法去除极值。这种方法的核心思想是，将数据中超出指定百分位范围（例如，下四分位数Q1和上四分位数Q3）的极端值，统一替换为该范围的边界值，从而规范数据的分布范围。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_3.png)

上一节我们介绍了数据清洗的重要性，本节中我们来看看如何具体实现基于百分位的去极值操作。

## 操作目标与思路 🎯

![](img/9fd06e44426ee7b4afd8b88d599b8e37_5.png)

![](img/9fd06e44426ee7b4afd8b88d599b8e37_7.png)

我们的目标是：对于给定的数据列，计算其第一四分位数（Q1，25%）和第三四分位数（Q3，75%）。将所有小于Q1的值都设置为Q1，将所有大于Q3的值都设置为Q3。这样，数据就被规范到了一个由Q1和Q3定义的固定范围内。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_9.png)

以下是实现此目标的基本步骤：
1.  选择需要处理的数据列。
2.  编写一个函数，该函数能够接收一列数据以及指定的下限和上限百分位。
3.  在函数内部，计算对应百分位的具体数值。
4.  使用`numpy`的`clip`函数，将数据限制在计算出的最小值和最大值之间。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_11.png)

### 第二步：编写去极值函数

我们将编写一个名为 `filter_by_percentile` 的函数。该函数接收三个参数：一个Pandas Series类型的数据序列，以及希望保留的数据范围的下限和上限百分位。

import numpy as np
import pandas as pd

def filter_by_percentile(series, low_percentile=0.25, high_percentile=0.75):

    参数:
        series: Pandas Series, 待处理的数据序列。
        low_percentile: float, 下限百分位（如0.25代表Q1）。

### 第三步：计算百分位对应的具体值

![](img/9fd06e44426ee7b4afd8b88d599b8e37_29.png)

在函数内部，我们需要计算传入的 `low_percentile` 和 `high_percentile` 所对应的具体数值。Pandas的 `quantile` 方法可以方便地完成这个计算。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_31.png)

```python
    # 计算指定百分位对应的值
    Q = series.quantile([low_percentile, high_percentile])
    low_value = Q.iloc[0]  # 获取下限值（如Q1）
    high_value = Q.iloc[1] # 获取上限值（如Q3）
```

**公式解释**：`Q = series.quantile([0.25, 0.75])` 会返回一个包含两个值的Series，其中 `Q.iloc[0]` 是25%分位数（Q1），`Q.iloc[1]` 是75%分位数（Q3）。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_33.png)

### 第四步：应用范围限制

得到边界值后，我们使用NumPy的 `np.clip` 函数将原始数据中所有小于 `low_value` 的值设置为 `low_value`，将所有大于 `high_value` 的值设置为 `high_value`。

```python
    # 使用np.clip将数据限制在[low_value, high_value]范围内
    filtered_series = np.clip(series, low_value, high_value)
    return filtered_series
```

![](img/9fd06e44426ee7b4afd8b88d599b8e37_35.png)

![](img/9fd06e44426ee7b4afd8b88d599b8e37_37.png)

**代码解释**：`np.clip(series, a_min, a_max)` 函数会返回一个新的序列，其中所有小于 `a_min` 的元素都被替换为 `a_min`，所有大于 `a_max` 的元素都被替换为 `a_max`，介于两者之间的元素保持不变。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_39.png)

### 完整函数代码

将以上步骤组合起来，就得到了完整的去极值函数。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_41.png)

```python
def filter_by_percentile(series, low_percentile=0.25, high_percentile=0.75):
    “““
    基于百分位法去除极值。
    “““
    Q = series.quantile([low_percentile, high_percentile])
    low_value = Q.iloc[0]
    high_value = Q.iloc[1]
    filtered_series = np.clip(series, low_value, high_value)
    return filtered_series
```

## 注意事项与实际应用 🔧

![](img/9fd06e44426ee7b4afd8b88d599b8e37_59.png)

![](img/9fd06e44426ee7b4afd8b88d599b8e37_61.png)

在理论示例中，我们使用了25%和75%作为分位点，这使得处理范围较大，以便在图表上更明显地展示效果。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_63.png)

但在实际的量化因子分析或机器学习数据预处理中，为了更精确地剔除真正的异常值，我们通常会使用更严格的范围。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_65.png)

![](img/9fd06e44426ee7b4afd8b88d599b8e37_67.png)

以下是实际应用中更常见的参数设置：
*   **更严格的范围**：例如使用 **2.5%** 和 **97.5%** 作为分位点（`low_percentile=0.025, high_percentile=0.975`）。这样可以只剔除分布两端各2.5%的极端数据，保留大部分信息。
*   **因子特异性**：不同的因子或特征数据可能有不同的分布特性，最佳的百分位阈值可能需要通过回测或实验来确定。

![](img/9fd06e44426ee7b4afd8b88d599b8e37_69.png)
