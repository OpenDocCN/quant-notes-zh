## 🐛 使用PDB进行调试

在编写代码时，我们经常会遇到报错，需要定位问题所在。IPython提供了一个名为`%pdb`的魔术命令，它可以自动在代码报错前进入调试模式，方便我们检查变量状态。

![](img/05078c34f3a16eb3457795bbd8d8b796_5.png)

`%pdb`是一个开关命令：
*   `%pdb on`：开启自动调试模式。
*   `%pdb off`：关闭自动调试模式。

当开启`%pdb`后，如果运行的代码中某一行即将报错，执行会在报错前自动暂停，并进入交互式调试器（PDB）。此时，我们可以使用PDB命令来检查当前环境中的变量。

![](img/05078c34f3a16eb3457795bbd8d8b796_7.png)

以下是一个示例函数，它会导致除零错误：
```python
def test(a, b):
    c = a / b  # 当b为0时，此行会报错
    return c
```

![](img/05078c34f3a16eb3457795bbd8d8b796_9.png)

![](img/05078c34f3a16eb3457795bbd8d8b796_11.png)

使用`%paste`魔术命令运行`test(3, 0)`。开启`%pdb`后，执行会在`c = a / b`这一行之前暂停，并进入PDB调试模式。

![](img/05078c34f3a16eb3457795bbd8d8b796_13.png)

![](img/05078c34f3a16eb3457795bbd8d8b796_15.png)

在PDB调试模式下，常用的命令是`p`（print），用于打印变量的值。例如，输入`p a`和`p b`可以分别查看变量`a`和`b`的当前值，从而快速定位问题。

以下是PDB的一些常用命令：
*   `h`：查看帮助文档。
*   `q`：退出调试器。
*   `n`：执行下一行代码。
*   `break`：设置断点。

调试完成后，使用`%pdb off`即可关闭自动调试功能。之后运行代码若再出错，将直接显示错误信息而不会进入调试模式。

## 🔍 使用命令历史

IPython支持类似Linux命令行的历史记录功能，可以方便地检索和复用之前输入过的命令。

使用**上箭头键**可以逐条回溯历史命令。此外，还可以进行**前缀搜索**：先输入命令的开头几个字符，再按上箭头键，IPython会筛选出以这些字符开头的历史命令供你选择。

## 📥 获取输入与输出结果

![](img/05078c34f3a16eb3457795bbd8d8b796_17.png)

![](img/05078c34f3a16eb3457795bbd8d8b796_19.png)

在交互式环境中，我们有时会直接运行表达式查看结果，但之后又需要用到这个结果。IPython提供了便捷的方式来获取上一行的输出和输入。

*   **获取输出**：使用单个下划线`_`可以获取上一行代码的输出结果。
    ```python
    # 示例
    3 + 5
    # 输出: 8
    result = _ * 2  # result = 8 * 2
    ```
*   **获取更早的输出**：使用两个下划线`__`可以获取上上一行的输出，三个下划线`___`则获取更早的，依此类推。更通用的方式是使用`_N`（N为行号）来获取指定行的输出。
*   **获取输入**：使用`_iN`（N为行号）可以获取指定行的输入字符串。例如，`_i72`会获取第72行输入的原始代码文本。

## 📂 目录标签系统（Bookmark）

当需要在多个项目目录间频繁切换时，反复输入冗长的`cd`路径非常麻烦。IPython的`%bookmark`魔术命令可以创建目录标签，实现快速跳转。

![](img/05078c34f3a16eb3457795bbd8d8b796_21.png)

![](img/05078c34f3a16eb3457795bbd8d8b796_23.png)

以下是`%bookmark`的常用操作：
*   **创建书签**：`%bookmark 标签名 目录路径`
*   **列出所有书签**：`%bookmark -l`
*   **跳转到书签目录**：`cd 标签名`
*   **删除单个书签**：`%bookmark -d 标签名`
*   **删除所有书签**：`%bookmark -r`

![](img/05078c34f3a16eb3457795bbd8d8b796_25.png)

![](img/05078c34f3a16eb3457795bbd8d8b796_27.png)

例如，将当前目录保存为标签`myproj`：`%bookmark myproj`。之后在任何位置只需输入`cd myproj`即可快速切换回该目录。

## 📓 IPython Notebook (Jupyter)

IPython Notebook（现已发展为Jupyter Notebook）是一个基于Web的交互式计算环境。它将代码、文本说明、公式、图表和结果整合在一个文档中，非常适合进行数据分析、教学和演示。

要使用Notebook，需要先安装`jupyter`模块：
```bash
pip install jupyter
```

安装完成后，在系统命令行输入`jupyter notebook`，它会在浏览器中打开一个文件管理界面。在这里可以创建新的Notebook文件（后缀为`.ipynb`）。

在Notebook中，你可以创建“单元格”（Cell）。每个单元格可以是：
*   **代码单元格**：输入并运行Python代码，结果会直接显示在单元格下方。
*   **Markdown单元格**：使用Markdown语法编写富文本，用于添加说明、标题或公式。

![](img/05078c34f3a16eb3457795bbd8d8b796_29.png)

Notebook支持将包含代码、运行结果和图文说明的完整文档导出为多种格式，如Python脚本（`.py`）、HTML网页（`.html`）或PDF文档（`.pdf`）。这使得它成为分享数据分析过程和结果的绝佳工具。
