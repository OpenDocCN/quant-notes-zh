# 金融量化分析：P6：05：金融分析 📊

## 概述
在本节课中，我们将学习金融分析的核心方法。金融分析是判断股票买卖时机、预测价格走势的关键手段，避免盲目投资。我们将介绍基本面分析和技术面分析两大体系，并重点讲解K线与均线这两个基础技术指标。

## 基本面分析
上一节我们介绍了金融和股票的基础知识，本节中我们来看看如何分析股票。金融分析的第一种方法是基本面分析。基本面分析的核心是评估公司的运营状况，即我们之前提到的公司自身因素。它通过分析宏观经济、行业前景和公司具体经营情况，来判断股票的投资价值。

以下是基本面分析的三个层面：

1.  **宏观经济面分析**：分析国家的财政政策、货币政策等，判断整体经济环境是鼓励资金进入股市还是倾向于储蓄。但需注意，宏观经济规律并非总是与股市表现一致。
2.  **行业分析**：评估特定行业（如教育、IT、钢铁等）的整体发展前景。可以通过观察该行业内几只代表性股票的走势来辅助判断。
3.  **公司分析**：这是最具体的层面。例如，若要投资贵州茅台，就需要分析该公司的公开财务数据。上市公司定期（如每季度、每年）发布经审计的财报，数据客观公正。投资者可以通过分析财报（如利润、每股收益）或实地考察，判断公司运营是否良好、是否盈利，从而决定是否买入其股票。

## 技术面分析
除了分析公司基本面，另一种主流方法是技术面分析。技术面分析认为，所有信息都已蕴含在市场交易数据中。其核心是通过研究历史价格走势和一系列技术指标来预测未来价格动向。

以下是两个基础且重要的技术指标：

### K线图
K线图是展示股票每日价格变动的图表。一根K线包含四个关键价格：开盘价、收盘价、最高价和最低价。

*   **阳线与阴线**：
    *   阳线通常显示为红色或空心，表示当日股价上涨，即收盘价高于开盘价。
    *   阴线通常显示为绿色或蓝色实心，表示当日股价下跌，即收盘价低于开盘价。
*   **K线结构**：
    *   **实体**：中间的矩形部分。在阳线中，实体的**下边缘**表示开盘价，**上边缘**表示收盘价；在阴线中则相反，实体的**上边缘**表示开盘价，**下边缘**表示收盘价。
    *   **影线**：实体上下方的细线。**上影线**的顶端表示当日最高价，**下影线**的底端表示当日最低价。

通过一根K线，即可直观了解股票在一天内的价格波动情况。此外，还存在一些特殊形态的K线，如开盘价等于收盘价的“十字星”，或没有影线的K线，这些形态在后期的策略分析中具有特定意义。

### 移动平均线（MA）
均线是另一种常用的技术指标，用于平滑价格数据，显示特定周期内的平均价格趋势。图中常见的白色线（MA5）和黄色线（MA60）就是均线。

![](img/07229b15fe1cbc7ef93c4c9244ce4f4b_7.png)

![](img/07229b15fe1cbc7ef93c4c9244ce4f4b_9.png)

*   **计算方法**：移动平均线是取前N个交易日（例如5日或60日）的收盘价（或其他代表性价格）计算出的平均值。随着时间推移，不断剔除最早一天的数据并加入最新一天的数据重新计算平均值，因此称为“移动”平均。
*   **常见类型**：
    *   **5日均线（MA5）**：代表最近5个交易日的平均价格，反映短期趋势。
    *   **60日均线（MA60）**：代表最近60个交易日的平均价格，反映中长期趋势。
*   **公式表示（以收盘价为例）**：
    `MA(N) = (C1 + C2 + ... + CN) / N`
    其中，`C1` 代表今日收盘价，`C2` 代表昨日收盘价，以此类推，`N` 为周期数。

将每日计算出的移动平均值连接起来，就形成了均线。均线能有效过滤日常波动，帮助投资者识别趋势。后续在讲解量化策略时，我们会深入探讨基于均线的交易策略，例如“双均线策略”。

## 总结
本节课我们一起学习了金融分析的两大支柱：基本面分析和技术面分析。基本面分析侧重于公司价值，通过宏观经济、行业和公司三个层面进行评估；技术面分析则聚焦于市场行为，通过研究历史价格图表和技术指标（如K线和均线）来寻找交易信号。理解这些基础概念是进行更深入量化策略研究和实践的第一步。
