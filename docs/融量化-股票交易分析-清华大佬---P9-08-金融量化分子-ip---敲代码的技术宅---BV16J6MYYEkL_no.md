在本节课中，我们将要学习iPython中一个非常实用且有趣的高级功能——魔术命令。魔术命令以百分号开头，能够极大地提升我们在交互式环境中的工作效率，例如直接运行外部脚本、粘贴执行代码片段以及精确测量代码运行时间等。

## 魔术命令简介

上一节我们介绍了iPython的基本交互功能，本节中我们来看看如何利用魔术命令来执行更高级的操作。魔术命令是iPython特有的功能，它们以 `%` 符号开头，可以直接在交互式命令行中执行，无需退出当前环境。

## 运行外部Python脚本

![](img/bcc6fe2af92c164e57091d91cdb0b97d_3.png)

在标准的Python命令行中，要运行一个外部`.py`文件，通常需要退出命令行，然后使用 `python filename.py` 命令。在iPython中，我们可以使用 `%run` 魔术命令直接在交互器中运行脚本。

![](img/bcc6fe2af92c164e57091d91cdb0b97d_5.png)

**代码示例：**
```python
# 假设我们有一个名为 hello.py 的文件，内容为 print("Hello World")
%run hello.py
```
执行上述命令后，`hello.py` 文件中的代码将被执行，结果会显示在当前的iPython会话中。

## 粘贴并执行代码

有时，我们可能从编辑器或其他地方复制了一段代码，希望直接在iPython中测试。虽然可以直接粘贴，但可能会遇到格式问题（如缩进错误）。iPython提供了 `%paste` 魔术命令来更优雅地处理这个问题。

![](img/bcc6fe2af92c164e57091d91cdb0b97d_7.png)

以下是使用 `%paste` 命令的步骤：
1.  从任何地方复制一段代码到剪贴板。
2.  在iPython中输入 `%paste` 并回车。
3.  iPython会先打印出剪贴板中的代码内容，然后用一个分隔符隔开，最后执行这段代码。

![](img/bcc6fe2af92c164e57091d91cdb0b97d_9.png)

这个命令特别适合测试较长的代码片段中的某一部分，而无需单独创建文件。

![](img/bcc6fe2af92c164e57091d91cdb0b97d_11.png)

![](img/bcc6fe2af92c164e57091d91cdb0b97d_13.png)

## 测量代码执行时间

在性能分析和优化时，我们经常需要测量某段代码或函数的执行时间。虽然可以使用Python的 `time` 模块，但对于运行时间极短的代码，测量结果可能不准确（例如显示为0秒）。iPython的 `%timeit` 魔术命令可以解决这个问题。

`%timeit` 命令会自动将目标代码运行多次，然后计算其平均执行时间，从而得到更精确的结果，尤其适用于微操作的性能评估。

**代码示例：**
```python
import random

# 创建一个包含1万个随机数的列表
my_list = [random.random() for _ in range(10000)]

# 测量排序这个列表所需的时间
%timeit sorted(my_list)
```
执行后，`%timeit` 会输出类似 `146 µs ± 7.42 µs per loop` 的结果，表示平均每次排序耗时约146微秒，并给出了误差范围。它通过多次运行（例如“7次运行，每次循环100万次”）来确保测量的稳定性。

## 总结

本节课中我们一起学习了iPython中三个核心的魔术命令：
*   **`%run`**：用于在交互式环境中直接运行外部的Python脚本文件。
*   **`%paste`**：用于安全地执行剪贴板中的代码，避免直接粘贴可能带来的格式错误。
*   **`%timeit`**：用于精确测量代码片段的执行时间，通过多次运行取平均值，特别适合性能分析和微基准测试。
