在本节课中，我们将学习数据分析流程中的关键一步：数据清洗。我们将以一个数据分析岗位招聘信息的数据集为例，手把手演示如何将原始数据整理成干净、可用于分析的数据。

上一节我们介绍了数据分析的流程和明确了分析需求。本节中，我们来看看如何获取数据并进行清洗，为后续的分析工作打下坚实基础。

## 获取数据

我们从一个招聘网站上采集了数据分析师岗位的招聘信息数据。数据已经以表格形式呈现，包含了职位ID、公司名称、城市、薪资范围等多个字段。

## 数据清洗步骤

数据清洗是确保分析结果准确性的基础。以下是数据清洗的常规步骤，我们将逐一进行操作。

### 1. 选择子集

选择子集是指判断表格中所有列是否都有必要进行分析。在本案例中，我们默认所有列都需要，因此无需进行此步操作。

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_3.png)

### 2. 列名重命名

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_5.png)

检查列名是否明确、无歧义。当前数据的列名清晰，因此也无需重命名。

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_7.png)

### 3. 删除重复值

对于招聘数据，每个职位应该对应唯一的职位ID。如果职位ID重复，则代表该职位信息重复。

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_9.png)

以下是查找和删除重复值的操作流程：

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_11.png)

首先，我们可以使用条件格式功能高亮显示重复的职位ID。
1.  选中“职位ID”列。
2.  点击【开始】选项卡下的【条件格式】。
3.  选择【突出显示单元格规则】->【重复值】。
4.  使用默认格式，点击确定。重复的ID会被标记出来。

为了更清晰地查看，我们可以进行排序：
1.  点击【数据】选项卡下的【排序】。
2.  主要关键字选择“职位ID”，顺序选择“升序”或“降序”。
3.  点击确定。重复的行会排列在一起。

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_13.png)

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_15.png)

确认重复行内容一致后，即可删除：
1.  选中数据区域。
2.  点击【数据】选项卡下的【删除重复值】。
3.  在对话框中，确保只勾选“职位ID”作为判断依据。
4.  点击确定，完成删除。

### 4. 缺失值处理

缺失值（空值）是数据清洗中常见的问题。我们需要先找出缺失值，再决定如何处理。

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_17.png)

我们可以使用 `COUNTIF` 函数统计每一列的空值数量。公式如下：
```excel
=COUNTIF(区域, “”)
```
例如，`=COUNTIF(A2:A5032, “”)` 可以统计A列从第2行到第5032行的空单元格数量。

统计后发现，“城市”列有2个空值，“公司所在商区”列有超过1400个空值。

对于缺失值，一般有三种处理方案：
1.  **直接删除**：当缺失值数量很少，且删除后对整体分析影响不大时采用。
2.  **人为判断填写**：适用于文本型数据，且可以根据其他列信息进行合理推断时采用。
3.  **填充缺失值**：当缺失值比例较大，删除会影响分析时采用。对于数值型数据，常用均值或中位数填充；对于文本型数据，可能需要根据业务逻辑填充或标记为“未知”。

在本案例中：
*   “城市”列的2个空值，我们可以根据“公司名称”或“商区”信息推断并手动填写（例如，填写为“上海”）。
*   “公司所在商区”列缺失值过多，人为填写不现实，删除则会损失大量数据。因此，我们决定在后续分析中**不使用这一列**，将其忽略。

### 5. 一致化处理

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_19.png)

一致化处理旨在将数据格式标准化，便于计算和比较。我们发现“薪水”列是一个文本区间（如“20-30K”），不便于直接计算平均薪资。

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_21.png)

处理步骤如下：
1.  **分列提取极值**：利用“-”作为分隔符，将薪资范围拆分成“最低薪资”和“最高薪资”两列。
    *   选中“薪水”列。
    *   点击【数据】选项卡下的【分列】。
    *   选择“分隔符号”，点击下一步。
    *   勾选“其他”，输入“-”，点击下一步，然后完成。
2.  **清理文本字符**：拆分后的数值带有“K”或“以上”、“以下”等字符，需要清理。
    *   选中拆分出的两列数据。
    *   使用 `Ctrl+H` 打开“查找和替换”对话框。
    *   在“查找内容”中输入“K”，“替换为”留空，点击“全部替换”。同样操作替换掉“以上”和“以下”。
3.  **计算平均薪资**：新增一列“平均薪资”，公式为 `(最低薪资 + 最高薪资) / 2`。
    ```excel
    =AVERAGE(最低薪资单元格, 最高薪资单元格)
    ```
    或者
    ```excel
    = (最低薪资单元格 + 最高薪资单元格) / 2
    ```
    经过以上步骤，我们得到了干净的数字型薪资数据。

### 6. 异常值处理与数据过滤

在“职位名称”列中，并非所有岗位都是纯粹的“数据分析师”，还混杂了“数据工程师”、“产品经理”等职位。我们需要过滤出目标岗位。

我们可以使用函数判断职位名称是否包含关键词（如“数据运营”、“数据分析”、“分析师”），并标记出目标行。

以下是使用 `FIND` 和 `COUNT` 函数组合进行判断的示例公式：
```excel
=IF(COUNT(FIND({“数据运营”,”数据分析”,”分析师”}, L2))>0, “是”, “否”)
```
*   `FIND(查找文本, 源文本)`：在源文本中查找指定文本，返回其起始位置（数字），如果找不到则返回错误。
*   `FIND({“A”,”B”,”C”}, L2)`：表示在L2单元格中依次查找“A”、“B”、“C”，返回一个位置数组。
*   `COUNT(数组)`：统计数组中数字的个数（忽略错误值）。
*   `IF(条件, “是”, “否”)`：如果统计结果大于0，说明至少找到一个关键词，则返回“是”，否则返回“否”。

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_23.png)

操作流程：
1.  在数据表新增一列（如“是否目标岗位”），输入上述公式并向下填充。
2.  使用筛选功能，筛选出该列为“是”的所有行。
3.  将这些行数据复制粘贴到一个新的工作表中，作为我们后续分析的**干净数据集**。

## 总结

本节课中我们一起学习了数据清洗的核心步骤和操作技巧。我们从一个原始的招聘数据表出发，逐步完成了：
1.  **删除重复值**，确保数据的唯一性。
2.  **处理缺失值**，根据实际情况采用删除、填充或忽略的策略。
3.  **进行一致化处理**，特别是将文本型薪资区间转换为可计算的数值。
4.  **过滤异常值**，通过函数精准筛选出目标分析对象。

![](img/1ba64f90ef933a5e8a4485d938bd4e8c_25.png)
