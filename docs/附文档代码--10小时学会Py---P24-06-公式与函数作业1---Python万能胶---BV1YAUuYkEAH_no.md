## 第一题：制作九九乘法表

![](img/bbf7e34de80ff30c2aea77950f46cd61_2.png)

这道题考察的是公式与函数中关于“引用”的操作。九九乘法表是一个经典的练习，它要求我们利用单元格的相对引用和绝对引用来实现动态计算。

![](img/bbf7e34de80ff30c2aea77950f46cd61_4.png)

### 第一步：理解乘法表结构

![](img/bbf7e34de80ff30c2aea77950f46cd61_5.png)

乘法表的结构是：列方向上的数字（1到9）乘以行方向上的数字（1到9）。例如，“一三得三”表示1乘以3等于3，“二三得六”表示2乘以3等于6。

### 第二步：构建基础乘法公式

首先，我们在一个单元格（例如B30）中输入基础乘法公式。我们希望实现的效果是：当公式向下拖动时，行号（乘数）变化；当公式向右拖动时，列号（被乘数）变化。

1.  初始公式：在单元格B30中输入 `=B$30 * $A31`。
    *   `B$30`：这里的`$`符号锁定了行号30。这意味着无论公式如何向下拖动，它始终引用第30行的值（即1）。
    *   `$A31`：这里的`$`符号锁定了列A。这意味着无论公式如何向右拖动，它始终引用A列的值（即1）。
2.  验证拖动效果：
    *   将B30的公式向下拖动，会得到 `1*1, 1*2, 1*3, ... 1*9` 的结果。
    *   将B30的公式向右拖动，会得到 `1*1, 2*1, 3*1, ... 9*1` 的结果。

![](img/bbf7e34de80ff30c2aea77950f46cd61_7.png)

以下是这个公式的代码表示：

```excel
=B$30 * $A31
```

### 第三步：完善混合引用以实现完整表格

上一步的公式只能生成第一行和第一列。为了生成完整的9x9表格，我们需要调整引用方式，让行和列都能在拖动时正确变化。

1.  修改公式：将B30的公式改为 `=B$30 * $A31`。
    *   `B$30`：列B前的`$`被移除，允许列在向右拖动时变化（B->C->D...）。行30前的`$`保留，锁定行号。
    *   `$A31`：列A前的`$`保留，锁定列号。行31前没有`$`，允许行在向下拖动时变化（31->32->33...）。
2.  生成完整表格：将这个修改后的公式从B30单元格开始，同时向下和向右拖动填充至9行9列的区域，即可生成完整的乘法计算结果。

![](img/bbf7e34de80ff30c2aea77950f46cd61_9.png)

以下是完善后的公式代码：

![](img/bbf7e34de80ff30c2aea77950f46cd61_10.png)

```excel
=B$30 * $A31
```

### 第四步：使用IF函数优化显示格式

通常我们需要的九九乘法表是一个左下三角的格式（即只显示 `行号 >= 列号` 的部分，例如有“二三得六”，但不显示“三二得六”）。这需要使用IF函数进行条件判断。

1.  判断逻辑：我们希望当**列号小于或等于行号**时，才显示乘法算式，否则显示为空。
    *   以单元格B30（对应1*1）和A31（对应1）为例，条件是 `B$30 <= $A31`。
2.  构建IF公式：将原有公式嵌套进IF函数中。在B30单元格输入：
    ```excel
    =IF(B$30 <= $A31, B$30 * $A31, "")
    ```
    *   **条件**：`B$30 <= $A31`。`B$30`锁定行，`$A31`锁定列，确保拖动时比较关系正确。
    *   **结果为真**：执行乘法 `B$30 * $A31`。
    *   **结果为假**：返回空字符串 `""`。
3.  生成最终表格：将此公式从B30单元格向下向右拖动填充，即可得到一个格式整洁的九九乘法表。

以下是最终公式代码：

![](img/bbf7e34de80ff30c2aea77950f46cd61_12.png)

![](img/bbf7e34de80ff30c2aea77950f46cd61_14.png)

```excel
=IF(B$30 <= $A31, B$30 * $A31, "")
```

## 总结

![](img/bbf7e34de80ff30c2aea77950f46cd61_19.png)

本节课我们一起学习了如何利用Excel公式制作九九乘法表。我们首先通过混合引用（`B$30`和`$A31`）实现了基础乘法运算的动态填充。随后，我们引入IF函数，通过条件判断 `B$30 <= $A31` 来控制输出，最终生成了一个标准的左下三角形式的九九乘法表。这个练习很好地融合了绝对引用、相对引用以及逻辑函数的应用。
