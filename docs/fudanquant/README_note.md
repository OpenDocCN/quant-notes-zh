# 复旦机器学习与量化投资笔记：1：线性回归模型入门

在本节课中，我们将要学习机器学习中最基础且重要的模型之一——线性回归。我们将了解它的核心思想、数学表达以及如何用代码实现一个简单的预测。

## 概述

线性回归是一种用于建立因变量（目标）与一个或多个自变量（特征）之间线性关系的统计方法。其目标是找到一条直线（或超平面），使得所有数据点到这条直线的距离之和最小。

上一节我们介绍了机器学习的基本概念，本节中我们来看看如何用线性模型进行预测。

## 核心概念与数学表达

线性回归的核心是假设目标值 `y` 和特征值 `x` 之间存在线性关系。对于单个特征的情况，这种关系可以用一条直线来表示。

其数学模型可以用以下公式描述：

**`y = w * x + b`**

在这个公式中：
*   **`y`** 是我们要预测的目标值。
*   **`x`** 是输入的特征值。
*   **`w`** 是权重（或斜率），表示 `x` 对 `y` 的影响程度。
*   **`b`** 是偏置（或截距），表示当 `x` 为 0 时 `y` 的值。

我们的目标是找到最合适的 `w` 和 `b`，使得模型预测的 `y` 值与真实的 `y` 值之间的误差最小。

## 损失函数与优化

为了找到最优的 `w` 和 `b`，我们需要一个标准来衡量模型预测的好坏，这个标准就是损失函数。

在线性回归中，最常用的损失函数是均方误差。以下是其公式：

**`MSE = (1/n) * Σ(y_i - (w*x_i + b))^2`**

其中：
*   `n` 是数据点的数量。
*   `y_i` 是第 `i` 个数据点的真实值。
*   `(w*x_i + b)` 是模型对第 `i` 个数据点的预测值。

我们的目标转化为：寻找一组 `w` 和 `b`，使得 `MSE` 的值最小。这个过程称为“优化”。

## 梯度下降算法

梯度下降是一种常用的优化算法，用于寻找损失函数的最小值。其基本思想是沿着损失函数下降最快的方向（梯度负方向）逐步调整参数。

参数更新的规则可以用以下公式表示：

**`w = w - α * (∂MSE/∂w)`**
**`b = b - α * (∂MSE/∂b)`**

其中 `α` 是学习率，控制每次更新的步长。

理解了原理后，我们来看看如何用代码实现一个简单的线性回归。

## 代码实现示例

以下是使用 Python 和 NumPy 库实现简单线性回归的示例代码。

```python
import numpy as np

# 生成模拟数据
np.random.seed(42)
X = 2 * np.random.rand(100, 1)  # 特征
y = 4 + 3 * X + np.random.randn(100, 1)  # 目标值，带有一些噪声

# 初始化参数
w = np.random.randn(1)
b = np.random.randn(1)
learning_rate = 0.1
n_iterations = 1000

# 梯度下降
for iteration in range(n_iterations):
    # 计算预测值
    y_pred = w * X + b
    # 计算误差
    error = y_pred - y
    # 计算梯度
    w_gradient = (2/len(X)) * np.dot(X.T, error)
    b_gradient = (2/len(X)) * np.sum(error)
    # 更新参数
    w = w - learning_rate * w_gradient
    b = b - learning_rate * b_gradient

print(f“训练后的权重 w: {w[0]:.4f}“)
print(f“训练后的偏置 b: {b[0]:.4f}“)
```

这段代码演示了从零开始实现梯度下降来训练线性回归模型的过程。

## 关键步骤总结

以下是构建一个线性回归模型通常包含的几个关键步骤：

1.  **数据准备**：收集并整理特征 `X` 和目标值 `y`。
2.  **模型初始化**：为参数 `w` 和 `b` 设置初始值（通常为随机小值）。
3.  **前向传播**：根据当前参数计算预测值 `y_pred = w * X + b`。
4.  **计算损失**：使用损失函数（如MSE）计算预测值与真实值之间的误差。
5.  **反向传播**：计算损失函数关于每个参数（`w`， `b`）的梯度。
6.  **参数更新**：使用梯度下降算法，沿着梯度负方向更新参数。
7.  **重复迭代**：重复步骤3至6，直到损失收敛或达到预设的迭代次数。

## 总结

本节课中我们一起学习了线性回归模型的基础知识。我们首先了解了其核心的线性假设 `y = w*x + b`，然后引入了均方误差作为衡量模型好坏的损失函数。为了最小化损失，我们介绍了梯度下降这一优化算法，并通过代码示例演示了完整的实现过程。线性回归是理解更复杂模型的基石，掌握其原理至关重要。
