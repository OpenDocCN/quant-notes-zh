# 机器学习：P9：集成学习与提升方法 🚀

在本节课中，我们将学习机器学习中的两种重要方法：集成学习与提升方法。我们将从回顾决策树开始，然后深入探讨基于决策树的提升方法，特别是梯度提升（Gradient Boosting）及其具体实现XGBoost，同时也会介绍AdaBoost的理论基础。课程内容力求简单直白，让初学者能够理解核心概念。

## 回顾：决策树与回归树 🌳

![](img/0fd4850de1a07d789782706f27d4a6ba_1.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_3.png)

上一节我们介绍了决策树作为分类模型。实际上，决策树也可以用于回归问题。本节中，我们来看看如何用决策树进行回归。

### 一维回归树示例

![](img/0fd4850de1a07d789782706f27d4a6ba_5.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_7.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_9.png)

考虑一个简单的监督学习问题：数据特征`x`是一维的，目标`y`也是连续的。假设数据分布如下图所示，在某个点`c`处，`y`的函数值有明显变化。



![](img/0fd4850de1a07d789782706f27d4a6ba_11.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_13.png)

我们的目标是找到一个学习函数来逼近从`x`到`y`的映射。使用决策树的思想，当只有一个特征`x`时，我们从根节点开始。由于只有一个特征，我们无需计算信息增益，直接根据`x`是否小于某个分割点`d`将数据分为两部分。

![](img/0fd4850de1a07d789782706f27d4a6ba_15.png)

*   左分支（`x < d`）：用常数`g`来预测`y`。
*   右分支（`x >= d`）：用常数`w`来预测`y`。

因此，我们需要确定三个参数：分割点`d`、左分支预测值`g`和右分支预测值`w`。为了评估参数的好坏，我们需要定义一个损失函数。例如，使用**L2损失函数**（均方误差），我们的目标是最小化以下总和：

**左分支损失**：`∑ (g - y_i)^2`，对所有满足 `x_i < d` 的 `i` 求和。
**右分支损失**：`∑ (w - y_j)^2`，对所有满足 `x_j >= d` 的 `j` 求和。

![](img/0fd4850de1a07d789782706f27d4a6ba_17.png)

在L2损失下，最优的`g`和`w`分别是其对应分支中所有`y`值的**平均值**：
*   `g = (1/k) * ∑ y_i` （`k`是左分支样本数）
*   `w = (1/(n-k)) * ∑ y_j` （`n-k`是右分支样本数）

![](img/0fd4850de1a07d789782706f27d4a6ba_19.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_21.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_23.png)

问题就转化为寻找最佳分割点`d`，使得总损失最小。

![](img/0fd4850de1a07d789782706f27d4a6ba_25.png)

### 高维回归树

当特征有`k`维时，思路类似。每个特征`i`都有一个分割点`c_i`。根据所有特征是否大于其各自的分割点，整个特征空间被划分为`2^k`个子区域（立方体）。在每个子区域`I_j`内，我们用一个常数`y_j`来逼近目标值。

我们的目标是最小化总损失：`∑_{j=1}^{2^k} ∑_{i in I_j} (y_j - y_i)^2`。

同样，在L2损失下，每个区域的最优`y_j`是该区域内所有`y_i`的平均值。真正的挑战在于如何高效地找到所有`k`个分割点`c_1, c_2, ..., c_k`，因为可能的区域数量随特征数指数增长。

![](img/0fd4850de1a07d789782706f27d4a6ba_27.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_28.png)

### 回归树的构建算法

为了避免组合爆炸，我们采用递归的决策树算法来构建**回归树（Regression Tree）**，过程如下：

1.  **从根节点开始**：遍历所有特征，对每个特征，寻找一个最佳分割点`c`。
2.  **计算分割后损失**：假设按特征`i`和分割点`c`将数据分为左右两部分。计算两部分的预测值（即`y`的平均值），并计算分割后的总L2损失。
3.  **选择最佳分割**：选择能使分割后总损失最小的那个特征`i`和分割点`c`，作为当前节点的分割规则。
4.  **递归生长**：对生成的左右子节点，重复步骤1-3。
5.  **停止条件**：为了避免过拟合，需要设置停止条件，例如：树达到最大深度、节点样本数少于阈值、或节点损失减少小于某个值等。

![](img/0fd4850de1a07d789782706f27d4a6ba_30.png)

通过上述过程建立的树，就是用于回归的决策树。

![](img/0fd4850de1a07d789782706f27d4a6ba_32.png)

## 提升（Boosting）方法的核心思想 📈

![](img/0fd4850de1a07d789782706f27d4a6ba_36.png)

上一节我们介绍了回归树，但单个树的拟合能力可能有限。本节中我们来看看如何通过“提升”来组合多个弱模型，形成一个强模型。

![](img/0fd4850de1a07d789782706f27d4a6ba_38.png)

提升方法基于一个直观的想法：如果我们的模型（弱学习器）预测结果不够精确，我们可以训练一个新的模型来专门学习当前模型的**残差**（预测值与真实值的差距），然后将多个模型的预测结果叠加，从而得到更精确的预测。

![](img/0fd4850de1a07d789782706f27d4a6ba_40.png)

### 梯度提升（Gradient Boosting）直观理解

![](img/0fd4850de1a07d789782706f27d4a6ba_42.png)

假设我们已有数据`(x_i, y_i)`。我们先用一个弱学习器（例如一棵回归树）进行学习，得到初始预测函数`f_0(x)`。计算残差：`y_i^1 = y_i - f_0(x_i)`。

![](img/0fd4850de1a07d789782706f27d4a6ba_44.png)

接下来，我们训练第二个弱学习器`f_1(x)`，但这次的学习目标不是原始`y_i`，而是上一轮的残差`y_i^1`。如果`f_1`能很好地拟合残差，那么组合模型`F_1(x) = f_0(x) + f_1(x)`应该更接近真实值`y_i`，因为：
`y_i - F_1(x_i) = [y_i - f_0(x_i)] - f_1(x_i) = y_i^1 - f_1(x_i) ≈ 0`。

![](img/0fd4850de1a07d789782706f27d4a6ba_46.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_48.png)

如果残差仍然较大，我们可以继续这个过程，训练`f_2(x)`去拟合`y_i^2 = y_i^1 - f_1(x_i)`，以此类推。最终，我们将得到一个强学习器：`F(x) = f_0(x) + f_1(x) + f_2(x) + ...`。

这种方法的核心是**用新的模型去拟合之前模型预测的误差**。

## 梯度提升决策树（GBDT）与 XGBoost 🌲➡️🚀

![](img/0fd4850de1a07d789782706f27d4a6ba_52.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_54.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_56.png)

上一节我们了解了提升的框架。本节中，我们具体看看如何将决策树作为弱学习器，并实现梯度提升，最终引出强大的XGBoost算法。

![](img/0fd4850de1a07d789782706f27d4a6ba_58.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_60.png)

### 从梯度下降到梯度提升

我们更形式化地定义问题。给定损失函数`L(y, F(x))`（例如L2损失），我们希望找到函数`F(x)`最小化总损失 `∑ L(y_i, F(x_i))`。

![](img/0fd4850de1a07d789782706f27d4a6ba_62.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_64.png)

梯度提升模仿了梯度下降，但是是在函数空间进行的。假设我们当前模型为`F_{m-1}(x)`，我们想增加一个新函数`f_m(x)`（一棵树）来改进模型：`F_m(x) = F_{m-1}(x) + f_m(x)`。

根据梯度下降思想，`f_m(x)`应该指向损失函数减少最快的方向，即负梯度方向。对于每个样本`i`，损失函数`L`对当前预测值`F_{m-1}(x_i)`的负梯度为：
`r_{mi} = - [∂L(y_i, F(x_i)) / ∂F(x_i)]_{F=F_{m-1}}`

**对于L2损失**：`L = (y - F)^2`，负梯度 `r_{mi} = y_i - F_{m-1}(x_i)`，正好是残差。这就是之前直观理解中，新树去拟合残差的原因。

![](img/0fd4850de1a07d789782706f27d4a6ba_66.png)

**对于一般损失函数**：我们可以让新树`f_m(x)`去拟合这些负梯度值`r_{mi}`。拟合完成后，还需要确定一个步长`γ_m`（学习率），最终更新模型：`F_m(x) = F_{m-1}(x) + γ_m * f_m(x)`。

![](img/0fd4850de1a07d789782706f27d4a6ba_68.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_70.png)

这个过程就称为**梯度提升**。当弱学习器是决策树时，就是**梯度提升决策树（GBDT）**。

![](img/0fd4850de1a07d789782706f27d4a6ba_72.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_74.png)

### XGBoost：优化的GBDT

![](img/0fd4850de1a07d789782706f27d4a6ba_76.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_78.png)

XGBoost（eXtreme Gradient Boosting）是GBDT的一种高效实现，它在速度和性能上做了大量优化。其核心改进在于构建树时，使用了更精确的**二阶泰勒展开**来近似损失函数，并加入了正则化项来控制模型复杂度。

![](img/0fd4850de1a07d789782706f27d4a6ba_80.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_82.png)

在构建第`m`棵树时，XGBoost的目标函数如下：
`Obj = ∑ L(y_i, F_{m-1}(x_i) + f_m(x_i)) + Ω(f_m)`
其中`Ω(f_m)`是正则项，惩罚树的复杂度（如叶子节点数、叶子权重）。

![](img/0fd4850de1a07d789782706f27d4a6ba_84.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_86.png)

将这个目标函数在`F_{m-1}`处进行**二阶泰勒展开**：
`Obj ≈ ∑ [L(y_i, F_{m-1}) + g_i * f_m(x_i) + 1/2 * h_i * f_m^2(x_i)] + Ω(f_m)`
其中，`g_i`是一阶导数，`h_i`是二阶导数。

移除常数项`L(y_i, F_{m-1})`后，问题转化为寻找树结构`f_m`以最小化：
`∑ [g_i * f_m(x_i) + 1/2 * h_i * f_m^2(x_i)] + Ω(f_m)`

![](img/0fd4850de1a07d789782706f27d4a6ba_88.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_90.png)

通过推导（具体过程略），可以得出对于一个确定的树结构，最优的叶子节点权重`w_j*`以及此时的目标函数值（损失减少量）有简洁的公式。这使得XGBoost在寻找最佳分割点时，可以贪婪地选择能带来最大损失减少的分割方案，计算非常高效。

![](img/0fd4850de1a07d789782706f27d4a6ba_92.png)

**XGBoost的特点总结**：
*   **二阶导数**：利用损失函数的海森矩阵信息，使优化更准确。
*   **正则化**：显式控制模型复杂度，有助于防止过拟合。
*   **并行处理**：在特征排序和分割点查找上可以并行计算。
*   **灵活性**：支持自定义损失函数（只要一阶、二阶可导）。
*   **缺失值处理**：自动学习缺失值的处理方向。

![](img/0fd4850de1a07d789782706f27d4a6ba_94.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_96.png)

在实践中，XGBoost是解决许多表格数据问题的强大工具。

![](img/0fd4850de1a07d789782706f27d4a6ba_98.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_100.png)

## AdaBoost：从理论到算法的提升方法 ⚖️

![](img/0fd4850de1a07d789782706f27d4a6ba_104.png)

前面我们主要讨论了用于回归问题的梯度提升。本节中，我们转向分类问题，介绍另一种经典的提升算法——AdaBoost，并从理论上解释它如何将弱分类器提升为强分类器。

![](img/0fd4850de1a07d789782706f27d4a6ba_106.png)

### 问题设定与弱分类器定义

![](img/0fd4850de1a07d789782706f27d4a6ba_108.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_110.png)

考虑二分类问题，标签`y ∈ {-1, +1}`。我们有一个弱分类算法，可以产生分类器`h(x)`。所谓“弱分类器”，是指其分类效果略好于随机猜测。形式化定义：对于一个数据分布`D`（样本权重`d_i`），若分类器`h`的错误率`ε = P_{i~D} (h(x_i) ≠ y_i) < 1/2`，且存在一个`λ > 0`使得`ε ≤ 1/2 - λ`，则`h`是一个弱分类器。

![](img/0fd4850de1a07d789782706f27d4a6ba_112.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_114.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_116.png)

AdaBoost的目标是：**将多个这样的弱分类器组合起来，形成一个错误率任意低的强分类器**。

### AdaBoost算法步骤

![](img/0fd4850de1a07d789782706f27d4a6ba_118.png)

以下是AdaBoost的核心迭代步骤：

![](img/0fd4850de1a07d789782706f27d4a6ba_120.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_122.png)

1.  **初始化权重**：对于训练集中的`n`个样本，初始化权重`D_1(i) = 1/n`。
2.  **对于每一轮 `t = 1, 2, ..., T`**：
    *   **训练弱分类器**：使用当前样本权重分布`D_t`训练一个弱分类器`h_t(x)`。
    *   **计算错误率**：`ε_t = ∑_{i: h_t(x_i)≠y_i} D_t(i)`。
    *   **计算分类器权重**：`α_t = (1/2) * ln((1 - ε_t) / ε_t)`。错误率越低，`α_t`越大，该弱分类器在最终组合中话语权越重。
    *   **更新样本权重**：对于每个样本`i`，
        `D_{t+1}(i) = (D_t(i) * exp(-α_t * y_i * h_t(x_i))) / Z_t`
        其中`Z_t`是归一化因子，确保`D_{t+1}`是一个概率分布。
        *   **核心作用**：如果样本`i`被`h_t`正确分类（`y_i * h_t(x_i) = +1`），其权重乘以`e^{-α_t} < 1`，权重减小。
        *   如果被错误分类（`y_i * h_t(x_i) = -1`），其权重乘以`e^{α_t} > 1`，权重增大。
        *   这样，下一轮的分类器`h_{t+1}`会**更加关注之前被分错的样本**。
3.  **组合最终分类器**：经过`T`轮后，得到`T`个弱分类器及其权重。最终的强分类器为：
    `H(x) = sign( ∑_{t=1}^{T} α_t * h_t(x) )`

![](img/0fd4850de1a07d789782706f27d4a6ba_124.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_126.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_128.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_130.png)

### AdaBoost的理论保证（简述）

![](img/0fd4850de1a07d789782706f27d4a6ba_132.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_134.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_136.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_138.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_140.png)

AdaBoost的神奇之处有理论支撑。可以证明，最终分类器`H(x)`在训练集上的错误率上界随着轮数`T`增加而指数下降。

![](img/0fd4850de1a07d789782706f27d4a6ba_142.png)

一个关键的证明思路是考察指数损失函数 `exp(-y * F(x))`，其中`F(x) = ∑ α_t h_t(x)`。可以证明，AdaBoost的每一轮迭代实际上是在最小化这个指数损失函数的某种上界。通过推导，最终训练错误率的上界满足：
`(1/n) ∑ I(H(x_i) ≠ y_i) ≤ exp(-2 ∑_{t=1}^{T} λ_t^2)`
其中`λ_t = 1/2 - ε_t`。由于每个`λ_t > 0`，这个上界随着`T`增大而指数趋近于0。这从理论上解释了AdaBoost的有效性。

1.  **梯度提升（GBDT/XGBoost）**：主要用于回归问题，其核心思想是让新的学习器（决策树）去拟合当前模型预测的**残差**（或更一般地，损失函数的负梯度）。XGBoost是其高效实现，通过二阶泰勒展开和正则化，在精度和效率上表现出色。
2.  **AdaBoost**：主要用于分类问题，其核心思想是**调整样本权重**，让后续的弱分类器重点关注之前分错的样本，最后通过加权投票组合弱分类器。它有坚实的理论保证，证明其能将弱分类器提升为强分类器。

![](img/0fd4850de1a07d789782706f27d4a6ba_168.png)

![](img/0fd4850de1a07d789782706f27d4a6ba_170.png)

这两种方法都体现了“集腋成裘”的思想，通过巧妙的方式组合多个简单模型，从而获得强大的预测能力。它们是机器学习工具箱中非常重要且实用的算法。
