## 核心概念与表示方式 💡

在本课程中，为了精确描述算法原理，我们将频繁使用数学公式和伪代码。例如：

*   线性回归模型的目标是找到一条直线（或超平面）来拟合数据，其公式通常表示为：
    `y = w^T * x + b`
    其中 `y` 是预测值，`x` 是特征向量，`w` 是权重系数，`b` 是偏置项。

*   感知机模型的更新规则可以用以下伪代码表示：
    ```
    初始化权重 w
    For each 样本 (x_i, y_i) in 训练集:
        计算预测值 y_hat = sign(w^T * x_i)
        If y_hat != y_i:
            更新权重 w = w + learning_rate * y_i * x_i
    ```
    掌握这些表示方法将有助于你更深刻地理解算法机制。
