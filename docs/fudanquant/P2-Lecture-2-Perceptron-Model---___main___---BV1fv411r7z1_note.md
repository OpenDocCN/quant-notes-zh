## 回顾：不同损失函数的影响

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_3.png)

上一节我们介绍了假设集合、数据点和损失函数的概念。本节中，我们通过一个简单的一维例子，来直观感受选择不同损失函数如何导致不同的“最优”解。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_5.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_7.png)

考虑在实数轴上有 *n* 个点：`x1, x2, ..., xn`。我们希望找到一个实数 `x*`，使其到所有点的某种“距离”之和最小。我们比较三种常见的损失函数：

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_9.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_11.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_13.png)

以下是三种损失函数下的优化问题定义：

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_15.png)

1.  **L1损失**：寻找 `x*`，使得 **∑ |x* - xi|** 最小。
2.  **L2损失**：寻找 `x*`，使得 **∑ (x* - xi)²** 最小。
3.  **L∞损失**：寻找 `x*`，使得 **max |x* - xi|** 最小。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_17.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_19.png)

它们的解具有截然不同的性质：

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_21.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_23.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_25.png)

*   **L1损失**的解是**中位数**。如果 `n` 为偶数，最优解是中间区间内的任意一点；如果 `n` 为奇数，最优解就是最中间的那个点。
*   **L2损失**的解是**均值**，即 `(x1 + x2 + ... + xn) / n`。
*   **L∞损失**的解是**最大值和最小值的终点**，即 `(min(xi) + max(xi)) / 2`。

这个简单的例子表明，**选择不同的损失函数，会得到性质完全不同的最优模型**。这为我们后续选择模型和损失函数提供了重要启示。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_27.png)

## 引入感知机模型 🧠

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_29.png)

现在，我们进入今天的主题——感知机。感知机是机器学习中一个先驱性的分类模型，其思想深刻而方法简洁。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_31.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_33.png)

### 问题定义

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_35.png)

感知机解决的是一个**监督式二分类问题**。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_37.png)

*   **输入**：一组样本 `(x1, y1), (x2, y2), ..., (xn, yn)`。
    *   `xi` 是一个 `k` 维向量，代表样本的 `k` 个特征。
    *   `yi` 是样本标签，仅取两个值，例如 **+1** 和 **-1**。
*   **目标**：找到一个函数 `f(x)`，能够根据输入 `x` 预测其类别 `y`。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_39.png)

从几何角度看，在二维平面上，`xi` 是点，`yi` 用两种颜色（如红与黑）表示。我们的目标是找到一条直线，将两种颜色的点分开。

### 损失函数与线性模型

对于分类问题，一个自然的损失函数是“0-1损失”：
```
loss(f(xi), yi) = 0, 如果 f(xi) = yi
                  1, 如果 f(xi) ≠ yi
```
整体损失就是分类错误的样本数。我们希望损失为0，即所有样本都被正确分类。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_41.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_43.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_45.png)

然而，如果我们允许 `f(x)` 是任意函数，很容易构造一个在训练样本上损失为0，但在新样本上完全无效的模型（例如，把所有训练样本外的点都预测为同一类）。这被称为**过拟合**。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_47.png)

因此，我们必须限制函数 `f(x)` 的形式。感知机选择了一类简单而强大的函数：**线性函数**。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_49.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_51.png)

### 线性函数与超平面

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_53.png)

在 `k` 维空间中，一个线性函数的形式为：
```
f(x) = wᵀx + b
```
其中 `w` 是一个 `k` 维向量（**权重向量**），`b` 是一个实数（**偏置项**）。`wᵀ` 表示 `w` 的转置（行向量），与列向量 `x` 做内积。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_55.png)

这个线性函数 `f(x) = 0` 定义了一个 `(k-1)` 维的**超平面**。向量 `w` 正是这个超平面的**法向量**。该超平面将整个空间自然分成两部分：
*   `f(x) > 0` 的区域
*   `f(x) < 0` 的区域

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_57.png)

感知机的核心思想就是：**寻找一个超平面，使得所有正类样本（`yi=+1`）落在 `f(x)>0` 的一侧，所有负类样本（`yi=-1`）落在 `f(x)<0` 的一侧**。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_59.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_61.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_63.png)

### 感知机问题形式化

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_65.png)

综合以上，感知机问题可以形式化如下：
给定样本 `(xi, yi)`，其中 `yi ∈ {+1, -1}`，寻找权重向量 `w` 和偏置 `b`，使得对于所有 `i`，满足：
```
yi * (wᵀxi + b) > 0
```
这个条件等价于要求每个样本都被正确分类。

为了简化记号，我们可以通过“升维”将 `b` 并入 `w`。定义新的向量：
```
x' = [x; 1] (在x末尾添加一个元素1)
w' = [w; b]
```
则原条件变为：
```
yi * (w'ᵀ x'i) > 0
```
此后，我们将省略撇号，仍用 `w` 和 `x` 表示升维后的向量。问题简化为：寻找 `w`，使得 **yi (wᵀxi) > 0** 对所有 `i` 成立。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_67.png)

## 感知机学习算法 🔄

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_69.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_71.png)

现在关键问题是：如何找到满足条件的 `w`？感知机采用了一种直观的**迭代更新（学习）** 方法。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_73.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_75.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_77.png)

### 算法直观

假设我们当前有一个权重向量 `w`，它未能正确分类所有样本。存在某个样本 `(xn, yn)` 被错误分类，即 `yn (wᵀxn) ≤ 0`。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_79.png)

*   **情况A**：`yn = +1`，但 `wᵀxn < 0`。这意味着 `w` 与 `xn` 的夹角太大。我们应该让 `w` 更靠近 `xn`。更新规则：`w_new = w_old + xn`。
*   **情况B**：`yn = -1`，但 `wᵀxn > 0`。这意味着 `w` 与 `xn` 的夹角太小。我们应该让 `w` 远离 `xn`。更新规则：`w_new = w_old - xn`。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_81.png)

综合两种情况，更新规则可以统一为：
```
w_new = w_old + yn * xn
```
这个更新发生在**每一个分类错误的样本上**。其几何意义是：将分类错误的样本所对应的向量 `yn*xn` 加到当前权重向量上，以调整决策超平面的方向。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_83.png)

### 算法流程与收敛性证明

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_85.png)

迭代算法在数学中很常见，例如二分法求根、牛顿迭代法。感知机算法也是一种迭代，并且有一个重要的保证：**对于线性可分的数据集，该算法一定在有限步内收敛**。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_87.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_89.png)

以下是算法的正式描述和收敛性证明要点：

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_91.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_93.png)

1.  **前提假设**：数据集是**线性可分的**，即存在某个最优权重向量 `w*`，使得 `yi (w*ᵀxi) > 0` 对所有 `i` 成立。
2.  **算法初始化**：随机初始化权重向量 `w0`（例如全零向量）。
3.  **迭代过程**：
    *   遍历所有样本，找到任意一个被当前 `wk` 错误分类的样本 `(xi, yi)`（即 `yi (wkᵀxi) ≤ 0`）。
    *   按规则更新权重：`w_{k+1} = wk + yi * xi`。
    *   重复此过程，直到没有样本被错误分类。

**收敛性证明思路**：
证明的核心在于说明权重向量 `w` 的模长增长有上界，而其与最优向量 `w*` 的内积增长有下界，两者矛盾将迫使迭代在有限步内停止。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_95.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_97.png)

*   **引理1（模长增长上界）**：由更新公式和错误分类条件可推导出，`||w_{k+1}||² ≤ ||wk||² + R²`，其中 `R = max ||xi||`。因此，`||wk||²` 的增长速度不超过 `O(k)`。
*   **引理2（与最优向量内积增长下界）**：由更新公式和线性可分假设可推导出，`w*ᵀ w_{k+1} ≥ w*ᵀ wk + γ`，其中 `γ = min yi(w*ᵀxi) > 0`。因此，`w*ᵀ wk` 的增长速度至少是 `O(k)`。
*   **矛盾与结论**：结合柯西-施瓦茨不等式 `w*ᵀ wk ≤ ||w*|| * ||wk||`，左边是 `O(k)`，右边是 `O(√k)`。当 `k` 足够大时，此不等式不可能成立。因此，`k` 不可能无限增大，算法必在有限步 `K` 内停止。此时的 `w_K` 就是一个能将所有样本正确分类的权重向量。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_99.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_101.png)

## 总结与展望 📚

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_103.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_105.png)

本节课中我们一起学习了机器学习中的经典模型——感知机。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_107.png)

我们首先回顾了损失函数的选择如何影响模型结果。然后，我们正式定义了感知机模型：它是一个用于解决线性可分二分类问题的线性模型。其目标是找到一个超平面（由权重向量 `w` 和偏置 `b` 定义）完美分离两类数据。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_109.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_111.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_113.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_115.png)

核心的学习算法是一种基于错误驱动的迭代更新方法：`w_new = w_old + yi * xi`，其中 `(xi, yi)` 是当前被错误分类的样本。我们证明了，对于线性可分的数据，该算法保证在有限步内收敛到一个解。

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_117.png)

![](img/1e78ba246faefb6b7cc0b5ed0f9df457_119.png)

感知机是神经网络和支持向量机的基础，其思想至关重要。然而，它也有局限性，最主要的就是只能处理线性可分的数据。在实践中，数据往往不是线性可分的，这就需要我们引入更复杂的模型，例如多层感知机（神经网络）或使用核方法的支持向量机。
