## 概述：收益率指标

设计并运行一个交易策略后，首要问题是评估其盈利能力。这通常通过计算收益率来实现。收益率直接反映了策略在一段时间内的盈亏情况。

![](img/aad5173dbde07277ea42f501db4d7ad1_4.png)

![](img/aad5173dbde07277ea42f501db4d7ad1_5.png)

## 回测收益率

![](img/aad5173dbde07277ea42f501db4d7ad1_6.png)

![](img/aad5173dbde07277ea42f501db4d7ad1_7.png)

![](img/aad5173dbde07277ea42f501db4d7ad1_8.png)

回测收益率衡量的是策略在整个回测周期内的总收益情况。它计算的是最终资产相对于初始资产的增长比率。

![](img/aad5173dbde07277ea42f501db4d7ad1_10.png)

以下是回测收益率的计算公式：
**总收益率 = (最终资产 - 初始资产) / 初始资产**

![](img/aad5173dbde07277ea42f501db4d7ad1_11.png)

这个值可能是正数（盈利），也可能是负数（亏损），直观地说明了“投资有风险，策略不一定稳赚”。不同的策略会得到不同的收益率结果，有的可能翻倍，有的则盈利较少。

![](img/aad5173dbde07277ea42f501db4d7ad1_13.png)

## 年化收益率

理解了总收益后，我们面临另一个问题：如何比较不同时间长度的策略？这时就需要引入年化收益率。

![](img/aad5173dbde07277ea42f501db4d7ad1_15.png)

年化收益率是将总收益率转化为一个标准化的年度收益率指标，便于在不同时间跨度的策略之间进行公平比较。当向他人展示一个运行了多年的策略时，年化收益率比总收益率更具参考价值。

![](img/aad5173dbde07277ea42f501db4d7ad1_17.png)

![](img/aad5173dbde07277ea42f501db4d7ad1_18.png)

它本质上是对总收益率进行公式变换，得到一个年度化的收益水平。在分析回测结果时，年化收益率通常是需要重点观察的核心指标。

以下是年化收益率的计算公式：
**年化收益率 = (1 + 总收益率) ^ (250 / 回测总交易日数) - 1**

![](img/aad5173dbde07277ea42f501db4d7ad1_20.png)

公式解析：
*   **R**：代表总收益率，即我们之前计算过的回测收益率。
*   **M**：代表一年的平均交易日数，通常取250或252天。
*   **N**：代表策略回测所使用的总交易日数。

## Python计算示例

![](img/aad5173dbde07277ea42f501db4d7ad1_22.png)

假设我们有一个回测数据 `DataFrame` 名为 `df`，其中包含各策略的最终资产 `final_value` 和初始资产 `initial_capital`。我们已经计算出总收益率 `total_return`。

![](img/aad5173dbde07277ea42f501db4d7ad1_23.png)

![](img/aad5173dbde07277ea42f501db4d7ad1_24.png)

![](img/aad5173dbde07277ea42f501db4d7ad1_25.png)

以下是计算年化收益率的Python代码示例：

![](img/aad5173dbde07277ea42f501db4d7ad1_26.png)

```python
# 假设 total_return 是包含各策略总收益率的Series
# N 是回测的总交易日数，例如 730 天
N = len(df) # 或者具体的回测天数
M = 250 # 一年按250个交易日计算

![](img/aad5173dbde07277ea42f501db4d7ad1_28.png)

# 计算年化收益率
annualized_return = (1 + total_return) ** (M / N) - 1

# 将结果添加到数据框中或进行展示
df[‘年化收益率’] = annualized_return
print(df[[‘策略名称‘， ’总收益率‘， ’年化收益率‘]])
```

执行代码后，我们可以得到每个策略的年化收益率。通常，年化收益率的值会与总收益率有所不同。例如，一个总收益率为113%（即1.13）的策略，在回测了约730天（约3年）后，其年化收益率可能约为20%多。通过年化指标，我们能更清晰地看到策略每年平均的盈利能力。

本节课中我们一起学习了量化策略评估的两个核心收益指标。
1.  **回测收益率**：反映了策略在整个测试周期内的总盈亏幅度，是评估策略效果的起点。
2.  **年化收益率**：将总收益率标准化为年度指标，消除了回测时间长度的影响，是进行跨策略比较和绩效陈述的关键指标。

![](img/aad5173dbde07277ea42f501db4d7ad1_35.png)

![](img/aad5173dbde07277ea42f501db4d7ad1_36.png)

理解并计算这两个指标，是客观评价一个交易策略是否具备潜在盈利能力的基石。
