# 量化交易入门：1：零门槛开通量化交易权限指南 🚀

在本节课中，我们将要学习如何以零资金门槛开通量化交易权限。我们将了解不同券商的要求差异，并为你提供清晰的选择建议。

## 概述

很多人认为量化交易的门槛很高，这种看法并不准确。实际上，个人开通量化交易权限，不少券商已经实现了零资金门槛。这意味着只要你拥有证券账户，就有机会开通此功能。

## 券商门槛差异

核心的差异点在于，不同券商设定的开通门槛各不相同。有的券商门槛高，有的门槛低，有的则没有门槛。此外，同一家券商的不同营业部，可能要求的资产门槛也不一样。这种现象较为特殊。

例如：
*   东方财富的量化权限以前要求100万资产。
*   中金公司要求资产达到50万。
*   中信证券可能要求500万。
*   国金证券有的营业部2万即可开通，有的则要求30万。
*   当然，也有几家券商支持无门槛开通。

## 开通建议与注意事项

开通量化交易权限，几乎都不会要求额外增加佣金。如果你计划开通量化交易权限，我的建议很简单直接：选择佣金低、流程简单、资产门槛低的券商。

如果你不想逐一咨询各家券商的开通门槛，也可以直接私信我获取相关信息。

## 主流量化平台介绍

量化交易权限在不同券商有不同的名称：
*   在东方财富，它被称为 **Choice量化**。
*   在同花顺，它被称为 **SuperMind量化**。
*   在其他券商，通常是开通 **QMT** 或 **PTrade** 权限。

我个人建议选择 **QMT**，因为它的入门更加简单。QMT又分为大QMT和Mini QMT（极简模式）。最好选择能同时为你开通两种模式的券商。

## 后续学习路径

券商量化交易权限开通后，另一个较大的门槛是，你需要投入一些时间学习Python编程语言。因为当前主流的量化交易策略开发都使用Python。

当然，Python入门也比较简单。通常来说，具备初中以上文化程度、能熟练使用电脑，如果有人指导，花费半个月左右的时间即可掌握基础。

## 总结

本节课中，我们一起学习了如何零门槛开通量化交易权限。我们了解了券商间的门槛差异，获得了选择券商的实用建议，并认识了主流的量化交易平台（如QMT）。最后，我们明确了开通权限后学习Python的必要性和可行性。量化交易的大门正在向更多人敞开。
