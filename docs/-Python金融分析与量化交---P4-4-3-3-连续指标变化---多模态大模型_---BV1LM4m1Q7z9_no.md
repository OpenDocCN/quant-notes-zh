## 简单增长率累加的问题

![](img/31c32a13f289515f8436cc48eed83e89_4.png)

上一节我们介绍了如何计算单期增长率。本节中我们来看看，能否将多个时间点的增长率直接相加，以得到一段时间的总增长率。

例如，假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

![](img/31c32a13f289515f8436cc48eed83e89_6.png)

以下是计算过程：
1.  第一到第二天的增长率：`(50 - 100) / 100 = -0.5`
2.  第二到第三天的增长率：`(75 - 50) / 50 = 0.5`

如果简单累加增长率：`-0.5 + 0.5 = 0`。这个结果表示总增长率为0，即股价没有变化。

但观察原始数据，股价从100元变为75元，实际上是下跌的。这说明**简单的增长率累加无法正确反映时间序列的整体变化**。

![](img/31c32a13f289515f8436cc48eed83e89_8.png)

## 使用对数计算连续增长率

![](img/31c32a13f289515f8436cc48eed83e89_10.png)

对于连续时间的计算，我们需要借助数学中的对数公式。核心思想是：**对数的加法等价于原数的乘法**。

我们计算连续的对数收益率，其公式为：
`log_return = log(今日价格 / 昨日价格)`

![](img/31c32a13f289515f8436cc48eed83e89_12.png)

以之前的例子计算：
1.  `log(50 / 100)`
2.  `log(75 / 50)`

![](img/31c32a13f289515f8436cc48eed83e89_14.png)

![](img/31c32a13f289515f8436cc48eed83e89_16.png)

![](img/31c32a13f289515f8436cc48eed83e89_18.png)

将对数收益率累加：`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/31c32a13f289515f8436cc48eed83e89_20.png)

![](img/31c32a13f289515f8436cc48eed83e89_22.png)

这个结果直接反映了**最后一天价格（75）与第一天价格（100）之间的整体比例关系**，避免了简单累加带来的问题。

![](img/31c32a13f289515f8436cc48eed83e89_24.png)

![](img/31c32a13f289515f8436cc48eed83e89_26.png)

## 在Python中实现连续增长率计算

![](img/31c32a13f289515f8436cc48eed83e89_28.png)

![](img/31c32a13f289515f8436cc48eed83e89_30.png)

以下是使用Pandas计算连续对数收益率的步骤：

![](img/31c32a13f289515f8436cc48eed83e89_32.png)

```python
import pandas as pd
import numpy as np

![](img/31c32a13f289515f8436cc48eed83e89_34.png)

![](img/31c32a13f289515f8436cc48eed83e89_36.png)

# 假设 `data` 是一个包含股价的Pandas Series
# 1. 计算每日的对数收益率
log_returns = np.log(data / data.shift(1))

# 2. 计算累积对数收益率（即从起始日到每日的连续变化）
cumulative_log_returns = log_returns.cumsum()

![](img/31c32a13f289515f8436cc48eed83e89_38.png)

![](img/31c32a13f289515f8436cc48eed83e89_40.png)

# 3. 将累积对数收益率还原为实际的价值倍数（假设起始价值为1单位）
cumulative_value = np.exp(cumulative_log_returns)
```

![](img/31c32a13f289515f8436cc48eed83e89_42.png)

**代码解释**：
*   `data.shift(1)`：将数据整体向上移动一格，从而得到“前一天”的价格。
*   `np.log(...)`：计算自然对数。
*   `.cumsum()`：计算累积和。对于序列 `[a, b, c]`，`cumsum` 的结果是 `[a, a+b, a+b+c]`。
*   `np.exp(...)`：指数函数，将对数结果还原为实际倍数。

![](img/31c32a13f289515f8436cc48eed83e89_44.png)

## 结果解读与应用

![](img/31c32a13f289515f8436cc48eed83e89_46.png)

![](img/31c32a13f289515f8436cc48eed83e89_48.png)

执行上述代码后，`cumulative_value` 序列表示：**假设在时间序列起始点投资了1个单位（如1元），那么在每个后续时间点上，这笔投资相当于多少单位**。

例如，如果2010年1月1日的 `cumulative_value` 为1，而2018年12月31日的 `cumulative_value` 为10，则意味着：
*   2010年投入的1元钱，到2018年底价值相当于10元钱。
*   这只股票在8年间的整体增长倍数为10倍。

![](img/31c32a13f289515f8436cc48eed83e89_50.png)

![](img/31c32a13f289515f8436cc48eed83e89_52.png)

这种方法能更真实、连贯地反映资产在一段连续时间内的价值增长轨迹，是金融分析中评估长期投资回报的常用指标。

## 常用时间序列操作总结

本节课中我们一起学习了金融时间序列分析中的几个核心操作：

![](img/31c32a13f289515f8436cc48eed83e89_58.png)

![](img/31c32a13f289515f8436cc48eed83e89_60.png)

1.  **计算差异值**：使用 `.diff()` 方法可以方便地计算相邻数据的差值。
2.  **计算单期增长率**：公式为 `(当期值 - 上期值) / 上期值`。
3.  **计算连续增长率（对数收益率）**：使用 `np.log(当期值 / 上期值)`，这种方法计算出的序列可以安全地进行累加以考察长期累积效应。
4.  **计算累积收益**：先计算对数收益率，再使用 `.cumsum()` 进行累积，最后用 `np.exp()` 还原为价值倍数。这回答了“期初的1单位投资，在后续每个时点价值多少”的问题。

![](img/31c32a13f289515f8436cc48eed83e89_62.png)

![](img/31c32a13f289515f8436cc48eed83e89_64.png)

![](img/31c32a13f289515f8436cc48eed83e89_65.png)

通过掌握这些操作，你可以对金融时间序列进行基础的变化分析，为更复杂的量化策略打下基础。
