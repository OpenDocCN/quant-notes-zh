## 工具包简介与获取

上一节我们介绍了量化分析的基本流程，本节中我们来看看如何利用现成的工具来简化这些工作。Alphalens是一个专门用于因子分析的Python工具包。

它的核心功能是帮助我们计算因子评价指标（如IC值）并生成各种分析图表。这意味着我们无需从零开始编写复杂的计算和绘图代码。

![](img/060b565a5e9fe07785822fa50037a609_5.png)

以下是关于Alphalens的基本信息：
*   **GitHub链接**：可以在其GitHub页面找到源代码和基础信息。
*   **官方文档**：提供了详细的使用说明和API参考。

![](img/060b565a5e9fe07785822fa50037a609_7.png)

![](img/060b565a5e9fe07785822fa50037a609_9.png)

安装Alphalens非常简单，只需在命令行中执行以下代码即可完成：
```bash
pip install alphalens
```
不过，在本课程提供的在线平台中，该工具包已经预先安装好了，方便我们直接使用。

## 在线研究环境配置

![](img/060b565a5e9fe07785822fa50037a609_19.png)

接下来，我们需要一个可以运行代码的环境。由于分析需要获取特定的金融数据，在个人电脑上操作可能比较麻烦。因此，我们将使用课程平台提供的在线环境。

在平台中，我们需要进入“投资研究”模块，而不是之前创建策略的“回测”模块。这个模块提供了一个类似Jupyter Notebook的交互式编程环境。

以下是进入在线研究环境的步骤：
1.  在平台左侧导航栏找到并点击“投资研究”。
2.  点击“新建”按钮，创建一个新的Python 3 Notebook文件。
3.  将文件重命名为“因子分析”或其他你喜欢的名称。

![](img/060b565a5e9fe07785822fa50037a609_21.png)

这样，我们就得到了一个在平台服务器上运行的代码编写工具。后续的所有代码都将在其中编写和执行。我会将完整的代码提供给大家，你可以直接上传到自己的平台账户，或者跟着视频一步步编写。
