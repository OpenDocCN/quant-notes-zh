## 创建Python项目与文件

上一节我们介绍了课程目标，本节中我们来看看如何创建一个Python项目并编写第一个程序。

![](img/4ddb91dba075640b40fd5d3e9300d5cf_7.png)

![](img/4ddb91dba075640b40fd5d3e9300d5cf_9.png)

首先，在开发环境中创建一个Python软件包。Python软件包与普通文件夹的区别在于，软件包目录下会有一个特殊的标识（通常是一个点或 `__init__.py` 文件），这表示它是一个可被Python识别的模块包。

以下是创建步骤：
1.  在项目根目录上点击右键。
2.  选择“新建” -> “Python软件包”。
3.  输入包名（例如“基础语法”）并回车。

![](img/4ddb91dba075640b40fd5d3e9300d5cf_11.png)

![](img/4ddb91dba075640b40fd5d3e9300d5cf_13.png)

创建完成后，系统会自动生成一个名为 `__init__.py` 的空文件，这是包的初始化文件。

接下来，我们需要创建一个具体的Python文件来编写代码。选择“新建” -> “Python文件”，并命名为“字符编码.py”。

![](img/4ddb91dba075640b40fd5d3e9300d5cf_15.png)

## 理解与使用字符编码

现在我们已经创建了文件，本节中我们来探讨字符编码的概念及其在Python中的使用。

字符编码决定了计算机如何存储和显示文本。如果编码设置不正确，可能会导致程序无法运行或出现乱码。Python 3默认使用UTF-8编码，它能很好地支持包括中文在内的多种语言。

![](img/4ddb91dba075640b40fd5d3e9300d5cf_17.png)

![](img/4ddb91dba075640b40fd5d3e9300d5cf_19.png)

我们的第一个程序通常是打印“Hello, World!”。在Python中，使用 `print()` 函数来实现。

![](img/4ddb91dba075640b40fd5d3e9300d5cf_21.png)

```python
print('Hello, World!')
```

运行这段代码，会在控制台输出“Hello, World!”。

![](img/4ddb91dba075640b40fd5d3e9300d5cf_23.png)

同样，我们也可以打印中文：
```python
print('你好，世界！')
```

然而，在某些环境下，直接打印中文可能会遇到编码问题。为了确保代码的兼容性，我们可以在Python文件的开头显式地声明使用UTF-8编码。这是通过一条特殊的注释来实现的。

```python
# -*- coding: utf-8 -*-
print('你好，世界！')
```

添加这行注释后，Python解释器就会明确知道这个文件使用UTF-8编码来解析其中的字符，从而避免潜在的乱码问题。UTF-8编码集同时兼容英文的ASCII码和中文的GBK等编码。
