# 量化交易入门：第1课：MC软件介绍 🚀

在本节课中，我们将学习量化交易中一个重要的第三方平台——MultiCharts（简称MC）。我们将了解为什么初学者可以从学习MC开始，以及如何下载、安装并初步认识这个软件的核心界面和功能。

## 为什么选择第三方平台？

上一节我们介绍了量化交易的基本概念，本节中我们来看看为什么初学者可以从第三方平台入手。

如果你有编程基础，可以直接使用维纳平台或其他开源平台。但如果你没有编程基础，直接学习编程语言的过程会很漫长。一些第三方平台，如TB、文华财经（文华6/文华9）和MC，提供了更简洁或更丰富的内置语言，能让你更快地写出自己的策略。

简单来说，使用第三方平台可以让你快速实现策略逻辑（只要不太复杂），从而获得持续学习的动力。相比之下，学习开源平台可能需要更长时间才能写出有效策略。

此外，第三方平台还能与开源平台进行策略对比，帮助你分析不同平台在相同条件下的开仓差异。

总而言之，第三方平台的优势在于它封装了数据、逻辑等复杂问题，你只需学习如何使用即可。而开源平台则需要你解决更多底层问题，如数据获取、框架兼容性等。

![](img/2bce2563f30e56bacd2f9cb16b7fd788_1.png)

学习一门第三方平台的语言（如MC的EasyLanguage），其核心的流程控制、循环等逻辑与Python、C++等通用编程语言是相通的，能为后续学习打下良好基础。

![](img/2bce2563f30e56bacd2f9cb16b7fd788_3.png)

## MultiCharts平台简介

我们介绍的是MC平台。首先声明，我不是MC的工作人员，也没有任何合作协议。我最初学习量化时使用的是MC，后来也用过TB。我个人认为MC功能强大，尤其在图表集成和算法方面表现优秀。它的语言**EasyLanguage**（也称PowerLanguage）非常容易上手。

虽然MC在中国的用户可能不是最多，且部分功能需要收费，但它仍然是一个非常好的平台。其语言逻辑清晰，学会后对学习Python或C++等语言有很大帮助。

## 软件下载与安装

首先，你需要下载MC软件。

![](img/2bce2563f30e56bacd2f9cb16b7fd788_5.png)

以下是下载与安装的步骤：
1.  在百度搜索“MultiCharts”。
2.  进入官网后，点击“立即下载”。
3.  根据你的电脑系统（32位或64位）选择对应版本下载。
4.  下载完成后，你会得到一个名为`MCCN`的压缩包，双击其中的`.exe`文件进行安装。
5.  安装过程是“下一步”式的，非常简单。

![](img/2bce2563f30e56bacd2f9cb16b7fd788_7.png)

![](img/2bce2563f30e56bacd2f9cb16b7fd788_9.png)

**注意**：下载页面有一个“V12商品合约的设定档”。如果你的MC软件一段时间未用，需要下载此文件并导入，以更新本地合约列表（例如新增的期货合约RB2401），这样才能向服务器申请该合约的数据。

![](img/2bce2563f30e56bacd2f9cb16b7fd788_11.png)

## 注册与登录

安装完成后，需要进行注册和登录。

以下是注册与登录的流程：
1.  首次打开软件，会提示填写注册名和注册码。
2.  你需要用手机号在官网注册一个账户。
3.  登录官网后，在个人中心找到“注册名”和“注册码”，复制并粘贴到软件中即可完成激活。
4.  **重要**：一个账户只能同时登录一台电脑。如需多台登录，需额外付费或关闭其他电脑上的MC。
5.  软件支持内盘、外盘期货、股票、期权、外汇等多种市场。如需挂实盘交易，需通过其合作期货商开户。

## 核心界面与功能概览

![](img/2bce2563f30e56bacd2f9cb16b7fd788_13.png)

![](img/2bce2563f30e56bacd2f9cb16b7fd788_15.png)

登录后，你会看到MC的主界面，这里集成了多个功能模块。熟悉这些界面是后续学习的基础。

### 1. 交易总管

交易总管用于管理交易账户。以下是其主要功能：
*   查看所有挂载账户的持仓信息。
*   查看资金情况。
*   查看委托单、挂单、成交记录及撤单详情。
*   “洗价”队列：管理等待价格条件满足的委托单。

### 2. 图表窗口

![](img/2bce2563f30e56bacd2f9cb16b7fd788_17.png)

图表窗口是进行分析和策略加载的核心区域。以下是其基本操作：
*   可以加载商品（如螺纹钢）并设置周期（如日线）。
*   可以插入指标、绘图和交易信号。
*   通过“背景拖动”按钮拖动图表，用滚轮左右滑动，用工具栏按钮缩放。
*   “样式”菜单可切换K线图、收盘线图等。
*   可以保存当前工作区，方便下次直接打开。

### 3. 公式编辑器

![](img/2bce2563f30e56bacd2f9cb16b7fd788_19.png)

公式编辑器是编写策略代码的地方。以下是三种主要类型：
*   **函数**：封装常用代码块（例如，计算特定时间点应发送的委托量）。调用函数`AverageFC`可计算移动平均。
*   **指标**：用于在图表上画线（例如，移动平均线）。它内部会调用函数来实现计算。
*   **信号**：用于产生买卖开平仓指令，即我们常说的**交易策略**。

### 4. 报价管理器

报价管理器管理所有可获取数据的合约设置。以下是其作用：
*   设置不同品种（股票、期货）的交易时间、夜盘等参数。
*   导入“商品合约设定档”以更新合约列表。

### 5. 投资组合交易

![](img/2bce2563f30e56bacd2f9cb16b7fd788_21.png)

投资组合交易用于多策略、多合约的回测或实盘交易。以下是其用途：
*   添加多个策略和商品进行统一回测。
*   评估策略在选定品种组合上的整体表现。

![](img/2bce2563f30e56bacd2f9cb16b7fd788_23.png)
