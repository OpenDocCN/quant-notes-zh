## 第三步：散点图与分布分析

![](img/860ad166748e6a027a68a2aab7318a1d_61.png)

除了折线图，散点图能更清晰地展示两个变量间的分布关系。同时，我们还可以观察每个变量自身的分布情况。

![](img/860ad166748e6a027a68a2aab7318a1d_63.png)

以下是具体操作步骤：

1.  计算数据的日收益率（或对数收益率）作为分析基础。
2.  使用 `pandas.plotting.scatter_matrix` 函数绘制散点图矩阵。

```python
# 计算收益率（方法之一：今日值/昨日值 - 1）
returns = data / data.shift(1) - 1

![](img/860ad166748e6a027a68a2aab7318a1d_65.png)

![](img/860ad166748e6a027a68a2aab7318a1d_67.png)

# 绘制散点图矩阵，对角线显示直方图
pd.plotting.scatter_matrix(returns, alpha=0.5, figsize=(10, 10), diagonal='hist', hist_kwds={'bins': 50})
```

在对角线位置，我们看到了每个指标自身的分布直方图。我们也可以将直方图替换为更平滑的核密度估计图。

```python
# 绘制散点图矩阵，对角线显示核密度估计图
pd.plotting.scatter_matrix(returns, alpha=0.5, figsize=(10, 10), diagonal='kde')
```

![](img/860ad166748e6a027a68a2aab7318a1d_69.png)

![](img/860ad166748e6a027a68a2aab7318a1d_71.png)

从散点图中，我们可以更明确地看到 `SPX` 收益率与 `VIX` 收益率呈负相关关系（点云呈向下倾斜趋势）。对角线上的图则展示了每个变量自身的分布形态。

## 第四步：引入回归分析

![](img/860ad166748e6a027a68a2aab7318a1d_79.png)

散点图展示了关系，但我们还可以进一步量化这种关系。这就是回归分析的目的——找到一条最能代表数据点趋势的直线（或曲线）。

![](img/860ad166748e6a027a68a2aab7318a1d_81.png)

![](img/860ad166748e6a027a68a2aab7318a1d_83.png)

以下是核心概念：

![](img/860ad166748e6a027a68a2aab7318a1d_85.png)

*   **回归线**：一条穿过数据点的直线，其公式为 **y = a * x + b**。
    *   **y** 是因变量（如 `VIX` 收益率）。
    *   **x** 是自变量（如 `SPX` 收益率）。
    *   **a** 是斜率（回归系数），表示 `x` 变化一单位时 `y` 的平均变化量。**a为负值即表示负相关**。
    *   **b** 是截距。

![](img/860ad166748e6a027a68a2aab7318a1d_87.png)

![](img/860ad166748e6a027a68a2aab7318a1d_89.png)

![](img/860ad166748e6a027a68a2aab7318a1d_91.png)

在散点图上添加这条回归线，可以让人一眼看出关系的强度和方向。我们将在后续课程中详细讲解如何使用 `statsmodels` 或 `scikit-learn` 库进行计算并绘制回归线。
