## 数据导入与初步观察

首先，我们需要导入必要的Python库并加载数据集。这个数据集包含了从雅虎财经获取的股票代码信息。

```python
import pandas as pd
import numpy as np
# 其他必要的库...
```

数据被导入后，我们首先查看其前几行，以了解数据结构。

```python
data_one = pd.read_csv('your_data_file.csv')
print(data_one.head())
```

观察数据后，我们发现原始数据包含一些无关的网站信息和无效行。列名也不规范，例如包含`ticker`、`name`、`exchange`、`category`和`country`等信息。我们的目标是专注于分析美国股票。

## 数据清洗与筛选

上一节我们查看了原始数据，本节中我们来看看如何将其清洗为可用的格式。首先，我们需要筛选出美国的股票数据。

以下是数据清洗的关键步骤：

1.  **跳过无效行**：使用`skiprows`参数在读取数据时直接跳过前两行无效信息。
    ```python
    data_one = pd.read_csv('your_data_file.csv', skiprows=2)
    ```
2.  **筛选美国股票**：在数据清洗后，我们创建一个新的DataFrame，仅保留`country`列为`USA`的行。
    ```python
    pro_two = data_one[data_one['country'] == 'USA']
    ```
3.  **检查结果**：确认筛选后的数据只包含美国股票，并查看其规模。
    ```python
    print(pro_two['country'].unique()) # 应只输出['USA']
    print(pro_two.shape) # 查看数据行数与列数
    ```

经过筛选，数据从近10万行减少到约2.2万行，这大大降低了后续处理的计算量。

## 提取股票代码列表

数据清洗完成后，我们需要提取出所有美国股票的代码列表，以便后续逐个获取其历史数据。

以下是提取股票代码列表的方法：

1.  **提取`ticker`列**：从清洗后的`pro_two` DataFrame中提取出股票代码列。
    ```python
    tickers = pro_two['ticker'].tolist()
    ```
2.  **确认列表长度**：检查提取出的股票代码数量，应与筛选后的数据行数一致。
    ```python
    print(len(tickers))
    ```

至此，我们得到了一个包含所有待查询美国股票代码的列表`tickers`。
