## 学习格局与目标

![](img/8d9183396e657077fafbe02735cf621e_6.png)

视频开始前，我们先思考一个问题：新手学习Python量化交易，仅仅是为了做量化交易吗？

![](img/8d9183396e657077fafbe02735cf621e_8.png)

对于没有接触过Python的新手，建议将学习Python量化交易的格局放大。量化交易只是一个具体的学习情境。在交易、职场和学习的道路上，能够善用Python才是学习的终极目标。

![](img/8d9183396e657077fafbe02735cf621e_10.png)

## 从Excel到Python

![](img/8d9183396e657077fafbe02735cf621e_12.png)

大部分同学都熟悉Excel的功能或快捷键。

![](img/8d9183396e657077fafbe02735cf621e_14.png)

Python能进行量化交易，Excel同样可以。我在GitHub上放置了一段代码，展示了现在用Excel实现策略向量回测，再迁移到Python实现的过程。有兴趣的同学可以下载该Python代码。

这段代码在聚宽量化交易平台上运行。同学无需部署Python环境，也无需另外接入数据源。只要能打开网页，就能够学习Python。有兴趣可以观看相关讲解视频。

新手通常只要有人提点一下，跨过那道门槛，打开思路，Python量化交易的学习道路就会一路平坦。Python量化交易并不神秘。

![](img/8d9183396e657077fafbe02735cf621e_16.png)

![](img/8d9183396e657077fafbe02735cf621e_18.png)

## Python作为Excel Plus

![](img/8d9183396e657077fafbe02735cf621e_20.png)

这类同学可以把Python当做**Excel Plus**来理解。这个Excel Plus是一种“胶水语言”。

![](img/8d9183396e657077fafbe02735cf621e_22.png)

以下是Python作为Excel Plus能实现的部分功能：

![](img/8d9183396e657077fafbe02735cf621e_24.png)

*   **数据处理**：能进行排序和筛选。
*   **逻辑实现**：能编写逻辑规则，细化量化交易策略。
*   **网络与通信**：能进行网络爬虫，处理大数据，还能发送企业微信信息。
*   **应用开发**：能做网页，开发用户界面（UI）。
*   **高级算法**：能实现机器学习与AI算法。
*   **开源生态**：有大量大神开发的Python开源代码可供使用。

![](img/8d9183396e657077fafbe02735cf621e_26.png)

对于新手同学来说，能够利用这些开源资源是一个巨大优势。

![](img/8d9183396e657077fafbe02735cf621e_28.png)

## 学习路径与优势

![](img/8d9183396e657077fafbe02735cf621e_30.png)

Python主打语法简单、工具包丰富、学习曲线平缓。利用已有的Excel基础来学习Python是最快速、最容易理解上手的方式，并且投资回报比最大。

![](img/8d9183396e657077fafbe02735cf621e_32.png)

用Excel的思维方式思考，用Excel的方法解决问题，再转换到Python实现。就跟操作Excel一样，只要有想法和思路，Python编程实现就容易多了。

![](img/8d9183396e657077fafbe02735cf621e_34.png)

Python量化交易没有那么高大上。先有解题思路，写出框架，再寻求优化。

**Excel有思路，Python就能有思路。Excel写得出来，Python就能写得出来。Excel写不出来的，Excel Plus（Python）能够帮你实现。**

![](img/8d9183396e657077fafbe02735cf621e_36.png)

## 量化交易的适用范围

![](img/8d9183396e657077fafbe02735cf621e_38.png)

快手量化交易的范围很广，包括但不限于股票、期货、期权、可转债等。只要你的交易规则相似，策略就能够复制到其他交易品种。

![](img/8d9183396e657077fafbe02735cf621e_40.png)

只要更换交易平台、行情接口和下单接口，就能上策略。前提是你是否愿意动手开始编写策略。

![](img/8d9183396e657077fafbe02735cf621e_42.png)

有些策略在某些交易品种上可能不合适，但换个交易品种，说不定策略就可行了。不必一开始接触新的交易品种，就投入过多。多看看，多接触学习。在理财和交易的赛道上，很多底层的交易逻辑是相通的。

![](img/8d9183396e657077fafbe02735cf621e_44.png)
