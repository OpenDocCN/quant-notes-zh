### 理解回测收益率
首先讲解最重要、最基本的指标——回测收益率。

![](img/1c8691cb049e246b49374485dd3d914f_15.png)

回测收益率的含义是：当你应用一个策略（例如，基于2015年至2020年的股票数据，设定买卖规则）进行历史测试后，最终获得的收益。关键点在于，它表示的不是绝对金额，而是相对于初始本金所获得的收益比例。

例如，结果显示收益率是4%点多，这意味着经过这些年的操作，最终赚取的钱相对于初始本金增加了约4%。

![](img/1c8691cb049e246b49374485dd3d914f_19.png)

![](img/1c8691cb049e246b49374485dd3d914f_23.png)

![](img/1c8691cb049e246b49374485dd3d914f_21.png)

### 计算公式
回测收益率的计算公式非常简单。

**公式**：
`回测收益率 = (期末资产价值 - 期初资产价值) / 期初资产价值`

用符号表示，若`P_t`代表期末的卖出总价值，`P_0`代表期初的投入本金，则公式为：
`收益率 = (P_t - P_0) / P_0`

其核心思想是：分子表示你赚了多少钱（期末价值减期初价值），分母是期初的本金，两者相除得到收益率。

![](img/1c8691cb049e246b49374485dd3d914f_25.png)

![](img/1c8691cb049e246b49374485dd3d914f_23.png)

回测收益率是评估策略效果最常用的指标，它直观地反映了策略的盈利能力。

![](img/1c8691cb049e246b49374485dd3d914f_27.png)

### 代码计算与可视化
接下来，我们通过代码计算回测收益率，并可视化其随时间的变化。

我们可以绘制一张图，展示从起始日（如2015年）到结束日，回测收益率随时间的变化曲线。这需要计算每个时间点相对于起始点的资产比值。

计算思路是：用每个时间点的资产价值`P_t`除以起始点的资产价值`P_0`。我们可以对公式稍作变形，先计算比值，最后再统一减去1。

![](img/1c8691cb049e246b49374485dd3d914f_25.png)

让我们开始编写代码。

![](img/1c8691cb049e246b49374485dd3d914f_27.png)

第一步是计算每个时间点的资产与初始资产的比值。我们构建一个DataFrame来存储这个比值。

```python
# 计算每个时间点的收盘价与初始收盘价的比值
price_ratio = data['close'] / data['close'].iloc[0]
```

这里，`data['close'].iloc[0]`获取了第一期（期初）的收盘价作为`P_0`。

计算完成后，我们可以绘制比值的变化图。

```python
import matplotlib.pyplot as plt
plt.figure(figsize=(12, 6))
price_ratio.plot()
plt.title('资产相对价值变化 (相对于期初)')
plt.xlabel('日期')
plt.ylabel('相对价值')
plt.grid(True)
plt.show()
```

![](img/1c8691cb049e246b49374485dd3d914f_29.png)

这张图展示了初始价值设为1的情况下，资产价值的相对变化。你可以理解为，期初投入1元钱，通过策略运作，这笔钱随时间变成了多少。

![](img/1c8691cb049e246b49374485dd3d914f_31.png)

但这还不是最终的回测收益率。我们需要计算整个回测周期的总收益率。

根据公式，最终的回测收益率等于**最后一期的比值减去1**。

```python
# 计算最终的回测收益率
final_return = price_ratio.iloc[-1] - 1
```

为了展示更清晰，我们可以将每日的收益率（即每日比值减1）也表格化展示。

```python
# 计算每日的回测收益率（每日相对于期初的收益率）
daily_return_vs_start = price_ratio - 1
```
