## 数组形状变换

上一节我们介绍了数组的索引与切片，本节中我们来看看如何改变数组的形状。数组的`shape`属性描述了其维度结构，例如一个五行六列的二维数组。`reshape`方法可以改变数组的形状，但新形状必须能容纳原数组的所有元素。

**代码示例：**
```python
import numpy as np
# 创建一个5行6列的二维数组
arr = np.arange(30).reshape(5, 6)
print(“原始数组形状：”, arr.shape) # 输出：(5, 6)

# 将二维数组变形为一维数组
arr_1d = arr.reshape(30)
print(“变形后数组：”, arr_1d.shape) # 输出：(30,)

# 将一维数组变形为多维数组
arr_new = arr_1d.reshape(2, 15) # 变形为2行15列
print(“新数组形状：”, arr_new.shape) # 输出：(2, 15)
```
变形操作要求新形状的元素总数必须与原数组一致。此操作在数据预处理，尤其是机器学习中调整输入数据维度时非常有用。

## 数组拼接操作

接下来，我们学习如何将多个数组组合在一起。拼接操作指将多个NumPy数组沿指定轴向（横向或纵向）连接起来。

以下是拼接操作的关键点：
*   使用`np.concatenate()`函数进行拼接。
*   参数`axis`决定拼接方向：`axis=0`表示沿行方向（纵向）拼接；`axis=1`表示沿列方向（横向）拼接。
*   进行拼接的数组必须具有相同的维度。

**代码示例：**
```python
# 沿列方向（纵向）拼接
arr_vertical = np.concatenate((arr, arr), axis=0)
print(“纵向拼接后形状：”, arr_vertical.shape) # 输出：(10, 6)

# 沿行方向（横向）拼接
arr_horizontal = np.concatenate((arr, arr), axis=1)
print(“横向拼接后形状：”, arr_horizontal.shape) # 输出：(5, 12)
```
拼接操作的一个典型应用是图像处理，例如将多张图片拼接成宫格图。需要注意的是，如果两个数组维度相同但行或列数不一致，拼接将会失败。

## 聚合统计函数

现在，我们转向数组的聚合与统计操作。这些函数能快速对数组整体或沿某个轴向进行计算。

常用的聚合操作包括求和、求极值和平均值：
*   `sum()`: 计算所有元素之和。
*   `max()` / `min()`: 查找所有元素的最大值/最小值。
*   `mean()`: 计算所有元素的平均值。
*   通过指定`axis`参数（0为列，1为行），可以计算轴向的聚合值。

**代码示例：**
```python
# 计算所有元素的和
total_sum = arr.sum()
print(“数组总和：”, total_sum)

# 计算每一列的最大值
col_max = arr.max(axis=0)
print(“每列最大值：”, col_max)

# 计算每一行的平均值
row_mean = arr.mean(axis=1)
print(“每行平均值：”, row_mean)
```

## 数学与统计函数

除了基本聚合，NumPy还提供了丰富的数学和统计函数。

以下是一些常用函数：
*   `np.sin()`, `np.cos()`, `np.tan()`: 计算数组元素的正弦、余弦、正切值。
*   `np.around(a, decimals=0)`: 将数组`a`中的元素四舍五入到指定小数位（`decimals`）。
*   `np.ptp()`: 返回数组沿指定轴向最大值与最小值的差（极差）。
*   **方差**与**标准差**：衡量数据离散程度的关键指标。

方差（`var`）和标准差（`std`）的计算公式如下：
*   **方差公式**：$$\sigma^2 = \frac{1}{N}\sum_{i=1}^{N}(x_i - \mu)^2$$
*   **标准差公式**：$$\sigma = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(x_i - \mu)^2}$$
其中，$x_i$ 是数据点，$\mu$ 是数据的均值，$N$ 是数据点总数。标准差是方差的平方根，两者都用于度量数据围绕平均值的波动情况。

**代码示例：**
```python
data = np.array([85, 90, 78, 92, 88])
# 计算方差
variance = np.var(data)
print(“方差：”, variance)
# 计算标准差
std_dev = np.std(data)
print(“标准差：”, std_dev)
```

## 矩阵运算

最后，我们探讨NumPy中的矩阵运算。矩阵本质上是二维数组，在线性代数和机器学习中至关重要。

![](img/7379b04992d19ed2bea0c0e9194c023f_3.png)

需要掌握的核心操作包括：
*   `arr.T` 或 `np.transpose(arr)`: 返回数组的转置（行变列，列变行）。
*   `np.dot(A, B)` 或 `A @ B`: 计算两个矩阵的**矩阵乘法**。

矩阵乘法并非简单的元素对应相乘。对于矩阵 $A$（$m \times n$）和矩阵 $B$（$n \times p$），其乘积 $C$（$m \times p$）中每个元素 $C_{ij}$ 的计算方式为：
$$C_{ij} = \sum_{k=1}^{n} A_{ik} \times B_{kj}$$
即 $C$ 的第 $i$ 行第 $j$ 列元素，等于 $A$ 的第 $i$ 行与 $B$ 的第 $j$ 列对应元素乘积之和。

![](img/7379b04992d19ed2bea0c0e9194c023f_5.png)

**代码示例：**
```python
# 创建两个矩阵
A = np.array([[2, 1], [4, 3]])
B = np.array([[1, 2], [1, 0]])
# 矩阵乘法
C = np.dot(A, B) # 等价于 A @ B
print(“矩阵乘法结果：\n”, C)
# 输出：
# [[3 4]
# [7 8]]
```
计算过程解析：结果矩阵`C`中，左上角元素`3` = 2×1 + 1×1；右上角元素`4` = 2×2 + 1×0；左下角元素`7` = 4×1 + 3×1；右下角元素`8` = 4×2 + 3×0。
