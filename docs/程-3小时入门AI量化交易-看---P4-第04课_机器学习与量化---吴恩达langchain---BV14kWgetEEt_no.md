## 机器学习基础概念

上一节我们介绍了量化交易的数据准备，本节中我们来看看机器学习的核心思想。

机器学习与传统编程的核心区别在于规则的来源。在传统编程中，程序员根据经验定义明确的规则（例如，如果MA5上穿MA10则看涨）。而在机器学习中，模型从海量的历史数据中自动学习出这些规则。这种方法能够发现人眼难以察觉的高维数据中的隐藏关系，从而可能得出更科学、更合理的交易策略。

机器学习主要分为三大类：

以下是三种主要的机器学习类型：
*   **监督学习**：模型从带有标签（Y值）的数据中学习。在量化中，Y可以是“涨/跌”（分类问题）或具体的价格/收益率（回归问题）。
*   **非监督学习**：模型从未标记的数据中发现内在结构。例如，对多只股票的时间序列进行聚类，发现哪些股票走势相似。
*   **强化学习**：智能体在与环境（如股市）的交互中学习，通过尝试不同的行动（买/卖）并获得奖励（收益）来优化策略。其理论基础是马尔可夫决策过程。

## Scikit-learn 使用流程

上一节我们了解了机器学习的基本类型，本节中我们来看看如何使用Python中最流行的机器学习库Scikit-learn。

![](img/5281913dcc904118cc9f28404cd8b304_4.png)

Scikit-learn（sklearn）提供了简洁一致的API，极大简化了建模过程。一个典型的机器学习流程包含以下三步：

以下是使用Scikit-learn的标准三步流程：
1.  **数据准备**：获取并整理数据。训练集 `X_train` 通常是一个 `n_samples × n_features` 的矩阵，`y_train` 是对应的标签向量（分类）或实值向量（回归）。务必使用 `.shape` 检查数据维度。
2.  **模型训练**：选择一个模型（如线性回归、SVM），创建实例，并用训练数据拟合它。`model.fit(X_train, y_train)` 这一行代码会学习出模型的最优参数。
3.  **模型评估**：使用训练好的模型对测试集 `X_test` 进行预测 `y_pred = model.predict(X_test)`，然后将预测结果 `y_pred` 与真实值 `y_test` 进行比较，使用评分函数（如准确率、均方误差）评估模型性能。

**核心公式**：`model.fit(X_train, y_train)` -> `y_pred = model.predict(X_test)` -> `score = evaluation_function(y_test, y_pred)`

**重要提示**：模型在测试集上的评分（score）好，**完全不等于**在真实回测或实盘中表现好。评分只是一个初步的筛选，用于淘汰明显无效的模型。

## 从线性回归到正则化

上一节我们熟悉了sklearn的基本流程，本节中我们深入探讨最基础也是最重要的线性模型家族。

### 线性回归
线性回归假设输出Y与输入X之间存在线性关系。其目标是找到一组参数θ，使得预测值与真实值之间的误差平方和最小（最小二乘法）。

**核心公式**：
*   模型：`Y = X · θ` （其中X是N×D矩阵，θ是D×1向量）
*   闭式解：`θ = (X^T · X)^(-1) · X^T · Y`

### 领回归
在实际计算中，矩阵 `(X^T · X)` 可能不可逆（病态）。数值计算中的一个技巧是给它加上一个小的对角矩阵 `λI`。有趣的是，这样做不仅解决了计算问题，往往还能提升模型在测试集上的泛化性能。

这引导出了领回归。它在最小二乘法的损失函数中，增加了一个对参数θ的L2范数（平方和）惩罚项，目的是防止参数值过大。

**核心公式**：
损失函数：`J(θ) = ||Y - Xθ||² + λ||θ||²`
其中 `λ` 是控制惩罚力度的超参数。几何解释是，在最小化误差的同时，限制参数向量θ在一个圆（或超球）内。

### 拉索回归
领回归会让所有参数同时缩小，但不会将任何参数**恰好**变为零。拉索回归采用了L1范数（绝对值之和）作为惩罚项。

**核心公式**：
损失函数：`J(θ) = ||Y - Xθ||² + λ|θ|`
由于其几何特性（菱形约束区域与椭圆误差等高线相交于坐标轴），拉索回归倾向于产生稀疏解，即会将一些不重要的特征对应的参数**压缩至零**，从而实现**特征选择**。

## 从线性到非线性：核方法

上一节我们介绍了带正则化的线性模型，本节中我们来看看如何用线性模型处理非线性问题。

一个巧妙的思路是：保持模型本身是线性的，但对输入特征X进行非线性变换。例如，将一维特征X扩展为 `[X, X², X³]`，然后对这个新的三维特征做线性回归，拟合出的就是一个曲线。

核方法（如径向基函数网络）将这一思想推向极致。它在输入空间的多个位置（如训练样本点或聚类中心）放置一系列的“基函数”（通常是高斯核），然后学习这些基函数输出的线性组合，以此来逼近任意复杂的函数。

**核心公式（高斯核）**：
`K(x, μ) = exp(-||x - μ||² / (2σ²))`
其中 `μ` 是基函数的中心位置，`σ` 控制函数的宽度。模型最终预测为：`y = Σ θ_i · K(x, μ_i)`

这种方法本质上是使用一组非线性基函数对原始特征空间进行重构，然后在新的空间中进行线性回归。

## 模型选择与交叉验证

上一节我们学会了构建多种模型，本节中我们来解决一个关键问题：如何选择模型的最佳超参数（如λ、σ）？

我们不能直接在测试集上选择超参数，这会导致“数据窥探”，使评估结果过于乐观。科学的做法是使用**交叉验证**。

标准流程如下：
1.  将数据分为**训练集**、**验证集**和**测试集**（例如70%，15%，15%）。
2.  在训练集上，用不同的超参数组合训练多个模型。
3.  在**验证集**上评估这些模型，选择性能最好的超参数组合。
4.  用选出的超参数，在**训练集+验证集**上重新训练最终模型。
5.  在**测试集**上评估最终模型，得到对外汇报的性能指标。
6.  如果决定部署，最后使用**全部数据**（训练+验证+测试）和选定超参数训练最终的生产模型。

对于多个超参数，可以使用网格搜索配合热力图来直观展示不同组合的性能。

## 量化中的特征工程

前面我们学习了模型本身，本节中我们回到量化交易的起点：如何定义模型的输入X？

对于每一个时间点t，我们需要构建一个特征向量 `X_t`，用以预测下一个时间点的目标 `Y_t`（例如 `price(t+1)`）。`X_t` 的定义是量化策略的核心，体现了研究者对市场的理解。

特征可以来源于：
*   **量价数据**：开盘价、收盘价、最高价、最低价、成交量。
*   **技术指标**：移动平均线（MA）、相对强弱指数（RSI）、布林带（Bollinger Bands）、MACD等。这些指标是对原始量价数据的数学变换。
*   **其他信息**：新闻情绪、宏观经济数据等（需要将其编码为数值特征）。

**实践建议**：可以从实现经典的技术指标函数库开始，这是构建特征池的基础。特征工程没有固定套路，需要结合领域知识进行大量实验和验证。

## 总结
本节课中我们一起学习了机器学习在量化交易中的应用基础。我们从最简单的线性回归出发，探讨了通过L1/L2正则化提升模型泛化能力的领回归和拉索回归，并介绍了通过核方法将线性模型用于非线性拟合的技巧。我们重点掌握了使用交叉验证进行模型选择和超参数调优的科学流程。最后，我们明确了量化建模的关键一步——特征工程，即如何为每个时间点构建有预测力的特征向量X。这些知识为我们后续构建和评估量化交易模型奠定了坚实的理论基础。
