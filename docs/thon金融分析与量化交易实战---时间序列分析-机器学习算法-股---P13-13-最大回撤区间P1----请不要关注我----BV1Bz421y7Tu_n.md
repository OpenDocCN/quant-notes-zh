## 最大回撤的定义与意义
上一节我们介绍了多种策略评估指标，本节中我们来看看**最大回撤**。

最大回撤描述的是在投资过程中，资产净值从最高点到后续最低点的**最大跌幅**。它衡量的不是策略的盈利能力，而是策略在运行期间可能经历的**最大风险**或“惨烈程度”。

具体来说，它寻找的是投资期间内，从某个峰值到之后某个谷值**下跌最严重**的那段区间。即使策略最终是盈利的，但中间可能经历了一段非常痛苦的下跌过程，最大回撤就是量化这段痛苦程度的指标。

其核心公式如下：

**最大回撤 = max( (P_i - P_j) / P_i )**

![](img/621695af6a704bd8dd929ba33498052a_4.png)

其中：
*   **P_i** 代表第 i 天的资产价格（或净值）。
*   **P_j** 代表第 i 天之后某一天（第 j 天）的资产价格（或净值），且 **P_j < P_i**。
*   公式的含义是：遍历所有可能的 i 和 j（j > i），计算从 i 日高点到 j 日低点的**回撤率**，并取其中的最大值。

## 最大回撤的计算思路
理解了最大回撤的概念后，我们来看看如何通过编程计算它。计算的关键在于高效地找到那个“最高点”和其后的“最低点”。

![](img/621695af6a704bd8dd929ba33498052a_8.png)

![](img/621695af6a704bd8dd929ba33498052a_10.png)

以下是计算最大回撤的核心步骤：

![](img/621695af6a704bd8dd929ba33498052a_12.png)

![](img/621695af6a704bd8dd929ba33498052a_14.png)

1.  **计算历史最高点序列**：我们需要为每一天计算一个值，即“到该天为止的历史最高净值”。这可以通过Pandas的 `.cummax()` 方法轻松实现。
2.  **计算每日回撤**：用每一天的“历史最高净值”减去当天的“实际净值”，就得到了每一天相对于其之前最高点的回撤值。
3.  **计算每日回撤率**：将每日回撤值除以对应的“历史最高净值”，得到每日的回撤率。
4.  **找出最大值**：在所有每日回撤率中，找出**最大值**，这就是整个考察期内的最大回撤率。

![](img/621695af6a704bd8dd929ba33498052a_16.png)

![](img/621695af6a704bd8dd929ba33498052a_18.png)

## 使用Python计算最大回撤
我们将通过一个简单的代码示例来演示上述计算过程。假设我们有一个包含资产净值的时间序列 `data[‘net_value’]`。

![](img/621695af6a704bd8dd929ba33498052a_24.png)

![](img/621695af6a704bd8dd929ba33498052a_26.png)

首先，我们计算累积最大值序列：
```python
# 计算到每一天为止的历史最高净值
cumulative_max = data['net_value'].cummax()
```

![](img/621695af6a704bd8dd929ba33498052a_28.png)

![](img/621695af6a704bd8dd929ba33498052a_30.png)

![](img/621695af6a704bd8dd929ba33498052a_32.png)

接下来，我们计算每日回撤和回撤率：
```python
# 计算每日回撤（历史最高 - 当前净值）
drawdown = cumulative_max - data['net_value']

![](img/621695af6a704bd8dd929ba33498052a_34.png)

![](img/621695af6a704bd8dd929ba33498052a_36.png)

# 计算每日回撤率
drawdown_pct = drawdown / cumulative_max
```

![](img/621695af6a704bd8dd929ba33498052a_38.png)

![](img/621695af6a704bd8dd929ba33498052a_40.png)

最后，找出最大回撤率及其发生的日期：
```python
# 计算最大回撤率
max_drawdown = drawdown_pct.max()

![](img/621695af6a704bd8dd929ba33498052a_42.png)

![](img/621695af6a704bd8dd929ba33498052a_44.png)

# 找到最大回撤率发生的日期（即回撤最严重的那一天）
max_drawdown_date = drawdown_pct.idxmax()

![](img/621695af6a704bd8dd929ba33498052a_46.png)

![](img/621695af6a704bd8dd929ba33498052a_48.png)

# 为了找到回撤区间的起始点（最高点），我们需要找到在 max_drawdown_date 之前，净值等于 cumulative_max 的日期。
# 通常，起始点是 max_drawdown_date 之前，cumulative_max 值最后一次发生变化的日期。
peak_before_drawdown = cumulative_max.loc[:max_drawdown_date]
# 找到峰值点（即回撤开始点）
peak_date = peak_before_drawdown[peak_before_drawdown == peak_before_drawdown.max()].index[-1]
```

![](img/621695af6a704bd8dd929ba33498052a_50.png)

![](img/621695af6a704bd8dd929ba33498052a_52.png)

执行这段代码，我们就能得到最大回撤率（例如0.0307，即3.07%），以及回撤开始的峰值日期和回撤结束的谷值日期，从而确定了**最大回撤区间**。

![](img/621695af6a704bd8dd929ba33498052a_54.png)

![](img/621695af6a704bd8dd929ba33498052a_56.png)

![](img/621695af6a704bd8dd929ba33498052a_58.png)

## 总结
本节课中我们一起学习了**最大回撤**这个关键的风险评估指标。

![](img/621695af6a704bd8dd929ba33498052a_66.png)

我们首先明确了最大回撤的定义：它衡量的是资产从历史最高点到之后最低点的**最大跌幅**，反映了投资策略可能面临的**最糟糕情况**。

![](img/621695af6a704bd8dd929ba33498052a_68.png)

![](img/621695af6a704bd8dd929ba33498052a_70.png)

接着，我们拆解了计算最大回撤的四个步骤：计算历史最高序列、计算每日回撤、计算每日回撤率、最终找出最大值。并通过Python代码演示了如何实现这一计算过程，得到了最大回撤率及其对应的起止时间点。

![](img/621695af6a704bd8dd929ba33498052a_72.png)

理解并监控最大回撤至关重要，它能帮助投资者评估策略的风险承受能力。一个盈利策略如果伴随巨大的最大回撤，意味着投资者需要有极强的心理承受力才能坚持到底。因此，最大回撤与夏普比率、年化收益率等指标结合，才能对策略做出更全面的评价。
