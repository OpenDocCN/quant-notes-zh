## 特征选择的意义与方法

上一节我们介绍了如何从原始数据中构建训练集（X和Y）。本节中我们来看看如何优化这些特征，即特征选择。

当我们拍脑袋选出一系列特征（例如300个）后，其中可能包含冗余特征、无效特征（如数值全为零的列）。特征选择的目标就是从原始特征中筛选出一个高质量的子集，因为分类器的表现高度依赖于输入特征的质量。

特征选择主要有三种思路：

1.  **子集选择**：从原始P个特征中，选出一个包含K个特征的子集（K<P），然后仅用这个子集进行预测。选出的特征值本身不变。
2.  **系数收缩**：不直接剔除特征，而是通过让模型训练中不重要的特征系数自动为零（或接近零）来实现选择。例如，Lasso回归就属于这种方法。
3.  **降维**：通过某种函数变换，将原始的P维特征向量映射到一个更低维度（如20维）的新空间。新特征的每一维都是原始特征的某种线性或非线性组合，其物理意义已发生改变。主成分分析（PCA）、自编码器（Autoencoder）都属于降维方法。

---

## 子集选择的具体策略

我们首先聚焦于第一种方法：如何选出一个最优的特征子集。

评价一个特征子集好坏的标准，通常是它在回归任务上的表现，例如R方值（R²）。我们的目标是找到能使R方值最大化的那个特征组合。然而，如果暴力枚举所有可能的特征子集，其计算复杂度是指数级的（2^P），在P较大时不可行。

因此，我们需要一些启发式方法来逼近最优解。

以下是几种常见的子集选择策略：

*   **前向选择**：这是一种贪心算法。开始时，特征子集为空。在每一步，从剩余未选的特征中，选择一个能使得模型性能（如R方）提升最大的特征加入子集。如此反复，直到性能不再显著提升或达到预定特征数。
*   **后向选择**：与前向选择相反。开始时包含所有特征。在每一步，尝试移除一个特征，移除那个对模型性能损害最小的特征。重复此过程，直到达到停止条件。
*   **暴力枚举（仅适用于极小P）**：当特征数非常少（例如5个）时，可以枚举所有可能的特征组合（1个特征的所有可能，2个特征的所有组合...），并选择性能最好的那个子集。

## 遗传算法：受生物进化启发的优化方法

上一节我们介绍了基于贪心策略的特征选择方法。本节中我们来看看一种完全不同、但更强大的全局优化方法——遗传算法。

遗传算法和神经网络一样，都是受生物现象启发的算法，虽然没有非常坚实的理论背景，但在实践中极其有效。它能解决各种优化问题，包括特征选择、投资组合优化等。

### 核心思想与生物基础

遗传算法的理论依据很简单：
1.  **种群与适应度**：一个种群由许多个体组成。每个个体对环境有一个“适应度”。适应环境的个体存活并繁衍，不适应的则被淘汰。
2.  **基因与遗传**：个体的特性由其“基因型”（如一串二进制代码）决定。在繁衍过程中，基因会通过“交叉”和“变异”产生新的组合。

对应到计算机模型：
*   **基因型**：用一串编码（如二进制串）表示问题的解。例如，在300个特征的选择问题中，一个300位的二进制串`10011001...`就代表一个解，其中`1`表示选择该特征，`0`表示不选。
*   **表现型**：基因型所对应的具体解。在上述例子中，表现型就是被选中的那组特征。
*   **适应度函数**：评价一个解好坏的函数。在特征选择中，适应度函数可以是所选特征子集在验证集上的R方值。

### 算法流程

遗传算法通过模拟自然选择的过程来寻找最优解：

1.  **初始化种群**：随机生成一定数量（如1000个）的基因型（二进制串）。
2.  **评估适应度**：计算种群中每个个体的适应度（如R方值）。
3.  **选择**：根据适应度高低，以一定概率选择优秀的个体进入“交配池”。适应度越高，被选中的概率越大。一种简单实现是：计算个体i被选中的概率 `P_i = fitness_i / sum(fitness)`，然后生成一个0-1的随机数，若随机数小于 `P_i` 则选中该个体。
4.  **交叉**：从“交配池”中随机选取两个个体作为父母，以一定概率交换它们基因串的某些片段，产生两个新的子代个体。
5.  **变异**：以很小的概率（如0.05%），随机改变子代个体基因串上的某些位（0变1或1变0）。
6.  **迭代**：用新生成的子代个体组成新的种群，替换旧种群。重复步骤2-5，进行多轮（如100代）演化。
7.  **输出**：演化结束后，种群中适应度最高的个体即为所求的近似最优解。

### 遗传算法的特点与应用

遗传算法的优势在于它能处理复杂的非凸优化问题，且对问题形式要求宽松，只需要能定义适应度函数即可。其缺点是可能需要较大的种群规模和迭代次数，且不能保证收敛到全局最优。

**课后实践**：尝试用遗传算法解决一个经典问题——猜测密码。例如，真实密码是“hello world”。你有一个函数，输入一个字符串，输出该字符串与真实密码匹配的字符个数。请设计遗传算法，从随机字符串开始，演化出正确的密码。

## 神经网络与反向传播算法

遗传算法为我们提供了一种全局搜索的利器。本节中，我们将深入另一大“玄学”法宝——神经网络，并彻底理解其核心训练算法：反向传播。

### 从生物神经元到数学模型

神经网络灵感来源于大脑神经元。一个简单的神经元模型（感知机）如下：
它接收多个输入信号 `X1, X2, ..., Xn`，每个信号乘以一个权重 `W1, W2, ..., Wn`，然后求和。如果加权和超过某个阈值 `T`，则神经元“激活”，输出1；否则输出0。

用公式表示其决策过程：
`输出 = 1 if (∑(Wi * Xi) > T) else 0`

为了便于后续的优化计算，我们做两个重要改进：
1.  **消除阈值T**：引入一个偏置项 `W0`，其输入恒为1。这样，决策规则变为 `输出 = 1 if (W0 + ∑(Wi * Xi) > 0) else 0`。阈值 `T` 被吸收进了权重 `W0`。
2.  **软化激活函数**：将阶梯状的硬阈值函数替换为光滑可导的西格莫德函数（Sigmoid）。Sigmoid函数定义为 `σ(x) = 1 / (1 + e^{-x})`，其输出在0到1之间连续变化，且导数有简便形式：`σ'(x) = σ(x) * (1 - σ(x))`。

### 网络训练与损失函数

一个神经网络由多层这样的神经元连接而成。我们的目标是找到一组权重 `W`，使得网络对于训练集中的输入 `X`，能输出尽可能接近真实值 `Z` 的预测值 `D`。

为此，我们定义一个损失函数 `P` 来衡量预测的好坏，例如均方误差：
`P = -1/2 * (Z - D)^2`
我们的目标就是最大化 `P`（即最小化误差）。`P` 是所有权重 `W` 的函数，因此神经网络的训练问题，转化为了寻找使 `P` 最大化的那组 `W`。

### 梯度下降与链式法则

求解最大化问题常用梯度下降法。对于某个权重 `Wj`，其更新规则为：
`Wj_new = Wj_old + α * (∂P/∂Wj)`
其中 `α` 是学习率。核心难点在于如何高效计算损失函数 `P` 对每一个权重 `Wj` 的偏导数 `∂P/∂Wj`。

以最简单的两层网络为例（输入X -> 隐藏层 -> 输出Z）。根据链式法则，损失 `P` 对第二层权重 `W2` 的偏导可以分解：
`∂P/∂W2 = (∂P/∂Z) * (∂Z/∂P2) * (∂P2/∂W2)`
其中：
*   `∂P/∂Z = (Z - D)` （损失函数对输出的偏导）
*   `∂Z/∂P2 = σ(P2) * (1 - σ(P2))` （Sigmoid函数的导数）
*   `∂P2/∂W2 = Y` （第二层输入Y，即第一层的输出）

类似地，可以求出 `P` 对第一层权重 `W1` 的偏导，链条更长：
`∂P/∂W1 = (∂P/∂Z) * (∂Z/∂P2) * (∂P2/∂Y) * (∂Y/∂P1) * (∂P1/∂W1)`

### 反向传播的巧妙之处：动态规划

对于深层网络，如果对每个权重都独立地使用链式法则从头计算偏导，计算量会随着网络层数指数级增长。反向传播算法的精妙之处在于，它采用了一种**动态规划**的思想，从输出层向输入层反向计算。

具体来说：
1.  首先计算输出层的误差 `∂P/∂Z`。
2.  反向传递时，计算下一层（靠近输出）的误差时，可以复用上一层已经计算好的结果。
3.  这样，每个神经元的误差只需计算一次，就可以用于更新所有与之相连的权重。

这种方法将计算复杂度从指数级降低到了与网络层数呈线性关系，使得训练深层神经网络成为可能。**这才是反向传播算法真正的核心贡献**。

## 循环神经网络简介

前面介绍的前馈神经网络在处理时序数据时，无法记忆历史信息。循环神经网络（RNN）应运而生，它让网络具备了“记忆”能力。

RNN的核心思想是：网络在当前时刻 `t` 的输出 `Y_t`，不仅依赖于当前输入 `X_t`，还依赖于网络在上一时刻 `t-1` 的隐藏状态 `H_{t-1}`。这个隐藏状态充当了网络的“记忆”，包含了历史信息。

![](img/86567fee413a1e4a02181891a6e8c0a3_2.png)

![](img/86567fee413a1e4a02181891a6e8c0a3_4.png)

![](img/86567fee413a1e4a02181891a6e8c0a3_6.png)

其数学表达可简化为：
`H_t = σ( U * X_t + W * H_{t-1} + b)`
`Y_t = V * H_t + c`
其中 `U, W, V` 是权重矩阵，`b, c` 是偏置项，`σ` 是激活函数。

![](img/86567fee413a1e4a02181891a6e8c0a3_8.png)

![](img/86567fee413a1e4a02181891a6e8c0a3_10.png)

![](img/86567fee413a1e4a02181891a6e8c0a3_12.png)

![](img/86567fee413a1e4a02181891a6e8c0a3_14.png)

通过这种结构，RNN在预测 `Y_t` 时，理论上能考虑到所有历史输入 `X_0, X_1, ..., X_{t-1}` 的信息。长短期记忆网络（LSTM）是RNN的一种改进，通过引入更复杂的“门”结构（输入门、遗忘门、输出门），有效缓解了RNN训练中的梯度消失问题，能够学习更长期的依赖关系。
