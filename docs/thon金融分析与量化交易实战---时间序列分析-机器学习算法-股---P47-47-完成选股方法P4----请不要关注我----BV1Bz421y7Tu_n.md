# 量化交易实战课程：P47：完成选股方法

## 概述
在本节课中，我们将学习如何完成一个核心的选股函数。我们将通过拼接多个因子数据表、计算综合得分并进行排序，最终筛选出排名前十的股票。整个过程将涉及Pandas DataFrame的操作、NumPy数组的创建以及数据排序。

---

## 数据拼接操作
上一节我们介绍了如何获取单个因子的数据，本节中我们来看看如何将多个因子的数据表合并在一起。

以下是实现数据拼接的步骤：

1.  使用一个DataFrame的`.join()`方法去连接另一个DataFrame。
2.  将拼接后的新DataFrame赋值给一个变量，例如`df_combined`。

代码实现如下：
```python
df_combined = df_down.join(df_second)
```
现在，两个DataFrame已经成功拼接起来了。

---

## 创建得分列并初始化
数据合并后，我们需要一个列来存放每只股票的综合得分。

我们在这个合并后的DataFrame中新建一个名为“得分值”的列，并暂时将其所有值初始化为0。
```python
df_combined[‘得分值’] = 0
```

---

## 计算综合总分
初始化完成后，我们需要实际计算这个“得分值”。思路是将所有因子的分数相加。

假设我们有六个因子指标，计算总分就是将这六个指标的值进行求和。
```python
# 计算所有因子列的总和，假设因子列名为 ‘factor1’, ‘factor2’...
df_combined[‘得分值’] = df_combined[[‘factor1’, ‘factor2’, ‘factor3’, ‘factor4’, ‘factor5’, ‘factor6’]].sum(axis=1)
```
这样，`df_combined[‘得分值’]`列中就存储了每只股票的综合总分。

---

## 根据总分排序
得到总分后，下一步是根据得分对股票进行排序，以找出表现最好的股票。

我们需要按“得分值”列进行降序排列，因为分数越高代表股票越好。
```python
df_sorted = df_combined.sort_values(by=‘得分值’, ascending=False)
```
排序完成后，`df_sorted`这个DataFrame中的股票就已经按照综合得分从高到低排列好了。

---

## 提取前十名股票
排序不是最终目的，我们的目标是获取排名前十的股票代码。

![](img/688bcc58e3e6f52be295005a46b7154e_1.png)

我们不需要所有数据，只需要排名前十的股票索引（即股票代码）。因此，我们从排序后的DataFrame中提取索引并转换为列表，然后取前十个元素。
```python
top_10_stocks = df_sorted.index.tolist()[:10]
```
至此，`top_10_stocks`列表中就包含了我们选出的前十名股票。

![](img/688bcc58e3e6f52be295005a46b7154e_3.png)

## 总结
本节课中我们一起学习了如何构建一个完整的选股函数。整个过程可以概括为以下几步：

1.  **读取并拼接因子数据**：将多个因子的数据表合并为一个统一的DataFrame。
2.  **初始化与计算综合得分**：创建得分列，并将所有因子分数相加得到总分。
3.  **排序与筛选**：根据总分进行降序排序，并提取排名前十的股票代码。

![](img/688bcc58e3e6f52be295005a46b7154e_7.png)

这个函数是量化选股策略的核心，执行一次即可获得基于当前多个指标综合评分最高的前十只股票，为后续的交易决策提供直接依据。
