## 什么是字符串？

上一节我们介绍了列表等数据结构，本节中我们来看看字符串。字符串是由一系列字符组成的序列，在Python中，用单引号（`'`）或双引号（`"`）括起来的内容就是字符串。

```python
string1 = '佟掌柜微信'
string2 = "we love money"
string3 = "list, dict, 123"
```
字符串的含义是“所见即所得”，引号内的所有内容，包括空格，都会被原样输出。

## 字符串的拼接

了解了字符串的定义后，我们来看看如何将多个字符串组合在一起，即字符串的拼接。

以下是几种常见的字符串拼接方法：

*   **使用 `join()` 方法**：`‘分隔符’.join(列表或元组)`。例如，`‘’.join([‘佟掌柜’, ‘热爱’, ‘搞钱’])` 会输出 `‘佟掌柜热爱搞钱’`。
*   **使用加号 `+`**：直接连接字符串。例如，`‘3w.’ + ‘i love money’ + ‘.com’` 会得到 `‘3w.i love money.com’`。
*   **在字符串中直接包含分隔符**：例如，`‘grade/is/good’`，字符串本身包含了斜杠 `/`。

## 字符串的转义符

有时我们需要在字符串中包含特殊字符（如引号本身），或者表示制表符、换行符等，这时就需要用到转义符（`\`）。

以下是转义符的用法示例：

*   **输出包含引号的字符串**：`‘what\’s up’` 可以正确输出 `what‘s up`。如果没有转义符 `\`，解释器会因引号匹配错误而报错。
*   **表示特殊字符**：`\t` 代表制表符（Tab），`\n` 代表换行符。
*   **输出反斜杠本身**：使用两个反斜杠 `\\` 来输出一个反斜杠 `\`。
*   **原始字符串**：在字符串前加 `r` 或 `R`，如 `r‘C:\Users\path’`，会将其中的反斜杠视为普通字符，而不是转义符。这在处理文件路径时非常有用。

## 字符串的选取与切片

字符串可以像列表一样进行索引和切片操作，因为字符串本质上也是一个字符序列。

以下是字符串选取操作的示例：

*   `name = ‘佟掌柜爱量化’`
*   `name[0]` 选取第0位字符：`‘佟’`
*   `name[:4]` 选取从开始到第4位（不包含）的字符：`‘佟掌柜爱’`
*   `name[2:]` 选取从第2位到末尾的字符：`‘爱量化’`
*   `len(name)` 获取字符串长度：`6`
*   `name[-1]` 选取最后一个字符：`‘化’`

## 字符串的大小写转换

Python提供了方便的方法来转换字符串中字母的大小写。

以下是大小写转换的方法：

*   **`upper()`**：将所有字母转换为大写。
*   **`lower()`**：将所有字母转换为小写。
*   **`title()`**：将每个单词的首字母大写，其余小写。
*   **`capitalize()`**：将整个字符串的首字母大写，其余小写。
*   **`swapcase()`**：将字符串中的大写字母转为小写，小写字母转为大写。

例如，对于字符串 `‘i love Python’`：
*   `upper()` 结果是 `‘I LOVE PYTHON’`
*   `title()` 结果是 `‘I Love Python’`

## 字符串的常用检索方法

我们经常需要在一个字符串中查找特定内容，Python提供了多种检索方法。

以下是常用的字符串检索方法：

*   **`count(‘子串’)`**：统计子串在字符串中出现的次数。区分大小写。
*   **`index(‘子串’)`**：查找子串首次出现的位置索引。如果子串不存在，会引发错误。
*   **`startswith(‘前缀’)`**：检查字符串是否以指定前缀开头，返回 `True` 或 `False`。
*   **`endswith(‘后缀’)`**：检查字符串是否以指定后缀结尾，返回 `True` 或 `False`。

## 字符串的分割与修剪

数据处理中，我们常需要将长字符串拆分成多个部分，或者去掉不需要的空格。

以下是字符串分割与修剪的方法：

*   **分割 `split()`**：`split(‘分隔符’， 最大分割次数)`。默认以空格、`\t`、`\n`、`\r` 为分隔符。例如，`‘我爱你 Python’.split()` 得到 `[‘我爱你’， ‘Python’]`。
*   **修剪 `strip()`**：移除字符串**开头和结尾**的指定字符（默认为空格）。例如，`‘ 佟掌柜 ‘.strip()` 得到 `‘佟掌柜’`。

## 字符串的其他操作

除了上述操作，字符串还支持一些其他实用功能。

以下是字符串的其他操作示例：

*   **相加与相乘**：`‘刘备’ + ‘关羽’ + ‘张飞’` 得到 `‘刘备关羽张飞’`。`‘刘备’ * 3` 得到 `‘刘备刘备刘备’`。
*   **成员判断**：使用 `in` 关键字判断子串是否存在，如 `‘佟’ in ‘佟掌柜’` 返回 `True`。
*   **替换 `replace()`**：`replace(‘旧子串’， ‘新子串’)`。例如，`‘佟掌柜很丑’.replace(‘丑’， ‘帅’)` 得到 `‘佟掌柜很帅’`。

## 字符串格式化的应用

字符串格式化能将变量按特定格式嵌入到字符串模板中，是生成动态文本的强大工具。

以下是使用 `%` 进行格式化的方法（旧式，但直观）：
```python
name = ‘佟掌柜’
login_time = 10
cost = 6.8888
info = ‘你好，%s，这是你今天第%d次查询银行卡，余额%.2f元。’ % (name, login_time, cost)
print(info) # 输出：你好，佟掌柜，这是你今天第10次查询银行卡，余额6.89元。
```
*   `%s` 对应字符串变量。
*   `%d` 对应整数变量。
*   `%.2f` 对应浮点数变量，并保留两位小数（四舍五入）。

在实际量化编程中，这常用于批量生成查询语句或文件名。例如，批量下载多个合约数据：
```python
contract_list = [‘IF2401’， ‘IF2402’， ‘IC2401’]
for contract in contract_list:
    sql = “SELECT * FROM market_data WHERE contract=‘%s’” % contract
    # 执行sql查询...
```
这样只需维护一个合约列表，就能自动生成所有查询，极大提高了效率。
