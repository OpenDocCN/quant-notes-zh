## Python的核心优势
上一节我们介绍了Python的流行程度，本节中我们来看看Python为何能击败一众对手，成为最受欢迎的编程语言之一。

Python最大的优点就是简单易学，语法简洁优美，新手也能轻松看懂。其对于数据的处理，也因为它自带各种工具包，而拥有得天独厚的优势。相对于其他编程语言，Python还有着很多优势。

正因为如此，国家才会如此重视Python编程教育，企业也对能使用Python的求职者青睐有加。我们在百度指数变化中可以看到，Python已经远超Java以及其他竞争对手，成为现今最受欢迎的编程语言。

## Python能做什么？
说了这么多，Python究竟能干什么呢？Python可以应用于我们生活工作中非常广泛的领域。

![](img/5ec691621398a42203c9c0cfe1e30c84_8.png)

以下是几个具体的应用示例：

![](img/5ec691621398a42203c9c0cfe1e30c84_10.png)

*   **自动化办公**：例如，老板给了我们一个任务，把很多Excel文件里面的所有数据整合到同一个Excel表格中。我们当然可以逐个复制粘贴，但是这种方法费时费力。而Python却能轻松帮你搞定。你需要做的仅仅是花费一点时间来输入代码，最后点一下运行。Python就能把处理后的Excel表格准确无误地展现在你眼前。
*   **批量下载**：Python可以一键下载多部电影。
*   **邮件与图像处理**：Python可以实现批量自动发送邮件、批量抠图等等。有些功能甚至在十行代码内就可以搞定。
