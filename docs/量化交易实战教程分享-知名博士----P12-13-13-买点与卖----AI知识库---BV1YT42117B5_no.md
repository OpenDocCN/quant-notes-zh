## 计算移动平均线

![](img/ea2875472fd31cb47775e100196b0b61_34.png)

上一节我们完成了数据清洗，本节中我们来看看如何计算技术指标。我们将计算短期和长期简单移动平均线（SMA），这是识别趋势和交叉点的基础。

首先，定义短期和长期的窗口大小。由于我们的数据集时间跨度较长（约7-8年），为了在图表中更清晰地观察趋势，我们将窗口设置得比通常的5日和20日更大一些。

```python
# 定义短期和长期移动平均的窗口大小
short_window = 40
long_window = 100
```

![](img/ea2875472fd31cb47775e100196b0b61_36.png)

接下来，我们使用`rolling`方法计算移动平均线，并将结果作为新列添加到数据框中。

![](img/ea2875472fd31cb47775e100196b0b61_38.png)

```python
# 计算短期移动平均线 (SMA)
apple_data['SMA_short'] = apple_data['AAPL'].rolling(window=short_window).mean()

![](img/ea2875472fd31cb47775e100196b0b61_40.png)

![](img/ea2875472fd31cb47775e100196b0b61_42.png)

# 计算长期移动平均线 (SMA)
apple_data['SMA_long'] = apple_data['AAPL'].rolling(window=long_window).mean()

![](img/ea2875472fd31cb47775e100196b0b61_44.png)

# 查看计算结果
print(apple_data.tail())
```

![](img/ea2875472fd31cb47775e100196b0b61_46.png)

![](img/ea2875472fd31cb47775e100196b0b61_48.png)

计算完成后，数据的前面部分会出现一些`NaN`值，这是因为移动平均计算需要足够的窗口数据。这是正常现象。

## 标记交易信号

![](img/ea2875472fd31cb47775e100196b0b61_68.png)

为了进行更系统的分析，我们需要将交叉点转化为明确的交易信号。我们将创建一个新的列`Position`，用于标记当前短期均线是高于还是低于长期均线。

![](img/ea2875472fd31cb47775e100196b0b61_70.png)

![](img/ea2875472fd31cb47775e100196b0b61_72.png)

在创建信号之前，需要先删除移动平均计算产生的`NaN`值。

```python
# 删除因计算移动平均而产生的缺失值
apple_data = apple_data.dropna()

![](img/ea2875472fd31cb47775e100196b0b61_74.png)

# 创建交易信号列：当短期均线高于长期均线时标记为1，否则为0
apple_data['Position'] = np.where(apple_data['SMA_short'] > apple_data['SMA_long'], 1, 0)

![](img/ea2875472fd31cb47775e100196b0b61_76.png)

# 查看带有信号的数据
print(apple_data[['AAPL', 'SMA_short', 'SMA_long', 'Position']].head())
```

![](img/ea2875472fd31cb47775e100196b0b61_78.png)

`Position`列的值为1代表短期趋势强于长期趋势（潜在持有或买入区间），为0则代表相反（潜在卖出或观望区间）。

![](img/ea2875472fd31cb47775e100196b0b61_80.png)

我们也可以将交易信号可视化：

## 策略思路与后续分析

通过上述步骤，我们已经能够识别出基于双均线交叉理论的潜在买卖点。这引出了一个重要的量化交易概念：**主动策略与被动持有的对比**。

*   **被动持有策略**：假设在最早的时间点买入股票并一直持有到最后，其收益完全取决于股价的最终涨跌。
*   **主动交易策略**：根据我们生成的交易信号（如`Position`列）进行操作。理论上，在“死亡交叉”（卖出信号）时卖出，在“黄金交叉”（买入信号）时买入，可以规避部分下跌风险，并尝试捕捉上涨机会。

![](img/ea2875472fd31cb47775e100196b0b61_88.png)

本节课中我们一起学习了如何：
1.  导入和预处理金融时间序列数据。
2.  计算短期和长期简单移动平均线（SMA）。
3.  将移动平均线可视化，并观察其交叉点。
4.  根据均线位置关系生成简单的二进制交易信号（1/0）。

![](img/ea2875472fd31cb47775e100196b0b61_90.png)

这为我们后续进行策略回测和收益计算奠定了坚实的基础。在接下来的课程中，可以基于这些信号模拟交易，计算策略收益，并与简单的买入持有策略进行比较。
