## 九九乘法表的实现

![](img/b11e3c0df6ef0e49c212b084a7b5d924_1.png)

上一节我们介绍了混合引用的概念，本节中我们来看看如何利用它来制作一个九九乘法表。

我们分析乘法表的规律：只有当行号小于或等于列号时，才需要显示乘法结果。例如，在第二行第一列（2x1）的单元格中，行号2大于列号1，因此该单元格应为空。反之，在第二行第二列（2x2）的单元格中，行号等于列号，则需要显示结果。

根据这个条件，我们可以使用IF函数进行判断。公式的逻辑是：如果当前单元格的行号（例如A31）大于或等于列号（例如B30），则显示乘积，否则显示为空。

![](img/b11e3c0df6ef0e49c212b084a7b5d924_3.png)

![](img/b11e3c0df6ef0e49c212b084a7b5d924_5.png)

为了实现公式的拖动填充，我们需要使用混合引用。在公式中，我们希望行号在向下拖动时变化，列号在向右拖动时变化。因此，我们在行号前加美元符号（`$`）锁定行，在列号前加美元符号（`$`）锁定列。

以下是实现公式：
```excel
=IF($A31>=B$30, $A31*B$30, "")
```
将这个公式向右再向下拖动，即可生成完整的九九乘法表。这是一个理解混合引用的经典案例。

## 多条件判断：成绩统计

![](img/b11e3c0df6ef0e49c212b084a7b5d924_7.png)

![](img/b11e3c0df6ef0e49c212b084a7b5d924_9.png)

接下来，我们处理一个学生成绩统计问题。需要判断学生是否所有科目都及格，以及是否有任意科目及格。

以下是需要使用的核心函数：
*   **AND函数**：所有条件都为真时返回`TRUE`。
*   **OR函数**：任意一个条件为真时返回`TRUE`。
*   **IF函数**：根据条件返回不同的结果。

**判断三门科目是否全部及格**
我们使用`IF`嵌套`AND`函数。条件是数学、计算机、英语三科成绩均大于等于60。
```excel
=IF(AND(B2>=60, C2>=60, D2>=60), "TRUE", "FALSE")
```
这个公式的含义是：如果三个条件都满足，则返回字符串“TRUE”，否则返回“FALSE”。

**判断是否有任意科目及格**
我们使用`IF`嵌套`OR`函数。条件是三门科目中任意一科成绩大于等于60。
```excel
=IF(OR(B2>=60, C2>=60, D2>=60), "TRUE", "FALSE")
```
这个公式的含义是：如果任意一个条件满足，则返回字符串“TRUE”，否则返回“FALSE”。

## 阶梯费用计算：水电气账单

![](img/b11e3c0df6ef0e49c212b084a7b5d924_11.png)

在实际工作中，我们常遇到阶梯定价的计算，例如水电费。这需要用到`IF`函数的嵌套。

**计算电费**
电费分为三个阶梯：0-240度，241-400度，400度以上。每个阶梯单价不同。
我们需要从高用量到低用量进行判断：
1.  首先判断用量是否大于400度。
2.  如果不是，再判断是否大于240度。
3.  如果也不是，则属于第一阶梯。

![](img/b11e3c0df6ef0e49c212b084a7b5d924_13.png)

以下是电费计算公式：
```excel
=IF(E2>400, E2*$K$4, IF(E2>240, E2*$K$3, E2*$K$2))
```
注意，单价（`$K$2`， `$K$3`， `$K$4`）需要使用绝对引用，以保证公式下拉时引用位置不变。

**计算水费**
水费的计算逻辑与电费类似，也分为多个阶梯。区别在于，水费总额还需加上固定的“水资源费”和“污水处理费”。因此，在完成阶梯水费计算后，需要额外加上这两项费用。
```excel
=IF(C2>176, C2*$H$4, IF(C2>120, C2*$H$3, C2*$H$2)) + C2*($I$2+$J$2)
```
公式前半部分计算阶梯水费，后半部分`C2*($I$2+$J$2)`计算附加费。

## 条件求和与计数

![](img/b11e3c0df6ef0e49c212b084a7b5d924_15.png)

这部分练习考察`SUMIF`和`COUNTIF`函数的使用。

**条件求和 (`SUMIF`)**
例如，统计所有“借方”金额的总和。
```excel
=SUMIF(A:A, "借", B:B)
```
这个公式在A列（条件区域）中查找“借”（条件），并对对应的B列（求和区域）数值进行求和。

**条件计数 (`COUNTIF`)**
例如，统计考勤表中“迟到”（标记为B）的次数。
```excel
=COUNTIF(B2:K2, "B")
```
这个公式在B2到K2的区域（条件区域）中，统计等于“B”（条件）的单元格个数。

## 从身份证提取信息

最后，我们学习如何从身份证号码中提取出生日期、计算年龄和工龄，并查找籍贯。

**提取出生日期**
身份证号码的第7到14位是出生日期。我们使用`MID`函数提取。
```excel
=MID(A2, 7, 8)
```
提取后，数据可能是文本格式，需要通过“分列”功能或`DATE`函数转换为真正的日期格式。

**计算年龄和工龄**
使用`DATEDIF`函数可以方便地计算两个日期之间的差值。
*   **计算年龄**：
    ```excel
    =DATEDIF(B2, TODAY(), "Y")
    ```
    其中B2是出生日期，`TODAY()`是当前日期，“Y”表示返回整年数。
*   **计算工龄（精确到月）**：
    ```excel
    =DATEDIF(C2, TODAY(), "Y") & "年" & MOD(DATEDIF(C2, TODAY(), "M"), 12) & "月"
    ```
    先用`DATEDIF(..., "M")`计算总月数，再用`MOD`函数求除以12的余数得到剩余月数，与整年数拼接。

**查找省市信息**
我们需要根据身份证前6位（地址码）在辅助表中查找对应的省市。
1.  先用`LEFT`函数提取身份证前6位。
2.  再用`VLOOKUP`函数进行查找。
```excel
=VLOOKUP(LEFT(A2, 6), $M$2:$N$100, 2, 0)
```
这个公式提取A2单元格的前6位，然后在绝对引用的区域`$M$2:$N$100`中精确查找（第4参数为0），并返回该区域的第2列（省市名称）。
