在本节课中，我们将学习如何使用一个强大的量化选股工具——条件选股平台。我们将从如何一键获取已回测策略的选股结果开始，逐步深入到如何利用丰富的指标进行自定义选股，以满足个性化的投资分析需求。

## 策略回测与一键选股

上一节我们介绍了量化策略的回测，本节中我们来看看如何将回测好的策略应用于每日选股。

首先，对于平台的新用户，可以通过“量化示例”功能快速上手。这里已经整理好了我们分享过的策略模板。

以下是具体操作步骤：

1.  进入“量化示例”页面。
2.  选择你感兴趣的策略，例如“北向日增仓策略”。
3.  点击“修改回测条件”，系统会自动复制该策略的所有条件。
4.  你可以修改策略名称或回测时间，然后点击“开始回测”。
5.  回测完成后，该策略会出现在“我的策略”列表中。

完成策略回测后，就可以使用“条件选股”功能了。

以下是获取策略每日选股结果的操作步骤：

1.  在平台主页进入“条件选股”栏目。
2.  在“我的策略”中选择一个已回测的策略，例如“北向日增仓优化策略”。
3.  系统会自动填充该策略的所有选股条件。
4.  你只需选择“基准时间”（即你想选股的日期）。
5.  点击“查看结果”，平台会立即列出该日期所有符合条件的股票。

对于实盘应用，你可以在每个交易日收盘、数据更新后，使用此功能筛选出符合策略的股票，为次日的交易计划做准备。

> **重要提示**：平台数据虽经严格核对，但仍可能存在个别误差。提供的结果仅供研究参考，建议对关键数据进行二次核对。

## 自定义选股功能详解

掌握了策略的一键选股后，本节我们来探索更灵活的自定义选股功能。这能帮助你实现更复杂、个性化的选股逻辑。

要实现自定义选股，需要理解并填写三个核心栏目：**选股条件**、**股票标的池**和**选股排序**。

![](img/92701d8290ebbaf902857b3fa883a8fe_2.png)

以下是各栏目的详细说明：

*   **选股条件**：需要设定基准日期，并添加一个或多个筛选条件。条件可以是单一指标的范围，也可以是多个指标的组合。
*   **股票标的池**：用于限定选股范围。例如，可以勾选“非ST股”来排除风险警示股票，或勾选“沪深300成分股”只在特定板块内选股。不勾选则默认在全A股市场选择。
*   **选股排序**：选择用于对最终结果排序的指标，并指定升序或降序。例如，选择“换手率”并点击“升序”，结果将按换手率从小到大排列。

## 自定义选股实例演示

为了帮助大家更好地理解和使用自定义选股功能，下面通过三个具体例子来演示。

**实例一：筛选高价股**

目标：选出2024年3月1日，收盘价大于100元、非ST股、非北交所上市，并按换手率升序排列的所有股票。

操作步骤如下：

1.  **基准时间**：选择 `2024-03-01`。
2.  **添加条件**：选择“行情指标”下的“收盘价(不复权)”，最小值设为 `100`。
3.  **股票标的池**：勾选“非ST股”和“非北交所上市”。
4.  **选股排序**：选择“行情指标”下的“换手率(不复权)”，排序选“升序”。
5.  点击“查看结果”。

**实例二：复合技术指标选股**

目标：选出2024年2月10日，换手率在3%到5%之间、MACD大于0、收盘价大于5日均线、非ST股、非北交所上市，并按总市值升序排列的所有股票。

操作步骤如下：

1.  **基准时间**：选择 `2024-02-10`。
2.  **添加条件1**：选择“行情指标”下的“换手率(不复权)”，最小值 `3`，最大值 `5`。
3.  **添加条件2**：选择“技术指标”下的“MACD”，最小值 `0`。
4.  **添加条件3（核心）**：这里需使用“自定义选股指标”。因为“收盘价 > 5日均线”等价于 `收盘价 - 5日均线 > 0`。
    *   指标一选择“收盘价(不复权)”。
    *   指标二选择“5日均线”。
    *   组合方式点击 `指标一 - 指标二`。
    *   最小值设为 `0`。
5.  **股票标的池**：勾选“非ST股”和“非北交所上市”。
6.  **选股排序**：选择“估值指标”下的“总市值”，排序选“升序”。
7.  点击“查看结果”。

**实例三：技术形态选股（金叉）**

目标：选出2024年3月12日，同时满足MACD金叉且5日均线上穿10日均线的中证500成分股，并按收盘价升序排列。

操作分析：平台“形态指标”暂未直接提供“金叉”选项，但我们可以通过指标组合来等价实现。
*   **MACD金叉**：等价于 `当日MACD > 0` 且 `前一日MACD < 0`。
*   **5日均线上穿10日均线**：等价于 `当日5日均线 - 当日10日均线 > 0` 且 `前一日10日均线 - 前一日5日均线 > 0`。

操作步骤如下：

1.  **基准时间**：选择 `2024-03-12`。
2.  **添加条件1 (当日MACD>0)**：选择“技术指标”下的“MACD”，最小值 `0`。
3.  **添加条件2 (前一日MACD<0)**：再次选择“MACD”，点击“高级选项”，在“向前天数”填 `1`，最大值填 `0`。
4.  **添加条件3 (5日线上穿10日线-当日)**：选择“自定义选股指标”。
    *   指标一选“5日均线”，指标二选“10日均线”。
    *   组合方式为 `指标一 - 指标二`，最小值 `0`。
5.  **添加条件4 (5日线上穿10日线-前一日)**：再次选择“自定义选股指标”。
    *   指标一选“5日均线”，在高级选项中“向前天数”填 `1`。
    *   指标二选“10日均线”，在高级选项中“向前天数”填 `1`。
    *   组合方式为 `指标二 - 指标一`，最小值 `0`。
6.  **股票标的池**：勾选“中证500成分股”。
7.  **选股排序**：选择“行情指标”下的“收盘价(不复权)”，排序选“升序”。
8.  点击“查看结果”。
