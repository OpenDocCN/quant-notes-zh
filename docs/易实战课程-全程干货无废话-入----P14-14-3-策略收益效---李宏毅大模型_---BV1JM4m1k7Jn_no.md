在本节课中，我们将学习如何计算和比较量化策略的收益。我们将通过一个具体的例子，分析“双均线策略”相对于“什么都不做”（即持有原始资产）的收益表现。核心在于理解如何计算每日收益率，并应用策略信号来调整这些收益率，最终通过累加和对数还原得到总收益。

![](img/f6719b8a71987489304ca58cb748de37_3.png)

---

上一节我们介绍了如何生成交易信号，本节中我们来看看如何基于这些信号计算策略的最终收益。

首先，我们需要计算股价的每日收益率。收益率不能简单地将每日价格变化相加，而是需要使用对数收益率进行计算，以符合金融建模的连续复利假设。

![](img/f6719b8a71987489304ca58cb748de37_5.png)

![](img/f6719b8a71987489304ca58cb748de37_7.png)

以下是计算对数收益率的步骤：
1.  获取股价序列 `price`。
2.  计算每日的对数收益率，公式为：`log_return = np.log(price / price.shift(1))`。
3.  在Python中，这通常通过Pandas的`shift`方法实现。

![](img/f6719b8a71987489304ca58cb748de37_9.png)

```python
# 假设df[‘Close’]是收盘价序列
returns = np.log(df[‘Close’] / df[‘Close’].shift(1))
```

![](img/f6719b8a71987489304ca58cb748de37_11.png)

执行以上代码后，我们得到了一个名为`returns`的序列，它代表了资产每日的自然增长率。

![](img/f6719b8a71987489304ca58cb748de37_13.png)

![](img/f6719b8a71987489304ca58cb748de37_15.png)

---

![](img/f6719b8a71987489304ca58cb748de37_17.png)

接下来，我们需要将之前生成的策略信号应用到收益率上。我们的策略信号`position`是一个由1和-1组成的序列，其中1代表跟随市场走势，-1代表反向操作（做空或反向交易）。

以下是应用策略信号计算策略收益率的方法：
1.  策略信号`position`也需要进行`shift(1)`操作，以确保使用前一天的信号来决定今天的操作。
2.  策略每日收益率等于原始对数收益率乘以调整后的信号。
3.  公式为：`strategy_returns = returns * position.shift(1)`。

![](img/f6719b8a71987489304ca58cb748de37_19.png)

![](img/f6719b8a71987489304ca58cb748de37_21.png)

```python
# 假设position是策略信号序列
strategy_returns = returns * position.shift(1)
```

![](img/f6719b8a71987489304ca58cb748de37_23.png)

这样，在`position`为-1的区间，策略收益将与市场原始收益相反，从而实现“低买高卖”或反向套利的目标。

![](img/f6719b8a71987489304ca58cb748de37_25.png)

![](img/f6719b8a71987489304ca58cb748de37_27.png)

---

![](img/f6719b8a71987489304ca58cb748de37_29.png)

在得到策略的每日收益率序列后，我们需要计算其累计总收益。由于我们使用的是对数收益率，累加非常方便，只需对序列求和即可。

以下是计算并还原总收益的步骤：
1.  分别对原始收益率序列`returns`和策略收益率序列`strategy_returns`求和，得到总的对数收益。
2.  使用指数函数`np.exp()`将对数总收益还原为实际的总资产倍数。
3.  这个倍数代表从期初投入1元本金，到期末能获得多少元。

![](img/f6719b8a71987489304ca58cb748de37_31.png)

![](img/f6719b8a71987489304ca58cb748de37_33.png)

```python
# 计算总的对数收益
total_log_return_buy_hold = returns.sum()
total_log_return_strategy = strategy_returns.sum()

![](img/f6719b8a71987489304ca58cb748de37_35.png)

![](img/f6719b8a71987489304ca58cb748de37_37.png)

# 还原为实际收益倍数
final_value_buy_hold = np.exp(total_log_return_buy_hold)
final_value_strategy = np.exp(total_log_return_strategy)
```

执行计算后，我们可以比较两个结果。例如，原始持有策略的最终价值可能是4.0，而双均线策略的最终价值可能是5.8。这表明在该段历史数据上，我们的策略获得了约45%的超额收益（5.8 / 4.0 - 1）。

本节课中我们一起学习了量化策略收益分析的全过程。我们从计算每日对数收益率开始，接着将策略信号应用到收益率上以得到策略每日收益，最后通过累加和对数还原计算出策略的最终总收益，并与基准收益进行比较。这个过程是量化交易回测的核心环节，帮助你客观评估一个策略的历史表现。
