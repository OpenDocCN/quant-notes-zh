在本节课中，我们将深入探讨线性回归中的一个核心概念——似然函数。我们将学习如何通过似然函数来估计模型参数，并理解其背后的数学原理，特别是如何将复杂的乘法问题转化为更易处理的加法问题。

## 从误差到参数估计

上一节我们介绍了误差项服从高斯分布。本节中我们来看看如何利用这个性质来求解模型参数。

我们已知数据 `X` 和标签 `Y`，当前的目标是求解参数 `θ`。误差项 `ε` 服从均值为0的高斯分布，其概率密度函数为：

```
P(ε) = (1 / √(2πσ²)) * exp(-ε² / (2σ²))
```

这个表达式是关于误差项 `ε` 的。但我们最终关心的不是具体的误差值，而是参数 `θ`。因此，我们需要将表达式转换为与 `θ` 相关。

![](img/6de0c4bca64af9afecca8493986a4219_2.png)

根据线性回归模型 `y = θᵀx + ε`，我们可以得到 `ε = y - θᵀx`。将其代入高斯分布公式，得到：

```
P(y|x; θ) = (1 / √(2πσ²)) * exp(-(y - θᵀx)² / (2σ²))
```

这个公式可以通俗地解释为：在给定参数 `θ` 和数据 `x` 的条件下，预测值 `θᵀx` 成为真实值 `y` 的可能性。我们的目标是找到一组 `θ`，使得这个可能性对于所有数据样本都尽可能大。

## 理解似然函数

为了找到最优的 `θ`，我们需要考虑所有数据样本。这里就引入了**似然函数**的概念。

似然函数 `L(θ)` 描述了在给定当前参数 `θ` 的情况下，观测到现有所有数据 `(x, y)` 的联合概率。我们希望这个概率越大越好。由于我们假设数据样本是独立同分布的，联合概率等于每个样本边缘概率的乘积。因此，似然函数定义为：

```
L(θ) = ∏_{i=1}^{m} P(y^{(i)} | x^{(i)}; θ)
```

![](img/6de0c4bca64af9afecca8493986a4219_4.png)

以下是关于似然函数累乘形式的解释：
*   **为什么是累乘？** 因为我们需要用大量数据来共同确定最合适的参数。数据越多，估计结果越可靠。在“独立同分布”的假设下，联合概率密度恰好等于各样本概率密度的乘积。
*   **似然函数的作用**：它帮助我们评估哪一组参数 `θ` 能最好地解释已观测到的数据。例如，在赌场观察前5位客人都赢了，我们会认为控制输赢的参数使得“赢”的概率很高，从而预测自己也会赢。似然函数就是量化这种“参数解释数据”的合理性。

## 从乘法到加法：对数似然

直接对似然函数 `L(θ)` 进行最大化求解非常困难，因为它涉及大量项的连乘。

为了解决这个问题，我们使用一个数学技巧：取对数。因为对数运算可以将乘法转换为加法：
```
log(a * b) = log(a) + log(b)
```

我们将似然函数取自然对数，得到**对数似然函数** `ℓ(θ)`：
```
ℓ(θ) = log L(θ) = ∑_{i=1}^{m} log P(y^{(i)} | x^{(i)}; θ)
```

这里有一个关键点：我们关心的是使函数值最大的参数点（**极值点**），而不是函数的最大值本身。对函数取对数会改变函数的极值，但不会改变极值点的位置。因此，最大化 `L(θ)` 与最大化 `ℓ(θ)` 是等价的，但后者（一个求和问题）在数学上容易处理得多。

通过取对数，我们成功地将一个复杂的连乘问题转化为了一个相对简单的求和问题，为后续的参数求解铺平了道路。
