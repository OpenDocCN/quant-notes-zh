# 计算机毕业设计：P1：项目概述与核心技术栈

在本节课中，我们将要学习一个结合了Python、大语言模型与股票数据分析的毕业设计项目。该项目旨在通过现代技术手段，对股票数据进行获取、分析、预测与可视化，并构建一个基础的股票推荐系统。

## 项目简介

本项目是一个综合性的计算机毕业设计，主题为“基于Python与大语言模型的股票数据分析与预测系统”。它涵盖了从数据获取到智能推荐的完整流程，适合作为大数据、人工智能或金融科技方向的毕业设计课题。

上一节我们介绍了项目的整体面貌，本节中我们来看看项目所涉及的核心技术栈。

## 核心技术栈

![](img/401171fad2c95e2e7e775f7032c13818_0.png)

以下是构建本系统所需的主要技术与工具：

*   **编程语言**：**Python**。因其丰富的数据科学库和简洁的语法，成为本项目的首选。
*   **数据处理与分析**：
    *   **Pandas**：用于数据清洗、转换与分析的核心库。
    *   **NumPy**：提供高效的数值计算功能。
*   **数据可视化**：
    *   **Matplotlib**：基础绘图库，用于创建静态图表。
    *   **Seaborn**：基于Matplotlib，提供更美观的统计图形。
    *   **Plotly** / **PyEcharts**：用于创建交互式图表，增强用户体验。
*   **大语言模型集成**：
    *   **OpenAI API** 或 **国内大模型API**：用于生成分析报告、解读市场情绪或辅助决策。
    *   提示词工程：设计有效的提示（Prompt）来引导大模型完成特定任务，例如：
        ```python
        prompt = f"""
        根据以下股票{stock_code}在日期{date}的数据：
        开盘价：{open_price}，收盘价：{close_price}，最高价：{high_price}，最低价：{low_price}，成交量：{volume}。
        请用简短的语言分析当日市场多空力量对比。
        """
        ```
*   **机器学习/量化分析**：
    *   **Scikit-learn**：提供经典的机器学习算法，用于价格预测或分类。
    *   **预测模型示例**：可以使用线性回归、时间序列模型（如ARIMA）或更复杂的模型进行尝试。一个简单的线性回归公式表示为：
        **`收盘价 ≈ w1 * 开盘价 + w2 * 成交量 + b`**
        其中 `w1`, `w2` 为权重，`b` 为偏置项。
*   **数据存储**：
    *   **MySQL** / **PostgreSQL**：用于存储结构化的股票历史数据、用户信息等。
    *   **CSV/JSON文件**：用于临时存储或备份数据。
*   **Web框架（可选，用于展示系统）**：
    *   **Flask** 或 **Django**：用于构建展示数据分析结果和推荐列表的Web应用程序。

## 核心功能模块

在了解了技术栈之后，我们进一步分解系统的核心功能模块。

以下是系统计划实现的几个关键功能模块：

![](img/401171fad2c95e2e7e775f7032c13818_2.png)

1.  **股票数据获取与处理模块**
    *   从公开API（如Tushare、AkShare）或财经网站获取历史与实时数据。
    *   使用Pandas进行数据清洗（处理缺失值、异常值）和格式化。

2.  **股票数据分析与可视化模块**
    *   计算关键技术指标（如移动平均线MA、相对强弱指数RSI）。
    *   通过Matplotlib/Plotly绘制K线图、成交量图、指标趋势图。

3.  **大模型智能分析模块**
    *   将处理好的数据或摘要输入大语言模型。
    *   获取模型对股票走势的文本分析、风险提示或新闻舆情解读。

4.  **股票预测与推荐模块**
    *   基于历史数据训练预测模型，预测未来价格走势（分类：涨/跌）。
    *   结合量化策略（如基于指标的金叉死叉）和大模型分析，生成股票推荐列表。

![](img/401171fad2c95e2e7e775f7032c13818_4.png)

5.  **系统展示与交互模块（毕业设计演示重点）**
    *   构建一个简单的Web界面或本地图形界面，展示数据图表、分析报告和推荐结果。
