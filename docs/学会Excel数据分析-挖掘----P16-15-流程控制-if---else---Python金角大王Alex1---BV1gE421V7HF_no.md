# Python数据分析入门：P16：流程控制-if...else

在本节课中，我们将要学习Python中一个非常核心的概念——流程控制。通过`if...else`语句，我们可以让程序根据不同的条件做出不同的决策，从而编写出更智能、更有趣的程序。

![](img/a2dd0e064b17b50171021bb2490625dc_1.png)

## 什么是流程控制？

上一节我们介绍了变量和数据类型，它们是程序的基础。本节中我们来看看如何让程序“思考”和“选择”。

想象你站在一个岔路口，面前有两条路。你会根据一些条件（比如哪条路更平坦、风景更好）来决定走哪一条。程序中的流程控制与此类似：我们预先定义好条件，程序会判断条件是否满足，然后执行对应的代码路径。这个过程就叫做流程控制。

在几乎所有编程语言中，流程控制的核心语法都是`if...else`。`if`是“如果”的意思，`else`是“否则”的意思。根据路径的多少，流程控制可以分为单分支、双分支和多分支。

## 单分支流程控制

单分支是最简单的形式，它只有一条路径，程序可以选择“走”或“不走”。

以下是单分支`if`语句的基本语法：

```python
if 条件:
    条件成立时要执行的代码
```

**注意**：
1.  条件后面必须有一个冒号 `:`。
2.  条件成立后要执行的代码必须**缩进**（通常是4个空格或一个Tab键）。缩进是Python用来标识代码块归属关系的方式。

![](img/a2dd0e064b17b50171021bb2490625dc_3.png)

让我们看一个具体的例子：

```python
salary = 8000
if salary < 10000:
    print("You make too less money and ugly, so you cannot find a girlfriend.")
```

在这段代码中：
*   我们定义了一个变量 `salary` 并赋值为8000。
*   `if` 语句判断 `salary < 10000` 这个条件是否成立。
*   因为8000确实小于10000，条件成立，所以会执行下面缩进的 `print` 语句。

如果将 `salary` 改为18000，条件 `18000 < 10000` 不成立，那么缩进的 `print` 语句就不会执行。

无论条件是否成立，没有缩进的代码都会照常执行：

```python
salary = 18000
if salary < 10000:
    print("条件成立时执行的语句")
print("这句话无论条件如何都会执行")
```

## 双分支流程控制

双分支提供了两条明确的路径：条件成立走一条，不成立走另一条。

以下是双分支`if...else`语句的基本语法：

```python
if 条件:
    条件成立时要执行的代码
else:
    条件不成立时要执行的代码
```

让我们修改之前的例子：

```python
salary = 8000
if salary < 10000:
    print("You make too less money and ugly, so you cannot find a girlfriend.")
else:
    print("Not bad! Now you can go out and find some pretty girls.")
```

在这段代码中：
*   如果 `salary < 10000` 成立，则执行第一个 `print` 语句。
*   否则（即 `salary >= 10000`），则执行 `else` 后面的 `print` 语句。

这样，程序必定会执行两个代码块中的一个，不会出现两者都不执行的情况。

## 多分支流程控制

当我们需要判断多种情况时，可以使用多分支结构 `if...elif...else`。

以下是多分支语句的基本语法：

```python
if 条件1:
    满足条件1时执行的代码
elif 条件2:
    满足条件2时执行的代码
elif 条件3:
    满足条件3时执行的代码
else:
    以上条件都不满足时执行的代码
```

`elif` 是 `else if` 的缩写，意思是“否则如果”。程序会从上到下依次判断条件，一旦某个条件成立，就执行对应的代码块，然后跳出整个判断结构。

让我们看一个判断成绩等级的例子：

```python
score = 85
if score >= 90:
    print("成绩优秀")
elif score >= 80:
    print("成绩良好")
elif score >= 60:
    print("成绩及格")
else:
    print("成绩不及格")
```

在这个例子中，程序会先判断 `score >= 90`，不成立；接着判断 `score >= 80`，成立，于是打印“成绩良好”，后面的 `elif` 和 `else` 就不再判断了。

## 总结

本节课中我们一起学习了Python流程控制的核心——`if...else`语句。

我们掌握了：
1.  **单分支 (`if`)**：用于条件成立时执行特定操作。
2.  **双分支 (`if...else`)**：用于在两种可能的情况中选择其一执行。
3.  **多分支 (`if...elif...else`)**：用于在多种可能的情况中选择其一执行。

![](img/a2dd0e064b17b50171021bb2490625dc_5.png)

关键要点包括：条件后的冒号 `:` 必不可少，通过**缩进**来定义代码块，以及 `elif` 的使用逻辑。掌握了这些，你的程序就具备了基本的“决策”能力，可以开始处理更复杂的逻辑了。
