## 高盛量化交易课程笔记：1-1：专业人士与零售交易员（上）🎯

在本节课中，我们将要学习专业交易员与零售交易员在思维方式和行为模式上的核心区别。理解这些差异是建立成功交易体系的第一步。

### 思维模式的差异

专业人士将交易视为一门基于概率和风险管理的生意。他们依靠系统化的方法和严格的纪律。零售交易员则常常受情绪驱动，将交易视为快速致富的途径，决策缺乏一致性。

### 决策依据的不同

专业人士的决策基于深入的数据分析和预设的交易计划。他们使用**赔率计算**来评估每一次交易机会。零售交易员更依赖直觉、小道消息或市场情绪，决策过程随意。

> 核心观点：专业交易是重复执行具有数学优势的策略，而非追逐单次交易的输赢。

## 高盛量化交易课程笔记：1-2：专业人士与零售交易员（下）⚖️

上一节我们介绍了专业与零售交易员在思维模式上的差异，本节中我们来看看他们在具体行为和实践上的不同。

### 风险管理实践

以下是专业人士与零售交易员在风险管理上的关键区别：

专业人士在每笔交易前就设定好明确的止损点和盈利目标。他们严格控制单笔交易的风险敞口，通常不超过总资本的1%-2%。零售交易员经常没有止损计划，或在亏损时选择“扛单”，导致小额亏损演变成重大损失。

### 资金与头寸管理

专业人士使用科学的资金管理策略，如凯利公式或其变体，来动态调整头寸大小。公式可以简化为：

`f* = (bp - q) / b`

其中：
*   `f*` 是最佳下注比例
*   `b` 是赔率（盈利/亏损）
*   `p` 是获胜概率
*   `q` 是失败概率 (1-p)

零售交易员往往全仓进出，或根据感觉随意调整仓位，缺乏系统性。

### 持续学习与复盘

专业人士将交易记录视为宝贵数据，定期复盘分析，优化策略。零售交易员很少记录交易，更倾向于将亏损归咎于市场或运气，而非从自身寻找原因。

## 高盛量化交易课程笔记：2-1：分布与赔率计算（上）🎲

在理解了交易员的思维差异后，我们需要掌握量化决策的基础：概率分布与赔率计算。本节课将介绍如何用数学眼光看待市场波动。

### 理解价格分布

市场价格变动并非随机游走，而是在一定时期内呈现出可被描述的统计分布。常见的如正态分布（钟形曲线），但金融市场常呈现“尖峰厚尾”特征，即极端事件发生的概率比正态分布预测的更高。

### 计算预期价值

交易的核心是计算一笔交易的预期价值（Expected Value, EV）。如果EV为正，长期执行则有利可图。计算公式为：

`EV = (获胜概率 * 平均盈利) - (失败概率 * 平均亏损)`

例如，一次交易有60%概率盈利100元，40%概率亏损80元，则：
`EV = (0.6 * 100) - (0.4 * 80) = 60 - 32 = +28元`

这是一个正期望值的交易。

## 高盛量化交易课程笔记：2-2：分布与赔率计算（下）📊

上一节我们介绍了预期价值的基本概念，本节将深入探讨如何结合赔率来优化决策，并理解概率在实战中的应用。

### 赔率与概率的结合

赔率（Odds）指的是潜在盈利与潜在亏损的比值。好的交易机会不仅需要高胜率，也需要有利的赔率。一个胜率40%但赔率为3:1（赚3块赔1块）的策略，其EV可能高于一个胜率60%但赔率为1:1的策略。

计算：`EV = (0.4 * 3) - (0.6 * 1) = 1.2 - 0.6 = +0.6`

### 应用实例：止损与止盈的设置

设置止损和止盈的本质是主动定义一笔交易的赔率。例如，你愿意承担50点的风险去博取150点的利润，那么你设定的赔率就是 **150:50 = 3:1**。接下来，你需要评估市场条件达成150点利润的概率是否值得这个赔率。

### 样本空间与统计显著性

单次交易的结果具有很大偶然性。专业人士关注的是在大样本（多次交易）下策略的统计表现。不要因为连续几次亏损就否定一个正EV的策略，也不要因为几次盈利就迷信一个有缺陷的方法。

## 高盛量化交易课程笔记：3-1：波动性评估（上）🌊

在能够计算赔率之后，我们必须学会评估市场的波动性，因为波动性直接决定了风险的大小和止损位置的合理性。本节课开始学习波动性的含义与度量。

### 什么是波动性

波动性衡量的是资产价格在一定时期内的变动幅度和频率。高波动性意味着价格在短时间内剧烈变动，机会多但风险也大；低波动性则相反。

### 历史波动率

历史波动率是通过计算过去一段时间内价格回报率的标准差来得到的。它是波动性最直接的度量。在Python中，可以使用`pandas`库快速计算：

```python
import pandas as pd
import numpy as np

# 假设`prices`是一个包含价格序列的Pandas Series
returns = prices.pct_change() # 计算日收益率
historical_volatility = returns.std() * np.sqrt(252) # 年化波动率（假设252个交易日）
```

## 高盛量化交易课程笔记：3-2：波动性评估（中）📈

上一节我们学习了历史波动率，本节将介绍另一种重要的波动率——隐含波动率，并探讨波动率在实战中的意义。

### 隐含波动率

隐含波动率不是从历史价格中计算出来的，而是从期权市场的当前价格中反推出来的。它代表了市场对未来一段时间内资产价格波动幅度的预期。通常使用**Black-Scholes模型**反向求解得出。

> 隐含波动率是市场情绪的“温度计”，高隐含波动率往往预示着市场不确定性或恐慌情绪加剧。

### 波动率与仓位管理

波动性评估直接影响仓位大小。市场波动性高时，应降低仓位以控制单笔交易的绝对风险；市场波动性低时，可以适当增加仓位以捕捉趋势。许多专业交易员使用基于波动性的仓位模型，例如，让每笔交易的风险敞口（止损额）等于账户净值的固定比例。

## 高盛量化交易课程笔记：3-3：波动性评估（下）🛡️

在了解了波动性的度量和意义后，本节我们将学习如何利用波动性来设置更科学的止损，并介绍波动性聚类现象。

### 基于波动性的止损

传统的固定点数止损可能在高波动市场中被轻易触发，或在低波动市场中显得过于宽松。基于平均真实波幅（ATR）的止损是更自适应的方法。ATR反映了近期市场的平均波动范围。

止损距离可以设置为：`止损价格 = 入场价格 - N * ATR`
其中N是一个乘数（如1.5或2），根据你的风险容忍度和策略测试结果来调整。

### 波动性聚类

金融时间序列中一个著名现象是“波动性聚类”，即高波动时期之后往往跟着高波动时期，低波动时期之后往往跟着低波动时期。这意味着风险不是均匀分布的，在经历大幅波动后，应警惕后续可能继续出现大波动，并相应调整风控措施。

## 高盛量化交易课程笔记：4-1：愿望框架与获得洞察力（上）💡

掌握了概率、赔率和波动性这些基础工具后，我们需要一个框架来系统地生成和筛选交易想法。本节介绍“愿望框架”，它帮助我们将市场观察转化为可交易的洞察。

### 什么是愿望框架

愿望框架是一个结构化的思维流程，用于识别市场中可能存在的、可被利用的规律性或低效性。它要求交易员主动提出一个关于市场行为的“愿望”或假设，然后去寻找证据验证或证伪它。

### 框架的第一步：观察与假设

从市场数据、新闻、宏观经济事件或图表形态中观察到一个有趣的现象。例如：“我注意到每次美联储利率决议后，美元指数在最初波动后，倾向于在接下来24小时内向某个方向持续运动。” 这就是一个初始假设。

## 高盛量化交易课程笔记：4-2：愿望框架与获得洞察力（下）🔍

上一节我们提出了交易假设，本节将完成愿望框架的后续步骤：收集数据、验证假设并将其转化为具体的交易规则。

### 数据收集与回溯测试

提出假设后，需要收集历史数据来检验。以上一节的美元指数为例，你需要：
1.  获取历史上所有美联储决议的日期和时间。
2.  获取决议后24小时内美元指数的分钟或小时数据。
3.  定义一个具体的“持续运动”规则（例如，决议后第4根小时线收盘价是否高于第1根小时线收盘价）。
4.  统计所有历史案例中，该规则成立的次数和比例。

这个过程就是简单的回溯测试，可以用代码实现：

```python
# 伪代码示例
win_count = 0
total_events = len(fed_dates)
for date in fed_dates:
    data = get_price_data_after(date, hours=24)
    if data[‘close’][3] > data[‘close’][0]: # 简化规则
        win_count += 1
win_rate = win_count / total_events
```

### 形成交易洞察与规则

如果回溯测试显示该现象具有统计显著性（例如胜率显著高于50%），且具备合理的赔率，那么它就从一个模糊的“愿望”变成了一个潜在的“交易洞察”。最后，需要将这个洞察具体化为可执行的交易规则，包括：
*   **触发条件**：何时入场（例如，决议后第1根小时线收盘）。
*   **风险管理**：止损和止盈如何设置（例如，基于决议日ATR的2倍）。
*   **仓位大小**：根据波动性和账户风险计算。

## 高盛量化交易课程笔记：5-1：相关指标（上）📉

在构建交易系统时，我们需要借助各种技术指标和宏观指标来量化市场状态、识别趋势和动量。从本节开始，我们将用多讲内容系统介绍一系列关键的相关指标。

### 趋势跟踪指标

趋势是你的朋友。以下是一些核心的趋势跟踪指标：

*   **移动平均线**：包括简单移动平均线（SMA）和指数移动平均线（EMA）。EMA对近期价格赋予更高权重，反应更灵敏。常用策略包括双均线交叉（如快线EMA(12)上穿慢线EMA(26)作为买入信号）。
*   **MACD**：由快线（DIF）、慢线（DEA）和柱状图（Histogram）组成。它用于识别趋势的强度、方向和转折点。当快线从下往上穿越慢线时，是潜在的看涨信号。

### 动量指标

动量指标衡量价格变化的速度和强度。

*   **相对强弱指数**：衡量近期价格涨跌幅度的对比，判断市场是否处于超买（通常>70）或超卖（通常<30）状态。公式基于一定周期内上涨日和下跌日的平均涨幅和跌幅计算得出。

## 高盛量化交易课程笔记：6-1：扼要概述与自上而下训练（上）🔄

在学习了大量指标和概念后，我们需要进行一次阶段性的总结，并引入“自上而下”的分析方法，将宏观、行业与个股（或币种）分析结合起来。

### 到目前为止我们学到了什么

本课程前半部分构建了一个量化交易员的思维和工具体系：
1.  **思维模式**：像专业人士一样思考，注重概率、期望值和风险管理。
2.  **核心工具**：理解分布、计算赔率（EV）、评估波动性（历史与隐含波动率、ATR）。
3.  **方法论**：运用“愿望框架”将市场观察转化为可测试的交易洞察。
4.  **技术武器**：掌握趋势、动量、波动性、成交量等多类指标，用于量化市场状态。

### 引入自上而下分析法

自上而下分析法要求我们先分析整体市场环境（大盘指数、宏观经济），再筛选强势行业或板块，最后在其中选择最具潜力的个别资产进行交易。这种方法能确保你的交易方向与更大的资金流向一致，提高胜算。

## 高盛量化交易课程笔记：6-2：自上而下训练与估值补充（下）🏛️

上一节我们介绍了自上而下分析的框架，本节将深入其具体步骤，并补充一个重要的估值工具——市盈率与增长比率。

### 自上而下分析的三层结构

以下是实施自上而下分析的具体步骤：

1.  **宏观经济与市场层面**：分析利率政策、通胀数据、GDP增长、地缘政治等。判断当前市场整体处于风险偏好（Risk-On）还是风险规避（Risk-Off）模式。这决定了你应该倾向于买入风险资产还是持有避险资产。
2.  **行业/板块层面**：在确定市场大方向后，寻找受益于当前宏观环境的行业。例如，在低利率和经济增长期，科技、消费 discretionary 板块可能表现突出。可以使用相对强度分析，比较各板块指数与大盘指数的表现。
3.  **个股/具体资产层面**：在强势板块中，运用之前学到的技术分析、指标和愿望框架，筛选出图形最好、动量最强或估值最合理的具体标的进行交易。

### 补充：市盈率与增长比率

PEG比率是市盈率（PE）相对于盈利增长速度的比值，用于评估成长型公司的估值是否合理。公式为：

`PEG比率 = 市盈率 / 盈利增长率`

例如，一家公司市盈率30倍，预期年盈利增长率20%，则PEG = 30 / 20 = 1.5。通常PEG < 1被认为估值可能被低估，但需结合行业和公司特性判断。

## 高盛量化交易课程笔记：7-1：资本配置与风险管理（上）🚦

拥有了分析工具和框架后，如何分配有限的资本并管理风险，就成为决定长期盈利的关键。本节开始，我们将深入探讨资本配置的“把关”过程。

### 资本配置的原则

资本配置决定了你将多少资金分配给不同的策略、资产类别或具体的交易机会。核心原则是**分散化**和**风险均衡**。
*   **分散化**：不要把所有的资金都投入单一资产或高度相关的资产中。分散可以降低非系统性风险。
*   **风险均衡**：确保你的投资组合中，每个部分所贡献的整体风险是大致均衡的，而不是简单地均衡资金金额。例如，高波动性资产的仓位应低于低波动性资产，以使它们的风险贡献相近。

### 机会筛选漏斗

不是每一个看似不错的机会都值得交易。你需要一个严格的筛选流程：
1.  **第一层：宏观与市场环境** 是否符合你的“自上而下”分析结论？如果不符合，直接排除。
2.  **第二层：策略契合度** 该机会是否符合你已验证的某个交易策略的所有入场条件？
3.  **第三层：风险回报比** 计算的潜在盈利/潜在亏损（赔率）是否达到你的最低要求（例如 > 2:1）？
4.  **第四层：绝对风险** 根据波动性（如ATR）设置的止损，导致的单笔交易最大亏损是否在账户风险预算内（如 < 2%）？

## 高盛量化交易课程笔记：7-2：资本配置与风险管理（中）⚙️

上一节我们介绍了资本配置的原则和筛选漏斗，本节将探讨具体的仓位 sizing 模型和风险管理中的“纪律”部分。

### 仓位计算模型

确定交易机会后，需要精确计算投入的仓位。最常用的方法是基于风险的仓位计算：

`仓位大小 = （账户总风险金额） / （每股或每单位风险）`

其中：
*   **账户总风险金额** = 账户净值 * 单笔交易最大风险比例（例如1%）。
*   **每股/每单位风险** = 入场价格 - 止损价格。

例如，账户有10万元，单笔风险设为1%，即1000元。你计划在股价100元买入，止损设在95元，则每股风险5元。可买入股数为：`1000 / 5 = 200股`。总投资额为 `200 * 100 = 20,000元`。

### 纪律的重要性

风险管理中最难的部分是执行纪律。这包括：
*   **严格执行止损**：止损单一旦设定，就必须视为不可撤销的命令。避免因“再等等看”的心理而移动止损。
*   **遵守仓位规则**：不要在情绪激动时（连续盈利后过度自信，或连续亏损后急于翻本）擅自加大仓位。
*   **定期复盘**：每周或每月检查所有交易记录，分析亏损交易是源于策略失效还是纪律松懈。

## 高盛量化交易课程笔记：7-3：风险管理（下）与交易计划🎯

在课程的最后部分，我们将整合所有知识，讨论日内交易的特殊性，并最终形成一份可操作的交易计划。

### 日内交易注意事项

日内交易持仓时间短，需要更专注的盘面观察和更快的反应速度。关键点包括：
*   **流动性**：只交易高流动性的品种，确保能快速进出。
*   **时间框架**：主要使用分钟图（如5分钟、15分钟）进行分析。
*   **新闻风险**：避开重大宏观经济数据发布前后等波动极不稳定的时段。
*   **当日止损**：除了单笔交易止损，还应设置“当日最大亏损限额”，一旦达到立即停止当天所有交易。

### 制定你的交易计划

一份完整的交易计划是你的作战手册，应包含以下部分：

1.  **交易哲学与目标**：简述你的核心交易理念和合理的年度/月度收益目标。
2.  **市场与时间**：你交易哪些市场（股票、外汇、加密货币等）？在什么时间段交易？
3.  **策略概要**：清晰描述你使用的1-3个主要交易策略（例如：基于EMA交叉的趋势跟踪策略）。
4.  **入场条件**：每个策略的明确、无歧义的入场信号（需包含指标参数）。
5.  **风险管理**：
    *   单笔交易风险比例（如1%）。
    *   止损设置规则（如基于ATR的2倍）。
    *   止盈设置规则（如风险回报比2:1）。
    *   每日/每周最大亏损限额。
6.  **仓位计算**：明确使用何种模型计算仓位（基于风险的模型）。
7.  **交易日志与复盘**：规定如何记录交易，以及复盘的频率和重点。
