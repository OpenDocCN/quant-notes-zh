# 金融量化：P32：6-中性化处理方法通俗解释

在本节课中，我们将要学习量化交易中的一个重要概念——**中性化**。我们将通过通俗的例子解释其核心目的，并了解其计算方法，为后续的实战应用打下基础。

## 概述：什么是中性化？

中性化的核心目的是**提纯**。它旨在从因子中提取出独特、有价值的信息，剔除与其他因子（如市值）高度相关的共性部分，从而避免选股结果出现我们不希望看到的倾向性。

## 中性化的通俗解释

上一节我们概述了中性化的目的，本节中我们来看看一个具体的例子，以便更好地理解。

假设我们设计了一个选股策略，其中使用了四个不同的因子（A、B、C、D）。理论上，这四个因子应从不同角度帮助我们筛选股票。

然而，在实际操作中，无论我们如何调整这四个因子的权重或组合，最终选出的股票池总是高度相似。这是为什么呢？

原因可能在于，这四个因子虽然名义上不同，但其内部绝大部分信息都与同一个因素——例如**市值**——高度相关。这就好比：
*   因子A（如市净率）绝大部分受市值影响。
*   因子B、C、D也绝大部分受市值影响。

在这种情况下，最终的选股结果几乎完全由“市值”这一个因素主导，其他因子自身的独特信息被淹没，无法发挥其应有的作用。这导致策略缺乏多样性，选股结果过于集中。

那么，什么是“提纯”呢？就是**从每个因子中，剔除掉与市值（或其他共性因素）相关的部分，只保留该因子自身独特、有价值的信息**。这个过程就是**中性化**。它让我们能更清晰地观察每个因子的独立贡献。

## 为何需要进行中性化？

以下是进行中性化处理的主要原因：

*   **避免选股倾向性**：如果不进行中性化，选股结果可能会过度集中于受某个共性因素（如市值）影响的股票，无法体现策略的多样性。
*   **提取因子真实信息**：中性化能剥离因子的共性噪音，让我们更准确地评估该因子本身对股票收益的预测能力。
*   **提升策略有效性**：使用提纯后的因子构建策略，理论上能获得更稳定、更独立的Alpha（超额收益）来源。

## 一个具体因子：市净率

![](img/83365015cfc3713b2ecbff90d6a2b037_1.png)

在后续的实战中，我们将以**市净率**为例进行中性化操作。这里先简要介绍这个因子。

**市净率**的计算公式为：
`市净率(PB) = 每股股价 / 每股净资产`
其中，每股净资产 = (公司总资产 - 总负债) / 总股数，代表公司账面上属于股东们的纯净价值。

一般来说，较低的市净率可能意味着股票被低估，具有更高的投资安全边际和潜在上涨空间，被认为是更具投资价值的信号。

## 中性化的计算方法

上一节我们了解了为何要进行中性化，本节中我们来看看其核心的计算思路。在量化实践中，对因子（如市净率）进行市值中性化，通常采用以下回归方法：

1.  **建立回归模型**：将被中性化的因子（如市净率）作为因变量 `Y`，将需要剔除的共性因素（如市值的对数）作为自变量 `X`，建立线性回归模型。
    `Y = α + β * X + ε`
    其中，`ε` 代表回归的残差项。

2.  **提取残差**：上述回归模型中的残差 `ε`，即因子 `Y` 中无法被 `X`（市值）解释的部分，就是我们需要的**提纯后**的中性化因子值。
    `中性化因子值 = ε = Y - (α + β * X)`

这个残差序列已经剥离了市值的影响，保留了市净率因子独立于市值的那部分特有信息，可以用于后续的选股策略。
