## 数据清洗的必要性

![](img/6730e818186044c641d9190016671774_9.png)

![](img/6730e818186044c641d9190016671774_11.png)

上一节我们介绍了数据分析的基本流程，本节中我们来看看为何需要进行数据清洗。在进行数据分析时，原始数据可能存在各种问题，这些问题会影响分析结果的准确性。

![](img/6730e818186044c641d9190016671774_13.png)

![](img/6730e818186044c641d9190016671774_15.png)

![](img/6730e818186044c641d9190016671774_17.png)

以下是数据清洗主要处理的三种问题：
1.  **缺失值**：指数据中的空值（NaN或None）。在某些分析需求下，这些空值没有意义，反而会干扰分析。
2.  **重复值**：指数据中完全相同的记录。重复值通常没有必要进行多次分析，需要过滤掉。
3.  **异常值**：指由于数据采集手段不同而产生的、明显偏离正常范围的数据点。异常值同样会干扰分析结论。

![](img/6730e818186044c641d9190016671774_19.png)

因此，在pandas数据清洗过程中，我们需要针对这三种情况进行处理。

![](img/6730e818186044c641d9190016671774_21.png)

## 缺失值的两种形式：NaN与None

![](img/6730e818186044c641d9190016671774_23.png)

在开始清洗缺失值之前，我们需要理解Python中两种“空”的区别：`NaN`（Not a Number）和 `None`。

![](img/6730e818186044c641d9190016671774_25.png)

![](img/6730e818186044c641d9190016671774_27.png)

`None`是一个Python对象，其类型为`NoneType`。
```python
type(None)  # 输出：NoneType
```

![](img/6730e818186044c641d9190016671774_29.png)

![](img/6730e818186044c641d9190016671774_31.png)

而在数据分析中，我们通常使用`numpy`模块提供的`NaN`，它是一个特殊的浮点数。
```python
import numpy as np
type(np.nan)  # 输出：float
```

![](img/6730e818186044c641d9190016671774_33.png)

![](img/6730e818186044c641d9190016671774_35.png)

![](img/6730e818186044c641d9190016671774_37.png)

为什么数据分析中要使用`NaN`而不是`None`呢？关键在于它们参与运算的能力不同。

![](img/6730e818186044c641d9190016671774_39.png)

![](img/6730e818186044c641d9190016671774_41.png)

`None`是对象类型，无法直接参与算术运算。
```python
None + 1  # 会引发 TypeError
```

![](img/6730e818186044c641d9190016671774_43.png)

而`NaN`是浮点类型，可以参与运算，但结果依然是`NaN`。
```python
np.nan + 1  # 输出：nan
```

![](img/6730e818186044c641d9190016671774_45.png)

在数据分析中，常常需要对原始数据进行各种运算处理。如果原始数据中的空值是`NaN`形式，则不会中断整个运算流程；如果是`None`形式，则可能在运算中引发错误。因此，数据分析中应使用`NaN`形式的空值。

![](img/6730e818186044c641d9190016671774_47.png)

![](img/6730e818186044c641d9190016671774_49.png)

**一个重要的特性是**：在pandas中，如果遇到了`None`形式的空值，pandas会自动将其强制转换为`NaN`形式。尽管如此，理解这两种空的本质区别仍然非常重要。

![](img/6730e818186044c641d9190016671774_51.png)

现在，我们来看如何在pandas中处理数据中的缺失值。我们将通过一个包含缺失值的示例数据集来演示。

![](img/6730e818186044c641d9190016671774_61.png)

首先，导入必要的模块并创建一组包含缺失值的示例数据。
```python
import pandas as pd
import numpy as np
from pandas import DataFrame, Series

![](img/6730e818186044c641d9190016671774_63.png)

![](img/6730e818186044c641d9190016671774_65.png)

# 创建一个8行6列的随机整数DataFrame
df = DataFrame(np.random.randint(0, 100, size=(8, 6)))
# 人为制造几个缺失值
df.iloc[2, 3] = None        # 第2行，第3列
df.iloc[4, 4] = np.nan      # 第4行，第4列
df.iloc[5, 2] = None        # 第5行，第2列
print(df)
```
运行后，你会看到数据中有三个位置显示为`NaN`，这就是我们需要处理的缺失值。

在DataFrame中，处理缺失值通常意味着删除包含缺失值的整行数据（因为删除单列可能会改变数据结构）。以下是两种主要的处理方法。

![](img/6730e818186044c641d9190016671774_67.png)

![](img/6730e818186044c641d9190016671774_69.png)

### 方法一：使用 `isnull()`/`notnull()` 结合 `any()`/`all()` 进行过滤

![](img/6730e818186044c641d9190016671774_71.png)

![](img/6730e818186044c641d9190016671774_73.png)

这种方法的核心是定位哪些行包含缺失值，然后将其删除。

![](img/6730e818186044c641d9190016671774_75.png)

![](img/6730e818186044c641d9190016671774_77.png)

![](img/6730e818186044c641d9190016671774_79.png)

`df.isnull()` 会返回一个与原数据形状相同的布尔型DataFrame，其中`True`表示该位置是缺失值。
`df.notnull()` 则与之相反，`False`表示缺失值。

![](img/6730e818186044c641d9190016671774_81.png)

![](img/6730e818186044c641d9190016671774_83.png)

![](img/6730e818186044c641d9190016671774_85.png)

我们需要结合`any(axis=1)`或`all(axis=1)`来在行方向上判断。
*   `any(axis=1)`：如果一行中存在任何一个`True`，则返回`True`。
*   `all(axis=1)`：如果一行中所有值都是`True`，则返回`True`。

![](img/6730e818186044c641d9190016671774_87.png)

![](img/6730e818186044c641d9190016671774_89.png)

以下是操作步骤：
1.  **使用 `isnull().any(axis=1)`**：找出哪些行至少包含一个缺失值。
    ```python
    # 获取一个布尔序列，True表示该行包含缺失值
    condition = df.isnull().any(axis=1)
    print(condition)
    ```
2.  **使用布尔索引过滤行**：将上一步得到的布尔序列作为行索引，即可筛选出不含缺失值的行。
    ```python
    # 直接筛选，保留 condition 为 False 的行（即不含缺失值的行）
    cleaned_df = df.loc[~condition]
    print(cleaned_df)
    ```
    **固定搭配**：`isnull()` 通常与 `any()` 搭配使用；`notnull()` 通常与 `all()` 搭配使用（`df.notnull().all(axis=1)` 为 `False` 的行表示包含缺失值）。

![](img/6730e818186044c641d9190016671774_91.png)

![](img/6730e818186044c641d9190016671774_93.png)

![](img/6730e818186044c641d9190016671774_95.png)

### 方法二：使用 `dropna()` 函数直接删除

pandas提供了一个更简便的内置函数 `dropna()` 来直接删除包含缺失值的行或列。

![](img/6730e818186044c641d9190016671774_103.png)

### 方法三：使用 `fillna()` 进行填充覆盖

![](img/6730e818186044c641d9190016671774_123.png)

![](img/6730e818186044c641d9190016671774_125.png)

![](img/6730e818186044c641d9190016671774_127.png)

![](img/6730e818186044c641d9190016671774_129.png)

除了删除，另一种处理缺失值的方法是将其填充为其他值。这在删除成本过高（例如，缺失值太多，删除后剩余数据过少）时尤为有用。

使用 `df.fillna()` 函数可以进行填充。最简单的填充是使用一个固定值。
```python
# 将所有缺失值填充为 0
filled_df = df.fillna(value=0)
print(filled_df)
```
然而，用固定值填充往往不够合理。更常见的做法是使用相邻的值进行填充，这可以通过 `method` 参数实现。
```python
# 向前填充：用缺失值前面的那个有效值来填充
filled_df_ffill = df.fillna(method='ffill', axis=0) # 沿列方向（垂直）向前填充
print(filled_df_ffill)

# 向后填充：用缺失值后面的那个有效值来填充
filled_df_bfill = df.fillna(method='bfill', axis=0) # 沿列方向（垂直）向后填充
print(filled_df_bfill)
```
参数说明：
*   `method='ffill'` 或 `'pad'`：向前填充。
*   `method='bfill'` 或 `'backfill'`：向后填充。
*   `axis`：填充方向。`axis=0`（默认）沿列方向（垂直）查找前后值；`axis=1` 沿行方向（水平）查找前后值。

**选择删除还是填充？**
*   **删除**：当缺失值数量较少，删除后对整体数据影响不大时，优先选择删除。
*   **填充**：当缺失值数量较多，删除会导致数据量锐减、成本过高时，考虑使用合理的策略（如前后填充、均值填充等）进行填充。但需注意，任何填充都是一种估算，并非原始真实值。
