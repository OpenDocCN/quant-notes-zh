![](img/318ecf62912b686f41b9254278768882_1.png)

在本节课中，我们将学习IPython中一个非常实用的高级功能——魔术命令。魔术命令以百分号开头，能够直接在交互式环境中执行文件、粘贴代码、测量时间等，极大地提升了开发和测试的效率。

## 魔术命令简介

上一节我们介绍了IPython的基本交互功能，本节中我们来看看IPython提供的“魔术命令”。魔术命令是以 `%` 开头的特殊指令，它们为交互式编程提供了许多便捷的高级功能。

## 运行外部Python文件

![](img/318ecf62912b686f41b9254278768882_3.png)

在标准的Python命令行中，若想运行一个已写好的Python文件，通常需要退出命令行，再执行 `python filename.py` 命令。在IPython中，我们可以使用魔术命令直接运行，无需退出。

![](img/318ecf62912b686f41b9254278768882_5.png)

以下是使用 `%run` 命令运行脚本的步骤：
1.  首先，创建一个简单的Python文件，例如 `hello.py`，内容为 `print(“Hello World”)`。
2.  在IPython交互环境中，输入命令 `%run hello.py`。
3.  该命令会执行 `hello.py` 文件，并输出结果。

**代码示例**：
```python
# 假设 hello.py 文件内容为 print(“Hello World”)
%run hello.py
# 输出：Hello World
```

## 粘贴并执行代码

![](img/318ecf62912b686f41b9254278768882_7.png)

有时我们需要从编辑器或其他地方复制一段代码到IPython中执行。直接粘贴可能会遇到格式问题（如缩进错误）。IPython提供了 `%paste` 魔术命令来解决这个问题。

![](img/318ecf62912b686f41b9254278768882_9.png)

`%paste` 命令会执行系统剪贴板中的代码，并自动处理可能存在的格式问题。这对于测试长代码中的某一部分非常有用。

![](img/318ecf62912b686f41b9254278768882_11.png)

![](img/318ecf62912b686f41b9254278768882_13.png)

**操作步骤**：
1.  在任何文本编辑器（如Sublime Text, VS Code）中复制一段代码。
2.  在IPython中输入 `%paste` 并回车。
3.  IPython会先打印出剪贴板中的代码，然后执行它。

## 精确测量代码运行时间

在性能分析和优化时，我们经常需要测量某段代码的执行时间。虽然可以使用Python的 `time` 模块，但对于运行时间极短的代码，单次测量可能不准确（结果为0秒）。

IPython的 `%timeit` 魔术命令通过多次运行代码并计算平均耗时，能够提供非常精确的时间测量结果，尤其适用于微操作的性能评估。

**使用方法**：
在需要测量的代码行或代码块前加上 `%timeit`。

**代码示例**：
```python
import random

# 创建一个包含1万个随机数的列表
my_list = [random.random() for _ in range(10000)]

# 测量列表排序操作的时间
%timeit my_list.sort()
```
执行后，`%timeit` 会输出类似 `146 µs ± 742 ns per loop` 的结果，表示平均每次循环耗时146微秒，并给出了误差范围。它通过自动调整运行次数（可能达到百万次）来获得稳定、精确的测量值。

![](img/318ecf62912b686f41b9254278768882_15.png)

本节课中我们一起学习了IPython的三个核心魔术命令：`%run` 用于直接运行外部脚本，`%paste` 用于粘贴并执行剪贴板中的代码，`%timeit` 用于精确测量代码段的执行时间。掌握这些命令能显著提升你在IPython环境下的开发与调试效率。
