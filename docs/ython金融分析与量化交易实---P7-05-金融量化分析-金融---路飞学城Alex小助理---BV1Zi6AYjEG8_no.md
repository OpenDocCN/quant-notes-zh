# 量化交易教程：P7：05 金融量化分析-金融分析 📈

在本节课中，我们将要学习金融分析的核心方法。上一节我们介绍了金融和股票的基础知识，本节中我们来看看如何通过系统性的分析来判断股票的买卖时机，而不是盲目操作。

金融分析主要分为两种方法：基本面分析和技术面分析。

## 基本面分析 🏢

基本面分析的核心是评估公司的运营状况，即我们之前提到的“公司自身因素”。它通过分析当前经济环境、行业前景以及公司具体的财务状况，来判断其股票是否值得购买。

以下是基本面分析的三个层面：

1.  **宏观经济面分析**：分析国家的财政政策、货币政策等宏观因素，判断整体经济环境是鼓励资金进入股市还是倾向于储蓄。但需注意，宏观经济规律并非总是与股市表现完全一致。
2.  **行业分析**：评估特定行业（如教育、IT、能源等）的整体发展前景。可以通过观察该行业内几只具有代表性的股票走势来辅助判断。
3.  **公司分析**：这是最具体的一层。上市公司需定期公开经审计的财务报告（财报）。投资者通过分析财报数据（如利润、每股收益等）、关注公司新闻甚至实地考察，来判断公司运营的好坏。如果公司运营良好、盈利能力强，则其股票可能具备投资价值。

## 技术面分析 📊

![](img/a0e2f18ff7374bd4aaaf0182a44d1746_1.png)

技术面分析的核心观点是：**所有信息都已蕴含在市场交易数据中**。它不关注公司内在价值，而是通过分析历史的市场价格和交易量走势，并借助一系列技术指标，来预测未来的价格变动。

技术分析依赖于定义明确的技术指标。以下是两个最基础且重要的技术指标介绍。

### K线图

![](img/a0e2f18ff7374bd4aaaf0182a44d1746_3.png)

![](img/a0e2f18ff7374bd4aaaf0182a44d1746_5.png)

K线图是展示股票每日价格走势的图表。一根K线包含了四个关键价格：开盘价、收盘价、最高价和最低价。

K线分为阳线和阴线：
*   **阳线**（通常为红色或空心）：表示当日股价上涨，即收盘价高于开盘价。
*   **阴线**（通常为绿色或实心）：表示当日股价下跌，即收盘价低于开盘价。

一根标准K线的构成如下：
*   **实体**：中间的矩形部分。实体的**上沿和下沿分别代表开盘价和收盘价**。
*   **影线**：实体上下方的细线。上影线的顶端代表**最高价**，下影线的底端代表**最低价**。

**公式/规则**：
*   对于阳线：`收盘价 > 开盘价`
*   对于阴线：`收盘价 < 开盘价`
*   实体高度 = |收盘价 - 开盘价|
*   上影线长度 = 最高价 - 实体上沿
*   下影线长度 = 实体下沿 - 最低价

特殊形态的K线（如十字星、光头光脚线等）具有不同的市场含义，我们将在后续课程中详细探讨。

### 移动平均线（MA）

移动平均线（Moving Average， MA）是通过计算过去一段时间内股价的平均值，并将每日的点连接起来形成的平滑曲线。它用于反映股价的趋势。

常见的均线有5日均线（MA5）、10日均线（MA10）、60日均线（MA60）等。数字代表计算平均值所取的天数。

![](img/a0e2f18ff7374bd4aaaf0182a44d1746_7.png)

![](img/a0e2f18ff7374bd4aaaf0182a44d1746_9.png)

**计算公式（以N日均线为例）**：
`MA(N) = (P1 + P2 + ... + PN) / N`
其中，`P1` 代表今天的收盘价，`P2` 代表昨天的收盘价，以此类推，`PN` 代表N天前的收盘价。这个计算每天都会向前“移动”一天。

**应用示例**：
在股票软件中，我们常看到不同颜色的均线交织。例如，当短期均线（如MA5）从下向上穿过长期均线（如MA60）时，这可能是一个买入信号（俗称“金叉”），反之则为卖出信号（“死叉”）。这就是著名的“双均线策略”的基础，我们会在量化策略部分深入讲解。
