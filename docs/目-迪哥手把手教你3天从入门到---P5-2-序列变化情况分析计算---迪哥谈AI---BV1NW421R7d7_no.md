在本节课中，我们将学习如何分析时间序列数据的变化情况。具体来说，我们将计算两种关键指标：**每日差异值**和**每日变化率**，并通过统计和可视化来理解数据的整体趋势。这些是量化金融分析中非常基础和重要的步骤。

上一节我们介绍了数据的基本处理，本节中我们来看看如何计算和分析序列的变化。

## 计算每日差异值

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_5.png)

首先，我们需要计算每一天与前一天的指标差异值。这个指标反映了价格的绝对变化量。

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_7.png)

在Pandas中，计算这个差异值非常简单。我们可以使用`.diff()`方法，它会自动用后一天的数据减去前一天的数据。

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

执行上述代码后，会输出前5天的差异值。例如，1月5号的数据等于1月5号的原始值减去1月4号的原始值。由于1月4号之前没有数据，所以第一行的差异值会显示为`NaN`（空值）。

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_9.png)

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_10.png)

`.diff()`方法默认计算相邻一天的差异。我们还可以通过指定`periods`参数来计算间隔多天的差异，例如两天的差异（`periods=2`）。

以下是使用`.diff()`方法时需要注意的几点：
*   第一行数据由于没有前一天的参考，计算结果为`NaN`。
*   可以通过`periods`参数灵活计算不同时间间隔的差异。
*   这个指标反映了价格的绝对变化，但有时相对变化（百分比）更有意义。

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_12.png)

## 计算每日变化率（增长率）

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_14.png)

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_15.png)

差异值告诉我们价格变化了多少，但没有考虑变化的相对幅度。例如，从1元涨到2元（涨了1元）和从10000元涨到10001元（也涨了1元）的意义完全不同。因此，我们引入**变化率**或**增长率**的概念。

变化率的计算公式为：
**变化率 = (后一天的值 - 前一天的值) / 前一天的值**

在Pandas中，我们可以直接使用`.pct_change()`方法来计算这个增长率。

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_17.png)

```python
# 计算每日变化率（增长率）
pct_data = data.pct_change()
print(pct_data.head())
```

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_19.png)

计算出的结果是一个百分比值，它更直观地反映了价格变化的相对强度。通常，股价的日变化率数值较小。

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_20.png)

## 分析变化率的统计特征

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_22.png)

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_23.png)

得到每日变化率后，我们可以进一步分析其统计特征，例如计算所有交易日增长率的平均值，以了解该股票或指标的整体平均日波动情况。

```python
# 计算增长率的平均值
mean_pct = pct_data.mean()
print(mean_pct)
```

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_25.png)

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_26.png)

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_27.png)

## 可视化统计结果

为了更直观地展示各个指标的平均增长率，我们可以将其可视化。由于平均值是单个数值，适合用条形图来展示。

```python
# 绘制平均增长率的条形图
mean_pct.plot(kind='bar', figsize=(16, 8))
```

![](img/0ea82e797fe2885be2a4e5ea0c4504ce_29.png)

通过图表，我们可以一目了然地看出哪个指标的平均日增长率最高或最低，便于进行横向比较。
