在本节课中，我们将要学习如何处理Pandas Series中的缺失值。缺失值是数据分析中常见的问题，它们可能由数据未记录、采集错误等原因造成。正确处理缺失值是保证分析结果准确性的重要步骤。

上一节我们介绍了Series的基本操作，本节中我们来看看如何处理其中的缺失数据。

## 什么是缺失值

在Series中，缺失数据通常表示为`NaN`（Not a Number）值。出现缺失数据时，有时我们可以忽略它，但在进行进一步运算或生成图表时，这些缺失值可能会带来问题，因此需要处理。

![](img/9940b39ad072df9661807d1c050cea9b_3.png)

## 缺失值处理方法

处理缺失值主要有两种思路：删除缺失值或填充缺失值。以下是具体的操作方法。

### 方法一：删除缺失值

![](img/9940b39ad072df9661807d1c050cea9b_5.png)

第一种处理方式是将包含缺失值的行直接删除。首先，我们需要判断哪些行是缺失的。

Pandas提供了`isnull()`函数来判断缺失值。它返回一个布尔型Series，其中`True`表示该位置是缺失值。

![](img/9940b39ad072df9661807d1c050cea9b_7.png)

```python
SR.isnull()
```

与`isnull()`相对的还有一个`notnull()`函数，它返回`True`表示该位置**不是**缺失值。

![](img/9940b39ad072df9661807d1c050cea9b_9.png)

```python
SR.notnull()
```

利用布尔索引，我们可以过滤掉缺失值。

![](img/9940b39ad072df9661807d1c050cea9b_11.png)

```python
SR[SR.notnull()]
```

![](img/9940b39ad072df9661807d1c050cea9b_13.png)

此外，Pandas还提供了一个直接删除缺失值的函数`dropna()`。

```python
SR.dropna()
```

![](img/9940b39ad072df9661807d1c050cea9b_15.png)

这个函数会直接删除所有包含缺失值的行。

### 方法二：填充缺失值

第二种处理方式是为缺失值赋予一个新值。这可以通过`fillna()`函数实现。

例如，我们可以将所有缺失值填充为0。

```python
SR.fillna(0)
```

**请注意**：Pandas的数据处理函数（如`fillna`、`dropna`）默认不会修改原始Series，而是返回一个新的Series。如果你想保存处理后的结果，需要将其赋值给一个新变量。

![](img/9940b39ad072df9661807d1c050cea9b_17.png)

```python
SR_filled = SR.fillna(0)
```

![](img/9940b39ad072df9661807d1c050cea9b_19.png)

除了填充固定值，我们还可以根据实际情况选择更合理的填充方式。例如，将缺失值填充为该Series的平均值，这样可以保持数据的整体趋势。

![](img/9940b39ad072df9661807d1c050cea9b_21.png)

```python
SR.fillna(SR.mean())
```

这里`SR.mean()`会自动忽略缺失值，计算剩余有效数据的平均值，非常方便。

## 总结

本节课中我们一起学习了Pandas Series缺失值的两种核心处理方法：
1.  **删除**：使用`dropna()`函数或通过布尔索引`SR[SR.notnull()]`过滤。
2.  **填充**：使用`fillna()`函数，可以填充固定值（如0），也可以填充统计值（如平均值`SR.mean()`）。

![](img/9940b39ad072df9661807d1c050cea9b_23.png)

理解并熟练运用这些方法，是进行数据清洗、为后续分析准备高质量数据的关键步骤。
