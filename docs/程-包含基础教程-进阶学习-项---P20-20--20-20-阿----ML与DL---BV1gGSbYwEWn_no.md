# 量化交易教程：P20：阿尔法与贝塔概述 📊

在本节课中，我们将要学习量化交易策略评估中的两个核心概念：阿尔法（α）和贝塔（β）。理解这两个指标，有助于我们区分投资收益的来源，并明确量化策略的优化方向。

## 收益的构成

上一节我们介绍了策略评估的多种指标，本节中我们来看看收益的深层构成。投资收益通常由两部分组成。

一部分收益与整体市场环境相关。当市场大环境向好时，大部分投资都能获得收益。这部分收益可以理解为“随波逐流”赚到的钱。

另一部分收益则与市场波动关系不大，主要来源于投资者自身的策略、独特的洞察力或操作手法。例如，通过深入分析公司财务、运营状况，并制定相应策略所获得的收益。

阿尔法和贝塔就是分别用来衡量这两部分收益的指标。

## 阿尔法（α）与贝塔（β）的定义

![](img/05f75b56a486cc74d3bcafd8a2aac1c9_1.png)

以下是阿尔法与贝塔的核心定义：

*   **阿尔法（α）**：衡量**超额收益**。它描述的是投资策略独立于市场波动所带来的回报，反映了策略本身的有效性或投资者的“选股能力”。阿尔法越高，说明策略超越市场基准的表现越好。
    *   公式关联：在资本资产定价模型（CAPM）中，阿尔法是回归方程 `E(Ri) = Rf + βi * [E(Rm) - Rf] + α` 中的截距项。其中 `E(Ri)` 是资产预期收益，`Rf` 是无风险利率，`E(Rm)` 是市场预期收益。

*   **贝塔（β）**：衡量**系统性风险**或市场相关性。它反映了投资策略对大盘波动的敏感程度。贝塔值为1表示策略波动与市场同步；大于1表示波动比市场更剧烈；小于1则表示波动比市场平缓。
    *   公式关联：贝塔是上述CAPM公式中的斜率系数 `βi`，计算公式为 `β = Cov(Ri, Rm) / Var(Rm)`，即资产收益与市场收益的协方差除以市场收益的方差。

## 图解收益分解

我们可以通过收益分解图来直观理解这两个概念。

![](img/05f75b56a486cc74d3bcafd8a2aac1c9_1.png)

如图所示：
*   **基准收益（市场收益）**：代表市场大环境（如沪深300指数）的整体走势带来的收益。这部分收益对应**贝塔**。
*   **策略收益**：代表采用特定策略后获得的总收益。
*   **超额收益**：策略收益减去基准收益后的部分，即策略战胜市场的部分。这部分收益对应**阿尔法**。

因此，总收益可以看作由市场收益（贝塔收益）和超额收益（阿尔法收益）两部分构成。

## 量化策略的关注重点

对于量化交易者而言，市场整体的贝塔收益是我们难以控制和预测的。因此，策略研发的核心目标就聚焦于如何获取稳定且可持续的**阿尔法收益**。

我们的关注点在于：如何通过数据挖掘、模型构建和策略优化，创造出能够独立于市场涨跌、持续产生正向超额收益的阿尔法策略。

## 总结

![](img/05f75b56a486cc74d3bcafd8a2aac1c9_5.png)

本节课中我们一起学习了阿尔法（α）和贝塔（β）的概念。贝塔衡量了策略随市场波动而获得的收益，即系统性风险部分；阿尔法则衡量了策略通过自身能力战胜市场所获得的超额收益。在量化交易实践中，持续寻找和优化阿尔法来源是我们的核心任务。后续在因子策略分析等章节中，我们将深入探讨如何具体计算和运用这些指标。
