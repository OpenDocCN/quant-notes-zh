## 构建回归方程

上一节我们介绍了数据的基本处理，本节中我们来看看如何构建两个金融指标之间的回归方程。构建回归方程的方法很多，许多工具包都可以完成。这里我们使用NumPy库中的`polyfit`函数来计算回归系数。

![](img/e5867d217fe158a260656201e5c0eba3_5.png)

![](img/e5867d217fe158a260656201e5c0eba3_7.png)

`polyfit`函数可以帮助我们计算一个回归模型的系数。它需要传入几个参数：

![](img/e5867d217fe158a260656201e5c0eba3_9.png)

![](img/e5867d217fe158a260656201e5c0eba3_11.png)

*   **第一个参数 (x)**: 自变量数据，即指标X。
*   **第二个参数 (y)**: 因变量数据，即指标Y。
*   **第三个参数 (deg)**: 回归方程的阶数。

![](img/e5867d217fe158a260656201e5c0eba3_13.png)

![](img/e5867d217fe158a260656201e5c0eba3_15.png)

这个参数`deg`决定了回归方程的形式。例如，当`deg=1`时，我们得到的是**一次线性回归方程**，其形式为：
`Y = k * X + b`
这包含了X的一次项（kX）和零次项（常数b）。

![](img/e5867d217fe158a260656201e5c0eba3_17.png)

如果指定`deg=2`，则方程会包含X的二次项、一次项和常数项，可以拟合曲线。但在当前任务中，我们只关注线性关系，因此指定`deg=1`即可。

![](img/e5867d217fe158a260656201e5c0eba3_19.png)

在计算之前，必须确保数据中没有空值（NaN），否则会导致计算错误。我们可以使用Pandas的`dropna`方法进行清理。

![](img/e5867d217fe158a260656201e5c0eba3_21.png)

以下是构建回归方程的核心代码步骤：

```python
# 假设 df 是包含‘X’和‘Y’两列数据的DataFrame
# 1. 清理数据中的空值
df.dropna(inplace=True)

![](img/e5867d217fe158a260656201e5c0eba3_23.png)

![](img/e5867d217fe158a260656201e5c0eba3_25.png)

# 2. 提取自变量X和因变量Y
x_data = df[‘X’].values
y_data = df[‘Y’].values

![](img/e5867d217fe158a260656201e5c0eba3_27.png)

# 3. 使用polyfit计算回归系数 (k, b)
coefficients = np.polyfit(x_data, y_data, deg=1)
k, b = coefficients
```
执行上述代码后，我们就得到了回归方程的系数k和b，从而确定了具体的回归方程。

![](img/e5867d217fe158a260656201e5c0eba3_29.png)

## 可视化回归结果

得到回归方程后，我们可以将其可视化，与原始数据散点图叠加，直观地观察拟合效果。

![](img/e5867d217fe158a260656201e5c0eba3_33.png)

以下是绘制散点图和回归线的步骤：

![](img/e5867d217fe158a260656201e5c0eba3_35.png)

![](img/e5867d217fe158a260656201e5c0eba3_37.png)

首先，我们需要用回归方程计算出对应于每个X值的预测Y值。然后使用Matplotlib进行绘图。

![](img/e5867d217fe158a260656201e5c0eba3_39.png)

```python
import matplotlib.pyplot as plt

![](img/e5867d217fe158a260656201e5c0eba3_41.png)

![](img/e5867d217fe158a260656201e5c0eba3_43.png)

# 1. 绘制原始数据散点图
plt.scatter(x_data, y_data, s=16, label=‘原始数据’)

![](img/e5867d217fe158a260656201e5c0eba3_45.png)

![](img/e5867d217fe158a260656201e5c0eba3_47.png)

# 2. 计算回归线对应的Y值
y_pred = k * x_data + b  # 利用求得的k和b计算预测值

![](img/e5867d217fe158a260656201e5c0eba3_49.png)

![](img/e5867d217fe158a260656201e5c0eba3_51.png)

# 3. 在同一图中绘制回归线
plt.plot(x_data, y_pred, color=‘red’, label=‘回归线’)

![](img/e5867d217fe158a260656201e5c0eba3_53.png)

plt.legend()
plt.show()
```
通过这张图，我们可以清晰地看到两个指标之间的线性关系趋势。需要注意的是，在金融分析中，我们常常对增长率、收益率等处理后的序列进行回归，而不是原始价格，这一点在应用时需要根据具体问题判断。

## 计算相关系数

![](img/e5867d217fe158a260656201e5c0eba3_57.png)

除了回归方程，衡量两个变量之间线性关系强度的常用指标是**相关系数**。在Pandas中，计算相关系数非常简单。

![](img/e5867d217fe158a260656201e5c0eba3_59.png)

对于一个包含多列数据的DataFrame，可以直接使用`.corr()`方法计算所有列两两之间的相关系数矩阵。

![](img/e5867d217fe158a260656201e5c0eba3_61.png)

```python
# 计算DataFrame中所有列的相关系数矩阵
correlation_matrix = df[[‘X‘, ’Y‘]].corr()
print(correlation_matrix)
```
对角线上的值永远是1（变量与自身的完全相关），而非对角线上的值则显示了‘X’和‘Y’之间的相关系数。这个矩阵是对称的。

![](img/e5867d217fe158a260656201e5c0eba3_63.png)

![](img/e5867d217fe158a260656201e5c0eba3_65.png)

## 附加题：相关系数的动态分析

![](img/e5867d217fe158a260656201e5c0eba3_69.png)

上一节我们计算了整体的相关系数，本节中我们来看看如何分析相关系数随时间的变化情况。这是一个更深入的分析，可以揭示两个指标间关系的稳定性或周期性。

思路是使用**滚动窗口**。我们随着时间推移，在每一个时间点，取最近一段时间（窗口）的数据来计算这两个指标的相关系数，从而得到一个随时间变化的相关系数序列。

以下是实现动态相关系数计算与可视化的代码：

![](img/e5867d217fe158a260656201e5c0eba3_71.png)

```python
# 设置滚动窗口的大小，例如250个交易日
window_size = 250

![](img/e5867d217fe158a260656201e5c0eba3_73.png)

![](img/e5867d217fe158a260656201e5c0eba3_75.png)

# 计算指标X相对于指标Y的滚动相关系数
rolling_corr = df[‘X’].rolling(window=window_size).corr(df[‘Y’])

![](img/e5867d217fe158a260656201e5c0eba3_77.png)

![](img/e5867d217fe158a260656201e5c0eba3_79.png)

# 绘制滚动相关系数随时间的变化图
rolling_corr.plot(figsize=(11, 6))
plt.title(‘滚动相关系数变化图 (窗口大小={})’.format(window_size))
plt.ylabel(‘相关系数’)
plt.show()
```
这张图展示了在每一个时间点上，基于过去一定窗口期数据计算出的相关系数。通过它，我们可以观察两个金融指标间的关联性是持续稳定，还是在某些市场阶段发生了显著变化。
