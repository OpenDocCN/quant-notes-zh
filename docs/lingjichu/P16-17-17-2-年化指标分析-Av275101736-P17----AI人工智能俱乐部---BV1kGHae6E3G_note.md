在本节课中，我们将要学习如何计算和解读量化策略分析中的两个核心收益指标：**回测收益率**与**年化收益率**。理解这两个指标对于评估一个交易策略的长期表现至关重要。

上一节我们介绍了如何计算策略的最终收益，本节中我们来看看如何将这些收益转化为更标准化的年化指标，以便进行横向比较。

## 理解回测收益率

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_8.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_10.png)

回测收益率衡量的是策略在整个回测周期内的总盈亏情况。它计算的是从开始到结束，你的投资组合价值变化的百分比。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_12.png)

**公式**如下：
`回测收益率 = (最终价值 - 初始价值) / 初始价值`

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_14.png)

这个指标直接反映了策略的最终效果。例如，有的策略可能实现了翻倍收益（收益率 > 100%），而有的策略则可能出现亏损（收益率为负值）。这直观地说明了“投资有风险”，策略并非稳赚不赔。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_16.png)

## 引入年化收益率

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_18.png)

理解了总收益后，我们面临第二个问题：如何比较不同时间长度的策略表现？一个运行了10年的策略和一个运行了2年的策略，仅看总收益率是不公平的。这时就需要**年化收益率**。

年化收益率将总收益标准化，转化为假设每年都能获得相同收益率的等效年度收益率。这使得不同时间跨度的策略之间具有可比性。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_20.png)

以下是年化收益率的计算公式：

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_22.png)

**公式**：
`年化收益率 = (1 + 总收益率) ^ (250 / N) - 1`

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_24.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_26.png)

其中：
*   **总收益率**：即我们上面计算出的回测收益率。
*   **N**：策略回测的总交易日天数。
*   **250**：通常假设一年有250个交易日（这是一个市场惯例近似值）。

## 计算示例

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_28.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_30.png)

假设我们有一个策略，其回测数据总天数 `N = 730` 天（大约2年），计算出的总收益率为 `R`。

我们可以用Python代码来计算年化收益率：

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_32.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_34.png)

```python
# 假设 total_return 是计算好的总收益率
total_return = 0.5  # 例如，总收益为50%
N = 730  # 回测总交易日
trading_days_per_year = 250

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_36.png)

# 计算年化收益率
annualized_return = (1 + total_return) ** (trading_days_per_year / N) - 1
print(f"年化收益率: {annualized_return:.2%}")
```

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_38.png)

执行计算后，我们会得到每个策略对应的年化收益率。通常，年化收益率会比总收益率数值小，因为它将收益分摊到了每一年。例如，一个在3.4年内获得113%总收益的策略，其年化收益率可能约为20%左右。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_40.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_42.png)

观察年化收益率能让我们更清晰地看到策略的**持续盈利能力**，而不仅仅是最终结果。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_44.png)

## 核心指标总结

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_46.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_48.png)

本节课中我们一起学习了量化策略分析中两个最关键的收益指标：

1.  **回测收益率**：反映策略在整个测试周期内的**最终总盈亏**，是评估策略效果的起点。
2.  **年化收益率**：将总收益标准化为年度比率，用于**公平比较**不同时间长度策略的盈利能力，是更常用的核心评估指标。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_50.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_52.png)

理解并计算这两个指标，是判断一个交易策略是否具备长期有效性的基础。在后续的分析中，我们还将结合风险指标（如最大回撤、夏普比率等）来更全面地评估策略。
