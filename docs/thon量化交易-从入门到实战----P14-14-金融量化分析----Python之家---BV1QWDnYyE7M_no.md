## 概述

花式索引允许我们使用一个整数数组作为索引，来选取原数组中任意位置、无规律的元素，并组成一个新的数组。这对于处理金融数据中需要提取特定时间点或特定股票的数据非常有用。

## 一维数组的花式索引

对于一个一维数组，如果我们想选取其中第1、3、4、6、7号元素（注意索引从0开始），可以使用一个包含这些索引的列表。

以下是具体做法：

```python
import numpy as np
A = np.arange(20)  # 创建一个0到19的数组
indices = [1, 3, 4, 6, 7]  # 指定要选取的索引
result = A[indices]
print(result)  # 输出: [1 3 4 6 7]
```

![](img/4250ab99462adcda0b5c99b938410488_5.png)

**核心概念**：`result = A[[indices_list]]`。将索引列表放在一个方括号内传递给数组，即可实现花式索引。

![](img/4250ab99462adcda0b5c99b938410488_7.png)

## 二维数组的花式索引

对于二维数组，花式索引的规则与一维类似，但可以通过逗号分隔行和列的索引方式，实现更复杂的选取。

以下是几种索引方式混合使用的示例：

1.  **常规索引与切片混合**：选取第0行，第2到第4列。
    ```python
    A_2d = np.arange(20).reshape(4,5)
    result = A_2d[0, 2:5]
    ```
2.  **常规索引与布尔型索引混合**：选取第0行中所有大于2的元素。
    ```python
    result = A_2d[0, A_2d[0] > 2]
    ```
3.  **花式索引与全切片混合**：选取第1行和第3行的所有列。
    ```python
    result = A_2d[[1, 3], :]
    ```

## 重要注意事项：避免行列同时使用花式索引

一个关键的注意事项是，**不能在逗号两边同时使用花式索引列表**。这种写法不会选取行索引和列索引的“组合”，而是会选取索引对指定的单个元素。

例如，`A_2d[[1,3], [1,3]]` 并不会选取(1,1), (1,3), (3,1), (3,3)这四个位置的值。它会被解释为选取(1,1)和(3,3)这两个位置的值。

```python
# 错误示范：这不会得到四个值
wrong_result = A_2d[[1, 3], [1, 3]]  # 输出: [6, 18]
```

## 如何正确选取行列交叉的多个值？

如果需要选取如(1,1), (1,3), (3,1), (3,3)这样的多个交叉点，需要分两步进行：

以下是分步选取的方法：

1.  首先，使用花式索引选取目标行（第1行和第3行），并保留所有列。
    ```python
    step1 = A_2d[[1, 3], :]
    ```
2.  然后，在第一步的结果上，再次使用花式索引选取目标列（第1列和第3列），并保留所有行。
    ```python
    final_result = step1[:, [1, 3]]
    ```
    也可以合并为一行：
    ```python
    final_result = A_2d[[1, 3], :][:, [1, 3]]
    ```

本节课中我们一起学习了NumPy的花式索引。我们了解到：
*   花式索引通过传递一个索引列表 `array[[indices]]` 来选取无规律的元素。
*   在二维数组中，可以灵活混合使用普通索引、切片、布尔索引和花式索引。
*   需要特别注意，**避免在行和列的位置上同时使用花式索引列表**，否则会产生非预期的结果。正确的做法是分步进行索引操作。
