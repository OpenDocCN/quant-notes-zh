# 量化交易教程：P46：工具包绘图展示 📊

在本节课中，我们将学习如何使用量化分析工具包中的绘图功能，来直观地展示因子的IC值（信息系数）及其时间序列走势。通过可视化，我们可以更清晰地评估因子的有效性和稳定性。

## 绘图工具导入与使用

上一节我们介绍了如何计算因子的相关系数（IC值）。本节中，我们来看看如何将这些计算出的IC值通过图表进行可视化展示。

我们导入一个名为 `PROLOATING` 的工具包，它专门用于帮助我们进行结果展示。

以下是绘图的核心代码：

```python
# 假设我们已经计算得到了IC值序列 ic_series
# 使用工具包中的函数绘制IC时间序列图
PROLOATING.plot_ic_time_series(ic_series)
```

![](img/91d23050cce8baadbfe65c697de6653e_1.png)

执行这段代码后，系统会默认生成一张图表。

## 图表结果解读

![](img/91d23050cce8baadbfe65c697de6653e_3.png)

生成的图表包含以下主要元素：
*   **蓝色线**：代表每日计算出的实际IC值走势。由于每日波动可能较大，这条线通常看起来起伏不定。
*   **绿色线**：代表以一个月为周期计算的平均IC值走势。这条线能帮助我们过滤掉日度波动，观察更长期的趋势。

图表还会在角落标注两个重要的统计指标：
*   **均值 (Mean)**：整个时间段内IC值的平均值。
*   **标准差 (STD)**：IC值波动程度的度量。

我们主要观察绿色平均线。从理想情况看，我们希望IC值越大越好，并且平均线能呈现出一定的上升或稳定趋势。如果平均线长期平稳且数值较小，则可能意味着该因子与收益率的相关性不强，有效性较低。

![](img/91d23050cce8baadbfe65c697de6653e_1.png)

## 信息比率简介

除了图表，工具包还会计算并展示一个名为 **信息比率 (Information Ratio)** 的指标。

信息比率的计算公式为：
**信息比率 = IC均值 / IC标准差**

这个比率用于描述因子IC值的稳定性。比值越大，说明因子的有效性越稳定；比值越小，则意味着因子的表现波动较大，稳定性不足。理解这个指标有助于我们从另一个维度评估因子质量。

![](img/91d23050cce8baadbfe65c697de6653e_5.png)

## 其他绘图功能

工具包的绘图功能非常丰富。除了绘制单期IC时间序列，它还能展示不同周期（如5期、10期）的IC走势，方便进行多周期对比分析。

![](img/91d23050cce8baadbfe65c697de6653e_7.png)

![](img/91d23050cce8baadbfe65c697de6653e_5.png)

API文档中还提供了其他多种分析图表，例如：
*   **直方图 (Histogram)**：查看IC值的分布情况。
*   **QQ图 (Q-Q Plot)**：检验IC值是否服从正态分布。
*   **热力图 (Heatmap)**：用颜色深浅直观展示不同维度下的IC值。

![](img/91d23050cce8baadbfe65c697de6653e_7.png)
