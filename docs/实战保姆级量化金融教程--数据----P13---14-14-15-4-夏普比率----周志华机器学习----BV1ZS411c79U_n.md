## 夏普比率的定义与意义
上一节我们介绍了投资回报率，本节中我们来看看如何衡量风险与收益的关系。夏普比率描述的是：对于承担的每一单位风险，投资者能获得多少超额收益。

![](img/ed80283b0da939415ed3fbc2bab6a63e_1.png)

我们可以用一个生活中的例子来理解：一份在战乱地区日薪数万的工作，其高薪对应的是极高的生命风险。夏普比率就是用来评估，为了获得一定的收益，我们所承担的风险是否“值得”。

![](img/ed80283b0da939415ed3fbc2bab6a63e_3.png)

**核心公式**：
`夏普比率 = (投资组合预期收益率 - 无风险利率) / 投资组合收益率的标准差`

指标越高，意味着在承担相同风险的情况下，获得的超额收益越高，该投资组合的表现也就越优秀。在选股时，在其他条件相近的情况下，我们倾向于选择夏普比率更高的股票或组合。

## 夏普比率的计算逻辑
理解了夏普比率的含义后，我们来看看它的具体计算逻辑。它主要比较两类收益率：

1.  **投资组合的收益率**：你选择的股票或基金等投资产品带来的回报。
2.  **无风险收益率**：通常指国债、银行存款等几乎无风险投资的收益率，例如年化3%或5%。

计算时，首先用投资组合收益率减去无风险收益率，得到“超额收益”。然后，用这个超额收益除以投资组合收益率自身的波动程度（即标准差）。标准差衡量了收益的稳定性，波动越大，风险越高。

所以，夏普比率本质上计算的是 **“每单位风险所产生的超额收益”**。

## 代码实现与结果分析
理论部分已经清晰，现在让我们通过代码实际计算一下夏普比率。在计算前，我们需要确保收益率数据是完整的，因此通常会处理缺失值。

![](img/ed80283b0da939415ed3fbc2bab6a63e_7.png)

以下是计算夏普比率的关键步骤代码：

```python
# 假设 returns 是投资组合的日收益率序列
# risk_free_rate 是无风险年化利率，例如 0.05 (5%)
# 计算年化夏普比率
import numpy as np

# 计算超额收益（日度）
excess_returns = returns - risk_free_rate / 250  # 假设一年有250个交易日

# 计算夏普比率
sharpe_ratio = np.sqrt(250) * (excess_returns.mean() / excess_returns.std())
```

*   股票C的夏普比率为负（-0.15），说明其收益甚至未能覆盖无风险收益，在承担风险后反而获得了更差的回报，应避免选择。
