## 数据查询与预处理
在交易日开始前，我们需要获取股票池中每只股票的财务数据。这就像数据挖掘，我们需要从海量指标中提取有用的信息。

所有可用的数据指标都可以在平台的帮助文档中找到。对于本次示例，我们将重点查询“营业收入”这一财务指标。

以下是查询财务数据的基本步骤：
1.  使用 `query` 函数指定要查询的指标。
2.  使用 `filter` 方法过滤出属于我们股票池的股票。
3.  使用 `order_by` 方法对结果进行排序。
4.  使用 `limit` 方法限制返回的股票数量。

```python
def before_trading_start(context):
    # 构建查询：获取营业收入指标
    q = query(
        fundamentals.financial_indicator.operating_revenue
    ).filter(
        fundamentals.financial_indicator.code.in_(context.stock_pool)
    ).order_by(
        fundamentals.financial_indicator.operating_revenue.desc() # 按营收降序排列
    ).limit(10) # 只取前10名

    # 执行查询，并将结果存入context
    context.filtered_stocks = get_fundamentals(q)
    # 打印结果以供检查
    print(context.filtered_stocks)
```

在这段代码中，我们查询了股票池中所有股票的营业收入，然后按照营收从高到低排序，最后只选取排名前10的股票。查询结果是一个DataFrame，我们将其保存在 `context.filtered_stocks` 中以便后续交易逻辑使用。其他暂时不用的函数（如 `handle_data` ）可以先使用 `pass` 语句跳过。
