上一节我们介绍了如何构建用于分析的特征数据。本节中，我们来看看如何对这些特征数据进行统计分析，并评估一个基于统计规则的简单策略效果。

![](img/b9683effda3a56e24ea6cab0c0d04afb_4.png)

首先，我们需要对数据进行变换，以便进行统计。我们关注的是“前一天涨跌”和“前两天涨跌”这两个特征组合下，当天股价涨跌（即结果）的分布情况。

![](img/b9683effda3a56e24ea6cab0c0d04afb_6.png)

![](img/b9683effda3a56e24ea6cab0c0d04afb_8.png)

以下是具体的数据处理步骤：

![](img/b9683effda3a56e24ea6cab0c0d04afb_10.png)

```python
# 假设 df 是包含特征列 ‘feature_1‘, ‘feature_2‘ 和结果列 ‘direction‘ 的DataFrame
# 使用 groupby 对两个特征进行分组，并统计每组中结果的数量
grouped_result = df.groupby([‘feature_1‘, ‘feature_2‘, ‘direction‘]).size()
```

![](img/b9683effda3a56e24ea6cab0c0d04afb_12.png)

执行上述代码后，我们得到一个分组统计结果。为了更清晰地展示所有可能的组合（即使某些组合没有出现），我们可以使用 `unstack` 方法并填充缺失值为0。

![](img/b9683effda3a56e24ea6cab0c0d04afb_14.png)

![](img/b9683effda3a56e24ea6cab0c0d04afb_16.png)

```python
# 将结果‘direction‘展开为列，并填充缺失值为0
pivot_table = grouped_result.unstack(fill_value=0)
```

![](img/b9683effda3a56e24ea6cab0c0d04afb_18.png)

此时，我们得到一个清晰的透视表。行索引是特征组合 `(feature_1, feature_2)`，列是当天的涨跌结果 `direction`（例如1代表涨，-1代表跌），表格中的数值代表该情况出现的次数。

![](img/b9683effda3a56e24ea6cab0c0d04afb_20.png)

![](img/b9683effda3a56e24ea6cab0c0d04afb_22.png)

![](img/b9683effda3a56e24ea6cab0c0d04afb_24.png)

接下来，我们需要从这个统计表中找出规律。我们的目标是：根据“前一天涨跌”和“前两天涨跌”的特征，预测当天最可能出现的涨跌情况。

以下是分析统计结果并制定策略的过程：

我们观察透视表，计算在每种特征组合下，涨（1）和跌（-1）哪种情况出现的次数更多。例如，当 `(feature_1=1, feature_2=1)`，即前两天都上涨时，我们统计发现接下来一天下跌的次数多于上涨的次数。

![](img/b9683effda3a56e24ea6cab0c0d04afb_26.png)

基于这个观察，我们可以制定一个简单的规则策略：
*   如果 `feature_1 == 1` 且 `feature_2 == 1`，则预测当天 `direction = -1`（跌）。
*   否则，预测当天 `direction = 1`（涨）。

这个规则可以用NumPy的 `where` 函数来实现：

![](img/b9683effda3a56e24ea6cab0c0d04afb_28.png)

```python
import numpy as np
# 根据特征列生成预测信号
data[‘pred_signal‘] = np.where((data[‘feature_1‘] == 1) & (data[‘feature_2‘] == 1), -1, 1)
```

![](img/b9683effda3a56e24ea6cab0c0d04afb_30.png)

策略制定完成后，我们需要评估它的效果。首先，我们可以计算这个简单策略的预测准确率。

以下是评估预测准确性的方法：

![](img/b9683effda3a56e24ea6cab0c0d04afb_36.png)

我们将策略预测的信号 `pred_signal` 与实际涨跌情况 `direction` 进行比较。

![](img/b9683effda3a56e24ea6cab0c0d04afb_38.png)

```python
# 计算预测正确的天数
correct_predictions = (data[‘pred_signal‘] == data[‘direction‘]).value_counts()
print(correct_predictions)
```

![](img/b9683effda3a56e24ea6cab0c0d04afb_40.png)

输出结果可能显示，在总共2135天中，有1102天预测正确，1033天预测错误。准确率略高于50%，说明这个基于统计的简单规则比随机猜测稍好一些。

![](img/b9683effda3a56e24ea6cab0c0d04afb_42.png)

![](img/b9683effda3a56e24ea6cab0c0d04afb_44.png)

然而，预测准确率并不能完全代表策略的盈利能力。最终，我们需要在回测中查看策略带来的实际收益。

![](img/b9683effda3a56e24ea6cab0c0d04afb_46.png)

以下是进行简单回测和结果对比的步骤：

我们计算并对比两种情况的累计收益：
1.  **实际收益**：完全跟随大盘（即 `direction`）买卖的收益。
2.  **策略收益**：按照我们的统计策略（即 `pred_signal`）买卖的收益。

![](img/b9683effda3a56e24ea6cab0c0d04afb_48.png)

![](img/b9683effda3a56e24ea6cab0c0d04afb_50.png)

```python
# 计算每日收益率（这里假设 returns 是预先计算好的日收益率序列）
# 实际收益序列
actual_returns = data[‘direction‘].shift(1) * data[‘returns‘]
# 策略收益序列
strategy_returns = data[‘pred_signal‘].shift(1) * data[‘returns‘]

![](img/b9683effda3a56e24ea6cab0c0d04afb_52.png)

![](img/b9683effda3a56e24ea6cab0c0d04afb_54.png)

# 计算累计收益
cumulative_actual_returns = (1 + actual_returns).cumprod()
cumulative_strategy_returns = (1 + strategy_returns).cumprod()
```

![](img/b9683effda3a56e24ea6cab0c0d04afb_56.png)

绘制两条累计收益曲线进行对比。结果可能会显示，虽然两者最终可能都是亏损的，但基于统计的策略 `strategy_returns` 的亏损幅度小于单纯跟随大盘的 `actual_returns`。这证明了即使简单的统计规律也可能带来一定的风险控制效果。
