## 绘制小提琴图

上一节我们介绍了数据的基本情况，本节中我们来看看如何使用`seaborn`库的核心函数来绘制小提琴图。除了常规的数据处理库，我们还需要导入`seaborn`和`matplotlib`进行绘图。

以下是绘制小提琴图所需的完整代码步骤：

```python
import seaborn as sns
import matplotlib.pyplot as plt

# 设置中文字体，防止乱码
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 设置图形大小
plt.figure(figsize=(8, 5))

# 使用seaborn绘制小提琴图
sns.violinplot(data=df,
               x='city',      # X轴为城市
               y='AQI',       # Y轴为AQI指标
               width=0.8,     # 设置小提琴的宽度
               linewidth=2,   # 设置轮廓线宽
               saturation=0.9, # 设置颜色饱和度
               fliersize=2,   # 设置离群值标记点的大小
               orient='v',    # 绘制垂直方向的小提琴图
               inner='box'    # 在小提琴图内部绘制箱形图
              )

# 显示图形
plt.show()
```

## 代码参数详解

在上一节中，我们运行了绘制小提琴图的代码。本节中，我们将详细解释代码中各个关键参数的作用，以帮助你理解和定制自己的图表。

以下是代码中主要参数的含义说明：

*   **`data=df`**：指定用于绘图的数据框。
*   **`x='city'`**：指定作为X轴（分类变量）的数据列，此处为城市名称。
*   **`y='AQI'`**：指定作为Y轴（数值变量）的数据列，此处为AQI指数。
*   **`width=0.8`**：控制每个小提琴图形的宽度。值越大，图形越宽。
*   **`linewidth=2`**：控制小提琴图形轮廓线的粗细。
*   **`saturation=0.9`**：控制颜色的饱和度。1为完全饱和，0为完全不饱和（灰度）。
*   **`fliersize=2`**：控制箱形图中离群值（异常点）标记的大小。
*   **`orient='v'`**：设置小提琴图的方向。`'v'`表示垂直，`'h'`表示水平。
*   **`inner='box'`**：设置在小提琴图内部显示的统计图形。`'box'`表示显示箱形图，其他选项如`'quartile'`（四分位线）或`'stick'`（数据点）也可用。

## 结果解读

运行上述代码后，我们将得到一个垂直方向的小提琴图。图中每个小提琴代表一个城市，其形状展示了该城市AQI数据的分布特征。

我们可以从以下几个方面解读图表：

*   **长度（Y轴范围）**：表示该城市AQI值的整体范围。长度越长，说明AQI值的波动范围越大。
*   **宽度**：表示数据在特定取值上的密度。中间部分越宽，说明数据在该值附近越集中；越窄则说明数据越分散。
*   **内部箱形图**：提供了中位数、四分位数等统计信息，便于进行更精确的比较。
*   **颜色差异**：不同颜色帮助我们快速区分不同的城市。

通过这个图表，我们可以直观地比较不同城市空气质量指数的分布差异，例如哪个城市的AQI普遍较高，哪个城市的数据波动更大。
