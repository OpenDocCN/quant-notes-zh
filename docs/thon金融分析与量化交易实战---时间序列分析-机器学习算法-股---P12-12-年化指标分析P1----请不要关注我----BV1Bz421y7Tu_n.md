## 回测收益率

![](img/9b4ea79be040573bac6aad39d1c69010_10.png)

![](img/9b4ea79be040573bac6aad39d1c69010_12.png)

回测收益率衡量的是在整个回测周期内，策略带来的总收益与初始本金的比值。它直接回答了“这个策略最终让我赚了（或亏了）多少钱”的问题。投资有风险，策略的收益可能为正，也可能为负。

以下是计算回测收益率的逻辑：
```python
# 假设 final_value 是回测结束时的总资产，initial_value 是初始本金
backtest_return = (final_value - initial_value) / initial_value
```
这个指标是评估策略有效性的首要标准。

![](img/9b4ea79be040573bac6aad39d1c69010_14.png)

## 年化收益率

![](img/9b4ea79be040573bac6aad39d1c69010_16.png)

![](img/9b4ea79be040573bac6aad39d1c69010_18.png)

理解了总收益后，另一个关键指标是年化收益率。当回测周期较长（例如数年）时，年化收益率能将总收益标准化，转化为一个“平均每年”的收益水平，便于在不同时间长度的策略之间进行比较。

![](img/9b4ea79be040573bac6aad39d1c69010_20.png)

年化收益率的计算公式如下：
**年化收益率 = (1 + 总收益率) ^ (年交易日数 / 回测总交易日数) - 1**

![](img/9b4ea79be040573bac6aad39d1c69010_22.png)

![](img/9b4ea79be040573bac6aad39d1c69010_24.png)

其中：
*   **总收益率 (R)**：即上文计算的回测收益率。
*   **年交易日数 (M)**：通常取250或252天。
*   **回测总交易日数 (N)**：策略回测数据的总天数。

![](img/9b4ea79be040573bac6aad39d1c69010_26.png)

以下是使用Python计算年化收益率的示例：
```python
# 假设 total_return 是计算好的总收益率，trade_days_per_year=250, total_trade_days 是数据总天数
annualized_return = (1 + total_return) ** (trade_days_per_year / total_trade_days) - 1
```
通过这个转换，我们可以得到更标准化、更具可比性的收益指标。虽然数值可能与总收益率相近，但意义不同，通常是分析时重点观察的指标。

![](img/9b4ea79be040573bac6aad39d1c69010_30.png)

## 实例计算与对比

![](img/9b4ea79be040573bac6aad39d1c69010_32.png)

![](img/9b4ea79be040573bac6aad39d1c69010_34.png)

让我们通过一个实例来观察这两个指标的差异。假设我们有一个回测周期为730天（约3年）的策略数据。

![](img/9b4ea79be040573bac6aad39d1c69010_36.png)

![](img/9b4ea79be040573bac6aad39d1c69010_38.png)

以下是计算步骤：
1.  首先计算各股票在整个回测期内的总收益率（回测收益率）。
2.  然后应用年化收益率公式，将总收益率转化为年化水平。

![](img/9b4ea79be040573bac6aad39d1c69010_40.png)

![](img/9b4ea79be040573bac6aad39d1c69010_42.png)

计算结果会显示，一个在3年内获得113%总收益的策略，其年化收益率可能在20%左右。而一个总收益为负的策略，其年化亏损率也会相应调整。这清晰地展示了将长期收益“年化”后的效果。

![](img/9b4ea79be040573bac6aad39d1c69010_44.png)

本节课中我们一起学习了量化策略分析的两个核心收益指标。**回测收益率**反映了策略在整个测试期的绝对盈亏情况，而**年化收益率**则提供了标准化的年度平均收益水平，是进行策略横向对比的关键依据。理解并计算这两个指标，是评估任何交易策略表现的基础。
