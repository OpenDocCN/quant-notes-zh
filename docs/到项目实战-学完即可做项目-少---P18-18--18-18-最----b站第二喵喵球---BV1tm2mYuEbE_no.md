## 概述

最大回撤是评估投资策略风险的关键指标。它描述的是在特定时间段内，资产净值从最高点到后续最低点的最大跌幅。这个指标不衡量策略的盈利能力，而是衡量其可能面临的最大风险程度。

## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何评估策略的风险。最大回撤描述的是投资过程中“最惨”的阶段。

它衡量的是从任意一个历史高点开始，到之后某个低点为止，资产价值下跌的最大幅度。这个“区间”代表了投资者可能承受的最大账面亏损和心理压力。

## 最大回撤的计算原理

理解了最大回撤的概念后，我们来看看如何用数学公式来描述它。其核心是找到资产价格序列中跌幅最大的那个“峰-谷”组合。

最大回撤的计算公式如下：

![](img/6c276977b5139a5a356a24c3603aa98f_5.png)

**最大回撤 = max( (P_i - P_j) / P_i )**

![](img/6c276977b5139a5a356a24c3603aa98f_7.png)

![](img/6c276977b5139a5a356a24c3603aa98f_9.png)

其中：
*   **P_i** 代表在时间点 `i` 的资产价格（或净值）。
*   **P_j** 代表在时间点 `i` 之后（`j > i`）的资产价格。
*   我们需要遍历所有可能的 `i` 和 `j` 组合，找出使 `(P_i - P_j) / P_i` 这个比值最大的情况。这个比值就是回撤率。

这个公式计算的是相对跌幅（百分比），比绝对金额更能反映风险的严重程度。

![](img/6c276977b5139a5a356a24c3603aa98f_11.png)

## 使用Python计算最大回撤

![](img/6c276977b5139a5a356a24c3603aa98f_13.png)

![](img/6c276977b5139a5a356a24c3603aa98f_15.png)

理论部分已经清晰，本节中我们将动手使用Python的Pandas库来实现最大回撤的计算。我们将通过一个分步示例来演示。

![](img/6c276977b5139a5a356a24c3603aa98f_17.png)

首先，我们需要一个资产价格的时间序列数据。这里我们假设已经有一个名为 `df` 的DataFrame，其中包含一列价格数据 `‘close’`。

![](img/6c276977b5139a5a356a24c3603aa98f_19.png)

以下是计算最大回撤的步骤：

![](img/6c276977b5139a5a356a24c3603aa98f_21.png)

1.  **计算历史最高值序列**：使用 `cummax()` 函数，为每个时间点计算截至该点的历史最高价格。
    ```python
    # 计算截至每一天的历史最高价
    historical_max = df[‘close’].cummax()
    ```

![](img/6c276977b5139a5a356a24c3603aa98f_23.png)

![](img/6c276977b5139a5a356a24c3603aa98f_25.png)

2.  **计算每日回撤**：用历史最高价序列减去当日的实际价格，得到每日的“潜在亏损”金额。
    ```python
    # 计算每日的回撤（从历史最高点下跌了多少）
    drawdown = historical_max - df[‘close’]
    ```

![](img/6c276977b5139a5a356a24c3603aa98f_27.png)

![](img/6c276977b5139a5a356a24c3603aa98f_29.png)

3.  **计算每日回撤率**：将每日回撤金额除以对应的历史最高价，得到回撤率。
    ```python
    # 计算每日的回撤率（百分比）
    drawdown_pct = drawdown / historical_max
    ```

![](img/6c276977b5139a5a356a24c3603aa98f_31.png)

4.  **找出最大回撤**：在所有每日回撤率中，找到最大值，这就是整个考察期内的最大回撤。
    ```python
    # 计算整个时间段内的最大回撤率
    max_drawdown = drawdown_pct.max()
    print(f“最大回撤率为：{max_drawdown:.2%}”)
    ```

![](img/6c276977b5139a5a356a24c3603aa98f_33.png)

将以上步骤整合，一个完整的计算函数如下：
```python
def calculate_max_drawdown(price_series):
    “””
    计算价格序列的最大回撤率
    :param price_series: pandas Series，资产价格序列
    :return: 最大回撤率（小数形式）
    “””
    # 计算历史最高点序列
    historical_max = price_series.cummax()
    # 计算每日回撤率
    drawdown_pct = (historical_max - price_series) / historical_max
    # 返回最大回撤率
    return drawdown_pct.max()
```

## 最大回撤在策略分析中的应用

![](img/6c276977b5139a5a356a24c3603aa98f_39.png)

学会了如何计算，我们来看看这个指标在实际分析中如何发挥作用。在专业的回测系统中，最大回撤会被自动计算并可视化。

它通常以图表形式呈现，在资产净值曲线上，用阴影高亮标出发生最大回撤的那个时间段。这能直观地告诉策略开发者：
*   策略在历史上经历过的最糟糕时期是什么时候。
*   在那段时期，资产缩水了百分之多少。
*   从高峰跌到谷底持续了多长时间。

![](img/6c276977b5139a5a356a24c3603aa98f_41.png)

![](img/6c276977b5139a5a356a24c3603aa98f_43.png)

评估策略时，不能只看总收益率。一个高收益但伴随极大回撤的策略，可能因为投资者无法承受中途的巨大亏损而被迫提前止损，从而无法实现最终的收益。因此，最大回撤是衡量策略稳健性和可执行性的关键。
