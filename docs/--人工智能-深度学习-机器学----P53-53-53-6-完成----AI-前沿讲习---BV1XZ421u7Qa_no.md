## 计算持仓股票
首先，我们需要一个函数来获取当前账户中持有的所有股票。

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_11.png)

以下是`get_holdings`函数的实现步骤：

```python
def get_holdings(context):
    holdings = []
    for position in context.portfolio.positions:
        if context.portfolio.positions[position].quantity > 0:
            holdings.append(position)
    return holdings
```

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_13.png)

这个函数遍历投资组合中的所有持仓，检查其数量是否大于零，并将持有的股票代码添加到列表中。

## 计算买卖清单
获取当前持仓后，我们需要将其与最新的股票排名进行比较，以确定需要买入和卖出的股票。

以下是计算买卖清单的逻辑：

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_15.png)

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_17.png)

```python
# 获取最新排名和当前持仓
stocks = get_stocks(context)
holdings = get_holdings(context)

# 计算需要买入的股票（在新排名中但不在当前持仓中）
to_buy = list(set(stocks) - set(holdings))

# 计算需要卖出的股票（在当前持仓中但不在新排名中）
to_sell = list(set(holdings) - set(stocks))
```

## 执行卖出操作
确定了卖出清单后，我们需要执行卖出操作。通常的策略是先卖出，再买入。

以下是执行卖出操作的代码：

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_19.png)

```python
for stock in to_sell:
    order_target_percent(stock, 0)
```

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_23.png)

这段代码遍历`to_sell`列表中的每只股票，并使用`order_target_percent`函数将其持仓比例调整为0%，即全部卖出。

## 执行买入操作
卖出完成后，我们执行买入操作。买入时，通常将可用资金平均分配到每只待买入的股票上。

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_29.png)

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_31.png)

以下是执行买入操作的步骤：

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_33.png)

![](img/2a44a57ac8a9ce1525ae6f52378df9ca_35.png)

1.  首先检查是否有股票需要买入。如果没有，则直接返回。
    ```python
    if len(to_buy) == 0:
        return
    ```
2.  计算可用于投资的资金总额。
    ```python
    total_value = context.portfolio.total_value
    ```
3.  计算每只股票应分配的资金。
    ```python
    average_value = total_value / len(to_buy)
    ```
4.  遍历买入清单，为每只股票下单。
    ```python
    for stock in to_buy:
        order_target_value(stock, average_value)
    ```
