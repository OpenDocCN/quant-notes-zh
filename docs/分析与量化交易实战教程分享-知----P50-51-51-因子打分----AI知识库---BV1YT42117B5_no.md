## 🔄 遍历因子并排序
上一节我们介绍了如何将因子分为“越高越好”和“越低越好”两类，并得到了两个DataFrame。本节中，我们来看看如何遍历这些因子，并对它们进行排序。

![](img/b719f4c28640f0c80873e39fc2ba7eab_6.png)

以下是遍历“越高越好”因子DataFrame并进行排序的步骤：

1.  首先，我们需要导入必要的库：`pandas` 用于数据处理，`numpy` 用于数值计算。
    ```python
    import pandas as pd
    import numpy as np
    ```

2.  遍历 `up` 这个DataFrame中的每一个因子列。
    ```python
    for factor in up.columns:
        # 获取当前因子的数据列
        current_factor_data = up[[factor]]
    ```

3.  对当前因子的数据进行排序。我们使用 `sort_values` 方法，并指定 `inplace=True` 参数，这样排序结果会直接更新到原DataFrame中，无需重新赋值。
    ```python
        # 按照当前因子值进行降序排序（越高越好）
        current_factor_data.sort_values(by=factor, ascending=False, inplace=True)
    ```

## ⚖️ 为排序后的因子打分
排序完成后，我们需要为每只股票在该因子上的表现进行打分。简单起见，我们采用线性打分法：排名第一的股票得分最高，排名最后的股票得分最低。

我们将使用 `numpy.linspace` 函数来生成一个线性的分数序列。这个函数可以生成指定范围内均匀间隔的数字。

![](img/b719f4c28640f0c80873e39fc2ba7eab_10.png)

以下是 `numpy.linspace` 的基本用法：
```python
# 语法：np.linspace(start, stop, num)
# start: 序列的起始值
# stop: 序列的结束值
# num: 要生成的样本数量

# 示例：生成从1到300的300个数字
scores = np.linspace(1, 300, 300)
print(scores) # 输出: [  1.   2.   3. ... 298. 299. 300.]
```

![](img/b719f4c28640f0c80873e39fc2ba7eab_12.png)

![](img/b719f4c28640f0c80873e39fc2ba7eab_14.png)

对于“越高越好”的因子，我们希望因子值最大的股票（排名第一）获得最高分（300分）。因此，我们可以使用 `np.linspace(300, 1, 股票数量)` 来生成一个从300递减到1的分数序列，并赋值给排序后的因子列。

![](img/b719f4c28640f0c80873e39fc2ba7eab_16.png)

以下是具体的打分代码：
```python
    # 假设股票总数为300
    stock_count = 300
    # 为“越高越好”的因子生成分数：第一名300分，最后一名1分
    up[factor] = np.linspace(stock_count, 1, stock_count)
```

## 📉 处理“越低越好”的因子
处理“越低越好”因子的逻辑与“越高越好”类似，但排序和打分的顺序相反。

![](img/b719f4c28640f0c80873e39fc2ba7eab_24.png)

以下是处理“越低越好”因子DataFrame的步骤：

![](img/b719f4c28640f0c80873e39fc2ba7eab_26.png)

![](img/b719f4c28640f0c80873e39fc2ba7eab_28.png)

1.  遍历 `down` 这个DataFrame中的每一个因子列。
    ```python
    for factor in down.columns:
        # 获取当前因子的数据列
        current_factor_data = down[[factor]]
    ```

![](img/b719f4c28640f0c80873e39fc2ba7eab_30.png)

2.  对当前因子的数据进行升序排序（因为越低越好）。
    ```python
        # 按照当前因子值进行升序排序（越低越好）
        current_factor_data.sort_values(by=factor, ascending=True, inplace=True)
    ```

![](img/b719f4c28640f0c80873e39fc2ba7eab_32.png)

3.  为排序后的因子打分。对于“越低越好”的因子，因子值最小的股票（排名第一）应获得最高分。因此，我们同样使用 `np.linspace(300, 1, 股票数量)` 来生成分数序列。
    ```python
        # 为“越低越好”的因子生成分数：第一名（值最小）300分，最后一名1分
        down[factor] = np.linspace(stock_count, 1, stock_count)
    ```

## 🧩 合并数据并计算总分
现在，我们得到了两个已经打好分的DataFrame：`up` 和 `down`。每个DataFrame都是300行（股票）乘以若干列（因子），每个单元格的值是该股票在该因子上的得分。

![](img/b719f4c28640f0c80873e39fc2ba7eab_38.png)

为了计算每只股票的综合得分，我们需要将所有因子的得分合并到一起。由于 `up` 和 `down` 的因子列名不同，我们可以使用 `pandas.concat` 函数将它们按列合并。

以下是合并DataFrame的代码：
```python
# 将‘越高越好’和‘越低越好’的因子得分表按列合并
combined_scores = pd.concat([up, down], axis=1)
```

合并后的 `combined_scores` DataFrame 包含了所有因子的得分。接下来，计算每只股票的总分就非常简单了，只需对每一行（即每一只股票）的所有因子得分进行求和。

![](img/b719f4c28640f0c80873e39fc2ba7eab_40.png)

以下是计算总分的代码：
```python
# 计算每只股票的综合总分（对所有因子得分求和）
total_score = combined_scores.sum(axis=1)
```

变量 `total_score` 就是一个Series，索引是股票代码，值是该股票的综合得分。得分越高的股票，理论上越值得投资。
