在本节课程中，我们将学习数据预处理中的一个核心步骤——标准化。标准化是数据挖掘和机器学习中非常常用的技术，其目的是消除不同特征（指标）之间由于量纲和取值范围不同带来的影响，使数据更易于分析和建模。

![](img/f665a864eed1a255edec0fe9a6a338de_3.png)

## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看如何通过标准化处理数据。

标准化操作的核心目标是：**使不同特征（维度）的数值处于同一量级，以便公平地进行比较和分析**。

为了理解标准化的目的，我们可以观察一个简单的二维数据分布。假设我们有两个特征X1和X2。X1的取值范围可能较大（例如从-10到10），而X2的取值范围可能较小（例如从-1到1）。这种取值范围的差异，可能会让某些机器学习算法（如基于距离的KNN、SVM等）过度关注取值范围大的特征，从而影响模型的准确性和公平性。

因此，标准化的出发点就是为了让所有特征在各个维度上的取值范围尽可能相同，让我们能用同样的“眼光”去观察它们，避免数值层面的差异对最终结果造成不必要的影响。

## 标准化的公式与步骤

标准化的具体操作遵循一个明确的数学公式。以下是标准化的核心公式：

**`z = (x - μ) / σ`**

其中：
*   **`z`** 代表标准化后的值。
*   **`x`** 代表原始数据值。
*   **`μ`** 代表该特征所有数据的**均值**。
*   **`σ`** 代表该特征所有数据的**标准差**。

这个公式可以分解为两个连续的步骤：

1.  **去均值（中心化）**：`x - μ`
    *   **目的**：将数据的中心移动到零点。原始数据通常不是以原点（0）为中心对称分布的。减去均值后，数据在各个维度上都会以零为中心对称。

2.  **除标准差（缩放）**：`(x - μ) / σ`
    *   **目的**：将数据的分布“缩放”到标准范围。仅完成中心化后，不同特征的取值范围（离散程度）依然不同。除以标准差可以解决这个问题。
    *   **原理**：标准差（σ）衡量了数据的离散程度。取值范围大、数据分散的特征，其标准差也大；反之则小。用中心化后的数据除以各自的标准差，相当于对离散程度大的数据进行“压缩”，对离散程度小的数据进行“拉伸”。最终，所有特征的数据分布都会具有近似相同的尺度（通常均值约为0，标准差为1）。

通过这两个步骤，数据不仅以零为中心对称，而且各特征的取值范围也变得近似相同。

## 标准化的代码实现

理解了原理后，我们可以动手实现标准化。在Python中，我们可以根据公式手动计算，也可以利用现成的库函数。

![](img/f665a864eed1a255edec0fe9a6a338de_5.png)

![](img/f665a864eed1a255edec0fe9a6a338de_7.png)

以下是手动实现标准化的示例代码：

```python
def standardize(series):
    """
    对输入的Pandas Series进行标准化处理。
    参数:
        series: 待处理的数据序列。
    返回:
        标准化后的数据序列。
    """
    # 计算均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    # 应用标准化公式： (x - μ) / σ
    return (series - mean_val) / std_val
```

![](img/f665a864eed1a255edec0fe9a6a338de_9.png)

代码非常简单：首先计算序列的均值和标准差，然后对序列中的每个值应用标准化公式。
