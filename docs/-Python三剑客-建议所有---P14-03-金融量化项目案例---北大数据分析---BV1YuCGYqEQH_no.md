## 需求分析

![](img/eb0f637a18b38045b3181a702dc4f1c8_17.png)

上一节我们明确了项目目标，本节中我们来看看如何对需求进行具体分析。

首先，我们需要明确几个关键点：

1.  **时间范围**：策略从2010年1月1日开始，到当前日期（案例中以2020年2月18日为例）为止。我们需要从原始数据中筛选出这个时间段的数据。
2.  **交易单位**：一手股票代表100股。所有买卖操作都以“手”为单位进行。
3.  **买卖规则**：
    *   **买入**：每月第一个交易日，买入一手（100股）股票。
    *   **卖出**：每年最后一个交易日，卖出当年持有的所有股票。
4.  **价格基准**：为了简化计算，我们统一使用**开盘价**作为买卖股票的单价。
5.  **收益计算**：总收益 = 卖出股票所得总金额 + 手中剩余股票估值 - 买入股票所花总金额。

基于以上分析，一个完整的年份（例如2010年）内，投资者会买入12手股票（共1200股），并在年底卖出这1200股。对于未结束的年份（例如2020年），买入的股票尚未卖出，需要按当前市价估算其价值。

## 计算买入股票的总花费

![](img/eb0f637a18b38045b3181a702dc4f1c8_51.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_53.png)

首先，我们来计算投资者买入股票总共花费了多少钱。

我们需要找到每月第一个交易日的数据行，并提取其开盘价。

![](img/eb0f637a18b38045b3181a702dc4f1c8_55.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_57.png)

以下是实现步骤：

![](img/eb0f637a18b38045b3181a702dc4f1c8_59.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_61.png)

1.  使用`resample`方法按月份分组数据，并获取每月的第一行数据（即每月第一个交易日）。
2.  从结果中提取`open`（开盘价）列。
3.  将每个开盘价乘以100（因为每手100股），然后求和，得到总花费。

![](img/eb0f637a18b38045b3181a702dc4f1c8_63.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_65.png)

```python
# 获取每月第一个交易日的数据
df_monthly = new_df.resample(‘M‘).first()

![](img/eb0f637a18b38045b3181a702dc4f1c8_67.png)

# 计算买入股票的总花费：每月开盘价 * 100股，然后求和
cost = df_monthly[‘open‘].sum() * 100
print(f“买入股票总花费：{cost}“)
```

## 计算卖出股票的总收入

![](img/eb0f637a18b38045b3181a702dc4f1c8_77.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_79.png)

接下来，我们计算投资者卖出股票总共获得了多少钱。

![](img/eb0f637a18b38045b3181a702dc4f1c8_81.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_83.png)

我们需要找到每年最后一个交易日的数据行。这里需要注意，对于未结束的年份（2020年），其“最后一行”数据并非真正的年终数据，因此需要特殊处理。

![](img/eb0f637a18b38045b3181a702dc4f1c8_85.png)

以下是实现步骤：

![](img/eb0f637a18b38045b3181a702dc4f1c8_87.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_89.png)

1.  使用`resample`方法按年份分组数据，并获取每年的最后一行数据。
2.  剔除2020年的数据（因为股票在2020年末并未卖出）。
3.  从结果中提取`open`（开盘价）列。
4.  将每个开盘价乘以1200（因为每年卖出1200股），然后求和，得到已卖出股票的总收入。

![](img/eb0f637a18b38045b3181a702dc4f1c8_91.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_93.png)

```python
# 获取每年最后一个交易日的数据
df_yearly = new_df.resample(‘A‘).last()

![](img/eb0f637a18b38045b3181a702dc4f1c8_95.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_97.png)

# 剔除2020年的数据（切片操作）
df_yearly = df_yearly[:-1]

![](img/eb0f637a18b38045b3181a702dc4f1c8_99.png)

# 计算卖出股票的总收入：每年年底开盘价 * 1200股，然后求和
receive = df_yearly[‘open‘].sum() * 1200
print(f“卖出股票总收入：{receive}“)
```

## 计算剩余股票的价值及总收益

![](img/eb0f637a18b38045b3181a702dc4f1c8_109.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_111.png)

最后，我们来计算总收益。这包括已卖出股票的收入和手中剩余股票的价值。

![](img/eb0f637a18b38045b3181a702dc4f1c8_113.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_115.png)

对于2020年，投资者买入了股票但尚未卖出。我们需要估算这部分股票的价值：
*   **剩余股数**：2020年已过去的月份数（案例中为2个月）* 100股/月 = 200股。
*   **估值单价**：使用最近一个交易日（2020年2月18日）的**收盘价**进行估值。

![](img/eb0f637a18b38045b3181a702dc4f1c8_117.png)

```python
# 计算剩余股票的价值（2020年买入的）
last_price = new_df[‘close‘].iloc[-1]  # 获取最近交易日的收盘价
last_money = last_price * 200  # 剩余200股的价值
print(f“剩余股票估值：{last_money}“)

# 计算总收益
total_profit = receive + last_money - cost
print(f“此策略总收益为：{total_profit}“)
```

![](img/eb0f637a18b38045b3181a702dc4f1c8_119.png)

![](img/eb0f637a18b38045b3181a702dc4f1c8_121.png)

公式总结如下：
**总收益 = 卖出收入 + 剩余股票估值 - 买入成本**

4.  最后，综合已卖出收入、剩余股票估值和总成本，计算出了该投资策略的最终收益。
