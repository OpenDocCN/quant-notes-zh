## 回测收益率

![](img/66c71774d0f5621ffc2d146a42d629b8_16.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_18.png)

回测收益率是评估策略效果最直接的指标。它表示在整个回测周期结束时，你的初始投资最终变成了多少。这个值可能是正的（盈利），也可能是负的（亏损），直观地反映了策略的最终盈亏情况。

![](img/66c71774d0f5621ffc2d146a42d629b8_20.png)

以下是计算回测收益率的逻辑：
```python
# 假设 final_value 是回测结束时的总资产，initial_value 是初始本金
backtest_return = (final_value - initial_value) / initial_value
```
这个指标是分析策略时最重要的一个。设计好一个策略之后，首先需要看这个策略能带来多少最终收益。

## 年化收益率

![](img/66c71774d0f5621ffc2d146a42d629b8_26.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_28.png)

理解了策略的最终收益后，另一个关键问题是：这个策略的赚钱效率如何？这时就需要引入年化收益率的概念。

![](img/66c71774d0f5621ffc2d146a42d629b8_30.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_32.png)

年化收益率将不同时间长度的投资回报标准化为年度比率，便于进行横向比较。例如，一个耗时10年的策略和一个耗时2年的策略，仅比较总收益是不公平的，年化收益能更好地衡量其盈利能力。

![](img/66c71774d0f5621ffc2d146a42d629b8_34.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_36.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_38.png)

年化收益率的计算公式如下：
**年化收益率 = (1 + 总收益率) ^ (250 / N) - 1**

![](img/66c71774d0f5621ffc2d146a42d629b8_40.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_42.png)

公式解析：
*   **总收益率 (R)**：即上文计算出的回测收益率。
*   **250 (M)**：代表一年中大约的交易天数（通常取250或252天）。
*   **N**：代表你回测数据的总交易天数。

![](img/66c71774d0f5621ffc2d146a42d629b8_44.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_46.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_48.png)

以下是使用Python计算年化收益率的示例代码：
```python
# 假设 total_return 是总收益率，N 是回测总天数
annual_return = (1 + total_return) ** (250 / N) - 1
```
在实际分析中，年化收益率通常是比总收益率更受关注的核心指标，它能更公允地反映策略的持续盈利能力。

![](img/66c71774d0f5621ffc2d146a42d629b8_50.png)

![](img/66c71774d0f5621ffc2d146a42d629b8_52.png)

本节课中我们一起学习了金融量化分析中的两个核心收益指标。我们首先了解了**回测收益率**，它直接反映了策略的最终盈亏结果。接着，我们深入探讨了**年化收益率**，它通过标准化时间周期，为我们提供了衡量策略盈利效率的关键视角。掌握这两个指标的计算与解读，是科学评估交易策略的第一步。
