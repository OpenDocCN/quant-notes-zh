## 优化测试与策略逻辑验证

![](img/b26de451fb7378a5b5290a93e274bfc3_38.png)

![](img/b26de451fb7378a5b5290a93e274bfc3_40.png)

为了在相对较好的市场环境中检验策略逻辑，我们将回测时间改为2016年1月4日至2017年1月4日。这一次，策略的回测收益明显高于基准收益，取得了更好的效果。

![](img/b26de451fb7378a5b5290a93e274bfc3_42.png)

![](img/b26de451fb7378a5b5290a93e274bfc3_44.png)

我们的策略核心逻辑是：每月初根据财务指标（如市盈率PE、市净率PB）对股票进行打分，并买入排名靠前的股票。以下是选择排名前十股票的代码逻辑示意：
```python
# 假设 df_ranked 是已经打好分并排序的股票DataFrame
selected_stocks = df_ranked.iloc[0:10]  # 选择前10名
```
这个结果初步验证了“指标越好，未来收益可能越高”的选股逻辑。

![](img/b26de451fb7378a5b5290a93e274bfc3_46.png)

![](img/b26de451fb7378a5b5290a93e274bfc3_48.png)

![](img/b26de451fb7378a5b5290a93e274bfc3_50.png)

为了进行对比实验，我们将选股范围改为排名最差的股票（例如从倒数10名中选股）：
```python
selected_stocks = df_ranked.iloc[-10:]  # 选择后10名（最差的）
```
回测结果显示，选择“差生”组合的收益比基准收益更差。这一正一反的对比实验，有力地证明了我们所用财务指标的有效性。

## 核心总结与策略评价

![](img/b26de451fb7378a5b5290a93e274bfc3_68.png)

![](img/b26de451fb7378a5b5290a93e274bfc3_70.png)

本节课中我们一起学习了如何对一个量化策略进行回测分析与评价。

![](img/b26de451fb7378a5b5290a93e274bfc3_72.png)

![](img/b26de451fb7378a5b5290a93e274bfc3_74.png)

我们验证的**打分法策略**是一种直观且常用的量化选股方法。其核心公式可以概括为：
**最终得分 = 指标A得分 + 指标B得分 + ...**
通过综合多个有效财务指标，筛选出综合评分最高的股票组合。

![](img/b26de451fb7378a5b5290a93e274bfc3_76.png)

总结来说，本次策略分析表明：
1.  策略在多数情况下能产生一定的超额收益，尤其在非极端熊市环境下效果更佳。
2.  通过对比“优等生”组合与“差生”组合，验证了所选底层因子的有效性。
3.  策略表现受整体市场环境影响较大，在熊市中可能表现平平，这是所有股票策略面临的共同挑战。
