在本节课中，我们将要学习如何分析时间序列数据的变化情况。具体来说，我们将计算两种核心指标：**每日差异值**和**每日变化率**，并学习如何对它们进行统计和可视化。这些是金融数据分析中非常基础且重要的操作。

上一节我们介绍了如何加载和查看数据，本节中我们来看看如何计算数据的变化。

## 计算每日差异值

![](img/c4bbd34e217d180c13e6d988684e8738_6.png)

首先，我们需要计算每一天指标相对于前一天的差异值。例如，对于股票价格，我们想知道今天比昨天涨了或跌了多少。在Pandas中，这可以通过一个简单的方法实现。

![](img/c4bbd34e217d180c13e6d988684e8738_8.png)

以下是计算每日差异值的步骤：

```python
# 假设 df 是我们的DataFrame数据
diff = df.diff()
print(diff.head())
```

![](img/c4bbd34e217d180c13e6d988684e8738_10.png)

执行这段代码后，会得到一个新的DataFrame。其中，每一行的值等于当前行（当天）的数据减去前一行（前一天）的数据。对于数据的第一行，由于没有前一天的数据可减，结果会是 `NaN`（空值）。

![](img/c4bbd34e217d180c13e6d988684e8738_12.png)

## 计算每日变化率（增长率）

仅仅知道差异值有时不够直观。例如，从1元涨到2元和从10000元涨到10001元，虽然都涨了1元，但意义完全不同。因此，我们更常使用**变化率**或**增长率**来衡量变化幅度。

变化率的计算公式是：
`变化率 = (后一天的值 - 前一天的值) / 前一天的值`

![](img/c4bbd34e217d180c13e6d988684e8738_14.png)

![](img/c4bbd34e217d180c13e6d988684e8738_16.png)

在Pandas中，我们可以直接调用一个内置函数来计算这个指标。

![](img/c4bbd34e217d180c13e6d988684e8738_18.png)

以下是计算每日变化率的步骤：

```python
# 计算每日变化率（百分比变化）
pct_change = df.pct_change()
print(pct_change.head())
```

![](img/c4bbd34e217d180c13e6d988684e8738_20.png)

计算完成后，我们可以对结果进行格式化，例如保留三位小数，使其更易读。

## 对变化率进行统计分析

![](img/c4bbd34e217d180c13e6d988684e8738_22.png)

得到每日变化率序列后，我们可能希望了解其整体统计特性，例如平均增长率是多少。

![](img/c4bbd34e217d180c13e6d988684e8738_24.png)

以下是计算平均变化率的步骤：

![](img/c4bbd34e217d180c13e6d988684e8738_26.png)

![](img/c4bbd34e217d180c13e6d988684e8738_28.png)

```python
# 计算各列的平均变化率
mean_pct_change = pct_change.mean()
print(mean_pct_change)
```

![](img/c4bbd34e217d180c13e6d988684e8738_30.png)

## 可视化平均变化率

![](img/c4bbd34e217d180c13e6d988684e8738_32.png)

为了更直观地展示各个指标的平均变化率，我们可以将其绘制成条形图。

![](img/c4bbd34e217d180c13e6d988684e8738_34.png)

以下是绘制平均变化率条形图的步骤：

```python
import matplotlib.pyplot as plt

![](img/c4bbd34e217d180c13e6d988684e8738_36.png)

# 绘制条形图
mean_pct_change.plot(kind='bar', figsize=(16, 8))
plt.title('各指标平均变化率')
plt.show()
```

![](img/c4bbd34e217d180c13e6d988684e8738_38.png)

通过这个图表，我们可以一目了然地看到哪个指标在观察期内的平均增长表现最好或最差。
