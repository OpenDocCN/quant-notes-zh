## 什么是策略评估指标？
当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以判断策略的盈亏情况。然而，“赚了”或“赔了”的描述过于笼统。我们需要更精确的指标来回答以下问题：
*   策略具体赚了多少？
*   与大盘相比，策略有何优势？
*   执行该策略会面临哪些风险？这些风险是否值得承担？
*   与其他策略相比，本策略的优势在哪里？

这些问题的答案，就构成了策略的评估指标。在后续课程中，我们将使用专业的策略回测平台。现在，我们先通过Python代码来深入理解这些基础指标的含义和计算方法。

## 核心指标：回测收益率
在众多评估指标中，回测收益率是最基本且最重要的一个。它衡量的是策略在整个回测周期内，最终为初始本金带来的回报比例。

![](img/f8378cdb65fc384311ad942625964511_17.png)

![](img/f8378cdb65fc384311ad942625964511_19.png)

![](img/f8378cdb65fc384311ad942625964511_21.png)

例如，一个4.5%的回测收益率意味着，相对于初始投入的本金，策略最终带来了4.5%的增值。

### 计算公式
回测收益率的计算公式如下：

![](img/f8378cdb65fc384311ad942625964511_23.png)

**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

![](img/f8378cdb65fc384311ad942625964511_25.png)

用符号表示为：
**Return = (P_t / P_0) - 1**

![](img/f8378cdb65fc384311ad942625964511_27.png)

其中：
*   **P_t** 代表回测结束时的资产总值（卖出所有持仓后的现金）。
*   **P_0** 代表回测开始时的初始本金。

这个公式的核心思想是计算**最终盈利相对于初始本金的增长比例**。

### 代码实现与可视化
上一节我们介绍了回测收益率的概念和公式，本节中我们来看看如何用Python计算并展示它。我们可以计算每一天相对于起始日的累计收益率，从而观察收益随时间的变化趋势。

以下是计算过程：
1.  计算每日资产净值与初始净值的比值。
2.  将最终比值减1，得到总回测收益率。
3.  可视化整个回测期间的收益率曲线。

![](img/f8378cdb65fc384311ad942625964511_29.png)

![](img/f8378cdb65fc384311ad942625964511_31.png)

```python
# 1. 计算每日净值与初始净值的比值序列
# 假设‘close’列代表每日策略的资产净值
cumulative_return_ratio = data['close'] / data['close'].iloc[0]

![](img/f8378cdb65fc384311ad942625964511_33.png)

![](img/f8378cdb65fc384311ad942625964511_35.png)

# 2. 计算最终的回测收益率
final_return = cumulative_return_ratio.iloc[-1] - 1

![](img/f8378cdb65fc384311ad942625964511_37.png)

![](img/f8378cdb65fc384311ad942625964511_39.png)

# 3. 将结果整理成DataFrame以便查看
result_df = pd.DataFrame({
    '回测收益率': [final_return]
}, index=['最终结果'])
print(result_df)

![](img/f8378cdb65fc384311ad942625964511_41.png)

# 4. 绘制收益率变化曲线
plt.figure(figsize=(10, 6))
plt.plot(cumulative_return_ratio.index, cumulative_return_ratio.values - 1, label='累计收益率曲线')
plt.axhline(y=0, color='r', linestyle='--', linewidth=1, label='盈亏平衡线')
plt.title('策略回测收益率走势图')
plt.xlabel('日期')
plt.ylabel('收益率')
plt.legend()
plt.grid(True)
plt.show()
```
执行代码后，我们会得到最终的回测收益率数值，并看到一张图表。这张图显示了从开始到结束，策略累计收益率是如何随时间演变的。曲线在零轴上方表示盈利，下方表示亏损。
