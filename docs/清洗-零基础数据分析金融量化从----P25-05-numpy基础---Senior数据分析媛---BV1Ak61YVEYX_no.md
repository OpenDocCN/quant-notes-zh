## 数组访问的基本原理

计算机中的数组在内存中是顺序存储的。数组索引（如0, 1, 2, 3...）实际上表示的是每个元素相对于数组起始内存地址的**偏移量**。这就是为什么编程语言中的数组索引通常从0开始。

例如，一个存储整数的数组，每个整数占4个字节。第一个元素的地址偏移量为`0`，第二个元素的地址偏移量为`4`，第三个为`8`，以此类推。索引值`i`对应的元素地址可以通过公式计算：
`元素地址 = 首地址 + i * 元素字节大小`

这种基于偏移量的访问方式高效且直接，是数组数据结构的核心。

## 一维数组的访问

一维数组的访问方式与Python列表类似，使用中括号`[]`和索引值。

```python
import numpy as np
L1 = [‘a‘, ‘b‘, ‘c‘, ‘d‘, ‘e‘] # Python列表
N1 = np.array(L1) # 转换为NumPy数组

# 访问第一个元素
print(L1[0]) # 输出: ‘a‘
print(N1[0]) # 输出: ‘a‘
```

## 多维数组的访问

对于多维数组，NumPy提供了更直观的访问方式。

### 直接访问与间接访问

我们首先创建一个二维数组作为示例：
```python
N2 = np.random.randint(0, 100, size=(5, 5))
print(N2)
```
假设我们想访问第二行第三列的元素（索引为`[1, 2]`）。

**间接访问**（不推荐）：
这种方式先获取行，再获取列，生成了一个中间对象。
```python
row = N2[1] # 先获取第二行，这是一个一维数组
element = row[2] # 再获取该行的第三个元素
```

**直接访问**（推荐）：
使用逗号分隔的索引元组，直接定位到目标元素，没有中间环节，效率更高且更安全。
```python
element = N2[1, 2] # 直接访问第二行第三列的元素
```
**原则**：进行只读操作时，两种方式均可；但进行赋值或修改操作时，务必使用直接访问方式。

### 行访问与列访问

以下是访问行和列的方法：

*   **访问单行**：`N2[1]` 或 `N2[1, :]`
*   **访问单列**：`N2[:, 1]`
*   **访问多行**：`N2[[1, 3]]` 或 `N2[1:4, :]` (切片访问第1,2,3行)
*   **访问多列**：`N2[:, [0, 2]]` 或 `N2[:, 1:3]` (切片访问第1,2列)
*   **访问子矩阵**：`N2[1:3, 1:3]` 访问一个2x2的子区域。

### 使用列表索引访问多个不连续元素

我们可以传递一个索引列表来访问多个不连续位置的元素。

一维数组示例：
```python
N1 = np.array([‘a‘, ‘b‘, ‘c‘, ‘d‘, ‘e‘])
indices = [1, 3, 0] # 指定要访问的索引位置
print(N1[indices]) # 输出: [‘b‘ ‘d‘ ‘a‘]
```

二维数组示例（访问`(1,0)`和`(2,2)`两个位置的元素）：
```python
row_indices = [1, 2]
col_indices = [0, 2]
# 注意：这里row_indices和col_indices会配对，即访问(1,0)和(2,2)
print(N2[row_indices, col_indices])
```

### 使用布尔列表（掩码）进行访问

布尔列表（掩码）可以用于条件筛选，`True`对应的位置将被选中。

```python
mask = [True, False, True, False, False] # 长度必须与数组在该维度上的长度一致
print(N1[mask]) # 输出N1中第0个和第2个元素: [‘a‘ ‘c‘]
```

对于二维数组：
```python
# 行掩码：选择第0行和第2行
row_mask = [True, False, True, False, False]
print(N2[row_mask])

# 列掩码：选择所有行的第1列和第3列
col_mask = [False, True, False, True, False]
print(N2[:, col_mask])
```
**注意**：布尔列表的长度必须与所要访问的**维度长度**严格匹配。

## 数组的切片操作

切片用于访问数组中连续的一部分。语法为`start:stop:step`，遵循**左闭右开**原则。

一维数组切片：
```python
N1 = np.random.randint(0, 100, 10)
print(N1[2:5]) # 访问索引2到4的元素
print(N1[::2]) # 每隔一个元素取一个，步长为2
print(N1[::-1]) # 反转数组
```

## 数组的变形（Reshape）

`reshape`函数可以改变数组的形状而不改变其数据。新形状的元素总数必须与原数组一致。

```python
N2 = np.random.randint(0, 100, size=(4,5)) # 4行5列，共20个元素
print(N2.shape) # 输出: (4, 5)

# 变为10行2列
N2_reshaped = N2.reshape((10, 2))
print(N2_reshaped.shape) # 输出: (10, 2)

# 变为一行（1行20列）
N2_row = N2.reshape((1, -1)) # -1 表示自动计算该维度大小
print(N2_row.shape) # 输出: (1, 20)

# 变为一列（20行1列）
N2_col = N2.reshape((-1, 1))
print(N2_col.shape) # 输出: (20, 1)
```
**关键点**：使用`-1`可以让NumPy自动计算该维度的合适大小，这在处理高维数据时非常方便。

## 访问机制总结

以下是NumPy数组访问的核心要点总结：

1.  **基础访问**：NumPy数组支持与Python列表类似的索引访问，但对于高维数组，建议使用直接访问方式`array[dim1_idx, dim2_idx, ...]`。
2.  **行列访问**：
    *   行访问：`array[row_idx]` 或 `array[row_idx, :]`
    *   列访问：`array[:, col_idx]`
3.  **多元素访问**：使用索引列表`array[[idx1, idx2, ...]]`可以访问多个不连续的元素。
4.  **条件访问**：使用布尔掩码`array[bool_mask]`可以根据条件筛选元素。务必确保掩码长度与对应维度匹配。
5.  **切片访问**：使用`start:stop:step`语法获取连续子集，支持步长和反向切片。
6.  **数组变形**：使用`reshape()`函数改变数组维度，元素总数需保持不变。可用`-1`自动推算某一维度大小。
