## 增长率累加的问题

![](img/ad8d63463837348a628ea4705c02ad3c_4.png)

对于连续时间序列，我们能否将每天的增长率简单累加，作为总体的增长率呢？答案是否定的。

以下是一个例子来说明这个问题：
*   第一天股价：100元
*   第二天股价：50元
*   第三天股价：75元

计算过程如下：
*   第二天增长率 = (50 - 100) / 100 = -0.5
*   第三天增长率 = (75 - 50) / 50 = 0.5
*   增长率累加和 = (-0.5) + 0.5 = 0

累加结果显示总增长率为0，意味着股票“没涨没跌”。但观察原始数据，股价从100元变为75元，实际上是下跌的。这说明**简单的增长率加法无法正确反映整体变化**。

![](img/ad8d63463837348a628ea4705c02ad3c_6.png)

![](img/ad8d63463837348a628ea4705c02ad3c_8.png)

## 解决方案：引入对数

为了正确计算连续时间序列的累积变化，我们需要借助数学中的对数公式。

![](img/ad8d63463837348a628ea4705c02ad3c_10.png)

核心思想是：将对数形式的每日变化率进行累加，其数学本质等同于计算首日与末日的整体比例关系。

**公式推导如下：**

![](img/ad8d63463837348a628ea4705c02ad3c_12.png)

假设我们有三天的股价：`P1`, `P2`, `P3`。
使用对数计算每日变化并累加：
`log(P2/P1) + log(P3/P2) = log( (P2/P1) * (P3/P2) ) = log(P3/P1)`

![](img/ad8d63463837348a628ea4705c02ad3c_14.png)

![](img/ad8d63463837348a628ea4705c02ad3c_16.png)

![](img/ad8d63463837348a628ea4705c02ad3c_18.png)

这个结果 `log(P3/P1)` 恰恰反映了**第三天股价相对于第一天股价的整体变化**。通过对数累加，我们绕过了直接加法带来的问题，直接得到了期初与期末的关系。

![](img/ad8d63463837348a628ea4705c02ad3c_20.png)

形象地说，这解决了“最初投入的一元钱，经过一系列涨跌后，相当于最终多少钱”的问题。

## 代码实现

![](img/ad8d63463837348a628ea4705c02ad3c_28.png)

![](img/ad8d63463837348a628ea4705c02ad3c_30.png)

以下是使用Python和Pandas计算连续累积收益率的步骤。

### 第一步：计算每日对数收益率

![](img/ad8d63463837348a628ea4705c02ad3c_32.png)

![](img/ad8d63463837348a628ea4705c02ad3c_34.png)

首先，我们需要计算后一天股价除以前一天股价的对数值。

![](img/ad8d63463837348a628ea4705c02ad3c_36.png)

```python
# 假设 data 是一个包含股价的Pandas Series或DataFrame列
# 使用 shift(1) 将数据向上移动一格，得到“前一天”的数据
shifted_data = data.shift(1)

![](img/ad8d63463837348a628ea4705c02ad3c_38.png)

# 计算每日的对数收益率：log(当天价格 / 前一天价格)
log_returns = np.log(data / shifted_data)

![](img/ad8d63463837348a628ea4705c02ad3c_40.png)

# 查看结果（保留几位小数）
print(log_returns.head().round(4))
```

![](img/ad8d63463837348a628ea4705c02ad3c_42.png)

代码解释：
*   `data.shift(1)` 生成一个序列，其中每个值都是原数据中**前一个交易日**的值。
*   `np.log(data / shifted_data)` 计算每日的对数收益率。
*   结果 `log_returns` 中的每个值，代表了从上一日到当前日的连续增长率。

![](img/ad8d63463837348a628ea4705c02ad3c_44.png)

![](img/ad8d63463837348a628ea4705c02ad3c_46.png)

### 第二步：计算累积收益率

接下来，我们需要将对数收益率累加起来，以反映从起点到任意时间点的总变化。

![](img/ad8d63463837348a628ea4705c02ad3c_48.png)

![](img/ad8d63463837348a628ea4705c02ad3c_50.png)

```python
# 使用 cumsum() 函数进行累加
cumulative_log_returns = log_returns.cumsum()

# 将对数累积收益率还原为实际的价值倍数（即“1元钱变成了多少元”）
cumulative_returns = np.exp(cumulative_log_returns)

![](img/ad8d63463837348a628ea4705c02ad3c_52.png)

![](img/ad8d63463837348a628ea4705c02ad3c_54.png)

# 绘制结果图
cumulative_returns.plot(figsize=(16, 8))
plt.title('累积收益走势 (以期初1元为单位)')
plt.ylabel('价值倍数')
plt.xlabel('日期')
plt.show()
```

![](img/ad8d63463837348a628ea4705c02ad3c_56.png)

代码解释：
*   `.cumsum()` 是累积求和函数。例如，输入 `[a, b, c]`，输出为 `[a, a+b, a+b+c]`。
*   对 `log_returns` 使用 `.cumsum()`，得到 `cumulative_log_returns`，其最后一个值就是 `log(最终价格/最初价格)`。
*   `np.exp()` 是 `np.log()` 的逆运算，用于将对数值还原为实际的比例倍数。
*   最终 `cumulative_returns` 序列的图表显示：假设在第一天投资1元（或1个单位），到后续每一天，这笔投资相当于多少元。

![](img/ad8d63463837348a628ea4705c02ad3c_58.png)

## 本章要点总结

![](img/ad8d63463837348a628ea4705c02ad3c_60.png)

本节课中我们一起学习了金融时间序列中连续变化情况的分析方法：

![](img/ad8d63463837348a628ea4705c02ad3c_62.png)

![](img/ad8d63463837348a628ea4705c02ad3c_64.png)

1.  **问题识别**：简单的增长率**不能**直接累加来衡量整体变化，因为它会扭曲最终结果。
2.  **数学工具**：引入**对数**是解决该问题的关键。通过对数变换，每日变化率的累加等价于计算期初与期末的整体比例。
    *   **核心公式**：`log(P_t / P_{t-1})` 的累加 = `log(P_end / P_start)`
3.  **代码实现**：
    *   使用 `data.shift(1)` 获取前一期数据。
    *   使用 `np.log(data / data.shift(1))` 计算每日对数收益率。
    *   使用 `.cumsum()` 计算累积对数收益率。
    *   使用 `np.exp()` 将累积对数收益率还原为实际的价值累积倍数。
4.  **结果解读**：最终得到的图表直观展示了“期初1单位投资在任意时间点的价值”，是量化分析中评估资产长期表现的重要指标。
