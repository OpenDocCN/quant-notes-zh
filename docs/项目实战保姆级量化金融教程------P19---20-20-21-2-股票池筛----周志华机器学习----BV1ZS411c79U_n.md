## 数据预处理与查询
在每日交易开始前，我们需要对股票池进行预处理和筛选。这涉及到查询股票的财务数据。

![](img/023f6a5fe66cd6a1d16566c2d53a396b_9.png)

以下是实现每日预处理的关键步骤：

1.  **导入查询模块**：我们需要使用平台提供的研究函数来查询财务数据。
2.  **构建查询语句**：查询语句（Query）用于指定我们想要获取的数据字段。
3.  **设置过滤条件**：过滤条件（Filter）用于限定查询的股票范围，这里我们限定在初始的沪深300股票池内。
4.  **排序与限制**：对查询结果进行排序，并限制返回的股票数量，例如只保留营收最高的10只股票。

## 实现每日筛选逻辑
我们将筛选逻辑写在 `before_trading` 函数中，确保在每个交易日开始前执行。

![](img/023f6a5fe66cd6a1d16566c2d53a396b_11.png)

以下是 `before_trading` 函数中筛选逻辑的代码实现：

![](img/023f6a5fe66cd6a1d16566c2d53a396b_13.png)

![](img/023f6a5fe66cd6a1d16566c2d53a396b_15.png)

![](img/023f6a5fe66cd6a1d16566c2d53a396b_17.png)

```python
def before_trading(context):
    # 使用query对象查询财务数据，这里查询营业总收入指标
    q = query(
        fundamentals.financial_indicator.operating_revenue
    ).filter(
        fundamentals.eod_derivative_indicator.code.in_(get_index_stocks('沪深300'))
    ).order_by(
        fundamentals.financial_indicator.operating_revenue.desc() # 按营收降序排列
    ).limit(10) # 只取前10名

    # 执行查询，获取结果DataFrame
    df = get_fundamentals(q)

    # 将筛选出的股票列表存入context，供后续交易函数使用
    context.stock_list = df.columns.values

    # 打印查看结果（实际运行时可以注释掉）
    print(df)
```

**代码解释**：
*   `fundamentals.financial_indicator.operating_revenue` 是查询营业总收入的字段。
*   `get_index_stocks(‘沪深300’)` 用于获取沪深300指数的成分股代码列表。
*   `order_by(…desc())` 表示按指定字段降序排列。
*   `limit(10)` 限制返回结果的数量为10条。
*   `get_fundamentals(q)` 是执行查询的函数，返回一个DataFrame。
