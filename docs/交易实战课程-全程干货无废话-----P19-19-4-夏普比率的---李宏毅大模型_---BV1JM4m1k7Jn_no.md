## 夏普比率的定义与作用

上一节我们介绍了投资回报率，本节中我们来看看如何衡量风险与收益的关系。夏普比率描述的是：对于投资者承担的每一单位风险，能够获得多少超额收益。

![](img/ea4c7aeaf1767773d15ed641721be66b_1.png)

我们可以用一个生活中的例子来理解：假设一份工作（如高风险地区的雇佣兵）日薪非常高，但其风险也极高。我们关心的是，为了获得这份高薪，所承担的巨大风险是否“值得”。夏普比率就是用来量化这个“值不值”的指标。

![](img/ea4c7aeaf1767773d15ed641721be66b_3.png)

**夏普比率的核心公式**如下：
`夏普比率 = (投资组合预期收益率 - 无风险收益率) / 投资组合收益率的标准差`

![](img/ea4c7aeaf1767773d15ed641721be66b_5.png)

指标越高，意味着在承担相同风险的情况下，获得的超额收益越高。因此，在比较多个投资选项时，我们通常倾向于选择夏普比率更高的那个。

## 夏普比率的计算逻辑

理解了夏普比率的作用后，我们来具体看看它的计算逻辑。它主要比较两个部分：一是你的投资组合带来的超额收益，二是该投资组合的波动性（即风险）。

*   **分子部分（超额收益）**：用你的投资组合收益率减去一个“无风险收益率”。无风险收益率可以理解为几乎没有任何损失本金风险的投资所能获得的回报，例如国债或银行保本理财的利率。这部分差值代表了你的投资策略超越市场无风险基准的收益。
*   **分母部分（风险）**：使用投资组合收益率的标准差来衡量。标准差越大，说明收益的波动越大，即风险越高。

所以，整个公式衡量的是 **“每承担一单位风险，能换来多少超额回报”**。

## 在Python中计算夏普比率

理论部分我们已经清晰，现在让我们动手在Python中实现夏普比率的计算。我们将基于之前计算好的股票日收益率数据来进行。

![](img/ea4c7aeaf1767773d15ed641721be66b_7.png)

首先，我们需要准备好数据。在计算收益率时，我们通常需要对数据中的缺失值进行处理，例如使用前一天的收益率进行填充，以保证计算的连续性。

![](img/ea4c7aeaf1767773d15ed641721be66b_9.png)

以下是计算夏普比率的关键步骤代码：

```python
import numpy as np
import pandas as pd

# 假设 returns 是一个Pandas Series或DataFrame，包含了投资组合的日收益率序列
# 假设无风险年化收益率为 5%
risk_free_rate = 0.05
# 将年化无风险利率转换为日利率。假设一年有250个交易日
daily_risk_free_rate = risk_free_rate / 250

# 计算超额收益（日收益率 - 日无风险利率）
excess_returns = returns - daily_risk_free_rate

# 计算夏普比率
# 公式： (超额收益的均值 / 超额收益的标准差) * sqrt(年化因子)
# 年化因子通常取252（美股）或250（A股），这里以250为例
sharpe_ratio = (excess_returns.mean() / excess_returns.std()) * np.sqrt(250)

![](img/ea4c7aeaf1767773d15ed641721be66b_11.png)

print(sharpe_ratio)
```

![](img/ea4c7aeaf1767773d15ed641721be66b_13.png)

运行以上代码后，我们会得到各个股票或投资组合的夏普比率。结果解读非常简单：数值越大越好。正值表示承担风险获得了正回报，负值则意味着承担风险反而导致了亏损。在我们的示例中，“中兴”股票的夏普比率最高，表明它在承担相同风险的情况下，能为我们带来最高的超额收益。
