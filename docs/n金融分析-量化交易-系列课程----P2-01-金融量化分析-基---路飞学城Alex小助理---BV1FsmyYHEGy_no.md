## 什么是金融？

上一节我们介绍了课程的整体目标，本节中我们来看看金融的基本定义。

从定义上讲，金融是对现有资源进行重新整合之后，实现价值和利润的等效流通。这个概念可能有些抽象。在日常生活中，金融常被理解为与钱打交道的行为，目的是让钱增值。但金融并非完全是不劳而获的投机行为，它对国家、企业和个人都有积极作用。

举个例子：假设一位有闲置资金的投资者，遇到一位有优秀创业点子但缺乏资金的创业者。投资者通过购买创业者公司的股票，将资金提供给创业者。几年后公司成功上市，价值增长，创业者获得了发展资金，投资者也获得了投资回报。这个过程促进了经济发展，实现了双赢，这就是金融活动的一种体现。

## 常见的金融工具

理解了金融的基本概念后，我们来看看实现这些金融活动的具体载体，即金融工具。金融工具是在金融市场中可交易的金融资产。

以下是几种常见的金融工具：

*   **股票**：代表对一家公司的所有权份额。购买股票即成为该公司的股东，可分享公司成长带来的收益（如股价上涨、分红），也需承担相应的风险。这是我们本课程后续重点讨论的核心。
*   **期货**：一种标准化合约，约定在未来某一特定时间和地点，以特定价格交割一定数量的某种资产（如大宗商品、金融资产）。买卖双方基于对资产未来价格走势的不同判断进行交易，用于套期保值或投机。
    *   **核心机制**：`期货合约 = 未来交割约定`
*   **黄金**：一种传统的贵金属投资品，常被视为避险资产和保值工具。其价格受全球供需、货币政策、地缘政治等因素影响，但长期来看相对稳定。
*   **外汇**：指不同国家货币之间的兑换交易。投资者通过预测汇率波动来赚取差价。汇率波动受两国经济状况、利率政策、国际贸易等因素影响。通常波动较小，个人投资者参与较少。
*   **基金**：由基金公司汇集众多投资者的资金，交由专业的基金经理进行统一管理和投资（可能投资于股票、债券、期货等多种资产）。适合缺乏专业知识和时间的个人投资者。
    *   **运作模式**：`投资者资金 -> 基金公司 -> 基金经理管理 -> 投资组合 -> 收益/风险 -> 返还投资者`

## 重点解析：股票

在众多金融工具中，股票是金融市场中最常见、最核心的投资工具之一，也是我们量化分析的主要对象。

股票代表持有者对一家上市公司的部分所有权。购买公司股票，就意味着成为了该公司的股东。股东的权利通常包括分享公司利润（通过分红）和对公司重大事项的投票权。股票的价格在公开市场上实时波动，受公司业绩、行业前景、宏观经济、市场情绪等多种因素影响。

投资者通过低买高卖股票来获取资本利得，或通过长期持有获得分红收益。但同时，如果公司经营不善或市场下行，股价下跌，投资者也会面临亏损的风险。因此，对股票进行分析和估值至关重要，而量化分析正是利用计算机和数学模型来辅助这一过程的有力工具。
