# 机器学习：P58：1-KMEANS算法概述 🎯

![](img/eb066a2cb31c15bd56944393b5f0078e_0.png)

在本节课中，我们将要学习机器学习中一个非常重要的分支——聚类。我们将从聚类的基本概念入手，重点介绍最经典、最实用的K-Means算法，并详细讲解其核心原理、优化目标以及应用中的关键点。

## 什么是聚类？🤔

上一节我们介绍了有监督学习，其特点是数据带有标签。本节中我们来看看聚类，它属于无监督学习问题。

聚类要解决的问题与分类有相似之处，但核心区别在于：我们手中没有每个数据点所属类别的标签。聚类算法的目标，是根据数据本身的特征，将相似的数据点自动划分到同一组（称为“簇”）中。

例如，观察右侧的散点图。在原始数据集中，这些点并没有颜色标记。聚类算法会根据一种相似度度量方式，将位置相近、特征相似的点归为一类。最终，我们可能得到三个不同的簇，分别用绿色、红色和蓝色表示。

## 聚类的挑战 ⚠️

聚类算法原理看似简单，但在实际应用中面临一些挑战，主要体现在评估和调参上。

![](img/eb066a2cb31c15bd56944393b5f0078e_2.png)

在有监督学习中，我们可以通过预测值与真实标签的对比，使用准确率、F1值等指标进行评估和模型选择。然而，在无监督的聚类任务中，由于没有“标准答案”，评估聚类结果的好坏以及比较不同参数下的聚类效果，都变得相对困难。

![](img/eb066a2cb31c15bd56944393b5f0078e_4.png)

## K-Means算法核心概念 🧠

![](img/eb066a2cb31c15bd56944393b5f0078e_6.png)

接下来，我们将介绍聚类中最经典、应用最广的K-Means算法。

### 1. 指定K值

![](img/eb066a2cb31c15bd56944393b5f0078e_8.png)

K-Means算法需要我们预先指定一个关键参数：**K值**。K值代表我们希望将数据聚合成多少个簇。

![](img/eb066a2cb31c15bd56944393b5f0078e_10.png)

*   **K=3**：算法会将所有数据点聚合成3个簇。
*   **K=4**：算法会将所有数据点聚合成4个簇。

![](img/eb066a2cb31c15bd56944393b5f0078e_12.png)

### 2. 质心

![](img/eb066a2cb31c15bd56944393b5f0078e_14.png)

每个簇都有一个中心点，称为**质心**。质心是该簇内所有数据点在各个特征维度上的平均值。

对于一个二维数据点（X, Y），其所在簇的质心坐标计算方式为：
`质心坐标 = (该簇所有点的X坐标平均值, 该簇所有点的Y坐标平均值)`

### 3. 距离度量

聚类依据“物以类聚”的原则，将相似的点分在一起。如何衡量两个点是否相似？最常用的方法是计算它们之间的**距离**。距离越近，相似度越高。

在K-Means中，最常使用的是**欧式距离**。对于两个点 `p1(x1, y1)` 和 `p2(x2, y2)`，其欧式距离计算公式为：
`距离 = sqrt((x1 - x2)^2 + (y1 - y2)^2)`

**重要提示**：在使用距离度量前，通常需要对数据进行**标准化**处理。这是因为如果不同特征（如X轴和Y轴）的数值范围差异巨大，数值范围大的特征会在距离计算中占据主导地位，从而影响聚类效果。标准化（如归一化到[0,1]区间）可以确保所有特征在计算中具有同等的重要性。

![](img/eb066a2cb31c15bd56944393b5f0078e_16.png)

### 4. 优化目标

![](img/eb066a2cb31c15bd56944393b5f0078e_18.png)

与大多数机器学习算法一样，K-Means通过优化一个目标函数来求解。它的优化目标是：**最小化每个簇中所有样本点到其质心的距离之和**。

用公式可以表示为：
`最小化 J = Σ（i=1 到 K） Σ（x 属于簇Ci） || x - μi ||^2`
其中：
*   `K` 是簇的个数。
*   `Ci` 代表第 `i` 个簇。
*   `μi` 是第 `i` 个簇的质心。
*   `|| x - μi ||` 是样本点 `x` 到其质心 `μi` 的欧式距离。

![](img/eb066a2cb31c15bd56944393b5f0078e_20.png)

![](img/eb066a2cb31c15bd56944393b5f0078e_22.png)

这个目标函数的直观理解是：我们希望每个簇内部的点都尽可能紧密地围绕在质心周围。如果一个点离当前簇的质心很远，但离另一个簇的质心更近，那么在算法迭代过程中，它就会被重新分配到更近的那个簇中去，从而优化整体目标。

## 总结 📝

本节课中我们一起学习了聚类的基本概念以及K-Means算法的核心原理。

![](img/eb066a2cb31c15bd56944393b5f0078e_24.png)

我们了解到：
1.  聚类是一种无监督学习方法，目标是将相似的数据自动分组。
2.  K-Means算法需要预先指定簇的数量K。
3.  算法通过计算数据点到**质心**的**距离**来判断相似性，常用欧式距离，且计算前需做数据标准化。
4.  算法的优化目标是**最小化所有样本点到其所属簇质心的距离平方和**，通过迭代不断调整数据点归属和质心位置来实现。

![](img/eb066a2cb31c15bd56944393b5f0078e_26.png)

K-Means算法以其简单、高效的特点，成为聚类任务中最基础和常用的工具之一。
