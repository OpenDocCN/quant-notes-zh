在本节课中，我们将对Pandas的核心数据结构之一——Series进行总结。我们将回顾其核心特性、关键操作以及处理数据时可能遇到的问题和解决方案。

## 概述

![](img/941c49632f9f76eefcc38c425e42e3d9_3.png)

Series是Pandas库中一维的、带标签的数组。它结合了Python字典和NumPy数组的特性，是进行金融数据分析和量化交易的基础数据结构。本节将系统性地总结Series的核心概念与操作方法。

## Series的核心特性

上一节我们介绍了Series的基本操作，本节中我们来看看它的核心特性总结。

Series是字典与数组的集合体。它支持通过整数下标（如数组）和自定义标签（如字典）两种方式进行数据访问与操作。

![](img/941c49632f9f76eefcc38c425e42e3d9_5.png)

以下是Series作为“数组+字典”集合体的具体表现：

![](img/941c49632f9f76eefcc38c425e42e3d9_7.png)

*   **类数组操作**：支持按下标进行索引、切片、布尔值索引。支持两个Series之间或一个Series与一个标量之间的加减乘除等运算。
*   **类字典操作**：支持按标签进行索引，也支持使用`in`操作符检查标签是否存在。

![](img/941c49632f9f76eefcc38c425e42e3d9_9.png)

## 整数索引的歧义与解决方案

![](img/941c49632f9f76eefcc38c425e42e3d9_11.png)

当Series的索引（`index`）也是整数时，使用中括号`[]`进行索引可能会产生歧义：它可能被解释为按下标索引，也可能被解释为按标签索引。

为了解决这个问题，Pandas提供了`.iloc`和`.loc`这两个属性来明确索引方式：

*   **`.iloc`**：用于**下标位置**索引（integer-location）。例如，`series.iloc[0]`获取第一个元素。
*   **`.loc`**：用于**标签**索引（label-location）。例如，如果索引标签是`[5, 10, 15]`，那么`series.loc[10]`获取标签为`10`的元素。

![](img/941c49632f9f76eefcc38c425e42e3d9_13.png)

## 数据对齐与缺失值

![](img/941c49632f9f76eefcc38c425e42e3d9_15.png)

当两个Series对象进行运算（如加减乘除）时，Pandas会按照**标签自动对齐**数据，然后对标签相同的值进行计算。

![](img/941c49632f9f76eefcc38c425e42e3d9_17.png)

如果某个标签只存在于其中一个Series中，那么在结果Series中，该标签对应的值将是缺失值，表示为`NaN`（Not a Number）。

![](img/941c49632f9f76eefcc38c425e42e3d9_19.png)

## 缺失值数据处理

![](img/941c49632f9f76eefcc38c425e42e3d9_21.png)

在数据分析中，处理缺失值（`NaN`）是一个常见步骤。Pandas提供了两种主要方法：

![](img/941c49632f9f76eefcc38c425e42e3d9_23.png)

以下是处理缺失数据的两种基本方法：

![](img/941c49632f9f76eefcc38c425e42e3d9_25.png)

1.  **删除缺失值**：使用`.dropna()`函数直接移除包含`NaN`的行。
    ```python
    series_without_na = series.dropna()
    ```
2.  **填充缺失值**：使用`.fillna(value)`函数，将`NaN`替换为指定的值（如`0`或平均值）。
    ```python
    series_filled = series.fillna(0) # 用0填充所有NaN
    ```

## 总结

![](img/941c49632f9f76eefcc38c425e42e3d9_27.png)

本节课中我们一起学习了Pandas Series数据结构的核心要点。我们明确了Series是结合了数组和字典特性的带标签一维数组。我们理解了使用`.iloc`和`.loc`来清晰区分按位置和按标签索引的重要性。我们掌握了数据按标签自动对齐的规则，以及由此可能产生的缺失值`NaN`。最后，我们学会了使用`.dropna()`和`.fillna()`来处理这些缺失数据。Series的其他功能，如布尔索引和花式索引，其理念均继承自NumPy，这为后续学习提供了良好基础。
