# Python金融分析与量化交易实战教程：P50：1-回归问题概述 📈

在本节课中，我们将要学习机器学习中的一个核心概念——回归问题。我们将从基本定义出发，解释什么是回归问题，并通过一个具体的银行贷款额度预测例子，来理解回归问题的数据构成、目标以及核心的数学表达形式。

---

## 什么是回归问题？🤔

在机器学习的有监督学习中，主要存在两种类型的问题：分类问题和回归问题。

分类问题很简单。例如，你去银行申请贷款，银行最终会给出一个明确的决定：贷款给你（类别“1”）或不贷款给你（类别“0”）。分类算法的输出是几个明确的类别。

那么，什么叫做回归问题呢？同样是去银行，但这次我们不问“能不能贷”，而是问“能贷多少钱”。这个“多少钱”是一个具体的数字，它可能是一个取值范围，例如从0到10万之间的任何值，比如3万6200、4万八千九、9万7600。所有可能性都存在。也就是说，我们需要在一定范围内找出一个可能的值。这类问题就叫做回归问题。

我们今天要讲解的就是一个回归问题。

---

## 回归问题的数据形式 📊

上一节我们介绍了回归问题的基本概念，本节中我们来看看一个回归任务中数据通常长什么样。

以下是数据表格的一个示例：

| 工资 (X1) | 年龄 (X2) | 额度 (Y) |
| :--- | :--- | :--- |
| ... | ... | ... |

在这个表格中：
*   **工资** 是我们数据的第一列，记作 **X1**。
*   **年龄** 是第二列，记作 **X2**。
*   **额度** 是第三列，记作 **Y**。

Y 是什么意思呢？我们刚才提到，线性回归是一种有监督算法。作为有监督算法，我们在训练模型时，必须有一个**标签**或者说 **Y 值**。这个 Y 值表示模型在训练过程中参考的目标指标。

按照当前这份数据，工资 X1 和年龄 X2 表示在建模时我要参考的两个指标。而目标，即回归方程要做出的结果，就是通过 X1 和 X2 去预测最终可能的额度 Y 等于多少。

*   **X1, X2**：是我要利用的**特征**。
*   **Y**：是我最终想要得到的**结果**（标签）。

这就是数据集必须呈现的需求。当做回归任务时，你至少需要两个东西：**特征 (X)** 和 **标签 (Y)**。

---

## 建立回归方程 🧮

我们再来想一想，现在无非就是要预测 Y 的值到底等于 2 万、7 万还是其他具体数字。这个值跟什么有关呢？在数据中，只给出了 X1 和 X2。Y 值肯定既跟 X1 有关，也跟 X2 有关。

那么，我可能想建立这样一个方程：`Y = X1 + X2`。直接相加好像不行。为什么？直接相加相当于 X1 的系数是 1，X2 的系数也是 1，这表示工资和年龄对最终结果的影响是一样的。

但考虑一下，工资和年龄，哪个对贷款额度的影响更大？相对来说应该是工资。年龄的影响可能更小。所以，我可以为它们分别加上权重系数。

以下是更合理的方程形式：

`Y = θ1 * X1 + θ2 * X2`

在这里：
*   **θ1** 是与 X1（工资）组合的权重。
*   **θ2** 是与 X2（年龄）组合的权重。

![](img/6f8b2535ba164621cc725cb99f71f191_1.png)

例如，假设 θ1 = 3，θ2 = 1，那就表示工资的影响是年龄的 3 倍。建立回归方程，本质上就是要找出最合适的 θ1 和 θ2。

![](img/6f8b2535ba164621cc725cb99f71f191_3.png)

在回归方程中，X1、X2 和 Y 在训练数据中都是已知的。我们还差的东西就是 **θ1** 和 **θ2**。因此，整个线性回归的核心目标就是：**求解出最合适的 θ1 和 θ2**。这件事解决了，问题就解决了。

---

## 线性方程的拟合问题 📉

上一节我们明确了目标是求解权重 θ，本节中我们来看看求解时会遇到的一个关键问题。

考虑一下，我们构建的回归方程 `Y = θ1 * X1 + θ2 * X2` 能否完美覆盖所有的数据点？当前方程是线性的。对于一个线性方程，它无法做出复杂的弯曲，它表示的是一个平面（在二维特征是直线）。

因此我们遇到了一个问题：**一个线性平面无法完美拟合所有的数据点**。

既然无法满足所有点，那我们应该怎么办？我们应该让这个平面**尽可能多地拟合当前的数据**。也就是说，在众多可能的线性平面中（例如画在上面或下面的平面），我们应该选择那个**拟合数据点最多的平面**，或者说，让数据点到这个平面的“距离”总体最小。

这个要求是合理的。线性方程虽然不能满足所有点，但它应该尽可能好地去拟合数据。

---

## 总结 ✨

本节课中我们一起学习了回归问题的核心内容：
1.  **定义**：回归问题是预测一个连续数值的问题，与输出离散类别的分类问题不同。
2.  **数据**：回归任务需要**特征 (X)** 和**连续值标签 (Y)**。
3.  **目标**：建立形如 `Y = θ1 * X1 + θ2 * X2 + ...` 的线性方程，核心是找到最优的权重参数 **θ**。
4.  **挑战**：线性模型无法穿过所有数据点，因此我们的优化目标是找到一条直线（或平面），使其**尽可能好地拟合所有数据点**，即整体误差最小。

![](img/6f8b2535ba164621cc725cb99f71f191_5.png)

这为我们接下来学习如何求解最优的 θ 参数（即模型训练）打下了基础。
