## 索引方法概述

与`Series`类似，`DataFrame`也可以通过中括号`[]`进行索引。但需要注意的是，直接使用中括号时，默认先选择列，再选择行，这与二维数组先选行后选列的习惯不同。

例如，对于一个`DataFrame`对象`df`，`df[‘列名’]`会先选择列，然后可以在此基础上选择行。然而，当行索引是整数时，这种方式容易产生混淆和错误。

因此，更推荐使用`.loc`和`.iloc`属性进行索引，它们能更清晰地区分标签索引和位置索引。

## 使用.loc和.iloc属性

以下是使用`.loc`和`.iloc`的正确方法：

*   **`.loc`**：基于**标签**进行索引。
*   **`.iloc`**：基于**整数位置**进行索引。

两者的通用语法是：`df.loc[行选择器, 列选择器]` 或 `df.iloc[行选择器, 列选择器]`。选择器之间用逗号隔开，左边是行，右边是列。

### 获取单个值

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_1.png)

要获取标签为`’A’`的行、列名为`’one’`的单元格的值（即1.0），推荐写法如下：

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_3.png)

```python
# 使用 .loc 通过标签获取
value = df.loc[‘A‘, ‘one‘]
```

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_5.png)

### 获取整行或整列数据

如果想获取某一整行或整列的数据，可以配合切片使用。

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_7.png)

获取标签为`’A’`的整行数据：
```python
# 行选择‘A‘，列使用‘:‘切片表示选择所有列
row_a = df.loc[‘A‘, :]
```

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_8.png)

获取列名为`’two’`的整列数据：
```python
# 行使用‘:‘切片表示选择所有行，列选择‘two‘
col_two = df.loc[:, ‘two‘]
```

## 灵活的索引组合

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_10.png)

`.loc`和`.iloc`的强大之处在于，行和列的选择器可以灵活组合，包括：
*   **单个标签/位置**：如 `‘A‘` 或 `0`。
*   **标签/位置列表（花式索引）**：如 `[‘A‘, ‘C‘]` 或 `[0, 2]`。
*   **标签/位置切片**：如 `‘A‘:‘C‘` 或 `0:2`。
*   **布尔数组**：用于条件筛选。

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_12.png)

例如，你可以进行如下混合索引：
```python
# 选择‘A‘和‘C‘两行，以及‘one‘到‘two‘的列（包含‘two‘）
subset = df.loc[[‘A‘, ‘C‘], ‘one‘:‘two‘]
```

这意味着你可以非常方便且灵活地获取`DataFrame`中任意子集的数据。

## 总结

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_14.png)

本节课中我们一起学习了`DataFrame`的核心索引方法。我们了解到：
1.  应优先使用`.loc`（基于标签）和`.iloc`（基于整数位置）属性进行索引，而非连续的中括号。
2.  索引的基本格式为`df.loc[行选择器, 列选择器]`，行选择在前，列选择在后。
3.  选择器可以是单个值、列表、切片或布尔数组，并且可以自由组合，以满足复杂的数据选取需求。

![](img/b7c3cad6a1581ef633b48c0d674ea5e4_16.png)

掌握这些索引技巧是高效进行数据清洗、分析和可视化的关键一步。
