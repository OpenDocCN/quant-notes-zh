## 数组形状变换（Reshape）

上一节我们介绍了数组的索引与切片，本节中我们来看看如何改变数组的形状。变形操作指的是在不改变数组数据的前提下，重新排列数组的维度。

例如，我们有一个原始的二维数组 `arr`，其形状为五行六列。

```python
import numpy as np
arr = np.random.rand(5, 6)  # 创建一个5行6列的二维数组
print(arr.shape)  # 输出: (5, 6)
```

我们可以使用 `reshape` 方法将其变形。例如，将这个二维数组变形为一维数组。原始数组共有30个元素（5*6），因此变形后的一维数组也必须恰好有30个位置。

```python
arr_1d = arr.reshape(30)  # 将二维数组变形为有30个元素的一维数组
print(arr_1d.shape)  # 输出: (30,)
```

反之，一维数组也可以变形为多维数组，只要变形后容器的总元素数量不变即可。

```python
arr_1 = np.arange(30)  # 创建一个有30个元素的一维数组
arr_2d_1 = arr_1.reshape(2, 15)  # 变形为2行15列的二维数组
arr_2d_2 = arr_1.reshape(6, 5)   # 变形为6行5列的二维数组
```

**核心概念**：变形操作 `array.reshape(new_shape)` 要求新形状的总元素数 `np.prod(new_shape)` 必须等于原数组的总元素数。

## 数组拼接（Concatenate）

接下来，我们学习如何将多个数组拼接在一起。拼接操作指的是将多个NumPy数组沿指定轴向（横向或纵向）连接起来。

以下是拼接操作的基本步骤：

1.  使用 `np.concatenate()` 函数。
2.  将要拼接的数组放入一个元组中作为第一个参数。
3.  通过 `axis` 参数指定拼接方向。
    *   `axis=0`：沿第一个轴（行方向，纵向）拼接。
    *   `axis=1`：沿第二个轴（列方向，横向）拼接。

```python
# 创建两个相同的二维数组
arr_a = np.array([[1, 2], [3, 4]])
arr_b = np.array([[5, 6], [7, 8]])

# 纵向拼接（行数增加）
result_v = np.concatenate((arr_a, arr_b), axis=0)
# 结果: [[1, 2], [3, 4], [5, 6], [7, 8]]

# 横向拼接（列数增加）
result_h = np.concatenate((arr_a, arr_b), axis=1)
# 结果: [[1, 2, 5, 6], [3, 4, 7, 8]]
```

**注意**：进行拼接的数组必须具有相同的维度，并且除拼接轴外，其他维度的长度必须一致。拼接操作在图像处理（如制作拼图）和数据合并中非常有用。

## 聚合操作（Aggregation）

聚合操作是指对数组中的一组值执行计算，返回一个单一的汇总值。NumPy提供了多种聚合函数。

以下是常用的聚合函数及其用法：

*   `sum()`: 计算所有元素的和。
*   `min()`: 找出所有元素中的最小值。
*   `max()`: 找出所有元素中的最大值。
*   `mean()`: 计算所有元素的平均值。

这些函数都可以通过 `axis` 参数指定计算方向，从而得到每行或每列的聚合结果。

```python
arr = np.array([[1, 2, 3], [4, 5, 6]])

print(arr.sum())          # 输出所有元素的和: 21
print(arr.sum(axis=0))    # 输出每列的和: [5 7 9]
print(arr.max(axis=1))    # 输出每行的最大值: [3 6]
```

## 数学与统计函数

除了基本聚合，NumPy还包含丰富的数学和统计函数。

### 通用数学函数

这些函数对数组中的每个元素独立进行计算。

```python
arr = np.array([0, np.pi/2, np.pi])
print(np.sin(arr))  # 计算正弦值: [0.0000000e+00 1.0000000e+00 1.2246468e-16]
print(np.cos(arr))  # 计算余弦值
print(np.round(3.14159, 2))  # 四舍五入到小数点后两位: 3.14
```

### 统计函数

统计函数用于分析数据的分布特征，其中**方差**和**标准差**尤为重要。

*   **方差（Variance）**: 衡量数据点与平均值之间的离散程度。公式为：
    `方差 = Σ(xi - μ)² / N`
    其中 `xi` 是每个数据点，`μ` 是平均值，`N` 是数据点总数。
*   **标准差（Standard Deviation）**: 方差的平方根。它和方差表示的含义相同，但单位与原始数据一致。公式为：
    `标准差 = sqrt(方差)`

```python
data = np.array([85, 90, 78, 92, 88, 76, 95, 89])

variance = np.var(data)  # 计算方差
std_dev = np.std(data)   # 计算标准差

print(f"方差: {variance:.2f}")
print(f"标准差: {std_dev:.2f}")
```

**核心概念**：标准差和方差是衡量数据波动性或分散程度的关键指标。在金融中，它们常用来度量资产价格或收益的风险。

## 矩阵运算

最后，我们探讨NumPy中的矩阵运算。矩阵是二维数组的一种特殊形式，在线性代数和机器学习中广泛应用。

### 矩阵创建与转置

我们可以创建特殊的矩阵，如单位矩阵，也可以对任何二维数组进行转置操作（行变列，列变行）。

```python
# 创建一个3x3的单位矩阵
identity_matrix = np.eye(3)
# 输出:
# [[1. 0. 0.]
#  [0. 1. 0.]
#  [0. 0. 1.]]

![](img/4c159ef356a5305464503a4c21f76b3c_3.png)

# 创建一个普通矩阵并转置
matrix = np.array([[1, 2], [3, 4], [5, 6]])
matrix_transpose = matrix.T
# 转置前 shape: (3, 2)
# 转置后 shape: (2, 3)，内容为 [[1, 3, 5], [2, 4, 6]]
```

### 矩阵乘法

矩阵乘法是核心操作，规则不同于数组的逐元素乘法。使用 `np.dot()` 或 `@` 运算符进行计算。

**计算规则**：对于矩阵A（m×n）和矩阵B（n×p），结果矩阵C（m×p）中的元素 `C[i, j]` 是A的第i行与B的第j列对应元素乘积之和。

![](img/4c159ef356a5305464503a4c21f76b3c_5.png)

```python
A = np.array([[2, 1], [4, 3]])  # 2x2 矩阵
B = np.array([[1, 2], [1, 0]])  # 2x2 矩阵

# 矩阵乘法
C = np.dot(A, B)  # 或者使用 C = A @ B
print(C)
# 计算过程：
# C[0,0] = 2*1 + 1*1 = 3
# C[0,1] = 2*2 + 1*0 = 4
# C[1,0] = 4*1 + 3*1 = 7
# C[1,1] = 4*2 + 3*0 = 8
# 输出: [[3 4], [7 8]]
```

![](img/4c159ef356a5305464503a4c21f76b3c_7.png)

**核心概念**：矩阵乘法 `np.dot(A, B)` 要求矩阵A的列数等于矩阵B的行数。
