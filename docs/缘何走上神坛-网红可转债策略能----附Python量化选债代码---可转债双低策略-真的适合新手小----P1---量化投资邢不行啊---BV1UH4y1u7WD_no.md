## 什么是可转债？

上一节我们介绍了课程的背景，本节中我们来看看策略的核心基础——可转债。

![](img/e3051d03b58b537f656ecd743ee60490_3.png)

![](img/e3051d03b58b537f656ecd743ee60490_5.png)

可转债，全称为可转换公司债券，本质上是一种企业债券。但它与传统债券不同，持有人有权在未来的某个时间点，按照事先约定的价格，将债券转换为发行公司的股票。

![](img/e3051d03b58b537f656ecd743ee60490_7.png)

以下是可转债与传统债券的主要区别：

*   **本质**：可转债是**债券**，但附带了转股期权。
*   **收益**：作为债券，它提供固定的利息收益。作为期权，它允许投资者分享公司股价上涨的收益。
*   **风险**：债券属性提供了下跌保护（有债底），股票属性则提供了上涨弹性。

其核心概念可以用一个简单的公式来描述转股价值：
`转股价值 = (100 / 转股价) * 正股股价`
其中，`100`代表可转债的面值（通常为100元），`转股价`是约定的转换价格，`正股股价`是发行公司股票的当前市场价格。

![](img/e3051d03b58b537f656ecd743ee60490_9.png)

## 可转债的“双低策略”逻辑

![](img/e3051d03b58b537f656ecd743ee60490_11.png)

![](img/e3051d03b58b537f656ecd743ee60490_13.png)

理解了可转债的基本概念后，本节我们来看看基于其特性设计的“双低策略”。

![](img/e3051d03b58b537f656ecd743ee60490_15.png)

![](img/e3051d03b58b537f656ecd743ee60490_17.png)

该策略的核心思想是寻找价格低且转股溢价率低的可转债进行投资。这两个指标“双低”，意味着债券的防守性（价格接近债底）和进攻性（转股溢价率低，股性强）都比较好。

![](img/e3051d03b58b537f656ecd743ee60490_19.png)

![](img/e3051d03b58b537f656ecd743ee60490_21.png)

策略的具体操作步骤如下：

![](img/e3051d03b58b537f656ecd743ee60490_23.png)

1.  **计算指标**：对市场上所有可转债，计算其“双低值”。
    `双低值 = 可转债价格 + 转股溢价率 * 100`
    （注：转股溢价率通常以百分比表示，乘以100是为了与价格在数值上匹配进行排序。）

![](img/e3051d03b58b537f656ecd743ee60490_25.png)

2.  **筛选排序**：删除正股处于ST、退市或停牌状态的可转债。将剩余可转债按“双低值”从小到大排序。

![](img/e3051d03b58b537f656ecd743ee60490_27.png)

3.  **构建组合**：买入“双低值”最小的N只可转债（例如5只），并平均分配资金。

![](img/e3051d03b58b537f656ecd743ee60490_29.png)

![](img/e3051d03b58b537f656ecd743ee60490_31.png)

4.  **定期调仓**：以周为单位进行调仓。在每周最后一个交易日收盘后执行步骤1-3，选出新的组合，并在下一周的第一个交易日开盘时换仓。

![](img/e3051d03b58b537f656ecd743ee60490_33.png)

## 策略的优化思路

![](img/e3051d03b58b537f656ecd743ee60490_51.png)

面对基础策略的回撤，我们能否进行优化呢？本节提供一些基于原策略的改进思路。

“双低策略”的逻辑基础是扎实的，我们可以在此基础上增加其他筛选维度来优化。以下是一些可行的优化方向：

![](img/e3051d03b58b537f656ecd743ee60490_53.png)

*   **结合剩余期限**：在距离转股期结束时间较长的可转债中（例如剩余年限最长的50只里），挑选双低值最小的标的。这可以避免即将到期的转债因时间价值衰减带来的不确定性。
*   **结合规模因子**：同时考虑“双低值”和“转债余额”或“流通规模”，规避规模过小、容易被资金操纵的个券。
*   **动态调整权重**：不采用等权配置，而是根据双低值、波动率或其他指标动态分配买入权重。
*   **设置止损止盈**：为策略加入简单的止损和止盈规则，以控制单次亏损和锁定部分利润。

这些优化思路的核心，都是在原有“双低”的股债平衡逻辑上，增加一层风险控制或收益增强的过滤器。投资者可以根据自己的理解进行测试和组合。
