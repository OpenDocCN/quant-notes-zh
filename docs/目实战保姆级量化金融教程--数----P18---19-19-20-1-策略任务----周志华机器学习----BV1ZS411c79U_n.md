## 创建策略文件

![](img/41d60838a2cedc45d3ce400c4c4ca00f_6.png)

![](img/41d60838a2cedc45d3ce400c4c4ca00f_2.png)

首先，我们需要在平台中创建一个新的策略。点击“新建策略”按钮，在弹出的对话框中为策略命名。例如，这是我们的第一个示例，可以命名为 `simple_demo`。策略类型选择股票交易，点击确定。

![](img/41d60838a2cedc45d3ce400c4c4ca00f_4.png)

确定后，平台会打开一个代码编辑界面。这是我们编写策略代码的地方。平台通常会提供一个包含几个核心函数模块的模板，例如：
*   `initialize`：初始化函数。
*   `before_trading`：每日开盘前运行的函数。
*   `handle_bar`：处理每个交易Bar（如每分钟、每日）数据的函数。

我们的工作就是在这几个模块中填充具体的策略逻辑。

## 策略目标与需求分析

![](img/41d60838a2cedc45d3ce400c4c4ca00f_6.png)

接下来，我们明确本次策略的任务需求。我们将构建一个基于沪深300指数的选股策略。

![](img/41d60838a2cedc45d3ce400c4c4ca00f_8.png)

核心思路是：从沪深300指数的成分股中，始终持有表现最好的10只股票。随着市场变化，这10只“最好”的股票会动态更新。我们的持仓也需要随之调整，卖出不再属于“最好”行列的股票，买入新进入“最好”行列的股票。

为了实现这个目标，我们需要完成以下步骤：
1.  获取沪深300指数的所有成分股，作为我们的股票池。
2.  定义一个评价股票“好坏”的标准（例如，某个财务指标）。
3.  每天根据这个标准，对股票池中的所有股票进行排序。
4.  选出排名前10的股票。
5.  调整我们的持仓，使其与这10只股票保持一致。

## 策略逻辑分解与模块规划

![](img/41d60838a2cedc45d3ce400c4c4ca00f_8.png)

现在，我们将上述步骤分配到策略模板的不同函数模块中。

**1. 初始化函数 `initialize`**
在策略初始化时，我们需要获取并设定股票池。因此，在 `initialize` 函数中，我们要完成获取沪深300指数成分股的任务。
```python
def initialize(context):
    # 获取沪深300指数成分股，并设置为策略的股票池
    stock_pool = get_index_stocks(‘000300.XSHG‘)
    set_universe(stock_pool)
```

**2. 盘前处理函数 `before_trading`**
这个函数在每个交易日开盘前运行。所有与当日交易决策相关、但不涉及实际下单的数据查询和处理工作，都应放在这里。
对于我们的策略，需要在这里完成：
*   查询股票池中所有股票的选定财务指标（例如，盈利数据）。
*   根据该指标对股票进行排序。
*   选出排名前10的股票代码，并保存下来，供交易函数使用。
```python
def before_trading(context):
    # 获取股票池
    stock_list = context.universe
    # 查询财务数据，例如：净利润（net_profit）
    fundamental_data = get_fundamentals(query(valuation.code, income.net_profit), stock_list)
    # 按净利润降序排序
    sorted_stocks = fundamental_data.sort_values(by=‘net_profit‘, ascending=False)
    # 选取前10只股票代码
    context.top_10_stocks = sorted_stocks[‘code‘].iloc[:10].tolist()
```

**3. 交易处理函数 `handle_bar`**
这个函数在每次Bar数据更新时（例如每日收盘）运行，负责执行实际的交易逻辑。
在这里，我们需要：
*   获取当前账户的持仓情况。
*   将当前持仓与 `before_trading` 中计算出的 `context.top_10_stocks`（目标持仓）进行比较。
*   卖出目标持仓中没有的股票。
*   买入我们尚未持有的目标持仓股票。
```python
def handle_bar(context, bar_dict):
    current_holdings = list(context.portfolio.positions.keys())
    target_holdings = context.top_10_stocks

    # 卖出逻辑：当前持有但不在目标列表中的股票
    stocks_to_sell = [s for s in current_holdings if s not in target_holdings]
    for stock in stocks_to_sell:
        order_target_value(stock, 0) # 将该股票持仓调整为0

    # 买入逻辑：在目标列表中但当前未持有的股票
    # 假设将资金平均分配到10只目标股票上
    cash_per_stock = context.portfolio.cash / len(target_holdings)
    stocks_to_buy = [s for s in target_holdings if s not in current_holdings]
    for stock in stocks_to_buy:
        order_value(stock, cash_per_stock) # 买入价值为cash_per_stock的股票
```

## 总结

![](img/41d60838a2cedc45d3ce400c4c4ca00f_12.png)

本节课我们一起学习了如何分析并规划一个量化交易策略。我们以“始终持有沪深300中财务指标最好的10只股票”为例，将策略分解为三个部分：
1.  **初始化 (`initialize`)**：设定策略的股票池。
2.  **盘前处理 (`before_trading`)**：进行数据获取、计算和选股，确定当日的目标持仓。
3.  **交易处理 (`handle_bar`)**：比较当前持仓与目标持仓，执行买卖订单以调整至目标状态。
