## 策略核心思想：波动率聚集效应 📈

上一节我们介绍了CTA策略的基本概念，本节中我们来看看一个具体的实战策略。这个策略的核心思想来源于金融学中的“波动率聚集效应”。

**波动率聚集效应**指的是：如果市场在一段时间内波动剧烈，那么接下来一段时间内，市场很可能继续保持较大的波动；反之，如果市场波动平缓，那么接下来也可能继续平缓。简单来说，历史的波动率在一定程度上可以预测未来的波动率。

基于此，当价格在短时间内波动达到一个由历史波动率定义的阈值时，我们认为这种大幅波动可能已经完成，价格将向均值回归。这就是我们策略盈利的逻辑基础。

## 策略规则详解 ⚙️

理解了策略思想后，我们来看看具体的交易规则。这是一个应用于外汇市场（欧元/日元）的高频策略，观察周期为5分钟。

### 关键指标计算

首先，我们需要计算一个衡量历史波动的关键指标——`Range`。

*   **Range（波动范围）**：取过去一段时间（例如10分钟）内的最高价与最低价之差。
    *   公式：`Range = 最高价(过去N期) - 最低价(过去N期)`

这个`Range`代表了上一个周期的波动幅度。

### 开仓、止盈与止损条件

以下是策略的三个核心价格点位：

1.  **开仓价（Long Open Price）**：
    *   当下一个5分钟K线的**开盘价**向下跌破特定价位时，触发开多仓。
    *   计算公式：`Long_Open_Price = 下一K线开盘价 - K1 * Range`
    *   **逻辑**：价格从开盘价向下已经波动了`K1 * Range`的幅度，基于波动率聚集效应，我们认为向下的波动可能已大部分完成，因此在此位置开多仓，博取价格反弹。

2.  **止盈价（Stop Win Price）**：
    *   开仓后，价格上涨到此价位则平仓获利。
    *   计算公式：`Stop_Win_Price = 开仓价 + K2 * Range`
    *   **逻辑**：价格向上反弹的幅度同样与历史波动范围`Range`相关。

3.  **止损价（Stop Loss Price）**：
    *   开仓后，价格继续下跌到此价位则平仓止损。
    *   这是一个**绝对止损**价，计算公式：`Stop_Loss_Price = 开仓价 - 固定止损点数`

**参数说明**：`K1`和`K2`是需要优化的参数，决定了开仓和止盈的敏感度。`Range`的计算周期（如10分钟）和策略运行周期（5分钟）也是经过参数寻优确定的。

## 策略执行流程与伪代码 🤖

为了更清晰地展示策略的运行逻辑，我们使用伪代码（用中文描述的程序逻辑）来演示。这是一种逐行计算持仓和信号的方法，适用于复杂策略。

**策略状态初始化**：初始持仓`holding = 0`（无持仓）。

以下是每一步的逻辑判断：

```
如果 当前没有持仓 (holding == 0):
    检查是否满足开仓条件：
        如果 当前K线的最低价 <= Long_Open_Price：
            执行开仓操作：
                1. 设置 holding = 1
                2. 记录开仓价
                3. 根据公式计算当前的止盈价和止损价
            检查开仓后是否立即触发止损：
                如果触发，则：
                    计算止损收益
                    设置 holding = 0
                否则：
                    以当前K线收盘价计算当期持仓收益
    否则（未满足开仓条件）：
        当期策略收益 = 0

如果 当前持有仓位 (holding == 1):
    先检查是否触发止损条件：
        如果触发，则：
            计算止损收益
            设置 holding = 0
    再检查是否触发止盈条件：
        如果触发，则：
            计算止盈收益
            设置 holding = 0
    如果以上均未触发：
        以当前K线收盘价计算当期持仓收益
```

**关键细节**：在持仓状态下，我们**先检查止损，再检查止盈**。这是因为在回测中，我们采用保守假设。如果同一根K线内同时触及止盈和止损价，我们统一按止损处理。这样回测出的收益更严格，如果策略依然表现良好，则说明其健壮性更强。

## 其他CTA策略思路拓展 💡

除了上述均值回归型策略，CTA策略还有很多其他思路。这里再简要介绍一种联动型策略。

**策略思想**：外盘大宗商品的剧烈波动会影响国内相关股票的价格。例如，当国际黄金价格大幅上涨时，A股市场的黄金股票也可能上涨。

**策略规则**：
1.  **开仓**：当外盘黄金期货价格向上突破过去N周期的最高价时，于次日开盘买入A股黄金板块股票。
2.  **平仓**：采用移动止损法，即从买入后的最高点回撤超过一定比例时平仓。

这个策略体现了跨市场联动的思想，将商品交易（CTA）与股票交易结合起来。
