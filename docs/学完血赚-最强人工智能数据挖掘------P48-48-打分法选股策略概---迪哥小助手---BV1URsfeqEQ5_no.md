# 人工智能数据挖掘实战：P48：打分法选股策略概述

## 📖 课程概述
在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分选股法。我们将了解其核心思想、实施步骤以及背后的逻辑，为后续的实战操作打下基础。

## 🎯 策略核心思想
上一节我们介绍了因子分析的基本概念，本节中我们来看看如何综合运用多个因子进行选股。

因子打分选股的核心思想是：**不依赖单一因子，而是综合多个因子对股票进行综合评价**。这类似于学生的总成绩由各科成绩加总而成。具体而言，我们会为每只股票的每个因子计算一个得分，然后将所有因子的得分汇总，得到一个总分。最后，根据总分对所有股票进行排名，选取排名靠前的股票构建投资组合。

以下是该策略的直观表示：
假设我们有300只股票和4个因子（A, B, C, D），我们需要为每只股票计算一个总分。
```
股票1: 因子A得分 + 因子B得分 + 因子C得分 + 因子D得分 = 总分1
股票2: 因子A得分 + 因子B得分 + 因子C得分 + 因子D得分 = 总分2
...
股票300: 因子A得分 + 因子B得分 + 因子D得分 = 总分300
```
然后对所有总分进行排序，选取排名前10或前20的股票。

## 🔑 关键前提条件
在实施打分策略前，有一个至关重要的前提条件需要明确。

我们需要预先知道**每个因子与未来收益率的相关性方向**。也就是说，对于某个特定因子，我们需要判断它是“越大越好”还是“越小越好”。

![](img/bd5bcdeef9d5798c422cf274466cb46c_1.png)

例如：
*   **每股收益（EPS）**：通常认为越大越好。
*   **市盈率（PE）**：在某些估值模型中，可能认为越小越好（代表更便宜）。

这个判断依据可以来源于：
1.  金融领域的先验知识和研究报告。
2.  我们自己通过历史数据计算出的因子IC值（信息系数）等指标。

在后续的实战中，我们将选择一组已知相关性方向的因子，一组是“越大越好”，另一组是“越小越好”。这是策略能够正确执行的基础。

## 📝 本节总结
本节课中我们一起学习了因子打分选股策略的概述。我们了解到，该策略通过综合多个因子对股票进行打分和排序，以此选出综合表现最优的股票组合。同时，我们强调了实施该策略的关键前提：必须明确每个因子与预期收益的相关性方向（正相关或负相关）。在下一节中，我们将进入实战环节，学习如何在平台上具体实现这一策略。
