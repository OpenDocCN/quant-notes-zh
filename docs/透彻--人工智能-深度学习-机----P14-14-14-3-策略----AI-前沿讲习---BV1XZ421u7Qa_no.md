# Python量化交易：14：策略收益效果分析 📈

在本节课中，我们将学习如何计算和比较量化交易策略的收益。我们将通过一个具体的例子，分析在应用“双均线策略”后，投资收益与不采取任何策略（即简单持有）相比有何变化。核心在于理解如何计算每日报酬率，并利用策略信号调整这些报酬率，最终通过累加和对数还原得到总收益。

上一节我们介绍了如何生成交易信号，本节中我们来看看如何基于这些信号计算策略的最终收益。

首先，我们需要计算股价的每日报酬率。报酬率反映了资产价格每日的相对变化，是计算收益的基础。计算公式如下：

**报酬率 = (当日股价 / 前一日股价) - 1**

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_5.png)

在代码中，我们使用Pandas的`shift`方法来获取前一日的数据：

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_7.png)

```python
returns = data['close'] / data['close'].shift(1) - 1
```

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_9.png)

执行上述代码后，我们得到了一个代表每日增长率的`returns`序列。

---

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_11.png)

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_13.png)

接下来，我们将策略信号应用到报酬率的计算中。我们的“双均线策略”会生成一个名为`position`的信号序列，其中：
*   `position = 1` 表示策略判断与市场走势一致，我们跟随市场。
*   `position = -1` 表示策略判断市场走势将反转，我们采取与市场相反的操作。

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_15.png)

因此，策略的每日收益应该是策略信号与市场报酬率的乘积。同时，需要注意信号的使用也应进行`shift(1)`操作，以确保使用前一日收盘后生成的信号来指导今日的交易。

策略收益的计算公式如下：

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_17.png)

**策略每日收益 = 前一日策略信号 * 当日市场报酬率**

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_19.png)

对应的代码如下：

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_21.png)

```python
strategy_returns = position.shift(1) * returns
```

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_23.png)

执行这段代码后，`strategy_returns`就代表了应用我们的策略后，每日的实际收益率。

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_25.png)

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_27.png)

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_29.png)

为了对比策略效果，我们需要计算从起始点到结束点的累计总收益。由于每日收益是相乘关系（复利），直接累加并不正确。我们之前引入对数收益率是为了便于计算，现在需要将其还原。

以下是计算累计总收益的步骤：
1.  对策略每日收益序列和原始每日收益序列分别求和，得到对数空间下的总增长。
2.  使用指数函数 `exp()` 将对数总增长还原为实际的累计总收益倍数。

具体操作如下，我们需要先处理可能存在的缺失值（NaN），然后进行计算：

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_31.png)

```python
# 处理缺失值
returns = returns.dropna()
strategy_returns = strategy_returns.dropna()

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_33.png)

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_35.png)

# 计算累计总收益（还原过程）
cumulative_returns = np.exp(returns.sum())
cumulative_strategy_returns = np.exp(strategy_returns.sum())
```

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_37.png)

执行计算后，我们可以查看结果。例如，输出可能显示：
*   原始持有策略的累计收益：`cumulative_returns` ≈ 4.0
*   双均线策略的累计收益：`cumulative_strategy_returns` ≈ 5.8

![](img/c5a57f88b81f4cf3e46936dc3c13d9e3_39.png)

这个结果意味着，如果初始投入1元钱，在不采取策略的情况下最终变为约4元，而在应用我们的双均线策略后，最终变为约5.8元，策略带来了额外的收益提升。

本节课中我们一起学习了量化策略收益分析的核心流程。我们首先计算了市场的每日报酬率，然后利用策略生成的交易信号调整这些报酬率，得到策略的每日收益。最后，通过求和与指数还原，我们计算并比较了原始持有策略与应用双均线策略后的累计总收益，直观地展示了策略的有效性。这个过程是量化交易中评估策略性能的基础环节。
