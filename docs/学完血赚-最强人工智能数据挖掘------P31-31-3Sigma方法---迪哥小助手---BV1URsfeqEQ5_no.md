# 人工智能数据挖掘实战：P31：3-Sigma方法实例 📊

在本节课中，我们将要学习一种在数据预处理中常用的去极值方法——3-Sigma方法。该方法基于统计学中的正态分布原理，能够有效地识别并处理数据中的异常值。

## 正态分布与极值概念 📈

上一节我们介绍了去极值的概念，本节中我们来看看基于统计学的3-Sigma方法。理解该方法需要先了解正态分布。

正态分布，也称为高斯分布，是一种常见的连续概率分布。如果理解起来比较困难，可以将其视为一种“正常”的数据分布模式。

为了便于理解，可以举一个例子：假设你去银行贷款，银行多借给你3块或5块钱是正常情况。但如果多借或少借你10万或8万，这就属于不正常的情况了。

在标准正态分布图中，中心位置是均值。均值代表数据的平均水平。例如，银行贷款的误差可能偏向多借（右侧）或少借（左侧）。

![](img/90ae4d243a1dd154e9f518008c778a6c_1.png)

现在观察分布区间。靠近均值的区间（图中红色阴影部分）面积较大，代表这些事件发生的概率较高。当区间向两侧扩展得越远，事件发生的概率就越低。

我们现在要做的是去极值。极值可能落在哪里？它可能落在均值附近吗？均值附近是大概率发生的正常数据。极值通常落在远离均值的地方。离均值越远、偏差越大的数据点，越有可能是离群点。

## 3-Sigma方法原理 🧮

3-Sigma方法正是基于上述思想。在正态分布中，我们定义了一个由均值（μ）和标准差（σ）决定的区间。

下图展示了以均值为中心，向两侧分别扩展1倍、2倍和3倍标准差所得到的区间。

![](img/90ae4d243a1dd154e9f518008c778a6c_1.png)

![](img/90ae4d243a1dd154e9f518008c778a6c_3.png)

观察区间可以发现，一旦选择的区间越大（例如±3σ），数据落在左右极端区域的可能性就越小。

3-Sigma方法的原理如下：假设我们对原始数据做了某种变换，使其理论上服从高斯分布（正态分布）。我们先不讨论如何变换，仅假设它符合正态分布规则。

当我们指定一个正负3倍标准差的区间（即 `[μ - 3σ, μ + 3σ]`）时，这个区间包含了大约99.7%的数据。这个值在统计学中是一个常用指标。

![](img/90ae4d243a1dd154e9f518008c778a6c_5.png)

![](img/90ae4d243a1dd154e9f518008c778a6c_6.png)

于是我们可以设定一个规则：将数据中最正常的99.7%视为有效数据，而将剩下的约0.3%当作极值进行处理和规范。这就叫做3-Sigma方法。它通过计算一个下限和一个上限来界定数据的合理范围。

![](img/90ae4d243a1dd154e9f518008c778a6c_7.png)

在高斯分布中，只需要知道两个值：均值（μ）和标准差（σ），就可以计算出上限和下限。

![](img/90ae4d243a1dd154e9f518008c778a6c_9.png)

*   **下限公式**：`lower_bound = μ - N * σ`
*   **上限公式**：`upper_bound = μ + N * σ`

![](img/90ae4d243a1dd154e9f518008c778a6c_10.png)

其中，N通常取3，代表3倍标准差。

## 3-Sigma方法代码实现 💻

![](img/90ae4d243a1dd154e9f518008c778a6c_12.png)

理解了原理后，我们来看看如何用代码实现3-Sigma方法。实现过程非常简单。

![](img/90ae4d243a1dd154e9f518008c778a6c_13.png)

首先，我们定义一个函数。函数名可以参照之前的命名方式，这里我们命名为处理3-Sigma的方法。参数包括数据（`data`）和倍数`N`（默认为3）。

以下是实现步骤：

![](img/90ae4d243a1dd154e9f518008c778a6c_15.png)

![](img/90ae4d243a1dd154e9f518008c778a6c_16.png)

1.  计算数据列的均值（`mean`）。
2.  计算数据列的标准差（`std`）。
3.  根据公式计算上限和下限。
4.  返回计算得到的下限和上限。

![](img/90ae4d243a1dd154e9f518008c778a6c_17.png)

![](img/90ae4d243a1dd154e9f518008c778a6c_18.png)

以下是具体的Python代码实现：

![](img/90ae4d243a1dd154e9f518008c778a6c_19.png)

```python
def three_sigma(data, n=3):
    """
    使用3-Sigma方法计算数据的正常值范围。
    参数:
        data: 输入的数据序列（例如Pandas Series）。
        n: 标准差的倍数，默认为3。
    返回:
        lower_bound: 正常值范围下限。
        upper_bound: 正常值范围上限。
    """
    # 计算均值和标准差
    mean = data.mean()
    std = data.std()

    # 计算下限和上限
    lower_bound = mean - n * std
    upper_bound = mean + n * std

    return lower_bound, upper_bound
```

代码执行后，我们就得到了基于当前数据和指定N值的正常值范围。

为了更直观地展示，我们可以取N=1（即包含约68.3%的数据）进行演示，这样正常与异常区域的对比会更明显。

```python
# 假设 df[‘column_name‘] 是我们的数据列
lower, upper = three_sigma(df[‘column_name‘], n=1)
print(f"正常值范围下限: {lower:.2f}")
print(f"正常值范围上限: {upper:.2f}")
```

*   **方法本质**：3-Sigma是一种基于数据服从正态分布的假设，通过均值加减若干倍标准差来确定数据合理范围的方法。
*   **核心公式**：关键在于计算下限 `μ - Nσ` 和上限 `μ + Nσ`。
*   **处理逻辑**：去极值不是简单删除数据，而是定义一个合理的上下界，并将所有超出此范围的数据规范到边界上或进行其他处理，使其保留在预设的边界内。
*   **步骤地位**：这是我们在数据预处理“三步走”当中的第一步——数据清洗与去极值。
