# 量化交易教程：P2：股票基本知识与分类 📈

在本节课中，我们将要学习股票的基本概念、核心作用以及常见的分类方式。理解这些基础知识是进行金融量化分析的第一步。

## 股票的定义与作用

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

为了更形象地理解，我们可以设想一个场景：一位创业者需要资金，而投资者看好他的项目。投资者将资金投入公司，公司则向投资者发行股票作为凭证。这代表投资者对公司出资，并因此成为公司的所有者之一。

股票的核心作用主要有两点：

1.  **出资证明与股东权利**：持有股票意味着你是公司的股东。这赋予了你参与公司重大决策的权利，例如在股东大会上投票。
2.  **获取收益**：作为股东，你有两种主要的获利方式：
    *   **公司分红**：当公司盈利时，可能会将部分利润按持股比例分配给股东。
    *   **交易获利**：你可以将持有的股票在公开市场上卖给其他投资者。如果卖出价格高于买入价格，即可获得价差收益。

上一节我们介绍了股票的基本定义和作用，本节中我们来看看公司如何向公众发行股票。

## 什么是上市与IPO

所谓“上市”，是指企业通过证券交易所，首次公开向全社会投资者增发股票以募集资金的过程。

公司不能随意向公众募集资金。它需要达到一定标准，并向证监会等监管机构提出申请。监管机构审核通过后，公司才能将其股票放在证券交易所这个公开市场上进行交易。所有符合条件的投资者都可以买卖这只股票。

公司首次公开发行股票的行为，就叫做 **IPO**。

理解了公司如何进入公开市场后，接下来我们了解一下市场上不同类型的股票。

## 股票的分类

股票可以按照不同的标准进行分类，以下是两种常见的分类方式。

### 按公司业绩分类

根据公司的经营和财务状况，股票通常被分为以下几类：

*   **蓝筹股**：指资本雄厚、信誉优良的大型公司股票。例如我国的**中石油**、**中石化**。它们通常规模巨大，经营稳定。
*   **绩优股**：指业绩优良公司的股票。它们可能规模不及蓝筹股，但盈利能力持续强劲。例如**贵州茅台**。
*   **ST股**：中文名为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记。这是为了提醒投资者该公司存在较高风险。

### 按上市地区分类

根据公司股票上市交易的地理位置和交易货币，可以分为：

*   **A股**：在中国大陆（上海、深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。
*   **H股**：在中国香港上市的公司股票。
*   **N股**：在美国纽约上市的公司股票。
*   **S股**：在新加坡上市的公司股票。

不同的股市有不同的交易规则。对于A股，有两个重要的规则需要了解：

1.  **涨跌幅限制**：A股设有每日价格涨跌幅限制，通常为**10%**。即一只股票当日的价格波动幅度不能超过前一日收盘价的10%。这是为了降低市场波动，保护投资者。
2.  **T+1交易制度**：“T”代表交易日。在A股，投资者**当天买入的股票，必须等到下一个交易日才能卖出**。这被称为“T+1”交割制度，旨在减少市场中的过度投机行为。

相比之下，港股、美股等市场多实行“T+0”制度（当天可买卖），且没有涨跌幅限制。
