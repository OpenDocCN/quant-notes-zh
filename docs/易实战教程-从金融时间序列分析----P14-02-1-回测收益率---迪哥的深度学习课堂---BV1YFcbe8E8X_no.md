## 回测收益率解读

![](img/934c7fc69ca1f4a0368b969d896370a5_7.png)

我们先从最重要的基础指标——回测收益率开始。

![](img/934c7fc69ca1f4a0368b969d896370a5_9.png)

![](img/934c7fc69ca1f4a0368b969d896370a5_5.png)

回测收益率衡量的是，在回测周期内，策略最终获得的收益相对于初始本金的比率。例如，你基于2015年至2020年的某只股票数据设计了一个买卖策略，回测收益率告诉你，在整个回测期结束后，你的本金增值了多少。它是一个百分比数值，例如4.25%，表示你的策略使本金增长了约4.25%。

![](img/934c7fc69ca1f4a0368b969d896370a5_11.png)

![](img/934c7fc69ca1f4a0368b969d896370a5_7.png)

![](img/934c7fc69ca1f4a0368b969d896370a5_13.png)

### 计算公式

![](img/934c7fc69ca1f4a0368b969d896370a5_15.png)

回测收益率的计算公式非常简单：

![](img/934c7fc69ca1f4a0368b969d896370a5_17.png)

**回测收益率 = (期末资产价值 / 期初资产价值) - 1**

![](img/934c7fc69ca1f4a0368b969d896370a5_19.png)

![](img/934c7fc69ca1f4a0368b969d896370a5_21.png)

用公式表示为：
`R = (P_t / P_0) - 1`

其中：
*   `P_t` 代表回测结束时的资产总价值（卖出所有头寸后的总资金）。
*   `P_0` 代表回测开始时的初始本金。

这个指标直观地反映了策略的绝对盈利能力，是我们评估策略表现时最常用的起点。

## Python代码实战计算

![](img/934c7fc69ca1f4a0368b969d896370a5_23.png)

上一节我们介绍了回测收益率的概念和公式，本节中我们来看看如何用Python代码实际计算它。我们将使用一个本地数据文件进行演示。

首先，导入必要的工具包并进行基础设置。

![](img/934c7fc69ca1f4a0368b969d896370a5_25.png)

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便后续图表显示中文
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False
```

![](img/934c7fc69ca1f4a0368b969d896370a5_27.png)

接着，读取本地数据文件。这里假设数据文件的第一列是日期。

```python
# 读取数据，指定第一列为索引（日期）
data = pd.read_csv('your_data.csv', index_col=0)
# 查看数据前几行，了解数据结构
print(data.head())
```

![](img/934c7fc69ca1f4a0368b969d896370a5_11.png)

数据中包含了股票在不同日期的多个指标。我们的目标是计算基于股价（例如收盘价）走势的回测收益率。

![](img/934c7fc69ca1f4a0368b969d896370a5_13.png)

假设我们关注的是收盘价（‘close’列），并以此模拟策略的资产价值变化。

以下是计算并绘制累计收益率曲线及最终回测收益率的步骤：

首先，计算每个时间点相对于期初的资产比值。这相当于公式中的 `(P_t / P_0)`。

![](img/934c7fc69ca1f4a0368b969d896370a5_29.png)

```python
# 假设‘close’列代表我们的资产净值
price_series = data['close']
# 计算每个时点相对于期初的比值
cumulative_return_ratio = price_series / price_series.iloc[0]
# 将结果转换为DataFrame以便后续操作
cumulative_df = pd.DataFrame(cumulative_return_ratio, columns=['累计比值'])
```

![](img/934c7fc69ca1f4a0368b969d896370a5_31.png)

然后，我们可以将这个累计比值的变化可视化，它展示了资产随时间相对增长的过程。

![](img/934c7fc69ca1f4a0368b969d896370a5_33.png)

```python
# 绘制累计收益率曲线
plt.figure(figsize=(10, 6))
cumulative_df['累计比值'].plot()
plt.title('资产累计收益走势图')
plt.ylabel('相对期初的比值')
plt.xlabel('日期')
plt.grid(True)
plt.show()
```

![](img/934c7fc69ca1f4a0368b969d896370a5_35.png)

![](img/934c7fc69ca1f4a0368b969d896370a5_29.png)

![](img/934c7fc69ca1f4a0368b969d896370a5_37.png)

![](img/934c7fc69ca1f4a0368b969d896370a5_39.png)

上图显示，从期初的1开始，资产价值随时间如何变化。接着，根据公式计算最终的回测收益率。

```python
# 计算最终的回测收益率：期末比值 - 1
final_return = cumulative_df['累计比值'].iloc[-1] - 1
# 将结果整理成表格形式展示
result_df = pd.DataFrame([final_return], columns=['回测收益率'], index=['最终结果'])
print(result_df)
```

![](img/934c7fc69ca1f4a0368b969d896370a5_41.png)

![](img/934c7fc69ca1f4a0368b969d896370a5_41.png)

这样，我们就得到了策略在整个回测期间的总收益率。这个数值可以是正的（盈利），也可以是负的（亏损）。在本例的演示数据中，我们得到了一个正收益。

![](img/934c7fc69ca1f4a0368b969d896370a5_43.png)
