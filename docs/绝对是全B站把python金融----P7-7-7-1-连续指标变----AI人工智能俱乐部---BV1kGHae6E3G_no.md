## 概述

上一节我们介绍了如何计算时间序列的简单差异和增长率。本节中我们来看看如何分析一个连续时间序列的累积变化，例如计算股票最初投资的1元钱，在多年后相当于多少钱。这需要一种特殊的数学处理方法。

## 为什么增长率不能直接累加？

我们首先通过一个例子说明，为什么不能将每天的增长率简单相加来得到总体的增长率。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_4.png)

假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

计算每日增长率：
*   第二天增长率 = (50 - 100) / 100 = **-0.5**
*   第三天增长率 = (75 - 50) / 50 = **0.5**

如果简单累加：-0.5 + 0.5 = **0**。这个结果暗示股票从第一天到第三天“没有涨跌”。

但观察实际价格：从100元变为75元，实际上是下跌了。这表明，**简单的增长率累加无法正确反映资产价值的整体变化**。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_6.png)

## 引入对数收益率

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_7.png)

为了正确计算连续时间的累积变化，我们需要借助数学中的对数。核心思想是计算**对数收益率**并进行累加。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_9.png)

对于相邻两天的价格 `P_t`（今天）和 `P_{t-1}`（昨天），对数收益率的计算公式为：

**`log_return = log(P_t / P_{t-1})`**

对数的一个关键性质是：**对数的加法等价于其内部数值的乘法**。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_11.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_13.png)

沿用之前的例子：
*   `log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_15.png)

最终结果 `log(75/100)` 直接反映了**第三天价格与第一天价格的比例关系**。通过对数收益率的累加，我们就能准确计算出任意一天相对于起始点的累积变化。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_17.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_18.png)

形象地说，这解决了“最初投资的1元钱，在经历了市场无数次涨跌后，最终相当于多少钱”的问题。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_19.png)

## 在Python中实现计算

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_23.png)

以下是在Pandas中计算对数收益率及还原累积价值的步骤。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_24.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_25.png)

### 1. 计算每日的对数收益率

首先，我们需要获取“前一天”的数据。Pandas的 `.shift()` 方法可以将数据序列向上或向下移动。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_27.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_28.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_29.png)

```python
# 假设 `data` 是一个包含股价的Pandas Series或DataFrame列
# 计算对数收益率
log_returns = np.log(data / data.shift(1))
```

`data.shift(1)` 将整个序列向上移动一格，使得每个位置的值都变成其“前一天”的值。`data / data.shift(1)` 计算每日价格比，`np.log()` 计算其自然对数。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_31.png)

### 2. 计算累积对数收益率

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_33.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_35.png)

接下来，我们需要将对数收益率从起始点开始累加起来。这可以使用NumPy的 `.cumsum()` 函数（累积求和）轻松完成。

```python
# 计算累积对数收益率
cumulative_log_returns = log_returns.cumsum()
```

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_37.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_38.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_39.png)

`.cumsum()` 函数会返回一个新序列，其中每个位置的值是原序列从开始到该位置所有值的和。

### 3. 还原为实际价值比例

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_41.png)

由于我们之前取了对数，现在需要将其还原为实际的价值倍数。这通过指数函数 `np.exp()` 来实现。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_43.png)

```python
# 将累积对数收益率还原为实际的价值倍数（假设起始点为1单位）
cumulative_value_ratio = np.exp(cumulative_log_returns)
```

`cumulative_value_ratio` 序列中的每个值，就代表了**最初投资的1单位资金，到该时间点所对应的价值**。例如，值为2.5表示最初的1元钱现在值2.5元。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_45.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_46.png)

## 常用操作总结

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_54.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_55.png)

本节课中我们一起学习了金融时间序列分析中的几个核心操作：

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_56.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_57.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_58.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_59.png)

1.  **计算差异值**：使用 `.diff()` 方法直接计算相邻数据的差值。
2.  **计算简单增长率**：通过 `(后值 - 前值) / 前值` 的公式实现。
3.  **计算对数收益率**：这是分析连续变化的核心，公式为 **`log(P_t / P_{t-1})`**，使用 `np.log(data / data.shift(1))` 计算。
4.  **分析累积变化**：通过对数收益率的累积和（`.cumsum()`）并还原（`np.exp()`），得到从起点开始的连续价值变化轨迹。

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_61.png)

![](img/d0f3726b0b407e6a051bb83eef6b0c8e_62.png)

关键在于理解，对于需要**连续复合计算**的场景（如长期投资回报），对数收益率比简单增长率更科学、更准确。Pandas和NumPy提供的工具（`.shift()`, `.cumsum()`, `np.log`, `np.exp`）让我们能够高效地实现这些分析。
