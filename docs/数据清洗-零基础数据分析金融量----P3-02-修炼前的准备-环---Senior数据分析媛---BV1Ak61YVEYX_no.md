## Anaconda：数据分析的集成环境

上一节我们对数据分析进行了初步介绍。本节中，我们来看看数据分析所对应的开发环境搭建流程。

首先，我们需要了解一个核心工具：**Anaconda**。Anaconda是一个**集成环境**。这意味着它已经为我们预先打包好了进行数据分析和机器学习开发所需的全部软件包和依赖库。

**核心概念**：
*   **集成环境**：一个预先配置好所有必要工具和库的软件包。

因此，如果你想进行数据分析或机器学习开发，第一步就是在你的电脑上安装Anaconda。安装后，你就可以直接使用这个集成的环境开始工作。

以下是安装Anaconda的步骤和注意事项：

1.  **下载安装包**：访问Anaconda官网，根据你的操作系统（Windows、macOS或Linux）下载对应的安装程序。
2.  **执行安装**：运行下载的安装程序，按照提示进行“下一步”安装即可。
3.  **重要注意事项**：安装路径（目录）**不能包含中文或特殊符号**。建议安装在某个磁盘的根目录下，例如 `C:\Anaconda3` 或 `D:\Anaconda3`。

## Jupyter Notebook：基于浏览器的开发工具

安装好Anaconda后，我们就拥有了开发环境。接下来，我们需要一个编写和执行代码的工具，这就是 **Jupyter Notebook**。

![](img/af6924cbe12a5d5d7943838884b12bab_5.png)

Jupyter Notebook（或简称Jupyter）是Anaconda提供的一个**基于浏览器的可视化开发工具**。我们将在Jupyter中编写数据分析代码并查看执行结果。它无需单独安装，Anaconda安装完成后即可使用。

![](img/af6924cbe12a5d5d7943838884b12bab_7.png)

### 启动Jupyter Notebook
启动Jupyter Notebook非常简单，只需在系统的终端（或命令提示符）中输入一条指令。

**启动指令**：
```bash
jupyter notebook
```
输入上述指令并按下回车键，系统会自动启动一个本地服务并打开你的默认浏览器，显示Jupyter的主界面。这个界面展示的是你当前终端所在目录的文件结构。

### 创建与使用笔记本文件
在Jupyter主界面，我们可以创建一个新的笔记本文件来编写代码。

**操作步骤**：
1.  点击右上角的 **`New`** 按钮。
2.  在下拉菜单中选择 **`Python 3`**。这将创建一个新的、后缀名为 `.ipynb` 的源代码文件。

创建文件后，你会看到编辑界面。界面中的每个可编辑区域称为一个 **`Cell`**（单元格）。

### Cell的两种模式
Cell有两种工作模式，用于不同目的：

*   **`Code` 模式**：在此模式的Cell中，你可以编写和运行Python代码。
*   **`Markdown` 模式**：在此模式的Cell中，你可以使用Markdown语法编写格式化的文本、笔记或注释。

**模式切换**：
*   选中一个Cell后，工具栏的下拉菜单会显示当前模式（Code或Markdown）。
*   通过下拉菜单可以手动切换模式。

**示例**：
在Code模式的Cell中输入 `print("Hello World")`，然后运行该Cell，即可在下方看到输出结果。在Markdown模式的Cell中输入 `# 这是标题`，运行后，文本会以标题格式显示。

### 常用快捷键
熟练使用快捷键可以极大提升在Jupyter中的工作效率。以下是几个最常用的快捷键：

*   **添加Cell**：
    *   `A`：在当前选中Cell的上方插入一个新Cell。
    *   `B`：在当前选中Cell的下方插入一个新Cell。
*   **删除Cell**：`X` 删除当前选中的Cell。
*   **切换Cell模式**：
    *   `M`：将当前Cell切换到 **Markdown模式**。
    *   `Y`：将当前Cell切换到 **Code模式**。
*   **运行Cell**：`Shift + Enter` 运行当前Cell，并跳转到下一个Cell。
*   **代码自动补全**：`Tab` 键可以触发代码自动补全提示。
*   **查看帮助文档**：将光标放在某个函数或方法名上，按 `Shift + Tab` 可以查看其简要帮助文档。
