## 使用`date_range`函数

![](img/e10b530f77cae9a5c93134b3e8eb8566_11.png)

![](img/e10b530f77cae9a5c93134b3e8eb8566_13.png)

`date_range`函数是Pandas中用于生成时间范围的核心工具。它允许你指定开始时间、结束时间、序列长度以及频率。

![](img/e10b530f77cae9a5c93134b3e8eb8566_15.png)

以下是`date_range`函数的主要参数：
*   **`start`**：指定时间序列的开始时间。
*   **`end`**：指定时间序列的结束时间。
*   **`periods`**：指定要生成的时间点的数量（长度）。
*   **`freq`**：指定生成时间点的频率，例如每天（‘D’）、每小时（‘H’）。

![](img/e10b530f77cae9a5c93134b3e8eb8566_17.png)

![](img/e10b530f77cae9a5c93134b3e8eb8566_19.png)

你可以通过指定`start`和`end`来生成一个时间范围，也可以指定`start`和`periods`来生成一个固定长度的序列。

## 参数`freq`详解

![](img/e10b530f77cae9a5c93134b3e8eb8566_21.png)

`freq`参数决定了生成时间点的间隔。它是一个非常灵活的参数，支持多种频率。

![](img/e10b530f77cae9a5c93134b3e8eb8566_23.png)

![](img/e10b530f77cae9a5c93134b3e8eb8566_25.png)

以下是`freq`参数的一些常用取值：
*   **`‘D’`**：每天（默认值）。
*   **`‘H’`**：每小时。
*   **`‘W’`**：每周（默认从周日开始）。
*   **`‘W-MON’`**：每周一。
*   **`‘B’`**：工作日（周一至周五）。
*   **`‘M’`**：每月最后一天。
*   **`‘MS’`**：每月第一天。
*   **`‘T’` 或 `‘min’`**：每分钟。
*   **`‘S’`**：每秒。
*   **`‘A’` 或 `‘Y’`**：每年最后一天。
*   **`‘AS’` 或 `‘YS’`**：每年第一天。

![](img/e10b530f77cae9a5c93134b3e8eb8566_27.png)

![](img/e10b530f77cae9a5c93134b3e8eb8566_29.png)

你甚至可以组合使用，例如`‘2H30min’`表示每2小时30分钟。Pandas库在处理时间序列方面功能非常强大。

![](img/e10b530f77cae9a5c93134b3e8eb8566_31.png)

![](img/e10b530f77cae9a5c93134b3e8eb8566_33.png)

## 时间序列的转换

![](img/e10b530f77cae9a5c93134b3e8eb8566_35.png)

生成的时间序列对象是Pandas的`DatetimeIndex`类型。你可以根据需要将其转换为其他格式。

![](img/e10b530f77cae9a5c93134b3e8eb8566_37.png)

![](img/e10b530f77cae9a5c93134b3e8eb8566_39.png)

例如，可以将其转换为Python标准的`datetime`对象列表，或者转换为字符串列表以便于显示或存储。虽然某些转换方法（如`.to_pydatetime()`）在未来版本中可能会有变化，但Pandas提供了灵活的方式在不同格式间进行转换。

![](img/e10b530f77cae9a5c93134b3e8eb8566_41.png)

本节课中我们一起学习了如何使用Pandas的`date_range`函数生成时间范围序列，了解了其核心参数`start`、`end`、`periods`和`freq`的用法，并看到了`freq`参数支持的各种频率设置。掌握这些知识，你就能轻松创建用于金融数据分析或其他时间序列任务的时间索引了。
