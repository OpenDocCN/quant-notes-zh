在本节课中，我们将学习如何计算并分析一个量化交易策略的最终收益。我们将通过一个具体的例子，对比“什么都不做”的原始收益与使用“双均线策略”调整后的策略收益，从而评估策略的有效性。

![](img/7398ce8bdf3979553def41adc802998d_1.png)

上一节我们介绍了如何通过双均线交叉来生成交易信号（`position`）。本节中，我们来看看如何利用这些信号来计算策略的最终收益。

![](img/7398ce8bdf3979553def41adc802998d_3.png)

## 计算原始收益率

首先，我们需要计算股价的日收益率。在金融分析中，我们通常使用对数收益率，因为它具有更好的数学性质（如可加性）。计算公式如下：

**对数收益率公式：**
`log_return = log(price_t / price_{t-1})`

在代码中，我们使用Pandas的`shift`方法来获取前一天的股价，并进行计算。

```python
# 计算对数收益率
data[‘return’] = np.log(data[‘close’] / data[‘close’].shift(1))
```

![](img/7398ce8bdf3979553def41adc802998d_5.png)

![](img/7398ce8bdf3979553def41adc802998d_7.png)

执行这段代码后，我们得到了一个名为`return`的列，它代表了股价每日的增长（或下跌）率。

## 计算策略收益率

![](img/7398ce8bdf3979553def41adc802998d_9.png)

接下来，我们将根据上一节生成的交易信号`position`来计算策略收益率。我们的策略逻辑是：当`position`为1时，表示我们预测股价将上涨，因此遵循原始收益率；当`position`为-1时，表示我们预测股价将下跌，因此需要“反向操作”，即获得与原始收益率相反的收益。

![](img/7398ce8bdf3979553def41adc802998d_11.png)

策略收益率的计算公式可以理解为：
**策略收益率 = 交易信号 * 原始收益率**

![](img/7398ce8bdf3979553def41adc802998d_13.png)

在代码中实现如下：

![](img/7398ce8bdf3979553def41adc802998d_15.png)

```python
# 计算策略收益率。注意：position也需要进行shift操作，以确保使用前一天的信号来决定今天的操作。
data[‘strategy_return’] = data[‘position’].shift(1) * data[‘return’]
```

这里的关键点是，`position`列也需要使用`.shift(1)`。这是因为我们在`t`日收盘后（或`t+1`日开盘前）才能根据`t`日的均线交叉情况生成信号，并用于`t+1`日的交易决策。

![](img/7398ce8bdf3979553def41adc802998d_17.png)

## 累计收益与结果还原

现在我们已经有了每日的策略收益率。为了知道从开始到结束的总收益，我们需要将这些每日的收益率累加起来。由于我们使用的是对数收益率，累加操作就是简单的求和。

![](img/7398ce8bdf3979553def41adc802998d_19.png)

累计对数收益率的公式为：
**累计对数收益率 = Σ(每日对数收益率)**

![](img/7398ce8bdf3979553def41adc802998d_21.png)

求和之后，我们需要将对数收益率还原为实际的资产倍数。还原公式为：
**最终资产倍数 = exp(累计对数收益率)**

![](img/7398ce8bdf3979553def41adc802998d_23.png)

这个结果表示：最初投入的1元钱，最终会变成多少钱。

![](img/7398ce8bdf3979553def41adc802998d_25.png)

以下是计算累计收益并还原的代码步骤：

![](img/7398ce8bdf3979553def41adc802998d_27.png)

![](img/7398ce8bdf3979553def41adc802998d_29.png)

```python
# 1. 删除可能存在缺失值的行（例如第一行由于shift操作产生的NaN）
data.dropna(inplace=True)

# 2. 分别计算原始收益率和策略收益率的累计和
cumulative_log_return_original = data[‘return’].sum()
cumulative_log_return_strategy = data[‘strategy_return’].sum()

# 3. 将对数累计收益还原为最终资产倍数
final_value_original = np.exp(cumulative_log_return_original)
final_value_strategy = np.exp(cumulative_log_return_strategy)

print(f“原始策略最终资产倍数：{final_value_original:.2f}“)
print(f“双均线策略最终资产倍数：{final_value_strategy:.2f}“)
```

![](img/7398ce8bdf3979553def41adc802998d_31.png)

## 结果解读

![](img/7398ce8bdf3979553def41adc802998d_33.png)

![](img/7398ce8bdf3979553def41adc802998d_35.png)

执行以上代码后，我们可能会得到类似这样的结果：
- 原始策略最终资产倍数：4.20
- 双均线策略最终资产倍数：5.80

![](img/7398ce8bdf3979553def41adc802998d_37.png)

这个结果意味着：
*   **如果什么都不做（买入并持有）**，最初投资的1元钱，最终会变成4.20元。
*   **如果使用我们的双均线策略进行交易**，最初投资的1元钱，最终会变成5.80元。

通过对比可以看出，在这个案例的历史数据回测中，双均线策略获得了比简单持有更高的收益（5.80 > 4.20），这表明该策略在历史阶段是有效的。

本节课中我们一起学习了量化策略收益分析的全过程。我们从计算基本的对数收益率开始，然后结合交易信号计算出策略的每日收益率，最后通过累加和指数还原，得到了策略的最终收益表现，并与基准（买入持有）进行了对比。这是评估任何一个量化策略是否有效的核心步骤。
