## 什么是聚类？

上一节我们介绍了有监督学习，其特点是数据带有标签。本节中我们来看看无监督学习中的聚类问题。

聚类要解决的问题与分类类似，但数据本身没有标签。我们的目标是将相似的数据点自动归为一组，形成不同的“簇”。例如，在右侧的图中，原始数据没有颜色标记，算法需要根据数据点之间的相似度，自动将它们划分为绿色、红色和蓝色三个簇。

聚类算法虽然原理相对简单，但面临两大主要挑战：
1.  **评估困难**：由于没有真实标签，很难像有监督学习那样通过精度、召回率等指标来准确评估聚类结果的好坏。
2.  **参数调节困难**：例如，改变聚类数量（K值）会得到不同的结果，但没有标准答案来判断哪个结果更优。

## K-Means算法核心概念

![](img/384d0ded4b6f165bca1c89efc3618807_6.png)

K-Means算法是聚类算法中最简单且最实用的方法之一。它的核心思想是：将数据点划分到K个簇中，使得每个数据点到其所属簇的“质心”距离之和最小。

![](img/384d0ded4b6f165bca1c89efc3618807_8.png)

![](img/384d0ded4b6f165bca1c89efc3618807_10.png)

以下是K-Means算法的几个核心概念：

![](img/384d0ded4b6f165bca1c89efc3618807_12.png)

### 1. 指定K值
K-Means算法需要我们预先指定一个参数：**K值**。K值代表我们希望将数据聚合成多少个簇。例如，设定K=3，算法最终会将数据分成三堆；设定K=4，则会分成四堆。这是算法运行前必须确定的参数。

![](img/384d0ded4b6f165bca1c89efc3618807_14.png)

### 2. 质心
每个簇都有一个中心点，称为**质心**。质心是该簇中所有数据点在各个维度上的平均值。例如，在二维数据中，质心的X坐标是簇内所有点X坐标的平均值，Y坐标是Y坐标的平均值。质心在算法的迭代优化过程中起着关键作用。

### 3. 距离度量
判断两个数据点是否相似，需要基于**距离**来计算。最常用的距离度量是**欧式距离**。对于两个点，计算它们在各个维度上差值的平方和再开方。

**公式**：对于二维空间中的点A(x1, y1)和B(x2, y2)，其欧式距离为：
`distance = sqrt((x1 - x2)^2 + (y1 - y2)^2)`

**重要提示**：在使用距离度量（尤其是欧式距离）前，通常需要对数据进行**标准化**（如归一化）。这是因为如果不同特征（维度）的数值范围差异巨大（例如一个特征值在0~1，另一个在100~1000），数值范围大的特征会主导距离计算，导致结果失真。标准化可以将所有特征缩放到相近的范围内（如0~1或-1~1）。

![](img/384d0ded4b6f165bca1c89efc3618807_16.png)

### 4. 优化目标
K-Means算法通过优化一个目标函数来工作。这个目标函数是：最小化所有数据点到其所属簇质心的距离总和。

![](img/384d0ded4b6f165bca1c89efc3618807_18.png)

**公式**：优化目标可以表示为最小化以下函数：
`J = Σ（i=1 到 K） Σ（x 属于簇Ci） || x - μi ||^2`
其中：
*   `K` 是簇的数量。
*   `Ci` 是第i个簇。
*   `x` 是簇Ci中的一个数据点。
*   `μi` 是簇Ci的质心。
*   `|| x - μi ||^2` 是点x到质心μi的欧式距离的平方。

![](img/384d0ded4b6f165bca1c89efc3618807_20.png)

![](img/384d0ded4b6f165bca1c89efc3618807_22.png)

**通俗解释**：我们希望每个簇内的点都尽可能紧密地围绕在质心周围。如果一个点离当前簇的质心很远，但离另一个簇的质心更近，那么在算法迭代过程中，它就会被重新分配到更近的那个簇里去。这个过程不断重复，直到所有数据点的分配不再变化，或达到最大迭代次数。

## 总结

本节课中我们一起学习了聚类的基本概念以及K-Means算法的核心原理。我们了解到：
*   聚类是一种无监督学习方法，用于将相似数据自动分组。
*   K-Means算法需要预先指定簇的数量K。
*   算法的核心是计算数据点到质心的距离（通常用欧式距离），并通过迭代优化，使每个点到其所属质心的距离之和最小。
*   数据标准化是使用距离度量的重要前置步骤。
*   聚类的评估和参数选择是其应用中的主要挑战。

K-Means算法以其简洁高效的特点，成为数据分析、客户分群、图像分割等众多领域的常用工具。
