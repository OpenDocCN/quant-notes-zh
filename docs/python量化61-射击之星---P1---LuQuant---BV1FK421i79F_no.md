## 烛台模式简介
上一节我们介绍了课程目标，本节中我们来看看将要用到的几种基础烛台模式。请注意，以下列表并非详尽无遗，我们仅关注几种基础形态。

以下是几种常见的烛台模式：
*   **十字星**：显示出不确定的市场结局。
*   **射击之星（流星）**：在上升趋势中，出现具有长上影线和小实体的蜡烛，可能预示趋势反转。
*   **锤子线**：与射击之星形态相反，具有长下影线，在下降趋势中可能预示反转。
*   **看涨吞没模式**：在下降趋势中，一根大阳线完全覆盖前一根阴线，可能宣布下降趋势结束。
*   **看跌吞没模式**：与看涨吞没形态相反，在上升趋势中，一根大阴线完全覆盖前一根阳线，可能预示上升趋势结束。

在本视频中，我们将特别关注**射击之星（流星）**模式，因为它能清晰地显示卖方进入市场并拒绝某一价格水平。

## 射击之星模式的编程识别
上一节我们回顾了几种烛台形态，本节中我们来看看如何用Python代码具体识别“射击之星”模式。

识别射击之星（卖出信号）需要满足几个几何条件：
1.  上影线长度相对于烛台实体长度的比例需超过一定阈值。
2.  下影线长度应相对较短。

我们可以用以下公式来描述这些条件。首先定义几个变量：
*   `high`：最高价
*   `open`：开盘价
*   `close`：收盘价
*   `low`：最低价
*   `body_diff`：实体长度，计算公式为 `abs(open - close)`。为避免除零错误，可设置一个最小值（如0.002）。
*   `high_diff`：上影线长度，计算公式为 `high - max(open, close)`。
*   `low_diff`：下影线长度，计算公式为 `min(open, close) - low`。

接着，我们计算两个比率：
*   **比率1 (ratio1)**：`high_diff / body_diff`。此值应较大，表示上影线长。
*   **比率2 (ratio2)**：`low_diff / high_diff`。此值应较小，表示下影线短。

在代码中，我们可以设置条件，例如 `ratio1 > 2.5` 且 `ratio2 < 0.3`，来初步筛选出射击之星形态。

## 结合技术指标增强信号
单独依靠烛台形态可能不足以产生可靠的交易信号。因此，我们将其与**相对强弱指数（RSI）** 结合使用。

对于**卖出信号**，我们不仅要求出现射击之星形态，还要求RSI值处于特定区间（例如，高于50%但低于70%）。这表示市场可能处于超买状态，但趋势并非极端强劲，增加了反转可能性。

对于**买入信号**（对应倒锤子或锤子线形态），我们则寻找RSI值低于50%但高于30%的情况。

在Python中，我们可以使用`pandas_ta`库方便地计算ATR和RSI指标。

```python
import pandas_ta as ta
# 计算ATR（平均真实波幅）和RSI
df[‘atr’] = ta.atr(df[‘high’], df[‘low’], df[‘close’], length=14)
df[‘rsi’] = ta.rsi(df[‘close’], length=14)
```

## 交易策略与回测设置
现在我们已经能识别交易信号，本节将定义具体的交易策略和回测规则。

我们采用以下策略逻辑：
1.  **入场**：当识别到符合条件（烛台形态+RSI）的买入或卖出信号时，在下一根K线开盘时入场。
2.  **止损**：使用**平均真实波幅（ATR）** 来设置动态止损。
    *   对于买入订单，止损位为 `入场价格 - ATR值`。
    *   对于卖出订单，止损位为 `入场价格 + ATR值`。
    *   同时，我们设置一个最小止损距离（例如1.1点），以防止止损过近。
3.  **止盈**：我们设定一个固定的**盈亏比（Risk-Reward Ratio）**。例如，设置盈亏比为2，这意味着止盈距离是止损距离的2倍。
    *   对于买入订单，止盈位为 `入场价格 + (2 * ATR值)`。
    *   对于卖出订单，止盈位为 `入场价格 - (2 * ATR值)`。

在回测中，我们将在信号出现后，向前检查一定数量的K线（例如20根），观察价格是**先触及止损**还是**先触及止盈**，从而判断该笔交易是盈利还是亏损。

理论上，当盈亏比为2时，策略胜率只需高于33.3%即可实现盈利。

## Python代码实现与结果分析
上一节我们制定了策略规则，本节我们来看看核心的代码实现与回测结果。

以下是信号识别函数的核心逻辑片段：

```python
def identify_signals(df):
    signals = [0] * len(df) # 初始化信号列表
    for i in range(1, len(df)):
        # 计算实体、影线长度及比率
        body_diff = abs(df[‘open’].iloc[i] - df[‘close’].iloc[i])
        if body_diff < 0.002:
            body_diff = 0.002
        high_diff = df[‘high’].iloc[i] - max(df[‘open’].iloc[i], df[‘close’].iloc[i])
        low_diff = min(df[‘open’].iloc[i], df[‘close’].iloc[i]) - df[‘low’].iloc[i]
        ratio1 = high_diff / body_diff
        ratio2 = low_diff / high_diff if high_diff != 0 else 0
        # 判断射击之星（卖出信号）条件
        if (ratio1 > 2.5 and ratio2 < 0.3 and body_diff > 0.03 and
            50 < df[‘rsi’].iloc[i] < 70):
            signals[i] = 1 # 卖出信号
        # 判断买入信号条件（此处略，逻辑对称）
    return signals
```

在对美元/日元15年的日线数据回测后，我们得到以下结果：
*   **卖出信号**：共产生36次信号，准确率约为61%。
*   **买入信号**：共产生92次信号，准确率约为42%。

两个信号的准确率均高于理论盈亏平衡点（33.3%），表明该基础策略框架在历史数据上具有盈利潜力。
