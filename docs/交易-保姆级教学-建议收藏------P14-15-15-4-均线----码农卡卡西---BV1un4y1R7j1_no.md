## 遍历参数组合并计算策略表现

接下来，我们将遍历所有参数组合，对每一组参数重新计算双均线策略，并评估其表现。以下是实现这一过程的核心步骤。

![](img/323248d6e46e3580283d54608c3b9dc6_68.png)

![](img/323248d6e46e3580283d54608c3b9dc6_70.png)

我们将使用一个循环来遍历`product`函数生成的所有参数对。在循环内部，我们将重复之前的数据处理、指标计算和策略评估步骤。

![](img/323248d6e46e3580283d54608c3b9dc6_72.png)

![](img/323248d6e46e3580283d54608c3b9dc6_74.png)

![](img/323248d6e46e3580283d54608c3b9dc6_76.png)

```python
import pandas as pd

![](img/323248d6e46e3580283d54608c3b9dc6_78.png)

![](img/323248d6e46e3580283d54608c3b9dc6_80.png)

![](img/323248d6e46e3580283d54608c3b9dc6_82.png)

![](img/323248d6e46e3580283d54608c3b9dc6_84.png)

# 初始化一个空的DataFrame来存储所有结果
results = pd.DataFrame()

![](img/323248d6e46e3580283d54608c3b9dc6_86.png)

![](img/323248d6e46e3580283d54608c3b9dc6_88.png)

![](img/323248d6e46e3580283d54608c3b9dc6_90.png)

# 遍历所有参数组合
for sma1, sma2 in product(short_window_range, long_window_range):
    # 1. 重新加载并处理数据
    data = pd.read_csv('apple_stock_price.csv', index_col='Date', parse_dates=True)
    data = data.dropna()  # 去除缺失值

    # 2. 计算收益率和移动平均线
    data['returns'] = data['Close'].pct_change()
    data['SMA1'] = data['Close'].rolling(window=sma1).mean()
    data['SMA2'] = data['Close'].rolling(window=sma2).mean()
    data = data.dropna()  # 再次去除因计算产生的缺失值

    # 3. 生成交易信号
    data['position'] = 0
    data.loc[data['SMA1'] > data['SMA2'], 'position'] = 1
    data.loc[data['SMA1'] <= data['SMA2'], 'position'] = 0

    # 4. 计算策略收益
    data['strategy'] = data['position'].shift(1) * data['returns']
    data = data.dropna()  # 去除策略信号产生的缺失值

    # 5. 汇总本次参数组合的结果
    total_return = data['returns'].sum()
    total_strategy_return = data['strategy'].sum()
    outperformance = total_strategy_return - total_return

    # 6. 将结果添加到总表中
    temp_df = pd.DataFrame({
        'SMA1': [sma1],
        'SMA2': [sma2],
        'Total_Return': [total_return],
        'Strategy_Return': [total_strategy_return],
        'Outperformance': [outperformance]
    })
    results = pd.concat([results, temp_df], ignore_index=True)
```
