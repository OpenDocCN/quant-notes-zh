## 金融分析方法
金融分析主要分为两种方法：基本面分析和技术面分析。

### 基本面分析
基本面分析的核心是评估公司的运营状况，即我们之前提到的公司自身因素。通过分析当前经济状况、行业前景以及公司具体运营情况，来决定是否购买其股票。

基本面分析包含以下三个方面：

以下是宏观经济面分析：
*   分析国家的财政政策、货币政策等。
*   判断资金流向是倾向于股市还是储蓄。
*   注意宏观经济分析存在不确定性，规律并非总是有效。

以下是行业分析：
*   判断整个行业的发展状况。
*   例如评估教育、IT、钢铁、煤炭等行业的前景。
*   可以提取行业内有代表性的股票，观察其走势。

以下是公司分析：
*   具体分析目标上市公司。
*   上市公司需定期公开经审计的财务报告（财报）。
*   通过分析财报数据（如盈利情况）来判断公司经营状况。
*   若预判公司经营良好，可在财报发布前购入股票；若不了解，则依据发布的财报数据做决策。

### 技术面分析
技术面分析的核心观点是：所有信息都已蕴含在市场交易数据中。它通过分析历史市场走势和技术指标来预测未来价格动向。

以下是技术面分析的核心：
*   分析历史时期（如前一个月）的市场走势。
*   不仅看涨跌，更依赖定义好的技术指标。
*   常见技术指标包括K线、MA均线、KDJ、MACD等。

接下来，我们简要介绍两个基础技术指标：K线和MA均线。

## 技术指标详解

### K线图
K线图是展示股票每日价格走势的图表。横轴代表时间，纵轴代表股票价格。图中的每一个柱状体称为一根K线。

一根K线包含四个价格信息：开盘价、收盘价、最高价和最低价。

K线分为阳线和阴线：
*   **阳线**：通常为红色或空心，表示当日股价上涨（收盘价高于开盘价）。
*   **阴线**：通常为绿色或实心，表示当日股价下跌（收盘价高于开盘价）。

K线的结构如下：
*   **实体**：中间的矩形部分。
    *   阳线中，实体的**下边缘**表示开盘价，**上边缘**表示收盘价。
    *   阴线中，实体的**上边缘**表示开盘价，**下边缘**表示收盘价。
*   **影线**：实体上下方的细线。
    *   **上影线**顶端表示当日**最高价**。
    *   **下影线**底端表示当日**最低价**。

因此，通过一根K线即可获取四个关键价格。K线存在一些特殊形态，例如十字星（开盘价等于收盘价）或光头光脚线（无影线），这些形态可用于后续分析。

### MA移动平均线
均线（MA， Moving Average）是取前若干天价格的平均值，并将每日的平均值点连接起来形成的曲线。

![](img/fcc2b2872b229b2d12d293882dee0a7e_7.png)

![](img/fcc2b2872b229b2d12d293882dee0a7e_9.png)

常见的均线包括：
*   **MA5**：5日均线，取前5天价格的平均值。
*   **MA60**：60日均线，取前60天价格的平均值。

计算公式（以收盘价为例）：
`MA(N) = (P1 + P2 + ... + PN) / N`
其中，`P1` 到 `PN` 代表过去N个交易日的收盘价。

均线的作用是平滑价格波动，反映近期趋势。例如，双均线策略（如MA5与MA60的比较）是量化交易中的常用策略。

## 总结
本节课我们一起学习了金融分析的两大方法。基本面分析侧重于公司运营、行业与经济环境；技术面分析则聚焦于历史价格数据与图表形态，并详细介绍了K线和MA均线这两个基础技术指标。理解这些是进行后续量化策略学习的重要基础。
