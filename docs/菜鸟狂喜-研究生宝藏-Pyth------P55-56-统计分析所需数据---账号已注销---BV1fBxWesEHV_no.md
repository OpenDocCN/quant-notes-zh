## 数据转换与分类

![](img/cb0710759521ddab183437f76945bf61_3.png)

![](img/cb0710759521ddab183437f76945bf61_5.png)

首先，我们需要将连续的价格变化指标（如涨跌幅）转换为离散的分类，例如“涨”（用1表示）和“跌”（用0表示）。这样做的目的是为了便于后续进行分组统计。

![](img/cb0710759521ddab183437f76945bf61_7.png)

![](img/cb0710759521ddab183437f76945bf61_9.png)

以下是实现这一转换的步骤：

![](img/cb0710759521ddab183437f76945bf61_11.png)

![](img/cb0710759521ddab183437f76945bf61_13.png)

1.  **定义转换函数**：创建一个函数，用于判断每个指标值是大于0（涨）还是小于等于0（跌）。
2.  **应用函数**：将此函数应用到我们关心的指标列上，生成新的分类列。
3.  **查看结果**：验证转换后的数据是否符合预期。

![](img/cb0710759521ddab183437f76945bf61_15.png)

![](img/cb0710759521ddab183437f76945bf61_17.png)

让我们开始编写代码。首先，我们定义一个名为 `clip_ins` 的函数来完成分类工作。

![](img/cb0710759521ddab183437f76945bf61_19.png)

![](img/cb0710759521ddab183437f76945bf61_21.png)

```python
def clip_ins(data, bins=[0]):
    global col_names
    col_names = []
    # 遍历我们之前定义好的特征列名（例如代表‘前一天’和‘前两天’涨跌幅的列）
    for col in feature_columns:
        # 为新列命名，在原列名后加‘_B’后缀
        new_col_name = col + '_B'
        # 判断原数据列的值是否大于0，大于0则为1（涨），否则为0（跌）
        data[new_col_name] = (data[col] > 0).astype(int)
        # 记录新生成的列名
        col_names.append(new_col_name)
```

![](img/cb0710759521ddab183437f76945bf61_23.png)

![](img/cb0710759521ddab183437f76945bf61_25.png)

接下来，我们调用这个函数来处理我们的数据 `data`。

```python
clip_ins(data)
```

![](img/cb0710759521ddab183437f76945bf61_27.png)

执行完成后，我们可以查看数据的前几行，确认新列已添加。

![](img/cb0710759521ddab183437f76945bf61_29.png)

![](img/cb0710759521ddab183437f76945bf61_31.png)


## 整合实际走势数据

![](img/cb0710759521ddab183437f76945bf61_39.png)

![](img/cb0710759521ddab183437f76945bf61_41.png)

为了进行统计分析，我们还需要一列代表“当天”实际涨跌情况的数据。原始数据中通常包含一个名为 `direction` 的列，它记录了价格的实际变动方向（例如，1代表上涨，-1代表下跌，0代表不变）。

![](img/cb0710759521ddab183437f76945bf61_43.png)

我们的目标是：统计在“前一天”和“前两天”的特定涨跌组合下，“当天”出现各种实际走势的频率。

![](img/cb0710759521ddab183437f76945bf61_45.png)

![](img/cb0710759521ddab183437f76945bf61_47.png)

## 使用分组统计

![](img/cb0710759521ddab183437f76945bf61_49.png)

![](img/cb0710759521ddab183437f76945bf61_51.png)

现在，我们拥有了所需的所有数据列：`前一天分类`、`前两天分类` 和 `实际方向`。接下来，使用Pandas的 `groupby` 方法进行分组统计。

![](img/cb0710759521ddab183437f76945bf61_53.png)

![](img/cb0710759521ddab183437f76945bf61_55.png)

以下是进行分组统计的步骤：

![](img/cb0710759521ddab183437f76945bf61_57.png)

![](img/cb0710759521ddab183437f76945bf61_59.png)

1.  **指定分组键**：按照 `前一天分类`、`前两天分类` 和 `实际方向` 这三列进行分组。
2.  **计算组大小**：统计每个唯一组合在历史数据中出现的次数。

![](img/cb0710759521ddab183437f76945bf61_61.png)

![](img/cb0710759521ddab183437f76945bf61_63.png)

让我们用代码实现：

![](img/cb0710759521ddab183437f76945bf61_65.png)

![](img/cb0710759521ddab183437f76945bf61_67.png)

```python
# 进行分组，统计每种情况出现的次数
grouped = data.groupby([col_names[0], col_names[1], ‘direction’]).size()
print(grouped)
```

![](img/cb0710759521ddab183437f76945bf61_69.png)

执行这段代码后，我们将得到一个序列（Series）。它展示了所有可能的组合及其出现次数。例如，输出 `(0, 0, 1) 25` 表示在历史数据中，“前两天跌、前一天跌、当天涨”这种情况出现了25次。

![](img/cb0710759521ddab183437f76945bf61_71.png)

本节课中我们一起学习了如何为简单的统计分析准备数据。我们通过将连续指标二值化（涨/跌），并整合实际走势数据，最终利用 `groupby` 方法统计了各种历史情况发生的频率。这为我们下一节基于历史概率制定交易策略提供了必要的数据基础。
