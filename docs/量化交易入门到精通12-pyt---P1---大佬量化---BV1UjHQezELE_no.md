# 量化交易入门到精通：12：Python基础语法之保留字 🐍

在本节课中，我们将要学习Python编程语言中的一个基础但至关重要的概念：**保留字**。理解保留字是编写正确、无歧义Python代码的第一步。

## 概述

![](img/4476f4d61faeed802c809086a76c4901_1.png)

保留字，也称为关键字，是Python语言内部已经定义并具有特殊功能的单词。它们不能被用作变量名、函数名或其他标识符名称。本节课将介绍如何查看Python的所有保留字，并通过实例说明为什么不能将它们用作自定义名称。

## 查看Python保留字

我们可以通过Python内置的`keyword`模块来查看所有的保留字。以下是具体步骤：

1.  首先，我们需要导入`keyword`模块。
2.  然后，使用`keyword.kwlist`属性来获取保留字列表。
3.  最后，打印这个列表以查看所有内容。

以下是实现此功能的代码：

![](img/4476f4d61faeed802c809086a76c4901_3.png)

```python
import keyword
print(keyword.kwlist)
```

执行上述代码后，控制台会输出一个列表，其中包含了Python当前版本的所有保留字。例如：`[‘False‘, ‘None‘, ‘True‘, ‘and‘, ‘as‘, ‘assert‘, …]`。这个列表中的单词就是Python的保留字。

## 为什么不能使用保留字作为标识符

![](img/4476f4d61faeed802c809086a76c4901_5.png)

保留字在Python语法中具有特定的含义和用途。如果将它们用作变量名等标识符，解释器将无法区分你是在使用关键字的功能，还是在引用一个自定义的变量，从而导致语法错误或歧义。

以下是几个错误的示例，演示了使用保留字作为变量名时会发生的情况：

![](img/4476f4d61faeed802c809086a76c4901_7.png)

*   **示例一：使用 `and`**
    尝试将变量命名为 `and` 并赋值：
    ```python
    and = 1  # 这行代码会报错
    ```
    执行时，Python会提示类似 `SyntaxError: invalid syntax` 的错误，明确指出 `and` 是一个关键字。

*   **示例二：使用 `True`**
    尝试将变量命名为 `True`：
    ```python
    True = 1  # 这行代码同样会报错
    ```
    这会引发 `SyntaxError: cannot assign to True` 的错误，因为 `True` 是代表布尔真值的保留字，不能被重新赋值。

![](img/4476f4d61faeed802c809086a76c4901_9.png)

*   **示例三：使用 `class`**
    尝试将变量命名为 `class`：
    ```python
    class = 1  # 这行代码也会报错
    ```
    错误信息会更加详细，例如 `SyntaxError: invalid syntax`，因为 `class` 是用于定义类的关键字。

![](img/4476f4d61faeed802c809086a76c4901_11.png)

从以上例子可以看出，使用保留字作为标识符会导致程序无法正常运行。因此，在命名变量、函数或类时，必须避开这些保留字。

![](img/4476f4d61faeed802c809086a76c4901_13.png)

## 总结

本节课中，我们一起学习了Python的保留字。我们首先了解了如何通过 `import keyword; print(keyword.kwlist)` 来查看所有保留字。更重要的是，我们通过几个具体的错误示例，明白了为什么**绝对不能**使用这些保留字来命名自己的变量或函数。记住并避开这些保留字，是编写正确Python代码的基本要求。在接下来的课程中，我们将开始学习如何使用这些保留字来构建有效的Python语句和程序结构。
