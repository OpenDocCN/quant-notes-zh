## 数据处理与PEG计算

获取到原始数据后，我们需要进行清洗和计算。首先，根据PEG策略的定义，我们需要过滤掉市盈率（PE）或净利润增长率（G）为负值的股票，因为负值没有经济学意义。

以下是数据处理流程：

1.  使用布尔索引过滤数据，保留 `pe_ratio > 0` 且 `inc_net_profit_year_on_year > 0` 的股票。
2.  计算PEG值，公式为：**PEG = PE / G**。注意，原始数据中的增长率G是百分比形式，计算时可能需要除以100。
3.  根据计算出的PEG值对股票进行升序排序，PEG值越小的股票理论上估值越低、成长性越好。

```python
# 过滤掉PE或G为负值的股票
df = df[(df[‘pe_ratio‘] > 0) & (df[‘inc_net_profit_year_on_year‘] > 0)]

# 计算PEG指标
df[‘peg‘] = df[‘pe_ratio‘] / df[‘inc_net_profit_year_on_year‘]

# 按PEG值升序排序
df = df.sort_values(by=‘peg‘)
```

## 策略总结与扩展思考

本节课中，我们一起学习了PEG策略的完整实现过程。PEG本质上是一个复合因子，它结合了估值（PE）和成长性（G），帮助我们寻找那些估值相对较低且具备良好成长潜力的股票。

与之前学习的均值回归等因子类似，PEG因子也可以作为多因子模型中的一个有效组成部分。你可以尝试将PEG与其他技术或基本面因子结合，构建更复杂的选股模型，以进一步提升策略的稳定性和收益潜力。

此外，从实践角度来看，长线持仓策略（如本PEG策略）通常比高频短线交易更为稳健，收益表现也可能更好。在A股市场环境下，适当控制交易频率，避免过度交易，是策略成功的重要因素之一。
