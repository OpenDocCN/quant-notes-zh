![](img/cfadc7792f417dba5011d70847122ae5_3.png)

## 什么是最大回撤？

上一节我们介绍了策略回测的基本概念，本节中我们来看看如何量化策略的风险。最大回撤描述的是投资过程中资产价值下跌“最惨”的时期。

具体来说，它衡量的是在选定周期内，任一历史时点往后推，资产净值走到最低点时的收益率回撤幅度的最大值。它关注的是下行风险。

## 最大回撤的计算原理

为了更精确地理解，我们可以通过公式来看。最大回撤的计算公式通常表示为：

**最大回撤 = max[(Pi - Pj) / Pi]**

![](img/cfadc7792f417dba5011d70847122ae5_5.png)

其中：
*   **Pi** 表示第 i 天的资产价格（峰值）。
*   **Pj** 表示第 i 天之后的某一天（j > i）的资产价格（谷底）。

![](img/cfadc7792f417dba5011d70847122ae5_7.png)

![](img/cfadc7792f417dba5011d70847122ae5_9.png)

![](img/cfadc7792f417dba5011d70847122ae5_11.png)

这个公式的核心是遍历所有可能的时间点组合（i, j），找出资产价值缩水比例最大的那个区间。这个比例反映了投资者在该区间可能承受的最大账面损失。

## 如何用Python计算最大回撤？

![](img/cfadc7792f417dba5011d70847122ae5_13.png)

![](img/cfadc7792f417dba5011d70847122ae5_15.png)

在Python中，我们可以借助Pandas库高效地计算最大回撤。以下是计算步骤的分解：

![](img/cfadc7792f417dba5011d70847122ae5_17.png)

首先，我们需要一个代表资产每日净值的时间序列数据。假设我们有一个Pandas Series对象 `data`。

![](img/cfadc7792f417dba5011d70847122ae5_19.png)

![](img/cfadc7792f417dba5011d70847122ae5_21.png)

以下是计算最大回撤的关键步骤：

![](img/cfadc7792f417dba5011d70847122ae5_23.png)

![](img/cfadc7792f417dba5011d70847122ae5_25.png)

1.  **计算历史最大值序列**：使用 `cummax()` 函数获取到每一天为止的历史最高净值。
    ```python
    historical_max = data.cummax()
    ```
    这行代码会生成一个新序列，其中每个值都是原序列到该日期为止的最大值。

![](img/cfadc7792f417dba5011d70847122ae5_27.png)

![](img/cfadc7792f417dba5011d70847122ae5_29.png)

![](img/cfadc7792f417dba5011d70847122ae5_31.png)

2.  **计算每日回撤**：用历史最高值减去当日的实际净值，得到每日的回撤值。
    ```python
    drawdown = historical_max - data
    ```

![](img/cfadc7792f417dba5011d70847122ae5_33.png)

3.  **计算每日回撤率**：将每日回撤值除以对应的历史最高值，得到回撤比例。
    ```python
    drawdown_pct = drawdown / historical_max
    ```

![](img/cfadc7792f417dba5011d70847122ae5_35.png)

4.  **找出最大回撤**：在所有每日回撤率中，最大值就是我们要找的**最大回撤率**。
    ```python
    max_drawdown = drawdown_pct.max()
    ```
    同时，我们也可以找出最大回撤发生的区间（开始日期和结束日期）。

![](img/cfadc7792f417dba5011d70847122ae5_37.png)

![](img/cfadc7792f417dba5011d70847122ae5_39.png)

## 最大回撤的实际意义

![](img/cfadc7792f417dba5011d70847122ae5_41.png)

在策略回测报告中，最大回撤是一个关键的风险指标。它直观地展示了策略在最坏情况下的表现。

![](img/cfadc7792f417dba5011d70847122ae5_43.png)

![](img/cfadc7792f417dba5011d70847122ae5_45.png)

例如，在2008年全球金融危机中，股市从高点大幅下跌，许多投资者经历了巨大的回撤。最大回撤指标能帮助投资者了解，如果类似情况重演，自己的策略可能承受多大的损失，从而更好地进行风险管理和资产配置。

一个策略可能最终总收益很高，但如果其最大回撤也很大，意味着投资者需要极强的心理承受能力来度过中间漫长的亏损期。因此，最大回撤与收益率、夏普比率等指标同样重要，需要综合评估。
