因为现在有一些股票不满足我们的要求，所以我们需要执行调仓。为了方便观察计时器是否起作用，我们可以先打印一条信息。

以下是调仓操作的具体步骤：
1.  对于需要删除的每一只股票，我们将其持仓量设置为零，即全部卖出。这里使用 `order_target_percent` 函数。
    ```python
    order_target_percent(stock, 0)
    ```
2.  卖出不符合条件的股票后，我们需要买入新的股票。对于当前股票池中的每一只股票，我们平均分配资金进行买入。
    ```python
    order_target_percent(stock, 1.0 / len(stock_list))
    ```

我们的策略是基于市值和市净率等因子进行选股，并遵循了“三步走”的流程，其中也包括了数据预处理操作。

![](img/4dd5019f5943f76916c9f1770de85ff5_123.png)

![](img/4dd5019f5943f76916c9f1770de85ff5_107.png)

![](img/4dd5019f5943f76916c9f1770de85ff5_125.png)

![](img/4dd5019f5943f76916c9f1770de85ff5_127.png)

以下是整个策略的核心步骤总结：
1.  **初始化与定时**：在策略的 `__init__` 构造函数中，设置定时器（如每月、每周或每日），决定调仓频率。
2.  **初步过滤**：首先过滤掉不符合基本条件的股票（如ST股、停牌股）。
3.  **因子查询与处理**：对剩余的股票，查询所需的因子数据（如市值、市净率）。然后对因子数据进行预处理，例如中性化处理。
4.  **股票筛选**：根据处理后的因子值进行筛选。例如，我们认为市净率越低越好，因此筛选出市净率小于等于某个分位数的股票。
5.  **调仓操作**：
    *   卖出：检查当前持仓中，有哪些股票不在最新的股票池里，将这些股票的持仓比例设置为0（`order_target_percent(stock, 0)`）。
    *   买入：对最新股票池中的所有股票，平均分配资金买入（`order_target_percent(stock, 1.0/len(stock_list))`）。
