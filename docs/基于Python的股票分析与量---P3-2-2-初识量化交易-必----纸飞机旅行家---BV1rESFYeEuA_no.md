### 基于交易产品的分类

基于交易产品的分类，主要关注量化策略所交易的具体金融产品类型。

以下是几种常见的策略类型：

![](img/cbf4b64422844f08494fa49588b1816a_15.png)

![](img/cbf4b64422844f08494fa49588b1816a_17.png)

*   **股票策略**：顾名思义，这类策略主要交易股票。策略在股票市场中进行，关注股票的价格变动。
*   **CTA策略 (管理期货)**：这类策略主要交易期货。对于金融知识不太了解的初学者，可以将其理解为一种基于多空交易机制、关注资金头寸的策略。它不关心股价的绝对涨跌，而是关注价格波动。波动越频繁，CTA策略通常获利机会越多。
*   **期权策略**：这类策略主要交易期权，更关注波动率等复杂因素。
*   **FOHF策略 (Fund of Funds， 母基金)**：这类策略投资于其他基金（如股票基金、债券基金、期货基金等），将这些基金作为投资标的。其目的是通过分散投资来更好地控制风险，并追求更高的收益。

### 基于盈利模式的分类

基于盈利模式的分类，关注策略如何获取利润。

以下是三种主要的盈利模式：

*   **单边多空策略**：策略方向明确，要么做多（看涨），要么做空（看跌）。例如“追涨杀跌”或相反的“追跌杀涨”就属于此类。
*   **套利策略**：这种策略追求稳定的、无风险的收益。其核心思想是捕捉不同资产之间价格关系的短暂失衡。例如，当资产A价格上涨而资产B价格下跌时，通过按一定比例同时买入和卖出A、B，无论市场整体方向如何，都可能锁定利润。博彩公司设计赔率也是类似的套利逻辑。
*   **对冲策略**：这类策略旨在降低投资组合的整体风险。传统的套利策略就是一种对冲策略。现代对冲策略还包括：
    *   **阿尔法策略**：追求超越市场基准（如沪深300指数）的超额收益。
    *   **市场中性策略**：策略收益与市场涨跌无关，旨在获取绝对收益。
    *   **事件驱动策略**：基于特定公司事件（如并购、财报发布）进行交易。

### 基于策略信号的分类

基于策略信号的分类，可以理解为策略背后所依赖的数学模型或思想流派。

以下是几种常见的策略信号类型：

*   **多因子模型**：这是量化选股的核心方法之一。它将股票的收益归因于多个“因子”（如市值、市盈率PE、行业、动量等）。策略通过寻找有效的因子组合来预测股票表现。公式可以简化为：`股票收益 = 因子1的暴露度 * 因子1的收益 + 因子2的暴露度 * 因子2的收益 + ... + 残差`。
*   **均值回归**：该模型认为，资产价格围绕其长期均值波动，偏离均值后最终会回归。例如，一只股票短期内大幅上涨后，可能预示着未来会下跌；反之亦然。
*   **动量效应**：该模型认为，资产价格具有延续原有运动趋势的惯性。即过去一段时间表现好的资产，未来短期内可能继续表现好；表现差的可能继续差。
*   **二八轮动**：该策略思想源于“二八定律”，认为市场上20%的股票（或20%的时间）贡献了80%的收益。策略旨在捕捉这领头上涨的20%部分。
*   **海龟交易法则**：这是一种趋势跟踪策略，它通过设定价格通道（如最高价/最低价的N日突破）来产生买卖信号，旨在捕捉大的价格趋势。
*   **机器学习策略**：这是较新的领域，利用机器学习算法（如随机森林、支持向量机、深度学习等）从历史数据中自动发现模式和规律，并用于预测。例如，可以将各种市场特征输入神经网络进行训练。代码框架可能如下：
    ```python
    # 伪代码示例
    from sklearn.ensemble import RandomForestRegressor
    model = RandomForestRegressor()
    model.fit(X_train, y_train) # X_train是特征数据，y_train是收益标签
    predictions = model.predict(X_test)
    ```

## 本节总结

![](img/cbf4b64422844f08494fa49588b1816a_21.png)

本节课中我们一起学习了量化交易的几种主要分类方式。我们回顾了基于**交易产品**（如股票、期货、期权）的分类，理解了基于**盈利模式**（单边、套利、对冲）的区别，并初步认识了基于**策略信号**背后的不同数学模型和思想流派（如多因子、均值回归、机器学习等）。理解这些分类有助于我们构建对量化交易领域的整体认知框架，为后续深入学习具体策略打下基础。量化交易模型从设计到实盘是一个需要不断迭代和优化的过程，其最终检验标准是在市场中持续盈利。
