## 研发投入数据概述 📊

![](img/b4852800f3c9b9ec95b5bd1d146208c6_11.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_13.png)

研发投入是指公司为研究与开发项目所支付的费用。上市公司在每个季度的财务报告中都会披露相应的研发费用数据。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_15.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_17.png)

例如，一份年报可能显示某公司在2023年的研发费用为28亿元。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_19.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_21.png)

我们统计了A股历史上所有股票在财报中的研发费用数据，并整理成了一份全面的数据集。这份数据不仅包含每家公司每个季度的研发费用，还汇总了其最近四个季度的研发费用总和。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_23.png)

## A股研发投入的初步统计 🏆

![](img/b4852800f3c9b9ec95b5bd1d146208c6_25.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_27.png)

利用上述数据，我们进行了一些简单的统计。一个有趣的发现是，在2023年，A股研发费用最多的公司是中国建筑，其总投入达到460亿元。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_29.png)

这似乎与大众对建筑行业“科技含量不高”的刻板印象相悖。进一步查看其研报，会发现该公司拥有大量专利和院士工作站，其创新水平不容小觑。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_31.png)

在研发费用榜单中，除了位居第二的比亚迪，其他公司无一例外都是大型国有企业。这表明，许多大型国企能成为行业龙头，不仅依靠其国资背景，其科研水平和创新能力也处于行业前列。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_33.png)

关于研发费用数据，还有一个重要的背景知识：**研发费用**这个字段是从2018年三季报才开始被强制要求披露的。在此之前，绝大部分公司不会单独披露此项，而是将其合并到“管理费用”中。当然，也有部分公司（如中兴通讯）多年来一直坚持披露。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_35.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_37.png)

> **注意**：也存在公司通过操纵研发费用进行财务造假的情况，例如将非研发支出计入研发费用以获取税收优惠或粉饰报表。这些是不正当手段，仅作了解，切勿模仿。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_39.png)

## 构建基于研发费用的选股策略 💡

![](img/b4852800f3c9b9ec95b5bd1d146208c6_41.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_43.png)

上一节我们介绍了研发费用的基本概念和数据情况，本节中我们来看看如何用它构建一个量化选股策略。我们的核心问题是：买入研发费用高的公司股票，能否帮助我们赚钱？

![](img/b4852800f3c9b9ec95b5bd1d146208c6_45.png)

我们设计了一个简单的月度调仓策略：

![](img/b4852800f3c9b9ec95b5bd1d146208c6_47.png)

1.  **选股时点**：在每个月的最后一个交易日收盘后。
2.  **排序标准**：按照**过去四个季度研发费用总和**从大到小对所有股票进行排序。
3.  **筛选**：剔除当时处于ST状态、退市或停牌的股票。
4.  **构建组合**：买入研发费用最高的10只股票。
5.  **交易**：在下个月第一个交易日开盘时，等权重买入该组合，并持有整个月。
6.  **调仓**：在下个月最后一个交易日收盘时卖出全部股票，然后重复步骤1-5，构建新的组合。

**策略逻辑公式化**：
```
每月末，选择股票池中满足条件（非ST、非退市、非停牌）的股票。
依据指标：研发费用_过去一年总和 = SUM(最近四个季度的研发费用)
对股票按该指标降序排列，选取前10名构成投资组合。
```

![](img/b4852800f3c9b9ec95b5bd1d146208c6_49.png)

我们使用历史数据和Python代码对这个策略进行了回测。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_51.png)

## 策略回测结果与分析（绝对值）📉

![](img/b4852800f3c9b9ec95b5bd1d146208c6_53.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_55.png)

程序运行结果显示，代表策略净值的橙色曲线从初始的1元增长到约1.199元，略微跑赢了代表大盘的沪深300指数。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_57.png)

然而，该策略的年化收益率仅为3.95%，最大回撤却超过30%。这意味着，单纯买入研发费用绝对值最高的公司，其收益甚至可能不如银行存款。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_59.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_61.png)

这个结果提示我们，使用研发费用的**绝对值**存在明显问题：它很容易选中那些规模庞大、营收基数大的公司（如大型国企）。这些公司研发投入的绝对值自然很高，但这并不一定代表其**重视研发**的程度更高。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_63.png)

## 引入研发费用占比指标 🔍

![](img/b4852800f3c9b9ec95b5bd1d146208c6_65.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_67.png)

上一节我们发现绝对值策略效果不佳，本节我们引入一个更合理的指标：**研发费用占比**。

其计算公式为：
```
研发费用占比 = 研发费用 / 主营业务收入
```
这个指标越大，说明公司越愿意将收入投入到研发中，更能体现公司对研发的重视程度。

例如：
*   公司A年营收1万亿元，研发投入100亿元，研发费用占比为 **1%**。
*   公司B年营收500亿元，研发投入50亿元，研发费用占比为 **10%**。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_69.png)

虽然公司A的研发投入绝对值更高，但公司B显然更注重研发。如果研发能有效转化为产出，公司B可能受益更大。

那么，买入研发费用占比高的公司，效果如何呢？

![](img/b4852800f3c9b9ec95b5bd1d146208c6_71.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_73.png)

我们修改了策略，将选股标准从“研发费用绝对值”改为“研发费用占比”，并进行了回测。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_75.png)

## 策略回测结果与分析（相对值）📊

回测结果令人失望。代表策略的净值曲线从1元下跌至0.72元，与沪深300指数的表现同样糟糕。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_77.png)

该策略的年化收益率为-7.18%，最大回撤超过50%。这表明，即使使用了理论上更合理的相对值指标，如果将其应用于全市场所有股票，依然无法获得有效收益。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_79.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_81.png)

至此我们发现，无论是研发费用的绝对值还是相对值，似乎都无法作为独立的选股因子在全市场范围内使用。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_83.png)

## 分行业研究的突破 🎯

![](img/b4852800f3c9b9ec95b5bd1d146208c6_85.png)

![](img/b4852800f3c9b9ec95b5bd1d146208c6_87.png)

恭喜留下来的同学，下面才是真正的正片开始。前面的策略在全市场失效，促使我们思考一个关键问题：研发投入对所有行业都同样重要吗？

![](img/b4852800f3c9b9ec95b5bd1d146208c6_89.png)

显然不是。对于酒店、奶茶等行业，将大量营收投入研发可能并非明智之举；而对于科技、高端制造等行业，研发则是生命线。之前策略的问题在于对所有行业“一刀切”。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_91.png)

因此，我们改进思路：**分行业进行选股**。只在特定行业内，选择研发费用占比最高的公司。

以下是我们在两个行业中的测试结果：

![](img/b4852800f3c9b9ec95b5bd1d146208c6_93.png)

**1. 纺织服饰行业**
*   **策略**：在纺织服饰行业内，每月选择研发费用占比最高的3只股票。
*   **结果**：策略净值从1元增长至2.82元，年化收益率达26%，最大回撤44%。该策略不仅大幅跑赢沪深300指数，也显著跑赢了行业平均涨幅（行业年化收益仅3.9%），产生了约22%的超额收益。

**2. 建筑材料行业**
*   **策略**：在建筑材料行业内，每月选择研发费用占比最高的3只股票。
*   **结果**：策略年化收益率为23%，相比整个行业也有约20%的超额收益。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_95.png)

当然，并非所有行业都适用。例如在商贸零售行业，使用该指标的策略年化收益为-12%，最大回撤70%，大幅跑输行业。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_97.png)

## 结论与总结 🏁

![](img/b4852800f3c9b9ec95b5bd1d146208c6_99.png)

本节课中我们一起学习了如何利用研发投入数据构建量化选股策略，并得出了以下核心结论：

![](img/b4852800f3c9b9ec95b5bd1d146208c6_101.png)

1.  **单一因子无效**：在全市场范围内，单纯使用研发费用的**绝对值**或**相对值（占比）** 作为选股因子，无法获得稳定超额收益。
2.  **行业属性关键**：研发投入的重要性因行业而异。在技术创新驱动型行业（如部分制造业、信息技术业），高研发费用占比是公司竞争力的重要体现，相关选股策略可能有效。
3.  **量化需要精细化**：成功的量化策略往往需要对市场进行细分和深入理解。一个因子在某些领域是“宝藏”，在另一些领域则可能是“陷阱”。

![](img/b4852800f3c9b9ec95b5bd1d146208c6_103.png)

我们已将研发费用占比因子在更多行业的表现结果整理成表。通过该表，你可以清晰了解哪些行业适合将研发费用作为选股指标，哪些行业则不适合。

> **核心收获**：在量化投资中，脱离行业背景孤立地使用财务指标通常是危险的。**“研发费用占比”是一个有效的选股指标，但它的有效性高度依赖于所处的行业环境。**
