在本节课中，我们将学习一个基础的股票序列分析策略。该策略的核心是计算短期和长期移动平均线，并通过观察它们的交叉点来判断买入和卖出的时机，即识别“黄金交叉”与“死亡交叉”。

## 策略概述与核心概念

![](img/b1870125c7bbf1d18e75c0f615dc65a2_5.png)

![](img/b1870125c7bbf1d18e75c0f615dc65a2_6.png)

该策略需要计算两个关键指标：
*   **短期移动平均线 (M1)**：通常使用较短的时间窗口计算，例如5个交易日。
*   **长期移动平均线 (M2)**：通常使用较长的时间窗口计算，例如20或30个交易日。

策略逻辑基于这两个均线的交叉：
*   **黄金交叉**：当短期均线从下方上穿长期均线时，通常被视为**买入信号**。
*   **死亡交叉**：当短期均线从上方下穿长期均线时，通常被视为**卖出信号**。

## 计算短期与长期移动平均线

![](img/b1870125c7bbf1d18e75c0f615dc65a2_8.png)

![](img/b1870125c7bbf1d18e75c0f615dc65a2_10.png)

首先，我们需要从股票数据中计算出短期和长期的移动平均值。我们将以苹果公司（AAPL）的股价数据为例。

![](img/b1870125c7bbf1d18e75c0f615dc65a2_11.png)

以下是计算步骤的代码实现：

![](img/b1870125c7bbf1d18e75c0f615dc65a2_13.png)

```python
# 假设 `data` 是一个包含苹果股价的DataFrame，其中‘AAPL’是股价列
# 计算短期移动平均（例如，窗口为10天）
data[‘M1’] = data[‘AAPL’].rolling(window=10).mean()

![](img/b1870125c7bbf1d18e75c0f615dc65a2_15.png)

# 计算长期移动平均（例如，窗口为30天）
data[‘M2’] = data[‘AAPL’].rolling(window=30).mean()

![](img/b1870125c7bbf1d18e75c0f615dc65a2_16.png)

![](img/b1870125c7bbf1d18e75c0f615dc65a2_17.png)

![](img/b1870125c7bbf1d18e75c0f615dc65a2_18.png)

![](img/b1870125c7bbf1d18e75c0f615dc65a2_19.png)

# 查看包含原始股价和两个均线的数据前几行
print(data[[‘AAPL‘, ’M1‘, ’M2‘]].head())
```

**代码解释**：
*   `data[‘AAPL’].rolling(window=10)` 创建了一个窗口大小为10的滚动窗口对象。
*   `.mean()` 对这个滚动窗口内的数据计算平均值，从而得到移动平均线。
*   我们将结果分别赋值给新列 `M1`（短期均线）和 `M2`（长期均线）。

> **注意**：在实际操作中，窗口大小的选择（如5天、20天）是常见的，但本例因数据周期较长（2010-2018年），为让图表更清晰，我们使用了稍大的窗口（10天和30天）。

## 量化交叉信号

![](img/b1870125c7bbf1d18e75c0f615dc65a2_36.png)

![](img/b1870125c7bbf1d18e75c0f615dc65a2_37.png)

为了更精确地定位交叉点，我们可以通过计算创建一个信号列，明确标识短期均线是高于还是低于长期均线。

![](img/b1870125c7bbf1d18e75c0f615dc65a2_39.png)

以下是使用 `np.where` 函数生成信号的代码：

```python
import numpy as np

![](img/b1870125c7bbf1d18e75c0f615dc65a2_41.png)

# 生成交易信号：当M1 > M2时为1，否则为-1
data[‘position’] = np.where(data[‘M1’] > data[‘M2’], 1, -1)

![](img/b1870125c7bbf1d18e75c0f615dc65a2_42.png)

![](img/b1870125c7bbf1d18e75c0f615dc65a2_44.png)

# 为了清晰展示信号，使用双Y轴进行绘图
ax = data[[‘AAPL‘, ’M1‘, ’M2‘]].plot(figsize=(12, 6))
# 在右侧Y轴绘制‘position’信号线（取值为1或-1）
data[‘position’].plot(ax=ax, secondary_y=True, style=‘g-’)
plt.show()
```

**代码解释**：
*   `np.where(condition, x, y)` 是一个条件赋值函数。当条件为真时返回值 `x`，为假时返回值 `y`。
*   这里，`condition` 是 `data[‘M1’] > data[‘M2’]`。如果短期均线高于长期均线，`position` 列赋值为 `1`，否则赋值为 `-1`。
*   在图表中，**右侧Y轴**的绿色信号线清晰地展示了“1”和“-1”的转换点，这些点正好对应了图表左侧的**黄金交叉**（信号由-1变为1）和**死亡交叉**（信号由1变为-1）。
