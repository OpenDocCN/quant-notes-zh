## 概述

回归分析用于研究一个或多个自变量（X）与一个因变量（Y）之间的关系。本节课程将演示如何使用Python的`statsmodels`库进行回归建模，从数据读取、相关系数计算到模型构建与解读。

我们使用`pandas`库读取数据文件，并使用`scipy`计算相关系数。以下代码展示了如何计算并展示数据集中各变量间的相关系数矩阵。

## 一元线性回归

上一节我们查看了变量间的相关性，本节中我们来看看如何建立具体的一元回归模型。一元回归是最简单的回归形式，它描述一个因变量（Y）与一个自变量（X）之间的关系。

以下是使用`statsmodels`库进行一元线性回归的步骤。我们以“每股收益”为因变量（Y），“每股净资产”为自变量（X）进行建模。

```python
import statsmodels.api as sm
# 定义自变量和因变量
X = df[‘每股净资产’]
Y = df[‘每股收益’]
# 为模型添加常数项（截距）
X = sm.add_constant(X)
# 建立OLS模型并拟合
model = sm.OLS(Y, X).fit()
# 查看模型摘要
print(model.summary())
```

![](img/c8177b1dd58d116cb0d1654f63c27931_7.png)

模型摘要提供了丰富的信息，我们需要重点关注以下几点：
*   **R-squared**：模型解释力，此处为0.786。
*   **系数显著性**：查看自变量`每股净资产`的`P>|t|`值。若小于0.05，则认为该变量对因变量的影响是显著的。本例中该系数显著为正。
*   **截距项**：本例中截距项显著为负。

结果表明，每股净资产越高，每股收益也越高，二者存在显著的正向关系。

## 多元线性回归

在实际研究中，因变量往往受多个因素影响。因此，我们需要使用多元回归模型。本节我们将在一元回归的基础上，引入更多自变量。

![](img/c8177b1dd58d116cb0d1654f63c27931_9.png)

以下是进行多元回归的代码示例。我们在模型中加入了“每股现金流量”和“总股本”作为额外的自变量。

![](img/c8177b1dd58d116cb0d1654f63c27931_11.png)

```python
# 定义多个自变量
X_multi = df[[‘每股净资产‘， ’每股现金流量‘， ’总股本‘]]
Y = df[‘每股收益’]
# 添加常数项
X_multi = sm.add_constant(X_multi)
# 建立并拟合多元OLS模型
model_multi = sm.OLS(Y, X_multi).fit()
# 查看模型摘要
print(model_multi.summary())
```

对比一元回归模型，多元回归模型的结果显示：
*   **R-squared提升**：从0.786增加至0.87，表明新加入的变量提升了模型的整体解释力。
*   **模型评价指标改善**：AIC和BIC值均有所下降，说明模型拟合效果更好。
*   **系数解读**：
    *   `每股净资产`系数显著为正。
    *   `每股现金流量`系数显著为正。
    *   `总股本`系数显著为负。这符合经济学直觉：在总收益不变的情况下，总股本（分母）越大，每股收益（分子）越小。

从代码可以看出，使用Python进行多元回归与一元回归的步骤非常相似，主要区别在于自变量的数量。
