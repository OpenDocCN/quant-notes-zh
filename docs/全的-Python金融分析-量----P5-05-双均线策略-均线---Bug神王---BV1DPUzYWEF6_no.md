## 均线概念与计算原理

数据准备就绪后，我们来看核心概念：移动平均线。

移动平均线是技术分析中常用的指标。对于每一个交易日，都可以计算出前N天收盘价的移动平均值。将这些每日的移动平均值连接起来，形成的曲线就是N日移动平均线，简称“均线”。

常用的均线周期有5日、10日（短期均线）、30日、60日（中期均线）等。

![](img/8fce2d7fb5c3a7b36f9167c43d82c34c_7.png)

其计算公式为：
**MA(N) = (C1 + C2 + C3 + … + CN) / N**
其中，`MA(N)`代表第N日的移动平均值，`C1`到`CN`分别代表前N个交易日的收盘价。

为了更直观地理解，假设我们有连续8天的收盘价。计算5日均线时：
*   首先计算第1到第5天收盘价的平均值，得到第一个点（X1）。
*   然后计算第2到第6天收盘价的平均值，得到第二个点（X2）。
*   接着计算第3到第7天收盘价的平均值，得到第三个点（X3）。
*   最后计算第4到第8天收盘价的平均值，得到第四个点（X4）。
将X1, X2, X3, X4这四个点连接起来，就构成了这8天时间窗口内的5日均线。

## 计算5日与30日均线

理解了原理后，我们使用Python计算茅台的5日和30日移动平均线。

![](img/8fce2d7fb5c3a7b36f9167c43d82c34c_9.png)

以下是计算步骤：

1.  **计算5日均值**：使用`rolling`窗口函数，窗口大小为5，并对每个窗口计算均值（`mean`）。
    ```python
    ma5 = df[‘close’].rolling(5).mean()
    ```
    这里，`ma5`是一个`Series`，其前4个值为`NaN`（因为前4天无法构成完整的5日窗口），从第5个值开始才是有效的5日移动平均值。

2.  **计算30日均值**：同理，计算30日移动平均线。
    ```python
    ma30 = df[‘close’].rolling(30).mean()
    ```
    `ma30`的前29个值为`NaN`。

至此，我们得到了5日和30日的移动平均值序列，这些值就是绘制均线所需的坐标点。

## 均线可视化

虽然matplotlib绘图模块的详细讲解不在本节范围，但我们可以初步使用它来观察计算出的均线。

以下是绘制均线的代码：

```python
import matplotlib.pyplot as plt
%matplotlib inline # 在Jupyter Notebook中显示图表

plt.plot(ma5[50:80], label=‘MA5’) # 绘制部分区间的5日均线
plt.plot(ma30[50:80], label=‘MA30’) # 绘制部分区间的30日均线
plt.legend() # 显示图例
plt.show()
```

运行代码后，可以看到两条曲线。波动较为频繁的是5日均线，相对平缓的是30日均线。两条线的交叉点（金叉或死叉）在双均线策略中具有重要的交易信号意义。
