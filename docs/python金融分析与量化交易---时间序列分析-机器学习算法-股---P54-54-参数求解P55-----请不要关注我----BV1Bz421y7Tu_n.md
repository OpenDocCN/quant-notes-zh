## 🔍 从最大似然到最小二乘
上一节我们介绍了如何通过最大似然估计来构建线性回归的目标。本节中，我们来看看如何对这个目标进行化简和求解。

首先，我们回顾一下似然函数。为了便于计算，我们对其取对数，将连乘转换为累加。

以下是化简后的对数似然函数：
```
L(θ) = Σ_{i=1}^{m} [log(A) + log(B)] + 常数项
```
其中，`A` 是与参数 `θ` 无关的常数部分，`B` 是包含 `θ` 的核心部分。

我们的目标是使似然函数 `L(θ)` 的值**越大越好**。由于 `L(θ)` 等于一个常数项减去一个恒为正值的部分，因此，要使 `L(θ)` 最大，就必须让减去的这个部分**越小越好**。

于是，我们单独提取出这个需要最小化的部分，定义为一个新的目标函数 `J(θ)`：
```
J(θ) = 1/2 * Σ_{i=1}^{m} (y_i - θ^T x_i)^2
```
这个函数有一个更广为人知的名字：**最小二乘法**。它的含义是，我们希望模型预测值 `θ^T x_i` 与真实值 `y_i` 之间的误差平方和越小越好。

![](img/008d0978b0943b9be554a0862d36ab28_2.png)

**核心要点**：
*   最大似然估计是许多机器学习算法的理论基础。
*   在线性回归的特定假设下，最大化似然等价于最小化误差平方和。
*   最小二乘法的目标就是找到一组参数 `θ`，使 `J(θ)` 最小。

## 🧮 参数求解：解析解
现在，我们有了明确的目标函数 `J(θ)`。接下来，我们看看如何求解能使 `J(θ)` 最小的参数 `θ`。

![](img/008d0978b0943b9be554a0862d36ab28_8.png)

这里需要注意，`x` 不是一个单独的数字，而是一个包含多个特征的向量。因此，`X` 是一个矩阵（每行是一个样本，每列是一个特征），`y` 是一个向量，`θ` 也是一个向量。

![](img/008d0978b0943b9be554a0862d36ab28_10.png)

我们的目标是：
```
argmin_θ J(θ)
```
在数学上，求解函数极值的一个常用方法是求导并令导数为零。因此，我们需要计算 `J(θ)` 对 `θ` 的偏导数，并令其等于0。

![](img/008d0978b0943b9be554a0862d36ab28_12.png)

![](img/008d0978b0943b9be554a0862d36ab28_14.png)

以下是求解的关键步骤和最终公式：
1.  将 `J(θ)` 用矩阵形式重写：`J(θ) = 1/2 (Xθ - y)^T (Xθ - y)`
2.  对 `θ` 求偏导。这里会用到矩阵求导的公式（了解即可，无需记忆）。
3.  令导数等于零，得到方程：`X^T X θ = X^T y`
4.  为了解出 `θ`，我们在等式两边同时左乘 `(X^T X)` 的逆矩阵（如果存在）。

最终，我们得到参数 `θ` 的**解析解（闭式解）**：
```
θ = (X^T X)^{-1} X^T y
```

**代码描述核心公式**：
```python
# 假设 X 是特征矩阵，y 是目标值向量
import numpy as np
# 使用解析解直接计算参数 theta
theta = np.linalg.inv(X.T @ X) @ X.T @ y
```

## 🤔 对解析解的思考
通过公式 `θ = (X^T X)^{-1} X^T y`，我们似乎直接“计算”出了答案。但这里引出了两个重要问题：

**1. 这算是“学习”吗？**
这个过程更像是一次性的数学计算，而非我们通常理解的、通过多次迭代试错来逐步改进的“学习”过程。它缺乏渐进优化的步骤。

**2. 公式永远成立吗？**
不总是成立。这个解成立的前提是矩阵 `(X^T X)` 必须是**可逆**的。但在实际数据中，如果特征之间存在严重的多重共线性，或者样本数少于特征数，这个矩阵就可能是奇异的（不可逆），导致无法求解。

正是由于这些局限性（缺乏迭代学习过程、对矩阵可逆性的依赖），我们在实际中并不总是使用这种解析解方法。下一节，我们将介绍另一种更通用、更接近“学习”本质的求解方法——**梯度下降法**，它能很好地解决这些问题。

## 📝 总结
本节课中我们一起学习了线性回归模型的参数求解全过程：
1.  我们从**最大似然估计**出发，推导出了**最小二乘法**的目标函数。
2.  通过矩阵运算和求导，得到了参数 `θ` 的**解析解公式**：`θ = (X^T X)^{-1} X^T y`。
3.  我们分析了该解析解方法的优点（直接计算）和缺点（非迭代学习、依赖矩阵可逆性）。
