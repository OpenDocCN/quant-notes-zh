在本节课中，我们将学习量化交易策略中的一个核心环节——因子中性化（或称因子提纯）。我们将通过一个具体的例子，即从市净率（PB）因子中剔除市值（Size）的影响，来详细讲解其背后的原理和实现步骤。

上一节我们介绍了因子标准化等预处理方法，本节中我们来看看如何通过回归分析来“提纯”因子，使其更纯粹地反映我们希望衡量的信息。

## 因子中性化的核心思想 🎯

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_2.png)

假设我们有一个市净率（PB）因子。我们发现，市净率与市值（Size）存在较大关联。这意味着市净率因子中，包含了一部分由市值解释的信息。为了更纯粹地使用市净率因子，我们希望剔除其中与市值相关的部分。

这个过程可以类比为：将市净率因子视为一个整体，市值是其中的一个组成部分。我们的目标是将市值这个“蓝色”部分从市净率这个“红色”整体中分离并剔除出去。

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_4.png)

## 如何实现因子提纯？ 🔍

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_6.png)

实现因子提纯的核心方法是线性回归。以下是具体的操作步骤：

第一步，建立回归方程。我们将因子（如市净率）作为因变量 `Y`，将希望剔除的影响因素（如市值）作为自变量 `X`。建立如下回归方程：
`Y = W * X + B`
其中，`W` 是回归系数，`B` 是截距项。通过求解这个方程，我们就能知道市值 `X` 解释了因子 `Y` 中的多大一部分。

第二步，计算预测值与残差。通过回归方程，我们可以得到因子 `Y` 的预测值 `Y_pred`：
`Y_pred = W * X + B`
这个预测值 `Y_pred` 代表了因子中能被市值所解释的部分。而真实值 `Y` 与预测值 `Y_pred` 之间的差异，即残差 `Residual`，则代表了市值无法解释的部分，也就是我们“提纯”后想要保留的因子信息：
`Residual = Y - Y_pred`

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_8.png)

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_10.png)

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_12.png)

因此，因子中性化的过程，就是用因子的真实值减去其被其他变量（如市值）解释的部分，从而得到更“纯净”的因子值。

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_14.png)

## 策略任务流程概述 📈

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_16.png)

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_18.png)

接下来，我们将把因子中性化整合到一个完整的选股策略流程中。以下是该策略的主要步骤：

![](img/8eb2ba08be8345fe299f0ce5f80a37e9_20.png)

以下是策略执行的具体步骤列表：
1.  **数据准备**：获取市净率（PB）和市值（Size）等基础因子数据。
2.  **股票池初筛**：剔除不符合条件的股票，例如停牌股、ST股、上市时间不足半年的新股等。
3.  **因子数据处理**：对筛选后的因子数据依次进行去极值、标准化和中性化处理。
4.  **策略选股**：基于处理后的“纯净”因子制定选股规则。例如，筛选出市净率小于0.2的股票作为买入候选。
5.  **回测与评估**：根据选股结果进行回测，评估策略的有效性。
