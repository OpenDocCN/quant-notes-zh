## 第一步：去极值处理

上一节我们介绍了预处理的三步走策略，本节中我们来看看第一步——去极值。去极值的目的是处理数据中的离群点或异常值，而不是简单地删除它们。常见的做法是设定一个界限，将超出界限的数值“拉回”到界限值上。

### 分位数去极值法

![](img/3644bfe05121ed35db93e0f85f0e8209_9.png)

分位数是一种描述数据分布位置的方法。最熟悉的概念是中位数，它代表数据中间位置的值。与均值相比，中位数对极端值不敏感，更能代表数据的整体情况。

![](img/3644bfe05121ed35db93e0f85f0e8209_10.png)

![](img/3644bfe05121ed35db93e0f85f0e8209_11.png)

![](img/3644bfe05121ed35db93e0f85f0e8209_12.png)

除了中位数（Q2），常用的还有第一四分位数（Q1）和第三四分位数（Q3）。
*   **Q1**：表示所有数据中，从小到大排列处于前25%位置的值。
*   **Q2（中位数）**：表示处于50%位置的值。
*   **Q3**：表示处于75%位置的值。

![](img/3644bfe05121ed35db93e0f85f0e8209_14.png)

在去极值操作中，我们利用这些分位数来设定数据的正常范围边界。

### 代码示例：分位数去极值

在实际操作前，需要导入必要的工具包并加载数据。这里以股价数据为例进行演示，在实际平台操作时会使用真实的因子数据（如市净率）。

```python
# 导入必要的库
import pandas as pd
import numpy as np

![](img/3644bfe05121ed35db93e0f85f0e8209_16.png)

# 加载数据（此处为示例，实际使用因子数据）
data = pd.read_csv(‘stock_data.csv’)
factor_data = data[‘close’]  # 假设‘close’列是我们的因子数据

# 定义分位数去极值函数
def winsorize_by_quantile(series, q_low=0.025, q_high=0.975):
    """
    使用分位数进行去极值处理。
    参数:
        series: 待处理的数据序列（Pandas Series）。
        q_low: 下分位数，默认2.5%。
        q_high: 上分位数，默认97.5%。
    返回:
        处理后的数据序列。
    """
    low_bound = series.quantile(q_low)
    high_bound = series.quantile(q_high)
    # 将小于下界的值设为下界，大于上界的值设为上界
    series_winsorized = series.clip(lower=low_bound, upper=high_bound)
    return series_winsorized

# 应用去极值函数
processed_data = winsorize_by_quantile(factor_data)
print(“原始数据范围:”, factor_data.min(), “-”, factor_data.max())
print(“去极值后范围:”, processed_data.min(), “-”, processed_data.max())
```

这段代码定义了一个函数，它计算因子的指定分位数（如2.5%和97.5%）作为边界，然后将所有超出边界的数据值调整到边界值上。
