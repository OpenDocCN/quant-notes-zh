## 综合打分法详解 🧮

![](img/44e6fc6cafab239c8dd062634753565e_19.png)

![](img/44e6fc6cafab239c8dd062634753565e_20.png)

综合打分法的核心思想是：将多个有效因子结合起来，为每只股票计算一个综合得分，然后根据得分高低选择股票。

![](img/44e6fc6cafab239c8dd062634753565e_21.png)

![](img/44e6fc6cafab239c8dd062634753565e_23.png)

以下是实施综合打分法的具体步骤：

![](img/44e6fc6cafab239c8dd062634753565e_25.png)

![](img/44e6fc6cafab239c8dd062634753565e_27.png)

**第一步：确定当期有效因子**
根据IC值筛选结果，确定当前月份用于打分的因子列表。

![](img/44e6fc6cafab239c8dd062634753565e_29.png)

![](img/44e6fc6cafab239c8dd062634753565e_31.png)

**第二步：对单个因子进行分组打分**
对于每个有效因子，我们根据其IC值的正负来决定打分逻辑：
*   如果因子的IC值 **大于0**，意味着因子值与未来收益率正相关。因此，我们给因子值**较大**的股票打**高分**。
*   如果因子的IC值小于0（本例中未出现），则给因子值**较小**的股票打**高分**。

![](img/44e6fc6cafab239c8dd062634753565e_33.png)

具体的打分操作是将该因子下的所有股票按因子值大小分为5组（Quintile）：
*   因子值最小的20%股票得 **1分**
*   次小的20%股票得 **2分**
*   ... 以此类推 ...
*   因子值最大的20%股票得 **5分**

![](img/44e6fc6cafab239c8dd062634753565e_35.png)

**第三步：计算股票综合得分**
遍历所有有效因子，对每只股票在每个因子上的得分进行求和，得到该股票的综合得分。

![](img/44e6fc6cafab239c8dd062634753565e_37.png)

![](img/44e6fc6cafab239c8dd062634753565e_39.png)

```python
# 伪代码示例：计算股票综合得分
def calculate_composite_score(stock, effective_factors):
    total_score = 0
    for factor in effective_factors:
        # 获取该股票在此因子上的分组（1-5）
        group = get_factor_quintile(stock, factor)
        # 根据IC值方向调整分数
        if ic_dict[factor] > 0:
            score = group  # IC>0，组号即分数
        else:
            score = 6 - group # IC<0，反向打分（第5组得1分，第1组得5分）
        total_score += score
    return total_score
```

![](img/44e6fc6cafab239c8dd062634753565e_41.png)

运行上述代码后，我们会得到一个股票及其综合得分的列表。例如，股票A得4分，股票B得19分。得分越高，代表该股票在多数有效因子上的表现越好。

**第一步：选择股票**
我们按综合得分从高到低排序，选择排名靠前的股票。常见的做法是选择排名前5%、前10%或前20%的股票。

*   持有3个月后，计算该期间投资组合的收益率。组合收益率等于所有成分股票收益率的算术平均值。

## 绩效评估指标 📈

![](img/44e6fc6cafab239c8dd062634753565e_89.png)

![](img/44e6fc6cafab239c8dd062634753565e_91.png)

仅看收益曲线不够全面，我们还需要一套量化指标来评估策略的风险与收益特征。

![](img/44e6fc6cafab239c8dd062634753565e_93.png)

![](img/44e6fc6cafab239c8dd062634753565e_95.png)

以下是关键的绩效评估指标及其计算逻辑：

![](img/44e6fc6cafab239c8dd062634753565e_97.png)

![](img/44e6fc6cafab239c8dd062634753565e_99.png)

**1. 累计收益率**
策略在整个回测期间的总收益。
`累计收益率 = (期末净值 / 期初净值) - 1`

![](img/44e6fc6cafab239c8dd062634753565e_101.png)

![](img/44e6fc6cafab239c8dd062634753565e_103.png)

**2. 年化收益率**
将累计收益率折算成年平均收益，便于比较。
`年化收益率 = (1 + 累计收益率)^(252 / 回测天数) - 1`

![](img/44e6fc6cafab239c8dd062634753565e_105.png)

**3. 最大回撤**
策略净值从最高点到最低点的最大跌幅，衡量历史最大亏损风险。
`最大回撤 = (波峰净值 - 波谷净值) / 波峰净值`

![](img/44e6fc6cafab239c8dd062634753565e_107.png)

![](img/44e6fc6cafab239c8dd062634753565e_109.png)

**4. 夏普比率**
衡量每承受一单位总风险，所获得的超额收益。
`夏普比率 = (年化收益率 - 无风险利率) / 年化波动率`

![](img/44e6fc6cafab239c8dd062634753565e_111.png)

![](img/44e6fc6cafab239c8dd062634753565e_113.png)

**5. 索提诺比率**
类似于夏普比率，但只考虑下行波动率（坏波动），更关注下跌风险。
`索提诺比率 = (年化收益率 - 无风险利率) / 年化下行波动率`

![](img/44e6fc6cafab239c8dd062634753565e_115.png)

![](img/44e6fc6cafab239c8dd062634753565e_117.png)

**6. 阿尔法 & 贝塔**
基于资本资产定价模型：
*   **贝塔**：衡量策略相对于市场基准的波动性（系统性风险）。
*   **阿尔法**：衡量策略超越市场基准的超额收益（选股能力）。
`策略收益 = 无风险利率 + 贝塔 * (市场收益 - 无风险利率) + 阿尔法`

![](img/44e6fc6cafab239c8dd062634753565e_119.png)

![](img/44e6fc6cafab239c8dd062634753565e_121.png)

**7. 信息比率**
衡量策略超越基准的稳定性和效率。
`信息比率 = (策略收益 - 基准收益)的年化均值 / 跟踪误差的年化值`

![](img/44e6fc6cafab239c8dd062634753565e_123.png)

![](img/44e6fc6cafab239c8dd062634753565e_125.png)

将前5%、10%、20%股票组合的绩效指标与基准对比后，我们通常会发现：
*   前5%组合的**年化收益率**和**夏普比率**最高。
*   但前20%组合的**最大回撤**可能更大，表明其风险更高。
*   所有策略组合的**阿尔法**和**信息比率**均为正，说明策略能持续创造超额收益。

## 总结

![](img/44e6fc6cafab239c8dd062634753565e_133.png)

![](img/44e6fc6cafab239c8dd062634753565e_135.png)

本节课中，我们一起学习了量化选股策略构建与评估的核心流程。

![](img/44e6fc6cafab239c8dd062634753565e_137.png)

![](img/44e6fc6cafab239c8dd062634753565e_139.png)

1.  **综合打分法**：我们利用IC值筛选有效因子，通过分组打分的方式计算每只股票的综合得分。
2.  **组合构建**：根据得分高低选择排名靠前的股票，以等权重、季度调仓的方式构建投资组合。
3.  **收益计算**：计算组合的季度收益率，并绘制累计收益曲线与基准对比。
4.  **绩效评估**：通过累计收益率、年化收益率、最大回撤、夏普比率、阿尔法贝塔等一系列指标，全面评估策略的收益能力和风险水平。

![](img/44e6fc6cafab239c8dd062634753565e_141.png)

综合打分法是一种直观有效的多因子合成方法。结果显示，基于有效因子打分的选股策略，在长期能够显著跑赢市场基准，为量化投资提供了坚实的实践基础。
