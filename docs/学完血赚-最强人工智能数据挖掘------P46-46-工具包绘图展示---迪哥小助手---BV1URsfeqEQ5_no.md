# 金融量化分析：P46：工具包绘图展示 📊

在本节课中，我们将学习如何使用工具包对计算出的因子IC值进行可视化展示。通过绘制图表，我们可以更直观地观察因子与收益率相关性的时间序列走势和稳定性，从而评估因子的有效性。

---

![](img/6d075cb24fdc6976029dd03cf3d8a4b4_1.png)

上一节我们介绍了如何计算因子的IC值（信息系数）。算完之后，我们更关注相关性大的因子，因为与收益率相关性越强，因子可能越有价值。

但有一个问题：我们计算出的IC值每天都不一样。仅从数值结果来看，其变化趋势可能不够明显。因此，我们可以将结果绘制成图来观察。

![](img/6d075cb24fdc6976029dd03cf3d8a4b4_3.png)

以下是绘图所需的步骤和代码：

1.  首先，导入一个名为 `proloting` 的工具包，它可以帮助我们进行结果展示。
2.  调用 `protem.plot` 方法，并指定绘制IC时间序列图。具体函数名可以通过查阅API文档确认，例如 `ic_time_series`。
3.  将我们之前计算好的IC值序列（例如存储在变量 `ic_series` 中）作为参数传入该函数。

核心绘图代码如下：
```python
# 假设已导入绘图工具包并计算好IC序列 ic_series
protem.plot.ic_time_series(ic_series)
```

执行代码后，图表会默认显示出来。生成的图表主要包含以下元素：

*   **蓝色线**：代表每日实际的IC值走势，波动范围通常较大。
*   **绿色线**：默认以一个月为窗口计算的IC均值移动平均线，用于观察长期趋势。
*   **统计信息**：图表通常会标注IC序列的均值（Mean）和标准差（Std），以及由 **均值 / 标准差** 计算得出的信息比率（Information Ratio）。

由于蓝色线波动剧烈，我们主要观察绿色均线的走势。通过当前图表可以发现，绿色均线整体比较平稳，没有显示出明显的上升或下降趋势，且平均值较小。这表明该因子与收益率的相关性不够强，也不够稳定。

![](img/6d075cb24fdc6976029dd03cf3d8a4b4_5.png)

![](img/6d075cb24fdc6976029dd03cf3d8a4b4_7.png)

这里提到的**信息比率**（Information Ratio）是一个辅助指标，其计算公式为：
**IR = Mean(IC) / Std(IC)**
它用于描述IC序列的稳定性。比值越大，说明标准差相对均值越小，即IC值越稳定；比值越小，则说明波动性越大。

除了单期IC序列图，工具包还能绘制不同计算周期（如5期、10期）的IC结果图，以及QQ图、直方图等其他分析图表。这些功能在API文档中均有列出，当需要深入分析因子时，可以灵活调用。
