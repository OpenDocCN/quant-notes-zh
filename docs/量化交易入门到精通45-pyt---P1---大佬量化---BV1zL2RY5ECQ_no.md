## 什么是正则表达式？

上一节我们介绍了课程目标，本节中我们来看看正则表达式的定义。正则表达式是一种用于描述文本模式的特殊字符串，可以用来**匹配**、**查找**、**替换**甚至**分割**文本。

## 为什么学习正则表达式？

在量化交易中，我们经常需要处理大量文本信息。例如：
*   从财务报表中提取关键数据。
*   从新闻文章中提取市场情绪。
*   验证用户输入的股票代码格式是否正确。

掌握正则表达式能极大提升处理这类任务的效率。

## 正则表达式基础语法

正则表达式由一系列特殊字符和普通字符组成。以下是几个最核心的元字符：

![](img/4274c7a5328f427a73993abb70da70f3_1.png)

*   **`.` (点号)**：匹配**任意单个**字符（除了换行符）。
    *   公式：`a.c` 可以匹配 `"abc"`、`"a&c"`、`"a c"`。
*   **`*` (星号)**：匹配前面的字符**零次或多次**。
    *   公式：`ab*c` 可以匹配 `"ac"`、`"abc"`、`"abbc"`。
*   **`+` (加号)**：匹配前面的字符**一次或多次**。
    *   公式：`ab+c` 可以匹配 `"abc"`、`"abbc"`，但不能匹配 `"ac"`。
*   **`\d`**：匹配**任意一个数字**，相当于 `[0-9]`。
*   **`{m,n}`**：匹配前面的字符**至少m次，至多n次**。
    *   公式：`a{2,4}` 可以匹配 `"aa"`、`"aaa"`、`"aaaa"`。

## Python `re` 模块实战

了解了基本概念后，我们进入实战环节。Python通过内置的`re`模块提供正则表达式功能。首先需要导入它：

```python
import re
```

### 实战一：匹配股票代码

假设我们需要匹配以“6”开头，且总长度为6位的股票代码（例如`600001`）。

以下是实现此功能的步骤和代码：

1.  **定义正则表达式**：`r‘^6\d{5}$‘`
    *   `^6` 表示字符串必须以“6”开头。
    *   `\d{5}` 表示后面紧跟5个数字。
    *   `$` 表示字符串在此结束。
2.  **准备测试数据**：一个包含多种格式代码的列表。
3.  **使用`re.match()`进行匹配**：从字符串起始位置开始匹配。

```python
import re

# 1. 定义正则表达式：匹配以6开头的6位数字股票代码
pattern = r‘^6\d{5}$‘

# 2. 准备测试数据
test_codes = [‘600001‘, ‘000001‘, ‘600ABC‘, ‘1234567‘, ‘600004‘]

# 3. 循环匹配并输出结果
for code in test_codes:
    if re.match(pattern, code):
        print(f“股票代码 {code} 符合规则“)
    else:
        print(f“股票代码 {code} 不符合规则“)
```

运行上述代码，输出结果将清晰显示哪些代码符合我们的规则。

### 实战二：从文本中提取日期

接下来，我们学习如何从一段文本中提取“YYYY-MM-DD”格式的日期。

以下是提取日期的核心步骤：

1.  **定义正则表达式**：`r‘\d{4}-\d{2}-\d{2}‘`
    *   `\d{4}` 匹配4位数字的年份。
    *   `-` 匹配连接符。
    *   `\d{2}` 匹配2位数字的月份和日期。
2.  **使用`re.findall()`进行查找**：返回所有匹配到的字符串列表。

```python
import re

# 1. 定义包含日期的文本
text = “报告生成于2024-01-09，预测下一交易日2024-01-10的市场走势。“

# 2. 定义正则表达式并提取所有日期
date_pattern = r‘\d{4}-\d{2}-\d{2}‘
dates_found = re.findall(date_pattern, text)

# 3. 输出结果
print(“提取到的日期：“, dates_found)  # 输出: [‘2024-01-09‘, ‘2024-01-10‘]
```

### 实战三：替换字符串中的内容

有时我们需要将文本中的特定模式替换为新内容。例如，将“万元”单位替换为“元”，并换算数值。

以下是替换字符串的方法：

1.  **使用`re.sub()`函数**：其基本格式为 `re.sub(模式, 替换内容, 原始字符串)`。
2.  **在模式中使用分组`()`**：被括号括起来的部分可以被在替换内容中引用，`\1` 代表第一个分组。

```python
import re

text = “公司上季度营收为1000万元，净利润为200万元。“
# 将“数字+万元”替换为“数字*10000+元“
# (\d+) 是一个分组，匹配一个或多个数字
new_text = re.sub(r‘(\d+)万元‘, r‘\1‘ + ‘0000元‘, text)

print(“替换后的文本：“, new_text)
# 输出: 公司上季度营收为10000000元，净利润为2000000元。
```

### 实战四：分割字符串

最后，我们来看如何用正则表达式分割字符串。这比简单的`str.split()`更灵活，可以按复杂的规则分割。

以下是分割字符串的示例：

1.  **使用`re.split()`函数**：按匹配到的模式进行分割。

![](img/4274c7a5328f427a73993abb70da70f3_3.png)

```python
import re

text = “苹果|香蕉,橘子 西瓜；芒果“
# 使用正则匹配 “|” 或 “,” 或空格 或 “；” 作为分隔符
# [|, ；] 是一个字符集，匹配其中任意一个字符
# \s 匹配任何空白字符，包括空格、制表符、换页符等
result = re.split(r‘[|,;\s]+‘, text)

![](img/4274c7a5328f427a73993abb70da70f3_5.png)

print(“分割后的列表：“, result)
# 输出: [‘苹果‘, ‘香蕉‘, ‘橘子‘, ‘西瓜‘, ‘芒果‘]
```
