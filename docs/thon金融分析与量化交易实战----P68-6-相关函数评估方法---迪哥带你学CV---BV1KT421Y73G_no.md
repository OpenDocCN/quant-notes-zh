# Python金融分析与量化交易实战课程：P68：6-相关函数评估方法 📊

在本节课中，我们将学习时间序列分析中两个至关重要的评估工具：自相关函数（ACF）和偏自相关函数（PACF）。它们能帮助我们为ARIMA模型选择合适的P值和Q值。

## 相关系数基础 📈

在深入探讨自相关函数之前，我们先回顾一下相关系数的概念。相关系数用于衡量两个变量之间的线性关系强度和方向。

*   **公式**：相关系数通常用 `ρ` 表示，取值范围为 **[-1, 1]**。
*   **正相关（ρ ≈ 1）**：一个变量增大，另一个变量也增大。
*   **负相关（ρ ≈ -1）**：一个变量增大，另一个变量减小。
*   **无相关（ρ ≈ 0）**：两个变量之间没有线性关系。

## 自相关函数（ACF）详解 🔄

上一节我们介绍了通用的相关系数，本节中我们来看看自相关函数。自相关函数描述的是**同一个时间序列**在不同时间点（即不同“阶数”）上的观测值之间的相关性。

*   **核心概念**：计算当前值 `Y_t` 与过去值 `Y_{t-k}` 之间的相关系数，其中 `k` 称为滞后阶数。
*   **公式**：`ACF(k) = Corr(Y_t, Y_{t-k})`
*   **取值范围**：ACF值同样在 **[-1, 1]** 之间。
*   **图形解读**：ACF图以滞后阶数 `k` 为横轴，以ACF值为纵轴。图中通常包含两条虚线，代表**95%的置信区间**。如果ACF值落在置信区间内，则认为在该阶数上相关性不显著。

简而言之，ACF图展示了时间序列自身在不同时间跨度上的“记忆”或惯性。

## 偏自相关函数（PACF）详解 🎯

说完了自相关函数，我们再来看看偏自相关函数。PACF在ACF的基础上更进一步，它衡量的是在**剔除了中间所有滞后项（`Y_{t-1}, Y_{t-2}, ..., Y_{t-k+1}`）的影响**后，当前值 `Y_t` 与过去值 `Y_{t-k}` 之间的**直接**相关性。

*   **核心区别**：
    *   **ACF**：包含了中间所有滞后项的间接影响。
    *   **PACF**：隔离了中间项，只衡量 `Y_t` 与 `Y_{t-k}` 的纯粹直接关系。
*   **作用**：PACF图对于识别AR模型的阶数 `p` 特别有用。

了解了理论之后，我们来看看如何用Python工具绘制这两个图。我们将使用 `statsmodels` 库。

![](img/103d7a865157a240285bcc9e2115629d_5.png)

以下是安装 `statsmodels` 库的步骤，如果使用 `pip` 安装遇到网络问题，可以采用手动下载安装的方式：

1.  **使用pip安装（推荐）**：
    ```bash
    pip install statsmodels
    ```

2.  **手动安装（备选方案）**：
    *   访问 [Python Extension Packages for Windows](https://www.lfd.uci.edu/~gohlke/pythonlibs/) 网站。
    *   搜索 “statsmodels”，找到与你的Python版本（如cp36表示Python 3.6）和系统位数（win32或amd64）对应的 `.whl` 文件并下载。
    *   在命令行中进入 `.whl` 文件所在目录，执行：
    ```bash
    pip install 下载的文件名.whl
    ```

![](img/103d7a865157a240285bcc9e2115629d_7.png)
