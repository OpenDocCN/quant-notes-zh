## 什么是因子选股策略？🎯

所谓因子选股，其核心在于“因子”。因子指的就是**标准**。当我们需要从一个庞大的股票池（例如300只股票）中筛选出少数几只（例如10只）时，必须依据某种标准。这个标准就是因子。

常见的因子主要分为两类：
1.  **财务指标因子**：例如公司的市值、净利润、增长率、市盈率（P/E）、净资产收益率（ROE）等。
2.  **行情指标因子**：例如股票过去几个月的收益率、波动率等。

无论选择哪种数据作为因子，其本质都是一种量化的筛选标准。

## 策略的核心逻辑 ⚙️

因子选股策略的操作非常简单直接：

1.  **选定一个因子**（例如，公司市值）。
2.  **确定筛选方向**：根据因子的特性，决定是选择因子值**最大**的N只股票，还是**最小**的N只股票。
3.  **定期调仓**：并非每日操作，而是每隔一段较长的周期（例如一个月或一个季度）检查一次。
    *   卖出当前持仓中已不符合“因子值最大/最小”标准的股票。
    *   买入新进入标准的股票。
    *   确保调仓后，持有的始终是当前符合该因子标准的N只股票。

## 小市值策略实例 📉

接下来，我们以实现一个经典的“小市值策略”为例，具体展示因子选股策略的运作。

**策略描述**：该策略认为小市值公司成长空间更大，股价上涨潜力更优。因此，策略始终持有股票池中市值最小的N只股票。

以下是该策略的具体步骤：

1.  **初始化**：在策略开始时，从股票池中选出市值最小的N只股票，建立初始持仓。
2.  **定期调仓**：假设调仓周期为一个月。
    *   在调仓日，重新计算所有股票的市值。
    *   检查当前持仓：哪些股票仍在“市值最小Top N”的名单中，哪些已经掉出名单。
    *   **卖出**掉出名单的股票。
    *   **买入**新进入“市值最小Top N”名单的股票。
3.  **循环执行**：每月重复步骤2，始终保持持仓为当前市值最小的N只股票。

其核心操作可以用以下伪代码描述：
```python
# 伪代码示例
初始化股票池 stock_pool
设定持仓股票数量 N
设定调仓周期（如30天）

while 策略运行:
    当前日期 = 获取当前日期()
    if 当前日期 == 调仓日:
        计算所有股票的市值
        找出市值最小的前N只股票 -> new_top_n
        当前持仓股票 -> current_holdings
        
        # 卖出不在新名单中的股票
        for stock in current_holdings:
            if stock not in new_top_n:
                卖出(stock)
        
        # 买入新名单中尚未持有的股票
        for stock in new_top_n:
            if stock not in current_holdings:
                买入(stock)
    
    等待至下一个调仓日
```
