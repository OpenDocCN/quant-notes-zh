在本节课中，我们将学习一个量化因子从构思到计算的全过程。我们将通过一个具体案例，观察市场现象，剖析核心指标，并逐步演示如何将原始的高频量价数据转化为一个可用的交易因子。整个过程将涵盖数据处理、阈值计算、信号筛选以及因子构建。

## 观察市场与数据准备 📊

上一节我们介绍了课程目标，本节中我们来看看如何准备基础数据。计算因子首先需要获取和处理数据。

我们读取了股票的分钟级别数据。数据中包含每只股票的复权因子、每分钟的开盘价、最高价、最低价、收盘价、成交量以及成交额。所有平台用户都可以免费获取这些分钟数据。

![](img/4acc9554cb5e214a094a8b4dc073085a_1.png)

数据中构建了一列名为 `dd`，其作用是将日期时间 `D` 转换为纯日期格式，以便于后续按天进行数据处理。

## 计算成交量激增时刻 ⚡

在准备好基础数据后，我们需要从中提取有意义的信号。本节的核心是识别成交量“激增”的时刻。

首先，我们截取每天最后240分钟的交易数据。这是因为A股每天有241条分钟数据，多出的一条是盘前竞价数据，我们通常将其聚合处理，因此只取倒数240行。

以下是计算激增时刻的关键步骤：

1.  **计算成交量变动**：计算每一分钟的成交量与上一分钟成交量的差值。
    ```python
    volume_change = current_volume - previous_volume
    ```
2.  **计算未来收益**：为每一分钟计算其未来N分钟（例如F1, F2, … FX）的收益率。
3.  **定义激增阈值**：计算当天所有分钟成交量变动的均值（`mean`）和标准差（`std`），将两者相加作为一个阈值（`threshold`）。
    ```python
    threshold = mean(volume_change) + std(volume_change)
    ```
4.  **筛选激增时刻**：将每一分钟的成交量变动值与阈值比较。如果变动值大于阈值，则该分钟被标记为“激增时刻”。
    ```python
    is_surge = volume_change > threshold
    ```

经过过滤后，我们只保留那些被标记为激增时刻的数据行。

## 构建因子：计算波动率与适度水平 📈

上一节我们筛选出了成交量激增的时刻，本节中我们利用这些时刻的数据来构建最终的因子。

在激增时刻，我们关注价格未来的波动情况。具体做法是：针对每一个激增时刻，计算其未来N分钟收益率的标准差。这个标准差反映了该激增事件后价格的波动剧烈程度。

```python
# 假设 future_returns 是未来N分钟的收益率序列
future_volatility = std(future_returns)
```

我们期望找到一个“适度”的波动水平。波动太大或太小都可能不是理想的信号。因此，我们引入“适度水平”的概念：

1.  **计算截面均值**：在每一天，计算全市场所有股票在激增时刻的未来波动率的平均值。
    ```python
    market_avg_volatility = mean(all_stocks_future_volatility)
    ```
2.  **计算偏离度**：对于单只股票，计算其波动率与当日全市场平均波动率的绝对距离。这个距离代表了该股票波动相对于市场的“适度”程度。
    ```python
    deviation = abs(stock_volatility - market_avg_volatility)
    ```
3.  **聚合为日度因子**：将一只股票一天内所有激增时刻计算出的偏离度求平均值，就得到了该股票当日的因子值。
    ```python
    daily_factor = mean(deviation)
    ```

最终，每只股票每一天都会得到一个因子值，例如名为 `alpha_107` 的因子。这个因子从高频的分钟量价数据表，通过一系列计算，最终聚合成一个日度的因子概念。

## 因子风格分析 🎭

在构建出因子后，我们需要理解它的特性。一个有趣的现象是：这个仅使用量价数据构建的因子，最终呈现出一定的“小市值”风格倾向。

这并非因为因子中直接使用了市值数据，而是由构建逻辑间接导致的：

1.  **激增阈值是自适应的**：大市值和小市值股票都使用各自成交量变动的均值和标准差作为阈值，因此两者都可能出现激增时刻。
2.  **截面均值隐含小市值偏向**：在计算适度水平时，我们使用了全市场股票的简单算术平均。由于A股市场中，小市值股票的数量远多于大市值股票，这个“截面均值”实际上更偏向于反映中小市值股票的整体波动水平。
3.  **因子含义**：因子值（偏离度）越小，说明该股票的波动率越接近这个由中小市值主导的市场平均水平，即被认为越“适度”。因此，具有中小市值波动特性的股票会获得更小的因子值（这是一个负向因子，即值越小可能越好）。

这种构建方式的优势在于：它不再是单纯地选择低市值股票，而是用量价信息刻画出了“类似于小市值股”的活跃特性。即使是一只大市值股票，只要它在成交量激增时表现出稳定（波动率接近市场平均水平）的特性，也会被纳入优选范围。这比单纯使用市值数据更具信息量。

## 总结与回顾 ✅

本节课中我们一起学习了一个完整量化因子的产生过程：

1.  **从数据出发**：我们获取并理解了高频的分钟量价数据。
2.  **定义核心信号**：我们通过“均值加标准差”的方法，从成交量变动中识别出“激增时刻”。
3.  **构建因子逻辑**：在激增时刻，我们计算未来价格波动的标准差，并通过与全市场平均波动水平的比较，定义出一个衡量“波动适度性”的因子。
4.  **分析因子风格**：我们探讨了因子为何会隐含小市值风格，并理解了这种通过量价数据间接捕捉风格的方法，比直接使用风格数据更为细腻和有效。

整个过程展示了如何将市场观察（成交量激增）转化为具体的指标（波动率），再通过严谨的计算步骤，最终构建出一个融合了量价信息的交易因子。这为设计属于自己的量化因子提供了一个清晰的范例。
