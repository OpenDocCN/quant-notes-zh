在本节课中，我们将深入学习Python中的列表数据类型。列表是Python中最基础且最常用的数据结构之一，它允许我们有序地存储和操作多个数据项。我们将从回顾列表的基本定义开始，逐步探索其更高级的操作，包括增删改查、列表嵌套、合并以及一些实用的方法。

## 列表的定义与特性 📝

上一节我们初步认识了列表。本节中，我们来看看列表的正式定义和核心特性。

列表的语法是使用中括号 `[]` 定义，其中的元素用逗号 `,` 分隔。每个元素都有一个对应的索引（从0开始），用于标记和访问数据。

**代码示例：定义一个列表**
```python
my_list = ['苹果', 123, True, ['嵌套', '列表']]
```

以下是列表的几个关键特性：
*   **可存放多个值**：一个列表可以包含任意数量的元素。
*   **有序性**：列表中的元素按照插入顺序排列，并且这个顺序是固定的。
*   **可变性**：列表创建后，可以对其中的元素进行增加、删除、修改等操作，这与不可变的字符串不同。
*   **可存放任意数据类型**：列表中的元素可以是数字、字符串、布尔值，甚至是另一个列表（即列表嵌套）。
*   **容量理论无限**：列表可以容纳的元素数量仅受计算机内存的限制。

## 列表的“增删改查”操作 🔧

了解了列表的基本概念后，我们来看看如何操作列表中的数据，即“增删改查”。

### 增加元素

以下是向列表中添加元素的几种主要方法：

*   **追加（Append）**：使用 `append()` 方法在列表的**末尾**添加一个元素。
    ```python
    names = ['金角大王', '黑姑娘', 'rain']
    names.append('eva')  # 结果：['金角大王', '黑姑娘', 'rain', 'eva']
    ```
*   **插入（Insert）**：使用 `insert(索引位置, 元素)` 方法在列表的**指定索引位置**插入一个元素。
    ```python
    names.insert(2, '新元素')  # 在索引2（第三个位置）插入
    ```
*   **合并（Extend）**：使用 `extend(另一个列表)` 方法将另一个列表中的所有元素**合并**到当前列表的末尾。注意，这会直接修改原列表。
    ```python
    names2 = ['狗蛋', '绿毛']
    names.extend(names2)  # names现在包含了names2的所有元素
    ```

### 删除元素

接下来，我们学习如何从列表中移除元素。

*   **`del` 语句**：这是Python的内置语句，可以删除指定索引的元素，甚至删除整个列表变量。
    ```python
    del names[1]  # 删除索引为1的元素
    ```
*   **`pop()` 方法**：默认删除并返回列表的**最后一个**元素。也可以指定索引 `pop(索引)` 来删除特定位置的元素。
    ```python
    last_item = names.pop()  # 删除最后一个元素并保存到变量
    second_item = names.pop(1)  # 删除索引为1的元素
    ```
*   **`remove()` 方法**：根据**元素的值**进行删除。它会移除列表中**第一个**匹配到的指定值。
    ```python
    names.remove('黑姑娘')  # 删除第一个值为‘黑姑娘’的元素
    ```
*   **`clear()` 方法**：清空整个列表，使其变为一个空列表 `[]`。
    ```python
    names.clear()
    ```

### 修改元素

修改列表元素是最直接的操作，通过索引赋值即可完成。

**代码示例：修改元素**
```python
names[0] = '银角大王'  # 将第一个元素改为‘银角大王’
names[-1] = '最后的元素'  # 使用负索引修改最后一个元素
```

### 查找元素

最后，我们学习如何在列表中查找元素及其位置。

*   **`index()` 方法**：返回指定元素**第一次出现**的索引位置。如果元素不存在，程序会报错。
    ```python
    position = names.index('rain')  # 查找‘rain’的索引
    ```
*   **`in` 关键字**：用于判断一个元素是否存在于列表中，返回 `True` 或 `False`。
    ```python
    is_present = 'eva' in names  # 检查‘eva’是否在列表中
    ```
*   **`count()` 方法**：统计某个元素在列表中出现的次数。
    ```python
    num = names.count('eva')  # 统计‘eva’出现的次数
    ```

一个常见的操作流程是：先使用 `in` 判断元素是否存在，如果存在，再用 `index()` 获取其索引，最后进行修改或其他操作。

## 列表的高级操作 🚀

掌握了基础的“增删改查”后，本节中我们来看看列表的一些更高级的特性，如嵌套和合并。

### 列表的嵌套

列表可以包含其他列表作为其元素，这称为列表嵌套。访问嵌套列表中的元素需要使用多次索引。

**代码示例：列表嵌套与访问**
```python
# 创建一个嵌套列表
names = ['金角大王', '黑姑娘', ['mike', 'rachel'], 'rain']
# 访问主列表的第三个元素（它是一个子列表）
sublist = names[2]  # 结果是 ['mike', 'rachel']
# 访问子列表中的第二个元素
name = names[2][1]  # 结果是 ‘rachel’
# 修改嵌套列表中的元素
names[2][1] = '修改后的rachel'
```

### 列表的合并（回顾）

我们之前提到的 `extend()` 方法就是列表合并的典型操作。它直接将另一个列表的所有元素“融合”进当前列表，而不是创建一个新列表。

**代码示例：列表合并**
```python
list_a = ['alex', 'jack']
names = ['金角大王', '黑姑娘']
names.extend(list_a)  # names现在为 ['金角大王', '黑姑娘', 'alex', 'jack']
```
