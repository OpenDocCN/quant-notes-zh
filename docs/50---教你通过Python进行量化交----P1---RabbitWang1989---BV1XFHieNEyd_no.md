## 第二步：调整股票代码格式

雅虎财经（yfinance）库识别股票代码的格式与我们获取的原始格式不同。原始格式为`交易所代码+公司代码`（如`SSE600519`），而yfinance需要的格式是`公司代码.交易所代码`（如`600519.SS`）。因此，我们需要编写一个函数进行转换。

![](img/107d17a20bf13892b860279380259773_18.png)

![](img/107d17a20bf13892b860279380259773_20.png)

![](img/107d17a20bf13892b860279380259773_22.png)

```python
def yfinance_ticker(ticker):
    return ticker[-6:] + ‘.’ + ticker[:2]

![](img/107d17a20bf13892b860279380259773_24.png)

csi300[‘yfinance_ticker’] = csi300[‘Ticker’].apply(lambda x: yfinance_ticker(x))
```

## 第五步：模拟交易回测

现在，我们根据“均值回归”策略的逻辑进行回测模拟：期初投入100万元，在每个交易日买入当日跌幅最大的十只股票（等权重投资），并在下一个交易日全部卖出，然后用收回的资金重复这一过程。

我们继续使用ChatGPT来生成回测模拟代码。

**Prompt示例**:
“我有两个DataFrame：

2.  `lowest_returns_per_day`: 包含`Date`和`Ticker`列，列出了每个交易日跌幅最大的十只股票。
请编写Python代码模拟以下交易策略：期初资本为1，000，000元。在每个交易日，买入`lowest_returns_per_day`中对应的十只股票，每只股票投入金额相等（即总资本的1/10）。在下一个交易日，以收盘价卖出所有股票。循环这个过程直到最后一天。请输出两个结果：1) 每日交易明细的DataFrame。2) 每日结束后投资组合总市值的DataFrame。”
