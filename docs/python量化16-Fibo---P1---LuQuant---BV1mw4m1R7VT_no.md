# Python量化交易：16：斐波那契回撤外汇交易指标 - P1

在本节课中，我们将学习如何在Python中实现一个完全自动化的斐波那契回撤交易策略。我们将从加载数据开始，逐步讲解如何检测趋势、识别关键分型点、计算斐波那契水平，并最终生成交易信号。课程结尾，我们将通过回测来评估该策略的潜在表现。

## 概述

斐波那契回撤是一种基于斐波那契数列的技术分析方法，主要用于帮助交易者决定入场时机。它本身不预测趋势方向，而是在已知趋势方向的前提下，寻找价格回撤到特定水平（如“黄金区域”）时入场，以优化风险回报比。

## 数据准备与趋势检测

![](img/ccfed420874d435ac532ea61a00a31fa_1.png)

![](img/ccfed420874d435ac532ea61a00a31fa_2.png)

首先，我们需要加载历史价格数据并计算用于判断趋势的技术指标。

我们加载了2003年至2023年的欧元/美元1小时数据。数据中剔除了零成交量的蜡烛图（即市场休市时段）。接着，我们使用`ta`（技术分析）库计算了两个指标：相对强弱指数（RSI）和移动平均线（MA）。移动平均线将用于趋势检测。

为了提升计算速度，我们暂时只使用前2000根蜡烛进行实验。在策略构建完成后，可以扩展到整个数据集。

以下是检测趋势的简单方法：我们检查最近15根蜡烛的收盘价是位于移动平均线之上还是之下。

![](img/ccfed420874d435ac532ea61a00a31fa_4.png)

```python
def detect_trend(data, ma_period=200, lookback=15):
    # 计算移动平均线
    data['MA'] = data['close'].rolling(window=ma_period).mean()
    # 获取最近 lookback 根蜡烛
    recent_candles = data['close'].iloc[-lookback:]
    recent_ma = data['MA'].iloc[-lookback:]

    if (recent_candles > recent_ma).all():
        return 2  # 上升趋势
    elif (recent_candles < recent_ma).all():
        return 1  # 下降趋势
    else:
        return 0  # 趋势不明
```
这个函数返回三个值：`1`代表下降趋势，`2`代表上升趋势，`0`代表趋势不明。参数`lookback`（回顾的蜡烛数量）可以根据你使用的时间框架进行调整。

## 生成斐波那契回撤信号

![](img/ccfed420874d435ac532ea61a00a31fa_8.png)

上一节我们介绍了如何判断市场趋势，本节中我们来看看如何在确定的趋势中生成具体的交易信号。

![](img/ccfed420874d435ac532ea61a00a31fa_10.png)

生成信号的核心函数需要以下参数：数据框`df`、当前蜡烛的索引`current_idx`、用于寻找分型点的回顾蜡烛数量`lookback`、以及一个较小的间隔`gap`。

![](img/ccfed420874d435ac532ea61a00a31fa_12.png)

*   **`lookback`**：用于检测分型点（局部高点或低点）所回顾的蜡烛总数。
*   **`gap`**：当前蜡烛与最近检测到的分型点之间允许的最小间隔蜡烛数。这是为了避免将距离太近的波动误判为有效的分型。

函数逻辑如下：首先确认当前市场趋势。例如，在上升趋势中，我们寻找一个先出现的局部低点（最小分型），后出现的局部高点（最大分型）。这两个分型点之间的价格变动必须超过一个设定的`price_diff_threshold`，以确保有足够的波动空间进行回撤交易。

接着，我们计算关键的斐波那契水平：
*   **黄金区域 (L1)**：`最高价 - 价格差 * 0.62`
*   **止盈水平 (L2)**：`最高价 - 价格差 * 0.78`
*   **止损水平 (L3)**：`0`（或可根据策略设定）

我们并非精确地在62%水平入场，而是设置一个围绕它的“区域”（例如60%-64%），以容纳市场噪音。这是通过`zone_threshold`参数实现的。

![](img/ccfed420874d435ac532ea61a00a31fa_14.png)

最后，我们检查当前蜡烛的收盘价是否落入这个“黄金区域”，并且确认在它之前的若干根蜡烛（数量为`gap`）的最高价都高于黄金区域。这确保了价格是从上方回落到该区域的，符合“反弹点”的定义。

以下是生成多头信号的核心逻辑代码片段：
```python
# 假设已找到有效的高低分型点：highest_price, lowest_price
price_diff = highest_price - lowest_price
L1 = highest_price - price_diff * 0.62  # 黄金区域中心
L2 = highest_price - price_diff * 0.78  # 止盈
L3 = 0  # 止损

![](img/ccfed420874d435ac532ea61a00a31fa_16.png)

zone_threshold = 0.01 * highest_price  # 区域阈值示例
# 检查当前收盘价是否在黄金区域内
if abs(current_close - L1) < zone_threshold:
    # 检查前 gap 根蜡烛的最高价是否都高于 L1
    previous_highs = df['high'].iloc[current_idx - gap : current_idx]
    if (previous_highs > L1).all():
        return {'signal': 2, 'tp': L2, 'sl': L3, 'min_idx': min_idx, 'max_idx': max_idx}
```
如果所有条件满足，函数返回一个包含信号类型、止盈止损位及分型点索引的字典。否则返回`0`（无信号）。对于下降趋势，应用对称的逻辑。

我们使用一个简单的回测函数，基于生成的信号、止盈和止损水平进行模拟交易。为了更贴近实际交易管理，我们为每笔交易设置了两个止盈目标：

*   **止盈2 (TP2)**：设置为TP1的一半，用于提前平掉一半的头寸，锁定部分利润。

## 总结与实用建议

本节课中，我们一起学习了如何用Python构建一个基于斐波那契回撤的自动化交易信号系统。我们从数据准备、趋势判断开始，详细讲解了信号生成的逻辑与代码实现，并通过可视化和回测对策略进行了验证。

核心要点总结如下：
1.  **斐波那契回撤是一个入场工具**：它帮助在趋势中寻找风险回报比更优的入场点，但不负责判断趋势方向。
2.  **系统化与容错性**：通过设置“区域”而非精确水平，以及要求价格从特定方向进入区域，策略能够更好地适应市场噪音。
3.  **风险管理至关重要**：回测显示策略可能产生高回报，但也伴随高回撤。止盈、止损和仓位管理是策略成功的关键。
4.  **人机结合的应用场景**：一个实用的方式是将此自动化系统作为辅助工具。交易者可以先手动评估大趋势和市场结构，然后让系统自动监控回撤情况，并在条件满足时发出通知，由交易者做出最终决策。
