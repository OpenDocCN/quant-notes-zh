## 什么是滑动窗口？🪟
上一节我们介绍了时间序列数据的基本处理。本节中，我们来看看如何通过滑动窗口来平滑数据。

滑动窗口是一种数据处理技术，它通过一个固定大小的“窗口”在时间序列数据上连续滑动，并对窗口内的数据进行统计计算（如求平均值）。这种方法可以避免因单个数据点的偶然波动或误差而导致的判断偏差，使得分析结果更加平稳和可靠。

例如，我们有一份从2016年到2017年共365天的数据。如果我们想了解2月5日的情况，单独看这一天的数据可能不够全面。更好的方法是取2月1日到2月10日这十天的数据，计算其平均值，用这个平均值来代表2月5日前后的大致情况。这个“十天”就是一个窗口。

**滑动窗口的核心操作流程如下：**
1.  定义一个窗口长度（例如10天）。
2.  将窗口放置在时间序列的起始位置（例如1月1日至1月10日），计算窗口内数据的统计量（如均值）。
3.  将窗口向右滑动一个单位（例如移动到1月2日至1月11日），再次计算统计量。
4.  重复步骤3，直到窗口滑动到时间序列的末端。

这个过程类似于卷积神经网络中的卷积核滑动操作。使用滑动窗口可以使数据的浮动范围变小，过滤掉一些离群点或噪声，从而得到更具代表性的趋势信息。

## 使用Pandas实现滑动窗口 🛠️
理解了滑动窗口的概念后，本节我们来看看如何在Python中利用Pandas库来实现它。

![](img/e8bab39a33b87a8127edd3d7c86acca8_3.png)

![](img/e8bab39a33b87a8127edd3d7c86acca8_5.png)

Pandas提供了非常简便的函数来处理滑动窗口。我们首先需要创建一个时间序列数据。

以下是创建示例时间序列数据的代码：
```python
import pandas as pd
import numpy as np

![](img/e8bab39a33b87a8127edd3d7c86acca8_7.png)

# 创建一个包含600个随机数的时间序列，频率为天(D)
ts = pd.Series(np.random.randn(600), index=pd.date_range('2023-01-01', periods=600, freq='D'))
```

![](img/e8bab39a33b87a8127edd3d7c86acca8_9.png)

创建好数据后，就可以使用`.rolling()`方法来创建滑动窗口对象。

![](img/e8bab39a33b87a8127edd3d7c86acca8_11.png)

以下是创建并应用滑动窗口的代码：
```python
# 创建一个窗口大小为10的滑动窗口对象
r = ts.rolling(window=10)

![](img/e8bab39a33b87a8127edd3d7c86acca8_13.png)

# 计算每个窗口的均值
rolling_mean = r.mean()
```
在这段代码中，`ts.rolling(window=10)`定义了一个长度为10的窗口。`.mean()`方法则计算了每个窗口内数据的平均值。对于序列的前9个数据点，因为无法构成一个完整的长度为10的窗口，所以其计算结果会是`NaN`（非数字）。从第10个数据点开始，才会输出有效的窗口均值。

![](img/e8bab39a33b87a8127edd3d7c86acca8_15.png)

除了均值，你还可以计算其他统计指标。

以下是滑动窗口支持的部分统计方法：
*   `max()`: 计算窗口最大值
*   `min()`: 计算窗口最小值
*   `std()`: 计算窗口标准差
*   `sum()`: 计算窗口总和
*   `median()`: 计算窗口中位数

## 可视化对比效果 📈
理论结合实践，为了更直观地理解滑动窗口的效果，本节我们将原始数据与经过窗口平滑后的数据绘制在同一张图上进行对比。

![](img/e8bab39a33b87a8127edd3d7c86acca8_19.png)

我们可以使用Matplotlib库来绘制图表，直观展示滑动窗口的平滑效果。

![](img/e8bab39a33b87a8127edd3d7c86acca8_21.png)

以下是绘制对比图的代码：
```python
import matplotlib.pyplot as plt

![](img/e8bab39a33b87a8127edd3d7c86acca8_23.png)

![](img/e8bab39a33b87a8127edd3d7c86acca8_25.png)

plt.figure(figsize=(12, 6))
# 绘制原始数据，红色虚线
ts.plot(color='red', linestyle='--', label='Original Data')
# 绘制滑动窗口均值数据，蓝色实线
rolling_mean.plot(color='blue', label='Rolling Mean (window=10)')

![](img/e8bab39a33b87a8127edd3d7c86acca8_27.png)

plt.legend()
plt.title('Original Data vs. Rolling Mean')
plt.show()
```
运行上述代码后，你会得到一张对比图。图中红色虚线代表原始数据，波动较大；蓝色实线代表经过窗口大小为10的均值平滑后的数据，其曲线更加平缓，趋势也更清晰。这证明了滑动窗口能有效平滑噪声，凸显时间序列的主要变化趋势。
