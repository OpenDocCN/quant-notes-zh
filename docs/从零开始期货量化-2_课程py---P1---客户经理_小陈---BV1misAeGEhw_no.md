## Python是什么？

Python是一种我们与计算机沟通的语言。通过Python，计算机可以理解我们的指令，并完成我们交给它的具体任务。有了Python，我们可以轻松地将脑中的想法转化为量化交易系统，进而转化为利润。这完美诠释了知识就是生产力。

## 实战案例：感受Python的魅力

![](img/3f054891726f1ddc38b97eeb546e08f0_9.png)

为了直观地感受Python，我们来看一个基于Python编写的“市场行情预测器”案例。这个程序结合了当下热点，旨在演示Python的基本功能。

以下是该预测程序的核心演示逻辑：

1.  程序启动，等待用户输入问题。
2.  如果用户没有输入，程序会提示“老板还没有提问呢”。
3.  如果用户输入了问题（例如“明天行情如何”），程序会进行“运算”并给出预测结果。

运行程序后，它给出的结论是：**明天大盘不是涨就是跌，平的概率极小**。这个预测“考虑周全”，逻辑“严谨”，以一种幽默的方式揭示了程序背后的简单逻辑。

通过这个演示可以发现，Python编程本身并不枯燥，可以实现非常多有意思的事情。

## 程序代码逐行解析

我们的程序虽然小，但“五脏俱全”。下面将逐行讲解程序是如何实现的。

### 1. 导入功能模块

程序的第一行代码如下：
```python
from 佟掌柜 import money_gpt
```
这行代码的作用是从名为“佟掌柜”的脚本文件中，导入一个叫 `money_gpt` 的函数。

**什么是函数？** 我们可以把做量化比作做菜。数据是原材料，而函数就是锅、碗、瓢、盆、刀、叉这些工具。每个函数都封装了一项特定功能。定义函数的目的就是为了提高效率，并且让整体代码看起来更简洁、清晰。

### 2. Python的运行顺序

Python代码的执行顺序是**从上到下，逐行运行**，就像我们读书的顺序一样。

### 3. 获取用户输入

接下来是这行代码：
```python
question = input(“请输入您的问题: “)
```
`question` 是一个我们自己定义的**变量**，用于存储信息。`input` 是一个函数，它的作用就是在程序运行时，让用户输入一些内容，并将输入的内容赋值给 `question` 这个变量。Python的语法接近自然语言，便于我们通过“说人话”来控制计算机。

### 4. 输出与逻辑判断

以下是程序的核心判断部分：
```python
print(“*” * 100)

if question:
    money_gpt(question)
else:
    print(“老板，你还没有提问呢！“)
```
*   `print` 是Python的核心函数之一，用于将结果输出或“打印”到屏幕上。`“*” * 100` 是为了打印一行星号，让输出看起来更美观，方便阅读。
*   `if` 和 `else` 是**逻辑判断**语句，英文意思是“如果”和“否则”。这行代码的意思是：**如果** `question` 变量中有内容（即用户提了问题），那么就执行 `money_gpt` 函数；**否则**（即用户没提问题），就输出“老板，你还没有提问呢！”。

### 5. 深入 `money_gpt` 函数内部

现在，我们来看看被调用的 `money_gpt` 函数内部是如何工作的。函数定义如下：
```python
import time

def money_gpt(question):
    print(“正在调用量子纠缠计算模块...”)
    time.sleep(0.3)
    # ... (后续循环打印等操作)
    print(“预测结果：明日大盘不是涨就是跌，平的概率极小。”)
```
*   `import time`：这行代码导入了Python的**时间模块**，使我们能使用与时间相关的功能。
*   `def money_gpt(question):`：`def` 是**定义**函数的关键字。这行代码定义了一个名为 `money_gpt` 的函数，它接受一个 `question` 参数。
*   `time.sleep(0.3)`：这是时间模块的一个功能，让程序**暂停**0.3秒再执行下一行。这样做是为了让提示信息有逐字显示的效果，提升用户体验。去掉它，结果会瞬间弹出，体验感不佳。

### 6. 循环结构的使用

函数内使用了循环结构：
```python
for i in range(5):
    print(i+1)

for element in [“金”, “木”, “水”, “火”, “土”]:
    print(element)
```
*   `for ... in ...` 是一种**循环**语句。当你需要重复执行类似操作多次（三次以上）时，使用循环可以极大简化代码。
*   第一个循环会依次输出数字1到5。
*   第二个循环会依次输出“金”、“木”、“水”、“火”、“土”这几个字。

## 课程总结 🎯

本节课中，我们一起学习了以下核心内容：

1.  **Python的本质**：一种与计算机沟通、将想法转化为工具和利润的语言。
2.  **程序的基本结构**：我们通过一个市场预测程序，实战演练了Python代码。
3.  **关键编程概念**：
    *   **导入模块**：使用 `from ... import ...` 来引入现有功能。
    *   **变量**：如 `question`，用于存储信息。
    *   **输入/输出**：`input` 获取输入，`print` 进行输出。
    *   **逻辑判断**：使用 `if...else` 结构让程序根据不同条件执行不同操作。
    *   **定义函数**：使用 `def` 关键字封装特定功能。
    *   **使用模块**：如 `import time` 来利用时间控制功能。
    *   **循环结构**：使用 `for ... in ...` 来高效处理重复任务。
