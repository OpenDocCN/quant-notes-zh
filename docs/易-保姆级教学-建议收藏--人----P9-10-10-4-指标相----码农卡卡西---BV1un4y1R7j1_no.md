## 第四步：散点图与分布分析

除了观察时间序列上的走势，我们还可以通过散点图来直接观察两个变量数值点之间的分布关系，这有助于我们判断其相关性的强度和模式。

以下是绘制散点图矩阵的代码，它同时展示了每个变量的分布（对角线）以及两两变量间的散点图：

![](img/be840d45af0ef03d96b4189132b82328_65.png)

![](img/be840d45af0ef03d96b4189132b82328_67.png)

```python
from pandas.plotting import scatter_matrix

# 计算每日收益率（或变化率），以便分析变化量之间的关系
returns = np.log(data / data.shift(1))
returns = returns.dropna()

# 绘制散点图矩阵，对角线显示核密度估计图
scatter_matrix(returns, alpha=0.5, figsize=(10, 10), diagonal=‘kde’)
plt.show()
```

![](img/be840d45af0ef03d96b4189132b82328_69.png)

![](img/be840d45af0ef03d96b4189132b82328_71.png)

在这个矩阵图中：
*   对角线上的图是每个变量自身的**核密度估计图**，它展示了该变量取值的分布情况，比直方图更平滑。
*   非对角线上的图是两个变量之间的**散点图**。从`SPX`和`VIX`的散点图中，我们可以更明显地看到数据点从左上到右下分布的负相关趋势。

## 第五步：进行回归分析

![](img/be840d45af0ef03d96b4189132b82328_73.png)

![](img/be840d45af0ef03d96b4189132b82328_75.png)

散点图展示了关系，而回归分析可以量化这种关系。我们可以在散点图上添加一条回归线，使得关系更加一目了然。

![](img/be840d45af0ef03d96b4189132b82328_77.png)

![](img/be840d45af0ef03d96b4189132b82328_79.png)

以下是使用`seaborn`库绘制带回归线的散点图的代码：

![](img/be840d45af0ef03d96b4189132b82328_81.png)

```python
import seaborn as sns

![](img/be840d45af0ef03d96b4189132b82328_83.png)

![](img/be840d45af0ef03d96b4189132b82328_85.png)

sns.set(style=“whitegrid”)
sns.regplot(x=returns[‘SPX’], y=returns[‘VIX’], scatter_kws={‘alpha’:0.5})
plt.xlabel(‘SPX Daily Returns’)
plt.ylabel(‘VIX Daily Returns’)
plt.title(‘Regression Plot: SPX vs VIX Returns’)
plt.show()
```

![](img/be840d45af0ef03d96b4189132b82328_87.png)

![](img/be840d45af0ef03d96b4189132b82328_89.png)

这条回归线的斜率和方向，清晰地印证了我们之前观察到的负相关关系。斜率本身（即回归系数）就量化了这种关系的强度。

![](img/be840d45af0ef03d96b4189132b82328_91.png)

## 总结

![](img/be840d45af0ef03d96b4189132b82328_93.png)

![](img/be840d45af0ef03d96b4189132b82328_95.png)

本节课中我们一起学习了如何分析两个金融指标间的相关性。我们首先提取了目标数据，然后通过**双Y轴折线图**观察了它们在时间序列上的互动。接着，我们使用**散点图矩阵**更直观地展示了变量间的分布与关系。最后，我们通过**带回归线的散点图**量化了这种关系，明确地展示了`SPX`与`VIX`指数收益率之间的负相关性。这些可视化工具是探索性数据分析（EDA）的核心，能帮助我们快速形成对数据关系的直观理解。
