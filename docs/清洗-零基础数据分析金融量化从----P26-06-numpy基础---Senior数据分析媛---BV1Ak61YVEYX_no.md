## 数组的连接

上一节我们介绍了数组的索引和切片，本节中我们来看看如何将多个数组组合在一起，即数组的连接操作。NumPy的连接操作主要用于将多个数组在指定轴向上合并，但它缺乏业务字段的概念，因此在实际业务表处理中应用有限。

以下是数组连接的基本语法和示例：

*   **`np.concatenate` 函数**：这是最通用的连接函数。
    *   语法：`np.concatenate((array1, array2, ...), axis=0)`
    *   参数 `axis=0` 表示纵向连接，`axis=1` 表示横向连接。
    *   连接时，在指定的轴向上，所有数组的维度大小必须一致。

![](img/3b10130705cc82fa26476d82a5fc71a3_1.png)

```python
import numpy as np
# 创建两个示例数组
n1 = np.random.randint(0, 100, size=(5, 5))
n2 = np.random.randint(0, 100, size=(5, 5))

# 纵向连接
result_v = np.concatenate((n1, n2), axis=0)
# 横向连接
result_h = np.concatenate((n1, n2), axis=1)
```

*   **便捷函数**：为了更直观，NumPy提供了 `hstack` (横向) 和 `vstack` (纵向) 函数。
    *   `np.hstack((n1, n2))` 等同于 `np.concatenate((n1, n2), axis=1)`
    *   `np.vstack((n1, n2))` 等同于 `np.concatenate((n1, n2), axis=0)`

## 数组的切分

![](img/3b10130705cc82fa26476d82a5fc71a3_3.png)

![](img/3b10130705cc82fa26476d82a5fc71a3_5.png)

了解了如何合并数组后，我们再来看看其逆操作——数组的切分。切分与切片不同，切片是获取数据的某一部分，而切分是将一个数组均等地或按指定位置分割成多个子数组。

以下是数组切分的方法：

*   **`np.split` 函数**：基本切分函数。
    *   语法：`np.split(array, indices_or_sections, axis=0)`
    *   如果 `indices_or_sections` 是一个整数 `N`，则数组将在指定轴向上被均分成 `N` 份。
    *   如果 `indices_or_sections` 是一个列表 `[a, b, ...]`，则表示在索引 `a`, `b`, ... 处进行切割。
    *   参数 `axis` 控制切分的方向。

```python
n1 = np.random.randint(0, 100, size=(6, 6))
# 将数组纵向均分成3份
parts = np.split(n1, 3, axis=0)
# 在列方向索引2和4处进行切割
parts2 = np.split(n1, [2, 4], axis=1)
```

*   **便捷函数**：同样提供了 `hsplit` (横向切分) 和 `vsplit` (纵向切分)。
    *   `np.hsplit(n1, 3)` 将数组水平方向分成3等份。
    *   `np.vsplit(n1, 2)` 将数组垂直方向分成2等份。

## 副本与视图

在进行数组操作时，理解数据的复制方式至关重要，这涉及到“副本”与“视图”的概念，以及编程中常说的深拷贝与浅拷贝。

*   **浅拷贝**：仅复制对象的引用（内存地址）。多个变量指向同一块内存数据，修改其中一个，其他变量访问的数据也会改变。
    *   在NumPy中，直接赋值 (`n2 = n1`) 就是浅拷贝。
*   **深拷贝**：复制对象本身及其包含的所有数据，生成一个全新的、独立的内存对象。修改副本不会影响原数据。
    *   在NumPy中，使用 `copy()` 方法实现数组的深拷贝。

```python
n1 = np.array([1, 2, 3])
n2 = n1 # 浅拷贝
n3 = n1.copy() # 深拷贝

n1[0] = 100
print(n2) # 输出：[100 2 3]，n2随之改变
print(n3) # 输出：[1 2 3]，n3保持不变
```

需要注意的是，在更复杂的嵌套对象结构中，`copy()` 可能只是“一层深拷贝”，若要完全递归复制所有层级对象，需要使用Python标准库的 `copy.deepcopy()`。但在NumPy数组的语境下，`copy()` 通常已足够。

## 聚合计算

聚合计算是指将一组数据汇总为单个标量值的过程，例如求和、求平均值、找最大值等。这是数据分析中提取关键指标的基础。

以下是常用的聚合函数：

*   **整体聚合**：默认对整个数组的所有元素进行计算。
    *   `array.sum()`: 求和
    *   `array.mean()`: 求平均值
    *   `array.std()`: 求标准差
    *   `array.max()` / `array.min()`: 求最大值/最小值
*   **轴向聚合**：通过 `axis` 参数可以指定沿某个轴进行聚合。
    *   `axis=0`: 对每**列**进行计算（纵向压缩）。
    *   `axis=1`: 对每**行**进行计算（横向压缩）。

```python
n1 = np.ones((5, 4))
print(n1.sum()) # 整体求和，输出 20.0
print(n1.sum(axis=0)) # 对每列求和，输出 [5. 5. 5. 5.]
print(n1.sum(axis=1)) # 对每行求和，输出 [4. 4. 4. 4. 4.]
```

*   **特殊聚合**：
    *   `array.any()`: 数组中是否存在至少一个 `True`（或非零值）。
    *   `array.all()`: 数组中的所有元素是否都为 `True`（或非零值）。
    *   处理空值：NumPy有特殊的空值 `np.nan`，它与任何值运算结果都是 `np.nan`。可以使用 `np.nansum()`, `np.nanmean()` 等函数忽略空值进行聚合。

## 数组运算与广播机制

NumPy最强大的特性之一是其向量化运算能力和广播机制，这使得我们可以避免编写低效的循环，直接对数组进行批量操作。

*   **元素级运算**：两个形状相同的数组进行 `+`, `-`, `*`, `/` 等运算时，是对应位置的元素分别进行运算。

```python
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
print(a + b) # 输出：[5 7 9]
print(a * b) # 输出：[4 10 18]
```

*   **广播机制**：当两个数组形状不同时，NumPy会尝试通过复制数据来扩展较小数组的维度，使其形状与较大数组兼容，然后再执行元素级运算。广播遵循特定规则：
    1.  从尾部维度开始比较。
    2.  维度大小相等，或其中一个为1，则兼容。
    3.  缺失的维度被视为1。
    4.  兼容后，在大小为1的维度上复制数据以匹配另一个数组。

![](img/3b10130705cc82fa26476d82a5fc71a3_15.png)

![](img/3b10130705cc82fa26476d82a5fc71a3_17.png)

![](img/3b10130705cc82fa26476d82a5fc71a3_19.png)

```python
# 数组与标量运算（标量被广播到数组每个元素）
a = np.array([[1,2,3], [4,5,6]])
print(a + 10) # 输出：[[11 12 13] [14 15 16]]

# 列向量与行向量相加
col = np.array([[1], [2], [3]]) # 形状 (3, 1)
row = np.array([10, 20, 30])    # 形状 (3,)
print(col + row) # 输出：[[11 21 31] [12 22 32] [13 23 33]]
# 解释：row被广播为 [[10,20,30],[10,20,30],[10,20,30]]，然后与col相加
```

*   **矩阵乘法**：注意，`*` 是元素乘法，而矩阵乘法应使用 `@` 运算符或 `np.dot()` 函数。

```python
A = np.array([[1,2],[3,4]])
B = np.array([[5,6],[7,8]])
print(A * B) # 元素乘法，输出：[[ 5 12] [21 32]]
print(A @ B) # 矩阵乘法，输出：[[19 22] [43 50]]
# 计算过程：[[1*5+2*7, 1*6+2*8], [3*5+4*7, 3*6+4*8]]
```
