## 股票的定义与作用

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

为了更形象地解释，我们可以设想一个场景：一位创业者需要资金，而投资者看好他的项目。投资者将资金投入公司，公司则向投资者发行股票作为凭证。例如，一个公司初始市值为5亿，由五位出资人（包括创始人）各出资1亿建立，那么每位出资人将获得公司20%的股票。

股票的核心作用主要有两点：
1.  **出资证明与股东身份**：持有股票意味着你是公司的所有者之一，拥有相应的投票权等股东权利。
2.  **获取收益**：作为股东，可以通过两种主要方式获利：
    *   **公司分红**：当公司盈利时，股东可以按持股比例获得利润分红。
    *   **交易获利**：股东可以在市场上将股票转让给他人，通过买卖价差获利。对于广大普通股民而言，交易获利是更常见的参与方式。

## 公司上市与IPO

公司要向社会公众募集资金，需要经过“上市”流程。

**上市**是指企业通过证券交易所，首次公开向公众增发股票以募集资金的过程。公司不能随意向公众集资，必须达到一定标准（如持续盈利能力、规范的财务制度等），并经过证监会审核批准后，才能在证券交易所挂牌交易。这样，所有符合条件的投资者都可以买卖该公司的股票。

**IPO**即首次公开募股，特指公司第一次上市并公开发行股票的行为。

## 股票的分类

了解股票的分类有助于我们后续进行更有针对性的分析。主要有两种分类方式。

### 按公司业绩分类

![](img/5294b8e26ef604668488083939df1d3f_1.png)

根据公司的经营表现，股票可分为以下几类：

*   **蓝筹股**：指资本雄厚、信誉优良的大型公司股票。例如中国的“中石油”、“中石化”。名称源于赌场中价值最高的蓝色筹码。
*   **绩优股**：指业绩优良、盈利稳定的公司股票。例如“贵州茅台”，其特点是持续为股东创造价值。
*   **ST股**：中文为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记，以警示投资者该公司存在较高风险。

### 按上市地区分类

根据股票上市交易的地点和结算货币，可分为：

*   **A股**：在中国大陆（上海、深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。
*   **H股**：在中国香港上市的公司股票。
*   **N股**：在美国纽约上市的公司股票。
*   **S股**：在新加坡上市的公司股票。

不同市场的交易规则存在差异，这对于量化策略的设计至关重要。以下是两个关键规则示例：

**A股关键规则**：
*   **涨跌幅限制**：为防止股价剧烈波动，保护投资者，A股设有每日涨跌幅限制，通常为**10%**。即：
    `今日最高价 <= 昨日收盘价 * 1.1`
    `今日最低价 >= 昨日收盘价 * 0.9`
*   **T+1交割制度**：当日买入的股票，需到下一个交易日才能卖出。这旨在减少市场投机行为。
    ```python
    # 模拟T+1规则：今日买入无法今日卖出
    if action == ‘buy’ and date == today:
        can_sell = False # 今日不可卖
    else:
        can_sell = True # 非买入日或次日可卖
    ```

**其他市场规则**：
*   **港股（H股）、美股**等市场通常实行**T+0**交易（当日可买卖）且**无涨跌幅限制**，波动可能更为剧烈。
*   **B股**规则稍复杂，涉及**T+1**交割和**T+3**交收（资金到账时间）。
