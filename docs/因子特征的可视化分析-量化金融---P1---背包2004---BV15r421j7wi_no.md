# 量化金融与机器学习2024：P1：因子特征的可视化分析 📊

![](img/20081930d520fb67e141d34ef0f15a71_1.png)

在本节课中，我们将学习如何对构建的金融因子特征进行分析和可视化。这对于理解数据特性、筛选有效特征以及撰写研究报告都至关重要。我们将使用一个专门为特征分析设计的Python库——`sweetviz`，来高效地生成多种分析图表。

![](img/20081930d520fb67e141d34ef0f15a71_3.png)

![](img/20081930d520fb67e141d34ef0f15a71_5.png)

## 概述
在构建了多个金融因子特征后，我们需要对其进行分析，以了解每个特征的分布形态、与目标变量的相关性以及特征之间的相互关系。本节将介绍几种核心的可视化分析方法，帮助我们从数据中获得直观的认识。

---

## 1. 特征分布分析 📈

上一节我们介绍了特征构建，本节中我们来看看如何了解单个特征的数值分布形态。这是数据分析的基础步骤。

### 1.1 使用描述性统计
最快速的方法是使用Pandas的`describe()`函数，它可以给出特征的四分位数、最大值、最小值、平均值和标准差等统计信息。

```python
df['pe_ratio'].describe()
```

![](img/20081930d520fb67e141d34ef0f15a71_7.png)

![](img/20081930d520fb67e141d34ef0f15a71_9.png)

![](img/20081930d520fb67e141d34ef0f15a71_11.png)

### 1.2 绘制分布图
为了更直观地查看分布，我们可以使用`sweetviz`库中的函数。它不仅绘图，还附带统计功能。以下是绘制静态市盈率（`pe_ratio`）分布图的代码：

![](img/20081930d520fb67e141d34ef0f15a71_13.png)

![](img/20081930d520fb67e141d34ef0f15a71_15.png)

![](img/20081930d520fb67e141d34ef0f15a71_17.png)

![](img/20081930d520fb67e141d34ef0f15a71_19.png)

```python
# 假设 df 是包含特征的数据框，'pe_ratio' 是其中一列
sv.analyze(df[['pe_ratio']])
```

![](img/20081930d520fb67e141d34ef0f15a71_21.png)

![](img/20081930d520fb67e141d34ef0f15a71_23.png)

通过分布图，我们可以观察到市盈率呈现“长尾分布”，即大多数股票的市盈率较低（小于50），但有少量股票的市盈率极高。这符合股票市场的实际情况。

### 2.1 绘制散点图
散点图可以直观展示两个变量之间的关系。我们将股息率（`dividend_yield`）放在X轴，将10日收益率（`return_10d`）放在Y轴。

以下是绘制散点图的代码示例：
```python
sv.compare(df[['dividend_yield', 'return_10d']])
```

### 2.2 使用核密度估计图
除了散点图，还可以使用核密度估计（KDE）图来展示数据的聚集情况，但计算量较大，对于大数据集可能需要采样。

![](img/20081930d520fb67e141d34ef0f15a71_71.png)

![](img/20081930d520fb67e141d34ef0f15a71_73.png)

![](img/20081930d520fb67e141d34ef0f15a71_74.png)

![](img/20081930d520fb67e141d34ef0f15a71_76.png)

![](img/20081930d520fb67e141d34ef0f15a71_78.png)

```python
# 对数据进行采样以加速
df_sample = df.sample(frac=0.1)
sv.compare(df_sample[['dividend_yield', 'return_10d']], feat_cfg=sv.FeatureConfig(force_num=['dividend_yield']))
```

我们使用一个包含分类标签（如`label_up`，上涨为1，下跌为0）的特征列作为分组依据。

执行后，对于每个数值特征（如市盈率），工具会分别绘制上涨组（黄色）和下跌组（蓝色）的分布曲线。通过对比可以发现，虽然两组分布形态相似，但在某些区间（如中等市盈率区间）上涨股票的比例略高，这暗示了该特征可能具有一定的区分能力。

![](img/20081930d520fb67e141d34ef0f15a71_189.png)

### 4.2 分类散点图
我们还可以在散点图中用颜色区分涨跌类别，以观察在二维特征空间中，不同类别的样本是否可以被区分。

```python
sv.compare(df[['pe_ratio', 'pb_ratio', 'label_up']], ‘label_up’)
```

![](img/20081930d520fb67e141d34ef0f15a71_191.png)

![](img/20081930d520fb67e141d34ef0f15a71_193.png)

如果某个特征或特征组合能使得两类样本的点在图中形成相对分离的簇，则说明该特征对于预测涨跌可能是有效的。

## 总结
本节课中我们一起学习了金融因子特征的可视化分析方法。我们掌握了：
1.  使用**分布图**了解单个特征的数值形态。
2.  通过**散点图**和**相关性热力图**分析特征与目标、特征与特征之间的关系。
3.  利用**分类对比图**观察特征在不同样本组（如涨/跌）中的分布差异。

![](img/20081930d520fb67e141d34ef0f15a71_211.png)

![](img/20081930d520fb67e141d34ef0f15a71_213.png)

![](img/20081930d520fb67e141d34ef0f15a71_215.png)

这些分析能帮助我们深入理解数据，为后续的特征工程和模型构建提供重要依据。建议同学们尝试用不同的因子替换示例代码中的特征，亲自生成并解读这些图表，以获得更深刻的体会。
