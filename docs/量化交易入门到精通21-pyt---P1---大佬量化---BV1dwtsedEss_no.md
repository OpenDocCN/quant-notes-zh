## 基础输出

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_5.png)

上一节我们介绍了变量的赋值，本节中我们来看看如何使用`print`函数输出这些变量的值。`print`函数可以将括号内的内容显示在屏幕上。

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_7.png)

例如，我们定义了两个变量：
```python
名称 = “贵州茅台”
代码 = “600519”
```
请注意，等号和引号都必须是英文符号，使用中文符号会导致程序报错。

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_9.png)

我们可以使用`print`函数直接输出这些变量：
```python
print(名称, 代码)
```
执行这行代码，屏幕上会显示：`贵州茅台 600519`。

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_11.png)

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_13.png)

## 格式化输出

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_15.png)

除了直接输出，Python还提供了更灵活的格式化输出方式，可以让输出的语句更清晰、易读。

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_17.png)

以下是两种常见的格式化输出方法：

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_19.png)

**方法一：使用逗号分隔**
在`print`函数中，用逗号分隔多个变量或字符串，它们会以空格连接起来输出。
```python
print(“股票名称：”, 名称, “股票代码：”, 代码)
```
输出结果为：`股票名称： 贵州茅台 股票代码： 600519`

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_21.png)

**方法二：使用f-string（格式化字符串字面值）**
这是一种更现代、更简洁的格式化方法。在字符串前加上字母`f`或`F`，然后在字符串内部用大括号`{}`包裹变量名，这个`{}`被称为**占位符**。
```python
print(f”股票名称是{名称}，股票代码是{代码}”)
```
输出结果为：`股票名称是贵州茅台，股票代码是600519`

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_23.png)

为了让输出结果更醒目，我们还可以在字符串中添加一些分隔符：
```python
print(f”—股票名称：{名称}，股票代码：{代码}—”)
```
输出结果为：`—股票名称：贵州茅台，股票代码：600519—`

![](img/00c2fc4cfb8cfd068d73fbc949a97fbc_25.png)
