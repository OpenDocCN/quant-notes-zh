# Python数据分析实战：P33：06 产品维度指标 📊

在本节课中，我们将学习数据分析中至关重要的产品维度指标。这些指标帮助我们衡量产品的市场表现、用户价值以及商业效益。

上一节我们介绍了用户和行为数据指标，本节中我们来看看如何从产品本身出发，评估其商业价值。

## GMV：成交总额 💰

产品数据最核心的指标之一是**GMV**，即成交总额。它反映了在指定时间段内，所有订单的总金额。

![](img/42616c8cc15143de45ca38e96dd1f2d3_1.png)

**GMV** 的计算公式为：
```
GMV = 销售额 + 取消订单金额 + 拒收订单金额 + 退款订单金额
```

![](img/42616c8cc15143de45ca38e96dd1f2d3_2.png)

GMV不仅仅是实际销售额，它包含了所有产生交易的流水。例如，用户下单后申请退款，这笔金额依然会计入GMV。这个指标反映了用户与产品之间发生的所有交易关系，即使某些交易最终并未为公司带来实际收入。

![](img/42616c8cc15143de45ca38e96dd1f2d3_4.png)

## ARPU 与 ARPPU：用户价值指标 👥

除了GMV，衡量用户价值的指标同样重要。

以下是两个关键的用户价值指标：
*   **ARPU**：平均每用户收入。计算公式为 `总收入 / 总用户数`。它衡量的是在全体用户中，平均每个用户带来的收入。
*   **ARPPU**：平均每付费用户收入。计算公式为 `总收入 / 付费用户数`。它专门衡量那些实际进行了消费的用户所带来的平均价值。

## SPU 与 SKU：商品管理标识 🏷️

![](img/42616c8cc15143de45ca38e96dd1f2d3_6.png)

在零售和电商领域，清晰地区分商品至关重要，这时就需要用到SPU和SKU。

以下是它们的定义与区别：
*   **SPU**：标准化产品单元。它是一个商品集合的最小单位，代表一类产品。例如，“iPhone 13”就是一个SPU。
*   **SKU**：库存量单位。它是识别产品的唯一标识符，基于不同的属性（如颜色、尺寸、配置）进行区分。例如，“iPhone 13 128GB 蓝色”就是一个独立的SKU。

简单来说，一个SPU（如一件特定款式的衣服）下会包含多个SKU（如该款式衣服的S码红色、M码白色等）。

## ROI：投资回报率 📈

对于公司运营而言，**ROI**是一个极其重要的综合性指标，它衡量的是投资与所获收益之间的关系。

![](img/42616c8cc15143de45ca38e96dd1f2d3_8.png)

**ROI** 的计算公式为：
```
ROI = (收益 - 成本) / 成本
```
或者更简单地理解为 `投资回报率 = 产生交易的金额 / 活动投放成本`。

![](img/42616c8cc15143de45ca38e96dd1f2d3_9.png)

例如，某次营销活动花费了100元，最终带来了150元的销售额，那么这次活动的ROI就是 `(150-100)/100 = 0.5`，即50%。这个指标贯穿了从投入成本到最终产生价值的全过程，是评估商业活动效率的核心。

## 核心指标总结 📋

通过以上学习，我们已经掌握了数据分析中各个维度的关键指标。为了帮助大家系统记忆，以下是需要重点掌握的指标列表：

**用户维度指标**
*   **DAU**：日活跃用户数
*   **日活率**：日活跃用户占总用户的比例
*   **新增用户数**：新注册或首次访问的用户数
*   **留存率**：包括次日留存率、7日留存率、月留存率
*   **客单价**：平均每个订单的金额
*   **ARPU/ARPPU**：平均每用户收入 / 平均每付费用户收入
*   **单位访客成本**：获取一个访客所需的平均成本

**行为维度指标**
*   **PV**：页面浏览量
*   **UV**：独立访客数
*   **跳出率**：只浏览一个页面便离开的访问比例
*   **平均访问时长**：用户每次访问在网站或App上停留的平均时间

**产品维度指标**
*   **GMV**：成交总额，即流水
*   **ROI**：投资回报率
