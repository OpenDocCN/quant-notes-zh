# 量化交易教程：P6：04 金融量化分析-影响股价因素&股票买卖知识 📈

在本节课中，我们将要学习影响股票价格的主要因素，并了解股票买卖的基本流程与规则。理解这些基础知识是进行金融量化分析的第一步。

## 影响股价的六大因素

上一节我们介绍了股票的基本概念，本节中我们来看看哪些因素会决定股票价格的涨跌。影响股价的因素可以归纳为以下六个方面。

### 1. 公司自身因素
这是影响股价最根本的因素。公司的经营状况直接决定了其长期价值。如果公司发展良好，未来预期收益高，其股价通常会上涨。反之，如果公司出现重大负面事件或经营不善，股价就会下跌。

### 2. 市场因素
这是影响股价最直接的因素。股价在短期内由市场的供求关系决定。买的人多，卖的人少，股价就会上涨；卖的人多，买的人少，股价就会下跌。这与其他商品的定价原理一致。

### 3. 行业因素
整个行业的发展前景会影响行业内所有公司的股价。例如，一个蓬勃发展的朝阳行业（如人工智能）会带动相关公司股价上涨；而一个衰退的行业则可能导致行业内公司股价普遍下跌。

### 4. 心理因素
投资者的心理和情绪，尤其是从众心理，会对股价产生非理性的影响。例如，当市场出现恐慌性抛售时，即使公司基本面没有恶化，股价也可能因投资者的集体不理智行为而暴跌。

### 5. 经济因素
国家层面的宏观经济政策和指标会影响股市整体表现。例如，当银行存款利率上升时，可能会吸引资金从股市流向银行，导致市场资金减少，股价承压。

### 6. 政治因素
国际或国内的政治局势、军事冲突风险等事件会严重影响市场信心和资金流向。政治不稳定通常会导致投资者避险情绪升温，抛售风险资产，从而引发股价下跌。

## 股票买卖流程与规则

![](img/cffd5717872eaec44827030268af4452_1.png)

了解了影响股价的因素后，我们来看看个人投资者如何进行股票买卖，以及需要遵守哪些市场规则。

### 开户与委托
个人不能直接在证券交易所买卖股票，必须通过证券公司（券商）进行。以下是基本步骤：
1.  在券商处开设证券账户。
2.  通过券商提供的系统（如交易软件）连接至交易所服务器。
3.  提交买入或卖出指令，这个过程称为“委托”。

### 交易日与交易时间
股票市场并非全天候开放交易。

*   **交易日**：通常为每周一至周五（非法定节假日）。
*   **交易时间**：一般为每个交易日的上午9:30至11:30，下午13:00至15:00。其中，开盘前和收盘前有特殊的竞价阶段。

以下是交易时段的具体划分：

*   **开盘集合竞价 (9:15 - 9:25)**：在这段时间内提交的委托不会立即成交。交易所会收集所有买卖申报，在9:25一次性集中撮合，以产生当日的**开盘价**。其核心原则是**使成交量最大化**。
*   **连续竞价 (9:30 - 11:30, 13:00 - 14:57)**：这是主要的交易时段。交易所系统会对买卖申报进行几乎连续的撮合（例如每几秒一次），促成实时交易。
*   **收盘集合竞价 (仅深圳交易所，14:57 - 15:00)**：在最后三分钟，系统再次采用集合竞价方式，收集所有委托并在15:00一次性撮合，以产生当日的**收盘价**。上海交易所的收盘价为连续竞价阶段的最后一笔成交价。

### 重要交易制度
我们之前已经介绍过两个关键制度，这里再次强调：

1.  **T+1交易制度**：当天（T日）买入的股票，必须等到下一个交易日（T+1日）才能卖出。
2.  **涨跌停板限制**：为防止股价过度波动，A股市场设有涨跌幅限制。通常，普通股票在一个交易日内的价格波动被限制在前一交易日收盘价的±10%以内。
