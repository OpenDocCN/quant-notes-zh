## 数据拼接操作

上一节我们介绍了如何为单个因子进行打分。本节中我们来看看如何将多个因子的打分结果合并成一张大表，以便进行综合评估。

![](img/3a9ca046cd11321692768703f2a7c7e5_4.png)

![](img/3a9ca046cd11321692768703f2a7c7e5_6.png)

拼接操作的核心是使用Pandas的`concat`函数，将一个DataFrame与另一个DataFrame进行合并。

以下是拼接两个DataFrame的代码：

![](img/3a9ca046cd11321692768703f2a7c7e5_8.png)

```python
df_combined = pd.concat([df1, df2], axis=1)
```

![](img/3a9ca046cd11321692768703f2a7c7e5_10.png)

我们使用已有的`df_down` DataFrame，通过`concat`方法将第二个DataFrame（即其他因子的打分结果）合并进来。合并后，我们为这个新的大表起一个名字，例如`df_combined`。

![](img/3a9ca046cd11321692768703f2a7c7e5_12.png)

## 创建得分列并初始化

![](img/3a9ca046cd11321692768703f2a7c7e5_14.png)

现在我们已经有了一个包含所有因子评分的大表，但表中还没有综合排名的信息。

![](img/3a9ca046cd11321692768703f2a7c7e5_16.png)

接下来，我们需要新建一列来存放最终的得分值。首先，我们初始化这一列，使其值全部为零。

![](img/3a9ca046cd11321692768703f2a7c7e5_18.png)

以下是创建并初始化得分列的代码：

![](img/3a9ca046cd11321692768703f2a7c7e5_20.png)

```python
import numpy as np
df_combined[‘得分’] = np.zeros((df_combined.shape[0], 1))
```

![](img/3a9ca046cd11321692768703f2a7c7e5_22.png)

在这段代码中，`np.zeros`用于创建一个零值矩阵。矩阵的维度是`(样本数量, 1)`，即行数等于数据样本总数（例如300），列数为1。这样，我们就为每个股票创建了一个初始得分为0的列。

![](img/3a9ca046cd11321692768703f2a7c7e5_24.png)

## 计算综合总分并排序

初始化得分列后，我们需要计算每个股票的综合总分。综合总分是各个因子得分的加总。

以下是计算总分并排序的步骤：

![](img/3a9ca046cd11321692768703f2a7c7e5_26.png)

首先，我们对所有因子列进行横向求和。假设我们有六个因子列，求和操作如下：

![](img/3a9ca046cd11321692768703f2a7c7e5_28.png)

```python
total_score = df_combined[[‘因子1’, ‘因子2’, ‘因子3’, ‘因子4’, ‘因子5’, ‘因子6’]].sum(axis=1)
df_combined[‘得分’] = total_score
```

![](img/3a9ca046cd11321692768703f2a7c7e5_30.png)

![](img/3a9ca046cd11321692768703f2a7c7e5_32.png)

计算完总分后，我们需要根据得分进行排序，以找出排名最高的股票。我们希望分数高的排在前面，因此使用降序排序。

![](img/3a9ca046cd11321692768703f2a7c7e5_34.png)

以下是排序操作的代码：

```python
df_sorted = df_combined.sort_values(by=‘得分’, ascending=False)
```

![](img/3a9ca046cd11321692768703f2a7c7e5_36.png)

排序完成后，`df_sorted`就是一个按照综合得分从高到低排列的DataFrame。

![](img/3a9ca046cd11321692768703f2a7c7e5_38.png)

## 提取最终选股结果

![](img/3a9ca046cd11321692768703f2a7c7e5_40.png)

排序之后，我们通常只关心排名最靠前的股票，例如前十名。

![](img/3a9ca046cd11321692768703f2a7c7e5_42.png)

我们不需要返回整个排序后的DataFrame，只需要提取排名前十的股票代码（即索引）即可。

![](img/3a9ca046cd11321692768703f2a7c7e5_44.png)

![](img/3a9ca046cd11321692768703f2a7c7e5_46.png)

以下是提取前十名股票代码的代码：

![](img/3a9ca046cd11321692768703f2a7c7e5_48.png)

```python
top_10_stocks = df_sorted.index[:10].tolist()
```

![](img/3a9ca046cd11321692768703f2a7c7e5_50.png)

在这段代码中，`.index`获取了排序后DataFrame的索引（即股票代码），`[:10]`切片取前十个，`.tolist()`将其转换为列表格式。这样，`top_10_stocks`列表就包含了我们最终选出的前十只股票。

![](img/3a9ca046cd11321692768703f2a7c7e5_52.png)

## 函数流程总结

![](img/3a9ca046cd11321692768703f2a7c7e5_54.png)

本节课中我们一起学习了`get_stars`选股函数的核心构建流程。

![](img/3a9ca046cd11321692768703f2a7c7e5_56.png)

该函数的完整逻辑可以概括为以下几步：
1.  **读取因子数据**：获取各个因子的原始数据。
2.  **进行打分操作**：对每个因子进行标准化和评分。
3.  **拼接数据**：将所有因子的评分结果合并到一张大表中。
4.  **计算总分**：对每个股票的所有因子得分进行求和，得到综合得分。
5.  **排序与筛选**：根据综合得分进行降序排序，并提取排名前十的股票代码作为最终选股结果。

![](img/3a9ca046cd11321692768703f2a7c7e5_58.png)

执行一次这个函数，就能返回基于当前所有指标计算出的、综合排名前十的股票列表，从而完成一次完整的量化选股过程。
