## 概述

许多交易者都希望找到一套成熟的、能稳定获利的交易策略。然而，市场行情并非一成不变，它通常由**趋势**与**震荡**两种形态交替组成。一段大涨大跌的趋势行情后，往往会跟随一段震荡盘整，这是市场的常态。因此，我们的交易策略必须能够应对这种动态变化。

---

## 策略设计的三个核心问题

在设计任何交易策略时，我们通常会面临三个核心问题。理解它们是构建有效策略的基础。

以下是这三个关键因素：

1.  **胜率**：指交易总次数中，盈利交易次数的比例。公式可以表示为：
    `胜率 = 盈利交易次数 / 总交易次数`
2.  **赔率（盈亏比）**：指平均每笔盈利交易的利润与平均每笔亏损交易的损失之比。例如，一笔交易亏损10万，另一笔盈利30万，那么赔率就是3倍。
3.  **频率**：指策略在特定时间周期内（如一年）可执行的交易次数。

![](img/2773b4a6b97b627bcfdea21ee9e5605a_5.png)

一个理想的策略似乎应该同时具备高胜率、高赔率和高频率。然而，这构成了一个“不可能三角”，同时满足这三点的策略极难找到。策略的长期稳定运营是最大的挑战。

## 量化交易的四阶段流程

对于初学者，建议将量化交易过程系统地分为四个阶段。这有助于理清思路，逐步构建体系。

上一节我们介绍了策略设计的核心矛盾，本节中我们来看看构建一个完整策略的具体步骤。

![](img/2773b4a6b97b627bcfdea21ee9e5605a_13.png)

以下是四个关键阶段：

![](img/2773b4a6b97b627bcfdea21ee9e5605a_15.png)

*   **第一阶段：选择交易品种**
    你的选择不限于股票。除了A股，还可以考虑期货、期权，甚至境外的加密货币等。不同品种的波动特性、趋势形态和你的熟悉程度都不同，找到适合自己的市场环境至关重要。
*   **第二阶段：选择交易标的（选股/选币）**
    在选定市场后，需要进一步筛选具体的交易标的。可以通过基本面分析、技术趋势、波动率或市场消息等方式，从一个初始的“股票池”或“标的池”中持续跟踪，寻找机会。
*   **第三阶段：选择交易时机（择时）**
    确定具体标的后，需要决定何时进场与出场。这涉及到你的交易周期：是日内交易（看tick数据）、短线跨日（看分钟线、日线），还是长期持有（看周线、月线）。不同的数据粒度（如日K线、tick级委托数据）能揭示不同的市场信息。
*   **第四阶段：资金与仓位管理**
    这是风险控制的核心。你需要决定：是集中投资单一标的还是分散投资多个标的？进场时是全仓投入还是分批建仓？出场时是逐步平仓还是一次性清仓？是否会使用移动止损/止盈等技术指标来管理风险？良好的仓位管理可以在胜率不高时，通过提高赔率或容错率来保护资金。

这整套流程构成了一个完整的量化交易策略框架。每位交易者都可以在其中找到适合自己的路径。

## 策略与行情的匹配

![](img/2773b4a6b97b627bcfdea21ee9e5605a_17.png)

很多交易者在设计策略时，忽略了策略类型与市场行情的匹配。我们必须考虑策略是适用于**趋势市**还是**震荡市**。

![](img/2773b4a6b97b627bcfdea21ee9e5605a_19.png)

结合人性的弱点和对未来预测的不确定性，在对市场波动率和趋势有一定判断后，应选择与之匹配的交易策略。因为未来是未知的，任何交易都存在不确定性，而匹配的策略能帮助我们降低这种不确定性。

接下来，我们从两个维度来介绍经典策略。

### 维度一：趋势 vs. 震荡

*   **趋势策略**：当市场呈现明确的单边上涨或下跌时，最简单的策略就是顺势而为，尽快跟上趋势。
*   **震荡策略**：当价格在一定范围内反复上下波动时，策略是在下跌时买入，上涨时卖出，进行区间操作。

![](img/2773b4a6b97b627bcfdea21ee9e5605a_21.png)

![](img/2773b4a6b97b627bcfdea21ee9e5605a_23.png)

### 维度二：容错率（仓位管理）

容错率关系到你能否承受判断失误的代价。如果你极度看好某个标的，认为处于爆发起点，可能会选择全仓进场以提高潜在收益，但这会显著降低容错率——一旦判断错误，损失较大。

![](img/2773b4a6b97b627bcfdea21ee9e5605a_25.png)

以下是几种结合了行情判断与仓位管理的经典策略：

*   **海龟交易法则（趋势策略，低容错率）**
    这是一个经典的趋势跟踪策略。其核心规则是：当价格突破近期高点时买入进场；当价格跌破近期低点时卖出出场。它通常采用全仓操作，赌趋势的延续，因此胜率相对较低，容错率也较低。
    ```python
    # 海龟策略进场信号伪代码示例
    if current_price > highest(price, N): # 突破N日高点
        enter_long() # 开多仓
    elif current_price < lowest(price, N): # 跌破N日低点
        exit_long() # 平多仓
    ```
*   **布林通道策略（震荡策略）**
    布林通道由中轨（均线）和上下轨（标准差通道）构成。在震荡市中，当价格跌至下轨附近时买入，涨至上轨附近时卖出，通过“买低卖高”的区间操作获利。
*   **马丁格尔策略（高容错率策略）**
    该策略通过在亏损时加倍下注来摊平成本。例如，第一次交易亏损后，第二次交易投入双倍资金，以此类推，直到某一次盈利，从而覆盖之前的所有亏损并实现盈利。它通过仓位控制极大地提高了容错率，但需要强大的资金支持和心理承受能力，因为连续下跌会导致仓位急剧加重。
*   **网格交易策略（震荡策略，高容错率）**
    网格策略是震荡策略的典型代表。它在预设的价格区间内，设置一系列等间距的买入和卖出挂单。价格下跌时触发买入，上涨时触发卖出，自动进行“低买高卖”。与马丁策略等待大反弹不同，网格策略赚取的是区间内多次小波动的利润。它将交易分散到多个时点，提高了胜率和稳定性，但单次收益率可能降低。
