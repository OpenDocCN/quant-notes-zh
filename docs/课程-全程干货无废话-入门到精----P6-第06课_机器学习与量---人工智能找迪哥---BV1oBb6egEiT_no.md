## 概述

在本节课中，我们将要学习机器学习在量化交易中的一个关键预处理步骤——特征选择。我们将探讨三种主要的特征选择方法，并深入讲解两个在实践中极其有效但理论背景相对“玄学”的算法：遗传算法和神经网络的反向传播原理。掌握这些内容，将帮助你构建更高质量的预测模型。

## 特征选择的意义与方法

上一节我们介绍了如何从原始数据中构建用于监督学习的训练集（X和Y）。本节中我们来看看如何优化这些特征。

特征选择的核心意义在于提升模型输入X的质量。我们最初选择的特征可能多达数百个，其中可能包含冗余特征、无效特征（如数值全为零的列）或噪声。高质量的输入特征能让分类器或回归器发挥更好的性能，而低质量的输入即使用再好的模型也难以得到理想结果。

特征选择主要有以下三种思路：

1.  **子集选择**：从原始的P个特征中，选出一个包含K个特征的子集（K < P），然后仅基于这个子集进行预测。选出的特征值本身不变。
2.  **系数收缩/正则化**：不直接剔除特征，而是通过修改模型（如在线性回归中加入L1正则化），让不重要特征的系数自动趋近于零，从而实现特征选择。我们之前讲过的LASSO回归就是这种方法。
3.  **降维**：这是一种完全不同的方法。它将原始的P维特征向量通过一个函数（线性或非线性）变换到一个新的低维空间（例如Z，维度为M，M < P）。新特征的每一维都是原始特征的某种组合，其物理意义已发生改变。主成分分析（PCA）、自编码器（Autoencoder）等都是降维技术。

接下来，我们首先详细看看如何进行子集选择。

## 子集选择：从暴力到贪心

子集选择的目标是：从P个特征中找出一个最优的K特征子集，使得基于该子集构建的模型在某个评价指标（如回归中的R²、AIC或BIC）上表现最好。

然而，寻找全局最优子集是一个组合优化问题。如果我们暴力枚举所有可能的特征组合，其计算复杂度是指数级的（2^P），在P较大时不可行。因此，我们需要一些启发式方法来逼近最优解。

以下是几种常见的子集选择策略：

*   **完全子集选择（暴力枚举）**：对于每个可能的子集大小k（k=1, 2, ..., P），尝试所有C(P, k)种组合，选择评价指标最优的那个子集。这只在特征数很少时可行。
*   **前向选择**：这是一种贪心算法。
    1.  开始时，模型不包含任何特征。
    2.  尝试将每个特征单独加入模型，选择能带来最大评价指标提升的那个特征。
    3.  固定已选入的特征，在剩余特征中再次选择能带来最大提升的特征加入。
    4.  重复步骤3，直到评价指标不再显著提升或达到预设特征数。
*   **后向选择**：与前向选择相反。
    1.  开始时，模型包含所有特征。
    2.  尝试移除每一个特征，选择移除后导致评价指标下降最少的那个特征并将其剔除。
    3.  重复步骤2，直到评价指标开始显著下降或达到预设特征数。

**评价指标**：在回归任务中，常用R²值（决定系数）来衡量模型对数据方差的解释比例，其值越接近1越好。AIC（赤池信息准则）和BIC（贝叶斯信息准则）则是同时考虑模型拟合优度和复杂度的指标，值越小越好。

## 遗传算法：受进化论启发的优化艺术

上一节我们介绍了基于贪心策略的特征选择方法。本节中我们来看看一种完全不同的、受生物进化启发的全局优化算法——遗传算法。它不仅可用于特征选择，还能解决各种复杂的优化问题。

遗传算法的灵感来源于达尔文的自然选择学说和孟德尔的遗传学。其核心思想是：一个“种群”中的“个体”（代表问题的候选解）通过模拟“选择”、“交叉”和“变异”等生物进化过程，逐步演化出对环境（即优化目标）适应度更高的个体。

### 核心概念与流程

1.  **基因型与表现型**：
    *   **基因型**：个体内部表示，通常是一个编码串（如二进制串`10011001`）。在特征选择问题中，每一位代表一个特征是否被选中（1选中，0不选）。
    *   **表现型**：基因型对应的实际解。在特征选择中，表现型就是被选中的那组特征列。

2.  **适应度函数**：用于评价个体好坏的函数。在特征选择中，适应度函数可以是基于选中特征子集构建的模型的R²值。我们的目标是找到使适应度函数最大化的基因型。

3.  **算法流程**：
    *   **初始化**：随机生成一个包含N个个体的初始种群（每个个体是一个随机基因型串）。
    *   **迭代（演化）**：重复以下步骤直到满足终止条件（如达到最大迭代次数或适应度收敛）：
        *   **选择**：根据个体的适应度高低，以一定概率选择出部分优秀个体作为“父母”。适应度越高，被选中的概率越大。一种简单实现是：计算个体i的适应度`f_i`占种群总适应度的比例`p_i = f_i / sum(f)`，然后通过生成随机数来决定是否选中该个体。
        *   **交叉**：随机配对选出的“父母”个体，以某种方式交换他们的部分基因片段，产生新的“后代”个体。这模拟了生物的有性繁殖。
        *   **变异**：以一个小概率，随机改变新个体基因串上的某些位（如二进制串的0变1，1变0）。这为种群引入了新的遗传多样性。
    *   **输出**：迭代结束后，种群中适应度最高的个体即为所求的（近似）最优解。

遗传算法的强大之处在于其全局搜索能力，尤其适用于非凸、不可导或搜索空间巨大的复杂优化问题。它的缺点是需要精心设计编码方式、适应度函数和参数（种群大小、交叉变异概率等），且不能保证收敛到全局最优解，但通常能找到一个非常好的近似解。

**一个有趣的作业**：尝试用遗传算法破解一个密码字符串（如“hello world”）。适应度函数定义为猜测字符串与目标字符串匹配的字符数量。你的基因型就是字符串每个位置字符的编码。

## 神经网络与反向传播算法原理

遗传算法为我们提供了一种强大的优化工具。现在，我们将目光转向另一个受生物学启发的、在当今人工智能领域占据统治地位的模型——人工神经网络，并深入理解其核心训练算法：反向传播。

### 从生物神经元到数学模型

生物神经元接收来自其他神经元的电信号（通过树突），当这些信号的加权总和超过某个阈值时，神经元就会“激活”，产生一个输出信号（通过轴突传递给其他神经元）。

最简单的数学模型——**感知机**，模拟了这一过程：
*   输入：特征向量 `X = [x1, x2, ..., xn]`
*   计算加权和：`z = w1*x1 + w2*x2 + ... + wn*xn + b`。其中`w_i`是权重，`b`是偏置项（用于替代阈值，使得比较对象变为0）。
*   激活函数：早期使用阶跃函数，输出0或1。但因其不可导，后来被平滑的Sigmoid函数等替代。Sigmoid函数公式为：`σ(z) = 1 / (1 + e^{-z})`，它将输入映射到(0,1)区间。

一个神经网络就是由大量这样的神经元相互连接而成。

### 反向传播算法：神经网络如何学习

神经网络的学习目标是：找到一组权重参数W，使得网络的预测输出`Y_pred`与真实标签`Y_true`之间的差距（损失函数）最小化。这是一个典型的优化问题。

**梯度下降**是解决此类优化问题的基本方法。参数更新公式为：
`W_new = W_old - α * ∂Loss/∂W`
其中α是学习率，`∂Loss/∂W`是损失函数对权重的梯度（偏导数）。

**反向传播算法的精妙之处，就在于它高效地计算出了这个梯度`∂Loss/∂W`。**

对于复杂的多层网络，直接计算每个权重对最终损失的偏导非常困难。反向传播算法运用了**链式法则**和**动态规划**的思想：

1.  **前向传播**：输入数据从网络第一层流向最后一层，计算每一层的激活值以及最终的损失。
2.  **反向传播**：从最后一层开始，反向计算损失函数对每一层净输入（加权和）的偏导数，这个值称为“误差项”或“δ”。
    *   对于输出层，δ可以直接由损失函数和激活函数导数算出。
    *   对于隐藏层，当前层的δ等于后一层的δ乘以后一层权重，再乘以当前层激活函数的导数。**这正是链式法则的应用**。
3.  **计算梯度**：一旦有了某一层的δ，那么损失对该层权重W的梯度就很容易计算了：`∂Loss/∂W = δ * 前一层激活值`。
4.  **更新权重**：利用计算出的梯度，按照梯度下降公式更新所有权重。

反向传播的“动态规划”特性体现在：在反向计算过程中，每一层的δ只计算一次，就可以用于计算该层所有参数的梯度，以及继续向更前层传播误差。这避免了大量重复计算，使得训练深层网络成为可能。

### 循环神经网络简介

前面介绍的是前馈神经网络，其输出仅由当前输入决定。但在处理时间序列数据（如股价）时，我们常希望模型能记住过去的信息。**循环神经网络（RNN）** 应运而生。

![](img/3e51ec0bd2971dfc7534c38fd144a34b_2.png)

![](img/3e51ec0bd2971dfc7534c38fd144a34b_4.png)

RNN的核心思想是：网络不仅接收当前时刻的输入`X_t`，还接收上一个时刻网络的隐藏状态`H_{t-1}`。这个隐藏状态`H_{t-1}`承载了历史信息。当前时刻的输出`Y_t`和新的隐藏状态`H_t`由`X_t`和`H_{t-1}`共同决定。公式可以简化为：
`H_t = σ(W_hh * H_{t-1} + W_xh * X_t + b)`
`Y_t = W_hy * H_t + b_y`
这使得RNN具有了短期记忆能力。

**长短期记忆网络（LSTM）** 是RNN的一种改进，它通过引入“门”机制（输入门、遗忘门、输出门），更精细地控制信息的保留与遗忘，有效缓解了RNN在训练长序列时容易出现的梯度消失或爆炸问题，从而能够学习更长期的依赖关系。
