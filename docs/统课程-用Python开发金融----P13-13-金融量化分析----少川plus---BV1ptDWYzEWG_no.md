## 概述

![](img/b436671f17a2ff017b98deec71f74356_10.png)

布尔型索引的核心思想是：**传入一个与数组长度相同的布尔值数组（或列表），NumPy会返回所有对应位置为`True`的元素**。这通常结合数组的条件运算（如比较运算）来使用，可以一步完成复杂的数据筛选。

![](img/b436671f17a2ff017b98deec71f74356_16.png)

![](img/b436671f17a2ff017b98deec71f74356_18.png)

## 布尔型索引的原理

![](img/b436671f17a2ff017b98deec71f74356_20.png)

其工作原理可以分为两步：

![](img/b436671f17a2ff017b98deec71f74356_22.png)

1.  **数组与标量的运算**：首先，对数组执行一个条件判断（例如 `array > 5`）。这个操作会返回一个与原数组形状相同的布尔值数组，其中每个位置表示原数组对应元素是否满足该条件。
    ```python
    # 示例：数组与标量比较，返回布尔数组
    import numpy as np
    a = np.array([8, 3, 6, 2, 9])
    bool_array = a > 5  # 返回 [True, False, True, False, True]
    ```

![](img/b436671f17a2ff017b98deec71f74356_24.png)

2.  **使用布尔数组进行索引**：然后，将这个布尔数组放入原数组的索引括号 `[]` 中。NumPy会自动选取所有在布尔数组中对应位置为 `True` 的元素，并返回一个新数组。
    ```python
    # 使用上一步得到的布尔数组进行索引
    result = a[bool_array]  # 返回 [8, 6, 9]
    ```

![](img/b436671f17a2ff017b98deec71f74356_26.png)

在实际使用中，我们通常将这两步合并为一行简洁的代码：`a[a > 5]`。

![](img/b436671f17a2ff017b98deec71f74356_28.png)

## 基础用法示例

![](img/b436671f17a2ff017b98deec71f74356_30.png)

![](img/b436671f17a2ff017b98deec71f74356_31.png)

让我们通过一个具体例子来理解。假设我们有一个数组，需要筛选出所有大于5的元素。

![](img/b436671f17a2ff017b98deec71f74356_33.png)

![](img/b436671f17a2ff017b98deec71f74356_35.png)

以下是使用Python内置`filter`函数和NumPy布尔索引的对比：

![](img/b436671f17a2ff017b98deec71f74356_37.png)

```python
import numpy as np

![](img/b436671f17a2ff017b98deec71f74356_39.png)

![](img/b436671f17a2ff017b98deec71f74356_41.png)

# 创建一个示例列表
data_list = [8, 3, 6, 2, 9]

![](img/b436671f17a2ff017b98deec71f74356_43.png)

# 方法一：使用Python的filter和lambda（相对繁琐）
filtered_list = list(filter(lambda x: x > 5, data_list))
print(filtered_list)  # 输出: [8, 6, 9]

![](img/b436671f17a2ff017b98deec71f74356_45.png)

![](img/b436671f17a2ff017b98deec71f74356_47.png)

# 方法二：使用NumPy布尔型索引（简洁高效）
data_array = np.array(data_list)
filtered_array = data_array[data_array > 5]
print(filtered_array)  # 输出: [8 6 9]
```

![](img/b436671f17a2ff017b98deec71f74356_49.png)

可以看到，NumPy的布尔型索引语法 `array[array > 5]` 非常直观和简洁。

![](img/b436671f17a2ff017b98deec71f74356_51.png)

## 组合条件筛选

![](img/b436671f17a2ff017b98deec71f74356_53.png)

![](img/b436671f17a2ff017b98deec71f74356_55.png)

![](img/b436671f17a2ff017b98deec71f74356_56.png)

布尔型索引的强大之处在于可以方便地组合多个筛选条件。NumPy使用位运算符来组合布尔数组，而不是Python的关键字`and`和`or`。

![](img/b436671f17a2ff017b98deec71f74356_58.png)

以下是组合条件的使用方法：

![](img/b436671f17a2ff017b98deec71f74356_60.png)

![](img/b436671f17a2ff017b98deec71f74356_62.png)

*   **与运算 (`&`)**：筛选同时满足多个条件的元素。**注意：每个条件必须用括号括起来**。
    ```python
    # 筛选大于5且为偶数的元素
    a = np.array([8, 3, 6, 2, 9, 12])
    result = a[(a > 5) & (a % 2 == 0)]
    print(result)  # 输出: [8, 6, 12]
    ```

*   **或运算 (`|`)**：筛选满足任意一个条件的元素。
    ```python
    # 筛选大于5或是偶数的元素
    result = a[(a > 5) | (a % 2 == 0)]
    print(result)  # 输出: [8, 3, 6, 2, 9, 12]
    ```

**重要提示**：务必使用 `&` 和 `|` 而不是 `and` 和 `or`，并且每个条件表达式都需要用括号包裹，这是因为运算符的优先级问题。

## 常见错误与注意事项

![](img/b436671f17a2ff017b98deec71f74356_64.png)

在使用布尔型索引时，初学者可能会遇到以下问题：

1.  **长度不匹配**：用于索引的布尔数组必须与原数组长度严格一致。
    ```python
    # 错误示例：分步筛选后，第二步的数组长度已改变
    temp = a[a > 5]  # temp长度为3
    # result = temp[temp % 2 == 0] # 这样写temp的长度已不是原数组a的长度
    # 正确做法应使用组合条件一次完成：a[(a > 5) & (a % 2 == 0)]
    ```

![](img/b436671f17a2ff017b98deec71f74356_66.png)

2.  **错误使用运算符**：如前所述，组合条件时应使用 `&` 和 `|`，而非 `and` 和 `or`。

![](img/b436671f17a2ff017b98deec71f74356_68.png)
