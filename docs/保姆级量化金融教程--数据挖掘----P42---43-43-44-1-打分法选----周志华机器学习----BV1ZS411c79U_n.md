在本节课中，我们将学习量化选股中一个常用且直观的策略——因子打分法。我们将了解其核心思想、实施步骤，并理解为何需要预先知道因子的方向性。

## 策略核心思想

上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中，我们来看看一个更综合的方法：因子打分法。

该策略的核心思想是：**为每只股票的多个因子分别打分，然后计算总分，最后根据总分排名来选取表现最佳的股票**。这类似于学生考试，总成绩由各科成绩加总而成，总分高的学生排名靠前。

假设我们有300只股票和A、B、C、D四个因子指标。我们的目标不是单独看某个因子，而是为每只股票的每个因子打分，例如：
*   股票1：A因子得10分，B因子得15分，C因子得8分，D因子得12分，**总分 = 45分**。
*   股票2：A因子得8分，B因子得20分，C因子得10分，D因子得10分，**总分 = 48分**。

计算所有股票的总分后，进行排名，选取总分最高的前10或20只股票作为投资组合。在下一次调仓时，重复此过程。

## 关键前提：因子方向性

在实施打分前，有一个至关重要的前提条件：**我们必须预先知道每个因子与预期收益率的相关性是正向还是负向**。

这是因为，打分需要依据因子数值的大小，但并非数值越大打分就一定越高。例如：
*   **正向因子**：因子值越大，预期收益越好。如`每股收益(EPS)`、`净资产收益率(ROE)`。对于这类因子，数值越大，我们应给予**越高**的分数。
*   **负向因子**：因子值越小，预期收益越好。如`市盈率(PE)`、`市净率(PB)`。对于这类因子，数值越小，我们应给予**越高**的分数。

如何确定因子的方向性呢？以下是两种主要方法：
1.  **查阅研究报告**：各大券商和金融机构会发布大量研究报告，其中包含对各类因子有效性的分析和经验结论。
2.  **自行计算分析**：我们可以像之前学习因子分析时一样，通过计算因子与收益率的**IC值**或进行相关性分析来验证其方向。

## 策略实施步骤概述

了解核心思想和前提后，因子打分法的实施可以概括为以下几个关键步骤：

以下是实施因子打分法选股的关键步骤：
1.  **因子选择与预处理**：选择一组方向性已知的因子，并处理数据中的缺失值、异常值。
2.  **因子标准化**：将不同量纲和范围的因子值转换到同一尺度（如Z-Score标准化），使其具有可比性。
3.  **因子打分**：根据因子的方向性，对标准化后的因子值进行打分。例如，对正向因子，数值越大分数越高。
4.  **计算总分**：将每只股票在所有因子上的得分相加（或加权平均），得到该股票的综合总分。
    *   公式：`股票总分 = 因子1得分 + 因子2得分 + ... + 因子N得分`
5.  **排序与选股**：根据总分对所有股票进行降序排列，选取排名靠前的股票构成投资组合。
6.  **定期调仓**：在设定的周期（如每月）重复上述过程，更新投资组合。
