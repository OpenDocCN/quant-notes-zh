## 从CAPM模型看机器学习
现在，我们从一个经典金融模型切入，看看机器学习视角如何理解它。

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_37.png)

### 什么是CAPM模型？
资本资产定价模型描述的是单个资产（或投资组合）的预期收益与市场整体收益之间的关系。

其核心公式为：
`R_i = α_i + β_i * R_m + ε_i`
其中：
*   `R_i`：资产i在时刻t的收益率。
*   `R_m`：市场组合在时刻t的收益率。
*   `β_i`：资产i的系统性风险系数，衡量资产收益对市场收益变动的敏感程度。
*   `α_i`：资产i的超额收益，代表无法被市场解释的部分。
*   `ε_i`：随机误差项，期望值为0。

**市场有效假说**认为，从长期看，`α_i`的期望值为零，即无法持续获得超越市场的收益。

### 机器学习的视角
让我们暂时忘记金融理论，纯粹从预测模型的角度看：

1.  **问题定义**：我们想预测招商银行股票每天的收益率`Y`。
2.  **模型假设**：`Y`是某些输入变量`X`的函数，加上一个期望为零的随机噪声`ε`。即 `Y = f(X) + ε`。
3.  **CAPM的假设**：CAPM认为，影响`Y`的变量`X`只有一个，就是市场收益率`R_m`，并且它们之间的关系是线性的，即 `f(X) = α + β * X`。

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_39.png)

**结论**：从机器学习角度看，CAPM就是一个简单的**单变量线性回归模型**。`β`是斜率，`α`是截距。我们可以用历史数据`(R_m, R_i)`进行拟合，求出`α`和`β`。

### 模型的局限与拓展
现实世界远比线性模型复杂：
*   `α`真的为零吗？如果不为零，就存在获得超额收益的可能。
*   关系真的是线性的吗？我们可以引入更多变量（特征），并使用更复杂的非线性模型。
*   **技术分析**的本质，就是寻找除了`R_m`之外的其他有效特征`X`（如移动平均线、动量、布林带等），并寻找`Y = f(X)`中更优的函数`f`。

这个视角将传统的金融分析无缝地引向了机器学习领域：**量化交易的核心，就是寻找有预测力的特征`X`，并建立准确的预测模型`f`。**

## 技术分析简介
技术分析认为，可以从历史价格和交易量数据中提取出预测未来走势的特征。

### 常见技术指标（特征）
以下是几类基本的技术指标，它们都可以作为机器学习模型中的特征`X`：

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_49.png)

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_51.png)

1.  **动量**：衡量价格变动的速度和强度。公式近似为：`(当前价格 / N期前价格) - 1`。
2.  **简单移动平均线**：过去N个时间点价格的平均值，用于平滑价格序列，反映趋势。
3.  **布林带**：在移动平均线上下各画出两条标准差带，用以衡量价格波动性和判断超买超卖状态。

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_53.png)

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_55.png)

### 特征工程的重要性
在将技术指标输入模型前，必须进行**特征标准化**。因为不同指标的量纲和取值范围差异巨大，不做标准化会导致模型被取值大的特征主导。

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_57.png)

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_59.png)

标准化通常采用：
`X_standardized = (X - mean(X)) / std(X)`

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_61.png)

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_63.png)

### 技术分析的适用场景
*   **多指标组合**：单一指标预测能力有限，需要多个指标组合使用。
*   **短时间尺度**：技术分析更适用于日内或短期的预测，长期预测效力下降。
*   **异类检测**：寻找与大盘走势不一致的股票，这些地方可能存在套利机会。

![](img/790cdaa1d72fdbe2fcfbeaf55bff559c_65.png)

与之相对的是**基本面分析**，它更适用于长期投资。选择哪种方法取决于你的交易时间尺度。

## 机器学习模型基础
现在，我们正式进入如何学习`f(X)`的部分。

### 问题的形式化
我们有历史数据，即许多对`(X, Y)`。我们的目标是找到一个函数`f`，使得对于新的`X_new`，能预测出`Y_new`。模型总可以表示为：`Y = f(X) + ε`，其中`ε`是噪声。

### 模型复杂度与过拟合
1.  **K近邻**：最简单的非参数方法。预测新样本时，找训练集中最相似的K个样本，用它们的`Y`值平均作为预测。但在高维数据中效果差（维数灾难）。
2.  **线性回归**：假设`f`是线性的，即`Y = β_0 + β_1*X_1 + ... + β_p*X_p`。目标是找到一组`β`，使预测误差最小。
3.  **过拟合**：如果模型过于复杂（如高阶多项式），它可能会完美拟合训练数据中的每一个点（包括噪声），但在未见过的测试数据上表现糟糕。这就是**过拟合**。

### 偏差-方差权衡
这是机器学习的核心概念：
*   **偏差**：模型在训练集上的错误率。模型越复杂，偏差可能越低。
*   **方差**：模型对于不同训练集的敏感度。模型越复杂，方差通常越高。
*   **权衡**：我们需要在偏差和方差之间取得平衡，选择一个复杂度适中的模型，使其在训练集和测试集上都有良好表现，即**泛化能力**强。
