## 概述

![](img/833d6c3098f06cdd40c207e2c9f9ced1_8.png)

![](img/833d6c3098f06cdd40c207e2c9f9ced1_10.png)

上一节我们完成了股票历史交易数据的预处理，包括将日期列转换为时间序列并设置为行索引。本节中，我们将利用处理好的数据，实现一个具体的分析需求：找出收盘价相对于开盘价涨幅超过3%的所有交易日。这个技能是量化分析的基础，能帮助我们识别股票的强势上涨日期。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_12.png)

## 需求分析与实现步骤

我们的目标是找出满足 `(收盘价 - 开盘价) / 开盘价 > 0.03` 条件的所有日期。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_14.png)

以下是实现此需求的核心步骤：

1.  **构建布尔条件**：根据涨跌幅公式，创建一个布尔序列（Series），其中`True`表示该交易日满足上涨3%的条件。
2.  **应用布尔索引**：将这个布尔序列作为原始DataFrame的行索引，从而筛选出所有`True`对应的行数据。
3.  **提取日期**：从筛选出的行数据中，获取其行索引（即日期），得到最终结果。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_16.png)

![](img/833d6c3098f06cdd40c207e2c9f9ced1_18.png)

### 步骤一：构建布尔条件

![](img/833d6c3098f06cdd40c207e2c9f9ced1_20.png)

首先，我们需要将需求转化为代码逻辑。在Pandas中，我们可以直接对整列数据进行向量化运算。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_22.png)

核心公式如下：
`(df[‘close’] - df[‘open’]) / df[‘open’] > 0.03`

![](img/833d6c3098f06cdd40c207e2c9f9ced1_24.png)

对应的Python代码如下：
```python
# 假设 df 是我们的股票数据DataFrame，且已处理好索引
condition = (df[‘close’] - df[‘open’]) / df[‘open’] > 0.03
```
执行这行代码后，`condition` 变量将成为一个布尔类型的Series，其索引与`df`相同，值为`True`或`False`，分别代表对应日期是否满足上涨条件。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_26.png)

### 步骤二：应用布尔索引筛选数据

![](img/833d6c3098f06cdd40c207e2c9f9ced1_28.png)

在Pandas中，有一个非常实用的技巧：**可以将一个布尔序列直接作为DataFrame的行索引，用于筛选数据**。其规则是：只保留布尔序列中值为`True`所对应的行，忽略值为`False`的行。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_30.png)

![](img/833d6c3098f06cdd40c207e2c9f9ced1_32.png)

因此，我们可以这样筛选数据：
```python
filtered_df = df[condition]
```
此时，`filtered_df` 是一个新的DataFrame，它只包含了原数据中所有满足上涨3%条件的交易日数据。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_34.png)

![](img/833d6c3098f06cdd40c207e2c9f9ced1_36.png)

### 步骤三：提取目标日期

![](img/833d6c3098f06cdd40c207e2c9f9ced1_38.png)

我们的最终目标是日期，而不是整行数据。由于之前已将日期设置为行索引，因此满足条件的日期就是 `filtered_df` 的行索引。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_40.png)

我们可以通过以下代码获取这些日期：
```python
up_dates = filtered_df.index
```
`up_dates` 就是一个包含了所有收盘价比开盘价上涨超过3%的日期的索引对象。

![](img/833d6c3098f06cdd40c207e2c9f9ced1_42.png)

![](img/833d6c3098f06cdd40c207e2c9f9ced1_44.png)

## 核心技巧总结

![](img/833d6c3098f06cdd40c207e2c9f9ced1_46.png)

在本节中，我们一起学习了如何捕获股票上涨的日期。整个过程的核心在于熟练运用 **布尔索引（Boolean Indexing）**。

我们总结出一个重要的数据处理经验：
> 在数据分析过程中，如果通过逻辑判断产生了一组布尔值，下一步通常就是将这组布尔值作为原数据的行索引，从而快速、高效地筛选出目标数据。
