# Python数据分析实战：P34：06：数据类型-字典的各种操作 📚

在本节课中，我们将学习Python字典的核心操作，包括增加、删除、修改、查找、循环以及嵌套。字典是Python中一种非常强大的数据结构，它以键值对的形式存储数据，能够高效地通过键来访问值。

---

## 字典的增加操作 ➕

![](img/0bcf244da5ce585feabb13197941caf1_1.png)

上一节我们介绍了字典的基本概念，本节中我们来看看如何向字典中添加新的键值对。

![](img/0bcf244da5ce585feabb13197941caf1_3.png)

字典的增加操作很简单。与列表使用 `append` 方法不同，字典直接通过指定新的键名并赋值来增加元素。

**代码示例：**
```python
d = {}  # 创建一个空字典
d[‘name‘] = ‘ALEX‘  # 增加一个键值对
d[‘age‘] = 22       # 再增加一个键值对
d[1] = ‘随便写了‘    # 键也可以是数字
```
执行后，字典 `d` 中就有了三个键值对。

需要注意的是，如果增加的键名在字典中已经存在，那么该操作会变成修改操作，即用新的值覆盖该键原有的值。

**代码示例：**
```python
d[‘name‘] = ‘金角大王‘  # 此时‘name‘键已存在，此操作为修改
```
执行后，键 `‘name‘` 对应的值从 `‘ALEX‘` 被修改为 `‘金角大王‘`。

## 字典的删除操作 🗑️

学会了如何增加数据，接下来我们学习如何删除字典中的数据。字典的删除操作主要有几种方法。

以下是几种常用的删除方法：
*   **`del` 语句**：这是一个通用的删除方法，可以删除字典中指定的键。
    ```python
    del d[1]  # 删除键为1的键值对
    ```
*   **`pop(key)` 方法**：删除指定键的键值对，并返回被删除的值。如果键不存在，会报 `KeyError` 错误。
    ```python
    value = d.pop(‘name‘)  # 删除‘name‘键，并将其值赋给value
    ```
*   **`clear()` 方法**：清空整个字典，使其变为空字典。
    ```python
    d.clear()  # 清空字典d
    ```

## 字典的查找操作 🔍

数据存好了，我们自然需要知道如何取出数据。字典的查找，即取值操作，是其核心功能。

取值操作非常简单，直接通过键名即可获取对应的值。

**代码示例：**
```python
dic = {‘name‘: ‘ALEX‘, ‘age‘: 22, ‘job‘: ‘黑姑娘‘}
value = dic[‘name‘]  # 取出键‘name‘对应的值‘ALEX‘
```

字典**不支持**像列表那样的切片操作，无法一次性取出多个值。

要检查一个键是否存在于字典中，可以使用 `in` 关键字。

**代码示例：**
```python
print(‘name‘ in dic)  # 输出：True
print(‘salary‘ in dic) # 输出：False
```
需要注意的是，字典的设计是通过键来查找值，而不是通过值来查找键。如果想通过值反向查找，效率会很低。

此外，字典还提供了几个有用的方法来获取所有键、值或键值对。

以下是几个重要的方法：
*   **`keys()`**：返回字典中所有的键。
    ```python
    all_keys = dic.keys()  # 返回类似 [‘name‘, ‘age‘, ‘job‘] 的视图对象
    ```
*   **`values()`**：返回字典中所有的值。
    ```python
    all_values = dic.values() # 返回所有值的视图对象
    ```
*   **`items()`**：返回字典中所有的键值对，每个键值对以元组形式表示。
    ```python
    all_items = dic.items() # 返回类似 [(‘name‘, ‘ALEX‘), (‘age‘, 22), (‘job‘, ‘黑姑娘‘)] 的视图对象
    ```

## 字典的循环操作 🔄

掌握了基本的增删改查后，我们来看看如何遍历字典中的所有数据。循环字典有多种方式。

以下是几种循环字典的方法：
*   **循环所有的键**：直接循环字典对象，默认就是循环其键。
    ```python
    for key in dic:
        print(key)  # 依次打印出每个键
    ```
*   **同时循环键和值（推荐）**：在循环键的同时，通过键取出值。这是官方推荐的高效方法。
    ```python
    for key in dic:
        print(key, dic[key])  # 同时打印键和对应的值
    ```
*   **使用 `items()` 方法循环**：该方法将键值对打包成元组进行循环。
    ```python
    for key, value in dic.items():
        print(key, value)  # 直接获取键和值
    ```
    虽然这种方法直观，但因为它涉及一个转换过程，在处理大量数据时效率不如上一种方法。

## 其他常用操作

除了核心的增删改查和循环，还有一些实用的操作需要了解。

**求字典长度**
使用内置的 `len()` 函数可以获取字典中键值对的数量。这个函数是全局的，也适用于列表、字符串等。

**代码示例：**
```python
length = len(dic)  # 返回字典dic中键值对的数量，例如 3
str_len = len(“Hello“) # 返回字符串长度 5
```

**字典的嵌套**
字典的值可以是任意数据类型，包括另一个字典，这就形成了嵌套字典。

**代码示例：**
```python
company = {
    ‘jack‘: {‘salary‘: 5000, ‘age‘: 25},
    ‘alex‘: {‘salary‘: 8000, ‘age‘: 30}
}
```
要访问嵌套字典中的值，需要逐层指定键。

**代码示例：**
```python
jack_salary = company[‘jack‘][‘salary‘]  # 获取jack的薪水，结果为5000
```
嵌套的层数没有限制，但层数过多会影响代码的可读性。
