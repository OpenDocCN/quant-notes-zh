# 量化交易教程：P10：11 金融量化分析-numpy-array创建

在本节课中，我们将要学习如何使用NumPy库创建数组。这是进行金融数据分析和量化计算的基础。我们将介绍多种创建数组的方法，包括创建全零、全一、空数组以及特定范围的数组，并解释它们各自的特点和适用场景。

上一节我们介绍了NumPy数组（ndarray）的一些常用属性，本节中我们来看看如何创建这些数组。我们已经知道可以使用`np.array()`方法将列表转换为数组，但NumPy还提供了更多便捷的函数来创建具有特定内容的数组。

## 创建全零数组

![](img/409e6526c5b851a442fee595d44e0480_1.png)

![](img/409e6526c5b851a442fee595d44e0480_3.png)

以下是创建全零数组的方法。使用`np.zeros()`函数可以快速创建一个所有元素都为0的数组。

![](img/409e6526c5b851a442fee595d44e0480_5.png)

```python
import numpy as np
# 创建一个长度为10的全零数组，默认数据类型为浮点数（float64）
a = np.zeros(10)
print(a)
print(a.dtype)  # 输出：float64

# 创建一个长度为10的全零整数数组
a_int = np.zeros(10, dtype=int)
print(a_int)
print(a_int.dtype)  # 输出：int64
```
`np.zeros()`函数默认创建浮点数类型的数组，这是为了在进行除法等运算时避免类型错误。如果需要整数数组，可以通过`dtype`参数指定。

![](img/409e6526c5b851a442fee595d44e0480_7.png)

## 创建全一数组

![](img/409e6526c5b851a442fee595d44e0480_9.png)

与全零数组类似，以下是创建全一数组的方法。使用`np.ones()`函数可以创建一个所有元素都为1的数组。

![](img/409e6526c5b851a442fee595d44e0480_11.png)

```python
# 创建一个长度为5的全一数组，默认数据类型为浮点数
b = np.ones(5)
print(b)

![](img/409e6526c5b851a442fee595d44e0480_13.png)

# 创建一个3x2的全一整数矩阵
b_int = np.ones((3, 2), dtype=int)
print(b_int)
```
同样，`np.ones()`默认生成浮点数组，使用`dtype`参数可以改变数据类型。

![](img/409e6526c5b851a442fee595d44e0480_15.png)

![](img/409e6526c5b851a442fee595d44e0480_17.png)

## 创建空数组

![](img/409e6526c5b851a442fee595d44e0480_19.png)

接下来，我们看看如何创建空数组。使用`np.empty()`函数可以分配一块内存空间但不初始化其值，因此数组内容是不确定的随机值。

![](img/409e6526c5b851a442fee595d44e0480_21.png)

![](img/409e6526c5b851a442fee595d44e0480_23.png)

```python
# 创建一个长度为100的空数组
c = np.empty(100)
print(c)
```
`np.empty()`只分配内存，不进行初始化（即不将内存值设置为0或1）。这意味着数组中的值是分配到的内存块中残留的旧数据。这种方法比`np.zeros()`或`np.ones()`速度稍快，但前提是你后续会覆盖数组中的所有值。如果不确定是否会覆盖所有值，建议使用`np.zeros()`。

![](img/409e6526c5b851a442fee595d44e0480_25.png)

![](img/409e6526c5b851a442fee595d44e0480_27.png)

## 创建序列数组

在Python列表中，我们使用`range()`生成序列。NumPy提供了类似的`np.arange()`函数来创建数组序列。

![](img/409e6526c5b851a442fee595d44e0480_29.png)

以下是使用`np.arange()`创建序列数组的方法。

![](img/409e6526c5b851a442fee595d44e0480_31.png)

```python
# 生成0到99的整数序列（前闭后开）
seq1 = np.arange(100)
print(seq1)

![](img/409e6526c5b851a442fee595d44e0480_33.png)

# 生成10到20的整数序列，步长为2
seq2 = np.arange(10, 20, 2)
print(seq2)

![](img/409e6526c5b851a442fee595d44e0480_35.png)

![](img/409e6526c5b851a442fee595d44e0480_37.png)

# 生成0到5的序列，步长为0.5（支持小数步长，这是原版range不具备的）
seq3 = np.arange(0, 5, 0.5)
print(seq3)
```
`np.arange(start, stop, step)`的规则与`range()`类似，是“前闭后开”的。它的优势在于步长（`step`）可以是小数。

![](img/409e6526c5b851a442fee595d44e0480_39.png)

![](img/409e6526c5b851a442fee595d44e0480_41.png)

## 创建线性间隔数组

![](img/409e6526c5b851a442fee595d44e0480_43.png)

另一个创建序列的函数是`np.linspace()`，它用于在指定的区间内生成等间隔的数值。

![](img/409e6526c5b851a442fee595d44e0480_45.png)

![](img/409e6526c5b851a442fee595d44e0480_47.png)

以下是使用`np.linspace()`创建数组的方法。

![](img/409e6526c5b851a442fee595d44e0480_49.png)

![](img/409e6526c5b851a442fee595d44e0480_51.png)

```python
# 在0到100之间均匀生成101个点（包含终点）
lin1 = np.linspace(0, 100, 101)
print(lin1)
print(lin1.size)  # 输出：101

# 在-10到10之间均匀生成10000个点，用于绘制平滑的函数图像
x = np.linspace(-10, 10, 10000)
y = x ** 2  # 计算y = x^2
# 此处y可以与绘图库（如matplotlib）结合绘制函数图像
```
`np.linspace(start, stop, num)`的参数`num`指定了要生成的样本数量，区间是“前闭后闭”的，即包含起点和终点。这在需要均匀采样（例如绘制函数图像）时非常有用。

![](img/409e6526c5b851a442fee595d44e0480_53.png)

## 创建单位矩阵

![](img/409e6526c5b851a442fee595d44e0480_55.png)

![](img/409e6526c5b851a442fee595d44e0480_57.png)

最后，我们介绍一个在线性代数中常用的函数`np.eye()`，用于创建单位矩阵。

![](img/409e6526c5b851a442fee595d44e0480_59.png)

![](img/409e6526c5b851a442fee595d44e0480_61.png)

以下是创建单位矩阵的方法。

![](img/409e6526c5b851a442fee595d44e0480_63.png)

![](img/409e6526c5b851a442fee595d44e0480_65.png)

```python
# 创建一个3x3的单位矩阵
identity_matrix = np.eye(3)
print(identity_matrix)
```
单位矩阵是一个方阵，主对角线上的元素为1，其余元素为0。在金融量化分析中，除非涉及特定的线性代数运算，否则使用频率相对较低。

![](img/409e6526c5b851a442fee595d44e0480_67.png)

![](img/409e6526c5b851a442fee595d44e0480_69.png)

本节课中我们一起学习了多种创建NumPy数组的方法：`np.zeros()`, `np.ones()`, `np.empty()`, `np.arange()`, `np.linspace()`和`np.eye()`。每种方法都有其特定的用途，从初始化固定值数组到生成数值序列，为后续的数据处理和计算奠定了坚实的基础。理解这些创建方式的区别，能帮助你在量化分析中选择最合适的工具。
