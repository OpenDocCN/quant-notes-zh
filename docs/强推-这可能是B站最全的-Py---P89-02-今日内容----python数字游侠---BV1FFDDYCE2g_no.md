## 抽样分布 📊

上一节我们介绍了抽样分布的基本概念，本节中我们将深入探讨几种重要的抽样分布。

这些分布均与正态分布相关，或者说是在正态分布的基础上推导出来的。主要包括T分布、卡方分布和F分布。在此处，我们主要基于理论介绍这几种分布。后续构建模型时，会用到这些分布，因为当数据符合某种分布时，我们就可以使用对应的公式来计算相关的概率值。

我们将介绍抽样分布，并再次详细讲解中心极限定理，以确保大家能够充分理解。

![](img/78bdd077e5e2e98b94faae6bcd969bce_7.png)

## 参数估计 🔍

讲完抽样分布之后，我们将进入参数估计部分。

参数估计主要包括以下两种方法：
*   **点估计**：用样本统计量的某个取值直接作为总体参数的估计值。
*   **区间估计**：在点估计的基础上，给出一个包含总体参数真值的概率范围（置信区间）。

参数估计的内容相对比较简单。
