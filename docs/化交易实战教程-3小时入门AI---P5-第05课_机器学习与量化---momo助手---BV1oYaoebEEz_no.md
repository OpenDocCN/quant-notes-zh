## 构建监督学习训练集
我们的目标是建立一个监督学习框架。假设原始数据表如下（`T` 代表时间点）：

![](img/d5530717112685e1663bce4acffb9e33_3.png)

| T | open | close | high | low | ... |
|---|---|---|---|---|---|
| 0 | ... | ... | ... | ... | ... |
| 1 | ... | ... | ... | ... | ... |
| ... | ... | ... | ... | ... | ... |

对于每一个时刻 `T0`，我们需要：
*   **特征 `X`**：将 `T0` 时刻及之前的所有信息，转换为一组新的特征 `[X1, X2, ..., XN]`。
*   **标签 `Y`**：`T0` 时刻之后我们想预测的目标，例如 `T0+1` 时刻的涨跌（分类）或价格/收益率（回归）。

**核心公式**：
`Y(T0) = f( X(T0), X(T0-1), ... )`
其中 `f` 是我们需要训练的模型。

输入特征 `X` 的质量对模型性能起决定性作用，即“垃圾进，垃圾出”。本节课我们先学习如何生成丰富的特征，下节课再学习如何从中筛选和优化。

## 特征（因子）选择方法论
特征的选择完全取决于你对所预测金融产品的理解。这是一个结合经验、灵感和实验的过程。

**基本原则**：将你认为所有可能相关的因素都作为候选特征。例如，预测沪深300指数，你可能会考虑人民币汇率、黄金价格、标普500指数等。初始阶段应“多多益善”。

**常见错误**：在计算 `T0` 时刻的特征时，不慎使用了 `T0+1` 时刻的未来数据，这将导致模型在回测中表现虚假的优秀，必须避免。

## 常见特征类型与计算
以下是几种常见的特征类型，所有计算函数均在课程附件代码中提供。

### 1. 时间滞后特征
最简单的特征，即过去若干个时间点的价格本身。
*   **代码示例**：`time_lags(symbol, start_date, end_date, window=5)` 返回过去5个时间点的价格序列。

### 2. 价格变化率特征
计算当前时刻与之前时刻的价格相对变化。
*   **公式**：`(Price(T) - Price(T-1)) / Price(T-1)`

### 3. 基于滚动窗口的统计特征
计算过去一个时间窗口内的统计量，如最高价、最低价，并构造标准化特征。
*   **公式**：
    *   `X1 = (Price(T) - Low(T-λ, T)) / (High(T-λ, T) - Low(T-λ, T))`
    *   `X2 = (Price(T) - Price(T-1)) / (High(T-λ, T) - Low(T-λ, T))`
    *   其中 `λ` 是窗口大小，为超参数。

### 4. 技术指标特征
许多经典的技术分析指标可以作为特征。以下是指标及其核心公式/概念：
*   **顺势指标**：`CCI = (Typical Price - MA(Typical Price)) / (0.015 * Mean Deviation)`，用于判断超买超卖。
*   **移动平均线**：`MA(λ) = Average(Price over window λ)`，用于平滑趋势。
*   **变动率指标**：`ROC = (Price(T) - Price(T-n)) / Price(T-n) * 100%`。
*   **布林带**：中轨为 `MA(λ)`，上下轨为中轨 ± `k * Std(Price over window λ)`。
*   **强力指数**：`Force Index = (Close(T) - Close(T-1)) * Volume(T)`。

**建议**：建立你自己的特征计算工具库（如 `feature_utils.py`），并逐步实现更多技术指标。

## 特征到训练集的最终转换
通过上述方法，对于每个时间点 `T`，我们都能计算出一个 `D` 维的特征向量。将所有时间点的特征向量堆叠，就得到了特征矩阵 `X_train`（形状为 `N_samples × D`）。

对应的标签 `Y_train`（形状为 `N_samples × 1`）需要从原始数据中计算：
*   **回归问题**：`Y` 是下一时刻的价格或收益率（一个实数）。
*   **分类问题**：`Y` 可以是 `+1/-1`（涨/跌），或 `0/1`，或进行 one-hot 编码（如大涨、小涨、平盘、小跌、大跌）。

至此，我们得到了标准的机器学习训练集 `(X_train, y_train)` 和测试集 `(X_test, y_test)`。

## 核心预测模型介绍
有了训练集，我们就可以应用机器学习模型。以下介绍几种在量化领域常用且有效的模型。

### 1. 集成学习：随机森林与梯度提升树
集成学习通过组合多个弱学习器来获得一个强学习器，通常能稳定提升性能。

**随机森林**
*   **核心思想**：并行训练多棵决策树，每棵树在随机采样的数据子集和随机选择的特征子集上训练，最终通过投票或平均做出预测。
*   **优点**：抗过拟合能力强，训练树的数量越多通常效果越好（或至少不差），可解释性较好（通过特征重要性）。
*   **代码示例（sklearn）**：
    ```python
    from sklearn.ensemble import RandomForestClassifier
    model = RandomForestClassifier(n_estimators=100, max_depth=5, n_jobs=-1)
    model.fit(X_train, y_train)
    ```

**梯度提升树**
*   **代表库**：`XGBoost`
*   **核心思想**：串行训练多棵决策树，后一棵树学习前一棵树的残差，通过不断修正错误来提升模型。
*   **特点**：在许多数据集上表现优于随机森林，但理论保障稍弱，需要仔细调参。
*   **代码示例（xgboost）**：
    ```python
    import xgboost as xgb
    model = xgb.XGBClassifier(n_estimators=100, max_depth=3)
    model.fit(X_train, y_train)
    ```

### 2. 广义线性模型
这类模型简单有效，常作为性能基准。
*   **逻辑回归**：用于分类问题。`P(Y=1) = sigmoid(W^T * X)`，其中 `sigmoid(z) = 1 / (1 + exp(-z))`。
*   **LASSO/岭回归**：用于回归问题，在线性回归基础上加入正则化项防止过拟合。
*   **支持向量机**：用于分类或回归，寻找最大化间隔的超平面。

### 3. 神经网络
*   **多层感知机**：包含输入层、多个隐藏层和输出层。每层进行线性变换后接激活函数。
*   **核心概念**：
    *   **激活函数**：如 ReLU，引入非线性。
    *   **Dropout**：训练时随机“关闭”部分神经元，是一种高效的集成学习技巧，防止过拟合。
    *   **反向传播**：通过链式求导和动态规划思想，高效计算梯度以更新网络权重。
*   **代码示例（Keras）**：
    ```python
    from keras.models import Sequential
    from keras.layers import Dense, Dropout
    model = Sequential()
    model.add(Dense(512, activation='relu', input_dim=D))
    model.add(Dropout(0.2))
    model.add(Dense(256, activation='relu'))
    model.add(Dense(1, activation='sigmoid')) # 分类
    model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])
    model.fit(X_train, y_train, epochs=10, batch_size=128, validation_data=(X_test, y_test))
    ```

## 模型调优与验证
**切勿使用默认参数直接交易！** 必须对模型超参数进行调优。
*   **方法**：交叉验证。将训练集进一步划分为训练 fold 和验证 fold，在验证 fold 上评估不同参数组合的效果。
*   **高级技巧**：对于训练成本高的模型（如深度网络），可以使用贝叶斯优化等更智能的搜索方法替代暴力网格搜索。
