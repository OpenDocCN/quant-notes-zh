# 量化交易实战：02：股票基本知识与分类 📈

在本节课中，我们将要学习股票的基础概念、核心作用以及常见的分类方式。理解这些知识是进行金融量化分析的基石。

## 股票的定义与作用

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

为了更形象地解释，我们可以设想一个场景：一位创业者需要资金，而投资者看好他的项目。投资者将资金投入公司，公司则向投资者发行股票作为凭证。例如，一个公司初始市值为5亿，由五位出资人各出资1亿建立，那么每位出资人将获得公司20%的股票。

股票的核心作用主要有两点：
1.  **出资证明与股东身份**：持有股票意味着你是公司的所有者之一，拥有相应的股东权利，例如参与股东大会和投票。
2.  **获取收益**：作为股东，可以通过两种主要方式获利：
    *   **公司分红**：当公司盈利时，股东有权按持股比例获得利润分配。
    *   **交易获利**：股东可以在市场上将股票转让给他人，通过买卖价差获利。对于广大普通股民而言，交易获利是更常见的参与方式。

## 公司上市与IPO

上一节我们介绍了股票的基本概念，本节中我们来看看公司如何向公众发行股票，即“上市”。

所谓上市，就是企业通过证券交易所，首次公开向全社会投资者增发股票以募集资金的过程。公司不能随意向公众募集资金，必须满足严格的条件（如持续盈利能力、规范的财务制度等），并经过证监会审核批准后，才能在证券交易所挂牌交易。

公司寻求上市的主要原因是为了获得更广泛的融资渠道。相对于向少数富人融资，上市可以让公司面向全国乃至全球数以千万计的投资者，汇聚大众资金，实现快速扩张。

公司第一次公开发行股票并上市的行为，就称为 **IPO**。

## 股票的分类

了解了上市的概念后，我们来看看股票有哪些常见的分类方式。

### 按公司业绩分类

![](img/a8986a6c6e5c1f21b6a732f206c9b0fc_1.png)

以下是三种常见的按业绩划分的股票类型：

*   **蓝筹股**：指资本雄厚、信誉优良的巨型公司发行的股票。例如中国的“中石油”、“中石化”。它们通常规模大、经营稳定。
*   **绩优股**：指业绩优良公司的股票。这类公司可能规模不及蓝筹股，但盈利能力突出且增长稳定，例如“贵州茅台”。
*   **ST股**：中文名为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记，以警示投资者该公司存在较高风险。

### 按上市地区分类

股票根据其上市交易的地点和所用货币不同，主要有以下分类：

*   **A股**：在中国大陆（上海、深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。
*   **H股**：在中国香港上市的公司发行的股票。
*   **N股**：在美国纽约上市的公司发行的股票。
*   **S股**：在新加坡上市的公司发行的股票。

### 主要股市的交易规则差异

不同市场的交易规则各有特点，其中A股的两项核心规则对量化交易策略设计影响重大：

1.  **涨跌幅限制**：A股设有每日价格涨跌幅限制，通常为**10%**（部分股票为5%或20%）。这意味着股票每日的收盘价相对于前一日收盘价的涨跌幅度不能超过这个比例。公式可表示为：
    `当日最高价 <= 前日收盘价 * (1 + 涨跌幅限制比例)`
    `当日最低价 >= 前日收盘价 * (1 - 涨跌幅限制比例)`
    此规则旨在平抑市场波动，保护投资者。

2.  **T+1交割制度**：在A股市场，当日（T日）买入的股票，需要到下一个交易日（T+1日）才能卖出。与之对应的是“T+0”制度（如港股、美股），即当日买入可当日卖出。T+1制度降低了市场的日内投机频率。
