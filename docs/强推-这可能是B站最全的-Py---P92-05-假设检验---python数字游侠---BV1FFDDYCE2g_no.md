在本节课中，我们将要学习统计学中一个非常重要的概念——**假设检验**。我们将通过一个生动的案例，理解假设检验的基本思想、核心步骤以及如何应用它来判断一个关于总体的假设是否成立。

## 概述

假设检验是一种统计推断方法，用于根据样本数据对总体参数提出假设并进行判断。它与参数估计不同，参数估计是用样本数据去推测总体参数，而假设检验是在总体参数已知或假设已知的情况下，去验证这个假设是否与样本数据相符。

## 假设检验的基本原理

上一节我们介绍了参数估计，本节中我们来看看假设检验。假设检验的原理基于**小概率反证法**。小概率原理认为，在一次随机抽样中，小概率事件（通常指概率小于0.05的事件）几乎不可能发生。如果在一次抽样中，小概率事件发生了，我们就有理由怀疑原假设的正确性。

## 假设检验的步骤

以下是进行假设检验的一般流程：

1.  **提出假设**：首先，我们需要建立两个对立的假设。**原假设（H₀）** 通常是我们要检验或试图推翻的假设。**备择假设（H₁）** 是原假设的对立面，是我们希望证实的假设。
2.  **确定检验水准**：设定一个显著性水平，通常用 **α** 表示，最常用的是 α = 0.05。这代表我们愿意接受5%的犯错风险（即错误地拒绝原假设）。
3.  **选择检验统计量并计算**：根据数据类型和检验目的，选择合适的统计量（如Z值、T值）并计算其数值。
4.  **做出决策**：将计算得到的统计量或对应的P值与显著性水平α进行比较。如果P值小于α，我们就有足够的证据**拒绝原假设**，接受备择假设；反之，则**不能拒绝原假设**。

## 核心概念与分布

在进行假设检验时，不同的总体参数需要使用不同的统计分布。以下是常见的对应关系：

*   **均值检验**：
    *   大样本（n ≥ 30）或总体标准差σ已知时，使用 **Z分布（正态分布）**。
    *   小样本（n < 30）且总体标准差σ未知时，使用 **T分布**。
*   **比例检验**：单样本比例检验通常也使用 **Z分布**。
*   **方差检验**：检验单个总体方差时，使用 **卡方分布**。
*   **方差比检验**：比较两个总体方差时，使用 **F分布**。

这些分布意味着，在处理相应问题时，我们默认其统计量服从该分布，并套用对应的公式进行计算。

## 案例详解：薯片重量检验

### 问题背景

![](img/a8b1f8ba7bd01820aa9878b660b6da93_9.png)

一家超市接到投诉，称某品牌60克薯片包装内薯片太少。店方怀疑是运输导致碎屑过多，但顾客可能认为重量不足。已知该薯片标称总体均值为60克，总体标准差为3克。

### 检验步骤

![](img/a8b1f8ba7bd01820aa9878b660b6da93_11.png)

**1. 提出假设**

![](img/a8b1f8ba7bd01820aa9878b660b6da93_13.png)

我们的目的是检验这批薯片的平均重量是否真的是60克。
*   原假设 H₀: μ = 60克 （总体均值等于60克）
*   备择假设 H₁: μ ≠ 60克 （总体均值不等于60克）
由于备择假设是“不等于”，这是一个**双侧检验**。

**2. 确定检验水准**
设定显著性水平 α = 0.05。

**3. 收集数据并计算统计量**
我们从这批货中随机抽取了50袋薯片（n=50 > 30，属于大样本），计算样本平均重量为58克。
由于检验的是均值，且是大样本，我们使用Z检验。计算Z统计量的公式为：
`Z = (样本均值 - 总体均值) / (总体标准差 / √样本量)`
代入数据：
`Z = (58 - 60) / (3 / √50) ≈ -4.71`

**4. 做出决策**
在显著性水平α=0.05的双侧检验中，对应的临界Z值为±1.96。
我们计算出的Z值约为-4.71，其绝对值远大于1.96，落在了拒绝域内。
同时，我们也可以计算P值（Z=-4.71对应的双侧概率），该值远小于0.05。
因此，我们**拒绝原假设 H₀**。

![](img/a8b1f8ba7bd01820aa9878b660b6da93_15.png)

### 结论解释

拒绝原假设意味着，在一次随机抽样中，出现样本均值为58克（与60克差异如此之大）的情况是一个小概率事件。既然这个小概率事件发生了，我们就有理由认为，当前这批薯片所在的总体，其平均重量已经不再是60克，可能存在偷工减料的问题。虽然这个结论仍有约5%的可能是错误的（即第一类错误），但我们愿意承担这个风险。

## 单侧检验

![](img/a8b1f8ba7bd01820aa9878b660b6da93_21.png)

上一节我们介绍了双侧检验，本节中我们来看看单侧检验。单侧检验用于检验总体参数是否大于或小于某个特定值。

### 案例：灯泡寿命检验

合同规定灯泡寿命不低于1000小时。已知灯泡寿命服从正态分布，标准差为200小时。从一批货中抽取100个，测得样本均值为960小时。批发商是否应购买？

![](img/a8b1f8ba7bd01820aa9878b660b6da93_23.png)

**1. 提出假设**
我们只关心寿命是否低于标准。
*   原假设 H₀: μ ≥ 1000小时 （寿命符合或高于标准）
*   备择假设 H₁: μ < 1000小时 （寿命低于标准）
由于备择假设是“小于”，这是一个**左侧检验**。

**2. 确定检验水准**
α = 0.05。

**3. 计算统计量**
`Z = (960 - 1000) / (200 / √100) = -40 / 20 = -2`

![](img/a8b1f8ba7bd01820aa9878b660b6da93_25.png)

**4. 做出决策**
对于α=0.05的左侧检验，临界Z值为-1.645。
计算出的Z = -2 < -1.645，落在拒绝域内。
因此，我们**拒绝原假设 H₀**。

### 结论解释

![](img/a8b1f8ba7bd01820aa9878b660b6da93_27.png)

我们有足够的统计证据表明，这批灯泡的平均寿命低于1000小时的标准，因此批发商不应购买。

![](img/a8b1f8ba7bd01820aa9878b660b6da93_29.png)
