## 简单增长率累加的问题

上一节我们介绍了如何计算简单的增长率。本节中我们来看看，如果我们想计算一段时间内的“总”增长率，能否将每天的增长率简单相加。

![](img/02298efe0d9c375aa93ccce08f7d3b17_4.png)

例如，假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

以下是计算过程：
1.  第一到第二天的增长率：(50 - 100) / 100 = -0.5
2.  第二到第三天的增长率：(75 - 50) / 50 = 0.5

如果我们将这两个增长率相加：-0.5 + 0.5 = 0。结果为零，似乎表示股票“没涨没跌”。但观察原始价格，从100元变为75元，实际上是下跌的。这说明，**简单的增长率不能直接相加**来反映整体的连续变化。

![](img/02298efe0d9c375aa93ccce08f7d3b17_6.png)

## 引入对数收益率

![](img/02298efe0d9c375aa93ccce08f7d3b17_8.png)

那么，对于连续的时间序列，我们该如何进行正确的累加操作呢？我们需要借助一个数学工具：**对数**。

![](img/02298efe0d9c375aa93ccce08f7d3b17_10.png)

具体做法是，计算每日的**对数收益率**。公式为：
`log_return = log(今日价格 / 昨日价格)`

为什么这样做？因为对数的加法等价于其内部数值的乘法。让我们用之前的例子来说明：

![](img/02298efe0d9c375aa93ccce08f7d3b17_12.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_14.png)

`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/02298efe0d9c375aa93ccce08f7d3b17_16.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_18.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_20.png)

最终结果 `log(75/100)` 直接反映了**第三天价格与第一天价格的比值关系**。这意味着，将对数收益率累加起来，就等价于计算了从第一天到最后一天的整体变化倍数。

![](img/02298efe0d9c375aa93ccce08f7d3b17_22.png)

形象地说，如果你在第一天用1元钱投资，经过一系列涨跌，将这些每日的对数收益率累加，再通过指数函数还原，就能知道这1元钱在最后一天变成了多少钱。

![](img/02298efe0d9c375aa93ccce08f7d3b17_24.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_26.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_28.png)

## 使用Python进行计算与可视化

![](img/02298efe0d9c375aa93ccce08f7d3b17_30.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_32.png)

接下来，我们看看如何在Python中实现上述计算。

![](img/02298efe0d9c375aa93ccce08f7d3b17_34.png)

### 1. 计算对数收益率

![](img/02298efe0d9c375aa93ccce08f7d3b17_36.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_38.png)

首先，我们需要获取“昨日价格”。这可以通过Pandas的 `shift()` 函数实现，它将序列整体向上或向下移动。

![](img/02298efe0d9c375aa93ccce08f7d3b17_40.png)

```python
# 假设 data 是一个包含股价的Pandas Series
昨日价格 = data.shift(1) # 将数据向上移动一格，得到前一天的价格
对数收益率 = np.log(data / 昨日价格) # 计算每日对数收益率
```

![](img/02298efe0d9c375aa93ccce08f7d3b17_42.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_44.png)

### 2. 计算累积收益（1元投资的未来价值）

![](img/02298efe0d9c375aa93ccce08f7d3b17_46.png)

得到每日的对数收益率后，我们需要对其进行累加，以反映从起点开始的累积变化。这可以通过 `cumsum()`（累积求和）函数实现。

![](img/02298efe0d9c375aa93ccce08f7d3b17_48.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_50.png)

然后，由于我们之前取了对数，现在需要通过指数函数 `np.exp()` 将其还原为实际的价值倍数。

```python
# 计算累积对数收益率
累积对数收益率 = 对数收益率.cumsum()

![](img/02298efe0d9c375aa93ccce08f7d3b17_52.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_54.png)

# 还原为实际价值（假设初始投资为1单位）
累积价值 = np.exp(累积对数收益率)

![](img/02298efe0d9c375aa93ccce08f7d3b17_56.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_58.png)

# 绘制结果图
累积价值.plot(figsize=(16, 8))
```

执行以上代码后，我们将得到一张图表。图表中，每条曲线都从1开始，表示在起始时间（如2010年）投资了1元钱。曲线在后续时间点上的值，就代表了这1元钱在对应时刻的价值。例如，亚马逊（AMZN）的曲线在2018年达到了约10，意味着2010年投资在亚马逊的1元钱，到2018年价值约10元。

## 核心操作总结

![](img/02298efe0d9c375aa93ccce08f7d3b17_64.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_66.png)

本节课中我们一起学习了金融时间序列中几种核心的变化情况分析方法：

![](img/02298efe0d9c375aa93ccce08f7d3b17_68.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_70.png)

1.  **计算差异值**：使用 `data.diff()` 可以轻松得到相邻时间点的绝对差值。
2.  **计算简单增长率**：公式为 `(后值 - 前值) / 前值`。这种方法直观，但无法正确累加。
3.  **计算对数收益率**：公式为 `np.log(后值 / 前值)`。这是分析连续时间序列变化的标准方法，因为它具有可加性。
4.  **计算累积收益**：通过对数收益率的累积和 (`cumsum()`) 再取指数 (`np.exp()`)，可以计算出初始投资在整个时间序列中任意时刻的对应价值。

![](img/02298efe0d9c375aa93ccce08f7d3b17_72.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_74.png)

![](img/02298efe0d9c375aa93ccce08f7d3b17_76.png)

通过掌握这些操作，我们能够更准确、更有意义地分析金融资产价格的连续变化趋势。
