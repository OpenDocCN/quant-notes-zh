## 夏普比率的定义与意义
上一节我们介绍了投资中的风险概念，本节中我们来看看如何量化评估风险与收益的关系。

夏普比率描述的是：在承担单位风险的情况下，所能获得的超额收益是多少。它衡量了投资组合的风险调整后收益。指标越高，意味着对于每单位风险，投资者获得的补偿（收益）越高，该投资也就越“划算”。

为了便于理解，可以举一个生活中的例子：一份在战乱地区日薪数万的工作，其高薪对应的是极高的生命风险。夏普比率所做的，就是量化评估这种“高风险”是否换来了足够高的“高回报”。

![](img/9f4e38f801969e71b19fb9310c0043ce_1.png)

在选股或评估投资策略时，如果单从夏普比率这一指标看，我们倾向于选择夏普比率更高的选项，因为它意味着更高的风险调整后收益。

![](img/9f4e38f801969e71b19fb9310c0043ce_3.png)

## 夏普比率的计算公式
理解了夏普比率的意义后，我们来看看它的具体计算方法。

![](img/9f4e38f801969e71b19fb9310c0043ce_5.png)

计算夏普比率需要三个核心要素：
1.  投资组合的预期收益率
2.  无风险收益率（例如国债收益率）
3.  投资组合收益率的标准差（代表风险）

其核心思想是：计算投资组合收益率超出无风险收益率的部分（即超额收益），再除以投资组合的风险（波动率）。公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   `Rp` 代表投资组合的平均收益率。
*   `Rf` 代表无风险收益率。
*   `σp` 代表投资组合收益率的标准差。

举例说明：假设银行保本理财的固定年化收益为5%（可视为无风险收益），而某个投资组合的年化收益为15%。同时，该组合收益的波动（标准差）较大。夏普比率就是用来衡量，为了获得这额外的10%（15%-5%）收益，你所承担的额外风险是否值得。

## 使用Python计算夏普比率
掌握了理论公式后，我们将在Python中实践如何计算多只股票的夏普比率。

首先，我们需要准备好股票的历史收益率数据。在计算收益率时，通常需要对数据中的缺失值进行处理，例如使用前一天的收盘价进行填充。

![](img/9f4e38f801969e71b19fb9310c0043ce_7.png)

假设我们已经有了一个包含多只股票日收益率的DataFrame `returns`，并且设定年化无风险收益率 `risk_free_rate` 为0.05（5%）。年交易天数按250天估算。

以下是计算年化夏普比率的关键步骤代码：

![](img/9f4e38f801969e71b19fb9310c0043ce_9.png)

```python
import numpy as np
import pandas as pd

# 假设 returns 是一个DataFrame，每一列代表一只股票的日收益率序列
# 计算每只股票的平均日收益率和日收益率的标准差
mean_daily_returns = returns.mean()
std_daily_returns = returns.std()

# 设定年化无风险利率，并转换为日度
annual_risk_free_rate = 0.05
daily_risk_free_rate = annual_risk_free_rate / 250

# 计算超额收益（日度）
excess_daily_returns = mean_daily_returns - daily_risk_free_rate

# 计算年化夏普比率
# 年化超额收益 = 日度超额收益均值 * 年交易天数
# 年化波动率 = 日度收益率标准差 * sqrt(年交易天数)
sharpe_ratios = excess_daily_returns / std_daily_returns * np.sqrt(250)

![](img/9f4e38f801969e71b19fb9310c0043ce_11.png)

![](img/9f4e38f801969e71b19fb9310c0043ce_13.png)

# 查看结果
print(sharpe_ratios)
```

计算完成后，我们会得到一个夏普比率序列。正值表示投资获得了超越无风险收益的回报。其中，**夏普比率最高的股票，意味着其单位风险带来的超额收益最高**，从该指标看是最优选择。负值则意味着其收益尚不及无风险收益，通常不在考虑范围内。
