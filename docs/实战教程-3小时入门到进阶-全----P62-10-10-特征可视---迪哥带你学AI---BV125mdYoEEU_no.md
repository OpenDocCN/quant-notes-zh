## 特征构建

上一节我们介绍了数据的基本处理，本节中我们来看看如何从历史回报率序列中构建有效的特征。我们的目标是创建两个新特征：`lag_1`（前一日回报率）和`lag_2`（前两日回报率）。

![](img/5364ec09d61682a907cc852667711280_10.png)

核心操作是使用Pandas的`shift`函数，该函数可以将序列中的数据按指定步长向前或向后移动。

![](img/5364ec09d61682a907cc852667711280_12.png)

```python
# 假设我们有一个名为‘returns’的日回报率序列
# 创建 lag_1 特征：将‘returns’序列向下移动1行
data[‘lag_1‘] = data[‘returns‘].shift(1)
# 创建 lag_2 特征：将‘returns’序列向下移动2行
data[‘lag_2‘] = data[‘returns‘].shift(2)
```

![](img/5364ec09d61682a907cc852667711280_14.png)

![](img/5364ec09d61682a907cc852667711280_16.png)

为了更通用和整洁，我们可以将上述过程封装成一个函数。

![](img/5364ec09d61682a907cc852667711280_18.png)

![](img/5364ec09d61682a907cc852667711280_20.png)

以下是创建滞后特征的函数步骤：

![](img/5364ec09d61682a907cc852667711280_22.png)

1.  定义函数`create_lag_features`，接收原始数据框和要创建的滞后特征数量作为参数。
2.  在函数内部，循环创建指定数量的滞后特征列。
3.  使用`shift(i)`方法创建第`i`天的滞后特征。
4.  为生成的新列命名，例如`lag_1`，`lag_2`。
5.  调用函数后，删除因移动而产生的缺失值（NaN）。

![](img/5364ec09d61682a907cc852667711280_24.png)

![](img/5364ec09d61682a907cc852667711280_26.png)

```python
def create_lag_features(data, n_lags=2):
    """
    为数据框创建滞后特征。
    参数:
        data: 包含‘returns‘列的数据框。
        n_lags: 要创建的滞后特征数量。
    返回:
        添加了滞后特征列的数据框。
    """
    for i in range(1, n_lags + 1):
        col_name = f‘lag_{i}‘
        data[col_name] = data[‘returns‘].shift(i)
    # 删除因创建滞后特征而产生的缺失值行
    data = data.dropna()
    return data

![](img/5364ec09d61682a907cc852667711280_28.png)

![](img/5364ec09d61682a907cc852667711280_30.png)

# 使用函数创建特征
data = create_lag_features(data, n_lags=2)
```

![](img/5364ec09d61682a907cc852667711280_32.png)

![](img/5364ec09d61682a907cc852667711280_34.png)

![](img/5364ec09d61682a907cc852667711280_36.png)

执行上述代码后，数据框中将新增`lag_1`和`lag_2`两列，分别代表前一日和前两日的回报率。由于最开始的几天没有更早的历史数据，会产生缺失值，我们使用`dropna()`方法将其剔除。

## 特征可视化

![](img/5364ec09d61682a907cc852667711280_40.png)

特征构建完成后，直观地理解特征与目标变量（当日回报率`returns`）的关系至关重要。我们将通过散点图进行可视化，其中横轴和纵轴分别代表两个特征，而每个数据点的颜色则代表当日的实际回报率大小。

我们使用Matplotlib库的散点图功能，并引入颜色映射（Colormap）来用不同颜色表示数值大小。

![](img/5364ec09d61682a907cc852667711280_42.png)

![](img/5364ec09d61682a907cc852667711280_44.png)

以下是绘制特征可视化散点图的步骤：

1.  使用`data.plot.scatter()`方法绘制散点图。
2.  指定`x=‘lag_1‘`，`y=‘lag_2‘`，将两个特征分别映射到横纵坐标。
3.  指定`c=data[‘returns‘]`，将当日回报率作为颜色映射的依据。
4.  选择一个颜色映射方案（如`‘coolwarm‘`），暖色代表正收益，冷色代表负收益。
5.  添加`colorbar=True`参数显示颜色条，便于解读颜色对应的数值。
6.  添加参考线（如x=0和y=0的轴线），使图表更清晰。

![](img/5364ec09d61682a907cc852667711280_46.png)

```python
import matplotlib.pyplot as plt

# 绘制散点图
ax = data.plot.scatter(x=‘lag_1‘,
                       y=‘lag_2‘,
                       c=data[‘returns‘],        # 用颜色表示当日回报率
                       cmap=‘coolwarm‘,          # 颜色映射：冷色到暖色
                       figsize=(10, 6),
                       colorbar=True,            # 显示颜色条
                       alpha=0.6)

# 添加参考线
ax.axhline(y=0, color=‘red‘, linestyle=‘--‘, linewidth=0.5)  # 水平线 y=0
ax.axvline(x=0, color=‘red‘, linestyle=‘--‘, linewidth=0.5)  # 垂直线 x=0

![](img/5364ec09d61682a907cc852667711280_48.png)

# 设置图表标题和坐标轴标签
ax.set_title(‘特征可视化：lag_1 vs lag_2 (颜色表示当日回报率)‘)
ax.set_xlabel(‘前一日回报率 (lag_1)‘)
ax.set_ylabel(‘前两日回报率 (lag_2)‘)

![](img/5364ec09d61682a907cc852667711280_50.png)

![](img/5364ec09d61682a907cc852667711280_52.png)

plt.tight_layout()
plt.show()
```

![](img/5364ec09d61682a907cc852667711280_54.png)

生成的图表中，每个点代表一个交易日。点的位置由`lag_1`和`lag_2`决定，颜色则反映了该交易日的实际回报率（`returns`）。通过观察点的颜色分布，我们可以初步判断在特定的历史回报率组合下，次日更可能上涨（暖色点聚集）还是下跌（冷色点聚集）。这为后续构建预测模型提供了直观的洞察。
