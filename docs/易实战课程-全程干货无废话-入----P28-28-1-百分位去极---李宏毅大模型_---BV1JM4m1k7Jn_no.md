# Python金融分析与量化交易实战课程：P28：百分位去极值方法 📊

在本节课中，我们将学习如何对金融因子数据进行预处理。因子是影响投资决策的指标，例如市净率或营收增长率。处理这些数据是构建有效量化模型的关键步骤。我们将按照一个三步走的流程进行讲解：去极值、标准化和中性化。

## 数据预处理三步走 🚶‍♂️

上一节我们介绍了因子数据的基本概念。本节中，我们来看看处理因子数据的三个核心步骤。

以下是数据预处理的三个主要步骤：

1.  **去极值**：处理数据中的离群点或异常值。
2.  **标准化**：将不同量纲或取值范围的数据调整到同一尺度。
3.  **中性化**：在金融因子分析中，消除因子与某些基准（如市值）的相关性。

![](img/2e6cdd4113b45b00242fcd775dd626bd_5.png)

前两个步骤在机器学习和数据挖掘中很常见。第三个步骤“中性化”是金融因子分析中的特殊处理。接下来，我们将按照这个顺序，详细讲解每一步的具体做法。

## 第一步：去极值方法 🎯

![](img/2e6cdd4113b45b00242fcd775dd626bd_11.png)

![](img/2e6cdd4113b45b00242fcd775dd626bd_13.png)

![](img/2e6cdd4113b45b00242fcd775dd626bd_15.png)

去极值的目的是处理那些远离数据主体分布的异常值。直接删除这些数据点可能造成信息损失。一个更常见的做法是设定一个合理的边界，将超出边界的极值“拉回”到边界上。例如，如果设定上界为5，而一个数据点是10，我们就把这个点的值改为5。

![](img/2e6cdd4113b45b00242fcd775dd626bd_17.png)

以下是几种常见的去极值方法，我们先从分位数法开始。

![](img/2e6cdd4113b45b00242fcd775dd626bd_19.png)

### 分位数去极值法

分位数是理解数据分布位置的重要工具。中位数是大家熟悉的概念，它代表数据排序后处于中间位置的值。与均值相比，中位数对极值不敏感，更能反映数据的“典型”情况。

![](img/2e6cdd4113b45b00242fcd775dd626bd_21.png)

在分位数去极值法中，我们通常关注三个关键分位点：

*   **Q1（第一四分位数）**：代表数据中25%位置的值。
*   **Q2（中位数）**：代表数据中50%位置的值。
*   **Q3（第三四分位数）**：代表数据中75%位置的值。

**公式**：
对于一个数据集，先将其从小到大排序。中位数（Q2）的位置可以通过公式 `(n+1)/2` 计算（n为数据个数）。Q1和Q3则分别是数据下半部分和上半部分的中位数。

![](img/2e6cdd4113b45b00242fcd775dd626bd_23.png)

![](img/2e6cdd4113b45b00242fcd775dd626bd_25.png)

利用这些分位数，我们可以定义一个合理的数值范围，并将超出此范围的极值进行调整，从而在不丢弃数据的前提下，削弱异常值的影响。

![](img/2e6cdd4113b45b00242fcd775dd626bd_27.png)
