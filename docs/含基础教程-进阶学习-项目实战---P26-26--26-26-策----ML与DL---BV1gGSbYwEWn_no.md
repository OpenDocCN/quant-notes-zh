接下来，我们进入策略的核心交易部分，即`handle_bar`函数。这里需要根据每日最新的股票池，动态调整持仓。

![](img/93c5d0380a010f62dedf97802b3581a0_55.png)

![](img/93c5d0380a010f62dedf97802b3581a0_57.png)

以下是该函数的主要逻辑：
1.  **判断持仓状态**：首先检查当前账户是否有持仓。
2.  **卖出不在股票池的股票**：遍历当前持仓，如果某只股票不在当日最新的优选股票池中，则将其全部卖出。
3.  **买入股票池中的股票**：遍历当日最新的优选股票池，对池中的每只股票进行买入操作。为了简化，这里采用等权重分配资金的方式。

![](img/93c5d0380a010f62dedf97802b3581a0_59.png)

![](img/93c5d0380a010f62dedf97802b3581a0_61.png)

![](img/93c5d0380a010f62dedf97802b3581a0_63.png)

对应的关键交易代码如下：
```python
# 获取当前持仓
current_positions = context.portfolio.positions
# 卖出逻辑：如果持仓股票不在最新股票池中，则清仓
for stock in current_positions.keys():
    if stock not in context.stock_list:
        order_target_percent(stock, 0)
# 买入逻辑：对股票池中的股票进行等权重买入
for stock in context.stock_list:
    order_target_percent(stock, 1.0 / len(context.stock_list))
```

现在，让我们来解读回测报告中的几个核心指标，这些指标是衡量策略好坏的关键：
*   **累计收益率**：策略在整个回测期间的总收益。图中通常以曲线形式展示，并与基准（如沪深300指数）进行对比。
*   **年化收益率**：将累计收益率折算为每年的平均收益率，便于不同周期策略的比较。
*   **最大回撤**：策略净值从最高点到最低点的最大跌幅，用于衡量策略可能面临的最大亏损风险。**公式**可表示为：`最大回撤 = (波峰价值 - 波谷价值) / 波峰价值`。
*   **夏普比率**：衡量策略承担单位风险所获得的超额回报。**公式**为：`夏普比率 = (策略年化收益率 - 无风险利率) / 策略收益的年化标准差`。该值为正且越大越好，若为负则说明收益不如无风险投资。

![](img/93c5d0380a010f62dedf97802b3581a0_79.png)

![](img/93c5d0380a010f62dedf97802b3581a0_81.png)

在示例的回测结果中，策略的累计收益和年化收益可能为负，这表明在该回测时段内策略是亏损的。然而，通过对比“策略收益曲线”与“基准收益曲线”，可以评估策略是否跑赢了市场大盘（即是否少亏或抗跌）。同时，最大回撤和夏普比率提供了从风险和收益性价比角度的评估。
