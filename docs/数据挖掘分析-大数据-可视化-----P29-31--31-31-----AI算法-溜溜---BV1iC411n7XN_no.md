## 正态分布与异常值原理 📈

上一节我们介绍了分位数和中位数绝对偏差法，本节中我们来看看基于正态分布的3Sigma方法。它的原理基于一个常见的统计学概念：正态分布。

如果大家理解起来比较麻烦，你可以将其视为一个“正常”的分布。以前在讲解机器学习时经常举一个例子：你去银行贷款，银行多借给你三块五块，这挺正常；但如果多借或少借你十万八万，这就不正常了。

我们来看这样一个走势。这里我先画一条线，我们以标准正态分布为例。在标准正态分布中，均值是零。比如现在我要做一件事，我有一个“误差”——就像银行贷款，他可能多借也可能少借。多借就往右边点，少借就往左边点。

但是大家来看，对于我们现在的区间来说，比如我画第一个红色阴影区间，它的面积相对来说比较大，代表这些事发生的可能性比较高。当我把这个区间越往外放的时候，这个可能性发生的概率就越低了。

想一想，我们现在要干什么？是不是要做一个去极值的问题？那极值可能落到哪里？用红色画，落到均值附近可能吗？在均值附近、大概率发生的事，那就是正常的数据。极值一般落在哪里？落在比较偏远的地方。离均值越远、偏差越大的点，越有可能是离群点。

![](img/f925739fbe32d3bc664d1122d9ad67ef_2.png)

## 3Sigma方法的定义 🎯

什么是3Sigma方法？它是这样一件事。

你看这里，我们画的这个图中有 `X` 的均值，以及均值加减一倍、两倍、三倍标准差（σ）的区间。从这个区间当中我们能发现一件事：一旦你选的区间越大（例如加减三倍标准差），数据点落在这个区间之外的可能性就越小。

3Sigma方法的思想是：假设一份数据经过某种变换后，理论上服从高斯分布（正态分布）。我们先不管怎么变换，就假设它服从正态分布。那么，当我指定一个正负三倍标准差的区间时，这个区间包含了多少数据呢？这个值在统计学中已经给出：约 **99.7%**。

我们现在发现一件事：如果我认为数据中，最正常的99.7%的数据可以保留，剩下0.3%的数据当作极值进行处理规范，是不是可以？这就叫做3Sigma方法。它相当于通过均值和标准差来确定一个数据的正常范围下限和上限。

在正态分布中，你需要计算两个值：均值（μ）和标准差（σ）。这样就能把上限和下限全部计算出来。

![](img/f925739fbe32d3bc664d1122d9ad67ef_4.png)

**核心公式：**
*   **上限（Upper Bound）** = `μ + n * σ`
*   **下限（Lower Bound）** = `μ - n * σ`
其中，`n` 通常取3（即3Sigma），代表标准差的倍数。

![](img/f925739fbe32d3bc664d1122d9ad67ef_6.png)

## Python代码实现 💻

![](img/f925739fbe32d3bc664d1122d9ad67ef_8.png)

好了，这个叫做3Sigma方法。我们来写一下代码，其实也很简单。

![](img/f925739fbe32d3bc664d1122d9ad67ef_10.png)

以下是实现3Sigma去极值函数的步骤：

1.  **定义函数**：函数名可以沿用之前的风格，我们将其命名为 `three_sigma`。
    ```python
    def three_sigma(data, n=3):
    ```

![](img/f925739fbe32d3bc664d1122d9ad67ef_12.png)

2.  **计算均值和标准差**：对于传入的`data`序列（通常是一列数据），我们需要计算其均值（mean）和标准差（std）。
    ```python
        mean = data.mean()
        std = data.std()
    ```

![](img/f925739fbe32d3bc664d1122d9ad67ef_14.png)

3.  **计算上下限**：根据公式，用均值加上`n`倍标准差得到上限，减去`n`倍标准差得到下限。
    ```python
        upper = mean + n * std
        lower = mean - n * std
    ```

![](img/f925739fbe32d3bc664d1122d9ad67ef_16.png)

4.  **返回结果**：函数返回一个包含下限和上限的序列（Series）。
    ```python
        return pd.Series([lower, upper], index=['lower', 'upper'])
    ```

![](img/f925739fbe32d3bc664d1122d9ad67ef_18.png)

完整函数代码如下：
```python
def three_sigma(data, n=3):
    """
    使用3Sigma方法计算数据的正常值范围。
    参数:
        data: pandas Series，待处理的数据列。
        n: int，标准差的倍数，默认为3。
    返回:
        pd.Series，包含‘lower’（下限）和‘upper’（上限）。
    """
    mean = data.mean()  # 计算均值
    std = data.std()    # 计算标准差
    upper = mean + n * std  # 计算上限
    lower = mean - n * std  # 计算下限
    return pd.Series([lower, upper], index=['lower', 'upper'])
```
