## 通用函数概述

NumPy的通用函数可以对数组进行批量运算。除了基本的加减乘除，NumPy还提供了许多其他数学运算函数。

以下是常见的通用函数分类：

*   **一元函数**：接受一个数组作为输入。
*   **二元函数**：接受两个数组作为输入。

### 绝对值运算

![](img/1bb137372d2164374d4e02467a046706_5.png)

`abs`函数用于计算数组中每个元素的绝对值。

![](img/1bb137372d2164374d4e02467a046706_7.png)

```python
import numpy as np
A = np.array([-5, -3, 0, 3, 5])
result = np.abs(A)  # 或直接使用 abs(A)
# 结果: [5 3 0 3 5]
```

### 平方根运算

![](img/1bb137372d2164374d4e02467a046706_11.png)

`sqrt`函数用于计算数组中每个元素的平方根。注意，负数开方会得到特殊值`nan`。

```python
A = np.array([-5, -3, 0, 3, 5])
result = np.sqrt(A)
# 结果: [nan nan 0. 1.73205081 2.23606798]
```

### 取整函数

NumPy提供了多种取整方式，适用于不同的数学场景。

以下是四种主要的取整函数：

*   `np.trunc`: 向零取整。例如，`1.6`变为`1`，`-1.6`变为`-1`。
*   `np.floor`: 向下取整（向负无穷方向）。例如，`1.6`变为`1`，`-1.6`变为`-2`。
*   `np.ceil`: 向上取整（向正无穷方向）。例如，`1.6`变为`2`，`-1.6`变为`-1`。
*   `np.round`: 四舍五入。小数部分小于0.5向零取整，大于等于0.5向远离零的方向取整。

```python
A = np.array([-1.6, -1.0, 1.0, 1.6])
print(np.trunc(A))   # [-1. -1.  1.  1.]
print(np.floor(A))   # [-2. -1.  1.  1.]
print(np.ceil(A))    # [-1. -1.  1.  2.]
print(np.round(A))   # [-2. -1.  1.  2.]
```

### 分离整数与小数部分

`modf`函数可以将数组中每个元素的整数部分和小数部分分离，并分别返回两个数组。

```python
A = np.array([1.6, -1.6, 2.3])
integer_part, fractional_part = np.modf(A)
# integer_part: [ 0.6 -0.6  0.3]
# fractional_part: [ 1. -1.  2.]
```

### 特殊浮点数值的判断与处理

在数值计算中，会遇到两种特殊的浮点数值：`nan`和`inf`。

*   **`nan` (Not a Number)**: 表示“不是一个数字”，通常由无效的数学运算产生，如`0/0`、负数开方等。**关键特性：`nan`不等于任何值，包括它自身**。
*   **`inf` (Infinity)**: 表示“无穷大”，通常由一个非零数除以零产生。

判断这些特殊值需要使用专门的函数：

```python
A = np.array([0, 1, 2, 0])
B = np.array([1, 2, 0, 0])
C = A / B  # 可能出现 inf 和 nan
# C: [ 0.   0.5  inf  nan]

# 判断 nan
nan_mask = np.isnan(C)
# 结果: [False False False  True]

# 过滤掉 nan
C_without_nan = C[~np.isnan(C)]

# 判断 inf
inf_mask = np.isinf(C)
# 结果: [False False  True False]

# 过滤掉 inf (方法一)
C_without_inf_1 = C[C != np.inf]
# 过滤掉 inf (方法二)
C_without_inf_2 = C[~np.isinf(C)]
```

## 二元通用函数

二元函数对两个数组的对应元素进行操作。

### 最大值与最小值

`maximum`和`minimum`函数用于逐元素比较两个数组，返回对应位置的最大值或最小值。

```python
A = np.array([3, 4, 1, 7])
B = np.array([2, 5, 6, 3])

max_result = np.maximum(A, B)
# 结果: [3 5 6 7] (逐元素比较：max(3,2), max(4,5)...)

min_result = np.minimum(A, B)
# 结果: [2 4 1 3] (逐元素比较：min(3,2), min(4,5)...)
```
