# 量化投资：第1章：如何精确量化投资者情绪 📊

![](img/02fb6595daa8610209ba778241a816b3_0.png)

在本节课中，我们将学习如何将虚无缥缈的市场情绪转化为客观、可量化的指标。巴菲特的名言“别人贪婪我恐惧，别人恐惧我贪婪”以及A股的格言“买在无人问津处，卖在人声鼎沸时”，都强调了洞察市场情绪的重要性。我们将从多个角度出发，介绍七种量化市场情绪的方法，帮助你更科学地理解市场。

## 1.1：问卷调查法 📝

![](img/02fb6595daa8610209ba778241a816b3_2.png)

上一节我们提到了市场情绪的重要性，本节中我们来看看最直接的方法——问卷调查。

![](img/02fb6595daa8610209ba778241a816b3_4.png)

![](img/02fb6595daa8610209ba778241a816b3_6.png)

通过直接询问投资者对未来市场的看法，可以获取最直观的情绪数据。例如，在2019年针对上证指数是否会回补特定缺口的调查中，80%的受访者认为会跌破，而市场后续走势与这一预期一致。

![](img/02fb6595daa8610209ba778241a816b3_8.png)

![](img/02fb6595daa8610209ba778241a816b3_10.png)

然而，问卷调查方法存在明显缺陷：信息可靠性无法保证，过程费时费力，在现代量化分析中已不常被推荐使用。

## 1.2：搜索指数分析法 🔍

![](img/02fb6595daa8610209ba778241a816b3_12.png)

除了直接询问，我们还可以通过分析大众的搜索行为来洞察情绪。

![](img/02fb6595daa8610209ba778241a816b3_14.png)

![](img/02fb6595daa8610209ba778241a816b3_16.png)

当市场关注度上升时，相关关键词的搜索量会显著增加。大型搜索引擎如百度、谷歌都提供搜索指数工具。

以下是获取百度指数的示例步骤：
1.  访问百度指数官网。
2.  输入“上证指数”等关键词。
3.  观察搜索指数与市场K线走势的相关性，例如在2015年牛市中两者高度同步。

![](img/02fb6595daa8610209ba778241a816b3_18.png)

这种方法也有局限性，例如特定搜索引擎用户群体的变化可能影响数据的代表性。

![](img/02fb6595daa8610209ba778241a816b3_20.png)

## 1.3：舆情大数据分析（AI文本分析） 🤖

![](img/02fb6595daa8610209ba778241a816b3_22.png)

网络上的海量文本信息是分析市场情绪的宝贵数据源。

![](img/02fb6595daa8610209ba778241a816b3_24.png)

每天，财经媒体和社交平台会产生大量带有情绪倾向的文本（如文章标题、投资者评论）。我们可以利用人工智能技术对它们进行自动分类。

处理流程可以概括为：
1.  收集财经新闻、社交媒体文本。
2.  使用自然语言处理（NLP）模型分析文本情感。
3.  将文章自动分类为“看涨”、“看跌”或“中性”。
4.  统计每日各类观点的比例，形成情绪时间序列。

**代码示例（概念性）**：
```python
# 伪代码：使用情感分析模型对文本进行分类
sentiment_score = sentiment_analyzer.predict(text_content)
if sentiment_score > threshold_positive:
    label = "看涨"
elif sentiment_score < threshold_negative:
    label = "看跌"
else:
    label = "中性"
```
需要说明的是，尽管技术先进，但单纯依赖舆情大数据构建的交易策略，其有效性和稳定性在实战中表现参差不齐。

![](img/02fb6595daa8610209ba778241a816b3_26.png)

![](img/02fb6595daa8610209ba778241a816b3_28.png)

## 1.4：分析师评级汇总 📈

![](img/02fb6595daa8610209ba778241a816b3_30.png)

券商分析师的评级报告提供了另一种专业视角的情绪指标。

![](img/02fb6595daa8610209ba778241a816b3_32.png)

国内分析师会在研究报告中给出股票评级，如“买入”、“增持”、“中性”、“减持”、“卖出”。这些评级集中反映了专业机构对市场的看法。

![](img/02fb6595daa8610209ba778241a816b3_34.png)

![](img/02fb6595daa8610209ba778241a816b3_36.png)

我们可以从财经网站获取并汇总这些评级数据。以下是观察方法：
- **看整体市场情绪**：统计全市场“买入”评级占比的变化。占比上升通常意味着市场情绪转向积极。
- **看个股情绪变化**：关注特定股票的评级是否发生剧烈调整（如从“买入”下调至“减持”），这可能预示基本面或情绪的变化。

![](img/02fb6595daa8610209ba778241a816b3_38.png)

## 1.5：市场交易数据指标 💹

![](img/02fb6595daa8610209ba778241a816b3_40.png)

![](img/02fb6595daa8610209ba778241a816b3_42.png)

市场自身产生的数据是构建情绪指标最直接、最可靠的来源。

![](img/02fb6595daa8610209ba778241a816b3_43.png)

许多反映市场多空力量、资金动向的指标，其本质就是情绪指标。以下是一些核心例子：

![](img/02fb6595daa8610209ba778241a816b3_45.png)

**1. 看跌看涨比率（PCR）**
PCR指标衡量市场上看跌期权与看涨期权的交易量比值，直接反映了投资者的多空情绪对比。

![](img/02fb6595daa8610209ba778241a816b3_47.png)

**公式**：`PCR = 看跌期权交易量 / 看涨期权交易量`

![](img/02fb6595daa8610209ba778241a816b3_49.png)

![](img/02fb6595daa8610209ba778241a816b3_51.png)

**2. 北向资金流向**
代表香港及国际投资者对A股市场的情绪。资金净流入表示看好，净流出表示看空。相关数据可在港交所官网盘后获取。

![](img/02fb6595daa8610209ba778241a816b3_53.png)

**3. 市场资金流数据**
交易软件中显示的主力资金、散户资金流入流出情况，体现了不同交易群体对市场的看法。

![](img/02fb6595daa8610209ba778241a816b3_55.png)

**4. 龙虎榜数据**
披露机构席位和大额资金的买卖动向，可以洞察“聪明钱”的情绪。

![](img/02fb6595daa8610209ba778241a816b3_57.png)

## 1.6：投资者行为数据 👥

![](img/02fb6595daa8610209ba778241a816b3_59.png)

投资者的开户和杠杆使用行为是情绪的重要外在表现。

![](img/02fb6595daa8610209ba778241a816b3_61.png)

**1. 新开户投资者数量**
市场情绪高涨时，新开户人数会激增。该数据可在“中国证券登记结算有限责任公司”官网查询。新股民增速的变化往往领先于股价指数，但增速过快也可能是市场过热的警示信号。

![](img/02fb6595daa8610209ba778241a816b3_63.png)

**2. 融资融券（两融）余额**
“融资”是投资者借钱买入（做多），“融券”是投资者借券卖出（做空）。两融余额的总量直接反映了市场中杠杆交易的热度。

![](img/02fb6595daa8610209ba778241a816b3_65.png)

**公式**：`融资余额` 增加，表明市场看多情绪浓厚；`融券余额` 增加，则表明看空情绪升温。

![](img/02fb6595daa8610209ba778241a816b3_67.png)

![](img/02fb6595daa8610209ba778241a816b3_69.png)

## 1.7：衍生品价格信号 📉

![](img/02fb6595daa8610209ba778241a816b3_71.png)

股指期货的价格揭示了市场对未来的预期。

![](img/02fb6595daa8610209ba778241a816b3_73.png)

**升水与贴水**：
- **期货价格 > 现货指数价格**：称为“升水”，表示市场预期未来上涨，情绪偏乐观。
- **期货价格 < 现货指数价格**：称为“贴水”，表示市场预期未来下跌，情绪偏悲观。

![](img/02fb6595daa8610209ba778241a816b3_75.png)

![](img/02fb6595daa8610209ba778241a816b3_77.png)

**贴水率计算公式**：
`贴水率 = (期货价格 - 现货指数价格) / 现货指数价格`

![](img/02fb6595daa8610209ba778241a816b3_79.png)

通过实时计算股指期货的升贴水率，可以量化市场对未来的情绪。长期贴水通常意味着市场信心不足。

![](img/02fb6595daa8610209ba778241a816b3_81.png)

![](img/02fb6595daa8610209ba778241a816b3_83.png)

## 更多情绪指标示例

![](img/02fb6595daa8610209ba778241a816b3_85.png)

除了上述七种核心方法，还有许多其他指标可用于刻画情绪：
- **成交量与换手率**：短期内急剧放大通常伴随情绪升温。
- **公募基金仓位**：仓位越高，说明机构对后市越看好。
- **基金净赎回率**：赎回率上升，反映基民（散户）情绪悲观。
- **大股东/高管增减持**：增持行为代表内部人士看好，减持则代表看空。

![](img/02fb6595daa8610209ba778241a816b3_87.png)

## 总结

![](img/02fb6595daa8610209ba778241a816b3_89.png)

本节课中我们一起学习了量化市场情绪的多种方法。我们从最主观的问卷调查，谈到利用搜索指数和AI分析舆情大数据，再到聚焦分析师评级、市场交易数据（如PCR、北向资金）、投资者行为（开户数、两融余额）以及衍生品信号（股指期货升贴水）。这些方法各有优劣，但共同目标是将难以捉摸的“市场情绪”转化为可定义、可计算、可回溯的客观指标，为理性的量化投资决策提供依据。记住，市场情绪是重要的辅助工具，但并非唯一的决策依据，需要结合其他分析方法综合使用。
