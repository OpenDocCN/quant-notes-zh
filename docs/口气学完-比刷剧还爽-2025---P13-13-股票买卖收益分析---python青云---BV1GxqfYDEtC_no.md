## 需求分析

![](img/77f6032da2ae43e0f8297e8d0a3471ae_17.png)

上一节我们介绍了数据的基本处理，本节中我们来看看如何将业务需求转化为具体的计算步骤。

首先，我们需要明确策略的规则和时间范围。

以下是需求的核心要点：
*   **时间范围**：从2010年1月1日到2020年2月18日（案例制作时的“昨天”）。
*   **买入规则**：每月的第一个交易日，买入一手股票。
*   **卖出规则**：每年的最后一个交易日，卖出持有的所有股票。
*   **交易单位**：一手股票等于100股。
*   **交易价格**：使用开盘价作为买卖股票的单价。
*   **收益计算**：总收益 = 卖出股票总收入 + 手中剩余股票估值 - 买入股票总成本。

![](img/77f6032da2ae43e0f8297e8d0a3471ae_19.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_21.png)

基于以上规则，我们可以进行更详细的分析：
1.  **一个完整年份的买卖数量**：一年有12个月，每月买入一手（100股），则全年买入 `12 * 100 = 1200` 股。在年末最后一个交易日，需要将这1200股全部卖出。
2.  **非完整年份的处理**：案例结束于2020年2月，因此2020年只买入了股票（1月和2月），但还未到年末，所以这部分股票尚未卖出，需要单独计算其价值。

![](img/77f6032da2ae43e0f8297e8d0a3471ae_25.png)

# 假设原始数据框名为 df，且索引已转换为DatetimeIndex
new_df = df[‘2010-01-01‘:‘2020-02-18‘]

## 计算买入股票的总成本

![](img/77f6032da2ae43e0f8297e8d0a3471ae_47.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_49.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_51.png)

接下来，我们计算投资者买入股票所花费的总金额。这需要找到每月第一个交易日的开盘价。

![](img/77f6032da2ae43e0f8297e8d0a3471ae_53.png)

以下是实现步骤：
1.  使用`resample(‘M‘)`方法按月份对`new_df`进行分组。
2.  使用`.first()`方法获取每个月的第一行数据，即每月第一个交易日的数据。
3.  从这些数据中提取`open`（开盘价）列。
4.  将每个开盘价乘以100（一手股票），然后求和，得到总成本。

![](img/77f6032da2ae43e0f8297e8d0a3471ae_55.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_57.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_59.png)

```python
# 获取每月第一个交易日的数据
df_monthly = new_df.resample(‘M‘).first()

![](img/77f6032da2ae43e0f8297e8d0a3471ae_61.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_63.png)

# 计算买入股票的总成本：每月开盘价 * 100股，然后求和
cost = df_monthly[‘open‘].sum() * 100
print(f“买入股票总成本：{cost}“)
```
代码`new_df.resample(‘M‘).first()`是关键，它帮助我们高效地提取了每月所需的特定数据点。

## 计算卖出股票的总收入

![](img/77f6032da2ae43e0f8297e8d0a3471ae_69.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_71.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_73.png)

现在，我们来计算卖出股票获得的收入。这需要找到每年最后一个交易日的开盘价。

![](img/77f6032da2ae43e0f8297e8d0a3471ae_75.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_77.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_79.png)

以下是实现步骤：
1.  使用`resample(‘A‘)`方法按年份对`new_df`进行分组（‘A‘代表年末）。
2.  使用`.last()`方法获取每年的最后一行数据，即每年最后一个交易日的数据。
3.  由于2020年未结束，其“最后一行”实际上是数据截止日（2月18日），我们需要将其剔除。
4.  从剩余数据中提取`open`列，将每个开盘价乘以1200（一年买入的总股数），然后求和，得到已卖出股票的总收入。

![](img/77f6032da2ae43e0f8297e8d0a3471ae_81.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_83.png)

```python
# 获取每年最后一个交易日的数据
df_yearly = new_df.resample(‘A‘).last()

![](img/77f6032da2ae43e0f8297e8d0a3471ae_85.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_87.png)

# 剔除2020年的数据（因为2020年股票未卖出）
df_yearly = df_yearly[:-1]

![](img/77f6032da2ae43e0f8297e8d0a3471ae_89.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_91.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_93.png)

# 计算卖出股票的总收入：每年末开盘价 * 1200股，然后求和
receive = df_yearly[‘open‘].sum() * 1200
print(f“卖出股票总收入：{receive}“)
```

## 计算剩余股票的价值及总收益

![](img/77f6032da2ae43e0f8297e8d0a3471ae_101.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_103.png)

最后，我们需要计算2020年买入但未卖出的股票价值，并最终计算总收益。

![](img/77f6032da2ae43e0f8297e8d0a3471ae_105.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_107.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_109.png)

以下是实现步骤：
1.  **计算剩余股数**：2020年买入了1月和2月两手的股票，共 `2 * 100 = 200` 股。
2.  **确定估值单价**：使用数据截止日（2020年2月18日）的收盘价作为剩余股票的单价。
3.  **计算剩余价值**：剩余股数乘以收盘价。
4.  **计算总收益**：应用公式 **总收益 = 卖出收入 + 剩余价值 - 买入成本**。

![](img/77f6032da2ae43e0f8297e8d0a3471ae_111.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_113.png)

```python
# 获取最后一天的收盘价，作为剩余股票的单价
last_price = new_df[‘close‘].iloc[-1]

![](img/77f6032da2ae43e0f8297e8d0a3471ae_115.png)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_117.png)

# 计算剩余股票的价值 (2020年买入了2手，共200股)
last_money = 200 * last_price
print(f“手中剩余股票估值：{last_money}“)

![](img/77f6032da2ae43e0f8297e8d0a3471ae_119.png)

# 计算总收益
total_profit = receive + last_money - cost
print(f“此策略总收益为：{total_profit}“)
```
