### 绝对值运算

标准库的 `abs` 函数可以取绝对值。在NumPy中，可以使用 `np.abs` 对数组的每个元素取绝对值。

![](img/3cd5a0856190f48dd1ef1f999da7c45b_9.png)

![](img/3cd5a0856190f48dd1ef1f999da7c45b_11.png)

```python
import numpy as np
A = np.array([-5, -3, 0, 3, 5])
result = np.abs(A)  # 结果为 [5 3 0 3 5]
```

### 平方根运算

使用 `np.sqrt` 函数可以对数组中的每个元素进行开平方运算。需要注意的是，负数开方会得到 `nan`（非数字）值。

```python
A = np.array([-5, -3, 0, 3, 5])
result = np.sqrt(A)  # 结果为 [nan nan 0. 1.73205081 2.23606798]
```

### 取整函数

Python提供了多种取整方式，NumPy也提供了对应的数组版本。

以下是四种主要的取整方式：

*   **向零取整**：`np.trunc` 或 `np.fix`。直接舍弃小数部分。
    *   例如：`np.trunc([1.6, -1.6])` 结果为 `[ 1., -1.]`
*   **向下取整**：`np.floor`。取不大于原数的最大整数。
    *   例如：`np.floor([1.6, -1.6])` 结果为 `[ 1., -2.]`
*   **向上取整**：`np.ceil`。取不小于原数的最小整数。
    *   例如：`np.ceil([1.6, -1.6])` 结果为 `[ 2., -1.]`
*   **四舍五入**：`np.round` 或 `np.rint`。遵循“四舍六入五成双”的规则（银行家舍入法）。
    *   例如：`np.round([1.6, -1.6, 1.5, 2.5])` 结果可能为 `[ 2., -2., 2., 2.]`

### 分离整数与小数部分

`np.modf` 函数可以将数组中每个元素的整数部分和小数部分分开，并返回两个独立的数组。

```python
A = np.array([1.6, -1.6, 3.0])
integer_part, fractional_part = np.modf(A)
# integer_part 为 [ 0.6 -0.6  0. ]
# fractional_part 为 [ 1. -1.  3.]
```

### 特殊值：nan与inf

在数值计算中，会遇到两种特殊的浮点数值：

*   **nan**：`Not a Number`，表示一个无意义的或未定义的结果（如 `0/0` 或负数的平方根）。
*   **inf**：`Infinity`，表示无穷大（如 `5/0`）。

`nan` 有一个特殊性质：它不等于任何值，甚至不等于它自己。因此，判断数组中是否存在 `nan` 不能使用 `==`，而必须使用 `np.isnan` 函数。判断 `inf` 可以使用 `==` 或 `np.isinf` 函数。

```python
A = np.array([0, 1, 2, 3])
B = A / A  # 结果为 [nan  1.  1.  1.]
print(np.isnan(B))  # 结果为 [ True False False False]

C = np.array([5, 6])
D = np.array([0, 2])
E = C / D  # 结果为 [inf  3.]
print(np.isinf(E))  # 结果为 [ True False]
```

## 二元通用函数

二元函数接收两个数组作为输入，并对它们的对应元素进行运算。

### 最大值与最小值

`np.maximum` 和 `np.minimum` 函数用于逐元素比较两个数组，返回每个位置上的最大值或最小值。

```python
A = np.array([3, 4, 5])
B = np.array([2, 5, 1])
max_result = np.maximum(A, B)  # 结果为 [3 5 5]
min_result = np.minimum(A, B)  # 结果为 [2 4 1]
```
