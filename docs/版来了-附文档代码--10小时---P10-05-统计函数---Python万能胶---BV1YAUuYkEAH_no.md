## 替换函数补充：`replace`

![](img/df4c6d8199d99ce0e8ede6de168286f3_3.png)

在讲解统计函数之前，我们先补充一个之前提到的文本替换函数 `replace`。`replace` 函数用于替换文本中的指定内容。

以下是 `replace` 函数的基本语法和操作步骤：

1.  **原始文本**：需要进行替换操作的字符串。
2.  **起始位置**：指定从原始文本的哪个位置开始替换。
3.  **替换长度**：指定要替换多少个字符。
4.  **新内容**：指定替换后的新字符串。

![](img/df4c6d8199d99ce0e8ede6de168286f3_5.png)

例如，假设我们有一个手机号码 `“13812345678”`，我们希望隐藏中间四位（第4位到第7位），可以这样操作：

```python
# 原始手机号码
phone_number = “13812345678”
# 从第4位开始（索引为3），替换4个字符为“****”
hidden_number = phone_number.replace(phone_number[3:7], “****”)
print(hidden_number)  # 输出：138****5678
```

![](img/df4c6d8199d99ce0e8ede6de168286f3_7.png)

![](img/df4c6d8199d99ce0e8ede6de168286f3_9.png)

**注意**：在实际的 `replace` 函数调用中，通常直接指定要被替换的旧字符串和新字符串，例如 `str.replace(old, new)`。上述示例演示了按位置替换的思路。

## 统计函数（一）：基础数学运算

接下来，我们进入统计函数的学习。首先介绍几个基础的数学运算函数。

![](img/df4c6d8199d99ce0e8ede6de168286f3_12.png)

### 向下取整函数：`int`

![](img/df4c6d8199d99ce0e8ede6de168286f3_14.png)

`int()` 函数用于将一个数字向下舍入取整。对于正数，它直接舍弃小数部分；对于负数，它向数值更小的方向取整。

**公式**：`int(number)`

例如：
*   `int(5.7)` 返回 `5`。
*   `int(-7.1)` 返回 `-8`（因为-8比-7.1更小）。

### 四舍五入函数：`round`

`round()` 函数用于对数字进行四舍五入，可以指定保留的小数位数。

![](img/df4c6d8199d99ce0e8ede6de168286f3_16.png)

**公式**：`round(number, ndigits)`
*   `number`：要四舍五入的数字。
*   `ndigits`：保留的小数位数。

例如：
*   `round(3.14159, 2)` 返回 `3.14`。
*   `round(3.14159, 0)` 返回 `3.0`。

### 取余函数：`mod`

![](img/df4c6d8199d99ce0e8ede6de168286f3_18.png)

`mod()` 函数用于计算两数相除的余数。

**公式**：`mod(dividend, divisor)`
*   `dividend`：被除数。
*   `divisor`：除数。

例如：
*   `mod(5, 3)` 返回 `2`（因为5除以3，商1余2）。

## 统计函数（二）：数据聚合计算

除了基础运算，统计函数还包含一系列用于数据聚合计算的函数，它们对一组数据进行分析。

以下是常用的数据聚合函数及其说明：

*   **最大值**：`max(data_range)` - 返回数据区域中的最大值。
*   **最小值**：`min(data_range)` - 返回数据区域中的最小值。
*   **平均值**：`average(data_range)` - 返回数据区域的算术平均值。
*   **求和**：`sum(data_range)` - 返回数据区域中所有数值的总和。
*   **计数**：`count(data_range)` - 返回数据区域中数值项的个数。

![](img/df4c6d8199d99ce0e8ede6de168286f3_20.png)

这些函数的参数通常是一个数据区域（例如一个列表或数组）。例如，计算列表 `[44, 68, 55, 60]` 的各项统计值：
*   `max([44, 68, 55, 60])` 返回 `68`。
*   `min([44, 68, 55, 60])` 返回 `44`。
*   `average([44, 68, 55, 60])` 返回 `56.75`。
*   `sum([44, 68, 55, 60])` 返回 `227`。
*   `count([44, 68, 55, 60])` 返回 `4`。
