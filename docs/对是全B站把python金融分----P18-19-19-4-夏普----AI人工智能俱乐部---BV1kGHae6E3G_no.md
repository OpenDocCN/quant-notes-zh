# Python金融量化与股票交易：P18：19.19.4-夏普比率的作用 📈

在本节课中，我们将要学习夏普比率这一核心概念。夏普比率是衡量投资绩效的重要指标，它描述了在承担单位风险的情况下，我们能获得多少超额收益。理解并计算夏普比率，对于评估和选择投资组合至关重要。

上一节我们介绍了投资回报率与风险的概念，本节中我们来看看如何用一个综合指标来评估“性价比”。

## 夏普比率的含义

夏普比率描述的是风险与收益之间的关系。它回答了一个核心问题：对于我承担的每一单位风险，我能获得多少额外的回报？

我们可以用一个生活中的例子来理解：假设一份工作的日薪非常高，但工作地点在战乱地区，风险极大。高薪是对高风险的一种补偿。在投资中也是如此，高风险通常伴随着高收益预期。夏普比率就是用来衡量这种“风险补偿”是否划算的指标。

![](img/2379257567e6199a7dc980d3d01af2eb_1.png)

![](img/2379257567e6199a7dc980d3d01af2eb_2.png)

**夏普比率越高，意味着在承担相同风险的情况下，获得的超额收益越高，该投资组合的“性价比”就越好。** 因此，当我们比较多个股票或投资组合时，在其他条件相似的情况下，通常会倾向于选择夏普比率更高的那个。

## 夏普比率的计算公式

理解了夏普比率的含义后，我们来看看如何计算它。计算需要三个核心要素：投资组合的收益率、无风险收益率以及投资组合的风险（波动率）。

![](img/2379257567e6199a7dc980d3d01af2eb_4.png)

其标准计算公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率。
*   **Rf**：无风险收益率（例如国债收益率、银行定期存款利率）。
*   **σp**：投资组合收益率的标准差，代表风险。

这个公式的逻辑是：先计算投资组合超越无风险收益的超额收益（`Rp - Rf`），再将其除以组合的风险（`σp`）。结果就是每承担一单位风险所获得的超额收益。

在实际计算年化夏普比率时，如果我们的收益率数据是日度的，需要进行年化处理。以下是计算年化夏普比率的Python代码步骤：

以下是计算年化夏普比率的关键步骤代码：

```python
import numpy as np
import pandas as pd

# 假设 `returns` 是一个Pandas Series，包含了投资组合的日收益率序列
# 假设无风险年化收益率 `risk_free_rate` 为 5%
risk_free_rate = 0.05

![](img/2379257567e6199a7dc980d3d01af2eb_6.png)

# 1. 计算超额收益（年化）
excess_returns = returns - risk_free_rate / 250  # 假设一年有250个交易日

# 2. 计算夏普比率（年化）
# 先计算超额收益的均值
mean_excess_return = excess_returns.mean()
# 再计算收益率的标准差
std_return = returns.std()
# 最后计算年化夏普比率
sharpe_ratio = (mean_excess_return * 250) / (std_return * np.sqrt(250))
```

![](img/2379257567e6199a7dc980d3d01af2eb_8.png)

**代码解释**：
1.  `excess_returns = returns - risk_free_rate / 250`：计算每日的超额收益。将年化无风险利率除以250（近似年交易日数），得到日度无风险利率，再从每日收益中减去它。
2.  `mean_excess_return = excess_returns.mean()`：计算超额收益的日度平均值。
3.  `sharpe_ratio = (mean_excess_return * 250) / (std_return * np.sqrt(250))`：计算年化夏普比率。
    *   `mean_excess_return * 250`：将日度平均超额收益年化。
    *   `std_return * np.sqrt(250)`：将日收益波动率（标准差）年化。这是根据“波动率随时间平方根增长”的统计特性。
    *   两者相除即得到年化夏普比率。

## 实例应用与结果解读

当我们对多个股票或策略计算夏普比率后，就可以进行比较。

假设我们计算了A、B、C三只股票的夏普比率，结果分别为：**股票A: 1.2， 股票B: 0.8， 股票C: -0.3**。

以下是基于夏普比率的决策分析：

*   **股票A (夏普比率 1.2)**：单位风险带来的超额收益最高，是三者中风险调整后收益最好的选择。
*   **股票B (夏普比率 0.8)**：风险调整后收益尚可，但不如股票A。
*   **股票C (夏普比率 -0.3)**：其收益甚至无法覆盖无风险收益，承担风险反而带来了负的超额回报，通常应避免。

![](img/2379257567e6199a7dc980d3d01af2eb_10.png)

因此，**夏普比率越大越好**。它为投资者提供了一个统一的标准，来比较不同风险水平下的投资表现。
