## 计算每日差异值

在金融数据分析中，我们不仅关心当天的绝对数值（如股价），更关心其相对于前一天的变化量。这个变化量被称为**差异值**。

![](img/3cba70010d4131058b7def4ab85de7c2_8.png)

计算差异值的逻辑是：用后一天的数据减去前一天的数据。在Pandas中，这可以通过一个内置函数轻松实现。

![](img/3cba70010d4131058b7def4ab85de7c2_10.png)

以下是计算每日差异值的代码：

```python
# 假设 `data` 是一个包含金融时间序列的DataFrame
diff_data = data.diff()
print(diff_data.head())
```

![](img/3cba70010d4131058b7def4ab85de7c2_12.png)

执行上述代码后，你会得到一个新的DataFrame。例如，1月5日的数据将是1月5日的原始值减去1月4日的原始值。由于1月4日之前没有数据，所以1月4日的差异值会显示为`NaN`（空值）。

![](img/3cba70010d4131058b7def4ab85de7c2_14.png)

这个指标在时间序列分析中非常常用。`diff()`函数还可以通过参数指定计算间隔，例如计算两天的差异（`periods=2`）或一周的差异。

## 计算增长率（变化率）

差异值是一个绝对变化量，但有时我们更关心相对变化。例如，股价从1元涨到2元（涨幅100%）和从10000元涨到10001元（涨幅0.01%）的意义完全不同。因此，我们需要计算**增长率**。

![](img/3cba70010d4131058b7def4ab85de7c2_16.png)

![](img/3cba70010d4131058b7def4ab85de7c2_18.png)

增长率的计算公式为：
**增长率 = (后一天的值 - 前一天的值) / 前一天的值**

![](img/3cba70010d4131058b7def4ab85de7c2_20.png)

同样，Pandas提供了现成的函数来计算这个指标。

以下是计算每日增长率的代码：

![](img/3cba70010d4131058b7def4ab85de7c2_22.png)

```python
# 计算增长率
pct_change_data = data.pct_change()
# 设置显示格式，保留三位小数
pd.set_option('display.float_format', lambda x: '%.3f' % x)
print(pct_change_data.head())
```

执行后，你会看到数据变成了百分比形式，表示每日相对于前一天的涨跌幅度。金融资产的日增长率通常数值较小。

# 计算各列的平均增长率
