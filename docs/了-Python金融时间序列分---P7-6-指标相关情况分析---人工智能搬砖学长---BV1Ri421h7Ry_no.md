为了直观地观察两个指标随时间变化的趋势，最直接的方法是绘制折线图。但由于两个指标的数值范围可能差异很大，将它们画在同一坐标系下会难以观察。因此，我们使用双Y轴的方法，将它们放在同一张图中但使用不同的坐标轴。

![](img/0eadc3471017522c8adba6012d574829_45.png)

![](img/0eadc3471017522c8adba6012d574829_49.png)

2.  然后，我们改进方法，使用`secondary_y`参数创建第二个Y轴。
3.  为了更清晰地观察短期趋势，我们可以只选取一段时间（例如2010年至2012年）的数据进行绘图。

## 第三步：绘制散点图矩阵进行深入分析

![](img/0eadc3471017522c8adba6012d574829_73.png)

折线图展示了时间序列上的趋势，而散点图能更直接地展示两个变量之间的分布关系。Pandas提供了`scatter_matrix`函数，可以方便地绘制多个变量两两之间的散点图，并在对角线位置展示每个变量自身的分布（直方图或核密度估计图）。

以下是绘制散点图矩阵的步骤：

![](img/0eadc3471017522c8adba6012d574829_75.png)

![](img/0eadc3471017522c8adba6012d574829_77.png)

1.  首先，我们需要计算指标的日收益率（或对数收益率），以分析变化率之间的关系。公式为：`收益率 = log(当日价格 / 前一日价格)`。
2.  然后，使用`pandas.plotting.scatter_matrix`函数绘图。

```python
import numpy as np
import pandas as pd
from pandas.plotting import scatter_matrix

# 计算对数收益率
returns = np.log(data / data.shift(1))
returns = returns.dropna() # 删除第一个NaN值

![](img/0eadc3471017522c8adba6012d574829_79.png)

![](img/0eadc3471017522c8adba6012d574829_81.png)

![](img/0eadc3471017522c8adba6012d574829_83.png)

# 绘制散点图矩阵，对角线显示直方图
scatter_matrix(returns, alpha=0.5, figsize=(12, 12), diagonal='hist', hist_kwds={'bins': 50})
```

![](img/0eadc3471017522c8adba6012d574829_85.png)

![](img/0eadc3471017522c8adba6012d574829_87.png)

我们也可以将对角线的图形从直方图改为更平滑的核密度估计（KDE）图：

![](img/0eadc3471017522c8adba6012d574829_89.png)

![](img/0eadc3471017522c8adba6012d574829_91.png)

```python
# 绘制散点图矩阵，对角线显示核密度估计图
scatter_matrix(returns, alpha=0.5, figsize=(12, 12), diagonal='kde')
```

![](img/0eadc3471017522c8adba6012d574829_93.png)

![](img/0eadc3471017522c8adba6012d574829_95.png)

![](img/0eadc3471017522c8adba6012d574829_97.png)

在生成的散点图矩阵中：
*   **对角线**：展示了单个变量（如`SPX收益率`）的分布情况。直方图显示了数据在不同区间的频数，而KDE图提供了更平滑的概率密度曲线。
*   **非对角线**：展示了两个变量（如`SPX收益率`和`VX收益率`）的散点图。从图中点的分布趋势可以清晰地看出它们呈负相关关系。

## 第四步：进行回归分析（思路引导）

![](img/0eadc3471017522c8adba6012d574829_107.png)

![](img/0eadc3471017522c8adba6012d574829_109.png)

![](img/0eadc3471017522c8adba6012d574829_111.png)

通过散点图，我们已经可以直观地看到关系。为了量化这种关系，我们可以进行线性回归分析。虽然本节没有展开代码实现，但其核心思路是：
*   目标是找到一个线性方程 **`y = a * x + b`** 来拟合散点图中的数据点，其中`y`是VX收益率，`x`是SPX收益率。
*   系数 **`a`** 就是回归系数（或斜率），它量化了`x`变化一个单位时，`y`平均变化多少。我们预期`a`为一个负数，证实负相关。
*   我们可以在散点图上画出这条回归线，使得关系一目了然。
