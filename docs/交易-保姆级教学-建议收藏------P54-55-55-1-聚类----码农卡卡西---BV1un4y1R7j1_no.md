接下来，我们导入要使用的数据。在上节课中，我们提到需要从数据中提取特征。在本例中，我们选择了两个特征：股票前一天的指标和前两天的指标。这两个指标将用于后续的建模。

# 假设data是已经加载的Pandas DataFrame

features = data[['feature_1', 'feature_2']]

## 第三步：建立K-Means聚类模型 🤖

![](img/01ef4f71c2e8df7a2a94d736fd0af495_23.png)

![](img/01ef4f71c2e8df7a2a94d736fd0af495_25.png)

之前我们使用回归方法进行分析，本节我们将尝试聚类方法。在`sklearn`库中，有一个非常实用的算法叫做**K-Means**。

首先，我们从`sklearn.cluster`模块导入`KMeans`类。

```python
from sklearn.cluster import KMeans
```

然后，我们实例化一个K-Means模型。这里需要一个关键参数：`n_clusters`，即我们希望将数据分成多少个簇（K值）。在我们的任务中，目标相对简单：判断股票是上涨还是下跌。因此，我们设定`n_clusters=2`，将数据分为两个簇。

```python
model = KMeans(n_clusters=2)
```

![](img/01ef4f71c2e8df7a2a94d736fd0af495_27.png)

![](img/01ef4f71c2e8df7a2a94d736fd0af495_29.png)

## 第四步：训练模型并预测 🔍

![](img/01ef4f71c2e8df7a2a94d736fd0af495_31.png)

模型实例化后，我们使用`.fit()`方法，传入准备好的特征数据来训练模型。

![](img/01ef4f71c2e8df7a2a94d736fd0af495_33.png)

```python
model.fit(features)
```

训练完成后，模型会显示一些默认参数，例如初始化中心点的方法、最大迭代次数等。这些参数在`sklearn`的官方文档中有详细说明。

接着，我们使用训练好的模型对原始数据进行预测，得到每个数据点所属的簇（0或1）。

```python
data['cluster'] = model.predict(features)
```

![](img/01ef4f71c2e8df7a2a94d736fd0af495_35.png)

为了更直观地理解结果，我们可以将预测的标签0和1映射为我们更熟悉的“-1”（代表下跌）和“1”（代表上涨）。

![](img/01ef4f71c2e8df7a2a94d736fd0af495_37.png)

```python
data['cluster'] = data['cluster'].where(data['cluster'] == 1, -1)
```

## 第五步：可视化聚类结果 🎨

![](img/01ef4f71c2e8df7a2a94d736fd0af495_45.png)

为了直观地查看聚类效果，我们可以将结果用散点图画出来。不同颜色代表不同的簇。

![](img/01ef4f71c2e8df7a2a94d736fd0af495_47.png)

```python
plt.scatter(features.iloc[:, 0], features.iloc[:, 1], c=data['cluster'], cmap='coolwarm')
plt.xlabel('Feature 1 (前一天指标)')
plt.ylabel('Feature 2 (前两天指标)')
plt.title('K-Means聚类结果')
plt.show()
```

![](img/01ef4f71c2e8df7a2a94d736fd0af495_49.png)

在生成的图中，蓝色点可能代表一个簇（例如下跌趋势），红色点代表另一个簇（例如上涨趋势）。通过观察可以发现，当两个特征值都为负时（即昨天和前天都下跌），数据点更可能被归为下跌簇。

## 第六步：评估聚类策略效果 💹

![](img/01ef4f71c2e8df7a2a94d736fd0af495_53.png)

仅仅得到聚类标签还不够，我们需要评估基于此构建的交易策略是否有效。我们计算一下，如果按照聚类标签进行交易（例如，标签为1时买入，为-1时卖出或做空），最终的收益情况如何。

![](img/01ef4f71c2e8df7a2a94d736fd0af495_55.png)

首先，我们需要将之前对收益率取对数的操作还原。

![](img/01ef4f71c2e8df7a2a94d736fd0af495_57.png)

![](img/01ef4f71c2e8df7a2a94d736fd0af495_59.png)

```python
# 假设原始收益率列名为‘returns’，我们已对其取对数存储在‘log_returns’中
data['strategy_returns'] = data['log_returns'] * data['cluster']
# 将对数收益率还原为实际收益率
data['cumulative_returns'] = (data['strategy_returns']).cumsum().apply(np.exp)
```

![](img/01ef4f71c2e8df7a2a94d736fd0af495_61.png)

然后，我们可以对比“什么都不做”的基准收益和聚类策略的收益。

![](img/01ef4f71c2e8df7a2a94d736fd0af495_63.png)

```python
baseline_return = data['returns'].cumsum().apply(np.exp).iloc[-1]
strategy_return = data['cumulative_returns'].iloc[-1]

![](img/01ef4f71c2e8df7a2a94d736fd0af495_65.png)

![](img/01ef4f71c2e8df7a2a94d736fd0af495_67.png)

print(f"基准收益: {baseline_return}")
print(f"聚类策略收益: {strategy_return}")
```

![](img/01ef4f71c2e8df7a2a94d736fd0af495_69.png)

**注意**：由于K-Means聚类本身具有随机性（例如初始中心点的选择），每次运行的结果可能不同，导致收益评估不稳定。在实际应用中，这可能是一个需要关注的问题。

![](img/01ef4f71c2e8df7a2a94d736fd0af495_71.png)

![](img/01ef4f71c2e8df7a2a94d736fd0af495_75.png)

本节课我们一起学习了如何在量化交易中应用K-Means聚类分析。我们完成了从数据准备、模型建立、预测到结果可视化和简单策略评估的全过程。

![](img/01ef4f71c2e8df7a2a94d736fd0af495_79.png)

需要强调的是，聚类是一种**无监督学习**方法，在缺乏明确标签的量化场景中，其解释性和稳定性可能不如有监督学习方法（如回归、分类）。因此，它通常不作为首选的策略构建核心方法，而是在探索性数据分析或特定场景下的一种补充工具。
