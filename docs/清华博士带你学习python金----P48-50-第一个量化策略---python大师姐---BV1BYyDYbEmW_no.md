在本节课中，我们将学习如何查看和分析量化策略的回测结果。我们将了解回测平台提供的详细报告页面，并解读其中关键的图表和金融指标，以评估策略的绩效。

## 查看详细回测报告

上一节我们介绍了如何运行一个简单的策略。本节中我们来看看如何获取更详细的分析报告。

如果你点击“运行回测”按钮，平台会生成一个更详细的报告页面。这个页面包含了多种图表和数据指标，用于深入分析策略表现。

![](img/0ff93fdc493afab45d6dd74c504c4e4c_1.png)

## 解读回测图表

报告页面中的图表直观地展示了策略的运行情况。

第一张图是策略收益与基准（如大盘指数）的对比曲线。在我们的例子中，蓝线代表策略收益，红线代表基准收益。由于测试时间设定在2015年6月到2016年6月（股灾期间），两者表现都较差，且走势相近。

第二张图是每日盈亏柱状图。它显示了策略在每个交易日是盈利还是亏损，以及具体的金额。柱状图向上为盈利，向下为亏损。

第三张图是每日买卖柱状图。它显示了每个交易日的买入和卖出总量，向上代表净买入，向下代表净卖出。

## 理解关键绩效指标

除了图表，报告还提供了一系列量化的金融指标，它们是判断策略优劣的核心依据。

以下是报告中一些重要的指标及其解释：

*   **策略收益**：策略在整个回测期间的总收益率。计算公式为 `(最终资产 - 初始资产) / 初始资产`。在我们的例子中，策略亏损了41%。
*   **基准收益**：作为比较基准的市场指数（如沪深300）在同一时期的收益率。本例中基准亏损了34%。
*   **策略年化收益**：将策略收益折算成年利率的形式，便于比较不同时间长度的策略。计算公式涉及复利，通常为 `年化收益 = (1 + 总收益)^(250 / N) - 1`，其中N是回测总交易日数。
*   **阿尔法（Alpha）**：衡量策略超越市场基准的超额收益能力。**Alpha > 0** 表示策略获得了超额收益。
*   **贝塔（Beta）**：衡量策略收益相对于市场基准波动的敏感性，即系统性风险。Beta=1表示与市场波动同步。
*   **夏普比率（Sharpe Ratio）**：衡量策略每承受一单位总风险，能产生多少超额回报。比率越高，说明风险调整后的收益越好。
*   **胜率**：盈利的交易次数占总交易次数的比例。
*   **盈亏比**：总盈利金额与总亏损金额的比值。**盈亏比 > 1** 表示总体盈利大于亏损。
*   **最大回撤**：策略在回测期间，资产净值从最高点到之后最低点的最大跌幅。这是衡量策略下行风险的关键指标，数值越小越好。

## 查阅详细交易记录

报告页面还提供了更详细的子报表，供深度分析。

例如，“交易详情”列出了回测中模拟的每一笔买卖委托，包括日期、股票代码、买卖方向、成交价格和数量。“每日持仓收益”则展示了每日收盘后的持仓市值及浮动盈亏。

如果你想深入研究策略为何成功或失败，可以仔细查看这些详细数据。
