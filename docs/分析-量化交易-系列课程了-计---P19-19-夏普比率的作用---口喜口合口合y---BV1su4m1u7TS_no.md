## 夏普比率的定义与作用
上一节我们介绍了投资回报率，本节中我们来看看如何衡量风险与收益的关系。

夏普比率描述的是投资组合每承受一单位风险，能够产生多少超额回报。它衡量的是风险调整后的收益。指标越高，意味着在承担相同风险的情况下，获得的超额收益越高。

![](img/017bdb9eab900609c7e549eb90ac0c78_1.png)

我们可以通过一个类比来理解：假设有两份工作，一份是稳定的文员工作，月薪5000元；另一份是叙利亚的雇佣兵工作，日薪3万元。后者的高薪对应着极高的生命风险。在投资中，夏普比率就是帮助我们判断，为了获得更高的收益，所承担的额外风险是否“值得”。

![](img/017bdb9eab900609c7e549eb90ac0c78_3.png)

![](img/017bdb9eab900609c7e549eb90ac0c78_5.png)

因此，在选股或构建投资组合时，在其他条件相似的情况下，我们倾向于选择夏普比率更高的资产，因为它意味着更高的“性价比”。

## 夏普比率的计算公式
理解了夏普比率的作用后，我们来看看它的具体计算方法。

夏普比率的核心思想是比较投资组合的收益与无风险收益的差值，并将其与投资组合的风险（波动性）相关联。公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率
*   **Rf**：无风险收益率（例如国债收益率）
*   **σp**：投资组合收益率的标准差（代表风险）

这个公式计算的是**每单位总风险所获得的超额收益**。我们希望这个值越大越好。

## 使用Python计算夏普比率
接下来，我们将在Python环境中实际计算一组股票的夏普比率。

![](img/017bdb9eab900609c7e549eb90ac0c78_7.png)

首先，我们需要准备好数据。这里假设我们已经有了股票的历史价格数据，并计算出了每日的回报率序列 `returns`。在计算回报率时，通常需要对缺失值进行处理，例如使用前一天的数值进行填充。

![](img/017bdb9eab900609c7e549eb90ac0c78_9.png)

```python
import numpy as np
import pandas as pd

# 假设 returns 是一个Pandas DataFrame，每一列代表一只股票的每日回报率
# 例如：returns = data.pct_change().fillna(method='ffill')

# 设置年化无风险利率，例如3%
risk_free_rate = 0.03
# 计算年化夏普比率，假设一年有250个交易日
days_per_year = 250

# 计算每只股票的平均超额收益（年化）
excess_return = returns.mean() * days_per_year - risk_free_rate
# 计算每只股票收益的标准差（年化）
return_std = returns.std() * np.sqrt(days_per_year)
# 计算夏普比率
sharpe_ratio = excess_return / return_std

![](img/017bdb9eab900609c7e549eb90ac0c78_11.png)

print(sharpe_ratio)
```

![](img/017bdb9eab900609c7e549eb90ac0c78_13.png)

代码说明：
1.  `excess_return`：计算了投资组合的年化平均收益率与无风险收益率的差值。
2.  `return_std`：计算了投资组合年化收益率的标准差。注意，日波动率需要乘以交易日数量的平方根来进行年化。
3.  `sharpe_ratio`：将超额收益除以风险（标准差），得到最终的夏普比率。

![](img/017bdb9eab900609c7e549eb90ac0c78_15.png)

计算结果是一个序列，展示了每只股票的夏普比率。数值最大的股票，意味着其单位风险带来的超额收益最高，从该指标看是最优选择。负值则表明其收益尚不及无风险收益。
