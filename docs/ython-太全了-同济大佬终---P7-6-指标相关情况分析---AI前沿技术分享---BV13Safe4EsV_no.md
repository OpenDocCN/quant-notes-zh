## 第三步：使用双Y轴进行可视化

将两个指标放在同一坐标系但使用左右不同的Y轴，可以更清晰地观察它们的反向变动关系。

![](img/1161c8a1e138073a78191d748ac3cef8_47.png)

![](img/1161c8a1e138073a78191d748ac3cef8_49.png)

以下是创建双Y轴图表的步骤：

![](img/1161c8a1e138073a78191d748ac3cef8_51.png)

![](img/1161c8a1e138073a78191d748ac3cef8_53.png)

1.  首先绘制第一个序列（如SPX），并获取当前的坐标轴对象。
2.  在该坐标轴对象上，使用 `twinx()` 方法创建共享X轴的第二个Y轴。
3.  在第二个Y轴上绘制第二个序列（如VIX）。

![](img/1161c8a1e138073a78191d748ac3cef8_55.png)

![](img/1161c8a1e138073a78191d748ac3cef8_57.png)

对应的代码如下：
```python
import matplotlib.pyplot as plt

![](img/1161c8a1e138073a78191d748ac3cef8_59.png)

fig, ax1 = plt.subplots(figsize=(10, 6))

![](img/1161c8a1e138073a78191d748ac3cef8_61.png)

# 在第一个Y轴（左侧）绘制SPX
color = ‘tab:blue’

color = ‘tab:green’
ax2.set_ylabel(‘VIX’, color=color)

ax2.tick_params(axis=‘y’, labelcolor=color)

fig.tight_layout()
plt.show()
```
如果数据时间跨度较长，图表可能过于密集。我们可以选取其中一段时间（例如2010年至2012年）的数据进行绘制，这样趋势会更明显。从图中可以观察到，当SPX指数下降时，VIX指数往往上升，这初步揭示了它们之间的负相关关系。

## 第四步：散点图与分布分析

除了折线图，散点图是观察两个变量关系的强大工具。Pandas库提供了便捷的 `scatter_matrix` 函数，可以一次性生成散点图矩阵和每个变量的分布图。

以下是绘制散点图矩阵的步骤：

![](img/1161c8a1e138073a78191d748ac3cef8_67.png)

![](img/1161c8a1e138073a78191d748ac3cef8_69.png)

1.  首先，我们计算指标的日收益率（或对数收益率）作为分析基础，以观察变化率之间的关系。公式为：`收益率 = (今日价格 / 昨日价格) - 1` 或使用对数收益率 `log(今日价格 / 昨日价格)`。
2.  使用 `pandas.plotting.scatter_matrix` 函数，传入收益率数据。
3.  通过参数设置散点透明度、对角线图形（直方图或核密度估计图）等。

![](img/1161c8a1e138073a78191d748ac3cef8_71.png)

对应的代码如下：
```python
import pandas as pd
import numpy as np

![](img/1161c8a1e138073a78191d748ac3cef8_73.png)

![](img/1161c8a1e138073a78191d748ac3cef8_75.png)

![](img/1161c8a1e138073a78191d748ac3cef8_77.png)

# 计算对数收益率
returns = np.log(data / data.shift(1)).dropna()

# 绘制散点图矩阵，对角线显示核密度估计图
pd.plotting.scatter_matrix(returns,
                           alpha=0.5, # 散点透明度
                           figsize=(10, 10),
                           diagonal=‘kde’, # 对角线画核密度估计图
                           density_kwds={‘bins’: 50})
plt.show()
```
在这个矩阵图中：
*   **对角线** 显示的是每个变量自身的分布情况（核密度估计图），它比直方图更平滑。
*   **非对角线** 的散点图展示了两个变量两两之间的关系。从SPX和VIX的散点图中，我们可以更清晰地看到数据点从左上到右下分布的负相关趋势。

## 第五步：进行回归分析

![](img/1161c8a1e138073a78191d748ac3cef8_85.png)

![](img/1161c8a1e138073a78191d748ac3cef8_87.png)

散点图已经直观展示了关系，但我们还可以用一条回归线来量化这种关系。虽然本段代码未直接实现，但其思路是在散点图的基础上，使用`scipy`或`statsmodels`库拟合一个线性模型 **y = β₀ + β₁x + ε**，并将拟合的直线添加到散点图上。回归系数 **β₁** 可以明确告诉我们关系强度和方向（正或负）。

## 总结

![](img/1161c8a1e138073a78191d748ac3cef8_91.png)

![](img/1161c8a1e138073a78191d748ac3cef8_93.png)

本节课中我们一起学习了如何分析金融指标间的相关性。
1.  **数据准备**：我们首先选取了需要分析的指标列（SPX和VIX）。
2.  **可视化探索**：我们通过绘制**双Y轴折线图**，在同一张图上观察了两个指标走势的反向关系。
3.  **深入分析**：通过计算收益率并绘制**散点图矩阵**，我们不仅看到了两个指标间的负相关散点分布，还通过对角线上的**核密度估计图**了解了各自的数据分布形态。
这些图表工具能帮助我们快速、直观地把握变量间的关联，为后续的量化建模（如回归分析）打下坚实基础。
