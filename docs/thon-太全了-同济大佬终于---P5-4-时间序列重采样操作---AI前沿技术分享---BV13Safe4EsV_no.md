# 金融量化分析：P5：4-时间序列重采样与窗口操作 📊

在本节课中，我们将要学习时间序列分析中的两个核心操作：重采样与时间窗口操作。这些技术对于调整数据频率、分析数据在不同时间尺度上的模式以及计算滑动统计指标至关重要。

![](img/c95539b6105d650083c1f49ff7fc6b45_1.png)

## 重采样操作 🔄

![](img/c95539b6105d650083c1f49ff7fc6b45_3.png)

上一节我们介绍了时间序列的基础概念，本节中我们来看看如何改变数据的时间频率，即重采样。重采样是指将时间序列数据从一个频率转换到另一个频率的过程。例如，将每日数据汇总为每周或每月数据。

![](img/c95539b6105d650083c1f49ff7fc6b45_5.png)

以下是重采样的基本步骤：

![](img/c95539b6105d650083c1f49ff7fc6b45_7.png)

![](img/c95539b6105d650083c1f49ff7fc6b45_9.png)

1.  **指定目标频率**：你需要告诉程序希望以什么时间单位进行重采样，例如“W”代表周，“M”代表月，“Y”代表年。
2.  **指定聚合方法**：在频率转换时，需要决定如何聚合原始数据。例如，将一周内的每日数据计算为周平均值、周总和或周第一个值等。

![](img/c95539b6105d650083c1f49ff7fc6b45_11.png)

在Python的pandas库中，可以使用`resample()`方法轻松完成。核心代码如下：

```python
# 假设df是一个带有日期时间索引的DataFrame
# 以周为单位进行重采样，并计算每周的平均值
weekly_data = df.resample('W').mean()
# 以月为单位进行重采样，并计算每月的第一个值
monthly_data = df.resample('M').first()
```

### 标签位置参数

![](img/c95539b6105d650083c1f49ff7fc6b45_13.png)

![](img/c95539b6105d650083c1f49ff7fc6b45_15.png)

在重采样时，还有一个重要参数是`label`，它决定了聚合后数据点的标签（索引）使用时间窗口的哪一端。

以下是`label`参数的两种主要选项：

![](img/c95539b6105d650083c1f49ff7fc6b45_17.png)

![](img/c95539b6105d650083c1f49ff7fc6b45_19.png)

*   **`label=‘right’`**：默认选项。使用时间窗口的**结束日期**作为该聚合结果的标签。例如，汇总1月1日至1月7日的数据，标签显示为1月7日。这更符合“截至某日”的观察逻辑。
*   **`label=‘left’`**：使用时间窗口的**开始日期**作为标签。同上例，标签会显示为1月1日。

![](img/c95539b6105d650083c1f49ff7fc6b45_21.png)

![](img/c95539b6105d650083c1f49ff7fc6b45_23.png)

你可以根据分析需求选择合适的标签。例如，在金融分析中，通常使用`label=‘right’`来代表截至某日的统计值。

```python
# 使用右标签（默认）
monthly_right = df.resample('M', label='right').mean()
# 使用左标签
monthly_left = df.resample('M', label='left').mean()
```

## 时间窗口（滑动窗口）操作 🪟

了解了如何改变数据频率后，我们来看看如何在原有频率上计算滑动统计量。时间窗口操作不是改变数据点，而是在一个固定长度的“窗口”内滑动，并计算窗口内数据的统计指标（如均值、最小值、最大值、标准差等）。这有助于分析数据的局部趋势和波动。

![](img/c95539b6105d650083c1f49ff7fc6b45_25.png)

![](img/c95539b6105d650083c1f49ff7fc6b45_27.png)

例如，对于一个每日股价序列，我们可以计算其10日移动平均线，即窗口大小为10的均值序列。

### 执行窗口操作

![](img/c95539b6105d650083c1f49ff7fc6b45_29.png)

在pandas中，主要使用`rolling()`方法来实现。

![](img/c95539b6105d650083c1f49ff7fc6b45_31.png)

以下是进行时间窗口操作的关键步骤：

![](img/c95539b6105d650083c1f49ff7fc6b45_33.png)

1.  **处理缺失值**：在计算窗口统计量之前，**必须**先处理原始数据中的缺失值（NaN）。否则，只要窗口内包含一个NaN，整个窗口的统计结果就会是NaN。通常使用`dropna()`方法移除含有缺失值的行。
2.  **创建滚动对象**：对处理后的数据列调用`rolling(window=N)`，其中`N`是窗口大小。
3.  **应用聚合函数**：对创建的滚动对象应用聚合函数，如`.mean()`， `.min()`， `.std()`等。

![](img/c95539b6105d650083c1f49ff7fc6b45_35.png)

![](img/c95539b6105d650083c1f49ff7fc6b45_37.png)

核心操作代码如下：

![](img/c95539b6105d650083c1f49ff7fc6b45_39.png)

```python
# 1. 处理缺失值
df_clean = df.dropna()

![](img/c95539b6105d650083c1f49ff7fc6b45_41.png)

![](img/c95539b6105d650083c1f49ff7fc6b45_43.png)

# 2. & 3. 计算10日滑动窗口的最小值
rolling_min = df_clean[‘column_name’].rolling(window=10).min()

# 可以同时计算多个窗口统计量并添加到DataFrame中
df_clean[‘rolling_min’] = df_clean[‘column_name’].rolling(window=10).min()
df_clean[‘rolling_max’] = df_clean[‘column_name’].rolling(window=10).max()
df_clean[‘rolling_mean’] = df_clean[‘column_name’].rolling(window=10).mean()
df_clean[‘rolling_std’] = df_clean[‘column_name’].rolling(window=10).std()

![](img/c95539b6105d650083c1f49ff7fc6b45_45.png)

![](img/c95539b6105d650083c1f49ff7fc6b45_47.png)

# 查看结果
print(df_clean.tail())
```
**注意**：前`N-1`个窗口（例如窗口大小为10时，前9个数据点）由于数据不足，其统计结果将为NaN，这是正常现象。
