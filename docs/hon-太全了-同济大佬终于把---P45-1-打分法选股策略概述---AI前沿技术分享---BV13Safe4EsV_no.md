# 量化投资入门：P45：1-打分法选股策略概述

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分选股法。我们将从策略的基本概念入手，解释其核心思想，并梳理实施该策略所需的关键步骤和先验知识。

## 策略核心思想

上一节我们介绍了因子分析的基本概念，本节中我们来看看如何综合运用多个因子来构建选股策略。

因子打分选股的核心思想，是模仿学生通过各科成绩总分进行排名和筛选的过程。在股票投资中，我们不只依赖单一因子（如市盈率）来判断股票好坏，而是综合考察多个因子（如估值、成长、价量等维度下的具体指标）。

假设我们有300支股票（股票1， 股票2， …， 股票300），并为每支股票计算了A、B、C、D四个因子的数值。我们的目标不是单独看某个因子，而是为每个因子打分，然后计算一个**总分**。最后，我们根据总分对所有股票进行排名，并选取排名靠前的股票（如前10名）作为投资组合。这就像根据学生的语文、数学、英语、地理总分来选拔优等生。

**核心公式**可以表示为：
`股票总分 = f(因子A得分， 因子B得分， 因子C得分， 因子D得分)`
其中，`f`通常为加权求和函数。

## 策略实施的关键前提

在具体实施打分之前，我们必须明确一个关键的先验知识：每个因子与股票未来收益率的相关性是正向还是负向。

以下是理解这一点的两个关键：

1.  **正向因子**：因子数值**越大**，通常预示着未来收益率**可能越高**。例如，每股收益、净资产收益率等。
2.  **负向因子**：因子数值**越小**，通常预示着未来收益率**可能越高**。例如，市盈率、市净率（在估值过高时）等。

明确因子方向至关重要，因为它决定了我们打分的规则：对于正向因子，数值越大得分应越高；对于负向因子，数值越小得分应越高。

我们可以通过两种方式获取这些先验知识：
*   **查阅研究报告**：参考券商或研究机构发布的因子研究结论。
*   **自行计算验证**：通过历史数据计算因子与收益率的**信息系数（IC）** 等指标进行验证。

## 策略执行步骤概述

基于以上思想，我们可以梳理出因子打分选股策略的基本步骤：

以下是实施该策略的主要环节：

1.  **因子选择与方向确认**：选取多个有效的因子，并明确每个因子与预期收益的相关方向（正/负）。
2.  **数据获取与处理**：获取股票池（如沪深300成分股）在特定时点的各因子原始数据。
3.  **因子标准化与打分**：对每个因子的原始数据进行处理（如去极值、标准化），然后根据其方向性转换为统一的得分（例如百分制排名得分）。
4.  **计算综合得分**：将每个股票的各因子得分按既定权重（如等权）相加，得到该股票的综合总分。
    `综合总分 = 权重A * 得分A + 权重B * 得分B + ...`
5.  **排序与选股**：根据综合总分对所有股票进行降序排序，选取排名最高的N支股票构成投资组合。
6.  **定期调仓**：策略通常按固定周期（如每月）重复上述过程，更新股票组合。

本节课中我们一起学习了因子打分选股策略的基本框架。我们了解到，该策略通过综合多个因子的信息，为股票计算一个综合评分，并依据评分高低进行选股。实施策略的关键在于明确每个因子的作用方向，并依此进行合理的打分与加权。在接下来的课程中，我们将在量化平台上具体实现这一策略。
