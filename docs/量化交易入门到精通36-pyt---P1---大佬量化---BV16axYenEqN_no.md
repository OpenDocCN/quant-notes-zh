## 什么是循环语句？

循环语句就像一位勤劳的助手，可以帮我们重复执行某些代码，省时省力。例如，你想在A股市场上找到所有过去一年中每个季度净利润都增长的股票。如果手动逐一检查，效率极低。此时，循环语句就能派上用场，它可以遍历所有股票数据，配合条件判断，最终筛选出目标股票。

Python中的循环主要分为两种：`while`循环和`for`循环。本节课我们先学习`while`循环。

## while循环：执着的小助手

`while`循环就像一个执着的小助手，只要条件满足，就会一直执行下去。它的基本语法结构如下：

```python
while 判断条件:
    执行语句
```

以下是使用`while`循环时需要注意的要点：
*   **冒号**：`while`语句末尾必须有一个冒号`:`。
*   **缩进**：循环体（即要重复执行的代码块）必须缩进。通常使用4个空格或一个`Tab`键进行缩进。

## 代码示例与解析

让我们通过一个具体的例子来理解`while`循环是如何工作的。

![](img/ce3508935c6ce35bf022901876a5a3ae_5.png)

```python
# 初始化循环次数为3
循环次数 = 3

![](img/ce3508935c6ce35bf022901876a5a3ae_7.png)

# 当循环次数大于0时，执行循环体内的代码
while 循环次数 > 0:
    print(f"当前循环次数：{循环次数}")
    # 每次循环后，循环次数减1
    循环次数 -= 1  # 这等同于：循环次数 = 循环次数 - 1
```

![](img/ce3508935c6ce35bf022901876a5a3ae_9.png)

这段代码的预期效果是打印出三行信息。我们来执行并分析其过程：

1.  **初始状态**：变量`循环次数`被赋值为`3`。
2.  **第一次循环**：判断`3 > 0`为`True`，进入循环体。打印`当前循环次数：3`，然后执行`循环次数 -= 1`，`循环次数`的值变为`2`。
3.  **第二次循环**：判断`2 > 0`为`True`，进入循环体。打印`当前循环次数：2`，然后`循环次数`的值变为`1`。
4.  **第三次循环**：判断`1 > 0`为`True`，进入循环体。打印`当前循环次数：1`，然后`循环次数`的值变为`0`。
5.  **循环结束**：再次判断条件`0 > 0`，结果为`False`，`while`循环停止执行。

程序最终输出结果为：
```
当前循环次数：3
当前循环次数：2
当前循环次数：1
```

代码中的 `循环次数 -= 1` 是关键，它确保了循环变量在每次迭代后发生变化，最终使条件变为`False`，从而避免程序陷入无限循环。
