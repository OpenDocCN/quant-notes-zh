## 对比不同股票组合的表现

![](img/6f04cc155fb06481cc36333e0cf65b85_44.png)

之前我们一直选择综合打分**排名前十**的股票构建投资组合。一个自然的疑问是：如果选择排名靠后的股票，结果会怎样？

![](img/6f04cc155fb06481cc36333e0cf65b85_46.png)

![](img/6f04cc155fb06481cc36333e0cf65b85_48.png)

以下是修改股票选择范围的代码：
```python
# 原代码：选择排名第1到第10的股票（前十名）
selected_stocks = ranked_stocks.iloc[0:10]

![](img/6f04cc155fb06481cc36333e0cf65b85_50.png)

# 修改后：选择排名靠后的股票（例如从第10名往后）
selected_stocks = ranked_stocks.iloc[10:]
```

![](img/6f04cc155fb06481cc36333e0cf65b85_52.png)

执行回测后发现：
*   当选择**排名前十**的股票时，策略年化收益约为**10%**，显著跑赢基准（基准收益为负）。
*   当选择**排名靠后（最差）**的股票时，策略收益（约 **-10%**）比基准收益（约 **-10%**）更差。

![](img/6f04cc155fb06481cc36333e0cf65b85_54.png)

![](img/6f04cc155fb06481cc36333e0cf65b85_56.png)

这个对比实验有力地验证了我们打分指标的有效性：**得分高的股票组合确实能产生更好的收益，而得分低的组合表现更差。**

![](img/6f04cc155fb06481cc36333e0cf65b85_58.png)

![](img/6f04cc155fb06481cc36333e0cf65b85_60.png)

## 解读详细回测报告

![](img/6f04cc155fb06481cc36333e0cf65b85_62.png)

除了汇总指标，回测系统还会生成详细的交易记录。

![](img/6f04cc155fb06481cc36333e0cf65b85_64.png)

以下是报告中的关键部分：
*   **交易详情**：显示策略在**每个月**进行的调仓操作，例如4月、5月、6月等，符合我们设定的月度调仓逻辑。
*   **持仓详情**：展示了每个调仓日具体的股票持仓列表和数量。
*   **账户信息**：记录了初始资金（如100万）和最终资金（如77万）的变化，直观反映了策略的盈亏情况。

![](img/6f04cc155fb06481cc36333e0cf65b85_66.png)

## 策略总结与要点回顾

![](img/6f04cc155fb06481cc36333e0cf65b85_70.png)

本节课中我们一起学习了如何对一个量化策略进行全面的总结与分析。

![](img/6f04cc155fb06481cc36333e0cf65b85_72.png)

首先，我们执行了策略回测并观察了核心指标，如收益、最大回撤和超额收益。

![](img/6f04cc155fb06481cc36333e0cf65b85_74.png)

接着，我们通过延长和调整回测时间区间，认识到策略表现与市场整体环境密切相关，在熊市中可能难以获得突出收益。

![](img/6f04cc155fb06481cc36333e0cf65b85_76.png)

然后，我们进行了一个重要的对比实验：通过改变股票选择范围（选前十 vs 选后十），验证了打分模型的有效性。**公式**可以简单表示为：
**策略收益 = f(股票打分排名)**，其中排名越靠前，预期收益越高。

![](img/6f04cc155fb06481cc36333e0cf65b85_78.png)

最后，我们学习了如何解读详细的回测报告，包括交易流水、持仓变化和账户资金曲线。

![](img/6f04cc155fb06481cc36333e0cf65b85_80.png)

打分法是一种直观且常用的多因子选股方法，能够快速得到一个相对有效的投资组合。但需要注意的是，任何策略都无法完全规避市场系统性风险，其表现会随着牛熊市转换而波动。在实践中，持续优化因子、调整权重和进行严格的风险管理至关重要。
