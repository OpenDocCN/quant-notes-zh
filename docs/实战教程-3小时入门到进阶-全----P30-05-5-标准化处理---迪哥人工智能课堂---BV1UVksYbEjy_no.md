在本节课中，我们将要学习数据预处理中一个至关重要的步骤——标准化。标准化是数据挖掘和机器学习中非常常用的技术，其核心目的是消除不同特征（指标）之间由于量纲和取值范围不同带来的影响，使数据更易于分析和建模。

![](img/bcb93f1d311124e7f24dded4c5a45276_3.png)

## 标准化的目的与原理 🎯

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体作用。

为了理解标准化的目的，我们可以观察一个包含不同指标的数据集。假设数据有两个特征，X1和X2。X1的取值范围可能较大，例如从-10到10；而X2的取值范围可能较小，例如从-1到1。这种取值范围的差异会导致在后续分析中，数值大的特征可能对结果产生不成比例的巨大影响。

标准化的出发点就是为了使不同的指标在各个维度上的取值范围尽可能相同。这样，我们可以用同样的“眼光”去观察它们，避免因纯粹的数值差异而对分析结果造成误导。

## 标准化的核心公式 📐

标准化的具体操作遵循一个明确的数学公式。以下是标准化的核心公式：

**`z = (x - μ) / σ`**

在这个公式中：
*   **`x`** 代表原始数据值。
*   **`μ`** 代表该特征所有数据的平均值（均值）。
*   **`σ`** 代表该特征所有数据的标准差。

这个公式可以分解为两个连续的步骤，让我们逐一理解。

### 第一步：去均值（中心化）

公式中的 **`(x - μ)`** 部分被称为“去均值”或“中心化”。它的目的是将数据分布的中心移动到零点。

原始数据通常不是以坐标原点为中心对称分布的。执行去均值操作后，数据在各个维度上都会以零为中心变得对称。这解决了数据分布中心不一致的问题。

### 第二步：除以标准差（缩放）

然而，仅仅去均值并不能解决不同特征取值范围不同的问题。这时就需要公式中的 **`/ σ`** 部分，即除以标准差。

标准差衡量了一组数据的离散程度。取值范围小、数据点集中的特征，其标准差也小；取值范围大、数据点分散的特征，其标准差也大。

通过除以各自的标准差，我们实现了：
1.  对取值范围大的特征，除以一个较大的标准差，能将其数值范围“压缩”到一个合理的区间。
2.  对取值范围小的特征，除以一个较小的标准差，能将其数值范围“放大”到一个可比较的区间。

最终，经过“去均值”和“除以标准差”两步操作，数据不仅以零为中心对称，而且各个特征的数值都被转换到了一个近似的、较小的标准区间内（通常大部分数据落在[-3, 3]之间）。

## 标准化的代码实现 💻

![](img/bcb93f1d311124e7f24dded4c5a45276_5.png)

理解了原理后，我们可以动手实现标准化。以下是使用Python和Pandas手动实现标准化的方法：

![](img/bcb93f1d311124e7f24dded4c5a45276_7.png)

首先，我们需要计算数据的均值和标准差。

```python
import pandas as pd

![](img/bcb93f1d311124e7f24dded4c5a45276_9.png)

# 假设 `data` 是一个Pandas Series 或 DataFrame 的一列
mean_val = data.mean()  # 计算均值 μ
std_val = data.std()    # 计算标准差 σ
```

![](img/bcb93f1d311124e7f24dded4c5a45276_11.png)

![](img/bcb93f1d311124e7f24dded4c5a45276_13.png)

然后，应用标准化公式：

```python
def standardize(series):
    """
    对输入的Pandas Series进行标准化。
    参数:
        series: 待标准化的数据序列。
    返回:
        标准化后的数据序列。
    """
    mean_val = series.mean()
    std_val = series.std()
    return (series - mean_val) / std_val

![](img/bcb93f1d311124e7f24dded4c5a45276_15.png)

# 应用标准化函数
standardized_data = standardize(data)
```

![](img/bcb93f1d311124e7f24dded4c5a45276_17.png)

## 使用Scikit-learn库快速标准化 🚀

![](img/bcb93f1d311124e7f24dded4c5a45276_19.png)

在实际项目中，我们通常使用成熟的机器学习库来简化操作。Scikit-learn提供了非常便捷的标准化工具。

以下是使用Scikit-learn进行标准化的方法：

```python
from sklearn.preprocessing import StandardScaler

![](img/bcb93f1d311124e7f24dded4c5a45276_21.png)

# 1. 初始化标准化器
scaler = StandardScaler()

![](img/bcb93f1d311124e7f24dded4c5a45276_23.png)

# 2. 拟合并转换数据（假设`data`是二维DataFrame）
# fit计算数据的均值和标准差，transform应用转换
standardized_data_sklearn = scaler.fit_transform(data)

![](img/bcb93f1d311124e7f24dded4c5a45276_25.png)

# 或者，对于单列数据（需转换为二维格式）
standardized_data_sklearn = scaler.fit_transform(data.values.reshape(-1, 1))
```

![](img/bcb93f1d311124e7f24dded4c5a45276_27.png)

使用`StandardScaler`只需一两行代码即可完成标准化，并且能方便地保存拟合好的参数，用于后续对新数据的转换。

![](img/bcb93f1d311124e7f24dded4c5a45276_29.png)
