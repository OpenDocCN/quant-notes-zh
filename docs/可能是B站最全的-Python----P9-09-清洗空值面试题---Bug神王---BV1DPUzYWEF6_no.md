现在，`df`中包含了我们需要处理的四列温度数据，其中存在一些空值。

以下是处理这些空值的两种主要方法：

![](img/23b1d4b3002ac969bd5307f88799f798_39.png)

![](img/23b1d4b3002ac969bd5307f88799f798_41.png)

**方法一：删除包含空值的行**
这是最直接的方法，使用`dropna()`函数可以删除任何包含空值的行。

![](img/23b1d4b3002ac969bd5307f88799f798_43.png)

```python
# axis=0 表示按行操作，删除包含空值的行
df_cleaned_drop = df.dropna(axis=0)
```

![](img/23b1d4b3002ac969bd5307f88799f798_45.png)

我们也可以使用`notnull()`和`all()`函数组合来实现相同的效果，它先检查每行是否全为非空值，然后索引出结果为`True`的行。

![](img/23b1d4b3002ac969bd5307f88799f798_47.png)

![](img/23b1d4b3002ac969bd5307f88799f798_49.png)

![](img/23b1d4b3002ac969bd5307f88799f798_51.png)

```python
df_cleaned_drop_alt = df[df.notnull().all(axis=1)]
```

![](img/23b1d4b3002ac969bd5307f88799f798_53.png)

![](img/23b1d4b3002ac969bd5307f88799f798_55.png)

原始数据有1060行，删除空值后剩余982行。由于删除的数据量不大，这种方法在此场景下是可行的。

![](img/23b1d4b3002ac969bd5307f88799f798_57.png)

![](img/23b1d4b3002ac969bd5307f88799f798_59.png)

**方法二：使用邻近值填充空值**
删除数据可能会造成信息损失。另一种方法是使用空值前面或后面的有效值来填充它，这特别适用于时间序列数据，因为相邻时刻的读数通常接近。

我们使用`fillna()`方法，并指定填充方向（`method`参数）。

![](img/23b1d4b3002ac969bd5307f88799f798_61.png)

![](img/23b1d4b3002ac969bd5307f88799f798_63.png)

```python
# 首先向前填充（用同一列上一个非空值填充）
df_filled = df.fillna(method='ffill', axis=0)
# 然后向后填充（用同一列下一个非空值填充），以确保所有空值都被处理
df_filled = df_filled.fillna(method='bfill', axis=0)
```

填充完成后，我们需要验证是否所有空值都已被处理。

![](img/23b1d4b3002ac969bd5307f88799f798_65.png)

![](img/23b1d4b3002ac969bd5307f88799f798_67.png)

```python
# 检查每列是否还存在任何空值，返回False表示该列已无空值
check_columns = df_filled.isnull().any(axis=0)
# 检查每行是否还存在任何空值
check_rows = df_filled.isnull().any(axis=1)
```

![](img/23b1d4b3002ac969bd5307f88799f798_69.png)

如果`check_columns`和`check_rows`的检查结果均为`False`，则说明数据中的空值已被成功清洗。
