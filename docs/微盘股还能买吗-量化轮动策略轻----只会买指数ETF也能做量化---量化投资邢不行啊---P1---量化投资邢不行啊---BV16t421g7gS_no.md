## 市场周期的启示 📈📉

以上的这些案例其实都在不断的告诉我们：这个市场没有永远的优和差，有的只是不断的周期往复。

其实不仅A股是这样的，美股也是如此。在美股股票市场的一两百年历史上，长期来看也是小市值的股票涨的更多一点。但是在60年代末到70年代初，已有好几年的时间，整个美国股市只有那些最大的50只股票正在上涨，其他股票都在下跌，历史上把当时的场景叫做“漂亮50”行情，大家都会抱团买入那些最好的公司。但是美股历史上，这段抱团投资的热潮很快就结束了。在1972年到1974年，这些抱团的“漂亮50”股票开始下跌，最多的比如说迪士尼，它从高点下跌了80%，还有麦当劳也从高点下跌了60%。这像不像你当年买的某安集团？而同期的标准普尔500指数只下跌了37%。

![](img/c82e918ac65af2319a1e4c1c19cce196_23.png)

![](img/c82e918ac65af2319a1e4c1c19cce196_25.png)

所以大家千万不要觉得，现在表现好的股票和策略，未来是只涨不跌的。就像我们的小市值股票，它之前几年表现都非常好，不断的上涨，但是也会遇到回调的时候，而且一回调就非常的猛烈。这一切其实都是周期的不断的重复，涨得多的自然要跌，跌的多的自然也要涨。

而我们作为投资者要做的是什么？其实也很简单，那就不要一根筋的只买小股票或者大股票。

## 策略构想：大小盘轮动 🔄

![](img/c82e918ac65af2319a1e4c1c19cce196_29.png)

我们可以尝试对它进行轮动。所以本期视频，我们就尝试构建一个大小盘轮动策略。

![](img/c82e918ac65af2319a1e4c1c19cce196_31.png)

首先，我们用大盘指数来代替大盘股股票，这主要是为了简单一点，大家也好理解。那么我们这里面选择沪深300、上证50、还有上证指数，这几个指数作为大盘指数的代表。然后同时，我们也使用创业板指数、上证小盘、中证1000这几个指数来代表小盘的股票。

![](img/c82e918ac65af2319a1e4c1c19cce196_33.png)

当你不断的在股票市场中进行交易，其实你会自觉或者不自觉地总结出来一些炒股的规律。比如说我自己就注意到有这样一种现象，我给大家举个例子：比如说在2010年9月28号这一天，沪深300指数最近一个月的涨幅是1.08%，而同期的创业板指数是跌了30.93%。那么明显可以看到，最近一个月我们沪深300指数是比较强的。

![](img/c82e918ac65af2319a1e4c1c19cce196_35.png)

那么接下来的一个交易日，沪深300指数跌了0.21%，而创业板指数跌了1.62%。虽然两者都是下跌，但明显沪深300指数跌的更少一点，它的表现更好。也就是说这个大小盘的指数，它的次日的表现延续了之前一个月的表现：之前一个月表现越好，下一天的表现也会越好。

我们再把时间快进到2010年的11月17号，在这里的17、18、19三天，这三天之前一个月的走势，发现是创业板的走势反而比沪深300指数要好很多。那么在接下来的一天当中，创业板是涨了1.61%，好于沪深300指数的1.42%。18、29号这两天也是。

![](img/c82e918ac65af2319a1e4c1c19cce196_37.png)

那么通过我举的这些例子，你会发现，当过去一段时间表现好的那个指数，它在下一个交易日它的好的表现可能会延续。当然，这个肯定不是绝对的，我这边举的这个例子都是我精挑细选出来的，很多时候它并不是按照这个规律表现。但是如果你在市场上混久了，你会发现好像这个规律确实在很大部分情况是会发生的。

那么我们很多量化投资的策略，其实就是来自于我们实盘交易的一些经验。我们完全可以把这些经验规则化、数量化，形成一套完整的、自动化的买卖逻辑。这样我们就把经验提取到了一个量化交易的策略。那我们接下来就来尝试着这么做。

![](img/c82e918ac65af2319a1e4c1c19cce196_39.png)

## 策略规则与回测验证 💡

![](img/c82e918ac65af2319a1e4c1c19cce196_41.png)

首先，我们这个策略针对的是大盘和小盘指数。我们默认使用沪深300、上证50和上证指数来代表大盘；创业板、上证小盘、中证1000来代表小盘指数。

![](img/c82e918ac65af2319a1e4c1c19cce196_43.png)

![](img/c82e918ac65af2319a1e4c1c19cce196_45.png)

每天收盘后，我们计算出每个指数之前20天的涨跌幅。哪个指数涨幅最大，我们就在接下来的一天持有哪个指数，如此循环往复。

有人听了这个描述可能还不是完全的清楚，我这边可以再举个例子：比如说我们现在是2010年的9月15日，这一天已经收盘了，然后我们统计到所有指数在前20天的涨跌幅。如图所示，一比就知道上证小盘指数涨得最多，那么我就在第二天开盘的时候，以开盘价去买入上证小盘指数ETF即可。可以想见，在未来几天，上证小盘指数仍然是过去20天涨得最多的一个指数，那么我们可能在未来几天持续持有这个小盘指数。

![](img/c82e918ac65af2319a1e4c1c19cce196_47.png)

![](img/c82e918ac65af2319a1e4c1c19cce196_49.png)

所以这个策略虽然日频发出信号，但它并不是那种换仓很频繁的策略。原理非常简单，在操作当中也是完全可行的，甚至你手工交易就可以。

![](img/c82e918ac65af2319a1e4c1c19cce196_51.png)

那这个策略到底能不能赚钱呢？这我们就需要借助指数的全部历史数据，并编写Python代码进行验证了。数据和代码我都已经准备好了。

![](img/c82e918ac65af2319a1e4c1c19cce196_53.png)

以下是策略的核心逻辑描述：

![](img/c82e918ac65af2319a1e4c1c19cce196_55.png)

![](img/c82e918ac65af2319a1e4c1c19cce196_57.png)

```python
# 伪代码示例：大小盘轮动策略核心逻辑
for 每个交易日 in 所有交易日:
    计算 每个候选指数 过去20个交易日的涨跌幅
    选择 涨跌幅最大的指数
    在下一个交易日开盘时，买入该指数对应的ETF
```

我们运行一下程序，看看结果如何。程序运行结果如图所示，代表策略的净值曲线从初始的1元涨到了10.64元，年化收益19.02%，非常不错，远远地跑赢了所有的指数。

但是作为策略来说，这个策略它的最大回撤有点大，高达51%。当然这个主要是由于这个策略永远满仓持有导致的，不管当时的行情好坏，他都会满仓的去持有。但是作为一个实际放钱的策略，我们不希望他的回撤这么大。这个时候我们就需要想一个办法，是不是可以去尝试优化一下这个策略，并且正好借这个案例去讲讲，我们是如何产生优化策略的思路，以及最终的优化结果到底是什么样的。

![](img/c82e918ac65af2319a1e4c1c19cce196_59.png)

## 策略优化：引入空仓机制 🛡️

![](img/c82e918ac65af2319a1e4c1c19cce196_61.png)

我们原来策略最大的问题就是它最大回撤太大，接近50%。为什么我们会在意最大回撤呢？因为这个影响我们实盘中是否能顺利地把我们策略执行下去。并且如果你的策略在第一年回撤50%，哪怕按照我们这个22%的年化收益策略，也需要到3年或者4年之后才能回本，这是非常亏的一件事情。

![](img/c82e918ac65af2319a1e4c1c19cce196_63.png)

那么想要降低回撤，我们就要去看一下我们原来策略，他回撤的开始时间和结束时间，是2015年的6月到2016年的6月，这段时间其实也就是2015年股灾的时候，市场整体处于一个极度的熊市当中。

![](img/c82e918ac65af2319a1e4c1c19cce196_65.png)

我们来看其中一个具体的交易日，2015年的6月25日。在此之前的20天，大盘指数和小盘指数都出现了较大幅度的下跌，然后之后几天的表现也不是很好。按照这个策略跑之后，五天表现是亏了12.7%。

![](img/c82e918ac65af2319a1e4c1c19cce196_67.png)

那看到这里你可能就能发现问题：什么问题呢？就是我们在市场整体下跌的时候，这个策略还是只是在大盘和小盘之间，选择跌的少的去买入。但是因为市场一直在跌，你不管选择什么之后，总归是下跌，这个就有点不合理了。当前市场明明处于熊市，但我们还在不断的买。

那更好的情况呢，就是我们发现大盘和小盘指数，它都在下跌的时候，我就停止我们的操作。比如说我们可以这样来改进我们的策略：我们在原来策略的基础上加一个条件，我们看一下大盘和小盘的涨幅是不是都小于零。如果都小于零的话，我下一天我就空仓了，我就不交易了，我就持有现金了。这是一个很简单也很合理的修改。

![](img/c82e918ac65af2319a1e4c1c19cce196_69.png)

那么我想知道这个修改它到底会不会有更好的效果呢？如果你不会编程的话，那你可能统计起来就很麻烦，但是你会编程的话就很简单，我们只需要在原有的代码上修改几行就可以了。

![](img/c82e918ac65af2319a1e4c1c19cce196_71.png)

以下是优化后的策略逻辑描述：

![](img/c82e918ac65af2319a1e4c1c19cce196_73.png)

```python
# 伪代码示例：优化后的大小盘轮动策略（引入空仓）
for 每个交易日 in 所有交易日:
    计算 大盘指数组 过去20日的平均涨跌幅
    计算 小盘指数组 过去20日的平均涨跌幅
    
    if 大盘平均涨幅 < 0 and 小盘平均涨幅 < 0:
        下个交易日空仓（持有现金）
    else:
        计算 每个候选指数 过去20个交易日的涨跌幅
        选择 涨跌幅最大的指数
        在下一个交易日开盘时，买入该指数对应的ETF
```

![](img/c82e918ac65af2319a1e4c1c19cce196_75.png)

我们运行一下代码来看看结果如何。程序运行结果如图所示，橙色曲线就是我们改进之后的一个结果。可以看到，相比对于原策略10.64的净值，改进后策略净值变成了12.69，它的确是让我们多赚钱。更重要的是，最大回撤由原来的51.1%，变成了26.8%，这个提升其实是巨大的。它让一个不可能拿来实盘的策略，变得基本上可以进一步研究去实盘了。

## 更多优化思路与扩展玩法 🧠

![](img/c82e918ac65af2319a1e4c1c19cce196_81.png)

而且我们可以尝试进一步的去优化。比如我们之前是各用了三个指数代替大小盘策略，那么我们是否可以增加更多的指数，一起来进行轮动呢？就比如说我们是看过去的一个月，也就是过去20天的涨跌幅，我们是不是可以把这边的20改成过去的五天、十天、四十天呢？你都可以自己去试一下。其实有了代码之后，这个试起来都非常的方便，代码里面的这些都是参数，你改一下这个参数，很快就可以得到相应的结果。

![](img/c82e918ac65af2319a1e4c1c19cce196_83.png)

并且如果你把思路开阔一点的话，可以把这个指数变成基金，那你是不是可以拿这个策略，在两个基金之间进行轮动呢？当然其实我是不建议的，因为基金换仓的成本太高了。或者有的同学他投资某马赛克市场，我们可以在之间做轮动，收益会非常的爆炸。

再来讲一个更专业的玩法。我们之前讲的都是一些现货，只能买入做多。但是如果我们能够交易股指期货的话，我们就可以构建多空的组合：我们发现大盘过去一段时间比小盘涨得多的时候，我们可以做多大盘同时做空小盘。那我们的收益就完全来自于两个指数之间的差异，然后我们整体的资金曲线画出来，就会非常的平稳。当然这个就比较复杂了，我们就不详细讲。

![](img/c82e918ac65af2319a1e4c1c19cce196_85.png)

## 总结 📚

![](img/c82e918ac65af2319a1e4c1c19cce196_87.png)

本节课中我们一起学习了如何构建一个基于市场周期的大小盘轮动量化策略。我们从市场现象中发现了“动量延续”的规律，并将其转化为一个简单的量化规则：每日持有过去20日涨幅最大的指数。

![](img/c82e918ac65af2319a1e4c1c19cce196_89.png)

通过历史回测，我们发现基础策略能获得显著超越指数的收益，但存在回撤过大的问题。于是我们进一步优化，引入了“双跌则空仓”的风控条件，有效降低了最大回撤，提升了策略的实用性。

![](img/c82e918ac65af2319a1e4c1c19cce196_91.png)

最后，我们还探讨了参数优化、资产扩展（如基金、加密货币）以及利用股指期货进行多空对冲等进阶思路。这个案例完整展示了从一个市场观察到形成策略、再到回测验证与迭代优化的量化投资基本流程。希望对你有所启发。
