上一节我们介绍了如何获取和预处理股票数据。本节中，我们来看看如何利用这些已知的指标数据，对股票进行综合打分与排名。

## 打分方法详解

以下是打分法的核心步骤：首先，我们需要为每个指标确定其优劣方向（即越大越好还是越小越好）；其次，将每个指标的数值映射到具体的分数；最后，汇总所有指标的分数得到总分，并据此进行排名。

### 1. 数据准备与方向确定

假设我们有一个股票池（例如沪深300成分股），并为每只股票计算了四个指标：A、B、C、D。根据先验知识，我们确定：
*   指标 **A**：越大越好。
*   指标 **B**：越大越好。
*   指标 **C**：越小越好（例如负债率）。
*   指标 **D**：越小越好（例如市值）。

每只股票的指标值都已通过归一化处理，范围在0到1之间。以下是两只示例股票的数据：

| 股票ID | 指标A | 指标B | 指标C | 指标D |
| :--- | :--- | :--- | :--- | :--- |
| id1 | 0.71 | 0.28 | 0.39 | 0.11 |
| id2 | 0.45 | 0.17 | 0.81 | 0.02 |

### 2. 设计打分区间与规则

接下来，我们需要为每个指标的数值区间分配分数。这里提供两种常见的思路：

**方法一：按固定区间划分**
将0到1的数值范围（或百分位）划分为若干个区间，并为每个区间赋予分数。

对于 **“越大越好”** 的指标（如A、B）：
*   `[0.8, 1.0]` -> 5分
*   `[0.6, 0.8)` -> 4分
*   `[0.4, 0.6)` -> 3分
*   `[0.2, 0.4)` -> 2分
*   `[0.0, 0.2)` -> 1分

对于 **“越小越好”** 的指标（如C、D）：
*   `[0.0, 0.2)` -> 5分
*   `[0.2, 0.4)` -> 4分
*   `[0.4, 0.6)` -> 3分
*   `[0.6, 0.8)` -> 2分
*   `[0.8, 1.0]` -> 1分

**方法二：按排名直接赋分**
例如，对于300只股票，直接将某个指标从大到小排序，排名第1的给300分，排名第300的给1分。这种方法更直接，避免了划分区间的主观性。

### 3. 计算单指标得分

我们以**方法一**为例，为示例股票计算各指标得分。

*   **股票 id1**:
    *   指标A (0.71): 属于 `[0.6, 0.8)` 区间，得 **4分**。
    *   指标B (0.28): 属于 `[0.2, 0.4)` 区间，得 **2分**。
    *   指标C (0.39): 属于 `[0.2, 0.4)` 区间。注意C是“越小越好”，因此得 **4分**。
    *   指标D (0.11): 属于 `[0.0, 0.2)` 区间。D是“越小越好”，因此得 **5分**。
    *   **总分** = 4 + 2 + 4 + 5 = **15分**。

*   **股票 id2**:
    *   指标A (0.45): 得 **3分**。
    *   指标B (0.17): 得 **1分**。
    *   指标C (0.81): 属于 `[0.8, 1.0]` 区间，是“越小越好”中的最差区间，得 **1分**。
    *   指标D (0.02): 属于 `[0.0, 0.2)` 区间，是“越小越好”中的最优区间，得 **5分**。
    *   **总分** = 3 + 1 + 1 + 5 = **10分**。

### 4. 汇总排名与选股

计算出所有300只股票的总分后，我们只需根据总分进行从高到低的排序。选择排名最高的前N只（例如前10只）股票，作为下一次调仓时买入或重点关注的对象。

## 整体策略流程梳理 🚀

基于上述打分法，我们可以梳理出一个完整的月度调仓策略流程。以下是该流程的核心步骤：

1.  **确定股票池**：首先，明确我们的选股范围。在本例中，我们基于**沪深300指数**的成分股作为初选股票池。
2.  **设置调仓周期**：定义一个定时调仓函数（例如每月执行一次）。在量化平台中，这个函数通常命名为 `rebalance`。
3.  **执行 `rebalance` 函数**：这是策略的核心，每次调仓时执行。
    *   **数据获取与预处理**：获取当前时刻股票池中所有股票的指定指标（A, B, C, D）数据。
    *   **因子打分**：按照前述的“越大越好”或“越小越好”规则，为每只股票的每个指标计算得分。
    *   **计算总分**：对每只股票，将其所有指标的得分相加，得到综合总分。
    *   **排序与选股**：根据总分对所有股票进行降序排序，选出排名前10的股票。
    *   **调仓操作**：卖出当前持仓中不在前10名单的股票，并买入新进入前10名单的股票，使投资组合与选股结果一致。

这个流程结构清晰，将复杂的多因子选股问题分解为可顺序执行的模块化步骤。接下来，我们将尝试用这种打分法构建一个简单的策略，并回测其历史表现。
