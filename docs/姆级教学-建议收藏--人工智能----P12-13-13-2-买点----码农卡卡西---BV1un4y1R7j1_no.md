## 计算短期与长期移动平均线

移动平均线是技术分析中常用的指标。我们将计算短期（例如42天）和长期（例如252天）的简单移动平均。

![](img/09228be5a5e32011ce8c2b5030ace0e0_36.png)

首先，定义短期和长期的窗口大小。

![](img/09228be5a5e32011ce8c2b5030ace0e0_38.png)

```python
short_window = 42
long_window = 252
```

![](img/09228be5a5e32011ce8c2b5030ace0e0_40.png)

然后，计算对应的移动平均线，并将结果添加到数据框中。

![](img/09228be5a5e32011ce8c2b5030ace0e0_42.png)

![](img/09228be5a5e32011ce8c2b5030ace0e0_44.png)

```python
stock_data_clean['SMA_Short'] = stock_data_clean['AAPL'].rolling(window=short_window).mean()
stock_data_clean['SMA_Long'] = stock_data_clean['AAPL'].rolling(window=long_window).mean()
```

![](img/09228be5a5e32011ce8c2b5030ace0e0_48.png)

## 标记买卖信号

![](img/09228be5a5e32011ce8c2b5030ace0e0_64.png)

![](img/09228be5a5e32011ce8c2b5030ace0e0_66.png)

交叉点可以转化为交易信号。当短期均线上穿长期均线时，被视为“黄金交叉”，是买入信号；反之，下穿时则为“死亡交叉”，是卖出信号。

![](img/09228be5a5e32011ce8c2b5030ace0e0_68.png)

![](img/09228be5a5e32011ce8c2b5030ace0e0_70.png)

我们可以在数据框中添加一个标志列来记录这种关系。

![](img/09228be5a5e32011ce8c2b5030ace0e0_72.png)

```python
stock_data_clean['Position'] = np.where(stock_data_clean['SMA_Short'] > stock_data_clean['SMA_Long'], 1, 0)
print(stock_data_clean[['AAPL', 'SMA_Short', 'SMA_Long', 'Position']].head())
```

![](img/09228be5a5e32011ce8c2b5030ace0e0_74.png)

其中，`Position`为1代表短期均线高于长期均线（潜在持有或买入区间），为0则代表低于（潜在卖出或空仓区间）。

![](img/09228be5a5e32011ce8c2b5030ace0e0_76.png)

## 策略思路：从被动持有到主动交易

![](img/09228be5a5e32011ce8c2b5030ace0e0_88.png)

上一节我们介绍了如何通过均线交叉识别买卖点，本节中我们来看看基于此的策略思路。

如果从最初的时间点买入并一直持有到最终时间点，其收益取决于股价的最终涨跌。这是一种被动的投资策略。

然而，如果我们利用均线交叉信号进行主动交易：
*   在“黄金交叉”（买入信号）出现时买入或持有。
*   在“死亡交叉”（卖出信号）出现时卖出。

理论上，这种策略可以帮助我们在市场下跌趋势形成前卖出，在上涨趋势启动前买入，从而可能获得比简单持有更高的收益，或规避部分下跌风险。接下来的课程中，我们将具体计算和比较这两种策略的回报率。
