## 函数是什么？

上一节我们通过一个发邮件的例子，初步感受了函数的作用。本节中我们来看看函数的正式定义和特性。

编程中的“函数”一词来源于数学，但两者概念不完全相同。在Python等编程语言中，函数有特定的含义。在英文中，不同语言对它的称呼不同：Basic语言称之为子程序（Subroutine），C语言称之为函数（Function），Java中称之为方法（Method）。在Python中，它被称为函数（Function）。

![](img/f4a1b96aa6a81c125e859b5296c3a526_3.png)

编程中函数的定义是：**将一组语句的集合，通过一个名字（函数名）封装起来。如果想执行这组语句，只需调用其函数名即可。**

![](img/f4a1b96aa6a81c125e859b5296c3a526_5.png)

想象一下上节课的例子，正是将一组发送邮件的语句，通过函数名`send_mail`包装起来。之后每次想发送邮件时，直接调用`send_mail()`就可以了。这个东西就叫函数。

![](img/f4a1b96aa6a81c125e859b5296c3a526_7.png)

## 函数的作用与特性

![](img/f4a1b96aa6a81c125e859b5296c3a526_9.png)

那么，函数具体有什么作用和特性呢？它的核心作用主要有以下三点：

![](img/f4a1b96aa6a81c125e859b5296c3a526_11.png)

以下是函数的三大核心特性：

1.  **减少重复代码**：这是函数最重要的功能。将需要重复使用的代码块定义成函数，可以避免在程序中多次编写相同的代码。
2.  **使程序变得可扩展**：当需求变更时，如果代码逻辑集中在函数内部，则只需修改函数定义一处，所有调用该函数的地方都会自动生效。例如，若要将发邮件功能改为群发，只需修改函数内部的实现逻辑。
3.  **使程序变得易维护**：道理与可扩展性类似。当需要修复bug或优化功能时，只需在函数内部进行修改，而无需在程序的各个角落寻找并修改重复的代码片段。这大大降低了维护成本。

![](img/f4a1b96aa6a81c125e859b5296c3a526_13.png)

## 函数的语法定义

![](img/f4a1b96aa6a81c125e859b5296c3a526_15.png)

了解了函数的概念与作用后，我们来看看在Python中如何定义一个函数。其基本语法结构如下：

![](img/f4a1b96aa6a81c125e859b5296c3a526_17.png)

```python
def function_name(parameters):
    """可选的函数文档字符串"""
    # 函数体：要执行的一组语句
    statement1
    statement2
    ...
    return value  # 可选的返回值
```

![](img/f4a1b96aa6a81c125e859b5296c3a526_19.png)

*   **`def`**： 关键字，是`define`的缩写，用于声明一个函数。
*   **`function_name`**： 函数名，遵循变量命名规则，用于调用函数。
*   **`parameters`**： 参数列表，位于括号内。参数是可选的，可以为空。参数是函数接收外部数据的接口。
*   **`:`**： 冒号，表示函数头部的结束，下一行开始是函数体。
*   **函数体**： 缩进的代码块，包含了函数要执行的所有语句。
*   **`return`**： 可选语句，用于从函数返回一个值。如果省略，函数默认返回`None`。

定义函数后，它只是被存储在内存中，并不会立即执行。必须通过**调用**来执行函数，调用方式是使用函数名后跟括号`()`。

## 动手定义一个函数

让我们根据上面的语法，动手定义一个简单的函数。

```python
def say_hi(name, age):
    print(f"Hello, my name is {name}.")
    print(f"I am {age} years old.")
    # 这里可以写更多行代码

![](img/f4a1b96aa6a81c125e859b5296c3a526_21.png)

# 函数定义完成后，必须调用才能执行
say_hi("Alex", 30)
```

![](img/f4a1b96aa6a81c125e859b5296c3a526_23.png)

运行这段代码，将会输出：
```
Hello, my name is Alex.
I am {age} years old.
```

在这个例子中：
1.  我们使用`def`关键字定义了一个名为`say_hi`的函数。
2.  它接受两个参数：`name`和`age`。
3.  函数体内有两行`print`语句。
4.  最后，我们通过`say_hi("Alex", 30)`调用了这个函数，并传入了具体的参数值。

你可以尝试在函数体内编写更多行代码，它们都会在函数被调用时按顺序执行。
