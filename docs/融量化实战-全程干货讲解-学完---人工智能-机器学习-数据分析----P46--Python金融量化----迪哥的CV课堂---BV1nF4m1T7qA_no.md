# Python金融量化：P46：工具包绘图展示

在本节课中，我们将学习如何使用专门的绘图工具包来可视化金融因子的分析结果，特别是IC值（信息系数）的时间序列图。通过图表，我们可以更直观地评估因子的有效性和稳定性。

上一节我们介绍了如何计算因子与收益率之间的相关系数（IC值）。本节中我们来看看如何将这些计算结果通过图表清晰地展示出来。

## 绘制IC值时间序列图

计算得到的IC值每天都会变化。为了更清晰地观察其趋势和特征，我们需要将其可视化。这里我们使用一个名为 `plotting` 的工具包来绘图。

![](img/3dd6b1c2bba11de243f8c65269a4f8ff_1.png)

以下是绘制IC值时间序列图的步骤：

![](img/3dd6b1c2bba11de243f8c65269a4f8ff_3.png)

1.  导入绘图工具包。
2.  调用 `plot_ic_ts` 函数。
3.  将计算好的IC序列数据传入该函数。

相关代码如下：
```python
# 假设 ic 是之前计算好的IC值序列
plotting.plot_ic_ts(ic)
```
执行上述代码后，系统会自动生成图表。

## 解读IC时序图

生成的图表包含以下关键信息：

*   **蓝色曲线**：代表每日实际的IC值走势，波动通常较大。
*   **绿色曲线**：代表IC值的滚动平均值（默认窗口为一个月），它平滑了每日波动，更能反映长期趋势。
*   **统计指标**：图表通常会标注IC序列的均值（mean）和标准差（std）。

在分析时，我们应主要关注**绿色曲线（滚动平均线）**。理想的因子其IC值应显著大于0，并且绿色曲线能展现出一定的上升或稳定趋势。如果绿色曲线长期在0值附近徘徊且数值较小，则说明该因子与收益率的相关性较弱，预测能力可能不足。

## 理解信息比率（IR）

图表中还会计算一个名为**信息比率**的指标，其公式为：
**IR = mean(IC) / std(IC)**
这个比率描述了IC值的稳定性。**IR值越大，说明因子的预测能力越稳定**。因为这意味着IC值的均值相对其波动（标准差）更高，结果更可靠。
