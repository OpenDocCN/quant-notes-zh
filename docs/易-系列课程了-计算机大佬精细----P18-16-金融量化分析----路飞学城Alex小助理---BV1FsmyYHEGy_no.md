### 方差与标准差

![](img/4cee9569949b92e1743c6934867c477b_7.png)

方差用于衡量一组数据的离散程度。计算方法是：每个数据点减去平均值，将差值平方，然后求和，最后除以数据点的个数。

![](img/4cee9569949b92e1743c6934867c477b_9.png)

![](img/4cee9569949b92e1743c6934867c477b_11.png)

**方差公式**：
`方差 = Σ(每个数据值 - 平均值)² / 数据个数`

标准差是方差的平方根，它同样表示数据的离散程度，但单位与原始数据相同。

**标准差公式**：
`标准差 = sqrt(方差)`

以下是计算过程的示例：
假设有一组数据 `[1, 2, 3, 4, 5]`，其平均值为3。
*   方差计算：`((1-3)² + (2-3)² + (3-3)² + (4-3)² + (5-3)²) / 5 = 2.0`
*   标准差计算：`sqrt(2.0) ≈ 1.414`

在NumPy中，`A.var()` 用于计算方差，`A.std()` 用于计算标准差。

标准差的一个应用是估计数据范围。对于许多数据分布（如正态分布），大约68%的数据会落在“平均值 ± 1倍标准差”的范围内，大约95%的数据会落在“平均值 ± 2倍标准差”的范围内。

### 其他统计函数

NumPy还提供了其他常用的统计函数：
*   `A.min()`：返回数组中的最小值。
*   `A.max()`：返回数组中的最大值。
*   `A.argmin()`：返回最小值所在位置的索引（下标）。
*   `A.argmax()`：返回最大值所在位置的索引（下标）。

## 随机数生成 🎲

了解了统计方法后，我们来看看NumPy的随机数生成功能。NumPy在 `np.random` 子模块下提供了丰富的随机数生成函数，其优势在于可以批量生成数组形式的随机数。

Python标准库的 `random` 模块常用函数有：
*   `random.random()`：生成一个[0.0, 1.0)之间的随机浮点数。
*   `random.randint(a, b)`：生成一个[a, b]之间的随机整数。
*   `random.choice(seq)`：从序列中随机选择一个元素。
*   `random.shuffle(seq)`：将序列顺序随机打乱。

NumPy的 `np.random` 模块重新实现了这些函数，并增加了直接生成数组的功能。

以下是NumPy中对应函数的使用示例：

**生成随机整数数组**
`np.random.randint(0, 10, size=10)` 会生成一个包含10个[0,10)区间随机整数的数组。
`np.random.randint(0, 10, size=(3, 5))` 会生成一个3行5列的二维随机整数数组。

**生成随机浮点数数组**
`np.random.rand(10)` 会生成一个包含10个[0.0, 1.0)区间随机浮点数的数组。

**从给定序列中随机选择（可重复）**
`np.random.choice([1, 3, 4, 5], size=10)` 会从给定列表中随机选择10次（允许重复），生成一个长度为10的数组。

**均匀分布随机数**
`np.random.uniform(2.0, 4.0, size=10)` 会生成10个在[2.0, 4.0)区间内均匀分布的随机浮点数。

![](img/4cee9569949b92e1743c6934867c477b_13.png)

`np.random.shuffle()` 函数与标准库用法一致，用于打乱数组顺序。
