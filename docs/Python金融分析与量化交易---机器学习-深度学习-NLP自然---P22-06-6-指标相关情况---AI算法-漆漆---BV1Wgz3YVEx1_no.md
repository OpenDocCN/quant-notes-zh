要展示两个指标之间的关系，最直接的方法是绘图。我们可以将它们画在同一张图上进行对比。

![](img/4f4f28c161751e26730e879d0cad7a0a_35.png)

以下是绘制双轴折线图的代码：
```python
data2.plot(secondary_y='VIX', figsize=(10, 6))
```
执行这段代码后，我们可以观察图表。但有时数据时间跨度太长（例如8年），会导致趋势不够清晰。为了更明显地观察关系，我们可以选取其中一段时间的数据。

![](img/4f4f28c161751e26730e879d0cad7a0a_43.png)

以下是选取特定时间段数据的代码：
```python
data_subset = data2['2010-01-01':'2012-12-31']
data_subset.plot(secondary_y='VIX', figsize=(10, 6))
```
现在，观察两年的数据，变化幅度更加清晰。我们可以发现，当蓝色线（SPX）下降时，绿色线（VIX）通常上升。这种负相关关系可以通过绘图直观地观察出来。

## 第三步：绘制散点图矩阵

![](img/4f4f28c161751e26730e879d0cad7a0a_57.png)

![](img/4f4f28c161751e26730e879d0cad7a0a_59.png)

除了折线图，我们还可以通过散点图来观察变量间的分布和两两关系。在绘制散点图前，我们可以先计算指标的日收益率，这能更好地反映变化。

以下是计算日收益率的代码：
```python
returns = np.log(data2 / data2.shift(1))
print(returns.head())
```
有了收益率数据后，我们可以绘制散点图矩阵来更全面地观察关系。

![](img/4f4f28c161751e26730e879d0cad7a0a_61.png)

![](img/4f4f28c161751e26730e879d0cad7a0a_63.png)

以下是绘制散点图矩阵的代码：
```python
pd.plotting.scatter_matrix(returns,
                           alpha=0.2,          # 设置点的透明度
                           diagonal='kde',     # 对角线绘制核密度估计图
                           figsize=(10, 10))
```
执行代码后，我们会得到一个矩阵图。对角线部分显示的是每个变量自身的分布（核密度估计图），而非对角线部分则是两两变量之间的散点图。

从散点图中，我们可以更清晰地看到SPX收益率与VIX收益率呈现负相关趋势，这与折线图的观察结果一致。核密度估计图是直方图的一种平滑版本，能更优雅地展示数据分布。

## 第四步：进行回归分析

散点图展示了关系，但如果我们能在图上添加一条回归线，或者计算出具体的回归系数，就能更精确地量化这种关系。

![](img/4f4f28c161751e26730e879d0cad7a0a_77.png)

![](img/4f4f28c161751e26730e879d0cad7a0a_79.png)

以下是进行简单线性回归分析的示例思路（需配合`scikit-learn`或`statsmodels`库）：
```python
# 示例：使用statsmodels进行OLS回归
import statsmodels.api as sm
X = returns['SPX'].dropna()
y = returns['VIX'].dropna()
X = sm.add_constant(X)  # 添加常数项
model = sm.OLS(y, X).fit()
print(model.summary())
```
通过回归分析，我们可以得到具体的回归方程和统计指标（如R平方、P值），从而科学地验证两个指标之间是否存在显著的负相关关系。
