## 数据变换与统计目标

上一节我们通过聚类方法对数据进行了分组。拿到分组结果后，接下来需要进行统计分析，目的是观察在特定的特征组合下，市场上涨或下跌的可能性哪一种更大。

![](img/2ee03d977a9e16a8f40707d91fa86721_16.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_18.png)

首先，我们需要对数据稍作变换。目前，我们将两个特征（如前一天的涨跌和前两天的涨跌）与结果（当天的涨跌）组合在了一起。为了更清晰地统计，我们需要对结果列进行操作。

![](img/2ee03d977a9e16a8f40707d91fa86721_20.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_22.png)

以下是关键的数据变换步骤：
```python
# 假设df是包含特征和结果的数据框，'result'是结果列
# 使用unstack方法将结果数据转换为更易统计的格式
pivot_table = df.groupby(['feature1', 'feature2'])['result'].size().unstack(fill_value=0)
```
执行这段代码后，我们会得到一个数据透视表。这个表格的行由两个特征的不同组合构成，列则是结果（例如1代表涨，-1代表跌），单元格内的数字表示该特征组合下对应结果出现的次数。这样，我们的统计目标就非常明确了：观察在不同特征变化时，最终的市场走势是如何变化的。

## 基于统计结果的策略制定

![](img/2ee03d977a9e16a8f40707d91fa86721_28.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_30.png)

数据准备好之后，我们就可以开始制定交易策略了。核心思想是：找出历史上出现频率最高的走势模式，并假设未来会重复这种模式。

![](img/2ee03d977a9e16a8f40707d91fa86721_32.png)

以下是制定策略的具体步骤：

![](img/2ee03d977a9e16a8f40707d91fa86721_34.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_36.png)

1.  **分析统计结果**：观察数据透视表，找出对于每一个特征组合，是上涨的次数多还是下跌的次数多。
2.  **制定规则**：根据统计结果制定买卖规则。例如，如果数据显示“前一天涨且前两天涨”时，第三天大概率下跌，那么我们的策略就可以是：当出现这种特征组合时，我们预测下跌，并采取相应的交易动作（如卖出或做空）。
3.  **生成交易信号**：根据上述规则，为历史数据中的每一天生成一个预测信号。

以下是根据统计结果生成预测信号的示例代码：
```python
import numpy as np

![](img/2ee03d977a9e16a8f40707d91fa86721_38.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_40.png)

# 假设data是原始数据框，包含‘day1’（前一天）和‘day2’（前两天）两个特征列
# 根据统计观察，我们制定规则：当day1和day2都为1（涨）时，预测为-1（跌），否则预测为1（涨）
data['position'] = np.where((data['day1'] + data['day2']) == 2, -1, 1)
```
这段代码中，`np.where`函数是一个条件赋值函数。它检查`day1`和`day2`的和是否等于2（即两者都为1）。如果条件为真，则在新的`position`列中赋值为-1（代表看跌）；否则赋值为1（代表看涨）。这个`position`列就是我们基于简单统计得出的交易信号。

## 策略效果评估

![](img/2ee03d977a9e16a8f40707d91fa86721_48.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_50.png)

策略制定完成后，我们必须评估其效果。我们将策略产生的预测信号与实际的市场走势进行比较，并计算其模拟收益。

![](img/2ee03d977a9e16a8f40707d91fa86721_52.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_54.png)

评估过程分为两步：

![](img/2ee03d977a9e16a8f40707d91fa86721_56.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_58.png)

1.  **预测准确率统计**：比较预测信号`position`与实际走势`direction`列，统计预测正确的天数。
    ```python
    # 统计预测正确的次数
    correct_predictions = (data['position'] == data['direction']).value_counts()
    print(correct_predictions)
    ```
    输出结果会显示`True`（预测正确）和`False`（预测错误）各自的数量。这可以直观地看出策略的预测能力。

![](img/2ee03d977a9e16a8f40707d91fa86721_60.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_62.png)

![](img/2ee03d977a9e16a8f40707d91fa86721_64.png)

2.  **模拟收益计算**：将策略信号应用到实际的价格变动上，计算累积收益，并与单纯跟随大盘的收益进行对比。
    ```python
    # 计算策略收益：策略信号 * 当日收益率
    strategy_returns = data['position'].shift(1) * data['returns']
    # 计算累积收益
    cumulative_strategy_returns = (1 + strategy_returns).cumprod()
    # 计算大盘基准累积收益
    cumulative_market_returns = (1 + data['returns']).cumprod()

    # 最终可以比较 cumulative_strategy_returns.iloc[-1] 和 cumulative_market_returns.iloc[-1]
    ```
    通过对比最终的策略累积收益和基准累积收益，我们可以判断这个简单的统计策略是否带来了超额收益，或者是否减少了损失。

在本节课的示例中，基于统计的策略预测正确率略高于50%，并且其模拟收益虽然也为负，但亏损幅度小于单纯持有大盘的亏损。这说明该统计策略在一定程度上起到了风险控制的作用。
