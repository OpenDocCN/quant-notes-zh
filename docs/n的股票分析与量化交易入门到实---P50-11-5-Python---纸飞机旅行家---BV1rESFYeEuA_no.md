## 索提诺比率 📉

![](img/fbb2d4a7a9a6b4cdc3763a5f32f23478_3.png)

首先给大家介绍索提诺比率。索提诺比率表示每承担一单位的下行风险，将获得多少超额回报。它与夏普比率非常相似。

它和夏普比率不一样的地方在于，夏普比率使用策略的总波动率，而索提诺比率使用下行的波动率。下行波动率表示策略收益在向下波动（即亏损时）的波动幅度。索提诺比率考虑的是最差情况，即下行波动率。

一般来说，下行波动率总是越小越好。索提诺比率越高，说明在承担相同单位下行风险时，能获得更高的超额回报率。索提诺比率可以看作是夏普比率在衡量股票风险时的一种修正方式。

![](img/fbb2d4a7a9a6b4cdc3763a5f32f23478_5.png)

它的算法与夏普比率基本一致，但分母的波动率不同。其公式为：
**索提诺比率 = (Rp - Rf) / σpd**
其中，**σpd** 代表策略的下行波动率。

## 信息比率 📈

![](img/fbb2d4a7a9a6b4cdc3763a5f32f23478_9.png)

接下来给大家介绍信息比率。信息比率衡量每单位超额风险所带来的超额收益。

![](img/fbb2d4a7a9a6b4cdc3763a5f32f23478_11.png)

它与夏普比率和索提诺比率有差异。夏普比率的分母是总风险，索提诺比率的分母是下行风险，而信息比率的分母是超额风险。超额风险指的是除了市场共有风险外，策略或所选股票特有的风险。例如，投资特定板块（如中概股、人工智能股）可能面临的泡沫风险就属于超额风险。

信息比率主要就是衡量这部分特有风险的。信息比率越大，说明该策略单位跟踪误差所获得的超额收益越高。我们的投资目标是在承担适当风险下，尽可能追求高的信息比率。

![](img/fbb2d4a7a9a6b4cdc3763a5f32f23478_13.png)

信息比率的算法也类似于夏普比率，其公式为：
**信息比率 = (Rp - Rb) / σt**
其中，**σt** 指的是策略与基准每日收益差值的年化标准差，这个差值就代表了超额风险。

## 策略波动率 📊

最后一个是策略的波动率。无论是夏普比率、索提诺比率还是信息比率，其分母都与波动率有关，只是具体算法不同。

首先介绍波动率的定义。波动率是衡量股票或衍生品价格波动范围的指标，即涨跌的震荡幅度。波动率常被用来衡量风险，一般认为高波动率代表着高风险，而高风险通常对应着高潜在回报。

![](img/fbb2d4a7a9a6b4cdc3763a5f32f23478_19.png)

接下来介绍波动率的算法。波动率常用 **σp** 表示，其计算公式为：
**σp = sqrt(250 / (N-1) * Σ(Rp_i - Rp_bar)^2)**
其中：
*   **250** 代表一年大约有250个交易日。
*   **N** 是策略执行的总天数。
*   **Rp_i** 是策略每日的收益率。
*   **Rp_bar** 是策略每日收益率的平均值。

这个公式计算的是策略收益率的年化标准差。



![](img/fbb2d4a7a9a6b4cdc3763a5f32f23478_21.png)

以下是波动率的回测查看方法。

波动率的回测查看同样是点击左侧的“波动率”选项，即可看到该策略的波动率指标。波动率越大，说明该策略的风险系数越高。



## 本章小结 ✅

本章是量化回测风险指标的第二部分内容。我们介绍了三个常见指标：索提诺比率、信息比率和策略波动率。

索提诺比率和信息比率都类似于夏普比率，用于表征单位风险所获得的超额收益情况，但具体算法不同。索提诺比率与夏普比率的最大差异在于分母，它采用的是下行波动率，即策略在亏损时的波动幅度。信息比率的分母则是超额风险，即策略特有的、超出市场普遍风险之外的风险，其算法是策略与基准每日收益差值的年化标准差。

策略波动率是衡量风险的基础指标，高波动率通常代表高风险。其算法是策略收益率年化标准差的计算。
