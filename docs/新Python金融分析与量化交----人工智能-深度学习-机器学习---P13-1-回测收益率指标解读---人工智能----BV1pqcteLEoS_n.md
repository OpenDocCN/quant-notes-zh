## 什么是策略评估指标？
当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以判断该策略是盈利还是亏损。然而，“盈利”或“亏损”本身是模糊的。我们需要更精确的指标来回答一系列问题：具体赚了多少？与大盘相比有何优势？承担了哪些风险？这些风险是否值得？与其他策略相比有何优劣？这些问题的答案，就是通过策略评估指标来获得的。

## 回测收益率
在众多指标中，我们首先介绍**回测收益率**。它是最基本、最重要的指标之一。

![](img/ed4c4969045a1c643a4e023aa413a5b7_9.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_21.png)

**回测收益率**衡量的是，在回测周期内，执行某个策略所获得的总收益相对于初始本金的比率。例如，你基于2015年至2020年的某只股票数据设计了一个策略，规定了买入和卖出的时机。回测收益率告诉你，最终你赚取的钱相对于最初投入的本金增加了多少百分比。它回答的是“折腾一番后，我的本金增值了多少”这个问题。

![](img/ed4c4969045a1c643a4e023aa413a5b7_23.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_11.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_13.png)

它的计算公式如下：

![](img/ed4c4969045a1c643a4e023aa413a5b7_15.png)

**公式：**
`回测收益率 = (期末资产价值 - 期初资产价值) / 期初资产价值`

![](img/ed4c4969045a1c643a4e023aa413a5b7_17.png)

或者等价于：
`回测收益率 = (期末资产价值 / 期初资产价值) - 1`

![](img/ed4c4969045a1c643a4e023aa413a5b7_19.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_21.png)

其中，`期末资产价值`可以理解为策略结束时的卖出总金额，`期初资产价值`则是策略开始时的投入本金。

![](img/ed4c4969045a1c643a4e023aa413a5b7_25.png)

这个指标直观地反映了策略的整体盈利能力，是我们评估策略表现时最先关注的数字。

![](img/ed4c4969045a1c643a4e023aa413a5b7_23.png)

## 用Python计算回测收益率
上一节我们介绍了回测收益率的概念，本节中我们来看看如何用Python代码进行计算和可视化。我们将使用一份示例股票数据来模拟策略回测后的每日资产价值序列。

首先，导入必要的工具包，并进行一些基础设置（如中文字体），以便后续画图。

![](img/ed4c4969045a1c643a4e023aa413a5b7_25.png)

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，确保图表能正常显示中文标签
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False
```

![](img/ed4c4969045a1c643a4e023aa413a5b7_27.png)

第一步，读取本地数据文件。这里我们假设数据文件包含了一列代表每日策略资产净值或股票收盘价的数据。

```python
# 读取数据，指定第一列为日期索引
data = pd.read_csv('your_stock_data.csv', index_col=0)
# 查看数据前几行
print(data.head())
```

![](img/ed4c4969045a1c643a4e023aa413a5b7_13.png)

这份数据包含了股票在不同日期的多个指标。为了演示，我们将其中的价格序列视为执行某个策略后，每日的资产总值。我们的目标是计算相对于第一天（期初）的每日收益率，并最终得到整个回测期间的总收益率。

以下是计算过程：

1.  **计算每日相对于期初的资产比值**：用每日的价格除以第一天的价格。
2.  **计算总回测收益率**：取最后一天的比值，然后减去1。

同时，我们可以绘制资产比值随时间变化的曲线，直观展示策略的增长过程。

![](img/ed4c4969045a1c643a4e023aa413a5b7_29.png)

```python
# 1. 计算每日资产相对于期初的比值
# 假设我们关注‘Close’这一列作为资产价值
price_series = data['Close']
initial_price = price_series.iloc[0] # 期初价格
relative_value = price_series / initial_price

![](img/ed4c4969045a1c643a4e023aa413a5b7_31.png)

# 将结果转换为DataFrame以便后续操作
relative_value_df = pd.DataFrame(relative_value, columns=['Relative_Value'])

![](img/ed4c4969045a1c643a4e023aa413a5b7_33.png)

# 绘制资产比值变化图
plt.figure(figsize=(12, 6))
plt.plot(relative_value_df.index, relative_value_df['Relative_Value'])
plt.title('策略资产相对价值变化曲线')
plt.xlabel('日期')
plt.ylabel('相对期初的价值比')
plt.grid(True)
plt.show()
```

![](img/ed4c4969045a1c643a4e023aa413a5b7_35.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_31.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_37.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_39.png)

上图展示了从期初（比值为1）开始，资产价值随时间的变化情况。接下来，我们计算最终的回测收益率。

```python
# 2. 计算总回测收益率
final_relative_value = relative_value_df.iloc[-1, 0] # 获取最后一个比值
backtest_return = final_relative_value - 1

![](img/ed4c4969045a1c643a4e023aa413a5b7_41.png)

# 将结果以更美观的格式展示
result_df = pd.DataFrame({
    '回测收益率': [backtest_return]
}, index=['最终结果'])
print(result_df)
```

![](img/ed4c4969045a1c643a4e023aa413a5b7_43.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_47.png)

![](img/ed4c4969045a1c643a4e023aa413a5b7_45.png)

执行以上代码，我们就得到了该策略在回测期间的总收益率。这个数值可能是正数（盈利），也可能是负数（亏损）。在本示例中，我们直接使用了原始股价数据，你可以将其想象为应用了某个买卖策略后产生的每日资产净值序列。
