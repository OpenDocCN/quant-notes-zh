## 核心概念与学习路径 🧭

了解目录结构后，我们来看看本课程可能涉及的核心领域。量化交易的核心是利用数学模型和计算机程序来制定交易策略。一个基础的策略回测流程可以用伪代码描述：

```python
# 策略回测简化框架
初始化资金和持仓
for 每一天 in 历史数据时间范围：
    根据当前市场数据（价格、指标等）生成交易信号
    if 产生买入信号：
        执行买入操作
    elif 产生卖出信号：
        执行卖出操作
计算最终收益、夏普比率等绩效指标
```

机器学习在其中扮演的角色，通常是利用历史数据训练模型，让模型来预测未来的价格走势或生成更优的交易信号。其核心公式可以抽象为寻找一个函数 `f`：

**`信号 = f(历史市场数据, 宏观经济指标, 新闻情感... )`**

我们的课程将从基础概念讲起，逐步深入到如何构建和评估这样的模型。
