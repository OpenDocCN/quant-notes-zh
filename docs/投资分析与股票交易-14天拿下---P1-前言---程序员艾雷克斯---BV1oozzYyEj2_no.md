# Python金融量化投资分析与股票交易：1：前言 🚀

在本节课中，我们将要学习Python金融量化投资的核心价值与基本思路。量化投资的核心在于将投资逻辑转化为代码，通过历史数据回测来验证策略的有效性，从而在实战前筛选出有潜力的策略，提高投资成功率。

---

投资最大的一个特点是能用代码来实现投资逻辑和想法，然后在历史数据上检验想法的优劣。如果在历史上表现很差，这个策略可以直接放弃，避免浪费资金和时间进行实盘测试。

如果策略在历史上表现良好，我们会尝试改进优化，然后投入实战使用。有历史数据的支撑，赚钱的概率会高很多。但很多人会遇到这样的问题：他们有一些策略和交易的想法，并想知道这些想法在历史回测中的表现。

然而，他们苦于自己不会写代码，或者找不到相关数据，以至于一直无法得到答案。

本节课中我们一起学习了量化投资的基本流程：**将投资想法转化为代码 -> 进行历史数据回测 -> 根据回测结果决定是否优化或放弃策略**。掌握编程技能是实现这一过程的关键，它能帮助你独立验证想法，在投资决策中占据主动。
