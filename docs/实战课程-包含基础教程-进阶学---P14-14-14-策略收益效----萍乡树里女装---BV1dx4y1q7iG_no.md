## 计算股价日增长率

上一节我们介绍了双均线策略的信号生成。本节中，我们来看看如何基于股价数据计算每日的增长率。

![](img/3c2275c539cd688b66f43ffba61df32d_3.png)

在量化分析中，我们通常使用对数收益率，因为它具有更好的统计特性（如可加性）。计算当前股价相对于前一天股价的对数增长率公式如下：

`log_return = log(price_t / price_{t-1})`

在Python的pandas中，我们可以通过以下步骤实现：

```python
# 假设‘close’是包含收盘价的列
data[‘return’] = np.log(data[‘close’] / data[‘close’].shift(1))
```

执行这段代码后，我们得到了一个名为 `return` 的新列，它代表了股价每日的对数增长率。

## 根据策略信号调整收益

![](img/3c2275c539cd688b66f43ffba61df32d_8.png)

![](img/3c2275c539cd688b66f43ffba61df32d_9.png)

![](img/3c2275c539cd688b66f43ffba61df32d_10.png)

我们已经有了市场本身的收益率。接下来，需要根据我们的双均线策略信号来调整这个收益。

我们的策略信号 `position` 列取值为1或-1。其中，1代表跟随市场原始走势（买入或持有），-1代表进行反向操作（例如，在策略判断为下跌趋势时做空）。因此，策略的每日收益率应为市场收益率乘以当日的仓位信号。同时，需要注意仓位信号应用于第二天的收益，因此也需要进行移位操作。

![](img/3c2275c539cd688b66f43ffba61df32d_12.png)

以下是计算策略收益的步骤：

![](img/3c2275c539cd688b66f43ffba61df32d_14.png)

1.  将仓位信号列 `position` 向后移动一天，以对齐收益率。
2.  将调整后的仓位信号与市场对数收益率相乘，得到策略的对数收益率。

![](img/3c2275c539cd688b66f43ffba61df32d_15.png)

对应的代码如下：

![](img/3c2275c539cd688b66f43ffba61df32d_17.png)

```python
# 计算策略收益率：市场收益率 * 前一天的仓位信号
data[‘strategy_return’] = data[‘return’] * data[‘position’].shift(1)
```

![](img/3c2275c539cd688b66f43ffba61df32d_19.png)

## 计算并比较累计收益

![](img/3c2275c539cd688b66f43ffba61df32d_20.png)

![](img/3c2275c539cd688b66f43ffba61df32d_21.png)

现在，我们有了两条收益率序列：原始市场收益率 (`return`) 和策略收益率 (`strategy_return`)。为了比较最终收益，我们需要将它们从对数形式还原为累计收益。

累计收益的计算过程如下：
1.  对收益率序列进行累加，得到总的对数收益。
2.  使用指数函数 `exp()` 将对数总收益还原为实际的总收益倍数（即1元本金最终变为多少元）。

![](img/3c2275c539cd688b66f43ffba61df32d_23.png)

以下是实现代码：

![](img/3c2275c539cd688b66f43ffba61df32d_25.png)

```python
# 计算累计对数收益
cum_log_return = data[[‘return‘， ’strategy_return‘]].cumsum()

# 将对数收益还原为实际收益倍数
total_return = np.exp(cum_log_return)

# 查看最终的总收益
final_return = total_return.iloc[-1]
print(final_return)
```

![](img/3c2275c539cd688b66f43ffba61df32d_27.png)

![](img/3c2275c539cd688b66f43ffba61df32d_28.png)

执行上述代码后，我们可以得到两个数字：一个代表从期初持有股票到期末的收益倍数，另一个代表使用双均线策略所获得的收益倍数。通过对比这两个数字，可以清晰地评估策略是否带来了超额收益。
