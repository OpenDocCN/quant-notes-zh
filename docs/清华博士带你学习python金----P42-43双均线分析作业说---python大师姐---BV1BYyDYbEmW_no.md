### 什么是移动平均线？
移动平均线是技术分析中常用的指标。对于每个交易日，计算前N天收盘价的平均值，将这些平均值连接起来，就形成了N日移动平均线。

**公式**：
`MA(N)_t = (Price_{t} + Price_{t-1} + ... + Price_{t-N+1}) / N`

常用的均线周期有5日、10日、30日、60日、120日和240日。其中：
*   5日、10日均线被视为**短期**均线，反应灵敏。
*   30日、60日均线被视为**中期**均线。
*   120日、240日均线被视为**长期**均线，走势更为平缓。

短期均线相对于长期均线波动更大、更敏感。

### 双均线策略的核心：金叉与死叉
双均线策略基于短期和长期两根均线的相对位置关系来产生交易信号。

*   **金叉**：当短期均线从下方**上穿**长期均线时，被视为**买入信号**。
*   **死叉**：当短期均线从上方**下穿**长期均线时，被视为**卖出信号**。

这个策略存在一定的滞后性，信号可能出现在价格趋势已经开始之后。

## 作业任务说明
以下是本次作业需要完成的几个具体步骤。

### 第一步：数据获取与均线计算
使用 `tushare` 包获取一支股票的历史行情数据。接着，使用 `pandas` 包计算该股票的5日均线（MA5）和60日均线（MA60）。

![](img/9e8efbef09ddb05f39b1bb87c6bfb0ca_1.png)

### 第二步：数据可视化
使用 `matplotlib` 包绘制三根曲线：每日收盘价线、5日均线（MA5）和60日均线（MA60）。通过图表直观观察均线的走势与交叉情况。

### 第三步：识别金叉与死叉日期
根据计算出的MA5和MA60数据，找出所有的金叉日期和死叉日期。

**判断逻辑**：
比较相邻两天的均线大小关系。
*   如果前一天满足 `MA5 < MA60`，而后一天满足 `MA5 > MA60`，则后一天发生的交叉为**金叉**。
*   如果前一天满足 `MA5 > MA60`，而后一天满足 `MA5 < MA60`，则后一天发生的交叉为**死叉**。

一种实现方式是使用循环遍历数据，比较每一天与后一天的关系。

![](img/9e8efbef09ddb05f39b1bb87c6bfb0ca_3.png)

### 第四步：模拟交易策略
假设从2010年1月1日开始，初始资金为10万元。遵循以下规则进行模拟交易：
1.  **买入**：出现**金叉**时，用全部可用资金买入股票，但必须以“手”（100股）为单位购买。
2.  **卖出**：出现**死叉**时，将持有的股票全部卖出。
3.  金叉与死叉会交替出现，因此策略是“买入 -> 卖出 -> 买入 -> 卖出……”的循环。

![](img/9e8efbef09ddb05f39b1bb87c6bfb0ca_5.png)

最终，计算到今天为止，这个策略的收益率如何，是盈利还是亏损。
