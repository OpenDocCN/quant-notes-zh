首先，我们来介绍NumPy模块的基本使用。NumPy模块在后期的学习中，主要是为pandas模块提供**数据源**。等我们学完pandas，你就会明白什么是数据源以及NumPy如何提供数据源。

![](img/6f684861d38d5da6aece479d0310a1d5_11.png)

简单来说，NumPy可以理解为一个**一维或多维的数组**。如果你有其他编程语言（如C、Java）的经验，应该知道数组的概念。如果只接触过Python，Python中有列表（list），你可以暂时将NumPy数组看作一种特殊的列表，或者将列表看作一种特殊的数组。它们都是一种**数据结构**，即一种可以装载数据的**容器**。

![](img/6f684861d38d5da6aece479d0310a1d5_13.png)

**NumPy数组**就是这个容器，它可以装载相关数据。暂时我们将NumPy模块视为一维或多维的数组。

*   **一维数组**：类似于列表 `[1, 2, 3]`
*   **二维数组**：类似于列表套列表 `[[1, 2, 3], [4, 5, 6]]`
*   **更高维数组**：以此类推

数组与列表是有区别的，稍后会介绍。目前，你只需知道NumPy数组主要装载**数值型数据**。因此，NumPy模块的重点在于**数值计算**。如果后期需要对一组数值进行运算，我们可以先将待计算的数据装入NumPy数组，再利用NumPy数组的特性进行运算。

此外，NumPy模块也是Python**科学计算库的基础**。这意味着NumPy内部已经封装好了各种科学计算所需的工具类。例如，如果需要对一组数据进行反三角函数运算，我们无需手动实现，直接调用NumPy中封装好的工具类即可。

**NumPy模块的核心要点**：
1.  它是一个容器，装载**数值型数据**，并侧重于**数值运算**。
2.  它是Python科学计算库的基础，提供了丰富的**科学计算工具类**。

## 创建NumPy数组（方式一：`np.array()`） 🛠️

介绍完NumPy模块后，接下来我们看看如何使用NumPy数组。首先，你需要创建它。

![](img/6f684861d38d5da6aece479d0310a1d5_15.png)

要使用NumPy，需要先导入模块。业界惯例是将其导入并别名为 `np`。

![](img/6f684861d38d5da6aece479d0310a1d5_17.png)

```python
import numpy as np
```

NumPy模块通常不需要单独安装，安装Anaconda集成环境时已包含。

导入模块后，我们来创建数组。第一种方式是使用 `np.array()` 函数。

```python
# 创建一维数组
arr1 = np.array([1, 2, 3])
arr1
```

在Jupyter Notebook的Cell中，直接输入变量名并按执行快捷键，即可显示变量的内容，无需`print`。

`np.array()` 函数的第一个参数 `object` 是数据源，即要放入数组的数据。第二个参数 `dtype` 表示数组元素的**数据类型**（不是数组本身的数据类型）。如果不指定 `dtype`，则采用数据源的原始类型。

同样，我们可以创建多维数组：

![](img/6f684861d38d5da6aece479d0310a1d5_19.png)

```python
# 创建二维数组
arr2 = np.array([[1, 2, 3], [4, 5, 6]])
arr2
```

![](img/6f684861d38d5da6aece479d0310a1d5_21.png)

返回的结果显示为两行三列，这是一个二维数组（一个平面）。你可以创建更高维度的数组，但最常用的是**一维、二维和三维数组**，因为更高维度不便于人类理解。

目前，我们已经掌握了一种创建NumPy数组的方式。

![](img/6f684861d38d5da6aece479d0310a1d5_23.png)

## 数组与列表的核心区别 ⚖️

![](img/6f684861d38d5da6aece479d0310a1d5_27.png)

掌握了初步的数组创建方式后，我们来看一个问题：数组与列表到底有什么区别？我们说数组可以是特殊的列表，列表也可以是特殊的数组，但它们必然存在区别。

让我们通过一个实验来观察：

![](img/6f684861d38d5da6aece479d0310a1d5_29.png)

```python
arr_mixed = np.array([1, 2.2, ‘three’])
arr_mixed
```

你会发现，尽管原始数据包含了整型、浮点型和字符串型，但最终数组中的所有元素都被强制转换成了**字符串类型**。这不会在Python普通列表中发生，因为列表允许元素类型不同。

在NumPy数组中，存储的数组元素类型**必须是统一的**。如果原始数据类型不同，NumPy会尝试将它们转换为同一类型。这里体现了数据类型的**优先级**：**字符串 > 浮点型 > 整型**。
*   如果存在字符串类型，所有元素转为字符串。
*   如果没有字符串但有浮点型，所有元素转为浮点型。
*   如果只有整型，则保持为整型。

![](img/6f684861d38d5da6aece479d0310a1d5_31.png)

这就是NumPy数组与Python列表的一个核心区别：**NumPy数组要求元素类型一致，而列表允许元素类型不同**。

![](img/6f684861d38d5da6aece479d0310a1d5_33.png)

![](img/6f684861d38d5da6aece479d0310a1d5_35.png)

## 创建NumPy数组（方式二：从外部数据读取） 🖼️

接下来，我们看看NumPy数组的另一种创建方式。这里我们有一个需求：将外部的一张图片读取并加载到NumPy数组中，然后尝试改变数组元素的数值，观察对原始图片的影响。

NumPy数组是一个容器，容器装载数据。不同的容器（如列表、字典、数组）对数据的组织形式不同，并且基于容器自身的特性，可以对数据进行不同形式的操作。我们重点学习NumPy数组这个容器，看它有哪些特性，能对数据做何种处理。

如何将外部图片装载到NumPy数组中呢？假设当前目录下有一张图片 `1.jpg`。图片本质上是由二进制数据（字节类型）构成的，而二进制数据也是数值型数据，因此可以装入NumPy数组。

![](img/6f684861d38d5da6aece479d0310a1d5_37.png)

![](img/6f684861d38d5da6aece479d0310a1d5_39.png)

在进行此操作前，我们需要先接触“三剑客”中的另一个模块：`matplotlib`。这里我们先初步使用它。

```python
import matplotlib.pyplot as plt

# 读取图片到NumPy数组
image_array = plt.imread(‘1.jpg’)
image_array
```

`plt.imread()` 读取图片后，会将数据存储在一个NumPy数组中。`image_array` 就是这个数组。打印它，你会发现返回的是一个**三维NumPy数组**（有三个中括号）。一维数组一个中括号，二维数组两个中括号，三维数组三个中括号。这个三维数组中存储的就是图片的像素数据。

![](img/6f684861d38d5da6aece479d0310a1d5_41.png)

![](img/6f684861d38d5da6aece479d0310a1d5_43.png)

我们可以显示这张图片：

![](img/6f684861d38d5da6aece479d0310a1d5_45.png)

```python
plt.imshow(image_array)
```

![](img/6f684861d38d5da6aece479d0310a1d5_47.png)

`plt.imshow()` 的作用是将NumPy数组进行可视化展示。由于这个数组存储的是图片数据，因此展示出来就是原始图片。

![](img/6f684861d38d5da6aece479d0310a1d5_49.png)

![](img/6f684861d38d5da6aece479d0310a1d5_51.png)

现在，`image_array` 中存储的是原始图片数据。如果我们对数组进行数值运算，例如让每个元素都减去100，会怎样呢？

```python
# 修改数组数据（对每个元素减去100）
modified_image_array = image_array - 100
plt.imshow(modified_image_array)
```

![](img/6f684861d38d5da6aece479d0310a1d5_53.png)

![](img/6f684861d38d5da6aece479d0310a1d5_55.png)

再次显示图片，你会发现图片效果发生了明显变化（通常变暗）。这个差异源于我们对图片的原始数据进行了统一的减法运算。

![](img/6f684861d38d5da6aece479d0310a1d5_57.png)

![](img/6f684861d38d5da6aece479d0310a1d5_59.png)

这揭示了图片处理软件（如美图秀秀）的原理：它们本质上是对图片的像素数据（存储在类似数组的结构中）进行特定的数学运算和算法处理。例如，“瘦脸”功能就是对脸部区域的像素数据应用特定算法进行变换。

![](img/6f684861d38d5da6aece479d0310a1d5_61.png)

![](img/6f684861d38d5da6aece479d0310a1d5_63.png)

因此，`plt.imread()` 是第二种创建NumPy数组的方式，它返回一个存储图片数据的**多维NumPy数组**。

## 创建NumPy数组（方式三：使用NumPy内置函数） 🔢

![](img/6f684861d38d5da6aece479d0310a1d5_65.png)

![](img/6f684861d38d5da6aece479d0310a1d5_67.png)

最后，我们介绍第三种创建NumPy数组的方式：使用NumPy提供的一系列内置函数。

![](img/6f684861d38d5da6aece479d0310a1d5_69.png)

![](img/6f684861d38d5da6aece479d0310a1d5_71.png)

以下是几个常用函数：

![](img/6f684861d38d5da6aece479d0310a1d5_73.png)

**1. `np.zeros()` 和 `np.ones()`**
这两个函数用于创建元素全为0或全为1的数组。参数 `shape` 指定数组的形状（维度）。

![](img/6f684861d38d5da6aece479d0310a1d5_75.png)

```python
# 创建一个3行4列，元素全为1的二维数组
arr_ones = np.ones((3, 4))
arr_ones

# 创建一个2行3列，元素全为0的二维数组
arr_zeros = np.zeros((2, 3))
arr_zeros
```

![](img/6f684861d38d5da6aece479d0310a1d5_77.png)

**2. `np.linspace()`**
此函数返回一个**一维等差数列数组**。参数 `start` 为起始值，`stop` 为结束值，`num` 为要生成的样本数量。

![](img/6f684861d38d5da6aece479d0310a1d5_79.png)

```python
# 生成从0到100之间均匀分布的20个数
arr_lin = np.linspace(0, 100, 20)
arr_lin
```

![](img/6f684861d38d5da6aece479d0310a1d5_81.png)

**3. `np.arange()`**
此函数类似于Python的 `range()`，返回一个**一维等差数列数组**。参数 `start` 为起始值，`stop` 为结束值（不包含），`step` 为步长。

![](img/6f684861d38d5da6aece479d0310a1d5_83.png)

```python
# 生成从10到50（不包含50），步长为2的数列
arr_arange = np.arange(10, 50, 2)
arr_arange
```

![](img/6f684861d38d5da6aece479d0310a1d5_85.png)

`np.linspace()` 和 `np.arange()` 都只能生成**一维数组**。

**4. `np.random.randint()`**
此函数用于生成指定范围内的**随机整数数组**。参数 `low` 为最小值，`high` 为最大值（不包含），`size` 为数组形状。

![](img/6f684861d38d5da6aece479d0310a1d5_87.png)

![](img/6f684861d38d5da6aece479d0310a1d5_89.png)

```python
# 生成一个5行3列，元素为0到100之间随机整数的二维数组
arr_rand = np.random.randint(0, 100, size=(5, 3))
arr_rand
```

![](img/6f684861d38d5da6aece479d0310a1d5_91.png)

至此，我们介绍了三种创建NumPy数组的方式：
1.  **`np.array()`**：从列表等序列创建。
2.  **`plt.imread()`**：从外部图片文件读取创建。
3.  **NumPy内置函数**：如 `zeros`, `ones`, `linspace`, `arange`, `random.randint` 等。

![](img/6f684861d38d5da6aece479d0310a1d5_93.png)

![](img/6f684861d38d5da6aece479d0310a1d5_95.png)

![](img/6f684861d38d5da6aece479d0310a1d5_97.png)

这些方式都可以创建NumPy数组，并且目前我们所创建的数组中存储的数据都是**数值型数据**。创建出NumPy数组后，我们才能进一步使用它进行各种数据操作和科学计算。
