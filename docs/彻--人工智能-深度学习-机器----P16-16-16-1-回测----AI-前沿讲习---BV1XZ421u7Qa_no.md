## 回测收益率

回测收益率是最直观的策略评估指标，它衡量了策略在整个回测周期内，相对于初始本金的总体盈利或亏损比例。

![](img/794c140353a56ce73f63781fee9dc41e_11.png)

![](img/794c140353a56ce73f63781fee9dc41e_13.png)

其计算公式如下：

![](img/794c140353a56ce73f63781fee9dc41e_15.png)

![](img/794c140353a56ce73f63781fee9dc41e_17.png)

**回测收益率 = (期末资产总值 - 期初资产总值) / 期初资产总值**

![](img/794c140353a56ce73f63781fee9dc41e_19.png)

![](img/794c140353a56ce73f63781fee9dc41e_21.png)

或者等价于：

**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

![](img/794c140353a56ce73f63781fee9dc41e_23.png)

其中，`期末资产总值`代表策略结束时的总价值，`期初资产总值`代表策略开始时的初始本金。

![](img/794c140353a56ce73f63781fee9dc41e_25.png)

接下来，我们通过Python代码来实际计算一个示例数据的回测收益率。

![](img/794c140353a56ce73f63781fee9dc41e_27.png)

首先，导入必要的工具包并加载数据。

```python
import pandas as pd
import matplotlib.pyplot as plt

# 设置中文字体（示例，根据实际环境调整）
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设第一列是日期
data = pd.read_csv('your_stock_data.csv', index_col=0, parse_dates=True)
# 假设我们关注‘close’列，即每日收盘价
price_series = data['close']
print(price_series.head())
```

假设我们的策略就是简单地持有这只股票，那么资产价值的变化就等同于股价的变化。我们可以计算每个时间点相对于期初的资产比值。

![](img/794c140353a56ce73f63781fee9dc41e_29.png)

![](img/794c140353a56ce73f63781fee9dc41e_31.png)

```python
# 计算每个时间点资产相对于期初的比值
relative_value = price_series / price_series.iloc[0]

![](img/794c140353a56ce73f63781fee9dc41e_33.png)

![](img/794c140353a56ce73f63781fee9dc41e_35.png)

# 绘制资产相对价值变化曲线
plt.figure(figsize=(10, 6))
relative_value.plot()
plt.title('资产相对价值走势')
plt.ylabel('相对价值 (期初=1)')
plt.xlabel('日期')
plt.grid(True)
plt.show()
```

![](img/794c140353a56ce73f63781fee9dc41e_37.png)

![](img/794c140353a56ce73f63781fee9dc41e_39.png)

最后，根据公式计算整个回测周期的总收益率。

```python
# 计算回测收益率： (期末价值/期初价值) - 1
backtest_return = (price_series.iloc[-1] / price_series.iloc[0]) - 1

![](img/794c140353a56ce73f63781fee9dc41e_41.png)

![](img/794c140353a56ce73f63781fee9dc41e_43.png)

# 将结果整理成DataFrame展示
result_df = pd.DataFrame({
    '回测收益率': [backtest_return]
}, index=['策略表现'])
print(result_df)
```

![](img/794c140353a56ce73f63781fee9dc41e_45.png)

![](img/794c140353a56ce73f63781fee9dc41e_47.png)

执行以上代码，我们将得到策略在整个回测期间的回测收益率。这个数值如果是正数代表盈利，负数代表亏损。

![](img/794c140353a56ce73f63781fee9dc41e_49.png)

本节课中我们一起学习了回测收益率。它是最基础的评价指标，直接反映了策略在历史测试中的总盈亏情况。然而，仅凭总收益率不足以全面评价策略，例如它没有考虑获得收益所花费的时间（年化收益）和所承担的风险（波动、回撤）。在接下来的章节中，我们将继续学习其他重要的评估指标。
