# Python虚拟货币量化交易：P1：云服务器环境搭建 🚀

在本节课中，我们将学习如何为Python虚拟货币量化交易搭建一个云服务器环境。整个过程包括购买并配置腾讯云轻量应用服务器、安装宝塔面板，以及设置Python编程环境。通过本教程，即使是初学者也能一步步完成环境的搭建。

## 购买与配置云服务器 💻

首先，推荐使用腾讯云的轻量应用服务器。在购买时，建议选择东京地区的服务器，型号选择入门型（约30元/月）即可满足需求。

![](img/fa50bce282ec2a8304ccfb4f06ece382_1.png)

购买完成后，进入腾讯云控制台。

![](img/fa50bce282ec2a8304ccfb4f06ece382_1.png)

![](img/fa50bce282ec2a8304ccfb4f06ece382_3.png)

## 登录与获取宝塔面板信息 🔐

在控制台中，选择“应用管理”并登录。登录后，系统会提供一段命令。复制并执行该命令，即可获取宝塔面板的登录地址、用户名和密码。

![](img/fa50bce282ec2a8304ccfb4f06ece382_3.png)

## 配置服务器防火墙 🛡️

在登录宝塔面板之前，需要先配置服务器的防火墙规则。宝塔面板默认使用8888端口，因此需要在防火墙中添加一条允许8888端口的规则。

![](img/fa50bce282ec2a8304ccfb4f06ece382_5.png)

![](img/fa50bce282ec2a8304ccfb4f06ece382_5.png)

添加规则后，即可使用之前获取的地址、用户名和密码登录宝塔面板。

![](img/fa50bce282ec2a8304ccfb4f06ece382_7.png)

![](img/fa50bce282ec2a8304ccfb4f06ece382_7.png)

## 安装与绑定宝塔面板 ⚙️

登录宝塔面板后，系统会提示绑定宝塔账号，按指引完成绑定即可。至此，宝塔面板的安装与基本配置就完成了。

上一节我们完成了服务器和面板的部署，本节中我们来看看如何搭建Python环境。因为量化交易主要使用Python，所以这是必不可少的一步。

## 下载Anaconda 📥

首先，访问Anaconda官网下载安装包。请注意选择适合的版本，教程中使用的版本可以确保流程顺利。

下载完成后，需要在服务器上创建一个目录来存放安装包。以下是操作步骤：

1.  进入服务器的 `/usr` 目录。
2.  在 `/usr` 目录下，新建一个名为 `anaconda` 的文件夹。

## 上传安装文件到服务器 📤

新建文件夹后，需要将本地下载好的Anaconda安装包上传到这个文件夹中。可以使用XFTP等文件传输工具进行上传。

![](img/fa50bce282ec2a8304ccfb4f06ece382_9.png)

XFTP工具可以从其官网下载。文件传输完成后，即可通过终端进行安装。

![](img/fa50bce282ec2a8304ccfb4f06ece382_9.png)

## 在终端中安装Anaconda 🖥️

打开服务器的终端，准备执行安装命令。以下是安装前的准备工作：

![](img/fa50bce282ec2a8304ccfb4f06ece382_11.png)

1.  获取root权限，方便后续操作：`sudo su`
2.  切换到Anaconda安装包所在的目录：`cd /usr/anaconda`

![](img/fa50bce282ec2a8304ccfb4f06ece382_11.png)

准备工作完成后，执行安装命令。安装命令格式如下：
```bash
bash Anaconda3-2023.03-1-Linux-x86_64.sh
```
请将命令中的文件名替换为你实际下载的Anaconda安装包名称。

## 配置环境变量 ⚙️

![](img/fa50bce282ec2a8304ccfb4f06ece382_13.png)

安装完成后，还需要配置环境变量，以便系统能识别Anaconda的命令。

1.  编辑用户的环境变量配置文件（例如 `~/.bashrc`）。
2.  在文件末尾添加以下内容（具体路径请根据你的安装目录调整）：
    ```bash
    export PATH="/root/anaconda3/bin:$PATH"
    ```
3.  保存文件后，执行以下命令使配置生效：
    ```bash
    source ~/.bashrc
    ```

![](img/fa50bce282ec2a8304ccfb4f06ece382_13.png)

执行完上述步骤后，可以输入 `conda --version` 来验证Anaconda是否安装成功。如果显示版本号，则说明环境已搭建完成。

![](img/fa50bce282ec2a8304ccfb4f06ece382_15.png)

![](img/fa50bce282ec2a8304ccfb4f06ece382_15.png)

本节课中我们一起学习了Python量化交易云服务器环境的完整搭建流程。我们首先购买并配置了腾讯云服务器，然后安装了宝塔面板用于可视化管理。接着，我们下载并安装了Anaconda，为Python开发提供了强大的环境支持。最后，通过配置环境变量，确保了Python和Conda命令可以正常使用。现在，你已经可以在云服务器上运行量化交易代码了。
