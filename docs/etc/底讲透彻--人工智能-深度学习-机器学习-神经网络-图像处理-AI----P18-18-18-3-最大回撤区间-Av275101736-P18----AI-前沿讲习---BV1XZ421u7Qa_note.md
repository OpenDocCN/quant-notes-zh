## 概述

在策略分析中，我们不仅关注最终收益，也需要评估策略的风险。最大回撤就是衡量投资过程中可能遭遇的最严重亏损的指标。

## 什么是最大回撤？

上一节我们介绍了策略的整体表现，本节中我们来看看如何衡量策略的风险。最大回撤描述的是在投资过程中，资产净值从最高点到之后最低点的最大跌幅。它衡量的是“最惨”的亏损阶段。

**公式**可以表示为：
`最大回撤 = max[(Pi - Pj) / Pi]`，其中 `Pi` 是时间 `i` 的资产价值，`Pj` 是 `i` 之后某个时间 `j` 的资产价值。

简单来说，它寻找的是整个投资周期内，从某个高点开始到后续低点，资产缩水最严重的那个区间。

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_5.png)

## 为什么最大回撤很重要？

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_7.png)

最大回撤是一个关键的风险指标。即使一个策略长期来看是盈利的，但如果其最大回撤过大，意味着投资者可能在过程中承受巨大的心理压力和资金风险。例如，2008年股市从高点大幅下跌，许多投资者承受了巨大亏损，这正是最大回撤所描述的风险。

## 如何计算最大回撤？

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_9.png)

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_11.png)

理解了概念后，我们来看看如何用Python进行计算。核心思路是遍历所有可能的“高点-低点”组合，找出跌幅最大的那个。

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_13.png)

以下是计算最大回撤的关键步骤：

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_15.png)

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_17.png)

1.  **计算累积最大值序列**：首先，我们需要计算到每一天为止的历史最高净值。这可以通过Pandas的 `cummax()` 函数实现。
    ```python
    # 假设 df[‘net_value’] 是资产净值序列
    cumulative_max = df[‘net_value’].cummax()
    ```
2.  **计算每日回撤**：然后，用这个历史最高值减去当日的实际净值，得到每日的潜在回撤值。
    ```python
    drawdown = cumulative_max - df[‘net_value’]
    ```
3.  **计算回撤比例**：将每日回撤值除以对应的历史最高值，得到回撤比例。
    ```python
    drawdown_pct = drawdown / cumulative_max
    ```
4.  **找出最大值**：最后，在所有回撤比例中找出最大值，即为最大回撤。
    ```python
    max_drawdown = drawdown_pct.max()
    ```

让我们通过一个简化的代码片段来演示整个过程：

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_27.png)

```python
import pandas as pd

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_29.png)

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_31.png)

# 假设这是我们的资产净值数据
data = {‘date’: pd.date_range(start=‘2023-01-01’, periods=10, freq=‘D’),
        ‘net_value’: [100, 102, 105, 103, 101, 99, 98, 102, 107, 110]}
df = pd.DataFrame(data).set_index(‘date’)

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_33.png)

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_35.png)

# 计算累积最大值
df[‘cumulative_max’] = df[‘net_value’].cummax()

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_37.png)

# 计算回撤和回撤比例
df[‘drawdown’] = df[‘cumulative_max’] - df[‘net_value’]
df[‘drawdown_pct’] = df[‘drawdown’] / df[‘cumulative_max’]

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_39.png)

# 找出最大回撤及其发生日期
max_dd_value = df[‘drawdown_pct’].max()
max_dd_date = df[‘drawdown_pct’].idxmax()

![](img/d152d5f3a7ad649eee27bd7b0a8ef7cc_41.png)

print(f“最大回撤为：{max_dd_value:.2%}”)
print(f“最大回撤发生在日期：{max_dd_date}”)
```
