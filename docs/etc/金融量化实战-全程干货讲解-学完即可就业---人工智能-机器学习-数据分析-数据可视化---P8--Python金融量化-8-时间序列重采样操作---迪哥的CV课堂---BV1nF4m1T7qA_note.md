# Python金融量化：8：时间序列重采样与窗口操作 📊

在本节课中，我们将要学习时间序列分析中的两个核心操作：**重采样**与**时间窗口操作**。重采样允许我们改变数据的时间频率，而时间窗口操作则用于计算滑动窗口内的统计指标。掌握这些技能对于金融数据的多维度分析至关重要。

## 重采样操作 🔄

![](img/f6b913d8587309c29993d34c5a4a8689_1.png)

![](img/f6b913d8587309c29993d34c5a4a8689_3.png)

上一节我们介绍了时间序列的基础处理，本节中我们来看看如何改变数据的时间频率，即重采样。

正常数据通常以天为单位进行统计。重采样操作允许我们将统计单位进行调整，例如，从按天统计转换为按周、按月或按年统计。这通过 `resample` 方法实现。

![](img/f6b913d8587309c29993d34c5a4a8689_5.png)

![](img/f6b913d8587309c29993d34c5a4a8689_7.png)

在重采样过程中，需要指定一个核心参数：新的时间频率单位。例如，可以指定以“周”（‘W’）为单位。同时，还需要指定在重采样后，如何聚合每个新时间窗口内的数据，例如计算均值、第一个值、中位数或其他统计指标。

![](img/f6b913d8587309c29993d34c5a4a8689_9.png)

![](img/f6b913d8587309c29993d34c5a4a8689_11.png)

以下是重采样的基本代码示例：

```python
# 假设 df 是一个以日期为索引的DataFrame
# 以周为单位进行重采样，并计算每周的均值
weekly_data = df.resample('W').mean()
print(weekly_data.head())
```

![](img/f6b913d8587309c29993d34c5a4a8689_13.png)

执行上述代码后，原始数据会被聚合到以周为单位的窗口中，并计算每个窗口内数据的平均值，从而得到一份新的时间序列。

我们不仅可以按周重采样，还可以按月（‘M’）、按年（‘Y’）或其他频率进行。例如，按月重采样的代码如下：

```python
monthly_data = df.resample('M').mean()
print(monthly_data.head())
```

![](img/f6b913d8587309c29993d34c5a4a8689_15.png)

![](img/f6b913d8587309c29993d34c5a4a8689_17.png)

在重采样时，有一个重要参数 `label` 需要注意。它决定了新时间窗口的标签（索引）是使用窗口的起始日期（‘left’）还是结束日期（‘right’）。

以下是 `label` 参数的使用示例：

![](img/f6b913d8587309c29993d34c5a4a8689_19.png)

![](img/f6b913d8587309c29993d34c5a4a8689_21.png)

```python
# 使用窗口的结束日期作为标签（默认行为）
monthly_data_right = df.resample('M', label='right').mean()
print(monthly_data_right.head())

![](img/f6b913d8587309c29993d34c5a4a8689_23.png)

![](img/f6b913d8587309c29993d34c5a4a8689_25.png)

# 使用窗口的起始日期作为标签
monthly_data_left = df.resample('M', label='left').mean()
print(monthly_data_left.head())
```

`label` 参数的选择取决于分析需求。在金融分析中，通常使用 `label=‘right’`（即用窗口结束日期作为标签），因为这更符合观察逻辑——例如，1月31日的数据代表的是整个1月份的统计结果。而使用 `label=‘left’` 时，12月31日的标签可能对应的是1月份的数据，这容易造成混淆。

## 时间窗口（滚动）操作 🪟

了解了如何改变数据频率后，我们来看看如何在固定的时间窗口内进行滑动计算，即时间窗口操作。

![](img/f6b913d8587309c29993d34c5a4a8689_27.png)

时间窗口操作，也称为滚动窗口或滑动窗口操作，其概念是：在时间序列上定义一个固定长度的窗口（例如10天），然后让这个窗口从序列起始位置开始，每次向后滑动一个单位（例如1天），并计算每个窗口内的统计指标。

![](img/f6b913d8587309c29993d34c5a4a8689_29.png)

以下是实现时间窗口操作的核心步骤：

![](img/f6b913d8587309c29993d34c5a4a8689_31.png)

1.  **数据准备**：首先，需要处理数据中的缺失值，因为缺失值会影响窗口统计的计算结果。
2.  **定义窗口**：使用 `rolling` 方法并指定窗口大小。
3.  **应用统计函数**：对每个窗口应用所需的统计函数，如 `min()`, `max()`, `mean()`, `std()` 等。

![](img/f6b913d8587309c29993d34c5a4a8689_33.png)

以下是完整的代码示例：

![](img/f6b913d8587309c29993d34c5a4a8689_35.png)

```python
# 1. 处理缺失值
df_clean = df.dropna()

![](img/f6b913d8587309c29993d34c5a4a8689_37.png)

# 2. 定义窗口大小，例如10天
window_size = 10

![](img/f6b913d8587309c29993d34c5a4a8689_39.png)

# 3. 计算滚动窗口的最小值
rolling_min = df_clean[‘column_name’].rolling(window=window_size).min()
print(rolling_min.head(15))
```

![](img/f6b913d8587309c29993d34c5a4a8689_41.png)

![](img/f6b913d8587309c29993d34c5a4a8689_43.png)

在初始的几个窗口中，由于数据点不足以填满整个窗口（例如前9天），计算结果会显示为 `NaN`。从第10个数据点开始，才会得到有效的窗口统计值。

![](img/f6b913d8587309c29993d34c5a4a8689_45.png)

我们可以同时对多个统计指标进行计算，并将结果添加到原始数据中，以便进行综合分析。

![](img/f6b913d8587309c29993d34c5a4a8689_47.png)

![](img/f6b913d8587309c29993d34c5a4a8689_49.png)

以下是为数据添加多个滚动统计指标的示例：

![](img/f6b913d8587309c29993d34c5a4a8689_51.png)

```python
# 计算并添加多个滚动统计指标
df_clean[‘rolling_min’] = df_clean[‘column_name’].rolling(window=window_size).min()
df_clean[‘rolling_max’] = df_clean[‘column_name’].rolling(window=window_size).max()
df_clean[‘rolling_mean’] = df_clean[‘column_name’].rolling(window=window_size).mean()
df_clean[‘rolling_std’] = df_clean[‘column_name’].rolling(window=window_size).std()

![](img/f6b913d8587309c29993d34c5a4a8689_53.png)

# 查看添加了统计指标后的数据
print(df_clean.tail())
```

![](img/f6b913d8587309c29993d34c5a4a8689_55.png)

![](img/f6b913d8587309c29993d34c5a4a8689_57.png)

执行以上操作后，数据框中会新增四列，分别代表每个时间点所在的前向窗口（包含该点及之前9个点）的最小值、最大值、均值和标准差。

![](img/f6b913d8587309c29993d34c5a4a8689_59.png)

**重要提示**：在进行任何窗口操作之前，务必先处理数据中的缺失值（`NaN`）。如果窗口内存在缺失值，大部分统计计算的结果也会是 `NaN`，这将导致大量数据无法使用。

![](img/f6b913d8587309c29993d34c5a4a8689_61.png)
