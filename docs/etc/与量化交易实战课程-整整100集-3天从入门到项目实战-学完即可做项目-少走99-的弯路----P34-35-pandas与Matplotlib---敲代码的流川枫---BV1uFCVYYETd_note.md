## pandas与Matplotlib的便捷结合

上一节我们介绍了`plot`函数及其相关用法，本节中我们来看看pandas库与Matplotlib库之间一个非常便捷的关联功能。

我们之前创建过DataFrame，有时我们希望将这个DataFrame直接输出为图表。例如，我们可以使用之前的一个股票数据文件`601318.csv`。

首先，我们查看一下这个文件的内容，并进行一些数据处理。

```python
# 假设已导入pandas为pd
df = pd.read_csv('601318.csv')
# 将‘date’列转换为日期时间对象，并设置为索引
df['date'] = pd.to_datetime(df['date'])
df.set_index('date', inplace=True)
```

这个数据集中可能包含一些字符串列或无用的序列。我们使用花式索引选出我们需要的四列价格数据：开盘价(`open`)、收盘价(`close`)、最高价(`high`)、最低价(`low`)。

```python
price_df = df[['open', 'close', 'high', 'low']]
```

现在，我们得到了一个只包含这四列价格的DataFrame。

## 直接绘制DataFrame

如果我想把这个DataFrame直接做成一个图表，有没有简便的方法呢？答案是肯定的。

我们可以直接在DataFrame对象上调用`.plot()`方法，然后使用`plt.show()`显示图表。

```python
price_df.plot()
plt.show()
```

运行这段代码，Matplotlib会很智能地直接生成图表。图表中，索引列（即我们设置的日期`date`）自动变成了X轴的坐标，并且智能地显示了如2008、2010、2012等年份标签。纵轴则是价格，四列数据被绘制成了四条线。

为什么看起来几乎像一条线呢？这是因为对于股票数据，每一天的开盘价、收盘价、最高价和最低价通常都非常接近，且数据量很大，导致四条线重叠在一起，难以分辨。图表中其实用不同颜色区分了它们，但在密集的数据下不够明显。如果想看得更清晰，可以在弹出的图表窗口中调整视图。

## 绘制Series对象

不仅DataFrame可以这样操作，Series对象也可以。Series是单列数据，调用`.plot()`方法后，绘制的就是一条线。

```python
# 例如，只绘制收盘价
df['close'].plot()
plt.show()
```

这是我们介绍的pandas库与Matplotlib库的一个便捷关系：**可以直接用DataFrame或Series对象调用`.plot()`方法来绘图**。
