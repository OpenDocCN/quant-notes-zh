## 数据准备与合并

![](img/40d766e036083661ddad18d9c0639651_9.png)

```python
df_merged = pd.merge(df1, df2, how='inner', on='姓名')
```

## 处理合并中的缺失值

![](img/40d766e036083661ddad18d9c0639651_45.png)

在合并操作中，可能会遇到缺失值。pandas提供了不同的合并方式（如`how='left'`、`how='right'`、`how='outer'`）来控制缺失值的出现。

![](img/40d766e036083661ddad18d9c0639651_47.png)

以下是不同合并方式的简要说明：
*   **`how='inner'`**：只保留两个数据集都有的键。
*   **`how='left'`**：保留左侧数据集的所有键，右侧缺失的值为NaN。
*   **`how='right'`**：保留右侧数据集的所有键，左侧缺失的值为NaN。
*   **`how='outer'`**：保留所有键，缺失的部分用NaN填充。

![](img/40d766e036083661ddad18d9c0639651_49.png)

```python
# 左合并示例
df_left_merge = pd.merge(df1, df2, how='left', on='姓名')
```

## 数据筛选

![](img/40d766e036083661ddad18d9c0639651_57.png)

合并数据后，我们经常需要根据条件筛选出特定的行。

![](img/40d766e036083661ddad18d9c0639651_59.png)

以下是几种常见的数据筛选方法：
*   **单条件筛选**：筛选出统计学成绩大于80的行。
    ```python
    df_filtered = df_merged[df_merged['统计学'] > 80]
    ```
*   **多条件筛选（与）**：筛选出统计学成绩为88且数学成绩为100的行。
    ```python
    df_filtered = df_merged[(df_merged['统计学'] == 88) & (df_merged['数学'] == 100)]
    ```
*   **多条件筛选（或）**：筛选出统计学成绩为85、88或90的行。
    ```python
    df_filtered = df_merged[df_merged['统计学'].isin([85, 88, 90])]
    ```
*   **区间筛选**：筛选出统计学成绩在85到100之间（含）的行。
    ```python
    df_filtered = df_merged[(df_merged['统计学'] >= 85) & (df_merged['统计学'] <= 100)]
    ```

## 数据排序

![](img/40d766e036083661ddad18d9c0639651_67.png)

我们还可以根据某一列的值对数据进行排序。例如，根据经济学成绩进行升序排列。

![](img/40d766e036083661ddad18d9c0639651_69.png)

```python
df_sorted = df_merged.sort_values(by='经济学')
```

![](img/40d766e036083661ddad18d9c0639651_71.png)

![](img/40d766e036083661ddad18d9c0639651_73.png)

若要降序排列，可以设置参数`ascending=False`。

![](img/40d766e036083661ddad18d9c0639651_75.png)

```python
df_sorted_desc = df_merged.sort_values(by='经济学', ascending=False)
```
