## 回测收益率回顾

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_4.png)

上一节我们介绍了回测收益率，它衡量了策略在整个回测周期内的总盈亏情况。其计算公式为：

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_6.png)

**最终价值 / 初始价值 - 1**

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_8.png)

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_10.png)

计算结果显示，不同策略的收益有正有负，这直观地说明了投资存在风险，并非所有策略都能稳定盈利。有些策略收益很高，实现了翻倍；而有些策略收益则相对较少。这个指标是策略分析中最重要的一环，用于评估策略的最终盈利能力。

## 年化收益率介绍

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_12.png)

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_14.png)

理解了策略的总收益后，本节我们来看看另一个关键指标：年化收益率。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_16.png)

你可能会经常听到“年化收益”这个词。回测收益率反映了整个周期（可能是数年）的总收益，但当向他人展示或比较不同时间长度的策略时，直接比较总收益并不公平。因此，我们需要一个标准化指标，即年化收益率，它将总收益折算为每年的平均收益，便于跨周期比较。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_18.png)

以下是年化收益率的计算公式：

**年化收益率 = (1 + 总收益率) ^ (年交易日数 / 回测总交易日数) - 1**

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_20.png)

其中：
*   **总收益率 (R)**：即我们之前计算的回测收益率。
*   **年交易日数 (M)**：通常取250或252天，代表一年的平均交易日数量。
*   **回测总交易日数 (N)**：你的回测数据所覆盖的总交易日数量。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_22.png)

在我们的示例中，回测数据共有730个交易日样本，大约对应3-4年。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_24.png)

## 使用Python计算年化收益率

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_26.png)

接下来，我们通过Python代码将上述公式实现。假设我们已经有了存储各策略回测收益率的Series `ret`。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_28.png)

```python
# 假设 ret 是一个Pandas Series，包含了各策略的回测收益率
# 例如：ret = pd.Series([-0.10, 1.13, 0.05, ...], index=[策略A， 策略B...])

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_30.png)

# 定义年交易日数和回测总交易日数
M = 250
N = len(data) # 假设data是回测期的日线数据DataFrame，共730条

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_32.png)

# 计算年化收益率
annual_ret = (1 + ret) ** (M / N) - 1

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_34.png)

# 将结果重命名为“回测年化收益”以便区分
annual_ret.name = ‘回测年化收益’

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_36.png)

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_38.png)

# 展示结果
print(annual_ret)
```

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_40.png)

执行代码后，我们会得到每个策略的年化收益率。与回测收益率对比，数值会有所不同。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_42.png)

例如，某个策略的回测总收益率为1.13（即113%），但其年化收益率可能约为0.2几（即20%+）。对于收益为负的策略，其年化亏损率也会是一个更小的负数。这两个指标共同提供了更全面的收益视角。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_44.png)

## 核心指标总结

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_46.png)

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_48.png)

本节课中我们一起学习了量化策略收益评估的两个核心指标。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_50.png)

1.  **回测收益率**：反映策略在整个测试周期内的**最终总盈亏**，是评估策略成败的绝对指标。
2.  **年化收益率**：将总收益标准化，转化为**每年的平均收益**，是进行跨时间周期策略比较和行业沟通的标准指标。

![](img/28aec50f13c7fbbfb0d5c2a47ce719bc_52.png)

在策略分析中，我们不仅要关注最终赚了多少（回测收益率），也要关注其按年折算的盈利能力（年化收益率），两者结合才能对策略绩效做出更准确的判断。
