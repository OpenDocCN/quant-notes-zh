## 概述
在本节课程中，我们将学习如何完成一个核心的选股方法。我们将把多个因子数据表进行拼接，计算每个股票的综合得分，并根据得分进行排序，最终筛选出排名前十的股票。整个过程将涉及Pandas库的数据合并、列操作、求和与排序等关键技能。

---

上一节我们介绍了如何为单个因子数据打分。本节中，我们来看看如何整合所有因子数据，并基于综合得分完成最终的股票筛选。

![](img/2fd857fb2e7b47ece97e29476f5dfe1c_4.png)

### 数据表拼接操作
首先，我们需要将处理好的多个因子数据表合并成一张大表。这可以通过Pandas的`concat`操作来实现。

以下是实现拼接的代码：
```python
# 假设 df1 和 df2 是两个需要合并的DataFrame
df_combined = pd.concat([df1, df2], axis=1)
```
在这段代码中，我们使用`pd.concat`函数，并指定`axis=1`参数，表示按列方向进行拼接。拼接完成后，我们为这个新的合并数据表指定一个名称，例如`df_rank`。

![](img/2fd857fb2e7b47ece97e29476f5dfe1c_6.png)

### 创建得分列并计算总分
现在，我们拥有了一张包含所有因子数据的大表，但还没有综合得分。接下来，我们需要新建一列来存放每个股票的总得分。

以下是创建得分列并计算总分的步骤：
1.  首先，在`df_rank`中新建一个名为“得分”的列，并初始化为0。
2.  然后，计算每个股票在所有因子上的得分总和。

以下是相应的代码实现：
```python
# 新建一列，初始值为0，行数与数据表相同
df_rank[‘得分’] = 0

# 假设因子数据列名为 ‘factor1‘, ’factor2‘, … ’factor6‘
# 计算这六列的和，并将结果赋值给‘得分’列
df_rank[‘得分’] = df_rank[[‘factor1‘, ’factor2‘, ’factor3‘, ’factor4‘, ’factor5‘, ’factor6‘]].sum(axis=1)
```
这段代码中，`sum(axis=1)`表示对每一行（即每一个股票）的指定列进行横向求和，从而得到该股票的综合得分。

### 根据总分排序并筛选前十
得到总分后，下一步是根据得分进行排序，并选出排名最高的股票。

以下是排序与筛选的步骤：
1.  使用`sort_values`方法，依据“得分”列进行降序排列（分数高的排前面）。
2.  从排序后的结果中，提取排名前十的股票代码。

以下是实现代码：
```python
# 按‘得分’列降序排序
df_sorted = df_rank.sort_values(by=‘得分‘, ascending=False)

# 获取排序后前十个股票的索引（即股票代码），并转换为列表
top_10_stocks = df_sorted.head(10).index.tolist()
```
在这段代码中，`ascending=False`参数指定了降序排序。`head(10)`用于获取前10行数据，`.index.tolist()`则将这10行对应的索引（股票代码）提取出来并转化为列表格式。

![](img/2fd857fb2e7b47ece97e29476f5dfe1c_8.png)

### 整合为选股函数
最后，我们将以上所有步骤整合到一个名为`get_stocks`的函数中。这个函数封装了从读取数据、打分、合并到最终排序筛选的全过程。

![](img/2fd857fb2e7b47ece97e29476f5dfe1c_10.png)

以下是`get_stocks`函数的逻辑概要：
1.  读取各因子数据。
2.  对每个因子数据进行标准化和打分。
3.  将所有因子的得分表拼接起来。
4.  计算每个股票的总得分并排序。
5.  返回得分排名前十的股票列表。

![](img/2fd857fb2e7b47ece97e29476f5dfe1c_12.png)

![](img/2fd857fb2e7b47ece97e29476f5dfe1c_14.png)

执行一次这个函数，就能得到基于当前所有因子数据计算出的、最值得关注的十只股票。

## 总结
本节课中我们一起学习了如何完成一个完整的选股方法。我们首先将多个因子数据表拼接成一张综合表，然后为每个股票计算综合得分，最后根据得分进行降序排序并筛选出前十名的股票。这个过程是量化选股策略的核心步骤，它将分散的因子分析结果整合成了一个可执行的股票列表。
