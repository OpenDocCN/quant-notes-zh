## 正态分布与极值概念

上一节我们介绍了其他去极值方法，本节中我们来看看基于统计理论的3Sigma方法。理解3Sigma方法，首先需要了解正态分布。

正态分布是一种常见的概率分布。如果理解起来比较麻烦，可以将其视为一种“正常”的分布。例如，在银行贷款场景中，银行多借或少借你少量金额是正常情况，但多借或少借你巨额款项则属于异常情况。

我们来看一个标准正态分布的示意图。图中有一条均值线，在标准正态分布中，均值为0。这可以类比为误差：银行贷款可能多借也可能少借，多借偏向右侧，少借偏向左侧。

在图中，靠近均值的区间（例如红色阴影区域）面积较大，代表这些事件发生的概率较高。当区间越往外扩展时，事件发生的概率就越低。

我们现在要处理的是去极值问题。极值可能落在哪里？它不太可能落在均值附近（红色区域），因为那是大概率发生的正常数据。极值一般落在远离均值的地方。离均值越远、偏差越大的数据点，越有可能是离群点。

## 3Sigma方法原理

那么，什么是三倍西格玛（3Sigma）？请看下图。

![](img/67188a99696400013c4dc1615dd01b23_1.png)

图中，`X` 代表均值。我们标记了均值加减一倍、两倍、三倍标准差（西格玛）的区间。可以发现，一旦选择的区间越大（例如加减三倍标准差），数据点落在这个区间之外的可能性就越小。

![](img/67188a99696400013c4dc1615dd01b23_1.png)

3Sigma方法的思路如下：我们有一份原始数据。假设可以对数据做某种变换，使其理论上服从高斯分布（即正态分布）。暂且不管如何变换，先假设它服从正态分布。

当我指定正负三倍标准差（`±3σ`）作为边界时，这个区间内包含了多少比例的数据？这个值在统计学中是已知的。

![](img/67188a99696400013c4dc1615dd01b23_3.png)

如图所示，`±3σ` 的区间包含了约99.7%的数据。于是，我们可以认为数据中最正常的99.7%作为有效数据，剩下的0.3%视为极值并进行规范化处理。这就叫做3Sigma方法。

上限和下限由什么决定？在高斯分布中，只需要计算两个值：均值（`μ`）和标准差（`σ`）。公式如下：
*   上限：`upper_bound = μ + n * σ`
*   下限：`lower_bound = μ - n * σ`
其中 `n` 通常取3。
