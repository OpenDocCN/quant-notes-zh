# Python量化金融课程：P1：课程介绍与招生信息 📢

## 课程概述

本课程由北京大学顾佳峰老师主讲，将于8月2日至4日在北京大学举办。课程旨在结合Python编程与量化金融实践，帮助学员从基础入门到实际应用。

课程设计注重理论与实践结合，内容前沿且实用性强。无论你是Python初学者，还是希望将Python应用于金融领域的从业者，本课程都能提供系统的学习路径。

---

## 课程内容安排

上一节我们介绍了课程的整体情况，本节中我们来看看具体的课程内容安排。课程内容分为三个主要部分，循序渐进。

![](img/62f78d38fe1b63b98f8fe0c8e66eeea7_1.png)

以下是课程的核心模块：

1.  **Python编程基础**
    *   面向Python基础较弱的学员，夯实编程基础。

2.  **Python统计模型**
    *   学习在金融分析中常用的统计方法与模型。

3.  **量化金融实战应用**
    *   将所学知识应用于财经数据分析，完成量化金融实践。

课程内容设计确保了学员能够从零开始，最终掌握使用Python进行量化金融分析的能力。

## 授课方式与证书

了解了师资力量后，我们来看看灵活的学习方式与课程收获。课程提供线上与线下两种参与模式。

以下是授课与证书详情：

*   **授课时间**：8月2日至4日（共三天）。
*   **授课形式**：
    *   线下：在北京大学校内上课。
    *   线上：通过同步直播参与，内容与线下完全一致。
*   **课程证书**：完成课程后可获得证书。此证书可作为升学（如保研、考博）、出国或求职时的补充材料，具有实用价值。
