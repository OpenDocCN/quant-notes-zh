## 求平均值

![](img/33b3b58cb128e1f5ca88bde08a19ed26_3.png)

上一节我们介绍了如何处理缺失值，本节中我们来看看如何计算数据的平均值。pandas中的`mean()`方法用于计算平均值。

对于一个DataFrame对象，`mean()`方法默认会对**每一列**分别计算平均值，并返回一个Series对象。例如，对于一个包含两列数据的DataFrame，`df.mean()`会返回一个长度为2的Series，每个元素对应一列的平均值。

**代码示例**：
```python
df.mean()
```

如果想按照行来计算平均值，可以通过设置参数`axis=1`来实现。

**代码示例**：
```python
df.mean(axis=1)
```

## 求和

与求平均值类似，pandas也提供了`sum()`方法用于求和。

以下是`sum()`方法的使用方式：
*   默认情况下，`df.sum()`会按列求和。
*   设置参数`axis=1`，可以按行求和。

**代码示例**：
```python
df.sum()        # 按列求和
df.sum(axis=1)  # 按行求和
```

## 数据排序

在数据分析中，经常需要对数据进行排序。pandas提供了两种主要的排序方式：按值排序和按索引排序。

### 按值排序

按值排序是指根据某一列或某一行具体的数值大小进行排序，使用`sort_values()`方法。

以下是按值排序的关键参数：
*   `by`：指定根据哪一列（或行）的值进行排序。
*   `ascending`：控制排序顺序，`True`为升序（默认），`False`为降序。
*   当排序的列中存在缺失值（NaN）时，包含NaN的行不会参与排序比较，统一被放置在结果的最后。

**代码示例**：
```python
# 按‘two’列的值升序排序
df.sort_values(by='two')
# 按‘two’列的值降序排序
df.sort_values(by='two', ascending=False)
```

### 按索引排序

按索引排序是指根据行索引（行标签）或列索引（列名）进行排序，使用`sort_index()`方法。

以下是按索引排序的说明：
*   对行索引排序：`df.sort_index()` 会按照A, B, C, D的顺序排列行。
*   对列索引排序：`df.sort_index(axis=1)` 会按照列名的顺序排列列。
*   同样可以使用`ascending`参数控制升序或降序。

**代码示例**：
```python
# 按行索引升序排序
df.sort_index()
# 按列索引降序排序
df.sort_index(axis=1, ascending=False)
```

## 其他通用函数
