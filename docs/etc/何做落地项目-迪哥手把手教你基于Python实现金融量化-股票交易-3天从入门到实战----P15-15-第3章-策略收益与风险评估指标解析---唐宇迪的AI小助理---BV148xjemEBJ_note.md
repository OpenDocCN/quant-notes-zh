## 回测收益率
回测收益率是最基础、最直观的评估指标。它衡量了策略在整个回测周期内，相对于初始本金所获得的总收益百分比。例如，一个4%的回测收益率意味着，相对于初始投入，策略最终赚取了4%的利润。

其计算公式如下：
**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

假设我们得到的是某只股票经过策略买卖信号处理后的每日资产净值序列。我们将其视为策略的每日净值。

以下是计算回测收益率和绘制累计收益曲线的步骤：

```python
# 1. 计算每日净值相对于期初净值的比值（即累计净值）
cumulative_return = data / data.iloc[0]

![](img/f31ae5644097cf6980a6e2ceae087f45_23.png)

# 2. 绘制累计收益曲线
cumulative_return.plot(figsize=(12, 6))
plt.title('策略累计收益曲线')
plt.xlabel('日期')
plt.ylabel('累计净值（期初为1）')
plt.grid(True)
plt.show()

# 3. 计算整个回测周期的总收益率
total_return = cumulative_return.iloc[-1] - 1
print(f"回测总收益率: {total_return.values[0]:.2%}")
```

## 其他关键评估指标
理解了最基本的收益指标后，一个完整的策略评估还需要考虑风险和收益的平衡。以下是几个常用的核心评估指标。

### 1. 年化收益率
回测收益率反映的是整个周期的总收益，但不同策略的运行周期长短不同，无法直接比较。年化收益率将收益标准化到年度维度，便于比较。

**年化收益率 = (1 + 总收益率)^(252 / 回测交易日数) - 1**
（假设一年有252个交易日）

### 2. 最大回撤
最大回撤衡量的是策略在历史上从任一峰值跌落到随后谷底的最大跌幅。它描述了投资者可能面临的最大亏损幅度，是评估策略风险的关键指标。

![](img/f31ae5644097cf6980a6e2ceae087f45_29.png)

**最大回撤 = (波峰价值 - 波谷价值) / 波峰价值**

![](img/f31ae5644097cf6980a6e2ceae087f45_31.png)

### 3. 夏普比率
夏普比率衡量的是策略每承受一单位总风险（波动率），能产生多少超额回报（超过无风险利率的收益）。该比率越高，说明策略的风险调整后收益越好。

![](img/f31ae5644097cf6980a6e2ceae087f45_33.png)

![](img/f31ae5644097cf6980a6e2ceae087f45_35.png)

**夏普比率 = (策略年化收益率 - 无风险利率) / 策略年化波动率**

![](img/f31ae5644097cf6980a6e2ceae087f45_37.png)

![](img/f31ae5644097cf6980a6e2ceae087f45_39.png)

### 4. 索提诺比率
索提诺比率与夏普比率类似，但它只考虑下行风险（即不好的波动），而不惩罚上行波动。对于更关注下跌风险的投资者，这个指标更有参考价值。

**索提诺比率 = (策略年化收益率 - 无风险利率) / 策略下行波动率**

![](img/f31ae5644097cf6980a6e2ceae087f45_41.png)

![](img/f31ae5644097cf6980a6e2ceae087f45_43.png)

### 5. 阿尔法与贝塔
- **贝塔**：衡量策略收益相对于市场基准（如沪深300指数）波动的敏感性。贝塔等于1表示与市场波动同步，大于1表示波动比市场大，小于1则比市场稳定。
- **阿尔法**：衡量策略超越市场基准的超额收益能力。正的阿尔法意味着策略有能力跑赢市场。
