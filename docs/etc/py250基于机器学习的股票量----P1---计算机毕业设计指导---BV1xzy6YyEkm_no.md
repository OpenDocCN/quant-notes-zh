## 数据获取与准备 📥

数据是机器学习项目的基石。对于股票量化分析，我们需要获取高质量的历史市场数据。

以下是几种常用的数据获取途径：

![](img/5ef670d2f21af8a4a9a5d7f6b19d74fe_3.png)

*   **免费公开接口**：例如 `yfinance` 库（雅虎财经）、`akshare` 库（AkShare）等，可以方便地获取股票历史行情。
*   **专业金融数据平台**：如Tushare、Baostock等，提供更规范、更稳定的数据服务。
*   **本地数据库**：对于高频或大量数据，可以建立本地数据库进行存储和管理。

获取到原始数据后，我们需要对其进行清洗和预处理，使其适合模型训练。

![](img/5ef670d2f21af8a4a9a5d7f6b19d74fe_4.png)

数据预处理通常包括以下几个关键步骤：

1.  **处理缺失值**：对于缺失的数据点，可以采用向前填充、向后填充或插值法进行处理。例如，在Pandas中可以使用 `df.fillna(method='ffill')`。
2.  **处理异常值**：识别并处理由于数据错误产生的异常价格或成交量。
3.  **数据标准化/归一化**：将不同尺度的特征数据转换到同一尺度，以加快模型收敛并提高性能。常用方法有Min-Max标准化和Z-Score标准化。
    *   **Min-Max标准化公式**：`X_scaled = (X - X.min) / (X.max - X.min)`
    *   **Z-Score标准化公式**：`X_scaled = (X - μ) / σ`，其中μ是均值，σ是标准差。
4.  **构建特征与标签**：从原始数据中构造特征（如移动平均线、相对强弱指数RSI等），并定义预测目标（标签），例如“第二天的涨跌”或“未来N日的收益率”。

![](img/5ef670d2f21af8a4a9a5d7f6b19d74fe_5.png)
