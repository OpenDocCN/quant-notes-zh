## 百分比堆积条形图

上一节我们介绍了堆积条形图，用于比较组内构成与组间总额。本节中我们来看看百分比堆积条形图。

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_9.png)

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_11.png)

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_13.png)

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_15.png)

百分比堆积条形图与普通堆积条形图的区别在于坐标轴单位。普通堆积条形图的刻度以具体数量为单位，而百分比堆积条形图的刻度以百分比为单位，每个条形总长度均为100%，用于清晰展示各组内部的构成比例关系。

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_17.png)

以下是创建百分比堆积条形图的步骤：
1.  选中需要绘制的数据区域。
2.  点击“插入”选项卡，在条形图区域选择“百分比堆积条形图”。

**核心概念**：图表中每个条形代表一个组别，条形的分段长度表示该类别在该组内的百分比占比。

通过此图，我们可以分析组内的构成关系。例如，可以观察到“北方”地区“底妆”品类的销售占比更高。但需要注意的是，由于各条形总长固定为100%，因此无法直接比较不同组别的销售总额大小。

## 坐标轴的高级调整

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_27.png)

图表坐标轴的刻度、范围和顺序都可以进行自定义调整，以适应不同的展示需求。

默认情况下，坐标轴的上下边界会根据数据系列的最大值和最小值自动设定。例如，一个销量图表，其刻度范围可能是从0到14万（即各组数据之和的最大值）。

我们可以手动调整这个范围：
*   **调整边界**：可以手动设置坐标轴的最大值和最小值，例如将最大值设为20万，或将最小值设为-2000。
*   **逆序刻度**：可以将坐标轴的刻度值顺序反转。
*   **添加次坐标轴**：当需要同时展示量级差异巨大的两组数据时，可以使用次坐标轴。

以下是使用次坐标轴的场景与步骤：
**应用场景**：当两组数据（如“眼妆销量”和“刷子销量”）单位或量级差异显著时，量级小的数据在图表中可能几乎无法显示。
**操作步骤**：
1.  先为两组数据创建簇状条形图。
2.  选中量级较小的数据系列（如“刷子”）。
3.  右键点击，选择“设置数据系列格式”。
4.  在系列选项中，将其绘制在“次坐标轴”上。
5.  分别调整主、次坐标轴的数据系列格式（如填充、重叠比例），确保两组数据都能清晰可见。

通过此功能，可以制作更高级的对比图表，例如左右对称的旋风图。其核心思路是：将主坐标轴范围设置为对称（如-8000到8000），将其中一个数据系列置于次坐标轴，并将次坐标轴也设置为相同的对称范围且进行“逆序刻度”处理，最后调整坐标轴标签位置即可。

## 散点图与趋势分析

接下来，我们探讨散点图以及如何利用它进行有预测能力的趋势分析。

散点图主要用于展示两个变量之间的关系。但仅仅画出数据点只是描述性统计。真正的趋势分析需要能够基于历史数据进行预测。

**核心概念**：预测的本质是找到变量之间的数学关系（规律）。例如，假设房价（Y）与面积（X）存在线性关系，可以用公式 **`Y = W * X + B`** 表示。其中，**`W`** 是斜率，**`B`** 是截距。一旦从历史数据中计算出 **`W`** 和 **`B`**，就可以预测新面积对应的房价。

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_29.png)

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_31.png)

那么，如何找到最合适的 **`W`** 和 **`B`** 呢？关键在于找到一条直线，使得所有历史数据点到这条直线的“距离”之和最小。这个“距离”即预测值与真实值的误差。通过最小化所有误差的平方和（即“损失函数”），利用数学方法（如求偏导数）可以解出最优的 **`W`** 和 **`B`**。这个过程称为**线性回归**。

在Excel中，我们可以轻松实现线性回归并得到预测线：
1.  根据两组数据（如“信用历史”和“信用分”）插入散点图。
2.  点击图表上的“+”号，添加“趋势线”。
3.  在趋势线设置中，选择“线性”。
4.  勾选“显示公式”，图表上就会显示回归方程 **`Y = W * X + B`** 的具体数值。

![](img/e8f14de74ca7a6bb9bb7ced6717ce8d5_33.png)

有了这个公式，就可以进行预测。虽然这仅是单变量线性回归，但它揭示了数据分析从描述现状到预测未来的关键跨越。在实际业务中，更复杂的多变量问题（如预测用户流失）会涉及更多因素和更复杂的模型，但其核心思想相通。

## 气泡图、雷达图、瀑布图与漏斗图

最后，我们快速了解几种特色图表及其应用场景。

**气泡图**：可视为散点图的增强版，用点的大小表示第三维数据。通过分别设置X轴、Y轴和气泡大小对应的数据系列，甚至可以配合颜色区分第四维（如不同类别），从而在一张图上展示多维信息。

以下是创建多系列气泡图的步骤：
1.  点击“插入”图表中的“气泡图”。
2.  右键图表，选择“选择数据”。
3.  逐一添加系列，分别设置每个系列的X值、Y值和气泡大小值。

**雷达图**：适用于展示多个维度的对比。例如，可以比较不同人员在“身份特质”、“行为偏好”等多项指标上的表现。
**创建方法**：选中数据，在“插入图表”的“所有图表”中选择“雷达图”。

**瀑布图**：用于展示一系列正值和负值对累计总数的贡献。常用于分析随时间变化的利润构成或项目进度。
**创建方法**：选中包含初始值、中间增减值和最终值的数据序列，直接插入“瀑布图”。

**漏斗图**：直观展示流程中各阶段的转化情况，是互联网行业分析用户行为路径、转化率的常用工具。
**创建方法**：选中表示各阶段数量的数据，插入“漏斗图”。

**核心思想**：这些图表不仅是可视化工具，背后对应着深刻的业务分析框架（如AARRR模型）。使用它们的关键在于将业务过程“颗粒化”（拆解为清晰、可量化的环节），从而实现精细化管理和过程监控。

## 组合图表

当需要在一个图表中组合使用不同类型的图表（如柱形图与折线图），且它们量纲差异很大时，可以使用组合图表。

**核心目的**：主要解决不同数据系列因量级差异过大而无法在同一标准坐标轴下清晰显示的问题。
**创建方法**：
1.  选中数据，点击“插入”选项卡中的“组合图”。
2.  为不同的数据系列分别指定图表类型（如系列A用簇状柱形图，系列B用折线图）。
3.  为量纲差异大的系列勾选“次坐标轴”。

本质上，组合图表是自动、便捷地实现了主次坐标轴的配合使用。
