## 打分法原理介绍

上一节我们介绍了如何获取和预处理因子数据。本节中，我们来看看如何根据这些因子数据为股票打分。

打分法的核心思想是：为每个因子设定评分规则，然后汇总所有因子的分数，得到每只股票的总分，最后根据总分进行排名和选股。

以下是打分的基本步骤：

1.  **确定因子方向**：首先明确每个因子是“越大越好”还是“越小越好”。例如，盈利能力指标通常越大越好，而负债率指标则越小越好。
2.  **划分区间并赋值**：将每个因子的数值范围划分为若干个区间。根据因子的方向，为每个区间赋予相应的分数。
    *   对于“越大越好”的因子：数值越大的区间，分数越高。
    *   对于“越小越好”的因子：数值越小的区间，分数越高。
3.  **计算单因子得分**：根据每只股票在该因子上的实际数值，确定其落入哪个区间，从而得到该因子的得分。
4.  **汇总总分**：将一只股票在所有因子上的得分相加，得到该股票的总分。
5.  **排序与选股**：对所有股票的总分进行排序，选择总分最高的若干只股票作为投资组合。

## 打分法实例演示

![](img/58435b3dab7674f47b20a84287d037ec_5.png)

假设我们有一个包含300只股票（ID1 到 ID300）的池子，并选取了四个因子：A、B、C、D。其中，A和B是“越大越好”的因子，C和D是“越小越好”的因子。

我们首先需要为每个因子的数值划分区间并设定分数。假设所有因子值都已归一化到 [0, 1] 区间。

以下是区间划分和分数设定的示例：

**对于“越大越好”的因子（如A、B）：**
*   数值在 [0.8, 1.0] 区间：得 **5分**
*   数值在 [0.6, 0.8) 区间：得 **4分**
*   数值在 [0.4, 0.6) 区间：得 **3分**
*   数值在 [0.2, 0.4) 区间：得 **2分**
*   数值在 [0.0, 0.2) 区间：得 **1分**

**对于“越小越好”的因子（如C、D）：**
*   数值在 [0.0, 0.2) 区间：得 **5分**
*   数值在 [0.2, 0.4) 区间：得 **4分**
*   数值在 [0.4, 0.6) 区间：得 **3分**
*   数值在 [0.6, 0.8) 区间：得 **2分**
*   数值在 [0.8, 1.0] 区间：得 **1分**

现在，我们有两支股票的因子数据：

*   **ID1**: A=0.71, B=0.28, C=0.39, D=0.11
*   **ID2**: A=0.45, B=0.17, C=0.81, D=0.02

根据上述规则进行计算：

**ID1 得分：**
*   因子A (0.71): 落入 [0.6, 0.8)，得 **4分**
*   因子B (0.28): 落入 [0.2, 0.4)，得 **2分**
*   因子C (0.39): 是“越小越好”因子，落入 [0.2, 0.4)，得 **4分**
*   因子D (0.11): 是“越小越好”因子，落入 [0.0, 0.2)，得 **5分**
*   **总分** = 4 + 2 + 4 + 5 = **15分**

**ID2 得分：**
*   因子A (0.45): 落入 [0.4, 0.6)，得 **3分**
*   因子B (0.17): 落入 [0.0, 0.2)，得 **1分**
*   因子C (0.81): 是“越小越好”因子，落入 [0.8, 1.0]，得 **1分**
*   因子D (0.02): 是“越小越好”因子，落入 [0.0, 0.2)，得 **5分**
*   **总分** = 3 + 1 + 1 + 5 = **10分**

计算完成后，ID1的总分（15分）高于ID2（10分）。在300只股票的池子中完成全部计算并排序后，我们可以选择总分排名前10的股票。

除了划分区间的方法，也可以采用直接排序打分法：例如，对于300只股票，将某个“越大越好”的因子从大到小排序，排名第1的给300分，排名第300的给1分。核心目标是得到一个可比较的总分用于排名。

## 整体策略流程梳理 🚀

![](img/58435b3dab7674f47b20a84287d037ec_7.png)

![](img/58435b3dab7674f47b20a84287d037ec_9.png)

理解了打分法后，我们来看整个量化策略的实现流程。这个流程将指导我们后续的代码编写。

以下是策略的核心步骤：

1.  **确定股票池**：首先，需要确定选股的范围。在本例中，我们使用**沪深300指数**成分股作为初始股票池。
2.  **设置调仓周期**：确定策略调仓再平衡的频率。常见的频率是每月或每季度。我们将设置一个定时器，在每月特定的交易日触发调仓函数。
3.  **实现调仓函数**：这是策略的核心函数（例如 `rebalance`）。在该函数中，我们将执行以下操作：
    *   **数据获取与预处理**：读取当前股票池中所有股票的选定因子数据（例如A、B、C、D），并进行必要的清洗和标准化。
    *   **因子打分**：应用上述打分法，为每只股票的每个因子计算得分。
    *   **汇总与排序**：汇总每只股票的所有因子得分，得到总分，然后按总分从高到低排序。
    *   **生成交易清单**：选择总分排名前10的股票，作为本期要调入或持有的投资组合。同时，卖出不在前10名中的现有持仓。
4.  **回测与评估**：将上述逻辑在历史数据中运行，评估策略的收益、风险等表现指标。

这个流程结构清晰，将选股逻辑封装在每月执行的函数中，是量化策略中常见的实现模式。
