## 注意事项详解

![](img/224f32fc39680ad88696e84d4162f694_7.png)

![](img/224f32fc39680ad88696e84d4162f694_9.png)

以下是创建数据透视表前，必须检查源数据的五个关键点。

![](img/224f32fc39680ad88696e84d4162f694_11.png)

### 1. 列标题不能为空

![](img/224f32fc39680ad88696e84d4162f694_13.png)

![](img/224f32fc39680ad88696e84d4162f694_15.png)

列标题是数据透视表识别和分组字段的依据。如果标题单元格为空，Excel会自动将下方第一行数据误认为标题，导致整个数据透视表结构混乱。

**错误演示**：假设“车型”列标题为空，创建数据透视表后，字段列表中将出现“华北”、“北京”等本应是数据的值作为字段名，完全无法进行正确分析。

**正确做法**：确保数据区域第一行的每个单元格都有明确的列标题。

### 2. 列标题不能重复

![](img/224f32fc39680ad88696e84d4162f694_17.png)

数据透视表依赖唯一的列标题来区分不同字段。如果存在重复的标题（例如，两列都命名为“品牌”），Excel会自动添加“品牌2”这样的后缀，造成混淆，让他人难以理解数据含义。

![](img/224f32fc39680ad88696e84d4162f694_19.png)

**错误演示**：将“车型”列也改名为“品牌”。创建透视表后，字段列表中会出现“品牌”和“品牌2”，用户无法直观判断哪个字段对应原始数据中的哪一列。

![](img/224f32fc39680ad88696e84d4162f694_21.png)

![](img/224f32fc39680ad88696e84d4162f694_23.png)

**正确做法**：为每一列设置唯一且含义清晰的标题。

### 3. 数据区域不能存在空行

![](img/224f32fc39680ad88696e84d4162f694_25.png)

![](img/224f32fc39680ad88696e84d4162f694_27.png)

数据区域中的空行会被视为数据的终止。如果源数据中间存在完全空白的行，数据透视表将只读取空行之上的数据，导致数据丢失。

![](img/224f32fc39680ad88696e84d4162f694_29.png)

![](img/224f32fc39680ad88696e84d4162f694_31.png)

**错误演示**：在数据中间插入一个空行，创建的数据透视表将不包含空行以下的数据。

**正确做法**：确保用于创建透视表的连续数据区域中没有任何空行。

### 4. 避免使用合并单元格

![](img/224f32fc39680ad88696e84d4162f694_33.png)

![](img/224f32fc39680ad88696e84d4162f694_35.png)

合并单元格会破坏数据的规整性。在数据透视表中，只有合并单元格左上角那个单元格的值会被识别，其他被合并的单元格内容会被视为空值。

**错误演示**：将同一品牌的多行数据合并。创建透视表后，该品牌的总销量会被拆分为两部分：一部分是合并单元格左上角的值（被正确归类），另一部分则被归入“（空白）”项中。

**正确做法**：在用于分析的明细数据表中，避免使用合并单元格。如需批量取消合并并填充内容，可使用以下步骤：
1.  取消所有合并单元格。
2.  选中该列数据，按 `F5` 键打开“定位”对话框，选择“定位条件” -> “空值”。
3.  在活动单元格中输入公式 `=↑`（即等于上方单元格），然后按 `Ctrl + Enter` 批量填充。

### 5. 保持数据类型一致

同一列中的数据应保持相同的数据类型（如全部为数字、文本或日期）。如果一列中混杂了文本格式的数字，数据透视表在求和时可能会将其忽略或进行计数，导致计算结果错误。

**错误演示**：将“销量”列中的某些数字设置为文本格式。创建透视表对销量求和时，这些文本数字不会被计入总和，导致结果偏小。

![](img/224f32fc39680ad88696e84d4162f694_37.png)

**正确做法**：确保数值列格式统一。可以将整列数据转换为数字格式，或使用“数据”选项卡下的“分列”功能进行批量格式转换。

![](img/224f32fc39680ad88696e84d4162f694_39.png)

## 课程总结

![](img/224f32fc39680ad88696e84d4162f694_41.png)

本节课我们一起学习了创建数据透视表的五项重要注意事项：
1.  **列标题**需**唯一**且**非空**。
2.  **数据区域**应**连续**、**无空行**。
3.  **避免合并单元格**，保持数据粒度。
4.  确保**同列数据类型一致**。
遵循这些原则，可以保证数据透视表的基础数据干净、规范，从而快速、准确地生成所需的汇总分析报告，为后续的数据洞察打下坚实基础。下一节，我们将开始探索数据透视表的常用功能与高级技巧。
