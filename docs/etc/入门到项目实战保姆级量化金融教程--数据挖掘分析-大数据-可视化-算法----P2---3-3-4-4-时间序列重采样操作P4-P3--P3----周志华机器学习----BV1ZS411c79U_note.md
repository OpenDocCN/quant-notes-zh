## 重采样操作 🔄

![](img/cb78d41f73524cdf942bf5861b1db079_10.png)

上一节我们介绍了时间序列的基本处理。本节中，我们来看看如何改变数据的时间频率，这个过程称为**重采样**。

![](img/cb78d41f73524cdf942bf5861b1db079_12.png)

原始数据通常以天为单位。我们可以将其调整为以周、月或年为单位进行统计。这通过 `resample` 方法实现。重采样时，需要指定新的时间频率单位，并选择聚合数据的方式（如计算均值、第一个值或中位数）。

以下是重采样的基本代码结构：
```python
# 假设 df 是一个以日期为索引的DataFrame
# 以周为单位进行重采样，并计算每周的均值
weekly_data = df.resample('W').mean()
```

![](img/cb78d41f73524cdf942bf5861b1db079_14.png)

例如，将日数据重采样为周数据后，系统会按周（7天一个窗口）聚合数据。默认情况下，结果标签（索引）使用每个时间窗口的**右边界**（最后一天）日期。

![](img/cb78d41f73524cdf942bf5861b1db079_16.png)

我们还可以通过 `label` 参数控制结果标签：
```python
# 使用时间窗口的左边界（第一天）作为标签
weekly_data_left = df.resample('W', label='left').mean()
# 使用时间窗口的右边界（最后一天）作为标签，这是默认行为
weekly_data_right = df.resample('W', label='right').mean()
```
选择 `label='left'` 或 `label='right'` 取决于分析需求。在金融场景中，使用右边界（`'right'`）通常更符合逻辑，因为它代表该时间段结束时的汇总情况。

## 时间窗口（滑动窗口）操作 🪟

![](img/cb78d41f73524cdf942bf5861b1db079_20.png)

![](img/cb78d41f73524cdf942bf5861b1db079_22.png)

接下来，我们探讨**时间窗口操作**，也称为滑动窗口分析。这与重采样不同，它是在一个固定大小的窗口上滑动，并计算每个窗口的统计量。

![](img/cb78d41f73524cdf942bf5861b1db079_24.png)

![](img/cb78d41f73524cdf942bf5861b1db079_26.png)

假设我们有100天的数据。我们可以定义一个大小为10天的窗口：
*   第一个窗口：第1天到第10天。
*   第二个窗口：第2天到第11天（窗口向后滑动1天）。
*   第三个窗口：第3天到第12天。
*   以此类推...

![](img/cb78d41f73524cdf942bf5861b1db079_28.png)

![](img/cb78d41f73524cdf942bf5861b1db079_30.png)

![](img/cb78d41f73524cdf942bf5861b1db079_32.png)

这使我们能够计算一系列连续的统计指标，例如过去10天的移动最小值、最大值或标准差。

![](img/cb78d41f73524cdf942bf5861b1db079_34.png)

![](img/cb78d41f73524cdf942bf5861b1db079_36.png)

![](img/cb78d41f73524cdf942bf5861b1db079_38.png)

以下是实现滑动窗口操作的步骤：

![](img/cb78d41f73524cdf942bf5861b1db079_40.png)

首先，**必须对数据进行预处理，处理缺失值**，否则窗口计算会产生大量 `NaN` 结果。
```python
# 处理缺失值
df_cleaned = df.dropna()
```

![](img/cb78d41f73524cdf942bf5861b1db079_42.png)

![](img/cb78d41f73524cdf942bf5861b1db079_44.png)

![](img/cb78d41f73524cdf942bf5861b1db079_46.png)

然后，使用 `rolling` 方法创建窗口并计算统计量。以下代码计算每个10天窗口的最小值：
```python
window_size = 10
# 对某一列进行窗口计算
rolling_min = df_cleaned['column_name'].rolling(window=window_size).min()
```

![](img/cb78d41f73524cdf942bf5861b1db079_48.png)

![](img/cb78d41f73524cdf942bf5861b1db079_50.png)

我们可以同时为数据添加多个窗口统计列：
```python
# 为清洗后的数据添加多个窗口统计列
df_cleaned['min_10'] = df_cleaned['column_name'].rolling(window=window_size).min()
df_cleaned['max_10'] = df_cleaned['column_name'].rolling(window=window_size).max()
df_cleaned['mean_10'] = df_cleaned['column_name'].rolling(window=window_size).mean()
df_cleaned['std_10'] = df_cleaned['column_name'].rolling(window=window_size).std()

![](img/cb78d41f73524cdf942bf5861b1db079_52.png)

![](img/cb78d41f73524cdf942bf5861b1db079_54.png)

# 查看结果（例如最后几行）
print(df_cleaned.tail())
```
这样，我们就得到了每个时间点基于其**前10天（包含当前点）** 窗口计算出的最小值、最大值、均值和标准差。

本节课中我们一起学习了：
1.  **时间序列重采样**：使用 `resample()` 方法改变数据的时间频率（如日变周），并通过 `label` 参数控制聚合结果的标签。
2.  **时间窗口操作**：使用 `rolling()` 方法创建滑动窗口，计算连续时间窗口内的统计指标（如移动平均）。**进行窗口计算前，务必处理数据中的缺失值**。
