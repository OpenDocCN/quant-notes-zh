## 章节 1：策略回顾与优化目标

上一节我们介绍了一个基础的EMA交叉策略。本节中，我们来看看本次优化的具体目标。

我们的目标是优化之前简要介绍的EMA策略。我们将研究在计算移动平均线斜率时需要考虑的**蜡烛数量**，这个参数被称为**回望期（Lookback Period N）**。我们需要一定数量的蜡烛来计算这些移动平均线的平均斜率。

因此，我们可以考虑使用20、10、5或任意数量的蜡烛。我们将对此进行处理。我们还将检查是否可以增加一些过滤条件。例如，在移动平均线斜率上，我们可以尝试测试“斜率小于-10”或某个特定斜率限制的条件，而不是简单的“斜率为正或负”。

我们将研究这些参数，以尝试为趋势识别创建更好的过滤器。最后，我们将尝试不同的交易管理方法，包括考虑**固定止损止盈值**、**追踪止损**以及与**平均真实波幅（ATR）** 相关的止损止盈，看看这是否能提升策略表现。

该系统并非高频交易风格，因为我们研究的是15分钟时间框架，所以它更像是日间交易策略。相同的系统也可以在更低的时间框架上研究，但由于市场噪音，控制起来可能更困难。

## 章节 2：代码框架与数据准备

上一节我们明确了优化方向，本节中我们来看看实现策略的代码框架和初始数据准备。

我们从Jupyter Notebook文件开始。我们首先创建一个数据框（DataFrame），并进行数据清洗。接着，我们计算三个不同周期的指数移动平均线（EMA）以及它们的斜率。这就是我们开始测试策略的起点。

以下是初始设置的核心代码部分：

```python
# 示例代码结构：数据准备与指标计算
import pandas as pd

# 假设 df 是包含价格数据的DataFrame
# 计算快速、中速、慢速EMA
df[‘ema_fast’] = df[‘close’].ewm(span=12).mean()
df[‘ema_medium’] = df[‘close’].ewm(span=26).mean()
df[‘ema_slow’] = df[‘close’].ewm(span=50).mean()

# 计算EMA的斜率（此处以简单差分为例）
df[‘slope_fast’] = df[‘ema_fast’].diff()
df[‘slope_medium’] = df[‘ema_medium’].diff()
df[‘slope_slow’] = df[‘ema_slow’].diff()
```

首先，我将采用1根蜡烛的回望期。这意味着我将采用最近1根蜡烛的移动平均线值，来计算快速、中速和慢速EMA斜率的平均值。如果计算正常，我将执行信号检查，即我们在上一个视频中介绍的入场信号。至此，我们处理的是入场点的总信号。

我们刚刚提出的这个策略在整体有趋势的市场中效果很好，给了我们一些正回报。但一年100%的回报对于股票波动来说并不是非常理想。所以，我们在这个阶段要做的就是增加测试中考虑的蜡烛数量，即移动平均线斜率的回望期参数。

## 章节 4：优化回望期参数

上一节我们通过回测发现了策略在横盘市场的弱点，本节中我们来看看通过调整核心参数——回望期（N）来尝试改进。

例如，我将回望期参数改为30根蜡烛。重新对之前的数据切片进行回测，我们的回报变为124%，比之前的测试稍好一些。检查净值曲线：我们一开始有更稳定的增长，然后净值不断增加达到约700%。但接下来有一段回撤期。

检查发生了什么：我们在这两个峰值之间的横盘期损失了大部分利润。因为在交易期间的这一部分，我们大部分交易都是亏损的。所以我们知道了算法的缺点和优点，以及它何时会失效。

有两种解决方案：
1.  尝试添加更多过滤器（例如更多指标），但这可能有点困难。
2.  简单地在我们直观看到市场有趋势时运行算法，而不是让它24小时运行。我们可以协助算法，检查何时应该执行，何时不应该执行。

我将尝试增加回望期参数的值，检查是否采用更大的值（比如40左右）能解决问题。这里我们获得了248%的回报，因此我们增加了回报，并且没有损失太多资产。再次考虑我们损失大部分利润的那个时间窗口，如果我们能确保算法跳过甚至不在这个窗口中进行交易，我们此时可以赚取大约600%到700%的利润。

还值得一提的是，今年上半年的第一部分测试期可能也是横盘市场，就像净值没有太大改善一样。如果我们此时观察市场，可以看到它是横盘趋势，不是很清楚。只有当市场开始出现变化的峰值时，我们才开始看到权益部分有所改善。这再次证明我们的算法非常适合趋势市场，但在横盘市场上表现不佳。

## 章节 5：优化止损与止盈

在调整了趋势过滤参数后，本节中我们来看看资金管理的关键部分：止损和止盈的设置。

关于止损和止盈值，我们当然可以增加它们，看看改变这些值对我们交易的影响。这些是我们的统计数据。如果我们现在将止损止盈设得很大，回报是负的（例如-2%）。这只是为了展示我们的模型对这些值有多么敏感，以及管理交易和资金管理有多么重要。

许多人认为，如果你有完美的交易信号和高的胜率，就足以拥有盈利策略。实际上，超过一半的策略盈利依赖于良好的资金管理或交易管理系统。

在这里，我们可以尝试将这些值更改为70点和75点（止损70，止盈75）。再试一次，我们现在得到了-61%的回报，所以我们的策略突然变成了亏损策略。似乎该策略不适应过大的止损止盈距离。

因此，我认为我们最好采用较小的值，例如35点止损和35点止盈。这里我们有29%的利润。所以我们知道我们不应该过多增加止损和止盈距离，因为这是15分钟的时间框架，过大的距离对盈利策略来说并不积极。

让我们回到40点止损和45点止盈再试一次，这里我们的回报约为101%。所以看起来40点止损、45点止盈是这两个参数之间的一个较优点。止损距离太短或止盈距离太宽都不好。

## 章节 6：引入ATR动态止损止盈

在测试了固定止损止盈后，本节中我们引入一个更适应市场波动的工具：平均真实波幅（ATR），来设置动态的止损止盈。

另一个想法是尝试使用与ATR相关的止损和止盈。我们在程序中的某个时刻已经计算了ATR，它是我们数据框中的一列。我们将使用这个值来计算止损距离。该距离将与ATR成比例，例如2倍ATR距离、1.5倍ATR距离等，具体取决于你希望止损相对于蜡烛的最后收盘价设置多远。

我们从1.5倍ATR距离开始，然后设置止盈止损比率（例如1.5）。首先，我们设置比率=1，这意味着我们的止损和止盈距离相等。当然，这些计算包含在我们的买入和卖出交易执行函数中。

运行回测，我们在大约6个月内获得了6%的回报。如果放回到12个月，今年我们有57%的回报，这是一个正向回报。我们可以再次绘制净值曲线。现在我们有很大的亏损。

当然，我们可以改变止盈止损的比率，以及止损的ATR倍数。此时，我们可以尝试使用等于2倍ATR的止损，并暂时将止盈止损比率保持为1。我们需要一次更改一个参数来验证对结果的影响。这里我们获得了86%的回报，比之前的结果有所改进。

如果我将ATR倍数增加到2.5，再检查一次，我们得到了246%的回报。这很好。如果此时我们回落到1.83倍ATR，回报约为95%。所以我认为大约2倍ATR是这个特定参数的最佳点。

现在我们要更改止盈止损比率，将其设置为1.2。我们获得了大约95%的回报。这是一个轻微的改进。这意味着我们可以进一步测试1.5的止盈止损比率。现在我们回到45%的回报。让我们回到比率1.2，保留这些参数，我们的利润约为95%。所以没有什么特别引人注目的，但它仍然是一个获胜策略。

## 章节 7：测试追踪止损

另一种常用的高级资金管理方法是追踪止损，本节中我们来测试它的效果。

我们将使用ATR来设置追踪止损。此时我们只使用追踪止损，不设定固定的止盈目标。我们将跟随价格移动，只要价格持续朝着有利方向变动，止损值就会随之修改。

这为我们提供了356%的回报（在12个月的测试期内）。我们几乎从2000开始。这是一个不同的12个月期间，我们运行并检查结果。这里我们有2866%的回报。所以在这种情况下，追踪止损工作得很好。

让我们尝试在不同的时间段开始，例如索引30000。如果我们测试整个策略（三年数据），我们的回报率为445%。我们在数据框中包含了整整三年的数据。以索引0作为开始举例，这里我们的回报率为负39%。所以从这个角度来看，这不是一个在所有时期都获胜的策略。

所以在某些时候，我们必须决定将哪种交易管理方法纳入我们的交易系统中。我们需要选择稳健的方法，不仅能带来高额正回报，而且能在三年或更长时间内保持稳健，以获得确定的结果，并避免在运行交易系统时出现意外的巨大亏损。

这样，我们就可以使用之前测试中表现最稳定的系统之一：45点固定止盈和40点固定止损，再进行一次大约30个月（三年期间）的测试。在这三年期间，我们的回报率为445%。这是三年期间的净值曲线，我们可以看到，净值在某种程度上达到了10100%的回报率，这很好。但随后我们又出现了一次下降。只有我们能够提高该系统中的过滤能力，并尝试限制这些下降周期，策略才会更完美。

## 章节 8：总结与核心要点

本节课中我们一起学习了如何对一个简单的EMA交叉策略进行多方面的优化。

请记住，我们在这个教程中采取了一些非常简单的东西：我们采用了三条移动平均线，以及价格上穿快速移动平均线作为信号。这不是一个复杂的系统，它真的非常简单和基本。

我们采取了以下优化步骤：
1.  **调整趋势敏感度**：通过改变计算EMA斜率的回望期（N）来优化趋势过滤。
2.  **优化资金管理**：测试了固定止损止盈、基于ATR的动态止损止盈以及追踪止损。
3.  **进行稳健性检验**：在不同时间切片上进行多次回测，评估策略的稳定性。

核心发现是：
*   该策略在**趋势市场**中表现良好，在**横盘市场**中容易亏损。
*   **参数优化**（如回望期N）对表现有显著影响。
*   **资金管理**（止损止盈设置）是策略盈利的关键组成部分，有时比信号本身更重要。
*   没有一种设置在所有市场时期都完美，需要在**盈利性**和**稳健性**之间做出权衡。
