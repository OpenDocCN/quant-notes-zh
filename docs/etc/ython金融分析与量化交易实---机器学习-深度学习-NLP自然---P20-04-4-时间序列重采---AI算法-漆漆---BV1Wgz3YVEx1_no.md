## 重采样操作

![](img/22975185d311962f5226441f53c12e99_3.png)

上一节我们介绍了时间序列的基本概念，本节中我们来看看如何改变其观测频率，即重采样。

重采样是指将时间序列数据从一个时间频率转换到另一个时间频率的过程。例如，原始数据是按天统计的，我们可以将其转换为按周、按月或按年进行统计。

![](img/22975185d311962f5226441f53c12e99_5.png)

![](img/22975185d311962f5226441f53c12e99_7.png)

进行重采样操作时，需要使用 `resample()` 方法，并至少指定一个参数来告知新的时间单位。同时，你需要指定在新频率下如何聚合数据，例如计算均值、第一个值、中位数或其他统计指标。

![](img/22975185d311962f5226441f53c12e99_9.png)

以下是重采样的基本代码示例，假设我们有一个名为 `data` 的DataFrame，其索引为日期：

![](img/22975185d311962f5226441f53c12e99_11.png)

```python
# 以‘周’（‘W’）为单位进行重采样，并计算每周的均值
weekly_data = data.resample('W').mean()
print(weekly_data.head())
```

执行后，数据会按周（默认从周日开始）进行分组，并计算每周数据的平均值作为新数据点。

我们也可以按月进行重采样：

```python
# 以‘月’（‘M’）为单位进行重采样，并计算每月的第一个值
monthly_data = data.resample('M').first()
print(monthly_data.head())
```

![](img/22975185d311962f5226441f53c12e99_13.png)

![](img/22975185d311962f5226441f53c12e99_15.png)

在重采样时，有一个重要的参数是 `label`，它决定了新数据点的标签（索引）使用时间窗口的哪一端。

以下是 `label` 参数的作用：
*   **`label=‘right’` (默认)**：使用时间窗口的**结束日期**作为该窗口数据的标签。例如，一月份的数据聚合后，标签显示为1月31日。
*   **`label=‘left’`**：使用时间窗口的**开始日期**作为标签。例如，一月份的数据聚合后，标签显示为1月1日。

![](img/22975185d311962f5226441f53c12e99_17.png)

![](img/22975185d311962f5226441f53c12e99_19.png)

```python
# 使用窗口起始日（1月1日）作为标签
monthly_data_left = data.resample('M', label='left').mean()
# 使用窗口结束日（1月31日）作为标签（默认行为）
monthly_data_right = data.resample('M', label='right').mean()
```

![](img/22975185d311962f5226441f53c12e99_21.png)

选择 `‘left’` 或 `‘right’` 取决于具体分析需求。在金融分析中，通常使用 `‘right’`（默认值），因为它表示截至到该日期为止的统计结果，更符合常规的观察逻辑。

![](img/22975185d311962f5226441f53c12e99_23.png)

## 时间窗口（滑动窗口）操作

了解了如何改变数据频率后，我们来看看如何在固定的时间窗口内进行滑动计算，这是计算移动平均线等指标的基础。

时间窗口操作是指定义一个固定长度的窗口（例如10天），让这个窗口在时间序列上滑动。对于每一个窗口位置，我们都可以计算其内部数据的统计量（如最小值、最大值、均值、标准差等）。

![](img/22975185d311962f5226441f53c12e99_25.png)

这个过程就像用一个固定长度的框，在时间轴上一步步移动，每移动一步就计算一次框内数据的特征。

![](img/22975185d311962f5226441f53c12e99_27.png)

以下是创建并计算滑动窗口统计量的步骤：

首先，我们需要确保数据没有缺失值，否则会影响窗口计算。

![](img/22975185d311962f5226441f53c12e99_29.png)

```python
# 1. 处理缺失值
data_clean = data.dropna()
```

![](img/22975185d311962f5226441f53c12e99_31.png)

接着，使用 `rolling()` 方法创建窗口。例如，创建一个窗口大小为10的滚动窗口对象，并计算每个窗口的最小值。

```python
# 2. 定义窗口大小
window_size = 10
# 3. 对清洗后的数据列进行滚动计算，此处计算每个窗口的最小值
rolling_min = data_clean[‘column_name’].rolling(window=window_size).min()
print(rolling_min.tail(15))
```

![](img/22975185d311962f5226441f53c12e99_33.png)

最初几个窗口（前9个）的结果会是 `NaN`，因为数据不足以填满一个长度为10的窗口。从第10个数据点开始，每个值都代表包含它自身及前9个数据在内的窗口的最小值。

![](img/22975185d311962f5226441f53c12e99_35.png)

我们可以一次性为数据添加多个基于窗口的统计列。

![](img/22975185d311962f5226441f53c12e99_37.png)

以下是同时计算最小值、最大值、均值和标准差的示例：

![](img/22975185d311962f5226441f53c12e99_39.png)

![](img/22975185d311962f5226441f53c12e99_41.png)

```python
# 一次性计算多个滚动统计量
data_clean[‘rolling_min’] = data_clean[‘column_name’].rolling(window=window_size).min()
data_clean[‘rolling_max’] = data_clean[‘column_name’].rolling(window=window_size).max()
data_clean[‘rolling_mean’] = data_clean[‘column_name’].rolling(window=window_size).mean()
data_clean[‘rolling_std’] = data_clean[‘column_name’].rolling(window=window_size).std()

# 查看计算结果
print(data_clean[[‘rolling_min‘， ‘rolling_max‘， ‘rolling_mean‘， ‘rolling_std‘]].tail())
```

这样，我们就得到了原始数据以及其对应的滑动窗口统计指标，这些指标对于分析数据趋势和波动性非常有帮助。

![](img/22975185d311962f5226441f53c12e99_43.png)

![](img/22975185d311962f5226441f53c12e99_45.png)
