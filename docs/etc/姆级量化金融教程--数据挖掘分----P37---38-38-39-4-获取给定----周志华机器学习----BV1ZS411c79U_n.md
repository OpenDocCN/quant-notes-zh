上一节我们介绍了如何获取单日的因子数据。本节中，我们来看看如何扩展这个过程，以获取一个完整时间区间内的所有数据。

![](img/501508a90b763a08f3cc7ac5adc078ef_18.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_20.png)

核心思路是使用一个循环，遍历区间内的每一个交易日，分别获取当天的数据，然后将这些每日的数据汇总到一个总表中。

![](img/501508a90b763a08f3cc7ac5adc078ef_22.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_24.png)

以下是实现这一过程的具体步骤：

![](img/501508a90b763a08f3cc7ac5adc078ef_26.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_28.png)

1.  **确定循环范围**：首先，我们需要知道要遍历多少天。这可以通过查询日期序列的长度来实现。
    ```python
    total_days = len(date_sequence)
    ```

![](img/501508a90b763a08f3cc7ac5adc078ef_30.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_32.png)

2.  **遍历每一天**：使用 `for` 循环，遍历从第0天到第 `total_days-1` 天的索引。
    ```python
    for i in range(total_days):
        # 处理第i天的数据
    ```

![](img/501508a90b763a08f3cc7ac5adc078ef_34.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_36.png)

3.  **获取每日数据**：在循环体内，调用获取单日数据的函数，但将参数从固定的 `0` 改为循环变量 `i`，以获取第 `i` 天的数据。
    ```python
    daily_data = get_factor_data(i)  # 假设的函数，实际中需替换为您的数据获取方法
    ```

![](img/501508a90b763a08f3cc7ac5adc078ef_38.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_40.png)

4.  **提取有效数据**：获取的数据通常是多维的。我们需要从中提取出有用的部分。根据示例，有效数据在第三个维度（索引为2）。
    ```python
    useful_data = daily_data[0, 2, :]  # 提取特定维度的数据
    ```

![](img/501508a90b763a08f3cc7ac5adc078ef_42.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_44.png)

5.  **构建每日DataFrame**：对于每一天，我们创建一个临时的DataFrame来存储两列信息：日期和当天的因子值。
    ```python
    daily_df = pd.DataFrame()
    daily_df[‘factor_value‘] = useful_data  # 存储因子值
    daily_df[‘date‘] = current_date  # 存储当前日期，current_date需根据索引i获取
    ```

![](img/501508a90b763a08f3cc7ac5adc078ef_46.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_48.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_50.png)

6.  **拼接至总表**：在循环开始前，初始化一个空的DataFrame作为总表。在每次循环中，将当天构建的 `daily_df` 拼接到这个总表中。
    ```python
    # 循环前初始化
    total_df = pd.DataFrame()
    
    # 循环内拼接
    total_df = pd.concat([total_df, daily_df], ignore_index=True)
    ```

在实现过程中，需要注意几个关键点。首先，数据获取和拼接操作可能比较耗时，尤其是在处理较长的时间区间（如一年）时，需要耐心等待。其次，在拼接操作后，必须将结果赋值回总表变量，否则数据不会保存。

![](img/501508a90b763a08f3cc7ac5adc078ef_66.png)

![](img/501508a90b763a08f3cc7ac5adc078ef_68.png)

最后，检查生成的 `total_df`，它应该包含三列关键信息：股票名称、日期以及对应的因子指标值。这样就完成了从指定区间内提取并整合全部数据的过程。
