# 金融量化分析：P5：04 影响股价因素与股票买卖知识 📈

在本节课中，我们将学习影响股票价格的核心因素，并了解股票买卖的基本流程与规则。理解这些基础知识是进行量化分析的前提。

## 影响股价的六大因素

上一节我们介绍了股票的基本概念，本节中我们来看看哪些因素会决定股票价格的涨跌。影响股价的因素可以归纳为以下六点。

### 1. 公司自身因素
这是影响股价最根本的因素。公司的经营状况直接决定了其长期价值。例如，一家公司首次公开募股（IPO）时，会根据其市值确定发行价。假设一家公司市值为50亿人民币，将其拆分为1亿股，则每股发行价为5元。此后，若公司经营良好，市值增长至70亿，则每股价格可能上涨至7元。反之，若公司出现重大丑闻或业绩下滑，股价就会下跌。

### 2. 市场因素
这是影响股价最直接的因素。股价的短期波动主要由市场供求关系决定。其原理类似于商品买卖：
*   **买盘 > 卖盘**：供不应求，股价上涨。
*   **卖盘 > 买盘**：供过于求，股价下跌。

### 3. 行业因素
整个行业的发展前景会影响行业内所有公司的股价。例如，当人工智能行业成为热点时，相关公司的股票可能普遍上涨。反之，如果一个行业前景黯淡，相关公司的股价也可能集体承压。

### 4. 心理因素
投资者的情绪和非理性行为会加剧市场波动。例如，当看到有大额抛售时，其他投资者可能因恐慌而跟随抛售，导致股价非理性下跌。历史上著名的“黑色星期五”股灾，部分原因就是交易故障引发的连锁恐慌抛售。

### 5. 经济因素
国家层面的宏观经济政策和指标对股市有广泛影响。例如：
*   **利率**：银行存款利率上升，可能吸引资金从股市流入银行，导致股市资金减少，股价承压。
*   **货币政策、外汇政策**等也会影响市场流动性和投资者信心。

### 6. 政治因素
国际或国内的政治事件和稳定性会影响市场。例如，当地缘政治紧张局势升级时（如国家间发生摩擦），投资者可能因避险情绪而抛售股票，导致股市下跌。相反，某些特定板块（如军工）可能因事件驱动而上涨。

## 股票买卖流程与规则

了解了影响价格的因素后，我们来看看在实际中如何买卖股票，以及需要遵守哪些市场规则。

### 1. 开户与委托
个人投资者不能直接在交易所买卖股票，必须通过证券公司（券商）进行。流程如下：
1.  在券商处开设证券账户。
2.  通过券商提供的系统（如软件、电话）连接交易所。
3.  提交买卖指令，这个过程称为“委托”。

### 2. 交易日与交易时间
股票交易并非随时可以进行，它有严格的时间规定。
*   **交易日**：通常为每周一至周五（法定节假日除外）。
*   **交易时间**：以下为一般情况（具体以交易所公告为准）：
    *   **开盘集合竞价**：9:15 - 9:25。此期间接受委托，但不立即成交，交易所会在9:25一次性撮合所有订单，以产生当日的**开盘价**。其核心原则是使成交量最大化。
    *   **连续竞价**：9:30 - 11:30, 13:00 - 14:57。此期间系统对接收到的委托进行**连续、高频的撮合**（如每几秒一次），达成实时成交。
    *   **收盘集合竞价**（深圳交易所）：14:57 - 15:00。此期间接受委托但不撮合，在15:00一次性撮合，以产生**收盘价**。上海交易所的收盘价为最后一笔连续竞价的成交价。

### 3. 核心交易制度
以下是两个至关重要的市场规则：
*   **T+1 制度**：指当日（T日）买入的股票，需到下一个交易日（T+1日）才能卖出。该制度旨在抑制过度投机。
*   **涨跌停板限制**：为控制股价单日波动幅度，A股市场设有涨跌幅限制（通常为±10%）。这意味着股票价格在一个交易日内，上涨不能超过前一日收盘价的10%，下跌也不能超过10%。其价格计算公式为：
    **涨停价 = 前收盘价 × (1 + 涨跌幅限制比例)**
    **跌停价 = 前收盘价 × (1 - 涨跌幅限制比例)**
