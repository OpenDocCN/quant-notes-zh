## 概述
在本节课中，我们将要学习一个名为`akshare`的免费开源Python库。它是获取量化交易所需底层数据（如K线、财务、宏观数据）的强大工具。我们将了解它的功能、使用方法、优缺点，并帮助你快速上手。

## 什么是akshare？
`akshare`是一个基于Python的开源金融数据接口库。它的核心功能是帮助用户从各大财经网站（如上海证券交易所、深圳证券交易所、新浪、雪球等）爬取公开的金融数据，并将数据整理成易于分析的`pandas DataFrame`格式。

上一节我们介绍了`akshare`的基本定位，本节中我们来看看它的数据覆盖范围。

## akshare的数据源
`akshare`的数据源非常广泛，涵盖了多个金融领域。以下是其主要的数据分类：

![](img/8d28edbeea12a3dfeed219a86d015397_9.png)

*   **股票数据**：A股、港股、美股等市场的行情、基本面数据。
*   **期货数据**：国内各大商品交易所的期货行情。
*   **债券与期权数据**：相关的行情与指标。
*   **外汇数据**：主要货币对的汇率信息。
*   **宏观数据**：包括GDP、CPI、利率等经济指标。
*   **公募与私募基金数据**：基金净值、持仓等信息。
*   **波动率数据**：如VIX指数等。

![](img/8d28edbeea12a3dfeed219a86d015397_11.png)

这些数据都是网页上的公开信息，`akshare`通过编写爬虫代码将其结构化地获取下来。

## 如何使用akshare？
`akshare`的使用非常直观。每个数据接口都是一个Python函数，调用后会返回一个`pandas DataFrame`。

以下是获取上海证券交易所市场概况数据的代码示例：

```python
import akshare as ak

![](img/8d28edbeea12a3dfeed219a86d015397_13.png)

![](img/8d28edbeea12a3dfeed219a86d015397_15.png)

# 使用 ak.stock_sse_summary() 函数获取上交所市场概况数据
sse_summary_df = ak.stock_sse_summary()
# 打印查看数据
print(sse_summary_df)
```

![](img/8d28edbeea12a3dfeed219a86d015397_17.png)

![](img/8d28edbeea12a3dfeed219a86d015397_19.png)

运行上述代码，你将得到一个包含总市值、成交量、上市公司数量等信息的`DataFrame`，其内容与官方网站上公布的数据一致。这证明了`akshare`有效地完成了数据抓取和格式转换的工作。

## 优点与缺点总结
本节课中我们一起学习了`akshare`的核心功能。最后，我们来总结一下它的主要优点和需要注意的缺点。

![](img/8d28edbeea12a3dfeed219a86d015397_25.png)

**优点：**
1.  **数据全面免费**：覆盖股票、期货、债券、宏观等多个领域，数据源免费。
2.  **使用便捷**：接口返回`pandas DataFrame`格式，与Python数据分析生态无缝衔接。
3.  **持续维护**：开源社区活跃，能较快适应数据源的变化。

![](img/8d28edbeea12a3dfeed219a86d015397_27.png)

**缺点：**
1.  **依赖网络爬虫**：本身不存储数据，实时获取受限于目标网站的反爬机制，不适合高频实盘交易。
2.  **可能被限制**：短时间内频繁请求可能导致IP被暂时封锁。
3.  **数据口径需注意**：不同接口的股票代码格式可能不统一（例如`600000.SH`与`sh600000`），使用时需要留意并做标准化处理。

![](img/8d28edbeea12a3dfeed219a86d015397_29.png)
