## 策略原理与思路介绍

MA-RSI策略综合运用了MA和RSI这两个常见的技术指标。MA（移动平均线）主要反映价格的长期趋势特征，而RSI（相对强弱指数）则是一个用于判断短期超买超卖状态的动量指标。通过结合这两个指标，我们可以筛选出处于**上升趋势**中的**超卖**个股，从而缩小选股范围。在此基础上，辅以简单的卖出和止损策略，以期实现较为稳定的收益。

该策略的重点在于：首先综合两个指标选出符合条件的股票，然后通过风控策略管理交易。它不追求极高的收益率，而是强调策略的稳定性和风险控制。

以下是该策略的核心组成部分：

**1. 选股范围**
该策略首先会设定一个选股池，其筛选条件包括：
*   选取市值大于500亿的股票（即大市值股票）。
*   排除ST股、科创板、创业板、退市股和次新股。
*   将剩余股票按照净利润环比增长率进行降序排列。

**2. 买卖信号**
*   **买入信号**：需同时满足两个条件。
    *   条件一：`K线价格 > MA200`。这确保了股票处于长期上升趋势中。
    *   条件二：`RSI(10) < 25`。这表示股票处于短期超卖状态，可能存在反弹机会。
*   **卖出信号**：满足以下任一条件即触发卖出。
    *   条件一：`RSI(10) > 40`。这表示短期已从超卖区域反弹，可考虑获利了结。
    *   条件二：`持有天数 >= N天`（例如10或11天）。这是为了定期换仓，控制风险。
    *   条件三：`个股自买入后下跌超过5%`。这是止损逻辑，用于控制单笔亏损。

> **注意**：策略中的具体参数（如MA200、RSI(10)、阈值25和40、持有天数N、止损比例5%）是基于历史经验设定的。在实际应用中，你可以根据自己的回测结果和风险偏好进行调整和优化。

## 策略实现思路 🧠

了解策略原理后，我们来看看如何用代码实现它。整个实现过程可以清晰地分为三步：

**第一步：环境初始化**
导入必要的函数库（如技术分析库），并设置回测环境。这包括设定基准指数、开启避免未来数据模式、设置交易费用、滑点等参数，并初始化一些用于记录状态的变量（如持仓数量、持有天数计数器）。

**第二步：选股逻辑**
在每个交易日，执行选股函数。该函数需要完成两项任务：
1.  **过滤无效股票**：根据选股范围，剔除ST股、科创板、创业板、退市股和次新股。
2.  **指标信号选股**：在过滤后的股票池中，计算每只股票的MA200和RSI(10)值，并筛选出同时满足`K线 > MA200`且`RSI(10) < 25`的股票，形成当日的“买入候选列表”。

**第三步：交易执行**
根据买卖信号执行交易。对于买入候选列表中的股票，如果没有持仓则买入；对于已持仓的股票，则判断是否满足三个卖出条件（RSI(10) > 40、持有到期、跌幅超阈值）中的任意一个，若满足则卖出。

**2. 选股逻辑中的股票过滤函数**
在选股前，我们需要一个函数来过滤掉不符合条件的股票（如ST股、科创板等）。
```python
def filter_stocks(stock_list):
    """
    过滤股票列表，剔除ST、退市、科创、创业、次新股
    :param stock_list: 初始股票列表
    :return: 过滤后的股票列表
    """
    df = get_fundamentals(query(
        valuation.code, valuation.market_cap
    ).filter(
        valuation.code.in_(stock_list)
    ), date=None)
    
    if df.empty:
        return []
    
    filtered_list = []
    for stock in df['code']:
        # 获取股票信息
        stock_info = get_security_info(stock)
        name = stock_info.display_name
        code = stock_info.code
        
        # 过滤条件
        # 1. 过滤ST、*ST、退市股票
        if 'ST' in name or '退' in name:
            continue
        # 2. 过滤次新股（假设上市200天以上为非次新）
        if stock_info.start_date is None or (context.current_dt.date() - stock_info.start_date).days < 200:
            continue
        # 3. 过滤创业板(300开头)和科创板(688开头)
        if code.startswith('300') or code.startswith('688'):
            continue
            
        filtered_list.append(stock)
    
    return filtered_list
```
> **代码说明**：`filter_stocks`函数通过遍历股票列表，检查股票名称是否包含特定字符、上市天数以及股票代码开头数字，来剔除不符合条件的股票。这是一个基础的实现方法，性能上有优化空间，例如可以尝试用向量化操作或更高效的查询方式。
