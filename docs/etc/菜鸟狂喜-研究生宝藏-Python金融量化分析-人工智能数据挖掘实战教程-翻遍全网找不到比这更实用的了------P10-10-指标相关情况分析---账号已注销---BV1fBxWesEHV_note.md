## 概述

![](img/b2468aac11d0192642449c6e83c0d6dd_9.png)

回归分析的核心是探究变量之间的关系，例如观察一个变量的走势或增长率是否会影响另一个变量。我们将通过以下步骤完成分析：
1.  选择并提取两个感兴趣的指标数据。
2.  通过绘制图表直观地观察它们的关系。
3.  使用散点图矩阵更全面地分析数据分布和两两关系。
4.  最终通过回归分析量化它们之间的相关性。

## 第三步：散点图矩阵分析

上一节我们通过折线图观察了时间序列上的关系。本节中，我们来看看如何通过散点图矩阵来同时观察单个变量的分布以及两个变量之间的关系。

![](img/b2468aac11d0192642449c6e83c0d6dd_63.png)

首先，我们计算指标的日收益率（或日变化率），这通常比原始价格数据更能反映波动关系。

```python
import numpy as np
returns = np.log(data / data.shift(1))
print(returns.head())
```

![](img/b2468aac11d0192642449c6e83c0d6dd_65.png)

接下来，我们使用Pandas的`scatter_matrix`函数来绘制散点图矩阵。

![](img/b2468aac11d0192642449c6e83c0d6dd_67.png)

![](img/b2468aac11d0192642449c6e83c0d6dd_69.png)

```python
from pandas.plotting import scatter_matrix

scatter_matrix(returns,
               alpha=0.5,        # 设置点的透明度，避免重叠
               figsize=(10, 10),
               diagonal='kde',   # 对角线显示核密度估计图
               hist_kwds={'bins': 50}) # 如果对角线用直方图，可设置分组数
```

**代码参数解释**：
*   `alpha`：散点图中点的透明度，用于在数据点密集时提高可读性。
*   `diagonal`：指定对角线上的图形类型。`'kde'`表示核密度估计图，它通过平滑的曲线展示单个变量的分布；`'hist'`则表示直方图。
*   `hist_kwds`：当`diagonal='hist'`时，用于向直方图传递额外参数，例如设置分组数量`bins`。

![](img/b2468aac11d0192642449c6e83c0d6dd_71.png)

![](img/b2468aac11d0192642449c6e83c0d6dd_73.png)

生成的矩阵图中：
*   **对角线**上的图展示了每个变量自身的分布情况（例如`SPX`收益率分布）。
*   **非对角线**上的散点图展示了两两变量之间的关系。从`SPX`和`VIX`的散点图中，可以清晰地看到点云呈向下倾斜的趋势，这直观地证实了它们之间的负相关关系。

## 第四步：引入回归分析

![](img/b2468aac11d0192642449c6e83c0d6dd_81.png)

散点图已经清晰地展示了关系，但我们还可以更进一步，用一条回归线来量化这种关系。虽然本节课没有直接绘制回归线，但其思路是在散点图的基础上，拟合一个线性方程（如 `y = kx + b`），并将这条线画在散点图上。回归系数`k`（斜率）的大小和正负，可以精确地描述相关的强度和方向。

![](img/b2468aac11d0192642449c6e83c0d6dd_83.png)

![](img/b2468aac11d0192642449c6e83c0d6dd_85.png)

例如，一个负的斜率`k`就明确表示`x`增加时`y`减少，这与我们观察到的`SPX`与`VIX`的关系一致。在后续实践中，可以使用`statsmodels`或`scikit-learn`等库轻松完成线性回归拟合和绘图。

## 总结

本节课中我们一起学习了金融指标相关性分析的完整流程：
1.  **数据准备**：从数据集中精确提取目标指标列。
2.  **可视化探索**：通过共享X轴的双Y轴折线图，在时间序列上直观观察指标的互动关系。
3.  **深入分析**：利用散点图矩阵，同时考察各指标的分布形态以及两两之间的相关关系。
4.  **量化分析（方向）**：了解到可以通过回归分析，用具体的数学公式（如 `y = kx + b`）来量化变量间关系的强度和方向。
