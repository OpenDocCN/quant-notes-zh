### 3. 计算因子与选股

![](img/06194506c3e04a6057b0245196cd6a5e_49.png)

这是策略最关键的步骤。我们需要：
1.  获取`stock_pool`中股票在调仓日的相关数据（如市值、市盈率等）。
2.  基于这些数据计算我们感兴趣的因子（例如估值因子、质量因子）。
3.  对因子进行预处理（去极值、标准化、行业中性化）。
4.  根据处理后的因子值对股票进行排序，选出排名靠前的股票作为当期买入标的。

![](img/06194506c3e04a6057b0245196cd6a5e_51.png)

![](img/06194506c3e04a6057b0245196cd6a5e_53.png)

```python
    # 获取股票基本面数据（此处为示例，需替换为实际因子计算）
    fundamental_df = get_fundamentals(

    # 假设我们以市值为因子（小市值效应），进行升序排列（市值越小排名越靠前）
    factor_series = fundamental_df.set_index('code')['market_cap']

    N = 10
    selected_stocks = factor_series.nsmallest(N).index.tolist()
```

### 4. 调仓操作

![](img/06194506c3e04a6057b0245196cd6a5e_57.png)

![](img/06194506c3e04a6057b0245196cd6a5e_59.png)

确定本期要持有的股票列表(`selected_stocks`)后，我们需要调整持仓：
*   卖出当前持有但不在新列表中的股票。
*   买入在新列表中但当前未持有的股票。
*   通常我们会将资金平均分配到每只目标股票上。

![](img/06194506c3e04a6057b0245196cd6a5e_61.png)

```python
    # 卖出当前持有，但不在目标列表中的股票
    for stock in context.portfolio.positions:
        if stock not in selected_stocks:
            order_target_value(stock, 0)

    # 将资金平均分配到目标股票上
    if len(selected_stocks) > 0:
        weight = 1.0 / len(selected_stocks)
        for stock in selected_stocks:
            order_target_percent(stock, weight)
```

![](img/06194506c3e04a6057b0245196cd6a5e_63.png)

![](img/06194506c3e04a6057b0245196cd6a5e_65.png)

*   **`order_target_value(stock, 0)`**：将指定股票的持仓价值调整到0，即清仓卖出。
*   **`order_target_percent(stock, weight)`**：将指定股票的持仓调整到占总资金的一定比例（`weight`）。

## 总结

![](img/06194506c3e04a6057b0245196cd6a5e_69.png)

![](img/06194506c3e04a6057b0245196cd6a5e_71.png)

本节课中我们一起学习了如何将一个因子选股策略从概念转化为代码。我们构建了一个按月调仓的策略框架，其核心流程包括：
1.  **设置定时器**：使用`run_monthly`定期触发调仓函数。
2.  **获取与过滤股票池**：确定可交易的股票范围。
3.  **因子计算与选股**：获取数据，计算并处理因子，根据因子值筛选出当期要投资的股票。
4.  **执行调仓**：根据新的股票列表，卖出旧仓位，买入新仓位，完成组合更新。

![](img/06194506c3e04a6057b0245196cd6a5e_73.png)
