## 概述

![](img/47720cbd6a02b21bc0bd766625373211_7.png)

![](img/47720cbd6a02b21bc0bd766625373211_9.png)

面试题描述了一个冷库温度数据清洗的场景。数据来自一个Excel表格，记录了七个温度采集设备每分钟的温度读数。我们的目标是清洗前四个设备数据中的空值，以便后续用于建立温度场关系模型，预测后三个设备的温度。虽然模型构建部分涉及机器学习，不在本节讨论范围，但数据预处理是我们可以完成的。

![](img/47720cbd6a02b21bc0bd766625373211_11.png)

首先，我们需要加载数据并查看其结构。

![](img/47720cbd6a02b21bc0bd766625373211_13.png)

![](img/47720cbd6a02b21bc0bd766625373211_15.png)

```python
import pandas as pd
df = pd.read_excel('./data/test_data.xlsx')
```

![](img/47720cbd6a02b21bc0bd766625373211_17.png)

加载后的`DataFrame` `df` 中，列`1`到`7`代表七个温度采集设备，每一行代表某一分钟所有设备采集到的温度数据。

![](img/47720cbd6a02b21bc0bd766625373211_19.png)

## 提取目标数据

我们的处理重点是前四个设备（列`1`到`4`）的数据。因此，我们先从原始数据中提取这四列。

```python
df = df[['1', '2', '3', '4']]
```

![](img/47720cbd6a02b21bc0bd766625373211_21.png)

![](img/47720cbd6a02b21bc0bd766625373211_23.png)

现在，`df` 只包含我们需要处理的四列数据，其中存在一些空值（NaN）。

![](img/47720cbd6a02b21bc0bd766625373211_25.png)

![](img/47720cbd6a02b21bc0bd766625373211_27.png)

## 方法一：删除含有空值的行

![](img/47720cbd6a02b21bc0bd766625373211_29.png)

一种直接的处理方式是删除任何包含空值的整行数据。Pandas提供了 `dropna` 方法来实现这个功能。

![](img/47720cbd6a02b21bc0bd766625373211_31.png)

![](img/47720cbd6a02b21bc0bd766625373211_33.png)

在Pandas中，`axis`参数用于指定操作的方向：`axis=0`表示行，`axis=1`表示列。由于我们要删除行，所以设置`axis=0`。

![](img/47720cbd6a02b21bc0bd766625373211_35.png)

```python
df_dropped = df.dropna(axis=0)
```

![](img/47720cbd6a02b21bc0bd766625373211_37.png)

执行后，`df_dropped` 将只包含在所有四列上都有完整数据的行。原始数据有1060行，删除空值行后剩余982行，删除的成本不高，因此这个方法是可行的。

![](img/47720cbd6a02b21bc0bd766625373211_39.png)

我们也可以使用 `notnull()` 和 `all()` 方法的组合来实现相同的效果。

![](img/47720cbd6a02b21bc0bd766625373211_41.png)

```python
# 检查每一行是否所有值都不为空
mask = df.notnull().all(axis=1)
# 使用布尔索引选择非空行
df_dropped_alternative = df.loc[mask]
```

![](img/47720cbd6a02b21bc0bd766625373211_43.png)

`df_dropped_alternative` 的结果与 `df_dropped` 完全相同。

## 方法二：使用近邻值填充空值

![](img/47720cbd6a02b21bc0bd766625373211_45.png)

![](img/47720cbd6a02b21bc0bd766625373211_47.png)

删除行可能会丢失数据。另一种方法是使用相邻的值来填充空值，这通常适用于时间序列数据，因为相邻时刻的读数通常相近。

![](img/47720cbd6a02b21bc0bd766625373211_49.png)

![](img/47720cbd6a02b21bc0bd766625373211_51.png)

我们使用 `fillna` 方法进行填充。`method`参数指定填充方式：
*   `ffill` 或 `pad`: 使用前一个有效值向前填充。
*   `bfill` 或 `backfill`: 使用后一个有效值向后填充。

![](img/47720cbd6a02b21bc0bd766625373211_53.png)

由于每一列代表一个设备随时间变化的温度，我们应沿着列方向（`axis=0`）用该设备前一刻或后一刻的温度来填充当前的空值。

![](img/47720cbd6a02b21bc0bd766625373211_55.png)

![](img/47720cbd6a02b21bc0bd766625373211_57.png)

首先尝试向前填充：

![](img/47720cbd6a02b21bc0bd766625373211_59.png)

```python
df_filled = df.fillna(method='ffill', axis=0)
```

向前填充后，数据开头如果存在空值，则可能无法被填充（因为前面没有值）。为了确保所有空值都被处理，我们可以再执行一次向后填充。

![](img/47720cbd6a02b21bc0bd766625373211_61.png)

![](img/47720cbd6a02b21bc0bd766625373211_63.png)

```python
df_filled = df_filled.fillna(method='bfill', axis=0)
```

现在，`df_filled` 中的所有空值都已被其列方向上的前一个或后一个有效值所替换。

## 验证清洗结果

![](img/47720cbd6a02b21bc0bd766625373211_65.png)

![](img/47720cbd6a02b21bc0bd766625373211_67.png)

无论采用哪种方法，清洗后都需要验证是否还存在空值。我们可以使用 `isnull()` 和 `any()` 方法来检查。

对于删除法，可以直接检查行数是否减少。
对于填充法，可以检查每列是否还存在空值：

![](img/47720cbd6a02b21bc0bd766625373211_69.png)

![](img/47720cbd6a02b21bc0bd766625373211_71.png)

```python
# 检查每一列是否还有空值
columns_with_null = df_filled.isnull().any(axis=0)
print(columns_with_null)
```

![](img/47720cbd6a02b21bc0bd766625373211_73.png)

如果输出显示所有列都是`False`，则说明所有空值已被成功清洗。

![](img/47720cbd6a02b21bc0bd766625373211_75.png)

## 总结

![](img/47720cbd6a02b21bc0bd766625373211_77.png)

本节课我们一起学习了如何处理数据集中的空值。我们通过一个冷库温度数据的面试题，实践了两种核心方法：
1.  **删除**：使用 `df.dropna(axis=0)` 直接删除含有空值的行。适用于空值较少、删除成本低的场景。
2.  **填充**：使用 `df.fillna(method='ffill/bfill', axis=0)` 用前向或后向的邻近值填充空值。适用于时间序列等数据，能更好地保留数据规模。

![](img/47720cbd6a02b21bc0bd766625373211_79.png)

![](img/47720cbd6a02b21bc0bd766625373211_81.png)
