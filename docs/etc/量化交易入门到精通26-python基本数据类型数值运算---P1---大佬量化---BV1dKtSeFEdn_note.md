# 量化交易入门到精通：26：Python基本数据类型数值运算 🧮

在本节课中，我们将要学习Python中数值类型的基本运算操作。这些运算就像使用计算器或Excel一样，是进行任何数据处理和量化分析的基础。我们将涵盖加法、减法、乘法、除法、取余和幂运算，并通过简单的例子来理解它们。

---

## 加法运算

首先，我们来看最简单的加法运算。在Python中，使用加号 `+` 进行加法计算。

```python
result = 1 + 1
print(result)
```

运行上述代码，输出结果为 `2`。

---

## 减法运算

上一节我们介绍了加法，本节中我们来看看减法运算。减法使用减号 `-` 进行计算。

```python
result = 1 - 1
print(result)
```

运行上述代码，输出结果为 `0`。

---

## 乘法运算

接下来是乘法运算。乘法使用星号 `*` 进行计算。

```python
result = 2 * 3
print(result)
```

运行上述代码，输出结果为 `6`。

---

## 除法运算

除法运算稍微复杂一些，Python提供了两种除法运算符。

以下是两种除法运算的介绍：

*   **单斜杠除法 (`/`)**: 执行标准的除法，结果总是**浮点数**（即带小数点的数）。
    ```python
    result = 10 / 2
    print(result)  # 输出 5.0
    ```
*   **双斜杠除法 (`//`)**: 执行**整除**运算，结果会向下取整为整数。
    ```python
    result = 10 // 2
    print(result)  # 输出 5
    ```

---

## 取余运算

除了基本的四则运算，我们还需要了解取余运算。取余运算使用百分号 `%`，它返回除法运算后的余数。

```python
result = 10 % 2
print(result)  # 输出 0，因为10能被2整除

result = 10 % 3
print(result)  # 输出 1，因为10除以3等于3余1
```

---

## 幂运算（乘方）

最后，我们来看幂运算，也就是乘方计算。幂运算使用两个星号 `**` 来表示。

![](img/32bcc3c97b0d379c60a6bd823659b0b1_1.png)

```python
result = 2 ** 2
print(result)  # 输出 4，即2的2次方

result = 2 ** 3
print(result)  # 输出 8，即2的3次方
```

![](img/32bcc3c97b0d379c60a6bd823659b0b1_3.png)

---

本节课中我们一起学习了Python数值运算的核心操作。我们掌握了使用 `+`、`-`、`*` 进行加、减、乘运算，理解了 `/` 和 `//` 两种除法的区别，学会了用 `%` 进行取余计算，以及用 `**` 进行幂运算。这些是构建更复杂量化策略和数据分析的基石。
