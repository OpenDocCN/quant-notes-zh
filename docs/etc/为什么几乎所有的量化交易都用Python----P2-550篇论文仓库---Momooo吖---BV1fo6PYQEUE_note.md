## 仓库使用指南

![](img/a47dd425b2235a365f3968cee611db9b_11.png)

了解了仓库的内容后，接下来我们看看这个宝藏仓库具体该如何使用。主要可以从以下四个方面入手。

![](img/a47dd425b2235a365f3968cee611db9b_11.png)

![](img/a47dd425b2235a365f3968cee611db9b_13.png)

以下是具体的使用场景和建议：

1.  **探索研究方向**：如果你是刚刚进入人工智能领域的同学，还不确定选择哪个具体方向，可以先去阅读对应领域的综述类论文，了解该方向主要研究什么内容，以及自己是否感兴趣。之后，直接去仓库中对应的文件夹下查找相关论文即可。

2.  **寻找论文灵感**：如果你在写论文时缺乏创新点（Idea），这个论文仓库绝对可以帮到你。你需要持续精读自己研究领域的论文，并泛读其他领域的论文，不断总结论文中的创新点。只有这样，你才能培养出自己提出创新想法的能力。例如，很多人会结合人工智能（AI）技术来发表论文。

3.  **作为个人论文词典**：你可以把这个仓库当作一个可扩展的词典。将来当你读到新的、仓库中尚未收录的优秀论文时，可以将其添加进去，不断充实这个资源库。

4.  **查找代码实现**：我为每篇论文都提供了代码地址。因此，如果你找不到某篇论文的代码实现，可以来这里查看一下。

只要你是从事人工智能相关方向的同学，这个资源一定会对你有用。
