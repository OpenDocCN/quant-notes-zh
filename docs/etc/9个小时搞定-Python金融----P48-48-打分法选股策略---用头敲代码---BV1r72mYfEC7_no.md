# Python金融量化分析：P48：因子打分法选股策略概述

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分法。我们将了解其核心思想、实施前提，并最终明确如何通过综合多个因子的评分来筛选出表现优异的股票。

## 策略核心思想

上一节我们介绍了如何通过单一因子（如IC值）来评估股票。然而，仅依赖单一因子进行判断可能过于片面。本节中，我们来看看如何将多个因子结合起来，形成一个更综合的选股策略。

因子打分法的核心思想类似于学生的综合成绩评定。假设我们有300只股票（例如沪深300成分股），每只股票都有A、B、C、D四个不同的因子指标（如估值、成长、价量等）。我们的目标是从中选出表现最好的10只或20只股票进行投资。

具体做法是：为每只股票的每个因子进行**打分**。例如，股票1的A因子得10分，B因子得15分，C和D因子也各有其分。然后，我们将所有因子的分数**加总**，得到一个**总分**。最后，根据总分对所有股票进行**排名**，选取排名靠前的股票作为投资组合。这就像根据学生的语文、数学、天文、地理各科成绩计算总分，再根据总分排名来选拔优等生。

**核心公式**可以表示为：
`股票总分 = 因子A得分 + 因子B得分 + 因子C得分 + 因子D得分`

我们的最终目标就是计算出这个总分并进行排序筛选。

## 策略实施前提

![](img/b032eb875ece5fe3fb5dbd1cc1e400dd_1.png)

在开始为因子打分之前，我们必须明确一个关键的**先验知识**：每个因子与股票未来收益率之间的**相关性方向**。

这是什么意思呢？并非所有因子都是数值越大越好。例如：
*   **正向因子**：因子值越大，预期收益越高。例如**每股收益(EPS)**、**净资产收益率(ROE)**，通常被认为是“越大越好”。
*   **负向因子**：因子值越小，预期收益越高。例如**市盈率(PE)**、**市净率(PB)**，在价值投资中常被认为是“越小越好”。

如果我们在打分时，对一个负向因子也采用“数值越大得分越高”的规则，那么我们的选股逻辑就会出现根本性错误。

因此，在应用打分法前，我们必须通过以下方式确定每个因子的属性：
1.  查阅券商的研究报告，获取市场经验共识。
2.  通过历史数据进行因子分析，计算因子与收益率的**IC值**或**Rank IC**来验证其相关性方向。

在后续的实战编程中，我们将选择几组已知属性的因子（一组正向、一组负向）作为示例，这个属性判断是我们的已知条件。
