# 量化交易入门到精通：11：Python基础语法之标识符 🏷️

在本节课中，我们将要学习Python编程中的一个基础但至关重要的概念——标识符。标识符是我们为变量、函数等元素命名时必须遵循的规则集合。理解并正确使用标识符，是编写规范、可读Python代码的第一步。

## 什么是标识符？

简单来说，标识符就是我们为变量、函数等起的名字。就像在数学中用X、Y、Z代表未知数一样，在编程中我们也需要名字来指代数据。然而，Python对标识符的命名有严格的要求。

## 标识符的核心规则

以下是定义Python标识符时必须遵守的核心规则：

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_1.png)

1.  **开头字符**：标识符的第一个字符必须是字母（a-z, A-Z）或下划线（_）。不能以数字或其他符号开头。
    *   **正确示例**：`variable`, `_private`, `Var1`
    *   **错误示例**：`1var`, `@name`

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_3.png)

2.  **后续字符**：第一个字符之后，标识符可以由字母、数字（0-9）或下划线（_）组成。
    *   **正确示例**：`my_var_1`, `totalAmount`, `MAX_VALUE`

3.  **区分大小写**：Python是区分大小写的语言。这意味着 `name`、`Name` 和 `NAME` 是三个完全不同的标识符。
    *   **示例**：`age = 10` 和 `Age = 20` 是两个独立的变量。

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_5.png)

4.  **支持中文命名**：Python 3.x 支持使用Unicode字符（包括中文）作为标识符。这使得代码对中文使用者更加友好，但在团队协作或开源项目中需谨慎使用。
    *   **示例**：`市盈率 = 20`，`股票名称 = “贵州茅台”`

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_7.png)

### 规则1：开头必须是字母或下划线

以下代码演示了正确和错误的命名方式：

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_15.png)

```python
# 正确的命名
x = 1
_variable = 2
Variable1 = 3

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_17.png)

# 错误的命名（会导致语法错误）
# 1var = 4  # 错误：以数字开头
# @name = 5 # 错误：以符号开头
```

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_19.png)

### 规则2与3：后续字符与大小写敏感

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_21.png)

以下是关于后续字符组合和大小写区分的示例：

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_23.png)

```python
# 后续字符可以是字母、数字、下划线
my_var_2 = 10
Total123 = 20

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_25.png)

# Python区分大小写
value = 100
Value = 200  # 这是另一个变量
VALUE = 300  # 这是第三个变量

print(value)  # 输出：100
print(Value)  # 输出：200
```

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_27.png)

### 规则4：使用中文标识符

Python允许使用中文进行变量命名，这在处理本地化业务逻辑时可能很有用。

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_29.png)

```python
# 使用中文作为变量名
市盈率 = 20
换手率 = 15.5
股票名称 = “贵州茅台”

![](img/d7c021d3f0ad8bf5ed0c6dd1148bf68a_31.png)

# 使用f-string格式化输出
print(f”股票: {股票名称}, 市盈率: {市盈率}, 换手率: {换手率}%”)
# 输出：股票: 贵州茅台, 市盈率: 20, 换手率: 15.5%
```
