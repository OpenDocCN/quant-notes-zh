## 概述：什么是统计学？

上一节我们介绍了数据分析的整体框架，本节中我们来看看数据分析的重要工具——统计学。统计学是一门关于收集、处理、分析、解释数据并从数据中得出结论的科学。它主要分为两大模块：描述统计和推断统计。

![](img/fd0bfade0d19267f176e7466ae49551b_5.png)

*   **描述统计**：研究如何对数据进行搜集、处理、汇总、图表描述、概括与分析，以呈现数据的总体特征。
*   **推断统计**：研究如何利用样本数据来推断总体特征的统计方法。

推断统计建立在描述统计的基础之上。本节课，我们的重点是掌握描述统计。

## 描述统计的核心：三大趋势

当我们拿到一组数据（例如一个班级所有学生的考试成绩）时，描述统计主要从三个维度来帮助我们理解它：

1.  **集中趋势**：数据向哪个中心值靠拢？
2.  **离散趋势**：数据之间的差异或波动有多大？
3.  **分布形状**：数据整体的分布形态是怎样的？

接下来，我们将逐一深入探讨。

### 1. 集中趋势：数据的中心点

集中趋势度量了一组数据向中心值靠拢的程度。最常用的三个指标是平均数、中位数和众数。

以下是衡量集中趋势的三个核心指标：

*   **平均数 (Mean)**：所有数据之和除以数据个数。它反映了数据的平均水平。
    *   **公式**：`平均数 = (数据1 + 数据2 + ... + 数据n) / n`
    *   **优点**：充分利用所有数据，易于理解和计算。
    *   **缺点**：对极端值（异常大或异常小的值）非常敏感。例如，张三(3000元)、李四(4000元)、王五(5000元)和马云(40000元)的平均工资是13000元，这个值并不能很好地代表前三人的收入情况。
*   **中位数 (Median)**：将数据按大小排序后，位于正中间的那个数。如果数据个数为偶数，则取中间两个数的平均值。
    *   **优点**：不受极端值影响。在上面的例子中，中位数是4500元，更能代表大多数人的收入水平。
    *   **缺点**：只考虑了数据的位置，没有利用所有数据的信息。
*   **众数 (Mode)**：一组数据中出现次数最多的那个数。
    *   **主要用途**：常用于分类数据。例如，在一组零件日领用量数据中，领用10个的天数最多，那么众数就是10。

**Excel实现**：
*   平均数：`=AVERAGE(数据区域)`
*   中位数：`=MEDIAN(数据区域)`
*   众数：`=MODE(数据区域)`

### 2. 离散趋势：数据的波动性

了解了数据的中心后，我们还需要知道数据围绕这个中心的波动情况，即离散趋势。离散程度大，说明数据越分散；离散程度小，说明数据越集中。

以下是衡量离散趋势的几个关键指标：

![](img/fd0bfade0d19267f176e7466ae49551b_13.png)

*   **方差 (Variance)** 与 **标准差 (Standard Deviation)**：这是最常用的离散度量。方差是每个数据与平均数之差的平方的平均数。标准差是方差的平方根。标准差更常用，因为它与原始数据的单位一致。
    *   **总体方差公式**：`σ² = Σ(Xi - μ)² / N` （其中σ是标准差，μ是总体平均数）
    *   **样本方差公式**：`s² = Σ(Xi - x̄)² / (n-1)` （其中s是样本标准差，x̄是样本平均数，n-1称为“自由度”）
    *   **意义**：方差和标准差越大，说明数据波动越大。
*   **变异系数 (Coefficient of Variation, CV)**：当需要比较两组单位不同或均值相差很大的数据的离散程度时，需要使用变异系数。它是标准差与平均数的比值。
    *   **公式**：`CV = (标准差 / 平均数) * 100%`
    *   **例子**：比较配件领用量（件）和维修费（元）的波动性。直接比较标准差不合理，计算变异系数后更具可比性。
*   **四分位差 (Interquartile Range, IQR)**：将数据从小到大排序并四等分，处于25%位置的值叫下四分位数(Q1)，处于75%位置的值叫上四分位数(Q3)。IQR = Q3 - Q1。IQR反映了中间50%数据的离散范围，对极端值不敏感。

![](img/fd0bfade0d19267f176e7466ae49551b_15.png)

![](img/fd0bfade0d19267f176e7466ae49551b_17.png)

**Excel实现**：
*   方差（样本）：`=VAR.S(数据区域)`
*   标准差（样本）：`=STDEV.S(数据区域)`
*   百分位数：`=PERCENTILE.INC(数据区域, 百分比)` （例如，0.5代表中位数）
*   四分位数：`=QUARTILE.INC(数据区域, 参数)` （参数：0最小值，1下四分位，2中位数，3上四分位，4最大值）

### 3. 分布形状：数据的轮廓

最后，我们通过分布形状来观察数据整体的分布形态，主要看偏度和峰度。

*   **偏度 (Skewness)**：描述数据分布对称性的指标。
    *   **偏度 ≈ 0**：分布形态左右对称（如正态分布）。
    *   **偏度 > 0**：正偏态（右偏），数据集中在左侧，右侧有长尾。平均值 > 中位数。
    *   **偏度 < 0**：负偏态（左偏），数据集中在右侧，左侧有长尾。平均值 < 中位数。
*   **峰度 (Kurtosis)**：描述数据分布陡峭程度的指标。
    *   **峰度 ≈ 0**：与正态分布陡峭程度相同。
    *   **峰度 > 0**：尖峰分布，数据更集中在平均数附近。
    *   **峰度 < 0**：平峰分布，数据分布更为分散。

**Excel实现**：
*   偏度：`=SKEW(数据区域)`
*   峰度：`=KURT(数据区域)`

## 高效工具：Excel描述统计分析

在Excel中，我们可以一次性计算出所有主要的描述统计指标。

![](img/fd0bfade0d19267f176e7466ae49551b_25.png)

![](img/fd0bfade0d19267f176e7466ae49551b_27.png)

**操作步骤**：
1.  点击【数据】选项卡。
2.  点击【数据分析】（若未找到，需在“文件-选项-加载项”中启用“分析工具库”）。
3.  在弹出的对话框中选择【描述统计】，点击【确定】。
4.  选择输入数据区域，勾选【汇总统计】，指定输出位置，点击【确定】。

![](img/fd0bfade0d19267f176e7466ae49551b_28.png)

![](img/fd0bfade0d19267f176e7466ae49551b_30.png)

![](img/fd0bfade0d19267f176e7466ae49551b_32.png)

Excel将生成一个包含平均数、标准误差、中位数、众数、标准差、方差、峰度、偏度、极差、最小值、最大值、求和、观测数等指标的表格。

## 总结

本节课中我们一起学习了描述统计的核心内容。我们首先明确了统计学的定义及其两大分支。然后，我们深入探讨了描述统计的三大方面：

1.  **集中趋势**：通过平均数、中位数、众数来寻找数据的中心。
2.  **离散趋势**：通过方差、标准差、变异系数、四分位差来度量数据的波动或差异。
3.  **分布形状**：通过偏度和峰度来观察数据分布的对称性和陡峭度。

![](img/fd0bfade0d19267f176e7466ae49551b_34.png)

![](img/fd0bfade0d19267f176e7466ae49551b_36.png)

掌握这些指标的计算、含义及应用场景，是进行任何数据分析的第一步。它们能帮助我们对数据形成快速而准确的初步认识，为后续更深入的分析（如推断统计、建模预测）打下坚实的基础。
