## 构建回归方程

![](img/ccf184a94c0e08bdf28510abb0defe9e_11.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_13.png)

构建回归方程的方法有很多。许多工具包都可以完成这个任务。这里我们使用NumPy库中的`polyfit`函数来计算回归系数。该函数可以帮助我们拟合一个多项式回归模型。

![](img/ccf184a94c0e08bdf28510abb0defe9e_15.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_17.png)

以下是构建回归方程的关键步骤：

![](img/ccf184a94c0e08bdf28510abb0defe9e_19.png)

1.  **准备数据**：首先，我们需要两个指标的数据。在本例中，我们使用`dow_jones`和`vix`这两个指标。
2.  **调用函数**：使用`np.polyfit(x, y, deg)`函数。其中`x`和`y`是数据序列，`deg`参数指定多项式的阶数。
    *   `deg=1`：表示拟合一个线性方程 `y = kx + b`，包含`x`的一次项和常数项。
    *   `deg=2`：表示拟合一个二次方程 `y = ax² + bx + c`，包含`x`的二次项、一次项和常数项。
3.  **处理数据**：在计算前，必须确保数据中没有空值（NaN），否则可能导致计算错误。我们可以使用`dropna()`方法清理数据。

![](img/ccf184a94c0e08bdf28510abb0defe9e_21.png)

以下是核心代码示例：

![](img/ccf184a94c0e08bdf28510abb0defe9e_23.png)

```python
import numpy as np
import pandas as pd

# 假设 df 是包含 ‘dow_jones‘ 和 ‘vix‘ 列的DataFrame
# 清理数据中的空值
df[['dow_jones', 'vix']].dropna(inplace=True)

![](img/ccf184a94c0e08bdf28510abb0defe9e_25.png)

# 提取数据
x = df['dow_jones'].values
y = df['vix'].values

![](img/ccf184a94c0e08bdf28510abb0defe9e_27.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_29.png)

# 使用 polyfit 计算线性回归系数 (deg=1)
# 结果返回 [k, b]，即 y = k*x + b 中的斜率和截距
coefficients = np.polyfit(x, y, deg=1)
k, b = coefficients
print(f“回归方程为: y = {k:.4f} * x + {b:.4f}“)
```

![](img/ccf184a94c0e08bdf28510abb0defe9e_31.png)

**注意**：在金融分析中，我们通常处理的是指标的增长率或收益率，而非原始价格。在构建回归方程前，需要考虑是否需要对数据进行预处理（例如，计算对数收益率），以使关系分析更具经济意义。

## 可视化回归方程

![](img/ccf184a94c0e08bdf28510abb0defe9e_35.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_37.png)

得到回归方程的系数后，我们可以在散点图上将这条回归线画出来，使关系更直观。

以下是可视化步骤：

![](img/ccf184a94c0e08bdf28510abb0defe9e_39.png)

1.  绘制两个指标的散点图。
2.  根据回归方程 `y = kx + b`，计算出一系列对应的`y_pred`值。
3.  将回归线绘制在同一个图上。

![](img/ccf184a94c0e08bdf28510abb0defe9e_41.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_43.png)

```python
import matplotlib.pyplot as plt

![](img/ccf184a94c0e08bdf28510abb0defe9e_45.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_47.png)

# 绘制散点图

# 使用相同的 x 值范围来计算预测的 y 值

![](img/ccf184a94c0e08bdf28510abb0defe9e_53.png)
```

## 计算相关系数

除了回归方程，相关系数是衡量两个变量之间线性关系强度和方向的另一个重要指标。它的取值范围在-1到1之间。

![](img/ccf184a94c0e08bdf28510abb0defe9e_67.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_69.png)

在Pandas中，计算两个序列的相关系数非常简单。

![](img/ccf184a94c0e08bdf28510abb0defe9e_71.png)

以下是计算相关系数的方法：

![](img/ccf184a94c0e08bdf28510abb0defe9e_73.png)

```python
# 计算两个列的相关系数
correlation = df[‘dow_jones‘].corr(df[‘vix‘])
print(f“Dow Jones 与 VIX 的相关系数为: {correlation:.4f}“)

![](img/ccf184a94c0e08bdf28510abb0defe9e_75.png)

# 如果要计算整个DataFrame中所有数值列两两之间的相关系数矩阵
corr_matrix = df.corr()
print(corr_matrix)
```

![](img/ccf184a94c0e08bdf28510abb0defe9e_77.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_79.png)

相关系数矩阵是一个对称矩阵，对角线上的值均为1（每个变量与自身的完全相关）。

## 附加任务：随时间变化的相关系数

在金融分析中，指标间的相关性并非一成不变。我们可以通过计算滚动窗口（滑动窗口）内的相关系数，来观察其随时间的变化情况。

![](img/ccf184a94c0e08bdf28510abb0defe9e_85.png)

以下是实现步骤：

![](img/ccf184a94c0e08bdf28510abb0defe9e_87.png)

1.  **定义窗口**：选择一个时间窗口大小，例如250个交易日（约一年）。
2.  **滚动计算**：对其中一个指标序列，在每一个滚动窗口内，计算其与另一个指标序列的相关系数。
3.  **可视化结果**：将计算出的滚动相关系数绘制成折线图。

![](img/ccf184a94c0e08bdf28510abb0defe9e_89.png)

```python
# 设置滚动窗口大小
window_size = 250

![](img/ccf184a94c0e08bdf28510abb0defe9e_91.png)

![](img/ccf184a94c0e08bdf28510abb0defe9e_93.png)

# 计算滚动相关系数
# rolling_corr 序列的每个点代表从当前点向前回溯 window_size 个数据点所构成的窗口内的相关系数
rolling_corr = df[‘dow_jones‘].rolling(window=window_size).corr(df[‘vix‘])

![](img/ccf184a94c0e08bdf28510abb0defe9e_95.png)

# 绘制滚动相关系数变化图
plt.figure(figsize=(16, 6))
rolling_corr.plot()
plt.title(f‘Rolling Correlation (Window={window_size}) between Dow Jones and VIX‘)
plt.xlabel(‘Date‘)
plt.ylabel(‘Correlation‘)
plt.axhline(y=0, color=‘black‘, linestyle=‘--‘, alpha=0.3) # 添加y=0的参考线
plt.grid(True, alpha=0.3)
plt.show()
```

![](img/ccf184a94c0e08bdf28510abb0defe9e_97.png)

通过这张图，我们可以清晰地看到`dow_jones`和`vix`之间的相关性如何随着市场环境的变化而波动。这比单一的静态相关系数包含了更多的信息。
