我们的想法是，将股票数据从横向排列（每列一个股票）转换为纵向排列（每行一个股票）。通常，我们更喜欢纵向查看数据，例如：第一个股票的财务营收情况，第二个股票的财务营收情况，依此类推。

![](img/a45a2305529b3707f985ce05ba3e258a_13.png)

![](img/a45a2305529b3707f985ce05ba3e258a_15.png)

以下是实现数据转置的步骤：
1.  对存储筛选结果的 `DataFrame` 执行转置操作。
2.  打印转置后的结果以验证格式是否正确。
3.  从转置后的数据中提取股票名称（索引），并将其存储起来供后续交易使用。

上一节我们完成了交易前的准备工作。本节中，我们来看看如何在 `handle_bar` 函数中实现具体的交易逻辑。

![](img/a45a2305529b3707f985ce05ba3e258a_44.png)

首先，我们需要判断当前是否持有股票。这里需要参考平台的API文档来获取持仓信息。API文档中 `context.portfolio.positions` 是一个字典，它包含了当前所有的持仓信息。

![](img/a45a2305529b3707f985ce05ba3e258a_46.png)

![](img/a45a2305529b3707f985ce05ba3e258a_48.png)

以下是交易逻辑的核心步骤：
1.  **判断持仓状态**：检查 `context.portfolio.positions` 是否为空。
2.  **卖出不在股票池中的股票**：如果持仓不为空，则遍历当前持仓。如果某只股票不在我们最新的 `stock_pool` 列表中，则将其全部卖出。
3.  **买入股票池中的股票**：遍历最新的 `stock_pool` 列表。对于池中的每只股票，我们执行买入操作。为了简化，我们采用等权重分配资金的方式买入。

![](img/a45a2305529b3707f985ce05ba3e258a_50.png)

![](img/a45a2305529b3707f985ce05ba3e258a_52.png)

```python
def handle_bar(context, bar_dict):
    # 获取当前持仓
    current_positions = context.portfolio.positions

    # 判断是否有持仓
    if len(current_positions) > 0:
        # 卖出逻辑：遍历当前持仓
        for stock in current_positions.keys():
            if stock not in context.stock_pool:
                # 使用order_target_percent函数，设置目标持仓比例为0，即全卖
                order_target_percent(stock, 0)

    # 买入逻辑：遍历股票池
    for stock in context.stock_pool:
        # 计算每只股票应占的资金比例（等权重）
        target_percent = 1.0 / len(context.stock_pool)
        # 使用order_target_percent函数，调整该股票持仓至目标比例
        order_target_percent(stock, target_percent)
```

上一节我们完成了策略代码的编写。本节中，我们来看看回测运行后的结果，并分析关键的性能指标。

![](img/a45a2305529b3707f985ce05ba3e258a_64.png)

运行回测后，系统会生成一份报告。报告可能显示策略在所选时间段内收益为负，这通常与回测的时间段选择有关，可以尝试调整时间范围。

![](img/a45a2305529b3707f985ce05ba3e258a_66.png)

![](img/a45a2305529b3707f985ce05ba3e258a_68.png)

我们设计了一个策略，现在从整体上查看关键指标。回测报告主要包含以下部分：
*   **收益概况**：对比策略收益与基准（如沪深300指数）收益。
*   **风险指标**：如最大回撤、夏普比率等。

![](img/a45a2305529b3707f985ce05ba3e258a_70.png)

![](img/a45a2305529b3707f985ce05ba3e258a_72.png)

以下是需要关注的核心指标：
*   **年化收益率**：策略和基准在一年内的平均收益率。即使都为负，策略若比基准少亏损，也体现了其相对价值。
*   **最大回撤**：策略净值从高点回落的最大幅度。本例中为17%左右，属于可接受范围。
*   **夏普比率**：衡量单位风险所获得的超额回报。在收益为负时，该比率通常也为负。

![](img/a45a2305529b3707f985ce05ba3e258a_74.png)

![](img/a45a2305529b3707f985ce05ba3e258a_76.png)
