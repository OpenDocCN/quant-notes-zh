## 回测结果分析

程序运行后，我们可以在日志中看到定期的调仓记录。同时，回测引擎会生成策略的性能报告。

![](img/9918089a6536262cdf8e9d4cd6962bda_83.png)

我们主要关注以下几个核心指标：

![](img/9918089a6536262cdf8e9d4cd6962bda_85.png)

![](img/9918089a6536262cdf8e9d4cd6962bda_87.png)

*   **策略收益率 vs 基准收益率**：对比策略的累计收益与市场基准（如沪深300指数）的收益。理想情况下，策略收益应跑赢基准。
*   **年化收益率**：策略在一年内的平均收益率。公式为：
    `年化收益率 = (1 + 总收益率)^(252 / 交易天数) - 1`
*   **夏普比率**：衡量策略承担单位风险所获得的超额回报。公式为：
    `夏普比率 = (策略年化收益率 - 无风险利率) / 策略收益率的年化标准差`
    该值越高，说明风险调整后的收益越好。
*   **最大回撤**：策略净值从峰值到谷底的最大跌幅。该值越小，说明策略控制下行风险的能力越强。

![](img/9918089a6536262cdf8e9d4cd6962bda_89.png)

![](img/9918089a6536262cdf8e9d4cd6962bda_91.png)

在本例的回测结果中，虽然整体市场基准收益为负，但我们的策略实现了小幅正收益，并且在夏普比率和最大回撤等风险指标上表现优于基准。这表明基于市值和市净率因子的简单筛选和定期调仓策略，在测试区间内取得了一定的正向效果。

本节课我们一起学习了量化策略从构建到评估的完整流程：

![](img/9918089a6536262cdf8e9d4cd6962bda_113.png)

![](img/9918089a6536262cdf8e9d4cd6962bda_115.png)

![](img/9918089a6536262cdf8e9d4cd6962bda_117.png)

1.  **初始化与定时**：在策略初始化函数中设定再平衡的频率（如每月）。
2.  **股票池筛选**：
    *   **初步过滤**：剔除不符合基本条件的股票（如ST股、停牌股）。
    *   **因子获取与处理**：获取所需的因子数据（如市值、市净率），并进行标准化、中性化等预处理。
    *   **因子筛选**：根据因子逻辑（如选择市净率低的股票）筛选出最终的股票池。
3.  **调仓执行**：
    *   **卖出**：卖出当前持有但不在新股票池中的股票。
    *   **买入**：按照既定规则（如等权重）买入新股票池中的股票。
4.  **效果评估**：通过回测，分析策略的收益率、风险指标（夏普比率、最大回撤），并与基准进行比较，以评估策略的有效性。

![](img/9918089a6536262cdf8e9d4cd6962bda_119.png)

![](img/9918089a6536262cdf8e9d4cd6962bda_121.png)

![](img/9918089a6536262cdf8e9d4cd6962bda_123.png)

需要注意的是，本教程中的策略仅为演示流程，实际应用中需要更严谨的因子研究、风险控制和实盘验证。回测结果不代表未来表现。
