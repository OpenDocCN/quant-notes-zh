## 业务指标 📊

业务指标是衡量业务状况的量化标准。理解并选择合适的指标是数据分析的第一步。

## 常用分析方法 🔍

以下是数据分析中几种核心的分析方法，每种方法都从不同角度帮助我们洞察数据。

![](img/0ff35cce6f2d8f0db404ee146afded7f_3.png)

### 逻辑树分析

![](img/0ff35cce6f2d8f0db404ee146afded7f_5.png)

逻辑树分析是一种结构化分解问题的方法，有助于锻炼系统性思维。它通常用于将复杂问题拆解为更小、更易管理的子问题。

![](img/0ff35cce6f2d8f0db404ee146afded7f_7.png)

### 多维度拆解分析

面对总体数据时，我们需要从不同维度进行拆解和细化，以定位到可具体分析的点。多维度拆解分析正是实现这一目的的关键技术。

![](img/0ff35cce6f2d8f0db404ee146afded7f_9.png)

### 对比分析

![](img/0ff35cce6f2d8f0db404ee146afded7f_11.png)

对比分析是最常用的方法之一，通过比较不同群体、不同时间点或不同条件下的数据来发现差异和趋势。

### 归因分析（假设检验法）

归因分析用于探究业务问题的根本原因。其核心是**假设检验法**：先提出一个推测（假设），然后收集证据去验证这个假设是否正确。

### RFM分析模型

RFM模型是一种客户价值分析工具，通过三个关键指标对客户进行分类：
*   **R（Recency）**：最近一次消费时间。
*   **F（Frequency）**：消费频率。
*   **M（Monetary）**：消费金额。

### 漏斗分析模型

![](img/0ff35cce6f2d8f0db404ee146afded7f_13.png)

漏斗分析模型用于追踪和优化用户在不同阶段的行为转化过程。一个著名的扩展模型是AARRR模型，它描述了用户生命周期的五个阶段：
*   **Acquisition（获取）**
*   **Activation（激活）**
*   **Retention（留存）**
*   **Revenue（收入）**
*   **Referral（推荐）**

![](img/0ff35cce6f2d8f0db404ee146afded7f_14.png)

### 杜邦分析法

杜邦分析法是一种用于评估企业财务状况和盈利能力的经典方法。它通过分解净资产收益率（ROE）来深入分析业绩驱动因素。其核心公式为：
**净资产收益率（ROE）= 销售净利率 × 总资产周转率 × 权益乘数**
由于涉及专业财务知识，本课程仅作了解，建议学员通过配套文档进行深入学习。
