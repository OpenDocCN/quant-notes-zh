`product` 函数能够帮助我们生成两个或多个序列的笛卡尔积。例如，给定两个参数空间 `[20, 24]` 和 `[180, 190]`，`product` 会生成所有组合：`(20, 180)`, `(20, 190)`, `(24, 180)`, `(24, 190)`。

以下是循环内的主要步骤：

![](img/8c1b34740f80cd93b73050f0585a2118_94.png)

1.  **数据准备**：重新加载苹果公司（AAPL）的股价数据，并处理缺失值。
2.  **计算指标**：根据当前的 `sma1` 和 `sma2` 参数，计算短期均线、长期均线和日收益率。
3.  **生成交易信号**：当短期均线上穿长期均线时，产生买入信号（仓位为1）；下穿时，产生卖出信号（仓位为0）。
4.  **计算策略收益**：根据交易信号和日收益率，计算策略的累计收益。
5.  **记录结果**：将当前参数组合、基准收益、策略收益以及策略相对于基准的收益差（`out`）保存下来。

![](img/8c1b34740f80cd93b73050f0585a2118_102.png)

![](img/8c1b34740f80cd93b73050f0585a2118_104.png)

![](img/8c1b34740f80cd93b73050f0585a2118_108.png)

```python
# 示例代码结构
for sma1, sma2 in product(sma1_range, sma2_range):
    # 1. 加载并预处理数据
    data = ... # 加载AAPL数据
    data = data.dropna()

    # 2. 计算技术指标
    data[‘returns’] = ... # 计算日收益率
    data[‘SMA1’] = data[‘Adj Close’].rolling(window=sma1).mean()
    data[‘SMA2’] = data[‘Adj Close’].rolling(window=sma2).mean()
    data = data.dropna()

    # 3. 生成交易信号
    data[‘position’] = ... # 根据SMA1和SMA2的交叉情况赋值0或1

    # 4. 计算策略收益
    data[‘strategy’] = data[‘position’].shift(1) * data[‘returns’]
    data = data.dropna()

    # 5. 汇总本次回测结果
    result_table = pd.DataFrame({
        ‘SMA1’: [sma1],
        ‘SMA2’: [sma2],
        ‘returns’: [data[‘returns’].sum()],
        ‘strategy’: [data[‘strategy’].sum()],
        ‘out’: [data[‘strategy’].sum() - data[‘returns’].sum()]
    })

    # 6. 将本次结果追加到总结果表中
    results = pd.concat([results, result_table], ignore_index=True)
```
