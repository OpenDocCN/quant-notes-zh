# 金融时间序列分析：P14：金融时间序列-II-卡尔曼滤波 📈

在本节课中，我们将要学习卡尔曼滤波的理论基础、推导过程及其在金融时间序列分析中的应用。卡尔曼滤波是一种强大的状态空间模型估计算法，特别适用于处理含有噪声的观测数据，并估计系统内部不可直接观测的状态。

## 理论介绍与问题定义 🔍

![](img/5bc79037fb0f16244aaebb31bf77f43b_1.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_2.png)

在介绍卡尔曼滤波理论之前，我们需要先弄清楚几种状态估计的情况。

以下是三种常见的估计问题：

1.  **预测问题**：基于从第1时刻到第T-1时刻的观测值 `Y1, ..., Y(T-1)`，来估计第T时刻的状态 `X_T`。这称为向前一步预测。
2.  **滤波问题**：基于到第T时刻为止的所有观测值 `Y1, ..., Y_T`，来估计第T时刻的内部状态 `X_T`。这是卡尔曼滤波的核心。
3.  **平滑问题**：基于整个时间序列（例如到第N时刻，N > T）的所有观测值，回过头来估计第T时刻的状态 `X_T`。这利用了未来的信息对当前状态进行平滑。

![](img/5bc79037fb0f16244aaebb31bf77f43b_4.png)

在本节讨论卡尔曼滤波时，我们主要用到前两个问题：**预测**和**滤波**。这两个问题在与高斯过程结合时尤为有效，即假设模型中的噪声项 `V_t` 和 `Z_t` 都是独立的标准正态分布。

## 卡尔曼滤波的实现过程 ⚙️

![](img/5bc79037fb0f16244aaebb31bf77f43b_6.png)

上一节我们定义了问题，本节中我们来看看卡尔曼滤波的具体实现过程。

![](img/5bc79037fb0f16244aaebb31bf77f43b_8.png)

首先，我们需要介绍卡尔曼滤波的两条核心状态空间模型方程：
*   **状态更新方程（动态方程）**：`X_t = A * X_(t-1) + V_t`
*   **观测方程**：`Y_t = C * X_t + Z_t`

![](img/5bc79037fb0f16244aaebb31bf77f43b_10.png)

其中，`V_t` 和 `Z_t` 是独立的高斯噪声。通过这两条方程，我们可以得到如下概率描述：
*   状态 `X_t` 的概率分布仅依赖于前一状态 `X_(t-1)`，因为 `V_t` 是独立的噪声。
*   观测 `Y_t` 的概率分布完全由当前状态 `X_t` 决定。

![](img/5bc79037fb0f16244aaebb31bf77f43b_12.png)

这个过程可以形象地理解为：存在一个内部的、隐藏的状态过程 `X_t` 在自主演化。在每一个时刻 `t`，这个内部过程会对外产生一个我们能观测到的信号 `Y_t`。因此，我们站在外部，观测到的是绿色的 `Y_t` 序列，但真正驱动整个模型的是里面这个隐藏的动态更新过程 `X_t`。卡尔曼滤波的这一整套隐藏状态与观测过程，实际上是**隐马尔可夫模型**的一个应用。

## 滤波与预测的两步流程 🔄

![](img/5bc79037fb0f16244aaebb31bf77f43b_14.png)

现在让我们走一遍滤波与预测的流程。滤波过程是反向的，因为我们拥有观测值 `Y_t`，目标是估计状态 `X_t`。

卡尔曼滤波是一个两步迭代过程：
1.  **预测步**：利用到 `t-1` 时刻为止的信息，预测 `t` 时刻的状态。即计算概率 `P(X_t | Y1, ..., Y_(t-1))`。
2.  **更新步（滤波步）**：结合 `t` 时刻的新观测 `Y_t`，更新对 `t` 时刻状态的估计。即计算概率 `P(X_t | Y1, ..., Y_t)`。

![](img/5bc79037fb0f16244aaebb31bf77f43b_16.png)

**核心思想是**：首先通过预测步得到一个先验估计，然后利用新观测数据在更新步中对这个先验估计进行修正，得到后验估计（即滤波结果）。

### 预测步推导

![](img/5bc79037fb0f16244aaebb31bf77f43b_18.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_20.png)

预测步的目标是计算 `P(X_t | Y1, ..., Y_(t-1))`。

我们从联合概率 `P(X_t, X_(t-1) | Y1, ..., Y_(t-1))` 开始，利用概率乘法法则和模型的马尔可夫性质（即给定 `X_(t-1)`，`X_t` 与更早的观测独立），可以将其拆解。然后，通过对 `X_(t-1)` 积分（或求和），边缘化掉这个变量，最终得到我们想要的 `P(X_t | Y1, ..., Y_(t-1))` 的表达式。这个表达式是一个高斯分布，其均值和协方差可以通过公式递推得到。

### 更新步推导

更新步的目标是计算 `P(X_t | Y1, ..., Y_t)`。

![](img/5bc79037fb0f16244aaebb31bf77f43b_22.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_24.png)

我们使用贝叶斯公式：
`P(X_t | Y1, ..., Y_t) ∝ P(Y_t | X_t) * P(X_t | Y1, ..., Y_(t-1))`

![](img/5bc79037fb0f16244aaebb31bf77f43b_26.png)

这里：
*   `P(X_t | Y1, ..., Y_(t-1))` 是**预测步**得到的**先验分布**。
*   `P(Y_t | X_t)` 是**观测方程**给出的**似然函数**。

![](img/5bc79037fb0f16244aaebb31bf77f43b_28.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_30.png)

通过计算这两个高斯分布的乘积，并归一化，我们可以得到后验分布 `P(X_t | Y1, ..., Y_t)`，它同样是一个高斯分布。其均值和协方差的更新公式就是卡尔曼滤波的核心公式。

![](img/5bc79037fb0f16244aaebb31bf77f43b_32.png)

## 卡尔曼滤波公式与迭代 🔢

![](img/5bc79037fb0f16244aaebb31bf77f43b_34.png)

上一节我们推导了滤波的理论步骤，本节中我们来看看具体的迭代公式。

![](img/5bc79037fb0f16244aaebb31bf77f43b_36.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_38.png)

给定状态空间模型：
*   `X_t = A * X_(t-1) + V_t`, `V_t ~ N(0, Q)`
*   `Y_t = C * X_t + Z_t`, `Z_t ~ N(0, R)`

![](img/5bc79037fb0f16244aaebb31bf77f43b_40.png)

并假设在 `t-1` 时刻，我们已经得到了滤波估计：状态均值 `X_(t-1|t-1)` 和状态协方差 `P_(t-1|t-1)`。

![](img/5bc79037fb0f16244aaebb31bf77f43b_42.png)

**预测步（从 t-1 到 t）**：
1.  预测状态均值：`X_(t|t-1) = A * X_(t-1|t-1)`
2.  预测状态协方差：`P_(t|t-1) = A * P_(t-1|t-1) * A^T + Q`

![](img/5bc79037fb0f16244aaebb31bf77f43b_44.png)

**更新步（在 t 时刻）**：
1.  计算卡尔曼增益 `K_t`：
    `K_t = P_(t|t-1) * C^T * (C * P_(t|t-1) * C^T + R)^(-1)`
2.  更新状态均值（滤波结果）：
    `X_(t|t) = X_(t|t-1) + K_t * (Y_t - C * X_(t|t-1))`
3.  更新状态协方差：
    `P_(t|t) = (I - K_t * C) * P_(t|t-1)`

![](img/5bc79037fb0f16244aaebb31bf77f43b_46.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_48.png)

**公式解读**：
*   `X_(t|t-1)`：基于 `t-1` 时刻信息对 `t` 时刻状态的预测。
*   `Y_t - C * X_(t|t-1)`：观测残差或新息，代表了实际观测与预测观测之间的差异。
*   **卡尔曼增益 `K_t`**：是一个权重矩阵，决定了我们应该在多大程度上相信新的观测 `Y_t`。当观测噪声 `R` 很小时，增益 `K_t` 大，更信任新观测；当预测不确定性 `P_(t|t-1)` 大时，增益 `K_t` 也大，也更倾向于用新观测来修正。
*   `X_(t|t)`：结合了预测和新观测信息后的最优状态估计。

![](img/5bc79037fb0f16244aaebb31bf77f43b_50.png)

## 应用实例：随机游走与线性回归 📊

![](img/5bc79037fb0f16244aaebb31bf77f43b_52.png)

### 随机游走滤波
考虑一个简单的随机游走状态模型：
`X_t = X_(t-1) + W_t`, `Y_t = X_t + e_t`
应用卡尔曼滤波后，可以有效地从充满噪声的观测序列 `Y_t` 中，提取出相对平滑的真实状态信号 `X_t` 的估计。

![](img/5bc79037fb0f16244aaebb31bf77f43b_54.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_56.png)

### 时变参数线性回归
在金融中，很多关系并非一成不变。例如，配对交易中的对冲比率 `β` 可能随时间变化。我们可以用卡尔曼滤波来估计时变参数。

![](img/5bc79037fb0f16244aaebb31bf77f43b_58.png)

将线性回归模型 `Y_t = α_t + β_t * X_t + e_t` 改写为状态空间形式：
*   **状态**：`θ_t = [α_t, β_t]^T`
*   **状态方程**：假设参数缓慢变化，例如 `θ_t = θ_(t-1) + V_t`（随机游走）。
*   **观测方程**：`Y_t = [1, X_t] * θ_t + Z_t`

![](img/5bc79037fb0f16244aaebb31bf77f43b_60.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_62.png)

通过卡尔曼滤波，我们可以动态地更新对 `α_t` 和 `β_t` 的估计，从而更灵活地捕捉市场关系的变化。

![](img/5bc79037fb0f16244aaebb31bf77f43b_64.png)

以下是一个使用 `pykalman` 库实现卡尔曼滤波进行时变参数估计的简要示例框架：

![](img/5bc79037fb0f16244aaebb31bf77f43b_74.png)

```python
# 安装库: pip install pykalman
from pykalman import KalmanFilter
import numpy as np
import pandas as pd

![](img/5bc79037fb0f16244aaebb31bf77f43b_76.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_78.png)

# 假设我们有两只ETF的价格序列: price_a 和 price_b
# 例如，price_a 是标的资产，price_b 是我们想对其建模的资产

![](img/5bc79037fb0f16244aaebb31bf77f43b_80.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_82.png)

# 1. 准备观测数据：price_b 的收益率作为 Y， price_a 的收益率作为特征 X
# 2. 定义卡尔曼滤波器参数
# 这里简化设置，实际应用中需要根据情况调整初始值和噪声协方差
kf = KalmanFilter(
    transition_matrices=[1], # 状态转移矩阵 A，这里假设状态是随机游走
    observation_matrices=np.ones((len(price_a), 1)), # 观测矩阵 C，这里设计为包含常量和price_a收益率的矩阵，示例简化
    initial_state_mean=0,
    initial_state_covariance=1,
    observation_covariance=1, # R
    transition_covariance=0.01 # Q
)

![](img/5bc79037fb0f16244aaebb31bf77f43b_84.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_86.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_88.png)

# 3. 应用滤波和平滑
# state_means, state_covs = kf.filter(Y) # 仅滤波
state_means, state_covs = kf.smooth(Y) # 平滑，使用全部数据估计每个时刻状态

![](img/5bc79037fb0f16244aaebb31bf77f43b_90.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_92.png)

![](img/5bc79037fb0f16244aaebb31bf77f43b_94.png)

# state_means 就包含了每个时刻估计出的时变参数（例如α和β）
```
