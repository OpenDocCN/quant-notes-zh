在本节课中，我们将学习如何分析时间序列数据的变化情况。具体来说，我们将计算两种关键指标：**每日差异值**和**增长率**。这些指标对于理解股票价格或其他金融数据的动态变化至关重要。

上一节我们介绍了数据的基本处理，本节中我们来看看如何量化数据的变化。

## 计算每日差异值

![](img/abda14a8a53874c8c41766d82ec825e3_6.png)

首先，我们需要计算每一天与前一天的指标差异值。例如，在股票数据中，我们不仅关心当天的收盘价，更关心它相比前一天是上涨还是下跌了多少。

![](img/abda14a8a53874c8c41766d82ec825e3_8.png)

在Pandas中，计算这个差异值非常简单。我们可以使用`.diff()`方法，它会自动用当前行的值减去前一行的值。

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

![](img/abda14a8a53874c8c41766d82ec825e3_10.png)

![](img/abda14a8a53874c8c41766d82ec825e3_12.png)

执行上述代码后，你会看到一个新的DataFrame。例如，1月5日的值等于1月5日的数据减去1月4日的数据。由于1月4日之前没有数据，所以第一行会显示为`NaN`（Not a Number）。

这个指标在时间序列分析中非常常用。`.diff()`方法默认计算相邻一天的差异，但你也可以通过指定参数来计算间隔多天的差异（例如，`.diff(periods=2)`计算与前两天的差异）。

为了养成良好习惯，当你对函数参数不熟悉时，建议查阅官方文档或相关资料。

## 计算增长率

![](img/abda14a8a53874c8c41766d82ec825e3_18.png)

差异值是一个绝对值，但有时我们更关心相对变化，即**增长率**。例如，从1元涨到2元是100%的增长，而从10000元涨到10001元增长幅度则微乎其微。

增长率的计算公式是：`(后一天的值 - 前一天的值) / 前一天的值`。

![](img/abda14a8a53874c8c41766d82ec825e3_20.png)

Pandas同样提供了现成的方法`.pct_change()`来计算这个百分比变化。

```python
# 计算每日增长率
pct_data = data.pct_change()
print(pct_data.head())
```

![](img/abda14a8a53874c8c41766d82ec825e3_22.png)

计算后，数据会从具体的差异值变为百分比形式的增长率。通常增长率的值会很小，例如0.001（即0.1%），这在股票市场中是正常现象。

![](img/abda14a8a53874c8c41766d82ec825e3_24.png)

![](img/abda14a8a53874c8c41766d82ec825e3_26.png)

## 分析增长率的统计特性

![](img/abda14a8a53874c8c41766d82ec825e3_28.png)

![](img/abda14a8a53874c8c41766d82ec825e3_30.png)

得到每日增长率序列后，我们可以进一步分析其整体统计特性。例如，计算所有交易日增长率的平均值，可以了解该股票或指标在观察期内的平均表现。

![](img/abda14a8a53874c8c41766d82ec825e3_32.png)

```python
# 计算增长率的平均值
mean_growth = pct_data.mean()
print(mean_growth)
```

![](img/abda14a8a53874c8c41766d82ec825e3_34.png)

为了更直观地展示各个指标的平均增长率，我们可以将其可视化。由于平均值是单个数值，适合用条形图来展示。

以下是绘制平均增长率条形图的代码：

![](img/abda14a8a53874c8c41766d82ec825e3_36.png)

```python
# 绘制平均增长率的条形图
mean_growth.plot(kind='bar', figsize=(16, 8))
```

![](img/abda14a8a53874c8c41766d82ec825e3_38.png)

通过这个图表，各个指标的平均增长情况一目了然，便于我们进行横向比较。

![](img/abda14a8a53874c8c41766d82ec825e3_40.png)

![](img/abda14a8a53874c8c41766d82ec825e3_42.png)
