## 凸函数的扩展

![](img/f0ad11ff800711f7a1dece891b51b9ea_8.png)

![](img/f0ad11ff800711f7a1dece891b51b9ea_10.png)

![](img/f0ad11ff800711f7a1dece891b51b9ea_12.png)

上一节我们介绍了凸函数的基本定义，本节中我们来看看如何将一个定义在凸集上的凸函数，扩展至整个空间。

![](img/f0ad11ff800711f7a1dece891b51b9ea_14.png)

![](img/f0ad11ff800711f7a1dece891b51b9ea_16.png)

![](img/f0ad11ff800711f7a1dece891b51b9ea_18.png)

![](img/f0ad11ff800711f7a1dece891b51b9ea_20.png)

对于一个凸函数 `f(x)`，我们可以进行如下扩展：如果 `x` 在 `f` 的定义域 `dom f` 内，则函数值保持不变；如果 `x` 在定义域外，则将其函数值设为正无穷大 `+∞`。

![](img/f0ad11ff800711f7a1dece891b51b9ea_22.png)

用公式描述这个扩展函数 `f̃(x)` 如下：
```
f̃(x) = { f(x),   if x ∈ dom f
       { +∞,     if x ∉ dom f
```

经过这样的扩展后，`f̃(x)` 是否仍然是一个凸函数？我们可以从凸函数的两个条件来验证。

首先，扩展后函数的定义域是整个 `n` 维空间，这显然是一个凸集。

![](img/f0ad11ff800711f7a1dece891b51b9ea_24.png)

其次，我们需要验证凸不等式是否成立。我们可以通过想象一个“铁锅”的比喻来理解：原来的凸函数像是锅底部分，扩展后，锅底以外的区域都被提升到了“天空”（正无穷）的高度。现在，考虑任意两点 `x` 和 `y`，以及 `θ ∈ [0, 1]`，有以下几种情况：

以下是所有可能的情况分析：
*   **情况一**：`x` 和 `y` 都在原定义域内。此时不等式就是原凸函数的不等式，自然成立。
*   **情况二**：`x` 和 `y` 都在原定义域外。此时不等式两边都是 `+∞`，`+∞ ≤ +∞` 成立。
*   **情况三**：一点在内，一点在外。此时不等式右边为 `+∞`，左边为一个有限值或 `+∞`，因此 `f̃(θx + (1-θ)y) ≤ +∞` 恒成立。

综上所述，经过此扩展得到的函数 `f̃(x)` 仍然是一个凸函数。在后续讨论中，我们通常会默认使用这种扩展后的凸函数，因为它定义在整个空间上，处理起来更为方便。

## 示性函数示例

为了加深理解，我们来看一个具体的例子：集合的示性函数。

设 `C` 是 `n` 维空间中的一个凸集。我们可以定义其示性函数 `f_c(x)`：
```
f_c(x) = { 0,   if x ∈ C
         { 未定义, if x ∉ C
```
这个函数的定义域是凸集 `C`，并且在定义域上恒为0，因此它是一个凸函数。

接着，我们将其扩展至整个空间，得到扩展的示性函数 `I_c(x)`：
```
I_c(x) = { 0,       if x ∈ C
         { +∞,     if x ∉ C
```
根据上一节的结论，`I_c(x)` 也是一个凸函数。

现在，考虑另一个函数 `g(x)`：
```
g(x) = { 0,   if x ∈ C
       { 1,   if x ∉ C
```
`g(x)` 是凸函数吗？我们以一维情况为例，令 `C = [-1, 1]`。函数图像像一个“平底帽”，在区间内值为0，区间外值为1。我们可以轻易找到两点（例如 `x=-2, y=2`），使得凸不等式不成立。因此，`g(x)` 既不是凸函数也不是凹函数。这个例子说明了为什么扩展时必须使用 `+∞`，使用有限值（如1）会破坏凸性。

## 凸函数的一阶条件

![](img/f0ad11ff800711f7a1dece891b51b9ea_26.png)

![](img/f0ad11ff800711f7a1dece891b51b9ea_27.png)

![](img/f0ad11ff800711f7a1dece891b51b9ea_29.png)

前面我们介绍了凸函数的两个基本定义，本节中我们将学习凸函数的第三个等价定义——一阶条件，它涉及函数的梯度（一阶导数）。

假设函数 `f: R^n → R` 是可微的（即梯度 `∇f` 在定义域内处处存在），并且定义域 `dom f` 是一个凸集。那么，`f` 是凸函数当且仅当对于定义域内任意两点 `x, y`，下式成立：
```
f(y) ≥ f(x) + ∇f(x)^T (y - x)
```
其中 `∇f(x)^T` 表示梯度向量 `∇f(x)` 的转置。

这个不等式有什么几何意义呢？我们考虑一维情况 `(n=1)`。不等式右边 `f(x) + f'(x)(y - x)` 正是函数在点 `x` 处的切线方程在 `y` 点的值。因此，一阶条件表明：**一个可微函数是凸的，当且仅当其图像总是在其任意一点的切线的上方**。就像“铁锅”的锅身始终在任何一根贴在锅底的棍子（切线）之上。

这个性质是凸函数最重要的性质之一。它与优化问题密切相关。例如，如果对于一个可微凸函数 `f`，我们找到一点 `x*` 使得其梯度为零 `∇f(x*) = 0`，那么对于任意 `y`，有：
```
f(y) ≥ f(x*) + 0^T (y - x*) = f(x*)
```
这意味着 `f(x*)` 就是函数的最小值，`x*` 是一个全局最优点。如果函数是严格凸的，那么这个最优点还是唯一的。

## 一阶条件的证明（一维情况）

![](img/f0ad11ff800711f7a1dece891b51b9ea_31.png)

为了更深刻地理解一阶条件，我们尝试用凸函数的基本定义（定义一）来证明它在一维情况下的等价性。证明分为两个方向。

**方向一：若 `f` 凸，则一阶条件成立。**

已知 `f` 是凸函数，其定义域 `dom f` 是凸集。在定义域内任取 `x, y`，且 `x ≠ y`。对于任意 `t ∈ (0, 1]`，根据凸函数的定义有：
```
f(x + t(y - x)) ≤ (1-t)f(x) + t f(y)
```
整理上式：
```
f(x + t(y - x)) - f(x) ≤ t (f(y) - f(x))
```
由于 `t > 0`，两边同除以 `t`：
```
[ f(x + t(y - x)) - f(x) ] / t ≤ f(y) - f(x)
```
现在，令 `t → 0+`。不等式左边是函数 `f` 在 `x` 点沿 `(y-x)` 方向的方向导数的定义，在一维情况下就是导数 `f'(x)`。因此取极限得到：
```
f'(x) (y - x) ≤ f(y) - f(x)
```
移项即得：
```
f(y) ≥ f(x) + f'(x) (y - x)
```
这正是我们要证明的一阶条件。

![](img/f0ad11ff800711f7a1dece891b51b9ea_33.png)

![](img/f0ad11ff800711f7a1dece891b51b9ea_35.png)

**方向二：若一阶条件成立，则 `f` 凸。**

已知定义域为凸集，且对定义域内任意 `u, v` 有 `f(v) ≥ f(u) + f'(u) (v - u)`。现在需要证明 `f` 是凸的，即对任意 `x, y ∈ dom f` 和 `θ ∈ [0, 1]`，有：
```
f(θx + (1-θ)y) ≤ θ f(x) + (1-θ) f(y)
```
令 `z = θx + (1-θ)y`。根据已知的一阶条件，我们有两个不等式：
1.  对点 `z` 和 `x`：`f(x) ≥ f(z) + f'(z) (x - z)`
2.  对点 `z` 和 `y`：`f(y) ≥ f(z) + f'(z) (y - z)`

将第一个不等式乘以 `θ`，第二个不等式乘以 `(1-θ)`，然后相加：
```
θ f(x) + (1-θ) f(y) ≥ [θ + (1-θ)] f(z) + f'(z) [θ(x - z) + (1-θ)(y - z)]
```
化简右边括号内项：
```
θ(x - z) + (1-θ)(y - z) = θx + (1-θ)y - [θ + (1-θ)]z = z - z = 0
```
因此，不等式变为：
```
θ f(x) + (1-θ) f(y) ≥ f(z) = f(θx + (1-θ)y)
```
这正是凸函数的定义。证明完毕。

本节课中我们一起学习了凸函数的扩展方法、示性函数的例子，以及凸函数至关重要的**一阶条件**及其在一维情况下的证明。一阶条件 `f(y) ≥ f(x) + ∇f(x)^T (y - x)` 是可微凸函数的充要条件，具有清晰的几何意义：函数图像始终在其切平面之上。下节课我们将探讨高维情况下的一阶条件证明。
