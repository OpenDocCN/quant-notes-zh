## 计算每日差异值

![](img/62c64fd0f959d7269bb425459098712d_6.png)

首先，我们需要计算每一天指标相对于前一天的差异值。例如，在股票数据中，我们不仅关心当天的价格，更关心它相比前一天是上涨还是下跌了多少。这个差异值可以通过后一天的数据减去前一天的数据得到。

![](img/62c64fd0f959d7269bb425459098712d_8.png)

在Pandas中，计算这种差异非常简单，我们可以直接使用`.diff()`方法。

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

执行上述代码后，你会看到一个新的DataFrame。例如，1月5日的数据是0.052，这个值正是由1月5日的数据减去1月4日的数据得到的。而1月4日对应的值是`NaN`（Not a Number），因为在这一天之前没有数据可供相减。

![](img/62c64fd0f959d7269bb425459098712d_10.png)

![](img/62c64fd0f959d7269bb425459098712d_12.png)

`.diff()`方法默认计算的是相邻一天（即周期为1）的差异。我们也可以计算间隔多天的差异。

以下是调整计算周期的方法：
*   **`periods=1`**： 默认值，计算后一天与前一天的差异。
*   **`periods=2`**： 计算当前数据与两天前数据的差异。
*   **`periods=7`**： 计算当前数据与一周前数据的差异。

## 计算增长率（变化率）

![](img/62c64fd0f959d7269bb425459098712d_16.png)

仅仅知道差异值有时还不够。例如，一支股票从1元涨到2元（差异1元）和从10000元涨到10001元（差异也是1元），其意义完全不同。前者翻倍，后者几乎没变。因此，我们常常需要计算**增长率**或**变化率**。

![](img/62c64fd0f959d7269bb425459098712d_18.png)

增长率的计算公式为：**`(后一个值 - 前一个值) / 前一个值`**。

Pandas同样提供了现成的方法`.pct_change()`来计算这个百分比变化。

![](img/62c64fd0f959d7269bb425459098712d_20.png)

```python
# 计算每日增长率
pct_change_data = data.pct_change()
print(pct_change_data.head())
```

计算后，数据会从具体的差异值转换为百分比。例如，0.052可能变为0.05（即5%）。为了显示更清晰，我们可以设置显示的小数位数。

![](img/62c64fd0f959d7269bb425459098712d_22.png)

```python
# 设置显示格式，保留3位小数
pd.set_option('display.float_format', lambda x: '%.3f' % x)
print(pct_change_data.head())
```

![](img/62c64fd0f959d7269bb425459098712d_24.png)

![](img/62c64fd0f959d7269bb425459098712d_26.png)

在金融数据中，日增长率通常是一个较小的数值，例如0.1%或0.2%，这属于正常范围。

![](img/62c64fd0f959d7269bb425459098712d_28.png)

## 统计分析与可视化

得到增长率序列后，我们可以对其进行统计分析，例如计算所有交易日增长率的平均值，以了解该指标的整体趋势。

![](img/62c64fd0f959d7269bb425459098712d_32.png)

![](img/62c64fd0f959d7269bb425459098712d_34.png)

```python
# 计算各列增长率的平均值
mean_pct_change = pct_change_data.mean()
print(mean_pct_change)
```

为了更直观地比较不同指标的平均增长率，我们可以将其可视化。由于平均值是单个数值，适合用条形图来展示。

以下是绘制平均增长率条形图的步骤：
1.  调用`.plot()`方法。
2.  通过`kind`参数指定图形类型为条形图（`‘bar’`）。
3.  通过`figsize`参数设置图形大小。

![](img/62c64fd0f959d7269bb425459098712d_36.png)

```python
# 绘制平均增长率的条形图
mean_pct_change.plot(kind='bar', figsize=(16, 8))
```

![](img/62c64fd0f959d7269bb425459098712d_38.png)

生成的图表将清晰地展示各个指标的平均增长水平，Y轴的值通常较小，符合金融数据的特征。通过这张图，我们可以一目了然地对比不同指标的增长表现。

![](img/62c64fd0f959d7269bb425459098712d_40.png)
