## 获取股票数据 📊

![](img/dc7f92db21b221695d750ab200f05593_15.png)

在设定了策略的运行框架后，我们现在需要获取具体的股票数据。我们将获取市场上所有股票的基本信息，作为后续筛选的起点。

![](img/dc7f92db21b221695d750ab200f05593_17.png)

我们使用一个名为 `get_all_securities` 的API函数来获取数据。这个函数可以返回指定市场（例如A股）的所有证券信息。

![](img/dc7f92db21b221695d750ab200f05593_19.png)

```python
# 获取所有股票数据，'cs' 参数通常代表普通股票
all_stocks = get_all_securities(types=['cs'])
```

获取到所有股票数据后，我们通常会得到一个包含股票代码、名称等信息的表格。然而，在实际策略中，我们很少会交易所有股票，因此需要对初始股票池进行过滤。

## 初步筛选股票池 🔍

![](img/dc7f92db21b221695d750ab200f05593_23.png)

我们已经获得了基础的股票列表，但直接使用全部股票并不现实。本节中，我们将对初始股票池进行初步筛选，剔除不符合基本条件的股票，例如ST股、次新股或流动性不足的股票。

![](img/dc7f92db21b221695d750ab200f05593_25.png)

以下是常见的初步筛选步骤：

![](img/dc7f92db21b221695d750ab200f05593_27.png)

*   **剔除ST/PT股票**：这些股票存在退市风险，通常需要排除。
*   **剔除上市时间过短的股票**：例如，剔除上市不足60个交易日的股票，以避免次新股异常波动的影响。
*   **剔除流动性差的股票**：例如，剔除日均成交额过低的股票，确保策略的可交易性。

![](img/dc7f92db21b221695d750ab200f05593_29.png)

具体的筛选条件可以根据策略需求进行调整。经过这一步，我们就得到了一个更干净、更适合进行因子分析的备选股票池。
