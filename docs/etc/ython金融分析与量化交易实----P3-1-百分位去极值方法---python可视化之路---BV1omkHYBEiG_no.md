# 金融数据分析：P3：百分位去极值方法

在本节课中，我们将学习如何对金融因子数据进行预处理。这是构建量化模型前至关重要的一步，良好的数据预处理能有效提升模型的稳定性和预测能力。我们将重点介绍三种核心的预处理方法：去极值、标准化和中性化。本节课，我们先从第一步——去极值开始。

## 因子数据简介

在开始处理之前，我们先明确什么是因子。因子可以理解为影响你投资决策的标准或指标。

例如，在购买股票时，你可能会依据某些标准进行筛选。这个标准就是因子。一个常见的因子是市净率，通常认为较低的市净率可能意味着更大的上涨空间。另一个例子是营收增长率，高增长率的公司可能被视为优质投资标的。

简单来说，因子就是那些可能对最终投资结果产生影响的各项指标。

![](img/772d16b5108683c561156463c19d38fc_1.png)

当我们获得这些因子数据后，不能直接用于建模。就像其他数据挖掘任务一样，我们需要先对数据进行预处理。本节课将详细介绍预处理的三个核心步骤。

## 预处理三步走策略

以下是因子数据预处理的三个主要步骤：

1.  **去极值**：处理数据中的异常点或离群点。
2.  **标准化**：将不同量纲和取值范围的因子数据转换到统一的尺度上。
3.  **中性化**：消除因子数据中某些系统性偏差的影响（如市值、行业的影响）。

![](img/772d16b5108683c561156463c19d38fc_3.png)

前两步在机器学习中很常见，而第三步“中性化”在因子策略分析中尤为重要。接下来，我们将按照这个顺序，首先讲解如何去极值。

![](img/772d16b5108683c561156463c19d38fc_5.png)

## 第一步：去极值方法

![](img/772d16b5108683c561156463c19d38fc_7.png)

![](img/772d16b5108683c561156463c19d38fc_9.png)

去极值的目标是处理那些远离数据主体分布的异常值。直接删除这些数据点可能造成信息损失，因此更常见的做法是“修正”它们，将其拉回到合理的边界内。

![](img/772d16b5108683c561156463c19d38fc_11.png)

以下是几种常用的去极值方法，我们将重点介绍第一种：百分位法。

![](img/772d16b5108683c561156463c19d38fc_13.png)

![](img/772d16b5108683c561156463c19d38fc_15.png)

### 方法一：百分位去极值法

这种方法利用数据的分位数来设定边界。分位数是统计学中的概念，用于描述数据分布的位置。

![](img/772d16b5108683c561156463c19d38fc_17.png)

*   **中位数 (Q2)**：将数据从小到大排列后，处于正中间位置的值。它比平均值更能抵抗极端值的影响。
*   **下四分位数 (Q1)**：所有数据中，有25%的数据小于等于这个值。
*   **上四分位数 (Q3)**：所有数据中，有75%的数据小于等于这个值。

**计算步骤：**
1.  计算数据的 **Q1** 和 **Q3**。
2.  计算四分位距：`IQR = Q3 - Q1`。
3.  设定正常值的下限和上限：
    *   下限：`lower_bound = Q1 - k * IQR`
    *   上限：`upper_bound = Q3 + k * IQR`
    （通常 `k` 取 1.5 或 3，在金融数据分析中常取 1.5）
4.  遍历数据，将所有小于下限的值设置为下限，所有大于上限的值设置为上限。

**代码示例：**
```python
import numpy as np

def winsorize_percentile(data, k=1.5):
    """
    使用百分位法（IQR）进行去极值处理。
    :param data: 输入的数据序列（如一个因子的值）
    :param k: 范围乘数，通常为1.5
    :return: 去极值处理后的数据
    """
    q1 = np.percentile(data, 25)
    q3 = np.percentile(data, 75)
    iqr = q3 - q1
    lower_bound = q1 - k * iqr
    upper_bound = q3 + k * iqr
    # 将超出边界的值拉回边界
    data_winsorized = np.clip(data, lower_bound, upper_bound)
    return data_winsorized
```

### 方法二：标准差去极值法（简述）

此方法假设数据服从正态分布。它计算数据的均值（`mean`）和标准差（`std`），然后设定边界为均值上下若干倍的标准差（如 `mean ± 3 * std`），并将边界外的值修正到边界上。这种方法对极端值比较敏感。

### 方法三：绝对中位差去极值法（简述）

这是标准差法的一种稳健变体。它使用中位数代替均值，使用绝对中位差（MAD）代替标准差来计算边界，对异常值的鲁棒性更强。
