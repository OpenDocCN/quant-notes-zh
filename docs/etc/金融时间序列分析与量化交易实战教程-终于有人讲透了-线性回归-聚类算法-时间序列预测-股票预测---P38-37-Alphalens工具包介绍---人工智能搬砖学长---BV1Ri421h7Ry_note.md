## 工具包简介与获取

![](img/18b770cdc5d0a17f0c56b5713646690f_5.png)

上一节我们介绍了量化分析的基本流程，本节中我们来看看如何利用现成工具提升效率。Alphalens是一个功能丰富的因子分析工具包，其核心价值在于自动化计算与可视化。

![](img/18b770cdc5d0a17f0c56b5713646690f_7.png)

以下是关于Alphalens的基本信息：
*   **官方资源**：其源代码和文档托管在GitHub上。
*   **核心功能**：专门用于金融因子分析，内置了计算和绘图功能。
*   **学习建议**：官方提供了详细的示例（examples）和API文档，是深入学习的最佳途径。

![](img/18b770cdc5d0a17f0c56b5713646690f_9.png)

安装该工具包非常简单，在命令行中使用以下命令即可：
```bash
pip install alphalens
```
不过，在本课程后续的实战环节中，我们将在已预装好所有必要环境的在线平台中编写和运行代码，因此本地安装步骤是可选的。

![](img/18b770cdc5d0a17f0c56b5713646690f_11.png)

## 在线研究环境配置

![](img/18b770cdc5d0a17f0c56b5713646690f_13.png)

由于在本地环境中获取和处理金融数据较为复杂，接下来我们将切换到指定的在线平台进行操作。该平台提供了专门的数据分析和研究环境。

以下是进入在线研究环境的步骤：
1.  在平台界面中，找到并点击“投资研究”功能模块。
2.  系统会启动一个类似于Jupyter Notebook的交互式编程环境。
3.  在该环境中新建一个Python 3笔记文件，我们可将其命名为“因子分析”。

![](img/18b770cdc5d0a17f0c56b5713646690f_15.png)

![](img/18b770cdc5d0a17f0c56b5713646690f_17.png)

这个在线环境允许我们在服务器端直接获取数据并执行分析代码，后续所有实战代码都将在该环境中编写和运行。相关代码会提供给大家，可以上传到自己的平台账户或跟随视频教程编写。
