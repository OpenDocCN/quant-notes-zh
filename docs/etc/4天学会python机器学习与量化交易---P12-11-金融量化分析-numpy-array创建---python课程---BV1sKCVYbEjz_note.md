## 创建特定内容的数组

![](img/710db53d5d5847706d701cbf301d8922_3.png)

![](img/710db53d5d5847706d701cbf301d8922_5.png)

以下是几种创建特定内容数组的方法。

### 创建全零数组
使用`np.zeros()`函数可以创建一个所有元素都为0的数组。默认情况下，数组的元素类型是浮点数（`float64`），这是为了在进行除法等运算时避免类型错误。

**代码示例：**
```python
import numpy as np
a = np.zeros(10) # 创建一个长度为10的全零数组，元素类型为float64
print(a.dtype) # 输出：float64

![](img/710db53d5d5847706d701cbf301d8922_7.png)

![](img/710db53d5d5847706d701cbf301d8922_9.png)

# 如果需要整数类型的全零数组，可以指定dtype参数
a_int = np.zeros(10, dtype=int)
print(a_int.dtype) # 输出：int64
```

### 创建全一数组
与`np.zeros()`类似，`np.ones()`函数用于创建所有元素都为1的数组。

**代码示例：**
```python
b = np.ones(5) # 创建一个长度为5的全一数组，元素类型为float64
b_int = np.ones(5, dtype=int) # 创建整数类型的全一数组
```

![](img/710db53d5d5847706d701cbf301d8922_11.png)

![](img/710db53d5d5847706d701cbf301d8922_13.png)

### 创建“空”数组
`np.empty()`函数用于创建一个“空”数组。这里的“空”并非指元素值为零或None，而是指分配了内存空间但未进行初始化，因此数组中的值是内存中残留的、不可预测的随机值。

![](img/710db53d5d5847706d701cbf301d8922_15.png)

**代码示例：**
```python
c = np.empty(100)
print(c[:5]) # 输出一些随机值，例如：[2.12199579e-314 6.36598737e-314 1.06099790e-313 1.48539705e-313 1.90979621e-313]
```

![](img/710db53d5d5847706d701cbf301d8922_17.png)

![](img/710db53d5d5847706d701cbf301d8922_19.png)

**为什么使用`np.empty()`？**
创建数组需要两步：1. 向计算机申请内存空间；2. 将内存空间的值初始化为指定内容（如0或1）。`np.empty()`只执行第一步，速度更快。当你确定后续操作会覆盖数组中的所有值时，可以使用它来提升性能。如果不确定，则应使用`np.zeros()`或`np.ones()`。

## 创建序列数组

除了填充固定值，我们经常需要创建具有特定序列的数组。

![](img/710db53d5d5847706d701cbf301d8922_25.png)

### 使用`np.arange()`
`np.arange()`函数类似于Python内置的`range()`，用于生成一个等差序列的数组。不同之处在于，`np.arange()`的步长（`step`）可以是小数。

![](img/710db53d5d5847706d701cbf301d8922_27.png)

**代码示例：**
```python
# 生成0到99的整数序列
seq1 = np.arange(100)
# 生成10到20（不包含20），步长为2的序列
seq2 = np.arange(10, 20, 2)
# 生成0到1，步长为0.1的序列（步长可以是小数）
seq3 = np.arange(0, 1, 0.1)
```

![](img/710db53d5d5847706d701cbf301d8922_29.png)

### 使用`np.linspace()`
`np.linspace()`函数用于在指定的区间内，生成指定数量的、等间隔的数值。它的第三个参数是**要生成的元素数量**，而不是步长。

![](img/710db53d5d5847706d701cbf301d8922_31.png)

![](img/710db53d5d5847706d701cbf301d8922_33.png)

**代码示例：**
```python
# 在0到100之间生成101个等间隔的点（包含0和100）
lin1 = np.linspace(0, 100, 101)
print(lin1.size) # 输出：101
print(lin1[0], lin1[-1]) # 输出：0.0 100.0

![](img/710db53d5d5847706d701cbf301d8922_35.png)

![](img/710db53d5d5847706d701cbf301d8922_37.png)

# 在0到10之间生成100个等间隔的点
lin2 = np.linspace(0, 10, 100)
```

![](img/710db53d5d5847706d701cbf301d8922_39.png)

**`np.linspace()`的应用场景**
它在科学计算中非常有用，例如绘制函数图像。因为数学函数是连续的，而计算机只能处理离散的点。我们可以使用`np.linspace()`在定义域内生成足够密集的点，然后计算函数值，从而近似地画出连续的函数图像。

![](img/710db53d5d5847706d701cbf301d8922_41.png)

![](img/710db53d5d5847706d701cbf301d8922_43.png)

**示例：绘制y=x²在[-10, 10]区间的图像**
```python
import matplotlib.pyplot as plt # 后续会学的绘图库
x = np.linspace(-10, 10, 10000) # 生成10000个点
y = x ** 2 # 计算每个x对应的y值
plt.plot(x, y) # 绘制图像
plt.show()
```

### 创建单位矩阵
`np.eye(N)`函数用于创建一个N×N的单位矩阵。单位矩阵是一个方阵，其主对角线上的元素为1，其余元素为0。

![](img/710db53d5d5847706d701cbf301d8922_51.png)

![](img/710db53d5d5847706d701cbf301d8922_53.png)

**代码示例：**
```python
identity_matrix = np.eye(3)
print(identity_matrix)
# 输出：
# [[1. 0. 0.]
#  [0. 1. 0.]
#  [0. 0. 1.]]
```
如果不涉及线性代数运算，这个函数使用频率较低。

本节课中我们一起学习了多种创建NumPy数组的方法：
1.  **`np.zeros()` / `np.ones()`**：创建填充固定值的数组。
2.  **`np.empty()`**：快速分配未初始化的内存空间。
3.  **`np.arange()`**：创建等差序列数组，步长可为小数。
4.  **`np.linspace()`**：在指定区间创建指定数量的等分点，常用于数值模拟和绘图。
5.  **`np.eye()`**：创建单位矩阵。
