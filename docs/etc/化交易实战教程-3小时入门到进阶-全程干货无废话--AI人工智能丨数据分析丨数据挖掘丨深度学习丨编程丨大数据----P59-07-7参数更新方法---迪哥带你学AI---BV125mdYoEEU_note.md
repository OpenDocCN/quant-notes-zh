# 机器学习入门：07-7：参数更新方法

在本节课中，我们将学习梯度下降算法的核心步骤——参数更新。我们将详细解释如何通过计算损失函数的梯度，并沿着梯度的反方向更新模型参数，从而找到使损失函数最小的最优解。

上一节我们介绍了梯度下降的基本思想，本节中我们来看看如何将这个思想转化为具体的数学公式和计算步骤。

## 参数更新的独立性

在优化过程中，我们需要找到最优的参数值，例如 θ₀ 和 θ₁。这里有一个关键问题：是同时优化所有参数，还是分别优化每个参数？

答案是分别优化。这是因为每个参数对应着数据中一个独立的特征。例如，θ₀ 对应特征 X₀，θ₁ 对应特征 X₁。在数据样本相互独立的前提下，特征 X₀ 和 X₁ 之间没有关系，因此它们对应的参数 θ₀ 和 θ₁ 在优化过程中也应独立处理。

这意味着，在三维空间中（以 θ₀ 和 θ₁ 为轴），我们会分别寻找 θ₀ 的优化方向和 θ₁ 的优化方向，让它们各自沿着最合适的方向前进，最终共同达到一个综合最优的结果。

## 梯度下降的步骤

![](img/f342ab0aa8af13c55833ed518fa4bf28_1.png)

寻找损失函数“山谷”最低点的过程可以概括为以下三步，这是一个反复迭代的过程：

1.  **求偏导**：计算损失函数 J 对每个参数 θⱼ 的偏导数，以确定下降方向。
2.  **走一小步**：沿着计算出的梯度反方向，移动一个很小的步长。步长过大可能导致“跌倒”，即不收敛或收敛效果差。
3.  **更新参数**：根据移动的方向和步长，更新所有参数的值。然后回到第一步，用新的参数值重新计算，直到满足停止条件。

## 损失函数与梯度计算

在数学上，我们如何描述这个“下山”任务呢？首先需要一个评估模型好坏的损失函数。

对于一个包含 *m* 个样本的数据集，常用的均方误差损失函数定义如下：

**公式：**
`J(θ) = (1/(2m)) * Σ (y⁽ⁱ⁾ - h_θ(x⁽ⁱ⁾))²`
其中，`h_θ(x⁽ⁱ⁾)` 是模型的预测值，`y⁽ⁱ⁾` 是真实值。

这个函数计算了所有样本预测误差的平方和，并取平均值。平方项放大了较大误差的影响，使得模型对糟糕的预测更加敏感。

接下来，我们需要计算损失函数 `J(θ)` 对某个特定参数 `θⱼ` 的偏导数，以得到该参数的更新方向。

**公式：**
`∂J/∂θⱼ = (1/m) * Σ (h_θ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾`

以下是这个梯度公式的推导要点：
*   对 `θⱼ` 求偏导时，损失函数中与 `θⱼ` 无关的项视为常数，导数为零。
*   平方项求导会产生系数 2，与公式中的 `1/2` 相乘后得到系数 1。
*   括号内 `(y - h_θ(x))` 对 `h_θ(x)` 求导会产生一个负号。
*   `xⱼ⁽ⁱ⁾` 是第 `i` 个样本在第 `j` 个特征上的值。

## 参数更新规则

计算得到梯度 `∂J/∂θⱼ` 后，我们知道了函数在 `θⱼ` 方向上的上升最快的方向。为了最小化损失函数，我们需要沿着梯度的**反方向**（即下降最快的方向）前进。

因此，参数 `θⱼ` 的更新规则如下：

**公式：**
`θⱼ := θⱼ - α * (∂J/∂θⱼ)`
即：
`θⱼ := θⱼ - α * [(1/m) * Σ (h_θ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾]`

在这个公式中：
*   `:=` 表示赋值操作。
*   `α` 称为**学习率**，它控制着我们每次更新参数时所走的“步长”大小。
*   `- α * (∂J/∂θⱼ)` 整体代表了沿着梯度反方向移动的一步。

通过不断重复“计算梯度 -> 更新参数”的过程，参数 `θ` 会逐步调整，使得损失函数 `J(θ)` 的值不断减小，最终逼近最小值。

本节课中我们一起学习了梯度下降算法的参数更新机制。我们明确了参数需要分别独立优化，掌握了从定义损失函数、计算梯度到最终更新参数的三步流程，并推导出了核心的参数更新公式 `θⱼ := θⱼ - α * (∂J/∂θⱼ)`。理解这个更新规则是掌握梯度下降乃至众多优化算法的基础。
