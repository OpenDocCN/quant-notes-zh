## MAD法原理介绍
上一节我们介绍了百分位法去极值，本节中我们来看看另一种方法：MAD法。MAD是Median Absolute Deviation的缩写，即中位数绝对偏差。虽然公式看起来稍显复杂，但计算过程其实很简单。

![](img/ca08c67412832f5bd04456b1ca52e68c_6.png)

MAD法的核心思想是，通过数据的中位数和其绝对偏差的中位数来定义一个合理的数值范围，并剔除超出此范围的极端值。

![](img/ca08c67412832f5bd04456b1ca52e68c_8.png)

![](img/ca08c67412832f5bd04456b1ca52e68c_10.png)

以下是MAD法的计算步骤：
1.  计算原始数据序列的中位数。
2.  计算序列中每个数据点与中位数之差的绝对值，形成一个新的序列。
3.  计算这个新序列的中位数，这个值就是MAD。
4.  设定一个系数N（通常取1.4826），计算上界和下界：
    *   上界 = 中位数 + N * MAD
    *   下界 = 中位数 - N * MAD
5.  将原始数据中超出 [下界, 上界] 范围的值替换为边界值。

```python
def filter_extreme_MAD(series, n=1.4826):
    """
    使用MAD法对序列进行去极值处理。
    :param series: 待处理的pandas Series数据
    :param n: MAD的倍数，默认为1.4826
    :return: 处理后的序列
    """
    # 1. 计算原始序列的中位数
    median = series.quantile(0.5)

    # 2. 计算每个数据点与中位数差的绝对值
    abs_deviation = (series - median).abs()

    # 3. 计算绝对偏差序列的中位数，即MAD值
    mad = abs_deviation.quantile(0.5)

    # 4. 计算上界和下界
    high = median + n * mad
    low = median - n * mad

    # 5. 将超出范围的值替换为边界值
    filtered_series = series.clip(lower=low, upper=high)

    return filtered_series
```
