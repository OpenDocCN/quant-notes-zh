## 重复数据的清洗

![](img/36f32f68df195e69b538401970a52702_15.png)

![](img/36f32f68df195e69b538401970a52702_17.png)

重复数据通常指数据集中完全相同的行。这些重复项可能由数据采集或合并过程中的错误导致，如果不处理，会影响分析的准确性。

以下是创建并处理重复数据的步骤：

![](img/36f32f68df195e69b538401970a52702_19.png)

![](img/36f32f68df195e69b538401970a52702_21.png)

1.  首先，我们创建一个包含重复行的示例数据集。
    ```python
    import pandas as pd
    import numpy as np

    # 创建一个8行4列的DataFrame
    df = pd.DataFrame(np.random.randint(0, 100, size=(8, 4)))
    print("原始数据：")
    print(df)

    # 人为制造重复行：将第3、4、5行的数据设置为与第2行相同
    df.iloc[3] = df.iloc[2]
    df.iloc[4] = df.iloc[2]
    print("\n制造重复行后的数据：")
    print(df)
    ```

![](img/36f32f68df195e69b538401970a52702_23.png)

2.  使用 `drop_duplicates()` 方法清洗重复数据。该方法的关键参数是 `keep`，用于指定保留哪一行。
    *   `keep=‘first’`：保留第一次出现的重复行，删除后续的重复行。这是默认设置。
    *   `keep=‘last’`：保留最后一次出现的重复行。
    *   `keep=False`：删除所有重复的行。

    ```python
    # 默认保留首次出现的行
    df_cleaned = df.drop_duplicates()
    print("\n清洗重复数据后（keep='first'）：")
    print(df_cleaned)

    # 保留最后一次出现的行
    df_cleaned_last = df.drop_duplicates(keep='last')
    print("\n清洗重复数据后（keep='last'）：")
    print(df_cleaned_last)
    ```

![](img/36f32f68df195e69b538401970a52702_25.png)

通过以上操作，我们可以轻松地移除数据集中的冗余信息。

## 异常数据的清洗

![](img/36f32f68df195e69b538401970a52702_27.png)

异常值是指与数据集中的其他观测值显著不同的数据点。例如，在蔬菜大棚的温度记录中，出现120℃的读数显然是不合理的。处理异常值通常需要先定义判定规则。

![](img/36f32f68df195e69b538401970a52702_29.png)

接下来，我们学习如何根据特定条件（例如，数值超过两倍标准差）来识别并删除异常值。

![](img/36f32f68df195e69b538401970a52702_31.png)

![](img/36f32f68df195e69b538401970a52702_33.png)

以下是清洗异常值的完整流程：

1.  生成一个模拟数据集。
    ```python
    # 创建一个1000行3列的数据集，数值范围在0到1之间
    df = pd.DataFrame(np.random.random(size=(1000, 3)), columns=['A', 'B', 'C'])
    print("原始数据形状：", df.shape)
    ```

![](img/36f32f68df195e69b538401970a52702_35.png)

2.  定义异常值判定条件。本例中，我们判定 **C列中数值大于该列两倍标准差** 的数据为异常值。
    ```python
    # 计算C列的两倍标准差
    twice_std = df['C'].std() * 2
    print("C列的两倍标准差为：", twice_std)

    # 创建布尔掩码，标记出异常值（True表示异常）
    condition = df['C'] > twice_std
    print("\n异常值标记（前10个）：")
    print(condition.head(10))
    ```

![](img/36f32f68df195e69b538401970a52702_37.png)

![](img/36f32f68df195e69b538401970a52702_39.png)

3.  利用布尔索引过滤掉异常值所在的行。我们需要保留正常值（即条件为False的行），因此对条件取反。
    ```python
    # 对条件取反，然后用于索引DataFrame，保留正常数据
    df_cleaned = df[~(df['C'] > twice_std)] # 等价于 df[~condition]
    print("\n清洗异常值后的数据形状：", df_cleaned.shape)
    print("清洗后C列最大值：", df_cleaned['C'].max())
    ```

![](img/36f32f68df195e69b538401970a52702_41.png)

![](img/36f32f68df195e69b538401970a52702_43.png)

执行后，新的数据框 `df_cleaned` 中将不再包含C列大于两倍标准差的异常数据行。
