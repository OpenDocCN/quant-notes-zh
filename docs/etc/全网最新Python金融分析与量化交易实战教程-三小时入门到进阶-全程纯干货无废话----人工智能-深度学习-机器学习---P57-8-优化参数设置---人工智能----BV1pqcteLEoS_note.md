# Python金融分析与量化交易实战教程：P57：8-优化参数设置 📈

在本节课中，我们将学习梯度下降算法中关键的参数设置，包括批量大小和学习率。理解这些概念对于高效训练机器学习模型至关重要。

## 批量梯度下降的问题

上一节我们介绍了梯度下降的基本思想，本节中我们来看看其具体实现中的关键参数。首先，考虑批量梯度下降的公式：

**公式：** `θ_new = θ_old - α * (1/M) * Σ(∇J(θ; x_i, y_i))`

其中，`M` 代表样本总数。当 `M` 非常大时，例如一百万，每次更新参数都需要计算一百万次梯度。这导致每次参数更新的速度非常慢。

![](img/26f76a37338b63c435526a15799183f6_1.png)

批量梯度下降的优点是，它使用所有样本的平均梯度方向，通常能稳定地朝向最优解收敛。但其显著缺点是，当样本数量巨大时，计算成本过高，训练速度极慢。

## 随机梯度下降的引入

既然使用所有样本太慢，一个自然的想法是每次只使用一个样本来更新参数，这就是随机梯度下降。

![](img/26f76a37338b63c435526a15799183f6_3.png)

![](img/26f76a37338b63c435526a15799183f6_5.png)

**公式：** `θ_new = θ_old - α * ∇J(θ; x_i, y_i)`

此时，每次迭代的速度相比批量梯度下降提升了约 `M` 倍。然而，随机梯度下降带来了新的问题：由于每次只依赖一个样本，更新方向受单个样本的噪声或离群值影响很大，导致优化路径不稳定，收敛过程波动剧烈。

以下是两种方法收敛路径的对比示意图：
*   **批量梯度下降**：路径平滑，稳定地朝向最低点。
*   **随机梯度下降**：路径曲折，围绕最低点大幅波动。

## 小批量梯度下降：一种折中方案

为了综合批量梯度下降的稳定性和随机梯度下降的速度，我们引入了小批量梯度下降。

**公式：** `θ_new = θ_old - α * (1/B) * Σ(∇J(θ; x_i, y_i))` （其中 `i` 从 1 到 `B`）

这里，`B` 代表批量大小，即每次迭代用于计算梯度的样本数量。我们既不用全部样本，也不用单个样本，而是使用一小批样本。

以下是关于批量大小 `B` 选择的要点：
*   **常见取值**：程序员通常选择 2 的幂次方，如 64、128、256。
*   **`B` 越大**：梯度估计越精确，收敛更稳定，但每次迭代速度慢，占用更多内存/显存。
*   **`B` 越小**：每次迭代速度快，但梯度估计噪声大，收敛路径波动大。
*   **选择原则**：在硬件（内存/显存）和可接受训练时间的限制下，通常选择较大的批量大小。64、128、256 是实践中常见的起点。

## 学习率的作用

在参数更新公式中，另一个关键参数是学习率 `α`。

![](img/26f76a37338b63c435526a15799183f6_7.png)

**公式：** `θ_new = θ_old - α * Gradient`

学习率 `α` 控制着每次参数更新沿着梯度方向前进的步长。

以下是关于学习率的要点：
*   **作用**：它决定了每次更新的幅度。乘以计算出的梯度方向，得到实际的更新量。
*   **常见取值**：通常设置为一个较小的正数，例如 0.01 或 0.001。
*   **设置原则**：
    *   学习率过大可能导致更新步伐太大，在最优解附近震荡甚至发散。
    *   学习率过小会导致收敛速度非常缓慢。
    *   通常从一个较小的值（如 0.01）开始，如果模型收敛不佳或训练过慢，再尝试调小它。
*   **调参**：学习率是模型训练中最重要的超参数之一，需要根据实际情况进行调整。

![](img/26f76a37338b63c435526a15799183f6_9.png)

## 梯度计算的核心

![](img/26f76a37338b63c435526a15799183f6_11.png)

在以上所有公式中，核心操作都是计算损失函数 `J(θ)` 对参数 `θ` 的梯度 `∇J(θ)`。这通过反向传播算法实现，其本质是链式法则。在计算时，我们将模型预测（涉及 `θ` 和 `x`）与真实标签 `y` 代入损失函数，然后对 `θ` 求导。`x` 和 `y` 在求导过程中被视为已知常数。

![](img/26f76a37338b63c435526a15799183f6_13.png)

本节课中我们一起学习了梯度下降算法的关键参数设置。我们首先分析了批量梯度下降计算慢和随机梯度下降不稳定的缺点，进而引出了折中的小批量梯度下降方法，并详细讨论了批量大小 `B` 的选择策略。接着，我们解释了学习率 `α` 的作用和设置技巧。

需要明确的是，在绝大多数现代机器学习问题中，我们不再使用解析法直接求解，而是依赖梯度下降这类优化算法进行迭代求解。梯度下降是最基础、最经典的优化方法。后续课程中，我们还将接触到在其基础上改进的、更高级的优化器。
