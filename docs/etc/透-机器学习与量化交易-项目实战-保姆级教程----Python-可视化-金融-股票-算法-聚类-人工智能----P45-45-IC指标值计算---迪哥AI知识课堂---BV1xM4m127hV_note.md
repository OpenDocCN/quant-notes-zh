# 机器学习与量化交易：P45：IC指标值计算 📊

在本节课中，我们将学习如何计算IC（信息系数）指标值。IC值是衡量我们构建的因子（指标）与股票未来实际收益率之间相关性强弱的关键指标。我们将通过获取股票价格数据、进行数据格式转换，最终计算出斯皮尔曼秩相关系数来完成这一任务。

## 获取收盘价数据

上一节我们介绍了如何构建因子，本节中我们来看看如何获取计算IC值所需的实际收益率数据。计算收益率需要股票的每日收盘价。

我们使用 `get_price` 函数来获取指定股票池在特定时间段内的价格数据。

```python
# 定义股票池和日期范围
stock_pool = ['你的股票代码列表']  # 请替换为实际的股票代码列表
start_date = '2019-01-01'
end_date = '2020-01-01'

# 获取价格数据
price_data = get_price(security=stock_pool,
                       start_date=start_date,
                       end_date=end_date,
                       fields=['close'])
```

执行上述代码后，`price_data` 是一个包含多只股票、多个交易日收盘价的数据结构。为了后续处理方便，我们通常将其转换为二维的DataFrame格式，并设置清晰的索引和列名。

![](img/e80915444c9fa8abbad91d5f081ece97_1.png)

```python
# 转换数据格式并设置索引、列名
price_df = price_data['close']  # 提取收盘价
price_df.index.name = 'date'    # 设置索引名为‘date’
price_df.columns.name = 'code'  # 设置列名为‘code’
```

完成这一步后，我们就得到了一个规整的收盘价DataFrame，索引是日期，列是股票代码，值为对应的收盘价。

![](img/e80915444c9fa8abbad91d5f081ece97_3.png)

## 数据格式转换

有了因子数据和价格数据后，我们需要将它们转换为特定库（如Alphalens）要求的统一格式，以便进行后续的因子分析。

以下是进行数据格式转换的关键步骤。我们需要使用一个工具函数，它接收处理好的因子数据和价格数据作为输入。

```python
# 假设已导入必要的工具库（如alphalens.utils）
# 将因子数据和价格数据转换为Alphalens可用的格式
factor_data = convert_to_forward_returns_format(factor_data=your_processed_factor_df,
                                                prices=price_df)
```

这个转换函数会生成一个新的数据结构。在这个结构中，每一行对应一个股票在特定日期的数据，并包含以下关键列：
*   **factor**: 你计算的因子值。
*   **factor_quantile**: 因子值被自动分成的组别（例如分为5组）。组号越小，代表因子值在同一天所有股票中排名越低（数值越小）；组号越大，代表因子值排名越高。
*   **1D**、**5D**、**10D**: 分别代表未来1天、5天、10天的股票收益率。我们通常使用 **1D**（次日收益率）来计算IC值。其计算公式为：
    **收益率 (1D) = (明日收盘价 - 今日收盘价) / 今日收盘价**

转换完成后，我们就得到了包含因子值、因子分组以及未来收益率的标准数据集。

## 计算IC值

现在，我们拥有了格式统一的数据，可以开始计算IC值了。IC值通常使用斯皮尔曼秩相关系数来计算，它衡量的是因子排名与未来收益率排名之间的单调关系。

以下是计算IC值的具体操作。我们调用性能分析模块中的相关函数。

执行上述代码后，`ic_series` 是一个时间序列，索引为日期，值为该交易日所有股票的因子值与下一期收益率之间的秩相关系数（IC值）。我们可以查看其前几行以了解计算结果。

```python

print(ic_series.head())
```

输出的IC值序列，其**IC**列就是我们计算出的每日IC指标值。这个值介于-1到1之间。一般来说，IC值大于0表示因子与未来收益正相关（因子值越大，预期收益越高），IC值越稳定且显著大于0，说明因子的预测能力越强。
