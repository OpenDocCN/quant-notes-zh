# 金融量化分析：P4：影响股价因素与股票买卖知识 📈

在本节课中，我们将学习影响股票价格的主要因素，并了解股票买卖的基本流程与规则。理解这些基础知识是进行量化分析的前提。

## 影响股价的六大因素

上一节我们介绍了股票的基本概念，本节中我们来看看哪些因素会影响股票的价格。以下是六个核心因素。

![](img/aa7e89cde239616280e743d90ff822a6_1.png)

1.  **公司自身因素**：这是影响股价最根本的因素。公司的经营状况、盈利能力、发展前景等基本面直接决定了其长期价值。公式上，股价可以视为公司未来现金流的折现值。
2.  **市场因素**：这是影响股价最直接的因素。短期内，股价由买卖双方的供需关系决定。买盘多于卖盘则价格上涨，反之则下跌。
3.  **行业因素**：公司所属行业的整体景气度和发展趋势会影响行业内所有公司的股价。例如，新兴行业（如人工智能）可能带动相关股票上涨。
4.  **心理因素**：投资者的情绪和非理性行为，如从众心理、恐慌性抛售或过度乐观，都会在短期内剧烈影响股价。
5.  **经济因素**：国家宏观经济状况，如利率、通货膨胀率、货币政策等，会影响整个市场的资金流动性和投资意愿。
6.  **政治因素**：国家政策、国际关系、地缘政治事件等都可能引发市场波动，影响投资者信心。

## 股票买卖的基本流程与规则

了解了影响股价的因素后，我们来看看实际操作中买卖股票的步骤和必须遵守的市场规则。以下是关键流程与概念。

*   **开户与委托**：个人投资者不能直接在交易所买卖股票，必须通过证券公司（券商）开户。买卖指令需要通过券商的系统提交到交易所，这个过程称为“委托”。
*   **交易日**：股票交易并非每天24小时进行。交易日通常为非法定节假日的周一至周五。
*   **交易时间**：在交易日中，有特定的交易时段。以上海/深圳证券交易所为例：
    *   **开盘集合竞价** (9:15 - 9:25)：此期间提交的买卖委托不会立即成交，而是在9:25一次性集中撮合，以产生当日的**开盘价**。其核心原则是最大化成交量。
    *   **连续竞价** (9:30 - 11:30, 13:00 - 14:57)：此期间交易系统对接收到的委托进行连续、高频的撮合，投资者可实时成交。
    *   **收盘集合竞价** (仅深交所：14:57 - 15:00)：与开盘类似，此期间委托集中至15:00撮合，产生**收盘价**。上交所的收盘价为当日最后一笔交易的成交价。
*   **T+1交易制度**：当日（T日）买入的股票，需等到下一个交易日（T+1日）才能卖出。资金卖出后，当日可用但不可取，需T+1日方可提取。
*   **涨跌停板限制**：为抑制过度投机，A股市场设有涨跌幅限制。普通股票每日价格波动不得超过前一交易日收盘价的±10%（ST股为±5%）。价格达到限幅时即停止交易。
