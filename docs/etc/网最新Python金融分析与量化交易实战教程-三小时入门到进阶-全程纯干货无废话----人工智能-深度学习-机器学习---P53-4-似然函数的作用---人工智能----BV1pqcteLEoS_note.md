在本节课中，我们将深入探讨线性回归中的一个核心概念——似然函数。我们将学习如何通过似然函数来估计模型参数，并理解其背后的数学原理，特别是如何将复杂的乘法问题转化为更易处理的加法问题。

## 误差项与高斯分布

上一节我们介绍了线性回归模型的基本形式。现在，我们面临的核心问题是：如何求解模型参数 `θ`？我们知道数据 `X` 和标签 `Y`，当前缺少的是参数 `θ` 的具体值。

我们之前已经解释过，模型的误差项 `ε` 服从均值为0的高斯分布（正态分布）。我们可以先列出高斯分布的表达式：

**公式：**
```
P(ε) = (1 / √(2πσ²)) * exp(-ε² / (2σ²))
```

在这个公式中，`exp` 表示以自然常数 `e`（约等于2.718）为底的指数函数。由于我们假设误差的均值 `μ` 为0，因此公式中省略了 `(ε - μ)` 项，直接写作 `ε²`。

这个分布是关于误差项 `ε` 的。然而，我们最终关心的并不是具体的误差值，而是参数 `θ`。因此，我们需要将这个与 `ε` 相关的表达式，转换成与 `θ` 相关的形式。

![](img/8c2f8abcb99b400e9699300b35f252b0_2.png)

根据线性回归模型 `Y = θX + ε`，我们可以进行转换：`ε = Y - θX`。将这个关系代入高斯分布公式，我们就得到了一个关于 `θ` 和已知数据 `X`、`Y` 的表达式。

## 似然函数的直观理解

现在，我们来通俗地解释一下这个转换后的式子意味着什么。我们的目标是找到参数 `θ`，使得 `θ` 与 `X` 组合后得到的预测值，与真实值 `Y` 越接近越好。

换句话说，我们希望找到一组参数 `θ`，使得这组参数与我们的数据 `X` 组合后，“恰好”得到真实观测值 `Y` 的可能性（概率）最大。这就是“似然”的含义——在给定参数下，观测到当前数据的可能性。

为了加深理解，可以举一个例子：假设我去澳门赌场，想试试手气但不敢先玩，于是观察前面五位玩家。如果五位玩家都赢了，那么我会认为赌场有一组固定的“参数”控制输赢，并且这组参数使得“赢”这个结果出现的可能性很高。因此，我推断自己接下来去玩，赢的可能性也很大。这里，“五位玩家都赢”是我观测到的数据，我根据这个数据去估计那组控制输赢的“参数”很可能是有利的。似然函数做的就是类似的事情：寻找最可能产生我们已观测到数据的参数。

## 似然函数的数学形式

在数学上，对于所有独立同分布的样本点，其联合概率等于各自概率的乘积。我们的数据恰好满足独立同分布假设。因此，整体的似然函数 `L(θ)` 是所有单个样本点似然的乘积：

**公式：**
```
L(θ) = ∏ (i=1 to m) P(Y_i | X_i; θ)
```

以下是关于这个公式的两点关键解释：
1.  **为什么是累乘（∏）？** 因为我们希望利用大量数据来共同确定最合适的参数。数据点越多，估计的结果理论上越可靠。在“独立同分布”的前提下，所有数据同时出现的联合概率就是每个数据点出现概率的乘积。
2.  **为什么是乘法？** 正如上一点所述，这是由概率论中“独立事件的联合概率”定义所决定的。

## 从乘法到加法：对数似然

虽然从原理上我们得到了似然函数，但直接求解这个连乘式非常困难。想象一下，如果有1000个数据点，就要计算1000个项的乘积，这在计算上是极其不便的。

一个常见的技巧是将乘法问题转化为加法问题，因为加法在数学上更易于处理（例如求导、优化）。转换的工具就是对数函数。根据对数的性质，乘积的对数等于对数的和：

**公式：**
```
log(A * B) = log(A) + log(B)
```

因此，我们对似然函数 `L(θ)` 取自然对数，得到对数似然函数 `log L(θ)`：

**公式：**
```
log L(θ) = ∑ (i=1 to m) log( P(Y_i | X_i; θ) )
```

现在，一个关键的疑问是：对原函数取对数，改变了函数本身，那么求出的解还会是原来的解吗？答案是：我们关心的“极值点”不会改变。我们最终的目标是找到使似然函数 `L(θ)` 取得最大值的那个参数 `θ` 的值（即极大似然估计）。给函数加上一个单调递增的函数（如对数函数），虽然函数值本身变了，但函数取最大值的位置（即极值点）并没有改变。因为我们只关心“在哪里取得最大”，而不关心“最大值具体是多少”，所以这个转化是等价且有效的。

通过取对数，我们将复杂的连乘问题转化为了连加问题，这为后续通过求导等数学工具来求解参数 `θ` 铺平了道路。
