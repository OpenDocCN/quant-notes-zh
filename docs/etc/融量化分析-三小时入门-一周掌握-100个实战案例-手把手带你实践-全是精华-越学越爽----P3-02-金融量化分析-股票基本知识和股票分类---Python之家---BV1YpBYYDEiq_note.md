# 金融量化分析：P3：02 股票基本知识与分类 📈

在本节课中，我们将要学习股票的核心概念、作用以及常见的分类方式。理解这些基础知识是进行金融量化分析的第一步。

## 股票的定义与作用

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

为了更形象地解释，我们可以设想一个场景：一位创业者需要资金，而投资者（如亿万富翁）愿意提供资金。投资者将钱给创业者，创业者则给予投资者公司的股票作为凭证。这表示投资者对公司出资了。

假设公司初始由五个人各出资1亿元，公司总市值即为5亿元。那么每位出资人将获得公司20%的股票。这代表了他们的股东身份和出资证明。

### 股票的核心作用

以下是股票的两个主要作用：

1.  **证明股东身份与权利**：持有股票意味着你是公司的所有者（股东）之一。作为股东，你拥有参与公司重大决策（如在股东大会上投票）的权利。
2.  **获取收益**：股东主要通过两种方式获利：
    *   **公司分红**：当公司盈利时，可能会将部分利润按持股比例分配给股东。公式可以表示为：`个人分红 = 公司净利润 × 个人持股比例`。
    *   **交易获利**：股东可以在市场上将股票转让给他人。如果卖出价格高于买入价格，即可赚取差价，实现获利或“套现”。

对于资金量较小的普通股民而言，这两种获利方式同样适用，只是涉及的金额和公司决策参与度不同。

## 公司上市与IPO

上一节我们介绍了股票的基本作用，本节中我们来看看公司如何向公众发行股票，即“上市”。

所谓上市，就是企业通过证券交易所，首次公开向社会增发股票以募集资金的过程。

公司不能随意向公众募集资金。它需要达到一定体量，并向证监会提交申请，披露财务报表等各种信息。经审核确认公司运营良好、风险可控后，才允许其上市。

上市后，公司的股票就能在证券交易所挂牌交易，所有股民都可以公开买卖这只股票。这相当于公司获得了向广大社会公众募集资金的资格。

**IPO** 即首次公开募股，指的就是公司第一次上市并公开发行股票的行为。

## 股票的分类

![](img/34233ad4346fc4993019f2298646bccf_1.png)

了解了上市的概念后，我们来看看股票有哪些常见的分类方式。

### 按公司业绩分类

按业绩分类，主要有以下三种常见类型：

*   **蓝筹股**：指资本雄厚、信誉优良的大公司股票。例如中石油、中石化。它们通常规模大、经营稳定。
*   **绩优股**：指业绩优良公司的股票。它们可能规模不如蓝筹股庞大，但盈利能力持续且强劲。例如贵州茅台。
*   **ST股**：中文称为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记，以提醒投资者该公司存在较高风险。

### 按上市地区分类

按股票上市和交易的地域不同，主要分为：

*   **A股**：在中国大陆（上海、深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。
*   **H股**：在中国香港上市的公司股票。
*   **N股/S股**：分别指在美国纽约、新加坡上市的公司股票。

### 不同股市的交易规则

不同的股市有不同的交易规则，其中A股有两条重要限制：

1.  **涨跌幅限制**：A股股票每日价格涨跌幅度不得超过前一日收盘价的10%（新股上市初期等特殊情况除外）。这旨在防止股价短期剧烈波动，保护投资者。公式表示：`今日价格上限 = 昨日收盘价 × 1.10`， `今日价格下限 = 昨日收盘价 × 0.90`。
2.  **T+1交割制度**：“T”代表交易当天。在A股，当天买入的股票，当天不能卖出，必须等到下一个交易日（T+1）才能卖出。这旨在减少市场过度投机。相比之下，港股、美股等多实行T+0制度，即当天买入可当天卖出。

B股规则类似但更复杂，涉及资金交收（T+3）。而H股、N股等市场通常没有涨跌幅限制，且多为T+0交易。

本节课中我们一起学习了股票的基础知识。我们首先明确了股票是股东身份的凭证，其作用在于确权与获利（分红与交易）。接着，我们了解了公司通过IPO上市向公众募集资金的过程。最后，我们掌握了股票按业绩（蓝筹股、绩优股、ST股）和按上市地（A股、B股、H股等）的分类方法，并认识了A股独特的涨跌幅限制和T+1交易规则。这些概念是构建金融量化分析知识体系的基石。
