## 均线概念与计算

数据准备就绪后，我们来看第二个核心需求：计算股票的5日均线和30日均线。首先，我们需要理解什么是均线。

![](img/06c688a8eebc22dc77138eabaab55684_7.png)

在股票分析中，对于每一个交易日，都可以计算出前N个交易日的移动平均值。将这些移动平均值连接起来形成的曲线，就称为N日移动平均线，简称均线。常用的均线有5日、10日、30日、60日等。其中，5日和10日均线通常被视为短期均线。

均线的计算公式如下：
**MA = (C1 + C2 + C3 + ... + CN) / N**
其中，`MA` 代表移动平均值，`C1` 到 `CN` 代表连续N个交易日的收盘价。

为了更直观地理解，我们举例说明5日均线的计算过程。假设有连续8天的收盘价，计算5日均线意味着：
*   首先计算第1天到第5天收盘价的平均值，得到第一个点 `X1`。
*   接着计算第2天到第6天收盘价的平均值，得到第二个点 `X2`。
*   然后计算第3天到第7天收盘价的平均值，得到第三个点 `X3`。
*   最后计算第4天到第8天收盘价的平均值，得到第四个点 `X4`。
将 `X1`, `X2`, `X3`, `X4` 这四个点连接起来，就构成了5日均线。30日、60日等均线的计算原理与此相同。

## 使用Pandas计算均线

![](img/06c688a8eebc22dc77138eabaab55684_9.png)

理解了均线的概念后，我们使用Pandas来计算具体的均线值。计算的核心是 `rolling` 函数，它可以方便地实现滑动窗口计算。

以下是计算5日和30日均线的代码：

```python
# 计算5日均线值
ma5 = df['close'].rolling(5).mean()
# 计算30日均线值
ma30 = df['close'].rolling(30).mean()
```
*   `df['close'].rolling(5)` 会创建一个窗口大小为5的滚动窗口对象。
*   `.mean()` 会对每个窗口内的数据（即连续的5个收盘价）计算平均值。
*   对于 `ma5`，前4个值会是 `NaN`（非数字），因为至少需要5个数据才能计算第一个5日均值。`ma30` 的前29个值同理。

## 均线可视化

计算出均线数据后，我们可以将其可视化，以便更直观地观察走势。这里我们使用 `matplotlib` 库进行简单的绘图。

```python
import matplotlib.pyplot as plt
# 启用Jupyter Notebook的嵌入式绘图
%matplotlib inline

# 绘制5日均线
plt.plot(ma5[50:80], label='MA5')
# 绘制30日均线
plt.plot(ma30[50:80], label='MA30')
plt.legend() # 显示图例
plt.show()
```
通过图表，我们可以清晰地看到两条均线的走势以及它们之间的交叉情况，这是双均线策略判断买卖信号的重要依据。
