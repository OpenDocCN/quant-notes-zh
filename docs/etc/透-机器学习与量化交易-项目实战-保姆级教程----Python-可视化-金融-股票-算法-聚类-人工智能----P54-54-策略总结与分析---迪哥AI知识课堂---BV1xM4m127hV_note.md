接下来，我们选择一个更长的时间段进行回测。回测很少只做几个月，因为难以看出效果，通常选择3到5年是比较常见的。

回测很快得出了结果。在2016年1月4日到2018年1月4日这个时间段，我们的策略结果与基准收益相差不大，没有太明显的优势。

以下是策略回测结果中包含的一些关键信息和分析点：

![](img/69478a4c642f559015f828992d6b5d1c_68.png)

![](img/69478a4c642f559015f828992d6b5d1c_70.png)

*   **交易详情**：记录了每次调仓和交易的具体信息。
*   **调仓频率**：策略是每月进行调仓，从交易详情中可以看到4月、5月、6月直至11月的调仓记录。
*   **持仓详情**：展示了每个时间点的持仓情况。
*   **账户信息**：例如，初始资金为100万，最终变成了77万，反映了策略在该时间段内的盈亏。

![](img/69478a4c642f559015f828992d6b5d1c_72.png)

我们将代码改回默认设置，即选择排名前十的股票。这个效果相对好一些。大家拿到代码后，可以默认使用这个设置进行尝试。

![](img/69478a4c642f559015f828992d6b5d1c_74.png)

我们的策略是基于报告，总结出哪些指标越高越好，哪些指标越低越好，从而构建了一个简单的打分回测策略。

![](img/69478a4c642f559015f828992d6b5d1c_76.png)

![](img/69478a4c642f559015f828992d6b5d1c_78.png)

打分法在量化交易中应用比较广泛，也比较直接，通常能获得一个相对不错的结果。当然，策略也会受到熊市等市场环境的影响，导致在某些时间段收益一般或回测效果不佳，这都是由市场决定的常见现象。
