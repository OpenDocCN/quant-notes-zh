![](img/7182af84796862231a53cd636587fd36_3.png)

## 什么是最大回撤？

上一节我们介绍了策略分析中的一些基本概念，本节中我们来看看如何评估策略的风险。最大回撤描述的是在投资过程中，资产净值从任一历史高点下跌到后续低点的最大幅度。它衡量的是投资者可能经历的最糟糕情况。

具体来说，它描述的是“惨劲儿”，即在投资过程中，资产价值下跌最严重、持续时间最长的那段区间。即使一个策略最终是盈利的，但中间可能经历一段非常痛苦的下跌过程，最大回撤就是量化这段“痛苦”程度的指标。

## 最大回撤的公式与含义

为了更精确地理解，我们可以用公式来描述最大回撤。假设 `Pi` 代表第 `i` 天的资产净值，`Pj` 代表第 `i` 天之后某一天（第 `j` 天）的资产净值。

最大回撤的计算公式可以表示为：

**最大回撤 = max( (Pi - Pj) / Pi )**，其中 `i < j`

这个公式的含义是：遍历所有可能的 `i` 和 `j`（`i` 在前，`j` 在后），计算从高点 `Pi` 到低点 `Pj` 的跌幅比例，然后取其中的最大值。这个最大值就是整个投资期间的最大回撤率。

## 为什么最大回撤很重要？

![](img/7182af84796862231a53cd636587fd36_5.png)

在评估一个投资策略时，我们不仅要看它最终能赚多少钱（收益率），更要看它可能带来多大的风险。最大回撤就是一个核心的风险指标。

![](img/7182af84796862231a53cd636587fd36_7.png)

它之所以重要，是因为：
*   **衡量极端风险**：它告诉你，在最坏的情况下，你的资产可能会缩水多少。
*   **影响投资心态**：巨大的回撤会考验投资者的心理承受能力，可能导致在低点恐慌性卖出。
*   **评估策略稳健性**：一个回撤控制得好的策略，通常更稳健，也更容易被长期执行。

![](img/7182af84796862231a53cd636587fd36_9.png)

例如，2008年全球金融危机时，股市从高点大幅下跌，许多投资者承受了巨大的账面亏损，这正是最大回撤在实际中的体现。

![](img/7182af84796862231a53cd636587fd36_11.png)

## 如何用Python计算最大回撤？

![](img/7182af84796862231a53cd636587fd36_13.png)

理解了概念后，我们来看看如何用代码实现最大回撤的计算。我们将使用Pandas库来高效地处理时间序列数据。

![](img/7182af84796862231a53cd636587fd36_15.png)

以下是计算最大回撤的核心步骤：

![](img/7182af84796862231a53cd636587fd36_17.png)

![](img/7182af84796862231a53cd636587fd36_19.png)

1.  计算历史最高点的累积序列。
2.  计算每日的“回撤”，即当日历史最高点与当日净值的差值。
3.  计算每日的回撤率。
4.  找出所有回撤率中的最大值。

![](img/7182af84796862231a53cd636587fd36_21.png)

让我们通过代码来演示这个过程。首先，我们假设有一个包含资产净值的数据序列 `data[‘net_value’]`。

![](img/7182af84796862231a53cd636587fd36_23.png)

![](img/7182af84796862231a53cd636587fd36_25.png)

```python
import pandas as pd

![](img/7182af84796862231a53cd636587fd36_27.png)

# 假设 df 是一个DataFrame，其中 ‘net_value’ 列是每日资产净值
# 计算到当前日期为止的历史最高净值序列
historical_max = df[‘net_value’].cummax()

![](img/7182af84796862231a53cd636587fd36_29.png)

![](img/7182af84796862231a53cd636587fd36_31.png)

# 计算每日的回撤（历史最高 - 当日净值）
drawdown = historical_max - df[‘net_value’]

# 计算每日的回撤率（回撤 / 历史最高）
drawdown_ratio = drawdown / historical_max

![](img/7182af84796862231a53cd636587fd36_33.png)

![](img/7182af84796862231a53cd636587fd36_35.png)

# 最大回撤率就是所有回撤率中的最大值
max_drawdown_ratio = drawdown_ratio.max()
print(f“最大回撤率为：{max_drawdown_ratio:.2%}”)
```

![](img/7182af84796862231a53cd636587fd36_37.png)

**代码解释**：
*   `.cummax()` 方法会生成一个序列，其中每个值都是原始序列到当前位置为止的最大值。
*   `historical_max - df[‘net_value’]` 得到的是每日相对于历史最高点的下跌绝对值。
*   将这个差值除以历史最高点，就得到了下跌的百分比，即回撤率。
*   最后，`.max()` 方法找出整个回撤率序列中的最大值，这就是我们要找的最大回撤。

![](img/7182af84796862231a53cd636587fd36_39.png)
