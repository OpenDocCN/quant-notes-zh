## 回测收益率 📊

![](img/6c53dbba73c949b784e85586065d8a1a_8.png)

回测收益率是评估策略表现最直接的指标。它表示在整个回测周期结束后，你的初始投资最终变成了多少，即总收益与初始本金的比值。

![](img/6c53dbba73c949b784e85586065d8a1a_10.png)

![](img/6c53dbba73c949b784e85586065d8a1a_12.png)

这个指标直接回答了“这个策略让我赚了（或亏了）多少钱”的问题。计算结果显示，不同策略的收益有正有负，这直观地体现了投资的风险性——并非所有策略都能保证盈利。

以下是计算回测收益率的逻辑：
```python
# 假设 final_value 是最终资产总值，initial_value 是初始本金
backtest_return = (final_value - initial_value) / initial_value
```

## 年化收益率 🔄

![](img/6c53dbba73c949b784e85586065d8a1a_14.png)

然而，仅看总收益率是不够的。如果一个策略运行了10年获得50%的收益，而另一个策略运行2年就获得40%的收益，直接比较总收益是不公平的。因此，我们需要引入**年化收益率**。

![](img/6c53dbba73c949b784e85586065d8a1a_16.png)

![](img/6c53dbba73c949b784e85586065d8a1a_18.png)

年化收益率将不同时间长度的收益标准化为“年均收益”，使得不同周期的策略表现可以公平比较。它是业内更常用、更关键的评估指标。

![](img/6c53dbba73c949b784e85586065d8a1a_20.png)

年化收益率的计算公式如下：
**年化收益率 = (1 + 总收益率)^(250 / N) - 1**

![](img/6c53dbba73c949b784e85586065d8a1a_22.png)

![](img/6c53dbba73c949b784e85586065d8a1a_24.png)

公式解析：
*   **总收益率 (R)**：即上文计算出的回测总收益率。
*   **250**：代表一年中大约的交易日数量（市场惯例）。
*   **N**：代表你的回测数据所覆盖的总交易日数。

例如，我们的示例数据有730个交易日（约2.9年），那么计算幂指数就是 250 / 730。

![](img/6c53dbba73c949b784e85586065d8a1a_26.png)

![](img/6c53dbba73c949b784e85586065d8a1a_28.png)

以下是Python计算示例：
```python
# 假设 total_return 是总收益率，trade_days 是总交易日数（例如730）
annual_return = (1 + total_return) ** (250 / trade_days) - 1
```

![](img/6c53dbba73c949b784e85586065d8a1a_30.png)

计算完成后，我们可以将结果展示在表格中。对比回测收益率，年化收益率数值通常会更小（对于盈利策略）或亏损更少（对于亏损策略），因为它将总收益分摊到了每一年。

![](img/6c53dbba73c949b784e85586065d8a1a_32.png)

![](img/6c53dbba73c949b784e85586065d8a1a_34.png)

## 核心指标总结 ✅

![](img/6c53dbba73c949b784e85586065d8a1a_36.png)

本节课中我们一起学习了量化策略分析中两个最基础的收益指标：

![](img/6c53dbba73c949b784e85586065d8a1a_38.png)

![](img/6c53dbba73c949b784e85586065d8a1a_40.png)

1.  **回测收益率**：反映策略在整个测试周期内的**总盈亏**情况，是评估策略绝对表现的起点。
2.  **年化收益率**：将总收益标准化为**年均收益**，是进行跨周期、跨策略比较的**核心指标**。

![](img/6c53dbba73c949b784e85586065d8a1a_42.png)

理解并计算这两个指标，是科学评估任何一个交易策略是否有效的第一步。在后续的分析中，我们还将结合风险指标（如最大回撤、夏普比率等）来更全面地衡量策略性能。
