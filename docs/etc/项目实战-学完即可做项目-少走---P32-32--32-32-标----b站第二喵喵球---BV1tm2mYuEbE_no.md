## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体做法。

标准化的核心目标是：**使不同指标（或维度）的数据具有可比性**。例如，在分析股票数据时，价格和交易量这两个指标的数值范围差异巨大。如果直接使用原始数据，数值范围大的指标（如价格）可能会在模型中占据主导地位，从而掩盖数值范围小的指标（如某些比率）的作用。标准化通过数学变换，将所有指标转换到相似的尺度上。

标准化的数学公式如下：

**`z = (x - μ) / σ`**

其中：
*   **`x`** 是原始数据。
*   **`μ`** 是该数据所在维度的平均值（均值）。
*   **`σ`** 是该数据所在维度的标准差。
*   **`z`** 是标准化后的数据。

这个公式可以分解为两个步骤来理解：

1.  **去均值（中心化）**：`x - μ`。这一步将数据的中心移动到零点。原本不以零为中心分布的数据，经过此操作后，会以坐标原点为中心对称分布。
2.  **除标准差（缩放）**：`(x - μ) / σ`。这一步调整数据的分布范围。标准差（σ）衡量了数据的离散程度。取值范围大、分布散的数据，其标准差也大；反之则小。通过除以各自的标准差，不同维度的数据被“压缩”或“拉伸”到近似的分布范围。

最终，标准化后的数据具有以下特性：
*   均值为0。
*   标准差为1。
*   各维度数据处于相似的数值区间内。

## 标准化的代码实现

理解了原理后，我们可以动手实现标准化函数。以下是使用Python和Pandas库实现标准化的方法。

首先，我们需要计算数据的均值和标准差。

```python
import pandas as pd

# 假设我们有一个Pandas Series 或 DataFrame 列，名为 ‘data_series‘
mean_value = data_series.mean()  # 计算均值 μ
std_value = data_series.std()    # 计算标准差 σ
```

接着，应用标准化公式：

![](img/b219d0f3e92600712cf898ea0e3be882_5.png)

```python
def standardize(series):
    """
    对输入的Pandas Series进行标准化处理。
    参数:
        series: 待标准化的Pandas Series数据。
    返回:
        标准化后的Pandas Series数据。
    """
    mean_val = series.mean()  # 计算均值
    std_val = series.std()    # 计算标准差
    # 应用公式： (x - μ) / σ
    standardized_series = (series - mean_val) / std_val
    return standardized_series

![](img/b219d0f3e92600712cf898ea0e3be882_7.png)

# 使用函数
standardized_data = standardize(data_series)
```
