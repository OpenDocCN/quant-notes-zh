## 从似然函数到最小二乘法

上一节我们介绍了如何通过最大似然估计来寻找最优参数。本节中我们来看看如何将最大化似然函数的问题转化为一个更易求解的优化问题。

首先，我们回顾似然函数。它是一个累乘的形式，为了便于计算，我们通常取其对数，将累乘转化为累加。

```math
L(\theta) = \prod_{i=1}^{m} p(y_i | x_i; \theta) \quad \rightarrow \quad \ell(\theta) = \sum_{i=1}^{m} \log p(y_i | x_i; \theta)
```

对于线性回归，我们假设误差服从高斯分布。经过推导，对数似然函数可以写成以下形式：

```math
\ell(\theta) = \text{常数项} - \frac{1}{2} \sum_{i=1}^{m} (y_i - \theta^T x_i)^2
```

我们的目标是最大化似然函数 `\ell(\theta)`。观察上式，常数项是固定的，因此最大化 `\ell(\theta)` 等价于最小化减号后面的部分。这部分就是我们的目标函数，通常称为损失函数或代价函数。

```math
J(\theta) = \frac{1}{2} \sum_{i=1}^{m} (y_i - \theta^T x_i)^2
```

![](img/ca3f25849f756326a983458cafb15426_2.png)

这个目标函数有一个广为人知的名字：**最小二乘法**。它的直观含义是，我们希望模型预测值 `\theta^T x_i` 与真实值 `y_i` 之间的差距（误差的平方）总和越小越好。

![](img/ca3f25849f756326a983458cafb15426_4.png)

以下是理解这个转换的几个关键点：
*   **累乘变累加**：取对数是为了简化计算。
*   **最大变最小**：由于对数似然函数的结构，最大化它等价于最小化一个平方和项。
*   **误差项假设**：整个推导基于误差服从高斯分布的假设，这是最小二乘法的基本前提。

![](img/ca3f25849f756326a983458cafb15426_6.png)

## 参数求解：解析解推导

![](img/ca3f25849f756326a983458cafb15426_8.png)

现在，我们有了明确的目标：最小化损失函数 `J(\theta)`。接下来我们看看如何求解出使 `J(\theta)` 最小的参数 `\theta`。

首先，我们需要用矩阵形式重写损失函数。注意，这里的 `X` 不是单个数字，而是包含所有样本特征的数据矩阵；`\theta` 是参数向量；`y` 是真实值向量。

![](img/ca3f25849f756326a983458cafb15426_10.png)

![](img/ca3f25849f756326a983458cafb15426_12.png)

```math
J(\theta) = \frac{1}{2} (X\theta - y)^T (X\theta - y)
```

为了找到最小值，一个标准的数学方法是求导并令导数等于零。我们对 `J(\theta)` 关于 `\theta` 求偏导。

以下是求导过程中的关键步骤和公式：
*   利用矩阵求导法则。
*   注意到 `X^TX` 是一个对称矩阵。
*   应用公式：若 `A` 对称，则 `\frac{\partial}{\partial \theta} (\theta^T A \theta) = 2A\theta`。

经过求导和化简，我们令导数等于零，得到所谓的**正规方程**：

```math
X^T X \theta = X^T y
```

我们的目标是解出 `\theta`。如果 `X^TX` 是可逆矩阵，我们可以在等式两边同时左乘它的逆矩阵 `(X^TX)^{-1}`，从而得到参数的解析解：

![](img/ca3f25849f756326a983458cafb15426_14.png)

```math
\theta = (X^T X)^{-1} X^T y
```

这个公式就是线性回归参数的**解析解**或**闭式解**。它意味着，只要我们有了数据 `X` 和 `y`，就可以通过一次矩阵运算直接计算出最优参数 `\theta`。

---

## 对解析解的思考

虽然我们得到了一个漂亮的数学解，但我们需要思考两个重要问题。

**1. 这算是“学习”吗？**
观察公式 `\theta = (X^TX)^{-1}X^Ty`，整个过程是直接的数学计算，没有模型根据数据反馈逐步调整参数（即迭代优化）的过程。这更像是一种“一步到位”的求解，而非我们通常理解的渐进式“学习”。

**2. 解析解总是存在吗？**
解析解的存在依赖于 `X^TX` 矩阵的可逆性。然而，在以下情况中，`X^TX` 可能不可逆：
*   **特征数量大于样本数量**（即列数多于行数）。
*   **特征之间存在严格的线性关系**（即多重共线性）。
当矩阵不可逆时，解析解公式将无法直接使用。

![](img/ca3f25849f756326a983458cafb15426_16.png)

正是由于这些限制，在实际的机器学习应用中，尤其是面对大规模、高维数据时，我们更常使用下一节将要介绍的**梯度下降**等迭代优化算法来求解参数。这些算法不直接求逆，能够处理不可逆的情况，并且更贴合“学习”的概念。

## 总结
本节课中我们一起学习了线性回归模型的参数求解全过程。
1.  我们首先通过**最大似然估计**，将对数似然函数最大化问题转化为了**最小二乘**损失函数最小化问题。
2.  然后，我们通过矩阵求导，推导出了参数 `\theta` 的**解析解**公式：`\theta = (X^TX)^{-1}X^Ty`。
3.  最后，我们讨论了解析解方法的优缺点，它虽然简洁，但可能面临矩阵不可逆的问题，并且缺乏迭代“学习”的过程，这为我们引入下一节的优化算法做好了铺垫。
