## 交易逻辑的实现

![](img/f6058d6c41775079ceafc865b2a1745c_38.png)

![](img/f6058d6c41775079ceafc865b2a1745c_40.png)

接下来，我们进入`handle_bar`函数，实现具体的交易逻辑。我们不再使用`pass`占位，而是编写实际的代码。

![](img/f6058d6c41775079ceafc865b2a1745c_42.png)

首先，我们需要判断当前账户是否持有股票。我们将参考API文档来编写这部分代码。

![](img/f6058d6c41775079ceafc865b2a1745c_44.png)

以下是判断和操作持仓的关键API：

![](img/f6058d6c41775079ceafc865b2a1745c_46.png)

![](img/f6058d6c41775079ceafc865b2a1745c_48.png)

*   **`context.portfolio.positions`**：这是一个字典，包含了当前所有持仓的信息。字典的键（keys）是持有的证券代码。

![](img/f6058d6c41775079ceafc865b2a1745c_50.png)

![](img/f6058d6c41775079ceafc865b2a1745c_44.png)

![](img/f6058d6c41775079ceafc865b2a1745c_52.png)

根据API，我们编写以下逻辑：

![](img/f6058d6c41775079ceafc865b2a1745c_54.png)

1.  **获取当前持仓**：`current_positions = context.portfolio.positions`
2.  **判断持仓状态**：
    *   如果持仓字典为空（`len(current_positions) == 0`），说明是策略运行的第一天，我们需要执行买入操作。
    *   如果持仓不为空，则需要根据最新的股票池调整持仓。

以下是具体的交易逻辑步骤：

**第一步：卖出不在新股票池中的股票**
我们需要遍历当前持有的股票，如果某只股票不在我们最新筛选出的股票池中，则将其卖出。

![](img/f6058d6c41775079ceafc865b2a1745c_56.png)

![](img/f6058d6c41775079ceafc865b2a1745c_58.png)

```python
# 遍历当前持仓
for stock in current_positions.keys():
    if stock not in context.stock_pool:  # 假设股票池存储在context.stock_pool中
        # 卖出该股票，目标持仓比例为0%（即全卖）
        order_target_percent(stock, 0)
```

**第二步：买入股票池中当前未持有的股票**
接着，我们遍历最新的股票池，买入那些我们尚未持有的股票。

![](img/f6058d6c41775079ceafc865b2a1745c_60.png)

```python
# 遍历目标股票池
for stock in context.stock_pool:
    if stock not in current_positions:
        # 买入该股票，采用等权重配置
        target_percent = 1.0 / len(context.stock_pool)
        order_target_percent(stock, target_percent)
```

![](img/f6058d6c41775079ceafc865b2a1745c_62.png)

交易函数`order_target_percent`用于调整某只股票的持仓达到目标百分比。其核心参数为：
*   `stock`: 证券代码。
*   `percent`: 目标持仓占总投资组合价值的百分比（0为清仓，1为满仓）。

![](img/f6058d6c41775079ceafc865b2a1745c_64.png)

![](img/f6058d6c41775079ceafc865b2a1745c_70.png)

![](img/f6058d6c41775079ceafc865b2a1745c_66.png)

![](img/f6058d6c41775079ceafc865b2a1745c_68.png)

这样，我们就完成了一个简单的“定期调仓、等权重配置”的交易策略逻辑。

## 策略回测结果分析

策略编写完成后，我们运行回测来查看结果。

![](img/f6058d6c41775079ceafc865b2a1745c_82.png)

![](img/f6058d6c41775079ceafc865b2a1745c_72.png)

回测结果可能因选择的时间段不同而有很大差异。我们需要关注以下几个核心指标来评估策略表现：

![](img/f6058d6c41775079ceafc865b2a1745c_74.png)

1.  **累计收益率**：策略在整个回测期间的总收益。图中通常包含“策略收益”曲线和“基准收益”（如沪深300指数）曲线。
2.  **年化收益率**：将累计收益率折算成年化的形式，便于比较。
3.  **最大回撤**：策略净值从最高点到最低点的最大跌幅，用于衡量策略的风险。**公式**为：
    `最大回撤 = (峰值 - 谷值) / 峰值`
4.  **夏普比率**：衡量每承受一单位风险，能获得多少超额回报。**公式**为：
    `夏普比率 = (策略年化收益率 - 无风险利率) / 策略年化波动率`
    当收益为负时，夏普比率通常也为负。

![](img/f6058d6c41775079ceafc865b2a1745c_76.png)

![](img/f6058d6c41775079ceafc865b2a1745c_86.png)

![](img/f6058d6c41775079ceafc865b2a1745c_78.png)

在结果页面，我们可以：
*   **对比策略与基准**：观察策略曲线是否跑赢基准指数。即使两者都为负，策略若能减少亏损，也体现了一定价值。
*   **分析风险指标**：查看最大回撤是否在可接受范围内，夏普比率是否优于基准。
*   **查看交易详情**：点击右侧“交易详情”，可以复盘每一天的具体买卖操作，检查逻辑是否符合预期。

## 总结
