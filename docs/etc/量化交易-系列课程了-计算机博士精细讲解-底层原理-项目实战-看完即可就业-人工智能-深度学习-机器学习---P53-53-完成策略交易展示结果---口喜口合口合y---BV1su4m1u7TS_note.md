## 获取当前持仓股票

![](img/62c56098414167409089c8f55997012e_9.png)

![](img/62c56098414167409089c8f55997012e_11.png)

首先，我们需要一个函数来获取当前投资组合中持有的所有股票。这个函数的核心是遍历账户持仓信息，筛选出持有数量大于零的股票。

以下是获取持仓股票列表的代码步骤：

```python
def get_holdings(context):
    # 初始化一个空列表，用于存放持仓股票代码
    holdings = []
    # 从上下文信息中获取当前所有持仓
    positions = context.portfolio.positions
    # 遍历每一个持仓
    for position in positions:
        # 获取该股票的持有数量
        quantity = context.portfolio.positions[position].quantity
        # 如果持有数量大于0，则将其加入持仓列表
        if quantity > 0:
            holdings.append(position)
    # 返回持仓股票列表
    return holdings
```

## 计算交易清单

获取了当前持仓和新股票排名后，下一步是计算需要进行的交易。我们需要找出需要买入的股票（在新排名中但不在当前持仓中）和需要卖出的股票（在当前持仓中但不在新排名中）。

![](img/62c56098414167409089c8f55997012e_15.png)

以下是计算买卖清单的逻辑：

![](img/62c56098414167409089c8f55997012e_17.png)

1.  **需要买入的股票 (`to_buy`)**: 新股票排名列表 (`stocks`) 减去当前持仓列表 (`holdings`)。
2.  **需要卖出的股票 (`to_sell`)**: 当前持仓列表 (`holdings`) 减去新股票排名列表 (`stocks`)。

![](img/62c56098414167409089c8f55997012e_19.png)

为了进行集合的差集运算，我们需要将列表转换为集合（`set`）。

![](img/62c56098414167409089c8f55997012e_21.png)

```python
# 假设 stocks 是新排名列表，holdings 是当前持仓列表
stocks_set = set(stocks)
holdings_set = set(holdings)

# 计算需要买入和卖出的股票
to_buy = list(stocks_set - holdings_set)  # 在stocks中但不在holdings中
to_sell = list(holdings_set - stocks_set) # 在holdings中但不在stocks中
```

## 执行交易订单

![](img/62c56098414167409089c8f55997012e_23.png)

计算出买卖清单后，我们就可以执行具体的交易操作了。通常的策略是先卖出不再需要的股票，再用释放的资金买入新的股票。

以下是执行交易的代码框架：

![](img/62c56098414167409089c8f55997012e_29.png)

```python
# 1. 执行卖出操作：遍历需要卖出的股票列表
for stock in to_sell:
    # order_target_value 函数将目标股票的持仓价值调整到指定金额
    # 这里设置为0，代表清仓该股票
    order_target_value(stock, 0)

# 2. 执行买入操作
# 首先检查是否有股票需要买入
if len(to_buy) == 0:
    return  # 没有需要买入的股票，直接返回

# 计算平均每只股票分配的资金
average_cash = total_cash / len(to_buy)

![](img/62c56098414167409089c8f55997012e_39.png)

# 遍历需要买入的股票列表，进行买入

    # 使用平均资金买入每只股票

```
