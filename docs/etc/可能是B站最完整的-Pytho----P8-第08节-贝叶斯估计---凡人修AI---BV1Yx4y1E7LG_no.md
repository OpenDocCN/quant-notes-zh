## 贝叶斯统计简介

在引入贝叶斯统计之前，我们先对最大似然估计做一个回顾。我们将比较MLE和贝叶斯统计方法的基本原理。

### 最大似然估计回顾

最大似然估计提供了一种给定观测数据来评估模型参数的方法。我们需要对数据的分布有一个假设，即模型形式已知，但参数未知。

举个例子，假设我们要估计全国人口的身高，并认为身高服从正态分布。我们需要估计这个正态分布的均值μ和方差σ²。我们不可能测量每一个人，因此通过抽样（例如从34个城市抽取1万个样本）来估计参数。

MLE的推导流程基于以下假设和步骤：
1.  样本独立同分布。若不满足，则无法通过连乘得到联合概率分布。
2.  假设模型概率密度函数为 `f(x|θ)`，其中θ为未知参数。
3.  在样本独立的情况下，其联合概率分布（似然函数）为每个样本点概率的乘积：
    **L(θ) = ∏ f(x_i | θ)**
4.  为便于优化，通常对似然函数取对数，得到对数似然函数：
    **log L(θ) = Σ log f(x_i | θ)**
5.  通过对数似然函数关于参数θ最大化，得到参数的MLE估计值 `θ_hat`：
    **θ_hat = argmax_θ log L(θ)**

### 贝叶斯估计原理

与最大似然估计相对应的是贝叶斯估计。MLE完全依赖于观测数据，而贝叶斯估计引入了额外的“先验”信息。

贝叶斯估计有两条核心假设：
1.  与MLE相同，需要对数据的分布形式 `f(x|θ)` 有假设。
2.  不同之处在于，贝叶斯不再将参数θ视为未知的确定值，而是视为一个**未知的随机变量**。因此，参数θ本身也有一个分布，我们称之为**先验分布** `P(θ)`。

先验分布 `P(θ)` 代表了我们在获得任何观测数据之前，基于前人经验或个人认知对参数θ的初步判断。

当我们获得观测数据 `D` 后，利用贝叶斯公式来更新对θ的认识，得到**后验分布** `P(θ|D)`。贝叶斯公式如下：
**P(θ|D) ∝ P(θ) * P(D|θ)**
其中，`P(D|θ)` 就是似然函数。得到后验分布后，我们通常取其期望值作为参数的贝叶斯估计。

## 贝叶斯推断与共轭先验

上一节我们比较了MLE和贝叶斯估计的哲学差异。本节中，我们来看看贝叶斯参数估计的具体步骤，并学习一个能简化计算的重要概念——共轭先验。

贝叶斯参数估计的步骤如下：
1.  基于假设的分布 `f(x|θ)` 和观测数据 `D`，计算数据的联合概率分布（似然函数）`P(D|θ)`。
2.  定义参数θ的损失函数（例如，平方损失函数）。
3.  根据先验知识，确定参数θ的先验分布 `P(θ)`。
4.  根据贝叶斯公式，计算后验分布 `P(θ|D) = P(θ)P(D|θ) / P(D)`。
5.  结合后验分布和损失函数，通过最小化期望损失来求得参数的贝叶斯估计。

在第四步计算后验分布时，如果先验分布 `P(θ)` 和似然函数 `P(D|θ)` 具有特殊的配对关系，使得后验分布 `P(θ|D)` 与先验分布属于**同一分布族**，则称该先验分布为**共轭先验**。这能极大简化计算，因为我们可以直接得到后验分布的解析表达式。

### 抛硬币示例

我们通过一个简单的抛硬币问题来演示贝叶斯推断和共轭先验。目标是利用贝叶斯推断，随着抛硬币数据的增多，不断更新我们对硬币正面朝上概率θ的估计。

我们假设每次抛硬币是伯努利试验，似然函数服从二项分布。如果我们选择先验分布 `P(θ)` 为**贝塔分布**，那么计算出的后验分布 `P(θ|D)` 恰好也是一个贝塔分布。这就是一个典型的共轭先验例子。

根据贝叶斯公式：
**P(θ|D) ∝ P(θ) * P(D|θ)**
将贝塔先验和二项式似然代入：
**P(θ) = Beta(θ; α, β) ∝ θ^(α-1) * (1-θ)^(β-1)**
**P(D|θ) = Binom(k; n, θ) ∝ θ^k * (1-θ)^(n-k)**
因此，后验分布正比于：
**P(θ|D) ∝ θ^(α+k-1) * (1-θ)^(β+n-k-1)**
这正是贝塔分布 `Beta(θ; α+k, β+n-k)` 的核。我们看到，后验分布的参数由先验参数（α, β）加上观测数据（k次正面，n次总抛掷）的信息组合而成。

例如，先验设为 `Beta(θ; 12, 12)`，观测到50次抛掷中10次正面（k=10, n=50）。则后验分布为 `Beta(θ; 12+10, 12+50-10) = Beta(θ; 22, 52)`。我们可以直接利用这个解析式进行分析和预测。

## 马尔可夫链蒙特卡洛原理

当模型不是共轭先验时，计算后验分布 `P(θ|D)` 或其后验期望会变得非常困难，常常涉及高维复杂积分。传统数值积分方法在参数维度高时会遭遇“维度灾难”，需要指数级增长的采样点才能保证精度，这在计算上不可行。

此时，我们需要借助**马尔可夫链蒙特卡洛** 方法。MCMC的核心思想是通过构建一个马尔可夫链，使其平稳分布恰好就是我们想要的后验分布 `P(θ|D)`。通过模拟这个链的长期状态，我们就能获得来自后验分布的样本，从而用这些样本来近似计算后验统计量（如均值、中位数）。

MCMC算法（以Metropolis-Hastings算法为例）的简单步骤如下：
1.  在参数空间随机选择一个初始点 `X0`。
2.  根据一个提议分布，建议一个新的位置 `X*`。
3.  计算接受概率 `A = min(1, P(X*|D)/P(X0|D))`。
4.  以概率 `A` 接受新位置 `X1 = X*`，否则拒绝移动 `X1 = X0`。
5.  重复步骤2-4足够多次（如10000次迭代）。
6.  丢弃前期的“预热”样本后，剩下的样本序列即可视为来自后验分布 `P(θ|D)` 的近似独立样本。

在Python中，我们可以使用 `PyMC3` 或 `Stan` 等库方便地实现MCMC采样。

### MCMC模拟示例

我们使用MCMC来模拟之前抛硬币的例子（50次抛掷，10次正面），并假设先验均值为0.5，标准差为0.1。以下是使用 `PyMC3` 的示意代码：

```python
import pymc3 as pm
import numpy as np

# 观测数据
n = 50
k = 10

with pm.Model() as coin_model:
    # 定义先验分布：硬币正面概率 theta
    theta = pm.Normal('theta', mu=0.5, sigma=0.1)
    # 定义似然函数：二项分布
    obs = pm.Binomial('obs', n=n, p=theta, observed=k)
    # 使用MCMC采样
    trace = pm.sample(2000, tune=1000, return_inferencedata=False)

# 查看后验分布的统计摘要
pm.summary(trace)
# 绘制后验分布轨迹图和密度图
pm.traceplot(trace)
```

运行后，MCMC采样得到的 `theta` 的后验分布均值应接近我们之前用解析方法计算出的贝塔后验的均值。

## 贝叶斯线性回归模型

掌握了贝叶斯推断和MCMC工具后，我们可以将其应用于更实用的模型。本节介绍贝叶斯线性回归。

在经典线性回归 `y = Xβ + ε` 中，我们通过最小二乘法求得点估计 `β_hat`。在贝叶斯框架下，我们将回归系数 `β` 和噪声方差 `σ²` 都视为随机变量，并为其指定先验分布（例如，`β` 服从多元正态先验，`σ²` 服从逆伽马先验）。

模型设定如下：
-   先验：`β ~ N(μ0, Σ0)`, `σ² ~ InvGamma(a, b)`
-   似然：`y | X, β, σ² ~ N(Xβ, σ²I)`

通过贝叶斯公式，我们可以推导出参数 `(β, σ²)` 的后验分布 `P(β, σ² | y, X)`。对于共轭先验情况，其后验分布也有解析形式。对于非共轭或更复杂的模型（如包含正则化的贝叶斯岭回归），则可以使用前面介绍的MCMC方法进行采样推断。

贝叶斯线性回归的优势在于，其结果不是一个单一的点估计，而是参数完整的后验分布。这使我们能够量化估计的不确定性，并直接进行预测分布的推导。

## 贝叶斯随机波动率模型

在金融时间序列分析中，波动率（资产收益率的方差）通常不是常数，而是随时间变化的。随机波动率模型是刻画这种现象的强大工具，而贝叶斯方法为其估计提供了自然框架。

一个基本的SV模型可以表示为：
**收益率方程：** `r_t = exp(h_t/2) * ε_t,  ε_t ~ N(0,1)`
**波动率方程：** `h_t = μ + φ*(h_{t-1} - μ) + σ_η * η_t,  η_t ~ N(0,1)`
其中，`h_t = log(σ_t²)` 是对数波动率，是一个潜在的不可观测状态变量。

估计SV模型的难点在于潜在状态序列 `{h_t}` 需要与模型参数 `(μ, φ, σ_η)` 一起被推断。在贝叶斯框架下，我们可以为所有未知量（参数和潜在状态）指定先验，然后基于观测到的收益率序列 `{r_t}`，利用MCMC方法（如Gibbs采样或Hamiltonian Monte Carlo）从它们的联合后验分布中进行采样。这样，我们不仅能得到参数估计，还能得到整个波动率路径 `{h_t}` 的后验分布，从而量化其不确定性。
