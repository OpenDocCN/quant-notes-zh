## 特征选择的意义与方法

上一节我们学习了如何从原始数据构建训练集（X和Y）。本节中，我们来看看如何对这些训练集进行第一步优化，即特征选择。

特征选择的核心在于：我们最初选择的特征（X）往往是凭经验选取的，可能包含大量冗余特征（例如某一列数值全为零），这会严重影响后续分类器或回归器的性能。高质量的特征是模型成功的基础。

特征选择主要有三种思路：

1.  **子集选择**：从原始的P个特征中，选出一个包含K个特征的子集（K < P），然后仅基于这个子集进行预测。
2.  **系数收缩/正则化**：不直接剔除特征，而是在模型训练时（如使用Lasso回归），让不重要特征的系数自动趋近于零，从而实现特征选择。
3.  **降维**：通过某种函数变换，将原始的P维特征向量映射到一个全新的低维空间（如Z维， Z < P）。新特征的每一维都是原始特征的某种线性或非线性组合，其物理意义已发生改变。例如主成分分析（PCA）、自编码器（Autoencoder）都属于降维方法。

---

## 子集选择的具体策略

在三种方法中，子集选择最为直观。其难点在于：如果我们有P个特征，每个特征都有“选”或“不选”两种状态，那么所有可能的子集组合数量是2^P，这是一个指数级复杂度的问题，无法暴力枚举最优解。

因此，我们需要一些启发式方法来逼近最优解。最优解通常定义为能使模型在验证集上性能指标（如R方）最高的特征子集。

以下是几种常见的逼近策略：

*   **暴力枚举**：当特征数P很小时，可以枚举所有特征数从1到P的所有组合，并选择性能最好的子集。公式上，即对于每个k（1 ≤ k ≤ P），计算所有C(P, k)种组合的性能，取最优。
*   **前向选择**：一种贪心算法。过程如下：
    1.  开始时，最优子集S为空。
    2.  遍历所有不在S中的单个特征，将每个特征加入S并训练模型，保留能使模型性能提升最大的那个特征，将其加入S。
    3.  重复步骤2，直到加入新特征无法带来显著性能提升，或达到预设特征数。
*   **后向选择**：与前向选择相反，开始时包含所有特征。然后尝试剔除每一个特征，并计算剔除后模型的性能下降程度。剔除那个导致性能下降最少的特征。重复此过程，直到剩余特征数满足要求。

**简单来说**：前向选择是“从零开始招最优秀的员工”，后向选择是“从满员开始开除最可有可无的员工”。

## 遗传算法：受进化论启发的优化器

在介绍了基础的子集选择方法后，我们来看一种更通用、更强大的优化算法——遗传算法。它不仅可用于特征选择，还能解决各类复杂的优化问题。

遗传算法的灵感来源于生物进化理论：
1.  **种群与适应度**：一个由多个个体组成的种群，个体对环境适应度高的存活并繁衍，适应度低的被淘汰。
2.  **基因与变异**：个体的特性由其基因（DNA）决定。在繁衍过程中，基因会发生**变异**（某些位点随机改变）和**交叉互换**（不同个体的基因片段交换）。

### 算法建模与应用

在计算机中，我们这样模拟该过程：

1.  **基因编码**：将待优化问题的解编码成“基因型”。例如，在特征选择中，一个长度为P的二进制串 `[1,0,1,1,0,...]` 就是一个基因型，`1`代表选择该特征，`0`代表不选。
2.  **初始化种群**：随机生成N个这样的基因型（个体），构成初始种群。
3.  **适应度函数**：定义一个函数 `fitness(基因型)` 来评价个体的好坏。在特征选择中，`fitness` 就是使用该特征子集训练模型后得到的R方值。
4.  **选择**：根据个体的适应度，以一定概率选择优秀的个体进入“交配池”。适应度越高，被选中的概率越大。一种实现方法是：计算个体i的适应度占种群总适应度的比例 `p_i = fitness_i / sum(fitness)`，然后生成一个0-1的随机数，若随机数小于 `p_i` 则选中该个体。
5.  **交叉与变异**：
    *   **交叉**：从交配池中随机选取两个个体作为“父母”，随机选择一个位置，交换它们部分基因片段，产生“后代”。
    *   **变异**：以很小的概率（如0.05），随机改变后代基因串上的某些位点（0变1或1变0）。
6.  **迭代**：用新产生的后代种群替代旧种群，重复步骤4-6。经过多轮迭代后，种群中的个体适应度会越来越高，最终我们就能得到一个或一组近似最优解。

**核心优势**：遗传算法不依赖于问题的具体数学形式，只要你能定义`适应度函数`，它就能在庞大的解空间中进行搜索，特别擅长处理非凸、非线性等复杂优化问题。

**示例作业**：用遗传算法“猜密码”。设定一个目标字符串（如“hello world”）。定义适应度函数为猜测字符串与目标字符串相同位置的字符匹配数量。随机生成一堆字符串作为初始种群，通过选择、交叉、变异，观察算法如何逐步逼近正确答案。

## 神经网络与反向传播算法

接下来，我们探讨另一大“玄学”利器——神经网络，并深入其核心训练算法：反向传播。

### 从生物神经元到数学模型

生物神经元接收多个输入信号，进行加权求和，如果总和超过某个阈值，则产生输出（激发）。模仿此过程，最简单的数学模型——感知机如下：

给定输入向量 `X = [x1, x2, ..., xn]`，权重 `W = [w1, w2, ..., wn]`，阈值 `b`。
计算加权和：`z = w1*x1 + w2*x2 + ... + wn*xn + b`。
输出：`y = 1 if z > 0 else 0`。

为了便于优化，我们做了两个关键改进：
1.  将阈值 `b` 视为一个永远输入为1的特征所对应的权重 `w0`，这样只需学习权重 `W`。
2.  将阶梯状的输出函数替换为光滑可导的Sigmoid函数：`σ(z) = 1 / (1 + e^{-z})`，其导数有优美形式：`σ'(z) = σ(z) * (1 - σ(z))`。

### 反向传播算法：计算权重的梯度

神经网络的目的是找到一组权重 `W`，使得网络预测的输出 `Y_pred` 与真实值 `Y_true` 尽可能接近。我们定义损失函数，例如均方误差：`L = 1/2 * (Y_true - Y_pred)^2`。训练目标就是最小化 `L`。

梯度下降法告诉我们，更新权重的公式是：`W_new = W_old - α * ∂L/∂W`，其中 `α` 是学习率。因此，问题的关键在于高效计算损失函数 `L` 对每一个权重 `w` 的偏导数 `∂L/∂w`。

对于多层网络，直接求导非常复杂。反向传播算法的精妙之处在于利用**链式法则**和**动态规划**思想，从输出层向输入层反向逐层计算梯度。

**以一个最简单的两层网络为例（输入X -> 隐藏层H -> 输出Y）**：
1.  前向传播：计算网络输出及每一层的激活值。
2.  反向传播：
    *   先计算损失 `L` 对输出 `Y` 的梯度：`∂L/∂Y`。
    *   计算 `Y` 对其输入 `Z2`（即隐藏层激活值经权重变换后）的梯度：`∂Y/∂Z2`（这里用到Sigmoid导数）。
    *   计算 `Z2` 对权重 `W2` 的梯度：`∂Z2/∂W2 = H`。
    *   根据链式法则，`∂L/∂W2 = (∂L/∂Y) * (∂Y/∂Z2) * (∂Z2/∂W2)`。至此，第二层权重梯度求得。
    *   为了求第一层权重 `W1` 的梯度，我们需要继续反向传播 `∂L/∂H`，然后重复类似过程：`∂L/∂W1 = (∂L/∂H) * (∂H/∂Z1) * (∂Z1/∂W1)`，而 `∂L/∂H` 可以从 `∂L/∂Z2` 计算得到。

**算法的关键**：在计算下层权重梯度时，上层已经计算好的梯度可以被复用。这使得计算复杂度从指数级降低到与网络层数呈线性关系，使得训练深层网络成为可能。

## 循环神经网络简介

前面介绍的都是前馈神经网络，其输出仅依赖于当前输入。但在处理时间序列数据（如股价）时，当前的输出可能依赖于过去的输入。循环神经网络应运而生。

RNN的核心思想是让网络具有“记忆”。它在每个时间步 `t` 接收当前输入 `X_t`，并同时接收上一个时间步的隐藏状态 `H_{t-1}`，共同计算当前输出 `Y_t` 和新的隐藏状态 `H_t`。

其核心计算公式为：
`H_t = σ( W_hh * H_{t-1} + W_xh * X_t + b_h )`
`Y_t = W_hy * H_t + b_y`

其中，`σ` 是激活函数，`W_hh`, `W_xh`, `W_hy` 是权重矩阵，`b_h`, `b_y` 是偏置项。**注意**，不同时间步共享相同的权重矩阵。

通过这种结构，信息可以在时间维度上传递，`H_t` 理论上包含了之前所有时间步 `X_0, X_1, ..., X_t` 的信息。

实践中，基础RNN存在梯度消失/爆炸问题，难以学习长期依赖。因此，更常用的是其改进版本——**长短时记忆网络**。LSTM通过引入“门”机制（输入门、遗忘门、输出门），更精细地控制信息的遗忘与记忆，有效地缓解了长期依赖问题。对于使用者而言，可以将其视为一个更复杂的、但输入输出格式与RNN相同的“黑盒”单元。
