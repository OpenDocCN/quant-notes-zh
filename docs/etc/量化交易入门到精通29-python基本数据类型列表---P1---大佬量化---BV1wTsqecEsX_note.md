# 量化交易入门到精通：29：Python基本数据类型列表

在本节课中，我们将要学习Python中一个非常重要的基本数据类型——列表（List）。列表就像一个可以存放多个物品的容器，在量化交易中，我们常用它来管理股票池、持仓组合等数据。我们将学习如何创建列表、访问列表中的元素、修改列表内容等基本操作。

## 列表的创建与打印

列表使用方括号 `[]` 来创建，其中的元素用逗号分隔。在量化场景下，我们可以用它来定义一个股票池。

以下是创建一个包含银行股股票名称的列表的代码：

```python
# 定义一个股票池列表
stock_pool = ["工商银行", "农业银行", "中国银行", "建设银行"]
print(stock_pool)
```

运行这段代码，控制台会输出整个列表：`[‘工商银行’， ‘农业银行’， ‘中国银行’， ‘建设银行’]`。

**注意**：编写代码时，所有符号（如逗号、括号）必须使用英文半角符号。如果使用了中文符号，程序会报错。例如，错误信息 `SyntaxError: invalid character` 通常就是由中文符号引起的。

## 访问列表中的单个元素

列表中的每个元素都有一个位置索引，**索引从0开始计数**。我们可以通过索引来访问列表中的特定元素。

以下是访问列表元素的几种方式：

*   **正向索引**：从列表开头开始计数，0代表第一个元素，1代表第二个，以此类推。
    ```python
    print(stock_pool[0])  # 输出：工商银行
    print(stock_pool[2])  # 输出：中国银行
    ```
*   **反向索引**：从列表末尾开始计数，-1代表最后一个元素，-2代表倒数第二个，以此类推。
    ```python
    print(stock_pool[-1]) # 输出：建设银行
    print(stock_pool[-3]) # 输出：农业银行
    ```

## 截取列表片段（切片）

有时我们不需要整个列表，而只需要其中的一部分。这时可以使用“切片”操作，其语法为 `list[start:end]`。切片会返回从索引 `start` 到 `end-1` 的元素组成的新列表。

以下是切片操作的示例：

```python
# 截取索引1到2（不包含3）的元素
print(stock_pool[1:3]) # 输出：[‘农业银行’， ‘中国银行’]
```
这段代码截取了原列表中索引为1和2的元素。记住，切片是“左闭右开”区间。

## 修改列表中的元素

列表是可变的，这意味着我们可以直接修改其中某个位置的值。这在更新股票池时非常有用。

以下是修改列表元素的示例，我们将“农业银行”替换为“招商银行”：

```python
# 将索引为1的元素（农业银行）修改为招商银行
stock_pool[1] = “招商银行”
print(stock_pool) # 输出：[‘工商银行’， ‘招商银行’， ‘中国银行’， ‘建设银行’]
```
运行后，原列表中的第二个元素就被更新了。
