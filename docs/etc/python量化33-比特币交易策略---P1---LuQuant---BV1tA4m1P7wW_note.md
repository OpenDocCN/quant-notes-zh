## 策略逻辑与核心概念

![](img/d318d8cdba22c6d8a0ed8e87828898e8_3.png)

![](img/d318d8cdba22c6d8a0ed8e87828898e8_4.png)

上一节概述了课程目标，本节中我们来看看策略的核心逻辑与构建基础。

![](img/d318d8cdba22c6d8a0ed8e87828898e8_6.png)

该策略结合了两个技术指标：**100周期指数移动平均线（EMA）**和**VWAP**。策略的核心思想是识别趋势并寻找入场点。

*   **趋势判断**：当价格在一段时间内持续位于100EMA上方时，我们判断为上升趋势，只寻找买入机会。反之，当价格持续位于100EMA下方时，判断为下降趋势，只寻找卖出机会。
*   **信号确认**：趋势判断需同时得到VWAP指标的确认。即，在上升趋势中，价格也需位于VWAP上方；在下降趋势中，价格也需位于VWAP下方。
*   **入场触发**：在趋势确立的前提下，当价格足够接近VWAP线时，触发交易信号。

策略使用三个可调参数：
1.  **`n_candles_ema`**: 用于确认趋势所需的连续位于EMA上方或下方的K线数量。
2.  **`n_candles_vwap`**: 用于确认趋势所需的连续位于VWAP上方或下方的K线数量（为简化，可与`n_candles_ema`设为相同值）。
3.  **`max_distance`**: 定义价格“足够接近”VWAP线的最大距离阈值。

止损与止盈基于**平均真实波幅（ATR）** 计算：
*   **止损距离** = `ATR * stop_loss_atr_multiplier`
*   **止盈距离** = `止损距离 * take_profit_ratio`

## 信号生成

数据准备就绪后，接下来我们编写代码来生成交易信号。以下是生成信号的步骤：

首先，我们根据EMA判断趋势方向。
```python
# 计算连续位于EMA上方或下方的K线数量
df['ema_signal'] = ... # 根据n_candles_ema参数生成趋势信号（如：1为看跌，2为看涨）
```

接着，我们根据VWAP进行趋势确认。
```python
# 计算连续位于VWAP上方或下方的K线数量
df['vwap_signal'] = ... # 根据n_candles_vwap参数生成趋势确认信号
```

最后，我们结合趋势和价格接近度来生成最终交易信号。
```python
def generate_total_signal(row, max_distance):
    buy_condition = (row['ema_signal'] == 2) and (row['vwap_signal'] == 2)
    sell_condition = (row['ema_signal'] == 1) and (row['vwap_signal'] == 1)

    # 计算当前K线最高价和最低价与VWAP的最近距离
    distance_to_vwap = min(abs(row['high'] - row['vwap']), abs(row['low'] - row['vwap']))

    if buy_condition and distance_to_vwap <= max_distance:
        return 2  # 买入信号
    elif sell_condition and distance_to_vwap <= max_distance:
        return 1  # 卖出信号
    else:
        return 0  # 无信号

df['total_signal'] = df.apply(generate_total_signal, axis=1, args=(max_distance,))
```
