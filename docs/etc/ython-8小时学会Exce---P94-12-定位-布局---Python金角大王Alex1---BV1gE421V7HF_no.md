# Python Web开发：P94：12 定位与布局 🧭

在本节课中，我们将要学习CSS中的`position`定位属性。定位是控制网页元素位置的核心技术之一，它能实现元素脱离正常的文档流，固定在特定位置，或相对于自身进行偏移。理解定位是制作复杂网页布局的基础。

## 定位的作用与效果

上一节我们介绍了盒模型和浮动，本节中我们来看看定位。定位是用来做什么的呢？让我们通过一个实际效果来理解。

![](img/d4f83696fffa1b2a483c2c55edb052d8_1.png)

打开知乎网站，向下滚动页面。你会发现页面内容在不断向下移动，但顶部的菜单栏却始终固定在浏览器窗口的顶部。这个效果类似于Excel中的“冻结窗格”，表头可以一直固定不动。

这个固定效果就是通过CSS的`position`属性实现的。通过浏览器的开发者工具（审查元素），可以查看到菜单栏元素的`position`属性被设置为`fixed`。如果取消这个属性，菜单栏就会随页面一起滚动。

![](img/d4f83696fffa1b2a483c2c55edb052d8_3.png)

## 定位的主要属性

CSS的`position`属性有几种不同的值。对于本课程，我们只需要掌握以下三种：
*   `fixed`：固定定位
*   `relative`：相对定位
*   `static`：静态定位（默认值）

我们暂时不涉及`absolute`（绝对定位），因为它与`relative`高度相似但解释起来更复杂。掌握以上三种已足够应对多数场景。

### 固定定位：`fixed`

`fixed`定位生成一个固定定位的元素。该元素的位置**相对于浏览器窗口**进行定位，即使页面滚动，它也不会移动。

元素的具体位置通过`left`、`top`、`right`、`bottom`这四个属性来规定。

让我们通过代码示例来理解。假设我们有一个蓝色的`<div>`盒子，默认紧贴浏览器左上角。

```css
.box1 {
    width: 200px;
    height: 100px;
    background-color: blue;
}
```

![](img/d4f83696fffa1b2a483c2c55edb052d8_5.png)

现在我们希望这个蓝色盒子相对于浏览器窗口，**顶部**和**左侧**各偏移50像素。实现方法如下：

![](img/d4f83696fffa1b2a483c2c55edb052d8_7.png)

```css
.box1 {
    position: fixed; /* 1. 设置定位模式为 fixed */
    top: 50px;      /* 2. 距离浏览器顶部 50px */
    left: 50px;     /* 3. 距离浏览器左侧 50px */
    width: 200px;
    height: 100px;
    background-color: blue;
}
```

应用此样式后，你会观察到两个现象：
1.  蓝色盒子确实移动到了距离窗口左上角(50px, 50px)的位置。
2.  原本在蓝色盒子下方的其他元素（如图片）会向上移动，占据蓝色盒子原来的位置。

**第一个现象的解释**：`top: 50px;`和`left: 50px;`使元素相对于浏览器窗口进行了偏移。这与`margin`（外边距）产生的视觉效果类似，但有本质区别。`margin`是增大了盒子自身的边距，而`fixed`定位是实实在在地改变了盒子的位置，盒子本身的尺寸和边距并未改变。在布局时，通常更推荐使用定位来进行移动。

**第二个现象的解释**：这涉及到“图层”的概念。默认情况下，所有元素都在同一个图层（想象成一张纸）上按顺序排列。当给一个元素设置`fixed`定位时，相当于把它从原来的图层“拿”出来，放到了一个新的、独立的图层上。原图层上该元素的位置就空了出来，后面的元素自然会向上填充。

![](img/d4f83696fffa1b2a483c2c55edb052d8_9.png)

**`fixed`的核心特性**：无论页面如何滚动，`fixed`定位的元素都会相对于浏览器窗口保持在设定的位置。例如，即使页面很长，有滚动条，蓝色盒子也会始终固定在距离窗口顶部50px、左侧50px的地方。

### 相对定位：`relative`

`relative`定位生成相对定位的元素。该元素的位置是**相对于其正常位置**进行偏移。

![](img/d4f83696fffa1b2a483c2c55edb052d8_11.png)

同样通过示例来理解。假设蓝色盒子内部有一段文字“Hello”。

![](img/d4f83696fffa1b2a483c2c55edb052d8_13.png)

```html
<div class="box1">
    <p>Hello</p>
</div>
```

![](img/d4f83696fffa1b2a483c2c55edb052d8_15.png)

现在我们想让这段文字相对其原本的位置进行移动。

![](img/d4f83696fffa1b2a483c2c55edb052d8_17.png)

![](img/d4f83696fffa1b2a483c2c55edb052d8_19.png)

```css
.box1 p {
    color: red;
    position: relative; /* 设置定位模式为 relative */
    left: 100px;        /* 相对于原始位置向右移动100px */
    /* top: 100px; */   /* 相对于原始位置向下移动100px */
}
```

![](img/d4f83696fffa1b2a483c2c55edb052d8_21.png)

应用`left: 100px;`后，“Hello”文字会**从其原本所在的位置**向右移动100像素。它既不是相对于浏览器窗口移动，也不是相对于其父级`<div>`移动，而是相对于**自己本应出现的位置**进行偏移。

![](img/d4f83696fffa1b2a483c2c55edb052d8_23.png)

![](img/d4f83696fffa1b2a483c2c55edb052d8_25.png)

![](img/d4f83696fffa1b2a483c2c55edb052d8_27.png)

如果同时设置`top: 100px;`，文字则会从其原始位置同时向右和向下移动。这就是“相对定位”的含义。

### 静态定位：`static`

![](img/d4f83696fffa1b2a483c2c55edb052d8_29.png)

![](img/d4f83696fffa1b2a483c2c55edb052d8_31.png)

`static`是`position`属性的默认值。这意味着每个元素默认都处于静态定位。

![](img/d4f83696fffa1b2a483c2c55edb052d8_33.png)

当一个元素的定位是`static`时，任何`top`、`bottom`、`left`、`right`以及`z-index`属性都将被忽略，不会产生任何效果。元素会遵循正常的文档流进行排列。

![](img/d4f83696fffa1b2a483c2c55edb052d8_35.png)

## 字体与文本属性 🖋️

在学习了定位之后，我们简要了解一下CSS中控制字体和文本样式的常用属性。这些属性没有复杂的逻辑，主要通过实践来掌握。

![](img/d4f83696fffa1b2a483c2c55edb052d8_37.png)

![](img/d4f83696fffa1b2a483c2c55edb052d8_39.png)

以下是常用的字体和文本属性，你可以通过W3School等网站查看每个属性的具体示例和可取值：

*   `color`： 设置文本颜色。
*   `text-align`： 设置文本的水平对齐方式（如`center`居中）。
*   `text-decoration`： 设置文本装饰（如下划线`underline`、删除线`line-through`）。
*   `font-family`： 设置字体族（如`“PingFang SC”`、`“Microsoft YaHei”`）。
*   `font-size`： 设置字体大小。
*   `font-style`： 设置字体样式（如斜体`italic`）。
*   `font-weight`： 设置字体粗细（如加粗`bold`）。

`font`属性是一个简写属性，可以在一个声明中设置所有字体属性。

示例：为所有`<h3>`标题设置样式
```css
h3 {
    font-family: “PingFang SC”, sans-serif; /* 字体 */
    font-size: 30px;                       /* 大小 */
    font-style: italic;                     /* 斜体 */
    color: #333;                            /* 颜色 */
}
```

## 综合练习与总结

本节课中我们一起学习了CSS的核心布局属性——定位（`position`），并简要了解了字体文本样式。

**总结要点**：
1.  `position: fixed;` 使元素相对于浏览器窗口固定，不随页面滚动。
2.  `position: relative;` 使元素相对于其自身原始位置进行偏移。
3.  `position: static;` 是默认值，元素处于正常文档流中。
4.  定位常与`top`、`right`、`bottom`、`left`属性配合使用。
5.  字体和文本属性用于控制内容的视觉呈现，需多加练习熟悉。
