在本节课中，我们将要学习一个重要的风险指标——**最大回撤**。它用于衡量投资策略在特定时间段内可能面临的最糟糕情况，即资产净值从峰值到谷底的最大跌幅。理解最大回撤对于评估投资风险至关重要。

![](img/5c137b165196c5f09e5b70be30235f37_3.png)

## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看**最大回撤**的具体含义。

最大回撤描述的并非策略的盈利能力，而是其**风险程度**。它衡量的是在投资过程中，资产价值从历史最高点下跌到随后最低点的最大幅度。这个“跌幅”或“缩水”的程度，反映了投资者可能经历的最“惨”阶段。

用公式可以更精确地描述。假设 `P_i` 代表第 `i` 天的资产价格（或净值），`P_j` 代表 `i` 天之后某一天 `j` 的价格。最大回撤就是寻找所有可能的 `i` 和 `j`（其中 `i < j`）组合中，计算出的最大相对跌幅：

**最大回撤公式：**
`Max Drawdown = max_{i, j (i<j)} [ (P_i - P_j) / P_i ]`

这个公式计算的是从高点 `P_i` 到低点 `P_j` 的**百分比跌幅**。遍历所有时间点对，找到其中跌幅最大的那个，就是最大回撤。

## 为什么最大回撤很重要？

![](img/5c137b165196c5f09e5b70be30235f37_7.png)

![](img/5c137b165196c5f09e5b70be30235f37_9.png)

最大回撤是一个关键的风险指标。一个策略可能长期来看是盈利的，但如果其最大回撤非常大，意味着投资者需要承受巨大的心理压力和市场波动风险。历史案例，如2008年全球金融危机时股市的暴跌，就导致了巨大的回撤，凸显了管理此类风险的重要性。

## 如何用Python计算最大回撤？

![](img/5c137b165196c5f09e5b70be30235f37_15.png)

![](img/5c137b165196c5f09e5b70be30235f37_17.png)

理解了概念后，我们来看看如何用Python的Pandas库进行计算。虽然量化回测框架通常内置此功能，但手动实现有助于深入理解其原理。

![](img/5c137b165196c5f09e5b70be30235f37_19.png)

核心思路是：我们需要找到历史上每一个时间点之前出现的最高点（峰值），然后计算当前价格相对于该峰值的回撤，最后找出所有回撤中的最大值。

![](img/5c137b165196c5f09e5b70be30235f37_21.png)

![](img/5c137b165196c5f09e5b70be30235f37_23.png)

以下是计算步骤：

![](img/5c137b165196c5f09e5b70be30235f37_25.png)

![](img/5c137b165196c5f09e5b70be30235f37_27.png)

1.  **计算累积最大值**：对于时间序列中的每一个点，计算到该点为止出现过的历史最高值。
2.  **计算每日回撤**：用该历史最高值减去当日的实际值，得到当日潜在的“下跌额”。
3.  **计算回撤率**：将“下跌额”除以对应的历史最高值，得到当日的回撤百分比。
4.  **找出最大值**：在所有每日回撤率中，找出最大值，即为最大回撤。

![](img/5c137b165196c5f09e5b70be30235f37_29.png)

![](img/5c137b165196c5f09e5b70be30235f37_31.png)

让我们通过代码示例来演示：

![](img/5c137b165196c5f09e5b70be30235f37_33.png)

![](img/5c137b165196c5f09e5b70be30235f37_35.png)

```python
import pandas as pd

# 假设 df[‘net_value’] 是代表资产净值的时间序列数据
# 步骤1：计算累积最大值（到当前日期为止的历史最高净值）
cumulative_max = df['net_value'].cummax()

![](img/5c137b165196c5f09e5b70be30235f37_37.png)

![](img/5c137b165196c5f09e5b70be30235f37_39.png)

# 步骤2：计算每日回撤（下跌额）
drawdown_value = cumulative_max - df['net_value']

![](img/5c137b165196c5f09e5b70be30235f37_41.png)

# 步骤3：计算每日回撤率（百分比）
drawdown_pct = drawdown_value / cumulative_max

![](img/5c137b165196c5f09e5b70be30235f37_43.png)

![](img/5c137b165196c5f09e5b70be30235f37_45.png)

# 步骤4：找出最大回撤率
max_drawdown = drawdown_pct.max()
print(f"最大回撤率为：{max_drawdown:.2%}")
```

这段代码中，`.cummax()` 方法至关重要，它返回序列的累积最大值。通过比较累积最大值和当前值，我们就能轻松计算出每个时间点的回撤情况。
