## 数据预处理与查询
在交易日开始前，我们需要对股票池进行预处理和筛选。这通常在 `before_trading` 函数中完成。

![](img/a4b607d1fda783cbba1f896246542028_11.png)

![](img/a4b607d1fda783cbba1f896246542028_13.png)

以下是实现股票筛选的步骤：

1.  **查询财务指标**：我们需要从数据接口查询所需的财务数据，例如营业收入。
2.  **应用过滤条件**：确保我们只处理股票池内的股票。
3.  **排序与限制**：根据指标对股票进行排序，并选取排名靠前的若干只。

### 使用帮助文档
所有可用的查询指标和函数都在官方帮助文档中。进行数据查询前，务必查阅相关文档以了解函数用法和可用字段。

### 编写查询逻辑
我们将使用 `query` 函数来获取财务数据。核心是构建一个查询语句，它包含三个部分：查询字段、过滤条件和排序限制。

```python
def before_trading(context):
    # 构建查询：获取营业收入数据，并筛选、排序
    query_object = query(
        fundamentals.financial_indicator.operating_revenue # 查询营业收入指标
    ).filter(
        fundamentals.financial_indicator.stockcode.in_(context.stock_pool) # 过滤：股票代码在股票池内
    ).order_by(
        fundamentals.financial_indicator.operating_revenue.desc() # 按营业收入降序排列
    ).limit(10) # 限制只取前10名

    # 执行查询，将结果存入context以备后续使用
    context.filtered_stocks = get_fundamentals(query_object)
    # 打印结果以供检查
    print(context.filtered_stocks)
```

## 代码调试与修正
运行上述代码后，如果发现查询结果为空，可能是股票池的指定方式有误。

![](img/a4b607d1fda783cbba1f896246542028_17.png)

![](img/a4b607d1fda783cbba1f896246542028_19.png)

**问题排查**：直接使用“沪深300”作为股票代码列表可能无法被识别。我们需要使用平台提供的特定函数来获取指数成分股。

![](img/a4b607d1fda783cbba1f896246542028_21.png)

![](img/a4b607d1fda783cbba1f896246542028_23.png)

![](img/a4b607d1fda783cbba1f896246542028_25.png)

**修正方法**：使用 `get_index_stocks` 函数来获取沪深300指数的成分股代码列表。

```python
def initialize(context):
    # 修正：获取沪深300指数的成分股列表作为股票池
    context.stock_pool = get_index_stocks('000300.XSHG') # 沪深300指数代码
```
