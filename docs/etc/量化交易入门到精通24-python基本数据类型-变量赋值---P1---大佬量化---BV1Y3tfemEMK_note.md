在本节课中，我们将要学习Python编程中一个非常基础且核心的概念：变量赋值。变量是存储数据的容器，理解如何为变量赋值是编写任何程序的第一步。我们将通过简单的例子，介绍几种不同的变量赋值方法。

## 变量赋值基础

上一节我们介绍了课程概述，本节中我们来看看如何进行基础的变量赋值。在Python中，使用等号 `=` 可以为变量赋值。等号左边是变量名，右边是想要存储的值。

以下是几个基础变量赋值的例子：

```python
counter = 100
miles = 1000.0
name = "大佬量化"
```

*   `counter = 100`：将整数 `100` 赋值给变量 `counter`。
*   `miles = 1000.0`：将浮点数 `1000.0` 赋值给变量 `miles`。
*   `name = "大佬量化"`：将字符串 `"大佬量化"` 赋值给变量 `name`。

这三个变量分别存储了整数、浮点数和字符串这三种基本数据类型。

## 多个变量同时赋值

除了逐个赋值，Python还允许我们同时为多个变量进行赋值。这可以让代码更加简洁。

以下是两种常见的多变量赋值方式：

1.  **为多个变量赋予相同的值**：
    我们可以让多个变量同时指向同一个值。
    ```python
    a = b = c = 1
    ```
    执行这行代码后，变量 `a`、`b`、`c` 的值都将是 `1`。

2.  **为多个变量赋予不同的值（序列解包）**：
    我们也可以在一行内为多个变量分别赋予不同的值。
    ```python
    a, b, c = 1, 2, 3
    ```
    执行这行代码后，变量 `a` 的值为 `1`，变量 `b` 的值为 `2`，变量 `c` 的值为 `3`。等号左右两侧的值按顺序一一对应。

我们可以通过 `print` 函数来验证这些赋值操作的结果：

![](img/6bcf3e95147eb4b8dae201758a84574a_2.png)

```python
print(a, b, c) # 输出：1 2 3
```
