## 什么是命令行参数？

![](img/c6234e4b250ca39f05bb8295bb64d40b_3.png)

命令行参数是指在终端（或命令提示符）中执行程序时，跟在程序名称后面输入的额外信息。这些信息会被传递给程序内部使用。

![](img/c6234e4b250ca39f05bb8295bb64d40b_5.png)

例如，在PyCharm、终端、CMD或PowerShell等“黑乎乎”的页面中，都可以通过命令行与程序交互。这在服务器环境中尤其常见，因为服务器通常没有图形化界面。

![](img/c6234e4b250ca39f05bb8295bb64d40b_7.png)

## 为什么需要命令行参数？

在量化交易中，我们可能写好一个策略，但希望灵活调整其参数。例如，止损比例可能是5%，止盈是10%。根据不同的风险偏好或市场判断，我们可能需要修改这些参数。使用命令行参数，我们可以在不修改程序代码、不重启程序的情况下，直接传入新的参数值。

## 如何使用命令行参数？

Python通过`sys`模块来获取命令行参数。以下是具体步骤：

1.  **导入模块**：首先需要导入`sys`模块。
2.  **访问参数**：`sys.argv`是一个列表，它存储了所有的命令行参数。其中，`sys.argv[0]`是脚本的名称，`sys.argv[1:]`才是用户传入的参数。

以下是一个简单的代码示例：

```python
import sys

# sys.argv[0] 是脚本名 'command_line_args.py'
# 用户传入的参数从 sys.argv[1] 开始
script_name = sys.argv[0]
stock_code = sys.argv[1]
stock_name = sys.argv[2]
start_date = sys.argv[3]
end_date = sys.argv[4]

print(f"脚本名称: {script_name}")
print(f"股票代码: {stock_code}")
print(f"股票名称: {stock_name}")
print(f"开始日期: {start_date}")
print(f"结束日期: {end_date}")
```

假设我们将上述代码保存为 `command_line_args.py`。

## 如何运行带参数的程序？

运行带参数的程序需要在终端中使用特定的命令格式。以下是操作步骤：

首先，打开终端（如CMD、PowerShell或PyCharm的Terminal），并切换到你的Python脚本所在的目录。可以使用 `cd` 命令来切换目录。

```
cd /path/to/your/script
```

![](img/c6234e4b250ca39f05bb8295bb64d40b_9.png)

然后，使用 `python` 命令执行脚本，并在后面跟上参数。参数之间用空格分隔。

![](img/c6234e4b250ca39f05bb8295bb64d40b_11.png)

以下是运行示例：

```
python command_line_args.py 600519 贵州茅台 20240101 20241231
```

![](img/c6234e4b250ca39f05bb8295bb64d40b_13.png)

**重要提示**：命令开头必须是 `python`（或 `python3`），以告诉系统使用Python解释器来运行后面的脚本文件。如果直接输入脚本文件名，系统可能会用其他程序（如记事本）打开它，而不会执行。

执行上述命令后，终端将输出：

```
脚本名称: command_line_args.py
股票代码: 600519
股票名称: 贵州茅台
开始日期: 20240101
结束日期: 20241231
```

![](img/c6234e4b250ca39f05bb8295bb64d40b_15.png)

## 核心概念与注意事项

以下是使用命令行参数时需要了解的几个要点：

*   **参数顺序**：程序按照 `sys.argv[1]`、`sys.argv[2]`... 的顺序来读取参数，因此传入参数的顺序必须与代码期望的顺序一致。
*   **参数数量**：在访问 `sys.argv` 的元素之前，最好检查参数数量是否足够，以避免 `IndexError`（索引错误）。例如，可以使用 `len(sys.argv)` 进行判断。
*   **参数类型**：通过 `sys.argv` 获取的所有参数都是字符串（`str`）类型。如果程序中需要数字或日期类型，需要进行相应的类型转换（如 `int()`, `float()`, `datetime.strptime()`）。
