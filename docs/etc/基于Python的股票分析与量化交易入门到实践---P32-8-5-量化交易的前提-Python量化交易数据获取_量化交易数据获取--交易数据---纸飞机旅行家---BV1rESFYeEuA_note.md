## 获取行情数据 📈

上一节我们介绍了如何获取标的信息，本节中我们来看看量化交易中最核心的数据——股票的交易数据。获取行情数据主要使用 `get_price` 函数。

`get_price` 函数用于获取证券的行情数据，支持股票、债券、基金等多种标的。它可以同时获取多个标的的多个数据字段。

以下是 `get_price` 函数的主要参数介绍：

*   **security**：支持单个股票代码或一个股票代码的列表。本课程主要关注股票代码。
*   **count** 与 **start_date**：二者选一，不可同时使用。`count` 指定返回的数据条数，`start_date` 指定开始日期。
*   **end_date**：指定结束日期，与 `start_date` 配合使用。
*   **frequency**：指定数据频率。格式为 `xd`（x天）或 `xm`（x分钟），例如 `1d` 表示日线，`60m` 表示60分钟线。默认是 `daily`（日线）。
*   **fields**：一个列表，指定需要获取的数据字段，如开盘价、收盘价等。默认不传则获取所有字段。
*   **skip_paused**：布尔值，指定是否跳过非交易日（如停牌、未上市日期）。默认为 `False`。
*   **fq**：复权选项，主要处理股票分红、拆股等对价格的影响。
*   **panel**：布尔值，用于控制多股票返回数据的格式。在 `pandas 0.25` 版本后，`Panel` 数据结构已被移除，建议设置为 `False`。
*   **fill_paused**：布尔值，指定对停牌期间的价格是否进行处理。`True` 表示用前一个交易日的收盘价填充，`False` 则用空值填充。

`get_price` 是量化交易中最常用的接口之一，虽然它最高只提供分钟级数据，但对于学习和初步实践已经足够。

## 获取龙虎榜数据 🏆

接下来，我们介绍另一个常用数据——龙虎榜数据。龙虎榜揭示了哪些股票在当日交易中出现异常波动，以及是哪些证券营业部在大量买入或卖出。

获取龙虎榜数据的函数是 `get_billboard_list`。

以下是 `get_billboard_list` 函数的主要参数介绍：

*   **stock_list**：股票代码的列表。如果为空，则返回指定日期内所有上榜的股票。
*   **start_date** 与 **end_date**：指定查询的开始和结束日期。
*   **count**：指定返回的数据条数。与 `start_date` 和 `end_date` 参数二选一使用。

以上便是获取交易数据（行情数据和龙虎榜数据）两个核心函数的基本介绍。

## 代码实战环节 💻

理论介绍完毕，现在进入代码实战环节。我们将通过具体示例演示如何使用这些函数。

### 实战一：获取行情数据

首先，我们演示如何使用 `get_price` 获取单只股票的日线行情数据。

```python
# 获取平安银行（000001）最近5个交易日的日线数据
df = get_price('000001.XSHE', count=5, frequency='daily', fields=['open', 'close', 'high', 'low', 'volume', 'money'])
print(df)
```

![](img/57ec49d7380d293f8eb3a962c6e34083_1.png)

运行上述代码，将输出包含开盘价、收盘价、最高价、最低价、成交量和成交额的数据框。

接下来，我们获取指定时间段的分钟级数据。

```python
# 获取平安银行在2023年3月1日全天的分钟级开盘价和收盘价
df_minute = get_price('000001.XSHE', start_date='2023-03-01', end_date='2023-03-01', frequency='1m', fields=['open', 'close'])
print(df_minute.head(20)) # 查看前20行
```

这段代码将输出该股票在指定日期的每分钟开盘价和收盘价。

最后，我们演示如何同时获取多只股票的行情数据。例如，获取中证100指数所有成分股近3天的数据。

```python
# 首先获取中证100指数（000903）的成分股列表
stock_list = get_index_stocks('000903.XSHG')
# 获取这些股票最近3个交易日的行情数据
df_multi = get_price(stock_list, count=3, frequency='daily', panel=False)
print(df_multi.shape) # 查看数据维度
# 筛选出平安银行的数据进行查看
df_000001 = df_multi[df_multi['code'] == '000001.XSHE']
print(df_000001)
```

通过设置 `panel=False`，返回的数据是一个标准的数据框，便于使用 `pandas` 进行筛选和分析。

### 实战二：获取龙虎榜数据

现在，我们来获取最新的龙虎榜数据。

```python
# 获取最近一个交易日的龙虎榜数据
df_billboard = get_billboard_list(count=1)
print(df_billboard.shape)
# 查看前10条数据的关键字段
print(df_billboard.head(10)[['code', 'abnormal_name', 'sales_depart_name', 'rank']])
```

运行代码后，可以看到上榜股票的代码、异常波动原因、营业部名称和排名等信息。

我们也可以获取最近多个交易日的龙虎榜数据。

```python
# 获取最近2个交易日的龙虎榜数据，并筛选关键字段
df_billboard_2days = get_billboard_list(count=2)
# 选择我们关心的列进行查看
selected_cols = ['code', 'abnormal_name', 'sales_depart_name', 'rank']
print(df_billboard_2days[selected_cols].head(15))
```

从输出中，可以清晰地看到哪些股票因“涨幅偏离值达7%”等原因上榜，以及是哪些营业部在活跃交易。
