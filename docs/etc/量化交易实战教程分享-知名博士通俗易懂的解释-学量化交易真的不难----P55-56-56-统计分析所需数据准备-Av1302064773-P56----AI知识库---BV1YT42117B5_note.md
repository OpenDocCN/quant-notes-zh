统计分析听起来很简单。其核心思路是：我们拥有“前一天”和“前两天”的指标数据。我们可以基于历史频率进行统计。例如，我们可以统计当“前两天”和“前一天”都是下跌时，第三天上涨的可能性大还是下跌的可能性大。同样，我们也可以统计“前一天跌、前两天涨”等不同组合下，第三天的表现。通过计算各种情况出现的概率，我们可以基于历史统计规律来制定交易策略。

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_5.png)

这个方法本身非常传统，并非推荐的实际交易策略。本节的**核心目标**是学习如何用Python实现这一数据处理过程。

## 第一步：数据标记与分类

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_9.png)

首先，我们需要对数据进行标记。我们要统计“前两天”和“前一天”的数据情况，并按涨跌区间进行分类。

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_11.png)

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_13.png)

具体来说，我们需要判断“前两天”和“前一天”的指标值是上涨（大于0）还是下跌（小于或等于0）。然后，我们将判断结果以新的列添加到数据中。

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_15.png)

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_17.png)

以下是实现这一步骤的代码：

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_19.png)

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_21.png)

```python
def create_binary_data(data):
    """
    将特征数据根据与0的比较结果转换为二值（0或1）。
    1 代表指标值 > 0 (上涨)
    0 代表指标值 <= 0 (下跌或不变)
    """
    # 定义需要处理的列名列表，这里以‘L1’（前一天）和‘L2’（前两天）为例
    global col_names
    col_names = ['L1', 'L2']  # 假设这是之前定义好的全局变量

    # 遍历每一列特征
    for col in col_names:
        # 为新列命名，例如‘L1’变为‘L1_GB’
        new_col_name = col + '_GB'
        # 判断该列数据是否大于0，结果转换为整数（True->1, False->0）
        data[new_col_name] = (data[col] > 0).astype(int)

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_23.png)

# 调用函数处理数据
create_binary_data(data)
```

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_25.png)

执行上述代码后，我们的数据框会增加两列，例如 `L1_GB` 和 `L2_GB`。它们的值为 **1**（表示对应日期的指标为“涨”）或 **0**（表示“跌”或“平”）。

为了查看处理结果，我们可以使用以下代码：

## 第二步：分组统计频率

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_41.png)

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_43.png)

数据标记完成后，接下来我们需要进行分组统计。目标是统计出“前一天”（L1_GB）和“前两天”（L2_GB）每一种涨跌组合下，“当天”实际涨跌（`direction`）的分布情况。

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_45.png)

我们可以使用Pandas库中强大的 `groupby` 方法来实现。以下是具体操作：

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_47.png)

```python
# 按照‘L1_GB‘, ’L2_GB‘, ’direction‘ 进行分组，并计算每种组合出现的次数
grouped = data.groupby(['L1_GB', 'L2_GB', 'direction']).size()

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_49.png)

# 打印分组统计结果
print(grouped)
```

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_51.png)

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_53.png)

这个结果非常直观。例如，第一行 `(0, 0, -1)` 表示：当前一天标记为跌（0）、前两天标记为跌（0）时，当天实际下跌（-1）的情况在历史数据中出现了150次。同理，`(0, 0, 1)` 表示相同的前置条件下，当天实际上涨（1）的情况出现了90次。

如果某种组合（如 `(1, 0, 0)`，如果direction没有0值）在历史中从未出现，`groupby` 结果中将不会包含该行。

本节课中我们一起学习了为统计分析策略准备数据的关键两步。

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_71.png)

1.  **数据标记**：我们将连续的技术指标值，通过**与零比较**，转换为代表涨跌的二值标签（0和1）。这通过一个简单的函数 `create_binary_data` 实现。
2.  **频率统计**：我们利用Pandas的 `groupby()` 方法，对“前一天状态”、“前两天状态”和“当天实际方向”进行分组，并统计每种组合出现的**次数**。这为我们提供了历史条件概率的原始数据。

![](img/ea0f713ba83e39f75f2b50a6bf3119d2_73.png)

通过这两步，我们就得到了一个清晰的“历史经验表”。在接下来的课程中，可以基于此表计算概率（例如，`(0,0)`条件下，下跌概率=150/(150+90)≈62.5%），并据此构建最简单的“统计跟随”交易策略。这个过程充分展示了如何将交易想法转化为可执行的Python代码。
