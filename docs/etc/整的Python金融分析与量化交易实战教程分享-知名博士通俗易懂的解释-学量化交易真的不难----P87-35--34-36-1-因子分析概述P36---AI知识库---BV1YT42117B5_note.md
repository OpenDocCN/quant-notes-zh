# 量化交易完全自学教程：P87：35. 因子分析概述

在本节课中，我们将要学习因子分析的核心概念。因子分析是量化交易中筛选有效预测指标的关键步骤，目的是从众多候选因子中，找出那些与未来股票收益率有较强相关性的因子。

## 因子分析的目标

上一节我们介绍了量化交易的基本框架，本节中我们来看看如何筛选有效的预测因子。假设我们拥有A股票的大量因子数据，例如基本面指标（如市盈率）和技术指标（如移动平均线）。这些因子可以划分为多个大类，每个大类下又可细分为众多具体指标。

现在，我们面临一个实际问题：如果手头有300个因子，在进行策略回测或设计时，是全部使用还是只选用其中一部分？我们需要对这些因子进行排序和评估，判断哪些因子对最终的投资收益有正面影响，哪些没有或甚至有负面影响。换句话说，我们需要根据每个因子对收益的“贡献度”进行打分和排序。

## 如何评估因子好坏

那么，如何判断一个因子的好坏呢？最基础的方法是观察因子与收益率之间的关系。首先，我们需要明确收益率的定义。

收益率通常指股票价格的变化率。以日收益率为例，其计算公式为：
`日收益率 = (当日收盘价 - 前一日收盘价) / 前一日收盘价`

我们的目标是实现长期盈利，因此希望找到那些与日收益率变化存在稳定关系的因子。假设我们有一年的数据（约250个交易日），对于某个因子F和收益率R，我们每天都会得到一对数值（F_t, R_t）。我们需要分析因子F的数值变化与收益率R的变化之间存在何种关系：是正相关、负相关，还是无关？

## 因子分析的核心：IC值分析

![](img/f5af0a8f30b417129109f490cf842492_1.png)

其实说白了，因子分析就是做上述相关性的量化评估。分析中主要进行两项工作，第一项是**因子的IC分析**。

IC（Information Coefficient，信息系数）是一个衡量相关性的指标。在单因子分析中，我们计算**特定因子**与**未来一期收益率**之间的相关性。这里通常使用**斯皮尔曼秩相关系数**。其计算公式虽未在此列出，但其取值范围在-1到1之间：
*   **越接近+1**：表示因子与收益率**正相关**越强。
*   **越接近-1**：表示因子与收益率**负相关**越强。
*   **越接近0**：表示因子与收益率基本**无关**。

由于我们有300个因子，而收益率序列相对固定，因此分析过程类似于一个循环：遍历每一个因子，分别计算其与收益率序列的斯皮尔曼相关系数。这个计算出的相关系数值，就被称为该因子的**IC值**。

## IC值的解读与应用

计算出IC值后，我们需要对其进行分析。以下是IC值分析中常见的可视化图表及其含义：

![](img/f5af0a8f30b417129109f490cf842492_1.png)

![](img/f5af0a8f30b417129109f490cf842492_3.png)

*   **IC值序列图**：图中蓝色折线展示了某个因子在历史上一段时间内（例如几个月）每日IC值的变化情况。它反映了该因子预测能力的稳定性。
*   **IC值移动平均线**：图中绿色线条是蓝色IC值序列的移动平均线（例如10日移动平均）。它用于平滑每日IC值的波动，帮助我们更清晰地观察因子预测能力的长期趋势。图表前期的空值是因为移动平均计算需要一定的窗口期数据。

![](img/f5af0a8f30b417129109f490cf842492_3.png)

![](img/f5af0a8f30b417129109f490cf842492_5.png)

通过观察IC值序列及其移动平均线，我们可以评估因子：
1.  **预测方向**：IC值长期平均为正还是为负，决定了因子预示的是上涨还是下跌。
2.  **预测强度**：IC值的绝对值大小，衡量了因子预测能力的强弱。绝对值越大，预测越有效。
3.  **稳定性**：IC值序列的波动程度。波动越小，因子的预测能力越稳定。

我们的目标是寻找那些IC值**绝对值较大**（即与收益率关系紧密）、**方向稳定**且**序列较平稳**的因子。对于那些IC值接近零或极不稳定的因子，则可以考虑剔除。

## 总结

![](img/f5af0a8f30b417129109f490cf842492_7.png)

本节课中我们一起学习了因子分析的基础。我们首先明确了因子分析的目标是从海量指标中筛选有效因子。接着，我们介绍了通过计算因子与收益率之间的**斯皮尔曼秩相关系数**来得到**IC值**，以此量化因子的预测能力。最后，我们讲解了如何通过分析IC值序列图及其移动平均线来评估因子的预测方向、强度和稳定性，从而完成因子的初步筛选。这是构建量化模型至关重要的一步。
