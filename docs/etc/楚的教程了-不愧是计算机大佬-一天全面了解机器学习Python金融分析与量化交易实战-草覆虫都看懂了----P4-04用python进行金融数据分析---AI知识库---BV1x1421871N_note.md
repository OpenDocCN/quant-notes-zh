## 机器学习概述 🤖

上一节我们介绍了量化交易的基本概念，本节中我们来看看机器学习的核心思想。

传统的编程方法是由程序员或交易员根据观察到的规则（例如，移动平均线交叉）来定义交易策略，并将其编写成`if-else`程序。计算机根据这些预设规则输出信号。

![](img/165a2a35ce8bb4032edb6b617efa9b0e_5.png)

机器学习与传统方法最根本的不同在于，其规则不是由程序员制定的，而是从数据中自动学习出来的。程序通过分析大量历史数据（输入`X`和对应的输出`Y`），自动生成一系列规则。这种方法的好处在于，计算机能够从高维数据中发现人类难以直观观察到的隐藏关系。

机器学习主要分为三类：
*   **监督学习**：给定输入`X`和对应的标签`Y`（可以是具体数值或类别），学习从`X`到`Y`的映射关系。在量化中，预测股价涨跌（分类）或具体价格（回归）是常见任务。
*   **无监督学习**：只有输入`X`，没有标签`Y`。目标是发现数据内在的结构，例如对股票进行聚类分组，或对高维特征进行降维。
*   **强化学习**：智能体在与环境（如股票市场）的交互中，通过尝试不同的动作（如买卖）并根据获得的奖励来学习最优策略。其理论基础是马尔可夫决策过程。

## 使用scikit-learn进行机器学习 🛠️

![](img/165a2a35ce8bb4032edb6b617efa9b0e_7.png)

`scikit-learn`是Python中一个功能强大且广泛使用的机器学习库。除了深度神经网络，它涵盖了大多数经典机器学习算法。

一个典型的`scikit-learn`建模流程分为三步：
1.  **准备数据**：获取并划分训练集和测试集。`X_train`通常是一个`n_samples × n_features`的矩阵，`y_train`是对应的标签向量或矩阵。
2.  **训练模型**：初始化一个模型（如`LinearRegression()`），调用`model.fit(X_train, y_train)`方法。这个过程会基于训练数据学习出模型的最优参数。
3.  **评估模型**：使用训练好的模型对测试集进行预测`model.predict(X_test)`，然后将预测结果`y_pred`与真实标签`y_test`进行比较，使用评分函数（如准确率、均方误差）评估模型性能。

**注意**：**绝对不能在训练集上评估模型性能**，这会导致过拟合，即模型只记住了训练数据而无法泛化到新数据。必须使用独立的测试集。

## 从线性回归到高级线性模型 📊

![](img/165a2a35ce8bb4032edb6b617efa9b0e_9.png)

![](img/165a2a35ce8bb4032edb6b617efa9b0e_11.png)

上一节我们了解了机器学习的基本流程，本节中我们来看看最基础也是最重要的模型家族——线性模型。

![](img/165a2a35ce8bb4032edb6b617efa9b0e_13.png)

### 普通最小二乘法（OLS）

线性回归模型假设输出`Y`与输入`X`之间存在线性关系，其公式为：
`Y = X · θ`
其中，`X`是特征矩阵，`θ`是待学习的参数向量。模型的目标是找到一组参数`θ`，使得预测值`Xθ`与真实值`Y`之间的误差平方和最小，因此也称为“最小二乘法”。

该模型有一个解析解（闭式解）：
`θ = (X^T X)^(-1) X^T y`
这意味着我们可以直接通过矩阵运算得到最优参数。

### 岭回归（Ridge Regression）

在实践中，直接求`(X^T X)`的逆可能遇到数值不稳定的问题（矩阵病态）。一个计算技巧是给`(X^T X)`加上一个小的单位矩阵`λI`再求逆。有趣的是，人们发现这样做不仅解决了计算问题，而且得到的模型在测试集上的表现往往比普通最小二乘更好。

这引导出了岭回归，它在最小二乘的目标函数中增加了一个惩罚项：
`J(θ) = ||y - Xθ||^2 + λ ||θ||^2`
其中，`λ`是一个超参数，`||θ||^2`是参数向量`θ`的L2范数平方（即各参数平方和）。

**几何解释**：最小二乘的损失函数在参数空间中形成一个“椭圆”，而惩罚项`λ ||θ||^2`的约束是一个“圆”。岭回归的最优解是这两个图形相切的点。通过控制`λ`，我们可以让解在“最小二乘解”和“零向量”之间平滑移动。`λ`越大，惩罚越重，参数值会被“压缩”得更接近零，这有助于防止过拟合，提升模型泛化能力。

### Lasso回归

岭回归虽然能压缩参数，但不会将任何参数恰好压缩到零。然而，在特征选择中，我们常常希望模型能自动识别出不重要的特征，并将其系数设为零，从而实现特征筛选。

Lasso回归通过修改惩罚项实现了这一点：
`J(θ) = ||y - Xθ||^2 + λ ||θ||_1`
这里，`||θ||_1`是参数向量`θ`的L1范数（即各参数绝对值之和）。

**几何解释**：L1惩罚项在参数空间中的约束区域是一个“菱形”。菱形与椭圆的切点更容易落在坐标轴上，这意味着某些参数`θ_i`会恰好为零。因此，Lasso回归天然具备特征选择的能力。

## 从线性到非线性：核方法 🔄

上一节我们讨论了线性模型的增强版，本节中我们来看看如何让线性模型处理非线性关系。

一个直观的想法是：我们仍然使用线性模型，但对输入特征`X`进行非线性变换。例如，对于一维输入`x`，我们可以构造新的特征`[x, x^2, x^3, ...]`，然后在这个新的高维空间里进行线性回归。这样，在原输入空间里，模型就表现为非线性。

核方法（如径向基函数网络）将这一思想推向极致。它使用一组“基函数”（通常是高斯函数）来对输入空间进行变换。每个基函数的形式类似于：
`φ_i(x) = exp(-γ ||x - μ_i||^2)`
其中，`μ_i`是基函数的中心点，`γ`控制函数的宽度。

模型输出是这些基函数的线性组合：
`y_pred = Σ θ_i · φ_i(x)`
理论上，足够多的基函数可以以任意精度逼近任何连续函数。

**关键问题**：
1.  **基函数中心`μ_i`如何确定？** 一个常见做法是对训练数据`X`进行K-Means聚类，将聚类中心作为`μ_i`。
2.  **基函数数量如何确定？** 这相当于选择聚类的数量`K`，需要通过交叉验证来选择。
3.  **宽度参数`γ`如何确定？** 同样需要通过交叉验证来选择。

## 模型选择与交叉验证 🎯

无论是岭回归的`λ`、Lasso的`λ`，还是核方法的`γ`和`K`，这些都需要人为设定的参数称为“超参数”。如何科学地选择超参数是机器学习中的关键步骤。

![](img/165a2a35ce8bb4032edb6b617efa9b0e_15.png)

![](img/165a2a35ce8bb4032edb6b617efa9b0e_17.png)

**错误做法**：直接在测试集上尝试不同超参数，选择表现最好的那个。这会导致“数据窥探偏差”，因为你使用测试集信息来调整模型，最终报告的测试集性能会过于乐观，无法反映模型在全新数据上的真实表现。

**正确做法：交叉验证**
1.  将数据分为三部分：**训练集**（如70%）、**验证集**（如15%）、**测试集**（如15%）。
2.  在训练集上用**不同的超参数组合**训练多个模型。
3.  在**验证集**上评估这些模型的性能，选择在验证集上性能最好的超参数组合。
4.  使用选出的最佳超参数，在**训练集+验证集**上重新训练一个最终模型。
5.  在**测试集**上评估这个最终模型的性能，此性能可作为对外汇报的指标。
6.  如果决定部署模型，最后可以使用**全部数据**（训练+验证+测试）以最佳超参数重新训练一个用于生产的模型。

对于多个超参数，可以使用网格搜索（Grid Search）遍历所有组合，并用热力图可视化不同组合在验证集上的性能。

## 定义金融特征（X）💡

![](img/165a2a35ce8bb4032edb6b617efa9b0e_21.png)

本节课我们一起学习了多种强大的建模工具。然而，所有模型都需要输入特征`X`。在量化交易中，如何为每个时间点`t`构造有意义的特征向量`X_t`，是策略成功与否的核心。

特征可以来源于多方面：
*   **价格信息**：开盘价、收盘价、最高价、最低价。
*   **成交量信息**。
*   **技术指标**：移动平均线（MA）、相对强弱指数（RSI）、布林带（Bollinger Bands）、MACD等。这些指标通常由量价数据计算得出。
*   **其他数据**：新闻情绪、宏观经济指标等（需要将其编码为数值特征）。

构建特征集是一个创造性的过程，体现了你对市场规律的理解。你可以从实现常用的技术指标函数开始，逐步构建和丰富你的特征库。
