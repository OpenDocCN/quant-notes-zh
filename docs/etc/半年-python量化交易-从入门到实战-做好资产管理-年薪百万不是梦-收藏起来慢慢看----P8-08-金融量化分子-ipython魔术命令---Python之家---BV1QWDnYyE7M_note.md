# Python量化交易：P8：08 金融量化分子-IPython魔术命令 🪄

在本节课中，我们将要学习IPython中一个非常实用且强大的高级功能——魔术命令。这些命令以百分号开头，能够极大地提升我们在交互式环境中的工作效率，例如直接运行外部脚本、粘贴执行代码片段以及精确测量代码运行时间等。

---

上一节我们介绍了IPython的基础环境，本节中我们来看看如何利用魔术命令来提升我们的开发体验。

魔术命令是IPython提供的一系列以百分号（`%`）开头的特殊命令。它们允许我们在不退出交互式环境的情况下，执行一些通常需要在外部命令行或编辑器中完成的操作。

## 运行外部Python脚本

![](img/d3f1fce5c258c6ba50a59fe688eff871_3.png)

在标准的Python命令行中，要运行一个Python文件，通常需要先退出命令行，然后执行类似 `python script.py` 的命令。在IPython中，我们可以使用 `%run` 命令直接运行脚本。

**公式/代码**：`%run 文件名.py`

![](img/d3f1fce5c258c6ba50a59fe688eff871_5.png)

例如，假设我们在当前文件夹下有一个名为 `hello.py` 的Python文件，其内容为 `print(“Hello World”)`。我们可以在IPython中直接输入：

```python
%run hello.py
```

执行后，`Hello World` 会直接输出在当前的交互式环境中，仿佛代码是在这里逐行执行的一样。

## 粘贴并执行代码

![](img/d3f1fce5c258c6ba50a59fe688eff871_7.png)

有时，我们可能从编辑器或其他地方复制了一段代码，希望在IPython中快速测试。虽然可以直接粘贴，但可能会遇到格式问题（如缩进错误）。IPython提供了 `%paste` 命令来解决这个问题。

![](img/d3f1fce5c258c6ba50a59fe688eff871_9.png)

以下是 `%paste` 命令的使用方法：

![](img/d3f1fce5c258c6ba50a59fe688eff871_11.png)

1.  首先，从任何地方复制一段代码到剪贴板。
2.  在IPython中输入 `%paste` 并回车。
3.  IPython会先打印出剪贴板中的代码内容，然后用一个分隔符隔开，最后执行这段代码。

![](img/d3f1fce5c258c6ba50a59fe688eff871_13.png)

这个功能对于测试长代码中的特定片段非常有用，无需单独创建文件。

## 精确测量代码运行时间

在性能分析和优化时，我们经常需要测量一段代码的执行时间。虽然可以使用Python的 `time` 模块，但对于运行时间极短的代码，单次测量的结果可能不准确（例如显示为0秒）。

IPython的 `%timeit` 命令通过自动多次运行代码并计算平均时间来提供高精度的测量结果。

**公式/代码**：`%timeit 代码语句`

例如，我们想测量对一个包含1万个随机数的列表进行排序所需的时间：

```python
import random
lst = [random.random() for _ in range(10000)]

%timeit sorted(lst)
```

`%timeit` 会智能地决定运行次数（例如，“7次运行，每次循环100万次”），并输出类似“146 µs ± 742 ns per loop”的结果，这表示平均每次排序耗时约146微秒，误差在正负742纳秒之间。这对于优化频繁调用的微小操作至关重要。

![](img/d3f1fce5c258c6ba50a59fe688eff871_15.png)

本节课中我们一起学习了IPython的三个核心魔术命令：`%run`、`%paste` 和 `%timeit`。`%run` 命令让我们能在交互环境中直接运行外部脚本；`%paste` 命令方便我们安全地粘贴和执行剪贴板中的代码；`%timeit` 命令则为代码性能分析提供了高精度的计时工具。掌握这些命令，能让你在数据分析和量化交易的探索过程中更加得心应手。
