## 序列的相关性与随机游走

上一节我们介绍了课程的整体框架，本节中我们首先来看看序列的相关性。通过绘制时间序列及其滞后项的散点图，我们可以直观地观察序列的自相关性。

以下是分析序列相关性的关键步骤：
1.  绘制原始时间序列图。
2.  绘制时间序列与其一阶滞后项的散点图。
3.  绘制时间序列与其更高阶滞后项的散点图，观察相关性衰减情况。

为了量化这种相关性，我们引入自相关函数。

### 自相关函数

自相关函数用于衡量时间序列与其自身滞后项之间的线性相关性。其数学定义如下：

对于一个时间序列 `{X_t}`，在时滞 `k` 处的自相关函数 `ρ_k` 定义为：
`ρ_k = Corr(X_t, X_{t-k})`

![](img/518076225d4ccd8ff04e70acee92896d_5.png)

我们可以通过绘制不同时滞 `k` 对应的 `ρ_k` 图像来判断时间序列的相关性结构。

![](img/518076225d4ccd8ff04e70acee92896d_7.png)

![](img/518076225d4ccd8ff04e70acee92896d_9.png)

![](img/518076225d4ccd8ff04e70acee92896d_11.png)

*   **白噪声**：除 `k=0` 时 `ρ_0=1` 外，其他所有 `ρ_k` 均接近于0。
*   **随机游走**：所有时滞 `k` 的 `ρ_k` 都接近于1，表明极强的持续性。
*   **MA过程**：自相关函数在 `q` 阶后“截尾”（显著为零），`q` 即为MA的阶数。
*   **AR过程**：自相关函数呈“拖尾”状（逐渐衰减至零），无法直接判断阶数。

由于ACF无法判断AR过程的阶数，我们需要引入偏自相关函数。

### 偏自相关函数

偏自相关函数衡量的是在控制了中间滞后项的影响后，`X_t` 与 `X_{t-k}` 之间的纯相关性。

*   **AR过程**：偏自相关函数在 `p` 阶后“截尾”，`p` 即为AR的阶数。

![](img/518076225d4ccd8ff04e70acee92896d_13.png)

通过联合使用ACF和PACF图，我们可以初步判断时间序列适合的模型类型（AR、MA或ARMA）及其可能的阶数。

![](img/518076225d4ccd8ff04e70acee92896d_15.png)

## 平稳时间序列模型

在理解了如何判断序列的相关性后，我们正式介绍三类经典的平稳时间序列模型。

### 移动平均模型

MA模型将当前值表示为当前及过去一系列白噪声的线性组合。

![](img/518076225d4ccd8ff04e70acee92896d_17.png)

一阶MA模型的公式为：
`X_t = ε_t + θ * ε_{t-1}`
其中 `ε_t` 是均值为0、方差为常数的白噪声。

![](img/518076225d4ccd8ff04e70acee92896d_19.png)

### 自回归模型

![](img/518076225d4ccd8ff04e70acee92896d_21.png)

AR模型将当前值表示为自身过去一系列值的线性组合。

![](img/518076225d4ccd8ff04e70acee92896d_23.png)

一阶AR模型的公式为：
`X_t = φ * X_{t-1} + ε_t`

![](img/518076225d4ccd8ff04e70acee92896d_25.png)

### 自回归移动平均模型

![](img/518076225d4ccd8ff04e70acee92896d_27.png)

ARMA模型结合了AR和MA模型，提供了更一般的表达形式。

![](img/518076225d4ccd8ff04e70acee92896d_29.png)

![](img/518076225d4ccd8ff04e70acee92896d_31.png)

ARMA(p, q)模型的公式为：
`X_t = φ_1*X_{t-1} + ... + φ_p*X_{t-p} + ε_t + θ_1*ε_{t-1} + ... + θ_q*ε_{t-q}`

## 从非平稳到平稳的转换

大多数金融时间序列（如股价）是非平稳的。为了应用ARMA模型，我们需要先将它们转换为平稳序列。

![](img/518076225d4ccd8ff04e70acee92896d_33.png)

![](img/518076225d4ccd8ff04e70acee92896d_35.png)

![](img/518076225d4ccd8ff04e70acee92896d_37.png)

一个非平稳序列 `X_t` 通常可以分解为三个部分：
`X_t = Trend_t + Seasonal_t + Y_t`
其中 `Y_t` 是一个平稳序列。

以下是常见的转换方法：
1.  **去趋势**：如果序列有明显的趋势（如线性增长），可以通过回归拟合趋势线，然后对残差进行建模。
2.  **差分**：计算序列的连续值之差。一阶差分公式为 `∇X_t = X_t - X_{t-1}`。对于随机游走，一阶差分后即为白噪声。
3.  **去季节性**：如果序列有周期性（如月度数据），可以进行季节性差分，公式为 `X_t - X_{t-s}`，其中 `s` 为周期长度。

![](img/518076225d4ccd8ff04e70acee92896d_39.png)

## 时间序列建模流程

![](img/518076225d4ccd8ff04e70acee92896d_41.png)

![](img/518076225d4ccd8ff04e70acee92896d_43.png)

综合以上知识，一个完整的时间序列建模流程如下：

![](img/518076225d4ccd8ff04e70acee92896d_45.png)

![](img/518076225d4ccd8ff04e70acee92896d_47.png)

![](img/518076225d4ccd8ff04e70acee92896d_49.png)

1.  **可视化与平稳化**：绘制序列图，检查趋势和季节性。通过差分、去趋势等方法获得平稳序列。
2.  **模型识别与定阶**：绘制平稳序列的ACF和PACF图，初步判断ARMA模型的阶数 `(p, q)`。
3.  **参数估计**：使用最大似然估计等方法拟合ARMA(p, q)模型。
4.  **模型诊断**：检验模型的残差是否为白噪声（无自相关性）。如果不是，则需要调整模型阶数。
5.  **预测与应用**：使用拟合好的模型进行预测，并将结果转换回原始序列的尺度。

![](img/518076225d4ccd8ff04e70acee92896d_51.png)

![](img/518076225d4ccd8ff04e70acee92896d_52.png)

![](img/518076225d4ccd8ff04e70acee92896d_54.png)

在模型选择时，为了避免过拟合，我们使用AIC准则，其公式为：
`AIC = -2 * log(Likelihood) + 2 * (p + q + 1)`
选择AIC值最小的模型。

![](img/518076225d4ccd8ff04e70acee92896d_56.png)

## 异方差模型

对于金融时间序列，我们经常观察到“波动率聚集”现象——大幅波动往往集中出现。这意味着残差的方差并非常数，而是随时间变化的，即存在“异方差性”。标准的ARMA模型无法刻画这一点。

### ARCH/GARCH模型

为了对时变波动率建模，我们引入ARCH和GARCH模型。

*   **ARCH模型**：条件方差仅依赖于过去残差的平方。
    ARCH(1)模型设定为：
    `σ_t² = ω + α * ε_{t-1}²`
    `r_t = σ_t * z_t`，其中 `z_t` 是标准正态分布。
*   **GARCH模型**：条件方差依赖于过去残差的平方和过去的条件方差本身，具有更长的记忆性。
    GARCH(1,1)模型设定为：
    `σ_t² = ω + α * ε_{t-1}² + β * σ_{t-1}²`

![](img/518076225d4ccd8ff04e70acee92896d_58.png)

我们可以先对收益率序列建立ARMA模型，再对其残差建立GARCH模型，从而同时刻画序列的条件均值（收益）和条件方差（风险）。
