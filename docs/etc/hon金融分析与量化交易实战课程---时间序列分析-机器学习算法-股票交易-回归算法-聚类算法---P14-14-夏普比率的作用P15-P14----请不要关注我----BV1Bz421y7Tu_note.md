## 夏普比率的定义与意义
上一节我们提到了投资中的风险概念，本节中我们来看看如何量化地衡量风险与收益的关系。

夏普比率描述的是：在承担单位风险的情况下，所能获得的超额收益是多少。这个指标越高，意味着投资组合在承担相同风险时，能获得更高的回报，或者说，为了获得相同回报，所需承担的风险更低。

为了便于理解，可以想象一个生活中的例子：一份在战乱地区日薪极高的工作，其高薪对应的是极高的生命风险。夏普比率就是用来评估这种“风险溢价”是否值得的量化工具。在选股或选择投资组合时，在其他条件相似的情况下，我们倾向于选择夏普比率更高的选项。

## 夏普比率的计算公式
理解了夏普比率的意义后，我们来看看如何计算它。其核心思想是比较投资组合的收益与无风险收益的差异，并将这个差异与投资组合的波动性（风险）相关联。

![](img/b8dd1f4abf8488dd088195c5df127f30_4.png)

公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率
*   **Rf**：无风险收益率（例如国债收益率、银行定期存款利率）
*   **σp**：投资组合收益率的标准差（代表波动性，即风险）

**公式解读**：
1.  `(Rp - Rf)` 计算的是投资组合超越无风险收益的超额收益。
2.  用超额收益除以组合的风险 `σp`，就得到了**单位风险所对应的超额收益**。

## 计算示例与代码实现
接下来，我们通过一个简化的Python示例，演示如何计算多只股票的夏普比率。假设我们已经有了股票的历史日收益率数据。

以下是计算夏普比率的关键步骤：

1.  准备数据：获取投资组合的历史收益率序列。
2.  设定参数：确定无风险收益率（本例按年化5%计算）和年化交易日数（通常为250天）。
3.  应用公式：计算年化夏普比率。

```python
import numpy as np
import pandas as pd

# 假设 `returns` 是一个Pandas DataFrame，每一列代表一只股票的历史日收益率
# 例如：returns = pd.DataFrame({‘stock_A’: […], ‘stock_B’: […], ‘stock_C’: […]})

![](img/b8dd1f4abf8488dd088195c5df127f30_6.png)

# 步骤1: 处理数据缺失值（用前一天的收益率填充）
returns_filled = returns.fillna(method=‘ffill’)

![](img/b8dd1f4abf8488dd088195c5df127f30_8.png)

# 步骤2: 设定参数
risk_free_rate = 0.05  # 年化无风险收益率，设为5%
trading_days_per_year = 250  # 年化交易日数

# 步骤3: 计算夏普比率
# 计算每只股票的平均日收益率
mean_daily_return = returns_filled.mean()
# 计算每只股票日收益率的标准差
std_daily_return = returns_filled.std()

# 计算年化夏普比率
# 先计算超额收益（日度）
excess_return = mean_daily_return - (risk_free_rate / trading_days_per_year)
# 年化夏普比率 = (平均日超额收益 * 年交易日数) / (日标准差 * 年交易日数的平方根)
sharpe_ratio = (excess_return * trading_days_per_year) / (std_daily_return * np.sqrt(trading_days_per_year))

print(“各股票的夏普比率：”)
print(sharpe_ratio)
print(“\n夏普比率最高的股票是：”, sharpe_ratio.idxmax(), “，比率为：”, sharpe_ratio.max())
```

**代码说明**：
*   我们首先填充了可能存在的缺失数据，确保计算连续性。
*   关键计算在于将日度数据年化。夏普比率通常以年化形式呈现以便比较。
*   最终，`sharpe_ratio` 序列给出了每只股票的夏普比率，数值越大代表风险调整后收益越好。
