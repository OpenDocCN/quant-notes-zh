import pandas as pd

## 均线概念与计算
数据准备完成后，我们进入核心部分：计算移动平均线。首先，我们来理解什么是均线。

在股票分析中，对于每一个交易日，都可以计算出前N天收盘价的移动平均值。将这些移动平均值连接起来形成的线，就称为N日移动平均线，简称均线。常用的均线有5日、10日、30日、60日等。

均线的计算公式如下：
`MA = (C1 + C2 + C3 + ... + CN) / N`
其中，`MA`代表移动平均值，`C1`到`CN`代表连续N天的收盘价。

为了更直观地理解，假设我们有连续8天的收盘价。计算5日均线时，我们首先计算第一个5天（第1天到第5天）的收盘价均值，得到点X1；然后计算第二个5天（第2天到第6天）的均值，得到点X2；以此类推。最后，将X1、X2、X3等点连接起来，就形成了5日均线。

理解了概念后，我们开始计算贵州茅台股票的5日均线和30日均线。计算均线可以使用pandas的`rolling`方法。

![](img/c5364c7f64e4432c9e0e08ec10e95483_9.png)

以下是计算步骤：

首先，从处理好的数据中提取收盘价序列。
```python
# 提取收盘价序列
close_series = df['close']
```

接着，使用`rolling`函数结合`mean`方法计算移动平均值。
```python
# 计算5日移动平均线（MA5）
ma5 = close_series.rolling(5).mean()

# 计算30日移动平均线（MA30）
ma30 = close_series.rolling(30).mean()
```
在计算结果中，`ma5`序列的前4个值为`NaN`（非数字），因为前4天不足以构成一个5日周期。同样，`ma30`序列的前29个值也为`NaN`。从第5天和第30天开始，我们得到了有效的移动平均值。

## 均线可视化
计算出均线数据后，我们可以将其可视化，以便观察走势。这里我们使用matplotlib库进行简单的绘图。

```python
import matplotlib.pyplot as plt
%matplotlib inline

# 绘制5日均线
plt.plot(ma5[50:80], label='MA5')

# 绘制30日均线
plt.plot(ma30[50:80], label='MA30')

# 添加图例
plt.legend()
plt.show()
```
通过图表，我们可以清晰地看到5日均线和30日均线的走势以及它们的交叉情况，这是双均线策略中判断买卖信号的关键。
