首先，我们需要对模型输出的结果数据进行处理。当前的数据包含了特征列（如前一天的涨跌）和结果列（当天的实际涨跌），但它们被组合在一起。我们需要将它们区分开来，以便进行分组统计。

![](img/bc0194a7cf769150e1b40ae577127b02_14.png)

![](img/bc0194a7cf769150e1b40ae577127b02_16.png)

以下是数据处理步骤：

![](img/bc0194a7cf769150e1b40ae577127b02_18.png)

![](img/bc0194a7cf769150e1b40ae577127b02_20.png)

1.  使用 `groupby` 方法，根据特征列（例如“前一天”和“前两天”的涨跌情况）对数据进行分组。
2.  对分组后的结果列（即“当天涨跌”）进行计数统计，观察在不同特征组合下，各种结果出现的频率。

![](img/bc0194a7cf769150e1b40ae577127b02_22.png)

为了更清晰地展示所有可能的组合（包括计数为0的情况），我们使用 `unstack` 和 `fillna` 方法对结果进行重塑和填充。

![](img/bc0194a7cf769150e1b40ae577127b02_24.png)

![](img/bc0194a7cf769150e1b40ae577127b02_26.png)

```python
# 假设 df 是包含特征和结果的数据框
# ‘feature1‘, ‘feature2‘ 是特征列名，‘result‘ 是结果列名
grouped_result = df.groupby([‘feature1‘, ‘feature2‘])[‘result‘].size().unstack().fillna(0)
```

![](img/bc0194a7cf769150e1b40ae577127b02_28.png)

执行上述操作后，我们得到一个数据透视表。行索引代表不同的特征组合，列索引代表不同的结果（例如涨或跌），表格中的数值代表该组合出现的次数。这使我们能够一目了然地看到，在特定的历史走势（特征）下，未来涨跌（结果）的分布情况。

![](img/bc0194a7cf769150e1b40ae577127b02_30.png)

![](img/bc0194a7cf769150e1b40ae577127b02_32.png)

基于上述统计表格，我们可以制定一个简单的交易策略。核心思想是：选择在特定特征组合下，出现概率更高的结果作为我们的预测方向。

![](img/bc0194a7cf769150e1b40ae577127b02_34.png)

例如，通过观察统计表，我们可能发现：当“前一天涨”且“前两天涨”（即两个特征值均为1）时，第二天“下跌”的次数多于“上涨”的次数。那么，我们的策略就可以定义为：当出现这种特征组合时，我们预测下跌；否则，预测上涨。

![](img/bc0194a7cf769150e1b40ae577127b02_36.png)

以下是策略制定的代码实现：

![](img/bc0194a7cf769150e1b40ae577127b02_38.png)

```python
import numpy as np

![](img/bc0194a7cf769150e1b40ae577127b02_40.png)

![](img/bc0194a7cf769150e1b40ae577127b02_42.png)

# 假设 data 是我们的原始数据框，包含 ‘feature1‘ 和 ‘feature2‘ 列
# 我们根据统计规律生成预测信号 ‘position‘
position = np.where((data[‘feature1‘] + data[‘feature2‘]) == 2, -1, 1)
```
这段代码的含义是：计算两个特征值的和，如果和等于2（即两个特征都为1，代表连续两天上涨），则预测值为-1（代表看跌）；否则，预测值为1（代表看涨）。

![](img/bc0194a7cf769150e1b40ae577127b02_44.png)

策略制定完成后，我们需要评估其效果。评估方法是：将我们的策略预测结果 `position` 与实际的涨跌情况 `direction` 进行比较，计算预测正确的天数。

![](img/bc0194a7cf769150e1b40ae577127b02_50.png)

![](img/bc0194a7cf769150e1b40ae577127b02_52.png)

以下是效果评估的代码：

![](img/bc0194a7cf769150e1b40ae577127b02_54.png)

```python
# 计算预测正确的次数
correct_predictions = (position == data[‘direction‘]).value_counts()
```
通过比较，我们可能得到类似“预测正确1102天，错误1033天”的结果。这表明基于简单统计的策略，其预测准确率略高于随机猜测（50%）。

![](img/bc0194a7cf769150e1b40ae577127b02_56.png)

![](img/bc0194a7cf769150e1b40ae577127b02_58.png)

![](img/bc0194a7cf769150e1b40ae577127b02_60.png)

然而，预测准确率并非最终目的，在金融市场中，我们更关心策略能否带来正的收益。因此，我们需要进行简单的回测，计算如果按照此策略进行交易，最终的累计收益率是多少。

我们将策略信号 `position` 与股票的实际日收益率 `returns` 相乘，得到策略每日的收益率，然后计算其累计收益。

![](img/bc0194a7cf769150e1b40ae577127b02_62.png)

![](img/bc0194a7cf769150e1b40ae577127b02_64.png)

```python
# 计算策略的累计收益
strategy_returns = data[‘returns‘] * position
cumulative_strategy_returns = (1 + strategy_returns).cumprod()
```
将基于统计策略的累计收益与单纯买入持有的收益进行对比。结果可能显示，虽然策略预测准确率不高，但通过规避某些大概率下跌的情况，其最终亏损可能小于单纯持有。
