## 工具包简介与获取

上一节我们介绍了量化分析的基本流程，本节中我们来看看如何利用现成工具提升效率。Alphalens是一个专为因子分析设计的Python工具包。

![](img/2c7d7e07acefec2339ba170c5ebc769d_5.png)

以下是关于Alphalens的基本信息：
*   **核心功能**：自动化完成因子分析中的计算与可视化任务。
*   **官方资源**：其代码托管于GitHub，并有详细的使用文档。
*   **安装方法**：在本地环境可通过 `pip install alphalens` 命令进行安装。

![](img/2c7d7e07acefec2339ba170c5ebc769d_7.png)

![](img/2c7d7e07acefec2339ba170c5ebc769d_9.png)

不过，在本课程后续的实践中，我们将直接在提供的在线平台中编写代码，该平台已预装好所有必要的工具包，方便我们直接获取数据并进行操作。

![](img/2c7d7e07acefec2339ba170c5ebc769d_11.png)

## 平台代码编写环境

![](img/2c7d7e07acefec2339ba170c5ebc769d_13.png)

![](img/2c7d7e07acefec2339ba170c5ebc769d_15.png)

接下来，我们切换至实际编写代码的环境。在所使用的量化平台中，我们需要进入“投资研究”模块，而非“回测”模块。

以下是进入并创建分析环境的步骤：
1.  在平台左侧导航栏找到并点击“投资研究”。
2.  界面会打开一个类似Jupyter Notebook的在线编程环境。
3.  点击“新建”按钮，创建一个新的Python 3笔记文件。
4.  可以将该文件重命名为“因子分析”或其他描述性名称。

![](img/2c7d7e07acefec2339ba170c5ebc769d_17.png)

后续所有的分析代码都将在该平台的这个笔记文件中编写和执行。平台服务器已经配置好了所需的数据接口和工具包（包括Alphalens），这比在本地操作更为便捷。

![](img/2c7d7e07acefec2339ba170c5ebc769d_19.png)
