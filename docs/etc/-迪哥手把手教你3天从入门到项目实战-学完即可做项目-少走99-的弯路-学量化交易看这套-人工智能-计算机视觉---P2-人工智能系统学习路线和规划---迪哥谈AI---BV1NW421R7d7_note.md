## 概述

学习人工智能有多种方法，但把握重点是关键。本节将阐述学习的优先级，并强调应将**源码学习**作为核心重点。算法和论文可以先行梳理，但最终要回归到源码的深入理解上。以下将按照基础、应用和重点领域的顺序，详细说明学习内容，特别是为刚入门、对后续方向感到迷茫的同学提供清晰的指引。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_22.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_23.png)

*   **Python与数学：点到为止**
    *   学习Python和数学知识时，应遵循“点到为止”的原则。不需要进行系统性、深入的学习，只需掌握基本概念和用法即可。后续在实际应用中边用边查，效率更高。
    *   学习方法提示：学习初期热情最高，不应将热情消耗在最基础的理论学习上，而应快速进入实践环节。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_24.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_26.png)

*   **机器学习：目标导向学习**
    *   学习重点取决于你的后续目标。
    *   **公式**: 如果你的目标是数据挖掘或数据分析，则需要系统、详细地学习机器学习。这包括掌握十大经典算法的原理、公式推导（例如，线性回归的损失函数 `L = Σ(y_i - ŷ_i)²`）、代码实践和实验分析。
    *   **代码**: 如果你的目标是计算机视觉，那么机器学习主要起铺垫作用，用于熟悉基础概念（如损失函数、梯度下降），为深度学习做准备。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_27.png)

*   **数据挖掘：结合项目实践**
    *   无论为了就业、论文还是项目包装，都建议了解数据挖掘。它代表了一个完整项目从数据到结果的流程。
    *   在AI领域，图像、文本、结构化数据都会涉及。学习数据挖掘时，应结合具体案例和项目，有目的地练习Python代码，而非孤立地系统学习Python。

*   **夯实算法基础**
    *   必须详细掌握卷积层的参数（如卷积核大小、步长）、池化层的参数与优势，以及它们的代码实现细节。同样，Transformer的每一步代码实现也需要透彻理解。
    *   打好这些基本功，后续学习任何新知识都会事半功倍。这就像拥有“九阳神功”护体，学什么都快。熟练掌握深度学习基本算法和**PyTorch框架**是快速进步的关键。

    *   学习过程不能只看理论，必须结合代码、项目和框架。课程选用**PyTorch**框架，因为它是当前最主流、核心的工具。
    *   **代码**: `import torch; model = torch.nn.Linear(10, 2)`。无需担心学不会，工具重在上手使用，会越用越熟练。

*   **掌握必要工具：OpenCV**
    *   对于计算机视觉（CV）方向，OpenCV是重要的图像处理工具。同样采取“快速熟悉”的策略，了解其基本模块和功能，后续使用时随时查阅即可。

## 综合项目与领域拓展

掌握了核心基础后，我们需要通过综合项目来融会贯通，并拓展到各个前沿领域。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_61.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_63.png)

在项目学习阶段，一定要广泛涉猎。不能只局限于自己当前的任务。例如，做目标检测也需要了解其他模块的技术，因为很多模块是通用的。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_64.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_65.png)

以下是各领域项目与拓展的学习内容：

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_69.png)

*   **目标检测**
    *   需要系统学习YOLO全系列（v1-v8）等经典算法。
    *   但目标检测不仅限于YOLO，还应广泛了解DETR、Deformable DETR、Focal Loss、半监督检测等主流算法。尽可能掌握多种算法的源码级细节，能加深对该领域的理解。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_71.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_72.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_73.png)

*   **图像分割**
    *   分割是CV领域与检测同等重要的核心方向。需要学习分割领域的经典与前沿算法，掌握其核心模块。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_74.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_75.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_78.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_79.png)

*   **项目模板应用**
    *   做项目和论文要“站在巨人的肩膀上”。课程会提供涵盖分类、分割、检测、OCR、生成等各类任务的完整项目模板。
    *   你需要做的主要工作是在模板上进行“增删改查”的调整，以适应自己的具体任务。商汤的MM系列等优秀框架会被详细解读。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_80.png)

*   **视觉前沿与拓展**
    *   **基础任务拓展**：行为识别、姿态估计、目标跟踪等经典视觉任务的源码实现。
    *   **Transformer**：Transformer在各类下游任务（如CV、NLP）中的应用与实现，这是当前必备的热点技术。
    *   **图神经网络**：学习如何处理具有关联关系的数据，掌握图神经网络（GNN）的应用。
    *   **自动驾驶相关**：点云处理、深度估计、车道线检测、三维重建等与无人驾驶相关的技术。
    *   **前沿热点**：对比学习、多模态项目的原理与实践。
    *   **其他经典方向**：缺陷检测、重识别（ReID）、生成对抗网络（GAN）、图像修复、超分辨率等。
    *   **强化学习与生成模型**：强化学习（RL）原理，以及如何结合RL打造生成模型（如GPT系列），了解其架构与源码。
    *   **专题应用**：医学影像（检测、分割、知识图谱、CT识别）等垂直领域专题。
    *   **大模型**：NLP大模型（如GPT）、视觉大模型、自监督任务等最新技术。
    *   **模型部署**：学习如何将训练好的模型部署到服务器或嵌入式设备（如开发板），结合物联网应用。

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_82.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_83.png)

![](img/8ac64133b59c1e4e5634a10cb5d5bb8e_84.png)

*   **其他AI领域**
    *   **自然语言处理**：文本数据处理、任务建模基础，以及Hugging Face等工具库的使用，和NLP下游任务模板。
    *   **时间序列**：时间序列预测专题，学习如何结合图网络或Transformer处理时序数据。
    *   **知识图谱**：知识图谱的构建与应用。
    *   **语音与推荐系统**：语音识别、推荐系统等相关任务。
