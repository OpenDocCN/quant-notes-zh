## 调整回测周期进行验证

![](img/83afa3232269eead421adeca939246b9_24.png)

![](img/83afa3232269eead421adeca939246b9_26.png)

为了更全面地评估策略，我们需要在不同的市场周期中进行测试。短期回测（如几个月）容易受到市场偶然波动的影响，通常不具备代表性。因此，我们接下来将回测周期延长至2年或更长时间。

![](img/83afa3232269eead421adeca939246b9_28.png)

![](img/83afa3232269eead421adeca939246b9_30.png)

![](img/83afa3232269eead421adeca939246b9_32.png)

以下是调整回测时间段的尝试与观察：

![](img/83afa3232269eead421adeca939246b9_34.png)

*   **2016年1月4日至2018年1月4日**：在这个为期两年的周期里，策略收益与基准收益相差不大，没有表现出明显的优势。
*   **2018年1月4日至2020年1月4日**：策略收益（约2%）略高于基准收益（约0%），但优势依然不够显著。这可能与2018年股市整体处于熊市有关，市场环境对所有策略都构成了挑战。
*   **2016年1月4日至2017年1月4日**：在这个为期一年的周期内，策略表现较好，回测收益明显高于基准收益，验证了策略在特定市场阶段的有效性。

![](img/83afa3232269eead421adeca939246b9_36.png)

![](img/83afa3232269eead421adeca939246b9_38.png)

## 探索策略选股逻辑的边界

![](img/83afa3232269eead421adeca939246b9_40.png)

![](img/83afa3232269eead421adeca939246b9_42.png)

我们的策略核心是选取综合打分排名靠前的股票。一个自然的疑问是：如果我们反其道而行之，选择排名靠后的股票，结果会怎样？

![](img/83afa3232269eead421adeca939246b9_44.png)

![](img/83afa3232269eead421adeca939246b9_46.png)

我们修改了选股逻辑，从选取排名**前10**的股票改为选取排名**最后10**（即指标最差）的股票，并在2016-2017年这个周期内进行回测。

![](img/83afa3232269eead421adeca939246b9_48.png)

![](img/83afa3232269eead421adeca939246b9_50.png)

结果符合预期：基准收益已经是负值（约-10%），而选择“最差”股票组合的策略，其亏损幅度比基准还要大。这个对比实验有力地证明了我们选取的财务指标（如市盈率PE、市净率PB等）的有效性——指标好的股票组合整体表现优于市场，而指标差的股票组合表现则弱于市场。

![](img/83afa3232269eead421adeca939246b9_52.png)

![](img/83afa3232269eead421adeca939246b9_54.png)

## 策略总结与展望

![](img/83afa3232269eead421adeca939246b9_72.png)

![](img/83afa3232269eead421adeca939246b9_74.png)

本节课中我们一起学习了如何对一个量化策略进行多角度的回测分析与总结。

![](img/83afa3232269eead421adeca939246b9_76.png)

我们的**打分策略**逻辑清晰直接：通过计算一系列财务指标（如 `PE`, `PB`）的综合得分，并每月买入排名靠前的股票。回测表明，该策略在多数情况下能够跑赢市场基准，尤其是在非极端熊市的环境中。同时，通过反向测试（买入排名靠后的股票）进一步验证了所选指标的有效性。

![](img/83afa3232269eead421adeca939246b9_78.png)

![](img/83afa3232269eead421adeca939246b9_80.png)

然而，策略表现受市场环境影响很大。在整体熊市（如2018年）中，策略可能仅能取得与市场持平或略好的收益，这是所有市场参与者都需要面对的系统性风险。打分法是量化投资中常用且直观的方法，能够帮助投资者快速构建一个基础策略。大家可以在提供的代码基础上，尝试调整指标权重、选股数量或调仓频率，以进一步优化策略，并理解不同参数对策略表现的影响。
