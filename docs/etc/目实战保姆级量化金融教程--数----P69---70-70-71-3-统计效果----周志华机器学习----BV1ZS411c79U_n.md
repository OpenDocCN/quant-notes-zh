## 数据变换与统计准备

![](img/0352c402569c4ddd06e79611050c77b7_6.png)

上一节我们介绍了如何创建特征列。本节中，我们来看看如何对数据进行变换，以便进行统计分析。

![](img/0352c402569c4ddd06e79611050c77b7_8.png)

![](img/0352c402569c4ddd06e79611050c77b7_10.png)

现在，我们拥有包含特征（如前一天的涨跌`lag1`、前两天的涨跌`lag2`）和结果（当天涨跌`direction`）的数据。为了统计不同特征组合下结果的分布，我们需要对数据进行聚合。

![](img/0352c402569c4ddd06e79611050c77b7_12.png)

以下是具体操作步骤：

![](img/0352c402569c4ddd06e79611050c77b7_14.png)

1.  首先，我们按特征列`lag1`和`lag2`进行分组，并统计每组中结果`direction`的数量。
2.  然后，使用`unstack`方法将结果重塑为一个更易读的交叉表格式，缺失值用0填充。

![](img/0352c402569c4ddd06e79611050c77b7_16.png)

![](img/0352c402569c4ddd06e79611050c77b7_18.png)

```python
# 按特征分组并统计结果数量，然后重塑数据
result = data.groupby(['lag1', 'lag2', 'direction'])['direction'].count().unstack(fill_value=0)
print(result)
```

![](img/0352c402569c4ddd06e79611050c77b7_20.png)

执行上述代码后，我们将得到一个表格。表格的行索引是`lag1`和`lag2`的特征组合（例如`(-1, -1)`表示前两天都下跌），列索引是结果`direction`（-1或1），表格中的数值代表该特征组合下出现对应结果的次数。这清晰地展示了特征与结果之间的关联。

![](img/0352c402569c4ddd06e79611050c77b7_22.png)

## 制定并应用统计策略

基于上一步得到的统计表格，我们可以观察哪种特征组合下，上涨或下跌的可能性更大。

![](img/0352c402569c4ddd06e79611050c77b7_24.png)

以下是策略制定的逻辑：

![](img/0352c402569c4ddd06e79611050c77b7_26.png)

1.  观察统计表，我们发现当`lag1`和`lag2`都为1（即前两天连续上涨）时，第三天`direction`为-1（下跌）的次数多于为1（上涨）的次数。
2.  因此，我们制定一个简单策略：如果`lag1`和`lag2`的和等于2，则预测第三天会下跌（标记为-1）；否则，预测第三天会上涨（标记为1）。

```python
# 根据特征组合预测走势
data['pred_stat'] = np.where((data['lag1'] + data['lag2']) == 2, -1, 1)
```

![](img/0352c402569c4ddd06e79611050c77b7_28.png)

这样，我们就得到了一个基于历史统计的预测列`pred_stat`。

![](img/0352c402569c4ddd06e79611050c77b7_30.png)

## 评估策略效果

![](img/0352c402569c4ddd06e79611050c77b7_36.png)

![](img/0352c402569c4ddd06e79611050c77b7_38.png)

策略制定后，我们需要评估它的效果。我们将从预测准确率和模拟收益两个角度进行分析。

![](img/0352c402569c4ddd06e79611050c77b7_40.png)

### 预测准确率评估

![](img/0352c402569c4ddd06e79611050c77b7_42.png)

我们将策略预测的结果`pred_stat`与实际结果`direction`进行比较，计算预测正确的天数。

![](img/0352c402569c4ddd06e79611050c77b7_44.png)

```python
# 计算预测正确的次数
correct_predictions = (data['pred_stat'] == data['direction']).value_counts()
print(correct_predictions)
```

![](img/0352c402569c4ddd06e79611050c77b7_46.png)

![](img/0352c402569c4ddd06e79611050c77b7_48.png)

输出结果可能显示，例如有1102次预测正确，1033次预测错误。这表明基于统计的策略比随机猜测略好，但优势并不明显。

### 模拟收益对比

![](img/0352c402569c4ddd06e79611050c77b7_50.png)

![](img/0352c402569c4ddd06e79611050c77b7_52.png)

最后，我们模拟两种投资方式的收益：
*   **跟随大盘**：每天均按照实际涨跌`direction`进行买卖。
*   **跟随策略**：每天按照我们的统计预测`pred_stat`进行买卖。

![](img/0352c402569c4ddd06e79611050c77b7_54.png)

我们计算两种方式的累计收益并进行比较。

![](img/0352c402569c4ddd06e79611050c77b7_56.png)

```python
# 计算两种策略的累计收益
returns_market = (data['returns'] * data['direction']).cumsum()
returns_strategy = (data['returns'] * data['pred_stat']).cumsum()

![](img/0352c402569c4ddd06e79611050c77b7_58.png)

# 绘制收益曲线对比图
import matplotlib.pyplot as plt
plt.plot(returns_market, label='Market Returns')
plt.plot(returns_strategy, label='Strategy Returns')
plt.legend()
plt.show()

![](img/0352c402569c4ddd06e79611050c77b7_60.png)

![](img/0352c402569c4ddd06e79611050c77b7_62.png)

# 输出最终收益
print("大盘最终收益:", returns_market.iloc[-1])
print("策略最终收益:", returns_strategy.iloc[-1])
```

![](img/0352c402569c4ddd06e79611050c77b7_64.png)

![](img/0352c402569c4ddd06e79611050c77b7_66.png)

结果显示，虽然两种方式最终可能都亏损，但基于统计的策略`returns_strategy`的亏损幅度`0.xx`小于单纯跟随大盘的亏损幅度。这说明该简单统计策略起到了一定的风险控制作用。
