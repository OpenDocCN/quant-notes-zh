## 概述：因子分析结果解读

上一节我们介绍了如何实现一个自定义因子，本节中我们来看看如何评估这个因子的表现。因子分析是量化策略开发中至关重要的一步，它帮助我们判断一个因子是否具有预测能力，以及其稳定性和实用性。

## 结果展示函数：`create_full_tear_sheet`

在聚宽量化平台中，用于展示因子分析结果的函数是 `create_full_tear_sheet`。理解其核心参数是正确解读分析结果的前提。

以下是该函数的关键参数及其含义：

*   **`demeaned`**：是否使用超额收益进行计算。`True` 表示使用，`False` 表示不使用。
*   **`group_adjust`**：是否对收益进行行业中性化处理。`True` 表示进行行业调整，`False` 表示不调整。
*   **`by_group`**：是否按行业分组展示结果。`True` 表示按行业展示，`False` 表示不按行业展示。
*   **`turnover_periods`**：调仓周期，通常设置为 `[1， 5， 10]`，代表持仓1天、5天、10天后进行调仓。
*   **`avgretplot`**：一个元组 `(5， 15)`，表示因子预测的天数，即计算过去5天和未来15天的收益。
*   **`std_bar`**：是否在图表中显示标准差。`True` 表示显示，`False` 表示不显示。

## 因子分析报告的核心内容

聚宽平台的因子分析报告主要包含三个部分：收益分析、IC分析和换手率分析。

### 收益分析

收益分析主要评估因子本身的盈利能力。需要注意的是，量化平台中因子的收益计算通常指股票在调仓周期内的涨跌幅比例，而非具体的价格差额。公式可以简化为：
`收益 = (当期收盘价 - 上期收盘价) / 上期收盘价`

收益分析从以下三个方面进行观察：

1.  **分位数平均收益**：展示在持仓1天、5天、10天后，各个因子分位数组合所能获得的平均收益。
2.  **分位数累计收益**：展示在上述不同持仓周期下，各分位数组合的累计收益情况。
3.  **多空组合收益**：指做多因子值最大的分位数组（如第5分位），同时做空因子值最小的分位数组（如第1分位），所形成的投资组合的收益。

### IC分析

![](img/e0b2acc3377130a1919416b8c6471611_5.png)

IC（Information Coefficient，信息系数）分析用于衡量因子的预测能力。IC值表示因子值与未来收益之间的相关性，取值范围在-1到1之间。**IC值越大，表示因子的预测能力越强**。

IC分析主要有两种方法：

*   **Normal IC**：计算因子载荷（即因子值）与未来收益之间的相关系数。
*   **Rank IC**：计算因子值的排序序数与未来收益的排序序数之间的相关系数。聚宽平台默认使用Rank IC。

![](img/e0b2acc3377130a1919416b8c6471611_7.png)

由于单日IC值波动较大，报告中通常会提供月度IC的移动均线作为参考，使趋势更平滑。

### 换手率分析

换手率分析用于观察在不同时间周期下，因子各分位组内个股的进出（买卖）情况。常见的观察周期为1天、5天和10天。

**换手率计算方法举例**：假设某因子第1分位组持有30只股票，一天后有1只股票被调出（卖出），那么该分位组单日换手率为 `1/30 ≈ 3.3%`。

![](img/e0b2acc3377130a1919416b8c6471611_9.png)

![](img/e0b2acc3377130a1919416b8c6471611_11.png)

换手率分析主要有两个价值：

1.  **因子稳定性**：换手率低的因子，其因子值在时间序列上持续性更好，表明选股结果稳定，收益不依赖于频繁交易。
2.  **交易成本考量**：高换手率意味着需要更频繁的交易来维持投资组合，从而产生更高的手续费和冲击成本。在因子收益相近的情况下，**低换手率的因子因其更低的交易成本，最终净收益往往更高**。

## 代码实战：分析移动平均因子

![](img/e0b2acc3377130a1919416b8c6471611_13.png)

下面我们通过一个简单的例子，使用5日移动平均（MA5）因子进行实战分析。

![](img/e0b2acc3377130a1919416b8c6471611_15.png)

```python
# 导入必要的库
import warnings
warnings.filterwarnings(‘ignore’) # 忽略警告信息
from jqdata import *
from jqfactor import *

![](img/e0b2acc3377130a1919416b8c6471611_17.png)

# 1. 定义自定义因子：5日移动平均线
class MA5(Factor):
    name = ‘ma5‘
    max_window = 5
    dependencies = [‘close‘]

    def calc(self， data):
        # 计算最近5日的收盘价均值
        return data[‘close‘].iloc[-5:].mean()

# 2. 进行单因子分析
# 创建因子实例
factor = MA5()

# 设置分析参数
q = 8 # 分为8个组
mask = get_index_stocks(‘000300.XSHG‘) # 股票池：沪深300指数成分股
# 进行因子分析并生成完整报告
create_full_tear_sheet(factor=factor，
                         start_date=‘2020-01-01‘，
                         end_date=‘2021-12-31‘，
                         mask=mask，
                         quantiles=q，
                         groupby=None， # 不按行业分组
                         demeaned=False，
                         group_adjust=False，
                         by_group=False，
                         turnover_periods=[1， 5， 10]，
                         avgretplot=(5， 15)，
                         std_bar=False)
```

运行以上代码后，系统会生成一份完整的因子分析报告。报告将依次展示：

1.  **因子描述性统计**：如因子值的最大值、最小值、均值、标准差等。
2.  **收益分析图表**：包括各分位数平均收益、多空组合收益、累计收益等。
3.  **IC分析图表**：展示不同周期（1日、5日、10日）的IC值序列、IC分布图以及月度IC均值。
4.  **换手率分析图表**：展示1日、5日、10日调仓周期下，各分位组的换手率情况。

通过观察这些图表，我们可以对MA5因子的表现进行评估。例如，如果多数分位数的平均收益为负，或IC值长期接近0或为负，则说明该因子在当前市场环境下可能无效。同时，观察换手率可以评估其实用性和交易成本。
