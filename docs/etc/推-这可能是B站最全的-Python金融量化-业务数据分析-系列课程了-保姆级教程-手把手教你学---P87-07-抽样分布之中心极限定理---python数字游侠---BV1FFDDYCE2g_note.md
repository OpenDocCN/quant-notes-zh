## 统计推断概述

![](img/d6cadc5c05045610fcd55655fef509a0_1.png)

上一节我们介绍了描述性统计，本节中我们来看看如何从样本推断总体。在日常生活中，我们经常根据已知信息对未知情况进行判断。例如，通过一个人的衣着和言谈举止判断其身份，或通过脸色判断其心情和身体状况。这些过程都包含推断，其结果可能对也可能错。

![](img/d6cadc5c05045610fcd55655fef509a0_3.png)

在统计学中，推断同样存在，但它需要依赖数据来得出推测性的结论。**统计推断**，就是通过样本数据对总体数据进行推测的过程。我们可以对样本计算均值、中位数、标准差等指标，并以此来推测总体的相应指标。目前，应用最广泛、最成熟的是对**均值**的推断。

![](img/d6cadc5c05045610fcd55655fef509a0_5.png)

## 抽样与抽样误差

进行统计推断的前提是抽样。在抽样过程中，我们会遇到**抽样误差**。它是指由于随机抽样，导致**样本均值**与**总体均值**之间存在的差异。

![](img/d6cadc5c05045610fcd55655fef509a0_7.png)

**公式**：`抽样误差 = 样本均值 - 总体均值`

例如，假设总体的真实均值是16，而我们抽取样本计算出的样本均值是15，那么抽样误差就是 -1。

然而，在现实中，我们通常**不知道总体的真实均值**。如果我们知道，就不需要抽样了。因此，我们无法直接计算这个误差。为了解决这个问题，我们需要进行多次抽样实验，分析这个误差是否存在某种分布规律。如果存在，我们就可以根据这个规律来估算误差的大小。

## 样本均值的分布规律

设想我们对同一个总体进行无数次样本量为n的随机抽样，每次抽样都会计算出一个样本均值。这无数个样本均值本身也会形成一个分布，即**样本均值的抽样分布**。

中心极限定理揭示了这一分布的关键规律：

**中心极限定理**：设从均值为μ、方差为σ²的任意总体中，抽取样本量为n的样本。当n充分大时（通常认为n > 30），样本均值的抽样分布近似服从**正态分布**。

这个正态分布的参数是：
*   **均值**：等于总体均值 μ。
*   **方差**：等于总体方差 σ² 除以样本量 n，即 σ²/n。
*   **标准差**：这个分布的标准差有一个专门名称，叫做**标准误**，其公式为 σ/√n。

**公式**：`样本均值抽样分布 ~ N(μ, σ²/n)`
**标准误公式**：`标准误 = σ / √n`

标准误衡量的是不同样本均值之间的离散程度，它本质上就是**抽样误差大小的度量**。离散程度越大，说明抽样误差可能越大；离散程度越小，说明样本均值越集中在总体均值附近，抽样误差越小。

## 影响抽样误差的因素

以下是影响标准误（即抽样误差大小）的两个主要因素：

1.  **总体内部的变异程度（σ）**：总体中各个个体之间的差异越大（σ越大），抽样误差通常也越大。因为数据本身就很分散，抽到极端值的可能性增加，导致样本均值波动更大。
2.  **样本量的大小（n）**：样本量越大（n越大），抽样误差越小。因为更大的样本能更好地代表总体，样本均值会更稳定地接近总体均值。从公式 `σ/√n` 也可以直观看出，n增大，分母变大，标准误减小。

在实际应用中，我们通常不知道总体标准差σ，因此会用样本标准差S来估计它。此时，标准误的估计值为 `S / √n`。

## 从正态分布到t分布

根据中心极限定理，我们知道 `(样本均值 - 总体均值) / 标准误` 的分布形态。当使用真实的总体标准差σ计算标准误时，这个统计量服从标准正态分布。

**公式**：`Z = (x̄ - μ) / (σ/√n) ~ N(0, 1)`

然而，现实中σ未知，我们只能用样本标准差S来替代。这时，上述统计量就不再严格服从标准正态分布，而是服从**t分布**。

**公式**：`t = (x̄ - μ) / (S/√n) ~ t(n-1)`

t分布的形状与正态分布类似，但尾部更厚，其具体形态取决于自由度（这里为n-1）。当样本量很大时，t分布非常接近正态分布。

## 总结

![](img/d6cadc5c05045610fcd55655fef509a0_9.png)

本节课中我们一起学习了中心极限定理及其意义。我们了解到，无论总体是什么分布，只要样本量足够大，样本均值的分布就会接近正态分布。这个定理为我们通过样本推断总体（尤其是总体均值）提供了理论依据。同时，我们认识了衡量抽样误差大小的指标——标准误，并了解了影响它的两个因素。最后，我们引出了当总体标准差未知时，用于推断的重要分布——t分布，为后续学习假设检验和置信区间奠定了坚实的基础。
