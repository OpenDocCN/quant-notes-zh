## 简单增长率累加的问题

首先，我们思考一个问题：能否将一段时间内每天的增长率简单相加，来得到这段时间的总增长率？

以下是一个例子来说明这个问题：
*   假设一只股票第一天价格是100元。
*   第二天价格跌至50元。
*   第三天价格涨至75元。

![](img/0fbebddeab41634639c2235e4068bee4_6.png)

计算每日增长率：
*   第二天增长率 = `(50 - 100) / 100 = -0.5`
*   第三天增长率 = `(75 - 50) / 50 = 0.5`

如果将这两个增长率相加：`(-0.5) + 0.5 = 0`。结果为零，似乎表示股票既没涨也没跌。

但观察实际价格：从第一天的100元到第三天的75元，股票实际上是下跌的。这说明，**简单的增长率累加并不能反映资产价值的真实累积变化**。

## 引入对数计算连续变化

![](img/0fbebddeab41634639c2235e4068bee4_8.png)

为了解决上述问题，在分析连续时间序列的累积变化时，我们需要借助数学中的对数。

核心思想是：计算每日的对数收益率，然后进行累加。对数收益率的加法等价于价格比率的乘法，这能正确反映从期初到期末的总变化。

![](img/0fbebddeab41634639c2235e4068bee4_10.png)

计算公式如下：
`总对数收益率 = log(第N天价格 / 第一天价格) = log(第二天价格/第一天价格) + log(第三天价格/第二天价格) + ... + log(第N天价格/第N-1天价格)`

以前面的例子计算：
`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`
这个结果直接反映了第一天（100元）和第三天（75元）之间的总差异。

形象地说，这相当于计算：最初投入的1元钱，在经过一系列涨跌后，最终变成了多少钱。

![](img/0fbebddeab41634639c2235e4068bee4_12.png)

## 在Python中实现计算

![](img/0fbebddeab41634639c2235e4068bee4_14.png)

以下是实现连续变化分析的步骤和代码。

![](img/0fbebddeab41634639c2235e4068bee4_16.png)

### 步骤1：计算每日的对数收益率

![](img/0fbebddeab41634639c2235e4068bee4_18.png)

首先，我们需要计算后一天价格除以前一天价格，然后取对数。这需要获取“前一天”的数据。

![](img/0fbebddeab41634639c2235e4068bee4_20.png)

![](img/0fbebddeab41634639c2235e4068bee4_22.png)

我们可以使用Pandas的`.shift()`函数将数据整体向上移动一格，从而得到前一天的数据。

```python
import pandas as pd
import numpy as np

![](img/0fbebddeab41634639c2235e4068bee4_24.png)

# 假设 `data` 是一个包含股价的Pandas Series
# 计算每日的对数收益率
log_return = np.log(data / data.shift(1))
```

### 步骤2：计算累积对数收益率

得到每日的对数收益率后，我们需要对其进行累加，以得到从起始日到每一天的累积变化。这可以使用NumPy的`.cumsum()`函数（累积求和）来实现。

![](img/0fbebddeab41634639c2235e4068bee4_26.png)

```python
# 计算累积对数收益率
cumulative_log_return = log_return.cumsum()
```

![](img/0fbebddeab41634639c2235e4068bee4_28.png)

### 步骤3：还原为实际价值比例

累积对数收益率表示的是“倍数”的对数。为了得到直观的“初始1元相当于当前多少元”，我们需要通过指数函数将其还原。

```python
# 将累积对数收益率还原为实际的价值比例（初始1元的当前价值）
cumulative_value_ratio = np.exp(cumulative_log_return)
```

![](img/0fbebddeab41634639c2235e4068bee4_30.png)

![](img/0fbebddeab41634639c2235e4068bee4_32.png)

### 步骤4：可视化结果

最后，我们可以将计算结果绘制成图，直观地展示每只股票初始投资的价值增长轨迹。

![](img/0fbebddeab41634639c2235e4068bee4_34.png)

```python
import matplotlib.pyplot as plt

cumulative_value_ratio.plot(figsize=(16, 8))
plt.title('累积价值变化 (以初始为1单位)')
plt.ylabel('价值比例')
plt.xlabel('日期')
plt.show()
```

![](img/0fbebddeab41634639c2235e4068bee4_36.png)

![](img/0fbebddeab41634639c2235e4068bee4_38.png)

执行以上代码后，我们将得到一张图表。图中每条曲线从1开始，展示了如果在该股票最初投入1个单位资金，到后续每个时间点，这笔资金将变成多少。例如，亚马逊(AMZN)的曲线最高，说明其从期初到期末的累积增长倍数最大。
