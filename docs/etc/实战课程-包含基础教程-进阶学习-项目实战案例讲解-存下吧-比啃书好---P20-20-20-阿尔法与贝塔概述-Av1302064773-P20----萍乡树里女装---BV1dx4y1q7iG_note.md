# 量化交易教程：P20：阿尔法与贝塔概述 📊

在本节课中，我们将要学习量化策略评估中的两个核心概念：阿尔法（α）和贝塔（β）。理解这两个指标对于区分策略收益的来源至关重要。

## 收益的构成

上一节我们介绍了多种策略评估指标，本节中我们来看看收益的深层构成。投资所获得的收益可以分解为两个部分。

一部分收益与整体市场环境相关。当市场大环境向好时，投资者通常更容易赚钱。这部分收益被称为市场收益。

另一部分收益则与市场波动关联不大，主要源于投资者自身的努力。例如，通过深入研究公司财务、运营状况等信息，凭借独到的眼光和策略操作来获利。这部分收益被称为超额收益。

## 阿尔法（α）与贝塔（β）的定义

以下是阿尔法与贝塔的核心定义，它们分别用于衡量上述两部分收益。

*   **贝塔（β）**：衡量策略收益与市场收益的关联程度，反映了策略对大盘波动的敏感性。它代表**系统性风险**。公式上，它常出现在以下回归模型中：
    `策略收益 = α + β * 市场收益 + 误差项`
*   **阿尔法（α）**：衡量策略获得的、与市场波动无关的超额收益。它代表**非系统性风险**，通常用于评估策略本身或投资经理的选股与择时能力。

图中展示了三种收益：
*   **策略收益**：执行某个策略获得的总收益。
*   **基准收益**（市场收益）：代表市场大环境走势的收益，例如沪深300指数的收益。

在这个框架下，**总收益（策略收益）由市场收益和超额收益两部分构成**。市场收益部分由贝塔（β）系数刻画，超额收益部分由阿尔法（α）系数刻画。

如上所述，通过建立策略收益与市场收益的回归方程，可以求解出阿尔法（α）和贝塔（β）的值。具体的计算方法将在后续讲解因子分析时详细介绍。

对于投资者而言，理解这两者的侧重点不同：
*   市场收益（贝塔）受宏观环境影响，个人难以改变或控制。
*   超额收益（阿尔法）则直接体现了策略的有效性和投资能力。

因此，在量化策略开发中，**我们的核心目标往往是追求稳定且显著的阿尔法（α）**，即创造超越市场基准的超额收益。

## 其他相关指标

在策略评估中，除了阿尔法和贝塔，还会用到其他指标。以下是部分常见风险与收益指标，初学者了解其含义即可，无需死记公式。

*   **最大回撤**：衡量策略历史上从峰值到谷底的最大亏损幅度。
*   **夏普比率**：衡量每承受一单位总风险，能获得多少超额回报。
*   **索提诺比率**：类似夏普比率，但只考虑下行风险。
*   **信息比率**：衡量超额收益的稳定性，即单位主动风险带来的超额收益。
*   **基准收益**：作为比较基准的市场或指数收益。即使什么都不做（完全复制基准），也可能获得正收益或负收益。
