## 双均线策略概述

![](img/f3e852823fad42f5b62e7f9a28724c45_6.png)

上一节我们介绍了时间序列分析的基础，本节中我们来看看如何将其应用于具体的交易策略。双均线策略是技术分析中常用的方法，其核心思想是通过比较两条不同周期的移动平均线来识别市场趋势的潜在变化。

![](img/f3e852823fad42f5b62e7f9a28724c45_8.png)

## 什么是双均线

![](img/f3e852823fad42f5b62e7f9a28724c45_10.png)

在股票数据中，我们每天都能获得一个收盘价。如果只看单个价格点的浮动，很难把握整体的价格变化趋势。为了更清晰地描述趋势，我们引入移动平均线的概念。

![](img/f3e852823fad42f5b62e7f9a28724c45_12.png)

![](img/f3e852823fad42f5b62e7f9a28724c45_14.png)

双均线，顾名思义，指的是两条移动平均线。一条是短期均线，另一条是长期均线。

![](img/f3e852823fad42f5b62e7f9a28724c45_16.png)

## 短期均线与长期均线

![](img/f3e852823fad42f5b62e7f9a28724c45_18.png)

![](img/f3e852823fad42f5b62e7f9a28724c45_20.png)

以下是短期均线的计算方式：

短期均线通常选取较短的周期，例如5个交易日（代表一周）。计算方法是取最近5个交易日的收盘价，计算其算术平均值。随着时间窗口的滑动，我们可以得到一系列的平均值，将这些点连接起来，就形成了短期均线。

其计算公式可以表示为：
`短期均线值 = (P1 + P2 + P3 + P4 + P5) / 5`
其中 P1 到 P5 代表最近5个交易日的收盘价。

![](img/f3e852823fad42f5b62e7f9a28724c45_22.png)

![](img/f3e852823fad42f5b62e7f9a28724c45_24.png)

长期均线的原理相同，但选取的周期更长，例如20个交易日（约一个月）或60个交易日（约一个季度）。长、短期均线的唯一区别在于计算时所选取的时间窗口大小。

![](img/f3e852823fad42f5b62e7f9a28724c45_26.png)

## 金叉与死叉：买卖信号

绘制出短期和长期两条均线后，它们之间的相对位置和交叉关系就构成了交易信号。

![](img/f3e852823fad42f5b62e7f9a28724c45_28.png)

以下是两种关键的交叉形态：

![](img/f3e852823fad42f5b62e7f9a28724c45_30.png)

1.  **黄金交叉（金叉）**：当短期均线从下方向上穿越长期均线时，形成金叉。这通常被视为**买入信号**。它表明近期价格上涨动能开始超越长期趋势，市场可能进入上升通道。
2.  **死亡交叉（死叉）**：当短期均线从上方向下穿越长期均线时，形成死叉。这通常被视为**卖出信号**。它表明近期价格下跌压力开始显现，市场可能转为下跌趋势。

![](img/f3e852823fad42f5b62e7f9a28724c45_32.png)

![](img/f3e852823fad42f5b62e7f9a28724c45_34.png)

## 策略逻辑与应用

![](img/f3e852823fad42f5b62e7f9a28724c45_36.png)

![](img/f3e852823fad42f5b62e7f9a28724c45_38.png)

理解了金叉和死叉后，我们就可以设计一个简单的交易策略：在金叉出现时买入股票，在死叉出现时卖出股票。理论上，如果能够准确识别这些信号并执行交易，就可以实现“低买高卖”，从而获取收益。

![](img/f3e852823fad42f5b62e7f9a28724c45_40.png)

当然，实际市场更为复杂，双均线策略会产生一定的滞后性，并且存在假信号。在后续的实践中，我们将使用Python来具体实现这个策略，并观察其效果。

![](img/f3e852823fad42f5b62e7f9a28724c45_42.png)
