## 为什么需要时间序列索引？

![](img/446a5c32b398a50571424e4064099ab1_10.png)

首先需要明确一点，为什么在预处理时需要将`date`列转换为时间序列，并再次将其设置为行索引。

![](img/446a5c32b398a50571424e4064099ab1_12.png)

这是因为后续的分析需求都会用到日期。将原始数据中的日期字符串转换为时间序列后，我们就可以利用时间序列特有的属性和方法，更高效地完成后续的分析任务。因此，请先跟随步骤进行操作。

## 需求分析：找出上涨超过3%的日期

![](img/446a5c32b398a50571424e4064099ab1_14.png)

接下来，我们需要找出目标股票（例如茅台）所有收盘价比开盘价上涨超过3%的日期。

这个分析的目的是总结和观察该股票涨幅的分布情况。例如，如果发现上涨日期集中在3月或6月，那么未来在这些月份购买该股票，获利的可能性或许会更大。

## 实现思路与伪代码

![](img/446a5c32b398a50571424e4064099ab1_16.png)

以下是实现该需求的核心逻辑。我们用伪代码来描述：

![](img/446a5c32b398a50571424e4064099ab1_18.png)

**伪代码逻辑**：
`(收盘价 - 开盘价) / 开盘价 > 0.03`

![](img/446a5c32b398a50571424e4064099ab1_20.png)

这个公式表示收盘价比开盘价上涨了3%。我们的目标是找出所有满足这个条件的日期。

![](img/446a5c32b398a50571424e4064099ab1_22.png)

## 将伪代码转换为实际代码

![](img/446a5c32b398a50571424e4064099ab1_24.png)

在Pandas中，开盘价和收盘价分别是DataFrame中的`open`和`close`列。我们可以直接对这两列进行向量化运算。

![](img/446a5c32b398a50571424e4064099ab1_26.png)

**核心代码公式**：
```python
( df[‘close’] - df[‘open’] ) / df[‘open’] > 0.03
```

这个表达式会对整个数据列进行计算，并返回一个布尔值Series。其中，`True`表示该行数据（即该日期）满足上涨3%的条件，`False`则表示不满足。

![](img/446a5c32b398a50571424e4064099ab1_28.png)

![](img/446a5c32b398a50571424e4064099ab1_30.png)

## 处理布尔值：一个重要的Pandas技巧

![](img/446a5c32b398a50571424e4064099ab1_32.png)

当我们在数据分析中得到一组布尔值时，下一步通常是将这组布尔值作为原始数据的行索引进行筛选。

![](img/446a5c32b398a50571424e4064099ab1_34.png)

![](img/446a5c32b398a50571424e4064099ab1_36.png)

这是因为在Pandas中，如果将布尔值Series放入DataFrame的`.loc`索引器中，就可以直接取出所有`True`对应的行数据，同时自动忽略`False`对应的行。

**经验总结**：
如果产生了布尔值，则下一步马上将布尔值作为原数据的行索引，以取出目标数据。

![](img/446a5c32b398a50571424e4064099ab1_38.png)

## 获取最终结果：满足条件的日期

![](img/446a5c32b398a50571424e4064099ab1_40.png)

通过布尔索引筛选出满足条件的行数据后，我们需要的并不是整行数据，而是这些行的索引，即日期。

![](img/446a5c32b398a50571424e4064099ab1_42.png)

![](img/446a5c32b398a50571424e4064099ab1_44.png)

由于我们在预处理时已将日期设置为行索引，因此直接获取筛选结果的`.index`属性，即可得到所有收盘价比开盘价上涨超过3%的日期。

**完整代码示例**：
```python
# 计算布尔条件
condition = (df[‘close’] - df[‘open’]) / df[‘open’] > 0.03
# 使用布尔索引筛选行
filtered_df = df.loc[condition]
# 获取筛选结果的索引，即目标日期
result_dates = filtered_df.index
```
