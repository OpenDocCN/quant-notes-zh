## 数据准备 📊

上一节我们介绍了如何获取股票数据。本节中，我们来看看如何读取并处理这些数据，为计算均线做准备。

首先，我们需要读取之前保存的股票历史行情数据。这里我们以“贵州茅台”的数据为例，数据已保存在名为“茅台.csv”的本地文件中。

以下是读取和处理数据的步骤：

```python
import pandas as pd

# 1. 读取CSV文件数据
df = pd.read_csv('茅台.csv')

# 2. 删除无用的列（例如默认的索引列‘Unnamed: 0’）
df = df.drop(labels='Unnamed: 0', axis=1)

# 3. 将‘date’列转换为时间序列类型
df['date'] = pd.to_datetime(df['date'])

df.set_index('date', inplace=True)
```

## 理解均线概念 📉

在计算均线之前，我们需要理解什么是均线。均线，全称移动平均线，是技术分析中用于识别趋势的常用工具。

对于每一个交易日，都可以计算出前N天收盘价的移动平均值。将这些每日的移动平均值连接起来，形成的曲线就是N日移动平均线，简称均线。常用的均线有5日、10日（短期均线）、30日、60日（中期均线）等。

![](img/df4304747766dcbe935db6d50c6c6e21_7.png)

其计算公式如下：
**MA = (C1 + C2 + C3 + ... + CN) / N**
其中：
- **MA** 代表移动平均值。
- **C1, C2, ..., CN** 分别代表第1天、第2天到第N天的收盘价。
- **N** 代表计算均线的周期（例如5或30）。

简单来说，5日均线就是连续计算每日“前5天收盘价的平均值”，并将这些平均值连成线。

## 计算双均线 🔢

理解了均线的概念和公式后，本节我们来看看如何用Python计算股票的5日均线和30日均线。

我们将使用Pandas库中的`rolling`函数。这个函数可以创建一个“滑动窗口”，让我们方便地计算指定窗口大小（如5天）内的统计值（如平均值）。

![](img/df4304747766dcbe935db6d50c6c6e21_9.png)

以下是计算5日和30日均线的代码：

```python
# 计算5日移动平均线（MA5）
ma5 = df['close'].rolling(5).mean()
# 计算30日移动平均线（MA30）
ma30 = df['close'].rolling(30).mean()

# 查看计算结果
print(“5日均线前10个值：”)
print(ma5.head(10))
print(“\n30日均线前10个值：”)
print(ma30.head(10))
```

**代码解释**：
- `df[‘close’]` 选取了收盘价序列。
- `.rolling(5)` 创建了一个宽度为5的滑动窗口。
- `.mean()` 对这个窗口内的数据计算平均值。
- 对于`ma5`，前4天的值为`NaN`（非数字），因为至少需要5个数据点才能计算第一个5日均值。第5个值才是第一个有效的5日移动平均值。`ma30`同理，前29个值为`NaN`。

## 可视化均线 📈

计算出均线数据后，我们可以将其可视化，直观地观察股票价格趋势以及不同周期均线之间的关系。

虽然Matplotlib绘图模块的详细用法不在本节重点，但我们可以先使用它来快速绘制均线图。

```python
import matplotlib.pyplot as plt
# 确保图表在Jupyter Notebook中内嵌显示
%matplotlib inline

# 绘制收盘价曲线（可选，用于对比）
plt.plot(df[‘close’][50:80], label=‘Close Price’)
# 绘制5日均线
plt.plot(ma5[50:80], label=‘MA5’)
# 绘制30日均线
plt.plot(ma30[50:80], label=‘MA30’)

plt.title(‘Stock Price with Moving Averages’)
plt.legend() # 显示图例
plt.show()
```

运行代码后，我们将看到三条曲线：收盘价线、5日均线和30日均线。均线相比股价曲线更为平滑。短期均线（如MA5）对价格变化更敏感，波动较大；长期均线（如MA30）更为平缓，代表长期趋势。两条均线的交叉点（金叉或死叉）是双均线交易策略的重要信号，这将在后续课程中深入探讨。
