在本节课中，我们将继续学习优化问题的基本概念，特别是如何将实际问题（如最短路径问题）建模为优化问题，并介绍优化问题的几种重要分类方法。我们还将简要回顾优化理论的发展历史。

## 最短路径问题的优化建模

上一节我们介绍了优化问题的一般形式。本节中我们来看看如何将一个具体问题——最短路径问题——表述为一个优化问题。

假设我们有一个无向图。图中的每个圆圈表示一个顶点，两个顶点之间的连线是一条边。每条边都有一个权重。例如，一条边的权重是2，另一条是3。这个权重可以表示从一个顶点到另一个顶点所需的时间（例如2毫秒记为2，3毫秒记为3），实质上代表了从一个顶点移动到另一个顶点的代价。

![](img/efa9591d360a37fde2838ea43821acb7_1.png)

最短路径问题是指：给定一个起点（源点）和一个终点，我们希望找到从起点到终点的一条路径，使得路径上所有边的权重之和最小。这是一个经典问题，有许多算法可以解决，如迪杰斯特拉算法、贝尔曼-福特算法等。本课程的重点不是这些算法，而是展示如何将此类问题写成一个优化问题。

给定一个无向图，其顶点集合为 **V**，边的集合为 **E**。**V** 和 **E** 共同定义了这个无向图。再给定一个源点 **s** 和一个目标点 **t**。我们希望找到最短路径。

设 **w_ij** 表示边 **(i, j)** 的权重。我们引入决策变量 **x_ij**。如果边 **(i, j)** 被选入路径，则 **x_ij = 1**；否则 **x_ij = 0**。目标是使所选边的总权重最小。

因此，目标函数可以写为：
`最小化 ∑_{(i,j)∈E} w_ij * x_ij`

然而，仅此目标无法保证我们选择的边构成一条从 **s** 到 **t** 的有效路径。我们需要添加约束来确保路径的连续性。

以下是关于路径的约束条件：
*   对于源点 **s**：从 **s** 出发的边数比进入 **s** 的边数多1。即：`∑_{j} x_sj - ∑_{j} x_js = 1`
*   对于目标点 **t**：进入 **t** 的边数比从 **t** 出发的边数多1。即：`∑_{j} x_tj - ∑_{j} x_jt = -1` （或表述为 `∑_{j} x_jt - ∑_{j} x_tj = 1`）
*   对于所有其他顶点 **v**（既非 **s** 也非 **t**）：进入 **v** 的边数等于从 **v** 出发的边数。即：`∑_{j} x_vj - ∑_{j} x_jv = 0`

此外，我们还有决策变量的约束：`x_ij ∈ {0, 1}`。

这个完整的数学模型（包含目标函数和上述所有约束）就是最短路径问题的优化描述。这个问题本质上是一个需要从众多选择中挑选出最优组合的问题，通常比较困难。但有趣的是，对于这个特定问题，我们可以将难处理的约束 `x_ij ∈ {0, 1}` 松弛为 `x_ij ≥ 0`。松弛后，目标函数和所有约束都是关于 **x_ij** 的线性表达式，因此问题变成了一个**线性规划**问题，而线性规划是相对容易求解的。至于为什么这种松弛对最短路径问题是有效的，这是一个经典的结论，感兴趣的读者可以深入研究。

## 优化问题的分类

我们刚刚看到了一个可以转化为线性规划的例子。现在，让我们系统地了解优化问题的几种主要分类方式。

### 线性规划 vs. 非线性规划

这种分类基于问题中函数的性质。

考虑优化问题的一般形式，包含目标函数 **f_0(x)** 和约束函数 **f_1(x), ..., f_m(x)**。如果所有这些函数都是线性函数，那么该问题称为**线性规划**问题。

线性函数的定义是：对于函数 **f_i** 和任意向量 **x, y** 及标量 **α, β**，满足 **f_i(αx + βy) = αf_i(x) + βf_i(y)**。

例如，我们松弛后的最短路径问题就是一个线性规划。

![](img/efa9591d360a37fde2838ea43821acb7_3.png)

![](img/efa9591d360a37fde2838ea43821acb7_5.png)

线性规划问题有一个很好的几何解释：其可行解集（由线性约束构成）是一个**凸多面体**，目标函数是线性的。其最优解总是在这个凸多面体的某个顶点（或边上）取得。著名的**单纯形法**就是利用这一性质来求解线性规划。

如果问题中至少有一个函数（无论是目标函数还是约束函数）是非线性的，那么该问题称为**非线性规划**问题。

### 凸规划 vs. 非凸规划

![](img/efa9591d360a37fde2838ea43821acb7_7.png)

这是本课程的核心分类，它更本质地区分了问题的难度。

![](img/efa9591d360a37fde2838ea43821acb7_8.png)

![](img/efa9591d360a37fde2838ea43821acb7_9.png)

如果一个优化问题满足以下两个条件，则它是一个**凸规划**问题：
1.  目标函数 **f_0(x)** 是凸函数。
2.  可行解集是一个凸集。

![](img/efa9591d360a37fde2838ea43821acb7_11.png)

凸函数的定义（初步印象）是：函数图像上任意两点之间的线段都位于图像上方。用公式表示，对于定义域内任意 **x, y** 和任意 **θ ∈ [0, 1]**，有：
`f(θx + (1-θ)y) ≤ θf(x) + (1-θ)f(y)`

凸集的定义是：集合中任意两点之间的连线上的所有点仍然属于该集合。

![](img/efa9591d360a37fde2838ea43821acb7_13.png)

**凸规划问题是相对容易求解的优化问题。** 相反，**非凸规划问题通常是难以求解的。**

任何线性规划问题都是凸规划的特例（因为线性函数既是凸函数也是凹函数，且线性约束构成的集合是凸集）。在几十年前，人们常认为“线性=容易，非线性=困难”。但现在我们认识到，更准确的分类是“凸=相对容易，非凸=困难”。

### 光滑优化 vs. 非光滑优化

这种分类主要针对目标函数 **f_0(x)**。

如果目标函数在其定义域内处处可微（导数连续），则称为**光滑优化**问题。如果目标函数在某些点不可微（例如有“尖角”），则称为**非光滑优化**问题。例如，绝对值函数 `f(x) = |x|` 在 `x=0` 处不可微。

光滑问题通常比非光滑问题稍容易处理一些，但这种难度差异不像凸与非凸那样本质。

### 连续优化 vs. 离散优化

这种分类基于可行解集的性质。

如果可行解集是连续区域（如实数域上的一个区间或区域），则是**连续优化**问题。如果可行解集是离散的点（例如整数集合、图的节点集合），则是**离散优化**问题，也称为组合优化。

![](img/efa9591d360a37fde2838ea43821acb7_15.png)

离散优化问题通常非常困难，因为它们往往是非凸的。连续优化问题则不一定，它可能是凸的（容易），也可能是非凸的（困难）。

### 单目标优化 vs. 多目标优化

到目前为止，我们讨论的都是**单目标优化**，即只有一个目标函数 **f_0(x)** 需要最小化。

但在现实中，许多问题本质上是**多目标优化**，即需要同时最小化多个目标函数：`F(x) = (f_1(x), f_2(x), ..., f_k(x))`。例如，调控房价可能需要同时考虑经济增长、社会稳定等多个目标。

在多目标优化中，通常不存在一个解能在所有目标上都达到最优。取而代之的是**帕累托最优**的概念：一个解是帕累托最优的，当且仅当不存在另一个解能在所有目标上都不差于它，且至少在一个目标上严格优于它。所有帕累托最优解构成的集合称为**帕累托前沿**。

一种常见的处理方法是将其转化为单目标问题，例如通过加权求和：`最小化 w_1*f_1(x) + w_2*f_2(x) + ... + w_k*f_k(x)`。但调整权重不一定能获得整个帕累托前沿。本课程将主要关注单目标优化。

## 本课程的目标与内容

我们反复强调，凸优化问题是“容易”的问题。本课程的主要目标之一，就是学习如何将实际问题建模或近似为一个凸优化问题。正如数学家希尔伯特所言：“一个清晰表述的问题，相当于解决了一半。” 对于优化，我们可以说：“将一个实际问题转化为凸优化问题，就完成了大部分工作。”

因此，本课程将围绕以下主线展开：
1.  **建模**：如何根据实际任务构造出容易（凸）的优化问题。
2.  **分析**：凸优化问题具有哪些优良性质？我们如何利用这些性质？
3.  **求解**：简单介绍一些求解凸优化问题的基本算法。

具体内容将包括：
*   凸集、凸函数、凸优化问题的基本定义与性质。
*   凸优化理论（如对偶理论）。
*   凸优化算法简介。

需要强调的是，“容易”是相对的。本课程讨论的“容易”，主要指问题本身的结构（凸性）使得其全局最优解可以被高效找到。我们通常假设目标函数和约束函数的值、乃至其导数都相对容易计算。我们不专门讨论函数值本身极难计算的问题（这类问题有专门的研究领域和算法）。

## 优化理论发展简史

![](img/efa9591d360a37fde2838ea43821acb7_17.png)

![](img/efa9591d360a37fde2838ea43821acb7_19.png)

最后，让我们简要回顾优化这门学科的发展历程，这有助于理解其核心思想与应用背景。

![](img/efa9591d360a37fde2838ea43821acb7_21.png)

*   **早期根源（17-18世纪）**：求解非线性方程 `f(x)=0` 的**牛顿-拉弗森法**，以及求解线性方程组的高斯、赛德尔等人的工作，本质上都隐含着优化的思想（例如，求方程根可转化为最小化 `[f(x)]^2`）。
*   **约束处理的里程碑（18世纪）**：**拉格朗日**引入了**拉格朗日乘子法**，通过将约束条件整合到目标函数中，为处理约束优化问题奠定了基石。这是本课程的核心概念之一。
*   **二战及战后发展（20世纪中期）**：
    *   **动态规划**（贝尔曼，1940s）：解决多阶段决策优化问题，核心思想是“从后往前”优化。
    *   **博弈论**（冯·诺依曼，1944年）：研究多个决策者相互竞争下的优化问题。纳什（1950年）提出了**纳什均衡**概念。
    *   **线性规划的提出与求解**：坎托罗维奇（1939年）从苏联生产计划问题中抽象出线性规划模型。丹齐格（1947年）提出了高效且实用的**单纯形法**。
*   **现代算法突破（20世纪中后期）**：
    *   人们发现单纯形法不是**多项式时间算法**（即最坏情况下求解时间可能随问题规模指数增长）。
    *   哈奇扬（1979年）提出了第一个求解线性规划的多项式时间算法（椭球法），但实际效率不高。
    *   卡马卡（1984年）提出了**内点法**，它既是多项式时间算法，在实际应用中对于大规模问题也往往非常高效。此后，内点法被推广到求解非线性凸优化问题，成为现代优化算法的主流之一。
*   **当代应用浪潮（2000年至今）**：随着计算能力的提升，优化方法被广泛应用于通信、机器学习、金融、数据科学等众多领域。如何将优化理论与各领域的具体问题结合，成为研究和应用的热点。这也正是本课程希望引导大家进入的方向。

## 总结

本节课中，我们一起学习了：
1.  如何将**最短路径问题**建模为一个带有0-1整数约束的优化问题，并了解到通过松弛可以将其转化为更容易求解的**线性规划**问题。
2.  优化问题的几种关键**分类**：线性/非线性、凸/非凸、光滑/非光滑、连续/离散、单目标/多目标。其中，**凸与非凸的区分是最本质的难度分水岭**。
3.  本课程的**目标**是学习如何建模、分析和求解凸优化问题。
4.  优化理论的**发展简史**，从牛顿、拉格朗日到线性规划、内点法，展示了这门学科从解决数学基础问题到处理实际复杂应用的演进过程。
