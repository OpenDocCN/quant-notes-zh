## 回顾：抛硬币的贝叶斯估计问题

![](img/3590216e21883106597263cdeb398b78_1.png)

上一节我们介绍了贝叶斯推断的核心思想。本节中，我们来看一个具体的例子：基于抛硬币实验对硬币的公平性参数 `θ` 进行贝叶斯估计。

![](img/3590216e21883106597263cdeb398b78_3.png)

![](img/3590216e21883106597263cdeb398b78_5.png)

我们的实验设定如下：
*   进行了 `N = 50` 次抛掷。
*   观测到正面（Head）的次数 `z = 10`。
*   我们假设参数 `θ`（硬币正面朝上的概率）的先验分布是贝塔分布（Beta Distribution）。
*   先验分布的参数设为 `α = 12`, `β = 12`。根据贝塔分布的性质，其均值为 `0.5`，方差约为 `0.1`，这代表我们初始认为硬币是基本公平的。

![](img/3590216e21883106597263cdeb398b78_7.png)

根据贝叶斯定理和共轭先验的性质，在得到观测数据后，`θ` 的后验分布也是一个贝塔分布，其参数更新为：
*   `α_post = α + z = 12 + 10 = 22`
*   `β_post = β + (N - z) = 12 + (50 - 10) = 52`

![](img/3590216e21883106597263cdeb398b78_9.png)

接下来，我们将使用马尔可夫链蒙特卡洛方法进行模拟，以验证这个理论结果。

![](img/3590216e21883106597263cdeb398b78_11.png)

## Python实践：使用MCMC验证后验分布

![](img/3590216e21883106597263cdeb398b78_13.png)

![](img/3590216e21883106597263cdeb398b78_15.png)

我们将使用 `PyMC3` 库来实现MCMC采样，模拟后验分布。

![](img/3590216e21883106597263cdeb398b78_17.png)

以下是实现此过程所需的Python库：

![](img/3590216e21883106597263cdeb398b78_19.png)

```python
import matplotlib.pyplot as plt
import numpy as np
import pymc3 as pm
import scipy.stats as stats
```

![](img/3590216e21883106597263cdeb398b78_21.png)

![](img/3590216e21883106597263cdeb398b78_23.png)

核心参数设置与MCMC建模步骤如下：

```python
# 1. 定义观测数据
n = 50  # 总实验次数
z = 10  # 观测到的正面次数

# 2. 定义先验参数
alpha_prior = 12
beta_prior = 12

# 3. 理论后验参数（用于对比）
alpha_post_theory = alpha_prior + z
beta_post_theory = beta_prior + (n - z)

![](img/3590216e21883106597263cdeb398b78_25.png)

![](img/3590216e21883106597263cdeb398b78_27.png)

# 4. 使用PyMC3构建模型
with pm.Model() as coin_model:
    # 定义先验分布：θ ~ Beta(12, 12)
    theta = pm.Beta('theta', alpha=alpha_prior, beta=beta_prior)
    
    # 定义似然函数：观测数据z服从二项分布
    y = pm.Binomial('y', n=n, p=theta, observed=z)
    
    # 5. 进行MCMC采样（使用Metropolis-Hastings算法）
    # 初始点采用最大后验概率估计
    start = pm.find_MAP()
    step = pm.Metropolis()
    trace = pm.sample(100000, step=step, start=start, progressbar=False)
```

运行上述代码后，我们可以绘制采样得到的后验分布，并与理论推导的贝塔分布进行对比。

![](img/3590216e21883106597263cdeb398b78_1.png)

从上图可以清晰地看到：
*   **绿色曲线**代表理论推导的后验分布 `Beta(22, 52)`。
*   **MCMC采样结果**的直方图与理论曲线几乎完全重合。
*   **先验分布** `Beta(12, 12)` 以0.5为中心。
*   在得到“50次实验仅10次正面”的观测数据后，后验分布的中心向左偏移（低于0.5），这符合我们的直觉：数据表明这枚硬币可能不是完全公平的。

这个实验验证了MCMC方法在估计复杂后验分布时的有效性。在大多数实际应用中，后验分布没有解析解，我们必须依赖MCMC这类数值方法进行推断。

## 从基础回归到贝叶斯线性回归

理解了贝叶斯推断的基本流程后，我们来看一个在机器学习中至关重要的应用：**贝叶斯线性回归**。

传统的（频率学派）线性回归通过最小化残差平方和来拟合数据。这种方法，如同最大似然估计，完全依赖于观测数据，缺乏对模型参数的先验约束，容易导致**过拟合**（Overfitting）。为了解决过拟合，常采用交叉验证（Cross-Validation）等方法，但这会浪费部分数据用于测试，且计算成本较高。

贝叶斯线性回归提供了另一种思路：
1.  **引入先验**：对回归系数 `w` 引入一个先验分布（例如正态分布），表达我们对参数可能取值的初始信念。
2.  **数据利用充分**：它使用全部数据来更新后验信念，无需分割训练集和测试集。
3.  **自动控制复杂度**：通过选择合适的先验，可以自然地避免过拟合，达到我们期望的模型复杂度。
4.  **输出是分布**：其预测结果不是一个单一值，而是一个概率分布，包含了预测的不确定性。

## 贝叶斯线性回归的理论简述

假设我们的线性模型为：
`y = w^T * x + ε`，其中 `ε ~ N(0, σ²)`

在贝叶斯框架下：
*   我们为参数 `w` 假设一个先验分布，例如 `w ~ N(0, α^(-1)I)`。
*   似然函数为 `y | x, w ~ N(w^T * x, σ²)`。

![](img/3590216e21883106597263cdeb398b78_31.png)

利用共轭先验的性质（正态先验与正态似然），后验分布 `p(w | X, y)` 也是一个正态分布。通过推导，最大化后验概率等价于最小化以下损失函数：

`L(w) = (β/2) * Σ(y_i - w^T * x_i)² + (α/2) * w^T * w`

其中：
*   第一项是普通最小二乘的误差项。
*   第二项是参数的L2范数惩罚项。

**这正是岭回归（Ridge Regression）的目标函数！** 因此，在特定先验假设下，贝叶斯线性回归与岭回归有紧密联系。贝叶斯方法的优势在于，它通过迭代更新（Sequential Update）来学习：将前一步得到的后验分布作为新的先验，结合新数据点，不断更新对参数 `w` 的认知，直至后验分布收敛。

## Python实践：贝叶斯线性回归

![](img/3590216e21883106597263cdeb398b78_41.png)

接下来，我们使用 `PyMC3` 实现一个简单的贝叶斯线性回归。

![](img/3590216e21883106597263cdeb398b78_43.png)

![](img/3590216e21883106597263cdeb398b78_45.png)

```python
import pymc3 as pm
import numpy as np
import matplotlib.pyplot as plt

![](img/3590216e21883106597263cdeb398b78_47.png)

# 生成模拟数据
np.random.seed(42)
X = np.linspace(0, 1, 50)
true_slope = 2.0
true_intercept = 1.0
Y = true_intercept + true_slope * X + np.random.normal(0, 0.2, size=X.shape)

# 使用PyMC3构建贝叶斯线性回归模型
with pm.Model() as linear_model:
    # 使用GLM模块快速构建模型
    # 公式表示：Y 由 X 线性解释，噪声服从正态分布
    pm.glm.GLM.from_formula('Y ~ X', data={'X': X, 'Y': Y}, family=pm.glm.families.Normal())
    
    # 进行MCMC采样（使用NUTS算法，效率更高）
    trace = pm.sample(3000, tune=1000, progressbar=False)

![](img/3590216e21883106597263cdeb398b78_49.png)

# 绘制结果
pm.plot_posterior(trace)
plt.show()
```
