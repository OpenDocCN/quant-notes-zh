## 夏普比率的定义与意义

上一节我们介绍了投资回报率，本节中我们来看看如何衡量“性价比”更高的投资。夏普比率描述的是：对于承担的每一单位风险，我们能获得多少超额收益。

我们可以用一个生活中的例子来理解：假设有两份工作，一份是稳定的办公室文员，日薪300元；另一份是去战乱地区执行任务，日薪3万元。高薪的背后对应着极高的风险。夏普比率就是用来量化这种“风险与收益是否匹配”的指标。

**夏普比率越高，意味着在承担相同风险的情况下，获得的超额收益越高，投资效率也就越好。** 在选股或构建投资组合时，我们通常会倾向于选择夏普比率更高的标的。

## 夏普比率的计算公式

![](img/76d81414fd39da8c09762f69a636035e_1.png)

理解了夏普比率的意义后，我们来看看如何计算它。其核心思想是比较投资组合的收益与无风险收益的差异，并将这个差异与投资组合自身的波动性（风险）进行比较。

![](img/76d81414fd39da8c09762f69a636035e_3.png)

公式如下：

![](img/76d81414fd39da8c09762f69a636035e_5.png)

**`夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差`**

*   **投资组合收益率**：你选择的股票或投资组合的回报率。
*   **无风险收益率**：通常指国债等几乎无风险投资的收益率，作为一个基准。
*   **投资组合收益率的标准差**：衡量投资组合收益的波动性，波动越大代表风险越高。

简单来说，这个公式计算的是 **“每承担一单位风险，能换来多少超出无风险收益的额外回报”**。

## 使用Python计算夏普比率

接下来，我们通过一个简化的Python代码示例，演示如何计算多只股票的夏普比率。假设我们已经有了股票的历史日收益率数据。

以下是计算夏普比率的关键步骤：

1.  准备数据：获取股票的历史日收益率序列。通常需要处理缺失值（例如用前一天的收益率填充）。
2.  设定参数：确定年化无风险收益率（例如5%）和一年的交易天数（例如250天）。
3.  进行计算：应用夏普比率公式，注意将日收益率进行年化处理。

```python
import numpy as np
import pandas as pd

# 假设 `returns_df` 是一个DataFrame，每一列代表一只股票的日收益率序列
# 例如：returns_df[‘股票A‘]， returns_df[‘股票B‘]

# 步骤1: 处理缺失值（前向填充）
returns_df_filled = returns_df.fillna(method=‘ffill‘)

# 步骤2: 设定参数
risk_free_rate = 0.05  # 年化无风险收益率，设为5%
trading_days_per_year = 250  # 一年大约250个交易日

# 步骤3: 计算年化夏普比率
def calculate_annual_sharpe_ratio(return_series):
    # 计算平均日超额收益率 (投资组合日收益率 - 无风险日收益率)
    excess_daily_returns = return_series - risk_free_rate / trading_days_per_year

    # 计算平均日超额收益率的均值
    mean_excess_return = excess_daily_returns.mean()

    # 计算日收益率的标准差
    std_daily_return = return_series.std()

    # 计算年化夏普比率
    # 公式: (平均日超额收益率 * 年交易天数) / (日标准差 * sqrt(年交易天数))
    annual_sharpe_ratio = (mean_excess_return * trading_days_per_year) / (std_daily_return * np.sqrt(trading_days_per_year))

    return annual_sharpe_ratio

![](img/76d81414fd39da8c09762f69a636035e_7.png)

# 对每一只股票计算夏普比率
sharpe_ratios = {}
for stock in returns_df_filled.columns:
    sharpe_ratios[stock] = calculate_annual_sharpe_ratio(returns_df_filled[stock])

# 将结果转换为Series以便查看
sharpe_series = pd.Series(sharpe_ratios)
print(“各股票年化夏普比率：“)
print(sharpe_series)

# 找出夏普比率最高的股票
best_stock = sharpe_series.idxmax()
print(f“\n夏普比率最高的股票是：{best_stock}， 比率为：{sharpe_series[best_stock]:.4f}“)
```

```

```

根据计算结果，我们可以清晰地比较不同资产的表现：
