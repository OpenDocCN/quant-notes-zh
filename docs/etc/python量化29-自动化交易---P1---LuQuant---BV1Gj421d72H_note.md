## 策略逻辑说明 💡
上一节我们概述了课程内容，本节中我们来看看策略的具体规则。

该策略基于日线时间框架，入场和离场规则如下：

**入场条件（同时满足）：**
1.  RSI（14）的值低于或等于25。
2.  ADX（14）的值高于20。
3.  价格在200周期指数移动平均线（EMA）之上交易，以确认上升趋势。

**离场条件（满足其一即可）：**
1.  RSI（14）的值上穿超过75。
2.  价格触及动态止损位。止损位设置为最近两根K线最低价减去2%的幅度。

此外，策略限制同一时间只持有一个多头头寸，以避免过度交易。

## 信号生成与可视化 🔍
在计算出所有指标后，下一步是根据策略规则生成交易信号。

我们遍历数据中的每一天，检查是否同时满足三个入场条件。当条件满足时，我们生成一个买入信号（标记为2）。为了直观理解信号的产生位置，我们可以将买入信号点绘制在价格图表上。

```python
# 生成交易信号
for i in range(len(data)):
    if (data.loc[i, 'RSI_14'] <= 25) and \
       (data.loc[i, 'ADX_14'] > 20) and \
       (data.loc[i, 'trend_signal'] == 2): # trend_signal 来自趋势判断函数
        data.loc[i, 'signal'] = 2 # 买入信号
```

通过图表可视化，我们可以观察到买入信号通常出现在价格小幅回调但整体处于上升趋势的区域。

## 策略回测与结果分析 📈
信号生成后，我们需要通过历史回测来评估策略的实际表现。本节我们将使用`backtesting.py`库进行回测。

以下是回测过程中的关键设置与逻辑：

1.  **初始设置**：初始资金1000美元，使用5倍杠杆，暂不考虑佣金和隔夜费用。
2.  **入场逻辑**：当出现买入信号且当前无持仓时，以99%的仓位开立多头头寸。
3.  **离场逻辑**：
    *   当`RSI`上穿75时平仓。
    *   当价格下跌触及动态止损位时平仓。止损价计算公式为：
        `止损价 = min(当前K线最低价， 前一根K线最低价) * (1 - 0.02)`
        **公式**：`stop_loss_price = min(low[i], low[i-1]) * (1 - stop_loss_percent)`
4.  **持仓限制**：同一时间只允许有一笔未平仓交易。

回测结果显示，该策略在十年测试期内取得了显著回报。

```python
# 动态止损计算示例
stop_loss_percent = 0.02 # 2% 止损
current_low = data.loc[i, 'Low']
previous_low = data.loc[i-1, 'Low']
stop_loss_price = min(current_low, previous_low) * (1 - stop_loss_percent)
```

**回测关键结果：**
*   **最终净值**：约3595美元（起始于1000美元）。
*   **总回报率**：约260%。
*   **胜率**：70%。
*   **总交易次数**：102次（约年均10次交易）。

权益曲线在整个回测期内整体呈现持续增长态势，期间虽有回撤，但能快速恢复并创新高。
