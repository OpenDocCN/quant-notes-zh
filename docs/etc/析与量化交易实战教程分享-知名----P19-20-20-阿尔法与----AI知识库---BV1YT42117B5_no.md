# 量化交易完全可自学教程：P19：20.20.阿尔法与贝塔概述

在本节课中，我们将要学习量化交易中两个核心概念：阿尔法（Alpha）和贝塔（Beta）。理解这两个指标对于评估投资策略的收益来源至关重要。

## 收益的构成

上一节我们介绍了策略评估的多种指标，本节中我们来看看收益的构成。投资所获得的收益可以分解为两个部分。

第一部分收益与整体市场环境相关。当市场大环境向好时，大部分投资都能获得收益。这部分收益是跟随市场波动而产生的。

第二部分收益则与市场整体波动无关。它来源于投资者独特的策略、敏锐的观察力或深入的公司基本面分析。例如，通过研究公司财务和运营状况，做出独特的投资决策所带来的收益。

因此，总收益可以看作由“市场收益”和“超额收益”两部分组成。

## 阿尔法（Alpha）与贝塔（Beta）的定义

基于上述收益构成，我们引入阿尔法和贝塔这两个指标来分别衡量这两部分。

**贝塔（Beta）** 衡量的是策略收益与市场收益的关联程度，即策略对大盘波动的敏感性。它反映了“跟随市场”所能获得的收益部分。贝塔值高，意味着策略收益随市场波动大。

**阿尔法（Alpha）** 衡量的是与市场波动无关的超额收益。它代表了策略通过自身努力（如选股、择时）所创造的独特价值。阿尔法是评估策略主动管理能力的关键指标。

在策略收益分析图中：

*   **我的策略收益**：代表策略获得的总收益。
*   **超额收益**：是“我的策略收益”减去“基准策略收益”的部分，即策略创造的独特价值。

因此，**总收益 = 市场收益 + 超额收益**。其中，市场收益部分与贝塔系数挂钩，超额收益部分与阿尔法系数挂钩。

## 阿尔法与贝塔的计算

从数学角度看，阿尔法和贝塔可以通过一个线性回归模型来求解。我们将策略的超额收益与市场的超额收益进行回归分析。

其核心公式可以表示为：
**策略超额收益 = Alpha + Beta * 市场超额收益 + 误差项**

在这个模型中：
*   **Beta** 是回归系数，表示市场收益变动时，策略收益的平均变动幅度。
*   **Alpha** 是截距项，代表市场收益为零时，策略所能获得的平均超额收益。

具体的求解方法（如普通最小二乘法）将在后续讲解因子分析时详细介绍。目前，理解其代表的经济含义更为重要。

## 量化交易的核心关注点

理解阿尔法和贝塔后，我们需要明确量化交易的核心目标。

市场整体走势（贝塔部分）是投资者难以控制或预测的。因此，策略研究的核心聚焦点在于如何获取稳定且可持续的**超额收益（阿尔法）**。

我们的目标是构建能够产生显著正阿尔法的策略。这意味着策略能够持续跑赢市场基准，证明其具备真正的选股或择时能力。

## 总结

本节课中我们一起学习了阿尔法（Alpha）和贝塔（Beta）的概念。

*   总收益可分解为**市场收益（贝塔）** 和**超额收益（阿尔法）**。
*   **贝塔**衡量策略与市场的关联性，代表系统性风险带来的收益。
*   **阿尔法**衡量策略独立于市场的盈利能力，是评估策略主动管理能力的核心指标。
*   量化交易的核心目标是寻找并获取稳定的**阿尔法收益**。
