在本节课中，我们将学习如何使用Python的条件判断语句（`if-elif-else`）来实现一个根据月薪输出不同心态的程序。这个练习将帮助我们巩固多分支流程控制的理解。

## 概述

![](img/d8bd2d5fb91e55218b10125719a10e5e_3.png)

我们将创建一个程序，根据用户输入的月薪数值，输出对应的职场心态描述。这涉及到多个条件的判断，是练习`if-elif-else`语句的绝佳案例。

## 程序需求分析

以下是程序需要实现的需求逻辑：
*   月薪 >= 100,000：输出“公司是我家”。
*   月薪 >= 50,000：输出“996像呼吸一样自然”。
*   月薪 >= 30,000：输出“老板说什么都是对的”。
*   月薪 >= 20,000：输出“老板说啥就是啥，给钱就行”。
*   月薪 >= 10,000：输出“老板说的有点问题，但我不说话”。
*   月薪 < 10,000：输出“老板我是你爹”。

## 代码实现步骤

上一节我们明确了需求，本节中我们来看看如何用代码一步步实现它。

首先，我们需要获取用户输入的工资。使用`input()`函数获取输入，并用`int()`将其转换为整数类型。

```python
salary = int(input("请输入你的月薪："))
```

![](img/d8bd2d5fb91e55218b10125719a10e5e_5.png)

![](img/d8bd2d5fb91e55218b10125719a10e5e_7.png)

接下来，我们使用`if-elif-else`结构进行多条件判断。判断顺序应从高到低。

![](img/d8bd2d5fb91e55218b10125719a10e5e_9.png)

```python
if salary >= 100000:
    print("公司是我家")
elif salary >= 50000:
    print("996像呼吸一样自然")
elif salary >= 30000:
    print("老板说什么都是对的")
elif salary >= 20000:
    print("老板说啥就是啥，给钱就行")
elif salary >= 10000:
    print("老板说的有点问题，但我不说话")
else:
    print("老板我是你爹")
```

## 关键概念解析

在运行了上面的代码后，我们需要理解两个核心的运行机制。

**1. 程序的执行流程**
程序会**从上到下**依次判断每个条件。当发现**第一个**为`True`的条件时，就会执行对应的代码块，然后**立即结束**整个`if-elif-else`结构，其后的条件即使也满足也不会再被判断。

**2. else语句的作用**
最后的`else`语句是**可选的**。它代表了当**所有**`if`和`elif`条件都不满足时，程序会执行的路径。如果不写`else`，那么当所有条件都不满足时，程序将不会执行任何输出。
