## 概述

夏普比率是一个用于衡量投资组合风险调整后收益的指标。它描述了在承担单位风险的情况下，投资者能够获得多少超额回报。简单来说，它帮助我们判断一项投资是否“划算”。

夏普比率描述的是：对于单位风险而言，我们能获得多少收益。指标越高，意味着在承担相同风险的情况下，获得的收益越高。因此，在比较不同投资选择时，我们通常倾向于选择夏普比率更高的选项。

## 夏普比率的计算逻辑

理解了夏普比率的作用后，我们来看看它是如何计算的。

其核心思想是比较投资组合的收益与无风险收益（如国债、银行存款）的差异。计算公式如下：

**夏普比率 = (投资组合的平均收益率 - 无风险收益率) / 投资组合收益率的标准差**

*   **分子（超额收益）**：代表投资组合超越“保本”收益的部分。
*   **分母（标准差）**：代表投资组合收益的波动性，即风险。

这个公式计算的是**单位风险所对应的超额收益**。我们当然希望这个值越大越好。

## 计算示例

![](img/00253baa40a7326f76b7ccb544b3fa80_7.png)

接下来，我们通过一个简化的例子来演示夏普比率的计算过程。

假设我们分析了几只股票，并计算了它们的日收益率。在计算前，我们通常需要处理数据中的缺失值，例如用前一天的收益率来填充当天的缺失值。

![](img/00253baa40a7326f76b7ccb544b3fa80_9.png)

以下是计算夏普比率的关键步骤：

1.  **计算超额收益**：用投资组合的日收益率减去无风险日收益率（例如年化5%的无风险收益，需转换为日度：`0.05 / 250`）。
2.  **计算年化夏普比率**：计算上述超额收益的均值，除以其标准差，最后乘以交易日总数的平方根进行年化。

```python
# 假设 returns 是投资组合的日收益率序列，rf_rate 是无风险日收益率
excess_returns = returns - rf_rate
sharpe_ratio = (excess_returns.mean() / excess_returns.std()) * np.sqrt(250)
```
