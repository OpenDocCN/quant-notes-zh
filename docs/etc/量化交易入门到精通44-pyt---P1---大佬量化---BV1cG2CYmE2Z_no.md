## 定义类与构造函数

上一节我们介绍了面向对象编程的基本思想，本节中我们来看看如何在Python中定义一个类。

![](img/a20289c78ff572f9fac0d83db4b27d14_1.png)

在Python中，使用关键字 `class` 来定义一个类。类就像一个模板，用于创建具有相同属性和方法的对象。`__init__` 是一个特殊的构造函数，在创建类的实例时会自动调用，用于初始化对象的属性。`self` 参数代表类的实例本身。

以下是定义一个“股票公司”类的示例代码：

```python
class StockCompany:
    def __init__(self, name, stock_code, price):
        self.name = name          # 公司名称
        self.stock_code = stock_code # 股票代码
        self.price = price        # 股票价格
```

在这段代码中：
*   `StockCompany` 是类名。
*   `__init__` 是构造函数。
*   `self.name`, `self.stock_code`, `self.price` 是类的属性，用于描述对象的特征。

## 创建对象与访问属性

定义好类之后，下一步就是创建具体的对象，这个过程也称为“实例化”。

创建对象很简单，只需像调用函数一样使用类名，并传入初始化所需的参数。创建后，可以通过 `对象名.属性名` 的方式来访问对象的属性。

以下是创建对象并访问其属性的示例：

```python
# 创建StockCompany类的一个实例（对象）
hs = StockCompany(“华泰证券”, “600000”, 12.5)

# 访问对象的属性
print(hs.name)        # 输出：华泰证券
print(hs.stock_code)  # 输出：600000
print(hs.price)       # 输出：12.5
```

你可以将 `“华泰证券”` 替换为 `“贵州茅台”`，用同一个类模板创建不同的公司对象。

## 定义与调用类的方法

除了属性，类还可以包含方法（即类中定义的函数）。方法是对象可以执行的操作。

在类中定义方法与定义普通函数类似，但第一个参数必须是 `self`，它指向实例对象本身。通过 `self`，方法可以访问和修改对象的属性。

以下是为 `StockCompany` 类添加方法的示例：

```python
class StockCompany:
    def __init__(self, name, stock_code, price):
        self.name = name
        self.stock_code = stock_code
        self.price = price

    # 定义一个类的方法：获取股票信息
    def get_stock_info(self):
        info = f“股票名称：{self.name}，代码：{self.stock_code}，价格：{self.price}”
        return info

    # 定义一个带参数的方法：计算加法（示例）
    def add(self, x, y):
        return x + y

# 创建对象
hs = StockCompany(“华泰证券”, “600000”, 12.5)

# 调用无参数的方法
stock_info = hs.get_stock_info()
print(f“获取到的股票信息为：{stock_info}”)
# 输出：获取到的股票信息为：股票名称：华泰证券，代码：600000，价格：12.5

# 调用带参数的方法
result = hs.add(10, 20)
print(f“10 + 20 的结果为：{result}”)
# 输出：10 + 20 的结果为：30
```

## 面向对象的核心概念

我们学习了类的定义、对象的创建以及方法的调用。面向对象编程还有三个更核心的要素：**封装、继承和多态**。

*   **封装**：将数据（属性）和操作数据的方法捆绑在一起，隐藏内部实现细节，只暴露必要的接口。上面的类就是一个封装的例子。
*   **继承**：允许一个类（子类）继承另一个类（父类）的属性和方法，从而实现代码复用和扩展。
*   **多态**：指不同类的对象对同一消息（方法调用）做出不同的响应，增加了程序的灵活性。

在实际的量化交易项目中，你会看到大量运用这些概念设计的类结构，它们使系统更加模块化和易于管理。
