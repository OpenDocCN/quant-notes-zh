## 概述
在本节课中，我们将要学习量化交易中一个常用的选股策略——因子打分法。我们将了解该策略的核心思想、实施步骤以及背后的逻辑。

## 策略核心思想
上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看如何综合多个因子进行更全面的选股。

因子打分选股策略的核心思想是：**为每只股票的多个因子分别打分，然后计算总分，最后根据总分排名来筛选股票**。这类似于学生根据各科成绩的总分进行排名。

假设我们有300只股票（Stock1, Stock2, ..., Stock300）和四个因子（A, B, C, D）。我们的目标是从中选出排名前N（例如前10）的股票。

以下是该过程的抽象表示：

```python
# 伪代码示例：计算股票总分
def calculate_total_score(stock_data):
    """
    stock_data: 包含各因子得分的字典或DataFrame
    例如：{'因子A': 85, '因子B': 90, '因子C': 78, '因子D': 88}
    """
    total_score = stock_data['因子A'] + stock_data['因子B'] + stock_data['因子C'] + stock_data['因子D']
    return total_score

# 对每只股票计算总分并排序
ranked_stocks = sorted(all_stocks, key=calculate_total_score, reverse=True)
selected_stocks = ranked_stocks[:10]  # 选取前10名
```

## 关键前提条件
在实施打分之前，我们必须明确一个关键前提：**每个因子与预期收益率的相关性方向**。

![](img/0465ad757be65346bebcb81ddc31f1b3_1.png)

*   **正相关因子**：因子值**越大**，预期收益可能**越高**。例如：每股收益（EPS）、净资产收益率（ROE）。
*   **负相关因子**：因子值**越小**，预期收益可能**越高**。例如：市盈率（PE）、市净率（PB）。

这个方向性知识是我们的“先验知识”，它可以通过查阅券商研究报告或通过历史数据进行因子分析（如计算IC值）来获得。在后续的实际策略编写中，我们将预设一组“越高越好”和一组“越低越好”的因子。

## 策略实施步骤
基于以上思想，因子打分选股策略可以概括为以下几个步骤：

以下是实施该策略的关键步骤列表：

1.  **因子选择与方向确认**：选择一组有效的因子，并明确每个因子与收益的相关性方向（正或负）。
2.  **数据标准化**：由于不同因子的量纲和范围不同（例如，市盈率是几十，而营收增长率是百分比），需要将它们标准化到统一的尺度（如Z-score标准化），以便公平地相加。
3.  **因子打分**：根据标准化后的因子值及其方向进行打分。对于正相关因子，值越大得分越高；对于负相关因子，值越小得分越高。
4.  **计算总分**：将每只股票在所有因子上的得分相加，得到该股票的综合总分。
5.  **排序与筛选**：对所有股票按总分进行降序排列，选取排名靠前的股票构成投资组合。
6.  **定期调仓**：策略通常按固定周期（如每月）重复执行以上步骤，卖出不在新名单中的股票，买入新进入名单的股票。

## 总结
本节课中我们一起学习了因子打分法选股策略。该策略通过综合评估多个因子对股票进行量化评分，并依据总分排名进行筛选，比单一因子选股更为全面。实施该策略的关键在于明确各因子的收益相关性方向，并完成数据标准化、打分、汇总和排序等步骤。在接下来的课程中，我们将在量化平台中实际编写这一策略。
