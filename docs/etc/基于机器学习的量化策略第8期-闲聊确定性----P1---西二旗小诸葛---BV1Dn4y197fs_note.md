## 模型演进：引入确定性概念

在回顾了整体表现后，我们进入核心部分，探讨模型的优化。本次分享的重点是新增的“确定性”模型。

在A股数千只股票中，策略倾向于避开高风险标的（如ST股票），专注于寻找上涨“确定性”更高的机会。所谓“确定性高”，指的是某些行业或板块本身质地优良（如高毛利率、强现金流），其内部有多只股票都符合盈利模型的条件。

当前的选股模型每日会稳定产出约90至100多只符合条件的股票。由于资金有限，无法全部买入，因此需要从中进行二次筛选。

为了解决这个问题，我们引入了“确定性分数” **`P`**。这是一个介于0到1之间的值，在机器学习中，可以理解为一个经过Sigmoid函数处理的概率输出。

**`P = σ(x)`**，其中 **`0 ≤ P ≤ 1`**

这个分数代表了模型预测该股票上涨的置信度。例如，一只股票有95%的可能性上涨，另一只有80%的可能性，策略会选择前者。

## 策略模型框架

接下来，我们具体看看整合了确定性模型后的完整策略框架。

策略模型主要由三部分组成：
1.  **选股模型**：负责初筛，每日产出约90-120只候选股票。
2.  **择时模型**：为候选股票生成一个择时分数 **`S`**。该模型已加入周线判断以延长持股周期。为了避免行业过度集中，在应用分数 **`S`** 时会考虑行业分散配置。
3.  **确定性模型**：这是新增部分，为每只股票计算一个确定性分数 **`P`**。

最终的买入决策依据综合分数进行，公式如下：

**`最终分数 = P × S`**

系统将按照 **`最终分数`** 的高低对股票进行排序，优先买入分数更高的标的。这一机制确保了投资组合向高置信度的机会倾斜。
