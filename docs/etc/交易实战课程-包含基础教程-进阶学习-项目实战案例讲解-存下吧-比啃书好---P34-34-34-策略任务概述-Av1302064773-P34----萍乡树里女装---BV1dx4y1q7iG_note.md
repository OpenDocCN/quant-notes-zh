# 量化交易教程：P34：因子中性化（提纯）与策略构建概述

![](img/52687b9521cb012375b5c0d645702fbe_0.png)

在本节课中，我们将学习量化交易中一个重要的数据处理步骤——因子中性化（或称因子提纯），并了解如何基于处理后的因子构建一个简单的选股策略。我们将通过一个具体的例子，即从市净率（PB）因子中剔除市值（Market Cap）的影响，来详细讲解其原理和实现思路。

![](img/52687b9521cb012375b5c0d645702fbe_2.png)

## 因子中性化的核心思想 🎯

上一节我们介绍了因子处理的基本概念，本节中我们来看看如何从因子中剔除我们不想要的“杂质”。

因子中性化的目标是提纯因子。例如，我们发现市净率（PB）因子与市值（Market Cap）存在较强的相关性。这意味着PB因子中包含了一部分市值的信息。如果我们只想研究PB本身对股价的影响，就需要从PB因子中剔除市值的影响。这个过程就像从一个混合信号中分离出我们想要的特定成分。

## 如何实现因子提纯？🔍

核心思路是使用线性回归来量化并剔除我们不想要的成分。以下是实现步骤的分解。

### 第一步：建立回归模型

![](img/52687b9521cb012375b5c0d645702fbe_9.png)

![](img/52687b9521cb012375b5c0d645702fbe_10.png)

![](img/52687b9521cb012375b5c0d645702fbe_11.png)

我们通过建立一个回归方程，来量化“杂质”（如市值）对目标因子（如PB）的解释程度。

![](img/52687b9521cb012375b5c0d645702fbe_13.png)

![](img/52687b9521cb012375b5c0d645702fbe_14.png)

*   **模型设定**：我们将目标因子（如PB）作为因变量 `Y`（真实值），将需要剔除的因子（如市值）作为自变量 `X`。
*   **回归方程**：建立如下线性关系：
    `Y ≈ W * X + B`
    其中，`W` 是权重（系数），`B` 是截距。我们的目标是求解出 `W` 和 `B`。

![](img/52687b9521cb012375b5c0d645702fbe_15.png)

### 第二步：理解回归结果

求解出 `W` 和 `B` 后，我们可以得到市值对PB的“预测值”。

![](img/52687b9521cb012375b5c0d645702fbe_17.png)

![](img/52687b9521cb012375b5c0d645702fbe_18.png)

![](img/52687b9521cb012375b5c0d645702fbe_19.png)

![](img/52687b9521cb012375b5c0d645702fbe_20.png)

![](img/52687b9521cb012375b5c0d645702fbe_21.png)

*   **预测值的含义**：`预测值 = W * X + B`。这个值代表了PB因子中能够被市值所解释的那一部分。
*   **残差的含义**：用PB因子的真实值 `Y` 减去预测值，得到残差（Residual）：
    `残差 = Y - (W * X + B)`
    这个残差就是市值解释不了的那部分PB信息，即我们“提纯”后的、与市值无关的纯PB因子。

整个过程可以概括为：**用因子真实值减去其能被其他变量（如市值）解释的部分，剩余部分即为中性化（提纯）后的因子**。

![](img/52687b9521cb012375b5c0d645702fbe_23.png)

## 构建一个简单的因子选股策略 📈

理解了中性化原理后，我们将其融入一个完整的策略流程中。以下是构建策略的关键步骤。

![](img/52687b9521cb012375b5c0d645702fbe_25.png)

![](img/52687b9521cb012375b5c0d645702fbe_26.png)

![](img/52687b9521cb012375b5c0d645702fbe_28.png)

首先，我们需要获取并准备基础数据。

![](img/52687b9521cb012375b5c0d645702fbe_29.png)

![](img/52687b9521cb012375b5c0d645702fbe_30.png)

以下是数据处理与策略执行的主要步骤：

![](img/52687b9521cb012375b5c0d645702fbe_32.png)

1.  **数据获取**：获取股票池的市净率（PB）和市值（Market Cap）数据。
2.  **股票池过滤**：剔除不符合条件的股票，例如：
    *   停牌的股票。
    *   ST/*ST股票。
    *   上市时间过短（如少于半年）的股票。
3.  **因子处理**：对保留的股票数据依次进行处理：
    *   **去极值**：处理异常值。
    *   **标准化**：将数据缩放到统一尺度。
    *   **中性化**：使用上述回归方法，从PB因子中剔除市值的影响。
4.  **选股逻辑**：基于处理后的纯PB因子制定规则。例如，买入**中性化后PB值小于0.2**的股票。
5.  **执行交易**：根据选股结果进行买入和卖出操作。
