在本节课中，我们将要学习金融量化分析中两个核心概念：阿尔法（α）和贝塔（β）。理解这两个指标对于评估投资策略的优劣至关重要。

## 收益的构成

上一节我们介绍了策略评估的多种指标，本节中我们来看看如何区分收益的来源。一项投资的收益通常由两部分构成。

以下是收益的两个主要组成部分：

*   **市场收益**：这部分收益与整体市场环境相关。当市场大环境向好时，大部分投资都能获得收益。这部分收益是“随大流”获得的。
*   **超额收益**：这部分收益与市场整体波动关系不大，主要来源于投资者独特的策略、敏锐的观察或主动管理。例如，通过深入分析公司财务、运营状况后做出的精准投资决策所带来的收益。

## 阿尔法（α）与贝塔（β）的定义

我们使用阿尔法（α）和贝塔（β）这两个指标来分别衡量上述两部分收益。

### 贝塔（β）：衡量市场相关性

贝塔衡量的是你的投资策略对市场波动的敏感性，它代表了“市场收益”部分。贝塔反映了策略的系统性风险。

*   **公式含义**：在资本资产定价模型（CAPM）中，单个资产或策略的预期收益与市场收益之间的关系可以表示为：
    `E(Ri) = Rf + β * (E(Rm) - Rf)`
    其中，`E(Ri)`是资产预期收益，`Rf`是无风险利率，`E(Rm)`是市场预期收益，`β`即为我们所说的贝塔系数。
*   **简单理解**：贝塔值高，意味着策略收益随市场涨跌波动大；贝塔值低，则意味着策略受市场影响较小。

### 阿尔法（α）：衡量策略超额能力

![](img/42b077f4b515725b724ac160b0754f51_1.png)

阿尔法衡量的是与市场波动无关的超额收益，它代表了“超额收益”部分。阿尔法反映了策略的非系统性风险，即管理者通过技能创造的额外价值。

*   **公式含义**：阿尔法实际上是上述CAPM模型中的残差项，即实际收益与模型预测收益的差值：
    `α = 实际收益 - [Rf + β * (市场收益 - Rf)]`
*   **简单理解**：正阿尔法表示策略跑赢了市场基准，获得了超额收益；负阿尔法则表示策略跑输了市场。

## 阿尔法和贝塔的计算与意义

那么，阿尔法和贝塔具体如何计算呢？在实际分析中，我们通常通过线性回归的方法来估算它们。

我们可以将策略收益与市场收益的关系看作一个线性方程：
`策略收益 = α + β * 市场收益 + ε`
其中，`ε`为随机误差项。通过历史数据拟合这个方程，即可求解出α和β的估计值。

对于投资者而言，理解这两者的侧重点不同：

1.  **贝塔（市场收益）难以控制**：市场整体的涨跌是外部宏观因素，个人投资者通常无法改变或预测。因此，这部分收益具有较大的不确定性。
2.  **阿尔法（超额收益）是追求目标**：正因为市场难以掌控，量化策略的核心目标就转向了寻求稳定的、正的阿尔法。这意味着要通过模型、算法和独特的洞察力，创造出独立于市场涨跌的收益。

所以，在策略评估中，我们通常更关注**阿尔法**的大小和稳定性，因为它直接体现了策略本身的有效性和附加价值。

## 总结

本节课中我们一起学习了金融量化中的两个基石概念：阿尔法（α）和贝塔（β）。

*   **贝塔（β）** 衡量策略收益与市场收益的相关性，代表随市场波动获得的收益。
*   **阿尔法（α）** 衡量策略超越市场基准的超额收益，代表由策略自身能力创造的收益。

理解并区分收益的来源，能帮助我们更客观地评估一个策略的真实表现：是搭了市场的“顺风车”，还是真正具备了“跑赢市场”的硬实力。在后续的因子策略分析等课程中，我们将深入探讨如何构建和优化以获取阿尔法为核心的量化策略。
