## 需求分析

![](img/441131e5e891765166dd6fd25b5de500_11.png)

![](img/441131e5e891765166dd6fd25b5de500_13.png)

上一节我们识别出了股票数据中的金叉和死叉信号。本节中，我们将利用这些信号来执行买卖操作。

![](img/441131e5e891765166dd6fd25b5de500_15.png)

![](img/441131e5e891765166dd6fd25b5de500_17.png)

具体的需求是：假设一个人从2010年1月1日开始，拥有10万元初始资金。每当出现金叉信号时，他尽可能多地买入股票；每当出现死叉信号时，他将手中持有的所有股票卖出。我们需要计算到数据截止日期（例如2020年）为止，他的总收益是多少。

![](img/441131e5e891765166dd6fd25b5de500_19.png)

![](img/441131e5e891765166dd6fd25b5de500_21.png)

![](img/441131e5e891765166dd6fd25b5de500_23.png)

![](img/441131e5e891765166dd6fd25b5de500_25.png)

![](img/441131e5e891765166dd6fd25b5de500_27.png)

![](img/441131e5e891765166dd6fd25b5de500_29.png)

![](img/441131e5e891765166dd6fd25b5de500_31.png)

以下是实现此需求的关键点分析：
*   **买卖单价**：使用交易信号发生当日的开盘价作为买卖股票的单价。
*   **买卖时机**：金叉时间点买入，死叉时间点卖出。
*   **交易单位**：股票交易以“手”为单位（1手=100股），因此买入时需计算10万元最多能买多少手。
*   **特殊情况**：如果数据最后一天是金叉（买入），则手中会持有未卖出的股票。这部分剩余股票的价值，需按最后一天的收盘价计算，并计入总收益。

### 步骤一：准备交易信号序列

![](img/441131e5e891765166dd6fd25b5de500_41.png)

首先，我们需要将之前找出的金叉和死叉时间点整合并排序，形成一个清晰的交易信号序列。

![](img/441131e5e891765166dd6fd25b5de500_43.png)

```python
# 假设 golden_date 是金叉日期列表， death_date 是死叉日期列表
# 创建Series，用1标记金叉，用0标记死叉
s1 = pd.Series(1, index=golden_date)  # 金叉标识
s2 = pd.Series(0, index=death_date)   # 死叉标识

![](img/441131e5e891765166dd6fd25b5de500_45.png)

# 合并两个Series并按日期索引排序
s = pd.concat([s1, s2]).sort_index()

# 筛选出2010年及以后的数据
s = s['2010':'2020']
```
现在，`s` 这个Series的索引是日期，值为1代表该日是金叉（买入信号），值为0代表该日是死叉（卖出信号）。

### 步骤三：遍历信号执行交易

![](img/441131e5e891765166dd6fd25b5de500_85.png)

![](img/441131e5e891765166dd6fd25b5de500_87.png)

核心逻辑是遍历交易信号序列 `s`，根据值是1还是0来决定买入或卖出。

![](img/441131e5e891765166dd6fd25b5de500_89.png)

以下是交易逻辑的代码框架：

![](img/441131e5e891765166dd6fd25b5de500_91.png)

![](img/441131e5e891765166dd6fd25b5de500_93.png)

![](img/441131e5e891765166dd6fd25b5de500_95.png)

```python
for i in range(len(s)):
    # 获取当前信号的日期
    current_time = s.index[i]
    # 判断信号类型
    if s.iloc[i] == 1:  # 金叉，买入
        # 1. 获取买入单价（当日开盘价）
        buy_price = df.loc[current_time, 'open']
        # 2. 计算最多能买多少手（1手=100股）
        # 一手股票所需金额 = buy_price * 100
        hand_count = money // (buy_price * 100)
        # 3. 计算买入股数，并更新持有数量
        buy_count = hand_count * 100
        hold += buy_count
        # 4. 从可用资金中扣除买入花费
        money -= buy_count * buy_price

    else:  # 死叉，卖出
        # 1. 获取卖出单价（当日开盘价）
        sell_price = df.loc[current_time, 'open']
        # 2. 卖出全部持有股票，获得资金
        money += hold * sell_price
        # 3. 清空持有股数
        hold = 0
```

### 步骤四：计算最终收益

循环结束后，需要处理可能剩余的股票（如果最后一天是金叉），并计算总收益。

![](img/441131e5e891765166dd6fd25b5de500_99.png)

![](img/441131e5e891765166dd6fd25b5de500_101.png)

![](img/441131e5e891765166dd6fd25b5de500_102.png)

![](img/441131e5e891765166dd6fd25b5de500_104.png)

```python
# 计算剩余股票的价值（按最后一天的收盘价）
last_price = df.iloc[-1]['close']  # 获取最后一天的收盘价
last_money = hold * last_price

![](img/441131e5e891765166dd6fd25b5de500_106.png)

![](img/441131e5e891765166dd6fd25b5de500_108.png)

![](img/441131e5e891765166dd6fd25b5de500_110.png)

![](img/441131e5e891765166dd6fd25b5de500_112.png)

# 计算总收益
# 总收益 = 最终现金 + 剩余股票价值 - 初始本金
total_earn = (money + last_money) - first_money
print(f“经过模拟交易，最终总收益为：{total_earn:.2f}元”)
```
