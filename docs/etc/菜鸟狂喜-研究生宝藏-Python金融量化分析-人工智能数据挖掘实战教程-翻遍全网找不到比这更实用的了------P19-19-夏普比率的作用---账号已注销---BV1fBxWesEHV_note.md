## 概述

夏普比率是一个用于衡量投资组合风险调整后收益的指标。它描述了在承担单位风险的情况下，投资者能够获得多少超额回报。简单来说，它帮助我们判断一项投资是否“划算”。

## 夏普比率的定义与意义

上一节我们介绍了投资回报率，本节中我们来看看如何结合风险来评估收益。

![](img/f2c177fd101b7fa28e066b4229719bce_1.png)

夏普比率描述的是：对于单位风险而言，投资所能带来的收益是多少。指标越高，意味着在承担相同风险的情况下，获得的收益越高。因此，在比较不同投资选项时，我们通常倾向于选择夏普比率更高的那个。

![](img/f2c177fd101b7fa28e066b4229719bce_3.png)

为了便于理解，我们可以将其类比为一份高风险、高报酬的工作。例如，一份在战区的日薪极高的工作，其高报酬对应的是极高的生命风险。夏普比率就是用来量化这种“风险与回报是否匹配”的关系。

![](img/f2c177fd101b7fa28e066b4229719bce_5.png)

## 夏普比率的计算公式

理解了其意义后，我们来看看如何计算夏普比率。

它的核心思想是比较投资组合的收益与无风险收益（如国债利息）之间的差额，再除以投资组合收益的波动性（风险）。具体公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率。
*   **Rf**：无风险收益率（例如国债年化收益率）。
*   **σp**：投资组合收益率的标准差（代表风险）。

这个公式计算的是**每承担一单位风险（波动）所获得的超额收益**。

## 在Python中计算夏普比率

接下来，我们将在Python环境中实际计算几只股票的夏普比率。假设我们已经有了股票的历史日收益率数据。

以下是计算夏普比率的关键步骤：

1.  **准备数据**：确保收益率数据完整，对缺失值进行填充（例如使用前一天的收益率）。
2.  **设定参数**：确定无风险收益率（例如年化5%）和一年的交易天数（例如250天）。
3.  **执行计算**：应用上述公式进行计算。需要注意的是，由于我们通常使用日收益率数据，在计算年化夏普比率时需要进行调整。

![](img/f2c177fd101b7fa28e066b4229719bce_7.png)

```python
import numpy as np
import pandas as pd

![](img/f2c177fd101b7fa28e066b4229719bce_9.png)

# 假设 returns 是一个包含多只股票日收益率的 DataFrame
# 填充缺失值（使用前向填充）
returns_filled = returns.fillna(method='ffill')

# 设定年化无风险收益率和年交易天数
risk_free_rate = 0.05
trading_days_per_year = 250

# 计算超额收益（日度）
excess_returns = returns_filled - risk_free_rate / trading_days_per_year

# 计算夏普比率（年化）
# 公式： (平均超额日收益 * 年交易天数) / (日收益标准差 * sqrt(年交易天数))
# 简化后为： 平均超额日收益 / 日收益标准差 * sqrt(年交易天数)
sharpe_ratios = (excess_returns.mean() / returns_filled.std()) * np.sqrt(trading_days_per_year)

print(sharpe_ratios)
```

执行以上代码后，我们将得到每只股票的夏普比率。结果是一个数值序列。

## 如何解读结果

![](img/f2c177fd101b7fa28e066b4229719bce_11.png)

计算完成后，我们需要解读这些数字。

![](img/f2c177fd101b7fa28e066b4229719bce_13.png)

*   **夏普比率为正**：表示投资组合的收益超过了无风险收益，数值越大越好。
*   **夏普比率为负**：表示投资组合的收益还不如无风险收益，通常应避免。
*   **比较不同股票**：在正值的范围内，夏普比率最高的股票，意味着其“风险收益性价比”最高。

![](img/f2c177fd101b7fa28e066b4229719bce_15.png)

例如，如果股票A的夏普比率为1.2，股票B的夏普比率为0.8，那么在其他条件相似的情况下，股票A是更优的选择。
