## Excel发展简史 📜

上一节我们介绍了课程的整体框架，本节中我们来看看Excel这款工具是如何诞生和发展的。

世界上第一款电子表格软件并非Excel，而是在1978年推出的**VisiCalc**。当时人们使用纸笔或黑板进行手工计算，修改数据非常麻烦。VisiCalc的发明者希望设计一款电子表格来解决这个问题。

从VisiCalc的界面截图可以看出，其结构已经定型：列由字母（A, B, C...）标识，行由数字（1, 2, 3...）标识，形成了一个二维表格。这为后来的电子表格软件奠定了基础。

![](img/d2a395736089afdab2fc026c6593870b_13.png)

![](img/d2a395736089afdab2fc026c6593870b_15.png)

![](img/d2a395736089afdab2fc026c6593870b_17.png)

然而，第一款获得巨大成功的电子表格是**Lotus 1-2-3**。它于1983年问世，集成了表格处理、图表和数据库三大功能，在DOS系统上非常流行。

![](img/d2a395736089afdab2fc026c6593870b_19.png)

![](img/d2a395736089afdab2fc026c6593870b_21.png)

![](img/d2a395736089afdab2fc026c6593870b_23.png)

微软公司早期也开发了名为**Multiplan**的电子表格，但未能竞争过Lotus 1-2-3。为此，微软秘密召开会议，决定开发一款集众家之长的软件。于是在1985年，**Excel** 1.0 for Mac诞生了。

![](img/d2a395736089afdab2fc026c6593870b_25.png)

Excel真正成为市场主导，得益于**Windows操作系统的普及**以及微软的**捆绑销售策略**。当Windows系统流行后，Lotus 1-2-3未能很好地适配Windows版本，而Excel则与Windows深度绑定，最终赢得了市场。

此后，Excel不断迭代更新：
*   **1993年**：Excel 5.0被集成到Microsoft Office套件中。
*   **1994年**：开始支持**VBA**，这是一种依附于Excel的编程语言，用于实现自动化操作。
*   **1997年**：Excel 97是一个重要转折点，界面和功能都有显著提升。
*   **2003年**：Excel 2003在中国非常流行。
*   **2007年及以后**：界面改为Ribbon功能区，功能更智能，但核心架构变化不大。

目前，常见的版本有2007、2010、2016、2019等。了解版本差异对文件兼容性很重要。

### 工作簿与文件格式

![](img/d2a395736089afdab2fc026c6593870b_32.png)

在Excel中，我们创建的文件称为**工作簿**。其文件后缀名标识了版本：
*   **`.xlsx`**：2007版及以后版本使用的格式。
*   **`.xls`**：97-2003版使用的格式。

**高级版本软件可以打开低级版本的文件，但反之则不行。** 例如，Excel 2016可以打开`.xls`文件，但Excel 2003无法打开`.xlsx`文件。

![](img/d2a395736089afdab2fc026c6593870b_34.png)

![](img/d2a395736089afdab2fc026c6593870b_36.png)

### 功能区与选项卡

![](img/d2a395736089afdab2fc026c6593870b_38.png)

打开工作簿后，顶部的区域称为**功能区**。所有操作命令都集中在这里。功能区按功能分类为不同的**选项卡**，如“开始”、“插入”、“公式”、“数据”等。

![](img/d2a395736089afdab2fc026c6593870b_40.png)

**操作技巧**：双击当前选项卡标签（如“开始”）可以折叠或展开功能区，这在屏幕空间有限时非常有用。

### 单元格、行与列

![](img/d2a395736089afdab2fc026c6593870b_42.png)

工作表由网格构成，每个格子是一个**单元格**，它是存储和操作数据的基本单位。

![](img/d2a395736089afdab2fc026c6593870b_44.png)

*   **列**：由字母标识（A, B, C, ..., Z, AA, AB...）。
*   **行**：由数字标识（1, 2, 3...）。
*   **单元格地址**：由列标和行号组成，例如`A1`代表A列第1行的单元格。这是公式中引用数据的基础。
*   **编辑栏**：位于功能区下方，显示当前活动单元格的地址和内容。当你输入公式时，单元格显示计算结果，而编辑栏显示原始公式。

Excel工作表的容量很大：
*   **行数**：`1,048,576`行
*   **列数**：`16,384`列（A - XFD）

**注意**：虽然Excel能处理百万行数据，但当数据量达到10万行级别时，操作会明显变慢。对于大数据量或复杂分析，建议使用Python等编程工具，Excel更适合中小型数据集的快速处理。

### 工作表

一个工作簿可以包含多个**工作表**（默认为Sheet1, Sheet2...），就像一本笔记本包含多页纸。以下是工作表的常用操作：

以下是工作表的常用操作列表：
*   **重命名**：双击工作表标签，或右键点击选择“重命名”。
*   **更改标签颜色**：右键点击工作表标签，选择“工作表标签颜色”。
*   **插入/删除**：点击右侧的`+`号插入新工作表；右键点击标签选择“删除”。
*   **移动或复制**：
    *   **在同一工作簿内**：直接拖动标签可移动；按住`Ctrl`键拖动可复制。或右键选择“移动或复制”，勾选“建立副本”即可复制。
    *   **在不同工作簿间**：右键选择“移动或复制”，在“工作簿”下拉列表中选择目标工作簿，再选择位置，决定是否“建立副本”。

### 冻结窗格

当表格数据很多，向下或向右滚动时，标题行或列会消失，影响查看。**冻结窗格**功能可以锁定特定的行或列。

![](img/d2a395736089afdab2fc026c6593870b_46.png)

![](img/d2a395736089afdab2fc026c6593870b_48.png)

![](img/d2a395736089afdab2fc026c6593870b_50.png)

![](img/d2a395736089afdab2fc026c6593870b_52.png)

![](img/d2a395736089afdab2fc026c6593870b_54.png)

操作路径：**“视图”选项卡 -> “冻结窗格”**。
*   **冻结首行**：滚动时保持第一行可见。
*   **冻结首列**：滚动时保持第一列可见。
*   **自定义冻结**：例如想冻结前两行和前两列，需先选中`C3`单元格（即第三行第三列的交叉点），然后点击“拆分”，再点击“冻结窗格”。这样滚动时，左上角区域将保持固定。

## 常用快捷方式 ⚡

![](img/d2a395736089afdab2fc026c6593870b_62.png)

最后，我们介绍一些能极大提升效率的快捷操作。熟练使用快捷键是Excel高手的标志之一。

以下是部分核心快捷键列表：
*   `Ctrl + N`：新建工作簿
*   `Ctrl + S`：保存工作簿
*   `Ctrl + C` / `Ctrl + V`：复制/粘贴
*   `Ctrl + Z`：撤销上一步操作
*   `Ctrl + Y`：恢复撤销的操作
*   `Ctrl + F`：查找
*   `Ctrl + H`：替换
*   `Ctrl + 方向键`：快速跳转到数据区域的边缘
*   `Ctrl + Shift + 方向键`：快速选中连续数据区域
*   `F2`：编辑当前选中的单元格
*   `F4`：重复上一步操作；在编辑公式时，循环切换单元格引用类型（绝对/相对引用）
