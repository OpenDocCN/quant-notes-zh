## 需求分析

![](img/d40940a1d87855b7fbaf62fcc0466685_17.png)

上一节我们明确了课程目标，本节中我们来看看具体的需求细节。

首先，我们需要明确策略的时间范围。原始数据从2001年开始，但我们的策略从2010年1月1日执行到2020年2月（课程制作时）。因此，第一步是筛选出这个时间段的数据。

其次，需要理解交易单位。“一手”股票在A股市场代表**100股**。这意味着每次买入操作都是以100股为单位进行的。

最后，计算收益的核心公式是：
**总收益 = 卖出股票总金额 + 剩余股票价值 - 买入股票总金额**

## 计算买入股票的总成本

![](img/d40940a1d87855b7fbaf62fcc0466685_51.png)

我们的策略是每月第一个交易日买入一手（100股）股票。因此，我们需要先找出每个月的第一个交易日。

![](img/d40940a1d87855b7fbaf62fcc0466685_53.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_55.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_57.png)

以下是实现步骤：
1.  使用`resample(‘M‘)`方法按月份对数据进行分组。
2.  使用`.first()`方法获取每个月的第一行数据，即当月第一个交易日的数据。
3.  从这些数据中提取‘open‘（开盘价）列。
4.  每个开盘价乘以100（一手），然后对所有月份的结果求和，即为总成本。

![](img/d40940a1d87855b7fbaf62fcc0466685_59.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_61.png)

```python
# 获取每月第一个交易日的数据
df_monthly = new_df.resample(‘M‘).first()

![](img/d40940a1d87855b7fbaf62fcc0466685_63.png)

# 计算买入股票的总花费：每月开盘价 * 100股，然后求和
cost = df_monthly[‘open‘].sum() * 100
print(f“买入股票总成本：{cost}“)
```

## 计算卖出股票的总收入

![](img/d40940a1d87855b7fbaf62fcc0466685_75.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_77.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_79.png)

策略规定在每年最后一个交易日卖出所有股票。但需要注意，2020年尚未结束，因此2020年买入的股票不会被卖出，需要单独处理。

![](img/d40940a1d87855b7fbaf62fcc0466685_81.png)

以下是实现步骤：
1.  使用`resample(‘A‘)`方法按年份对数据进行分组。
2.  使用`.last()`方法获取每年的最后一行数据，即当年最后一个交易日的数据。
3.  由于2020年数据不应卖出，我们需要剔除最后一年的数据。
4.  从剩下的每年数据中提取‘open‘（开盘价）列。
5.  每年卖出的是当年累计买入的所有股票。一个完整年份有12个月，买入12手，即**1200股**。因此，每年的卖出收入为：当年最后交易日的开盘价 * 1200。
6.  对所有年份的卖出收入求和。

![](img/d40940a1d87855b7fbaf62fcc0466685_83.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_85.png)

```python
# 获取每年最后一个交易日的数据
df_yearly = new_df.resample(‘A‘).last()

![](img/d40940a1d87855b7fbaf62fcc0466685_87.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_89.png)

# 剔除2020年的数据（因为2020年未结束，不卖出）
df_yearly = df_yearly[:-1]

![](img/d40940a1d87855b7fbaf62fcc0466685_91.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_93.png)

# 计算卖出股票的总收入：每年最后交易日的开盘价 * 1200股，然后求和
receive = df_yearly[‘open‘].sum() * 1200
print(f“卖出股票总收入：{receive}“)
```

## 计算剩余股票价值及总收益

![](img/d40940a1d87855b7fbaf62fcc0466685_103.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_105.png)

截至2020年2月，投资者手中还持有2020年买入但未卖出的股票。我们需要估算这部分股票的价值，并将其计入总资产。

![](img/d40940a1d87855b7fbaf62fcc0466685_107.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_109.png)

以下是实现步骤：
1.  确定剩余股票数量：2020年已过去的月份里，每月买入一手，假设到2月共买入2手，即**200股**。
2.  确定估价单价：使用最新一个交易日（即数据最后一行）的**收盘价(close)** 来估算剩余股票价值。
3.  计算剩余股票价值。
4.  最终计算总收益。

![](img/d40940a1d87855b7fbaf62fcc0466685_111.png)

![](img/d40940a1d87855b7fbaf62fcc0466685_113.png)

```python
# 计算剩余股票价值：最新收盘价 * 200股
last_price = new_df[‘close‘][-1]  # 获取最后一个交易日的收盘价
last_money = last_price * 200
print(f“剩余股票估值：{last_money}“)

# 计算总收益
total_profit = receive + last_money - cost
print(f“此策略总收益为：{total_profit}“)
```
