## 策略初始化与定时调仓

上一节我们介绍了必要的工具包，本节中我们来看看如何初始化我们的交易策略。一个量化策略需要明确其运行的时间频率，例如是按日、按周还是按月执行。

![](img/bfc4610411a3a5f905318d253c749b1a_9.png)

在我们的策略中，我们将采用**按月调仓**的方式。这意味着策略会每月定期运行一次，重新评估股票池，卖出不再符合策略标准的股票，并买入新符合标准的股票。这种频率在实战中较为常见，既能捕捉市场变化，又避免了过于频繁交易产生的高额成本。

![](img/bfc4610411a3a5f905318d253c749b1a_11.png)

为了实现定时调仓，我们需要在策略的初始化函数中设置一个定时器。这个定时器会指定策略运行的周期。

![](img/bfc4610411a3a5f905318d253c749b1a_13.png)

![](img/bfc4610411a3a5f905318d253c749b1a_15.png)

```python
# 示例：在策略初始化函数中设置按月运行的定时器
self.schedule_rebalance(event_rule=date_rules.month_start())
```

## 数据预处理与股票筛选

![](img/bfc4610411a3a5f905318d253c749b1a_27.png)

![](img/bfc4610411a3a5f905318d253c749b1a_29.png)

获取到全市场股票列表只是开始。在实际策略中，我们很少会交易所有股票，通常需要根据一定的规则（如市值、流动性、上市时间等）进行筛选，构建一个更精简、更适合策略的股票池。

因此，在后续步骤中，我们将对这些股票的ID进行过滤，剔除不符合我们预设条件的股票，例如ST股票、上市时间过短的新股、或者交易不活跃的股票等。这一步是为后续的因子选股策略打下坚实的基础。
