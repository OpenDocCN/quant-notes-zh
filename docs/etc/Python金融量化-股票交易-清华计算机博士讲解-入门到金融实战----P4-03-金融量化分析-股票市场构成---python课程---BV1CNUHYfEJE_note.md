# Python金融量化+股票交易：P4：03 金融量化分析-股票市场构成 📊

在本节课中，我们将要学习股票市场的构成，了解市场中有哪些参与者，以及影响股价的主要因素。理解这些基础知识是进行后续量化分析的前提。

## 公司和投资者 💼

上一节我们介绍了股票的分类，本节中我们来看看股票市场的构成。首先，市场中最核心的参与者是公司和投资者。公司是融资方，需要资金；投资者是出资方，提供资金进行投资。

## 监管机构与交易所 ⚖️

但是，上市和股票交易不能由公司和投资者直接进行，以避免暗箱操作。因此，需要监管机构。最常见的监管机构是证监会，负责监管证券行业。公司上市需要向证监会提交各种材料，证监会负责判断公司是否存在欺诈、洗钱等违法行为。证监会有权决定公司能否上市，甚至可以将已上市的公司撤下。

第二个是证券业协会，其作用相对较弱，主要负责主办证券从业资格考试等事务。

第三个是交易所。交易所提供股票交易的场所，例如中国的上海和深圳交易所。在电子化交易之前，投资者需要到交易所现场排队交易。现在，交易主要通过互联网连接到交易所的系统进行。交易所的功能是处理各种买卖股票的请求。

## 证券中介机构 🔗

还有一个重要的参与者是证券中介机构，俗称券商。个人投资者不能直接去交易所买卖股票，必须通过券商开户。这是因为在交易所拥有交易席位的成本非常高。券商在交易所有席位，他们开发软件（如各种炒股软件）供投资者使用。投资者通过券商的软件提交交易指令，券商再通过其在交易所的席位将指令下达，从而完成股票买卖。

## 中国的交易所与板块 🇨🇳

中国的交易所有两个，一个在上海，一个在深圳。每个交易所都有不同的板块。

以下是上海和深圳交易所的板块划分：
*   **上海交易所**：只有一个主板。
*   **深圳交易所**：分为三个板块：主板、中小板和创业板。中小板和创业板是国家为鼓励小公司或创业公司发展而设立的。主板上市要求严格，而创业板要求相对较低，例如可能要求连续几年达到一定净利润即可。

## 大盘指数 📈

对于每一个板块，都有一个代表其整体走势的“大盘”指数。指数反映了该板块内所有股票的综合表现。例如，上海主板的大盘指数叫做**沪指**（上证指数）。深圳主板的指数叫做**深成指**，中小板的指数是**中小板指**，创业板的指数是**创业板指**。

指数就像一个趋势图。如果构成该指数的大部分股票（代表大部分市值）在上涨，那么指数就会上涨，表明市场整体向好；反之则表明市场行情不佳。它提供了一个概括性的视角来观察整个市场的趋势。

## 影响股价的因素 ⚙️

接下来我们说一下影响股价的因素。股价的波动受到多种力量驱动。

以下是影响股价的主要因素：
*   **公司基本面**：这是公司的内在价值，包括盈利能力（如净利润）、财务状况、成长潜力等。公式可以简单表示为：`股价 ≈ 公司价值 / 总股本`。
*   **宏观经济环境**：国家的经济增长率、利率、通货膨胀率、货币政策等都会影响整个股市的情绪和资金流向。
*   **行业趋势**：公司所在行业是处于上升期还是衰退期，有无技术革新或政策扶持。
*   **市场情绪与资金流动**：投资者的心理预期、恐慌或贪婪情绪，以及市场上资金的充裕程度（流动性）会短期内显著影响股价。
*   **政策与新闻**：政府的相关政策、国际关系、以及突发的公司或行业新闻都会导致股价波动。
