## 为何使用NumPy？

在深入概念之前，我们先通过两个例子直观感受NumPy带来的便利。

### 示例一：货币换算

假设我们已知50家公司的市值（单位：美元），需要将其全部转换为人民币（汇率为6.8）。

**传统Python列表方法**：
```python
import random
# 生成50个100到200之间的随机浮点数，模拟公司市值（万美元）
a = [random.uniform(100, 200) for _ in range(50)]
# 使用列表推导式进行换算
b = [value * 6.8 for value in a]
```

![](img/b8e8e879f00b8767c8d875092cd04752_11.png)

![](img/b8e8e879f00b8767c8d875092cd04752_13.png)

**NumPy数组方法**：
```python
import numpy as np
# 将列表转换为NumPy数组
a_np = np.array(a)
# 直接进行数组与标量的乘法运算
b_np = a_np * 6.8
```
使用NumPy，一行代码即可完成整个数组的运算，简洁高效。

### 示例二：计算购物车总金额

![](img/b8e8e879f00b8767c8d875092cd04752_21.png)

假设我们有两个列表，分别存储商品单价和对应数量，需要计算总金额。

![](img/b8e8e879f00b8767c8d875092cd04752_23.png)

**NumPy数组方法**：
```python
import numpy as np
# 生成单价和数量数组
price = np.array([random.uniform(10, 20) for _ in range(50)])
quantity = np.array([random.randint(1, 10) for _ in range(50)])
# 对应元素相乘得到各商品金额，再求和
total = (price * quantity).sum()
```
NumPy允许数组进行逐元素运算，并提供了便捷的求和函数，极大地简化了代码。

![](img/b8e8e879f00b8767c8d875092cd04752_25.png)

## NumPy核心：ndarray对象

![](img/b8e8e879f00b8767c8d875092cd04752_31.png)

![](img/b8e8e879f00b8767c8d875092cd04752_33.png)

NumPy的核心是`ndarray`对象，即N维数组对象。它由同质元素组成，并具有固定大小。

### 创建数组

使用`np.array()`函数，传入一个序列（如列表）即可创建数组。
```python
import numpy as np
arr1 = np.array([1, 2, 3, 4, 5]) # 从列表创建
arr2 = np.array(range(10)) # 从可迭代对象创建
```

### 数组与列表的区别

![](img/b8e8e879f00b8767c8d875092cd04752_37.png)

![](img/b8e8e879f00b8767c8d875092cd04752_39.png)

![](img/b8e8e879f00b8767c8d875092cd04752_41.png)

理解数组与Python列表的区别至关重要，以下是两个核心区别：

![](img/b8e8e879f00b8767c8d875092cd04752_43.png)

![](img/b8e8e879f00b8767c8d875092cd04752_45.png)

1.  **元素类型必须相同**：列表可以容纳不同类型的数据（如整数、字符串混合）。数组要求所有元素必须是相同的数据类型。
2.  **大小固定**：列表可以动态地添加或删除元素。数组在创建时大小就固定了，无法直接改变其元素数量（但可以修改元素值）。

![](img/b8e8e879f00b8767c8d875092cd04752_47.png)

![](img/b8e8e879f00b8767c8d875092cd04752_49.png)

这种同质性和固定大小的特性，是NumPy能够进行高效底层运算和内存管理的原因。

## 数组的常用属性

创建数组后，我们可以通过一系列属性来了解其信息。

![](img/b8e8e879f00b8767c8d875092cd04752_51.png)

```python
import numpy as np
# 创建一个二维数组
arr = np.array([[1, 2, 3], [4, 5, 6]])
```

![](img/b8e8e879f00b8767c8d875092cd04752_53.png)

以下是该数组的一些关键属性：

![](img/b8e8e879f00b8767c8d875092cd04752_55.png)

*   **`dtype`**: 查看数组元素的数据类型。
    ```python
    print(arr.dtype) # 输出：int64
    ```
*   **`size`**: 查看数组元素的总个数。
    ```python
    print(arr.size) # 输出：6
    ```
*   **`shape`**: 查看数组的维度形状，返回一个表示各维度长度的元组。
    ```python
    print(arr.shape) # 输出：(2, 3) 表示2行3列
    ```
*   **`ndim`**: 查看数组的维数。
    ```python
    print(arr.ndim) # 输出：2
    ```
*   **`T`**: 获取数组的转置（行变列，列变行）。
    ```python
    print(arr.T)
    # 输出：
    # [[1 4]
    #  [2 5]
    #  [3 6]]
    ```

## NumPy的数据类型（dtype）

NumPy有自己的一套数据类型，用于精确定义数组中元素的种类和精度。常见类型如下：

*   **布尔型**: `bool_`
*   **整型**: `int8`, `int16`, `int32`, `int64`
*   **无符号整型**: `uint8`, `uint16`, `uint32`, `uint64`
*   **浮点型**: `float16`, `float32`, `float64`

类型名中的数字（如`int32`中的32）通常表示该类型在内存中占用的**位数**。位数越多，能表示的数值范围越大或精度越高。例如，`int64`是64位整数，`float64`是双精度浮点数。

在大多数科学计算中，我们主要使用`int64`、`float64`等数字类型。

## 理解多维数组

![](img/b8e8e879f00b8767c8d875092cd04752_61.png)

*   **一维数组**：可以视为一条线上的点序列。
*   **二维数组**：可以视为一个表格或矩阵，具有行和列。
*   **三维数组**：可以想象为一本书，具有页、行和列。

![](img/b8e8e879f00b8767c8d875092cd04752_63.png)

例如，一个`shape`为`(2, 3, 4)`的三维数组，可以理解为2页，每页有3行4列的数据。
