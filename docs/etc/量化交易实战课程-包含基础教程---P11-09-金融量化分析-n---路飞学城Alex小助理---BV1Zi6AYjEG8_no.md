## 为什么要使用NumPy？

![](img/d190290694250565261b8307aa66b947_3.png)

![](img/d190290694250565261b8307aa66b947_5.png)

上一节我们介绍了NumPy是数据分析的基础包，本节中我们来看看它为何如此重要。NumPy最主要的功能是提供了一个名为`ndarray`的多维数组对象，它能高效地处理批量数据运算。

![](img/d190290694250565261b8307aa66b947_7.png)

![](img/d190290694250565261b8307aa66b947_9.png)

以下是两个直观的例子，展示了NumPy相比纯Python列表的优势：

**示例一：货币批量转换**
假设我们已知50家公司的市值（单位：美元），需要将所有市值转换为人民币（假设汇率为6.8）。

*   **使用纯Python列表的方法**：通常需要编写循环，遍历列表中的每个元素进行计算。
*   **使用NumPy的方法**：将列表转换为NumPy数组后，可以直接进行数组与标量的乘法运算。

```python
import numpy as np
import random

# 模拟50家公司的市值（美元），范围在100万到200万之间
a_list = [random.uniform(100, 200) for _ in range(50)]
print(“原始市值列表（美元）:”, a_list)

![](img/d190290694250565261b8307aa66b947_11.png)

![](img/d190290694250565261b8307aa66b947_13.png)

# 转换为NumPy数组
a_array = np.array(a_list)
# 批量转换为人民币
result = a_array * 6.8
print(“转换后市值数组（人民币）:”, result)
```

![](img/d190290694250565261b8307aa66b947_15.png)

![](img/d190290694250565261b8307aa66b947_17.png)

**示例二：计算购物车总金额**
假设我们有一个商品价格列表和一个对应商品数量的列表，需要计算总金额。

![](img/d190290694250565261b8307aa66b947_19.png)

![](img/d190290694250565261b8307aa66b947_21.png)

*   **使用NumPy的方法**：将两个列表转换为数组后，可以直接进行数组间的逐元素乘法，然后求和。

![](img/d190290694250565261b8307aa66b947_23.png)

```python
# 模拟50件商品的价格和数量
prices = [random.uniform(10, 20) for _ in range(50)]
quantities = [random.randint(1, 10) for _ in range(50)]

![](img/d190290694250565261b8307aa66b947_25.png)

# 转换为NumPy数组
prices_array = np.array(prices)
quantities_array = np.array(quantities)

![](img/d190290694250565261b8307aa66b947_27.png)

![](img/d190290694250565261b8307aa66b947_29.png)

# 计算每件商品的总价，然后求和
total = (prices_array * quantities_array).sum()
print(“购物车总金额为:”, total)
```

![](img/d190290694250565261b8307aa66b947_31.png)

通过以上例子可以看到，NumPy的数组运算**语法简洁**（无需显式循环）且**执行效率更高**，特别适合处理金融领域中的大规模数值计算。

## NumPy数组（ndarray）的核心概念

上一节我们通过例子感受了NumPy的便捷，本节中我们来深入了解一下其核心对象——`ndarray`数组。

![](img/d190290694250565261b8307aa66b947_35.png)

![](img/d190290694250565261b8307aa66b947_37.png)

### 创建数组
使用`np.array()`函数，传入一个列表或其他可迭代对象，即可创建NumPy数组。

![](img/d190290694250565261b8307aa66b947_39.png)

![](img/d190290694250565261b8307aa66b947_41.png)

![](img/d190290694250565261b8307aa66b947_43.png)

```python
import numpy as np
arr1 = np.array([1, 2, 3, 4, 5])  # 从列表创建
arr2 = np.array(range(10))        # 从range对象创建
print(arr1)
print(arr2)
```

![](img/d190290694250565261b8307aa66b947_45.png)

### 数组与列表的区别
理解数组与Python列表的区别至关重要：

![](img/d190290694250565261b8307aa66b947_47.png)

![](img/d190290694250565261b8307aa66b947_49.png)

1.  **元素类型必须相同**：列表可以容纳不同类型的元素（如整数、字符串混合），而数组要求所有元素必须是同一种数据类型（通常是数字）。
2.  **大小不可变**：列表可以动态地添加或删除元素（如使用`append`方法）。数组在创建后，其大小（即元素总数）是固定的，无法直接添加新元素。这是其实现高性能内存访问的基础。

## 数组的常用属性

![](img/d190290694250565261b8307aa66b947_51.png)

创建数组后，我们可以通过一系列属性来了解它的信息。以下是几个最常用的属性：

**`dtype`**
查看数组中元素的数据类型。NumPy有自己的一套数据类型体系，如`int64`, `float32`等，用于精确控制内存使用和计算精度。

![](img/d190290694250565261b8307aa66b947_53.png)

![](img/d190290694250565261b8307aa66b947_55.png)

```python
arr = np.array([1, 2, 3])
print(arr.dtype)  # 输出: int64 (在64位系统上)
```

![](img/d190290694250565261b8307aa66b947_57.png)

**`size`**
返回数组中元素的总个数。

![](img/d190290694250565261b8307aa66b947_59.png)

```python
arr = np.array([[1, 2, 3], [4, 5, 6]])
print(arr.size)   # 输出: 6
```

**`shape`**
返回一个元组，表示数组的“形状”，即每个维度上有多少个元素。这对于多维数组尤其重要。

```python
arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
print(arr_2d.shape)  # 输出: (2, 3)，表示2行3列

arr_3d = np.array([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])
print(arr_3d.shape)  # 输出: (2, 2, 2)，可以理解为2页，每页2行2列
```

**`ndim`**
返回数组的维数。

```python
arr_1d = np.array([1, 2, 3])
arr_2d = np.array([[1, 2], [3, 4]])
print(arr_1d.ndim)  # 输出: 1
print(arr_2d.ndim)  # 输出: 2
```

![](img/d190290694250565261b8307aa66b947_61.png)

![](img/d190290694250565261b8307aa66b947_63.png)

**`.T` (转置)**
对于二维数组，`.T`属性可以返回其转置矩阵（行变列，列变行）。这在处理矩阵运算或某些数据排列时非常有用。

```python
arr = np.array([[1, 2, 3], [4, 5, 6]])
print(“原始数组:\n”, arr)
print(“转置数组:\n”, arr.T)
# 输出:
# 原始数组:
#  [[1 2 3]
#   [4 5 6]]
# 转置数组:
#  [[1 4]
#   [2 5]
#   [3 6]]
```
