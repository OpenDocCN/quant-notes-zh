以下是昨日讲解的核心知识点：

*   **数据分析概述**：介绍了数据分析的发展方向以及数据分析工程师所需的基本技能。
*   **Excel基础**：讲解了Excel软件的发展历史，并详细介绍了工作簿、工作表、功能区、单元格等基本概念。

    *   快速选中连续区域：`Ctrl + Shift + 方向键（右/下）`
    *   快速定位到数据区域边界：`Ctrl + 方向键（上/下/左/右）`
*   **操作技巧**：学习了多项实用的数据处理技巧。

接下来，我们详细回顾一下昨日讲解的操作技巧。

以下是昨日重点讲解的Excel操作技巧列表：

1.  **定位**：使用 `Ctrl + G` 或 `F5` 键快速定位到特定单元格或区域。
2.  **选择性粘贴**：主要用于两个场景：
    *   对整列数据进行批量运算（如统一加薪）。
    *   将公式计算结果转换为静态数值。
3.  **查找与替换**：使用 `Ctrl + F` 进行查找，`Ctrl + H` 进行替换，并学习了通配符（`*` 匹配多个字符，`?` 匹配单个字符）的应用。
4.  **日期时间格式转换**：可以通过更智能的“分列”功能快速转换格式。
5.  **删除重复值**：快速识别并删除数据表中的重复记录。
6.  **分列与合并**：将一列数据按规则拆分为多列，或将多列数据合并为一列。
7.  **快速填充**：系统可根据已有数据模式自动填充，也支持用户自定义填充规则。

**关于日期格式转换的补充说明**：在“数据”选项卡中使用“分列”功能时，选择“分隔符号”，在第二步中无需进行任何操作，直接点击“下一步”，在第三步的“列数据格式”中选择“日期”，即可快速完成格式转换。

### 模块一：公式与函数 📊

![](img/5870b5481293d2136042ab988db99f62_14.png)

![](img/5870b5481293d2136042ab988db99f62_16.png)

公式与函数是Excel进行数据计算和分析的核心。我们将系统地学习以下内容：

![](img/5870b5481293d2136042ab988db99f62_18.png)

*   **基本概念**：理解公式与函数的定义与区别。
*   **运算符**：学习算术、比较、文本连接和引用运算符的使用。
*   **单元格引用**：掌握相对引用、绝对引用和混合引用的原理与应用。
*   **常用函数分类**：我们将学习以下几类常用函数：
    *   **逻辑函数**：如 `IF` 函数，用于条件判断。
    *   **文本函数**：如 `LEFT`, `RIGHT`, `MID`，用于处理文本字符串。
    *   **统计函数**：如 `SUM`, `AVERAGE`, `COUNT`，用于基础统计计算。
    *   **查找与引用函数**：如 `VLOOKUP`, `INDEX`, `MATCH`，用于数据查询与匹配。
    *   **日期函数**：如 `TODAY`, `DATEDIF`，用于处理日期数据。
