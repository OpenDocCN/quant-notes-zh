### 特征值与特征向量的定义与意义

![](img/fb068c453291535dcc8519d8854483fe_13.png)

![](img/fb068c453291535dcc8519d8854483fe_15.png)

对于一个给定的方阵 **A**，其特征值 **λ** 和特征向量 **x** 满足以下关系：
**A x = λ x**
这意味着，矩阵 **A** 作用于特征向量 **x** 时，仅对 **x** 进行缩放（系数为 **λ**），而不改变其方向。特征值和特征向量能够深刻刻画矩阵的本质。

![](img/fb068c453291535dcc8519d8854483fe_17.png)

![](img/fb068c453291535dcc8519d8854483fe_18.png)

![](img/fb068c453291535dcc8519d8854483fe_20.png)

以下是求解特征值与特征向量的基本步骤：
1.  根据定义建立方程：**A x = λ x**。
2.  移项得到：**(A - λ I) x = 0**，其中 **I** 是单位矩阵。
3.  要使该方程有非零解 **x**，矩阵 **(A - λ I)** 必须不可逆，即其行列式为零：**det(A - λ I) = 0**。
4.  解此关于 **λ** 的方程，得到 **N** 个特征值（可能包含重根）。
5.  将每个特征值 **λ_i** 代回方程 **(A - λ_i I) x = 0**，求解得到对应的特征向量 **x_i**。

![](img/fb068c453291535dcc8519d8854483fe_22.png)

### 对称矩阵与奇异值分解（SVD）

![](img/fb068c453291535dcc8519d8854483fe_24.png)

![](img/fb068c453291535dcc8519d8854483fe_25.png)

在实际应用中，我们经常遇到一类特殊的矩阵——对称矩阵（**A = A^T**）。对称矩阵具有两个优美性质：
1.  所有特征值都是实数。
2.  不同特征值对应的特征向量互相正交。

![](img/fb068c453291535dcc8519d8854483fe_27.png)

![](img/fb068c453291535dcc8519d8854483fe_29.png)

![](img/fb068c453291535dcc8519d8854483fe_30.png)

对于对称矩阵 **A**，我们可以进行谱分解：
**A = Q Σ Q^T**
其中，**Q** 是由特征向量组成的正交矩阵（**Q^T Q = I**），**Σ** 是由特征值组成的对角矩阵。

![](img/fb068c453291535dcc8519d8854483fe_32.png)

对于更一般的任意矩阵 **A**（不一定是方阵或对称阵），我们可以进行奇异值分解（SVD）：
**A = U Σ V^T**
其中，**U** 和 **V** 都是正交矩阵，**Σ** 是对角矩阵，其对角线元素称为奇异值。SVD 是数据分析中的强大工具，主成分分析（PCA）即基于此。

![](img/fb068c453291535dcc8519d8854483fe_34.png)

![](img/fb068c453291535dcc8519d8854483fe_35.png)

## 基于特征值的资产组合优化理论

![](img/fb068c453291535dcc8519d8854483fe_39.png)

理解了矩阵分解的基础后，我们将其应用于金融领域。资产组合优化的核心问题之一是：给定一组资产，如何分配资金以最小化投资组合的整体风险？

![](img/fb068c453291535dcc8519d8854483fe_45.png)

### 问题建模：最小化投资组合风险

假设我们有 **N** 个资产。在时刻 **t**，我们能观察到每个资产的收益率，构成一个 **N** 维向量。通过历史数据，我们可以计算出这 **N** 个资产收益率的协方差矩阵 **Σ**，它衡量了资产之间的联动风险和各自的波动风险。

![](img/fb068c453291535dcc8519d8854483fe_48.png)

我们的目标是确定一个权重向量 **w = (w_1, w_2, ..., w_N)^T**，其中 **w_i** 表示分配给第 **i** 个资产的资金比例，且满足 **∑ w_i = 1**。投资组合的风险（方差）可以用以下公式衡量：
**风险 = w^T Σ w**

![](img/fb068c453291535dcc8519d8854483fe_50.png)

![](img/fb068c453291535dcc8519d8854483fe_52.png)

![](img/fb068c453291535dcc8519d8854483fe_53.png)

![](img/fb068c453291535dcc8519d8854483fe_54.png)

![](img/fb068c453291535dcc8519d8854483fe_55.png)

因此，我们的优化问题转化为：在约束 **∑ w_i = 1** 下，寻找 **w** 使得 **w^T Σ w** 最小化。

![](img/fb068c453291535dcc8519d8854483fe_57.png)

![](img/fb068c453291535dcc8519d8854483fe_58.png)

![](img/fb068c453291535dcc8519d8854483fe_60.png)

![](img/fb068c453291535dcc8519d8854483fe_62.png)

![](img/fb068c453291535dcc8519d8854483fe_64.png)

### 解决方案：特征组合

一个优美而重要的结论是，上述最小化问题的解与协方差矩阵 **Σ** 的特征系统密切相关。具体而言：
*   使投资组合风险最小的权重向量 **w**，近似对应于协方差矩阵 **Σ** 的最小特征值所对应的特征向量（需经过归一化处理，使权重之和为1）。
*   此时的最小风险值，近似等于那个最小的特征值。

以下是基于特征值的资产配置步骤：
1.  计算资产收益率的协方差矩阵 **Σ**。
2.  对 **Σ** 进行特征分解，得到特征值 **λ_1 ≤ λ_2 ≤ ... ≤ λ_N** 和对应的特征向量。
3.  选取最小特征值 **λ_1** 对应的特征向量 **v_1**。
4.  对 **v_1** 进行归一化（使其各分量之和为1），得到的向量即为最优权重向量 **w***。

![](img/fb068c453291535dcc8519d8854483fe_66.png)

![](img/fb068c453291535dcc8519d8854483fe_67.png)

![](img/fb068c453291535dcc8519d8854483fe_68.png)

![](img/fb068c453291535dcc8519d8854483fe_70.png)

**注意**：在实践中，最小特征值对应的方向往往代表市场整体方向，与大盘指数相似。为了获得超越市场的收益，许多策略会选择第二小或后续特征值对应的特征组合。

### 方法二：最小方差组合

![](img/fb068c453291535dcc8519d8854483fe_77.png)

![](img/fb068c453291535dcc8519d8854483fe_78.png)

![](img/fb068c453291535dcc8519d8854483fe_80.png)

![](img/fb068c453291535dcc8519d8854483fe_82.png)

我们也可以通过求解带约束的优化问题来直接得到权重。即最小化 **w^T Σ w**，并满足 **∑ w_i = 1**。利用拉格朗日乘子法，可以得到解析解：
**w* = (Σ^{-1} 1) / (1^T Σ^{-1} 1)**
其中 **1** 是元素全为1的向量。这种方法得到的组合称为“全局最小方差组合”。

随机矩阵理论研究了当矩阵元素完全随机时（例如，从某个分布中独立抽取），其特征值的分布规律。对于由纯噪声构成的协方差矩阵，其特征值会服从一个特定的分布（如Marchenko-Pastur分布）。

### 滤波算法步骤

![](img/fb068c453291535dcc8519d8854483fe_124.png)

![](img/fb068c453291535dcc8519d8854483fe_126.png)

我们可以通过对比实际协方差矩阵的特征值分布与随机矩阵理论预测的分布，来识别并滤除噪声成分：

1.  **特征分解**：计算经验协方差矩阵 **Σ** 的特征值 **{λ_i}** 和特征向量。
2.  **噪声识别**：根据随机矩阵理论（如Marchenko-Pastur分布），确定一个特征值区间，落在此区间内的特征值很可能主要源于随机噪声。
3.  **矩阵滤波**：将这些被认为是噪声的特征值置零（或收缩），同时保留其他特征值。然后利用滤波后的特征值和原来的特征向量重构一个新的协方差矩阵 **Σ_filtered**。
    **Σ_filtered = Q · diag(λ‘_1, λ’_2, ..., λ‘_N) · Q^T**
    其中，**λ‘_i** 是滤波后的特征值（噪声特征值被设为0或缩小），**Q** 是特征向量矩阵。
4.  **优化求解**：使用滤波后的协方差矩阵 **Σ_filtered** 代替原始的 **Σ**，代入前述的特征组合法或最小方差法公式中，计算最终的资产权重 **w**。

![](img/fb068c453291535dcc8519d8854483fe_128.png)

![](img/fb068c453291535dcc8519d8854483fe_130.png)

![](img/fb068c453291535dcc8519d8854483fe_132.png)

这种方法通过去除随机噪声，有望得到更稳健、更反映真实信号的风险估计和资产配置方案。

**核心内容总结**：
1.  **数学基础**：特征值/特征向量和奇异值分解（SVD）是理解数据结构和进行降维、去噪的基石。
2.  **资产配置理论**：马科维茨的现代投资组合理论告诉我们，可以通过优化权重来平衡风险与收益。最小化风险的问题与协方差矩阵的特征系统有直接联系。
3.  **实践方法**：我们掌握了两种求解最优权重的方法——基于最小特征值的特征组合法，以及基于优化理论的最小方差组合法。
4.  **高级技巧**：引入随机矩阵理论对协方差矩阵进行滤波，可以有效地减少估计误差，提升策略的稳健性。
5.  **论文研读**：初步了解了如何快速阅读和批判性分析相关领域的研究论文。
