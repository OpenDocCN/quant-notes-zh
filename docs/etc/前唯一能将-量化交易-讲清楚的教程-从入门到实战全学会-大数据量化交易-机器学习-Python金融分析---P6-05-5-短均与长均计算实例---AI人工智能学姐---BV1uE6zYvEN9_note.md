## 概述

![](img/128675725bd11a14f444fb5826f2b3e8_7.png)

![](img/128675725bd11a14f444fb5826f2b3e8_9.png)

移动平均线是量化交易中常用的技术指标。通过比较短期和长期移动平均线的相对位置，我们可以判断市场的趋势变化。当短期均线从下方上穿长期均线时，形成“黄金交叉”，通常被视为买入信号。反之，当短期均线从上方下穿长期均线时，形成“死亡交叉”，通常被视为卖出信号。

上一节我们介绍了数据获取与处理的基础知识，本节中我们来看看如何具体计算这些指标并应用于分析。

## 计算短期与长期移动平均线

![](img/128675725bd11a14f444fb5826f2b3e8_11.png)

首先，我们需要计算两个关键指标：短期移动平均线和长期移动平均线。我们以苹果公司（AAPL）的股票数据为例。

![](img/128675725bd11a14f444fb5826f2b3e8_13.png)

![](img/128675725bd11a14f444fb5826f2b3e8_15.png)

![](img/128675725bd11a14f444fb5826f2b3e8_17.png)

以下是计算步骤：

1.  **选择数据**：我们使用苹果公司的历史股价数据。
2.  **定义窗口**：短期均线通常使用较小的窗口（如5天、10天），长期均线使用较大的窗口（如20天、30天）。由于我们的数据时间跨度较长，为了图表更清晰，我们使用10天作为短期窗口，30天作为长期窗口。
3.  **进行计算**：使用Pandas的`.rolling()`和`.mean()`方法计算移动平均值。

![](img/128675725bd11a14f444fb5826f2b3e8_19.png)

![](img/128675725bd11a14f444fb5826f2b3e8_21.png)

核心计算代码如下：

![](img/128675725bd11a14f444fb5826f2b3e8_23.png)

![](img/128675725bd11a14f444fb5826f2b3e8_25.png)

```python
# 假设 `df` 是包含苹果股价的DataFrame，其中‘Close’列为收盘价
# 计算10日短期移动平均线
df[‘M1’] = df[‘Close’].rolling(window=10).mean()
# 计算30日长期移动平均线
df[‘M2’] = df[‘Close’].rolling(window=30).mean()
```

![](img/128675725bd11a14f444fb5826f2b3e8_27.png)

## 绘制均线图进行分析

计算完成后，我们可以将股价、短期均线和长期均线绘制在同一张图上进行观察。

![](img/128675725bd11a14f444fb5826f2b3e8_29.png)

以下是绘图与分析过程：

![](img/128675725bd11a14f444fb5826f2b3e8_31.png)

![](img/128675725bd11a14f444fb5826f2b3e8_33.png)

1.  **绘制图表**：使用Matplotlib或Pandas内置的绘图功能，将三条线绘制出来。
2.  **识别交叉点**：在图表上，我们可以清晰地看到短期均线（M1）与长期均线（M2）相交的位置。
3.  **判断信号**：
    *   **死亡交叉**：当短期均线从上方向下穿越长期均线时，表明短期趋势转弱，可能预示下跌，是卖出信号。
    *   **黄金交叉**：当短期均线从下方向上穿越长期均线时，表明短期趋势转强，可能预示上涨，是买入信号。

## 使用数值信号增强分析

![](img/128675725bd11a14f444fb5826f2b3e8_41.png)

为了更精确、自动化地识别交叉信号，我们可以创建一个数值信号列。这个信号列明确标记出短期均线是高于还是低于长期均线。

![](img/128675725bd11a14f444fb5826f2b3e8_43.png)

以下是创建信号列的步骤：

![](img/128675725bd11a14f444fb5826f2b3e8_45.png)

![](img/128675725bd11a14f444fb5826f2b3e8_47.png)

1.  **定义信号规则**：我们设定当短期均线（M1）大于长期均线（M2）时，信号为`1`；反之，信号为`-1`。
2.  **使用NumPy计算**：`np.where(condition, value_if_true, value_if_false)`函数非常适合完成这个任务。

![](img/128675725bd11a14f444fb5826f2b3e8_49.png)

核心代码如下：

![](img/128675725bd11a14f444fb5826f2b3e8_51.png)

![](img/128675725bd11a14f444fb5826f2b3e8_53.png)

```python
import numpy as np
# 创建信号列
df[‘position’] = np.where(df[‘M1’] > df[‘M2’], 1, -1)
```

![](img/128675725bd11a14f444fb5826f2b3e8_55.png)

3.  **双坐标轴绘图**：由于信号值（1和-1）与股价数值范围差异巨大，为了在同一张图上清晰展示，我们使用双坐标轴。主坐标轴（左侧）显示股价和均线，副坐标轴（右侧）显示信号线。

```python
import matplotlib.pyplot as plt
fig, ax1 = plt.subplots(figsize=(12,6))
# 在主坐标轴上绘制股价和均线
ax1.plot(df[‘Close’], label=‘Price’, alpha=0.5)
ax1.plot(df[‘M1’], label=‘M1 (10-day)’)
ax1.plot(df[‘M2’], label=‘M2 (30-day)’)
ax1.set_ylabel(‘Price’)
ax1.legend(loc=‘upper left’)
# 创建副坐标轴
ax2 = ax1.twinx()
# 在副坐标轴上绘制信号线
ax2.plot(df[‘position’], label=‘Signal’, color=‘black’, linestyle=‘--’)
ax2.set_ylabel(‘Signal (1/-1)’)
ax2.legend(loc=‘upper right’)
plt.show()
```

![](img/128675725bd11a14f444fb5826f2b3e8_57.png)

![](img/128675725bd11a14f444fb5826f2b3e8_59.png)

在最终的图表中，黑色的信号线在`1`和`-1`之间切换。信号线从`-1`变为`1`的转折点，对应着“黄金交叉”；从`1`变为`-1`的转折点，则对应着“死亡交叉”。这使得交易信号的识别变得一目了然。

![](img/128675725bd11a14f444fb5826f2b3e8_61.png)
