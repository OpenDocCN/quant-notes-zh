def before_trading(context):

## 实现交易逻辑

在`handle_bar`函数中，我们需要根据每日筛选出的股票池执行实际的买卖操作。逻辑分为两部分：卖出不再持有的股票，买入新入选的股票。

![](img/0e65afff204ee77299559593b94a5cb1_55.png)

![](img/0e65afff204ee77299559593b94a5cb1_57.png)

首先，我们需要判断当前账户的持仓情况。可以通过`context.portfolio.positions`获取一个包含所有持仓信息的字典。

![](img/0e65afff204ee77299559593b94a5cb1_59.png)

![](img/0e65afff204ee77299559593b94a5cb1_61.png)

以下是交易逻辑的实现步骤：

![](img/0e65afff204ee77299559593b94a5cb1_63.png)

![](img/0e65afff204ee77299559593b94a5cb1_65.png)

![](img/0e65afff204ee77299559593b94a5cb1_67.png)

1.  **判断持仓状态**：检查当前持仓字典是否为空。若为空，则为策略运行第一天，执行初始建仓。
2.  **卖出操作**：遍历当前持仓。如果某只股票不在当日最新的股票池中，则将其全部卖出。
3.  **买入操作**：遍历当日股票池。如果某只股票不在当前持仓中，则进行买入。

![](img/0e65afff204ee77299559593b94a5cb1_69.png)

这里我们使用`order_target_percent`函数来下单，它可以指定交易后该股票资产占总资产的目标比例。

![](img/0e65afff204ee77299559593b94a5cb1_71.png)

```python
def handle_bar(context, bar_dict):
    # 获取当前持仓和当日股票池
    current_positions = context.portfolio.positions
    target_pool = context.portfolio

    # 卖出逻辑：持仓中有，但目标池中没有的股票
    for stock in current_positions.keys():
        if stock not in target_pool:
            # 卖出全部持仓，目标比例设为0
            order_target_percent(stock, 0)

    # 买入逻辑：目标池中有，但持仓中没有或不足的股票
    # 这里采用等权分配的方式买入
    if len(target_pool) > 0:
        weight = 1.0 / len(target_pool)  # 计算每只股票应占的比例
        for stock in target_pool:
            # 买入至目标权重
            order_target_percent(stock, weight)
```

## 运行回测与结果分析

![](img/0e65afff204ee77299559593b94a5cb1_75.png)

完成代码后，我们可以运行回测。系统会生成一份详细的报告，其中包含多个评估策略性能的指标。

![](img/0e65afff204ee77299559593b94a5cb1_77.png)

![](img/0e65afff204ee77299559593b94a5cb1_79.png)

主要关注以下四个核心指标：

![](img/0e65afff204ee77299559593b94a5cb1_81.png)

![](img/0e65afff204ee77299559593b94a5cb1_83.png)

1.  **回测年化收益**：我们的策略在整个回测期间获得的年化收益率。
2.  **基准年化收益**：同期简单持有基准（如沪深300指数）所能获得的年化收益率。对比策略收益与基准收益，是衡量策略有效性的关键。
3.  **最大回撤**：策略净值从最高点到最低点的最大跌幅。这个指标反映了策略可能面临的最大亏损风险，数值越小越好。
4.  **夏普比率**：衡量策略承担单位风险所获得的超额回报。通常为正数且越大越好。如果策略收益为负，夏普比率也可能为负。

![](img/0e65afff204ee77299559593b94a5cb1_85.png)

![](img/0e65afff204ee77299559593b94a5cb1_87.png)

报告还会提供收益曲线图、交易详情列表等。通过交易详情，可以查看每一天的具体买卖操作。
