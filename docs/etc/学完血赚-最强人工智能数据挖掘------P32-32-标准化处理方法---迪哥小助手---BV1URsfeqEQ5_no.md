# 人工智能数据挖掘实战：P32：标准化处理方法 📊

在本节课中，我们将要学习数据预处理中一个至关重要的步骤——标准化。标准化能够消除不同特征（指标）之间由于量纲和取值范围不同带来的影响，使数据更适合进行后续的机器学习建模。

## 为什么需要标准化？🤔

![](img/b75a0b923a112ff8e90c173a5bb20ff7_1.png)

![](img/b75a0b923a112ff8e90c173a5bb20ff7_2.png)

上一节我们介绍了数据预处理的重要性，本节中我们来看看一个具体且常用的方法：标准化。

假设我们有一组数据，包含两个特征 X1 和 X2。观察下图，我们可以发现一个问题：X1 和 X2 的取值范围是不同的。X1 的取值范围较大，而 X2 的取值范围较小。

![](img/b75a0b923a112ff8e90c173a5bb20ff7_1.png)
![](img/b75a0b923a112ff8e90c173a5bb20ff7_2.png)

在机器学习中，如果特征的取值范围差异很大，某些算法（如基于距离的KNN、SVM，或使用梯度下降的模型）可能会赋予取值范围大的特征更高的权重，从而影响模型的准确性和性能。标准化的核心目的，就是为了让所有特征处于**相同的尺度**上，以便我们能用公平的眼光去比较和分析它们。

## 标准化的原理与公式 📐

标准化通常通过一个简单的数学变换来实现，其目标是使数据符合均值为0、标准差为1的标准正态分布。

标准化的计算公式如下：

**`z = (x - μ) / σ`**

其中：
*   **`z`** 是标准化后的值。
*   **`x`** 是原始数据值。
*   **`μ`** 是该特征所有数据的**均值**。
*   **`σ`** 是该特征所有数据的**标准差**。

这个公式可以分解为两个步骤来理解：

1.  **去均值（`x - μ`）**：将每个数据点减去其所在特征的平均值。这一步的目的是**将数据的中心移动到0点**，使数据以原点为中心对称分布。
2.  **除标准差（`/ σ`）**：将去均值后的数据除以其标准差。这一步的目的是**调整数据的离散程度**。

为了理解“除标准差”的作用，我们可以看一个例子：假设有两组人，第一组是工资相近的普通员工，他们的工资标准差很小；第二组包含了普通员工和亿万富翁，他们的工资标准差非常大。对于取值范围本身较小的第一组数据，除以一个小的标准差，其尺度变化不大；对于取值范围巨大的第二组数据，除以一个大的标准差，能显著缩小其数值范围。最终，两组数据经过处理后，其取值范围会变得近似。

通过这两步操作，数据不仅以0为中心对称，而且各个特征的数值范围也被调整到相近的水平。

## 标准化的代码实现 💻

理解了原理后，我们可以自己动手实现标准化函数，也可以使用现成的工具库。

![](img/b75a0b923a112ff8e90c173a5bb20ff7_4.png)

![](img/b75a0b923a112ff8e90c173a5bb20ff7_5.png)

以下是手动实现标准化的Python代码示例：

```python
def standardize(data):
    """
    对输入的数据进行标准化处理。
    参数:
        data: 一个数值列表或NumPy数组。
    返回:
        标准化后的数据。
    """
    # 计算数据的均值和标准差
    mean_val = np.mean(data)
    std_val = np.std(data)
    # 应用标准化公式: (x - mean) / std
    standardized_data = (data - mean_val) / std_val
    return standardized_data
```

![](img/b75a0b923a112ff8e90c173a5bb20ff7_7.png)

代码非常简单：首先计算数据的均值和标准差，然后对每个数据点应用公式 `(x - mean) / std` 即可。

![](img/b75a0b923a112ff8e90c173a5bb20ff7_8.png)

![](img/b75a0b923a112ff8e90c173a5bb20ff7_9.png)

在实际项目中，我们更常使用成熟的机器学习库。以下是使用 `scikit-learn` 库进行标准化的方法：

```python
from sklearn.preprocessing import StandardScaler

# 创建StandardScaler对象
scaler = StandardScaler()
# 拟合数据并计算均值和标准差
scaler.fit(data)
# 转换数据
standardized_data = scaler.transform(data)
# 或者一步完成拟合和转换
# standardized_data = scaler.fit_transform(data)
```

使用 `scikit-learn` 的优势在于其接口统一，能轻松处理训练集和测试集（需用训练集的参数转换测试集），并且集成在完整的数据处理流程中。

## 总结 📝

![](img/b75a0b923a112ff8e90c173a5bb20ff7_21.png)

![](img/b75a0b923a112ff8e90c173a5bb20ff7_22.png)

本节课中我们一起学习了数据标准化处理。
*   **目的**：消除不同特征间量纲和取值范围的差异，使数据处于同一尺度，提升模型性能。
*   **原理**：通过公式 **`z = (x - μ) / σ`**，先减去均值使数据中心化，再除以标准差调整数据离散度。
*   **实现**：可以手动计算实现，也可以便捷地使用 `sklearn.preprocessing.StandardScaler` 工具。
*   **应用**：标准化是数据挖掘和机器学习中极为常用的预处理步骤，对于许多算法来说是必不可少的。
