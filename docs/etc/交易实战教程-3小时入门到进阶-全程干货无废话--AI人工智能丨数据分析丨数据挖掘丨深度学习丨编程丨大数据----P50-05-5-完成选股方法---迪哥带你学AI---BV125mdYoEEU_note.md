在本节课中，我们将学习如何将多个股票因子评分表进行合并，并计算综合得分以筛选出排名前十的股票。这是构建量化选股策略的关键一步。

![](img/a26db05aa64bf4371e6b18c5d0954344_2.png)

## 数据表的拼接操作

上一节我们介绍了如何为单个因子数据打分。本节中，我们来看看如何将多个因子的评分结果整合到一起。

首先，我们需要将两个包含不同因子评分的数据表（DataFrame）进行拼接。在Pandas中，这可以通过`concat`操作轻松完成。

以下是拼接两个DataFrame的代码：

```python
# 假设 df1 和 df2 是两个已经包含因子评分的数据表
df_combined = pd.concat([df1, df2], axis=1)
```

![](img/a26db05aa64bf4371e6b18c5d0954344_4.png)

执行上述代码后，`df_combined` 将成为一个合并后的大表，其中包含了来自 `df1` 和 `df2` 的所有列。

## 计算综合得分并排序

现在我们已经有了一个包含所有因子评分的大表，但是还没有一个综合的排名信息。接下来，我们需要计算每只股票的综合得分并进行排序。

以下是计算总分并排序的步骤：

1.  **新建得分列**：首先，在合并后的DataFrame中新建一列，用于存放综合得分。初始值可以设为0。
    ```python
    df_combined[‘score’] = 0
    ```

![](img/a26db05aa64bf4371e6b18c5d0954344_6.png)

2.  **计算总分**：将多个因子的评分相加，得到每只股票的总分。假设我们有6个因子列。
    ```python
    # 对指定的因子列进行横向求和
    factor_columns = [‘factor1’, ‘factor2’, ‘factor3’, ‘factor4’, ‘factor5’, ‘factor6’]
    df_combined[‘score’] = df_combined[factor_columns].sum(axis=1)
    ```

![](img/a26db05aa64bf4371e6b18c5d0954344_8.png)

3.  **按总分排序**：根据计算出的`score`列进行降序排列，分数最高的股票排在最前面。
    ```python
    df_sorted = df_combined.sort_values(by=‘score’, ascending=False)
    ```

## 获取最终选股结果

排序完成后，我们的目标就变成了从排序后的列表中提取出排名前十的股票代码。

以下是获取前十名股票列表的代码：

```python
# 获取排序后DataFrame的索引（即股票代码），并转换为列表
top_10_stocks = df_sorted.index.tolist()[:10]
```

![](img/a26db05aa64bf4371e6b18c5d0954344_10.png)

至此，`top_10_stocks` 这个列表就包含了根据当前所有因子综合评分选出的前十名股票。

![](img/a26db05aa64bf4371e6b18c5d0954344_12.png)

## 函数封装与流程总结

![](img/a26db05aa64bf4371e6b18c5d0954344_14.png)

我们可以将以上所有步骤封装到一个函数中，例如 `get_top_stocks()`。

![](img/a26db05aa64bf4371e6b18c5d0954344_16.png)

该函数的主要流程可以概括为：
1.  读取各因子数据并进行打分。
2.  将所有因子的评分表拼接成一个总表。
3.  在总表中计算每只股票的综合得分。
4.  根据综合得分进行降序排序。
5.  返回排名前十的股票代码列表。

![](img/a26db05aa64bf4371e6b18c5d0954344_18.png)

每次调用这个函数，它都会基于最新的因子数据，计算出当前时刻综合排名前十的股票，为我们的交易决策提供依据。

![](img/a26db05aa64bf4371e6b18c5d0954344_20.png)

本节课中我们一起学习了量化选股的核心步骤：**多因子数据合并、综合得分计算与排序筛选**。通过将不同维度的因子评价整合并量化，我们能够系统化、自动化地筛选出具有潜力的投资标的，这是构建稳健量化策略的重要基础。
