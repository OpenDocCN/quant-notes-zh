# 量化交易教程：P5：04 金融量化分析-影响股价因素与股票买卖知识 📈

在本节课中，我们将要学习影响股票价格的主要因素，并了解股票买卖的基本流程与规则。理解这些基础知识是进行金融量化分析的第一步。

## 影响股价的六大因素

上一节我们介绍了股票的基本概念，本节中我们来看看哪些因素会影响股票价格的波动。影响股价的因素可以归纳为以下六个方面。

### 1. 公司自身因素
这是影响股价最根本的因素。公司的经营状况直接决定了其长期价值。例如，公司首次公开募股（IPO）时，会根据其市值确定发行价格。假设一家公司市值为50亿人民币，发行1亿股，则每股发行价为5元。公式表示为：
**发行价 = 公司总市值 / 发行总股数**
此后，若公司经营良好，市值增长至70亿，则每股价格理论上会相应上涨。反之，若公司出现重大负面事件（如创始人丑闻），股价则可能下跌。

### 2. 市场因素
这是影响股价最直接的因素。股价的短期波动主要由市场供求关系决定。当买股票的人多于卖股票的人时，供不应求，股价上涨；反之，当卖的人多于买的人时，供过于求，股价下跌。这类似于任何商品的市场规律。

### 3. 行业因素
整个行业的发展前景会影响行业内所有公司的股价。例如，如果人工智能行业前景被普遍看好，那么相关公司的股票可能集体上涨。反之，如果一个行业被认为正在衰落，其相关公司的股价则可能普遍下跌。

### 4. 心理因素
投资者的心理和情绪，尤其是从众心理，会对股价产生非理性影响。例如，当市场出现恐慌性抛售时，即使公司基本面没有恶化，股价也可能因投资者的集体不理智行为而暴跌。历史上著名的“黑色星期五”股灾就与此类因素有关。

### 5. 经济因素
国家层面的宏观经济政策和指标会影响股市整体表现。例如，当银行存款利率上升时，可能吸引资金从股市流向银行，导致市场资金减少，从而可能引发股价下跌。其他如外汇政策、存款准备金率等也属于此类因素。

### 6. 政治因素
国际或国内的政治局势、军事冲突风险等事件会严重影响市场信心和股价。例如，当两个国家关系紧张、爆发军事冲突的风险升高时，相关国家的股市可能下跌，但军工类股票可能因预期需求增加而上涨。

## 股票买卖流程与规则

了解了影响股价的因素后，我们来看看个人投资者如何进行股票买卖，以及需要遵守哪些市场规则。

以下是股票买卖的主要步骤与关键概念：

1.  **开户与委托**
    个人不能直接进入交易所买卖股票，必须通过证券公司（券商）开户。开户后，投资者通过券商的系统提交买卖指令，这个过程称为“委托”。

2.  **股票交易日**
    股票交易所并非全天候营业。交易日通常为非法定节假日的周一至周五。在节假日，交易所同样休市。

3.  **交易时间**
    在每个交易日内，交易也并非24小时进行。以中国A股市场为例，主要时间阶段如下：
    *   **开盘集合竞价**（9:15 - 9:25）：此期间接受买卖委托，但不立即成交。交易所会在9:25一次性对之前所有委托进行撮合，以产生当日的**开盘价**。其核心原则是使成交量最大化。
    *   **连续竞价**（9:30 - 11:30, 13:00 - 14:57）：这是主要的交易时段。交易所系统每隔几秒就会对当前的买卖委托进行一次撮合，成交是连续进行的。
    *   **收盘集合竞价**（深市：14:57 - 15:00）：仅深圳交易所有此阶段。此期间提交的委托不会立即撮合，而是在15:00统一撮合，以产生**收盘价**。上海交易所的收盘价则为连续竞价阶段的最后一笔成交价。

4.  **交易制度**
    *   **T+1制度**：指当日（T日）买入的股票，要到下一个交易日（T+1日）才能卖出。卖出股票获得的资金，当日可以用于买入其他股票，但需到下一交易日才能从证券账户转出到银行账户。
    *   **涨跌停板限制**：为防止股价过度波动，A股市场设有涨跌幅限制。普通股票在一个交易日内的价格波动，不得超过前一交易日收盘价的±10%（ST等特殊股票限制为±5%）。
