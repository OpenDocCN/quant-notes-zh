# 量化交易与Python金融分析实战课程：P44：45. IC指标值计算

在本节课中，我们将学习如何计算IC（信息系数）指标值。IC值是衡量选股因子预测能力的关键指标，它通过计算因子值与未来收益率之间的相关性来评估因子的有效性。我们将从获取收盘价数据开始，进行数据格式转换，最终完成IC值的计算。

## 获取收盘价数据

上一节我们介绍了因子的计算，本节中我们来看看如何获取计算IC值所需的实际收益率数据。计算收益率需要股票的每日收盘价。

以下是获取收盘价数据的步骤：

![](img/a70843cce1f62ec4c2329af99d309034_1.png)

*   使用 `get_price` 函数获取指定股票池在特定时间段内的价格数据。
*   指定股票代码列表、起始日期和结束日期。
*   从返回的多维价格数据中，提取出我们需要的“收盘价”数据，并将其转换为二维的DataFrame格式，以便于后续处理。

```python
# 获取收盘价数据示例
price = get_price(stock_list, start_date='2019-01-01', end_date='2020-01-01')
# 提取收盘价并整理格式
close_price = price['close'].unstack()
close_price.index.name = 'date'
close_price.columns.name = 'code'
```

![](img/a70843cce1f62ec4c2329af99d309034_3.png)

## 数据格式转换

获取到因子数据和收盘价数据后，我们需要将它们转换为计算IC值所需的特定格式。这个转换过程通常由一个工具函数完成。

以下是进行数据格式转换的说明：

*   需要准备两个参数：处理好的因子数据和处理好的收盘价数据。
*   调用专用的格式转换函数，将原始数据重组为包含日期、股票代码、未来多期收益率、因子值及因子分组的新数据结构。
*   在这个新结构中，因子值会根据其数值大小被自动划分到不同的分组中（例如分为5组），分组编号越小代表因子值越小。

```python
# 数据格式转换示例
from utility import convert_to_forward_returns_data
formatted_data = convert_to_forward_returns_data(factor_data=processed_factor_data,
                                                 prices=close_price)
```

## 计算IC指标值

数据准备就绪后，我们就可以计算核心的IC指标值了。IC值本质上是因子值与未来一期（或其他期限）收益率之间的相关系数。

![](img/a70843cce1f62ec4c2329af99d309034_5.png)

![](img/a70843cce1f62ec4c2329af99d309034_7.png)

以下是计算IC值的步骤：

![](img/a70843cce1f62ec4c2329af99d309034_9.png)

*   使用量化分析库中提供的IC计算函数。
*   将上一步格式化好的数据作为参数传入该函数。
*   函数会返回一个包含每日IC值的结果序列。

![](img/a70843cce1f62ec4c2329af99d309034_11.png)

```python
# 计算IC值示例
from alphalens.performance import factor_information_coefficient
ic_series = factor_information_coefficient(formatted_data)
print(ic_series.head())
```

![](img/a70843cce1f62ec4c2329af99d309034_13.png)

本节课中我们一起学习了IC指标值的完整计算流程。我们从获取收盘价数据出发，经历了必要的数据格式转换，最终利用专用函数计算出了衡量因子预测能力的IC值。理解并掌握IC值的计算，是评估和筛选有效量化因子的基础。
