## 什么是聚类？🤔

上一节我们介绍了机器学习中的有监督学习，其特点是拥有带标签的数据。本节中我们来看看聚类，它属于无监督学习。

聚类要解决的问题是：我们手中没有每个数据属于哪个类别的标签。聚类算法的目标，是将相似的数据点自动归为一组。例如，观察右侧的散点图，原始数据并没有颜色标记。聚类算法会根据数据点之间的相似度，自动将它们分成不同的组（如绿色、红色、蓝色三组）。

![](img/9a4f185f9b9e36aec1010105658da246_2.png)

聚类算法从原理上看相对简单，但它面临两大主要挑战：
1.  **评估困难**：由于没有真实标签，很难客观评价聚类结果的好坏。
2.  **参数调节困难**：例如，改变聚类数量（K值）会得到不同结果，但缺乏标准答案来判断哪个结果更优。

## K-Means算法核心概念 🔑

![](img/9a4f185f9b9e36aec1010105658da246_12.png)

K-Means算法是聚类算法中最简单、最常用的一种。它的核心思想是“物以类聚”，通过迭代寻找数据的最佳分组。

![](img/9a4f185f9b9e36aec1010105658da246_14.png)

以下是K-Means算法的几个核心概念：

### 1. K值（聚类数量）
K-Means算法需要我们预先指定一个参数：**K值**。K值代表我们希望将数据聚合成多少个簇（组）。
*   如果设定 `K=3`，算法会将数据分成3堆。
*   如果设定 `K=4`，则分成4堆。
**公式表示**：`K` 是一个需要用户定义的超参数。

### 2. 质心
每个簇都有一个中心点，称为**质心**。质心是该簇内所有数据点在各个维度上的平均值。
*   对于一个二维数据点 `(x, y)`，其所在簇的质心坐标为 `(mean(x), mean(y))`。
**代码描述**：对于一个包含 `n` 个样本的簇，其质心坐标计算为：
`质心坐标 = (sum(所有样本的X坐标)/n, sum(所有样本的Y坐标)/n)`

### 3. 距离度量
要判断两个数据点是否相似，需要计算它们之间的距离。最常用的距离是**欧几里得距离**（简称欧式距离）。
*   对于二维空间中的点 `A(x1, y1)` 和 `B(x2, y2)`，其欧式距离为：`sqrt((x1-x2)^2 + (y1-y2)^2)`。
**重要提示**：在使用距离度量前，通常需要对数据进行**标准化**（如归一化），以确保不同特征（如X轴和Y轴）的数值范围一致，避免某个特征因数值过大而主导距离计算。

![](img/9a4f185f9b9e36aec1010105658da246_16.png)

### 4. 优化目标
K-Means算法的目标是：让**每个簇内的数据点尽可能靠近该簇的质心**，同时让**不同簇的质心尽可能远离**。
其优化的目标函数（也称为惯性或簇内平方和）可以表示为：
**`最小化 J = Σ（i=1 到 K） Σ（x 属于簇Ci） || x - μi ||^2`**
其中：
*   `K` 是簇的数量。
*   `Ci` 是第 `i` 个簇。
*   `x` 是簇 `Ci` 中的一个数据点。
*   `μi` 是簇 `Ci` 的质心。
*   `|| x - μi ||^2` 是数据点 `x` 到其质心 `μi` 的欧式距离平方。

![](img/9a4f185f9b9e36aec1010105658da246_18.png)

这个目标函数的意义是：我们希望所有数据点到其所属簇质心的距离总和越小越好。在迭代过程中，算法会不断调整数据点的归属和质心的位置，以最小化这个目标函数。

## 总结 📝

本节课中我们一起学习了K-Means聚类算法的基本概念。我们了解到：
1.  聚类是一种无监督学习方法，用于将相似数据自动分组。
2.  K-Means算法需要预先指定聚类数量K。
3.  算法的核心是**质心**和**距离度量**（常用欧式距离）。
4.  算法的优化目标是最小化每个簇内数据点到其质心的距离总和。
