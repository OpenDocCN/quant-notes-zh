在本节课中，我们将要学习数据预处理中的一个核心步骤——标准化。标准化是量化交易和机器学习中非常基础且重要的操作，它能帮助我们消除不同数据指标因量纲和取值范围不同带来的影响。

![](img/d421ec43e7e43dc57d401d0bd76a2a6b_3.png)

## 为什么需要标准化？ 🤔

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体目的。

假设我们有一组数据，包含两个指标X1和X2。X1的取值范围较大，例如从-10到10；而X2的取值范围较小，例如从-1到1。如果我们直接用这些原始数据进行分析或建模，取值范围大的指标（如X1）可能会对结果产生不成比例的巨大影响，从而掩盖了其他指标的作用。这并非我们期望的。

标准化的核心目的，就是为了让不同维度的数据指标具有可比性。它通过数学变换，使得处理后的数据满足以下两个特性：
1.  以零为中心对称分布。
2.  各维度的数值大致分布在一个相近的、较小的区间内。

## 标准化的原理与公式 📐

标准化的操作原理非常简单，其核心公式如下：

**`z = (x - μ) / σ`**

在这个公式中：
*   **`x`** 代表原始数据。
*   **`μ`** 代表该维度数据的平均值（均值）。
*   **`σ`** 代表该维度数据的标准差。

这个公式可以分解为两个步骤来理解：

**第一步：去均值**
操作：`x - μ`
目的：将数据整体的中心平移到零点。原始数据可能不以零点为中心，此操作能确保处理后的数据在各个维度上都以零为中心对称。

**第二步：除标准差**
操作：`(去均值后的结果) / σ`
目的：调整数据的“伸缩”范围，解决各维度取值范围不同的问题。

为了理解除标准差的作用，可以想象两个小组：
*   **小组A**：成员收入都在1万到2万之间，差异小，计算出的标准差`σ`也小。
*   **小组B**：成员收入从1万到数千亿，差异巨大，计算出的标准差`σ`非常大。

对于取值范围本身较小的数据（小组A），除以一个较小的标准差，其数值范围变化相对温和。对于取值范围巨大的数据（小组B），除以一个非常大的标准差，能将其数值大幅度“压缩”到一个合理的区间。最终，经过这两步操作，不同维度的数据就被“拉”到了相近的尺度上。

## 如何用代码实现标准化？ 💻

在Python中，我们可以轻松实现标准化。以下是手动实现的代码示例：

首先，我们需要计算数据的均值和标准差。

![](img/d421ec43e7e43dc57d401d0bd76a2a6b_5.png)

![](img/d421ec43e7e43dc57d401d0bd76a2a6b_7.png)

```python
import numpy as np

def standardize(series):
    """
    对输入的序列进行标准化处理。
    参数:
        series: 一个pandas Series或类似数组的数据结构。
    返回:
        标准化后的序列。
    """
    # 计算均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    
    # 应用标准化公式: (x - μ) / σ
    standardized_series = (series - mean_val) / std_val
    
    return standardized_series
```

![](img/d421ec43e7e43dc57d401d0bd76a2a6b_9.png)

代码逻辑非常清晰：先计算均值和标准差，然后对数据序列应用标准化公式。

## 总结 📝

![](img/d421ec43e7e43dc57d401d0bd76a2a6b_35.png)

本节课中我们一起学习了数据标准化处理。
*   **目的**：消除不同数据维度因量纲和取值范围差异造成的影响，使数据具有可比性。
*   **原理**：通过 **`z = (x - μ) / σ`** 公式，先中心化（去均值），再缩放（除标准差）。
*   **实现**：可以手动根据公式计算，也可以利用像Scikit-learn中的`StandardScaler`这样的工具快速实现。
*   **效果**：处理后的数据以零为中心，且各维度数值分布区间相近。

![](img/d421ec43e7e43dc57d401d0bd76a2a6b_37.png)

标准化是数据预处理中极为常用的一步，在进入量化建模或机器学习算法之前，对数据进行标准化往往是一个良好的起点。
