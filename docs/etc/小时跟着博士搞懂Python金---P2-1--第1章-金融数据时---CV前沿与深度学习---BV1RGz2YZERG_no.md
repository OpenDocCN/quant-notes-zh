首先，我们需要导入数据，并将其时间列设置为索引，以便进行时间序列分析。

以下是导入数据并设置时间索引的步骤：

![](img/e4e44a45d00b773a4da1331957125503_7.png)

1.  使用 `pandas` 的 `read_csv` 函数读取CSV文件。
2.  通过 `index_col` 参数指定日期列作为数据框的索引。
3.  使用 `parse_dates` 参数将日期字符串自动解析为标准的日期时间格式。

![](img/e4e44a45d00b773a4da1331957125503_9.png)

![](img/e4e44a45d00b773a4da1331957125503_10.png)

```python
import pandas as pd

![](img/e4e44a45d00b773a4da1331957125503_11.png)

# 读取数据，将第一列（日期）设为索引，并解析日期格式
data = pd.read_csv('data.csv', index_col=0, parse_dates=True)
```

## 基础统计分析

![](img/e4e44a45d00b773a4da1331957125503_43.png)

![](img/e4e44a45d00b773a4da1331957125503_44.png)

在初步观察了数据走势后，接下来我们需要对数据进行更深入的统计分析，以了解数据的整体情况。

![](img/e4e44a45d00b773a4da1331957125503_47.png)

以下是进行基础统计分析的两个主要方法：

![](img/e4e44a45d00b773a4da1331957125503_48.png)

![](img/e4e44a45d00b773a4da1331957125503_49.png)

![](img/e4e44a45d00b773a4da1331957125503_50.png)

1.  **使用 `describe` 方法获取描述性统计**：此方法可以快速计算并展示每个数值列的关键统计指标，如计数、均值、标准差、最小值、四分位数和最大值。我们可以使用 `round` 方法控制结果的小数位数。

    ```python
    # 计算描述性统计，并保留两位小数
    summary_stats = data.describe().round(2)
    print(summary_stats)
    ```

2.  **使用 `agg` 方法聚合自定义统计指标**：如果只需要特定的统计指标，可以使用 `aggregate`（或简写为 `agg`）函数，将多个统计函数的结果汇总在一起。

    ```python
    # 自定义聚合统计：最小值、最大值、均值、标准差和中位数
    custom_stats = data.agg(['min', 'max', 'mean', 'std', 'median'])
    print(custom_stats)
    ```
