# 假设数据已加载到变量 `data` 中

## 应用K-Means聚类算法

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_20.png)

上一节我们导入了数据和工具包，本节中我们来看看如何应用K-Means聚类算法进行建模。

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_22.png)

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_24.png)

这次的任务与之前不同。之前我们使用回归方法进行预测，而这次我们将使用聚类方法。在`sklearn`库中，我们选择`KMeans`模块。K-Means是一种将数据点划分为K个簇的经典聚类算法。

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_26.png)

以下是建模步骤：

首先，我们实例化一个K-Means模型。这里需要指定一个关键参数`n_clusters`，即我们希望数据被分成多少个簇。在我们的任务中，目标是将股票涨跌情况分为两类（涨和跌），因此我们将K值设为2。

```python
# 实例化K-Means模型，设定簇的数量为2
model = KMeans(n_clusters=2, random_state=42)
```

然后，我们使用`.fit()`方法，将选定的特征数据传入模型进行训练。

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_28.png)

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_30.png)

```python
# 使用特征数据拟合模型
model.fit(features)
```

模型拟合完成后，会显示一些默认参数，例如初始化中心点的方法、最大迭代次数等。K-Means算法由于初始中心点的随机性，每次运行结果可能略有不同。`sklearn`中的实现默认会进行多次初始化并选择最佳结果。

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_32.png)

接下来，我们使用训练好的模型对数据进行预测，得到每个数据点所属的簇标签（0或1）。

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_34.png)

```python
# 预测每个数据点所属的簇
cluster_labels = model.predict(features)
data['cluster'] = cluster_labels
```

为了更直观地对应涨（1）和跌（-1），我们对预测标签进行映射转换。

```python
# 将聚类标签0和1映射为-1和1，以对应“跌”和“涨”
data['cluster'] = data['cluster'].apply(lambda x: 1 if x == 1 else -1)
```

## 策略回测与评估

然而，仅仅得到分类簇并不足以证明策略的有效性。我们需要进行回测，评估基于此聚类结果的交易策略收益。

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_56.png)

首先，我们计算按照聚类信号进行交易（假设簇标签为1时买入，-1时卖出或做空）的收益序列，并与原始收益进行对比。注意，如果之前对收益进行了对数处理，需要先进行还原。

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_58.png)

```python
# 计算聚类策略的收益（假设‘returns’是原始日收益率列）
# 策略：当cluster为1时，获得当日收益；为-1时，获得当日负收益（即做空）
data['strategy_returns'] = data['cluster'] * data['returns']

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_60.png)

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_62.png)

# 如果‘returns’是对数收益，需还原为简单收益（假设）
# data['simple_returns'] = np.exp(data['returns']) - 1
# data['strategy_simple_returns'] = data['cluster'] * data['simple_returns']

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_64.png)

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_66.png)

# 计算累计收益
data['cumulative_returns'] = (1 + data['returns']).cumprod()
data['cumulative_strategy_returns'] = (1 + data['strategy_returns']).cumprod()
```

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_68.png)

然后，我们可以计算并比较基准收益和策略收益。

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_70.png)

```python
final_baseline_return = data['cumulative_returns'].iloc[-1] - 1
final_strategy_return = data['cumulative_strategy_returns'].iloc[-1] - 1

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_72.png)

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_74.png)

print(f"基准累计收益率: {final_baseline_return:.2%}")
print(f"聚类策略累计收益率: {final_strategy_return:.2%}")
```

![](img/b3e95e22664fc6aa6fd0d6eb0ebb3ae8_76.png)

**注意**：由于K-Means聚类具有随机性，每次运行的结果和策略收益可能不同。在实际应用中，聚类分析在量化策略中并不常用，主要原因在于其缺乏明确的标签，难以进行严谨的经济学解释和稳定的性能评估。它通常作为探索性数据分析或在其他方法不适用时的备选方案。
