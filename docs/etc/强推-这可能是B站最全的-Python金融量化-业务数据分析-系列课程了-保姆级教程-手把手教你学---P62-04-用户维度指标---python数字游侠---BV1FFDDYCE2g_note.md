## 什么是业务指标？

![](img/aaa8521706f9d747c50584d86fcfa0a7_1.png)

![](img/aaa8521706f9d747c50584d86fcfa0a7_3.png)

上一节我们介绍了数据分析需要建立在业务理解之上。本节中我们来看看，如何通过“业务指标”来量化地理解业务。

业务指标是衡量业务好坏的标准。一个好的、正确的指标能够清晰地反映业务的经营状况。例如，对于一家销售公司，**销售额** 是其核心指标，因为它直接反映了公司赚钱的能力。销售额越高，通常意味着经营越好；反之则可能存在问题。

![](img/aaa8521706f9d747c50584d86fcfa0a7_5.png)

![](img/aaa8521706f9d747c50584d86fcfa0a7_7.png)

但仅凭单一指标可能不够全面。例如，销售额上升的同时，如果前期投入成本巨大，整体利润可能并未改善。因此，我们需要一套综合的指标体系来全面评估业务。

另一个例子是手机应用（APP）。衡量一款APP是否健康、受欢迎，一个核心指标是 **日活跃用户数（DAU）**，即一天内有多少用户使用了该APP并产生了价值。DAU越高，通常说明APP越受欢迎。

## 为何要学习业务指标？

我们为什么要费心去学习和衡量这些指标呢？原因在于，数据分析的最终目的是为了了解和改进业务。通过指标，我们可以：

![](img/aaa8521706f9d747c50584d86fcfa0a7_25.png)

1.  **发现问题**：当别人说“业务不好”时，指标（如销售额下降50%）提供了具体、可度量的依据。
2.  **定位问题**：通过分析业务流程中各个环节的指标（如从浏览商品到下单的转化率），可以精准定位问题出在哪个环节。
3.  **评估潜力与吸引投资**：在向投资方展示时，用户数、付费用户占比、利润等指标是证明业务潜力和价值的关键证据。

需要注意的是，公司的核心指标和数据通常都是保密的，这是公司的核心资产。在工作中必须具有严格的数据保密意识。

## 用户维度指标详解

![](img/aaa8521706f9d747c50584d86fcfa0a7_27.png)

由于不同行业的指标差异很大，我们以互联网电商行业为例，将指标分为用户数据、用户行为数据和产品数据三类。本节我们重点讲解用户数据层面的指标。

以下是用户数据层面的核心指标列表：

### 1. 日新增用户数

**日新增用户数** 指的是在某一天内，首次使用某产品（如APP、网站）的用户数量。“新增”强调是第一次访问。

*   **核心意义**：反映产品的拉新能力与效果。在产品初期，新增用户数是衡量增长势头的重要指标。
*   **潜在问题**：单纯追求新增用户数可能导致“僵尸用户”问题，即用户因短期利益（如扫码送礼）下载后便不再活跃，无法产生长期价值。

### 2. 活跃用户数

**活跃用户数** 指在特定时间段内，进行了指定操作的用户数量。它比新增用户数更能体现为用户带来的价值。

*   **定义方式**：不同公司定义不同，常见的有：
    *   打开应用即算活跃（较宽松）。
    *   用户登录后才算活跃（如游戏、电商）。
    *   完成指定行为（如阅读、评论、下单）才算活跃（较严格）。
*   **按时间维度划分**：
    *   **日活跃用户数（DAU）**：一天内的活跃用户数。行业常简称为“日活”。
    *   **周活跃用户数（WAU）**：一周内的活跃用户数。
    *   **月活跃用户数（MAU）**：一个月内的活跃用户数。

### 3. 活跃率

**活跃率** 是活跃用户数占总用户数的比例，公式为：
`活跃率 = 活跃用户数 / 总用户数`

*   **核心意义**：比率指标通常比绝对数更能反映产品的健康程度和用户粘性。活跃率低可能意味着产品对用户的吸引力不足。

### 4. 留存率

**留存率** 衡量的是“留住用户”的能力。它考察的是，在某个时间点（如第一天）新增的用户，在之后的一段时间（如第N天）仍然活跃的比例。

*   **计算公式**：`第N天留存率 = (第1天新增用户中在第N天仍活跃的用户数) / (第1天新增用户总数)`
*   **常见类型**：
    *   **次日留存率**：新增用户次日仍活跃的比例。
    *   **7日留存率**：新增用户第7天仍活跃的比例。
    *   **月留存率**：新增用户第30天仍活跃的比例。
*   **行业参考标准（4-2-1法则）**：次日留存率约40%，7日留存率约20%，月留存率约10%，可视为产品处于健康水平。
*   **核心意义**：反映产品的用户粘性和长期价值。留住一个老用户的成本通常远低于获取一个新用户。

### 5. 单位获客成本

**单位获客成本** 指平均获取一个客户（或独立访客）所花费的成本。

*   **计算公式**：`单位获客成本 = 广告投放总费用 / 广告带来的独立访客数`
*   **核心意义**：评估市场推广渠道的效果和效率。需要结合该渠道用户带来的**收入**和**转化率**进行综合分析。如果成本上升而收入或转化未提升，则说明该推广渠道可能存在问题。

### 6. 客单价与ARPU/ARPPU

这部分指标关注用户的价值贡献。

*   **客单价**：平均每个订单的金额。常用于零售和电商。
    *   **计算公式**：`客单价 = 订单总金额 / 订单数量`
*   **平均每用户收入（ARPU）**：平均每个用户（通常指注册用户）带来的收入。
    *   **计算公式**：`ARPU = 总收入 / 总用户数`
*   **平均每付费用户收入（ARPPU）**：平均每个付费用户带来的收入。
    *   **计算公式**：`ARPPU = 总收入 / 付费用户数`
*   **核心意义**：ARPU反映整体用户的平均价值，ARPPU则更精准地反映付费用户的消费能力和价值，常用于游戏、订阅服务等行业。

## 总结

本节课中我们一起学习了用户维度的核心业务指标。我们从衡量拉新效果的**日新增用户数**开始，过渡到衡量用户参与度的**活跃用户数**与**活跃率**，进而深入到衡量用户粘性的**留存率**。最后，我们学习了评估推广效率的**单位获客成本**和衡量用户价值的**客单价、ARPU/ARPPU**。

![](img/aaa8521706f9d747c50584d86fcfa0a7_29.png)

理解这些指标，不仅能够帮助我们量化地评估业务状态，更是与业务方、投资方进行有效沟通的基础。记住，指标的意义在于驱动决策，发现问题并指导行动，而不仅仅是冰冷的数字。
