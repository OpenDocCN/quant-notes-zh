# AI模型体验教程：P1：幻方量化DeepSeek-V3模型初体验 🚀

在本节课中，我们将学习如何体验幻方量化公司最新推出的DeepSeek-V3大语言模型。我们将了解其核心特点，并通过官网进行实际功能测试。

## 概述

DeepSeek-V3是一款由幻方量化公司推出的新一代大语言模型。它采用了**MOE（Mixture of Experts）混合专家架构**，具有成本低、生成速度快的特点。该模型拥有**671B**参数，在多项基准测试中性能接近GPT-4o和Claude-3.5-Sonnet。

    *   缓存未命中时：**2元 / 百万输入token**
    *   输出token：**8元 / 百万输出token**
    *   目前提供45天的免费体验期。

## 官网功能体验指南

了解了模型的特点后，接下来我们进入DeepSeek官网进行实际操作体验。

官网地址为：**chat.deepseek.com**。界面设计简洁，主要功能清晰。

![](img/027b43d9c68e511a99ec1303100eb301_13.png)

![](img/027b43d9c68e511a99ec1303100eb301_15.png)

以下是官网的核心功能区域介绍：

![](img/027b43d9c68e511a99ec1303100eb301_17.png)

![](img/027b43d9c68e511a99ec1303100eb301_19.png)

*   **对话输入框**：位于界面下方，用于输入问题。
*   **功能按钮**：
    *   **深度思考**：点击后模型会进行更深入、步骤更详细的推理（通常以英文进行）。
    *   **联网搜索**：点击后模型可以实时搜索互联网信息来回答问题。
*   **文件上传**：支持上传多种格式文件，包括PDF、Word、Excel、PPT、图片等。最多可上传50个文件，每个文件最大100MB。
*   **新建对话**：用于开启一个全新的对话线程。

## 实际功能测试

![](img/027b43d9c68e511a99ec1303100eb301_27.png)

现在，我们将使用几个不同的问题来测试模型的各项功能。

![](img/027b43d9c68e511a99ec1303100eb301_29.png)

![](img/027b43d9c68e511a99ec1303100eb301_31.png)

### 基础问答测试

![](img/027b43d9c68e511a99ec1303100eb301_33.png)

首先，我们不启用任何特殊功能，进行一个基础问题测试。
> 问题：“先有鸡还是先有蛋？”
> 模型回答速度非常快，答案流畅生成。

![](img/027b43d9c68e511a99ec1303100eb301_35.png)

![](img/027b43d9c68e511a99ec1303100eb301_37.png)

### 启用“深度思考”模式

![](img/027b43d9c68e511a99ec1303100eb301_39.png)

接下来，我们测试“深度思考”功能。该模式下模型会进行更复杂的推理。
1.  点击“深度思考”按钮，按钮颜色会发生变化表示已启用。
2.  输入问题，例如：“通用人工智能（AGI）什么时候可以实现？”
3.  模型会以英文进行分步推理，生成详细的分析过程。
4.  您可以要求模型将推理结果转为中文，例如输入：“请将上面的分析转为中文。”

![](img/027b43d9c68e511a99ec1303100eb301_41.png)

![](img/027b43d9c68e511a99ec1303100eb301_43.png)

### 启用“联网搜索”模式

![](img/027b43d9c68e511a99ec1303100eb301_45.png)

![](img/027b43d9c68e511a99ec1303100eb301_47.png)

最后，我们测试联网搜索功能，获取实时信息。
1.  点击“联网搜索”按钮。
2.  输入需要查询实时信息的问题，例如：“马斯克的身价是多少？”
3.  模型会显示正在搜索网页，并基于搜索结果给出答案（例如：“超过4000亿美元”）。
