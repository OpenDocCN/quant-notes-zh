## 生成时间范围：date_range函数

![](img/d431c85215199708186b2f8326b72621_5.png)

`date_range` 函数是 Pandas 中用于生成等间距时间序列的核心工具。它可以根据指定的开始时间、结束时间、序列长度和频率来创建 `DatetimeIndex` 对象。

![](img/d431c85215199708186b2f8326b72621_7.png)

以下是 `date_range` 函数的基本参数：

*   **start**：指定时间序列的开始时间。
*   **end**：指定时间序列的结束时间。
*   **periods**：指定要生成的时间点的数量（序列长度）。
*   **freq**：指定时间序列的频率，例如“D”代表天，“H”代表小时。

![](img/d431c85215199708186b2f8326b72621_9.png)

## 参数使用示例

![](img/d431c85215199708186b2f8326b72621_11.png)

### 1. 指定开始和结束时间

![](img/d431c85215199708186b2f8326b72621_13.png)

通过设置 `start` 和 `end` 参数，可以生成一个从开始日期到结束日期的每日时间序列。默认频率 `freq='D'` 表示按天生成。

![](img/d431c85215199708186b2f8326b72621_15.png)

```python
import pandas as pd

![](img/d431c85215199708186b2f8326b72621_17.png)

# 生成从2010-01-01到2010-05-01的每日日期
date_index = pd.date_range(start='2010-01-01', end='2010-05-01')
print(date_index)
```
**输出示例**：`DatetimeIndex(['2010-01-01', '2010-01-02', '2010-01-03', ..., '2010-04-30', '2010-05-01'], dtype='datetime64[ns]', freq='D')`

![](img/d431c85215199708186b2f8326b72621_19.png)

### 2. 指定开始时间和序列长度

如果不指定结束时间，可以使用 `periods` 参数来定义序列的长度。例如，从2010年1月1日开始，生成60个连续的日期。

```python
# 从2010-01-01开始，生成60个连续的日期
date_index = pd.date_range(start='2010-01-01', periods=60)
print(date_index)
```

### 3. 调整生成频率

![](img/d431c85215199708186b2f8326b72621_21.png)

`freq` 参数非常强大，它允许你以不同的时间间隔生成序列。以下是一些常用的频率代码：

![](img/d431c85215199708186b2f8326b72621_23.png)

*   `'D'`： 日历日（默认）
*   `'H'`： 小时
*   `'W'`： 周（默认从周日开始，`'W-MON'` 表示从周一开始）
*   `'B'`： 工作日（周一至周五）
*   `'M'`： 月末
*   `'MS'`： 月初
*   `'Q'`： 季末
*   `'A'` 或 `'Y'`： 年末
*   `'T'` 或 `'min'`： 分钟
*   `'S'`： 秒

![](img/d431c85215199708186b2f8326b72621_25.png)

```python
# 生成每小时的时间点
hourly_index = pd.date_range(start='2010-01-01', periods=24, freq='H')
print(hourly_index)

![](img/d431c85215199708186b2f8326b72621_27.png)

# 生成每周一的时间点
weekly_monday_index = pd.date_range(start='2010-01-01', periods=5, freq='W-MON')
print(weekly_monday_index)

![](img/d431c85215199708186b2f8326b72621_29.png)

![](img/d431c85215199708186b2f8326b72621_31.png)

# 生成工作日（周一至周五）的时间点
business_days_index = pd.date_range(start='2010-01-01', periods=10, freq='B')
print(business_days_index)
```

`freq` 参数还支持更复杂的偏移量组合，例如“2H30min”表示每2小时30分钟。

![](img/d431c85215199708186b2f8326b72621_33.png)

![](img/d431c85215199708186b2f8326b72621_35.png)

## 时间索引对象的转换

`date_range` 生成的是一个 `DatetimeIndex` 对象。你可以方便地将其转换为其他格式。

![](img/d431c85215199708186b2f8326b72621_37.png)

*   **转换为 Python 原生的 datetime 对象列表**：使用 `.to_pydatetime()` 方法。
*   **转换为字符串列表**：虽然 `DatetimeIndex` 本身打印出来像字符串，但你可以使用 `.strftime()` 方法进行格式化，或通过列表推导式转换。

![](img/d431c85215199708186b2f8326b72621_39.png)

```python
# 转换为Python datetime对象列表（注意：未来版本可能废弃，建议使用 .to_list()）
py_datetime_list = date_index.to_pydatetime()

# 转换为特定格式的字符串列表
str_list = [date.strftime('%Y-%m-%d') for date in date_index]
print(str_list)
```
