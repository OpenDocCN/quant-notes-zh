## 概述

文件是存储数据的基本方式之一。在量化交易中，我们经常需要从CSV、Excel等格式的文件中读取股票历史数据，或者将分析结果写入新的文件。Python提供了内置的`open()`函数以及更高级的库（如`pandas`）来帮助我们完成这些任务。本节我们将学习如何使用这些工具。

## 打开文件：`open()`函数

要操作一个文件，首先需要打开它。Python使用`open()`函数来打开文件，该函数会返回一个文件对象，用于后续的读写操作。

**核心语法**：
```python
file_object = open(file_path, mode, encoding)
```

以下是`open()`函数参数的详细说明：
*   **`file_path`**：字符串，表示要打开的文件路径。
*   **`mode`**：字符串，指定打开文件的模式。
*   **`encoding`**：字符串，指定文件的编码格式（如 `‘gbk‘`, `‘utf-8‘`），对处理中文文本尤为重要。

### 文件打开模式

文件模式决定了你可以对文件进行何种操作。以下是常用的模式：

*   **`‘r‘`**：**只读**模式。这是默认模式，用于读取文件内容，不能写入。
*   **`‘w‘`**：**写入**模式。如果文件已存在，则会**清空原有内容**再写入；如果文件不存在，则创建新文件。
*   **`‘a‘`**：**追加**模式。在文件末尾添加新内容，不会清空原有数据。如果文件不存在，则创建新文件。
*   **`‘x‘`**：**创建**模式。创建新文件并写入。如果文件已存在，则会报错。
*   **`‘b‘`**：**二进制**模式。用于处理图片、音频等非文本文件，需与其他模式结合使用（如 `‘rb‘`, `‘wb‘`）。
*   **`‘+‘`**：**更新**模式。允许读写操作，需与其他模式结合使用（如 `‘r+‘`, `‘w+‘`）。

> **重要提示**：使用`‘w‘`模式时要格外小心，因为它会直接覆盖原文件。

## 读取文件内容

上一节我们介绍了如何打开文件，本节中我们来看看如何读取文件中的数据。我们将使用贵州茅台的日线数据（CSV格式）作为示例。

### 使用`pandas`读取CSV文件

![](img/811de668698702445980fe2e172730de_7.png)

![](img/811de668698702445980fe2e172730de_9.png)

对于结构化数据（如CSV），使用`pandas`库是最方便高效的方法。`pandas`的`read_csv()`函数可以轻松将文件读入一个`DataFrame`（类似表格的数据结构）。

![](img/811de668698702445980fe2e172730de_11.png)

**核心代码**：
```python
import pandas as pd
df = pd.read_csv(‘文件路径/贵州茅台.csv‘, encoding=‘gbk‘)
print(df)
```

![](img/811de668698702445980fe2e172730de_13.png)

执行上述代码后，`df`变量中就存储了文件的所有数据。直接打印`df`会以精简格式显示，如果希望看到完整内容，可以使用`to_string()`方法：
```python
print(df.to_string())
```

![](img/811de668698702445980fe2e172730de_15.png)

![](img/811de668698702445980fe2e172730de_17.png)

### 使用`with`语句安全地打开文件

手动管理文件的打开和关闭容易出错。Python的`with`语句可以确保文件在使用后被正确关闭，即使中途发生异常也是如此。这是推荐的文件操作方式。

**核心代码**：
```python
with open(‘贵州茅台.csv‘, ‘r‘, encoding=‘gbk‘) as file:
    df = pd.read_csv(file)
    print(df.to_string())
# 文件在此处会自动关闭
```

## 写入和追加内容

学会了读取，我们再来看看如何向文件中写入数据。这常用于保存策略输出或日志。

### 写入模式（‘w‘）

使用`‘w‘`模式会创建一个新文件（或清空已有文件）并写入内容。

**核心代码**：
```python
with open(‘写入文件测试.txt‘, ‘w‘, encoding=‘utf-8‘) as file:
    file.write(‘Hello World!\n‘)  # ‘\n‘ 表示换行
```

### 追加模式（‘a‘）

使用`‘a‘`模式会在文件末尾添加新内容，保留原有数据。

**核心代码**：
```python
with open(‘写入文件测试.txt‘, ‘a‘, encoding=‘utf-8‘) as file:
    file.write(‘这是新追加的一行。\n‘)
```

## 读取文件的指定部分

有时我们不需要读取整个文件，而只关心前几行数据。以下是读取指定行数的方法。

我们可以先使用`readlines()`方法将所有行读入一个列表，然后通过切片或循环获取需要的部分。

**核心代码**：
```python
with open(‘写入文件测试.txt‘, ‘r‘, encoding=‘utf-8‘) as file:
    all_lines = file.readlines()  # 读取所有行到列表
    # 打印前2行
    for i in range(2):
        print(all_lines[i])
```
