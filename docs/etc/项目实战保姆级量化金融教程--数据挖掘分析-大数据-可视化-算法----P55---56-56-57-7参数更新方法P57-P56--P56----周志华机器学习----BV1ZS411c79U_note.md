# 机器学习：P55-P57：参数更新方法

在本节课中，我们将学习机器学习中一个至关重要的概念：**梯度下降法**。我们将深入探讨如何通过计算损失函数的梯度来更新模型参数，从而找到使模型性能最优的参数值。整个过程将分为三个核心步骤，我们会用简单直白的语言和公式来解释每一步。

## 概述

我们的目标是优化一个模型，例如线性回归模型 `hθ(x) = θ₀ + θ₁x₁`，使其预测结果尽可能准确。这需要通过调整参数 `θ₀` 和 `θ₁` 来实现。本节课的核心问题是：在优化过程中，我们是**同时**更新所有参数，还是**分别**更新每个参数？我们将通过分析参数之间的关系和梯度下降的数学原理来解答这个问题，并最终推导出参数更新的通用公式。

## 参数更新：分别进行还是同时进行？

上一节我们介绍了损失函数的概念，本节我们来看看如何通过优化损失函数来更新参数。首先需要明确一个关键点：参数 `θ₀` 和 `θ₁` 的更新是**分别进行**的，而不是同时作为一个整体进行优化。

为什么是分别进行呢？这源于数据样本之间的独立性。在模型中，`θ₀` 对应的是特征 `x₀`（通常为常数1），`θ₁` 对应的是特征 `x₁`。由于每个数据样本的特征 `x₀` 和 `x₁` 之间没有依赖关系，是相互独立的，因此它们所对应的参数 `θ₀` 和 `θ₁` 在优化过程中也互不影响。

我们可以将优化过程想象成在一个三维空间中寻找山谷的最低点（即损失函数的最小值）。`θ₀` 和 `θ₁` 构成了这个空间的两个坐标轴。在寻找最低点时，我们需要为 `θ₀` 找到一个合适的下降方向，同时也为 `θ₁` 找到另一个合适的下降方向。这两个参数沿着各自的方向独立前进，最终共同抵达一个综合最优的位置。

![](img/bb8221f30d9589b4ea6f201356e51127_1.png)

因此，在数学上，我们需要：
1.  计算损失函数 `J` 对 `θ₀` 的偏导数，得到 `θ₀` 的更新方向。
2.  计算损失函数 `J` 对 `θ₁` 的偏导数，得到 `θ₁` 的更新方向。
这两个计算过程是独立的，参数之间“各玩各的”，互不干扰。

## 梯度下降法的三个步骤

寻找山谷最低点（即最小化损失函数）的过程，可以概括为以下三个步骤，并且这是一个需要反复迭代的过程：

1.  **计算偏导数（求梯度）**：计算损失函数 `J(θ)` 对每个参数 `θⱼ` 的偏导数 `∂J/∂θⱼ`。偏导数的值指示了在该参数方向上，函数值上升最快的方向。
2.  **沿反方向走一小步**：沿着梯度（偏导数）的**反方向**移动一小步。因为梯度方向是上升最快的方向，其反方向就是下降最快的方向。**步长必须很小**，如果步长太大，可能导致“跌倒”，即优化过程不收敛、震荡甚至发散。
3.  **更新参数**：根据移动的步长更新当前参数的值。更新后，我们就得到了一个新的参数点，然后回到第1步，重新计算在新参数点处的梯度，如此反复，直至收敛。

## 数学描述：从损失函数到更新公式

上面我们介绍了梯度下降的流程，现在来看看如何在数学上精确描述这个“下山”任务。首先，我们需要一个衡量模型在所有样本上整体表现的损失函数，而不仅仅是单个样本。

对于一个有 `m` 个样本的数据集，常用的损失函数是**均方误差**：
`J(θ) = (1/(2m)) * Σ (y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾))²`
其中，`Σ` 表示对 `i` 从 `1` 到 `m` 求和。
*   `(1/m)` 求平均，确保损失函数不受样本数量影响。
*   `1/2` 是一个为了方便后续求导计算的系数，求导后平方项会产生因子2，与之约分后形式更简洁。
*   平方项 `(真实值 - 预测值)²` 放大了预测误差，使得模型对大的错误更为敏感。

接下来，我们的目标是对某个特定参数 `θⱼ` 求偏导 `∂J/∂θⱼ`。假设我们的预测函数为 `hθ(x) = θ₀x₀ + θ₁x₁ + ... + θⱼxⱼ + ...`。

以下是求偏导的关键步骤推导：
1.  当对 `θⱼ` 求偏导时，求和符号 `Σ` 和系数 `1/(2m)` 可以提到导数外面。
2.  对平方项 `(y - hθ(x))²` 求关于 `θⱼ` 的导数，运用链式法则：先对平方项求导得 `2*(y - hθ(x))`，再乘以内层函数 `-hθ(x)` 对 `θⱼ` 的导数。
3.  `hθ(x)` 对 `θⱼ` 求偏导，由于 `hθ(x)` 是 `θⱼxⱼ` 与其它项的线性加和，而其它项与 `θⱼ` 无关，因此导数结果就是 `xⱼ`。
4.  综合以上，并注意到内层求导产生的负号，我们得到：
    `∂J/∂θⱼ = (1/m) * Σ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾`
    这里 `xⱼ⁽ⁱ⁾` 表示第 `i` 个样本的第 `j` 个特征值。

现在我们得到了梯度（偏导数）。为了**下降**，我们需要沿着梯度的**反方向**移动。梯度的反方向就是加上一个负号：`-∂J/∂θⱼ`。

最后，参数 `θⱼ` 的更新公式如下：
`θⱼ := θⱼ - α * ∂J/∂θⱼ`
即：
`θⱼ := θⱼ - α * [(1/m) * Σ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾]`

其中：
*   `:=` 表示赋值操作。
*   `α` 是一个非常重要的超参数，称为**学习率**。它控制着我们沿着梯度反方向前进的“步长”大小。
*   `(1/m) * Σ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾` 就是损失函数 `J` 对参数 `θⱼ` 的偏导数。

这个公式就是梯度下降法的核心。对于每一个参数 `θⱼ`，我们都按照这个公式独立地进行更新。

## 总结

本节课中我们一起学习了梯度下降法的原理与实现。我们首先明确了模型参数需要**分别进行更新**，因为其对应的特征是独立的。然后，我们将优化过程分解为**计算梯度、确定方向、更新参数**三个核心步骤。最后，我们从均方误差损失函数出发，一步步推导出了适用于线性模型的梯度下降参数更新公式：
`θⱼ := θⱼ - α * [(1/m) * Σ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾]`
理解这个公式的每一个组成部分及其意义，是掌握梯度下降法乃至许多现代机器学习算法的基础。
