## 第一题：展示不同类别的销售额对比

上一节我们介绍了课程概述，本节中我们来看看第一道作业题。题目要求展示不同类别的销售额对比情况。

以下是具体操作步骤：

![](img/b2116524a5e0f2d3bd167fd39c98eace_9.png)

1.  首先，使用数据透视表对数据进行分类汇总。
    *   将“类别”字段拖拽至“行”区域。
    *   将“销售额”字段拖拽至“值”区域。
2.  接着，基于创建好的数据透视表，插入数据透视图。
    *   点击数据透视表任意单元格。
    *   在菜单栏选择“分析” -> “数据透视图”。
    *   选择柱状图或条形图，点击“确定”。
3.  最后，为图表添加数据标签，使数值显示更清晰。
    *   点击图表上的柱形。
    *   右键选择“添加数据标签”。

![](img/b2116524a5e0f2d3bd167fd39c98eace_11.png)

完成以上步骤后，即可得到清晰的各类别销售额对比图。

## 第二题：展示不同地区的销量对比

完成了类别对比，我们再来看看如何对比不同地区的销量。其操作思路与第一题类似，但分析的维度是“地区”和“销量”。

以下是具体操作步骤：

![](img/b2116524a5e0f2d3bd167fd39c98eace_17.png)

1.  创建新的数据透视表。
    *   将“地区”字段拖拽至“行”区域。
    *   将“数量”（即销量）字段拖拽至“值”区域。
2.  为了提升图表可读性，可以对数据进行排序。
    *   点击数据透视表中“数量”列的任意单元格。
    *   选择“降序”排列，使销量最高的地区排在最前。
3.  插入数据透视图，并选择合适的图表类型（如柱状图）。

![](img/b2116524a5e0f2d3bd167fd39c98eace_19.png)

![](img/b2116524a5e0f2d3bd167fd39c98eace_21.png)

经过排序后，图表将呈现阶梯式上升的效果，能更直观地展示各地区销量的差异。

![](img/b2116524a5e0f2d3bd167fd39c98eace_23.png)

分析完静态对比，本节我们学习如何展示单一产品（如收纳盒）随时间变化的趋势。这里我们演示使用数据透视表筛选的方法。

![](img/b2116524a5e0f2d3bd167fd39c98eace_25.png)

以下是具体操作步骤：

![](img/b2116524a5e0f2d3bd167fd39c98eace_27.png)

![](img/b2116524a5e0f2d3bd167fd39c98eace_29.png)

1.  创建一个包含时间、子类别和销售额的数据透视表。
    *   将“订单日期”拖拽至“行”区域，并右键将其分组为“季度”或“年”。

3.  插入数据透视图，并选择“折线图”。

![](img/b2116524a5e0f2d3bd167fd39c98eace_31.png)

生成的折线图可以清晰地反映收纳盒在2016年至2019年间各季度的销售额变化趋势。从图中可以看出，其销售额整体呈上升趋势。

接下来，我们学习如何进行区域性占比分析。本题要求展示华东地区内，不同类别产品销售数量的占比情况。

![](img/b2116524a5e0f2d3bd167fd39c98eace_39.png)

## 附加技巧与问题解答

在操作过程中，你可能会遇到一些细节问题。以下是两个常见问题的解决方法：

*   **如何隐藏数据透视表中的分类汇总？**
    在数据透视表设计选项卡中，找到“分类汇总”选项，选择“不显示分类汇总”即可。
*   **图表没有标准答案**
    图表的样式、颜色和布局都可以根据个人喜好和报告需求进行调整。学习的关键是掌握方法和逻辑，而非追求与演示完全一致。

## 题目二与题目三思路提示

*   **题目二（雷达图）**：用于分析多个对象（如用户）在多个维度（如能力指标）上的表现。选择数据后，直接插入“雷达图”即可。离中心点越远，代表在该维度上的能力值越强。
*   **题目三（费用数据分析）**：
    1.  **费用占比分析**：使用`SUM`函数计算各类别年度总费用，然后插入饼图或柱状图，分析费用构成。例如，可能发现“工资”是最大的费用支出项。
    2.  **费用趋势与异常分析**：插入折线图分析“总费用”或单项费用（如“工资”）在12个月中的变化。如果发现某个月份（如2月）费用异常高，可结合业务知识（如发放年终奖）进行深入分析，并建议进一步细分数据以验证猜想。
