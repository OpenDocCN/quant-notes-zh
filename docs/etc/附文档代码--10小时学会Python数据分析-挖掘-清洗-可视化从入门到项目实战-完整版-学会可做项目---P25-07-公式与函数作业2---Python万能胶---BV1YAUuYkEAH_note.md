## 九九乘法表的实现

上一节我们介绍了单元格引用的基本概念，本节中我们来看看如何利用混合引用来制作一个标准的九九乘法表。

![](img/6cdb56c49ec5a6de7396f7252792a828_1.png)

我们分析单元格显示内容的规律：当行号小于或等于列号时，单元格应显示乘法算式的结果；否则，单元格应为空。

因此，我们可以使用一个`IF`函数进行判断。其核心逻辑是：如果当前单元格的行号（例如A31）大于或等于列号（例如B30），则进行计算，否则返回空值。

为了实现公式的拖动填充，我们需要使用混合引用。我们希望公式在**向下**拖动时，**行号**发生变化；在**向右**拖动时，**列号**发生变化。所以，在引用列标时需要添加美元符号`$`进行固定。

![](img/6cdb56c49ec5a6de7396f7252792a828_3.png)

以下是实现公式：
```excel
=IF($A31>=B$30, $A31*B$30, "")
```
将这个公式输入到乘法表区域的第一个单元格（例如B2），然后向右再向下拖动填充，即可生成完整的九九乘法表。这个练习是理解**混合引用**的经典案例。

![](img/6cdb56c49ec5a6de7396f7252792a828_4.png)

## 多条件成绩判断

接下来，我们处理一个学生成绩判断问题。这里需要根据多门课程的成绩，判断学生是否“全部通过”或“至少通过一门”。

这需要用到逻辑函数`AND`和`OR`与`IF`函数的嵌套。

![](img/6cdb56c49ec5a6de7396f7252792a828_6.png)

![](img/6cdb56c49ec5a6de7396f7252792a828_8.png)

*   **`AND`函数**：所有条件都满足时返回`TRUE`。
*   **`OR`函数**：至少一个条件满足时返回`TRUE`。

假设数学、计算机、英语成绩分别在C、D、E列，及格线为60分。

以下是具体的公式应用：

1.  **判断三门均通过**：
    我们需要检查三个条件是否同时成立。使用`AND`函数组合这些条件。
    ```excel
    =IF(AND(C2>=60, D2>=60, E2>=60), "通过", "不通过")
    ```
    这个公式的意思是：如果数学、计算机、英语成绩都大于等于60，则返回“通过”，否则返回“不通过”。

2.  **判断至少通过一门**：
    我们只需检查是否有任意一门课程及格。使用`OR`函数。
    ```excel
    =IF(OR(C2>=60, D2>=60, E2>=60), "通过", "不通过")
    ```
    这个公式的意思是：如果数学、计算机、英语中有任意一门成绩大于等于60，则返回“通过”，否则返回“不通过”。

## 阶梯水电费计算

在实际工作中，我们常遇到类似水电费的分段计费问题。本节我们以电费为例，学习如何使用`IF`函数的嵌套进行逻辑判断。

电费计价规则分为三个阶梯：
*   第一阶梯：用量 ≤ 240度，单价为`P1`
*   第二阶梯：240度 < 用量 ≤ 400度，单价为`P2`
*   第三阶梯：用量 > 400度，单价为`P3`

![](img/6cdb56c49ec5a6de7396f7252792a828_12.png)

假设电费单价表在单元格`$M$2:$M$4`，当前用电量在B列。

计算逻辑是：首先判断是否大于最高阈值，否则再判断是否大于下一阈值。以下是实现公式：
```excel
=IF(B2>400, B2*$M$4, IF(B2>240, B2*$M$3, B2*$M$2))
```
**公式解析**：
1.  首先判断`B2>400`，如果成立，则执行`B2*$M$4`（第三阶梯）。
2.  如果不成立，则进入下一个`IF`判断`B2>240`。如果成立，则执行`B2*$M$3`（第二阶梯）。
3.  如果也不成立，则执行`B2*$M$2`（第一阶梯）。

注意，单价区域的引用`$M$2:$M$4`使用了绝对引用`$`，是为了保证公式向下拖动时，引用的单价位置不会改变。

水费的计算逻辑与此类似，但可能需要在计算结果上额外加上固定的水资源费和污水处理费，即`水费 = 阶梯水价计算部分 + 用水量 * (水资源费单价 + 污水处理费单价)`。

## 数据提取与计算

最后，我们处理一个从身份证号码中提取出生日期、计算年龄和工龄，并查找籍贯信息的问题。这综合运用了文本函数、日期函数和查找函数。

以下是关键步骤：

1.  **提取出生日期**：
    身份证号码的第7到14位是出生年月日。使用`MID`函数进行提取。
    ```excel
    =MID(A2, 7, 8)
    ```
    提取出的是一串文本“19800323”，需要将其转换为Excel可识别的日期格式。选中该列数据，使用“数据”选项卡中的“分列”功能，在第三步中选择“日期”格式即可。

2.  **计算年龄**：
    使用`DATEDIF`函数计算当前日期与出生日期之间的整年数。
    ```excel
    =DATEDIF(C2, TODAY(), "Y")
    ```
    其中`C2`是出生日期，`TODAY()`获取当前日期，`"Y"`表示返回整年数。

3.  **计算精确到月的工龄**：
    同样使用`DATEDIF`函数，但分别计算整年数和剩余的月数。
    *   整年数：`=DATEDIF(E2, TODAY(), "Y")` （假设E2是入职日期）
    *   剩余月数：`=DATEDIF(E2, TODAY(), "YM")` （`"YM"`参数忽略年和日，只返回月差）
    最终工龄可以表示为：`=工龄整年数 & "年" & 剩余月数 & "个月"`

4.  **根据身份证查找籍贯**：
    身份证前6位是地址码。我们可以先提取前6位，然后用`VLOOKUP`函数在一个预先准备好的“地址码-地区”对照表中进行查找。
    *   提取地址码：`=LEFT(A2, 6)`
    *   查找省份（假设对照表中地址码在A列，省份在B列）：
        ```excel
        =VLOOKUP(LEFT(A2, 2), $对照表!$A:$B, 2, FALSE)
        ```
        这里用`LEFT(A2,2)`提取前两位（省级代码），在对照表中精确查找对应的省份名称。
    *   查找市/区：方法相同，只需提取前4位或6位，并在对照表中找到对应的列即可。

这个练习表明，解决复杂问题通常需要将多个函数**嵌套**使用，例如将`LEFT`/`MID`的结果作为`VLOOKUP`的查找值。熟练掌握每个函数的功能是灵活组合应用的前提。
