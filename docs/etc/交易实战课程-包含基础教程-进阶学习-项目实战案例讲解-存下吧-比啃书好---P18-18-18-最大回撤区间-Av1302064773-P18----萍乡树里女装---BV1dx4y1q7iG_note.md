## 概述

最大回撤是衡量投资策略风险的关键指标。它描述的是在特定时间段内，资产净值从最高点到后续最低点的最大跌幅。这个指标不直接评估策略的盈利能力，而是评估其可能面临的最大风险，即投资者可能经历的最“惨”时期。

## 最大回撤的定义

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何量化风险。最大回撤不能仅从字面理解，它描述的是投资过程中资产价值下跌的“惨劲儿”。

具体来说，在炒股或执行策略的过程中，资产价格会有涨跌。**最大回撤区间**指的就是从某个高点开始，到后续某个低点结束，资产价值下跌最严重、持续时间最长的那段时期。它反映了你可能承受的最大账面亏损幅度。

## 核心概念与公式

![](img/285625e58d614778f5d83d5df4d13946_4.png)

最大回撤的计算基于一个核心思想：寻找资产价格序列中，从前期高点（`Pi`）到后期低点（`Pj`）的最大相对跌幅。

![](img/285625e58d614778f5d83d5df4d13946_6.png)

其计算公式可以表示为：

![](img/285625e58d614778f5d83d5df4d13946_8.png)

**最大回撤 = max( (Pi - Pj) / Pi )**

![](img/285625e58d614778f5d83d5df4d13946_10.png)

![](img/285625e58d614778f5d83d5df4d13946_12.png)

其中：
*   `Pi` 代表第 `i` 天的资产价格（或净值）。
*   `Pj` 代表第 `i` 天之后（`j > i`）的某一天资产价格。
*   我们需要遍历所有可能的 `i` 和 `j`（`j > i`）组合，找出 `(Pi - Pj) / Pi` 这个比值最大的情况。

![](img/285625e58d614778f5d83d5df4d13946_14.png)

这个比值（百分比）就是最大回撤率，它描述了资产从高点“缩水”的程度。

![](img/285625e58d614778f5d83d5df4d13946_16.png)

![](img/285625e58d614778f5d83d5df4d13946_17.png)

## 计算过程解析

![](img/285625e58d614778f5d83d5df4d13946_19.png)

![](img/285625e58d614778f5d83d5df4d13946_20.png)

理解了公式后，我们来看看如何用Python和Pandas进行计算。核心是使用 **`.cummax()`** 函数。

![](img/285625e58d614778f5d83d5df4d13946_22.png)

![](img/285625e58d614778f5d83d5df4d13946_23.png)

以下是计算最大回撤的关键步骤：

![](img/285625e58d614778f5d83d5df4d13946_25.png)

![](img/285625e58d614778f5d83d5df4d13946_26.png)

1.  **计算历史最高值序列**：对资产价格序列使用 `.cummax()` 函数，得到截至每一天的历史最高价格。
    ```python
    # 假设 df[‘close’] 是每日收盘价序列
    historical_max = df[‘close’].cummax()
    ```
2.  **计算每日潜在回撤**：用每一天对应的历史最高价减去当天的实际价格，得到每一天相对于其历史高点的下跌值。
    ```python
    drawdown = historical_max - df[‘close’]
    ```
3.  **计算每日回撤率**：将每日下跌值除以其对应的历史最高价，得到每日的回撤率。
    ```python
    drawdown_pct = drawdown / historical_max
    ```
4.  **找出最大回撤**：在所有每日回撤率中，最大值就是整个考察期内的**最大回撤**。
    ```python
    max_drawdown = drawdown_pct.max()
    ```

## 实际意义与案例

![](img/285625e58d614778f5d83d5df4d13946_28.png)

![](img/285625e58d614778f5d83d5df4d13946_29.png)

最大回撤是一个至关重要的**风险指标**。它告诉你，即使一个策略长期是盈利的，但在过程中你可能需要忍受多大的账面亏损。

![](img/285625e58d614778f5d83d5df4d13946_31.png)

例如，在2008年全球金融危机中，股市从高点大幅下跌。许多投资者经历了巨大的回撤，资产严重缩水。最大回撤指标能清晰地量化这种极端情况下的风险，帮助投资者评估自己能否承受这样的波动。

![](img/285625e58d614778f5d83d5df4d13946_33.png)

![](img/285625e58d614778f5d83d5df4d13946_34.png)

在策略回测报告中，最大回撤通常会与年化收益率、夏普比率等一同列出，用于全面评估策略的收益风险比。

## 总结

本节课中我们一起学习了**最大回撤**这一概念。我们了解到：
1.  最大回撤是衡量投资策略下行风险的核心指标，反映了资产从高点至低点的最大跌幅。
2.  其核心计算公式为 **`max( (Pi - Pj) / Pi )`**，需要通过编程遍历所有价格点对进行计算。
3.  使用Pandas的 **`.cummax()`** 函数可以高效地计算历史最高序列，进而推导出最大回撤。
4.  该指标的意义在于帮助投资者理解策略可能带来的最大心理压力与资金风险，是风控管理中不可或缺的一环。
