## 模型构建与初步可视化

![](img/93c9f35621157dfb90036080a497af08_15.png)

![](img/93c9f35621157dfb90036080a497af08_17.png)

![](img/93c9f35621157dfb90036080a497af08_19.png)

上一节我们介绍了时间序列的基本概念，本节中我们来看看如何使用Prophet构建模型并进行初步的可视化。

![](img/93c9f35621157dfb90036080a497af08_21.png)

执行`create_model`函数后，它会帮助我们完成建模和预测，并生成图表。图表中的黑色点代表观测值，绿色点代表模型预测值，浅绿色区域代表置信区间。这样就构成了一个基本的时间序列预测图。

![](img/93c9f35621157dfb90036080a497af08_23.png)

![](img/93c9f35621157dfb90036080a497af08_25.png)

```python
# 示例：创建Prophet模型并进行拟合
from prophet import Prophet
model = Prophet()
model.fit(df_history)
future = model.make_future_dataframe(periods=0)
forecast = model.predict(future)
```

![](img/93c9f35621157dfb90036080a497af08_27.png)

![](img/93c9f35621157dfb90036080a497af08_29.png)

模型构建完成后，我们可以使用Prophet框架自带的`plot_components`函数来展示模型发现的规律。该函数可以绘制出数据的年度趋势、月度趋势以及年度内各月份的趋势。

![](img/93c9f35621157dfb90036080a497af08_31.png)

![](img/93c9f35621157dfb90036080a497af08_33.png)

以下是绘制模型组件图的步骤：
1.  拟合模型后，调用 `model.plot_components(forecast)`。
2.  函数会自动生成趋势图、年度季节性和周季节性等子图。

## 深入模型参数与源码

![](img/93c9f35621157dfb90036080a497af08_35.png)

了解了基本流程后，我们深入看看模型实例化时的关键参数。

在`create_model`函数中，首先对模型进行实例化。实例化时需要指定一些参数，例如是否绘制季节性变化（`yearly_seasonality`, `weekly_seasonality`等）。一个关键参数是`changepoint_prior_scale`，它控制着模型对趋势变化的灵活度，默认值为0.05。我们通常可以先使用默认值。

![](img/93c9f35621157dfb90036080a497af08_37.png)

如果想深入了解参数设计或模型定义，可以按住Ctrl键（在IDE中）点击`Prophet`类跳转到源码中阅读。源码中包含了所有可配置的参数，如：
*   `yearly_seasonality`: 是否拟合年度趋势。
*   `weekly_seasonality`: 是否拟合周度趋势。
*   `holidays`: 可以传入节假日数据框。
*   `seasonality_mode`: 季节性模式（加性或乘性）。

![](img/93c9f35621157dfb90036080a497af08_39.png)

在实例化模型后，代码还添加了一个以30.5天为周期的季节性成分。最后，将配置好的模型返回。

![](img/93c9f35621157dfb90036080a497af08_41.png)

![](img/93c9f35621157dfb90036080a497af08_43.png)

## 数据训练与历史拟合

![](img/93c9f35621157dfb90036080a497af08_45.png)

![](img/93c9f35621157dfb90036080a497af08_47.png)

有了模型，下一步就是用历史数据进行训练和拟合。

训练时，代码通过`self.training`参数控制使用最近多少年的数据进行训练，例如设置为3则表示仅使用最近3年的数据。使用`model.fit()`方法即可完成训练。

![](img/93c9f35621157dfb90036080a497af08_49.png)

![](img/93c9f35621157dfb90036080a497af08_51.png)

训练完成后，使用`model.make_future_dataframe()`生成待预测的时间数据框。当参数`periods=0`时，表示不预测未来的新时间点，只对历史数据进行拟合和预测。此时，`future`数据框与历史数据的时间范围一致。

![](img/93c9f35621157dfb90036080a497af08_53.png)

![](img/93c9f35621157dfb90036080a497af08_55.png)

2.  用绿色线条绘制模型对历史数据的预测值（`ds`和`yhat`）。
3.  用浅绿色区域填充置信区间（`yhat_lower`和`yhat_upper`）。

![](img/93c9f35621157dfb90036080a497af08_79.png)

![](img/93c9f35621157dfb90036080a497af08_81.png)

![](img/93c9f35621157dfb90036080a497af08_83.png)

调用`model.plot_components(forecast)`函数，可以绘制出分解后的趋势图。从生成的图中我们可以分析出：
*   **年度趋势**：显示从2015年到2018年，整体价格呈上升趋势。
*   **月度趋势**：显示在每月初价格有上升趋势，在月末有下降趋势。
*   **年度内月份效应**：显示12月和1月的价格上涨幅度最大，而7月、9月和10月则呈现下降趋势。

这些规律为我们提供了交易思路：例如，在每月月底价格可能下降时，可以考虑减少买入操作。

## 识别趋势突变点

![](img/93c9f35621157dfb90036080a497af08_101.png)

![](img/93c9f35621157dfb90036080a497af08_103.png)

时间序列中常存在趋势发生变化的点，Prophet可以自动识别这些“突变点”。

![](img/93c9f35621157dfb90036080a497af08_105.png)

![](img/93c9f35621157dfb90036080a497af08_107.png)

突变点是指趋势发生转折的位置，例如由下降转为上升，或上升速度突然加快。在Prophet中，这通过观察时间序列**二阶导数**的正负变化来判断。

![](img/93c9f35621157dfb90036080a497af08_109.png)

以下是识别和可视化突变点的步骤：
1.  拟合模型后，可以从`model.changepoints`属性中获取模型自动识别的突变点日期。
2.  计算突变点处的趋势变化率（相当于二阶导数的符号）。
3.  根据变化率为正（positive，增长加速）或为负（negative，增长减速），用不同颜色的竖线在趋势图上标记出来。

![](img/93c9f35621157dfb90036080a497af08_111.png)

![](img/93c9f35621157dfb90036080a497af08_113.png)

通过这张图，我们可以直观地看到历史数据中趋势发生重大变化的时间点。

## 进行未来预测

![](img/93c9f35621157dfb90036080a497af08_115.png)

![](img/93c9f35621157dfb90036080a497af08_117.png)

最后，我们将利用训练好的模型对未来进行预测。

![](img/93c9f35621157dfb90036080a497af08_119.png)

预测未来与历史拟合的步骤类似，关键区别在于`make_future_dataframe`函数中的`periods`参数。例如，设置`periods=180`，表示在历史数据结束后，再预测未来180天的值。

![](img/93c9f35621157dfb90036080a497af08_121.png)

![](img/93c9f35621157dfb90036080a497af08_123.png)

此时生成的`future`数据框将包含未来的日期，对其进行预测后，`forecast`对象中也会包含未来日期的预测值`yhat`和置信区间。绘图时，未来时间段只有绿色的预测线及置信区间，没有黑色观测点。

![](img/93c9f35621157dfb90036080a497af08_125.png)

从预测结果图可以看到，模型基于历史趋势对未来做出了延伸预测。预测结果可能显得比较“保守”，没有出现特别剧烈的波动。这是因为模型默认的`changepoint_prior_scale`参数（控制趋势变化灵活度）设置为一个适中的值。**通过调整这个参数，我们可以让模型的预测显得更“激进”或更“平稳”**。

![](img/93c9f35621157dfb90036080a497af08_127.png)

![](img/93c9f35621157dfb90036080a497af08_129.png)
