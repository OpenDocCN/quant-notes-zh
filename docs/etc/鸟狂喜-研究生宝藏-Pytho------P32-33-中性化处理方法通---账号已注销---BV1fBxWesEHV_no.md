## 什么是中性化？

上一节我们介绍了多种量化因子，本节中我们来看看**中性化**。它的核心目的是进行**提纯**。

为了理解“提纯”，我们先看一个例子。假设我们设计了一个选股策略，使用了A、B、C、D四个不同的因子。理论上，这四个因子应该从不同角度帮助我们筛选股票。

然而，在实际操作中，无论我们如何调整这四个因子的组合，最终选出的股票池总是高度相似。为什么会这样？

原因可能在于，这四个因子虽然名义上不同，但其内部绝大部分信息都与同一个底层因素（例如**市值**）高度相关。例如：
*   因子A（如市净率）的数值很大程度上受市值影响。
*   因子B、C、D的数值也主要受市值影响。

这样一来，我们看似使用了四个因子，但实际上起决定性作用的几乎都是“市值”这一个因素。我们无法获得因子A、B、C、D各自独特的、有价值的信息。这就像从四个不同的人身上，只找到了他们共同的特点，而忽略了每个人的个性。

**中性化**要做的，就是从每个因子中，剥离掉这些共性的、我们不希望它主导结果的干扰因素（如市值的影响），从而“提纯”出该因子自身独特的、有价值的信息部分。这个过程就叫做中性化。

## 中性化在量化中的应用

在量化交易中，我们经常使用多个指标（因子）对股票池进行筛选。但在使用因子选股时，有时会因为这些因子共同受到某个强相关因素（如市值、行业）的影响，导致选出的股票具有我们不希望的倾向性，例如总是集中在某类大盘股或某个行业。

如果不进行中性化处理，选股结果就会比较集中，无法真正体现不同因子的筛选差异。例如，市净率因子与市值通常有很高的相关性，直接使用未经处理的市净率选股，效果会大打折扣。

为了更好地理解后续的实战案例，这里简要介绍我们将用到的因子——**市净率**。

**市净率**的计算公式为：
`市净率(PB) = 每股股价 / 每股净资产`
其中，**每股净资产** = (公司总资产 - 公司总负债) / 总股本，它代表了每股股票所代表的公司净资产的多少。

![](img/cd9039b9f467b8aa3b19657571b42a43_1.png)

在投资中，通常认为较低的市净率可能意味着股票被低估，其未来上涨的潜力或安全边际相对更高。

## 中性化的计算方法

由于在本地Notebook中无法直接获取所需的金融因子数据（如市净率、市值），我们将在量化平台中演示具体操作。这里先讲解其核心计算思想。

中性化通常通过回归分析来实现。以“剔除市值影响”为例，其基本步骤如下：

以下是核心计算步骤：

1.  **建立回归模型**：将被分析的因子（如市净率）作为因变量 `Y`，将需要剔除的干扰因素（如市值）作为自变量 `X`。
    `Y = α + β * X + ε`
    其中，`Y`是因子值，`X`是市值（或其他需要中性化的变量），`α`是截距，`β`是系数，`ε`是残差。

2.  **进行回归拟合**：使用所有股票的数据，通过线性回归等方法，拟合出上述方程，得到系数 `β`。

3.  **计算残差（中性化后的因子）**：对于每一只股票，计算其因子实际值与回归模型预测值之间的差值。
    `中性化因子值 = ε = Y - (α + β * X)`
    这个残差 `ε` 就是剔除了市值影响后，该因子“纯净”的部分。它代表了这只股票的市净率相对于同市值水平股票的平均市净率，是偏高还是偏低。
