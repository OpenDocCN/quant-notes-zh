以下是短期回测（例如几个月）的核心指标观察：
*   **基准收益**：基准组合在该时间段内表现为亏损。
*   **策略收益**：我们的策略虽然没有获得高额利润，但成功稳住了资金，没有出现大幅亏损。
*   **最大回撤**：策略的最大回撤约为17%，处于可接受范围。
*   **超额收益**：策略相对于基准获得了约67%的超额收益，短期表现不错。

除了收益曲线和概要指标，回测系统还提供了详细的交易记录。

![](img/c03e18ca4328821b15c3a518d7668335_56.png)

以下是交易详情与账户信息的解读：
*   **交易详情**：列表显示了每月调仓的具体记录，例如4月、5月、6月等，验证了策略按月调仓的逻辑。
*   **持仓详情**：展示了每次调仓后持有的具体股票和数量。
*   **账户信息**：记录了初始资金（如100万）和最终资金（如77万）的变化，直观反映了策略的盈亏情况。

打分法是一种在量化交易中常用且直观的策略构建方法。它通过综合多个财务或技术指标，对股票进行排序和选择。

![](img/c03e18ca4328821b15c3a518d7668335_64.png)

以下是关于打分策略的总结：
*   **优点**：逻辑简单直接，易于理解和实现，通常能获得一个相对稳定的结果。
*   **局限性**：策略效果受市场环境影响较大。在熊市期间，策略收益可能表现平平甚至不佳，这是市场系统性风险决定的常见现象。
