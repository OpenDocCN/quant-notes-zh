## 随机过程模型回顾

上一节我们介绍了蒙特卡罗模拟的基础。本节中，我们来看看几种关键的随机过程模型，它们是模拟资产价格路径的基础。

![](img/0b311f50277b9da4cc039119e5d2bdb0_3.png)

### 奥恩斯坦-乌伦贝克过程

![](img/0b311f50277b9da4cc039119e5d2bdb0_5.png)

首先分析奥恩斯坦-乌伦贝克过程。该过程的积分结果 `ET` 经过数学分析，其均值为零，方差为以下表达式的正态分布。

`ET ~ N(0, Var(ET))`

![](img/0b311f50277b9da4cc039119e5d2bdb0_7.png)

由于前面部分均为常数，因此 `XT` 的均值是前一部分，方差是后一部分的正态分布。所以，OU过程本质上是一个正态分布。

### CIR过程及其变体

![](img/0b311f50277b9da4cc039119e5d2bdb0_9.png)

接下来看一个更复杂的CIR过程。CIR过程本身不像OU过程或几何布朗运动那样有解析解。它是一个满足均值回复的过程。

![](img/0b311f50277b9da4cc039119e5d2bdb0_11.png)

均值回复的速率为 `K`，长期均值为 `X_bar`。公式中的前一部分称为漂移项，后一部分是随机项，即一个维纳过程乘以标准差 `σ` 和 `√Xt`。

![](img/0b311f50277b9da4cc039119e5d2bdb0_13.png)

`dXt = K(X_bar - Xt)dt + σ√Xt dWt`

![](img/0b311f50277b9da4cc039119e5d2bdb0_15.png)

CIR过程具有均值回复性质，趋向于回复到长期均值 `X_bar`。其回复的波动与当前股价有关：价格越高，波动率越大，在均值附近震荡的幅度也越高。

![](img/0b311f50277b9da4cc039119e5d2bdb0_17.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_19.png)

CIR过程可以使用欧拉离散化方法进行离散。但这种方法有时会导致 `Xt` 出现负值。为了避免这种情况，可以采用对数欧拉过程。

![](img/0b311f50277b9da4cc039119e5d2bdb0_21.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_23.png)

令 `Yt = log(Xt)`，则等式右边可以改写为相应的表达式。这样可以避免模拟到负的股价。这种对数欧拉过程与CIR过程结合，常用于利率曲线的模拟。在本课程中，我们主要使用几何布朗运动来模拟股价。

## 欧式期权定价的蒙特卡罗模拟

理解了基础模型后，我们具体来看欧式期权的蒙特卡罗模拟定价。

![](img/0b311f50277b9da4cc039119e5d2bdb0_25.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_27.png)

首先，我们假设标的资产价格满足几何布朗运动。满足该运动，可以得到期末 `T` 时刻股价的解析式：

![](img/0b311f50277b9da4cc039119e5d2bdb0_28.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_30.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_32.png)

`ST = S0 * exp((r - σ²/2)T + σ√T * Z)`

![](img/0b311f50277b9da4cc039119e5d2bdb0_34.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_36.png)

其中：
*   `S0` 是初始股价。
*   `r` 是无风险利率。
*   `σ` 是股价波动率。
*   `Z` 是一个标准正态分布 `N(0,1)` 的随机变量。

![](img/0b311f50277b9da4cc039119e5d2bdb0_38.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_40.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_42.png)

对于亚式期权，其收益依赖于整条价格路径，并非仅依赖于到期日价格。其收益是股价均值减去行权价 `K`。因此，我们不能直接使用期末值，而需要模拟出每一个小时间点 `ti` 的股价。

![](img/0b311f50277b9da4cc039119e5d2bdb0_44.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_46.png)

`收益 = max(平均(S_ti) - K, 0)`

![](img/0b311f50277b9da4cc039119e5d2bdb0_48.png)

以下是常见期权的收益函数：
*   **看涨期权**：`max(ST - K, 0)`
*   **看跌期权**：`max(K - ST, 0)`
*   **亚式期权**：`max(平均(S_ti) - K, 0)`

![](img/0b311f50277b9da4cc039119e5d2bdb0_50.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_52.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_54.png)

### 蒙特卡罗模拟的误差分析

![](img/0b311f50277b9da4cc039119e5d2bdb0_56.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_58.png)

在期权定价中，我们需要了解蒙特卡罗模拟的误差具体是多少。

我们知道样本均值 `μ_hat` 满足正态分布，其均值就是真实的 `μ`（无偏估计），方差为 `σ/√N`，其中 `N` 是模拟的路径数量。

![](img/0b311f50277b9da4cc039119e5d2bdb0_60.png)

这意味着误差服从以零为均值、`σ/√N` 为标准差的正态分布。我们可以构造统计量 `Y`：

`Y = (μ_hat - μ) / (σ/√N) ~ N(0,1)`

通常，我们需要设定一个置信水平。例如，我们希望误差小于某个值（如1%）的概率为99.7%。通过标准正态分布表可知，当概率为99.7%时，`Y` 的绝对值约为3。

因此，`3 * σ / √N = 0.01`。可以反解出 `N`：

`N = (3 * σ / 0.01)²`

这样，我们就得到了需要模拟的次数，以确保我们有99.7%的信心认为误差小于1%。

在模拟过程中，我们需要收集以下信息：
*   **样本均值**：用于估计真实均值 `μ`。
*   **样本方差**：用于估计真实方差 `σ²`。我们需要在每一步迭代中更新样本均值和样本方差。

### Python 实现示例

让我们看一个Python示例，定价一个欧式看跌期权。

![](img/0b311f50277b9da4cc039119e5d2bdb0_62.png)

假设参数如下：
*   到期时间 `T = 1` 年
*   时间步数 `N_steps = 52`（每周一步）
*   无风险利率 `r = 0.1`
*   初始股价 `S0 = 100`
*   行权价 `K = 100`
*   波动率 `σ = 0.25`
*   模拟路径数 `N_paths = 400000`

根据布莱克-斯科尔斯公式计算的理论价格为 `5.46`。通过蒙特卡罗模拟计算的价格约为 `5.47`，误差为 `0.015`。

以下是核心代码步骤的简述：
1.  定义所有参数（`T`, `r`, `S0`, `K`, `σ`, `N_paths`, `N_steps`）。
2.  离散化时间轴。
3.  导入或编写几何布朗运动的路径生成函数。
4.  定义看跌期权的收益函数：`max(K - ST, 0)`。
5.  对每条路径计算收益，并乘以折现因子 `exp(-r*T)`。
6.  计算所有折现后收益的均值作为期权价格估计。
7.  计算样本标准差以估计误差。
8.  可视化前100条模拟路径。

## 美式期权定价

![](img/0b311f50277b9da4cc039119e5d2bdb0_64.png)

接下来，让我们来介绍美式期权的定价。美式期权在基本概念上与欧式期权类似。

![](img/0b311f50277b9da4cc039119e5d2bdb0_66.png)

回顾欧式期权的参数：
*   `T`：到期时间
*   `r`：无风险利率
*   `S0`：初始股价
*   `K`：行权价
*   `σ`：标的资产波动率

美式期权与欧式期权的唯一区别在于**它可以提前执行**。

![](img/0b311f50277b9da4cc039119e5d2bdb0_68.png)

### 提前执行的理论分析

![](img/0b311f50277b9da4cc039119e5d2bdb0_70.png)

对于美式看涨期权，如果标的股票**没有分红**，则**永远不应提前执行**。原因有三：
1.  **货币的时间价值**：提前执行需要立即支付行权价 `K`，不如将这笔资金进行无风险投资，到期再执行。
2.  **下行风险保护**：持有期权而非股票，可以避免股价下跌带来的损失。如果提前执行，就失去了期权的保护作用。
3.  **期权价值**：看涨期权的价格总是高于其内在价值。提前执行只能获得内在价值，而出售期权可以获得更高的期权价格。

然而，对于美式看跌期权，情况则不同，有时提前执行可能是最优选择。

### 定价挑战与最小二乘蒙特卡罗方法

由于可能提前执行，我们不能使用简单的蒙特卡罗方法为美式期权定价。

在每一个时间点 `ti`，美式期权的价值 `V(ti)` 是以下两者的最大值：
`V(ti) = max( IntrinsicValue(ti), E[ e^{-rΔt} * V(ti+1) | F(ti) ] )`
其中，`IntrinsicValue(ti)` 是立即执行的收益，第二项是继续持有期权的期望折现值。

![](img/0b311f50277b9da4cc039119e5d2bdb0_72.png)

问题在于，对于单条模拟路径上的一个点，我们无法计算条件期望 `E[...]`。如果粗暴地用下一条路径的折现值代替，会严重丧失准确性。

![](img/0b311f50277b9da4cc039119e5d2bdb0_74.png)

因此，我们采用**最小二乘蒙特卡罗** 方法来拟合最优执行边界。

![](img/0b311f50277b9da4cc039119e5d2bdb0_76.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_78.png)

### LSM 算法步骤

![](img/0b311f50277b9da4cc039119e5d2bdb0_80.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_82.png)

整个LSM算法是向后递推的：
1.  **初始化**：在到期日 `T`，期权价值等于其收益。
2.  **向后递推**：对于时间点 `t = T-1, T-2, ..., 0`：
    a. 收集所有在 `t` 时刻处于“实值”状态的路径。
    b. 以这些路径在 `t` 时刻的股价为自变量，以其在下一时刻 `t+1` 的期权价值折现到 `t` 时刻的值 `Y` 为因变量。
    c. 使用一组基函数（例如，股价的二次多项式：`1, S, S²`）对 `(S, Y)` 进行最小二乘回归，拟合出条件期望函数 `E[Y|S]`。
    d. 对于每条路径，比较立即执行的收益和回归得到的继续持有价值 `E[Y|S]`。
    e. 如果立即执行收益更高，则在该路径的 `t` 时刻执行，期权价值等于执行收益；否则，期权价值等于折现后的下一时刻价值。
3.  **得出价格**：将所有路径在 `t=0` 时刻的期权价值取平均，即得到美式期权的估计价格。

![](img/0b311f50277b9da4cc039119e5d2bdb0_84.png)

![](img/0b311f50277b9da4cc039119e5d2bdb0_86.png)

*   `fit_u`: 拟合特定时间点的条件期望函数（执行边界）。
