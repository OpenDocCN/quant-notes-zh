## 字符串常量和变量

字符串是编程中处理文本数据的基本类型。字符串常量是用单引号 `'` 或双引号 `"` 括起来的任意文本，例如 `'Hello'` 或 `"Python"`。常量意味着其值在程序运行中不会改变。

当字符串本身包含引号、换行等特殊字符时，需要使用反斜杠 `\` 进行转义。例如，`\'` 表示一个单引号字符，`\"` 表示一个双引号字符。

字符串变量则是我们自定义的、可以存储字符串值的变量。例如：
```python
name = "崔老师"
```

---

## 字符串的索引与切片

Python 提供了灵活的方式来访问字符串中的单个或多个字符。

字符串中的每个字符都有一个位置索引。索引可以从左向右（正索引），从 `0` 开始计数；也可以从右向左（负索引），从 `-1` 开始计数。

例如，对于字符串 `name = "ABCDEF"`：
*   `name[1]` 的值是 `'B'`
*   `name[-2]` 的值是 `'E'`

除了访问单个字符，我们还可以通过“切片”操作获取字符串的一部分。切片使用冒号 `:` 分隔起始和结束索引，其格式为 `[start:end]`，注意结果包含起始索引的字符，但不包含结束索引的字符。

以下是几个切片示例：
*   `name[1:4]` 的结果是 `'BCD'`
*   `name[:3]` 的结果是 `'ABC'`（从开头到索引2）
*   `name[3:]` 的结果是 `'DEF'`（从索引3到结尾）

## 字符串的基本运算

Python 为字符串提供了几种基本的运算操作。

以下是字符串支持的运算：

*   **连接（+）**：将两个字符串拼接在一起。例如：`"Hello" + "World"` 的结果是 `"HelloWorld"`。
*   **重复（*）**：将字符串重复指定次数。例如：`"Hi" * 3` 的结果是 `"HiHiHi"`。
*   **索引（[]）**：通过索引获取字符串中特定位置的字符。
*   **切片（[:]）**：通过切片获取字符串的一个子串。

## 字符串格式化输出

![](img/37a1abaf711a2419f9081ff4424366eb_21.png)

在实际应用中，我们经常需要将变量值嵌入到一段固定的文本中进行输出，这就需要用到字符串格式化。

### 方法一：使用百分号 `%` 格式化

![](img/37a1abaf711a2419f9081ff4424366eb_23.png)

这种方法使用 `%` 作为占位符，在字符串后通过 `%` 连接一个元组来提供实际值。

常用的格式符有：
*   `%d`：格式化整数。
*   `%f`：格式化浮点数，可以指定精度，如 `%.2f` 表示保留两位小数。
*   `%s`：格式化字符串。

![](img/37a1abaf711a2419f9081ff4424366eb_25.png)

![](img/37a1abaf711a2419f9081ff4424366eb_26.png)

示例代码如下：
```python
name = "崔老师"
age = 45
height = 1.74
print("我是%s，今年%d岁，身高%.2f米" % (name, age, height))
```

![](img/37a1abaf711a2419f9081ff4424366eb_27.png)

### 方法二：使用 `format()` 方法格式化

![](img/37a1abaf711a2419f9081ff4424366eb_29.png)

这是更现代、更灵活的格式化方法。它在字符串中使用花括号 `{}` 作为占位符，然后调用字符串的 `.format()` 方法传入变量。

![](img/37a1abaf711a2419f9081ff4424366eb_31.png)

`format()` 方法支持按位置传递参数，也支持按名称传递参数。

![](img/37a1abaf711a2419f9081ff4424366eb_33.png)

![](img/37a1abaf711a2419f9081ff4424366eb_35.png)

示例代码如下：
```python
name = "崔老师"
age = 45
height = 1.74
print("我是{}，今年{}岁，身高{:.2f}米".format(name, age, height))
# 或指定顺序
print("我是{0}，今年{1}岁，身高{2:.2f}米".format(name, age, height))
```
