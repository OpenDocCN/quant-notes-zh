![](img/951a42a82f4615098f0d2c9f7b884c1b_2.png)

## 计算基础收益率

![](img/951a42a82f4615098f0d2c9f7b884c1b_4.png)

上一节我们介绍了如何通过均线交叉生成交易信号。本节中，我们来看看如何基于这些信号计算策略的实际收益。

首先，我们需要计算股票每日的基础收益率。在金融分析中，为了便于进行多期收益的累加，我们通常使用对数收益率。对数收益率的计算公式如下：

**公式：** `log_return = log(price_t / price_{t-1})`

以下是计算对数收益率的代码步骤：

![](img/951a42a82f4615098f0d2c9f7b884c1b_6.png)

```python
# 假设‘price’是包含股价的Pandas Series
# 计算对数收益率
log_returns = np.log(price / price.shift(1))
```

![](img/951a42a82f4615098f0d2c9f7b884c1b_8.png)

这段代码计算了当前价格与前一日价格比值的自然对数，其结果就是每日的对数收益率。

![](img/951a42a82f4615098f0d2c9f7b884c1b_10.png)

## 应用策略信号调整收益

![](img/951a42a82f4615098f0d2c9f7b884c1b_12.png)

在计算出基础收益率后，我们需要根据之前生成的交易信号来调整它。我们的策略信号（`position`）是一个由1和-1组成的序列。

![](img/951a42a82f4615098f0d2c9f7b884c1b_14.png)

*   当 `position` 为 1 时，表示我们遵循原始市场走势（即做多）。
*   当 `position` 为 -1 时，表示我们采取与原始市场走势相反的操作（例如，在预测下跌时做空，从而在市场下跌时获利）。

![](img/951a42a82f4615098f0d2c9f7b884c1b_16.png)

因此，策略的每日收益率可以通过将基础收益率与策略信号相乘得到：

**公式：** `strategy_return = position * log_return`

![](img/951a42a82f4615098f0d2c9f7b884c1b_18.png)

以下是应用策略信号计算策略收益的代码：

![](img/951a42a82f4615098f0d2c9f7b884c1b_20.png)

```python
# 假设‘position’是交易信号序列，同样需要进行滞后一期处理，以避免使用未来数据
strategy_log_returns = position.shift(1) * log_returns
```

![](img/951a42a82f4615098f0d2c9f7b884c1b_22.png)

注意，这里对 `position` 也使用了 `.shift(1)` 操作，这是为了确保我们在第t天使用的是第t-1天收盘后产生的信号，避免“前瞻性偏差”，即使用未来信息进行交易决策。

![](img/951a42a82f4615098f0d2c9f7b884c1b_24.png)

![](img/951a42a82f4615098f0d2c9f7b884c1b_26.png)

## 比较策略最终收益

![](img/951a42a82f4615098f0d2c9f7b884c1b_28.png)

计算出调整后的策略收益率序列后，我们需要将其与原始收益率进行比较，看策略是否带来了超额收益。

![](img/951a42a82f4615098f0d2c9f7b884c1b_30.png)

![](img/951a42a82f4615098f0d2c9f7b884c1b_32.png)

由于我们使用的是对数收益率，将一段时间内的所有每日对数收益率相加，即可得到该时间段的总对数收益率。最后，通过指数函数 `exp()` 将其还原为最终的价值比率。

![](img/951a42a82f4615098f0d2c9f7b884c1b_34.png)

以下是计算并比较最终收益的代码：

![](img/951a42a82f4615098f0d2c9f7b884c1b_36.png)

```python
# 删除可能存在的缺失值（例如第一行由于shift操作产生的NaN）
clean_log_returns = log_returns.dropna()
clean_strategy_returns = strategy_log_returns.dropna()

![](img/951a42a82f4615098f0d2c9f7b884c1b_38.png)

# 计算总对数收益率
total_log_return = clean_log_returns.sum()
total_strategy_log_return = clean_strategy_returns.sum()

# 将总对数收益率还原为最终的价值倍数
final_value_buy_hold = np.exp(total_log_return)  # 买入持有的最终价值
final_value_strategy = np.exp(total_strategy_log_return) # 策略的最终价值

![](img/951a42a82f4615098f0d2c9f7b884c1b_40.png)

![](img/951a42a82f4615098f0d2c9f7b884c1b_42.png)

print(f"买入持有策略：1元最终变为 {final_value_buy_hold:.2f} 元")
print(f"双均线策略：1元最终变为 {final_value_strategy:.2f} 元")
```

![](img/951a42a82f4615098f0d2c9f7b884c1b_44.png)

在这个示例中，我们可能看到类似的结果：
*   买入持有：1元 → 约4元
*   双均线策略：1元 → 约5.8元

![](img/951a42a82f4615098f0d2c9f7b884c1b_46.png)

这表明，在我们回测的这段历史数据中，应用双均线策略获得了比简单持有更高的收益。

![](img/951a42a82f4615098f0d2c9f7b884c1b_48.png)

## 总结

![](img/951a42a82f4615098f0d2c9f7b884c1b_50.png)

本节课中我们一起学习了量化策略收益分析的核心步骤。

![](img/951a42a82f4615098f0d2c9f7b884c1b_52.png)

1.  **计算基础收益**：我们使用对数收益率来度量每日的价格变化，便于多期累加。
2.  **应用策略调整**：我们将生成的交易信号（如双均线交叉产生的 `position` 序列）与基础收益率相乘，得到策略的每日收益率。关键是要对信号进行滞后处理，确保回测的严谨性。
3.  **评估策略效果**：通过累加策略期的对数收益率并还原为最终价值，与基准（如买入持有）的最终价值进行对比，从而客观评估策略是否创造了超额收益。

![](img/951a42a82f4615098f0d2c9f7b884c1b_54.png)

![](img/951a42a82f4615098f0d2c9f7b884c1b_56.png)

这个过程是量化回测的基础，通过它我们可以定量地判断一个交易思路在历史数据上的表现，为策略优化提供依据。
