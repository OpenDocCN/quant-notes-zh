## 精英中的精英：新财富分析师

在分析师群体中，“新财富分析师”被认为是王牌中的王牌。这些分析师是由社保、公募、保险、私募等专业机构投资者投票评选出的。由于这些机构是分析师的主要服务对象，因此由他们来投票比较合理。分析师推荐的股票能否帮这些机构赚钱，是投票的重要参考依据。

可以说，新财富分析师大部分是业内公认的佼佼者。当然，有投票的地方就有“江湖”，这个评选也存在一些场外因素。

![](img/99304283d3e3dae25910cbb00651cab2_26.png)

那么，这个群体到底是有用还是没用？是专家还是“砖家”？大家各有看法。但在量化交易者看来，分析师工作的核心是推荐股票，评判其业务能力的最佳指标，就是他们推荐股票的实际收益。

## 数据准备与处理思路

![](img/99304283d3e3dae25910cbb00651cab2_34.png)

上一节我们介绍了新财富分析师，本节中我们来看看如何用数据验证他们的荐股能力。

![](img/99304283d3e3dae25910cbb00651cab2_36.png)

2022年的新财富分析师评选已经结束。我们整理出了一份名单，共涉及25个行业，120位明星分析师。接下来，我们将用数据验证这120多位分析师在2022年推荐的股票，如果跟着买到底能不能赚钱。

![](img/99304283d3e3dae25910cbb00651cab2_38.png)

进行这样的研究，光有名单不够，还需要知道他们在2022年具体推荐了哪些股票。分析师通常以研究报告的形式发布推荐。一份研报的第一页会明确指出，某位分析师在某年某月某日推荐了某只股票，并给出“买入”等评级。整篇报告则包含具体的分析。

![](img/99304283d3e3dae25910cbb00651cab2_40.png)

![](img/99304283d3e3dae25910cbb00651cab2_42.png)

需要注意的是，不同券商使用的评级话术可能不同，但大致可以理解为分析师在当下对具体股票给出的“优良中差”评级。

![](img/99304283d3e3dae25910cbb00651cab2_44.png)

个人投资者一般很难直接看到这些研报，通常需要付费。当然，现在很多分析师也会把研报发布在自己的公众号上。分析师每年发布的研报数量巨大，大约在4万篇左右，手动统计非常繁琐。

![](img/99304283d3e3dae25910cbb00651cab2_46.png)

![](img/99304283d3e3dae25910cbb00651cab2_48.png)

因此，我们准备了一份从2008年至今的完整历史数据。这份数据包含了每位分析师撰写的每一篇研报，记录了发布日期、推荐的股票、分析师姓名以及给出的评级。

![](img/99304283d3e3dae25910cbb00651cab2_50.png)

我们从这份数据中筛选出2022年所有新财富分析师的相关记录。筛选后的数据如下图所示。我们发现，分析师可能对同一只股票连续发布多次研报。

![](img/99304283d3e3dae25910cbb00651cab2_52.png)

为了方便验证，我们只保留分析师**第一次**对某只股票给出**最高评级**的那次记录。例如，某分析师多次推荐“某某股份”，第一次给出第二档评级，第二次才给出第一档的最高评级，那么我们只保留第二次的记录。

![](img/99304283d3e3dae25910cbb00651cab2_54.png)

![](img/99304283d3e3dae25910cbb00651cab2_56.png)

我们将研究分析师在给出最高评级后，该股票在未来5天、20天、60天以及120天的收益表现。对所有分析师都进行相同的处理。

![](img/99304283d3e3dae25910cbb00651cab2_58.png)

虽然可以使用Excel完成这个统计，但过程会比较复杂。我们将借助Python代码来实现，高效地查看结果。

## 分析结果：2022年新财富分析师表现

![](img/99304283d3e3dae25910cbb00651cab2_64.png)

运行程序后，我们得到了以下结果。

![](img/99304283d3e3dae25910cbb00651cab2_66.png)

![](img/99304283d3e3dae25910cbb00651cab2_68.png)

在120位新财富分析师中，有97位在2022年以最高评级推荐过股票，占比81%。其中有46人一年内推荐了超过20只股票，甚至有一位“劳模”推荐了102只。

![](img/99304283d3e3dae25910cbb00651cab2_70.png)

在具体推荐中，有些人很成功，其推荐的股票在后续周期内出现了很高的涨幅。当然，也有一些分析师推荐后股票一路下跌。

![](img/99304283d3e3dae25910cbb00651cab2_72.png)

但个人的表现不能说明问题，我们需要看新财富分析师群体的整体表现。我们统计了分析师推荐股票后，在未来一段时间内的平均涨幅。

![](img/99304283d3e3dae25910cbb00651cab2_74.png)

具体结果如图所示。可以看到，在推荐后的5天和20天，平均涨幅是下跌的；但在60天和120天后，平均涨幅转为上涨。然而，不管在哪个周期，他们推荐股票的胜率都远低于50%，即下跌的股票比上涨的多。

![](img/99304283d3e3dae25910cbb00651cab2_76.png)

这个结果能直接说明分析师不给力吗？显然不能。因为2022年代表大盘的沪深300指数本身下跌了22%。因此，评价分析师的表现不能只看绝对涨幅，而要看**相对表现**，即推荐的股票是否比大盘少跌或多涨。

![](img/99304283d3e3dae25910cbb00651cab2_78.png)

![](img/99304283d3e3dae25910cbb00651cab2_80.png)

我们通过程序计算了推荐股票相对于大盘的平均超额收益。结果显示，无论是短期还是中长期，这些金牌分析师推荐的股票都能稳定跑赢大盘。特别是在推荐120天后，相比大盘有约10%的超额收益。

![](img/99304283d3e3dae25910cbb00651cab2_82.png)

当然，也有观点认为，新财富分析师是做行业研究的，他们只在单一行业内推荐股票。如果某个行业整体比大盘涨得多，那么该行业的分析师也会受益。因此，应该与相应的行业指数对比，例如医药分析师的推荐应与医药行业指数对比。

![](img/99304283d3e3dae25910cbb00651cab2_84.png)

这个想法非常正确。我们也计算了相对于行业指数的结果。总体来看，分析师推荐的股票仍然可以跑赢自身所在的行业指数。

至此，我们可以基本确定，2022年的新财富分析师们算是实至名归，他们在2022年推荐的股票确实表现不错。

## 深入思考：对实际投资的真正帮助

恭喜还没有离开的同学，因为接下来要讲的才是真正的干货。

![](img/99304283d3e3dae25910cbb00651cab2_90.png)

我们之前的分析证明了2022年最优秀的新财富分析师，在2022年推荐的股票确实不错。但这除了证明评选结果并非乱投以外，对我们2023年的实际投资有什么帮助呢？其实帮助不大。

![](img/99304283d3e3dae25910cbb00651cab2_92.png)

只有证明**去年的**明星分析师在**今年**推荐的股票仍然有超额收益，才对投资者有持续的帮助。就像一个理论，如果只能解释过去而不能预测未来，那就毫无意义。

当然，2023年尚未结束，我们无法知道2022年明星分析师在2023年的表现。但我们可以研究**2021年**的新财富明星分析师，看看他们在接下来的2022年推荐的股票表现如何。

我们自己的实盘策略中，就有一些是基于分析师推荐的股票来选股的。这体现了量化投资的一个核心思路：我们有时不需要通过复杂的财务分析或技术分析来预测股票涨跌，而是通过量化方法，找出市场上经常做对的人，跟着他操作。

我们不需要保证每个分析师推荐的股票未来一定上涨，只需要保证他们**整体是有效的**，那么同样可以得到一个优秀的投资策略。

## 分析师行业细节与信息滞后性

![](img/99304283d3e3dae25910cbb00651cab2_114.png)

关于分析师行业，还有很多细节值得了解。只有了解这些，才能更好地基于其荐股数据构建策略。

![](img/99304283d3e3dae25910cbb00651cab2_116.png)

例如，分析师如果想推荐某只股票，他第一时间通常不会发布研报，而是优先在自己的客户交流群内通知，或直接私信服务的基金经理。甚至在非正式场合就把信息传递了。之后，才会发布相关的研报，而这些研报有时是助理撰写的。

![](img/99304283d3e3dae25910cbb00651cab2_118.png)

因此，当我们看到公开研报时，信息已经有一定的滞后性。但即便如此，大家也看到了，这样的信息仍然能够产生超额收益。

![](img/99304283d3e3dae25910cbb00651cab2_120.png)

所以，不要抱怨信息的滞后性。滞后的信息也有价值，关键在于使用数据的人是否会利用。

## 总结

![](img/99304283d3e3dae25910cbb00651cab2_126.png)

本节课中，我们一起学习了如何利用Python和大数据分析券商分析师荐股的有效性。我们通过处理历史研报数据，计算了2022年及2021年新财富分析师推荐股票的短期、中长期收益，并与大盘及行业指数进行了对比。

![](img/99304283d3e3dae25910cbb00651cab2_128.png)

关键结论如下：
1.  新财富分析师推荐的股票，在中长期（60天、120天）能产生正向的平均绝对收益。
2.  更重要的是，这些股票在**所有观察周期内都能稳定跑赢大盘和行业指数**，产生超额收益。
3.  这种有效性具有一定的持续性，上一年的明星分析师在次年依然能提供有价值的推荐。
4.  这为量化投资提供了一个实用思路：**跟随市场上持续有效的信号源**，而非盲目预测。

![](img/99304283d3e3dae25910cbb00651cab2_130.png)

量化投资的魅力在于用数据验证想法，将市场中的规律转化为可执行的策略。希望本教程能帮助你打开思路，用更客观、更数据驱动的方式看待市场信息。
