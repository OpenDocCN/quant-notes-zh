## 概述

在命令行程序中，我们经常需要获取用户输入的信息。Python提供了一个非常简单的内置函数 `input()` 来实现这个功能。本节我们将学习如何使用 `input()` 函数，并处理输入数据的类型转换。

## 读取用户输入

读取用户输入的核心指令是 `input()` 函数。它是一个内置函数，可以接收一个字符串参数作为给用户的提示信息。

![](img/51686b1df05662f1a2bfa4aea5ee463e_3.png)

![](img/51686b1df05662f1a2bfa4aea5ee463e_5.png)

**代码示例：**
```python
input("请输入您的密码：")
```
执行这行代码时，程序会暂停，等待用户在命令行中输入内容，输入完成后按回车键，程序才会继续执行。

![](img/51686b1df05662f1a2bfa4aea5ee463e_7.png)

为了后续使用用户输入的值，我们需要将其存储在一个变量中。

**代码示例：**
```python
username = input("请输入您的用户名：")
print(username)
```
这样，用户输入的内容就被保存在变量 `username` 中，并可以打印出来。

![](img/51686b1df05662f1a2bfa4aea5ee463e_9.png)

## 实践：收集用户信息

上一节我们介绍了如何使用 `input()` 函数。本节中我们来看看如何用它来编写一个收集用户基本信息的小程序。

以下是创建一个收集姓名、年龄、家乡和爱好信息的程序步骤：

1.  使用 `input()` 函数提示用户输入姓名，并存入变量 `name`。
2.  使用 `input()` 函数提示用户输入年龄，并存入变量 `age`。
3.  使用 `input()` 函数提示用户输入家乡，并存入变量 `hometown`。
4.  使用 `input()` 函数提示用户输入爱好，并存入变量 `hobby`。
5.  最后，使用 `print()` 函数将所有信息打印出来。

**完整代码示例：**
```python
name = input("请输入您的姓名：")
age = input("请输入您的年龄：")
hometown = input("请输入您的家乡：")
hobby = input("请输入您的爱好：")

print(name)
print(age)
print(hometown)
print(hobby)
```
运行程序后，它会依次提示用户输入，并将最终结果输出到屏幕上。

## 处理输入数据的类型

现在，我们来尝试一个计算题：让用户输入两个数字，然后程序计算它们的乘积。

**初步尝试代码：**
```python
number1 = input("请输入第一个数字：")
number2 = input("请输入第二个数字：")
print(number1 * number2)
```
运行这段代码并输入数字（例如10和6）后，程序会报错：`TypeError: can‘t multiply sequence by non-int of type ‘str’`。错误信息表明，我们试图将两个字符串进行乘法运算，这是不允许的。

为什么输入的数字会变成字符串？这是因为 **`input()` 函数获取的所有输入，都会以字符串（str）类型返回**，即使输入的内容看起来是数字。

为了验证这一点，我们可以使用 `type()` 函数查看变量的数据类型。

**代码示例：**
```python
print(type(number1), type(number2))
```
输出结果会显示两者都是 `<class ‘str’>`。

为了让数字计算能够正常进行，我们需要将字符串转换为整数（int）。转换方法非常简单，使用 `int()` 函数即可。

**修正后的代码：**
```python
number1 = input("请输入第一个数字：")
number2 = input("请输入第二个数字：")

# 将字符串转换为整数
number1_int = int(number1)
number2_int = int(number2)

print(number1_int * number2_int)
```
这样，两个数字就能正确相乘了。

同理，我们也可以使用 `str()` 函数将整数转换回字符串。

**代码示例：**
```python
result = number1_int * number2_int
print("乘积是：" + str(result))
```
**重要提示：** `int()` 函数只能转换“看起来像整数”的字符串，例如 `“10”`、`“-5”`。如果尝试转换 `“Alex”` 这样的字符串，程序将会报错。
