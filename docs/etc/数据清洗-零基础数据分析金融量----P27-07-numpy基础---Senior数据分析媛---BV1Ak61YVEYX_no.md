## NumPy排序的两种方式

在NumPy中，排序有两种基本方式：改变原始数组和不改变原始数组。

### 生成随机数据

首先，我们使用 `np.random.permutation` 函数生成一组随机索引，用于演示排序。

```python
import numpy as np
# 生成0到9的随机排列
data = np.random.permutation(10)
print("原始数据：", data)
```

这个函数生成的是一组不重复的随机索引。我们可以利用它对数组进行随机重排。

```python
# 创建一个5x5的随机数组
arr = np.random.randint(0, 10, size=(5, 5))
print("原始数组：\n", arr)
# 使用随机索引重排行
shuffled_arr = arr[data[:5]] # 取前5个索引对应行
print("重排后的数组：\n", shuffled_arr)
```

### 1. 不改变原始数组的排序

这种方式会返回一个新的排序后的数组，原始数组保持不变。

```python
# 不改变原始数据
sorted_values = np.sort(data)
print("排序后的值（新数组）：", sorted_values)
print("原始数据未改变：", data)
```

### 2. 改变原始数组的排序

这种方式直接在原始数组的内存位置进行排序，没有返回值。

```python
# 改变原始数据
data.sort() # 注意：这是数组对象的方法
print("排序后（原始数据已改变）：", data)
```

这种方式适用于数据量较小、需要就地排序的场景，例如按姓名或成绩排序。

## 部分排序：`np.partition`

当数据量很大时（例如几万或几十万条），我们可能只关心数据的前几个最大值或最小值，而不需要进行全局排序。这时可以使用部分排序 `np.partition` 来提高效率。

![](img/31a273c429eb0dd2c3914c058009fa3d_7.png)

`np.partition` 函数接受一个参数 `k` 作为分界点。
*   当 `k` 为正数时，返回数组中最小的 `k` 个数（位于数组前 `k` 位）。
*   当 `k` 为负数时，返回数组中最大的 `|k|` 个数（位于数组后 `|k|` 位）。

分界点之外的数据顺序是不确定的。

```python
# 生成1万个随机数
large_data = np.random.permutation(10000)
print("大型数据集（前10个）：", large_data[:10])

# 获取最小的2个数
partitioned_small = np.partition(large_data, 2)
print("部分排序后（取最小2个），前5个值：", partitioned_small[:5])
# 此时前两个数是最小的，但后面的数顺序是乱的

# 获取最大的2个数
partitioned_large = np.partition(large_data, -2)
print("部分排序后（取最大2个），后5个值：", partitioned_large[-5:])
# 此时最后两个数是最大的，但前面的数顺序是乱的
```

**效率对比**：部分排序的效率远高于全局排序，尤其是在只需要获取头部或尾部部分数据时。

```python
import timeit
# 测试部分排序（取最大2个）的时间
time_partial = timeit.timeit(lambda: np.partition(large_data.copy(), -2), number=100)
print(f"部分排序100次平均时间：{time_partial/100*1e6:.2f} 微秒")

![](img/31a273c429eb0dd2c3914c058009fa3d_9.png)

# 测试全局排序的时间

```

## 算法原理简述（以冒泡排序为例）

为了帮助理解部分排序的效率，我们简要回顾一下冒泡排序的原理。

假设对数组 `[3, 2, 5, 6, 7, 4, 1, 9, 8]` 进行排序：
1.  **第一轮**：相邻元素两两比较，大的下沉。经过本轮，**最大值9**被确定并移动到末尾。
2.  **第二轮**：对剩余元素（最大值之前的区间）进行同样操作，确定**次大值8**。
3.  以此类推...

如果我们只想要最大的两个数（9和8），那么进行两轮排序后就可以停止，无需完成全部 `n-1` 轮排序。这就是部分排序能够提升效率的核心思想。但需要注意的是，在部分排序中，分界点之外的元素顺序是没有保证的。
