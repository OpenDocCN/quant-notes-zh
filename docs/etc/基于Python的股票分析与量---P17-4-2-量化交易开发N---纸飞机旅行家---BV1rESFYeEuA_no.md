## 📈 移动平均线简介

移动平均线是分析时间序列数据趋势的常用工具。在股票分析中，常见的移动平均线包括5日均线、10日均线、20日均线等。它通过计算指定时间窗口内股价的平均值，来平滑短期波动，揭示长期趋势。

## 🧮 核心数学工具：卷积

在介绍具体均线算法前，我们先了解一个关键的数学工具：卷积。卷积描述了两个函数相互作用，生成第三个函数的过程。在信号处理中，它可以用来描述过去信号对当前状态的影响，是一种时空响应的叠加。在计算移动平均时，卷积提供了一种非常便捷的实现方式。

Numpy库中提供了卷积函数 `np.convolve`。其基本用法如下：

```python
import numpy as np
result = np.convolve(a, b, mode='full')
```
其中 `a` 和 `b` 是两个一维数组。`mode` 参数决定了输出数组的长度：
*   `‘full’`：默认模式，返回长度为 `N+M-1` 的完整卷积结果。
*   `‘same’`：返回与最长输入数组长度相同的中心部分。
*   `‘valid’`：只返回完全重叠的点，长度为 `max(M, N) - min(M, N) + 1`。

我们将主要使用 `‘full’` 模式。

## 📊 简单移动平均线

简单移动平均线是最基础的均线类型。它计算指定时间窗口内股价的算术平均值，且窗口内每个数据点的权重相等（等权）。例如，5日均线就是最近5个交易日收盘价的平均值。

以下是计算简单移动平均线的步骤：

1.  **生成权重数组**：根据目标时间窗口（如5日），创建一个长度相同、元素值均为 `1/窗口长度` 的一维数组。这代表了等权。
2.  **卷积运算**：将股价序列与上一步生成的权重数组进行卷积运算。
3.  **结果截取**：由于卷积结果的边缘效应，我们需要截取有效的中心部分作为最终的均线值。
4.  **可视化**：使用绘图库将原始股价与计算出的均线绘制在一起进行对比。

## 📉 指数移动平均线

指数移动平均线是对简单移动平均线的改进。它的核心区别在于，EMA赋予近期价格更高的权重，而远期价格的权重则以指数形式衰减。这使得EMA对价格的最新变化反应更灵敏。

以下是计算指数移动平均线的步骤，它比SMA多了一个权重衰减的步骤：

1.  **权重初始化**：创建一个从0到`窗口长度-1`的数组。
2.  **指数衰减**：对上一步的数组应用指数函数 `np.exp`，并进行线性变换（如使用 `np.linspace` 生成从-1到0的序列），以实现权重的指数衰减。
3.  **权重归一化**：将衰减后的权重数组归一化，使其总和为1。
4.  **卷积运算**：将股价序列与归一化后的权重数组进行卷积。
5.  **结果截取与可视化**：与SMA类似，截取有效结果并绘图。

## 💻 代码实战

现在，让我们通过代码来具体实现上述两种移动平均线的计算。我们将使用平安银行2009年1月至2月初的股价数据作为示例，并借助 `matplotlib` 库进行可视化。

首先，我们实现简单移动平均线：

```python
import numpy as np
import matplotlib.pyplot as plt

# 假设 `close_prices` 是包含收盘价的一维Numpy数组
close_prices = np.array([...]) # 你的股价数据

window = 5 # 设置时间窗口，例如5日均线
# 1. 生成等权权重数组
weights = np.ones(window) / window
# 2. 进行卷积运算
sma_full = np.convolve(weights, close_prices, mode='full')
# 3. 截取有效结果（中心部分，长度与原始数据相同）
sma = sma_full[window-1: -window+1]

# 4. 可视化
plt.plot(close_prices, label='Close Price')
plt.plot(sma, label=f'{window}-Day SMA')
plt.legend()
plt.show()
```

接下来，我们实现指数移动平均线：

```python
window = 5 # 时间窗口
# 1. & 2. 创建并应用指数衰减权重
x = np.arange(window) # 生成数组 [0, 1, 2, 3, 4]
# 使用线性空间和指数函数创建衰减权重
weights_ema = np.exp(np.linspace(-1, 0, window))
# 3. 权重归一化
weights_ema /= weights_ema.sum()
# 4. 卷积运算
ema_full = np.convolve(weights_ema, close_prices, mode='full')
# 5. 截取有效结果并可视化
ema = ema_full[window-1: -window+1]

plt.plot(close_prices, label='Close Price')
plt.plot(ema, label=f'{window}-Day EMA')
plt.legend()
plt.show()
```

运行代码后，你可以观察到EMA线比SMA线更加贴近近期的股价变化，对趋势的反应更为迅速。

## 📝 总结

本节课中我们一起学习了股价分析中两个重要的趋势指标：简单移动平均线和指数移动平均线。

*   我们首先引入了**卷积**的概念，它作为计算移动平均的数学工具，能够便捷地处理时间序列的滑动窗口计算。
*   **简单移动平均线**在时间窗口内采用等权计算，实现简单，是趋势分析的基础。
*   **指数移动平均线**引入了权重衰减机制，赋予近期数据更高权重，因此对价格新变化的敏感度更高。
*   在代码实战部分，我们利用 `numpy.convolve` 函数实现了两种均线的计算，并使用 `matplotlib` 将结果可视化，直观地比较了它们与原始股价的趋势关系。
