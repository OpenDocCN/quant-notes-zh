## 风险敞口决定潜在收益 💡

上一节我们介绍了多种策略和因子挖掘方法。本节中我们来看看风险控制。从本质上说，一个策略的潜在收益是由其承担的风险决定的。风险和收益同源。

![](img/c6c86d49d920819716fbda3bdb487338_3.png)

![](img/c6c86d49d920819716fbda3bdb487338_4.png)

![](img/c6c86d49d920819716fbda3bdb487338_6.png)

![](img/c6c86d49d920819716fbda3bdb487338_8.png)

一个简单的例子是购买保险。购买保险需要支付保费，而保险公司根据平均出险率计算，可以获得盈利。作为保险买家，我们付出保费，回报是在重大突发事件时获得损失弥补。

回到金融市场，有两种类似的收益风格：
*   **购买保险者**：希望在市场剧烈波动或下跌时获得额外收益。这类投资者需要接受其回报率可能低于市场平均回报的事实。
*   **出售保险者**：通过赚取额外的风险溢价来获得更高回报。其更高的收益率并非源于投资水平更高，而是承担了额外的风险溢价。

因此，类似于保险公司的利润，如果某些资产容易受到特定风险影响，购买这些资产就可以获得额外的风险补偿回报。

**核心观点**：风险敞口决定了我们的收益。投资者投资股票而非更安全的债券，是期待从股票中获得额外的回报，这部分回报来自于一系列的风险因素。

## 风险因子与风险偏好 🔄

![](img/c6c86d49d920819716fbda3bdb487338_10.png)

![](img/c6c86d49d920819716fbda3bdb487338_12.png)

我们强调的风险因子，例如公司规模（市值），在历史上曾被认为是阿尔法收益，但后来被论证为一种风险因子。不同时期，大小市值公司的表现会轮动。

在债券中，长期债券的收益率通常高于短期债券。公式可以表示为：
**长期债券收益率 ≈ 短期债券收益率 + 期限风险溢价**
因为出借更长期限的资金需要承担更大的不确定性风险，因此要求获得更多的风险补偿。

持续的**风险溢价**会导致不同资产类别（如股票与债券、长期债与短期债）之间的收益率产生区别。

另一个关键点是**风险偏好**。人们对风险的偏好并非一成不变，而是随时间变化的量。如果能够捕捉到人们风险偏好的变化，就存在“低买高卖”的交易机会。我们赌的是风险偏好会出现均值回归。

![](img/c6c86d49d920819716fbda3bdb487338_14.png)

然而，这一点需要谨慎对待。市场由投资者的信心和交易构成，不存在完全正确的估值或市场状态，这需要每个人自行判断。

![](img/c6c86d49d920819716fbda3bdb487338_16.png)

## 夏普比率与收益分布 📈

作为一个理性的投资者，我们关心资产的**夏普比率**。其公式为：
**夏普比率 = (资产预期收益率 - 无风险利率) / 资产收益率的标准差**

但一个理性的假设是：对于具有相同夏普比率的两个资产，我们在选择时没有偏好。然而，这个假设可能并不完全正确。因为具有相同夏普比率的资产，可能存在不同的收益分布。

*   **一种分布**：类似于购买保险，大部分时间小幅亏损，偶尔获得大额收益。
*   **另一种分布**：类似于出售保险，大部分时间稳定赚取小钱，但在突发事件时可能产生大额亏损。

这两种分布体现的是**偏度**的问题。负偏态意味着收益分布左侧有长尾（偶尔大亏），正偏态意味着收益分布右侧有长尾（偶尔大赚）。

以下我们将通过编程，生成具有相同夏普比率但不同偏度的收益分布。

### 生成特定偏度的收益分布

我们将编写一个函数，根据给定的年化夏普比率、目标偏度、目标波动率和样本数量，生成相应的收益分布序列。

```python
import numpy as np
import matplotlib.pyplot as plt

![](img/c6c86d49d920819716fbda3bdb487338_18.png)

def generate_skewed_returns(target_sharpe, target_skew, target_vol, num_samples, days_in_year=252):
    """
    生成具有特定夏普比率和偏度的日度收益率序列。
    
    参数:
    target_sharpe: 目标年化夏普比率
    target_skew: 目标偏度
    target_vol: 目标年化波动率
    num_samples: 生成的样本数量
    days_in_year: 一年中的交易日数，默认为252
    
    返回:
    samples: 生成的日度收益率序列
    """
    # 计算年化收益率和目标日度波动率
    annual_return = target_sharpe * target_vol  # 简化假设无风险利率为0
    daily_vol = target_vol / np.sqrt(days_in_year)
    daily_return = annual_return / days_in_year
    
    # 使用伽马分布来构造具有特定偏度的分布
    # 伽马分布的形状参数 (shape parameter)
    shape_param = 4.0 / (abs(target_skew) + 1e-6) ** 2  # 加极小值防止除零
    
    # 伽马分布的尺度参数 (scale parameter)
    scale_param = daily_vol / np.sqrt(shape_param)
    
    # 从伽马分布中采样
    gamma_samples = np.random.gamma(shape_param, scale_param, num_samples)
    
    # 调整符号以匹配目标偏度的方向
    adjusted_sign = -1.0 if target_skew < 0 else 1.0
    adjusted_samples = [x * adjusted_sign for x in gamma_samples]
    
    # 计算当前样本的均值，并调整到目标日度收益率
    current_mean = np.mean(adjusted_samples)
    adjustment = daily_return - current_mean
    final_samples = [x + adjustment for x in adjusted_samples]
    
    return final_samples

# 生成正偏度和负偏度的收益序列示例
np.random.seed(42)  # 设置随机种子以保证结果可复现
positive_skew_returns = generate_skewed_returns(target_sharpe=1.0, target_skew=1.0,
                                                 target_vol=0.20, num_samples=2500)
negative_skew_returns = generate_skewed_returns(target_sharpe=1.0, target_skew=-1.0,
                                                 target_vol=0.20, num_samples=2500)

![](img/c6c86d49d920819716fbda3bdb487338_22.png)

![](img/c6c86d49d920819716fbda3bdb487338_23.png)

![](img/c6c86d49d920819716fbda3bdb487338_25.png)

![](img/c6c86d49d920819716fbda3bdb487338_27.png)

# 绘制直方图进行可视化
fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].hist(positive_skew_returns, bins=50, edgecolor='black', alpha=0.7)
axes[0].set_title('正偏度收益分布 (大部分小亏，偶尔大赚)')
axes[0].set_xlabel('日度收益率')
axes[0].set_ylabel('频率')

axes[1].hist(negative_skew_returns, bins=50, edgecolor='black', alpha=0.7, color='orange')
axes[1].set_title('负偏度收益分布 (大部分小赚，偶尔大亏)')
axes[1].set_xlabel('日度收益率')
axes[1].set_ylabel('频率')

plt.tight_layout()
plt.show()
```

运行以上代码，我们可以直观地看到两种分布形态的差异。这两种形态孰优孰劣取决于个人或机构的风格。

*   **正偏度策略**（如趋势跟踪、买入期权）：大部分时间可能承受小额亏损或交易成本，但能限制最大亏损，在黑天鹅事件中抗冲击力较强。这符合机构投资者风控要求，但执行起来可能“反人性”。
*   **负偏度策略**（如做市、卖出波动率、套利）：大部分时间稳定赚取小钱，但极端行情下可能产生巨大回撤。个人投资者“止盈不止损”的行为容易导致这种分布。

**核心要点**：给定相同的年化波动率和收益率，可以产生完全不同的收益形态。在策略评估时，必须注意其中可能暗含的偏度风险溢价。

## 其他重要风险类型 ⚠️

除了上述讨论，在实际交易中还需考虑以下几种风险：

以下是几种关键的风险类型：
1.  **周期性风险与资产配置**：人们的风险偏好随经济周期变化。例如，美林投资时钟模型阐述了在不同经济阶段（复苏、过热、滞胀、衰退）应配置不同类型的资产（股票、债券、商品、现金）。在通胀时期，持有过多现金会导致实际购买力下降。
2.  **流动性风险**：当资金规模较大时，需要考虑资产的流动性。在小盘股中投入大量资金可能难以在不显著影响价格的情况下快速清仓。机构投资者偏好股票、期货等流动性好的资产。低流动性资产（如房地产、私募股权）通常要求更高的流动性风险补偿，这也能部分解释小市值公司的高收益。
3.  **交易壁垒风险**：看到市场机会不代表能参与。例如，外国投资者投资A股有渠道限制；高频交易需要昂贵的硬件和链路投入；私募股权投资需要尽职调查成本。必须权衡收益与这些壁垒成本。
4.  **行为金融学风险**：投资者心理可能导致非理性行为，如倾向于“止盈”兑现成功感，而厌恶“止损”承认错误。基于行为金融学的策略可以挖掘这类市场非有效性。
5.  **运营风险**：技术故障可能导致灾难性损失，例如骑士资本因高频交易系统错误在45分钟内巨亏4.6亿美元。这强调了系统稳定性和风控流程的重要性。
