## 核心概念解析

上一节我们介绍了课程的整体目标，本节中我们来看看固定收益分析中的几个核心概念。理解这些概念是进行后续量化建模的基础。

### 计息基准

计息基准是计算债券利息的基础规则。它定义了如何计算两个付息日之间的应计利息。

**公式**：`应计利息 = 票面金额 × 票面利率 × (计息天数 / 计息基准)`

### 到期收益率

到期收益率是使债券未来现金流的现值等于其当前市场价格的贴现率。它是衡量债券投资回报的关键指标。

**公式**：`债券价格 = Σ (现金流_t / (1 + YTM)^t)`，其中YTM为到期收益率。

## 主要分析工具

掌握了基本概念后，我们来看看在固定收益分析中会用到的主要工具和模型。

### 收益率曲线

收益率曲线描绘了不同期限债券的收益率之间的关系。它是分析利率风险和进行产品定价的基石。

以下是构建收益率曲线的常见步骤：
*   收集不同期限的标准债券的市场数据。
*   利用插值法（如样条插值）拟合出连续的曲线。
*   对曲线进行分析，解读市场对未来利率的预期。

### 债券及其衍生品

固定收益市场不仅包括基础债券，还包含由其衍生出的各类金融产品。

以下是常见的产品类型：
*   **普通债券**：定期付息，到期还本。
*   **浮动利率债券**：票面利率随参考利率（如LIBOR）定期调整。
*   **利率互换**：双方交换固定利率和浮动利率现金流的合约。
