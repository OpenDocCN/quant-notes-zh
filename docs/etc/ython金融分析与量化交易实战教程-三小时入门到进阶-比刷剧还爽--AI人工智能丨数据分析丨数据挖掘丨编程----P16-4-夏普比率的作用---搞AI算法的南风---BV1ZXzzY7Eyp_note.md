## 概述

夏普比率是衡量投资组合风险调整后收益的关键指标。它描述了在承担单位风险的情况下，投资者能获得多少超额回报。简单来说，它帮助我们判断一个高收益的投资是否值得其伴随的高风险。

## 夏普比率的核心概念

上一节我们介绍了投资回报率，本节中我们来看看如何结合风险来评估收益的质量。

![](img/d8aeca4fc040f6e4464db36087668f6c_1.png)

夏普比率描述的是：对于每承担一单位的风险，我们能获得多少收益。这个指标越高，意味着在相同的风险水平下，我们能获得更高的回报，因此投资效率也越高。

![](img/d8aeca4fc040f6e4464db36087668f6c_3.png)

在选股或选择投资组合时，如果仅从夏普比率这一指标来看，我们应选择比率更高的选项，因为它代表了更高的风险调整后收益。

![](img/d8aeca4fc040f6e4464db36087668f6c_5.png)

## 夏普比率的计算公式

理解了夏普比率的意义后，我们来看看如何具体计算它。

计算夏普比率需要比较两个关键数值：一是你投资组合的收益率，二是市场的无风险收益率（例如国债利率）。我们需要计算这两者之间的差值，即超额收益。

然后，再用这个超额收益除以你投资组合收益率的标准差（代表风险）。公式如下：

**夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差**

在具体计算年化夏普比率时，公式通常写作：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   `Rp` 代表投资组合的平均收益率。
*   `Rf` 代表无风险收益率。
*   `σp` 代表投资组合收益率的标准差。

## 使用Python计算夏普比率

接下来，我们将在Python中实践这个计算过程。假设我们已经有了股票的历史回报率数据。

以下是计算夏普比率的步骤：

![](img/d8aeca4fc040f6e4464db36087668f6c_7.png)

首先，我们需要准备数据。在计算回报率时，我们通常需要处理缺失值。一种常见的方法是使用前一天的数值进行填充。

```python
# 假设 returns 是包含各股票每日回报率的DataFrame
# 使用前向填充方法处理缺失值
returns_filled = returns.fillna(method='ffill')
```

然后，我们定义无风险收益率（例如年化5%），并根据交易日数量（假设一年250个交易日）将其转化为日度利率。

```python
# 定义年化无风险收益率
risk_free_rate = 0.05
# 转化为日度无风险收益率
daily_risk_free_rate = risk_free_rate / 250
```

最后，我们按照公式计算每只股票的夏普比率。计算年化夏普比率时，需要将日度标准差进行年化处理（乘以交易日数的平方根）。

```python
import numpy as np

# 计算超额收益（日度）
excess_returns = returns_filled - daily_risk_free_rate

![](img/d8aeca4fc040f6e4464db36087668f6c_9.png)

# 计算夏普比率
# 1. 计算超额收益的均值（日度）
mean_excess_return = excess_returns.mean()
# 2. 计算回报率的标准差（日度）
std_daily_return = returns_filled.std()
# 3. 计算年化夏普比率
sharpe_ratios = (mean_excess_return * 250) / (std_daily_return * np.sqrt(250))

![](img/d8aeca4fc040f6e4464db36087668f6c_11.png)

print(sharpe_ratios)
```

![](img/d8aeca4fc040f6e4464db36087668f6c_13.png)

运行代码后，我们会得到每只股票的夏普比率。数值越大，代表该股票在承担单位风险时获得的收益越高。例如，在结果中，“中兴”的夏普比率最高，意味着它在我们评估的股票中风险收益效率最高。负的夏普比率通常意味着收益还不如无风险利率，不是理想的选择。
