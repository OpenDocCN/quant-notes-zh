## 概述

## 条件求和：`SUMIF`函数

![](img/e080225803dfcde9941f24bdc3012a29_5.png)

![](img/e080225803dfcde9941f24bdc3012a29_7.png)

当我们需要对满足特定条件的数据进行求和时，可以使用`SUMIF`函数。例如，计算员工级别为“A类”的销售额总和。

以下是`SUMIF`函数的使用步骤：

![](img/e080225803dfcde9941f24bdc3012a29_9.png)

1.  **函数结构**：`=SUMIF(range, criteria, [sum_range])`
    *   `range`：条件区域，即用于条件判断的单元格区域。
    *   `criteria`：条件，即需要满足的标准。
    *   `[sum_range]`：求和区域，即实际需要求和的单元格区域。如果省略，则对`range`区域求和。

![](img/e080225803dfcde9941f24bdc3012a29_11.png)

2.  **具体操作**：
    *   在目标单元格输入公式：`=SUMIF(C:C, “A类”, D:D)`
    *   其中，`C:C`是“员工级别”列（条件区域），`“A类”`是条件，`D:D`是“销售额”列（求和区域）。
    *   按下回车键，即可得到A类员工的销售额总和。

这个函数非常适合对单一条件的数据进行快速汇总。当你有多个类别（如A类、B类、C类）需要分别求和时，使用`SUMIF`函数配合单元格引用可以高效地批量完成。

## 多条件求和：`SUMIFS`函数

如果求和的条件不止一个，我们就需要使用`SUMIFS`函数。例如，计算“工作年限大于5年”且“员工级别为A类”的销售额总和。

以下是`SUMIFS`函数的使用步骤：

1.  **函数结构**：`=SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...)`
    *   `sum_range`：求和区域，即实际需要求和的单元格区域。
    *   `criteria_range1`：第一个条件区域。
    *   `criteria1`：第一个条件。
    *   `[criteria_range2, criteria2], ...`：可选的其他条件区域和条件，可以添加多组。

2.  **具体操作**：
    *   在目标单元格输入公式：`=SUMIFS(D:D, B:B, “>5”, C:C, “A类”)`
    *   其中，`D:D`是“销售额”列（求和区域），`B:B`是“工作年限”列（第一个条件区域），`“>5”`是第一个条件，`C:C`是“员工级别”列（第二个条件区域），`“A类”`是第二个条件。
    *   按下回车键，即可得到满足两个条件的销售额总和。

![](img/e080225803dfcde9941f24bdc3012a29_13.png)

与`SUMIF`不同，`SUMIFS`将求和区域放在了参数的第一位，这样在设置多个条件时逻辑更清晰。

![](img/e080225803dfcde9941f24bdc3012a29_15.png)

## 条件计数：`COUNTIF`函数

学会了条件求和，条件计数就很容易理解了。`COUNTIF`函数用于统计满足特定条件的单元格数量。例如，统计员工级别为“A类”的员工有多少个。

以下是`COUNTIF`函数的使用步骤：

![](img/e080225803dfcde9941f24bdc3012a29_19.png)

1.  **函数结构**：`=COUNTIF(range, criteria)`
    *   `range`：需要计数的单元格区域。
    *   `criteria`：计数的条件。

![](img/e080225803dfcde9941f24bdc3012a29_21.png)

2.  **具体操作**：
    *   在目标单元格输入公式：`=COUNTIF(C:C, “A类”)`
    *   其中，`C:C`是“员工级别”列，`“A类”`是条件。
    *   按下回车键，即可得到A类员工的数量。

`COUNTIF`函数比`SUMIF`更简单，因为它只需要指定在哪里找（区域）和找什么（条件），然后直接返回符合条件的记录条数。

## 多条件计数：`COUNTIFS`函数

同样地，当计数条件有多个时，我们使用`COUNTIFS`函数。例如，统计“工作年限大于等于10年”且“销售额大于等于2000”的员工有多少个。

以下是`COUNTIFS`函数的使用步骤：

1.  **函数结构**：`=COUNTIFS(criteria_range1, criteria1, [criteria_range2, criteria2], ...)`
    *   `criteria_range1`：第一个条件区域。
    *   `criteria1`：第一个条件。
    *   `[criteria_range2, criteria2], ...`：可选的其他条件区域和条件。

![](img/e080225803dfcde9941f24bdc3012a29_23.png)

![](img/e080225803dfcde9941f24bdc3012a29_25.png)

2.  **具体操作**：
    *   在目标单元格输入公式：`=COUNTIFS(B:B, “>=10”, D:D, “>=2000”)`
    *   其中，`B:B`是“工作年限”列（第一个条件区域），`“>=10”`是第一个条件，`D:D`是“销售额”列（第二个条件区域），`“>=2000”`是第二个条件。
    *   按下回车键，即可得到同时满足两个条件的员工数量。

`COUNTIFS`函数可以看作是`COUNTIF`的多条件版本，其参数结构与`SUMIFS`类似，只是去掉了最前面的求和区域。

![](img/e080225803dfcde9941f24bdc3012a29_27.png)

## 总结

本节课中我们一起学习了四个强大的条件统计函数：
*   **`SUMIF`**：对满足**单一条件**的数据进行求和。
*   **`SUMIFS`**：对满足**多个条件**的数据进行求和。
*   **`COUNTIF`**：统计满足**单一条件**的单元格数量。
*   **`COUNTIFS`**：统计满足**多个条件**的单元格数量。

![](img/e080225803dfcde9941f24bdc3012a29_29.png)

掌握这些函数，你就能在数据分析中轻松实现按条件汇总和计数，这是处理业务数据、进行量化分析的基础技能。请务必通过练习来熟悉每个函数的参数顺序和写法。
