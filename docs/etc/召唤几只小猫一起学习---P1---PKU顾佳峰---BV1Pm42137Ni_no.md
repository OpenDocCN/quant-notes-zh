# 召唤几只小猫一起学习：P1：基础概念与准备

在本节课中，我们将学习如何通过编程“召唤”几只小猫，并让它们一起学习。这是一个入门级的编程概念，我们将从最基础的部分开始，确保初学者能够轻松理解。

## 概述

我们将从理解“召唤”和“小猫”这两个核心概念开始。在编程中，“召唤”通常意味着创建或实例化一个对象，而“小猫”可以代表一个具有特定属性和行为的实体。本节课的目标是掌握创建多个对象并让它们执行简单任务的基本方法。

上一节我们介绍了课程的整体目标，本节中我们来看看具体需要做哪些准备工作。

## 核心概念：对象与实例

在编程中，**对象**是数据和相关操作的集合。我们可以把“小猫”看作一个对象。每只具体的小猫，都是这个对象的一个**实例**。创建实例的过程，就是我们所说的“召唤”。

我们可以用一个简单的类定义来描述小猫：

```python
class Cat:
    def __init__(self, name):
        self.name = name

    def learn(self):
        print(f"{self.name} 正在学习。")
```

上面的代码定义了一个 `Cat` 类。`__init__` 是一个特殊方法，用于在创建实例（即召唤小猫）时初始化其属性，比如名字。`learn` 是这个小猫对象可以执行的一个行为。

## 召唤多只小猫

理解了如何定义一个小猫类之后，我们就可以召唤多只小猫了。以下是召唤三只小猫并让它们开始学习的步骤：

![](img/6331d902e928061820a90bb52308289d_3.png)

1.  **实例化对象**：使用类来创建具体的实例。
2.  **存储对象**：通常将创建的对象存储在列表或字典中，以便管理。
3.  **调用方法**：让每个实例执行其行为。

让我们看看具体的代码实现：

```python
# 召唤三只小猫，并给它们起名
cat1 = Cat("咪咪")
cat2 = Cat("喵喵")
cat3 = Cat("小白")

# 将小猫们放入一个列表
cat_class = [cat1, cat2, cat3]

# 让所有小猫一起学习
for cat in cat_class:
    cat.learn()
```

运行这段代码，你将在控制台看到三只小猫依次开始学习的输出。
