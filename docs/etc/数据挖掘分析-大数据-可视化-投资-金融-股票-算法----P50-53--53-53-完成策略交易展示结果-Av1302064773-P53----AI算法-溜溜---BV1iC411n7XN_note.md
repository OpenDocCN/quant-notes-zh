## 获取当前持仓股票

![](img/bcf99f54102b08d931892a0825a4f933_8.png)

![](img/bcf99f54102b08d931892a0825a4f933_10.png)

上一节我们介绍了如何根据财务指标对股票进行排名。本节中，我们来看看如何根据新的排名结果进行调仓。首先，我们需要一个函数来获取当前投资组合中持有的所有股票。

![](img/bcf99f54102b08d931892a0825a4f933_12.png)

以下是 `get_holdings` 函数的实现步骤：

![](img/bcf99f54102b08d931892a0825a4f933_14.png)

1.  从 `context` 对象中获取当前的持仓信息 `positions`。
2.  遍历 `positions` 列表。
3.  检查每只股票的持仓数量 `quantity`。
4.  如果持仓数量大于0，则将该股票代码加入持仓列表。
5.  返回最终的持仓股票列表。

![](img/bcf99f54102b08d931892a0825a4f933_16.png)

```python
def get_holdings(context):
    # 初始化持仓列表
    holdings = []
    # 获取当前所有持仓
    positions = context.portfolio.positions
    # 遍历持仓
    for position in positions:
        # 检查持仓数量是否大于0
        if positions[position].quantity > 0:
            # 将股票代码加入列表
            holdings.append(position)
    # 返回持仓列表
    return holdings
```

## 计算交易清单

![](img/bcf99f54102b08d931892a0825a4f933_18.png)

获取了当前持仓后，我们需要将其与最新的股票排名进行比较，以确定需要进行的交易。

以下是计算买入和卖出清单的逻辑：

1.  调用 `get_holdings(context)` 函数获取当前持仓列表 `holdings`。
2.  将最新的股票排名列表 `stocks` 和持仓列表 `holdings` 都转换为集合（Set）类型，以便进行集合运算。
3.  需要**买入**的股票是：在最新排名中，但不在当前持仓中的股票。公式为：`to_buy = set(stocks) - set(holdings)`。
4.  需要**卖出**的股票是：在当前持仓中，但不在最新排名中的股票。公式为：`to_sell = set(holdings) - set(stocks)`。

```python
# 获取当前持仓
holdings = get_holdings(context)
# 将列表转换为集合
stocks_set = set(stocks)
holdings_set = set(holdings)

![](img/bcf99f54102b08d931892a0825a4f933_20.png)

# 计算需要买入的股票
to_buy = stocks_set - holdings_set
# 计算需要卖出的股票
to_sell = holdings_set - stocks_set
```

## 执行卖出操作

在量化交易中，通常遵循“先卖后买”的原则，以确保有足够的资金进行买入。现在，我们来执行卖出操作。

![](img/bcf99f54102b08d931892a0825a4f933_26.png)

以下是卖出操作的实现步骤：

1.  遍历需要卖出的股票列表 `to_sell`。
2.  对每只股票，使用 `order_target_percent` 函数，将其目标持仓比例设置为0%，即清仓该股票。

```python
# 执行卖出操作
for stock in to_sell:
    # 将股票持仓比例调整为0%，即全部卖出
    order_target_percent(stock, 0)
```

## 执行买入操作

![](img/bcf99f54102b08d931892a0825a4f933_30.png)

卖出完成后，我们获得了现金，接下来执行买入操作。我们需要将资金平均分配到每一只需要买入的股票上。

![](img/bcf99f54102b08d931892a0825a4f933_32.png)

以下是买入操作的实现步骤：

![](img/bcf99f54102b08d931892a0825a4f933_34.png)

1.  检查 `to_buy` 列表是否为空。如果为空，则直接返回，无需操作。
2.  计算可用于投资的**总资产价值**。公式为：`total_value = context.portfolio.portfolio_value`。
3.  计算每只待买入股票应分配的资金。公式为：`per_stock_value = total_value / len(to_buy)`。
4.  遍历需要买入的股票列表 `to_buy`。
5.  对每只股票，使用 `order_target_value` 函数，买入指定金额的股票。

![](img/bcf99f54102b08d931892a0825a4f933_36.png)

![](img/bcf99f54102b08d931892a0825a4f933_38.png)

```python
# 执行买入操作
if len(to_buy) == 0:
    # 如果没有需要买入的股票，则返回
    return
else:
    # 计算总资产
    total_value = context.portfolio.portfolio_value
    # 计算每只股票应买入的金额（平均分配）
    per_stock_value = total_value / len(to_buy)
    # 遍历买入列表
    for stock in to_buy:
        # 买入指定金额的股票
        order_target_value(stock, per_stock_value)
```

在代码编写过程中，常见的错误包括：
*   **标点符号错误**：如使用中文逗号 `，` 代替英文逗号 `,`。
*   **类型错误**：对列表直接进行减法操作。需要先将列表转换为集合。
*   **语法错误**：如缺少括号、引号或冒号。

![](img/bcf99f54102b08d931892a0825a4f933_58.png)
