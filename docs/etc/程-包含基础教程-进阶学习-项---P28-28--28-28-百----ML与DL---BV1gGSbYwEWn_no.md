## 什么是因子？🔍

因子是影响最终结果（如股票收益）的指标。在投资股票时，投资者通常会依据某些标准进行筛选，这些标准就是因子。例如，市净率较低的股票可能被认为有更大的上涨空间；营收增长率较高的公司可能被视为优质投资标的。因此，因子可以理解为对投资结果产生影响的各类指标。

## 因子数据处理概述 📈

当我们获得因子数据后，不能直接用于建模，需要进行预处理。这类似于数据挖掘任务，我们需要探究每个因子（X1, X2, X3...）对最终收益（Y）的影响，并从中选择合适的因子。

以下是因子数据预处理的三个核心步骤：
1.  **去极值**：处理数据中的离群点或异常值。
2.  **标准化**：将不同量纲和取值范围的因子数据转换到统一的尺度上。
3.  **中性化**：消除因子数据中与某些系统性风险（如行业、市值）的相关性，这是量化金融中的特殊步骤。

## 第一步：去极值处理 🎯

极值是指数据中远离主体分布的数据点。直接删除这些数据可能造成信息损失，因此更常见的做法是“拉回”这些极值，使其不超过我们设定的合理边界。

### 百分位去极值法

一种常用的去极值方法是基于分位数（百分位）的方法。分位数可以帮助我们理解数据的分布情况。

![](img/23f7ffea0bd63ec156c3f87120163522_5.png)

以下是几个关键的分位数概念：
*   **中位数 (Q2)**：将数据从小到大排列后，处于正中间位置的值。对于奇数个数据，中位数是中间那个数；对于偶数个数据，中位数是中间两个数的平均值。中位数对极值不敏感，能更好地反映数据的中心趋势。
*   **下四分位数 (Q1)**：所有数据中，处于25%位置的值。
*   **上四分位数 (Q3)**：所有数据中，处于75%位置的值。

![](img/23f7ffea0bd63ec156c3f87120163522_7.png)

**四分位距 (IQR)** 是一个重要的概念，它衡量了数据中间50%部分的离散程度，计算公式为：
`IQR = Q3 - Q1`

![](img/23f7ffea0bd63ec156c3f87120163522_9.png)

![](img/23f7ffea0bd63ec156c3f87120163522_11.png)

基于IQR，我们可以设定正常值的范围。通常，将下界设为 `Q1 - k * IQR`，上界设为 `Q3 + k * IQR`，其中 `k` 是一个常数（常用1.5或3）。任何低于下界或高于上界的值都被视为极值，并将其数值调整到对应的边界值上。

![](img/23f7ffea0bd63ec156c3f87120163522_13.png)

```python
# 示例：使用pandas计算分位数和IQR
import pandas as pd

![](img/23f7ffea0bd63ec156c3f87120163522_15.png)

![](img/23f7ffea0bd63ec156c3f87120163522_17.png)

# 假设 factor_data 是一个包含因子值的Pandas Series
Q1 = factor_data.quantile(0.25)
Q3 = factor_data.quantile(0.75)
IQR = Q3 - Q1

# 设定边界（这里k=1.5）
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

![](img/23f7ffea0bd63ec156c3f87120163522_19.png)

# 将极值“拉回”到边界
factor_data_clipped = factor_data.clip(lower_bound, upper_bound)
```

### 为何使用分位数而非均值？

均值容易受到极端值的影响。例如，若将公司高管的极高薪资与普通员工薪资一起计算平均工资，结果会显著偏高，无法代表大多数员工的真实情况。而中位数（Q2）作为分位数的一种，能避免这个问题，更能反映数据的典型情况。

def winsorize_by_quantile(series, k=1.5):
    Q1 = series.quantile(0.25)
    Q3 = series.quantile(0.75)
    IQR = Q3 - Q1
    lower_limit = Q1 - k * IQR
    upper_limit = Q3 + k * IQR
    return series.clip(lower_limit, upper_limit)
