在本节课中，我们将学习如何计算和比较量化交易策略的收益。我们将基于之前构建的双均线策略，分析其相对于“买入并持有”策略的收益表现。核心在于理解如何计算每日收益率，并根据策略信号调整收益计算方式，最终评估策略的有效性。

上一节我们介绍了如何通过双均线交叉生成交易信号（`position`）。本节中我们来看看如何利用这些信号来计算策略的最终收益，并与不做任何操作的基准收益进行对比。

## 计算基准收益率

![](img/c5076ca133111d17398e5e99a04aa235_5.png)

首先，我们需要计算股票每日的基准收益率。这代表了在不采取任何策略、单纯持有股票的情况下，每日的价值变化率。

![](img/c5076ca133111d17398e5e99a04aa235_7.png)

![](img/c5076ca133111d17398e5e99a04aa235_9.png)

计算公式为：
`收益率 = (当日收盘价 / 前一日收盘价) - 1`

![](img/c5076ca133111d17398e5e99a04aa235_11.png)

在Python的pandas中，我们可以通过以下代码实现：
```python
# 假设data是一个包含‘Close’列的DataFrame
data[‘returns’] = data[‘Close’] / data[‘Close’].shift(1) - 1
```
这里，`shift(1)`函数将收盘价序列向下移动一位，使得当日的收盘价可以与前一日的收盘价进行运算。

![](img/c5076ca133111d17398e5e99a04aa235_13.png)

## 计算策略收益率

![](img/c5076ca133111d17398e5e99a04aa235_15.png)

接下来，我们将策略信号应用到收益率计算中。我们的`position`列值为1或-1，其含义如下：
*   **`position = 1`**：表示遵循原始市场走势（即基准收益率）。
*   **`position = -1`**：表示进行反向操作，即在市场下跌时我们获得正收益，在市场上涨时我们获得负收益（这在实际交易中通常通过做空实现）。

![](img/c5076ca133111d17398e5e99a04aa235_17.png)

因此，策略的每日收益率计算公式为：
`策略收益率 = position * 基准收益率`

![](img/c5076ca133111d17398e5e99a04aa235_19.png)

![](img/c5076ca133111d17398e5e99a04aa235_21.png)

以下是实现代码：
```python
# 注意：position也需要进行shift(1)操作，以确保使用前一日生成的信号来影响当日的收益
data[‘strategy’] = data[‘position’].shift(1) * data[‘returns’]
```
执行这段代码后，`data` DataFrame中会新增一列`strategy`，它代表了根据双均线策略调整后的每日收益率。

![](img/c5076ca133111d17398e5e99a04aa235_23.png)

## 累计收益与结果对比

![](img/c5076ca133111d17398e5e99a04aa235_25.png)

![](img/c5076ca133111d17398e5e99a04aa235_27.png)

![](img/c5076ca133111d17398e5e99a04aa235_29.png)

为了评估长期表现，我们需要将每日的收益率累计起来，计算总收益。这涉及到将收益率序列还原为累计净值。

以下是计算累计总收益的步骤：
1.  **处理缺失值**：由于使用了`shift`操作，数据首行会产生缺失值（NaN），需要先将其填充为0。
2.  **计算累计和**：对处理后的收益率序列进行累加。
3.  **还原为净值**：将对数收益率（或简单收益率累加）通过指数函数还原为最终的累计净值。对于简单收益率，累计净值公式为 `(1 + 收益率).cumprod()`。但这里我们使用的是累加和对数近似，最终通过 `np.exp(累计和)` 来还原。

![](img/c5076ca133111d17398e5e99a04aa235_31.png)

实现代码如下：
```python
# 1. 填充缺失值
data[[‘returns’, ‘strategy’]] = data[[‘returns’, ‘strategy’]].fillna(0)

![](img/c5076ca133111d17398e5e99a04aa235_33.png)

![](img/c5076ca133111d17398e5e99a04aa235_35.png)

# 2. & 3. 计算累计总收益（假设我们使用了对数收益率，此处进行还原）
# 如果returns是简单收益率，则应使用 (1+data[‘returns’]).cumprod()
cumulative_returns = np.exp(data[‘returns’].cumsum())
cumulative_strategy = np.exp(data[‘strategy’].cumsum())

![](img/c5076ca133111d17398e5e99a04aa235_37.png)

# 查看最终的总收益倍数
final_return = cumulative_returns.iloc[-1] # 基准最终净值
final_strategy = cumulative_strategy.iloc[-1] # 策略最终净值

![](img/c5076ca133111d17398e5e99a04aa235_39.png)

print(f“基准总收益: {final_return:.2f}“)
print(f“策略总收益: {final_strategy:.2f}“)
```
通过对比`final_return`和`final_strategy`，我们可以直观地看到策略是否跑赢了市场。例如，如果初始资金为1元：
*   基准收益为4元，意味着1元变成了4元。
*   策略收益为5.8元，意味着1元变成了5.8元。
这清楚地表明，在该段历史数据上，我们的双均线策略获得了超越基准的收益。

![](img/c5076ca133111d17398e5e99a04aa235_41.png)

![](img/c5076ca133111d17398e5e99a04aa235_43.png)

本节课中我们一起学习了量化策略收益分析的全过程。我们从计算每日基准收益率开始，接着将策略信号融入其中得到策略收益率，最后通过累计计算和净值还原来对比策略与基准的最终表现。结果显示，双均线策略在示例数据上取得了更优的收益。需要注意的是，这只是一个简化的示例，实际策略评估还需考虑交易成本、滑点等因素。
