在本节课中，我们将要学习Python中两种最基础的数据类型：数字和字符串。理解数据类型是编程的基石，它决定了计算机如何处理我们提供的数据。

## 什么是数据类型？

人类可以轻易区分数字和文字。例如，`25`是数字，`“你好”`是文字。数字可以进行数学运算，文字可以组成句子。

但计算机本身无法自动区分。除非我们明确告知计算机某个值是数字还是字符串，否则它不知道如何处理。因此，编程语言引入了**数据类型**的概念，对生活中的各种数据（如数字、文字、列表）进行明确的分类和定义。

在Python中，主要的数据类型包括：整数(`int`)、浮点数(`float`)、字符串(`str`)、布尔值(`bool`)、列表(`list`)、元组(`tuple`)、字典(`dict`)、集合(`set`)。本节课我们先学习数字和字符串。

## 数字类型 🔢

数字类型主要分为整数和浮点数，它们统称为数字类型，可以进行数学运算。

### 整数 (`int`)

整数是不带小数部分的数字，在Python中的类型名称为 `int`。

**定义与类型检查**
定义整数时，直接书写数字，**不能加引号**。我们可以使用 `type()` 函数来检查一个值的类型。

```python
age = 25
print(type(age))  # 输出：<class 'int'>
```

**核心概念**
*   **定义整数**：直接赋值，不加引号。例如：`num = 100`
*   **类型检查**：使用 `type(变量名)` 函数。
*   **重要区别**：`25` 是整数，而 `"25"` 是字符串。

### 浮点数 (`float`)

浮点数即带有小数点的数字，也就是我们常说的小数。

```python
price = 9.99
print(type(price))  # 输出：<class 'float'>
height = 1.75
```
数字类型（包括正数、负数、整数、浮点数）都可以进行加(`+`)、减(`-`)、乘(`*`)、除(`/`)等数学运算。

上一节我们介绍了数字类型，它用于数值计算。接下来，我们看看用于处理文本信息的字符串类型。

## 字符串类型 📝

字符串是由零个或多个字符组成的有序序列，用于表示文本信息。在Python中，字符串的类型名称为 `str`。

### 字符串的定义

字符串需要使用引号括起来。Python支持三种引号方式：
1.  **单引号**：`‘这是一个字符串’`
2.  **双引号**：`“这也是一个字符串”`
3.  **三引号（三个单引号或双引号）**：用于定义多行字符串。

```python
name = "Alex"  # 双引号
hometown = ‘山东’  # 单引号
```
单引号和双引号在功能上没有区别，可以互换使用。选择哪种通常取决于字符串内容本身。

**何时使用双引号？**
当字符串内部包含单引号时，建议使用双引号包裹，以避免冲突。

```python
# 字符串内包含单引号，外部使用双引号
sentence = "I‘m Alex."
```

**何时使用三引号？**
三引号主要用于定义**多行字符串**。单引号和双引号定义的字符串必须写在一行内。

```python
# 使用三引号定义多行字符串
multi_line_msg = “““
我的名字是Alex。
我来自山东。
我喜欢编程。
”””
print(multi_line_msg)
```
在以上代码中，换行符 `\n` 会被保留，当使用 `print()` 函数输出时，会显示为真正的换行。

### 字符串的常用方法

字符串作为Python中的一个“类”（`class`），拥有许多内置的“方法”（函数），可以对字符串进行各种操作。你可以通过 `变量名.` 后按 `Tab` 键（在某些编辑器中）或查看帮助来了解这些方法。

以下是几个简单示例：

*   **`upper()`**: 将字符串中的所有字母转换为大写。
*   **`lower()`**: 将字符串中的所有字母转换为小写。
*   **`center(width, fillchar)`**: 返回一个指定宽度并居中的字符串，剩余部分用指定字符填充。

```python
msg = “hello world”
print(msg.upper())  # 输出：HELLO WORLD

name = “Alex”
print(name.center(20, ‘-‘))  # 输出：——–Alex——–
# 注意：宽度计算基于字节，可能与字符数直观感受不同，初步了解效果即可。
```

### 字符串的运算

与数字的数学运算不同，字符串有自己独特的运算方式。

**字符串拼接（`+`）**
使用加号 `+` 可以将多个字符串连接成一个新的字符串。

```python
first_name = “张”
last_name = “三”
full_name = first_name + last_name
print(full_name)  # 输出：张三
```

**字符串重复（`*`）**
使用乘号 `*` 可以将字符串重复指定的次数。

```python
tag = “重要！“
print(tag * 3)  # 输出：重要！重要！重要！
```

**重要限制**
字符串的 `+` 和 `*` 运算只能在**字符串之间**进行。尝试将字符串与数字直接相加会导致错误。

![](img/0f5381b50d2cc381cbc1436f9f6839cb_3.png)

```python
name = “Alex”
age = 22
# print(name + age)  # 这行代码会报错：TypeError: can only concatenate str (not “int“) to str
```
如果需要拼接字符串和数字，必须先将数字转换为字符串，我们将在后续课程中学习转换方法。

## 总结 🎯

本节课中我们一起学习了Python编程中最基础的两种数据类型：
1.  **数字类型**：包括整数(`int`)和浮点数(`float`)。定义时直接书写数字，用于数学计算。
2.  **字符串类型**(`str`)：用于表示文本，定义时需用引号括起。我们学习了单引号、双引号和三引号的不同用途，并实践了字符串的拼接(`+`)、重复(`*`)操作以及几个常用的方法（如`upper()`, `lower()`）。
