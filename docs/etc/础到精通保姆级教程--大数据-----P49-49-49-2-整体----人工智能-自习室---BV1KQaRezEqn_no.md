## 因子打分方法详解

上一节我们介绍了如何获取和预处理因子数据。本节中，我们来看看如何为这些因子进行打分，以便对股票进行综合评估。

首先，我们拥有每个股票的多个因子数据。例如，我们有四个因子：A、B、C、D。其中，因子A和B是“越大越好”的指标（如收益率），而因子C和D是“越小越好”的指标（如负债率、市值）。

以下是打分的基本步骤：

1.  **确定打分方向**：根据因子的性质（越大越好或越小越好），确定其得分与数值大小的关系。
2.  **划分数值区间**：将每个因子的数值范围划分为若干个区间。
3.  **分配分数**：为每个区间分配一个分数。对于“越大越好”的因子，数值大的区间得分高；对于“越小越好”的因子，数值小的区间得分高。
4.  **计算单因子得分**：根据每个股票在该因子上的具体数值，找到其所属区间，并获得对应的分数。
5.  **计算总分**：将一只股票在所有因子上的得分相加，得到其综合总分。
6.  **排序与选股**：对所有股票的总分进行排序，选择排名最高的股票作为投资组合。

### 打分示例

假设我们有两个股票（id1, id2）和四个因子（A, B, C, D）。它们的归一化数值如下：

| 股票ID | 因子A | 因子B | 因子C | 因子D |
| :----- | :---- | :---- | :---- | :---- |
| id1    | 0.71  | 0.28  | 0.39  | 0.11  |
| id2    | 0.45  | 0.17  | 0.81  | 0.02  |

**已知条件**：
*   因子A、B：**越大越好**。
*   因子C、D：**越小越好**。

**区间划分与分数分配**：
我们将数值范围（0-1）等分为5个区间，并为“越大越好”和“越小越好”两种类型分别制定打分规则。

```python
# “越大越好”型因子打分规则
# 区间: [1.0, 0.8) -> 5分, [0.8, 0.6) -> 4分, [0.6, 0.4) -> 3分, [0.4, 0.2) -> 2分, [0.2, 0.0] -> 1分

# “越小越好”型因子打分规则
# 区间: [0.0, 0.2) -> 5分, [0.2, 0.4) -> 4分, [0.4, 0.6) -> 3分, [0.6, 0.8) -> 2分, [0.8, 1.0] -> 1分
```

**计算过程**：
*   **id1**:
    *   因子A (0.71): 属于 `[0.6, 0.8)`， **越大越好** => **4分**
    *   因子B (0.28): 属于 `[0.2, 0.4)`， **越大越好** => **2分**
    *   因子C (0.39): 属于 `[0.2, 0.4)`， **越小越好** => **4分**
    *   因子D (0.11): 属于 `[0.0, 0.2)`， **越小越好** => **5分**
    *   **总分** = 4 + 2 + 4 + 5 = **15分**

*   **id2**:
    *   因子A (0.45): 属于 `[0.4, 0.6)`， **越大越好** => **3分**
    *   因子B (0.17): 属于 `[0.0, 0.2)`， **越大越好** => **1分**
    *   因子C (0.81): 属于 `[0.8, 1.0]`， **越小越好** => **1分**
    *   因子D (0.02): 属于 `[0.0, 0.2)`， **越小越好** => **5分**
    *   **总分** = 3 + 1 + 1 + 5 = **10分**

**排序结果**：id1 (15分) > id2 (10分)。因此，id1的排名更高。

> **提示**：打分方法有多种变体。例如，可以不划分区间，直接对300只股票的某个因子值从大到小排序，排名第1的给300分，排名第300的给1分。核心目标是得到一个可比较的综合总分。

## 整体策略流程梳理 🔄

理解了打分法后，我们来看一个完整的月度调仓策略流程。该流程将指导我们后续的代码实现。

![](img/65e827e4477c46f27335f40fbe1b0c34_11.png)

以下是策略执行的核心步骤：

1.  **确定股票池**：首先，需要确定选股的范围。在本例中，我们使用**沪深300指数**的成分股作为初始股票池。
2.  **设置调仓周期**：设定策略的调仓频率。通常，月度或季度调仓较为常见。这里我们使用**月度调仓**。
3.  **实现调仓函数**：这是策略的核心，我们将其命名为 `rebalance`。该函数在每个调仓日被调用，并执行以下操作：
    *   **数据获取与预处理**：获取当前股票池内所有股票的指定因子数据（如A、B、C、D），并进行必要的清洗与归一化。
    *   **因子打分**：应用上述打分方法，为每只股票的每个因子计算得分。
    *   **计算综合得分**：将每只股票在所有因子上的得分相加，得到其最终总分。
    *   **生成交易清单**：根据总分对所有股票进行降序排序，选取排名前10的股票作为本期拟买入的标的。
    *   **执行调仓**：卖出不在清单中的持仓，并买入清单中的股票，使投资组合与清单一致。

这个流程相对清晰简洁。接下来，我们将尝试用代码实现这个基于打分法的选股策略，并回测其历史表现，观察其收益效果。
