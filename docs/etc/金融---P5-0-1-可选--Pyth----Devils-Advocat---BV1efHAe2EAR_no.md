# 量化交易：Python入门之数据分析【1/4】：0-1 线上Jupyter环境配置 🚀

在本节课中，我们将学习如何快速配置一个无需本地安装的Python编程环境。这对于初学者或希望快速上手的同学来说，是一个非常便捷的入门方式。

## 概述

我们将使用一个名为Jupyter的在线环境。Jupyter是一个交互式笔记本，允许你编写代码、运行代码、查看结果，并添加文本说明，非常适合数据分析和学习。本节将引导你访问并使用这个线上环境。

## 环境配置步骤

以下是配置线上Jupyter环境的详细步骤。

![](img/ca2c1829a0282673c269d9cde7414801_1.png)

**第一步：访问在线Jupyter网站**

首先，打开你的浏览器。在地址栏中，你可以通过两种方式访问在线Jupyter环境。

![](img/ca2c1829a0282673c269d9cde7414801_3.png)

*   方法一：在搜索引擎中搜索关键词“在线版jupyter”。
*   方法二：直接在地址栏输入 `jupyter.org` 官网地址，然后找到“Try”相关的链接。

**第二步：进入Jupyter Lab界面**

通过上述任一方法，你将进入Jupyter的官方页面。在页面上找到并点击“Jupyter Lab”的入口。

**第三步：创建新的笔记本文件**

![](img/ca2c1829a0282673c269d9cde7414801_5.png)

进入Jupyter Lab后，你需要创建一个新的工作文件。点击页面上的“Start Files”或“Start a new notebook”按钮。有时，你可能需要再点击一个“加号”（+）图标来创建一个全新的笔记本。

![](img/ca2c1829a0282673c269d9cde7414801_6.png)

**第四步：开始编写和运行代码**

现在，你已经成功创建了一个Jupyter笔记本文件。在这个界面中，你可以输入Python代码并立即运行。例如，你可以尝试输入以下代码来测试环境：

```python
import numpy as np
A = np.array([1, 2, 3])
print(A)
```

输入代码后，按下 `Shift + Enter` 键即可运行该单元格的代码，并在下方看到输出结果。这省去了在个人电脑上安装Python和各种库的复杂步骤。

![](img/ca2c1829a0282673c269d9cde7414801_8.png)
