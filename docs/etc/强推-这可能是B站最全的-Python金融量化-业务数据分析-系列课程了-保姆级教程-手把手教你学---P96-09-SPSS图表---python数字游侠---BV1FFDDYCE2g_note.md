## 图表构建器

![](img/508f314deb8813916eafa4f9bab1c24f_3.png)

上一节我们介绍了SPSS的基本界面，本节中我们来看看如何使用图表构建器创建图形。

![](img/508f314deb8813916eafa4f9bab1c24f_5.png)

在SPSS中，构建图表主要有两种方式。第一种是使用图表构建器，操作方式直观。

![](img/508f314deb8813916eafa4f9bab1c24f_7.png)

以下是使用图表构建器创建柱状图的步骤：
1.  点击菜单栏的“图形”，选择“图表构建器”。
2.  在弹出的对话框中，从左侧的图表类型中选择“柱状图”，并将其拖拽到右侧的画布区域。
3.  画布上会显示图表的基本结构。此时，将数据集中想要分析的字段（例如“城市”）拖拽到画布上X轴的位置。
4.  点击“确定”，SPSS会自动生成一个显示每个城市人数计数的柱状图。

![](img/508f314deb8813916eafa4f9bab1c24f_9.png)

如果生成的图形效果不理想，可以更换分析的字段。例如，将“学历”字段拖拽到X轴，可以生成不同学历的人数分布柱状图，其视觉效果通常会更清晰。

## 图表编辑

创建图表后，我们可以对图表进行编辑以改善其外观。

![](img/508f314deb8813916eafa4f9bab1c24f_11.png)

双击生成的图表，会打开“图表编辑器”窗口。在编辑器中，可以选中图表的各个部分（如标题、坐标轴、柱条）进行修改。

以下是可进行的编辑操作示例：
*   **修改颜色**：选中柱状图的柱条，在右侧的属性面板中，可以更改其填充颜色与边框。
*   **调整样式**：可以修改图表的标题文字、大小、坐标轴标签等。
*   **更改图表类型**：虽然不如构建器直观，但也可以在编辑器中进行部分调整。

需要注意的是，SPSS的核心优势在于强大的统计分析功能，其图表功能相对基础，主要用于辅助展示分析结果。

## 旧对话框创建图表

除了图表构建器，SPSS还保留了传统的“旧对话框”方式来创建图表。

点击“图形”菜单下的“旧对话框”，可以看到各种图表类型，如条形图、饼图等。

以下是使用旧对话框创建饼图的简要过程：
1.  选择“饼图”，并选择“个案摘要”。
2.  在定义分区时，将分类变量（如“城市”）放入“定义分区依据”的框内。
3.  点击“确定”生成饼图。

这种方式步骤稍多，但逻辑与图表构建器类似，都是通过指定变量来定义图表内容。

## 基础统计分析功能简介

图表是数据展示的一部分，SPSS更强大的功能在于其统计分析模块。

在“分析”菜单下，有丰富的统计模型，如相关、回归、降维等。对于初学者，可以从基础的描述性统计开始。

例如，使用“分析” -> “描述统计” -> “探索”功能，可以对连续变量（如“年龄”）进行计算，得到均值、中位数、标准差、最小值、最大值等一系列描述性统计量。

![](img/508f314deb8813916eafa4f9bab1c24f_13.png)

另一个常用功能是交叉表（也称列联表），用于分析两个分类变量之间的关系。

![](img/508f314deb8813916eafa4f9bab1c24f_15.png)

以下是创建交叉表的步骤：
1.  点击“分析” -> “描述统计” -> “交叉表”。
2.  将一个分类变量（如“城市”）放入“行”框，将另一个分类变量（如“性别”）放入“列”框。
3.  点击“确定”，SPSS会生成一个表格，展示不同城市下不同性别人数的交叉分布情况。这在后续的分类数据分析（如卡方独立性检验）中会经常用到。

![](img/508f314deb8813916eafa4f9bab1c24f_17.png)

本节课中我们一起学习了在SPSS中创建图表的两种主要方法：直观的图表构建器和传统的旧对话框。我们了解到可以对生成的图表进行基本的编辑，但SPSS的图表功能相对简洁，其核心优势在于丰富的统计分析模块。最后，我们简要介绍了描述性统计和交叉表这两个基础且实用的分析功能，为后续深入学习数据分析打下基础。
