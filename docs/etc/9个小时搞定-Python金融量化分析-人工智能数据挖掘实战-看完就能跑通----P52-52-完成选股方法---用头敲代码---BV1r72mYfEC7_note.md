## 数据表拼接操作

上一节我们介绍了如何获取并处理单个因子数据。本节中，我们来看看如何将多个因子的数据表合并成一张大表，以便进行后续的综合分析。

![](img/801a6019bec000329bbefcced96d9754_4.png)

以下是拼接两个DataFrame的步骤：

![](img/801a6019bec000329bbefcced96d9754_6.png)

1.  使用第一个DataFrame的`.join()`方法。
2.  将第二个DataFrame作为参数传入该方法。

![](img/801a6019bec000329bbefcced96d9754_8.png)

```python
df_merged = df1.join(df2)
```

![](img/801a6019bec000329bbefcced96d9754_10.png)

现在，两个DataFrame已经拼接完成。我们为这个合并后的新表指定一个名字，例如 `df_rank`。此时，这张大表中包含了所有因子的原始数据，但还没有综合排名的信息。

![](img/801a6019bec000329bbefcced96d9754_12.png)

## 创建并计算综合得分列

![](img/801a6019bec000329bbefcced96d9754_14.png)

为了进行排名，我们需要创建一个新的列来存放每只股票的综合得分。

![](img/801a6019bec000329bbefcced96d9754_16.png)

首先，新建一个名为“得分值”的列，并暂时用零值填充。这里使用NumPy的`zeros`函数来创建一个指定维度的零值数组。

![](img/801a6019bec000329bbefcced96d9754_18.png)

```python
import numpy as np
df_rank[‘得分值’] = np.zeros((300, 1))
```

![](img/801a6019bec000329bbefcced96d9754_20.png)

上述代码创建了一个300行、1列的零值矩阵，并将其赋值给新列。但此时列中的数据全是零，我们需要计算真实的总分。

接下来，我们要计算总分并进行排序。现在每个指标的分值都已具备，我们需要将这些指标的分值合并计算。

具体操作是，对六个指标列进行横向求和。可以使用`.sum(axis=1)`方法来实现。

![](img/801a6019bec000329bbefcced96d9754_22.png)

```python
df_rank[‘得分值’] = df_rank[[‘指标1’, ‘指标2’, ‘指标3’, ‘指标4’, ‘指标5’, ‘指标6’]].sum(axis=1)
```

![](img/801a6019bec000329bbefcced96d9754_24.png)

计算完总和后，我们便得到了每只股票的综合得分 `score`。

![](img/801a6019bec000329bbefcced96d9754_26.png)

## 根据得分排序并获取结果

![](img/801a6019bec000329bbefcced96d9754_28.png)

得到综合得分后，下一步是根据得分对股票进行排序，并筛选出排名靠前的股票。

![](img/801a6019bec000329bbefcced96d9754_30.png)

使用 `.sort_values()` 方法进行排序。我们需要指定按“得分值”这一列排序，并且因为要选出分数最高的股票，所以排序顺序应为降序。

![](img/801a6019bec000329bbefcced96d9754_32.png)

```python
df_sorted = df_rank.sort_values(by=‘得分值’, ascending=False)
```

![](img/801a6019bec000329bbefcced96d9754_34.png)

排序完成后，我们通常只关心最终的股票列表。因此，可以从排序后的DataFrame中提取出我们需要的部分。

![](img/801a6019bec000329bbefcced96d9754_36.png)

我们只需要`score`值以及对应的股票代码。首先，取出“得分值”这一列，它是一个Series对象。

```python
score_series = df_sorted[‘得分值’]
```

![](img/801a6019bec000329bbefcced96d9754_38.png)

![](img/801a6019bec000329bbefcced96d9754_40.png)

我们的目标是选出前十名的股票，而不是得分。因此，需要获取这些高分股票对应的索引（即股票代码），并将其转换为列表。最后，使用切片取出前十个元素。

![](img/801a6019bec000329bbefcced96d9754_42.png)

```python
top_10_stocks = score_series.index.tolist()[:10]
```

![](img/801a6019bec000329bbefcced96d9754_44.png)

## 方法流程总结

![](img/801a6019bec000329bbefcced96d9754_46.png)

本节课中我们一起学习了 `get_stocks` 方法的核心构建流程。

![](img/801a6019bec000329bbefcced96d9754_48.png)

1.  **读取因子数据**：获取多个单因子数据表。
2.  **数据拼接**：将所有因子数据合并到一张大表中。
3.  **计算综合得分**：对多个因子值进行求和，得到每只股票的总分。
4.  **排序与筛选**：根据总分进行降序排序，并提取出排名前十的股票代码列表。

![](img/801a6019bec000329bbefcced96d9754_50.png)

至此，最关键的一个函数已经完成。执行一次 `get_stocks` 方法，它就会基于当前日期的所有指标数据，返回综合排名前十的股票列表，为后续的回测和交易决策打下基础。
