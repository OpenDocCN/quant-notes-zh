## 数据预处理与指标查询

![](img/f51c87a2811e98c52b03f76911f71f99_21.png)

在交易日开始前，我们通常需要进行数据预处理。核心任务是查询相关财务指标，并根据这些指标对股票池进行筛选。

以下是实现这一过程的关键步骤：

1.  **理解查询函数**：我们使用平台提供的 `get_fundamentals` 函数来查询财务数据。该函数的核心参数是一个 `query` 对象，用于指定查询的指标、过滤条件和排序规则。
2.  **构建查询语句**：查询语句（`query`）决定了我们要获取哪些数据。我们需要从帮助文档中查找所需的财务指标字段。
3.  **应用过滤条件**：通过 `filter` 方法，我们可以限定只查询属于我们股票池（沪深300）的股票。
4.  **进行排序**：使用 `order_by` 方法，我们可以根据选定的指标（如营业收入）对结果进行排序，例如降序排列以获取营收最高的股票。
5.  **限制结果数量**：最后，使用 `limit` 方法限制返回的股票数量，例如只选取排名前10的股票。

## 编写筛选代码

现在，让我们将上述步骤转化为具体的代码。我们将在 `before_trading_start` 函数中执行每日的筛选逻辑。

![](img/f51c87a2811e98c52b03f76911f71f99_23.png)

![](img/f51c87a2811e98c52b03f76911f71f99_25.png)

![](img/f51c87a2811e98c52b03f76911f71f99_27.png)

```python
def before_trading_start(context):
    # 构建财务数据查询
    q = query(
        fundamentals.financial_indicator.operating_revenue  # 查询营业收入指标
    ).filter(
        fundamentals.financial_indicator.code.in_(context.stock_pool)  # 过滤条件：股票代码在沪深300池中
    ).order_by(
        fundamentals.financial_indicator.operating_revenue.desc()  # 按营业收入降序排列
    ).limit(10)  # 只取前10名

    # 执行查询，将结果（DataFrame格式）保存到context中供后续使用
    context.filtered_stocks = get_fundamentals(q)

    # 打印查询结果以供检查
    print(context.filtered_stocks)
```

![](img/f51c87a2811e98c52b03f76911f71f99_29.png)

![](img/f51c87a2811e98c52b03f76911f71f99_31.png)

**代码要点说明**：
*   `fundamentals.financial_indicator.operating_revenue` 是查询营业收入的具体字段路径，这需要参考平台的API文档。
*   `.in_(context.stock_pool)` 确保了只筛选股票池内的股票。最初尝试时若忘记使用 `get_index_stocks` 获取成分股列表，会导致查询结果为空。
*   `.desc()` 表示降序排序，`.asc()` 表示升序排序。
*   查询结果是一个Pandas DataFrame，包含了股票代码和对应的营业收入数据。
