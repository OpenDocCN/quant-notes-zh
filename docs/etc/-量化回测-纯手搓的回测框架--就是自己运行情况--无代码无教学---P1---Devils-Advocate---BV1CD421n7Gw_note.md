## 什么是量化回测？📊

![](img/c6d15cd0406c4634e717f61b909ae558_13.png)

![](img/c6d15cd0406c4634e717f61b909ae558_14.png)

![](img/c6d15cd0406c4634e717f61b909ae558_15.png)

量化回测是一种使用历史数据来验证交易策略有效性的方法。通过模拟过去的市场环境，我们可以评估一个策略的潜在表现，而无需投入真实资金。

一个基本的回测框架通常包含以下几个核心部分：

![](img/c6d15cd0406c4634e717f61b909ae558_30.png)

![](img/c6d15cd0406c4634e717f61b909ae558_31.png)

![](img/c6d15cd0406c4634e717f61b909ae558_33.png)

1.  **数据模块**：负责加载和处理历史市场数据。
2.  **策略模块**：定义买入和卖出的逻辑规则。
3.  **执行模块**：模拟订单的执行，包括成交价格和手续费。
4.  **投资组合模块**：记录当前的现金、持仓和总资产。
5.  **分析模块**：计算并输出策略的各项绩效指标。

## 数据模块：历史数据的处理 📈

![](img/c6d15cd0406c4634e717f61b909ae558_44.png)

![](img/c6d15cd0406c4634e717f61b909ae558_46.png)

数据是回测的基石。我们需要获取并整理历史价格数据，例如每日的开盘价、最高价、最低价和收盘价。

![](img/c6d15cd0406c4634e717f61b909ae558_50.png)

![](img/c6d15cd0406c4634e717f61b909ae558_51.png)

![](img/c6d15cd0406c4634e717f61b909ae558_52.png)

![](img/c6d15cd0406c4634e717f61b909ae558_53.png)

以下是处理数据时常见的步骤：

![](img/c6d15cd0406c4634e717f61b909ae558_54.png)

![](img/c6d15cd0406c4634e717f61b909ae558_55.png)

![](img/c6d15cd0406c4634e717f61b909ae558_56.png)

![](img/c6d15cd0406c4634e717f61b909ae558_57.png)

*   **数据获取**：从文件（如CSV）或在线API加载数据。
*   **数据清洗**：处理缺失值或异常值。
*   **数据转换**：计算技术指标，例如移动平均线。其公式为：
    **`SMA = (P1 + P2 + ... + Pn) / n`**
    其中 `P` 代表价格，`n` 是周期数。

策略模块是回测的大脑，它根据数据和预设规则生成交易信号。

![](img/c6d15cd0406c4634e717f61b909ae558_66.png)

一个简单的策略示例是“双均线交叉策略”：当短期均线上穿长期均线时买入，下穿时卖出。

![](img/c6d15cd0406c4634e717f61b909ae558_70.png)

![](img/c6d15cd0406c4634e717f61b909ae558_71.png)

![](img/c6d15cd0406c4634e717f61b909ae558_72.png)

![](img/c6d15cd0406c4634e717f61b909ae558_73.png)

在代码中，策略逻辑可能这样表示：
```python
if short_ma > long_ma and not holding_position:
    signal = “BUY”
elif short_ma < long_ma and holding_position:
    signal = “SELL”
else:
    signal = “HOLD”
```

## 投资组合与执行模块：模拟交易过程 💰

投资组合模块跟踪我们的资金状态。核心是以下公式：
**`总资产 = 现金 + ∑(持仓数量 × 当前价格)`**

![](img/c6d15cd0406c4634e717f61b909ae558_82.png)

![](img/c6d15cd0406c4634e717f61b909ae558_83.png)

![](img/c6d15cd0406c4634e717f61b909ae558_84.png)

![](img/c6d15cd0406c4634e717f61b909ae558_85.png)

![](img/c6d15cd0406c4634e717f61b909ae558_86.png)

执行模块则负责模拟订单的成交。它需要考虑：
*   **成交价格**：通常使用下一根K线的开盘价模拟买入。
*   **交易成本**：包括佣金和滑点（预期价格与实际成交价的差异）。

![](img/c6d15cd0406c4634e717f61b909ae558_87.png)

![](img/c6d15cd0406c4634e717f61b909ae558_107.png)

![](img/c6d15cd0406c4634e717f61b909ae558_108.png)

以下是典型的回测主循环步骤：

![](img/c6d15cd0406c4634e717f61b909ae558_111.png)

![](img/c6d15cd0406c4634e717f61b909ae558_112.png)

![](img/c6d15cd0406c4634e717f61b909ae558_113.png)

![](img/c6d15cd0406c4634e717f61b909ae558_114.png)

1.  初始化：设置起始资金、加载数据。
2.  循环遍历每一个历史交易日：
    *   更新数据指标（如计算均线）。
    *   运行策略，获取交易信号。
    *   执行模块处理信号，更新投资组合。
3.  回测结束后，进入分析阶段。

![](img/c6d15cd0406c4634e717f61b909ae558_115.png)

![](img/c6d15cd0406c4634e717f61b909ae558_116.png)

![](img/c6d15cd0406c4634e717f61b909ae558_117.png)

![](img/c6d15cd0406c4634e717f61b909ae558_118.png)

分析模块会计算关键绩效指标，例如：
*   **总收益率**：`(最终资产 - 初始资产) / 初始资产`
*   **年化收益率**
*   **最大回撤**：资产净值从峰值到谷底的最大跌幅。
*   **夏普比率**：衡量风险调整后的收益。
