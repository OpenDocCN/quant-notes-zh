在本节课中，我们将要学习一种名为“3-Sigma”的去极值方法。该方法基于统计学中的正态分布原理，通过计算数据的均值和标准差来识别并处理异常值。我们将从原理入手，理解其背后的逻辑，然后通过代码实现，并最终将其应用于数据预处理流程中。

## 原理介绍

上一节我们介绍了去极值的概念，本节中我们来看看基于统计学原理的3-Sigma方法。

其核心思想基于正态分布（也称为高斯分布）。正态分布是一种常见的概率分布，其形状呈钟形曲线。在金融数据等许多场景中，我们可以假设数据经过变换后近似服从正态分布。

![](img/844ccdba27597ddecaa004b1f87d2231_1.png)

在正态分布中，数据大多集中在均值附近。离均值越远的数据点，出现的概率就越低。极值，即异常值，通常就落在这些远离均值的低概率区域。

3-Sigma方法正是利用了这一特性。这里的“Sigma”（σ）指的是标准差，它是衡量数据离散程度的一个指标。我们通过计算**均值（μ）**和**标准差（σ）**，来定义一个数据的“正常”范围。

以下是其核心公式定义的上限和下限：

**上限 = μ + N * σ**
**下限 = μ - N * σ**

![](img/844ccdba27597ddecaa004b1f87d2231_3.png)

其中，N通常取3。根据统计学经验，在正态分布中，数据落在均值正负3倍标准差区间内的概率约为99.7%。这意味着，我们将落在这个区间外的0.3%的数据视为极值进行处理。

![](img/844ccdba27597ddecaa004b1f87d2231_5.png)

## 代码实现

![](img/844ccdba27597ddecaa004b1f87d2231_7.png)

理解了原理之后，我们来看看如何用代码实现3-Sigma方法。

![](img/844ccdba27597ddecaa004b1f87d2231_9.png)

首先，我们需要计算指定数据列的均值和标准差。然后，根据给定的倍数N，计算出上限和下限。最后，将所有超出此范围的数据规范到边界上。

![](img/844ccdba27597ddecaa004b1f87d2231_11.png)

以下是该函数的核心代码实现：

![](img/844ccdba27597ddecaa004b1f87d2231_13.png)

![](img/844ccdba27597ddecaa004b1f87d2231_15.png)

```python
def sigma_method(data_series, n=3):
    """
    使用3-Sigma方法处理极值。
    参数:
        data_series: 待处理的数据序列（pandas Series）。
        n: Sigma的倍数，默认为3。
    返回:
        series: 处理后的数据序列。
        lower_limit: 计算出的下限。
        upper_limit: 计算出的上限。
    """
    # 计算均值和标准差
    mean = data_series.mean()
    std = data_series.std()
    
    # 根据公式计算上下限
    upper_limit = mean + n * std
    lower_limit = mean - n * std
    
    # 将超出范围的数据规范到边界上
    processed_series = data_series.clip(lower=lower_limit, upper=upper_limit)
    
    return processed_series, lower_limit, upper_limit
```

我们来回顾一下关键点：
*   **原理基础**：该方法基于数据服从正态分布的假设，利用**均值（μ）**和**标准差（σ）**来定义正常数据的范围。
*   **核心公式**：通过公式 **μ ± N * σ** 计算上下限，通常N=3以涵盖99.7%的数据。
*   **操作实质**：去极值并非简单删除异常点，而是将其规范到计算出的**下限**和**上限**之内，使所有数据都落在预设的合理区间中。
*   **代码实现**：过程简洁，主要步骤包括计算均值与标准差、根据公式确定边界、并使用`clip`函数进行数据规范。

![](img/844ccdba27597ddecaa004b1f87d2231_41.png)

这是数据预处理“三步走”中第一步“去极值”的又一种有效工具。掌握它，能帮助我们在量化分析中更好地处理噪声数据，为后续建模打下坚实基础。
