## 预测效果评估

在构建模型之前，我们需要了解如何评估一个分类预测模型的好坏。以下是两个最基础的评估指标。

![](img/e1f054176307dafc9592c44d5a5ccb64_6.png)

### 命中率

![](img/e1f054176307dafc9592c44d5a5ccb64_8.png)

![](img/e1f054176307dafc9592c44d5a5ccb64_10.png)

命中率是最常见的评估统计量，用于衡量预测的准确率。其计算公式为：

![](img/e1f054176307dafc9592c44d5a5ccb64_12.png)

**命中率 = (预测正确的样本数) / (总样本数)**

命中率越接近1，说明模型的训练效果越好。

![](img/e1f054176307dafc9592c44d5a5ccb64_14.png)

### 混淆矩阵

![](img/e1f054176307dafc9592c44d5a5ccb64_16.png)

混淆矩阵是一个2x2的矩阵，可以更细致地展示预测结果。假设我们将“上涨”标记为1，“下跌”标记为0，那么混淆矩阵的结构如下：

| 真实\预测 | 预测为1（涨） | 预测为0（跌） |
| :--- | :--- | :--- |
| **真实为1（涨）** | UT (正确预测上涨) | UF (错误预测下跌) |
| **真实为0（跌）** | DF (错误预测上涨) | DT (正确预测下跌) |

*   **UT** 和 **DT** 位于主对角线上，代表预测正确的样本数，我们希望它们越大越好。
*   **UF** 和 **DF** 代表预测错误的样本数，我们希望它们越小越好。

其中，**UF** 被称为**第一类错误**（假阳性），例如将一个信用良好、不会违约的客户预测为会违约。**DF** 被称为**第二类错误**（假阴性），例如将一个信用差、会违约的客户预测为不会违约。在风控等场景中，第二类错误通常带来的损失更大，因此我们更倾向于控制第二类错误。

## 常用分类算法简介

上一节我们介绍了如何评估模型，本节中我们来看看几种用于预测的机器学习算法。关于这些算法的理论知识，我们将在后续的机器学习章节深入探讨。

![](img/e1f054176307dafc9592c44d5a5ccb64_18.png)

![](img/e1f054176307dafc9592c44d5a5ccb64_20.png)

以下是几种基础的分类算法及其核心思想：

![](img/e1f054176307dafc9592c44d5a5ccb64_22.png)

1.  **逻辑回归**
    *   **核心思想**：假设在给定特征`X`的条件下，分类`Y`属于某一类的概率服从一个逻辑函数（Logistic函数）。通过最大化所有观测值的联合条件概率（即最大似然估计）来求解模型参数。
    *   **公式**：`P(Y=1|X) = 1 / (1 + exp(-(w*X + b)))`，其中`w`是权重，`b`是偏置项。

2.  **判别分析**
    *   **核心思想**：基于贝叶斯定理。首先假设特征`X`对于每个类别`Y`都服从一个特定的分布（通常是多元高斯分布），并且每个类别有一个先验概率。通过最大似然估计求出参数后，对于新的样本，计算其属于每个类别的后验概率，概率最高的类别即为预测结果。
    *   **LDA与QDA**：线性判别分析假设所有类别共享**相同的协方差矩阵**，这使得决策边界是线性的。二次判别分析则允许每个类别有自己的协方差矩阵，决策边界是二次的，因此更为灵活。

3.  **支持向量机**
    *   **核心思想**：寻找一个最优的分类超平面，使得两个类别的样本点到该平面的**间隔**最大化。对于线性不可分的情况，采用“软间隔”分类器，允许一些样本点位于间隔内部或错误的一侧，这些点被称为“支持向量”。
    *   **核函数**：为了处理非线性问题，SVM引入了核函数，将数据映射到高维空间使其线性可分。常用核函数包括：
        *   多项式核
        *   高斯核（RBF核）
        *   Sigmoid核

![](img/e1f054176307dafc9592c44d5a5ccb64_24.png)

![](img/e1f054176307dafc9592c44d5a5ccb64_26.png)

4.  **决策树与随机森林**
    *   **决策树**：一种树形结构的分类器，非常直观。它从根节点开始，根据某个特征将数据划分为子集，递归地进行下去，直到满足停止条件（如纯度足够高）。划分特征的选择通常基于信息增益或基尼不纯度等准则。
    *   **随机森林**：为了克服单棵决策树容易过拟合和受数据扰动影响大的缺点，随机森林采用了**Bagging**思想。它构建多棵决策树，每棵树在训练时不仅对样本进行随机抽样，还对特征进行随机选择。最终的预测结果是所有树预测结果的综合（如投票或平均）。这种方法特别适用于特征相关性高、数据噪音大的场景。

![](img/e1f054176307dafc9592c44d5a5ccb64_28.png)

除了上述方法，还有如梯度提升树、神经网络等更高级的算法。自然语言处理也是近期火热的方向，它可以从文本数据（如新闻、社交媒体评论）中提取情绪特征，用于预测市场走势，尽管有时发现的关联看似无厘头，但数据驱动的方法可能揭示出潜在的相关性。

### 关键参数设置简介

![](img/e1f054176307dafc9592c44d5a5ccb64_56.png)

![](img/e1f054176307dafc9592c44d5a5ccb64_58.png)

![](img/e1f054176307dafc9592c44d5a5ccb64_60.png)

在调用这些模型时，了解一些关键参数有助于调整模型：

![](img/e1f054176307dafc9592c44d5a5ccb64_62.png)

*   **逻辑回归**：
    *   `penalty`：正则化类型，常用`l2`。
    *   `C`：正则化强度的倒数，C值越小，正则化越强。
*   **判别分析**：
    *   `solver`：求解算法，如`svd`（奇异值分解）。
*   **支持向量机**：
    *   `C`：惩罚系数，控制对错误分类的容忍度。C越大，越不能容忍错误，模型可能过拟合。
    *   `kernel`：核函数类型，如`linear`（线性），`rbf`（高斯核）。
    *   `gamma`：高斯核的参数，影响单个样本的影响范围。
