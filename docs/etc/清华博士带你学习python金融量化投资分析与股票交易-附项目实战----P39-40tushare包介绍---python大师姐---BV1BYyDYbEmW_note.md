## 核心接口：获取历史行情数据

Tushare提供了多种数据接口，其中最常用的是获取历史行情数据的接口。

以下是`get_k_data`接口的核心参数说明：

![](img/8aa06c30206a4be217305b104adb135f_29.png)

*   **`code`**：股票代码，用于指定获取哪只股票的数据。例如：`‘601318’`。
*   **`ktype`**：K线类型。默认为日K线（`‘D’`），还可选择周K线（`‘W’`）、月K线（`‘M’`）以及分钟线（如`‘5’`代表5分钟线）。
*   **`autype`**：复权类型。默认为前复权（`‘qfq’`）。复权处理了股票分红、送股等行为对股价的影响，初学者使用默认值即可。
*   **`index`**：是否为指数。设置为`True`时，`code`参数应传入指数代码（如`‘000016’`代表上证50）。
*   **`start`**：数据开始日期，格式为`‘YYYY-MM-DD’`。
*   **`end`**：数据结束日期，格式为`‘YYYY-MM-DD’`。不设置则默认为当前日期。

![](img/8aa06c30206a4be217305b104adb135f_31.png)

![](img/8aa06c30206a4be217305b104adb135f_33.png)

调用该接口会返回一个Pandas的DataFrame对象，包含日期（`date`）、开盘价（`open`）、收盘价（`close`）、最高价（`high`）、最低价（`low`）、成交量（`volume`）等列。

**代码示例：获取中国平安（601318）的历史数据并保存**
```python
import tushare as ts

![](img/8aa06c30206a4be217305b104adb135f_35.png)

# 获取股票‘601318’从最早到现在的日K线数据
df = ts.get_k_data(‘601318’, start=‘1988-01-01’)
# 将数据保存到CSV文件
df.to_csv(‘601318.csv’, index=False)
```
运行上述代码，即可将中国平安的历史行情数据下载并保存到本地文件`601318.csv`中。

## 其他常用数据接口

除了历史行情，Tushare还提供了其他多种数据接口，以满足不同的分析需求。

![](img/8aa06c30206a4be217305b104adb135f_41.png)

![](img/8aa06c30206a4be217305b104adb135f_43.png)

以下是部分其他常用接口的简介：

![](img/8aa06c30206a4be217305b104adb135f_45.png)

*   **实时行情**：`get_today_all()`，用于获取所有股票今日的行情概览。
*   **实时分笔数据**：`get_today_ticks()`，用于获取某只股票当日的详细分笔成交数据。
*   **历史分笔数据**：`get_tick_data()`，用于获取某只股票历史某日的分笔成交数据。
*   **大盘指数**：`get_index()`，用于获取主要大盘指数的实时行情数据。

![](img/8aa06c30206a4be217305b104adb135f_47.png)

每个接口都有详细的说明文档和调用示例，大家可以根据实际需求查阅Tushare的官方文档进行调用。对于本课程后续的简单分析示例，使用`get_k_data`接口获取的历史数据已经足够。
