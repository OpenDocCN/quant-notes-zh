# 量化交易入门：03：股票市场构成 🏛️

在本节课中，我们将要学习股票市场的基本构成。了解市场中有哪些参与者以及他们各自扮演的角色，是进行金融分析和量化交易的重要基础。

## 市场参与者

上一节我们介绍了股票的分类，本节中我们来看看股票市场的构成，即市场中有哪些参与者。

首先是公司和投资者。公司是需要融资的一方，投资者是提供资金的一方。公司通过上市向投资者融资。

但是，股票交易不能让公司和投资者直接进行交易，这是为了防止暗箱操作。

以下是市场中的主要监管与服务机构：

*   **证监会**：这是证券行业的监管机构。公司想要上市，需要向证监会提交各种材料。证监会负责判断公司是否有欺诈、洗钱等违法行为，并决定公司能否上市或是否应退市。
*   **证券业协会**：这是一个行业自律组织，作用相对较弱。例如，证券从业资格考试由其主办。
*   **交易所**：交易所提供股票交易的场所。在中国，主要有上海和深圳两家交易所。早期交易者需要亲自去交易所排队交易，现在则通过网络连接到交易所的系统。交易所的功能是处理各种买卖请求。
*   **证券中介机构（券商）**：个人投资者不能直接去交易所买卖股票。投资者需要在券商（如中信证券、中金公司等）开户。券商在交易所有交易席位，投资者通过券商开发的软件（如同花顺）提交交易指令，券商再通过其席位将指令传达给交易所，完成股票买卖。

## 中国的交易所与板块

接下来我们了解一下中国的交易所及其板块划分。

中国有两个主要的证券交易所：上海证券交易所和深圳证券交易所。

每个交易所内部又划分为不同的板块：

*   **上海证券交易所**：主要有一个主板。
*   **深圳证券交易所**：分为三个板块：主板、中小板和创业板。

中小板和创业板是国家为鼓励小规模或初创公司发展而设立的。这些公司可能体量不够大，但成长性好，可以在这些板块上市融资。主板上市要求严格（如对净利润有高要求），而创业板等要求相对较低。

## 大盘指数

对于每一个板块，我们通常用“大盘”来指代其整体表现。大盘其实就是一个指数。

指数反映了该板块内所有股票的综合表现。例如，上海主板的大盘叫做**沪指**（上证指数），深圳主板的叫做**深成指**，中小板的叫做**中小板指**，创业板的叫做**创业板指**。

简单来说，指数就像是一个平均值或趋势图。它概括了整个市场盘子（所有股票的总和）的走势，用于判断市场整体是向好还是向坏，而不必逐一分析成千上万只个股。
