## 概述

![](img/bf2d5e5385bfb308644e1a96142f0268_10.png)

## 凸函数的一阶条件回顾
上一节我们介绍了凸函数的几种定义。本节中，我们来看看当凸函数一阶可微时，它所满足的一个等价条件。

如果一个凸函数 `f` 在其凸定义域 `D` 上一阶可微，那么对于定义域内任意两点 `x` 和 `y`，以下不等式恒成立：

![](img/bf2d5e5385bfb308644e1a96142f0268_12.png)

![](img/bf2d5e5385bfb308644e1a96142f0268_14.png)

**公式：** `f(y) ≥ f(x) + ∇f(x)^T (y - x)`

![](img/bf2d5e5385bfb308644e1a96142f0268_16.png)

![](img/bf2d5e5385bfb308644e1a96142f0268_18.png)

![](img/bf2d5e5385bfb308644e1a96142f0268_20.png)

这个不等式的几何意义是：函数图像在其任意一点 `x` 处的切线（或超平面）总是位于函数图像的下方。这可以形象地理解为，在一个凸的“锅底”上放一根“棍子”（切线），这根棍子永远不会高于锅的表面。

![](img/bf2d5e5385bfb308644e1a96142f0268_22.png)

![](img/bf2d5e5385bfb308644e1a96142f0268_24.png)

## 拟凸函数的一阶条件
与凸函数类似，可微的拟凸函数也有其特定的一阶条件，但形式有所不同。

![](img/bf2d5e5385bfb308644e1a96142f0268_26.png)

对于一个定义在凸集上的可微拟凸函数 `f`，其等价的一阶条件是一个逻辑关系：对于定义域内任意两点 `x` 和 `y`，如果 `f(y) ≤ f(x)`，则必有：

![](img/bf2d5e5385bfb308644e1a96142f0268_28.png)

![](img/bf2d5e5385bfb308644e1a96142f0268_30.png)

**公式：** `∇f(x)^T (y - x) ≤ 0`

反之，如果对于任意 `x` 和 `y`，上述逻辑关系都成立，那么函数 `f` 就是拟凸的。这是一个独特的定义，因为它通过一个逻辑蕴含关系（而非简单的不等式）来刻画函数性质。

为了帮助理解，我们在一维情况下简要证明这个逻辑关系的双向性。

![](img/bf2d5e5385bfb308644e1a96142f0268_32.png)

**证明思路（正向，拟凸 => 逻辑关系）：**
1.  根据拟凸函数定义，对于任意 `θ ∈ [0, 1]`，有 `f(θx + (1-θ)y) ≤ max{f(x), f(y)}`。
2.  假设 `f(y) ≤ f(x)`，则上述不等式变为 `f(θx + (1-θ)y) ≤ f(x)`。
3.  通过代数变形，并令 `θ` 趋近于1（考虑极限），可以从上述不等式中推导出 `∇f(x)^T (y - x) ≤ 0`。

![](img/bf2d5e5385bfb308644e1a96142f0268_34.png)

**证明思路（反向，逻辑关系 => 拟凸）：**
1.  已知对于任意 `x, y`，若 `f(y) ≤ f(x)`，则有 `∇f(x)^T (y - x) ≤ 0`。
2.  要证明 `f` 拟凸，即证明对于任意 `θ ∈ [0, 1]`，有 `f(θx + (1-θ)y) ≤ max{f(x), f(y)}`。
3.  不妨设 `f(x) ≥ f(y)`。通过构造并利用已知的逻辑关系（此时 `x` 和 `θx + (1-θ)y` 分别扮演原关系中的 `x` 和 `y`），可以证明该不等式成立。

![](img/bf2d5e5385bfb308644e1a96142f0268_36.png)

![](img/bf2d5e5385bfb308644e1a96142f0268_38.png)

## 一阶条件的意义对比
理解了凸函数和拟凸函数的一阶条件后，我们来看看它们在实际优化问题中意义的重大区别。

![](img/bf2d5e5385bfb308644e1a96142f0268_40.png)

对于凸函数，如果在其定义域内找到一点 `x*`，使得梯度为零（`∇f(x*) = 0`），那么这一点就是全局极小点。因为将 `∇f(x*) = 0` 代入凸函数的一阶条件，会得到对于所有 `y`，`f(y) ≥ f(x*)`。

然而，对于拟凸函数，梯度为零的点不再具有这种明确的优化意义。将 `∇f(x*) = 0` 代入拟凸函数的一阶逻辑关系，前提 `f(y) ≤ f(x*)` 成立时，结论 `0 ≤ 0` 是一个永真但无信息的陈述。这意味着，拟凸函数的驻点（梯度为零的点）可能是极小点、极大点或鞍点，需要进一步判断。

**示例：**
考虑函数 `f(x) = x^3`。这是一个拟凸函数（其下水平集是凸的）。在 `x=0` 处，梯度 `f'(0)=0`，但这一点显然不是全局极小点。

## 凸函数与拟凸函数的二阶条件
上一节我们对比了一阶条件的意义，本节中我们来看看它们的二阶条件有何不同。

![](img/bf2d5e5385bfb308644e1a96142f0268_42.png)

对于二阶可微的凸函数 `f`，其等价条件是：在其定义域内每一点 `x` 处的海森矩阵（Hessian matrix）`∇²f(x)` 都是半正定的。

**公式（凸函数）：** `∇²f(x) ⪰ 0`，对于所有 `x ∈ D`。

对于二阶可微的拟凸函数 `f`，其二阶条件同样是一个逻辑关系：对于定义域内任意 `x` 和 `y`，如果满足 `∇f(x)^T (y - x) = 0`，那么必有：

![](img/bf2d5e5385bfb308644e1a96142f0268_44.png)

**公式（拟凸函数）：** `(y - x)^T ∇²f(x) (y - x) ≥ 0`

![](img/bf2d5e5385bfb308644e1a96142f0268_46.png)

这个条件可以理解为：**仅在梯度方向与位移 `(y-x)` 正交的那些“临界”方向上，海森矩阵才需要是半正定的**。这比凸函数要求海森矩阵处处半正定要弱得多。

![](img/bf2d5e5385bfb308644e1a96142f0268_48.png)

**一维情况下的简化理解：**
在一维情况下，拟凸函数的二阶条件变为：若 `f'(x) * (y - x) = 0`，则 `f''(x) * (y - x)^2 ≥ 0`。
*   如果 `f'(x) = 0`，那么对于任意非零的 `(y-x)`，可推出 `f''(x) ≥ 0`。这意味着在驻点处，二阶导数必须非负。
*   这对比凸函数的条件 `f''(x) ≥ 0` 对于所有 `x` 成立，再次说明拟凸函数的条件更弱。

**示例对比：**
*   `f(x) = x^2`：既是凸函数也是拟凸函数。在 `x=0`（驻点）处，`f''(0)=2 > 0`，满足两者条件。
*   `f(x) = x^3`：是拟凸函数，但不是凸函数。在 `x=0`（驻点）处，`f''(0)=0 ≥ 0`，满足拟凸函数的二阶条件，但不满足凸函数处处二阶导非负的条件（例如在 `x<0` 时，`f''(x) < 0`）。

## 对数凸与对数凹函数
除了拟凸/拟凹，函数还有一类重要的扩展形式：对数凸函数和对数凹函数。

**定义：**
*   **对数凸函数**：对于一个正值函数 `f(x) > 0`，如果 `log f(x)` 是凸函数，则称 `f(x)` 是对数凸的。
*   **对数凹函数**：对于一个正值函数 `f(x) > 0`，如果 `log f(x)` 是凹函数，则称 `f(x)` 是对数凹的。

**引入意义：**
在实际问题（如概率建模、效用函数）中，经常遇到函数的连乘形式。取对数可以将连乘转化为连加，简化运算。此时，判断原函数的对数凸/凹性就变得非常重要。

![](img/bf2d5e5385bfb308644e1a96142f0268_50.png)

**与凸/凹函数的关系：**
以下是几种函数类型之间的强弱关系：
1.  **对数凸 => 凸**：如果一个函数是对数凸的，那么它一定是凸的。因为 `f(x) = exp(log f(x))`，而指数函数 `exp(·)` 是凸且单调递增的，根据凸函数的保凸运算规则，凸函数经过单调递增的凸函数复合后仍是凸函数。
2.  **凹 => 对数凹**：如果一个正值函数是凹的，那么它一定是对数凹的。因为对数函数 `log(·)` 是凹且单调递增的，凹函数经过单调递增的凹函数复合后仍是凹函数（即 `log f(x)` 是凹的）。
3.  反之则不必然成立。凸函数不一定是对数凸的（如 `f(x)=x`），对数凹函数也不一定是凹的。

## 总结
本节课我们一起学习了可微拟凸函数的一阶和二阶条件，并与凸函数的相应条件进行了深入对比。
*   **一阶条件**：拟凸函数由逻辑关系 `f(y) ≤ f(x) ⇒ ∇f(x)^T (y - x) ≤ 0` 刻画，其梯度为零的点不一定是极值点，这与凸函数有本质区别。
*   **二阶条件**：拟凸函数仅要求在梯度与搜索方向正交的临界方向上，海森矩阵半正定，这比凸函数要求处处半正定要宽松。
*   **函数扩展**：我们还介绍了对数凸/凹函数，它们在实际问题中处理连乘形式时非常有用，并且与凸/凹函数存在明确的强弱关系。

理解这些条件的差异，对于后续判断优化问题的性质以及选择求解算法至关重要。下一章，我们将把凸集和凸函数的概念结合起来，正式定义“凸优化问题”，并探讨其优良性质。
