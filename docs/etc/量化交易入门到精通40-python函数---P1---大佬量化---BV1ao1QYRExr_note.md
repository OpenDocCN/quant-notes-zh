在本节课中，我们将要学习Python编程中一个极其重要的概念——函数。函数就像炒菜的菜谱，将一系列操作步骤封装起来，以后需要时直接调用即可，这能让代码更简洁、高效且易于维护。

## 认识函数

上一节我们介绍了控制流，本节中我们来看看如何将代码块组织成可复用的单元——函数。

Python使用 `def` 关键字来定义函数，其基本结构如下：
```python
def 函数名(参数):
    # 函数体
```

### 定义无参数函数

以下是一个定义无参数函数的例子：
```python
def function1():
    print("Hello World")
```
定义函数后，需要通过函数名加括号 `()` 来调用它，才能执行其中的代码。
```python
function1()  # 输出：Hello World
```

### 定义有参数函数

函数可以接收参数，使功能更加灵活。以下是定义有参数函数的例子：
```python
def function2(name):
    print(f"Hello {name}")
```
调用时，需要传入相应的值。
```python
function2("Python")  # 输出：Hello Python
```
参数在函数定义时称为“形参”（形式参数），在调用时传入的具体值称为“实参”（实际参数）。形参的名称（如`name`）与调用时使用的变量名无关。

## 参数传递：可变与不可变对象

在传递参数时，需要理解Python中对象的可变性。这就像炒菜时，有些配料（如盐）加了就是新的，不影响原来的；而有些（如汤底）改变了就会影响整锅汤。

### 不可变对象

数字、字符串、元组等属于不可变对象。在函数内部修改它们时，实际上是创建了一个新对象，不会影响函数外部的原始值。
```python
def change_value(a):
    print(f"函数内修改前a的id: {id(a)}")
    a = 10
    print(f"函数内修改后a的id: {id(a)}")

x = 1
print(f"调用前x的id: {id(x)}")
change_value(x)
print(f"调用后x的值: {x}")  # x的值仍然是1，未被改变
```

### 可变对象

列表、字典等属于可变对象。在函数内部修改它们的内容，会影响函数外部的原始对象。
```python
def change_list(my_list):
    my_list.append(40)
    print(f"函数内my_list的值: {my_list}")

my_list = [10, 20, 30]
print(f"调用前my_list的值: {my_list}")
change_list(my_list)
print(f"调用后my_list的值: {my_list}")  # my_list变成了[10, 20, 30, 40]
```

## 函数的参数类型

函数参数有多种类型，就像做菜可以根据需要选择不同的调料。

### 必需参数

必需参数就像炒菜必不可少的油和盐，必须按顺序、按数量传递，否则程序会报错。
```python
def function4(a, b):  # a和b是必需参数
    print(f"a={a}, b={b}")

function4(1, 2)  # 正确
# function4(1)     # 错误，缺少一个必需参数
```

### 关键字参数

使用关键字参数调用函数时，可以用参数名来指定值，这样就不必拘泥于参数定义的顺序。
```python
def function5(a, b):
    print(f"a={a}, b={b}")

function5(b=3, a=4)  # 输出：a=4, b=3
```

### 默认参数

定义函数时，可以为参数指定默认值。调用时若不传递该参数，则使用默认值。
```python
def function6(a, b=2):  # b的默认值为2
    print(f"a={a}, b={b}")

function6(10)       # 输出：a=10, b=2
function6(10, 20)   # 输出：a=10, b=20
```

### 不定长参数

不定长参数允许函数接受任意数量的参数，使用 `*args` 表示，这些参数会被封装成一个元组。
```python
def function7(*args):
    print(f"传入的参数是: {args}")
    print(f"参数类型是: {type(args)}")
    # 遍历元组中的所有参数
    for arg in args:
        print(arg)

function7(1, 2, 3, 4, 5)
```
