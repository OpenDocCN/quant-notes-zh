## 因子打分法详解

上一节我们介绍了如何获取和处理股票数据。本节中，我们来看看如何利用这些数据对股票进行综合评价，即**因子打分法**。

### 打分法的基本思想
打分法的核心思想是为每只股票的每个指标（因子）赋予一个分数，然后将所有指标的分数汇总，得到该股票的总分。最后，根据总分对所有股票进行排名，选择排名靠前的股票作为投资组合。

### 数据准备与因子方向
首先，我们需要明确每个指标的好坏方向。例如，有些指标是“越大越好”（如收益率），有些则是“越小越好”（如负债率）。

以下是示例数据，假设我们有4个指标（A, B, C, D）和300只股票（id1 到 id300）：

| 股票ID | 因子A | 因子B | 因子C | 因子D |
| :--- | :--- | :--- | :--- | :--- |
| id1 | 0.71 | 0.28 | 0.39 | 0.11 |
| id2 | 0.45 | 0.17 | 0.81 | 0.02 |
| ... | ... | ... | ... | ... |
| id300 | ... | ... | ... | ... |

**因子方向定义：**
*   因子A：**越大越好**
*   因子B：**越大越好**
*   因子C：**越小越好**
*   因子D：**越小越好**

### 划分区间与设定分值
接下来，我们需要为每个因子的数值划分区间，并为每个区间设定对应的分数。数值通常需要进行归一化处理，使其落在0到1之间。

以下是划分区间和设定分值的两种情形：

**1. 对于“越大越好”的因子（如A、B）：**
数值越大，得分越高。

| 数值区间 | 得分 |
| :--- | :--- |
| [0.8, 1.0] | 5 |
| [0.6, 0.8) | 4 |
| [0.4, 0.6) | 3 |
| [0.2, 0.4) | 2 |
| [0.0, 0.2) | 1 |

**2. 对于“越小越好”的因子（如C、D）：**
数值越小，得分越高。

| 数值区间 | 得分 |
| :--- | :--- |
| [0.0, 0.2) | 5 |
| [0.2, 0.4) | 4 |
| [0.4, 0.6) | 3 |
| [0.6, 0.8) | 2 |
| [0.8, 1.0] | 1 |

### 计算单因子得分
根据上述规则，我们可以计算每只股票在每个因子上的得分。

以下是计算示例：

*   **股票 id1 打分：**
    *   因子A = 0.71 → 落在 [0.6, 0.8) 区间 → **得分：4**
    *   因子B = 0.28 → 落在 [0.2, 0.4) 区间 → **得分：2**
    *   因子C = 0.39 → 落在 [0.2, 0.4) 区间 → **得分：4** (注意：C是越小越好)
    *   因子D = 0.11 → 落在 [0.0, 0.2) 区间 → **得分：5** (注意：D是越小越好)
    *   **总分 = 4 + 2 + 4 + 5 = 15**

*   **股票 id2 打分：**
    *   因子A = 0.45 → 落在 [0.4, 0.6) 区间 → **得分：3**
    *   因子B = 0.17 → 落在 [0.0, 0.2) 区间 → **得分：1**
    *   因子C = 0.81 → 落在 [0.8, 1.0] 区间 → **得分：1**
    *   因子D = 0.02 → 落在 [0.0, 0.2) 区间 → **得分：5**
    *   **总分 = 3 + 1 + 1 + 5 = 10**

### 汇总与选股
对池子中的所有股票（例如沪深300成分股）重复上述打分过程，得到每只股票的总分。

然后，对所有股票按总分从高到低进行排序。最终，选择排名前十的股票作为下一次调仓的目标投资组合。

**代码逻辑示意：**
```python
# 假设 scores 是一个字典，键为股票ID，值为总分
sorted_stocks = sorted(scores.items(), key=lambda x: x[1], reverse=True)
top_10_stocks = [stock_id for stock_id, _ in sorted_stocks[:10]]
```

## 整体策略流程梳理

![](img/ed30745435204c88e6baec34ca7b2ed2_5.png)

理解了打分法的核心步骤后，我们来看整个量化策略的实现流程。

![](img/ed30745435204c88e6baec34ca7b2ed2_7.png)

### 第一步：确定股票池
首先，需要确定选股的范围。在本例中，我们使用**沪深300指数**的成分股作为初始股票池。这确保了我们在具有一定流动性和市场代表性的大盘股中进行选择。

![](img/ed30745435204c88e6baec34ca7b2ed2_9.png)

### 第二步：设置调仓周期
策略通常不是每日交易，而是定期调整。常见的调仓周期有每月或每季度。我们需要设置一个**定时器**，在每个调仓日触发相应的选股和调仓函数。

### 第三步：实现调仓函数
这是策略的核心部分，我们将其命名为 `rebalance` 函数。在该函数中，我们需要依次完成以下任务：

以下是 `rebalance` 函数内需要执行的关键步骤列表：

1.  **获取数据**：在调仓日，获取股票池中所有股票的最新因子数据（A, B, C, D）。
2.  **因子打分**：应用前面讲解的打分法，为每只股票的每个因子计算得分。
3.  **计算总分**：将每只股票的所有因子得分相加，得到其综合总分。
4.  **排序选股**：根据总分对所有股票进行降序排序，选出排名前十的股票。
5.  **执行调仓**：将投资组合的权重调整到新选出的十只股票上（例如，等权重配置）。

### 第四步：回测与评估
将上述流程在历史数据中运行，可以评估该打分策略的收益、风险等表现，例如观察策略能否实现超越基准的收益。

## 总结
本节课中我们一起学习了量化选股中的**因子打分法**。我们首先明确了因子的好坏方向，然后通过划分区间的方式为每个因子赋值，接着汇总得到股票的总分并据此进行排名选股。最后，我们梳理了整合打分法到一个完整量化策略中的关键流程：确定股票池、设置调仓周期、实现核心调仓函数。这是一种直观且常用的多因子选股方法，为后续构建更复杂的模型奠定了基础。
