## ⚠️ 第一讲：理解风险与收益

上一节我们介绍了多种策略框架和因子挖掘方法。本节中，我们来看看交易中更为根本的要素：风险。

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_3.png)

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_5.png)

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_6.png)

一个策略的潜在收益，本质上是由其承担的潜在风险决定的。风险与收益同源。策略所承担的潜在风险，决定了其潜在的收益。

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_8.png)

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_10.png)

举一个简单的例子：购买保险需要支付保费。按照平均出险率计算，保险公司可以获得盈利。作为保险买家，我们付出保费，其回报是在重大突发事件时获得相应的损失弥补。

回到金融市场，我们有两种类似的收益风格：
*   如果我们希望成为购买保险的人，即在市场剧烈波动或下跌时获得额外收益，那么我们需要接受一个事实：平时的回报率可能会低于市场平均回报。
*   与我们做对手盘的人，即出售保险的人，会通过赚取额外的风险溢价来获得更高回报。从某种意义上说，并非其投资水平更高，而是其赚取了承担额外风险所对应的溢价。

所以，类似于保险公司的利润，如果某些资产容易受到特定风险的影响，那么购买这些资产就可以获得额外的回报。

我们想强调的一点是：**风险敞口决定了我们的收益**。

为什么投资者会选择投资股票，而不是债券等更安全的资产？因为投资者期望从股票中获得额外的回报。这些回报来自于一系列的风险因素。

例如，曾经有一段时间，小市值公司可以稳定跑赢大市值公司，这后来被证实不是一个阿尔法因子，而是一个风险因子。事实上，后期可能发现大市值公司跑赢小市值公司。我们强调的这是一个市值或公司规模因子。

在债券中也有类似情况：较长期债券的收益率通常高于短期债券。因为本质上，借出更长时间的钱需要承担更长期限的风险，因此可以获得更多的风险补偿。

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_12.png)

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_14.png)

所以说，持续的风险溢价会导致不同资产之间的收益率产生区别，例如股票和债券，长期债券与短期债券。

另外需要考虑的一点是：人们对风险的偏好是保持不变的吗？显然不是。这是一个随时间变化的量。如果风险随时间变化，并且我们能够捕捉到人们风险偏好的变化，那么就存在低买高卖的交易机会。我们赌的是人们的风险偏好会出现均值回归。

例如，当前我们认为某项资产存在溢价，可以卖出该资产。我们认为等过一段时间，人们的风险偏好回归正常时，资产价格也会相应回归正常水平。当然，这一点需要谨慎，因为投资者的交易心理会变化，市场由投资者的交易行为构成。因此，并不存在一个完全正确的解答，到底什么是正确的估值或市场状态，这需要每个人有自己的判断。

另外还想强调的一点是：作为一个理性的投资者，我们关心的是资产的夏普比率。但“理性”的假设是否正确呢？对于具有相同夏普比率的两种资产，我们在选择时真的没有偏好吗？事实上，这个理性假设可能并不完全正确。

因为具有相同夏普比率的资产，可能存在不同的收益分布。一种分布类似于购买保险：大部分时候小亏，偶尔出现大额盈利。另一种分布则相反：大部分时候稳定小赚，但作为保险的卖方，在突发事件时可能会大亏。这体现的其实是**偏度**的问题。

接下来，我们通过编程来看一下具有相同夏普比率但不同偏度的两种资产的收益分布。

## 📊 第二讲：偏度与收益分布

上一节我们介绍了风险与偏度的概念。本节中，我们通过代码来具体生成和可视化两种不同偏度的收益分布。

以下是生成具有目标夏普比率和偏度的收益分布的核心步骤：

1.  **设定目标参数**：我们需要年化夏普比率、目标偏度和目标波动率。
    ```python
    target_sharpe = 1.0  # 年化夏普比率
    target_skew = 1.0    # 目标偏度 (正偏度)
    target_vol = 0.20    # 年化波动率 20%
    size = 2500          # 样本数量
    ```
2.  **计算日度参数**：将年化参数转化为日度参数（假设一年有252个交易日）。
    ```python
    days_in_year = 252
    annual_return = target_sharpe * target_vol
    daily_return = annual_return / days_in_year
    daily_vol = target_vol / np.sqrt(days_in_year)
    ```
3.  **生成伽马分布样本**：我们使用伽马分布来构造具有特定偏度的分布。伽马分布的概率密度函数为：`f(x; k, θ) = x^(k-1) * e^(-x/θ) / (θ^k * Γ(k))`，其中 `k` 是形状参数，`θ` 是尺度参数。
    ```python
    # 计算伽马分布参数
    eps = 1e-6  # 防止除零
    shape_param = 2.0 / (abs(target_skew) + eps)
    scale_param = daily_vol / np.sqrt(shape_param)
    # 生成样本
    samples = np.random.gamma(shape=shape_param, scale=scale_param, size=size)
    ```
4.  **调整偏度方向与均值**：根据目标偏度的正负调整样本方向，并将其均值调整到目标日度收益率。
    ```python
    adjusted_signal = 1.0 if target_skew >= 0 else -1.0
    sample_mean = shape_param * scale_param * adjusted_signal
    adjustment = daily_return - sample_mean
    adjusted_samples = [x * adjusted_signal + adjustment for x in samples]
    ```
5.  **可视化结果**：绘制收益分布的直方图。
    ```python
    import matplotlib.pyplot as plt
    plt.hist(adjusted_samples, bins=100)
    plt.grid(True)
    plt.title(f‘Sharpe={target_sharpe}, Skew={target_skew}, Vol={target_vol}‘)
    plt.show()
    ```

**结果分析**：
*   **正偏度分布**：大部分交易日小幅亏损，但偶尔有几天获得巨大盈利。这种策略类似于“买入期权”或“趋势跟踪”，具有较好的抗黑天鹅能力。
*   **负偏度分布**：大部分交易日稳定小幅盈利，但偶尔遭遇巨大亏损。这种策略类似于“卖出期权”或“做市”，平时赚钱，但极端行情下风险很大。

这两种分布没有绝对的优劣，取决于个人风格和资金性质。机构投资者通常更倾向于具有正偏度的策略，以控制最大回撤。个人投资者需要清楚自己的策略属于哪种类型，并了解其隐含的风险。

## 🧩 第三讲：风险的多维度考量

上一节我们通过偏度深入了解了收益分布的风险特征。本节中，我们系统性地总结量化交易中需要关注的多种风险类型。

以下是交易中需要系统考虑的主要风险维度：

1.  **长期风险溢价**：不同资产类别（如股票 vs 债券）或同一资产的不同属性（如长期债 vs 短期债）之间存在持续的风险溢价，这是收益的根本来源之一。
2.  **时变的风险偏好**：市场的风险偏好并非恒定，会随着经济周期、市场情绪等因素变化。例如，美林投资时钟模型就描述了在不同经济阶段应配置何种资产。投资者需要判断当前市场的风险偏好阶段。
3.  **偏度风险**：如前所述，即使夏普比率相同，收益分布的形态（偏度）可能截然不同，这代表了不同的风险特征。在评估策略时，需要进行压力测试，考察极端情形下的表现。
4.  **流动性风险**：当资金规模达到一定程度时，需要考虑资产的流动性。在小盘股中投入大量资金可能导致难以出货。机构投资者偏好流动性好的资产（如股票、期货），以便应对申购赎回。低流动性资产（如房地产、私募股权）通常要求更高的流动性风险补偿。
5.  **交易壁垒与成本**：看到利润不等于能赚到利润。交易可能受到法规壁垒（如外资投资A股的限制）、技术壁垒（如高频交易需要机房托管和专线）或尽职调查成本（如私募股权投资）的限制。必须权衡收益与成本。
6.  **行为金融学风险**：传统金融学的“理性人”假设常被打破。例如，投资者倾向于“止盈”以获得成就感，但不愿“止损”以回避承认错误。基于行为金融学的策略可以捕捉这些非理性行为带来的机会。
7.  **运营风险**：指由内部流程、人员、系统或外部事件导致损失的风险。一个著名的案例是骑士资本因高频交易服务器程序错误，在一小时内亏损了4.5亿美元。

**核心总结**：风险与收益同源。成功的量化交易不仅在于寻找阿尔法，更在于清晰地识别、衡量和管理上述各类风险。务必对自己的资金负责，谨慎使用杠杆。

## 🔄 第四讲：最小扭转法——一种改进的因子正交化方法

上一节我们全面探讨了各类风险。本节中，我们转向因子处理的一个具体技术：如何更好地对因子进行正交化处理以进行风险预算和分散化。

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_22.png)

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_24.png)

传统的风险平价策略旨在配置风险，而非资产。我们希望每种资产（或因子）对组合整体风险的贡献度相同。为了准确计算贡献度，理想状态是让不同的风险因子之间互不相关（正交）。

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_25.png)

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_27.png)

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_28.png)

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_30.png)

主成分分析是常用的正交化方法。它将原始因子转换为一组新的、互不相关的主成分。然而，PCA有两个主要缺点：
1.  **统计不稳定性**：尤其是对于特征值较小的成分，结果不稳定。
2.  **可解释性差**：得到的主成分是原始因子的线性组合，其经济含义模糊，难以解释。

**最小扭转法** 旨在解决这些问题。它的核心思想是：寻找一个转换矩阵，将原始因子旋转为一组新的正交因子，但同时要求这个旋转尽可能“小”，即新的因子与原始因子的跟踪误差最小。

**数学表述**：
假设我们有 `K` 个原始因子向量 `f`，其协方差矩阵为 `Σ_f`。我们希望找到一个扭转矩阵 `T`，使得新的因子 `z = T f` 满足：
*   `z` 的各个分量互不相关（即 `Cov(z)` 是对角阵）。
*   扭转矩阵 `T` 在满足上述条件下，最小化其与单位矩阵 `I` 的“距离”，这个距离用归一化跟踪误差衡量：
    `min_T || T - I ||_F`， subject to `Cov(z) = D` (D为对角阵)
    其中 `||·||_F` 是弗罗贝尼乌斯范数。

**与PCA的直观对比**：
*   **PCA**：像将一组散乱的点重新映射到一组全新的、方差最大的正交坐标轴上。新坐标轴方向可能完全偏离原始因子方向。
*   **最小扭转法**：像将这组点进行一个尽可能轻微的“旋转”，直到它们在新的坐标系下变得正交。新坐标轴方向与原始因子方向非常接近。

**优势**：
1.  **保持可解释性**：由于旋转最小，新因子与原始因子高度相似，因此更容易理解其经济含义。
2.  **稳定性更好**：对于相关性较弱的资产组合，PCA可能给出不稳定的权重分配（例如，将所有权重集中于一个主成分），而最小扭转法能得到更符合直觉的、分散化的结果。
3.  **适用于风险预算**：得到正交因子后，可以清晰计算每个因子对组合风险的边际贡献，从而更精确地进行风险预算和配置。

**实现**：该方法的求解涉及一个带约束的优化问题，通常使用迭代算法。原作者提供了MATLAB代码。在Python中，可以利用优化库（如 `scipy.optimize`）来实现。

**应用建议**：在因子投资中，当您挖掘出一批候选因子后，可以使用最小扭转法替代PCA，对这些因子进行正交化处理。这样既能降低因子间的多重共线性，又能最大程度保留因子的原始经济意义，便于后续的因子选择和组合构建。

## 🎲 第五讲：基于贝叶斯引源模型的交易策略

上一节我们介绍了一种改进的因子处理方法。本节中，我们来看一个完全不同的策略框架：基于贝叶斯引源模型的预测与交易。

传统的线性回归模型 `y = Xβ + ε` 在高维金融数据（特征维度 >> 样本数量）中容易过拟合，即使使用Lasso等正则化方法。

本文提出了一种**贝叶斯引源模型**。其核心思想是：
1.  **假设存在K个潜在的“引源”**：每个引源代表市场的一种特定状态或模式。
2.  **每个数据点由某个引源生成**：对于第 `t` 个样本，它属于第 `k` 个引源的概率为 `π_k`。在该引源下，其特征 `x_t` 由引源的中心 `s_k` 加上高斯噪声生成，其标签 `y_t` 也服从该引源下的条件分布。
3.  **预测转化为贝叶斯推断**：给定新的观测特征 `x`，预测其标签 `y` 的问题，转化为计算后验概率：
    `P(y|x) ∝ Σ_{k=1}^K P(y|x, 源=k) * P(源=k|x)`
    其中，`P(源=k|x)` 可以通过计算 `x` 与各引源中心 `s_k` 的相似度来估计。

**策略实现步骤（以预测比特币短期价格方向为例）**：
1.  **数据准备**：使用限价订单簿数据，构造特征（如价格序列、买卖不平衡度 `TI`）和标签（未来10秒价格平均变化）。
2.  **第一阶段：模式发现**：
    *   使用历史数据（如过去30分钟窗口）。
    *   采用聚类算法（如K-Means）对所有可能的时间序列片段进行聚类。
    *   从聚类结果中选取最具代表性的 `M` 个簇中心作为“引源” `{s_1, ..., s_M}`。这些簇中心就是发现的典型市场模式。
3.  **第二阶段：参数学习**：
    *   在新的历史数据窗口上，对于每个引源 `s_j`，学习一个简单的线性模型，用于预测在该模式下未来价格的变动。模型参数 `w_j` 通过回归学习得到。
4.  **第三阶段：在线交易**：
    *   对于新的时刻，获取当前特征 `x`。
    *   计算 `x` 与每个引源 `s_j` 的相似度（如负的欧氏距离平方）。
    *   根据相似度加权聚合各引源的预测结果 `w_j^T x`，得到最终的价格变动预测 `Δp`。
    *   如果 `Δp > 阈值T`，则买入；如果 `Δp < -T`，则卖空。

![](img/d9bbfb0fa29a5ac2a84bd394aa282414_32.png)

**策略特点与创新**：
*   **非参数与贝叶斯结合**：避免了传统线性回归设定固定参数形式的限制，通过数据驱动的模式发现来捕捉市场结构。
*   **可解释的模式**：发现的引源（簇中心）是具体的价格序列模式，可以可视化，便于理解策略的逻辑。
*   **适应市场变化**：通过定期重新执行模式发现和参数学习，可以适应市场状态的变化。

**对A股/期货市场的启示**：
*   可以尝试将此框架应用于A股或商品期货市场。
*   关键调整可能在于：特征工程（需寻找有效的市场微观特征）、聚类算法与数量的选择（A股模式可能更复杂，需要更多簇）、交易频率和阈值的设定。
*   这个框架的价值在于提供了一种从数据中自动发现“有效市场模式”并进行贝叶斯决策的思路，不同于传统的因子挖掘范式。

## 🎯 课程总结
本节课我们一起学习了量化交易中三个层次的重要内容：

1.  **风险认知**：深入理解了风险与收益的同源关系，认识了包括偏度风险、流动性风险、运营风险在内的多种风险类型，并学会了通过编程分析收益分布的特征。
2.  **技术工具**：学习了一种名为“最小扭转法”的先进因子正交化技术。它在保持因子可解释性的同时实现风险因子的分散化，优于传统的主成分分析方法。
3.  **策略框架**：探讨了一个基于贝叶斯引源模型的交易策略。该框架通过聚类发现市场模式，并利用贝叶斯推断进行决策，为处理高维金融数据提供了新思路。

量化交易的核心不仅在于寻找预测信号（阿尔法），更在于对风险的全面管理、对策略特征的深刻理解，以及不断吸收新的方法论来优化整个投资系统。希望本课内容能帮助大家在量化交易的道路上建立更稳固、更敏锐的认知框架。
