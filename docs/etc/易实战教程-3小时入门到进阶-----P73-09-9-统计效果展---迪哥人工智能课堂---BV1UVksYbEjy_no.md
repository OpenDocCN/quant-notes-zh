上一节我们介绍了如何使用 `groupby` 进行数据分组。本节中，我们来看看如何对分组结果进行统计分析，并基于此制定策略。

![](img/746af06258e4bb378d603b1f2db46b5f_4.png)

结果拿到手之后，接下来需要做统计。统计的目标是观察哪一种可能性多，哪一种可能性少。

![](img/746af06258e4bb378d603b1f2db46b5f_6.png)

![](img/746af06258e4bb378d603b1f2db46b5f_8.png)

首先，需要对数据做一点变换。当前需要观察的是前两个特征与最终结果的关系，但数据被组合在一起了。因此，对数据的最后这一列（即结果列）进行操作。

![](img/746af06258e4bb378d603b1f2db46b5f_10.png)

以下是具体操作，使用 `unstack` 方法并填充空值：

![](img/746af06258e4bb378d603b1f2db46b5f_12.png)

```python
# 对分组结果进行unstack操作，并填充空值为0
result = data.groupby(['feature1', 'feature2'])['result'].size().unstack(fill_value=0)
```

![](img/746af06258e4bb378d603b1f2db46b5f_14.png)

执行后，所有特征组合的结果都会显示出来，没有值的地方会填充为0。此时得到的结果表格，行和列分别是两个特征，表格内的值是对应组合下的结果数量。这便于观察特征不同变化时，最终结果的分布情况。

接下来，需要基于此进行统计，制定策略。

以下是执行策略的步骤。我们将数据传入，指定代表“前一天涨跌”和“前两天涨跌”的两列特征。

```python
# 计算两个特征的和，判断是否等于2（即前两天都上涨）
data['sum'] = data[['day-1', 'day-2']].sum(axis=1)
condition = data['sum'] == 2
```

然后，根据统计结果进行判断。通过观察发现，当“前一天涨”且“前两天涨”时，第三天大概率下跌；其他情况下则大概率上涨。

基于这个观察，使用 `np.where` 来生成交易信号：

![](img/746af06258e4bb378d603b1f2db46b5f_16.png)

```python
import numpy as np
# 根据特征和是否等于2，生成交易信号：下跌为-1，上涨为1
data['position'] = np.where(condition, -1, 1)
```

这个判断纯粹是根据当前数据统计结果数出来的。不同数据集的结果可能不同。这只是一种简单的统计方法演示。

![](img/746af06258e4bb378d603b1f2db46b5f_18.png)

统计方法完成后，需要计算策略的预测效果。

![](img/746af06258e4bb378d603b1f2db46b5f_20.png)

将实际的涨跌方向 `direction` 与我们基于统计生成的 `position` 信号进行比较，查看预测正确的天数。

![](img/746af06258e4bb378d603b1f2db46b5f_22.png)

![](img/746af06258e4bb378d603b1f2db46b5f_24.png)

```python
# 比较预测信号与实际方向一致的次数
correct_predictions = (data['position'] == data['direction']).value_counts()
print(correct_predictions)
```

结果显示，有1102天预测正确，1033天预测错误。基于统计的方法，预测正确的次数略多一些。

![](img/746af06258e4bb378d603b1f2db46b5f_26.png)

![](img/746af06258e4bb378d603b1f2db46b5f_28.png)

既然预测正确次数更多，最后来评估该策略在回测中的收益表现。

使用与之前评估聚类策略相同的回测函数，但传入基于统计方法生成的 `position` 列。

![](img/746af06258e4bb378d603b1f2db46b5f_30.png)

```python
# 计算基于统计策略的累计收益
returns = data['returns'] * data['position']
cumulative_returns = (returns + 1).cumprod()
print(cumulative_returns.iloc[-1])
```

![](img/746af06258e4bb378d603b1f2db46b5f_32.png)

![](img/746af06258e4bb378d603b1f2db46b5f_34.png)

执行后，得到策略的最终累计收益。结果显示，虽然仍然是亏损，但基于统计的策略比单纯跟随大盘走势亏损得更少。这说明了统计策略在一定程度上的有效性。

![](img/746af06258e4bb378d603b1f2db46b5f_36.png)

![](img/746af06258e4bb378d603b1f2db46b5f_38.png)

本节课中我们一起学习了如何对分组数据进行统计分析，并基于统计概率制定简单的量化交易策略。核心操作包括使用 `groupby` 与 `unstack` 分析特征组合，通过观察统计规律生成交易信号，并最终通过回测评估策略效果。

![](img/746af06258e4bb378d603b1f2db46b5f_44.png)

这种基于统计的策略核心思想是**比较不同情况下的可能性大小**，选择概率更大的结果作为预测。虽然本例中的策略相对简单，但清晰地展示了数据分析在量化交易中的一个基本应用出发点：从历史数据中寻找统计规律以指导未来决策。
