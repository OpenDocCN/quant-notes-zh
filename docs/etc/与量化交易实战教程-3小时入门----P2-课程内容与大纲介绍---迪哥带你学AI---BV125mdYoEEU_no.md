## 课程风格与工具 🛠️

![](img/0e18a471a7f0723284d19539963bf577_19.png)

本课程是一门纯实战课程，全程通过代码演示进行教学，不依赖PPT，也不教授基础的金融或炒股知识。重点完全放在“如何用Python实现任务”上。

![](img/0e18a471a7f0723284d19539963bf577_21.png)

以下是课程中将使用的主要工具和环境介绍：

![](img/0e18a471a7f0723284d19539963bf577_23.png)

*   **开发环境**：我们将使用 **Jupyter Notebook** 作为主要的代码演示和练习环境。它非常适合进行交互式的数据分析和探索。
*   **核心工具包**：课程将重点依赖两个Python库：
    1.  **NumPy**：用于高效的数值计算。其核心是`ndarray`（多维数组）对象。
        ```python
        import numpy as np
        array = np.array([1, 2, 3, 4, 5])
        ```
    2.  **Pandas**：用于数据分析和处理，尤其是表格型和时间序列数据。其核心是`DataFrame`（数据框）对象。
        ```python
        import pandas as pd
        df = pd.DataFrame({‘Open‘: [100, 101], ‘Close‘: [102, 103]})
        ```
*   **回测平台**：我们将使用专业的量化回测平台来验证交易策略。在平台上，你可以编写策略逻辑，然后**在历史数据上模拟交易**，并得到详细的回测报告，包括：
    *   **收益曲线**：策略净值随时间的变化。
    *   **基准对比**：策略收益与大盘指数（如沪深300）收益的对比。
    *   **交易详情**：每日买卖记录、持仓情况。
    *   **风险指标**：最大回撤、夏普比率等。

## 详细课程大纲 📋

![](img/0e18a471a7f0723284d19539963bf577_29.png)

以下是课程三个模块的详细内容分解。

### 模块一：Python必备工具包实战
本模块为零基础学员打造，目标是打好Python数据处理的基础。
以下是本模块的核心学习点：
1.  **环境配置**：Python及相关库的安装与配置。
2.  **Python基础**：课程所需的Python核心语法精讲。
3.  **NumPy详解**：数组创建、操作、数学计算。
4.  **Pandas详解**：`DataFrame`与`Series`的数据读取、清洗、筛选、分组聚合。

![](img/0e18a471a7f0723284d19539963bf577_31.png)

![](img/0e18a471a7f0723284d19539963bf577_33.png)

### 模块二：Python金融数据分析实战
本模块将理论与实践结合，专注于金融数据。
以下是本模块的核心学习点：
1.  **金融数据获取**：如何获取股票、基金等历史行情数据。
2.  **数据分析实战**：对价格、成交量等数据进行统计分析与可视化。
3.  **时间序列分析**：金融时间序列的常用处理方法。
4.  **初探回测**：在量化平台上完成第一个简单的回测项目，理解回测流程与核心指标。

### 模块三：量化交易策略深入
本模块深入量化交易策略的研发与优化。
以下是本模块的核心学习点：
1.  **经典策略实现**：学习并实现如动量策略、均值回归等经典量化策略。
2.  **策略建模与分析**：基于数据特征建立模型，制定交易信号。
3.  **回测进阶**：策略的优化、参数调优以及如何避免过拟合。
4.  **绩效分析**：深入分析回测结果，从多维度评估策略的有效性与稳健性。
