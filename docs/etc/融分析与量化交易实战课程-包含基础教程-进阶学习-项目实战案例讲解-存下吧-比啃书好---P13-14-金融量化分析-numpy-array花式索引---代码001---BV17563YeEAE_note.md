## 概述
在本节课中，我们将要学习NumPy数组的另一种高级索引方式——花式索引。我们将了解它的基本概念、使用方法，以及如何将其与其他索引方式结合使用，以灵活地从数组中提取数据。

## 花式索引的基本概念
上一节我们介绍了布尔型索引，本节中我们来看看花式索引。花式索引用于从数组中选取一组**没有规律**的特定元素。例如，对于一个数组，我们可能需要选取其第1、3、4、6、7号元素来组成一个新的数组。

**核心概念**：花式索引通过在索引位置传入一个整数列表或数组来实现。
```python
import numpy as np
a = np.arange(20)
selected = a[[1, 3, 4, 6, 7]]
```

## 花式索引的用法
以下是花式索引的具体操作步骤：
1.  创建一个NumPy数组。
2.  定义一个包含所需元素位置的整数列表。
3.  将该列表作为索引传入原数组。

注意，花式索引与普通索引（单个数字）和切片（`start:stop:step`）不同，它允许我们指定任意顺序的索引组合。

## 二维数组的花式索引
对于二维数组，花式索引的规则同样适用。NumPy提供了四种主要的索引方式：普通索引、切片、布尔型索引和花式索引。在二维数组中，我们可以通过逗号分隔行和列的索引来组合使用这些方法。

例如，我们可以用常规索引选择行，用切片选择列：
```python
arr_2d = np.arange(20).reshape(4, 5)
# 选择第0行，第2到第4列（不包含第4列）
result = arr_2d[0, 2:4]
```
也可以组合常规索引和布尔型索引：
```python
# 选择第0行中所有大于2的元素
result = arr_2d[0, arr_2d[0] > 2]
```

## 索引组合的注意事项与技巧
虽然多种索引方式可以混合使用，但有一个重要的限制：**逗号两边不能同时使用花式索引**，否则会产生意想不到的结果。这种写法会被解释为选取坐标`(i, j)`和`(m, n)`位置上的值，而不是分别按行和列进行花式索引。

例如，如果想选取第1、3行与第1、3列交叉的四个元素（即位置(1,1), (1,3), (3,1), (3,3)），不能直接写`arr_2d[[1,3], [1,3]]`。

以下是正确的分步操作方法：
1.  首先使用花式索引选取指定的行，并选取所有列。
    ```python
    rows = arr_2d[[1, 3], :]
    ```
2.  然后在结果上，选取所有行和指定的列。
    ```python
    final_result = rows[:, [1, 3]]
    ```
这种方法通过两次索引操作，清晰地实现了行列分别筛选的目的。
