## 数据表拼接操作

上一节我们介绍了如何获取和处理单个因子数据。本节中，我们来看看如何将多个因子数据表合并成一张大表，以便进行综合计算。

以下是实现数据拼接的步骤：

1.  使用Pandas的`concat`函数进行拼接。
2.  将拼接后的新`DataFrame`命名为`df_rank`。

具体的拼接操作代码如下：
```python
df_rank = pd.concat([df1, df2], axis=1)
```
这段代码将两个`DataFrame`（`df1`和`df2`）按列（`axis=1`）拼接在一起。

![](img/05eae10175cb3eaccd093cadbd946bef_4.png)

## 计算综合得分并排序

现在我们已经有了包含所有因子数据的大表。接下来，我们需要为每只股票计算一个综合得分，并根据得分进行排序。

以下是计算总分并排序的步骤：

1.  在`df_rank`中新建一个名为“得分值”的列，初始值设为0。
2.  计算所有因子列的总和，得到每只股票的综合得分。
3.  根据综合得分进行降序排列，得分最高的股票排在最前面。

实现计算的代码如下：
```python
# 新建得分列，初始化为0
df_rank[‘得分值’] = 0

# 计算所有因子列的总和（假设因子列名为‘factor1’， ‘factor2’...）
df_rank[‘得分值’] = df_rank[[‘factor1’， ‘factor2’， ‘factor3’]].sum(axis=1)

# 根据‘得分值’列进行降序排序
df_rank_sorted = df_rank.sort_values(by=‘得分值’， ascending=False)
```

## 获取最终选股结果

完成排序后，我们的目标就变得很明确：获取排名前十的股票代码。

以下是获取最终结果的步骤：

1.  从排序后的`DataFrame`中提取“得分值”这一列（一个`Series`对象）。
2.  获取这个`Series`的索引，即股票代码列表。
3.  选取列表中的前10个元素，即为最终选出的股票。

![](img/05eae10175cb3eaccd093cadbd946bef_6.png)

实现最终筛选的代码如下：
```python
# 提取得分列
score_series = df_rank_sorted[‘得分值’]

![](img/05eae10175cb3eaccd093cadbd946bef_8.png)

# 获取股票代码列表并选取前十名
top_10_stocks = score_series.index.tolist()[:10]
```

## 函数流程总结

![](img/05eae10175cb3eaccd093cadbd946bef_12.png)

本节课中我们一起学习了`get_stocks`函数的核心流程。这个函数完成了以下工作：

![](img/05eae10175cb3eaccd093cadbd946bef_14.png)

1.  **读取因子数据**：获取多个维度的股票指标。
2.  **数据拼接**：将所有因子数据合并到一张表中。
3.  **计算与打分**：对每个因子进行处理并计算综合得分。
4.  **排序与筛选**：根据综合得分降序排列，并返回排名前十的股票代码。

![](img/05eae10175cb3eaccd093cadbd946bef_16.png)

每次调用这个函数，它都会基于最新的因子数据，计算出当前时刻综合排名前十的股票，从而完成一次选股操作。这是构建自动化量化交易策略的重要基础。
