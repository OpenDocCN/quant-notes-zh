## 为什么要学习SPSS？🤔

SPSS是一款统计分析软件，其理论基础完全基于统计学知识。它涵盖了统计设计的完整流程，包括数据搜集、整理、分析和报告，这与数据分析的通用流程类似。

在实际工作中，并非所有企业都会使用SPSS。部分企业在构建模型或进行测试时可能会用到它。但使用SPSS必然需要统计学知识。

以下是SPSS的主要优点与劣势：
*   **优点**：功能强大、兼容性好、易于使用、扩展性高。
*   **劣势**：在统计计算速度上相对较慢。

## SPSS的发展史 📜

![](img/429dc2365327df25fe5cba8e767eb4a6_5.png)

![](img/429dc2365327df25fe5cba8e767eb4a6_7.png)

SPSS最初由美国斯坦福大学的三位研究生在20世纪60年代研发。其发展历程如下：
*   **70年代**：成立了SPSS公司。
*   **80年代**：推出了微型机版本。
*   **90年代**：开发了Windows版本（V6至V10）。
*   **2010年**：被IBM公司收购，更名为IBM SPSS Statistics。
*   **版本迭代**：从SPSS 11更新至25（目前常用稳定版本），界面和功能不断优化。

![](img/429dc2365327df25fe5cba8e767eb4a6_9.png)

与SPSS同类的统计分析软件还有SAS和R。SAS专业但难度较高且收费；R是一门编程语言，与Python类似。相比之下，SPSS和Python更易于学习和上手。

![](img/429dc2365327df25fe5cba8e767eb4a6_11.png)

SPSS采用按钮式操作界面，直观易用。被IBM收购后，它与Office办公软件无缝衔接，兼容性良好，并且支持与Python连接进行二次开发。

## SPSS操作界面 🖥️

在深入学习软件功能前，我们先了解其文件格式和操作界面。

SPSS默认生成的文件后缀名为 **`.sav`**。同时，它也支持导入多种格式的数据文件，如Excel（`.xls`, `.xlsx`）、CSV（`.csv`）、文本文件等，体现了其良好的兼容性。

打开一个`.sav`文件后，界面标题会显示两部分信息：文件名和数据集名称（如“数据集1”）。

![](img/429dc2365327df25fe5cba8e767eb4a6_15.png)

SPSS主要包含四大窗口：

![](img/429dc2365327df25fe5cba8e767eb4a6_17.png)

### 1. 数据编辑窗口

这是SPSS的核心工作区，由两个视图构成：
*   **数据视图**：用于查看和编辑具体的数据内容。行标为数字，列标直接显示字段名（变量名）。
*   **变量视图**：用于定义和修改变量的属性。它类似于数据库的表设计，可以设置变量的**名称**、**类型**、**宽度**、**标签**、**值**等。

这种设计结合了电子表格的直观性和数据库的结构化管理优点。

### 2. 结果查看器窗口

![](img/429dc2365327df25fe5cba8e767eb4a6_19.png)

在执行任何分析操作后，输出结果会显示在此窗口。它采用树形目录结构，清晰展示分析结果的各个部分（如标题、统计表、图表），方便用户快速定位和查阅。

![](img/429dc2365327df25fe5cba8e767eb4a6_21.png)

### 3. 语法编辑器窗口

![](img/429dc2365327df25fe5cba8e767eb4a6_23.png)

![](img/429dc2365327df25fe5cba8e767eb4a6_25.png)

![](img/429dc2365327df25fe5cba8e767eb4a6_27.png)

SPSS的所有菜单操作，本质上都在后台生成了对应的命令语法。你可以通过“文件”->“新建”->“语法”打开语法编辑器。将生成的语法粘贴至此并运行，可以复现之前的分析操作。这揭示了软件底层基于代码执行的原理。

### 4. 脚本编辑器窗口

![](img/429dc2365327df25fe5cba8e767eb4a6_29.png)

![](img/429dc2365327df25fe5cba8e767eb4a6_31.png)

![](img/429dc2365327df25fe5cba8e767eb4a6_33.png)

SPSS支持与其他编程语言交互以实现二次开发。通过“文件”->“新建”->“脚本”，可以打开脚本编辑器，并选择使用Python或Basic（VBA）语言编写脚本，从而扩展SPSS的功能。

## 基本使用方式 🛠️

### 菜单与对话框

SPSS主要通过菜单和对话框进行操作。例如，点击“分析”->“描述统计”->“探索”，会弹出一个对话框。用户可以在对话框中设置因变量、因子等，并点击“统计图”等按钮选择需要的输出内容。
对话框下方有多个按钮：
*   **确定**：执行分析。
*   **粘贴**：将当前设置对应的语法粘贴到语法编辑器窗口。
*   **重置**：清空当前选择。
*   **取消**：关闭对话框。
*   **帮助**：获取帮助信息。

### 变量类型与属性

在变量视图中，有几个关键属性需要理解：
*   **类型**：主要有数值、日期和字符串三种。
*   **标签**：对变量含义的文本说明（例如，变量“S0”的标签可以是“城市”）。
*   **值**：对于分类变量（如城市、性别），可以用数字编码来代表不同类别（例如，1=男，2=女）。在数据视图中，可以通过点击工具栏的“值标签”图标切换显示数字编码或实际的文本标签。
*   **测量**：定义变量的测量尺度，是统计分析的重要依据。
    *   **标度**：表示连续性的数值变量（如年龄、收入）。
    *   **有序**：表示有序分类变量（如学历：初中、高中、大学）。
    *   **名义**：表示无序分类变量（如城市、职业）。
