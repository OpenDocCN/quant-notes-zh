在本节课中，我们将学习如何处理Pandas Series中的缺失数据。缺失数据是数据分析中常见的问题，学会处理它们对于后续的计算和可视化至关重要。

上一节我们介绍了Series的基本操作，本节中我们来看看如何处理其中的缺失值。

## 什么是缺失数据

![](img/4153a41cdc56582ef786d3df75f5548c_3.png)

在Series中，可能会出现缺失数据，通常表示为`NaN`值。有时我们可以忽略这些缺失值，但在进行数学运算或生成图表时，它们可能会带来问题或影响观感，因此需要处理。

## 缺失数据处理方法

处理缺失数据主要有两种思路：删除缺失值或填充缺失值。

![](img/4153a41cdc56582ef786d3df75f5548c_5.png)

### 方法一：删除缺失值

![](img/4153a41cdc56582ef786d3df75f5548c_7.png)

第一种方法是直接删除包含缺失数据的行。

![](img/4153a41cdc56582ef786d3df75f5548c_9.png)

首先，我们需要判断哪些数据是缺失的。以下是相关的函数：

*   `SR.isnull()`: 判断每个元素是否为缺失值。返回一个布尔型Series，是`NaN`则对应`True`，否则为`False`。
*   `SR.notnull()`: 判断每个元素是否**不是**缺失值。是`NaN`则对应`False`，否则为`True`。

![](img/4153a41cdc56582ef786d3df75f5548c_11.png)

利用布尔索引的特性，我们可以过滤掉缺失值：

![](img/4153a41cdc56582ef786d3df75f5548c_13.png)

```python
# 使用 notnull() 进行过滤
SR_filtered = SR[SR.notnull()]
```

![](img/4153a41cdc56582ef786d3df75f5548c_15.png)

此外，Pandas提供了一个更直接的方法来删除缺失值：

```python
# 使用 dropna() 函数删除所有包含 NaN 的行
SR_dropped = SR.dropna()
```

### 方法二：填充缺失值

第二种方法是为缺失值赋予一个具体的值。

使用`fillna()`函数可以实现填充。例如，将所有缺失值填充为0：

![](img/4153a41cdc56582ef786d3df75f5548c_17.png)

```python
# 将缺失值填充为 0
SR_filled_zero = SR.fillna(0)
```

![](img/4153a41cdc56582ef786d3df75f5548c_19.png)

**重要提示**：Pandas的数据处理方法（如`dropna`, `fillna`）默认不会修改原始Series，而是返回一个新的Series对象。如果你想保存处理后的结果，必须将其赋值给一个新变量。

![](img/4153a41cdc56582ef786d3df75f5548c_21.png)

在实际应用中，填充的值可以根据业务逻辑决定。除了填充固定值（如0），另一种常见的做法是填充统计值，例如该列的平均值。

```python
# 将缺失值填充为该Series的平均值
SR_filled_mean = SR.fillna(SR.mean())
```

这里有一个便利之处：`SR.mean()`在计算平均值时，会自动忽略`NaN`值，只对有效数据进行计算，这大大简化了处理流程。

![](img/4153a41cdc56582ef786d3df75f5548c_23.png)

本节课中我们一起学习了处理Pandas Series中缺失数据的两种核心方法：删除（`dropna`）与填充（`fillna`）。掌握这些方法能有效清理数据，为后续的金融分析与建模打下坚实基础。
