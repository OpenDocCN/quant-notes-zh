## 双均线策略概述

![](img/50fb95b30295dd83f06598ab3a4d90b7_7.png)

上一节我们介绍了量化分析的基本框架，本节中我们来看看一个具体的交易策略：双均线策略。

双均线策略的核心是使用两条不同周期的移动平均线来分析价格趋势。移动平均线能帮助我们平滑价格波动，从而更清晰地观察整体趋势，而非仅仅关注单日的价格变化。

![](img/50fb95b30295dd83f06598ab3a4d90b7_9.png)

## 什么是双均线？

![](img/50fb95b30295dd83f06598ab3a4d90b7_11.png)

![](img/50fb95b30295dd83f06598ab3a4d90b7_13.png)

“双均线”中的“双”指的是两条均线。通常，一条是短期均线，另一条是长期均线。

![](img/50fb95b30295dd83f06598ab3a4d90b7_15.png)

![](img/50fb95b30295dd83f06598ab3a4d90b7_17.png)

*   **短期均线**：计算周期较短，例如5天或10天的收盘价平均值。它能反映价格的近期变化趋势。
    *   **公式**：`SMA_short = (P1 + P2 + ... + Pn) / n`，其中 `n` 为短期周期（如5）。
*   **长期均线**：计算周期较长，例如20天、40天或60天的收盘价平均值。它能反映价格的中长期趋势。
    *   **公式**：`SMA_long = (P1 + P2 + ... + Pm) / m`，其中 `m` 为长期周期（如20）。

短期均线对价格变化更敏感，波动较大；长期均线则更为平滑，趋势更稳定。两者的相对位置和交叉关系，构成了交易信号的基础。

![](img/50fb95b30295dd83f06598ab3a4d90b7_19.png)

## 核心概念：金叉与死叉

![](img/50fb95b30295dd83f06598ab3a4d90b7_21.png)

![](img/50fb95b30295dd83f06598ab3a4d90b7_23.png)

当我们把短期均线和长期均线绘制在同一张图上时，它们的交叉点就具有重要的指示意义。以下是两种关键的交叉形态：

![](img/50fb95b30295dd83f06598ab3a4d90b7_25.png)

1.  **黄金交叉（金叉）**
    *   **定义**：短期均线从下方向上穿越长期均线。
    *   **市场含义**：这通常意味着短期趋势开始转强，并可能带动长期趋势向上。被视为一个潜在的**买入信号**。
    *   **图示**：短期均线（如红色线）向上穿过长期均线（如蓝色线）。

![](img/50fb95b30295dd83f06598ab3a4d90b7_27.png)

![](img/50fb95b30295dd83f06598ab3a4d90b7_29.png)

2.  **死亡交叉（死叉）**
    *   **定义**：短期均线从上方向下穿越长期均线。
    *   **市场含义**：这通常意味着短期趋势开始走弱，并可能引发长期趋势下跌。被视为一个潜在的**卖出信号**。
    *   **图示**：短期均线（如红色线）向下穿过长期均线（如蓝色线）。

![](img/50fb95b30295dd83f06598ab3a4d90b7_31.png)

简单来说，金叉提示“可能上涨，考虑买入”，死叉提示“可能下跌，考虑卖出”。

## 策略逻辑与应用

理解了金叉和死叉后，我们就可以构建一个简单的交易策略逻辑。

这个策略的思路是：通过程序自动识别金叉和死叉的发生，并据此执行买卖操作。

![](img/50fb95b30295dd83f06598ab3a4d90b7_33.png)

![](img/50fb95b30295dd83f06598ab3a4d90b7_35.png)

以下是该策略的基本操作规则：
*   当出现**黄金交叉**时，执行买入操作。
*   当出现**死亡交叉**时，执行卖出操作。

理论上，如果每次信号都预测准确，通过“低买高卖”的循环，就能实现盈利。在实际应用中，我们会使用Python等工具，计算移动平均线，并编程实现信号的自动识别与策略的回测验证。
