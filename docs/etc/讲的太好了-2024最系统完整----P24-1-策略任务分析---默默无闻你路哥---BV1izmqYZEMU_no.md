## 策略目标概述

![](img/e052a18ca152baba536bde7fb810a28f_3.png)

我们的目标是构建一个基于沪深300指数的选股策略。该策略的核心思想是：始终持有沪深300成分股中表现最好的10只股票，并定期（例如每日）根据最新数据更新持仓，以保持投资组合的“最优”状态。

## 策略实现步骤分解

![](img/e052a18ca152baba536bde7fb810a28f_5.png)

上一节我们介绍了策略的整体目标，本节中我们来看看具体的实现步骤。整个策略的实现将分为三个主要模块：初始化、盘前处理和盘中交易。

### 第一步：创建策略框架

首先，我们需要在交易平台中创建一个新的策略。

![](img/e052a18ca152baba536bde7fb810a28f_1.png)

在平台界面中找到“新建策略”按钮并点击。在弹出的对话框中，为策略命名，例如 `simple_demo`。策略类型选择“股票”。确认后，平台会生成一个可编辑的代码界面。

![](img/e052a18ca152baba536bde7fb810a28f_3.png)

生成的代码框架通常包含几个关键的函数模块，这些是需要我们完成的部分：
*   `initialize`：初始化函数，在策略启动时运行一次。
*   `before_trading`：盘前处理函数，在每个交易日开始前运行。
*   `handle_bar`：盘中处理函数，在每个交易时段（如每分钟或每日）运行。

![](img/e052a18ca152baba536bde7fb810a28f_5.png)

### 第二步：明确策略逻辑与任务

我们的策略逻辑是：从沪深300指数的300只成分股中，定期（例如每日）根据某个选股指标（如公司盈利数据）进行排序，选出排名前10的股票作为当期持仓。

以下是策略运行的核心循环：
1.  **获取股票池**：在初始化阶段，获取沪深300指数的所有成分股代码。
2.  **盘前选股**：在每个交易日开始前，获取股票池中所有股票的财务数据（例如盈利），并按照指标值进行降序排序，选出前10只股票。
3.  **盘中调仓**：在交易时段，对比当前实际持仓与盘前选出的“最优10只股票”列表。
    *   如果持仓股票不在新列表中，则卖出。
    *   如果新列表中的股票尚未持有，则买入。
    *   目标是使实际持仓与“最优列表”完全一致。

### 第三步：模块功能规划

基于以上逻辑，我们可以规划各个函数模块的具体任务。

**在 `initialize` 初始化函数中**，我们需要完成基础设置。平台默认生成的例子可能只针对单只股票，我们需要修改为获取整个沪深300指数成分股。

```python
# 示例代码结构
def initialize(context):
    # 获取沪深300指数成分股，并设置为策略的股票池
    stock_pool = get_index_stocks('000300.SH')
    set_universe(stock_pool)
```

**在 `before_trading` 盘前处理函数中**，我们需要执行每日的数据查询和处理工作，这些操作与实际交易指令无关。

以下是 `before_trading` 函数需要完成的任务列表：
*   从股票池中查询所有股票的财务数据（例如 `income` 盈利）。
*   对查询到的盈利数据进行降序排序。
*   选取盈利最高的前10只股票，将其代码列表保存到一个全局变量中，供 `handle_bar` 函数使用。

**在 `handle_bar` 盘中处理函数中**，我们将执行实际的交易逻辑。假设初始资金为10万元。

以下是 `handle_bar` 函数的核心处理流程：
1.  **获取当前持仓**：查询账户当前持有的所有股票。
2.  **对比持仓与目标**：将当前持仓列表与 `before_trading` 中计算出的“最优10只股票”列表进行对比。
3.  **执行调仓**：
    *   卖出当前持有但不在新目标列表中的股票。
    *   买入在新目标列表中但尚未持有的股票。
    *   目标是使持仓股票与目标列表完全一致。

## 总结

![](img/e052a18ca152baba536bde7fb810a28f_15.png)

本节课中，我们一起学习了如何为一个简单的量化交易策略进行任务分析。我们明确了策略的目标——持续持有沪深300中表现最佳的10只股票，并将实现过程分解为 **初始化**、**盘前选股** 和 **盘中调仓** 三个清晰的步骤。每个步骤对应交易平台API的一个函数模块，为后续的代码编写打下了坚实的基础。虽然逻辑描述起来有些复杂，但核心思想非常简单：定期筛选并换入更好的股票。
