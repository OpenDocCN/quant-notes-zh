## 什么是金融？
金融的定义是对现有资源进行重新整合之后，实现价值和利润的等效流通。这个定义可能听起来抽象，但通俗理解，金融涉及资金的流动和配置。它并非完全是不劳而获的投机行为。金融活动对整个经济体系和个人都有积极作用。

例如，一位拥有闲置资金的投资者，可以将资金投入一个有发展潜力的创业公司。投资者通过购买该公司的股票提供资金，公司则利用这笔资金发展业务。几年后公司若成功上市，价值增长，投资者获得回报，公司也得以壮大。这个过程实现了双赢，促进了经济发展，让资金流向需要它的地方。这就是金融活动的一个典型例子。

## 常见的金融工具
在日常生活中，我们更常接触到的是具体的金融工具，即可在金融市场中交易的金融资产。以下是几种主要的金融工具：

![](img/9a40fa74231ba1d964cd96528e0cb91d_3.png)

以下是几种主要的金融工具介绍：

*   **股票**：代表对一家公司的所有权份额。购买股票即成为该公司的股东，可分享公司利润（分红）并承担经营风险。它是本课程后续讨论的核心。
*   **期货**：一种标准化合约，约定在未来某一特定时间和地点，以约定价格交割一定数量的某种资产（如大宗商品）。其特点是**高风险、高收益**。交易双方基于对未来价格走势的不同判断进行交易，旨在规避风险或进行投机。
*   **黄金**：一种传统的避险资产和保值工具。其价格相对稳定，波动通常小于股票。价格主要受全球供应量（如金矿开采）、货币政策（如货币增发）和市场需求影响。
*   **外汇**：指不同国家货币之间的兑换交易，例如人民币与美元的汇率。汇率会因两国经济状况、利率政策等因素波动。大型金融机构常利用汇率波动进行套利，因其波动幅度对个人投资者而言通常较小。
*   **基金**：由基金公司汇集众多投资者的资金，交由专业的基金经理进行统一管理和投资（可投资于股票、债券等多种资产）。个人投资者通过购买基金份额，间接参与投资。其特点是**风险与收益通常低于直接投资股票**，适合缺乏专业知识和时间的投资者。

## 重点：理解股票
上一节我们介绍了多种金融工具，本节中我们重点来看看其中最核心的一种——股票。

股票是股份有限公司为筹集资金而发行给股东的所有权凭证。每股股票代表股东对企业拥有一个基本单位的所有权。购买股票意味着成为公司的部分所有者。

其核心逻辑可以用一个简单的公式表示：
**股东权益 = 公司总资产 - 公司总负债**

作为股东，你的收益主要来自两个方面：
1.  **资本利得**：以较低价格买入股票，再以较高价格卖出，赚取差价。
2.  **股息分红**：公司盈利后，将部分利润按持股比例分配给股东。

同时，你也需要承担相应的风险，即公司经营不善导致股价下跌或破产的可能性。
