# 金融量化分析：P19：夏普比率的作用 📈

在本节课中，我们将要学习一个在金融投资中至关重要的评估指标——夏普比率。我们将了解它的定义、计算方式以及它在实际选股策略中的应用。

## 概述

夏普比率用于衡量一项投资在承担单位风险时，所能获得的超额收益。简单来说，它帮助我们判断“冒这个风险去获取收益，到底值不值得”。这个指标越高，意味着投资组合的风险调整后收益越好。

## 夏普比率的定义与意义

上一节我们介绍了投资中的风险概念，本节中我们来看看如何量化风险与收益的关系。

夏普比率描述的是：对于每承担一单位的风险，你能获得多少超额回报。这类似于一个权衡：高收益往往伴随着高风险，但我们需要知道这个“高风险”换来的“高收益”是否划算。

![](img/03d4359bfa1e38d0be166b246cb51caa_1.png)

![](img/03d4359bfa1e38d0be166b246cb51caa_2.png)

举个例子，一份日薪极高但身处战区的雇佣兵工作，其高薪对应的是极高的生命风险。夏普比率就是用来评估这类“风险溢价”是否合理的工具。在选股时，我们计算各只股票的夏普比率，数值更高的股票，意味着其单位风险带来的收益更高，通常是更优的选择。

其核心公式如下：

**夏普比率 = (投资组合预期收益率 - 无风险收益率) / 投资组合收益率的标准差**

![](img/03d4359bfa1e38d0be166b246cb51caa_4.png)

## 夏普比率的计算逻辑

理解了夏普比率的意义后，我们来看看它的具体计算逻辑。计算主要涉及三个部分：

1.  **投资组合的收益率**：你的投资（或单只股票）在一段时间内的回报。
2.  **无风险收益率**：理论上没有任何风险的投资所能获得的收益率，例如国债、银行存款的固定利率。这是一个基准。
3.  **投资组合收益率的标准差**：用于衡量投资回报的波动性，即风险大小。波动越大，风险越高。

计算步骤是：首先用投资组合收益率减去无风险收益率，得到“超额收益”；然后将这个超额收益除以投资组合的风险（标准差），结果就是夏普比率。

## 实战计算：用Python计算夏普比率

理论部分已经清晰，现在让我们进入实战环节，使用Python来计算一组股票数据的夏普比率。

首先，我们需要准备数据。在计算收益率之前，通常需要处理数据中的缺失值（例如因停牌导致的数据空缺）。一个常见的做法是用前一天的数值进行填充。

```python
# 假设 df 是包含股票每日收盘价的数据框
# 计算每日收益率
returns = df.pct_change()
# 填充缺失值（用前一个有效值填充）
returns = returns.fillna(method='ffill')
```

接下来，我们设定无风险收益率。这里假设年化无风险收益率为5%。

```python
risk_free_rate = 0.05
```

![](img/03d4359bfa1e38d0be166b246cb51caa_6.png)

然后，我们按照公式计算年化夏普比率。需要注意的是，我们通常将日收益率转换为年化收益率进行计算。

```python
import numpy as np

# 计算年化夏普比率
# 假设一年有250个交易日
trading_days_per_year = 250

![](img/03d4359bfa1e38d0be166b246cb51caa_8.png)

# 计算超额收益的平均值（年化）
excess_return = returns.mean() * trading_days_per_year - risk_free_rate
# 计算收益率的标准差（年化）
volatility = returns.std() * np.sqrt(trading_days_per_year)

# 计算夏普比率
sharpe_ratio = excess_return / volatility
```

以下是计算过程的关键点列表：
*   **年化处理**：由于收益率数据通常是日度的，需要乘以年交易天数（如250）将其年化。标准差年化则需要乘以年交易天数的平方根。
*   **结果解读**：计算出的 `sharpe_ratio` 是一个序列，对应每一只股票（或每一列数据）的夏普比率。

## 结果分析与应用

计算完成后，我们得到了一个关键的结果。查看结果，我们可能会发现类似以下的输出：

```
股票A    0.85
股票B    1.32
股票C   -0.15
股票D    0.45
```

在这个假设的结果中：
*   **股票B**的夏普比率最高（1.32），意味着在承担相同风险的情况下，它能提供最高的超额收益，是最优选择。
*   **股票A**和**股票D**的夏普比率为正，但低于股票B。
*   **股票C**的夏普比率为负，意味着其收益甚至未能覆盖无风险收益，是应该避免的选择。

因此，在构建投资组合时，我们会优先选择夏普比率更高的资产，目标是使整个组合的夏普比率最大化。
