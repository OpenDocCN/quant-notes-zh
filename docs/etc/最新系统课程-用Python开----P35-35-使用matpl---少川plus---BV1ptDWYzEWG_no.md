## 计算机如何绘制曲线
上一节我们介绍了matplotlib的基本绘图功能，本节中我们来看看如何用它绘制数学函数图像。计算机屏幕上显示的图像，本质上都是由一个个像素点构成的。所谓的“线”或“曲线”，实际上是由一系列非常密集的点连接而成。例如，在绘图软件中画一个圆，放大后会发现它是由许多小点组成的。绘制数学函数图像也是同样的道理：我们需要生成一系列密集的X坐标点，然后根据函数公式计算出对应的Y坐标点，最后将这些点连接起来，就形成了函数图像。

## 生成密集的数据点
以下是生成密集数据点的核心方法。我们将使用NumPy库中的`linspace`函数来创建一个均匀分布的数组。

![](img/097636dc7c65a1114f0dfc8d3bdd27ca_3.png)

```python
import numpy as np
# 生成从-100到100之间，包含10000个点的数组
x = np.linspace(-100, 100, 10000)
```
`linspace`函数的第一个参数是起点，第二个参数是终点，第三个参数是要生成的点的总数。这与`arange`函数不同，`arange`指定的是步长，而`linspace`指定的是点的数量。通过生成足够多的点（例如10000个），我们可以得到一条看起来非常平滑的曲线。

## 计算函数值并绘图
有了X坐标数组后，我们就可以根据数学公式计算Y坐标值。NumPy数组支持向量化运算，这意味着我们可以直接对整个数组进行数学运算，而无需使用循环。

![](img/097636dc7c65a1114f0dfc8d3bdd27ca_5.png)

![](img/097636dc7c65a1114f0dfc8d3bdd27ca_7.png)

以下是计算三个函数值并绘图的步骤：
1.  **定义函数**：根据公式 `y = f(x)` 计算Y值。
2.  **绘制图像**：使用`plt.plot`函数分别绘制三条曲线。
3.  **添加样式**：为每条曲线设置不同的颜色和标签，以便区分。
4.  **显示图例**：调用`plt.legend()`显示图例。
5.  **调整视图**：如果曲线值域相差过大，可以使用`plt.ylim`调整Y轴显示范围，以便更好地观察所有曲线。

```python
import matplotlib.pyplot as plt
import numpy as np

# 生成X坐标点
x = np.linspace(-100, 100, 10000)

# 计算三个函数的Y值
y1 = x                     # y = x
y2 = x ** 2                # y = x^2
y3 = 3 * (x ** 3) + 5 * (x ** 2) + 2 * x + 1  # y = 3x^3 + 5x^2 + 2x + 1

# 绘制三条曲线
plt.plot(x, y1, color='red', label='y = x')
plt.plot(x, y2, color='green', label='y = x^2')
plt.plot(x, y3, color='blue', label='y = 3x^3 + 5x^2 + 2x + 1')

# 添加图例
plt.legend()

# 显示图形
plt.show()
```

## 图像显示与坐标轴调整
在绘制过程中，可能会遇到因为不同函数值域（Y值范围）相差巨大，导致某些曲线在图中显示不明显的问题。例如，三次函数 `y = x^3` 的增长速度远快于线性函数 `y = x`。为了解决这个问题，我们可以手动设置Y轴的显示范围。

```python
# 在调用 plt.show() 之前，设置Y轴显示范围
plt.ylim(-1000, 1000)
```
通过调整`ylim`的参数，我们可以将视图聚焦在感兴趣的区间，从而使所有曲线都能清晰可见。同样，也可以使用`plt.xlim`来调整X轴的显示范围。

## 总结
本节课中我们一起学习了使用matplotlib和NumPy绘制数学函数图像的方法。核心步骤包括：使用`np.linspace`生成密集的X点数组，利用NumPy的向量化运算高效计算Y值，最后用`plt.plot`绘制并美化图像。当曲线值域差异较大时，可以使用`plt.ylim`或`plt.xlim`来调整坐标轴范围。这种方法在数学实验、数据分析和科学计算中都非常常用。
