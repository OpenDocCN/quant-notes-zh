## 概述
在本节课中，我们将学习图表的基础知识。图表是数据可视化的重要工具，能够将枯燥的数据转化为直观、易于理解的图形，从而更有效地传达分析结果。我们将从图表的意义讲起，并学习如何创建和设置几种最基础的图表类型。

## 为什么需要图表？🤔

![](img/9744305cb731c87818a840fab06b1c7c_9.png)

在开始学习具体操作之前，我们先理解学习图表的意义。数据分析不仅是对数字进行处理，更重要的是将分析结果有效地传达给他人。

![](img/9744305cb731c87818a840fab06b1c7c_11.png)

试想一下，如果你想向别人描述一只猫有多么可爱，使用语言描述（如“毛软软的，特别活泼”）可能效果有限。但如果直接展示一张这只猫的照片，其视觉冲击力会立刻让人感受到它的可爱，并产生想养的欲望。

![](img/9744305cb731c87818a840fab06b1c7c_13.png)

![](img/9744305cb731c87818a840fab06b1c7c_15.png)

图表的作用与此类似。面对一堆原始数据，人们可能难以快速抓住重点。例如，下面这张表格：

![](img/9744305cb731c87818a840fab06b1c7c_9.png)

仅看数字，我们很难直观地比较各城市的营收情况。但如果将这些数据转化为柱状图：

![](img/9744305cb731c87818a840fab06b1c7c_11.png)

我们就能一眼看出“广州地区的营收情况较好”。图表用更直观、更通俗易懂的形式呈现分析结果，使汇报内容更具吸引力，从而达到更好的沟通效果。这就是我们学习图表的核心意义。

## 认识图表的基本结构 🏗️

上一节我们了解了图表的重要性，本节中我们来看看一个图表由哪些部分组成。掌握图表结构是后续进行自定义设置的基础。

![](img/9744305cb731c87818a840fab06b1c7c_17.png)

![](img/9744305cb731c87818a840fab06b1c7c_19.png)

![](img/9744305cb731c87818a840fab06b1c7c_21.png)

我们以最简单的柱状图为例。首先选择“超一线城市营收”数据中2018年的数据，点击【插入】->【推荐的图表】->【柱状图】即可创建。

创建完成后，我们来认识图表的各个组成部分：

![](img/9744305cb731c87818a840fab06b1c7c_23.png)

*   **图表标题**：位于图表顶部，用于说明图表的中心思想或数据代表的意义，例如“2018年超一线城市营收”。
*   **绘图区**：图表的核心区域，包含数据图形（如柱状）和网格线。
*   **数据系列**：绘图区中代表具体数据的图形，例如每个城市的柱子就是一个系列。
*   **数据标签**：可显示在系列图形上，用于标注该系列的具体数值。
*   **坐标轴**：
    *   **水平轴（分类轴）**：通常显示分类项目，如城市名称。
    *   **垂直轴（数值轴）**：通常显示数值刻度。
*   **图例**：用于说明图表中不同颜色或图案所代表的数据系列。

![](img/9744305cb731c87818a840fab06b1c7c_25.png)

了解这些部分有什么用呢？当我们想对图表的特定部分进行格式调整时，需要先选中它。选中不同部分后，右键菜单或侧边栏会出现对应的格式设置选项。

例如：
*   选中**绘图区**，右键选择“设置绘图区格式”，可以设置填充颜色和边框。
*   选中**图表标题**，则会出现“设置图表标题格式”的选项，可以设置文本填充、阴影和对齐方式。
*   选中**坐标轴**，可以设置坐标轴的刻度、标签等格式。
*   选中**数据系列**或**数据标签**，也有其对应的格式设置。

这种“选中即对应”的交互方式，是我们自定义图表美观度和清晰度的关键。

![](img/9744305cb731c87818a840fab06b1c7c_27.png)

## 创建与设置柱状图 📊

![](img/9744305cb731c87818a840fab06b1c7c_29.png)

![](img/9744305cb731c87818a840fab06b1c7c_31.png)

认识了图表结构后，本节我们来学习如何创建和设置最常用的图表之一——柱状图。

![](img/9744305cb731c87818a840fab06b1c7c_33.png)

柱状图主要用于在**垂直方向**上比较不同类别的数据大小。创建方法很简单：选中数据区域，点击【插入】->【推荐的图表】->【柱状图】即可。

![](img/9744305cb731c87818a840fab06b1c7c_35.png)

**如何同时展示两年数据进行对比？**
如果数据源中包含2018和2019两年数据，创建图表时会自动用不同颜色的柱子代表不同年份，形成对比。这展示了柱状图的另一个用途：比较同一类别（如北京）在不同时间点的数据。

**选择数据源的两种方法：**
1.  **拖动选择**：创建图表后，直接拖动数据区域右下角来扩展或缩小数据范围。这是较新版本Excel提供的便捷功能。
2.  **“选择数据”对话框**：右键点击图表，选择“选择数据”。在弹出的对话框中，可以精确地添加、编辑或删除“图例项（系列）”和“水平轴标签”。例如，可以删除2018年系列，再通过“添加”按钮重新引入。

![](img/9744305cb731c87818a840fab06b1c7c_37.png)

**常用美化操作：**
*   **删除网格线**：点击选中网格线，按 `Delete` 键删除，可使图表看起来更简洁。
*   **添加数据标签**：点击图表旁的 `+` 号，勾选“数据标签”，即可在每个柱子上显示具体数值。

![](img/9744305cb731c87818a840fab06b1c7c_39.png)

## 创建与设置条形图 ↔️

上一节我们介绍了垂直方向的柱状图，本节中我们来看看它的“近亲”——条形图。

![](img/9744305cb731c87818a840fab06b1c7c_41.png)

条形图与柱状图原理相同，只是将柱子**水平放置**。它是在**水平方向**上比较不同类别的数据。创建方法类似：选中数据，点击【插入】->【推荐的图表】->【条形图】。

**何时使用条形图？**
当分类项目的名称较长时，使用柱状图可能导致水平轴标签显示不全或拥挤。此时，使用条形图将类别置于垂直轴，让名称有充足空间显示，图表会更具可读性和美观性。

![](img/9744305cb731c87818a840fab06b1c7c_43.png)

![](img/9744305cb731c87818a840fab06b1c7c_45.png)

**设置条形图：**
其设置选项与柱状图基本一致。同样可以删除网格线、添加数据标签。数据标签的位置可以选择“居中”、“轴内侧”、“数据标签外”等，通常“数据标签外”是较清晰的选择。

## 创建与设置折线图 📈

前面两节我们学习了用于比较的柱状图和条形图，本节我们学习用于观察趋势的折线图。

折线图用于显示**一段时间内**数据的变化趋势或波动情况。它的水平轴通常是时间，垂直轴是数值。

**创建单个系列的折线图：**
选择某首歌（如“爱的就是你”）各月的播放量数据，插入折线图。图表会显示这首歌热度随时间上升或下降的趋势。

![](img/9744305cb731c87818a840fab06b1c7c_47.png)

![](img/9744305cb731c87818a840fab06b1c7c_49.png)

**创建多个系列的折线图进行对比：**
选择多首歌各月的数据，插入折线图。图表会同时绘制多条折线，便于我们对比不同歌曲在同一时期的热度走势。例如，可以清晰看到“小苹果”在某个时间点后热度显著下降。

![](img/9744305cb731c87818a840fab06b1c7c_51.png)

![](img/9744305cb731c87818a840fab06b1c7c_53.png)

![](img/9744305cb731c87818a840fab06b1c7c_55.png)

**折线图的设置与美化：**
*   同样可以删除网格线以使图表更简洁。
*   对于折线图，添加数据标签可能会使图表显得杂乱，通常观察整体趋势更为重要，因此可以不添加。
*   在【设计】选项卡中可以选择不同的图表样式，如改为深色背景，这取决于个人审美和汇报场景的需求。

## 创建与设置散点图 🔘

![](img/9744305cb731c87818a840fab06b1c7c_57.png)

最后，我们来学习用于分析数据关系的散点图。

散点图由一系列点组成，用于展示两个**数值变量**（二维数据）之间的潜在关系或相关性。例如，分析“身高”和“体重”之间的关系。

![](img/9744305cb731c87818a840fab06b1c7c_59.png)

![](img/9744305cb731c87818a840fab06b1c7c_61.png)

**创建散点图：**
选中身高和体重两列数据，点击【插入】->【推荐的图表】->【散点图】。图表中每个点代表一个个体，其横坐标是体重，纵坐标是身高。

**添加趋势线：**
为了更清晰地判断关系，可以添加趋势线。点击图表旁的 `+` 号，勾选“趋势线”。如果点大致沿一条斜线分布，则说明两者存在一定的线性相关关系。

![](img/9744305cb731c87818a840fab06b1c7c_63.png)

**计算相关系数：**
要量化相关性的强弱，可以计算相关系数。首先需要启用“数据分析”功能：
1.  点击【文件】->【选项】->【加载项】。
2.  在“管理”下拉框中选择“Excel加载项”，点击【转到】。
3.  勾选“分析工具库”，点击【确定】。
启用后，在【数据】选项卡会出现“数据分析”按钮。
4.  点击【数据分析】，选择“相关系数”，点击【确定】。
5.  在对话框中，选择身高和体重的数据区域作为“输入区域”，指定一个输出位置，点击【确定】即可得到相关系数。结果越接近1或-1，相关性越强。

**突出显示特定数据点：**
如果想强调图表中的某个异常点：
1.  单击一次选中所有数据点，再单击一次即可单独选中目标数据点。
2.  右键选择“设置数据点格式”。
3.  在“标记”选项中，选择“内置”并更改点的类型和大小。
4.  在“填充”中，选择“纯色填充”并更换颜色。
这样，该点就会在图表中突出显示。

**在图表中添加说明：**
为了进一步解释，可以在图表中插入文本框或形状。
*   **插入文本框**：点击【插入】->【文本框】，在图表上绘制并输入文字（如“异常点”）。
*   **插入形状**：点击【插入】->【形状】，可以选择箭头等形状来引导视线或进行标注。
这些元素能帮助观众更好地理解图表的重点。

## 总结 🎯

![](img/9744305cb731c87818a840fab06b1c7c_67.png)

本节课中我们一起学习了数据可视化中图表的基础知识。
1.  我们首先理解了**图表的意义**在于将数据直观化，提升沟通效果。
2.  然后，我们认识了图表的**基本结构**，包括标题、绘图区、坐标轴、图例等，并知道如何选中各部分进行格式设置。
3.  接着，我们动手创建了四种基础图表：
    *   **柱状图**：用于垂直比较不同类别的数据。
    *   **条形图**：用于水平比较不同类别的数据，尤其适合类别名称较长时。
    *   **折线图**：用于展示数据随时间的变化趋势。
    *   **散点图**：用于分析两个数值变量之间的相关性，并学习了添加趋势线和计算相关系数的方法。
4.  最后，我们还掌握了如何通过设置格式、添加文本框和形状来美化和增强图表的表达能力。
