## 数组与标量/数组的运算

![](img/a78eb34739c47dc8a2ba8379f5de0890_10.png)

![](img/a78eb34739c47dc8a2ba8379f5de0890_12.png)

NumPy数组支持高效的批量运算，这比手动编写循环要快得多。

![](img/a78eb34739c47dc8a2ba8379f5de0890_14.png)

![](img/a78eb34739c47dc8a2ba8379f5de0890_16.png)

**与标量运算**：标量即单个数值。数组可以与标量进行逐元素的数学运算。

![](img/a78eb34739c47dc8a2ba8379f5de0890_17.png)

![](img/a78eb34739c47dc8a2ba8379f5de0890_19.png)

```python
import numpy as np
A = np.arange(10)  # 创建数组 [0, 1, 2, ..., 9]
print(A + 1)   # 每个元素加1
print(A * 3)   # 每个元素乘以3
print(3 / A)   # 用3除以每个元素（注意：除以0会产生警告，结果为正无穷inf）
```

![](img/a78eb34739c47dc8a2ba8379f5de0890_21.png)

![](img/a78eb34739c47dc8a2ba8379f5de0890_23.png)

**与数组运算**：两个形状相同的数组可以进行逐元素的加减乘除和比较运算。

![](img/a78eb34739c47dc8a2ba8379f5de0890_25.png)

![](img/a78eb34739c47dc8a2ba8379f5de0890_27.png)

```python
B = np.arange(10, 20)  # 创建数组 [10, 11, 12, ..., 19]
print(A + B)  # 对应位置相加
print(A > B)  # 比较对应位置，返回布尔值数组
```

## 一维数组的索引与切片

![](img/a78eb34739c47dc8a2ba8379f5de0890_31.png)

![](img/a78eb34739c47dc8a2ba8379f5de0890_33.png)

索引用于获取单个元素，切片用于获取一个子数组。

**索引**：通过位置下标获取元素，下标从0开始。
```python
print(A[0])  # 获取第一个元素
print(A[-1]) # 获取最后一个元素
```

**切片**：通过`[start:stop:step]`的格式获取一个片段。规则是“前闭后开”。
```python
print(A[0:4])   # 获取索引0到3的元素：[0, 1, 2, 3]
print(A[:5])    # 获取前5个元素
print(A[5:])    # 获取从索引5开始到最后的元素
print(A[::2])   # 每隔一个取一个元素
```

![](img/a78eb34739c47dc8a2ba8379f5de0890_35.png)

![](img/a78eb34739c47dc8a2ba8379f5de0890_37.png)

**重要区别：视图与副本**
NumPy数组的切片返回的是原数组数据的**视图**（view），而非副本（copy）。这意味着修改切片会直接影响原数组。这是为了在处理大数据时节省内存和时间。

```python
C = A[0:4]  # C是A的一个切片视图
C[0] = 100  # 修改C的第一个元素
print(A[0]) # 输出100，A的第一个元素也被修改了！
```

如果希望获得一个独立的副本，需要使用`copy()`方法：
```python
D = A[0:4].copy()
D[0] = 999
print(A[0]) # 输出100，原数组A不受影响
```

## 二维数组的索引与切片

对于二维数组（矩阵），我们需要通过行和列两个维度来定位元素。

**索引**：有两种等价的写法。
```python
# 创建一个3行5列的二维数组
arr_2d = np.arange(15).reshape(3, 5)
# 获取第0行第0列的元素
print(arr_2d[0][0])
print(arr_2d[0, 0])  # 更推荐这种逗号分隔的写法
# 获取第2行第3列的元素（值为13）
print(arr_2d[2, 3])
```

**切片**：使用逗号分隔行和列的切片规则。格式为`[行切片, 列切片]`。
- 逗号**左边**控制**行**的选取。
- 逗号**右边**控制**列**的选取。

以下是具体示例：
```python
# 获取前两行、前三列的子矩阵（即元素 0,1,2,5,6,7）
print(arr_2d[0:2, 0:3])

# 获取第1行到最后一行、第2列到最后一列的子矩阵（即元素 7,8,9,12,13,14）
print(arr_2d[1:, 2:])

# 获取所有行、第1列的元素（返回一个一维数组）
print(arr_2d[:, 1])
```

同样，二维数组的切片也是视图操作。如果需要独立副本，记得使用`.copy()`。

![](img/a78eb34739c47dc8a2ba8379f5de0890_39.png)

本节课中我们一起学习了NumPy数组的索引与切片。核心要点包括：
1.  数组支持高效的批量运算。
2.  一维数组的索引切片与列表类似，但切片结果是原数组的**视图**。
3.  二维数组使用`[行, 列]`的格式进行索引和切片，逗号左边切行，右边切列。
4.  使用`.copy()`方法可以创建数据的独立副本，避免意外修改原数据。
