## 打分方法介绍

上一节我们介绍了如何获取和预处理因子数据。本节中，我们来看看如何为这些因子进行打分。打分是量化选股中的关键步骤，其核心思想是将不同量纲和方向的指标，转化为统一的分数，以便进行综合比较。

以下是两种常见的打分方法：

1.  **区间划分法**：将每个因子的数值范围划分为若干区间，并为每个区间赋予一个分数。
2.  **排序打分法**：直接根据因子值在所有股票中的排序位置来赋予分数。

## 区间划分法示例

我们通过一个具体的例子来理解区间划分法。假设我们有300只股票（ID1 到 ID300），每只股票有四个因子（A, B, C, D）。我们知道：
*   因子 **A** 和 **B** 是 **越大越好**。
*   因子 **C** 和 **D** 是 **越小越好**。

假设所有因子值都已归一化到 [0, 1] 区间。我们首先为每个因子划分区间并设定分数规则。

以下是因子A（越大越好）的评分规则：

| 数值区间 | 得分 |
| :--- | :--- |
| [0.8, 1.0] | 5 |
| [0.6, 0.8) | 4 |
| [0.4, 0.6) | 3 |
| [0.2, 0.4) | 2 |
| [0.0, 0.2) | 1 |

以下是因子C（越小越好）的评分规则：

| 数值区间 | 得分 |
| :--- | :--- |
| [0.0, 0.2) | 5 |
| [0.2, 0.4) | 4 |
| [0.4, 0.6) | 3 |
| [0.6, 0.8) | 2 |
| [0.8, 1.0] | 1 |

现在，我们为两只示例股票进行打分。

以下是股票 ID1 的因子值和对应得分：

*   **A = 0.71** -> 落在 [0.6, 0.8) 区间 -> 得分：**4**
*   **B = 0.28** -> 落在 [0.2, 0.4) 区间 -> 得分：**2**
*   **C = 0.39** -> 落在 [0.2, 0.4) 区间 -> 得分：**4** (因子C越小越好)
*   **D = 0.11** -> 落在 [0.0, 0.2) 区间 -> 得分：**5** (因子D越小越好)
*   **总分** = 4 + 2 + 4 + 5 = **15**

![](img/96103efe22cc6fdb7588dd0daab4621b_6.png)

以下是股票 ID2 的因子值和对应得分：

*   **A = 0.45** -> 落在 [0.4, 0.6) 区间 -> 得分：**3**
*   **B = 0.17** -> 落在 [0.0, 0.2) 区间 -> 得分：**1**
*   **C = 0.81** -> 落在 [0.8, 1.0] 区间 -> 得分：**1** (因子C越小越好)
*   **D = 0.02** -> 落在 [0.0, 0.2) 区间 -> 得分：**5** (因子D越小越好)
*   **总分** = 3 + 1 + 1 + 5 = **10**

对所有300只股票完成上述打分后，我们得到每只股票的总分。接下来，只需根据总分进行排序，选择排名最高的前10只股票，即可作为下一次调仓的目标。

## 整体策略流程梳理

![](img/96103efe22cc6fdb7588dd0daab4621b_10.png)

基于上述打分法，我们可以梳理出一个完整的月度调仓策略流程。以下是该策略的核心步骤：

1.  **确定股票池**：首先，我们需要确定选股的范围。本例中，我们使用 **沪深300指数** 成分股作为初始股票池。
2.  **设置调仓周期**：设定策略的调仓频率。通常，月度或季度调仓较为常见。我们将设置一个定时器，每月触发一次调仓函数。
3.  **实现调仓函数**：这是策略的核心。在每次调仓时，执行以下操作：
    *   **数据获取与预处理**：获取股票池中所有股票的指定因子数据（如A, B, C, D），并进行必要的清洗和归一化处理。
    *   **因子打分**：根据预设的规则（如区间划分法），为每只股票的每个因子进行打分。
    *   **计算总分**：将每只股票的所有因子得分相加，得到该股票的综合总分。
    *   **排序与选股**：将所有股票按总分从高到低排序，选取排名前N（例如前10）的股票。
    *   **调仓操作**：卖出当前持仓中不在新选股列表中的股票，买入新选股列表中尚未持有的股票，使投资组合与选股列表一致。
