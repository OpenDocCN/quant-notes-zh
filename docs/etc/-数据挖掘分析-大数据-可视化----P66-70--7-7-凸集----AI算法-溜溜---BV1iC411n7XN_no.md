## 课程回顾

上一节我们介绍了一些典型的凸集，包括球、椭球、多面体和单纯形。这些集合结构相对简单，易于识别。



![](img/f830936e5483bc9004a5a71e847118d1_19.png)

![](img/f830936e5483bc9004a5a71e847118d1_21.png)

为了巩固理解，我们考虑一个集合：`{x | x <= 0}`。这个集合是凸集吗？是的，因为它是一个多面体，由线性不等式构成。它也是一个单纯形吗？根据定义，单纯形是有限个仿射无关点的凸包。在一维空间中，我们可以选择点 `x0 = 0` 和 `x1` 为一个足够小的负数，它们的凸包就是这个集合，因此它也是一个单纯形。当有疑问时，回归定义进行推导是可靠的方法。



![](img/f830936e5483bc9004a5a71e847118d1_23.png)

![](img/f830936e5483bc9004a5a71e847118d1_25.png)

上节课我们还介绍了对称矩阵集合，包括对称正定矩阵和对称半正定矩阵。这里需要更正一个口误：正定矩阵是指所有特征值都大于零的方阵，而不是奇异值。感谢提出指正的同学。



![](img/f830936e5483bc9004a5a71e847118d1_27.png)

![](img/f830936e5483bc9004a5a71e847118d1_29.png)

如果优化变量是矩阵，我们会涉及许多矩阵操作。对此有疑问的同学，强烈建议查阅教材附录中关于矩阵运算的基础知识。

## 保凸运算简介

![](img/f830936e5483bc9004a5a71e847118d1_48.png)

![](img/f830936e5483bc9004a5a71e847118d1_50.png)

![](img/f830936e5483bc9004a5a71e847118d1_54.png)

本节的核心内容是：**我们可以通过哪些变换，使得一个凸集在变换之后仍然是一个凸集**。掌握这些保凸运算是分析凸优化问题的有效工具。

![](img/f830936e5483bc9004a5a71e847118d1_59.png)

## 凸集的交集

![](img/f830936e5483bc9004a5a71e847118d1_61.png)

![](img/f830936e5483bc9004a5a71e847118d1_62.png)

![](img/f830936e5483bc9004a5a71e847118d1_63.png)

![](img/f830936e5483bc9004a5a71e847118d1_64.png)

![](img/f830936e5483bc9004a5a71e847118d1_65.png)

第一个保凸运算非常简单：**如果两个集合都是凸集，那么它们的交集也是凸集**。



![](img/f830936e5483bc9004a5a71e847118d1_67.png)

![](img/f830936e5483bc9004a5a71e847118d1_69.png)

这个结论很容易理解。考虑两个凸集 `S1` 和 `S2`，以及它们交集 `S1 ∩ S2` 中的任意两点。根据凸集定义，这两点间的线段既属于 `S1` 也属于 `S2`，因此必然属于它们的交集。



![](img/f830936e5483bc9004a5a71e847118d1_71.png)

![](img/f830936e5483bc9004a5a71e847118d1_73.png)

![](img/f830936e5483bc9004a5a71e847118d1_74.png)

![](img/f830936e5483bc9004a5a71e847118d1_75.png)

![](img/f830936e5483bc9004a5a71e847118d1_76.png)

这个结论可以推广到任意多个凸集的情况：**任意一族凸集的交集仍然是凸集**。



![](img/f830936e5483bc9004a5a71e847118d1_77.png)

![](img/f830936e5483bc9004a5a71e847118d1_78.png)

![](img/f830936e5483bc9004a5a71e847118d1_79.png)

![](img/f830936e5483bc9004a5a71e847118d1_80.png)

![](img/f830936e5483bc9004a5a71e847118d1_82.png)

用公式表达如下：设 `A` 是一个索引集，对于每个 `α ∈ A`，`S_α` 是一个凸集，那么它们的交集 `∩_{α∈A} S_α` 也是凸集。



![](img/f830936e5483bc9004a5a71e847118d1_84.png)

![](img/f830936e5483bc9004a5a71e847118d1_86.png)

![](img/f830936e5483bc9004a5a71e847118d1_88.png)

![](img/f830936e5483bc9004a5a71e847118d1_90.png)

**注意**：凸集的并集不一定是凸集。可以很容易地构造两个凸集，使得连接它们中某两点的线段并不完全落在并集中。

![](img/f830936e5483bc9004a5a71e847118d1_92.png)

![](img/f830936e5483bc9004a5a71e847118d1_94.png)

![](img/f830936e5483bc9004a5a71e847118d1_96.png)

## 仿射函数

![](img/f830936e5483bc9004a5a71e847118d1_100.png)

![](img/f830936e5483bc9004a5a71e847118d1_102.png)

接下来我们介绍一个重要的保凸变换：**仿射函数**。



![](img/f830936e5483bc9004a5a71e847118d1_104.png)

![](img/f830936e5483bc9004a5a71e847118d1_106.png)

![](img/f830936e5483bc9004a5a71e847118d1_107.png)

![](img/f830936e5483bc9004a5a71e847118d1_109.png)

仿射函数是一种线性映射。具体来说，如果一个函数 `f: R^n -> R^m` 可以写成 `f(x) = Ax + b` 的形式，其中 `A` 是一个 `m×n` 的矩阵，`b` 是一个 `m` 维向量，那么这个函数就是仿射函数。



![](img/f830936e5483bc9004a5a71e847118d1_111.png)

![](img/f830936e5483bc9004a5a71e847118d1_112.png)

![](img/f830936e5483bc9004a5a71e847118d1_113.png)

![](img/f830936e5483bc9004a5a71e847118d1_114.png)

![](img/f830936e5483bc9004a5a71e847118d1_115.png)

![](img/f830936e5483bc9004a5a71e847118d1_117.png)

关于凸集和仿射函数的核心结论是：**如果集合 `S ⊆ R^n` 是凸集，那么它经过仿射函数 `f` 映射后的像 `f(S) = {f(x) | x ∈ S}` 也是凸集**。



![](img/f830936e5483bc9004a5a71e847118d1_119.png)

这个结论直观上不难理解。仿射变换包括缩放、旋转、平移等，这些操作不会改变一个物体的“凸性”。例如，将一个足球（凸集）拉伸成一个橄榄球，它仍然是凸集。



![](img/f830936e5483bc9004a5a71e847118d1_121.png)

![](img/f830936e5483bc9004a5a71e847118d1_123.png)

![](img/f830936e5483bc9004a5a71e847118d1_125.png)

我们可以从定义出发严格证明：在 `f(S)` 中任取两点 `y1 = f(x1)`, `y2 = f(x2)`，对于任意 `θ ∈ [0,1]`，有：
`θ y1 + (1-θ) y2 = θ(Ax1+b) + (1-θ)(Ax2+b) = A(θ x1 + (1-θ) x2) + b = f(θ x1 + (1-θ) x2)`
由于 `S` 是凸集，`θ x1 + (1-θ) x2 ∈ S`，因此 `f(θ x1 + (1-θ) x2) ∈ f(S)`，证毕。

![](img/f830936e5483bc9004a5a71e847118d1_127.png)

![](img/f830936e5483bc9004a5a71e847118d1_129.png)

![](img/f830936e5483bc9004a5a71e847118d1_130.png)

![](img/f830936e5483bc9004a5a71e847118d1_131.png)

## 仿射函数的逆映射

![](img/f830936e5483bc9004a5a71e847118d1_133.png)

![](img/f830936e5483bc9004a5a71e847118d1_134.png)

![](img/f830936e5483bc9004a5a71e847118d1_135.png)

![](img/f830936e5483bc9004a5a71e847118d1_137.png)

仿射函数的逆映射也能保持凸性。



![](img/f830936e5483bc9004a5a71e847118d1_139.png)

![](img/f830936e5483bc9004a5a71e847118d1_141.png)

设 `g: R^k -> R^n` 是一个仿射函数，`S ⊆ R^n` 是一个凸集。那么，原像集合 `g^{-1}(S) = {x | g(x) ∈ S}` 也是凸集。



![](img/f830936e5483bc9004a5a71e847118d1_142.png)

![](img/f830936e5483bc9004a5a71e847118d1_143.png)

![](img/f830936e5483bc9004a5a71e847118d1_144.png)

![](img/f830936e5483bc9004a5a71e847118d1_145.png)

这意味着，如果一个集合在像空间中是凸的，那么所有能映射到该集合的原始点构成的集合，在原像空间中也是凸的。正映射和逆映射都是保凸的。

![](img/f830936e5483bc9004a5a71e847118d1_147.png)

![](img/f830936e5483bc9004a5a71e847118d1_149.png)

## 仿射映射的例子

![](img/f830936e5483bc9004a5a71e847118d1_151.png)

![](img/f830936e5483bc9004a5a71e847118d1_153.png)

以下是几个常见的、属于仿射映射的保凸运算：



![](img/f830936e5483bc9004a5a71e847118d1_155.png)

![](img/f830936e5483bc9004a5a71e847118d1_157.png)

**缩放与平移**：
*   **缩放**：如果 `S` 是凸集，`α` 是一个标量，则缩放后的集合 `αS = {αx | x ∈ S}` 也是凸集。
*   **平移**：如果 `S` 是凸集，向量 `a`，则平移后的集合 `S + a = {x + a | x ∈ S}` 也是凸集。



![](img/f830936e5483bc9004a5a71e847118d1_161.png)

![](img/f830936e5483bc9004a5a71e847118d1_162.png)

![](img/f830936e5483bc9004a5a71e847118d1_163.png)

![](img/f830936e5483bc9004a5a71e847118d1_164.png)

![](img/f830936e5483bc9004a5a71e847118d1_166.png)

这两种操作都是仿射映射的特例，不会改变集合的凸性。

## 凸集的和

另一个重要的保凸运算是：**两个凸集的和仍然是凸集**。



这里“和”的定义是：`S1 + S2 = {x + y | x ∈ S1, y ∈ S2}`。注意，这不是并集，而是将两个集合中所有元素两两相加得到的新集合。



![](img/f830936e5483bc9004a5a71e847118d1_168.png)

如何用仿射映射的观点理解呢？我们可以构造一个新的集合 `S = S1 × S2`，即所有形如 `(x, y)` 的点构成的集合，其中 `x ∈ S1`, `y ∈ S2`。如果 `S1` 和 `S2` 是凸的，那么 `S` 也是凸的。



![](img/f830936e5483bc9004a5a71e847118d1_170.png)

![](img/f830936e5483bc9004a5a71e847118d1_172.png)

然后，我们定义一个仿射函数 `f(x, y) = x + y`。这个函数将 `S` 映射到 `S1 + S2`。由于 `f` 是仿射映射且 `S` 是凸集，根据前面的结论，`S1 + S2 = f(S)` 也是凸集。



![](img/f830936e5483bc9004a5a71e847118d1_174.png)

**思考**：一个凸集与一个非凸集的和，有可能是凸集吗？答案是可能的，这取决于具体集合。

## 线性矩阵不等式（LMI）

![](img/f830936e5483bc9004a5a71e847118d1_182.png)

![](img/f830936e5483bc9004a5a71e847118d1_183.png)

![](img/f830936e5483bc9004a5a71e847118d1_184.png)

![](img/f830936e5483bc9004a5a71e847118d1_185.png)

在控制理论中非常重要的一个概念是**线性矩阵不等式（LMI）**，它的解集是凸集。



![](img/f830936e5483bc9004a5a71e847118d1_187.png)

![](img/f830936e5483bc9004a5a71e847118d1_189.png)

![](img/f830936e5483bc9004a5a71e847118d1_191.png)

一个LMI具有如下形式：
`A(x) = x1*A1 + x2*A2 + ... + xn*An ≼ B`
其中 `Ai` 和 `B` 是对称矩阵，`x` 是变量向量，符号 `≼` 表示矩阵不等式，即 `B - A(x)` 是一个半正定矩阵。



![](img/f830936e5483bc9004a5a71e847118d1_193.png)

![](img/f830936e5483bc9004a5a71e847118d1_194.png)

![](img/f830936e5483bc9004a5a71e847118d1_196.png)

**命题**：满足上述LMI的所有 `x` 构成的集合 `{x | A(x) ≼ B}` 是一个凸集。



![](img/f830936e5483bc9004a5a71e847118d1_198.png)

**证明思路**：
1.  定义仿射函数 `f(x) = B - A(x)`。这是一个从变量 `x` 所在空间到对称矩阵空间的映射。
2.  我们已知对称半正定矩阵的集合是一个凸锥（是凸集）。
3.  原集合 `{x | A(x) ≼ B} = {x | f(x) ≽ 0}` 正是这个半正定矩阵凸集通过仿射函数 `f` 的**逆映射**得到的原像。
4.  根据“仿射函数的逆映射保凸”的结论，该解集是凸集。



![](img/f830936e5483bc9004a5a71e847118d1_200.png)

![](img/f830936e5483bc9004a5a71e847118d1_201.png)

![](img/f830936e5483bc9004a5a71e847118d1_202.png)

![](img/f830936e5483bc9004a5a71e847118d1_203.png)

如果觉得矩阵空间抽象，可以尝试一个简化理解：将矩阵退化为标量（1x1矩阵），那么LMI就变成了普通的线性不等式，其解集显然是一个凸集（半空间）。

![](img/f830936e5483bc9004a5a71e847118d1_205.png)

![](img/f830936e5483bc9004a5a71e847118d1_207.png)

## 椭球是球的仿射像

![](img/f830936e5483bc9004a5a71e847118d1_209.png)

最后我们看一个例子：**任何椭球都可以看作是一个单位球经过仿射变换得到的**。



回忆椭球的定义：`ε = {x | (x - xc)^T P^{-1} (x - xc) ≤ 1}`，其中 `P` 是对称正定矩阵，`xc` 是中心。



考虑单位球：`B = {u | ||u||_2 ≤ 1}`。
定义仿射变换：`f(u) = P^{1/2} u + xc`，其中 `P^{1/2}` 是满足 `(P^{1/2})^2 = P` 的矩阵（即P的平方根）。



![](img/f830936e5483bc9004a5a71e847118d1_211.png)

![](img/f830936e5483bc9004a5a71e847118d1_213.png)

![](img/f830936e5483bc9004a5a71e847118d1_215.png)

那么，将单位球 `B` 中的所有点通过 `f` 进行变换，得到的集合 `f(B)` 恰好就是椭球 `ε`。这直观地说明了仿射变换的保凸性：球是凸集，变换后的椭球也是凸集。

![](img/f830936e5483bc9004a5a71e847118d1_217.png)

![](img/f830936e5483bc9004a5a71e847118d1_218.png)

## 总结

![](img/f830936e5483bc9004a5a71e847118d1_222.png)

本节课我们一起学习了保持凸集性质的几种基本运算：
1.  **任意多个凸集的交集**仍是凸集。
2.  **仿射函数**（及其逆映射）将凸集映射为凸集。缩放、平移、求和等都是其特例。
3.  **线性矩阵不等式（LMI）的解集**是凸集，这可以通过仿射映射的逆映射性质来证明。



掌握这些保凸运算，使我们能够识别和构造复杂的凸集，这是理解和建立凸优化模型的基础。下一节我们将继续探讨其他的保凸运算。
