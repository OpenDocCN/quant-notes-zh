## 🔍 误差项与参数求解目标
上一节我们介绍了误差项服从高斯分布。本节中我们来看看如何利用这个性质来求解模型参数。

我们知道数据 `X` 和标签 `Y` 都是已知的。现在缺少的是模型的参数 `θ`。我们的目标是找到最合适的参数 `θ`。

既然误差项 `ε` 服从均值为0的高斯分布，我们可以写出其概率密度函数：
```
P(ε) = (1 / √(2πσ²)) * exp(-ε² / (2σ²))
```
由于均值 `μ` 为0，公式中省略了 `(ε - μ)` 项，直接写作 `ε²`。

这个分布是关于误差项 `ε` 的。但我们最终关心的不是具体的误差值，而是参数 `θ`。因此，我们需要将表达式转换成与 `θ` 相关。

根据线性回归定义，`Y = θX + ε`，可以推导出 `ε = Y - θX`。将这个关系代入高斯分布公式，我们就得到了一个关于 `θ` 和已知数据 `(X, Y)` 的表达式。

通俗地解释：我们寻找参数 `θ`，使得 `θ` 与 `X` 组合后得到的结果，与真实值 `Y` 越接近越好。换句话说，我们希望参数 `θ` 使得预测值成为真实值 `Y` 的**可能性**最大。

## 🤔 似然函数的作用与直观理解
为了解释似然函数的作用，我们举一个例子。

假设我在赌场观察别人赌博。赌场的输赢由一组内在的、固定的参数控制。我观察到：
*   第一个玩家赢了。
*   第二个玩家也赢了。
*   第三、四、五个玩家都赢了。

在控制输赢的参数不变的前提下，我观察到的所有样本（玩家）结果都是“赢”。那么我会认为，接下来我去玩的时候，赢的可能性也极高（接近100%）。

这说明了什么？**似然函数描述的就是这样一件事：什么样的参数，在与我们的数据组合后，最有可能得到我们观测到的结果。** 我们希望找到一组固定的参数 `θ`，它能最好地解释我们手中所有已知的数据。

现在，请注意似然函数中的符号 `Π`（累乘）。为什么是累乘？

1.  **数据量影响置信度**：如果只看到一个人赢，我可能觉得是侥幸。如果看到一万个人都赢，我就非常确信这组参数导致“赢”的结果。累乘 `i=1 to m` 意味着我们希望利用**大量数据**来找到最可靠的参数，数据越多，估计越准。
2.  **独立同分布假设**：我们之前假设数据是独立同分布的。在概率论中，**独立事件的联合概率等于各事件边缘概率的乘积**。因此，使用累乘在数学上是完全正确的。

## ➕ 从乘法问题到加法问题：引入对数
虽然似然函数的概念很清晰，但直接求解它是一个乘法问题，非常困难。想象一下，如果有1000个数据点，就要计算1000个概率值的乘积，这在计算上几乎是不可行的。

相比之下，**加法问题则容易处理得多**。那么，如何将乘法转换为加法呢？答案是使用**对数运算**。

在对数运算中有一个重要性质：**乘积的对数等于对数的和**。
```
log(a * b) = log(a) + log(b)
```
这样，我们就可以通过对数，将复杂的连乘问题转化为相对简单的连加问题。

但这里有一个关键问题：给函数加上对数，函数本身的值就改变了，这样求出的解还和原来一样吗？

我们需要明确优化目标：我们真正关心的是让似然函数取最大值的那个**参数点 `θ`**（即极值点），而不是似然函数的最大值具体是多少。加上对数函数（一个单调递增函数）会改变函数的极值（最大值大小），但**不会改变极值点的位置**。因此，优化对数似然函数和优化原始似然函数，在寻找最优参数 `θ` 这一点上是**等价的**。

通过这个转换，我们成功地将一个棘手的乘法优化问题，转化为了一个更容易处理的加法优化问题。

---

## 📝 构建对数似然函数
现在，我们来看如何具体构建用于求解的对数似然函数。

首先，回顾原始的似然函数是连乘形式：
`L(θ) = Π P(ε_i)`
其中 `ε_i = y_i - θ x_i`。

在其前面加上对数 `log`，利用对数性质，连乘就转化为了连加：
`log L(θ) = Σ log P(ε_i)`

接下来的步骤，就是将高斯分布的概率密度公式 `P(ε)` 代入 `log P(ε_i)` 中进行具体推导和化简，最终得到一个关于参数 `θ` 的表达式。这个表达式就是我们的**对数似然函数**，最大化它就能找到最优参数 `θ`。

## 🎯 总结
本节课中我们一起学习了似然函数的核心思想及其在线性回归参数估计中的应用。

![](img/73fc52221869e648f2f72635ab173c9f_6.png)

1.  **目标转换**：我们将求解参数 `θ` 的问题，转化为寻找能使预测值成为真实值的**可能性最大**的参数。
2.  **似然函数**：它表示在给定参数下，观察到当前所有数据的联合概率。我们通过**最大化似然函数**来估计参数。
3.  **对数变换**：由于直接优化连乘形式的似然函数计算困难，我们引入对数运算，将其转化为连加形式的**对数似然函数**。这种变换不改变最优参数点的位置，但大大简化了求解过程。
