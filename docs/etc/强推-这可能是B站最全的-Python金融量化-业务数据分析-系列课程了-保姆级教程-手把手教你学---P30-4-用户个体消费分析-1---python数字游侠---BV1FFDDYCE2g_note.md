# Python金融量化+业务数据分析：P30：4.用户个体消费分析

在本节课中，我们将学习如何对用户个体的消费数据进行分析。我们将计算每个用户的消费总金额和总次数，并通过散点图和直方图来可视化这些数据的分布情况，从而洞察用户消费行为。

## 计算用户消费总金额与总次数

上一节我们介绍了数据的基本结构，本节中我们来看看如何计算每个用户的消费总金额和总次数。

首先，我们需要计算每个用户的消费总金额。这需要基于用户ID进行分组，然后对消费金额列求和。

```python
# 计算每个用户的消费总金额
user_total_amount = df.groupby('user_id')['order_amount'].sum()
```

接下来，我们需要计算每个用户的消费总次数。消费次数可以通过统计每个用户ID在原始数据中出现的行数来获得。

```python
# 计算每个用户的消费总次数
user_total_count = df.groupby('user_id').size()
# 或者使用特定列（如订单日期）的计数
# user_total_count = df.groupby('user_id')['order_date'].count()
```

## 绘制消费金额与产品数量的散点图

在计算出每个用户的消费总金额和消费产品总数量后，我们可以通过散点图来观察两者之间的关系。

以下是绘制散点图的步骤：
1.  计算每个用户消费的产品总数量。
2.  使用散点图函数，将产品数量作为自变量（X轴），消费金额作为因变量（Y轴）。

```python
# 计算每个用户消费的产品总数量
user_total_products = df.groupby('user_id')['order_product'].sum()

import matplotlib.pyplot as plt
```

## 分析消费金额的分布（直方图）

为了更细致地了解用户消费金额的分布，特别是小额消费用户的情况，我们将绘制消费金额在1000元以内的用户分布直方图。

以下是实现步骤：
1.  计算每个用户的消费总金额。
2.  筛选出消费金额小于等于1000元的用户。
3.  对筛选后的消费金额数据绘制直方图。

```python
# 计算每个用户的消费总金额
user_amount_summary = df.groupby('user_id').sum()

# 筛选消费金额在1000元以内的用户
low_amount_users = user_amount_summary.query('order_amount <= 1000')

# 提取消费金额列并绘制直方图
low_amount_users['order_amount'].hist()
plt.xlabel('消费金额（≤1000元）')
plt.ylabel('用户数量')
plt.title('小额消费用户金额分布直方图')
plt.show()
```

从直方图中可以看到，第一个柱子（通常对应0-100元区间）最高。这说明在消费金额不超过1000元的用户中，消费额在100元以内的用户数量是最多的。

## 分析消费产品数量的分布（直方图）

类似地，我们也可以分析用户消费产品数量的分布情况，重点关注购买数量在100个以内的用户。

以下是具体步骤：
1.  计算每个用户的消费产品总数量。
2.  筛选出消费产品数量小于等于100个的用户。
3.  对筛选后的产品数量数据绘制直方图。

```python
# 计算每个用户的消费产品总数量（已在上文计算）
# user_total_products = df.groupby('user_id')['order_product'].sum()

# 筛选消费产品数量在100个以内的用户
low_product_users = user_total_products[user_total_products <= 100]

# 绘制直方图
low_product_users.hist()
plt.xlabel('消费产品数量（≤100个）')
plt.ylabel('用户数量')
plt.title('低购买量用户产品数量分布直方图')
plt.show()
```

该直方图显示，购买产品数量在0-10个之间的用户是最多的。这与消费金额的分布结论一致，都表明大部分用户属于低频、低额的消费群体。
