## 概述

最大回撤是衡量投资策略风险的关键指标，它描述了在特定时间段内，资产净值从最高点到后续最低点的最大跌幅。这个指标不评估策略的盈利能力，而是评估其可能面临的最大风险。

## 什么是最大回撤？

上一节我们介绍了策略评估的多种指标，本节中我们来看看最大回撤。最大回撤描述的是投资过程中“最惨”的阶段。在炒股过程中，价格必然有涨有跌。最惨的阶段，即价格持续下跌最严重的那个区间，就称为最大回撤区间。

它不能仅从字面理解，其核心是衡量资产缩水的严重程度。例如，股票价格从15元持续下跌至1元的过程，就是一个典型的最大回撤区间。这个指标帮助我们评估，即使策略最终盈利，中间过程可能承受多大的压力和风险。2008年股市从6000多点暴跌至1600多点，就是一次巨大的市场回撤，导致许多人承受了巨额亏损。

因此，最大回撤是一个风险指标，它告诉你策略在最糟糕的时候会让你“多惨”。

![](img/07829cc0addd3860f75b5c5f40252fa3_5.png)

## 最大回撤的计算公式

![](img/07829cc0addd3860f75b5c5f40252fa3_7.png)

理解了概念后，我们来看看如何量化计算最大回撤。其标准计算公式如下：

**最大回撤 = max( (Pi - Pj) / Pi )**

![](img/07829cc0addd3860f75b5c5f40252fa3_9.png)

![](img/07829cc0addd3860f75b5c5f40252fa3_11.png)

其中：
*   **Pi** 代表第 i 天的股价（或资产净值）。
*   **Pj** 代表第 i 天之后某一天（第 j 天）的股价。
*   我们需要遍历所有可能的 i 和 j（j > i），计算从 i 到 j 的跌幅，并找出其中的最大值。

![](img/07829cc0addd3860f75b5c5f40252fa3_13.png)

这个公式计算的是从某个高点（Pi）到其后某个低点（Pj）的相对缩水比例。

![](img/07829cc0addd3860f75b5c5f40252fa3_15.png)

![](img/07829cc0addd3860f75b5c5f40252fa3_17.png)

## 使用Python计算最大回撤

![](img/07829cc0addd3860f75b5c5f40252fa3_19.png)

![](img/07829cc0addd3860f75b5c5f40252fa3_21.png)

理论需要实践来巩固，接下来我们使用Python的Pandas库来演示如何计算最大回撤。我们将通过一个简单的示例来理解计算过程。

![](img/07829cc0addd3860f75b5c5f40252fa3_23.png)

![](img/07829cc0addd3860f75b5c5f40252fa3_25.png)

首先，我们需要理解一个Pandas的操作：`cummax()`。它与我们熟悉的`cumsum()`（累积和）类似，但计算的是累积最大值。

![](img/07829cc0addd3860f75b5c5f40252fa3_27.png)

![](img/07829cc0addd3860f75b5c5f40252fa3_29.png)

```python
import pandas as pd

![](img/07829cc0addd3860f75b5c5f40252fa3_31.png)

# 创建一个示例序列
s = pd.Series([2, 5, 3, 8, 4, 7])
print("原始序列:")
print(s)
print("\n累积最大值序列:")
print(s.cummax())
```
执行上述代码，你会发现`cummax()`会返回序列中到当前位置为止的最大值。例如，对于第三个元素3，其累积最大值是前三个数中的最大值5。

![](img/07829cc0addd3860f75b5c5f40252fa3_33.png)

![](img/07829cc0addd3860f75b5c5f40252fa3_35.png)

现在，我们将这个操作应用到金融时间序列数据上。假设`df[‘close’]`是我们的每日收盘价序列。

![](img/07829cc0addd3860f75b5c5f40252fa3_37.png)

计算最大回撤的步骤如下：
1.  计算序列的累积最大值序列。
2.  用累积最大值序列减去原始序列，得到每日相对于历史高点的回撤值。
3.  将这个回撤值除以对应的累积最大值（即公式中的Pi），得到每日的回撤比例。
4.  在所有回撤比例中找出最大值，即为最大回撤。

![](img/07829cc0addd3860f75b5c5f40252fa3_39.png)

以下是代码实现：

![](img/07829cc0addd3860f75b5c5f40252fa3_41.png)

```python
# 假设 df 是一个包含收盘价‘close’列的DataFrame
# 计算累积最大值
cum_max = df['close'].cummax()
# 计算每日回撤（累积最大值 - 当日收盘价）
drawdown = cum_max - df['close']
# 计算每日回撤比例
drawdown_pct = drawdown / cum_max
# 计算最大回撤
max_drawdown = drawdown_pct.max()
print(f"最大回撤为: {max_drawdown:.2%}")
```
这段代码的核心思想是：对于每一天，我们都找出到那天为止的历史最高价，然后计算当前价格从那个最高价下跌了多少比例。所有天数中，这个比例的最大值，就是整个时间段的最大回撤。

在实际的策略回测框架（如Backtrader、Zipline）中，这些指标通常会被自动计算并展示在分析图表中。图表中会用阴影区域标出最大回撤发生的具体区间，直观地展示策略的风险时段。
