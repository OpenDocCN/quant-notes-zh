## 因子数据简介

上一节我们介绍了课程的整体框架，本节中我们来看看什么是因子数据。在量化交易中，因子可以被理解为选股的标准或依据。例如，投资者在购买股票时，通常会依据某些想法或标准进行筛选，这些标准就是因子。

以下是几个常见的因子例子：
*   **市净率**：较低的市净率可能意味着股票被低估，未来上涨空间较大。
*   **营收增长率**：较高的营收增长率通常被认为是公司经营状况良好的信号。

简单来说，**因子**就是可能对最终投资结果（如收益率）产生影响的各项指标。我们的目标就是分析这些因子（如 X1, X2, X3...）如何影响最终结果（Y），并从中筛选出有效的因子来构建策略。

## 数据预处理三步走

![](img/584eef96bd1e260d8602c62b72a80755_1.png)

在直接使用因子数据进行建模前，我们需要对其进行预处理。本节将介绍一个标准的三步走流程。

以下是数据预处理的三个核心步骤：
1.  **去极值**：处理数据中的异常点或离群点。
2.  **标准化**：将不同量纲和取值范围的因子数据转换到统一的尺度上。
3.  **中性化**：消除因子数据中与某些系统性风险（如行业、市值）的相关性，这在金融因子分析中尤为重要。

接下来，我们将按照这个顺序，首先详细讲解如何去极值。

## 去极值方法：百分位法

![](img/584eef96bd1e260d8602c62b72a80755_3.png)

第一步是处理数据中的极值。极值，或称离群点，是那些明显偏离数据主体分布的值。直接删除这些数据可能造成信息损失，因此常见的做法是将其“拉回”到合理的边界内。

![](img/584eef96bd1e260d8602c62b72a80755_5.png)

![](img/584eef96bd1e260d8602c62b72a80755_7.png)

例如，假设我们设定数据的合理上界为5，而某个数据点的值为10。我们不直接删除10，而是将其修正为5，使其不再“越界”。

![](img/584eef96bd1e260d8602c62b72a80755_9.png)

有多种方法可以确定这个“边界”，我们先介绍第一种：**分位数法**。

![](img/584eef96bd1e260d8602c62b72a80755_11.png)

![](img/584eef96bd1e260d8602c62b72a80755_13.png)

### 分位数概念

![](img/584eef96bd1e260d8602c62b72a80755_15.png)

分位数是统计学中用于描述数据位置的概念。最广为人知的是**中位数**，它代表了数据集的中间值。与均值相比，中位数对极值不敏感，能更好地反映数据的集中趋势。

以下是几个关键的分位点：
*   **Q1（第一四分位数）**：将所有数据从小到大排列，位于25%位置的值。
*   **Q2（中位数）**：位于50%位置的值。
*   **Q3（第三四分位数）**：位于75%位置的值。

![](img/584eef96bd1e260d8602c62b72a80755_17.png)

### 百分位去极值操作

基于分位数，我们可以定义数据的正常范围。通常，我们会计算一个称为**四分位距（IQR）** 的指标：
`IQR = Q3 - Q1`

然后，设定数据的上下界：
*   **下界（Lower Bound）** = `Q1 - k * IQR`
*   **上界（Upper Bound）** = `Q3 + k * IQR`

其中，`k` 是一个常数，通常取1.5。任何小于下界或大于上界的数值，都会被修正为对应的边界值。

![](img/584eef96bd1e260d8602c62b72a80755_19.png)

以下是使用Python进行百分位去极值处理的示例代码框架：

```python
import pandas as pd
import numpy as np

# 假设 df[‘factor’] 是我们要处理的因子数据列
def winsorize_percentile(series, k=1.5):
    Q1 = series.quantile(0.25)
    Q3 = series.quantile(0.75)
    IQR = Q3 - Q1
    lower_bound = Q1 - k * IQR
    upper_bound = Q3 + k * IQR
    # 将超出边界的值拉回边界
    series_winsorized = series.clip(lower=lower_bound, upper=upper_bound)
    return series_winsorized

# 应用函数
# df[‘factor_winsorized’] = winsorize_percentile(df[‘factor’])
```
