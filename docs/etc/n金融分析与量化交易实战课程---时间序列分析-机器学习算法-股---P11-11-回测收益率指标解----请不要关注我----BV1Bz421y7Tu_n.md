## 策略评估指标概述

当我们完成一个交易策略后，必须在历史数据中进行测试，以判断该策略是盈利还是亏损。然而，仅用“赚”或“赔”来描述是不够的。我们需要更精确的指标来回答以下问题：策略具体赚了多少？与大盘相比有何优势？策略承担了哪些风险？这些风险是否值得？与其他策略相比有何优势？这些问题的答案就构成了策略的评估指标。

![](img/0564a5d9fa0c473b97c9f309304260ec_4.png)

后续课程中，我们将使用专业的策略回测平台自动生成这些指标。本节课的目标是让大家理解这些核心指标的含义和基本计算方法，以便能看懂回测报告。

## 回测收益率

![](img/0564a5d9fa0c473b97c9f309304260ec_10.png)

第一个也是最基本的指标是**回测收益率**。它衡量的是策略在整个回测周期内，相对于初始本金所获得的总回报。

![](img/0564a5d9fa0c473b97c9f309304260ec_12.png)

![](img/0564a5d9fa0c473b97c9f309304260ec_14.png)

其计算公式为：
`回测收益率 = (期末资产价值 / 期初资产价值) - 1`

![](img/0564a5d9fa0c473b97c9f309304260ec_16.png)

![](img/0564a5d9fa0c473b97c9f309304260ec_18.png)

其中，`期末资产价值`可以理解为策略结束时的总资产（包括现金和持仓市值），`期初资产价值`则是策略开始时的本金。这个指标直接反映了策略的最终盈利（或亏损）能力。

下面，我们通过代码来实际计算并可视化这一指标。

假设我们加载的数据包含了策略模拟交易后，每日的资产净值（或可用收盘价近似代替）。接下来，我们计算每日资产相对于期初的比值。

![](img/0564a5d9fa0c473b97c9f309304260ec_28.png)

![](img/0564a5d9fa0c473b97c9f309304260ec_30.png)

```python
# 计算每日资产相对于期初的比值
# 即公式中的 (P_t / P_0)
relative_value = data / data.iloc[0]
# 绘制资产相对价值变化曲线
relative_value.plot(figsize=(12, 6))
plt.title('资产相对价值走势')
plt.show()
```

![](img/0564a5d9fa0c473b97c9f309304260ec_32.png)

![](img/0564a5d9fa0c473b97c9f309304260ec_34.png)

这张图展示了从期初（基准为1）开始，资产价值随时间的变化情况。上升曲线代表盈利，下降代表亏损。

![](img/0564a5d9fa0c473b97c9f309304260ec_36.png)

![](img/0564a5d9fa0c473b97c9f309304260ec_38.png)

最后，我们计算整个回测周期的总收益率。

![](img/0564a5d9fa0c473b97c9f309304260ec_40.png)

![](img/0564a5d9fa0c473b97c9f309304260ec_42.png)

```python
# 计算总回测收益率
# 取最后一日的相对价值，然后减去1
total_return = relative_value.iloc[-1] - 1

![](img/0564a5d9fa0c473b97c9f309304260ec_44.png)

![](img/0564a5d9fa0c473b97c9f309304260ec_46.png)

# 将结果整理成DataFrame以便查看
result_df = pd.DataFrame(total_return.values, columns=['回测收益率'], index=total_return.index)

```
