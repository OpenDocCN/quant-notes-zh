## 回测收益率

![](img/336acda579cb639984818075756b31ea_11.png)

回测收益率是最基本、最直观的策略评估指标。它衡量的是，在回测周期内，策略最终获得的收益相对于初始本金的增长率。

![](img/336acda579cb639984818075756b31ea_13.png)

例如，你基于2015年至2020年的某只股票数据设计了一个买卖策略。回测收益率告诉你，执行这个策略到最终，你的本金增值了多少百分比。图中显示约为4%，这意味着多年操作下来，你的资金相对于本金增长了约4%。



![](img/336acda579cb639984818075756b31ea_15.png)

![](img/336acda579cb639984818075756b31ea_17.png)

![](img/336acda579cb639984818075756b31ea_23.png)

![](img/336acda579cb639984818075756b31ea_19.png)

![](img/336acda579cb639984818075756b31ea_21.png)

它的计算公式如下：

**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

其中：
*   **期末资产总值 (P_t)**：回测结束时的资产价值（卖出所有头寸后的总资金）。
*   **期初资产总值 (P_0)**：回测开始时的初始本金。

![](img/336acda579cb639984818075756b31ea_23.png)

这个公式的核心是计算资产净值的相对增长。它直接反映了策略在历史数据上的整体盈利能力。



![](img/336acda579cb639984818075756b31ea_27.png)

![](img/336acda579cb639984818075756b31ea_25.png)

接下来，我们通过Python代码来实际计算和可视化回测收益率的变化过程。首先，导入必要的工具包并加载数据。

![](img/336acda579cb639984818075756b31ea_27.png)

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体（示例）
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设第一列是日期
data = pd.read_csv('your_stock_data.csv', index_col=0, parse_dates=True)
print(data.head())
```



![](img/336acda579cb639984818075756b31ea_13.png)

数据包含了股票在不同时间点的价格信息。我们将基于这些价格数据模拟策略的资产净值曲线。



![](img/336acda579cb639984818075756b31ea_19.png)

我们的目标是计算每个时间点相对于起始点的资产比值，并最终得到回测收益率。以下是计算步骤：

1.  **计算累积净值曲线**：将每个时间点的价格与初始价格相除，得到资产相对初始本金的比值序列。这相当于将公式 `(P_t / P_0)` 应用于每一个时间点 `t`。
2.  **计算最终回测收益率**：取序列最后一个值（期末净值），然后减去1。

![](img/336acda579cb639984818075756b31ea_29.png)

![](img/336acda579cb639984818075756b31ea_31.png)

```python
# 1. 计算每个时间点相对于起始点的净值比
# 假设‘close’列代表我们策略的每日资产净值
cumulative_return_ratio = data['close'] / data['close'].iloc[0]

![](img/336acda579cb639984818075756b31ea_33.png)

# 可视化净值增长曲线
plt.figure(figsize=(10, 6))
cumulative_return_ratio.plot()
plt.title('策略累积净值增长曲线 (相对于初始本金)')
plt.ylabel('净值比')
plt.xlabel('日期')
plt.grid(True)
plt.show()
```



![](img/336acda579cb639984818075756b31ea_35.png)

![](img/336acda579cb639984818075756b31ea_31.png)

![](img/336acda579cb639984818075756b31ea_37.png)

上图展示了资产净值随时间的变化。比值大于1表示盈利，小于1表示亏损。

![](img/336acda579cb639984818075756b31ea_39.png)

```python
# 2. 计算整个回测周期的总收益率
final_return = cumulative_return_ratio.iloc[-1] - 1

![](img/336acda579cb639984818075756b31ea_41.png)

# 将结果整理成DataFrame以便查看
result_df = pd.DataFrame({
    ‘回测收益率’: [final_return]
}, index=[‘最终结果’])
print(result_df)
```

执行代码后，我们便得到了该策略在回测周期内的总收益率。这个数值就是评估平台报告中的“回测收益”。需要说明的是，本例中我们直接使用了股价数据模拟净值，实际策略的净值计算会涉及买卖点、仓位、手续费等更复杂的因素。
