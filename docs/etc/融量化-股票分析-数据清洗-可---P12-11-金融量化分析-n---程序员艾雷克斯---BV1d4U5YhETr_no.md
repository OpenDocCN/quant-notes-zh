### 创建全零数组

使用`np.zeros()`函数可以创建一个所有元素都为0的数组。默认情况下，数组的元素类型是浮点数（`float64`）。

**代码示例：**
```python
import numpy as np
a = np.zeros(10)
print(a)
print(a.dtype)  # 输出：float64
```

![](img/66ac23920344569770bc69d75c8cc7a8_7.png)

如果你希望创建整数类型的全零数组，可以通过`dtype`参数指定。

![](img/66ac23920344569770bc69d75c8cc7a8_9.png)

**代码示例：**
```python
a_int = np.zeros(10, dtype=int)
print(a_int.dtype)  # 输出：int64
```

### 创建空数组

![](img/66ac23920344569770bc69d75c8cc7a8_21.png)

![](img/66ac23920344569770bc69d75c8cc7a8_23.png)

`np.empty()`函数用于创建一个“空”数组。这里的“空”并不意味着数组元素是0或None，而是指分配了内存空间，但不会主动初始化其中的值。因此，数组中的值是内存中残留的、未定义的随机数据。

![](img/66ac23920344569770bc69d75c8cc7a8_25.png)

![](img/66ac23920344569770bc69d75c8cc7a8_27.png)

**代码示例：**
```python
c = np.empty(10)
print(c)  # 输出内容为随机值，例如：[2.12199579e-314 6.36598737e-314 ...]
```

**为什么使用`np.empty()`？**
创建数组需要先向计算机申请一块固定大小的内存。`np.zeros()`和`np.ones()`在申请内存后，会执行第二步操作：将每个位置的值分别设置为0或1。而`np.empty()`只执行第一步（申请内存），不执行初始化操作，因此速度更快。当你确定后续会覆盖数组中的所有值时，可以使用`np.empty()`来提升效率。如果不确定，则使用`np.zeros()`或`np.ones()`更安全。

## 创建序列数组

在数据处理中，经常需要生成一个具有规律数字序列的数组。NumPy提供了类似Python内置`range()`的函数。

### 使用`np.arange()`

![](img/66ac23920344569770bc69d75c8cc7a8_37.png)

`np.arange()`函数与`range()`用法相似，用于生成一个等差序列的数组。它支持整数和小数步长。

![](img/66ac23920344569770bc69d75c8cc7a8_39.png)

![](img/66ac23920344569770bc69d75c8cc7a8_41.png)

**代码示例：**
```python
# 生成0到99的整数序列
seq1 = np.arange(100)
print(seq1[:5])  # 输出：[0 1 2 3 4]

![](img/66ac23920344569770bc69d75c8cc7a8_43.png)

# 指定起始值、结束值和步长
seq2 = np.arange(10, 20, 0.5)
print(seq2)  # 输出：[10.  10.5 11.  11.5 ... 19.5]
```
注意：`np.arange(start, stop, step)`生成的序列包含`start`，不包含`stop`。

![](img/66ac23920344569770bc69d75c8cc7a8_45.png)

### 使用`np.linspace()`

![](img/66ac23920344569770bc69d75c8cc7a8_47.png)

`np.linspace()`函数用于在指定的区间内，生成指定数量的、等间隔的数值。它与`np.arange()`的关键区别在于：第三个参数不是步长（step），而是生成数组的元素个数（num）。

![](img/66ac23920344569770bc69d75c8cc7a8_49.png)

![](img/66ac23920344569770bc69d75c8cc7a8_51.png)

**代码示例：**
```python
# 在0到10之间生成5个等间隔的点
lin_seq = np.linspace(0, 10, 5)
print(lin_seq)  # 输出：[ 0.   2.5  5.   7.5 10. ]
```
注意：`np.linspace(start, stop, num)`默认包含结束值`stop`。生成的数组长度为`num`，相邻两个值的间隔为 `(stop - start) / (num - 1)`。

![](img/66ac23920344569770bc69d75c8cc7a8_53.png)

![](img/66ac23920344569770bc69d75c8cc7a8_55.png)

**应用场景：**
`np.linspace()`在科学计算和可视化中非常有用。例如，要绘制一个连续函数（如 y = x²）的图像，我们无法计算无穷多个点，但可以在一个区间（如[-10, 10]）内生成足够多（如10000个）的等间隔点来近似模拟。

**代码示例（概念演示）：**
```python
import matplotlib.pyplot as plt # 后续会学的绘图库
x = np.linspace(-10, 10, 10000)
y = x ** 2  # 计算y = x²
# plt.plot(x, y) # 用这10000个点画出平滑的抛物线
```

### 创建单位矩阵

![](img/66ac23920344569770bc69d75c8cc7a8_65.png)

![](img/66ac23920344569770bc69d75c8cc7a8_67.png)

`np.eye(N)`函数用于创建一个N×N的单位矩阵。单位矩阵是一个方阵，其主对角线上的元素为1，其余元素为0。

![](img/66ac23920344569770bc69d75c8cc7a8_69.png)

**代码示例：**
```python
identity_matrix = np.eye(3)
print(identity_matrix)
# 输出：
# [[1. 0. 0.]
#  [0. 1. 0.]
#  [0. 0. 1.]]
```
如果不涉及线性代数运算，这个函数使用频率较低。
