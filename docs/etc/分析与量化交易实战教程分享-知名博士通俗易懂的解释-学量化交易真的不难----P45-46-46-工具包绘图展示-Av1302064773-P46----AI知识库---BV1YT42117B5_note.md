# 量化交易完全可自学教程：P45：46. 工具包绘图展示

## 概述
在本节课中，我们将学习如何使用量化分析工具包中的绘图功能，来可视化因子的IC值（信息系数）及其时间序列。我们将重点关注如何解读IC值图表，并理解信息比率等辅助指标的含义。

上一节我们介绍了如何计算因子的IC值，本节中我们来看看如何将这些计算结果通过图表直观地展示出来。

## 绘制IC值时间序列图
计算完IC值后，我们需要对其进行可视化分析。IC值每天都会变化，仅看数字结果不够直观。我们可以使用名为`POLOATING`的工具包来绘图。

以下是绘制IC值时间序列图的步骤：
1.  导入绘图工具包。
2.  使用`plot`方法下的`ic_time_series`函数。
3.  将计算好的IC值序列（`SD`）作为参数传入并执行。

## 解读IC值图表
生成的图表包含以下关键信息：
*   **蓝色线**：代表每日实际的IC值走势，波动范围通常较大。
*   **绿色线**：默认以一个月为周期计算IC值的移动平均值，用于观察长期趋势。
*   **统计值**：图表会标注IC值的整体均值（`mean`）和标准差（`STD`），以及由 **`mean / STD`** 计算得到的信息比率。

观察图表时，我们主要关注**绿色移动平均线**。因为蓝色线波动剧烈，难以判断整体趋势。我们希望IC值越大越好，这表示因子与收益率的相关性越强。

从当前图表的绿色线来看，IC值的移动平均值较小且走势平稳，没有显示出明显的上升或下降趋势。这可能意味着该因子与未来收益率的相关性不强，不是我们理想中的有效因子。

## 理解信息比率
信息比率是一个辅助指标，其计算公式为：
**`信息比率 = IC均值 / IC标准差`**
它用于描述IC值序列的稳定性。
*   比值**越大**，说明IC值的标准差相对均值较小，序列越稳定。
*   比值**越小**，说明波动性较大，稳定性较差。

这个指标有助于我们判断因子表现的一致性，但在此阶段，我们只需简单理解其概念即可。

## 工具包的其他绘图功能
除了时间序列图，该工具包还提供了其他分析图表，例如不同计算周期的IC结果。

![](img/31e36b78298bdef65e8b873e99ea4e7f_7.png)

![](img/31e36b78298bdef65e8b873e99ea4e7f_5.png)

通过查阅API文档，我们还可以找到更多绘图函数，用于深入分析因子。以下是部分可用的可视化功能：
*   **直方图**：查看IC值的分布情况。
*   **QQ图**：检验IC值是否服从正态分布。
*   **热力图**：用颜色矩阵展示因子在不同分组下的表现。
