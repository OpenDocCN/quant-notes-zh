## 因子分析概述

![](img/f12d8b62b62df6df870b66d2f5b42504_3.png)

上一章我们介绍了量化策略的回测，本节中我们来看看量化策略的另一个核心：因子分析。

![](img/f12d8b62b62df6df870b66d2f5b42504_5.png)

因子分析是股票投资的常用思路，指的是利用**因子**（在机器学习中可能称为特征或指标），使各种股票收益最大化的投资方法。这些因素可能包括波动性、动量、股票规模等。

### 因子分析的原理

因子分析属于统计学范畴，其核心原理是**降维**思想。它与当前火热的深度学习方向相反，旨在将影响股价变化的多种可能因素，缩减为特定的几种。其主要方法是将相关性高的变量聚合在一起，找到这些特征指标，从而减少需要分析的维度，降低分析的复杂度，以便更合理、更容易地预测股价。

## 因子类别与流程

了解了因子分析的基本概念后，本节我们来看看因子有哪些类别，以及进行分析的一般流程。

![](img/f12d8b62b62df6df870b66d2f5b42504_7.png)

因子分析主要有两个类别：
1.  **量化选股因子**：这类因子多为财务类指标，例如营业利润率、销售净利率、营收环比增长率等。
2.  **量化择时因子**：这类因子主要是一些技术指标，例如均线（MA）、换手率、波动率、KDJ、MACD等。

实际的量化策略通常比较复杂，会同时结合选股因子和择时因子。

![](img/f12d8b62b62df6df870b66d2f5b42504_9.png)

以下是因子分析常见的四步流程：
1.  **因子构造/选择**：从众多指标中选择或构造出有效的因子。在应用层面，我们主要使用现有因子；当对市场有更深理解后，可以尝试创新构造新因子。
2.  **因子选股**：根据选定的因子筛选股票，构建股票池。
3.  **策略执行**：基于因子判断买入和卖出信号，进行交易。
4.  **策略回测与优化**：对策略进行回测，根据回测结果发现并优化因子或策略，循环迭代直至满意。

![](img/f12d8b62b62df6df870b66d2f5b42504_11.png)

## 常用基础因子

![](img/f12d8b62b62df6df870b66d2f5b42504_13.png)

在介绍了因子的类别和流程后，本节我们具体了解一些在量化平台上常用的基础因子。这些因子不局限于上述两大类。

以下是六种常用的基础因子：

**1. 价量因子**
通过量化平台的 `get_price()` 函数获取，主要包括开盘价、收盘价、最高价、最低价、成交量、成交金额等。

![](img/f12d8b62b62df6df870b66d2f5b42504_15.png)

**2. 财务数据因子**
指当日可见的最新单季财务指标，通常按季度更新。常见因子包括：
*   动态市盈率（PE Ratio）
*   换手率（Turnover Ratio）
*   市净率（PB Ratio）
*   股票总市值
*   股票流通市值

![](img/f12d8b62b62df6df870b66d2f5b42504_17.png)

**3. 行业因子**
表征股票所属的行业分类，是一个**哑变量**（即取值为0或1）。常见的行业分类标准包括证监会行业分类、申万行业分类以及各量化平台自有的分类。

**4. 概念因子**
表征股票所属的概念板块，同样是一个**哑变量**。常见概念板块包括智能电网、物联网、高端装备制造、一带一路、ChatGPT、5G等。如果股票属于某个概念，则返回1，否则返回0。

![](img/f12d8b62b62df6df870b66d2f5b42504_19.png)

**5. 指数因子**
表征股票是否为特定指数的成分股，也是一个**哑变量**。例如，可以判断股票是否属于上证指数、沪深300、中证500等。如果是，则返回1，否则返回0。

**6. 资金流因子**
通过 `get_money_flow()` 函数获取，反映资金流向情况。常用字段包括：
*   涨跌幅
*   主力净额、主力净占比
*   超大单净额、超大单净占比
*   大单净额、大单净占比
*   中单净额、中单净占比
*   小单净额、小单净占比

## 本章总结

本节课中我们一起学习了因子分析的概述。

我们首先介绍了**因子分析的定义**，它是指利用特定因子、特征或指标使股票投资收益最大化的方法，其核心原理是通过**降维**来简化分析。

接着，我们探讨了**因子的类别与流程**。因子主要分为量化选股因子（多为财务指标）和量化择时因子（多为技术指标）。分析流程通常包括因子构造、因子选股、策略执行和回测优化四个步骤。

最后，我们详细介绍了**六种常用的基础因子**：价量因子、财务数据因子、行业因子、概念因子、指数因子和资金流因子。其中，行业、概念和指数因子均为哑变量，用于表征股票的属性归属。

![](img/f12d8b62b62df6df870b66d2f5b42504_21.png)

掌握这些基础概念，是构建和优化量化交易策略的重要第一步。
