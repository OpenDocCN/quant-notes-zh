## 平均结果与连续增长率

上一节我们介绍了如何计算平均结果，本节中我们来看看如何计算连续时间的增长率。

增长率通常用于衡量相邻两个时间点之间的变化比例。其计算公式为：

**增长率 = (后一天价格 - 前一天价格) / 前一天价格**

然而，当我们尝试将多个时间段的增长率直接相加，以得到从起始点到终点的“总增长率”时，会遇到问题。

![](img/203f6083048c3d90a767765328bf441b_4.png)

## 直接累加增长率的问题

以下是直接累加增长率导致的问题示例。

假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

计算各阶段增长率：
*   第一天到第二天: (50 - 100) / 100 = -0.5
*   第二天到第三天: (75 - 50) / 50 = 0.5

![](img/203f6083048c3d90a767765328bf441b_6.png)

![](img/203f6083048c3d90a767765328bf441b_7.png)

若将增长率直接相加：-0.5 + 0.5 = 0。这个结果为0，似乎表示股票从第一天到第三天“没有涨跌”。但观察原始价格，从100元变为75元，实际上是下跌的。这说明**增长率不能直接进行加法运算**。

## 使用对数计算连续累积变化

![](img/203f6083048c3d90a767765328bf441b_9.png)

对于连续时间的序列，若想进行有意义的累加操作，需要借助对数运算。

核心思路是计算**对数收益率**，其公式为：

**对数收益率 = log(后一天价格 / 前一天价格)**

![](img/203f6083048c3d90a767765328bf441b_11.png)

这样做的好处在于，对数收益率的累加等价于价格比率的连乘。例如：

![](img/203f6083048c3d90a767765328bf441b_13.png)

log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)

![](img/203f6083048c3d90a767765328bf441b_15.png)

![](img/203f6083048c3d90a767765328bf441b_16.png)

最终结果 `log(75/100)` 直接反映了**最后一天价格与第一天价格的比例关系**。将所有时间段的**对数收益率**累加起来，再通过指数函数还原，就能得到从起始点到任意时间点的累积增长倍数。

![](img/203f6083048c3d90a767765328bf441b_17.png)

![](img/203f6083048c3d90a767765328bf441b_18.png)

![](img/203f6083048c3d90a767765328bf441b_19.png)

## 在Python中实现连续变化分析

以下是使用Pandas和NumPy实现上述分析的步骤。

![](img/203f6083048c3d90a767765328bf441b_21.png)

![](img/203f6083048c3d90a767765328bf441b_22.png)

![](img/203f6083048c3d90a767765328bf441b_23.png)

### 1. 计算对数收益率

首先，我们需要获取“前一天”的数据。可以使用Pandas的 `shift()` 函数将数据向上移动一格。

![](img/203f6083048c3d90a767765328bf441b_25.png)

```python
# 假设 data 是一个包含股价的Pandas Series或DataFrame
shifted_data = data.shift(1)  # 数据向上移动一格，得到“前一天”的数据
```

![](img/203f6083048c3d90a767765328bf441b_26.png)

![](img/203f6083048c3d90a767765328bf441b_27.png)

然后，计算对数收益率。

```python
import numpy as np

![](img/203f6083048c3d90a767765328bf441b_29.png)

# 计算对数收益率：log(当天价格 / 前一天价格)
log_returns = np.log(data / shifted_data)
```

![](img/203f6083048c3d90a767765328bf441b_31.png)

### 2. 计算累积收益并还原为价值

![](img/203f6083048c3d90a767765328bf441b_33.png)

接下来，我们对对数收益率进行累积求和，这代表了从起点到每个时间点的总对数收益。

![](img/203f6083048c3d90a767765328bf441b_34.png)

```python
# 累积求和
cumulative_log_returns = log_returns.cumsum()
```

最后，使用指数函数 `np.exp()` 将累积对数收益率还原为实际的累积增长倍数（即：最初的1单位投资，相当于当前时点的价值）。

![](img/203f6083048c3d90a767765328bf441b_36.png)

![](img/203f6083048c3d90a767765328bf441b_38.png)

```python
# 还原为实际价值倍数
cumulative_value = np.exp(cumulative_log_returns)
```

### 3. 可视化结果

![](img/203f6083048c3d90a767765328bf441b_40.png)

![](img/203f6083048c3d90a767765328bf441b_41.png)

我们可以绘制图表，直观展示各股票初始投资随时间增长的情况。

![](img/203f6083048c3d90a767765328bf441b_42.png)

```python
import matplotlib.pyplot as plt

cumulative_value.plot(figsize=(10, 6))
plt.title('初始1单位投资的累积价值变化')
plt.ylabel('累积价值倍数')
plt.xlabel('时间')
plt.show()
```

![](img/203f6083048c3d90a767765328bf441b_44.png)

![](img/203f6083048c3d90a767765328bf441b_45.png)

图表将显示，例如，在起始时间（如2010年）投资于各公司的1元钱，到结束时间（如2018年）分别变成了多少元。这清晰地反映了不同股票在时间序列上的整体表现。
