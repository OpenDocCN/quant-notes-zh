# 量化交易完全可自学教程：P27：百分位去极值方法

在本节课中，我们将学习如何对因子数据进行预处理。预处理是量化分析中至关重要的一步，它可以帮助我们清洗数据、消除异常值的影响，并使不同尺度的数据具有可比性，从而为后续的建模和分析打下坚实基础。

## 什么是因子

在开始学习预处理方法之前，我们首先需要理解什么是“因子”。在量化交易中，因子可以被理解为选择股票的标准或依据。

例如，在购买股票时，你可能会依据某些标准进行筛选。这些标准就是因子。一个常见的因子是市净率，较低的市净率可能意味着股票有更大的上涨空间。另一个例子是营收增长率，较高的增长率可能表明公司经营状况良好，其股票更具投资价值。

简单来说，因子就是那些可能对最终投资结果产生影响的指标。我们的目标就是分析这些因子，找出哪些对收益有显著影响，并据此构建投资策略。

## 因子数据预处理三步走

当我们拿到原始的因子数据后，通常不能直接用于建模。我们需要对其进行一系列预处理操作。本节将介绍一个标准的三步走预处理流程。

![](img/7b8fa749d07675049f6e11123a5bdc8d_1.png)

以下是因子数据预处理的三个核心步骤：
1.  **去极值**：处理数据中的异常值或离群点。
2.  **标准化**：将不同量纲和范围的因子数据转换到统一的尺度上。
3.  **中性化**：消除因子数据中与某些系统性风险（如行业、市值）的相关性。

前两步（去极值和标准化）在机器学习和数据挖掘领域非常常见。第三步（中性化）则是金融因子分析中特有的重要步骤。接下来，我们将按照这个顺序，详细讲解每一步的具体做法。

## 第一步：去极值处理

上一节我们介绍了预处理的三步走策略，本节中我们来看看第一步——去极值。数据中可能存在一些远离主体分布的极端值，这些极值可能会扭曲统计分析的结果。我们的目标不是简单地删除它们，而是将其“拉回”到合理的边界内。

一种常见且有效的方法是**百分位去极值法**。其核心思想是：我们不使用容易受极值影响的均值来设定边界，而是使用更能代表数据分布位置的分位数。

![](img/7b8fa749d07675049f6e11123a5bdc8d_3.png)

### 理解分位数

![](img/7b8fa749d07675049f6e11123a5bdc8d_5.png)

分位数是统计学中用于描述数据位置的概念。最熟悉的是**中位数**，它将数据分为两等份。与均值相比，中位数对极值不敏感，更能代表数据的“中心”位置。

![](img/7b8fa749d07675049f6e11123a5bdc8d_7.png)

除了中位数，常用的还有**四分位数**：
*   **Q1（第一四分位数）**：将所有数据从小到大排列，位于25%位置的值。
*   **Q2（第二四分位数）**：即中位数，位于50%位置的值。
*   **Q3（第三四分位数）**：位于75%位置的值。

![](img/7b8fa749d07675049f6e11123a5bdc8d_9.png)

### 百分位去极值法的操作

![](img/7b8fa749d07675049f6e11123a5bdc8d_11.png)

该方法利用四分位数来定义数据的正常范围。我们通常将超出一定百分位范围的值视为极值，并将其调整到边界上。

![](img/7b8fa749d07675049f6e11123a5bdc8d_13.png)

具体步骤如下：
1.  计算因子数据的**Q1**和**Q3**。
2.  计算**四分位距（IQR）**：`IQR = Q3 - Q1`。
3.  设定**下限（Lower Bound）**和**上限（Upper Bound）**：
    *   `下限 = Q1 - k * IQR`
    *   `上限 = Q3 + k * IQR`
    *   其中，`k`是一个常数，通常取1.5（温和）或3（严格）。
4.  遍历所有数据点，将任何小于下限的值设置为下限，将任何大于上限的值设置为上限。

![](img/7b8fa749d07675049f6e11123a5bdc8d_15.png)

以下是该方法的代码描述：
```python
import numpy as np

def percentile_winsorize(data, k=1.5):
    """
    使用百分位法对数据进行去极值处理。
    :param data: 输入的一维数据数组
    :param k: 范围乘数，默认1.5
    :return: 去极值处理后的数据
    """
    Q1 = np.percentile(data, 25)
    Q3 = np.percentile(data, 75)
    IQR = Q3 - Q1
    lower_bound = Q1 - k * IQR
    upper_bound = Q3 + k * IQR
    # 将极值拉回到边界
    data_winsorized = np.clip(data, lower_bound, upper_bound)
    return data_winsorized
```

### 代码示例演示

让我们通过一个简单的示例来演示这个过程。假设我们有一组代表某因子的模拟数据。

```python
# 导入必要的库
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 生成包含极值的示例数据
np.random.seed(42)
normal_data = np.random.randn(100) * 10 + 50  # 100个正态分布数据，均值50，标准差10
outliers = np.array([120, -30])  # 加入两个极端值
factor_data = np.concatenate([normal_data, outliers])

print(“原始数据描述：”)
print(pd.Series(factor_data).describe())
print(“\n原始数据前5个值：”, factor_data[:5])

![](img/7b8fa749d07675049f6e11123a5bdc8d_19.png)

# 应用百分位去极值函数
k = 1.5
Q1 = np.percentile(factor_data, 25)
Q3 = np.percentile(factor_data, 75)
IQR = Q3 - Q1
lower_bound = Q1 - k * IQR
upper_bound = Q3 + k * IQR

print(f“\nQ1: {Q1:.2f}, Q3: {Q3:.2f}”)
print(f“IQR: {IQR:.2f}”)
print(f“下限 (Q1 - {k}*IQR): {lower_bound:.2f}”)
print(f“上限 (Q3 + {k}*IQR): {upper_bound:.2f}”)

# 处理数据
processed_data = np.clip(factor_data, lower_bound, upper_bound)

print(“\n处理后的数据描述：”)
print(pd.Series(processed_data).describe())

# 可视化对比
fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].boxplot(factor_data)
axes[0].set_title(‘原始数据箱线图’)
axes[0].axhline(y=upper_bound, color=‘r’, linestyle=‘–‘, alpha=0.5, label=‘上限’)
axes[0].axhline(y=lower_bound, color=‘g’, linestyle=‘–‘, alpha=0.5, label=‘下限’)
axes[0].legend()

![](img/7b8fa749d07675049f6e11123a5bdc8d_21.png)

axes[1].boxplot(processed_data)
axes[1].set_title(‘去极值后数据箱线图’)
plt.tight_layout()
plt.show()
```

![](img/7b8fa749d07675049f6e11123a5bdc8d_23.png)

运行以上代码，你将看到原始数据中的极端值（120和-30）被拉回到了基于分位数计算出的上下限附近，数据的分布范围变得更为合理，且保留了大部分数据信息。
