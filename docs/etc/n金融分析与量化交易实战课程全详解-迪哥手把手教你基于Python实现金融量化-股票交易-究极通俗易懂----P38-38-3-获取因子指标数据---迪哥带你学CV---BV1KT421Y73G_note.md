在本节课中，我们将学习如何使用Python工具包获取金融因子指标数据。我们将从指定日期范围开始，然后查询特定因子（如市盈率）在所有A股股票中的数值，并最终获取和分析这些数据。

## 概述

上一节我们介绍了因子分析的基本概念和所需工具包。本节中，我们来看看如何具体获取指定日期范围内的因子数据。与回测框架不同，这里需要我们自己编写代码来获取多日数据。

## 第一步：选择日期数据

首先，我们需要选择一个日期范围，例如从2019年1月1日到2020年1月1日。这个日期列表将用于后续的数据查询。

以下是获取交易日期的代码：

```python
trading_dates = get_trading_dates('2019-01-01', '2020-01-01')
```

这段代码定义了一个函数，用于获取指定时间段内的所有交易日。每完成一步，我们都应该检查结果是否正确。

这段代码定义了一个查询对象`q`，它指定了要查询的因子（市盈率）并过滤出股票池中的股票。
