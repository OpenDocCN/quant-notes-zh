## 什么是最大回撤？📉

在策略分析报告中，我们经常会看到“最大回撤”这个指标。

![](img/a1d16e479542f5fab7b365e38e630f8a_2.png)

最大回撤描述的是投资过程中可能遭遇的最大亏损幅度。它不能仅从字面理解，其核心是衡量投资过程中的“惨烈程度”。

具体来说，在股票交易过程中，价格必然有涨有跌。最大回撤指的就是从某个高点开始，到后续某个低点为止，资产价值下跌最严重的那段区间。它反映了投资者在策略执行期间可能承受的最大账面亏损。

我们可以通过公式来更精确地理解这个概念。最大回撤的计算公式通常表示为：

**最大回撤 = max( (Pi - Pj) / Pi )**

![](img/a1d16e479542f5fab7b365e38e630f8a_4.png)

其中：
*   **Pi** 代表第 i 天的资产价格（或净值）。
*   **Pj** 代表第 i 天之后（j > i）的某一天资产价格。

这个公式的意思是：遍历所有可能的时间点 i 和其后的时间点 j，计算从 i 到 j 的跌幅百分比，然后取其中的最大值。这个最大值就是最大回撤。它描述的是一段时期内资产缩水的最大比例。

![](img/a1d16e479542f5fab7b365e38e630f8a_6.png)

因此，最大回撤是一个风险指标。它不直接评估策略的盈利能力，而是评估策略在执行过程中可能带来的最大风险。即使一个策略最终是盈利的，但其间可能经历一段非常痛苦的下跌过程，这段过程就是最大回撤区间。了解这个指标，有助于投资者评估自己能否承受这样的风险。

![](img/a1d16e479542f5fab7b365e38e630f8a_8.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_10.png)

例如，在2008年股市大跌期间，许多投资者经历了巨大的回撤，这正说明了风险控制的重要性。

![](img/a1d16e479542f5fab7b365e38e630f8a_12.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_14.png)

## 如何计算最大回撤？🧮

![](img/a1d16e479542f5fab7b365e38e630f8a_16.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_18.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_20.png)

在我们的策略回测报告中，系统会自动统计并标出最大回撤区间。

![](img/a1d16e479542f5fab7b365e38e630f8a_22.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_26.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_4.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_28.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_30.png)

现在，我们来手动实现计算最大回撤的逻辑。核心思路是：我们需要遍历所有可能的区间，计算其回撤值，然后找出最大值。

![](img/a1d16e479542f5fab7b365e38e630f8a_32.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_34.png)

Pandas库提供了高效的方法来进行此类计算。首先，我们介绍一个关键函数：`cummax()`。它与我们之前学过的`cumsum()`（累积和）类似，但计算的是累积最大值。

![](img/a1d16e479542f5fab7b365e38e630f8a_36.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_38.png)

以下是`cummax()`函数的一个简单示例：

![](img/a1d16e479542f5fab7b365e38e630f8a_40.png)

```python
import pandas as pd

![](img/a1d16e479542f5fab7b365e38e630f8a_42.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_44.png)

# 创建一个示例序列
s = pd.Series([2, 5, 3, 8, 1, 6])
print(s.cummax())
```
输出结果会是：`[2, 5, 5, 8, 8, 8]`。这表示：
*   对于第一个元素，之前最大值是2。
*   对于前两个元素，最大值是5。
*   对于前三个元素，最大值是5（因为5>3）。
*   以此类推，`cummax()`会记录到当前位置为止所遇到的最大值。

![](img/a1d16e479542f5fab7b365e38e630f8a_46.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_48.png)

理解了`cummax()`后，我们就可以用它来计算最大回撤。计算步骤如下：

![](img/a1d16e479542f5fab7b365e38e630f8a_50.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_52.png)

1.  假设我们有一个代表每日资产净值的序列 `price_series`。
2.  计算该序列的累积最大值序列：`cummax_series = price_series.cummax()`。这个序列的每个点都代表了“到当天为止的历史最高点”。
3.  计算每日的回撤值：`drawdown_series = (cummax_series - price_series) / cummax_series`。这个序列的每个值代表了“当前价格相对于历史最高点的回撤比例”。
4.  最后，从`drawdown_series`中找出最大值，即为最大回撤：`max_drawdown = drawdown_series.max()`。

![](img/a1d16e479542f5fab7b365e38e630f8a_54.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_56.png)

让我们结合一段模拟的股价数据来演示这个过程：

![](img/a1d16e479542f5fab7b365e38e630f8a_58.png)

```python
# 假设这是某股票一段时间内的收盘价
close_prices = pd.Series([100, 105, 102, 98, 96, 110, 108, 115, 112, 105])

![](img/a1d16e479542f5fab7b365e38e630f8a_60.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_62.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_64.png)

# 步骤1：计算累积最大值
cummax_prices = close_prices.cummax()
print(“累积最大值序列:”, cummax_prices.values)

![](img/a1d16e479542f5fab7b365e38e630f8a_66.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_68.png)

# 步骤2：计算每日回撤比例
daily_drawdown = (cummax_prices - close_prices) / cummax_prices
print(“每日回撤比例:”, daily_drawdown.values)

![](img/a1d16e479542f5fab7b365e38e630f8a_70.png)

![](img/a1d16e479542f5fab7b365e38e630f8a_72.png)

# 步骤3：计算最大回撤
max_drawdown_value = daily_drawdown.max()
print(“最大回撤为:”, max_drawdown_value)
```
通过这段代码，我们可以清晰地看到每一天相对于之前高点的回撤情况，并最终得到整个期间的最大回撤值。
