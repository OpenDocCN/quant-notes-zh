## 概述

![](img/0165400e7342484cec3a847966d99684_7.png)

![](img/0165400e7342484cec3a847966d99684_9.png)

移动平均线是金融分析中常用的技术指标，用于平滑股价波动并识别趋势。短期移动平均线对价格变化反应灵敏，而长期移动平均线则反映更长期的趋势。当短期均线从下方上穿长期均线时，形成“黄金交叉”，通常被视为买入信号；反之，当短期均线从上方下穿长期均线时，形成“死亡交叉”，通常被视为卖出信号。本节我们将通过代码实现这一分析过程。

---

## 计算短期移动平均线

![](img/0165400e7342484cec3a847966d99684_11.png)

首先，我们需要计算短期移动平均线。通常，短期窗口可以选择5天（代表一周的交易日），但根据数据的时间跨度，我们可以适当调整窗口大小。这里我们以苹果公司（AAPL）的股票数据为例，计算一个10天的短期移动平均线。

![](img/0165400e7342484cec3a847966d99684_13.png)

![](img/0165400e7342484cec3a847966d99684_15.png)

![](img/0165400e7342484cec3a847966d99684_17.png)

**代码实现：**
```python
# 假设df是包含苹果股价的DataFrame，列名为‘AAPL’
short_window = 10
df['M1'] = df['AAPL'].rolling(window=short_window).mean()
```
这段代码使用`rolling`方法创建一个滑动窗口，并用`mean`函数计算窗口内数据的平均值，结果存储在名为`M1`的新列中。

## 计算长期移动平均线

![](img/0165400e7342484cec3a847966d99684_23.png)

接下来，我们计算长期移动平均线。常见的长期窗口是20天或30天（代表约一个月的交易日）。我们继续使用苹果股票数据，计算一个30天的长期移动平均线。

![](img/0165400e7342484cec3a847966d99684_25.png)

![](img/0165400e7342484cec3a847966d99684_27.png)

**代码实现：**
```python
long_window = 30
df['M2'] = df['AAPL'].rolling(window=long_window).mean()
```
同样，我们使用`rolling`和`mean`方法，将30天窗口的平均值存储在名为`M2`的新列中。

## 识别黄金交叉与死亡交叉

![](img/0165400e7342484cec3a847966d99684_41.png)

![](img/0165400e7342484cec3a847966d99684_43.png)

在图表上，我们可以手动识别交叉点。当绿色的短期均线（M1）从下方上穿红色的长期均线（M2）时，形成**黄金交叉**，暗示可能上涨，是潜在的买入时机。当短期均线从上方下穿长期均线时，形成**死亡交叉**，暗示可能下跌，是潜在的卖出时机。

![](img/0165400e7342484cec3a847966d99684_45.png)

为了更精确地识别这些交叉点，我们可以用代码生成交易信号。

![](img/0165400e7342484cec3a847966d99684_47.png)

**代码实现：**
```python
import numpy as np

![](img/0165400e7342484cec3a847966d99684_49.png)

![](img/0165400e7342484cec3a847966d99684_51.png)

# 生成交易信号：当短期均线大于长期均线时为1，否则为-1
df['position'] = np.where(df['M1'] > df['M2'], 1, -1)

![](img/0165400e7342484cec3a847966d99684_53.png)

# 可视化信号
fig, ax1 = plt.subplots(figsize=(12, 6))

![](img/0165400e7342484cec3a847966d99684_55.png)

ax1.plot(df['AAPL'], label='AAPL Price', color='blue', alpha=0.5)
ax1.plot(df['M1'], label='Short-term MA', color='green')
ax1.plot(df['M2'], label='Long-term MA', color='red')
ax1.set_ylabel('Price')
ax1.legend(loc='upper left')

# 创建第二个Y轴来显示交易信号
ax2 = ax1.twinx()
ax2.plot(df['position'], label='Trading Signal', color='purple', linestyle='--')
ax2.set_ylabel('Signal (1: Buy, -1: Sell)')
ax2.legend(loc='upper right')

![](img/0165400e7342484cec3a847966d99684_57.png)

![](img/0165400e7342484cec3a847966d99684_59.png)

plt.show()
```
在这张图中，紫色虚线代表交易信号。当信号值为1时（图表上部），表示短期均线高于长期均线（黄金交叉区域）。当信号值为-1时（图表下部），表示短期均线低于长期均线（死亡交叉区域）。信号的翻转点即为我们寻找的交叉点。

![](img/0165400e7342484cec3a847966d99684_61.png)

## 总结

![](img/0165400e7342484cec3a847966d99684_63.png)

本节课我们一起学习了如何利用移动平均线进行简单的量化分析。我们首先计算了股票的短期（10天）和长期（30天）移动平均线，然后通过可视化方法观察了“黄金交叉”和“死亡交叉”这两种重要的技术信号。最后，我们使用`np.where`函数自动生成了交易信号，并通过双Y轴图表清晰地展示了信号与股价走势的关系。

*   **移动平均计算**：`df['MA'] = df['Price'].rolling(window=N).mean()`
*   **信号生成**：`df['signal'] = np.where(df[‘短期MA’] > df[‘长期MA’], 1, -1)`
