## 简单增长率累加的问题

首先，我们思考一个问题：能否将各个时间点的增长率简单相加，来得到总体的增长率？

以下是一个例子来说明这个问题。

![](img/eab154cd75de7007ffc4f10ba5ce371d_4.png)

*   假设一只股票第一天的价格是100元。
*   第二天价格跌至50元。
*   第三天价格涨至75元。

计算这两段时间的增长率：
*   第二天相对于第一天的增长率：`(50 - 100) / 100 = -0.5`
*   第三天相对于第二天的增长率：`(75 - 50) / 50 = 0.5`

如果简单地将增长率相加：`-0.5 + 0.5 = 0`。这个结果为0，似乎意味着股票从第一天到第三天既没涨也没跌。

但观察原始数据：第一天100元，第三天75元，股票实际上是下跌的。

因此，对于增长率来说，**不能直接做加法**。那么，对于连续的时间序列，如果我们想进行累加操作来反映整体变化，该怎么办呢？

![](img/eab154cd75de7007ffc4f10ba5ce371d_6.png)

## 引入对数收益率

对于连续时间的计算，我们需要借助一个数学工具：**对数**。

![](img/eab154cd75de7007ffc4f10ba5ce371d_10.png)

具体方法是计算**对数收益率**。其核心思想是，将对数收益率进行累加，等价于计算第一天与最后一天价格比值的对数。

计算公式如下：
`总对数收益率 = log(第N天价格 / 第一天价格) = log(第二天价格/第一天价格) + log(第三天价格/第二天价格) + ... + log(第N天价格/第N-1天价格)`

为什么这样可行？因为对数的加法法则：`log(a) + log(b) = log(a * b)`。

应用到我们的例子：
`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/eab154cd75de7007ffc4f10ba5ce371d_12.png)

![](img/eab154cd75de7007ffc4f10ba5ce371d_14.png)

结果 `log(75/100)` 直接反映了第一天（100元）和最后一天（75元）之间的整体关系。

![](img/eab154cd75de7007ffc4f10ba5ce371d_16.png)

![](img/eab154cd75de7007ffc4f10ba5ce371d_18.png)

形象地说，这相当于计算：如果初始股价为1个单位，经过一系列涨跌后，最终相当于多少个初始单位。

![](img/eab154cd75de7007ffc4f10ba5ce371d_20.png)

## 在Python中计算连续复合增长

![](img/eab154cd75de7007ffc4f10ba5ce371d_22.png)

![](img/eab154cd75de7007ffc4f10ba5ce371d_24.png)

以下是实现连续复合增长分析的具体步骤。

![](img/eab154cd75de7007ffc4f10ba5ce371d_26.png)

### 1. 计算每日对数收益率

![](img/eab154cd75de7007ffc4f10ba5ce371d_28.png)

![](img/eab154cd75de7007ffc4f10ba5ce371d_30.png)

首先，我们需要计算后一天价格与前一天价格比值的自然对数。

这里的关键是获取“前一天”的数据。我们可以使用Pandas的 `shift()` 函数将数据整体向上移动一格。

![](img/eab154cd75de7007ffc4f10ba5ce371d_32.png)

```python
import pandas as pd
import numpy as np

![](img/eab154cd75de7007ffc4f10ba5ce371d_34.png)

![](img/eab154cd75de7007ffc4f10ba5ce371d_36.png)

# 假设 `data` 是一个包含股价的Pandas Series
# 计算对数收益率
log_returns = np.log(data / data.shift(1))
```

代码解释：
*   `data.shift(1)` 将数据向上移动一位，使得每个位置的值都变成其“前一天”的值。
*   `data / data.shift(1)` 计算每日价格与前一日价格的比值。
*   `np.log()` 计算该比值的自然对数，得到每日的对数收益率。

### 2. 计算累积对数收益率

![](img/eab154cd75de7007ffc4f10ba5ce371d_38.png)

![](img/eab154cd75de7007ffc4f10ba5ce371d_40.png)

接下来，我们需要将每日的对数收益率累加起来，得到从起始日到每一天的累积对数收益率。这可以使用 `np.cumsum()` 函数（累计求和）来实现。

```python
# 计算累积对数收益率
cumulative_log_returns = log_returns.cumsum()
```

![](img/eab154cd75de7007ffc4f10ba5ce371d_42.png)

`np.cumsum()` 函数会返回一个序列，其中每个位置的值是原始序列从开始到该位置所有值的和。

![](img/eab154cd75de7007ffc4f10ba5ce371d_44.png)

### 3. 还原为实际价值倍数

![](img/eab154cd75de7007ffc4f10ba5ce371d_46.png)

累积对数收益率反映了以对数形式表示的增长。为了更直观地理解（例如，“最初1块钱现在值多少钱”），我们需要将其还原为实际的价值倍数。这通过指数函数 `np.exp()` 来实现。

![](img/eab154cd75de7007ffc4f10ba5ce371d_48.png)

```python
# 将累积对数收益率还原为价值倍数（初始价值为1）
normalized_price = np.exp(cumulative_log_returns)
```

![](img/eab154cd75de7007ffc4f10ba5ce371d_50.png)

现在，`normalized_price` 序列中的每个值就代表了：假设起始日投资额为1个单位，到该日期时，投资价值变为多少倍。

### 4. 可视化结果

![](img/eab154cd75de7007ffc4f10ba5ce371d_52.png)

![](img/eab154cd75de7007ffc4f10ba5ce371d_54.png)

最后，我们可以将结果绘制成图，直观地展示各只股票从起始日以来的价值增长轨迹。

```python
import matplotlib.pyplot as plt

![](img/eab154cd75de7007ffc4f10ba5ce371d_56.png)

![](img/eab154cd75de7007ffc4f10ba5ce371d_58.png)

normalized_price.plot(figsize=(16, 8))
plt.title(‘初始1单位价值的增长情况’)
plt.ylabel(‘价值倍数’)
plt.xlabel(‘日期’)
plt.show()
```

这张图可以清晰地告诉我们，如果从起始日（例如2010年）在每只股票上投入1块钱，到结束日（例如2018年），这笔投资会变成多少钱。例如，亚马逊的股票可能让1块钱变成约10块钱，而苹果的股票可能变成约4块钱。
