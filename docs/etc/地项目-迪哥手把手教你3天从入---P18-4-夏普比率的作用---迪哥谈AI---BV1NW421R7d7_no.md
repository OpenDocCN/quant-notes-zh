## 夏普比率的定义与作用
上一节我们介绍了风险的概念，本节中我们来看看如何量化地衡量风险与收益的关系。

夏普比率描述的是：在承担单位风险的情况下，所能获得的超额收益是多少。这类似于评估一份工作的“性价比”。例如，一份日薪极高但身处战区的雇佣兵工作，其高薪对应的是极高的生命风险。在金融投资中，我们也需要评估：为了获得一定的收益，我们需要承担多大的风险，这个风险是否值得。

**夏普比率越高，意味着在承担相同单位风险时，获得的超额收益越高，该投资或策略的“性价比”就越好。** 当我们面对多个股票或投资组合时，在其他条件相似的情况下，应优先选择夏普比率更高的选项。

其核心思想是计算投资组合的收益与“无风险收益”的差值，再除以该投资组合收益的波动性（风险）。

## 夏普比率的计算公式
理解了夏普比率的作用后，我们来看看它的具体计算方法。

公式涉及两个关键部分：
1.  **超额收益**：投资组合的平均收益率减去市场的无风险收益率（例如国债利率）。
2.  **风险（波动）**：投资组合收益率的标准差，代表了收益的波动程度。

夏普比率的计算公式如下：

![](img/1ac83bb99b6d516812ed84949557a3e4_4.png)

**Sharpe Ratio = (E(Rp) - Rf) / σp**

其中：
*   `E(Rp)`：投资组合的预期收益率（平均收益率）。
*   `Rf`：无风险收益率。
*   `σp`：投资组合收益率的标准差。

在实际计算年化夏普比率时，通常会对日度或月度收益率进行年化处理。假设我们使用日度收益率数据，年化夏普比率的计算公式为：

**Sharpe Ratio (annualized) = (E(Rp_daily) - Rf_daily) / σp_daily * sqrt(252)**

这里乘以 `sqrt(252)`（约等于根号252，假设一年有252个交易日）是为了将基于日数据的比率年化。

## 代码实现与计算示例
理论需要实践来巩固，接下来我们通过Python代码演示如何计算夏普比率。

在计算前，我们需要确保收益率数据的完整性。通常需要对数据中的缺失值进行处理，例如使用前一天的收益率进行填充。

以下是计算夏普比率的关键步骤代码：

```python
import numpy as np
import pandas as pd

# 假设 ‘returns‘ 是一个Pandas Series，包含了投资组合的日度收益率序列
# 假设无风险年化利率为 5%
risk_free_rate_annual = 0.05
# 转换为日度无风险利率 (假设252个交易日)
risk_free_rate_daily = risk_free_rate_annual / 252

![](img/1ac83bb99b6d516812ed84949557a3e4_6.png)

# 计算投资组合日收益率的平均值和标准差
mean_return_daily = returns.mean()
std_return_daily = returns.std()

# 计算年化夏普比率
sharpe_ratio_annual = (mean_return_daily - risk_free_rate_daily) / std_return_daily * np.sqrt(252)

print(f"年化夏普比率为: {sharpe_ratio_annual:.4f}")
```

以下是计算结果示例：
*   股票A的夏普比率：-0.15
*   股票B的夏普比率：0.82
*   股票C的夏普比率：1.35

根据“夏普比率越高越好”的原则，股票C（夏普比率1.35）是三者中最优的选择，因为它在承担单位风险时获得了最高的超额收益。股票A的夏普比率为负，意味着其收益甚至未能覆盖无风险收益，不是好的选择。
