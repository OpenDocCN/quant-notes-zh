## 策略逻辑验证：选择“最差”股票

![](img/152da72386cadc1213b5f00a949b0253_46.png)

![](img/152da72386cadc1213b5f00a949b0253_48.png)

为了进一步验证策略逻辑（即我们选取的财务指标是否真的能区分股票好坏），我们做了一个反向测试。

![](img/152da72386cadc1213b5f00a949b0253_50.png)

![](img/152da72386cadc1213b5f00a949b0253_52.png)

之前，我们的策略是选择综合得分**排名前十**的股票。现在，我们修改代码，改为选择排名**最后十名**（即指标最差）的股票。

![](img/152da72386cadc1213b5f00a949b0253_54.png)

代码修改部分如下（示意）：
```python
# 原代码：选择得分最高的前10只股票
selected_stocks = ranked_stocks.iloc[:10]

![](img/152da72386cadc1213b5f00a949b0253_56.png)

![](img/152da72386cadc1213b5f00a949b0253_58.png)

# 改为：选择得分最低的后10只股票
selected_stocks = ranked_stocks.iloc[-10:]
```

![](img/152da72386cadc1213b5f00a949b0253_60.png)

![](img/152da72386cadc1213b5f00a949b0253_62.png)

回测结果对比：
*   **选择前十（好股票）**：策略收益约为10%，显著跑赢基准（基准收益为负）。
*   **选择后十（差股票）**：策略收益为负，且比基准收益更差。

![](img/152da72386cadc1213b5f00a949b0253_64.png)

这个反向测试有力地证明了我们的打分模型是有效的：得分高的股票组合确实能产生更好的回报，而得分低的股票组合表现更差。

## 总结

![](img/152da72386cadc1213b5f00a949b0253_78.png)

![](img/152da72386cadc1213b5f00a949b0253_80.png)

本节课中我们一起学习了如何对一个量化策略进行全面的总结与分析。

1.  **回测需多周期验证**：策略在短期可能表现良好，但需要在不同的市场周期（牛、熊、震荡市）中进行长期回测，以检验其稳健性。
2.  **策略逻辑得到验证**：我们通过正向（选优）和反向（选劣）测试，证实了基于特定财务指标的打分模型能够有效区分股票的未来表现。核心公式可概括为：
    **综合得分 = Σ(指标_i × 权重_i)**
    根据得分排序选择股票。
3.  **关注关键指标**：分析策略时，应重点关注**策略收益**、**基准收益**、**超额收益**、**最大回撤**等核心指标。
4.  **市场环境影响**：任何策略的表现都无法脱离整体市场环境，在熊市中策略收益不佳是常见现象。

![](img/152da72386cadc1213b5f00a949b0253_82.png)

![](img/152da72386cadc1213b5f00a949b0253_84.png)

打分法是一种直观且常用的多因子选股方法。虽然它可能无法在所有市场环境下都表现出色，但为我们提供了一个系统化、可量化的投资决策框架。大家可以在提供的代码基础上，尝试调整指标、权重或回测参数，进一步探索和完善策略。
