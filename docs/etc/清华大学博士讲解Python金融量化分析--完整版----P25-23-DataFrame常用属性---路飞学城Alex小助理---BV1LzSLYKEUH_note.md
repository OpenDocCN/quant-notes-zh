# Python金融量化分析：P25：DataFrame常用属性 📊

在本节课中，我们将学习Pandas中`DataFrame`对象的几个常用属性。这些属性可以帮助我们快速查看和了解数据的基本结构，例如行索引、列索引、数据值以及一些统计摘要。

---

上一节我们介绍了`DataFrame`对象的创建方式，本节中我们来看看它有哪些常用的属性。

## 行索引与值

与`Series`对象类似，`DataFrame`也有`index`和`values`属性。
*   `index`属性用于获取行索引，即数据最左侧的索引标签。
*   `values`属性用于获取数据值。但与`Series`返回一维数组不同，`DataFrame`的`values`返回的是一个二维数组，其中每一行是一个一维数组。

以下是代码示例：
```python
# 假设 df 是一个已创建的 DataFrame 对象
print(df.index)   # 输出行索引，例如 Index(['A', 'B', 'C', 'D'], dtype='object')
print(df.values)  # 输出一个二维数组，包含所有数据值
```

## 转置与列索引

`DataFrame`是一个二维结构，因此它比`Series`多了一个列索引，并且可以进行转置操作。
*   `T`属性用于获取`DataFrame`的转置，即行和列互换。
*   `columns`属性用于获取列索引，即数据顶部的列名标签。

以下是代码示例：
```python
print(df.T)        # 输出转置后的 DataFrame
print(df.columns)  # 输出列索引，例如 Index(['one', 'two'], dtype='object')
```
**注意**：当数据中包含缺失值`NaN`（一个特殊的浮点数）时，整列数据可能会被自动转换为浮点类型。可以使用`.astype()`方法进行类型转换。

## 数据描述统计

`describe()`是一个非常有用的方法（注意，它是一个方法，需要加括号调用），它可以快速生成数据各列的统计摘要。

以下是`describe()`方法返回的主要统计信息：
*   **count**：非缺失值的数量。
*   **mean**：平均值。
*   **std**：标准差。
*   **min**：最小值。
*   **25%**：第一四分位数。
*   **50%**：中位数。
*   **75%**：第三四分位数。
*   **max**：最大值。

以下是代码示例：
```python
print(df.describe())  # 输出各列的统计摘要
```
