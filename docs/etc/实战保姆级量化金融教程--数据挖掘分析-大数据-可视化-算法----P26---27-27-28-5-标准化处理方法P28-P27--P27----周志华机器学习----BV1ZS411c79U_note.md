在本节课中，我们将要学习数据预处理中的一个核心步骤——标准化。标准化是数据挖掘和机器学习中非常常用的技术，其目的是消除不同特征（指标）之间由于量纲和取值范围不同带来的影响，使数据更易于分析和建模。

![](img/2074bec8368b6c878c9b0cf115ef4fb7_3.png)

## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体作用。

标准化操作的核心目标是：**使不同特征（维度）的数值处于同一量级，以便公平地进行比较和分析**。例如，在一个数据集中，特征A的取值范围是0到100，而特征B的取值范围是0到1。如果不做处理，特征A的微小波动就可能完全掩盖特征B的变化，从而影响模型对特征B的识别。

为了更直观地理解，我们可以看一个例子。假设我们有两个特征X1和X2，它们的取值范围不同。X1的取值范围较大，而X2的取值范围较小。如果我们直接用原始数据进行分析，数值的差异可能会对结果产生不必要的影响。

## 标准化的数学公式

标准化的具体操作通过一个数学公式完成。该公式分为两个步骤：

**标准化公式**：
`z = (x - μ) / σ`

其中：
*   `x` 是原始数据。
*   `μ` 是该特征所有数据的**均值**。
*   `σ` 是该特征所有数据的**标准差**。

这个公式可以分解为两个连续的操作：

1.  **去均值**：`x - μ`
    *   **目的**：将数据集的中心移动到零点。经过这一步，数据在各个维度上将以零为中心对称分布。
2.  **除标准差**：`(x - μ) / σ`
    *   **目的**：缩放数据，使其标准差变为1。这一步解决了不同特征取值范围不同的问题。取值范围大的特征，其标准差通常也大，除以较大的标准差后，其数值范围会被压缩。反之亦然。最终使所有特征的数值分布在一个相近的尺度内。

## 标准化的代码实现

理解了原理后，我们可以动手实现标准化函数。以下是使用Python和Pandas实现标准化的步骤：

首先，我们需要计算数据的均值和标准差。

```python
import pandas as pd

![](img/2074bec8368b6c878c9b0cf115ef4fb7_5.png)

![](img/2074bec8368b6c878c9b0cf115ef4fb7_7.png)

def standardize(series):
    """
    对输入的Pandas Series进行标准化处理。
    参数:
        series: 待标准化的数据序列。
    返回:
        标准化后的数据序列。
    """
    # 计算均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    # 应用标准化公式
    standardized_series = (series - mean_val) / std_val
    return standardized_series
```

代码非常简单清晰：先计算均值和标准差，然后对序列中的每个值应用标准化公式。

## 使用Scikit-learn进行标准化

![](img/2074bec8368b6c878c9b0cf115ef4fb7_11.png)

除了手动实现，我们也可以利用成熟的机器学习库来快速完成标准化。Scikit-learn库中的`StandardScaler`类提供了高效且功能更全面的标准化工具。

![](img/2074bec8368b6c878c9b0cf115ef4fb7_13.png)

以下是使用Scikit-learn进行标准化的方法：

![](img/2074bec8368b6c878c9b0cf115ef4fb7_15.png)

```python
from sklearn.preprocessing import StandardScaler
import numpy as np

![](img/2074bec8368b6c878c9b0cf115ef4fb7_17.png)

![](img/2074bec8368b6c878c9b0cf115ef4fb7_19.png)

# 假设我们有一个二维数据
data = np.array([[30, 200], [25, 180], [35, 220]])
# 创建StandardScaler对象
scaler = StandardScaler()
# 拟合数据并计算均值和标准差
scaler.fit(data)
# 进行转换
standardized_data = scaler.transform(data)
print(standardized_data)
```

使用`sklearn.preprocessing.StandardScaler`的优势在于，它可以方便地处理整个数据集（而不仅仅是单个序列），并且其`fit`方法计算的参数（均值和标准差）可以保存下来，用于后续新数据的转换，保证训练集和测试集使用相同的缩放标准。

## 总结

![](img/2074bec8368b6c878c9b0cf115ef4fb7_35.png)

![](img/2074bec8368b6c878c9b0cf115ef4fb7_37.png)

本节课中我们一起学习了数据标准化处理。
*   **目的**：消除不同特征间量纲和取值范围的差异，使模型能平等地对待所有特征。
*   **原理**：通过 **`z = (x - μ) / σ`** 公式，先进行去中心化（减均值），再进行缩放（除以标准差）。
*   **实现**：可以手动使用Pandas计算均值和标准差来实现，也可以直接调用Scikit-learn库中的`StandardScaler`工具，后者更适合生产环境和复杂数据集。

![](img/2074bec8368b6c878c9b0cf115ef4fb7_39.png)

标准化是数据预处理流程中几乎必不可少的一步，能有效提升许多机器学习算法的性能和稳定性。
