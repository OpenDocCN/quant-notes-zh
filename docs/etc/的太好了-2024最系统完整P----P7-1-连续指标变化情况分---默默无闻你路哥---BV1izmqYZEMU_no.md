## 概述

上一节我们介绍了如何计算时间序列的差异值和简单增长率。本节中，我们来看看如何分析一个连续时间序列的累积变化，例如计算一笔投资从初始到最终的总价值变化。我们将发现，直接累加增长率是无效的，并学习使用对数收益率进行正确计算的方法。

## 为什么增长率不能直接累加？

![](img/a36219c6cc568445a2cb6991f46926d3_4.png)

首先，我们思考一个问题：平均结果可以进行加法操作，那么连续时间的增长率可以累加吗？

以下是一个例子：
*   第一天股票价格：100元
*   第二天股票价格：50元
*   第三天股票价格：75元

计算每日增长率：
*   第二天增长率：(50 - 100) / 100 = -0.5
*   第三天增长率：(75 - 50) / 50 = 0.5

![](img/a36219c6cc568445a2cb6991f46926d3_6.png)

如果我们将增长率累加：-0.5 + 0.5 = 0。累加结果为零，似乎意味着股票从第一天到第三天既没涨也没跌。

![](img/a36219c6cc568445a2cb6991f46926d3_8.png)

但观察原始数据：价格从100元变为75元，实际上是下跌了。这表明，**增长率不能直接进行累加**，因为累加结果无法正确反映第一天与最后一天之间的整体变化。

![](img/a36219c6cc568445a2cb6991f46926d3_10.png)

## 解决方案：引入对数收益率

对于连续时间的计算，我们需要借助数学中的对数公式。

核心思想是：将对数收益率进行累加，等价于计算首尾价格的比例。

![](img/a36219c6cc568445a2cb6991f46926d3_12.png)

![](img/a36219c6cc568445a2cb6991f46926d3_14.png)

![](img/a36219c6cc568445a2cb6991f46926d3_16.png)

**公式推导如下：**

![](img/a36219c6cc568445a2cb6991f46926d3_18.png)

![](img/a36219c6cc568445a2cb6991f46926d3_20.png)

![](img/a36219c6cc568445a2cb6991f46926d3_22.png)

假设我们计算每日的对数收益率（以自然常数 *e* 为底）：
`log(第二天价格 / 第一天价格) + log(第三天价格 / 第二天价格)`

![](img/a36219c6cc568445a2cb6991f46926d3_24.png)

根据对数运算法则，加法可以转化为乘法：
`= log( (第二天价格/第一天价格) * (第三天价格/第二天价格) )`

![](img/a36219c6cc568445a2cb6991f46926d3_26.png)

![](img/a36219c6cc568445a2cb6991f46926d3_28.png)

约分后得到：
`= log(第三天价格 / 第一天价格)`

![](img/a36219c6cc568445a2cb6991f46926d3_30.png)

![](img/a36219c6cc568445a2cb6991f46926d3_32.png)

这个结果直接反映了**第三天价格相对于第一天价格的整体变化比例**。

形象地说，这相当于：如果初始股价为1元，经过一段时间涨跌后，最终股价是多少元。我们通过累加各阶段的对数收益率，就能得到这个最终结果。

![](img/a36219c6cc568445a2cb6991f46926d3_34.png)

![](img/a36219c6cc568445a2cb6991f46926d3_36.png)

![](img/a36219c6cc568445a2cb6991f46926d3_38.png)

## 在Python中计算连续变化

以下是实现连续变化分析的步骤和代码。

![](img/a36219c6cc568445a2cb6991f46926d3_40.png)

![](img/a36219c6cc568445a2cb6991f46926d3_42.png)

### 步骤1：计算每日的对数收益率

![](img/a36219c6cc568445a2cb6991f46926d3_44.png)

![](img/a36219c6cc568445a2cb6991f46926d3_46.png)

我们需要计算 `当日价格 / 前一日价格` 的自然对数。首先，要获取“前一日价格”序列，可以使用Pandas的 `shift()` 函数将数据向上移动一格。

![](img/a36219c6cc568445a2cb6991f46926d3_48.png)

```python
# 假设 data 是一个包含股价的Pandas Series
# 计算每日对数收益率
returns = np.log(data / data.shift(1))
print(returns.head())
```
*   `data.shift(1)`：将整个序列的值向后移动一个位置，从而得到每一天对应的“前一日价格”。
*   `np.log(...)`：计算自然对数，得到对数收益率。

![](img/a36219c6cc568445a2cb6991f46926d3_50.png)

![](img/a36219c6cc568445a2cb6991f46926d3_52.png)

### 步骤2：计算累积对数收益率并还原为实际价值

![](img/a36219c6cc568445a2cb6991f46926d3_54.png)

![](img/a36219c6cc568445a2cb6991f46926d3_56.png)

仅仅得到每日的对数收益率还不够，我们需要知道从起始日到任意一天的累积变化。这可以通过对对数收益率序列进行**累积求和**来实现。

![](img/a36219c6cc568445a2cb6991f46926d3_58.png)

![](img/a36219c6cc568445a2cb6991f46926d3_60.png)

累积求和后，我们得到的是累积对数收益率。为了将其还原为“初始1单位对应的实际价值”，需要对其进行指数运算（即 `e^累积对数收益率`）。

![](img/a36219c6cc568445a2cb6991f46926d3_62.png)

```python
# 计算累积对数收益率
cumulative_log_returns = returns.cumsum()

![](img/a36219c6cc568445a2cb6991f46926d3_64.png)

# 还原为实际价值（假设初始投资为1单位）
cumulative_value = np.exp(cumulative_log_returns)

![](img/a36219c6cc568445a2cb6991f46926d3_66.png)

![](img/a36219c6cc568445a2cb6991f46926d3_68.png)

# 绘制结果图
cumulative_value.plot(figsize=(16, 8))
```
*   `.cumsum()`：对序列进行累积求和。例如，输入[1, 2, 3]，输出[1, 3, 6]。
*   `np.exp(...)`：进行指数运算，将对数尺度还原为线性尺度。

![](img/a36219c6cc568445a2cb6991f46926d3_70.png)

执行以上代码后，我们将得到一张图表。图表中，每条曲线都从1开始，展示了如果在该股票上初始投入1单位资金，随着时间的推移，这笔投资的价值会如何变化。这直观地反映了不同股票的长期增长能力。

![](img/a36219c6cc568445a2cb6991f46926d3_72.png)

![](img/a36219c6cc568445a2cb6991f46926d3_78.png)

![](img/a36219c6cc568445a2cb6991f46926d3_80.png)

1.  **识别问题**：我们首先指出，简单的增长率（`(今日-昨日)/昨日`）**不能直接累加**，因为其累加结果无法正确反映期初与期末的整体关系。
2.  **引入解决方案**：我们引入了**对数收益率**的概念。通过对数变换，将乘法关系（价格连续相乘）转化为加法关系（对数收益率相加），使得累加变得有意义。
3.  **Python实现**：我们分两步在Python中实现了计算：
    *   使用 `shift(1)` 获取滞后数据，结合 `np.log` 计算每日对数收益率。
    *   使用 `.cumsum()` 对对数收益率进行累积求和，再通过 `np.exp` 将结果还原为易于理解的“初始单位价值”。
4.  **结果解读**：最终绘制的图表清晰地展示了不同资产从期初到任意时间点的价值累积效果，这是量化分析中评估长期趋势的常用工具。
