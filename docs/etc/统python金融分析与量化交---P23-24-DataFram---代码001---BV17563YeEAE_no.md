# 量化交易教程：P23：DataFrame索引和切片

## 📋 概述
在本节课中，我们将学习如何从DataFrame对象中获取数据。DataFrame是Pandas库中一个二维的、表格型的数据结构，理解其索引和切片操作是进行数据分析的基础。我们将重点介绍通过标签和位置索引来访问数据的方法，并比较不同方式的优缺点。

## 🔍 DataFrame数据获取的基本方法
上一节我们介绍了DataFrame的一些常用属性。本节中我们来看看如何获取DataFrame表格中的具体数值。

与Series类似，我们可以使用中括号 `[]` 来获取DataFrame的值。例如，假设我们想获取标签为 `‘A’` 的行和 `‘one’` 的列交叉位置的值。

**注意**：在DataFrame中，使用中括号索引时，通常是**先指定列，再指定行**。这与二维数组（先指定行，再指定列）的访问顺序不同，更类似于数据库查询语句 `SELECT column FROM table` 的逻辑。

```python
# 假设df是一个DataFrame
value = df['one']['A']  # 先列后行
```

然而，如果行索引是整数类型，这种方式可能会产生混淆或错误。因此，我们不推荐连续使用两个中括号进行链式索引。

## ✅ 推荐的索引方法：`.loc` 与 `.iloc`
为了避免混淆，我们建议根据索引类型（标签或位置）明确使用 `.loc` 或 `.iloc` 属性，并且避免连续使用两个中括号。

以下是更安全、更推荐的数据获取方法：

### 使用 `.loc` 通过标签索引
`.loc` 属性用于通过行和列的**标签名**进行索引。其语法是 `df.loc[行索引, 列索引]`，用逗号分隔。

```python
# 获取行标签为‘A’，列标签为‘one’的值
value = df.loc['A', 'one']
```

如果想获取某一整行的所有数据，可以这样操作：

```python
# 获取行标签为‘A’的所有列数据
row_a = df.loc['A', :]  # 冒号‘:’表示选择所有列
# 或者简写为
row_a = df.loc['A']
```

### 使用 `.iloc` 通过位置索引
`.iloc` 属性用于通过行和列的**整数位置**（从0开始）进行索引。其语法与 `.loc` 类似。

![](img/210c04bd70d660f095c893cdbb8a098c_1.png)

```python
# 获取第0行，第1列的值（假设‘one’列在第1列）
value = df.iloc[0, 1]

![](img/210c04bd70d660f095c893cdbb8a098c_3.png)

# 获取第0行的所有数据
row_0 = df.iloc[0, :]  # 或 df.iloc[0]
```

**关键区别**：
*   **`.loc`**：基于标签（label）的索引。
*   **`.iloc`**：基于整数位置（integer position）的索引。

![](img/210c04bd70d660f095c893cdbb8a098c_5.png)

## 🧩 灵活的切片与索引组合
DataFrame的索引功能非常强大，允许在行和列维度上使用多种索引方式自由组合。

以下是 `.loc` 和 `.iloc` 支持的一些高级索引方式，可以任意搭配使用：

1.  **切片（Slice）**：使用 `start:end` 的格式选择连续范围。
2.  **列表（List）**：传递一个标签或位置的列表，选择多个不连续的行或列。
3.  **布尔数组（Boolean Array）**：通过一个布尔值数组（通常由条件判断产生）来筛选数据。

![](img/210c04bd70d660f095c893cdbb8a098c_7.png)

![](img/210c04bd70d660f095c893cdbb8a098c_8.png)

**示例组合**：

```python
import pandas as pd

# 假设有一个DataFrame df
df = pd.DataFrame({'one': [1.0, 2.0, 3.0],
                   'two': [4.0, 5.0, 6.0],
                   'three': [7.0, 8.0, 9.0]},
                  index=['A', 'B', 'C'])

![](img/210c04bd70d660f095c893cdbb8a098c_10.png)

# 示例1：使用.loc，行选择标签列表['A','C']，列选择切片‘one’:‘two’
result1 = df.loc[['A', 'C'], 'one':'two']

![](img/210c04bd70d660f095c893cdbb8a098c_12.png)

# 示例2：使用.iloc，行选择位置切片0:2（即第0,1行），列选择位置列表[0, 2]（即第0,2列）
result2 = df.iloc[0:2, [0, 2]]

# 示例3：使用布尔索引选择‘one’列大于1.5的行，并只显示‘two’和‘three’列
condition = df['one'] > 1.5
result3 = df.loc[condition, ['two', 'three']]
```

通过这种方式，可以非常灵活地从DataFrame中提取出所需的任何数据子集。
