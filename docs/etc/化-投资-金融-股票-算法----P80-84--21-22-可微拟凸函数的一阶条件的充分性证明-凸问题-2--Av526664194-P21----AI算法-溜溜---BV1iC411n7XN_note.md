# 凸优化：第一章：优化问题基本概念与等价变换

在本节课中，我们将学习优化问题的核心概念，包括局部最优解、可行解集、最优解集的定义及其相互关系。我们还将探讨如何将非标准形式的优化问题，通过一系列等价变换，转化为标准形式，以便于应用现有的求解方法。

## 局部最优解的定义

上一节我们介绍了优化问题的一般形式，本节中我们来看看什么是局部最优解。

一个解 \( X \) 被称为局部最优解，意味着存在一个半径 \( R > 0 \)。以 \( X \) 为球心、\( R \) 为半径可以构成一个球 \( B_R(X) \)。该球与可行解集 \( D \) 的交集构成了一个局部可行解集 \( \Omega = D \cap B_R(X) \)，并且 \( X \) 本身也在 \( \Omega \) 中。

对于集合 \( \Omega \) 中的任意一点 \( Z \)，目标函数 \( f_0(Z) \) 的取值都不小于 \( f_0(X) \)。也就是说，\( X \) 在其局部邻域内的取值是最小的。需要注意的是，验证一个点是否为局部最优解，本质上需要求解一个局部范围内的全局优化问题。

## 局部最优解的图示

为了更直观地理解，我们考虑一个无约束的非凸函数优化问题。对于一个凸函数，其局部最优解就是全局最优解。但对于非凸函数，可能存在多个互不相连的局部最优解。

下图展示了一个非凸函数的例子：
*   **全局最优解**：既是局部最优，也是全局最优的点。
*   **局部最优解**：在某个小邻域内取值最优，但不是全局最优的点。

所有局部最优解构成的集合称为**局部最优解集**，记为 \( S_{\text{local}} \)。

## 优化问题中的集合关系

到目前为止，我们介绍了优化问题中涉及的几个关键集合。以下是这些集合及其关系：

1.  **定义域 \( D \)**：所有函数 \( f_i(x) \) 和 \( h_i(x) \) 定义域的交集。这是最大的集合。
2.  **可行解集**：满足所有约束 \( f_i(x) \leq 0 \) 和 \( h_i(x) = 0 \) 的点 \( x \) 构成的集合，是 \( D \) 的子集。
3.  **最优解集 \( S \)**：所有使得目标函数 \( f_0(x) \) 取到全局最小值的点构成的集合，是可行解集的子集。
4.  **\( \epsilon \)-次优解集 \( S_\epsilon \)**：所有满足 \( f_0(x) \leq p^* + \epsilon \) 的点构成的集合，其中 \( p^* \) 是最优值。当 \( \epsilon = 0 \) 时，\( S_\epsilon = S \)；当 \( \epsilon \to \infty \) 时，\( S_\epsilon \) 趋近于可行解集。
5.  **局部最优解集 \( S_{\text{local}} \)**：所有局部最优解构成的集合。它一定包含全局最优解集 \( S \)，但与 \( \epsilon \)-次优解集 \( S_\epsilon \) 的关系不确定，取决于 \( \epsilon \) 的取值。

这些集合的包含关系可以概括为：**最优解集 \( S \) ⊆ 局部最优解集 \( S_{\text{local}} \) ⊆ 可行解集 ⊆ 定义域 \( D \)**。\( \epsilon \)-次优解集 \( S_\epsilon \) 与局部最优解集 \( S_{\text{local}} \) 有交集，但不一定存在包含关系。

## 约束形式与活动约束

我们通常将不等式约束写为 \( f_i(x) \leq 0 \) 而非 \( f_i(x) < 0 \)。原因在于实际问题中，最优解往往出现在约束的边界上。

对于一个可行解 \( x \)，针对某个不等式约束 \( f_i(x) \leq 0 \)：
*   如果 \( f_i(x) = 0 \)，则称该约束在点 \( x \) 处是**活动的**。
*   如果 \( f_i(x) < 0 \)，则称该约束在点 \( x \) 处是**不活动的**。

从工程角度，任何形如 \( f_i(x) < 0 \) 的约束，总可以通过调整边界（例如，将 \( x < 100 \) 改为 \( x \leq 99.99 \)）转化为 \( \leq \) 形式。从数学角度，也可以通过引入函数（如对数函数）将严格不等式纳入定义域，从而转化为标准形式。

## 可行性问题与等价变换

有一类重要的优化问题称为**可行性问题**，它不追求最优，只寻找一个满足所有约束的解。其标准形式为：
```
find x
subject to f_i(x) ≤ 0, i = 1, ..., m
           h_i(x) = 0, i = 1, ..., p
```
它可以等价地转化为标准优化问题：`minimize 0 subject to ...`。因为极小化一个常数，其最优解集就是所有可行解。

将非标准问题转化为标准形式是优化中的重要技巧。以下是几种常见的等价变换：

### 1. 缩放变换
对目标函数和约束函数进行正系数缩放，不改变问题本质。
*   **目标函数**：`minimize α₀ f₀(x)`，其中 `α₀ > 0`。
*   **不等式约束**：`α_i f_i(x) ≤ 0`，其中 `α_i > 0`。
*   **等式约束**：`β_i h_i(x) = 0`，其中 `β_i ≠ 0`。
这种变换常用于统一不同约束的量纲或权重。

### 2. 函数变换
引入单调函数进行复合变换。
*   设 `φ₀: R → R` 是单调递增函数，则 `minimize φ₀(f₀(x))` 与原问题等价。
*   设 `φ_i: R → R` 满足 `φ_i(u) ≤ 0 ⇔ u ≤ 0`，则约束 `φ_i(f_i(x)) ≤ 0` 与原约束 `f_i(x) ≤ 0` 等价。
*   设 `ψ_i: R → R` 满足 `ψ_i(u) = 0 ⇔ u = 0`，则约束 `ψ_i(h_i(x)) = 0` 与原约束 `h_i(x) = 0` 等价。
**例子**：`minimize ||Ax - b||₂` 等价于 `minimize ||Ax - b||₂²`，因为平方函数在非负实数域上是单调递增的。

### 3. 消除等式约束
等式约束会增加问题复杂度。如果等式约束 `h_i(x) = 0, i=1,...,p` 的解集可以用参数形式 `x = φ(z)` 表示，其中 `z ∈ R^k`，那么原优化问题可以等价地转化为关于 `z` 的、无等式约束的问题：
```
minimize f₀(φ(z))
subject to f_i(φ(z)) ≤ 0, i = 1, ..., m
```
**例子**：线性等式约束 `Ax = b`。
*   若 `A` 可逆，则解唯一 `x = A⁻¹b`，问题退化为常量。
*   更一般地，设 `x = Fz + x₀`，其中 `F` 是 `A` 的零空间矩阵，`x₀` 是 `Ax = b` 的一个特解。则原 `n` 维问题可转化为关于 `z` 的 `(n - rank(A))` 维问题，同时消除了所有等式约束。

## 总结

本节课中我们一起学习了优化问题的基本概念。我们明确了局部最优解的定义及其与全局最优解的关系，梳理了定义域、可行解集、最优解集等关键集合的层次结构。我们还深入探讨了将不等式约束写成 `≤` 形式的原因，并介绍了活动约束的概念。最后，我们掌握了多种将非标准优化问题（如可行性问题、含盒约束问题）通过缩放、函数复合、消除等式约束等方法，等价转化为标准形式的技巧，这是应用通用优化算法求解实际问题的关键一步。
