## 公式与地址引用

上一节我们介绍了函数的具体应用，本节我们先来回顾公式与地址引用的基础。

**公式**是以等号（`=`）开头，用于建立数据间联系并执行计算的算法。它需要使用运算符。

**单元格引用**决定了公式中使用的数据来源。引用方式主要有三种：

以下是三种引用方式的对比：

![](img/9af89227e96e9629f277d5deeccd4249_3.png)

*   **相对引用**：公式中的单元格地址会随着公式位置的移动而自动调整。可以理解为“如影随形”。
    *   示例：`=A1+B1`
*   **绝对引用**：公式中的单元格地址固定不变，无论公式复制到何处。可以理解为“以不变应万变”。
    *   示例：`=$A$1+$B$1`
*   **混合引用**：单元格地址的行号或列标之一固定，另一个相对变化。可以理解为“审时度势”。
    *   示例：`=$A1` (列绝对，行相对) 或 `=A$1` (列相对，行绝对)

![](img/9af89227e96e9629f277d5deeccd4249_5.png)

决定引用方式的符号是美元符号（`$`）。在行号或列标前添加`$`，该部分就变为绝对引用。

## 函数总结

理解了公式的基础后，我们来系统回顾已学习的各类函数。

### 逻辑函数

逻辑函数用于进行条件判断。

![](img/9af89227e96e9629f277d5deeccd4249_7.png)

*   **IF函数**：根据条件返回不同的值。
    *   公式：`=IF(逻辑测试, [值为真时的结果], [值为假时的结果])`
*   **AND函数**：所有条件都为真时返回`TRUE`。
    *   公式：`=AND(条件1, [条件2], ...)`
*   **OR函数**：任一条件为真时返回`TRUE`。
    *   公式：`=OR(条件1, [条件2], ...)`
*   **IF函数嵌套**：在IF函数的结果中再嵌入IF函数，用于多条件判断。

![](img/9af89227e96e9629f277d5deeccd4249_8.png)

### 文本函数

![](img/9af89227e96e9629f277d5deeccd4249_9.png)

文本函数用于处理和操作文本字符串。

![](img/9af89227e96e9629f277d5deeccd4249_10.png)

![](img/9af89227e96e9629f277d5deeccd4249_11.png)

*   **LEFT函数**：从文本左侧提取指定数量的字符。
    *   公式：`=LEFT(文本, [字符数])`
*   **RIGHT函数**：从文本右侧提取指定数量的字符。
    *   公式：`=RIGHT(文本, [字符数])`
*   **MID函数**：从文本指定位置开始提取指定数量的字符。
    *   公式：`=MID(文本, 开始位置, 字符数)`
*   **LEN函数**：返回文本字符串的字符数。
    *   公式：`=LEN(文本)`
*   **TEXT函数**：将数值转换为按指定数字格式表示的文本。
    *   公式：`=TEXT(数值, 格式代码)`
*   **REPLACE函数**：替换文本中指定位置和长度的字符。
    *   公式：`=REPLACE(原文本, 开始位置, 字符数, 新文本)`

### 查找与引用函数

查找与引用函数用于在数据区域中查找和引用特定信息。

*   **VLOOKUP函数**：垂直查找，在表格的首列查找指定的值，并返回当前行中指定列的值。
    *   公式：`=VLOOKUP(查找值, 表格区域, 列序数, [匹配类型])`
*   **MATCH函数**：返回指定值在数组中的相对位置。
    *   公式：`=MATCH(查找值, 查找区域, [匹配类型])`

接下来，我们补充讲解几个与行列位置相关的函数。

*   **ROW函数**：返回指定单元格的行号。
    *   公式：`=ROW([单元格引用])`，如果省略参数，则返回公式所在单元格的行号。
*   **COLUMN函数**：返回指定单元格的列号。
    *   公式：`=COLUMN([单元格引用])`，如果省略参数，则返回公式所在单元格的列号。
*   **COLUMNS函数**：返回数组或引用的列数。
    *   公式：`=COLUMNS(数组或区域)`

### 统计函数

![](img/9af89227e96e9629f277d5deeccd4249_13.png)

统计函数用于对数据进行数学和统计计算。

![](img/9af89227e96e9629f277d5deeccd4249_14.png)

![](img/9af89227e96e9629f277d5deeccd4249_15.png)

*   **INT函数**：将数字向下舍入到最接近的整数。
    *   公式：`=INT(数字)`
*   **MOD函数**：返回两数相除的余数。
    *   公式：`=MOD(被除数, 除数)`
*   **ROUND函数**：将数字四舍五入到指定的小数位数。
    *   公式：`=ROUND(数字, 小数位数)`
*   **AVERAGE函数**：计算参数的平均值。
    *   公式：`=AVERAGE(数值1, [数值2], ...)`
*   **MAX/MIN函数**：返回一组值中的最大值/最小值。
    *   公式：`=MAX(数值1, [数值2], ...)` / `=MIN(数值1, [数值2], ...)`
*   **SUM函数**：计算参数之和。
    *   公式：`=SUM(数值1, [数值2], ...)`
*   **COUNT函数**：计算包含数字的单元格个数。
    *   公式：`=COUNT(值1, [值2], ...)`
*   **COUNTIF函数**：计算满足给定条件的单元格个数。
    *   公式：`=COUNTIF(条件区域, 条件)`
*   **SUMIF函数**：对满足条件的单元格求和。
    *   公式：`=SUMIF(条件区域, 条件, [求和区域])`
*   **COUNTIFS/SUMIFS函数**：多条件计数/求和。
    *   公式：`=COUNTIFS(条件区域1, 条件1, [条件区域2, 条件2], ...)` / `=SUMIFS(求和区域, 条件区域1, 条件1, [条件区域2, 条件2], ...)`

### 日期函数

日期函数用于处理日期和时间值。

*   **YEAR/MONTH/DAY函数**：返回日期中的年/月/日。
    *   公式：`=YEAR(日期)` / `=MONTH(日期)` / `=DAY(日期)`
*   **TODAY函数**：返回当前日期。
    *   公式：`=TODAY()`
*   **NOW函数**：返回当前日期和时间。
    *   公式：`=NOW()`
*   **DATE函数**：将提供的年、月、日参数组合成一个日期。
    *   公式：`=DATE(年, 月, 日)`
*   **DATEDIF函数**：计算两个日期之间的天数、月数或年数。
    *   公式：`=DATEDIF(开始日期, 结束日期, 单位代码)`
