上一节我们介绍了如何计算策略的回测收益率，它反映了策略在整个回测周期内的总盈亏情况。本节中我们来看看如何将总收益转化为更标准、更便于比较的年化收益率。

## 理解回测收益率

![](img/6e9a39b63df2a45e8ba07667c3245549_8.png)

![](img/6e9a39b63df2a45e8ba07667c3245549_10.png)

回测收益率是评估策略表现的基础指标。它计算的是策略在整个回测周期结束后，与初始本金相比的盈亏比例。

![](img/6e9a39b63df2a45e8ba07667c3245549_12.png)

以下是回测收益率的核心含义：
*   它直接反映了策略的最终盈亏结果。
*   结果可能为正（盈利），也可能为负（亏损），这体现了“投资有风险”的原则。
*   它是一个总体的、绝对的比例值，例如“翻倍”意味着收益率为100%。

![](img/6e9a39b63df2a45e8ba07667c3245549_14.png)

## 引入年化收益率

![](img/6e9a39b63df2a45e8ba07667c3245549_16.png)

![](img/6e9a39b63df2a45e8ba07667c3245549_18.png)

对于策略来说，理解了总收益后，还有一个关键值叫做年化收益率。

你可能会想，你已经知道了赚了多少。但如果你用10年时间实现了一个收益率，而别人只用了1年实现了相同的收益率，两者的效率显然是不同的。因此，在金融领域，我们通常更关注年化收益，它将不同时间长度的收益标准化到“每年”的维度，便于横向比较。

![](img/6e9a39b63df2a45e8ba07667c3245549_20.png)

年化收益率是对回测收益率进行公式变换后得到的结果。它通常是我们重点观察的核心指标。

![](img/6e9a39b63df2a45e8ba07667c3245549_22.png)

## 年化收益率的计算公式

![](img/6e9a39b63df2a45e8ba07667c3245549_24.png)

![](img/6e9a39b63df2a45e8ba07667c3245549_26.png)

年化收益率的计算公式其实很简单。我们来分解一下其中的变量：

*   **R**：表示总的回测收益率。这是我们上一节已经计算好的值。
*   **M**：代表一年的交易日数量。在金融计算中，通常取 **250** 或 **252** 天，而非365天。
*   **N**：代表你的回测数据所覆盖的总交易日数量。例如，在我们的示例数据中，总共有 **730** 个交易日样本。

![](img/6e9a39b63df2a45e8ba07667c3245549_28.png)

![](img/6e9a39b63df2a45e8ba07667c3245549_30.png)

基于以上变量，年化收益率（Annualized Return）的计算公式如下：

**公式：`年化收益率 = (1 + 总收益率) ^ (一年交易日数 / 总交易日数) - 1`**

![](img/6e9a39b63df2a45e8ba07667c3245549_32.png)

![](img/6e9a39b63df2a45e8ba07667c3245549_34.png)

用代码可以表示为：
```python
# 假设 total_return 是总收益率，trade_days_per_year=250， total_trade_days 是总交易日数
annualized_return = (1 + total_return) ** (trade_days_per_year / total_trade_days) - 1
```

![](img/6e9a39b63df2a45e8ba07667c3245549_36.png)

![](img/6e9a39b63df2a45e8ba07667c3245549_38.png)

## 计算实例与结果对比

![](img/6e9a39b63df2a45e8ba07667c3245549_40.png)

让我们实际计算一下。我们将这个新指标命名为“回测年化收益”。

![](img/6e9a39b63df2a45e8ba07667c3245549_42.png)

执行计算后，我们可以将年化收益的结果与之前的回测总收益进行对比。你会发现两者数值存在差异。

![](img/6e9a39b63df2a45e8ba07667c3245549_44.png)

![](img/6e9a39b63df2a45e8ba07667c3245549_46.png)

例如，某个策略在730天（大约3年）内的总收益可能是113%。但它的年化收益经过计算可能只有20%多。另一个最初显示为负收益的策略，其年化负收益的数值也会有所不同。

![](img/6e9a39b63df2a45e8ba07667c3245549_48.png)

这再次说明，在评估策略时，你不仅需要关注最终的总收益率，还必须考察其年化收益率，后者能更公平地反映策略的盈利能力。

![](img/6e9a39b63df2a45e8ba07667c3245549_50.png)

![](img/6e9a39b63df2a45e8ba07667c3245549_52.png)

本节课中我们一起学习了量化策略收益评估的两个核心指标。我们首先回顾了**回测收益率**，它表示策略在整个周期内的总盈亏。接着，我们深入探讨了**年化收益率**，它通过一个标准公式将总收益转化成年度比例，是进行不同周期策略对比的关键工具。理解这两个指标是分析和优化任何交易策略的基础。
