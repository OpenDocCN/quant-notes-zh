## 策略评估指标简介
当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以判断策略的盈亏情况。但“赚了”或“赔了”的描述过于笼统。我们需要更精确的指标来回答一系列问题：具体赚了多少？与大盘相比有何优势？策略承担了哪些风险？这些风险是否值得？与其他策略相比有何优劣？这些问题的答案就构成了策略的评估指标。

## 核心指标：回测收益率
在众多评估指标中，回测收益率是最基本且最重要的一个。它衡量的是策略在整个回测周期内，最终获得的总体收益相对于初始本金的比率。

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_24.png)

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_26.png)

**公式**：
`回测收益率 = (期末资产总值 - 期初本金) / 期初本金`

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_28.png)

也可以表示为：
`回测收益率 = (P_T / P_0) - 1`
其中，`P_T`代表回测结束时的资产价格（或总值），`P_0`代表回测开始时的初始本金。

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_30.png)

这个指标直观地告诉我们：“折腾了一番后，我的本金最终变成了原来的多少倍？”例如，4.23%的收益率意味着本金增长了约4.23%。

### 计算与可视化回测收益率
为了更深入地理解，我们可以计算每一天相对于期初的累计收益率，并绘制其走势图。

以下是计算步骤：
1.  计算每日资产净值与期初净值的比值。
2.  将上述比值减去1，得到每日的累计收益率序列。
3.  取序列最后一个值，即为整个回测周期的总收益率。

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_32.png)

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_34.png)

```python
# 1. 计算每日净值与期初净值的比值
cumulative_return_ratio = data / data.iloc[0]

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_36.png)

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_38.png)

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_40.png)

# 绘制累计收益率走势图
plt.figure(figsize=(10, 6))
cumulative_return_ratio.plot()
plt.title('累计收益率走势')
plt.ylabel('净值比 (相对于期初)')
plt.xlabel('日期')
plt.grid(True)
plt.show()

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_42.png)

# 2. 计算最终的回测收益率 (总收益率)
final_return = cumulative_return_ratio.iloc[-1] - 1

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_44.png)

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_46.png)

![](img/fbcdc142f2a531ad3985fbc7320eb6c1_48.png)

# 3. 将结果整理为DataFrame以便查看
result_df = pd.DataFrame({
    ‘回测收益率’: final_return.values
}, index=[‘策略最终收益’])
print(result_df)
```
通过运行以上代码，你可以得到两个输出：
1.  **一张图表**：展示了资产净值随时间增长（或下跌）的相对路径。从图中可以清晰看到策略的盈利周期和回撤期。
2.  **一个表格**：清晰地列出了策略在整个回测期间实现的最终总收益率。
