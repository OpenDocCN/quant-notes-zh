在本节课中，我们将要学习一种新的循环结构——`while`循环。与之前学习的`for`循环不同，`while`循环的次数可以是不确定的，它根据条件是否成立来决定是否继续执行循环体。我们将通过简单的例子来理解其语法和应用场景。

## 什么是while循环？

上一节我们介绍了`for`循环，它通常用于遍历已知次数的序列。本节中我们来看看`while`循环。

`while`循环，又名“死循环”，其核心特点是：只要条件满足，循环就会一直执行下去。这意味着循环的次数可以是不固定的。

**公式**：`while`循环的基本语法结构如下：
```python
while 条件:
    # 循环体代码
```
当`条件`为真（`True`）时，就会执行循环体内的代码。执行完毕后，会再次检查`条件`，如果依然为真，则继续执行，如此反复。

## 编写一个死循环

以下是创建一个无限循环（死循环）的示例：

![](img/a950671d3047b3b79cdc6bea95880a8b_3.png)

```python
while True:
    # 这里可以执行任何操作
    print("这是一个死循环")
```
在这个例子中，因为条件`True`永远为真，所以循环会永不停止地执行下去。

![](img/a950671d3047b3b79cdc6bea95880a8b_5.png)

## 控制循环次数

虽然`while`循环可以无限执行，但我们通常需要控制它的循环次数。这可以通过在循环体内修改影响条件的变量来实现。

以下是限制循环次数的示例：

```python
count = 0
while count < 10:
    print(f"这是第 {count + 1} 次循环")
    count += 1  # 每次循环计数器加1
```
在这个例子中，我们设置了一个计数器`count`。只要`count`小于10，循环就会继续。每次循环，`count`都会增加1，直到`count`等于10时，条件不再满足，循环结束。

## while循环实战：猜年龄游戏

![](img/a950671d3047b3b79cdc6bea95880a8b_9.png)

现在，让我们应用`while`循环来改写之前用`for`循环实现的“猜年龄”游戏，并限制只能猜三次。

以下是实现代码：

![](img/a950671d3047b3b79cdc6bea95880a8b_11.png)

```python
age = 26  # 设定的正确年龄
count = 0  # 初始化猜测次数计数器

![](img/a950671d3047b3b79cdc6bea95880a8b_13.png)

![](img/a950671d3047b3b79cdc6bea95880a8b_15.png)

while count < 3:
    guess = int(input("请输入你猜的年龄："))
    if guess == age:
        print("恭喜你，猜对了！")
        break  # 猜对后使用break跳出循环
    elif guess > age:
        print("猜大了，再试试！")
    else:
        print("猜小了，再试试！")
    count += 1  # 每次猜测后计数器加1

![](img/a950671d3047b3b79cdc6bea95880a8b_17.png)

![](img/a950671d3047b3b79cdc6bea95880a8b_19.png)

# 循环结束后执行的代码
print("游戏结束。")
```
在这个游戏中：
1.  我们使用`while count < 3`来控制最多只能猜三次。
2.  每次猜测后，计数器`count`会加1。
3.  如果猜对了，使用`break`语句提前跳出循环。
4.  循环结束后，会执行后续的代码（如“游戏结束”的提示）。

## break与continue在while循环中的使用

与`for`循环一样，`while`循环中也可以使用`break`和`continue`语句来控制循环流程。
*   **`break`**：立即终止整个循环。
*   **`continue`**：跳过本次循环中剩余的代码，直接进入下一次循环条件判断。

它们的用法与在`for`循环中完全相同。

## 总结

本节课中我们一起学习了`while`循环。我们了解到：

2.  可以通过在循环体内改变条件变量的值，来控制`while`循环的执行次数。
3.  `while`循环同样支持`break`和`continue`语句来控制循环流程。
4.  通过“猜年龄”游戏的实战，我们掌握了如何用`while`循环实现有次数限制的交互逻辑。
