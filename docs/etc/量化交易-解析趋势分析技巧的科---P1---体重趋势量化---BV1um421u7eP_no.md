# 量化交易：1：从体重趋势理解量化分析基础

在本节课中，我们将学习量化分析的基本概念。我们将通过一个非常生活化的例子——分析体重变化趋势，来理解量化交易中趋势分析的核心思想。这种方法将帮助我们摆脱主观猜测，学会用数据和规则来客观地判断市场方向。

## 体重记录的启示

量化分析的核心在于将模糊的感觉转化为清晰的数据和规则。为了理解这一点，我们可以先看一个生活中的例子：记录体重变化。

假设你开始健身，并每天记录自己的体重。一周后，你得到以下数据：

```
周一：70.5公斤
周二：70.2公斤
周三：70.8公斤
周四：70.0公斤
周五：69.8公斤
周六：69.5公斤
周日：69.3公斤
```

仅凭感觉，你可能会说“体重好像下降了”。但在量化分析中，我们不能依赖“好像”。我们需要一个明确的规则来判断趋势。

## 定义量化趋势规则

上一节我们看到了原始的体重数据，本节中我们来看看如何制定一个量化的规则来判断趋势。

一个简单有效的规则是：**比较当前体重与三天前的平均体重**。如果当前体重低于过去三天的平均体重，则认为体重处于下降趋势；反之，则认为处于上升或盘整趋势。

我们可以用以下公式来描述这个规则：

**趋势信号 = 当前体重 < 过去三天体重的平均值 ? “下降” : “非下降”**

以下是应用该规则分析上述数据的具体步骤：

1.  **计算三天平均体重**：从周四开始，我们可以计算前三天（周一、周二、周三）的体重平均值。
    *   周四的参照均值 = (70.5 + 70.2 + 70.8) / 3 = 70.5公斤
2.  **比较判断**：将当前体重与这个均值比较。
    *   周四体重70.0公斤 < 70.5公斤 → 信号为“下降”。
3.  **持续计算**：对后续每一天重复此过程。
    *   周五均值（周二、三、四）= (70.2 + 70.8 + 70.0) / 3 = 70.33公斤，体重69.8 < 70.33 → “下降”。
    *   周六均值（周三、四、五）= (70.8 + 70.0 + 69.8) / 3 = 70.2公斤，体重69.5 < 70.2 → “下降”。
    *   周日均值（周四、五、六）= (70.0 + 69.8 + 69.5) / 3 = 69.77公斤，体重69.3 < 69.77 → “下降”。

通过这个简单的规则，我们得出了清晰、一致的结论：从周四开始，体重持续处于下降趋势。这比单纯的感觉“好像瘦了”要可靠得多。

## 从体重到价格趋势

理解了如何量化分析体重趋势后，我们可以将这一思维完全迁移到金融市场的价格分析中。

在量化交易中，股价、币价等金融资产的价格，就相当于我们的“体重”。它们每天上下波动，形成一系列数据。我们的目标同样是识别其中的趋势。

我们可以定义完全类似的规则：**比较当前价格与过去一段时间（例如N天）的平均价格**。这个“平均价格”在金融领域有一个专门且强大的指标，称为**移动平均线**。

例如，在判断股价趋势时，一个常用的方法是：
**如果当前股价 > 20日移动平均线，则认为处于上升趋势；如果当前股价 < 20日移动平均线，则认为处于下降趋势。**

用伪代码可以表示为：
```python
if current_price > moving_average(price, window=20):
    trend = "上升"
else:
    trend = "下降"
```

这条移动平均线，就像为杂乱的股价波动描绘出了一条平滑的趋势轨道。价格在轨道之上，代表势头向上；价格在轨道之下，代表势头向下。这为我们提供了客观、可回溯的交易决策依据。

## 总结

本节课中我们一起学习了量化分析的基础思维模式。

我们从一个记录体重的例子出发，展示了如何将“感觉变轻了”这种主观判断，转化为“当前体重是否低于过去三天平均体重”的客观规则。随后，我们将这一生活化的量化逻辑无缝衔接到了金融市场，引入了**移动平均线**这一核心工具，用它来清晰界定价格运行的趋势状态。

这种“定义数据、制定规则、执行判断”的过程，正是量化交易的核心。它摒弃了情绪干扰，让决策建立在一致性和纪律性之上。在接下来的课程中，我们将基于这一基础，探索更复杂的量化模型和策略。
