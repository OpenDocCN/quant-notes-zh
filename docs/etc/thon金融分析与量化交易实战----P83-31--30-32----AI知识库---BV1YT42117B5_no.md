## 🧹 第一步：构建股票过滤函数
在分析股票之前，我们需要先过滤掉一些不符合交易条件的股票，例如停牌股、ST股和刚上市的新股。以下是构建过滤函数的具体步骤。

![](img/fbdf7a47cea43281989844138b88d635_20.png)

### 1. 过滤停牌股票
首先，我们需要判断并过滤掉处于停牌状态的股票。我们使用一个函数来实现这个功能。

![](img/fbdf7a47cea43281989844138b88d635_22.png)

![](img/fbdf7a47cea43281989844138b88d635_24.png)

```python
def filter_suspended_stocks(stock_list):
    """
    过滤掉停牌的股票。
    :param stock_list: 初始股票列表
    :return: 过滤掉停牌股后的股票列表
    """
    filtered_stocks = []
    for stock in stock_list:
        # 使用API判断股票是否全天停牌
        if not is_suspended(stock):
            filtered_stocks.append(stock)
    return filtered_stocks
```
**核心逻辑**：遍历股票列表，使用 `is_suspended(stock)` API判断每只股票是否停牌，将未停牌的股票加入新列表并返回。

### 2. 过滤ST股票
接下来，我们需要过滤掉被标记为ST（特别处理）的股票，这类股票通常存在较高风险。

![](img/fbdf7a47cea43281989844138b88d635_32.png)

![](img/fbdf7a47cea43281989844138b88d635_34.png)

![](img/fbdf7a47cea43281989844138b88d635_36.png)

```python
def filter_st_stocks(stock_list):
    """
    过滤掉ST股票。
    :param stock_list: 经过停牌过滤后的股票列表
    :return: 过滤掉ST股后的股票列表
    """
    filtered_stocks = []
    for stock in stock_list:
        # 使用API判断股票是否为ST股
        if not is_st(stock):
            filtered_stocks.append(stock)
    return filtered_stocks
```
**核心逻辑**：遍历股票列表，使用 `is_st(stock)` API判断每只股票是否为ST股，将非ST股加入新列表。

### 3. 过滤新股
最后，我们需要过滤掉上市时间过短的新股，以确保股票有足够的历史数据可供分析。这里我们设定一个阈值（例如180天）。

![](img/fbdf7a47cea43281989844138b88d635_44.png)

![](img/fbdf7a47cea43281989844138b88d635_46.png)

![](img/fbdf7a47cea43281989844138b88d635_48.png)

```python
def filter_new_stocks(stock_list, days_threshold=180):
    """
    过滤掉上市天数少于阈值的股票。
    :param stock_list: 经过前两步过滤的股票列表
    :param days_threshold: 上市天数阈值，默认为180天
    :return: 过滤掉新股后的股票列表
    """
    filtered_stocks = []
    for stock in stock_list:
        # 获取股票上市以来的天数
        days_listed = get_days_listed(stock)
        # 如果上市天数大于阈值，则保留
        if days_listed > days_threshold:
            filtered_stocks.append(stock)
    return filtered_stocks
```
**核心逻辑**：遍历股票列表，使用 `get_days_listed(stock)` 获取每只股票的上市天数，只保留上市天数超过设定阈值（如180天）的股票。
