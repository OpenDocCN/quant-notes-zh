## 什么是最大回撤？🎢

上一节我们介绍了策略的收益表现，本节中我们来看看如何评估策略的风险。最大回撤描述的是投资过程中，资产净值从最高点到之后最低点的**最大跌幅**。

它衡量的是投资者在策略运行期间可能经历的“最惨”阶段。例如，你的投资组合价值从100元涨到150元，然后又跌到80元，那么这期间的最大回撤就是从150元跌到80元的过程。

![](img/8639642d696eb514b7a3ac0df8eef4c7_1.png)

在回测分析图表中，最大回撤通常用一个区间（例如图中的阴影部分）来标示，它代表了净值下跌最严重的那段时期。

---

## 最大回撤的意义：衡量“惨劲儿”📉

最大回撤是一个**风险指标**。它不直接评价策略的好坏，而是评估策略的**风险程度**。

即使一个策略长期来看是盈利的，但如果它的最大回撤非常大，意味着投资者在过程中可能需要承受巨大的心理压力和资金波动风险。历史上，巨大的回撤可能导致投资者无法坚持策略而提前退出，造成实际亏损。

例如，2008年股市从高点大幅下跌，许多投资者经历了巨大的资产缩水，这个过程就可以用最大回撤来描述其惨烈程度。

**核心概念**：最大回撤描述的是投资过程中“最惨”的那段下跌区间及其幅度。

## 如何计算最大回撤？🧮

理解了概念后，我们来看看如何用数学和代码来计算最大回撤。

![](img/8639642d696eb514b7a3ac0df8eef4c7_7.png)

最大回撤的计算目标是：在所有可能的“高点”和之后的“低点”组合中，找到跌幅最大的那个。

其**公式**可以表示为：

![](img/8639642d696eb514b7a3ac0df8eef4c7_9.png)

**最大回撤 = max[ (Pi - Pj) / Pi ]**

![](img/8639642d696eb514b7a3ac0df8eef4c7_11.png)

其中：
*   **Pi** 代表第 i 天的资产净值（一个高点）。
*   **Pj** 代表第 j 天的资产净值（i 天之后的某个低点，且 Pj < Pi）。
*   公式计算的是从高点 Pi 到低点 Pj 的**相对跌幅**。
*   我们需要遍历所有可能的 i 和 j (j > i)，找出这个比值的最大值。

![](img/8639642d696eb514b7a3ac0df8eef4c7_13.png)

## 使用Python计算最大回撤💻

![](img/8639642d696eb514b7a3ac0df8eef4c7_15.png)

以下是使用Pandas库高效计算最大回撤的步骤和代码逻辑。

![](img/8639642d696eb514b7a3ac0df8eef4c7_17.png)

首先，我们需要一个代表每日资产净值的序列，这里用 `price_series` 表示。

![](img/8639642d696eb514b7a3ac0df8eef4c7_19.png)

**计算步骤**：
1.  计算该序列到当前为止的**历史最大值**序列。这代表了到每一天为止，投资者曾达到的“最佳状态”。
2.  用这个“历史最大值”序列减去**当日实际净值**序列，得到每日的“潜在亏损”序列。
3.  将“潜在亏损”序列除以对应的“历史最大值”序列，得到每日的“回撤比率”序列。
4.  在这个“回撤比率”序列中，找到**最大值**，这就是整个考察期内的最大回撤。

![](img/8639642d696eb514b7a3ac0df8eef4c7_21.png)

以下是核心代码演示：

![](img/8639642d696eb514b7a3ac0df8eef4c7_23.png)

![](img/8639642d696eb514b7a3ac0df8eef4c7_25.png)

```python
import pandas as pd

![](img/8639642d696eb514b7a3ac0df8eef4c7_27.png)

# 假设 price_series 是一个包含每日净值的Pandas Series
# 1. 计算历史累计最大值序列
historical_max = price_series.cummax()

![](img/8639642d696eb514b7a3ac0df8eef4c7_29.png)

# 2. 计算每日回撤（历史最高 - 当日净值）
drawdown = historical_max - price_series

![](img/8639642d696eb514b7a3ac0df8eef4c7_31.png)

# 3. 计算每日回撤比率
drawdown_ratio = drawdown / historical_max

# 4. 找到最大回撤比率
max_drawdown = drawdown_ratio.max()

![](img/8639642d696eb514b7a3ac0df8eef4c7_33.png)

print(f"最大回撤为：{max_drawdown:.2%}")
```

![](img/8639642d696eb514b7a3ac0df8eef4c7_35.png)

**代码解释**：
*   `.cummax()` 函数是计算累计最大值的关键。对于序列 `[a, b, c, ...]`，它返回 `[a, max(a,b), max(a,b,c), ...]`。
*   通过上述操作，我们避免了复杂的双重循环，利用向量化计算高效地得到了结果。

![](img/8639642d696eb514b7a3ac0df8eef4c7_37.png)
