## 打分法的基本原理 🧮

上一节我们介绍了如何获取和预处理股票数据。本节中，我们来看看如何利用这些数据对股票进行评价和筛选。核心方法是“打分法”。

打分法的核心思想是：为每个待评价的股票，根据其在不同指标上的表现进行评分，然后将各指标得分汇总，得到一个总分。最后，根据总分对所有股票进行排序，选择排名靠前的股票作为投资组合的候选。

以下是打分法的关键步骤：

1.  **确定评价指标与方向**：首先，需要明确使用哪些指标（如市盈率、市净率、负债率等），并确定每个指标是“越大越好”还是“越小越好”。这是打分的基础。
2.  **为每个指标的数据划分区间并赋值**：将每个指标的所有股票数据，按照数值大小划分成若干个区间。根据指标的好坏方向，为每个区间赋予相应的分数。
3.  **计算单指标得分**：根据每只股票在每个指标上的具体数值，确定其落入哪个区间，从而得到该指标上的得分。
4.  **汇总总分**：将一只股票在所有选定指标上的得分相加，得到其综合总分。
5.  **排序与筛选**：根据综合总分对所有股票进行排序，选择总分最高的若干只股票。

## 打分法实例详解 📝

为了更清晰地理解，我们通过一个具体的例子来演示打分过程。

假设我们有一个股票池（例如沪深300成分股），并选取了四个评价指标：A、B、C、D。其中：
*   **A因子**：越大越好。
*   **B因子**：越大越好。
*   **C因子**：越小越好。
*   **D因子**：越小越好。

我们首先需要为每个指标的数据划分打分区间。这里假设所有指标的数据都已归一化到 `[0, 1]` 的范围。

以下是打分区间的设计：

*   **对于“越大越好”的指标（如A、B）**：
    *   数值在 `[0.8, 1.0]` 区间：得 **5分**
    *   数值在 `[0.6, 0.8)` 区间：得 **4分**
    *   数值在 `[0.4, 0.6)` 区间：得 **3分**
    *   数值在 `[0.2, 0.4)` 区间：得 **2分**
    *   数值在 `[0.0, 0.2)` 区间：得 **1分**

*   **对于“越小越好”的指标（如C、D）**：
    *   数值在 `[0.0, 0.2)` 区间：得 **5分**
    *   数值在 `[0.2, 0.4)` 区间：得 **4分**
    *   数值在 `[0.4, 0.6)` 区间：得 **3分**
    *   数值在 `[0.6, 0.8)` 区间：得 **2分**
    *   数值在 `[0.8, 1.0]` 区间：得 **1分**

现在，我们有两支股票 `id1` 和 `id2`，它们在四个指标上的数值如下：

| 股票ID | A值 | B值 | C值 | D值 |
| :--- | :--- | :--- | :--- | :--- |
| id1 | 0.71 | 0.28 | 0.39 | 0.01 |
| id2 | 0.45 | 0.17 | 0.81 | 0.02 |

接下来，我们根据上述规则为每只股票的每个指标打分：

*   **id1 打分过程**：
    *   **A (0.71)**：属于“越大越好”指标，落在 `[0.6, 0.8)` 区间，得 **4分**。
    *   **B (0.28)**：属于“越大越好”指标，落在 `[0.2, 0.4)` 区间，得 **2分**。
    *   **C (0.39)**：属于“越小越好”指标，落在 `[0.2, 0.4)` 区间，得 **4分**。
    *   **D (0.01)**：属于“越小越好”指标，落在 `[0.0, 0.2)` 区间，得 **5分**。
    *   **总分**：`4 + 2 + 4 + 5 = 15分`

*   **id2 打分过程**：
    *   **A (0.45)**：属于“越大越好”指标，落在 `[0.4, 0.6)` 区间，得 **3分**。
    *   **B (0.17)**：属于“越大越好”指标，落在 `[0.0, 0.2)` 区间，得 **1分**。
    *   **C (0.81)**：属于“越小越好”指标，落在 `[0.8, 1.0]` 区间，得 **1分**。
    *   **D (0.02)**：属于“越小越好”指标，落在 `[0.0, 0.2)` 区间，得 **5分**。
    *   **总分**：`3 + 1 + 1 + 5 = 10分`

计算完成后，`id1` 得15分，`id2` 得10分。如果我们计划选择总分最高的前10只股票，那么 `id1` 将会入选，而 `id2` 可能不会（取决于其他298只股票的得分情况）。

> **补充说明**：除了划分固定区间，打分法还有其他变体。例如，可以直接根据数值在所有股票中的排名（百分位）来打分。如果某指标“越大越好”，那么排名第1的（最大值）给最高分，排名最后的给最低分。这种方法同样有效，核心目标都是得到一个可比较的综合评分。

理解了打分法的核心后，我们可以将其嵌入到一个完整的量化交易策略流程中。以下是构建一个基于月度调仓的选股策略的整体步骤：

![](img/7f78b04f890f3a3a2ec99bc2e7f331bb_9.png)

1.  **确定股票池**：首先，定义策略操作的股票范围。在本例中，我们选择 **沪深300指数成分股** 作为初选股票池。这对应代码中设置 `universe`。

2.  **设置调仓周期**：确定策略的调仓频率。常见的频率有每月、每季度。本例中，我们设置为 **每月调仓一次**。这需要通过设置定时器函数（如 `schedule_function`）来实现，在每月固定的时间点触发调仓逻辑。

3.  **实现调仓函数**：这是策略的核心。我们将定义一个名为 `rebalance` 的函数，它会在每个调仓日被自动调用。该函数内部包含以下子步骤：
    *   **数据获取与预处理**：从数据源查询当前股票池中所有股票在本调仓周期所需的指标数据（如A, B, C, D）。确保数据是最新且经过清洗的。
    *   **指标打分**：对获取到的每一个指标，按照我们预先定义的规则（“越大越好”或“越小越好”）以及区间划分，为每只股票进行打分。这对应于上面实例详解的部分。
    *   **计算综合得分**：将每只股票在所有指标上的得分相加，得到其 **综合总分**。
        ```python
        # 假设 df 是包含各指标得分的DataFrame
        df[‘total_score‘] = df[[‘score_A‘, ‘score_B‘, ‘score_C‘, ‘score_D‘]].sum(axis=1)
        ```
    *   **排序与选股**：根据综合总分对所有股票进行降序排序，并选取排名前N（例如，前10）的股票作为本次调仓的目标持仓。
        ```python
        # 按总分降序排序，并取前10名
        target_stocks = df.nlargest(10, ‘total_score‘)[‘symbol‘].tolist()
        ```
    *   **执行交易**：根据计算出的 `target_stocks`，调整当前持仓，卖出不在名单中的股票，买入名单中新出现的股票，使投资组合与目标名单一致。

4.  **回测与评估**：将上述逻辑在历史数据上运行（回测），观察策略的收益曲线、夏普比率、最大回撤等指标，以评估策略的有效性。
