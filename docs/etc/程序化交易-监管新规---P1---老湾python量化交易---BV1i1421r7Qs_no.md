# 程序化交易监管新规解读：P1：监管背景与量化交易生态入门 🧭

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_1.png)

在本节课中，我们将要学习2023年沪深津三大交易所发布的程序化交易监管新规的核心背景，并澄清关于量化交易的常见误解，同时介绍当前A股市场自动化交易的合规生态。

## 监管新规发布背景

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_3.png)

2023年9月，沪深津三大交易所发布了关于加强程序化交易管理的通知。随后，在端午节前的最后一个交易日，三大交易所正式发布了一份详细的程序化交易监管细则文件。这份文件内容详尽，对程序化交易的各个方面进行了明确规定。

由于细则内容繁多，以下将为大家梳理这份监管文件的核心要点。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_5.png)

## 监管核心：规范高频交易

本次监管的主要目的是对高频交易进行整改。相对于A股市场中的普通投资者，高频交易在技术、信息或交易速度上都存在明显优势。此外，高频交易在某些时点上也存在策略趋同和交易共振等问题，这加剧了市场的波动性。

因此，监管的绝大部分措施旨在定义并规范高频交易，以缩小普通量化交易者与机构主力在量化交易能力上的差距，核心目标是维护市场公平。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_7.png)

## 量化交易的常见误解

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_9.png)

对于不了解量化交易的朋友来说，量化交易常被误解为躲在计算机背后的“黑客”行为。人们认为他们拥有计算能力超强、网速极快的超级计算机，仅凭技术优势就能轻松获利。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_11.png)

但实际上，量化交易只是帮助投资者实现自动化交易的工具。量化交易可以是“印钞机”，也可能是“碎钞机”。无论是机构主力还是散户，如果没有好的交易策略，使用量化交易同样会亏损。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_13.png)

既然自动化交易只是一个工具，建议各位同学以更广阔的视角看待量化交易。成熟的量化交易策略，其应用市场包括但不限于A股、CTA、外汇或某个特定市场，而其底层策略通常由Python编写。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_15.png)

一套成熟的策略，在更换行情接口或交易接口后，就可以跨市场运作。

量化交易也只是学习Python的一个应用场景。办公室里的Excel处理、学校里的作业都可以用Python来完成。

## A股市场自动化交易的现状

想在A股市场实现自动化交易，难点在于新手通常不知道如何找到合法合规且能快速上手的量化交易平台。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_17.png)

但简单之处在于，由于市场竞争激烈，目前普通交易者能够找到低门槛、且配备完善行情数据与下单接口的量化交易平台。有些交易者会使用“按键精灵”这类插件来模拟键盘鼠标操作，在电脑上的炒股软件中下单。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_19.png)

然而，这类插件往往存在各种风险。目前A股市场已有面向普通交易者的成熟自动化交易平台，普通交易者实在没有必要冒险使用这些灰色插件。监管对于A股市场的自动化交易，一直有严格的要求和限制。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_21.png)

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_23.png)

## 合规的量化交易平台

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_25.png)

由于技术开发成本和运维团队等因素，一般券商很难独立开发一套仅供自己客户使用的量化交易平台。因此，市场上出现了由第三方软件公司开发、并销售给券商的量化交易平台，例如PTrade、QMT和掘金等。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_27.png)

这些量化交易平台可以通过券商内部系统，合法合规地对接、封装乃至直达交易所的行情与下单接口。其稳定性与速度，与之前提到的灰色插件完全不在同一个量级。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_29.png)

很多人会误解PTrade和QMT，以为它们和同花顺、通达信这类炒股软件一样，找到链接下载就可以直接使用。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_31.png)

由于PTrade和QMT是券商向第三方软件公司采购的服务，并且需要通过券商的内部系统来对接交易所的行情数据与下单接口，因此，即便有下载链接，也需要获得券商提供的正式账号或测试账号，才能够登录使用。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_33.png)

这里需要重点强调：PTrade和QMT的一切操作合法合规，券商也不能随意中断你的交易。所以，想下载PTrade或QMT的同学，可以先联系自己在券商的客户经理。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_35.png)

如果你的客户经理不熟悉PTrade或QMT，或者你所在的券商并未采购这些平台，又或者申请门槛太高，同学可以在后台私信我。

## 不同市场的交易生态

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_37.png)

顺便提一下，不同市场的自动化交易有着不同的生态。在某个特定的市场，交易所的行情数据与交易接口是完全开放的API，交易者可以自行部署自动化交易策略。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_39.png)

在学习Python量化交易的过程中，新手同学常常会遇到各种各样的“坑”。这些“坑”包括Python语法、量化框架、市场交易品种以及交易规则等。这些小问题可能会困扰新手好几个月，但有时只需要有人适时提点几句，问题就可以迎刃而解，从而加快学习速度。

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_41.png)

要知道，A股市场不仅能投资股票，还能做ETF、可转债、国债、商品期货甚至国外指数。如果你能从更高的视角看待，就会发现自动化交易策略可以有很多不同的应用和变化，不要总是固守自己熟悉的那一小块领域。

## 总结

![](img/fd8c14157ce5c1c6c7767cb193c98a5b_43.png)

本节课中，我们一起学习了2023年程序化交易监管新规出台的背景与核心目标，即重点规范高频交易以维护市场公平。我们澄清了量化交易作为自动化工具的本质，并介绍了在A股市场进行合规自动化交易的途径，主要是通过券商提供的PTrade、QMT等专业平台。最后，我们提醒大家要以开放的视野看待量化交易在不同市场的应用，并鼓励在遇到学习瓶颈时积极寻求指导。
