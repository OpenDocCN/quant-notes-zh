## 拼接数据到总表 🔗

![](img/13b126f8059bd3c363a22d0b39b6fb6e_54.png)

在每次循环中构建好临时`DataFrame`后，我们需要将其拼接到之前初始化好的总`DataFrame`中。这可以通过`pandas.concat`函数实现。

![](img/13b126f8059bd3c363a22d0b39b6fb6e_56.png)

以下是完成数据拼接的完整流程：

![](img/13b126f8059bd3c363a22d0b39b6fb6e_58.png)

![](img/13b126f8059bd3c363a22d0b39b6fb6e_60.png)

```python
import pandas as pd

![](img/13b126f8059bd3c363a22d0b39b6fb6e_62.png)

![](img/13b126f8059bd3c363a22d0b39b6fb6e_64.png)

# 初始化一个空的DataFrame作为总表
total_df = pd.DataFrame()

![](img/13b126f8059bd3c363a22d0b39b6fb6e_66.png)

# 假设 dates 是日期序列，get_daily_factor 是获取每日数据的函数
for i in range(len(dates)):
    # 获取第i天的因子数据
    daily_factor_value = get_daily_factor(i)

    # 构建临时DataFrame
    temp_df = pd.DataFrame({
        'factor_value': daily_factor_value,
        'date': dates[i]
    })

    # 关键步骤：将临时DataFrame拼接到总表，并重新赋值给total_df
    total_df = pd.concat([total_df, temp_df], ignore_index=True)

![](img/13b126f8059bd3c363a22d0b39b6fb6e_68.png)

![](img/13b126f8059bd3c363a22d0b39b6fb6e_70.png)

# 循环结束后，查看总表的前几行
print(total_df.head())
```

![](img/13b126f8059bd3c363a22d0b39b6fb6e_72.png)

![](img/13b126f8059bd3c363a22d0b39b6fb6e_74.png)

**请注意**：`pd.concat`操作会返回一个新的`DataFrame`，必须将其赋值回`total_df`变量，否则拼接无效。

![](img/13b126f8059bd3c363a22d0b39b6fb6e_76.png)

## 总结 🎯

![](img/13b126f8059bd3c363a22d0b39b6fb6e_78.png)

本节课中我们一起学习了如何获取一个给定时间区间内的全部量化数据。我们通过以下步骤实现了这一目标：
1.  **遍历日期**：使用`for`循环遍历目标日期序列。
2.  **逐日获取**：在循环内调用接口，获取每一天的数据。
3.  **构建记录**：将单日的数据和日期封装成一个临时的`DataFrame`。
4.  **汇总成表**：使用`pd.concat`函数将所有临时`DataFrame`拼接成一个总表。

![](img/13b126f8059bd3c363a22d0b39b6fb6e_80.png)

![](img/13b126f8059bd3c363a22d0b39b6fb6e_82.png)

这个过程虽然需要手动编写循环，但它是批量获取时间序列数据的标准方法。现在，你得到了一个包含日期和对应因子值的完整`DataFrame`，可以用于后续的回测或分析。
