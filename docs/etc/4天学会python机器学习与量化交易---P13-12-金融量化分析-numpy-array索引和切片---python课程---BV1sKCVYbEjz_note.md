标量在数学上指单个的数值。NumPy数组可以与标量进行逐元素的运算。

![](img/116fb4ea2b785494ba1dc630b60516f0_10.png)

以下是数组与标量运算的示例：

![](img/116fb4ea2b785494ba1dc630b60516f0_12.png)

```python
import numpy as np
A = np.arange(10)  # 创建一个0到9的数组

print(A * 3)  # 每个元素乘以3
print(3 / A)  # 用3除以每个元素（注意：除以0会产生警告）
```

## 数组的索引

索引用于访问数组中的单个元素。其规则与Python列表类似。

![](img/116fb4ea2b785494ba1dc630b60516f0_29.png)

### 一维数组索引

![](img/116fb4ea2b785494ba1dc630b60516f0_31.png)

对于一维数组，使用单个索引即可访问元素。

```python
print(A[0])  # 访问第一个元素
```

![](img/116fb4ea2b785494ba1dc630b60516f0_33.png)

### 二维数组索引

对于二维数组，索引需要指定行和列。有两种等价的写法。

以下是二维数组索引的示例：

```python
# 创建一个3行5列的二维数组
A_2d = np.arange(15).reshape(3, 5)
print(A_2d[0][0])  # 方法一：先取行，再取列
print(A_2d[0, 0])  # 方法二：使用逗号分隔行和列索引
```

![](img/116fb4ea2b785494ba1dc630b60516f0_35.png)

## 数组的切片

切片用于获取数组的一个子集。NumPy数组的切片与列表有相似之处，但也有一个关键区别。

### 一维数组切片

一维数组的切片语法与列表相同：`[start:stop:step]`，遵循“前闭后开”原则。

```python
C = A[0:4]  # 获取索引0到3的元素
print(C)
```

### 切片与视图

NumPy数组切片的关键特性是：**切片返回的是原数组数据的视图（view），而非副本（copy）**。这意味着修改切片会直接影响原数组。

以下是视图行为的示例：

```python
A = np.arange(10)
B = list(range(10))  # 创建一个普通列表作为对比
C = A[0:4]  # A的切片
D = B[0:4]  # B的切片

C[0] = 20  # 修改NumPy数组切片
D[0] = 20  # 修改列表切片

print(A[0])  # 输出20，原数组被修改
print(B[0])  # 输出0，原列表未被修改
```

这种行为是为了在处理大型数据时节省内存和时间。如果希望获得一个独立的副本，需要使用`copy()`方法。

```python
C_copy = A[0:4].copy()
C_copy[0] = 100
print(A[0])  # 输出20，原数组不受影响
```

### 二维数组切片

对于二维数组，可以使用逗号分隔对行和列分别进行切片，语法为`[行切片, 列切片]`。

以下是二维数组切片的示例：

```python
A_2d = np.arange(15).reshape(3, 5)
# 获取第0、1行，第0、1列的元素（即左上角2x2的子矩阵）
sub_matrix = A_2d[0:2, 0:2]
print(sub_matrix)

# 获取第1、2行，第2到最后一列的元素
sub_matrix2 = A_2d[1:3, 2:]
print(sub_matrix2)
```

记住，逗号左边是行切片，右边是列切片。这种灵活的切片方式是处理多维数据（如金融时间序列或面板数据）的强大工具。
