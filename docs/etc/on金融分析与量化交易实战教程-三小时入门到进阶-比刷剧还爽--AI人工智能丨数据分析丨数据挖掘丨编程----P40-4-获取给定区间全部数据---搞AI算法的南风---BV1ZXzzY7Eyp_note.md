## 遍历日期序列

![](img/cf62a450e853627dd450b23540815381_11.png)

我们的目标是处理一个日期序列。要获取其中每一天的数据，最直接的方法是使用循环。我们可以通过 `for` 循环遍历序列中的每一个日期。

![](img/cf62a450e853627dd450b23540815381_13.png)

以下是实现遍历的核心思路：
```python
# 假设 dates 是一个日期序列
for i in range(len(dates)):
    current_date = dates[i]
    # 在此处获取并处理 current_date 的数据
```

## 获取每日数据并处理

![](img/cf62a450e853627dd450b23540815381_15.png)

在循环体内，我们需要获取每一天对应的因子数据。这个数据通常是多维数组，我们需要提取出对我们有用的部分。

![](img/cf62a450e853627dd450b23540815381_17.png)

![](img/cf62a450e853627dd450b23540815381_19.png)

以下是处理每日数据的步骤：
1.  根据循环索引 `i` 获取当日的原始数据。
2.  原始数据是一个三维数组，我们通常只关心第三个维度的全部数据。
3.  提取有用数据，代码示例如下：
    ```python
    # 假设 daily_raw_data 是通过接口获取的当日原始数据
    useful_data = daily_raw_data[0, 2, :]  # 提取特定维度的数据
    ```

![](img/cf62a450e853627dd450b23540815381_21.png)

## 构建并汇总数据表

我们不仅需要获取数据，还需要将其组织起来。理想的数据表应包含日期、股票名称和对应的因子值三列。

![](img/cf62a450e853627dd450b23540815381_23.png)

以下是构建总数据表的流程：
1.  在循环开始前，初始化一个空的 `DataFrame` 作为总表。
2.  在每次循环中，为当日数据创建一个临时的 `DataFrame`，包含“日期”和“因子值”两列。
3.  将这个临时 `DataFrame` 拼接到总表中。

![](img/cf62a450e853627dd450b23540815381_25.png)

![](img/cf62a450e853627dd450b23540815381_27.png)

以下是关键代码示例：
```python
import pandas as pd

# 初始化总表
total_df = pd.DataFrame()

![](img/cf62a450e853627dd450b23540815381_29.png)

for i in range(len(dates)):
    # 1. 获取当日有用数据
    useful_data = get_daily_data(i)  # 假设的函数

    # 2. 创建当日数据表
    daily_df = pd.DataFrame()
    daily_df['因子值'] = useful_data
    daily_df['日期'] = dates[i]

    # 3. 拼接至总表
    total_df = pd.concat([total_df, daily_df], ignore_index=True)
```

![](img/cf62a450e853627dd450b23540815381_31.png)

![](img/cf62a450e853627dd450b23540815381_33.png)

**注意**：`pd.concat` 操作后必须将结果重新赋值给变量，否则拼接无效。对于较长时间区间的数据，此过程可能需要一些时间。
