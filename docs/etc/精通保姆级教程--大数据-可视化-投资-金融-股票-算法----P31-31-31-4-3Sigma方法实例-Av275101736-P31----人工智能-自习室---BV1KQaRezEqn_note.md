## 正态分布与3-Sigma原理 📈

上一节我们介绍了分位数和中位数绝对偏差去极值法，本节中我们来看看基于正态分布的3-Sigma方法。

首先，我们需要理解正态分布。正态分布，也称为高斯分布，是一种常见的连续概率分布。在正态分布中，数据大多集中在均值附近，离均值越远的数据点出现的概率越低。

![](img/c85290e424606ae6bd05ad3e3c4e2680_1.png)

如果理解正态分布有困难，可以将其想象为一个“正常”的分布。例如，去银行贷款，银行多借或少借你少量金额（如三块五块）是正常情况。但如果多借或少借你10万8万，这就属于异常情况了。

在标准正态分布中，均值为0。我们可以将数据点视为围绕均值的误差或波动。数据点落在均值附近（红色阴影区域）的概率较高，代表这些是正常数据。而数据点落在远离均值的区域（如两侧的尾部）的概率则非常低，这些点很可能就是我们需要处理的极值或离群点。

3-Sigma方法正是基于这一思想。它利用正态分布的特性，设定一个以均值为中心、上下各延伸3倍标准差（σ）的区间。根据统计学经验，在正态分布中，约有99.7%的数据会落在这个区间内。

**公式**：
*   区间上限 = 均值 + 3 × 标准差
*   区间下限 = 均值 - 3 × 标准差

因此，3-Sigma方法认为，落在这个区间内的数据是“正常”的，而落在区间外的数据（约0.3%）则被视为“极值”，需要进行规范化处理。

理解了原理后，我们来看看如何在Python中实现3-Sigma方法。其核心是计算数据的均值和标准差，然后根据指定的倍数N（例如N=3）来确定上下限。

![](img/c85290e424606ae6bd05ad3e3c4e2680_9.png)

以下是实现步骤：

![](img/c85290e424606ae6bd05ad3e3c4e2680_11.png)

首先，我们需要计算选定数据列的均值和标准差。

![](img/c85290e424606ae6bd05ad3e3c4e2680_13.png)

```python
# 计算均值和标准差
mean = data_column.mean()   # 均值
std = data_column.std()     # 标准差
```

![](img/c85290e424606ae6bd05ad3e3c4e2680_15.png)

接着，根据均值和标准差计算数据的上下限。

![](img/c85290e424606ae6bd05ad3e3c4e2680_17.png)

```python
# 设定倍数 N，例如 N=3 即为 3-Sigma
N = 3

![](img/c85290e424606ae6bd05ad3e3c4e2680_19.png)

![](img/c85290e424606ae6bd05ad3e3c4e2680_21.png)

# 计算上限和下限
upper_limit = mean + N * std
lower_limit = mean - N * std
```

![](img/c85290e424606ae6bd05ad3e3c4e2680_23.png)

![](img/c85290e424606ae6bd05ad3e3c4e2680_25.png)

最后，对原始数据进行规范化，将所有值限制在计算出的上下限之内。

![](img/c85290e424606ae6bd05ad3e3c4e2680_27.png)

```python
# 将超出上限的值设为上限，低于下限的值设为下限
normalized_data = data_column.clip(lower=lower_limit, upper=upper_limit)
```

![](img/c85290e424606ae6bd05ad3e3c4e2680_29.png)

![](img/c85290e424606ae6bd05ad3e3c4e2680_31.png)

## 方法总结与选择 🤔

![](img/c85290e424606ae6bd05ad3e3c4e2680_37.png)

本节课中我们一起学习了三种主要的去极值方法。需要强调的是，“去极值”并非简单地将异常数据丢弃，而是通过定义合理的上界和下界，将所有数据规范到这个范围内。

![](img/c85290e424606ae6bd05ad3e3c4e2680_39.png)

以下是三种方法的简要回顾：

![](img/c85290e424606ae6bd05ad3e3c4e2680_41.png)

![](img/c85290e424606ae6bd05ad3e3c4e2680_43.png)

*   **分位数法**：基于数据自身的分布百分位点设定边界。
*   **中位数绝对偏差法**：利用中位数和绝对偏差的稳健统计量来识别离群点。
*   **3-Sigma法**：假设数据服从正态分布，利用均值和标准差来划定正常数据范围。

![](img/c85290e424606ae6bd05ad3e3c4e2680_45.png)

在实际应用中，选择哪种方法取决于具体的任务需求和数据特性。你可以根据实际情况进行实验，选择最合适的数据预处理策略。这三种方法都是数据预处理“三步走”中重要的第一步，为后续分析奠定基础。
