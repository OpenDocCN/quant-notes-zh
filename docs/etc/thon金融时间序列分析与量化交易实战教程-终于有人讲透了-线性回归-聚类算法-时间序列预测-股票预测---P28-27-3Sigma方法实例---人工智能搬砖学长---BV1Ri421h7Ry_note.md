## 概述
在本节课中，我们将要学习一种名为“3Sigma”的数据去极值方法。该方法基于统计学中的正态分布原理，用于识别并处理数据中的异常值，是金融数据分析中常用的预处理技术。

---

## 3Sigma方法原理

上一节我们介绍了其他去极值方法，本节中我们来看看基于统计学原理的3Sigma方法。

3Sigma方法的原理非常简单。请看下图，图中绘制了一个正态分布曲线。如果理解正态分布有困难，可以将其视为一种“正常”的分布模式。例如，在银行贷款场景中，银行多借或少借你少量金额是正常情况，但多借或少借你十万八万则属于异常情况。

![](img/8fd9b34947ae6dad0c761745f50bb63e_1.png)

![](img/8fd9b34947ae6dad0c761745f50bb63e_1.png)

我们来看这个走势。以标准正态分布为例，图中有一个均值为0的点。这好比贷款误差，可能多借（向右偏移）也可能少借（向左偏移）。图中红色阴影区域面积较大，代表这些情况发生的概率较高。当区间越往外扩展时，事件发生的概率就越低。

我们现在要做的是去极值。极值可能落在哪里？落在红色区域（均值附近）吗？这里发生的是大概率事件，属于正常数据。极值通常落在远离均值的地方。离均值越远、偏差越大的点，越有可能是离群点。

那么，什么是三倍西格玛（3Sigma）？请看下图。

![](img/8fd9b34947ae6dad0c761745f50bb63e_3.png)

![](img/8fd9b34947ae6dad0c761745f50bb63e_3.png)

图中，X代表均值。我们标记了均值加减一倍、两倍、三倍标准差（Sigma）的区间。可以发现，区间选得越大（例如加减三倍标准差），数据落在左右极端区域的可能性就越小。

3Sigma方法的思路如下：假设我们有一份数据，并且可以通过某种变换使其理论上服从高斯分布（正态分布）。当我们指定正负三倍标准差的范围时，这个区间内包含了多少数据呢？统计学的结论是：**约99.7%**的数据会落在这个区间内。

因此，如果我们认为数据中最正常的99.7%是有效数据，剩下的0.3%作为极值进行处理，这就是3Sigma方法。在高斯分布中，确定上下限只需要两个值：**均值（mean）**和**标准差（std）**。

![](img/8fd9b34947ae6dad0c761745f50bb63e_5.png)

有了均值和标准差，我们就能计算出上限和下限。

![](img/8fd9b34947ae6dad0c761745f50bb63e_5.png)

![](img/8fd9b34947ae6dad0c761745f50bb63e_7.png)

![](img/8fd9b34947ae6dad0c761745f50bb63e_7.png)

## 3Sigma方法代码实现

![](img/8fd9b34947ae6dad0c761745f50bb63e_9.png)

理解了原理后，我们来看看如何用代码实现3Sigma方法。

![](img/8fd9b34947ae6dad0c761745f50bb63e_11.png)

实现非常简单。我们创建一个函数，其逻辑与之前的方法类似。

![](img/8fd9b34947ae6dad0c761745f50bb63e_13.png)

首先，我们需要计算指定数据列的均值和标准差。

![](img/8fd9b34947ae6dad0c761745f50bb63e_15.png)

```python
# 计算均值和标准差
mean = data_series.mean()
std = data_series.std()
```

接着，根据指定的倍数N（例如N=3代表3Sigma），计算上下限：

![](img/8fd9b34947ae6dad0c761745f50bb63e_17.png)

```python
# 计算上限和下限
upper_limit = mean + N * std
lower_limit = mean - N * std
```

![](img/8fd9b34947ae6dad0c761745f50bb63e_19.png)

最后，函数返回这两个界限值，用于后续的数据裁剪（将所有数据规范到`[lower_limit, upper_limit]`区间内）。

以下是完整的函数示例：

![](img/8fd9b34947ae6dad0c761745f50bb63e_21.png)

```python
def three_sigma_method(data_series, n=3):
    """
    使用3Sigma方法计算去极值的上下限。
    参数:
        data_series: pandas Series，待处理的数据列。
        n: int，标准差的倍数，默认为3。
    返回:
        lower_limit, upper_limit: 计算出的下限和上限。
    """
    mean = data_series.mean()
    std = data_series.std()
    
    upper_limit = mean + n * std
    lower_limit = mean - n * std
    
    return lower_limit, upper_limit
```

![](img/8fd9b34947ae6dad0c761745f50bb63e_23.png)

我们可以调用这个函数，并传入数据和倍数N。例如，如果我们希望保留约68.3%的数据（对应N=1），可以这样调用：

![](img/8fd9b34947ae6dad0c761745f50bb63e_25.png)

```python
lower, upper = three_sigma_method(data['column_name'], n=1)
```

![](img/8fd9b34947ae6dad0c761745f50bb63e_27.png)

然后，我们可以根据得到的`lower`和`upper`值，对原始数据进行裁剪，将所有小于下限的值设为下限，所有大于上限的值设为上限。

![](img/8fd9b34947ae6dad0c761745f50bb63e_29.png)

```python
clipped_data = data['column_name'].clip(lower=lower, upper=upper)
```

通过绘图，我们可以直观地看到处理前后的数据对比，以及上下限的位置。

## 方法选择与核心要点

![](img/8fd9b34947ae6dad0c761745f50bb63e_39.png)

主要就是三种去极值方法，我们已经全部介绍完毕。

![](img/8fd9b34947ae6dad0c761745f50bb63e_41.png)

以下是选择方法时需要考虑的几点：

![](img/8fd9b34947ae6dad0c761745f50bb63e_43.png)

*   **方法无绝对优劣**：使用哪种方法都可以，取决于实际任务需求。这些方法都是预处理中的不同策略。
*   **鼓励实验**：大家可以自行尝试这些方法，观察它们在不同数据上的效果。

**核心要点**：
我们的第一步操作叫做“去极值”。需要注意，**去极值不是直接丢弃数据，而是对其进行规范化**。具体做法是定义好一个下界和一个上界，然后让所有数值都保持在我们指定好的下界和上界之内。
