## 夏普比率的定义与作用
上一节我们介绍了投资回报率，本节中我们来看看如何衡量“性价比”更高的投资。夏普比率描述的是：对于承担的每一单位风险，我们能获得多少超额收益。

我们可以用一个生活中的例子来理解：一份在叙利亚日薪3万的雇佣兵工作，其高薪对应的是极高的生命风险。在投资中也是如此，高收益往往伴随高风险。夏普比率就是用来评估，为了获得一定的超额收益，我们所承担的风险是否“值得”。指标越高，意味着单位风险带来的收益越高，该投资组合的“性价比”就越好。

![](img/5ac04f0a7b0c0cf7f8f2344db175bcb9_2.png)

在选股或构建投资组合时，如果单从夏普比率这一指标看，我们应选择**夏普比率更高**的那个，因为它代表了更好的风险调整后收益。

## 夏普比率的计算原理
理解了夏普比率的作用后，我们来看看它是如何计算的。这涉及到两个关键概念：投资组合的收益率和无风险收益率。

![](img/5ac04f0a7b0c0cf7f8f2344db175bcb9_4.png)

在金融领域，通常将国债等几乎无风险的资产收益率作为基准，称为无风险收益率（例如年化3%或5%）。当我们投资股票或基金时，获得的收益率往往高于这个无风险收益率，高出的这部分就是**超额收益**。

夏普比率的核心思想是：用投资组合的**超额收益**，除以其收益的**波动性（标准差）**。波动性代表了风险，标准差越大，说明收益波动越剧烈，风险越高。

因此，夏普比率的公式可以表示为：
`夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差`

## 使用Python计算夏普比率
接下来，我们通过Python代码来实际计算夏普比率。假设我们已经有了一个包含多只股票历史收益率的数据。

首先，我们需要处理数据并计算每日收益率。这里需要注意，原始数据可能存在缺失值（如停牌），我们需要进行填充处理。

```python
# 假设‘returns_df’是一个DataFrame，包含了多只股票的每日收益率
# 处理缺失值：使用前一天的收益率填充当天的缺失值
returns_df_filled = returns_df.fillna(method='ffill')

![](img/5ac04f0a7b0c0cf7f8f2344db175bcb9_6.png)

# 设置年化无风险收益率，例如5%
risk_free_rate = 0.05
# 计算年化夏普比率，假设一年有250个交易日
trading_days_per_year = 250

# 计算每只股票的平均年化收益率
annualized_return = returns_df_filled.mean() * trading_days_per_year

![](img/5ac04f0a7b0c0cf7f8f2344db175bcb9_8.png)

# 计算每只股票收益率的标准差（年化）
annualized_std = returns_df_filled.std() * np.sqrt(trading_days_per_year)

# 计算夏普比率
sharpe_ratio = (annualized_return - risk_free_rate) / annualized_std

print(sharpe_ratio)
```

以下是计算步骤的关键点列表：
1.  **数据预处理**：使用`.fillna(method=‘ffill’)`方法向前填充缺失的收益率数据，确保时间序列连续。
2.  **计算年化收益率**：将日收益率均值乘以年交易天数（通常为250天），得到年化收益率。
3.  **计算年化标准差**：将日收益率的标准差乘以年交易天数的平方根，得到年化波动率（风险）。
4.  **应用公式**：将（年化收益率 - 无风险收益率）除以年化标准差，得到最终的夏普比率。

运行代码后，我们会得到每只股票对应的夏普比率。数值越大越好。例如，如果“中兴通讯”的夏普比率最高，则意味着在这组股票中，投资它承受每单位风险所获得的超额回报是最优的。负的夏普比率通常意味着其收益尚不及无风险收益，不是好的选择。
