## 计算短期与长期移动平均线

上一节我们介绍了移动平均线的概念，本节中我们来看看如何用代码具体计算。

首先，我们需要选定分析的股票数据。这里我们以苹果公司（AAPL）的股价为例。

![](img/242ed186b2446332d413c6d8d906e404_11.png)

![](img/242ed186b2446332d413c6d8d906e404_13.png)

![](img/242ed186b2446332d413c6d8d906e404_15.png)

以下是计算移动平均线的步骤：

![](img/242ed186b2446332d413c6d8d906e404_17.png)

1.  **导入数据**：加载苹果公司的历史股价数据。
2.  **定义窗口**：短期平均线通常使用较小的窗口（如5天、10天），长期平均线使用较大的窗口（如20天、30天）。由于我们的数据时间跨度较长，为让图表更清晰，我们将使用10天和30天作为示例。
3.  **进行计算**：使用Pandas的滚动窗口函数计算指定窗口内的股价平均值。

![](img/242ed186b2446332d413c6d8d906e404_19.png)

核心计算代码如下：

![](img/242ed186b2446332d413c6d8d906e404_21.png)

![](img/242ed186b2446332d413c6d8d906e404_23.png)

```python
# 假设 df 是包含苹果股价的DataFrame，其中‘Close’为收盘价列
# 计算10日短期移动平均线
df[‘M1’] = df[‘Close’].rolling(window=10).mean()
# 计算30日长期移动平均线
df[‘M2’] = df[‘Close’].rolling(window=30).mean()
```

![](img/242ed186b2446332d413c6d8d906e404_27.png)

## 识别黄金交叉与死亡交叉

在图表上，我们可以清晰地看到短期均线（M1）和长期均线（M2）的交叉情况。

![](img/242ed186b2446332d413c6d8d906e404_41.png)

![](img/242ed186b2446332d413c6d8d906e404_43.png)

以下是两种关键交叉的定义：

![](img/242ed186b2446332d413c6d8d906e404_45.png)

*   **死亡交叉**：当短期移动平均线从上方向下穿越长期移动平均线时，形成死亡交叉。这通常被视为下跌趋势开始的信号，提示投资者考虑卖出或观望。
*   **黄金交叉**：当短期移动平均线从下方向上穿越长期移动平均线时，形成黄金交叉。这通常被视为上涨趋势开始的信号，提示投资者考虑买入。

![](img/242ed186b2446332d413c6d8d906e404_47.png)

## 使用信号指标进行增强分析

![](img/242ed186b2446332d413c6d8d906e404_49.png)

![](img/242ed186b2446332d413c6d8d906e404_51.png)

为了让交叉信号更加一目了然，我们可以创建一个信号指标来明确标记短期均线是高于还是低于长期均线。

![](img/242ed186b2446332d413c6d8d906e404_53.png)

![](img/242ed186b2446332d413c6d8d906e404_55.png)

我们可以使用`np.where`函数来生成这个信号列：

```python
import numpy as np
# 生成信号：短期均线大于长期均线时为1，小于时为-1
df[‘position’] = np.where(df[‘M1’] > df[‘M2’], 1, -1)
```

![](img/242ed186b2446332d413c6d8d906e404_57.png)

为了在图表中清晰地展示这个信号，我们可以使用双Y轴进行绘图。左边Y轴显示股价和均线，右边Y轴显示信号指标（1或-1）。

![](img/242ed186b2446332d413c6d8d906e404_59.png)

![](img/242ed186b2446332d413c6d8d906e404_61.png)

```python
fig, ax1 = plt.subplots(figsize=(12,6))
ax1.plot(df[‘Close’], label=‘Price’, color=‘blue’, alpha=0.5)
ax1.plot(df[‘M1’], label=‘10-Day MA’, color=‘green’)
ax1.plot(df[‘M2’], label=‘30-Day MA’, color=‘red’)
ax1.set_ylabel(‘Price’)
ax1.legend(loc=‘upper left’)

ax2 = ax1.twinx() # 创建共享X轴的第二个Y轴
ax2.plot(df[‘position’], label=‘Signal’, color=‘purple’, linestyle=‘--’)
ax2.set_ylabel(‘Signal (1/-1)’)
ax2.legend(loc=‘upper right’)
plt.show()
```

![](img/242ed186b2446332d413c6d8d906e404_63.png)

在生成的图表中，当紫色信号线从-1跃升到1时，对应图表中的“黄金交叉”点；当从1下降到-1时，则对应“死亡交叉”点。
