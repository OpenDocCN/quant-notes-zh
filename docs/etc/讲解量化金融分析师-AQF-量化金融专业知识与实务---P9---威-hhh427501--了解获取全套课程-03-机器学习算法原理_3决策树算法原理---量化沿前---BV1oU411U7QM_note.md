## 决策树的基本思想

上一节我们介绍了支持向量机等算法，本节中我们来看看决策树如何工作。决策树的核心思想是通过一系列简单的“是/否”问题，对数据进行层层划分，最终达到分类或回归的目的。

### 一个简单的分类例子

假设我们有一个音乐推荐场景，需要根据“节奏”（快/慢）和“音调”（高/低）两个特征来判断用户是否喜欢某首歌。数据分布如下图所示，用户只喜欢“节奏快”且“音调低”的音乐（图中圈出部分）。

![](img/4c6b9270f562817a9722e5ff0934d51c_0.png)

如果用线性分类器，可能需要多条直线才能划分出这个区域，但决策树可以轻松解决。

以下是构建决策树进行判断的步骤：
1.  **第一个问题**：音调是否低？
    *   如果 **否**（音调高），则直接判断为 **不喜欢**。
    *   如果 **是**（音调低），则进入下一个问题。
2.  **第二个问题**：节奏是否快？
    *   如果 **是**（节奏快），则判断为 **喜欢**。
    *   如果 **否**（节奏慢），则判断为 **不喜欢**。

通过这样两个问题，我们就完成了一个非线性分类。

### 一个更复杂的分类例子

考虑一个更复杂的二维数据分布，如下图所示。我们需要将图中的圆圈（喜欢）和叉号（不喜欢）分开。

![](img/4c6b9270f562817a9722e5ff0934d51c_1.png)

决策树可以这样构建：
1.  **第一个问题**：X 是否小于 3？
    *   **是**：进入左分支。
        *   **第二个问题**：Y 是否小于 2？
            *   **是**：分类为 **不喜欢**（叉号）。
            *   **否**：分类为 **喜欢**（圆圈）。
    *   **否**（X ≥ 3）：进入右分支。
        *   **第二个问题**：Y 是否小于 4？
            *   **是**：分类为 **不喜欢**（叉号）。
            *   **否**：分类为 **喜欢**（圆圈）。

通过这棵树，复杂的非线性边界被一系列简单的判断条件所描述。

## 决策树在量化投资中的应用

在量化投资中，我们可以使用类似的方法。例如，根据股票的“市值”和“板块”特征来预测其涨跌。

假设我们有如下数据样本：

| 市值 | 板块 | 涨跌 |
| :--- | :--- | :--- |
| 大 | 消费 | 涨 |
| 大 | 周期 | 涨 |
| 中 | 消费 | 涨 |
| 中 | 成长 | 跌 |
| 小 | 消费 | 跌 |
| 小 | 周期 | 跌 |

算法可以自动学习并生成如下决策树：
1.  **是否为大市值？**
    *   **是** → 预测为 **涨**。
    *   **否** → 进入下一问题。
2.  **是否为消费板块？**
    *   **否** → 预测为 **跌**。
    *   **是** → 进入下一问题。
3.  **是否为中市值？**
    *   **是** → 预测为 **涨**。
    *   **否**（即小市值）→ 预测为 **跌**。

当新数据（例如一只“小市值消费股”）进入时，模型会沿着这棵树进行判断，最终输出预测结果。

### 如何选择最佳划分点？

你可能会问，为什么第一个问题要问“是否为大市值”，而不是“是否为消费板块”？这涉及到一个核心概念：**信息增益**。算法会选择那个能最大程度减少数据“不确定性”或“混乱度”的特征作为节点划分依据。信息增益高的特征会被优先用来提问。其计算通常基于**熵**或**基尼不纯度**等公式，例如：

**熵的公式**：`Entropy = -Σ(p_i * log2(p_i))`
其中 `p_i` 是第 i 类样本所占的比例。

**信息增益**：`Information Gain = Entropy(父节点) - Σ( (N_child / N_parent) * Entropy(子节点) )`
算法会选择信息增益最大的特征进行划分。

## 从单棵树到森林：集成学习

单棵决策树能力有限，容易过拟合。将多棵决策树组合起来，可以构建更强大、更稳定的模型，这种方法称为**集成学习**。主要有两种思路：**并行** 和 **串行**。

### 并行方法：随机森林 🌲🌲🌲

随机森林是**并行**集成方法的代表。其核心思想是“**投票**”或“**取平均**”。

**工作原理**：
1.  从原始数据集中**有放回地随机抽样**（Bootstrap Sampling），生成多个不同的子数据集。
2.  用每个子数据集独立训练一棵决策树。
3.  对于分类任务，所有树进行**投票**，少数服从多数；对于回归任务，取所有树的**预测平均值**。

**优势**：
*   通过随机抽样和随机选择特征，确保了树之间的差异性，降低了模型整体过拟合的风险。
*   效果通常比单棵决策树好很多。

### 串行方法：AdaBoost

![](img/4c6b9270f562817a9722e5ff0934d51c_9.png)

AdaBoost是**串行**集成方法的代表。其核心思想是“**关注错误**”，即后续的树重点纠正前面树分错的样本。

![](img/4c6b9270f562817a9722e5ff0934d51c_11.png)

**工作原理**：
1.  初始时，所有训练样本具有相同的权重。
2.  训练第一棵决策树。
3.  根据这棵树的分类结果，**增加分错样本的权重**，降低分对样本的权重。这使得在训练下一棵树时，模型会更关注之前分错的“难题”。
4.  重复步骤2-3，依次训练多棵树。
5.  最终预测时，**错误率越低的树，其“话语权”（权重）越高**，将所有树的加权结果结合起来得到最终预测。

**优势**：
*   通过自适应地调整样本权重，将弱分类器提升为强分类器。
*   对难以分类的样本有很好的处理能力。

本节课中我们一起学习了决策树算法及其在量化金融中的应用。
*   我们首先了解了决策树如何通过一系列“是/否”问题解决复杂的非线性分类问题。
*   接着，我们探讨了决策树构建的核心——**信息增益**，它帮助算法选择最优的划分特征。
*   最后，我们学习了如何通过**集成学习**提升模型性能：**随机森林**通过并行构建多棵树并投票来提升稳定性和准确率；**AdaBoost**通过串行训练，让后续模型聚焦于前序模型的错误，以提升整体性能。
