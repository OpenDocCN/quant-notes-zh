## 数据读取与初步探索

首先，我们需要读取股票数据。这里我们选择微软（MSFT）的股票历史数据。

![](img/120e4d73254c8705cbd67b3a7286e69c_1.png)

```python
# 示例：使用yfinance库获取微软股票数据
import yfinance as yf
msft = yf.Ticker("MSFT")
hist = msft.history(period="max")
print(hist.head())
```

执行上述代码后，我们会得到一个包含日期（`Date`）和多个金融指标（如开盘价、收盘价、成交量等）的`DataFrame`。我们主要关注的是收盘价（`Close`），这是我们后续要预测的目标变量。

初次运行代码时，可能会因为需要下载和加载库而稍慢。如果代码报错，通常是因为未安装`yfinance`、`pandas`等必要的库，需要先安装它们。

## 数据预处理与关键指标提取

读取数据后，我们需要进行一些预处理，并提取关键信息。

上一节我们介绍了如何读取数据，本节中我们来看看如何对数据进行预处理并提取关键指标。

以下是数据预处理的关键步骤：
1.  **重置索引**：将日期列设置为索引，便于时间序列操作。
2.  **指定目标变量**：明确我们要预测的指标，例如收盘价 `Y = df[‘Close’]`。
3.  **计算衍生指标**：例如每日价格变动 `df[‘Daily_Change’] = df[‘Close’] - df[‘Open’]`。
4.  **确定时间范围**：找出数据的最早日期（`start_date = df.index.min()`）和最晚日期（`end_date = df.index.max()`）。
5.  **计算极值**：找出历史最高价、最低价及其发生的日期。

通过这些步骤，我们可以对数据有一个全面的初步了解，例如知道微软股票数据从1986年3月13日开始，整体呈上升趋势。

## 时间序列可视化

拿到并处理好数据后，下一步通常是将其可视化，以直观地观察趋势和模式。

以下是绘制时间序列图的基本方法：
1.  **绘制整体趋势**：以日期为横轴，收盘价为纵轴，绘制股票价格随时间变化的曲线。
2.  **标注关键点**：在图上标出历史最高点、最低点以及当前价格点。
3.  **自定义观察区间**：可以指定起始日期和结束日期，只绘制特定时间段内的数据，例如观察2000年至2010年的走势。
4.  **多指标对比**：可以同时绘制多个指标，例如将收盘价和成交量画在同一张图上（使用双Y轴），以观察价量关系。

可视化能帮助我们快速识别数据的整体趋势（如长期上涨）、周期性波动以及异常点（如交易量突增的日期）。
