## 概述

我们将比较VWAP和移动平均线这两个交易指标，并用它们来构建交易机器人。策略的核心思想是：当价格连续一段时间位于指标曲线下方时，视为下降趋势，寻找做空机会；当价格连续位于指标曲线上方时，视为上升趋势，寻找做多机会。具体的入场信号是价格“反弹接近”指标曲线时。我们将使用Python进行完整的策略实现和回测分析。

## 为什么VWAP指标很重要？ 📈

上一节我们介绍了本课程的目标，本节中我们来看看为什么VWAP是一个值得关注的指标。

观察VWAP可以帮助交易者判断证券是否在以公允价值交易。这是一个关键信息，因为：
*   如果当前价格**高于**VWAP曲线，可能表明该证券被**高估**。
*   如果当前价格**低于**VWAP曲线，可能表明该证券被**低估**。

简单来说，价格往往会在某个时刻向VWAP水平回归收敛。正如我们在示例中看到的，当价格蜡烛线与黄色VWAP曲线之间的差距拉大后，价格最终会收敛并反弹回VWAP水平。

---

## VWAP的计算公式 📊

我们可以清楚地注意到，VWAP（黄色曲线）与简单移动平均线的计算方式不同。

VWAP的计算公式如下：
`VWAP = (累计 (典型价格 * 交易量)) / 累计交易量`

其中，**典型价格**的计算公式为：
`典型价格 = (最高价 + 最低价 + 收盘价) / 3`

公式中的“累计”是指自交易时段开始以来的总和。这个时段可以根据目标设定为每日、每周或每月。

VWAP通常比简单移动平均线（SMA）包含更多信息，主要是因为它**包含了交易量数据**。请注意，VWAP表达式中的两个关键组成部分都与交易量相关。

我们还可以利用VWAP来确定交易的最佳入场和出场点，因为它也充当着随价格变动的动态支撑位和阻力位。这正是本视频所展示策略中使用VWAP的原因。

## 策略逻辑说明 🧠

我们将使用的策略逻辑如下。

首先，我们会检查一系列连续蜡烛线是位于VWAP曲线（或移动平均线）的下方还是上方。我们将使用这两个指标分别进行比较。

如果蜡烛线位于曲线下方，那么我们将其视为空头信号。当某根蜡烛线非常接近我们的对比曲线时，我们将确定入场点。

以下是具体的判断步骤：

![](img/8b8daca151f03ff11574ad3018380cd0_6.png)

1.  **识别趋势**：如果有一系列蜡烛线（例如6或7根）完全位于移动平均线下方，那么我们处于下降趋势，将寻找做空机会。
2.  **寻找入场点**：如果这些蜡烛线之后，出现一根价格非常接近移动平均线的蜡烛线，我们预期价格会从这一动态阻力位反弹并延续下降趋势。此时我们进入空头头寸。
3.  **应用至VWAP**：对VWAP曲线应用相同的方法。当在VWAP曲线下方出现一系列蜡烛线，随后某根蜡烛线反弹并非常接近VWAP水平时，我们在此时进入空头头寸。
4.  **多头逻辑**：相反，当蜡烛线位于曲线上方，并且某一根蜡烛线向下回调并非常接近曲线时，我们进入多头头寸。

为了进行比较，我们将在本视频中依次应用完全相同的策略逻辑，一次依赖于移动平均线，另一次依赖于VWAP曲线。

## 代码实现：数据准备与指标计算 💻

以下是策略的Jupyter Notebook实现步骤。

首先，我们加载数据。数据是2019年至2022年之间的比特币/美元15分钟K线数据。

```python
# 示例：使用pandas加载数据
import pandas as pd
data = pd.read_csv(‘bitcoin_data.csv‘)
```

接着，我们清理单元格中的数据格式，然后使用`pandas_ta`技术分析模块来计算VWAP和指数移动平均线（EMA）。

```python
# 示例：计算指标
import pandas_ta as ta
data[‘VWAP‘] = ta.vwap(data[‘high‘], data[‘low‘], data[‘close‘], data[‘volume‘])
data[‘EMA‘] = ta.ema(data[‘close‘], length=20)
```

---

## 代码实现：生成交易信号 ⚙️

现在我们可以计算交易信号。我们使用一个参数`back_candles`，它代表需要测试的连续蜡烛线数量，以判断它们是否高于或低于曲线（无论是移动平均线还是VWAP曲线）。

在这个单元格中，我们首先针对指数移动平均线（EMA）进行计算。我们检查每一组`back_candles`数量的连续蜡烛线，测试它们是否都高于或低于特定的曲线，然后将信号存储在一个名为`ema_signal`的新列中。

信号值将是整数：
*   `2` 或 `1`：代表明确的做多或做空信号。
*   `0`：代表没有特定信号。

我们使用完全相同的方法计算VWAP信号，并将其添加为数据框中的新列。显然，如果你想测试10根连续蜡烛线，可以更改`back_candles`这个数字。

## 代码实现：整合最终入场信号 🎯

在计算出基础的EMA和VWAP趋势信号后，我们需要计算包含“接近曲线”条件的最终入场信号，我们称之为`total_ema_signal`和`total_vwap_signal`。

如果出现任何价格非常接近曲线的蜡烛线，并且我们假设价格将从该曲线反弹并反转方向，那么这些信号将叠加在先前计算的趋势信号之上。因此，我们必须引入一个“接近距离”参数，在这里我称之为`my_close_distance`，并设置为100。这个值可能需要根据不同的交易品种进行优化。

![](img/8b8daca151f03ff11574ad3018380cd0_10.png)

在`total_ema_signal`函数中：
1.  首先检查`ema_signal`是否等于2（长期趋势，即所有测试的蜡烛线位于EMA曲线上方）。
2.  如果是，则添加另一个条件：检查蜡烛线的最高价或最低价与EMA曲线之间的距离，取更接近的那个距离，判断是否小于或等于`my_close_distance`参数。
3.  如果满足条件，则返回2作为最终的EMA入场信号（多头入场点）。
4.  对空头头寸（`ema_signal == 1`）执行相同的反向操作。

`total_vwap_signal`函数也完全相同，只是这里依赖的是VWAP值，而不是EMA曲线。

## 代码实现：可视化与回测设置 📉

我们可以使用专门的单元格来可视化这些信号点。在图表上，紫色点代表空头入场头寸，绿色点可能代表多头入场头寸。放大后，我们可以看到在曲线下方有一系列蜡烛线，同时有一根蜡烛线在价格上非常接近曲线。

现在进入回测部分。我们使用`backtesting.py`包进行回测。我们采用基于ATR（平均真实波幅）的交易管理方法。

```python
# 计算ATR
data[‘ATR‘] = ta.atr(data[‘high‘], data[‘low‘], data[‘close‘], length=14)
```

![](img/8b8daca151f03ff11574ad3018380cd0_12.png)

我们设置止盈止损比率（TP/SL Ratio）为2.5，止损距离为0.8倍的ATR值。因此，当前头寸的止损距离使用`0.8 * ATR`计算。

在回测逻辑中：
*   如果出现`total_signal == 2`的信号，并且当前没有持仓，我们则计算止损和止盈距离，并使用50%的净值开立买入头寸。
*   如果出现`total_signal == 1`的信号（空头头寸），我们应用相同的方法。

接着，我们创建一个继承自`backtesting.py`中`Strategy`类的自定义策略类，使用10万作为起始现金，并设置保证金或杠杆为5。为了便于与之前不使用佣金的策略进行比较，此处暂不包含佣金。

## 回测结果分析：基于移动平均线（EMA）的策略 📊

运行基于EMA信号的策略回测后，我们得到以下结果：
*   **总回报率**：296%
*   **夏普比率**：0.6
*   **年化回报率**：约70%

![](img/8b8daca151f03ff11574ad3018380cd0_16.png)

从资金曲线可以看到，模型在三年数据中表现良好。起始阶段出现回撤，随后在近两年时间里表现挣扎，之后随着价格波动性增加，资金曲线出现了良好的攀升。这表明，当市场波动性非常有限时，这个模型可能无法很好地工作。每当出现大幅波动时，模型似乎表现更佳。

## 参数优化与风险提示 ⚠️

![](img/8b8daca151f03ff11574ad3018380cd0_25.png)

我们大约有4到5个关键参数可以调整：
1.  `back_candles`（连续蜡烛线数量）
2.  `my_close_distance`（接近距离参数）
3.  仓位大小
4.  杠杆
5.  止盈止损比率

![](img/8b8daca151f03ff11574ad3018380cd0_26.png)

你可以根据自己的交易风格轻松调整和微调这些参数，尝试找到一个平衡点，以提高夏普比率并减少回撤周期。

例如，如果你想提高夏普比率，通常需要降低保证金或杠杆。如果我们把杠杆从5降低到2，重新运行回测，结果会发生变化：
*   **总回报率**下降到123%。
*   **夏普比率**从0.6增加到了1.02。

![](img/8b8daca151f03ff11574ad3018380cd0_28.png)

![](img/8b8daca151f03ff11574ad3018380cd0_29.png)

因此，还需要考虑一个事实：如果使用高杠杆账户，可能会降低夏普比率，因为显然你增加了风险，回撤期也可能更加严重。

本节课中，我们一起学习了如何构建并回测一个基于趋势和动态支撑阻力位的交易策略。我们详细对比了使用VWAP指标和移动平均线（EMA）指标在同一策略框架下的表现。

![](img/8b8daca151f03ff11574ad3018380cd0_33.png)

核心要点总结：
1.  **策略逻辑**：通过判断价格连续位于指标线上/下来定义趋势，在价格回调接近指标线时入场。
2.  **指标对比**：在本案例的回测中，基于VWAP的策略获得了更高的总回报（473% vs 296%），但资金曲线波动更大，夏普比率并未显著优于EMA策略。
3.  **关键参数**：策略表现严重依赖于`back_candles`、`close_distance`、杠杆及风控参数，需要根据具体市场情况进行优化。
4.  **风险控制**：高杠杆在放大收益的同时也放大了风险和回撤，合理降低杠杆有助于提升夏普比率，优化风险调整后收益。
