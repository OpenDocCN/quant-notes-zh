# Python量化交易教程：P1：基于移动平均线与蜡烛图形态的策略优化 🕯️📈

![](img/41bce7672e32ea917d7d3ffde7d60eee_1.png)

## 概述
在本节课中，我们将学习如何优化一个基于移动平均线的简单交易策略。我们将通过添加一个关于蜡烛图（K线）形态的新条件，来改进策略的性能。课程将涵盖策略逻辑、Python代码实现、以及不同止损止盈设置下的回测结果对比。

## 策略逻辑回顾 🔄
上一节我们介绍了策略的基础框架。本节中我们来看看其核心逻辑。

![](img/41bce7672e32ea917d7d3ffde7d60eee_5.png)

该策略使用三条指数移动平均线（EMA）来判断市场趋势。
*   **快速EMA**：周期最短。
*   **中等EMA**：周期居中。
*   **慢速EMA**：周期最长。

**趋势判断公式**：
当 `快速EMA > 中等EMA > 慢速EMA` 且三条均线斜率均为正时，判定为**上升趋势**。
当 `快速EMA < 中等EMA < 慢速EMA` 且三条均线斜率均为负时，判定为**下降趋势**。

![](img/41bce7672e32ea917d7d3ffde7d60eee_7.png)

**原始入场信号**：
*   在**上升趋势**中，寻找**买入**信号：一根蜡烛的**开盘价低于快速EMA**且**收盘价高于快速EMA**。
*   在**下降趋势**中，寻找**卖出**信号：一根蜡烛的**开盘价高于快速EMA**且**收盘价低于快速EMA**。

---

## 引入蜡烛图形态条件 🕯️
上一节我们介绍了基础的均线交叉入场信号。本节中，我们将在该信号上增加一个关于蜡烛图形态的过滤条件，以确认趋势的强度。

这个新条件是：入场蜡烛的影线（或称“灯芯”、“尾巴”）必须很短。
*   对于**买入信号**（看涨蜡烛），要求其**下影线**非常短。这表示价格在开盘后没有大幅向下试探，上涨动能强劲。
*   对于**卖出信号**（看跌蜡烛），要求其**上影线**非常短。这表示价格在开盘后没有大幅向上试探，下跌动能强劲。

**代码描述条件**：
假设 `Wick_Limit` 是我们设定的影线长度阈值（例如0.00025）。
对于买入信号，需满足：`(开盘价 - 最低价) <= Wick_Limit`
对于卖出信号，需满足：`(最高价 - 开盘价) <= Wick_Limit`

## 数据准备与回测设置 💻
以下是进行策略回测前需要完成的准备工作。

1.  **导入数据**：使用2019年至2022年的EUR/USD 15分钟级别K线数据。
2.  **数据清洗**：删除成交量为零的K线，并重置数据框索引。
3.  **计算指标**：
    *   添加快速、中等、慢速三条指数移动平均线（EMA）。
    *   添加平均真实波幅（ATR），用于动态设置止损止盈。
    *   计算三条EMA的斜率，用于辅助判断趋势方向。
4.  **生成信号**：
    *   首先根据三条EMA的排列顺序和斜率生成**趋势信号**。
    *   然后结合趋势和K线穿越快速EMA的条件，生成基础的**入场信号**。
    *   最后，应用本节介绍的“短影线”条件，生成**最终入场信号**。
5.  **设置回测参数**：
    *   初始本金：30单位。
    *   杠杆：1:100。
    *   测试了四种止损止盈设置方法：
        *   **固定点数**：止损45点，止盈45点。
        *   **ATR倍数**：止损距离为2倍ATR，止盈为止损的1.2倍。
        *   **ATR追踪止损**：仅使用基于ATR的追踪止损，无固定止盈。
        *   **前N根K线极值**：根据信号出现前6根K线的最低（买入）或最高（卖出）价设置止损。

## 回测结果对比 📊
上一节我们设置了回测环境。本节中，我们来比较添加“短影线”条件前后的策略表现。

以下是使用四种不同止损止盈方法时，策略在三年数据上的回测结果对比。

**1. 固定点数止损止盈**
*   **原始策略**：总回报445%，胜率55%。资金曲线波动大，存在显著回撤期。
*   **增加短影线条件后**：总回报1805%，胜率59%。资金曲线呈现稳定的上升趋势。

**2. ATR倍数止损止盈**
*   **原始策略**：总回报74%，胜率47%。资金曲线有回撤。
*   **增加短影线条件后**：总回报249%，胜率55%。资金曲线整体向上。

**3. ATR追踪止损**
*   **原始策略**：总回报-39%，胜率36%。策略失效。
*   **增加短影线条件后**：总回报394%，胜率42.5%。策略由亏转盈，但资金曲线仍有较大回撤。

**4. 前N根K线极值止损**
*   **原始策略**：总回报-89%，胜率40%。策略失效。
*   **增加短影线条件后**：总回报373%，胜率49%。策略由亏转盈。

## 总结
本节课中我们一起学习了如何通过添加蜡烛图形态过滤条件来优化一个基于移动平均线的交易策略。核心要点如下：

![](img/41bce7672e32ea917d7d3ffde7d60eee_9.png)

1.  **策略基础**：利用三条EMA的排列与斜率判断趋势，利用K线穿越快速EMA作为入场触发。
2.  **关键优化**：增加了“入场K线需具备短影线”的条件，以过滤假突破，确认趋势动能。
3.  **回测验证**：在多种止损止盈设置下，增加该条件后，策略的总回报率和资金曲线稳定性普遍得到显著提升，甚至将一些原本亏损的策略方法转变为盈利。
4.  **核心启示**：单一的入场信号容易受到市场噪音干扰。结合价格行为（如蜡烛图形态）进行过滤，是提高策略稳健性的有效方法之一。

![](img/41bce7672e32ea917d7d3ffde7d60eee_11.png)
