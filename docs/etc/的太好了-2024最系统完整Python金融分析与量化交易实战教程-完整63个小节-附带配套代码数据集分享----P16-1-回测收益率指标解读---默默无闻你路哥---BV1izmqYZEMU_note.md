## 回测收益率详解 💹

回测收益率是最基本、最直观的策略评估指标。它衡量的是，在回测周期结束时，你的策略资产净值相对于初始本金增长的百分比。

![](img/e819a77f552379077aaa3aa2bb0f9583_6.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_7.png)

举个例子，如果你用一套策略测试了从2015年到2020年的某只股票数据，根据策略规则进行买卖，最终会得到一个结果。这个结果不是指你赚了多少钱的绝对值，而是相对于你最初投入的本金，资产增值了多少。例如，如果回测收益率是40%，就意味着经过这些年的操作，你的总资产比最初的本金多了40%。

它的计算公式如下：

![](img/e819a77f552379077aaa3aa2bb0f9583_9.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_16.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_10.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_11.png)

**公式**：
`回测收益率 = (期末资产净值 - 期初资产净值) / 期初资产净值`

![](img/e819a77f552379077aaa3aa2bb0f9583_12.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_13.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_14.png)

其中，`P_T` 代表回测结束时的资产净值（卖出所有资产后的总资金），`P_0` 代表回测开始时的初始本金。这个公式计算的是策略在整个回测期间的总回报率。

## 代码实战：计算回测收益率 💻

上一节我们介绍了回测收益率的概念和公式，本节中我们来看看如何用Python代码进行计算和可视化。我们将使用一个示例数据集来模拟策略回测后的每日资产净值。

![](img/e819a77f552379077aaa3aa2bb0f9583_16.png)

首先，我们需要导入必要的工具包并加载数据。

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便显示
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

![](img/e819a77f552379077aaa3aa2bb0f9583_18.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_19.png)

# 读取数据，假设第一列是日期
data = pd.read_csv('your_data.csv', index_col=0)
print(data.head())
```

![](img/e819a77f552379077aaa3aa2bb0f9583_6.png)

数据加载后，我们假设其中一列（例如‘close’）代表了我们策略回测过程中每天的资产净值。接下来，我们计算相对于第一天的资产净值变化比率。

```python
# 计算每日资产净值相对于初始值的比率
# 即公式中的 P_t / P_0
value_ratio = data['close'] / data['close'].iloc[0]

# 绘制资产净值变化曲线
plt.figure(figsize=(12, 6))
plt.plot(value_ratio.index, value_ratio.values)
plt.title('资产净值相对变化曲线')
plt.xlabel('日期')
plt.ylabel('净值比率 (P_t / P_0)')
plt.grid(True)
plt.show()
```

![](img/e819a77f552379077aaa3aa2bb0f9583_21.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_21.png)

上图展示了资产净值随时间的变化情况，起点为1。最后，我们计算整个回测期间的总收益率。

```python
# 计算最终的回测收益率
# 即公式： (P_T / P_0) - 1
final_return = value_ratio.iloc[-1] - 1

![](img/e819a77f552379077aaa3aa2bb0f9583_23.png)

![](img/e819a77f552379077aaa3aa2bb0f9583_24.png)

# 将结果整理成DataFrame展示
result_df = pd.DataFrame({
    ‘回测收益率’: [final_return]
}, index=[‘最终结果’])
print(result_df)
```

![](img/e819a77f552379077aaa3aa2bb0f9583_27.png)

执行以上代码，我们就得到了策略在整个回测期间的回测收益率。这个值可能是正数（盈利），也可能是负数（亏损）。它直观地告诉我们策略的最终表现。
