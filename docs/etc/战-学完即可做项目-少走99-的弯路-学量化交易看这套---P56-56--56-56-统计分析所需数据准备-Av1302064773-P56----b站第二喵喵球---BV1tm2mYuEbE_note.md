## 数据准备与特征构建

![](img/23bbc2b6e94125066165f38898c2280a_9.png)

![](img/23bbc2b6e94125066165f38898c2280a_11.png)

要实现上述统计，我们首先需要将原始的股价变化数据，转换为能清晰表示“涨”和“跌”的二值化特征。

![](img/23bbc2b6e94125066165f38898c2280a_13.png)

![](img/23bbc2b6e94125066165f38898c2280a_15.png)

![](img/23bbc2b6e94125066165f38898c2280a_17.png)

以下是构建特征的具体步骤：

![](img/23bbc2b6e94125066165f38898c2280a_19.png)

![](img/23bbc2b6e94125066165f38898c2280a_21.png)

1.  **定义特征列**：我们需要关注“前一天”和“前两天”的股价变化数据。假设这些数据已经存在于我们的`DataFrame`中，列名分别为`L1`和`L2`。
2.  **二值化处理**：将连续的价格变化值转换为简单的0和1。规则是：若变化值大于0（上涨），则标记为1；若小于或等于0（下跌或持平），则标记为0。
3.  **生成新列**：为原始数据中的`L1`和`L2`列分别创建对应的二值化新列，例如`L1-B`和`L2-B`。

![](img/23bbc2b6e94125066165f38898c2280a_23.png)

![](img/23bbc2b6e94125066165f38898c2280a_25.png)

## 代码实现

我们将通过一个函数来完成上述特征构建过程。

![](img/23bbc2b6e94125066165f38898c2280a_27.png)

```python
def clip_ins(data, bs=0):
    """
    将指定的特征列二值化（0或1）。
    :param data: 包含特征数据的DataFrame
    :param bs: 判断阈值，默认为0
    """
    global col_names  # 声明使用全局变量来存储新列名
    col_names = []    # 初始化列名列表

    # 假设‘L1’和‘L2’是我们要处理的特征列名
    feature_columns = [‘L1‘, ‘L2‘]

    for col in feature_columns:
        # 生成新列名，例如‘L1-B’
        new_col_name = col + ‘-B‘
        # 二值化判断：大于阈值bs为1，否则为0
        data[new_col_name] = (data[col] > bs).astype(int)
        # 记录新列名
        col_names.append(new_col_name)

![](img/23bbc2b6e94125066165f38898c2280a_29.png)

![](img/23bbc2b6e94125066165f38898c2280a_31.png)

# 调用函数，处理数据
clip_ins(data)
print(data[[‘L1-B‘, ‘L2-B‘]].head())
```

![](img/23bbc2b6e94125066165f38898c2280a_33.png)

执行以上代码后，数据集中会增加`L1-B`和`L2-B`两列。例如，`[1, 0]`表示前一天上涨(`L1-B=1`)，前两天下跌(`L2-B=0`)。

## 整合实际涨跌方向

![](img/23bbc2b6e94125066165f38898c2280a_39.png)

![](img/23bbc2b6e94125066165f38898c2280a_41.png)

为了进行统计，我们还需要知道在每一种“前两天”情况下，股票实际的涨跌结果。我们假设原始数据中已有一列`direction`，表示当日实际的涨跌（例如，1为涨，0为跌或平）。

![](img/23bbc2b6e94125066165f38898c2280a_43.png)

![](img/23bbc2b6e94125066165f38898c2280a_45.png)

现在，我们的数据中包含了：
*   **条件特征**：`L1-B`（前一天状态）、`L2-B`（前两天状态）。
*   **结果标签**：`direction`（当日实际结果）。

## 分组统计频率

![](img/23bbc2b6e94125066165f38898c2280a_51.png)

接下来，我们使用`pandas`的`groupby`功能，对`[‘L1-B‘, ‘L2-B‘, ‘direction‘]`这三个列进行分组，并统计每种组合出现的次数。这直接反映了各种历史情况下的结果分布。

![](img/23bbc2b6e94125066165f38898c2280a_53.png)

![](img/23bbc2b6e94125066165f38898c2280a_55.png)

![](img/23bbc2b6e94125066165f38898c2280a_57.png)

```python
# 按‘L1-B‘, ‘L2-B‘, ‘direction‘进行分组，并计算每组大小
grouped = data.groupby([‘L1-B‘, ‘L2-B‘, ‘direction‘])
frequency_stats = grouped.size()
print(frequency_stats)
```

![](img/23bbc2b6e94125066165f38898c2280a_59.png)

![](img/23bbc2b6e94125066165f38898c2280a_61.png)

执行这段代码后，我们将得到一个类似如下的统计结果：
*   `(0, 0, 0): 120` 表示前两天跌、前一天跌、当日实际也跌的情况出现了120次。
*   `(0, 0, 1): 85` 表示前两天跌、前一天跌、但当日实际涨的情况出现了85次。
*   `(0, 1, 1): 200` 表示前两天涨、前一天跌、当日实际涨的情况出现了200次。
*   ……

## 总结

![](img/23bbc2b6e94125066165f38898c2280a_73.png)
