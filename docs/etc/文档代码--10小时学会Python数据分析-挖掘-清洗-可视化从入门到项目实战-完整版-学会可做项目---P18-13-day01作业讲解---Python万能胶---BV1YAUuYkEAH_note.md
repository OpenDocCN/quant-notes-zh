## 作业1：选择性清除条件格式 🧹

![](img/15bfbdcf6494a8781bfd4833b827ae96_5.png)

上一节我们介绍了条件格式的基本应用，本节中我们来看看如何精确地清除它。

![](img/15bfbdcf6494a8781bfd4833b827ae96_7.png)

题目要求是：清除选中单元格中已设置的条件格式，但保留其他手动设置的格式（如背景色）。

**核心思路**：使用“定位条件”功能，精确选中所有应用了条件格式的单元格，然后仅清除这些单元格的格式。

以下是具体操作步骤：

![](img/15bfbdcf6494a8781bfd4833b827ae96_9.png)

1.  选中目标数据区域。
2.  按下 `Ctrl + G` 打开“定位”对话框，点击“定位条件”。
3.  在弹出的窗口中，选择“条件格式”，然后点击“确定”。此时，所有应用了条件格式的单元格会被选中。
4.  在“开始”选项卡的“编辑”组中，点击“清除”，然后选择“清除格式”。

![](img/15bfbdcf6494a8781bfd4833b827ae96_11.png)

这样操作后，只有条件格式被清除，其他手动设置的单元格格式得以保留。

![](img/15bfbdcf6494a8781bfd4833b827ae96_13.png)

## 作业2：快速对比两列数据的差异 🔍

![](img/15bfbdcf6494a8781bfd4833b827ae96_15.png)

在日常工作中，我们经常需要对比两列数据是否一致。本节介绍两种高效的方法。

第一种方法是使用“定位条件”。

以下是具体操作步骤：

1.  同时选中需要对比的两列数据。
2.  按下 `Ctrl + G` 打开“定位”对话框，点击“定位条件”。
3.  选择“行内容差异单元格”，然后点击“确定”。Excel会自动选中同行中内容不同的单元格。
4.  为这些选中的单元格填充一个醒目的颜色（如黄色）即可。

第二种方法是使用“条件格式”中的公式规则，这种方法在对比包含公式计算或小数位的数据时可能更精确。

以下是具体操作步骤：

![](img/15bfbdcf6494a8781bfd4833b827ae96_17.png)

![](img/15bfbdcf6494a8781bfd4833b827ae96_18.png)

1.  选中需要对比的数据区域（例如A列和B列）。
2.  点击“开始”选项卡下的“条件格式”，选择“新建规则”。
3.  在规则类型中，选择“使用公式确定要设置格式的单元格”。
4.  在公式框中输入：`=NOT(EXACT($A1,$B1))`
    *   `EXACT` 函数用于精确比较两个文本字符串是否完全相同。
    *   `NOT` 函数将结果取反，即当两单元格内容**不**相同时，公式返回 `TRUE`，触发格式设置。
5.  点击“格式”按钮，设置填充颜色，然后点击“确定”。

应用此规则后，两列中内容不相同的行会自动高亮显示。

## 作业3：合并两列数据并跳过空白单元格 ➕

当我们需要将两列数据合并到一列，且两列数据交错无重复时，可以使用“选择性粘贴”。

![](img/15bfbdcf6494a8781bfd4833b827ae96_20.png)

以下是具体操作步骤：

1.  复制第一列数据，粘贴到目标列。
2.  复制第二列数据。
3.  右键点击目标列的第一个单元格，选择“选择性粘贴”。
4.  在弹出的对话框中，勾选“跳过空单元格”选项，然后点击“确定”。

这样，第二列的非空数据就会自动填充到目标列的空白位置，实现两列数据的无缝合并。

## 作业4：合并上下半年报表数据 📈

当两个结构完全相同的表格（如上半年和下半年报表）需要合并时，有两种简便方法。

![](img/15bfbdcf6494a8781bfd4833b827ae96_22.png)

![](img/15bfbdcf6494a8781bfd4833b827ae96_23.png)

第一种方法是使用“选择性粘贴”中的“加”运算。

![](img/15bfbdcf6494a8781bfd4833b827ae96_25.png)

以下是具体操作步骤：

1.  复制下半年报表的数据区域。
2.  选中上半年报表对应的数据区域的起始单元格。
3.  右键点击，选择“选择性粘贴”。
4.  在“运算”区域选择“加”，然后点击“确定”。这样，下半年数据就会累加到上半年数据上。

![](img/15bfbdcf6494a8781bfd4833b827ae96_27.png)

第二种方法是使用公式引用，适用于需要动态链接的情况。

假设上半年数据在 `M5` 单元格，对应的下半年数据在 `M12` 单元格，则可以在汇总单元格输入公式：`=M5+M12`。向右向下拖动填充柄，即可快速完成所有数据的合并计算。

![](img/15bfbdcf6494a8781bfd4833b827ae96_29.png)

## 作业5：为空白单元格快速求和 ➕

这个作业是对之前课程的复习，核心是定位空值并快速求和。

以下是具体操作步骤：

1.  选中包含空白单元格的数据区域。
2.  按下 `Ctrl + G`，打开“定位条件”，选择“空值”并确定，所有空白单元格会被选中。
3.  直接按下 `Alt + =`（自动求和快捷键），Excel会自动为每个空白单元格填充其上方的数据之和。

![](img/15bfbdcf6494a8781bfd4833b827ae96_31.png)

## 作业6：快速生成大量连续序号 🔢

如果需要生成1到10000的连续序号，手动输入效率低下。我们可以使用“序列”填充功能。

以下是具体操作步骤：

![](img/15bfbdcf6494a8781bfd4833b827ae96_35.png)

1.  在起始单元格（如A1）输入数字 `1`。
2.  选中该单元格，在“开始”选项卡的“编辑”组中，点击“填充”，选择“序列”。
3.  在“序列”对话框中：
    *   “序列产生在”选择“列”。
    *   “步长值”输入 `1`。
    *   “终止值”输入 `10000`。
4.  点击“确定”，Excel会自动在A列填充1到10000的序号。

![](img/15bfbdcf6494a8781bfd4833b827ae96_37.png)

## 作业7：拆分数据并提取不重复值 🧩

本题要求将用“VS”连接的国家名拆分，并分别提取不重复值。这需要结合“分列”和“删除重复项”功能。

![](img/15bfbdcf6494a8781bfd4833b827ae96_42.png)

以下是具体操作步骤：

1.  **替换分隔符**：选中数据，按 `Ctrl + H` 打开“查找和替换”对话框。将“VS”全部替换为逗号 `,`。
2.  **分列**：保持数据选中状态，点击“数据”选项卡下的“分列”。选择“分隔符号”，点击“下一步”，勾选“逗号”作为分隔符，点击“完成”。数据会被拆分成多列。
3.  **删除重复值**：将拆分后的各列数据分别复制到单独的区域。选中其中一列，点击“数据”选项卡下的“删除重复项”，点击“确定”以保留唯一值。对其他列重复此操作。

![](img/15bfbdcf6494a8781bfd4833b827ae96_44.png)

## 作业8：从文本中提取评分 ⭐

![](img/15bfbdcf6494a8781bfd4833b827ae96_46.png)

最后一题要求从“工作能力88”这类文本中，将描述和分数分开。这里介绍三种方法。

**方法一：使用“分列”功能**
利用空格作为分隔符进行分列，是最直接的方法。

![](img/15bfbdcf6494a8781bfd4833b827ae96_48.png)

**方法二：使用 `Ctrl + E`（快速填充）**
1.  手动在第一个单元格输入对应的分数（如88）。
2.  选中该单元格及下方区域，按下 `Ctrl + E`，Excel会自动识别模式并填充所有分数。

**方法三：使用函数（更灵活）**
如果文本结构规律（如分数都是最后2位），可以使用函数批量提取。
例如，在目标单元格输入公式：`=RIGHT(A1, 2)`。这个公式会提取A1单元格文本最右侧的2个字符（即分数）。向下拖动即可填充所有行。

![](img/15bfbdcf6494a8781bfd4833b827ae96_50.png)
