在本节课中，我们将要学习一个重要的风险指标——最大回撤。它描述了在投资过程中，资产价值从最高点到最低点的最大跌幅，是衡量策略风险的关键指标。

![](img/769fc62a5d023f043d6c48d452102a15_3.png)

## 什么是最大回撤？📉

上一节我们介绍了策略的收益评估，本节中我们来看看如何评估风险。最大回撤描述的是投资过程中“最惨”的阶段。

它衡量的是从任意一个历史高点，到之后某个低点之间，资产价值的最大百分比跌幅。公式定义如下：

**最大回撤 = max( (Pi - Pj) / Pi )**

其中：
*   **Pi** 代表第 i 天的资产价值（历史高点）。
*   **Pj** 代表第 i 天之后第 j 天的资产价值（后续低点）。

这个指标的核心是找出整个投资周期内，资产缩水最严重的那个连续下跌区间。它不直接评价策略好坏，而是揭示策略可能面临的最大风险。例如，即使一个策略长期盈利，但过程中可能经历一段剧烈的下跌，这段下跌就是最大回撤区间，它考验投资者的承受能力。

## 如何计算最大回撤？🧮

![](img/769fc62a5d023f043d6c48d452102a15_5.png)

理解了概念后，我们来看看如何用Python代码实现最大回撤的计算。核心思路是：遍历所有可能的高点（Pi）和其后的低点（Pj），找出 `(Pi - Pj) / Pi` 的最大值。

![](img/769fc62a5d023f043d6c48d452102a15_7.png)

Pandas库提供了便捷的方法来进行滚动计算。以下是一个关键函数 `cummax()` 的介绍，它能帮助我们高效地找到历史高点序列。

`cummax()` 函数会计算序列的累积最大值。对于序列中的每一个位置，它返回的是从序列开始到当前位置所有值中的最大值。

![](img/769fc62a5d023f043d6c48d452102a15_9.png)

```python
import pandas as pd

![](img/769fc62a5d023f043d6c48d452102a15_11.png)

![](img/769fc62a5d023f043d6c48d452102a15_13.png)

# 创建一个示例序列
s = pd.Series([3, 1, 4, 1, 5, 9, 2])
print(s.cummax())
# 输出: 0    3
#       1    3 (1和3比，最大值是3)
#       2    4 (4和之前的最大值3比，最大值是4)
#       3    4
#       4    5
#       5    9
#       6    9
```

![](img/769fc62a5d023f043d6c48d452102a15_15.png)

基于 `cummax()`，我们可以按步骤计算最大回撤：

![](img/769fc62a5d023f043d6c48d452102a15_17.png)

以下是计算单日回撤并找出最大值的步骤：

![](img/769fc62a5d023f043d6c48d452102a15_19.png)

![](img/769fc62a5d023f043d6c48d452102a15_21.png)

1.  计算历史最高点序列：`historical_max = price_series.cummax()`
2.  计算每日的回撤值：`daily_drawdown = (historical_max - price_series) / historical_max`
3.  找出整个序列中的最大回撤值：`max_drawdown = daily_drawdown.max()`

![](img/769fc62a5d023f043d6c48d452102a15_23.png)

![](img/769fc62a5d023f043d6c48d452102a15_25.png)

让我们结合股价数据示例来演示：

![](img/769fc62a5d023f043d6c48d452102a15_27.png)

```python
# 假设 df[‘close’] 是我们的股价序列
price_series = df[‘close’]

![](img/769fc62a5d023f043d6c48d452102a15_29.png)

![](img/769fc62a5d023f043d6c48d452102a15_31.png)

# 步骤1: 计算累积最大值（历史高点）
rolling_max = price_series.cummax()

![](img/769fc62a5d023f043d6c48d452102a15_33.png)

# 步骤2: 计算每日回撤
daily_drawdown = (rolling_max - price_series) / rolling_max

![](img/769fc62a5d023f043d6c48d452102a15_35.png)

![](img/769fc62a5d023f043d6c48d452102a15_37.png)

# 步骤3: 找到最大回撤
max_drawdown = daily_drawdown.max()

print(f”最大回撤为: {max_drawdown:.2%}”)
```

![](img/769fc62a5d023f043d6c48d452102a15_39.png)

![](img/769fc62a5d023f043d6c48d452102a15_41.png)

执行以上代码，我们便可以得到该股价序列在整个观察期内的最大回撤百分比。这个值对应着策略回测图表中阴影部分最深的那个区间。
