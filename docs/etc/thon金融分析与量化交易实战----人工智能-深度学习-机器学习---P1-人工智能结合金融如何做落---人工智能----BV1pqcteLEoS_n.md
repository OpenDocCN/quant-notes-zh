# Python金融分析与量化交易实战教程：P1：人工智能结合金融如何做落地项目 🚀

## 概述
在本节课中，我们将探讨人工智能如何与金融领域结合以完成实际项目。我们将从两个主要角度进行分析：学术研究的角度与实际项目开发的角度。

## 学术研究角度：从理论算法入手
上一节我们介绍了课程概述，本节中我们来看看如何从学术研究的角度结合AI与金融。撰写论文时，建议从理论模型和算法入手，而非直接使用回测平台或分析股市信息。核心应回归到**时间序列预测**模型上。

以下是当前时间序列预测中三种比较主流的方法，可供参考：

1.  基于Transformer架构的模型，例如 **Informer**。
2.  基于傅里叶变换的模型，例如 **TimesNet**，它结合了卷积和傅里叶变换。
3.  结合大语言模型的方法，例如 **Time-LLM**，利用大模型进行跨领域的知识迁移。

总之，进行学术研究应专注于理论算法的创新与改进。

## 实际项目角度：构建智能工作流
上一节我们介绍了学术研究的思路，本节中我们来看看实际项目开发的角度。在实际项目中，任务不仅限于时间序列预测。因为影响金融市场的因素非常多，本质上我们需要处理和分析多源、广泛的知识与数据。

此时，工作的重点有所不同。建议大家多关注**智能体（Agent）**的搭建。目前，许多大型企业和学术机构都已推出专用的智能体，用于回测、量化分析和自动交易。

如果进行实际项目开发，建议尝试构建一个智能体，并根据自身需求融入个性化的设计和工作流程。

## 总结
本节课中我们一起学习了人工智能与金融结合的两个实践路径。在学术研究上，应聚焦于时间序列预测等核心算法的理论创新；在实际项目开发上，则可以探索构建智能体（Agent）来整合复杂的工作流与多源数据，以解决更实际的金融问题。
