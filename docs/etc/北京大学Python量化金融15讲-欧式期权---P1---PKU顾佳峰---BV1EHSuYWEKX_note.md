## 欧式期权基本概念

基于BSM模型，我们来具体了解欧式期权。欧式期权赋予买方在特定到期日，以特定价格买入或卖出某种资产的权利，而非义务。这种权利只能在到期日当天行使。

以下是欧式期权的两种主要类型：
*   **看涨期权**：赋予持有者在到期日以执行价格买入资产的权利。
*   **看跌期权**：赋予持有者在到期日以执行价格卖出资产的权利。

![](img/4b9d30c735acfe5db505162da2460e18_7.png)

从交易双方的角度看，买入看涨期权意味着从卖方处买入一项买入资产的权利；卖出看涨期权则意味着向买方出售这项权利。

![](img/4b9d30c735acfe5db505162da2460e18_9.png)

## 使用Python进行蒙特卡洛模拟

理解了期权类型后，本节中我们来看看如何用Python进行定价模拟。我们将使用蒙特卡洛方法，通过大量随机模拟来估算欧式看涨期权的价格。

实现过程主要依赖`NumPy`库。首先需要设定模型所需的各项参数。

以下是实现模拟所需的关键参数和步骤：
1.  **导入库**：`import numpy as np`
2.  **设定参数**：
    *   `S0`: 标的资产初始价格
    *   `K`: 期权的执行价格
    *   `T`: 期权到期时间（年）
    *   `r`: 无风险利率
    *   `sigma`: 标的资产价格的波动率
    *   `I`: 蒙特卡洛模拟的次数
3.  **核心算法**：生成随机路径，计算到期日股价及期权收益。
    *   生成随机数：`np.random.standard_normal(I)`
    *   计算到期日股价 `ST`：`ST = S0 * np.exp((r - 0.5 * sigma ** 2) * T + sigma * np.sqrt(T) * np.random.standard_normal(I))`
    *   计算期权到期收益 `hT`：`hT = np.maximum(ST - K, 0)`
    *   计算期权现值 `C0`：`C0 = np.exp(-r * T) * np.mean(hT)`

例如，设定参数 `S0=100.0`, `K=105.0`, `T=1.0`, `r=0.05`, `sigma=0.2`, `I=100000` 进行模拟，可能得到期权价格约为 **7.971**。

若将模拟次数 `I` 增加到100万次，其他参数不变，得到的价格可能变为 **8.016**。模拟次数的增加会影响结果的精度。投资者可以通过调整这些参数，模拟在不同市场条件下的期权价格，为投资决策提供参考。
