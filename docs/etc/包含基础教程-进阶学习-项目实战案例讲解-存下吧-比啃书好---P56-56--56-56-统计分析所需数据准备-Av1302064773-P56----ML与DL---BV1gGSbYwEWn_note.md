上一节我们介绍了量化交易的基本概念，本节中我们来看看如何为统计分析准备数据。我们的目标是：基于“前一天”和“前两天”的指标（例如涨跌幅），判断它们是上涨（>0）还是下跌（<0），然后统计在这些不同历史组合下，“当天”实际涨跌的频次。

首先，我们需要对原始数据进行处理，将连续的价格变化指标转换为代表“涨”或“跌”的离散标签（例如，用1表示涨，0表示跌）。

以下是实现这一转换的核心代码步骤：

![](img/537691195ef289cd4100dd47e261c2fc_25.png)

```python
def clip_bins(data, bins=[0]):
    """
    将数据根据阈值（默认为0）进行二值化处理。
    大于等于阈值的标记为1，小于阈值的标记为0。
    """
    global new_cols
    new_cols = []
    for col in data.columns:
        # 创建新列名，在原列名后添加‘_b’
        new_col = col + ‘_b‘
        # 使用pandas的cut函数进行分箱，这里简化为与0比较
        # 实际等价于：data[new_col] = (data[col] >= 0).astype(int)
        data[new_col] = pd.cut(data[col], bins=bins, labels=False)
        new_cols.append(new_col)
```

```python
# 假设‘L1_b‘和‘L2_b‘是处理后的特征列，‘direction‘是实际涨跌列
grouped = data.groupby([‘L1_b‘, ‘L2_b‘, ‘direction‘])
# 计算每个分组的大小（即频次）
result_counts = grouped.size()
print(result_counts)
```

本节课中我们一起学习了为统计分析准备数据的关键步骤：
1.  **数据二值化**：将连续的金融指标（如涨跌幅）转换为离散的涨跌标签（0和1），便于进行类别统计。
2.  **分组频次统计**：使用`groupby`方法，按照“前一天状态”、“前两天状态”和“当天实际方向”进行分组，并统计各组合出现的次数。

![](img/537691195ef289cd4100dd47e261c2fc_73.png)

这个过程是许多传统统计交易策略的基础。虽然该方法本身较为简单，但它清晰地展示了如何将原始数据转化为可用于分析决策的格式。在接下来的课程中，我们将利用这些统计结果来构建并回测简单的交易策略。
