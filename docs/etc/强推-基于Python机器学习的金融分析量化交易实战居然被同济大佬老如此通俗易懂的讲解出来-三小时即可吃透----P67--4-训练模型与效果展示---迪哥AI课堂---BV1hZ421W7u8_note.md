## 数据准备：构建训练序列 📊

![](img/f6e5d8b280755015a548f5291e1fd5ed_12.png)

上一节我们介绍了RNN模型的结构，本节中我们来看看如何准备符合模型输入要求的数据。核心任务是将原始的时间序列数据转换为模型能够处理的序列样本。

### 滑动窗口生成序列

![](img/f6e5d8b280755015a548f5291e1fd5ed_18.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_20.png)

我们需要使用滑动窗口的方法，从原始数据中截取固定长度的子序列作为输入（X），并将子序列的下一个值作为预测目标（Y）。

![](img/f6e5d8b280755015a548f5291e1fd5ed_22.png)

以下是生成训练数据的核心函数 `get_train_data` 的步骤解析：

1.  **初始化空列表**：创建 `data_x` 和 `data_y` 列表，用于存放所有生成的输入序列和对应的标签。
    ```python
    data_x = []
    data_y = []
    ```

![](img/f6e5d8b280755015a548f5291e1fd5ed_24.png)

2.  **确定序列数量**：假设总数据长度为4000，每个序列长度为50。由于最后一个序列不能越界，因此可生成的序列总数为 `总长度 - 序列长度`。
    ```python
    for i in range(4000 - 50):
    ```

![](img/f6e5d8b280755015a548f5291e1fd5ed_26.png)

3.  **应用滑动窗口**：在循环中，调用 `nice_window` 函数获取从索引 `i` 开始、长度为51（50个输入步长 + 1个目标值）的窗口数据。
    ```python
    window = nice_window(data, start=i, length=51)
    ```

![](img/f6e5d8b280755015a548f5291e1fd5ed_28.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_30.png)

4.  **分割输入与目标**：将窗口内的前49个数据点作为输入特征 `X`，将第50个数据点作为预测目标 `Y`。
    ```python
    x = window[0:49]  # 输入序列
    y = window[49]    # 预测目标
    ```

![](img/f6e5d8b280755015a548f5291e1fd5ed_32.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_34.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_36.png)

5.  **保存序列**：将生成的 `x` 和 `y` 分别添加到 `data_x` 和 `data_y` 列表中。
    ```python
    data_x.append(x)
    data_y.append(y)
    ```

![](img/f6e5d8b280755015a548f5291e1fd5ed_38.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_40.png)

完成循环后，我们得到了形状为 `(3950, 49, 1)` 的 `data_x` 和形状为 `(3950, 1)` 的 `data_y`。这符合RNN的输入要求：`(样本数, 时间步长, 特征维度)`。

![](img/f6e5d8b280755015a548f5291e1fd5ed_42.png)

## 模型训练与回调设置 ⚙️

![](img/f6e5d8b280755015a548f5291e1fd5ed_44.png)

准备好数据后，下一步是将数据输入模型进行训练。为了提高训练效率和模型质量，我们会使用一些回调函数。

![](img/f6e5d8b280755015a548f5291e1fd5ed_46.png)

以下是配置和启动模型训练的关键步骤：

![](img/f6e5d8b280755015a548f5291e1fd5ed_48.png)

1.  **编译模型**：指定优化器、损失函数和评估指标（此步骤通常在模型构建时已完成）。
    ```python
    model.compile(optimizer='adam', loss='mse')
    ```

![](img/f6e5d8b280755015a548f5291e1fd5ed_50.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_52.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_54.png)

2.  **定义回调函数**：回调函数可以在训练过程中执行特定操作。
    *   **EarlyStopping**：监控验证集损失（`val_loss`）。如果该指标在连续多个周期内没有改善，则提前终止训练，防止过拟合和资源浪费。
        ```python
        early_stop = EarlyStopping(monitor='val_loss', patience=5)
        ```
    *   **ModelCheckpoint**：在每个训练周期后保存模型。设置 `save_best_only=True` 可以确保只保存验证集指标最好的那个模型。
        ```python
        checkpoint = ModelCheckpoint('best_model.h5', monitor='val_loss', save_best_only=True)
        ```

3.  **开始训练**：调用 `fit` 方法，传入训练数据、训练轮次、批次大小以及定义好的回调函数列表。
    ```python
    history = model.fit(data_x, data_y,
                        epochs=100,
                        batch_size=32,
                        validation_split=0.2,
                        callbacks=[early_stop, checkpoint])
    ```

![](img/f6e5d8b280755015a548f5291e1fd5ed_56.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_58.png)

对于本教程中的正弦波预测任务，由于问题相对简单，模型损失会迅速下降，训练很快就能达到很好的效果。

训练完成后，我们需要在测试集上评估模型性能，并直观地查看预测效果。

### 测试流程

![](img/f6e5d8b280755015a548f5291e1fd5ed_68.png)

测试过程通常包括以下步骤：
1.  加载在验证集上表现最好的已保存模型。
2.  使用与训练数据相同的方式，为测试数据构建输入序列。
3.  让模型对测试序列进行预测。
4.  将模型的预测结果与真实值进行比较。

![](img/f6e5d8b280755015a548f5291e1fd5ed_70.png)

![](img/f6e5d8b280755015a548f5291e1fd5ed_72.png)
