# 量化策略基础：P1：利用Python计算皮尔逊相关系数 📈

在本节课中，我们将学习如何利用Python代码计算皮尔逊相关系数，并将其应用于量化策略分析中，以寻找历史行情中的相似K线走势。

## 概述

皮尔逊相关系数是衡量两个变量之间线性相关程度的统计指标。在量化分析中，我们可以用它来比较当前K线形态与历史K线形态的相似性，从而预测未来可能的走势。本节将逐步演示这一过程。

## 数据准备与处理

首先，我们需要准备用于分析的历史数据。在本案例中，我们站在2024年2月5日的时点进行分析，因此筛选出该日期之前的所有数据。

接下来，我们需要计算未来若干天的收益率。这个收益率数据将用于分析：当历史行情中出现相似的K线走势后，其未来一段时间的收益表现如何。

我们设定一个参数 `length = 100`，表示将当前时点之前的100根K线作为基准K线，用于后续的相似性比对。

## 核心计算过程

上一节我们介绍了数据准备，本节中我们来看看如何通过循环计算K线组合的相关性。

首先，初始化一个DataFrame，我们将其命名为 `df_result`。这个DataFrame用于记录在历史数据中寻找到的、与当前基准K线形态相似的K线组合。记录的信息包括该组合的起始日期、截止日期以及计算出的相关系数。

以下是计算相关性的核心步骤：

1.  **循环遍历历史数据**：从历史数据中，依次截取与基准K线数量相同（即100根）的K线序列。
2.  **计算多维度相关性**：一根K线包含开盘价、最高价、最低价、收盘价四个价格数据。我们需要分别计算当前基准序列与历史序列在这四个价格序列上的皮尔逊相关系数。
3.  **合成最终相关系数**：将上述四个相关系数求平均值，得到代表两个K线组合整体形态相似性的最终相关系数。其公式可以表示为：
    **最终相关系数 = (corr(开盘价) + corr(最高价) + corr(最低价) + corr(收盘价)) / 4**

以下是该过程的简化代码逻辑示意：

```python
# 假设 baseline_prices 是基准K线的四价数据，history_segment 是历史片段K线的四价数据
open_corr = pearsonr(baseline_prices[‘open‘], history_segment[‘open‘])[0]
high_corr = pearsonr(baseline_prices[‘high‘], history_segment[‘high‘])[0]
low_corr = pearsonr(baseline_prices[‘low‘], history_segment[‘low‘])[0]
close_corr = pearsonr(baseline_prices[‘close‘], history_segment[‘close‘])[0]

final_correlation = (open_corr + high_corr + low_corr + close_corr) / 4
```

## 结果汇总与展示

计算完成后，我们将记录所有相似片段信息的 `df_result` 与原始历史数据拼接起来，以便进行综合分析。

最后，打印出最终的数据。在本案例的运行结果中，相关性最高的三个历史K线片段与当前基准形态的相关系数大约都在0.924左右，这些相似形态出现的时间点分别是在1994年。
