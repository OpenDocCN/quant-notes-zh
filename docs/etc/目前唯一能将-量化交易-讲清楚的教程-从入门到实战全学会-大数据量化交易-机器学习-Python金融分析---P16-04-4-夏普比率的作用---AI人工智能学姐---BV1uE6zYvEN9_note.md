## 概述

上一节我们介绍了投资回报率的计算。本节中，我们来看看如何衡量“性价比”，即承担单位风险能获得多少超额收益。夏普比率正是这样一个指标，它帮助我们评估投资组合的风险调整后表现。

## 夏普比率的核心概念

夏普比率描述的是：对于每承担一单位的风险，我们能获得多少超额收益。指标越高，意味着在承担相同风险的情况下，获得的收益越高，投资策略的“性价比”也就越好。

![](img/2fef2fce2dbda98b5e466cbde009d722_1.png)

我们可以用一个生活中的例子来理解：一份在叙利亚日薪数万的雇佣兵工作，其高薪对应的是极高的生命风险。夏普比率就像是在衡量，为了获得这份高薪，所承担的风险是否“值得”。

![](img/2fef2fce2dbda98b5e466cbde009d722_3.png)

在选股或评估投资策略时，如果单从夏普比率这个指标来看，我们应选择**夏普比率更高**的股票或策略，因为它意味着单位风险带来的收益更高。

![](img/2fef2fce2dbda98b5e466cbde009d722_5.png)

## 夏普比率的计算公式

理解了概念后，我们来看看如何计算夏普比率。它的计算基于两个核心对比：投资组合的收益率与无风险收益率。

*   **无风险收益率**：可以理解为几乎没有任何风险就能获得的收益，例如银行存款利息、国债收益等。这是一个基准。
*   **投资组合收益率**：就是你实际的投资策略或组合所获得的收益率。
*   **投资组合收益率的标准差**：用于衡量投资组合的风险或波动性。

夏普比率的计算公式如下：

**夏普比率 = (投资组合年化收益率 - 无风险年化收益率) / 投资组合年化收益率的标准差**

这个公式可以拆解为两步：
1.  计算**超额收益**：你的投资组合收益比无风险收益多出来的部分。
2.  计算**风险调整后的收益**：将超额收益除以组合的风险（波动率），得到单位风险对应的超额收益。

## 使用Python计算夏普比率

以下是使用Python和Pandas库计算多只股票夏普比率的步骤和代码示例。我们假设已经获取了股票的历史价格数据，并计算了每日收益率。

首先，我们需要准备数据。计算夏普比率前，通常需要对收益率数据中的缺失值进行处理。

```python
# 假设 returns 是一个Pandas DataFrame，包含多只股票的每日收益率
# 处理缺失值：使用前一天的收益率填充当天的缺失值
returns = returns.fillna(method='ffill')
```

接下来，我们定义无风险收益率（例如年化5%）并计算夏普比率。计算时需要注意将日收益率年化。

![](img/2fef2fce2dbda98b5e466cbde009d722_7.png)

```python
import numpy as np

# 设置年化无风险收益率，例如5%
risk_free_rate = 0.05
# 一年通常按250个交易日计算
trading_days_per_year = 250

# 计算超额收益率：日收益率 - 日化无风险收益率
excess_returns = returns - risk_free_rate / trading_days_per_year

# 计算夏普比率
# 步骤：1. 计算超额收益率的年化均值；2. 计算日收益率的标准差并年化；3. 两者相除
sharpe_ratios = (excess_returns.mean() * trading_days_per_year) / (returns.std() * np.sqrt(trading_days_per_year))

# 查看结果
print(sharpe_ratios)
```

执行以上代码后，我们将得到一个夏普比率序列。例如，输出可能类似：

```
股票A   -0.15
股票B    0.82
股票C    1.35
股票D    0.10
dtype: float64
```

## 如何解读结果

![](img/2fef2fce2dbda98b5e466cbde009d722_11.png)

根据计算结果，我们可以对不同的投资标的进行比较：

![](img/2fef2fce2dbda98b5e466cbde009d722_13.png)

*   **夏普比率为正**：表示投资组合跑赢了无风险收益。**数值越大越好**，例如股票C的夏普比率为1.35，意味着其单位风险带来的超额收益最高，是当前选择中最“划算”的。
*   **夏普比率为负**：表示投资组合的收益还不如无风险收益，例如股票A。通常我们会避免选择夏普比率为负的标的。
*   **比较决策**：在多个正值的夏普比率中，应优先选择数值最大的那个，因为它提供了最好的风险调整后回报。
