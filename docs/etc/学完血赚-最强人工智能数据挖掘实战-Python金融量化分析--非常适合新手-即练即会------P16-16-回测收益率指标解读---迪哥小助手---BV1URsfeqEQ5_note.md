## 什么是策略评估指标？
当我们完成一个交易策略的构建后，必须在历史数据中进行测试。策略评估指标的作用就是量化地告诉我们这个策略的表现。它需要回答几个关键问题：策略的绝对收益是多少？相比市场基准有何优势？策略承担了哪些风险？这些风险是否值得？以及与其他策略相比有何优劣？这些问题的答案都依赖于一系列评估指标的计算与分析。

## 回测收益率详解
### 定义与概念
**回测收益率**是最基本的策略收益指标。它衡量的是，在回测周期内，策略最终获得的收益相对于初始投入本金的比例。

例如，你从2015年到2020年对某只股票运行一个策略（决定何时买入、何时卖出），最终会得到一个结果。这个结果不是赚了多少钱的绝对值，而是相对于你最初投入的本金，资产增值了多少。如果指标显示为40%，就意味着你的本金在回测期间增长了40%。

### 计算公式
回测收益率的计算公式非常直观：

![](img/57b881d5b91a93be8c39e3843622d0b3_13.png)

![](img/57b881d5b91a93be8c39e3843622d0b3_14.png)

**回测收益率 = (期末资产总值 - 期初资产总值) / 期初资产总值**

![](img/57b881d5b91a93be8c39e3843622d0b3_16.png)

用符号表示为：
**R = (P_T - P_0) / P_0**

其中：
*   **P_T** 代表回测结束时的资产总值（卖出所有头寸后的总资金）。
*   **P_0** 代表回测开始时的初始本金。

![](img/57b881d5b91a93be8c39e3843622d0b3_18.png)

![](img/57b881d5b91a93be8c39e3843622d0b3_19.png)

这个公式计算的是策略在整个回测期间的总回报率。

## Python实战：计算回测收益率
上一节我们介绍了回测收益率的概念和公式，本节中我们来看看如何用Python代码实际计算它。我们将使用一份示例股票数据来模拟策略回测后的资产曲线。

首先，导入必要的工具包并加载数据。

```python
import pandas as pd

print(data.head())
```
执行后，你会看到数据的格式，其中包含如收盘价等字段。我们可以将`data['close']`的走势视为策略运行后每日的资产净值曲线。

### 计算过程分步解析
计算回测收益率的核心是计算期末相对于期初的增长比例。

1.  **计算累积净值曲线**：首先，我们计算每个时间点资产净值相对于期初净值的比值。这形成了一条以1为起点的累积收益曲线。
    ```python
    # 计算每个时间点的净值相对于期初的比值
    cumulative_return = data['close'] / data['close'].iloc[0]
    # 绘制累积收益曲线
    plt.figure(figsize=(10, 6))
    cumulative_return.plot()
    plt.title('策略累积收益曲线 (基准: 期初=1)')
    plt.ylabel('净值比例')
    plt.grid(True)
    plt.show()
    ```
    这张图展示了从期初开始，你的1元本金随时间的变化情况。

![](img/57b881d5b91a93be8c39e3843622d0b3_26.png)

![](img/57b881d5b91a93be8c39e3843622d0b3_27.png)

![](img/57b881d5b91a93be8c39e3843622d0b3_28.png)

2.  **计算最终回测收益率**：根据公式，回测收益率是期末净值比例减去1。
    ```python
    # 计算最终的回测收益率
    final_return = cumulative_return.iloc[-1] - 1
    # 将结果整理成DataFrame以便展示
    result_df = pd.DataFrame({
        '回测收益率': [final_return]
    }, index=['最终值'])
    print(result_df)
    ```
    执行这段代码，就会输出策略在整个回测期间的总收益率。如果值为正，表示盈利；为负，则表示亏损。
