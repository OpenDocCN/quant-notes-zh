## 概述

当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以评估其表现。评估不能仅用“赚钱”或“赔钱”来概括，需要更精确的量化指标。这些指标能告诉我们赚了多少、相比市场基准有何优势、承担了哪些风险，以及与其他策略相比的优劣。本节课，我们将聚焦于第一个关键指标：回测收益率。

## 回测收益率详解

![](img/b0eefa481d5ede80af03da65d4992340_5.png)

![](img/b0eefa481d5ede80af03da65d4992340_7.png)

回测收益率是评估策略表现最常用的指标，它衡量了策略在整个回测周期内，相对于初始本金的总体盈利或亏损比例。

![](img/b0eefa481d5ede80af03da65d4992340_9.png)

其核心计算公式如下：

![](img/b0eefa481d5ede80af03da65d4992340_11.png)

**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

![](img/b0eefa481d5ede80af03da65d4992340_13.png)

![](img/b0eefa481d5ede80af03da65d4992340_15.png)

其中：
*   **期末资产总值**：策略在回测结束时的总价值（包括现金和持仓市值）。
*   **期初资产总值**：策略在回测开始时的初始本金。

![](img/b0eefa481d5ede80af03da65d4992340_17.png)

![](img/b0eefa481d5ede80af03da65d4992340_19.png)

这个公式计算的是策略的**累计收益率**。结果为正表示盈利，为负表示亏损。

![](img/b0eefa481d5ede80af03da65d4992340_21.png)

## 代码实战：计算回测收益率

![](img/b0eefa481d5ede80af03da65d4992340_23.png)

上一节我们介绍了回测收益率的概念和公式，本节中我们来看看如何用Python代码实际计算它。我们将使用一个示例数据集来模拟策略回测后的每日资产净值曲线。

以下是计算步骤：

![](img/b0eefa481d5ede80af03da65d4992340_25.png)

![](img/b0eefa481d5ede80af03da65d4992340_27.png)

1.  导入必要的工具包并加载数据。
2.  计算每日资产净值相对于初始净值的比值（即累计收益曲线）。
3.  根据公式计算整个周期的总回测收益率。

```python
# 1. 导入工具包
import pandas as pd
import matplotlib.pyplot as plt

# 设置中文字体（示例，根据系统调整）
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设‘close’列代表策略每日的资产净值
data = pd.read_csv('your_strategy_data.csv', index_col=0, parse_dates=True)
# 为演示，我们假设‘close’列就是净值。实际中，这应是策略计算出的每日总资产。
net_value = data['close']

# 2. 计算累计收益曲线：每日净值 / 初始净值
cumulative_return_ratio = net_value / net_value.iloc[0]

![](img/b0eefa481d5ede80af03da65d4992340_29.png)

![](img/b0eefa481d5ede80af03da65d4992340_31.png)

# 绘制累计收益曲线图
plt.figure(figsize=(12, 6))
cumulative_return_ratio.plot()
plt.title('策略累计收益曲线')
plt.ylabel('累计收益倍数 (初始净值=1)')
plt.xlabel('日期')
plt.grid(True)
plt.show()

![](img/b0eefa481d5ede80af03da65d4992340_33.png)

# 3. 计算总回测收益率：(最终净值 / 初始净值) - 1
total_return = (net_value.iloc[-1] / net_value.iloc[0]) - 1

![](img/b0eefa481d5ede80af03da65d4992340_35.png)

![](img/b0eefa481d5ede80af03da65d4992340_37.png)

![](img/b0eefa481d5ede80af03da65d4992340_39.png)

# 将结果整理成DataFrame展示
result_df = pd.DataFrame({
    '回测收益率': [total_return]
}, index=['策略表现'])
print(result_df)
```

**代码解释**：
*   `net_value.iloc[0]` 获取了期初的资产净值（初始本金）。
*   `net_value.iloc[-1]` 获取了期末的资产净值。
*   `cumulative_return_ratio` 序列展示了从期初到任意一天，资产相对于本金的变化倍数。当该值大于1时，表示盈利；小于1时，表示亏损。
*   `total_return` 即为我们最终计算出的回测收益率。
