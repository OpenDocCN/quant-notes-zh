## 自相关函数

上一节我们介绍了ARIMA模型的基本概念，本节中我们来看看如何确定模型中的阶数。首先介绍**自相关函数**。

自相关函数用于衡量同一个时间序列在不同时间点上的观测值之间的相关性。它描述的是当前值`Y_t`与过去值`Y_{t-k}`（k为滞后阶数）之间的线性关系。

相关系数用于衡量两个变量之间的关联程度。其取值范围在-1到1之间：
*   **+1** 表示完全正相关。
*   **-1** 表示完全负相关。
*   **0** 表示无线性相关关系。

自相关函数是相关系数概念在时间序列自身的延伸。我们计算`Y_t`与`Y_{t-1}`、`Y_t`与`Y_{t-2}`等不同滞后阶数下的相关系数。

自相关函数的公式如下：

\[
ACF(k) = \frac{\text{Cov}(Y_t, Y_{t-k})}{\sqrt{\text{Var}(Y_t) \cdot \text{Var}(Y_{t-k})}}
\]

其中，`Cov`代表协方差，`Var`代表方差，`k`代表滞后阶数。

ACF的取值范围同样在-1到1之间。我们通常通过ACF图来观察其随滞后阶数`k`变化的趋势。图中会包含一个置信区间（常取95%），用于判断哪些自相关系数是统计显著的。

以下是解读ACF图的关键点：
*   **横轴**：滞后阶数 `k`。
*   **纵轴**：自相关系数 `ACF(k)` 的值。
*   **置信区间**：图中的虚线区域。如果ACF值超出此区域，则认为该阶数的自相关性是显著的。

## 偏自相关函数

说完了自相关函数，接下来我们介绍**偏自相关函数**。PACF在ARIMA模型定阶中扮演着同样关键的角色。

偏自相关函数衡量的是，在剔除了中间滞后项（`Y_{t-1}, Y_{t-2}, ..., Y_{t-k+1}`）的影响后，当前值`Y_t`与过去值`Y_{t-k}`之间的纯相关性。

可以这样理解：
*   **ACF** 计算`Y_t`与`Y_{t-k}`的相关性时，包含了通过中间变量`Y_{t-1}, ..., Y_{t-k+1}`传递的间接影响。
*   **PACF** 则移除了这些中间变量的干扰，只衡量`Y_t`与`Y_{t-k}`之间的直接影响。

因此，PACF能更清晰地揭示时间序列自身的真实结构，特别是在识别AR模型的阶数`p`时非常有用。

## 使用Python绘制ACF与PACF图

了解了核心概念后，我们来看看如何用Python工具实现可视化。我们将使用`statsmodels`库。

首先需要安装`statsmodels`库。如果在使用`pip`安装时遇到网络问题，可以手动下载`.whl`文件进行安装。

以下是安装步骤的代码示例：

```python
# 方法一：使用pip在线安装（推荐网络通畅时使用）
# 在Anaconda Prompt或终端中执行：
# pip install statsmodels

![](img/50951204ddbd56f7d5eb57266e6d5843_1.png)

# 方法二：手动安装.whl文件（解决网络问题）
# 1. 从指定网站下载对应Python版本和系统位数的.whl文件（如statsmodels-0.13.2-cp39-cp39-win_amd64.whl）
# 2. 在命令行中进入.whl文件所在目录
# 3. 执行安装命令，例如：
# pip install statsmodels-0.13.2-cp39-cp39-win_amd64.whl
```

![](img/50951204ddbd56f7d5eb57266e6d5843_3.png)

安装成功后，即可使用以下代码绘制ACF和PACF图。

以下是绘制图形的完整代码示例：

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf

![](img/50951204ddbd56f7d5eb57266e6d5843_5.png)

# 假设我们有一个名为‘data’的时间序列数据（Pandas Series格式）
# data = pd.read_csv('your_data.csv', index_col='date', parse_dates=True)['value']

# 设置图形大小
plt.figure(figsize=(12, 6))

# 绘制ACF图 (Autocorrelation Function)
plt.subplot(121) # 1行2列的第1个位置
plot_acf(data, lags=20, ax=plt.gca()) # lags参数指定显示的最大滞后阶数
plt.title('自相关函数图')

# 绘制PACF图 (Partial Autocorrelation Function)
plt.subplot(122) # 1行2列的第2个位置
plot_pacf(data, lags=20, ax=plt.gca(), method='ywm') # method指定PACF的计算方法
plt.title('偏自相关函数图')

plt.tight_layout() # 自动调整子图间距
plt.show()
```

![](img/50951204ddbd56f7d5eb57266e6d5843_7.png)

代码说明：
1.  导入必要的库：`pandas`, `numpy`, `matplotlib.pyplot` 以及 `statsmodels` 中的绘图函数。
2.  `plot_acf()` 函数用于绘制自相关函数图。
3.  `plot_pacf()` 函数用于绘制偏自相关函数图。
4.  参数 `lags` 控制显示的滞后阶数范围。
5.  图形通常并排显示，便于对比分析。
