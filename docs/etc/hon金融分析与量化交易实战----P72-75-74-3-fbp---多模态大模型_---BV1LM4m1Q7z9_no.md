首先是 `model = cf.create_model()` 这一行。我们跳转到 `create_model` 函数的定义中。第一步是重载绘图工具，然后实例化模型。在实例化时，会指定一系列参数。

**关键参数说明：**
*   `seasonality_mode`: 是否绘制季节性变化图，可设为 `True` 或 `False`。
*   `changepoint_prior_scale`: 这是一个关键参数，默认值为 `0.05`。Prophet框架的默认值也是 `0.05`，我们直接使用即可。这个参数控制模型对趋势变化的灵活度。

如果你想查看所有参数的设计或模型定义，可以在IDE中（如PyCharm或Eclipse）使用 `Ctrl+鼠标左键` 点击函数名跳转到源码。

![](img/d23681d8f22286f89f440b51210cb32e_23.png)

在源码中，你可以详细阅读每个参数的含义。除了上述参数，你还可以指定是否按年(`yearly_seasonality`)、按周(`weekly_seasonality`)展示趋势，是否考虑节假日(`holidays`)等。由于源码内容较多，我们在此不深入阅读，感兴趣可以自行研究。

![](img/d23681d8f22286f89f440b51210cb32e_33.png)

在实例化模型时，我们传入了指定的参数，并额外添加了一个以30.5天为周期的变化趋势。这样，我们就得到了一个基础的时间序列模型。

```python
# 示例：创建Prophet模型并添加自定义季节性
from prophet import Prophet
model = Prophet(changepoint_prior_scale=0.05, yearly_seasonality=True)
model.add_seasonality(name='monthly', period=30.5, fourier_order=5)
```

模型构建完成后，需要加载数据进行训练。这里有一个细节：代码中使用了 `self.training_years` 参数。

![](img/d23681d8f22286f89f440b51210cb32e_35.png)

![](img/d23681d8f22286f89f440b51210cb32e_29.png)

它的含义是，我们可以只用最近几年的历史数据进行训练和预测。例如，如果只想用近3年的数据，就可以设置 `training_years=3`。这样，模型就只基于最近3年的数据来学习规律。

![](img/d23681d8f22286f89f440b51210cb32e_37.png)

![](img/d23681d8f22286f89f440b51210cb32e_39.png)

训练时，调用 `model.fit()` 方法。训练完成后，使用 `make_future_dataframe` 方法生成待预测的时间段。这里将参数 `periods` 设置为 `0`。

![](img/d23681d8f22286f89f440b51210cb32e_41.png)

![](img/d23681d8f22286f89f440b51210cb32e_43.png)

将 `periods` 设为 `0` 意味着我们只对历史数据进行拟合和预测，而不对未来新的时间点进行预测。稍后我们会演示设置 `periods=90` 或 `100` 来进行未来预测的情况。目前，我们只关注模型在历史数据上的表现。

![](img/d23681d8f22286f89f440b51210cb32e_45.png)

那么，历史数据如何预测呢？`make_future_dataframe` 生成的 `future` 数据框，当 `periods=0` 时，其时间范围与训练数据一致。本质上，就是用同一份数据训练，再用同一份数据预测。这个过程执行起来可能会稍慢。

![](img/d23681d8f22286f89f440b51210cb32e_47.png)

预测完成后，结果中会包含几个自动添加的列，如 `yhat`、`yhat_lower`、`yhat_upper`。我们可以将 `yhat` 作为预测值，而 `yhat_lower` 和 `yhat_upper` 则构成了置信区间。

![](img/d23681d8f22286f89f440b51210cb32e_49.png)

![](img/d23681d8f22286f89f440b51210cb32e_31.png)

![](img/d23681d8f22286f89f440b51210cb32e_51.png)

![](img/d23681d8f22286f89f440b51210cb32e_53.png)

接下来是绘图部分。我们创建子图，分别绘制：
*   观测值：`ds`（日期）和 `y`（真实值），用黑色点表示。
*   预测值：`future[‘ds’]`（日期）和 `future[‘yhat’]`（预测值），用绿色线表示。
*   置信区间：使用 `fill_between` 函数绘制 `yhat_lower` 和 `yhat_upper` 之间的区域。

![](img/d23681d8f22286f89f440b51210cb32e_55.png)

![](img/d23681d8f22286f89f440b51210cb32e_57.png)

这样，我们就得到了最终的预测图。中间的浅绿色区域是置信区间，黑色点是真实值，绿色线是模型的预测值。

## 突变点分析 🔍

![](img/d23681d8f22286f89f440b51210cb32e_85.png)

上一节我们分析了数据的周期性趋势。本节中，我们来探讨一个重要的概念：突变点。

![](img/d23681d8f22286f89f440b51210cb32e_87.png)

突变点是指时间序列中趋势发生显著变化的点。例如，从下降趋势突然转为上升的拐点，或者上升速度突然加快/减慢的点。在Prophet中，这被称为 `changepoint`。

![](img/d23681d8f22286f89f440b51210cb32e_89.png)

![](img/d23681d8f22286f89f440b51210cb32e_71.png)

我们可以通过观察时间序列的二阶导数的正负情况来分析这些突变点。

![](img/d23681d8f22286f89f440b51210cb32e_91.png)

以下是分析突变点的步骤：
1.  创建模型并进行训练。
2.  获取预测结果和训练集数据，并按日期合并。
3.  从模型中提取计算出的突变点。
4.  根据突变点的索引找到对应的日期。
5.  检查每个突变点处的二阶导数值（模型已计算好），判断其大于零（正向变化）还是小于零（负向变化）。
6.  根据导数值的正负，用不同颜色的线条在图上标出这些突变点。

![](img/d23681d8f22286f89f440b51210cb32e_93.png)

![](img/d23681d8f22286f89f440b51210cb32e_95.png)

![](img/d23681d8f22286f89f440b51210cb32e_85.png)

在结果图中，绿色线条表示趋势转为正向增长的突变点，红色线条表示趋势转为负向增长的突变点。

![](img/d23681d8f22286f89f440b51210cb32e_97.png)

这样，我们就成功识别并可视化了时间序列中的关键转折点。

## 未来预测 🚀

![](img/d23681d8f22286f89f440b51210cb32e_97.png)

前面我们主要关注对历史数据的拟合。接下来，我们看看如何用模型预测未来。

![](img/d23681d8f22286f89f440b51210cb32e_99.png)

![](img/d23681d8f22286f89f440b51210cb32e_101.png)

预测未来的步骤与之前类似，核心区别在于 `make_future_dataframe` 函数中的 `periods` 参数。例如，我们设置 `periods=180`，表示要预测未来180天。

```python
# 预测未来180天
future = model.make_future_dataframe(periods=180)
forecast = model.predict(future)
```

![](img/d23681d8f22286f89f440b51210cb32e_103.png)

此时，生成的 `future` 数据框将包含从历史数据开始日期，一直延续到未来180天后的所有日期。模型会对这些未来的日期也给出预测值。

![](img/d23681d8f22286f89f440b51210cb32e_105.png)

绘图方法不变。在结果图中，黑色点（真实值）只到当前日期（例如2018年1月26日），而绿色预测线会向后延伸180天，展示未来的预测趋势。

![](img/d23681d8f22286f89f440b51210cb32e_107.png)

从预测图可以看到，模型基于历史趋势，推测未来价格可能会先经历一段下降，然后再上升。这体现了“物极必反”的思想，上涨过多后可能会回调。当然，这个预测可能相对保守，实际的未来波动或许更大。

![](img/d23681d8f22286f89f440b51210cb32e_107.png)

预测的“保守”与“奔放”程度，可以通过一个关键参数来调整。

![](img/d23681d8f22286f89f440b51210cb32e_109.png)

## 核心参数：changepoint_prior_scale ⚙️

前面我们多次提到了 `changepoint_prior_scale` 参数，现在来详细解释它的作用。

这个参数控制着模型对于趋势变化的灵活度或敏感度。
*   **值较大**（如 `0.5`）：模型会更灵活，允许趋势线出现更多的突变点，从而更好地拟合历史数据中的波动，但在预测未来时可能波动更大、更“奔放”。
*   **值较小**（如 `0.01`）：模型会更保守，趋势线更平滑，对历史数据的波动不那么敏感，预测未来的趋势也会更平缓。

![](img/d23681d8f22286f89f440b51210cb32e_111.png)

你可以通过调整这个参数，使你的时间序列模型更适合你的数据特性和预测需求。例如，如果你认为未来趋势会剧烈变化，可以适当调高该值；如果你希望预测更平滑稳健，则可以调低该值。

![](img/d23681d8f22286f89f440b51210cb32e_113.png)
