上一节我们介绍了NumPy、Pandas和Matplotlib这三大数据分析工具。本节中，我们来看看如何获取真实的金融数据进行分析。要进行金融分析，首先需要数据。`tushare` 包就是一个专门用于获取国内股票数据的Python接口。

## 安装与导入
安装 `tushare` 包非常简单，通常使用 `pip` 命令即可。在安装过程中，如果提示缺少某些依赖包，只需根据提示安装相应的依赖包，然后再安装 `tushare` 即可成功。

以下是导入 `tushare` 的代码，通常我们使用 `ts` 作为其别名以简化代码。
```python
import tushare as ts
```

![](img/743f9a9e5524ba91aa5b7370604f6dfe_27.png)

## 核心接口：获取历史行情数据
`tushare` 提供了多个数据接口，其中最常用的是获取历史行情数据的接口。该接口可以获取股票的日K线、周K线、月K线等数据。

获取历史行情数据的主要函数是 `get_k_data()`。它返回一个Pandas的DataFrame结构，方便我们进行后续的数据处理和分析。

![](img/743f9a9e5524ba91aa5b7370604f6dfe_29.png)

以下是 `get_k_data()` 函数的基本用法示例：
```python
df = ts.get_k_data(code='601318')
```
执行上述代码后，`df` 变量中将存储股票代码为‘601318’的历史行情数据。数据默认包含以下列：日期（date）、开盘价（open）、收盘价（close）、最高价（high）、最低价（low）、成交量（volume）和股票代码（code）。

![](img/743f9a9e5524ba91aa5b7370604f6dfe_31.png)

![](img/743f9a9e5524ba91aa5b7370604f6dfe_33.png)

## 函数参数详解
`get_k_data()` 函数有几个重要的参数，用于控制获取数据的范围、类型和内容。

![](img/743f9a9e5524ba91aa5b7370604f6dfe_35.png)

以下是各参数的详细说明：
*   **code**：股票代码。例如，`‘601318’` 代表中国平安。
*   **ktype**：K线类型。默认为 `‘D’`（日K线）。其他选项包括 `‘W’`（周K线）、`‘M’`（月K线）、`‘5’`（5分钟K线）等。
*   **autype**：复权类型。默认为 `‘qfq’`（前复权）。复权处理涉及股票分红、增发时的股价调整，对于初学者，使用默认的前复权即可。
*   **index**：是否为指数。当需要获取大盘指数数据（如上证指数 `‘000001’`）时，需将此参数设置为 `True`。
*   **start**：数据开始日期。格式为 `‘YYYY-MM-DD’`。如果不指定，默认从较早的日期开始。
*   **end**：数据结束日期。格式为 `‘YYYY-MM-DD’`。如果不指定，默认获取到最新日期。

![](img/743f9a9e5524ba91aa5b7370604f6dfe_37.png)

![](img/743f9a9e5524ba91aa5b7370604f6dfe_39.png)

一个获取指定时间段内股票数据的示例如下：
```python
df = ts.get_k_data(code='601318', start='2007-01-01', end='2017-12-31')
```

## 数据的保存与应用
获取到数据后，我们可以将其保存到本地文件（如CSV格式），以便后续重复使用，无需每次都从网络下载。

![](img/743f9a9e5524ba91aa5b7370604f6dfe_41.png)

![](img/743f9a9e5524ba91aa5b7370604f6dfe_43.png)

保存数据到CSV文件的代码如下：
```python
df.to_csv(‘601318.csv’)
```
保存之后，在后续的分析中，我们只需使用Pandas读取这个CSV文件即可。

## 其他数据接口简介
除了历史行情数据，`tushare` 还提供了其他多种数据接口，可以满足不同的分析需求。

![](img/743f9a9e5524ba91aa5b7370604f6dfe_49.png)

以下是部分其他接口的简介：
*   **实时行情数据**：获取当日交易数据。
*   **分笔历史数据**：获取详细的历史成交记录。
*   **实时分笔数据**：获取最新的成交明细。
*   **大盘指数数据**：获取各类大盘指数的实时或历史数据。

![](img/743f9a9e5524ba91aa5b7370604f6dfe_51.png)

这些接口都有对应的函数，具体用法可以参考 `tushare` 的官方文档。对于本课程后续的简单分析项目，掌握 `get_k_data()` 接口获取历史数据已经足够。
