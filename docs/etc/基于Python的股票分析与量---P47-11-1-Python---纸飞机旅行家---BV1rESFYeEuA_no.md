## 量化策略回测的整体流程 🔄

上一章我们介绍了量化择时，本节中我们来看看量化策略回测。量化策略回测的整体流程可以概括为七个主要步骤。

以下是量化策略回测的七个核心步骤：
1.  **选择股票池和实现回测函数**：根据策略选定股票范围，并编写核心的回测函数。
2.  **设定回测参数**：包括回测的起止时间、初始资金以及调仓频率。
3.  **载入历史数据并初始化**：平台调用`handle_data`函数，为策略运行做好准备。
4.  **平台自动处理订单**：根据策略逻辑和市场价格，系统生成并尝试执行交易订单。
5.  **处理未完成订单**：系统会自动取消无法成交的订单（例如，试图卖出的数量超过持仓量）。
6.  **输出运行日志**：在回测过程中记录交易详情、持仓变化等信息。
7.  **生成回测报告**：回测结束后，系统自动生成包含收益曲线、各项指标的分析报告。

本章内容与第一章介绍的回测流程在深度和侧重点上有所不同。本章主要聚焦于在聚宽平台上实现回测的具体流程、内部机制以及常见的回测分析指标。

## 量化策略回测的步骤分解 🔍

上一节我们介绍了回测的整体框架，本节中我们将对每一步进行详细分解，并说明在聚宽平台上如何操作。

### 第一步：策略准备与函数实现

这一步是回测的准备工作，核心是在量化平台上编写交易策略。重点在于实现`handle_data`函数，该函数定义了每个调仓周期（如每天）的具体交易逻辑。

在页面上，这对应于“编写策略”的部分。我们需要在此选择股票池（例如基于白马股或MACD策略），并完成`handle_data`函数的代码编写。

![](img/b2772fdb94f5adcb107810864c2b24d3_1.png)

### 第二步：设定回测参数

这一步需要设定回测的基本环境。我们需要在平台上指定回测的开始与结束日期、初始资金金额以及策略运行的频率（例如每日运行）。

这里需要注意区分“回测时间段”和“数据获取日期”。回测时间段是你希望模拟交易的时期，而数据日期则取决于你在`get_data`等函数中指定的参数，它可能包含更早的历史数据用于指标计算。

### 第三步：数据载入与函数调用

![](img/b2772fdb94f5adcb107810864c2b24d3_3.png)

设定好参数后，点击运行，平台便会开始载入选定时间段的历史数据，并调用我们编写好的`handle_data`函数。此时，系统日志开始输出，标志着回测正式启动运行。

### 第四步及以后：平台自动化处理

![](img/b2772fdb94f5adcb107810864c2b24d3_5.png)

从第四步开始，后续步骤主要由聚宽量化平台自动完成。

以下是平台自动处理的关键环节：
*   **处理订单**：平台根据`handle_data`函数发出的指令以及当时的市场行情，生成并尝试执行交易订单。
*   **取消未完成订单**：对于因各种原因（如资金不足、涨停无法买入等）无法立即成交的订单，平台会自动取消。开发者也可以主动调用`get_open_orders()`获取未成交订单列表，并调用`cancel_order()`取消指定订单。
*   **输出日志**：平台会自动记录每一笔交易的详情。开发者也可以在代码中任意位置使用`log.info()`或`print()`函数添加自定义日志，以便于调试和观察策略运行状态。
*   **生成回测报告**：当所有回测日期运行完毕后，平台会自动生成一份详细的回测报告。

回测报告通常包含策略收益曲线图、基准（如沪深300指数）收益曲线对比图、每日持仓和交易明细表格，以及一系列量化评估指标（如年化收益率、阿尔法、贝塔、夏普比率、最大回撤等）。

## 实战演示：MACD策略回测实例 💻

![](img/b2772fdb94f5adcb107810864c2b24d3_7.png)

前面我们分解了回测的各个步骤，现在让我们通过一个具体的MACD策略实例，来直观感受整个回测流程在聚宽平台上是如何运行的。

我们使用一个预先编写好的MACD交易策略进行演示。策略代码的核心部分在`handle_data`函数中，它计算MACD指标，并根据金叉死叉信号决定买入或卖出操作。

在平台界面上，我们设定了回测时间段、初始资金（例如10万元）和调仓频率（每日）。首先点击“编译运行”，平台开始逐日模拟交易，并在下方输出区域打印每一日的交易日志。

初步运行完成后，我们可以看到一个简要的回测结果摘要。例如，策略总收益率为19%，而同期沪深300指数下跌-5.57%，策略表现出了超额收益（阿尔法为0.13）。同时，我们也能看到夏普比率为0.38，最大回撤为-17.54%等关键指标。

为了获得更详尽的分析，我们可以点击“运行回测”来生成完整的回测报告。报告会以丰富的图表形式展示策略收益曲线与基准曲线的对比，并列出所有详细的评估指标。此外，我们还可以查看“交易详情”和“每日持仓”等标签页，深入了解策略每一笔操作的具体情况。
