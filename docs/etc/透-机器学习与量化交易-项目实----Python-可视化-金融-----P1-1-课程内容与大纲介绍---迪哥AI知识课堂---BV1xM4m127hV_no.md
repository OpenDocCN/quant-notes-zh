![](img/94d239be965d1f76001f5ba838e765c9_27.png)

*   **Jupyter Notebook**：用于演示Python工具包的使用和完成数据分析案例。核心工具包包括用于数值计算的 **`NumPy`** 和用于数据分析的 **`Pandas`**。
*   **量化回测平台**：用于将编写好的交易策略应用到历史数据中，模拟交易并分析结果。在平台中，我们可以编写策略、执行回测，并观察详细的交易记录、持仓情况和账户信息。

![](img/94d239be965d1f76001f5ba838e765c9_31.png)

例如，在回测平台中，我们可以初始化一个模拟账户并运行策略：
```python
# 示例：初始化回测上下文
context = initialize(context)  # 初始化策略上下文
context.cash = 100000  # 设置初始资金为10万元
```
回测结束后，可以分析策略相对于基准（如大盘指数）的**超额收益**等各项指标。

## 详细课程大纲

接下来，我们详细看一下课程大纲的三个模块。

![](img/94d239be965d1f76001f5ba838e765c9_41.png)

以下是课程大纲的详细分解：

![](img/94d239be965d1f76001f5ba838e765c9_43.png)

1.  **第一模块：Python必备工具包实战**
    *   本模块为基础部分，讲解Python核心知识点及环境配置。
    *   重点介绍 **`NumPy`** （数值计算）和 **`Pandas`** （数据处理）库的使用。

![](img/94d239be965d1f76001f5ba838e765c9_45.png)

2.  **第二模块：Python金融数据分析实战**
    *   本模块进入实战，使用真实股票数据。
    *   内容涵盖数据获取、分析、建模、统计。
    *   引导在量化平台中完成第一个回测项目，评估策略历史表现。

![](img/94d239be965d1f76001f5ba838e765c9_47.png)

![](img/94d239be965d1f76001f5ba838e765c9_49.png)

3.  **第三模块：量化交易策略深入实战**
    *   本模块深入量化交易核心。
    *   讲解经典交易策略、实战中的数据处理方法、因子分析。
    *   探讨如何改进策略，以期在历史回测中获得更优的收益结果。
