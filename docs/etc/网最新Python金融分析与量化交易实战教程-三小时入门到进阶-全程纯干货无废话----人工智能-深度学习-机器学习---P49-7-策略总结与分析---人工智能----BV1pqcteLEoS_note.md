![](img/57e643a2d04c039ec2bf886de37f9bc4_36.png)

![](img/57e643a2d04c039ec2bf886de37f9bc4_38.png)

1.  **测试2016年至2018年**：将回测时间设置为2016年1月4日至2018年1月4日。执行速度很快。结果显示，策略收益与基准收益相差不大，没有显著优势。
2.  **测试2018年至2020年**：将回测时间修改为2018年1月4日至2020年1月4日。策略逻辑和各项计算指标均无误。但在此区间内，策略收益（约2%）仅略高于基准收益（约0%），效果仍不明显。这可能与2018年的熊市行情有关。
3.  **测试2016年全年**：为避免熊市影响，测试2016年1月4日至2017年1月4日的表现。此时策略收益明显高于基准收益，效果改善。
4.  **测试排名靠后的股票**：此前策略是选取综合评分**排名前10**的股票。现在修改代码，改为选取**排名最后**的股票（即指标表现最差的股票）。回测结果显示，使用差股票的收益（约-77万）比基准收益（约-10%）更差。这从反面验证了评分指标的有效性——评分高的股票组合表现优于评分低的组合。
