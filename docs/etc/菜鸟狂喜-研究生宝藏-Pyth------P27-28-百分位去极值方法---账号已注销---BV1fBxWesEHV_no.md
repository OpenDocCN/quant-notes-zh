## 数据预处理三步走
因子数据的预处理通常遵循三个步骤：去极值、标准化和中性化。前两个步骤在机器学习和数据挖掘中很常见，而中性化则在因子策略分析中尤为重要。本节课，我们将首先深入探讨**去极值**的方法。

## 去极值方法：百分位法
处理极值（或异常值）的一种常见方法是设定边界，将超出边界的数值“拉回”到边界上，而不是直接删除。这能保留数据样本，同时减少极端值的影响。以下是实现这一目标的一种方法。

![](img/85f881c47648b3a173680bd2900ac5ff_7.png)

![](img/85f881c47648b3a173680bd2900ac5ff_9.png)

### 理解分位数
分位数是帮助我们理解数据分布位置的关键概念。最熟悉的分位数是中位数，它代表了数据集的中间值。与均值相比，中位数对极端值不敏感，更能代表数据的“典型”情况。

![](img/85f881c47648b3a173680bd2900ac5ff_11.png)

![](img/85f881c47648b3a173680bd2900ac5ff_13.png)

![](img/85f881c47648b3a173680bd2900ac5ff_15.png)

除了中位数，常用的还有四分位数：
*   **第一四分位数 (Q1)**：所有数据中，有25%的数据小于等于此值。
*   **中位数 (Q2)**：所有数据中，有50%的数据小于等于此值。
*   **第三四分位数 (Q3)**：所有数据中，有75%的数据小于等于此值。

![](img/85f881c47648b3a173680bd2900ac5ff_17.png)

![](img/85f881c47648b3a173680bd2900ac5ff_19.png)

### 利用分位数设定边界
我们可以利用Q1和Q3来定义数据的正常范围。一个常用的方法是计算**四分位距**，并以此设定上下界。

四分位距的计算公式为：
```
IQR = Q3 - Q1
```

![](img/85f881c47648b3a173680bd2900ac5ff_21.png)

基于IQR，我们可以设定正常值的下限和上限：
```
下限 = Q1 - k * IQR
上限 = Q3 + k * IQR
```
其中，`k`是一个常数，通常取1.5。任何小于下限或大于上限的数值都被视为极值。

### 处理极值
对于识别出的极值，我们将其数值调整到对应的边界上：
*   如果数值小于下限，则将其设置为下限的值。
*   如果数值大于上限，则将其设置为上限的值。

![](img/85f881c47648b3a173680bd2900ac5ff_23.png)

![](img/85f881c47648b3a173680bd2900ac5ff_25.png)

这种方法通过以下代码实现：
```python
def percentile_winsorize(data, k=1.5):
    """
    使用百分位法对数据进行去极值处理。
    :param data: 输入的数据序列（如Pandas Series）
    :param k: 用于计算边界的常数，默认为1.5
    :return: 处理后的数据序列
    """
    Q1 = data.quantile(0.25)
    Q3 = data.quantile(0.75)
    IQR = Q3 - Q1
    lower_bound = Q1 - k * IQR
    upper_bound = Q3 + k * IQR
    # 将超出边界的值拉回到边界
    data_winsorized = data.clip(lower=lower_bound, upper=upper_bound)
    return data_winsorized
```

## 总结
本节课中，我们一起学习了因子数据预处理的第一步——去极值。我们重点介绍了**百分位法**，该方法通过计算数据的四分位数和四分位距来定义正常值的范围，并将超出此范围的极值调整到边界上。这种方法能有效平滑异常值的影响，为后续的标准化和中性化处理奠定基础。下一节，我们将继续学习数据预处理的第二步：标准化。
