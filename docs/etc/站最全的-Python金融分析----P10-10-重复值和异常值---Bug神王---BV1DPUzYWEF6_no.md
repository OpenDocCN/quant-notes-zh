### 处理重复数据 🔄

![](img/585719e9b013841d1bf490bb16fa0a7b_15.png)

![](img/585719e9b013841d1bf490bb16fa0a7b_17.png)

重复数据指的是在数据集中完全相同的行。这些重复项可能会影响分析结果的准确性，因此需要被移除。

以下是创建包含重复行数据集的步骤：

![](img/585719e9b013841d1bf490bb16fa0a7b_19.png)

![](img/585719e9b013841d1bf490bb16fa0a7b_21.png)

1.  首先，我们使用NumPy生成一个8行4列的随机整数DataFrame。
    ```python
    import pandas as pd
    import numpy as np

    DF = pd.DataFrame(np.random.randint(0, 100, size=(8, 4)))
    ```
2.  接着，我们手动创建几行重复数据。例如，让索引为2、4、5的行数据都等于`[0, 0, 0, 0]`。
    ```python
    DF.iloc[2] = [0, 0, 0, 0]
    DF.iloc[4] = [0, 0, 0, 0]
    DF.iloc[5] = [0, 0, 0, 0]
    ```
    现在，DataFrame中就有了三行完全相同的重复数据。

![](img/585719e9b013841d1bf490bb16fa0a7b_23.png)

清洗重复数据非常简单，使用Pandas的`.drop_duplicates()`方法即可。

![](img/585719e9b013841d1bf490bb16fa0a7b_25.png)

该方法的关键参数是`keep`，用于指定保留哪一行重复数据：
*   `keep=‘first‘`：保留第一次出现的行（默认值）。
*   `keep=‘last‘`：保留最后一次出现的行。
*   `keep=False`：删除所有重复的行。

我们通常使用默认设置，保留首次出现的行并删除后续重复项。
```python
DF_cleaned = DF.drop_duplicates(keep=‘first‘)
```
执行后，只有索引为2的行被保留，索引为4和5的重复行已被删除。

![](img/585719e9b013841d1bf490bb16fa0a7b_27.png)

### 处理异常数据 🚨

![](img/585719e9b013841d1bf490bb16fa0a7b_29.png)

异常数据是指那些明显偏离正常范围或不符合业务逻辑的值。例如，一个蔬菜大棚的温度传感器读数突然达到120°C，这显然是一个异常值。

![](img/585719e9b013841d1bf490bb16fa0a7b_31.png)

![](img/585719e9b013841d1bf490bb16fa0a7b_33.png)

处理异常值通常分为两步：**定义异常判定条件**，然后**根据条件筛选或删除数据**。

我们通过一个示例来学习。假设我们有一个包含1000行、3列（A, B, C）的随机数据集，数值范围在0到1之间。
```python
DF = pd.DataFrame(np.random.random(size=(1000, 3)), columns=[‘A‘, ‘B‘, ‘C‘])
```
我们的任务是：**清洗掉C列中数值大于该列两倍标准差的异常值**。

![](img/585719e9b013841d1bf490bb16fa0a7b_35.png)

以下是处理步骤：

![](img/585719e9b013841d1bf490bb16fa0a7b_37.png)

![](img/585719e9b013841d1bf490bb16fa0a7b_39.png)

1.  **计算判定阈值**：首先，计算C列的两倍标准差。
    ```python
    twice_std = DF[‘C‘].std() * 2
    ```
2.  **生成布尔掩码**：判断C列中哪些值大于这个阈值。结果为True表示是异常值。
    ```python
    condition = DF[‘C‘] > twice_std
    ```
3.  **取反并筛选**：我们的目的是保留正常值（即`condition`为False的行）。因此对布尔序列取反（`~`），然后用它来索引原DataFrame。
    ```python
    DF_cleaned = DF[~condition]
    ```

![](img/585719e9b013841d1bf490bb16fa0a7b_41.png)

![](img/585719e9b013841d1bf490bb16fa0a7b_43.png)

操作完成后，新的`DF_cleaned`数据集中，C列的所有值都将小于或等于两倍标准差，异常值已被成功过滤掉。
