## 概述

上一节我们介绍了投资回报率，本节中我们来看看如何衡量“性价比”更高的投资。夏普比率正是用来描述：在承担单位风险的情况下，我们能获得多少超额收益。指标越高，意味着投资组合的“风险回报效率”越好。

## 夏普比率的定义与意义

![](img/645957605eff965fad3c75f11f3debc1_1.png)

夏普比率描述的是，对于单位风险而言，投资所能带来的收益。指标越高，表示在承担相同风险的情况下，获得的收益越高。

![](img/645957605eff965fad3c75f11f3debc1_3.png)

在选股或评估投资组合时，如果单看夏普比率这个指标，我们倾向于选择数值更高的，因为它意味着更高的风险调整后收益。

![](img/645957605eff965fad3c75f11f3debc1_5.png)

## 夏普比率的计算公式

理解风险与收益的关系后，我们来看如何计算这个指标。计算时，我们需要考虑两个关键部分：投资组合的超额收益和其波动性（风险）。

以下是夏普比率的计算公式：

**夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差**

其中：
*   **投资组合收益率**：是你所选择的股票或投资组合的回报率。
*   **无风险收益率**：通常指国债等几乎无风险投资的收益率，例如年化5%。
*   **投资组合收益率的标准差**：代表了投资组合收益的波动性，即风险。

这个公式的核心思想是：计算每承担一单位风险（标准差），能获得多少超出无风险收益的回报。

## Python代码实现

接下来，我们通过Python代码来计算夏普比率。在计算前，我们需要确保收益率数据是完整的，因此通常会处理缺失值。

![](img/645957605eff965fad3c75f11f3debc1_7.png)

以下是计算夏普比率的关键步骤代码：

![](img/645957605eff965fad3c75f11f3debc1_9.png)

```python
import numpy as np
import pandas as pd

# 假设 returns 是一个包含各股票每日收益率的Pandas Series或DataFrame
# 步骤1: 处理缺失值（用前一天的收益填充）
returns_filled = returns.fillna(method='ffill')

# 步骤2: 设定年化无风险收益率，例如5%，并转换为日度
risk_free_rate = 0.05
daily_risk_free_rate = risk_free_rate / 250  # 假设一年有250个交易日

# 步骤3: 计算超额收益
excess_returns = returns_filled - daily_risk_free_rate

# 步骤4: 计算夏普比率
# 公式: (超额收益的均值 / 超额收益的标准差) * sqrt(交易日数量)
sharpe_ratio = (excess_returns.mean() / excess_returns.std()) * np.sqrt(250)
```
