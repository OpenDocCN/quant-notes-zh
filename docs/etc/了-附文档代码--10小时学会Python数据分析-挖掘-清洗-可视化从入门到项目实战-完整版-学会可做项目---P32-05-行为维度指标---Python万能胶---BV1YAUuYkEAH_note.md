# Python数据分析：P32：05 行为维度指标

![](img/917251fb4cd8a0e55a1a9cff9417509b_1.png)

在本节课中，我们将学习数据分析中至关重要的行为维度指标。这些指标帮助我们理解用户如何与产品（尤其是网站或应用）进行交互，是评估产品表现和用户粘性的核心。

![](img/917251fb4cd8a0e55a1a9cff9417509b_2.png)

上一节我们介绍了数据分析的基础概念，本节中我们来看看如何量化用户行为。

![](img/917251fb4cd8a0e55a1a9cff9417509b_4.png)

## 页面浏览量（PV）与点击量

![](img/917251fb4cd8a0e55a1a9cff9417509b_5.png)

![](img/917251fb4cd8a0e55a1a9cff9417509b_6.png)

页面浏览量（Page View， PV）通常也被称为点击量。其核心计算逻辑是：每当用户的前端页面（如浏览器）向网站后端服务器发送一次请求，就记录为一次PV。

例如，当你打开一个网站首页，这算一次PV。点击页面上的一个链接进入详情页，这又算一次PV。即使你只是刷新当前页面，也会向后端发送新的请求，因此同样算作一次PV。

![](img/917251fb4cd8a0e55a1a9cff9417509b_8.png)

![](img/917251fb4cd8a0e55a1a9cff9417509b_9.png)

**核心公式**：
`PV = 页面请求次数`

![](img/917251fb4cd8a0e55a1a9cff9417509b_10.png)

## 独立访客数（UV）

独立访客数（Unique Visitor， UV）用于统计一天内访问网站的唯一用户数量。它通常以用户的浏览器Cookie作为识别依据。

其核心规则是：**同一个Cookie在一天内的多次访问，仅被计为一个独立访客**。这种统计方式存在一个局限性：如果多人共用同一台电脑且不清除Cookie，系统仍会将其识别为同一个用户。

![](img/917251fb4cd8a0e55a1a9cff9417509b_12.png)

![](img/917251fb4cd8a0e55a1a9cff9417509b_13.png)

![](img/917251fb4cd8a0e55a1a9cff9417509b_14.png)

**核心概念**：
`UV = 基于Cookie识别的日唯一用户数`

![](img/917251fb4cd8a0e55a1a9cff9417509b_15.png)

## 基于IP的统计方式

另一种统计用户数的方式是基于IP地址。即，将同一个IP地址在一天内的访问计为一个用户。

![](img/917251fb4cd8a0e55a1a9cff9417509b_17.png)

![](img/917251fb4cd8a0e55a1a9cff9417509b_18.png)

然而，这种方式的问题比UV更大。例如，在办公室、学校等局域网环境下，所有用户可能共享同一个公网IP，导致大量实际用户被统计为一个人。因此，在互联网数据分析中，**UV是比IP更常用、更准确的用户数量指标**。

## 平均访问时长

![](img/917251fb4cd8a0e55a1a9cff9417509b_20.png)

平均访问时长衡量用户每次访问在网站上停留的平均时间。计算方法是用户从进入网站到离开网站的总时长，除以访问次数（通常使用PV）。

![](img/917251fb4cd8a0e55a1a9cff9417509b_21.png)

**核心公式**：
`平均访问时长 = 总访问时长 / 访问次数（PV）`

![](img/917251fb4cd8a0e55a1a9cff9417509b_23.png)

## 跳出率

![](img/917251fb4cd8a0e55a1a9cff9417509b_24.png)

跳出率用于衡量网站内容对访客的吸引力。它指的是那些**只访问了一个页面就离开网站**的访问次数，占总访问次数的比例。

例如，有100个用户访问了网站，其中30个用户只看了一个页面就关闭了网站，那么跳出率就是30%。

![](img/917251fb4cd8a0e55a1a9cff9417509b_26.png)

**核心公式**：
`跳出率 = （只访问一个页面的访问次数 / 总访问次数） * 100%`

## 转化率

转化率衡量网站促使用户完成特定目标行为的效率。“转化”是指潜在用户完成了我们所期望的行为，例如点击商品详情、加入购物车、提交表单等。

![](img/917251fb4cd8a0e55a1a9cff9417509b_28.png)

计算方式是特定行为的完成次数除以触发该行为的访问次数。

![](img/917251fb4cd8a0e55a1a9cff9417509b_30.png)

以下是转化率计算的示例：
1.  有100次访问点击了网站首页，其中50次进一步点击了商品详情页。那么，首页到详情页的转化率为 `50 / 100 = 50%`。
2.  在这50个进入详情页的用户中，有10人点击了“加入购物车”。那么，详情页到加购的转化率为 `10 / 50 = 20%`。

![](img/917251fb4cd8a0e55a1a9cff9417509b_31.png)

**核心公式**：
`转化率 = （完成期望行为的次数 / 总访问次数） * 100%`

## 复购率

复购率反映用户的忠诚度和产品的粘性。它指的是**购买两次及以上的用户数**，占**总购买用户数**的比例。

一个高复购率通常意味着用户对产品或服务高度认可。例如，电商平台（如淘宝）的复购率通常很高，因为用户会频繁购物。而一些工具型应用（如抢票软件），其复购率可能呈现周期性高峰（如节假日），这同样能说明用户对其的依赖。

**核心公式**：
`复购率 = （购买次数 ≥ 2 的用户数 / 总购买用户数） * 100%`

![](img/917251fb4cd8a0e55a1a9cff9417509b_33.png)

本节课中我们一起学习了PV、UV、平均访问时长、跳出率、转化率和复购率这六个关键的行为维度指标。理解并计算这些指标，是进行用户行为分析和产品优化决策的基础。
