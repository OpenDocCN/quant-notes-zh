# Python金融分析与量化交易实战课程：P44：第9章：因子打分选股实战

## 概述
在本节课中，我们将学习量化交易中的一个重要策略——因子打分选股。我们将了解如何将多个因子综合起来，为每只股票计算一个总分，并依据总分排名来筛选出表现最优的股票组合。

上一节我们介绍了如何评估单个因子的有效性，本节中我们来看看如何综合多个因子进行选股。

## 因子打分选股的核心思想
因子打分选股的核心思想是：**不依赖单一因子，而是综合多个因子对股票进行综合评价**。

我们可以将这个过程类比为学生考试：
*   每个因子（如市盈率、市净率）就像一门科目（如语文、数学）。
*   我们需要为每只股票在每个因子上“打分”。
*   最后，将所有因子的分数汇总，得到一个“总分”。
*   根据总分对所有股票进行排名，选择排名靠前的股票构建投资组合。

这种方法旨在通过多维度评估，选出整体“素质”最优的股票。

## 实施前的关键前提
在开始为因子打分之前，我们必须明确一个关键前提：**了解每个因子与预期收益率的相关性方向**。

以下是需要明确的两类因子：
1.  **正向因子**：因子值越大，通常预期收益越高。例如：`每股收益(EPS)`、`净资产收益率(ROE)`。
2.  **负向因子**：因子值越小，通常预期收益越高。例如：`市盈率(PE)`、`市净率(PB)`（在价值投资框架下）。

如何确定因子的方向？可以通过查阅券商研究报告获得先验知识，或者通过历史数据计算因子与收益率的**信息系数(IC)** 等指标来验证。

这个方向性判断是打分的基础，因为它决定了我们打分的原则：对于正向因子，数值越大得分应越高；对于负向因子，数值越小得分应越高。

## 因子打分选股的实施步骤
理解了核心思想和前提后，我们来看看具体的实施步骤。以下是实现因子打分选股策略的关键环节：

![](img/98ffd7ef2fab049879617b8bcf94a909_1.png)

1.  **选择因子池**
    确定用于评价股票的因子集合，例如：`市盈率(PE)`、`市净率(PB)`、`净资产收益率(ROE)`、`营收增长率`等。

2.  **确定因子方向**
    根据先验知识或历史分析，明确步骤1中每个因子属于“正向因子”还是“负向因子”。

3.  **因子数据标准化（归一化）**
    由于不同因子的量纲和范围差异巨大（例如市盈率可能是几十，而营收增长率是百分比），直接相加不合理。我们需要将每个因子的值映射到统一的评分尺度上（如0-100分）。
    常用方法是使用**分位数排名**或**Z-Score标准化**。
    *   **分位数排名**：计算该因子值在全部股票中的百分位排名。
    *   **Z-Score标准化**：`z = (x - μ) / σ`，其中`x`为原始值，`μ`为均值，`σ`为标准差。

4.  **根据方向调整分数**
    对标准化后的分数进行调整，确保所有因子都是“得分越高越好”。
    *   对于**正向因子**，标准化分数可直接作为得分。
    *   对于**负向因子**，需要用 `1 - 标准化分数`（或类似转换）使其反转，从而数值小的原始值得高分。

5.  **计算综合得分**
    将每只股票在所有因子上的调整后得分相加（或求加权平均），得到该股票的综合得分。
    `综合得分 = 因子A得分 + 因子B得分 + ... + 因子N得分`
    也可以为不同因子赋予不同的权重：
    `综合得分 = w_a * 因子A得分 + w_b * 因子B得分 + ... + w_n * 因子N得分`

6.  **排序与选股**
    根据综合得分对所有股票进行降序排序。选择排名前N（例如前10或前20）的股票作为当期投资组合。

7.  **定期调仓**
    该策略通常需要定期（例如每月或每季度）重复步骤3-6，重新计算分数、排序并更新投资组合，以反映最新的市场信息。
