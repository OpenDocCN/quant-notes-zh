在充满变数的金融市场中，交易往往是一场逆人性的挑战。交易者容易受到情绪、贪欲和愤怒等影响，从而做出非理性的决策。正是这些非理性的决策，导致账户亏损累累。为了规避情绪化的交易陷阱，我们需要借助科学的工具和方法。

我们最容易犯的错误是不止损，这等于埋下了一颗定时炸弹。第二容易犯的错误是不止盈，这会让之前的努力白费。

为了避免这些情绪化交易陷阱，我们开发了一个自动止盈止损的小工具。

## 止损板块详解

![](img/a088e755d254d49db1719bf67b31a919_11.png)

止损板块分为**硬止损**和**软止损**两种策略。

![](img/a088e755d254d49db1719bf67b31a919_13.png)

### 硬止损（固定比例止损）

![](img/a088e755d254d49db1719bf67b31a919_15.png)

硬止损，即固定比例止损。当持仓亏损达到预设的百分比时，例如亏损达到 `10%`，系统会自动执行卖出操作，以此防止损失进一步扩大。

**核心逻辑公式**：
`触发卖出条件：当前亏损比例 >= 预设止损比例（如10%）`

### 软止损（时间止损）

软止损，我们称之为时间止损。这并非指真正的亏损，而是指在规定的时间内，如果价格没有达到我们预期的涨幅，系统也会自动进行清仓操作。这对交易有时效性的热点板块很有意义。

![](img/a088e755d254d49db1719bf67b31a919_17.png)

**核心逻辑描述**：
`如果在预设时间T内，价格涨幅未达到预期目标R，则执行清仓。`

## 止盈板块详解

![](img/a088e755d254d49db1719bf67b31a919_19.png)

接下来我们看看止盈板块的功能。止盈策略旨在帮助我们在盈利时锁定利润，避免因贪婪而回吐收益。

![](img/a088e755d254d49db1719bf67b31a919_21.png)

以下是止盈板块包含的几种策略：

![](img/a088e755d254d49db1719bf67b31a919_23.png)

*   **小涨部分止盈**：这是一种仓位管理策略。当价格小幅上涨时，我们卖出部分仓位来覆盖交易佣金和手续费等成本，确保这部分交易不亏钱。
*   **大涨部分止盈/清仓**：当价格大幅上涨时，我们可以选择继续减仓或者直接清仓，以此来锁定可观的利润。
*   **冲高回落止盈**：这是一个非常实用的策略。当价格快速上涨到某一阈值（例如开盘涨幅达 `9.5%`），但后续盘中价格从高点回落一定幅度（例如 `3%`）时，工具会自动捕捉到这一“冲高回落”趋势，帮助我们在相对高位卖出，避免因贪婪而错过最佳卖出时机。

![](img/a088e755d254d49db1719bf67b31a919_25.png)

**冲高回落止盈逻辑公式**：
`触发卖出条件：最高价涨幅 >= 阈值H 且 当前价从最高价回落幅度 >= 阈值R`

![](img/a088e755d254d49db1719bf67b31a919_27.png)

## 利润保护机制：移动止盈线

值得一提的是，每次执行止盈操作后，系统都会自动计算并更新一条“保护线”，即**移动止盈线**。这条线的目的是为了保证本次交易的最终结果为正收益，防止盈利单变亏损单。

![](img/a088e755d254d49db1719bf67b31a919_29.png)
