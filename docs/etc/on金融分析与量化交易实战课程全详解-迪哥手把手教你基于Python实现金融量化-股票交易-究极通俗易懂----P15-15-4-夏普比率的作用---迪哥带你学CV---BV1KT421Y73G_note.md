# Python金融分析与量化交易实战课程：P15：15.4-夏普比率的作用 📈

在本节课中，我们将要学习夏普比率这一核心金融指标。夏普比率用于衡量投资组合在承担单位风险时所获得的超额回报，是评估投资绩效的重要工具。

## 夏普比率的定义与意义

上一节我们介绍了投资回报率，本节中我们来看看如何结合风险来评估收益质量。夏普比率描述的是，对于单位风险而言，我们能获得多少收益。指标越高，意味着在承担相同风险的情况下，获得的超额回报越高。

我们可以用一个生活中的例子来理解：假设有两份工作，一份是稳定的办公室工作，日薪500元；另一份是高风险地区的雇佣兵工作，日薪5万元。虽然雇佣兵的日薪高，但其风险也极高。夏普比率就像是在计算“每单位风险的日薪”，帮助我们判断高风险是否带来了足够高的回报补偿。

![](img/aad0f5a11207de6666e46183a1a1b1b5_1.png)

在选股或构建投资组合时，我们通常会比较多个选项的夏普比率。在其他条件相似的情况下，夏普比率更高的投资更优，因为它意味着“性价比”更高——用更少的风险换取了更多的收益。

![](img/aad0f5a11207de6666e46183a1a1b1b5_3.png)

## 夏普比率的计算方法

![](img/aad0f5a11207de6666e46183a1a1b1b5_5.png)

理解了夏普比率的意义后，我们来看看如何计算它。其核心思想是比较投资组合的收益与无风险收益的差异，再除以投资组合的风险（波动性）。

计算公式如下：
`夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差`

**公式解析：**
*   **分子 (`投资组合收益率 - 无风险收益率`)**: 代表投资获得的**超额回报**。无风险收益率通常可以用国债收益率等近似代替。
*   **分母 (`投资组合收益率的标准差`)**: 代表投资组合的**风险**（波动性）。标准差越大，说明收益波动越大，风险越高。

因此，夏普比率衡量的是**每承担一单位的总风险，能获得多少超额回报**。

## 基于Python计算夏普比率

接下来，我们将在Python环境中实际计算一组股票的夏普比率。假设我们已经有了股票的历史日收益率数据。

以下是计算夏普比率的关键步骤：

1.  **准备数据**：确保收益率数据是完整的，对缺失值进行填充（例如，用前一天的收益率填充）。
2.  **设定参数**：确定无风险收益率（例如年化5%）和年化交易日数（通常为250天）。
3.  **应用公式**：按照上述公式进行计算，并对结果进行年化处理。

```python
import numpy as np
import pandas as pd

![](img/aad0f5a11207de6666e46183a1a1b1b5_7.png)

# 假设 `returns` 是一个Pandas DataFrame，每一列代表一只股票的日收益率序列
# 例如：returns = pd.DataFrame({‘股票A’: [0.01, -0.02, ...], ‘股票B’: [...]})

# 步骤1: 处理缺失值（前向填充）
returns_filled = returns.fillna(method=‘ffill’)

# 步骤2: 设定参数
risk_free_rate = 0.05  # 年化无风险收益率，例如5%
trading_days_per_year = 250  # 年化交易日数

# 步骤3: 计算年化夏普比率
# 计算超额收益的均值（年化）
excess_return = returns_filled - risk_free_rate / trading_days_per_year
avg_excess_return = excess_return.mean() * trading_days_per_year

# 计算收益率的标准差（年化）
std_return = returns_filled.std() * np.sqrt(trading_days_per_year)

![](img/aad0f5a11207de6666e46183a1a1b1b5_9.png)

# 计算夏普比率
sharpe_ratio = avg_excess_return / std_return

![](img/aad0f5a11207de6666e46183a1a1b1b5_11.png)

print(“各股票的夏普比率:”)
print(sharpe_ratio)
```

运行以上代码后，我们会得到每只股票的夏普比率。结果解读非常简单：**数值越大越好**。正数表示获得了正向的超额回报，负数则表示表现还不如无风险投资。例如，如果“中兴通讯”的夏普比率最高，那么在承担相同风险的前提下，投资它获得的超额回报是最高的。
