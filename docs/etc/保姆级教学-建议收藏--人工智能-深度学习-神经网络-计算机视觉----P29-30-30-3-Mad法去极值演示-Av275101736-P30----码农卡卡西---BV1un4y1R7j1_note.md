在本节中，我们将学习第二种去极值方法：MAD法。MAD法在公式上看似稍复杂，但计算过程其实很简单。

## 概述：MAD法原理

MAD法要求先计算一个名为MAD的值。具体步骤如下：

![](img/70f437291d8551f8037c44c9831ab68b_4.png)

首先，给定一个数据序列，例如 `[1, 2, 3, 4, ...]`。我们计算这个序列的中位数，假设中位数是4。

接着，对于序列中的每一个数，计算其与中位数的**绝对差值**。例如，第一个数1与中位数4的绝对差值是 `|1-4| = 3`。对所有数据执行此操作，会得到一个新的序列，这个序列由所有绝对差值构成。

然后，对这个由绝对差值构成的新序列，再次求取它的中位数。这个最终得到的中位数，就是所谓的MAD值。

![](img/70f437291d8551f8037c44c9831ab68b_4.png)

![](img/70f437291d8551f8037c44c9831ab68b_6.png)

## 确定数据边界

![](img/70f437291d8551f8037c44c9831ab68b_8.png)

获得MAD值后，就可以确定数据的合理范围了。我们通过以下公式设定上下边界：

![](img/70f437291d8551f8037c44c9831ab68b_10.png)

![](img/70f437291d8551f8037c44c9831ab68b_12.png)

*   **上界** = 中位数 + N × MAD
*   **下界** = 中位数 - N × MAD

这里的N值通常有一个固定取值：**1.4826**。这个值是为了使MAD估计与正态分布的标准差估计保持一致。

![](img/70f437291d8551f8037c44c9831ab68b_14.png)

设定好上下界后，剩余的操作就与上一节介绍的百分位法类似了：将超出上界的数据替换为上界值，将低于下界的数据替换为下界值。

![](img/70f437291d8551f8037c44c9831ab68b_16.png)

![](img/70f437291d8551f8037c44c9831ab68b_18.png)

MAD法本质上是提供了另一种确定数据边界的方法，以实现去极值。

## 代码实现

![](img/70f437291d8551f8037c44c9831ab68b_26.png)

理解了原理后，代码实现就很简单了。我们将按照上述公式逐步计算。

![](img/70f437291d8551f8037c44c9831ab68b_28.png)

以下是一个名为 `filter_mad` 的函数实现：

```python
def filter_mad(series, n):
    # 1. 计算原始序列的中位数
    median = series.quantile(0.5)

    # 2. 计算每个数据与中位数的绝对差值，然后求该差值序列的中位数，即MAD值
    mad = (series - median).abs().quantile(0.5)

    # 3. 计算上下边界
    high = median + n * mad
    low = median - n * mad

    # 4. 根据边界对数据进行裁剪
    return series.clip(lower=low, upper=high)
```
