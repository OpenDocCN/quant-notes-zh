![](img/9f23b8dc873976a3cf37b461b70c3b7e_1.png)

在本节课中，我们将学习如何安装 MySQL 8.0 数据库。MySQL 8.0 版本引入了对数据分析工作至关重要的“窗口函数”功能，掌握其安装是后续学习的基础。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_3.png)

## 课程概述

![](img/9f23b8dc873976a3cf37b461b70c3b7e_5.png)

本节教程将详细演示 MySQL 8.0 的安装步骤。我们选择安装 8.0 版本，而非之前学习的 5.7 版本，主要原因在于 8.0 版本新增了“窗口函数”模块。窗口函数在数据分析工作中使用频率很高，能简化复杂的联表查询操作。同时，掌握窗口函数也是数据分析岗位面试中的常见考点。因此，为了后续课程内容（窗口函数讲解）的顺利进行，我们需要先完成 MySQL 8.0 的安装。

## 安装步骤详解

![](img/9f23b8dc873976a3cf37b461b70c3b7e_7.png)

接下来，我们将一步步完成 MySQL 8.0 的安装。请确保您已下载好 MySQL 8.0 的 Windows MSI 安装包。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_9.png)

![](img/9f23b8dc873976a3cf37b461b70c3b7e_11.png)

### 启动安装程序

![](img/9f23b8dc873976a3cf37b461b70c3b7e_13.png)

首先，双击下载好的 `.msi` 格式安装文件以启动安装向导。

### 同意许可协议

![](img/9f23b8dc873976a3cf37b461b70c3b7e_15.png)

![](img/9f23b8dc873976a3cf37b461b70c3b7e_17.png)

安装程序启动后，您将看到初始界面。第一步操作是同意软件许可协议。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_19.png)

以下是需要执行的操作：
*   勾选 **“I accept the license terms”**（我接受许可条款）。
*   点击 **“Next”**（下一步）按钮。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_21.png)

![](img/9f23b8dc873976a3cf37b461b70c3b7e_23.png)

### 选择安装类型

进入此步骤后，请勿急于点击下一步。由于 MSI 安装包包含许多组件，我们只需要安装 MySQL 服务器本身。

以下是需要执行的操作：
*   选择 **“Server only”**（仅安装服务器）选项。
*   点击 **“Next”**（下一步）按钮。

### 检查安装就绪状态

此时，界面会显示即将安装的产品信息（例如 MySQL Server 8.0.xx）和默认的安装路径。

以下是需要执行的操作：
*   确认信息无误后，点击 **“Execute”**（执行）按钮开始安装。
*   等待安装进度条完成。
*   安装完成后，点击 **“Next”**（下一步）按钮。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_25.png)

![](img/9f23b8dc873976a3cf37b461b70c3b7e_27.png)

### 产品配置

![](img/9f23b8dc873976a3cf37b461b70c3b7e_29.png)

![](img/9f23b8dc873976a3cf37b461b70c3b7e_31.png)

安装完成后，进入产品配置阶段。在此界面，我们需要进行一些关键设置。

以下是需要执行的操作：
*   保持默认选择，点击 **“Next”**（下一步）。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_33.png)

### 配置网络与端口

![](img/9f23b8dc873976a3cf37b461b70c3b7e_35.png)

在此界面，我们需要设置 MySQL 服务的端口号。如果您的电脑上已经安装了 MySQL 5.7（默认端口3306），为了避免冲突，必须为 8.0 版本修改端口。

以下是需要执行的操作：
*   找到 **“Port Number”**（端口号）设置项。
*   将端口号从默认的 `3306` 修改为一个未占用的端口，例如 `3308`。
*   点击 **“Next”**（下一步）。

### 选择身份验证方法

此步骤涉及设置密码加密方式。为了后续连接时更方便，我们选择传统的加密方式。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_37.png)

以下是需要执行的操作：
*   选择 **“Use Legacy Authentication Method”**（使用传统身份验证方法）选项。
*   点击 **“Next”**（下一步）。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_39.png)

### 设置 root 账户密码

这是关键步骤，需要为 MySQL 的 root（超级管理员）账户设置密码。请务必牢记此密码。

以下是需要执行的操作：
*   在 **“MySQL Root Password”** 和 **“Repeat Password”** 输入框中，输入相同的密码（例如 `123456`）。
*   （可选）可以跳过添加额外用户的步骤。
*   点击 **“Next”**（下一步）。

### 配置 Windows 服务

在此步骤，我们需要指定 MySQL 8.0 在 Windows 中显示的服务名称。如果与已存在的 MySQL 服务名冲突，可以修改。

以下是需要执行的操作：
*   查看 **“Windows Service Name”**。
*   如果系统提示冲突（通常会有感叹号），可将名称修改为 `MySQL80` 等以示区分。
*   确认无误后，点击 **“Next”**（下一步）。

### 应用配置并完成安装

进入最后一步，安装程序将应用所有配置。

以下是需要执行的操作：
*   确认配置摘要，特别是端口号（应为之前修改的，如 `3308`）。
*   点击 **“Execute”**（执行）使配置生效。
*   等待配置完成后，点击 **“Finish”**（完成）按钮。至此，MySQL 8.0 安装完毕。

## 测试连接

![](img/9f23b8dc873976a3cf37b461b70c3b7e_41.png)

安装完成后，我们需要验证 MySQL 8.0 服务是否正常运行。可以使用数据库管理工具（如 Navicat）进行连接测试。

![](img/9f23b8dc873976a3cf37b461b70c3b7e_43.png)

1.  打开 Navicat。
2.  点击“连接”，选择“MySQL”。
3.  在连接设置中：
    *   **连接名**：自定义，例如 `MySQL8.0`。
    *   **主机**：`localhost` 或 `127.0.0.1`。
    *   **端口**：填写安装时设置的端口（例如 `3308`）。
    *   **用户名**：`root`。
    *   **密码**：输入安装时设置的 root 账户密码。
4.  点击“测试连接”。如果显示“连接成功”，则表明 MySQL 8.0 已安装并配置成功。
