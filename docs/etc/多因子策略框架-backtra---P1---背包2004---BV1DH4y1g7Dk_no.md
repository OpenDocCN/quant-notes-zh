# 量化金融与机器学习：P1：多因子策略框架实现 🧮

在本节课中，我们将学习如何将一个单因子策略扩展为多因子策略。我们将使用市盈率和总市值两个因子，通过整合它们的排序信息来构建一个更稳健的选股策略。

## 概述

上一节我们介绍了如何使用单一因子（如市盈率）进行选股。本节中，我们来看看当面对多个因子时，如何将它们结合起来，形成一个综合的选股逻辑。核心思想是分别对每个因子进行排序，然后将排序序号相加，得到一个综合排名，最后根据这个综合排名选择股票。

## 多因子整合思路

当只有一个因子时，我们直接对该因子排序并选取头部股票。现在有两个因子，我们不能简单地只取某一个因子的头部股票。

我们的处理思路如下：
1.  分别对每个因子进行排序，得到每个股票在每个因子下的排序序号。
2.  将每个股票的多个因子排序序号相加，得到一个“总序号”。
3.  根据这个“总序号”进行排序，数值越小（即综合排名越靠前）的股票越优。
4.  选取总序号排名靠前的股票作为投资组合。

以下是具体的实现步骤。

## 代码实现与解析

首先，我们引入必要的模块和风控规则（如避免涨停股），这些与单因子策略一致。假设我们已经获取了交易当日所有有效股票的数据 `df`。

```python
# 假设 df 是包含股票代码、市盈率(pe_ratio)、总市值(total_market_cap)等字段的DataFrame
# 风控筛选步骤已包含在df的获取过程中
```

接下来，我们需要对两个因子分别排序。我们使用Pandas的 `.rank()` 方法。

```python
# 对市盈率进行排序：我们希望选取市盈率高的股票，因此按降序排序（ascending=False）
pe_rank = df[‘pe_ratio’].rank(ascending=False)

![](img/cc9a6f92ac77e17d0ce11bfc82ea649d_1.png)

# 对总市值进行排序：我们希望选取市值小的股票，因此按升序排序（ascending=True）
cap_rank = df[‘total_market_cap’].rank(ascending=True)
```

![](img/cc9a6f92ac77e17d0ce11bfc82ea649d_3.png)

![](img/cc9a6f92ac77e17d0ce11bfc82ea649d_5.png)

`.rank()` 方法为序列中的每个值生成一个排序序号。对于市盈率降序排序，值最大的股票序号为1；对于总市值升序排序，值最小的股票序号为1。

现在，我们将两个因子的排序序号相加，得到综合排名。

```python
# 将综合排名作为新列添加到原DataFrame中
df.loc[:, ‘composite_rank’] = pe_rank + cap_rank
```

**注意**：使用 `df.loc[:, ‘composite_rank’]` 的赋值方式可以避免Pandas产生警告。

最后，我们根据综合排名进行排序，并选取排名最靠前的N只股票。

```python
# 根据综合排名升序排序
df_sorted = df.sort_values(by=‘composite_rank’)

![](img/cc9a6f92ac77e17d0ce11bfc82ea649d_7.png)

# 选取排名前5的股票代码
top_n = 5
stock_list = df_sorted.head(top_n)[‘stock_code’].tolist()
```

![](img/cc9a6f92ac77e17d0ce11bfc82ea649d_9.png)

![](img/cc9a6f92ac77e17d0ce11bfc82ea649d_11.png)
