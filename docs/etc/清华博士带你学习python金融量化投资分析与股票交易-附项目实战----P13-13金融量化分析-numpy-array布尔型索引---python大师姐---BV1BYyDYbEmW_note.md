## 布尔型索引的原理

![](img/8c02069804922d796f6870e831e85f66_24.png)

![](img/8c02069804922d796f6870e831e85f66_26.png)

布尔型索引的原理可以分为两步：
1.  数组与标量进行运算，得到一个布尔数组。
2.  将这个布尔数组传入原数组的索引中，从而筛选出对应位置为`True`的元素。

![](img/8c02069804922d796f6870e831e85f66_28.png)

以下是其核心概念：
```python
# 假设有一个数组 a
a = np.array([8, 3, 6, 2, 9])
# 第一步：数组与标量运算，得到布尔数组
bool_array = a > 5  # 结果为 [True, False, True, False, True]
# 第二步：使用布尔数组进行索引
result = a[bool_array]  # 结果为 [8, 6, 9]
# 更简洁的一步写法
result = a[a > 5]
```

## 与Python传统方法的对比

![](img/8c02069804922d796f6870e831e85f66_37.png)

![](img/8c02069804922d796f6870e831e85f66_39.png)

以下是使用Python内置`filter`函数和NumPy布尔型索引的对比。

![](img/8c02069804922d796f6870e831e85f66_41.png)

![](img/8c02069804922d796f6870e831e85f66_43.png)

**传统Python方法（使用filter和lambda）：**
```python
import numpy as np
a_list = [8, 3, 6, 2, 9]
filtered_list = list(filter(lambda x: x > 5, a_list))
```

![](img/8c02069804922d796f6870e831e85f66_45.png)

![](img/8c02069804922d796f6870e831e85f66_47.png)

**NumPy布尔型索引方法：**
```python
import numpy as np
a_array = np.array([8, 3, 6, 2, 9])
filtered_array = a_array[a_array > 5]
```
可以看到，NumPy的方法更加简洁直观。

## 组合条件筛选

![](img/8c02069804922d796f6870e831e85f66_53.png)

![](img/8c02069804922d796f6870e831e85f66_55.png)

![](img/8c02069804922d796f6870e831e85f66_56.png)

有时我们需要使用多个条件进行筛选，例如“选出数组中所有大于5的偶数”。这涉及到组合布尔条件。

![](img/8c02069804922d796f6870e831e85f66_60.png)

![](img/8c02069804922d796f6870e831e85f66_62.png)

**方法一：分步筛选**
```python
import numpy as np
a = np.array([8, 3, 6, 2, 9, 4])
# 第一步：筛选大于5的数
b = a[a > 5]  # b = [8, 6, 9]
# 第二步：从b中筛选偶数
result = b[b % 2 == 0]  # result = [8, 6]
```

![](img/8c02069804922d796f6870e831e85f66_64.png)

**方法二：使用位运算符组合条件（一步完成）**
需要注意的是，不能直接使用Python的关键字`and`和`or`，而应使用位运算符`&`（与）和`|`（或），并且每个条件需要用括号括起来。
```python
import numpy as np
a = np.array([8, 3, 6, 2, 9, 4])
# 筛选大于5的偶数
result = a[(a > 5) & (a % 2 == 0)]  # 使用 & 表示“与”
# 筛选大于5的数或所有的偶数（使用 | 表示“或”）
result_or = a[(a > 5) | (a % 2 == 0)]
```

## 注意事项

![](img/8c02069804922d796f6870e831e85f66_66.png)

在使用组合条件进行布尔型索引时，有两点需要特别注意：
1.  必须使用位运算符`&`和`|`，而不是逻辑关键字`and`和`or`。
2.  每个独立的布尔条件必须用括号`()`括起来，以确保运算优先级正确。
