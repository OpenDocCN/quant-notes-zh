## 什么是最大回撤？

上一节我们介绍了策略分析的基本概念，本节中我们来看看最大回撤。最大回撤描述的是在投资过程中，资产价值从历史最高点到之后最低点的最大百分比跌幅。它衡量的是投资过程中可能经历的“最惨”阶段。

**核心公式**：
`最大回撤 = max[(P_i - P_j) / P_i]`，其中 `i < j`，`P_i` 代表第 `i` 天的价格（峰值），`P_j` 代表第 `j` 天的价格（谷底）。

这个指标不能仅从字面理解。它并非评估策略绝对好坏，而是评估策略的风险。即使一个策略最终盈利，其过程也可能存在一段非常痛苦的下跌时期，最大回撤就是量化这段“惨劲儿”有多惨。

## 最大回撤的计算逻辑

理解了最大回撤的含义后，我们来看看如何通过计算找到这个区间。核心思路是：遍历所有可能的价格点对（`i`, `j`，且 `i < j`），找出 `(P_i - P_j) / P_i` 这个比值最大的情况。

以下是计算最大回撤的关键步骤：

1.  **寻找历史峰值序列**：对于数据集中的每一天，计算到该天为止的历史最高价格。这可以通过 `cummax()` 函数实现。
2.  **计算每日潜在回撤**：用当天的历史最高价格减去当天的实际价格，得到每日的“下跌深度”。
3.  **计算每日回撤率**：将每日的“下跌深度”除以对应的历史最高价格，得到每日的回撤百分比。
4.  **找出最大值**：在所有每日回撤百分比中，找出最大值，即为整个时间段的最大回撤。

## 使用Python代码实现计算

理论清晰后，我们通过Python代码来具体实现上述计算过程。我们将使用Pandas库进行数据处理。

![](img/5efe584476ca392583815eadd71fa0bd_8.png)

首先，我们导入必要的库并准备示例数据。

![](img/5efe584476ca392583815eadd71fa0bd_10.png)

![](img/5efe584476ca392583815eadd71fa0bd_12.png)

```python
import pandas as pd
import numpy as np

![](img/5efe584476ca392583815eadd71fa0bd_14.png)

# 假设df是一个包含价格数据的DataFrame，其中'close'列为每日收盘价
# 这里我们创建一些示例数据
dates = pd.date_range('2023-06-01', periods=10, freq='D')
prices = [100, 102, 98, 95, 93, 96, 99, 101, 97, 94]
df = pd.DataFrame({'close': prices}, index=dates)
print("原始价格数据:")
print(df)
```

![](img/5efe584476ca392583815eadd71fa0bd_16.png)

接下来，我们按照计算逻辑逐步实现。

![](img/5efe584476ca392583815eadd71fa0bd_17.png)

![](img/5efe584476ca392583815eadd71fa0bd_19.png)

```python
# 1. 计算历史最高价序列（累积最大值）
df['cumulative_max'] = df['close'].cummax()
print("\n累积最大值序列:")
print(df[['close', 'cumulative_max']])

![](img/5efe584476ca392583815eadd71fa0bd_20.png)

# 2. 计算每日的下跌深度（历史最高 - 当前价）
df['drawdown_depth'] = df['cumulative_max'] - df['close']
print("\n每日下跌深度:")
print(df['drawdown_depth'])

![](img/5efe584476ca392583815eadd71fa0bd_22.png)

![](img/5efe584476ca392583815eadd71fa0bd_23.png)

# 3. 计算每日的回撤率（下跌深度 / 历史最高价）
df['drawdown_rate'] = df['drawdown_depth'] / df['cumulative_max']
print("\n每日回撤率:")
print(df['drawdown_rate'])

![](img/5efe584476ca392583815eadd71fa0bd_25.png)

![](img/5efe584476ca392583815eadd71fa0bd_26.png)

# 4. 找出最大回撤
max_drawdown = df['drawdown_rate'].max()
print(f"\n最大回撤为: {max_drawdown:.2%}")

# 5. （可选）找出最大回撤发生的结束日期（谷底日期）
max_dd_end_date = df['drawdown_rate'].idxmax()
print(f"最大回撤的谷底日期是: {max_dd_end_date.date()}")
```

![](img/5efe584476ca392583815eadd71fa0bd_28.png)

![](img/5efe584476ca392583815eadd71fa0bd_29.png)

运行以上代码，你将看到每一步的中间结果，并最终得到最大回撤的值及其发生的日期。

## 最大回撤的实际意义

![](img/5efe584476ca392583815eadd71fa0bd_33.png)

计算完成后，我们需要理解这个数字在实战中的意义。最大回撤是一个关键的风险控制指标。

![](img/5efe584476ca392583815eadd71fa0bd_34.png)

以下是最大回撤在投资决策中的主要作用：

*   **评估策略风险承受度**：一个最大回撤为50%的策略，意味着你可能需要忍受资产腰斩的考验。投资者需要根据自身的风险承受能力来选择策略。
*   **优化资产配置**：通过比较不同策略或资产的最大回撤，可以更好地进行组合配置，降低整体投资组合的波动风险。
*   **策略改进的依据**：分析最大回撤发生的时段和市场环境，可以帮助我们反思策略逻辑，寻找改进空间，例如增加止损机制。
