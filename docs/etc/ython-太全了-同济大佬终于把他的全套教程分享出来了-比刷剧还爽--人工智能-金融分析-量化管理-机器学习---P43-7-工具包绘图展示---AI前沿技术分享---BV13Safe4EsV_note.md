# 量化投资入门：P43：7-工具包绘图展示 📊

在本节课中，我们将学习如何使用绘图工具包对计算出的因子IC值进行可视化分析。通过图表，我们可以更直观地观察因子与收益率的相关性随时间变化的趋势和稳定性。

上一节我们介绍了如何计算因子的IC值（信息系数），即因子值与下一期收益率之间的相关系数。本节中我们来看看如何将这些数值结果转化为直观的图表。

## 绘制IC值时间序列图

![](img/6b4c58806944d40470552def0fa332b3_1.png)

计算出的IC值每天都会变化。为了更清晰地观察其走势，我们需要将其绘制成图。这里我们引入一个名为 `alphalens` 的工具包，它专门用于因子分析的可视化。

以下是使用 `alphalens` 绘制IC值时间序列图的步骤：

![](img/6b4c58806944d40470552def0fa332b3_3.png)

1.  **导入绘图函数**：从 `alphalens` 中导入绘制时间序列图的函数 `plot_ic_ts`。
2.  **执行绘图**：将之前计算好的因子数据（通常是一个 `pandas.Series` 或 `DataFrame`）传入该函数。
3.  **分析图表**：观察生成的图表，理解其中各条曲线的含义。

对应的核心代码如下：

```python
from alphalens import plotting
# 假设 factor_data 是之前计算好的因子IC值数据
plotting.plot_ic_ts(factor_data)
```

执行代码后，会生成一张图表。图表中通常包含两条线：
*   **蓝色曲线**：代表每日的实际IC值，波动通常较大。
*   **绿色曲线**：代表IC值的滚动平均值（默认以一个月为窗口），它平滑了每日的波动，更能反映趋势。

## 解读图表与关键指标

通过生成的图表，我们可以进行以下分析：

*   **观察绿色趋势线**：由于蓝色线波动剧烈，我们主要关注绿色移动平均线。我们希望因子与收益率的正相关性越强越好，即IC值越大越好。同时，我们也希望看到IC值能保持一个相对稳定或向上的趋势。
*   **理解信息比率（IR）**：图表中通常会标注一个名为 **信息比率（Information Ratio）** 的指标。其计算公式为：
    `IR = IC均值 / IC标准差`
    这个比率描述了IC值的稳定性。**IR值越大，说明因子的预测能力越稳定**；反之，则说明因子的有效性波动较大。

## 探索其他分析图表

![](img/6b4c58806944d40470552def0fa332b3_5.png)

`alphalens` 工具包功能强大，除了时间序列图，还提供了多种分析图表，用于全方位评估因子质量。

![](img/6b4c58806944d40470552def0fa332b3_7.png)

以下是其他几种有用的可视化图表：

![](img/6b4c58806944d40470552def0fa332b3_9.png)

*   **IC值直方图与QQ图**：用于检验IC值的分布是否接近正态分布。
*   **分位数组合收益热力图**：展示按因子值分组的股票组合在不同时期的平均收益，用颜色深浅表示收益高低。
*   **因子自相关图**：分析因子值自身在不同时间间隔上的相关性，用于判断因子的衰减速度。

![](img/6b4c58806944d40470552def0fa332b3_11.png)

这些图表在后续深入分析因子和构建策略时非常有用，本节课暂不逐一演示，大家可以在实践中根据需要调用对应的API函数进行探索。
