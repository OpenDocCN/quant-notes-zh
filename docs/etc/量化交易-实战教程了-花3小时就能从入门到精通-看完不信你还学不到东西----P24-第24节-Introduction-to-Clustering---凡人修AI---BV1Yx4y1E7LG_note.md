## 概述：无监督学习与聚类

上一节我们介绍了监督学习的多种模型。本节中，我们来看看机器学习中另一个重要分支：无监督学习。与监督学习不同，无监督学习的数据样本没有预先定义的标签或响应变量Y。它的目标是发现数据内部的结构或模式，其中最核心的技术之一就是聚类分析。

### 什么是无监督学习？

在监督学习中，我们拥有特征X和对应的响应Y（无论是分类标签还是连续数值）。模型的学习过程受到已知Y的“监督”。而在无监督学习中，我们只有特征X。例如，在天文学中分析星团，我们只有星星在三维空间中的坐标（X），目标是发现哪些星星可能属于同一个星团（潜在的类别Y），但这个Y在训练前是未知的。聚类就是将具有相似特征的样本自动分组，使得组内样本相似度高，组间样本相似度低。

无监督学习的一个主要挑战是缺乏评估模型好坏的客观标准。因为没有真实的Y，我们无法像监督学习那样计算均方误差或使用交叉验证来比较模型。模型效果的判断往往依赖于领域知识或直观的可视化结果。

### K均值聚类算法

![](img/fdecbca367067ad28fc7706513273640_3.png)

K均值聚类是无监督学习中最基础且应用最广泛的算法之一。其核心思想非常简单：将数据划分为K个簇，每个簇由其所有样本点的均值（质心）来代表。

以下是K均值聚类的具体算法步骤：

1.  **初始化**：在数据空间中随机选择K个点作为初始质心。
2.  **分配步骤**：对于数据集中的每一个样本点，计算其到K个质心的距离，并将其分配给距离最近的质心所属的簇。
3.  **更新步骤**：对于每一个新形成的簇，重新计算该簇所有样本点的均值，并将该均值设为新的质心。
4.  **迭代**：重复步骤2和步骤3，直到质心的位置不再发生变化（或变化小于某个阈值），此时算法收敛。

算法公式化描述如下：
*   给定样本集 \( D = \{x_1, x_2, ..., x_m\} \)。
*   目标是最小化**畸变函数** \( J \)：
    \[ J = \sum_{i=1}^{m} \sum_{k=1}^{K} w_{ik} \| x_i - \mu_k \|^2 \]
    其中，\( w_{ik} = 1 \) 当样本 \( x_i \) 属于簇 \( k \)，否则为0。\( \mu_k \) 是簇 \( k \) 的质心。

K均值算法收敛速度快，但它存在一个关键问题：由于畸变函数J是非凸的，算法只能保证收敛到局部最优解，而非全局最优解。最终结果严重依赖于初始质心的选择。

为了解决这个问题，一个常见的实践方法是：
**多次运行K均值算法**，每次使用不同的随机初始质心，然后选择畸变函数J值最小的那次结果作为最终输出。

![](img/fdecbca367067ad28fc7706513273640_5.png)

### Python中的K均值聚类示例

让我们通过一个Python例子来直观理解K均值。我们模拟三个二维高斯分布（正态分布）的数据点。

```python
# 导入必要的库
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs

![](img/fdecbca367067ad28fc7706513273640_7.png)

# 生成模拟数据：三个簇
X, y_true = make_blobs(n_samples=300, centers=3, cluster_std=[1.0, 0.5, 0.3], random_state=0)

# 应用K均值聚类，假设K=3
kmeans = KMeans(n_clusters=3, random_state=0)
y_kmeans = kmeans.fit_predict(X)

![](img/fdecbca367067ad28fc7706513273640_9.png)

# 可视化结果
plt.scatter(X[:, 0], X[:, 1], c=y_kmeans, s=50, cmap='viridis')
centers = kmeans.cluster_centers_
plt.scatter(centers[:, 0], centers[:, 1], c='red', s=200, alpha=0.75, marker='X')
plt.title('K-Means Clustering (K=3)')
plt.show()
```

在这个例子中，我们“幸运地”知道了真实簇数（K=3）。然而在现实中，我们通常不知道最佳的K值。我们可以尝试不同的K值（例如K=4），并通过可视化结果结合业务直觉来判断哪个K值更合理。例如，K=4可能会将一个本应完整的簇不合理地分割开。

## 聚类在金融中的应用：OHLC模式分类

![](img/fdecbca367067ad28fc7706513273640_11.png)

接下来，我们探讨聚类在量化交易中的一个应用：基于股票日线OHLC（开盘价、最高价、最低价、收盘价）数据进行模式分类。

![](img/fdecbca367067ad28fc7706513273640_13.png)

基本思路如下：
1.  **特征构建**：将每日的OHLC数据转换为三个比率特征，以消除绝对价格的影响：
    *   `high / open`
    *   `low / open`
    *   `close / open`
2.  **聚类分析**：使用K均值等算法对这些三维特征进行聚类。理想情况下，数据会被聚成几类，分别代表不同的市场状态，例如“大涨”、“大跌”、“横盘”等。
3.  **交易策略**：模型训练好后，当新的一天的数据到来时，计算其特征并归类。如果它被归到“大涨”类，我们可以在当天收盘前建立多头仓位，预期次日上涨获利。

![](img/fdecbca367067ad28fc7706513273640_15.png)

## 神经网络简介

![](img/fdecbca367067ad28fc7706513273640_41.png)

神经网络是当前深度学习领域的基础。它通过模拟人脑神经元的工作方式，构建多层网络结构来处理复杂模式。本节简要介绍两种基本类型。

### 人工神经网络/前馈神经网络

![](img/fdecbca367067ad28fc7706513273640_43.png)

![](img/fdecbca367067ad28fc7706513273640_45.png)

![](img/fdecbca367067ad28fc7706513273640_47.png)

这是最简单、最直观的神经网络。信息从输入层单向流动，经过一个或多个隐藏层，最终到达输出层。每一层由多个神经元（节点）组成，神经元之间的连接具有权重，并通过激活函数引入非线性。

![](img/fdecbca367067ad28fc7706513273640_49.png)

![](img/fdecbca367067ad28fc7706513273640_51.png)

一个简单的神经元输出公式为：
\[ a = g(\sum_{i=1}^{n} w_i x_i + b) \]
其中，\( x_i \) 是输入，\( w_i \) 是权重，\( b \) 是偏置，\( g \) 是激活函数（如Sigmoid, ReLU）。

![](img/fdecbca367067ad28fc7706513273640_53.png)

![](img/fdecbca367067ad28fc7706513273640_55.png)

![](img/fdecbca367067ad28fc7706513273640_57.png)

### 循环神经网络

RNN专门用于处理序列数据（如时间序列、文本）。它的特点是神经元之间不仅存在层间连接，还存在同层或跨时间步的连接，形成一个“循环”，使得网络具有记忆之前信息的能力。这使得RNN在预测时间序列下一个值或理解自然语言上下文方面非常有效。

此外，还有**卷积神经网络**（CNN），它在处理图像、语音等网格状数据方面表现出色，通过卷积核高效提取局部特征。CNN是计算机视觉领域的基石。

![](img/fdecbca367067ad28fc7706513273640_59.png)

![](img/fdecbca367067ad28fc7706513273640_61.png)

## 降维技术：主成分分析

![](img/fdecbca367067ad28fc7706513273640_65.png)

![](img/fdecbca367067ad28fc7706513273640_67.png)

降维是无监督学习的另一项核心技术，用于减少数据集的特征数量，同时尽可能保留原始信息。这有助于缓解“维数灾难”，提高模型效率，并实现数据可视化。

![](img/fdecbca367067ad28fc7706513273640_69.png)

![](img/fdecbca367067ad28fc7706513273640_71.png)

![](img/fdecbca367067ad28fc7706513273640_73.png)

降维主要分两类：
1.  **特征选择**：从原有特征中直接选出一个子集。我们在之前课程中已接触过，例如使用LASSO回归或决策树中的特征重要性进行选择。
2.  **特征提取**：将原有特征变换到一个新的低维空间。**主成分分析**（PCA）是最经典的特征提取方法。

![](img/fdecbca367067ad28fc7706513273640_75.png)

![](img/fdecbca367067ad28fc7706513273640_77.png)

### 主成分分析

![](img/fdecbca367067ad28fc7706513273640_79.png)

![](img/fdecbca367067ad28fc7706513273640_81.png)

PCA的目标是找到数据中方差最大的方向（主成分），并将数据投影到这些方向上。第一个主成分是方差最大的方向，第二个主成分是与第一主成分正交且方差次大的方向，依此类推。

![](img/fdecbca367067ad28fc7706513273640_83.png)

数学上，PCA通过求解数据协方差矩阵的特征值和特征向量来实现。转换后的新特征（主成分）是原始特征的线性组合，且彼此不相关。

![](img/fdecbca367067ad28fc7706513273640_85.png)

在金融中，PCA常用于：
*   **风险因子分析**：从大量资产收益率中提取少数几个关键风险因子。
*   **数据预处理**：在构建多因子模型前，对高度相关的因子进行降维，消除多重共线性。
*   **可视化**：将高维投资组合数据降至2维或3维进行绘图观察。

使用`scikit-learn`实现PCA非常简单：
