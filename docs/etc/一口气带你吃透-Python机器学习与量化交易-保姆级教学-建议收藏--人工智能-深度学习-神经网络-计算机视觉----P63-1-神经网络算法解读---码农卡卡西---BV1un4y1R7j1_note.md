## 神经网络的基本思想

上一节我们概述了课程内容，本节中我们来看看神经网络的核心思想。深度学习或神经网络的一个代表性特点是“端到端”。

*   **传统算法**：像“把大象放进冰箱”一样，需要分步骤、分流程处理。例如，三维重建任务需要先估计深度，再估计点云，最后生成结果。
*   **深度学习/神经网络**：强调端到端。输入是原始数据，输出直接是最终结果，简化了中间过程。模型通过**损失函数**来学习目标，中间步骤由网络自动完成。

## 神经网络的发展简史

了解了基本思想后，我们简要回顾一下神经网络的发展历程。

*   **起源（1960年代）**：最初被称为“感知机”或“多层感知机（MLP）”。受限于计算性能，未能流行。
*   **再次出现（1997年）**：卷积神经网络开始应用于银行支票笔迹识别等实际场景。
*   **爆发（2012年）**：AlexNet（一个卷积神经网络）在ImageNet图像分类竞赛中以巨大优势夺冠，使深度学习重新进入主流视野。
*   **国内兴起（2014年左右）**：深度学习在国内成为创业和投资热点。

## 数据与挑战

在深入算法之前，我们需要理解神经网络处理的数据以及面临的核心挑战。

图像数据是计算机视觉中最常见的数据形式，它本质上是一个由长、宽、颜色通道组成的**矩阵**。在深度学习中，所有数据都被视为矩阵进行处理。

实际项目中，一个重大挑战是**跨域问题**。例如，在股票预测中，用历史数据训练的模型在实际应用中可能效果很差。这是因为历史数据和新数据的**分布**可能不同，模型在历史数据中学到的规律无法泛化到新数据上。这也是许多定制化AI项目难以产品化的原因。

## 神经网络的核心计算

现在，让我们进入核心部分，看看神经网络如何进行数学计算。

假设我们有一个输入图像，其矩阵形状为 `32 * 32 * 3`（即3072个像素点）。在传统神经网络中，我们首先将其**展平**成一个包含3072个特征的列向量 `X`。

我们的目标是做一个十分类任务。神经网络不会直接输出类别ID，而是输出该样本属于**各个类别的概率**。

为了得到十个类别的概率，我们需要十组不同的**权重参数** `W`（每组也是一个3072维的向量）。计算过程如下：
`输出值 = W * X + b`
其中，`*` 表示矩阵乘法（这里是向量点积），`b` 是**偏置项**。每组权重 `W` 和偏置 `b` 共同计算出一个类别的得分（尚未归一化为概率）。偏置项的数量与输出类别数一致，此处为10个。

**权重参数 `W` 的初始化**通常是随机的。在实际应用中，使用他人训练好的**预训练模型**作为初始化起点，往往比完全随机初始化效果更好，训练也更稳定。

## 损失函数与正则化

模型做出了预测，我们如何判断其好坏并指导它改进呢？这就需要损失函数。

损失函数衡量模型预测结果与真实标签之间的差异。差异越大，损失越大，说明模型需要调整。在分类任务中，常用的损失函数是**交叉熵损失**。

以下是计算交叉熵损失的关键步骤：
1.  **原始得分**：模型输出十个类别的原始得分。
2.  **Softmax归一化**：通过 `softmax` 函数将原始得分转换为概率。公式为：`概率_i = exp(得分_i) / sum(exp(得分_j))`。这确保了所有类别概率之和为1，且放大了得分间的差异。
3.  **计算损失**：只关注正确类别的预测概率。使用负对数似然计算损失：`损失 = -log(正确类别概率)`。当预测概率为1（完美）时，损失为0；概率越小，损失越大。

为了防止模型过于复杂、只记住训练数据（即**过拟合**），我们引入**正则化**，常见的是L2正则化。它会对权重参数 `W` 的大小进行惩罚，鼓励模型使用更小、更平滑的权重，从而提升泛化能力。
`总损失 = 数据损失（如交叉熵损失） + λ * 正则化惩罚项`
其中，λ 是控制正则化强度的超参数，正则化惩罚项通常是所有权重平方和。

## 反向传播与优化

计算出损失后，我们需要更新权重参数 `W` 以减少损失，这个过程通过**反向传播**和**梯度下降**实现。

*   **梯度下降**：将损失函数想象成一座山，我们的目标是找到山谷（损失最小点）。梯度方向是山坡最陡的方向。我们沿着梯度的**反方向**（即下山方向）更新权重，每次更新一小步（步长由学习率控制）。
*   **反向传播**：是一种高效计算损失函数对网络中**所有权重参数梯度**的方法。它利用链式法则，从输出层开始，逐层向前计算梯度。
*   **优化器**：简单的梯度下降存在一些问题。实践中常用更高级的优化器，如 **Adam**，它自适应地调整学习率，训练效果更好、更稳定。

权重参数是分别独立更新的。由于参数数量巨大（通常百万甚至千万级），即使部分参数未达到最优，整体模型性能也能接受。

## 神经网络的层次结构与本质

神经网络通常由多层结构组成，每一层都在进行**特征变换**。

*   **输入层**：接收原始数据（如3072个像素值）。
*   **隐藏层**：每一层都对上一层的输出进行线性变换（`WX+b`）和非线性激活（如ReLU函数），从而提取和组合出新的、更抽象的特征。深层网络可以看作进行“粗加工”到“细加工”的多级特征提取。
*   **输出层**：根据最终提取的特征，输出预测结果（如类别概率）。

神经网络的本质是一个强大的**特征学习器**。它自动从数据中学习出对当前任务最有效的特征表示，取代了传统方法中需要人工设计特征的过程。

## 其他重要概念

以下是构建和训练神经网络时涉及的其他关键技术点。

*   **激活函数**：为网络引入非线性，使其能够拟合复杂函数。早期使用Sigmoid函数，但容易导致梯度消失。现在最常用的是**ReLU函数**：`f(x) = max(0, x)`，它计算简单，能缓解梯度消失问题。
*   **数据预处理**：标准化是常见步骤，使每个特征维度均值为0，方差为1。公式为：`X_new = (X - mean) / std`。这有助于模型稳定、快速收敛。
*   **Dropout**：一种防止过拟合的正则化技术。在训练时，随机“丢弃”（暂时禁用）一部分神经元，迫使网络不过度依赖某些特定的特征，从而增强泛化能力。

## 过拟合演示

我们可以通过一个交互demo直观理解过拟合。在一个复杂的二分类数据集上：
*   使用少量神经元（如3个），模型可能无法很好地区分所有数据点（欠拟合）。
*   使用适当数量的神经元（如5个），模型可以学到合理的决策边界。
*   使用过多神经元（如15个），模型会“死记硬背”训练数据，甚至为个别噪声点创造复杂的边界，导致在训练集上表现完美，但在新数据上表现很差，这就是**过拟合**。
