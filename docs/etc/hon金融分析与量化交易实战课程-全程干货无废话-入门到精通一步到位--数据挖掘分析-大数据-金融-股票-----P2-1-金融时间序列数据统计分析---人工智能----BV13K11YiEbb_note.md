## 数据导入与时间索引设置

首先，我们需要导入数据，并将日期列设置为数据框的索引，同时确保其格式为标准的时间格式。

以下是读取数据并设置时间索引的代码：

```python
# 假设数据文件为 ‘financial_data.csv‘，其中包含名为 ‘date‘ 的日期列
import pandas as pd

# 读取数据，并指定 ‘date‘ 列为索引，同时解析日期格式
data = pd.read_csv(‘financial_data.csv‘, index_col=‘date‘, parse_dates=True)
```

执行上述代码后，数据框的索引将变为标准的 `DatetimeIndex` 格式。数据中可能包含空值，例如某些日期（如1月1日）的指标值为空，这是正常现象。

为了更清晰地查看数据，我们可以使用 `.head()` 方法预览前几行。

```python
# 查看数据前几行
print(data.head())
```

## 数据可视化：展示各指标走势

上一节我们完成了数据的导入，本节中我们来看看如何将多个金融指标的走势进行可视化展示。

以下是两种常见的绘图方法：

**方法一：将所有指标绘制在同一张图中进行对比。**

```python
# 使用pandas内置的绘图功能，将所有序列画在一张图上
data.plot(figsize=(15, 10))
```

**方法二：将每个指标绘制在独立的子图中，以便更清晰地观察单个走势。**

```python
# 使用 subplots 参数，将每个序列画在独立的子图中
data.plot(subplots=True, figsize=(15, 20))
```

两种方法各有优势。在同一张图中便于进行横向对比，而子图形式则能更清晰地展示每个指标的独立走势。可以根据分析需求选择合适的方式。

当前数据包含多个指标，例如不同公司（如苹果、微软）的股价，以及汇率、黄金价格等其他金融指标。

## 数据的基本统计分析

在完成数据导入和初步可视化后，我们需要对数据进行基本的统计分析，以了解数据的整体情况。

首先，我们可以查看数据的基本信息，包括样本数量、时间范围、数据类型和缺失值情况。

```python
# 查看数据基本信息
print(f“样本数量：{len(data)}“)
print(f“时间范围：{data.index.min()} 到 {data.index.max()}“)
print(“数据类型：“)
print(data.dtypes)
print(“\n缺失值统计：“)
print(data.isnull().sum())
```

从输出中，我们可以看到数据共有2216个样本，时间跨度从2010年1月1日到2018年6月29日。各列均为浮点数类型，且存在不同程度的缺失值。

---

## 计算描述性统计指标

接下来，我们计算各项指标的核心统计量，如计数、均值、标准差、最小值、最大值、四分位数等。

最直接的方法是使用 `.describe()` 函数。

```python
# 计算描述性统计，并保留两位小数
summary_stats = data.describe().round(2)
print(summary_stats)
```

`.describe()` 函数默认提供了丰富的统计指标。通过 `.round(2)` 可以控制输出结果的小数位数，使结果更清晰易读。

---

## 使用聚合函数自定义统计汇总

上一节我们使用了 `.describe()` 获取了全面的统计摘要。本节中我们来看看如何根据特定需求，使用聚合函数自定义需要汇总的统计指标。

以下是使用 `agg` 函数汇总特定统计指标的方法：

```python
# 自定义需要汇总的统计指标：最小值、最大值、均值、标准差、中位数
custom_stats = data.agg([‘min‘, ‘max‘, ‘mean‘, ‘std‘, ‘median‘])
print(custom_stats)
```

通过 `agg` 函数，我们可以灵活选择需要计算的统计量（如 `numpy.mean`, `numpy.std`），而无需包含 `.describe()` 中的所有项。这种方法在只需要关注部分核心指标时非常高效。
