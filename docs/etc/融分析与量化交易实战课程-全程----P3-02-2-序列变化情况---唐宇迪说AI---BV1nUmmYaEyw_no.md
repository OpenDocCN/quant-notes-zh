## 计算每日差异值

![](img/215e3bf48d227566d3f82f1d94acfdec_6.png)

首先，我们需要计算每一天指标与前一天的差异值。这个指标反映了数据每日的绝对变化量。

![](img/215e3bf48d227566d3f82f1d94acfdec_8.png)

在Pandas中，我们可以使用 `diff()` 函数轻松实现这个计算。该函数会计算当前行与前一行的差值。

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

![](img/215e3bf48d227566d3f82f1d94acfdec_10.png)

执行上述代码后，你会看到一个新的DataFrame。例如，1月5日的数据等于其原始值减去1月4日的值。1月4日对应的结果会是 `NaN`（非数字），因为其前一天没有数据可供相减。

![](img/215e3bf48d227566d3f82f1d94acfdec_12.png)

这个差异值指标在时间序列分析中非常常用。

## 计算增长率

差异值表示的是绝对变化，但有时我们更关心相对变化，即增长率。例如，股价从1元涨到2元（增长100%）和从10000元涨到10001元（增长0.01%）的意义截然不同。

![](img/215e3bf48d227566d3f82f1d94acfdec_20.png)

增长率的计算公式为：`(后一天的值 - 前一天的值) / 前一天的值`。

Pandas提供了 `pct_change()` 函数直接计算这个百分比变化。

![](img/215e3bf48d227566d3f82f1d94acfdec_22.png)

```python
# 计算每日增长率
pct_data = data.pct_change()
# 设置显示格式，保留三位小数
pd.set_option('display.float_format', lambda x: '%.3f' % x)
print(pct_data.head())
```

![](img/215e3bf48d227566d3f82f1d94acfdec_24.png)

计算后，数据将显示为百分比形式（例如0.052表示增长5.2%）。股票日增长率通常在这个量级。

![](img/215e3bf48d227566d3f82f1d94acfdec_26.png)

![](img/215e3bf48d227566d3f82f1d94acfdec_28.png)

## 分析增长率的统计特征

![](img/215e3bf48d227566d3f82f1d94acfdec_30.png)

得到每日增长率序列后，我们可以对其进行统计分析，例如计算所有交易日增长率的平均值。

![](img/215e3bf48d227566d3f82f1d94acfdec_32.png)

```python
# 计算各列增长率的平均值
mean_pct = pct_data.mean()
print(mean_pct)
```

![](img/215e3bf48d227566d3f82f1d94acfdec_34.png)

## 可视化统计结果

为了更直观地展示各指标的平均增长率，我们可以将其可视化。由于平均值是单个数值，适合用条形图表示。

![](img/215e3bf48d227566d3f82f1d94acfdec_36.png)

```python
# 绘制平均增长率的条形图
mean_pct.plot(kind='bar', figsize=(16, 8))
plt.show()
```

![](img/215e3bf48d227566d3f82f1d94acfdec_38.png)

通过图表，我们可以一目了然地比较不同指标的平均增长情况。Y轴的值通常较小，符合金融数据的普遍特征。
