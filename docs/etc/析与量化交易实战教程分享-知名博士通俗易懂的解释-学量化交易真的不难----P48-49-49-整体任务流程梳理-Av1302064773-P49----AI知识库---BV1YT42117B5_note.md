## 🧮 因子打分法详解

上一节我们介绍了如何获取和预处理因子数据。本节中我们来看看如何利用这些已知的因子值为股票打分。

打分的方法有很多种，我们通过一个具体的例子来讲解。

假设我们有一个包含300只股票（例如沪深300成分股）的股票池。我们选取了四个因子进行分析：
*   **因子A**: 越大越好
*   **因子B**: 越大越好
*   **因子C**: 越小越好
*   **因子D**: 越小越好

每只股票都有这四个因子的具体数值。为了便于演示，我们假设这些数值已经过归一化处理，范围在0到1之间。

![](img/310ab8741816871066b1d9b2a46d5351_5.png)

以下是两只示例股票的数据：

| 股票ID | 因子A | 因子B | 因子C | 因子D |
| :--- | :--- | :--- | :--- | :--- |
| ID1 | 0.71 | 0.28 | 0.39 | 0.11 |
| ID2 | 0.45 | 0.17 | 0.81 | 0.02 |

![](img/310ab8741816871066b1d9b2a46d5351_7.png)

### 方法一：区间划分打分法
这种方法首先将每个因子的取值范围划分为若干个区间，并为每个区间赋予一个分数。

![](img/310ab8741816871066b1d9b2a46d5351_9.png)

![](img/310ab8741816871066b1d9b2a46d5351_11.png)

以下是划分区间和赋分的原则：

1.  **对于“越大越好”的因子（如A、B）**：数值越大的区间，得分越高。
    *   `[0.8, 1.0]` -> 5分
    *   `[0.6, 0.8)` -> 4分
    *   `[0.4, 0.6)` -> 3分
    *   `[0.2, 0.4)` -> 2分
    *   `[0.0, 0.2)` -> 1分

2.  **对于“越小越好”的因子（如C、D）**：数值越小的区间，得分越高。
    *   `[0.0, 0.2)` -> 5分
    *   `[0.2, 0.4)` -> 4分
    *   `[0.4, 0.6)` -> 3分
    *   `[0.6, 0.8)` -> 2分
    *   `[0.8, 1.0]` -> 1分

![](img/310ab8741816871066b1d9b2a46d5351_13.png)

现在，我们来为示例股票打分：

*   **股票ID1**:
    *   因子A (0.71): 落入 `[0.6, 0.8)`，得 **4分**。
    *   因子B (0.28): 落入 `[0.2, 0.4)`，得 **2分**。
    *   因子C (0.39): 注意，C是“越小越好”。0.39落入 `[0.2, 0.4)`，对应 **4分**。
    *   因子D (0.11): “越小越好”，落入 `[0.0, 0.2)`，得 **5分**。
    *   **总分** = 4 + 2 + 4 + 5 = **15分**。

*   **股票ID2**:
    *   因子A (0.45): 落入 `[0.4, 0.6)`，得 **3分**。
    *   因子B (0.17): 落入 `[0.0, 0.2)`，得 **1分**。
    *   因子C (0.81): “越小越好”，落入 `[0.8, 1.0]`，得 **1分**。
    *   因子D (0.02): “越小越好”，落入 `[0.0, 0.2)`，得 **5分**。
    *   **总分** = 3 + 1 + 1 + 5 = **10分**。

通过这种方式，我们可以计算出股票池中所有300只股票的总分。

### 方法二：直接排序打分法
除了划分区间，还有一种更直接的方法。以下是具体步骤：

1.  对于“越大越好”的因子，将300只股票按该因子值**从大到小**排序。
2.  排名第1的股票赋予最高分（例如300分），排名第300的股票赋予最低分（例如1分）。也可以使用百分位数赋分。
3.  对于“越小越好”的因子，则将股票按该因子值**从小到大**排序，再进行赋分。
4.  最后，同样将各因子得分相加得到每只股票的总分。

![](img/310ab8741816871066b1d9b2a46d5351_15.png)

**核心目标**是获得一个可比较的总分指标。无论采用哪种打分方法，最终都是为了将所有股票的总分进行**排名**。

## 🔄 整体策略流程梳理

![](img/310ab8741816871066b1d9b2a46d5351_19.png)

![](img/310ab8741816871066b1d9b2a46d5351_21.png)

理解了打分方法后，我们来看整个量化策略的实现流程。这个流程就是我们接下来要编码实现的策略框架。

![](img/310ab8741816871066b1d9b2a46d5351_23.png)

![](img/310ab8741816871066b1d9b2a46d5351_25.png)

以下是策略的核心步骤：

![](img/310ab8741816871066b1d9b2a46d5351_27.png)

1.  **确定股票池**：首先，需要明确我们的选股范围。在本例中，我们将在`context`中指定初始股票池为**沪深300指数成分股**。

2.  **设置调仓周期**：确定策略再平衡（调仓）的频率。常见的频率是每月或每季度。我们需要设置一个**定时器**，在每月初触发调仓函数。

3.  **实现再平衡函数**：这是策略的核心函数（例如 `rebalance`）。在该函数中，我们将执行以下操作：
    *   **数据获取与预处理**：获取当前股票池内所有股票的选定因子数据，并进行必要的清洗与标准化。
    *   **因子打分**：应用上述打分法（如区间划分法），为每只股票的每个因子计算得分。
    *   **计算总分**：将每只股票的所有因子得分相加，得到其综合总分。
        ```python
        # 伪代码示例：计算总分
        total_score = score_factor_A + score_factor_B + score_factor_C + score_factor_D
        ```
    *   **筛选标的**：根据总分对所有股票进行**降序排名**，并选择排名最高的前N只（例如前10只）股票作为本次调仓的目标买入标的。
    *   **执行交易**：根据目标持仓比例，卖出不在名单中的股票，买入名单中的股票。

这个流程结构清晰，是因子选股策略的典型范式。接下来，我们将尝试用这种打分法构建一个简单的策略，并回测其历史表现。
