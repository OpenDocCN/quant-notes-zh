## 什么是业务指标？

业务指标是衡量业务好坏的标准。一个好的、正确的指标能够清晰地反映业务的核心状况。

例如，对于销售公司，**销售额**是衡量其经营情况的核心指标。销售额越高，通常意味着公司经营得越好。然而，仅看销售额可能不够全面，还需要结合成本、利润等指标综合判断。

## 为何要学习业务指标？

![](img/1aceeb9009b30c3e5f8fc2effb754832_15.png)

学习业务指标主要有三个目的：

1.  **了解公司基本业务**：明确公司做什么、卖什么产品、服务哪些对象。
2.  **评估业务经营情况**：通过核心指标判断业务哪里好、哪里不好。例如，通过对比月度销售额，可以直观看出业务是增长还是下滑。
3.  **定位业务流程问题**：通过分析用户在网站或APP中的行为路径（如浏览商品 -> 查看详情 -> 加入购物车 -> 支付），可以定位是哪个环节导致了用户流失，从而针对性优化。

![](img/1aceeb9009b30c3e5f8fc2effb754832_17.png)

为了更具体地理解，我们聚焦于互联网电商行业，将指标分为用户数据、用户行为数据和产品数据三类。首先，我们从用户数据层面开始。

## 用户数据指标

用户数据指标主要关注用户的获取、活跃、留存以及价值。

![](img/1aceeb9009b30c3e5f8fc2effb754832_19.png)

### 日新增用户数

**日新增用户数** 指的是产品每天新增的用户数量。“新增”是指用户第一次访问或使用产品。

例如，用户第一次下载并使用Keep健身APP，他就成为当天的日新增用户。如果用户卸载后重新下载，则不算新增，因为系统能通过设备唯一标识识别老用户。

这个指标反映了产品的“拉新”能力。初期，公司需要通过市场活动、广告投放等方式获取新用户。但需注意，单纯追求新增用户数可能带来“僵尸用户”（即注册后不再活跃的用户），因此需要结合其他指标综合评估。

### 活跃用户数

**活跃用户数** 指在产品中产生了有价值行为的用户数量。相对于新增用户，活跃用户更能体现产品的真实价值。

如何定义“活跃”因公司而异，常见标准有：
*   打开应用即算活跃（标准较宽松）。
*   用户登录账号才算活跃。
*   用户完成指定行为（如阅读文章、发布内容、完成购买）才算活跃（标准较严格）。

活跃用户数按时间维度可分为：
*   **日活跃用户数（DAU）**：当天产生活跃行为的用户数。行业内常简称为“日活”。
*   **周活跃用户数（WAU）**
*   **月活跃用户数（MAU）**

**活跃率** 是活跃用户数占总用户数的比例（公式：`活跃率 = 活跃用户数 / 总用户数`）。比率指标通常比绝对数更能反映产品的健康程度。

### 留存率

**留存率** 衡量产品留住用户的能力。它考察的是，新增用户在后续一段时间内，是否仍然会访问或使用你的产品。

计算公式为：
`第N天留存率 = (第1天新增用户中，在第N天仍访问的用户数) / 第1天新增用户总数`

常见的留存率指标有：
*   **次日留存率**：新增用户次日仍访问的比例。
*   **7日留存率**：新增用户第7天仍访问的比例。
*   **月留存率**：新增用户第30天仍访问的比例。

留存率至关重要，因为它关系到用户获取成本能否收回并产生长期价值。行业有一个“40-20-10”的模糊参考标准：次日留存率约40%，7日留存率约20%，月留存率约10%。

### 单位获客成本

**单位获客成本** 指平均获取一个客户所花费的成本。

计算公式为：
`单位获客成本 = 广告投放总费用 / 该广告带来的独立访客数`

这个指标用于评估推广渠道的效果。我们需要将单位获客成本与客户带来的平均收入、转化率等指标关联分析。如果成本持续上升，而收入或转化率没有相应增长，就说明推广策略可能存在问题。

### 客单价与用户价值

以下是衡量用户价值的相关指标：

*   **客单价**：平均每个订单的金额。常用于零售和电商。
    `客单价 = 订单总金额 / 订单数量`
*   **平均用户收入（ARPU）**：平均每个用户（包括非付费用户）带来的收入。反映整体用户的平均价值。
    `ARPU = 总收入 / 总用户数`
*   **平均付费用户收入（ARPPU）**：平均每个付费用户带来的收入。更聚焦于付费用户群体的消费能力。
    `ARPPU = 总收入 / 付费用户数`

对于游戏、订阅服务等产品，ARPU和ARPPU是评估用户价值和商业模式健康度的重要指标。
