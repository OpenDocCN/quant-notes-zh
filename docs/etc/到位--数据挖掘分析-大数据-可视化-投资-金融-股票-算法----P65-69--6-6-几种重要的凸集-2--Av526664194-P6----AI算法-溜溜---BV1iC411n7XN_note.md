## 多面体 📦

![](img/35135fd4ab48cf8002ede77c8e097182_21.png)

![](img/35135fd4ab48cf8002ede77c8e097182_23.png)

![](img/35135fd4ab48cf8002ede77c8e097182_25.png)

下面要讲的概念在线性规划中非常常用，叫做多面体。

![](img/35135fd4ab48cf8002ede77c8e097182_27.png)

![](img/35135fd4ab48cf8002ede77c8e097182_29.png)

![](img/35135fd4ab48cf8002ede77c8e097182_31.png)

多面体仍然是一个集合。所以多面体也是这样一个集合。

这个集合中的元素X满足以下性质：首先，它要满足一系列不等式，总共有M个不等式，形式为 **a_j^T x ≤ b_j**。另外，还有P个等式，形式为 **c_j^T x = d_j**。这样一个集合就称为多面体。

![](img/35135fd4ab48cf8002ede77c8e097182_33.png)

多面体实际上表达的意思是：第一个式子中的每个不等式实际上定义了一个半空间，第二个式子中的每个等式定义了一个超平面。所以，多面体实际上是一些半空间和一些超平面的交集。

![](img/35135fd4ab48cf8002ede77c8e097182_35.png)

![](img/35135fd4ab48cf8002ede77c8e097182_37.png)

![](img/35135fd4ab48cf8002ede77c8e097182_39.png)

只要选择一些半空间以及一些超平面，取它们的交集，就可以得到一个多面体。

![](img/35135fd4ab48cf8002ede77c8e097182_40.png)

![](img/35135fd4ab48cf8002ede77c8e097182_41.png)

![](img/35135fd4ab48cf8002ede77c8e097182_42.png)

![](img/35135fd4ab48cf8002ede77c8e097182_43.png)

![](img/35135fd4ab48cf8002ede77c8e097182_45.png)

多面体不一定是有界的。例如，考虑最简单的情况：多面体只有一个半空间约束。根据定义，它肯定是一个多面体，但它看起来不像通常的“体”，并且它是无界的，因为其取值可以趋于无穷大。为了符合我们的习惯，通常我们想象的多面体是体积有限的。对于这种体积有限的多面体，我们称之为有界多面体。这个概念比较简单，不再赘述。

![](img/35135fd4ab48cf8002ede77c8e097182_47.png)

![](img/35135fd4ab48cf8002ede77c8e097182_49.png)

我们关心的问题是：多面体是不是一个凸集？一个多面体一定是一个凸集。因为其中的每个约束条件都可以验证：任取两个点X1和X2，验证它们的凸组合 **θX1 + (1-θ)X2** 是否仍然满足这些线性不等式和等式约束，结果肯定是满足的。

![](img/35135fd4ab48cf8002ede77c8e097182_51.png)

![](img/35135fd4ab48cf8002ede77c8e097182_53.png)

![](img/35135fd4ab48cf8002ede77c8e097182_54.png)

多面体这个概念还是比较简单的。

![](img/35135fd4ab48cf8002ede77c8e097182_55.png)

![](img/35135fd4ab48cf8002ede77c8e097182_56.png)

![](img/35135fd4ab48cf8002ede77c8e097182_57.png)

## 单纯形 🔺

![](img/35135fd4ab48cf8002ede77c8e097182_60.png)

我们再讲一个稍微复杂一点的概念。

![](img/35135fd4ab48cf8002ede77c8e097182_62.png)

![](img/35135fd4ab48cf8002ede77c8e097182_63.png)

下面要讲的是一种比较特殊的多面体，一种非常特殊的多面体叫做单纯形。你们应该都听说过单纯形法。单纯形法这个概念与单纯形的定义是有关系的，它们两者不是一回事，但一定是有联系的。

![](img/35135fd4ab48cf8002ede77c8e097182_65.png)

什么东西叫做一个单纯形呢？定义时需要注意：为了定义一个单纯形，首先考虑一个N维空间。在这个空间里，选择K+1个点，从V0一直到VK。我们要求选择这些点时，向量 **V1 - V0, ..., VK - V0** 必须是线性无关的。这是选择这些点时的唯一要求。

![](img/35135fd4ab48cf8002ede77c8e097182_67.png)

选择了这些点后，与这些点相关，我们就可以定义一个单纯形：它是这些点的凸包。我们把它称为单纯形。

![](img/35135fd4ab48cf8002ede77c8e097182_69.png)

为了简单起见，把系数 θ₀ 到 θ_K 都大于等于零写作 **θ ≥ 0**（θ是向量），把系数之和等于1写作 **e^T θ = 1**（e是全1向量）。所以，这个概念就是：首先找K+1个点，满足 **V1 - V0, ..., VK - V0** 这K个向量线性无关。如果它们线性无关，就可以找这K+1个点的凸包。这个定义看起来不太容易理解。

![](img/35135fd4ab48cf8002ede77c8e097182_71.png)

我们找一个简单的例子来讲讲，看看在二维情况下会是什么样子。

![](img/35135fd4ab48cf8002ede77c8e097182_73.png)

例如，在空间中选择K=1，总共选择两个点V0和V1。那么 **V1 - V0** 这个向量是线性无关的（只要不等于零）。由这两点构造的单纯形是什么样子呢？是连接这两点的线段。也就是说，对这两点求它们的凸包即可。

![](img/35135fd4ab48cf8002ede77c8e097182_75.png)

再把K增大一点，变成三个点，即K=2。这时要考虑两个向量：**V1 - V0** 和 **V2 - V0**。只要这两个向量不共线，它们一定是线性无关的。与这三个点对应的单纯形是什么样子？是一个三角形（包含内部区域）。

第一个是单纯形（线段），第二个也是单纯形（三角形）。再把情况稍微扩大一点，考虑四个点。那么这三个向量是线性无关的吗？不是。在二维空间里，无法找到三个线性无关的向量。因为对前两个向量做线性组合，一定可以得到第三个向量。所以它不是线性无关的。既然不是线性无关，也就无法构造一个单纯形。

实际上我们有一个结论：对于一个二维空间，无法找到四个或以上的点来构造单纯形。也就是说，在二维空间里，单纯形只有两种情况：一种是线段，另一种是三角形。在二维空间里不存在一个四边形是单纯形，因为无法找到四个或以上的点使得相关向量线性无关。

那么在三维空间里，一个单纯形是什么样子的呢？假设在三维空间里选择四个点，这四个点能满足向量线性无关的条件，那么这样一个单纯形是一个四面体。在三维空间里，我们无法找到五面体、六面体使其成为单纯形。这个概念实际上在线性规划的单纯形法中有相似的概念，但这里不展开说。

下面我们再来看一件事：一个单纯形，它一定是一个多面体。我们要说明这样一件事情。从直观上这是非常好理解的，因为我们刚刚画的图可以很清楚看见一个单纯形一定是一个多面体。但在数学上应该怎么证明呢？

![](img/35135fd4ab48cf8002ede77c8e097182_77.png)

![](img/35135fd4ab48cf8002ede77c8e097182_79.png)

![](img/35135fd4ab48cf8002ede77c8e097182_81.png)

![](img/35135fd4ab48cf8002ede77c8e097182_83.png)

这里把证明写出来。这个证明比较复杂，但很有好处，可以让你们理解凸优化这门课中最简单的命题证明起来是什么样的情况。下面要证明的命题实际上是这门课里一个非常容易的命题。

![](img/35135fd4ab48cf8002ede77c8e097182_85.png)

我们要证明的命题是：一个单纯形是一种多面体。或者说，任意的一个单纯形，都可以用多面体的数学形式表达出来。这就是我们要证明的事情。

![](img/35135fd4ab48cf8002ede77c8e097182_87.png)

证明思路应该是用构造的方法：如果给你一个单纯形，那么一定可以把它写成多面体的形式。首先，想办法构造一个单纯形，接着验证里面的点是否可以用多面体描述出来。

![](img/35135fd4ab48cf8002ede77c8e097182_89.png)

实际上，我们把单纯形的定义重新抄了一遍：如果一个X是单纯形C中的一个点，那么它一定可以表达成V0到VK这K+1个点的凸组合。而且，向量 **V1 - V0, ..., VK - V0** 是线性无关的。相反，如果变化这些系数θ，就可以得到C中所有的X。如果确定一个θ，就得到C中的一个X；如果变化θ，就可以得到整个C。

![](img/35135fd4ab48cf8002ede77c8e097182_91.png)

下面定义一个新的向量和一个新的矩阵。新的向量很简单：把之前的θ向量中的θ₀去掉，只保留θ₁到θ_K，组成一个K维向量，称为y。那么y一定满足两个性质：首先，**y ≥ 0**（每个元素大于等于零）；其次，**e^T y ≤ 1**（元素之和小于等于1），因为去掉了θ₀。

![](img/35135fd4ab48cf8002ede77c8e097182_93.png)

![](img/35135fd4ab48cf8002ede77c8e097182_94.png)

另外，构造一个矩阵B，是把所有的K个向量 **V1 - V0, ..., VK - V0** 拼到一块构成的矩阵，它是一个N×K的矩阵。

![](img/35135fd4ab48cf8002ede77c8e097182_95.png)

![](img/35135fd4ab48cf8002ede77c8e097182_96.png)

![](img/35135fd4ab48cf8002ede77c8e097182_98.png)

下面再把X拆开。X一定是V0到VK的凸组合。把它拆一下，把V0拆出来，目的是把这K个线性无关的向量找出来。写成 **X = V0 + B y** 的形式。

下面要讲一件很重要的事情：因为这K个向量是线性无关的，即矩阵B中的列向量线性无关，所以矩阵B的秩一定等于K。因为它包含K个线性无关的向量，所以它的秩一定等于K。由于K≤N，所以它是一个列满秩矩阵。

![](img/35135fd4ab48cf8002ede77c8e097182_100.png)

既然它是列满秩矩阵，那么自然可以通过某种线性变换把它变成特定形式：任意一个列满秩矩阵，总可以通过一个线性变换，把它变成 **[I; 0]** 的形式，其中上面是K×K的单位矩阵I，下面是(N-K)×K的零矩阵。这是基本的线性代数知识。

![](img/35135fd4ab48cf8002ede77c8e097182_102.png)

怎么变呢？一定可以找到一个非奇异的方阵A（其所有特征值非零），使得 **A B = [I; 0]**。

![](img/35135fd4ab48cf8002ede77c8e097182_104.png)

![](img/35135fd4ab48cf8002ede77c8e097182_106.png)

![](img/35135fd4ab48cf8002ede77c8e097182_107.png)

我们之前讲过，**X = V0 + B y**。这个式子与什么等价呢？把式子左右两端都乘上A，得到 **A X = A V0 + A B y**。从左边到右边是成立的。但是，如果A不满足一定条件，无法从右边回到左边。这里有一个很重要的要求：A必须是非奇异矩阵。因为这样，就可以在等式两边左乘A的逆矩阵，得到原式。

![](img/35135fd4ab48cf8002ede77c8e097182_109.png)

![](img/35135fd4ab48cf8002ede77c8e097182_111.png)

既然这个式子成立，那么就好办了。把矩阵A拆成两部分：A1和A2。我们可以把这个式子拆成两个部分。实际上，这个式子相当于两个式子：我们把上面和下面的式子分开。

![](img/35135fd4ab48cf8002ede77c8e097182_113.png)

![](img/35135fd4ab48cf8002ede77c8e097182_115.png)

**A1 B** 应该等于单位矩阵I，所以这部分就变成了y。**A2 B** 应该等于零矩阵，所以这部分就没有了。这里还有个y存在。

![](img/35135fd4ab48cf8002ede77c8e097182_117.png)

![](img/35135fd4ab48cf8002ede77c8e097182_119.png)

![](img/35135fd4ab48cf8002ede77c8e097182_121.png)

y是什么意思呢？y实际上是系数向量θ的一部分，即从θ₁到θ_K。我们之前讲过，如果改变这些系数θ，就可以得到集合中任意的X。所以实际上，这时我们假设X是可变的。也就是说，对于集合中的任意一个X，这个式子都是成立的。

![](img/35135fd4ab48cf8002ede77c8e097182_123.png)

从这个式子可以得到什么推论呢？y是一个变量，但这个y要满足要求：**y ≥ 0** 且 **e^T y ≤ 1**。所以实际上，就可以把这个等式变成不等式，把它变成两个不等式和一个等式。

![](img/35135fd4ab48cf8002ede77c8e097182_125.png)

![](img/35135fd4ab48cf8002ede77c8e097182_127.png)

这两个不等式是根据y的两个性质得来的。把y的这两个性质带进去，就可以得到这两个不等式。我们再讲一下：如果变化y，就可以得到单纯形中所有的点。所以也就是说，任意的一个X（单纯形中任意的X），都可以用这个式子来描述。而且，任意一个满足这个式子的X，一定描述的是这个单纯形中的一个X。这个正向和反向都是成立的。

![](img/35135fd4ab48cf8002ede77c8e097182_129.png)

![](img/35135fd4ab48cf8002ede77c8e097182_131.png)

所以，任意一个单纯形，一定都可以用这样两个线性不等式和一个线性等式来描述。所以说，任意的一个单纯形都一定是一个多面体。证明结束。

![](img/35135fd4ab48cf8002ede77c8e097182_133.png)

听起来好像没有问题，但是一定要把这个弯转过来：这里面的y是遍历整个集合的变量。你们仔细去想想这个证明。

![](img/35135fd4ab48cf8002ede77c8e097182_135.png)

![](img/35135fd4ab48cf8002ede77c8e097182_137.png)

## 对称矩阵集合 ⬡

到目前为止，我们讲的凸集概念都可以认为是空间中的几何体。我们之前讲的这些凸集，包括球、椭球、多面体、单纯形，都可以利用空间想象力去想象它到底是什么样子。但是在数学里，我们更讲究抽象，有些东西是无法想象的。下面我们要讲这样一个凸集。

![](img/35135fd4ab48cf8002ede77c8e097182_139.png)

我们考虑一个对称矩阵的集合。把它记为 **S^n**。我们之前已经讲过对称正定矩阵。所以这个符号你们应该比较清楚了：S表示对称的意思，n表示n维空间。没有其他要求，只要求它对称，即 **X = X^T**。这是所谓对称矩阵的集合。

![](img/35135fd4ab48cf8002ede77c8e097182_141.png)

![](img/35135fd4ab48cf8002ede77c8e097182_143.png)

![](img/35135fd4ab48cf8002ede77c8e097182_144.png)

相应的，我们有一个对称的半正定矩阵的集合，中间加上一个加号，表示半正定的意思。也就是说，它的所有特征值一定都要大于等于零。后面这个式子，看起来写的是 **X ≥ 0**，但其实不是。一般如果我们写 **X ≥ 0**，表示的是矩阵中每个元素都大于等于零。但是，如果写成右边这个形式，描述的意思是它的特征值都要大于等于零。这是一个非常重大的区别，一定要记住。

![](img/35135fd4ab48cf8002ede77c8e097182_145.png)

![](img/35135fd4ab48cf8002ede77c8e097182_146.png)

![](img/35135fd4ab48cf8002ede77c8e097182_147.png)

既然有半正定矩阵，也相应会有正定矩阵。这里 **X > 0** 描述的并不是说X是一个正矩阵，而是指X是一个正定矩阵。这三个符号你们要记住，很多时候看文章或者推公式时可能会碰到。

![](img/35135fd4ab48cf8002ede77c8e097182_149.png)

![](img/35135fd4ab48cf8002ede77c8e097182_151.png)

但是我们这里讲的是一个凸集的概念。那么我们的问题是：这样三个集合是不是凸集？这就不好想象了。因为我们之前想象凸集时，都可以利用空间想象力，因为我们都是生活在空间里的。但是，我们每个人都不是生活在一个矩阵空间里的。这个矩阵空间是n×n矩阵的空间，它是矩阵空间的一个子空间。也就是说，这个空间中的每个元素都是一个矩阵。除非你是一个矩阵，否则你一定不能知道这个空间是什么样子，不能想象这个空间是什么样子。所以，这个时候我们就不能用我们的想象力了，要利用我们的逻辑思维能力。

![](img/35135fd4ab48cf8002ede77c8e097182_153.png)

![](img/35135fd4ab48cf8002ede77c8e097182_155.png)

下面我们要证明一件事情：首先，我们来看看对称半正定矩阵集合有什么样的性质。我们要证明的是，它不光是一个凸集，它还是一个凸锥。如果它是凸锥，那么它一定是个凸集。

![](img/35135fd4ab48cf8002ede77c8e097182_157.png)

证明实际上也是按部就班的，按照凸锥的定义来就可以了。也就是说，任意选择两个实数θ₁和θ₂（当然这两个数都要大于等于零），然后任意从集合中选择两个矩阵A和B（它们都是对称半正定的矩阵）。我们要证明的是它们的凸锥组合 **θ₁A + θ₂B** 仍然在这个集合内，仍然是一个对称半正定矩阵。

![](img/35135fd4ab48cf8002ede77c8e097182_159.png)

对称这个事情很好证明：如果A是对称矩阵，B是对称矩阵，那么 **θ₁A + θ₂B** 一定也是个对称矩阵，非常简单。所以我们要证明的就是半正定这一块。

什么样的一个矩阵是半正定的呢？我们要算它的特征值，但算特征值太麻烦了。我们一般有一个简单的描述：如果一个矩阵A（或B）是半正定矩阵，那么任取一个n维向量x，**x^T A x** 一定是要大于等于零的，**x^T B x** 也是要大于等于零的。这里的结果是一个标量，要大于等于零。这实际上就是关于半正定的定义。

![](img/35135fd4ab48cf8002ede77c8e097182_161.png)

现在有了这两个性质，我们就要证明新的点也要满足这个性质。要证明 **x^T (θ₁A + θ₂B) x ≥ 0**。这个值是不是大于等于零的呢？非常简单，把这个式子展开：**θ₁ (x^T A x) + θ₂ (x^T B x)**。它一定是大于等于零的。所以我们说，这样一个对称半正定矩阵集合，它一定是一个凸锥，所以它也一定是个凸集。

![](img/35135fd4ab48cf8002ede77c8e097182_163.png)

那么对称的矩阵集合它是不是一个凸锥呢？是的，因为我们这边其实就已经讲过了，验证对称是非常容易验证的。

那么对称的正定矩阵集合是不是一个凸锥呢？是还是不是？看起来好像是。但其实不是。为什么会这样呢？我们待会再讲。先讲一个简单的例子，很快你就会明白它为什么不是。

## 直观理解与特例 🔍

我们刚刚虽然讲的是，在考虑矩阵空间时要运用抽象思维。但很多时候抽象思维是不够用的，还是得用形象的思维。很简单，怎么样去想象形象思维呢？就是把n变成1，考虑最简单的情况。

当n变成1的时候，我们仍然可以去构造这样一些所谓的对称矩阵集合、对称的半正定矩阵集合以及对称的正定矩阵集合。这时候，我们的空间想象力就可以发挥作用了，因为这个集合中的每个元素都是一个点。

![](img/35135fd4ab48cf8002ede77c8e097182_165.png)

例如，当n=1时，对称的半正定矩阵集合就变成了非负的实数空间。首先，一个标量X一定等于它的转置。其次，要保证它的特征值大于等于零，实际上就是说这个元素本身要大于等于零。

相应的，正定矩阵的集合就变成了正实数的集合。一个正实数的集合，它是不是一个凸锥？不是。因为正数的集合不包含原点。如果它不包含原点，那么它一定不是凸锥。但是，它是一个凸集。这也就是我们为什么要讲它的原因，因为我们现在讲的是凸集。对称的矩阵集合就很简单了，就是整个实数空间。

![](img/35135fd4ab48cf8002ede77c8e097182_167.png)

现在回过头来再看看一般n维空间的情况。看看为什么我们没法证明对称正定矩阵集合是一个凸锥。问题出在哪里呢？因为对称正定矩阵是这样的：如果A和B是对称正定矩阵，那么 **x^T A x > 0** 和 **x^T B x > 0** 是满足的。现在，式子 **θ₁ (x^T A x) + θ₂ (x^T B x)** 到这里都还是大于等于零的，但不是严格大于零。原因很简单，因为我们的θ₁和θ₂是可以取零的。对于一个凸锥的要求是，只要θ₁和θ₂大于等于零就可以，不要求和为1。那为什么它是凸集呢？就是因为对于凸集，要求θ₁ + θ₂ = 1，这样就保证了对称正定矩阵集合是凸集，但它不再是凸锥了。

我们刚刚讲了n=1的情况。再讲一下n=2的时候是什么样的情况。n=2的时候，情况要稍微复杂一点，但是有个好处：我们可以把矩阵空间变成若干个实数的空间。因为任意一个矩阵，一定都是由矩阵中的这些元素构造出来的。我们来考虑矩阵中的元素。

本节课中我们一起学习了多面体、单纯形以及对称矩阵集合这三种重要的凸集。我们了解了多面体是半空间和超平面的交集，单纯形是特定点集的凸包，而对称半正定矩阵集合则是一个凸锥。通过具体的例子和证明，我们加深了对这些抽象几何概念的理解，并学会了如何运用逻辑而非仅凭想象来分析和证明凸集的性质。
