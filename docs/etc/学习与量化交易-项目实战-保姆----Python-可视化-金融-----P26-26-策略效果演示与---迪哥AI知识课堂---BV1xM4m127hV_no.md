以下是判断和交易逻辑的实现步骤：

第一步，获取当前持仓信息。
```python
# 获取当前持仓字典
current_positions = context.portfolio.positions
```

![](img/5e60c867dba0f00f29314be807296d07_54.png)

![](img/5e60c867dba0f00f29314be807296d07_56.png)

第二步，判断持仓是否为空。如果为空，说明是首次运行，需要买入股票池中的所有股票。如果不为空，则需要根据最新的股票池调整持仓。

以下是具体的逻辑判断和交易操作：

![](img/5e60c867dba0f00f29314be807296d07_58.png)

![](img/5e60c867dba0f00f29314be807296d07_60.png)

1.  如果当前持仓为空，则买入股票池中的所有股票。
    ```python
    if len(current_positions) == 0:
        # 买入逻辑
        for stock in stock_pool:
            order_target_percent(stock, target_percent)
    ```

![](img/5e60c867dba0f00f29314be807296d07_62.png)

![](img/5e60c867dba0f00f29314be807296d07_64.png)

2.  如果当前持仓不为空，则需要遍历现有持仓，卖出不在最新股票池中的股票。
    ```python
    else:
        # 卖出不在股票池中的股票
        for stock in current_positions:
            if stock not in stock_pool:
                order_target_percent(stock, 0)  # 卖出全部
    ```

![](img/5e60c867dba0f00f29314be807296d07_66.png)

3.  然后，遍历最新的股票池，买入当前未持有或需要调整仓位的股票。为了简化，这里采用平均分配资金的方式。
    ```python
        # 买入股票池中的股票（平均分配资金）
        target_percent = 1.0 / len(stock_pool)
        for stock in stock_pool:
            order_target_percent(stock, target_percent)
    ```

现在，我们来解读回测结果中的关键指标。主要关注前四个指标：

![](img/5e60c867dba0f00f29314be807296d07_76.png)

![](img/5e60c867dba0f00f29314be807296d07_78.png)

1.  **回测年化收益**：我们设计的策略所产生的年化收益率。
2.  **基准年化收益**：参照指数（如沪深300）在同期的年化收益率。
3.  **最大回撤**：策略在回测期间资产净值从峰值到谷底的最大跌幅，用于衡量策略的风险。图中显示为17%左右。
4.  **夏普比率**：衡量每承受一单位风险，能获得多少超额收益。如果收益为负，夏普比率通常也为负。

![](img/5e60c867dba0f00f29314be807296d07_80.png)

从结果概览中，我们可以对比策略收益与基准收益。虽然两者可能都为负，但策略的目标是相对基准减少亏损或获得超额收益。
