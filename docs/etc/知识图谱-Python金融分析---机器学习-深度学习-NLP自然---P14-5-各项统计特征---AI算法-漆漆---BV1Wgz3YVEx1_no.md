# 人工智能金融知识图谱与Python量化交易实战：P14：5-各项统计特征

![](img/9a4d8391481a5c3604530ba029693227_1.png)

在本节课中，我们将学习如何为知识图谱中的节点构建和提取多种统计特征。我们将使用DeepWalk算法生成节点嵌入，并处理风险特征与类别特征，为后续的机器学习模型准备数据。

## 节点嵌入特征生成

上一节我们介绍了图结构的基本概念，本节中我们来看看如何利用DeepWalk算法为图中的每个节点生成向量表示。

我们使用DeepWalk算法来构建每个顶点的特征表示。该算法通过在图中进行随机游走来生成节点的序列，然后利用这些序列训练一个词向量模型，从而将每个节点映射为一个固定维度的向量。

以下是生成节点嵌入特征的核心步骤：

![](img/9a4d8391481a5c3604530ba029693227_3.png)

1.  安装并配置DeepWalk环境。
2.  在指定的图数据上运行DeepWalk算法。
3.  将训练得到的节点ID与其对应的向量保存为结构化的数据。

![](img/9a4d8391481a5c3604530ba029693227_5.png)

运行完成后，我们会得到一个类似DataFrame的结构。其中第一列是节点的ID，后续各列则对应该节点向量的各个维度。例如，对于一个128维的向量，我们将得到从第0维到第127维的共128个数值。

这个数据集就是我们为每个顶点构建的特征表示结果。

## 风险特征处理

在获得了节点嵌入特征后，我们还需要处理数据集提供的其他特征。接下来，我们看看如何计算风险特征的统计值。

数据集中提供了一个名为“风险特征”的表格，它列出了每个ID对应的五项风险特征值。由于数据经过了脱敏处理，我们无法获知每个特征的具体业务含义，因此只能进行基础的统计计算。

以下是针对风险特征的计算方法：

*   **风险总和**：计算五项风险特征值的总和。
*   **风险比例**：分别计算每一项风险特征值在总和中所占的比例。

这些计算为我们提供了每个节点在风险维度上的基本统计特性。

## 类别特征处理

处理完风险特征后，我们转向另一种重要的特征类型：类别特征。本节中，我们将学习如何将文本形式的类别信息转化为模型可用的数值特征。

数据集中包含“类别特征”，每个ID对应一个或多个用特定符号（如“/”）分隔的类别标签。我们的目标是将这些文本标签转化为数值向量。

我们借鉴文本预处理中的词袋模型方法。具体流程如下：

1.  对原始类别字符串进行预处理，将复合标签（如“互金公司/b to b”）拆分为独立的标签。
2.  使用`sklearn`库中的`CountVectorizer`工具，在所有节点的类别标签上拟合一个词频统计模型。
3.  使用该模型将每个节点的类别标签转换为一个向量。这个向量的长度等于数据集中所有唯一类别标签的数量，向量中每个位置的值表示对应类别标签在该节点中出现的次数。

通过这种方法，我们得到了每个节点在类别维度上的统计特征。例如，如果数据集中共有44种不同的类别标签，那么每个节点的类别特征就是一个44维的向量。

## 课程总结

本节课中我们一起学习了为金融知识图谱节点构建统计特征的三种主要方法：
1.  使用**DeepWalk算法**生成节点的图结构嵌入向量。
2.  对脱敏的**风险特征**进行求和与比例计算，得到基础统计量。
3.  借鉴文本处理的**词袋模型**，将文本类别的**类别特征**转换为数值型的计数向量。

![](img/9a4d8391481a5c3604530ba029693227_7.png)

这些特征工程步骤为我们后续进行机器学习建模和量化交易分析准备了丰富且结构化的输入数据。
