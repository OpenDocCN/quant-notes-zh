## 绘制双轴折线图

![](img/7ce9804f85d396165885d361870520e6_35.png)

![](img/7ce9804f85d396165885d361870520e6_37.png)

为了观察两个指标随时间的变化关系，最直接的方法是绘制图表。然而，如果将它们画在同一个坐标系中，由于量纲和数值范围不同，趋势可能不清晰。

![](img/7ce9804f85d396165885d361870520e6_39.png)

![](img/7ce9804f85d396165885d361870520e6_41.png)

因此，我们采用双Y轴的方法，将两个序列绘制在同一张图上，但使用不同的纵坐标轴。这样能更直观地对比它们的走势。

![](img/7ce9804f85d396165885d361870520e6_43.png)

![](img/7ce9804f85d396165885d361870520e6_45.png)

![](img/7ce9804f85d396165885d361870520e6_47.png)

以下是绘制双轴折线图的步骤：

1.  使用Pandas的`plot`方法，并指定`secondary_y`参数为第二个列名（如`VIX`）。
2.  为了更清晰地观察短期趋势，我们可以通过日期索引筛选特定时间段的数据（例如2010年至2012年）。
3.  设置图形尺寸以获得更好的视觉效果。

![](img/7ce9804f85d396165885d361870520e6_49.png)

![](img/7ce9804f85d396165885d361870520e6_51.png)

![](img/7ce9804f85d396165885d361870520e6_53.png)

相关代码如下：
```python
# 筛选特定时间段的数据
data_subset = data['2010-01-01':'2012-12-31']
# 绘制双轴折线图
ax = data_subset.plot(secondary_y='VIX', figsize=(10,6))
```
通过观察图形，我们可以发现，当SPX指数下降时，VIX指数往往上升，显示出一种负相关的趋势。图表绘制通常不是一步到位的，需要根据初步结果不断调整参数，以使结论更加清晰。

## 绘制散点图与分布图

![](img/7ce9804f85d396165885d361870520e6_63.png)

除了折线图，散点图是观察两个变量关系的另一种有效方式。我们可以进一步绘制一个散点图矩阵，它不仅展示两两变量之间的关系，还能显示每个变量自身的分布。

以下是绘制散点图矩阵的步骤：

![](img/7ce9804f85d396165885d361870520e6_65.png)

1.  首先，我们计算指标的日收益率（或日变化率）作为分析数据，公式为：`收益率 = log(当日价格 / 前一日价格)`。
2.  使用Pandas的`scatter_matrix`函数来绘制图形。
3.  在函数参数中，我们可以指定：
    *   `alpha`：点的透明度，避免重叠点过多。
    *   对角线图形：可以选择绘制直方图（`hist`）或核密度估计图（`kde`）来展示单个变量的分布。
    *   `figsize`：图形尺寸。

![](img/7ce9804f85d396165885d361870520e6_67.png)

相关代码如下：
```python
import numpy as np
# 计算日收益率
returns = np.log(data / data.shift(1))
# 绘制散点图矩阵，对角线显示核密度估计图
pd.plotting.scatter_matrix(returns, alpha=0.2, figsize=(10, 10), diagonal='kde')
```
在生成的矩阵图中，对角线展示了每个指标自身的分布情况（如SPX收益率的分布），而非对角线的散点图则展示了两个指标之间的关系。从SPX与VIX的散点图中，可以更明显地看到数据点呈现向下倾斜的趋势，这再次印证了它们之间的负相关关系。核密度估计图能提供比直方图更平滑的分布曲线。

## 进行回归分析

![](img/7ce9804f85d396165885d361870520e6_73.png)

散点图可以直观展示关系，而回归分析则能量化这种关系。我们可以在散点图的基础上，添加一条回归线，使得关系模式一目了然。

![](img/7ce9804f85d396165885d361870520e6_75.png)

![](img/7ce9804f85d396165885d361870520e6_77.png)

![](img/7ce9804f85d396165885d361870520e6_79.png)

虽然Pandas的`scatter_matrix`功能强大，但添加回归线需要借助其他库（如Seaborn或Statsmodels）。其核心思想是拟合一个线性模型。

![](img/7ce9804f85d396165885d361870520e6_81.png)

![](img/7ce9804f85d396165885d361870520e6_83.png)

回归方程的基本形式为：
**y = β₀ + β₁x + ε**
其中，`y`是因变量（如VIX收益率），`x`是自变量（如SPX收益率），`β₁`是回归系数（斜率），它量化了`x`对`y`的影响程度和方向（正或负），`β₀`是截距，`ε`是误差项。

![](img/7ce9804f85d396165885d361870520e6_85.png)

![](img/7ce9804f85d396165885d361870520e6_87.png)

通过计算出的回归系数，我们可以明确地说：SPX收益率每变动一个单位，VIX收益率平均会如何变动。这为理解市场指标间的动力学提供了量化依据。

![](img/7ce9804f85d396165885d361870520e6_89.png)

![](img/7ce9804f85d396165885d361870520e6_91.png)
