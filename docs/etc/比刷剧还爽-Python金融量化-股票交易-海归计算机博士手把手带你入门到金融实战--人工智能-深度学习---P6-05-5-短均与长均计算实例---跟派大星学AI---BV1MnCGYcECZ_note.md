# 量化交易全教程：P6：05-5-短均与长均计算实例 📈

在本节课中，我们将学习如何通过计算股票的短期和长期移动平均线，来构建一个简单的分析策略。这个策略的核心是识别“黄金交叉”和“死亡交叉”，它们是判断买入和卖出时机的常见技术指标。

![](img/702df60c22c0cf8c9e8def134dd292a9_1.png)

## 策略概述

![](img/702df60c22c0cf8c9e8def134dd292a9_3.png)

我们将分析一个股票序列，计算其短期移动平均线和长期移动平均线。通过观察这两条线的交叉点，我们可以判断市场的潜在趋势变化。短期均线上穿长期均线被称为“黄金交叉”，通常被视为买入信号；反之，短期均线下穿长期均线被称为“死亡交叉”，通常被视为卖出信号。

![](img/702df60c22c0cf8c9e8def134dd292a9_5.png)

接下来，我们将按照步骤来实现这个策略。

![](img/702df60c22c0cf8c9e8def134dd292a9_7.png)

## 第一步：计算短期与长期移动平均线

![](img/702df60c22c0cf8c9e8def134dd292a9_9.png)

首先，我们需要计算两个关键指标：短期移动平均线和长期移动平均线。移动平均线是过去一段时间内股价的平均值，它能平滑价格波动，帮助我们看清趋势。

在计算之前，我们需要选择具体的数据。这里我们以苹果公司（AAPL）的股票数据为例进行分析。

以下是计算移动平均线的核心代码：

```python
# 假设 df 是包含苹果股价的DataFrame，列名为 ‘AAPL‘
# 计算短期移动平均线（例如10天窗口）
short_window = 10
df[‘M1‘] = df[‘AAPL‘].rolling(window=short_window).mean()

![](img/702df60c22c0cf8c9e8def134dd292a9_11.png)

# 计算长期移动平均线（例如30天窗口）
long_window = 30
df[‘M2‘] = df[‘AAPL‘].rolling(window=long_window).mean()
```

![](img/702df60c22c0cf8c9e8def134dd292a9_13.png)

![](img/702df60c22c0cf8c9e8def134dd292a9_15.png)

**代码解释**：
*   `df[‘AAPL‘].rolling(window=10)` 创建了一个窗口大小为10的滚动窗口对象。
*   `.mean()` 对这个窗口内的数据计算平均值，从而得到移动平均线。
*   短期均线（M1）对价格变化更敏感，长期均线（M2）则更能反映整体趋势。

![](img/702df60c22c0cf8c9e8def134dd292a9_17.png)

正常来说，短期窗口常设为5天（对应一周的交易日），长期窗口常设为20或30天。但由于我们的示例数据时间跨度较长（2010年至2018年），为了在图表中更清晰地展示交叉现象，我们适当增大了窗口值。

计算完成后，我们可以选取股价、短期均线和长期均线这三列数据进行查看。

![](img/702df60c22c0cf8c9e8def134dd292a9_19.png)

## 第二步：绘制均线图进行观察

![](img/702df60c22c0cf8c9e8def134dd292a9_21.png)

计算出指标后，最直观的方法是将其绘制成图表进行观察。上一节我们计算了移动平均线，本节中我们来看看如何将它们可视化。

![](img/702df60c22c0cf8c9e8def134dd292a9_23.png)

![](img/702df60c22c0cf8c9e8def134dd292a9_25.png)

我们将苹果股价、短期均线（M1）和长期均线（M2）绘制在同一张图上。

![](img/702df60c22c0cf8c9e8def134dd292a9_27.png)

```python
# 绘制股价与移动平均线
df[[‘AAPL‘, ‘M1‘, ‘M2‘]].plot(figsize=(12,6))
```

初次绘图可能会因为数据时间跨度太长而导致均线交叉过多，难以辨认。为了解决这个问题，我们可以截取数据中较近的一段时期（例如最后250个交易日，大约一年）重新绘图，这样趋势会更加清晰。

在清晰的图表中，通常：
*   蓝色线代表股票实际价格。
*   绿色线代表短期移动平均线（M1）。
*   红色线代表长期移动平均线（M2）。

现在，我们可以在图上寻找“交叉点”：
1.  **死亡交叉**：当绿色短期均线从上方向下穿过红色长期均线时，形成死亡交叉。这通常预示着下跌趋势的开始，是不适合买入或应考虑卖出的信号。
2.  **黄金交叉**：当绿色短期均线从下方向上穿过红色长期均线时，形成黄金交叉。这通常预示着上涨趋势的开始，是潜在的买入信号。

![](img/702df60c22c0cf8c9e8def134dd292a9_29.png)

![](img/702df60c22c0cf8c9e8def134dd292a9_31.png)

通过图表，我们可以直观地看到这些交叉点，并理解其市场含义。

![](img/702df60c22c0cf8c9e8def134dd292a9_33.png)

## 第三步：量化交叉信号

![](img/702df60c22c0cf8c9e8def134dd292a9_35.png)

虽然看图可以识别交叉，但为了更精确、更程序化地定位这些信号，我们可以创建一个新的指标来量化它。上一节我们通过看图识别交叉，本节我们通过计算来精确标记交叉点。

![](img/702df60c22c0cf8c9e8def134dd292a9_37.png)

![](img/702df60c22c0cf8c9e8def134dd292a9_39.png)

这个新指标将明确显示短期均线何时高于或低于长期均线。

以下是创建信号指标的核心代码：

```python
import numpy as np

![](img/702df60c22c0cf8c9e8def134dd292a9_41.png)

# 生成交易信号：短期均线大于长期均线时为1，小于时为-1
df[‘position‘] = np.where(df[‘M1‘] > df[‘M2‘], 1, -1)
```

![](img/702df60c22c0cf8c9e8def134dd292a9_43.png)

**代码解释**：
*   `np.where(condition, x, y)` 是NumPy的条件赋值函数。
*   `df[‘M1‘] > df[‘M2‘]` 是判断条件，比较短期均线是否大于长期均线。
*   如果条件为真（大于），`position` 列赋值为 `1`，代表“持有”或“看多”信号。
*   如果条件为假（小于），`position` 列赋值为 `-1`，代表“空仓”或“看空”信号。

![](img/702df60c22c0cf8c9e8def134dd292a9_45.png)

这样，`position` 列就像一个开关，其值在 `1` 和 `-1` 之间切换，切换点正好对应了均线的交叉点。

![](img/702df60c22c0cf8c9e8def134dd292a9_47.png)

## 第四步：可视化交易信号

![](img/702df60c22c0cf8c9e8def134dd292a9_49.png)

为了将交易信号与股价走势叠加对比，我们需要在图中增加一个坐标轴来专门显示这个信号序列。

![](img/702df60c22c0cf8c9e8def134dd292a9_51.png)

以下是绘制双Y轴图表的代码：

![](img/702df60c22c0cf8c9e8def134dd292a9_53.png)

```python
import matplotlib.pyplot as plt

![](img/702df60c22c0cf8c9e8def134dd292a9_55.png)

fig, ax1 = plt.subplots(figsize=(12,6))

# 第一个Y轴：绘制股价和均线
ax1.plot(df[‘AAPL‘], label=‘Price‘, alpha=0.5)
ax1.plot(df[‘M1‘], label=‘Short MA‘, alpha=0.7)
ax1.plot(df[‘M2‘], label=‘Long MA‘, alpha=0.7)
ax1.set_ylabel(‘Price‘)

# 第二个Y轴：绘制交易信号
ax2 = ax1.twinx()  # 创建共享X轴的第二个Y轴
ax2.plot(df[‘position‘], label=‘Signal‘, color=‘black‘, linestyle=‘--‘)
ax2.set_ylabel(‘Signal‘)
ax2.set_yticks([-1, 1])

![](img/702df60c22c0cf8c9e8def134dd292a9_57.png)

# 添加图例
ax1.legend(loc=‘upper left‘)
ax2.legend(loc=‘upper right‘)

![](img/702df60c22c0cf8c9e8def134dd292a9_59.png)

![](img/702df60c22c0cf8c9e8def134dd292a9_61.png)

plt.show()
```

在生成的图表中：
*   主图（左Y轴）显示价格和均线走势。
*   信号线（右Y轴）在 `1` 和 `-1` 之间波动。
*   当信号线从 `-1` 跃升到 `1` 时，对应 **黄金交叉**（买入信号）。
*   当信号线从 `1` 下降到 `-1` 时，对应 **死亡交叉**（卖出信号）。

这张图将价格分析与交易信号完美结合，使得策略的触发点一目了然。

![](img/702df60c22c0cf8c9e8def134dd292a9_63.png)
