## 构建并执行因子查询
有了股票池和交易日数据后，我们就可以构建查询语句来获取特定因子（例如市盈率`PE`）的数据。

![](img/e817469fb2fd55932cfa8e8860f73165_15.png)

![](img/e817469fb2fd55932cfa8e8860f73165_17.png)

查询需要用到`fundamentals`模块。以下是一个标准的查询示例，它查询市盈率指标，并过滤出属于我们股票池的股票：

```python
# 导入必要的模块，假设为fundamentals
from some_module import fundamentals

![](img/e817469fb2fd55932cfa8e8860f73165_19.png)

# 构建查询语句
q = query(fundamentals.eod_derivative_indicator.pe_ratio
         ).filter(fundamentals.stockcode.in_(stock_pool))
```

![](img/e817469fb2fd55932cfa8e8860f73165_21.png)

**代码解释**：
*   `query(...)`：构建一个查询。
*   `fundamentals.eod_derivative_indicator.pe_ratio`：指定要查询的因子，这里是市盈率。
*   `.filter(...)`：添加过滤条件。
*   `fundamentals.stockcode.in_(stock_pool)`：过滤条件为股票代码在我们定义的`stock_pool`中。

![](img/e817469fb2fd55932cfa8e8860f73165_23.png)

查询语句构建好后，需要执行它来获取数据。我们需要传入查询对象`q`和具体的查询日期。

![](img/e817469fb2fd55932cfa8e8860f73165_25.png)

我们先尝试查询第一个交易日（`trading_dates[0]`）的数据，以验证操作是否正确：

## 查看与理解数据
获取数据后，我们需要查看其结构以理解如何后续使用。

![](img/e817469fb2fd55932cfa8e8860f73165_33.png)

```python
# 查看数据的形状
print(factor_data.shape)
# 输出可能类似 (1, 1, 3000)，这是一个三维结构

# 查看前几条具体数据
print(factor_data.iloc[0, 0, :5])
```

![](img/e817469fb2fd55932cfa8e8860f73165_35.png)

![](img/e817469fb2fd55932cfa8e8860f73165_37.png)

**数据形状说明**：返回的数据可能是一个三维结构（例如`(1, 1, n)`）。前两个维度通常与查询的字段和条件有关，对于我们单因子、单日查询，它们通常是1。有效数据集中在最后一个维度，其长度`n`等于匹配的股票数量。打印出的前几条数据展示了股票代码与其对应的因子值。

![](img/e817469fb2fd55932cfa8e8860f73165_39.png)

需要注意的是，因为查询时只传入了一个日期（`trading_dates[0]`），所以这批数据只对应那一个交易日（例如2019年1月2日）。

![](img/e817469fb2fd55932cfa8e8860f73165_41.png)
