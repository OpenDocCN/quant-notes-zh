## 获取并存储股票数据

第一个需求是获取某只股票的历史行情数据。

我们可以通过查阅 `tushare` 的官方文档来找到获取历史行情数据的方法。文档中 `get_k_data` 方法可用于此目的。

以下是获取贵州茅台（股票代码：600519）历史数据的代码：

```python
# 获取股票历史数据
df = ts.get_k_data(code='600519', start='2000-01-01')
print(df)
```

代码说明：
*   `code`：传入字符串形式的股票代码。
*   `start`：数据开始时间。这里设置一个较早的日期以确保获取全部历史数据。
*   `end`：数据结束时间。若不指定，默认获取到最近一个交易日的数据。

执行后，我们将得到一个 `DataFrame`，包含日期（`date`）、开盘价（`open`）、收盘价（`close`）、最高价（`high`）、最低价（`low`）、成交量（`volume`）和股票代码（`code`）等列。

为了数据稳定性，我们可以将数据存储到本地：

```python
# 将数据存储到本地CSV文件
df.to_csv('maotai.csv')
```

通过调用 `to_csv` 方法，可以将 `DataFrame` 中的数据写入本地文件。

## 读取与清洗本地数据

接下来，我们将本地存储的数据读入 `DataFrame` 并进行处理。

```python
# 从本地CSV文件读取数据
df = pd.read_csv('maotai.csv')
print(df.head())
```

读取数据后，需要进行以下处理：

1.  **删除无用列**：读取的CSV文件通常会多出一列无意义的索引列，需要删除。
    ```python
    # 删除名为 ‘Unnamed: 0‘ 的列
    df.drop(labels='Unnamed: 0', axis=1, inplace=True)
    print(df.head())
    ```
    *   `labels`：指定要删除的列名。
    *   `axis=1`：表示操作对象是列（在 `drop` 方法中，1代表列，0代表行）。
    *   `inplace=True`：使操作直接修改原 `DataFrame`。

2.  **查看数据类型**：了解各列的数据类型，特别是日期列。
    ```python
    # 查看DataFrame的详细信息，包括每列数据类型和非空值数量
    print(df.info())
    ```
    通过 `info()` 方法，我们发现 `date` 列是对象（`object`，即字符串）类型，而不是时间序列类型。

3.  **转换日期格式**：将 `date` 列转换为时间序列类型，便于后续基于时间的分析。
    ```python
    # 将 ‘date‘ 列转换为 datetime 类型
    df['date'] = pd.to_datetime(df['date'])
    print(df.info())
    ```
    转换后，`date` 列的类型变为 `datetime64[ns]`。

4.  **设置日期为索引**：将时间序列设置为 `DataFrame` 的行索引，这是时间序列分析的常见做法。
    ```python
    # 将 ‘date‘ 列设置为行索引
    df.set_index('date', inplace=True)
    print(df.head())
    ```
    设置后，`date` 列不再是普通列，而是变成了行索引。
