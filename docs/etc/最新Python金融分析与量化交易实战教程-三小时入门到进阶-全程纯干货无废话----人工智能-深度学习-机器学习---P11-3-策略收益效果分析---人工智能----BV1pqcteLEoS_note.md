## 计算基础收益率

上一节我们介绍了双均线策略的信号生成，本节中我们来看看如何计算策略的收益。首先，我们需要计算股票每日的基础收益率。

基础收益率（日收益率）的计算公式是：当日收盘价除以前一日收盘价，然后取自然对数。这样做可以将乘法关系转换为加法关系，便于后续的累加计算。

以下是计算基础收益率的代码步骤：

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_5.png)

```python
# 假设 data 是包含股价的DataFrame，'Close' 是收盘价列
# 计算每日收益率：ln(今日收盘价 / 昨日收盘价)
data['returns'] = np.log(data['Close'] / data['Close'].shift(1))
```

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_7.png)

执行这段代码后，`data` 数据框中会新增一列 `returns`，它代表了股票每日的对数收益率。

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_9.png)

## 计算策略收益率

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_11.png)

接下来，我们将策略信号应用到收益率计算中。我们的策略信号 `position` 列中，1 代表持有（跟随市场），-1 代表反向操作（与市场走势相反）。

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_13.png)

策略收益率的计算逻辑是：用策略信号 `position` 乘以基础收益率 `returns`。当 `position` 为 1 时，策略收益等于市场收益；当 `position` 为 -1 时，策略收益等于市场收益的相反数，即实现了“低买高卖”的反转操作。

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_15.png)

以下是计算策略收益率的代码：

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_17.png)

```python
# 注意：策略信号也需要进行 shift(1) 操作，代表使用前一天的信号来决定今天的操作
data['strategy'] = data['position'].shift(1) * data['returns']
```

执行后，`data` 数据框中会新增一列 `strategy`，它代表了应用策略后的每日对数收益率。

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_19.png)

## 累计收益与结果对比

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_21.png)

在计算出每日的策略收益率后，我们需要将其累加起来，得到总的对数收益率，然后再通过指数函数还原为最终的资金倍数。

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_23.png)

以下是计算并比较最终收益的步骤：

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_25.png)

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_27.png)

1.  **处理缺失值**：由于使用了 `shift` 操作，数据首行会产生缺失值（NaN），需要先将其删除。
2.  **计算累计对数收益**：对 `returns` 列和 `strategy` 列分别求和，得到总的对数收益率。
3.  **还原为资金倍数**：使用指数函数 `np.exp()` 将对数收益率还原。结果表示：期初投入 1 元，期末能变成多少元。

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_29.png)

以下是实现代码：

```python
# 1. 删除含有缺失值的行
data.dropna(inplace=True)

# 2. & 3. 计算并还原累计收益
cumulative_returns = np.exp(data[['returns', 'strategy']].sum())
```

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_31.png)

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_33.png)

执行完上述代码后，`cumulative_returns` 将包含两个数值：
*   `cumulative_returns['returns']`：代表不做任何操作，单纯持有股票到期末的资金倍数。
*   `cumulative_returns['strategy']`：代表应用双均线策略到期末的资金倍数。

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_35.png)

以示例中的数据为例，最终对比结果可能是：
*   原始持有收益：1 元变成了约 4 元。
*   策略收益：1 元变成了约 5.8 元。

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_37.png)

这个结果表明，在此案例中，双均线策略相比简单的买入持有策略，获得了更高的超额收益。

## 总结

![](img/6d5de0a2ddf7537e5053a9507ba2b9b8_39.png)

本节课中我们一起学习了量化策略收益分析的核心流程。我们首先计算了股票的日对数收益率，然后通过将策略信号与收益率相乘，得到了策略的日收益率序列。最后，我们通过求和与指数还原，计算并对比了策略最终收益与基准收益。这个过程清晰地展示了如何定量评估一个交易策略的有效性。
