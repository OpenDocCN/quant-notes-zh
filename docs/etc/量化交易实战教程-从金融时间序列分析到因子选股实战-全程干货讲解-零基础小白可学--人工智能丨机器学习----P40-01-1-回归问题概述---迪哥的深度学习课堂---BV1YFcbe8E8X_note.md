# 深度学习与金融应用：P40：01-1-回归问题概述

在本节课中，我们将要学习机器学习中的核心问题之一——回归问题。我们将从基本概念入手，解释什么是回归，它与分类问题的区别，并通过一个具体的银行贷款额度预测例子，来理解线性回归模型的数据结构、目标以及核心挑战。

## 什么是回归问题？

在机器学习的有监督学习中，主要存在两种类型的问题：分类和回归。

分类问题有明确的类别划分。例如，向银行申请贷款，银行最终给出的结果是“批准”（类别1）或“拒绝”（类别0）。分类算法的输出是离散的类别标签。

回归问题则不同。同样是去银行，但这次询问的是“能贷给我多少钱？”。这个答案是一个具体的数值，例如3万、5万或9万。这个数值通常在一个连续的范围内取值。回归问题的目标就是预测这样一个连续的数值。本节课我们讨论的就是回归问题。

## 数据与模型目标

上一节我们介绍了回归问题的定义，本节中我们来看看用于回归建模的数据通常是什么样子，以及我们的目标是什么。

假设我们有一份数据集，旨在根据用户的“工资”和“年龄”来预测其“贷款额度”。数据可以整理成如下表格形式：

| 工资 (X1) | 年龄 (X2) | 额度 (Y) |
| :--- | :--- | :--- |
| ... | ... | ... |

在这个表格中：
*   **X1（工资）** 和 **X2（年龄）** 被称为**特征**。它们是模型进行预测时所依据的输入变量。
*   **Y（额度）** 被称为**标签**或**目标值**。它是我们希望模型预测出的结果。

作为一个有监督学习算法，线性回归建模的核心就是：利用已知的特征 `X` 和对应的标签 `Y`，学习出一个模型。之后，对于新的、只有特征 `X` 的数据，模型能够预测出其对应的 `Y` 值。

因此，我们的目标很明确：通过 `X1` 和 `X2` 来预测 `Y`。

## 建立回归方程

既然目标是预测 `Y`，而 `Y` 显然与 `X1` 和 `X2` 都相关，一个自然的想法是建立一个方程来描述这种关系。最直接的形式是线性组合：

`Y = θ1 * X1 + θ2 * X2`

这里，`θ1` 和 `θ2` 被称为**参数**或**权重**。它们代表了不同特征对最终结果的影响程度。

为什么不能简单地将 `X1` 和 `X2` 直接相加（即假设 `θ1 = θ2 = 1`）呢？因为不同特征的重要性通常不同。在贷款场景中，“工资”对“额度”的影响很可能远大于“年龄”。`θ1` 和 `θ2` 的数值就量化了这种重要性差异。例如，若 `θ1 = 3`，`θ2 = 1`，则意味着工资的影响力是年龄的3倍。

现在，在我们的数据集中，`X1`、`X2` 和 `Y` 都是已知的。那么，建立回归模型的核心任务就变成了：**寻找一组最优的参数 `θ1` 和 `θ2`**，使得由这个方程计算出的预测值尽可能准确。

## 模型的挑战与目标

![](img/6e202d629729981d0b978b84e9bc5ea3_3.png)

上一节我们明确了要求解参数 `θ1` 和 `θ2`，本节中我们来探讨求解时会遇到什么挑战，以及如何定义“最优”的参数。

我们遇到一个根本性问题：一个简单的线性方程 `Y = θ1*X1 + θ2*X2`，在几何上代表一个平面（或二维空间中的一条直线）。它几乎不可能完美地穿过所有真实的数据点。



既然无法让平面穿过所有点，我们的目标就需要调整为：**找到一个平面，使得所有数据点到这个平面的“距离”总体上最小**。换句话说，这个平面应该尽可能贴近大部分的数据点，即“拟合”得最好。

在下图中，虽然多个平面都是线性的，但显然中间的平面比上下两个更能代表数据的整体趋势。我们的目标就是找到这个“最佳拟合”平面。



因此，“求解最优的 `θ1` 和 `θ2`” 这句话，更精确的定义是：找到能使模型预测值与真实值之间总体误差最小的那组参数。

## 总结

本节课中我们一起学习了回归问题的基本框架：
1.  **回归 vs. 分类**：回归预测连续值，分类预测离散类别。
2.  **数据构成**：回归模型需要特征 `X` 和对应的连续标签 `Y`。
3.  **线性模型**：常用 `Y = θ1*X1 + θ2*X2 + ...` 这样的线性方程来建模。
4.  **核心参数**：权重参数 `θ` 决定了每个特征对预测结果的影响大小。
5.  **模型目标**：由于无法完美拟合所有数据，线性回归的目标是找到一组参数，使模型预测值与真实值之间的总体误差最小化。
