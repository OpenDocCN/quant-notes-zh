# 金融数据分析：P18：序列变化情况分析计算 📈

在本节课中，我们将学习如何分析时间序列数据的变化情况。具体来说，我们将计算两种关键指标：**每日差异值**和**每日变化率**，并通过可视化的方式展示这些指标的平均值。这些分析对于理解金融数据的波动趋势至关重要。

## 计算每日差异值

上一节我们介绍了数据的基本结构，本节中我们来看看如何计算序列中每一天与前一天的差异值。这个指标反映了数据在连续时间点上的绝对变化量。

在Pandas中，计算差异值非常简单，我们可以直接使用`.diff()`方法。该方法默认计算当前行与前一行（即前一天）的差值。

![](img/ac88f079855dab9a7e9ce99ead0b179a_1.png)

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

执行上述代码后，输出结果如下：

![](img/ac88f079855dab9a7e9ce99ead0b179a_1.png)

![](img/ac88f079855dab9a7e9ce99ead0b179a_3.png)

![](img/ac88f079855dab9a7e9ce99ead0b179a_5.png)

观察结果，例如1月5日的值0.052，是由当天的数据减去1月4日的数据得到的。对于序列的第一天（如1月4日），由于没有前一天的数据可减，结果会显示为`NaN`（非数字）。

这样，我们就得到了一个表示“每日与前一日相比的绝对变化量”的指标。这个指标在时间序列分析中非常常用。

`.diff()`方法不仅限于计算相邻两天的差异，我们还可以通过指定`periods`参数来计算间隔多天的差异。例如，`data.diff(periods=7)`可以计算每周的变化。

## 计算每日变化率（增长率）

![](img/ac88f079855dab9a7e9ce99ead0b179a_11.png)

了解了绝对差异值后，我们进一步探讨相对变化。对于金融数据，增长率（或变化率）往往比绝对差值更有意义。例如，股价从1元涨到2元（增长100%）与从10000元涨到10001元（增长0.01%）的“1元”含义截然不同。

变化率的计算公式为：`(后一天的值 - 前一天的值) / 前一天的值`。

在Pandas中，我们可以使用`.pct_change()`方法来直接计算这个百分比变化率。

![](img/ac88f079855dab9a7e9ce99ead0b179a_13.png)

```python
# 计算每日变化率（增长率）
pct_change_data = data.pct_change()
print(pct_change_data.head())
```

执行代码后，结果如下：

![](img/ac88f079855dab9a7e9ce99ead0b179a_17.png)

![](img/ac88f079855dab9a7e9ce99ead0b179a_15.png)

现在，数值表示的是增长率（例如0.0012表示增长0.12%），而非绝对差值。金融资产的日增长率通常就是这样一个较小的百分比数值。

## 可视化平均增长率

![](img/ac88f079855dab9a7e9ce99ead0b179a_21.png)

得到每日增长率序列后，我们可以进一步计算各个指标在整个时间段内的平均增长率，并用图表进行直观展示。

以下是实现步骤：
1.  首先计算增长率的平均值。
2.  然后使用条形图进行可视化，因为平均值是每个指标的一个单一汇总值。

![](img/ac88f079855dab9a7e9ce99ead0b179a_23.png)

![](img/ac88f079855dab9a7e9ce99ead0b179a_25.png)

```python
# 计算各列的平均增长率
mean_pct_change = pct_change_data.mean()

# 使用条形图可视化平均增长率
mean_pct_change.plot(kind='bar', figsize=(16, 8))
```

生成的图表如下：

![](img/ac88f079855dab9a7e9ce99ead0b179a_29.png)

![](img/ac88f079855dab9a7e9ce99ead0b179a_27.png)

![](img/ac88f079855dab9a7e9ce99ead0b179a_29.png)

通过这张条形图，各个指标的平均增长表现一目了然。Y轴的值通常较小，这符合金融数据增长率的特性。图表清晰地展示了不同指标在观察期内的平均变化趋势。
