## 正态分布与极值概念 📈

上一节我们介绍了去极值的概念，本节中我们来看看基于统计理论的3Sigma方法。理解3Sigma方法，首先要理解正态分布。

正态分布，也称为高斯分布，是一种常见的连续概率分布。如果理解起来比较麻烦，可以将其视为一种“正常”的分布。例如，在银行贷款场景中，银行多借给你三块五块，或者少借你三块五块，都是正常情况。但若多借或少借你十万八万，就属于不正常的情况。

在正态分布图中，我们画一条均值线。以标准正态分布为例，其均值为零。这可以类比为银行贷款的误差：多借钱对应均值右侧的点，少借钱对应左侧的点。

现在观察图中的区间。第一个用红色阴影标出的区间，其面积相对较大，代表这些事件发生的可能性较高。当区间越往外扩展时，事件发生的概率就越低。

我们现在要做的是去极值。极值可能落在哪里？它不太可能落在均值附近（红色区域），因为那是大概率发生的正常数据。极值一般落在远离均值的地方，即分布的两侧。离均值越远，偏差越大，越有可能是离群点。

## 3Sigma方法原理 🔍

![](img/f003f2234dc63d84e7d43b040cfe123b_1.png)

上一节我们理解了极值在分布中的位置，本节中我们来具体看看什么是3Sigma方法。

请看下图。我们在分布上标记了均值（X̄），以及均值加减一倍、两倍、三倍标准差（σ）的区间。

![](img/f003f2234dc63d84e7d43b040cfe123b_1.png)

从这个区间可以发现，一旦选择的区间越大（例如加减三倍标准差），数据落在左右极端区域的可能性就越小。

3Sigma方法的原理如下：我们假设可以对原始数据做某种变换，使其理论上服从正态分布（先不讨论如何变换）。当我们指定正负三倍标准差（±3σ）作为边界时，这个区间内包含了多少数据呢？

统计学的结论已经列出：**约99.7%**的数据会落在这个区间内。

因此，如果我们认为数据中99.7%的部分是“正常”的，剩下的0.3%作为极值进行处理和规范，这就是3Sigma方法。在高斯分布中，确定上下限只需要两个值：**均值（μ）**和**标准差（σ）**。计算公式如下：
*   上限：`upper = μ + n * σ`
*   下限：`lower = μ - n * σ`
其中 `n` 通常取3。

完整的函数代码如下：

```python
def three_sigma(data, n=3):
    """
    使用3Sigma方法计算数据的正常值范围。
    参数:
        data: pandas Series，待处理的数据列。
        n: int，标准差的倍数，默认为3。
    返回:
        lower: float，正常值范围下限。
        upper: float，正常值范围上限。
    """
    mean = data.mean()  # 计算均值
    std = data.std()    # 计算标准差
    upper = mean + n * std  # 计算上限
    lower = mean - n * std  # 计算下限
    return lower, upper
```

## 核心要点总结 ✅

![](img/f003f2234dc63d84e7d43b040cfe123b_41.png)

本节课中我们一起学习了3Sigma去极值方法。

![](img/f003f2234dc63d84e7d43b040cfe123b_43.png)

主要要点如下：
*   **原理基础**：该方法基于数据服从正态分布的假设，认为距离均值三倍标准差以外的数据属于小概率事件，可视为极值。
*   **核心计算**：关键在于计算数据的**均值（μ）**和**标准差（σ）**，并通过公式 **`μ ± nσ`** 确定正常值范围的上下限。
*   **方法本质**：去极值不是简单删除异常点，而是定义一个合理的**下界**和**上界**，然后将所有超出此范围的数值规范到边界上，使其保持在预设的合理区间内。
*   **方法对比**：3Sigma法是数据预处理“三步走”（通常指去极值、标准化、中性化）中，完成“去极值”这一步的多种有效策略之一。

![](img/f003f2234dc63d84e7d43b040cfe123b_45.png)
