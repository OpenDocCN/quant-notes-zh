## 模型评估指标

初步观察后，我们需要更系统地评估模型性能。以下是两个核心的评估方向：

![](img/f049d7ed36e9f5073c629c65911f8e57_33.png)

![](img/f049d7ed36e9f5073c629c65911f8e57_35.png)

1.  **误差评估**：计算预测值与真实值之间的差异。常用指标如平均绝对误差（MAE）。
    *   **公式**：`MAE = (1/n) * Σ|y_true - y_pred|`
    *   其中，`y_true`是真实值，`y_pred`是预测值，`n`是数据点数量。
2.  **趋势预测准确率**：评估模型是否能正确预测股价的“上涨”或“下跌”方向。这比单纯的数值误差对交易策略更具指导意义。

![](img/f049d7ed36e9f5073c629c65911f8e57_37.png)

为了进行严谨的评估，我们将数据划分为训练集和测试集。例如，使用2014年至2017年的数据作为训练集，2017年至2018年的数据作为测试集。

以下是计算趋势预测准确率的关键代码步骤：

```python
# 计算真实值和预测值每日的变化方向（1为上涨，0为下跌）
true_diff = np.sign(np.diff(y_true)).astype(int)  # y_true为测试集真实值
pred_diff = np.sign(np.diff(y_pred)).astype(int)  # y_pred为测试集预测值

# 计算上涨趋势的预测准确率
up_accuracy = np.mean(true_diff[true_diff == 1] == pred_diff[true_diff == 1])
# 计算下跌趋势的预测准确率
down_accuracy = np.mean(true_diff[true_diff == 0] == pred_diff[true_diff == 0])
```

## 评估结果分析

![](img/f049d7ed36e9f5073c629c65911f8e57_43.png)

运行评估代码后，我们得到了针对亚马逊股价预测模型的具体指标：

![](img/f049d7ed36e9f5073c629c65911f8e57_45.png)

*   **训练集平均误差（MAE）**：约18美元。
*   **测试集平均误差（MAE）**：约160美元。测试集误差远大于训练集误差，表明模型存在过拟合风险，泛化能力有待提升。
*   **上涨趋势预测准确率**：约56%。
*   **下跌趋势预测准确率**：约44%。

这些准确率接近50%，几乎等同于随机猜测，说明当前的基础模型在捕捉股价变化方向上效果不佳。
