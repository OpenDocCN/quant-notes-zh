## 概述

因子是影响最终结果（如股票收益）的指标。例如，市净率、营收增长率等都可以作为选股的因子。在利用因子进行建模或策略制定前，我们需要对原始的因子数据进行预处理，以确保数据的质量和有效性。预处理通常包括三个步骤：去极值、标准化和中性化。本节课我们将详细讲解第一步——去极值。

## 因子数据预处理三步走

上一节我们介绍了因子数据预处理的重要性，本节中我们来看看具体的三个步骤。

以下是因子数据预处理的三个核心步骤：

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_1.png)

1.  **去极值**：处理数据中的异常值或离群点。
2.  **标准化**：将不同取值范围的数据调整到同一尺度。
3.  **中性化**：在量化金融中，消除因子数据中与某些特定风险（如行业、市值）相关的系统性影响。

本节课，我们将按照这个顺序，首先聚焦于如何对数据进行去极值处理。

## 为什么需要去极值处理

在数据中，绝大多数数据点通常集中在某个范围内，但少数数据点可能远离主体数据分布，这些点被称为极值或离群点。直接使用包含极值的数据进行建模，可能会扭曲模型的判断，例如使均值失去代表性。因此，我们需要处理这些极值。

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_3.png)

一种直接的想法是删除这些极值点，但这可能会损失信息。更常见的做法是“修正”这些极值，将它们“拉回”到我们设定的合理边界内，而不是直接丢弃。这就是去极值方法的核心思想。

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_5.png)

## 基于分位数的去极值方法

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_7.png)

有多种方法可以实现去极值，本节介绍基于分位数的方法。分位数是统计学中的概念，用于描述数据分布的位置。

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_9.png)

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_11.png)

以下是几个关键的分位数概念：

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_13.png)

*   **中位数 (Q2)**：将数据从小到大排列后，处于正中间位置的值。它比均值更能抵抗极值的影响。
*   **下四分位数 (Q1)**：数据中所有数值由小到大排列后，处于25%位置的值。
*   **上四分位数 (Q3)**：数据中所有数值由小到大排列后，处于75%位置的值。

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_15.png)

基于这些分位数，我们可以计算出一个称为“四分位距”的指标：

**IQR = Q3 - Q1**

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_17.png)

四分位距反映了中间50%数据的波动范围。我们通常认为，在 `[Q1 - k * IQR, Q3 + k * IQR]` 这个区间内的数据是“正常”的，其中 `k` 是一个常数（常取1.5）。落在这个区间之外的数据点则被视为极值。

去极值的操作是：将所有小于下限 `(Q1 - k * IQR)` 的值设置为该下限，将所有大于上限 `(Q3 + k * IQR)` 的值设置为该上限。这样，极值就被“拉回”到边界上，而没有被删除。

## 代码示例

下面我们通过一个简单的例子，演示如何使用Python实现基于分位数的去极值方法。本例使用苹果公司股价数据作为演示数据。

![](img/bc3bc0f1c551b1cd7ab05890cbc5f44c_19.png)

```python
# 导入必要的工具包
import pandas as pd
import numpy as np

# 读取示例数据（这里以苹果股价收盘价为例）
# 假设我们有一个包含‘close’价格列的DataFrame ‘df’
# df = pd.read_csv(‘your_data.csv‘)
# 为演示，我们创建一个简单的示例数据
np.random.seed(42)
sample_data = np.random.normal(100, 15, 100) # 生成100个正态分布数据，均值100，标准差15
# 手动添加几个极值
sample_data[0] = 200
sample_data[1] = 50
df = pd.DataFrame({‘close‘: sample_data})

# 定义基于分位数的去极值函数
def winsorize_by_quantile(series, k=1.5):
    """
    使用分位数法对序列进行去极值处理。
    参数:
        series: pandas Series, 待处理的数据序列。
        k: float, 用于计算边界的乘数，默认为1.5。
    返回:
        处理后的pandas Series。
    """
    q1 = series.quantile(0.25)
    q3 = series.quantile(0.75)
    iqr = q3 - q1
    lower_bound = q1 - k * iqr
    upper_bound = q3 + k * iqr
    # 将小于下限的值设为下限，大于上限的值设为上限
    return series.clip(lower=lower_bound, upper=upper_bound)

# 应用去极值函数
df[‘close_winsorized‘] = winsorize_by_quantile(df[‘close‘])

# 查看处理前后的基本统计信息和极值情况
print(“处理前数据描述：“)
print(df[‘close‘].describe())
print(“\n处理后数据描述：“)
print(df[‘close_winsorized‘].describe())
print(“\n处理前极值（>Q3+1.5IQR 或 <Q1-1.5IQR）：“)
q1, q3 = df[‘close‘].quantile(0.25), df[‘close‘].quantile(0.75)
iqr = q3 - q1
outliers = df[(df[‘close‘] > q3 + 1.5*iqr) | (df[‘close‘] < q1 - 1.5*iqr)]
print(outliers[‘close‘])
print(“\n处理后，这些极值被修正为：“)
print(df.loc[outliers.index, ‘close_winsorized‘])
```

运行以上代码，你可以观察到原本的极值（如200和50）被修正到了基于分位数计算出的合理边界附近，而大部分正常数据保持不变。
