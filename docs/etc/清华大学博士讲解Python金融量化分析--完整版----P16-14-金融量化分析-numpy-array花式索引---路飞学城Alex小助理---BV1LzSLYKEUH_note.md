## 什么是花式索引？ 🤔

花式索引用于从数组中选取没有规律的元素。例如，对于一个数组，需要选取其第1、3、4、6、7号元素组成一个新的数组。这些索引位置没有规律。

![](img/0e75eaa8af3a6f0a76a35d438321e939_5.png)

以下是花式索引的基本语法：
```python
import numpy as np
a = np.arange(20)
selected = a[[1, 3, 4, 6, 7]]
```
在这个例子中，`[1, 3, 4, 6, 7]` 是一个索引列表，它告诉NumPy选取数组中这些特定位置的元素。

## 二维数组的花式索引 📊

![](img/0e75eaa8af3a6f0a76a35d438321e939_7.png)

对于二维数组，花式索引的规则与一维数组类似。NumPy提供了多种索引方式：普通索引、切片、布尔型索引和花式索引。在二维数组中，可以通过逗号分隔行和列的索引方式。

以下是几种索引方式混合使用的例子：
```python
arr = np.arange(20).reshape(4, 5)
# 选取第0行，第2到第4列（不包含第4列）
result1 = arr[0, 2:4]
# 选取第0行中所有大于2的元素
result2 = arr[0, arr[0] > 2]
```

## 花式索引的注意事项 ⚠️

需要注意的是，花式索引不能同时出现在逗号的两侧。如果尝试在行和列上都使用花式索引，结果可能不符合直觉。

例如，以下代码并不会选取第1、3行与第1、3列交叉的四个元素：
```python
# 这并不会选取 (1,1), (1,3), (3,1), (3,3) 这四个位置
wrong_result = arr[[1, 3], [1, 3]]
```
实际上，上述代码会选取 `(1,1)` 和 `(3,3)` 这两个位置的值，即 `6` 和 `18`。

## 如何正确选取行列子集？ ✅

如果想正确选取特定的行和列（例如第1、3行与第1、3列），需要分步操作。

以下是分步选取的方法：
1.  首先使用花式索引选取特定的行，并选取所有列。
2.  然后在结果上，选取所有行和特定的列。

具体代码如下：
```python
# 第一步：选取第1、3行，所有列
step1 = arr[[1, 3], :]
# 第二步：在step1的结果上，选取所有行，第1、3列
final_result = step1[:, [1, 3]]
```
这种方法可以确保我们得到期望的行列交叉子集。
