在本节课中，我们将学习如何利用Python和Pandas库，从股票数据中筛选出开盘价较前一日收盘价跌幅超过2%的日期。这是量化分析中一个常见的数据筛选需求。

上一节我们介绍了如何计算股票价格的涨跌幅，本节中我们来看看如何根据更复杂的条件（如与前一日数据对比）来筛选特定日期。

## 需求分析与伪代码

![](img/821d9f8249f10c2297e25258342a13bd_7.png)

需求是：找出该支股票所有“开盘价比前一日收盘价跌幅超过2%”的日期。

![](img/821d9f8249f10c2297e25258342a13bd_9.png)

首先，我们需要将需求转化为逻辑步骤，并用伪代码描述。

以下是核心计算逻辑的伪代码：
1.  计算跌幅：`(当日开盘价 - 前一日收盘价) / 前一日收盘价`
2.  判断条件：计算结果 `小于` `-0.02`

这里需要注意：`-0.03`（跌幅3%）比 `-0.02`（跌幅2%）更小。因此，要找到“跌幅超过2%”的日期，判断条件应为“计算结果 **小于** `-0.02`”。

![](img/821d9f8249f10c2297e25258342a13bd_11.png)

![](img/821d9f8249f10c2297e25258342a13bd_13.png)

## 代码实现与分步讲解

![](img/821d9f8249f10c2297e25258342a13bd_15.png)

接下来，我们将伪代码转化为实际的Python代码。我们将使用Pandas的`shift()`方法来获取“前一日收盘价”。

![](img/821d9f8249f10c2297e25258342a13bd_17.png)

以下是实现此需求的关键步骤：

![](img/821d9f8249f10c2297e25258342a13bd_19.png)

![](img/821d9f8249f10c2297e25258342a13bd_21.png)

1.  **获取前一日收盘价**：使用`df[‘close’].shift(1)`可以将收盘价序列整体向下移动一行，这样每一行的数据就变成了前一日的收盘价。
2.  **计算涨跌幅**：按照公式 `(df[‘open’] - df[‘close’].shift(1)) / df[‘close’].shift(1)` 进行计算。
3.  **应用筛选条件**：将计算结果与 `-0.02` 比较，得到一个布尔序列（True/False）。
4.  **提取日期**：将布尔序列作为原数据框的行索引，筛选出满足条件的行，然后提取这些行的索引（即日期）。

![](img/821d9f8249f10c2297e25258342a13bd_23.png)

```python
# 一行代码实现核心逻辑
result_dates = df.loc[(df[‘open’] - df[‘close’].shift(1)) / df[‘close’].shift(1) < -0.02].index
```

![](img/821d9f8249f10c2297e25258342a13bd_25.png)

![](img/821d9f8249f10c2297e25258342a13bd_27.png)

让我们分解这行代码：
*   `df[‘close’].shift(1)`：生成“前一日收盘价”序列。
*   `(df[‘open’] - df[‘close’].shift(1)) / df[‘close’].shift(1)`：计算当日开盘相对于前日收盘的涨跌幅。
*   `… < -0.02`：生成一个布尔掩码，标记跌幅超过2%的位置为True。
*   `df.loc[… ]`：使用布尔掩码从`df`中选取出满足条件的行。
*   `.index`：最后，从筛选出的行中提取日期索引。
