## 预处理三步走 🚶‍♂️

因子数据预处理主要包含三个核心步骤：去极值、标准化和中性化。以下是每一步的简要说明：

![](img/d7912d66164b6879b363cb899b624bf1_28.png)

![](img/d7912d66164b6879b363cb899b624bf1_30.png)

![](img/d7912d66164b6879b363cb899b624bf1_32.png)

1.  **去极值**：移除数据中的异常值（离群点），防止极端值对整体分析产生过大影响。
2.  **标准化**：将数据转换到均值为0、标准差为1的标准正态分布，使不同量纲或量级的指标可以进行比较和加权。
3.  **中性化**：消除因子数据中与其他已知变量（如市值）的相关性，提取因子本身的“纯净”信息。

## 第一步：去极值（三西格玛法）🔍

![](img/d7912d66164b6879b363cb899b624bf1_46.png)

![](img/d7912d66164b6879b363cb899b624bf1_48.png)

去极值的目的是处理数据中的异常点。我们采用常用的“三西格玛法”，该方法基于正态分布假设，将超出均值三倍标准差范围的值视为异常值并进行处理。

```python
def filter_three_sigma(series, n=3):
    """
    使用三西格玛法去除极值。
    参数:
        series: 待处理的因子数据序列（pandas Series）。
        n: 标准差倍数，默认为3。
    返回:
        处理后的数据序列。
    """
    # 计算均值和标准差
    mean = series.mean()
    std = series.std()
    
    # 计算上限和下限
    high = mean + n * std
    low = mean - n * std
    
    # 将超出范围的值截断到边界
    series = series.clip(low, high)
    
    return series
```

## 第二步：标准化（Z-Score）📏

![](img/d7912d66164b6879b363cb899b624bf1_76.png)

标准化是为了消除数据量纲的影响，使得不同因子之间具有可比性。标准化的公式是：`(原始值 - 均值) / 标准差`。

```python
def standardize(series):
    """
    对数据进行标准化（Z-Score标准化）。
    参数:
        series: 待标准化的数据序列（pandas Series）。
    返回:
        标准化后的数据序列（均值为0，标准差为1）。
    """
    # 计算均值和标准差
    mean = series.mean()
    std = series.std()
    
    # 应用标准化公式
    series_standardized = (series - mean) / std
    
    return series_standardized
```

## 第三步：中性化（回归法）⚖️

![](img/d7912d66164b6879b363cb899b624bf1_104.png)

![](img/d7912d66164b6879b363cb899b624bf1_106.png)

![](img/d7912d66164b6879b363cb899b624bf1_108.png)

中性化的目标是消除因子与其他已知变量（如市值）的系统性关联。例如，市净率因子可能本身就与市值大小相关。我们通过线性回归来剥离这种影响。

![](img/d7912d66164b6879b363cb899b624bf1_110.png)

![](img/d7912d66164b6879b363cb899b624bf1_112.png)

![](img/d7912d66164b6879b363cb899b624bf1_114.png)

其核心思想是：将待处理的因子作为因变量`Y`，将需要剥离的变量（如市值）作为自变量`X`，进行线性回归 `Y = W * X + b`。回归后得到的残差（真实值`Y`与回归预测值`Ŷ`的差），即为中性化后的因子值，它不再包含`X`的影响。

![](img/d7912d66164b6879b363cb899b624bf1_116.png)

![](img/d7912d66164b6879b363cb899b624bf1_118.png)

![](img/d7912d66164b6879b363cb899b624bf1_120.png)

![](img/d7912d66164b6879b363cb899b624bf1_122.png)

![](img/d7912d66164b6879b363cb899b624bf1_124.png)

```python
import statsmodels.api as sm

![](img/d7912d66164b6879b363cb899b624bf1_126.png)

![](img/d7912d66164b6879b363cb899b624bf1_128.png)

![](img/d7912d66164b6879b363cb899b624bf1_130.png)

def neutralize(factor_series, market_cap_series):
    """
    对因子进行市值中性化处理。
    参数:
        factor_series: 待中性化的因子数据（如市净率）。
        market_cap_series: 需要剥离的变量数据（如市值）。
    返回:
        中性化后的因子数据序列（残差序列）。
    """
    # 为自变量添加常数项（截距b）
    X = sm.add_constant(market_cap_series.astype(float))
    Y = factor_series.astype(float)
    
    # 使用普通最小二乘法进行线性回归
    model = sm.OLS(Y, X).fit()
    
    # 返回残差，即中性化后的因子值
    neutralized_factor = model.resid
    return neutralized_factor
```

在这段代码中：
1.  我们使用`statsmodels`库的`OLS`（普通最小二乘法）进行线性回归。
2.  `factor_series`是因变量`Y`，代表需要处理的因子。
3.  `market_cap_series`是自变量`X`，代表我们希望剥离的影响因素（这里是市值）。
4.  回归模型拟合后，`model.resid`属性直接给出了残差序列，这就是去除了市值影响后的“纯净”因子值。

3.  **中性化**：借助`statsmodels`库进行线性回归分析，通过提取残差来消除因子与特定变量（如市值）的相关性。
