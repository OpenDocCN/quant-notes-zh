在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分法。我们将了解其核心思想、实施步骤以及背后的逻辑，为后续的代码实现打下基础。

## 策略核心思想

上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看如何综合多个因子进行更全面的选股。

因子打分选股策略的核心思想是：**为每只股票的多个因子分别打分，然后计算总分，最后根据总分排名来筛选股票**。这类似于学生根据各科成绩的总分进行排名。

假设我们有300只股票（`stock_1`, `stock_2`, ..., `stock_300`）和四个因子（`factor_A`, `factor_B`, `factor_C`, `factor_D`）。我们的目标是每月从这300只股票中选出总分最高的前10只进行调仓。

这个过程可以抽象为以下步骤：
1.  获取所有股票在四个因子上的原始数据。
2.  根据每个因子的特性（越大越好或越小越好），将原始数据转换为标准化分数。
3.  对每只股票，将其在所有因子上的分数求和，得到**总分**。
4.  根据总分对所有股票进行**降序排名**。
5.  选取排名前N（例如10）的股票作为投资组合。

## 关键前提条件

在实施打分之前，我们必须明确一个关键的先验知识：**每个因子与预期收益率的相关性是正向还是负向的**。

以下是需要预先了解的信息：
*   **正向因子**：因子值越大，预期收益率通常越高。例如：`每股收益(EPS)`、`净资产收益率(ROE)`。
*   **负向因子**：因子值越小，预期收益率通常越高。例如：`市盈率(PE)`、`市净率(PB)`（在价值投资中通常认为越低越好）。

3.  我们自己通过历史数据计算因子IC值或进行收益率分析得到的结果。

## 策略实施步骤概述

以下是实施因子打分法选股策略的主要步骤：

1.  **因子选择与数据准备**：确定用于打分的因子列表，并获取股票在这些因子上的截面数据（同一时间点所有股票的数据）。
2.  **数据预处理**：处理缺失值、异常值，确保数据质量。
3.  **因子方向判断**：为每个因子指定其与收益的相关性方向（`direction = 1` 表示正向，`direction = -1` 表示负向）。
4.  **因子标准化与打分**：将每个因子的原始值转换为可比较的分数。常用方法是**分位数排名**或**Z-Score标准化**。
    *   **分位数排名**：将因子值从小到大排序，并计算其百分位排名。对于正向因子，排名越靠前（值越大）分数越高；对于负向因子，则相反。
    *   **公式示意（分位数排名）**：`score = rank(factor_value) * direction`
5.  **计算综合得分**：将一只股票在所有因子上的得分进行加权或简单求和，得到该股票的综合得分。
    *   **公式**：`total_score_i = sum(weight_j * score_ij)` 其中，`i`代表股票，`j`代表因子，`weight`是因子权重（初期可设为等权重1）。
6.  **股票筛选**：根据综合得分对所有股票进行排序，选择得分最高的前N只股票作为当期投资组合。
7.  **定期调仓**：在设定的周期（如每月初）重复上述步骤，更新投资组合。
