## 字符串的定义与创建

字符串在Python中就像一张股票的名称，例如“贵州茅台”或“工商银行”。它是由一系列字符组成的序列，可以使用单引号 `‘’` 或双引号 `“”` 来创建。

```python
# 使用单引号创建字符串
stock_1 = ‘贵州茅台’
# 使用双引号创建字符串
stock_2 = “工商银行”
```

这两种写法是等价的。我们可以通过 `print` 函数来输出它们。

```python
print(stock_1)
print(stock_2)
```

执行上述代码，会分别输出“贵州茅台”和“工商银行”。

---

## 字符串的切片操作

上一节我们介绍了如何创建字符串，本节中我们来看看如何从字符串中提取部分内容，这称为“切片”。

切片操作允许我们根据索引位置获取字符串的子串。在Python中，索引从0开始计数。

以下是切片操作的一个示例：

```python
stock = ‘贵州茅台’
# 从索引2（第三个字符）开始，截取到字符串末尾
substring = stock[2:]
print(substring)
```

这段代码会输出“茅台”。`[2:]` 表示从索引为2的字符开始，一直截取到字符串的最后一个字符。

---

## 字符串的拼接与重复

字符串支持使用加号 `+` 进行拼接，也支持使用乘号 `*` 进行重复。

以下是字符串拼接的示例：

```python
stock_name = ‘贵州茅台’
stock_code = ‘.SH’
full_code = stock_name + stock_code
print(full_code)
```

这段代码会输出“贵州茅台.SH”，即将两个字符串连接在一起。

以下是字符串重复的示例：

```python
stock = ‘贵州茅台’
repeated_stock = stock * 2
print(repeated_stock)
```

这段代码会输出“贵州茅台贵州茅台”，即将原字符串重复两次。

---

## 转义字符

有时我们需要在字符串中包含一些特殊字符，例如引号本身。这时就需要使用反斜杠 `\` 作为转义字符。

以下是使用转义字符的示例：

```python
# 错误写法：字符串中的引号会提前结束字符串定义
# text = ‘It‘s a stock.’

# 正确写法：使用转义字符
text = ‘It\‘s a stock.’
print(text)
```

![](img/65e6d10551383a592036ae1804433fcb_1.png)

这段代码会正确输出 `It‘s a stock.`。反斜杠 `\` 告诉Python，其后的单引号是字符串内容的一部分，而不是字符串的结束标记。
