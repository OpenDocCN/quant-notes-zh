## 什么是回归策略

回归策略的目标是预测未来。例如，在股票交易中，我们希望预测明天股价是上涨还是下跌。如果预测上涨，就可以买入；如果预测下跌，就可以提前卖出，等价格下跌后再买回。这样就能从中获利。

所谓的回归分析，就是借助我们已有的历史数据来预测未来的指标。我们可以利用今天、前一天、前两天甚至更早的数据。利用的数据特征越多，预测效果可能就越好。我们要做的是在历史数据中寻找走势规律，并预测接下来的走势。这就是回归策略分析。

## 第二步：数据预处理与特征工程

![](img/d2815f8363d461d479c929df75a53bad_10.png)

首先，从数据中提取我们需要的那一列数据。之前我们提到，为了得到一个具有累加性质的指标，我们计算了对数回报率。这里我们继续使用这种方法，对数据进行基本的预处理操作，包括处理缺失值。这些操作与之前的方法完全相同。

![](img/d2815f8363d461d479c929df75a53bad_10.png)

接下来，我们在数据中添加一个新的列。观察回报率数据，它有时为正，有时为负。在实际交易中，我们有时只关心价格变动的方向，即到底是上涨还是下跌，而不关心具体的涨跌幅度。

![](img/d2815f8363d461d479c929df75a53bad_12.png)

![](img/d2815f8363d461d479c929df75a53bad_12.png)

因此，我们创建一个名为`direction`的列，用来表示走势是向上还是向下。这个指标的制作方法非常简单。

![](img/d2815f8363d461d479c929df75a53bad_14.png)

![](img/d2815f8363d461d479c929df75a53bad_14.png)

在`returns`数据中，我们直接取其符号：如果是负数，则记为-1；如果是正数，则记为1。然后将其转换为整数类型。

**代码示例：**
```python
data['direction'] = np.sign(data['returns']).astype(int)
```

![](img/d2815f8363d461d479c929df75a53bad_16.png)

这样，我们就为数据引入了一个新列，用于表示价格的走势方向。

## 第三步：构建回归任务

![](img/d2815f8363d461d479c929df75a53bad_22.png)

现在，我们进入本节课的重点：如何进行回归分析。

![](img/d2815f8363d461d479c929df75a53bad_24.png)

![](img/d2815f8363d461d479c929df75a53bad_22.png)

首先明确我们要完成的任务。回归是一个机器学习任务，它基于历史指标（特征）来预测未来的值。这是一个**有监督学习**任务，意味着我们需要有特征`X`和标签`Y`。`X`是输入指标，`Y`是我们想要预测的答案。

![](img/d2815f8363d461d479c929df75a53bad_26.png)

在当前的回归任务中，我们需要找到`X`和`Y`。其中，`Y`的选择相对简单。

![](img/d2815f8363d461d479c929df75a53bad_24.png)

我们可以有两种选择来定义`Y`：
1.  预测具体的涨跌值，即把`returns`列作为目标值`Y1`。
2.  只预测涨跌方向，即把`direction`列作为目标值`Y2`。

![](img/d2815f8363d461d479c929df75a53bad_28.png)

![](img/d2815f8363d461d479c929df75a53bad_26.png)

这样，我们就可以进行两个对比实验，分别使用`Y1`和`Y2`作为预测目标，来观察不同预测目标的效果。

![](img/d2815f8363d461d479c929df75a53bad_28.png)
