## 计算每日收益率

上一节我们介绍了如何生成交易信号，本节中我们来看看如何根据这些信号计算策略的收益。首先，我们需要计算股价的每日收益率。

计算每日增长率的公式是：`当日收盘价 / 前一日收盘价`。在代码中，我们使用 `shift(1)` 方法来获取前一日的数据。

```python
# 假设 df[‘Close’] 是收盘价序列
returns = df[‘Close’] / df[‘Close’].shift(1)
```

![](img/8f703e25d41bac67d09f4b122432f6bc_5.png)

![](img/8f703e25d41bac67d09f4b122432f6bc_7.png)

执行上述代码后，我们得到了一个代表每日增长率的 `returns` 序列。

![](img/8f703e25d41bac67d09f4b122432f6bc_9.png)

## 结合策略信号计算策略收益率

仅仅有市场本身的收益率还不够，我们需要将之前生成的交易信号（`position`）结合起来。`position` 序列中的值为 1 或 -1。

![](img/8f703e25d41bac67d09f4b122432f6bc_11.png)

![](img/8f703e25d41bac67d09f4b122432f6bc_13.png)

*   当 `position` 为 1 时，代表我们遵循原趋势（买入并持有）。
*   当 `position` 为 -1 时，代表我们采取与原趋势相反的操作（例如，在股价下跌时做空，在股价上涨时卖出）。

![](img/8f703e25d41bac67d09f4b122432f6bc_15.png)

因此，策略的每日收益率应该是市场收益率乘以当日的交易信号。同样，我们需要对 `position` 序列进行 `shift(1)` 操作，以确保使用的是前一交易日结束时确定的信号来影响当日的收益。

![](img/8f703e25d41bac67d09f4b122432f6bc_17.png)

```python
strategy_returns = position.shift(1) * returns
```

执行这段代码后，`strategy_returns` 就代表了应用我们策略后的每日收益率。

![](img/8f703e25d41bac67d09f4b122432f6bc_19.png)

![](img/8f703e25d41bac67d09f4b122432f6bc_21.png)

## 从收益率还原为最终资金

![](img/8f703e25d41bac67d09f4b122432f6bc_23.png)

我们得到了每日的收益率，但这是一个相对值序列。要评估策略的最终效果，我们需要知道：如果最初投入1元钱，最终会变成多少钱。

![](img/8f703e25d41bac67d09f4b122432f6bc_25.png)

![](img/8f703e25d41bac67d09f4b122432f6bc_27.png)

这个过程分为两步：
1.  **对数化与累加**：首先对每日收益率取对数（`np.log`），然后将所有对数收益率累加。这等价于计算连续复利下的总增长。
2.  **指数还原**：对累加的结果取指数（`np.exp`），将其还原为最终的资金倍数。

![](img/8f703e25d41bac67d09f4b122432f6bc_29.png)

![](img/8f703e25d41bac67d09f4b122432f6bc_31.png)

以下是计算原始收益率和策略收益率最终结果的代码：

```python
# 计算原始收益的最终资金倍数
final_return = np.exp(np.log(returns).cumsum().iloc[-1])

# 计算策略收益的最终资金倍数。注意处理可能存在的缺失值（NaN）
final_strategy = np.exp(np.log(strategy_returns.fillna(0)).cumsum().iloc[-1])
```

![](img/8f703e25d41bac67d09f4b122432f6bc_33.png)

## 结果分析与比较

![](img/8f703e25d41bac67d09f4b122432f6bc_35.png)

![](img/8f703e25d41bac67d09f4b122432f6bc_37.png)

执行完上述计算后，我们可以比较两个结果。

![](img/8f703e25d41bac67d09f4b122432f6bc_39.png)

例如，假设我们得到：
*   `final_return = 4.0`
*   `final_strategy = 5.8`

![](img/8f703e25d41bac67d09f4b122432f6bc_41.png)

这个结果表明：
*   如果什么都不做（买入并持有），最初的1元钱最终会变成4元钱。
*   如果应用我们的双均线策略，最初的1元钱最终会变成5.8元钱。

![](img/8f703e25d41bac67d09f4b122432f6bc_43.png)

通过对比可以看出，我们的策略在此案例中获得了比简单持有更高的收益。

![](img/8f703e25d41bac67d09f4b122432f6bc_45.png)

![](img/8f703e25d41bac67d09f4b122432f6bc_47.png)

本节课中我们一起学习了量化策略收益分析的核心步骤：从计算每日收益率开始，到结合交易信号得出策略收益率，最后通过累加和对数还原计算出策略的最终资金增长倍数，并与基准（买入持有）进行比较。这是评估一个策略是否有效的关键环节。
