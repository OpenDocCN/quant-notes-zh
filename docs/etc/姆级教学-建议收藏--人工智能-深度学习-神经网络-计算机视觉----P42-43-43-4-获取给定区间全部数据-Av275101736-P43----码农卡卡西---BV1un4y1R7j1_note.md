核心思路是遍历日期序列中的每一天，对每一天都执行一次数据获取操作，然后将每天获取的数据拼接成一个完整的 `DataFrame`。

![](img/b62045ef714be3cb938a1943f794cb84_34.png)

以下是实现这一过程的具体步骤：

![](img/b62045ef714be3cb938a1943f794cb84_40.png)

![](img/b62045ef714be3cb938a1943f794cb84_42.png)

![](img/b62045ef714be3cb938a1943f794cb84_44.png)

![](img/b62045ef714be3cb938a1943f794cb84_46.png)

1.  **确定日期范围**：首先，需要明确要获取数据的时间区间，例如从 `2019-01-01` 到 `2020-01-01`。
2.  **遍历每一天**：使用 `for` 循环遍历这个日期区间内的每一天。
3.  **获取每日数据**：在循环体内，调用数据获取函数，传入当前循环到的日期索引 `i`，以获取当天的因子值。
4.  **处理返回数据**：数据获取函数通常返回一个三维数组。我们只需要其中包含实际数值的部分，通常通过索引 `[0, 2, :]` 来提取。
5.  **构建每日数据表**：将提取出的数值和对应的日期，构建成一个包含两列（日期和因子值）的 `DataFrame`。
6.  **拼接总表**：初始化一个空的 `DataFrame` 作为总表。在每次循环中，将新构建的每日数据表拼接到这个总表中。

```python
import pandas as pd

![](img/b62045ef714be3cb938a1943f794cb84_56.png)

# 假设 dates 是日期序列，get_daily_factor(i) 是获取第 i 天数据的函数
total_df = pd.DataFrame() # 初始化总表

![](img/b62045ef714be3cb938a1943f794cb84_62.png)

![](img/b62045ef714be3cb938a1943f794cb84_64.png)

![](img/b62045ef714be3cb938a1943f794cb84_66.png)

![](img/b62045ef714be3cb938a1943f794cb84_68.png)

for i in range(len(dates)):
    # 获取第 i 天的因子数据（三维数组）
    daily_data = get_daily_factor(i)
    
    # 提取有用的数值部分（假设在 [0,2,:] 位置）
    factor_values = daily_data[0, 2, :]
    
    # 构建当天的数据表
    daily_df = pd.DataFrame({
        'date': dates[i], # 日期列
        'factor_value': factor_values # 因子值列
    })
    
    # 将当天的数据表拼接到总表中，注意需要重新赋值给 total_df
    total_df = pd.concat([total_df, daily_df], ignore_index=True)
```
