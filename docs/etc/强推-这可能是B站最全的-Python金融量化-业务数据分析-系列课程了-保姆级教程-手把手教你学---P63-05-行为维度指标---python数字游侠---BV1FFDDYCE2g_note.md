在本节课中，我们将学习数据分析中几个核心的行为维度指标。这些指标是衡量用户与网站或应用互动情况的基础，对于理解用户行为、评估产品表现至关重要。

![](img/50e543df1e2a2b001f99909197659987_3.png)

上一节我们介绍了数据的基本概念，本节中我们来看看如何通过具体指标量化用户行为。

![](img/50e543df1e2a2b001f99909197659987_7.png)

## 点击量 (PV)

![](img/50e543df1e2a2b001f99909197659987_9.png)

点击量，也称为页面浏览量。其核心计算逻辑是：每当用户的前端页面向后端服务器发送一次请求，就记录为一次点击。

**核心概念**：`一次前端请求 = 一次点击量`

![](img/50e543df1e2a2b001f99909197659987_11.png)

![](img/50e543df1e2a2b001f99909197659987_13.png)

例如，当你打开一个网站首页，这算一次点击。随后你点击一个文章标题进入详情页，这又算一次点击。即使你只是刷新当前页面，也会因为产生了新的请求而被计入一次点击。

![](img/50e543df1e2a2b001f99909197659987_15.png)

## 独立访客数 (UV)

![](img/50e543df1e2a2b001f99909197659987_17.png)

独立访客数用于统计一天内访问网站的唯一用户数量。它通常以用户的浏览器Cookie作为唯一标识。

![](img/50e543df1e2a2b001f99909197659987_19.png)

**核心概念**：`同一Cookie在一天内的多次访问 = 一个独立访客`

![](img/50e543df1e2a2b001f99909197659987_21.png)

这意味着，如果同一台电脑（使用同一Cookie）在一天内被不同的人使用并访问网站，在UV统计中只会被计为一个用户。UV关注的是访问设备的唯一性，而非实际使用人数。

## 基于IP的统计

![](img/50e543df1e2a2b001f99909197659987_23.png)

为了尝试解决同一设备多人使用的问题，另一种统计方式是使用用户的IP地址作为唯一标识。

![](img/50e543df1e2a2b001f99909197659987_25.png)

**核心概念**：`同一IP地址在一天内的访问 = 一个用户（统计上）`

![](img/50e543df1e2a2b001f99909197659987_27.png)

然而，这种方法存在更明显的问题。例如，在办公室或学校的局域网中，许多用户会共享同一个公网IP，导致他们被统计为同一个用户。因此，在实际业务中，UV（基于Cookie）是更常用的用户数统计方式。

## 平均访问时长

![](img/50e543df1e2a2b001f99909197659987_29.png)

平均访问时长衡量用户每次访问网站平均停留了多长时间。它反映了网站内容对用户的吸引力。

![](img/50e543df1e2a2b001f99909197659987_31.png)

以下是计算平均访问时长的公式：

![](img/50e543df1e2a2b001f99909197659987_33.png)

**计算公式**：`平均访问时长 = 总访问时长 / 访问次数（点击量）`

![](img/50e543df1e2a2b001f99909197659987_35.png)

总访问时长是指用户从第一次打开网站到最后一次离开的总时间。访问次数通常使用点击量(PV)来计算。

## 跳出率

![](img/50e543df1e2a2b001f99909197659987_37.png)

跳出率是衡量网站内容吸引力的一个重要指标。它特指那些只访问了一个页面就离开网站的访问次数占总访问次数的比例。

**计算公式**：`跳出率 = （只访问一个页面就离开的访问次数）/ 总访问次数`

例如，有100个用户访问了网站，其中30个用户只看了一眼首页就关闭了浏览器，那么跳出率就是30%。高跳出率可能意味着着陆页内容与用户期望不符，或用户体验不佳。

![](img/50e543df1e2a2b001f99909197659987_39.png)

## 转化率

转化率用于衡量网站促使用户完成特定目标行为的效率。这个“目标行为”可以是任何业务期望的动作，例如点击商品详情、加入购物车、提交表单或完成支付。

![](img/50e543df1e2a2b001f99909197659987_41.png)

**计算公式**：`转化率 = 完成目标行为的次数 / 总访问次数`

![](img/50e543df1e2a2b001f99909197659987_43.png)

转化是一个层层递进的过程。例如：
1.  从网站首页到商品详情页的转化率。
2.  从商品详情页到加入购物车的转化率。
3.  从购物车到成功支付的转化率。
分析每一步的转化率，可以帮助我们定位用户流失的关键环节。

## 复购率

复购率衡量的是用户的忠诚度和产品的用户粘性。它指的是在特定时间段内，进行过两次或以上购买行为的用户数，占所有购买用户总数的比例。

**计算公式**：`复购率 = （购买次数 ≥ 2 的用户数）/ 总购买用户数`

高复购率通常意味着用户对产品或服务高度认可，愿意持续消费。例如，电商平台、订阅服务或周期性需求产品（如旅行订票APP）都非常关注这个指标。它比单纯的日活跃用户数更能反映产品的长期价值和用户留存情况。
