在本节课中，我们将学习如何处理Pandas Series中的缺失数据。缺失值是数据分析中常见的问题，正确处理它们对于后续的计算和可视化至关重要。

上一节我们介绍了Series的基本概念和操作，本节中我们来看看如何处理其中的缺失值。

## 什么是缺失值

![](img/eadc9a61771b0a00cd22d01d28145983_3.png)

在Series中可能会出现缺失数据，即`NaN`值。有时我们可以忽略这些缺失值，但在进行进一步运算或生成图表时，它们可能会带来问题，因此需要处理。

## 缺失值处理方法

![](img/eadc9a61771b0a00cd22d01d28145983_5.png)

处理缺失值主要有两种思路：删除缺失值或填充缺失值。

### 方法一：删除缺失值

![](img/eadc9a61771b0a00cd22d01d28145983_7.png)

第一种方法是直接删除包含缺失数据的行。

![](img/eadc9a61771b0a00cd22d01d28145983_9.png)

以下是判断和删除缺失值的相关函数：

*   **`sr.isnull()`**: 此函数返回一个布尔型Series，用于判断每个位置是否为缺失值。是缺失值则返回`True`，否则返回`False`。
    ```python
    sr.isnull()
    ```
*   **`sr.notnull()`**: 此函数是`isnull()`的反向操作。不是缺失值则返回`True`，是缺失值则返回`False`。
    ```python
    sr.notnull()
    ```
*   **`sr.dropna()`**: 此函数直接删除所有包含缺失值的行，返回一个新的Series。
    ```python
    sr.dropna()
    ```

![](img/eadc9a61771b0a00cd22d01d28145983_11.png)

![](img/eadc9a61771b0a00cd22d01d28145983_13.png)

我们可以利用布尔索引`sr[sr.notnull()]`来过滤掉缺失值，其效果与`sr.dropna()`相同。

### 方法二：填充缺失值

![](img/eadc9a61771b0a00cd22d01d28145983_15.png)

第二种方法是为缺失值赋予一个具体的值。

以下是填充缺失值的函数：

*   **`sr.fillna(value)`**: 此函数将Series中所有的`NaN`值替换为指定的`value`。例如，填充为0：
    ```python
    sr.fillna(0)
    ```

**重要提示**：Pandas的数据处理方法（如`dropna`, `fillna`）默认不会修改原始Series，而是返回一个处理后的新Series。如果需要保存结果，必须进行赋值操作，例如：`sr_filled = sr.fillna(0)`。

![](img/eadc9a61771b0a00cd22d01d28145983_17.png)

除了填充固定值，还可以根据实际情况选择更合理的填充方式。例如，有时缺失值可能只是未被记录，为了保持数据的连续性，可以将其填充为该Series的平均值。

```python
# 将缺失值填充为该Series的平均值
sr.fillna(sr.mean())
```

![](img/eadc9a61771b0a00cd22d01d28145983_19.png)

![](img/eadc9a61771b0a00cd22d01d28145983_21.png)

`sr.mean()`在计算平均值时会自动忽略`NaN`值，这为数据处理提供了极大的便利。

## 总结

本节课中我们一起学习了Pandas Series缺失值处理的两种核心方法：
1.  **删除**：使用`isnull()`, `notnull()`进行判断，使用`dropna()`进行删除。
2.  **填充**：使用`fillna()`函数，可以填充为固定值（如0），也可以填充为统计值（如平均值`mean()`）。

![](img/eadc9a61771b0a00cd22d01d28145983_23.png)

正确处理缺失值是数据清洗的关键步骤，能确保后续分析的准确性和有效性。
