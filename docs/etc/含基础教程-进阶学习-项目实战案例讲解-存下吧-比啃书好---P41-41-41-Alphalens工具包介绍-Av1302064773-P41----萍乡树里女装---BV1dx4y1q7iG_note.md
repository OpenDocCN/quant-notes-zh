## 工具包简介与获取

![](img/be45abd5735025372b64134d386b3219_7.png)

Alphalens是一个专门用于金融因子分析的Python工具包。它封装了因子评价的核心计算（如IC值）和结果可视化功能，使我们无需从零开始编写代码。

![](img/be45abd5735025372b64134d386b3219_9.png)

以下是关于Alphalens的基本信息：
*   **官方资源**：其GitHub仓库链接和详细的使用文档是学习和参考的最佳起点。
*   **安装方法**：在本地环境使用`pip install alphalens`命令即可安装。不过，在本课程后续的实战环节中，我们将使用已经预装了该工具包的在线量化平台，因此可以跳过安装步骤。
*   **学习方法**：建议初学者通过阅读官方文档中的示例（Examples）来快速上手。这些示例清晰地展示了工具包的基本用法。

## 在线研究环境配置

![](img/be45abd5735025372b64134d386b3219_13.png)

![](img/be45abd5735025372b64134d386b3219_14.png)

由于因子分析需要获取大量的金融数据，在个人电脑上操作较为麻烦。因此，我们将在一个在线的量化交易平台中进行代码编写和数据分析。

以下是进入在线研究环境的步骤：
1.  登录量化交易平台。
2.  在平台界面左侧找到并点击“投资研究”功能模块。
3.  系统会打开一个类似于Jupyter Notebook的交互式编程环境。
4.  在该环境中新建一个Python 3笔记文件，并为其命名（例如“因子分析”）。

![](img/be45abd5735025372b64134d386b3219_16.png)

![](img/be45abd5735025372b64134d386b3219_17.png)

这个在线环境本质上是在平台的服务器上运行代码，它提供了所需的数据接口和计算资源，我们只需专注于编写分析逻辑即可。后续的演示代码都将在此环境中运行。

![](img/be45abd5735025372b64134d386b3219_19.png)
