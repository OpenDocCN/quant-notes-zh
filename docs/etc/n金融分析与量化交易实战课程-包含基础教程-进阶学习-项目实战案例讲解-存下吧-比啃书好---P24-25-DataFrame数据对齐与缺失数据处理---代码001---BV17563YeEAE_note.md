## 概述
在本节课中，我们将要学习Pandas中`DataFrame`对象的两个重要功能：数据对齐与缺失数据处理。`DataFrame`作为二维数据结构，其数据对齐涉及行索引和列索引两个维度。同时，处理其中的缺失数据（NaN）是数据清洗的关键步骤。我们将通过简单的示例，清晰地讲解这些概念和操作方法。

![](img/366b2c038e11ffc4371834c46f8876a5_1.png)

## 数据对齐
上一节我们介绍了`Series`的数据对齐，本节中我们来看看`DataFrame`的数据对齐。与`Series`一样，`DataFrame`在进行算术运算时也会自动对齐数据。但由于`DataFrame`拥有行索引和列索引两个维度，因此需要在这两个维度上分别进行对齐。

例如，我们创建两个`DataFrame`对象`df1`和`df2`，它们的行索引顺序不同。当我们执行`df1 + df2`时，Pandas会自动根据行索引和列索引进行匹配对齐，然后执行运算。如果某个位置在其中一个`DataFrame`中不存在，或者对应的值是`NaN`，那么运算结果在该位置也会是`NaN`。

**核心概念**：数据对齐遵循索引匹配原则。运算时，系统会寻找两个`DataFrame`中行标签和列标签均相同的单元格进行操作。

![](img/366b2c038e11ffc4371834c46f8876a5_3.png)

```python
# 示例：两个DataFrame相加，自动按索引对齐
result = df1 + df2
```

![](img/366b2c038e11ffc4371834c46f8876a5_5.png)

如果两个`DataFrame`的列顺序不同，系统同样会根据列名进行对齐，而不是简单地按位置相加。

![](img/366b2c038e11ffc4371834c46f8876a5_7.png)

## 缺失数据处理
处理完数据对齐，本节我们聚焦于`DataFrame`中缺失数据的处理。`DataFrame`处理缺失数据的方法与`Series`对象大部分相似，但也存在一些关键区别。

### 填充缺失值
首先是相似的部分：填充缺失值。我们可以使用`fillna()`方法，将所有的`NaN`值替换为指定的值。

```python
# 示例：将DataFrame中所有NaN填充为0
df_filled = df2.fillna(0)
```

### 删除缺失值
不一样的地方主要体现在删除缺失值的`dropna()`方法上。在`DataFrame`中，`dropna()`的默认行为是：**只要某一行中存在任何一个缺失值（NaN），就会删除整行**。

```python
# 默认删除包含任何NaN的行
df_cleaned = df2.dropna()
```

然而，我们有时不希望如此严格。例如，某行有10列数据，仅1列是缺失值，其他9列数据仍有价值。这时，我们可以通过设置参数来改变删除逻辑。

以下是`dropna()`方法的关键参数说明：

*   **`how=‘any’`**：默认值。如果行（或列）中存在**任何**`NaN`，则删除该行（或列）。
*   **`how=‘all’`**：只有当行（或列）中**所有**值都是`NaN`时，才删除该行（或列）。
*   **`axis`**：决定删除操作的轴向。
    *   `axis=0` 或 `axis=‘index’`：默认值。按行操作，删除包含`NaN`的行。
    *   `axis=1` 或 `axis=‘columns’`：按列操作，删除包含`NaN`的列。

例如，如果我们希望只删除那些整行都是空值的行，可以这样操作：

```python
# 只删除全部为NaN的行
df_cleaned = df2.dropna(how=‘all’)
```

如果我们希望以列为单位，删除包含缺失值的列，则需要使用`axis`参数：

![](img/366b2c038e11ffc4371834c46f8876a5_9.png)

```python
# 删除包含任何NaN的列
df_cleaned = df2.dropna(axis=1)
# 或
df_cleaned = df2.dropna(axis=‘columns’)
```

![](img/366b2c038e11ffc4371834c46f8876a5_11.png)

**核心概念**：可以将`DataFrame`想象成一个二维表格，`axis=0`代表沿着垂直方向（行）操作，`axis=1`代表沿着水平方向（列）操作。

![](img/366b2c038e11ffc4371834c46f8876a5_13.png)

## 总结
本节课中我们一起学习了`DataFrame`的数据对齐与缺失数据处理。我们了解到，`DataFrame`的运算会自动根据行和列索引进行对齐。在缺失值处理方面，我们重点掌握了`fillna()`填充方法和`dropna()`删除方法，特别是`dropna()`方法中`how`和`axis`参数的不同用法，它们能帮助我们更灵活地清洗数据。这些操作与之前学习的`Series`相关方法原理一致，是构建稳健数据分析流程的基础。
