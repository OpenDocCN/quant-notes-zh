策略的核心是定义调仓周期。我们创建一个策略类，并在其初始化函数中设置按月调仓的定时器。

![](img/4745ed082599c545090ea79d8773e273_22.png)

```python
def initialize(context):
```

![](img/4745ed082599c545090ea79d8773e273_26.png)

![](img/4745ed082599c545090ea79d8773e273_28.png)

*   **`run_monthly`**：这是一个定时器函数，它会在每个月的指定交易日（这里设为每月1号）执行一次名为`rebalance`的函数。
*   **`rebalance`**：这是我们即将定义的核心调仓函数，负责在每个调仓日执行选股和买卖操作。

## 第三步：构建核心调仓函数

![](img/4745ed082599c545090ea79d8773e273_32.png)

![](img/4745ed082599c545090ea79d8773e273_34.png)

`rebalance`函数是策略的执行主体。以下是该函数需要完成的步骤概述：

![](img/4745ed082599c545090ea79d8773e273_36.png)

![](img/4745ed082599c545090ea79d8773e273_38.png)

1.  获取全市场股票列表。
2.  对股票进行初步筛选，构建股票池。
3.  获取股票的相关因子数据。
4.  对因子进行预处理（去极值、标准化、中性化）。
5.  根据处理后的因子得分，选择排名靠前的股票。
6.  执行买卖操作，将持仓调整为目标股票组合。
