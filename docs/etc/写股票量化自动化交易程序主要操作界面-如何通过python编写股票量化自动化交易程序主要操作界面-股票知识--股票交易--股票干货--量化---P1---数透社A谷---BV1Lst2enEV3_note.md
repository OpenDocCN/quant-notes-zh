# Python股票量化自动化交易程序：1：主要操作界面指南 📈

![](img/f19924dd4ebefe65b49088bc35bb0c13_0.png)

在本教程中，我们将学习一个Python股票量化自动化交易程序（以下简称“程序”）的主要操作界面。我们将了解其核心布局、各个功能模块的作用以及基本操作流程，帮助你快速上手。

## 界面概览与导航 🧭

程序的主界面是进行所有量化交易活动的中心。上一节我们介绍了课程目标，本节中我们来看看程序的主界面布局。

主界面左侧有一个工具栏，包含“量化交易”、“配置”和“交易”三个主要按钮控件。

*   **量化界面**：当前默认显示的主界面，用于策略回测、选股等量化分析功能。
*   **配置界面**：点击“配置”控件进入，用于设置程序的通用参数，例如行情图表的大小、边距等。
*   **交易界面**：点击“交易”控件进入，提供实盘交易相关的功能。这部分内容较多，会有单独的主题详细介绍。

当需要从子界面（如配置或交易界面）返回时，点击上方菜单栏的“主菜单”，选择“返回”，即可切回登录页面或主界面。

## 量化界面功能详解 ⚙️

量化界面是程序的核心，集成了数据管理、策略回测和选股等多种功能。上一节我们了解了整体导航，本节中我们来看看量化页面上各个控件的具体功能。

以下是量化界面主要功能模块的介绍：

*   **离线数据下载**：将最新的个股行情数据下载到本地。包含“开始下载”、“刷新文件”、“补全下载”、“停止下载”四个选项。下载时可以选择更新全市场数据、仅更新自选股票池或特定板块的数据，以缩短下载时间。
*   **股票池管理**：
    *   **导入我的股票池**：从本地文件导入自选股票列表到程序中。
    *   **导出我的股票池**：将程序中的自选股票代码和名称以CSV格式保存到本地。这样可以管理多份股票池，并与其他系统共享自选股列表。
*   **日志查看**：
    *   **系统日志**：显示程序系统级别的运行状态信息。
    *   **选股日志**：显示选股过程中的运行情况。
*   **组合分析**：将需要对比的股票添加到一个池中，进行对比分析。目前支持收益率、波动率等指标的对比。
*   **走势叠加分析**：可以将多只股票的行情走势图叠加在一起进行对比观察。

## 策略回测与择时信号 🧪

策略回测是验证交易想法有效性的关键步骤。上一节我们介绍了数据管理功能，本节中我们来看看如何利用策略进行回测和生成交易信号。

程序在“策略导航”中提供了多种择时策略，例如经典策略中的 **N日突破**、**ATR止盈止损** 和 **布林带突破** 策略。

使用策略进行回测的基本流程如下：
1.  在策略导航中选择一个策略。
2.  点击“回测参数”，设置回测的股票、时间范围等参数。
3.  点击“开始回测”，程序会根据历史数据运行策略并生成分析报告。

除了回测，策略还可用于在行情图上标注实时的择时信号。操作方法是：
1.  点击“实时参数”。
2.  输入股票代码并选择某个策略。
3.  填写行情参数，即可在行情界面上看到相应的买入/卖出信号。

例如，**均线交叉** 策略还支持批量扫描。你可以选择“自选股票池”，点击“开始扫描”后，程序会找出所有符合金叉（买入）或死叉（卖出）条件的股票，并考虑交易滑点，将其加入监控池，为下一个交易日的实盘操作做准备。

程序还支持 **多子图显示** 功能，可以同时显示两幅行情走势图进行对比分析，图表可以是动态或静态的。

## 核心功能：选股流程 🔍

选股是量化交易中至关重要的一环，好的选股能显著提升盈利效率。上一节我们探讨了策略应用，本节中我们聚焦于程序强大的选股功能。

程序主要通过三种模式筛选优质股票：**数据驱动选股**、**形态驱动选股** 和 **RPS排名选股**。

以下是选股功能的详细说明：

*   **数据驱动选股**：支持通过多种指标和数据进行筛选。
    *   **指标类型**：包括北向资金、每日涨停数据、季度基金持仓等。
    *   **自定义数据**：支持导入自定义的离线数据进行选股。
    *   **数据展示**：可以通过矩阵树图等形式显示行业、板块内个股的涨跌幅排行。
    *   **筛选过程**：刷新数据后，使用 **条件表达式** 对股票进行过滤。例如，可以编写公式筛选出 `市盈率 < 20 且 净利润增长率 > 30%` 的股票。
*   **形态驱动选股**：基于K线形态进行筛选。
    *   **支持形态**：包括双底形态、箱体突破、均线多头排列、单针探底回升等。
    *   **回测模式**：如果选择过去的日期进行选股，则进入回测模式，可以验证形态在历史中的有效性。
    *   **叠加分析**：可以在形态选股的结果基础上，叠加其他维度数据（如季度利润报表、资金持股、基本面指标）进行综合过滤。
    *   **股票池选项**：除了自选股和全市场，还可以聚焦于特定的 **概念板块池** 或 **行业板块池**，使选股更精准、快速。
    *   **参数与回测**：每种形态算法都有专用的参数配置界面。选股后，还可以对结果进行回测评估。
*   **RPS排名选股**：用于发现强势股。
    *   **数据准备**：使用前需先更新全市场个股的行情数据。
    *   **参数配置**：点击开始RPS排名选股后，会进入参数配置界面，例如设置观测排名范围（如前10%），以尽早发现启动的强势股。
    *   **结果应用**：生成RPS分析结果后，可通过数据驱动方式导入到程序中。
    *   **个股RPS跟踪**：该功能支持将个股股价与其RPS值进行可视化对比，用于辅助分析强势股的介入时机。设置监测的价量参数后，即可查看对比图表。

## 高级功能与使用技巧 💡

除了上述核心功能，程序还提供了一些高级管理指令和便捷操作。

在行情界面或相关列表中，右键点击股票通常会弹出高级功能菜单。除了添加自选股、加入组合分析池、导入离线数据等管理指令外，还包括：
*   查看个股现金流量表。
*   查看个股行情走势。
*   查看个股F10资料。
*   K线自动播放等。
