# 量化交易与Python金融分析实战：P46：工具包绘图展示

在本节课中，我们将学习如何使用量化分析工具包对计算出的因子IC值进行可视化展示。通过图表，我们可以更直观地评估因子的有效性和稳定性。

上一节我们介绍了如何计算因子与收益率之间的相关系数（IC值）。本节中我们来看看如何将这些数值结果转化为直观的图表。

## 绘制IC值时间序列图

![](img/145e2940d16d5810a3d617d731312075_1.png)

计算得到的IC值每天都会变化。为了更清晰地观察其趋势和稳定性，我们可以将其绘制成时间序列图。这里我们使用一个名为 `plotting` 的工具包来完成可视化。

![](img/145e2940d16d5810a3d617d731312075_3.png)

以下是绘制IC时间序列图的步骤：

1.  导入必要的绘图工具包。
2.  调用 `plot_ic_ts` 函数。
3.  将包含每日IC值的时间序列数据传入该函数。

```python
# 假设 ic_series 是包含每日IC值的时间序列数据
from alphalens import plotting
plotting.plot_ic_ts(ic_series)
```

执行代码后，工具包会自动生成图表。图表中主要包含两条线：
*   **蓝色线**：代表每日的实际IC值，波动通常较大。
*   **绿色线**：代表IC值的月度滚动平均值（即每月的平均值），能更平滑地显示长期趋势。

此外，图表还会标注出整个时间序列IC值的 **均值（Mean）** 和 **信息比率（IR）**。信息比率的计算公式为：
**IR = Mean / Std**
其中，`Std` 是IC值的标准差。该比率用于衡量IC值的稳定性，**比值越大，代表因子的预测效果越稳定**。

通过观察绿色均线，我们可以评估因子的整体有效性。理想情况下，我们希望看到稳定且较高的正IC值。如果均线长期在零值附近徘徊或数值很小，则说明该因子与收益率的相关性较弱，可能不是有效的选股因子。

## 查看其他分析图表

![](img/145e2940d16d5810a3d617d731312075_5.png)

除了时间序列图，工具包还提供了多种其他分析图表，用于从不同角度评估因子。

![](img/145e2940d16d5810a3d617d731312075_7.png)

以下是工具包支持的部分其他图表类型：

*   **直方图 (`plot_ic_histogram`)**：展示IC值的分布情况。
*   **QQ图 (`plot_ic_qq`)**：检验IC值是否符合正态分布。
*   **月度热力图 (`plot_monthly_ic_heatmap`)**：用颜色深浅展示不同月份的IC值表现。

![](img/145e2940d16d5810a3d617d731312075_9.png)

![](img/145e2940d16d5810a3d617d731312075_11.png)

这些函数为深入分析因子特性提供了全面的可视化工具。在实际构建量化策略时，可以灵活选用这些图表进行综合诊断。

![](img/145e2940d16d5810a3d617d731312075_13.png)
