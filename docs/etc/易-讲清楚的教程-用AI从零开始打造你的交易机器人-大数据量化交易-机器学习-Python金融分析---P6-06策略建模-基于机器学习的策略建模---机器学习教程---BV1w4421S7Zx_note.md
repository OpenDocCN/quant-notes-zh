## 特征选择的意义与方法

上一节我们学习了如何从原始数据中构建训练集（X和Y）。本节中，我们来看看如何优化这些输入特征。

原始的特征集（X）可能包含数百甚至上千个维度，这些特征往往是凭经验选择的，可能存在冗余（如某些特征高度相关）、无效（如某一列值全为零）或噪声。特征质量直接决定了模型性能的上限。无论分类器多么先进，如果输入特征质量很差，都难以得到理想的结果。

特征选择的核心目标是从原始特征中筛选出一个优质子集，以提升模型性能（如回归任务中的R方值）。主要有三种思路：

1.  **子集选择**：从原始P个特征中，直接挑选出K个特征（K<P）。选出的特征值本身不变。
2.  **系数收缩/正则化**：不直接剔除特征，而是在模型训练过程中（如使用Lasso回归），让不重要特征的系数自动收缩为零，从而实现特征选择。
3.  **降维**：通过某种函数变换，将原始的P维特征映射到一个全新的低维空间（如Z维， Z<P）。新特征的每一维都是原始特征的线性或非线性组合（例如主成分分析PCA、自编码器Autoencoder），其物理意义已发生改变。

以下是特征子集选择的几种具体方法：

*   **暴力枚举法**：尝试所有可能的特征组合。例如，从300个特征中选1个最好的、选2个最好的组合……以此类推。这种方法能保证找到最优解，但计算复杂度是指数级的（`O(2^P)`），仅适用于特征数很少的情况。
*   **前向选择法**：一种贪心算法。开始时特征集合为空，每次从未入选的特征中选择一个，使得加入后模型性能提升最大，直到性能不再显著提升为止。其复杂度约为 `O(P^2)`。
*   **后向选择法**：与前向选择相反。开始时包含所有特征，每次尝试剔除一个特征，选择对模型性能损害最小的那个剔除，直到性能开始显著下降。

## 遗传算法：受进化论启发的优化器 🧬

在介绍了传统的特征选择方法后，我们来看一种截然不同且功能强大的优化方法——遗传算法。它不仅可用于特征选择，还能解决各类复杂的优化问题。

遗传算法的灵感来源于生物进化论：一个种群中，适应环境的个体存活并繁衍，不适应环境的个体被淘汰。通过数代演化，种群整体适应性会不断提高。

### 核心概念与建模

1.  **基因型与表现型**：
    *   **基因型**：问题的编码表示。在特征选择中，我们可以用一个二进制串表示，例如 `基因型 = [1, 0, 0, 1, 1, 0]`，其中`1`代表选择该特征，`0`代表不选。
    *   **表现型**：基因型对应的实际解。在上述例子中，表现型就是被选中的特征子集。
2.  **适应度函数**：用于评价个体（一个解）好坏的函数。在特征选择中，适应度函数可以是所选特征子集在验证集上得到的R方值。我们的目标是找到使适应度函数最大化的基因型。
3.  **种群初始化**：随机生成一定数量（如1000个）的基因型（二进制串），构成初始种群。

### 算法的关键操作

遗传算法通过模拟自然进化过程来迭代优化种群：

*   **选择**：根据个体的适应度，以一定概率选择优秀的个体进入下一代。一种常见的方法是轮盘赌选择，即个体被选中的概率与其适应度成正比。
    ```python
    # 轮盘赌选择示例 (概念代码)
    import numpy as np
    fitness = np.array([0.8, 0.2, 0.7, 1.3, 0.1]) # 5个个体的适应度
    prob = fitness / fitness.sum() # 计算被选中的概率
    # 例如，prob = [0.26, 0.06, 0.23, 0.42, 0.03]
    selected_indices = np.random.choice(range(5), size=5, p=prob, replace=True)
    ```
*   **交叉**：随机选择两个父代个体，交换它们部分基因片段，产生新的子代个体。这模拟了生物的有性繁殖过程。
*   **变异**：以一个小概率随机改变个体基因型中的某些位（如将0变为1）。这为种群引入了新的基因，有助于跳出局部最优。

### 算法流程与特点

1.  初始化种群。
2.  计算每个个体的适应度。
3.  **循环以下步骤直到满足终止条件（如达到最大迭代次数）**：
    a. **选择**：根据适应度选择父代。
    b. **交叉**：对选出的父代进行交叉操作，产生子代。
    c. **变异**：对子代进行变异操作。
    d. 用新生成的子代种群替代旧种群（或部分替代）。
    e. 重新计算种群中个体的适应度。
4.  输出历代中适应度最高的个体（即最优解）。

**遗传算法的优点**：能够处理非凸、非线性、不可微的复杂优化问题，对问题领域知识要求低，通用性强。
**遗传算法的缺点**：不能保证收敛到全局最优解，参数（如种群大小、交叉变异概率）需要调优，计算开销可能较大。

**实践建议**：对于特征选择这类组合优化问题，遗传算法是一种非常有效的工具。你可以将每个特征子集编码为一个二进制个体，用模型在验证集上的性能作为适应度，通过多代演化来搜索高性能的特征组合。

## 神经网络与反向传播算法：从生物灵感到数学奇迹 🔄

接下来，我们深入探讨机器学习的另一个支柱——神经网络，特别是其核心训练算法：反向传播。

### 从生物神经元到数学模型

生物神经元接收来自其他神经元的信号，加权求和后，如果超过某个阈值则产生兴奋（输出信号），否则抑制。感知机模型对其进行了简化建模：

`加权和 S = Σ (wi * xi) + b` （其中b是偏置，替代了阈值）
`输出 y = f(S)`，其中 `f` 是激活函数。

最初的感知机使用阶跃函数作为`f`，但其不可导，不利于优化。因此，我们改用平滑的Sigmoid函数：
`σ(x) = 1 / (1 + e^(-x))`
其导数有很好的性质：`σ‘(x) = σ(x) * (1 - σ(x))`。

### 神经网络与优化目标

多层神经网络由大量这样的神经元连接而成。我们的目标是找到一组网络参数（权重`W`和偏置`b`），使得网络对于训练样本的预测输出 `ŷ` 尽可能接近真实标签 `y`。

为此，我们定义一个损失函数 `J(W, b)` 来衡量预测误差，例如均方误差：
`J = 1/2 * (y - ŷ)^2`
训练神经网络就是寻找最小化损失函数 `J` 的参数的过程。这本质上是一个优化问题。

### 梯度下降与链式法则

梯度下降法是解决此类优化问题的基本思想：参数沿着损失函数梯度的反方向更新，以逐步减小损失。
`W_new = W_old - α * ∂J/∂W`，其中 `α` 是学习率。

对于简单模型（如线性回归），`∂J/∂W` 很容易直接计算。但对于多层神经网络，损失 `J` 与底层参数 `W` 之间隔着复杂的复合函数关系，直接求导困难。

**反向传播算法的精髓**：它巧妙地运用了**链式法则**，从输出层开始，反向逐层计算损失函数对每一层参数的梯度。

考虑一个极简网络：输入x -> 权重w1 -> Sigmoid -> 权重w2 -> Sigmoid -> 输出z， 真实值为d。
损失 `J = 1/2*(d-z)^2`。

*   计算 `∂J/∂w2`：
    `∂J/∂w2 = (∂J/∂z) * (∂z/∂P2) * (∂P2/∂w2)`
    其中 `P2 = w2 * y`， `y` 是上一层的输出。
    每一项都可求：
    *   `∂J/∂z = -(d - z)`
    *   `∂z/∂P2 = z * (1 - z)` (Sigmoid导数)
    *   `∂P2/∂w2 = y`
*   计算 `∂J/∂w1`：
    `∂J/∂w1 = (∂J/∂z) * (∂z/∂P2) * (∂P2/∂y) * (∂y/∂P1) * (∂P1/∂w1)`
    其中 `P1 = w1 * x`。
    同样，每一项都可基于已知量或上一层梯度计算。

### 反向传播的高效性：动态规划思想

在复杂网络中，一个底层参数会通过多条路径影响最终损失。如果为每条路径独立计算梯度，复杂度会爆炸式增长。

反向传播算法的第二个巧妙之处在于它像**动态规划**一样，**复用已计算的梯度**。
1.  它从输出层开始反向计算。
2.  当计算到某一层时，该层神经元节点处的梯度 `δ` 已经求出。
3.  在计算前一层参数梯度时，直接使用这个 `δ`，而不需要重新追溯所有到输出层的路径。

这使得计算复杂度与网络层数呈线性关系，而非指数关系，从而让训练深度神经网络成为可能。

## 循环神经网络：引入时间记忆 ⏳

标准的神经网络（如前馈神经网络、卷积神经网络）在处理当前输入时，会“忘记”之前的输入。但在金融时间序列等场景中，历史信息至关重要。

循环神经网络（RNN）通过在神经元间引入循环连接，使网络具备记忆能力。其核心思想是：网络在当前时刻 `t` 的输出 `h_t`，不仅取决于当前输入 `x_t`，还取决于上一时刻的网络状态 `h_{t-1}`。

简化公式为：
`h_t = σ( W_hh * h_{t-1} + W_xh * x_t + b )`
其中 `W_hh` 和 `W_xh` 是权重矩阵，`σ` 是激活函数。

通过这种结构，信息可以在时间步之间传递，网络能够利用历史上下文进行预测。

![](img/0fd58ea1546f204dceed97dbc24457d9_1.png)

**长短期记忆网络（LSTM）** 是RNN的一种成功变体。它通过引入“门”机制（输入门、遗忘门、输出门），更精细地控制信息的保留与遗忘，有效解决了标准RNN在训练中可能遇到的梯度消失或爆炸问题，使其能够学习更长期的依赖关系。
