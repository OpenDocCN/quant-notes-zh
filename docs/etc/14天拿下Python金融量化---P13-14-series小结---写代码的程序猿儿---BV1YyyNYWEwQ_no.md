## 概述

![](img/52d38e96932879e8439540b8f9d20117_5.png)

Series是Pandas库中的一种一维数据结构，它结合了数组和字典的特性。本节将系统性地总结Series的核心功能，包括数据访问、索引、数据对齐以及缺失值处理。

![](img/52d38e96932879e8439540b8f9d20117_7.png)

## Series的核心特性

![](img/52d38e96932879e8439540b8f9d20117_9.png)

上一节我们介绍了Series的基本操作，本节中我们来看看它的核心特性总结。

Series是一个**字典**与**数组**的集合体。它既支持像数组一样通过整数下标进行访问，也支持像字典一样通过标签进行访问。

![](img/52d38e96932879e8439540b8f9d20117_11.png)

以下是Series支持的主要操作：

![](img/52d38e96932879e8439540b8f9d20117_13.png)

![](img/52d38e96932879e8439540b8f9d20117_15.png)

*   **数组式操作**：支持按下标进行索引、切片、布尔值索引。两个Series之间或Series与标量之间可以进行加减乘除等运算。
*   **字典式操作**：支持按标签进行索引，以及使用`in`操作符检查标签是否存在。

![](img/52d38e96932879e8439540b8f9d20117_17.png)

## 整数索引的歧义与解决方案

![](img/52d38e96932879e8439540b8f9d20117_19.png)

当Series的索引为整数时，直接使用中括号`[]`进行索引可能会产生歧义，因为它可能被解释为下标索引，也可能被解释为标签索引。

为了解决这个问题，Pandas提供了两个明确的属性：

![](img/52d38e96932879e8439540b8f9d20117_21.png)

*   **`.iloc[]`**：用于纯粹的、基于整数位置的索引（即下标索引）。
*   **`.loc[]`**：用于基于标签的索引。

![](img/52d38e96932879e8439540b8f9d20117_23.png)

![](img/52d38e96932879e8439540b8f9d20117_25.png)

例如：
```python
# 假设s是一个索引为[0, 1, 2]的Series
value_by_index = s.iloc[0]  # 获取第一个位置（下标）的元素
value_by_label = s.loc[0]   # 获取标签为0的元素
```

## 数据对齐

![](img/52d38e96932879e8439540b8f9d20117_27.png)

![](img/52d38e96932879e8439540b8f9d20117_29.png)

在Pandas中，当两个Series进行运算（如加减乘除）时，其运算是基于**标签对齐**的。

![](img/52d38e96932879e8439540b8f9d20117_31.png)

这意味着，运算会按照两个Series共有的标签进行。如果一个标签只存在于其中一个Series中，那么在结果Series中，该标签对应的值将是缺失值（NaN）。

![](img/52d38e96932879e8439540b8f9d20117_33.png)

## 缺失数据处理

![](img/52d38e96932879e8439540b8f9d20117_35.png)

在数据对齐过程中，常常会产生缺失值（NaN）。Pandas提供了两种主要方法来处理缺失数据。

![](img/52d38e96932879e8439540b8f9d20117_37.png)

以下是处理缺失数据的两种方法：

![](img/52d38e96932879e8439540b8f9d20117_38.png)

![](img/52d38e96932879e8439540b8f9d20117_40.png)

1.  **删除缺失值**：使用`dropna()`函数直接移除包含NaN的行。
    ```python
    series_without_na = original_series.dropna()
    ```
2.  **填充缺失值**：使用`fillna()`函数在缺失值的位置填入指定的新值（如0、平均值等）。
    ```python
    series_filled = original_series.fillna(0)  # 用0填充所有NaN
    ```

![](img/52d38e96932879e8439540b8f9d20117_42.png)

## Series的功能来源

![](img/52d38e96932879e8439540b8f9d20117_44.png)

除了我们介绍的这些Pandas特有功能（如标签索引、数据对齐、`.iloc`/`.loc`）外，Series的许多其他功能都继承自其底层依赖的NumPy数组。

![](img/52d38e96932879e8439540b8f9d20117_46.png)

例如，布尔型索引、花式索引（Fancy Indexing）等操作，其行为与NumPy数组一致。

![](img/52d38e96932879e8439540b8f9d20117_48.png)
