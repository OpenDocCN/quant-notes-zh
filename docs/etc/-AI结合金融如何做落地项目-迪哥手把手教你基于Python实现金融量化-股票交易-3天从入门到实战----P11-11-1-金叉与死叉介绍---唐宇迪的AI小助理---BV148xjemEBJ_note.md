## 概述

![](img/d6ef4545fc99ecad8095f24201f6e735_7.png)

![](img/d6ef4545fc99ecad8095f24201f6e735_9.png)

在股票交易数据分析中，直接观察每日收盘价的波动往往难以把握整体趋势。为了更清晰地描述价格变化趋势，我们引入了移动平均线的概念。本节课将介绍如何利用两条不同周期的移动平均线（即“双均线”）来生成交易信号，并解释金叉（买入信号）与死叉（卖出信号）的形成原理与作用。

![](img/d6ef4545fc99ecad8095f24201f6e735_11.png)

## 双均线概念

![](img/d6ef4545fc99ecad8095f24201f6e735_13.png)

![](img/d6ef4545fc99ecad8095f24201f6e735_15.png)

![](img/d6ef4545fc99ecad8095f24201f6e735_17.png)

上一节我们提到了从个体价格观察趋势的局限性，本节中我们来看看如何通过“平均”的方式来平滑数据，以观察趋势。

所谓“双均线”，指的是计算两条不同时间窗口的移动平均线。通常，一条是周期较短的“短期均线”，另一条是周期较长的“长期均线”。

![](img/d6ef4545fc99ecad8095f24201f6e735_19.png)

![](img/d6ef4545fc99ecad8095f24201f6e735_21.png)

*   **短期均线**：例如选择5个交易日（一周）或10个交易日作为窗口，计算该窗口内收盘价的平均值。它反应了近期价格的平均走势，对价格变化更为敏感。
*   **长期均线**：例如选择20个交易日（约一月）、60个交易日（约一季）作为窗口。它反应了较长期价格的平均走势，更为平滑，趋势性更强。

![](img/d6ef4545fc99ecad8095f24201f6e735_23.png)

![](img/d6ef4545fc99ecad8095f24201f6e735_25.png)

计算移动平均线的公式如下：
`MA(N) = (P1 + P2 + ... + PN) / N`
其中，`MA(N)`代表N日移动平均值，`P1`到`PN`代表最近N个交易日的收盘价。

通过计算并绘制这两条均线，我们可以得到比原始价格线更平滑的趋势线。

## 金叉与死叉

![](img/d6ef4545fc99ecad8095f24201f6e735_27.png)

![](img/d6ef4545fc99ecad8095f24201f6e735_29.png)

![](img/d6ef4545fc99ecad8095f24201f6e735_31.png)

在得到短期和长期两条均线后，它们的相对位置变化能为我们提供交易信号。以下是两种关键的交叉信号：

![](img/d6ef4545fc99ecad8095f24201f6e735_33.png)

![](img/d6ef4545fc99ecad8095f24201f6e735_35.png)

1.  **黄金交叉（金叉）**
    当短期均线从下方向上穿越长期均线时，形成黄金交叉。这通常意味着近期平均成本已超过长期平均成本，市场短期动能转强，可能开启上涨趋势，因此被视为**买入信号**。

2.  **死亡交叉（死叉）**
    当短期均线从上方向下穿越长期均线时，形成死亡交叉。这通常意味着近期平均成本已低于长期平均成本，市场短期动能转弱，可能开启下跌趋势，因此被视为**卖出信号**。

## 双均线交易策略

现在既然我们已经知道了金叉（买点）和死叉（卖点）的含义，那么对于一个股票数据序列，我们就可以设计一个基础的交易策略。

![](img/d6ef4545fc99ecad8095f24201f6e735_37.png)

该策略的核心逻辑非常简单：
*   当出现**金叉**时，执行买入操作。
*   当出现**死叉**时，执行卖出（或做空）操作。

![](img/d6ef4545fc99ecad8095f24201f6e735_39.png)

理论上，如果能够准确识别每一次交叉信号并正确执行交易，投资者就能在上涨趋势开始时买入，在下跌趋势开始时卖出或离场，从而获取收益。后续课程中，我们将使用Python来实际计算移动平均线，并基于历史数据回测这一策略的效果。
