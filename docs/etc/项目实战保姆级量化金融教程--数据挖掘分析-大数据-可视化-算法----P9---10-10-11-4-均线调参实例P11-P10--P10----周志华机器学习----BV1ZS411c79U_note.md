```python
# 定义短期和长期均线的参数范围
sma1_range = range(20, 61, 4)   # 从20到60，步长为4
sma2_range = range(180, 281, 10) # 从180到280，步长为10

![](img/61b575ab7569e375de6a1b7f3d3fa889_68.png)

# 初始化一个空的DataFrame来存储所有结果
results = pd.DataFrame()

![](img/61b575ab7569e375de6a1b7f3d3fa889_70.png)

# 遍历所有参数组合
for sma1, sma2 in product(sma1_range, sma2_range):
    # 在这里执行策略计算...
    # 1. 重新加载并预处理数据
    # 2. 计算收益率和均线
    # 3. 生成交易信号并计算策略收益
    # 4. 将本次参数组合的结果保存到results中
```

参数空间的意思是，例如 `sma1=20` 会与 `sma2=180, 190, 200...` 等所有值组合，`product` 函数会帮我们生成所有这样的配对。

## 🧮 在循环内执行策略计算

![](img/61b575ab7569e375de6a1b7f3d3fa889_76.png)

![](img/61b575ab7569e375de6a1b7f3d3fa889_78.png)

![](img/61b575ab7569e375de6a1b7f3d3fa889_80.png)

在每次循环中，我们需要为当前的 `sma1` 和 `sma2` 参数值重新执行一遍完整的策略流程。

![](img/61b575ab7569e375de6a1b7f3d3fa889_82.png)

![](img/61b575ab7569e375de6a1b7f3d3fa889_84.png)

![](img/61b575ab7569e375de6a1b7f3d3fa889_86.png)

以下是循环内需要完成的主要步骤：

![](img/61b575ab7569e375de6a1b7f3d3fa889_88.png)

![](img/61b575ab7569e375de6a1b7f3d3fa889_90.png)

1.  **重新加载数据**：为了保证每次计算独立，最好重新加载股票数据。
2.  **数据预处理**：处理缺失值。
3.  **计算指标**：计算收益率 `returns`、短期均线 `SMA1` 和长期均线 `SMA2`。
4.  **生成交易信号**：根据双均线交叉规则生成持仓信号 `position`。
5.  **计算策略收益**：根据信号计算策略的累计收益。
6.  **保存结果**：将当前参数组合及对应的策略收益、基准收益等信息保存下来。

![](img/61b575ab7569e375de6a1b7f3d3fa889_92.png)

![](img/61b575ab7569e375de6a1b7f3d3fa889_94.png)

核心计算公式和代码与之前一致：
*   收益率：`returns = data['Close'].pct_change()`
*   移动平均线：`sma1 = data['Close'].rolling(window=sma1).mean()`
*   交易信号：当短期均线上穿长期均线时，持仓为1；否则为0。

所有参数组合计算完毕后，我们将结果整理成一个清晰的表格（DataFrame），便于分析。

![](img/61b575ab7569e375de6a1b7f3d3fa889_110.png)

![](img/61b575ab7569e375de6a1b7f3d3fa889_112.png)

结果表应包含以下列：
*   `SMA1`: 短期均线窗口值
*   `SMA2`: 长期均线窗口值
*   `Returns`: 基准收益（买入并持有）
*   `Strategy`: 策略收益
*   `Outperformance`: 策略相对于基准的超额收益（`Strategy - Returns`）

![](img/61b575ab7569e375de6a1b7f3d3fa889_114.png)

通过观察 `Outperformance` 列，我们可以判断：
*   如果值为正，说明该参数下的策略表现优于简单持有。
*   如果值为负，说明该参数组合效果不佳，甚至不如不操作。
*   通过排序，我们可以快速找出表现最好的几组参数。
