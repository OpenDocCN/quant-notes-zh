上一节我们概述了课程内容，本节中我们来看看技术分析的基础——K线图。

![](img/3c8e4134ff81ede01c8d9a830350dee3_11.png)

![](img/3c8e4134ff81ede01c8d9a830350dee3_13.png)

K线图，也称为蜡烛图，起源于日本。它用于展示特定时间段内的开盘价、收盘价、最高价和最低价。一根典型的K线包含实体和影线两部分。

![](img/3c8e4134ff81ede01c8d9a830350dee3_15.png)

*   **实体**：表示开盘价与收盘价之间的价格区间。如果收盘价高于开盘价，实体通常为空心或红色，表示上涨；反之则为实心或绿色，表示下跌。
*   **上影线**：实体上方的细线，顶端代表该时间段内的最高价。
*   **下影线**：实体下方的细线，底端代表该时间段内的最低价。

在量化交易中，单根K线常被称为“Bar”。Bar的频率取决于所选择的时间周期。

以下是不同时间周期的Bar示例：
*   **日K线**：一根Bar代表一天（如3月1日、3月2日）。
*   **分钟K线**：一根Bar代表一分钟，包含该分钟内的开盘、收盘、最高、最低价。
*   **Tick数据**：代表一次价格跳动，是频率最高的数据，常用于高频交易策略。

## 技术指标概述

![](img/3c8e4134ff81ede01c8d9a830350dee3_17.png)

![](img/3c8e4134ff81ede01c8d9a830350dee3_19.png)

了解了K线的基本构成后，本节我们来看看基于K线数据衍生出的各类技术指标。技术指标通过数学公式处理价格和成交量数据，旨在揭示市场趋势、动量或超买超卖状态。

### 移动平均线

移动平均线通过计算过去一段时间价格的平均值来平滑价格曲线，帮助识别趋势方向。

*   **简单移动平均**：对过去N个周期的收盘价求算术平均值。公式为：`SMA = (P1 + P2 + ... + PN) / N`，其中`P`为价格，`N`为周期数。
*   **指数移动平均**：赋予近期价格更高的权重，使均线对价格变化反应更灵敏。计算公式涉及平滑系数，对近期数据赋予更大权重。

常见的交易信号是均线的“金叉”（短期均线上穿长期均线，视为买入信号）和“死叉”（短期均线下穿长期均线，视为卖出信号）。

### MACD指标

MACD是趋势跟踪动量指标，由快线、慢线和柱状图组成。

*   **计算**：MACD线是短期EMA与长期EMA的差值，信号线是MACD线的EMA，柱状图是MACD线与信号线的差值。
*   **应用**：
    *   **金叉/死叉**：MACD线上穿信号线为金叉（买入），下穿为死叉（卖出）。
    *   **背离**：价格创新低而MACD指标未创新低，可能预示底部反转。
    *   **零轴**：MACD在零轴上方通常表示多头市场，下方表示空头市场。

### KDJ指标

KDJ是随机振荡指标，反应灵敏，用于判断超买超卖区域。

*   **构成**：由K线、D线和J线三条曲线组成。
*   **应用**：主要通过K线与D线的金叉（买入）、死叉（卖出）来交易，尤其在指标处于低位（如低于20）或高位（如高于80）时发出的信号更受关注。

### RSI指标

RSI是相对强弱指标，衡量价格变动速度和幅度，评估超买超卖状况。

*   **计算**：基于一定时期内上涨和下跌幅度的平均值。
*   **应用**：RSI值通常在0到100之间。一般认为RSI超过70为超买区，可能回调；低于30为超卖区，可能反弹。

### CCI指标

CCI是顺势指标，衡量当前价格相对于统计平均价格的偏离程度。

*   **计算**：`CCI = (典型价格 - N期典型价格的SMA) / (0.015 * N期典型价格的平均绝对偏差)`。其中典型价格 = `(最高价 + 最低价 + 收盘价) / 3`。
*   **应用**：CCI可正可负。常见用法是以+100和-100作为分界线。
    *   CCI从下方上穿-100，可能视为买入机会。
    *   CCI从上方下穿+100，可能视为卖出信号。
    *   CCI在+100以上或-100以下运行，表示价格处于非常态区间，波动可能加剧。

### 布林带指标

布林带由一条中轨线和两条上下轨线组成，反映价格波动区间。

*   **计算**：
    *   中轨 = N期简单移动平均线
    *   上轨 = 中轨 + K * N期价格的标准差
    *   下轨 = 中轨 - K * N期价格的标准差
    *   （通常N=20， K=2）
*   **应用**：
    *   **区间交易**：价格触及上轨可能回调（卖出），触及下轨可能反弹（买入）。
    *   **趋势跟踪**：价格持续沿上轨运行表示强势上涨，沿下轨运行表示强势下跌。
    *   **收口与开口**：布林带收口预示波动率降低，可能即将突破；开口预示波动率增大，趋势可能延续。
