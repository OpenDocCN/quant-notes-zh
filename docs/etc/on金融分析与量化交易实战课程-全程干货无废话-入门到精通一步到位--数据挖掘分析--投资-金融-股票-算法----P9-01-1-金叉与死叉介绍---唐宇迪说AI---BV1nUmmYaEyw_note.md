在交易数据分析中，直接观察每日价格波动难以把握整体趋势。双均线策略通过计算不同时间窗口的移动平均线，帮助我们更清晰地识别价格趋势和潜在的买卖时机。

## 双均线是什么

![](img/e6f86d3c7632f08db36460e6bca6bcab_20.png)

![](img/e6f86d3c7632f08db36460e6bca6bcab_22.png)

上一节我们提到了趋势分析的重要性，本节中我们来看看如何具体实现。双均线策略涉及两条移动平均线。

![](img/e6f86d3c7632f08db36460e6bca6bcab_24.png)

![](img/e6f86d3c7632f08db36460e6bca6bcab_26.png)

*   **移动平均线 (MA)**：它通过计算指定时间段内收盘价的平均值来平滑价格数据，从而反映趋势。
*   **短均线 (Short MA)**：计算周期较短的移动平均线，例如5日或10日均线，对近期价格变化更敏感。
*   **长均线 (Long MA)**：计算周期较长的移动平均线，例如20日或60日均线，反映更长期的价格趋势。

![](img/e6f86d3c7632f08db36460e6bca6bcab_28.png)

短均线的计算公式如下，其中 `Close_i` 代表第 `i` 日的收盘价：
`短均线_t = (Close_{t-4} + Close_{t-3} + Close_{t-2} + Close_{t-1} + Close_t) / 5`
同理，长均线采用更长的计算窗口，例如20日。

## 金叉与死叉：核心交易信号

![](img/e6f86d3c7632f08db36460e6bca6bcab_30.png)

计算出长短两条均线后，它们的相对位置和交叉关系就构成了交易信号。

![](img/e6f86d3c7632f08db36460e6bca6bcab_32.png)

![](img/e6f86d3c7632f08db36460e6bca6bcab_34.png)

![](img/e6f86d3c7632f08db36460e6bca6bcab_36.png)

以下是两种关键的交叉信号：

![](img/e6f86d3c7632f08db36460e6bca6bcab_38.png)

1.  **黄金交叉 (Golden Cross / 金叉)**
    *   **定义**：短均线从下向上穿越长均线。
    *   **市场含义**：短期趋势开始强于长期趋势，通常被视为**买入信号**。
    *   **图示**：在价格图表上，短均线（如红线）上穿长均线（如蓝线）的点即为买点。

![](img/e6f86d3c7632f08db36460e6bca6bcab_40.png)

2.  **死亡交叉 (Death Cross / 死叉)**
    *   **定义**：短均线从上向下穿越长均线。
    *   **市场含义**：短期趋势开始弱于长期趋势，通常被视为**卖出信号**。
    *   **图示**：在价格图表上，短均线（如红线）下穿长均线（如蓝线）的点即为卖点。

## 双均线策略的逻辑

![](img/e6f86d3c7632f08db36460e6bca6bcab_42.png)

基于金叉和死叉信号，我们可以构建一个简单的交易策略。

![](img/e6f86d3c7632f08db36460e6bca6bcab_44.png)

![](img/e6f86d3c7632f08db36460e6bca6bcab_46.png)

策略逻辑可以用以下伪代码描述：
```python
# 初始化：计算短均线(MA_short)和长均线(MA_long)
for 每一天 in 数据:
    if MA_short 上穿 MA_long:  # 发生金叉
        执行买入操作
    elif MA_short 下穿 MA_long:  # 发生死叉
        执行卖出操作
```
理想情况下，在每次金叉时买入，死叉时卖出，通过捕捉趋势变化来获取收益。
