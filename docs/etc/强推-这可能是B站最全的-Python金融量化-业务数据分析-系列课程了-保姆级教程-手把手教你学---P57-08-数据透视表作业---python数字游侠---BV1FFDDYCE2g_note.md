## 第一题：部门消费占比与列联表分析

![](img/b0971f19325793144ed4875b53520e41_3.png)

上一节我们介绍了数据透视表的基础操作，本节中我们来看看如何计算部门消费占比并构建列联表。

### 1. 计算部门消费金额及占比

首先，我们需要计算每个部门的水电费总金额及其占总金额的百分比。

![](img/b0971f19325793144ed4875b53520e41_5.png)

**操作步骤如下：**
1.  选中原数据，点击“插入”选项卡中的“数据透视表”。
2.  将“部门”字段拖入“行”区域。
3.  将“金额”字段拖入“值”区域。
4.  在数据透视表的“值”区域中，点击“求和项：金额”的下拉箭头，选择“值字段设置”。
5.  在弹出的对话框中，切换到“值显示方式”选项卡。
6.  在“值显示方式”的下拉列表中，选择“总计的百分比”。
7.  点击“确定”。

此时，数据透视表将同时显示每个部门的消费金额及其占总金额的百分比。这个功能展示了数据透视表在计算相对值时的灵活性。

### 2. 构建费用类型的列联表

![](img/b0971f19325793144ed4875b53520e41_7.png)

接下来，我们构建一个二维表格，用于展示不同部门在不同费用项目（如水费、电费）上的具体金额。这种表格在统计学中称为“列联表”。

![](img/b0971f19325793144ed4875b53520e41_9.png)

**操作步骤如下：**
1.  新建一个数据透视表，或将现有透视表字段清空。
2.  将“部门”字段拖入“行”区域。
3.  将“项目名称”字段拖入“列”区域。
4.  将“金额”字段拖入“值”区域。

![](img/b0971f19325793144ed4875b53520e41_11.png)

完成以上步骤后，即可得到一个以部门为行、以费用类型为列的二维金额汇总表，即列联表。它能清晰展示数据在两个维度上的分布情况。

![](img/b0971f19325793144ed4875b53520e41_13.png)

![](img/b0971f19325793144ed4875b53520e41_15.png)

## 第二题：多维度数值计算

![](img/b0971f19325793144ed4875b53520e41_17.png)

这道题主要考察对值字段汇总方式的掌握，要求计算最大值、最小值、求和及计数。

![](img/b0971f19325793144ed4875b53520e41_19.png)

![](img/b0971f19325793144ed4875b53520e41_21.png)

以下是需要计算的四个数值：
*   **最大值**：找出指定字段中的最大数值。
*   **最小值**：找出指定字段中的最小数值。
*   **求和**：计算指定字段所有数值的总和。
*   **计数**：统计指定字段中非空项目的数量。

实现方法很简单：将需要分析的字段多次拖入“值”区域，然后分别对每个“值”字段进行设置，在“值字段设置”的“值汇总方式”选项卡中选择对应的计算类型（最大值、最小值、求和、计数）即可。

## 第三题：计算相对于基年的价格变化趋势

上一节我们处理了静态占比，本节中我们来看看如何分析动态变化，即计算各年份价格相对于基准年份（2009年）的增长百分比。

![](img/b0971f19325793144ed4875b53520e41_23.png)

**操作步骤如下：**
1.  插入数据透视表，将“采购年份”拖入“行”区域，“物料名称”拖入“列”区域，“单价”拖入“值”区域。
2.  在“值”区域中，点击“求和项：单价”，选择“值字段设置”。
3.  在“值显示方式”选项卡中，选择“差异百分比”。
4.  在“基本字段”中选择“采购年份”，在“基本项”中选择“2009”。
5.  点击“确定”。

![](img/b0971f19325793144ed4875b53520e41_25.png)

![](img/b0971f19325793144ed4875b53520e41_27.png)

完成设置后，数据透视表将显示其他各年份价格相对于2009年的变化百分比。例如，2010年触摸屏价格显示为“10%”，即表示其价格比2009年增长了10%。这个功能非常适合用于分析时间序列数据的变化趋势。

![](img/b0971f19325793144ed4875b53520e41_29.png)

## 第四题：使用数据透视图进行多城市销量对比

![](img/b0971f19325793144ed4875b53520e41_31.png)

本节我们将结合数据透视表与数据透视图，对北京、上海、广州的销量进行动态对比分析。

**操作步骤如下：**
1.  基于源数据创建数据透视表。
2.  将“日期”字段拖入“行”区域。**请注意**：如果源数据中的日期是标准的日期格式，数据透视表会自动将其按年、季度、月进行分级，方便进行不同时间粒度的分析。
3.  将“销售区域”字段拖入“列”区域，并通过筛选功能只保留“北京”、“上海”、“广州”。
4.  将“销量”字段拖入“值”区域。
5.  选中数据透视表任意单元格，点击“分析”选项卡中的“数据透视图”，选择“折线图”等图表类型。

创建完成后，数据透视图与数据透视表是联动的。例如，在数据透视表中点击“年”前面的加号（+）展开到“月”，数据透视图也会同步更新，显示按月变化的趋势。需要注意的是，数据透视图的数据源直接绑定到其关联的数据透视表，因此不能像普通图表那样随意更改数据源区域，调整数据需要通过操作数据透视表来实现。
