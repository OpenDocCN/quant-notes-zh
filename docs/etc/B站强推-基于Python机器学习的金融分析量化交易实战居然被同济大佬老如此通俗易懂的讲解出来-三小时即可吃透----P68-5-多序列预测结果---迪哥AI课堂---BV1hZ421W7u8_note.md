上一节我们介绍了模型的训练过程，本节中我们来看看如何使用训练好的模型进行多步预测。核心思想是：**使用前 `time_step` 个已知数据点预测下一个点，然后将这个预测点加入已知序列，再预测下一个点，如此循环**。

![](img/91394bea95788f06c5816fbe714c76d4_34.png)

以下是预测循环的关键步骤：

1.  **初始化序列**：从测试数据中取出第一个长度为 `time_step` 的序列作为初始输入。
2.  **单步预测**：将当前序列输入模型，得到下一个时间点的预测值。
3.  **更新序列**：将预测值添加到当前序列的末尾，并移除序列最前端的一个值，从而生成一个新的、长度仍为 `time_step` 的序列。
4.  **循环迭代**：重复步骤2和3，直到预测出指定数量的未来值。

![](img/91394bea95788f06c5816fbe714c76d4_36.png)

为了便于调试和理解，代码中设置了一个 `debug` 标志。当 `debug=True` 时，程序会打印出每一步的中间结果。

![](img/91394bea95788f06c5816fbe714c76d4_38.png)

![](img/91394bea95788f06c5816fbe714c76d4_40.png)

![](img/91394bea95788f06c5816fbe714c76d4_42.png)

```python
# 预测循环的核心代码逻辑
for i in range(predict_length):
    # 1. 将当前序列转换为模型需要的三维格式 (1, time_step, 1)
    current_seq = data[i:i+time_step].reshape((1, time_step, 1))
    # 2. 进行单步预测
    predicted_value = model.predict(current_seq)[0, 0]
    # 3. 更新序列：移除第一个值，加入预测值
    data = np.append(data[1:], predicted_value)
```
