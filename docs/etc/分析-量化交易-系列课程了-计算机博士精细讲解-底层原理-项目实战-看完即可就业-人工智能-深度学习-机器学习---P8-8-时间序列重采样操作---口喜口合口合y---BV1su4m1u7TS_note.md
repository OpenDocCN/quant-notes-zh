## 重采样操作 🔄

![](img/fa418296736ce785cb8820ed5fd73cea_3.png)

上一节我们介绍了时间序列的基本概念，本节中我们来看看如何对时间序列进行重采样。重采样是指将数据从一个时间频率转换到另一个时间频率的过程。例如，原始数据可能是按天记录的，我们可以将其转换为按周、按月或按年进行统计。

![](img/fa418296736ce785cb8820ed5fd73cea_5.png)

要进行重采样，我们需要使用 `resample` 方法。这个方法至少需要一个参数，即指定新的时间频率单位。

![](img/fa418296736ce785cb8820ed5fd73cea_7.png)

![](img/fa418296736ce785cb8820ed5fd73cea_9.png)

以下是重采样的基本步骤：

![](img/fa418296736ce785cb8820ed5fd73cea_11.png)

1.  指定重采样的频率，例如 `'W'` 代表周，`'M'` 代表月。
2.  指定在重采样后，每个时间段内数据的聚合方式，例如计算均值、第一个值、中位数等。

![](img/fa418296736ce785cb8820ed5fd73cea_13.png)

```python
# 假设 df 是一个以日期为索引的DataFrame
# 按周进行重采样，并计算每周数据的平均值
weekly_data = df.resample('W').mean()
```

执行上述代码后，我们便得到了按周重采样后的结果。重采样的过程相当于设定了一个时间窗口（例如一周），并将窗口内的数据按指定规则聚合。

![](img/fa418296736ce785cb8820ed5fd73cea_15.png)

我们不仅可以按周重采样，也可以按月或其他频率进行。

![](img/fa418296736ce785cb8820ed5fd73cea_17.png)

```python
# 按月进行重采样，并计算每月数据的第一个值
monthly_data = df.resample('M').first()
```

![](img/fa418296736ce785cb8820ed5fd73cea_19.png)

在重采样时，有一个重要的参数需要注意：**标签**。它决定了聚合后的数据点使用哪个时间戳作为索引。

![](img/fa418296736ce785cb8820ed5fd73cea_21.png)

![](img/fa418296736ce785cb8820ed5fd73cea_23.png)

以下是标签参数 `label` 的两种主要选项：

![](img/fa418296736ce785cb8820ed5fd73cea_25.png)

*   `label='right'`：默认选项。使用时间窗口的**结束日期**作为该窗口数据的标签。
*   `label='left'`：使用时间窗口的**开始日期**作为该窗口数据的标签。

例如，按月重采样时，`label='right'` 会用当月的最后一天（如1月31日）来标记该月的数据；而 `label='left'` 则会用当月的第一天（如1月1日）来标记。选择哪种方式取决于具体的分析逻辑和需求，通常 `label='right'` 更符合观察习惯，因为它表示的是截至该日期为止的统计结果。

## 时间窗口操作 🪟

![](img/fa418296736ce785cb8820ed5fd73cea_27.png)

接下来，我们探讨时间窗口操作。如果说重采样是改变观察数据的“尺度”，那么时间窗口操作则是在原有尺度上进行“滑动观察”。

![](img/fa418296736ce785cb8820ed5fd73cea_29.png)

时间窗口操作，也称为滑动窗口，其概念是：设定一个固定长度的窗口（例如10天），让这个窗口在时间序列上依次滑动。在每一个窗口位置，我们都对窗口内的数据进行计算。

![](img/fa418296736ce785cb8820ed5fd73cea_31.png)

例如，对于一个有100天的数据序列：
*   第一个窗口包含第1天到第10天的数据。
*   然后窗口向后滑动一天，第二个窗口包含第2天到第11天的数据。
*   接着窗口再滑动一天，第三个窗口包含第3天到第12天的数据，依此类推。

![](img/fa418296736ce785cb8820ed5fd73cea_33.png)

在Pandas中，我们使用 `rolling()` 方法来实现滑动窗口计算。

![](img/fa418296736ce785cb8820ed5fd73cea_35.png)

以下是进行时间窗口操作的基本步骤：

![](img/fa418296736ce785cb8820ed5fd73cea_37.png)

1.  首先，**对数据进行预处理，处理缺失值**。因为如果窗口内存在缺失值，可能会导致整个窗口的计算结果为NaN。
    ```python
    # 假设 data 是包含缺失值的DataFrame
    data_clean = data.dropna() # 删除含有缺失值的行
    ```
2.  使用 `rolling()` 方法指定窗口大小，并对某一列数据应用统计函数。
    ```python
    # 对清洗后的数据，以10天为窗口大小，计算每个窗口的最小值
    window_size = 10
    min_values = data_clean['column_name'].rolling(window=window_size).min()
    ```
    在结果中，前 `window_size - 1` 个值会是NaN，因为直到第10天，第一个窗口的数据才完整。

![](img/fa418296736ce785cb8820ed5fd73cea_39.png)

![](img/fa418296736ce785cb8820ed5fd73cea_41.png)

我们可以同时对多个统计指标进行计算，并将结果添加到原数据中。

![](img/fa418296736ce785cb8820ed5fd73cea_43.png)

以下是如何为数据添加多个窗口统计指标的示例：

![](img/fa418296736ce785cb8820ed5fd73cea_45.png)

![](img/fa418296736ce785cb8820ed5fd73cea_47.png)

```python
# 计算10天窗口内的最小值、最大值、均值和标准差
data_clean['min_10d'] = data_clean['column_name'].rolling(window=10).min()
data_clean['max_10d'] = data_clean['column_name'].rolling(window=10).max()
data_clean['mean_10d'] = data_clean['column_name'].rolling(window=10).mean()
data_clean['std_10d'] = data_clean['column_name'].rolling(window=10).std()

![](img/fa418296736ce785cb8820ed5fd73cea_49.png)

![](img/fa418296736ce785cb8820ed5fd73cea_51.png)

# 查看添加了统计指标后的数据
print(data_clean.tail())
```

![](img/fa418296736ce785cb8820ed5fd73cea_53.png)

这样，我们就得到了基于滑动窗口计算出的各种统计值，能够更细致地分析数据在短期内的变化趋势。

![](img/fa418296736ce785cb8820ed5fd73cea_55.png)

![](img/fa418296736ce785cb8820ed5fd73cea_57.png)

![](img/fa418296736ce785cb8820ed5fd73cea_59.png)

![](img/fa418296736ce785cb8820ed5fd73cea_61.png)

首先，我们学习了**重采样**，它允许我们将数据从一种时间频率（如日度）转换到另一种频率（如周度、月度），并通过指定聚合函数（如 `mean()`, `sum()`）和标签参数（`label`）来得到新的时间序列。
