1.  **卖出不符合条件的股票**：遍历需要删除的股票列表，使用 `order_target_percent` 函数将其持仓比例调整为0，即全部卖出。
    ```python
    for stock in stocks_to_delete:
        order_target_percent(stock, 0)
    ```

    ```python
    for stock in current_pool:
        order_target_percent(stock, 1.0 / len(current_pool))
    ```

在编写和运行策略时，常常会遇到各种错误。以下是调试过程中发现并解决的一些典型问题：

![](img/3c6e40e4b236a481e82839df5c04037d_81.png)

![](img/3c6e40e4b236a481e82839df5c04037d_83.png)

*   **中性化处理参数缺失**：在调用中性化函数时，漏传了必要的参数（如因子列名）。需要补全参数。
*   **模型未训练**：在使用统计模型（如OLS）进行中性化前，必须先调用 `fit()` 方法进行训练，然后才能计算残差。
*   **函数名拼写错误**：例如，将 `quantile` 误拼为 `qant`，导致程序报错。需要仔细检查并修正拼写。
*   **变量名引用错误**：在循环或函数调用中，使用了未定义或拼写错误的变量名（如 `stock_list`）。需要确保变量名一致。

![](img/3c6e40e4b236a481e82839df5c04037d_91.png)

策略运行完毕后，我们可以通过回测平台提供的分析报告来评估其效果。主要关注以下几个核心指标：

![](img/3c6e40e4b236a481e82839df5c04037d_107.png)

![](img/3c6e40e4b236a481e82839df5c04037d_109.png)

![](img/3c6e40e4b236a481e82839df5c04037d_111.png)

![](img/3c6e40e4b236a481e82839df5c04037d_113.png)

*   **累计收益率**：策略在整个回测期间的总收益。图中显示，该策略的收益曲线最终优于基准（如沪深300指数）。
*   **年化收益率**：将累计收益率折算成年化形式，便于比较。
*   **夏普比率**：衡量每承受一单位风险，所获得的超额回报。公式为：
    `Sharpe Ratio = (Rp - Rf) / σp`
    其中 `Rp` 是策略年化收益率，`Rf` 是无风险利率，`σp` 是策略收益的年化标准差。
*   **最大回撤**：策略净值从高点回落至最低点的最大幅度，用于衡量策略的风险。本例中最大回撤约为9%。

![](img/3c6e40e4b236a481e82839df5c04037d_119.png)

从结果来看，虽然该简单策略的绝对收益和夏普比率不算很高，但在回测期内其表现稳定地超越了基准指数，证明了基于市值和市净率因子进行筛选和定期调仓的基本逻辑是有效的。

本节课中我们一起学习了因子选股策略的完整实现与评估流程。我们来回顾一下核心步骤：

![](img/3c6e40e4b236a481e82839df5c04037d_137.png)

![](img/3c6e40e4b236a481e82839df5c04037d_139.png)

![](img/3c6e40e4b236a481e82839df5c04037d_141.png)

![](img/3c6e40e4b236a481e82839df5c04037d_143.png)

![](img/3c6e40e4b236a481e82839df5c04037d_145.png)

1.  **初始化与定时**：在 `initialize` 函数中设置定时器（如每月调仓）。
2.  **股票池筛选**：在定时函数中，首先过滤掉不满足基本条件的股票（如ST股、停牌股）。
3.  **因子获取与处理**：查询剩余股票的因子数据（如市值、市净率），并进行必要的预处理（如中性化处理）。
4.  **因子选股**：根据处理后的因子值进行排序和筛选，得到当期目标股票池。
5.  **执行调仓**：
    *   卖出当前持有但不在新股票池中的股票。
    *   将资金平均买入新股票池中的所有股票。
6.  **效果评估**：运行回测，通过累计收益、年化收益、夏普比率、最大回撤等指标，全面评估策略的盈利能力和风险水平。

![](img/3c6e40e4b236a481e82839df5c04037d_147.png)

![](img/3c6e40e4b236a481e82839df5c04037d_149.png)

![](img/3c6e40e4b236a481e82839df5c04037d_151.png)

![](img/3c6e40e4b236a481e82839df5c04037d_153.png)

这个流程清晰地展示了从因子选择到策略执行，再到绩效评估的量化交易策略开发闭环。需要注意的是，回测结果仅代表历史表现，实际市场环境复杂多变，此策略仅供学习参考。
