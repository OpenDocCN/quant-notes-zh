## 🧮 核心概念回顾

在本教程中，我们实践了年化收益率的计算。其核心公式用于将一段时期内的总收益率转化为一个可比较的年均收益率指标。

年化收益率的计算公式为：
**年化收益率 = (1 + 总收益率)^(1/年数) - 1**

在Python中，我们可以用类似以下的代码来实现计算：
```python
total_return = 1.783  # 标普500的总收益率+1，即 1 + 178.3%
years = 10
annualized_return = total_return ** (1 / years) - 1
print(f"年化收益率: {annualized_return:.2%}")
```
