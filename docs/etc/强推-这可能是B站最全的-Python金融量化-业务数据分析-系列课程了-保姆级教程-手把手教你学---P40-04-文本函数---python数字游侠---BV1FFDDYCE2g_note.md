在本节课中，我们将要学习Excel中处理文本数据的关键工具——文本函数。我们将重点掌握如何从文本中提取特定部分、计算文本长度以及转换文本格式，这些技能在数据清洗和预处理中至关重要。

上一节我们介绍了函数的基本概念，本节中我们来看看专门用于处理文本数据的函数。

文本函数是针对文本数据进行操作的函数。例如，身份证号码是一个包含多种信息的文本字符串，我们可以从中提取省份编号、出生日期和性别等信息。这时就需要使用文本函数。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_5.png)

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_7.png)

文本函数的核心是三个用于提取子字符串的函数：LEFT、RIGHT和MID。它们分别用于从文本的左侧、右侧和中间开始提取指定长度的字符。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_9.png)

以下是这三个函数的具体用法：

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_11.png)

*   **LEFT函数**：从文本左侧开始提取字符。
    *   **公式**：`=LEFT(text, [num_chars])`
    *   **参数说明**：
        *   `text`：需要提取的原始文本。
        *   `num_chars`：指定要从左侧开始提取的字符数量。
    *   **示例**：从身份证号中提取前3位地区码，公式为 `=LEFT(A2, 3)`。在Excel中，一个数字、字母或汉字均被视为一个字符。

*   **RIGHT函数**：从文本右侧开始提取字符。
    *   **公式**：`=RIGHT(text, [num_chars])`
    *   **参数说明**：
        *   `text`：需要提取的原始文本。
        *   `num_chars`：指定要从右侧开始提取的字符数量。
    *   **示例**：提取身份证号后6位，公式为 `=RIGHT(A2, 6)`。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_13.png)

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_15.png)

*   **MID函数**：从文本中间指定位置开始提取字符。
    *   **公式**：`=MID(text, start_num, num_chars)`
    *   **参数说明**：
        *   `text`：需要提取的原始文本。
        *   `start_num`：开始提取的起始位置（Excel中位置从1开始计数）。
        *   `num_chars`：指定要提取的字符数量。
    *   **示例**：从身份证号第7位开始，提取8位出生日期（YYYYMMDD），公式为 `=MID(A2, 7, 8)`。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_17.png)

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_19.png)

掌握了文本提取函数后，我们再来学习一个用于检查文本长度的函数：LEN函数。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_21.png)

LEN函数用于返回文本字符串的字符个数。在数据清洗中，常用来校验固定长度数据的准确性，例如身份证号应为18位，手机号为11位。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_23.png)

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_25.png)

*   **LEN函数**：计算文本字符串的长度。
    *   **公式**：`=LEN(text)`
    *   **参数说明**：
        *   `text`：要计算长度的文本。
    *   **示例**：检查身份证号长度是否为18位，公式为 `=LEN(A2)`。若结果不为18，则说明该条数据可能存在录入错误。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_27.png)

最后，我们学习一个功能强大的格式转换函数：TEXT函数。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_29.png)

TEXT函数可以将数值或日期转换为指定格式的文本。它不仅能够实现简单的日期格式转换，还支持复杂的自定义格式。

*   **TEXT函数**：将数值转换为按指定数字格式表示的文本。
    *   **公式**：`=TEXT(value, format_text)`
    *   **参数说明**：
        *   `value`：需要转换的数值、日期或引用包含数值的单元格。
        *   `format_text`：用双引号括起的文本形式的数字格式。
    *   **示例**：将日期“2023-10-27”转换为“2023年10月27日”的文本格式，公式为 `=TEXT(B2, "yyyy年mm月dd日")`。格式代码可以参考Excel单元格格式设置中的“自定义”类别。

![](img/ce94e5062448ae6cdbe9cb0567dfc01c_31.png)

本节课中我们一起学习了Excel中五个核心的文本函数：用于提取子串的LEFT、RIGHT、MID函数，用于计算长度的LEN函数，以及用于格式转换的TEXT函数。熟练运用这些函数，能极大地提升我们在数据处理中清洗、校验和转换文本数据的效率。
