## 🛤️ GDX轨道线

GDX轨道线通过反映中心线与收盘价的乖离，对趋势线进行平滑和修正，从而更精确地描述股价的运行规律。它由三根线组成：上方的压力线、下方的支撑线以及中间的济安线（中心线）。

以下是GDX轨道线的核心算法：

*   **济安线（中心线）**：其计算涉及一个中间变量 `AA`。
    *   `AA = (2 * 收盘价 + 最高价 + 最低价) / 4`
    *   `济安线 = 以AA为权重的收盘价动态移动平均`
*   **压力线**：在济安线的基础上向上偏移一定百分比。
    *   `压力线 = (1 + M/100) * 济安线`
*   **支撑线**：在济安线的基础上向下偏移一定百分比。
    *   `支撑线 = (1 - M/100) * 济安线`

其中，参数 `N` 通常取30，`M` 通常取9。

了解了GDX的构成后，我们来看它的买卖应用原则：
*   当股价上升触及压力线时，可视为卖出信号。
*   当股价下跌触及支撑线时，可视为买入信号。

## 🚀 JS加速线

这里的JS并非JavaScript，而是加速线。它的原理是通过对比当前股价与一段时间前的股价，来测量股价涨跌的速度，从而判断趋势运行的快慢。

JS指标包含四根线：JS线本身，以及JS线在不同周期下的简单移动平均线。

以下是JS加速线的计算方法：
*   **JS线**：`JS = 100 * (收盘价 - N日前收盘价) / (N * N日前收盘价)`
    *   参数 `N` 通常取5。
*   **JS均线**：分别计算JS线的5日(`MAJS1`)、10日(`MAJS2`)、20日(`MAJS3`)简单移动平均。

接下来是JS指标的应用方法：
*   当JS线向下交叉一根或多根JS均线时，为卖出信号。
*   当JS线向上交叉一根或多根JS均线时，为买入信号。

## 💻 代码实战

现在，我们将使用Python的`technical_analysis`库来实际计算GDX和JS指标。

首先，我们实现GDX轨道线的计算。

```python
# 导入技术分析库
import technical_analysis as ta

# 定义股票代码列表
security_list = ['0001002', '6012111']
# 设置计算日期
check_date = '2022-09-01'
# 计算GDX指标，参数N=14, M=9
jax, ylx, zcx = ta.GDX(security_list, check_date, N=14, M=9)

# 循环输出每只股票的结果
for i in range(len(security_list)):
    print(f"股票 {security_list[i]} 在 {check_date} 的GDX指标：")
    print(f"  济安线(JAX): {jax[i]}")
    print(f"  压力线(YLX): {ylx[i]}")
    print(f"  支撑线(ZCX): {zcx[i]}")
    print()
```

运行上述代码，可以得到每只股票的济安线、压力线和支撑线数值。当股价高于压力线时考虑卖出，低于支撑线时考虑买入。

接下来，我们实现JS加速线的计算。

```python
# 导入技术分析库
import technical_analysis as ta

![](img/f017b2531c1af843f3e2767918ade7bf_3.png)

# 定义单只股票代码
security = '000001'  # 例如平安银行
# 设置计算日期
check_date = '2022-09-01'
# 计算JS指标，参数N=5, M1=5, M2=10, M3=20
js, ma_js1, ma_js2, ma_js3 = ta.JS(security, check_date, N=5, M1=5, M2=10, M3=20)

# 输出JS指标的四根线数值
print(f"股票 {security} 在 {check_date} 的JS指标：")
print(f"  JS线: {js}")
print(f"  JS-5日均线(MAJS1): {ma_js1}")
print(f"  JS-10日均线(MAJS2): {ma_js2}")
print(f"  JS-20日均线(MAJS3): {ma_js3}")
```

运行代码后，我们将得到JS线及其三条均线的值。通过观察这些线之间的交叉关系（尤其是两根以上的交叉），可以判断买入或卖出的时机。

## 📝 本章小结

本节课中我们一起学习了趋向指标中的GDX轨道线和JS加速线。

*   **GDX轨道线**：由济安线、压力线和支撑线三线构成。压力线和支撑线是济安线的等比例偏移。应用时，股价触及压力线可卖出，触及支撑线可买入。
*   **JS加速线**：用于衡量股价变化速度，由JS线及其5日、10日、20日移动平均线共四根线组成。当JS线向下交叉多根均线时卖出，向上交叉时买入。
*   **代码实战**：我们使用`technical_analysis`库实现了这两个指标的计算。GDX函数主要参数为`N`和`M`，JS函数主要参数为`N`、`M1`、`M2`、`M3`。
