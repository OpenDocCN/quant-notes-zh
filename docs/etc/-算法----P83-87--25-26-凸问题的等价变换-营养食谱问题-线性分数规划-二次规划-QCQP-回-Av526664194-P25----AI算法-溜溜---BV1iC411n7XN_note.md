# 量化交易与Python金融分析实战：P25：凸问题的等价变换与二次规划

在本节课中，我们将深入探讨凸优化中的两个重要概念：线性分数规划的等价性证明，以及二次规划（QP）和二次约束二次规划（QCQP）的基本形式与应用。我们将通过严谨的数学推导和直观的几何解释，帮助你理解这些核心优化问题的本质。

## 线性分数规划等价性的进一步证明

上一节我们介绍了线性分数规划到线性规划的等价变换。有同学提出，若将变换后问题的约束直接改为 **Z > 0**，则两个问题完全等价。这个观点是正确的。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_1.png)

然而，标准线性规划形式通常使用 **Z ≥ 0** 而非 **Z > 0**。下面我们将证明，即使将约束放宽为 **Z ≥ 0**，原线性分数规划问题与变换后的线性规划问题仍然是等价的。关键在于证明当 **Z = 0** 时，两者依然等价。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_3.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_5.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_7.png)

### 当 Z = 0 时的情形分析

![](img/87cd93a11b81cf18a90c6ecae15fec9f_8.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_10.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_12.png)

当 **Z = 0** 时，我们无法直接通过之前的变换构造原问题的解。但我们知道原问题 **P0** 的可行解集非空，因此必然存在一个可行解 **X₀**。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_14.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_16.png)

我们需要证明：如果 **(Y, Z)** 是变换后问题 **P1** 的可行解，且 **Z = 0**，那么我们可以构造出原问题 **P0** 的可行解。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_18.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_20.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_22.png)

构造方式如下：令 **X = X₀ + tY**，其中 **t** 是任意大于等于零的实数。这意味着，如果我们能找到 **P1** 的一个可行解 **(Y, 0)**，那么从 **X₀** 出发，沿 **Y** 方向的整条射线上的点都是 **P0** 的可行解。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_24.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_26.png)

以下是验证过程：

![](img/87cd93a11b81cf18a90c6ecae15fec9f_28.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_30.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_32.png)

首先，由于 **(Y, 0)** 是 **P1** 的可行解，它满足以下条件：
1.  **GY ≤ 0** （因为 Z=0）
2.  **AY = 0**
3.  **eᵀY = 1** （其中 e 是全1向量）

![](img/87cd93a11b81cf18a90c6ecae15fec9f_33.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_34.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_36.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_38.png)

现在，验证构造出的 **X = X₀ + tY** 是否为 **P0** 的可行解：
*   **不等式约束**：`GX = GX₀ + tGY`。已知 `GX₀ ≤ h`（因 X₀ 可行），且 `GY ≤ 0`，`t ≥ 0`，故 `tGY ≤ 0`。因此 `GX ≤ h`。
*   **等式约束**：`AX = AX₀ + tAY = b + t*0 = b`。
*   **分母约束**：`cᵀX + d = (cᵀX₀ + d) + t(cᵀY)`。已知 `cᵀX₀ + d > 0`（因 X₀ 可行），且 `t ≥ 0`，`cᵀY` 为常数，故对于充分大的 t，该项仍大于零。实际上，只要 `t` 有限，该和也大于零。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_40.png)

因此，整条射线上的点 **X** 都满足 **P0** 的所有约束，即是其可行解。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_42.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_44.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_46.png)

### 目标函数值的极限等价性

![](img/87cd93a11b81cf18a90c6ecae15fec9f_48.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_50.png)

我们的最终目标是证明，存在射线上的某个点（或极限点），其目标函数值等于变换后问题在 **(Y, 0)** 处的目标函数值 **cᵀY**。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_52.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_54.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_56.png)

计算射线上的点 **X** 对应的原目标函数值：
`f₀(X) = (aᵀX + b) / (cᵀX + d)`

![](img/87cd93a11b81cf18a90c6ecae15fec9f_58.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_60.png)

将 `X = X₀ + tY` 代入，并考虑当 `t → ∞` 时的极限：
`lim_{t→∞} f₀(X₀ + tY) = lim_{t→∞} [ (aᵀX₀ + b + t(aᵀY)) / (cᵀX₀ + d + t(cᵀY)) ]`

![](img/87cd93a11b81cf18a90c6ecae15fec9f_62.png)

由于 `t` 趋于无穷大，分子和分母中的常数项 `(aᵀX₀ + b)` 和 `(cᵀX₀ + d)` 成为无穷小量。又因为 `cᵀY = 1`（来自约束 `eᵀY = 1` 及变换定义），分母的主导项为 `t`。分子中，只有 `t(aᵀY)` 项能与分母的 `t` 同级。

因此，该极限等于：
`lim_{t→∞} [ (aᵀY) / (cᵀY) ] = aᵀY / 1 = aᵀY`

![](img/87cd93a11b81cf18a90c6ecae15fec9f_64.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_66.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_68.png)

而这正是变换后线性规划问题在点 **(Y, 0)** 处的目标函数值。由此得证，即使 **Z = 0**，两个问题的最优值也是等价的。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_70.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_72.png)

## 二次规划（QP）简介

讲完线性（一次）规划，我们接下来看另一类重要的凸优化问题：**二次规划**。这类问题的目标函数是二次的。

一个二次规划的标准形式如下：

**最小化：** `(1/2) xᵀPx + qᵀx + r`
**约束条件：** `Gx ≤ h`, `Ax = b`

其中，要求目标函数是凸函数。对于二次函数 `(1/2) xᵀPx + qᵀx + r`，其为凸函数的充要条件是矩阵 **P** 是半正定矩阵（**P ≥ 0**）。同时，其不等式约束和等式约束都是仿射的（即线性的）。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_76.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_78.png)

简而言之，**二次规划就是目标函数为二次凸函数，且所有约束均为线性约束的优化问题**。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_80.png)

### 几何直观与线性规划的区别

![](img/87cd93a11b81cf18a90c6ecae15fec9f_82.png)

我们可以像分析线性规划一样画出二次规划的图像。约束集合仍然是一个多面体。在二维情况下，如果 **P** 是正定矩阵，目标函数的等高线是一系列椭圆（对于凸函数，等高线间距向内逐渐变密）。

那么最优解在哪里？与线性规划不同，二次规划的最优解不一定出现在约束多面体的顶点上。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_84.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_85.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_87.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_88.png)

考虑一个例子：如果无约束二次函数的最小值点本身就位于线性约束集合的内部，那么该点就是带约束二次规划的最优解。**这意味着二次规划的最优解可以出现在可行域的内部**。而线性规划的最优解则必然出现在可行域的顶点或边缘上，这是两者一个重要的区别。

## 二次约束二次规划（QCQP）

![](img/87cd93a11b81cf18a90c6ecae15fec9f_99.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_101.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_103.png)

有同学可能会问，如果约束中的不等式也是二次凸函数，那它不也是凸问题吗？为什么不能叫二次规划？

![](img/87cd93a11b81cf18a90c6ecae15fec9f_105.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_107.png)

确实，这类问题也是凸问题，但它有专门的名称：**二次约束二次规划**，英文通常简称为 **QCQP**。它并非二次规划的特例，而是一类更广泛的问题。

![](img/87cd93a11b81cf18a90c6ecae15fec9f_109.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_110.png)

QCQP 的标准形式如下：

![](img/87cd93a11b81cf18a90c6ecae15fec9f_111.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_112.png)

![](img/87cd93a11b81cf18a90c6ecae15fec9f_114.png)

**最小化：** `(1/2) xᵀP₀x + q₀ᵀx + r₀`
**约束条件：** `(1/2) xᵀPᵢx + qᵢᵀx + rᵢ ≤ 0, for i = 1, ..., m`
`Ax = b`

其中，要求所有矩阵 **Pᵢ** (i=0,1,...,m) 都是半正定的，以保证目标函数和所有不等式约束函数都是凸的。

### QCQP 的几何直观

我们可以画出QCQP的示意图。目标函数的等高线仍是椭圆。而不等式约束 `(1/2) xᵀPᵢx + qᵢᵀx + rᵢ ≤ 0` 定义的可行域可能是一个椭圆区域（或其它二次曲面围成的区域）。

最优解通常出现在目标函数等高线与约束边界椭圆的切点处。由于可行域边界是光滑曲线，不存在像多面体那样的“顶点”，这使得QCQP的求解在理论上比线性约束的二次规划（QP）更为复杂。

## 实际应用举例：信号恢复与正则化

下面我们通过信号处理中的经典例子，看看QP和QCQP如何应用于实际。

### 1. 最小二乘估计（无约束QP）

考虑一个带噪声的测量系统：`b = Ax + e`。其中 `x` 是待估计信号，`A` 是测量矩阵，`e` 是噪声，`b` 是观测值。

最常见的恢复方法是**最小二乘法**，即最小化误差的平方和：
**最小化：** `||Ax - b||₂²`

这显然是一个无约束的二次优化问题。展开目标函数：
`f(x) = xᵀ(AᵀA)x - 2bᵀAx + bᵀb`
矩阵 `AᵀA` 总是半正定的，因此这是一个无约束的二次规划（凸）。若 `AᵀA` 可逆（正定），则有解析解：`x* = (AᵀA)⁻¹Aᵀb`。

### 2. Lasso 问题（可转化为QP）

如果我们已知信号 `x` 是稀疏的（即大部分元素为零），可以在最小二乘中增加一个 **L1正则化** 项，形成 **Lasso** 问题：
**最小化：** `||Ax - b||₂² + λ||x||₁`，其中 `λ ≥ 0`。

这里 `||x||₁` 是L1范数（绝对值之和），它是非光滑、非凸的。但我们可以通过一个技巧将其转化为标准的二次规划。

**转化技巧**：将变量 `x` 分解为正部 `x₊` 和负部 `x₋`，即 `x = x₊ - x₋`，并约束 `x₊ ≥ 0`, `x₋ ≥ 0`。此时，`||x||₁ = sum(x₊) + sum(x₋)`。

将 `x = x₊ - x₋` 和 `||x||₁ = 1ᵀx₊ + 1ᵀx₋` 代入原问题，得到：
**最小化：** `||A(x₊ - x₋) - b||₂² + λ(1ᵀx₊ + 1ᵀx₋)`
**约束条件：** `x₊ ≥ 0`, `x₋ ≥ 0`

![](img/87cd93a11b81cf18a90c6ecae15fec9f_116.png)

这是一个关于新变量 `(x₊, x₋)` 的**二次规划问题**，因为目标函数是二次的，约束是线性的。这个分解技巧对于处理L1范数非常有效。

### 3. 岭回归（Ridge Regression）及其等价形式（QP与QCQP）

如果我们希望信号 `x` 的各元素幅度相对均匀，而非稀疏，可以使用 **L2正则化**，即 **岭回归**：
**最小化：** `||Ax - b||₂² + λ||x||₂²`，其中 `λ > 0`。

这显然是一个无约束的二次规划。由于增加了 `λ||x||₂²` 项，即使 `AᵀA` 不可逆，新的海森矩阵 `AᵀA + λI` 也是正定的，保证了问题的强凸性。

此外，岭回归有一个等价的约束形式：
**最小化：** `||Ax - b||₂²`
**约束条件：** `||x||₂² ≤ t`

对于每一个 `λ > 0`，都存在一个 `t ≥ 0`，使得这两个问题的最优解相同。第二个问题的目标函数是二次的，约束是二次的，因此它是一个**二次约束二次规划（QCQP）**。这种等价性在优化理论中可以通过对偶性来证明。

## 总结

本节课我们一起深入学习了凸优化中的两个核心板块：
1.  **线性分数规划的等价性**：我们完成了严格的证明，展示了如何通过变量替换和极限分析，将线性分数规划等价转化为标准的线性规划，即使考虑 `Z ≥ 0` 的情形。
2.  **二次规划及其扩展**：我们介绍了二次规划（QP）和二次约束二次规划（QCQP）的基本形式、几何意义及其与线性规划的区别。关键点在于QP的最优解可在可行域内部，而LP的最优解总在边界。
3.  **实际应用与转化技巧**：通过信号恢复的例子，我们看到了最小二乘（无约束QP）、Lasso（通过变量分解转化为QP）以及岭回归（无约束QP或其等价QCQP形式）这些重要模型如何纳入凸优化框架。其中，将L1范数通过 `x = x₊ - x₋` 分解转化为线性约束的技巧尤为重要。

理解这些基本问题的形式和转化方法，是构建和解决更复杂金融量化模型（如投资组合优化、风险控制模型等）的重要基础。
