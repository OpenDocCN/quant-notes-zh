## 核心操作与函数定义
上一节我们介绍了去极值的概念，本节中我们来看看如何具体实现基于百分位的去极值方法。核心思路是计算数据的下四分位数（Q1）和上四分位数（Q3），然后将小于Q1的值替换为Q1，大于Q3的值替换为Q3。

![](img/58d0b768eef33935b754fcc77a926b4d_7.png)

![](img/58d0b768eef33935b754fcc77a926b4d_9.png)

以下是实现该功能的核心函数：

```python
import pandas as pd
import numpy as np

![](img/58d0b768eef33935b754fcc77a926b4d_11.png)

![](img/58d0b768eef33935b754fcc77a926b4d_13.png)

def winsorize_by_percentile(series, low_percentile=0.25, high_percentile=0.75):
    """
    基于百分位对序列进行去极值处理（缩尾处理）。
    
    参数:
    series (pd.Series): 待处理的序列数据。
    low_percentile (float): 下百分位，默认0.25（Q1）。
    high_percentile (float): 上百分位，默认0.75（Q3）。
    
    返回:
    pd.Series: 处理后的序列。
    """
    # 计算指定百分位的值
    q = series.quantile([low_percentile, high_percentile])
    low_bound = q.iloc[0]  # 下界
    high_bound = q.iloc[1] # 上界
    
    # 使用np.clip将超出边界的值缩尾至边界
    winsorized_series = np.clip(series, low_bound, high_bound)
    
    return winsorized_series
```

## 分步解析
接下来，我们详细解析上述函数的每一步操作。

![](img/58d0b768eef33935b754fcc77a926b4d_19.png)

### 第一步：数据排序与分位数计算
为了找到数据的Q1和Q3，我们首先需要理解数据分布。`pandas.Series.quantile()` 函数可以直接帮助我们计算任意百分位的值，无需手动排序。

![](img/58d0b768eef33935b754fcc77a926b4d_21.png)

**公式**：`Q1 = 数据序列的25%分位数`， `Q3 = 数据序列的75%分位数`。

在代码中，我们通过 `series.quantile([0.25, 0.75])` 一次性计算出这两个边界值。

![](img/58d0b768eef33935b754fcc77a926b4d_23.png)

### 第二步：应用边界进行缩尾
获得边界值（`low_bound`, `high_bound`）后，我们需要将原始数据中所有小于下界的值设置为下界，所有大于上界的值设置为上界。`numpy.clip()` 函数完美实现了这个功能。

![](img/58d0b768eef33935b754fcc77a926b4d_25.png)

**公式**：`处理后的值 = max(min(原始值, high_bound), low_bound)`

![](img/58d0b768eef33935b754fcc77a926b4d_27.png)

这行代码 `np.clip(series, low_bound, high_bound)` 即完成了此操作。

## 参数调整与实际应用建议
在之前的例子中，我们使用了25%和75%作为分位点，这会将相当一部分数据视为“极值”并进行调整。

![](img/58d0b768eef33935b754fcc77a926b4d_47.png)

以下是实际应用中更常见的参数设置建议：
*   **更严格的范围**：在实际量化因子分析中，通常会使用更窄的范围，例如 **2.5% 和 97.5%** 分位点，这样只对真正极端的两端数据进行处理。
*   **调整方法**：只需在调用函数时传入自定义参数即可：
    ```python
    # 使用2.5%和97.5%分位点进行去极值
    processed_series_strict = winsorize_by_percentile(target_series, 0.025, 0.975)
    ```

![](img/58d0b768eef33935b754fcc77a926b4d_51.png)

选择何种参数取决于具体策略对数据异常值的容忍度，需要在回测中谨慎验证。
