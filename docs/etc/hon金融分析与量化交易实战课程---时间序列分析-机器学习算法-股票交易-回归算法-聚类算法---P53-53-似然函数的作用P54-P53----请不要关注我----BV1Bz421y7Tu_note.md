# 量化交易实战课程：P53-P54：似然函数的作用

![](img/4923672d3500615f1a94ead18a4b2c7d_0.png)

## 概述
在本节课程中，我们将学习线性回归中的一个核心概念——**似然函数**。我们将探讨如何利用似然函数来求解模型的最佳参数，并理解其背后的数学原理。课程将从误差项的高斯分布假设出发，逐步推导出似然函数，并解释如何通过取对数将其转化为更易求解的形式。

---

## 误差项与高斯分布
上一节我们介绍了线性回归模型 `Y = C^T X + ε` 以及误差项 `ε` 的概念。本节中我们来看看如何利用误差项的统计特性来求解参数 `C`。

我们已知数据 `X` 和标签 `Y`，目标是找到最优的参数 `C`。我们假设误差项 `ε` 服从均值为0的高斯分布（正态分布）。高斯分布的概率密度函数公式如下：

`P(ε) = (1 / √(2πσ²)) * exp(-ε² / (2σ²))`

由于我们假设误差的均值为0，因此公式中省略了减去均值 `μ` 的步骤。

---

## 从误差到参数：构建似然函数
上述高斯分布是关于误差项 `ε` 的。然而，我们最终关心的是参数 `C`，而非具体的误差值。因此，我们需要建立一个与 `C` 相关的函数。

![](img/4923672d3500615f1a94ead18a4b2c7d_2.png)

根据模型 `Y = C^T X + ε`，我们可以得到 `ε = Y - C^T X`。将这个关系代入高斯分布公式，我们就得到了一个关于参数 `C` 和数据 `(X, Y)` 的函数。

这个函数的直观解释是：**它描述了在给定参数 `C` 和数据 `X` 的情况下，观测到标签 `Y` 的可能性（概率）**。我们希望找到一组参数 `C`，使得这个可能性（概率）**最大化**。因为这意味着我们的模型预测值 `C^T X` 与真实值 `Y` 最接近。

---

## 理解似然函数：一个类比
为了帮助理解，我们可以举一个例子。假设你去澳门赌场，想试试手气但不敢先玩，于是先观察别人。

*   你观察到第一位玩家赢了。
*   你观察到第二位玩家也赢了。
*   接着第三、第四、第五位玩家都赢了。

赌场的输赢由一组内在的“参数”控制。你观察到所有样本（五位玩家）都赢了，你就会倾向于认为这组参数使得“赢”的概率很高，进而预测自己接下来玩也会赢。

似然函数描述的就是类似的过程：**寻找一组参数，使得这组参数与我们的观测数据组合后，恰好得到我们观测到的结果的可能性最大**。参数 `C` 是固定的（待求的），数据 `(X, Y)` 也是固定的（已知的）。

## 似然函数的数学形式：为什么是累乘？
以下是似然函数 `L(C)` 的核心形式及其解释：

`L(C) = ∏ (i=1 to m) P(Y_i | X_i; C)`

**为什么是累乘（∏）？**
1.  **数据独立性**：我们通常假设每个数据样本 `(X_i, Y_i)` 是独立同分布的。在独立性的前提下，所有样本同时出现的**联合概率**等于每个样本出现的**边缘概率**的乘积。因此，为了计算所有数据出现的总可能性，我们使用累乘。
2.  **数据越多越可信**：就像赌场的例子，观察到的赢家样本越多（`m` 越大），你对“参数利于赢”的信念就越强。累乘机制放大了大量数据一致性的效果。

---

## 从乘法到加法：引入对数似然
虽然似然函数的概念很清晰，但直接求解它非常困难。因为它是 `m` 个概率值的乘积。当 `m` 很大时（例如1000），直接处理一个干项连乘的表达式几乎是不可能的。

相比之下，加法运算则容易处理得多。数学上，我们可以通过对数运算将乘法转换为加法：

`log(A * B) = log(A) + log(B)`

因此，我们对似然函数取自然对数，得到**对数似然函数** `log L(C)`：

`log L(C) = ∑ (i=1 to m) log P(Y_i | X_i; C)`

**一个关键问题**：取对数后，函数值变了，我们求出的解还正确吗？
答案是肯定的。因为我们最终的目标是找到使函数值最大的参数 `C`，即寻找**极值点**，而非具体的极值大小。对原函数进行单调递增变换（如取对数），不会改变极值点的位置。因此，最大化 `L(C)` 与最大化 `log L(C)` 是等价的。

通过取对数，我们将复杂的连乘问题转化为了简单的求和问题，极大降低了后续求解的难度。

## 总结
本节课中我们一起学习了：
1.  **似然函数 `L(C)` 的定义**：它表示在给定参数 `C` 下，观测到现有数据的概率。
2.  **似然函数的作用**：我们的目标是找到使 `L(C)` 最大化的参数 `C`，这组参数能使模型的预测最贴合真实数据。
3.  **似然函数的形式**：基于误差项独立同分布的高斯假设，似然函数是每个样本概率的累乘。
4.  **对数似然函数**：为了便于计算，我们对似然函数取对数，将乘法问题转化为加法问题，且不改变极值点的位置，即 `argmax L(C) = argmax log L(C)`。
