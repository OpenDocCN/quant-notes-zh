# 量化金融与机器学习：P30：信用风险的IRC模型与方差缩减技术

在本节课中，我们将学习信用风险建模的核心——IRC模型，并深入探讨几种用于提升蒙特卡洛模拟效率的方差缩减技术。我们将从高斯Copula模型出发，理解信用违约的相关性，然后学习重要性采样、低差异序列等方法来优化计算。

## 计算信用违约损失

上一节我们介绍了信用风险的基本概念，本节中我们来看看如何具体计算一个资产组合的信用违约损失。例如，我们持有一个包含100个发行者的CDO。我们可以将这100个发行者视为100只债券。

这些债券不一定是国债，可能还包括企业债、公司债，或是一些地方政府发行的债券。我们的目标是计算它们组合起来可能带来的预期信用违约损失。

我们定义 **xi** 为第 **i** 个发行者的违约指示因子，其计算公式如下：

`xi = sqrt(ρ) * M + sqrt(1 - ρ) * εi`

其中：
*   **ρ** 是不同发行者之间的相关系数。理论上，每对发行者之间的相关性都不同，但为了计算方便，我们通常假设所有配对之间的 **ρ** 是相同的。
*   **M** 被称为 **系统风险因子**。对于每一条模拟路径，所有发行者的 **M** 值都相同。
*   **εi** 被称为 **特质风险因子**。它代表了每个发行者独有的、区别于其他发行者的风险因素。例如，对于第一个用户和第二个用户，他们的 **ε1** 和 **ε2** 完全不同，并且在不同的模拟路径中，这些值也不同。

![](img/eeb785dbc25c5ded53922c9f701a5049_1.png)

**xi** 和 **xj** 之间的相关性完全由公共的 **M** 决定，因为特质因子 **ε** 之间是相互独立的。因此，它们的协方差就是 **ρ**。当相关性 **ρ** 越高时，违约事件就越极端：要么大家都不违约，要么大家一起违约。这导致预期损失的分布会非常极端，集中在“零损失”和“最大损失”两端。

![](img/eeb785dbc25c5ded53922c9f701a5049_1.png)

## 相关性违约的计算过程

接下来，我们具体看看相关性违约是如何计算的。

首先，我们从标准正态分布中抽取系统风险因子 **M**。然后，对于每一个发行者 **i**，我们使用相同的 **M** 和独立抽取的特质因子 **εi**，通过公式 `xi = sqrt(ρ) * M + sqrt(1 - ρ) * εi` 计算出 **xi**。计算出的 **xi** 本身也服从正态分布。

接着，我们计算 **xi** 的概率值 **Fi**，即标准正态分布的累积分布函数值。我们将 **Fi** 与一个预设的 **违约阈值 pi** 进行比较。如果 **Fi < pi**，则认为该发行者违约；反之则不违约。

![](img/eeb785dbc25c5ded53922c9f701a5049_3.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_5.png)

这里存在一个问题：如果违约概率 **pi** 非常小（例如0.01），那么 **xi** 很难落入导致违约的极端值区域。因为正态分布下，大部分 **xi** 的值都集中在均值附近。

![](img/eeb785dbc25c5ded53922c9f701a5049_7.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_3.png)

为了解决这个问题，我们可以对公共因子 **M** 进行平移，应用之前提到过的 **重要性采样** 技术。

![](img/eeb785dbc25c5ded53922c9f701a5049_5.png)

## 在IRC模型中应用重要性采样

重要性采样技术主要应用于 **IRC模型**。IRC模型用于预测在一年内、99.9%置信度下的投资组合信用损失分位数，这是信用风险资本金计算的核心。该模型假设投资组合在一年内保持不变。

让我们将其具体化：假设我们购买了100份信用违约互换。总名义本金为1000万。我们假设前5个发行者的一年生存概率为0.95，其余为0.99。因此，它们的违约概率分别为0.05和0.01（后者属于罕见事件）。回收率假设服从10%到50%之间的均匀分布。回收率是指违约发生后，能追回的本金比例。我们使用一元高斯Copula模型，并设相关系数 **ρ = 70%**。

![](img/eeb785dbc25c5ded53922c9f701a5049_9.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_9.png)

我们可以通过平移系统风险因子 **M** 来进行重要性采样。其核心思想与之前正态分布的例子完全相同。平移的方向和距离由参数 **d** 控制。

在我们的Python示例中，我们通过网格搜索来寻找最优的 **d** 值，范围从-2.5到0，步长为0.25。结果发现，当 **d ≈ -1.5** 时，方差缩减比率最高，效率提升了52倍，这是一个非常优秀的结果。同时，我们也计算出了IRC模型下的预期总损失。虽然寻找最优 **d** 值需要额外时间，但整体上仍节省了大量计算时间。

为什么能降低方差？原理与正态分布的例子类似。在标准的蒙特卡洛模拟中，信用违约（罕见事件）发生的概率很低。通过平移 **M**，我们将更多的模拟路径引导至我们感兴趣的“违约区域”。在未使用重要性采样的标准模拟中，模拟200次可能没有一次触及违约区域；而使用后，我们能够更有效地捕捉到这些极端事件。

## 低差异序列方法

![](img/eeb785dbc25c5ded53922c9f701a5049_13.png)

接下来，让我们看看另一种独立的方法：**低差异序列**。

![](img/eeb785dbc25c5ded53922c9f701a5049_15.png)

蒙特卡洛模拟不准确的根源之一在于随机数的生成质量。例如，在[0,1]区间内生成200个随机样本点，其分布可能并不均匀，有些区域空白，有些区域则点过于密集。

![](img/eeb785dbc25c5ded53922c9f701a5049_17.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_17.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_19.png)

我们希望降低整个点集的 **差异度**，让点尽可能均匀分布。

![](img/eeb785dbc25c5ded53922c9f701a5049_21.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_19.png)

首先定义差异度 **D**。其表达式衡量了在任意一个子区间（或超立方体）B内，实际落点比例与理想均匀分布下应占面积比例的绝对差的最大值。理想情况下，差异度应为零。我们希望差异度越小越好。

![](img/eeb785dbc25c5ded53922c9f701a5049_23.png)

一些研究机构创造了特定的低差异序列，例如 **Sobol序列**。它易于实现，只需替换掉我们生成随机数时所用的均匀分布发生器即可。其原理是将[0,1]区间按二进制方式不断细分（如先取0.5，再取0.25，0.75，再取1/8，1/16的倍数等），直到覆盖整个区间，从而生成极其均匀的序列。生成均匀序列点后，我们再通过正态分布反函数得到最终的样本值。

![](img/eeb785dbc25c5ded53922c9f701a5049_23.png)

接下来看一个使用低差异序列为布朗运动定价的例子，比如为亚式期权定价。比较计算时间和误差，可以发现使用低差异序列后，误差降低了近十倍。这个例子使用的是Sobol序列。

具体调用方式如下：我们需要从网络（如斯坦福大学官网）下载 `sobol_seq` 库。导入后，调用 `sobol_seq.i4_sobol_generate` 函数来生成二维空间的500个样本点。

![](img/eeb785dbc25c5ded53922c9f701a5049_27.png)

下图展示了普通蒙特卡洛（左）与Sobol序列（右）生成的点集，可见右图分布更为均匀。

![](img/eeb785dbc25c5ded53922c9f701a5049_29.png)

生成序列后，将其代入亚式期权定价的蒙特卡洛函数中，即可比较两种方法计算出的价格。

## 其他方差缩减技术

![](img/eeb785dbc25c5ded53922c9f701a5049_27.png)

我们再介绍两种简单易实现的方法。

![](img/eeb785dbc25c5ded53922c9f701a5049_29.png)

第一种是 **矩匹配**。其思路非常直接：我们本意是生成一个均值为0、方差为1的标准正态分布样本，但由于生成的样本数量有限（如200个），其样本均值和方差可能并非精确的0和1。我们可以通过计算初始样本的均值 `m` 和标准差 `s`，然后对每个样本点进行变换：`(x - m) / s`，从而得到一个更接近标准正态分布的样本集。

![](img/eeb785dbc25c5ded53922c9f701a5049_31.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_36.png)

第二种是 **分层抽样**。其思想与低差异序列类似：如果我们要抽取N个样本，就将[0,1]区间均匀分成N层，然后在每一层内随机抽取一个点。接着，通过正态分布反函数将这些均匀分布的点转换为目标分布的样本点。

![](img/eeb785dbc25c5ded53922c9f701a5049_33.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_34.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_38.png)

例如，要生成2000个标准正态样本。未使用任何技术时，其直方图分布与理论正态分布略有差异。

![](img/eeb785dbc25c5ded53922c9f701a5049_36.png)

![](img/eeb785dbc25c5ded53922c9f701a5049_40.png)

使用分层抽样后，分布变得光滑许多。
