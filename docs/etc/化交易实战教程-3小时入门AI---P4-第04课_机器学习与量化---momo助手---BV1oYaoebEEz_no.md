## 从传统编程到机器学习

上一节我们介绍了数据获取与处理的基础。本节中我们来看看机器学习与传统编程在构建交易策略上的根本区别。

在传统的编程方法中，策略规则是由程序员或交易员人工定义的。例如，观察到移动平均线（MA5）上穿MA10后股票倾向于上涨，就将此规则编写成 `if-else` 语句。程序根据输入的指标（如MACD、MA5、MA10）判断并输出“看涨”或“看跌”信号。

机器学习则不同，其规则并非由人工制定，而是从数据中自动学习出来的。你只需要向程序提供大量“输入-输出”对（例如，当指标是这样时，结果是涨；是那样时，结果是跌），程序就能自动挖掘出一系列规则。这种方法的好处在于，计算机能够处理海量、高维的数据，并可能发现人类难以直观观察到的隐藏关系。

## 机器学习的三大类型

以下是机器学习的主要分类：

1.  **监督学习**：输入数据包含特征（X）和对应的标签（Y）。Y可以是一个具体的数值（回归问题），也可以是一个类别（分类问题，如“看涨”/“看跌”）。监督学习的目标是学习一个从X到Y的映射函数。
2.  **非监督学习**：输入数据只有特征（X），没有标签。其目标是发现数据内部的规律，例如将相似的股票聚为一类（聚类），或将高维特征压缩到低维空间以便可视化（降维）。
3.  **强化学习**：智能体（Agent）与环境（如股票市场）交互。智能体根据当前状态（State）采取行动（Action，如买入/卖出），环境则给予奖励（Reward）。其目标是学习一系列行动策略，以最大化长期累积奖励。其理论基础是马尔可夫决策过程（MDP）。

![](img/8d7dd39a748a36ef1f8d4417a5a6822e_2.png)

在量化交易中，监督学习（用于预测价格或涨跌）和非监督学习（用于发现股票聚类或降维）都非常常见。

## Scikit-learn 简介与使用流程

![](img/8d7dd39a748a36ef1f8d4417a5a6822e_4.png)

Scikit-learn（简称sklearn）是Python中一个功能强大且易用的机器学习库。除了深度神经网络，它涵盖了大多数经典的机器学习算法。

一个典型的sklearn建模流程包含以下三步：

1.  **准备数据**：获取并划分训练集和测试集。`X_train` 通常是一个 `(n_samples, n_features)` 的矩阵，`y_train` 是对应的标签向量或矩阵。务必在建模前使用 `X.shape` 和 `y.shape` 检查数据维度。
2.  **训练模型**：选择一个模型（如线性回归、SVM），创建模型实例，然后调用 `model.fit(X_train, y_train)` 方法进行训练。这个过程会学习出模型参数的最优组合。
3.  **评估模型**：使用训练好的模型对测试集进行预测 `model.predict(X_test)`，然后将预测结果 `y_pred` 与真实标签 `y_test` 进行比较，使用评分函数（如准确率、均方误差）评估模型性能。

**核心代码流程**：
```python
# 1. 准备数据
X_train, X_test, y_train, y_test = get_data(...) # 自定义函数
print(X_train.shape, y_train.shape)

# 2. 训练模型
from sklearn.linear_model import LinearRegression
model = LinearRegression() # 实例化模型
model.fit(X_train, y_train) # 训练模型

# 3. 评估模型
y_pred = model.predict(X_test) # 预测
score = model.score(X_test, y_test) # 评分
print(f"模型评分： {score}")
```

**重要提示**：模型在测试集上的评分（Score）好，并不等同于在回测（Backtest）中表现好，更不等于实盘能盈利。评分是模型筛选的第一步，但远非最终验证。

## 从线性回归到高级线性模型

本节我们将深入探讨线性模型家族，这是量化实践中应用非常广泛且有效的一类模型。

### 线性回归（OLS）

线性回归假设输出Y与输入X之间存在线性关系，其模型表示为：

**公式**：`Y = X · θ`

其中，`X` 是 `(n_samples, n_features)` 的特征矩阵，`θ` 是 `(n_features, )` 的参数向量。线性回归的目标是找到一组参数 `θ`，使得预测值与真实值之间的平方误差最小（最小二乘法）。

**目标函数（损失函数）**：
`J(θ) = (Y - Xθ)^T (Y - Xθ)`

通过令 `J(θ)` 对 `θ` 的偏导数为零，可以得到参数的闭式解（解析解）：

**公式**：`θ = (X^T X)^(-1) X^T Y`

### 领回归（Ridge Regression）

线性回归在特征多重共线性或数据病态时可能不稳定。领回归在线性回归的损失函数中增加了一个L2正则化项（惩罚项），旨在限制参数 `θ` 的大小，防止过拟合。

**目标函数**：
`J(θ) = (Y - Xθ)^T (Y - Xθ) + α * Σ(θ_i^2)`
其中，`α` 是控制正则化强度的超参数（`α >= 0`）。

**几何解释**：领回归等价于在最小化平方误差的同时，要求参数向量 `θ` 的L2范数（平方和）小于某个常数。这相当于在参数空间中，寻找平方误差等高线与L2范数约束圆的切点。通过调整 `α`，可以控制解从原点（全零）向线性回归最优解移动。领回归会使所有参数都缩小（Shrinkage），但通常不会将任何参数精确变为零。

### 拉索回归（Lasso Regression）

拉索回归与领回归类似，但使用L1正则化项。它同样可以防止过拟合，并具备特征选择的能力。

**目标函数**：
`J(θ) = (Y - Xθ)^T (Y - Xθ) + α * Σ|θ_i|`

**几何解释**：L1正则化的约束区域是一个菱形。与平方误差等高线相切时，切点更容易落在坐标轴上，这意味着某些特征对应的参数 `θ_i` 会精确地变为零。因此，拉索回归能够自动进行特征选择，产生稀疏的模型。

### 从线性到非线性：核方法（Kernel Method）/ 径向基函数（RBF）

如何用线性模型拟合非线性关系？一个巧妙的方法是对输入特征 `X` 进行非线性变换，然后在变换后的高维空间中做线性回归。

**核方法（以RBF为例）**：我们不再直接使用原始特征 `X`，而是定义一组“基函数”（例如高斯函数），将原始 `X` 映射到这些基函数上。模型输出 `Y` 被表示为这些基函数的线性组合。

**公式（单个基函数）**：`φ(x) = exp(-λ ||x - μ||^2)`
其中，`μ` 是基函数的中心点，`λ` 控制函数的宽度（胖瘦）。

**实际操作**：
1.  确定基函数的个数 `K` 和中心 `μ`。一种常见做法是对训练数据 `X` 进行K-means聚类，将 `K` 个聚类中心作为 `μ`。
2.  对于任意输入 `x`，计算它与 `K` 个中心 `μ` 的高斯核函数值，得到一个 `K` 维的新特征向量。
3.  对这个新的 `K` 维特征向量使用线性回归（通常配合正则化，如领回归）进行拟合。

通过这种方式，我们利用线性模型拟合了非线性关系。基函数的宽度 `λ` 和个数 `K` 是需要调节的超参数。

## 模型选择与交叉验证

上一节我们介绍了多种模型及其超参数。本节中我们来看看如何科学地选择模型和超参数，避免过拟合和欠拟合。

### 超参数选择的重要性

无论是领回归/拉索回归中的正则化强度 `α`，还是RBF中的宽度 `λ` 和中心个数 `K`，这些超参数的选择都至关重要。`α` 太大导致欠拟合（参数趋近于零），`α` 太小则可能导致过拟合。

### 交叉验证（Cross-Validation）流程

我们不能直接在测试集上选择超参数，否则会“窥探”测试集信息，导致评估结果过于乐观。标准的做法是使用验证集（Validation Set）。

以下是详细流程：

1.  **数据划分**：将全部数据划分为三部分：训练集（如70%）、验证集（如15%）、测试集（如15%）。
2.  **在验证集上选择超参数**：
    *   在训练集上，用不同的超参数组合（如 `α=0.1, 1, 10`）分别训练模型。
    *   将这些训练好的模型在**验证集**上进行评估，选择在验证集上性能最好的超参数组合。
3.  **重新训练与最终评估**：
    *   使用**选定的最佳超参数**，在**训练集+验证集**合并的数据上重新训练一个最终模型。
    *   用**测试集**对这个最终模型进行**一次**性能评估，此分数作为对外汇报的模型性能。
4.  **生产环境模型**：如果对测试集结果满意，在最终部署前，可以使用**全部数据**（训练+验证+测试）和最佳超参数，训练最终用于生产的模型。

对于多个超参数，可以使用网格搜索（Grid Search）暴力尝试所有组合，并通过热力图（Heatmap）可视化结果。Sklearn提供了 `GridSearchCV` 等工具来自动化这一过程。

## 定义量化模型的输入特征（X）

模型和评估方法都已就绪，最后也是最具创造性的环节是：如何为每个时刻 `t` 定义输入特征向量 `X_t`？

假设我们要预测下一时刻 `t+1` 的价格 `Y_{t+1}`。我们需要基于 `t` 时刻及之前的信息，构造一个特征向量 `X_t`。这些特征可以包括：

*   **价格信息**：过去N个时刻的收盘价、开盘价、最高价、最低价。
*   **衍生技术指标**：各种移动平均线（MA）、相对强弱指数（RSI）、布林带（Bollinger Bands）、MACD等。
*   **成交量信息**：过去N个时刻的成交量、成交额。
*   **其他数据**：新闻情绪指数、宏观经济指标等（如果可获得）。

**核心任务**：将你对市场的理解和洞察，转化为可计算的、数值化的特征。这是量化策略开发中“艺术”的部分，也是产生差异化策略的关键。

**实践建议**：可以从实现经典的技术指标函数库开始，逐步构建自己的特征池。
