## 数字类型 🔢

Python 3支持四种数字类型：整型、浮点型、布尔型和复数。我们首先来了解它们。

### 数字类型介绍

以下是Python中的四种数字类型：

*   **整型 (int)**：Python 3中的整型没有大小限制，可以当作长整型使用。例如：`i = 12`。
*   **浮点型 (float)**：由整数部分和小数部分组成，即使小数部分为0，只要包含小数点就是浮点型。例如：`f = 36.5` 或 `f2 = 36.`。
*   **布尔型 (bool)**：只有两个值：`True` 和 `False`。在Python 3中，`True` 的值为 `1`，`False` 的值为 `0`，它们可以和数字进行运算。例如：`b = True`。
*   **复数 (complex)**：由实数部分和虚数部分构成，可以用 `a + bj` 或 `complex(a, b)` 表示。例如：`c = 12j`。

### 数字类型转换

数字类型之间可以相互转换，转换方法是将目标数据类型作为函数名。

以下是数字类型转换的示例：
```python
# 定义一个浮点数
f = 36.5
# 转换为整数（向下取整）
int_num = int(f)  # 结果为 36

# 定义一个整数
i = 12
# 转换为浮点数
float_num = float(i)  # 结果为 12.0

# 转换为复数
complex_num = complex(i)  # 结果为 (12+0j)
```
> **注意**：类型转换函数（如 `int()`, `float()`）会返回转换后的结果，但不会改变原变量本身。如需改变原变量，需要将返回值重新赋值给它，例如：`f = int(f)`。

### 数字运算

Python支持基本的算术运算。以下是常见的运算符：

*   **加法**：`2 + 4 = 6`
*   **减法**：`2 - 4 = -2`
*   **乘法**：`2 * 4 = 8`
*   **除法**：`2 / 4 = 0.5`
*   **整除**：`2 // 4 = 0` （向下取整）
*   **取模（求余）**：`2 % 4 = 2`
*   **乘方**：`2 ** 4 = 16`

> **提示**：不同类型的数字混合运算时，Python会自动将整型转换为浮点型。例如 `2.0 + 4` 的结果是 `6.0`。

### 常用数学函数

Python内置了一些数学函数，也可以导入 `math` 模块进行更复杂的计算。

以下是部分常用函数：
```python
# 内置函数
abs(-2.4)   # 取绝对值，结果为 2.4
round(3.1415926, 2) # 四舍五入保留两位小数，结果为 3.14

# 使用 math 模块
import math
math.e          # 自然常数 e，约 2.71828
math.pi         # 圆周率 π
math.ceil(2.5)  # 向上取整，结果为 3
math.floor(2.5) # 向下取整，结果为 2
math.exp(1)     # e 的 1 次方，约 2.71828
math.log(10)    # 自然对数 ln(10)
math.sqrt(4)    # 平方根，结果为 2.0
```

## 字符串类型 📝

字符串是Python中非常重要且常用的数据类型。本节我们将学习字符串的创建、索引、切片、运算、方法及格式化。

### 字符串的创建

可以使用单引号、双引号或三引号来创建字符串。

以下是创建字符串的几种方式：
```python
# 单引号创建
s1 = 'Hello World'
# 双引号创建
s2 = "Hello World"
# 三引号创建多行字符串
s3 = '''Hello
World'''
s4 = """Hello
World"""
```
> **注意**：使用单引号或双引号时，前后引号必须匹配。三引号常用于创建多行字符串或作为多行注释。

### 字符串的索引和切片

字符串是序列类型，可以通过索引访问单个字符，通过切片访问子字符串。**索引从0开始**。

以下是索引和切片操作的示例：
```python
s = "Hello World"
# 索引
s[0]    # 获取第一个字符 'H'
s[-1]   # 获取最后一个字符 'd'

# 切片 [i:j]，遵循左闭右开原则
s[:2]   # 从开始到索引2（不含），结果为 'He'
s[2:4]  # 从索引2到4（不含），结果为 'll'
s[6:]   # 从索引6到末尾，结果为 'World'
```
> **关键点**：切片语法 `[i:j]` 表示获取从索引 `i` 到 `j-1` 的子串。字符串是**不可变**数据类型，无法通过索引直接修改其中的字符。

### 字符串的运算

字符串支持加法和乘法运算，以及成员运算符。

以下是字符串运算的示例：
```python
# 拼接
"Hello" + " " + "World"  # 结果为 'Hello World'
# 重复
"Hi" * 3                 # 结果为 'HiHiHi'
# 成员运算
"H" in "Hello"           # 结果为 True
"World" not in "Hello"   # 结果为 True
```
> **注意**：字符串拼接时，操作数必须都是字符串类型。若需与其他类型拼接，需先进行类型转换，例如 `"Price is " + str(10)`。

### 常用字符串方法

字符串对象有许多内置方法。可以使用 `dir(str)` 查看所有方法，用 `help(str.method)` 查看具体说明。

以下是几个常用的字符串方法：
```python
s = "hello world"
s.upper()       # 转为大写: 'HELLO WORLD'
s.lower()       # 转为小写: 'hello world'
s.title()       # 单词首字母大写: 'Hello World'
s.capitalize()  # 字符串首字母大写: 'Hello world'
s.split()       # 默认按空格拆分，得到列表: ['hello', 'world']
s.split('o')    # 按字符'o'拆分: ['hell', ' w', 'rld']
s.replace('o', 'a') # 替换: 'hella warld'
s.find('world') # 查找子串位置，返回索引 6
```
> **提示**：字符串方法通常返回处理后的新字符串，不会改变原字符串。如需改变原变量，需重新赋值，如 `s = s.upper()`。

### 字符串的转义

使用反斜杠 `\` 可以对特殊字符进行转义。

以下是常见的转义字符：
*   `\\`：反斜杠
*   `\'`：单引号
*   `\"`：双引号
*   `\n`：换行
*   `\t`：制表符

如果希望字符串中的反斜杠不被转义，可以使用**原始字符串**，在引号前加 `r`。
```python
# 普通字符串，\n被转义为换行
path1 = "C:\Desktop\news"  # 可能出错
# 原始字符串
path2 = r"C:\Desktop\news" # 正确表示路径
```

### 字符串的格式化

字符串格式化用于创建包含变量的模板。Python提供了多种格式化方法。

#### 1. 百分号 `%` 格式化

使用 `%s`（字符串）、`%d`（整数）、`%f`（浮点数）等作为占位符。
```python
name = "Alice"
age = 25
# 按顺序填充
"Hi, I'm %s. I'm %d years old." % (name, age)
# 使用字典填充
"Hi, I'm %(name)s. I'm %(age)d years old." % {'name': name, 'age': age}
```

#### 2. `format()` 方法格式化

使用 `{}` 作为占位符，功能更强大灵活。
```python
name = "Alice"
age = 25
# 按顺序填充
"Hi, I'm {}. I'm {} years old.".format(name, age)
# 按索引填充
"Hi, I'm {1}. I'm {0} years old.".format(age, name)
# 按关键字填充
"Hi, I'm {name}. I'm {age} years old.".format(name=name, age=age)
# 使用字典
info = {'name': 'Alice', 'age': 25}
"Hi, I'm {name}. I'm {age} years old.".format(**info)
```

#### 3. f-string 格式化 (Python 3.6+)

在字符串前加 `f` 或 `F`，直接在 `{}` 内写入变量或表达式，最为简洁直观。
```python
name = "Alice"
age = 25
# 直接嵌入变量
f"Hi, I'm {name}. I'm {age} years old."
# 嵌入表达式
f"Next year I will be {age + 1} years old."
# 格式化浮点数精度
price = 49.987
f"The price is {price:.2f}"  # 结果为 'The price is 49.99'
```
