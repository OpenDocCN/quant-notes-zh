## 时间对象基础

在Python基础课程中，你可能已经接触过处理时间的标准库。Python标准库中处理时间对象的类和方法位于`datetime`模块中。

`datetime`模块中有一个方法，可以将字符串转换为时间对象，即`strptime`方法。它的作用是解析字符串。记忆方法是：`p`代表解析（parse），`f`代表格式化（format）。因此，将字符串解析成对象的方法是`strptime`。

其基本用法是`datetime.datetime.strptime(字符串, 格式)`。例如，格式字符串可以是`%Y-%m-%d`。但这种转换比较麻烦，每次都需要输入格式字符串。

![](img/012dc1e38b3f08c850b58015601fe00f_3.png)

## 自动解析时间字符串

在实际数据中，时间字符串的格式可能多种多样，例如有人用`-`分隔，有人用`/`分隔，格式并不统一。手动处理每种格式非常繁琐。

![](img/012dc1e38b3f08c850b58015601fe00f_5.png)

![](img/012dc1e38b3f08c850b58015601fe00f_7.png)

有一个名为`dateutil`的库可以自动检测并解析多种常见的时间字符串格式。如果你安装了`pandas`，这个库通常已经自动安装好了，因为`pandas`依赖它。

![](img/012dc1e38b3f08c850b58015601fe00f_9.png)

`dateutil`库中有一个`parser`子模块，其中的`parse`函数可以直接转换字符串，无需指定格式。例如，`dateutil.parser.parse('2001-01-01')`。它支持各种格式，如`2001/01/01`或`Jan 1, 2001`。但请注意，它主要针对英文格式，像`2001年1月1日`这样的中文格式可能无法解析。

![](img/012dc1e38b3f08c850b58015601fe00f_11.png)

## 使用Pandas进行批量转换

如果我们有大量数据需要转换，手动使用`dateutil`仍然不够高效。为此，`pandas`提供了一个`to_datetime`函数。

![](img/012dc1e38b3f08c850b58015601fe00f_13.png)

这个函数可以传入一个列表或数组，批量将时间字符串转换为时间对象。例如：
```python
pd.to_datetime(['2001-01-01', '2001/01/02'])
```
转换后，我们会得到一个`DatetimeIndex`对象。这个对象非常适合用作数据索引，至于为什么用它做索引更好，我们将在后续课程中详细说明。
