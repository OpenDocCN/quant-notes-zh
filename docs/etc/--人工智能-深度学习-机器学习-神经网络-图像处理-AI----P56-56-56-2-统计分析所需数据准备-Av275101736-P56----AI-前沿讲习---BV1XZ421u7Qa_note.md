## 数据准备与转换

首先，我们需要对原始数据进行处理，将连续的价格变化指标转换为代表“涨”或“跌”的二元标签。

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_15.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_17.png)

以下是构建二元标签函数的步骤：

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_19.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_21.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_23.png)

1.  **定义函数**：创建一个名为 `CLIPINS` 的函数，用于处理数据。
2.  **遍历特征列**：对数据中代表“前一天”和“前两天”价格变化的每一列进行处理。
3.  **判断涨跌**：将每个特征值与0进行比较。大于0表示“涨”，标记为1；小于或等于0表示“跌”或“平”，标记为0。
4.  **存储结果**：将判断结果作为新列添加到原始数据框中。

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_25.png)

```python
def CLIPINS(data):
    global col_names
    col_names = []
    for col in [‘L1’, ‘L2’]:  # L1代表前一天，L2代表前两天
        new_col_name = col + ‘_B’
        data[new_col_name] = (data[col] > 0).astype(int)
        col_names.append(new_col_name)
```

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_27.png)

执行该函数后，数据框中会新增 `L1_B` 和 `L2_B` 两列，其值为0或1，分别代表对应日期的“跌”和“涨”。

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_29.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_31.png)

## 分组统计概率

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_43.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_45.png)

数据准备完成后，下一步是进行分组统计，计算在各种历史情况下，未来涨跌的分布频率。

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_47.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_49.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_51.png)

以下是使用 `groupby` 进行统计的步骤：

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_53.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_55.png)

1.  **选择分组列**：按照 `L1_B`（前一天状态）、`L2_B`（前两天状态）和 `direction`（实际结果）进行分组。
2.  **执行分组**：使用Pandas的 `groupby` 方法。
3.  **计算大小**：使用 `.size()` 方法统计每个唯一组合出现的次数。

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_57.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_59.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_61.png)

```python
grouped = data.groupby([‘L1_B’, ‘L2_B’, ‘direction’])
result = grouped.size()
print(result)
```

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_63.png)

![](img/3dd4f364ebc6996afd4118ef9fcf2f39_65.png)

执行以上代码后，我们将得到一个统计表。例如，结果 `(0, 0, 1): 150` 表示在历史数据中，前两天跌(`L2_B=0`)、前一天跌(`L1_B=0`)的情况下，第三天实际出现上涨(`direction=1`)的次数是150次。
