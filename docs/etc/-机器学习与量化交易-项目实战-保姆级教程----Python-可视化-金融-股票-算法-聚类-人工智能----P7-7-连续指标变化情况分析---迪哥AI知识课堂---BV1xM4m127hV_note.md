## 简单增长率累加的问题

上一节我们介绍了如何计算单期增长率。本节中我们来看看，如果我们想计算一个连续时间段内的“总”增长率，能否将各期的增长率简单相加。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_4.png)

例如，假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

以下是计算过程：
1.  第一到第二天的增长率：(50 - 100) / 100 = -0.5
2.  第二到第三天的增长率：(75 - 50) / 50 = 0.5

如果简单累加增长率：-0.5 + 0.5 = 0。这个结果为0，意味着股票“没有涨跌”。

但观察原始数据，股价从100元变为75元，实际上是下跌的。这说明**简单的增长率加法不能反映连续时间内的整体变化**。

## 引入对数变换计算连续回报

对于连续时间的计算，我们需要借助数学中的对数公式。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_10.png)

核心思想是：将对数形式的单期回报率相加，等价于计算首期与末期价格的直接比率。

**公式**：
`总连续回报 = log(第N天价格 / 第1天价格) = log(第2天价格/第1天价格) + log(第3天价格/第2天价格) + ... + log(第N天价格/第N-1天价格)`

![](img/5f0755a838e8bbcb382d2f9d71a83a88_12.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_14.png)

其中，`log`通常指自然对数（以e为底）。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_16.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_18.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_20.png)

**意义**：这个累加结果直接反映了“第一天投入的1元钱，在第N天相当于多少钱”。例如，累加结果为`log(2)`，则意味着最终价值是初始的2倍。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_22.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_24.png)

### 1. 计算单期对数回报率

首先，我们需要计算每一天相对于前一天的**对数回报率**。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_34.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_36.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_38.png)

**代码**：
```python
import pandas as pd
import numpy as np

# 假设 `data` 是一个包含股价的Pandas Series或DataFrame
# 使用 shift(1) 获取前一天的数据
shifted_data = data.shift(1)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_40.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_42.png)

# 计算对数回报率：log(当天价格 / 前一天价格)
returns = np.log(data / shifted_data)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_44.png)

# 查看结果
print(returns.head())
```
这段代码中，`data.shift(1)`将数据整体向上移动一行，使得每一行对应的是其“前一天”的数据。`np.log(data / shifted_data)`即计算了连续的对数回报率。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_46.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_48.png)

### 2. 计算累积连续回报（还原为价值）

![](img/5f0755a838e8bbcb382d2f9d71a83a88_50.png)

接下来，我们对这些单期的对数回报率进行累加，并通过指数函数`exp`将其还原为实际的累积增长倍数。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_52.png)

**代码**：
```python
# 对对数回报率进行累积求和
cumulative_returns = returns.cumsum()

![](img/5f0755a838e8bbcb382d2f9d71a83a88_54.png)

# 通过指数函数还原为价值倍数（假设初始投资为1单位）
cumulative_value = np.exp(cumulative_returns)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_56.png)

![](img/5f0755a838e8bbcb382d2f9d71a83a88_58.png)

# 绘制结果图
cumulative_value.plot(figsize=(16, 8))
```
*   `returns.cumsum()`：计算累积和。例如序列为`[a, b, c]`，则结果为`[a, a+b, a+b+c]`。
*   `np.exp(cumulative_returns)`：将对数累积回报转换回实际的价值倍数。如果`cumulative_returns`在某个时间点为`0.693`（即`log(2)`），那么`np.exp(0.693)`将约等于`2`，表示价值翻倍。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_60.png)

执行后，我们将得到一张图表，清晰地展示出从起始时间开始，假设投入1单位资金，在每个时间点对应的价值变化。例如，亚马逊（AMZN）的曲线可能显示，2010年的1美元到2018年变成了约10美元。

![](img/5f0755a838e8bbcb382d2f9d71a83a88_62.png)

本节课中我们一起学习了金融时间序列中连续变化情况的分析方法：

![](img/5f0755a838e8bbcb382d2f9d71a83a88_68.png)

1.  **差异值与简单增长率**：我们回顾了使用`.diff()`计算差异值，以及`(当期值-前期值)/前期值`计算简单增长率的方法。但这些方法不适合进行跨期累加。
2.  **对数回报率**：为了解决累加问题，我们引入了**对数回报率**的概念，其计算公式为`log(当期价格 / 上期价格)`。
3.  **累积价值计算**：通过对数回报率的累积和（`cumsum`），再经过指数运算（`exp`）还原，我们可以计算出“期初1单位投资在任意时间点的价值”这一重要指标。这比简单累加增长率更能准确反映资产的长期表现。
