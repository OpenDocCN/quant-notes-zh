# 金融量化分析：P20：阿尔法与贝塔概述 📊

在本节课中，我们将要学习投资策略评估中的两个核心概念：阿尔法（α）和贝塔（β）。理解这两个指标，能帮助我们区分投资收益的来源，并明确量化策略的优化方向。

## 收益的两种来源 💰

在投资中，你获得的收益通常可以分解为两个部分。

一部分收益与整体市场环境相关。当市场大环境向好时，大部分资产都会上涨，你自然也能从中获利。这部分收益可以看作是“随大流”获得的。

另一部分收益则与市场整体波动关系不大，它主要来源于你的个人努力。例如，通过深入研究公司财报、分析运营状况，凭借独到的眼光和策略进行操作所获得的收益。

因此，总收益可以看作由“市场收益”和“超额收益”两部分组成。

## 阿尔法（α）与贝塔（β）的定义 📈

上一节我们介绍了收益的两种来源，本节中我们来看看如何用指标来衡量它们。阿尔法和贝塔就是分别用于衡量这两部分的指标。

*   **贝塔（β）**：衡量投资组合收益对**市场波动**的敏感性。它反映了你的策略与大盘走势的关联程度，即“随大流”的部分。贝塔值高，意味着策略收益受市场影响大。
*   **阿尔法（α）**：衡量**超越市场**的超额收益。它描述的是与市场波动无关的、由你的策略或能力所带来的那部分收益。阿尔法是量化策略追求的核心目标。

简单来说：
*   市场收益与 **贝塔（β）** 挂钩。
*   超额收益与 **阿尔法（α）** 挂钩。

## 通过实例理解概念 📊

为了更好地理解，我们可以观察策略的回测结果图。图中通常包含几条曲线：

1.  **策略收益曲线**：代表你的策略所产生的总收益。
2.  **基准收益曲线**（如沪深300指数）：代表市场大环境的整体走势。
3.  **超额收益曲线**：由 **策略收益** 减去 **基准收益** 得到，它直观地展示了你的策略独立于市场所创造的价值。

因此，我们的总收益 = 市场收益（贝塔部分）+ 超额收益（阿尔法部分）。

## 阿尔法与贝塔的计算 🔢

那么，阿尔法和贝塔具体如何计算呢？在实际分析中，它们通常通过一个线性回归模型来估计。

![](img/dada101d8a9728546a37c4b7f1e17ba1_3.png)

模型公式可以表示为：
`策略收益 = α + β * 市场收益 + ε`

其中：
*   `α` 就是阿尔法值，代表回归线的截距。
*   `β` 就是贝塔值，代表回归线的斜率。
*   `ε` 是误差项。

通过历史数据拟合这个回归方程，我们就可以求解出具体的阿尔法（α）和贝塔（β）值。在后续讲解因子策略时，我们会深入探讨具体的计算方法。

## 量化策略的关注重点 🎯

了解这两个指标后，我们的策略目标就非常明确了。市场整体走势（贝塔部分）是个人无法控制的。因此，量化策略的核心关注点在于如何获取稳定且可持续的**阿尔法（α）**，即超额收益。

我们的目标就是通过构建有效的模型和策略，最大化阿尔法值。

## 其他常见评估指标 📝

除了阿尔法和贝塔，在策略评估中还会用到其他风险与收益指标。以下是其中一些常见的指标及其简要说明，初学者了解其含义即可，无需死记公式，实际应用中多有现成工具包可以调用。

*   **最大回撤**：衡量策略在历史上从峰值到谷底的最大亏损幅度，反映策略的风险。
*   **夏普比率**：衡量每承受一单位总风险，所获得的超额收益。比率越高，策略的风险调整后收益越好。
*   **基准收益**：即之前提到的市场收益，代表“什么都不做，单纯跟随大盘”所能获得的收益。
