1.  **判断是否需要调仓**：检查当前持仓中是否有股票不再满足选股条件（即不在最新的股票池中）。如果存在这样的股票，则需要进行调仓操作。
    ```python
    if len(stocks_to_delete) > 0:
        print(f‘执行调仓操作，时间：{context.current_dt}’)
    ```

![](img/6979bad779f7b9d48658fad9c6f4e3b5_25.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_27.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_29.png)

2.  **卖出不符合条件的股票**：对于所有需要删除的股票，将其持仓比例设置为0，即全部卖出。
    ```python
    for stock in stocks_to_delete:
        order_target_percent(stock, 0)
    ```

![](img/6979bad779f7b9d48658fad9c6f4e3b5_31.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_33.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_35.png)

3.  **买入符合条件的股票**：对于最新股票池中的所有股票，进行等权重买入。即每只股票的持仓比例平均分配。
    ```python
    for stock in stock_list:
        order_target_percent(stock, 1.0 / len(stock_list))
    ```

![](img/6979bad779f7b9d48658fad9c6f4e3b5_37.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_39.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_41.png)

通过以上三步，我们完成了每月一次的调仓操作，确保持仓始终由当前最符合因子条件的股票构成。

在编写和运行策略代码时，难免会遇到错误。我们需要根据错误提示，逐一检查和修正代码逻辑或语法问题。

以下是我们在调试过程中遇到并解决的主要问题：

![](img/6979bad779f7b9d48658fad9c6f4e3b5_65.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_67.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_69.png)

*   **中性化处理参数缺失**：在调用中性化函数时，缺少了必要的参数（如因子列名）。修正方法是将正确的参数传入函数。
*   **回归模型未训练**：在使用普通最小二乘法（OLS）进行中性化前，必须先调用 `fit()` 方法训练模型。
    ```python
    model = sm.OLS(y, X).fit()  # 先训练模型
    resid = y - model.predict(X)  # 再计算残差
    ```
*   **函数名拼写错误**：例如，将分位数函数 `quantile` 拼写错误。修正方法是检查并更正拼写。
*   **变量名引用错误**：在循环或判断中，使用了未定义或错误的变量名（如 `stock_list`）。修正方法是确保变量名前后一致。

## 回测结果分析

![](img/6979bad779f7b9d48658fad9c6f4e3b5_83.png)

代码运行后，我们可以在回测平台上查看策略的表现。回测结果提供了评估策略有效性的关键指标。

![](img/6979bad779f7b9d48658fad9c6f4e3b5_85.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_87.png)

以下是分析回测结果时关注的核心指标：

![](img/6979bad779f7b9d48658fad9c6f4e3b5_89.png)

*   **累计收益率**：策略在整个回测期间的总收益。与基准（如沪深300指数）对比，可以判断策略是否跑赢大盘。
*   **年化收益率**：将累计收益率折算为每年的平均收益率，便于不同周期策略的比较。
*   **夏普比率**：衡量每承受一单位风险，所获得的超额回报。公式为：
    `夏普比率 = (投资组合预期收益率 - 无风险利率) / 投资组合收益率的标准差`
    该值越高，说明风险调整后的收益越好。
*   **最大回撤**：在选定周期内，资产净值从最高点到最低点的最大跌幅。该指标反映了策略可能面临的最大亏损风险。

![](img/6979bad779f7b9d48658fad9c6f4e3b5_91.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_93.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_95.png)

在我们的示例回测中，策略的年化收益率和夏普比率虽然不高，但相较于基准指数（显示为负收益），策略实现了正收益，表现出了相对优势。最大回撤控制在10%以内，属于可接受范围。

本节课中我们一起学习了构建并评估一个完整因子选股策略的全过程。

![](img/6979bad779f7b9d48658fad9c6f4e3b5_117.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_119.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_121.png)

整个策略的流程可以总结为以下几步：
1.  **初始化与定时**：在 `initialize` 函数中设置定时器（如每月调仓）。
2.  **股票池过滤**：在 `before_trading_start` 或其他函数中，过滤掉ST股、停牌股等不符合条件的股票。
3.  **因子查询与处理**：获取所需因子数据（如市值、市净率），并进行必要的预处理（如中性化、标准化）。
4.  **选股**：根据处理后的因子值，应用选股逻辑（如选择市净率最低的20%股票），形成当期股票池。
5.  **调仓**：比较当期股票池与现有持仓，卖出“池外”股票，并等权重买入“池内”股票。
6.  **回测评估**：运行回测，通过累计收益、年化收益、夏普比率、最大回撤等指标综合评价策略效果。

![](img/6979bad779f7b9d48658fad9c6f4e3b5_123.png)

![](img/6979bad779f7b9d48658fad9c6f4e3b5_125.png)

需要注意的是，本课程中的策略示例主要用于演示流程和方法。实际应用中，因子的选择、处理方式、选股逻辑和风控措施都需要更深入的研究和严谨的验证。回测结果不代表未来表现，不可直接用于实盘交易。
