## 量化策略核心：3：策略程序的基本结构

![](img/a753f56a2d3cbd96e478634fcb7e02ca_37.png)

![](img/a753f56a2d3cbd96e478634fcb7e02ca_39.png)

一个完整的量化策略程序，无论使用何种平台（如穿云箭、聚宽、掘金），通常都围绕两个核心函数构建：

![](img/a753f56a2d3cbd96e478634fcb7e02ca_41.png)

![](img/a753f56a2d3cbd96e478634fcb7e02ca_43.png)

![](img/a753f56a2d3cbd96e478634fcb7e02ca_45.png)

![](img/a753f56a2d3cbd96e478634fcb7e02ca_47.png)

1.  **初始化函数**：策略启动时只运行一次。主要用于设置佣金、手续费、策略参数，以及创建初始的股票池。
    ```python
    def 初始化(上下文):
        设置佣金(费率=0.0003)
        股票池 = [‘000001’， ‘600000’]
        上下文.股票池 = 股票池
    ```

![](img/a753f56a2d3cbd96e478634fcb7e02ca_49.png)

![](img/a753f56a2d3cbd96e478634fcb7e02ca_51.png)

2.  **盘中运行函数**：在交易时间内被周期性或事件触发重复运行。负责从股票池获取行情、计算指标、产生交易信号并执行下单。
    ```python
    def 盘中运行(上下文， 行情数据):
        for 股票 in 上下文.股票池:
            数据 = 获取行情(股票)
            信号 = 计算交易信号(数据)
            if 信号 == ‘买入’:
                下单(股票， ‘买入’， 数量=100)
    ```

其核心流程可以概括为：**选取股票池 -> 计算指标 -> 根据信号交易**。

## 执行交易：5：连接交易接口

在策略中，当计算出买入或卖出信号后，需要调用交易接口来执行。不同的交易账户（股票、期货、外汇）需要使用不同的接口和指令。

*   **股票交易（以同花顺为例）**：
    ```python
    # 初始化连接
    初始化交易接口(‘同花顺’)
    # 买入指令
    下单(股票代码=‘000001’， 操作=‘买入’， 价格=现价， 数量=100)
    ```

*   **期货交易**：期货是双向交易，指令更多样。
    ```python
    # 初始化期货接口（如天勤）
    初始化交易接口(‘天勤期货’)
    # 开仓买入（做多）
    下单(合约=‘rb2405’， 操作=‘买开’， 数量=1)
    # 平仓卖出
    下单(合约=‘rb2405’， 操作=‘卖平’， 数量=1)
    ```

*   **替换交易接口**：策略逻辑不变，只需替换初始化连接和具体下单指令的代码，即可让同一个策略为不同的账户服务。

## 消息通知：6：发送交易提醒（QQ/微信）

如果你不想或不能进行全自动交易，可以让策略在发现交易机会时，通过QQ、微信、邮件等方式发送提醒，然后手动下单。

![](img/a753f56a2d3cbd96e478634fcb7e02ca_96.png)

![](img/a753f56a2d3cbd96e478634fcb7e02ca_98.png)

实现原理是：在策略的**盘中运行函数**中，当交易信号产生时，不执行下单指令，而是调用发送消息的函数。

![](img/a753f56a2d3cbd96e478634fcb7e02ca_100.png)

*   **发送QQ消息**：需要指定接收人的QQ昵称或群名。
    ```python
    # 消息接收列表
    联系人列表 = [‘独狼科普’， ‘小虾’， ‘独狼群’]
    # 当信号产生时
    for 联系人 in 联系人列表:
        发送QQ消息(联系人， f‘发现机会：{股票代码} 满足买入条件！’)
    ```

*   **发送微信消息**：原理与QQ类似，通过模拟微信客户端发送。
    ```python
    发送微信消息(‘文件传输助手’， ‘策略提醒：请注意股票 000001’)
    ```

![](img/a753f56a2d3cbd96e478634fcb7e02ca_102.png)

*   **发送邮件或短信**：邮件需要配置SMTP服务，短信则需要购买第三方服务（如腾讯云短信）。

![](img/a753f56a2d3cbd96e478634fcb7e02ca_104.png)

![](img/a753f56a2d3cbd96e478634fcb7e02ca_106.png)

![](img/a753f56a2d3cbd96e478634fcb7e02ca_107.png)

这种方法让你可以在上班或外出时，也能及时监控策略信号，不错过交易机会。
