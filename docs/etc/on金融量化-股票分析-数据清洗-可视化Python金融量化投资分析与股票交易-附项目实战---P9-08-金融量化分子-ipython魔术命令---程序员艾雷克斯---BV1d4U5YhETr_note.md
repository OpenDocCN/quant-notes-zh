## 魔术命令简介

上一节我们介绍了IPython的基本使用，本节中我们来看看IPython提供的“魔术命令”。魔术命令是以百分号（`%`）开头的一系列特殊指令，它们为交互式编程提供了许多便捷的高级功能。

## 运行外部Python脚本

![](img/02cd11689e9acb482d5025a3ce4ca951_3.png)

在标准的Python命令行中，若想运行一个已写好的Python文件，通常需要退出命令行，再使用`python 文件名.py`的命令来执行。在IPython中，我们无需退出。

我们可以使用`%run`魔术命令，直接在IPython交互器中运行外部的Python脚本文件。

![](img/02cd11689e9acb482d5025a3ce4ca951_5.png)

**操作步骤如下：**
1.  首先，在任意文本编辑器中创建一个简单的Python文件，例如`hello.py`，内容为：
    ```python
    print("Hello World")
    ```
2.  在IPython中，使用以下命令即可运行该文件：
    ```python
    %run hello.py
    ```
    执行后，会输出`Hello World`。这个功能允许我们在不离开交互环境的情况下测试和运行完整的脚本。

## 粘贴并执行代码

有时，我们可能需要从编辑器或其他地方复制一大段代码到IPython中进行测试。直接粘贴可能会因为缩进等问题导致错误。

![](img/02cd11689e9acb482d5025a3ce4ca951_7.png)

为了解决这个问题，IPython提供了`%paste`魔术命令。它会智能地处理剪贴板中的代码，然后执行。

![](img/02cd11689e9acb482d5025a3ce4ca951_9.png)

**以下是使用`%paste`命令的步骤：**
1.  在任何地方（如Sublime Text, VS Code等编辑器）复制一段代码到剪贴板。
2.  在IPython中输入`%paste`并回车。
3.  该命令会先打印出即将执行的代码内容，并用分隔符标出，然后自动执行这段代码。这避免了手动粘贴时可能出现的格式错误。

![](img/02cd11689e9acb482d5025a3ce4ca951_11.png)

## 精确测量代码执行时间

在性能分析和优化时，我们经常需要测量某段代码或函数的执行时间。虽然可以使用Python的`time`模块，但对于执行时间极短的代码，测量结果可能不精确（例如显示为0秒）。

IPython的`%timeit`魔术命令可以解决这个问题。它会自动多次运行指定的代码，并计算平均执行时间，从而得到非常精确的结果，尤其适用于微基准测试。

**让我们通过一个例子来理解`%timeit`的用法：**
假设我们需要测量对一个列表进行排序所需的时间。

```python
# 创建一个包含10000个随机数的列表
import random
my_list = [random.randint(1, 10000) for _ in range(10000)]

# 使用 %timeit 测量排序操作的时间
%timeit sorted(my_list)
```
执行上述命令后，`%timeit`会输出类似以下的结果：
```
1.46 ms ± 7.42 µs per loop (mean ± std. dev. of 7 runs, 1000 loops each)
```
这表示：`sorted(my_list)`这个操作平均每次执行需要1.46毫秒。`%timeit`自动运行了7轮，每轮循环执行了1000次排序，最后给出了平均值和标准差。这种方式能有效衡量短时间操作的性能。

**核心概念公式/代码描述：**
- 魔术命令通用格式：`%command` 或 `%%command`（用于单元格魔术命令）。
- 运行脚本：`%run script_name.py`
- 粘贴执行：`%paste`
- 测量时间：`%timeit statement_to_measure`
