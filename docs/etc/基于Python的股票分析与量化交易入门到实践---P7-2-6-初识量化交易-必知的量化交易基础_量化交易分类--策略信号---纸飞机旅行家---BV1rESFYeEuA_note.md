## 概述：什么是策略信号？

![](img/a1ae0564955174b2c4bb4e542c74ee96_3.png)

在上一节我们介绍了量化交易的盈利模式分类，本节中我们来看看量化策略的核心——策略信号。无论量化策略如何实现，其基本流程都遵循三个步骤：**信号捕捉 -> 交易 -> 建仓/平仓**。其中，**策略信号**就是第一步中捕捉到的、用于决定是否进行交易以及进行何种交易（买入或卖出）的特征信号。它是量化策略的决策核心。

为了更直观地理解，我们可以将策略信号想象成指导交易的“脉冲”。例如，观察股票价格走势图，策略会在特定的价格点（如低点）生成“买入”信号（记为1），在另一些点（如高点）生成“卖出”信号（记为0）。将这些信号与价格走势图拟合，就能看到策略是如何根据信号进行操作的。

## 策略信号的三大流派

策略信号分类下主要有三个流派，它们各自有不同的方法论和盈利逻辑。以下是这三个流派的详细介绍。

### 1. 多因子策略

![](img/a1ae0564955174b2c4bb4e542c74ee96_5.png)

多因子策略是一种非常常见且基础的量化策略。其核心思想是：找到与股票未来收益率最相关的一系列指标（即“因子”），并根据这些因子构建一个股票组合，目标是确保该组合在未来一段时间内能够跑赢或跑输市场指数。

**盈利模式**：
*   如果能确定股票组合未来表现**优于**指数，则对该组合进行**做多**操作。
*   如果能确定股票组合未来表现**差于**指数，则对该组合进行**做空**操作。

**举例**：
一个简单的多因子组合可能包含以下因子项：
*   资产负债率 < 5%
*   资产回报率 > 20%
*   每股净收益增长率 > 15%
*   市盈率 < 30
*   属于“新能源”板块

满足所有这些条件的股票将构成投资组合。如果模型预测该组合将跑赢大盘，则买入持有。

### 2. 交易模型

交易模型是基于现代金融理论、数学和计算机技术，构建的一套具有普遍性、可盈利、可量化、可执行的交易系统。你可以将其理解为自己编写的一套自动化交易程序或规则。

**盈利模式**：交易模型的盈利关键在于市场的走势**符合**模型所设定的逻辑和规则。只要价格波动在模型预设的范围内，模型就能通过高频的买卖操作获利。

**举例：网格交易法**
网格交易法是交易模型中一个经典的策略，特别适合震荡市场。其原理如下：
1.  设定一个基准价格（如10元）和网格间距（如0.1元）。
2.  在基准价建立初始仓位（如买入500股）。
3.  设定规则：价格每上涨一个网格（0.1元），则卖出一定数量（如100股）；价格每下跌一个网格（0.1元），则买入一定数量（如100股）。

通过这样的规则，在价格上下震荡的过程中，策略会自动进行“低买高卖”的操作，从而积累利润。其核心公式可以表示为：
```python
# 伪代码逻辑
if current_price >= last_trade_price + grid_size:
    sell(some_shares)
    update_last_trade_price(current_price)
elif current_price <= last_trade_price - grid_size:
    buy(some_shares)
    update_last_trade_price(current_price)
```
需要注意的是，如果市场出现单边上涨或下跌行情，网格交易法则可能失效，导致亏损或套牢。

### 3. 机器学习

![](img/a1ae0564955174b2c4bb4e542c74ee96_9.png)

机器学习流派是量化交易领域的新兴方向。它运用机器学习算法（如自然语言处理、知识图谱），试图从海量、非传统的（如新闻文本、社交媒体情绪）数据中，挖掘出与股价变动相关的、尚未被传统金融理论发现的规律和策略信号。

**盈利模式**：通过算法发现有效的、领先的“特征-股价”关联模式，并据此生成交易信号。其目标是从新的数据维度“挖掘”出能带来超额收益的交易机会。

**举例**：
研究机构尝试分析新闻事件对股价的影响。例如，通过文本挖掘技术分析关于某公司的新闻（如“董事长变更”、“新产品发布”），并利用机器学习模型判断该新闻是利好还是利空，以及其影响是否会带来股价在次日上涨或下跌的领先信号。如果模型能有效识别这种模式，就可以提前进行交易。

## 总结

本节课中我们一起学习了量化交易的核心——策略信号分类。
*   **策略信号**本质上是交易的指令信号，决定了量化策略何时买入或卖出，是整个策略的决策核心。
*   我们详细介绍了三大策略流派：
    1.  **多因子策略**：通过寻找与收益率相关的指标构建组合，通过判断组合与指数的相对强弱（跑赢或跑输）来进行做多或做空盈利。
    2.  **交易模型**：构建一套自动化的交易系统（如网格交易法），其盈利依赖于市场走势是否符合模型设定的震荡或趋势逻辑。
    3.  **机器学习**：利用先进的算法从更广阔的数据中挖掘潜在的、非传统的交易信号，旨在发现新的盈利模式。

![](img/a1ae0564955174b2c4bb4e542c74ee96_13.png)

理解这些基础分类，有助于我们在后续课程中更好地设计和选择适合自己的量化策略。
