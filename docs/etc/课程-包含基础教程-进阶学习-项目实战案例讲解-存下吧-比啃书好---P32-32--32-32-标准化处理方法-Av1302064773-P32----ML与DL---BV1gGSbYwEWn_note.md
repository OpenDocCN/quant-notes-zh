在本节课中，我们将要学习数据预处理中一个非常重要的步骤——标准化。标准化是机器学习和数据分析中常用的技术，旨在消除不同特征（或指标）之间由于量纲和取值范围不同带来的影响，使数据更易于分析和建模。

![](img/57a27db5390a0a5a0c3ed4819036f610_3.png)

## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体作用。

标准化操作的核心目标是：**使不同特征（指标）的数值处于同一量级，以便公平地进行比较和分析**。其原理基于一个简单的数学公式。

假设我们有一组数据，包含两个特征 `X1` 和 `X2`。它们的取值范围可能完全不同。例如，`X1` 的取值范围较大，而 `X2` 的取值范围较小。如果直接使用原始数据，取值范围大的特征可能会在模型计算中占据主导地位，从而影响结果的准确性。

为了解决这个问题，我们使用标准化。标准化的公式如下：

**`Z = (X - μ) / σ`**

其中：
*   `X` 是原始数据。
*   `μ` 是数据在该特征维度上的**均值**。
*   `σ` 是数据在该特征维度上的**标准差**。

这个公式可以分解为两个步骤：

1.  **去均值**：`X - μ`
    *   **目的**：将数据的中心移动到零点。这使得数据在各个维度上都以零为中心对称。
2.  **除标准差**：`(X - μ) / σ`
    *   **目的**：调整数据的“尺度”或“分散程度”。通过除以标准差，取值范围大的特征会被“压缩”，取值范围小的特征会被“拉伸”，最终使所有特征的数值分布在一个相近的范围内。

为了更直观地理解，可以想象两组人：
*   **第一组**：工资都在1万到2万之间，大家的收入差异小，计算出的标准差也小。
*   **第二组**：包含工资1万、2000亿、3000亿的人，收入差异极大，计算出的标准差非常大。

对第一组数据除以一个小的标准差，对第二组数据除以一个大的标准差，最终能使两组数据调整后的数值范围变得相近。这就是标准化让不同量纲数据变得可比的核心思想。

## 标准化的代码实现

理解了原理后，我们来看看如何用代码实现标准化。以下是手动实现标准化的方法：

```python
def standardize_series(series):
    """
    对输入的pandas Series进行标准化处理。
    参数:
        series: 待标准化的数据序列。
    返回:
        标准化后的数据序列。
    """
    # 计算均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    # 应用标准化公式： (X - μ) / σ
    standardized_series = (series - mean_val) / std_val
    return standardized_series
```

代码非常简单，核心就是计算数据的均值和标准差，然后应用公式。

## 使用现成工具库

在实际项目中，我们通常使用成熟的机器学习库来快速完成标准化，例如 `scikit-learn`。

![](img/57a27db5390a0a5a0c3ed4819036f610_9.png)

以下是使用 `scikit-learn` 进行标准化的方法：

![](img/57a27db5390a0a5a0c3ed4819036f610_11.png)

![](img/57a27db5390a0a5a0c3ed4819036f610_13.png)

```python
from sklearn.preprocessing import StandardScaler
import pandas as pd

# 假设 `data` 是一个DataFrame
scaler = StandardScaler()
# 拟合数据并转换
standardized_data = scaler.fit_transform(data)
# 将结果转换回DataFrame（可选）
standardized_df = pd.DataFrame(standardized_data, columns=data.columns)
```

![](img/57a27db5390a0a5a0c3ed4819036f610_15.png)

使用 `scikit-learn` 的 `StandardScaler` 可以一行代码完成标准化，并且能方便地处理整个数据集。

![](img/57a27db5390a0a5a0c3ed4819036f610_23.png)

```python
import pandas as pd
data = pd.Series([30, 25, 100, 45, 60])
print("原始数据：")
print(data)
```

![](img/57a27db5390a0a5a0c3ed4819036f610_25.png)

原始数据的取值范围差异很大（从25到100）。

![](img/57a27db5390a0a5a0c3ed4819036f610_31.png)

```python
standardized_data = standardize_series(data)
print("\n标准化后的数据：")
print(standardized_data)
```

![](img/57a27db5390a0a5a0c3ed4819036f610_33.png)

输出结果将显示，所有数值都被转换到了一个以0为中心、数值范围相对较小的区间内（通常在-3到3之间）。这验证了标准化的效果：**消除了量纲差异，使数据分布更均匀**。
