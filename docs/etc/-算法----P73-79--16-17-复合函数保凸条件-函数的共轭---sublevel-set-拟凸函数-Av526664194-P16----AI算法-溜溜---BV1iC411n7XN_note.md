# 凸优化：16：复合函数保凸条件、函数的共轭、α-sublevel set与拟凸函数

在本节课中，我们将深入学习凸函数分析中的几个核心概念。我们将首先回顾并深入探讨复合函数保持凸性的条件，然后介绍函数的共轭及其性质，接着定义并分析α-sublevel set，最后简要引入拟凸函数的概念。这些工具对于理解和构建凸优化问题至关重要。

## 复合函数保凸条件回顾

上一节我们介绍了判断函数组合凸凹性的四条规则。本节中，我们来看看其中关于复合函数保凸性的第一条规则，并深入理解其关键假设。

假设我们考虑一个复合函数 **F = h ∘ g**，即 **F(x) = h(g(x))**。如果满足以下两个条件：
1.  函数 **h** 是一个凸函数。
2.  函数 **h** 的扩展（即将其定义域扩展到整个实数空间后得到的函数）在整个实数空间上是单调非降的。
3.  函数 **g** 是一个凸函数。

那么，复合函数 **F** 就一定是一个凸函数。

这里需要强调一个关键点：**h 的扩展在整个空间上的单调性**是必要条件，而不仅仅是 **h** 在其原始定义域内的单调性。为了说明这一点，我们来看一个反例。

### 一个关键的反例

考虑以下两个函数：
*   **g(x) = x²**，其定义域为整个实数空间 **R**，显然是一个凸函数。
*   **h(z)** 定义如下：
    ```
    h(z) = 0, 当 1 ≤ z ≤ 2
    ```
    在其定义域 [1, 2] 上，**h(z)** 既是凸函数（因为其图像是一条线段），也是单调的（常数函数既是非增也是非降）。

如果我们忽略“**h** 的扩展”这一条件，仅根据 **h** 在其定义域内的凸性和单调性，可能会错误地推断 **F(x) = h(g(x))** 是凸函数。然而，让我们分析 **F(x)** 的实际形态。

函数 **F(x)** 在 **g(x)** 取值于 [1, 2] 时为零，即当 **x ∈ [-√2, -1] ∪ [1, √2]** 时 **F(x)=0**，在其他地方无定义。这个定义域 **{x | 1 ≤ x² ≤ 2}** 本身不是一个凸集（例如，取 x=-1 和 x=1，它们的凸组合 0 不在集合内）。因此，**F(x)** 甚至不是一个定义在凸集上的函数，更不可能是凸函数。

问题出在哪里？在于 **h** 的扩展。作为凸函数，**h** 的凸扩展在其定义域 [1, 2] 之外会被赋予值 **+∞**。这个扩展后的函数图像在 z<1 时是 +∞，在 [1,2] 区间是 0，在 z>2 时又是 +∞。这个扩展函数显然**不是**单调非降的。正是由于 **h** 的扩展缺乏单调性，导致复合函数保凸的规则失效。

## 重新审视指数函数例子

上一节我们遗留了一个关于指数函数的例子。本节中我们来看看如何巧妙地解决它。

我们想证明：如果 **g(x)** 是凸函数且 **g(x) ≥ 0**，对于任意 **p ≥ 1**，函数 **F(x) = (g(x))^p** 也是凸函数。

直接套用规则会遇到困难。如果我们定义 **h(z) = z^p**，其定义域为 **z ≥ 0**。**h(z)** 是凸且单调非降的，但其凸扩展在 **z < 0** 时为 **+∞**，导致扩展函数在 **z=0** 处从 **+∞** 跳变到 **0**，破坏了单调性。

解决方案是重新构造 **h(z)**，使其定义在整个实数空间 **R** 上，同时保持凸性和单调性。我们可以如下定义：

```
h(z) = { z^p,  如果 z ≥ 0
       { 0,    如果 z < 0
```

现在，这个新的 **h(z)** 定义在整个 **R** 上，其本身就是其自身的扩展。容易验证：
1.  在 **z ≥ 0** 时，它是凸且单调非降的。
2.  在 **z < 0** 时，它是常数 0。
3.  在 **z=0** 处，函数连续且从常数段平滑（非下降）地过渡到增长段。

因此，这个 **h(z)** 在整个 **R** 上是凸函数且单调非降。由于 **g(x) ≥ 0**，复合 **F(x) = h(g(x)) = (g(x))^p** 成立，并且满足复合函数保凸的所有条件，从而证明 **F(x)** 是凸函数。这个例子告诉我们，有时需要灵活地构造辅助函数来满足定理的条件。

## 函数的共轭

接下来，我们介绍一个在优化理论中极为重要的概念：函数的共轭。

### 定义与几何意义

对于一个函数 **f: R^n → R**，其共轭函数 **f*: R^n → R** 定义为：

**f*(y) = sup_{x ∈ dom f} ( y^T x - f(x) )**

其中，**sup** 表示上确界（可以理解为最大值，但更严谨），**dom f** 是 **f** 的定义域。

这个定义的几何意义非常直观：对于给定的向量 **y**，**y^T x** 是一个关于 **x** 的线性函数。**f*(y)** 的值就是这个线性函数 **y^T x** 与原始函数 **f(x)** 之间的最大垂直间隔（在 **x** 变化时）。你可以想象用一系列斜率为 **y** 的直线去“支撑”函数 **f(x)** 的图像，**f*(y)** 就是这些直线在 y 轴截距的相反数所能达到的最大值。

### 重要性质

函数的共轭有两个非常重要的性质：

1.  **可微情况下的最优性条件**：如果 **f(x)** 是可微的，那么使得上式中上确界达到的 **x̂**（如果存在）满足一阶最优性条件：**∇f(x̂) = y**。这是因为我们是对 **x** 求 **y^T x - f(x)** 的无约束极大值，令其梯度为零即得：**y - ∇f(x) = 0**。

2.  **共轭函数必为凸函数**：无论原函数 **f(x)** 是否是凸函数，其共轭函数 **f*(y)** 都一定是凸函数。这个性质在对偶理论中起着基石般的作用，它保证了无论原问题多么复杂，其对偶问题总是一个凸优化问题，这大大简化了求解。

### 共轭函数计算示例

以下是几个常见函数的共轭计算：

*   **仿射函数**：设 **f(x) = a^T x + b**。
    *   其共轭为：**f*(y) = sup_x ( y^T x - (a^T x + b) ) = sup_x ( (y - a)^T x - b )**。
    *   这是一个关于 **x** 的线性函数。如果 **y = a**，则线性项为零，**f*(a) = -b**。
    *   如果 **y ≠ a**，则该线性函数无上界，**f*(y) = +∞**。
    *   所以，**f*(y)** 是一个仅在 **y = a** 点取有限值 **-b**，在其他点均为 **+∞** 的函数，这是一个凸函数（尽管是退化的）。

*   **负对数函数**：设 **f(x) = -log(x)**，定义域 **dom f = R_{++}**（正实数）。
    *   其共轭为：**f*(y) = sup_{x>0} ( y x + log(x) )**。
    *   利用性质1，对内部函数求导并令其为零：**y + 1/x = 0**，解得 **x = -1/y**。
    *   此解要求 **x > 0**，即 **-1/y > 0**，所以 **y < 0**。
    *   将 **x = -1/y** 代入，得 **f*(y) = y * (-1/y) + log(-1/y) = -1 - log(-y)**，其中 **y < 0**。
    *   当 **y ≥ 0** 时，内部函数 **y x + log(x)** 随着 **x → 0+** 趋向 **-∞**，但随着 **x → +∞**，**y x** 主导并使整体趋向 **+∞**，因此上确界为 **+∞**。
    *   综上，**f*(y) = { -1 - log(-y), 若 y < 0; +∞, 若 y ≥ 0 }**。

*   **二次函数**：设 **f(x) = (1/2) x^T Q x**，其中 **Q** 是一个 **n×n** 对称正定矩阵，定义域为 **R^n**。
    *   其共轭为：**f*(y) = sup_x ( y^T x - (1/2) x^T Q x )**。
    *   利用性质1，求梯度并令其为零：**y - Q x = 0**，解得 **x = Q^{-1} y**（因为 **Q** 正定可逆）。
    *   将解代入：**f*(y) = y^T (Q^{-1} y) - (1/2) (Q^{-1} y)^T Q (Q^{-1} y) = y^T Q^{-1} y - (1/2) y^T Q^{-1} y = (1/2) y^T Q^{-1} y**。
    *   因此，二次函数的共轭仍是二次函数，只是将矩阵 **Q** 替换为其逆矩阵 **Q^{-1}**。这是一个非常优美且对称的性质。

### 二次共轭

一个自然的问题是：共轭的共轭 **f**(x)** 是否等于原函数 **f(x)**？答案是否定的。

我们知道，任何函数的共轭 **f*** 都是凸函数。那么 **f*** 的共轭 **f**** 也必然是凸函数。如果原函数 **f** 本身不是凸函数，那么 **f**** 就不可能等于 **f**。例如，一个非凸函数，其二次共轭会是它的凸包络（某种“最接近”的凸函数）。

然而，如果 **f** 是一个闭凸函数（closed convex function），那么我们有 **f** = f**。这类似于复数中，一个复数的共轭的共轭等于其自身，但对于函数，需要附加凸性和闭性条件才成立。

## α-sublevel set

在介绍了凸集和凸函数之后，我们现在来看一个直接连接这两个概念的工具：α-sublevel set。

### 定义

对于一个函数 **f: R^n → R** 和一个实数 **α ∈ R**，其 **α-sublevel set** 定义为：

**C_α = { x ∈ dom f | f(x) ≤ α }**

即，函数值不超过 **α** 的所有点的集合。你可以将其想象为函数图像在高度 **α** 以下的“等高线”所围成的区域在定义域上的投影。

### 与凸函数的关系

α-sublevel set 有一个非常重要的性质：

**如果 f(x) 是一个凸函数，那么对于任意 α ∈ R，其 α-sublevel set C_α 都是一个凸集。**

**证明**：为了证明 **C_α** 是凸集，我们需要验证其定义：任取 **x, y ∈ C_α** 以及任意 **θ ∈ [0, 1]**，证明它们的凸组合 **z = θx + (1-θ)y** 也属于 **C_α**。
1.  由于 **x, y ∈ dom f** 且 **dom f** 是凸集（凸函数定义要求），所以 **z ∈ dom f**。
2.  因为 **f** 是凸函数，所以有：**f(z) ≤ θ f(x) + (1-θ) f(y)**。
3.  由于 **x, y ∈ C_α**，所以 **f(x) ≤ α**，**f(y) ≤ α**。
4.  因此，**f(z) ≤ θ α + (1-θ) α = α**。
5.  结合1和4，**z ∈ C_α**。证毕。

这个性质非常有用。它意味着，对于一个凸函数，其所有“水平切片”给出的集合都是凸的。这在构建约束条件时尤为重要：形如 **f(x) ≤ 0** 的约束，如果 **f** 是凸函数，则其可行域是一个凸集，这保证了优化问题的良好结构。

## 拟凸函数简介

最后，我们简要介绍拟凸函数的概念，它与 sublevel set 有直接联系。

一个函数 **f: R^n → R** 被称为**拟凸函数**，如果其定义域是凸集，并且其所有的 α-sublevel set **C_α** 对于任意 **α ∈ R** 都是凸集。

比较一下：
*   **凸函数**：要求函数图像上任意两点的连线位于图像上方。这是一个很强的函数值条件。
*   **拟凸函数**：只要求每个水平集是凸的。这是一个更弱的几何条件。

显然，**所有凸函数都是拟凸函数**（因为我们已经证明了凸函数的 sublevel set 是凸集），但反之则不成立。拟凸函数比凸函数更广泛，例如，单峰函数（如正态分布密度函数）通常是拟凸的，但不是凸的。拟凸优化问题（目标函数或约束函数为拟凸）比凸优化问题更一般，但仍有较好的性质可供利用。

3.  我们定义了**α-sublevel set**，并证明了凸函数的所有 sublevel set 都是凸集，这建立了凸函数与凸集之间的一个重要桥梁。
