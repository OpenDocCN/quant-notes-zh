## 影响股价的六大因素

上一节我们介绍了股票的基本概念，本节中我们来看看哪些因素会影响股票价格的波动。影响股价的因素可以归纳为以下六点。

以下是影响股价的六大因素：

1.  **公司自身因素**：这是影响股价最根本的因素。公司的经营状况、盈利能力、发展前景以及重大事件（如丑闻）都会直接影响其股票价值。公式上，可以理解为：**股价 ≈ 公司内在价值 / 总股本**。公司价值增长，股价通常上涨；反之则下跌。

2.  **市场因素**：这是影响股价最直接的因素，表现为买卖双方的供需关系。当买盘大于卖盘（供不应求）时，股价上涨；当卖盘大于买盘（供过于求）时，股价下跌。

3.  **行业因素**：整个行业的发展趋势会影响行业内所有公司。例如，当人工智能行业蓬勃发展时，相关公司的股票可能普遍上涨；反之，若某个行业衰退，其公司的股票也可能随之下跌。

4.  **心理因素**：投资者的情绪和非理性行为会影响股价，例如从众心理。当市场出现恐慌性抛售时，即使公司基本面未发生重大变化，股价也可能因投资者的集体行为而暴跌。

5.  **经济因素**：国家宏观经济政策，如利率、存款准备金率、外汇政策等，会影响市场流动性和投资意愿。例如，存款利率上升可能导致资金从股市流向银行，从而对股价产生下行压力。

6.  **政治因素**：国际关系、地区局势、政府政策等政治事件会显著影响市场信心和资本流向。例如，地缘政治紧张局势可能引发市场恐慌，导致股市下跌；而相关领域的股票（如军工股）可能因事件驱动而上涨。

## 股票买卖流程与规则

了解了影响股价的因素后，我们再来看看个人投资者如何进行股票买卖，以及需要遵守哪些市场规则。

以下是股票买卖的关键步骤与规则：

1.  **开户与委托**：个人不能直接进入交易所买卖股票，必须通过证券公司（券商）开户。开户后，投资者通过券商的系统提交买卖指令，这个过程称为“委托”。

2.  **股票交易日**：A股市场并非每天都可以交易。交易日通常为每周一至周五（非法定节假日）。遇到国家法定节假日，交易所也会休市。

3.  **交易时间**：在每个交易日内，也有特定的交易时段。A股主要交易时间如下：
    *   **开盘集合竞价**：9:15 - 9:25。此期间提交的委托单不会立即成交，交易所会在9:25一次性集中撮合，以产生当日的**开盘价**。其核心原则是最大化成交量。
    *   **连续竞价（早市）**：9:30 - 11:30。在此期间，交易所系统对买卖申报进行逐笔连续撮合，投资者提交的委托若符合条件可即时成交。
    *   **连续竞价（午市）**：13:00 - 14:57（沪市）/ 14:57（深市）。此阶段同样是连续竞价。
    *   **收盘集合竞价（仅深市）**：14:57 - 15:00。深交所在最后3分钟采用集合竞价方式产生**收盘价**。沪市的收盘价为当日最后一笔交易前一分钟所有交易的成交量加权平均价。

4.  **T+1交易制度**：投资者当天买入的股票，必须等到下一个交易日（T+1日）才能卖出。当天卖出的股票，所得资金当天可用于买入股票，但需到下一个交易日才能从证券账户转出到银行账户。

5.  **涨跌停板限制**：为抑制过度投机，A股市场对普通股票设有每日价格涨跌幅限制。通常，一只股票在一个交易日内的交易价格，相对于上一个交易日的收盘价，涨幅和跌幅不得超过10%（ST等特殊股票为5%）。价格达到涨跌停板后，交易并未完全停止，但可能难以买入或卖出。
