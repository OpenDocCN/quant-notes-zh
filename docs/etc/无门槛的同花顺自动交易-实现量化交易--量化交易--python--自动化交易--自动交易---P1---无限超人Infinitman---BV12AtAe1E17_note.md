# 量化交易入门：1：基于同花顺客户端的简易自动化交易方案

## 概述
在本节课中，我们将要学习如何利用Python和同花顺客户端，实现一个低门槛的自动化交易方案。这种方法不依赖专业的量化交易软件（如QMT），而是通过模拟鼠标点击的方式在同花顺客户端内完成下单操作。

## 原理与局限性
上一节我们介绍了自动化交易的概念，本节中我们来看看这种基于同花顺的具体实现原理。

此方法并非通过券商提供的专业交易接口（API）实现，而是通过程序模拟鼠标点击操作，控制同花顺客户端的下单界面。因此，它存在以下核心缺陷：
*   **交易不稳定**：可能受网络延迟、客户端界面变化等因素影响，导致下单失败。
*   **非官方支持**：该方法依赖于客户端界面，若同花顺更新界面，相关代码可能需要调整。

**核心操作逻辑**可以用以下伪代码描述：
```python
启动同花顺下单程序 -> 定位输入框 -> 输入股票代码 -> 输入价格 -> 点击“买入”或“卖出”按钮
```

## 环境准备与代码演示
理解了原理后，我们需要进行环境准备。以下是实现此功能所需的步骤：

1.  **安装Python库**：使用 `pip` 安装名为 `easytrader` 的库。
    ```bash
    pip install easytrader
    ```
2.  **定位同花顺下单程序**：找到您电脑上同花顺软件的安装目录，并定位到名为 `xiadan.exe` 的可执行文件，记下其完整路径。
3.  **登录客户端**：**至关重要的一步**：必须手动打开并登录同花顺的 `xiadan.exe` 客户端，确保交易账户处于已登录状态。

接下来，我们通过一段示例代码来演示如何实现自动卖出操作。

![](img/3080b783f600f6121aebccba46c22be9_1.png)

![](img/3080b783f600f6121aebccba46c22be9_3.png)

```python
import easytrader

![](img/3080b783f600f6121aebccba46c22be9_5.png)

# 1. 创建用户对象并连接客户端
# 将路径替换为您自己的`xiadan.exe`路径

```
运行此代码后，程序将自动操控同花顺客户端，填入代码`000001`、价格`10.50`，并点击卖出100股。您可以在客户端的“委托”或“持仓”页面中查看到这笔挂单。

## 集成到量化策略
演示了单次下单后，我们可以将其集成到更完整的量化交易流程中。

这个自动化下单模块可以作为您量化策略的执行端。以下是典型的集成思路：
1.  使用Python进行市场数据分析，生成交易信号（例如：计算买入/卖出价格）。
2.  将计算得出的**股票代码**、**价格**和**数量**传递给上述的 `user.sell()` 或 `user.buy()` 函数。
3.  程序便会自动执行下单操作。

此外，`easytrader` 库也提供了其他辅助功能，例如：
*   `user.balance`：获取账户资金余额。
*   `user.position`：获取当前持仓信息。
*   `user.today_trades`：获取当日成交记录。
