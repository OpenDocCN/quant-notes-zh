## 概述：什么是支持向量机？

支持向量机（Support Vector Machine, SVM）是一种经典的机器学习算法。在深度学习兴起之前，它在机器学习领域有着非常广泛的应用。SVM包含两个主要分支：用于分类的SVC（Support Vector Classification）和用于回归的SVR（Support Vector Regression）。本节课我们将基于SVC来讲解其原理，理解了分类问题，回归问题也就触类旁通。

SVM的核心思想是通过一个“超平面”来对数据进行分割。对于二维数据，这个超平面就是一条直线；对于三维数据，就是一个平面。推广开来，在P维空间中，SVM试图找到一个P-1维的超平面，将数据点最优地分成两类。

## 核心思想：寻找最优决策边界

![](img/ad0bc726f21799971f9643e591fb72f8_2.png)

![](img/ad0bc726f21799971f9643e591fb72f8_4.png)

上一节我们概述了SVM的基本概念，本节中我们来看看它是如何找到那个分割数据的“最佳”超平面的。

SVM的目标是找到一个决策边界（即超平面），使得两类数据中离这个边界最近的那些点（称为“支持向量”）到边界的距离最大化。这个“距离”被称为“间隔”（Margin）。SVM的优化目标就是最大化这个间隔。

以下是理解这一过程的关键步骤：

1.  **识别支持向量**：决定最优超平面的并不是所有数据点，而是那些离潜在分界线最近的点，这些点被称为“支持向量”。
2.  **计算间隔**：间隔是指所有支持向量到决策边界距离中的最小值。
3.  **最大化间隔**：SVM算法通过调整超平面的位置和方向，使得这个最小距离达到最大。这样的超平面被认为是最鲁棒、泛化能力最强的。

为了更直观地理解，我们可以对比两种不同的分割方式。第一种分割线虽然能分开数据，但有些点离边界太近，间隔很小；第二种分割线则让所有支持向量都离边界足够远，从而实现了更大的间隔。显然，第二种是更好的分割方式。

## 关键概念：核函数（Kernel）

![](img/ad0bc726f21799971f9643e591fb72f8_8.png)

上一节我们介绍了SVM在线性可分数据上的工作原理，本节中我们来看看当数据本身线性不可分时，SVM如何应对。

![](img/ad0bc726f21799971f9643e591fb72f8_10.png)

![](img/ad0bc726f21799971f9643e591fb72f8_12.png)

![](img/ad0bc726f21799971f9643e591fb72f8_14.png)

有些数据在原始的低维空间中无法用一条直线（或一个平面）完美分开。这时，SVM引入了“核函数”这一强大工具。核函数的核心思想是：将数据从原始的低维空间映射到一个更高维的空间。在这个高维空间中，原本线性不可分的数据可能就变得线性可分了。

![](img/ad0bc726f21799971f9643e591fb72f8_16.png)

例如，在二维平面上的一堆环形分布的数据点无法用直线分开。但如果通过一个核函数将其映射到三维空间，我们可能就能找到一个平面（三维空间中的超平面）将其完美分割。映射完成后，问题又回到了在高维空间中寻找最大间隔超平面的标准SVM问题。

![](img/ad0bc726f21799971f9643e591fb72f8_22.png)

![](img/ad0bc726f21799971f9643e591fb72f8_24.png)

![](img/ad0bc726f21799971f9643e591fb72f8_28.png)

在实际应用中，有多种核函数可供选择，它们决定了数据映射到高维空间后的形态，从而影响决策边界的形状：

![](img/ad0bc726f21799971f9643e591fb72f8_30.png)

![](img/ad0bc726f21799971f9643e591fb72f8_32.png)

![](img/ad0bc726f21799971f9643e591fb72f8_34.png)

*   **线性核（Linear）**：`kernel = ‘linear’`。直接在原始空间寻找线性分界，决策边界是直线或平面。
*   **多项式核（Polynomial）**：`kernel = ‘poly’`。决策边界是多项式曲线，可以更弯曲以适应复杂数据。
*   **径向基函数核/高斯核（RBF）**：`kernel = ‘rbf’`。这是最常用的核函数之一，能产生非常灵活、复杂的非线性决策边界。

## 实例解析：从线性不可分到线性可分

为了加深对核函数的理解，我们来看一个具体的例子。假设在二维空间中有四个点：红点(0,3)和(3,0)，蓝点(1,2)和(2,1)。在二维平面上，我们无法找到一条直线将它们完美分开。

此时，我们可以通过新增一个特征维度（例如 `Z = X * Y`）将数据映射到三维空间：
*   红点(0,3) -> (0, 3, 0)
*   红点(3,0) -> (3, 0, 0)
*   蓝点(1,2) -> (1, 2, 2)
*   蓝点(2,1) -> (2, 1, 2)

在三维空间中，我们可以很容易地找到一个平面（例如在Z=1附近）将这两类点清晰地分开。这个“新增特征”的过程，其本质就是应用了一个特定的核函数（本例中类似于多项式核），实现了从低维不可分到高维可分的转换。

## SVM与逻辑回归的对比

![](img/ad0bc726f21799971f9643e591fb72f8_36.png)

![](img/ad0bc726f21799971f9643e591fb72f8_38.png)

![](img/ad0bc726f21799971f9643e591fb72f8_40.png)

在结束之前，我们简要对比一下SVM和之前学过的逻辑回归。两者虽然都可用于分类，但侧重点不同：
*   **逻辑回归**：所有数据点都参与决策边界的确定，其目标是找到概率分界线。
*   **支持向量机**：只关注“支持向量”，即最难分的边界点，其目标是最大化分类间隔，这通常使得SVM对噪声和异常值有更好的鲁棒性，并且通过核函数能处理更复杂的非线性问题。
