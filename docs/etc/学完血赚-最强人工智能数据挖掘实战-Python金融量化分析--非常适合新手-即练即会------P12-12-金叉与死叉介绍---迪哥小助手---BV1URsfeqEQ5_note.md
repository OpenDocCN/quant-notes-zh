## 什么是双均线？ 📊

上一节我们介绍了股票数据分析的基础，本节中我们来看看如何更好地描述股价的整体趋势。仅观察每日收盘价的波动，难以把握其长期走向。为了解决这个问题，我们引入“双均线”的概念。

![](img/d2446d9a6791387d49310332c810aba8_10.png)

双均线，顾名思义，指的是两条移动平均线。一条是短期均线，另一条是长期均线。

![](img/d2446d9a6791387d49310332c810aba8_13.png)

### 短期均线

短期均线用于反映股价近期的平均走势。例如，我们可以计算过去5个交易日（通常代表一周）收盘价的平均值。

![](img/d2446d9a6791387d49310332c810aba8_15.png)

![](img/d2446d9a6791387d49310332c810aba8_16.png)

![](img/d2446d9a6791387d49310332c810aba8_17.png)

![](img/d2446d9a6791387d49310332c810aba8_18.png)

**公式**：`短期均线值 = (第1天收盘价 + 第2天收盘价 + ... + 第5天收盘价) / 5`

![](img/d2446d9a6791387d49310332c810aba8_19.png)

![](img/d2446d9a6791387d49310332c810aba8_20.png)

![](img/d2446d9a6791387d49310332c810aba8_21.png)

![](img/d2446d9a6791387d49310332c810aba8_22.png)

计算时，我们采用滑动窗口的方式：先计算第1天到第5天的均值，接着计算第2天到第6天的均值，以此类推。将这些均值连接起来，就形成了一条比原始日K线更平滑的曲线，它能更清晰地显示短期趋势。

![](img/d2446d9a6791387d49310332c810aba8_23.png)

![](img/d2446d9a6791387d49310332c810aba8_24.png)

![](img/d2446d9a6791387d49310332c810aba8_25.png)

![](img/d2446d9a6791387d49310332c810aba8_26.png)

### 长期均线

![](img/d2446d9a6791387d49310332c810aba8_28.png)

长期均线的计算原理与短期均线完全相同，唯一的区别在于选取的窗口期更长。

**公式**：`长期均线值 = (过去N天收盘价之和) / N`，其中N通常取20、40或60等，分别代表约一个月、两个月或三个月的交易日。

![](img/d2446d9a6791387d49310332c810aba8_30.png)

![](img/d2446d9a6791387d49310332c810aba8_31.png)

![](img/d2446d9a6791387d49310332c810aba8_32.png)

长周期能过滤掉更多的短期噪音，更好地反映股价的长期趋势。

![](img/d2446d9a6791387d49310332c810aba8_33.png)

![](img/d2446d9a6791387d49310332c810aba8_34.png)

![](img/d2446d9a6791387d49310332c810aba8_35.png)

### 1. 黄金交叉（金叉）✅

![](img/d2446d9a6791387d49310332c810aba8_49.png)

当短期均线从下方向上穿越长期均线时，形成黄金交叉。这通常被视为一个**买入信号**。

![](img/d2446d9a6791387d49310332c810aba8_51.png)

![](img/d2446d9a6791387d49310332c810aba8_52.png)

![](img/d2446d9a6791387d49310332c810aba8_53.png)

![](img/d2446d9a6791387d49310332c810aba8_54.png)

**逻辑**：短期均线向上突破长期均线，表明近期市场平均成本开始高于长期平均成本，买方力量增强，股价短期内可能开启上涨趋势。此时买入，旨在“低买”，期待后续“高卖”。

![](img/d2446d9a6791387d49310332c810aba8_55.png)

![](img/d2446d9a6791387d49310332c810aba8_56.png)

### 2. 死亡交叉（死叉）❌

![](img/d2446d9a6791387d49310332c810aba8_58.png)

![](img/d2446d9a6791387d49310332c810aba8_59.png)

![](img/d2446d9a6791387d49310332c810aba8_60.png)

当短期均线从上方向下穿越长期均线时，形成死亡交叉。这通常被视为一个**卖出信号**。

![](img/d2446d9a6791387d49310332c810aba8_61.png)

![](img/d2446d9a6791387d49310332c810aba8_62.png)

**逻辑**：短期均线向下跌破长期均线，表明近期市场平均成本已低于长期平均成本，卖方力量占优，股价短期内可能进入下跌趋势。此时卖出，是为了规避下跌风险，锁定利润或减少亏损。

![](img/d2446d9a6791387d49310332c810aba8_64.png)

![](img/d2446d9a6791387d49310332c810aba8_65.png)

## 双均线交易策略 🎯

![](img/d2446d9a6791387d49310332c810aba8_66.png)

![](img/d2446d9a6791387d49310332c810aba8_67.png)

理解了金叉和死叉的含义后，我们就可以构建一个简单的交易策略。

![](img/d2446d9a6791387d49310332c810aba8_70.png)

**策略逻辑**：
1.  当出现**黄金交叉**时，执行买入操作。
2.  当出现**死亡交叉**时，执行卖出操作。

理想情况下，若能准确识别每一次交叉信号并据此操作，理论上可以实现“买在起涨点，卖在下跌前”，从而累积收益。

本节课中我们一起学习了双均线策略的核心概念。我们首先解释了短期均线和长期均线的计算方法及其意义，它们分别代表了股价的近期和长期趋势。接着，我们重点探讨了**金叉**与**死叉**这两个关键信号：金叉是短期均线上穿长期均线，提示买入机会；死叉是短期均线下穿长期均线，提示卖出时机。基于这两个信号，可以构建一套清晰的交易策略。记住，这个策略是许多量化交易模型的基础，在后续课程中我们将动手用代码实现它。
