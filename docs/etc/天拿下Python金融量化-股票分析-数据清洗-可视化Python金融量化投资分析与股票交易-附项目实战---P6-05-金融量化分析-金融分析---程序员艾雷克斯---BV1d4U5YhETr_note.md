# Python金融量化投资分析与股票交易：P6：05 金融量化分析-金融分析 📈

在本节课中，我们将要学习金融分析的核心方法。了解如何通过基本面和技术面分析来判断股票的投资价值，避免盲目买卖。

上一节我们介绍了金融和股票的基础知识。本节中，我们来看看如何对股票进行具体的分析。买卖股票不能仅凭感觉，需要借助科学的分析方法来判断其未来走势。盲目买卖无异于赌博，通常会导致亏损。

金融分析主要包含两大方面的分析方法：基本面分析和技术面分析。

## 基本面分析

基本面分析的核心是评估公司的实际运营状况。这主要基于我们之前提到的、影响股价的公司自身因素。你需要分析当前宏观经济状况、行业发展前景以及公司具体的经营情况，以此来决定是否购买其股票。

基本面分析包含以下三个层面：

以下是宏观经济面分析。这包括分析国家的财政政策、货币政策等。例如，判断国家当前是鼓励资金流入股市还是倾向于储蓄。但请注意，宏观经济分析不应过分依赖，因为市场表现有时并不完全遵循宏观规律。

以下是行业分析。这属于相对宏观的判断，需要评估整个行业的发展态势。例如，当年的教育、IT或传统工业（如钢铁、煤炭）哪个行业更具发展潜力。你可以通过提取该行业中几只具有代表性的股票，观察其整体走势来辅助判断。

以下是公司分析。这是最具体的一层。例如，如果你打算购买贵州茅台的股票，就需要深入研究这家公司。上市公司必须定期公开其财务数据。每年公司会发布几次财报，其中3月份左右发布年度财报，每个季度也会发布季度财报。如果你长期跟踪一家公司，对其经营状况有深入了解，甚至可以在利好财报发布前提前布局。对于普通投资者，则可以通过分析已发布的财务报表（如公司盈利、每股收益等）来客观评估公司价值。这些经过审计的财报数据通常是可靠的信息来源。综合财务数据、新闻资讯甚至实地考察，可以判断公司运营是否良好。如果公司运营出色、盈利能力强，则可以考虑购买其股票。

## 技术面分析

![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_1.png)

上一节我们介绍了基本面分析，它侧重于公司的内在价值。本节中我们来看看技术面分析。技术面分析的核心观点是：**所有信息都已蕴含在市场交易数据之中**。它通过分析历史市场走势（如前一个月、一周的价格变动）和技术指标来预测未来价格动向。但仅看涨跌并不足够，因此需要借助定义明确的技术指标。

以下是两种常见的技术指标：K线和移动平均线（MA）。

### K线图

K线图是展示股票每日价格走势的图表。如下图所示，横轴代表时间，纵轴代表股票价格。图中每一天都对应一根“K线”。



![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_3.png)

![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_1.png)


![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_5.png)

一根K线包含了四个关键价格：开盘价、收盘价、最高价和最低价。K线分为阳线和阴线。



![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_3.png)


*   **阳线**（通常为红色或空心）：表示当日股价上涨，即收盘价高于开盘价。
*   **阴线**（通常为绿色或蓝色/实心）：表示当日股价下跌，即收盘价低于开盘价。

K线的结构如下：



![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_5.png)


*   **实体**：中间的矩形部分。其实体的**上、下边缘**分别代表收盘价和开盘价（阳线），或开盘价和收盘价（阴线）。
*   **影线**：实体上方和下方的细线。**上影线顶端**代表当日最高价，**下影线底端**代表当日最低价。

K线可能存在一些特殊形态，例如“十字星”（开盘价等于收盘价）或没有影线的K线（开盘价/收盘价同时是最高价/最低价）。不同的K线形态可用于辅助分析。

### 移动平均线（MA）

介绍完K线，我们来看另一个重要指标：移动平均线。均线的作用是平滑价格波动，显示趋势方向。



![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_7.png)


![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_7.png)

图中白色的`MA5`线代表5日均线，黄色的`MA60`线代表60日均线。



![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_9.png)

![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_9.png)


*   **计算公式（以N日均线为例）**：
    `MA(N) = (P1 + P2 + ... + PN) / N`
    其中，`P1` 代表前第1天的收盘价（或特定价格），`PN`代表前第N天的价格。
*   **含义**：均线是取前若干天价格的平均值。例如，5日均线值是包括当天在内向前回溯5个交易日的价格平均值。将每一天的均线值连接起来，就形成了移动平均线。它英文全称是 **Moving Average**，反映了近期价格的平均趋势。



![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_11.png)


均线在量化策略中应用广泛，例如后续会讲到的“双均线交叉策略”。

![](img/1f3a83b8e3e6ad9bdb90bf848dbda2fb_11.png)

本节课中我们一起学习了金融分析的两大支柱：基本面分析和技术面分析。基本面分析从宏观经济、行业到公司层面评估内在价值；技术面分析则利用K线、均线等工具研究市场交易数据本身蕴含的规律。理解这些方法是进行理性投资和后续量化策略开发的基础。
