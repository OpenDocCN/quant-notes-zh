## 数据刷新与源数据更改 🔄

![](img/d523edbeca28f50d6924d758bb726ef3_1.png)

上一节我们介绍了如何创建数据透视表。本节中我们来看看当原始数据发生变化时，如何更新数据透视表。

当数据源中的数值被修改后，数据透视表不会自动更新。此时，需要手动刷新。

**操作步骤：**
1.  将鼠标光标定位在数据透视表区域内。
2.  此时会出现“数据透视表工具”上下文选项卡。
3.  点击“分析”标签页下的“刷新”按钮，选择“全部刷新”。

![](img/d523edbeca28f50d6924d758bb726ef3_3.png)

**代码/公式描述：**
*   功能位置：`数据透视表工具` -> `分析` -> `刷新` -> `全部刷新`

![](img/d523edbeca28f50d6924d758bb726ef3_5.png)

如果新增的数据行或列在原始数据区域的**外部**（例如在表格底部新增一行），仅刷新无效。因为创建透视表时已固定了数据源范围。此时需要更改数据源。

![](img/d523edbeca28f50d6924d758bb726ef3_7.png)

![](img/d523edbeca28f50d6924d758bb726ef3_9.png)

**操作步骤：**
1.  将鼠标光标定位在数据透视表区域内。
2.  点击“数据透视表工具”下的“分析”标签页。
3.  点击“更改数据源”按钮，重新选择包含新增数据的整个区域。

## 对计算结果进行排序 📈

![](img/d523edbeca28f50d6924d758bb726ef3_11.png)

我们常常需要查看销量最高或最低的项目。在普通表格中，我们可以直接使用排序功能，但在数据透视表中，直接使用功能区排序会报错。

正确的排序方法是利用数据透视表自带的排序选项。

**操作步骤：**
1.  点击行标签字段右侧的下拉箭头。
2.  选择“其他排序选项”。
3.  在弹出的对话框中，选择“降序排序（从大到小）”。
4.  在“排序依据”中，选择需要依据的数值字段（例如“求和项：销量”）。

**代码/公式描述：**
*   功能路径：`行标签下拉菜单` -> `其他排序选项` -> `降序排序` -> `依据：[求和项:销量]`

## 添加计算字段与计算项 ➕

有时我们需要在数据透视表的基础上进行新的计算，例如计算“均价”（收入/销量）。这可以通过添加“计算字段”实现。

**操作步骤（添加计算字段 - 针对列）：**
1.  将鼠标光标定位在数据透视表的列标签或数值区域。
2.  点击“数据透视表工具”下的“分析”标签页。
3.  点击“字段、项目和集”下拉菜单，选择“计算字段”。
4.  在对话框中，输入名称（如“均价”），在公式框中输入：`=收入额 / 销量`。
5.  点击“添加”后确定。

**代码/公式描述：**
*   功能路径：`分析` -> `字段、项目和集` -> `计算字段`
*   公式示例：`均价 = 收入额 / 销量`

如果需要在行方向上进行计算（例如计算两个品牌销量的差值），则需要使用“计算项”。

**操作步骤（添加计算项 - 针对行）：**
1.  将鼠标光标定位在数据透视表的行标签区域（例如“品牌”字段）。
2.  点击“数据透视表工具”下的“分析”标签页。
3.  点击“字段、项目和集”下拉菜单，此时“计算项”变为可用，点击它。
4.  在对话框中，输入名称（如“差值”），在公式框中构造计算，例如：`=宝马 - 丰田`。
5.  点击确定。

## 优化数据透视表外观 🎨

默认生成的数据透视表格式可能不符合阅读习惯。我们可以对其进行美化，使其更接近常见的报表样式。

以下是几个关键的优化步骤：

**操作列表：**
*   **隐藏字段按钮**：在“数据透视表工具”的“分析”标签页下，取消勾选“字段按钮”。
*   **以表格形式显示**：在“设计”标签页下，点击“报表布局”，选择“以表格形式显示”。
*   **隐藏分类汇总**：在“设计”标签页下，点击“分类汇总”，选择“不显示分类汇总”。
*   **合并居中标签单元格**：点击“分析”标签页下的“选项”，在“布局和格式”选项卡中，勾选“合并且居中排列带标签的单元格”。

**提示**：若不小心关闭了“数据透视表字段”窗格，只需在“分析”标签页下，点击“字段列表”按钮即可重新显示或隐藏。

## 使用切片器进行动态筛选 🔍

切片器提供了一种直观、交互式的筛选方式，可以动态控制数据透视表中显示的内容。

**操作步骤：**
1.  将鼠标光标定位在数据透视表区域内。
2.  点击“分析”标签页下的“插入切片器”。
3.  在弹出的对话框中，选择希望用于筛选的字段（例如“地区”、“品牌”）。
4.  点击确定后，会出现对应的切片器。点击切片器中的项目，数据透视表会立即显示筛选后的结果。
5.  按住 `Ctrl` 键可以多选。点击切片器右上角的“清除筛选器”图标可取消筛选。
6.  选中切片器，按 `Delete` 键可将其删除。

## 根据类别拆分生成多个工作表 📑

![](img/d523edbeca28f50d6924d758bb726ef3_15.png)

![](img/d523edbeca28f50d6924d758bb726ef3_17.png)

如果需要将数据按某个维度（如地区）拆分成独立的工作表分发给不同负责人，可以使用“显示报表筛选页”功能。

**操作步骤：**
1.  将需要拆分的字段（如“地区”）拖拽到数据透视表的“筛选器”区域。
2.  将鼠标光标定位在数据透视表区域内。
3.  点击“分析”标签页下的“选项”按钮，选择“显示报表筛选页”。
4.  在弹出的对话框中，选择要拆分的字段（即刚才放入筛选器的字段），点击确定。
5.  Excel会自动创建以每个筛选项命名的新工作表，并包含对应的数据透视表。

## 合并多个结构相同的工作表 🤝

当有多个格式完全相同的工作表（如各季度销售表）需要合并分析时，可以使用“多重合并计算数据区域”功能。

首先，需要将此功能添加到功能区或使用快捷键。

![](img/d523edbeca28f50d6924d758bb726ef3_27.png)

![](img/d523edbeca28f50d6924d758bb726ef3_29.png)

**添加到功能区的步骤：**
1.  点击“文件” -> “选项” -> “自定义功能区”。
2.  在“从下列位置选择命令”下拉菜单中选择“不在功能区中的命令”。
3.  在列表中找到并选中“数据透视表和数据透视图向导”。
4.  在右侧主选项卡列表中，选择“数据”选项卡，新建一个组，然后点击“添加”按钮。
5.  点击确定后，该功能会出现在“数据”选项卡下。

**使用快捷键**：按 `Alt` + `D` + `P` 键可直接调出向导。

![](img/d523edbeca28f50d6924d758bb726ef3_31.png)

**合并操作步骤：**
1.  调出“数据透视表和数据透视图向导”。
2.  选择“多重合并计算数据区域”，点击下一步。
3.  选择“创建单页字段”，点击下一步。
4.  依次选择每个需要合并的工作表的数据区域，并点击“添加”按钮。
5.  添加完所有区域后，点击下一步，选择放置数据透视表的位置，点击完成。

## 创建数据透视图 📉

![](img/d523edbeca28f50d6924d758bb726ef3_35.png)

![](img/d523edbeca28f50d6924d758bb726ef3_37.png)

数据透视图是基于数据透视表创建的图表，它能与数据透视表联动，实现动态的图表展示。

**操作步骤：**
1.  将鼠标光标定位在数据透视表区域内。
2.  点击“分析”标签页下的“数据透视图”按钮。
3.  在弹出的对话框中选择想要的图表类型（如柱形图），点击确定。
4.  生成的数据透视图会带有筛选字段。调整数据透视表中的筛选器或切片器，图表会同步更新。

![](img/d523edbeca28f50d6924d758bb726ef3_39.png)
