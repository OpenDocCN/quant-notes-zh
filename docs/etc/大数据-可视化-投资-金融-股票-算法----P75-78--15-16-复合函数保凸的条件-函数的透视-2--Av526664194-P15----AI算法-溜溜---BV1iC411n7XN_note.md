## 复合函数保凸条件的进一步探讨

![](img/cfca5e4a5865f22e9b247c409a95f84a_7.png)

上一节我们介绍了复合函数保凸的一般条件，本节中我们来看看几个具体的应用例子，并分析一个可能产生困惑的情况。

**例子三：函数 `1/z` 的扩展**
考虑函数 `1/z`。当 `z > 0` 时，它是一个单调递减的凸函数。如果我们对其进行扩展，它成为一个单调不增的函数。现在，假设 `h(z)` 是 `1/z` 的扩展（单调不增），且 `g(x)` 是一个凹函数。那么，根据复合函数保凸的第二条性质，复合函数 `g(x)/1`（即 `g(x)`）的某种形式）可能具有特定的凸性。但更常见的应用是：若 `g(x)` 是凸函数，则 `1/g(x)` 在 `g(x) > 0` 时是凸函数。

**例子四：凸函数的幂函数**
如果一个函数 `g(x)` 是凸函数，且其取值 `g(x) ≥ 0`，同时给定一个常数 `p ≥ 1`，那么函数 `[g(x)]^p` 也是一个凸函数。这是一个非常重要的性质，因为它简化了幂函数的凸性分析。我们只需判断内层函数 `g(x)` 的凸性即可。

![](img/cfca5e4a5865f22e9b247c409a95f84a_9.png)

![](img/cfca5e4a5865f22e9b247c409a95f84a_11.png)

为了应用复合函数保凸的性质，我们令外层函数 `h(z) = z^p`。现在，我们需要判断 `h(z)` 的凸凹性以及单调性。

以下是关键分析步骤：
1.  当 `p = 1` 时，`h(z) = z`。这是一条通过原点的直线，它既是凸函数又是凹函数。
2.  我们需要决定将其视为凸函数还是凹函数，以匹配复合函数保凸的条件。条件要求外层函数 `h` 是凸函数且单调非减。
3.  如果我们将 `h(z) = z` 视为凸函数，那么它的扩展（定义域扩展到整个实数轴）将不再是单调函数，这不符合“单调非减”的要求。
4.  因此，为了满足“凸且单调非减”的条件，我们**不能**简单地将 `p=1` 时的 `h(z)=z` 直接套用该性质。当 `p=1` 时，`[g(x)]^1 = g(x)`，其凸性直接由 `g(x)` 决定，无需通过复合函数性质判断。
5.  当 `p > 1` 时，`h(z) = z^p` 在 `z ≥ 0` 时是凸函数且单调递增，此时性质完全适用。

**核心结论**：对于凸函数 `g(x) ≥ 0`，当 `p ≥ 1` 时，函数 `[g(x)]^p` 是凸函数。当 `p=1` 时，即为 `g(x)` 本身；当 `p>1` 时，可通过复合函数（`h(z)=z^p`, `z≥0`）的保凸性来理解。

## 函数的透视

![](img/cfca5e4a5865f22e9b247c409a95f84a_13.png)

之前我们学习过“透视函数”，本节我们将介绍一个不同的概念——“函数的透视”。

**透视函数回顾**
透视函数 `P` 是一个从 `n+1` 维空间到 `n` 维空间的映射，定义如下：
`P(z, t) = z / t`，其中定义域要求 `t > 0`。

**函数的透视定义**
给定一个函数 `f: R^n → R`，它的透视函数 `g: R^n × R_{++} → R` 定义为：
`g(x, t) = t * f(x / t)`
其中，`t > 0`，并且 `x/t` 必须在 `f` 的定义域内。

**重要性质**
如果原函数 `f(x)` 是一个凸函数，那么它的透视函数 `g(x, t)` 也是一个凸函数（在定义域 `R^n × R_{++}` 上）。
这个性质非常有用，因为它允许我们将一个 `n` 维凸函数“提升”为一个 `n+1` 维的凸函数。

![](img/cfca5e4a5865f22e9b247c409a95f84a_15.png)

**证明要点**
在证明 `g(x,t)` 是凸函数时，需要注意两点：
1.  首先需要证明其定义域 `{ (x,t) | t>0, x/t ∈ dom f }` 是一个凸集。
2.  其次，在证明函数值的凸不等式时，必须对 `(x, t)` 这个 `n+1` 维的点做凸组合，而不是分别对 `x` 和 `t` 做凸组合。

我们称 `g(x,t)` 对于变量组 `(x,t)` 是“联合凸”的。这与“对于 `x` 是凸的”或“对于 `t` 是凸的”是不同的、更强的性质。

![](img/cfca5e4a5865f22e9b247c409a95f84a_17.png)

**例子一：欧几里得范数平方的透视**
令 `f(x) = x^T x = ||x||_2^2`（凸函数）。它的透视函数为：
`g(x, t) = t * f(x/t) = t * ( (x/t)^T (x/t) ) = (x^T x) / t`
因此，函数 `||x||_2^2 / t` 对于 `(x, t)`（其中 `t>0`）是联合凸的。

**例子二：负对数函数的透视**
令 `f(x) = -log(x)`（凸函数，定义域 `x>0`）。它的透视函数为：
`g(x, t) = t * (-log(x/t)) = -t log(x) + t log(t)`
根据性质，这个函数对于 `(x, t)`（其中 `x>0, t>0`）也是联合凸的。

**从透视函数到KL散度**
我们可以将负对数透视函数推广到向量形式。考虑两个向量 `u` 和 `v`（分量均大于0），构造如下函数：
`∑_{i} [ u_i * log(u_i / v_i) ]`
这个函数是凸的，因为它是多个（`u_i`, `v_i` 视为标量）凸的透视函数的和，且每个分量是独立的。

在此基础上，定义 **KL散度**：
`D_{KL}(u || v) = ∑_{i} [ u_i * log(u_i / v_i) - u_i + v_i ]`
由于减去了一个关于 `u` 的仿射函数 `∑ u_i`，加上了一个关于 `v` 的仿射函数 `∑ v_i`，而仿射变换保持凸性，因此 **KL散度 `D_{KL}(u || v)` 也是一个凸函数**。

![](img/cfca5e4a5865f22e9b247c409a95f84a_19.png)

![](img/cfca5e4a5865f22e9b247c409a95f84a_21.png)

KL散度是Bregman散度的一个特例。选择特定的凸函数（如负熵）可以导出KL散度，并且保证了其凸性，这在优化和信息论中非常有用。

## 函数的共轭

![](img/cfca5e4a5865f22e9b247c409a95f84a_32.png)

![](img/cfca5e4a5865f22e9b247c409a95f84a_34.png)

最后，我们介绍一个在凸优化对偶理论中至关重要的工具——函数的共轭。

![](img/cfca5e4a5865f22e9b247c409a95f84a_36.png)

**定义**
对于一个函数 `f: R^n → R`，它的共轭函数 `f^*: R^n → R` 定义为：
`f^*(y) = sup_{x ∈ dom f} { y^T x - f(x) }`
其中，`sup` 表示上确界（最大值）。

![](img/cfca5e4a5865f22e9b247c409a95f84a_37.png)

![](img/cfca5e4a5865f22e9b247c409a95f84a_39.png)

**几何解释（以一维为例）**
考虑函数 `y = f(x)`。对于给定的斜率 `y`，`y^T x - f(x)` 的几何意义是：斜率为 `y` 的直线 `l(x) = y * x` 与函数曲线 `f(x)` 在 `x` 点处的垂直距离。共轭函数值 `f^*(y)` 就是所有 `x` 点对应的这个垂直距离的最大值。在图像上，它对应于斜率为 `y` 的直线与函数 `f(x)` 的“最远”垂直距离。

**重要性质**
以下是共轭函数的两个关键性质：
1.  **可微函数的极值点**：如果 `f(x)` 可微，那么使 `y^T x - f(x)` 达到上确界的 `x` 满足 `∇f(x) = y`。这从求导为零（`y - f'(x) = 0`）可以直接得出。
2.  **共轭函数总是凸的**：无论原函数 `f(x)` 是否是凸函数，它的共轭函数 `f^*(y)` **一定是一个凸函数**。这是因为对于每一个固定的 `x`，`y^T x - f(x)` 都是 `y` 的线性（仿射）函数。`f^*(y)` 是无穷多个这样的线性函数的逐点上确界，而上确界运算保持凸性。

这个性质非常强大：它提供了一种将任意函数（甚至是非凸函数）转化为凸函数的方法。共轭函数在对偶理论中扮演着核心角色。

## 总结
本节课中我们一起学习了：
1.  **复合函数保凸条件的应用**：重点分析了幂函数 `[g(x)]^p` 的凸性条件，明确了当 `g(x)` 凸且非负、`p ≥ 1` 时，该幂函数为凸函数。
2.  **函数的透视**：学习了如何通过 `g(x,t) = t * f(x/t)` 从一个凸函数 `f` 构造出更高维的联合凸函数 `g`，并以欧几里得范数平方和负对数为例，推导出KL散度这一重要凸函数。
3.  **函数的共轭**：掌握了共轭函数 `f^*(y) = sup_x {y^T x - f(x)}` 的定义、几何意义及其核心性质——共轭函数总是凸的。这为后续学习优化问题的对偶理论打下了坚实基础。

这些概念是凸优化理论中的高级工具，理解它们有助于我们更深入地分析金融模型中的优化问题。
