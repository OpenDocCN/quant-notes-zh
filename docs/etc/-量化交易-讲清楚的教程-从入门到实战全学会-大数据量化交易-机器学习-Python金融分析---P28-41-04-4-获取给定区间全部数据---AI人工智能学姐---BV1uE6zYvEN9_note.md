## 概述

![](img/21a85d28bb69ba921374859022b41f8e_7.png)

![](img/21a85d28bb69ba921374859022b41f8e_9.png)

上一节我们介绍了如何获取单日的因子数据。本节中，我们来看看如何扩展这个功能，获取一个连续时间段内的所有数据。核心思路是遍历日期序列中的每一天，获取当天的数据，然后将每天的数据拼接成一个完整的 `DataFrame`。

## 遍历日期序列

![](img/21a85d28bb69ba921374859022b41f8e_11.png)

![](img/21a85d28bb69ba921374859022b41f8e_13.png)

首先，我们需要遍历指定的日期区间。假设我们有一个日期序列，目标是获取其中每一天的数据。

![](img/21a85d28bb69ba921374859022b41f8e_15.png)

以下是实现遍历的基本代码结构：
```python
# 假设 dates 是一个日期序列
for i in range(len(dates)):
    current_date = dates[i]
    # 在此处获取 current_date 的数据
```

![](img/21a85d28bb69ba921374859022b41f8e_17.png)

## 获取每日数据并处理

![](img/21a85d28bb69ba921374859022b41f8e_19.png)

在循环体内，对于每一天，我们需要调用获取因子数据的函数。该函数通常会返回一个三维数组，我们只需要其中包含实际数值的部分。

以下是获取并处理单日数据的代码：
```python
# 获取第 i 天的因子数据
daily_data = get_factor_on_date(i)  # 假设的获取函数
# 通常我们只关心第三个维度的数据（指标值）
value = daily_data[0, 2, :]  # 根据实际数据结构调整索引
```

![](img/21a85d28bb69ba921374859022b41f8e_21.png)

![](img/21a85d28bb69ba921374859022b41f8e_23.png)

## 构建每日数据表

![](img/21a85d28bb69ba921374859022b41f8e_25.png)

获取到每日的数值后，我们需要将其与对应的日期信息一起，构建成一个小的数据表（`DataFrame`）。

![](img/21a85d28bb69ba921374859022b41f8e_27.png)

以下是构建每日 `DataFrame` 的步骤：
1.  创建一个新的 `DataFrame`。
2.  将因子数值添加为一列，并为其命名（例如 `‘factor_value’`）。
3.  将当前日期添加为另一列（例如 `‘date’`）。

代码示例如下：
```python
import pandas as pd

![](img/21a85d28bb69ba921374859022b41f8e_29.png)

# 为第 i 天创建 DataFrame
daily_df = pd.DataFrame()
daily_df[‘factor_value‘] = value  # 添加因子值列
daily_df[‘date‘] = dates[i]       # 添加日期列
```

![](img/21a85d28bb69ba921374859022b41f8e_31.png)

![](img/21a85d28bb69ba921374859022b41f8e_33.png)

## 拼接成总表

![](img/21a85d28bb69ba921374859022b41f8e_35.png)

在每次循环中，我们都会生成一个代表单日数据的 `daily_df`。最后，我们需要将所有单日的数据表拼接起来，形成一个包含整个时间区间数据的总表。

![](img/21a85d28bb69ba921374859022b41f8e_37.png)

以下是拼接操作的实现：
1.  在循环开始前，初始化一个空的 `DataFrame` 作为总表。
2.  在每次循环中，使用 `pd.concat()` 函数将当天的 `daily_df` 拼接到总表中。

![](img/21a85d28bb69ba921374859022b41f8e_39.png)

代码示例如下：
```python
# 初始化一个空的 DataFrame 作为总表
total_df = pd.DataFrame()

![](img/21a85d28bb69ba921374859022b41f8e_41.png)

![](img/21a85d28bb69ba921374859022b41f8e_43.png)

# 在循环内部进行拼接
for i in range(len(dates)):
    # ... (获取数据并构建 daily_df 的代码)
    total_df = pd.concat([total_df, daily_df], ignore_index=True)
```

![](img/21a85d28bb69ba921374859022b41f8e_45.png)

**注意**：`pd.concat()` 操作后必须将结果赋值回原变量，否则拼接无效。
