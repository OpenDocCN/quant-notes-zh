## 第三步：计算短期与长期移动平均线

![](img/3e8f9020d8df01005b2d90758516f657_46.png)

![](img/3e8f9020d8df01005b2d90758516f657_48.png)

数据准备就绪后，我们可以开始计算技术指标。移动平均线是分析趋势的核心工具。我们将计算短期和长期简单移动平均线（SMA）。

![](img/3e8f9020d8df01005b2d90758516f657_50.png)

以下是计算移动平均线的步骤：

![](img/3e8f9020d8df01005b2d90758516f657_52.png)

1.  **定义窗口期**：设定短期（如42天）和长期（如252天）的移动平均窗口。
2.  **计算移动平均值**：对股价数据应用滚动窗口，并计算每个窗口内的平均值。

![](img/3e8f9020d8df01005b2d90758516f657_54.png)

![](img/3e8f9020d8df01005b2d90758516f657_56.png)

```python
# 定义短期和长期的窗口大小
short_window = 42
long_window = 252

![](img/3e8f9020d8df01005b2d90758516f657_58.png)

![](img/3e8f9020d8df01005b2d90758516f657_60.png)

# 计算短期移动平均线 (SMA)
apple_data['SMA_short'] = apple_data['AAPL'].rolling(window=short_window).mean()

![](img/3e8f9020d8df01005b2d90758516f657_62.png)

# 计算长期移动平均线 (SMA)
apple_data['SMA_long'] = apple_data['AAPL'].rolling(window=long_window).mean()

![](img/3e8f9020d8df01005b2d90758516f657_64.png)

![](img/3e8f9020d8df01005b2d90758516f657_66.png)

# 查看计算结果（尾部数据，因为前期窗口不足会产生NaN）
print(apple_data.tail())
```

![](img/3e8f9020d8df01005b2d90758516f657_68.png)

由于移动平均计算需要足够的历史数据，初始的一些行会显示为`NaN`（非数字），这是正常现象。

## 第五步：标记买卖信号位置

![](img/3e8f9020d8df01005b2d90758516f657_88.png)

为了量化分析，我们需要在数据中明确标记出这些信号。我们将创建一个新的列（`Position`），用于指示当前短期均线是高于还是低于长期均线。

![](img/3e8f9020d8df01005b2d90758516f657_90.png)

在此之前，需要先清除在计算移动平均线时产生的`NaN`值。

```python
# 删除因计算移动平均而产生的缺失值
apple_data.dropna(inplace=True)

![](img/3e8f9020d8df01005b2d90758516f657_92.png)

# 创建信号列：当短期均线高于长期均线时标记为1，否则为0
apple_data['Position'] = np.where(apple_data['SMA_short'] > apple_data['SMA_long'], 1, 0)

![](img/3e8f9020d8df01005b2d90758516f657_94.png)

![](img/3e8f9020d8df01005b2d90758516f657_96.png)

# 查看带有信号标记的数据
print(apple_data[['AAPL', 'SMA_short', 'SMA_long', 'Position']].head())
```

![](img/3e8f9020d8df01005b2d90758516f657_98.png)

`Position`列中的`1`代表短期趋势强于长期趋势（潜在持有或买入区间），`0`则相反。

## 第六步：理解策略逻辑与模拟思路

![](img/3e8f9020d8df01005b2d90758516f657_102.png)

现在，我们已经有了清晰的买卖信号标记。接下来探讨基于此信号的交易策略逻辑。

![](img/3e8f9020d8df01005b2d90758516f657_104.png)

*   **无策略情况**：如果从最初开始买入并一直持有到最后，你的收益完全取决于股价的最终涨跌。
*   **双均线策略**：根据我们标记的信号进行操作。
    *   当`Position`从0变为1时（黄金交叉），**买入**。
    *   当`Position`从1变为0时（死亡交叉），**卖出**。
    *   理论上，通过这种“高卖低买”的时机把握，可以在股价波动中获取比单纯持有更高的收益，或减少下跌时的损失。

例如，假设股价从100元跌至80元：
*   **单纯持有**：亏损20元。
*   **应用策略**：在下跌开始（死亡交叉）时以约100元卖出，在跌势企稳、出现买入信号（黄金交叉）时以约80元买回。不仅避免了亏损，同样的资金还能持有更多股份，实现了相对盈利。
