## 择时的核心要点与误区

![](img/685696ff0d9bc25476850b68c26d7f2f_17.png)

在了解了策略的表现后，我们需要掌握关于“择时”的两个关键认知：一个核心要点和一个重大误区。这对于正确应用任何择时策略都至关重要。

**一个核心要点**：择时的根本目的是指导**仓位配置**，而不仅仅是简单地判断市场是上涨还是下跌。其价值在于将趋势判断转化为具体的操作指令（如满仓、半仓、空仓）。

**一个重大误区**：将择时策略直接用于判断**个股**的多空趋势。这是错误的。择时分析的对象应该是整个市场或具有广泛代表性的板块。因此，择时策略应基于**指数**（如沪深300、上证指数）或多个指数的综合表现来构建，而不能应用于单个股票。
