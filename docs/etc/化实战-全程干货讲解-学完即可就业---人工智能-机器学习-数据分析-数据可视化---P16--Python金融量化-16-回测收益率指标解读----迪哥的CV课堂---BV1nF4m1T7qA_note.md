## 回测收益率详解

![](img/6e309e8e67622287ae97888849e76892_15.png)

![](img/6e309e8e67622287ae97888849e76892_17.png)

回测收益率是最基本、最直观的评估指标。它衡量的是，在整个回测周期结束后，你的初始本金通过策略操作最终变成了多少。其核心是计算相对收益。

![](img/6e309e8e67622287ae97888849e76892_19.png)

![](img/6e309e8e67622287ae97888849e76892_21.png)

**公式**如下：
`回测收益率 = (期末资产总值 - 期初本金) / 期初本金`
也可以表示为：
`回测收益率 = (P_t / P_0) - 1`
其中，`P_0`代表期初买入资产时的价格（对应期初本金），`P_t`代表期末卖出资产时的价格（对应期末资产总值）。

这个指标直接告诉我们，折腾了一番之后，相对于最初投入的钱，我们最终赚了（或亏了）百分之多少。

![](img/6e309e8e67622287ae97888849e76892_23.png)

## 实战计算：用Python计算回测收益率

![](img/6e309e8e67622287ae97888849e76892_25.png)

上一节我们介绍了回测收益率的概念和公式，本节中我们来看看如何用Python代码实际计算它。我们将使用一份示例股票数据来模拟策略回测后的每日资产净值走势。

![](img/6e309e8e67622287ae97888849e76892_27.png)

首先，导入必要的工具包并加载数据。

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便绘图显示
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设第一列是日期
data = pd.read_csv('your_stock_data.csv', index_col=0)
print(data.head())
```

假设`data[‘close’]`列代表了我们应用策略后，每日的资产净值。接下来，我们计算相对于第一天的净值比。

![](img/6e309e8e67622287ae97888849e76892_29.png)

```python
# 计算每日净值相对于期初净值的比值
net_value_ratio = data['close'] / data['close'].iloc[0]
# 绘制净值曲线
net_value_ratio.plot(figsize=(10, 6))
plt.title('策略净值增长曲线')
plt.ylabel('净值比 (相对于期初)')
plt.show()
```

![](img/6e309e8e67622287ae97888849e76892_31.png)

![](img/6e309e8e67622287ae97888849e76892_33.png)

这张图展示了从期初开始，每一块钱是如何随着策略运行而变化的。现在，我们来计算最终的回测收益率。

![](img/6e309e8e67622287ae97888849e76892_35.png)

![](img/6e309e8e67622287ae97888849e76892_37.png)

```python
# 计算最终的回测收益率： (期末净值 / 期初净值) - 1
final_return = (data['close'].iloc[-1] / data['close'].iloc[0]) - 1

![](img/6e309e8e67622287ae97888849e76892_39.png)

# 将结果以表格形式美观地展示
result_df = pd.DataFrame({
    '回测收益率': [final_return]
})
print(result_df)
```

![](img/6e309e8e67622287ae97888849e76892_41.png)

![](img/6e309e8e67622287ae97888849e76892_43.png)

执行以上代码，我们便得到了该策略在整个回测周期内的总收益率。这个值可能是正的（盈利），也可能是负的（亏损）。
