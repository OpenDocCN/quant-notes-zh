## 概述

上一节我们介绍了百分位法去极值，本节中我们来看看另一种基于中位数绝对偏差的方法，即MAD法。这种方法通过计算数据的中位数和绝对偏差来确定数据的正常范围，从而识别并处理异常值。

![](img/9e480e2b59826363622140087542b8e2_4.png)

## MAD法原理

MAD法的核心思想是：如果一个数据点距离数据中位数的距离，超过了某个基于“中位数绝对偏差”设定的阈值，那么它就被视为异常值。

以下是MAD值的计算步骤：

1.  首先，计算原始数据序列的中位数（Median）。
2.  然后，计算序列中每个数据点与中位数之差的绝对值，形成一个新的序列。
3.  最后，计算这个新序列的中位数，这个值就是MAD（Median Absolute Deviation）。

![](img/9e480e2b59826363622140087542b8e2_6.png)

![](img/9e480e2b59826363622140087542b8e2_8.png)

得到MAD值后，我们可以设定数据的正常范围。通常，上界和下界由以下公式决定：

![](img/9e480e2b59826363622140087542b8e2_10.png)

**上界 = 中位数 + N × MAD**
**下界 = 中位数 - N × MAD**

![](img/9e480e2b59826363622140087542b8e2_12.png)

其中，N是一个常数，通常取值为 **1.4826**。任何落在这个范围之外的数据点，都可以被视为极值并进行处理（例如，将其调整到边界值）。

![](img/9e480e2b59826363622140087542b8e2_14.png)

以下是实现MAD去极值功能的函数代码：

![](img/9e480e2b59826363622140087542b8e2_24.png)

```python
import pandas as pd
import numpy as np

![](img/9e480e2b59826363622140087542b8e2_26.png)

![](img/9e480e2b59826363622140087542b8e2_28.png)

def filter_extreme_mad(series, n):
    """
    使用MAD法对序列进行去极值处理。
    参数:
        series: 待处理的数据序列 (pd.Series)
        n: 用于计算边界的倍数，通常为1.4826
    返回:
        处理后的数据序列
    """
    # 1. 计算原始序列的中位数
    median = series.quantile(0.5)

    # 2. 计算每个数据点与中位数差值的绝对值
    abs_diff = (series - median).abs()

    # 3. 计算绝对值序列的中位数，即MAD值
    mad = abs_diff.quantile(0.5)

    # 4. 计算上界和下界
    high = median + n * mad
    low = median - n * mad

    # 5. 将超出边界的数据替换为边界值
    filtered_series = series.clip(lower=low, upper=high)

    return filtered_series
```
