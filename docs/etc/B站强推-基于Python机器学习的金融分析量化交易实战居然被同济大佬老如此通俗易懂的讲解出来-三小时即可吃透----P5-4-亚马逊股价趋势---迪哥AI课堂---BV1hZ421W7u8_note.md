## 模型评估方法

刚才我们构建并观察了基础模型的预测。本节中，我们来看看如何系统地评估这个模型的性能。

评估模型主要从两个方面进行：

1.  **预测误差**：比较预测值与真实值之间的差异。
2.  **趋势判断准确率**：判断模型是否能正确预测股价的上升或下降趋势。

以下是具体的评估步骤：

首先，划分训练集和测试集。例如，选取2014年至2017年的数据作为训练集，2017年至2018年的数据作为测试集。

![](img/b8c174852ccfbc415bbad1073369d6a1_39.png)

![](img/b8c174852ccfbc415bbad1073369d6a1_41.png)

```python
# 示例：划分训练集和测试集
train = df[‘2014-01-01‘:‘2017-01-01‘]
test = df[‘2017-01-01‘:‘2018-01-01‘]
```

![](img/b8c174852ccfbc415bbad1073369d6a1_43.png)

接着，在训练集上拟合模型，并预测测试集时间段内的股价。

然后，计算误差。误差可以通过平均绝对误差等指标来衡量，公式为：
**误差 = | 预测值 - 真实值 |**

![](img/b8c174852ccfbc415bbad1073369d6a1_45.png)

同时，评估趋势判断的准确性。方法是计算每一天股价相对于前一天的变动方向（上升或下降），然后对比模型预测的变动方向与实际变动方向是否一致。

```python
# 示例：计算预测趋势与实际趋势
predicted_trend = np.sign(np.diff(forecast[‘yhat‘].values))
actual_trend = np.sign(np.diff(test[‘y‘].values))
# 计算趋势判断正确的比例
trend_accuracy = np.mean(predicted_trend == actual_trend)
```

通过实际代码运行评估（过程中需注意关闭绘图窗口以让程序继续执行），我们得到以下关键指标：

*   **训练集平均误差**：约18
*   **测试集平均误差**：约160
*   **上升趋势预测正确率**：56%
*   **下降趋势预测正确率**：43.81%
