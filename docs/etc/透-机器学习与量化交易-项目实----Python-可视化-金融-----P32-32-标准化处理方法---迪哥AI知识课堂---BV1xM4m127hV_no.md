在本节课中，我们将要学习数据预处理中的一个核心步骤——标准化。标准化是机器学习项目中非常常见且重要的操作，它能帮助我们将不同量纲和取值范围的数据调整到同一尺度，从而避免某些特征因数值过大而对模型产生不成比例的影响。

![](img/0fe5a0328696cdaf528959290c374cb5_3.png)

## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体作用。标准化的核心目标是消除不同特征（或指标）在数值范围上的差异，使所有特征都能被“平等”地看待。

为了理解这一点，请看下图。假设我们有一个数据集，包含两个特征：X1和X2。

![](img/0fe5a0328696cdaf528959290c374cb5_1.png)

从图中可以看出，X1的取值范围较大（例如从-10到10），而X2的取值范围较小（例如从-2到2）。如果我们直接用这些原始数据训练模型，取值范围大的特征（如X1）可能会主导模型的学习过程，这并不是我们希望看到的。因此，我们需要进行标准化，让所有特征处于相似的数值区间。

## 标准化的公式与步骤

标准化的操作遵循一个明确的数学公式。其核心思想分为两步：去中心化（去均值）和按比例缩放（除标准差）。

标准化的公式如下：

**`z = (x - μ) / σ`**

其中：
*   **`x`** 是原始数据。
*   **`μ`** 是该特征所有数据的平均值（均值）。
*   **`σ`** 是该特征所有数据的标准差。

接下来，我们分解这个公式的两个步骤：

**第一步：去均值（`x - μ`）**
这一步的目的是将数据的中心移动到零点。如下图所示，原始数据可能并不以原点对称。减去均值后，数据在各个维度上都会以零为中心对称分布。

![](img/0fe5a0328696cdaf528959290c374cb5_3.png)

**第二步：除标准差（`/ σ`）**
去均值后，虽然数据中心对齐了，但不同特征的取值范围（离散程度）依然不同。这时就需要除以标准差。

标准差（`σ`）衡量了数据的离散程度。取值范围大、数据差异大的特征，其标准差也大；反之则小。

通过除以各自的标准差，我们实现了以下效果：
*   对于取值范围大的特征（标准差大），除以一个大数，使其数值范围缩小。
*   对于取值范围小的特征（标准差小），除以一个小数，其数值范围相对变化不大。

最终，所有特征的数据不仅以零为中心，其数值分布范围也变得近似相同。

![](img/0fe5a0328696cdaf528959290c374cb5_5.png)

## 标准化的代码实现

理解了原理后，我们来看看如何用代码实现标准化。以下是两种常见的方法。

![](img/0fe5a0328696cdaf528959290c374cb5_5.png)

### 方法一：手动实现

![](img/0fe5a0328696cdaf528959290c374cb5_7.png)

我们可以根据公式，手动计算均值和标准差，然后进行转换。以下是实现代码：

```python
def standardize(series):
    """
    对输入的序列进行标准化处理。
    参数:
        series: 一个pandas Series或类似数组的数据结构。
    返回:
        标准化后的序列。
    """
    # 计算均值和标准差
    mean = series.mean()
    std = series.std()
    # 应用标准化公式： (x - μ) / σ
    return (series - mean) / std
```

代码逻辑非常简单：
1.  计算输入数据的均值（`mean`）和标准差（`std`）。
2.  将数据减去均值，再除以标准差。
3.  返回处理后的结果。

![](img/0fe5a0328696cdaf528959290c374cb5_9.png)

### 方法二：使用Scikit-learn库

![](img/0fe5a0328696cdaf528959290c374cb5_11.png)

在实际项目中，我们通常使用成熟的机器学习库来完成此类操作，例如Scikit-learn。它提供了更高效、更稳定的实现。

![](img/0fe5a0328696cdaf528959290c374cb5_13.png)

以下是使用Scikit-learn中的`StandardScaler`进行标准化的方法：

```python
from sklearn.preprocessing import StandardScaler

![](img/0fe5a0328696cdaf528959290c374cb5_15.png)

# 假设 `data` 是一个二维数组或DataFrame
scaler = StandardScaler()
standardized_data = scaler.fit_transform(data)
```

![](img/0fe5a0328696cdaf528959290c374cb5_17.png)

使用`StandardScaler`通常只需一两行代码。你可以在Scikit-learn的官方文档中搜索“StandardScaler”来了解更多详细参数和用法。

## 总结

![](img/0fe5a0328696cdaf528959290c374cb5_37.png)

本节课中我们一起学习了数据标准化处理。
*   **目的**：消除不同特征因取值范围不同而带来的量纲影响，使模型能够平等地学习所有特征。
*   **原理**：通过 **`z = (x - μ) / σ`** 公式，先减去均值使数据以零为中心，再除以标准差使数据尺度一致。
*   **实现**：可以手动根据公式计算，也可以方便地使用Scikit-learn等库的`StandardScaler`工具。
*   **效果**：标准化后的数据各特征均值约为0，标准差约为1，处于相似的数值范围内。

![](img/0fe5a0328696cdaf528959290c374cb5_39.png)

标准化是数据预处理中极为基础和关键的一步，在大多数机器学习工作流中，它都是推荐甚至必需的步骤。
