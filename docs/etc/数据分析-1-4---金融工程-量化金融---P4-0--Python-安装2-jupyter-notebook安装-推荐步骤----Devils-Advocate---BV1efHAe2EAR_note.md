在本节课中，我们将学习如何安装Jupyter Notebook，并掌握其基础使用方法。Jupyter Notebook是一个交互式编程环境，非常适合数据分析、教学和探索性编程。

## Jupyter Notebook的安装

![](img/34929e37eafc348807c4bf466781a88e_4.png)

为了与课程内容保持同步，我们需要安装Jupyter Notebook。以下是安装步骤：

![](img/34929e37eafc348807c4bf466781a88e_5.png)

1.  打开命令提示符或终端。
2.  输入并执行以下命令来安装Jupyter Notebook：
    ```bash
    pip install jupyter
    ```
3.  等待安装过程完成。

安装完成后，我们可以启动Jupyter Notebook。

![](img/34929e37eafc348807c4bf466781a88e_7.png)

## 启动与创建Notebook

![](img/34929e37eafc348807c4bf466781a88e_9.png)

以下是启动Jupyter Notebook并创建新文件的方法：

![](img/34929e37eafc348807c4bf466781a88e_11.png)

1.  在命令提示符或终端中，输入以下命令启动Jupyter Notebook：
    ```bash
    jupyter notebook
    ```
2.  启动后，浏览器会自动打开Jupyter的主界面。
3.  在主界面中，点击右上角的“New”按钮。
4.  在下拉菜单中选择“Python 3”，即可创建一个新的Notebook文件。

## Notebook基础操作

![](img/34929e37eafc348807c4bf466781a88e_13.png)

现在，我们来看一下如何在Notebook中进行基础编程操作。

![](img/34929e37eafc348807c4bf466781a88e_15.png)

1.  在新建的Notebook单元格中，输入以下代码：
    ```python
    print("Hello World")
    ```
2.  按下 `Shift + Enter` 键运行该单元格。
3.  系统会执行代码，并在单元格下方输出结果“Hello World”。

运行成功后，界面会显示代码的执行结果。这表明Jupyter Notebook已经可以正常工作。

![](img/34929e37eafc348807c4bf466781a88e_17.png)
