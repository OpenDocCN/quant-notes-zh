## 数据录入与变量定义 📝

上一节我们介绍了课程概述，本节中我们来看看如何在SPSS中录入数据并定义变量属性。

打开SPSS数据视图，在变量视图中可以定义变量。例如，录入“性别”变量。

1.  在“名称”列输入“性别”。
2.  在“类型”列，点击单元格右侧的按钮，将类型从默认的“数值”改为“字符串”，宽度可设置为“2”。
3.  在“标签”列可以输入更详细的描述，如“受访者性别”。
4.  在“值”列，点击按钮进行值标签设置。例如，将“1”定义为“男”，“0”定义为“女”。这样在数据视图中，输入1或0就会显示对应的标签。
5.  在“测量”列，对于“性别”这类无序分类变量，应选择“名义”。

通过以上步骤，SPSS能基于统计学知识，智能地辅助我们完成数据录入和变量定义。

## 查找与替换 🔍

数据录入后，我们可能需要查找或修改特定内容。SPSS的查找与替换功能与Excel类似，快捷键也是 `Ctrl + F`。

调用查找与替换窗口后，需要注意一个关键点：**SPSS默认按列进行查找**。你需要在“查找”对话框中指定要搜索的变量（列），否则可能无法找到目标字符串。

例如，在“城市”列中查找“北京”，需要先在“查找”对话框的“查找”框中输入“北京”，然后在“变量”列表中选择对应的“城市”变量，再进行查找或替换操作。

**核心注意事项**：对于设置了值标签的变量（如性别，实际存储值为1/0，显示为男/女），直接查找显示值“男”可能失败。建议此类修改在变量视图的“值标签”设置中进行，或直接对实际存储值进行操作。

## 计算变量 🧮

在数据分析中，我们常需要基于现有变量生成新变量。这可以通过“计算变量”功能实现。

1.  点击菜单栏的 **转换 -> 计算变量**。
2.  在“目标变量”框中，输入新变量的名称，如 `New_Age`。
3.  在“数字表达式”框中，构建计算公式。例如，输入 `年龄 + 10`，表示新变量是原年龄加10的结果。
4.  点击“确定”，数据集中就会新增一列 `New_Age`。

![](img/d309103ebfea19ae7d16399573b5c798_1.png)

**代码示例（在“计算变量”对话框中实现）**：
```
目标变量： New_Age
数字表达式： 年龄 + 10
```
你也可以选择“如果...”按钮来设置条件，仅对满足条件的个案进行计算。

## 数据排序 🔼

对数据进行排序是常见的整理需求。SPSS支持单列和多列排序。

*   **单列排序**：在数据视图中，右键点击列标题，选择“升序排序”或“降序排序”。
*   **多列排序（个案排序）**：
    1.  点击菜单栏的 **数据 -> 个案排序**。
    2.  将需要排序的变量从左侧选入“排序依据”框。**框内变量的顺序决定了排序的优先级**，越靠上的变量优先级越高。
    3.  为每个变量选择“升序”或“降序”。
    4.  点击“确定”。系统会先按第一个变量排序，第一个变量值相同的个案，再按第二个变量排序，以此类推。

这与Excel中的“自定义排序”功能逻辑一致。

## 数据拆分与筛选 🪓

上一节我们介绍了如何排序，本节中我们来看看如何根据条件对数据进行分组或筛选。

### 拆分文件
“拆分文件”功能用于将数据按指定变量分组，后续的所有分析都会**分别在各组内独立进行**。

1.  点击 **数据 -> 拆分文件**。
2.  选择“比较组”或“按组组织输出”，并将分组变量（如“城市”）选入“分组依据”框。
3.  点击“确定”。此时进行任何分析（如描述统计），结果都会按“城市”分组显示。
4.  **重要**：使用完毕后，务必回到“拆分文件”设置，选择“分析所有个案，不创建组”，以免影响后续分析。

### 选择个案（筛选）
“选择个案”功能用于根据条件筛选出数据的子集。

1.  点击 **数据 -> 选择个案**。
2.  选择“如果条件满足”，然后点击“如果...”按钮设置条件（如 `id < 10`）。
3.  点击“继续”和“确定”。数据视图中，未被选中的个案前会标有斜杠。后续分析将只针对筛选出的个案进行。
4.  **重要**：使用完毕后，需回到“选择个案”，选择“所有个案”以取消筛选。

## 数据汇总与透视 📊

![](img/d309103ebfea19ae7d16399573b5c798_5.png)

在Excel中常用数据透视表进行分类汇总，SPSS中也有类似功能。

![](img/d309103ebfea19ae7d16399573b5c798_7.png)

1.  点击 **数据 -> 汇总**。
2.  在“分界变量”框中，放入分组变量（如“城市”）。
3.  在“变量摘要”框中，放入需要汇总的变量（如“年龄”）。
4.  点击“函数...”按钮，选择汇总函数，如“均值”、“个案数”等。
5.  点击“确定”。SPSS会生成一个新的数据集或变量，显示每个组的汇总结果。

![](img/d309103ebfea19ae7d16399573b5c798_9.png)

这个功能适用于快速计算各组的统计量。

## 数据重组（宽表转长表与长表转宽表）🔄

![](img/d309103ebfea19ae7d16399573b5c798_11.png)

有时数据格式不适合分析，需要进行重组。常见的是“宽表”转“长表”（一行变多行）及其逆操作。

![](img/d309103ebfea19ae7d16399573b5c798_13.png)

**场景**：原始数据中，每个人一行，记录了其在“焦虑”、“紧张”等4个症状上的评分。我们希望转换为每个人有4行数据，每行记录一个症状的评分。

### 宽表转长表（一行变多行）
1.  点击 **数据 -> 重构**。
2.  选择“将选定变量重构为个案”，点击“下一步”。
3.  选择“一个”索引变量，数量为1，点击“下一步”。
4.  在“个案组标识”中，选择标识每个个案的变量（如“个体ID”）。
5.  在“要转换的变量”中，将需要转换的多个变量（如4个症状评分）选入右侧框。
6.  后续步骤按默认设置，点击“完成”。数据即从一行一条记录，转换为每个症状一行记录。

### 长表转宽表（多行变一行）
这是上述过程的逆操作。
1.  点击 **数据 -> 重构**。
2.  选择“将选定个案重构为变量”，点击“下一步”。
3.  “标识变量”选择标识个案的变量（如“个体ID”）。“索引变量”选择标识转换后新变量名的变量（即“症状类型”）。
4.  按提示完成后续步骤，即可将数据恢复为一行一条记录的格式。

## 合并数据文件 🤝

![](img/d309103ebfea19ae7d16399573b5c798_19.png)

![](img/d309103ebfea19ae7d16399573b5c798_21.png)

分析时常需要合并多个数据文件。SPSS支持两种合并：纵向追加个案（添加行）和横向合并变量（添加列）。

### 纵向合并（添加个案）
用于合并结构相同或相似的数据集，将B文件的行追加到A文件尾部。
1.  在**活动数据集**中，点击 **数据 -> 合并文件 -> 添加个案**。
2.  选择要合并的另一个数据集文件。
3.  在对话框中，SPSS会自动匹配名称相同的变量。对于名称不同但含义相同的变量（如“身高”和“Hight”），可手动配对。仅存在于一个文件中的变量，可选择是否带入新数据集。
4.  点击“确定”完成合并。

### 横向合并（添加变量）
用于根据关键变量（如ID）将两个数据集中的变量合并到一起。
1.  在**活动数据集**中，点击 **数据 -> 合并文件 -> 添加变量**。
2.  选择要合并的另一个数据集文件。
3.  确保两个数据集都已按关键变量排序，并在对话框中选择“按照排序文件中的关键变量匹配个案”。通常选择“两个文件都提供个案”的选项。
4.  点击“确定”，第二个文件中的变量就会根据关键变量匹配并添加到第一个文件中。

## 标识重复个案 🏷️

在数据清洗时，需要检查重复记录。
1.  点击 **数据 -> 标识重复个案**。
2.  在“定义匹配个案的依据”框中，放入判断重复所依据的变量（如“姓名”和“身份证号”）。**只有所有这些变量值都相同的个案才会被判定为重复**。
3.  点击“确定”。SPSS会创建一个新变量指示个案是否重复，并在输出窗口中报告重复个案的数量和比例。

此功能有助于快速定位和审查可能的重复数据。
