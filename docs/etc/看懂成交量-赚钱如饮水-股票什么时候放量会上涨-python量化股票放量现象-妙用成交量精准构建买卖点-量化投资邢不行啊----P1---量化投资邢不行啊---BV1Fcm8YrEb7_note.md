## 概述

![](img/38ba960247f968d7c0761981136ee035_7.png)

成交量是股票市场中的重要指标，许多市场谚语都围绕它展开，例如“天量天价，地量地价”。这些经验之谈是量化策略开发的灵感来源，但其有效性需要通过历史数据来验证。本节课，我们将聚焦于“放量”现象，通过精确的量化定义和全面的数据回测，探究其与股价涨跌的关系。

## 什么是放量？

所谓放量，即成交量显著放大。当某只股票在某个交易日的成交量大幅高于往常时，即可称为放量。这通常意味着有大量资金进出，市场交易活跃。

从K线图上看，放量通常表现为成交量柱状图出现明显的“一柱擎天”形态。

## 放量后的股价走势

![](img/38ba960247f968d7c0761981136ee035_19.png)

放量后的股价走势并不确定，可能持续上涨，也可能转为下跌或横盘震荡。有一种主流观点认为，股价的相对位置（高位或低位）决定了放量后的表现。

![](img/38ba960247f968d7c0761981136ee035_21.png)

![](img/38ba960247f968d7c0761981136ee035_23.png)

相关的市场口诀是：“低位放量跟，跟错也要跟；高位放量跑，跑错也要跑。”意思是，在股价相对低位出现放量时应买入，在相对高位出现放量时应卖出。

![](img/38ba960247f968d7c0761981136ee035_25.png)

然而，仅凭几张图表举例不足以得出结论。我们需要借助全部A股历史数据和Python代码，对“高位放量”和“低位放量”进行量化定义和全面统计。

## 如何量化定义“高位”、“低位”与“放量”？

![](img/38ba960247f968d7c0761981136ee035_31.png)

将模糊的市场经验转化为计算机可执行的精确规则，是量化交易的关键一步。

![](img/38ba960247f968d7c0761981136ee035_33.png)

### 定义高位与低位

![](img/38ba960247f968d7c0761981136ee035_35.png)

![](img/38ba960247f968d7c0761981136ee035_37.png)

我们采用以下方法定义股价的相对位置：
*   计算股票过去120个交易日的收盘价。
*   将这些收盘价从大到小排序。
*   如果当日收盘价排名在前10%，则定义该日为 **“高位”**。
*   如果当日收盘价排名在后10%，则定义该日为 **“低位”**。

![](img/38ba960247f968d7c0761981136ee035_39.png)

**公式表示：**
`当日收盘价百分位 = (当日收盘价在过去120日中的排名) / 120`
若 `百分位 >= 0.9`，则为高位；若 `百分位 <= 0.1`，则为低位。

![](img/38ba960247f968d7c0761981136ee035_41.png)

当然，这只是其中一种定义方式，你可以根据自己的理解进行调整。

![](img/38ba960247f968d7c0761981136ee035_42.png)

### 定义放量

我们介绍一种基于换手率和布林带思想的定义方法：
1.  **计算换手率**：`换手率 = 当日成交量 / 流通股数`。换手率能更好地反映交易的活跃程度。
2.  **借鉴布林带公式**：布林带的中轨是N日收盘价的均值，上下轨是中轨加减M倍标准差。
3.  **构建换手率布林上轨**：计算过去100日换手率的均值（`mean_turnover`）和标准差（`std_turnover`）。
    *   **上轨** = `mean_turnover + 2 * std_turnover`
4.  **定义放量**：如果当日换手率 **大于** 上轨，则认为该日 **“放量”**。

![](img/38ba960247f968d7c0761981136ee035_44.png)

**代码描述：**
```python
# 假设 df[‘turnover’] 为换手率序列
mean_turnover = df[‘turnover’].rolling(100).mean()
std_turnover = df[‘turnover’].rolling(100).std()
upper_band = mean_turnover + 2 * std_turnover
is_volume_spike = df[‘turnover’] > upper_band # 结果为True代表放量
```

![](img/38ba960247f968d7c0761981136ee035_46.png)

## 数据验证与统计分析

![](img/38ba960247f968d7c0761981136ee035_54.png)

上一节我们完成了核心概念的量化和定义，本节我们将使用Python和全市场历史数据来进行验证。

![](img/38ba960247f968d7c0761981136ee035_56.png)

![](img/38ba960247f968d7c0761981136ee035_58.png)

我们运行编写好的程序，在A股所有股票的历史数据中，自动标记出每一个“高位放量”和“低位放量”的时点，并统计这些时点之后不同周期（1天、3天、5天、10天）的股价表现。

![](img/38ba960247f968d7c0761981136ee035_60.png)

以下是程序运行后的统计结果：

![](img/38ba960247f968d7c0761981136ee035_62.png)

![](img/38ba960247f968d7c0761981136ee035_64.png)

*   **高位放量**：自2007年以来，共出现约37万次。
    *   放量后1、3、5、10天，股价下跌的概率均显著大于50%（最低约53%）。
    *   但平均跌幅很小，接近零。这表明高位放量后，股价更可能进入横盘震荡，而非立即下跌。
*   **低位放量**：自2007年以来，共出现约4万次。
    *   放量后1、3、5、10天，股价上涨的概率稳步上升。
    *   平均收益率也同步上升，**放量后10天的平均收益率高达2.8%**，这是一个非常可观的数字。

数据验证表明，“低位放量看涨”的谚语在统计上是有效的，而“高位放量看跌”则更可能表现为横盘。

## 从结论到策略

![](img/38ba960247f968d7c0761981136ee035_68.png)

虽然我们得出了“低位放量后股价上涨概率和平均收益较高”的结论，但这本身还不是一个可交易的量化策略。

![](img/38ba960247f968d7c0761981136ee035_70.png)

一个完整的量化策略需要精确规定：
*   **买卖信号**：具体何时买入、何时卖出。
*   **仓位管理**：每次买卖多少资金。
*   **绩效评估**：根据策略交易能形成一条清晰的资金曲线，显示盈亏。

![](img/38ba960247f968d7c0761981136ee035_72.png)

如何将“低位放量”这个有效的市场现象，构建成一个具备以上要素的、可执行的量化策略，我们将在后续课程中详细讲解。
