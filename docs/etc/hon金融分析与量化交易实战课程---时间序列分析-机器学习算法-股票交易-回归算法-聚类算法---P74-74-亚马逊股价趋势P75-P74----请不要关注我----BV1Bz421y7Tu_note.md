## 模型评估方法
现在我们已经得到了预测结果，接下来需要评估模型的性能。以下是评估模型时需要考虑的两个核心方面：

### 1. 预测误差
预测误差衡量的是预测值与真实值之间的差异。我们可以将历史数据划分为训练集和测试集进行计算。
*   **训练集误差**：模型在训练数据上的预测误差。
*   **测试集误差**：模型在未见过数据（测试集）上的预测误差。

误差可以用绝对误差表示，公式为：
**误差 = | 真实值 - 预测值 |**

### 2. 趋势预测准确率
除了具体数值，股价的涨跌趋势同样重要。我们需要评估模型是否能正确预测价格的上升或下降趋势。
*   如果真实趋势是上升，模型预测是否为上升？
*   如果真实趋势是下降，模型预测是否为下降？

评估趋势准确率可以帮助我们理解模型对市场方向的把握能力。

## 实施模型评估
我们将通过代码演示具体的评估过程。首先，划分数据：使用2014年至2017年共3年的数据作为训练集，使用2017年至2018年共1年的数据作为测试集。

![](img/61cd51b832abecf95d90d12c08c3df1b_43.png)

![](img/61cd51b832abecf95d90d12c08c3df1b_45.png)

以下是评估的核心步骤代码片段：

```python
# 计算预测值与真实值的差异序列
diff = forecast[‘yhat’].diff()
# 使用sign函数将差异转换为趋势标签：上升为1，下降为0
trend_predicted = np.sign(diff).apply(lambda x: 1 if x > 0 else 0)
# 同样计算真实数据的趋势标签
trend_actual = np.sign(data[‘y’].diff()).apply(lambda x: 1 if x > 0 else 0)
# 计算趋势预测的正确率
trend_accuracy = (trend_predicted == trend_actual).mean()
```

![](img/61cd51b832abecf95d90d12c08c3df1b_47.png)

在评估过程中，如果遇到不熟悉的函数（如`np.sign`），可以查阅官方文档。`np.sign`函数的作用是：输入大于0返回1，小于0返回-1，等于0返回0。

评估完成后，我们得到以下关键指标：
*   **上升趋势预测正确率**：约56%
*   **下降趋势预测正确率**：约43%
*   **训练集平均绝对误差**：约18美元
*   **测试集平均绝对误差**：约160美元

从结果看，当前模型的趋势预测准确率接近随机猜测（50%），且测试集误差远大于训练集误差，说明模型存在改进空间。
