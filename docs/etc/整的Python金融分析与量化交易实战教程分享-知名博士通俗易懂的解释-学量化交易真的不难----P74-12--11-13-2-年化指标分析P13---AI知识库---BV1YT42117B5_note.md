## 回测收益率回顾

![](img/4b9455e32c339e4cbe704b87cebf0ba0_4.png)

上一节我们介绍了回测收益率，它衡量了策略在整个回测周期内的总盈亏情况。

![](img/4b9455e32c339e4cbe704b87cebf0ba0_6.png)

回测收益率计算的是策略最终收益与初始投入的比值。这个值可能是正的，也可能是负的，直观地反映了“投资有风险，收益不保证”这一事实。通过回测，我们可以看到有些策略可能获得翻倍收益，而有些则收益较少甚至亏损。

![](img/4b9455e32c339e4cbe704b87cebf0ba0_8.png)

这个指标是策略分析中最重要的一环，设计一个策略后，首先需要观察的就是它能带来多少回报。

![](img/4b9455e32c339e4cbe704b87cebf0ba0_10.png)

## 引入年化收益率

![](img/4b9455e32c339e4cbe704b87cebf0ba0_12.png)

理解了回测收益率后，本节中我们来看看另一个关键指标：年化收益率。

在实际交流中，如果回测周期长达数年，直接比较总收益率并不公平。年化收益率将不同时间长度的收益标准化，转化为假设投资期为一年所能获得的收益率，便于横向比较。

以下是年化收益率的计算公式：

![](img/4b9455e32c339e4cbe704b87cebf0ba0_14.png)

![](img/4b9455e32c339e4cbe704b87cebf0ba0_16.png)

**公式：**
`年化收益率 = (1 + 总收益率) ^ (250 / 总交易日数) - 1`

![](img/4b9455e32c339e4cbe704b87cebf0ba0_18.png)

其中：
*   **总收益率**：即我们之前计算的回测收益率。
*   **250**：代表一年中大约的平均交易日数量（也可使用252）。
*   **总交易日数**：策略回测所覆盖的总交易天数。

![](img/4b9455e32c339e4cbe704b87cebf0ba0_20.png)

## 代码实现计算

![](img/4b9455e32c339e4cbe704b87cebf0ba0_22.png)

现在，我们通过Python代码来实际计算年化收益率。假设我们已经有了一个包含回测收益率的数据。

![](img/4b9455e32c339e4cbe704b87cebf0ba0_24.png)

以下是计算年化收益率的代码步骤：

![](img/4b9455e32c339e4cbe704b87cebf0ba0_26.png)

```python
# 假设 df 是一个DataFrame，其中‘回测收益率’列已计算好
# 总交易日数
total_trading_days = len(df)  # 例如 730天
# 一年平均交易日数
trading_days_per_year = 250

![](img/4b9455e32c339e4cbe704b87cebf0ba0_28.png)

# 计算年化收益率
df['年化收益率'] = (1 + df['回测收益率']) ** (trading_days_per_year / total_trading_days) - 1

![](img/4b9455e32c339e4cbe704b87cebf0ba0_30.png)

# 展示结果
print(df[['回测收益率', '年化收益率']])
```

![](img/4b9455e32c339e4cbe704b87cebf0ba0_32.png)

执行上述代码后，我们会得到一个新的“年化收益率”列。可以观察到，年化收益率与回测收益率数值不同但关联紧密。例如，一个在700多天（约3年）内获得113%总收益的策略，其年化收益率可能约为20%左右。原本亏损的策略，其年化亏损率也会相应调整。

![](img/4b9455e32c339e4cbe704b87cebf0ba0_34.png)

## 核心指标总结

![](img/4b9455e32c339e4cbe704b87cebf0ba0_38.png)

本节课中我们一起学习了量化策略收益评估的两个核心指标。

![](img/4b9455e32c339e4cbe704b87cebf0ba0_40.png)

*   **回测收益率**：反映策略在整个测试周期内的**总盈亏**结果，是评估策略有效性的首要指标。
*   **年化收益率**：将总收益标准化为**年度比率**，消除了时间长度的影响，便于在不同策略间进行公平比较。

![](img/4b9455e32c339e4cbe704b87cebf0ba0_42.png)

在策略分析中，我们不仅要关注最终获得了多少收益，更要关注其按年折算后的盈利能力，两者结合才能对策略性能做出更全面的判断。
