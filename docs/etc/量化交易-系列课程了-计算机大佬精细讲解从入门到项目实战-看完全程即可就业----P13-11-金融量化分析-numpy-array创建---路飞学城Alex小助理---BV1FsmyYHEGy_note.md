*   **创建全零数组**：使用`np.zeros`函数。默认创建的是浮点型数组。
    ```python
    import numpy as np
    a = np.zeros(10) # 创建长度为10的全零数组，数据类型为float64
    a_int = np.zeros(10, dtype=int) # 创建长度为10的全零数组，并指定数据类型为整数
    ```

![](img/12522924cb07267c77c424570a9e301b_19.png)

![](img/12522924cb07267c77c424570a9e301b_21.png)

![](img/12522924cb07267c77c424570a9e301b_23.png)

![](img/12522924cb07267c77c424570a9e301b_25.png)

*   **创建全一数组**：使用`np.ones`函数。同样，默认创建浮点型数组。
    ```python
    b = np.ones(5) # 创建长度为5的全一数组，数据类型为float64
    b_int = np.ones(5, dtype=int) # 创建长度为5的全一数组，并指定数据类型为整数
    ```

![](img/12522924cb07267c77c424570a9e301b_27.png)

![](img/12522924cb07267c77c424570a9e301b_29.png)

*   **创建空数组**：使用`np.empty`函数。此函数只分配内存空间，但不初始化内存中的值，因此数组内容为内存中的随机残留值。
    ```python
    c = np.empty(100) # 创建长度为100的“空”数组，内容未初始化
    ```
    使用`empty`比`zeros`少了一步初始化操作，速度更快。但前提是你后续会覆盖数组中的所有值。如果不确定，建议使用`zeros`或`ones`。

*   **使用`arange`函数**：类似于Python内置的`range`函数，但功能更强大，可以生成数组。
    ```python
    # 生成0到99的整数序列
    seq1 = np.arange(100)
    # 生成10到20（不包含20）的整数序列
    seq2 = np.arange(10, 20)
    # 生成0到10，步长为0.5的序列
    seq3 = np.arange(0, 10, 0.5)
    ```
    `arange`是前闭后开区间，且步长可以是小数。

![](img/12522924cb07267c77c424570a9e301b_49.png)

*   **使用`linspace`函数**：在指定的起始值和结束值之间，生成等间隔的数值序列。第三个参数指定的是要生成的元素**数量**，而不是步长。
    ```python
    # 在0到100之间生成101个等间隔的数（包含0和100）
    lin1 = np.linspace(0, 100, 101)
    # 在-10到10之间生成10000个等间隔的数，用于绘制平滑的函数图像
    x = np.linspace(-10, 10, 10000)
    y = x ** 2 # 计算y = x^2
    ```
    `linspace`是前闭后闭区间。它在需要均匀采样或绘制函数图像时非常有用。

*   **创建单位矩阵**：使用`np.eye`函数。单位矩阵是一个方阵，主对角线上的元素为1，其余元素为0。
    ```python

    identity_matrix = np.eye(3)
    ```
    如果不涉及线性代数运算，这个函数使用频率较低。
