在本节课中，我们将学习量化策略编写中几个核心的设置函数。这些函数用于配置策略的运行环境，包括业绩基准、交易成本、滑点等，是构建真实、可执行策略的基础。

上一节我们介绍了量化策略的基本框架，本节中我们来看看如何通过具体的设置函数来配置和优化策略。

## 常用策略设置函数概述

策略设置函数通常在策略初始化阶段调用，用于定义策略的基准、成本、交易规则等。以下是五个常用的设置函数及其作用。

### 1. 设置业绩比较基准 📊

基准是衡量策略表现的参照物。例如，可以将策略的收益率与沪深300指数进行比较。在API中，使用 `set_benchmark` 函数来设置。

**公式/代码示例：**
```python
set_benchmark('000300.XSHG')  # 设置基准为沪深300指数
```

### 2. 设置佣金与印花税 💰

股票交易会产生手续费。买入时通常有佣金，卖出时则有佣金加印花税。使用 `set_order_cost` 函数来模拟这些成本。

**公式/代码示例：**
```python
# 设置佣金为万分之三，印花税为千分之一
set_order_cost(cost=0.0003, type='stock')
# 注：实际API参数可能更复杂，需参考具体平台文档
```

### 3. 设置滑点 ⚙️

滑点是指实际成交价格与预期下单价格之间的偏差。设置滑点可以模拟市场流动性不足或价格快速变动的情况。API中通常使用 `set_slippage` 函数。

![](img/8183b23adba746c22cdb5690783b8b1c_1.png)

**公式/代码示例：**
```python
set_slippage(price=0.002)  # 设置滑点为千分之二
```

### 4. 设置成交量比例 📈

此设置模拟大额订单无法一次性全部成交的场景。例如，你想买入100手，但市场只能提供50手。这通过调整成交量比例来模拟。

**公式/代码示例：**
```python
set_option('volume_ratio', 0.5)  # 设置成交量比例为50%
```

![](img/8183b23adba746c22cdb5690783b8b1c_3.png)

### 5. 设置动态复权模式 🔄

![](img/8183b23adba746c22cdb5690783b8b1c_5.png)

股票或基金在分红、拆股后，价格会发生变化。动态复权功能会调整历史价格，以反映真实收益。建议在回测中开启此选项以获得更准确的结果。

**公式/代码示例：**
```python
set_option('use_real_price', True)  # 开启真实价格（动态复权）
```

## 代码实战演示

接下来，我们进入实战环节，通过修改代码来观察每个设置函数对策略回测结果的具体影响。

### 设置业绩基准

以下是设置业绩基准的代码示例。我们将基准从默认值改为浦发银行，观察策略收益对比的变化。

```python
# 在initialize函数中设置
def initialize(context):
    set_benchmark('600000.XSHG')  # 浦发银行
    # ... 其他初始化代码
```
运行回测后，策略收益曲线会与浦发银行的股价走势进行比较，并计算阿尔法、贝塔等指标。

### 设置佣金与印花税

现在，我们为策略添加交易成本。

```python
def initialize(context):
    # 设置佣金和印花税（示例参数）
    set_order_cost(cost=0.0003, type='stock')  # 佣金万分之三
    # 注：印花税设置方式可能不同，需查阅具体API
    # ... 其他代码
```
添加成本后，频繁交易的策略收益会显著下降，因为每笔交易都会扣除费用。

### 设置滑点

添加滑点设置，模拟价格波动。

```python
def initialize(context):
    set_slippage(price=0.002)  # 千分之二的滑点
    # ... 其他代码
```
在某些情况下，设置滑点可能意外地提高策略收益，因为它模拟了订单未能立即以最优价成交的情况。

### 设置成交量比例

模拟订单无法全部成交的场景。

```python
def initialize(context):
    set_option('volume_ratio', 0.5)  # 仅50%的订单量能成交
    # ... 其他代码
```
这会导致策略的实际成交股数少于计划，影响最终收益。

![](img/8183b23adba746c22cdb5690783b8b1c_7.png)

### 设置动态复权

最后，我们开启动态复权。

```python
def initialize(context):
    set_option('use_real_price', True)  # 开启动态复权
    # ... 其他代码
```
开启后，回测使用的价格数据是经过分红、拆股调整的，更能反映持有资产的真实价值变化。关闭此功能，回测结果可能会失真。

## 本节总结

本节课中我们一起学习了量化策略中五个关键的设置函数：
1.  **`set_benchmark`**：用于设置业绩比较基准。
2.  **`set_order_cost`**：用于设置交易佣金和税费。
3.  **`set_slippage`**：用于设置交易滑点。
4.  **`set_option`（成交量比例）**：用于模拟订单部分成交的情况。
5.  **`set_option`（动态复权）**：用于开启或关闭价格复权调整。

![](img/8183b23adba746c22cdb5690783b8b1c_9.png)

这些函数都在策略的 `initialize` 初始化函数中调用，它们共同决定了回测环境的真实性和策略最终表现的评估标准。通过调整这些参数，我们可以让策略回测更贴近实际交易，从而做出更可靠的决策。在实际应用中，需要根据策略特点和市场环境合理配置这些参数。
