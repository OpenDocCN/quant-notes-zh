## 概述
在本节课中，我们将学习量化交易策略中的一个核心环节——**因子打分法**。我们将了解如何根据多个财务或市场指标（因子）对股票进行综合评分，并基于评分结果进行选股和调仓。整个过程将从数据准备开始，逐步讲解打分的具体方法，并最终梳理出一个完整的策略执行流程。

## 因子打分法详解

上一节我们介绍了如何获取和预处理股票因子数据。本节中我们来看看如何对这些因子进行综合评估，即“打分”。

打分法的核心思想是：为每个影响股票表现的因子（如市盈率、负债率等）设定评分规则，然后将每只股票在所有因子上的得分汇总，得到一个总分。最后，根据总分排名来选择表现最佳的股票。

### 打分规则设计

以下是设计打分规则时需要考虑的两个关键方向：

1.  **因子方向性**：每个因子对股票预期表现的影响方向不同。有些因子是“越大越好”（例如净资产收益率ROE），有些则是“越小越好”（例如资产负债率）。
2.  **分值映射**：需要将因子的实际数值（或百分位）映射到一个具体的分数上。通常，表现更好的区间应获得更高的分数。

### 打分计算示例

假设我们有4个因子（A, B, C, D）和2只股票（id1, id2）。已知：
*   A因子和B因子是**越大越好**。
*   C因子和D因子是**越小越好**。
*   所有因子值都已归一化到0-1之间。

首先，我们为每个因子划分数值区间并分配分数。对于“越大越好”的因子，高分对应高数值区间；对于“越小越好”的因子，高分对应低数值区间。

**区间与分值映射表：**

| 数值区间 | “越大越好”因子得分 | “越小越好”因子得分 |
| :--- | :--- | :--- |
| [0.8, 1.0] | 5 | 1 |
| [0.6, 0.8) | 4 | 2 |
| [0.4, 0.6) | 3 | 3 |
| [0.2, 0.4) | 2 | 4 |
| [0.0, 0.2) | 1 | 5 |

现在，假设我们查询到以下因子数据：

| 股票ID | 因子A | 因子B | 因子C | 因子D |
| :--- | :--- | :--- | :--- | :--- |
| id1 | 0.71 | 0.28 | 0.39 | 0.11 |
| id2 | 0.45 | 0.17 | 0.81 | 0.02 |

根据上表的规则，我们可以为每只股票的每个因子打分：

以下是id1股票的打分过程：
*   **因子A (0.71)**：属于“越大越好”类型。0.71落在[0.6, 0.8)区间，得 **4分**。
*   **因子B (0.28)**：属于“越大越好”类型。0.28落在[0.2, 0.4)区间，得 **2分**。
*   **因子C (0.39)**：属于“越小越好”类型。0.39落在[0.2, 0.4)区间，得 **4分**。
*   **因子D (0.11)**：属于“越小越好”类型。0.11落在[0.0, 0.2)区间，得 **5分**。
*   **id1总分**：4 + 2 + 4 + 5 = **15分**。

以下是id2股票的打分过程：
*   **因子A (0.45)**：得 **3分**。
*   **因子B (0.17)**：得 **1分**。
*   **因子C (0.81)**：属于“越小越好”类型。0.81落在[0.8, 1.0]区间，得 **1分**。
*   **因子D (0.02)**：得 **5分**。
*   **id2总分**：3 + 1 + 1 + 5 = **10分**。

计算完成后，我们对所有股票（例如沪深300成分股）的总分进行排序。在本例中，id1（15分）排名高于id2（10分）。在调仓时，我们便可以选取总分排名最高的前N只股票（例如前10名）作为投资组合。

> **其他打分方法**：除了划分固定区间，也可以采用百分位排序法。例如，将300只股票按某个因子从大到小排序，排名第1的给300分，排名第300的给1分。这种方法同样有效，核心目标是得到一个可比较的综合排名。

以下是策略执行的主要步骤：

1.  **确定股票池**：首先，定义策略操作的股票范围。本例中，我们将在沪深300指数的成分股中进行选择。
    ```python
    # 示例：设置初始股票池为沪深300成分股
    set_universe('沪深300')
    ```

2.  **设置调仓周期**：策略通常按固定频率（如每月或每季度）运行。我们需要设置一个定时器，在每月初触发调仓函数。
    ```python
    # 示例：每月第一个交易日执行rebalance函数
    run_monthly(rebalance, date_rule=1)
    ```

3.  **实现调仓函数**：这是策略的核心函数`rebalance()`。每次调仓时，它都会执行以下子步骤：
    *   **数据获取**：获取当前股票池中所有股票的指定因子数据（如A, B, C, D）。
    *   **因子打分**：根据预设规则（因子方向、区间划分），为每只股票的每个因子计算得分。
    *   **汇总排序**：将每只股票在所有因子上的得分相加，得到总分，并按总分从高到低排序。
    *   **生成交易清单**：选取总分排名前10的股票，作为本次调仓后希望持有的股票。
    *   **执行调仓**：卖出当前持仓中不在新清单里的股票，买入新清单中尚未持有的股票，使投资组合与清单一致。
