## 核心概念与策略概述

价格突破策略的核心是识别价格在特定通道内的运行，并在价格突破通道边界时进行交易。策略主要分为三个步骤：
1.  **检测枢轴点**：识别图表上的局部高点和低点。
2.  **拟合价格通道**：将连续的枢轴点拟合成上下两条趋势线，形成一个通道。
3.  **识别突破信号**：当价格K线完全突破通道边界时，生成交易信号。

上一节我们介绍了策略的整体框架，本节中我们来看看如何具体实现这些步骤。

## 第一步：检测枢轴点

![](img/528a847beb0b4d6616fd0150a2f1d313_5.png)

枢轴点是指价格高于或低于其两侧相邻K线的极值点。我们的算法需要自动识别这些点。

以下是检测枢轴点的核心代码逻辑：
```python
def detect_pivot(df, window):
    """
    检测枢轴高点和低点。
    :param df: 包含`high`, `low`列的DataFrame
    :param window: 在两侧检查的K线数量
    :return: 包含枢轴点标记的Series (1=高点， -1=低点)
    """
    highs = df['high']
    lows = df['low']
    pivot_highs = (highs == highs.rolling(window*2+1, center=True).max())
    pivot_lows = (lows == lows.rolling(window*2+1, center=True).min())
    # 标记：1为枢轴高点， -1为枢轴低点
    pivots = pd.Series(0, index=df.index)
    pivots[pivot_highs] = 1
    pivots[pivot_lows] = -1
    return pivots
```
该函数通过在一个滑动窗口内寻找最高价和最低价来定位枢轴点。

## 第二步：拟合有效价格通道

找到枢轴点后，我们需要将它们组合起来，判断是否能形成一个有效的价格通道。一个有效的通道需要满足几个条件。

以下是判断有效通道的条件列表：
*   **最少点数**：至少需要5个枢轴点（例如，2个高点+3个低点，或反之）。
*   **拟合优度**：将高点和低点分别进行线性拟合后，其**R²**（决定系数）应大于一个阈值（例如0.9）。**R²** 衡量了数据点与拟合直线的接近程度，公式为 **R² = 1 - (SS_res / SS_tot)**，其中 `SS_res` 是残差平方和，`SS_tot` 是总平方和。
*   **斜率平行**：高点和低点拟合出的两条趋势线的斜率差异应小于一个阈值（例如10%），以确保通道近似平行。

以下是生成突破信号的逻辑：
*   **看跌突破信号**：当一根K线的**开盘价**和**收盘价**都低于通道的下轨趋势线时，生成做空信号。
*   **看涨突破信号**：当一根K线的**开盘价**和**收盘价**都高于通道的上轨趋势线时，生成做多信号。

## 交易管理系统优化

![](img/528a847beb0b4d6616fd0150a2f1d313_16.png)

![](img/528a847beb0b4d6616fd0150a2f1d313_17.png)

仅仅有信号并不够，如何执行交易至关重要。我们将引入智能交易管理来提升回报。

### 订单挂单与回调入场

我们不直接在信号出现时以市价入场，而是采用挂单策略。例如，在看跌突破信号出现后，我们在当前收盘价上方一定百分比（如1%）的位置下一个**卖出限价单**。这样，如果价格在突破后出现预期中的回调，我们的订单将被触发，从而获得一个更有利的入场价。

### 止损与止盈设置

![](img/528a847beb0b4d6616fd0150a2f1d313_19.png)

![](img/528a847beb0b4d6616fd0150a2f1d313_20.png)

入场后，我们需要管理风险。
*   **止损**：设置在入场价下方一定百分比（如3%）的位置。
*   **止盈**：通过 **止盈距离 = 止损距离 × 止盈止损比率** 计算得出。例如，若止损距离为3%，比率为1.5，则止盈距离为4.5%。

![](img/528a847beb0b4d6616fd0150a2f1d313_22.png)

![](img/528a847beb0b4d6616fd0150a2f1d313_23.png)

### 引入RSI作为额外退出条件

![](img/528a847beb0b4d6616fd0150a2f1d313_25.png)

![](img/528a847beb0b4d6616fd0150a2f1d313_27.png)

除了固定的止损止盈，我们还可以使用技术指标如RSI来提前退出交易。
*   如果持有多头仓位且**RSI > 95**，则平仓。
*   如果持有空头仓位且**RSI < 5**，则平仓。
这可以帮助我们锁定利润或避免在趋势极端时回吐过多盈利。
