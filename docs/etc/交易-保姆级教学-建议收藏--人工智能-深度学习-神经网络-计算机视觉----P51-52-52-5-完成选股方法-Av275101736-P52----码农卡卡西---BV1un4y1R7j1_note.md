在本节课中，我们将学习如何将多个股票因子评分表进行合并，并计算综合得分以完成最终的选股排序。我们将通过拼接DataFrame、计算总分和排序等步骤，实现一个能够返回排名前十股票列表的函数。

## 拼接数据表

![](img/6ea203a72b406a37a4933900866b3ccb_4.png)

上一节我们介绍了如何为单个因子生成评分表。本节中，我们来看看如何将多个因子的评分表合并成一张大表。

![](img/6ea203a72b406a37a4933900866b3ccb_6.png)

拼接操作是指将一个DataFrame与另一个DataFrame进行连接。我们使用`concat`函数来完成这个操作。

![](img/6ea203a72b406a37a4933900866b3ccb_8.png)

![](img/6ea203a72b406a37a4933900866b3ccb_10.png)

以下是拼接操作的代码：
```python
df_merged = pd.concat([df1, df2], axis=1)
```
在这段代码中，`df1`和`df2`是两个需要合并的DataFrame，`axis=1`参数表示按列进行拼接。拼接完成后，我们得到了一个包含所有因子评分的新DataFrame，可以将其命名为`df_merged`。

![](img/6ea203a72b406a37a4933900866b3ccb_12.png)

## 计算综合得分并排序

![](img/6ea203a72b406a37a4933900866b3ccb_14.png)

![](img/6ea203a72b406a37a4933900866b3ccb_16.png)

现在我们已经有了包含所有因子评分的大表，但表中还没有综合排名信息。接下来，我们需要计算每只股票的综合得分并进行排序。

![](img/6ea203a72b406a37a4933900866b3ccb_18.png)

首先，我们在DataFrame中新建一个名为“得分值”的列，并暂时用零填充。
```python
df_merged[‘得分值’] = np.zeros((df_merged.shape[0], 1))
```
然后，我们需要计算总分。总分是各个因子评分的加和。计算完成后，我们将根据总分进行降序排序，以找出得分最高的股票。

![](img/6ea203a72b406a37a4933900866b3ccb_20.png)

以下是计算总分并排序的步骤：
1.  对每一行的所有因子评分进行求和，得到综合得分。
2.  根据综合得分列，对DataFrame进行降序排序。

相关代码如下：
```python
# 计算每一行的总分（假设因子评分列名为‘factor1’， ‘factor2’...）
df_merged[‘总分’] = df_merged[[‘factor1’， ‘factor2’， ...]].sum(axis=1)

![](img/6ea203a72b406a37a4933900866b3ccb_22.png)

# 根据‘总分’列进行降序排序
df_sorted = df_merged.sort_values(by=‘总分’， ascending=False)
```

![](img/6ea203a72b406a37a4933900866b3ccb_24.png)

![](img/6ea203a72b406a37a4933900866b3ccb_26.png)

## 获取前十名股票

![](img/6ea203a72b406a37a4933900866b3ccb_28.png)

排序完成后，我们的目标就变成了从排序后的结果中提取排名前十的股票代码。

![](img/6ea203a72b406a37a4933900866b3ccb_30.png)

我们不需要所有列的数据，只需要股票代码和对应的总分。因此，我们可以从排序后的DataFrame中提取股票代码索引，并将其转换为列表，然后取前十个元素。

![](img/6ea203a72b406a37a4933900866b3ccb_32.png)

以下是获取前十名股票列表的代码：
```python
# 获取排序后DataFrame的索引（即股票代码），并转换为列表
stock_list = df_sorted.index.tolist()

![](img/6ea203a72b406a37a4933900866b3ccb_34.png)

![](img/6ea203a72b406a37a4933900866b3ccb_36.png)

# 取前十个股票代码
top_10_stocks = stock_list[:10]
```
这样，`top_10_stocks`列表就包含了我们根据综合评分选出的前十只股票。

![](img/6ea203a72b406a37a4933900866b3ccb_38.png)

## 函数封装与流程总结

![](img/6ea203a72b406a37a4933900866b3ccb_40.png)

![](img/6ea203a72b406a37a4933900866b3ccb_42.png)

最后，我们将上述所有步骤封装到一个名为`get_top_stocks`的函数中。这个函数完成了整个选股流程。

![](img/6ea203a72b406a37a4933900866b3ccb_44.png)

函数的主要流程如下：
1.  **读取因子数据**：获取各个因子的评分DataFrame。
2.  **数据拼接**：使用`pd.concat`将所有因子评分表合并。
3.  **计算总分**：对每个股票的所有因子评分进行求和。
4.  **排序**：根据总分进行降序排序。
5.  **返回结果**：提取排名前十的股票代码列表并返回。

![](img/6ea203a72b406a37a4933900866b3ccb_46.png)

调用一次`get_top_stocks`函数，即可得到基于当前所有因子数据计算出的、综合排名前十的股票列表。

![](img/6ea203a72b406a37a4933900866b3ccb_48.png)

![](img/6ea203a72b406a37a4933900866b3ccb_50.png)

本节课中我们一起学习了如何整合多个股票因子评分，通过计算综合得分和排序，最终实现一个完整的选股方法。我们掌握了使用`pd.concat`拼接数据、使用`.sum()`和`.sort_values()`进行计算与排序，以及提取目标股票列表的关键步骤。
