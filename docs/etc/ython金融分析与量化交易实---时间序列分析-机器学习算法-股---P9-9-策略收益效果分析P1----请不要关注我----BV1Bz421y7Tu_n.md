在本节课中，我们将学习如何计算和比较量化策略的最终收益。我们将从计算股票的单日收益率开始，然后结合之前构建的“双均线策略”信号，计算策略的累计收益，并与“买入并持有”的原始收益进行对比。

## 计算单日收益率

![](img/54f280952336cba6ab37268b8d2bf03a_2.png)

上一节我们介绍了如何生成交易信号，本节中我们来看看如何计算策略的收益。首先，我们需要计算股票价格每日的涨跌幅度，即单日收益率。

![](img/54f280952336cba6ab37268b8d2bf03a_4.png)

在量化分析中，为了便于后续的累加计算，我们通常使用对数收益率。其计算公式为：

`对数收益率 = ln(当日收盘价 / 前一日收盘价)`

在Python的pandas库中，我们可以通过以下步骤实现：

1.  获取股价数据（例如收盘价 `close`）。
2.  计算 `close / close.shift(1)`，即当日价格除以前一日价格。
3.  对上述结果取自然对数 `np.log()`。

以下是计算对数收益率的代码：

![](img/54f280952336cba6ab37268b8d2bf03a_6.png)

```python
# 假设 df 是包含股价的DataFrame，'close' 是收盘价列
returns = np.log(df['close'] / df['close'].shift(1))
```

![](img/54f280952336cba6ab37268b8d2bf03a_8.png)

执行这段代码后，`returns` 列就存储了每日的对数收益率。

![](img/54f280952336cba6ab37268b8d2bf03a_10.png)

## 结合策略信号计算策略收益率

我们已经有了市场的原始收益率，现在需要将其与我们之前构建的策略信号结合起来。回顾一下，我们的“双均线策略”生成了一个名为 `position` 的信号列，其取值为 1 或 -1。

![](img/54f280952336cba6ab37268b8d2bf03a_12.png)

*   **`position = 1`**：表示策略判断市场处于上升趋势，我们应跟随市场，正常持有。
*   **`position = -1`**：表示策略判断市场处于下降趋势，我们应进行反向操作（例如做空），其收益将与市场原始走势相反。

![](img/54f280952336cba6ab37268b8d2bf03a_14.png)

因此，策略的每日收益率应为市场原始收益率与策略信号的乘积。计算公式为：

![](img/54f280952336cba6ab37268b8d2bf03a_16.png)

`策略日收益率 = position * 原始日收益率`

同样需要注意，为了在当天开盘时就执行信号，我们需要对 `position` 列也进行 `shift(1)` 操作，使用前一交易日收盘时生成的信号。代码如下：

![](img/54f280952336cba6ab37268b8d2bf03a_18.png)

```python
# 计算策略的日收益率
strategy_returns = df['position'].shift(1) * returns
df['strategy_returns'] = strategy_returns
```

## 计算并比较累计收益

![](img/54f280952336cba6ab37268b8d2bf03a_20.png)

为了评估策略的长期效果，我们需要计算累计收益，即观察初始的1元钱最终变成了多少钱。

![](img/54f280952336cba6ab37268b8d2bf03a_22.png)

由于我们使用的是对数收益率，累计收益的计算变得非常简单，只需将所有日期的收益率相加，然后通过指数函数 `np.exp()` 将其还原为实际的资金倍数。这个过程可以表示为：

![](img/54f280952336cba6ab37268b8d2bf03a_24.png)

`最终资金倍数 = exp( 收益率1 + 收益率2 + ... + 收益率N )`

![](img/54f280952336cba6ab37268b8d2bf03a_26.png)

以下是计算并比较“原始收益”与“策略收益”的步骤：

![](img/54f280952336cba6ab37268b8d2bf03a_28.png)

1.  分别对 `returns`（原始收益）和 `strategy_returns`（策略收益）列进行求和（`.sum()`），得到总的对数收益率。
2.  对求和结果使用 `np.exp()` 函数，得到最终的资产倍数。

![](img/54f280952336cba6ab37268b8d2bf03a_30.png)

在计算前，需要确保数据中没有缺失值（NaN）。我们可以使用 `.dropna()` 方法进行处理。

```python
# 删除含有缺失值的行，确保计算准确
df_clean = df[['returns', 'strategy_returns']].dropna()

# 计算累计对数收益并还原为资金倍数
cumulative_return = np.exp(df_clean['returns'].sum())
cumulative_strategy_return = np.exp(df_clean['strategy_returns'].sum())

![](img/54f280952336cba6ab37268b8d2bf03a_32.png)

print(f"买入并持有最终资金倍数: {cumulative_return:.2f}")
print(f"双均线策略最终资金倍数: {cumulative_strategy_return:.2f}")
```

假设输出结果为：
*   买入并持有最终资金倍数: 4.00
*   双均线策略最终资金倍数: 5.80

![](img/54f280952336cba6ab37268b8d2bf03a_34.png)

![](img/54f280952336cba6ab37268b8d2bf03a_36.png)

这个结果表明，如果初始投入1元钱：
*   单纯买入股票并一直持有，最终会变成 **4元**。
*   使用我们的双均线策略进行交易，最终会变成 **5.8元**。

![](img/54f280952336cba6ab37268b8d2bf03a_38.png)

显然，在该段历史数据上，我们的策略获得了超越市场基准的超额收益。

## 总结

本节课中我们一起学习了量化策略收益分析的核心流程。

![](img/54f280952336cba6ab37268b8d2bf03a_40.png)

1.  **计算基础收益**：我们首先学会了如何计算股票的对数日收益率，这是收益分析的基础。
2.  **融合策略信号**：接着，我们将策略生成的交易信号（`position`）与原始收益率相乘，得到了策略的日收益率序列。这体现了策略“在上涨时持有，在下跌时反向操作”的核心逻辑。
3.  **评估策略效果**：最后，我们通过求和与指数还原，计算了策略的累计收益，并与“买入并持有”的原始收益进行对比。结果显示，双均线策略在本案例中创造了更高的最终回报。

![](img/54f280952336cba6ab37268b8d2bf03a_42.png)

通过这一整套分析，我们能够定量地评估一个交易策略的历史表现，这是量化交易中至关重要的一步。
