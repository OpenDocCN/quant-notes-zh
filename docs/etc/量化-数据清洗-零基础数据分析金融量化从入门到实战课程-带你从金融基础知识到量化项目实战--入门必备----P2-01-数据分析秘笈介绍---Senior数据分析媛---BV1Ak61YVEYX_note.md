## 什么是数据分析？

![](img/2a240f854541fa183a8ac974f52c87c6_7.png)

上一节我们介绍了课程的整体安排，本节中我们来看看数据分析的核心定义。

数据分析是指从看似杂乱无章且量级庞大的数据中，提炼出有价值的信息，并总结出这些数据背后存在的内部规律。

![](img/2a240f854541fa183a8ac974f52c87c6_9.png)

用公式可以简单表示为：
**数据分析 = 从原始数据中提取有价值信息 + 总结内部规律**

## 数据分析的核心作用

理解了数据分析的定义后，我们来看看它最重要的价值体现在哪里。

![](img/2a240f854541fa183a8ac974f52c87c6_17.png)

![](img/2a240f854541fa183a8ac974f52c87c6_19.png)

数据分析最主要的作用是**实现数据价值的最大化**，这通常体现在提升综合收入或利润上。各行各业都需要有价值的产出，数据分析技能可以帮助我们基于原始数据，制定策略以实现利润最大化。

![](img/2a240f854541fa183a8ac974f52c87c6_21.png)

![](img/2a240f854541fa183a8ac974f52c87c6_23.png)

以下是数据分析价值最大化的几个具体体现：

![](img/2a240f854541fa183a8ac974f52c87c6_25.png)

![](img/2a240f854541fa183a8ac974f52c87c6_27.png)

1.  **分析用户消费行为**：商家可以基于历史消费记录，分析用户的消费习惯、回购能力等，从而制定更有效的促销活动方案、确定促销时间与力度。
2.  **优化广告投放策略**：通过分析用户观看、点击广告的历史行为数据，可以决定广告的投放时间、内容设计以及投放平台（如爱奇艺、腾讯视频），使广告效果和价值最大化。
3.  **辅助精准决策**：数据分析是用适当的方法对收集来的大量数据进行分析，最终得出的结论可以帮助人们做出精准判断，并采取合适行动。

![](img/2a240f854541fa183a8ac974f52c87c6_29.png)

## 为什么要学习数据分析？

了解了数据分析的概念和作用后，你可能会问，我们为什么要专门学习它呢？主要有以下三个原因。

以下是学习数据分析的三个核心原因：

![](img/2a240f854541fa183a8ac974f52c87c6_47.png)

![](img/2a240f854541fa183a8ac974f52c87c6_49.png)

1.  **岗位需求广泛**：在招聘平台（如Boss直聘）上，数据分析岗位的需求量很大。不同于仅限于互联网行业的开发岗，数据分析适用于所有产生数据的行业，包括传统制造业和新兴互联网行业，帮助企业通过数据做出正确决策、实现价值最大化。
2.  **数据竞赛平台活跃**：国内外存在许多数据竞赛平台（如阿里的天池、Kaggle）。企业和个人会在这些平台上发布带有赏金的数据分析任务，这为数据分析技能提供了实践和变现的渠道。
3.  **Python数据科学的基础**：Python在数据分析、机器学习领域有强大优势。其数据分析框架集成了大量数学和科学运算模块，是进行复杂数据分析和科学计算的基础。
4.  **机器学习的前置课程**：数据分析是转向机器学习领域的必备基础。扎实的数据分析能力能为后续学习机器学习提供良好的铺垫。

## 数据分析的基本流程

![](img/2a240f854541fa183a8ac974f52c87c6_51.png)

那么，进行一次完整的数据分析通常需要遵循怎样的步骤呢？

![](img/2a240f854541fa183a8ac974f52c87c6_53.png)

数据分析的实现通常遵循以下五步基本流程：

![](img/2a240f854541fa183a8ac974f52c87c6_55.png)

1.  **提出问题**：明确需要分析解决的具体问题。
2.  **准备数据**：通过内部获取、购买或网络爬虫等方式，收集与问题相关的数据。
3.  **分析数据**：运用数据分析相关的模块和技能，对准备好的数据进行处理和分析。
4.  **获得结论**：从分析结果中提炼出有价值的结论和信息。
5.  **结果可视化**：使用图表（如散点图、直方图、折线图）将分析结论直观地展示出来。
