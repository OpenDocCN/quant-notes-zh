在本节课中，我们将要学习Pandas中DataFrame对象的索引和切片操作。DataFrame是一个二维表格型数据结构，理解如何高效、准确地从中获取数据是进行数据分析的基础。

上一节我们介绍了DataFrame的一些常用属性，本节中我们来看看如何获取DataFrame中的具体数值。

## 索引的基本方法

与Series类似，DataFrame也可以通过中括号 `[]` 的方式获取值。但需要注意的是，DataFrame的索引顺序与二维数组不同。对于DataFrame，通常**先指定列索引，再指定行索引**。这类似于数据库查询中先选择列（SELECT column），再筛选行的逻辑。

例如，假设有一个DataFrame `df`，要获取列名为`‘one’`、行标签为`‘A’`的值，直观的写法是 `df[‘one’][‘A’]`。然而，这种连续使用两个中括号的方式并不推荐，尤其是在行索引为整数时容易产生混淆或错误。

## 推荐的索引方法：`.loc` 与 `.iloc`

为了避免混淆，Pandas提供了 `.loc` 和 `.iloc` 这两个属性进行索引，它们清晰地指明了索引方式。

*   **`.loc`**：**基于标签**进行索引。
*   **`.iloc`**：**基于整数位置**进行索引。

它们的通用语法是：
```python
df.loc[行索引, 列索引]
df.iloc[行位置, 列位置]
```
索引部分用逗号隔开，**前面是行，后面是列**。这与NumPy的索引顺序一致。

以下是具体的使用示例：

![](img/2c4b9f89f130d643ef5566f827788c89_3.png)

### 使用 `.loc` 获取单个值
要获取行标签为`‘A’`、列标签为`‘one’`的值，更安全的方法是：
```python
df.loc[‘A’, ‘one’]
```

![](img/2c4b9f89f130d643ef5566f827788c89_5.png)

### 使用 `.loc` 获取整行数据
如果想获取`‘A’`这一行的所有数据，列索引部分可以留空或使用切片 `:` 表示选择所有列。
```python
df.loc[‘A’, :]  # 获取A行，所有列
# 或简写为
df.loc[‘A’]
```

### 使用 `.iloc` 基于位置索引
`.iloc` 的用法与 `.loc` 类似，但传入的是整数位置。例如，获取第一行、第二列的数据：
```python
df.iloc[0, 1]
```

## 灵活的切片与索引组合

![](img/2c4b9f89f130d643ef5566f827788c89_8.png)

`.loc`和`.iloc`的强大之处在于，行和列的索引部分可以灵活地使用**切片、布尔索引或花式索引（传递一个列表）**，并且可以任意搭配。

以下是一些组合示例：

![](img/2c4b9f89f130d643ef5566f827788c89_10.png)

*   **选择特定的多行和多列**：
    ```python
    # 选择行标签为‘A’和‘C’的行，以及列标签为‘one’和‘two’的列
    df.loc[[‘A‘, ‘C‘], [‘one‘, ‘two‘]]
    ```

![](img/2c4b9f89f130d643ef5566f827788c89_12.png)

*   **使用切片选择连续的行和列**：
    ```python
    # 选择从‘A‘行到‘C‘行，以及从‘one‘列到‘two‘列的所有数据
    df.loc[‘A‘:‘C‘, ‘one‘:‘two‘]
    ```

*   **混合使用索引类型**：
    ```python
    # 行使用花式索引选择‘A‘和‘C‘，列使用切片选择所有列
    df.loc[[‘A‘, ‘C‘], :]
    ```

![](img/2c4b9f89f130d643ef5566f827788c89_14.png)

这种设计使得数据选取非常灵活和强大。

![](img/2c4b9f89f130d643ef5566f827788c89_16.png)

本节课中我们一起学习了DataFrame的索引与切片操作。核心在于掌握基于标签的 `.loc` 和基于位置的 `.iloc` 这两种推荐方法，并理解其 `[行, 列]` 的索引顺序。通过组合切片、列表和布尔值，可以高效地从DataFrame中提取出所需的任何数据子集，这是后续进行数据筛选、清洗和分析的关键步骤。
