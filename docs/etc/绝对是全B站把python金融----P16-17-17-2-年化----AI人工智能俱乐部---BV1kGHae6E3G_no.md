## 回测收益率：策略的最终成绩单

回测收益率是衡量一个交易策略在特定历史时间段内表现的最直接指标。它计算的是策略最终的总收益与初始本金的比值。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_4.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_5.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_6.png)

以下是回测收益率的含义：
*   它直接反映了策略在测试期间是盈利还是亏损。
*   数值为正表示盈利，为负则表示亏损。
*   数值大小直观地展示了盈利或亏损的程度，例如“翻倍”意味着收益率接近100%。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_7.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_8.png)

这个指标至关重要，因为设计好一个策略后，首先需要确认的就是这个策略能带来多少回报。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_10.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_11.png)

## 年化收益率：标准化的绩效标尺

理解了总收益后，我们面临一个新的问题：如果一个策略运行了10年，总收益是150%，另一个策略运行了2年，总收益是80%，哪个策略更好？直接比较总收益是不公平的，因为时间跨度不同。这时就需要引入**年化收益率**。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_13.png)

年化收益率是将不同时间周期的总收益，折算成相当于每年平均获得的收益率。这使得不同时间长度的策略之间可以进行公平比较。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_15.png)

在量化分析中，年化收益率通常是比总收益率更受关注的核心评价指标。

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_17.png)

### 年化收益率的计算公式

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_18.png)

年化收益率的计算基于总收益率，并考虑交易天数。其公式如下：

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_20.png)

**年化收益率 = (1 + 总收益率)^(M / N) - 1**

其中：
*   **R** 代表总收益率（即回测收益率）。
*   **M** 代表一年的平均交易日数量，通常取 **250** 或 **252** 天。
*   **N** 代表策略回测所用的总交易日数量。

### 代码实现示例

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_22.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_23.png)

假设我们已经计算出了总收益率 `total_return`，并且知道回测数据的总交易日数 `total_days`，我们可以用Python计算年化收益率：

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_25.png)

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_26.png)

```python
# 定义年交易日数
trading_days_per_year = 250

![](img/9c000ebf92a8a4bc8a5b383b196ce8f5_28.png)

# 计算年化收益率
annualized_return = (1 + total_return) ** (trading_days_per_year / total_days) - 1
```

例如，一个策略在730个交易日（约3年）内获得了1.13（即113%）的总收益率，其年化收益率计算如下：
`(1 + 1.13) ^ (250 / 730) - 1 ≈ 0.27`，即年化收益率约为27%。

通过对比可以发现，年化收益率数值通常比总收益率小，但它提供了标准化的年度绩效视角，是评估策略质量的关键。
