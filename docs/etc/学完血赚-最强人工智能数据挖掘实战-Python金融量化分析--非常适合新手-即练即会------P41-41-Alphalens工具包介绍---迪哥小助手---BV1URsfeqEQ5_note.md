## 工具包简介与安装

![](img/11711b7598838f9fa7a8a8999bc3bf3d_5.png)

![](img/11711b7598838f9fa7a8a8999bc3bf3d_6.png)

![](img/11711b7598838f9fa7a8a8999bc3bf3d_7.png)

**Alphalens** 是一个专门用于金融因子分析的Python工具包。它封装了计算IC值、分组收益、绘制分析图表等核心功能，使我们无需从零开始编写复杂的代码。

以下是获取与安装Alphalens的步骤：
*   **官方资源**：你可以通过其GitHub仓库和官方文档进行深入了解。
*   **安装方法**：安装过程非常简单，只需在命令行中执行 `pip install alphalens` 即可。
*   **平台便利性**：在本课程提供的量化平台中，Alphalens已经预先安装好，我们可以直接使用，无需手动安装。

## 量化分析平台操作

接下来，我们将在量化平台中编写代码进行实战。因为在该平台中可以便捷地获取金融数据，避免了本地配置数据源的麻烦。

![](img/11711b7598838f9fa7a8a8999bc3bf3d_16.png)

![](img/11711b7598838f9fa7a8a8999bc3bf3d_17.png)

以下是进入平台编写环境的步骤：
1.  登录量化平台，进入策略管理界面。
2.  在左侧菜单中找到并点击 **“投资研究”** 功能。
3.  系统会打开一个类似于Jupyter Notebook的在线编程环境。
4.  点击新建一个 **Python 3** 笔记文件，并将其重命名为“因子分析”。我们后续的所有代码都将在这个环境中编写和运行。
