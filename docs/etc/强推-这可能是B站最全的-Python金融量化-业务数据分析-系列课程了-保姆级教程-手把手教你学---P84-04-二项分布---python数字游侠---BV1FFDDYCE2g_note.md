## 概率分布概述

上一节我们介绍了数据分析的基础，本节中我们来看看概率分布。概率分布描述了随机变量取值的规律。随机变量发生的可能性大小称为概率，而概率存在的规律就称为概率分布。概率分布主要分为两类：离散型随机变量的概率分布和连续型随机变量的概率分布。

对于离散型随机变量，我们主要讲解二项分布和泊松分布。对于连续型随机变量，我们主要讲解正态分布。本节我们先聚焦于离散型随机变量。

![](img/8e9939d96b5c22cd47f3b805a9041182_5.png)

## 离散型随机变量

![](img/8e9939d96b5c22cd47f3b805a9041182_7.png)

我们以抛硬币作为例子来理解离散型随机变量。抛硬币的结果只有两种：正面或反面。我们可以将出现正面记为 **1**，出现反面记为 **0**。这里的 **1** 和 **0** 是离散的、不连续的数值，因此属于离散型随机变量。每次抛掷，出现正面或反面的概率都是 **1/2**。

## 伯努利分布

在讲解二项分布之前，我们需要先了解伯努利分布。伯努利分布是理解二项分布的基础。

![](img/8e9939d96b5c22cd47f3b805a9041182_9.png)

![](img/8e9939d96b5c22cd47f3b805a9041182_11.png)

在现实生活中，许多事件的结果只有两个。以下是伯努利试验的例子：
*   抛一次硬币：结果只有正面或反面。
*   检查一件产品质量：结果只有合格或不合格。
*   购买一张彩票：结果只有中奖或未中奖。
*   进行一次表白：结果只有成功或不成功。

这些只进行一次、且结果只有两个（成功或失败）的随机试验，称为伯努利试验。其结果的概率分布称为伯努利分布，也叫两点分布。

伯努利分布的概率非常简单。假设事件“成功”的概率为 **P**，事件“失败”的概率为 **Q**，那么两者关系为：
**P + Q = 1**

## 二项分布

![](img/8e9939d96b5c22cd47f3b805a9041182_13.png)

上一节我们介绍了单次的伯努利试验，本节中我们来看看多次试验的情况。二项分布描述的是在 **N** 次独立重复的伯努利试验中，我们所期望的结果（例如“成功”）恰好发生 **X** 次的概率分布规律。

我们通过一个具体问题来理解：连续抛掷一枚均匀硬币5次，恰好出现2次正面的概率是多少？

首先，我们手动推导所有可能的情况。在5次抛掷中，恰好2次正面的组合方式有10种（例如：正正反反反、正反正反反等）。每次抛掷出现正面的概率是 **1/2**，出现反面的概率也是 **1/2**。因此，任何一种特定的2正3反序列出现的概率是：
**(1/2)^2 * (1/2)^3 = 1/32**

因为有10种这样的序列，所以总概率为：
**10 * (1/32) = 10/32 = 5/16 ≈ 0.3125**

显然，我们不可能每次都手动列举。二项分布提供了通用公式来计算这个概率。

二项分布的概率质量函数公式为：
**P(X=k) = C(n, k) * p^k * (1-p)^(n-k)**
其中：
*   **n** 代表试验总次数。
*   **k** 代表期望的成功次数。
*   **p** 代表单次试验中成功的概率。
*   **C(n, k)** 是组合数，计算公式为 **n! / (k! * (n-k)!)**，代表从n次试验中选取k次成功的所有可能组合数。

对于抛硬币5次得2次正面的例子，代入公式：
**P(X=2) = C(5, 2) * (0.5)^2 * (0.5)^3 = 10 * 0.25 * 0.125 = 0.3125**

## 在Excel中计算二项分布

在实际应用中，我们可以使用Excel的`BINOM.DIST`函数来计算二项分布概率。

函数语法为：`BINOM.DIST(number_s, trials, probability_s, cumulative)`
*   `number_s`: 期望的成功次数（例如2）。
*   `trials`: 总试验次数（例如5）。
*   `probability_s`: 单次成功概率（例如0.5）。
*   `cumulative`: 逻辑值。`FALSE`计算恰好发生k次的概率；`TRUE`计算累积概率（即发生次数≤k的概率）。

![](img/8e9939d96b5c22cd47f3b805a9041182_15.png)

![](img/8e9939d96b5c22cd47f3b805a9041182_17.png)

计算抛5次硬币得2次正面的概率：
`=BINOM.DIST(2, 5, 0.5, FALSE)`，结果约为0.3125。

## 二项分布的特征与总结

符合以下三个条件的随机实验，其成功次数的分布就是二项分布：
1.  实验是在相同条件下独立重复进行的n次试验。
2.  每次试验只有两种可能的结果：“成功”或“失败”。
3.  每次试验中“成功”的概率p是相同的，“失败”的概率为1-p。

二项分布是n次独立伯努利试验的推广。其均值和方差分别为：
*   均值（期望值）: **E(X) = n * p**
*   方差: **Var(X) = n * p * (1-p)**

当试验次数n足够大时，二项分布的形态会趋近于正态分布。在统计学中，通常认为当 **n ≥ 30** 时，这种近似是合理的。在实际的机器学习或大数据分析中，由于数据量巨大，许多统计前提条件的影响会变得很小。
