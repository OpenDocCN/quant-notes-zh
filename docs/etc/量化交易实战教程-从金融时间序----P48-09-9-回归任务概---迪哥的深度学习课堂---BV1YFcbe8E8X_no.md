## 数据准备与预处理

上一节我们介绍了回归策略的基本概念，本节中我们来看看如何为回归任务准备数据。首先，我们需要导入必要的工具包并加载数据。

![](img/a46d93c16d5a43c05130d4aa4c55b600_3.png)

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
```

数据文件是 `data.csv`。本次我们使用欧元对美元的汇率数据作为分析对象。

![](img/a46d93c16d5a43c05130d4aa4c55b600_5.png)

```python
data = pd.read_csv('data.csv')
```

![](img/a46d93c16d5a43c05130d4aa4c55b600_7.png)

![](img/a46d93c16d5a43c05130d4aa4c55b600_9.png)

接下来，我们从数据中提取所需的列，并进行基本的预处理操作。为了使结果具有累加性，我们通常对价格数据取对数来计算回报率。

```python
# 计算对数回报率
data['returns'] = np.log(data['Close'] / data['Close'].shift(1))
# 处理缺失值
data.dropna(inplace=True)
```

![](img/a46d93c16d5a43c05130d4aa4c55b600_11.png)

为了简化问题，我们不仅关心回报率的具体数值，有时更关心其方向（涨或跌）。因此，我们创建一个新的列 `direction` 来表示走势。

```python
# 创建方向列，1代表上涨，-1代表下跌
data['direction'] = np.sign(data['returns']).astype(int)
```

以下是数据预处理后的一个可视化结果，它展示了回报率分布的基本情况。

```python
data['returns'].hist(bins=50)
plt.show()
```

## 构建回归任务

![](img/a46d93c16d5a43c05130d4aa4c55b600_21.png)

在准备好数据后，我们进入核心环节：构建回归任务。回归是机器学习中的一种有监督学习任务，意味着我们需要特征（X）和标签（Y）。

![](img/a46d93c16d5a43c05130d4aa4c55b600_23.png)

对于我们的金融预测任务，**Y（标签）** 是我们想要预测的未来值。这里我们可以尝试两种不同的预测目标进行对比实验：
*   **Y1**：预测明天的具体回报率数值（`returns`）。
*   **Y2**：预测明天是上涨还是下跌（`direction`）。

而 **X（特征）** 则是用于预测的历史信息。我们可以利用过去多天的数据（如前1天、前2天、前5天等指标）来构建特征。使用的历史特征越多，模型可能捕捉到的模式就越丰富。

![](img/a46d93c16d5a43c05130d4aa4c55b600_25.png)
