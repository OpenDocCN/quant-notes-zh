## 什么是最大回撤？📊

![](img/426c0a1024b6b593483549862e3a6ab6_2.png)

上一节我们介绍了策略的收益表现，本节中我们来看看如何评估策略的风险。最大回撤描述的是投资过程中“最惨”的阶段。

它衡量的是从任意一个历史高点，到之后某个低点之间，资产价值的最大跌幅。这个指标不能仅从字面理解，它核心描述的是投资过程中可能面临的最大亏损压力。

## 最大回撤的数学定义 📐

最大回撤的计算公式如下：

**公式：**
`最大回撤 = max_{i, j} ( (P_i - P_j) / P_i )`，其中 `i < j`。

在这个公式中：
*   `P_i` 代表第 `i` 天的资产价格（或净值）。
*   `P_j` 代表第 `i` 天之后的某一天 `j` 的资产价格。
*   我们需要遍历所有可能的 `i` 和 `j`（`i < j`），找出 `(P_i - P_j) / P_i` 这个比值最大的情况。

这个比值 `(P_i - P_j) / P_i` 表示从高点 `P_i` 到低点 `P_j` 的**缩水比例**。最大回撤就是所有可能时间段中，这个缩水比例的最大值。

## 为什么最大回撤很重要？⚠️

![](img/426c0a1024b6b593483549862e3a6ab6_4.png)

最大回撤是一个风险指标，而非收益指标。它告诉你，即使一个策略长期来看是赚钱的，但在过程中你可能会经历一段非常难熬的时期。

![](img/426c0a1024b6b593483549862e3a6ab6_6.png)

例如，在2008年金融危机期间，股市从高点大幅下跌，许多投资者承受了巨大的账面亏损。最大回撤区间就量化了这种“惨烈”的程度。了解策略的最大回撤，有助于你评估自己能否承受这样的波动，从而做好资金管理和风险控制。

![](img/426c0a1024b6b593483549862e3a6ab6_8.png)

## 如何用Python计算最大回撤？💻

![](img/426c0a1024b6b593483549862e3a6ab6_10.png)

![](img/426c0a1024b6b593483549862e3a6ab6_12.png)

理解了概念后，我们来看看如何用代码实现最大回撤的计算。核心思路是：先计算每个时间点之前的历史最高值，然后计算当前值与历史最高值的差距，最后找出最大的相对跌幅。

以下是计算最大回撤的关键步骤：

![](img/426c0a1024b6b593483549862e3a6ab6_14.png)

首先，我们需要计算到每一天为止的历史最高价（或净值）。这可以通过 `pandas` 的 `cummax()` 函数实现。

![](img/426c0a1024b6b593483549862e3a6ab6_16.png)

![](img/426c0a1024b6b593483549862e3a6ab6_17.png)

**代码示例：计算历史最高序列**
```python
import pandas as pd

![](img/426c0a1024b6b593483549862e3a6ab6_19.png)

![](img/426c0a1024b6b593483549862e3a6ab6_20.png)

# 假设 df[‘close’] 是每日收盘价序列
historical_max = df[‘close’].cummax()
```
`cummax()` 函数会返回一个序列，其中每个值都是到当前日期为止所有价格的最大值。

接下来，我们计算每日的“回撤”，即当前价格与历史最高价的差值。

![](img/426c0a1024b6b593483549862e3a6ab6_22.png)

![](img/426c0a1024b6b593483549862e3a6ab6_23.png)

**代码示例：计算每日回撤**
```python
daily_drawdown = (historical_max - df[‘close’]) / historical_max
```
这行代码计算了每日价格相对于历史最高点的回撤比例。

![](img/426c0a1024b6b593483549862e3a6ab6_25.png)

![](img/426c0a1024b6b593483549862e3a6ab6_26.png)

最后，最大回撤就是这个每日回撤序列中的最大值。

**代码示例：计算最大回撤**
```python
max_drawdown = daily_drawdown.max()
```
`max_drawdown` 的值就是我们要找的最大回撤率。

![](img/426c0a1024b6b593483549862e3a6ab6_28.png)

![](img/426c0a1024b6b593483549862e3a6ab6_29.png)

## 计算过程总结 📝

![](img/426c0a1024b6b593483549862e3a6ab6_31.png)

让我们将上述步骤整合，清晰地展示计算流程：

以下是计算最大回撤的完整逻辑：
1.  **计算历史最高序列**：对价格序列应用 `cummax()`，得到每个时间点之前的历史最高值。
2.  **计算每日回撤率**：用 `(历史最高值 - 当前值) / 历史最高值` 公式，计算每一天的回撤比例。
3.  **找出最大值**：在所有每日回撤率中，找出最大值，即为最大回撤。

![](img/426c0a1024b6b593483549862e3a6ab6_33.png)

![](img/426c0a1024b6b593483549862e3a6ab6_34.png)
