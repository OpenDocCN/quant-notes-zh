## 回测收益率回顾

![](img/664ddf11ef96da3ed572bf6e2529ef94_14.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_16.png)

回测收益率是评估策略有效性的首要指标。它计算了策略在整个回测周期结束后，初始资金的总变化率。其核心含义是：与初始投入相比，最终赚取或亏损的百分比。

![](img/664ddf11ef96da3ed572bf6e2529ef94_18.png)

以下是计算回测收益率的基本逻辑：
```python
# 假设 final_value 是最终资产总值，initial_value 是初始本金
backtest_return = (final_value - initial_value) / initial_value
```
这个值可能是正数（盈利），也可能是负数（亏损），直观地体现了“投资有风险”，策略并非稳赚不赔。有些策略可能实现翻倍收益，而有些则收益微薄甚至亏损。

![](img/664ddf11ef96da3ed572bf6e2529ef94_20.png)

## 年化收益率详解

![](img/664ddf11ef96da3ed572bf6e2529ef94_22.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_24.png)

理解了策略的总收益后，我们还需要一个标准化的指标来比较不同时间跨度策略的盈利能力，这就是年化收益率。

![](img/664ddf11ef96da3ed572bf6e2529ef94_26.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_28.png)

人们经常听到“年化收益”这个词。如果你的回测周期长达8年或10年，在向他人展示时，对方可能更关心一个标准化的年度收益表现。年化收益率正是将总收益率转化为假设每年复利一次所能得到的等效收益率。

![](img/664ddf11ef96da3ed572bf6e2529ef94_30.png)

以下是年化收益率的计算公式：
**年化收益率 = (1 + 总收益率) ^ (250 / N) - 1**

![](img/664ddf11ef96da3ed572bf6e2529ef94_32.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_34.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_36.png)

公式解析：
*   **总收益率 (R)**：即我们之前计算出的回测收益率。
*   **250 (M)**：代表一年中大约的交易日数量（通常取250或252天）。
*   **N**：代表策略回测所使用的总交易日数量。

![](img/664ddf11ef96da3ed572bf6e2529ef94_38.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_40.png)

例如，我们的示例数据共有730个交易日样本，大约相当于3年的数据。根据公式，我们可以进行如下计算：

![](img/664ddf11ef96da3ed572bf6e2529ef94_42.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_44.png)

```python
# 假设 total_return 是计算好的总收益率，N 是总交易日数
annualized_return = (1 + total_return) ** (250 / N) - 1
```

![](img/664ddf11ef96da3ed572bf6e2529ef94_46.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_48.png)

计算完成后，我们可以将年化收益率结果与回测收益率并列展示。对比两者可以发现，数值存在差异。年化收益率提供了一个更统一的标准，便于比较不同时间长度策略的盈利能力，因此通常是量化分析中重点观察的核心指标。

![](img/664ddf11ef96da3ed572bf6e2529ef94_50.png)

![](img/664ddf11ef96da3ed572bf6e2529ef94_52.png)

本节课中我们一起学习了量化策略分析的两个核心收益指标。我们回顾了**回测收益率**，它展示了策略的最终总盈亏。接着，我们深入探讨了**年化收益率**，它通过一个标准公式将总收益标准化，便于跨周期比较，是评估策略绩效更常用的关键指标。理解并计算这两个指标，是量化交易策略分析与评价的基础。
