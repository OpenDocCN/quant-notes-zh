## 概述

量化CTA策略是一种系统化的交易方法，它使用数学模型和计算机程序来分析市场数据并执行交易。本节课程将引导你理解其基本流程。

## 策略开发流程

一个完整的量化CTA策略开发包含多个关键环节。以下是构建策略所需遵循的主要步骤：

1.  **数据获取与处理**：这是策略的基础。我们需要获取高质量的历史价格、成交量等市场数据，并进行清洗和格式化，以备后续分析使用。
2.  **策略思想与信号生成**：基于对市场的理解，设计交易逻辑。例如，当短期移动平均线上穿长期移动平均线时，生成买入信号。其公式可表示为：`买入信号 = SMA(close, 10) > SMA(close, 30)`。
3.  **回测与验证**：将策略逻辑应用于历史数据，模拟交易过程，以评估策略在过去的盈利能力、风险等指标。
4.  **风险管理**：设定规则来控制每笔交易的潜在损失，例如使用固定百分比止损：`止损价 = 入场价 * (1 - 止损比例)`。
5.  **实盘部署**：将通过回测验证的策略连接到实盘交易接口，进行自动化交易。

上一节我们介绍了策略开发的整体流程，本节中我们来看看如何用Python代码实现一个简单的双均线策略信号生成。

## 核心代码示例

以下是一个使用Python和pandas库计算双均线交易信号的简化示例：

```python
import pandas as pd

# 假设df是一个包含‘close’价格的DataFrame
def dual_moving_average_strategy(df, short_window=10, long_window=30):
    # 计算短期和长期简单移动平均线
    df[‘SMA_short’] = df[‘close’].rolling(window=short_window).mean()
    df[‘SMA_long’] = df[‘close’].rolling(window=long_window).mean()
    
    # 生成交易信号：短期均线上穿长期均线为1（买入），下穿为-1（卖出），其余为0（持仓）
    df[‘signal’] = 0
    df.loc[df[‘SMA_short’] > df[‘SMA_long’], ‘signal’] = 1
    df.loc[df[‘SMA_short’] < df[‘SMA_long’], ‘signal’] = -1
    
    # 实际的交易信号是信号的变化点（金叉或死叉）
    df[‘position’] = df[‘signal’].diff()
    return df
```

理解了信号生成的基本代码后，我们需要知道如何评估一个策略的好坏。接下来，我们将探讨策略回测中需要关注的核心绩效指标。

## 关键绩效指标

在回测结束后，我们需要通过一系列指标来量化策略的表现。以下是几个最常用的评估指标：

*   **年化收益率**：策略在一年内的平均收益率，公式为 `(总收益率 + 1)^(252/交易天数) - 1`。
*   **夏普比率**：衡量每承受一单位总风险，所获得的超额回报。公式为 `(策略年化收益率 - 无风险利率) / 策略收益的年化标准差`。
*   **最大回撤**：策略从任一历史高点下跌的最大幅度，是衡量策略下行风险的关键指标。
*   **胜率**：盈利交易次数占总交易次数的比例。
