## 数据选择与重采样 🔍

![](img/3c3a77bb89965d2f18cd99796b011b7e_40.png)

在获得原始数据后，我们需要选择特定的分析序列并进行聚合。

![](img/3c3a77bb89965d2f18cd99796b011b7e_42.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_44.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_46.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_48.png)

1.  **选择分析列**：股票数据通常包含开盘价、最高价、最低价、收盘价等。我们选择`close`（收盘价）列作为预测目标。
2.  **进行重采样**：为了平滑数据并降低噪声，可以按周对收盘价进行重采样，并计算每周的平均收盘价。`W-MON`表示以每周一作为周期的开始。
    ```python
    weekly_data = data[‘close‘].resample(‘W-MON‘).mean()
    ```
3.  **划分训练集**：选择2000年至2015年的数据作为训练集，用于构建模型。

![](img/3c3a77bb89965d2f18cd99796b011b7e_58.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_60.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_64.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_66.png)

1.  **绘制序列图**：绘制训练期间收盘价的走势图。通常股票价格波动较大，呈现明显的非平稳性。
2.  **进行一阶差分**：对序列进行一阶差分以消除趋势。差分公式为：**y‘_t = y_t - y_{t-1}**。
    ```python
    diff_data = weekly_data.diff().dropna()
    ```
3.  **评估差分效果**：观察一阶差分后的序列图。如果序列围绕均值波动，且无明显趋势或周期性，则认为基本满足平稳性要求。若仍不平稳，可尝试二阶差分。

## 确定模型参数 (p, d, q) 🧮

![](img/3c3a77bb89965d2f18cd99796b011b7e_76.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_78.png)

确定序列平稳后，我们需要通过自相关图（ACF）和偏自相关图（PACF）来确定ARIMA模型的`p`和`q`参数。

![](img/3c3a77bb89965d2f18cd99796b011b7e_80.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_82.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_84.png)

以下是参数确定方法：

![](img/3c3a77bb89965d2f18cd99796b011b7e_86.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_88.png)

1.  **绘制ACF与PACF图**：对平稳化后的序列（即差分后的数据）绘制ACF和PACF图。
2.  **观察截尾点**：
    *   **ACF图**：用于确定移动平均阶数`q`。如果ACF在滞后`q`阶后突然截尾（落入置信区间），则`q`可选为该值。
    *   **PACF图**：用于确定自回归阶数`p`。如果PACF在滞后`p`阶后突然截尾，则`p`可选为该值。
3.  **参数选择**：在本案例中，ACF和PACF图均显示在一阶滞后后迅速落入置信区间，因此初步选择 **p=1， q=1**。差分阶数`d`即为之前进行的一阶差分，所以 **d=1**。
    > **提示**：更严谨的方法是计算不同(p, q)组合对应的AIC或BIC值，选择信息准则最小的组合。

## 模型训练与预测 🤖

![](img/3c3a77bb89965d2f18cd99796b011b7e_96.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_98.png)

参数确定后，即可构建并训练ARIMA模型，然后进行预测。

![](img/3c3a77bb89965d2f18cd99796b011b7e_100.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_102.png)

以下是模型训练与预测流程：

![](img/3c3a77bb89965d2f18cd99796b011b7e_104.png)

1.  **建立ARIMA模型**：使用`statsmodels`库，传入训练数据、参数`order=(p, d, q)`以及数据的频率（本例中为周数据‘W‘）。
    ```python
    from statsmodels.tsa.arima.model import ARIMA
    model = ARIMA(train_data, order=(1, 1, 1), freq=‘W‘)
    ```
2.  **训练模型**：调用`fit()`方法训练模型。可以通过`model.summary()`查看详细的模型统计信息，包括各系数的显著性等。
    ```python
    fitted_model = model.fit()
    ```
3.  **进行预测**：使用训练好的模型进行预测。需要指定预测的起始和结束时间。**起始时间必须在训练数据的时间范围内**，而结束时间可以是未来的任意时间点。
    ```python
    forecast = fitted_model.predict(start=pred_start_date, end=pred_end_date)
    ```

## 总结与拓展 💡

![](img/3c3a77bb89965d2f18cd99796b011b7e_136.png)

本节课中我们一起学习了使用ARIMA模型进行股票价格预测的完整流程。

![](img/3c3a77bb89965d2f18cd99796b011b7e_138.png)

![](img/3c3a77bb89965d2f18cd99796b011b7e_140.png)

**核心流程总结如下**：
1.  **数据准备**：读取数据，设置时间索引，选择目标序列（如收盘价），并进行重采样。
2.  **平稳化处理**：通过差分（通常一阶差分即可）使序列变得平稳。
3.  **参数定阶**：通过观察平稳序列的ACF和PACF图，初步确定ARIMA模型的`p`和`q`参数。
4.  **建模预测**：使用`(p, d, q)`参数构建ARIMA模型，训练后对未来进行预测。
5.  **评估验证**：可视化预测结果，对比趋势，评估模型有效性。
