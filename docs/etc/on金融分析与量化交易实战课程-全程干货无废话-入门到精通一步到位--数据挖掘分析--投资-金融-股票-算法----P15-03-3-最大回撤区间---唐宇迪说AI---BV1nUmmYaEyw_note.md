在本节课中，我们将要学习一个重要的风险指标——最大回撤。它描述的是在投资过程中，资产价值从最高点到最低点的最大跌幅，反映了策略可能面临的最糟糕情况。

## 概述

上一节我们介绍了策略分析中的一些基本概念，本节中我们来看看如何量化评估投资策略的风险。最大回撤正是这样一个关键的风险指标，它衡量的是投资过程中“最惨”的那段时期。

## 什么是最大回撤？

最大回撤不能仅从字面理解。它描述的是在炒股或投资过程中，资产价值下跌最严重、持续时间最长的那段区间。简单来说，它衡量的是你投资过程中“惨劲儿”有多大。

![](img/98b03c7ddfdaaed2647256f54d10e3fa_5.png)

**公式**如下：
`最大回撤 = max((P_i - P_j) / P_i)`
其中，`P_i` 表示某一时刻（第 `i` 天）的资产价值，`P_j` 表示 `i` 之后某一时刻（第 `j` 天）的资产价值，且 `P_j < P_i`。我们需要遍历所有可能的 `i` 和 `j` 组合，找出比值最大的那个。

![](img/98b03c7ddfdaaed2647256f54d10e3fa_7.png)

这个指标评估的不是策略的绝对好坏，而是策略的风险。即使一个策略最终是盈利的，但如果在过程中经历了巨大的回撤，也可能因为投资者无法承受心理压力或资金链断裂而失败。例如，2008年股市从高点大幅下跌，就让许多投资者承受了巨大损失。

## 如何计算最大回撤？

![](img/98b03c7ddfdaaed2647256f54d10e3fa_13.png)

![](img/98b03c7ddfdaaed2647256f54d10e3fa_15.png)

理解了概念后，我们来看看如何用Python代码进行计算。核心思路是：我们需要找出历史上资产价值从前期高点下跌的最大幅度。

![](img/98b03c7ddfdaaed2647256f54d10e3fa_17.png)

![](img/98b03c7ddfdaaed2647256f54d10e3fa_19.png)

以下是计算最大回撤的步骤：

![](img/98b03c7ddfdaaed2647256f54d10e3fa_21.png)

![](img/98b03c7ddfdaaed2647256f54d10e3fa_23.png)

![](img/98b03c7ddfdaaed2647256f54d10e3fa_25.png)

1.  **计算累积最大值**：首先，我们需要计算到每一天为止的历史最高价（累积最大值）。
2.  **计算每日回撤**：然后，用这个历史最高价减去当天的实际价格，得到每一天的“潜在下跌”幅度。
3.  **计算回撤比率**：接着，将这个下跌幅度除以对应的历史最高价，得到回撤比率。
4.  **找出最大值**：最后，在所有回撤比率中找出最大值，即为最大回撤。

![](img/98b03c7ddfdaaed2647256f54d10e3fa_27.png)

![](img/98b03c7ddfdaaed2647256f54d10e3fa_29.png)

我们将使用Pandas库的 `cummax()` 函数来高效地计算累积最大值。

![](img/98b03c7ddfdaaed2647256f54d10e3fa_31.png)

**代码**示例：
```python
import pandas as pd

![](img/98b03c7ddfdaaed2647256f54d10e3fa_33.png)

![](img/98b03c7ddfdaaed2647256f54d10e3fa_35.png)

![](img/98b03c7ddfdaaed2647256f54d10e3fa_37.png)

# 假设 df[‘close’] 是包含每日收盘价的序列
# 1. 计算累积最大值
cumulative_max = df[‘close’].cummax()
# 2. 计算每日回撤（下跌幅度）
drawdown = cumulative_max - df[‘close’]
# 3. 计算每日回撤比率
drawdown_pct = drawdown / cumulative_max
# 4. 找出最大回撤比率
max_drawdown = drawdown_pct.max()
print(f“最大回撤为：{max_drawdown:.2%}”)
```

![](img/98b03c7ddfdaaed2647256f54d10e3fa_39.png)

让我们分解一下这段代码：
*   `df[‘close’].cummax()`：生成一个新序列，其中每个值都是到该日为止 `df[‘close’]` 的最大值。
*   `cumulative_max - df[‘close’]`：用历史最高点减去当日收盘价，得到下跌的绝对值。
*   `drawdown / cumulative_max`：将下跌绝对值转化为相对于历史高点的百分比，这才是衡量“惨劲儿”的标准。
*   `.max()`：最终找出这个百分比序列中的最大值。

![](img/98b03c7ddfdaaed2647256f54d10e3fa_41.png)

在实际的策略回测框架（如Backtrader、Zipline）中，这些指标通常会被自动计算并展示在分析图表中。图表中会用阴影区域标出最大回撤发生的具体时间区间，直观地展示策略的风险阶段。
