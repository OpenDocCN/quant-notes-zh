## 回测收益率详解

上一节我们介绍了策略评估的必要性，本节中我们来看看最核心的指标——回测收益率。

![](img/8d154769d79f8d6e54e573bac7516333_11.png)

![](img/8d154769d79f8d6e54e573bac7516333_13.png)

回测收益率衡量的是，在回测周期内，策略最终的总收益相对于初始本金的增长比例。它是最直观的绩效指标，告诉你“折腾一番后，本金总共增加了多少百分比”。

![](img/8d154769d79f8d6e54e573bac7516333_15.png)

![](img/8d154769d79f8d6e54e573bac7516333_17.png)

其计算公式如下：

![](img/8d154769d79f8d6e54e573bac7516333_19.png)

![](img/8d154769d79f8d6e54e573bac7516333_21.png)

**公式**：`回测收益率 = (期末资产总值 - 期初资产总值) / 期初资产总值`

也可以表示为：`回测收益率 = (P_t / P_0) - 1`
其中，`P_t` 代表期末的资产价格（或总值），`P_0` 代表期初的资产价格（或本金）。

![](img/8d154769d79f8d6e54e573bac7516333_23.png)

接下来，我们将通过Python代码，基于一份示例股票数据，计算并可视化其回测收益率。

![](img/8d154769d79f8d6e54e573bac7516333_25.png)

首先，导入必要的工具包并加载数据。

![](img/8d154769d79f8d6e54e573bac7516333_27.png)

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便绘图显示
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，指定日期列为索引
data = pd.read_csv('your_stock_data.csv', index_col=0)
# 查看数据前几行
print(data.head())
```

假设数据包含股票的每日收盘价。我们将计算每个交易日相对于起始日的资产变化比率，并最终得出整个回测周期的总收益率。

以下是计算步骤：

1.  计算每日资产相对于期初的比值序列。
2.  根据比值序列绘制资产净值曲线。
3.  利用期末的比值计算总回测收益率。

![](img/8d154769d79f8d6e54e573bac7516333_29.png)

![](img/8d154769d79f8d6e54e573bac7516333_31.png)

```python
# 1. 计算每日资产净值（假设以收盘价代表资产价值）
# 即：每日收盘价 / 第一期收盘价
net_value = data['close'] / data['close'].iloc[0]

![](img/8d154769d79f8d6e54e573bac7516333_33.png)

![](img/8d154769d79f8d6e54e573bac7516333_35.png)

# 2. 绘制资产净值曲线
plt.figure(figsize=(10, 6))
net_value.plot()
plt.title('策略资产净值曲线')
plt.ylabel('净值（相对于期初）')
plt.xlabel('日期')
plt.grid(True)
plt.show()

![](img/8d154769d79f8d6e54e573bac7516333_37.png)

![](img/8d154769d79f8d6e54e573bac7516333_39.png)

# 3. 计算总回测收益率
# 回测收益率 = (期末净值 - 1)
total_return = net_value.iloc[-1] - 1

# 将结果整理成DataFrame展示
result_df = pd.DataFrame({
    '回测收益率': [total_return]
}, index=['策略'])
print(result_df)
```

![](img/8d154769d79f8d6e54e573bac7516333_41.png)

![](img/8d154769d79f8d6e54e573bac7516333_43.png)

执行以上代码后，你会看到一条展示资产净值随时间变化的曲线，以及一个显示具体回测收益率数值的表格。这个收益率数值（例如0.042表示4.2%）直接告诉你策略在测试期间的整体表现。
