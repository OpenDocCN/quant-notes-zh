在本节课中，我们将要学习一种在金融数据分析中常用的去极值方法——3 Sigma方法。我们将从正态分布的基本概念入手，理解3 Sigma方法的原理，并通过代码实例演示如何实现它。

## 正态分布与极值概念 📈

上一节我们介绍了去极值的概念，本节中我们来看看基于统计原理的3 Sigma方法。理解3 Sigma方法，首先需要了解正态分布。

正态分布是一种常见的概率分布，其形状呈钟形曲线。在正态分布中，大多数数据点会集中在均值附近，而远离均值的数据点出现的概率则较低。我们可以将数据中的正常波动看作是分布在均值附近的数据，而极值或异常值则是那些远离均值、出现概率极低的数据点。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_1.png)

为了更直观地理解，请看下图。图中展示了一个标准正态分布曲线，均值（μ）为0。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_1.png)

图中阴影区域代表了数据落在某个区间内的概率。靠近均值的红色区域面积较大，意味着数据落在此处的概率较高，属于正常数据。而越向两侧延伸，曲线下的面积越小，意味着数据落在此处的概率越低，这些远离均值的点就更可能是我们需要处理的极值或离群点。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_3.png)

## 3 Sigma方法原理 🔍

理解了正态分布后，我们来看什么是3 Sigma方法。Sigma（σ）代表标准差，是衡量数据离散程度的指标。

在正态分布中，数据落在特定均值加减标准差倍数区间内的概率是固定的。具体来说：
*   数据落在 **μ ± 1σ** 区间内的概率约为 **68.3%**。
*   数据落在 **μ ± 2σ** 区间内的概率约为 **95.4%**。
*   数据落在 **μ ± 3σ** 区间内的概率约为 **99.7%**。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_3.png)

3 Sigma方法正是基于这个原理。我们假设经过处理的金融数据近似服从正态分布。当我们设定上下限为均值加减3倍标准差（μ ± 3σ）时，理论上可以覆盖99.7%的数据。此时，我们将落在这个区间之外的数据视为极值。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_5.png)

**核心思想**：将数据的上下限定义为 **均值 ± N倍的标准差**。通常N=3，以保留99.7%的数据，并将剩余的0.3%作为极值进行规范化处理（如缩放到边界上），而不是直接删除。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_7.png)

因此，计算边界只需要两个统计量：
*   **均值（μ）**
*   **标准差（σ）**

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_9.png)

上下限的计算公式为：
**上限 = μ + N * σ**
**下限 = μ - N * σ**

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_11.png)

## 3 Sigma方法代码实现 💻

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_13.png)

接下来，我们将通过代码实现3 Sigma方法。我们将创建一个函数，为输入的数据序列计算基于指定N倍标准差的上下限。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_15.png)

首先，我们需要计算数据的均值和标准差。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_17.png)

```python
# 计算均值 (mean)
mean = data_series.mean()

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_19.png)

# 计算标准差 (std)
std = data_series.std()
```

有了均值和标准差，就可以根据公式计算上下限。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_21.png)

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_23.png)

以下是完整的函数实现：

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_25.png)

```python
def sigma_cut(data_series, n=3):
    """
    使用3 Sigma方法计算去极值的上下限。
    参数:
        data_series: 输入的数据序列（Pandas Series）。
        n: 标准差的倍数，默认为3。
    返回:
        lower_limit: 计算出的下限。
        upper_limit: 计算出的上限。
    """
    # 计算均值和标准差
    mean = data_series.mean()
    std = data_series.std()
    
    # 根据公式计算上下限
    upper_limit = mean + n * std
    lower_limit = mean - n * std
    
    return lower_limit, upper_limit
```

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_27.png)

现在，我们可以使用这个函数来处理数据。例如，我们有一列金融收益率数据 `returns`，我们想用1倍标准差（覆盖约68.3%的数据）来观察其边界。

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_29.png)

```python
# 应用函数，n=1
lower, upper = sigma_cut(returns, n=1)
print(f"下限: {lower}, 上限: {upper}")
```

![](img/aaa4df4ac1c7454395505e79c7cf6d0d_31.png)
