## NumPy模块简介 📊

![](img/6d8b81c37869dffb698e1255246e7151_7.png)

上一节我们介绍了数据分析环境的搭建，本节中我们来看看在数据分析中常用的核心模块。在数据分析领域，有三个至关重要的模块，它们被称为“数据分析三剑客”，分别是：**NumPy**、**pandas** 和 **matplotlib**。我们后续的所有数据分析操作都将围绕这三个模块展开。

![](img/6d8b81c37869dffb698e1255246e7151_9.png)

![](img/6d8b81c37869dffb698e1255246e7151_11.png)

首先，我们来认识第一个模块：NumPy。需要明确的是，在整个课程体系中，**pandas** 模块是我们的重点学习对象，它是数据分析的核心操作工具。NumPy 和 matplotlib 虽然同样重要，但相对于 pandas 而言，其核心地位稍低。NumPy 模块的一个重要角色是为 pandas 模块提供数据源，等我们学习到 pandas 时，你会更清楚地理解这种关系。

![](img/6d8b81c37869dffb698e1255246e7151_13.png)

那么，NumPy 究竟是什么？简单来说，NumPy 可以被理解为一个用于处理**一维或多维数组**的库。如果你有其他编程语言（如C、Java）的经验，应该对“数组”这个概念不陌生。如果只接触过Python，你可以暂时将NumPy数组看作一种特殊的**列表**，或者将列表看作一种特殊的数组。无论是列表还是数组，它们都是一种**数据结构**，即一种可以装载数据的**容器**。

NumPy数组这个容器主要装载**数值型数据**。因此，NumPy模块的核心在于**数值计算**。当我们需要对一组数值进行运算时，可以先将这些数据装入NumPy数组，再利用NumPy数组的特性进行高效计算。

此外，NumPy也是Python**科学计算库的基础**。这意味着NumPy内部已经封装好了各种用于科学计算的工具类。例如，如果我们需要对一组数据进行反三角函数运算，无需自己实现，直接调用NumPy中封装好的函数即可。

总结NumPy的两大重点：
1.  它是一个容器，用于装载和运算**数值型数据**。
2.  它是Python科学计算的基础，提供了丰富的预置科学计算工具。

## 创建NumPy数组的三种方式 🔧

![](img/6d8b81c37869dffb698e1255246e7151_15.png)

介绍完NumPy的基本概念后，接下来我们看看如何创建NumPy数组。要使用NumPy，首先需要导入模块。业界通常约定使用 `np` 作为别名。

![](img/6d8b81c37869dffb698e1255246e7151_17.png)

```python
import numpy as np
```

导入模块后，我们就可以开始创建数组了。NumPy模块在Anaconda集成环境中已预先安装，无需单独下载。

### 方式一：使用 `np.array()` 从现有数据创建

第一种，也是最直接的方式，是使用 `np.array()` 函数将现有的Python数据结构（如列表）转换为NumPy数组。

我们可以通过 `np.array()` 创建一维或多维数组。该函数的主要参数是 `object`（数据源）和可选的 `dtype`（指定数组元素的数据类型）。如果不指定 `dtype`，NumPy会自动推断。

![](img/6d8b81c37869dffb698e1255246e7151_19.png)

以下是创建一维和二维数组的示例：

![](img/6d8b81c37869dffb698e1255246e7151_21.png)

```python
# 创建一维数组
arr_1d = np.array([1, 2, 3])
print(arr_1d)
# 输出: array([1, 2, 3])

![](img/6d8b81c37869dffb698e1255246e7151_23.png)

![](img/6d8b81c37869dffb698e1255246e7151_25.png)

# 创建二维数组
arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
print(arr_2d)
# 输出: array([[1, 2, 3],
#              [4, 5, 6]])
```

![](img/6d8b81c37869dffb698e1255246e7151_27.png)

在实际应用中，我们最常使用的是一维、二维和三维数组，因为过高的维度不便于理解和操作。

![](img/6d8b81c37869dffb698e1255246e7151_29.png)

掌握了基本的数组创建后，我们来探讨一个关键问题：NumPy数组与Python列表的核心区别是什么？一个重要的区别在于**数组元素的类型必须统一**。

请看下面的例子：

![](img/6d8b81c37869dffb698e1255246e7151_31.png)

```python
arr_mixed = np.array([1, 2.2, ‘three‘])
print(arr_mixed)
# 输出: array([‘1‘, ‘2.2‘, ‘three‘], dtype=‘<U32‘)
```

![](img/6d8b81c37869dffb698e1255246e7151_33.png)

![](img/6d8b81c37869dffb698e1255246e7151_35.png)

你会发现，原本包含整型、浮点型和字符串的列表，在转换为NumPy数组后，所有元素都被强制转换成了**字符串类型**。这是因为NumPy数组要求元素类型一致，当类型不一致时，它会按照一个优先级顺序（字符串 > 浮点数 > 整数）进行**类型提升**，将所有元素转换为优先级最高的类型。而Python列表则允许元素类型各不相同。

### 方式二：使用 `plt.imread()` 从图像文件创建

第二种创建数组的方式与图像处理相关。我们可以将外部的图片文件读取并加载到NumPy数组中。图片本质上是由二进制数据构成的，这些数据可以视为数值，因此能够装入NumPy数组。

![](img/6d8b81c37869dffb698e1255246e7151_37.png)

![](img/6d8b81c37869dffb698e1255246e7151_39.png)

这个操作需要用到三剑客中的另一个模块：`matplotlib`。这里我们先初步使用它。

以下是读取图片并创建数组的步骤：

```python
import matplotlib.pyplot as plt

![](img/6d8b81c37869dffb698e1255246e7151_41.png)

![](img/6d8b81c37869dffb698e1255246e7151_43.png)

# 读取图片，数据将存储在NumPy数组中
image_array = plt.imread(‘1.jpg‘)
print(image_array.shape) # 查看数组形状，例如 (高度, 宽度, 通道数)
# 显示图片
plt.imshow(image_array)
plt.show()
```

![](img/6d8b81c37869dffb698e1255246e7151_45.png)

`plt.imread()` 读取的 `image_array` 是一个多维NumPy数组（通常是三维，代表图片的高度、宽度和颜色通道）。`plt.imshow()` 函数则用于将这个存储图像数据的数组可视化显示出来。

![](img/6d8b81c37869dffb698e1255246e7151_47.png)

![](img/6d8b81c37869dffb698e1255246e7151_49.png)

这种方式的强大之处在于，我们可以直接对数组中的数值进行运算，从而改变图像。例如，让所有像素值减去一个常数：

![](img/6d8b81c37869dffb698e1255246e7151_51.png)

```python
# 修改数组数据（所有像素值减100）
modified_image = image_array - 100
# 显示修改后的图片
plt.imshow(modified_image)
plt.show()
```

![](img/6d8b81c37869dffb698e1255246e7151_53.png)

![](img/6d8b81c37869dffb698e1255246e7151_55.png)

执行后你会发现图片的亮度或色彩发生了变化。这揭示了图像处理软件（如美图秀秀）的原理：它们本质上是在对构成图片的数值数据进行特定的数学运算和加工。

![](img/6d8b81c37869dffb698e1255246e7151_57.png)

### 方式三：使用NumPy内置函数创建特定数组

![](img/6d8b81c37869dffb698e1255246e7151_63.png)

第三种方式是使用NumPy提供的一系列内置函数，快速创建具有特定规律或形状的数组。

![](img/6d8b81c37869dffb698e1255246e7151_65.png)

![](img/6d8b81c37869dffb698e1255246e7151_67.png)

以下是几个常用函数：

![](img/6d8b81c37869dffb698e1255246e7151_69.png)

![](img/6d8b81c37869dffb698e1255246e7151_71.png)

![](img/6d8b81c37869dffb698e1255246e7151_73.png)

*   `np.zeros(shape)`：创建指定形状、元素全为0的数组。
*   `np.ones(shape)`：创建指定形状、元素全为1的数组。
*   `np.linspace(start, stop, num)`：在指定区间内创建**等差数列**。`num` 参数指定生成元素的个数。**返回一维数组**。
*   `np.arange(start, stop, step)`：在指定区间内，按照固定步长创建数组。**返回一维数组**。
*   `np.random.randint(low, high, size)`：创建指定形状、元素为指定范围内**随机整数**的数组。

![](img/6d8b81c37869dffb698e1255246e7151_75.png)

以下是使用示例：

![](img/6d8b81c37869dffb698e1255246e7151_77.png)

```python
# 创建3行4列的全1数组
arr_ones = np.ones((3, 4))
print(arr_ones)

![](img/6d8b81c37869dffb698e1255246e7151_79.png)

# 创建从0到100的20个等间隔数（等差数列）
arr_lin = np.linspace(0, 100, 20)
print(arr_lin)

![](img/6d8b81c37869dffb698e1255246e7151_81.png)

# 创建从10到50，步长为2的数组
arr_arange = np.arange(10, 50, 2)
print(arr_arange)

![](img/6d8b81c37869dffb698e1255246e7151_83.png)

# 创建5行3列，元素为0到100之间随机整数的数组
arr_random = np.random.randint(0, 100, size=(5, 3))
print(arr_random)
```

![](img/6d8b81c37869dffb698e1255246e7151_85.png)

请注意，`np.linspace` 和 `np.arange` 通常用于生成一维序列数组，而 `np.zeros`、`np.ones` 和 `np.random.randint` 可以通过 `shape` 或 `size` 参数创建任意维度的数组。

![](img/6d8b81c37869dffb698e1255246e7151_87.png)
