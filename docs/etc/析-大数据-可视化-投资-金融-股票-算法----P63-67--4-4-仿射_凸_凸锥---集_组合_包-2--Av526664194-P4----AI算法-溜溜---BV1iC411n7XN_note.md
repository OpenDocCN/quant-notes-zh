# 凸优化：4.4：仿射集、凸集与凸锥 + 集、组合、包（2）📐

在本节课中，我们将继续学习凸优化中的核心概念：仿射集、凸集与凸锥。我们将深入理解如何从任意集合构造出这些特殊的集合，并掌握仿射组合、凸组合与凸锥组合的定义及其几何意义。

---

上一节我们介绍了仿射集的定义和仿射包的概念。本节中，我们来看看如何直观地理解仿射包，并引入更重要的概念——凸集。

## 仿射包的直观理解 🧠

![](img/c36d363323e45ec9b0c062f4499736ac_1.png)

我们从一个简单的例子开始，考虑二维空间中的一个集合，它只包含两个点 **X1** 和 **X2**。这个集合本身不是一个仿射集。那么它的仿射包是什么呢？

按照定义，我们从集合中任意选取点（这里就是这两个点），并构造它们的仿射组合。对于两个点，其仿射组合的集合就是连接这两点的**直线**。因此，这条直线就是由点集 {X1, X2} 构造出的仿射包。

现在，我们添加第三个点 **X3**。由这三个点构成的集合，其仿射包是什么呢？它并不是连接每两点的三条线段。因为仿射组合可以是任意多个点的组合。对于这三个不共线的点，其所有仿射组合构成的集合，实际上就是**整个二维平面**。这个平面是包含原集合的最小仿射集。

![](img/c36d363323e45ec9b0c062f4499736ac_3.png)

![](img/c36d363323e45ec9b0c062f4499736ac_5.png)

如果集合 **C** 本身就是一个仿射集，那么它的仿射包就是它自身。

---

在理解了仿射集之后，我们接下来要关注本章的核心——凸集。

![](img/c36d363323e45ec9b0c062f4499736ac_7.png)

## 凸集的定义 📍

一个集合 **C** 是凸集，当且仅当连接其中任意两点的**线段**仍然包含在 **C** 内。

用数学语言描述：对于任意 **X1, X2 ∈ C** 和任意标量 **θ**，其中 **0 ≤ θ ≤ 1**，都有：
**θX1 + (1-θ)X2 ∈ C**

从这个定义可以立刻看出，**仿射集一定是凸集**。因为连接两点的整条直线都在集合内，那么连接这两点的线段也必然在集合内。因此，仿射集是凸集的一个特例。

## 凸组合

之前我们定义了仿射组合。相应地，我们定义凸组合。

以下是凸组合的定义：
考虑 **K** 个点 **X1, ..., XK** 和 **K** 个标量 **θ1, ..., θK**。如果这些标量满足：
1.  **θ1 + θ2 + ... + θK = 1**
2.  **θi ≥ 0**，对于所有 **i = 1, ..., K**

那么，由这些点和标量构成的线性组合 **θ1X1 + θ2X2 + ... + θKXK** 称为这 **K** 个点的一个**凸组合**。

![](img/c36d363323e45ec9b0c062f4499736ac_9.png)

![](img/c36d363323e45ec9b0c062f4499736ac_11.png)

与仿射集类似，凸集也可以用组合来定义：一个集合 **C** 是凸集，当且仅当其中任意元素的任意凸组合仍然在 **C** 内。

![](img/c36d363323e45ec9b0c062f4499736ac_13.png)

![](img/c36d363323e45ec9b0c062f4499736ac_15.png)

![](img/c36d363323e45ec9b0c062f4499736ac_17.png)

## 凸包

![](img/c36d363323e45ec9b0c062f4499736ac_19.png)

![](img/c36d363323e45ec9b0c062f4499736ac_21.png)

![](img/c36d363323e45ec9b0c062f4499736ac_23.png)

类似于仿射包，我们可以定义凸包。

![](img/c36d363323e45ec9b0c062f4499736ac_25.png)

集合 **C** 的**凸包**，是包含 **C** 的所有凸组合的集合。换句话说，它是包含 **C** 的**最小凸集**。

为了帮助理解，我们来看一些几何例子：
*   一个实心的六边形区域是一个凸集。
*   只有一个六边形边框的集合**不是**凸集（连接某些顶点的线段不在边框上）。
*   一个正方形的内部挖去两个点后**不是**凸集（连接这两个洞的线段穿过空洞）。
*   一个正方形的内部只挖去两个角上的点，它**是**凸集。
*   对于这些非凸的集合，它们的凸包就是将其“凹陷”或“缺口”部分用直线段填充后得到的最小凸形。
*   即使是离散的点集（例如教室里的同学们），也有凸包，即用一根“绳子”套住最外围的点所围成的区域。

讲完凸集，我们要介绍另一个重要的集合——锥。

![](img/c36d363323e45ec9b0c062f4499736ac_27.png)

## 锥与凸锥 🧊

![](img/c36d363323e45ec9b0c062f4499736ac_29.png)

![](img/c36d363323e45ec9b0c062f4499736ac_30.png)

![](img/c36d363323e45ec9b0c062f4499736ac_32.png)

一个集合 **C** 是一个**锥**，如果对于任意 **X ∈ C** 和任意标量 **θ ≥ 0**，都有 **θX ∈ C**。这意味着从原点出发，经过集合中任意点的整条射线都在集合内。

![](img/c36d363323e45ec9b0c062f4499736ac_34.png)

![](img/c36d363323e45ec9b0c062f4499736ac_36.png)

锥不一定是凸的。我们特别关注既是锥又是凸集的集合，称为**凸锥**。

![](img/c36d363323e45ec9b0c062f4499736ac_38.png)

![](img/c36d363323e45ec9b0c062f4499736ac_40.png)

一个集合 **C** 是凸锥，如果对于任意 **X1, X2 ∈ C** 和任意标量 **θ1, θ2 ≥ 0**，都有：
**θ1X1 + θ2X2 ∈ C**

![](img/c36d363323e45ec9b0c062f4499736ac_42.png)

![](img/c36d363323e45ec9b0c062f4499736ac_44.png)

![](img/c36d363323e45ec9b0c062f4499736ac_46.png)

![](img/c36d363323e45ec9b0c062f4499736ac_48.png)

几何上：
*   从原点出发的几条射线构成一个锥（像散开的火柴）。
*   一个“Y”字形不是锥，因为其分支交点不在原点。
*   一个以原点为顶点的实心冰淇淋蛋筒形状，既是锥又是凸集，因此是一个凸锥。

![](img/c36d363323e45ec9b0c062f4499736ac_50.png)

![](img/c36d363323e45ec9b0c062f4499736ac_52.png)

## 凸锥组合与凸锥包

![](img/c36d363323e45ec9b0c062f4499736ac_53.png)

![](img/c36d363323e45ec9b0c062f4499736ac_55.png)

![](img/c36d363323e45ec9b0c062f4499736ac_57.png)

类似地，我们可以定义凸锥组合。

![](img/c36d363323e45ec9b0c062f4499736ac_59.png)

![](img/c36d363323e45ec9b0c062f4499736ac_61.png)

以下是凸锥组合的定义：
考虑 **K** 个点 **X1, ..., XK** 和 **K** 个标量 **θ1, ..., θK**，其中 **θi ≥ 0**（不要求和为1）。那么组合 **θ1X1 + θ2X2 + ... + θKXK** 称为这 **K** 个点的一个**凸锥组合**。

![](img/c36d363323e45ec9b0c062f4499736ac_62.png)

![](img/c36d363323e45ec9b0c062f4499736ac_64.png)

集合 **C** 的**凸锥包**，是包含 **C** 的所有凸锥组合的集合，也就是包含 **C** 的**最小凸锥**。

![](img/c36d363323e45ec9b0c062f4499736ac_66.png)

![](img/c36d363323e45ec9b0c062f4499736ac_67.png)

![](img/c36d363323e45ec9b0c062f4499736ac_68.png)

例如：
*   二维平面中一组离散点的凸锥包，是以原点为顶点，将所有最外围的点用射线连接后所夹的扇形区域（一个“冰激凌锥”）。
*   一条过原点的直线段，其凸锥包是从原点出发、指向线段方向的一条射线。

![](img/c36d363323e45ec9b0c062f4499736ac_70.png)

## 概念总结与关系 🔗

![](img/c36d363323e45ec9b0c062f4499736ac_79.png)

![](img/c36d363323e45ec9b0c062f4499736ac_81.png)

本节课我们一起学习了三组核心概念：

![](img/c36d363323e45ec9b0c062f4499736ac_83.png)

**三种集合**：
1.  **仿射集**：包含任意两点间整条直线的集合。
2.  **凸集**：包含任意两点间线段的集合。
3.  **凸锥**：既是锥（对任意 θ ≥ 0 缩放封闭）又是凸集的集合。

![](img/c36d363323e45ec9b0c062f4499736ac_85.png)

**三种组合**（对于点 **X1, ..., XK** 和标量 **θ1, ..., θK**）：
1.  **仿射组合**：要求 **Σθi = 1**，对 **θi** 符号无限制。
2.  **凸组合**：要求 **Σθi = 1** 且 **θi ≥ 0**。
3.  **凸锥组合**：要求 **θi ≥ 0**，不要求和为1。

![](img/c36d363323e45ec9b0c062f4499736ac_87.png)

**三种包**：包含原集合的**最小**的相应类型的集合。

它们之间的关系是：
*   仿射集 ⇒ 凸集。
*   凸锥 ⇒ 凸集。
*   凸组合是仿射组合加上非负限制。
*   凸锥组合是凸组合去掉“和为1”的限制。

![](img/c36d363323e45ec9b0c062f4499736ac_89.png)

**特例讨论**：
*   单点集 **{X}**：是仿射集，也是凸集。只有当该点是原点时，它才是凸锥。
*   空集 **∅**：它既是仿射集，也是凸集，也是凸锥。

---

本节课中，我们一起深入学习了仿射集、凸集和凸锥的定义、几何意义以及它们之间的衍生概念（组合与包）。理解这些基础概念是后续学习凸函数和优化问题的重要基石。
