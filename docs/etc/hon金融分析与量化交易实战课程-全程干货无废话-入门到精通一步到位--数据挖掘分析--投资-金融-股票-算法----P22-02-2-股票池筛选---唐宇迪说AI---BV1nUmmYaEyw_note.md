## 交易前的数据预处理

![](img/4194e47a75fa0287600315478a52e2e2_9.png)

![](img/4194e47a75fa0287600315478a52e2e2_11.png)

在每日交易开始前，我们需要对股票池进行筛选。这个过程称为预处理，核心是获取并分析股票的财务数据。

![](img/4194e47a75fa0287600315478a52e2e2_13.png)

![](img/4194e47a75fa0287600315478a52e2e2_15.png)

以下是实现预处理的关键步骤：

![](img/4194e47a75fa0287600315478a52e2e2_17.png)

1.  **查询财务指标**：使用平台提供的`query`功能，查询我们关心的财务数据，例如营业收入。
2.  **应用过滤器**：通过`filter`条件，确保我们只处理股票池内的股票。
3.  **排序与筛选**：对查询结果进行排序（例如按营业收入降序排列），并限制返回的股票数量（例如只取前10名）。

```python
def before_trading_start(context):
    # 构建查询，获取营业收入指标
    q = query(
        fundamentals.eod_derivative_indicator.operating_revenue
    ).filter(
        fundamentals.eod_derivative_indicator.code.in_(context.stock_pool)
    ).order_by(
        fundamentals.eod_derivative_indicator.operating_revenue.desc()
    ).limit(10)

    # 执行查询，将结果存入context以备交易函数使用
    context.filtered_stocks = get_fundamentals(q)
    # 打印结果以供检查
    print(context.filtered_stocks)
```

## 查询功能的详细说明

量化交易本质上是数据挖掘，平台提供了丰富的指标可供查询。所有可用的指标和函数都在官方帮助文档中有详细说明。

以下是使用查询功能时需要注意的几点：

*   **参考文档**：所有可查询的指标（如财务数据、技术指标）都列在帮助文档中，使用前应仔细阅读。
*   **理解参数**：`query()`函数通常需要传入要查询的指标字段。
*   **组合使用**：可以链式调用`.filter()`, `.order_by()`, `.limit()`等方法对结果进行过滤、排序和数量限制。

如果查询结果为空，请检查以下方面：

![](img/4194e47a75fa0287600315478a52e2e2_27.png)

*   **股票池指定**：确保在`filter`中使用的股票池对象（如`context.stock_pool`）已被正确定义和赋值。最初我们可能错误地在股票列表中查找指数，需要修正为在指数列表中查找“沪深300”。
*   **字段名称**：确认查询的财务指标字段名称准确无误，可以参考帮助文档。
*   **回测日期**：确保回测的日期范围内，所查询的财务数据是存在的。
