在本节课中，我们将学习如何在一个交易平台上构建一个简单的量化交易策略。我们将通过一个具体的任务——从沪深300指数中动态选择并持有表现最佳的10只股票——来熟悉平台的核心API和策略编写流程。整个过程将分为初始化、交易前准备和实际交易执行三个主要步骤。

![](img/4f78a2ee23c78cfc1e34325bbb4f77b6_3.png)

## 策略目标概述 🎯

![](img/4f78a2ee23c78cfc1e34325bbb4f77b6_5.png)

我们的目标是创建一个策略，该策略始终持有沪深300指数中表现最好的10只股票。具体来说，我们会定期（例如每天）根据某个财务指标（如盈利能力）对沪深300中的所有股票进行排序，然后买入排名前10的股票。同时，我们会卖出手中已持有但不在新“前十”名单中的股票，以确保投资组合始终由当前表现最佳的股票构成。

## 策略实现步骤分解 🔧

上一节我们介绍了策略的整体目标，本节中我们来看看实现这个策略需要完成哪些具体步骤。以下是实现该策略的三个核心模块及其任务。

1.  **初始化 (`initialize`)**
    在策略开始运行时，我们需要获取沪深300指数的全部成分股，作为我们的初始股票池。

2.  **交易前处理 (`before_trading`)**
    在每个交易日开始前，我们需要从股票池中获取所有股票的财务数据（例如盈利数据），并根据选定的指标进行排序，筛选出当前表现最好的10只股票。

3.  **交易执行 (`handle_bar`)**
    在交易时段，我们需要对比当前持有的股票与`before_trading`阶段计算出的“最佳十股”名单。卖出已持有但不在新名单中的股票，并买入新名单中尚未持有的股票。

## 模块功能详解 📝

### 初始化模块：建立股票池

在初始化函数中，我们的首要任务是定义策略的股票池。我们将使用平台提供的API来获取沪深300指数的所有成分股代码。

```python
def initialize(context):
    # 获取沪深300指数成分股，并设置为策略的股票池
    stock_pool = get_index_stocks(‘000300.SH‘)
    set_universe(stock_pool)
```

### 交易前处理模块：筛选最佳股票

![](img/4f78a2ee23c78cfc1e34325bbb4f77b6_7.png)

在`before_trading`函数中，我们将执行每日的数据处理工作。这包括查询财务数据和排序筛选，这些操作不涉及实际交易。

以下是该模块需要完成的任务列表：
*   从股票池中获取所有股票的财务数据（例如，净利润）。
*   根据选定的财务指标（如净利润）对所有股票进行降序排序。
*   从排序结果中选取排名前10的股票代码，并存储起来供交易时段使用。

![](img/4f78a2ee23c78cfc1e34325bbb4f77b6_9.png)

![](img/4f78a2ee23c78cfc1e34325bbb4f77b6_11.png)

### 交易执行模块：调整投资组合

`handle_bar`函数是策略的核心交易逻辑发生的地方。在这里，我们将根据`before_trading`阶段得出的股票名单来调整实际持仓。

以下是该模块的核心逻辑：
*   **判断当前持仓**：查询当前账户中持有的所有股票。
*   **对比与调整**：
    *   卖出当前持有但不在新“前十”名单中的股票。
    *   用可用资金买入新“前十”名单中尚未持有的股票。
*   **目标状态**：使最终持仓与计算出的最佳十股名单完全一致。
