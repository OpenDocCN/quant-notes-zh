## 计算每日差异值

首先，我们需要计算每一天指标相对于前一天的差异值。这个指标反映了数据每日的绝对变化量。

![](img/37c1fb3b8418001839b5cf1cc360935d_5.png)

在Pandas中，我们可以使用 `diff()` 函数轻松实现这个计算。该函数默认计算当前行与前一行之间的差值。

![](img/37c1fb3b8418001839b5cf1cc360935d_7.png)

以下是计算差异值的代码：

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

执行上述代码后，你会看到一个新的DataFrame。例如，1月5日的数据是0.052，这个值是由1月5日当天的数据减去1月4日的数据得到的。由于1月4日之前没有数据，所以1月4日对应的行会显示为NaN（空值）。

`diff()` 函数默认计算相邻一天（即周期为1）的差异。如果你想计算间隔多天的差异，可以通过指定 `periods` 参数来实现。例如，`data.diff(periods=2)` 会计算当前数据与两天前数据的差异。

![](img/37c1fb3b8418001839b5cf1cc360935d_9.png)

![](img/37c1fb3b8418001839b5cf1cc360935d_10.png)

## 计算每日变化率（增长率）

仅仅知道绝对差异有时不够直观。例如，股价从1元涨到2元和从10000元涨到10001元，虽然都涨了1元，但意义完全不同。因此，我们更常使用变化率（增长率）来衡量相对变化。

![](img/37c1fb3b8418001839b5cf1cc360935d_12.png)

变化率的计算公式为：`(后一天的值 - 前一天的值) / 前一天的值`。

![](img/37c1fb3b8418001839b5cf1cc360935d_14.png)

Pandas提供了 `pct_change()` 函数来直接计算这个百分比变化。

![](img/37c1fb3b8418001839b5cf1cc360935d_15.png)

以下是计算增长率的代码：

```python
# 计算每日增长率
pct_data = data.pct_change()
print(pct_data.head())
```

计算完成后，数据会以小数形式表示增长率（例如0.01表示增长1%）。你可以使用 `.round(3)` 等方法将结果显示为更易读的格式。

![](img/37c1fb3b8418001839b5cf1cc360935d_17.png)

## 对变化率进行统计分析

得到每日的变化率后，我们可以对其进行统计分析，以了解整体趋势。

![](img/37c1fb3b8418001839b5cf1cc360935d_19.png)

![](img/37c1fb3b8418001839b5cf1cc360935d_20.png)

例如，我们可以计算所有交易日增长率的平均值，这能告诉我们该股票在观察期内的平均每日涨跌幅度。

![](img/37c1fb3b8418001839b5cf1cc360935d_22.png)

![](img/37c1fb3b8418001839b5cf1cc360935d_23.png)

以下是计算平均增长率的代码：

```python
# 计算平均增长率
mean_pct = pct_data.mean()
print(mean_pct)
```
