## 概述

![](img/e23167372a69f6db6262e35f0dec49c3_3.png)

`for`循环是Python中用于遍历序列（如列表、字符串等）的主要工具。与`while`循环不同，`for`循环通常用于已知迭代次数或需要遍历集合中每个元素的情况。本节我们将重点学习`for`循环的语法和基本应用。

![](img/e23167372a69f6db6262e35f0dec49c3_5.png)

## `for`循环的基本语法

`for`循环的语法结构与`while`循环有所不同。其基本格式如下：

```python
for 变量 in 序列:
    执行语句
```

*   **`for`**：循环语句的关键字。
*   **`变量`**：在每次循环中，该变量会被自动赋值为序列中的当前元素。
*   **`in`**：关键字，用于连接变量和要遍历的序列。
*   **`序列`**：可以是列表、字符串、元组或`range()`对象等可迭代对象。
*   **`:`**：冒号表示代码块的开始。
*   **`执行语句`**：需要重复执行的代码，必须缩进（通常是一个Tab键或4个空格）。

## 第一个`for`循环示例

为了更好地理解，我们可以回顾一下上节课用`while`循环实现的一个简单计数器。现在，我们用`for`循环来实现相同的功能。

![](img/e23167372a69f6db6262e35f0dec49c3_7.png)

以下是使用`while`循环的旧代码：
```python
循环次数 = 3
while 循环次数 > 0:
    print(“当前循环次数”, 循环次数)
    循环次数 -= 1
```

现在，我们使用`for`循环来改写：

![](img/e23167372a69f6db6262e35f0dec49c3_9.png)

```python
for 当前循环次数 in range(3):
    print(“当前循环次数”, 当前循环次数)
```

![](img/e23167372a69f6db6262e35f0dec49c3_11.png)

运行这段代码，你可能会发现输出结果是：
```
当前循环次数 0
当前循环次数 1
当前循环次数 2
```

这与预期略有不同。这是因为在编程中，索引通常从**0**开始。`range(3)`生成的是一个序列`[0, 1, 2]`。`for`循环会依次将0、1、2赋值给变量`当前循环次数`并执行打印操作。

![](img/e23167372a69f6db6262e35f0dec49c3_13.png)

为了让输出更符合习惯（从1开始计数），我们可以对变量进行简单的运算：

```python
for 当前循环次数 in range(3):
    print(“当前循环次数”, 当前循环次数 + 1)
```

现在输出就变成了：
```
当前循环次数 1
当前循环次数 2
当前循环次数 3
```

## 理解 `range()` 函数

![](img/e23167372a69f6db6262e35f0dec49c3_15.png)

在上面的例子中，我们使用了一个新函数 `range()`。它是`for`循环的得力助手，用于生成一个整数序列。

`range()`函数的基本用法：
*   `range(stop)`：生成从0开始，到`stop-1`结束的整数序列。例如`range(5)`生成`[0,1,2,3,4]`。
*   `range(start, stop)`：生成从`start`开始，到`stop-1`结束的整数序列。例如`range(1, 5)`生成`[1,2,3,4]`。
*   `range(start, stop, step)`：生成从`start`开始，到`stop-1`结束，步长为`step`的整数序列。例如`range(0, 10, 2)`生成`[0,2,4,6,8]`。

`for`循环会**遍历**`range()`函数产生的序列，依次将每个值赋给循环变量，并执行循环体内的代码。

## `for`循环的常见用途：遍历

`for`循环最典型的用途就是遍历各种数据结构。以下是几个常见的遍历示例：

![](img/e23167372a69f6db6262e35f0dec49c3_17.png)

### 遍历列表
列表是存储多个数据的有序集合，`for`循环可以轻松访问其中每一个元素。
```python
水果列表 = [“苹果”, “香蕉”, “橙子”]
for 水果 in 水果列表:
    print(“我喜欢吃”, 水果)
```
输出：
```
我喜欢吃 苹果
我喜欢吃 香蕉
我喜欢吃 橙子
```

### 遍历字符串
字符串也可以看作是一个由字符组成的序列。
```python
for 字符 in “Python”:
    print(字符)
```
输出：
```
P
y
t
h
o
n
```
