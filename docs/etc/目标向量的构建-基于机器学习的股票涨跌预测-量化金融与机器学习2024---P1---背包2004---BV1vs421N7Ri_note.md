# 量化金融与机器学习：P1：目标向量的构建

在本节课中，我们将学习如何为机器学习模型构建目标向量，以预测股票未来的涨跌。我们将把问题简化为一个二分类任务，并利用历史数据中的市盈率和总市值作为特征。

## 概述

上一节我们介绍了机器学习在量化金融中的应用背景。本节中，我们来看看如何具体构建一个用于预测股票涨跌的数据集。核心任务是：**基于某只股票在特定日期的特征（如市盈率、总市值），预测其在未来一段时间（例如30天后）的涨跌情况**。我们将把这个预测问题转化为一个二分类问题。

## 构建目标向量的步骤

以下是构建目标向量的核心步骤，我们将逐一详细说明。

### 1. 定义预测问题

首先，我们需要明确预测的目标。在我们的策略框架中，假设我们每30天进行一次调仓选股。那么，在调仓日（记为T日），我们需要预测的是：**从T日开始的未来30天内，某只股票的收益率是正还是负**。

![](img/97be2a859714efc72147e9571bac747f_1.png)

*   **如果未来30天收益率为正**，则标记为 **1**（代表“涨”）。
*   **如果未来30天收益率为负**，则标记为 **0**（代表“跌”）。

![](img/97be2a859714efc72147e9571bac747f_3.png)

![](img/97be2a859714efc72147e9571bac747f_5.png)

这样，我们就将连续的收益率预测问题，简化为了一个**二分类**问题。

### 2. 计算未来收益率

![](img/97be2a859714efc72147e9571bac747f_7.png)

要获得“未来30天收益率”这个标签，我们需要从历史数据中计算出来。Pandas库提供了非常便捷的金融计算函数。

**核心概念：`pct_change` 函数**
这个函数用于计算一个序列中相邻元素之间的百分比变化，即收益率。其公式可以表示为：
`收益率_t = (价格_t - 价格_{t-1}) / 价格_{t-1}`

![](img/97be2a859714efc72147e9571bac747f_9.png)

然而，我们的数据包含多只股票，且时间序列是连续的。直接对整个数据框的收盘价序列使用 `pct_change(30)` 计算的是任意相邻30行的价格变化，这可能是不同股票之间的比较，没有意义。

![](img/97be2a859714efc72147e9571bac747f_11.png)

![](img/97be2a859714efc72147e9571bac747f_13.png)

![](img/97be2a859714efc72147e9571bac747f_15.png)

**正确做法：分组计算**
我们必须先按股票代码进行分组，然后在每个股票自己的时间序列内计算收益率。这需要结合 `groupby` 和 `pct_change` 函数。

![](img/97be2a859714efc72147e9571bac747f_17.png)

```python
# 假设 df 是包含‘code’(股票代码)和‘close’(收盘价)列的数据框
# 计算每只股票自身时间序列上，当前价格相对于30天前价格的收益率
df[‘return_30d’] = df.groupby(‘code’)[‘close’].pct_change(30)
```

![](img/97be2a859714efc72147e9571bac747f_19.png)

![](img/97be2a859714efc72147e9571bac747f_21.png)

执行这段代码后，`df` 中会新增一列 `return_30d`，它表示**当天收盘价相对于30天前收盘价的收益率**。

![](img/97be2a859714efc72147e9571bac747f_23.png)

### 3. 将收益率平移为未来值

![](img/97be2a859714efc72147e9571bac747f_25.png)

![](img/97be2a859714efc72147e9571bac747f_27.png)

上一步计算出的 `return_30d` 是“过去30天的收益率”，即站在今天看**已经发生**的收益。但我们的目标是要预测“未来30天的收益率”。

因此，我们需要做一个关键操作：**将收益率列向前（向过去方向）平移30个时间单位**。这样，在T日这一行，`return_30d` 列的值就变成了从T日开始的**未来30天**的收益率。

```python
# 将收益率列向前平移30个时间单位，得到未来收益率
df[‘return_future_30d’] = df.groupby(‘code’)[‘return_30d’].shift(-30)
```

*   `shift(-30)` 函数将序列中的每个值向上移动30行。原来在第T+30行的值（即基于T日价格计算的未来收益率），现在被移到了第T行。
*   经过平移后，`df[‘return_future_30d’]` 列在T日的值，就代表了从T日持有到T+30日的预期收益率。

### 4. 将连续收益率转换为分类标签

现在我们已经有了未来30天的连续收益率 `return_future_30d`。为了进行二分类，需要将其转换为0和1的标签。

我们定义一个简单的转换规则：
*   若 `return_future_30d` > 0，则标签为 **1**（看涨）。
*   否则，标签为 **0**（看跌）。

可以使用Pandas的 `apply` 函数或更高效的向量化操作来实现：

```python
# 方法1：使用apply函数
df[‘target’] = df[‘return_future_30d’].apply(lambda x: 1 if x > 0 else 0)

# 方法2：使用向量化操作 (更高效)
df[‘target’] = (df[‘return_future_30d’] > 0).astype(int)
```

至此，我们成功创建了目标变量 `target`。每一行数据都包含了：
*   **特征**：当日的市盈率(`pe_ratio`)、总市值(`market_cap`)等。
*   **目标**：`target`，表示该股票在未来30天是否上涨（1为涨，0为跌）。

## 总结

本节课中我们一起学习了为股票涨跌预测模型构建目标向量的完整流程。我们首先将问题定义为二分类任务，然后利用 `groupby` 和 `pct_change` 函数计算每只股票的历史收益率，接着通过 `shift` 函数将收益率平移以获得“未来收益率”，最后将连续值转换为0/1分类标签。完成这些步骤后，我们就得到了一个包含特征(`X`)和目标(`y`)的标准数据集，可以无缝接入后续的机器学习模型进行训练和预测。

![](img/97be2a859714efc72147e9571bac747f_29.png)

接下来，我们就可以使用这个准备好的数据集，进行特征工程、模型选择、训练与评估等标准的机器学习流程了。
