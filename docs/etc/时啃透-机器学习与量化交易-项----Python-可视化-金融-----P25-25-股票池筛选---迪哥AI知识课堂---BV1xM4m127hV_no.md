在每日交易开始前（`before_trading`函数中），我们需要执行股票筛选。核心目标是：从沪深300成分股中，查询每只股票的财务数据（例如营业收入），并按照该指标降序排列，选出排名前10的股票。

![](img/0aa36b16e669d1a98c7ed2b8f2f41597_13.png)

以下是实现此功能的关键步骤：

### 1. 查询财务数据

我们需要使用平台提供的API来查询财务指标。所有可用的指标和函数都在官方帮助文档中，使用前应仔细查阅。

例如，要查询基础财务数据中的“营业总收入”指标，可以在文档的“财务数据”部分找到对应的字段。

### 2. 构建查询语句

查询的核心是构建一个`query`对象，它指定了要获取的数据字段、过滤条件、排序方式和数量限制。

```python
def before_trading(context):
    # 构建查询：获取营业总收入指标
    q = query(
        fundamentals.financial_indicator.operating_revenue
    ).filter(
        # 过滤条件：股票代码必须在沪深300成分股中
        fundamentals.financial_indicator.code.in_(context.stock_pool)
    ).order_by(
        # 排序：按营业总收入降序排列
        fundamentals.financial_indicator.operating_revenue.desc()
    ).limit(
        # 限制：只取前10名
        10
    )
```

本节课中我们一起学习了量化交易中股票池筛选的基本流程：
1.  **定义初始股票池**：通常选择一个宽基指数（如沪深300）作为起点。
2.  **构建查询逻辑**：利用平台的`query`函数，结合`filter`（过滤）、`order_by`（排序）和`limit`（限制）来筛选目标股票。
3.  **基于财务指标筛选**：本例中以“营业总收入”作为排序依据，选出了营收最高的10只股票。
4.  **调试与修正**：确保数据查询的代码和股票池的定义准确无误。
