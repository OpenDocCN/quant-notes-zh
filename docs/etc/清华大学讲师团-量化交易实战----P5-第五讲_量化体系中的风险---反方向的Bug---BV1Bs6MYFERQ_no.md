## 课程回顾与核心要素

上一节我们介绍了量化策略研发的迭代过程，本节中我们来看看风险控制如何融入整个体系。

![](img/c195bdb6f3964cc44472117da6bbed00_4.png)

一个完整的量化交易系统包含几个核心要素：股票池（交易池）、择时信号（买入与卖出）、风险控制和头寸管理。

![](img/c195bdb6f3964cc44472117da6bbed00_6.png)

*   **股票池（交易池）**：指你计划交易的所有标的集合，不仅限于股票，也包括期货、期权等。
*   **择时信号**：决定何时买入（进入）和何时卖出（退出）的信号。对于趋势型策略，出场信号往往比入场信号更为关键。
*   **风险控制**：本节课的重点，旨在管理交易中的不确定性，防止灾难性损失。
*   **头寸管理**：决定每次交易投入多少资金，与风险控制紧密相关，将在下节课详述。

许多人在设计策略时过度关注买入信号，但风险控制和头寸管理才是保证策略长期有效的基石。本节课，我们将聚焦于风险控制。

## 第一部分：风险的度量——波动率 📊

要控制风险，首先需要度量风险。最常用的风险度量指标是**波动率**，它衡量了资产价格在一段时间内上下波动的幅度。

**核心概念**：波动率越大，代表价格运动的不确定性越高，即风险越大。风险本身包含向上（盈利）和向下（亏损）两种可能，它衡量的是不确定性，而非单纯的亏损。

以下是两种常用的波动率计算方法：

1.  **标准差**：基于收盘价序列计算，反映了价格偏离其平均值的程度。
    *   **公式**：`σ = sqrt( Σ (Pi - μ)² / N )`，其中 `Pi` 为价格，`μ` 为均价，`N` 为周期数。
2.  **平均真实波幅（ATR）**：考虑了价格跳空的情况，能更全面地反映单根K线内的波动幅度。
    *   **计算步骤**：
        1.  计算单根K线的真实波幅（TR）：
            *   `TR = max( (当日最高价 - 当日最低价), abs(当日最高价 - 昨日收盘价), abs(当日最低价 - 昨日收盘价) )`
        2.  计算N日ATR（例如N=20）：
            *   `ATR = (TR1 + TR2 + ... + TRN) / N`

**波动率在系统各环节的应用**：
*   **股票池**：作为选股条件，筛选波动率符合策略风格（稳健型选低波动，激进型选高波动）的标的。
*   **择时信号**：辅助判断趋势启动（波动率从低位放大）或趋势末端（波动率异常放大）。
*   **风险控制（本节重点）**：用于设定科学的止盈止损阈值。
*   **头寸管理（下节重点）**：根据波动率大小动态分配仓位。

在风控环节，尤其是止盈止损中，使用ATR更为常见。

## 第二部分：常用的风险控制方法 🛡️

![](img/c195bdb6f3964cc44472117da6bbed00_8.png)

理解了如何度量风险后，我们来看看具体有哪些风险控制方法。风险可分为系统性风险（影响整个市场）和非系统性风险（影响个别标的）。

![](img/c195bdb6f3964cc44472117da6bbed00_10.png)

### 应对系统性风险

系统性风险如政策变化、国际关系、黑天鹅事件等，难以预测，但可通过以下方式应对：

*   **主动仓位管理**：当预判或已发生可能延续的系统性风险时，主动降低整体仓位，以控制总资产回撤。
*   **量化手段辅助判断**：使用大盘指数的技术指标（如均线排列、MACD）来判断市场整体趋势，并据此调整策略仓位或启用备用策略。
*   **策略失效监控**：将策略的收益曲线视为K线，应用技术指标（如均线死叉、出现历史最大回撤）来监控策略本身是否可能失效，并及时调整。

![](img/c195bdb6f3964cc44472117da6bbed00_12.png)

### 其他需注意的风险点

![](img/c195bdb6f3964cc44472117da6bbed00_14.png)

![](img/c195bdb6f3964cc44472117da6bbed00_16.png)

*   **流动性风险**：单只标的的持仓市值不宜超过其流通市值的2%；单日成交量不宜超过该标的成交量的10%，以保证能顺利进出。
*   **隔夜与节假日风险**：对于杠杆产品（如期货）或重大节假日前后，应严格控制或清空隔夜头寸，以防跳空开盘导致巨大损失。

### 策略内的核心风控：止盈止损

以上是外部和整体的风控，对于每笔交易内部，最系统的风控是止盈止损模块。以下是几种常见方法：

*   **固定比例法**：设定固定的盈利百分比（如8%）止盈，固定的亏损百分比（如5%）止损。方法简单，但略显主观。
*   **技术信号法**：利用技术指标（如均线死叉、跌破支撑位）作为出场信号。常与入场信号成对使用。
*   **跟踪止盈止损法（推荐）**：这是一种动态且高效的方法，尤其适合趋势跟踪策略。
    *   **逻辑**：从建仓后的最高点开始跟踪，当价格从该最高点回撤一定幅度时出场。
    *   **关键改进**：回撤幅度不再主观设定固定百分比，而是与波动率（如ATR）挂钩。例如，设定为“从最高点回撤2倍ATR时出场”。这意味着，只有当回撤幅度超过了近期正常波动的范围（视为异常波动）时才触发退出，避免被市场噪声“震”出趋势。

**宽进严出**：对于趋势型策略，常采用“宽进严出”的原则。即入场信号可以相对宽松以捕捉更多潜在趋势，而出场信号（含止盈止损）必须精心设计，确保截断亏损，让利润奔跑。

## 第三部分：实例——实现跟踪止盈止损模块 💻

上一节我们介绍了方法，本节中我们来看看如何在代码中实现一个基于ATR的跟踪止盈止损模块，并观察其对策略效果的提升。

我们基于一个已有的策略进行改进。原策略使用MACD底背离买入，顶背离卖出。分析其交易记录发现，仅以顶背离出场有时过晚，导致利润回吐。

**优化思路**：引入跟踪止盈止损模块作为辅助出场条件，以更及时地锁定利润或控制亏损。

**核心代码逻辑**：

1.  **盘前更新**：在每日开盘前，计算并更新两个关键数据：
    *   `最高价_至今`：从建仓日到昨日收盘期间的最高价。
    *   `ATR_建仓点`：根据当前股价复权情况，重新计算建仓点的ATR值（避免因除权等导致数值失效）。
2.  **盘中监控**：在交易时间内，实时执行以下判断：
    *   计算 `当前最高价 = max(最高价_至今， 今日最高价)`。
    *   计算 `止损价 = 当前最高价 - N * ATR_建仓点` （N为参数，如2或3）。
    *   如果 `当前股价 <= 止损价`，则执行卖出操作。

**回测效果对比**：
*   在均线择时策略中加入该模块后，年化收益率从1.61%提升至10.10%，夏普比率由负转正至0.80，最大回撤从36.6%大幅降低至16.7%。
*   在MACD择时策略中加入该模块后，年化收益率从10.18%略降至8.02%，但最大回撤从9.0%显著降低至5.0%，风险收益比（年化收益/最大回撤）得到优化。

可见，加入跟踪止盈止损模块能有效改善策略的风险收益特征。

## 第四部分：策略性能评估与优化方向 📈

增加了基础风控模块后，我们还可以从哪些角度进一步优化风险控制？

一个直接的思路是：在头寸管理上区分对待不同风险的标的。即对于波动率（风险）大的标的，分配更少的资金；对于波动率小的标的，分配更多的资金。这样可以从投资组合层面进一步平滑整体风险。这将是下节课**头寸管理**的核心内容。

此外，对波动率的应用可以更深入：

*   **绝对风险与相对风险**：我们不仅可以在不同标的间横向比较波动率（绝对风险），还可以观察单个标的波动率随时间的变化趋势（相对风险）。
*   **波动率形态**：如果某标的波动率呈现“圆形底”形态（从高位下降至低位再放大），可能预示着底部盘整结束、关注度上升。这既可作为一种风控警示（波动率可能进入放大周期，需谨慎），在特定策略下也可作为潜在的入场信号。

## 第五部分：深入探讨——波动率与风险的关系 🤔

最后，我们深入探讨一下波动率与市场的关系，加深对风险的理解。

![](img/c195bdb6f3964cc44472117da6bbed00_18.png)

![](img/c195bdb6f3964cc44472117da6bbed00_20.png)

*   **为何选择ETF**：对于普通投资者，投资指数ETF（如300ETF）是简单有效的策略。原因之一就是ETF的波动率通常低于个股，风险相对更小，心理冲击也更低，便于长期持有或定投。
*   **波动率与市场阶段**：在A股，波动率与市场走势常呈一定相关性。熊市和牛市末期通常波动率放大；牛市中期和震荡市波动率可能相对稳定或缩小。观察波动率的异常放大，有时可以作为市场情绪过热、趋势可能逆转的警示。
*   **下行风险**：有同学提出，上涨波动不是风险。确实，更精细的风险度量可专注于**下行波动率**，即只计算价格下跌时的波动。这对应着另一个绩效指标——**索提诺比率**，它用下行标准差代替总标准差，更能衡量单位下行风险带来的收益。

**总结**：风险控制是交易中你唯一能牢牢把握的部分。盈利需要市场给予机会（贝塔），而控制亏损、活到牛市来临则依赖于你自己的风控体系（阿尔法）。务必重视风控，将其置于策略设计的重中之重。
