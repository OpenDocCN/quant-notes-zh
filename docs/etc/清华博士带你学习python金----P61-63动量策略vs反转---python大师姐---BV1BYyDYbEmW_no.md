## 核心概念解析

![](img/470fa9fd63959b28eeeaa2418da1c417_3.png)

动量策略基于一个假设：过去一段时间表现优异的股票，在未来一段时间内将继续保持良好的表现。其核心思想是“趋势延续”。

![](img/470fa9fd63959b28eeeaa2418da1c417_5.png)

反转策略则基于相反的假设：过去一段时间表现不佳的股票，在未来一段时间内将发生逆转，表现会变好。其核心思想是“物极必反”。

这两个概念本身不一定是成熟的交易策略，而是市场存在的两种特性。量化研究常常需要探究在特定市场环境下，是动量效应更强还是反转效应更强，这可以为其他复杂策略的改进和优化提供思路。

![](img/470fa9fd63959b28eeeaa2418da1c417_7.png)

## 策略实现思路

以下是两种策略在选股层面的具体实现逻辑：

![](img/470fa9fd63959b28eeeaa2418da1c417_9.png)

动量策略：
1.  计算股票池中所有股票在过去一段时间（例如30天）的收益率。
2.  按照收益率从高到低排序。
3.  买入收益率最高的前N只股票。

反转策略：
1.  计算股票池中所有股票在过去一段时间（例如30天）的收益率。
2.  按照收益率从低到高排序（即表现最差的排在前面）。
3.  买入收益率最低（即表现最差）的前N只股票。

我们将使用一个量化平台（示例中使用了`history`函数）来获取数据并实现策略。请注意，以下代码是一个逻辑示例，实际运行需要相应的数据接口支持。

    # 设定基准为沪深300指数
    set_benchmark('000300.XSHG')
    # 设定持仓股票数量
    context.N = 10
    # 设定股票池（例如沪深300成分股）
    context.stocks = get_index_stocks('000300.XSHG')

# 每月调仓函数
def rebalance(context):
    # 获取股票池中所有股票过去30天的收盘价数据
    # history函数用于获取多只股票的历史数据
    df_close = history(30, field='close', security_list=context.stocks, df=True)
    # 转置DataFrame，使行索引为股票代码，列索引为日期
    df_close = df_close.T

    # 计算每只股票过去30天的收益率
    # 收益率 = (期末价格 - 期初价格) / 期初价格
    df_close['return'] = (df_close.iloc[:, -1] - df_close.iloc[:, 0]) / df_close.iloc[:, 0]

    # --- 动量策略选股部分 ---
    # 按收益率降序排列（收益率高的在前）
    df_sorted = df_close.sort_values(by='return', ascending=False)

**代码关键点说明**：
*   `history(30, field='close', ...)`：一次性获取股票列表中所有股票过去30天的收盘价，返回一个`DataFrame`。
*   `.T`：对`DataFrame`进行转置操作，便于按股票进行数据分析。
*   `df_close['return'] = (df_close.iloc[:, -1] - df_close.iloc[:, 0]) / df_close.iloc[:, 0]`：向量化计算每只股票的收益率。`iloc[:, -1]`获取最后一列（最近收盘价），`iloc[:, 0]`获取第一列（30天前收盘价）。
*   `sort_values(by='return', ascending=False)`：按`return`列排序，`ascending=False`为降序（动量策略），`ascending=True`为升序（反转策略）。

## 策略表现与思考

根据示例回测结果（在特定时间段和沪深300股票池中）：
*   **动量策略**的累计收益率为负（约-24%），跑输大盘。
*   **反转策略**的累计收益率为正，表现优于动量策略。

由此，我们可以得到一个初步的观察结论：在示例所测试的A股市场（沪深300）环境下，**反转效应可能强于动量效应**。这意味着“买入过去表现差的股票”的简单思路，在回测期内比“买入过去表现好的股票”表现更佳。

**重要提示**：
1.  这个结论具有极强的**时间局限性和市场局限性**。更换测试时间段或股票池（如全A股、美股），结果可能完全不同。
2.  动量策略和反转策略是非常基础的单因子策略，**不建议直接作为实盘交易策略**。它们更多用于市场特性研究和作为复杂策略的组成部分（例如，下一节将介绍的“羊驼交易法则”就融合了反转思想）。
3.  金融市场上存在大量关于“神奇反转策略”的讨论，但投资者需保持谨慎。任何简单策略的长期有效性都值得怀疑，成功的投资通常需要更严谨、更系统的模型。
