## 策略评估指标简介
当我们完成一个交易策略的构建后，需要在历史数据中进行测试，以判断策略的盈亏情况。然而，仅用“赚”或“赔”来描述是不够的。我们需要更精确的指标来回答以下问题：具体赚了多少？相比市场基准有何优势？策略承担了哪些风险？与其他策略相比有何特点？这些问题的答案，就是策略评估指标。

## 回测收益率
回测收益率是最基本、最常用的策略评估指标。它衡量了策略在整个回测周期内，相对于初始本金所获得的总收益比例。

![](img/06050d450941c78f11f457fe47007c16_13.png)

![](img/06050d450941c78f11f457fe47007c16_15.png)

**公式**如下：
`回测收益率 = (期末资产总值 - 期初本金) / 期初本金`

![](img/06050d450941c78f11f457fe47007c16_17.png)

![](img/06050d450941c78f11f457fe47007c16_19.png)

![](img/06050d450941c78f11f457fe47007c16_21.png)

也可以表示为：
`回测收益率 = (Pt / P0) - 1`
其中，`P0`代表期初买入资产时的价格（对应本金），`Pt`代表期末卖出资产时的价格（对应资产总值）。

这个指标直观地告诉我们，经过策略的一番操作后，我们的本金最终变成了原来的多少倍（减去1后就是增长率）。

![](img/06050d450941c78f11f457fe47007c16_23.png)

接下来，我们通过代码来实际计算并可视化回测收益率。

![](img/06050d450941c78f11f457fe47007c16_25.png)

## 代码实战：计算与可视化回测收益率
首先，我们导入必要的工具包并加载示例数据。

![](img/06050d450941c78f11f457fe47007c16_27.png)

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便绘图显示
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，指定日期列为索引
data = pd.read_csv('your_stock_data.csv', index_col=0)
print(data.head())
```

假设`data[‘close’]`列代表我们策略在每一天运行后的资产净值（或直接使用股价模拟）。我们将基于此计算每日相对于第一天的净值变化。

```python
# 计算每日净值相对于期初净值的比值
cumulative_return = data['close'] / data['close'].iloc[0]

![](img/06050d450941c78f11f457fe47007c16_29.png)

![](img/06050d450941c78f11f457fe47007c16_31.png)

# 绘制净值曲线
plt.figure(figsize=(10, 6))
cumulative_return.plot()
plt.title('策略累计净值走势')
plt.ylabel('净值倍数 (相对于期初)')
plt.xlabel('日期')
plt.grid(True)
plt.show()
```

![](img/06050d450941c78f11f457fe47007c16_33.png)

![](img/06050d450941c78f11f457fe47007c16_35.png)

上图展示了资产净值随时间增长（或下跌）的过程。现在，我们来计算整个回测周期的总收益率。

![](img/06050d450941c78f11f457fe47007c16_37.png)

![](img/06050d450941c78f11f457fe47007c16_39.png)

```python
# 计算总回测收益率： (最终净值 / 期初净值) - 1
total_return = cumulative_return.iloc[-1] - 1

![](img/06050d450941c78f11f457fe47007c16_41.png)

# 将结果整理成DataFrame以便展示
result_df = pd.DataFrame({
    ‘回测收益率’: [total_return]
}, index=[‘策略总收益’])
print(result_df)
```

![](img/06050d450941c78f11f457fe47007c16_43.png)

![](img/06050d450941c78f11f457fe47007c16_45.png)

执行以上代码，我们将得到一个具体的百分比数值，这就是该策略在回测期间的总收益率。
