## 布尔型索引的原理

![](img/36a08d320ce1ba322f2450c8c70dd099_12.png)

![](img/36a08d320ce1ba322f2450c8c70dd099_14.png)

布尔型索引的原理可以分为两步：

![](img/36a08d320ce1ba322f2450c8c70dd099_16.png)

1.  **数组与标量的比较运算**：首先，将一个数组与一个值（标量）进行比较（例如 `A > 5`）。这个操作会返回一个**布尔数组**，其长度与原数组相同，每个位置的值表示对应元素是否满足条件（`True` 或 `False`）。
2.  **使用布尔数组进行索引**：然后，将这个布尔数组放入原数组的方括号 `[]` 中。NumPy会自动选取所有在布尔数组中对应位置为 `True` 的元素，并返回一个新的数组。

![](img/36a08d320ce1ba322f2450c8c70dd099_18.png)

**核心公式**：
```python
# 筛选出数组arr中所有大于5的元素
result = arr[arr > 5]
```

## 从例子开始：筛选大于5的数

![](img/36a08d320ce1ba322f2450c8c70dd099_24.png)

为了理解布尔型索引的强大，我们先看一个用Python基础方法实现的例子。

![](img/36a08d320ce1ba322f2450c8c70dd099_26.png)

以下是使用Python列表和`filter`函数的方法：

![](img/36a08d320ce1ba322f2450c8c70dd099_28.png)

```python
# 创建一个列表
a = [8, 3, 6, 2, 9, 1]
# 使用filter和lambda函数筛选大于5的数
b = filter(lambda x: x > 5, a)
# 将结果转换为列表
list(b)  # 输出：[8, 6, 9]
```

![](img/36a08d320ce1ba322f2450c8c70dd099_30.png)

这个方法虽然可行，但代码略显繁琐。现在，我们看看用NumPy如何实现：

![](img/36a08d320ce1ba322f2450c8c70dd099_31.png)

![](img/36a08d320ce1ba322f2450c8c70dd099_33.png)

```python
import numpy as np
# 将列表转换为NumPy数组
A = np.array([8, 3, 6, 2, 9, 1])
# 使用布尔型索引，一行代码完成筛选
result = A[A > 5]
print(result)  # 输出：[8 6 9]
```

![](img/36a08d320ce1ba322f2450c8c70dd099_35.png)

![](img/36a08d320ce1ba322f2450c8c70dd099_37.png)

可以看到，`A[A > 5]` 这行代码非常直观和简洁，它直接表达了“从数组A中取出所有大于5的元素”这个意图。

## 深入理解原理

![](img/36a08d320ce1ba322f2450c8c70dd099_43.png)

![](img/36a08d320ce1ba322f2450c8c70dd099_45.png)

为了更清晰地理解 `A[A > 5]` 是如何工作的，我们可以将其拆解：

![](img/36a08d320ce1ba322f2450c8c70dd099_47.png)

1.  **第一步：生成布尔掩码**
    `A > 5` 这个表达式会对数组`A`中的每个元素执行比较运算。
    ```python
    mask = A > 5
    print(mask)  # 输出：[ True False  True False  True False]
    ```
    它返回了一个布尔数组`mask`，其中`True`表示对应位置的元素大于5。

![](img/36a08d320ce1ba322f2450c8c70dd099_49.png)

2.  **第二步：应用布尔索引**
    将上一步得到的布尔数组`mask`放入`A[]`中。
    ```python
    result = A[mask]
    print(result)  # 输出：[8 6 9]
    ```
    NumPy会取出`mask`中所有`True`位置对应的`A`中的元素。

![](img/36a08d320ce1ba322f2450c8c70dd099_51.png)

手动演示这个原理：
```python
arr = np.array([10, 20, 30, 40])
# 手动创建一个布尔数组，选择第0个和第3个元素
bool_mask = np.array([True, False, False, True])
print(arr[bool_mask])  # 输出：[10 40]
```

## 组合条件筛选

布尔型索引的强大之处在于可以方便地组合多个筛选条件。

![](img/36a08d320ce1ba322f2450c8c70dd099_60.png)

![](img/36a08d320ce1ba322f2450c8c70dd099_62.png)

### 案例一：筛选“大于5的偶数”
这需要同时满足两个条件（“与”关系）。

**错误尝试**：直接链式索引
```python
# 这是错误的！因为 A[A>5] 的结果数组长度变了，无法再与原始A的长度匹配
A[A>5][A%2==0]  # 会报错或得到错误结果
```

**正确方法**：使用位运算符 `&` (与)
我们需要将两个条件生成的布尔数组进行“与”运算。注意，每个条件必须用括号括起来。
```python
# 筛选大于5的偶数
result = A[(A > 5) & (A % 2 == 0)]
print(result)  # 输出：[6]
```
**重要提示**：在NumPy中组合布尔条件时，必须使用位运算符 `&` (与)、`|` (或)、`~` (非)，而不是Python的关键字 `and`、`or`、`not`，并且每个条件都需要用括号括起来。

### 案例二：筛选“大于5的数 或 偶数”
这满足任意一个条件即可（“或”关系）。

![](img/36a08d320ce1ba322f2450c8c70dd099_64.png)

使用位运算符 `|` (或)：
```python
# 筛选大于5的数 或者 是偶数
result = A[(A > 5) | (A % 2 == 0)]
print(result)  # 输出：[8 6 2 9]
```
在这个结果中，`8`和`6`满足“大于5”，`2`是偶数（但不大于5），`9`满足“大于5”。

## 总结

![](img/36a08d320ce1ba322f2450c8c70dd099_66.png)

本节课我们一起学习了NumPy中**布尔型索引**的核心用法：

![](img/36a08d320ce1ba322f2450c8c70dd099_68.png)

1.  **核心操作**：`array[condition]`，例如 `A[A > 5]`，可以极其高效地基于条件筛选数组元素。
2.  **工作原理**：分为两步——先通过比较运算生成布尔掩码数组，再用该掩码对原数组进行索引。
3.  **组合条件**：使用位运算符 `&` (与)、`|` (或) 来组合多个筛选条件，**切记每个独立条件需用括号括起来**。
4.  **优势**：相比Python基础的循环或`filter`函数，布尔型索引语法更简洁、可读性更强，并且由于是NumPy的向量化操作，执行效率也高得多。
