## 线性分类器基础

上一节我们介绍了机器学习的基本概念，本节中我们来看看分类任务中最基础的工具——线性分类器。

所谓分类器，其目标是根据用户定义的类别，将输入空间 **X** 划分成互不相交的区域。例如，在一个简单的二分类问题中，我们可以用一个平面将空间中的红色点（第一类）和蓝色点（第二类）分隔开。这些区域的边界被称为**决策面**。

根据预测函数形式的不同，决策面可能是线性的平面，也可能是二次的抛物线，甚至是更高维度的复杂曲面。其中，参数为线性的决策面所对应的分类器，就称为**线性分类器**。

## 逻辑回归原理

虽然名字中带有“回归”，但逻辑回归实际上是一种分类方法。我们先研究二分类问题，其原理可以自然地扩展到多分类场景。

逻辑回归的流程与线性回归相似，可以概括为以下三个步骤：

![](img/790449751cc1c886022660130542a5c8_5.png)

1.  **寻找预测函数**：我们需要找到一个合适的预测函数，称为 **hypothesis**，通常记作 **h**。在逻辑回归中，这个函数代表了给定特征 **x** 后，样本属于某个类别的**后验概率**。逻辑回归选择了 **Sigmoid 函数** 作为其预测函数。
2.  **构造损失函数**：我们需要一个函数来衡量预测输出与真实标签之间的偏差，这个函数称为**损失函数**或**代价函数**。在线性回归中，我们使用均方误差；在逻辑回归中，我们使用**负的对数似然函数**。
3.  **参数最优化**：我们的目标是找到一组参数，使得损失函数的值最小。在逻辑回归中，通常使用**梯度下降法**来迭代优化参数 **θ**，直到收敛。

接下来，让我们深入探讨这三个步骤的具体实现。

---

### 1. 预测函数：Sigmoid函数

首先，回顾线性回归的公式。假设线性表达式为 **Z**：
```
Z = θ₀ + θ₁X₁ + ... + θₙXₙ = θᵀX
```
其中，向量 **X** 的第一列通常为1，以包含截距项 θ₀。

对于逻辑回归，我们希望将输出映射为概率（介于0和1之间），因此选择了 **Sigmoid 函数** 作为预测函数 **h(x)**：
```
h(x) = g(θᵀX) = 1 / (1 + e^{-θᵀX})
```
其中，`g(z)` 就是Sigmoid函数：`g(z) = 1 / (1 + e^{-z})`。

Sigmoid函数的图像呈S形，其输出值严格在0到1之间，中点恰好是0.5。这个特性使其非常适合表示概率。**h(x)** 可以解释为：在给定特征 **x** 的条件下，样本属于正类（例如类别1）的后验概率。

**预测规则**：
- 如果 `h(x) >= 0.5`，则预测样本属于正类。
- 如果 `h(x) < 0.5`，则预测样本属于负类。

### 2. 损失函数：最大似然估计

有了预测函数，我们需要通过训练数据来估计参数 **θ**。这里采用**最大似然估计**的方法。

对于二分类问题，样本的标签 `y` 服从伯努利分布。根据预测函数 `h(x)` 的定义，我们有：
- 样本属于正类（y=1）的概率为：`P(y=1 | x; θ) = h(x)`
- 样本属于负类（y=0）的概率为：`P(y=0 | x; θ) = 1 - h(x)`

可以将这两个公式合并为一个表达式：
```
P(y | x; θ) = (h(x))^y * (1 - h(x))^(1-y)
```

![](img/790449751cc1c886022660130542a5c8_9.png)

假设我们有 `m` 个独立同分布的样本，其联合概率（似然函数）为所有样本概率的乘积：
```
L(θ) = ∏_{i=1}^{m} P(y^{(i)} | x^{(i)}; θ) = ∏_{i=1}^{m} (h(x^{(i)}))^{y^{(i)}} * (1 - h(x^{(i)}))^{1-y^{(i)}}
```

为了简化计算，我们通常取其对数，得到**对数似然函数**：
```
l(θ) = log L(θ) = ∑_{i=1}^{m} [ y^{(i)} log(h(x^{(i)})) + (1-y^{(i)}) log(1 - h(x^{(i)})) ]
```

![](img/790449751cc1c886022660130542a5c8_11.png)

![](img/790449751cc1c886022660130542a5c8_13.png)

我们的目标是最大化这个对数似然函数 `l(θ)`。为了将其转化为一个最小化问题，我们定义**损失函数 J(θ)** 为负的平均对数似然：
```
J(θ) = - (1/m) * l(θ) = - (1/m) * ∑_{i=1}^{m} [ y^{(i)} log(h(x^{(i)})) + (1-y^{(i)}) log(1 - h(x^{(i)})) ]
```
这样，**最小化 J(θ) 就等价于最大化似然函数 L(θ)**。

---

### 3. 参数优化：梯度下降法

![](img/790449751cc1c886022660130542a5c8_15.png)

现在，我们需要找到使损失函数 `J(θ)` 最小的参数 **θ**。由于 `J(θ)` 是凸函数，我们可以使用**梯度下降法**。

![](img/790449751cc1c886022660130542a5c8_17.png)

![](img/790449751cc1c886022660130542a5c8_19.png)

梯度下降法的核心是沿着损失函数梯度的反方向更新参数。首先，需要计算损失函数 `J(θ)` 关于每个参数 `θ_j` 的偏导数（梯度）。

经过推导（应用链式法则），可以得到梯度的简洁表达式：
```
∂J(θ)/∂θ_j = (1/m) * ∑_{i=1}^{m} ( h(x^{(i)}) - y^{(i)} ) * x_j^{(i)}
```

这个结果非常直观：梯度是**所有样本的预测误差 `(h(x) - y)` 乘以对应特征 `x_j` 后的平均值**。

![](img/790449751cc1c886022660130542a5c8_21.png)

得到梯度后，我们就可以进行参数更新。以下是梯度下降的迭代公式：
```
重复直到收敛 {
    θ_j := θ_j - α * (∂J(θ)/∂θ_j)   (对所有参数 j 同时更新)
}
```
其中：
- **α** 是**学习率**，一个重要的超参数，控制着每次更新的步长，通常设置在0到1之间（例如0.01, 0.1, 0.5）。
- 学习率的选择至关重要：过大可能导致无法收敛，过小则收敛速度慢。在实践中，常通过**交叉验证**来选择最优的α值。

## 总结

本节课中，我们一起学习了机器学习中经典的分类算法——逻辑回归。

我们首先理解了**线性分类器**和**决策面**的基本概念。然后，我们深入剖析了逻辑回归的三个核心步骤：
1.  使用 **Sigmoid 函数** 作为预测函数，将线性组合 `θᵀX` 的输出映射为概率。
2.  基于**最大似然估计**原理，构造了用于二分类的**对数似然损失函数**。
3.  介绍了使用**梯度下降法**来优化损失函数、求解模型参数 **θ** 的过程，并给出了参数更新的具体公式。
