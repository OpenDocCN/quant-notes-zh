# 从零开始卷量化系统：02：数据获取模块 📊

在本节课中，我们将学习如何构建量化系统的第一个核心模块——数据获取模块。我们将从引入必要的库开始，逐步完成登录数据接口、获取交易日历以及下载股票日线数据的过程。整个过程将采用封装函数的方式，以提高代码的复用性和效率。

## 第一步：引入必要的库 📦

首先，我们需要引入所有后续步骤中会用到的Python库。这包括数据分析库、数据接口库、绘图库以及时间处理库。

以下是需要引入的库列表：

*   `tushare`：用于获取金融数据的Python接口。
*   `pandas`：用于数据处理和分析的核心库。
*   `matplotlib`：用于数据可视化和绘图。
*   `datetime`：用于处理日期和时间。

## 第二步：登录数据接口 🔑

上一节我们引入了必要的库，本节中我们来看看如何登录数据接口以获取权限。要使用Tushare的数据API，你需要先注册并获取个人专属的token。

以下是获取和设置token的步骤：

1.  访问Tushare官网进行注册。
2.  登录后，在个人主页中找到你的`token`（一串类似密码的字符串）。
3.  复制你的`token`，并在代码中替换掉`‘your_token’`这个占位符。

完成这一步后，你就可以通过Tushare的在线数据库来调用所需的数据了。我们将这个登录步骤封装在一个函数里，以便复用。

```python
import tushare as ts
# 请将 ‘your_token’ 替换为你自己的token
ts.set_token(‘your_token’)
pro = ts.pro_api()
```
