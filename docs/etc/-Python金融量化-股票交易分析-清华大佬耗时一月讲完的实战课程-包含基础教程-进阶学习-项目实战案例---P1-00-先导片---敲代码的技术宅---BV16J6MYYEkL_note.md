# Python金融量化与股票交易分析：00：课程先导与概述

在本节课中，我们将对《Python金融量化与股票交易分析》课程进行整体介绍。本课程由浅入深，旨在帮助初学者掌握使用Python进行金融市场数据分析与量化策略构建的核心技能。

## 课程内容概览

本课程结构清晰，分为三个主要阶段，确保学习者能够循序渐进地掌握知识。

以下是课程涵盖的核心模块：

1.  **Python与量化基础**：学习Python编程语言基础、数据处理库以及金融市场基本概念。
2.  **数据分析与策略进阶**：深入讲解金融时间序列分析、技术指标计算与回测框架。
3.  **实战项目案例**：通过完整的股票分析、因子模型构建等实战项目，巩固所学知识。

## 学习目标与收获

完成本课程后，你将能够独立完成以下任务：

*   使用Python获取和处理金融数据。
*   计算并可视化常用的技术指标，例如移动平均线（MA）。
*   构建并回测简单的量化交易策略。
*   理解量化投资的基本流程和风险管理概念。

## 核心工具与库简介

本课程将主要使用以下Python库，它们是金融量化领域的标准工具。

*   **pandas**：用于高效的数据处理与分析。
*   **numpy**：提供强大的数值计算功能。
*   **matplotlib / seaborn**：用于创建数据可视化图表。
*   **TA-Lib**：专门用于计算技术分析指标。
*   **Backtrader / Zipline**：流行的量化策略回测框架。
