## 概述

上一节我们完成了数据的初步处理。本节中，我们来看看如何将数据转换为 `alphalens` 包所需的特定格式。该格式要求数据以日期和股票代码作为多级索引，这与我们当前的数据结构不同。

## 理解目标格式

`alphalens` 包要求的数据格式与我们当前的数据结构不同。它需要数据按照以下方式组织：

![](img/323b5dfb81196b1080682683bf6eb2bc_3.png)

*   **第一级索引**：日期（Date）。
*   **第二级索引**：股票代码或股票名称。
*   **数据列**：具体的因子指标值。

![](img/323b5dfb81196b1080682683bf6eb2bc_5.png)

其结构类似于一个多级索引的 `Series`，直观表示如下：

![](img/323b5dfb81196b1080682683bf6eb2bc_7.png)

```python
# 目标数据结构示意（非实际代码）
日期        股票
2019-01-01  A股票     1.3
            B股票     2.1
            C股票     0.8
2019-01-02  A股票     1.5
            B股票     2.0
            C股票     0.9
```

![](img/323b5dfb81196b1080682683bf6eb2bc_9.png)

而我们的数据目前可能是以日期为行、股票为列的 `DataFrame`。因此，我们需要进行格式转换。

## 执行格式转换

为了满足工具包的要求，我们必须将数据转换为上述格式。以下是转换的核心步骤。

![](img/323b5dfb81196b1080682683bf6eb2bc_11.png)

首先，我们需要重新设置数据的索引。我们将使用 `pandas` 的 `set_index` 方法，并指定多级索引。

```python
# 假设 df 是原始的 DataFrame，包含 ‘date‘, ‘stock_code‘, ‘factor_value‘ 等列
# 转换后，索引变为 (日期, 股票代码)，值列为因子值
formatted_series = df.set_index([‘date‘, ‘stock_code‘])[‘factor_value‘]
```

![](img/323b5dfb81196b1080682683bf6eb2bc_13.png)

让我们详细解析这段代码：
*   `df.set_index([‘date‘, ‘stock_code‘])`：将 ‘date‘ 和 ‘stock_code‘ 这两列设置为新的多级索引。
*   `[‘factor_value‘]`：从结果中只选取我们关心的因子值列，最终得到一个 `Series`。
*   `formatted_series`：这就是转换后符合要求的 `Series` 对象。

![](img/323b5dfb81196b1080682683bf6eb2bc_15.png)

执行此操作后，我们可以使用 `.head()` 方法查看结果，确认其格式是否符合预期：第一级是日期，第二级是股票代码，对应的值是因子指标。

![](img/323b5dfb81196b1080682683bf6eb2bc_17.png)

## 数据预处理

![](img/323b5dfb81196b1080682683bf6eb2bc_19.png)

在将数据输入分析模型前，通常还需要进行预处理，以消除异常值并统一量纲。这与我们之前课程中介绍的方法一致。

以下是两个关键的预处理步骤：

![](img/323b5dfb81196b1080682683bf6eb2bc_21.png)

1.  **去极值（Winsorization）**：将因子值中超出指定分位数（如上1%分位和下99%分位）的极端值替换为边界值，以防止异常点对分析产生过大影响。
    ```python
    # 去极值函数示例
    def winsorize(series):
        q_low = series.quantile(0.01)
        q_high = series.quantile(0.99)
        return series.clip(lower=q_low, upper=q_high)
    ```

![](img/323b5dfb81196b1080682683bf6eb2bc_23.png)

2.  **标准化（Standardization）**：将因子值转换为均值为0、标准差为1的标准正态分布，使得不同因子之间具有可比性。
    ```python
    # 标准化公式
    standardized_value = (原始值 - 均值) / 标准差
    ```
    在Pandas中，这可以通过 `(series - series.mean()) / series.std()` 轻松实现。

![](img/323b5dfb81196b1080682683bf6eb2bc_25.png)

在实际操作中，我们需要对**每一天**的横截面数据分别进行去极值和标准化处理。处理完成后，可以通过绘制直方图来观察因子值的分布是否变得更加规范。
