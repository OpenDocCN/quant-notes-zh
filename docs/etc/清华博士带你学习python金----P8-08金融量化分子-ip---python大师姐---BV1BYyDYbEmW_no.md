在本节课中，我们将学习IPython中一个非常实用且有趣的高级功能——魔术命令。魔术命令以百分号开头，能够极大地提升我们在交互式环境中的工作效率，例如直接运行外部脚本、粘贴执行代码以及精确测量代码运行时间等。

## 魔术命令简介

上一节我们介绍了IPython的基本交互功能。本节中，我们来看看IPython提供的“魔术命令”。魔术命令是以 `%` 符号开头的一系列特殊指令，它们为交互式编程环境增添了强大的工具。

## 运行外部Python脚本

![](img/6909ed4a1d105d0a04f886eaa18485e1_3.png)

在标准的Python命令行中，若想运行一个外部Python文件，通常需要退出命令行，再使用 `python filename.py` 命令。在IPython中，我们无需退出。

![](img/6909ed4a1d105d0a04f886eaa18485e1_5.png)

我们可以使用 `%run` 魔术命令来直接运行一个Python脚本文件。

**代码示例：**
```python
%run hello_world.py
```
执行上述命令后，`hello_world.py` 文件中的代码将在当前IPython会话中运行，其定义的变量和函数也会被导入到当前命名空间。

## 粘贴并执行代码

![](img/6909ed4a1d105d0a04f886eaa18485e1_7.png)

有时，我们可能从编辑器或其他地方复制了一段代码，希望在IPython中快速测试。除了直接粘贴，IPython还提供了 `%paste` 魔术命令。

![](img/6909ed4a1d105d0a04f886eaa18485e1_9.png)

![](img/6909ed4a1d105d0a04f886eaa18485e1_11.png)

以下是 `%paste` 命令的使用方法：
1.  首先，将你想要执行的代码复制到剪贴板。
2.  在IPython中输入 `%paste` 并回车。
3.  IPython会先打印出剪贴板中的代码内容，然后用一个分隔符隔开，最后执行这段代码。

![](img/6909ed4a1d105d0a04f886eaa18485e1_13.png)

这个命令特别适合执行从其他地方复制的、可能包含缩进或特殊格式的代码块，能避免直接粘贴时可能出现的格式错误。

## 精确测量代码运行时间

在性能分析和优化时，我们经常需要测量某段代码的执行时间。虽然可以使用Python的 `time` 模块，但对于运行时间极短的代码，单次测量可能不准确（例如结果为0秒）。

IPython的 `%timeit` 魔术命令可以解决这个问题。它会自动多次运行指定的代码，并计算平均运行时间，从而得到更精确的结果。

**公式/概念：**
`%timeit` 的工作原理是：**多次执行 -> 统计耗时 -> 计算平均值与标准差**。

**代码示例：**
```python
import random
# 创建一个包含10000个随机数的列表
my_list = [random.random() for _ in range(10000)]

# 测量排序这个列表所需的时间
%timeit sorted(my_list)
```
执行后，你会看到类似 `146 µs ± 742 ns per loop` 的输出。这表示平均每次循环耗时146微秒，正负误差742纳秒。`%timeit` 会自动决定运行次数（例如“7次运行，每次100万循环”），以确保测量结果的可靠性。这对于优化频繁调用的函数或代码片段非常有帮助。
