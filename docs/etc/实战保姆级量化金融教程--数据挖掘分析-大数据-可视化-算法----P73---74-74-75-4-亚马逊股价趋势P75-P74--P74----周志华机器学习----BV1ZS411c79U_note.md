在获得数据后，我们构建一个基础的Prophet模型进行预测。这个模型旨在捕捉数据的整体趋势，而非精确拟合每一个波动点，以避免过拟合。

![](img/97b48c0398f7541f915075e66c5c1909_41.png)

以下是构建和训练模型的代码框架：
```python
# 初始化模型
model = Prophet()
# 拟合历史数据
model.fit(df)
# 创建未来日期框架（例如未来90天）
future = model.make_future_dataframe(periods=90)
# 进行预测
forecast = model.predict(future)
```

模型预测结果（通常用绿线表示）通常比真实股价数据（通常用黑点表示）更加平滑和平稳。这种保守的预测策略有助于降低异常波动点对整体趋势判断的干扰。

## 模型性能评估

在得到预测结果后，我们需要系统地评估模型的性能。评估主要从两个维度进行：预测数值的准确性和预测趋势方向的正确性。

![](img/97b48c0398f7541f915075e66c5c1909_49.png)

以下是评估模型的核心步骤：

![](img/97b48c0398f7541f915075e66c5c1909_51.png)

![](img/97b48c0398f7541f915075e66c5c1909_53.png)

1.  **划分数据集**：将历史数据划分为训练集和测试集。例如，使用2014年至2017年的数据作为训练集，2017年至2018年的数据作为测试集。
2.  **计算预测误差**：在测试集上，计算模型预测值与真实值之间的绝对误差或平均绝对误差（MAE）。这反映了模型预测价格的精确度。
    *   训练集平均误差：反映模型对历史数据的拟合程度。
    *   测试集平均误差：反映模型对未来未知数据的预测能力。
3.  **评估趋势预测准确率**：分析模型是否能正确预测股价的上涨或下跌趋势。
    *   **计算每日变化**：通过比较相邻日期的预测值，得到预测的每日涨跌方向。
    *   **与真实趋势对比**：将预测的涨跌方向与真实的涨跌方向进行对比。
    *   **计算准确率**：分别统计上涨日和下跌日被正确预测的百分比。例如，模型可能正确预测了56%的上涨日和43%的下跌日。

![](img/97b48c0398f7541f915075e66c5c1909_55.png)

此外，还可以计算预测值的置信区间，以了解预测的不确定性范围。
