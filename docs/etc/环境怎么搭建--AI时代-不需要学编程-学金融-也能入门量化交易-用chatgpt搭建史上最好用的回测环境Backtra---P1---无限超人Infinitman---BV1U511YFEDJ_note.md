## 环境与工具准备

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_11.png)

上一节我们介绍了课程目标，本节中我们来看看搭建回测环境需要做哪些准备工作。

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_13.png)

首先，你需要准备以下两样核心工具：
1.  **Python环境**：Backtrader是一个Python库，因此需要在你的电脑上安装Python。具体的安装步骤在网上有大量教程，此处不再赘述。
2.  **Backtrader库**：这是我们将要使用的核心回测框架。它功能强大，支持股票、期货、比特币等多种资产的回测。由于是本地库，所有数据和策略都在你的电脑上运行，安全性高。

以下是安装Backtrader的具体步骤：
打开命令行工具（如CMD或终端），输入以下指令进行安装：
```bash
pip install backtrader
```
安装成功后，命令行会显示类似“Requirement already satisfied”的提示。

## 项目结构与数据获取

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_19.png)

环境搭建好后，我们需要组织项目文件并获取历史数据用于回测。

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_21.png)

首先，在电脑上创建一个新的文件夹作为项目根目录，例如命名为`MyBacktest`。在该文件夹内，再创建一个名为`data`的子文件夹，专门用于存放股票历史数据文件。

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_23.png)

接着，我们需要获取历史价格数据。一个常用且免费的数据来源是雅虎财经（Yahoo Finance）。
1.  访问雅虎财经网站。
2.  搜索你感兴趣的股票代码（例如：AMD）。
3.  找到历史数据（Historical Data）页面。
4.  选择时间范围（例如：最大范围Max），然后点击下载（Download），数据通常会以CSV格式保存。
5.  将下载好的CSV文件（如`AMD.csv`）放入之前创建的`data`文件夹中。

至此，项目在文件路径上的基础框架就搭建完成了。你可以使用任何文本编辑器或代码编辑器（如VS Code）打开`MyBacktest`文件夹进行后续操作。

## 利用ChatGPT生成回测代码

现在，我们将进入核心环节：使用ChatGPT帮助我们生成可运行的Backtrader回测代码。无论是免费的GPT-3.5/4o模型，还是付费的GPT-4，都能很好地完成此任务。

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_29.png)

与ChatGPT沟通时，指令清晰是关键。你可以这样开始：

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_31.png)

> “你好，请帮我使用Backtrader库搭建一个量化交易回测程序的基本框架。我的股票历史数据是AMD，数据文件是CSV格式，存放在项目文件夹下的`data`子目录里，文件名为`AMD.csv`。请为我生成一个完整的、可运行的Python代码，并应用一个简单的移动平均线交叉策略。”

ChatGPT在理解你的需求后，会生成包含以下几个核心模块的代码：
*   **数据加载模块**：从指定路径读取`AMD.csv`文件。
*   **策略类**：定义一个基于移动平均线（例如，短期均线上穿长期均线时买入，下穿时卖出）的交易逻辑。
*   **回测引擎设置**：初始化Backtrader的回测核心，并设置初始资金等参数。
*   **可视化输出**：添加代码以在回测结束后绘制资产曲线和交易信号图。

生成代码后，将其完整复制到你项目根目录下新建的Python文件中（例如`backtest.py`）。

## 运行回测与自定义优化

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_33.png)

代码生成后，我们即可运行第一次回测。

在终端中，进入你的项目目录`MyBacktest`，运行以下命令：
```bash
python backtest.py
```
程序将开始回测，并在结束后显示收益曲线图。图中绿色向上箭头代表买入信号，红色向下箭头代表卖出信号。你可以直观地看到策略在历史数据上的表现，例如总收益率、回撤等。

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_35.png)

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_37.png)

为了使回测结果更贴近真实交易，我们通常需要加入交易成本（佣金）。我们可以再次求助ChatGPT进行优化。

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_39.png)

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_41.png)

> “请在我刚才的Backtrader代码中加入佣金模块。我的券商佣金规则是：每股交易佣金为0.005美元，每笔交易最低收费1美元。请修改代码，让回测计算时计入这些费用。”

将ChatGPT返回的修改后的代码更新到你的`backtest.py`文件中，再次运行回测。你会发现，在计入佣金后，策略的最终收益可能会发生显著变化，这能帮助你更真实地评估策略的可行性。

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_43.png)

## 关键要点与总结

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_45.png)

本节课中我们一起学习了如何借助AI工具跨越编程门槛，快速搭建属于自己的量化交易回测系统。我们来总结几个关键要点：

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_47.png)

1.  **清晰表述**：与ChatGPT交互时，对文件路径、数据格式、策略想法和佣金规则等细节的描述务必准确清晰，这是获得有效代码的前提。
2.  **本地安全**：使用Backtrader在本地进行回测，确保了你的交易策略和私有数据不会上传到互联网，安全性高。
3.  **迭代优化**：回测是一个迭代过程。你可以基于初步结果，不断让ChatGPT帮你调整策略参数、尝试不同的指标（如RSI、MACD），或加入止损止盈逻辑，逐步优化你的交易系统。

![](img/a68515b9fa2b0b445c9fc4df1da91c5c_49.png)
