## 简单增长率累加的问题

上一节我们介绍了如何计算增长率。本节中我们来看看，能否将一段时间内每天的增长率简单累加，以反映整体的增长情况。

例如，假设有一只股票：
*   第一天价格：100元
*   第二天价格：50元
*   第三天价格：75元

![](img/7e079b4d2d01501723f2f430d3841a04_4.png)

我们分别计算增长率：
*   第二天增长率：(50 - 100) / 100 = -0.5
*   第三天增长率：(75 - 50) / 50 = 0.5

如果简单累加增长率：-0.5 + 0.5 = 0。这个结果为0，似乎意味着股票从第一天到第三天“没有涨跌”。

但观察原始数据：第一天100元，第三天75元，实际是下跌的。这说明**简单的增长率累加无法正确反映连续时间内的整体变化**。

## 引入对数计算连续复合增长率

![](img/7e079b4d2d01501723f2f430d3841a04_6.png)

![](img/7e079b4d2d01501723f2f430d3841a04_8.png)

对于连续时间的序列，如果想进行有意义的累加操作，我们需要借助数学中的对数。

核心思想是：将对数形式的日收益率进行累加，其效果等同于计算首日与末日价格的比例。

![](img/7e079b4d2d01501723f2f430d3841a04_10.png)

**公式推导**：
假设我们计算每日的对数收益率：`log(后一天价格 / 前一天价格)`。
那么两天的对数收益率之和为：
`log(价格2 / 价格1) + log(价格3 / 价格2) = log( (价格2 / 价格1) * (价格3 / 价格2) ) = log(价格3 / 价格1)`

可以看到，对数收益率的累加，最终等价于计算**最后一天价格与第一天价格之比的对数**。这正好描述了我们想要的整体差异。

![](img/7e079b4d2d01501723f2f430d3841a04_12.png)

形象地说，这解决了“第一天投入1元钱，经过一系列涨跌后，在最后一天相当于多少钱”的问题。

### 步骤1：计算对数收益率

![](img/7e079b4d2d01501723f2f430d3841a04_28.png)

![](img/7e079b4d2d01501723f2f430d3841a04_30.png)

首先，需要计算每天相对于前一天的比率，然后取对数。

```python
# 计算每日价格与前一日价格的比率
price_ratio = data / data.shift(1)
# 计算对数收益率 (以e为底)
log_returns = np.log(price_ratio)
```

![](img/7e079b4d2d01501723f2f430d3841a04_32.png)

![](img/7e079b4d2d01501723f2f430d3841a04_34.png)

这里`data.shift(1)`的作用是将数据整体向上移动一行，这样`data`中的当前行就对应了`data.shift(1)`中的前一行数据。

![](img/7e079b4d2d01501723f2f430d3841a04_36.png)

### 步骤2：计算累积对数收益率

接下来，我们需要将对数收益率进行累积求和，得到从起始日开始到每一天的累积变化。

![](img/7e079b4d2d01501723f2f430d3841a04_38.png)

![](img/7e079b4d2d01501723f2f430d3841a04_40.png)

```python
# 计算累积对数收益率
cumulative_log_returns = log_returns.cumsum()
```

![](img/7e079b4d2d01501723f2f430d3841a04_42.png)

`cumsum()`函数实现累加操作，例如输入`[a, b, c]`，输出`[a, a+b, a+b+c]`。

![](img/7e079b4d2d01501723f2f430d3841a04_44.png)

![](img/7e079b4d2d01501723f2f430d3841a04_46.png)

### 步骤3：还原为实际价值倍数

累积对数收益率表示的是价值倍数的对数。要得到“第一天1元钱相当于当天多少钱”的实际倍数，需要对其进行指数还原。

![](img/7e079b4d2d01501723f2f430d3841a04_48.png)

![](img/7e079b4d2d01501723f2f430d3841a04_50.png)

```python
# 通过指数函数还原为实际的价值倍数
cumulative_value_ratio = np.exp(cumulative_log_returns)
```

以下是本课程中涉及的核心操作总结：

![](img/7e079b4d2d01501723f2f430d3841a04_62.png)

![](img/7e079b4d2d01501723f2f430d3841a04_64.png)

*   **计算差异值**：使用`data.diff()`可以方便地计算相邻数据的差值。
*   **计算简单增长率**：公式为`(当期值 - 前一期值) / 前一期值`。
*   **计算连续复合增长率（对数收益率）**：公式为`log(当期值 / 前一期值)`。
*   **计算累积增长倍数**：先计算对数收益率，再使用`cumsum()`累加，最后用`exp()`函数还原。
