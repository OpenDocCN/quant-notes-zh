# 金融量化分析：P4：03 股票市场构成 🏛️

在本节课中，我们将要学习股票市场的构成。了解市场中有哪些参与者以及他们各自扮演的角色，是理解股票交易如何运作的基础。我们将从公司和投资者开始，逐步介绍监管机构、交易所和中介机构，最后解释股票指数和板块的概念。

## 公司和投资者

上一节我们介绍了股票的分类，本节中我们来看看股票市场的构成。市场中的参与者首先包括公司和投资者。

*   **公司**：需要融资的一方。
*   **投资者**：提供资金的一方。

公司通过上市向投资者融资，但公司和投资者不能直接进行交易。

## 监管与服务机构

为了防止暗箱操作，市场需要监管和服务机构。以下是主要的机构类型：

*   **证监会**：证券行业的监管机构。公司上市需向证监会提交材料，证监会负责审查公司是否存在违法行为（如欺诈、洗钱），并有权决定公司能否上市或将其退市。
*   **证券业协会**：作用相对较弱，例如主办证券从业资格考试。
*   **交易所**：提供股票交易的场所（如上海和深圳交易所）。早期交易者需亲临交易所，现在则通过网络连接。交易所负责处理所有买卖请求。

## 证券中介机构（券商）

个人投资者不能直接去交易所买卖股票，成本过高。这涉及到历史原因：在电子化之前，买卖股票需要实物凭证，手续繁琐。交易所通过出售昂贵的交易席位来设定门槛。

因此，证券中介机构（俗称券商）应运而生。券商（如中信证券、中金公司）在交易所有席位。他们开发交易软件（如同花顺），投资者通过软件将指令发送给券商，券商再利用其席位将指令传达至交易所，完成交易。

## 交易所与板块

中国有两个主要交易所：上海证券交易所和深圳证券交易所。每个交易所下设有不同的板块。

*   **上海证券交易所**：主要设有**主板**。
*   **深圳证券交易所**：设有**主板**、**中小板**和**创业板**。中小板和创业板是为规模较小或处于创业阶段的公司设立的，上市要求比主板宽松，例如创业板要求公司连续几年达到一定净利润即可。

## 大盘指数

对于每个板块，我们常用“大盘”来指代其整体表现。大盘实际上是一个**指数**，它反映了该板块内所有股票的综合表现。

*   **沪指/上证指数**：代表上海主板市场的整体走势。
*   **深成指**：代表深圳主板市场的整体走势。
*   **中小板指**：代表深圳中小板市场的整体走势。
*   **创业板指**：代表深圳创业板市场的整体走势。

指数就像是一个平均值或趋势图，将数百只股票的表现打包，用以判断整个市场是向好还是向坏。例如，当大盘指数上涨时，通常意味着市场内多数股票在上涨，整体行情向好。

本节课中我们一起学习了股票市场的构成。我们了解了市场中的核心参与者：融资的公司、投资的个人、进行监管的证监会、提供交易场所的交易所以及连接投资者与交易所的券商。同时，我们也认识了中国的两大交易所及其不同板块，并明白了大盘指数是反映市场整体走势的重要工具。理解这些基本构成是进行后续金融量化分析的重要前提。
