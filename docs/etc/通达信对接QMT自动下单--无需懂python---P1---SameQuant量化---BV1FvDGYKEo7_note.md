![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_7.png)

## 条件预警自动交易

除了手动操作，本工具还支持更智能的交易方式：利用通达信的**条件预警**功能实现全自动交易。

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_15.png)

以下是设置流程：

1.  在通达信中打开并设置**条件预警**功能。
2.  当条件预警被触发，符合条件的股票会自动被添加到软件的“买入股票池”中。

## 交易参数配置与自动卖出

了解了两种买入方式后，我们来看看如何配置交易参数以及实现自动卖出功能。本工具提供了灵活的配置选项。

**买入参数**：
*   **单笔买入金额**：您可以预设每笔委托买入的固定金额。
*   **加价幅度**：委托买入价会在股票最新价的基础上增加一定百分比。例如，设置加价 `1.5%`。
    *   计算公式：`委托买入价 = 最新价 * (1 + 加价百分比)`
    *   如果计算结果超过涨停价，则自动以涨停价委托。

**卖出参数**：
*   **减价幅度**：委托卖出价会在最新价的基础上减少一定百分比。例如，默认减少 `-1.5%`。
    *   计算公式：`委托卖出价 = 最新价 * (1 - 减价百分比)`

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_25.png)

**自动同步持仓**：
软件会自动从您的交易账户读取**可用余额大于零**的持仓，并同步到“可用持仓”股票池中，省去手动导入的麻烦。

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_27.png)

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_25.png)

**自动卖出流程**：
1.  您可以在通达信中设置条件预警，监控“可用持仓”池中的股票。
2.  当持仓股满足卖出条件时，预警会将其添加到“卖出股票池”。
3.  一旦“卖出股票池”出现新股票，软件将立即执行卖出委托。

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_29.png)

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_29.png)

我们可以进行演示：将股票加入卖出列表后，系统瞬间完成了卖出委托。

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_33.png)

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_31.png)

在QMT的委托记录中，可以立即查看到对应的卖出订单。

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_35.png)

![](img/f7798c7b2bc001e1e0f9bedb3f6876ea_33.png)
