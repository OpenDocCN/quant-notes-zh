## 数据查询与预处理

在交易开始前，我们通常需要获取股票的最新财务数据以辅助决策。这涉及到使用平台提供的查询功能。

所有可用的数据指标都在官方帮助文档中有详细说明。量化交易本质上是数据挖掘，涉及大量指标。本节课我们仅以营业收入为例进行演示。

以下是查询财务数据的基本步骤：

1.  **导入查询模块**：首先需要引入相关的查询函数。
2.  **构建查询语句（query）**：指定你想要获取的指标字段。
3.  **添加过滤条件（filter）**：限定查询范围，例如只查询股票池内的股票。
4.  **排序（order_by）**：根据指标对结果进行排序。
5.  **限制数量（limit）**：只保留排名靠前的若干只股票。

我们将这段逻辑写在策略的 `before_trading` 函数中，以便在每个交易日开始前执行。

![](img/fd2bb967d9b78e16092816739f06bd91_17.png)

![](img/fd2bb967d9b78e16092816739f06bd91_19.png)

```python
def before_trading(context):
    # 1. 构建查询，获取营业收入指标
    q = query(
        indicator.operating_revenue  # 营业收入指标
    ).filter(
        # 2. 过滤条件：股票代码必须在我们的股票池中
        valuation.code.in_(context.stock_pool)
    ).order_by(
        # 3. 排序：按营业收入降序排列（从高到低）
        indicator.operating_revenue.desc()
    ).limit(
        # 4. 限制：只取前10名
        10
    )

    # 执行查询，将结果（DataFrame格式）存入上下文
    context.fundamental_df = get_fundamentals(q)

    # 打印查看结果
    print(context.fundamental_df)
```
