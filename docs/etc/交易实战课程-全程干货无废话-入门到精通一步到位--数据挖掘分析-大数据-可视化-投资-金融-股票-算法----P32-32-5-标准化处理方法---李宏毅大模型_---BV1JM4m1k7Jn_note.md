在本节课中，我们将要学习数据预处理中一个至关重要的步骤——标准化。标准化是数据挖掘和机器学习中常用的技术，旨在消除不同特征（或指标）之间由于量纲和取值范围不同带来的影响，使数据更易于分析和建模。

![](img/3bafde627d18561d64cfc6124946fa8e_3.png)

## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体原理。

标准化操作的核心目标是让不同维度的数据具有可比性。假设我们有一组数据，包含两个指标X1和X2。如下图所示，X1和X2的取值范围通常不同。

![](img/3bafde627d18561d64cfc6124946fa8e_1.png)

如果直接使用这些原始数据进行分析，数值范围大的指标可能会对模型结果产生不成比例的巨大影响。为了用统一的尺度去衡量所有指标，避免数值层面的差异干扰分析结果，我们需要进行标准化处理。

## 标准化的公式分解

标准化通过一个数学公式来实现，这个公式可以分解为两个清晰的步骤。

标准化的核心公式如下：
**`z = (x - μ) / σ`**
其中，`x` 是原始数据，`μ` 是该维度数据的均值，`σ` 是该维度数据的标准差。

这个公式可以分解为两步来理解：

1.  **去均值（中心化）**：`x - μ`
    这一步的目的是将数据集的中心移动到零点。原始数据通常不以原点为中心对称。减去均值后，数据在各个维度上将以零为中心对称分布。

2.  **除标准差（缩放）**：`(x - μ) / σ`
    去均值后，各维度的取值范围差异依然存在。除以标准差就是为了解决这个问题。标准差反映了数据的离散程度。取值范围大、个体差异大的数据组，其标准差也大；反之则小。

以下是关于标准差如何调整取值范围的一个类比：
*   **第一组**：成员收入相近（例如都在1-2万元之间），标准差较小。
*   **第二组**：成员收入差异巨大（例如从1万元到数千亿元），标准差非常大。

对于取值范围大的数据，除以一个大的标准差，可以将其缩放至一个合理的范围；对于取值范围小的数据，除以一个小的标准差，可以避免其被过度压缩。最终，所有维度的数据都被调整到近似的数值范围内。

如图所示，经过完整的标准化处理后，数据不仅以零为中心对称，而且各维度的取值范围也变得相近。

## 标准化的代码实现

![](img/3bafde627d18561d64cfc6124946fa8e_7.png)

理解了原理后，我们来看看如何用代码实现标准化。在数据科学中，我们既可以从零开始实现，也可以借助成熟的工具库。

以下是手动实现标准化函数的示例：

![](img/3bafde627d18561d64cfc6124946fa8e_9.png)

![](img/3bafde627d18561d64cfc6124946fa8e_11.png)

```python
def standardize(series):
    """
    对输入的序列进行标准化处理。
    参数:
        series: pandas Series 或类似数组结构。
    返回:
        标准化后的序列。
    """
    # 计算均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    # 应用标准化公式: (x - mean) / std
    return (series - mean_val) / std_val
```

![](img/3bafde627d18561d64cfc6124946fa8e_13.png)

代码非常简单：先计算序列的均值和标准差，然后应用公式即可。

当然，在实际项目中，我们更常使用现成的工具库。例如，在 `scikit-learn` 库的预处理模块中，提供了 `StandardScaler` 类，可以更高效、便捷地完成标准化，通常只需一两行代码。如果你想使用，可以在 `scikit-learn` 官方文档中搜索 “StandardScaler” 查看详细用法。

## 总结

![](img/3bafde627d18561d64cfc6124946fa8e_35.png)

本节课中我们一起学习了数据标准化处理。
*   **目的**：消除不同特征间量纲和取值范围的差异，使数据具有可比性。
*   **原理**：通过 **`z = (x - μ) / σ`** 公式，先减去均值使数据以零为中心，再除以标准差缩放数据范围。
*   **效果**：处理后的数据各维度均以零为中心对称，且取值范围近似。
*   **实现**：可以手动实现计算，也可以使用如 `scikit-learn` 中的 `StandardScaler` 等工具高效完成。

![](img/3bafde627d18561d64cfc6124946fa8e_37.png)

标准化是数据预处理中的一个基础且关键的步骤，在金融数据分析、机器学习模型训练等场景中应用极为广泛。掌握它，能为后续的深入分析打下坚实的基础。
