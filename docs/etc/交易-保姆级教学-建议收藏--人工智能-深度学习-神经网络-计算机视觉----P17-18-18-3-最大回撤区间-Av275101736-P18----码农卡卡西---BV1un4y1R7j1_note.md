## 什么是最大回撤？

上一节我们介绍了策略分析中的一些基本概念，本节中我们来看看最大回撤。最大回撤描述的是在投资过程中，资产净值从最高点到之后最低点的最大跌幅。它衡量的是策略在一段时期内可能面临的最糟糕情况。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_1.png)

最大回撤不能仅从字面理解。它描述的是投资过程中“最惨”的阶段。在炒股过程中，价格会有下跌的时候。最惨烈、持续时间最长的下跌区间，就称为最大回撤区间。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_3.png)

## 最大回撤的公式与含义

为了更精确地理解，我们通过公式来看。最大回撤的计算公式通常表示为：

**最大回撤 = max[(Pi - Pj) / Pi]**

其中：
*   **Pi** 代表第 i 天的资产价格（或净值）。
*   **Pj** 代表第 i 天之后某一天（第 j 天）的资产价格，且 **j > i**。

这个公式的意思是：我们需要遍历所有可能的时间点 i 和它之后的时间点 j，计算从 i 到 j 的跌幅（**（Pi - Pj）/ Pi**），然后找出所有跌幅中的最大值。这个最大值就是最大回撤率。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_5.png)

它描述的并不是策略的盈利能力，而是策略的**风险**。风险意味着，即使一个策略长期来看是赚钱的，但在过程中可能会经历一段非常痛苦的下跌期。最大回撤就是量化这段“最痛苦”时期的严重程度。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_7.png)

例如，2008年股市从6000多点跌至1600多点，这个过程就是一次巨大的回撤，给许多投资者带来了严重亏损。最大回撤区间就是用来描述在某一段时间内，你的亏损有多严重。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_5.png)

![](img/c0cdf76a1113b5e2e4c8db21849cdade_9.png)

![](img/c0cdf76a1113b5e2e4c8db21849cdade_11.png)

在我们的策略回测结果中，通常会统计并标出最大回撤发生的区间。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_13.png)

## 如何用Python计算最大回撤？

![](img/c0cdf76a1113b5e2e4c8db21849cdade_15.png)

接下来，我们看看如何用代码实现最大回撤的计算。核心思路是：我们需要找出净值曲线在每一个时间点之前的最高点，然后计算当前净值相对于那个最高点的回撤，最后找出所有回撤中的最大值。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_17.png)

以下是计算步骤：

![](img/c0cdf76a1113b5e2e4c8db21849cdade_19.png)

首先，我们使用Pandas库。假设我们有一个代表每日净值的Series，名为 `net_value`。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_21.png)

![](img/c0cdf76a1113b5e2e4c8db21849cdade_23.png)

第一步，计算历史最高值序列。我们使用 `cummax()` 函数，它可以得到每个时间点之前（包括当前点）的历史最大值序列。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_25.png)

![](img/c0cdf76a1113b5e2e4c8db21849cdade_27.png)

```python
# 计算历史最高净值序列
historical_max = net_value.cummax()
```

`cummax()` 函数的作用是：对于序列中的每一个位置，返回从起始位置到该位置所有值中的最大值。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_29.png)

![](img/c0cdf76a1113b5e2e4c8db21849cdade_31.png)

第二步，计算每日的回撤。每日回撤 = （历史最高值 - 当日净值）/ 历史最高值。

```python
# 计算每日回撤率
daily_drawdown = (historical_max - net_value) / historical_max
```

![](img/c0cdf76a1113b5e2e4c8db21849cdade_33.png)

![](img/c0cdf76a1113b5e2e4c8db21849cdade_35.png)

第三步，从每日回撤中找出最大值，即为最大回撤率。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_37.png)

```python
# 计算最大回撤率
max_drawdown = daily_drawdown.max()
```

同时，我们也可以找出最大回撤发生的开始和结束日期（即回撤区间）。

![](img/c0cdf76a1113b5e2e4c8db21849cdade_39.png)

```python
# 找到最大回撤率所在的日期
end_date = daily_drawdown.idxmax()
# 找到最大回撤开始日期（回撤开始前的最高点日期）
# 我们需要找到在 end_date 之前，净值等于 historical_max[end_date] 的最后一个日期
start_date = net_value[:end_date][net_value[:end_date] == historical_max[end_date]].index[-1]
```

1.  **最大回撤**衡量的是投资策略可能面临的最大亏损幅度，是一个重要的**风险度量指标**。
2.  它的计算公式是寻找净值从某个高点下跌到后续低点的最大百分比跌幅。
3.  在Python中，我们可以借助 `cummax()` 函数高效地计算每日回撤，进而得到最大回撤率及其发生的区间。
