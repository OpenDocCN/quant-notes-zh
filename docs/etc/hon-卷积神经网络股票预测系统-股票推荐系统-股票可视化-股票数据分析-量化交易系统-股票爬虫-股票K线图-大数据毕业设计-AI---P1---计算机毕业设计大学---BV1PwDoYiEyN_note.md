# 计算机毕业设计：P1：项目概述与核心技术栈

在本课程中，我们将学习如何构建一个综合性的股票分析与预测系统。该系统将结合Python编程、数据爬取、可视化、深度学习和量化交易等多个技术领域，旨在为计算机相关专业的毕业设计提供一个完整、实用的项目范例。

## 项目概述

本项目旨在开发一个集股票数据获取、分析、可视化、预测与推荐于一体的智能系统。系统将自动从网络获取股票数据，通过卷积神经网络（CNN）等算法进行价格趋势预测，并结合量化交易策略为用户提供投资建议。整个项目流程覆盖了从数据源到最终应用的全链路。

上一节我们了解了项目的整体目标，接下来，我们将深入探讨实现这一目标所需的核心技术组件。

## 核心技术栈

以下是构建本系统所需的关键技术及其作用：

*   **Python**：作为项目的主要编程语言，因其丰富的数据科学和机器学习库而被选用。
*   **卷积神经网络**：一种深度学习模型，在本项目中用于从股票历史数据（如K线图）中提取特征并进行价格预测。其核心操作是卷积，公式可简化为 `特征图 = 卷积核 * 输入数据 + 偏置`。
*   **股票爬虫**：用于自动从财经网站（如新浪财经、东方财富）获取实时或历史的股票交易数据。通常使用 `requests` 库发送HTTP请求，并用 `BeautifulSoup` 或 `pandas` 解析HTML或JSON数据。
*   **股票数据分析**：对爬取到的原始数据进行清洗、计算技术指标（如移动平均线MA、相对强弱指数RSI）和统计分析，为后续建模做准备。
*   **股票可视化**：将分析结果和预测趋势以图表形式直观展示，主要使用 `matplotlib` 或 `plotly` 库绘制K线图、趋势线等。
*   **量化交易系统**：基于数学模型和计算机程序执行交易策略的系统。核心是根据预测信号和策略规则生成交易指令，例如：`if 预测价格 > 当前价格 * 1.02: 执行买入`。
*   **股票推荐系统**：综合预测结果、风险指标和用户偏好，筛选并推荐具有潜在投资价值的股票。

在明确了技术栈之后，我们可以预见系统将具备强大的数据处理和智能决策能力。

## 系统工作流程

系统的运行遵循一个清晰的逻辑链条：

![](img/c883037a9084196bd22f2c7a0f0d6554_3.png)

1.  **数据获取**：爬虫程序定期运行，抓取指定股票的历史及实时数据。
2.  **数据处理**：清洗数据，计算必要的技术指标，并将数据格式化为模型可接受的输入（例如，将价格序列转换为图像格式供CNN使用）。
3.  **模型训练与预测**：使用历史数据训练CNN模型，然后用训练好的模型对最新数据进行预测，输出未来价格走势的概率分布。
4.  **可视化展示**：将原始K线图、技术指标曲线和模型预测结果一同绘制在交互式图表中。
5.  **策略与推荐**：量化交易模块根据预测结果和预设策略评估买卖点；推荐系统则综合多项因素生成股票推荐列表。

本节课中，我们一起学习了“Python+卷积神经网络股票预测系统”项目的全貌。我们明确了这是一个融合了**数据爬取**、**CNN预测**、**可视化**和**量化交易**的综合性毕业设计。从数据源头到最终的投资建议，每个技术环节都至关重要，共同构成了一个完整的AI金融分析系统原型。在接下来的课程中，我们将逐一深入每个模块的实现细节。
