## 第二步：获取因子指标数据 📈

![](img/4d5360140194a79c7b3f94431a91aa2b_9.png)

有了交易日列表后，下一步是查询具体的因子指标数据。我们将以市盈率（P/E Ratio）为例，查询它在所有A股股票中的值。

![](img/4d5360140194a79c7b3f94431a91aa2b_11.png)

以下是获取因子数据的步骤：

![](img/4d5360140194a79c7b3f94431a91aa2b_13.png)

1.  **确定股票池**：首先，我们需要获取所有A股股票的代码列表，作为我们的分析对象。
2.  **构建查询语句**：使用工具包提供的 `query` 功能，指定要查询的因子（如 `fundamentals.eod_derivative_indicator.pe_ratio`）。
3.  **应用过滤器**：在查询中，我们通过 `filter` 条件限定只查询股票池中的股票。
4.  **执行查询**：最后，对指定的单个日期执行查询，获取该日所有股票的因子数据。

![](img/4d5360140194a79c7b3f94431a91aa2b_15.png)

```python
# 1. 获取股票池
stock_pool = get_all_securities('stock').index.tolist() # 示例函数，获取所有A股代码

![](img/4d5360140194a79c7b3f94431a91aa2b_17.png)

# 2. 构建查询语句
q = query(fundamentals.eod_derivative_indicator.pe_ratio).filter(
    fundamentals.eod_derivative_indicator.code.in_(stock_pool)
)

![](img/4d5360140194a79c7b3f94431a91aa2b_19.png)

# 3. 执行查询（以第一个交易日为例）
single_date_data = get_fundamentals(q, date=trading_dates[0])
```

![](img/4d5360140194a79c7b3f94431a91aa2b_21.png)

执行后，`single_date_data` 将包含指定交易日所有A股股票的市盈率数据。数据是一个三维结构，但通常我们只关心最后一个维度的具体数值。

![](img/4d5360140194a79c7b3f94431a91aa2b_23.png)

![](img/4d5360140194a79c7b3f94431a91aa2b_25.png)

```python
# 查看前5条数据
print(single_date_data.iloc[0, 0, :5])
```

输出会显示股票代码及其对应的市盈率数值。请注意，由于我们只查询了一天，结果中不包含日期列，日期信息由传入的 `trading_dates[0]` 参数决定。
