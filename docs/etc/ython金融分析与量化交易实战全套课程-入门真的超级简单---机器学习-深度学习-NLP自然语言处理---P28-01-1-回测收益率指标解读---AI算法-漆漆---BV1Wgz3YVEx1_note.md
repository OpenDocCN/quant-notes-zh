## 回测收益率

假设你构建了一个策略，基于2015年至2020年的某只股票数据，规定了买入和卖出的时机。回测收益率就是用来衡量这个策略最终为你赚了多少钱的指标。关键点在于，它表示的不是绝对的金额，而是相对于初始本金所获得的**收益比例**。例如，结果显示4.32%，就意味着你的策略使本金增值了约4.32%。

![](img/94785ce3b3248bf055165fe3d84c6e4e_25.png)

![](img/94785ce3b3248bf055165fe3d84c6e4e_21.png)

![](img/94785ce3b3248bf055165fe3d84c6e4e_27.png)

它的计算公式如下：

**回测收益率 = (期末资产总值 - 期初本金) / 期初本金**

用符号表示为：
**Return = (P_t - P_0) / P_0**

其中，`P_t` 代表策略结束时的资产总值（卖出所有头寸后的总金额），`P_0` 代表期初投入的本金。这个指标是评估策略表现最常用的指标，直观地反映了策略的盈利能力。

![](img/94785ce3b3248bf055165fe3d84c6e4e_23.png)

接下来，我们通过Python代码演示如何计算。我们不仅可以计算最终的总收益率，还可以观察收益率随时间的变化情况。

![](img/94785ce3b3248bf055165fe3d84c6e4e_25.png)

我们可以绘制一张图，来展示从期初开始，每一时刻的累计收益率走势。这需要计算每个时间点的资产值与期初值的比值。

![](img/94785ce3b3248bf055165fe3d84c6e4e_27.png)

![](img/94785ce3b3248bf055165fe3d84c6e4e_29.png)

以下是计算步骤的代码实现：

![](img/94785ce3b3248bf055165fe3d84c6e4e_31.png)

```python
# 计算每个时间点的资产与期初资产的比值（即累计净值）
cumulative_value = data[‘close‘] / data[‘close‘].iloc[0]

# 绘制累计净值曲线图
plt.figure(figsize=(10, 6))
cumulative_value.plot()
plt.title(‘策略累计净值走势‘)
plt.ylabel(‘累计净值‘)
plt.xlabel(‘日期‘)
plt.grid(True)
plt.show()
```

根据公式，最终收益率是最后一个时间点的累计净值减去1。

```python
# 计算最终回测收益率
final_return = cumulative_value.iloc[-1] - 1

![](img/94785ce3b3248bf055165fe3d84c6e4e_41.png)

# 以表格形式美观地展示结果
result_df = pd.DataFrame({
    ‘回测收益率‘: [final_return]
}, index=[‘最终值‘])
print(result_df)
```
