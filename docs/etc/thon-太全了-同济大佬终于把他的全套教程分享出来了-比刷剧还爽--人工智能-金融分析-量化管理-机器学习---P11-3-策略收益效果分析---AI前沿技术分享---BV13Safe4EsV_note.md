## 计算基础收益率

上一节我们介绍了股价数据，本节中我们来看看如何计算每日的收益率。收益率是衡量投资表现的核心指标，它表示资产价格在单位时间内的变化率。

![](img/8926439ee19ee666b4586ac7f0ed696b_5.png)

在计算过程中，我们不能直接累加每日的价格变化百分比，因为这是一个乘法过程。为了便于后续的累加计算，我们通常先计算对数收益率。

![](img/8926439ee19ee666b4586ac7f0ed696b_7.png)

以下是计算对数收益率的步骤：

![](img/8926439ee19ee666b4586ac7f0ed696b_9.png)

1.  获取股价序列 `data['Adj Close']`。
2.  计算每日的对数收益率，公式为：`log(当日收盘价 / 前一日收盘价)`。在代码中，这通常通过 `np.log(data['Adj Close'] / data['Adj Close'].shift(1))` 来实现。

![](img/8926439ee19ee666b4586ac7f0ed696b_11.png)

![](img/8926439ee19ee666b4586ac7f0ed696b_13.png)

执行上述代码后，我们得到一个名为 `returns` 的序列，它代表了股价每日的增长（或下跌）率。

![](img/8926439ee19ee666b4586ac7f0ed696b_15.png)

## 应用策略计算策略收益

![](img/8926439ee19ee666b4586ac7f0ed696b_17.png)

我们已经有了基础的收益率，接下来看看如何将我们的交易策略应用进去。本节中，我们将使用一个简化的“双均线”策略作为示例。该策略的核心是生成一个交易信号序列 `position`。

![](img/8926439ee19ee666b4586ac7f0ed696b_19.png)

`position` 序列中的值通常为 1 或 -1：
*   **1** 代表持有头寸，即跟随市场原始走势。
*   **-1** 代表反向头寸，即与市场原始走势相反。

![](img/8926439ee19ee666b4586ac7f0ed696b_21.png)

![](img/8926439ee19ee666b4586ac7f0ed696b_23.png)

以下是计算策略收益的步骤：

![](img/8926439ee19ee666b4586ac7f0ed696b_25.png)

![](img/8926439ee19ee666b4586ac7f0ed696b_27.png)

1.  确保 `position` 序列与 `returns` 序列在时间上对齐。由于交易决策是基于前一天的信息做出的，因此 `position` 序列也需要进行 `shift(1)` 操作。
2.  计算策略每日的收益率，公式为：`策略日收益率 = position * returns`。这意味着当 `position` 为 -1 时，我们将市场原始收益反转。
3.  将计算出的策略收益率序列保存，例如命名为 `strategy_returns`。

![](img/8926439ee19ee666b4586ac7f0ed696b_29.png)

## 累计收益计算与比较

计算出每日的策略收益率后，我们需要将其累加起来，得到从投资开始到结束的总收益，并与不做任何策略的原始累计收益进行比较。

![](img/8926439ee19ee666b4586ac7f0ed696b_31.png)

![](img/8926439ee19ee666b4586ac7f0ed696b_33.png)

由于我们之前计算的是对数收益率，累加非常方便。以下是计算和比较的步骤：

![](img/8926439ee19ee666b4586ac7f0ed696b_35.png)

![](img/8926439ee19ee666b4586ac7f0ed696b_37.png)

1.  **处理缺失值**：由于使用了 `shift` 操作，收益率序列的第一行会是缺失值（NaN），需要使用 `dropna()` 方法将其去除。
2.  **计算累计对数收益**：对处理后的 `returns` 和 `strategy_returns` 序列分别执行 `.sum()` 操作，得到总的对数收益。
3.  **还原为实际收益**：将对数收益通过指数函数 `np.exp()` 还原为实际的总收益倍数。例如，结果为 4.0 表示 1 元本金最终变成了 4 元。
4.  **比较结果**：对比 `原始累计收益` 和 `策略累计收益` 的数值。在我们的示例中，原始收益约为 4 元，而应用双均线策略后的收益约为 5.8 元，显示了策略的有效性。

## 总结

![](img/8926439ee19ee666b4586ac7f0ed696b_39.png)

本节课中我们一起学习了量化策略收益分析的全过程。我们首先计算了基础的日对数收益率，然后通过乘以交易信号序列 `position` 得到了策略的日收益率，最后通过累加和对数还原，比较了策略收益与原始市场收益的差异。这个过程是量化回测中评估策略有效性的基础环节。
