# 机器学习入门：P50：回归问题概述

在本节课中，我们将要学习机器学习中的一个核心问题类型：回归问题。我们将从基本概念入手，理解什么是回归，它与分类问题的区别，并初步了解线性回归模型是如何构建和思考的。

## 什么是回归问题？

上一节我们介绍了机器学习的基本概念，本节中我们来看看回归问题。

在监督学习领域，主要存在两种类型的问题：分类问题和回归问题。

以下是两种问题的核心区别：

*   **分类问题**：预测结果是离散的类别标签。例如，银行判断是否批准贷款申请，结果只有“批准”（类别1）或“拒绝”（类别0）。
*   **回归问题**：预测结果是连续的数值。例如，银行判断能批准多少额度的贷款，结果可能是在一个区间内的任意具体数值，如3.62万、4.89万或9.76万元。

我们今天要讨论的，就是回归问题。

## 回归问题的数据形式

要理解回归模型如何工作，首先需要了解其处理的数据是什么样子。

假设我们有一份数据集，旨在根据用户的“工资”和“年龄”来预测其“贷款额度”。这份数据可以整理成如下表格形式：

| 工资 (X1) | 年龄 (X2) | 额度 (Y) |
| :--- | :--- | :--- |
| ... | ... | ... |

![](img/3aaeee8b411074714a262d525f420d19_1.png)

在监督学习中，数据通常由两部分组成：

*   **特征 (X)**：用于进行预测的输入变量。在本例中，即“工资”和“年龄”，我们分别用 `X1` 和 `X2` 表示。
*   **标签 (Y)**：我们希望模型预测的目标输出。在本例中，即“贷款额度”，用 `Y` 表示。

因此，我们的目标是：利用已知的特征 `X1` 和 `X2`，去预测未知的标签 `Y` 的值。

## 构建线性回归模型

既然我们的目标是预测 `Y`，而 `Y` 显然与特征 `X1` 和 `X2` 相关，一个最直观的想法是建立一个数学方程来描述这种关系。

![](img/3aaeee8b411074714a262d525f420d19_3.png)

我们可以假设 `Y` 是 `X1` 和 `X2` 的线性组合：
`Y = θ1 * X1 + θ2 * X2`

在这个方程中：
*   `θ1` 和 `θ2` 被称为**参数**或**权重**。
*   它们代表了每个特征对最终结果 `Y` 的影响程度。例如，如果 `θ1 = 3`，`θ2 = 1`，则意味着工资 (`X1`) 对贷款额度的影响是年龄 (`X2`) 的3倍。

现在，在我们的数据集中，`X1`、`X2` 和对应的 `Y` 都是已知的。模型训练的核心目标就变成了：**寻找一组最合适的参数 `θ1` 和 `θ2`，使得这个方程能够最好地描述数据**。

## 模型的挑战与目标

然而，我们立刻会遇到一个现实问题：一条直线（或一个平面）几乎不可能完美穿过所有的数据点。

想象将数据点画在图上，我们的线性方程 `Y = θ1*X1 + θ2*X2` 对应的是一个平面。而实际的数据点（红色点）会散布在这个平面周围。



既然无法让平面穿过每一个点，那么一个合理的优化目标就是：**找到这样一个平面，使得所有数据点到这个平面的“距离”之和最小**。换句话说，这个平面应该尽可能贴近大部分的数据点，实现对数据的“最佳拟合”。

![](img/3aaeee8b411074714a262d525f420d19_5.png)

本节课中我们一起学习了回归问题的基本概念。我们明确了回归与分类的区别，认识了回归数据中特征(`X`)和标签(`Y`)的构成，并初步建立了线性回归模型的方程形式 `Y = θ1*X1 + θ2*X2`。最后，我们了解到模型的目标是寻找最佳参数，以最小化预测值与真实值之间的总体差异，从而实现对数据的最佳拟合。在接下来的课程中，我们将深入探讨如何具体求解这些参数 `θ`。
