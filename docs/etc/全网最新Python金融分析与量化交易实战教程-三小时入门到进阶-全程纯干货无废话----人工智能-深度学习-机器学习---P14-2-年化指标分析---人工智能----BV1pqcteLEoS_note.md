## 概述
在本节课程中，我们将学习量化交易策略分析中的两个核心收益指标：回测收益率与年化收益率。理解这两个指标是评估策略有效性的基础。

上一节我们介绍了策略回测的基本流程，本节中我们来看看如何具体计算和解读策略的收益表现。

## 回测收益率
回测收益率是评估策略表现最直接的指标。它表示在整个回测周期结束后，初始投资的总回报率。

![](img/510a727b4b0b7ea7b150974432f9b180_4.png)

以下是计算回测收益率的核心概念：
*   **含义**：策略在整个回测期间的总收益与初始本金的比值。
*   **结果解读**：结果可能为正（盈利）或为负（亏损），直观反映了策略的“盈亏”情况。

![](img/510a727b4b0b7ea7b150974432f9b180_6.png)

![](img/510a727b4b0b7ea7b150974432f9b180_0.png)

![](img/510a727b4b0b7ea7b150974432f9b180_8.png)

根据计算结果，不同策略的回测收益率有正有负，这说明了投资存在风险，策略并非稳赚不赔。有些策略收益率很高，甚至翻倍；有些则收益较少或出现亏损。这个比值清晰地展示了策略相对于初始投资的盈利或亏损程度。

![](img/510a727b4b0b7ea7b150974432f9b180_10.png)

![](img/510a727b4b0b7ea7b150974432f9b180_2.png)

![](img/510a727b4b0b7ea7b150974432f9b180_12.png)

这个指标被称为**回测收益率**，是策略分析中最重要的指标之一。设计好一个策略后，首先需要观察的就是这个策略能带来多少收益。

![](img/510a727b4b0b7ea7b150974432f9b180_14.png)

## 年化收益率
理解了策略的总收益后，我们还需要一个标准化的指标来比较不同时间长度策略的盈利能力，这就是年化收益率。

![](img/510a727b4b0b7ea7b150974432f9b180_16.png)

![](img/510a727b4b0b7ea7b150974432f9b180_4.png)

![](img/510a727b4b0b7ea7b150974432f9b180_18.png)

对于策略评估来说，除了知道总共赚了多少，另一个关键值是**年化收益**。

![](img/510a727b4b0b7ea7b150974432f9b180_6.png)

年化收益是一个更常用的比较指标。如果你的回测周期很长（例如8年或10年），在向他人展示时，对方可能更关心策略平均每年的盈利能力。因此，我们需要计算出年化收益。

![](img/510a727b4b0b7ea7b150974432f9b180_20.png)

![](img/510a727b4b0b7ea7b150974432f9b180_8.png)

![](img/510a727b4b0b7ea7b150974432f9b180_22.png)

年化收益率本质上是对回测收益率进行公式变换得到的。它假设收益在一年内均匀复利增长，从而将不同时间长度的收益转化为一个可比较的年平均收益率。

![](img/510a727b4b0b7ea7b150974432f9b180_24.png)

![](img/510a727b4b0b7ea7b150974432f9b180_10.png)

![](img/510a727b4b0b7ea7b150974432f9b180_26.png)

在策略分析过程中，**年化收益通常是重点观察的核心指标**。虽然它由总收益计算得出，但提供了更统一的衡量标准。

### 年化收益率计算公式
其计算公式如下：

![](img/510a727b4b0b7ea7b150974432f9b180_28.png)

`年化收益率 = (1 + 总收益率) ^ (M / N) - 1`

![](img/510a727b4b0b7ea7b150974432f9b180_30.png)

以下是公式中各个参数的含义：
*   **R**：表示总收益率，即我们之前计算出的回测收益率。
*   **M**：表示一年的交易日数量，通常取 **250** 或 **252** 天。
*   **N**：表示策略回测使用的总交易日数量。

### 计算示例
以我们的回测数据为例进行实际计算。我们的数据样本量（N）为730个交易日。

![](img/510a727b4b0b7ea7b150974432f9b180_32.png)

![](img/510a727b4b0b7ea7b150974432f9b180_12.png)

![](img/510a727b4b0b7ea7b150974432f9b180_34.png)

计算过程可以转化为如下Python代码：
```python
# 假设 total_return 是之前计算好的总收益率（回测收益率）
# N 是总交易日数，例如 730
# M 是年交易日数，例如 250

![](img/510a727b4b0b7ea7b150974432f9b180_36.png)

annualized_return = (1 + total_return) ** (250 / 730) - 1
```

![](img/510a727b4b0b7ea7b150974432f9b180_38.png)

![](img/510a727b4b0b7ea7b150974432f9b180_14.png)

![](img/510a727b4b0b7ea7b150974432f9b180_40.png)

执行计算后，我们将得到一个新的“年化收益”指标。将其与之前的“回测收益”结果对比，可以发现数值存在差异。

![](img/510a727b4b0b7ea7b150974432f9b180_42.png)

![](img/510a727b4b0b7ea7b150974432f9b150974432f9b180_18.png)

![](img/510a727b4b0b7ea7b150974432f9b180_44.png)

例如，某个策略在超过700天（约3-4年）的回测中，总收益率为1.13（即113%）。但其年化收益率计算出来约为0.2几（即20%+）。对于总收益为负的策略，其年化亏损率也会相应调整。

![](img/510a727b4b0b7ea7b150974432f9b180_46.png)

![](img/510a727b4b0b7ea7b150974432f9b180_20.png)

![](img/510a727b4b0b7ea7b150974432f9b180_48.png)

这再次说明，在策略评估过程中，我们不仅需要关注最终的**总收益率**，还必须考察标准化后的**年化收益率**，后者对于衡量策略的持续盈利能力更为关键。

![](img/510a727b4b0b7ea7b150974432f9b180_50.png)

## 总结
本节课我们一起学习了量化策略收益分析的两个核心指标。
1.  **回测收益率**：反映了策略在整个测试周期内的总盈亏情况，是结果最直观的体现。
2.  **年化收益率**：通过公式 `(1 + 总收益率) ^ (年交易日数 / 总交易日数) - 1` 将总收益标准化，转化为年平均收益，便于不同周期策略之间的比较。

![](img/510a727b4b0b7ea7b150974432f9b180_52.png)

理解并计算这两个指标，是评估和优化交易策略的第一步。
