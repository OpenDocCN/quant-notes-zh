上一节我们介绍了 `div` 标签的基本概念，本节中我们来看看如何利用多个 `div` 进行网页布局。

观察一个典型网页，其结构通常包含页头、侧边栏、主内容区和页脚等部分。这些部分可以通过多个 `div` 标签组合而成。

## 构建网页结构

以下是构建一个简单网页布局的步骤，我们使用四个 `div` 来代表四个主要区域。

```html
<div id="menu">这里是页头</div>
<div id="sidebar">这里是侧边栏</div>
<div id="content-box">这里是主内容区</div>
<div id="footer">这里是页脚</div>
```

*   **`id="menu"`**：代表页头区域，通常包含导航菜单。
*   **`id="sidebar"`**：代表左侧的侧边栏。
*   **`id="content-box"`**：代表右侧的主内容区。
*   **`id="footer"`**：代表页面底部的页脚。

![](img/64f5139fba3091160df35ec70988d703_7.png)

## 使用CSS添加样式

![](img/64f5139fba3091160df35ec70988d703_8.png)

为了区分这些区域并控制其布局，我们需要为它们添加CSS样式。我们通过 `id` 选择器（以 `#` 开头）来精确地选中每个 `div`。

```css
#menu {
    background-color: yellow;
    height: 150px;
}

![](img/64f5139fba3091160df35ec70988d703_10.png)

#sidebar {
    background-color: orange;
    height: 500px;
    width: 200px;
}

![](img/64f5139fba3091160df35ec70988d703_12.png)

#content-box {
    background-color: greenyellow;
    height: 500px;
    width: 800px;
}

#footer {
    background-color: black;
    height: 100px;
}
```

此时，四个 `div` 会按照默认的文档流从上到下依次排列。

## 实现横向排列

我们希望侧边栏和主内容区能够并排显示。这需要用到 CSS 的 `float` 属性。

![](img/64f5139fba3091160df35ec70988d703_14.png)

`float` 属性可以使元素脱离正常的文档流，并向左或向右“浮动”，直到其外边缘碰到包含框或另一个浮动元素的边缘为止。

![](img/64f5139fba3091160df35ec70988d703_16.png)

我们将侧边栏和主内容区都设置为向左浮动：

![](img/64f5139fba3091160df35ec70988d703_18.png)

```css
#sidebar {
    float: left;
    /* 其他样式 */
}

#content-box {
    float: left;
    /* 其他样式 */
}
```

应用 `float: left;` 后，这两个 `div` 会从左到右并排排列。

![](img/64f5139fba3091160df35ec70988d703_20.png)

## 清除浮动影响

![](img/64f5139fba3091160df35ec70988d703_22.png)

当元素浮动后，后续的元素可能会受到影响。为了使页脚 `div` 不受上方浮动元素的影响，能够正常地显示在下方，我们需要使用 `clear` 属性。

![](img/64f5139fba3091160df35ec70988d703_24.png)

`clear` 属性指定元素的哪一侧不允许出现浮动元素。我们为页脚添加 `clear: both;`，表示其左右两侧都不允许有浮动元素。

![](img/64f5139fba3091160df35ec70988d703_26.png)

```css
#footer {
    clear: both;
    /* 其他样式 */
}
```

![](img/64f5139fba3091160df35ec70988d703_28.png)

这样，页脚就会清除浮动布局的规则，重新从新的一行开始排列。

![](img/64f5139fba3091160df35ec70988d703_30.png)

![](img/64f5139fba3091160df35ec70988d703_32.png)

![](img/64f5139fba3091160df35ec70988d703_34.png)

## 使用百分比实现自适应宽度

![](img/64f5139fba3091160df35ec70988d703_36.png)

使用固定像素宽度（如 `800px`）的布局在不同尺寸的屏幕上可能表现不佳。我们可以使用百分比宽度来创建自适应布局。

![](img/64f5139fba3091160df35ec70988d703_38.png)

例如，我们希望侧边栏占宽度的20%，主内容区占80%，并且总宽度填满浏览器：

![](img/64f5139fba3091160df35ec70988d703_40.png)

```css
#sidebar {
    width: 20%;
    float: left;
}

![](img/64f5139fba3091160df35ec70988d703_42.png)

![](img/64f5139fba3091160df35ec70988d703_44.png)

#content-box {
    width: 80%;
    float: left;
}
```

![](img/64f5139fba3091160df35ec70988d703_46.png)

**注意**：并排浮动的两个元素的宽度百分比之和不能超过100%，否则第二个元素会被“挤”到下一行。

![](img/64f5139fba3091160df35ec70988d703_48.png)

使用百分比布局的好处是，当调整浏览器窗口大小时，布局会按比例自动适应。

![](img/64f5139fba3091160df35ec70988d703_50.png)

![](img/64f5139fba3091160df35ec70988d703_52.png)

本节课中我们一起学习了如何使用 `div` 和 CSS 进行基础的网页布局。核心要点包括：
1.  使用多个 `div` 划分网页结构。
2.  通过 `#id` 选择器为特定 `div` 添加样式。
3.  使用 **`float: left;`** 属性实现元素的横向排列。
4.  使用 **`clear: both;`** 属性清除浮动对后续元素的影响。
5.  使用百分比宽度（如 `width: 20%;`）创建自适应布局。
