# 量化交易入门：第1章：什么是量化交易？ 🏇💹

![](img/26864ca479264abc87327bf7119e5165_1.png)

在本节课中，我们将要学习量化交易的核心概念。我们将通过一个生动的赛马比喻，帮助你理解量化交易的基本思想、关键术语以及一个实际的应用场景。

## 到底什么是量化交易

![](img/26864ca479264abc87327bf7119e5165_3.png)

想象一下，你经营着一个拥有100匹马的赛马场。

![](img/26864ca479264abc87327bf7119e5165_5.png)

市面上有两种赛马游戏：一种是比赛谁跑得快，另一种是比赛谁跑得慢。

你面临的问题是：如何挑选参赛的马匹，才能让你获奖的几率最高？与其花费大量时间去细致了解每一匹马的性能，不如建立一个自动化的规则。

![](img/26864ca479264abc87327bf7119e5165_7.png)

例如，你认为腿长的马跑得更快。

![](img/26864ca479264abc87327bf7119e5165_9.png)

那么，你就可以按照腿长这个指标，为所有马匹排序。排名前50的马去参加“比快”的赛道，排名后50的马去参加“比慢”的赛道。整个过程全部由代码自动操作。

## 从赛马到股市

将上述思维应用到股市中：市场上有大约3000只股票。你灵机一动，按照某个规则（例如，市盈率低）给所有股票排序。

![](img/26864ca479264abc87327bf7119e5165_11.png)

排名靠前的若干只股票，你选择“做多”（买入，预期上涨）。排名靠后的若干只股票，你选择“做空”（卖出，预期下跌）。

**这个自动化的、基于规则的交易思维，就是“量化”。而你设想的那个排序规则（例如“腿长”或“低市盈率”），就被称为“阿尔法（Alpha）”。**

![](img/26864ca479264abc87327bf7119e5165_13.png)

阿尔法的核心是寻找能够**预测**股票未来表现的信号或规则。在量化模型中，这通常体现为一个公式或函数，用于计算每只股票的“得分”：

`股票得分 = f(因子1， 因子2， ...)`

例如，一个简单的阿尔法规则可以是：
`得分 = 市盈率的倒数 * 0.7 + 近期涨幅 * 0.3`

然后根据得分进行排序和交易。

## 量化顾问的机会

上一节我们介绍了量化交易的核心是寻找“阿尔法”。本节中我们来看看一个将这种能力转化为实际收益的机会。

“挖矿”（此处为代称）是一家跨国量化公司。他们每年在全球招募几次“量化顾问”，这些顾问的任务就是为公司寻找有效的“阿尔法”因子。

目前，该公司为推荐人和被推荐人设立了一项激励，金额大约为1400元。

成为顾问的收益结构如下：
*   **日常收益**：每天根据你提交的阿尔法因子的质量，以及你账号的权重，每个阿尔法会获得0到400元不等的回报。
*   **季度奖金**：每个季度，公司会对全球顾问进行排名，并发放季度奖金，奖金范围大约在1400元到14万元之间，分为多个梯度。
*   **比赛奖金**：公司会定期举办全球性比赛，比赛过程与日常寻找阿尔法类似，优胜者将获得额外奖金。

在实际操作中，初期（约一两个月）由于阿尔法质量不高和账号权重较低，月收益可能在1000到3000元。但度过这个阶段后，随着阿尔法质量提升并叠加季度奖金，月收益有机会达到上万元甚至更高。一位资深顾问的收益增长曲线可能呈现出指数增长的趋势。

总结来说：入门获取基础收益相对简单，但要获得高额回报则依赖于个人能力的提升。

## 兼职的投入与实操

这个兼职需要耗费大量时间吗？实际上，许多人每天只投入约20分钟。繁重的计算工作由云端服务器（云电脑）承担。

以下是典型的工作流程：
1.  你将设置好的策略代码发送到云端服务器。
2.  云服务器24小时不间断地运行代码，回测并生成阿尔法因子。
3.  你从结果中筛选并提交有效的阿尔法。

值得注意的是，初始的策略代码模板通常由该公司提供的培训课程给出，极大降低了入门门槛。当前参与人数不多，竞争尚不激烈。

## 重要注意事项与流程

关于之前提到的1400元新人激励，需要满足特定条件才能获得，并非简单报名即可。

以下是关键步骤和条件：
*   **填写推荐码**：报名时必须填写有效的推荐码，否则无法获得激励。
*   **完成学习**：需要参加公司安排的四节直播课程。
*   **坚持提交**：需要有累计约十天的阿尔法因子提交记录。

这些要求并不难达成。按照课程学习后，你将会积累大量阿尔法因子，每天甚至可能提交不完（每日有提交上限）。提出这一点，是为了提醒希望获得激励的朋友需要坚持完成前期流程。

此外，本次报名可能包含一道测试题和简历提交。对于初学者，题目可能看起来陌生，但在了解基础知识后会变得简单。你可以通过加入社群与大家共同探讨。

“挖矿”公司的操作非常正规：
*   成为顾问需要签署正式的顾问合同。
*   每日收益透明，在官方网站可实时查看。
*   报酬发放前，公司会代为处理税务事宜。

对于有志于进入金融行业的人而言，成为一名优秀的顾问，未来还有可能获得该公司的全职工作机会。
