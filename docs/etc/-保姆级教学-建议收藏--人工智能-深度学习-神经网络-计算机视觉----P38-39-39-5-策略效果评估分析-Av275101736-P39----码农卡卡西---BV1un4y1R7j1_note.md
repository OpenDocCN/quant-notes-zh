1.  **中性化函数参数缺失**：在调用中性化函数时，需要传入正确的参数，包括待处理的DataFrame和指标列名。
    ```python
    # 错误示例：缺少指标列名参数
    result = neutralize(df, factor_column)
    # 正确示例
    result = neutralize(df, factor_column, 'target_column_name')
    ```

2.  **模型未训练**：在使用线性回归等模型进行中性化处理前，必须先调用 `fit` 方法训练模型。
    ```python
    model = LinearRegression()
    model.fit(X, y)  # 必须先训练
    predictions = model.predict(X_new)
    ```

3.  **函数名拼写错误**：例如，将 `quantile` 误写为 `qantile`。
    ```python
    # 错误拼写
    df['quantile'] = df.groupby('date')['factor'].qantile(0.3)
    # 正确拼写
    df['quantile'] = df.groupby('date')['factor'].quantile(0.3)
    ```

4.  **变量名未定义**：在循环或函数中使用了未声明的变量。
    ```python
    # 错误示例：`stock_list` 未定义
    for stock in delete_list:
        order_target_percent(stock, 0)
    # 正确示例：确保 `delete_list` 已正确定义并赋值
    delete_list = current_hold - new_pool
    for stock in delete_list:
        order_target_percent(stock, 0)
    ```

策略回测完成后，平台会生成详细的绩效报告。我们需要关注以下几个核心评估指标：

![](img/c5ede77285026877dea53ff3f20c38b3_107.png)

![](img/c5ede77285026877dea53ff3f20c38b3_109.png)

![](img/c5ede77285026877dea53ff3f20c38b3_111.png)

![](img/c5ede77285026877dea53ff3f20c38b3_113.png)

*   **累计收益率**：策略在整个回测期间的总收益。公式为 `(最终净值 - 初始净值) / 初始净值`。
*   **年化收益率**：将累计收益率折算成年化的收益，便于与其他策略或基准比较。
*   **夏普比率**：衡量每承受一单位总风险，会产生多少超额报酬。公式为 `(策略年化收益率 - 无风险利率) / 策略年化波动率`。该值越高，说明风险调整后的收益越好。
*   **最大回撤**：在选定周期内，从任一高点净值回落至最低点的最大幅度。这是衡量策略下行风险的关键指标。
*   **阿尔法与贝塔**：
    *   **贝塔**：衡量策略收益相对于市场基准收益的波动性。贝塔为1表示与市场波动同步。
    *   **阿尔法**：衡量策略相对于市场基准的超额收益。正的阿尔法表明策略战胜了市场。
