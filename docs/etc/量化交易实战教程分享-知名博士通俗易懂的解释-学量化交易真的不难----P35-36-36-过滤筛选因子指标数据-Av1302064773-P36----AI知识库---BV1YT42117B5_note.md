在构建股票池时，我们通常需要剔除一些不符合交易条件的股票，例如停牌股、ST股和刚上市的新股。以下是实现这一筛选过程的步骤。

首先，我们需要编写一个过滤函数，其核心逻辑是遍历传入的股票列表，并根据特定条件保留符合条件的股票。

![](img/838e1052a1bb622495173e7ed39a5ff7_34.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_36.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_38.png)

以下是三个具体的过滤条件及其实现：

![](img/838e1052a1bb622495173e7ed39a5ff7_40.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_42.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_44.png)

1.  **过滤停牌股票**：使用API判断股票是否处于全面停牌状态。
    ```python
    def filter_suspended_stocks(stock_list):
        filtered_list = []
        for stock in stock_list:
            if not is_suspended(stock):  # 假设 is_suspended 是判断停牌的API函数
                filtered_list.append(stock)
        return filtered_list
    ```

![](img/838e1052a1bb622495173e7ed39a5ff7_46.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_48.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_50.png)

2.  **过滤ST股票**：使用API判断股票是否为ST股。
    ```python
    def filter_st_stocks(stock_list):
        filtered_list = []
        for stock in stock_list:
            if not is_st(stock):  # 假设 is_st 是判断ST股的API函数
                filtered_list.append(stock)
        return filtered_list
    ```

![](img/838e1052a1bb622495173e7ed39a5ff7_52.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_54.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_56.png)

3.  **过滤新股**：通过判断股票上市天数是否超过一定阈值（例如180天）来过滤新股。
    ```python
    def filter_new_stocks(stock_list, min_days=180):
        filtered_list = []
        for stock in stock_list:
            if get_listed_days(stock) > min_days:  # 假设 get_listed_days 是获取上市天数的函数
                filtered_list.append(stock)
        return filtered_list
    ```

# 构建查询语句，获取市净率(pb_ratio)和市值(market_cap)
query = query(fundamentals.eod_derivative_indicator.pb_ratio,

查询返回的数据格式可能是一个以指标为行、股票为列的 `DataFrame`。但为了后续分析方便，我们通常希望数据是以股票为行、指标为列。

![](img/838e1052a1bb622495173e7ed39a5ff7_118.png)

![](img/838e1052a1bb622495173e7ed39a5ff7_120.png)

# 转置数据框，使股票成为行，指标成为列
transposed_data = fundamental_data.T
