## 概述 📋

在本节课中，我们将学习如何为统计分析准备数据。我们将基于股票价格的前两天变化情况，通过简单的统计方法来预测第三天的涨跌趋势。核心思路是：统计历史数据中，在“前两天涨跌”的各种组合下，第三天出现“涨”或“跌”的频率，并基于出现概率更高的结果来制定交易策略。本节的重点是使用Python和Pandas库来完成数据的分组与统计。

## 核心思路与数据需求 💡

![](img/fab07557a7ce4de995c90678c99bf1de_2.png)

上一节我们介绍了基于技术指标的交易策略。本节中，我们来看看一种更直观的统计分析方法。

![](img/fab07557a7ce4de995c90678c99bf1de_4.png)

这种方法听起来更简单。其核心是：我们已有“前一天”和“前两天”的价格变化指标。现在，我们想基于历史频率进行统计。例如：
*   如果前两天都是下跌，统计一下第三天是下跌的可能性大，还是上涨的可能性大。
*   如果前一天下跌、前一天上涨，再统计接下来下跌或上涨的可能性。

![](img/fab07557a7ce4de995c90678c99bf1de_6.png)

![](img/fab07557a7ce4de995c90678c99bf1de_8.png)

以此类推，根据“前两天”和“前一天”的涨跌状态（共约四种组合），分别统计在每种组合下，当天实际涨跌的分布情况。我们基于历史数据中概率更高的结果来制定买卖策略。

![](img/fab07557a7ce4de995c90678c99bf1de_10.png)

需要强调的是，这种方法较为传统，此处主要用于让大家熟悉数据处理和统计流程。

## 第一步：数据标记与转换 🔧

![](img/fab07557a7ce4de995c90678c99bf1de_18.png)

![](img/fab07557a7ce4de995c90678c99bf1de_20.png)

在Python中实现上述需求，第一步是统计“前两天”和“前一天”的数据情况，并按“涨/跌”进行区间划分。

![](img/fab07557a7ce4de995c90678c99bf1de_22.png)

![](img/fab07557a7ce4de995c90678c99bf1de_24.png)

具体来说，我们需要判断每个时间点的“前两天”和“前一天”指标是大于0（涨）还是小于0（跌），并将判断结果用新的列标记出来。

![](img/fab07557a7ce4de995c90678c99bf1de_26.png)

以下是实现这一步骤的代码：

```python
def create_binary_data(data):
    """
    将特征数据转换为二元的涨跌标记（1为涨，0为跌）。
    """
    # 定义需要转换的列名，这里使用之前生成的特征列
    cols = ['L1', 'L2']  # 假设L1代表前一天变化，L2代表前两天变化
    for col in cols:
        # 创建新列名
        new_col_name = col + '_B'
        # 判断原数据列是否大于0，大于0标记为1（涨），否则为0（跌）
        data[new_col_name] = (data[col] > 0).astype(int)

# 调用函数处理数据
create_binary_data(data)
# 查看处理后的部分数据
print(data[['L1_B', 'L2_B']].head())
```

![](img/fab07557a7ce4de995c90678c99bf1de_28.png)

执行以上代码后，数据中会新增`L1_B`和`L2_B`两列。其中，`1`代表对应周期上涨，`0`代表下跌。例如，第一行数据`[1, 0]`表示：前一天涨了(`L1_B=1`)，前两天跌了(`L2_B=0`)。

## 第二步：整合实际走势数据 📊

![](img/fab07557a7ce4de995c90678c99bf1de_34.png)

为了进行统计，我们还需要一列代表“当天”实际涨跌情况的数据。原始数据中通常包含一个名为`direction`的列，它记录了价格的实际变动方向（例如，1代表上涨，-1代表下跌，0代表不变）。

![](img/fab07557a7ce4de995c90678c99bf1de_36.png)

因此，我们的分析将基于这三个字段进行：
*   `L2_B`: 前两天涨跌标记
*   `L1_B`: 前一天涨跌标记
*   `direction`: 当天实际涨跌方向

![](img/fab07557a7ce4de995c90678c99bf1de_38.png)

![](img/fab07557a7ce4de995c90678c99bf1de_40.png)

## 第三步：分组统计频率 📈

![](img/fab07557a7ce4de995c90678c99bf1de_42.png)

![](img/fab07557a7ce4de995c90678c99bf1de_44.png)

现在，我们想要分组统计各种情况出现的次数。例如：在`L2_B=0`且`L1_B=0`（即前两天都跌）的情况下，`direction`分别等于1、0、-1的次数各是多少。

![](img/fab07557a7ce4de995c90678c99bf1de_46.png)

Pandas库提供了一个非常便捷的方法`groupby`来完成这个任务。

![](img/fab07557a7ce4de995c90678c99bf1de_48.png)

以下是分组统计的代码：

![](img/fab07557a7ce4de995c90678c99bf1de_50.png)

![](img/fab07557a7ce4de995c90678c99bf1de_52.png)

```python
# 按照‘L2_B‘, ‘L1_B‘, ‘direction‘进行分组，并计算每组的大小（即行数）
grouped = data.groupby(['L2_B', 'L1_B', 'direction']).size()

![](img/fab07557a7ce4de995c90678c99bf1de_54.png)

# 打印分组统计结果
print(grouped)
```

![](img/fab07557a7ce4de995c90678c99bf1de_56.png)

![](img/fab07557a7ce4de995c90678c99bf1de_58.png)

执行这段代码后，我们将得到一个类似下表的结果：

![](img/fab07557a7ce4de995c90678c99bf1de_62.png)

这个结果清晰地展示了历史数据中所有“前两日状态”与“当日实际结果”的组合及其出现频次。例如，第一行表示：在历史中，出现“前两天跌(`L2_B=0`)、前一天跌(`L1_B=0`)”的情况后，当天实际下跌(`direction=-1`)的次数是105次。

## 总结 🎯

![](img/fab07557a7ce4de995c90678c99bf1de_70.png)

本节课中我们一起学习了为统计分析准备数据的关键步骤：
1.  **数据标记**：我们将连续的价格变化指标（`L1`, `L2`）转换为二元的涨跌标记（`L1_B`, `L2_B`），便于分类统计。
2.  **数据整合**：我们确认了用于统计的目标变量——当天的实际价格方向（`direction`）。
3.  **分组统计**：我们利用Pandas的`groupby`和`size`方法，高效地统计了在各种“历史状态”组合下，“当日结果”的分布频率。

![](img/fab07557a7ce4de995c90678c99bf1de_72.png)

得到这个统计结果后，我们就可以分析在特定历史模式下（如“前两日连续下跌”），次日最可能出现的走势，从而为制定简单的概率化交易策略提供数据基础。在接下来的课程中，我们将基于此统计结果进行策略的回测与评估。
