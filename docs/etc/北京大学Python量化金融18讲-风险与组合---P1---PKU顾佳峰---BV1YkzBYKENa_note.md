## 理论基础：夏普比率与风险

现代投资组合理论的基础由哈里·马可维茨的资产选择理论奠定。在此理论框架下，夏普比率是一个衡量投资绩效的重要指标。

![](img/d6fc38848a8ff6ae6d8cbf985d1606b5_3.png)

夏普比率的计算公式为：
**Sharpe Ratio = (Rp - Rf) / σp**
其中，`Rp` 是投资组合的预期收益率，`Rf` 是无风险利率，`σp` 是投资组合收益率的标准差（即风险）。

威廉·夏普于1966年提出了这一比率，并因此获得了诺贝尔经济学奖。在图形上，夏普比率代表了从无风险利率点出发，与有效前沿曲线相切的直线的斜率。

## 无风险与有风险情景对比

以下是两种情景下计算结果的对比分析。

**无风险情景 (Rf=0%)**
*   **资产配置权重 (W1, W2, W3):** 0.70, 0.20, 0.10
*   **组合预期收益率:** 10.5%
*   **组合风险 (标准差):** 0.096
*   **夏普比率:** 1.0926

**有风险情景 (Rf=3%)**
*   **资产配置权重 (W1, W2, W3):** 0.45, 0.35, 0.20
*   **组合预期收益率:** 15.25%
*   **组合风险 (标准差):** 0.15
*   **夏普比率:** 0.8199

通过对比，我们可以得出以下核心结论：
1.  **资产配置变化**：当引入正的无风险利率后，风险较低的资产（如股票1）配置权重下降，而风险较高但预期收益也更高的资产（如股票2和股票3）配置权重上升。
2.  **收益与风险关系**：有风险情景下的投资组合预期收益率（15.25%）显著高于无风险情景（10.5%），这符合“高收益伴随高风险”的原则，其组合风险也从0.096上升至0.15。
3.  **夏普比率变化**：有风险情景下的夏普比率（0.8199）低于无风险情景（1.0926）。这是因为在图形上，切线的起点从(0,0)点移到了(0, 0.03)点，导致与有效前沿相切的直线斜率变得更为平缓。

在有效前沿图中：
*   无风险情景下，切线从原点(0,0)出发。
*   有风险（例如3%）情景下，切线从点(0, 0.03)出发。
*   随着无风险利率的升高，这个起点会不断上移，导致切线越来越平缓，即夏普比率逐渐降低。这直观地展示了风险成本对投资组合最优选择的影响。
