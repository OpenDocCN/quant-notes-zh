## 数据预处理与指标查询

在设定好基础股票池后，我们需要对池中的股票进行预处理，即获取它们每日的财务数据。这就像数据挖掘，我们需要从海量指标中筛选出有用的信息。

以下是实现数据查询和筛选的步骤：

1.  **查阅帮助文档**：所有可用的财务指标和查询方法都在官方帮助文档中。你需要根据需求在文档中查找对应的指标字段和函数用法。
2.  **使用查询函数**：我们将使用一个特定的查询函数来获取财务数据。该函数的核心是构建一个`query`对象，用于指定要查询的指标。
3.  **构建查询语句**：在我们的例子中，我们想查询“营业总收入”这个财务指标。你可以在财务数据文档中找到这个字段的确切名称。
4.  **应用过滤器**：查询语句需要指定在哪些股票上执行。我们使用`filter`条件来限定只查询属于我们股票池（即沪深300）的股票。
5.  **排序与限制**：查询到所有数据后，我们根据“营业总收入”进行降序排列，然后使用`limit`函数只选取排名前10的股票。

将以上步骤整合，核心代码如下：

```python
# 在 before_trading 函数中执行筛选
query = query(指标.营业总收入)
    .filter(指标.股票代码.in_(self.pool)) # 过滤出股票池中的股票
    .order_by(指标.营业总收入.desc()) # 按营收降序排列
    .limit(10) # 只取前10名

self.selected_stocks = get_fundamentals(query) # 执行查询，结果存入上下文
print(self.selected_stocks) # 打印查看结果
```

## 调试与验证

在初次运行代码时，可能会遇到问题。例如，打印结果为空，这通常是因为股票池的名称指定不正确。

![](img/56ae895052c639e81640a9af1990f48f_21.png)

![](img/56ae895052c639e81640a9af1990f48f_22.png)

1.  **检查股票池代码**：确保用于过滤的股票池代码与初始化时设定的完全一致。沪深300指数在系统中的标准代码可能是`000300.SH`，而非简单的“沪深300”。你需要从系统的指数列表中查找并复制正确的代码。
2.  **修正代码**：将初始化股票池和过滤条件中的代码都修正为正确的标准代码。
3.  **重新回测**：修正后重新运行回测，观察输出日志。此时应该能正确打印出每一天筛选出的股票列表。
