# Python实战量化交易理财系统：2-2：如何通过Python API获取股票数据 📈

在本节课中，我们将学习如何通过Python API获取股票数据。这是构建量化交易系统的基础步骤，我们将介绍几种数据获取方式，并重点讲解如何使用免费的量化平台接口来获取实时和历史股票行情数据。

获取股票数据主要有三种方式。

第一种是通过网络爬虫，例如从东方财富等财经网站抓取数据。这种方式需要自行编写和维护爬虫程序，过程较为耗时费力。

![](img/b54e2ce8860165a6787c4274728d630b_1.png)

第二种是通过付费的数据服务商，例如Wind、彭博或某些炒股软件提供的专业接口。这类服务通常需要支付每年数万元的使用费。

![](img/b54e2ce8860165a6787c4274728d630b_3.png)

第三种是通过一些量化交易平台或开源代码库提供的免费API接口。这些接口通常提供试用期或一定额度的免费数据调用，适合个人学习和初期开发。

本教程将采用第三种方式，因为它免费且易于使用。

![](img/b54e2ce8860165a6787c4274728d630b_5.png)

![](img/b54e2ce8860165a6787c4274728d630b_7.png)

我们将使用聚宽（JoinQuant）社区提供的数据服务。首先，需要提交试用申请以获得账号。

以下是开通步骤：
1.  访问聚宽官网并找到数据试用申请页面。
2.  填写必要的个人信息，确保手机号正确。
3.  获取并输入手机验证码。
4.  提交申请后，系统会提供一个账号，默认密码通常是该账号的后六位。
5.  使用获得的账号和密码登录聚宽平台。

成功登录后，即可在试用期内免费调用其全部基础数据。

## 获取单支股票数据 📊

![](img/b54e2ce8860165a6787c4274728d630b_47.png)

环境配置妥当后，我们就可以开始获取数据了。首先，学习如何获取单支股票的行情数据。

聚宽数据提供了`get_price`接口来获取历史行情。以下是一个基础示例：

![](img/b54e2ce8860165a6787c4274728d630b_49.png)

```python
# 获取股票000001.XSHE（平安银行）在指定时间前的4条分钟线数据
data = get_price('000001.XSHE', end_date='2015-01-30 14:00:00', count=4, frequency='minute', fields=['open', 'close', 'high', 'low', 'volume', 'money'])
print(data)
```

![](img/b54e2ce8860165a6787c4274728d630b_51.png)

![](img/b54e2ce8860165a6787c4274728d630b_53.png)

对代码中的参数解释如下：
*   `‘000001.XSHE’`：股票代码。`XSHE`代表深圳证券交易所，`XSHG`代表上海证券交易所。
*   `end_date`：获取数据的截止时间。
*   `count`：获取的数据条数。
*   `frequency`：数据频率，例如`‘minute’`（分钟）、`‘daily’`（日线）。
*   `fields`：需要获取的字段列表，如开盘价(`open`)、收盘价(`close`)等。

我们可以修改参数来获取不同频率和时段的数据。例如，获取最近10个交易日的日线数据：

![](img/b54e2ce8860165a6787c4274728d630b_55.png)

```python
data = get_price('000001.XSHE', end_date='2021-05-30', count=10, frequency='daily', fields=['open', 'close', 'high', 'low', 'volume', 'money'])
print(data)
```

![](img/b54e2ce8860165a6787c4274728d630b_57.png)

获取到数据后，建议与东方财富等权威财经网站的数据进行比对，以验证数据的准确性。

## 获取多支股票数据 📑

![](img/b54e2ce8860165a6787c4274728d630b_59.png)

掌握了单支股票的获取方法后，接下来我们学习如何批量获取多支股票的信息。这通常需要先获得市场上所有股票的代码列表。

![](img/b54e2ce8860165a6787c4274728d630b_61.png)

使用`get_all_securities`接口可以获取平台支持的所有股票基础信息。

```python
# 获取所有股票信息列表
stock_list = get_all_securities(['stock'])
print(stock_list.head()) # 打印前5条数据查看
print("股票总数：", len(stock_list))
```

得到股票代码列表后，便可以循环遍历，批量获取每支股票的行情数据。以下是一个结合延时（防止请求过快）的示例：

```python
import time
from jqdatasdk import *
auth('你的账号', '你的密码')

![](img/b54e2ce8860165a6787c4274728d630b_63.png)

# 获取所有A股股票代码
stock_list = get_all_securities(['stock']).index.tolist()

![](img/b54e2ce8860165a6787c4274728d630b_65.png)

for stock_code in stock_list[:5]:  # 这里仅遍历前5支作为演示
    print(f"正在获取 {stock_code} 的数据...")
    # 获取该股票最近5个交易日的日线数据
    price_data = get_price(stock_code, end_date='2021-05-30', count=5, frequency='daily')
    print(price_data)
    time.sleep(3)  # 暂停3秒，避免请求频率过高
```

![](img/b54e2ce8860165a6787c4274728d630b_67.png)

这个脚本会依次获取并打印多支股票的历史行情数据，是构建股票数据池的基础。

![](img/b54e2ce8860165a6787c4274728d630b_69.png)
