## 算术运算符 ➕➖✖️➗

算术运算符用于执行基本的数学运算，如加、减、乘、除。

以下是Python中主要的算术运算符及其示例：

*   **`+`**：加法。例如：`a + b`。
*   **`-`**：减法。例如：`a - b`。
*   **`*`**：乘法。例如：`a * b`。
*   **`/`**：除法。例如：`a / b`。
*   **`%`**：取模（求余数）。返回除法运算的余数。例如：`10 % 3` 的结果是 `1`。
*   **`**`**：幂运算。返回x的y次幂。例如：`2 ** 3` 的结果是 `8`。
*   **`//`**：取整除。返回商的整数部分，向下取整。例如：`9 // 2` 的结果是 `4`。

**核心概念示例：**
```python
# 取模运算常用于判断奇偶数
number = 9
if number % 2 == 0:
    print(f"{number} 是偶数")
else:
    print(f"{number} 是奇数")
# 输出：9 是奇数
```

## 比较运算符 ⚖️

比较运算符用于比较两个值，并返回一个布尔值（`True` 或 `False`）。

以下是Python中的比较运算符：

![](img/bd56fd1520c6e894985fca07a4e005fa_15.png)

*   **`==`**：等于。判断两个值是否相等。**注意**：单个等号 `=` 是赋值，双等号 `==` 才是比较。
*   **`!=`**：不等于。
*   **`>`**：大于。
*   **`<`**：小于。
*   **`>=`**：大于等于。
*   **`<=`**：小于等于。

**核心概念示例：**
```python
a = 10
b = 20
print(a == b)  # 输出：False
print(a != b)  # 输出：True
print(a < b)   # 输出：True
```

## 逻辑运算符 🔗

逻辑运算符用于组合多个条件，进行更复杂的逻辑判断。

Python提供了三个逻辑运算符：

*   **`and`**：逻辑“与”。只有当**所有**条件都为 `True` 时，整个表达式才返回 `True`。
*   **`or`**：逻辑“或”。只要**任意一个**条件为 `True`，整个表达式就返回 `True`。
*   **`not`**：逻辑“非”。对条件取反，`True` 变 `False`，`False` 变 `True`。

**核心概念示例：**
```python
age = 25
has_license = True

# 使用 and：两个条件必须同时满足
can_drive = age >= 18 and has_license
print(can_drive)  # 输出：True

# 使用 or：满足任意一个条件即可
is_special = (age < 12) or (age > 65)
print(is_special) # 输出：False

# 使用 not：取反
is_minor = not (age >= 18)
print(is_minor)   # 输出：False
```

**关于混合运算的优先级**：当 `and` 和 `or` 混合使用时，`or` 的优先级低于 `and`。但为了避免混淆，最清晰的做法是使用括号 `()` 来明确指定运算顺序。

## 赋值运算符 💾

![](img/bd56fd1520c6e894985fca07a4e005fa_23.png)

赋值运算符用于为变量赋值。除了基本的赋值，还有将运算与赋值结合起来的复合赋值运算符。

以下是常见的赋值运算符：

*   **`=`**：基本赋值。例如：`x = 5`。
*   **`+=`**：加后赋值。`x += 3` 等价于 `x = x + 3`。
*   **`-=`**：减后赋值。`x -= 3` 等价于 `x = x - 3`。
*   **`*=`**：乘后赋值。`x *= 3` 等价于 `x = x * 3`。
*   **`/=`**：除后赋值。`x /= 3` 等价于 `x = x / 3`。
*   **`%=`**：取模后赋值。`x %= 3` 等价于 `x = x % 3`。

**核心概念示例：**
```python
counter = 1
counter += 1  # 等同于 counter = counter + 1
print(counter) # 输出：2
```

## 成员运算符 🔍

成员运算符用于测试一个元素是否存在于某个序列中，如字符串、列表、元组等。

![](img/bd56fd1520c6e894985fca07a4e005fa_27.png)

Python的成员运算符有两个：

*   **`in`**：如果在指定的序列中找到值，则返回 `True`。
*   **`not in`**：如果在指定的序列中没有找到值，则返回 `True`。

**核心概念示例：**
```python
fruits = ['apple', 'banana', 'orange']

# 检查元素是否在列表中
print('banana' in fruits)     # 输出：True
print('grape' not in fruits)  # 输出：True

# 检查字符是否在字符串中
name = "Alex"
print('A' in name)   # 输出：True
print('le' in name)  # 输出：True
print('xy' in name)  # 输出：False
```

**注意**：成员运算符不能直接用于测试数字是否在一个整数中，但可以用于测试数字是否在一个数字列表或范围内。
