## 量化择时的定义

量化择时是利用数量化的方法，通过对各种宏观微观指标的量化分析，试图找到影响大盘走势的关键信息，并对未来进行预测。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_5.png)

通俗来说，量化择时就是通过量化的方式来判断买点和卖点。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_7.png)

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_9.png)

## 量化择时的常用方法

量化择时主要有两种常用方法：**趋势量化择时**和**市场情绪量化择时**。

上一节我们介绍了量化择时的定义，本节中我们来看看这两种方法的具体区别。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_11.png)

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_13.png)

### 趋势量化择时

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_15.png)

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_17.png)

趋势量化择时的基本思想主要来源于技术分析。技术分析有一个基本理论，认为**趋势是存在延续性**的。通过分析当前和历史的趋势，可以预测未来。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_19.png)

以下是常见的趋势择时指标：
*   **MA（均线）**
*   **MACD（金叉）**
*   **DMA** 等

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_21.png)

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_23.png)

### 市场情绪量化择时

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_25.png)

市场情绪量化择时的方法与趋势择时完全不同。它的假设是：**当市场情绪热烈时，大盘会上涨；当市场情绪低迷时，大盘会下跌**。因此，它通过判断投资者的热情程度来判断大盘趋势。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_27.png)

以下是市场情绪择时常用的分析维度或数据来源：
*   **调查问卷**：对核心投资人或随机股民进行抽样调查。
*   **开户人数**：对比不同时间段市场新增开户人数。
*   **搜索指数**：分析百度、谷歌等平台对特定关键词的搜索热度。
*   **报告评级**：关注专业机构对股票发布的买入、持有、卖出等评级。
*   **融资融券数据**：观察市场整体的杠杆使用情况。
*   **舆情数据**：分析新闻稿、分析师文章等文本信息所反映的市场情绪。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_29.png)

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_31.png)

> **注意**：本系列课程主要介绍**趋势量化择时**。市场情绪择时需要建立在宏观或自建数据库基础上，暂不涉及。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_33.png)

## 技术指标理论

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_35.png)

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_37.png)

既然趋势量化择时源于技术分析，那么我们需要了解技术分析赖以成立的基石。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_39.png)

技术分析的所有指标和理论均基于以下三项市场假设：
1.  **市场行为涵盖一切信息**。
2.  **价格沿趋势移动**。
3.  **历史会重演**。

这三项假设缺一不可。如果市场行为不包含有效信息、价格无法预测、或历史毫无参考价值，那么技术分析的理论基础将不复存在。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_41.png)

### 技术指标的定义

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_43.png)

技术指标是技术分析中使用最多的一种方式。它试图通过建立**数学模型**和**完整的数学计算公式**，来衡量市场行为的各个方面，并最终计算出能够反映证券市场特定方面的数值（即技术指标值），用以辅助判断买点和卖点。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_45.png)

可以通俗地理解为：技术指标是将金融市场的信息“公式化”，从而预测未来。例如，一个简单的移动平均线（MA）计算公式为：
`MA(N) = (C1 + C2 + ... + CN) / N`
其中，`C`代表收盘价，`N`代表计算周期。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_47.png)

## 技术指标的分类

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_49.png)

技术指标种类繁多，根据其功能和特点，主要可分为以下四类。

### 1. 趋向指标

趋向指标是用于识别和追踪趋势的图形类指标。它的核心特点是**不试图预测价格的顶部和底部**，只反映趋势的方向和强度。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_51.png)

以下是常见的趋向指标：
*   MA（移动平均线）
*   MACD（指数平滑异同移动平均线）

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_53.png)

### 2. 反趋向指标（震荡指标）

反趋向指标，或称震荡指标，其理念与趋向指标相反。它认为市场是波动的，存在转折点。因此，这类指标**具有强烈的捕捉价格顶部和底部的意图**，重点在于判断市场的转折点。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_55.png)

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_57.png)

以下是常见的反趋向指标：
*   KDJ（随机指标）
*   RSI（相对强弱指标）

### 3. 压力支撑指标（通道指标）

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_59.png)

压力支撑指标，又称通道指标。它通过绘制**顶部压力线**和**底部支撑线**形成一个价格通道，以此来反映行情可能遇到的阻力和支撑区域。

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_61.png)

以下是常见的压力支撑指标：
*   布林带（Bollinger Bands）
*   薛斯通道（Xue‘s Channel）

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_63.png)

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_65.png)

### 4. 量价指标

![](img/6df3a8c8aab1c823cc34ceb5a528b8c2_67.png)

量价指标专注于分析**成交量与价格变动之间的关系**。其假设是，成交量的变化能够预示未来价格的走势。这类指标不关心顶部和底部，只聚焦于量价互动。

以下是常见的量价指标：
*   OBV（能量潮指标）
*   VOL（成交量指标）

## 本章总结

本节课中我们一起学习了量化择时的核心概念。

首先，我们明确了量化择时的定义：**通过量化方法寻找买卖点**。

接着，我们探讨了量化择时的两大流派：基于技术分析的**趋势量化择时**和基于投资者心理的**市场情绪量化择时**。本课程将重点讲解前者。

然后，我们深入了解了趋势择时所依赖的**技术指标理论**，它建立在市场行为涵盖信息、价格沿趋势移动、历史会重演三大假设之上。技术指标的本质是一套可量化的数学模型。

最后，我们将技术指标分为四大类：反映趋势的**趋向指标**、捕捉转折的**反趋向指标**、标识阻力与支撑的**压力支撑指标**，以及专注量价关系的**量价指标**。
