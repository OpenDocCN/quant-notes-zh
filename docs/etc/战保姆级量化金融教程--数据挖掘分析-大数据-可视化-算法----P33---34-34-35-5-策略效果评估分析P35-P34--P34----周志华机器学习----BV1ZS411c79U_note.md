1.  **卖出不再符合条件的股票**：遍历需要删除的股票列表，使用 `order_target_percent` 函数将其持仓比例调整为0。
    ```python
    for stock in stocks_to_delete:
        order_target_percent(stock, 0)
    ```
2.  **买入新筛选出的股票**：遍历当前股票池中的所有股票，使用 `order_target_percent` 函数进行买入，并平均分配资金。
    ```python
    for stock in stock_list:
        order_target_percent(stock, 1.0 / len(stock_list))
    ```

在编写和运行策略代码时，难免会遇到错误。以下是我们在调试过程中发现并修正的几个关键问题：

![](img/18a5b05661338fe728158a3a36873c8b_69.png)

![](img/18a5b05661338fe728158a3a36873c8b_71.png)

![](img/18a5b05661338fe728158a3a36873c8b_73.png)

1.  **中性化函数参数缺失**：在调用中性化处理函数时，漏传了必要的参数（如因子名称）。修正后，确保传入完整的 `DataFrame` 和指标名称。
2.  **模型未训练**：在使用普通最小二乘法（OLS）进行中性化前，必须先调用 `fit()` 方法训练模型，然后才能计算残差。
    ```python
    model = sm.OLS(y, X).fit()
    resid = model.resid
    ```
3.  **函数名拼写错误**：例如，将 `quantile` 误写为 `qantile`，将 `list` 误写为 `lis` 等。仔细检查并修正拼写错误。

## 策略流程总结

本节课中我们一起学习了构建一个完整因子选股策略的流程：

![](img/18a5b05661338fe728158a3a36873c8b_119.png)

![](img/18a5b05661338fe728158a3a36873c8b_121.png)

![](img/18a5b05661338fe728158a3a36873c8b_123.png)

1.  **初始化与定时器设置**：在 `initialize` 函数中设定调仓频率（如每月）。
2.  **股票池过滤**：在 `before_trading_start` 中过滤掉不满足基本条件的股票（如ST股、停牌股）。
3.  **因子获取与处理**：查询所需因子数据（如市值、市净率），并进行必要的预处理（如中性化）。
4.  **因子筛选**：根据因子逻辑（如市净率越低越好）筛选出符合条件的股票，形成最终股票池。
5.  **调仓执行**：
    *   卖出持仓中不在新股票池的股票。
    *   买入新股票池中的所有股票，并平均分配资金。
6.  **效果评估**：通过回测结果分析策略的收益、风险等关键指标。

![](img/18a5b05661338fe728158a3a36873c8b_125.png)

![](img/18a5b05661338fe728158a3a36873c8b_127.png)

这个流程为我们提供了一个清晰的量化策略开发框架。需要注意的是，本教程仅用于回测演示和学习，实际市场应用需考虑更多复杂因素。
