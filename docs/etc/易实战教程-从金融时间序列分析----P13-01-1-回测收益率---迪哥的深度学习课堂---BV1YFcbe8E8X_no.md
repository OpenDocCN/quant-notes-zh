在本节课中，我们将要学习量化策略评估中最基础且核心的指标——回测收益率。理解这个指标是评估一个策略是否盈利的第一步。

![](img/59da25498ce6e8fb2592594f59a895ab_1.png)

当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以判断该策略的盈亏情况。然而，“赚了”或“赔了”只是一个定性的描述。我们需要更精确的定量指标来回答：具体赚了多少？与市场大盘相比有何优势？承担了哪些风险？与其他策略相比有何特点？这些问题的答案，就构成了策略的评估指标体系。

以下是本节课将要介绍的核心评估指标之一。

## 回测收益率

![](img/59da25498ce6e8fb2592594f59a895ab_3.png)

回测收益率是最基本、最常用的策略评估指标。它衡量的是，在回测周期内，策略最终获得的收益相对于初始本金的增长比例。

![](img/59da25498ce6e8fb2592594f59a895ab_5.png)

![](img/59da25498ce6e8fb2592594f59a895ab_7.png)

其计算公式如下：

![](img/59da25498ce6e8fb2592594f59a895ab_9.png)

**回测收益率 = (期末资产总值 - 期初资产总值) / 期初资产总值**

或者等价地：

![](img/59da25498ce6e8fb2592594f59a895ab_11.png)

![](img/59da25498ce6e8fb2592594f59a895ab_13.png)

**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

![](img/59da25498ce6e8fb2592594f59a895ab_15.png)

其中，`期末资产总值` 可以理解为策略最终卖出所有资产后的总金额 `P_t`，`期初资产总值` 则是初始投入的本金 `P_0`。

![](img/59da25498ce6e8fb2592594f59a895ab_17.png)

![](img/59da25498ce6e8fb2592594f59a895ab_19.png)

这个指标直观地告诉我们，经过一段时间的策略运行，我们的本金变成了多少。例如，一个4.5%的回测收益率意味着，相对于初始投入，我们总共赚取了4.5%的利润。

![](img/59da25498ce6e8fb2592594f59a895ab_21.png)

## 实战计算：用Python计算回测收益率

上一节我们介绍了回测收益率的概念和公式，本节中我们来看看如何用Python代码实际计算它。我们将使用一份示例股票数据来模拟策略回测后的资产净值走势。

![](img/59da25498ce6e8fb2592594f59a895ab_23.png)

首先，我们需要导入必要的工具包并加载数据。

```python
import pandas as pd
import matplotlib.pyplot as plt

![](img/59da25498ce6e8fb2592594f59a895ab_25.png)

![](img/59da25498ce6e8fb2592594f59a895ab_27.png)

# 设置中文字体，方便后续图表显示中文
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设第一列是日期
data = pd.read_csv('your_stock_data.csv', index_col=0)
data.index = pd.to_datetime(data.index)  # 确保索引是日期时间类型
print(data.head())
```

假设 `data[‘close’]` 列代表了我们策略在每一天的资产净值（可以理解为每日的“收盘价”）。我们的目标是计算从开始到结束，资产净值相对于起始点的增长情况。

以下是计算并可视化回测收益率过程的步骤：

第一步，计算每个时间点的资产净值相对于起始点的比值。

![](img/59da25498ce6e8fb2592594f59a895ab_29.png)

```python
# 计算每日净值相对于期初净值的比值
relative_value = data['close'] / data['close'].iloc[0]

![](img/59da25498ce6e8fb2592594f59a895ab_31.png)

# 绘制资产净值相对变化曲线
plt.figure(figsize=(10, 6))
relative_value.plot()
plt.title('资产净值相对变化曲线')
plt.xlabel('日期')
plt.ylabel('相对净值 (相对于期初)')
plt.grid(True)
plt.show()
```

![](img/59da25498ce6e8fb2592594f59a895ab_33.png)

这张图展示了从一开始（净值为1）到结束，我们的资产净值是如何变化的。如果曲线向上，表示盈利；如果向下，则表示亏损。

![](img/59da25498ce6e8fb2592594f59a895ab_35.png)

![](img/59da25498ce6e8fb2592594f59a895ab_37.png)

第二步，根据公式计算最终的回测收益率。

![](img/59da25498ce6e8fb2592594f59a895ab_39.png)

```python
# 计算最终的回测收益率
final_return = (data['close'].iloc[-1] / data['close'].iloc[0]) - 1

![](img/59da25498ce6e8fb2592594f59a895ab_41.png)

# 将结果整理成DataFrame，便于查看
result_df = pd.DataFrame({
    ‘回测收益率’: [final_return]
}, index=[‘策略表现’])
print(result_df)
```

![](img/59da25498ce6e8fb2592594f59a895ab_43.png)

![](img/59da25498ce6e8fb2592594f59a895ab_45.png)

执行这段代码，我们就可以得到该策略在回测周期内的具体收益率数值。这个数值是评估策略表现的起点。

![](img/59da25498ce6e8fb2592594f59a895ab_47.png)

## 总结

![](img/59da25498ce6e8fb2592594f59a895ab_49.png)

本节课中我们一起学习了量化策略评估的基础——回测收益率。我们明确了它的定义：策略在回测结束后，总资产相对于初始本金的增长率。我们掌握了其核心计算公式，并通过Python代码实战，一步步计算并可视化了一份示例数据的回测收益率。理解这个指标，是后续学习更复杂的风险评估指标（如夏普比率、最大回撤等）和进行策略对比的基石。记住，回测收益率告诉我们策略“赚了多少”，但要想全面评估策略，我们还需要结合其他指标来分析“赚得是否稳健”、“风险有多大”。
