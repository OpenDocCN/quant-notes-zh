上一节我们介绍了如何构建回归模型来预测股票价格变动。本节中，我们来看看如何评估这些模型的预测结果，并将其转化为可执行的交易策略。

首先，我们已经得到了预测数据。现在，我们可以计算不同策略下的累计收益。以下是具体步骤：

![](img/ec8ef05f4f09853c14200da7287f6e37_5.png)

1.  **提取关键指标**：从数据中提取实际收益率（`returns`）以及两个回归模型的预测结果。
2.  **计算累计收益**：由于我们之前对收益率进行了对数处理（`log(A) + log(B) = log(A*B)`），这使得收益率具有可加性。我们可以使用累加操作（`cumsum`）来计算一段时间内的累计对数收益。
3.  **还原收益值**：累加后，需要将对数收益还原为实际收益。这可以通过指数函数 `np.exp()` 来实现。

```python
# 假设 data 是包含 returns, pred_1, pred_2 列的 DataFrame
# 计算累计对数收益
cumulative_log_returns = data[['returns', 'pred_1', 'pred_2']].cumsum()
# 将对数收益还原为实际收益
cumulative_returns = np.exp(cumulative_log_returns)
```

执行上述代码后，我们得到了三个最终收益值。它们分别代表：
*   **实际走势（returns）**：如果初始投入1元，最终价值约为0.8元，表示亏损。
*   **策略一（pred_1）**：最终价值约为0.94元，虽然也亏损，但比什么都不做（实际走势）亏得少。
*   **策略二（pred_2）**：最终价值约为1.3元，实现了盈利。

这个结果表明，通过回归模型预测每日的买卖决策，策略二取得了正向收益。

接下来，我们分析预测的准确性。我们可以将模型的预测方向（涨/跌）与实际涨跌方向进行比较。

![](img/ec8ef05f4f09853c14200da7287f6e37_7.png)

以下是评估预测准确率的步骤：

1.  **获取实际方向**：数据中已有的 `direction` 列代表实际的涨跌情况。
2.  **比较预测方向**：将回归模型的预测值（`pred_1`, `pred_2`）转换为涨跌方向（例如，大于0为涨，小于0为跌），并与实际方向进行比较。
3.  **统计正确率**：计算预测正确和错误的次数。

![](img/ec8ef05f4f09853c14200da7287f6e37_9.png)

```python
# 假设实际方向列名为 ‘direction‘，预测值列名为 ‘pred_1‘
# 将预测值转换为方向（1为涨，0为跌）
pred_direction = (data[‘pred_1‘] > 0).astype(int)
# 比较预测方向与实际方向
correct_predictions = (pred_direction == data[‘direction‘])
# 统计正确和错误的次数
value_counts = correct_predictions.value_counts()
```

![](img/ec8ef05f4f09853c14200da7287f6e37_11.png)

![](img/ec8ef05f4f09853c14200da7287f6e37_13.png)

对于策略一，预测错误1093次，正确1042次，正确率略低于50%，这与最终亏损的结果相符。
对于策略二，预测正确1096次，错误1039次，正确率略高于50%，这解释了为何最终能获得微薄盈利。预测正确的次数稍多，累积起来便产生了正向收益。

最后，为了更直观地展示整个时间周期内三种策略的收益变化过程，我们来绘制收益走势图。

我们之前计算的是最终的总收益。如果想看每一天的累计收益走势，应该使用 `cumsum` 函数。`cumsum` 函数进行的是累积求和：第一天的值是第一天的收益，第二天的值是前两天的收益之和，以此类推，第N天的值就是前N天收益的总和。

![](img/ec8ef05f4f09853c14200da7287f6e37_19.png)

![](img/ec8ef05f4f09853c14200da7287f6e37_21.png)

以下是绘制走势图的步骤：

![](img/ec8ef05f4f09853c14200da7287f6e37_23.png)

![](img/ec8ef05f4f09853c14200da7287f6e37_25.png)

1.  **计算每日累计收益**：对 `returns`、`pred_1`、`pred_2` 分别应用 `cumsum()` 函数，得到每日的累计对数收益。
2.  **还原为实际价值**：同样使用 `np.exp()` 将每日的累计对数收益还原为以1元为起点的实际资产价值。
3.  **绘制图表**：将三条曲线绘制在同一张图中进行对比。

```python
import matplotlib.pyplot as plt

![](img/ec8ef05f4f09853c14200da7287f6e37_27.png)

# 计算每日累计收益并还原
cumulative_value = np.exp(data[['returns', 'pred_1', 'pred_2']].cumsum())

# 绘制走势图
plt.figure(figsize=(16, 8))
plt.plot(cumulative_value[‘returns‘], label=‘实际走势‘, color=‘blue‘)
plt.plot(cumulative_value[‘pred_1‘], label=‘策略一‘, color=‘green‘)
plt.plot(cumulative_value[‘pred_2‘], label=‘策略二‘, color=‘red‘)
plt.xlabel(‘时间‘)
plt.ylabel(‘资产价值（起始为1元）‘)
plt.title(‘不同策略收益走势对比‘)
plt.legend()
plt.grid(True)
plt.show()
```

![](img/ec8ef05f4f09853c14200da7287f6e37_29.png)

从生成的图中可以清晰看到：
*   **蓝色曲线（实际走势）**：整体呈下跌趋势，1元最终跌至约0.8元。
*   **绿色曲线（策略一）**：虽然最终也下跌，但其走势与蓝色曲线差异明显，波动不同。
*   **红色曲线（策略二）**：在大部分时间内资产价值高于1元，实现了盈利，尤其在中期阶段收益显著。
