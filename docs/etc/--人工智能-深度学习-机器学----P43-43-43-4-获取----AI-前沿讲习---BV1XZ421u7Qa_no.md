上一节我们介绍了如何获取单日的因子数据，本节中我们来看看如何获取一个连续时间段内的全部数据。

![](img/207479a07827503314b30458d2cd29e9_15.png)

如果有一个日期序列，需要查询其中每一天的数据，最直接的方法是使用循环。我们可以通过`for`循环遍历日期序列中的每一天。

![](img/207479a07827503314b30458d2cd29e9_17.png)

![](img/207479a07827503314b30458d2cd29e9_19.png)

以下是实现这一过程的核心步骤：

![](img/207479a07827503314b30458d2cd29e9_21.png)

![](img/207479a07827503314b30458d2cd29e9_23.png)

1.  **确定日期区间长度**：首先，需要知道日期序列的总天数，这决定了循环的次数。
2.  **遍历每一天**：使用`for`循环，遍历日期序列中的每一个索引位置。
3.  **获取每日数据**：在循环体内，调用获取单日数据的函数，传入当前循环的索引`i`，以获取第`i`天的因子数据。该函数返回一个三维数组，我们通常只关心第三个维度的数据。
4.  **构建每日数据表**：将获取到的每日数据（因子值）与对应的日期信息组合，构建成一个临时的`DataFrame`。这个`DataFrame`应至少包含两列：`日期`和`因子值`。
5.  **汇总所有数据**：在循环开始前，创建一个空的`DataFrame`作为总表。在每次循环中，将步骤4构建的每日`DataFrame`拼接到这个总表中。

![](img/207479a07827503314b30458d2cd29e9_25.png)

以下是实现上述逻辑的代码框架：

![](img/207479a07827503314b30458d2cd29e9_27.png)

![](img/207479a07827503314b30458d2cd29e9_29.png)

```python
import pandas as pd

![](img/207479a07827503314b30458d2cd29e9_31.png)

![](img/207479a07827503314b30458d2cd29e9_33.png)

# 假设 dates 是日期序列，get_daily_factor 是获取单日数据的函数
total_days = len(dates)
total_data = pd.DataFrame() # 创建空的总表

![](img/207479a07827503314b30458d2cd29e9_35.png)

![](img/207479a07827503314b30458d2cd29e9_37.png)

for i in range(total_days):
    # 1. 获取第i天的原始数据（三维数组）
    daily_raw_data = get_daily_factor(i)
    # 2. 提取有用的因子值（假设在第三维）
    factor_values = daily_raw_data[0, 0, :]
    
    # 3. 构建每日的DataFrame
    daily_df = pd.DataFrame()
    daily_df['因子值'] = factor_values
    daily_df['日期'] = dates[i] # 假设dates[i]是具体的日期
    
    # 4. 将每日DataFrame拼接到总表
    total_data = pd.concat([total_data, daily_df], ignore_index=True)
```
