### 第一步：去极值
去极值的目的是处理数据中的异常值或离群点，防止它们对模型产生过大影响。我们将使用“三西格玛”方法来实现。

![](img/93ac7b211f121ce0e8459deead2a2caa_32.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_34.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_36.png)

**三西格玛法公式**：
`上限 = 均值 + 3 * 标准差`
`下限 = 均值 - 3 * 标准差`

![](img/93ac7b211f121ce0e8459deead2a2caa_38.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_40.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_42.png)

我们将数据中超出此范围的值“拉回”到边界上，而不是直接删除。

![](img/93ac7b211f121ce0e8459deead2a2caa_46.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_48.png)

```python
def filter_three_sigma(series, n=3):
    """
    使用三西格玛法去除极值
    :param series: 待处理的因子数据序列
    :param n: 西格玛倍数，默认为3
    :return: 处理后的数据序列
    """
    mean = series.mean()  # 计算均值
    std = series.std()   # 计算标准差
    
    # 计算上限和下限
    upper = mean + n * std
    lower = mean - n * std
    
    # 将超出范围的值规范到边界上
    return series.clip(lower, upper)
```

### 第二步：标准化
标准化（也称Z-Score标准化）的目的是消除不同因子之间量纲和数量级的差异，使所有数据处于同一尺度，便于比较和计算。

![](img/93ac7b211f121ce0e8459deead2a2caa_62.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_64.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_66.png)

**标准化公式**：
`标准化值 = (原始值 - 均值) / 标准差`

```python
def standardize(series):
    """
    对因子数据进行标准化（Z-Score）
    :param series: 待标准化的数据序列
    :return: 标准化后的数据序列
    """
    mean = series.mean()
    std = series.std()
    
    # 应用标准化公式
    return (series - mean) / std
```

### 第三步：中性化
中性化的目的是消除因子数据中与其他已知变量（如市值）的相关性，从而提取出因子“纯粹”的预测能力。我们通过线性回归来实现。

![](img/93ac7b211f121ce0e8459deead2a2caa_86.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_88.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_90.png)

![](img/93ac7b211f121ce0e8459deead2a2caa_92.png)

**核心思想**：
1.  以需要中性化的因子（如市净率）作为因变量 `Y`。
2.  以需要剔除影响的变量（如市值）作为自变量 `X`。
3.  建立 `Y = W * X + b` 的线性回归模型。
4.  计算残差（真实值 `Y` 与模型预测值 `Y_pred` 的差），这个残差就是去除了市值影响后的“中性化”因子。

```python
def neutralize(factor_series, market_cap_series):
    """
    对因子进行市值中性化处理
    :param factor_series: 待中性化的因子数据（如市净率），作为Y
    :param market_cap_series: 市值数据，作为X
    :return: 中性化后的因子数据（残差）
    """
    # 确保数据类型为float
    Y = factor_series.values.astype(float)
    X = market_cap_series.values.astype(float)
    
    # 添加常数项，用于拟合截距b
    X = sm.add_constant(X)
    
    # 使用普通最小二乘法进行线性回归
    model = sm.OLS(Y, X).fit()
    
    # 返回残差，即中性化后的因子值
    return model.resid
```

## 总结
本节课中我们一起学习了因子数据预处理的完整流程。我们首先通过**去极值**处理了异常数据，然后通过**标准化**统一了数据的尺度，最后通过**中性化**回归消除了因子与市值等变量的相关性，得到了更为“纯净”的因子。这三步是构建有效量化模型的基础，处理后的数据能显著提升后续因子分析和策略回测的可靠性。
