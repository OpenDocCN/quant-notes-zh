## 获取因子指标数据

有了交易日列表后，接下来我们需要获取具体的因子指标数据。这包括确定要分析的因子、构建股票池，并执行数据查询。

![](img/dc555521ef3ce7b0b0cd71249b926700_17.png)

首先，我们确定要分析的因子，例如`p_retro`。同时，我们需要构建一个股票池，这里我们选择获取所有A股股票。

![](img/dc555521ef3ce7b0b0cd71249b926700_19.png)

![](img/dc555521ef3ce7b0b0cd71249b926700_21.png)

```python
# 指定要分析的因子名称
factor_name = ‘p_retro’

# 获取所有A股股票代码，构建股票池
stock_pool = get_all_securities(types=[‘stock’], date=None).index.tolist()
```

![](img/dc555521ef3ce7b0b0cd71249b926700_23.png)

![](img/dc555521ef3ce7b0b0cd71249b926700_25.png)

接下来，我们需要构建查询语句来获取指定因子在股票池中的数据。查询语句通常较长，可以参考工具的API文档。

![](img/dc555521ef3ce7b0b0cd71249b926700_27.png)

```python
# 构建查询语句
q = query(fundamentals.financial_indicator.p_retro
         ).filter(fundamentals.financial_indicator.code.in_(stock_pool))
```

![](img/dc555521ef3ce7b0b0cd71249b926700_29.png)

查询语句构建完成后，我们需要执行它来获取数据。这里我们以`trading_dates`中的第一天为例进行查询。

![](img/dc555521ef3ce7b0b0cd71249b926700_31.png)

![](img/dc555521ef3ce7b0b0cd71249b926700_33.png)

```python
# 执行查询，获取指定日期的因子数据
# 假设我们查询第一个交易日的数据
query_date = trading_dates[0]
factor_data = get_fundamentals(q, date=query_date)
```

执行上述代码后，`factor_data`中就包含了指定日期所有股票对应的`p_retro`因子值。我们可以查看前几条数据以确认格式。

![](img/dc555521ef3ce7b0b0cd71249b926700_35.png)
