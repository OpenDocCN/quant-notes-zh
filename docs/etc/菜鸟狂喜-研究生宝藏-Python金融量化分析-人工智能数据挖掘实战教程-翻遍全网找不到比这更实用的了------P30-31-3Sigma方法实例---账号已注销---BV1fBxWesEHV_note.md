在本节课中，我们将要学习一种在数据预处理中常用的去极值方法——3-Sigma方法。我们将从正态分布的基本概念入手，理解3-Sigma方法的原理，并最终通过Python代码实现它，用于识别和处理数据中的异常值。

## 正态分布与3-Sigma原理 📈

上一节我们介绍了数据预处理的重要性，本节中我们来看看基于统计原理的3-Sigma方法。

3-Sigma方法的原理基于正态分布。正态分布是一种常见的概率分布，其形状呈钟形曲线。在正态分布中，大多数数据点会聚集在平均值附近，离平均值越远的数据点出现的概率越低。

为了便于理解，可以想象一个去银行贷款的例子。银行多借或少借你少量金额是正常情况，但如果金额偏差巨大（例如多借或少借10万元），这就属于异常情况。在数据中，这些“偏差巨大”的值就是我们需要关注的极值或离群点。

在标准正态分布中，平均值（μ）为0，标准差（σ）为1。数据分布情况可以通过距离平均值的标准差倍数来衡量。

![](img/f7b86523c4115c08248c6d4e89011b9d_1.png)

上图展示了以平均值为中心，分别加减1倍、2倍、3倍标准差所覆盖的数据范围。区间越大，覆盖的数据点距离平均值越远，但包含的数据量占总体的比例也越高。

3-Sigma方法的核心思想是：假设数据服从或近似服从正态分布，那么绝大多数（约99.7%）的正常数据会落在平均值加减3倍标准差的区间内。即：
**区间 = [μ - 3σ， μ + 3σ]**

因此，我们可以将此区间外的数据（约0.3%）视为极值进行处理，而不是直接丢弃。处理方式通常是将这些极值规范到区间的边界上。

## 3-Sigma方法的代码实现 💻

![](img/f7b86523c4115c08248c6d4e89011b9d_1.png)

理解了原理后，我们现在通过Python代码来实现3-Sigma方法。我们将计算数据的上下限，并将超出限值的数据规范到边界。

以下是实现3-Sigma去极值功能的关键步骤：

1.  **计算均值和标准差**：首先需要计算待处理数据列的算术平均值（μ）和标准差（σ）。
2.  **确定上下限**：根据公式 **下限 = μ - nσ** 和 **上限 = μ + nσ** 计算边界。其中n通常取3。
3.  **数据规范化**：遍历数据，将所有小于下限的值设置为下限，将所有大于上限的值设置为上限。

然后，计算关键指标。对于传入的数据列，我们使用 `.mean()` 方法计算均值，使用 `.std()` 方法计算标准差。

![](img/f7b86523c4115c08248c6d4e89011b9d_9.png)

## 核心要点总结 ✨

![](img/f7b86523c4115c08248c6d4e89011b9d_37.png)

本节课中我们一起学习了3-Sigma去极值方法。

![](img/f7b86523c4115c08248c6d4e89011b9d_39.png)

我们主要介绍了三种去极值策略，3-Sigma是其中之一。需要特别注意，**“去极值”不是简单地将异常数据删除**，而是通过定义合理的下界和上界，将所有数据规范到这个范围内。

![](img/f7b86523c4115c08248c6d4e89011b9d_41.png)

![](img/f7b86523c4115c08248c6d4e89011b9d_41.png)

![](img/f7b86523c4115c08248c6d4e89011b9d_43.png)

![](img/f7b86523c4115c08248c6d4e89011b9d_43.png)

具体做法是：对于小于下限的值，将其提升至下限；对于大于上限的值，将其降低至上限。这样就达到了平滑数据、减少极端值影响的目的，同时保留了所有的样本点。

![](img/f7b86523c4115c08248c6d4e89011b9d_45.png)

![](img/f7b86523c4115c08248c6d4e89011b9d_45.png)
