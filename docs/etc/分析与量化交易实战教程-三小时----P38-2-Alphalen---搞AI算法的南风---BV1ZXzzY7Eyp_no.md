## 工具包简介与获取

![](img/d04da2bc59c9fd01344cb92b58ff004a_5.png)

![](img/d04da2bc59c9fd01344cb92b58ff004a_7.png)

Alphalens是一个专门用于金融因子分析的Python工具包。接下来我们所有的计算和画图操作都可以借助它来完成，这能让我们更专注于策略逻辑本身。

![](img/d04da2bc59c9fd01344cb92b58ff004a_9.png)

![](img/d04da2bc59c9fd01344cb92b58ff004a_11.png)

以下是关于Alphalens的基本信息：
*   **GitHub链接**：可以访问其开源仓库查看源码和示例。
*   **官方文档**：提供了详细的使用说明和API参考。

![](img/d04da2bc59c9fd01344cb92b58ff004a_13.png)

安装Alphalens非常简单，只需在命令行中执行以下命令：
```bash
pip install alphalens
```
不过，在本课程后续的实战环节中，我们将使用一个在线的量化平台。该平台已经预装了Alphalens等必要的库，因此大家无需在本地安装即可直接使用。如果你想在本地环境进行实验，则可以自行安装。

![](img/d04da2bc59c9fd01344cb92b58ff004a_15.png)

对于想深入学习的同学，建议查阅官方文档和示例（Examples）。这些资料提供了最权威和详细的使用指南。我们课程中的案例也主要参考了官方的示例，并将其核心内容总结和简化，以帮助大家快速上手。

## 在线研究环境

![](img/d04da2bc59c9fd01344cb92b58ff004a_19.png)

由于直接获取和处理金融数据较为繁琐，我们将在量化平台提供的在线环境中编写代码。这个环境已经配置好了所需的数据和库。

接下来，我们切换到平台的“投资研究”模块。这个模块提供了一个类似于Jupyter Notebook的交互式编程环境，允许我们在平台的服务器上直接运行代码。

以下是进入在线研究环境的步骤：
1.  在平台界面左侧找到并点击“投资研究”。
2.  点击“新建”按钮，创建一个新的Python 3笔记本。
3.  将新建的笔记本重命名为“因子分析”。

![](img/d04da2bc59c9fd01344cb92b58ff004a_21.png)

![](img/d04da2bc59c9fd01344cb92b58ff004a_23.png)

之后，我们所有的示例代码都将在这个名为“因子分析”的笔记本中编写和运行。我会将完整的代码提供给大家，你可以直接上传到你的平台账户，或者跟随视频教程一步步编写。

![](img/d04da2bc59c9fd01344cb92b58ff004a_25.png)
