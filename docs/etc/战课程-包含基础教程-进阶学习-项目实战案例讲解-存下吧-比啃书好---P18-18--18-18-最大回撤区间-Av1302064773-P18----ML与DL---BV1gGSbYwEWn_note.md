## 概述 📖

最大回撤是衡量投资策略风险的关键指标。它描述的是在特定时间段内，资产净值从最高点到之后最低点的最大跌幅。这个指标不评估策略的盈利能力，而是评估其可能面临的最大风险程度。

## 什么是最大回撤？ 📉

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何评估策略的风险。最大回撤描述的是投资过程中“最惨”的阶段。

具体来说，它衡量的是从任意一个历史高点开始，到之后任意一个低点为止，资产价值下跌的最大幅度。这个“幅度”不仅考虑下跌的绝对金额，更关注下跌的百分比，即资产“缩水”的程度。

**公式**可以表示为：
`最大回撤 = max[(Pi - Pj) / Pi]`
其中，`Pi` 代表历史高点的价格，`Pj` 代表 `Pi` 之后某个低点的价格。我们需要遍历所有可能的 `i` 和 `j`（`j > i`）组合，找出比值最大的那个。

## 最大回撤的意义 ⚠️

理解最大回撤的计算后，我们来看看它的实际意义。它本质上是一个**风险指标**。

![](img/3cf72c2daeec5fa314c9dd5f558508d5_5.png)

![](img/3cf72c2daeec5fa314c9dd5f558508d5_7.png)

即使一个策略长期来看是盈利的，但在过程中可能会经历非常痛苦的下跌阶段。最大回撤就是量化这个“痛苦”的程度。例如，在2008年全球金融危机中，许多市场指数从高点下跌超过50%，这意味着投资者在最糟糕的阶段承受了资产腰斩的风险。了解策略的最大回撤，有助于投资者评估自己能否承受这样的波动，避免在真正的下跌中因恐慌而做出错误决策。

![](img/3cf72c2daeec5fa314c9dd5f558508d5_9.png)

在我们的策略回测报告中，最大回撤区间会被明确标识出来，如下图所示，它帮助我们发现策略最脆弱的时期。

## 如何计算最大回撤？ 🧮

![](img/3cf72c2daeec5fa314c9dd5f558508d5_11.png)

![](img/3cf72c2daeec5fa314c9dd5f558508d5_13.png)

理论介绍完毕，接下来我们动手实践如何用Python计算最大回撤。核心思路是利用Pandas库的向量化操作高效地完成计算。

![](img/3cf72c2daeec5fa314c9dd5f558508d5_15.png)

以下是计算最大回撤的关键步骤：

![](img/3cf72c2daeec5fa314c9dd5f558508d5_17.png)

1.  **计算历史最大值序列**：我们需要知道到每一天为止，历史上的最高净值是多少。这可以通过 `cummax()` 函数实现。
2.  **计算每日回撤**：用每一天对应的历史最高值减去当天的实际净值，得到每日的“潜在下跌”金额。
3.  **计算每日回撤率**：将上述差值除以对应的历史最高值，得到每日的回撤百分比。
4.  **找出最大值**：在所有每日回撤率中，找出最大值，即为该时间段内的最大回撤。

![](img/3cf72c2daeec5fa314c9dd5f558508d5_19.png)

![](img/3cf72c2daeec5fa314c9dd5f558508d5_21.png)

让我们通过代码来演示这个过程。假设我们有一个名为 `df` 的DataFrame，其中 `‘net_value’` 列代表每日净值。

![](img/3cf72c2daeec5fa314c9dd5f558508d5_23.png)

```python
import pandas as pd

![](img/3cf72c2daeec5fa314c9dd5f558508d5_25.png)

![](img/3cf72c2daeec5fa314c9dd5f558508d5_27.png)

![](img/3cf72c2daeec5fa314c9dd5f558508d5_29.png)

# 假设 df[‘net_value‘] 是净值序列
net_value = df[‘net_value‘]

![](img/3cf72c2daeec5fa314c9dd5f558508d5_31.png)

# 1. 计算到当日为止的历史最高净值序列
historical_max = net_value.cummax()

![](img/3cf72c2daeec5fa314c9dd5f558508d5_33.png)

# 2. 计算每日的下跌金额（历史最高 - 当前净值）
drawdown_amount = historical_max - net_value

![](img/3cf72c2daeec5fa314c9dd5f558508d5_35.png)

# 3. 计算每日的回撤率（下跌金额 / 历史最高净值）
drawdown_ratio = drawdown_amount / historical_max

![](img/3cf72c2daeec5fa314c9dd5f558508d5_37.png)

![](img/3cf72c2daeec5fa314c9dd5f558508d5_39.png)

# 4. 找出所有回撤率中的最大值，即最大回撤
max_drawdown = drawdown_ratio.max()

print(f“最大回撤为：{max_drawdown:.2%}“)
```

![](img/3cf72c2daeec5fa314c9dd5f558508d5_41.png)

![](img/3cf72c2daeec5fa314c9dd5f558508d5_43.png)

执行以上代码，我们就能得到该净值序列在整个回测期内的最大回撤百分比。例如，输出“最大回撤为：30.70%”，意味着该策略曾经历过资产从前期高点下跌超过30%的艰难时期。
