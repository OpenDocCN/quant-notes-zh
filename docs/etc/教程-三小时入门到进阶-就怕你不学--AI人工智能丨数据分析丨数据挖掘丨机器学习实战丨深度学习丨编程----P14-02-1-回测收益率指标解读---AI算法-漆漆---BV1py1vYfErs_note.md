回测收益率是最基本也是最重要的指标。它衡量的是，在回测周期内，执行某个策略（例如，基于特定规则在2015年至2020年间买卖某只股票）所获得的总收益。注意，它表示的不是绝对金额，而是相对于初始本金的收益率。例如，一个4%的回测收益率意味着，相对于投入的本金，策略最终赚取了4%的利润。



![](img/50d4da191c2fe0056f61449c2fa364a9_19.png)

![](img/50d4da191c2fe0056f61449c2fa364a9_25.png)

其计算公式如下：

![](img/50d4da191c2fe0056f61449c2fa364a9_27.png)

**回测收益率 = (P_t / P_0) - 1**

其中：
*   **P_t** 代表策略在回测结束时的资产总值（卖出后的金额）。
*   **P_0** 代表策略在回测开始时的初始本金（买入前的金额）。

分子 `(P_t - P_0)` 表示赚取的绝对利润，除以 `P_0` 后则转化为相对于本金的收益率，最后减去1得到净增长比例。



![](img/50d4da191c2fe0056f61449c2fa364a9_21.png)

这个指标是评估策略表现时最常用的，直观地反映了策略的最终盈利效果。



![](img/50d4da191c2fe0056f61449c2fa364a9_23.png)

接下来，我们通过Python代码来计算并可视化回测收益率。我们不仅可以计算最终的收益率，还可以观察收益率在整个回测周期内的变化过程。

我们可以计算每个时间点资产相对于初始本金的比值 `(P_i / P_0)`，这实际上就是累积净值曲线。最后一天的净值减1，即为整个周期的回测收益率。

![](img/50d4da191c2fe0056f61449c2fa364a9_29.png)

```python
# 计算每个时间点的资产与初始资产的比值
cumulative_value = data[‘close‘] / data[‘close‘].iloc[0]
# 将结果转换为DataFrame以便后续操作
df_ratio = pd.DataFrame(cumulative_value, columns=[‘cumulative_ratio‘])
```

上图展示了从期初净值1开始，策略资产随时间的变化情况。接下来，我们计算整个回测周期的最终收益率。



![](img/50d4da191c2fe0056f61449c2fa364a9_29.png)

最终的回测收益率就是净值曲线最后一个点的值减去1。

![](img/50d4da191c2fe0056f61449c2fa364a9_41.png)

![](img/50d4da191c2fe0056f61449c2fa364a9_43.png)

```python
# 计算最终回测收益率
final_return = df_ratio[‘cumulative_ratio‘].iloc[-1] - 1
# 将结果整理成表格形式展示
result_df = pd.DataFrame({‘回测收益率‘: [final_return]}, index=[‘最终值‘])
print(result_df)
```
