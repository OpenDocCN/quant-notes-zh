## 股票的定义与作用

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

为了更形象地解释，我们可以设想一个场景：一位创业者需要资金，而投资者愿意提供资金。投资者将资金给予公司，公司则向投资者发行股票作为凭证。这代表投资者对公司出资，并因此成为公司的所有者之一。

股票主要有两个作用：
1.  **出资证明与股东身份**：持有股票证明你对公司进行了出资，你是公司的股东。作为股东，你拥有相应的权利，例如在股东大会上投票。
2.  **获取收益**：作为股东，你可以通过两种主要方式获利：
    *   **公司分红**：当公司盈利时，可能会将部分利润按持股比例分配给股东。
    *   **交易获利**：你可以将持有的股票在市场上卖给其他投资者。如果卖出价格高于买入价格，你就获得了价差收益。

上一节我们介绍了股票的基本定义和作用，本节中我们来看看公司如何向公众发行股票。

## 公司上市与IPO

所谓“上市”，是指企业通过证券交易所，首次公开向全社会投资者增发股票以募集资金的过程。

公司不能随意向公众募集资金。它需要达到一定标准，并向证监会等监管机构提交申请，经过严格审核（如审查财务报表、经营状况等）后，才能获得在证券交易所挂牌交易的资格。

上市后，公司的股票就能被所有股民看到并进行买卖。这个首次公开募股的过程就叫做 **IPO**。

## 股票的分类

股票可以按照不同的标准进行分类。以下是两种常见的分类方式。

### 按公司业绩分类

根据公司的经营业绩，股票通常被分为以下几类：

![](img/3ead426d736a2147aaa31816f926a5e2_1.png)

*   **蓝筹股**：指资本雄厚、信誉优良的大型公司股票。例如中国的石油、石化、银行等巨头企业。其特点是体量大、经营稳定。
*   **绩优股**：指业绩优良公司的股票。这类公司可能规模不是最大，但盈利能力持续且强劲。例如某些消费、科技行业的龙头公司。
*   **ST股**：中文名为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记。这是为了提醒投资者该公司存在较高风险。

### 按上市地区分类

根据股票上市交易的地理位置和交易货币，可以分为：

*   **A股**：在中国大陆（上海、深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。
*   **H股**：在中国香港上市的公司股票。
*   **N股**：在美国纽约上市的公司股票。
*   **S股**：在新加坡上市的公司股票。

不同的股市有不同的交易规则。了解这些规则对投资至关重要。

## 主要股市交易规则简介

以下是几个主要市场部分核心规则的对比：

| 市场 | 涨跌幅限制 | 交易制度 (T+N) | 备注 |
| :--- | :--- | :--- | :--- |
| **A股** | 普通股票±10% | T+1 | 今日买入，下一交易日方可卖出。旨在减少投机。 |
| **B股** | 普通股票±10% | T+1 (交易)， T+3 (资金交收) | 卖出股票后，资金需3个交易日后方可提现。 |
| **H股 (港股)** | 无涨跌幅限制 | T+0 | 当日可多次买卖同一只股票。 |
| **N股 (美股)** | 无涨跌幅限制 | T+0 | 交易灵活，风险与机会并存。 |

**核心概念解释**：
*   **涨跌幅限制**：指一只股票在一个交易日内的价格波动不得超过前一个交易日收盘价的一定百分比。公式可表示为：
    `当日最高价 ≤ 前日收盘价 × (1 + 涨幅限制比例)`
    `当日最低价 ≥ 前日收盘价 × (1 - 跌幅限制比例)`
*   **T+N**：其中`T`代表交易日（Trade day），`N`代表天数。`T+0`表示当天买卖，资金和股票实时交割；`T+1`表示当天买入的股票，需等到下一个交易日（`T+1`）才能卖出。
