# Python金融分析与量化交易实战课程：P48：48.1-打分法选股策略概述 📊

## 概述
在本节课中，我们将要学习一种在选股过程中常用的策略——因子打分选股法。我们将了解该策略的核心思想、实施步骤以及背后的逻辑，为后续的实战编程打下基础。

## 策略核心思想
上一节我们介绍了因子分析的基本概念。本节中我们来看看如何将多个因子综合起来进行选股。

因子打分选股，顾名思义，其核心在于“打分”。我们不再孤立地看待单个因子对股票好坏的影响，而是像评判学生的综合成绩一样，为每只股票在多个因子上的表现分别打分，然后计算一个总分，最后根据总分排名来筛选股票。

具体而言，假设我们有300只股票（例如沪深300成分股），以及A、B、C、D四个不同的因子指标（如估值、成长、动量等）。我们的目标是每月从这300只股票中选出表现最好的10只或20只进行调仓。

## 策略执行步骤
以下是实施因子打分选股策略的关键步骤：

1.  **因子选择与方向判断**：首先，我们需要选择用于评价的因子，并预先知道每个因子与股票未来收益率的相关性是正向还是负向。例如，净资产收益率（ROE）通常是**越高越好**（正向因子），而市盈率（PE）在某些策略中可能是**越低越好**（负向因子）。这个判断可以基于金融研究报告或历史数据的统计分析（如计算IC值）得出。

2.  **为单因子打分**：对每只股票，在每个因子上的原始数值进行标准化处理，并转换为得分。对于正向因子，数值越大得分越高；对于负向因子，数值越小得分越高。常见的打分方法包括百分位排名、Z-Score标准化等。公式可以表示为：
    `单因子得分 = f(因子原始值, 因子方向)`
    其中，函数 `f` 根据因子方向对数值进行正向化处理。

![](img/f803969fec8ff8c2f3f1378cf044795d_1.png)

3.  **计算综合得分**：将每只股票在所有选定因子上的得分进行加权求和，得到该股票的综合总分。这类似于计算学生的总成绩：
    `综合总分 = w1 * 得分_A + w2 * 得分_B + w3 * 得分_C + w4 * 得分_D`
    其中，`w1, w2, w3, w4` 是各因子的权重，可以等权，也可以根据因子的预测能力分配不同权重。

4.  **排序与选股**：根据计算出的综合总分对所有股票进行降序排名。选取排名最高的N只股票（如前10名）作为当期投资组合的标的。

## 策略优势与注意事项
这种策略的优势在于综合了多方面的信息，避免了依赖单一因子可能产生的偏差，使选股逻辑更稳健。

然而，策略效果高度依赖于所选因子的有效性和因子方向的正确判断。如果预先判断的因子方向错误，或者因子本身与未来收益的相关性很弱甚至消失，那么打分结果将失去意义，导致策略失效。因此，因子研究和持续验证是策略成功的基础。
