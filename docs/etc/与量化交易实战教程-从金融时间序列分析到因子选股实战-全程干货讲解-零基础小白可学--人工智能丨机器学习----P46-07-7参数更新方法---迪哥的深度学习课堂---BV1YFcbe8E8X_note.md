在本节课中，我们将学习梯度下降算法中参数更新的具体方法。我们将深入探讨如何分别对模型中的不同参数进行优化，并详细拆解参数更新的数学推导过程。

## 参数优化：分别进行还是同时进行？

上一节我们介绍了梯度下降的基本思想，本节中我们来看看参数更新的具体策略。在优化目标函数以找到最优参数（如 θ₀ 和 θ₁）的过程中，一个关键问题是：这些参数是应该分别优化，还是放在一起同时优化？

答案是**分别进行优化**。原因在于，参数 θ₀ 对应特征 X₀，参数 θ₁ 对应特征 X₁。在数据样本中，X₀ 和 X₁ 是相互独立的。既然特征之间没有关系，那么它们对应的参数在优化过程中也应当互不影响。

因此，在优化过程中，我们分别对 θ₀ 和 θ₁ 进行优化。这可以想象在一个三维空间中，θ₀ 和 θ₁ 各代表一个坐标轴。我们前进时，会先为 θ₀ 寻找合适的方向，再为 θ₁ 寻找合适的方向。两者各自沿着自己的方向前进，最终综合得到一个结果。

![](img/43e412c84817eb64fa5e4fea970941b8_1.png)

具体来说，优化步骤如下：
1.  首先，用损失函数 J 对 θ₀ 求偏导，得到 θ₀ 的更新方向。
2.  然后，用损失函数 J 对 θ₁ 求偏导，得到 θ₁ 的更新方向。
3.  两个参数各自按照自己的方向进行更新，彼此之间没有影响。

## 梯度下降的核心步骤

![](img/43e412c84817eb64fa5e4fea970941b8_3.png)

寻找山谷最低点（即最小化损失函数）的过程，可以概括为以下三个核心步骤，并且这是一个反复迭代的过程。

以下是梯度下降算法的三个核心步骤：

1.  **求偏导**：计算损失函数关于当前各个参数的梯度（偏导数）。
2.  **沿梯度反方向走一小步**：沿着计算出的梯度反方向，以一个较小的步长（学习率）移动。**强调：必须是一小步**。步长过大可能导致“跌倒”，即方向错误、不收敛或收敛效果差。
3.  **更新参数**：根据移动的方向和步长，更新所有参数的值。更新后，回到步骤1，在新的参数点上重新计算，如此反复。

## 数学推导：参数更新公式

那么，如何用数学公式来描述这个“下山”任务呢？我们之前定义了损失函数（loss function）。

回顾一下，对于单个样本 i，其损失为：
`loss_i = 1/2 * (y_i - h_θ(x_i))^2`
其中 `h_θ(x_i)` 是我们的预测函数。

但评估一个模型的好坏，不能只看一个样本。我们希望模型能尽可能好地拟合所有样本。因此，我们使用所有 M 个样本的平均损失作为整体的损失函数 J(θ)：
`J(θ) = 1/M * Σ_{i=1}^{M} [1/2 * (y_i - h_θ(x_i))^2]`
这里加上平方项是为了放大预测误差较大的样本的影响，求和后除以 M 是计算平均损失。

现在，假设我们要对某个参数 θ_j 进行更新，我们需要计算损失函数 J(θ) 对 θ_j 的偏导数 ∂J/∂θ_j。

假设预测模型为线性模型：`h_θ(x_i) = θ_0 + θ_1*x_i1 + θ_2*x_i2 + ... + θ_j*x_ij + ...`
当我们对 θ_j 求偏导时，只有包含 θ_j 的项（即 θ_j * x_ij）会起作用，其他项视为常数，导数为零。

求导过程如下：
`∂J/∂θ_j = ∂/∂θ_j { 1/M * Σ_{i=1}^{M} [1/2 * (y_i - h_θ(x_i))^2] }`
`= 1/M * Σ_{i=1}^{M} [ (y_i - h_θ(x_i)) * ∂/∂θ_j ( - h_θ(x_i) ) ]` （链式法则，平方项导数为2*(y_i - h_θ)，与1/2相乘后系数为1；注意 h_θ(x_i) 前有负号）
`= 1/M * Σ_{i=1}^{M} [ (y_i - h_θ(x_i)) * ( - x_ij ) ]`
`= -1/M * Σ_{i=1}^{M} [ (y_i - h_θ(x_i)) * x_ij ]`

这里，`x_ij` 表示第 i 个样本的第 j 个特征值。

![](img/43e412c84817eb64fa5e4fea970941b8_5.png)

我们得到了梯度 ∂J/∂θ_j。在梯度下降中，我们需要沿着梯度的**反方向**（即下降最快的方向）更新参数。梯度的反方向就是 `- ∂J/∂θ_j`。

因此，参数 θ_j 的更新公式为：
`θ_j_new = θ_j_old - α * ∂J/∂θ_j`
`= θ_j_old + α * [1/M * Σ_{i=1}^{M} ( (y_i - h_θ(x_i)) * x_ij ) ]`

其中：
*   `θ_j_old` 是参数更新前的值。
*   `α` 是学习率（Learning Rate），即我们所说的“一小步”的步长。
*   `[1/M * Σ_{i=1}^{M} ( (y_i - h_θ(x_i)) * x_ij ) ]` 就是计算出的梯度方向（已取反）。

这个公式就是参数更新的核心。它表示：新的参数值等于旧的参数值，加上学习率乘以所有样本的预测误差与对应特征值的乘积的平均值。

## 总结

![](img/43e412c84817eb64fa5e4fea970941b8_7.png)

本节课中我们一起学习了梯度下降算法的参数更新方法。我们明确了参数需要**分别进行优化**，因为其对应的特征是独立的。我们梳理了梯度下降**求偏导、走小步、更新参数**的三步核心流程。最后，我们通过数学推导，得出了关键的参数更新公式 **`θ_j_new = θ_j_old + α * [1/M * Σ ( (y_i - h_θ(x_i)) * x_ij ) ]`**。理解这个公式的由来，对于后续手动实现或深入理解优化算法至关重要。
