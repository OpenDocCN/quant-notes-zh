## 趋向指标的基本概念

上一节我们简单介绍了趋向指标的定义。本节将进行更详细的讲解。

![](img/52d8037c4751d02bfc3775a9ef108f57_3.png)

趋向指标又称动向指标。上一节提到，趋向指标既不捉顶也不补底。本节将解释它为什么能做到这点。其基本原理是：通过分析股价在上升及下跌过程中的均衡（这里的均衡可以理解为供需关系受价格变动的影响），由于供需关系受价格变动引发从均衡到失衡的循环过程，并根据这个循环过程来进行趋势判断。趋向指标主要跟踪股价在上升及下跌中的均衡，重点关注股价及价格变动的均衡和失衡这个循环过程，并由此进行趋势研判。

![](img/52d8037c4751d02bfc3775a9ef108f57_5.png)

![](img/52d8037c4751d02bfc3775a9ef108f57_5.png)

接下来进行详细的原理介绍。在大多数指标中，基本都是以每一日股价的收盘走势及升幅跌幅来进行计算的。但除了趋向指标外，这种方法的不足是忽略了每一日的高价和低价的波动幅度。而趋向指标（如动向指标DMI）的重点就是关注每日高低价格的波动幅度，并通过这个波动幅度来预测未来的走势。大家记住一点：趋向指标主要是把每日股价的高低波动幅度计算在内，以此来预测整体股价的未来走势。

## MACD指标

![](img/52d8037c4751d02bfc3775a9ef108f57_7.png)

在介绍了基本原理后，接下来我们介绍MACD指标。之前可能已经简单了解过MACD，这里我们先回顾一下它的定义。

![](img/52d8037c4751d02bfc3775a9ef108f57_7.png)

MACD是在20世纪70年代由美国投资家杰拉尔德·阿佩尔提出的。它本质上是平滑处理后的均线差异程度，一般用来判断股价变化的方向、强度和趋势。这里有一幅图，重点标出了几个关键点：这是一个MACD股价均势图和K线图的例子。这里是一个点，称为“金叉”；这里是一个点，称为“死叉”。当快线上穿慢线，同时股价上扬，这就是金叉，通常被视为买入信号。当快线下穿慢线，这就是死叉，通常被视为卖出信号。当然，这不是绝对的，但这是一个大概率事件。图中这里也是一个死叉。

MACD主要由两根线构成：DIF线（快线）和DEA线（慢线）。当快线上穿慢线时，也就是红柱出现的第一天，称为金叉，是买进或持有的时期。当快线下穿慢线时，也就是绿柱出现的第一天，称为死叉，是卖出的时期。图中这个点就是死叉，因为它是绿柱出现的第一天。MACD的核心就是发现金叉和死叉，并遵循“金叉买进，死叉卖出”的原则。

## MACD的代码实现

接下来是MACD的代码实现。在聚宽平台中，MACD已经被抽象成一个基本函数，实现起来非常方便。

![](img/52d8037c4751d02bfc3775a9ef108f57_9.png)

![](img/52d8037c4751d02bfc3775a9ef108f57_9.png)

我们使用`technical_analysis`技术分析包，里面有专门的`MACD`函数。以下是该函数的参数说明：

以下是`MACD`函数的主要参数：
*   `security_list`：股票列表，可以传入单只股票代码或股票代码列表。
*   `check_date`：查询MACD值的日期。
*   `SHORT`：快线周期，默认为12。
*   `LONG`：慢线周期，默认为26。
*   `MID`：计算MACD的EMA均线周期，默认为9。

这个函数返回一个字典，包含DIF、DEA和MACD三个值，每个值本身也是一个以股票代码为键的字典。

```python
# 示例代码：计算MACD
from technical_analysis import *

# 计算股票‘000001.XSHE’在‘2023-01-01’的MACD值
result = MACD(security_list=['000001.XSHE'], check_date='2023-01-01', SHORT=12, LONG=26, MID=9)

# 获取DIF、DEA、MACD值
dif_dict = result['DIF']
dea_dict = result['DEA']
macd_dict = result['MACD']
```

## EMV指标

介绍完MACD，接下来我们介绍EMV指标。EMV是简易波动指标。

![](img/52d8037c4751d02bfc3775a9ef108f57_11.png)

EMV重点关注股票的中间价所对应的相对波动幅度。它的计算方式是：相对成交额乘以相对振幅，作为衡量股票中间价波动百分比的基数。EMV值增加代表成交量增加、价格上升；EMV值下跌代表缩量下跌、价格下跌。

EMV的计算公式如下：
*   `A = (今日最高价 + 今日最低价) / 2` （今日中间价）
*   `B = (前日最高价 + 前日最低价) / 2` （前日中间价）
*   `C = 今日最高价 - 今日最低价` （今日振幅）
*   `EM = (A - B) * C / 今日成交额`
*   `EMV = N日内EM的累加和`
*   `MAEMV = EMV的M日简单移动均线`

![](img/52d8037c4751d02bfc3775a9ef108f57_11.png)

一般参数N为14，参数M为9。

这里有一幅常见的EMV图。上面是K线图，下面是EMV和MAEMV线。图中A点从上往下穿过零轴，对应一个卖点；B点从下往上穿过零轴，对应一个买点。EMV有一条零轴线，这是它与MACD的一个区别。EMV的简单口诀是：当EMV从底部向上穿过零轴时买入（上升趋势）；当EMV从顶部向下穿过零轴时卖出（下跌趋势）。

同样，在聚宽平台中，EMV也被抽象成了函数。

以下是`EMV`函数的主要参数：

## UOS指标

介绍完EMV，再给大家介绍UOS指标。UOS叫终极波动指标。

![](img/52d8037c4751d02bfc3775a9ef108f57_15.png)

UOS是一种多方位的功能指标。它除了用于趋势确认以及判断超买超卖，还可以提供最适当的交易时机，并能加强其他指标的可靠度。下面介绍UOS的算法。

UOS的计算涉及几个步骤：
1.  `TH = max(今日最高价， 昨日收盘价)`
2.  `TL = min(今日最低价， 昨日收盘价)`
3.  `ACC1 = N1日内（收盘价 - TL）的累加和 / N1日内（TH - TL）的累加和`
4.  `ACC2 = N2日内（收盘价 - TL）的累加和 / N2日内（TH - TL）的累加和`
5.  `ACC3 = N3日内（收盘价 - TL）的累加和 / N3日内（TH - TL）的累加和`
6.  `UOS = (ACC1 * N2 * N3 + ACC2 * N1 * N3 + ACC3 * N1 * N2) * 100 / (N1*N2 + N2*N3 + N1*N3)`
7.  `MAUOS = UOS的M日平滑移动均线`

一般参数设置为：N1=7， N2=14， N3=28， M=6。

UOS指标同时适用于短线和长线分析：
*   **短线**（一天到一周）：上穿50可考虑抄底，下穿65可考虑卖出。
*   **中长线**（数周到一年）：上穿35可考虑抄底，下穿70可考虑卖出。

## 总结

本节课我们一起学习了三种重要的趋向指标：MACD、EMV和UOS。
*   **MACD**：通过快慢线的交叉（金叉/死叉）来判断买卖时机，核心是观察DIF与DEA的关系以及红绿柱的变化。
*   **EMV**：关注中间价的波动，通过EMV线与零轴的关系（上穿买入，下穿卖出）来指导交易。
*   **UOS**：一种多功能指标，结合了短、中、长周期，通过特定的数值阈值（如50， 65， 35， 70）来判断超买超卖和交易时机。
