## 绘制散点图矩阵
除了折线图，散点图矩阵是观察变量间关系及各自分布的强大工具。在绘制之前，我们通常计算指标的日收益率（或对数收益率）以进行平稳化处理。

以下是计算对数收益率并绘制散点图矩阵的步骤：
```python
import numpy as np
import pandas as pd
from pandas.plotting import scatter_matrix

![](img/f5d41b7febc83485213d4263af2f29df_65.png)

![](img/f5d41b7febc83485213d4263af2f29df_67.png)

# 计算对数收益率
returns = np.log(data / data.shift(1))
returns = returns.dropna() # 删除第一个NaN值

# 绘制散点图矩阵
fig, ax = plt.subplots(figsize=(10, 10))
# 使用pandas的scatter_matrix函数
scatter_matrix(returns,
               alpha=0.5, # 设置点的透明度
               diagonal=‘kde’, # 对角线绘制核密度估计图
               ax=ax)
plt.show()
```
**核密度估计** 是一种用于估计随机变量概率密度函数的非参数方法，能生成平滑的分布曲线。它与直方图展示的信息类似，但更连续平滑。在对角线上，我们也可以选择绘制直方图（`diagonal=‘hist’`）。

![](img/f5d41b7febc83485213d4263af2f29df_69.png)

![](img/f5d41b7febc83485213d4263af2f29df_71.png)

通过散点图矩阵，我们可以更清晰地看到SPX收益率与VIX收益率之间的负相关关系（左下和右上的散点图呈向下倾斜的趋势）。
