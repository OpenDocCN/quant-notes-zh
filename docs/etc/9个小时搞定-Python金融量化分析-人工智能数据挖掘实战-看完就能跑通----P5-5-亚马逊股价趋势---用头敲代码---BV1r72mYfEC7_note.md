# 初始化Prophet模型，不调整任何参数

## 模型评估方法

![](img/3148727e219221000e3edec5a3752336_37.png)

运行基础模型后，我们需要评估其性能。评估主要从两个方面进行：

1.  **预测误差**：计算预测值与真实值之间的差异。
2.  **趋势方向准确性**：判断模型预测的股价涨跌方向是否正确。

以下是评估步骤的简要介绍：

首先，划分数据集。我们以2017年1月24日为界，之前3年（2014-2017）的数据作为训练集，之后1年（2017-2018）的数据作为测试集。

```python
train = df[df[‘ds’] < ‘2017-01-24’] # 训练集
test = df[df[‘ds’] >= ‘2017-01-24’] # 测试集
```

然后，使用训练集拟合模型，并预测测试集时间段的价格。

```python
model = Prophet()
model.fit(train)
future = model.make_future_dataframe(periods=365) # 预测未来一年
forecast = model.predict(future)
```

接下来，进行具体的评估计算。

![](img/3148727e219221000e3edec5a3752336_39.png)

![](img/3148727e219221000e3edec5a3752336_41.png)

### 计算预测误差

![](img/3148727e219221000e3edec5a3752336_43.png)

预测误差可以通过平均绝对误差等指标来衡量。公式如下：
**误差 = | 真实值 - 预测值 |**

我们分别计算训练集和测试集上的平均误差。

![](img/3148727e219221000e3edec5a3752336_45.png)

### 评估趋势方向准确性

趋势方向评估是看模型预测的涨跌方向是否正确。我们通过比较相邻两天的价格差值的符号来判断趋势。

1.  计算真实价格和预测价格每日的变化差值。
2.  使用 `np.sign()` 函数获取差值的符号（正数为上涨，负数为下跌）。
3.  比较真实趋势与预测趋势的符号是否一致，统计正确的比例。

`np.sign()` 函数的行为是：输入大于0返回1，小于0返回-1，等于0返回0。

评估代码执行后，我们得到类似以下的结果输出和可视化图表：

图表中，深蓝色线为预测值，黑色线为真实值。可以观察到，当前基础模型的预测值较为平缓，与真实股价的波动存在较大差距。

*   下降趋势预测正确率：约44%

这些指标表明，当前模型在测试集上的预测误差较大，趋势预测准确率接近随机猜测（50%），模型性能有较大提升空间。
