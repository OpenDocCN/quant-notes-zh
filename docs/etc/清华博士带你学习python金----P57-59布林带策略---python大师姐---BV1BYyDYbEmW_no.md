## 布林带策略原理

布林带，英文名为Bollinger Bands，也被称为布林线或保利加通道。它是一种由三条线组成的技术分析工具。

上一节我们介绍了均线等基础概念，本节中我们来看看布林带的具体构成。

布林带由三条线组成：
*   **中线**：即**均线**，通常为20日简单移动平均线。
*   **上轨线**：可视为价格的**压力线**或**阻力线**。
*   **下轨线**：可视为价格的**支撑线**。

其核心思想是：价格大部分时间会在由上下轨线构成的“通道”内波动。当价格**向上突破上轨**时，可能意味着上涨动力耗尽，面临回调压力，是卖出信号。当价格**向下跌破下轨**时，可能意味着下跌过度，获得支撑，是买入信号。

## 布林带的计算公式

理解了基本概念后，我们来看看这三条线是如何通过数学公式计算出来的。

布林带的三条线定义如下：
*   **中线 (Middle Band, MB)**：`MB = MA(CLOSE, M)`。其中`MA`是移动平均函数，`CLOSE`是收盘价序列，`M`是周期（常取20）。
*   **上轨线 (Upper Band, UB)**：`UB = MB + (N * STD(CLOSE, M))`。
*   **下轨线 (Lower Band, LB)**：`LB = MB - (N * STD(CLOSE, M))`。

公式中的关键参数：
*   `M`：计算移动平均线和标准差的周期，通常为20。
*   `N`：标准差倍数，通常为2。这个参数决定了通道的宽度。
*   `STD`：计算`M`个周期内收盘价的标准差。

这个公式的统计学基础是：对于一组服从正态分布的数据，其均值加减2倍标准差的范围，大约包含了95.4%的数据点。虽然股价分布不完全符合正态分布，但此方法仍能有效地描述价格波动的“常态范围”。

## 策略逻辑与参数影响

掌握了计算方法，我们来看看具体的交易逻辑以及如何调整策略。

布林带是一个典型的择时策略，其交易规则非常简单：
1.  **买入信号**：当股价向下跌破布林带下轨线（LB）时，执行买入操作。
2.  **卖出信号**：当股价向上突破布林带上轨线（UB）时，执行卖出操作。

![](img/84df8b1c3451c78c3ff78d7e9af54864_1.png)

以下是关于策略参数`N`的几点说明：
*   `N`值越小，通道越窄，交易信号越频繁，但可能增加误判（如在半山腰交易）的风险。
*   `N`值越大，通道越宽，交易信号越稀少，可能错过一些交易机会，但信号可能更可靠。
*   进阶用法中，上下轨可以采用不同的`N`值，以构建非对称通道，适应不同的风险偏好（例如，下轨收紧以求买入更稳妥）。

## Python代码实现

![](img/84df8b1c3451c78c3ff78d7e9af54864_5.png)

最后，我们将上述原理转化为具体的Python代码。我们将使用`pandas`库进行数据处理。

假设我们已有股票收盘价数据`close_prices`（一个`pandas.Series`对象）。

```python
import pandas as pd
import numpy as np

# 定义布林带计算函数
def calculate_bollinger_bands(close, window=20, num_std=2):
    """
    计算布林带。
    参数:
        close: 收盘价序列 (pd.Series)
        window: 移动窗口周期，默认为20
        num_std: 标准差倍数，默认为2
    返回:
        包含中线、上轨、下轨的DataFrame
    """
    # 计算中线：简单移动平均线
    middle_band = close.rolling(window=window).mean()
    
    # 计算收盘价在窗口期内的标准差
    std = close.rolling(window=window).std()
    
    # 计算上轨和下轨
    upper_band = middle_band + (std * num_std)
    lower_band = middle_band - (std * num_std)
    
    # 将结果组合成DataFrame
    bb_df = pd.DataFrame({
        'Middle': middle_band,
        'Upper': upper_band,
        'Lower': lower_band
    })
    return bb_df

# 生成交易信号函数
def generate_bollinger_signals(close, bb_df):
    """
    根据布林带生成交易信号。
    规则:
        1. 当收盘价跌破下轨时，产生买入信号 (1)
        2. 当收盘价突破上轨时，产生卖出信号 (-1)
        3. 其余情况为持仓或观望 (0)
    """
    signals = pd.Series(index=close.index, data=0) # 初始化信号序列为0
    
    # 收盘价低于下轨 -> 买入信号
    signals[close < bb_df['Lower']] = 1
    # 收盘价高于上轨 -> 卖出信号
    signals[close > bb_df['Upper']] = -1
    
    # 为了简化，我们这里假设信号发出后立即交易。
    # 在实际应用中，可能需要考虑信号点是否有效（例如，是否在窗口期之后）。
    # 同时，通常需要避免连续的同向信号，这里仅作基础演示。
    return signals
```

**代码说明**：
1.  `calculate_bollinger_bands`函数负责计算布林带的三条线。
2.  `generate_bollinger_signals`函数根据价格与上下轨的位置关系，生成交易信号（1为买入，-1为卖出，0为无操作）。
3.  在实际回测中，您需要结合这些信号、初始资金和交易规则（如每次全仓买入卖出）来模拟交易，计算收益。
