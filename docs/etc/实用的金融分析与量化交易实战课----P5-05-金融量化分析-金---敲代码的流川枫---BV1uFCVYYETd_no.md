# 量化交易实战：05：金融分析

在本节课中，我们将要学习金融分析的核心方法。金融分析是判断股票买卖时机、评估投资价值的关键手段，它帮助我们避免盲目投资，做出更理性的决策。

上一节我们介绍了金融和股票的基础知识，本节中我们来看看如何对股票进行分析。金融分析主要分为两大流派：基本面分析和技术面分析。

## 基本面分析

基本面分析的核心是评估公司的内在价值。它通过研究公司的运营状况、财务状况以及所处的宏观经济环境，来判断其股票是否值得投资。这主要基于我们之前提到的“公司自身因素”。

![](img/f52a07de9b12ac88652a795d56041396_1.png)

以下是基本面分析的三个主要层面：

![](img/f52a07de9b12ac88652a795d56041396_3.png)

1.  **宏观经济面分析**：分析国家整体的财政政策、货币政策等，判断经济大环境的好坏。例如，政策是鼓励资金流入股市还是鼓励储蓄。但需注意，宏观经济与股市的关联并非绝对。
2.  **行业分析**：评估特定行业（如IT、教育、能源）的整体发展前景和景气度。可以通过观察该行业内几只代表性股票的走势来辅助判断。
3.  **公司分析**：这是最具体的一层。投资者需要深入研究目标公司的公开信息，特别是其定期发布的财务报表（如年报、季报）。这些财报经过审计，数据相对客观。通过分析公司的盈利能力、资产状况等财务数据，结合实地考察或新闻资讯，可以判断公司运营是否健康。如果公司运营良好、持续盈利，其股票可能具有投资价值。

![](img/f52a07de9b12ac88652a795d56041396_5.png)

## 技术面分析

上一节我们介绍了如何通过公司基本面判断价值，本节中我们来看看另一种截然不同的方法。技术面分析的核心观点是：**所有影响股价的信息都已反映在市场的历史交易数据中**。因此，它不关心公司本身，而是专注于研究股价的历史走势和交易量等市场行为数据，并从中寻找规律和趋势。

技术面分析通过观察历史价格图表和一系列技术指标来预测未来价格走势。需要注意的是，历史上涨不代表未来一定上涨，因此需要借助更复杂的指标进行判断。

以下是两个最基础且重要的技术指标：

![](img/f52a07de9b12ac88652a795d56041396_7.png)

![](img/f52a07de9b12ac88652a795d56041396_9.png)

1.  **K线**
    K线图是展示股票每日价格变动的图表。一根K线包含了四个关键价格：开盘价、收盘价、最高价和最低价。
    *   **阳线**（通常为红色或空心）：表示当日股价上涨，即收盘价高于开盘价。其实体的下端是开盘价，上端是收盘价。
    *   **阴线**（通常为绿色或实心）：表示当日股价下跌，即收盘价低于开盘价。其实体的上端是开盘价，下端是收盘价。
    *   实体上下方的细线称为“影线”，上影线顶端代表最高价，下影线底端代表最低价。

    公式表示单日价格区间为：`[最低价， 最高价]`。特殊形态如“十字星”（开盘价等于收盘价）或“光头光脚线”（无影线）都传递着不同的市场信号。

2.  **移动平均线**
    移动平均线是通过计算过去一段时间内股价的平均值，并将每日的平均值连成线而形成的趋势指标。它用于平滑价格波动，显示股价的中长期趋势。
    *   **计算**：例如，5日均线（MA5）的值是包括当天在内，向前回溯5个交易日的收盘价平均值。60日均线（MA60）则是回溯60个交易日的平均值。
    *   **应用**：均线能直观展示趋势。短期均线（如MA5）对价格变化更敏感，长期均线（如MA60）则更平缓，代表长期趋势。后续课程中会详细介绍利用短期均线与长期均线交叉来制定买卖信号的“双均线策略”。

    其计算公式可简化为：`MA(N) = (P1 + P2 + ... + PN) / N`，其中 `P` 代表每日的选定价格（通常为收盘价），`N` 代表计算周期。

![](img/f52a07de9b12ac88652a795d56041396_11.png)

本节课中我们一起学习了金融分析的两大支柱：基本面分析和技术面分析。基本面分析着眼于公司的内在价值，而技术面分析则专注于市场价格行为本身。理解这两种方法，是构建量化交易策略、进行独立投资决策的重要基础。在接下来的课程中，我们将学习如何将这些分析理念转化为具体的、可执行的量化策略。
