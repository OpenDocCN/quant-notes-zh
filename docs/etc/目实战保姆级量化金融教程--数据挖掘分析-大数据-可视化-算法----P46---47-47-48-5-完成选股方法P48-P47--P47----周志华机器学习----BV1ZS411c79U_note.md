# Python量化交易：P46-P48：完成选股方法

## 概述
在本节课中，我们将学习如何将多个股票因子数据表进行拼接，并基于这些因子计算综合得分，最终完成一个完整的选股方法。该方法的核心是计算总分并排序，以筛选出得分最高的前十只股票。

---

## 数据表的拼接操作
上一节我们介绍了如何获取和处理单个因子数据。本节中，我们来看看如何将多个因子数据表合并成一张大表。

以下是拼接两个DataFrame的步骤：

1.  使用`pd.concat`函数进行拼接。
2.  将拼接后的DataFrame赋予一个新的变量名。

```python
# 假设df1和df2是两个需要拼接的DataFrame
df_combined = pd.concat([df1, df2], axis=1)
```

现在，我们已经将两个DataFrame拼接起来了。我们给这个合并后的大表起一个新名字，例如`df_combined`。

---

## 计算总分并排序
现在，我们有了一个包含所有因子数据的大表，但表中还没有综合排名的信息。接下来，我们需要计算每只股票的综合得分并进行排序。

以下是计算总分并排序的步骤：

1.  在DataFrame中新建一列，用于存放得分，初始值可以设为0。
2.  将所有因子的分数相加，得到每只股票的总分。
3.  根据总分进行降序排序，分数最高的股票排名最前。

```python
# 新建一列‘score’，初始值为0
df_combined['score'] = 0

# 计算总分：假设有6个因子列，列名为‘factor1’到‘factor6’
df_combined['score'] = df_combined[['factor1', 'factor2', 'factor3', 'factor4', 'factor5', 'factor6']].sum(axis=1)

# 根据‘score’列进行降序排序
df_sorted = df_combined.sort_values(by='score', ascending=False)
```

---

## 获取最终选股结果
完成排序后，我们需要从排序结果中提取出我们最终要选择的股票。

以下是获取最终选股结果的步骤：

1.  从排序后的DataFrame中提取出‘score’列。
2.  获取得分最高的前十只股票的代码（即索引）。
3.  将股票代码转换为列表格式并返回。

```python
# 提取排序后的‘score’列，这是一个Series对象
score_series = df_sorted['score']

![](img/89f45b8988ef7d6c6aad181c7de6e414_1.png)

# 获取得分前十的股票代码（索引），并转换为列表
top_10_stocks = score_series.index[:10].tolist()

# 返回结果
return top_10_stocks
```

至此，我们就完成了选股方法的核心函数。该函数执行一次，就能返回基于当前所有因子数据计算出的得分前十的股票列表。

## 总结
本节课中我们一起学习了如何完成一个基础的选股方法。整个过程分为三步：首先，将多个因子数据表拼接成一张总表；其次，计算每只股票在所有因子上的综合得分并进行排序；最后，提取出得分最高的前十只股票作为最终的选股结果。这个方法构成了量化选股策略的一个核心模块。
