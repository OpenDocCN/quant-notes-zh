## 概述

上一节我们介绍了价值类因子，本节我们将聚焦于质量类因子。质量类因子主要关注公司的财务健康状况和资本结构，是评估一家公司内在质量的重要依据。

## 什么是质量类因子？🔍

质量类因子指与股票的财务质量、资本结构相关的因素。这些因素主要包括公司的盈利能力、盈利稳定性、资本结构、成长性、会计质量、派息能力以及投资能力等。简单来说，质量因子衡量的是公司财务报告的质量和整体资本结构的稳健性。

以下是质量类因子常用的两个核心指标：
*   **净资产收益率 (ROE)**
*   **总资产净利率 (ROA)**

接下来，我们将详细解析这两个指标。

## 核心指标详解

### 1. 净资产收益率 (ROE)

净资产收益率，简称ROE，是衡量公司利用股东权益创造利润效率的核心指标。

*   **公式**：`ROE = (税后利润 / 所有者权益) * 100%`
*   **通俗理解**：可以将其视为公司的“投资回报率”。它表示股东每投入一元钱，公司能为其赚取多少净利润。
*   **代码中的字段名**：`roe`
*   **评价标准**：该指标值越高，通常说明公司的盈利能力越强，为股东创造价值的能力越高，股票的质量也相对更好。

### 2. 总资产净利率 (ROA)

总资产净利率，简称ROA，反映的是公司运用其全部资产（包括自有资产和借入资产）所获得利润的水平。

*   **公式**：`ROA = (净利润 / 平均资产总额) * 100%`
*   **通俗理解**：可以理解为“公司的投入产出效率”。它表示公司每使用一元的资产（无论来源），平均能获得多少元的利润。
*   **代码中的字段名**：`roa`
*   **评价标准**：ROA越高，表明公司的资产运营越有效，整体的投入产出水平越高，是一家公司经营良好的体现。

**简单记忆**：对于初学者，只需记住`ROE`和`ROA`这两个指标都是**越高越好**，它们分别从股东视角和公司整体资产视角衡量了公司的质量。

## Coding实战环节 💻

![](img/e7c99d3debf884ef9710258a675bd0e0_1.png)

了解了核心概念后，我们进入代码实战环节。我们将使用`akshare`库获取股票基本面数据，并分别根据ROE和ROA进行选股。

以下是基于净资产收益率(ROE)进行选股的示例代码：

```python
# 导入akshare库
import akshare as ak

# 获取A股所有股票的基本面数据（这里以某个历史时点为例，实际回测需调整日期）
stock_fundamentals_df = ak.stock_financial_analysis_indicator(date="20230930")

# 筛选净资产收益率(roe)大于15%的股票
high_roe_stocks = stock_fundamentals_df[stock_fundamentals_df['roe'] > 15]

# 按ROE从高到低排序，并查看前10名
top_10_roe_stocks = high_roe_stocks.sort_values(by='roe', ascending=False).head(10)
print("净资产收益率(ROE)大于15%的前10只股票：")
print(top_10_roe_stocks[['股票代码', '股票名称', 'roe']])
```

**代码说明**：

3.  使用`sort_values`方法对结果进行排序，`ascending=False`表示降序排列。
4.  `head(10)`用于获取排序后的前10条记录。
5.  在实际策略中，阈值（如15%、8%）和回测日期需要根据具体研究和市场情况调整。
