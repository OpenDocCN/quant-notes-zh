## 图表操作回顾 📊

![](img/03489b43fd066872940284f6fccf2296_10.png)

上一节我们介绍了图表的基本组成部分和多种图表类型。本节中，我们来看看每种图表的核心用途和特点。

以下是上一讲中涉及的主要图表类型及其描述：

![](img/03489b43fd066872940284f6fccf2296_12.png)

*   **柱状图**：在垂直方向进行不同类型数据的对比。
*   **条形图**：在水平方向进行不同类型数据的对比。
*   **折线图**：反映一段时间内，某一类型或多个类型数据的变化趋势。公式可表示为：`y = f(t)`，其中 `y` 代表数据值，`t` 代表时间。
*   **散点图**：反映两种数据类型之间的相关关系。
*   **气泡图**：在散点图基础上增加气泡大小维度，可反映三种数据类型之间的联系。
*   **饼图**：体现部分与整体的关系，常用于展示占比。
*   **雷达图**：反应不同类型在多个维度上的综合对比分析。
*   **面积图**：通过面积的大小来进行不同类型数据的对比。
*   **组合图**：运用两种或以上不同的图表类型组合展示数据，常用于对比数值尺度差异巨大的多类数据。

以下是数据分析的标准步骤：

![](img/03489b43fd066872940284f6fccf2296_34.png)

![](img/03489b43fd066872940284f6fccf2296_36.png)

1.  **明确问题/需求**：理解业务方需要解决的具体问题。
2.  **确定分析思路**：规划从哪些维度、采用何种方法进行分析。
3.  **数据处理（数据清洗）**：对获取的原始数据进行整理，使其变得可用、可靠。
4.  **数据分析**：基于清洗后的数据和既定思路进行深入分析。
5.  **数据可视化**：将分析结果通过图表等形式直观呈现。
6.  **报告撰写与汇报**：形成分析报告并向相关方汇报工作成果。
7.  **提出建议与落地**：根据结论提出可执行的建议方案。
8.  **效果跟进**：跟踪建议落地后的效果，必要时进行迭代分析。

其中，**数据清洗**是数据分析中至关重要且耗时较多的一环。它的目的是提升数据质量。

以下是数据清洗中常见的操作步骤（非固定顺序，视实际情况选择）：

![](img/03489b43fd066872940284f6fccf2296_38.png)

*   **选择子集**：根据需要筛选相关的数据列。
*   **列重命名**：将不合适的列名改为清晰易懂的名称。例如，在Python的pandas库中：`df.rename(columns={'old_name': 'new_name'}, inplace=True)`。
*   **删除重复值**：识别并删除完全相同的重复行。
*   **缺失值处理**：处理数据中的空值。常用方法包括：
    *   若缺失量小，可直接删除。
    *   若为文本且量不大，可根据其他列信息人为修正。
    *   若为数值且缺失量大，可采用均值或中位数填充。当数据离散程度大时，优先使用中位数。
*   **一致化处理**：将数据统一为符合要求的格式。例如，将文本型数字区间“10k-15k”转换为可用于计算的数值。
*   **异常值处理**：识别并处理明显偏离正常范围的数据点。对于数值，可通过统计描述（如最大值、最小值）结合业务经验判断；对于文本，则需人工审查。
