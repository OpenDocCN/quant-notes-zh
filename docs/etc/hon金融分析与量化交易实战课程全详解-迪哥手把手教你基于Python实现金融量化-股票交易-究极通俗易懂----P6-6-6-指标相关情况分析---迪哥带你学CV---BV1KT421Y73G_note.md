## 第三步：改进可视化——双Y轴图

![](img/250faae981394ff3307e08340562e4e9_37.png)

为了解决量纲不同的问题，我们可以使用双Y轴图。这样，`SPX`和`VIX`可以共享同一个X轴（时间），但分别使用左右两个Y轴，从而清晰地展示各自的走势。

![](img/250faae981394ff3307e08340562e4e9_39.png)

![](img/250faae981394ff3307e08340562e4e9_41.png)

![](img/250faae981394ff3307e08340562e4e9_43.png)

以下是具体操作步骤：

![](img/250faae981394ff3307e08340562e4e9_45.png)

```python
# 绘制双Y轴图，SPX用左轴，VIX用右轴
ax = data['SPX'].plot(figsize=(10,6), color='blue', legend=True)
ax2 = ax.twinx()  # 创建共享X轴的第二个Y轴
data['VIX'].plot(ax=ax2, color='green', legend=True)
ax.set_ylabel('SPX')
ax2.set_ylabel('VIX')
```

![](img/250faae981394ff3307e08340562e4e9_47.png)

为了更清晰地观察短期关系，我们可以只选取一段时间的数据（例如2010年至2012年）进行绘图。从图中可以明显观察到，当`SPX`指数下跌时，`VIX`指数往往上升，初步揭示了它们之间的负相关关系。

## 第四步：散点图与分布观察

![](img/250faae981394ff3307e08340562e4e9_53.png)

![](img/250faae981394ff3307e08340562e4e9_55.png)

除了观察时间序列上的走势，我们还可以通过散点图来观察两个变量在任意时间点上的直接关系。同时，我们也可以查看每个变量自身的分布情况。

![](img/250faae981394ff3307e08340562e4e9_57.png)

以下是具体操作步骤：

![](img/250faae981394ff3307e08340562e4e9_59.png)

```python
# 计算每日收益率（或变化率），以便分析变化量之间的关系
returns = np.log(data / data.shift(1))
returns = returns.dropna()  # 删除首行因shift产生的NaN值

![](img/250faae981394ff3307e08340562e4e9_61.png)

# 使用pandas的scatter_matrix绘制散点图矩阵
from pandas.plotting import scatter_matrix
scatter_matrix(returns, alpha=0.5, figsize=(10, 10), diagonal='kde', bins=50)
```

在这张散点图矩阵中：
*   对角线上的图是每个变量的核密度估计图（KDE），展示了该变量自身的分布形态。
*   非对角线上的图是两两变量之间的散点图。从`SPX`和`VIX`的散点图中，可以更清晰地看到数据点从左上到右下分布的负相关趋势。

![](img/250faae981394ff3307e08340562e4e9_63.png)

## 第五步：进行回归分析

![](img/250faae981394ff3307e08340562e4e9_65.png)

散点图展示了关系，但我们还需要一个量化的描述。回归分析可以帮助我们拟合出一条最能代表数据点关系的直线，并用方程和相关系数来描述这种关系。

以下是具体操作步骤：

![](img/250faae981394ff3307e08340562e4e9_67.png)

```python
# 使用statsmodels库进行线性回归分析
import statsmodels.api as sm

![](img/250faae981394ff3307e08340562e4e9_69.png)

# 准备数据：X为SPX收益率，y为VIX收益率
X = returns['SPX']
y = returns['VIX']
# 为X添加常数项（截距）
X = sm.add_constant(X)

![](img/250faae981394ff3307e08340562e4e9_71.png)

![](img/250faae981394ff3307e08340562e4e9_73.png)

# 建立普通最小二乘（OLS）模型并拟合
model = sm.OLS(y, X).fit()

![](img/250faae981394ff3307e08340562e4e9_75.png)

![](img/250faae981394ff3307e08340562e4e9_77.png)

# 打印详细的回归结果摘要
print(model.summary())
```

![](img/250faae981394ff3307e08340562e4e9_79.png)

回归结果摘要中，我们需要重点关注：
1.  **回归系数**：`const`（截距）和`SPX`的系数。`SPX`的系数为负值，证实了负相关关系。
2.  **R-squared**：表示模型对数据变异的解释程度。
3.  **P>|t|**：对应系数的P值。通常P值小于0.05，我们认为该系数是统计显著的。

![](img/250faae981394ff3307e08340562e4e9_81.png)

![](img/250faae981394ff3307e08340562e4e9_83.png)

通过回归方程 **`VIX = const + coef * SPX`**，我们可以量化地描述：`SPX`每变动一个单位，`VIX`平均会如何变动。

![](img/250faae981394ff3307e08340562e4e9_85.png)

![](img/250faae981394ff3307e08340562e4e9_87.png)

## 总结

![](img/250faae981394ff3307e08340562e4e9_89.png)

本节课中我们一起学习了金融指标相关性分析的完整流程。
1.  我们首先**选取了目标指标**（SPX和VIX）的数据。
2.  然后通过绘制**双Y轴折线图**，在时间序列上直观观察到了它们的负相关走势。
3.  接着，我们利用**散点图矩阵**，同时观察了变量的分布和两两关系，使负相关模式更加清晰。
4.  最后，我们通过**线性回归分析**，量化了这种关系，得到了描述两者关系的具体数学公式和统计指标。
