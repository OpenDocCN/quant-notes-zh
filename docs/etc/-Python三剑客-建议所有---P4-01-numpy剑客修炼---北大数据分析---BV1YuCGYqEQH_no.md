## NumPy模块简介 📦

![](img/e0ae0afb6e4023c361c6746a15b75ac6_11.png)

首先，大致介绍一下NumPy模块。你可以将NumPy理解为一个一维或多维的数组。如果你有其他编程语言（如C语言、Java）的经验，你一定知道什么是数组。如果只接触过Python，Python中没有直接称为“数组”的结构，但我们学习过列表。如果你不知道什么是数组，可以暂时将数组视为一种特殊的列表，或者将列表视为一种特殊的数组。无论是列表还是数组，它们都表示一种数据结构，即一种可以装载数据的容器。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_13.png)

因此，NumPy模块对应的容器就叫做“数组”，这个数组可以装载相关数据。暂时，我们可以把NumPy模块当作一个一维或多维的数组。

那么，什么是一维，什么是多维呢？例如，我们之前接触列表较多。如果我定义一个列表 `a_list = [1, 2, 3]`，我们可以暂且称这个列表为一维列表。什么是二维列表呢？例如 `a_list = [[1, 2, 3], [4, 5, 6]]`，这就是列表套列表，我们称之为二维列表。三维列表的含义类似，就是列表套列表再套列表。这里举的是列表的例子，我们的数组也是一样的，有一维数组，也有多维数组。

数组和列表肯定是有区别的，至于区别是什么，稍后会进行介绍。所以，暂时你只需把NumPy模块当作一个一维或多维的数组即可。这个一维或多维的数组主要装载数值型数据。因此，NumPy模块的重点操作在于它对数值进行运算。在后期数据分析中，如果我们需要对一组数值进行某种形式的运算，就可以先将这组待计算的数值装载到NumPy数组中，然后基于NumPy数组的相关特性进行运算。

**第一点，NumPy模块重在数值计算。** 这意味着，如果我们想使用NumPy模块充当容器（即数组），那么数组中存放的数据必须是数值型数据。

**第二点，NumPy模块是Python科学计算库的基础。** 这意味着NumPy模块中已经为我们封装好了各种各样基于科学计算的操作所对应的工具类。例如，后期我们想对一组数据进行反三角函数的运算，我们不需要手动实现反三角函数，而是直接调用NumPy模块中封装好的工具类，这些工具类就可以帮我们对一组数值进行反三角函数运算。

综上所述，NumPy模块的介绍重点在于两点：
1.  它是一个容器，装载的数据是数值型数据，主要用于对数值型数据进行运算。
2.  它是Python科学计算库的基础，封装了各种科学计算工具类，可以直接基于这些工具类对数组中的数值型数据进行科学计算。

## 创建NumPy数组 🛠️

介绍完NumPy模块后，接下来我们看看如何使用NumPy数组。首先，你需要创建NumPy数组。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_15.png)

### 方法一：使用 `np.array()` 创建

![](img/e0ae0afb6e4023c361c6746a15b75ac6_17.png)

首先，需要导入NumPy模块，并约定俗成地为其起别名 `np`。

```python
import numpy as np
```

模块导入后，就可以创建数组了。NumPy模块需要单独安装吗？不需要。因为我们安装好Anaconda这样的集成环境后，集成环境中已经帮我们集成好了各种模块，包括NumPy。

现在，我们来创建一个一维数组。使用 `np.array()` 函数。

```python
arr = np.array([1, 2, 3])
print(arr)
```

`array()` 函数怎么用呢？如果不知道，可以按 `Shift + Tab` 打开该函数的帮助文档，查看其参数含义。主要参数：
*   `object`: 数据源，即要放入数组的数据。
*   `dtype`: 类型，表示数组元素的数据类型。如果不指定，则默认为传入数据的原始类型。

执行上述代码，会输出类似 `[1 2 3]` 的结果，表示创建好了一个一维NumPy数组。

能创建一维数组，同样可以创建多维数组。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_19.png)

```python
arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
print(arr_2d)
```

![](img/e0ae0afb6e4023c361c6746a15b75ac6_21.png)

现在，`arr_2d` 的数据源是一个二维列表，那么创建好的就是一个二维数组。输出是一个由两行三列组成的平面，表示一个二维NumPy数组。当然，你也可以创建三维、五维甚至十维数组，但通常我们使用最多的是一维、二维和三维数组，因为维度再高就不便于人类理解了。

至此，我们已经掌握了一种创建NumPy数组的方式。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_23.png)

### 数组与列表的区别 ⚖️

![](img/e0ae0afb6e4023c361c6746a15b75ac6_25.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_27.png)

掌握了初步的数组创建方式后，我们来看一个问题：数组和列表到底有什么区别？我们说数组可以是特殊的列表，列表也可以是特殊的数组，但它们之间一定有区别。

我们来测试一下：

```python
arr_mixed = np.array([1, 2.2, 'three'])
print(arr_mixed)
print(arr_mixed.dtype) # 查看数组元素的数据类型
```

![](img/e0ae0afb6e4023c361c6746a15b75ac6_29.png)

你会发现，数组中存储的数组元素类型被统一强转成了字符串类型。这很奇怪，明明原始数据有整型、浮点型和字符串型，为什么最终都变成了字符串型？这种情况在列表中不会发生，因为列表中的元素类型可以是不同的。

在数组的演示中，我们发现：**数组中存储的数组元素类型必须是统一的。** 如果原始数组元素类型不同，最终一定会被强转成同一类型。为什么会强转成字符串呢？如果没有字符串类型会怎样？

```python
arr_mixed2 = np.array([1, 2.2, 3.3])
print(arr_mixed2)
print(arr_mixed2.dtype)
```

![](img/e0ae0afb6e4023c361c6746a15b75ac6_31.png)

你会发现，没有字符串时，数据都被转成了浮点型。这意味着数据类型是有优先级的：字符串 > 浮点型 > 整型。如果原始数据中有字符串类型，最终都转成字符串；如果没有字符串但有浮点型，都转成浮点型；如果只有整型，就保持整型。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_33.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_35.png)

这就是列表和数组的一个主要区别。

### 方法二：从外部资源创建（如图片） 🖼️

接下来，我们看看NumPy数组的另一种创建形式。我们有一个需求：将外部的一张图片读取并加载到NumPy数组中，然后尝试改变数组元素的数值，观察对原始图片的影响。

NumPy数组是一个容器，容器里装载数据。不同的容器对数据的组织形式不同。例如，列表以列表元素形式组织数据，字典以键值对形式组织数据。数据装载到不同的容器中，我们就可以基于该容器的特性对数据进行不同形式的处理。我们重点学习的是NumPy数组这个容器，看它有什么特性，以及基于这些特性可以对数组元素进行哪些处理。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_37.png)

如何将外部图片装载到NumPy数组中呢？假设当前目录下有一张图片 `1.jpg`。图片本质上是由二进制数据构成的，二进制数据也是数值型数据，可以装载到NumPy数组中。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_39.png)

在进行这个操作之前，我们需要先接触“三剑客”中的最后一个模块：`matplotlib`。这里我们先初步使用一下。

```python
import matplotlib.pyplot as plt

# 读取图片，返回一个NumPy数组
image_array = plt.imread('1.jpg')
print(image_array)
```

`plt.imread()` 读取图片后，会将图片数据放在一个NumPy数组中。`image_array` 就是返回的数组，里面装载了图片内容。打印 `image_array`，你会发现它是一个三维NumPy数组。为什么是三维？一维数组有一个中括号，二维数组有两个中括号，三维数组有三个中括号。所以返回的是一个三维数组，里面存储的就是图片数据。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_41.png)

我们可以将这个图片数据显示出来：

![](img/e0ae0afb6e4023c361c6746a15b75ac6_43.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_45.png)

```python
plt.imshow(image_array)
plt.show()
```

`plt.imshow()` 的作用是将NumPy数组进行可视化展示。因为这里的NumPy数组存储的是图片数据，所以展示出来就是原始图片。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_47.png)

现在，`image_array` 中存储的是原始图片数据。如果我们对数组进行运算，例如让每个数组元素都减去100，会怎样？

![](img/e0ae0afb6e4023c361c6746a15b75ac6_49.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_51.png)

```python
# 对图片数据进行运算
modified_image_array = image_array - 100
# 再次显示图片
plt.imshow(modified_image_array)
plt.show()
```

再次显示的图片与原始图片会有明显差异。这个差异就源于我们将图片的原始数据都减去了100。你会发现，我们似乎对图片做了某种处理。这类似于使用美图秀秀等修图软件的原理。处理的本质就是对图片数据进行某种特定形式的处理和运算。例如，瘦脸功能就是通过特定算法，定位图片中脸部的数据，然后对其进行某种算术运算，从而达到瘦脸效果。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_53.png)

因此，使用 `plt.imread()` 可以返回一个NumPy数组，数组中的内容源于外部图片。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_55.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_57.png)

### 方法三：使用NumPy内置函数创建 🔢

![](img/e0ae0afb6e4023c361c6746a15b75ac6_59.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_61.png)

最后一种创建数组的方式，是使用NumPy提供的一些内置函数。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_63.png)

以下是几个常用函数：

*   `np.zeros(shape)`: 创建指定形状的全0数组。
*   `np.ones(shape)`: 创建指定形状的全1数组。
*   `np.linspace(start, stop, num)`: 创建等差数列的一维数组。
*   `np.arange(start, stop, step)`: 创建指定范围和步长的一维数组。
*   `np.random.randint(low, high, size)`: 创建指定范围和形状的随机整数数组。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_65.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_67.png)

我们来逐一看看：

![](img/e0ae0afb6e4023c361c6746a15b75ac6_69.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_71.png)

`np.zeros()` 和 `np.ones()` 可以创建多维数组，通过 `shape` 参数指定形状。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_73.png)

```python
# 创建一个3行4列的全0二维数组
arr_zeros = np.zeros((3, 4))
print(arr_zeros)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_75.png)

# 创建一个3行4列的全1二维数组
arr_ones = np.ones((3, 4))
print(arr_ones)
```

`np.linspace()` 和 `np.arange()` 主要用于创建一维数组。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_77.png)

```python
# 创建一个从0到100，包含20个元素的等差数列数组
arr_lin = np.linspace(0, 100, 20)
print(arr_lin)

# 创建一个从10到50，步长为2的等差数列数组
arr_arange = np.arange(10, 50, 2)
print(arr_arange)
```

![](img/e0ae0afb6e4023c361c6746a15b75ac6_79.png)

![](img/e0ae0afb6e4023c361c6746a15b75ac6_81.png)

`np.random.randint()` 可以创建随机整数数组，通过 `size` 参数指定形状。

```python
# 创建一个5行3列，元素在0到100之间的随机整数数组
arr_rand = np.random.randint(0, 100, size=(5, 3))
print(arr_rand)
```

![](img/e0ae0afb6e4023c361c6746a15b75ac6_83.png)

`np.zeros()` 和 `np.ones()` 可以返回一维或多维数组，而 `np.linspace()` 和 `np.arange()` 只能返回一维数组。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_85.png)

至此，我们就有三种方式创建NumPy数组了：
1.  使用 `np.array()` 直接创建。
2.  使用 `plt.imread()` 从图片文件创建。
3.  使用NumPy内置函数（如 `zeros`, `ones`, `linspace`, `arange`, `random.randint`）创建。

这几种方式都可以返回NumPy数组。现在，我们已经可以创建NumPy数组了。创建出数组后，我们才能使用它。并且，目前我们创建的这些NumPy数组中，存储的数据都是数值型数据。

![](img/e0ae0afb6e4023c361c6746a15b75ac6_87.png)
