# Python量化交易：P45：IC指标值计算

## 概述
在本节课中，我们将学习如何计算量化交易中的一个核心指标——IC值。IC值用于衡量我们构建的因子（例如技术指标）与股票未来实际收益率之间的相关性。我们将通过获取股票价格数据、计算收益率、进行数据格式转换，最终计算出斯皮尔曼秩相关系数（IC值）。

上一节我们介绍了如何构建和计算一个自定义的因子。本节中我们来看看如何评估这个因子的有效性，即计算其IC值。

## 获取收盘价数据
要计算IC值，我们需要两样东西：因子值和股票的实际收益率。因此，我们首先需要获取股票的每日收盘价数据，以便后续计算收益率。

以下是获取收盘价数据的步骤：

1.  使用 `get_price` 函数获取指定股票池在特定时间段内的价格数据。
2.  从返回的多维价格数据中，提取出我们需要的“收盘价”这一列，并将其整理成二维的 `DataFrame` 格式。
3.  为数据框设置清晰的索引和列名，方便后续处理。

![](img/5c6aafb456bf56bb923275939d1fd68b_1.png)

```python
# 假设 stock_pool 是之前定义好的股票代码列表
# 获取2019-01-01到2020-01-01的收盘价数据
price_data = get_price(
    security=stock_pool,
    start_date='2019-01-01',
    end_date='2020-01-01',
    fields=['close'] # 指定获取收盘价
)

# 提取收盘价数据，并整理格式
close_price = price_data['close'] # 假设返回数据是字典形式
close_price.index.name = 'date'   # 设置索引名为‘日期’
close_price.columns.name = 'code' # 设置列名为‘股票代码’
```
执行完上述代码后，我们就得到了一个清晰的二维表格，索引是日期，列是股票代码，表格内的值是每日的收盘价。

![](img/5c6aafb456bf56bb923275939d1fd68b_3.png)

## 数据格式转换
获取到原始因子数据和收盘价数据后，我们需要将它们转换成量化分析库要求的特定格式。这个格式转换函数通常比较长且复杂，其核心目的是将因子值和对应的未来收益率对齐，并可能进行分组处理。

以下是关键步骤：

1.  调用库中提供的格式转换函数（例如 `convert_to_forward_returns_data`）。
2.  将我们准备好的因子数据（`factor_data`）和价格数据（`close_price`）作为参数传入。
3.  转换后的数据会包含因子值、未来不同周期（如1天、5天、10天）的收益率，以及根据因子值大小自动划分的组别（例如分为5组，组号越大代表因子值越大）。

```python
# 导入必要的工具模块
from quant_utils import convert_to_forward_returns_data

# 进行数据格式转换
# 假设 factor_series 是之前计算好的因子值序列（与收盘价数据索引对齐）
aligned_data = convert_to_forward_returns_data(
    factor=factor_series,
    prices=close_price,
    periods=(1, 5, 10) # 指定计算未来1期、5期、10期的收益率
)
```
转换后，我们得到的数据结构包含了因子值、未来收益率以及因子分组信息。例如，`1D`列代表未来1天的收益率，计算公式为：`(明日收盘价 - 今日收盘价) / 今日收盘价`。分组信息（如1到5）表示该股票当日的因子值在所有股票中的相对大小排名。

## 计算IC值
数据准备就绪后，我们就可以计算IC值了。IC值通常使用斯皮尔曼秩相关系数来计算，它衡量的是因子排名与未来收益率排名之间的单调关系。

以下是计算IC值的方法：

![](img/5c6aafb456bf56bb923275939d1fd68b_5.png)

1.  使用量化分析库中提供的性能分析函数（例如 `factor_information_coefficient`）。
2.  将上一步转换好的对齐数据（`aligned_data`）传入该函数。
3.  函数会返回一个包含每日IC值的数据序列。

![](img/5c6aafb456bf56bb923275939d1fd68b_7.png)

![](img/5c6aafb456bf56bb923275939d1fd68b_9.png)

```python
# 导入性能分析模块
from quant_performance import factor_information_coefficient

# 计算IC值序列
ic_series = factor_information_coefficient(aligned_data)

![](img/5c6aafb456bf56bb923275939d1fd68b_11.png)

# 查看前几日的IC值
print(ic_series.head())
```
输出的IC值序列，其索引是日期，值就是该日期因子与未来收益率（默认使用1期收益率）的秩相关系数。这个值介于-1到1之间，越接近1表示正相关性越强，越接近-1表示负相关性越强，接近0则表示没有单调关系。

![](img/5c6aafb456bf56bb923275939d1fd68b_13.png)

## 总结
本节课中我们一起学习了IC指标值的完整计算流程。我们首先获取了股票的收盘价数据，然后将其与因子数据一同进行格式转换与对齐，最后调用库函数计算出了衡量因子预测能力的斯皮尔曼秩相关系数（IC值）。理解并计算IC值是量化因子研究与策略评估的基础。
