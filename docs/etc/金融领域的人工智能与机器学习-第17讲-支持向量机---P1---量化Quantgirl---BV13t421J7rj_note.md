## 从最大边距分类器到支持向量分类器

![](img/497ab2a7cea793a1dc638536ef6828a9_4.png)

上一节我们介绍了最大边距分类器，它试图找到一个能完美分隔两类数据的超平面，并最大化其边距。然而，这种方法对新加入的数据点非常敏感，决策边界可能会因为少数新观测而发生显著变化。

那么，我们能否让分类器对新观测不那么敏感、更宽容一些呢？这就是支持向量分类器要解决的问题。它并非严格追求一个完全分离的超平面，而是允许一些观测被错误分类，从而获得更稳健的决策边界。

### 支持向量分类器的优化问题

![](img/497ab2a7cea793a1dc638536ef6828a9_12.png)

与最大边距分类器相比，支持向量分类器的优化问题引入了新的变量。以下是其核心公式：

我们最大化边距 `M`，同时满足以下约束：
1.  对于每个观测 `i`，有 `y_i (β_0 + β_1 x_{i1} + ... + β_p x_{ip}) >= M(1 - ε_i)`。
2.  系数平方和约束：`Σ β_j^2 = 1`。
3.  松弛变量非负：`ε_i >= 0`。
4.  总误差约束：`Σ ε_i <= C`。

![](img/497ab2a7cea793a1dc638536ef6828a9_14.png)

**关键变化**：
*   **松弛变量 (ε_i)**：允许第 `i` 个观测位于边距的错误一侧，甚至被错误分类。`ε_i` 的值衡量了该观测“犯错”的程度。
*   **超参数 (C)**：这是一个通过交叉验证选择的非负调谐参数。它控制了对误分类的容忍度总和。`C` 值越大，模型对误分类的容忍度越高，边距可能越宽；`C` 值越小，模型越严格，边距可能越窄。

![](img/497ab2a7cea793a1dc638536ef6828a9_16.png)

### 超参数C的影响

![](img/497ab2a7cea793a1dc638536ef6828a9_18.png)

以下是不同C值对决策边界影响的说明：

*   **C值很大**：模型对观测值位于边距错误一侧有很高的容忍度，因此边距会较宽。
*   **C值减小**：模型对误分类的容忍度降低，边距随之变窄。

![](img/497ab2a7cea793a1dc638536ef6828a9_20.png)

## 从线性到非线性：支持向量机

尽管支持向量分类器比最大边距分类器更灵活，但它仍然是一个线性分类器。对于许多现实世界的数据集（例如呈环形分布的数据），线性决策边界无法进行有效分类。

![](img/497ab2a7cea793a1dc638536ef6828a9_22.png)

![](img/497ab2a7cea793a1dc638536ef6828a9_24.png)

我们真正需要的是一个非线性分类器。这就是支持向量机的用武之地。支持向量机通过使用“核技巧”将数据映射到更高维的特征空间，从而在该空间中找到一个线性的分离超平面，这个超平面映射回原始空间后，就形成了非线性的决策边界。

![](img/497ab2a7cea793a1dc638536ef6828a9_26.png)

![](img/497ab2a7cea793a1dc638536ef6828a9_28.png)

### 核函数：衡量相似性的关键

核函数是支持向量机的核心。它是一个函数 `K(x_i, x_i')`，用于量化两个观测 `x_i` 和 `x_i'` 之间的相似性（或距离）。在支持向量分类器的对偶形式中，其解可以表示为训练观测之间内积的线性组合。

![](img/497ab2a7cea793a1dc638536ef6828a9_30.png)

支持向量机通过用更通用的核函数替换这个内积，实现了非线性扩展。此时，分类决策函数变为：
`f(x) = β_0 + Σ_{i ∈ S} α_i y_i K(x, x_i)`
其中 `S` 是支持向量的索引集合（通常远小于总样本数 `n`）。

![](img/497ab2a7cea793a1dc638536ef6828a9_32.png)

### 常用的核函数类型

![](img/497ab2a7cea793a1dc638536ef6828a9_34.png)

以下是几种常用的核函数：

*   **线性核**：`K(x_i, x_i') = Σ_{j=1}^p x_{ij} x_{i'j}`。这等价于标准的支持向量分类器。
*   **多项式核**：`K(x_i, x_i') = (1 + Σ_{j=1}^p x_{ij} x_{i'j})^d`。其中 `d` 是多项式的次数，它允许决策边界为多项式曲线。
*   **径向基函数核**：`K(x_i, x_i') = exp(-γ Σ_{j=1}^p (x_{ij} - x_{i'j})^2)`。这是最常用的非线性核之一，`γ` 是一个正参数，控制单个观测的影响范围。它能产生非常灵活、复杂的决策边界。

![](img/497ab2a7cea793a1dc638536ef6828a9_36.png)

在实践中，径向基函数核通常能提供出色的分类性能，尤其适用于复杂的非线性模式。

![](img/497ab2a7cea793a1dc638536ef6828a9_38.png)

## 总结

本节课中我们一起学习了支持向量机的发展脉络与应用。
1.  我们从**最大边距分类器**出发，认识到其对数据微小变化的敏感性。
2.  通过引入**松弛变量**和**超参数C**，我们得到了更稳健的**支持向量分类器**，它允许一定程度的误分类以换取更好的泛化能力。
3.  为了处理非线性可分的数据，我们引入了**核函数**。通过使用**多项式核**或**径向基函数核**等非线性核，我们将数据映射到高维空间，从而构建出**支持向量机**，它能够在原始特征空间中形成复杂的非线性决策边界。

![](img/497ab2a7cea793a1dc638536ef6828a9_40.png)

![](img/497ab2a7cea793a1dc638536ef6828a9_42.png)

支持向量机因其强大的分类能力和清晰的数学基础，在金融领域的信用评分、市场趋势分类、风险识别等诸多场景中都有广泛应用。理解其从线性到非线的扩展逻辑，是掌握这一重要工具的关键。
