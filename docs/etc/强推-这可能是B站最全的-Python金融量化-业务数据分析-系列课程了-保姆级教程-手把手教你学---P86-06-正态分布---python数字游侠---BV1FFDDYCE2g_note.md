## 从离散到连续：正态分布的概念

![](img/bc58654414142b1974b0f0b08fb487db_5.png)

上一节我们介绍了离散型随机变量的概率分布，本节中我们来看看连续型随机变量的概率分布。连续型随机变量最著名的分布就是正态分布，也称为高斯分布。

正态分布由德国数学家高斯提出。在同一时期，法国数学家拉普拉斯也独立提出过，但由于高斯名气更高，因此常被称为高斯分布。不过，我们了解这个历史背景即可。

![](img/bc58654414142b1974b0f0b08fb487db_7.png)

![](img/bc58654414142b1974b0f0b08fb487db_9.png)

那么，什么是正态分布呢？在讲解理论之前，我们先思考几个生活中的例子。

*   **身高分布**：统计中国所有男性的身高，范围大致在1.6米到1.9米之间。哪个身高区间的人数最多呢？通常是1.7米到1.8米之间。
*   **智商分布**：假设智商范围在55到145之间，大部分人的智商都集中在100左右，例如85到115之间。

![](img/bc58654414142b1974b0f0b08fb487db_11.png)

![](img/bc58654414142b1974b0f0b08fb487db_13.png)

![](img/bc58654414142b1974b0f0b08fb487db_15.png)

这些现象说明了一个共同规律：在大量观测中，处于中间水平的数据占绝大多数，极高或极低的数据则相对较少。正态分布正是描述这种“中间多，两头少”分布规律的数学模型。它的图形呈对称的钟形。

![](img/bc58654414142b1974b0f0b08fb487db_17.png)

## 连续型随机变量的概率计算

![](img/bc58654414142b1974b0f0b08fb487db_19.png)

我们回到连续型随机变量的概率计算问题。对于离散型变量，我们可以直接计算取某个特定值的概率。但对于连续型变量，由于取值是连续的，取到**任何一个精确点**的概率理论上是**0**。

那么如何计算连续型变量的概率呢？答案是计算**区间概率**。我们可以通过计算概率密度曲线下特定区间的**面积**来得到该区间对应的概率。

![](img/bc58654414142b1974b0f0b08fb487db_21.png)

![](img/bc58654414142b1974b0f0b08fb487db_23.png)

以下是理解这个过程的一种方式：
1.  想象一个数据直方图，每个长方形的面积代表该区间数据出现的频率。
2.  当我们将区间划分得无限细，直方图顶部的轮廓就会趋近于一条光滑的曲线，即概率密度曲线。
3.  整条曲线下的总面积定义为1（代表100%的概率）。
4.  要计算变量落在某个区间（如a到b）的概率，只需计算该区间对应曲线下的面积。

![](img/bc58654414142b1974b0f0b08fb487db_25.png)

因此，对于服从正态分布的变量X，计算`P(a < X < b)`就转化为计算正态分布曲线下从a到b的面积。

![](img/bc58654414142b1974b0f0b08fb487db_27.png)

## 标准正态分布：统一计算的桥梁

每个正态分布都由两个参数决定：均值（μ，决定分布中心）和方差（σ²，决定分布的离散程度）。由于均值和方差不同，正态分布曲线会有不同的位置和形状，直接计算概率很不方便。

![](img/bc58654414142b1974b0f0b08fb487db_29.png)

为了解决这个问题，我们引入**标准正态分布**。标准正态分布是一个特殊的正态分布，其**均值μ=0，标准差σ=1**。

任何服从正态分布`N(μ, σ²)`的随机变量X，都可以通过以下公式转化为服从标准正态分布`N(0, 1)`的变量Z：
**`Z = (X - μ) / σ`**
这个过程称为**标准化**。标准化后，我们可以利用统一的标准正态分布表或函数来查询概率，大大简化了计算。

![](img/bc58654414142b1974b0f0b08fb487db_31.png)

## 实战演练：计算正态分布概率

让我们通过一个例子来实践如何计算正态分布的概率。

**问题**：某零件长度服从正态分布，平均长度（μ）为10mm，标准差（σ）为0.2mm。随机抽取一件零件，其长度小于9.4mm的概率是多少？

**解题思路**：
1.  我们要求的是 `P(X < 9.4)`。
2.  首先对X=9.4进行标准化，计算Z值：
    **`Z = (9.4 - 10) / 0.2 = -3`**
3.  问题转化为求标准正态分布下 `P(Z < -3)` 的概率。
4.  查询标准正态分布表（或使用统计软件/函数）。标准正态分布表通常给出的是 `P(Z < z)` 的值。
    *   查表得，`P(Z < 3) ≈ 0.9987`。
    *   由于正态分布的对称性，`P(Z < -3) = 1 - P(Z < 3) = 1 - 0.9987 = 0.0013`。

![](img/bc58654414142b1974b0f0b08fb487db_33.png)

![](img/bc58654414142b1974b0f0b08fb487db_35.png)

![](img/bc58654414142b1974b0f0b08fb487db_37.png)

因此，随机抽取一件零件长度小于9.4mm的概率约为**0.13%**。

**如何使用标准正态分布表**：
标准正态分布表一般根据Z值查找累积概率 `Φ(z) = P(Z ≤ z)`。
*   **根据Z值查概率**：例如Z=1.96，先找到1.9所在行，再找到0.06所在列，交叉点数值0.975即为 `P(Z ≤ 1.96)`。
*   **根据概率查Z值**：例如求概率为0.975对应的Z值，在表中找到最接近0.975的概率值，其对应的行标题和列标题之和即为Z值（1.96）。

## 重要经验法则

对于任何正态分布，数据落在以均值为中心的特定倍数标准差范围内的概率是固定的。这是一个非常实用的经验法则：
*   大约 **68%** 的数据落在均值左右 **1个标准差**（μ ± σ）范围内。
*   大约 **95%** 的数据落在均值左右 **2个标准差**（μ ± 2σ）范围内。更精确地，常用 **μ ± 1.96σ**。
*   大约 **99.7%** 的数据落在均值左右 **3个标准差**（μ ± 3σ）范围内。

这个法则在快速评估数据和进行统计推断（如置信区间）时非常有用。

本节课中我们一起学习了连续型随机变量的核心分布——正态分布。

3.  我们学习了通过标准化公式 **`Z = (X - μ) / σ`**，将任何正态分布转化为均值为0、标准差为1的标准正态分布，以便于查表计算概率。
4.  我们通过实例演练了如何利用标准正态分布计算特定事件的概率。
5.  我们了解了正态分布的经验法则，它给出了数据落在均值附近不同范围内的快速概率估计。
