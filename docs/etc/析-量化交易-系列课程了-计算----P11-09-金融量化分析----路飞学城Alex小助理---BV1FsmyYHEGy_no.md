### 使用`%pdb`命令进行调试 🐛

上一节我们介绍了IPython的魔术命令。本节中，我们来看看一个非常实用的调试命令：`%pdb`。

在编写代码时，经常会遇到报错。传统的调试方法是手动添加断点，然后逐步排查。`%pdb`命令是一个开关，可以自动进入调试模式。

![](img/aba396876e8b9e1be0500fd316d39fef_5.png)

**用法**：
```python
%pdb on  # 开启自动调试模式
%pdb off # 关闭自动调试模式
```

当开启`%pdb on`后，如果你粘贴或运行一段代码，IPython会在代码报错的那一行**之前**自动暂停，并进入交互式调试器（pdb）。此时，你可以检查当前环境中的变量状态。

**示例**：
假设我们有以下测试函数：
```python
def test(a, b):
    c = a / b  # 如果b为0，这里会报错
    return c
```
运行`test(3, 0)`会引发`ZeroDivisionError`（除零错误）。在`%pdb on`模式下，错误发生前，执行会停在`c = a / b`这一行。

![](img/aba396876e8b9e1be0500fd316d39fef_7.png)

进入pdb调试模式后，可以使用`p`命令（print）查看变量：
```
p a  # 输出: 3
p b  # 输出: 0
```
通过检查变量，可以快速定位到`b`的值为0，这就是错误根源。

![](img/aba396876e8b9e1be0500fd316d39fef_9.png)

![](img/aba396876e8b9e1be0500fd316d39fef_11.png)

以下是pdb调试器的常用命令：
*   `h`： 查看帮助文档。
*   `q`： 退出调试器。
*   `n`： 执行下一行代码。
*   `break`： 设置断点。

![](img/aba396876e8b9e1be0500fd316d39fef_13.png)

对于简单的错误检查，通常使用`p`命令查看变量就足够了。更复杂的调试可以使用专门的IDE（如PyCharm），但`%pdb`在快速交互式排查中非常方便。不使用调试功能时，记得用`%pdb off`关闭它。

![](img/aba396876e8b9e1be0500fd316d39fef_15.png)

### 使用命令历史记录 🔍

IPython提供了强大的命令历史功能，类似于Linux终端。

*   按**上箭头键**可以调出上一条命令。
*   如果你只记得命令的开头几个字符，可以先输入这些字符，再按上箭头键，IPython会**自动搜索**匹配的历史命令。

例如，你输入过`a = 1`和`a * b`。在空白行输入`a`然后按上箭头，会依次出现`a * b`和`a = 1`，方便你快速找回之前的操作。

### 获取输入与输出结果 📥📤

在Python标准交互环境中，如果不将表达式结果赋值给变量，该结果不会被保存。IPython提供了便捷的方法来获取最近的计算结果。

![](img/aba396876e8b9e1be0500fd316d39fef_17.png)

![](img/aba396876e8b9e1be0500fd316d39fef_19.png)

*   **获取输出**：使用单个下划线`_`获取上一行的输出结果。使用双下划线`__`获取上上一行的输出，依此类推。
    ```python
    In [1]: 3 + 5
    Out[1]: 8

    In [2]: _ * 2  # 获取上一行的输出8，并乘以2
    Out[2]: 16
    ```
*   **获取输入**：使用`_iX`（其中X是行号）可以获取指定行的输入内容（字符串形式）。例如，`_i1`会返回第一行输入的代码`'3 + 5'`。

这个功能在你临时想查看某个计算结果，但之后又需要用到它时非常有用。

### 目录标签系统（Bookmarks） 📂

![](img/aba396876e8b9e1be0500fd316d39fef_21.png)

如果你经常需要在多个项目目录间切换，反复输入长路径会很麻烦。IPython的`%bookmark`魔术命令可以创建目录的快捷方式（书签）。

![](img/aba396876e8b9e1be0500fd316d39fef_23.png)

以下是常用操作：
```python
%bookmark proj /path/to/your/project  # 创建名为‘proj’的书签，指向你的项目路径
%bookmark -l                           # 列出所有已保存的书签
cd proj                                # 使用书签快速切换到对应目录
%bookmark -d proj                      # 删除名为‘proj’的书签
%bookmark -r                           # 删除所有书签
```
这大大简化了在复杂目录结构中的导航。

### Jupyter Notebook 🌐

最后，我们介绍一个基于Web的、更强大的交互式计算环境：Jupyter Notebook。它是IPython的扩展，尤其受数据科学和金融分析领域的青睐。

首先需要安装：
```bash
pip install jupyter
```
安装完成后，在系统命令行输入`jupyter notebook`，它会自动在浏览器中打开一个本地服务器页面。

在Jupyter Notebook中，你可以：
1.  创建“单元格”来编写和运行代码，结果（包括图表）会直接显示在下方。
2.  将单元格类型切换为Markdown，用于编写格式化的文本说明，非常适合创建包含代码、结果和注释的技术文档或报告。
3.  将Notebook文件保存为多种格式，如`.ipynb`（原生格式）、`.py`（Python脚本）、`.html`或`.pdf`。

它就像一个增强版的Web版IPython，特别适合进行探索性数据分析、教学和创建可重复的研究报告。
