## 数据获取与初步查看

![](img/3f269163d9c8f2c962795f7a543bf211_1.png)

上一节我们介绍了课程背景，本节中我们来看看如何获取股票数据。在量化分析中，首先需要获得股票的历史数据。我们主要使用Tushare这个库。

使用Tushare前，需要在官网注册以获得一个`token`（接口密钥）。获取后，将其输入代码以初始化Tushare，并配合Pandas库进行数据分析。

以下是获取“中国平安”股票数据的步骤：
1.  导入必要的库：`tushare` 和 `pandas`。
2.  设置你的Tushare `token`。
3.  调用接口，输入股票代码（例如`‘000001.SZ’`）和日期范围（例如从`‘2015-01-01’`至今）。
4.  将获取的数据打印前几行进行查看，并保存到本地CSV文件。

```python
import tushare as ts
import pandas as pd

# 设置你的token
ts.set_token('你的token')
pro = ts.pro_api()

# 获取数据
df = pro.daily(ts_code='000001.SZ', start_date='20150101', end_date='20231115')
print(df.head())

# 保存到本地
df.to_csv('pingan_stock.csv', index=False)
```

执行后，我们将看到从2015年1月1日到最近一个交易日（例如上周五11月15日）的数据。打印出的前五行会显示每日的开盘价、最高价、最低价、收盘价等信息。

## 数据信息查看与理解

获取数据后，我们需要了解其基本结构。使用Pandas的`info()`和`describe()`方法可以快速查看。

以下是查看数据信息的操作：
1.  使用 `df.info()` 查看数据的列名、数据类型和非空值数量。
2.  使用 `df.describe()` 查看数值型列的基本统计信息，如均值、标准差、最小值、最大值和分位数。

```python
# 查看数据框架信息
print(df.info())

# 查看数值列的统计描述
print(df.describe())
```

从输出可知，数据共有11列，包含`ts_code`（股票代码）、`trade_date`（交易日期）、`open`（开盘价）等变量。观察值数量为2399个。除了`ts_code`是对象类型，其他列多为数值类型。

## 数据清洗与格式化

在实际分析中，原始数据往往包含不必要的信息或格式不统一。因此，数据清洗是关键一步。

![](img/3f269163d9c8f2c962795f7a543bf211_3.png)

上一节我们查看了数据的原始样貌，本节中我们对其进行清洗和格式化。核心操作包括：选择需要的列、重命名列、转换日期格式以及设置索引。

以下是数据清洗的具体步骤：
1.  **选择列**：从11列中选取我们关心的几列，如交易日期和价格数据。
2.  **重命名列**：将英文列名改为更易懂的中文名。
3.  **转换日期格式**：将字符串格式的日期转换为Pandas的`datetime`格式，并格式化为“年-月-日”。
4.  **设置索引**：将日期列设置为数据框的索引，便于时间序列分析。

```python
# 1. 选择需要的列
df_clean = df[['trade_date', 'open', 'high', 'low', 'close', 'vol']].copy()

# 2. 重命名列
df_clean.columns = ['日期', '开盘价', '最高价', '最低价', '收盘价', '成交量']

# 3. 转换日期格式并设置为索引
df_clean['日期'] = pd.to_datetime(df_clean['日期'])
df_clean.set_index('日期', inplace=True)

# 再次查看处理后的数据
print(df_clean.head())

# 保存清洗后的数据
df_clean.to_csv('pingan_stock_cleaned.csv')
```

处理后的数据索引变为日期，列名变为中文，更便于后续分析和绘图。

## 数据可视化：绘制价格走势图

数据清洗完成后，我们可以进行初步的可视化分析。最常用的是绘制股价走势图。

以下是绘制收盘价折线图的步骤：
1.  直接使用DataFrame的`plot`方法绘制指定列（如‘收盘价’）。
2.  默认图表标签为英文，我们需要将其改为中文。这需要先配置中文字体。
3.  添加中文标题和轴标签，使图表更易读。

```python
import matplotlib.pyplot as plt

# 设置中文字体（确保系统有相应字体，如SimHei）
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 绘制收盘价走势图
df_clean['收盘价'].plot(figsize=(12,6), title='中国平安收盘价格走势')
plt.xlabel('日期')
plt.ylabel('收盘价（元）')
plt.grid(True)
plt.show()
```

生成的图表展示了中国平安从2015年1月1日至今的收盘价走势。我们可以清晰地看到价格在不同时期的变化，例如国庆节期间的市场波动。
