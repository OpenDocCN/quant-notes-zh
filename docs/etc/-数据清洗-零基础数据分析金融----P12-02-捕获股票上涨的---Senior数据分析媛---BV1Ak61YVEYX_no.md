## 需求分析 🎯

![](img/a094ed6f826e69d54ce62b89793f677c_8.png)

我们的目标是：找出目标股票（例如茅台）所有**收盘价比开盘价上涨超过3%**的日期。这个需求可以帮助我们分析股票在哪些时间段表现强势，为投资决策提供参考。

![](img/a094ed6f826e69d54ce62b89793f677c_10.png)

## 核心思路与公式 💡

要判断收盘价是否比开盘价上涨超过3%，我们需要计算每日的涨幅百分比。其核心逻辑可以用以下公式表示：

![](img/a094ed6f826e69d54ce62b89793f677c_12.png)

**涨幅公式**：
`(收盘价 - 开盘价) / 开盘价 > 0.03`

这个公式计算出的结果是一个布尔值（True或False）。True表示该日期满足条件，False则表示不满足。

![](img/a094ed6f826e69d54ce62b89793f677c_14.png)

## 实现步骤 🛠️

以下是实现该需求的具体步骤。

### 步骤一：构建布尔条件

![](img/a094ed6f826e69d54ce62b89793f677c_16.png)

首先，我们需要将上述公式转化为Pandas代码。假设我们的数据存储在名为`df`的DataFrame中，其中`‘open’`列是开盘价，`‘close’`列是收盘价。

![](img/a094ed6f826e69d54ce62b89793f677c_18.png)

```python
# 计算每日涨幅是否超过3%，得到一个布尔序列
condition = (df[‘close’] - df[‘open’]) / df[‘open’] > 0.03
```

![](img/a094ed6f826e69d54ce62b89793f677c_20.png)

执行这行代码后，`condition`变量将成为一个布尔序列（Series），其索引是日期，值为True或False。

### 步骤二：利用布尔值筛选数据

![](img/a094ed6f826e69d54ce62b89793f677c_22.png)

在数据分析中，有一个非常实用的经验：**如果得到了一个布尔序列，下一步可以立即将其作为原数据的行索引进行筛选。**

这是因为，在Pandas中，当我们将一个布尔序列放入DataFrame的`loc`索引器中时，它会自动筛选出所有对应值为True的行，而忽略值为False的行。

![](img/a094ed6f826e69d54ce62b89793f677c_24.png)

```python
# 将布尔条件作为行索引，筛选出满足条件的行
filtered_df = df.loc[condition]
```

![](img/a094ed6f826e69d54ce62b89793f677c_26.png)

执行后，`filtered_df`将只包含那些收盘价比开盘价上涨超过3%的日期的所有数据（开盘价、收盘价、最高价、最低价等）。

### 步骤三：提取目标日期

![](img/a094ed6f826e69d54ce62b89793f677c_28.png)

![](img/a094ed6f826e69d54ce62b89793f677c_30.png)

我们的最终目标不是整行数据，而是具体的日期。由于之前我们已经将日期列设置为DataFrame的索引，因此这些日期就是筛选后数据的行索引。

![](img/a094ed6f826e69d54ce62b89793f677c_32.png)

```python
# 提取满足条件的日期
result_dates = filtered_df.index
```

![](img/a094ed6f826e69d54ce62b89793f677c_34.png)

现在，`result_dates`就是我们要找的所有满足上涨条件的日期列表。

![](img/a094ed6f826e69d54ce62b89793f677c_36.png)

## 代码总结 📝

将以上步骤整合，完整的代码如下：

![](img/a094ed6f826e69d54ce62b89793f677c_38.png)

![](img/a094ed6f826e69d54ce62b89793f677c_40.png)

```python
# 步骤1: 计算布尔条件
condition = (df[‘close’] - df[‘open’]) / df[‘open’] > 0.03

# 步骤2: 利用布尔条件筛选行数据
filtered_df = df.loc[condition]

![](img/a094ed6f826e69d54ce62b89793f677c_42.png)

![](img/a094ed6f826e69d54ce62b89793f677c_44.png)

# 步骤3: 获取筛选后数据的索引（即日期）
up_dates = filtered_df.index

# 打印结果
print(“收盘价比开盘价上涨超过3%的日期有：”)
print(up_dates)
```

![](img/a094ed6f826e69d54ce62b89793f677c_46.png)
