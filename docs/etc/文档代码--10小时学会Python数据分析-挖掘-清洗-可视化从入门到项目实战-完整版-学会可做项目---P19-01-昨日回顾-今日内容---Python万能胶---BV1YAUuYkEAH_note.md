## 昨日回顾

上一节我们介绍了Excel数据分析中的两大核心内容：公式与函数，以及数据透视表。

对于公式与函数而言，地址引用是一个很重要的点。我们按照不同类型将函数分为以下几类：
*   逻辑函数
*   文本函数
*   统计函数
*   查找与引用函数
*   日期函数

每一类函数都介绍了相关的常用操作。

第二块内容是数据透视表。数据透视表是一种快速对数据进行分类汇总计算的操作方式。它在Excel中属于比较灵活高效的使用方法，也是工作中非常常见和常用的一种方式。数据透视表有其自身的优势和使用前的注意事项。我们介绍了其常用功能，包括：
*   更新操作
*   排序操作
*   切片器操作
*   计算字段与计算项
*   多重合并
*   数据透视图操作
