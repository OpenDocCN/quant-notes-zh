## 概述

上一节我们介绍了如何计算时间序列的差异值和简单增长率。本节中，我们来看看如何分析一个连续时间内的指标变化，特别是计算初始价值相对于后续每一天的累积变化倍数。

## 为什么增长率不能直接累加？

![](img/280426f36f6bbb43abc3cbcae67946f3_4.png)

首先，我们思考一个问题：能否将一段时间内每天的增长率简单相加，作为这段时间的总增长率？

以下是一个例子来说明这个问题：
*   假设一只股票第一天的价格是100元。
*   第二天价格跌至50元。
*   第三天价格涨至75元。

计算过程如下：
*   第二天的增长率 = (50 - 100) / 100 = -0.5
*   第三天的增长率 = (75 - 50) / 50 = 0.5

如果将这两个增长率相加：-0.5 + 0.5 = 0。这个结果意味着总增长率为0，即股票价格没有变化。

然而，观察原始数据：第一天100元，第三天75元，价格实际上是下跌的。这说明，**增长率不能直接进行加法运算**，因为它无法正确反映从起始点到终点的整体变化。

## 如何计算连续时间的累积变化？

对于连续时间的计算，我们需要借助数学中的对数公式。

![](img/280426f36f6bbb43abc3cbcae67946f3_10.png)

核心思想是：将对数形式的每日变化率相加，等价于计算起始日与终止日价值的直接比率。

公式如下：
`总累积变化 = log(第2天价格/第1天价格) + log(第3天价格/第2天价格) + ... + log(第N天价格/第N-1天价格)`

根据对数运算法则，这个加法等于：
`log( (第2天价格/第1天价格) * (第3天价格/第2天价格) * ... * (第N天价格/第N-1天价格) )`

![](img/280426f36f6bbb43abc3cbcae67946f3_12.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_14.png)

分子分母约简后，得到：
`log(第N天价格 / 第1天价格)`

![](img/280426f36f6bbb43abc3cbcae67946f3_16.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_18.png)

这个结果正好反映了第N天相对于第1天的整体变化。最后，我们通过指数函数 `exp()` 将这个对数结果还原为实际的价值倍数。

![](img/280426f36f6bbb43abc3cbcae67946f3_20.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_22.png)

形象地说，这相当于计算：如果初始投资为1个单位（例如1元），在经过一系列涨跌后，在最后一天它相当于多少钱。

## 代码实现

![](img/280426f36f6bbb43abc3cbcae67946f3_28.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_30.png)

以下是使用Python和Pandas实现上述计算的步骤。

### 1. 计算每日的对数变化率

![](img/280426f36f6bbb43abc3cbcae67946f3_32.png)

首先，我们需要计算后一天价格除以前一天价格，然后取对数。这里使用 `shift()` 函数来获取前一天的数据。

![](img/280426f36f6bbb43abc3cbcae67946f3_34.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_36.png)

```python
import numpy as np
import pandas as pd

# 假设 `data` 是一个包含股价的Pandas Series或DataFrame
# 计算每日变化率：当天数据 / 前一天数据
daily_ratio = data / data.shift(1)

![](img/280426f36f6bbb43abc3cbcae67946f3_38.png)

# 计算对数变化率
log_returns = np.log(daily_ratio)
```

![](img/280426f36f6bbb43abc3cbcae67946f3_40.png)

### 2. 计算累积对数收益并还原为价值

![](img/280426f36f6bbb43abc3cbcae67946f3_42.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_44.png)

接下来，我们对对数变化率进行累加，然后通过指数函数还原，得到初始1单位投资在每一天对应的价值。

![](img/280426f36f6bbb43abc3cbcae67946f3_46.png)

```python
# 计算累积对数收益
cumulative_log_returns = log_returns.cumsum()

# 通过指数函数还原，得到价值序列 (初始价值为1)
cumulative_value = np.exp(cumulative_log_returns)
```

![](img/280426f36f6bbb43abc3cbcae67946f3_48.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_50.png)

### 3. 可视化结果

最后，我们可以将计算出的价值序列绘制成图，直观展示初始投资随时间增长的情况。

![](img/280426f36f6bbb43abc3cbcae67946f3_52.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_54.png)

```python
import matplotlib.pyplot as plt

cumulative_value.plot(figsize=(10, 6))
plt.title('初始1单位投资的价值增长曲线')
plt.ylabel('价值（相对于初始1单位）')
plt.xlabel('日期')
plt.grid(True)
plt.show()
```

![](img/280426f36f6bbb43abc3cbcae67946f3_56.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_58.png)

本节课中我们一起学习了金融时间序列中连续变化情况的分析方法：

![](img/280426f36f6bbb43abc3cbcae67946f3_64.png)

![](img/280426f36f6bbb43abc3cbcae67946f3_66.png)

1.  **识别问题**：我们首先指出，简单的增长率 (`(今日-昨日)/昨日`) 不能直接累加来衡量长期整体增长。
2.  **引入解决方案**：我们引入了对数变换，通过计算每日对数收益率 (`log(今日/昨日)`) 来解决这个问题。对数收益率的累加具有实际意义，它等价于计算起始日与终止日的总比率。
3.  **实现与还原**：在代码中，我们使用 `shift()` 获取滞后数据，用 `np.log()` 计算对数收益率，用 `cumsum()` 进行累加，最后用 `np.exp()` 将累积对数收益率还原为易于理解的价值倍数序列。
4.  **结果解读**：通过绘制最终的价值曲线，我们可以直观地比较不同资产在长期内的累积增长表现，例如看到“2010年投资1元在某股票，到2018年相当于X元”。
