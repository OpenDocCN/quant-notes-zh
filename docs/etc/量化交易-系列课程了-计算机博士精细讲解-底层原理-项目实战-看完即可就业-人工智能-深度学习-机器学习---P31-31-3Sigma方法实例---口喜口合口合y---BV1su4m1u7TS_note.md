## 正态分布与极值概念

上一节我们介绍了去极值的概念，本节中我们来看看基于统计原理的3Sigma方法。首先需要理解正态分布。

正态分布，也称为高斯分布，是一种常见的连续概率分布。如果数据服从正态分布，那么大部分数据会集中在均值附近，远离均值的数据点出现的概率较低。

为了便于理解，可以想象一个例子：你去银行贷款，银行多借或少借你少量金额（比如几块钱）是常见情况，这可以看作是“正常”数据。但如果银行多借或少借你一大笔钱（比如十万八万），这种情况就很少见，可以看作是“异常”或“极值”数据。

在正态分布曲线中，越靠近中心（均值）的区域，曲线下的面积越大，代表事件发生的概率越高。越远离中心的区域，面积越小，代表事件发生的概率越低。极值点通常就落在这些远离均值的低概率区域。

## 3Sigma方法原理

![](img/5212f0a301b485a783cfd50263fff7cf_1.png)

基于上述思想，3Sigma方法的核心是利用正态分布的性质来界定数据的正常范围。

在标准正态分布中，我们以均值（μ）为中心，向左右各扩展若干倍的标准差（σ）。标准差衡量了数据的离散程度。

以下是不同倍数标准差所覆盖的数据比例：
*   **μ ± 1σ**：覆盖约 **68.3%** 的数据。
*   **μ ± 2σ**：覆盖约 **95.4%** 的数据。
*   **μ ± 3σ**：覆盖约 **99.7%** 的数据。

3Sigma方法就是指，我们设定数据的正常范围为 **μ ± 3σ**。这意味着我们认为99.7%的数据是正常的，而落在范围之外的0.3%的数据则被视为极值点，需要进行处理。

其上下限的计算公式为：
*   **上限（Upper Bound）**: `μ + n * σ`
*   **下限（Lower Bound）**: `μ - n * σ`

其中，`n` 通常取3。当 `n=1` 时，我们处理的是68.3%的核心数据。

## 代码实现3Sigma方法

![](img/5212f0a301b485a783cfd50263fff7cf_5.png)

![](img/5212f0a301b485a783cfd50263fff7cf_7.png)

理解了原理后，我们来看看如何用Python代码实现3Sigma去极值函数。

首先，我们需要计算数据的均值（mean）和标准差（std）。

![](img/5212f0a301b485a783cfd50263fff7cf_9.png)

```python
def sigma_3(data, n=3):
    # 计算数据的均值
    mean = data.mean()
    # 计算数据的标准差
    std = data.std()
```

接下来，根据公式计算上限和下限。

![](img/5212f0a301b485a783cfd50263fff7cf_11.png)

```python
    # 计算上限：均值 + n倍标准差
    upper_bound = mean + n * std
    # 计算下限：均值 - n倍标准差
    lower_bound = mean - n * std
```

![](img/5212f0a301b485a783cfd50263fff7cf_13.png)

最后，函数返回计算出的上下限。在实际处理数据时，我们会将所有大于上限的值设置为上限值，所有小于下限的值设置为下限值，从而完成去极值操作。

```python
    # 返回下限和上限
    return lower_bound, upper_bound
```
