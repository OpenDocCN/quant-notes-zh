## 股票时间序列的定义与作用

上一节我们介绍了NumPy在股票分析中的应用，本节中我们来看看Pandas库，它是金融数据分析中一个极其重要的工具。我们首先需要理解什么是股票时间序列。

时间序列是金融领域最重要的时序数据类型。股价、汇率等都是时间序列数据。时间序列不仅包括年、月、季度、周、日，甚至还有小时、分钟。最低粒度是tick数据，相当于毫秒级。

时间序列主要分析以下两个维度：

以下是趋势分析：
*   趋势分析主要分析时间序列在某一方向上持续的向上或向下运动。
*   其本质是基于统计分析来预测股价的看涨或看跌趋势。如果预测准确，我们就能盈利。

以下是相关性分析：
*   相关性分析是时间序列中最重要的特征之一。
*   例如，A事件和B事件可能相关。比如发生战争时，军工股股价会上涨，而民生相关股票可能会下跌。这就是序列的相关性分析。

## Pandas时间序列常用函数

了解了时间序列的基本概念后，本节中我们来看看Pandas库中用于处理时间序列的一些核心函数。

以下是`datetime`数据类型：
*   `datetime`是Python中表示时间的常见数据结构。
*   它被广泛应用于金融时间序列分析，因为它可以很方便地进行各种时间运算，例如取年、取月、取日、取时、取分、取秒以及时间的加减等。

以下是`loc`函数：
*   `loc`函数（以及`iloc`、`at`等）主要用于对DataFrame进行条件筛选。
*   可以将其理解为SQL语句中的`WHERE`条件。

以下是`groupby`函数：
*   `groupby`函数可能有的同学用过，有的没有。没关系，我们会在程序中演示。
*   它相当于SQL中的`GROUP BY`，主要用于对DataFrame进行分组聚合操作。

接下来，我们介绍Pandas中关于时间处理的基本函数。老规矩，我们先读取数据。

```python
# 读取数据
df = pd.read_csv('stock_data.csv')
```

然后，我们为每一列进行赋值。DataFrame的赋值非常容易，IDE提示也很清晰。

```python
# 为列赋值（假设原始数据没有列名或需要重命名）
df.columns = ['stock_id', 'date', 'close', 'open', 'high', 'low', 'volume']
```

赋值完成后，我们对时间列进行处理。需要将`date`列单独处理，先转换其时间格式。

```python
# 将‘date’列转换为datetime格式
df['date'] = pd.to_datetime(df['date'])

# 从datetime列中提取年份
df['year'] = df['date'].dt.year

# 从datetime列中提取月份
df['month'] = df['date'].dt.month

# 输出处理结果
print(df[['date', 'year', 'month']].head())
```

转换完成后，我们可以输出查看结果。可以看到，日期被正确解析，并提取出了对应的年份和月份。

下面，我们计算最低的收盘价。和之前一样，我们先准备好基本的数据结构。

```python
# 假设df已经包含必要的列
# 计算最低收盘价
min_close = df['close'].min()
print(f"最低收盘价: {min_close}")

# 找到最低收盘价所在行的索引
min_index = df['close'].idxmin()
print(f"最低收盘价的索引: {min_index}")

# 使用loc函数取出该行所有数据
min_row = df.loc[min_index]
print(f"最低收盘价所在行的数据:\n{min_row}")
```

`idxmin()`函数可以找到最小值所在的索引。然后使用`loc`函数，通过索引取出整行数据。`loc`函数也可以用于取出特定列，这里不详细展开。

接下来，我们计算每月的平均收盘价与开盘价。老规矩，先确认数据结构。

```python
# 确保数据结构包含所需列
# 计算每月平均收盘价
monthly_avg_close = df.groupby('month')['close'].mean()
print("每月平均收盘价:")
print(monthly_avg_close)

# 计算每月平均开盘价
monthly_avg_open = df.groupby('month')['open'].mean()
print("\n每月平均开盘价:")
print(monthly_avg_open)
```

这里用到了`groupby`函数，按`month`列进行分组。分组后，需要指定要计算的列（如`‘close’`），再调用`mean()`函数求平均值。

最后，我们举个例子计算股票的日涨跌幅。涨跌幅即今日收盘价减去昨日收盘价。Pandas中有一个`diff`函数可以方便地计算差值。

```python
# 计算涨跌幅（今日收盘价 - 昨日收盘价）
df['rise'] = df['close'].diff()
print("日涨跌幅（差值）:")
print(df[['date', 'close', 'rise']].head())

![](img/f191a45670157025503d2f2dd79b32a3_3.png)

# 计算涨跌幅比例（(今日收盘价 - 昨日收盘价) / 昨日收盘价）
df['rise_ratio'] = df['close'].pct_change()
print("\n日涨跌幅比例:")
print(df[['date', 'close', 'rise_ratio']].head())
```

`diff()`函数计算当前行与前一行的差值。`pct_change()`函数则直接计算变化百分比。可以看到，第一天的数据因为没有前一日数据，所以结果为NaN。

## 本章小结

本节课中我们一起学习了Pandas在股票时间序列分析中的应用。

首先，我们回顾了股票时间序列。股票时间序列是金融数据中最重要的数据结构。股价、汇率都是时间序列数据。其时间粒度可以从年、半年度、季度、月度、周度、日度，到分钟级、秒级，甚至到毫秒级的tick数据。

它能进行哪些分析呢？
*   **趋势性分析**：基于统计分析预测价格的涨跌趋势，目的是为了盈利。
*   **相关性分析**：分析不同投资标的（如不同股票）之间是正相关还是负相关。

然后，我们介绍了Pandas中用于时间序列分析的核心函数：
*   **`datetime`数据类型**：用于方便地进行时间运算，如取年、月、进行加减等。
*   **条件筛选`loc`**：用于按条件筛选数据，本次举例比较简单，实际应用可能更复杂。
*   **分组聚合`groupby`**：在时间序列中常用于按时间（如月）或按股票代码进行分组。例如，A股有数千只股票，若要计算每只股票每月的平均价格，就需要使用`groupby`。
