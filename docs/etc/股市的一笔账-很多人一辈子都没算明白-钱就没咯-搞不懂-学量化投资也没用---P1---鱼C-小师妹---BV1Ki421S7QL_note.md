# 量化投资入门：第1课：理解盈亏不对称性 📊

在本节课中，我们将要学习一个在投资中至关重要但常被忽视的核心概念：**盈亏的不对称性**。很多人即使学习复杂的量化投资策略，如果算不明白这笔基础账，最终也难以实现稳定盈利。我们将通过几个简单的算术题来揭示这个原理。

## 盈亏不对称性：三个关键问题

上一节我们介绍了盈亏不对称性的重要性，本节中我们来看看具体的例子。以下是三个基础的算术问题，它们将直观地展示盈利和亏损对最终资产的影响并不对等。

1.  假如你有100万，先涨100%，再跌50%，还剩多少？
2.  假如你有100万，先跌50%，再涨50%，还剩多少？
3.  假如你有100万，先涨50%，再跌50%，还剩多少？

## 计算结果与核心原理

现在，我们来逐一计算并分析结果。

**第一题：先涨100%，再跌50%**
*   上涨100%后，资产变为：`100万 * (1 + 100%) = 200万`。
*   再下跌50%，资产变为：`200万 * (1 - 50%) = 100万`。
*   **最终结果：100万。**

**第二题：先跌50%，再涨50%**
*   下跌50%后，资产变为：`100万 * (1 - 50%) = 50万`。
*   再上涨50%，资产变为：`50万 * (1 + 50%) = 75万`。
*   **最终结果：75万。**

**第三题：先涨50%，再跌50%**
*   上涨50%后，资产变为：`100万 * (1 + 50%) = 150万`。
*   再下跌50%，资产变为：`150万 * (1 - 50%) = 75万`。
*   **最终结果：75万。**

通过计算可以清晰地看到：**相同幅度的上涨和下跌，其最终结果并不等于回到原点**。下跌对资产的侵蚀作用远大于上涨的修复作用。这就是盈亏的不对称性。

## 应对策略：盈利储蓄与纪律性操作

理解了盈亏不对称性后，我们需要制定策略来应对。核心思想是保护已实现的利润，并利用规则进行仓位管理。

以下是基于此原理的两个关键操作原则：

1.  **盈利部分储蓄**：在上涨过程中，应将部分盈利取出储蓄，不再全部投入市场。这能确保利润落袋为安，并为后续操作储备“弹药”。
2.  **纪律性仓位管理**：
    *   **上涨逐步减仓**：随着股价上涨，按比例减少持仓。例如，上涨10%则减仓10%，上涨100%则清仓。
    *   **下跌加倍补仓**：在“不追高”和“不碰垃圾股”的前提下，当股价从高点回踩时，按回踩比例的双倍进行补仓。例如，回踩10%则补仓20%，回踩25%则补仓50%。

## 策略执行的关键

上述策略的有效执行，离不开一个关键要素：**严明的纪律性**。投资操作必须像程序一样，触发预设条件就严格执行，避免受情绪干扰。否则，策略将形同虚设，最终结果仍是“赢了又输，竹篮打水一场空”。

## 总结

本节课中我们一起学习了投资中的**盈亏不对称性**原理。我们通过简单的算术题证明，亏损比盈利对资产的影响更大。因此，成功的投资不仅需要寻找机会，更需要通过**盈利储蓄**和**纪律性的仓位管理**（上涨减仓、下跌加倍补仓）来保护本金和利润。记住，没有纪律的执行，再好的策略也无用。
