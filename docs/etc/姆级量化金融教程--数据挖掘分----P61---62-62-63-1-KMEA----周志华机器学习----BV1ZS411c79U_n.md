## 聚类是什么？

上一节我们介绍了有监督学习，其特点是数据带有标签。本节中我们来看看无监督学习中的典型代表——聚类。

聚类要解决的问题是：当我们手中没有数据标签时，如何将数据分组。其目标是将相似的数据点归为一组（称为“簇”），从而在杂乱的数据中发现内在结构。例如，在右侧的图中，算法根据数据点的分布，将相似的点自动聚集成绿色、红色和蓝色三个不同的簇。

聚类算法虽然原理相对简单，但面临两大主要挑战：
1.  **评估困难**：由于没有真实标签，很难客观评价聚类结果的好坏。
2.  **参数调节困难**：例如，改变簇的数量（K值）会得到不同结果，但缺乏标准答案来判断哪个结果更优。

## K-Means算法核心概念

![](img/3728f605aa43a7748ff3633e9f96d659_6.png)

K-Means算法是聚类算法中最简单且最实用的方法之一。理解它需要掌握几个核心概念。

![](img/3728f605aa43a7748ff3633e9f96d659_8.png)

### 1. K值（簇的数量）

![](img/3728f605aa43a7748ff3633e9f96d659_10.png)

![](img/3728f605aa43a7748ff3633e9f96d659_12.png)

K-Means算法要求我们预先指定一个参数 **K**，它代表我们希望将数据聚合成多少个簇。
*   若设定 **K=3**，算法会将所有数据点分成3个簇。
*   若设定 **K=4**，则分成4个簇。
公式表示为：`目标簇数量 = K`

### 2. 质心

![](img/3728f605aa43a7748ff3633e9f96d659_14.png)

质心是每个簇的“中心点”，其位置由该簇内所有数据点在各维度上的平均值决定。
对于一个二维数据点（X, Y），其所在簇的质心坐标计算公式为：
`质心坐标 = (mean(所有点的X值), mean(所有点的Y值))`
每个簇都有其对应的质心，它在算法迭代过程中起到关键作用。

### 3. 距离度量

为了判断数据点是否“相似”，我们需要一个度量标准。最常用的方法是**欧氏距离**。
对于两个二维点 A(x1, y1) 和 B(x2, y2)，其欧氏距离计算公式为：
`距离 = sqrt((x1 - x2)^2 + (y1 - y2)^2)`
距离越近，我们认为两点越相似。

**重要提示**：在使用距离度量前，通常需要对数据进行**标准化**（如归一化）。这是因为如果不同特征（维度）的数值范围差异巨大（例如一个特征值在0~1，另一个在100~1000），数值范围大的特征会主导距离计算，从而影响聚类效果。标准化的目的是使所有特征处于相近的数值尺度。

![](img/3728f605aa43a7748ff3633e9f96d659_16.png)

![](img/3728f605aa43a7748ff3633e9f96d659_18.png)

### 4. 优化目标

与大多数机器学习算法一样，K-Means通过优化一个目标函数来求解。其目标是：**最小化每个簇内所有样本点到该簇质心的距离总和**。
用公式表示其优化目标为：
`最小化：∑（i=1 到 K）∑（x 属于簇Ci） || x - μi ||^2`
其中：
*   `K` 是簇的总数。
*   `Ci` 代表第 i 个簇。
*   `x` 是簇 Ci 中的一个数据点。
*   `μi` 是簇 Ci 的质心。
*   `|| x - μi ||^2` 是点 x 到其质心 μi 的欧氏距离平方。

![](img/3728f605aa43a7748ff3633e9f96d659_20.png)

![](img/3728f605aa43a7748ff3633e9f96d659_22.png)

**通俗解释**：算法试图让每个簇内部的点都尽可能紧密地围绕在质心周围。如果一个点离当前簇的质心很远，但离另一个簇的质心更近，那么在迭代过程中，它就会被重新分配到更近的那个簇中去，以此优化整体目标。

## 总结

本节课中我们一起学习了聚类的基本概念以及K-Means算法的核心思想。我们了解到：
1.  聚类是一种无监督学习方法，用于将无标签的数据按相似性分组。
2.  K-Means算法需要预先指定簇的数量K。
3.  算法的核心是**质心**和基于**欧氏距离**的相似性度量。
4.  算法的优化目标是**最小化簇内样本点到质心的距离之和**，通过迭代更新质心和样本点所属簇来实现。

K-Means以其简洁高效的特点，成为最常用的聚类算法之一，为后续探索更复杂的模型奠定了基础。
