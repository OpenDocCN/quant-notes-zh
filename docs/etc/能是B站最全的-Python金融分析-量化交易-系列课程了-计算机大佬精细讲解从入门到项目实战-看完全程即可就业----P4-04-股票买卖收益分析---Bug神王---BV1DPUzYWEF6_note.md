在本节课中，我们将学习如何分析一个特定的股票买卖策略的收益情况。我们将模拟一个从2010年1月1日开始，每月第一个交易日买入一手股票，并在每年最后一个交易日卖出所有股票的投资策略，并计算其截至2020年2月的总收益。通过这个案例，我们将深入实践Pandas的时间序列数据处理、数据重采样以及索引切片等核心操作。

## 需求分析

上一节我们介绍了案例背景，本节中我们来看看如何将需求转化为具体的计算步骤。

首先，我们需要明确计算收益的几个关键要素：时间范围、买卖规则、股票数量以及价格基准。

以下是需求分析的核心要点：

![](img/e33f22c47aac35356a928abb8093e934_15.png)

1.  **时间范围**：策略执行时间为2010年1月1日至2020年2月（案例制作时）。原始数据时间范围更广，因此需要首先将数据筛选至该时间段。
2.  **买卖规则**：
    *   **买入**：每月第一个交易日，买入一手股票。
    *   **卖出**：每年最后一个交易日，卖出当年持有的所有股票。
3.  **股票数量**：在A股市场，“一手”代表100股。因此，每次买入操作是100股。
4.  **价格基准**：为简化计算，我们统一使用**开盘价**作为买卖股票的单价。
5.  **收益计算**：总收益 = 卖出股票所得总金额 + 手中剩余股票估值 - 买入股票所花总金额。

首先，我们需要将股票数据的时间范围控制在2010年至2020年之间。假设我们的原始数据`df`的行索引已经是日期时间类型。

```python
# 筛选2010-01-01至2020-02-18期间的数据
new_df = df[‘2010-01-01‘:‘2020-02-18‘]
```
执行这段代码后，`new_df`将只包含指定时间范围内的数据，这是我们后续所有计算的基础。

## 计算买入股票的总花费

![](img/e33f22c47aac35356a928abb8093e934_41.png)

![](img/e33f22c47aac35356a928abb8093e934_43.png)

接下来，我们计算投资者买入股票的总花费。根据策略，我们需要找到每月第一个交易日的数据。

![](img/e33f22c47aac35356a928abb8093e934_45.png)

![](img/e33f22c47aac35356a928abb8093e934_47.png)

我们可以使用Pandas的`resample`方法，按月份对数据进行重采样，并取每个月的第一行数据，这正好对应了每月的第一个交易日。

![](img/e33f22c47aac35356a928abb8093e934_49.png)

```python
# 获取每月第一个交易日的行数据
df_monthly = new_df.resample(‘M‘).first()
```
`df_monthly`中存储了每个月的第一个交易日对应的所有信息（开盘价、收盘价等）。注意，这里的索引显示为月末日期，但数据本身是当月的第一行。

![](img/e33f22c47aac35356a928abb8093e934_51.png)

![](img/e33f22c47aac35356a928abb8093e934_53.png)

![](img/e33f22c47aac35356a928abb8093e934_55.png)

买入股票的总花费，就是将这些行中的开盘价分别乘以100（一手），然后求和。

![](img/e33f22c47aac35356a928abb8093e934_57.png)

![](img/e33f22c47aac35356a928abb8093e934_59.png)

```python
# 计算买入股票的总花费：每月开盘价 * 100股，然后求和
cost = df_monthly[‘open‘].sum() * 100
```
变量`cost`就代表了从2010年1月到2020年2月，所有买入操作的总成本。

![](img/e33f22c47aac35356a928abb8093e934_61.png)

## 计算卖出股票的总收入

![](img/e33f22c47aac35356a928abb8093e934_63.png)

![](img/e33f22c47aac35356a928abb8093e934_65.png)

![](img/e33f22c47aac35356a928abb8093e934_67.png)

现在，我们来计算卖出股票的总收入。根据策略，我们需要找到每年最后一个交易日的数据。

![](img/e33f22c47aac35356a928abb8093e934_69.png)

![](img/e33f22c47aac35356a928abb8093e934_71.png)

![](img/e33f22c47aac35356a928abb8093e934_73.png)

同样使用`resample`方法，但这次按年份(‘A‘)进行重采样，并取每年的最后一行数据。

![](img/e33f22c47aac35356a928abb8093e934_75.png)

![](img/e33f22c47aac35356a928abb8093e934_77.png)

```python
# 获取每年最后一个交易日的行数据
df_yearly = new_df.resample(‘A‘).last()
```
这里有一个特殊情况：2020年尚未结束，因此2020年的“最后一行”数据实际上是当前日期（2020年2月18日）的数据，这笔股票并未被卖出。我们需要在计算时排除它。

![](img/e33f22c47aac35356a928abb8093e934_79.png)

![](img/e33f22c47aac35356a928abb8093e934_81.png)

```python
# 排除2020年的数据（因为2020年的股票尚未卖出）
df_yearly = df_yearly[:-1]
```
现在，`df_yearly`中存储的是2010年至2019年，每年最后一个交易日的行数据。

![](img/e33f22c47aac35356a928abb8093e934_83.png)

![](img/e33f22c47aac35356a928abb8093e934_85.png)

![](img/e33f22c47aac35356a928abb8093e934_87.png)

对于一个完整的年份，投资者会在年底卖出当年买入的所有股票。一年有12个月，每月买入100股，所以每年卖出 **1200股**。因此，卖出股票的总收入计算如下：

![](img/e33f22c47aac35356a928abb8093e934_89.png)

![](img/e33f22c47aac35356a928abb8093e934_91.png)

```python
# 计算卖出股票的总收入：每年末开盘价 * 1200股，然后求和
receive = df_yearly[‘open‘].sum() * 1200
```
变量`receive`代表了2010年至2019年，所有卖出操作获得的总现金。

![](img/e33f22c47aac35356a928abb8093e934_93.png)

![](img/e33f22c47aac35356a928abb8093e934_95.png)

![](img/e33f22c47aac35356a928abb8093e934_97.png)

## 计算剩余股票价值及总收益

![](img/e33f22c47aac35356a928abb8093e934_99.png)

![](img/e33f22c47aac35356a928abb8093e934_101.png)

截至2020年2月，投资者手中还持有2020年1月和2月买入的股票，共200股。这部分资产的价值也应计入总收益。

![](img/e33f22c47aac35356a928abb8093e934_103.png)

![](img/e33f22c47aac35356a928abb8093e934_105.png)

我们使用**最后一天的收盘价**来估算这部分剩余股票的价值。

![](img/e33f22c47aac35356a928abb8093e934_107.png)

![](img/e33f22c47aac35356a928abb8093e934_109.png)

![](img/e33f22c47aac35356a928abb8093e934_111.png)

```python
# 获取数据集中最后一天的收盘价，作为剩余股票的单价
last_price = new_df[‘close‘][-1]

# 计算剩余股票的价值：200股 * 最后一天收盘价
last_money = 200 * last_price
```
最后，我们可以计算该投资策略的总收益了。

![](img/e33f22c47aac35356a928abb8093e934_113.png)

![](img/e33f22c47aac35356a928abb8093e934_115.png)

```python
# 计算总收益 = 卖出收入 + 剩余股票价值 - 买入成本
total_profit = receive + last_money - cost
```
执行以上代码后，`total_profit`就给出了这个策略的最终收益金额。

![](img/e33f22c47aac35356a928abb8093e934_117.png)
