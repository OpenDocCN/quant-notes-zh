## 深度学习简介

上一节我们介绍了课程安排。本节中，我们来看看什么是深度学习。

深度学习是机器学习的一个分支。其核心思想是让计算机自动从数据中学习特征，而无需人工干预进行特征提取。

例如，在识别汽车的任务中：
*   **传统机器学习**：需要人工定义特征（如车轮、车窗），再让模型分类。
*   **深度学习**：模型直接从原始图像数据中自动学习并识别出“汽车”这个概念。

在量化金融中，深度学习常用于数据的分类与预测。

## 构造示例数据

为了演示深度学习，我们首先需要一组数据。以下是构造两类示例数据的核心代码：

```python
# 假设代码：生成两类围绕不同中心点的随机数据
import numpy as np
# 第一类数据，中心点为 (2, 0)
class1_data = np.random.randn(100, 2) + [2, 0]
# 第二类数据，中心点为 (-2, 0)
class2_data = np.random.randn(100, 2) + [-2, 0]
```

我们构造了两类数据。第一类以点(2, 0)为中心生成随机点，第二类以点(-2, 0)为中心生成随机点。这两类数据在X=0的轴线附近有部分重合。

直观上看，`X = 0` 这条线可以作为一个有效的分类边界，将两类数据大体分开。

## 深度学习分类过程

数据构造好以后，我们接下来进行深度学习分类。该过程主要分为两部分：使用测试集进行初步评估，以及使用神经网络模型进行正式分类。

模型训练完成后，我们会得到损失值等指标，并可以可视化分类结果。通常会产生两张图：
1.  **测试集分类结果图**：显示使用测试数据得到的分类边界。
2.  **神经网络分类结果图**：显示神经网络学习到的最终分类边界。

在我们的例子中，两种图示都显示分类边界大致在 `X = 0` 附近。但神经网络学习到的边界通常更清晰、准确，侵入对方区域的点更少。

## 与传统方法对比

![](img/3e6f8237d1334375bdbbdfffe1fddd95_7.png)

上一节我们介绍了深度学习的分类效果。本节中，我们来看看它与传统机器学习方法的对比。

我们使用逻辑回归和随机森林两种经典模型在同一数据集上进行分类。

以下是两种模型的核心调用示例：
```python
# 逻辑回归模型
from sklearn.linear_model import LogisticRegression
model_lr = LogisticRegression()
model_lr.fit(X_train, y_train)

# 随机森林模型
from sklearn.ensemble import RandomForestClassifier
model_rf = RandomForestClassifier()
model_rf.fit(X_train, y_train)
```

两种方法得到的效果相似，其分类边界也基本在 `X = 0` 这条线上。虽然可以通过更细致的指标评估哪个模型更优，但在此例中效果接近。

![](img/3e6f8237d1334375bdbbdfffe1fddd95_9.png)

这个例子说明，用深度学习进行股票数据的分类与预测，其根本原理是一致的：即找到一种最佳方式，将不同的类别尽可能地区分开来。
