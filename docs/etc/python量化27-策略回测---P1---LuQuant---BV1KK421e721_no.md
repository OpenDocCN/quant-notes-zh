在本节课中，我们将学习一个结合VWAP（成交量加权平均价）、布林带和RSI（相对强弱指数）的短线交易策略，并使用Python进行完整的策略回测。我们将从数据准备、指标计算、信号生成，一直讲解到回测执行与结果分析。

## 概述 📋

本策略旨在捕捉短期价格波动。其核心逻辑是：首先使用VWAP判断市场主要趋势，然后利用布林带寻找价格在趋势内的回调入场点，最后用RSI指标进行交易确认。这是一个快速平仓策略，平均持仓时间约51分钟，适合希望减少隔夜风险和费用的交易者。

## 生成交易信号 ⚡

有了技术指标后，我们就可以根据策略逻辑生成具体的买入和卖出信号。

策略的交易信号由三部分条件共同决定：
1.  **趋势判断（VWAP信号）**：统计最近15根K线是全部位于VWAP线上方（上升趋势）还是下方（下降趋势）。
2.  **入场点判断（布林带信号）**：在上升趋势中，寻找收盘价跌破布林带下轨的K线作为买入机会；在下降趋势中，寻找收盘价升破布林带上轨的K线作为卖出机会。
3.  **信号确认（RSI过滤）**：对于买入信号，要求RSI值低于45；对于卖出信号，要求RSI值高于55。

以下是生成总信号的函数逻辑：

```python
def generate_total_signal(row):
    # 条件1: VWAP信号。2代表上升趋势，1代表下降趋势。
    # 条件2: 布林带突破。
    # 条件3: RSI确认。
    if row['VWAP_Signal'] == 2 and row['Close'] < row['BB_lower'] and row['RSI'] < 45:
        return 2  # 买入信号
    elif row['VWAP_Signal'] == 1 and row['Close'] > row['BB_upper'] and row['RSI'] > 55:
        return 1  # 卖出信号
    else:
        return 0  # 无信号

df['Total_Signal'] = df.apply(generate_total_signal, axis=1)
```

我们可以统计信号总数，这大致等于策略在回测期间可能执行的交易次数。
