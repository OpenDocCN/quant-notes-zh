## 选择日期范围

上一节我们介绍了所需的工具包，本节中我们来看看如何获取数据。与回测框架不同，因子分析模块没有内置的每日数据获取机制。因此，我们需要自己编写代码来获取指定日期范围内的数据。

![](img/012064ff29610e1228cd1a68247cd6f1_7.png)

第一步是选择一个日期范围。我们需要获取从起始日期到结束日期之间的所有交易日数据，以便后续查询。

以下是获取交易日期的函数：

![](img/012064ff29610e1228cd1a68247cd6f1_9.png)

```python
def get_trading_date(start_date, end_date):
    """
    获取指定日期范围内的所有交易日。
    参数:
        start_date (str): 起始日期，格式为'YYYY-MM-DD'。
        end_date (str): 结束日期，格式为'YYYY-MM-DD'。
    返回:
        trading_dates (list): 交易日列表。
    """
    # 此处调用平台API获取交易日列表，示例代码
    trading_dates = ['2019-01-02', '2019-01-03', ...] # 实际应从数据源获取
    return trading_dates

![](img/012064ff29610e1228cd1a68247cd6f1_11.png)

# 示例：获取2019年全年的交易日
trading_dates = get_trading_date('2019-01-01', '2020-01-01')
```

## 构建因子查询

![](img/012064ff29610e1228cd1a68247cd6f1_17.png)

接下来，我们需要构建查询语句来获取具体的因子数据。我们以`P/E`（市盈率）因子为例。

![](img/012064ff29610e1228cd1a68247cd6f1_19.png)

![](img/012064ff29610e1228cd1a68247cd6f1_21.png)

以下是构建查询的步骤：

1.  导入查询模块。
2.  指定要查询的因子字段。
3.  使用`filter`条件限定只查询股票池中的股票。

![](img/012064ff29610e1228cd1a68247cd6f1_23.png)

```python
from jqdata import query, valuation

![](img/012064ff29610e1228cd1a68247cd6f1_25.png)

# 构建查询语句
q = query(valuation.code, valuation.pe_ratio).filter(valuation.code.in_(stock_pool))
```

![](img/012064ff29610e1228cd1a68247cd6f1_27.png)

![](img/012064ff29610e1228cd1a68247cd6f1_29.png)
