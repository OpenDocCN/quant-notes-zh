# 量化交易完全可自学教程：P44：IC指标值计算

## 概述
在本节课中，我们将学习如何计算IC指标值。IC指标，即信息系数，用于衡量因子值与股票未来收益率之间的相关性。我们将通过获取股票价格数据、计算因子值与收益率，并最终使用专用函数计算IC值来完成这一过程。

## 数据准备：获取收盘价
上一节我们介绍了如何计算因子值，本节中我们来看看如何获取计算IC值所需的股票价格数据。计算IC值需要将因子值与股票的实际收益率进行关联，因此我们需要获取股票的每日收盘价数据。

以下是获取收盘价的步骤：
1.  使用 `get_price` 函数获取指定股票池在特定时间段内的价格数据。
2.  从返回的多维数据中提取“收盘价”这一列，将其转换为二维的DataFrame格式。
3.  为DataFrame设置清晰的索引和列名，以便后续处理。

![](img/2436841acc35c6d45c82132ea5a26be6_1.png)

```python
# 获取收盘价数据示例
price_data = get_price(securities=stock_pool, start_date='2019-01-01', end_date='2020-01-01')
# 提取收盘价并整理格式
close_price = price_data['close']
close_price.index.name = 'date'
close_price.columns.name = 'code'
```

![](img/2436841acc35c6d45c82132ea5a26be6_3.png)

## 数据格式转换
获取到因子数据和收盘价数据后，我们需要将它们转换为计算IC值所需的特定格式。这一步通常需要使用一个较长的专用函数来完成。

以下是格式转换的步骤：
1.  导入必要的工具库。
2.  调用格式转换函数，将处理好的因子数据和收盘价数据作为参数传入。
3.  转换后的数据会包含日期、股票代码、未来不同周期的收益率、因子值以及因子值所属的分组区间等信息。

```python
# 数据格式转换示例（函数名较长，通常直接复制使用）
from some_module import convert_to_factor_data_format

factor_data = convert_to_factor_data_format(factor_values=processed_factor_data, prices=close_price)
```
转换后的数据结构中，`1D`、`5D`、`10D` 等列代表未来1天、5天、10天的收益率。`factor` 列是因子值。`factor_quantile` 列是因子分组，系统默认将因子值从小到大分为5组（1到5），数字越大代表因子值越大。

## 计算IC指标值
数据准备就绪后，我们就可以计算IC指标值了。IC值本质上是因子值与未来一期收益率之间的相关系数。

以下是计算IC值的步骤：
1.  在相关工具库的 `performance` 模块中找到计算IC值的函数。
2.  将上一步转换好的 `factor_data` 作为参数传入该函数。
3.  函数将返回一个包含IC值序列的结果。

![](img/2436841acc35c6d45c82132ea5a26be6_5.png)

![](img/2436841acc35c6d45c82132ea5a26be6_7.png)

![](img/2436841acc35c6d45c82132ea5a26be6_9.png)

```python
# 计算IC指标值示例
from performance_module import calculate_information_coefficient

![](img/2436841acc35c6d45c82132ea5a26be6_11.png)

ic_series = calculate_information_coefficient(factor_data=factor_data)
# 查看前几行的IC值
print(ic_series.head())
```
执行上述代码后，我们将得到一个IC值序列。这个序列中的每个值，都代表了在特定日期，所有股票的因子值与它们下一期收益率之间的相关系数。

## 总结
本节课中我们一起学习了IC指标值的完整计算流程。我们首先获取了股票的收盘价数据，然后将其与因子数据一同转换为计算所需的特定格式，最后调用专用函数计算出了IC值序列。IC值是量化选股中评估因子有效性的核心指标，理解其计算过程是构建量化策略的重要基础。
