## 矩阵特征值与奇异值分解（SVD）基础

上一节我们概述了本节课的核心目标。本节中，我们来看看支撑这些金融应用所需的数学基础：矩阵的特征值与奇异值分解。

### 线性变换与矩阵
一个矩阵可以看作一个线性变换函数。对于一个 `N x N` 的方阵 `A`，它将一个 `N` 维向量 `x` 映射为另一个 `N` 维向量 `Ax`。理解矩阵的这种变换视角是理解其特征值的基础。

### 特征值与特征向量的定义
特征值和特征向量旨在刻画矩阵的本质。对于一个给定的方阵 `A`，我们寻找满足以下等式的标量 `λ`（特征值）和非零向量 `x`（特征向量）：
**公式：`Ax = λx`**
这个等式的几何意义是：寻找那些在经过线性变换 `A` 作用后，方向保持不变（仅被拉伸或压缩 `λ` 倍）的向量 `x`。

求解特征值就是解方程 `det(A - λI) = 0`，其中 `I` 是单位矩阵。一个 `N x N` 矩阵有 `N` 个特征值（可能是实数或复数）。

### 对称矩阵的特殊性质
在应用中，我们特别关注**对称矩阵**（即 `A = A^T`）。对称矩阵有两个优美性质：
1.  其特征值均为实数。
2.  其不同特征值对应的特征向量相互正交（垂直）。

### 矩阵对角化与谱定理
对于可对角化的矩阵 `A`，可以分解为：
**公式：`A = S Σ S^(-1)`**
其中，`S` 的列是特征向量，`Σ` 是对角矩阵，对角线元素是特征值。
对于对称矩阵 `A`，由于其特征向量正交，可以写成更简洁的形式（谱定理）：
**公式：`A = Q Σ Q^T`**
这里 `Q` 是正交矩阵（`Q^T = Q^(-1)`），其列是标准正交的特征向量。

### 奇异值分解（SVD）
对于任意矩阵 `A`（不一定是方阵或对称），都可以进行奇异值分解，这是更通用的工具：
**公式：`A = U Σ V^T`**
其中，`U` 和 `V` 都是正交矩阵，`Σ` 是对角矩阵（对角线元素称为奇异值）。SVD 的统计意义与主成分分析（PCA）紧密相关。

**代码示例（Python中使用numpy进行SVD）：**
```python
import numpy as np
# 假设 A 是一个矩阵
A = np.random.randn(5, 3)
U, S, Vt = np.linalg.svd(A, full_matrices=False)
# 重建矩阵 A
Sigma = np.diag(S)
A_reconstructed = U @ Sigma @ Vt
```

## 资产组合理论：利用特征值优化权重

上一节我们介绍了矩阵分解的数学工具。本节中我们来看看如何将这些工具应用于金融领域，解决资产组合的权重配置问题。

### 问题定义
假设我们选择了 `N` 支股票进行交易，初始资金为100万。核心问题是：如何分配资金，即确定每支股票的权重 `w1, w2, ..., wN`，使得在控制风险的同时优化收益？其中，权重向量 **`w`** 满足 `sum(w_i) = 1`。

### 收益率与协方差矩阵
首先，我们基于历史数据计算每支股票（或资产）的收益率序列。对于 `T` 个时间点和 `N` 个资产，我们可以计算：
*   `N` 维平均收益率向量 **`μ`**。
*   `N x N` 维协方差矩阵 **`Σ`**，用于衡量资产之间的风险和联动关系。

### 投资组合的风险度量
现代投资组合理论指出，一个投资组合的风险（方差）可以用以下公式表示：
**公式：`风险 = w^T Σ w`**
其中 **`w`** 是权重向量（列向量），**`w^T`** 是其转置（行向量）。

### 基于特征值的最小风险组合
一个重要的结论是：**使投资组合风险 `w^T Σ w` 最小化的权重向量 `w`，恰好是协方差矩阵 `Σ` 的最小特征值所对应的特征向量**（经过归一化，使权重之和为1）。
这意味着，我们可以通过计算协方差矩阵的特征值和特征向量，直接找到理论上的最小风险组合。

### 基于优化理论的另一种方法
我们也可以将问题形式化为一个约束优化问题：
> 最小化：`w^T Σ w`
> 约束条件：`sum(w_i) = 1`
使用拉格朗日乘子法可以求解，得到解析解：**`w* = (Σ^(-1) 1) / (1^T Σ^(-1) 1)`**，其中 **`1`** 是元素全为1的向量。

## 进阶方法：基于随机矩阵理论的滤波

上一节介绍了两种基础的权重配置方法。本节中我们来看看一种更高级的滤波技术，旨在剔除市场中的随机噪声。

### 随机矩阵理论（RMT）简介
金融时间序列中既包含真实信号，也包含大量随机噪声。随机矩阵理论研究的是由随机数构成的矩阵的性质。例如，一个纯随机矩阵的特征值分布服从特定的规律（如Marchenko-Pastur分布）。

### 滤波算法步骤
我们可以利用RMT来“过滤”协方差矩阵中的随机噪声成分，步骤如下：
1.  **计算原始协方差矩阵 `Σ` 的特征值 `λ_i` 和特征向量**。
2.  **识别并滤除噪声特征值**：根据Marchenko-Pastur分布，将落在该分布典型区间内（例如，均值附近一个标准差内）的特征值视为主要代表随机噪声。将这些特征值置为零。
    **代码逻辑：**
    ```python
    for i in range(len(eigenvalues)):
        if lower_bound < eigenvalues[i] < upper_bound: # 落在噪声区间
            eigenvalues[i] = 0
    ```
3.  **重构滤波后的协方差矩阵 `Σ_filtered`**：利用修改后的特征值和对应的特征向量，通过公式 `Σ_filtered = Q Λ_filtered Q^T` 重构矩阵。
4.  **基于新矩阵计算权重**：将滤波后的 `Σ_filtered` 代入前述的最小风险组合公式或优化方法中，计算新的权重 `w_filtered`。

### 方法效果
研究表明，使用滤波后的协方差矩阵计算出的投资组合权重，其样本外表现通常优于使用原始协方差矩阵的结果，因为它减少了过拟合随机噪声的风险。

## 机器学习量化交易论文研读

理论需要结合实际研究。以下是几篇相关论文的核心要点，展示了机器学习在量化交易中的不同应用思路。

### 论文一：基于深度神经网络的高频交易
*   **核心方法**：使用5分钟级高频数据，特征维度高达9895维。采用深度神经网络模型，并使用英特尔至强融核协处理器进行大规模并行训练（耗时约10小时）。
*   **可借鉴点**：面对高维特征和大数据量时，采用Mini-batch SGD训练和专用硬件加速是必要的工程实践。
*   **批判性思考**：使用CPU集群训练深度网络效率可能低于使用GPU（如NVIDIA Tesla）。研究应注重计算效率的优化。

### 论文二：结合深度信念网络（DBN）的股价预测
*   **核心方法**：特征工程相对简单（主要使用日度和月度收益率）。模型结构是先使用深度信念网络进行无监督预训练，再用逻辑回归进行微调分类。
*   **可借鉴点**：展示了“无监督预训练+有监督微调”的经典深度学习流程在金融时序数据上的应用。
*   **批判性思考**：特征构建较为基础，模型性能提升有限（错误率从50%降至46%）。更复杂的特征工程可能带来更大收益。

### 论文三：利用社交媒体情绪进行市场预测
*   **核心方法**：分析Twitter和Google搜索中“牛市”和“熊市”相关词汇的出现频率，将其作为市场情绪指标，并研究其对未来市场走势的预测能力。
*   **主要结论**：
    1.  社交媒体情绪对市场有预测作用。
    2.  Twitter数据比Google搜索数据更准确。
    3.  Twitter上的讨论会影响Google的搜索趋势。
    4.  该结论可能不适用于中国股市（因数据源不同）。
*   **研究启示**：除了价格和成交量等传统数据，另类数据（如新闻、社交媒体）是量化研究的重要方向。研究方法（提出假设、数据验证、得出结论）具有样板参考价值。

**研读论文的建议**：
1.  **重点阅读**：引言（了解问题与框架）和实验部分（看效果与方法）。
2.  **保持批判**：论文并非金科玉律，要思考其方法的局限性与可改进之处。
3.  **注重实践**：思考如何将论文中的思想与自己已学的模型、特征工程方法结合。
