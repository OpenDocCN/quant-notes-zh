在本节课中，我们将学习如何安装 MySQL 8.0 数据库。安装 MySQL 8.0 是为了后续学习数据分析中一个重要的功能模块——窗口函数（或称开窗函数）。这个功能在 MySQL 5.7 版本中并不存在，但在 8.0 及以后的版本中得到了支持。掌握窗口函数对于数据分析工作和相关面试都至关重要。

## 为什么需要安装 MySQL 8.0？ 🤔

![](img/beae75848e1799bbf14151f6d1851e81_5.png)

![](img/beae75848e1799bbf14151f6d1851e81_6.png)

上一节我们介绍了课程的整体安排，本节中我们来看看安装 MySQL 8.0 的具体原因。在公司环境中，MySQL 5.7 版本因其稳定性仍被广泛使用。然而，在数据分析领域，**窗口函数**是一个极其有用的工具，它能够简化复杂的联表查询操作。该功能仅在 MySQL 8.0 及以上版本中提供。因此，为了学习并实践这一核心技能，我们需要安装 MySQL 8.0。

## 安装步骤详解 📥

![](img/beae75848e1799bbf14151f6d1851e81_10.png)

以下是详细的 MySQL 8.0 安装步骤，请跟随指引逐步操作。

![](img/beae75848e1799bbf14151f6d1851e81_14.png)

![](img/beae75848e1799bbf14151f6d1851e81_15.png)

![](img/beae75848e1799bbf14151f6d1851e81_17.png)

### 1. 启动安装程序
首先，找到下载好的 MySQL 8.0 安装文件（通常为 `.msi` 格式），双击启动安装程序。

### 2. 接受许可协议
安装向导启动后，首先需要同意许可条款。
*   勾选 **“I accept the license terms”**。
*   点击 **“Next”**。

### 3. 选择安装类型
这一步需要选择安装的组件。
*   选择 **“Server only”** 选项，这表示只安装 MySQL 服务器核心组件。
*   点击 **“Next”**。

![](img/beae75848e1799bbf14151f6d1851e81_19.png)

![](img/beae75848e1799bbf14151f6d1851e81_21.png)

### 4. 检查产品与路径
安装程序会显示即将安装的产品信息（MySQL Server 8.0.xx）。
*   确认安装路径，通常保持默认即可，也可以根据需要修改。
*   点击 **“Next”**。

![](img/beae75848e1799bbf14151f6d1851e81_23.png)

![](img/beae75848e1799bbf14151f6d1851e81_25.png)

### 5. 执行安装
程序将开始安装过程，请耐心等待进度条完成。
*   安装完成后，点击 **“Next”**。

![](img/beae75848e1799bbf14151f6d1851e81_27.png)

![](img/beae75848e1799bbf14151f6d1851e81_28.png)

### 6. 产品配置
进入服务器配置环节，点击 **“Next”** 继续。

### 7. 配置网络与端口
此步骤需要配置服务器端口。
*   如果您的电脑上已经安装了 MySQL 5.7（默认端口3306），为了避免冲突，需要将 MySQL 8.0 的端口修改为其他值，例如 **3308**。
*   在 **“Port Number”** 处输入新端口号。
*   点击 **“Next”**。

### 8. 选择身份验证方法
这一步关系到连接数据库时的密码加密方式。
*   为了兼容性和便于连接，请选择第二个选项：**“Use Legacy Authentication Method”**。
*   点击 **“Next”**。

![](img/beae75848e1799bbf14151f6d1851e81_30.png)

![](img/beae75848e1799bbf14151f6d1851e81_32.png)

### 9. 设置 root 用户密码
这是最关键的一步，需要为默认的 `root` 用户设置密码。
*   在 **“MySQL Root Password”** 和 **“Repeat Password”** 输入框中，输入并确认您的密码。为了便于记忆，初学者可以设置为 `123456`。
*   点击 **“Next”**。

### 10. 配置 Windows 服务
此处可以配置 MySQL 在 Windows 中的服务名称。
*   如果服务名称与已存在的 MySQL 服务冲突，可以修改为例如 `MySQL80`。
*   确保没有错误提示（感叹号）。
*   点击 **“Next”**。

### 11. 应用配置并完成安装
在最终确认界面，点击 **“Execute”** 以应用所有配置。
*   等待所有配置项执行完毕，状态变为“Complete”。
*   点击 **“Finish”** 完成安装。

## 验证安装与连接 🔗

安装完成后，我们需要验证 MySQL 8.0 是否成功运行。

以下是使用图形化管理工具（如 Navicat）进行连接的步骤：
1.  打开 Navicat。
2.  点击“连接”，选择“MySQL”。
3.  在连接设置中：
    *   **连接名**：可自定义，如 `MySQL8.0`。
    *   **主机**：`localhost` 或 `127.0.0.1`。
    *   **端口**：输入安装时设置的端口号（例如 **3308**）。
    *   **用户名**：`root`。
    *   **密码**：输入安装时设置的 `root` 用户密码。
4.  点击“测试连接”。如果显示“连接成功”，则表明 MySQL 8.0 已安装并配置完成。
