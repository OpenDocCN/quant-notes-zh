## 构建特征

![](img/1a94a9b0828d682cad12441887a2378e_6.png)

![](img/1a94a9b0828d682cad12441887a2378e_8.png)

上一节我们介绍了数据的基本处理。本节中，我们来看看如何从历史回报率数据中构建用于预测的特征。

![](img/1a94a9b0828d682cad12441887a2378e_10.png)

我们的目标是预测当天的回报率（Y）。为此，我们需要使用历史数据作为特征（X）。例如，我们可以使用前一天和前两天的回报率来预测今天的回报率。

![](img/1a94a9b0828d682cad12441887a2378e_12.png)

以下是创建特征的具体步骤：

![](img/1a94a9b0828d682cad12441887a2378e_14.png)

1.  首先，我们定义一个函数来创建特征列。该函数接收一个包含历史数据的DataFrame。
2.  在函数内部，我们通过循环创建指定数量的特征列。例如，创建“前一天回报率”和“前两天回报率”。
3.  对于每个特征，我们使用Pandas的`.shift()`方法将`returns`列的数据向后移动指定的天数，从而得到历史数据。

![](img/1a94a9b0828d682cad12441887a2378e_16.png)

![](img/1a94a9b0828d682cad12441887a2378e_18.png)

![](img/1a94a9b0828d682cad12441887a2378e_20.png)

```python
def create_features(data):
    # 初始化一个列表，用于存储新创建的列名
    cols = []
    # 假设我们创建两个特征：前一天（lag1）和前两天（lag2）
    for i in range(1, 3):
        # 生成列名，例如 ‘lag_1‘, ‘lag_2‘
        col_name = ‘lag_{}‘.format(i)
        # 将历史数据（滞后i天）作为新列添加到DataFrame中
        data[col_name] = data[‘returns‘].shift(i)
        # 记录列名
        cols.append(col_name)
    # 删除因移位操作产生的缺失值行
    data.dropna(inplace=True)
    return data, cols

![](img/1a94a9b0828d682cad12441887a2378e_22.png)

![](img/1a94a9b0828d682cad12441887a2378e_24.png)

# 应用函数
data, feature_cols = create_features(data)
```

![](img/1a94a9b0828d682cad12441887a2378e_26.png)

![](img/1a94a9b0828d682cad12441887a2378e_28.png)

执行上述代码后，我们的数据中会新增`lag_1`和`lag_2`两列，分别代表前一天和前两天的回报率。同时，我们删除了包含缺失值的行，确保数据干净可用。

## 特征可视化

![](img/1a94a9b0828d682cad12441887a2378e_36.png)

![](img/1a94a9b0828d682cad12441887a2378e_38.png)

![](img/1a94a9b0828d682cad12441887a2378e_40.png)

在成功构建特征之后，我们需要直观地理解特征与目标变量之间的关系。本节我们将通过散点图进行可视化展示。

![](img/1a94a9b0828d682cad12441887a2378e_42.png)

我们将创建一个二维散点图，其中：
*   X轴代表特征一：前一天的回报率（`lag_1`）。
*   Y轴代表特征二：前两天的回报率（`lag_2`）。
*   每个数据点的颜色代表目标变量：当天的实际回报率（`returns`）。颜色越偏向暖色（如红色），表示正收益越大；颜色越偏向冷色（如蓝色），表示负收益越大。

![](img/1a94a9b0828d682cad12441887a2378e_44.png)

![](img/1a94a9b0828d682cad12441887a2378e_46.png)

以下是绘制该图的代码：

![](img/1a94a9b0828d682cad12441887a2378e_48.png)

![](img/1a94a9b0828d682cad12441887a2378e_50.png)

```python
import matplotlib.pyplot as plt

# 绘制散点图
plt.figure(figsize=(10, 6))
# s参数指定点的大小，c参数根据‘returns‘列的值映射颜色，cmap指定颜色映射方案
scatter = plt.scatter(data[‘lag_1‘], data[‘lag_2‘], c=data[‘returns‘], cmap=‘RdYlBu‘, s=30)
# 添加颜色条，用于解读颜色对应的数值
plt.colorbar(scatter, label=‘Actual Returns‘)

![](img/1a94a9b0828d682cad12441887a2378e_52.png)

# 添加参考线
plt.axhline(y=0, color=‘r‘, linestyle=‘--‘, alpha=0.5) # 水平线 y=0
plt.axvline(x=0, color=‘r‘, linestyle=‘--‘, alpha=0.5) # 垂直线 x=0

# 添加标题和坐标轴标签
plt.title(‘Feature Visualization: Lagged Returns vs Actual Returns‘)
plt.xlabel(‘Lag 1 (Previous Day Return)‘)
plt.ylabel(‘Lag 2 (Two Days Ago Return)‘)

![](img/1a94a9b0828d682cad12441887a2378e_54.png)

![](img/1a94a9b0828d682cad12441887a2378e_56.png)

plt.show()
```

![](img/1a94a9b0828d682cad12441887a2378e_58.png)

![](img/1a94a9b0828d682cad12441887a2378e_60.png)

通过这张图，我们可以快速观察到历史回报率与当日回报率之间可能存在的模式或集群效应。例如，当`lag_1`和`lag_2`均为正时，当天的回报率（颜色）是否也倾向于为正。这为后续的模型选择和分析提供了直观的洞察。
