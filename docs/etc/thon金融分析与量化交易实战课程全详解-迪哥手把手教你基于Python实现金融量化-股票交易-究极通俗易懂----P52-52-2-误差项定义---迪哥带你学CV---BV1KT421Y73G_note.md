## 概述
在本节课程中，我们将学习线性回归模型中的一个核心概念——误差项。我们将探讨如何构建一个拟合数据的平面（即回归方程），并理解真实值与预测值之间的差异是如何被定义和衡量的。这是理解机器学习模型如何“学习”的关键一步。

---

## 构建拟合平面

上一节我们讨论了用平面来拟合数据点的想法。现在，我们来具体定义这个平面。

首先，我们写出平面的方程。我们用 **hθ(x)** 来表示当前构建的平面。其公式为：
**hθ(x) = θ₀ + θ₁x₁ + θ₂x₂**

在这个公式中：
*   **x₁** 和 **x₂** 是我们的特征，例如工资和年龄。
*   **θ₁** 和 **θ₂** 被称为**权重项**。它们决定了每个特征（如工资、年龄）对最终结果（如贷款额度）的影响程度。
*   **θ₀** 被称为**偏置项**。它的作用是**对模型结果进行微调**。无论其他特征值如何变化，改变 θ₀ 会使整个预测平面向上或向下平行移动，从而弥补模型整体的系统性偏差。

需要强调的是，偏置项的作用是**微调**，模型的核心影响因素仍然是权重项 θ₁ 和 θ₂。

---

## 转换为矩阵形式

我们之前提到，数据处理通常基于矩阵运算。然而，当前的公式 `θ₀ + θ₁x₁ + θ₂x₂` 由于存在独立的 θ₀ 项，不方便直接写成矩阵乘法形式。

为了解决这个问题，我们需要对公式进行转换。观察发现，如果能为 θ₀ 也配对一个特征 `x₀`，那么整个公式就可以统一为矩阵形式：`θ₀x₀ + θ₁x₁ + θ₂x₂`。

![](img/125fb1da115d826acb3718b380299cb3_2.png)

那么，这个新构造的 `x₀` 应该是什么呢？`x₁` 和 `x₂` 都有实际含义（如工资、年龄），但 `x₀` 没有实际物理意义，它纯粹是为了计算方便而引入的。

以下是实现转换的方法：
我们需要在原始数据中**手动添加一列**，命名为 `x₀`。为了使 `θ₀ * x₀ = θ₀` 成立，这一列的所有值都应为 **1**。

因此，在数据预处理中，经常能看到添加一列全为1的数据，其目的就是为了**将方程转换为矩阵形式，便于后续计算**，并无其他特殊含义。

![](img/125fb1da115d826acb3718b380299cb3_4.png)

![](img/125fb1da115d826acb3718b380299cb3_6.png)

转换后，我们的回归方程可以简洁地表示为：
**hθ(x) = θᵀX**
其中，`θ` 是包含所有参数 `[θ₀, θ₁, θ₂]` 的向量，`X` 是包含所有特征 `[x₀, x₁, x₂]` 的向量。`θ` 是未知的、我们需要通过机器学习求解的参数。

## 定义误差项

现在，我们有了拟合平面（预测方程），就可以评估它的好坏。让我们从单个数据点来观察。

以某一个样本（例如图中的红点）为例：
*   我们知道它的特征值 `x₁` 和 `x₂`。
*   将其代入我们的平面方程 **hθ(x)**，可以得到一个对应的**预测值**（图中蓝点）。
*   这个样本本身还有一个**真实值**（图中红点），即历史数据中实际的结果（如银行实际批准的贷款额）。

**预测值**与**真实值**之间的差异，就是我们所说的**误差项**。我们用 `error` 或 `ε` 来表示它。

对于第 `i` 个样本，其误差可以表示为：
**ε⁽ⁱ⁾ = y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾)**
其中，`y⁽ⁱ⁾` 是真实值，`hθ(x⁽ⁱ⁾)` 是预测值。

显然，对于不同的样本，其误差值通常是不同的。

## 从误差到目标函数

理解误差项后，我们就可以探讨机器是如何“学习”的。机器学习的核心是：
1.  给机器提供数据。
2.  为机器设定一个**目标**（或称损失函数、目标函数）。
3.  机器自动调整参数，以尽可能地达到这个目标。

那么，我们的目标应该是什么？对于误差项，我们是希望它越大越好，还是越小越好？

误差代表了预测的不准确性。最理想的情况是预测完全准确，即 **预测值 = 真实值**，此时**误差为零**。因此，一个最直观的目标是：**我们希望所有样本的误差总和尽可能小，越接近于零越好**。

在机器学习中，几乎所有的损失函数都遵循这个基本原则：**损失函数的值越小，代表模型拟合得越好；当损失函数为零时，代表达到了完美的拟合**。

## 总结
本节课中，我们一起学习了线性回归模型的关键组成部分：
1.  我们定义了回归方程 **hθ(x) = θ₀ + θ₁x₁ + θ₂x₂**，并解释了**权重项**和**偏置项**的作用。
2.  我们学会了通过添加值为1的 `x₀` 列，将回归方程转换为矩阵形式 **θᵀX**，以便于计算。
3.  我们引入了**误差项** **ε** 的概念，它衡量了单个样本的预测值与真实值之间的差异。
4.  最后，我们建立了误差与机器学习目标之间的联系：**模型的学习目标就是找到一组参数 θ，使得所有样本的误差总和最小化**。这为我们下一节具体构建损失函数奠定了基础。
