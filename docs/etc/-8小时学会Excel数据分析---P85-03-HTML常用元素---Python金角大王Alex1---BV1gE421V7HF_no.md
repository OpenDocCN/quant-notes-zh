## 标题与段落 📝

![](img/3666ef9605002109928c5b8c30a843e2_11.png)

标题用于定义文档的层级结构，从`<h1>`到`<h6>`，重要性依次递减。段落则使用`<p>`标签定义，每个`<p>`标签会自动在其内容前后创建换行。

![](img/3666ef9605002109928c5b8c30a843e2_13.png)

![](img/3666ef9605002109928c5b8c30a843e2_15.png)

以下是标题和段落的示例代码：
```html
<h1>这是一级标题</h1>
<h2>这是二级标题</h2>
<p>这是一个段落。</p>
<p>这是另一个段落。</p>
```
`<h1>`是最大的标题，`<h6>`是最小的标题。每个`<p>`标签代表一个独立的段落，浏览器会自动在段落之间添加间距。

## 超链接 🔗

![](img/3666ef9605002109928c5b8c30a843e2_23.png)

![](img/3666ef9605002109928c5b8c30a843e2_25.png)

![](img/3666ef9605002109928c5b8c30a843e2_27.png)

超链接用于从一个页面跳转到另一个页面或资源。其核心是`<a>`标签，通过`href`属性指定目标地址。

![](img/3666ef9605002109928c5b8c30a843e2_29.png)

![](img/3666ef9605002109928c5b8c30a843e2_31.png)

以下是创建超链接的语法：
```html
<a href="https://目标网址">链接显示的文本</a>
```
`href`是`<a>`标签的属性，类似于函数的参数，用于指定点击后要跳转的网址。

默认情况下，链接会在当前浏览器标签页中打开。如果想在新标签页中打开链接，可以添加`target="_blank"`属性。
```html
<a href="https://www.luffycity.com" target="_blank">路飞学城</a>
```
这样点击链接时，会打开一个新的浏览器标签页，而原页面保持不变。

## 显示图片 🖼️

![](img/3666ef9605002109928c5b8c30a843e2_35.png)

`<img>`标签用于在网页中嵌入图像。它使用`src`属性来指定图片文件的路径，并且是一个自闭合标签（不需要结束标签）。

![](img/3666ef9605002109928c5b8c30a843e2_43.png)

![](img/3666ef9605002109928c5b8c30a843e2_45.png)

![](img/3666ef9605002109928c5b8c30a843e2_47.png)

以下是插入图片的基本语法：
```html
<img src="图片路径/图片文件名.jpg">
```
默认情况下，图片会以其原始尺寸显示。我们可以通过`width`和`height`属性来调整其大小。通常只设置`width`，高度会按比例自动缩放，以避免图片失真。
```html
<img src="heiguniang.jpg" width="600px">
```
需要注意的是，`<img>`标签和`<a>`标签默认不会像`<p>`标签那样自动换行。如果需要在它们之后换行，可以使用`<br>`标签。
```html
<img src="photo.jpg" width="200">
<br>
<a href="link.html">这是一个链接</a>
```

## 表格 📊

表格用于展示行列数据，使用`<table>`标签定义。表格由行(`<tr>`)和单元格(`<td>`)组成。`border`属性可以为表格添加边框。

![](img/3666ef9605002109928c5b8c30a843e2_55.png)

![](img/3666ef9605002109928c5b8c30a843e2_57.png)

![](img/3666ef9605002109928c5b8c30a843e2_59.png)

以下是创建一个简单表格的代码结构：
```html
<table border="1">
  <tr>
    <td>姓名</td>
    <td>年龄</td>
  </tr>
  <tr>
    <td>张三</td>
    <td>25</td>
  </tr>
</table>
```
*   `<table>`：定义整个表格。
*   `<tr>`：定义表格中的一行。
*   `<td>`：定义表格中的一个标准单元格。

![](img/3666ef9605002109928c5b8c30a843e2_61.png)

![](img/3666ef9605002109928c5b8c30a843e2_63.png)

![](img/3666ef9605002109928c5b8c30a843e2_65.png)

为了更清晰地表示表头，可以使用`<th>`标签代替`<td>`，浏览器通常会将`<th>`的内容加粗并居中显示。
```html
<table border="1">
  <tr>
    <th>姓名</th>
    <th>年龄</th>
  </tr>
  <tr>
    <td>李四</td>
    <td>30</td>
  </tr>
</table>
```
为了让表格内容看起来不那么拥挤，可以使用`cellpadding`属性来增加单元格内边距。
```html
<table border="1" cellpadding="10">
```

## 列表 📋

![](img/3666ef9605002109928c5b8c30a843e2_73.png)

![](img/3666ef9605002109928c5b8c30a843e2_75.png)

![](img/3666ef9605002109928c5b8c30a843e2_77.png)

![](img/3666ef9605002109928c5b8c30a843e2_79.png)

HTML支持有序列表和无序列表。

![](img/3666ef9605002109928c5b8c30a843e2_81.png)

![](img/3666ef9605002109928c5b8c30a843e2_83.png)

![](img/3666ef9605002109928c5b8c30a843e2_85.png)

*   **有序列表 (`<ol>`)**: 列表项会带有数字或字母序号。
*   **无序列表 (`<ul>`)**: 列表项通常以项目符号（如圆点）开头。

![](img/3666ef9605002109928c5b8c30a843e2_87.png)

两种列表的列表项都使用`<li>`标签定义。

![](img/3666ef9605002109928c5b8c30a843e2_89.png)

![](img/3666ef9605002109928c5b8c30a843e2_91.png)

以下是有序列表和无序列表的示例：
```html
<!-- 有序列表 -->
<ol>
  <li>第一项</li>
  <li>第二项</li>
</ol>

![](img/3666ef9605002109928c5b8c30a843e2_93.png)

![](img/3666ef9605002109928c5b8c30a843e2_95.png)

![](img/3666ef9605002109928c5b8c30a843e2_97.png)

![](img/3666ef9605002109928c5b8c30a843e2_99.png)

<!-- 无序列表 -->
<ul>
  <li>苹果</li>
  <li>香蕉</li>
</ul>
```
有序列表可以通过`start`属性指定起始序号。
```html
<ol start="50">
  <li>第五十项</li>
  <li>第五十一项</li>
</ol>
```
列表还可以进行嵌套，例如在某个列表项下再创建一个子列表。
```html
<ul>
  <li>咖啡
    <ul>
      <li>星巴克</li>
      <li>Costa</li>
    </ul>
  </li>
  <li>茶</li>
</ul>
```
