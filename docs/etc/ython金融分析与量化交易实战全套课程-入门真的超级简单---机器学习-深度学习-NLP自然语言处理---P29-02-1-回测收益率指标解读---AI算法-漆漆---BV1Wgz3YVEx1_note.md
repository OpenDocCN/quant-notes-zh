## 回测收益率

回测收益率是最基本、最重要的评估指标。它衡量的是，在回测周期内，策略最终获得的收益相对于初始本金的比率。

例如，你基于2015年至2020年的股票数据设计了一个策略，规定了买入和卖出的时机。回测收益率告诉你，执行这个策略后，你赚取的钱相对于最初投入的本金增加了多少百分比。图中显示约为4%，这意味着多年操作下来，你的资产相对于本金增长了约4%。

![](img/b692a2059082dcb8afb4b510b7c03610_23.png)

![](img/b692a2059082dcb8afb4b510b7c03610_23.png)

其计算公式如下：

![](img/b692a2059082dcb8afb4b510b7c03610_25.png)

**回测收益率 = (期末资产总值 - 期初本金) / 期初本金**

![](img/b692a2059082dcb8afb4b510b7c03610_27.png)

用符号表示为：
**Return = (P_t - P_0) / P_0**

其中：
*   **P_t** 代表回测结束时的资产总值（卖出所有头寸后的总金额）。
*   **P_0** 代表回测开始时的初始本金。

这个指标直观地反映了策略的整体盈利能力。

![](img/b692a2059082dcb8afb4b510b7c03610_27.png)

接下来，我们通过Python代码演示如何计算并可视化回测收益率的变化过程。我们不仅可以计算最终的总收益率，还可以观察收益率在整个回测周期内的走势。

计算思路是：计算每个时间点的资产净值与初始净值的比值，这个比值序列可以反映资产的相对增长情况。最后一天的比值减1，即为总回测收益率。

```python
# 假设‘close’列是我们的策略净值或股价，这里用它代表资产价值
price_series = data['close']

![](img/b692a2059082dcb8afb4b510b7c03610_29.png)

# 计算每个时间点相对于初始点的比值
cumulative_return = price_series / price_series.iloc[0]

![](img/b692a2059082dcb8afb4b510b7c03610_31.png)

# 绘制资产净值变化曲线
plt.figure(figsize=(12, 6))
cumulative_return.plot()
plt.title('策略净值相对增长曲线')
plt.xlabel('日期')
plt.ylabel('净值比 (初始值=1)')
plt.grid(True)
plt.show()
```

上图展示了资产净值相对于期初（设为1）的变化过程。现在，我们来计算最终的回测收益率。

![](img/b692a2059082dcb8afb4b510b7c03610_39.png)

```python
# 计算最终回测收益率：最后一天的净值比减去1
final_return = cumulative_return.iloc[-1] - 1

# 将结果整理成DataFrame以便清晰展示
result_df = pd.DataFrame({
    '回测收益率': [final_return]
}, index=['最终结果'])

![](img/b692a2059082dcb8afb4b510b7c03610_41.png)

print(result_df)
```
