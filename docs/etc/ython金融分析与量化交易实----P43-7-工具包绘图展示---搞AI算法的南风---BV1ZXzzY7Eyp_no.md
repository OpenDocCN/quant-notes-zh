## 绘图展示IC值
上一节我们介绍了如何计算因子与收益率之间的相关系数（IC值）。计算完成后，我们通常希望IC值越大越好，因为这表示因子与收益率的相关性越强。然而，IC值每天都会变化，仅从数值结果观察不够直观。因此，本节我们将学习如何将IC值绘制成图表进行可视化分析。

![](img/c2a32f155b22c9e72ea5e26029228712_1.png)

我们引入一个名为`proteint`的工具包来帮助我们进行展示。以下是绘制IC值时间序列图的具体步骤。

首先，我们需要从`proteint`工具包中调用绘图函数。通过查阅API文档，我们找到绘制IC时间序列图的函数是`plot_ic_ts`。

![](img/c2a32f155b22c9e72ea5e26029228712_3.png)

```python
# 导入绘图工具包
import proteint as pt

# 绘制IC值时间序列图
pt.plotting.plot_ic_ts(ic)
```

执行以上代码后，系统会自动生成IC值的时间序列图。

## 图表解读
生成的图表包含以下关键元素：
*   **蓝色曲线**：代表每日计算出的实际IC值，其波动范围通常较大。
*   **绿色曲线**：代表以月为周期计算的IC值移动平均线。它平滑了每日的波动，更能反映趋势。
*   **统计信息**：图表通常会标注IC值的均值（Mean）和标准差（Std），并计算信息比率（IR = Mean / Std）。

在分析时，我们主要观察绿色移动平均线。因为蓝色实际值波动剧烈，难以判断长期趋势。通过观察绿色曲线，我们可以评估因子在一段时间内的平均表现是否稳定且显著。

当前图表显示，绿色移动平均线整体较为平稳，且数值偏小。这表明该因子与收益率的相关性不够强，趋势性不明显，可能不是我们寻找的有效因子。我们期望找到一个IC值较大且趋势向上的因子。

## 补充概念：信息比率
信息比率（Information Ratio, IR）是评估因子稳定性的一个重要指标。其计算公式为：
**IR = IC均值 / IC标准差**
这个比值描述了因子IC值的稳定程度。比值越大，说明因子的IC值越稳定（标准差小），预测能力的一致性越好；比值越小，则说明因子的表现波动较大。

## 其他可视化工具
除了时间序列图，`proteint`工具包还提供了多种其他分析图表，例如：
*   **直方图（Histogram）**：用于观察IC值的分布情况。
*   **QQ图（Q-Q Plot）**：用于检验IC值是否服从正态分布。
*   **热力图（Heatmap）**：可以用不同颜色直观展示不同周期或不同因子间的IC值矩阵。
