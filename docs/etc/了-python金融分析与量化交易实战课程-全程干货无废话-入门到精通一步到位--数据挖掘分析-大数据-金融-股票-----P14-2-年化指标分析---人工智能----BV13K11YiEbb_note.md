## 回测收益率
回测收益率是评估策略效果最直接的指标。它表示在整个回测周期结束后，你的初始投资总共增长或亏损了多少。

![](img/b4fb986685e38c922228c24b1cc0c6a7_10.png)

其计算公式非常简单：
**回测收益率 = (最终总资产 - 初始总资产) / 初始总资产**

![](img/b4fb986685e38c922228c24b1cc0c6a7_12.png)

![](img/b4fb986685e38c922228c24b1cc0c6a7_14.png)

这个值可能是正的（盈利），也可能是负的（亏损）。它直接反映了策略在历史数据上的整体盈亏情况。

![](img/b4fb986685e38c922228c24b1cc0c6a7_16.png)

以下是计算回测收益率后可能得到的结果示例：
*   有些策略的回测收益率是负的，这表明投资存在风险，并非稳赚不赔。
*   有些策略的回测收益率非常高，甚至可能翻倍。
*   剩余的策略收益率可能相对较低。

![](img/b4fb986685e38c922228c24b1cc0c6a7_18.png)

这个指标是策略分析中最重要的一个。当你设计好一个策略后，首先需要看的就是这个策略能让你赚多少钱。

## 年化收益率
理解了回测收益率后，我们来看另一个关键指标：年化收益率。

![](img/b4fb986685e38c922228c24b1cc0c6a7_22.png)

![](img/b4fb986685e38c922228c24b1cc0c6a7_24.png)

你可能会想，我已经知道总共赚了多少，为什么还需要年化收益？这是因为回测周期可能很长，比如8年或10年。当向他人展示或比较不同策略时，一个统一的时间标准（通常是一年）更为直观和公平。年化收益率就是将不同时间长度的总收益，折算成相当于每年平均获得的收益率。

![](img/b4fb986685e38c922228c24b1cc0c6a7_26.png)

通常，在观察回测结果时，我们更关注年化收益率，它是我们重点观察的指标。

![](img/b4fb986685e38c922228c24b1cc0c6a7_28.png)

其计算公式如下：
**年化收益率 = (1 + 总收益率) ^ (250 / N) - 1**

![](img/b4fb986685e38c922228c24b1cc0c6a7_30.png)

公式解析：
*   **R**：代表总收益率，即我们上面计算出的回测收益率。
*   **M**：代表一年的平均交易日数量，通常取250或252天。
*   **N**：代表你的回测数据所覆盖的总交易日数量。

![](img/b4fb986685e38c922228c24b1cc0c6a7_32.png)

例如，如果你的回测数据有730个交易日（大约3年），总收益率为0.5（即50%），那么年化收益率的计算过程如下：

![](img/b4fb986685e38c922228c24b1cc0c6a7_34.png)

![](img/b4fb986685e38c922228c24b1cc0c6a7_36.png)

```python
# 假设总收益率 R = 0.5， 总交易日 N = 730
R = 0.5
N = 730
M = 250 # 一年平均交易日

![](img/b4fb986685e38c922228c24b1cc0c6a7_38.png)

年化收益率 = (1 + R) ** (M / N) - 1
```

![](img/b4fb986685e38c922228c24b1cc0c6a7_40.png)

计算完成后，你会发现年化收益率与总收益率在数值上有所不同。例如，一个总收益率为113%的策略，其年化收益率可能约为20%+；而一个总收益率为负的策略，其年化亏损率也会相应调整。

![](img/b4fb986685e38c922228c24b1cc0c6a7_42.png)

![](img/b4fb986685e38c922228c24b1cc0c6a7_44.png)

这再次说明，在策略评估过程中，你不仅需要看最终的总收益率，还需要关注折算后的年化收益率，以便在不同时间跨度的策略之间进行公平比较。

![](img/b4fb986685e38c922228c24b1cc0c6a7_46.png)

## 总结
本节课中我们一起学习了量化策略收益分析的两个核心指标。
1.  **回测收益率**：反映了策略在整个测试周期内的总盈亏情况，是最直接的收益衡量标准。
2.  **年化收益率**：将总收益标准化为年度收益，便于比较不同时间长度策略的盈利能力，是更常用的评估指标。

![](img/b4fb986685e38c922228c24b1cc0c6a7_50.png)

![](img/b4fb986685e38c922228c24b1cc0c6a7_52.png)

理解并计算这两个指标，是评估和优化你的交易策略的基础。
