# Python金融量化投资分析：P4：03 金融量化分析-股票市场构成 📊

在本节课中，我们将要学习股票市场的构成。了解市场中有哪些参与者以及他们各自扮演的角色，是进行金融量化分析的基础。上一节我们介绍了股票的分类，本节中我们来看看股票市场是如何组织和运作的。

## 市场参与者

股票市场并非只有买卖双方，它是一个由多个角色共同构成的复杂系统。以下是市场中的主要参与者。

### 1. 公司与投资者
公司是资金的需求方，通过发行股票进行融资。投资者是资金的供给方，通过购买股票进行投资。

### 2. 监管机构
公司和投资者不能直接进行交易，以避免暗箱操作。因此，需要监管机构来维持市场秩序。最常见的监管机构是**证监会**。
*   **证监会**：全称为中国证券监督管理委员会，是证券行业的监管机构。公司上市需要向证监会提交各种材料。证监会负责审查公司是否存在欺诈、洗钱等违法行为，并有权决定公司能否上市或将其退市。

### 3. 证券业协会
证券业协会的作用相对较弱，它是一个行业自律组织。例如，一些证券从业资格考试由其主办。

### 4. 交易所
交易所是提供股票集中交易场所的机构，可以理解为“大市场”。中国的两大交易所是上海证券交易所和深圳证券交易所。在电子化交易普及之前，投资者需要到交易所现场排队交易。现在，交易通过网络连接到交易所的系统进行。交易所的核心功能是处理所有买卖股票的请求。

### 5. 证券中介机构
证券中介机构通常被称为**券商**。个人投资者不能直接去交易所买卖股票，必须通过券商开户进行交易。
*   **券商的作用**：券商在交易所拥有交易席位，但席位成本高昂。为了赚回成本，券商开发了交易软件（如同花顺等）。投资者通过券商的软件提交交易指令，券商再通过其在交易所的席位，将指令传达给交易所，从而完成股票买卖。

## 中国交易所与板块

中国有两个主要的证券交易所，每个交易所下又分为不同的板块，以适应不同规模和发展阶段的企业。

### 上海证券交易所
上海证券交易所主要包含一个板块：
*   **主板**：上市要求最为严格，通常面向大型成熟企业。

### 深圳证券交易所
深圳证券交易所包含三个板块：
*   **主板**：与上海主板类似，面向大型成熟企业。
*   **中小板**：为中小企业提供融资渠道，但近年来上市数量减少。
*   **创业板**：为成长性高、但规模可能未达到主板要求的企业（如创业公司）设立。上市门槛相对较低，例如要求连续几年达到一定净利润即可。

## 大盘指数

对于每一个板块，我们常用一个综合性的指标来反映其整体表现，这就是**大盘指数**。

指数反映了该板块内所有股票的综合表现。例如，上海主板的大盘指数叫做**上证指数（沪指）**。它通过计算板块内众多股票价格的平均走势，来描绘整个市场的“晴雨表”。如果大部分权重股票的市值在上涨，指数就会上升，表明市场整体向好；反之则表明市场走弱。

以下是各板块对应的主要指数：
*   上海主板：**上证指数（沪指）**
*   深圳主板：**深证成指（深成指）**
*   深圳中小板：**中小板指**
*   深圳创业板：**创业板指**
