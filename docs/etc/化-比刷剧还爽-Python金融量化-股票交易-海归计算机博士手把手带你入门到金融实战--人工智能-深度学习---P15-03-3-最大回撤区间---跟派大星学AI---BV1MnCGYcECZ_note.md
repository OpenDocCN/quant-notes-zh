## 什么是最大回撤？
上一节我们介绍了策略分析中的各种指标，本节中我们来看看**最大回撤**。它描述的是在投资过程中，资产净值从最高点到之后最低点的**最大跌幅**。简单来说，它衡量的是你在投资过程中可能经历的最糟糕、最“惨”的阶段。

其核心公式如下：

**最大回撤 = max( (Pi - Pj) / Pi )**

其中：
*   **Pi** 代表某个时间点 i 的资产净值（或股价）。
*   **Pj** 代表在时间点 i **之后**的某个时间点 j 的资产净值。
*   我们需要遍历所有可能的 i 和 j (j > i)，找出比值最大的那个。

这个指标不直接评价策略的盈利能力，而是评估其**风险**。即使一个策略最终盈利，但如果其最大回撤很大，意味着投资者在过程中可能承受巨大的心理压力和资金波动风险。

## 如何计算最大回撤？
理解了概念后，我们来看看如何用Python进行计算。计算最大回撤的核心是找到净值曲线上的“高点到后续低点”的最大落差。

![](img/99cff1155602dacba839155c84c2835a_9.png)

以下是计算最大回撤的关键步骤：

![](img/99cff1155602dacba839155c84c2835a_11.png)

![](img/99cff1155602dacba839155c84c2835a_13.png)

1.  **计算累积最大值序列**：对于净值序列，计算从起始到每一天的**历史最高值**。这可以通过Pandas的`.cummax()`方法实现。
2.  **计算每日回撤**：用每天的**历史最高值**减去**当天的实际净值**，得到每天的潜在回撤值。
3.  **计算每日回撤率**：将每日回撤值除以对应的**历史最高值**，得到回撤率。
4.  **找出最大值**：在所有每日回撤率中，找出**最大值**，即为最大回撤率。

![](img/99cff1155602dacba839155c84c2835a_15.png)

让我们通过代码示例来演示这个过程：

![](img/99cff1155602dacba839155c84c2835a_17.png)

![](img/99cff1155602dacba839155c84c2835a_19.png)

```python
import pandas as pd

![](img/99cff1155602dacba839155c84c2835a_23.png)

![](img/99cff1155602dacba839155c84c2835a_25.png)

![](img/99cff1155602dacba839155c84c2835a_27.png)

# 假设‘net_value’是包含每日资产净值的Pandas Series
net_value = pd.Series([100, 105, 102, 110, 108, 115, 112, 109, 118, 115])

![](img/99cff1155602dacba839155c84c2835a_29.png)

![](img/99cff1155602dacba839155c84c2835a_31.png)

# 1. 计算历史最高净值序列
running_max = net_value.cummax()
print(“历史最高净值序列：”)
print(running_max)

![](img/99cff1155602dacba839155c84c2835a_33.png)

# 2. 计算每日回撤（历史最高 - 当日净值）
drawdown = running_max - net_value
print(“\n每日回撤值：”)
print(drawdown)

![](img/99cff1155602dacba839155c84c2835a_35.png)

![](img/99cff1155602dacba839155c84c2835a_37.png)

# 3. 计算每日回撤率
drawdown_pct = drawdown / running_max
print(“\n每日回撤率：”)
print(drawdown_pct)

![](img/99cff1155602dacba839155c84c2835a_39.png)

# 4. 找出最大回撤率
max_drawdown = drawdown_pct.max()
print(f”\n最大回撤率为：{max_drawdown:.2%}”)
```

## 在策略分析中的应用
在量化回测框架中（如Backtrader、Zipline等），最大回撤通常会被自动计算并作为关键风险指标展示在分析报告中。它帮助投资者回答一个问题：“在我的策略生效期间，我最坏的情况下会亏多少？”

![](img/99cff1155602dacba839155c84c2835a_43.png)

![](img/99cff1155602dacba839155c84c2835a_45.png)

一个优秀的策略需要在**收益**和**最大回撤**之间取得平衡。过大的回撤可能导致投资者在净值恢复前因恐慌而提前退出。
