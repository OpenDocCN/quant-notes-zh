## 策略评估指标简介
当我们完成一个交易策略后，必须在历史数据中进行测试，以判断策略的盈亏情况。但“赚了”或“赔了”是模糊的描述。我们需要更精确的指标来回答以下问题：
*   相比于初始本金，策略具体赚了多少？
*   与大盘相比，策略有何优势？
*   策略承担了哪些风险？这些风险是否值得？
*   与其他策略相比，本策略的优势在哪里？

这些问题的答案，就是策略评估指标要提供的信息。

### 回测收益率的定义与计算
我们先讲解最重要、最基本的指标——回测收益率。

![](img/b33cf51cbf33f374f615ae32bd69038c_21.png)

回测收益率衡量的是策略在回测周期内的总收益。例如，你基于2015年至2020年的某只股票数据设计了一个策略，规定了买入和卖出时机。回测收益率就是执行这个策略后，最终获得的**收益相对于初始本金**的比率。它不是一个具体的金额，而是一个百分比。例如，结果显示4.XX%，你就知道折腾多年后，你的本金大概增值了多少。

![](img/b33cf51cbf33f374f615ae32bd69038c_23.png)

![](img/b33cf51cbf33f374f615ae32bd69038c_23.png)

其计算公式如下：

**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

![](img/b33cf51cbf33f374f615ae32bd69038c_25.png)

其中，`P_T` 代表期末的资产价格（或总值），`P_0` 代表期初的资产价格（或总值）。分子 `(P_T - P_0)` 代表赚取的绝对利润，分母 `P_0` 代表初始投入。这个比率就是回测收益率。

![](img/b33cf51cbf33f374f615ae32bd69038c_27.png)

![](img/b33cf51cbf33f374f615ae32bd69038c_25.png)

这是策略评估中最常用的指标，直观地反映了策略的总体表现。

![](img/b33cf51cbf33f374f615ae32bd69038c_27.png)

### 代码实现与可视化
下面我们通过代码来计算回测收益率。我们不仅可以计算最终的总收益率，还可以观察收益率随时间的变化情况。

例如，我们可以绘制从2015年到2018年，策略净值（相对初始值）的变化曲线。这相当于计算每个时间点的 `P_k / P_0`。

具体做法如下：我们构建一个DataFrame，其中每个值都是当前时间点的数据与初始数据的比值。

```python
# 计算每个时间点相对于期初的净值
cumulative_return = data / data.iloc[0]
# 绘制净值曲线图
cumulative_return.plot(figsize=(12, 6))
plt.show()
```

![](img/b33cf51cbf33f374f615ae32bd69038c_29.png)

上图展示了一开始净值为1的情况下，通过策略运作，净值随时间的变化情况。

![](img/b33cf51cbf33f374f615ae32bd69038c_31.png)

但这还不是最终的回测收益率。根据公式，我们需要计算**最终**的净值比，然后减去1。

![](img/b33cf51cbf33f374f615ae32bd69038c_29.png)

```python
# 计算最终的回测收益率
final_return = cumulative_return.iloc[-1] - 1
# 将结果整理成表格展示
result_df = pd.DataFrame(final_return.values, columns=['回测收益率'], index=[‘最终结果’])
print(result_df)
```

## 总结
本节课中，我们一起学习了量化策略评估的核心——回测收益率。
*   **定义**：它衡量了策略在整个回测周期内，期末资产相对于期初资产的增长比率。
*   **公式**：**回测收益率 = (P_T / P_0) - 1**。
*   **意义**：这是评估策略盈利能力的首要且最直观的指标，告诉我们策略“总共赚了（或亏了）多少百分比”。
*   **计算**：我们通过Python代码演示了如何从资产净值序列中计算该指标，并绘制了净值变化曲线。
