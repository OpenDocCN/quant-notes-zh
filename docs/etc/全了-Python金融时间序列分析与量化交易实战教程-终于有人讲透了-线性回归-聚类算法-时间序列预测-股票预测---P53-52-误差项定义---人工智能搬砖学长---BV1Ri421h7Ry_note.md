## 概述
在本节课中，我们将学习线性回归模型中一个核心概念——误差项的定义。我们将探讨如何构建回归方程，如何将其转换为矩阵形式以便计算，以及如何定义衡量模型预测准确性的误差项。理解这些概念是后续进行模型训练和优化的基础。

![](img/f3e2a79363a025f77eaa05769ad90e86_4.png)

## 构建回归方程
上一节我们介绍了线性回归的基本思想，本节中我们来看看如何用数学公式来描述这个“最拟合”的平面。

首先，我们写出回归方程的公式。用 `Hθ(X)` 表示我们构建的拟合平面，其公式为：
`Hθ(X) = θ₀ + θ₁X₁ + θ₂X₂`
其中，`X₁` 和 `X₂` 分别代表影响结果的两个特征（例如工资和年龄），`θ₁` 和 `θ₂` 是它们的**权重项**。`θ₀` 被称为**偏置项**。

偏置项的作用是进行微调。无论 `θ₁`、`θ₂` 或特征值如何变化，`θ₀` 的改变会使整个平面向上或向下浮动，从而对模型的最终结果进行小幅修正，使其更精准。需要强调的是，偏置项是**微调**，而非大范围移动。在线性回归中，核心影响因素是权重项 `θ₁` 和 `θ₂`。

![](img/f3e2a79363a025f77eaa05769ad90e86_6.png)

## 转换为矩阵形式
为了便于后续的矩阵运算，我们需要将上述方程转换为矩阵形式。观察公式 `θ₀ + θ₁X₁ + θ₂X₂`，我们发现由于 `θ₀` 的存在，它无法直接写成矩阵乘法。

解决方案是引入一个新的特征 `X₀`。我们构造一个 `X₀`，使其与 `θ₀` 相乘，从而将公式重写为 `θ₀X₀ + θ₁X₁ + θ₂X₂`。这样，整个表达式就可以用矩阵表示了。

![](img/f3e2a79363a025f77eaa05769ad90e86_8.png)

以下是关于 `X₀` 的关键点：
*   `X₀` 是我们为了计算方便而**人为添加**的一列数据。
*   这一列的所有值通常设为 **1**，因为任何数乘以1都等于其本身。
*   在数据预处理中，经常能看到添加一列全为1的数据，目的就是为了实现这种矩阵转换，而非具有其他物理含义。

![](img/f3e2a79363a025f77eaa05769ad90e86_10.png)

![](img/f3e2a79363a025f77eaa05769ad90e86_12.png)

转换后，我们的回归方程 `Hθ(X)` 就成为了已知数据 `X` 与未知参数 `θ` 的矩阵运算结果，`θ` 正是我们需要求解的目标。

![](img/f3e2a79363a025f77eaa05769ad90e86_14.png)

## 定义误差项
现在，我们有了回归方程，可以计算预测值。以一个样本点为例，我们已知其特征 `X₁` 和 `X₂`，代入方程后，可以得到一个对应的**预测值**（用蓝色点表示）。

![](img/f3e2a79363a025f77eaa05769ad90e86_16.png)

同时，我们拥有该样本的**真实值**（用红色点表示），例如历史数据中银行实际批准贷款的金额。预测值与真实值之间的差异，就是我们所说的**误差项**（error）。

对于第 `i` 个样本，其误差可以表示为：
`error⁽ⁱ⁾ = y_true⁽ⁱ⁾ - y_pred⁽ⁱ⁾`
其中，`y_true` 是真实值，`y_pred` 是预测值。每个样本都有自己的误差项。

## 目标函数（损失函数）
机器学习的过程是：给定数据（data）和一个学习目标（目标函数），机器通过调整参数来逼近这个目标。

我们的目标应该基于误差项来定义。显然，我们希望模型的预测尽可能准确，即误差项**越小越好**。最理想的情况是误差为零，这意味着预测值与真实值完全一致。

因此，我们通常构建一个**损失函数**（或目标函数），其核心思想是：
*   当损失函数值为 **0** 时，达到完美状态。
*   损失函数值**越接近于0**，代表模型性能越好。

在机器学习中，绝大多数损失函数都遵循这一设计原则：最小化误差。

## 总结
本节课中我们一起学习了线性回归模型中误差项的定义。我们首先构建了包含权重项和偏置项的回归方程，并通过添加特征 `X₀` 将其转换为矩阵形式以便计算。接着，我们定义了预测值与真实值之间的差异为误差项。最后，我们引出了机器学习的关键——通过定义损失函数（以最小化误差为目标）来指导模型学习。理解误差项是评估和优化模型性能的基础。
