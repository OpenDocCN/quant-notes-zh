在本节课中，我们将学习如何对处理好的因子数据进行打分与排序。这是构建多因子模型的关键步骤，我们将遍历因子数据，根据因子特性（越高越好或越低越好）进行排序，并为其分配相应的分数，为后续的综合评分做准备。

## 遍历因子与排序

上一节我们介绍了如何将因子分为“越高越好”和“越低越好”两类。本节中，我们将分别遍历这两个DataFrame，对每个因子进行排序。

以下是遍历“越高越好”因子并进行排序的步骤：

1.  获取DataFrame的列名列表，以便遍历每一个因子。
2.  对当前因子列的数据进行排序。对于“越高越好”的因子，我们使用降序排列。
3.  使用 `inplace=True` 参数将排序结果直接应用到原始DataFrame中，避免额外的赋值操作。

```python
# 假设 fundamental_up 是存储“越高越好”因子的DataFrame
for factor in fundamental_up.columns.tolist():
    # 按当前因子列降序排序
    fundamental_up.sort_values(by=factor, ascending=False, inplace=True)
```

![](img/1a03eb48b084fe6e1863b805f89e9506_2.png)

## 为排序后的因子打分

排序完成后，我们需要为每个股票在该因子上的表现打分。为了简单直观，我们采用线性打分法：排名第一的股票得分最高，排名最后的股票得分最低。

![](img/1a03eb48b084fe6e1863b805f89e9506_4.png)

![](img/1a03eb48b084fe6e1863b805f89e9506_6.png)

我们将使用NumPy的 `linspace` 函数来生成分数序列。这个函数可以生成一个指定范围内均匀间隔的数字序列。

![](img/1a03eb48b084fe6e1863b805f89e9506_8.png)

![](img/1a03eb48b084fe6e1863b805f89e9506_10.png)

`linspace` 函数的公式为：
`numpy.linspace(start, stop, num=50, endpoint=True, retstep=False, dtype=None, axis=0)`
其中：
*   `start`：序列的起始值。
*   `stop`：序列的结束值。
*   `num`：要生成的样本数。

![](img/1a03eb48b084fe6e1863b805f89e9506_12.png)

例如，`np.linspace(1, 300, 300)` 会生成从1到300的300个等间隔数字。

![](img/1a03eb48b084fe6e1863b805f89e9506_14.png)

![](img/1a03eb48b084fe6e1863b805f89e9506_16.png)

对于“越高越好”的因子，我们希望排名第一（值最大）的股票获得最高分（300分），因此分数序列应为从300到1递减。我们可以通过设置 `start=300`, `stop=1` 来实现。

![](img/1a03eb48b084fe6e1863b805f89e9506_18.png)

![](img/1a03eb48b084fe6e1863b805f89e9506_20.png)

以下是具体的打分步骤：

![](img/1a03eb48b084fe6e1863b805f89e9506_22.png)

1.  导入必要的库：`numpy` 和 `pandas`。
2.  对于“越高越好”的因子，使用 `np.linspace(300, 1, num_of_stocks)` 生成从300到1的分数序列。
3.  对于“越低越好”的因子，使用 `np.linspace(1, 300, num_of_stocks)` 生成从1到300的分数序列（排名第一即值最小的股票得300分）。
4.  将生成的分数序列赋值给排序后的因子列。

![](img/1a03eb48b084fe6e1863b805f89e9506_24.png)

```python
import numpy as np
import pandas as pd

![](img/1a03eb48b084fe6e1863b805f89e9506_26.png)

![](img/1a03eb48b084fe6e1863b805f89e9506_28.png)

# 假设股票数量为300
num_of_stocks = 300

# 对“越高越好”的因子进行遍历和打分
for factor in fundamental_up.columns.tolist():
    fundamental_up.sort_values(by=factor, ascending=False, inplace=True)
    # 生成从300到1的分数并赋值
    fundamental_up[factor] = np.linspace(300, 1, num_of_stocks)

# 对“越低越好”的因子进行遍历和打分
for factor in fundamental_down.columns.tolist():
    fundamental_down.sort_values(by=factor, ascending=True, inplace=True) # 升序排列
    # 生成从1到300的分数并赋值
    fundamental_down[factor] = np.linspace(1, 300, num_of_stocks)
```

## 合并数据与后续步骤

完成所有因子的打分后，我们得到了两个新的DataFrame（`fundamental_up` 和 `fundamental_down`），其中存储的不再是原始因子值，而是每个股票在各个因子上的得分。

为了计算每个股票的综合得分，我们需要将这两个DataFrame合并。这可以通过pandas的合并操作（如 `concat`）来实现，确保每个股票在所有因子上的得分都在一行数据中，以便后续进行加总。

本节课中我们一起学习了因子打分与排序的核心流程：首先遍历因子并排序，然后利用 `np.linspace` 函数根据排序结果生成分数，最后为后续计算综合得分准备好数据。这是将原始因子数据转化为可比较、可加总的分数的重要一步。
