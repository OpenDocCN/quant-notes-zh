```bash
pip install alphalens
```
不过，在本课程后续的实战环节中，我们将在提供的在线量化平台进行操作，该平台已预装好Alphalens，因此无需自行安装。

## 学习方法建议
为了高效掌握Alphalens的使用，建议遵循以下学习路径。

![](img/6102a1fbf629054f4dc72f233f318b69_19.png)

1.  **查阅官方示例**：官方文档中包含了许多精心编写的示例（`examples`），这些是学习核心功能的最佳起点。
2.  **阅读教程文档**：官方提供的教程详细解释了工具包的设计理念和基本用法，有助于深入理解。
3.  **参考API文档**：当需要了解特定函数或模块的详细参数时，应查阅API文档。
4.  **结合课程案例**：本课程将官方文档的核心内容进行了梳理和总结，并准备了相应的实战案例，方便跟随学习。
