## 总体与样本

上一节我们介绍了概率分布，本节中我们来看看如何从数据中获取信息。总体和样本是统计学中最基本的概念。

总体是研究对象的全体，称为 **population**。样本是从总体中抽取的一部分数据或个体，称为 **sample**。

例如，全国所有成年男性的身高构成一个总体。而我们从中抽取的1万名成年男性的身高，则构成一个样本。我们的目标通常是获取总体的特征，即总体参数，例如总体均值 **μ** 和总体方差 **σ²**。总体参数通常用希腊字母表示。

![](img/9c941cdf087fecb3aa5df8b63b774e27_2.png)

然而，直接测量总体参数通常非常困难，因为总体数量往往过于庞大。例如，我们无法测量全国每一位成年男性的身高。因此，我们很难直接获得总体参数。

我们可以通过样本来推断总体，这个过程称为抽样估计或参数估计。参数估计的流程分为三个步骤。

以下是参数估计的三个步骤：

1.  **抽样**：从总体中抽取样本。样本的大小称为样本容量，记为 **n**。例如，抽取1万个人的身高，样本容量就是10000。
2.  **计算样本统计量**：根据样本的观测值计算样本统计量。例如样本均值 **X̄**、样本方差 **S²** 等。样本统计量通常用大写英文字母表示。
3.  **估计**：用计算出的样本统计量作为总体参数的估计值。例如，用样本均值 **X̄** 估计总体均值 **μ**，用样本方差 **S²** 估计总体方差 **σ²**。

样本统计量本质上是样本观测值的函数。接下来，我们看看几个常见的样本统计量。

## 常见样本统计量

以下是几个核心的样本统计量及其公式：

![](img/9c941cdf087fecb3aa5df8b63b774e27_4.png)

*   **样本均值 (Sample Mean)**：所有样本观测值的平均值。
    *   公式：**X̄ = (1/n) * Σ(Xi)**，其中 i 从 1 到 n。
    *   总体均值公式：**μ = (1/N) * Σ(Xi)**，其中 i 从 1 到 N（总体数量）。
*   **样本方差 (Sample Variance)**：衡量样本数据的离散程度。
    *   公式：**S² = [1/(n-1)] * Σ(Xi - X̄)²**
    *   这里除以 **(n-1)** 而非 **n**，是为了使样本方差成为总体方差的无偏估计。
    *   总体方差公式：**σ² = (1/N) * Σ(Xi - μ)²**
*   **样本标准差 (Sample Standard Deviation)**：样本方差的平方根，也是衡量离散程度的指标。
    *   公式：**S = √S²**
    *   总体标准差：**σ = √σ²**

## 标准差与标准误

初学者常常混淆标准差和标准误。标准差衡量的是**数据本身的分散程度**。而标准误衡量的是**样本统计量的抽样误差大小**，即用样本统计量估计总体参数时的准确性。

标准误本质上也是标准差，但它是样本统计量（如样本均值）的标准差。例如，样本均值的标准误公式为：
**SEM = σ / √n**

![](img/9c941cdf087fecb3aa5df8b63b774e27_6.png)

根据中心极限定理，样本均值 **X̄** 服从均值为 **μ**，方差为 **σ²/n** 的正态分布。因此，样本均值 **X̄** 的标准差（即标准误）就是 **σ/√n**。

标准误越小，说明样本统计量的估计越精确。从公式 **SEM = σ / √n** 可以看出，样本容量 **n** 越大，标准误越小。这很好理解：样本越大，对总体的代表性就越好，估计误差自然越小。极端情况下，如果样本就是整个总体（n = N），那么样本均值就等于总体均值，误差为零。

## 三大抽样分布

因为样本是随机抽取的，所以每次计算出的样本统计量也会不同。因此，样本统计量本身也是一个随机变量。样本统计量所服从的分布，就称为**抽样分布**。

在假设检验中，我们会用到三大重要的抽样分布：卡方分布、t分布和F分布。对于初学者，掌握其定义、构造式和图形特征即可。

### 卡方分布 (Chi-square Distribution)

![](img/9c941cdf087fecb3aa5df8b63b774e27_8.png)

如果 **n** 个相互独立的随机变量 **X₁, X₂, ..., Xₙ** 均服从标准正态分布 **N(0,1)**，则随机变量 **Y = X₁² + X₂² + ... + Xₙ²** 服从自由度为 **n** 的卡方分布，记为 **Y ~ χ²(n)**。

![](img/9c941cdf087fecb3aa5df8b63b774e27_10.png)

![](img/9c941cdf087fecb3aa5df8b63b774e27_12.png)

需要记住的要点：
1.  构成元素 **Xᵢ** 服从标准正态分布。
2.  构造式为各标准正态变量平方之和。
3.  图形特征：取值范围为 **(0, +∞)**，呈右偏（正偏）形态。
4.  期望为 **n**，方差为 **2n**。
5.  随着自由度 **n** 增加，分布形态逐渐接近正态分布。
6.  具有可加性：若 **Y₁ ~ χ²(n₁)**， **Y₂ ~ χ²(n₂)**，且相互独立，则 **Y₁ + Y₂ ~ χ²(n₁ + n₂)**。

### t分布 (t-Distribution)

设随机变量 **X ~ N(0,1)**， **Y ~ χ²(n)**，且 **X** 与 **Y** 相互独立，则随机变量 **T = X / √(Y/n)** 服从自由度为 **n** 的 t 分布，记为 **T ~ t(n)**。

需要记住的要点：
1.  **X** 服从标准正态分布，**Y** 服从卡方分布。
2.  构造式为 **T = X / √(Y/n)**。
3.  图形特征：关于 **y** 轴对称，呈“肥尾”形态（尾部比标准正态分布更厚）。
4.  期望为 **0**，方差为 **n/(n-2)** (n>2)。
5.  随着自由度 **n** 增加，t分布逐渐接近标准正态分布。通常当 **n ≥ 30** 时，可近似为标准正态分布。

### F分布 (F-Distribution)

设随机变量 **X ~ χ²(m)**， **Y ~ χ²(n)**，且 **X** 与 **Y** 相互独立，则随机变量 **F = (X/m) / (Y/n)** 服从第一自由度为 **m**、第二自由度为 **n** 的 F 分布，记为 **F ~ F(m, n)**。

![](img/9c941cdf087fecb3aa5df8b63b774e27_14.png)

需要记住的要点：
1.  **X** 和 **Y** 均服从卡方分布。
2.  构造式为 **F = (X/m) / (Y/n)**。
3.  图形特征：取值范围为 **(0, +∞)**，呈右偏形态。
4.  随着自由度 **m** 和 **n** 增加，分布逐渐接近正态分布。
5.  一个重要性质：若 **T ~ t(n)**，则 **T² ~ F(1, n)**。这个性质可以通过t分布和F分布的定义式推导得出。

到目前为止，我们在这门课上学过的所有概率分布可以总结如下：

*   **离散分布**：
    *   伯努利分布 (0-1分布)
    *   二项分布
    *   泊松分布 (二项分布的近似)
*   **连续分布**：
    *   均匀分布
    *   指数分布 (与泊松过程相关)
    *   正态分布
    *   对数正态分布 (其对数服从正态分布)
*   **三大抽样分布**：
    *   卡方分布
    *   t分布
    *   F分布

## 抽样分布定理

抽样分布定理是后续进行假设检验的理论基石。这里介绍三个核心定理，大家需要了解并记住其结论。

### 定理一

设 **X₁, X₂, ..., Xₙ** 是来自正态总体 **N(μ, σ²)** 的样本，**X̄** 和 **S²** 分别是样本均值和样本方差，则有：
**Y = (n-1)S² / σ² ~ χ²(n-1)**
并且 **X̄** 与 **S²** 相互独立。

> **推导思路**：通过对样本观测值进行标准化和配方，可以将其转化为一个自由度为 **n** 的卡方统计量与一个自由度为 **1** 的卡方统计量之差，利用卡方分布的可加性即得证。

![](img/9c941cdf087fecb3aa5df8b63b774e27_19.png)

### 定理二

![](img/9c941cdf087fecb3aa5df8b63b774e27_21.png)

设 **X₁, X₂, ..., Xₙ** 是来自正态总体 **N(μ, σ²)** 的样本，**X̄** 和 **S²** 分别是样本均值和样本方差，则有：
**T = (X̄ - μ) / (S/√n) ~ t(n-1)**

> **推导思路**：利用定理一的结论 **(n-1)S²/σ² ~ χ²(n-1)**，以及 **X̄** 的分布性质 **(X̄ - μ)/(σ/√n) ~ N(0,1)**，直接套用t分布的定义式即可证明。

### 定理三

![](img/9c941cdf087fecb3aa5df8b63b774e27_23.png)

设 **X₁, ..., Xₙ₁** 与 **Y₁, ..., Yₙ₂** 分别是来自两个独立的正态总体 **N(μ₁, σ₁²)** 与 **N(μ₂, σ₂²)** 的样本。**X̄**, **Ȳ**, **S₁²**, **S₂²** 分别为两样本的均值和方差，则有：
**F = (S₁²/σ₁²) / (S₂²/σ₂²) ~ F(n₁-1, n₂-1)**

![](img/9c941cdf087fecb3aa5df8b63b774e27_25.png)

> **推导思路**：对两个总体分别应用定理一，得到两个独立的卡方统计量，再根据F分布的定义式构造即可证明。

对于这三个定理，掌握其结论形式和应用场景至关重要，具体的推导过程了解即可。
