## 第三步：双Y轴折线图

![](img/9137f9c3929602891f95a2b867faf017_53.png)

![](img/9137f9c3929602891f95a2b867faf017_55.png)

为了更清晰地比较两个指标在相同时间点的互动，我们可以将它们绘制在同一张图上，但使用左右两个不同的Y轴。

![](img/9137f9c3929602891f95a2b867faf017_57.png)

以下是具体步骤：

![](img/9137f9c3929602891f95a2b867faf017_59.png)

1.  首先创建一个坐标轴。
2.  在该坐标轴上绘制第一个指标（如 `SPX`），并为其指定颜色和标签。
3.  通过 `twinx()` 方法创建共享同一X轴的第二个Y轴。
4.  在第二个Y轴上绘制第二个指标（如 `VIX`），并指定不同的颜色。
5.  为了更清晰地观察短期关系，我们可以只选取特定时间段（例如2010年至2012年）的数据进行绘图。

![](img/9137f9c3929602891f95a2b867faf017_61.png)

```python
# 选取2010年至2012年的数据
data_subset = data['2010-01-01':'2012-12-31']
# 创建图形和第一个Y轴
fig, ax1 = plt.subplots(figsize=(12, 6))
# 绘制SPX（左Y轴）
color = 'tab:blue'
ax1.set_xlabel('Date')
ax1.set_ylabel('SPX', color=color)
ax1.plot(data_subset.index, data_subset['SPX'], color=color)
ax1.tick_params(axis='y', labelcolor=color)
# 创建第二个Y轴（共享X轴）
ax2 = ax1.twinx()
# 绘制VIX（右Y轴）
color = 'tab:green'
ax2.set_ylabel('VIX', color=color)
ax2.plot(data_subset.index, data_subset['VIX'], color=color)
ax2.tick_params(axis='y', labelcolor=color)
plt.title('SPX vs VIX (2010-2012)')
plt.show()
```

从这张双Y轴图中，我们可以更直观地观察到一种现象：当 `SPX`（蓝线）下降时，`VIX`（绿线）往往上升，这暗示了二者之间存在负相关关系。可视化是一个迭代过程，通常从基本图形开始，然后不断调整参数（如时间范围、图形大小）以使模式更清晰。

## 第四步：散点图与分布分析

除了时间序列折线图，散点图是研究两个变量间关系的强大工具。它能直接展示一个变量相对于另一个变量的分布情况。

![](img/9137f9c3929602891f95a2b867faf017_67.png)

以下是具体步骤：

![](img/9137f9c3929602891f95a2b867faf017_69.png)

1.  我们使用Pandas的 `scatter_matrix` 函数来创建散点图矩阵。
2.  该矩阵的对角线可以显示每个变量自身的分布，我们可以选择绘制直方图（`hist`）或更平滑的核密度估计图（`kde`）。
3.  通过调整透明度（`alpha`）参数，可以避免数据点过度重叠。

![](img/9137f9c3929602891f95a2b867faf017_71.png)

![](img/9137f9c3929602891f95a2b867faf017_73.png)

```python
from pandas.plotting import scatter_matrix
# 计算指标的日收益率（作为示例，此处使用原始值，实际分析中常用收益率）
# 此处仅为展示绘图方法，实际分析应使用更有意义的衍生数据
# 绘制散点图矩阵，对角线显示核密度估计图
scatter_matrix(data, alpha=0.5, figsize=(8, 8), diagonal='kde')
plt.show()
```

![](img/9137f9c3929602891f95a2b867faf017_75.png)

![](img/9137f9c3929602891f95a2b867faf017_77.png)

在生成的矩阵图中：
*   **对角线**：显示了 `SPX` 和 `VIX` 各自的分布情况。`kde` 图比直方图提供了更平滑的分布曲线。
*   **非对角线散点图**：清晰地展示了 `SPX` 和 `VIX` 之间的负相关关系，点云呈现从左上到右下的趋势。

![](img/9137f9c3929602891f95a2b867faf017_79.png)

这种可视化方法比折线图更能凸显变量间的统计关系。选择直方图还是核密度估计图取决于个人偏好，两者都能有效展示数据分布。
