在本节课中，我们将学习如何计算和比较量化交易策略的收益。我们将通过一个具体的例子，分析“双均线策略”相对于“买入并持有”策略的收益效果。核心在于理解如何计算每日收益率，并应用策略信号来调整收益计算。

![](img/25eb9f8eb15052acd137695b585f7506_3.png)

上一节我们介绍了如何生成交易信号，本节中我们来看看如何基于这些信号计算策略的最终收益。

## 计算基础收益率

首先，我们需要计算股票每日的基础收益率。在金融分析中，我们通常使用对数收益率，因为它具有更好的统计特性（如可加性）。计算公式如下：

![](img/25eb9f8eb15052acd137695b585f7506_5.png)

![](img/25eb9f8eb15052acd137695b585f7506_7.png)

`对数收益率 = ln(当日收盘价 / 前一日收盘价)`

![](img/25eb9f8eb15052acd137695b585f7506_9.png)

以下是计算步骤：

![](img/25eb9f8eb15052acd137695b585f7506_11.png)

1.  获取股价数据。
2.  使用Pandas的`shift`函数将股价序列向后移动一位，得到前一日股价。
3.  将当日股价除以前一日股价，并取自然对数。

![](img/25eb9f8eb15052acd137695b585f7506_13.png)

对应的Python代码如下：
```python
# 假设data是一个包含‘Close’列的DataFrame
data[‘returns’] = np.log(data[‘Close’] / data[‘Close’].shift(1))
```

![](img/25eb9f8eb15052acd137695b585f7506_15.png)

## 应用策略计算策略收益率

![](img/25eb9f8eb15052acd137695b585f7506_17.png)

接下来，我们将之前生成的交易信号（`position`列）应用到收益率计算中。我们的策略逻辑是：当`position`为1时，表示跟随市场正常走势；当`position`为-1时，表示进行反向操作（例如做空或卖出）。

![](img/25eb9f8eb15052acd137695b585f7506_19.png)

策略收益率的计算公式为：
`策略收益率 = 交易信号 * 基础收益率`

![](img/25eb9f8eb15052acd137695b585f7506_21.png)

以下是具体操作步骤：

![](img/25eb9f8eb15052acd137695b585f7506_23.png)

![](img/25eb9f8eb15052acd137695b585f7506_25.png)

1.  确保交易信号（`position`列）也进行了`shift(1)`操作，以与收益率在时间上对齐（使用当日的信号决定当日的操作）。
2.  将对齐后的`position`列与`returns`列逐元素相乘。

![](img/25eb9f8eb15052acd137695b585f7506_27.png)

![](img/25eb9f8eb15052acd137695b585f7506_29.png)

对应的Python代码如下：
```python
# 假设data中已有‘position’和‘returns’列
data[‘strategy’] = data[‘position’].shift(1) * data[‘returns’]
```

## 计算并比较累计收益

计算出每日的策略收益率后，我们需要将其累加起来，得到总的对数收益率。然后，通过指数函数将其还原为实际的资金增长倍数（即1元本金最终变为多少元）。

![](img/25eb9f8eb15052acd137695b585f7506_31.png)

![](img/25eb9f8eb15052acd137695b585f7506_33.png)

计算累计总收益的步骤如下：

![](img/25eb9f8eb15052acd137695b585f7506_35.png)

![](img/25eb9f8eb15052acd137695b585f7506_37.png)

1.  对`returns`列和`strategy`列分别求和，得到总的对数收益率。
2.  使用`np.exp`函数将总对数收益率还原为资金倍数。

对应的Python代码如下：
```python
# 计算总对数收益率并还原为资金倍数
total_return = np.exp(data[‘returns’].sum())
total_strategy_return = np.exp(data[‘strategy’].sum())
```
通过比较`total_return`（买入持有收益）和`total_strategy_return`（策略收益），即可评估策略的有效性。例如，若结果为“买入持有：4元，策略收益：5.8元”，则表明该策略获得了超额收益。

![](img/25eb9f8eb15052acd137695b585f7506_39.png)

本节课中我们一起学习了量化策略收益分析的全过程。我们从计算每日对数收益率开始，接着将交易信号应用于收益率计算以得到策略每日收益，最后通过累加和指数还原来比较策略与基准的最终收益效果。这个过程是评估任何交易策略性能的基础。
