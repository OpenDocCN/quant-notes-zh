## 🔬 遗传规划算法简介
上一节我们介绍了人工挖掘因子的方法。本节中，我们来看看如何利用算法自动化地生成和筛选因子。

![](img/a306508a550da94a68d9d6fd8ec14453_11.png)

遗传规划是一种借鉴自然界生物进化思想的算法。它通过模拟遗传、交叉和变异等过程，从一组初始的简单因子（基因）出发，不断组合和演化，最终生成复杂且可能有效的新因子。

![](img/a306508a550da94a68d9d6fd8ec14453_13.png)

![](img/a306508a550da94a68d9d6fd8ec14453_15.png)

其核心优势在于能够高效地探索海量的因子组合空间。假设我们初始有10个因子和20种数学运算符，仅两两组合的第一步就能产生近900种新因子。如果重复此过程多次，因子数量将呈指数级爆炸增长。遗传规划的目的，正是从这近乎无限的组合中，找到那些可能有效的因子。

---

## 🧬 遗传规划的核心流程
遗传规划遵循一个清晰的迭代流程，其核心步骤包括生成、评估和进化。

以下是遗传规划算法的主要步骤：
1.  **随机生成第一代公式**：算法从基础的因子和运算符库中随机组合，生成初始的“公式种群”。
2.  **计算适应度**：评估每个公式（即新因子）的预测能力。在量化中，常用因子在回测区间的平均Rank IC或因子收益率作为适应度指标。
3.  **选择与进化**：选出适应度最高的公式作为“父代”，通过**交叉**（交换两个公式的部分结构）和**变异**（随机改变公式的某一部分）操作，生成新的“子代”公式。
4.  **迭代优化**：重复步骤2和3，经过多代进化后，保留适应度最高的一批公式作为挖掘出的新因子。

在量化领域，我们不仅使用基础的加减乘除、开方、对数等数学运算符，还需要引入时间序列相关的专业函数，以适应金融数据的特性。

以下是一些在量化遗传规划中常用的自定义函数示例：
*   **`ts_mean(df, n)`**: 计算时间序列在过去n个周期的均值。
*   **`ts_delay(df, n)`**: 将时间序列的值向后延迟n个周期。
*   **`ts_correlation(df1, df2, n)`**: 计算两个时间序列在过去n个周期的滚动相关系数。
*   **`ts_rank(df, n)`**: 计算时间序列在过去n个周期内的横截面排名。
*   **`industry_neutralize(df)`**: 对因子值进行行业中性化处理。

通过引入这些函数，算法能够合成出如 `correlation(ts_mean(volume, 5), high, 10)` 这类具有明确金融逻辑的复杂因子，这是纯数学运算符难以实现的。

## 🛠️ 实践：构建遗传规划框架
理论需要结合实践。本节中，我们将探讨如何动手实现一个轻量化的遗传规划框架。

![](img/a306508a550da94a68d9d6fd8ec14453_44.png)

![](img/a306508a550da94a68d9d6fd8ec14453_46.png)

![](img/a306508a550da94a68d9d6fd8ec14453_48.png)

虽然已有现成的库（如`gplearn`），但为了更灵活地适配量化需求（特别是时间序列处理）和优化运行效率，自己实现核心部分是有价值的。关键在于设计好“运算符”函数库和进化逻辑。

以下是一个计算滚动相关系数运算符的代码示例，展示了如何处理时间序列数据：

```python
import numpy as np

def rolling_correlation(x, y, d):
    """
    计算两个序列x和y的滚动相关系数。
    x, y: 输入序列（numpy数组）
    d: 滚动窗口长度
    返回: 滚动相关系数序列
    """
    n = len(x)
    result = np.full(n, np.nan) # 初始化结果数组为NaN

    if d <= 1:
        return result # 窗口太小无法计算

    for i in range(d-1, n):
        start_idx = i - d + 1
        x_window = x[start_idx:i+1]
        y_window = y[start_idx:i+1]

        # 检查窗口内是否有NaN
        mask = ~(np.isnan(x_window) | np.isnan(y_window))
        x_clean = x_window[mask]
        y_clean = y_window[mask]

        if len(x_clean) < 2: # 有效数据点不足
            result[i] = np.nan
            continue

        # 计算相关系数
        corr_matrix = np.corrcoef(x_clean, y_clean)
        result[i] = corr_matrix[0, 1] if corr_matrix.size == 4 else np.nan

    return result

# 标准化是因子处理的重要一步，通常在每个运算符计算后进行
def standardize(series):
    """对序列进行标准化（Z-score）"""
    mean_val = np.nanmean(series)
    std_val = np.nanstd(series)
    if std_val == 0:
        return np.zeros_like(series)
    return (series - mean_val) / std_val
```

实现框架时，需要循环执行以下步骤：
1.  从预定义的运算符库中随机选择并组合成公式树。
2.  评估公式，计算其生成因子的适应度（如Rank IC）。
3.  根据适应度进行选择、交叉和变异，产生新一代公式。
4.  重复迭代，直到达到终止条件（如指定代数或适应度收敛）。

## 💡 策略思考与进阶讨论
搭建好框架只是第一步，在实际应用中还有许多深层问题需要思考。

![](img/a306508a550da94a68d9d6fd8ec14453_50.png)

![](img/a306508a550da94a68d9d6fd8ec14453_51.png)

**1. 过拟合与因子失效**：遗传规划能挖掘出大量历史表现优异的因子，但其中许多可能存在过拟合。如何区分因子是暂时失效还是永久失效，是一个关键挑战。需要建立持续的监控体系，例如跟踪因子的**IC衰减曲线**，并设置科学的因子入库与淘汰机制。

**2. 预测目标的选择**：算法的适应度函数依赖于预测目标。目标的选择（如未来1日、5日、20日收益率）直接决定了策略的交易频率和逻辑。改变预测目标是调整策略特征的一种有效方式。

![](img/a306508a550da94a68d9d6fd8ec14453_53.png)

![](img/a306508a550da94a68d9d6fd8ec14453_55.png)

**3. 与其他方法的结合**：遗传规划可以与机器学习模型结合。例如，使用神经网络学习因子间的非线性关系，或用集成模型提升预测稳定性。此外，将基本面的先验知识（如产业链逻辑）融入因子生成过程，可能产生更具经济学解释性的稳健因子。

![](img/a306508a550da94a68d9d6fd8ec14453_57.png)

![](img/a306508a550da94a68d9d6fd8ec14453_59.png)

![](img/a306508a550da94a68d9d6fd8ec14453_61.png)

![](img/a306508a550da94a68d9d6fd8ec14453_63.png)

![](img/a306508a550da94a68d9d6fd8ec14453_65.png)

**4. 市场环境适应性**：此类因子挖掘策略的表现与市场波动性、流动性密切相关。在波动率低、同质化策略拥挤的市场环境中，因子有效性可能迅速衰减。因此，持续创新和多元化研究（如结合日内高频数据）至关重要。
