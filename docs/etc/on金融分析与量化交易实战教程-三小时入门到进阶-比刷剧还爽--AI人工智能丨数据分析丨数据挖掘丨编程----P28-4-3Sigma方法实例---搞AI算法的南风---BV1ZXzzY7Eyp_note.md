## 3-Sigma方法原理 📈

3-Sigma方法的原理基于正态分布（也称高斯分布）。正态分布是一种常见的概率分布，其形状呈对称的钟形曲线。

为了便于理解，可以将其想象为一种“正常”的分布。例如，在银行贷款场景中，银行批准给你的贷款额度与预期额度略有偏差（如多或少几元）是常见情况，这属于“正常”范围。但如果偏差极大（如多或少十万元），则属于“不正常”的极端情况。

![](img/aeed56e3f7150ebad359205a8e383762_1.png)

在正态分布曲线中，中心点代表**均值**（μ）。数据点离均值越近，出现的概率越高；离均值越远，出现的概率则越低。我们的目标是识别并处理那些远离均值、概率极低的“离群点”（极值）。

**3-Sigma** 中的 “Sigma”（σ） 指的是**标准差**，它衡量数据的离散程度。我们通过计算均值加减若干倍标准差来定义一个数据区间：
*   **均值 ± 1σ**：包含约68.3%的数据。
*   **均值 ± 2σ**：包含约95.4%的数据。
*   **均值 ± 3σ**：包含约99.7%的数据。

3-Sigma方法就是设定数据的正常范围为 **均值 ± 3倍标准差**。我们假设数据经过变换后服从正态分布，那么落在这个区间内的数据（占99.7%）被视为正常数据，而落在区间外的数据（占0.3%）则被视为极值，需要进行规范化处理。

简而言之，我们需要计算数据的**均值**和**标准差**，然后根据以下公式确定上下限：
*   **下限** = 均值 - N * 标准差
*   **上限** = 均值 + N * 标准差
其中，N通常取3。

## 代码实现 💻

以下是3-Sigma方法的代码实现步骤。我们将创建一个函数，输入一列数据和一个倍数N，返回基于该方法的上下限。

![](img/aeed56e3f7150ebad359205a8e383762_5.png)

首先，复制并重命名函数，以明确其功能。

![](img/aeed56e3f7150ebad359205a8e383762_7.png)

```python
def three_sigma(data_series, n=3):
```

![](img/aeed56e3f7150ebad359205a8e383762_9.png)

在函数内部，我们需要计算两个核心统计量：**均值**和**标准差**。

![](img/aeed56e3f7150ebad359205a8e383762_11.png)

```python
    mean = data_series.mean()   # 计算均值
    std = data_series.std()     # 计算标准差
```

![](img/aeed56e3f7150ebad359205a8e383762_13.png)

获得均值和标准差后，即可根据公式计算上限和下限。

![](img/aeed56e3f7150ebad359205a8e383762_15.png)

```python
    upper = mean + n * std      # 计算上限
    lower = mean - n * std      # 计算下限
    return lower, upper         # 返回下限和上限
```

![](img/aeed56e3f7150ebad359205a8e383762_17.png)

![](img/aeed56e3f7150ebad359205a8e383762_19.png)

至此，3-Sigma方法的函数就编写完成了。它的逻辑非常清晰：计算统计量，然后应用公式。

## 核心要点总结 ✅

![](img/aeed56e3f7150ebad359205a8e383762_35.png)

本节课我们一起学习了3-Sigma去极值方法。

![](img/aeed56e3f7150ebad359205a8e383762_37.png)

![](img/aeed56e3f7150ebad359205a8e383762_39.png)

*   **原理核心**：该方法基于数据服从正态分布的假设，通过**均值**和**标准差**来确定数据的正常范围。落在 **均值 ± N倍标准差** 区间外的点被视为离群值。
*   **代码实现**：关键在于计算 `data_series.mean()` 和 `data_series.std()`，并用公式 `mean ± n * std` 计算边界。
*   **方法对比**：与分位数法、MAD法一样，3-Sigma法是数据预处理中“去极值”的一种有效手段，不同方法各有特点，可根据实际情况选用。
*   **重要概念**：需要明确，“去极值”并非简单删除数据，而是通过设定**上界**和**下界**，将超出范围的数据规范到边界之内，从而减少极端值对整体分析的影响。

![](img/aeed56e3f7150ebad359205a8e383762_41.png)

这是构建稳健数据预处理流程的三步走策略中的重要一步。
