# 量化交易实战课程：P60：构建回归方程

## 概述
在本节课中，我们将学习如何利用Python的机器学习库构建一个回归方程模型，用于预测股票每日的涨跌趋势。我们将从导入工具包开始，逐步完成模型的训练、预测，并基于预测结果制定简单的交易策略。

---

## 第一步：导入必要的工具包
为了构建回归模型，我们需要借助一个强大的Python机器学习库。这里我们选择使用`scikit-learn`，它包含了绝大多数常用的机器学习算法。

以下是导入线性回归模型的代码：
```python
from sklearn.linear_model import LinearRegression
```

---

## 第二步：实例化模型
在`scikit-learn`中，使用任何模型的第一步都是对其进行实例化。实例化相当于创建一个模型对象，之后我们才能用它来训练和预测数据。

我们创建一个线性回归模型实例：
```python
model = LinearRegression()
```

---

## 第三步：训练模型
模型实例化后，下一步是使用我们的数据来训练它。训练过程在`scikit-learn`中通过`.fit()`方法完成。

`.fit()`方法需要传入两个关键参数：
*   **X**：特征数据，即用于预测的自变量。
*   **Y**：标签数据，即我们想要预测的目标变量。

我们将进行一个对比实验，使用两个不同的目标变量`Y`来训练模型，以观察哪种预测效果更好。

以下是训练模型的代码示例。注意，我们需要确保特征列名在函数外部可用，因此使用了`global`声明：
```python
# 假设 df 是我们的数据框，feature_columns 是特征列名的列表
# 实验一：使用实际收益率（returns）作为目标变量
model.fit(df[feature_columns], df[‘returns‘])

# 实验二：使用涨跌方向（direction）作为目标变量
model.fit(df[feature_columns], df[‘direction‘])
```

---

## 第四步：进行预测
模型训练完成后，就可以用它来对新的或已有的数据进行预测。预测功能通过`.predict()`方法实现。

我们将对原始数据分别使用两个训练好的模型进行预测，并将预测结果作为新列添加到数据框中：
```python
# 使用第一个模型（以returns为目标）进行预测
df[‘pred_returns‘] = model.predict(df[feature_columns])

# 使用第二个模型（以direction为目标）进行预测
df[‘pred_direction‘] = model.predict(df[feature_columns])
```

执行完成后，查看数据，我们可能会发现预测值与真实值存在差异。这是正常现象，因为模型（尤其是简单的线性回归）很难100%准确地预测金融市场。

---

## 第五步：解读预测结果并制定策略
对于交易策略而言，我们通常更关心涨跌的方向，而非具体的数值。因此，我们需要将连续的预测值转换为代表“买入”或“卖出”的信号。

我们可以通过判断预测值是否大于0来生成交易信号：
```python
# 将预测值转换为交易信号：大于0为1（买入），小于等于0为-1（卖出）
df[‘signal_returns‘] = df[‘pred_returns‘].apply(lambda x: 1 if x > 0 else -1)
df[‘signal_direction‘] = df[‘pred_direction‘].apply(lambda x: 1 if x > 0 else -1)
```

现在，`signal_returns`和`signal_direction`这两列就清晰地指示了每天应该执行的操作。

---

## 第六步：计算策略收益
有了交易信号，我们就可以模拟策略的收益情况。基本思路是：当信号为1（买入）时，获得当天的实际收益率；当信号为-1（卖出）时，获得当天实际收益率的相反数（即做空或避开损失）。

以下是计算策略收益的公式：
**策略日收益 = 交易信号 × 实际日收益率**

用代码实现如下：
```python
# 计算策略一的收益
df[‘strategy_return_1‘] = df[‘signal_returns‘] * df[‘returns‘]
# 计算策略二的收益
df[‘strategy_return_2‘] = df[‘signal_direction‘] * df[‘returns‘]
```

执行代码后，数据框中会新增两列，分别代表两个回归方程策略所获得的日收益率。通过对比这些收益率，我们可以初步评估哪个模型（哪个目标变量）构建的策略更有效。

**注意**：在Jupyter Notebook中反复执行单元格时，新的操作会叠加在原有数据上。如果想清除之前的列并重新开始，最简便的方法是重启内核（Kernel -> Restart & Run All）或从特定单元格开始重新运行整个流程。

## 总结
本节课我们一起学习了构建回归方程进行量化交易策略开发的完整流程：
1.  导入`scikit-learn`库中的线性回归模型。
2.  实例化模型对象。
3.  使用`.fit()`方法，分别以“收益率”和“涨跌方向”为目标变量训练模型。
4.  使用`.predict()`方法对数据进行预测。
5.  将连续的预测值转换为清晰的买卖信号（1或-1）。
6.  根据买卖信号计算策略的模拟收益。

![](img/dc1acdf73917e08cf4c15f2d4d9f6fff_5.png)

这个过程展示了如何将机器学习模型应用于金融时间序列数据，并基于模型输出制定可执行的交易策略。虽然本例使用的是基础线性回归，但整个框架可以扩展到更复杂的模型。关键在于通过实验对比，找到更适合预测目标变量的模型。
