在本节课中，我们将要学习量化策略回测中两个至关重要的收益指标：**回测收益率**与**年化收益率**。理解这两个指标是评估策略有效性的基础。

![](img/3087def75ea83bcf23a4ef8aa4048220_2.png)

上一节我们介绍了如何计算策略的最终盈亏，即回测收益率。本节中我们来看看如何将总收益转化为更标准、更易于比较的**年化收益率**。

![](img/3087def75ea83bcf23a4ef8aa4048220_4.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_6.png)

## 回测收益率回顾

![](img/3087def75ea83bcf23a4ef8aa4048220_8.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_10.png)

回测收益率衡量的是在整个回测周期结束后，策略的总盈亏情况。其计算公式为：
`总收益率 = (最终资产 - 初始资产) / 初始资产`

![](img/3087def75ea83bcf23a4ef8aa4048220_12.png)

这个指标直观地告诉你策略最终是赚是赔，以及赚赔的幅度。例如，一个策略的回测收益率为0.5，意味着你的资产增长了50%；若为-0.2，则意味着亏损了20%。

![](img/3087def75ea83bcf23a4ef8aa4048220_14.png)

## 年化收益率详解

![](img/3087def75ea83bcf23a4ef8aa4048220_16.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_18.png)

然而，仅看总收益率是不够的。如果一个策略用了10年获得50%的收益，而另一个策略只用1年就获得40%的收益，显然后者的资金使用效率更高。为了公平地比较不同时间长度策略的盈利能力，我们引入**年化收益率**。

年化收益率是将总收益率折算为以“年”为单位的平均收益率，使得不同周期的策略收益具有可比性。其核心公式如下：

`年化收益率 = (1 + 总收益率)^(年交易日数 / 回测总交易日数) - 1`

![](img/3087def75ea83bcf23a4ef8aa4048220_20.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_22.png)

以下是公式中各个参数的含义：
*   **总收益率**：即上文计算出的回测总收益率。
*   **年交易日数**：通常取250或252天，代表一年中平均的交易日数量。
*   **回测总交易日数**：你的策略回测所覆盖的总交易天数。

![](img/3087def75ea83bcf23a4ef8aa4048220_24.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_26.png)

### 计算示例

假设一个策略的回测总收益率为 `R = 0.5`（即50%），回测总交易日数 `N = 730`天（约3年），年交易日数 `M = 250`天。

![](img/3087def75ea83bcf23a4ef8aa4048220_28.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_30.png)

那么，该策略的年化收益率计算过程为：
`年化收益率 = (1 + 0.5)^(250 / 730) - 1 ≈ 0.156`

![](img/3087def75ea83bcf23a4ef8aa4048220_32.png)

这意味着，该策略折算成年均收益率约为15.6%。

![](img/3087def75ea83bcf23a4ef8aa4048220_34.png)

在实际的Python分析中，我们可以这样计算：

![](img/3087def75ea83bcf23a4ef8aa4048220_36.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_38.png)

```python
# 假设 total_return 是计算好的总收益率
total_return = 0.5
# 回测总天数
total_days = 730
# 年交易日数
trading_days_per_year = 250

![](img/3087def75ea83bcf23a4ef8aa4048220_40.png)

# 计算年化收益率
annualized_return = (1 + total_return) ** (trading_days_per_year / total_days) - 1
print(f"年化收益率为: {annualized_return:.2%}")
```

![](img/3087def75ea83bcf23a4ef8aa4048220_42.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_44.png)

执行代码后，你会得到与总收益率不同的年化数值。通常，对于盈利的策略，如果回测周期超过一年，年化收益率会低于总收益率；反之，对于亏损的策略，年化亏损率也会被“摊薄”。

![](img/3087def75ea83bcf23a4ef8aa4048220_46.png)

## 核心指标总结

![](img/3087def75ea83bcf23a4ef8aa4048220_48.png)

本节课中我们一起学习了评估量化策略收益的两个核心指标：
1.  **回测收益率**：反映策略在整个测试周期内的**最终总盈亏**，是评估策略成败的绝对指标。
2.  **年化收益率**：将总收益标准化为年均收益，是**比较不同周期策略盈利能力**和衡量资金使用效率的关键相对指标。

![](img/3087def75ea83bcf23a4ef8aa4048220_50.png)

![](img/3087def75ea83bcf23a4ef8aa4048220_52.png)

在分析策略时，必须同时关注这两个指标。一个优秀的策略，通常需要在可接受的风险下，展现出具有吸引力的年化收益率。
