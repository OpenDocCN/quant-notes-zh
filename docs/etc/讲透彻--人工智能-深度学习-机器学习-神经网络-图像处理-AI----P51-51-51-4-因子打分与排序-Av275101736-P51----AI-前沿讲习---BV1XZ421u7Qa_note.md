在本节课中，我们将学习如何对选出的因子进行排序和打分。这是构建多因子模型的关键步骤，我们将分别处理“越高越好”和“越低越好”两类因子，并为它们分配合理的分数。

![](img/8e0353555cf30605e648bae8165e7197_4.png)

上一节我们介绍了如何筛选和分类因子，本节中我们来看看如何为这些因子进行排序和打分。

## 遍历因子并排序

首先，我们需要遍历之前创建的两个DataFrame（分别代表“越高越好”和“越低越好”的因子），并对每个因子进行排序。

以下是遍历“越高越好”因子并排序的步骤：

```python
import numpy as np
import pandas as pd

# 假设我们已经有了两个DataFrame: df_up (越高越好) 和 df_down (越低越好)
# 遍历‘越高越好’的DataFrame
for factor in df_up.columns:
    # 对当前因子列进行排序，从低到高
    df_up.sort_values(by=factor, inplace=True)
```

![](img/8e0353555cf30605e648bae8165e7197_6.png)

代码解释：
*   `df_up.columns` 获取DataFrame的所有列名（即因子名）。
*   `df_up.sort_values(by=factor, inplace=True)` 按照名为 `factor` 的列对 `df_up` 进行排序。`inplace=True` 参数表示直接在原DataFrame上修改，无需重新赋值。

![](img/8e0353555cf30605e648bae8165e7197_8.png)

## 为排序后的因子打分

![](img/8e0353555cf30605e648bae8165e7197_10.png)

排序完成后，我们需要为每个股票在该因子上的表现打分。一个简单的方法是：排名第一的得最高分，排名最后的得最低分。我们将使用NumPy的 `linspace` 函数来生成分数序列。

![](img/8e0353555cf30605e648bae8165e7197_12.png)

![](img/8e0353555cf30605e648bae8165e7197_14.png)

`numpy.linspace` 函数用于创建等差数列，其基本公式为：
`np.linspace(start, stop, num)`
*   `start`：序列的起始值。
*   `stop`：序列的结束值。
*   `num`：要生成的样本数量。

![](img/8e0353555cf30605e648bae8165e7197_16.png)

![](img/8e0353555cf30605e648bae8165e7197_18.png)

以下是使用 `linspace` 进行打分的示例：

![](img/8e0353555cf30605e648bae8165e7197_20.png)

```python
# 假设有300支股票
num_stocks = 300

![](img/8e0353555cf30605e648bae8165e7197_22.png)

![](img/8e0353555cf30605e648bae8165e7197_24.png)

# 为‘越高越好’的因子打分：因子值越大，分数越高（从1到300）
scores_up = np.linspace(1, num_stocks, num_stocks)

![](img/8e0353555cf30605e648bae8165e7197_26.png)

![](img/8e0353555cf30605e648bae8165e7197_28.png)

# 为‘越低越好’的因子打分：因子值越小，分数越高（从300到1）
scores_down = np.linspace(num_stocks, 1, num_stocks)
```

![](img/8e0353555cf30605e648bae8165e7197_30.png)

现在，我们将打分逻辑整合到遍历循环中：

![](img/8e0353555cf30605e648bae8165e7197_32.png)

![](img/8e0353555cf30605e648bae8165e7197_34.png)

```python
# 为‘越高越好’的因子打分
for factor in df_up.columns:
    df_up.sort_values(by=factor, inplace=True)
    # 生成从1到300的分数，并赋值给当前因子列
    df_up[factor] = np.linspace(1, len(df_up), len(df_up))

# 为‘越低越好’的因子打分
for factor in df_down.columns:
    df_down.sort_values(by=factor, inplace=True)
    # 生成从300到1的分数，并赋值给当前因子列
    df_down[factor] = np.linspace(len(df_down), 1, len(df_down))
```

## 合并打分结果

![](img/8e0353555cf30605e648bae8165e7197_36.png)

完成对所有因子的打分后，我们得到了两个新的DataFrame（`df_up` 和 `df_down`），它们现在存储的是分数而非原始因子值。为了计算每支股票的综合得分，我们需要将这两个DataFrame合并。

![](img/8e0353555cf30605e648bae8165e7197_38.png)

以下是合并DataFrame的步骤：

```python
# 将两个打分后的DataFrame按行索引对齐相加，得到综合得分
total_score_df = df_up + df_down
```

*   此操作要求 `df_up` 和 `df_down` 具有相同的行索引（股票）和列结构（因子）。相加后，`total_score_df` 中每个单元格的值就是该股票在所有因子上的得分总和。

![](img/8e0353555cf30605e648bae8165e7197_40.png)

![](img/8e0353555cf30605e648bae8165e7197_42.png)

本节课中我们一起学习了因子打分与排序的完整流程。我们首先遍历两类因子并分别进行排序，然后利用 `np.linspace` 函数根据排序结果为每支股票分配分数，最后将两类因子的得分相加得到综合得分。这是构建多因子选股模型的基础，下一步我们将基于这个综合得分进行股票筛选。
