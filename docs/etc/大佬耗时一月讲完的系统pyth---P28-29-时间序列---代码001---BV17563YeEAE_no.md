## 概述

之前我们介绍了Pandas中关于时间对象的多种函数。生成了这些时间对象后，我们可以用它们来构成一个时间序列。所谓时间序列，就是以时间对象作为索引的Series或DataFrame。

## 创建时间序列

我们之前看到，`pandas.date_range`函数生成的索引类型是`DatetimeIndex`。这意味着我们可以用它来作为DataFrame或Series的索引。

例如，创建一个Series对象：

![](img/f9ef2a411a808275e25a9fe0e720ee14_3.png)

![](img/f9ef2a411a808275e25a9fe0e720ee14_5.png)

```python
import pandas as pd
import numpy as np

![](img/f9ef2a411a808275e25a9fe0e720ee14_7.png)

# 创建一个长度为100的数值数组作为数据
data = np.arange(100)

![](img/f9ef2a411a808275e25a9fe0e720ee14_9.png)

# 创建一个从2017-01-01开始，长度为100天的DatetimeIndex作为索引
index = pd.date_range('2017-01-01', periods=100)

![](img/f9ef2a411a808275e25a9fe0e720ee14_11.png)

# 创建时间序列Series
sr = pd.Series(data, index=index)
```

现在，我们创建了一个较长的Series对象。你可以看到它的索引值看起来像字符串，但实际上它是一个时间对象。通过检查`sr.index`，我们可以确认它是`DatetimeIndex`类型。

## 时间序列的切片操作

成为一个时间序列后，它带来了一些直观的好处。例如，我们可以方便地选取特定时间段的数据。

以下是几种切片方式：

*   **按年月切片**：选取2017年3月的所有数据。
    ```python
    sr['2017-03']
    ```
*   **按年切片**：选取2017年的所有数据。
    ```python
    sr['2017']
    ```
*   **按日期范围切片**：选取从2017年12月25日到2018年2月1日的数据。
    ```python
    sr['2017-12-25':'2018-02-01']
    ```

也就是说，你可以使用不完整的日期（如年、年月）或一个完整的日期范围进行切片，即使传入的是字符串格式也没有问题。

## 时间序列的重采样

上一节我们介绍了如何对时间序列进行切片，本节中我们来看看另一个强大的功能：重采样。

`resample`函数用于重新采样或重新取样。例如，我们有一系列日度数据，但希望查看每周或每月的汇总情况。

以下是重采样的示例：

*   **按周求和**：将数据按周分组，并计算每周的和。
    ```python
    sr.resample('W').sum()
    ```
    输出结果会显示每周的起始日期（如1月1日、1月8日）及该周数据的汇总值。
*   **按月求平均值**：将数据按月分组，并计算每月的平均值。
    ```python
    sr.resample('M').mean()
    ```

## 其他函数：truncate

除了切片和重采样，时间序列还有其他函数支持，例如`truncate`。

![](img/f9ef2a411a808275e25a9fe0e720ee14_15.png)

`truncate`函数可以根据日期截断数据。它有两个参数：`before`和`after`。
*   `before`：保留指定日期之后的数据。
*   `after`：保留指定日期之前的数据。

示例：
```python
# 保留2018年2月3日之后的数据
sr.truncate(before='2018-02-03')

# 保留2018年2月3日之前的数据
sr.truncate(after='2018-02-03')
```

不过，这个函数的意义不是很大，因为我们可以用更灵活的切片操作达到相同目的。例如，同时使用`before`和`after`参数可以保留中间段的数据，但这也可以通过两次切片来实现。
