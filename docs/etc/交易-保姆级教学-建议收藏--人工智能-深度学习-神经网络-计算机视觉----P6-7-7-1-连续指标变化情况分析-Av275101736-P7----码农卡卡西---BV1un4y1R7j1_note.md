## 简单增长率累加的问题

上一节我们介绍了如何计算增长率。本节中我们来看看，能否将一段时间内的各个增长率累加起来，以反映整体的增长情况。

例如，假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_4.png)

以下是计算过程：
1.  第一到第二天的增长率：`(50 - 100) / 100 = -0.5`
2.  第二到第三天的增长率：`(75 - 50) / 50 = 0.5`

如果简单地将增长率累加：`-0.5 + 0.5 = 0`。这个结果意味着总增长率为0，即股票价格没有变化。

但观察原始数据，股价从100元变为75元，实际上是下跌的。这说明**简单的增长率累加无法正确反映时间序列的整体变化**。

## 引入对数变换

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_6.png)

对于连续时间的序列，如果想进行有意义的累加操作，我们需要借助数学中的对数公式。

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_8.png)

核心思想是：将对数形式的日收益率进行累加，等价于计算首日与末日价格的整体比率。

**公式推导如下：**

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_10.png)

假设连续三天的价格为 `P1`, `P2`, `P3`。
我们计算每日的对数收益率：
`log(P2/P1) + log(P3/P2)`

根据对数运算法则 `log(a) + log(b) = log(a*b)`，上式可合并为：
`log( (P2/P1) * (P3/P2) ) = log(P3/P1)`

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_12.png)

这个结果直接反映了**第三天价格相对于第一天价格的整体变化**。推广开来，将一段时间内所有的对数收益率累加，就得到了这段时间的整体对数收益率，它等于末期价格与初期价格之比的对数。

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_14.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_16.png)

形象地说，这解决了“第一天投入1元钱，经过一系列涨跌后，在最后一天相当于多少钱”的问题。

## 在Python中实现连续变化分析

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_24.png)

我们将使用Pandas和NumPy库来实现上述计算。

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_26.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_28.png)

### 步骤一：计算对数收益率

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_30.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_32.png)

首先，我们需要计算每一天相对于前一天的比率，然后取自然对数。

```python
# 假设 `data` 是一个包含股价的Pandas Series或DataFrame列
# 使用 shift(1) 将数据向上移动一格，得到前一天的数据
shifted_data = data.shift(1)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_34.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_36.png)

# 计算每日的对数收益率：log(当天价格 / 前一天价格)
log_returns = np.log(data / shifted_data)
```

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_38.png)

### 步骤二：计算累积对数收益率

接下来，我们对这些每日的对数收益率进行累加。这可以使用 `np.cumsum()` 函数轻松完成。

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_40.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_42.png)

```python
# 计算累积对数收益率
cumulative_log_returns = log_returns.cumsum()
```

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_44.png)

`np.cumsum()` 函数的作用是返回累积和。例如，输入 `[1, 2, 3, 4]`，它会返回 `[1, 3, 6, 10]`。

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_46.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_48.png)

### 步骤三：还原为实际价值倍数

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_50.png)

由于我们之前取了对数，现在需要将其还原，以得到“初始1单位投资在每一天对应的实际价值”。

**公式：** `实际价值倍数 = exp(累积对数收益率)`

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_52.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_54.png)

```python
# 将对数收益率还原为实际的价值倍数
# 即：初始的1元钱，在每一天相当于多少钱
value_over_time = np.exp(cumulative_log_returns)
```

### 步骤四：可视化结果

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_56.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_58.png)

最后，我们可以将 `value_over_time` 绘制成图表，直观地展示初始投资随时间增长（或减少）的情况。

```python
# 绘制价值变化曲线
value_over_time.plot(figsize=(16, 8))
plt.title('初始投资价值随时间变化')
plt.ylabel('价值倍数 (初始为1)')
plt.xlabel('时间')
plt.show()
```

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_60.png)

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_62.png)

图表会显示，例如，亚马逊(AMZN)的股票在2010年投入1元，到2018年可能价值约10元；而苹果(AAPL)的1元投资可能增长到约4元。

![](img/53c6aecc44fb6cc5c22d3b7e0026157a_70.png)

1.  **差异值与简单增长率**：我们回顾了如何计算价格差异(`.diff()`)和简单增长率(`(当期-前期)/前期`)，但指出简单增长率不能直接累加。
2.  **对数收益率**：为了解决累加问题，我们引入了对数收益率 **`log(P_t / P_{t-1})`**。它的累加具有实际意义，代表了整个时间段的总收益。
3.  **累积收益与价值还原**：通过对数收益率的累积和(`np.cumsum()`)，再通过指数函数(`np.exp()`)还原，我们得到了“初始1单位投资在任意时刻的价值”这一核心指标。
4.  **实际应用**：这个方法清晰地展示了长期投资的复利效应，是量化分析中评估资产长期表现的重要工具。
