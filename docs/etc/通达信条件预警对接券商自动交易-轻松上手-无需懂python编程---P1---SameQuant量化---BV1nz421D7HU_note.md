# 通达信条件预警对接券商自动交易：P1：核心原理与快速上手 🚀

在本节课中，我们将学习如何将通达信软件的条件预警功能，与券商的QMT策略终端进行对接，从而实现无需Python编程的自动化交易。整个过程仅需会使用通达信的条件选股功能即可。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_1.png)

上一节我们介绍了课程目标，本节中我们来看看实现这一功能需要做的准备工作。

## 准备工作 📋

![](img/97fc3e3e2a5de88de4334130d5fab8e1_3.png)

以下是实现对接所需的两款核心软件：
1.  **通达信软件**：用于设置条件预警和股票池。
2.  **券商QMT策略终端**：用于接收预警信号并执行自动化交易。

在开始之前，我们需要在通达信软件中预先设置好三个自定义板块。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_5.png)

以下是需要在通达信“自定义板块”中创建的三个板块：
*   **买入股票池**：用于存放触发买入预警的股票。
*   **卖出股票池**：用于存放触发卖出预警的股票。
*   **盘后持仓池**：用于存放您计划监控卖出的持仓股票。

准备工作完成后，接下来我们启动QMT策略程序。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_7.png)

## 启动与监控 🖥️

在QMT策略终端中运行相应的监控程序。程序启动后，将自动执行以下操作：
*   监控证券账户的持仓和资金情况。
*   监控通达信软件中上述三个自定义板块的股票成分变化。

此时，如果板块中没有股票，QMT界面将显示为空。接下来，我们开启通达信的条件预警功能。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_9.png)

## 买入流程详解 ➕

在通达信中设置好买入条件预警，并将其输出目标指向 **“买入股票池”** 板块。当预警条件被触发时，符合条件的个股会自动被添加到该板块中。

与此同时，QMT策略程序会实时监控“买入股票池”板块。一旦发现有新的股票加入，程序会立即读取该信号，并自动执行买入操作。程序具备去重逻辑，对于已经买入的股票，即使再次触发买入信号也不会重复下单。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_11.png)

理解了买入流程后，我们再来看看对应的卖出流程是如何运作的。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_13.png)

## 卖出流程详解 ➖

卖出流程的监控逻辑与买入类似，但监控的源头不同。首先，在通达信中设置卖出条件预警时，其监控的股票池应设置为 **“盘后持仓池”**（即您当前持仓的股票列表）。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_15.png)

当持仓股触发卖出预警条件时，该股票会被自动添加到 **“卖出股票池”** 板块中。

QMT策略程序在监控到“卖出股票池”有新增股票时，便会立即触发对该股票的卖出操作。至此，一个完整的“买入-持有-卖出”自动化交易闭环就形成了。

除了基本的买卖触发，该系统还支持更多高级功能，让策略更贴合实际交易需求。

## 高级功能与价格设置 ⚙️

该系统支持丰富的订单价格类型，使交易策略更灵活。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_17.png)

以下是目前支持的订单价格类型（共11种）：
*   `最新价`
*   `买一价` 到 `买五价`
*   `卖一价` 到 `卖五价`

无论是买入还是卖出订单，您都可以根据策略选择以上任意一种价格进行委托。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_19.png)

此外，系统还支持实用的盘中监控功能。例如，对于持仓中已涨停的个股，您可以设置一个“封单金额”阈值（如3000万元）。

当该涨停股的封单金额低于您设定的阈值时，QMT程序将自动触发卖出操作，帮助您及时锁定利润或规避炸板风险。卖出价格同样可以在上述支持的价格类型中选择。

![](img/97fc3e3e2a5de88de4334130d5fab8e1_21.png)
