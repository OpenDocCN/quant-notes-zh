## 因子收益率与IC值的区别

上一节我们介绍了IC值，它衡量的是因子与收益率之间的相关性。本节中我们来看看因子收益率，它与IC值有本质区别。

IC值计算的是因子与收益率之间的相关性。现在我们要计算的是因子收益率本身，它是一个具体的数值，而非描述两者关系的系数。

## 因子收益率的定义与公式

![](img/127d73397a99e19c71e910e49ea378af_7.png)

因子收益率描述的是因子对收益率的直接贡献权重。它通过一个线性回归模型来体现。

其核心公式如下：

![](img/127d73397a99e19c71e910e49ea378af_9.png)

`收益率 = 因子 * 因子收益率 + 偏置项`

![](img/127d73397a99e19c71e910e49ea378af_11.png)

这个公式可以类比为一个回归方程：目标变量Y（收益率）与输入X（因子）之间存在关系。因子收益率就是公式中的权重参数 **W**。我们的目标就是求解出这个 **W** 的值。

![](img/127d73397a99e19c71e910e49ea378af_13.png)

对于多个因子（例如100或200个），我们可以为每个因子计算其对应的因子收益率。计算完成后，我们可以比较哪些因子的收益率更高，从而将其作为一个筛选指标。

![](img/127d73397a99e19c71e910e49ea378af_15.png)

## 因子收益率的用途

因子收益率主要有两个用途。

![](img/127d73397a99e19c71e910e49ea378af_17.png)

![](img/127d73397a99e19c71e910e49ea378af_19.png)

**第一，用于可视化分析。**
以下是因子收益率分析图的解读方式：
*   图中将因子取值区间（例如0-0.2， 0.2-0.4等）分为若干组。
*   对于每一组因子，观察其带来的平均收益大小。
*   例如，在图中，2号区间对应的平均收益较高，说明因子在该取值范围内能带来较大的收益。

![](img/127d73397a99e19c71e910e49ea378af_21.png)

![](img/127d73397a99e19c71e910e49ea378af_23.png)

![](img/127d73397a99e19c71e910e49ea378af_25.png)

通过这种可视化，我们可以分析每个因子在何种取值区间内更为有效。

![](img/127d73397a99e19c71e910e49ea378af_27.png)

**第二，作为因子筛选的定量指标。**
我们可以实际计算出每个因子的收益率数值。当需要从大量因子（如500个）中筛选出少数有效因子（如30个）时，因子收益率可以作为一个重要的参考指标。

![](img/127d73397a99e19c71e910e49ea378af_29.png)

![](img/127d73397a99e19c71e910e49ea378af_31.png)

当然，在实际筛选过程中，我们不会只依赖因子收益率。还需要综合其他指标进行评估，例如：
*   IC值（信息系数）
*   IC值的均值（IC Mean）
*   IC值的标准差（IC Std）
*   信息比率（IR = IC Mean / IC Std），它代表了因子的稳定性

![](img/127d73397a99e19c71e910e49ea378af_33.png)

![](img/127d73397a99e19c71e910e49ea378af_35.png)

最终，我们需要综合多项指标来评比，选出综合表现更优的因子。

![](img/127d73397a99e19c71e910e49ea378af_37.png)

![](img/127d73397a99e19c71e910e49ea378af_39.png)
