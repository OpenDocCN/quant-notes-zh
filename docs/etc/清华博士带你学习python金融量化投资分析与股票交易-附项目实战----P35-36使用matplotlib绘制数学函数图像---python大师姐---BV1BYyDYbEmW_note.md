在本节课中，我们将学习如何使用Python的matplotlib库来绘制数学函数图像。这是金融量化分析中可视化数据趋势和模型的基础技能。

## 概述

计算机绘制曲线或圆形时，实际上是通过连接一系列密集的点来近似实现的。例如，在绘图软件中放大一个圆，会发现它由许多小点组成。绘制数学函数图像也是同样的原理：我们需要生成一系列密集的X值，计算出对应的Y值，然后将这些点连接起来，就形成了函数曲线。

![](img/af76afce99084bf02bfb4c51e7112432_1.png)

## 生成密集的数据点

上一节我们介绍了绘图的基本概念，本节中我们来看看如何具体生成绘制函数所需的数据点。

为了绘制平滑的曲线，我们需要在X轴上定义一系列非常接近的点。这可以通过NumPy库的`linspace`函数来实现。`linspace`与`arange`不同，它允许我们指定将一段区间均匀分成多少份。

以下是使用`linspace`生成数据点的代码示例：

![](img/af76afce99084bf02bfb4c51e7112432_3.png)

```python
import numpy as np
# 在区间[-100, 100]内生成10000个均匀分布的点
x = np.linspace(-100, 100, 10000)
```

## 计算函数值并绘图

有了X值的数组后，我们就可以计算对应函数的Y值。NumPy数组支持向量化运算，可以直接对数组进行数学运算，非常方便。

![](img/af76afce99084bf02bfb4c51e7112432_5.png)

接下来，我们将绘制三个函数的图像：`y = x`， `y = x^2`， 以及 `y = 3x^3 + 5x^2 + 2x + 1`。

以下是完整的绘图步骤：

![](img/af76afce99084bf02bfb4c51e7112432_7.png)

```python
import matplotlib.pyplot as plt
import numpy as np

# 1. 生成X轴数据点
x = np.linspace(-100, 100, 10000)

# 2. 计算三个函数的Y值
y1 = x                     # y = x
y2 = x ** 2                # y = x^2
y3 = 3 * x**3 + 5 * x**2 + 2 * x + 1  # y = 3x^3 + 5x^2 + 2x + 1

# 3. 绘制三条曲线，并设置颜色和图例标签
plt.plot(x, y1, color='red', label='y = x')
plt.plot(x, y2, color='green', label='y = x^2')
plt.plot(x, y3, color='blue', label='y = 3x^3 + 5x^2 + 2x + 1')

# 4. 添加图例
plt.legend()
plt.show()
```

## 调整坐标轴范围

运行上述代码后，你可能会发现红色的直线（`y = x`）几乎看不见。这是因为三次函数`y3`的值增长极快，其Y值范围（例如达到数百万）远大于线性函数`y1`的范围（仅在-100到100之间），导致`y1`在图表中显示为一条贴近X轴的线。

为了解决这个问题，我们可以使用`ylim`函数来手动设置Y轴的显示范围，以便更清晰地观察所有曲线。

以下是调整Y轴范围的代码：

```python
# ... 前述计算x, y1, y2, y3的代码 ...

plt.plot(x, y1, color='red', label='y = x')
plt.plot(x, y2, color='green', label='y = x^2')
plt.plot(x, y3, color='blue', label='y = 3x^3 + 5x^2 + 2x + 1')

plt.legend()
# 设置Y轴显示范围为[-1000, 1000]
plt.ylim(-1000, 1000)
plt.show()
```

调整后，各条曲线就能在设定的视野内清晰显示了。你也可以根据需要调整X轴的范围（使用`xlim`函数）或生成数据点的区间（修改`linspace`的参数）。

## 总结

本节课中我们一起学习了使用matplotlib和NumPy绘制数学函数图像的核心方法。

![](img/af76afce99084bf02bfb4c51e7112432_9.png)

1.  **核心原理**：通过生成密集的X-Y点对并连接来近似曲线。
2.  **关键工具**：使用NumPy的`linspace`生成均匀分布的X值数组，并利用数组的向量化运算高效计算Y值。
3.  **绘图步骤**：调用`plt.plot()`绘制曲线，并用`label`和`legend`添加图例。
4.  **视图调整**：使用`plt.ylim()`和`plt.xlim()`调整坐标轴范围，以优化图像的显示效果。

这种方法在数学实验、数据分析及金融模型的图形化展示中都非常常用。
