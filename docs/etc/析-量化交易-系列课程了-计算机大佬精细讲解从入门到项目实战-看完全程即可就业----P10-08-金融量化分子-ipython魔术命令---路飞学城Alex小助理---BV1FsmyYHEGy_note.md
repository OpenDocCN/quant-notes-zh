在本节课中，我们将要学习IPython中一个非常实用的高级功能——魔术命令。魔术命令以百分号开头，能够极大地提升我们在交互式环境中的工作效率，例如运行外部脚本、粘贴代码块以及精确测量代码执行时间等。

## 魔术命令简介

上一节我们介绍了IPython的基本交互功能，本节中我们来看看IPython提供的“魔术命令”。魔术命令是以单个百分号 `%` 开头的特殊指令，它们为IPython环境增添了强大的扩展功能。

## 运行外部Python脚本

![](img/506ab1691538d303a5130833603c20b7_3.png)

在标准的Python命令行中，若想运行一个已写好的Python文件，通常需要退出交互模式，然后在终端执行 `python filename.py`。在IPython中，我们可以使用魔术命令 `%run` 直接在交互器中运行外部脚本，无需退出当前环境。

![](img/506ab1691538d303a5130833603c20b7_5.png)

以下是具体操作步骤：
1.  首先，在任意文本编辑器（如Vim、Sublime Text）中创建一个Python文件，例如 `hello.py`。
2.  在文件中写入简单的代码，例如 `print(“Hello World”)`。
3.  在IPython交互器中，输入命令 `%run hello.py` 即可执行该脚本。

**代码示例**：
```python
# 在IPython中输入
%run hello.py
```
执行后，脚本中的 `print` 语句输出会显示在IPython中。

## 粘贴并执行代码

![](img/506ab1691538d303a5130833603c20b7_7.png)

有时我们需要从编辑器或其他地方复制一段代码到IPython中执行。直接粘贴可能会因为缩进（如Tab键）等问题导致错误。为此，IPython提供了 `%paste` 魔术命令。

![](img/506ab1691538d303a5130833603c20b7_9.png)

`%paste` 命令会执行系统剪贴板中的代码内容。它首先将代码打印出来以便确认，然后自动执行。

![](img/506ab1691538d303a5130833603c20b7_11.png)

![](img/506ab1691538d303a5130833603c20b7_13.png)

**操作流程**：
1.  在任何地方复制一段Python代码。
2.  在IPython中输入 `%paste` 并回车。
3.  IPython会打印出剪贴板中的代码并执行它。

这对于测试长代码中的某一部分片段非常方便，无需单独创建文件。

## 测量代码执行时间

在性能分析和优化时，我们经常需要测量代码的执行时间。虽然可以使用Python的 `time` 模块，但对于执行时间极短的代码，测量结果可能不精确（例如显示为0秒）。

IPython的 `%timeit` 魔术命令可以解决这个问题。它会自动多次运行指定的代码，并计算平均执行时间，从而得到更精确的结果，尤其适用于微基准测试。

**使用方法**：
在需要计时的单行代码前加上 `%timeit`。

**代码示例**：
假设我们想测量对一个列表进行排序所需的时间。
```python
import random
# 生成一个包含1万个随机数的列表
my_list = [random.random() for _ in range(10000)]

# 使用 %timeit 测量排序时间
%timeit sorted(my_list)
```
执行后，`%timeit` 会输出类似 `146 µs ± 7.42 µs per loop` 的结果，表示平均每次循环耗时146微秒，并给出了误差范围。它会自动决定运行次数（如“7次运行，每次100万循环”），以确保测量的准确性。

![](img/506ab1691538d303a5130833603c20b7_15.png)

本节课中我们一起学习了IPython中三个核心的魔术命令：`%run` 用于直接运行外部脚本；`%paste` 用于安全地粘贴并执行剪贴板中的代码；`%timeit` 用于精确测量短小代码片的执行时间。掌握这些命令能显著提升你在IPython环境中进行数据分析、代码测试和性能优化的效率。
