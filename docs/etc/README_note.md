# Python入门：1：变量与数据类型

在本节课中，我们将要学习Python编程中最基础的两个概念：变量和数据类型。它们是构建所有程序的基石。

## 概述

变量是存储信息的容器，而数据类型则定义了容器中信息的种类和可执行的操作。理解它们对于编写有效的Python代码至关重要。

## 变量

变量可以被想象成一个贴有标签的盒子，用于存放数据。在Python中，创建一个变量并为其赋值非常简单。

以下是创建变量的基本语法：
```python
变量名 = 值
```
例如，`name = "小明"` 创建了一个名为 `name` 的变量，并将字符串“小明”存储在其中。变量名可以自由选择，但需遵循一些规则：只能包含字母、数字和下划线，且不能以数字开头。

上一节我们介绍了变量的基本概念，本节中我们来看看Python中主要的数据类型。

## 数据类型

数据类型决定了数据的性质以及可以对数据执行哪些操作。Python有几种内置的核心数据类型。

以下是几种最常见的数据类型：

*   **整数**：表示没有小数部分的数字，例如 `10`, `-5`, `0`。
*   **浮点数**：表示带有小数点的数字，例如 `3.14`, `2.0`, `-0.5`。
*   **字符串**：表示文本信息，由单引号或双引号包围，例如 `"你好世界"`, `'Python'`。
*   **布尔值**：表示逻辑值，只有两个：`True`（真）和 `False`（假）。

了解数据类型后，我们来看看如何检查一个变量的类型，这在编程中非常有用。

## 使用 `type()` 函数

`type()` 函数可以帮助我们查看任何变量或值的数据类型。

其使用格式为：
```python
type(要检查的对象)
```
例如，执行 `print(type(10))` 会输出 `<class 'int'>`，表明10是一个整数类型。

掌握了类型检查，我们有时还需要在不同类型之间进行转换。

## 类型转换

类型转换是指将一种数据类型转换为另一种。Python提供了几个内置函数来实现这一点。

以下是常用的类型转换函数：

*   `int()`：将数据转换为整数。
*   `float()`：将数据转换为浮点数。
*   `str()`：将数据转换为字符串。

例如，`int("123")` 会将字符串“123”转换为整数 `123`。但请注意，`int("abc")` 这样的转换会失败，因为“abc”无法被合理地解释为一个数字。

## 总结

本节课中我们一起学习了Python编程的基础。我们首先了解了**变量**作为数据容器的角色及其命名规则。接着，我们探讨了四种核心**数据类型**：整数、浮点数、字符串和布尔值。然后，我们学会了使用 `type()` 函数来检查数据的类型。最后，我们介绍了如何使用 `int()`, `float()`, `str()` 等函数进行**类型转换**。牢固掌握这些概念，将为后续学习更复杂的编程知识打下坚实的基础。
