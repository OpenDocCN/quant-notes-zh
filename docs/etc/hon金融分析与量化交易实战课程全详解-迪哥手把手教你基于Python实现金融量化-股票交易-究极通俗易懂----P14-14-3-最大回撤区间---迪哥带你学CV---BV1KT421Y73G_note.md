在本节课中，我们将要学习一个重要的风险指标——最大回撤。它描述了在投资过程中，资产价值从最高点到最低点的最大跌幅，是衡量策略风险的关键指标。

![](img/4e85d852f3f1e8426e8e444ee70a958a_3.png)

## 什么是最大回撤？📉

上一节我们介绍了策略分析中的各种指标，本节中我们来看看最大回撤。最大回撤描述的是投资过程中“最惨”的阶段。它衡量的是从任意一个历史高点，到之后任意一个低点，资产价值下跌的最大幅度。

具体来说，它关注的是在股票或策略净值下跌过程中，最严重、持续时间最长的那段下跌区间。这个指标不能仅从字面理解，它核心描述的是风险，即你可能经历的最大亏损幅度。

## 最大回撤的计算公式 📐

为了更精确地理解，我们来看最大回撤的数学定义。假设 `P_i` 代表第 `i` 天的资产价格（或策略净值），`P_j` 代表第 `i` 天之后第 `j` 天的价格。

最大回撤的计算公式如下：

**最大回撤 = max_{i, j (i<j)} [ (P_i - P_j) / P_i ]**

这个公式的含义是：遍历所有可能的 `i` 和 `j`（`i` 在前，`j` 在后），计算从 `i` 日到 `j` 日的相对跌幅 `(P_i - P_j) / P_i`，然后取其中的最大值。这个最大值就是整个投资期间的最大回撤率。

![](img/4e85d852f3f1e8426e8e444ee70a958a_5.png)

## 为什么最大回撤很重要？⚠️

![](img/4e85d852f3f1e8426e8e444ee70a958a_7.png)

最大回撤是一个风险指标，而非盈利指标。它告诉你，即使一个策略最终是盈利的，但在过程中你可能会经历多么“惨烈”的下跌。例如，在2008年金融危机中，股市从高点大幅下挫，许多投资者承受了巨大损失，这正是巨大回撤的体现。

![](img/4e85d852f3f1e8426e8e444ee70a958a_9.png)

了解最大回撤有助于你评估策略的风险承受能力。如果你无法承受策略历史数据中出现的最大回撤，那么这个策略可能不适合你。

![](img/4e85d852f3f1e8426e8e444ee70a958a_11.png)

## 使用Python计算最大回撤 💻

![](img/4e85d852f3f1e8426e8e444ee70a958a_13.png)

接下来，我们看看如何用Python的Pandas库来计算最大回撤。核心思路是利用 `.cummax()` 函数来动态计算历史最高点。

![](img/4e85d852f3f1e8426e8e444ee70a958a_15.png)

以下是计算最大回撤的步骤和示例代码：

![](img/4e85d852f3f1e8426e8e444ee70a958a_17.png)

![](img/4e85d852f3f1e8426e8e444ee70a958a_19.png)

首先，我们需要一个代表每日资产净值的时间序列数据。假设我们有一个名为 `net_value` 的Pandas Series。

![](img/4e85d852f3f1e8426e8e444ee70a958a_21.png)

![](img/4e85d852f3f1e8426e8e444ee70a958a_23.png)

1.  **计算历史最高点序列**：使用 `.cummax()` 方法，得到截至每一天的历史最高净值。
    ```python
    historical_max = net_value.cummax()
    ```
2.  **计算每日回撤**：用历史最高点减去当日的实际净值，得到每日的绝对回撤值。
    ```python
    drawdown = historical_max - net_value
    ```
3.  **计算每日回撤率**：将每日绝对回撤除以对应的历史最高点，得到回撤率。
    ```python
    drawdown_pct = drawdown / historical_max
    ```
4.  **找出最大回撤**：在所有每日回撤率中，找到最大值，即为最大回撤。
    ```python
    max_drawdown = drawdown_pct.max()
    ```

![](img/4e85d852f3f1e8426e8e444ee70a958a_25.png)

![](img/4e85d852f3f1e8426e8e444ee70a958a_27.png)

为了更直观，以下是一个完整的代码示例：

![](img/4e85d852f3f1e8426e8e444ee70a958a_29.png)

```python
import pandas as pd

![](img/4e85d852f3f1e8426e8e444ee70a958a_31.png)

# 假设 net_value 是一个包含每日净值的Pandas Series
# 例如：net_value = pd.Series([100, 105, 102, 98, 110, 108, 115, 112])

![](img/4e85d852f3f1e8426e8e444ee70a958a_33.png)

![](img/4e85d852f3f1e8426e8e444ee70a958a_35.png)

# 1. 计算历史最高点序列
historical_max = net_value.cummax()

![](img/4e85d852f3f1e8426e8e444ee70a958a_37.png)

# 2. 计算每日回撤（绝对值和百分比）
drawdown_abs = historical_max - net_value
drawdown_pct = drawdown_abs / historical_max

# 3. 计算最大回撤（率）
max_drawdown = drawdown_pct.max()

![](img/4e85d852f3f1e8426e8e444ee70a958a_39.png)

![](img/4e85d852f3f1e8426e8e444ee70a958a_41.png)

print(f“最大回撤率为：{max_drawdown:.2%}”)
```

执行这段代码，你将得到该净值序列在整个观察期内的最大回撤率。
