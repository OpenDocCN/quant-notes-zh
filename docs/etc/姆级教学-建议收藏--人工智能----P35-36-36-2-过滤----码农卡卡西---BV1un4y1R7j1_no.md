### 1. 过滤停牌股
我们首先定义一个函数，用于判断并过滤掉全年停牌的股票。这可以通过调用特定的API来实现。

![](img/bb9a774f54a00269b8be1c62e7099cf8_28.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_30.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_32.png)

```python
def filter_suspended_stocks(stock_list):
    """
    过滤掉全年停牌的股票。
    :param stock_list: 股票代码列表
    :return: 过滤后的股票代码列表
    """
    filtered_list = []
    for stock in stock_list:
        # 调用API判断股票是否全年停牌
        if not is_suspended(stock):
            filtered_list.append(stock)
    return filtered_list
```

### 2. 过滤ST股
接下来，我们需要过滤掉ST股（即被特别处理的股票）。逻辑与过滤停牌股类似。

![](img/bb9a774f54a00269b8be1c62e7099cf8_42.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_44.png)

```python
def filter_st_stocks(stock_list):
    """
    过滤掉ST股。
    :param stock_list: 股票代码列表
    :return: 过滤后的股票代码列表
    """
    filtered_list = []
    for stock in stock_list:
        # 调用API判断股票是否为ST股
        if not is_st(stock):
            filtered_list.append(stock)
    return filtered_list
```

### 3. 过滤新股
最后，我们过滤掉上市时间过短的新股。这里我们设定一个阈值（例如180天），只保留上市超过此天数的股票。

![](img/bb9a774f54a00269b8be1c62e7099cf8_54.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_56.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_58.png)

```python
def filter_new_stocks(stock_list, days_threshold=180):
    """
    过滤掉上市天数少于指定阈值的新股。
    :param stock_list: 股票代码列表
    :param days_threshold: 上市天数阈值，默认为180天
    :return: 过滤后的股票代码列表
    """
    filtered_list = []
    for stock in stock_list:
        # 获取股票上市天数
        days_listed = get_days_listed(stock)
        if days_listed > days_threshold:
            filtered_list.append(stock)
    return filtered_list
```

### 数据转置与清洗
查询返回的数据格式通常是横向的（指标为列，股票为行）。为了便于后续分析，我们通常需要将其转置为纵向格式（每行代表一只股票的一个指标）。同时，需要处理可能存在的空值。

![](img/bb9a774f54a00269b8be1c62e7099cf8_152.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_154.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_156.png)

```python
# 转置数据，使每行对应一只股票
transposed_data = fundamental_data.T

![](img/bb9a774f54a00269b8be1c62e7099cf8_158.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_160.png)

![](img/bb9a774f54a00269b8be1c62e7099cf8_162.png)

# 删除包含空值的行（简单处理）
cleaned_data = transposed_data.dropna()
```
