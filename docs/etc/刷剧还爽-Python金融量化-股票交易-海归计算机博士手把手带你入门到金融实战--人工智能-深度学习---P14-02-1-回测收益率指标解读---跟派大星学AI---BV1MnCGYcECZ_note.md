## 理解回测收益率

![](img/aa194881ee51b37c381c8cbeb4f52cae_19.png)

在众多指标中，回测收益率是最基本、最重要的一个。它衡量了策略在整个回测周期内，相对于初始本金所获得的总收益比例。

![](img/aa194881ee51b37c381c8cbeb4f52cae_21.png)

例如，假设我们针对某只股票从2015年到2020年的数据设计了一个买卖策略。回测收益率会告诉我们，最终我们的资产相对于最初投入的本金增长（或减少）了多少百分比。如下图所示，4.17%即表示总收益率为4.17%。

![](img/aa194881ee51b37c381c8cbeb4f52cae_21.png)

其计算公式非常直观：

![](img/aa194881ee51b37c381c8cbeb4f52cae_23.png)

**回测收益率 = (期末资产总值 - 期初本金) / 期初本金**

用符号表示为：
**Return = (P_t - P_0) / P_0**

![](img/aa194881ee51b37c381c8cbeb4f52cae_25.png)

其中：
*   **P_t** 代表回测结束时的资产总值（卖出所有头寸后的总金额）。
*   **P_0** 代表回测开始时的初始本金。

![](img/aa194881ee51b37c381c8cbeb4f52cae_27.png)

这个指标直接回答了“这个策略最终让我赚（赔）了百分之多少”这个问题。

## 计算与可视化回测收益率

为了更深入地理解，我们可以计算策略在回测期间每一天的累计收益率变化，并将其可视化。

计算思路是：将每一天的资产价值（这里用收盘价近似模拟）与第一天的初始价值相比，得到每日的累计净值，最后再通过公式计算总收益率。

以下是计算步骤：

```python
# 1. 计算每日累计净值：每日收盘价 / 初始日收盘价
cumulative_value = data[‘close‘] / data[‘close‘].iloc[0]

# 2. 绘制累计净值曲线图
plt.figure(figsize=(10, 6))
plt.plot(cumulative_value.index, cumulative_value.values)
plt.title(‘策略累计净值走势图‘)
plt.xlabel(‘日期‘)
plt.ylabel(‘累计净值‘)
plt.grid(True)
plt.show()
```

```python
# 3. 计算最终的回测收益率： (期末净值 - 期初净值) / 期初净值
# 由于期初净值被标准化为1，所以公式简化为：期末净值 - 1
final_return = cumulative_value.iloc[-1] - 1

![](img/aa194881ee51b37c381c8cbeb4f52cae_39.png)

# 4. 将结果整理成表格形式展示
result_df = pd.DataFrame({
    ‘回测收益率‘: [final_return]
}, index=[‘结果‘])
print(result_df)
```

## 本节总结

![](img/aa194881ee51b37c381c8cbeb4f52cae_45.png)

本节课中，我们一起学习了量化策略评估的基石——回测收益率。

![](img/aa194881ee51b37c381c8cbeb4f52cae_47.png)

*   **核心概念**：回测收益率衡量了策略在历史测试中获得的**总收益百分比**，其**公式**为 `(期末资产 - 期初本金) / 期初本金`。
*   **计算过程**：我们通过Python代码演示了如何从价格数据计算每日累计净值，并最终得到总收益率。
*   **重要意义**：这是评估策略盈利能力的第一个且最直接的指标。一个策略必须首先具备正的回测收益率，才有进一步分析其他指标（如风险、稳定性）的价值。

![](img/aa194881ee51b37c381c8cbeb4f52cae_49.png)
