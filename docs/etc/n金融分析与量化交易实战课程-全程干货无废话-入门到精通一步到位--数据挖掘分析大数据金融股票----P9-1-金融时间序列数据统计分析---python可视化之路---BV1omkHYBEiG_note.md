## 概述

---

## 数据读取与预处理

上一节我们介绍了课程目标，本节中我们来看看如何获取并准备我们的数据。

首先，我们需要读取包含金融时间序列数据的CSV文件。数据中的日期格式可能与标准格式不符，因此需要进行处理。

```python
import pandas as pd

# 读取数据，并指定‘Date’列为索引，同时解析日期格式
df = pd.read_csv('your_data.csv', index_col='Date', parse_dates=True)
```

执行上述代码后，`DataFrame`的索引将变为标准的`datetime`格式，方便后续基于时间的操作。数据中可能包含一些空值（例如节假日），这是金融数据的常见情况。

---

## 数据可视化

在准备好数据之后，我们可以通过可视化直观地观察各个金融指标（如股价、汇率）的走势。

以下是绘制数据走势的两种常用方法：

1.  **单图展示所有指标**：所有折线绘制在同一张图中，便于整体对比。
    ```python
    df.plot(figsize=(12, 8))
    ```
2.  **子图分别展示各指标**：每个指标拥有独立的子图，展示更清晰。
    ```python
    df.plot(subplots=True, figsize=(12, 8))
    ```

具体使用哪种方式取决于分析需求。我们的数据可能包含苹果、微软等公司的股价，以及黄金价格、汇率等多个指标。

---

## 基础统计分析

可视化帮助我们形成初步印象，接下来我们需要用具体的统计数字来描述数据。

最直接的方法是使用`.describe()`函数，它可以快速生成数据集的概要统计。

```python
# 生成基础统计描述，并保留两位小数
summary = df.describe().round(2)
```

`.describe()`默认返回计数、均值、标准差、最小值、四分位数和最大值。通过`.round(2)`，我们控制了输出结果的小数位数，使报表更简洁易读。

---

## 自定义指标聚合

有时我们只需要特定的统计指标，或者希望以自定义的格式汇总它们。这时可以使用`.agg()`聚合函数。

以下是使用聚合函数计算指定统计量的示例：

```python
import numpy as np

# 自定义聚合统计：最小值、最大值、均值、标准差、中位数
custom_stats = df.agg(['min', 'max', np.mean, np.std, np.median])
```

通过`.agg()`，我们可以自由组合`Pandas`内置函数（如`‘min’`）或`NumPy`函数（如`np.std`），灵活地生成所需的统计汇总表。这种方法比`.describe()`更具针对性。
