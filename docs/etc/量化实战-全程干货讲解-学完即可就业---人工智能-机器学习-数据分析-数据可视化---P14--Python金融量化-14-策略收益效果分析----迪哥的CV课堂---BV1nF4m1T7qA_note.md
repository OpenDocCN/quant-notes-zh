## 计算对数收益率

上一节我们介绍了策略信号，本节中我们来看看如何计算策略的收益。收益的计算是一个累加过程，但直接累加并不方便，因此我们引入对数收益率进行计算。

对数收益率的计算公式为：
`log_return = log(price_t / price_{t-1})`

以下是计算对数收益率的步骤：
1.  获取股价数据。
2.  计算当前股价除以前一日股价的结果。
3.  对这个比值取自然对数。

![](img/744ce5b7706f4769871028dc90343c54_5.png)

在代码中，我们可以使用Pandas的`shift`函数来获取前一日的数据，并进行计算。

![](img/744ce5b7706f4769871028dc90343c54_7.png)

```python
# 假设 df[‘Close’] 是股价序列
returns = np.log(df[‘Close’] / df[‘Close’].shift(1))
```

![](img/744ce5b7706f4769871028dc90343c54_9.png)

执行后，我们得到了一个表示每日增长率的`returns`序列。

![](img/744ce5b7706f4769871028dc90343c54_11.png)

## 结合策略信号计算策略收益

![](img/744ce5b7706f4769871028dc90343c54_13.png)

接下来，我们需要将计算出的收益率与我们的策略信号结合起来。我们的策略信号`position`是一个序列，其中的值通常为1或-1。

![](img/744ce5b7706f4769871028dc90343c54_15.png)

*   当`position`为1时，表示我们遵循原始市场走势。
*   当`position`为-1时，表示我们对原始市场走势进行反向操作（例如，在预测下跌时做空）。

![](img/744ce5b7706f4769871028dc90343c54_17.png)

因此，策略每日的收益率应为策略信号乘以原始对数收益率：
`strategy_returns = position * returns`

同样，这里需要注意时间对齐问题，策略信号也需要进行`shift`操作，以确保使用的是前一交易日结束时的信号来决定当日的操作。

![](img/744ce5b7706f4769871028dc90343c54_19.png)

```python
# 假设 position 是策略信号序列
strategy_returns = position.shift(1) * returns
```

![](img/744ce5b7706f4769871028dc90343c54_21.png)

计算完成后，我们可以查看`strategy_returns`序列。在数据初期，由于信号可能大部分时间相同，策略收益可能与原始收益看起来一致。

![](img/744ce5b7706f4769871028dc90343c54_23.png)

![](img/744ce5b7706f4769871028dc90343c54_25.png)

## 累计收益计算与对比

![](img/744ce5b7706f4769871028dc90343c54_27.png)

我们想看到策略的最终效果，就需要计算累计收益。这需要将每日的收益率进行累加。

![](img/744ce5b7706f4769871028dc90343c54_29.png)

然而，由于我们之前计算的是对数收益率，其累加和等价于计算总的对数收益。为了得到直观的“1元钱最终变成多少元”的结果，我们需要将对数收益还原为普通收益。

还原公式为：
`total_return = exp(sum(log_returns))`

![](img/744ce5b7706f4769871028dc90343c54_31.png)

以下是计算与对比的步骤：
1.  分别对原始收益率序列`returns`和策略收益率序列`strategy_returns`进行求和（`sum`操作）。
2.  对求和结果应用指数函数`np.exp`进行还原。
3.  对比两个结果。

![](img/744ce5b7706f4769871028dc90343c54_33.png)

```python
# 计算基准总收益（什么都不做）
benchmark_total_return = np.exp(returns.sum())
# 计算策略总收益
strategy_total_return = np.exp(strategy_returns.sum())

![](img/744ce5b7706f4769871028dc90343c54_35.png)

![](img/744ce5b7706f4769871028dc90343c54_37.png)

print(f“基准总收益：{benchmark_total_return:.2f}”)
print(f“策略总收益：{strategy_total_return:.2f}”)
```

例如，输出结果可能显示：
*   基准总收益：4.00 （意味著1元钱最终变为4元）
*   策略总收益：5.80 （意味著1元钱最终变为5.8元）

![](img/744ce5b7706f4769871028dc90343c54_39.png)

这表明，在我们使用的双均线策略下，收益比单纯持有股票更高。

![](img/744ce5b7706f4769871028dc90343c54_41.png)

## 总结

![](img/744ce5b7706f4769871028dc90343c54_43.png)

![](img/744ce5b7706f4769871028dc90343c54_45.png)

本节课中我们一起学习了量化策略收益分析的核心流程。我们首先通过计算对数收益率来获取每日的增长数据；然后，将收益率与代表买卖方向的策略信号相乘，得到策略的每日收益序列；最后，通过对收益序列进行累加和指数还原，计算出策略的最终总收益，并与基准收益进行对比，从而量化评估策略的绩效。
