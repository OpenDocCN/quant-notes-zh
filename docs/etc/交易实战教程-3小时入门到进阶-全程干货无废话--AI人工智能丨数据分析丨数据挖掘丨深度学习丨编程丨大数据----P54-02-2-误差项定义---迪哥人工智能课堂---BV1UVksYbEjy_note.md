在本节课中，我们将要学习线性回归模型中一个核心概念——误差项的定义。我们将探讨如何构建一个拟合数据的平面，并理解真实值与预测值之间的差异，这是后续定义损失函数、训练模型的基础。

## 构建拟合平面

上一节我们介绍了线性回归的基本思想，本节中我们来看看如何用数学公式来描述这个“最拟合”的平面。

首先，我们给出拟合平面的公式。假设我们有两个特征：工资（X1）和年龄（X2）。我们的预测值（即拟合平面上的点）可以表示为：

**hθ(x) = θ₀ + θ₁X₁ + θ₂X₂**

在这个公式中：
*   **θ₁** 和 **θ₂** 被称为**权重项**。它们分别代表了工资和年龄对最终预测结果（如贷款额度）的影响程度。
*   **θ₀** 被称为**偏置项**。它的作用是**对预测结果进行微调**。无论特征值如何变化，偏置项的改变会使整个预测平面向上或向下平移，以弥补模型可能存在的系统性偏差。需要强调的是，偏置项的作用是微调，而非大范围地改变模型。

## 转换为矩阵形式

我们之前提到，数据处理通常基于矩阵运算。然而，当前的公式 `θ₀ + θ₁X₁ + θ₂X₂` 由于存在独立的 `θ₀` 项，无法直接写成简洁的矩阵乘法形式。

![](img/5234c92e19dbe5805a2f38bcfe2e0b28_2.png)

为了解决这个问题，我们需要对公式进行转换。观察发现，如果为 `θ₀` 也配对一个特征 `X₀`，那么整个公式就可以统一为矩阵形式：`θ₀X₀ + θ₁X₁ + θ₂X₂`。

那么，`X₀` 应该是什么呢？`X₁`（工资）和 `X₂`（年龄）都有明确的现实含义，而 `X₀` 是我们为了计算方便**构造出来的**。为了让 `θ₀X₀` 等价于原来的 `θ₀`，`X₀` 的值必须恒为 **1**。

![](img/5234c92e19dbe5805a2f38bcfe2e0b28_4.png)

![](img/5234c92e19dbe5805a2f38bcfe2e0b28_6.png)

因此，在数据预处理阶段，我们通常会在原始数据矩阵中**手动添加一列**，这一列的所有值都为1，并将其命名为 `X₀`。这样做的目的纯粹是为了数学表达的便利，以便将模型转换为 `hθ(x) = θᵀX` 的矩阵形式，其中 `θ` 是包含所有参数 `(θ₀, θ₁, θ₂)` 的向量，`X` 是包含所有特征 `(X₀, X₁, X₂)` 的向量。

![](img/5234c92e19dbe5805a2f38bcfe2e0b28_8.png)

## 定义误差项

![](img/5234c92e19dbe5805a2f38bcfe2e0b28_10.png)

现在，我们有了拟合平面（预测模型）`hθ(x)`。接下来，我们考虑单个数据点。

对于任意一个样本（例如一位客户的记录），我们有其真实的特征值 `(X₁, X₂)` 和银行历史上实际给予的贷款额度 `Y`（**真实值**）。将特征值代入我们的平面方程 `hθ(x)`，会得到一个计算出来的贷款额度（**预测值**）。

在三维空间中，真实数据点（红色点）和拟合平面上的对应预测点（蓝色点）在竖直方向（Y轴方向）上存在一个**差值**。这个差值就是**误差项（error）**。

用公式表示即为：
**error⁽ⁱ⁾ = y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾)**
其中，上标 `(i)` 代表第 `i` 个样本，表明每个样本都有其独立的误差。

## 机器学习的核心：目标函数

机器学习可以通俗地理解为：给机器提供**数据（Data）** 和一个明确的**学习目标（Goal）**，机器会自动寻找最优参数来实现这个目标。这个“学习目标”在数学上被称为**损失函数（Loss Function）** 或**目标函数**。

对于我们的回归问题，最自然的目标就是让预测尽可能准确，即让**误差项尽可能小**。最理想的情况是误差为零，这意味着预测值与真实值完全吻合。

因此，我们的损失函数将围绕**最小化所有样本的误差**这一核心思想来构建。损失函数值越接近零，代表模型拟合得越好。这是定义几乎所有机器学习模型损失函数的基本出发点。
