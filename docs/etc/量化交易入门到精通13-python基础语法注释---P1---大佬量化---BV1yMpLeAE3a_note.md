# 量化交易入门到精通：13：Python基础语法注释

在本节课中，我们将要学习Python编程中一个非常基础但至关重要的概念——注释。注释是写在代码中，用于解释代码功能、逻辑或意图的文字，它不会被Python解释器执行。掌握注释的使用，能让你的代码更易读、更易维护，尤其是在团队协作中。

## 注释的种类与写法

![](img/c640e71b7ec2bece9ae8bb4929da41f1_1.png)

Python中的注释主要分为两种：单行注释和多行注释。它们各有其特定的语法格式。

### 单行注释

![](img/c640e71b7ec2bece9ae8bb4929da41f1_3.png)

单行注释用于对单行代码进行简短说明。其写法是以井号 `#` 开头，`#` 之后直到行尾的所有内容都会被Python视为注释。

![](img/c640e71b7ec2bece9ae8bb4929da41f1_5.png)

以下是单行注释的示例：
```python
# 这是一个单行注释
收盘价 = 100  # 这里定义了股票的收盘价
```

### 多行注释

![](img/c640e71b7ec2bece9ae8bb4929da41f1_7.png)

当需要解释的说明文字较长，或者需要注释掉一整段代码时，我们可以使用多行注释。Python中，多行注释有两种等效的写法。

第一种写法是使用三个连续的双引号 `"""` 将注释内容包裹起来。
```python
"""
这是一个多行注释的示例。
它可以跨越多行，
用于解释复杂的代码逻辑或函数功能。
"""
```

![](img/c640e71b7ec2bece9ae8bb4929da41f1_9.png)

第二种写法是使用三个连续的单引号 `'''`，其效果与双引号完全相同。
```python
'''
这也是一种多行注释的写法。
选择双引号或单引号取决于个人或团队的编码风格。
'''
```

## 注释的实际应用与价值

上一节我们介绍了注释的写法，本节中我们来看看注释在实际编程中扮演的角色及其重要性。

注释的核心目的是为了方便代码的标注和解释。它能让其他人（包括未来的你自己）更快地理解代码的意图和计算逻辑。

例如，在量化交易中，我们经常需要计算股票的收益率。没有注释的代码可能令人费解，而加上注释后，其逻辑就一目了然。

以下是一个结合了变量定义、计算和注释的完整示例：
```python
# 定义股票收盘价
收盘价 = 100
# 定义投资者的成本价
成本价 = 90

# 计算股票收益率
# 公式：收益率 = (收盘价 - 成本价) / 成本价
收益率 = (收盘价 - 成本价) / 成本价

# 打印计算结果
print(f"股票收益率为：{收益率}")
```
运行这段代码，将会输出计算出的收益率。通过注释，我们清晰地说明了每个变量的含义以及收益率的具体计算公式。
