## 概述

上一节我们介绍了百分位法去极值，本节中我们来看看另一种稳健的方法：MAD法。MAD是“Median Absolute Deviation”（中位数绝对偏差）的缩写。这种方法通过数据的中位数和偏差来定义正常值的范围，从而识别并处理异常值。

![](img/82bcfefac78acdf88c0627dda738c548_4.png)

## MAD法核心概念与计算步骤

MAD法的核心思想是：先计算数据序列的中位数，然后计算每个数据点与中位数之差的绝对值，再求这些绝对值的中位数（即MAD值）。最后，根据中位数和MAD值设定一个范围，超出此范围的值被视为极值。

以下是MAD法的具体计算步骤：

![](img/82bcfefac78acdf88c0627dda738c548_6.png)

![](img/82bcfefac78acdf88c0627dda738c548_8.png)

1.  **计算中位数**：首先，计算原始数据序列的中位数。公式表示为：
    `median = Median(原始序列)`

![](img/82bcfefac78acdf88c0627dda738c548_10.png)

2.  **计算绝对偏差**：接着，计算序列中每个数据点与中位数的绝对差值。公式表示为：
    `绝对偏差序列 = |原始序列 - median|`

![](img/82bcfefac78acdf88c0627dda738c548_12.png)

3.  **计算MAD值**：然后，计算上一步得到的绝对偏差序列的中位数。这个值就是MAD。公式表示为：
    `MAD = Median(绝对偏差序列)`

![](img/82bcfefac78acdf88c0627dda738c548_14.png)

![](img/82bcfefac78acdf88c0627dda738c548_16.png)

4.  **设定范围并处理**：最后，设定正常值的上下界。通常，上界为 `median + N * MAD`，下界为 `median - N * MAD`。`N`是一个常数，通常取 **1.4826**。原始数据中超出此范围的值将被视为极值并进行处理（如截断或替换）。

![](img/82bcfefac78acdf88c0627dda738c548_18.png)

![](img/82bcfefac78acdf88c0627dda738c548_20.png)

以下是实现MAD去极值功能的函数代码：

![](img/82bcfefac78acdf88c0627dda738c548_30.png)

```python
import pandas as pd
import numpy as np

def filter_extreme_MAD(series, n):
    """
    使用MAD法对序列进行去极值处理。
    参数:
        series: 待处理的数据序列 (pandas Series)
        n: 用于计算范围的倍数，通常为1.4826
    返回:
        处理后的序列
    """
    # 1. 计算中位数
    median = series.quantile(0.5)
    
    # 2. 计算绝对偏差序列
    abs_deviation = (series - median).abs()
    
    # 3. 计算MAD值 (绝对偏差序列的中位数)
    MAD = abs_deviation.quantile(0.5)
    
    # 4. 计算上下界
    high = median + n * MAD
    low = median - n * MAD
    
    # 5. 对极值进行截断处理
    filtered_series = series.clip(lower=low, upper=high)
    
    return filtered_series
```
