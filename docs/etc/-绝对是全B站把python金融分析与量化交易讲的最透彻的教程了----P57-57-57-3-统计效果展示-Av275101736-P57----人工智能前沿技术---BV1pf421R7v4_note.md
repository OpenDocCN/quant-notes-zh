## 数据变换与统计准备
在开始统计之前，我们需要对数据进行一些变换，以便更清晰地观察特征与结果之间的关系。

![](img/4d56d3f2ba3e4d65269d7758c7304909_22.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_24.png)

当前的数据中，特征（如前一天的涨跌、前两天的涨跌）和最终结果（当天的涨跌）是组合在一起的。我们需要将结果列单独提取出来，并进行透视操作，以便统计。

![](img/4d56d3f2ba3e4d65269d7758c7304909_26.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_28.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_30.png)

以下是数据变换的步骤：
1.  使用 `groupby` 对特征列进行分组。
2.  对结果列使用 `.size()` 方法进行计数。
3.  使用 `.unstack(fill_value=0)` 方法将结果展开，缺失值用0填充，确保所有特征组合都能显示出来。

![](img/4d56d3f2ba3e4d65269d7758c7304909_32.png)

执行上述操作后，我们将得到一个清晰的表格，其中行和列代表不同的特征组合，表格内的值则代表该组合出现的次数。这使我们能够直观地看到，在特定特征条件下（例如“前一天涨，前两天涨”），股票是涨的次数多还是跌的次数多。

![](img/4d56d3f2ba3e4d65269d7758c7304909_34.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_36.png)

## 基于统计的策略制定
得到统计表格后，我们可以分析哪种特征组合下，股票上涨或下跌的可能性更大。

![](img/4d56d3f2ba3e4d65269d7758c7304909_38.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_40.png)

以下是分析过程：
*   观察统计表格，我们发现当“前一天涨”且“前两天涨”（即特征值均为1）时，表格中对应的“下跌”次数（标记为-1）多于“上涨”次数（标记为1）。
*   基于此观察，我们可以制定一个简单的策略：当出现“前一天涨且前两天涨”的情况时，预测第三天会跌；在其他特征组合下，则预测第三天会涨。

![](img/4d56d3f2ba3e4d65269d7758c7304909_42.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_44.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_46.png)

这个策略可以用代码逻辑表示：
```python
# 假设 df[‘feature1’] 是前一天涨跌，df[‘feature2’] 是前两天涨跌
df[‘position’] = np.where((df[‘feature1’] == 1) & (df[‘feature2’] == 1), -1, 1)
```
这段代码创建了一个新的列 `position`，当两个特征值都为1时，赋值为-1（表示看跌/卖出信号），否则赋值为1（表示看涨/买入信号）。

![](img/4d56d3f2ba3e4d65269d7758c7304909_48.png)

## 策略效果评估
制定策略后，我们需要评估其有效性。我们将策略预测的涨跌方向与实际股票涨跌方向进行比较。

![](img/4d56d3f2ba3e4d65269d7758c7304909_50.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_52.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_54.png)

以下是评估方法：
1.  将策略生成的 `position` 列与实际结果 `direction` 列进行比较。
2.  使用 `(df[‘position’] == df[‘direction’]).value_counts()` 统计预测正确和错误的次数。
3.  在示例中，预测正确的次数（True）为1102次，错误的次数（False）为1033次，正确率略高于50%。

![](img/4d56d3f2ba3e4d65269d7758c7304909_56.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_58.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_60.png)

![](img/4d56d3f2ba3e4d65269d7758c7304909_62.png)

为了更直观地评估，我们可以计算按照此策略进行交易的理论收益，并与单纯跟随大盘走势的收益进行对比。
```python
# 计算策略收益：策略信号 * 实际收益率
strategy_returns = df[‘position’].shift(1) * df[‘returns’]
# 计算累计收益
cumulative_strategy_returns = (1 + strategy_returns).cumprod()
```
对比结果显示，基于此简单统计策略的累计收益虽然仍为亏损，但亏损幅度小于单纯持有股票的收益。这说明该统计策略在一定程度上起到了优化作用。
