在本节课中，我们将要学习一个在金融量化分析中至关重要的风险调整后收益指标——夏普比率。我们将理解它的核心概念、计算公式，并通过Python代码演示如何计算和解读它。

## 概述

上一节我们介绍了投资回报率等基础收益指标。本节中我们来看看如何衡量“性价比”，即单位风险所带来的收益。夏普比率正是这样一个指标，它帮助我们评估一项投资在承担风险后，是否获得了足够的超额回报。

## 夏普比率的含义

夏普比率描述的是，对于每承担一单位的风险，我们能获得多少超额收益。指标越高，意味着在承担相同风险的情况下，获得的收益越高。

![](img/ad535582da8c36018bccf2d2c6a8f4ec_1.png)

我们可以用一个类比来理解：假设有两份工作，一份是稳定的文员工作，月薪1万元；另一份是去战乱地区做雇佣兵，日薪3万元。后者的高薪对应着极高的生命风险。在投资中，夏普比率就是用来衡量“高风险”是否换来了足够高的“高回报”。我们总是希望这个指标越高越好。

![](img/ad535582da8c36018bccf2d2c6a8f4ec_3.png)

## 夏普比率的计算公式

![](img/ad535582da8c36018bccf2d2c6a8f4ec_5.png)

理解了其含义后，我们来看如何计算夏普比率。它的核心思想是比较投资组合的收益与无风险收益（如国债、银行存款利率）的差异。

公式如下：

**夏普比率 = (投资组合的平均收益率 - 无风险收益率) / 投资组合收益率的标准差**

*   **投资组合的平均收益率**：你的投资在一段时间内的平均回报。
*   **无风险收益率**：理论上没有任何风险的投资所能获得的收益率，通常参考国债利率。
*   **投资组合收益率的标准差**：衡量投资回报波动性的指标，标准差越大，代表风险越高。

这个公式计算的是“每单位总风险所获得的超额回报”。在具体计算年化夏普比率时，通常还会乘以根号下交易天数（如一年约250个交易日）进行标准化。

## 用Python计算夏普比率

以下是使用Python计算多只股票夏普比率的步骤和代码示例。我们将假设无风险年化收益率为5%。

首先，我们需要准备数据，计算每只股票的历史日收益率，并处理可能存在的缺失值。

```python
import pandas as pd
import numpy as np

![](img/ad535582da8c36018bccf2d2c6a8f4ec_7.png)

# 假设 `stock_data` 是一个DataFrame，包含多只股票的每日收盘价，索引为日期
# 计算日收益率
returns = stock_data.pct_change()
# 填充缺失值（例如停牌日），用前一天的收益率填充（通常填充0或前值）
returns = returns.fillna(method='ffill')
```

接下来，我们定义无风险利率并计算夏普比率。

![](img/ad535582da8c36018bccf2d2c6a8f4ec_9.png)

```python
# 设置无风险年化收益率，例如5%
risk_free_rate = 0.05
# 一年通常按250个交易日计算
trading_days_per_year = 250

# 计算超额收益（日度）
excess_returns = returns - risk_free_rate / trading_days_per_year

# 计算夏普比率（年化）
# 公式： (平均超额日收益 * 年化因子) / (日收益标准差 * 年化因子)
# 简化后： 平均超额日收益 / 日收益标准差 * sqrt(年化因子)
sharpe_ratios = (excess_returns.mean() / returns.std()) * np.sqrt(trading_days_per_year)

print(sharpe_ratios)
```

执行以上代码后，我们会得到一个Series，其中包含每只股票的年化夏普比率。

## 如何解读结果

![](img/ad535582da8c36018bccf2d2c6a8f4ec_11.png)

查看计算结果，例如：
*   股票A: 1.50
*   股票B: 0.80
*   股票C: -0.20

![](img/ad535582da8c36018bccf2d2c6a8f4ec_13.png)

解读如下：
*   **股票A的夏普比率为1.5**：这意味着每承担1个单位的总风险，可以获得1.5个单位的超额收益。这是三者中最优的选择。
*   **股票B的夏普比率为0.8**：其风险收益的“性价比”低于股票A。
*   **股票C的夏普比率为负**：这表明其收益甚至没有跑赢无风险利率（如银行存款），在承担风险后反而获得了更差的回报。通常应避免此类投资。

![](img/ad535582da8c36018bccf2d2c6a8f4ec_15.png)

**核心原则是：在比较不同投资时，夏普比率越高越好。**
