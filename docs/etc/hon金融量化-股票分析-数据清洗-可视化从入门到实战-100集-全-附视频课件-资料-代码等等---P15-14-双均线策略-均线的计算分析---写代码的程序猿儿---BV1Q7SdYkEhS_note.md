## 数据准备 📊

上一节我们介绍了量化分析的基本流程，本节中我们来看看如何准备用于双均线策略的数据。首先，我们需要获取并处理股票的历史行情数据。

### 读取历史数据

以下是读取本地存储的股票数据文件并做初步处理的步骤。

```python
import pandas as pd

# 从本地CSV文件读取数据
df = pd.read_csv('./茅台.csv')
# 删除无用列（假设第一列是索引列）
df = df.drop(labels=‘Unnamed: 0’, axis=1)
```

### 处理时间序列

![](img/3c74258a5578af554468082d1d3c4c2d_1.png)

数据读取后，我们需要将日期列转换为时间序列格式，并将其设置为数据框的行索引，以便进行时间序列分析。

```python
# 将‘date’列转换为时间序列

df.set_index(‘date’, inplace=True)
```

执行上述步骤后，数据框`df`的行索引就变成了时间戳，方便我们进行基于时间的计算和分析。

## 均线概念与计算 📉

上一节我们准备好了数据，本节中我们来看看什么是均线以及如何计算它。均线是技术分析中的重要工具。

### 什么是均线？

![](img/3c74258a5578af554468082d1d3c4c2d_5.png)

对于每一个交易日，都可以计算出前N天的移动平均值。将这些移动平均值连接起来形成的线，就叫做N日移动平均线，简称均线。
常用的均线有5日、10日、30日、60日、120日和240日等。其中，5日和10日均线被视为短期均线。

均线的计算公式如下：
**MA = (C1 + C2 + C3 + … + CN) / N**
其中，`MA`代表移动平均值，`C1`到`CN`分别代表第1天到第N天的收盘价。

### 计算5日与30日均线

根据公式，我们需要基于股票的收盘价序列来计算移动平均值。在Pandas中，可以使用`rolling`函数配合`mean`方法轻松实现。

以下是计算5日和30日均线的代码：

```python
# 计算5日均线值（MA5）
ma5 = df[‘close’].rolling(5).mean()
# 计算30日均线值（MA30）
ma30 = df[‘close’].rolling(30).mean()
```

![](img/3c74258a5578af554468082d1d3c4c2d_7.png)

**代码解释**：
*   `df[‘close’].rolling(5)`：在收盘价序列上创建一个窗口大小为5的滚动窗口对象。
*   `.mean()`：对这个滚动窗口内的值计算平均值。
*   对于`ma5`，前4个值会是`NaN`（非数字），因为至少需要5个数据点才能计算第一个5日均值。`ma30`同理，前29个值为`NaN`。

计算得到的`ma5`和`ma30`就是由一系列均值点构成的序列，将这些点连接起来，就分别形成了5日均线和30日均线。

## 均线可视化 📈

在计算出均线值之后，我们可以将其可视化，直观地观察两条均线的走势和交叉情况。

虽然Matplotlib模块的详细用法不在本节重点，但我们可以先用它来绘制图形。以下是绘图代码：

```python
import matplotlib.pyplot as plt
# 启用Jupyter Notebook的嵌入式绘图
%matplotlib inline

# 绘制5日均线
plt.plot(ma5[50:80], label=‘MA5’)
# 绘制30日均线
plt.plot(ma30[50:80], label=‘MA30’)
plt.legend() # 显示图例
plt.show()
```

**代码解释**：
*   `plt.plot()`用于绘制折线图。
*   `ma5[50:80]`表示只绘制`ma5`序列中索引50到80的数据点，这是为了在图形局部更清晰地观察均线形态。
*   `label`参数为线条添加标签，`plt.legend()`用于显示图例。

通过图表，我们可以清晰地看到短期均线（MA5）和长期均线（MA30）的波动以及它们之间的交叉关系，这是双均线策略产生交易信号的基础。
