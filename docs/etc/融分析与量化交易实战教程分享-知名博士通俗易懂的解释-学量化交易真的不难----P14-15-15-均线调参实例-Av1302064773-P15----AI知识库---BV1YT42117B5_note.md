1.  **加载并准备数据**：重新加载股票价格数据，并处理缺失值。
    ```python
    # 假设 data 是包含‘Close’列的DataFrame
    data = data[['Close']].dropna()
    ```

![](img/1ce417963191e85995b4d115ca1d59f9_64.png)

2.  **计算指标**：根据当前的 `sma1` 和 `sma2` 参数，计算短期均线、长期均线和收益率。
    ```python
    # 计算收益率
    data['returns'] = np.log(data['Close'] / data['Close'].shift(1))
    # 计算短期均线
    data['SMA1'] = data['Close'].rolling(window=sma1).mean()
    # 计算长期均线
    data['SMA2'] = data['Close'].rolling(window=sma2).mean()
    # 删除因滚动计算产生的缺失值
    data.dropna(inplace=True)
    ```

3.  **生成交易信号**：当短期均线上穿长期均线时，做多（标记为1）；反之，平仓（标记为0）。
    ```python
    data['position'] = np.where(data['SMA1'] > data['SMA2'], 1, 0)
    ```

![](img/1ce417963191e85995b4d115ca1d59f9_66.png)

![](img/1ce417963191e85995b4d115ca1d59f9_68.png)

![](img/1ce417963191e85995b4d115ca1d59f9_70.png)

4.  **计算策略收益**：根据持仓信号计算策略的每日收益，并与基准（简单持有）收益对比。
    ```python
    # 策略收益：持仓 * 收益率
    data['strategy'] = data['position'].shift(1) * data['returns']
    # 删除信号滞后产生的缺失值
    data.dropna(inplace=True)
    # 计算策略相对于基准的超额收益
    outperformance = data['strategy'].sum() - data['returns'].sum()
    ```

    result = {
        'SMA1': sma1,
        'SMA2': sma2,
        'Returns': data['returns'].sum(),
        'Strategy': data['strategy'].sum(),
        'Outperformance': outperformance
    }
