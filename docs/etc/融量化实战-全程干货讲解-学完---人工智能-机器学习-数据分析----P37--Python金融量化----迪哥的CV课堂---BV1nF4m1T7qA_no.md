## 第一步：去极值操作 🧹

![](img/75203756d5827b8190b61c7709f1d745_16.png)

![](img/75203756d5827b8190b61c7709f1d745_18.png)

![](img/75203756d5827b8190b61c7709f1d745_20.png)

上一节我们介绍了如何获取因子数据，本节中我们来看看如何处理数据中的异常值。去极值操作旨在处理数据中的离群点，防止极端值对后续分析产生过大影响。我们将使用“三西格玛”方法来实现。

![](img/75203756d5827b8190b61c7709f1d745_22.png)

![](img/75203756d5827b8190b61c7709f1d745_24.png)

**三西格玛法**的核心思想是：对于正态分布的数据，大约99.7%的数据点会落在均值加减三倍标准差的范围内。我们将此范围之外的值视为异常值，并将其调整到边界上。

![](img/75203756d5827b8190b61c7709f1d745_26.png)

![](img/75203756d5827b8190b61c7709f1d745_28.png)

以下是去极值操作的代码实现：

![](img/75203756d5827b8190b61c7709f1d745_30.png)

![](img/75203756d5827b8190b61c7709f1d745_32.png)

![](img/75203756d5827b8190b61c7709f1d745_34.png)

```python
def filter_three_sigma(series, n=3):
    """
    使用三西格玛法去除极值。
    :param series: 输入的因子数据序列（pandas Series）
    :param n: 西格玛倍数，默认为3
    :return: 处理后的数据序列
    """
    mean = series.mean()  # 计算均值
    std = series.std()    # 计算标准差
    max_range = mean + n * std  # 计算上限
    min_range = mean - n * std  # 计算下限
    # 将超出范围的值调整到边界
    return series.clip(lower=min_range, upper=max_range)
```

![](img/75203756d5827b8190b61c7709f1d745_36.png)

![](img/75203756d5827b8190b61c7709f1d745_38.png)

这个函数首先计算数据的均值和标准差，然后根据指定的倍数（默认为3）确定正常值的上下限。最后，使用`.clip()`方法将所有超出此范围的值“修剪”到边界值，而不是直接删除。

## 第二步：标准化操作 📏

![](img/75203756d5827b8190b61c7709f1d745_50.png)

![](img/75203756d5827b8190b61c7709f1d745_52.png)

![](img/75203756d5827b8190b61c7709f1d745_54.png)

完成去极值后，我们需要对不同因子进行标准化处理。标准化（也称Z-Score标准化）的目的是消除不同因子之间由于量纲和数量级不同带来的影响，使所有因子数据服从均值为0、标准差为1的标准正态分布。

![](img/75203756d5827b8190b61c7709f1d745_56.png)

![](img/75203756d5827b8190b61c7709f1d745_58.png)

标准化操作的公式如下：

![](img/75203756d5827b8190b61c7709f1d745_60.png)

![](img/75203756d5827b8190b61c7709f1d745_62.png)

**`z = (x - μ) / σ`**

![](img/75203756d5827b8190b61c7709f1d745_66.png)

![](img/75203756d5827b8190b61c7709f1d745_68.png)

其中，`x`是原始值，`μ`是均值，`σ`是标准差。

```python
def standardize_series(series):
    """
    对数据进行标准化（Z-Score标准化）。
    :param series: 输入的因子数据序列（pandas Series）
    :return: 标准化后的数据序列
    """
    mean = series.mean()
    std = series.std()
    # 应用标准化公式
    return (series - mean) / std
```

![](img/75203756d5827b8190b61c7709f1d745_82.png)

![](img/75203756d5827b8190b61c7709f1d745_84.png)

该函数计算序列的均值和标准差，然后对每个数据点应用上述公式。经过处理的数据，其分布形态不变，但均值为0，标准差为1，便于不同因子间的比较和合成。

## 第三步：中性化操作 ⚖️

![](img/75203756d5827b8190b61c7709f1d745_92.png)

![](img/75203756d5827b8190b61c7709f1d745_94.png)

最后一步是因子中性化。在量化投资中，许多因子（如市净率PB）可能受到其他风格因子（如市值）的影响。中性化的目的就是通过回归分析，剔除这些已知风格因子的影响，从而得到更“纯净”、更能反映股票自身特性的因子值。

![](img/75203756d5827b8190b61c7709f1d745_96.png)

![](img/75203756d5827b8190b61c7709f1d745_98.png)

我们使用**线性回归**的方法进行中性化。基本思路是：将待处理的因子作为因变量Y，将需要剔除影响的风格因子（如市值）作为自变量X，建立回归模型。然后，用原始因子值减去回归模型预测的值，得到的残差就是中性化后的因子值。

![](img/75203756d5827b8190b61c7709f1d745_100.png)

![](img/75203756d5827b8190b61c7709f1d745_102.png)

以下是中性化操作的代码实现，我们使用`statsmodels`库进行线性回归：

![](img/75203756d5827b8190b61c7709f1d745_104.png)

![](img/75203756d5827b8190b61c7709f1d745_106.png)

```python
import statsmodels.api as sm

![](img/75203756d5827b8190b61c7709f1d745_110.png)

def neutralize_factor(factor_series, style_series):
    """
    对因子进行中性化处理，剔除指定风格因子的影响。
    :param factor_series: 待中性化的因子序列（Y）
    :param style_series: 风格因子序列，如市值（X）
    :return: 中性化后的因子序列（残差）
    """
    # 为风格因子添加常数项（截距）
    X = sm.add_constant(style_series.astype(float))
    Y = factor_series.astype(float)
    
    # 使用普通最小二乘法（OLS）进行线性回归
    model = sm.OLS(Y, X).fit()
    
    # 返回回归的残差，即中性化后的因子值
    return model.resid
```

![](img/75203756d5827b8190b61c7709f1d745_112.png)

![](img/75203756d5827b8190b61c7709f1d745_114.png)

![](img/75203756d5827b8190b61c7709f1d745_116.png)

在这个函数中：
1.  `factor_series`是我们需要提纯的因子（例如市净率）。
2.  `style_series`是我们要剔除影响的风格因子（例如市值）。
3.  我们建立`Y = factor_series`关于`X = style_series`的线性回归模型。
4.  模型拟合后，`model.resid`（残差）即为原始因子值中无法被风格因子解释的部分，也就是中性化后的结果。

## 总结与回顾 🎯

![](img/75203756d5827b8190b61c7709f1d745_128.png)

![](img/75203756d5827b8190b61c7709f1d745_130.png)

本节课中我们一起学习了因子数据预处理的完整流程：

![](img/75203756d5827b8190b61c7709f1d745_132.png)

1.  **去极值**：我们使用**三西格玛法**处理异常值，通过设定均值加减三倍标准差的边界，将极端值调整至边界内，增强了数据的稳健性。
2.  **标准化**：我们采用**Z-Score标准化**，将不同量纲的因子数据转化为均值为0、标准差1的标准分布，为后续的因子比较和加权合成奠定了基础。
3.  **中性化**：我们运用**线性回归**技术，剥离了诸如市值等风格因子对目标因子的影响，从而得到了更纯粹、更能反映股票内在特性的因子值。

![](img/75203756d5827b8190b61c7709f1d745_134.png)

![](img/75203756d5827b8190b61c7709f1d745_136.png)

![](img/75203756d5827b8190b61c7709f1d745_137.png)

这三步预处理操作是构建有效多因子模型的关键准备工作。在实际应用中，通常需要按顺序执行：`去极值 -> 标准化 -> 中性化`。掌握这些方法，你将能够为后续的因子检验、筛选和模型构建准备好高质量、可比较的数据基础。
