# 量化交易全教程：P11：03-3-策略收益效果分析 📈

在本节课中，我们将学习如何计算和比较交易策略的收益。我们将通过一个具体的例子，分析使用“双均线策略”与不采取任何策略（即持有不动）两种情况下，初始投资的最终价值分别是多少。我们会从计算每日收益率开始，逐步构建策略收益，并进行最终的价值还原与比较。

![](img/1ba60da8122d11d14b336ffa68c1883a_1.png)

## 计算每日收益率

![](img/1ba60da8122d11d14b336ffa68c1883a_3.png)

上一节我们介绍了如何通过股价数据计算持仓信号。本节中，我们来看看如何基于股价计算每日的收益率。

收益率反映了资产价格每日的变化率。在量化分析中，我们通常使用对数收益率，因为它具有更好的数学性质（如可加性）。计算对数收益率的公式如下：

**公式：** `log_return = log(price_t / price_{t-1})`

在Python的pandas库中，我们可以通过以下步骤实现：

1.  获取股价序列。
2.  计算当前价格除以前一日价格的结果。
3.  对这个比值取自然对数。

![](img/1ba60da8122d11d14b336ffa68c1883a_5.png)

![](img/1ba60da8122d11d14b336ffa68c1883a_7.png)

以下是计算每日收益率的代码：

```python
# 假设 data[‘Close’] 是包含每日收盘价的序列
data[‘returns’] = np.log(data[‘Close’] / data[‘Close’].shift(1))
```

![](img/1ba60da8122d11d14b336ffa68c1883a_9.png)

这段代码执行后，`data[‘returns’]` 列就存储了每日的对数收益率。`shift(1)` 操作将价格序列向下移动一行，使得当前行的价格能与前一日的价格进行计算。

![](img/1ba60da8122d11d14b336ffa68c1883a_11.png)

## 构建策略收益序列

![](img/1ba60da8122d11d14b336ffa68c1883a_13.png)

在计算出基础收益率后，我们需要根据我们的交易策略来调整这些收益。我们的“双均线策略”会生成一个持仓信号（`position`），这个信号指示我们在某些时间段内应该采取与市场走势相反的操作。

![](img/1ba60da8122d11d14b336ffa68c1883a_15.png)

以下是构建策略收益序列的步骤：

1.  **理解持仓信号**：`position` 列中的值为1或-1。值为1时，表示我们跟随市场原始走势；值为-1时，表示我们采取与市场走势相反的操作（例如，市场下跌时我们视为盈利）。
2.  **应用策略**：策略收益等于持仓信号乘以原始收益率。这样，当 `position` 为-1时，原始收益率的符号会被反转。
3.  **注意时间对齐**：在计算时，需要确保持仓信号与收益率在时间上正确对应。通常，我们使用前一日收盘后生成的信号来指导次日的交易，因此需要对 `position` 列也进行 `shift(1)` 操作。

![](img/1ba60da8122d11d14b336ffa68c1883a_17.png)

以下是计算策略收益的代码：

```python
# 计算策略收益，注意对position进行shift操作以对齐时间
data[‘strategy_returns’] = data[‘position’].shift(1) * data[‘returns’]
```

![](img/1ba60da8122d11d14b336ffa68c1883a_19.png)

## 计算并比较最终收益

![](img/1ba60da8122d11d14b336ffa68c1883a_21.png)

现在我们已经有了原始收益率序列和策略收益率序列。接下来，我们需要将它们累积起来，还原为最终的投资价值，并进行比较。

![](img/1ba60da8122d11d14b336ffa68c1883a_23.png)

以下是计算并比较最终收益的步骤：

![](img/1ba60da8122d11d14b336ffa68c1883a_25.png)

1.  **处理缺失值**：由于使用了 `shift` 操作，数据的前几行会产生缺失值（NaN）。我们需要使用 `dropna()` 方法将其移除。
2.  **累积收益**：将对数收益率进行求和，即可得到整个时间段的总对数收益率。这是因为对数收益率具有可加性：`总收益 = 收益1 + 收益2 + … + 收益N`。
3.  **还原为价值**：将对数总收益通过指数函数 `exp()` 还原为最终的价值乘数。这个乘数表示初始1单位资金最终变成了多少。
4.  **比较结果**：分别计算“持有不动”和“使用策略”两种情景下的最终价值乘数，数值更大者代表策略更有效。

![](img/1ba60da8122d11d14b336ffa68c1883a_27.png)

![](img/1ba60da8122d11d14b336ffa68c1883a_29.png)

以下是实现这一过程的代码：

```python
# 1. 删除含有缺失值的行
data_clean = data[[‘returns‘， ‘strategy_returns‘]].dropna()

# 2. & 3. 分别计算两种情景下的最终价值乘数
final_return = np.exp(data_clean[‘returns‘].sum())
final_strategy = np.exp(data_clean[‘strategy_returns‘].sum())

print(f“持有不动最终价值： {final_return:.2f}“)
print(f“双均线策略最终价值： {final_strategy:.2f}“)
```

![](img/1ba60da8122d11d14b336ffa68c1883a_31.png)

执行这段代码后，我们可以得到两个数字。例如，结果可能显示：
*   持有不动：1元变成了4.00元。
*   双均线策略：1元变成了5.80元。

![](img/1ba60da8122d11d14b336ffa68c1883a_33.png)

![](img/1ba60da8122d11d14b336ffa68c1883a_35.png)

这表明，在我们回测的这段历史数据中，应用双均线策略获得了比简单持有更高的收益。

![](img/1ba60da8122d11d14b336ffa68c1883a_37.png)

## 本节总结

本节课中我们一起学习了量化策略收益分析的核心流程。

我们首先介绍了如何计算每日对数收益率。接着，我们讲解了如何将交易策略生成的持仓信号应用到收益率上，从而构建出策略专属的收益序列。最后，我们通过累积收益和指数还原，计算并比较了“无策略”和“有策略”两种情景下的最终投资价值。

![](img/1ba60da8122d11d14b336ffa68c1883a_39.png)

这个过程是量化回测中评估策略有效性的基础。通过这样的分析，我们可以客观地判断一个策略在历史数据中是否具备盈利潜力。请注意，这只是一个简化的示例，实际策略分析还需要考虑交易成本、滑点等因素。
