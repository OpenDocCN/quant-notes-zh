在本节课中，我们将学习如何使用SPSS软件导入和导出不同类型的数据文件。掌握这些操作是进行数据分析的基础，能够帮助我们高效地从各种来源获取数据，并将分析结果保存为所需的格式。

## 导入外部数据

上一节我们介绍了SPSS的基本界面和变量类型，本节中我们来看看如何将外部数据导入到SPSS中进行分析。SPSS支持多种数据格式，我们将重点学习三种常见类型的导入方法。

### 导入Excel文件

首先，我们学习如何导入最常见的Excel文件。以下是操作步骤：

![](img/2898ce9357d33aa8bfe4a5c181a8dcbc_1.png)

![](img/2898ce9357d33aa8bfe4a5c181a8dcbc_3.png)

1.  在SPSS主界面，点击菜单栏的“文件”，然后选择“打开” -> “数据”。
2.  在弹出的文件选择对话框中，点击“文件类型”下拉菜单，选择“Excel (*.xls, *.xlsx, *.xlsm)”。
3.  找到并选择目标Excel文件，点击“打开”。
4.  在随后出现的“打开Excel数据源”对话框中，需要注意几个关键选项：
    *   **工作表**：选择要导入的特定工作表（如果工作簿包含多个工作表）。
    *   **从第一行数据读取变量名**：如果Excel表的第一行是列标题（字段名），请勾选此选项。
    *   **范围**：可以指定要导入的特定数据区域（例如 A1:D100）。
5.  确认设置无误后，点击“确定”，数据即可成功导入SPSS数据视图。

> **核心概念**：变量名即数据表中的列名，在数据库中称为字段。SPSS会自动识别第一行是否为有效的变量名。

### 导入数据库文件（以MySQL为例）

接下来，我们看看如何连接并导入数据库中的数据。在操作前，需要确保已安装对应数据库的ODBC驱动。

以下是连接MySQL数据库的步骤：

1.  **安装驱动**：从MySQL官网下载并安装对应版本的MySQL ODBC驱动程序。
2.  **配置数据源**：
    *   在SPSS中，点击“文件” -> “导入数据” -> “数据库” -> “新建查询”。
    *   在“数据库向导”中，点击“添加”来配置新的ODBC数据源。
    *   选择已安装的MySQL ODBC驱动，并配置连接信息：
        *   **Data Source Name**：连接名称（可自定义）。
        *   **TCP/IP Server**：数据库服务器地址（本机可填 `127.0.0.1`）。
        *   **User** 和 **Password**：数据库的用户名和密码。
        *   **Database**：选择要连接的具体数据库。
    *   点击“Test”测试连接，成功即可。
3.  **导入数据**：
    *   选择配置好的数据源，点击“下一步”。
    *   在数据库表列表中，选择要导入的数据表。
    *   建议选择数据量较小的表进行测试，以避免SPSS处理过载。
    *   点击“完成”，数据将被导入。

> **核心概念**：此过程本质是在SPSS（客户端）与MySQL数据库（服务端）之间建立连接通道，从而直接读取服务器上的数据。

### 导入文本数据（如CSV、TXT）

最后，我们学习如何导入没有固定列结构的文本数据。SPSS提供了向导来帮助识别和分隔数据。

以下是导入文本数据的步骤：

1.  点击“文件” -> “导入数据” -> “文本数据”。
2.  选择目标文本文件（如 `.csv` 或 `.txt`），点击“打开”，启动文本导入向导。
3.  **向导第一步**：预览数据格式，通常选择“分隔”表示变量由特定符号（如逗号、制表符）分隔。
4.  **向导第二步**：
    *   确认“变量如何排列”为“分隔”。
    *   根据文件是否包含列标题，选择“是”或“否”来回答“文件开头是否包含变量名”。
5.  **向导第三步**：定义数据细节。
    *   “每行表示一个个案”保持默认。
    *   “要导入多少个案？”通常选择“全部”。
6.  **向导第四步**：指定分隔符。
    *   在“变量之间的分隔符是什么？”中，根据文件实际情况勾选，例如“制表符”或“逗号”。
    *   如果文本中有引号等限定符，在“文本限定符”中选择。
7.  **向导第五步**：指定变量格式（可选）。可以在此步为特定列（如日期列）指定数据格式。
8.  **完成导入**：在最后一步，**务必勾选“粘贴此语法”**，然后点击“完成”。此时会生成语法代码，需要在弹出的语法编辑器窗口中点击“运行”按钮执行，数据才会正式导入。
9.  **编码问题**：如果导入后中文显示为乱码（小方块），可在向导第一步的“编码”下拉框中，将编码改为“本地编码”或 `GB2312` 等再尝试。

> **核心概念**：在SPSS中，一行数据称为一个“个案”，一列数据称为一个“变量”。文本导入向导的作用是将无结构的文本数据，解析为SPSS可识别的个案和变量结构。

## 导出SPSS数据与分析结果

学会了导入数据，自然也需要掌握如何将SPSS中的数据和分析结果导出。我们主要学习两种导出类型：导出数据表和导出分析结果报告。

### 导出数据表至Excel

我们可以将SPSS数据视图中的整个或部分数据集导出为Excel文件，操作非常灵活。

1.  在SPSS数据视图窗口，点击“文件” -> “另存为”。
2.  在“另存为”对话框中，从“保存类型”下拉列表中选择“Excel (*.xls, *.xlsx)”。
3.  指定保存的文件名和位置。
4.  点击对话框右侧的“变量…”按钮，可以进行高级设置：
    *   **导出部分变量**：取消勾选不需要导出的变量，即可实现选择性导出。
    *   **保存到现有文件**：可以选择“将工作表追加到现有文件”，将数据作为一张新工作表添加到已有的Excel工作簿中（**操作时需确保目标工作簿已关闭**）。
    *   **保存值和标签**：可以选择导出“变量名”或“变量标签”，对于分类变量（如用1/2代表男/女），可以选择导出“值”或“值标签”，后者会直接导出“男”、“女”这样的文字。
5.  设置完成后，点击“保存”。

### 导出分析结果

对于在SPSS输出查看器中生成的分析报告（表格、图表等），我们也可以将其导出。

1.  在SPSS输出查看器窗口，选中要导出的结果表格或图表。
2.  点击“文件” -> “导出”。
3.  在导出对话框中，选择导出的格式（如Word、Excel、PDF、HTML等）。
4.  指定导出的文件名和位置。
5.  点击“确定”，即可将选中的分析结果导出为独立文件，方便插入报告或演示文稿中。
