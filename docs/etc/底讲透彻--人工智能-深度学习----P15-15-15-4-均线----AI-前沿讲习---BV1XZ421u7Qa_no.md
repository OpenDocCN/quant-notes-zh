## 参数优化的重要性

![](img/651dec9c6e59f482e6205ce92034c36f_12.png)

![](img/651dec9c6e59f482e6205ce92034c36f_14.png)

双均线策略的结果与短期和长期移动平均线的窗口大小密切相关。不同的参数组合可能导致截然不同的收益结果。因此，我们需要系统地测试不同的参数值，以寻找表现更优的组合。

![](img/651dec9c6e59f482e6205ce92034c36f_16.png)

![](img/651dec9c6e59f482e6205ce92034c36f_18.png)

以下是进行参数优化的核心步骤：

![](img/651dec9c6e59f482e6205ce92034c36f_20.png)

![](img/651dec9c6e59f482e6205ce92034c36f_22.png)

![](img/651dec9c6e59f482e6205ce92034c36f_24.png)

1.  **定义参数空间**：确定短期和长期移动平均线窗口的可能取值范围。
2.  **遍历所有组合**：使用工具生成所有可能的参数组合。
3.  **计算策略表现**：对每一组参数，执行完整的双均线策略回测。
4.  **汇总并比较结果**：将所有参数组合及其对应的策略收益汇总，进行比较分析。

## 代码实现：遍历参数组合

首先，我们需要导入必要的工具来生成参数组合。我们将使用`itertools`库中的`product`函数。

![](img/651dec9c6e59f482e6205ce92034c36f_38.png)

![](img/651dec9c6e59f482e6205ce92034c36f_40.png)

```python
from itertools import product
```

![](img/651dec9c6e59f482e6205ce92034c36f_42.png)

![](img/651dec9c6e59f482e6205ce92034c36f_44.png)

![](img/651dec9c6e59f482e6205ce92034c36f_46.png)

`product`函数可以帮助我们生成两个或多个序列的笛卡尔积，即所有可能的组合。例如，给定两个列表 `[1, 2]` 和 `[‘a‘, ‘b‘]`，`product` 会生成 `(1, ‘a‘)`， `(1, ‘b‘)`， `(2, ‘a‘)`， `(2, ‘b‘)`。

![](img/651dec9c6e59f482e6205ce92034c36f_48.png)

接下来，我们定义短期和长期移动平均线的参数空间，并开始遍历。

![](img/651dec9c6e59f482e6205ce92034c36f_50.png)

![](img/651dec9c6e59f482e6205ce92034c36f_52.png)

![](img/651dec9c6e59f482e6205ce92034c36f_54.png)

```python
# 定义参数空间
# 短期均线窗口从20到60，步长为4
short_window_range = range(20, 61, 4)
# 长期均线窗口从180到280，步长为10
long_window_range = range(180, 281, 10)

import pandas as pd
```

## 整合策略回测代码

![](img/651dec9c6e59f482e6205ce92034c36f_66.png)

在循环体内，我们需要为每一组参数 `(sma1, sma2)` 执行一次完整的双均线策略计算。这包括数据准备、计算收益率、计算移动平均线、生成交易信号和计算策略收益。

以下是整合后的代码框架：

![](img/651dec9c6e59f482e6205ce92034c36f_68.png)

![](img/651dec9c6e59f482e6205ce92034c36f_70.png)

```python
# 重新加载股票价格数据（例如苹果股票）
data = pd.read_csv(‘你的股票数据文件路径‘, index_col=‘Date‘, parse_dates=True)
price = data[‘Close‘]  # 假设收盘价列名为‘Close‘

![](img/651dec9c6e59f482e6205ce92034c36f_72.png)

![](img/651dec9c6e59f482e6205ce92034c36f_74.png)

# 1. 处理缺失值
price = price.dropna()

![](img/651dec9c6e59f482e6205ce92034c36f_76.png)

![](img/651dec9c6e59f482e6205ce92034c36f_78.png)

# 2. 计算每日收益率
returns = price.pct_change().dropna()

![](img/651dec9c6e59f482e6205ce92034c36f_80.png)

![](img/651dec9c6e59f482e6205ce92034c36f_82.png)

![](img/651dec9c6e59f482e6205ce92034c36f_84.png)

# 3. 计算移动平均线 (使用当前循环中的参数 sma1 和 sma2)
sma1_series = price.rolling(window=sma1).mean().dropna()
sma2_series = price.rolling(window=sma2).mean().dropna()

![](img/651dec9c6e59f482e6205ce92034c36f_86.png)

![](img/651dec9c6e59f482e6205ce92034c36f_88.png)

# 4. 生成交易信号：当短期均线上穿长期均线时买入(1)，下穿时卖出(-1)
# 为了对齐数据长度，需要对信号序列进行切片
aligned_price = price[sma2_series.index[0]:]  # 从长期均线第一个有效值开始对齐
aligned_returns = returns[aligned_price.index[0]:]
aligned_sma1 = sma1_series[aligned_price.index[0]:]
aligned_sma2 = sma2_series[aligned_price.index[0]:]

![](img/651dec9c6e59f482e6205ce92034c36f_90.png)

![](img/651dec9c6e59f482e6205ce92034c36f_92.png)

# 计算信号
signal = pd.Series(0, index=aligned_sma1.index)
signal[aligned_sma1 > aligned_sma2] = 1  # 短期线上穿，买入信号
signal[aligned_sma1 <= aligned_sma2] = -1 # 短期线下穿，卖出或做空信号
# 实际中可能更复杂，这里做简化。通常使用.shift(1)来避免未来函数。

![](img/651dec9c6e59f482e6205ce92034c36f_94.png)

![](img/651dec9c6e59f482e6205ce92034c36f_96.png)

![](img/651dec9c6e59f482e6205ce92034c36f_98.png)

# 5. 计算策略收益：假设信号是持仓方向（1为持有多头，-1为持有空头，0为空仓）
strategy_returns = signal.shift(1) * aligned_returns  # 使用前一天的信号决定今天交易
strategy_returns = strategy_returns.dropna()

![](img/651dec9c6e59f482e6205ce92034c36f_100.png)

![](img/651dec9c6e59f482e6205ce92034c36f_102.png)

# 6. 计算基准收益（买入并持有）和策略超额收益
benchmark_returns = aligned_returns[strategy_returns.index[0]:]
outperformance = strategy_returns.cumsum().iloc[-1] - benchmark_returns.cumsum().iloc[-1]  # 简化比较总收益

![](img/651dec9c6e59f482e6205ce92034c36f_104.png)

![](img/651dec9c6e59f482e6205ce92034c36f_106.png)

![](img/651dec9c6e59f482e6205ce92034c36f_108.png)

# 7. 将本次参数组合的结果保存到汇总表格中
temp_df = pd.DataFrame({
    ‘SMA1‘: [sma1],
    ‘SMA2‘: [sma2],
    ‘Benchmark_Return‘: [benchmark_returns.cumsum().iloc[-1]],
    ‘Strategy_Return‘: [strategy_returns.cumsum().iloc[-1]],
    ‘Outperformance‘: [outperformance]
})
results = pd.concat([results, temp_df], ignore_index=True)
```

![](img/651dec9c6e59f482e6205ce92034c36f_110.png)

![](img/651dec9c6e59f482e6205ce92034c36f_112.png)

## 结果分析与解读

代码运行完毕后，`results` DataFrame 中将包含所有参数组合的回测结果。我们可以通过排序来快速找到表现最好的参数组合。

![](img/651dec9c6e59f482e6205ce92034c36f_114.png)

![](img/651dec9c6e59f482e6205ce92034c36f_116.png)

```python
# 按超额收益降序排列，查看表现最好的参数
best_results = results.sort_values(by=‘Outperformance‘, ascending=False)
print(best_results.head(10))
```

![](img/651dec9c6e59f482e6205ce92034c36f_118.png)

通过分析结果表格，你可以观察到：
*   某些参数组合的策略收益远超基准（买入持有）。
*   另一些参数组合可能表现不佳，甚至不如基准。
*   这直观地展示了参数选择对量化策略效果的巨大影响。
