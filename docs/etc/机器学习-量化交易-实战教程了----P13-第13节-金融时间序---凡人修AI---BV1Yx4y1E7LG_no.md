# 量化交易与机器学习：13：金融时间序列-II-状态空间模型 📈

在本节课中，我们将学习状态空间模型及其核心算法——卡尔曼滤波。我们将从回顾平稳时间序列模型开始，逐步引入更灵活的状态空间模型，并探讨其如何将多种经典模型统一起来，最后简要介绍卡尔曼滤波的原理。

## 回顾：平稳时间序列模型

上一节我们介绍了最常见的平稳时间序列模型——ARMA(p, q)模型。理解这个模型有两个关键点。

第一，在该过程中，我们能够观测到的值恰好是时间序列本身，即ARMA过程的输出值。而驱动该过程的噪声项是不可观测的。

第二，该过程之所以平稳，一个很重要的原因是假设了所有系数（从 `φ1` 到 `φp`， `θ1` 到 `θq`）本身都不随时间变化。

这两点共同保证了ARMA过程是一个平稳的时间序列。

## 引入状态空间模型

然而，在许多模型的更新、动态对冲以及信号处理场景中，我们通常希望线性模型或动态时间序列模型的系数能够随时间变化。这样，模型就能更好地匹配当前的市场信号。为此，我们需要引入一种称为**状态空间模型**的框架。

状态空间模型包含两个主要的更新过程。第一个是**状态** `X_t`，这是我们真正关心的、驱动系统的内在状态。第二个是**输出** `Y_t`，这是一个可观测的外部表现。我们通常更关心其内在状态。

![](img/9c90575ec79650b5cb85d06d1298aaa7_1.png)

对于一个线性状态空间模型，其更新表达式由以下方程组描述：

1.  **状态更新方程**：描述状态 `X_t` 如何随时间演变。
    `X_t = A * X_{t-1} + B * Z_t`
    其中，`Z_t` 是驱动状态更新的噪声。

2.  **观测方程**：描述可观测的输出 `Y_t` 如何由当前状态 `X_t` 生成。
    `Y_t = C * X_t + D * V_t`
    其中，`V_t` 是观测噪声。

在这两个方程中，状态更新噪声 `Z_t` 和观测噪声 `V_t` 通常被假设为相互独立的白噪声序列（例如，均值为0的正态分布）。这就是一个带有观测的动态更新状态空间模型，其中所有关系都是线性的。

## 状态空间模型的普适性

状态空间模型是比我们之前介绍过的均值回复、异方差时间序列等模型更通用的表达式。接下来，我们看看如何将几种经典模型表示为状态空间模型。

### AR过程的状态空间表示

首先，让我们看一个AR(2)过程的例子。它满足：
`X_t = -φ1 * X_{t-1} - φ2 * X_{t-2} + Z_t`

我们定义状态向量 `X_t` 为 `[X_t, X_{t-1}]^T`。那么，状态空间表示如下：

![](img/9c90575ec79650b5cb85d06d1298aaa7_3.png)

*   **状态更新方程**：
    `[X_t; X_{t-1}] = [[-φ1, -φ2]; [1, 0]] * [X_{t-1}; X_{t-2}] + [1; 0] * Z_t`
    矩阵A的第一行体现了AR(2)的关系，第二行表示 `X_{t-1}` 简单地等于上一时刻的 `X_{t-1}`。

*   **观测方程**：
    对于AR过程，观测值就是状态本身，因此 `Y_t = X_t`。
    可以写为：`Y_t = [1, 0] * [X_t; X_{t-1}] + 0 * V_t`

对于一般的AR(p)过程，可以推广此方法。状态向量包含最近的p个观测值，状态转移矩阵A的第一行包含AR系数，下方是一个次对角线为1的矩阵。观测矩阵C始终为 `[1, 0, ..., 0]`，表示我们观测到的是当前状态向量的第一个元素。

### MA与ARMA过程的状态空间表示

同理，MA和ARMA过程也可以写成状态空间模型的形式。我们以MA(1)过程为例：
`Y_t = θ * Z_{t-1} + Z_t`

此时，状态向量应包含噪声项。我们定义 `X_t = [Z_t, Z_{t-1}]^T`。

*   **状态更新方程**：
    `[Z_t; Z_{t-1}] = [[0, 0]; [1, 0]] * [Z_{t-1}; Z_{t-2}] + [1; 0] * W_t`
    （这里 `W_t` 是一个新的驱动噪声，在标准表示中可能需要调整）

*   **观测方程**：
    `Y_t = [1, θ] * [Z_t; Z_{t-1}] + 0 * V_t`

对于ARMA(p, q)过程，状态向量需要同时包含最近的p个观测值和q个噪声项。状态转移矩阵A和观测矩阵C的结构会变得复杂，但核心思想不变：状态方程体现AR部分和噪声的递推关系，观测方程体现MA部分。

![](img/9c90575ec79650b5cb85d06d1298aaa7_5.png)

![](img/9c90575ec79650b5cb85d06d1298aaa7_7.png)

由此可见，状态空间模型本身更具一般性，能够对应并转换多种平稳时间序列模型。

## 状态空间模型的其他应用

除了表示经典时间序列模型，状态空间模型在其他领域也有广泛应用。

### 动态线性回归

在线性回归中，我们通常假设系数是固定的。但状态空间模型允许我们动态更新这些系数，这在金融领域非常有用，因为市场关系可能随时间变化。

*   **状态**：我们将回归系数 `β_t` 定义为隐藏状态 `X_t`。
*   **状态更新**：对于系数不随时间变化的简单情况，可写为 `β_t = β_{t-1}`。更复杂的模型可以加入学习率或演化噪声：`β_t = A * β_{t-1} + 噪声`。
*   **观测方程**：可观测的因变量 `Y_t` 由设计矩阵（自变量）与当前系数决定：`Y_t = (设计矩阵)_t * β_t + 观测噪声`。

**注意**：此处的状态 `X_t`（即系数 `β_t`）与传统线性回归中的自变量 `X` 角色相反，切勿混淆。

### 时变自回归模型

有时我们希望AR模型的系数能随时间变化，以更好地跟踪时间序列的趋势。此时，我们可以将AR系数本身作为状态进行更新。

![](img/9c90575ec79650b5cb85d06d1298aaa7_9.png)

*   **状态**：AR模型的系数向量 `φ_t`。
*   **状态更新**：`φ_t = A * φ_{t-1} + 状态噪声`。矩阵A可以包含学习率参数。
*   **观测方程**：`Y_t = (滞后观测值向量)_t * φ_t + 观测噪声`。这体现了AR模型本身的生成过程。

### 随机波动率模型

GARCH类模型也可以表示为状态空间形式。在GARCH(1,1)模型中，我们通常对去均值后的收益率平方建模，它等于波动率的平方加上一个扰动。

*   **状态**：潜在的波动率 `σ_t^2`。
*   **状态更新**：描述波动率如何根据上一期波动率和上一期收益率平方演化（即GARCH方程）。
*   **观测方程**：观测到的收益率平方 `r_t^2` 等于当前波动率加上一个观测误差。

这为使用卡尔曼滤波等算法在线估计时变的波动率提供了框架。

## 卡尔曼滤波简介

在介绍了状态空间模型的各种形式后，我们自然面临一个问题：如何根据观测数据 `Y_1, Y_2, ..., Y_t` 来估计不可观测的状态 `X_t`？这就是**卡尔曼滤波**要解决的核心问题。

卡尔曼滤波提供了一套最优（在最小均方误差意义下）的递推算法，用于在线估计状态空间模型中的隐藏状态。其推导过程涉及贝叶斯推断和线性高斯系统的性质，虽然有些复杂，但对于理解算法本质和后续变体开发非常有帮助。

![](img/9c90575ec79650b5cb85d06d1298aaa7_13.png)

**核心步骤**（简述）：
卡尔曼滤波在每个时间点t进行两个步骤：
1.  **预测**：基于 `t-1` 时刻的状态估计，预测 `t` 时刻的状态和观测值。
2.  **更新**：获得 `t` 时刻的实际观测值 `Y_t` 后，结合预测值，更新对 `t` 时刻状态的估计。

该算法高效且只需保留前一时刻的信息，非常适合实时处理金融时间序列数据。
