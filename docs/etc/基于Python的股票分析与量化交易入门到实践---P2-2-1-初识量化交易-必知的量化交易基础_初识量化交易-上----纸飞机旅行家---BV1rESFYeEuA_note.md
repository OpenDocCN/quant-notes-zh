## 🧠 什么是量化思想？
上一节我们介绍了课程的整体安排，本节中我们正式进入量化交易的学习。首先，我们来理解最核心的“量化思想”。

![](img/af0779532ec4e8c8a86f7d744164a8f4_5.png)

量化思想的核心在于将问题转化为数学模型，并通过计算寻找最优解。下面我们通过一个博彩公司的例子来具体说明。

假设一场足球赛在巴西队和阿根廷队之间进行，博彩公司需要为“巴西胜”、“平局”、“阿根廷胜”三个结果设定赔率。已知下注资金分布如下：
*   买巴西胜：4亿元
*   买平局：3亿元
*   买阿根廷胜：3亿元

博彩公司的目标是，无论比赛结果如何，都能稳定获利（例如1亿元）。如何设计赔率？

**解决方案如下：**
首先，总下注资金为 `4 + 3 + 3 = 10` 亿元。目标利润为1亿元，因此无论哪种结果发生，公司需要支付的总金额应为 `10 - 1 = 9` 亿元。

基于此，我们可以计算出保证盈利的赔率：
*   **巴西胜的赔率**：`总支付额 / 巴西胜下注额 = 9 / 4 = 2.25`
*   **平局的赔率**：`9 / 3 = 3.00`
*   **阿根廷胜的赔率**：`9 / 3 = 3.00`

![](img/af0779532ec4e8c8a86f7d744164a8f4_7.png)

只要按此赔率设置，无论最终赛果如何，博彩公司都能确保赚取1亿元利润。这个通过已知数据建立模型并计算确定方案的过程，就体现了量化思想。

## 🤖 什么是量化交易？
上面我们通过例子介绍了量化思想，现在来看看它在金融交易领域的应用——量化交易。

根据定义，**量化交易**是指以**先进的数学模型**替代人的主观判断，利用**计算机技术**从海量历史数据中筛选能带来超额收益的“大概率”策略，以**减少投资者情绪波动**，避免非理性决策。

量化交易主要涉及三个领域的知识：
以下是量化交易涉及的三大知识领域：
1.  **金融学**：提供理论基础和市场规则。包括：
    *   **金融工程**：设计并实现构成交易策略的数学模型。
    *   **金融衍生品**：理解股票、债券、期货等交易标的。
    *   **会计学**：核算收益、评估模型绩效并进行比较。
2.  **数学**：提供建模和分析工具。核心包括：
    *   **概率论与统计学**：评估策略的“大概率”胜算并进行统计验证。
    *   **博弈论**：分析金融市场中多方参与的博弈行为。
3.  **计算机技术**：实现策略的工具。主要包括：
    *   **编程**：撰写策略代码。
    *   **机器学习/人工智能**：开发更复杂的预测模型。
    *   **大数据技术**：处理高维、海量的金融数据。

这三个领域共同支撑了量化交易的实践。

## 📜 量化交易的发展历史
了解了量化交易的定义后，我们来看看它的发展历程。首先从国外开始。

![](img/af0779532ec4e8c8a86f7d744164a8f4_9.png)

以下是国外量化交易发展的关键里程碑：
*   **1952年**：哈里·马科维茨提出**现代投资组合理论**，奠定了理论基础。
*   **1964年**：威廉·夏普等人提出**资本资产定价模型**，为资产定价提供了核心工具。
*   **1982年**：“量化交易之父”詹姆斯·西蒙斯创立**文艺复兴科技公司**，其大奖章基金在长达40年的时间里取得了约**35%** 的年化收益率，展现了量化交易的巨大潜力。
*   **2008年**：全球金融危机使量化交易规模从约2500亿美元锐减至500亿美元，遭遇重大挫折。
*   **2011年至今**：行业复苏并蓬勃发展。2012年，高盛公司将大量传统交易员岗位替换为量化工程师。目前，量化投资在美国市场的占比已约**50%**，规模达数万亿美元。

![](img/af0779532ec4e8c8a86f7d744164a8f4_11.png)

上一页我们介绍了国外的量化交易发展历史，现在来接着看一下国内的情况。

![](img/af0779532ec4e8c8a86f7d744164a8f4_13.png)

我国量化交易起步较晚，但发展迅速。
以下是国内量化交易发展的关键节点：
*   **2004年**：光大保德信发行国内首只公募量化基金。
*   **2010年4月16日**：**沪深300股指期货**上市，标志着中国资本市场具备了做空和杠杆工具，量化交易有了发展的土壤。
*   **2015年**：**中证500股指期货**上市，量化基金得以进行更灵活的对冲和高频交易，进入启蒙期。
*   **2019年至今**：证监会放开公募基金转融通业务（融资融券），叠加人工智能技术的应用，量化交易进入飞速发展期。截至2021年，国内量化公募基金规模超2000亿元，私募量化基金规模估计超1.5万亿元。

## ⚖️ 量化交易 vs. 主观交易
在量化交易领域有一个生动的比喻：**量化交易像西医，主观交易像中医**。中医依靠个人经验的“望闻问切”，而西医依靠仪器检测和数据分析。

具体来说，两者主要有以下六个方面的区别：
以下是量化交易与主观交易的核心区别：
1.  **纪律性**：量化交易由模型和程序执行，完全**规避情绪干扰**；主观交易极易受交易员贪婪、恐惧等情绪影响。
2.  **精力**：量化交易可**7x24小时**不间断监控市场；主观交易依赖人工盯盘，受体力与精力限制。
3.  **效率**：量化交易可达**毫秒/秒级**的决策与执行速度；主观交易从决策到手动下单至少需要数十秒。
4.  **分析维度**：量化交易可处理**多维度、跨品种、全历史**的海量数据；主观交易主要依赖个人经验，分析范围有限。
5.  **可控性**：量化交易的风险和收益分布基于概率，**可回溯、可分析**，相对可控；主观交易的绩效波动大，可控性差。
6.  **可复制性**：量化交易策略以代码为载体，**易于复制和扩展**；主观交易能力依附于个人，难以复制。

## 🔄 量化交易的主要流程
理解了量化交易的特点后，我们来看看实施一个量化策略需要经历哪些步骤。其核心流程与大数据项目类似，是一个不断迭代优化的过程。

一个完整的量化交易流程通常包含以下环节：
以下是量化交易策略从研发到部署的核心步骤：
1.  **数据获取**：收集行情数据、历史数据、公司基本面信息等。
2.  **数据清洗**：对原始数据进行格式化、处理缺失值、去除异常值等，使其适用于模型。
3.  **策略编写**：将交易思路转化为具体的**计算机程序（代码）**。这是策略的核心。
4.  **策略回测**：在**历史数据**上运行策略，检验其盈利能力与风险。这是评估策略科学性的关键。
5.  **策略优化**：根据回测结果调整策略参数或逻辑。**回测与优化是一个循环迭代的过程**。
6.  **模拟交易**：在**当前实时市场环境**中，用虚拟资金运行策略，观察其在未来一段时间（通常为半年到一年）的表现，以检验策略在实盘中的适应性和防止“过拟合”。
7.  **实盘交易**：投入真实资金运行策略。通常需要经历**3-5年**的完整市场周期验证，才能证明策略的长期有效性。

整个流程从“数据获取”到“实盘交易”并非单向，在“模拟交易”阶段可能需返回“优化”环节，最终才能交付稳定的实盘策略。
