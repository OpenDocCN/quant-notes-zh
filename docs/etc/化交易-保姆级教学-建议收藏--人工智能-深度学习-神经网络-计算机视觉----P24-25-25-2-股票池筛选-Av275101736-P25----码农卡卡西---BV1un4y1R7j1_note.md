## 数据预处理与指标查询
设定好股票池后，我们需要在每日交易开始前进行数据预处理，即获取并筛选股票。这通常在 `before_trading` 函数中完成。

量化交易的核心是数据挖掘，平台提供了丰富的财务和行情指标供我们查询。所有可用的指标都在官方帮助文档中详细列出。

以下是查询财务数据的基本步骤：
1.  使用 `query` 函数构建查询。
2.  在查询中指定需要获取的指标字段。
3.  使用 `filter` 条件筛选目标股票池。
4.  使用 `order_by` 对结果进行排序。
5.  使用 `limit` 限制返回的股票数量。

## 构建筛选逻辑
现在，我们开始编写具体的筛选代码。我们的目标是：每日从沪深300成分股中，查询其营业收入，并选出该指标排名前十的股票。

首先，我们需要从帮助文档中找到“营业收入”对应的字段名。假设我们找到的字段是 `income`。

![](img/d0dd20fb3021a394e9f46c2da83269f7_15.png)

![](img/d0dd20fb3021a394e9f46c2da83269f7_17.png)

```python
def before_trading(context):
    # 构建查询
    q = query(
        # 查询营业收入指标
        income
    ).filter(
        # 筛选条件：股票代码在沪深300成分股中
        code.in_(get_index_stocks('沪深300'))
    ).order_by(
        # 按营业收入降序排列（从高到低）
        income.desc()
    ).limit(
        # 限制返回前10条结果
        10
    )

    # 执行查询，结果是一个DataFrame
    df = get_fundamentals(q)
    # 将筛选结果存入context，供后续函数使用
    context.stock_list = df['code'].tolist()
    # 打印查看结果
    print(df)
```

**代码解析**：
*   `query(income)`：声明我们要查询的指标是营业收入。
*   `.filter(code.in_(get_index_stocks(‘沪深300’)))`：确保我们只查询属于沪深300指数成分股的股票。`get_index_stocks` 是一个函数，用于获取指定指数的成分股列表。
*   `.order_by(income.desc())`：将查询结果按照营业收入从高到低（降序）排列。
*   `.limit(10)`：只取排名前10的股票。
*   `get_fundamentals(q)`：执行查询，返回一个包含股票代码和营业收入数据的DataFrame。

## 注意事项与调试
在初次运行策略时，如果发现查询结果为空，需要检查股票池的指定方式是否正确。确保使用了正确的函数（如 `get_index_stocks`）来获取指数成分股，而不是直接传入指数名称。
