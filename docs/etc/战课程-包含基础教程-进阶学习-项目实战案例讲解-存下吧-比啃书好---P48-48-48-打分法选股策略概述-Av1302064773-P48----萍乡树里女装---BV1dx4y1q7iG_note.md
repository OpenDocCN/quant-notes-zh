# 量化交易教程：P48：因子打分法选股策略概述 📊

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分法。我们将了解其核心思想、实施步骤以及背后的逻辑，为后续的实战编写打下基础。

## 策略核心思想

上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看如何综合多个因子进行更全面的评估。

因子打分选股策略的核心思想是：**综合多个因子对股票进行评分，根据总分排名来筛选股票**。这类似于学生的总成绩由各科成绩加总而成，我们通过计算每只股票在所有选定因子上的综合得分，并选取得分最高的股票构建投资组合。

## 策略流程详解

以下是因子打分法选股的基本步骤：

1.  **选择因子**：确定用于评价股票的多个因子（如估值、成长、质量、预期等维度下的具体指标）。
2.  **确定因子方向**：明确每个因子与预期收益率的相关性是正向（因子值越大越好）还是负向（因子值越小越好）。这是策略的先验知识。
3.  **数据标准化**：由于不同因子的量纲和范围不同，需要将它们标准化到同一尺度（如Z-Score），以便进行公平的加权和比较。公式如下：
    ```
    Z-Score = (原始值 - 该因子所有股票的平均值) / 该因子所有股票的标准差
    ```
4.  **因子打分**：根据标准化后的因子值进行打分。对于正向因子，值越大得分越高；对于负向因子，值越小得分越高。
5.  **计算综合得分**：将股票在各个因子上的得分进行加权求和，得到该股票的综合总分。公式可表示为：
    ```
    股票综合得分 = Σ(因子_i的权重 * 因子_i的得分)
    ```
    在基础版本中，通常假设各因子权重相等。
6.  **排序与选股**：对所有股票的综合得分进行排序，选取排名靠前（例如前10或20名）的股票作为当期投资组合。
7.  **定期调仓**：策略通常按月或按周定期执行上述步骤，卖出不在新名单中的股票，买入新入选的股票，完成调仓。

## 关键前提与已知条件

![](img/53f938b6b004150ab0cd87e3fd34ee11_1.png)

实施该策略有一个关键前提：**必须预先知道每个因子与未来收益率的关系方向（正相关或负相关）**。

这个信息可以通过两种方式获得：
*   **研究经验**：参考券商的研究报告或公认的金融学常识。例如，通常认为市盈率（PE）是负向因子（越低越好），而净资产收益率（ROE）是正向因子（越高越好）。
*   **历史分析**：像我们之前课程中那样，通过计算因子的IC值（信息系数）等指标，利用历史数据验证其有效性。

在后续的实战中，我们将选择一组已知方向的因子进行演示。

## 策略优势与意义

这种策略的优势在于其**综合性与直观性**。它避免了依赖单一因子可能带来的片面性，通过多维度评估提高了选股逻辑的稳健性。同时，“打分”和“排名”的概念非常直观，易于理解和实施。
