## 工具包简介与获取

上一节我们介绍了因子分析的基本概念，本节中我们来看看能帮助我们高效完成这项工作的工具。

![](img/f031ff46fcf6f302967bb793420a46d8_5.png)

![](img/f031ff46fcf6f302967bb793420a46d8_7.png)

![](img/f031ff46fcf6f302967bb793420a46d8_8.png)

Alphalens是一个专门用于金融因子分析的Python工具包。它封装了常见的因子评价指标计算和图表绘制功能。

![](img/f031ff46fcf6f302967bb793420a46d8_10.png)

以下是获取和了解Alphalens的步骤：
*   **安装**：通过pip命令即可安装。在你的命令行或Anaconda Prompt中输入 `pip install alphalens`。
*   **GitHub仓库**：你可以访问其GitHub页面查看源代码和基础信息。
*   **官方文档与示例**：官方提供了详细的使用文档和示例代码（Sample），这是学习其核心功能的最佳途径。

![](img/f031ff46fcf6f302967bb793420a46d8_12.png)

![](img/f031ff46fcf6f302967bb793420a46d8_14.png)

> 提示：在本课程后续的实战环节，我们将使用一个已经预装了Alphalens的量化平台，因此你无需在本地手动安装。如果你想在个人电脑上练习，则可以按上述步骤安装。

## 量化平台的研究环境

![](img/f031ff46fcf6f302967bb793420a46d8_16.png)

![](img/f031ff46fcf6f302967bb793420a46d8_18.png)

由于直接获取和分析金融数据较为复杂，接下来我们将在专业的量化平台中编写代码。

之前我们学习了在平台中创建并回测策略。今天我们不进行回测，而是专注于数据分析。因此，我们需要使用平台的“投资研究”功能。

以下是进入研究环境的操作流程：
1.  登录量化平台。
2.  在左侧功能栏中找到并点击“投资研究”。
3.  界面会加载一个类似于Jupyter Notebook的在线编程环境。
4.  新建一个Python 3笔记文件，我们可以将其命名为“因子分析”。

![](img/f031ff46fcf6f302967bb793420a46d8_20.png)

![](img/f031ff46fcf6f302967bb793420a46d8_22.png)

这个环境运行在平台的服务器上，可以方便地调用平台提供的数据接口和已安装好的工具包（如Alphalens）。后续所有实战代码都将在该环境中编写和执行。

![](img/f031ff46fcf6f302967bb793420a46d8_24.png)

本节课中我们一起学习了Alphalens工具包的作用与获取方式，并熟悉了将在其中进行代码编写的量化平台研究环境。接下来，我们就可以利用这个强大的工具来开展实际的因子分析工作了。
