# Python金融分析与量化交易实战教程：P43：1-打分法选股策略概述

## 概述
在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分选股法。我们将了解该策略的核心思想、实施步骤以及背后的逻辑，为后续的实战编程打下基础。

## 因子打分选股法核心思想
上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看如何综合多个因子进行更全面的选股。

因子打分选股法的核心思想是：**为每只股票的多个因子分别打分，然后计算一个综合总分，最后根据总分排名来筛选股票**。这类似于学生考试，总成绩由各科成绩加总而成，总分高的学生排名靠前。

假设我们有300只股票（`stock_1`, `stock_2`, ..., `stock_300`）和四个因子（`factor_A`, `factor_B`, `factor_C`, `factor_D`）。我们的目标是每月从这300只股票中选出综合表现最好的前10只进行调仓。

## 策略实施的前提条件
在开始打分之前，我们必须明确一个关键的先验知识：**每个因子与股票未来收益率的相关性是正还是负**。

![](img/997007154738568e0cd8d0d80b003afa_1.png)

*   **正相关因子**：因子值越大，预期收益率越高。例如，每股收益（EPS）、净资产收益率（ROE）通常被认为是“越大越好”的因子。
*   **负相关因子**：因子值越小，预期收益率越高。例如，市盈率（PE）、市净率（PB）在某些情况下被认为是“越小越好”的因子。

这个判断标准可以通过查阅券商研究报告获得，也可以通过历史数据进行因子分析（如计算IC值）来验证。在后续的实战中，我们将选择一组已知特性的因子进行演示。

## 策略执行步骤
以下是实施因子打分选股策略的关键步骤：

1.  **数据准备**：获取目标股票池（如沪深300成分股）在特定时间点的各项因子数据。
2.  **因子方向判断**：根据先验知识，确定每个因子是“正向因子”（越大越好）还是“负向因子”（越小越好）。
3.  **单因子打分**：对每个因子，在所有股票间进行排序并分配分数。对于正向因子，因子值最大的股票得分最高；对于负向因子，因子值最小的股票得分最高。常用的打分方法是**分位数排名**或**标准化后打分**。
    *   **公式示例（分位数排名）**：`因子得分 = (股票在该因子上的排名 / 股票总数) * 100`
4.  **计算综合得分**：将每只股票在所有因子上的得分进行加权求和，得到该股票的综合总分。
    *   **公式**：`综合总分 = w_A * 得分_A + w_B * 得分_B + w_C * 得分_C + w_D * 得分_D`
    *   其中 `w_A, w_B, ...` 代表各因子的权重，在基础策略中常采用等权重，即 `w_A = w_B = w_C = w_D = 0.25`。
5.  **排序与选股**：根据计算出的综合总分对所有股票进行降序排序，选取排名前N（如10或20）的股票作为当期投资组合。
6.  **定期调仓**：在设定的调仓周期（如每月初）重复上述步骤，更新投资组合。

## 总结
本节课中我们一起学习了因子打分选股策略。该策略通过综合评估多个因子对股票进行排序，比单一因子选股更为全面。其核心流程是：**判断因子方向 -> 对每个因子单独打分 -> 加权计算综合分 -> 根据总分排名选股**。在下一节的实战演示中，我们将在量化平台中具体实现这一策略。
