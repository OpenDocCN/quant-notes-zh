在本节课中，我们将要学习一种名为3-Sigma（三西格玛）的去极值方法。该方法基于统计学中的正态分布原理，通过计算数据的均值和标准差来识别并处理异常值。

## 3-Sigma方法原理

上一节我们介绍了去极值的概念，本节中我们来看看基于统计分布的3-Sigma方法。其核心思想源于正态分布（也称高斯分布）。

正态分布描述了许多自然现象的分布规律。我们可以将其想象为一种“正常”的分布模式。例如，在银行贷款场景中，银行批准给你的贷款额度在期望值（均值）附近有小幅波动是正常的，但如果波动过大（例如多贷或少贷10万元），则可能被视为异常情况。

![](img/06ea4c1df741081848dd458ba6a20ad4_1.png)

在正态分布曲线中，数据点越靠近均值，其出现的概率越高；越远离均值，其出现的概率则越低。我们的目标是识别并处理那些远离均值、概率极低的异常点（即极值）。

**3-Sigma** 正是基于这一原理。它通过计算均值（μ）和标准差（σ），设定一个以均值为中心、上下浮动若干倍标准差的区间。落在该区间内的数据被认为是“正常”的，而落在区间外的则被视为“异常”或“极值”。

具体来说，对于一个服从正态分布的数据集：
*   **μ ± 1σ** 的区间大约包含 **68.3%** 的数据。
*   **μ ± 2σ** 的区间大约包含 **95.4%** 的数据。
*   **μ ± 3σ** 的区间大约包含 **99.7%** 的数据。

因此，3-Sigma方法意味着我们将 **μ - 3σ** 设为下限，**μ + 3σ** 设为上限。理论上，99.7%的数据会落在这个范围内，剩下的0.3%则作为极值进行处理（不是删除，而是将其规范到边界值上）。

## 代码实现

![](img/06ea4c1df741081848dd458ba6a20ad4_5.png)

以下是3-Sigma去极值方法的代码实现步骤。我们将定义一个函数，它接收一列数据和倍数N（例如N=3代表3-Sigma），并返回处理该列数据的下限和上限。

![](img/06ea4c1df741081848dd458ba6a20ad4_7.png)

首先，我们需要计算数据的两个核心统计量：均值（mean）和标准差（std）。

![](img/06ea4c1df741081848dd458ba6a20ad4_9.png)

```python
# 计算均值和标准差
mean = data.mean()
std = data.std()
```

![](img/06ea4c1df741081848dd458ba6a20ad4_11.png)

![](img/06ea4c1df741081848dd458ba6a20ad4_13.png)

接着，根据均值和标准差计算上下限。公式如下：
*   **上限（upper_limit） = mean + N * std**
*   **下限（lower_limit） = mean - N * std**

以下是完整的函数定义：

![](img/06ea4c1df741081848dd458ba6a20ad4_15.png)

![](img/06ea4c1df741081848dd458ba6a20ad4_17.png)

```python
def sigma_3(data, n=3):
    """
    3-Sigma去极值方法
    :param data: 输入的数据序列（Pandas Series）
    :param n: 西格玛倍数，默认为3
    :return: 处理后的下限和上限
    """
    mean = data.mean()  # 计算均值
    std = data.std()    # 计算标准差
    
    upper_limit = mean + n * std  # 计算上限
    lower_limit = mean - n * std  # 计算下限
    
    return lower_limit, upper_limit
```

## 方法应用与比较

![](img/06ea4c1df741081848dd458ba6a20ad4_23.png)

现在，我们可以使用这个函数。例如，设定 `n=1`，我们将得到包含大约68.3%数据的区间边界。

![](img/06ea4c1df741081848dd458ba6a20ad4_25.png)

![](img/06ea4c1df741081848dd458ba6a20ad4_27.png)

```python
# 应用函数，假设df[‘column_name’]是我们的数据列
lower, upper = sigma_3(df['column_name'], n=1)
print(f"下限: {lower}, 上限: {upper}")
```

![](img/06ea4c1df741081848dd458ba6a20ad4_29.png)

![](img/06ea4c1df741081848dd458ba6a20ad4_31.png)

在实际的股票数据回测图中，以此方法计算出的边界线（通常用蓝色线表示）会标示出正常值的范围。不同的去极值方法（如分位数法、中位数绝对偏差法）得出的边界位置可能略有差异。

![](img/06ea4c1df741081848dd458ba6a20ad4_33.png)

以下是几种常见去极值方法的简要对比：
*   **分位数法**：直接根据数据的百分比位置设定边界，不假设数据分布。
*   **中位数绝对偏差法**：基于中位数和绝对偏差，对异常值不敏感。
*   **3-Sigma法**：假设数据服从正态分布，利用均值和标准差设定边界。

![](img/06ea4c1df741081848dd458ba6a20ad4_37.png)

具体选择哪种方法，需根据实际任务的数据特性和需求来决定。

需要再次强调的是，去极值（或称“缩尾”）处理的目的并非简单丢弃异常数据，而是定义一个合理的**下界（lower_limit）**和**上界（upper_limit）**，将所有数据规范到这个范围内，从而减少极端值对后续分析（如计算相关性、构建模型）的负面影响。这是数据预处理“三步走”中关键的第一步。
