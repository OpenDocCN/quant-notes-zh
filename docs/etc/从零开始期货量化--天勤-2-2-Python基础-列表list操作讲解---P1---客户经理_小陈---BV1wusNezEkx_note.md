## 什么是列表？

上一节我们介绍了Python的变量和基本数据类型，本节中我们来看看**列表**。

列表可以理解为一个“抽屉”或“容器”，用于存放一系列有序的数据。它使用方括号 `[]` 定义，其中的元素用逗号分隔。列表的强大之处在于它可以容纳**任何类型**的数据，例如整数、浮点数、字符串，甚至其他列表、字典等。

**公式/代码表示：**
```python
# 一个包含多种数据类型的列表示例
my_list = [1, 3.14, ‘hello‘, [‘a‘, ‘b‘], {‘key‘: ‘value‘}]
```

## 如何创建列表？

创建列表主要有两种方式。

以下是两种创建空列表的方法：
1.  使用 `list()` 函数：`L1 = list()`
2.  使用方括号 `[]`：`L2 = []`

这两种方式都会创建一个空的列表容器，等待我们放入数据。我们也可以直接在方括号内放入初始元素来创建列表。

## 列表的索引与切片

在操作列表元素之前，必须理解列表的**索引**。列表的索引从 **0** 开始计数，而不是1。这是计算机编程中的通用约定。

**公式/代码表示：**
```python
L = [‘A‘, ‘B‘, ‘C‘, ‘D‘, ‘E‘]
# 索引：   0     1     2     3     4
```
*   `L[0]` 获取第一个元素 ‘A‘。
*   `L[1:4]` 获取索引1到3的元素，即 `[‘B‘, ‘C‘, ‘D‘]`（切片包含起始索引，不包含结束索引）。
*   `L[::2]` 从头到尾，每隔一个元素取一个，即 `[‘A‘, ‘C‘, ‘E‘]`。

## 列表的常用操作（增删改查）

列表提供了丰富的方法来管理其中的元素。以下是核心的操作方法。

### 1. 增加元素
我们可以向列表中添加新元素。
*   **`append()`**: 在列表**末尾**添加一个元素。如果添加的是另一个列表，它会将整个列表作为一个元素加入。
    ```python
    L = [1, 2, 3]
    L.append(4)       # L 变为 [1, 2, 3, 4]
    L.append([5, 6])  # L 变为 [1, 2, 3, 4, [5, 6]]
    ```
*   **`extend()`**: 在列表末尾**一次性添加**另一个列表中的所有元素（将其“展开”添加）。
    ```python
    L = [1, 2, 3]
    L.extend([4, 5])  # L 变为 [1, 2, 3, 4, 5]
    ```
*   **`insert()`**: 在**指定索引位置**插入一个元素。
    ```python
    L = [1, 2, 3]
    L.insert(1, ‘a‘)  # 在索引1的位置插入‘a‘，L变为 [1, ‘a‘, 2, 3]
    ```

### 2. 删除元素
我们可以从列表中移除元素。
*   **`del` 语句**: 根据索引删除元素。
    ```python
    L = [1, 2, 3, 4, 5]
    del L[0]  # 删除索引0的元素，L变为 [2, 3, 4, 5]
    ```
*   **`remove()`**: 删除列表中**第一个**匹配到的指定值。
    ```python
    L = [1, 2, 3, 2, 5]
    L.remove(2)  # 删除第一个2，L变为 [1, 3, 2, 5]
    ```
*   **`pop()`**: 默认删除并返回列表**最后一个**元素。也可指定索引。
    ```python
    L = [1, 2, 3, 4, 5]
    last_item = L.pop()      # last_item为5，L变为[1,2,3,4]
    second_item = L.pop(1)   # second_item为2，L变为[1,3,4]
    ```
*   **`clear()`**: 清空整个列表。
    ```python
    L = [1, 2, 3]
    L.clear()  # L变为 []
    ```

### 3. 修改与查询元素
*   **修改**: 通过索引直接赋值。
    ```python
    L = [1, 2, 3]
    L[1] = 20  # L变为 [1, 20, 3]
    ```
*   **查询长度**: 使用 `len()` 函数获取列表元素个数。
    ```python
    L = [1, 2, 3, 4, 5]
    length = len(L)  # length为5
    ```
*   **统计次数**: 使用 `count()` 方法统计某元素出现的次数。
    ```python
    L = [1, 2, 2, 3, 2]
    count_of_2 = L.count(2)  # count_of_2为3
    ```

## 列表的排序与反转

列表可以方便地进行排序和顺序反转。

*   **`sort()`**: **原地排序**列表，即直接修改原列表。默认升序。
    ```python
    L = [3, 1, 4, 1, 5]
    L.sort()               # L变为 [1, 1, 3, 4, 5]
    L.sort(reverse=True)   # 降序排列，L变为 [5, 4, 3, 1, 1]
    ```
*   **`sorted()`**: **返回一个新的排序后的列表**，不改变原列表。
    ```python
    L = [3, 1, 4]
    new_list = sorted(L)   # new_list为 [1, 3, 4]，L仍为 [3, 1, 4]
    ```
*   **`reverse()`**: **原地反转**列表元素的顺序（不是排序）。
    ```python
    L = [1, 2, 3, 4]
    L.reverse()  # L变为 [4, 3, 2, 1]
    ```

## 列表在量化中的应用实例 💡

理论学习之后，我们来看一个列表在期货量化中的实际应用场景：**批量下载多个品种的日线数据**。

如果没有列表，我们需要为每个期货品种（如IF、IC、RB）单独写一行下载代码，非常繁琐。利用列表，我们可以将所有的品种代码存储在一个列表中，然后通过一个循环即可批量处理。

**公式/代码表示：**
```python
# 定义一个包含多个期货品种代码的列表
symbol_list = [‘IF888‘, ‘IC888‘, ‘RB888‘] # 示例代码，可扩展至79个品种

# 通过循环，一遍代码即可处理所有品种
for symbol in symbol_list:
    # 这里是调用天勤API下载`symbol`品种日线数据的函数
    # download_daily_data(symbol)
    print(f“正在下载 {symbol} 的数据...“)
```
通过这种方式，代码变得极其简洁和高效，这正是列表在自动化处理中的价值体现。
