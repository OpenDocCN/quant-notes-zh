# Python金融分析与量化交易实战教程：04-4-3：3-Sigma方法实例 📊

在本节课中，我们将要学习一种基于统计学原理的数据预处理方法——3-Sigma方法。该方法用于识别和处理数据中的异常值（极值），是金融数据分析和量化交易中常用的技术。

上一节我们介绍了其他几种去极值的方法，本节中我们来看看基于正态分布假设的3-Sigma方法。

## 3-Sigma方法原理 📈

3-Sigma方法的原理非常简单。大家先来看下面这张图，图中画了一个正态分布曲线。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_1.png)

正态分布，如果理解起来比较麻烦，可以将其看作一个“正常”的分布。例如，在银行贷款的场景中，银行多借或少借你几块钱是正常的，但多借或少借你十万八万就不正常了。

在标准正态分布中，均值（μ）为0。以误差为例，误差可能为正（多借钱）也可能为负（少借钱）。图中红色阴影区域面积较大，代表这些事件发生的概率较高。当区间越往外扩展时，事件发生的概率就越低。

我们现在要做的是去极值。极值通常不会落在均值附近（红色区域），因为那是大概率发生的正常数据。极值更可能落在远离均值的偏远区域，偏差越大，越有可能是离群点。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_1.png)

## 3-Sigma区间定义 🎯

3-Sigma是这样定义的：我们以均值为中心，向左右各扩展若干倍的标准差（σ）来划定一个区间。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_3.png)

图中展示了均值（X̄）加减1倍、2倍、3倍标准差所形成的区间。区间选得越大（例如±3σ），数据落在区间之外的可能性就越小。

3-Sigma方法基于一个假设：我们可以对数据进行某种变换，使其理论上服从高斯分布（正态分布）。当我们指定正负三倍标准差（±3σ）的区间时，这个区间内包含了约99.7%的数据。因此，我们可以将最正常的99.7%数据保留，将剩下的0.3%视为极值并进行规范化处理。

在高斯分布中，只要知道数据的均值（μ）和标准差（σ），就能计算出上限和下限：
*   **上限** = μ + n * σ
*   **下限** = μ - n * σ
其中，n通常取3。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_3.png)

## 代码实现 💻

以下是3-Sigma方法的Python代码实现。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_5.png)

首先，我们定义一个函数，其逻辑与之前的方法类似。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_5.png)

在函数中，我们需要计算传入数据序列的均值和标准差。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_7.png)

![](img/f4f2fa98506b443e4617ab4f9eea1c35_7.png)

![](img/f4f2fa98506b443e4617ab4f9eea1c35_9.png)

计算均值和标准差的代码如下：
```python
mean = data.mean()   # 计算均值
std = data.std()     # 计算标准差
```

得到这两个指标后，就可以计算上限和下限。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_11.png)

![](img/f4f2fa98506b443e4617ab4f9eea1c35_9.png)

![](img/f4f2fa98506b443e4617ab4f9eea1c35_13.png)

上限和下限的计算公式如下：
```python
up = mean + n * std  # 计算上限
low = mean - n * std # 计算下限
```

![](img/f4f2fa98506b443e4617ab4f9eea1c35_15.png)

函数最终返回处理后的数据序列，以及计算出的下限和上限值。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_17.png)

![](img/f4f2fa98506b443e4617ab4f9eea1c35_11.png)

![](img/f4f2fa98506b443e4617ab4f9eea1c35_19.png)

完整的函数定义如下：
```python
def three_sigma(data, n=3):
    mean = data.mean()
    std = data.std()
    up = mean + n * std
    low = mean - n * std
    # 将超出范围的值规范到边界
    data_capped = data.clip(lower=low, upper=up)
    return data_capped, low, up
```

## 核心要点总结 ✨

![](img/f4f2fa98506b443e4617ab4f9eea1c35_37.png)

本节课中我们一起学习了3-Sigma去极值方法。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_39.png)

主要要点如下：
*   **方法原理**：基于数据服从正态分布的假设，利用均值（μ）和标准差（σ）划定正常数据的范围（通常为μ ± 3σ），在此范围外的数据被视为极值。
*   **实现步骤**：计算数据的均值和标准差，根据公式计算上下限，最后将超出限值的数据规范到边界上。
*   **核心目标**：去极值的目的不是直接删除数据，而是通过定义合理的下界和上界，将所有数值规范到指定范围内，以保证后续分析的稳定性。

![](img/f4f2fa98506b443e4617ab4f9eea1c35_41.png)

这是我们在“三步走”数据预处理流程中的第一步——数据规范化处理的重要环节。
