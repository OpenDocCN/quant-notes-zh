在本节课中，我们将学习如何从股票历史交易数据中，找出所有收盘价比开盘价上涨超过3%的日期。这是量化分析中的一个基础但重要的步骤，可以帮助我们识别股票的强势上涨时段。

## 数据预处理回顾

![](img/9fbbb294e6138383a95781f8baed1566_8.png)

上一节我们介绍了如何对股票历史交易数据进行预处理，特别是将`date`列转换为时间序列并设置为行索引。这样做是因为后续的分析需求（例如查找特定日期）会频繁用到日期信息。将日期作为时间序列并设为索引后，我们可以利用Pandas时间序列的强大功能来高效地处理这些需求。

![](img/9fbbb294e6138383a95781f8baed1566_10.png)

## 核心需求分析

![](img/9fbbb294e6138383a95781f8baed1566_12.png)

本节中我们来看看具体的需求：找出该股票所有**收盘价比开盘价上涨3%**的日期。这个分析可以帮助我们总结股票的上涨规律，例如发现上涨多集中在某些月份，为未来的投资时机提供参考。

我们的目标是获取满足条件的日期列表。

![](img/9fbbb294e6138383a95781f8baed1566_14.png)

## 实现思路与伪代码

首先，我们需要明确计算涨幅的公式。收盘价比开盘价上涨3%的条件可以用以下公式表示：
```
(close - open) / open > 0.03
```
其中：
*   `open` 代表股票的开盘价。
*   `close` 代表股票的收盘价。
*   `0.03` 即3%。

![](img/9fbbb294e6138383a95781f8baed1566_16.png)

在Pandas的DataFrame中，`open`和`close`都是Series（序列）。因此，上述公式可以直接应用于整个数据列，进行向量化计算。

![](img/9fbbb294e6138383a95781f8baed1566_18.png)

## 代码实现步骤

![](img/9fbbb294e6138383a95781f8baed1566_20.png)

以下是实现该需求的具体步骤和代码。

![](img/9fbbb294e6138383a95781f8baed1566_22.png)

### 步骤1：构建布尔条件序列

![](img/9fbbb294e6138383a95781f8baed1566_24.png)

我们首先根据公式创建一个布尔序列（Boolean Series），其中`True`表示该日期满足上涨3%的条件，`False`则表示不满足。

![](img/9fbbb294e6138383a95781f8baed1566_26.png)

```python
# 假设df是我们的股票数据DataFrame，且已进行预处理
condition_series = (df[‘close’] - df[‘open’]) / df[‘open’] > 0.03
```
执行这行代码后，`condition_series`就是一个与`df`行数相同的Series，其值为`True`或`False`。

### 步骤2：利用布尔索引筛选数据

![](img/9fbbb294e6138383a95781f8baed1566_28.png)

![](img/9fbbb294e6138383a95781f8baed1566_30.png)

在数据分析中有一个实用经验：**如果得到了布尔值序列，下一步可以立即将其作为原数据的行索引进行筛选**。

![](img/9fbbb294e6138383a95781f8baed1566_32.png)

![](img/9fbbb294e6138383a95781f8baed1566_34.png)

这是因为在Pandas中，当我们将一个布尔序列放入DataFrame的`loc`索引器时，它会自动返回所有`True`值对应的行数据，并忽略`False`值对应的行。

![](img/9fbbb294e6138383a95781f8baed1566_36.png)

```python
# 将布尔序列作为行索引，筛选出满足条件的行
filtered_df = df.loc[condition_series]
```
此时，`filtered_df`包含了所有收盘价比开盘价上涨超过3%的日子的完整数据（开盘价、收盘价、最高价、最低价等）。

### 步骤3：提取目标日期

![](img/9fbbb294e6138383a95781f8baed1566_38.png)

![](img/9fbbb294e6138383a95781f8baed1566_40.png)

我们的最终目标是日期，而不是整行数据。由于在预处理时我们已经将日期设置为行索引（`df.index`），因此满足条件的行的索引就是我们要找的日期。

![](img/9fbbb294e6138383a95781f8baed1566_42.png)

```python
# 获取筛选后数据的行索引，即目标日期列表
target_dates = filtered_df.index
```
这样，`target_dates`就是一个包含了所有满足上涨条件的日期的Index对象。

![](img/9fbbb294e6138383a95781f8baed1566_44.png)

## 关键技巧总结

![](img/9fbbb294e6138383a95781f8baed1566_46.png)

本节课中我们一起学习了如何用Pandas高效筛选数据，核心技巧是**布尔索引（Boolean Indexing）**。我们通过一个向量化的比较运算得到布尔序列，然后利用`df.loc[boolean_series]`直接提取出所有符合条件的行。这个方法避免了繁琐的循环，代码简洁且执行效率高。

![](img/9fbbb294e6138383a95781f8baed1566_48.png)

记住这个模式：**定义条件 -> 生成布尔序列 -> 使用`.loc`筛选 -> 获取结果**。这对于处理DataFrame的筛选任务非常通用。

![](img/9fbbb294e6138383a95781f8baed1566_50.png)

![](img/9fbbb294e6138383a95781f8baed1566_51.png)

---
**本节课总结**：我们掌握了从股票数据中筛选特定价格行为日期的方法。核心是利用 `(close - open) / open > 0.03` 公式构造条件，并通过 **布尔索引** `df.loc[condition]` 来高效地提取满足条件的行数据，最终从行索引中获得目标日期列表。
