## 最大回撤的定义

在策略分析中，最大回撤是一个关键指标。

![](img/f61de89f15f4348ee4ba5442d5fd7540_3.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_1.png)

最大回撤描述的是投资过程中可能经历的最糟糕情况。它不能仅从字面理解，其核心是衡量投资过程中的“惨烈”程度。

具体来说，在投资过程中，资产价格必然有下跌的时候。**最大回撤区间**指的就是资产价值从某个高点下跌到后续低点，跌幅最大、持续时间最长的那段时期。

![](img/f61de89f15f4348ee4ba5442d5fd7540_3.png)

我们可以通过公式来更精确地理解这个概念。假设 `Pi` 代表第 `i` 天的资产价格，`Pj` 代表第 `i` 天之后某一天（第 `j` 天）的资产价格。最大回撤的计算就是寻找所有可能的 `i` 和 `j`（`j > i`）组合中，价值缩水最严重的那个区间。

例如，价格可能先经历一个为期5天、从10元跌到5元的下跌。之后价格反弹，又可能经历一个更剧烈的下跌，比如从15元跌到1元。我们需要遍历所有数据点，找出价值下降最严重、持续时间最长的那个区间。

最大回撤的计算公式通常包含一个比率，即跌幅除以初始高点价格，这代表了资产的**缩水比例**。

这个指标并非直接评估策略的盈利能力，而是评估其**风险**。风险意味着，即使一个策略最终是盈利的，但在过程中可能会经历一段非常痛苦的时期，导致投资者难以承受。最大回撤就是量化这段“最痛苦时期”的指标。

在投资中，风险控制至关重要。历史事件（如2008年股市暴跌）提醒我们，巨大的回撤可能带来严重后果。最大回撤区间就是用来描述在某段时间内，投资组合价值下跌的“惨烈”程度。

![](img/f61de89f15f4348ee4ba5442d5fd7540_5.png)

在我们的回测策略分析中，系统会自动统计并标识出回测期间内，资产曲线经历的最大回撤区间。

接下来，我们将学习如何计算这个指标。

## 最大回撤的计算方法

![](img/f61de89f15f4348ee4ba5442d5fd7540_7.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_5.png)

计算最大回撤需要比较多个可能的下跌区间，并从中找出最严重的一个。这涉及到对时间序列数据的遍历和比较。

我们可以借助Pandas库中的函数来简化计算。首先，了解一个关键函数：`cummax()`。

![](img/f61de89f15f4348ee4ba5442d5fd7540_7.png)

`cummax()` 是“累计最大值”函数。对于一个序列，它会返回从序列开始到当前位置为止的最大值。

为了演示，我们先创建一个简单的Pandas Series对象。

```python
import pandas as pd
import numpy as np

# 创建一个示例序列，包含一些价格数据和一个空值（NaN）
s = pd.Series([2913, 2925, np.nan, 2938, 2934])
print("原始序列:")
print(s)
```

![](img/f61de89f15f4348ee4ba5442d5fd7540_9.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_9.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_11.png)

现在，我们对这个序列应用 `cummax()` 函数。

![](img/f61de89f15f4348ee4ba5442d5fd7540_13.png)

```python
cum_max = s.cummax()
print("\n累计最大值序列:")
print(cum_max)
```

![](img/f61de89f15f4348ee4ba5442d5fd7540_11.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_15.png)

执行结果会显示，对于序列中的每个位置，其值等于从序列开始到该位置所有元素的最大值（空值NaN通常被忽略，不影响计算）。例如，第一个值就是它本身；第二个值是前两个的最大值；第三个值因为遇到NaN，可能保持前一个最大值，以此类推。

![](img/f61de89f15f4348ee4ba5442d5fd7540_13.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_17.png)

这与我们之前学过的 `cumsum()`（累计求和）函数逻辑类似，只是这里计算的是最大值。

![](img/f61de89f15f4348ee4ba5442d5fd7540_19.png)

那么，`cummax()` 如何帮助我们计算最大回撤呢？思路是：要找到“最惨”的下跌，我们需要知道每个时间点之前的**历史最高点**。用这个历史最高点减去当前价格，就能得到从历史高点至今的潜在亏损（回撤）幅度。

---

![](img/f61de89f15f4348ee4ba5442d5fd7540_21.png)

### 计算步骤分解

![](img/f61de89f15f4348ee4ba5442d5fd7540_23.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_25.png)

假设我们有一个代表资产净值的序列 `data`。

![](img/f61de89f15f4348ee4ba5442d5fd7540_27.png)

1.  **计算历史最高点序列**：对 `data` 应用 `cummax()`，得到每个时间点之前（含当前）的历史最高净值。
    ```python
    historical_max = data.cummax()
    ```

2.  **计算每日回撤**：用每个时间点的历史最高净值减去当日的实际净值。这个差值代表了如果投资者在历史最高点买入，持有到当日所面临的亏损。
    ```python
    drawdown = historical_max - data
    ```
    ![](img/f61de89f15f4348ee4ba5442d5fd7540_31.png)

3.  **计算每日回撤率**：为了更直观地比较不同价格水平下的回撤，我们计算回撤率，即回撤金额除以对应的历史最高净值。
    ```python
    drawdown_ratio = drawdown / historical_max
    ```
    ![](img/f61de89f15f4348ee4ba5442d5fd7540_41.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_29.png)

4.  **找出最大回撤率**：在所有每日回撤率中，最大值就是整个考察期内的**最大回撤**。
    ```python
    max_drawdown = drawdown_ratio.max()
    print(f"最大回撤率为: {max_drawdown:.2%}")
    ```
    ![](img/f61de89f15f4348ee4ba5442d5fd7540_39.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_31.png)

让我们用一个简化的代码块连贯地展示这个过程：

```python
# 假设 df[‘net_value’] 是资产净值序列
data = df['net_value']

![](img/f61de89f15f4348ee4ba5442d5fd7540_33.png)

# 步骤1: 计算历史最高点
historical_max = data.cummax()

![](img/f61de89f15f4348ee4ba5442d5fd7540_35.png)

# 步骤2: 计算回撤
drawdown = historical_max - data

![](img/f61de89f15f4348ee4ba5442d5fd7540_37.png)

# 步骤3: 计算回撤率
drawdown_ratio = drawdown / historical_max

# 步骤4: 找到最大回撤率
max_drawdown_ratio = drawdown_ratio.max()
print(f"最大回撤率为: {max_drawdown_ratio:.2%}")

# 我们也可以找到最大回撤发生的区间（结束点）
max_dd_end_index = drawdown_ratio.idxmax()
# 寻找这个结束点对应的开始点（历史最高点出现的日期）
# 注意：可能需要向前追溯，找到 historical_max 首次等于当前 historical_max 的日期
```

![](img/f61de89f15f4348ee4ba5442d5fd7540_37.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_39.png)

通过以上计算，我们得到了一个百分比数值，例如3.07%。这表示在该策略的回测历史中，资产净值从某个高点下跌到低点的最大缩水比例为3.07%。

## 总结

![](img/f61de89f15f4348ee4ba5442d5fd7540_43.png)

本节课我们一起学习了**最大回撤**这个核心风险指标。

![](img/f61de89f15f4348ee4ba5442d5fd7540_45.png)

![](img/f61de89f15f4348ee4ba5442d5fd7540_47.png)

*   **定义**：最大回撤描述了投资策略在特定时期内，资产净值从最高点到之后最低点的**最大可能跌幅**。它衡量的是投资过程中可能面临的最糟糕情况。
*   **意义**：它是一个重要的**风险指标**，而非盈利指标。帮助投资者了解策略的潜在波动性和可能承受的最大心理压力及资金损失。
*   **计算**：其核心计算思想是利用 `cummax()` 函数找出历史高点，然后计算当前净值相对于历史高点的回撤比率，最终找出所有比率中的最大值。
*   **应用**：在量化回测中，最大回撤是评估策略稳健性的关键依据之一。一个优秀的策略不仅要有高收益，也应有可控的最大回撤。

理解并监控最大回撤，对于管理投资风险、制定合理的资金管理策略至关重要。
