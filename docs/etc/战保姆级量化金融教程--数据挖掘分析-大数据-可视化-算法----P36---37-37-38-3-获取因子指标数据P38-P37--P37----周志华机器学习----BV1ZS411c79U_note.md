## 第一步：选择交易日日期范围 📅

![](img/3486def2cc0b9c048007899ff503fd6f_12.png)

要获取数据，首先需要确定查询的日期范围。我们将编写一个函数来获取指定时间段内的所有交易日。

![](img/3486def2cc0b9c048007899ff503fd6f_14.png)

```python
def get_trading_dates(start_date, end_date):
    """
    获取指定起始日期和结束日期之间的所有交易日。
    参数:
        start_date (str): 起始日期，格式为'YYYY-MM-DD'。
        end_date (str): 结束日期，格式为'YYYY-MM-DD'。
    返回:
        pandas.DatetimeIndex: 交易日期的列表。
    """
    # 调用平台API获取交易日历
    all_dates = get_trading_dates(start_date, end_date)
    return all_dates
```

![](img/3486def2cc0b9c048007899ff503fd6f_16.png)

![](img/3486def2cc0b9c048007899ff503fd6f_18.png)

现在，我们调用这个函数，获取2019年全年的交易日列表。

![](img/3486def2cc0b9c048007899ff503fd6f_20.png)

```python
# 获取2019-01-01到2020-01-01之间的所有交易日

print(trading_dates[:5]) # 查看前5个交易日，检查是否正确
```

## 第三步：编写因子查询语句 🔍

有了日期和股票，现在可以构建查询来获取具体的因子数据。我们以市盈率（P/E）因子为例。

![](img/3486def2cc0b9c048007899ff503fd6f_32.png)

以下是构建查询语句的步骤：

```python
# 1. 创建一个查询对象
from zipline.pipeline.factors import CustomFactor
# 假设我们查询的因子在Fundamentals中叫做‘pe_ratio’
# 注意：实际因子名需参考平台文档

![](img/3486def2cc0b9c048007899ff503fd6f_34.png)

# 2. 构建Pipeline查询
# 首先，创建一个因子对象
pe_ratio = Fundamentals.pe_ratio.latest

![](img/3486def2cc0b9c048007899ff503fd6f_36.png)

# 然后，创建Pipeline并设置筛选条件
def make_pipeline():
    return Pipeline(
        columns={
            'pe_ratio': pe_ratio, # 要获取的因子数据
        },
        screen=pe_ratio.notnull() & Fundamentals.stockcode.in_(stock_pool) # 筛选条件：因子值不为空且在股票池内
    )
```

![](img/3486def2cc0b9c048007899ff503fd6f_38.png)

为了确保查询语句正确，一个更直接的方法是参考平台的帮助文档或示例代码。核心是使用`query`函数和`filter`方法。

![](img/3486def2cc0b9c048007899ff503fd6f_40.png)

```python
# 示例：直接使用query API进行查询（具体语法取决于所用平台）
q = query(
    Fundamentals.pe_ratio
).filter(
    Fundamentals.stockcode.in_(stock_pool)
)
```

![](img/3486def2cc0b9c048007899ff503fd6f_42.png)

这段代码定义了一个查询`q`，意思是：获取股票池`stock_pool`中所有股票的市盈率数据。
