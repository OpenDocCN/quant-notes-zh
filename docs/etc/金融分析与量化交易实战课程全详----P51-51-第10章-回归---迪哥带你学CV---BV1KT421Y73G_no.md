## 什么是回归问题？🤔

上一节我们介绍了课程背景，本节中我们来看看什么是回归问题。

在机器学习的有监督学习中，主要存在两类问题：分类和回归。

*   **分类问题**：其输出结果是明确的类别。例如，银行判断是否批准贷款，结果只有“批准”（可标记为1）或“拒绝”（可标记为0）两种。
*   **回归问题**：其输出结果是一个具体的连续数值。例如，询问银行能批准多少贷款额度，结果可能是0到10万之间的任意数值，如3.62万、4.89万或9.76万。回归的目标是在一个取值范围内找到一个可能的预测值。

我们今天要讨论的就是回归问题。

---

## 数据构成与建模目标 📊

理解了问题类型后，我们来看看用于回归分析的数据通常是什么样子。

假设我们有一份数据，旨在根据用户的“工资”和“年龄”来预测其“贷款额度”。数据可以整理成如下表格形式：

| 工资 (X1) | 年龄 (X2) | 额度 (Y) |
| :--- | :--- | :--- |
| ... | ... | ... |

在这个数据集中：
*   **X1（工资）** 和 **X2（年龄）** 被称为**特征**，是模型进行预测时所依据的指标。
*   **Y（额度）** 被称为**标签**，是模型训练时需要参考并最终预测的目标值。

因此，构建回归模型的任务就是：利用已知的特征 **X1** 和 **X2**，去预测未知的标签 **Y** 的值。

---

## 构建线性回归方程 ➕

既然目标是预测Y，而Y与X1、X2都相关，我们很自然地会想到用一个方程来描述它们的关系。

最直接的想法可能是将它们简单相加：`Y = X1 + X2`。但这隐含了一个假设：工资和年龄对额度的影响程度完全相同（系数均为1）。这显然不合理，通常工资的影响会比年龄更大。

因此，我们需要为每个特征引入一个系数，来量化其影响程度。于是，我们可以建立如下线性方程：

**`Y = θ1 * X1 + θ2 * X2`**

在这个方程中：
*   **`θ1`** 和 **`θ2`** 是模型的**参数**（或称权重）。
*   例如，若 `θ1 = 3`，`θ2 = 1`，则意味着工资（X1）的影响是年龄（X2）的3倍。

现在，在我们的数据中，X1、X2和Y都是已知的。那么，线性回归建模的核心目标就变成了：

**寻找一组最合适的参数 `θ1` 和 `θ2`，使得这个方程能最好地描述数据特征与标签之间的关系。**

## 模型的拟合与目标 🎯

![](img/ea2065f3e3933408e901b9ec9d6ff78b_3.png)

我们已经明确了要求解参数，但“最合适”如何定义呢？这引出了线性回归的核心思想。

一个线性方程在几何上代表一个平面（对于两个特征而言）。而我们的数据点（样本）是散布在这个空间中的。如下图所示，我们几乎不可能找到一条直线或一个平面能完美穿过所有数据点。



![](img/ea2065f3e3933408e901b9ec9d6ff78b_1.png)

既然无法满足所有点，那么一个合理的目标就是：**找到那个能尽可能“接近”所有数据点的平面**。也就是说，这个平面应使得所有数据点到该平面的总体距离或误差最小。



![](img/ea2065f3e3933408e901b9ec9d6ff78b_5.png)

如上图所示，可能存在多个平面（A, B, C），我们的目标是找到那个能使数据点（红色点）整体上离它最近的那个平面（例如平面B）。衡量这种“接近程度”或“误差”的常见方法是使用**均方误差**等损失函数，而求解最优 `θ1`、`θ2` 的过程就是最小化这个损失函数。

---

## 总结 📝

本节课中我们一起学习了回归分析策略的基础：

1.  **区分了回归与分类问题**：回归预测连续值，分类预测离散类别。
2.  **认识了回归数据**：数据由特征（X）和标签（Y）构成。
3.  **建立了线性回归模型**：模型形式为 `Y = θ1 * X1 + θ2 * X2`，其核心是找到参数 θ。
4.  **明确了建模目标**：线性模型无法完美拟合所有点，因此目标是找到一组参数，使得模型预测值与真实值之间的总体误差最小。
