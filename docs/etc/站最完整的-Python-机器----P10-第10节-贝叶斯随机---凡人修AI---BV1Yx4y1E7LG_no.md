## 为什么需要随机波动率？🤔

上一节我们介绍了时间序列分析的基本思想。本节中，我们来看看为什么波动率不能简单地视为常数。

我们的基本思路是希望时间序列满足平稳性和均值回复性。平稳性意味着我们估计的均值和方差在长期内是有效的，这有助于我们理解长期的风险和收益。均值回复性则意味着，当序列偏离其长期均值时，我们有理由预期它会向均值回归，从而可以通过做多或做空来获取利润。

然而，现实中的时间序列往往不满足这些理想条件。因此，我们需要更复杂的模型来描述资产价格的运动。

## 从几何布朗运动到Heston模型 🔄

首先，一个常见的模型是几何布朗运动。其表达式如下：

`dS_t = μ * S_t * dt + σ * S_t * dW_t`

其中：
*   `S_t` 代表标的资产在时间 `t` 的价格。
*   `μ` 是漂移率（期望收益率）。
*   `σ` 是波动率（常数）。
*   `dW_t` 是维纳过程（标准布朗运动）的微分。

在这个模型中，漂移项和不确定项都与当前股价 `S_t` 成正比。这符合一个基本市场直觉：股价越高，其向上漂移的趋势和不确定性可能越大。

但是，几何布朗运动存在一个弊端：长期观测显示，波动率并不总是与股价 `S_t` 成正比。因此，这个模型对波动率的刻画过于粗糙。

为了解决这个问题，我们引入下面的二元Heston模型。该模型对标的资产价格波动的微分序列描述与上面类似，只是将常数波动率 `σ` 替换为随时间变化的 `√V_t`：

`dS_t = μ * S_t * dt + √V_t * S_t * dW_t^S`

同时，模型对波动率的平方 `V_t` 本身进行随机刻画：

`dV_t = κ * (θ - V_t) * dt + ξ * √V_t * dW_t^V`

以下是关键参数的解释：
*   `θ`：`V_t` 的长期平均水平。
*   `κ`：均值回复的速度。当 `V_t` 高于 `θ` 时，漂移项为负，趋势向下拉；反之则向上拉。
*   `ξ`：波动率的波动率，即扰动项的尺度。
*   `dW_t^S` 和 `dW_t^V` 是两个相关的维纳过程。

这个模型更符合市场直觉：波动率本身是随机且均值回复的，并且其扰动大小与其当前水平有关（通过 `√V_t` 体现）。

## 贝叶斯随机波动率模型 🎲

![](img/92d26ca3770488acae39507c8b9ab38f_1.png)

![](img/92d26ca3770488acae39507c8b9ab38f_3.png)

Heston模型没有解析解，通常需要通过蒙特卡洛模拟进行计算。然而，模型中的参数（`μ`, `κ`, `θ`, `ξ`）难以直接估计。这时，我们引入贝叶斯方法。

![](img/92d26ca3770488acae39507c8b9ab38f_5.png)

其核心思想如下：
首先，我们观察资产的对数收益率（log return）。我们认为它服从一个学生t分布，均值为零（符合长期回报率为零的直觉），方差与一个潜在变量有关。

![](img/92d26ca3770488acae39507c8b9ab38f_7.png)

具体模型设定为：
1.  对数收益率 `r_t` 服从学生t分布：`r_t ~ StudentT(ν, 0, λ_t)`。其中 `ν` 是自由度（控制分布的厚尾程度），`λ_t` 与方差相关。
2.  潜在变量 `s_t`（称为“潜在到达率”）满足一个随机游走过程：`s_t ~ Normal(s_{t-1}, σ²)`。这表示其以上一时刻值为基准进行上下扰动，符合市场感知。
3.  参数 `σ` 和 `ν` 本身也有先验分布，例如均值为0.2的指数分布和均值为10的指数分布。

![](img/92d26ca3770488acae39507c8b9ab38f_9.png)

通过贝叶斯推断（如MCMC采样），我们可以同时得到参数和潜在状态的后验分布，从而估计出时变的波动率。

![](img/92d26ca3770488acae39507c8b9ab38f_11.png)

## Python实现与应用示例 🐍

接下来，我们看看如何用Python（使用`pymc3`库）实现贝叶斯随机波动率模型。我们以亚马逊（AMZN）股票数据为例。

![](img/92d26ca3770488acae39507c8b9ab38f_13.png)

以下是建模的核心步骤：
1.  计算股票的百分比收益率（return）。
2.  定义参数 `ν` 和 `σ` 的先验分布（指数分布）。
3.  定义潜在变量 `s` 的随机游走过程（高斯分布）。
4.  将波动率过程与学生t分布连接，并将收益率数据作为观测值。

![](img/92d26ca3770488acae39507c8b9ab38f_15.png)

```python
# 示例代码结构 (基于 pymc3)
import pymc3 as pm

with pm.Model() as model:
    # 先验分布
    nu = pm.Exponential('nu', 1./10)
    sigma = pm.Exponential('sigma', 1./0.02)

    # 似然函数 (学生t分布)
```

## 模型的应用价值 💡

得到波动率的分布后，我们可以将其应用于多个重要领域：
1.  **资产价格模拟**：在蒙特卡洛模拟中，使用时变的波动率序列而非固定均值，可以更真实地估计股票或期权的价格。
2.  **风险度量**：计算风险价值（VaR）时，经典方法（方差-协方差法）依赖于历史波动率。如果结合随机波动率模型或GARCH模型，可以得到更准确且动态的VaR估计，公式为：`VaR = μ + σ * Z_α`，其中 `σ` 来自模型预测的波动率。
