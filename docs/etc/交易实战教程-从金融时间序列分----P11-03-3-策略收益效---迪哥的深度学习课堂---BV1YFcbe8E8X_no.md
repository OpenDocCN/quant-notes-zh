## 概述

![](img/fc061904726f2865085ed24de9de7b82_4.png)

上一节我们介绍了如何通过双均线策略生成交易信号。本节中，我们来看看如何将这些信号转化为具体的收益数据，并进行量化分析。核心在于计算策略的累计收益率，并与基准收益率进行对比。

## 计算基准收益率

首先，我们需要计算股票的基准收益率，即在不采取任何策略、单纯持有股票情况下的收益变化。这是一个累加的过程，但直接计算价格比率不方便，我们通常引入对数收益率。

对数收益率的计算公式为：
`log_return = log(price_t / price_{t-1})`

![](img/fc061904726f2865085ed24de9de7b82_6.png)

在Python中，我们可以通过股价序列的移位操作来计算每日的简单收益率（作为近似），代码如下：

![](img/fc061904726f2865085ed24de9de7b82_8.png)

```python
# 假设‘data’是一个包含股价‘Close’列的DataFrame
data[‘return’] = data[‘Close’] / data[‘Close’].shift(1) - 1
```

![](img/fc061904726f2865085ed24de9de7b82_10.png)

执行以上代码后，我们得到了一个名为`return`的列，它代表了股价每日的增长率。

![](img/fc061904726f2865085ed24de9de7b82_12.png)

## 计算策略收益率

![](img/fc061904726f2865085ed24de9de7b82_14.png)

接下来，我们根据上一节生成的交易信号来计算策略收益率。我们的策略逻辑是：当`position`信号为1时，表示我们预测市场将下跌，因此采取与市场走势相反的操作（例如做空或反向交易）；当`position`信号为0时，表示我们跟随市场原始走势。

![](img/fc061904726f2865085ed24de9de7b82_16.png)

策略收益率的计算方法是：用`position`信号乘以每日的基准收益率。需要注意的是，`position`信号也需要进行移位操作，以确保信号作用于正确的交易日。

![](img/fc061904726f2865085ed24de9de7b82_18.png)

以下是计算策略收益率的步骤：

1.  对`position`列进行移位操作。
2.  将移位后的`position`信号与基准收益率`return`相乘。

![](img/fc061904726f2865085ed24de9de7b82_20.png)

具体代码如下：

![](img/fc061904726f2865085ed24de9de7b82_22.png)

```python
# 对position信号进行移位，确保与收益率对齐
strategy_position = data[‘position’].shift(1)
# 计算策略每日收益率
data[‘strategy_return’] = strategy_position * data[‘return’]
```

![](img/fc061904726f2865085ed24de9de7b82_24.png)

![](img/fc061904726f2865085ed24de9de7b82_26.png)

执行完上述操作后，`data` DataFrame中会增加一列`strategy_return`，代表策略每日的收益率。

![](img/fc061904726f2865085ed24de9de7b82_28.png)

## 计算累计收益并对比

![](img/fc061904726f2865085ed24de9de7b82_30.png)

为了比较长期收益，我们需要将每日收益率累加起来，并还原为最终的资金增长倍数。这需要两个步骤：

1.  对收益率序列进行累加求和。
2.  将求和结果通过指数函数还原为最终价值（假设初始本金为1元）。

![](img/fc061904726f2865085ed24de9de7b82_32.png)

以下是计算并对比两种策略最终收益的代码：

![](img/fc061904726f2865085ed24de9de7b82_34.png)

```python
# 计算基准策略的最终价值（初始1元）
final_value_baseline = np.exp(data[‘return’].dropna().sum())
# 计算双均线策略的最终价值（初始1元）
final_value_strategy = np.exp(data[‘strategy_return’].dropna().sum())

![](img/fc061904726f2865085ed24de9de7b82_36.png)

print(f“基准（买入持有）策略最终价值：{final_value_baseline:.2f}”)
print(f“双均线策略最终价值：{final_value_strategy:.2f}”)
```

![](img/fc061904726f2865085ed24de9de7b82_38.png)

通过对比这两个数值，我们可以直观地看到策略是否带来了超额收益。例如，如果基准策略最终价值为4.0，而双均线策略最终价值为5.8，则说明我们的策略有效提升了收益。

## 总结

![](img/fc061904726f2865085ed24de9de7b82_40.png)

本节课中我们一起学习了量化策略收益分析的核心流程。我们首先计算了股票的基准收益率，然后结合交易信号计算了策略收益率，最后通过累加和还原操作，对比了策略与基准的最终收益效果。这个过程是量化交易回测中评估策略有效性的基础环节。虽然本例中的策略信号是预设的（用于教学演示），但实际应用中，信号将由模型根据历史数据动态生成。
