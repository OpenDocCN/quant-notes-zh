在本节课中，我们将要学习如何对一个量化交易策略的收益效果进行分析。我们将通过计算策略的累计收益，并与“什么都不做”（即持有原始资产）的收益进行对比，来评估策略的有效性。

![](img/e407ade22a8c87422fd0747ecb556cc1_3.png)

上一节我们介绍了如何通过双均线策略生成交易信号（`position`列）。本节中我们来看看如何基于这些信号计算策略的最终收益。

## 计算原始收益率

首先，我们需要计算股价的日收益率。收益率反映了资产价格每日的变化情况。

![](img/e407ade22a8c87422fd0747ecb556cc1_5.png)

以下是计算日收益率的步骤：
1.  获取股价数据。
2.  计算当前日股价除以前一日股价的结果。
3.  为了后续计算方便，我们通常使用对数收益率。

![](img/e407ade22a8c87422fd0747ecb556cc1_7.png)

在代码中，我们使用Pandas的`shift`方法将数据向下移动一位，以获取前一天的股价。

![](img/e407ade22a8c87422fd0747ecb556cc1_9.png)

```python
# 计算对数收益率
data['return'] = np.log(data['close'] / data['close'].shift(1))
```

![](img/e407ade22a8c87422fd0747ecb556cc1_11.png)

![](img/e407ade22a8c87422fd0747ecb556cc1_13.png)

执行以上代码后，我们得到了一个名为`return`的列，它代表了资产每日的增长率。

![](img/e407ade22a8c87422fd0747ecb556cc1_15.png)

## 计算策略收益率

![](img/e407ade22a8c87422fd0747ecb556cc1_17.png)

接下来，我们将根据策略信号来计算策略的收益率。我们的策略核心是：在`position`标记为`-1`的区间（即短期均线下穿长期均线的看跌区间），我们采取与市场走势相反的操作（例如做空或反向交易），从而将亏损转化为盈利。

以下是计算策略收益率的逻辑：
1.  策略收益率 = 交易信号 (`position`) × 原始收益率 (`return`)。
2.  当`position`为`1`时，表示跟随市场走势，收益率不变。
3.  当`position`为`-1`时，表示反向操作，收益率取反。

![](img/e407ade22a8c87422fd0747ecb556cc1_19.png)

![](img/e407ade22a8c87422fd0747ecb556cc1_21.png)

在应用`position`列时，同样需要进行`shift(1)`操作，以确保使用的是前一日生成的信号来指导当日的交易。

![](img/e407ade22a8c87422fd0747ecb556cc1_23.png)

```python
# 计算策略收益率，注意对position列也进行shift操作
data['strategy'] = data['position'].shift(1) * data['return']
```

![](img/e407ade22a8c87422fd0747ecb556cc1_25.png)

![](img/e407ade22a8c87422fd0747ecb556cc1_27.png)

计算完成后，我们得到了`strategy`列，它代表了执行策略后每日的收益率。

![](img/e407ade22a8c87422fd0747ecb556cc1_29.png)

## 比较累计收益

为了直观比较策略效果，我们需要将每日的收益率累加起来，并还原为最终的资金增长倍数（即1元本金最终变为多少元）。

![](img/e407ade22a8c87422fd0747ecb556cc1_31.png)

以下是计算并比较累计收益的步骤：
1.  分别对`return`列和`strategy`列的收益率进行求和，得到总的对数收益率。
2.  使用指数函数 `np.exp()` 将对数总收益率还原为资金倍数。
3.  比较两种情况下资金的最终价值。

![](img/e407ade22a8c87422fd0747ecb556cc1_33.png)

```python
# 计算并比较累计收益
total_return = np.exp(data['return'].sum())
total_strategy = np.exp(data['strategy'].sum())

![](img/e407ade22a8c87422fd0747ecb556cc1_35.png)

![](img/e407ade22a8c87422fd0747ecb556cc1_37.png)

print(f"原始持有收益: 1元 -> {total_return:.2f}元")
print(f"策略执行收益: 1元 -> {total_strategy:.2f}元")
```

在本例的演示中，执行策略后的收益（例如1元变为5.8元）高于单纯持有资产的收益（例如1元变为4.0元），这表明在当前数据回测下，该双均线策略带来了超额收益。

![](img/e407ade22a8c87422fd0747ecb556cc1_39.png)

本节课中我们一起学习了量化策略收益分析的核心流程：从计算基础资产收益率，到结合策略信号计算策略收益率，最后通过累计和还原来评估策略的最终效果。这个过程是检验任何一个交易策略是否有效的关键步骤。
