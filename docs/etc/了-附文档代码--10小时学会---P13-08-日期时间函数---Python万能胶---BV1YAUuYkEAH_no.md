在本节课中，我们将学习Excel中几个核心的日期与时间函数。这些函数能帮助我们获取当前时间、构造特定日期以及计算日期之间的差值，是数据处理中非常实用的工具。

---

![](img/465de7ef3d3cbbb265e9dea9d9bd44e5_4.png)

上一节我们介绍了其他类型的函数，本节中我们来看看专门处理日期和时间的函数。首先，我们来了解两个获取当前时间的函数。

## 系统自带函数：TODAY 与 NOW

以下是两个无需参数即可返回当前日期和时间的函数。

*   **`TODAY()`**：此函数返回当前的系统日期。
    *   **公式**：`=TODAY()`
    *   **结果示例**：`2020-06-17`

*   **`NOW()`**：此函数返回当前的系统日期和时间。
    *   **公式**：`=NOW()`
    *   **结果示例**：`2020-06-17 11:26:00`

---

了解了如何获取当前时间后，我们来看如何根据已有的年、月、日数据来构造一个标准的日期。

## 构造日期：DATE 函数

`DATE` 函数用于将独立的年、月、日数值组合成一个标准的日期格式。这在数据清洗，例如将拆分后的日期列重新合并时非常有用。

以下是 `DATE` 函数的语法和示例。

*   **语法**：`=DATE(年, 月, 日)`
*   **示例**：假设A1单元格是年份 `2020`，B1单元格是月份 `6`，C1单元格是日期 `17`。
    *   **公式**：`=DATE(A1, B1, C1)`
    *   **结果**：`2020-06-17`

掌握了日期的构造，我们接下来学习一个强大的函数，它可以精确计算两个日期之间相差的天数、月数或年数。

## 计算日期差值：DATEDIF 函数

`DATEDIF` 函数是一个隐藏函数（输入时无提示），用于计算两个日期之间的差值，其精度高于简单的日期相减。它需要三个参数。

以下是 `DATEDIF` 函数的语法和不同单位的用法。

![](img/465de7ef3d3cbbb265e9dea9d9bd44e5_10.png)

*   **语法**：`=DATEDIF(开始日期, 结束日期, 单位)`
*   **参数“单位”说明**：
    *   `"D"`：计算两个日期之间相差的**天数**。
        *   **示例**：`=DATEDIF("2020-01-01", "2021-03-20", "D")` 返回 `444`。
    *   `"M"`：计算两个日期之间相差的**月数**。
        *   **示例**：`=DATEDIF("2020-01-01", "2021-03-01", "M")` 返回 `14`。
    *   `"Y"`：计算两个日期之间相差的**年数**。
        *   **示例**：`=DATEDIF("2020-01-01", "2021-03-20", "Y")` 返回 `1`。

**注意**：当使用 `"M"` 或 `"Y"` 单位时，结果单元格格式需设置为“常规”或“数字”以正确显示数值，而非日期。

本节课中我们一起学习了四个关键的日期时间函数。我们首先通过 `TODAY` 和 `NOW` 获取当前时间，然后使用 `DATE` 函数组合生成标准日期，最后利用 `DATEDIF` 函数精确计算日期之间的差值。掌握这些函数能有效提升处理时间相关数据的效率。
