## 什么是换手率？📊

上一节我们介绍了本课程的目标，本节中我们来看看策略的核心指标——换手率。

![](img/0a3a299af0f567d9eb6db016f69fa2ad_5.png)

换手率是衡量股票流通性强弱的重要指标。其计算公式为：

![](img/0a3a299af0f567d9eb6db016f69fa2ad_7.png)

![](img/0a3a299af0f567d9eb6db016f69fa2ad_9.png)

**换手率 = 当日成交量 / 流通总股数**

![](img/0a3a299af0f567d9eb6db016f69fa2ad_11.png)

用通俗的话解释，换手率反映了市场中买卖某只股票的频率。换手率高，说明该股票交易活跃、关注度高；换手率低，则说明该股票相对冷门。

![](img/0a3a299af0f567d9eb6db016f69fa2ad_13.png)

这个指标不仅用于股票分析，也常用于基金评价。基金换手率高，通常意味着基金经理频繁交易，风格偏短线；换手率低，则说明基金经理倾向于长期持有，风格可能更偏向价值投资。

![](img/0a3a299af0f567d9eb6db016f69fa2ad_15.png)

![](img/0a3a299af0f567d9eb6db016f69fa2ad_17.png)

## 策略构建思路 💡

![](img/0a3a299af0f567d9eb6db016f69fa2ad_19.png)

![](img/0a3a299af0f567d9eb6db016f69fa2ad_21.png)

理解了换手率的基本概念后，本节我们来看看如何利用它来构建一个具体的选股策略。

我们的核心想法是：既然高换手率代表热门和分歧，那么反其道而行之，买入市场关注度较低（即低换手率）的股票，是否可能获得更好的收益？

以下是策略的具体执行步骤：

![](img/0a3a299af0f567d9eb6db016f69fa2ad_23.png)

![](img/0a3a299af0f567d9eb6db016f69fa2ad_25.png)

1.  **选股时间**：在每个月的最后一个交易日收盘后。
2.  **计算指标**：计算所有股票在过去20个交易日的平均换手率。
3.  **股票过滤**：剔除ST/*ST股票、已退市或停牌的股票，以及上市未满一年的股票。
4.  **排序选股**：将剩余股票按平均换手率**从低到高**进行排序。
5.  **构建组合**：买入排名最低（即换手率最小）的10只股票。
6.  **交易执行**：在次月第一个交易日开盘时，等权重买入这10只股票。
7.  **持有与调仓**：持有该组合整整一个月，在次月最后一个交易日全部卖出。然后重复步骤1-6，构建新的投资组合。

![](img/0a3a299af0f567d9eb6db016f69fa2ad_27.png)

通过这个按月调仓的循环，我们就得到了一个完整的量化选股策略。

## 策略回测结果 📉

![](img/0a3a299af0f567d9eb6db016f69fa2ad_37.png)

在明确了策略的构建方法后，我们需要用历史数据来验证其有效性。我们使用Python和A股历史数据对该策略进行了回测。

![](img/0a3a299af0f567d9eb6db016f69fa2ad_39.png)

![](img/0a3a299af0f567d9eb6db016f69fa2ad_41.png)

以下是回测的关键发现：

*   **买入高换手率股票**：如果策略改为每月买入**换手率最高**的10只股票，从2014年至今，初始的1元钱会亏损至0.01元。这印证了金融市场的经验：“人多的地方不要去”。高换手率往往意味着投资者分歧大，难以形成推动股价上涨的一致预期。
*   **买入低换手率股票**：执行我们上述的**低换手率策略**，回测结果截然不同。从2014年至今，初始的1元钱增长到了约4.88元，实现了接近5倍的收益，年化收益率约16%，显著跑赢了同期的沪深300指数。
*   **近期表现**：该策略在2024年上半年表现尤为突出，实现了24%的收益，且最大回撤仅为4%。其选股多为基本面扎实、暴雷风险低的大市值行业龙头，因此在市场波动时持有信心更足。

这个策略每月仅需调仓一次，操作简单，适合工作繁忙的投资者作为指数增强型策略来使用。

## 策略优化与拓展 🚀

![](img/0a3a299af0f567d9eb6db016f69fa2ad_47.png)

我们已经看到了基础的低换手率策略表现优异。本节中，我们可以在此基础上进行一些优化和拓展，以探索更多的可能性。

![](img/0a3a299af0f567d9eb6db016f69fa2ad_49.png)

![](img/0a3a299af0f567d9eb6db016f69fa2ad_51.png)

你可以尝试修改选股条件，回测不同的思路：

![](img/0a3a299af0f567d9eb6db016f69fa2ad_53.png)

*   **选择换手率稳定的股票**：使用换手率的标准差来衡量其波动性。买入过去一段时间内换手率波动最小（标准差最低）的股票。
*   **使用换手率变化率**：关注换手率自身的变动趋势，例如买入换手率下降最快的股票。
*   **多因子结合**：将低换手率与其他指标（如市盈率、市净率、股息率等）结合，构建更复杂的多因子选股模型。
