## 一个实际案例：筛选期货品种

上一节我们介绍了编程的基本概念，本节中我们来看看如何用条件判断解决实际问题。

假设我们有一个包含所有主力合约品种代码的列表 `z_l`，现在需要从中筛选出上海期货交易所（SHFE）的品种。从逻辑上讲，SHFE的品种代码中都包含“SHFE”字段。

以下是实现该逻辑的代码：

```python
z_l = [‘AG2306’, ‘AL2306’, ‘AU2306’, ‘BU2306’, ‘CU2306’, ‘FU2306’, ‘HC2306’, ‘NI2306’, ‘PB2306’, ‘RB2306’, ‘RU2306’, ‘SN2306’, ‘SP2306’, ‘SS2306’, ‘WR2306’, ‘ZN2306’, ‘A2305’, ‘B2305’, ‘C2305’, ‘CS2305’, ‘EG2305’, ‘FB2305’, ‘I2305’, ‘J2305’, ‘JD2305’, ‘JM2305’, ‘L2305’, ‘M2305’, ‘P2305’, ‘PP2305’, ‘V2305’, ‘Y2305’, ‘CF2305’, ‘CJ2305’, ‘CY2305’, ‘FG2305’, ‘JR2305’, ‘LR2305’, ‘MA2305’, ‘OI2305’, ‘PM2305’, ‘RI2305’, ‘RM2305’, ‘RS2305’, ‘SA2305’, ‘SF2305’, ‘SM2305’, ‘SR2305’, ‘TA2305’, ‘UR2305’, ‘WH2305’, ‘ZC2305’, ‘AP2305’, ‘CJ2305’, ‘CY2305’, ‘FG2305’, ‘JR2305’, ‘LR2305’, ‘MA2305’, ‘OI2305’, ‘PM2305’, ‘RI2305’, ‘RM2305’, ‘RS2305’, ‘SA2305’, ‘SF2305’, ‘SM2305’, ‘SR2305’, ‘TA2305’, ‘UR2305’, ‘WH2305’, ‘ZC2305’]

for i in z_l:
    if ‘SHFE’ in i:
        print(i)
```

这段代码的含义是：使用`for`循环遍历列表`z_l`中的每一个元素（临时变量`i`）。`if`语句判断字符串“SHFE”是否存在于当前元素`i`中。如果条件满足（为真），则执行`print(i)`打印该品种代码；如果不满足，则跳过，继续检查下一个元素。

运行这段代码，程序会输出所有包含“SHFE”的品种，从而完成筛选。

通过这个案例，我们可以看到`if`的基本应用：**当满足某个条件时，执行后续的代码；不满足则跳过**。

## `if`语句的基本形式

理解了基本逻辑后，我们来系统性地学习`if`语句的几种写法。

### 1. 单独使用 `if`

`if`可以单独使用，用于判断单一条件。

```python
score = int(input(“请输入分数: “))
if score > 90:
    print(“优秀”)
```

运行示例：
- 输入`95`，输出`优秀`。
- 输入`80`，无输出。

### 2. `if` 与 `else` 搭配使用

当条件不满足时，我们通常希望执行另一段代码，这时就需要`else`。

```python
num = int(input(“请输入一个数字: “))
if num % 2 == 0:
    print(f”{num} 是偶数”)
else:
    print(f”{num} 是奇数”)
```

运行示例：
- 输入`6`，输出`6 是偶数`。
- 输入`5`，输出`5 是奇数`。

`if-else`结构可以对“是”与“否”两种情况做出完整响应。

### 3. 多条件判断：`if`、`elif`、`else`

现实问题往往有多个条件分支。例如，游乐场按年龄分段收费：6岁以下免费，6-18岁半价8元，18岁以上全价16元。

```python
age = int(input(“请输入年龄: “))
if age < 6:
    price = 0
    print(“免费”)
elif age < 18:
    price = 8
    print(“票价: 8元”)
else:
    price = 16
    print(“票价: 16元”)
```

运行示例：
- 输入`20`，输出`票价: 16元`。
- 输入`3`，输出`免费`。
- 输入`10`，输出`票价: 8元`。

`elif`是`else if`的缩写，用于检查多个条件，并且**一旦某个条件满足，后续的`elif`或`else`将不再执行**。

## `if`语句的语法结构与运算符

### 语法结构

`if`语句的标准书写格式如下：

```python
if 条件A:
    # 执行操作1 (注意缩进)
elif 条件B:
    # 执行操作2
else:
    # 执行操作3
```

**书写要点**：
1.  `if`、`elif`、`else`后面要加英文冒号`:`。
2.  冒号下一行的代码必须**缩进**（通常为4个空格或一个Tab键）。缩进内的代码属于同一个代码块。
3.  条件判断的顺序是：先检查`条件A`，若为真则执行`操作1`后结束；若为假则检查`条件B`，依此类推；所有条件都不满足时，执行`else`下的`操作3`。

### 常用运算符

在条件判断中，我们经常使用以下运算符来构建条件：

**比较运算符**：
- `==`：等于（注意是双等号，用于比较）
- `!=`：不等于
- `>`：大于
- `<`：小于
- `>=`：大于等于
- `<=`：小于等于

**逻辑运算符**：
- `and`：与（条件同时满足）
- `or`：或（条件满足其一）
- `not`：非（取反）

**成员与身份运算符**（示例中已使用）：
- `in`：在序列中
- `not in`：不在序列中
- `is`：是同一个对象
- `is not`：不是同一个对象

## 进阶应用与技巧

### 嵌套 `if` 语句

在一个`if`或`else`的代码块中，可以再嵌入新的`if`语句，这称为嵌套。它可以处理更复杂的层级逻辑。

```python
score = int(input(“请输入成绩: “))
if score >= 90:
    print(“优秀”)
else:
    if score >= 80:
        print(“良好”)
    else:
        if score >= 70:
            print(“中等”)
        else:
            if score >= 60:
                print(“及格”)
            else:
                print(“不及格”)
```

这段代码实现了成绩分级。虽然此问题用一层`if-elif-else`也能解决，但嵌套结构展示了处理多层逻辑的可能性。运行结果符合预期（如输入`75`输出`中等`）。

### 更简洁的写法：条件表达式

Python提供了一种更简洁、类似口语的写法，称为条件表达式（或三元运算符）。

传统写法：
```python
score = 100
if score == 100:
    status = “满分”
else:
    status = “未满分”
print(score, status)
```

使用条件表达式简化：
```python
score = 100
status = “满分” if score == 100 else “未满分”
print(score, status)
```

两种写法结果相同。条件表达式 `x if condition else y` 的含义是：如果`condition`为真，则值为`x`，否则值为`y`。它让代码更紧凑，但逻辑复杂时可能影响可读性。
