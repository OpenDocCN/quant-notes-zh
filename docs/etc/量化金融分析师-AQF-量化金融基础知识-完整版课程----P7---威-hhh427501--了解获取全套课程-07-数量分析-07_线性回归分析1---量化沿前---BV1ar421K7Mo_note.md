在本节课中，我们将要学习线性回归分析的基础知识。线性回归是研究变量之间相关关系的重要工具，在量化金融等领域有广泛应用。我们将从基本概念入手，逐步理解其原理、计算方法以及如何检验其有效性。

## 回归分析概述

回归分析研究的是变量之间的相关关系。

变量之间的关系可以分为两种。一种是确定性的函数关系。例如，圆的面积 **S** 和半径 **R** 之间的关系是 **S = πR²**。知道 **R** 就能唯一确定 **S**。

另一种是相关关系。例如，身高和体重之间存在正相关关系，但并非一一对应。身高较高的人，体重通常也较重，但并非绝对。回归分析研究的正是这种相关关系。

需要特别注意，回归分析研究的是相关关系，而非因果关系。例如，研究发现股价与降雨量存在相关关系，但这并不意味着降雨量是导致股价变化的原因。

## 回归分析在量化金融中的应用

回归分析方法在众多领域都有广泛应用。在量化投资领域，一个主流的模型是多因子模型。

以下是检验因子有效性的常用回归方法：
1.  收集所有股票每一期的个股收益 **R**，以及前一期的因子值 **X**。
2.  将个股收益 **R** 对因子值 **X** 进行回归，得到方程 **R = a + bX**。
3.  如果回归系数 **b** 显著不为零，则说明该因子有效。**b > 0** 表示因子值越大，预期收益越高；**b < 0** 则表示因子值越小，预期收益越高。若 **b** 不显著，则说明该因子对收益的预测能力较弱。

另一个应用是配对交易。通过历史数据回归，可以找到两只高度相关股票（例如股票Y和股票X）的价格关系，如 **Y = 1.5X**。当实时价格偏离此历史关系时（例如Y的价格变为X的两倍），则进行买低卖高的套利操作，待价格关系恢复常态时平仓。

## 线性回归模型

线性回归是最简单、直观且常用的回归方法。我们将想要解释或预测的变量称为**因变量 Y**，将用来解释或控制的变量称为**自变量 X**。

当自变量只有一个时，称为**一元线性回归**；当自变量有多个（X₁, X₂, ..., Xₙ）时，称为**多元线性回归**。

本节我们重点讨论一元线性回归。考虑一个经典例子：工资水平 **Y** 与工作年限 **X** 之间的关系。通常认为两者正相关，工作年限越长，工资水平越高。

我们可以写出一个关系式：**Y = β₀ + β₁X**。其中，**β₀** 可理解为初始工资（当 **X=0** 时），**β₁** 表示每多工作一年，工资的平均增长额。

然而，这个关系式描述的是严格的函数关系，而现实中工资与工作年限并非一一对应，只是一种统计上的相关关系。因此，更准确的表述是工资水平的**条件期望**与工作年限成函数关系。

## 总体回归函数与样本回归函数

上一节我们提到了条件期望。严格来说，线性回归模型描述的是因变量 **Y** 的条件期望与自变量 **X** 之间的关系。

**总体回归函数** 描述了这种真实关系：**E(Y|Xᵢ) = β₀ + β₁Xᵢ**。它代表了站在“上帝视角”看到的 **X** 与 **Y** 之间的线性关系。对于任何一个具体的观测点 **i**，其实际值 **Yᵢ** 与这个条件期望的差值，用一个**随机误差项 εᵢ** 表示：**Yᵢ = β₀ + β₁Xᵢ + εᵢ**。

然而，我们无法获知总体全部数据，因此只能通过抽取样本来推断总体关系。

根据样本数据点绘制的图形称为**散点图**。通过样本点，我们可以拟合一条最优直线，称为**样本回归线**，其对应的函数为**样本回归函数**：**Ŷᵢ = β̂₀ + β̂₁Xᵢ**。

其中，**β̂₀** 和 **β̂₁** 是 **β₀** 和 **β₁** 的估计值，决定了直线的截距和斜率。对于每个样本观测点，其实际值 **Yᵢ** 与样本回归线上的预测值 **Ŷᵢ** 之间的差值称为**残差 eᵢ**：**Yᵢ = β̂₀ + β̂₁Xᵢ + eᵢ**。

## 参数估计：普通最小二乘法

现在的问题是，如何确定这条最优的样本回归线？最常用的方法是**普通最小二乘法**。

OLS的核心思想是使**残差平方和**最小，即寻找参数 **β̂₀** 和 **β̂₁**，使得 **∑eᵢ² = ∑(Yᵢ - Ŷᵢ)²** 的值达到最小。这意味着找到一条直线，使所有样本点到该直线的垂直距离的平方和最小。

以下是求解 **β̂₀** 和 **β̂₁** 的过程：
1.  设定目标函数：**Min ∑(Yᵢ - β̂₀ - β̂₁Xᵢ)²**。
2.  分别对 **β̂₀** 和 **β̂₁** 求偏导数，并令其等于零。
3.  求解方程组，得到估计量的计算公式。

最终推导出的OLS估计量为：
*   **β̂₁ = ∑(Xᵢ - X̄)(Yᵢ - Ȳ) / ∑(Xᵢ - X̄)²**
*   **β̂₀ = Ȳ - β̂₁X̄**

其中，**X̄** 和 **Ȳ** 分别是 **X** 和 **Y** 的样本均值。**β̂₁** 的公式也可以理解为 **X** 和 **Y** 的样本协方差除以 **X** 的样本方差。

**高斯-马尔可夫定理**指出，在满足一系列经典假设的条件下，OLS估计量是所有线性无偏估计量中方差最小的，即**最佳线性无偏估计**。

## 模型的基本假设

为了保证OLS估计量的优良性质，简单线性回归模型通常需要以下基本假设：
1.  **线性于参数**：模型对参数 **β₀** 和 **β₁** 是线性的，自变量本身可以是任何形式（如 **X²**, **ln(X)** 等）。
2.  **自变量非随机**：自变量 **X** 的值是给定的、可控制的，而非随机变量。
3.  **随机误差项零均值**：**E(εᵢ) = 0**。
4.  **同方差性**：所有随机误差项的方差都相同，即 **Var(εᵢ) = σ²**。
5.  **无自相关**：不同观测的随机误差项之间不相关，即 **Cov(εᵢ, εⱼ) = 0, (i ≠ j)**。
6.  **自变量与误差项不相关**：**Cov(Xᵢ, εᵢ) = 0**。
7.  **正态性假设（可选但常用）**：随机误差项服从正态分布，即 **εᵢ ~ N(0, σ²)**。此假设有助于进行严格的统计推断。

## 回归系数的显著性检验

得到回归系数 **β̂₁** 后，我们需要检验它是否显著不为零。这被称为**回归系数的显著性检验**。

**回归系数（如 β₁）** 表示 **X** 变动一单位对 **Y** 期望值的影响大小，其取值范围是全体实数。切勿将其与**相关系数**混淆，相关系数衡量的是线性相关的强度和方向，其取值范围在 **-1** 到 **1** 之间。

检验的目的是判断自变量 **X** 是否对因变量 **Y** 有显著的解释力。我们建立如下假设：
*   原假设 **H₀: β₁ = 0** （X对Y没有线性影响）
*   备择假设 **H₁: β₁ ≠ 0** （X对Y有线性影响）

若拒绝原假设，则认为 **X** 与 **Y** 之间存在显著的线性关系。

检验步骤如下：
1.  **陈述假设**：如上所述。
2.  **选择检验统计量**：在误差项方差 **σ²** 未知的情况下（通常如此），使用 **t** 统计量。可以证明，在满足模型假设的条件下，**t = (β̂₁ - 0) / SE(β̂₁)** 服从自由度为 **n-2** 的 **t** 分布。其中 **SE(β̂₁)** 是 **β̂₁** 的标准误。
3.  **确定显著性水平 α**：通常设为 0.05 或 0.01。
4.  **计算检验统计量的值**：利用样本数据计算出 **t** 值。
5.  **做出决策**：比较计算出的 **t** 值的绝对值与临界值 **t_{α/2}(n-2)**，或者计算 **p值** 并与 **α** 比较。若 |t| > 临界值 或 p值 < α，则拒绝原假设。
