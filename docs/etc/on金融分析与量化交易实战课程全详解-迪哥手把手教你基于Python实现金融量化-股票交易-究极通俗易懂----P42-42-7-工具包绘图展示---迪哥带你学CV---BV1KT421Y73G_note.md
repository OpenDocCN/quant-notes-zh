# Python金融量化实战课程：P42：42.7-工具包绘图展示 📊

在本节课程中，我们将学习如何使用量化分析工具包中的绘图功能，将计算得到的因子IC值（信息系数）进行可视化展示。通过图表，我们可以更直观地评估因子的有效性和稳定性。

上一节我们介绍了如何计算因子与未来收益率之间的相关系数（IC值）。本节中，我们来看看如何将这些数值结果绘制成图表，以便进行更深入的分析。

## 因子IC值的可视化

![](img/71bbcbbe9c76179463889b60ef934e6c_1.png)

计算得到的IC值每天都会变化。为了更清晰地观察其趋势和表现，我们需要将其绘制成时间序列图。以下是使用工具包进行绘图的具体步骤。

首先，我们需要导入绘图工具。在代码中，我们使用 `proloting` 工具包（此处应为 `alphalens` 或类似库的绘图模块，视频中口误为“PROLOATING”）来进行展示。

![](img/71bbcbbe9c76179463889b60ef934e6c_3.png)

```python
# 导入绘图模块
import alphalens.tears as tears
# 绘制IC值时间序列图
tears.plot_information_table(ic_data)
```

具体到API调用，我们需要找到绘制IC时间序列的函数。根据文档，该函数名为 `plot_ic_ts`（视频中口误为“IC1杠时间序列”）。执行该函数并传入我们之前计算好的IC数据（`ic_data`），即可生成图表。

```python
# 绘制IC时间序列图
tears.plot_ic_ts(ic_data)
```

## 图表解读与分析

图表生成后，默认会包含两条线：
*   **蓝色线**：代表每日计算的实际IC值。由于其波动范围通常较大，单独观察难以看出明显规律。
*   **绿色线**：代表IC值的滚动平均值，默认窗口期为一个月（约21个交易日）。这条线平滑了日度波动，更能反映因子的长期趋势。

图表下方还会统计并展示IC值的**均值（Mean）** 和**标准差（Std）**。此外，一个重要指标是**信息比率（Information Ratio, IR）**，其计算公式为：
**IR = Mean(IC) / Std(IC)**

信息比率用于衡量因子稳定性的程度。比值越大，说明因子的IC值越稳定（标准差相对均值更小）；比值越小，则说明因子的表现波动较大。

观察我们绘制的图表，绿色均线整体较为平稳，但数值偏小，且未显示出明显的上升或下降趋势。这表明当前测试的因子与未来收益率的相关性较弱，可能不是一个有效的预测指标。

## 工具包的其他绘图功能

![](img/71bbcbbe9c76179463889b60ef934e6c_5.png)

![](img/71bbcbbe9c76179463889b60ef934e6c_7.png)

除了IC时间序列图，量化分析工具包还提供了多种其他图表，用于全方位评估因子。以下是部分可用的绘图函数：

*   **`plot_ic_histogram`**: 绘制IC值的直方图，用于观察其分布形态。
*   **`plot_ic_qq`**: 绘制IC值的QQ图，用于检验其是否服从正态分布。
*   **`plot_quantile_statistics_table`**: 展示因子分位数相关的统计表格。
*   **`plot_monthly_ic_heatmap`**: 绘制月度IC值热力图，用颜色深浅直观展示不同月份的表现。

![](img/71bbcbbe9c76179463889b60ef934e6c_9.png)

这些函数在工具的API文档中均有列出。在实际进行深入的因子研究与策略构建时，可以灵活调用这些功能进行全面的分析。
