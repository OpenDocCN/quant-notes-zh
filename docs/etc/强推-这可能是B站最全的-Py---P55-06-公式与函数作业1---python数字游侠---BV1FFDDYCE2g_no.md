在本节课中，我们将学习如何利用Excel中的公式与函数，特别是混合引用的技巧，来构建一个经典的九九乘法表。我们将从基础公式开始，逐步引入IF函数，最终完成一个只显示下三角区域的完整乘法表。

## 第一题：构建九九乘法表

![](img/80484602d19e9c32e1d2da3d1e4d8056_1.png)

![](img/80484602d19e9c32e1d2da3d1e4d8056_3.png)

上一节我们介绍了公式与函数的基本概念，本节中我们来看看如何应用这些知识解决实际问题。第一道题考察的是公式中的引用操作，目标是创建一个九九乘法表。

![](img/80484602d19e9c32e1d2da3d1e4d8056_5.png)

### 理解需求与初步构建

乘法表的结构是：行方向（1到9）与列方向（1到9）的数字相乘。例如，第一行是1×1, 1×2, ..., 1×9。

以下是构建乘法表的第一步操作：

1.  在单元格B31中输入初始公式 `=B$30*A31`。这个公式表示用B30单元格的值（列标题1）乘以A31单元格的值（行标题1）。
2.  公式中，`B$30` 使用了混合引用，在行号30前加上了美元符号`$`。这表示在向下拖动填充时，行号30保持不变，始终引用第一行的乘数。
3.  按下回车键后，向下拖动B31单元格的填充柄，即可得到第一列的结果：1×1, 1×2, ..., 1×9。

### 应用混合引用完成表格

初步构建只完成了第一列，我们需要将公式向右拖动以填充整个表格。

![](img/80484602d19e9c32e1d2da3d1e4d8056_7.png)

以下是完成整个乘法表的关键步骤：

![](img/80484602d19e9c32e1d2da3d1e4d8056_9.png)

1.  观察向右拖动时的需求：我们希望列标（如B、C、D）能随之变化，以引用C30、D30等单元格的值。因此，需要将`B$30`中的列标B前的美元符号`$`去掉，改为`B$30`。
2.  同时，在向右拖动时，我们不希望公式中`A31`的列标A发生变化（始终引用A列的行标题），但向下拖动时，希望其行号能变化。因此，需要在`A31`的列标A前加上美元符号，改为`$A31`。
3.  因此，B31单元格的最终公式应为：`=B$30*$A31`。
4.  将此公式同时向下和向右拖动填充至整个9x9区域，即可生成完整的九九乘法表。

**核心公式**：
`=B$30*$A31`

### 使用IF函数优化显示效果

完整的乘法表包含了上三角区域（如2×1）和下三角区域（如1×2），通常我们只需要显示下三角部分（即行号大于等于列号的部分）。

以下是使用IF函数优化显示的逻辑与步骤：

![](img/80484602d19e9c32e1d2da3d1e4d8056_11.png)

1.  我们需要一个条件来判断当前单元格是否应该显示。逻辑是：当A列的行标题数字**大于等于**B列的列标题数字时，才进行计算和显示。
2.  在B31单元格中，将公式修改为：`=IF($A31>=B$30, B$30*$A31, "")`。
    *   `$A31>=B$30`：判断条件。`$A31`锁定列，代表当前行号；`B$30`锁定行，代表当前列号。
    *   `B$30*$A31`：如果条件为真（行号≥列号），则执行乘法运算。
    *   `""`：如果条件为假（行号<列号），则返回空字符串，单元格显示为空白。
3.  将此包含IF函数的公式向下和向右拖动填充，即可得到一个只显示下三角区域的、更标准的九九乘法表。

![](img/80484602d19e9c32e1d2da3d1e4d8056_12.png)

**优化后公式**：
`=IF($A31>=B$30, B$30*$A31, "")`
