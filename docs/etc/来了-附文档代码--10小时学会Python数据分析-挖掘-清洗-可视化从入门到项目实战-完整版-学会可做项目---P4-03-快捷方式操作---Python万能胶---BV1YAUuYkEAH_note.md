# Python数据分析：P4：03 快捷方式操作 🚀

在本节课中，我们将要学习Excel中一系列实用的键盘快捷方式。掌握这些快捷方式能显著提升数据处理和分析的工作效率，让你在操作时更加得心应手。

## 常用基础快捷方式

上一节我们介绍了学习快捷方式的重要性，本节中我们来看看最基础、最常用的一组快捷方式。这些操作是日常工作的基石。

![](img/dbd3a9d6d21e57d966e28446fbdd4866_1.png)

以下是几个最核心的快捷操作：

*   **`Ctrl + A`**：全选。此操作要求所选区域必须是**连续**的数据区域。如果数据中存在空行或空列，则只会选中连续部分。
*   **`Ctrl + Z`**：撤销上一步操作。
*   **`Ctrl + X` / `Ctrl + C` / `Ctrl + V`**：分别对应剪切、复制和粘贴。
*   **`Ctrl + B`**：将选定文本加粗。
*   **`Ctrl + S`**：保存当前工作簿。
*   **`Ctrl + F`**：打开“查找”对话框。`F` 代表 `Find`。
*   **`Ctrl + H`**：打开“替换”对话框。也可以先按 `Ctrl + F` 查找，再切换到“替换”标签页。
*   **`Alt + Enter`**：在单元格内强制换行。这与“自动换行”不同，强制换行允许你在任意位置手动开始新的一行。

*   **`Home`**：将光标移动到当前行的开头。
*   **`Ctrl + Home`**：快速跳转到工作表的 `A1` 单元格（起始位置）。
*   **`Ctrl + Shift + Home`**：选中从当前单元格到 `A1` 单元格的连续区域。
*   **`Ctrl + Shift + ↓ / ↑ / → / ←`**：向指定方向快速选择连续的数据区域，直到遇到空白单元格为止。
*   **`Ctrl + ↓ / ↑ / → / ←`**：将光标快速跳转到当前数据区域在相应方向的边界。
*   **`Ctrl + Page Up` / `Ctrl + Page Down`**：在多个工作表之间切换。

以下是一些能够提升操作流畅度的实用快捷方式：

![](img/dbd3a9d6d21e57d966e28446fbdd4866_34.png)

![](img/dbd3a9d6d21e57d966e28446fbdd4866_36.png)

*   **`Ctrl + N`**：在已打开一个Excel程序的前提下，新建一个空白工作簿。
*   **`Ctrl + 1`**：打开“设置单元格格式”对话框，用于调整数字、对齐、字体、边框等样式。
*   **`Ctrl + G`** 或 **`F5`**：打开“定位”对话框，用于快速跳转到特定单元格或选择特定类型的单元格（如公式、空值等）。
*   **`Ctrl + O`**：打开“打开文件”对话框。`O` 代表 `Open`。
*   **`Ctrl + P`**：打开“打印”设置界面。
*   **`Ctrl + R`**：向右填充，将选中区域最左侧单元格的内容或格式复制到右侧选中的单元格。`R` 代表 `Right`。
*   **`Ctrl + D`**：向下填充，将选中区域最上方单元格的内容或格式复制到下方选中的单元格。`D` 代表 `Down`。
*   **`Ctrl + ;`**：在活动单元格中输入当前日期。
*   **`Ctrl + 9`**：隐藏选定的行。
*   **`Ctrl + 0`**：隐藏选定的列。
*   **`Ctrl + W`**：关闭当前工作簿窗口（需谨慎使用，可能不提示保存）。
*   **`F2`**：进入活动单元格的编辑模式。
*   **`F4`**：重复上一步操作（主要针对插入/删除行/列、设置格式等鼠标操作）。在编辑公式时，`F4` 键可以在相对引用、绝对引用和混合引用之间循环切换。
*   **`F12`**：打开“另存为”对话框。
