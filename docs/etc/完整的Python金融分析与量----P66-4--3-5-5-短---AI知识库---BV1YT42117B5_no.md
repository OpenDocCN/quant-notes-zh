## 策略概述与核心概念

![](img/c718b101c2084439c16eb2c4e99db7a2_7.png)

我们将分析一个股票序列，通过计算其短期移动平均线和长期移动平均线，来判断买入和卖出的时机。这个策略的核心在于识别两条均线的交叉点。

![](img/c718b101c2084439c16eb2c4e99db7a2_9.png)

*   **黄金交叉**：短期均线从下方向上穿越长期均线，通常被视为**买入信号**，预示着价格可能开始上涨。
*   **死亡交叉**：短期均线从上方向下穿越长期均线，通常被视为**卖出信号**，预示着价格可能开始下跌。

## 计算短期与长期移动平均线

首先，我们需要选择目标股票并计算其移动平均线。我们以苹果公司（AAPL）的股票数据为例。

以下是计算步骤：

![](img/c718b101c2084439c16eb2c4e99db7a2_11.png)

![](img/c718b101c2084439c16eb2c4e99db7a2_13.png)

1.  **导入数据**：加载苹果公司的历史股价数据。
2.  **计算短期均线**：我们选择一个较短的窗口期来计算移动平均值。通常，5日（一周交易日）或10日均线较为常见。由于我们的数据时间跨度较长，这里选择10日窗口。
3.  **计算长期均线**：我们选择一个较长的窗口期。通常，20日（约一个月交易日）或30日均线较为常见。这里我们选择30日窗口。

![](img/c718b101c2084439c16eb2c4e99db7a2_15.png)

对应的Python代码如下：

```python
# 假设 df 是包含苹果股价的DataFrame，其中‘AAPL’是股价列
# 计算10日短期移动平均线 (M1)
df[‘M1’] = df[‘AAPL’].rolling(window=10).mean()

![](img/c718b101c2084439c16eb2c4e99db7a2_17.png)

![](img/c718b101c2084439c16eb2c4e99db7a2_19.png)

# 计算30日长期移动平均线 (M2)
df[‘M2’] = df[‘AAPL’].rolling(window=30).mean()
```

![](img/c718b101c2084439c16eb2c4e99db7a2_21.png)

执行上述代码后，我们的DataFrame中会新增两列：`M1`（10日均线）和`M2`（30日均线）。我们可以使用 `df.head()` 来查看前几行数据，确认计算成功。

![](img/c718b101c2084439c16eb2c4e99db7a2_23.png)

![](img/c718b101c2084439c16eb2c4e99db7a2_25.png)

*   当**绿色线（M1）向上穿越红色线（M2）**时，形成**黄金交叉**，提示可能上涨。

## 使用信号指标进行增强分析

手动观察交叉点不够精确。我们可以创建一个信号指标，来明确标记短期均线是高于还是低于长期均线。

我们定义：
*   当 `M1 > M2` 时，标记为 **1**（看多信号）。
*   当 `M1 < M2` 时，标记为 **-1**（看空信号）。

![](img/c718b101c2084439c16eb2c4e99db7a2_39.png)

![](img/c718b101c2084439c16eb2c4e99db7a2_41.png)

以下是使用NumPy的 `where` 函数创建信号列的代码：

![](img/c718b101c2084439c16eb2c4e99db7a2_43.png)

```python
import numpy as np

![](img/c718b101c2084439c16eb2c4e99db7a2_45.png)

# 创建信号列 ‘position‘
df[‘position’] = np.where(df[‘M1’] > df[‘M2’]， 1， -1)
```

![](img/c718b101c2084439c16eb2c4e99db7a2_47.png)

现在，我们可以将这个信号指标也绘制在图上。由于信号值（1和-1）与股价数值范围差异巨大，我们使用右侧的Y轴来单独显示它。

![](img/c718b101c2084439c16eb2c4e99db7a2_49.png)

以下是增强版绘图代码：

```python
fig， ax1 = plt.subplots(figsize=(14， 7))

# 左侧Y轴：绘制股价和均线
ax1.plot(df.index， df[‘AAPL’]， label=‘AAPL Price’， alpha=0.5)
ax1.plot(df.index， df[‘M1’]， label=‘10-day MA (M1)‘， color=‘green‘)
ax1.plot(df.index， df[‘M2’]， label=‘30-day MA (M2)‘， color=‘red‘)
ax1.set_xlabel(‘Date‘)
ax1.set_ylabel(‘Price‘)
ax1.legend(loc=‘upper left‘)

![](img/c718b101c2084439c16eb2c4e99db7a2_51.png)

# 创建右侧Y轴，用于绘制信号
ax2 = ax1.twinx()
ax2.plot(df.index， df[‘position’]， label=‘Signal (Position)‘， color=‘purple‘， linestyle=‘--‘)
ax2.set_ylabel(‘Signal (1/-1)‘)
ax2.set_ylim([-1.5， 1.5])
ax2.legend(loc=‘upper right‘)

![](img/c718b101c2084439c16eb2c4e99db7a2_53.png)

plt.title(‘AAPL Price， Moving Averages， and Trading Signal‘)
plt.show()
```

在这张增强图中：
*   紫色虚线代表我们的交易信号。
*   当信号线在 **+1** 时，表示处于“黄金交叉”后的看多区域。
*   当信号线在 **-1** 时，表示处于“死亡交叉”后的看空区域。
*   信号线从 **-1 跳变到 +1** 的时刻，就是**黄金交叉**点。
*   信号线从 **+1 跳变到 -1** 的时刻，就是**死亡交叉**点。
