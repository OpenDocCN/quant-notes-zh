在本节课中，我们将学习如何使用Python获取OKX交易所的比特币K线数据，并计算其移动平均线。整个过程仅需8行核心代码，非常适合初学者入门量化分析。

## 概述

![](img/4559ad512a3c1a863d8ceb013989a4cb_2.png)

我们将通过三个步骤完成目标：首先导入必要的库，然后获取交易所的K线数据，最后利用技术分析库计算移动平均线。下面我们开始逐步讲解。

## 代码讲解

以下是实现该功能所需的全部核心代码。

### 第一步：导入必要的库

首先，我们需要导入三个关键的Python库。

```python
import requests
import pandas as pd
import talib
```

*   **`requests`**：用于向交易所的API发送HTTP请求，以获取数据。
*   **`pandas`**：一个强大的数据处理库，我们将用它来整理和操作获取到的数据。
*   **`talib`**：技术分析库，它封装了超过200种技术指标的计算公式，可以让我们用一行代码计算出MA、MACD、RSI等复杂指标。

### 第二步：获取K线数据

上一节我们介绍了所需的工具库，本节中我们来看看如何获取数据。我们需要构造一个API请求地址，并向交易所发送请求。

```python
url = ‘https://www.okx.com/api/v5/market/candles?instId=BTC-USDT-SWAP&bar=1D&limit=300‘
response = requests.get(url)
data = response.json()
```

*   **`url`**：这是OKX交易所提供的K线数据API地址。其中包含三个关键参数：
    *   `instId=BTC-USDT-SWAP`：交易对代码，这里指比特币永续合约。你可以将其替换为任何交易所支持的币种，例如`ETH-USDT-SWAP`。
    *   `bar=1D`：K线周期，`1D`代表日线。你可以根据需要改为`1H`（1小时）、`15m`（15分钟）等。
    *   `limit=300`：获取的K线数量，最大值为300。
*   **`requests.get(url)`**：发送GET请求到指定的URL。
*   **`.json()`**：将服务器返回的数据解析为Python的字典或列表格式。

获取到的原始数据是一个列表，其中每条K线也是一个列表，包含时间、开盘价、最高价、最低价、收盘价等信息。但它的列名是数字，不便于阅读。

### 第三步：整理数据

为了更清晰地查看数据，我们使用pandas将其转换为`DataFrame`，并重命名列。

```python
df = pd.DataFrame(data[‘data’])
df.columns = [‘timestamp’, ‘open’, ‘high’, ‘low’, ‘close’, ‘vol’, ‘volCcy’, ‘confirm’]
```

运行后，数据将变得清晰易读。你会注意到最后一根K线（最新的那根）的`confirm`状态为`0`，表示该日线尚未走完；而历史K线的状态为`1`，表示已经收盘。

### 第四步：计算移动平均线

数据准备就绪后，现在进入核心环节：计算技术指标。得益于`talib`库，计算移动平均线变得非常简单。

```python
df[‘ma120’] = talib.MA(df[‘close’], timeperiod=120)
```

![](img/4559ad512a3c1a863d8ceb013989a4cb_4.png)

*   **`talib.MA()`**：`talib`库中计算移动平均的函数。
*   **`df[‘close’]`**：传入收盘价序列作为计算基础。
*   **`timeperiod=120`**：指定计算120周期的移动平均线，即MA120。

执行这行代码后，数据框`df`中就会新增一列`ma120`，里面就是计算好的120日移动平均值。

## 结果验证与总结

让我们来验证一下计算结果的准确性。由于最后一根日K线尚未收盘，我们选取倒数第二根K线的MA120值进行对比。

程序计算出的值为 **63265**。我们回到行情软件中查看同一根K线的MA120数值，结果完全一致。这证明了我们代码和`talib`库计算的准确性。

此外，MA120在长周期行情分析中常被视为重要的支撑或压力位。从历史走势图中可以看到，价格在触及MA120时，经常会发生趋势反转或延续，这体现了该指标的实用性。
