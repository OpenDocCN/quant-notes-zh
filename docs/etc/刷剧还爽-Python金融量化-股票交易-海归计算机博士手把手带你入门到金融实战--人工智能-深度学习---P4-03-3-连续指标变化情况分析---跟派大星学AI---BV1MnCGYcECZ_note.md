## 简单增长率累加的问题

上一节我们介绍了如何计算增长率。本节中我们来看看，如果我们想观察一个连续时间段内的整体增长，能否将每天的增长率简单累加。

![](img/ff67b8695ee71af7e3ee3f3589836b86_4.png)

例如，有一只股票：
*   第一天价格：100元
*   第二天价格：50元
*   第三天价格：75元

以下是计算过程：
1.  第二天增长率 = (50 - 100) / 100 = -0.5
2.  第三天增长率 = (75 - 50) / 50 = 0.5
3.  增长率累加和 = (-0.5) + 0.5 = 0

累加结果为0，似乎表示股票从第一天到第三天“没有涨跌”。但观察原始数据，股价从100元跌至75元，实际上是下跌的。这说明**简单的增长率累加无法正确反映连续时间内的整体变化**。

## 使用对数计算连续变化

![](img/ff67b8695ee71af7e3ee3f3589836b86_6.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_8.png)

对于连续时间的计算，我们需要借助数学中的对数公式。

![](img/ff67b8695ee71af7e3ee3f3589836b86_10.png)

核心思想是：将对数形式的日收益率进行累加，其本质是计算首日与末日价格的比值。

**公式推导如下：**

假设我们计算连续两天的对数收益率：
`log(第二天价格 / 第一天价格) + log(第三天价格 / 第二天价格)`

![](img/ff67b8695ee71af7e3ee3f3589836b86_12.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_14.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_16.png)

根据对数运算法则 `log(a) + log(b) = log(a * b)`，上式可转化为：
`log( (第二天价格/第一天价格) * (第三天价格/第二天价格) ) = log(第三天价格 / 第一天价格)`

![](img/ff67b8695ee71af7e3ee3f3589836b86_18.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_20.png)

最终，**对数收益率的累加和等价于计算末期价格与初期价格比值的对数**。这完美地反映了从期初到期末的整体变化。

![](img/ff67b8695ee71af7e3ee3f3589836b86_22.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_24.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_26.png)

## 在Python中实现连续变化分析

![](img/ff67b8695ee71af7e3ee3f3589836b86_28.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_30.png)

我们将通过代码实现上述过程，计算“期初1单位投资相当于期末多少价值”。

以下是实现步骤：

![](img/ff67b8695ee71af7e3ee3f3589836b86_32.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_34.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_36.png)

首先，计算每日的对数收益率。这需要用到`shift()`函数来获取前一天的数据。

```python
# 计算每日的对数收益率
# data['Adj Close'] 是调整后的收盘价序列
returns = np.log(data['Adj Close'] / data['Adj Close'].shift(1))
```

![](img/ff67b8695ee71af7e3ee3f3589836b86_38.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_40.png)

接下来，我们需要将对数收益率进行累积求和，这可以通过`cumsum()`函数实现。

![](img/ff67b8695ee71af7e3ee3f3589836b86_42.png)

```python
# 对对数收益率进行累积求和
cumulative_returns = returns.cumsum()
```

![](img/ff67b8695ee71af7e3ee3f3589836b86_44.png)

最后，为了得到以初期价格为基准（例如，初期设为1）的累计价值序列，我们需要将对数累积收益率还原为实际数值。

![](img/ff67b8695ee71af7e3ee3f3589836b86_46.png)

```python
# 将对数累积收益率还原为实际累积价值（期初基准为1）
cumulative_value = np.exp(cumulative_returns)
```

![](img/ff67b8695ee71af7e3ee3f3589836b86_48.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_50.png)

现在，`cumulative_value`这个序列就清晰地展示了：假设在时间序列起点投入1元钱，到每一个后续时间点，这笔投资的价值变成了多少。我们可以将其可视化。

![](img/ff67b8695ee71af7e3ee3f3589836b86_52.png)

```python
# 绘制累积价值曲线
cumulative_value.plot(figsize=(10, 6))
plt.title('累积投资价值变化 (期初基准为1)')
plt.ylabel('累积价值')
plt.xlabel('日期')
plt.show()
```

![](img/ff67b8695ee71af7e3ee3f3589836b86_54.png)

通过图表，我们可以直观地比较不同股票在相同时间段内的整体增长表现。例如，可能会发现期初的1元钱投资于某公司，到期末价值可能增长到10元；而投资于另一家公司，可能只增长到4元。

## 本节总结

![](img/ff67b8695ee71af7e3ee3f3589836b86_60.png)

本节课中我们一起学习了金融时间序列中连续变化情况的分析方法。

![](img/ff67b8695ee71af7e3ee3f3589836b86_62.png)

![](img/ff67b8695ee71af7e3ee3f3589836b86_64.png)

1.  **识别了简单方法的局限**：我们首先认识到，将每日增长率简单累加无法正确衡量连续时间内的整体变化。
2.  **引入了核心数学工具**：为了解决这个问题，我们引入了对数计算。通过对数收益率的累加，可以等价地得到期初与期末的整体关系。
3.  **掌握了Python实现**：我们使用`shift()`函数计算差值，用`np.log()`计算对数收益率，用`cumsum()`进行累积求和，最后用`np.exp()`还原为易于理解的累积价值序列。
4.  **完成了实用分析**：最终，我们能够计算出并可视化“期初1单位投资在后续各时间点的价值”，这是一个在量化分析中非常实用且直观的指标。
