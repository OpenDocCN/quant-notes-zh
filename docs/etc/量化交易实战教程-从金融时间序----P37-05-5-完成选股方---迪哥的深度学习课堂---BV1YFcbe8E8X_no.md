上一节我们介绍了如何为单个因子进行打分。本节中，我们来看看如何将多个因子的打分结果合并，并基于综合得分完成最终的选股排名。

## 拼接数据表

![](img/b48f3873d04d25e74b10cb03e3d8e316_4.png)

首先，我们需要将两个包含不同因子打分结果的DataFrame进行拼接操作。拼接操作使用`pd.concat`函数，将一个DataFrame与另一个DataFrame连接起来。

![](img/b48f3873d04d25e74b10cb03e3d8e316_6.png)

以下是拼接操作的代码：

![](img/b48f3873d04d25e74b10cb03e3d8e316_8.png)

```python
# 假设 df1 和 df2 是两个需要拼接的DataFrame
df_combined = pd.concat([df1, df2], axis=1)
```

![](img/b48f3873d04d25e74b10cb03e3d8e316_10.png)

拼接完成后，我们得到了一个包含所有因子数据的大表。为了后续计算，我们为这个合并后的DataFrame指定一个新的名称，例如`df_rank`。

```python
df_rank = df_combined
```

## 计算综合得分并排序

![](img/b48f3873d04d25e74b10cb03e3d8e316_12.png)

现在，我们有了包含所有因子打分的大表，但还没有综合排名信息。接下来，我们需要新建一列来存放每个股票的综合得分。

首先，我们初始化一个全为零的列，名为“得分值”。

```python
import numpy as np
# 假设有300个样本（股票）
df_rank['得分值'] = np.zeros((300, 1))
```

初始化后，我们需要实际计算综合得分。综合得分是各个因子得分的总和。假设我们有六个因子指标，我们需要对这六个指标进行求和。

以下是计算总分并排序的步骤：

![](img/b48f3873d04d25e74b10cb03e3d8e316_14.png)

1.  **计算总分**：对每个股票的所有因子得分进行求和。
2.  **排序**：根据总分进行降序排列，以找到得分最高的股票。

以下是相应的代码实现：

```python
# 计算所有因子列的总和，假设因子列名为‘factor1’， ‘factor2’...‘factor6’
factor_columns = ['factor1', 'factor2', 'factor3', 'factor4', 'factor5', 'factor6']
df_rank['总分'] = df_rank[factor_columns].sum(axis=1)

# 根据‘总分’列进行降序排序
df_rank_sorted = df_rank.sort_values(by='总分', ascending=False)
```

## 筛选前十名股票

![](img/b48f3873d04d25e74b10cb03e3d8e316_16.png)

排序完成后，我们只需要关注排名结果。我们的目标是获取综合得分排名前十的股票代码。

![](img/b48f3873d04d25e74b10cb03e3d8e316_18.png)

以下是获取前十名股票列表的代码：

![](img/b48f3873d04d25e74b10cb03e3d8e316_20.png)

```python
# 获取排序后DataFrame的索引（即股票代码），并转换为列表
top_10_stocks = df_rank_sorted.index[:10].tolist()
```

![](img/b48f3873d04d25e74b10cb03e3d8e316_22.png)

至此，`get_top_stocks`函数的核心逻辑就完成了。执行一次这个函数，它就会基于当前所有因子的数据，返回当天综合得分排名前十的股票列表。

![](img/b48f3873d04d25e74b10cb03e3d8e316_24.png)
