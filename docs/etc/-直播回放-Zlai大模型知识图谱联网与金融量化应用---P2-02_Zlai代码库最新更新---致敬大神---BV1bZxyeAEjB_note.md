## 新增模型详解

![](img/1aee77c78d413cd8213a840219b6ac3c_35.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_37.png)

本次更新在模型支持方面有显著增强，引入了多种专长不同的模型。

以下是新增的具体模型及其特点：

![](img/1aee77c78d413cd8213a840219b6ac3c_39.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_41.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_43.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_45.png)

1.  **千问VL多模态模型**：增加了通义千问视觉语言（VL）模型的两个尺寸版本，用于处理图像和文本的多模态任务。
2.  **LongAlpaca长文本生成模型**：增加了智谱开源的长文本生成模型，包括ChatGLM和Llama3.1两个版本。该模型专为生成超长文本设计，可输出数千甚至数万字，解决了普通模型输出长度限制（通常在500-800字）的问题。
3.  **MiniCPM小钢炮系列模型**：增加了轻量级但性能强劲的模型系列。
4.  **LongSee长文本引用模型**：增加了清华大学开源的长文本精准引用模型。该模型能精确定位并引用原文中的信息，使输出结果更忠于提供的材料，在需要严格引用的任务（如RAG）中表现更佳。
5.  **千问2.5全系列模型**：完整增加了千问2.5系列的所有尺寸模型。
6.  **GTE-Embedding-v3向量模型**：新增了效果较好的文本向量化模型，它支持OpenAI的嵌入接口规范，部署后可使用OpenAI SDK进行调用。

![](img/1aee77c78d413cd8213a840219b6ac3c_51.png)

## 新增工具与功能

![](img/1aee77c78d413cd8213a840219b6ac3c_61.png)

除了模型，本次更新还大幅丰富了可用的工具集，特别是在金融数据领域。

![](img/1aee77c78d413cd8213a840219b6ac3c_67.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_69.png)

以下是新增的工具和功能改进：

![](img/1aee77c78d413cd8213a840219b6ac3c_73.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_75.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_77.png)

1.  **Agent示例**：增加了基于知识图谱的智能体应用示例，将在后续章节详细介绍。
2.  **金融数据工具**：
    *   **证券研报工具**：支持下载指定板块、行业、地区的券商研报，供大模型进行总结分析。
    *   **财经新闻工具**：提供各类财经快讯、公司新闻等数据。
    *   **宏观经济数据工具**：提供CPI、PPI、GDP、PMI等多种宏观经济指标数据。
    *   **股票预测排行工具**：提供网络上的股票预测排名数据。
    *   **股票财务报表工具**：提供详细的上市公司财务数据。
3.  **Streamlit集成优化**：在消息机制中增加了 `streamlit_show` 方法，允许在Streamlit前端页面上直接、便捷地展示对话、图片、音频或图表等内容。
4.  **性能提升**：调整了大模型流式输出的响应机制，提升了文本的生成和返回速度。

## 项目目标与使用

![](img/1aee77c78d413cd8213a840219b6ac3c_93.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_95.png)

有观众询问该项目是否仅仅是模型的堆砌。实际上，整合模型是达成最终目标的第一步。

![](img/1aee77c78d413cd8213a840219b6ac3c_99.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_101.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_103.png)

![](img/1aee77c78d413cd8213a840219b6ac3c_105.png)

项目的核心目标分为两步：
1.  **模型测试与评估**：通过统一接口方便地调用和测试不同模型，了解它们在各种任务上的性能差异，筛选出优质模型。
2.  **构建决策辅助系统**：让这些模型能够灵活调用各类工具（如金融数据工具），对信息进行综合处理与分析，最终为用户提供财务决策方面的参考。
