## 逻辑函数概述

上一节我们介绍了函数学习的基本思路，即了解函数的应用场景和参数组成。本节中我们来看看逻辑函数。逻辑函数用于执行条件判断，其返回结果为TRUE或FALSE。我们首先学习的是IF函数。

## IF函数：基础条件判断

IF函数是最基础的逻辑判断函数。其作用类似于Python中的`if-else`语句：给定一个条件，如果条件成立则返回一个值，否则返回另一个值。

**函数公式**：
`=IF(条件, 条件成立时返回的值, 条件不成立时返回的值)`

我们通过一个实例来理解。假设需要根据销售人员的销售额计算提成比例：销售额大于等于1500时，提成比例为10%；否则为8%。

以下是具体操作步骤：

1.  在目标单元格输入等号`=`，开始输入函数。
2.  输入函数名`IF`，可以手动输入，也可以输入`I`后按`Tab`键自动补全并添加左括号。
3.  输入函数的三个参数，用英文逗号隔开：
    *   **条件**：选择销售额单元格，并输入 `>=1500`。
    *   **条件成立返回值**：输入 `10%`。
    *   **条件不成立返回值**：输入 `8%`。
4.  输入右括号`)`，按`Enter`键完成。

**代码示例**：
`=IF(B2>=1500, 10%, 8%)`

> **注意**：在向下拖动填充公式时，如果提成比例是固定值（如10%所在的单元格），需要使用绝对引用（按`F4`键添加`$`符号），以防止引用随拖动而改变。

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_3.png)

计算出提成比例后，提成金额的计算就很简单了，等于提成比例乘以销售额。

## 多条件判断：IF与AND、OR的嵌套

在实际业务中，判断条件往往不止一个。这时就需要将IF函数与AND（与）、OR（或）函数结合使用。

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_5.png)

### AND函数：所有条件必须同时成立

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_7.png)

AND函数用于判断多个条件是否**同时**为真。只有所有条件都成立，AND函数才返回TRUE。

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_9.png)

**函数公式**：
`=AND(条件1, 条件2, ...)`

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_11.png)

假设新的奖金规则是：员工级别为“A类”**且**工作年限大于等于10年，奖金为5000，否则为0。

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_13.png)

以下是实现方法：

1.  使用IF函数作为外层框架。
2.  将AND函数作为IF的**条件**参数。
3.  在AND函数内，用逗号分隔两个条件：`级别单元格="A类"` 和 `工作年限单元格>=10`。
4.  在IF函数中指定返回值：条件成立（AND返回TRUE）为5000，否则为0。

**代码示例**：
`=IF(AND(C2="A类", D2>=10), 5000, 0)`

> **注意**：在公式中引用文本（如“A类”）时，必须使用英文双引号`""`括起来。

### OR函数：任一条件成立即可

OR函数用于判断多个条件中是否**至少有一个**为真。只要有一个条件成立，OR函数就返回TRUE。

**函数公式**：
`=OR(条件1, 条件2, ...)`

假设奖金规则变为：员工级别为“A类”**或**客户评价等于5，奖金为5000，否则为0。

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_15.png)

实现方法与AND类似，只是将AND替换为OR。

**代码示例**：
`=IF(OR(C2="A类", E2=5), 5000, 0)`

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_17.png)

## IF函数的嵌套：处理多阶段条件

对于同一列数据需要划分多个阶段进行判断的情况（例如，根据分数段评定等级），可以使用IF函数的嵌套。

假设根据客户评价发放奖金：
*   评价 >= 4：奖金5000
*   3 <= 评价 < 4：奖金3000
*   评价 < 3：奖金0

这需要两层IF判断：

1.  第一层IF判断是否`>=4`，如果是则返回5000。
2.  如果否（即`<4`），则进入第二层IF，判断是否`>=3`。
3.  在第二层IF中，如果`>=3`则返回3000，否则（即`<3`）返回0。

**代码示例**：
`=IF(E2>=4, 5000, IF(E2>=3, 3000, 0))`

> **关键点**：在嵌套的IF函数中，后续的条件默认是在上一层条件**不成立**的前提下进行判断的。因此，第二层只需写`>=3`，而无需重复写`<4`的条件。

## 总结

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_19.png)

本节课中我们一起学习了Excel逻辑函数的核心用法。

*   **IF函数**是逻辑判断的基础，结构为`IF(条件, 真值, 假值)`。
*   **AND与OR函数**常与IF嵌套，用于处理多列数据的复合条件判断：AND要求所有条件同时满足，OR则只需满足其一。
*   **IF函数嵌套**用于处理对同一数据的多阶段、阶梯式判断。

![](img/ca8199fb5fe6db3d4ea8146e33bdd43f_21.png)

逻辑函数是数据分析和业务判断的基石，理解并熟练运用它们，能为后续更复杂的数据处理打下坚实基础。
