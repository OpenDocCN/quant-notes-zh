## 文本函数“三剑客”：LEFT, RIGHT, MID

文本函数主要针对文本字符串进行操作。例如，身份证号码是一个18位的文本字符串，其中包含了省份编号、出生日期和性别等信息。我们可以使用文本函数从中提取出所需的部分。

以下是三个核心的文本截取函数：

![](img/5eef5f422f35d0a7fd2224cf13d58518_5.png)

![](img/5eef5f422f35d0a7fd2224cf13d58518_7.png)

*   **LEFT函数**：从文本字符串的**左侧**开始提取指定数量的字符。
    *   **公式**：`=LEFT(text, [num_chars])`
    *   **参数说明**：
        *   `text`：需要处理的原始文本。
        *   `[num_chars]`：指定要从左侧开始提取的字符数量。一个数字、一个汉字或一个英文字母均被视为一个字符。

![](img/5eef5f422f35d0a7fd2224cf13d58518_9.png)

*   **RIGHT函数**：从文本字符串的**右侧**开始提取指定数量的字符。
    *   **公式**：`=RIGHT(text, [num_chars])`
    *   **参数说明**：参数含义与LEFT函数相同，只是提取方向改为从右侧开始。

![](img/5eef5f422f35d0a7fd2224cf13d58518_11.png)

*   **MID函数**：从文本字符串的**指定位置**开始提取指定数量的字符。
    *   **公式**：`=MID(text, start_num, num_chars)`
    *   **参数说明**：
        *   `text`：需要处理的原始文本。
        *   `start_num`：开始提取的起始位置（在Excel中，位置计数从1开始，而非0）。
        *   `num_chars`：指定要提取的字符数量。

> **注意**：在输入函数时，所有的逗号 `,` 和括号 `()` 都必须在英文输入法状态下输入，否则会导致公式错误。

### LEN函数：测量文本长度

![](img/5eef5f422f35d0a7fd2224cf13d58518_19.png)

LEN函数用于返回文本字符串中的字符个数。在数据清洗中，它常用来检查数据是否符合固定长度要求（如身份证号应为18位，手机号为11位）。

*   **公式**：`=LEN(text)`
*   **参数说明**：
    *   `text`：需要计算长度的文本。
*   **应用示例**：使用`=LEN(身份证号单元格)`，若结果不为18，则说明该身份证号码可能存在录入错误。

![](img/5eef5f422f35d0a7fd2224cf13d58518_21.png)

### TEXT函数：转换数值格式

![](img/5eef5f422f35d0a7fd2224cf13d58518_22.png)

![](img/5eef5f422f35d0a7fd2224cf13d58518_24.png)

TEXT函数可以将数值（包括日期和时间）转换为按指定格式显示的文本。它在统一数据展示形式时非常有用。

![](img/5eef5f422f35d0a7fd2224cf13d58518_26.png)

*   **公式**：`=TEXT(value, format_text)`
*   **参数说明**：
    *   `value`：需要转换格式的数值。
    *   `format_text`：用双引号括起的格式代码，用于定义转换后的显示样式。
*   **应用示例**：将日期转换为“YYYY-MM-DD”格式，公式为：`=TEXT(日期单元格, "yyyy-mm-dd")`。

> **提示**：`format_text`参数可以使用自定义格式代码，实现更个性化的显示需求，但这部分内容较为复杂，初学者了解其基础用法即可。
