## 概述
在本节课中，我们将学习量化交易的核心概念。我们将探讨量化交易的本质、它与传统交易方式的区别，以及从事量化交易所需要掌握的核心技能。课程将避免冗长的历史背景介绍，直接聚焦于量化交易的实际运作方式。

## 什么是量化交易？
上一节我们介绍了课程概述，本节中我们来看看量化交易究竟做了一件什么事。

传统炒股需要投资者长时间守在电脑前，紧盯市场动态和各种指标。即便如此，投资者也很难持续盈利。原因主要有两点：第一，人类决策具有主观性，可能导致判断不准确；第二，个人的精力、时间和计算能力有限，无法全面分析大量股票或跨越十年以上的历史数据。

量化交易的核心目标同样是赚钱，但方法不同。它将具体的交易策略交给计算机来制定和执行。例如，我们可以让计算机基于历史数据，找出在特定时间段内（如两年）能实现最大收益的买卖时机。这就是一种策略。

量化交易的本质是：**基于历史数据进行分析和策略设计**。这个过程常被称为“回测”。回测是指在历史数据中分析有价值的信息，并设计策略以最大化收益。简而言之，量化交易就是让计算机对历史数据进行数据挖掘，找出能赚钱的有效策略。

## 量化交易所需的核心技能
刚才我们简单介绍了量化交易，大家应该明白了它是基于历史数据设计盈利策略。那么，实现量化交易需要掌握哪些技能呢？

量化交易是一个交叉学科，需要多方面的知识。金融学知识需要了解，但并非重点，只需理解基本概念即可。真正的重点在于与计算机和数据处理相关的技能。

![](img/e383821810697f06427efcc70cddfc47_3.png)

以下是量化交易中至关重要的核心技能：

*   **机器学习算法**：用于基于数据预测未来走势或结果。
*   **统计方法**：用于计算和分析数据中的各项指标，挖掘其蕴含的信息。
*   **编程实践（如Python）**：用于亲自编程实现策略和数据处理。

在量化交易中，没有绝对的“必备工具”。任何能帮助你最大化收益的工具和方法都是好的。金融知识并非不重要，但相比之下，**如何处理、操纵数据并将数据转化为利润**更为关键。本质上，量化交易可以看作是数据挖掘领域一个当前非常热门且实用的分支。

## 总结
本节课中，我们一起学习了量化交易的核心内容。我们明确了量化交易是利用计算机对历史数据进行回测和策略优化的过程，以克服人类交易的主观性和能力局限。同时，我们了解了从事量化交易应重点掌握机器学习、统计学和编程（如Python）等数据处理技能，而非深究复杂的金融理论。量化交易的本质是数据驱动的智能决策。
