# 量化交易与机器学习：P31：重点抽样与测度变化 📊

在本节课中，我们将学习**重点抽样**这一重要的蒙特卡洛模拟技术。它主要用于高效模拟**少数事件**或计算**极端值**，在金融风险管理和衍生品定价中应用广泛。

上一节我们介绍了蒙特卡洛模拟的基础，本节中我们来看看如何通过改变概率测度来提升模拟效率。

## 重点抽样的应用场景 🎯

在金融领域，我们经常需要模拟罕见或极端事件来评估风险。常规的蒙特卡洛模拟在这些场景下效率极低，因为绝大多数模拟路径都不会落入我们关心的“关键区域”。

以下是几个典型的应用案例：

*   **信用违约互换定价**：需要模拟高信用评级客户的违约事件，其违约概率通常极低。
*   **债务抵押凭证定价**：比CDS更复杂，需要模拟不同层级资产的违约和回收情况。
*   **风险价值与潜在未来风险暴露计算**：需要计算投资组合在极端分位数（如99%、99.9%）下的损失。

## 效率问题：为何需要重点抽样？ ⚠️

我们通过一个简单的例子来理解效率问题。假设有一个彩票，中奖概率 `P` 极低，但奖金 `G` 很高。

*   **收益的方差**：对于伯努利分布，收益方差公式为 `Var = G^2 * P * (1-P)`。
*   **蒙特卡洛误差**：模拟的误差与 `sqrt(Var/N)` 成正比，其中 `N` 是模拟次数。
*   **所需模拟次数**：为了控制误差，需要的模拟次数 `N` 必须大于 `10^4 * (1-P)/P`。

![](img/c603120d90dfb3d58297fbe27f8437c7_1.png)

从下表可以看出，当事件概率极低时，所需的模拟次数呈爆炸式增长：

| 中奖概率 `P` | 所需最小模拟次数 `N` |
| :--- | :--- |
| 0.999 | ~10 条 |
| 0.5 | ~10^4 条 |
| 0.01 | ~10^6 条 |
| 0.0001 | ~10^8 条 |

在现实中，AAA级债券的年违约概率约为 `0.0001`（即0.01%）。要精确模拟此类事件，理论上需要上亿次的常规模拟，这在计算上是不可行的。因此，我们需要**重点抽样**来提高算法效率。

![](img/c603120d90dfb3d58297fbe27f8437c7_3.png)

## 重点抽样的直观原理 💡

![](img/c603120d90dfb3d58297fbe27f8437c7_5.png)

核心思想是：**改变概率分布，使模拟样本更多地落入我们感兴趣的区间**，然后对结果进行修正，使其与原分布的期望值保持一致。

![](img/c603120d90dfb3d58297fbe27f8437c7_7.png)

考虑两个等价的期望收益场景：
1.  以 `10^-8` 的概率获得 `$1,000,000`。
2.  以 `0.99` 的概率获得 `$1.01`。

两者的期望收益都是 `$1`。如果我们用第二种分布进行模拟，几乎每次模拟都能产生有效样本（获得奖金），从而极大地提高了模拟“中奖”这一事件的效率。重点抽样就是数学上实现这一“分布转换”并保持期望值不变的方法。

![](img/c603120d90dfb3d58297fbe27f8437c7_9.png)

## 重点抽样的数学框架与流程 🧮

![](img/c603120d90dfb3d58297fbe27f8437c7_11.png)

上一部分我们了解了直观原理，本节中我们来看看其严谨的数学定义和操作步骤。如果对数学推导不感兴趣，可以重点关注其结论和流程。

![](img/c603120d90dfb3d58297fbe27f8437c7_13.png)

假设我们需要计算随机变量 `X` 的函数 `g(X)` 在原始概率测度 `P` 下的期望：
`E_P[g(X)] = ∫ g(x) * f_P(x) dx`
其中 `f_P(x)` 是 `X` 在 `P` 测度下的概率密度函数。当 `f_P(x)` 使得 `g(x)` 显著不为零的区域（关键区域）很难被采样到时，直接模拟效率低下。

![](img/c603120d90dfb3d58297fbe27f8437c7_15.png)

![](img/c603120d90dfb3d58297fbe27f8437c7_17.png)

重点抽样引入一个新的、更容易采样的概率测度 `Q`，其密度函数为 `f_Q(x)`。通过测度变换公式，原期望可以等价地写为：
`E_P[g(X)] = ∫ [g(x) * f_P(x)/f_Q(x)] * f_Q(x) dx = E_Q[g(X) * L(X)]`
其中 `L(X) = f_P(X) / f_Q(X)` 被称为 **似然比** 或 **Radon-Nikodym导数**。

![](img/c603120d90dfb3d58297fbe27f8437c7_19.png)

**操作流程如下：**
1.  选择一个合适的辅助分布 `Q`，使其能高效地生成落入关键区域的样本。
2.  从新分布 `Q` 中抽取样本 `{x_i}`。
3.  对每个样本计算其权重 `w_i = g(x_i) * L(x_i)`，其中 `L(x_i) = f_P(x_i) / f_Q(x_i)`。
4.  计算加权平均作为原期望的估计：`E_P[g(X)] ≈ (1/N) * Σ w_i`。

## 案例：正态分布下的测度平移 📈

![](img/c603120d90dfb3d58297fbe27f8437c7_21.png)

让我们看一个具体例子。假设 `X` 服从标准正态分布 `N(0,1)`（即 `P` 测度），我们关心的是 `X > 3` 的尾部区域事件。直接采样很难获得该区域的样本。

我们可以将采样分布 `Q` 设为 `N(μ, 1)`，即把中心平移到我们感兴趣的尾部附近，例如 `μ = 3`。此时，似然比为：
`L(x) = f_P(x) / f_Q(x) = exp(-x^2/2) / exp(-(x-μ)^2/2) = exp(μ*x - μ^2/2)`

**效果对比：**
*   **常规抽样**：从 `N(0,1)` 采样，样本大多集中在0附近，`X>3` 的样本极少，每个样本权重相同（`1/N`）。
*   **重点抽样**：从 `N(3,1)` 采样，大量样本落在 `X>3` 的区域。每个样本的权重不再是 `1/N`，而是 `(1/N) * L(x_i)`。通过权重修正，最终估计的期望值与常规抽样一致，但方差大大降低。

## 总结 📝

![](img/c603120d90dfb3d58297fbe27f8437c7_27.png)

![](img/c603120d90dfb3d58297fbe27f8437c7_29.png)

本节课我们一起学习了**重点抽样**技术。
1.  **核心目的**：解决蒙特卡洛模拟在计算**少数事件**或**极端值**时效率低下的问题。
2.  **核心思想**：通过改变概率测度（从 `P` 到 `Q`），使模拟样本更多地落入“关键区域”，然后利用**似然比** `L(X)=f_P(X)/f_Q(X)` 对结果进行修正，保证估计的无偏性。
3.  **关键公式**：`E_P[g(X)] = E_Q[g(X) * L(X)]`。
4.  **应用价值**：在信用衍生品定价、极端风险计量等领域不可或缺，能将需要上亿次模拟的问题降低到万次级别，极大提升了计算可行性。
