### 求和与平均值

*   **`A.sum()`**：对数组`A`中的所有元素进行求和。
*   **`A.mean()`**：计算数组`A`中所有元素的算术平均值。

### 方差与标准差

![](img/840537ff40592e0232c76df0a2b9c77d_11.png)

以下是两个重要的统计概念，用于衡量数据的离散程度。

*   **方差（Variance）**：衡量每个数据点与平均值之间差异的平方的平均值。方差越大，数据越分散。
    *   **公式**：`方差 = Σ(每个数据 - 平均值)² / 数据个数`
    *   **NumPy方法**：`A.var()`
*   **标准差（Standard Deviation）**：方差的平方根。它和原始数据有相同的量纲，更常被用来估计数据的波动范围。
    *   **公式**：`标准差 = sqrt(方差)`
    *   **NumPy方法**：`A.std()`

**举例说明**：对于数组 `[1, 2, 3, 4, 5]`，其平均值为3。
*   方差计算：`((1-3)² + (2-3)² + (3-3)² + (4-3)² + (5-3)²) / 5 = 2.0`
*   标准差计算：`sqrt(2.0) ≈ 1.414`

在工业生产中，标准差常被用来评估产品质量的稳定性（良品率）。标准差越小，说明产品尺寸等指标越集中在平均值附近，一致性越好。

### 最大值与最小值

以下是查找数组中极值及其位置的方法。

*   **`A.max()`**：返回数组`A`中的最大值。
*   **`A.min()`**：返回数组`A`中的最小值。
*   **`A.argmax()`**：返回数组`A`中最大值所在的**索引（下标）**。
*   **`A.argmin()`**：返回数组`A`中最小值所在的**索引（下标）**。

## 随机数生成 🎲

接下来，我们探讨NumPy的随机数生成功能。它与Python内置的`random`模块功能类似，但优势在于能直接生成数组形式的随机数据。

NumPy的随机数函数位于`np.random`子模块下。

### 常用随机函数

以下是`np.random`中一些常用函数及其与内置`random`模块的对比。

*   **生成随机浮点数**：`np.random.rand(10)` 生成一个包含10个在`[0, 1)`区间内随机浮点数的数组。对应内置模块的`random.random()`。
*   **生成随机整数**：`np.random.randint(0, 10, (3, 5))` 生成一个3行5列、元素值在`[0, 10)`区间内的随机整数数组。第三个参数`shape`可以指定输出数组的维度。
*   **均匀分布随机数**：`np.random.uniform(2.0, 4.0, 10)` 生成10个在`[2.0, 4.0)`区间内均匀分布的随机浮点数。
*   **随机选择**：`np.random.choice([1, 3, 4, 5], size=10)` 从给定列表`[1, 3, 4, 5]`中随机选择10次（有放回），返回一个长度为10的数组。
*   **打乱顺序**：`np.random.shuffle(arr)` 将数组`arr`的元素顺序随机打乱。此操作直接修改原数组。

**核心优势**：`np.random`中的函数大多支持通过`size`参数直接指定输出数组的形状，实现了批量生成，比使用循环调用内置`random`函数效率高得多。
