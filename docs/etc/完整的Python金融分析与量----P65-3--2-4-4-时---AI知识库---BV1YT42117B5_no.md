## 重采样操作

![](img/e7e6673ce2308a187090df4fc2491e45_5.png)

上一节我们介绍了时间序列的基础操作，本节中我们来看看如何改变数据的时间频率，即重采样。

![](img/e7e6673ce2308a187090df4fc2491e45_7.png)

![](img/e7e6673ce2308a187090df4fc2491e45_9.png)

重采样是指将数据从一个时间频率转换到另一个时间频率。例如，原始数据可能是按天统计的，我们可以将其调整为按周、按月或按年进行统计。这通过 `resample` 方法实现。

![](img/e7e6673ce2308a187090df4fc2491e45_11.png)

在重采样过程中，你需要指定两个关键参数：
1.  **目标频率**：例如 `'W'` 代表周，`'M'` 代表月。
2.  **聚合函数**：指定如何合并新频率内的数据，例如取均值、第一个值或中位数。

以下是重采样的基本代码结构：
```python
# 假设 df 是一个以日期为索引的 DataFrame
# 按周重采样，并计算每周的均值
weekly_data = df.resample('W').mean()
# 按月重采样，并计算每月的第一个值
monthly_data = df.resample('M').first()
```
执行重采样后，即可得到按新频率聚合后的结果。

### 重采样中的标签设置

![](img/e7e6673ce2308a187090df4fc2491e45_13.png)

在重采样时，还有一个重要参数是 `label`，它决定了新时间点的标签是使用时间窗口的起始点还是结束点。

![](img/e7e6673ce2308a187090df4fc2491e45_15.png)

以下是 `label` 参数的作用：
*   **`label='right'` (默认)**：使用时间窗口的结束日期作为标签。例如，月度数据中，1月31日的值代表1月1日至1月31日数据的聚合结果。
*   **`label='left'`**：使用时间窗口的起始日期作为标签。例如，月度数据中，12月31日的值可能代表从12月31日开始的下一个月数据的聚合结果。

```python
# 使用窗口结束日期作为标签（默认）
df.resample('M', label='right').mean()
# 使用窗口起始日期作为标签
df.resample('M', label='left').mean()
```
选择哪种标签取决于分析逻辑。在金融数据分析中，通常使用 `label='right'`，因为它更符合“回顾过去一段时间”的观察习惯。

## 时间窗口（滑动窗口）操作

![](img/e7e6673ce2308a187090df4fc2491e45_17.png)

了解了如何改变数据频率后，我们来看看如何在固定的时间窗口内进行滑动计算，即时间窗口操作。

![](img/e7e6673ce2308a187090df4fc2491e45_19.png)

![](img/e7e6673ce2308a187090df4fc2491e45_21.png)

时间窗口操作不是改变数据频率，而是在一个固定长度的窗口（例如10天）内进行滑动计算。第一个窗口包含第1天到第10天的数据，第二个窗口包含第2天到第11天的数据，以此类推。这常用于计算移动平均线、移动标准差等指标。

实现时间窗口操作的核心方法是 `rolling()`。

![](img/e7e6673ce2308a187090df4fc2491e45_23.png)

![](img/e7e6673ce2308a187090df4fc2491e45_25.png)

![](img/e7e6673ce2308a187090df4fc2491e45_27.png)

以下是进行时间窗口操作的基本步骤：
1.  **数据准备**：首先需要处理缺失值，因为窗口内如果存在缺失值，会导致计算结果也为缺失值。
2.  **创建窗口**：使用 `rolling(window=n)` 方法指定窗口大小 `n`。
3.  **应用计算**：对每个窗口应用聚合函数，如 `.min()`, `.max()`, `.mean()`, `.std()`。

![](img/e7e6673ce2308a187090df4fc2491e45_29.png)

![](img/e7e6673ce2308a187090df4fc2491e45_31.png)

```python
# 1. 处理缺失值
data_clean = df['column_name'].dropna()

![](img/e7e6673ce2308a187090df4fc2491e45_33.png)

# 2. & 3. 创建窗口并计算10日移动最小值
rolling_min = data_clean.rolling(window=10).min()
```
前 `n-1` 个位置（此处为前9个）的结果会是 `NaN`，因为窗口尚未被数据填满。

![](img/e7e6673ce2308a187090df4fc2491e45_35.png)

![](img/e7e6673ce2308a187090df4fc2491e45_37.png)

### 同时计算多个窗口统计量

![](img/e7e6673ce2308a187090df4fc2491e45_39.png)

![](img/e7e6673ce2308a187090df4fc2491e45_41.png)

我们可以方便地为原始数据添加多个窗口统计列。

![](img/e7e6673ce2308a187090df4fc2491e45_43.png)

![](img/e7e6673ce2308a187090df4fc2491e45_45.png)

以下是如何为数据同时添加最小值、最大值、均值和标准差等窗口统计量：
```python
# 假设 data_clean 是处理好的 DataFrame
window_size = 10
data_clean['min_10'] = data_clean['target_column'].rolling(window=window_size).min()
data_clean['max_10'] = data_clean['target_column'].rolling(window=window_size).max()
data_clean['mean_10'] = data_clean['target_column'].rolling(window=window_size).mean()
data_clean['std_10'] = data_clean['target_column'].rolling(window=window_size).std()

![](img/e7e6673ce2308a187090df4fc2491e45_47.png)

# 查看结果
print(data_clean.tail())
```
这样，我们就得到了基于滑动窗口计算出的多种统计指标。

![](img/e7e6673ce2308a187090df4fc2491e45_49.png)

![](img/e7e6673ce2308a187090df4fc2491e45_51.png)

**重要提示**：在进行窗口计算前，务必先处理缺失值（`NaN`），否则计算结果中会出现大量无效的 `NaN` 值。

本节课中我们一起学习了时间序列分析的两个高级操作：
1.  **重采样**：使用 `resample()` 方法改变数据的时间频率（如日变周、月），并通过 `label` 参数控制时间标签的显示方式。
2.  **时间窗口操作**：使用 `rolling()` 方法创建滑动窗口，计算移动统计量（如移动平均、移动极值）。进行此操作前，**必须预处理数据，清除缺失值**，以确保计算结果的准确性。
