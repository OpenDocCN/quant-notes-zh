## 概述
在本节课中，我们将要学习一个在金融投资中至关重要的评估指标——**夏普比率**。我们将理解它的定义、作用，并通过一个简单的例子来学习如何计算它，从而帮助我们判断一项投资在承担风险后所获得的回报是否“划算”。

上一节我们介绍了投资中的风险概念，本节中我们来看看如何用一个具体的指标来衡量风险与收益的关系。

## 夏普比率的定义与作用
夏普比率描述的是：对于你承担的每一单位风险，你能获得多少超额收益。它帮助我们回答“这份高风险的工作（或投资）是否值得去做”这个问题。

例如，一份日薪极高但身处战区的雇佣兵工作，其风险巨大。夏普比率就是用来量化这种“高风险”是否对应了足够高的“高回报”。指标值越高，意味着**单位风险所获得的收益越高**，这项投资也就越有吸引力。

![](img/10192968c82c0a9b3f39dcd07ae82d7e_1.png)

在选股或选择投资组合时，如果单从夏普比率这个指标来看，我们通常会选择**夏普比率更高的选项**，因为它意味着在同等风险下能获得更好的回报，或者在同等收益下承担了更小的风险。

![](img/10192968c82c0a9b3f39dcd07ae82d7e_2.png)

## 夏普比率的计算公式
理解了夏普比率的作用后，我们来看看如何计算它。计算需要三个核心部分：投资组合的收益率、无风险收益率以及投资组合的风险（波动率）。

我们可以用一个生活中的例子来理解：银行会根据你的风险承受能力推荐理财产品。对于追求保本的客户，可能会推荐年化5%的国债（近似无风险收益）。对于能承受风险的客户，可能会推荐一个预期年化收益15%的投资组合。

![](img/10192968c82c0a9b3f39dcd07ae82d7e_4.png)

夏普比率关注的就是这**15%的预期收益**，比**5%的无风险收益**多出来的那部分（即超额收益），与投资组合自身波动性的比值。

以下是夏普比率的计算公式：

**夏普比率 = (投资组合的平均收益率 - 无风险收益率) / 投资组合收益率的标准差**

在具体计算年化夏普比率时，公式通常写作：
`Sharpe_Ratio = (portfolio_return.mean() - risk_free_rate) / portfolio_return.std() * sqrt(252)`
其中，`252`是一年大约的交易天数，用于将日度或周期收益率年化。

## 计算实例：使用Python计算夏普比率
接下来，我们通过一个Python代码示例，演示如何计算多只股票的夏普比率。假设我们已经有了股票的历史日收益率数据。

以下是计算步骤：
1.  准备数据：确保收益率数据没有缺失值（通常用前一天的收益率填充缺失值）。
2.  设定参数：确定无风险收益率（例如年化5%）。
3.  应用公式：按照上述公式进行计算。

```python
import numpy as np
import pandas as pd

# 假设‘returns’是一个Pandas DataFrame，每一列代表一只股票的日收益率序列
# 步骤1: 处理缺失值（前向填充）
returns_filled = returns.fillna(method='ffill')

![](img/10192968c82c0a9b3f39dcd07ae82d7e_6.png)

# 步骤2: 设定年化无风险收益率，例如5%
annual_risk_free_rate = 0.05
# 转换为日度无风险收益率（假设一年252个交易日）
daily_risk_free_rate = annual_risk_free_rate / 252

# 步骤3: 计算每只股票的年化夏普比率
# 计算超额收益（日度）
excess_daily_returns = returns_filled - daily_risk_free_rate

![](img/10192968c82c0a9b3f39dcd07ae82d7e_8.png)

# 计算年化夏普比率
# 年化超额收益率 = 日度超额收益均值 * 252
annualized_excess_return = excess_daily_returns.mean() * 252
# 年化波动率 = 日度收益率标准差 * sqrt(252)
annualized_volatility = returns_filled.std() * np.sqrt(252)

# 夏普比率 = 年化超额收益率 / 年化波动率
sharpe_ratios = annualized_excess_return / annualized_volatility

print(sharpe_ratios)
```

运行以上代码后，我们会得到每只股票的夏普比率。例如，输出可能显示股票A的夏普比率为1.2，股票B的夏普比率为0.8，股票C的夏普比率为-0.3。

## 如何解读结果
得到夏普比率后，我们需要学会解读：

*   **夏普比率 > 0**：投资组合的收益超过了无风险收益。数值越大越好，说明承担风险获得的补偿越高。例如，股票A（1.2）优于股票B（0.8）。
*   **夏普比率 < 0**：投资组合的收益还不如无风险收益（如国债），意味着承担了风险却没有获得正回报。例如股票C（-0.3），通常应避免。
*   **比较原则**：在同等市场条件下，比较不同投资策略或资产时，**夏普比率越高，其风险调整后的收益表现越优秀**。

![](img/10192968c82c0a9b3f39dcd07ae82d7e_10.png)

在我们的示例中，如果“中兴”股票对应的夏普比率最大，那么从该指标角度看，它就是单位风险下收益最高的选择。

## 总结
本节课中我们一起学习了**夏普比率**这个核心的金融评估指标。
1.  **作用**：它衡量的是**每承担一单位风险所获得的超额收益**，帮助投资者判断风险与收益是否匹配。
2.  **计算**：其核心公式为 `(组合收益 - 无风险收益) / 组合风险(标准差)`。我们使用Python演示了完整的计算过程。
3.  **应用**：在量化选股或策略评估中，我们通常倾向于选择**夏普比率更高**的投资选项，因为它代表了更优的风险收益比。
