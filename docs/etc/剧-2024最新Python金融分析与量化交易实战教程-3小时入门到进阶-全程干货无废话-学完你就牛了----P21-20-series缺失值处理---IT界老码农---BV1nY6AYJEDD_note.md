在本节课中，我们将要学习如何处理Pandas Series中的缺失数据。缺失值是数据分析中常见的问题，学会正确处理它们对于后续的计算和可视化至关重要。

上一节我们介绍了Series的基本操作，本节中我们来看看如何处理其中的缺失值。

## 缺失值的出现与影响

在Series中可能会出现缺失数据，即`NaN`值。出现缺失数据时，有时可以放任不管，但有时在进行进一步运算或生成图表时，这些缺失值会带来困扰，因此需要处理。

![](img/31673020dc68cf774fbbfe938a20dd63_3.png)

## 缺失值处理方法概述

处理缺失值主要有两种思路：一是直接删除含有缺失值的行；二是用某个值填充缺失的位置。

### 方法一：删除缺失值

![](img/31673020dc68cf774fbbfe938a20dd63_5.png)

以下是删除缺失值的相关操作。

![](img/31673020dc68cf774fbbfe938a20dd63_7.png)

首先，我们可以判断Series中哪些位置是缺失值。使用`isnull()`函数可以返回一个布尔型Series，其中`True`表示该位置是缺失值。

```python
SR.isnull()
```

![](img/31673020dc68cf774fbbfe938a20dd63_9.png)

与之相对的还有一个函数叫`notnull()`，它返回`True`表示该位置不是缺失值。

```python
SR.notnull()
```

![](img/31673020dc68cf774fbbfe938a20dd63_11.png)

利用布尔索引的特性，我们可以过滤掉缺失值。

![](img/31673020dc68cf774fbbfe938a20dd63_13.png)

```python
SR[SR.notnull()]
```

此外，Pandas还提供了一个直接删除缺失值的函数`dropna()`。

![](img/31673020dc68cf774fbbfe938a20dd63_15.png)

```python
SR.dropna()
```

这个函数会直接删除所有包含缺失值的行。

### 方法二：填充缺失值

除了删除，我们还可以为缺失值填充一个具体的数值。使用`fillna()`函数可以实现这个功能。

例如，将所有缺失值填充为0：

![](img/31673020dc68cf774fbbfe938a20dd63_17.png)

```python
SR.fillna(0)
```

**注意**：Pandas的数据处理操作（如`fillna`、`dropna`）默认不会修改原始Series，而是返回一个新的Series。如果你想保存处理后的结果，需要将其赋值给一个新变量。

![](img/31673020dc68cf774fbbfe938a20dd63_19.png)

```python
SR_filled = SR.fillna(0)
```

![](img/31673020dc68cf774fbbfe938a20dd63_21.png)

在实际分析中，填充值的选择需要根据业务逻辑决定。除了填充固定值（如0），另一种常见的做法是使用该列的平均值进行填充，这有助于保持数据的整体趋势。

```python
mean_value = SR.mean()  # 计算非缺失值的平均值
SR.fillna(mean_value)
```

`mean()`函数在计算时会自动跳过缺失值，非常方便。这正是Pandas等工具在处理真实世界不完整数据时的优势所在。

![](img/31673020dc68cf774fbbfe938a20dd63_23.png)

本节课中我们一起学习了Pandas Series缺失值处理的两种核心方法：删除（`dropna`）与填充（`fillna`）。理解并熟练运用这些方法，是进行数据清洗、保证分析结果可靠性的重要一步。
