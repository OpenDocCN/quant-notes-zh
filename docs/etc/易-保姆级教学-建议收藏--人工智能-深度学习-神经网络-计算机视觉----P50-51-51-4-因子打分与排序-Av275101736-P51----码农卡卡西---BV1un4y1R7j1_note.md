## 为因子分配分数

![](img/0cae33bbce6cc9e34b2e018d13084c94_20.png)

![](img/0cae33bbce6cc9e34b2e018d13084c94_22.png)

排序完成后，我们需要为每个股票在该因子上的表现分配一个分数。简单起见，我们按排名顺序直接分配分数：排名第一的得最高分（300分），排名最后的得最低分（1分）。

![](img/0cae33bbce6cc9e34b2e018d13084c94_24.png)

![](img/0cae33bbce6cc9e34b2e018d13084c94_26.png)

我们将使用NumPy的`linspace`函数来生成一个等差数列作为分数。

![](img/0cae33bbce6cc9e34b2e018d13084c94_28.png)

![](img/0cae33bbce6cc9e34b2e018d13084c94_30.png)

以下是`numpy.linspace`函数的用法介绍。它有三个主要参数：起始值、结束值和生成样本的数量。

![](img/0cae33bbce6cc9e34b2e018d13084c94_32.png)

```python
# 示例：生成从1到300的300个等间隔数字
scores = np.linspace(1, 300, 300)
print(scores) # 输出: [  1.   2.   3. ... 298. 299. 300.]
```

![](img/0cae33bbce6cc9e34b2e018d13084c94_34.png)

![](img/0cae33bbce6cc9e34b2e018d13084c94_36.png)

对于“越高越好”的因子，我们希望因子值最大的股票获得最高分。因此，分数序列应从1到300。

![](img/0cae33bbce6cc9e34b2e018d13084c94_38.png)

```python
    # 为“越高越好”的因子分配分数
    num_stocks = len(up_df) # 获取股票数量，此处为300
    up_df[factor] = np.linspace(1, num_stocks, num_stocks)
```

接下来，我们对“越低越好”的因子进行同样的操作，但分数顺序需要反转，即因子值最小的股票获得最高分。

```python
# 假设 `down_df` 是包含“越低越好”因子的DataFrame
for factor in down_df.columns:
    factor_data = down_df[factor]
    down_df.sort_values(by=factor, inplace=True)
    # 为“越低越好”的因子分配分数，分数从高到低
    num_stocks = len(down_df)
    down_df[factor] = np.linspace(num_stocks, 1, num_stocks)
```

## 合并数据并计算总分

完成两个DataFrame的因子打分后，我们需要将它们合并，并计算每个股票在所有因子上的总得分。

```python
# 将两个DataFrame相加，得到每个股票在所有因子上的总得分
total_score_df = up_df + down_df
# 计算每个股票的总分（各因子得分之和）
total_score_df['total_score'] = total_score_df.sum(axis=1)
```

本节课中我们一起学习了因子打分与排序的核心流程。我们首先遍历了“越高越好”和“越低越好”两类因子，分别对它们进行排序。接着，我们使用`np.linspace`函数根据排序结果为每个股票在单个因子上分配了分数。最后，我们将两个打分后的DataFrame合并，计算出了每个股票的综合得分。这是构建多因子选股模型的基础步骤。
