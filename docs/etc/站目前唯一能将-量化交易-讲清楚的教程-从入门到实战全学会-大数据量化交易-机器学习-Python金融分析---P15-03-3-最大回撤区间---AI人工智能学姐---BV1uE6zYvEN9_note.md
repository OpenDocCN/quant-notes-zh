## 什么是最大回撤？
在策略分析中，最大回撤是一个关键指标。它描述的是在投资过程中，资产净值从最高点到之后最低点的最大跌幅。这个指标衡量的是投资策略可能面临的最糟糕情况。

上一节我们介绍了策略的整体表现，本节中我们来看看如何评估策略的风险。

### 核心概念解释
最大回撤不能仅从字面理解。它描述的是投资过程中“最惨”的阶段。具体来说，它指在投资期间，资产价值持续下跌最严重的那段区间。

例如，股票价格从某个高点开始下跌，可能中间有短暂反弹，但整体趋势向下，直到触及一个低点。从起点的高点到终点的低点，这段价格下跌最剧烈的时期，就是最大回撤区间。

其数学定义可以用以下公式表示：

**最大回撤公式：**
`Max Drawdown = max[(P_i - P_j) / P_i]`
其中：
*   `P_i` 代表第 `i` 天的资产价格（高点）。
*   `P_j` 代表第 `i` 天之后某一天 `j` 的资产价格（低点），且 `j > i`。

这个公式计算的是所有可能的高点 `i` 和其后的低点 `j` 组合中，**相对跌幅**最大的那一个。`(P_i - P_j)` 是下跌的绝对值，除以 `P_i` 则得到相对缩水比例。

### 为什么最大回撤很重要？
最大回撤是一个**风险指标**。它不直接评价策略的盈利能力，而是揭示策略在运行过程中可能承受的最大压力。

![](img/4f097e76914c5d411d6ca31831d0c3b2_5.png)

即使一个策略最终是盈利的，但如果其最大回撤非常大，意味着投资者在过程中可能需要承受巨大的心理压力和资金风险。历史上，过大的回撤可能导致投资者在低点恐慌性卖出，从而无法享受到后续的反弹收益。例如，2008年全球金融危机时，股市从高点大幅下挫，许多投资者承受了巨大损失。

![](img/4f097e76914c5d411d6ca31831d0c3b2_7.png)

因此，评估一个策略时，不仅要看最终收益，还必须关注其最大回撤，以衡量策略的稳健性和风险控制能力。

## 如何计算最大回撤？
理解了概念后，我们来看看如何用Python代码计算最大回撤。我们将使用Pandas库来高效地完成这个计算。

![](img/4f097e76914c5d411d6ca31831d0c3b2_11.png)

以下是计算最大回撤的核心步骤：

![](img/4f097e76914c5d411d6ca31831d0c3b2_13.png)

1.  **计算历史最高点序列**：对于时间序列中的每一个点，计算截至到该点的历史最大值。
2.  **计算回撤值序列**：用每个点的“历史最高值”减去该点的“当前实际值”，得到潜在的下跌幅度。
3.  **计算回撤比率序列**：将下跌幅度除以对应的“历史最高值”，得到相对回撤比例。
4.  **找出最大值**：在所有回撤比例中，找出最大值，即为最大回撤。

![](img/4f097e76914c5d411d6ca31831d0c3b2_15.png)

### 使用Pandas进行计算
Pandas的`.cummax()`函数可以方便地计算滚动最大值（即历史最高点序列）。

![](img/4f097e76914c5d411d6ca31831d0c3b2_17.png)

**示例代码：**
```python
import pandas as pd

![](img/4f097e76914c5d411d6ca31831d0c3b2_19.png)

![](img/4f097e76914c5d411d6ca31831d0c3b2_21.png)

# 假设 df[‘close’] 是股票的每日收盘价序列
price_series = df[‘close’]

![](img/4f097e76914c5d411d6ca31831d0c3b2_23.png)

![](img/4f097e76914c5d411d6ca31831d0c3b2_25.png)

# 1. 计算截至每日的历史最高价
historical_max = price_series.cummax()

![](img/4f097e76914c5d411d6ca31831d0c3b2_27.png)

# 2. 计算每日的潜在下跌幅度（历史最高 - 当前价）
drawdown_value = historical_max - price_series

![](img/4f097e76914c5d411d6ca31831d0c3b2_29.png)

# 3. 计算每日的回撤比例（下跌幅度 / 历史最高）
drawdown_ratio = drawdown_value / historical_max

![](img/4f097e76914c5d411d6ca31831d0c3b2_31.png)

# 4. 找出整个序列中的最大回撤比例
max_drawdown = drawdown_ratio.max()

![](img/4f097e76914c5d411d6ca31831d0c3b2_33.png)

![](img/4f097e76914c5d411d6ca31831d0c3b2_35.png)

print(f“最大回撤为：{max_drawdown:.2%}”)
```

![](img/4f097e76914c5d411d6ca31831d0c3b2_37.png)

**代码解释：**
*   `price_series.cummax()`：生成一个新序列，其中每个位置的值都是原序列从开始到该位置的最大值。
*   `historical_max - price_series`：得到每个时间点上，当前价格相对于历史最高点的下跌绝对值。
*   `drawdown_value / historical_max`：将下跌绝对值转化为相对比例，这是衡量“惨烈程度”的关键。
*   `.max()`：最终找出这个比例序列中的最大值，即**最大回撤**。

在实际的策略回测框架（如Backtrader、Zipline等）中，这些计算通常已被封装好，可以直接输出最大回撤值和对应的区间图表。上述手动计算过程有助于我们深入理解其原理。
