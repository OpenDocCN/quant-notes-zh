## 概述

## 引入Matplotlib

![](img/0564967f2e9b9804924df81f0130606b_3.png)

在代码中，我们通常这样引入Matplotlib的绘图模块：

![](img/0564967f2e9b9804924df81f0130606b_5.png)

```python
import matplotlib.pyplot as plt
```

![](img/0564967f2e9b9804924df81f0130606b_7.png)

![](img/0564967f2e9b9804924df81f0130606b_9.png)

其中，`plt`是约定俗成的别名。`plt.plot()`函数用于绘图，`plt.show()`函数用于展示图形。

## 绘制基础折线图

`plt.plot()`函数最基本的功能是绘制折线图。它接受两个主要参数：X轴数据列表和Y轴数据列表。

```python
plt.plot([1, 2, 3, 4], [2, 4, 6, 8])
plt.show()
```

运行上述代码，会生成一条连接点(1,2)、(2,4)、(3,6)、(4,8)的直线。如果Y值改为`[2, 3, 6, 4]`，则会生成一条折线。

![](img/0564967f2e9b9804924df81f0130606b_11.png)

## 自定义线条样式

![](img/0564967f2e9b9804924df81f0130606b_13.png)

`plt.plot()`函数还有第三个可选参数，它是一个格式字符串，用于控制线条的**线型**、**标记点**和**颜色**。

格式字符串的语法为：`‘[颜色][标记点][线型]’`。

以下是几个关键部分的示例：

![](img/0564967f2e9b9804924df81f0130606b_15.png)

![](img/0564967f2e9b9804924df81f0130606b_17.png)

*   **颜色**：`‘b’`（蓝色）、`‘g’`（绿色）、`‘r’`（红色）、`‘k’`（黑色）。
*   **标记点**：`‘o’`（圆点）、`‘v’`（下三角）、`‘^’`（上三角）、`‘*’`（星号）、`‘+’`（加号）。
*   **线型**：`‘-’`（实线）、`‘--’`（虚线）、`‘:’`（点线）。

例如，`‘ro-’`表示绘制红色实线，并在数据点处用圆圈标记。`‘g--’`表示绘制绿色虚线，无标记点。

![](img/0564967f2e9b9804924df81f0130606b_19.png)

![](img/0564967f2e9b9804924df81f0130606b_21.png)

你也可以使用关键字参数分别指定，例如：`color=‘red‘, marker=‘o‘, linestyle=‘--‘`。

## 在同一图形中绘制多条线

只需多次调用`plt.plot()`函数，然后再调用`plt.show()`，即可在同一坐标系中叠加多条线。

![](img/0564967f2e9b9804924df81f0130606b_27.png)

![](img/0564967f2e9b9804924df81f0130606b_29.png)

```python
# 绘制第一条线：红色圆点实线
plt.plot([1, 2, 3, 4], [1, 4, 9, 16], ‘ro-‘)
# 绘制第二条线：蓝色三角虚线
plt.plot([1, 2, 3, 4], [2, 5, 10, 17], ‘b^--‘)
plt.show()
```

![](img/0564967f2e9b9804924df81f0130606b_31.png)

## 其他常用函数

![](img/0564967f2e9b9804924df81f0130606b_33.png)

除了`plot`，Matplotlib还提供了许多其他绘图函数，例如：
*   `plt.scatter()`: 绘制散点图。
*   `plt.bar()`: 绘制柱状图。
*   `plt.hist()`: 绘制直方图。
*   `plt.title()`: 为图表添加标题。
*   `plt.xlabel()` / `plt.ylabel()`: 为X/Y轴添加标签。
*   `plt.legend()`: 显示图例。
