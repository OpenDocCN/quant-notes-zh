# 量化分析入门：P1：首板后三连阴尾盘买入法量化验证 🧐

在本节课中，我们将学习如何对一个网络上流传的股票交易策略——“首板后三连阴尾盘买入法”——进行量化验证。我们将把博主的文字描述转化为精确的量化规则，并使用Python进行历史数据回测，通过数据来判断该策略的实战价值。

网上有很多人分享炒股经验。他们自称是高手或赚了很多钱，无私地分享高胜率战法或选股秘籍。我偶尔会刷到这类视频。如果看到高赞视频，一般会认真看完。如果我认为有研究价值，会进行详细的量化统计分析，用数据辨真假。这期我们就统计分析验证一个自称炒股3年赚256万的“首板后三连阴，尾盘5分钟买入法”。

## 策略原理解读 📺

我们先看一下这个视频。它的标题是“3年赚256万”。播放量只有4000多，但点赞、收藏、分享的比例接近20%。这个比例非常高，说明很多股民认可这个选股方法和战法。

视频中提到：“了解高5~7个点就可以直接卖掉，只要这种形态出现，并且符合条件，那第二天必然会有大阳线甚至涨停。今天我要跟大家去讲的就是如何在这一天阴线的位置进场，然后在次日，也就第二天阳线的位置，我们冲高卖出。”

博主展示了案例图，声称在第二根阳线大涨了8%，并举例在第三根阴线进场，次日冲高最多14%卖出。

## 策略条件梳理 📝

我总结了视频内容，将其选股条件整理如下：

以下是该策略的核心买入、卖出及选股条件：

![](img/0f7f0eb95ef729ee5e367a2b1b973a65_1.png)

1.  **买入**：首板后出现三连阴，在第三根阴线的尾盘最后5分钟买入。
2.  **止盈**：买入后次日，股价冲高5%到7%时卖出。
3.  **止损**：如果股价跌破首板涨停阳线的最低价（即起涨点），则止损。
4.  **首板要求**：必须是首个涨停板。分时图需震荡向上，显示盘中吸筹。不能是急速拉升的秒板，最好有一次炸板回封。尾盘必须封死涨停。
5.  **次日开盘**：首板后的第一天（即三连阴的第一天）必须是高开，高开一分钱也行，不能平开或低开。
6.  **三连阴形态**：三根都是小阴线，尤其是第二、第三根必须是小阴线，不能是大阴线。绝对不能跌破首板当日阳线的起涨点。
7.  **调整幅度**：三连阴整体调整到首板涨停阳线高度的1/2处为佳，最大下跌幅度不超过阳线的1/3处。
8.  **成交量**：三连阴期间成交量持续缩量为佳。第三根阴线的成交量必须小于第一根阴线。
9.  **K线纯粹性**：必须是真正的三连阴，中间不能夹杂阳线或假阳线。
10. **尾盘分时**：尾盘最后5分钟的分时图最好是窄幅震荡或小幅下跌，不能是快速拉升。
11. **股票筛选**：排除ST/*ST股票，排除首板是一字涨停的股票。

## 量化转换与回测 💻

上一节我们详细解读了策略的原始条件，本节中我们来看看如何将这些文字描述转化为可执行的Python代码，并进行历史回测。

我们将选股逻辑转化成Python语言，做一个量化回测。回测开始日期设为2018年1月1日，结束日期为最近一个交易日。

## 回测结果分析 📊

在将所有历史个股数据运行计算后，我们得到一个包含所有信号的CSV文件。接下来我们对这些数据进行统计分析。

统计期间从2018年至今，信号总数为 **741** 次。

以下是核心统计结果：

*   **次日平均涨幅**：`0.526%`
*   **次日收盘上涨概率**：`55.2%`
*   **触发5%止盈的次数**：`145` 次（占比约 `19.6%`）
*   **触发5%止损的次数**：`65` 次（占比约 `8.8%`）
*   **次日直接跌破起涨点止损的次数**：`50` 次（占比约 `6.7%`）

此外，我们还统计了所有信号在之后1到9个交易日的平均涨跌幅和收涨概率。数据显示，首日（即策略持有的次日）表现最佳，平均涨幅为正，上涨概率超过55%。但随着持有时间延长，平均涨幅和上涨概率呈逐渐下降趋势。

## 结论与总结 🎯

从以上统计结果我们可以得出基本结论：

1.  **策略有效性**：该策略在超短隔日交易（即当天尾盘买入，次日卖出）的框架下，具有一定的实战价值。次日上涨概率 `55.2%` 高于随机水平，且止盈次数显著高于止损次数。
2.  **策略定位**：这是一个典型的**超短线策略**，不适合中长期持有。数据表明，持有超过一天后，策略的胜率和收益率优势会衰减。
3.  **并非“圣杯”**：策略存在失败案例，胜率并非100%。约20%的概率能捕获5%以上的涨幅，但同时也有近10%的概率会在次日面临5%以上的亏损。

本节课中我们一起学习了如何对一个网络流传的股票策略进行量化验证。我们经历了从解读策略条件、转化为量化规则、编写回测代码到分析统计结果的完整流程。最终通过数据发现，“首板后三连阴尾盘买入法”作为一个超短隔日交易策略，具备一定的统计优势，但绝非稳赚不赔的“秘籍”。量化分析的价值就在于用客观数据替代主观感觉，帮助我们在复杂的市场中做出更理性的决策。
