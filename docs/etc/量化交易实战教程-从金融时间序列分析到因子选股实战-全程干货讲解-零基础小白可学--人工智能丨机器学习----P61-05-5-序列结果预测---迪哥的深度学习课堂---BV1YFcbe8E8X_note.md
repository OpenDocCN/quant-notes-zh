上一节我们介绍了如何预测未来的单个时间点。本节中，我们来看看如何预测未来一个时间段的序列结果。

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_5.png)

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_7.png)

预测未来多个时间点的方法其实很简单。关键在于数据准备和模型输出的设置。在默认的序列生成函数中，参数 `SINGLESTEP` 默认是 `False`。

```python
# 默认情况下，函数进行多标签（多步）预测
SINGLESTEP = False
```

这意味着默认进行的是多标签预测。因此，当我们不指定预测步数为1时，模型会预测整个未来的序列。

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_9.png)

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_11.png)

训练数据的准备方法与之前类似。主要区别在于标签的构建。以下是标签维度的关键点：

*   标签的第一个维度（样本数量）与之前相同。
*   标签的第二个维度不再是1，而是72。这表示标签包含的是未来72个时间点的值，即一个时间区间。

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_13.png)

标签的不同直接导致了模型输出的不同。在模型训练部分，我们构建了一个更复杂的网络结构。

我们使用了两层LSTM网络。第一层LSTM有32个隐藏单元。这里有一个关键点：当堆叠LSTM层时，需要将前一层LSTM的 `return_sequences` 参数设置为 `True`。

```python
# 堆叠LSTM层时，前一层需返回完整序列
model.add(LSTM(32, return_sequences=True))
```

这样做的原因是，第一个LSTM层的输出需要作为序列输入到第二个LSTM层中。如果只返回最后一个时间步的输出，则无法进行堆叠。

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_15.png)

接着，我们顺序添加第二层LSTM。最后，连接一个全连接层作为输出层。此时，输出层的神经元数量必须与标签的维度对应。

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_17.png)

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_19.png)

由于我们的标签是未来72个值，因此模型的输出也必须是72个值。

```python
# 输出层神经元数量需与预测步数（标签维度）一致
model.add(Dense(72))
```

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_21.png)

模型训练完成后，可以观察预测结果。在结果可视化图中，蓝色曲线代表实际值，红色曲线代表预测值。

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_23.png)

模型成功预测出了序列“先下降再上升”的整体趋势。虽然预测值在具体数值上可能不是特别精确，但趋势方向是基本一致的。这展示了神经网络捕捉时间序列模式的能力。

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_25.png)

这里简要回顾了在TensorFlow中完成时间序列预测任务的核心流程。本质上，我们的演示基于一个清晰的框架：

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_27.png)

1.  指定时间窗口（`lookback`）用于构建输入特征。
2.  指定标签，决定是预测单点还是未来序列。
3.  将数据组织成批量的、适合循环神经网络输入的格式。
4.  将数据输入到LSTM网络中进行训练和预测。

![](img/a439a68b6b89ac1f5abf6fd76fcfad86_29.png)

关键在于如何根据任务目标调整标签。标签若为单个值，则预测未来一个点；标签若为72个值，则预测未来72个点。

一个常见的问题是：模型预测出的72个值，是如何与未来72个时间点的真实值一一对应起来的？

这其实无需我们手动对齐。神经网络预测什么，完全由损失函数规定。初始时，神经网络并不知道这72个输出值各自代表哪个未来时间点。但通过损失函数（例如均方误差MSE）的计算，模型会学习到：如果第一个输出值对应第一个未来时间点的真实值，第二个输出值对应第二个未来时间点的真实值，依此类推，那么总的损失值就会降低。因此，模型会自动朝着这个方向进行优化。

**损失函数定义了我们的任务目标。** 在设计神经网络解决具体问题时，最核心的一环就是设计合适的损失函数，它将我们的业务需求转化为模型可优化的数学目标。

本节课中我们一起学习了如何实现时间序列的多步预测。我们了解了通过改变标签的维度（从1变为N）来让模型预测未来N个时间点，并掌握了在堆叠LSTM层时需要设置 `return_sequences=True` 的关键技巧。最后，我们认识到损失函数在引导模型学习正确对应关系中的核心作用。
