## 价值类因子定义
上一节我们介绍了规模类因子的选股方法，本节中我们来看看价值类因子选股。

价值类因子对应人们常说的价值投资策略。根据定义，价值投资是一种久经考验的投资策略。该策略的核心是通过购买那些**相对低价**的股票，这些股票的价值是基于基本面指标来衡量的，例如股息、账面价值、利润、现金流或其他衡量公司价值的方法。简而言之，价值投资的目标是寻找当前股价被**相对低估**的股票。

## 价值类因子常用指标
以下是价值类因子最常用的几个核心指标：
*   **市净率**
*   **市销率**
*   **市盈率**（分为动态市盈率与静态市盈率）

## 核心指标详解
### 市净率
市净率是最常用的价值指标之一，其公式为：
**市净率 = 每股市价 / 每股净资产**

一般来说，市净率较低的股票投资价值较高，反之则较低。但这个判断并非绝对，因为不同行业的市净率水平差异很大。例如，传统能源行业的市净率通常远低于互联网行业。在互联网行业，市净率普遍较高（如30-50）。如果市净率过高（如超过55-60），可能意味着股票存在泡沫；如果过低（如接近1或5），也可能表明公司基本面不佳。因此，市净率是一个重要的参考指标，但需要结合其他因素综合判断。

在代码中，市净率对应的字段通常是 `valuation.pb_ratio`。

### 市销率
市销率的公式为：
**市销率 = 股票股价 / 每股销售额**

这个指标在国内证券市场应用较多。它的主要作用是剔除那些市盈率很低，但**主营业务缺乏核心竞争力**、主要依靠非经常性损益来增加利润的股票。有些公司的主营业务可能不赚钱，但财报显示利润很好，可能是因为其收入来源于其他业务（如房地产）。将市销率与市盈率结合使用，可以帮助我们发现这类公司。

在代码中，市销率对应的字段通常是 `valuation.ps_ratio`。

### 市盈率
市盈率分为动态市盈率和静态市盈率。

**动态市盈率**：其计算基于**尚未真正实现的、对下一年度的盈利预测**。例如，在2022年，根据已发布的季度报告（一季报、半年报等）来预测2023年全年的每股收益。
**静态市盈率**：其计算基于**上一年度已实现的财报利润**。它表示按照当前的盈利水平，需要累积多少年才能达到当前的股价。通常所说的“市盈率”多指静态市盈率。

两者的公式如下：
*   **动态市盈率 = 股票现价 / 未来每股收益的预测值**
*   **静态市盈率 = 股票现价 / 每股收益**

一般来说，动态市盈率和静态市盈率由于算法不同，差异可能较大。市盈率指标越小，说明投资回收周期越短，风险相对越小。

在代码中，动态市盈率对应的字段可能是 `valuation.pcf_ratio`，而静态市盈率对应的字段通常是 `valuation.pe_ratio`。

## 代码实战环节 💻
接下来，我们将进入代码实战环节，演示如何使用这些价值类因子进行选股。

### 1. 基于市净率选股
首先，我们尝试使用市净率进行筛选。通常不会单独使用市净率一个指标。

```python
# 示例：筛选市净率小于1.5的股票
# 假设使用聚宽（JoinQuant）的API，字段名可能为 valuation.pb_ratio
# 此处为伪代码逻辑
filter_condition = (valuation.pb_ratio < 1.5)
order_by = valuation.pb_ratio.asc()
start_date = ‘2019-01-01’
# 执行查询并打印结果
```

运行后，将返回从2019年1月1日起，市净率小于1.5的股票列表。

### 2. 结合市净率与市销率选股
接下来，我们结合市净率和市销率进行筛选。

```python
# 示例：筛选市净率小于1.5 且 市销率小于0.5 的股票
filter_condition = (valuation.pb_ratio < 1.5) & (valuation.ps_ratio < 0.5)
# 其他逻辑与上例类似
```
运行后，将筛选出同时满足两个条件的股票。结果中可能出现市销率为负的情况，这通常意味着公司利润为负。

### 3. 结合市盈率与市销率选股
最后，我们演示结合动态市盈率、静态市盈率和市销率进行综合筛选。

```python
# 示例：筛选市销率<0.5，动态市盈率<6，静态市盈率在3到5之间的股票，并按静态市盈率排序
filter_condition = (valuation.ps_ratio < 0.5) & 
                   (valuation.pcf_ratio < 6) & 
                   (valuation.pe_ratio > 3) & 
                   (valuation.pe_ratio < 5)
order_by = valuation.pe_ratio.asc()
# 执行查询并打印结果
```
运行这段代码，将得到符合多重价值因子条件的股票列表。可以看到，静态市盈率为正的公司，其动态市盈率可能因季度亏损而发生变化。

## 总结
本节课中我们一起学习了价值类因子在量化选股中的应用。

价值类因子对应价值投资策略，其核心目标是找到那些被市场**低估**的股票。衡量标准主要看股价是否处于相对低位，并考察其股息、利润、现金流等基本面价值。

价值类因子的核心指标有三个：
1.  **市净率**：公式为 **PB = 股价 / 每股净资产**。代码字段通常为 `valuation.pb_ratio`。该指标越小通常越好，但需考虑行业差异。
2.  **市销率**：公式为 **PS = 股价 / 每股销售额**。代码字段通常为 `valuation.ps_ratio`。它用于评估公司主营业务的盈利能力，常与市盈率结合使用，以剔除主营业务不健康但利润表象好的公司。
3.  **市盈率**：分为动态和静态。
    *   **静态市盈率**：基于上一年财报利润，代码字段通常为 `valuation.pe_ratio`。
    *   **动态市盈率**：基于本年度已发布数据预测下一年利润，代码字段可能为 `valuation.pcf_ratio`。
