## 整体任务流程梳理

上一节我们介绍了如何获取和筛选用于评价股票的指标。本节中我们来看看如何利用这些指标对股票进行打分和排序。

在已知每个指标的优劣方向（例如，某些指标越大越好，某些越小越好）后，接下来的核心任务就是“打分”。

### 打分方法概述

以下是几种常见的打分方法，我们将重点介绍第一种基于区间划分的方法。

#### 方法一：基于数值区间划分打分

我们以一组示例数据来说明此方法。假设我们有300只股票（ID1到ID300），每只股票有A、B、C、D四个指标。已知：
*   **指标A和B**：数值越大越好。
*   **指标C和D**：数值越小越好。

每只股票的指标值（假设已归一化到0-1之间）如下表所示：

| 股票ID | 指标A | 指标B | 指标C | 指标D |
| :--- | :--- | :--- | :--- | :--- |
| ID1 | 0.71 | 0.28 | 0.39 | 0.11 |
| ID2 | 0.45 | 0.17 | 0.81 | 0.02 |

**第一步：为每个指标设定打分区间和规则。**
我们将0-1的数值范围划分为5个区间，并为“越大越好”和“越小越好”两类指标分别设定打分规则。

```python
# 区间划分
区间划分 = [(1.0, 0.8), (0.8, 0.6), (0.6, 0.4), (0.4, 0.2), (0.2, 0.0)]

# 越大越好型指标（如A、B）的得分映射
得分_越大越好 = {区间划分[0]: 5, 区间划分[1]: 4, 区间划分[2]: 3, 区间划分[3]: 2, 区间划分[4]: 1}

# 越小越好型指标（如C、D）的得分映射
得分_越小越好 = {区间划分[0]: 1, 区间划分[1]: 2, 区间划分[2]: 3, 区间划分[3]: 4, 区间划分[4]: 5}
```

**第二步：为每只股票的每个指标单独打分。**
根据指标值落入的区间和对应的打分规则进行评分。
*   **ID1评分过程**：
    *   指标A=0.71（越大越好），落入(0.8, 0.6)区间，得 **4分**。
    *   指标B=0.28（越大越好），落入(0.4, 0.2)区间，得 **2分**。
    *   指标C=0.39（越小越好），落入(0.4, 0.2)区间，得 **4分**。
    *   指标D=0.11（越小越好），落入(0.2, 0.0)区间，得 **5分**。
*   **ID2评分过程**：
    *   指标A=0.45（越大越好），落入(0.6, 0.4)区间，得 **3分**。
    *   指标B=0.17（越大越好），落入(0.2, 0.0)区间，得 **1分**。
    *   指标C=0.81（越小越好），落入(1.0, 0.8)区间，得 **1分**。
    *   指标D=0.02（越小越好），落入(0.2, 0.0)区间，得 **5分**。

**第三步：计算每只股票的总分并排序。**
将单只股票所有指标的得分相加，得到其综合总分。

| 股票ID | 总分计算 | 总分 |
| :--- | :--- | :--- |
| ID1 | 4 + 2 + 4 + 5 | **15** |
| ID2 | 3 + 1 + 1 + 5 | **10** |

计算出所有300只股票的总分后，按总分从高到低进行排序。选择排名最高的前10只股票，作为下一次投资组合调整（调仓）的目标。

#### 方法二：基于排名直接打分

另一种更简单的方法是直接根据指标值在所有股票中的排名来打分。例如，对于300只股票：
*   若指标“越大越好”，则排名第1（值最大）的股票得300分，排名第300的股票得1分。
*   若指标“越小越好”，则排名第1（值最小）的股票得300分，排名第300的股票得1分。

然后将各指标得分相加得到总分，再进行排序选股。这种方法避免了划分区间的步骤，同样有效。

### 策略实现流程总览 🚀

![](img/9dec5dd95f099b76eb1722693b2470bd_9.png)

基于上述打分法，一个完整的月度调仓策略流程可以梳理如下：

1.  **确定股票池**：首先，明确我们的选股范围。本例中，我们将从**沪深300指数**的成分股中挑选。
2.  **设置调仓周期**：定义一个定时调仓函数（如每月第一个交易日执行），函数的核心是`rebalance`（再平衡）操作。
3.  **实现`rebalance`函数**：这是策略的核心，其内部逻辑为：
    *   **数据获取**：读取当前时刻股票池中所有股票的预设指标数据（例如6个指标，前3个越大越好，后3个越小越好）。
    *   **指标打分**：使用上述打分法之一，为每只股票的每个指标计算得分。
    *   **汇总排序**：将每只股票的各指标得分相加，得到综合总分，并按总分降序排列。
    *   **生成交易清单**：选取总分排名前10的股票，作为本次调仓拟买入的标的。
