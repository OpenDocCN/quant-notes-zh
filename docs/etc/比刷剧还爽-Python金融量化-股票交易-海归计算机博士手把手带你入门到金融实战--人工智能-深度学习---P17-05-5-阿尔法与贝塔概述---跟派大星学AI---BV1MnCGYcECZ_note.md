# 量化交易全教程：P17：阿尔法与贝塔概述 📊

在本节课中，我们将要学习量化策略评估中的两个核心概念：阿尔法（α）和贝塔（β）。理解这两个指标，能帮助我们区分投资收益的来源，并明确量化策略的优化方向。

## 收益的构成

上一节我们介绍了多种策略评估指标，本节中我们来看看收益的构成。一项投资或策略所赚取的收益，通常可以分解为两个部分。

以下是收益的两个主要来源：

1.  **市场收益**：这部分收益与整体市场环境的好坏直接相关。当大盘上涨时，大部分投资都能获得收益，这部分收益是“随大流”获得的。
2.  **超额收益**：这部分收益与市场整体波动无关，它来源于投资者或策略自身的独特能力，例如精准的选股、优秀的择时或有效的风险管理。

## 阿尔法（α）与贝塔（β）的定义

基于上述收益构成，阿尔法和贝塔分别用于衡量这两部分。

*   **贝塔（β）**：衡量策略收益对市场波动的敏感性。它代表了“市场收益”部分。一个策略的贝塔值高，意味着其收益受大盘影响大。
*   **阿尔法（α）**：衡量策略超越市场基准的超额收益。它代表了“超额收益”部分，是剔除了市场影响后，纯粹由策略自身能力带来的收益。

![](img/26bdc7e7e845df9dc977ef5e498f630a_1.png)

我们可以用一个简单的线性回归模型来理解它们的关系：

**策略收益 = α + β × 市场收益 + 误差**

在这个公式中：
*   `β`（贝塔）是斜率，表示市场收益每变动1单位，策略收益的预期变动量。
*   `α`（阿尔法）是截距，表示当市场收益为零时，策略所能获得的收益，即超额收益。

图中包含三条曲线：

*   **基准收益**：代表市场整体表现的收益曲线（如沪深300指数收益）。这对应了“市场收益”。
*   **超额收益**：策略收益减去基准收益后的曲线。这直接体现了阿尔法（α）的累积效果。

我们的核心目标，就是通过优化策略，最大化这条**超额收益**曲线。

## 量化策略的关注重点

市场整体的走势（贝塔部分）是个人投资者难以控制和预测的。因此，在量化交易中，我们的主要精力应放在如何获取稳定的**阿尔法**上。

以下是量化策略开发的核心思路：

1.  承认市场波动（贝塔）的客观存在，但不将其作为主要盈利来源。
2.  专注于研究和构建能持续产生**正阿尔法**的策略模型。
3.  通过历史回测和风险评估，确保阿尔法收益的稳定性和可持续性。

## 总结

![](img/26bdc7e7e845df9dc977ef5e498f630a_5.png)

本节课中我们一起学习了阿尔法（α）和贝塔（β）的核心概念。贝塔衡量了策略收益与市场波动的关联程度，代表了系统性风险带来的收益；而阿尔法则衡量了策略超越市场的超额收益，代表了策略自身的选股或择时能力。在量化交易实战中，我们的核心目标是寻找并验证能持续产生正阿尔法的策略，这才是量化模型真正的价值所在。后续在因子分析等章节中，我们将深入探讨如何具体计算和优化这些指标。
