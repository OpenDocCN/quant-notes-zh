## 概述
在本节课中，我们将要学习夏普比率这一核心金融指标。夏普比率用于衡量投资组合在承担单位风险时所获得的超额回报，是量化投资中评估风险调整后收益的重要工具。

## 夏普比率的定义与意义
上一节我们介绍了风险的概念，本节中我们来看看如何量化风险与收益的关系。

夏普比率描述的是，在承担一定风险的情况下，投资所能获得的收益是否“值得”。它衡量的是“单位风险所带来的收益”。指标越高，意味着在承担相同风险的情况下，获得的收益越高，投资效率也就越高。

在选股或构建投资组合时，如果仅从夏普比率这一指标来看，我们倾向于选择夏普比率更高的资产或组合，因为它代表了更高的风险调整后收益。

## 夏普比率的计算原理
理解了夏普比率的意义后，我们来探讨其计算方法。计算夏普比率需要引入“无风险收益率”作为基准。

![](img/ce56ea195567b2a7ad5e07ea89a203c5_3.png)

例如，银行存款或国债通常被视为几乎无风险的投资，其收益率（如年化3%）可作为无风险收益率。而一个投资组合（如股票基金）可能提供15%的预期收益率。这两者之间的差值（15% - 3% = 12%）就是承担风险所获得的“超额收益”。

![](img/ce56ea195567b2a7ad5e07ea89a203c5_5.png)

夏普比率的核心思想，就是用这个“超额收益”除以投资组合收益的波动性（即风险，通常用标准差表示）。因此，夏普比率的计算公式可以表示为：

**夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差**

这个公式计算的是每承担一单位的总风险，能获得多少超额回报。

## 实战计算：用Python计算夏普比率
理论部分已经介绍完毕，现在让我们动手用Python代码实际计算夏普比率。计算前，我们需要准备好投资组合的历史收益率数据。

在之前的课程中，我们计算了股票的日收益率，并对数据中的缺失值进行了前向填充处理（即用前一天的数值填充当天的缺失值），以确保数据连续性。

以下是计算夏普比率的步骤与代码：

1.  **准备数据**：获取投资组合的历史日收益率序列 `returns`。
2.  **设定参数**：设定年化无风险收益率，例如5%（0.05）。由于我们使用的是日度数据，需要将其转化为日度无风险收益率。
3.  **计算超额收益**：用日收益率减去日化无风险收益率。
4.  **计算夏普比率**：计算超额收益的均值，除以超额收益的标准差，最后乘以根号下年化交易日数（通常为250天）进行年化。

```python
import numpy as np

# 假设 returns 是一个包含日收益率的Pandas Series或NumPy数组
# 设定年化无风险收益率为5%
risk_free_rate = 0.05
# 计算日化无风险收益率
daily_risk_free = risk_free_rate / 250

![](img/ce56ea195567b2a7ad5e07ea89a203c5_7.png)

# 计算超额收益率
excess_returns = returns - daily_risk_free

# 计算夏普比率（年化）
sharpe_ratio = np.mean(excess_returns) / np.std(excess_returns) * np.sqrt(250)

print(f"夏普比率（年化）为: {sharpe_ratio}")
```

运行以上代码后，我们可以得到计算出的夏普比率。例如，对于多只股票进行计算后，可能会得到如下结果：

*   中兴通讯： 0.85
*   股票A： -0.12
*   股票B： 0.45

## 结果解读与应用
计算结果已经呈现，我们该如何解读并应用它呢？

![](img/ce56ea195567b2a7ad5e07ea89a203c5_11.png)

根据计算结果，中兴通讯的夏普比率最高（0.85），为正值。股票B的夏普比率（0.45）次之，而股票A的夏普比率为负（-0.12）。

在比较时，我们遵循以下原则：
*   **夏普比率为正**：表明投资获得了超越无风险收益的超额回报。
*   **夏普比率为负**：表明投资表现甚至不如无风险资产，通常不予考虑。
*   **夏普比率越高越好**：在均为正值的选项中，夏普比率越高，代表其风险调整后的收益表现越优秀。

![](img/ce56ea195567b2a7ad5e07ea89a203c5_13.png)

![](img/ce56ea195567b2a7ad5e07ea89a203c5_15.png)

因此，在本例的策略中，仅从夏普比率角度评估，中兴通讯是最优选择。
