## 策略核心概念 📈

上一节我们初步接触了量化策略的编写流程。本节中，我们来看看一个基于技术分析、有科学依据的策略：双均线策略。

**移动平均线**（Moving Average, MA）代表了过去N个交易日股价的平均值。其计算公式为：
`MA(N) = (P1 + P2 + ... + PN) / N`
其中，`P1`到`PN`代表最近N个交易日的收盘价。将每个交易日的这个平均值连接起来，就形成了一条均线。例如，计算过去10天的平均值，就得到了10日均线；计算过去5天，则得到5日均线。均线反映了股票价格在近期历史中的整体波动趋势。

常见的均线周期包括5日、10日、30日、60日、120日和240日等。

**双均线策略**涉及两条周期不同的均线：一条短期均线（如5日线），一条长期均线（如20日线）。该策略的交易信号基于两条均线的交叉关系产生。

以下是该策略的两个关键信号点：

*   **金叉（Golden Cross）**：当短期均线从下方向上穿越长期均线时，被视为**买入信号**。这通常意味着股价开始呈现上涨趋势，短期内的上涨动能超过了长期趋势。
*   **死叉（Death Cross）**：当短期均线从上方向下穿越长期均线时，被视为**卖出信号**。这通常意味着股价下跌趋势开始，短期下跌速度超过了长期趋势。

简单概括，双均线策略就是：**金叉买入，死叉卖出**。
