## 计算每日差异值

首先，我们需要计算每一天指标相对于前一天的差异值。例如，在股票数据中，我们不仅关心当天的价格，更关心它相比前一天是上涨还是下跌了多少。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_6.png)

在Pandas中，计算差异值非常简单。我们可以使用`.diff()`方法，它会自动用当前行的值减去前一行的值。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_8.png)

以下是计算差异值的代码：

```python
# 假设df是我们的DataFrame数据
diff_result = df.diff()
print(diff_result.head())
```

执行上述代码后，你会看到一个新的DataFrame。例如，1月5日的数据是0.052，这个值就是由1月5日的数据减去1月4日的数据得到的。对于数据的第一行（如1月4日），由于没有前一天的数据可减，结果会显示为`NaN`（空值）。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_10.png)

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_12.png)

`.diff()`方法默认计算的是与前一天（即间隔为1天）的差异。我们也可以计算间隔多天的差异。

以下是`.diff()`方法的一些参数说明：
*   `periods`：指定计算差异的间隔。例如，`periods=2`表示用当前值减去前两天的值。
*   其他参数可以查阅Pandas官方文档。

养成查阅API文档的习惯非常重要。遇到不熟悉的函数或参数，最直接的方法是复制函数名，通过搜索引擎或官方文档进行查询。自己动手实践和探索是最好的学习方法。

## 计算变化率（增长率）

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_16.png)

仅仅知道差异值有时还不够。例如，股价从1元涨到2元是涨了1元，从10000元涨到10001元也是涨了1元，但两者的意义完全不同。前者翻了一倍，后者几乎没变。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_18.png)

因此，我们常常需要计算变化率或增长率。其计算公式为：
**变化率 = (后一个值 - 前一个值) / 前一个值**

在Pandas中，我们可以使用`.pct_change()`方法来直接计算这个百分比变化。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_20.png)

以下是计算变化率的代码：

```python
# 计算每日的百分比变化（增长率）
pct_change_result = df.pct_change()
print(pct_change_result.head())
```

查看结果，你会发现数值变成了百分比形式，例如0.0012代表增长了0.12%。股票每日的涨跌幅通常就在这个量级。我们可以使用`.round()`方法来控制显示的小数位数，例如`pct_change_result.round(3)`保留三位小数。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_22.png)

## 对变化结果进行统计与可视化

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_24.png)

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_26.png)

计算出每日的变化率后，我们可能希望了解整体的平均变化情况。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_28.png)

我们可以对变化率数据计算平均值等统计指标。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_30.png)

以下是计算平均变化率的代码：

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_32.png)

```python
# 计算各列的平均变化率
mean_pct_change = pct_change_result.mean()
print(mean_pct_change)
```

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_34.png)

得到平均变化率后，我们可以将其可视化，以便更直观地比较不同指标的平均表现。

由于平均值是单个数值，适合用条形图来展示。我们可以使用Pandas内置的绘图功能。

以下是绘制平均变化率条形图的代码：

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_36.png)

```python
# 绘制平均变化率的条形图
mean_pct_change.plot(kind='bar', figsize=(16, 8))
```

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_38.png)

通过这张图，各个指标在一段时间内的平均增长率一目了然，便于我们进行横向比较。

![](img/b32b1d2645fe12f632f6ddd3a2fd0b9a_40.png)
