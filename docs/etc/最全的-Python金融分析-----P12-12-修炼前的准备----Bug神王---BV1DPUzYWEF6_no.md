## Jupyter Notebook：可视化开发工具

![](img/ed2c4dd42b8b332247efd8476abf13ce_1.png)

![](img/ed2c4dd42b8b332247efd8476abf13ce_3.png)

**Jupyter Notebook** 是Anaconda提供的一个基于浏览器的可视化开发工具。我们将在其中进行数据分析相关代码的编写与执行。

Jupyter无需单独安装，安装好Anaconda后即可使用。

### 启动与界面

以下是Jupyter Notebook的基本使用方法：

![](img/ed2c4dd42b8b332247efd8476abf13ce_5.png)

![](img/ed2c4dd42b8b332247efd8476abf13ce_7.png)

1.  **启动**：在终端中录入指令 `jupyter notebook` 并按下回车。
    ```bash
    jupyter notebook
    ```
2.  **界面**：指令执行后，会启动默认浏览器并打开一个页面，显示当前目录下的文件结构。

### 创建与使用文件

启动后，我们需要新建一个文件来编写代码。

*   **新建文件**：点击页面上的“New”按钮，选择“Python 3”。这将创建一个后缀为 `.ipynb` 的源文件。
*   **Cell（单元格）**：文件中默认有一个可编辑区域，称为“Cell”。我们可以在Cell中编写内容并执行。
    *   **示例**：在Cell中输入 `print(“Hello World”)`，点击“Run”按钮或使用快捷键即可执行并看到结果。

### Cell的两种模式

Cell有两种工作模式，用于不同目的：

*   **Code模式**：用于编写和运行程序代码。
*   **Markdown模式**：用于编写格式化的笔记和注释。您可以使用Markdown语法来设置标题、列表等样式。

**模式切换**：可以通过Cell上方的下拉菜单选择模式，也可以使用快捷键进行切换。

### 常用快捷键

熟练掌握快捷键能极大提升开发效率。以下是几个核心快捷键：

*   **添加Cell**：
    *   `A`：在当前选中Cell的上方插入一个新Cell。
    *   `B`：在当前选中Cell的下方插入一个新Cell。
*   **删除Cell**：`X` 删除当前选中的Cell。
*   **切换Cell模式**：
    *   `M`：将当前Cell切换到 **Markdown模式**。
    *   `Y`：将当前Cell切换到 **Code模式**。
*   **运行Cell**：`Shift + Enter` 运行当前Cell，并跳转到下一个Cell。
*   **代码补全**：`Tab` 键可以触发代码自动补全功能。
*   **查看帮助**：在函数名上使用 `Shift + Tab` 可以快速查看该函数的帮助文档。
