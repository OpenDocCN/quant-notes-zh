## 数据准备与指标计算 📊

上一节我们介绍了课程概述，本节中我们来看看如何准备数据和计算技术指标。这是构建任何量化策略的第一步。

首先，我们需要导入历史价格数据并计算策略所需的技术指标。

以下是数据准备与计算的核心步骤：

![](img/66c959ab11e6e57655e46888b3e438d1_8.png)

1.  **导入数据**：使用CSV文件加载欧元/美元在2019年至2022年间的5分钟K线数据。
    ```python
    # 示例代码：使用pandas读取CSV数据
    import pandas as pd
    data = pd.read_csv('eurusd_5min.csv')
    ```
2.  **数据清洗**：删除市场休市时最高价等于最低价的无效K线，并将时间索引转换为正确的GMT格式。
3.  **计算技术指标**：
    *   计算慢速EMA（长度50）和快速EMA（长度30）用于判断趋势。
    *   计算布林带（长度15，标准差1.5）用于生成交易信号。
    *   计算平均真实波幅（ATR）用于设定动态止损止盈。
    ```python
    # 示例代码：使用TA-Lib或pandas计算指标
    data['ema_slow'] = data['close'].ewm(span=50).mean()
    data['ema_fast'] = data['close'].ewm(span=30).mean()
    # 此处省略布林带和ATR的具体计算代码
    ```

完成这些步骤后，我们的数据框将包含开盘价、最高价、最低价、收盘价、成交量以及所有计算出的技术指标列。

## 交易信号生成 ⚡

在准备好数据并计算出指标后，本节我们将定义具体的交易逻辑来生成买卖信号。

信号生成分为两步：首先是趋势判断，其次是入场点触发。

**趋势判断**：我们通过比较快速EMA和慢速EMA来确定市场趋势。
*   如果 `快速EMA > 慢速EMA`，则判定为**上升趋势**，此时只考虑做多。
*   如果 `快速EMA < 慢速EMA`，则判定为**下降趋势**，此时只考虑做空。
为了减少噪音，我们要求趋势在最近N根K线（例如6根，代表过去30分钟）内保持一致。

**入场点触发**：在确定的趋势方向上，利用布林带边缘寻找入场时机。
*   在**上升趋势**中，当价格**向下穿越布林带下轨**时，触发**多头（买入）**信号。
*   在**下降趋势**中，当价格**向上穿越布林带上轨**时，触发**空头（卖出）**信号。
其逻辑假设是，价格在触及布林带边缘后，倾向于向中轨回归。

我们将上述逻辑编写成函数，应用到历史数据上，生成一列明确的交易信号（例如，1代表空头信号，2代表多头信号，0代表无信号）。

## 策略回测与结果分析 📈

生成了交易信号后，我们需要在历史数据上模拟交易，以评估策略的潜在表现。本节我们将进行回测并分析关键绩效指标。

![](img/66c959ab11e6e57655e46888b3e438d1_10.png)

我们使用一个简单的回测引擎，初始资金设为250美元，并应用30倍杠杆。对于每一笔交易：
*   **止损**：根据入场点的ATR值动态设置。公式为：`止损距离 = 止损系数 * ATR`。
*   **止盈**：根据止损距离和固定的盈亏比设置。公式为：`止盈距离 = 止损距离 * 止盈止损比率`。

![](img/66c959ab11e6e57655e46888b3e438d1_12.png)

以下是回测得到的关键结果：
*   **总回报率**：超过25%（同期买入持有策略回报约为-7%）。
*   **胜率**：约40%。
*   **交易次数**：1671次。
*   **最大回撤**：约-17%。

从净值曲线图可以看出，在近三个月的测试期内，资产曲线整体呈现稳步增长，期间存在一些盘整期，但持续时间相对较短。这表明该策略框架具备盈利潜力。
