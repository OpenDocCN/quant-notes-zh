## 数据准备与处理

上一节我们介绍了`plot`函数及其周边功能。本节中我们来看看pandas库与Matplotlib库之间的便捷联系。

例如，我们之前创建了一个DataFrame。现在，我们希望将这个DataFrame输出为图表。我们可以使用之前的一个股票数据文件（例如`601318.csv`）进行演示。

首先，我们加载并查看数据。与之前操作一样，我们需要将日期列转换为日期时间对象，并将其设置为索引列。

```python
import pandas as pd

# 假设 df 是从 '601318.csv' 加载的DataFrame
df['date'] = pd.to_datetime(df['date'])
df.set_index('date', inplace=True)
```

由于数据中可能包含字符串列或无用的序列，我们使用花式索引筛选出需要的几列价格数据，例如开盘价(`Open`)、收盘价(`Close`)、最高价(`High`)和最低价(`Low`)。

```python
price_df = df[['Open', 'Close', 'High', 'Low']]
```

## 直接绘制DataFrame

现在，如果我们想将这个处理好的DataFrame直接绘制成图表，有一个非常简便的方法。

我们可以直接在DataFrame对象上调用`.plot()`方法，然后使用`plt.show()`显示图表。

```python
import matplotlib.pyplot as plt

price_df.plot()
plt.show()
```

执行上述代码后，Matplotlib会智能地生成一个图表。在这个图表中，索引列（即日期）会自动成为X轴的坐标，而筛选出的四列价格数据则成为Y轴数据，绘制成四条不同颜色的曲线。

图表可能显示为似乎只有一条线，这是因为数据量较大，且每日的开盘价、收盘价、最高价和最低价数值通常比较接近，导致多条线重叠在一起。仔细观察，仍能看到不同颜色的线条。如果想看得更清晰，可以在弹出的图表窗口中手动调整视图。

![](img/56975201d934c3b26b8d733cbbd20878_1.png)

这种方法同样适用于Series对象。Series是单列数据，调用`.plot()`方法将只绘制一条曲线。

**核心关系公式**：
`DataFrame.plot() -> Matplotlib Figure`
`Series.plot() -> Matplotlib Line`
