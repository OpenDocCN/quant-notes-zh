## 模型评估方法
既然模型预测存在偏差，我们就需要系统地评估它。本节中，我们来看看如何量化评估模型的性能。

评估主要从两个方面进行：

![](img/b52cdf173b4d7feb6d3337839165015a_41.png)

1.  **预测误差**：计算预测值与真实值之间的差异。我们可以将历史数据划分为训练集和测试集。例如，用2017年之前的数据训练，用2017年的数据测试，然后分别计算训练集和测试集上的平均误差。误差公式可以表示为：
    **误差 = |预测值 - 真实值|**

2.  **趋势预测准确性**：评估模型是否正确预测了股价的上升或下降趋势。如果真实趋势是上升的，模型预测是否也是上升？反之亦然。我们需要分别统计模型对上升趋势和下降趋势的预测正确率。

![](img/b52cdf173b4d7feb6d3337839165015a_25.png)

![](img/b52cdf173b4d7feb6d3337839165015a_27.png)

## 评估过程详解
下面我们通过代码调试（Debug）的方式，一步步理解评估过程。对于复杂代码，逐行调试观察变量值的变化是很好的学习方法。

首先，我们划分数据。设定训练集为3年数据（例如2014-2016年），测试集为1年数据（2017年）。测试集必须包含已知的真实值，因此只能使用历史数据。

接着，用训练集数据拟合（`fit`）Prophet模型。然后，用这个模型预测（`predict`）测试集时间段（未来365天）的股价。

预测完成后，我们得到预测值序列和测试集的真实值序列，用于计算评估指标。

以下是计算趋势准确性的核心步骤：
1.  计算每日股价的一阶差分（今日价 - 昨日价），得到 `difference`。正数表示当日相对于前一日上涨，负数表示下跌。
2.  使用 `np.sign()` 函数将差分值转化为趋势标签：正数为1（上涨），负数为-1（下跌），零为0。
3.  分别对比预测序列和真实序列的趋势标签，统计预测正确的比例。

![](img/b52cdf173b4d7feb6d3337839165015a_43.png)

```python
# np.sign(x) 返回值规则：

# 如果 x == 0， 返回 0
# 如果 x < 0， 返回 -1
```

同时，我们计算了平均绝对误差（MAE）。训练集的平均误差约为18美元，而测试集的平均误差高达160多美元，这表明模型在未知数据上泛化能力较差。
