## 增长率累加的问题

上一节我们介绍了如何计算增长率。本节中我们来看看，如果我们想计算一段时间内的“总”增长率，能否将每天的增长率简单相加。

例如，一只股票第一天的价格是100元，第二天跌至50元，第三天涨至75元。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_4.png)

*   第一天的增长率是 `(50 - 100) / 100 = -0.5`
*   第二天的增长率是 `(75 - 50) / 50 = 0.5`

如果将这两个增长率相加：`-0.5 + 0.5 = 0`。这个结果为零，似乎意味着股票从第一天到第三天既没涨也没跌。

然而，观察原始数据：第一天100元，第三天75元，股票实际上是下跌的。这说明，**增长率不能直接进行加法运算**，因为简单的累加无法反映第一天与最后一天之间的真实整体变化。

## 使用对数计算连续变化

对于连续时间序列的累加操作，我们需要借助数学中的对数公式。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_6.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_8.png)

核心思想是：将对数形式的每日变化率相加，等价于计算首日与末日价格的直接比率。

**公式推导如下：**

![](img/8bbe880e13807f9cff65d03b4f10fe9b_10.png)

假设我们计算每日的对数收益率：`log(后一天价格 / 前一天价格)`。

对于上述例子：
*   第一天到第二天：`log(50 / 100)`
*   第二天到第三天：`log(75 / 50)`

将对数收益率相加：
`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/8bbe880e13807f9cff65d03b4f10fe9b_12.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_14.png)

最终结果 `log(75/100)` 直接反映了第三天价格（75）与第一天价格（100）的比率关系。通过对数累加，我们巧妙地得到了反映整体变化的指标。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_16.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_18.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_20.png)

形象地说，这相当于计算：最初投资1元钱，在经过一系列涨跌后，在最终时刻相当于多少钱。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_22.png)

### 1. 计算对数收益率

首先，我们需要计算每日的对数收益率，即 `log(当日价格 / 前一日价格)`。这里使用 `shift()` 函数来获取前一日的数据。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_32.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_34.png)

```python
import numpy as np
import pandas as pd

![](img/8bbe880e13807f9cff65d03b4f10fe9b_36.png)

# 假设 data 是一个包含股价的Pandas Series
# 计算对数收益率
log_return = np.log(data / data.shift(1))
```

### 2. 计算累积收益率

![](img/8bbe880e13807f9cff65d03b4f10fe9b_38.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_40.png)

接下来，我们需要将对数收益率进行累积求和，以得到从起始日到每一天的累积变化。这可以通过 `cumsum()` 函数实现。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_42.png)

```python
# 计算累积对数收益率
cumulative_log_return = log_return.cumsum()
```

![](img/8bbe880e13807f9cff65d03b4f10fe9b_44.png)

### 3. 还原为实际价值倍数

![](img/8bbe880e13807f9cff65d03b4f10fe9b_46.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_48.png)

最后，将累积对数收益率通过指数函数 `exp()` 还原，即可得到“初始1单位投资在每一天对应的价值”。

```python
# 还原为实际价值倍数 (初始1元对应的价值)
cumulative_value = np.exp(cumulative_log_return)
```

![](img/8bbe880e13807f9cff65d03b4f10fe9b_50.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_52.png)

### 4. 可视化结果

我们可以将结果绘制成图，直观展示初始投资随时间增长的情况。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_54.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_56.png)

```python
import matplotlib.pyplot as plt

cumulative_value.plot(figsize=(16, 8))
plt.title('初始1单位投资的价值增长曲线')
```

本节课中我们一起学习了金融时间序列中连续变化情况的分析方法。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_66.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_68.png)

1.  **增长率不可直接累加**：我们通过实例说明了简单累加每日增长率会得到错误结论，无法反映资产价值的真实整体变化。
2.  **引入对数方法**：我们引入了对数收益率 `log(P_t / P_{t-1})` 的概念。其核心优势在于，对数收益率的累加 `∑log(P_t / P_{t-1})` 等价于计算 `log(P_T / P_0)`，完美反映了期初与期末的整体关系。
3.  **Python实现**：我们使用 `shift()` 获取滞后数据，用 `np.log()` 计算对数收益率，用 `cumsum()` 进行累积，最后用 `np.exp()` 还原为易于理解的价值倍数，并通过绘图进行可视化。

![](img/8bbe880e13807f9cff65d03b4f10fe9b_70.png)

![](img/8bbe880e13807f9cff65d03b4f10fe9b_72.png)

通过掌握这种方法，你可以准确地分析任何金融资产（如股票、基金）在特定时间段内的真实累积收益，这是量化分析和投资回顾中的一个关键技能。
