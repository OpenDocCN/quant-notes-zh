## Excel发展简史

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_12.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_13.png)

上一节我们介绍了课程的整体安排，本节中我们来看看Excel的起源与发展。Excel并非第一款电子表格软件，它的成功与操作系统的发展紧密相连。

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_15.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_16.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_17.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_19.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_20.png)

世界上第一款电子表格软件是1978年发布的**VisiCalc**。它诞生的原因是，在此之前人们使用纸笔或黑板进行数据计算，修改一个数字就需要重新计算所有相关结果，非常繁琐。VisiCalc确立了电子表格的基本结构：列由字母（A, B, C...）标识，行由数字（1, 2, 3...）标识，构成一个二维表格。

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_21.png)

随后，1983年出现了更成功的电子表格软件**Lotus 1-2-3**。它集成了三大功能：表格计算、图表和简易数据库，因此在DOS系统时代非常流行。

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_23.png)

微软公司早在1982年就推出了电子表格**Multiplan**，但未能竞争过Lotus 1-2-3。为此，微软秘密研发，于1985年推出了**Excel 1.0**（最初用于Mac系统）。Excel真正成为市场主导，是在1987年微软推出Windows操作系统后，采取了将Excel与Windows绑定的销售策略。而Lotus 1-2-3未能开发出好用的Windows版本，因此逐渐被市场淘汰。

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_25.png)

此后，Excel持续迭代更新，几个关键版本节点包括：
*   **1993年 Excel 5.0**：被集成到Microsoft Office套件中，并开始支持VBA编程语言。
*   **1997年 Excel 97**：这是一个功能得到重大增强的里程碑版本。
*   **2003年 Excel 2003**：该版本在中国市场非常普及。
*   **2007年及以后版本**：界面改为“功能区”（Ribbon），文件后缀名变为`.xlsx`，功能上更智能、更美观。

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_27.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_29.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_31.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_33.png)

目前，常见的版本主要分为两类：
*   **97-2003版**：文件后缀名为 **`.xls`**
*   **2007版及以后**：文件后缀名为 **`.xlsx`**

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_35.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_36.png)

## Excel界面与核心概念

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_45.png)

了解了历史后，我们正式打开Excel，认识其工作界面和核心组成部分。

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_47.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_48.png)

启动Excel后打开的文件称为**工作簿**。一个工作簿可以包含多个**工作表**，就像一本笔记本包含多页纸。

### 功能区与选项卡
窗口上方的区域称为**功能区**，它汇集了Excel的所有操作命令。功能区按功能划分为多个**选项卡**（如“开始”、“插入”、“公式”等）。
*   **操作技巧**：双击当前选中的选项卡标签（如“开始”），可以折叠或展开功能区，以节省屏幕空间。

### 单元格、行与列
工作表由网格构成，每个小格子称为**单元格**，它是存储数据的基本单位。
*   **单元格地址**：每个单元格的位置由其所在的**列标**（字母）和**行号**（数字）唯一确定，例如`A1`、`B3`。引用单元格地址是Excel公式计算的基础。
*   **编辑栏**：选中单元格后，编辑栏会显示该单元格的地址和内容。当单元格内是公式时，编辑栏显示公式本身，而单元格显示公式计算结果。
*   **工作表容量**：一个工作表最多有 **1,048,576行** 和 **16,384列**（XFD列）。虽然容量大，但处理超过10万行数据时性能会显著下降，此时应考虑使用Python等编程工具。

### 工作表基本操作
以下是针对工作表的常用操作：
*   **添加工作表**：点击工作表标签右侧的“+”号。
*   **重命名工作表**：双击工作表标签，或右键点击标签选择“重命名”。
*   **设置工作表标签颜色**：右键点击标签，选择“工作表标签颜色”。
*   **删除工作表**：右键点击标签，选择“删除”。
*   **移动或复制工作表**：
    *   在**同一工作簿内**移动：直接拖动工作表标签。
    *   在**同一工作簿内**复制：右键点击标签 -> “移动或复制” -> 勾选“建立副本”。
    *   **在不同工作簿间**移动或复制：右键点击标签 -> “移动或复制” -> 在“工作簿”下拉列表中选择目标工作簿 -> 选择位置（可选“建立副本”）。

### 冻结与拆分窗口
当表格数据很多时，滚动屏幕会导致标题行或列不可见。使用“冻结窗格”功能可以锁定特定区域。
*   **冻结首行/首列**：点击“视图”选项卡 -> “冻结窗格” -> 选择“冻结首行”或“冻结首列”。
*   **冻结多行多列**：选中你想冻结区域右下方的第一个单元格（例如，想冻结前两行和前两列，则选中`C3`单元格）-> 点击“视图”选项卡 -> 先点击“拆分”，再点击“冻结窗格”。

## 常用快捷方式
掌握快捷键能极大提升操作效率。以下是一些最常用的Excel快捷操作：

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_56.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_58.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_60.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_62.png)

![](img/607f72e4ae0679f8e43aa8833cdd2f5e_64.png)

*   **`Ctrl + N`**：新建工作簿
*   **`Ctrl + S`**：保存工作簿
*   **`Ctrl + C` / `Ctrl + X` / `Ctrl + V`**：复制 / 剪切 / 粘贴
*   **`Ctrl + Z` / `Ctrl + Y`**：撤销 / 恢复
*   **`Ctrl + F`**：查找
*   **`Ctrl + H`**：替换
*   **`Ctrl + 方向键`**：快速跳转到数据区域的边缘
*   **`Ctrl + Shift + 方向键`**：快速选中从当前单元格到数据区域边缘的范围
*   **`F2`**：编辑当前选中的单元格
*   **`F4`**：重复上一步操作；在编辑公式时，循环切换单元格引用类型（绝对引用/相对引用）
