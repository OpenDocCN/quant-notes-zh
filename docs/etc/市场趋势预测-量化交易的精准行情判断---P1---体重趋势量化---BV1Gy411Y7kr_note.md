# 市场趋势预测：量化交易的精准行情判断 - P1：体重趋势量化

## 概述
在本节课中，我们将要学习如何将量化交易中判断市场趋势的核心思想，应用于一个简单直观的模型——体重趋势预测。通过这个类比，初学者可以轻松理解量化分析的基本逻辑和步骤。

## 从体重变化理解趋势
量化交易的核心之一是判断市场趋势。为了便于理解，我们可以用一个生活中的例子——体重变化——来模拟这个过程。就像我们关心体重是上升还是下降一样，交易者关心的是资产价格的整体方向。

上一节我们介绍了趋势判断的重要性，本节中我们来看看如何具体量化一个趋势。

## 量化体重的“上涨”与“下跌”
要判断趋势，首先需要定义什么是“上涨”和“下跌”。在体重模型中，我们设定一个简单的规则：如果今天的体重比昨天重，则记为一次“上涨”；如果比昨天轻，则记为一次“下跌”。

以下是具体的判断逻辑，可以用伪代码表示：

```python
if 今日体重 > 昨日体重:
    趋势状态 = “上涨”
elif 今日体重 < 昨日体重:
    趋势状态 = “下跌”
else:
    趋势状态 = “持平”
```

通过连续多日应用这个规则，我们就可以得到一系列“上涨”和“下跌”的标记，这是分析趋势的基础数据。

## 计算连续上涨/下跌的天数
得到每日的状态标记后，下一步是计算连续上涨或下跌的天数。这能帮助我们判断趋势的强度和持续性。例如，连续上涨5天比涨1天跌1天的模式更能表明一个明确的上升趋势。

以下是计算连续天数的步骤描述：
1.  从第一天的数据开始检查。
2.  如果当前状态与前一天相同，则连续天数加1。
3.  如果状态发生变化，则重置连续天数，并从1开始重新计算。
4.  记录下每一段连续上涨或下跌的持续长度。

这个过程可以帮助我们过滤掉日常波动，抓住主要的价格运动方向。

## 定义并识别“趋势”
有了连续天数的数据，我们就可以正式定义“趋势”。在这个模型中，我们可以规定：**当连续上涨或下跌的天数达到或超过某个阈值（例如3天）时，就认为形成了一个“趋势”**。

这个阈值是一个关键参数，它决定了我们对趋势的敏感度。阈值设得小，容易识别出趋势但可能包含更多“噪音”；阈值设得大，识别出的趋势更可靠但可能反应滞后。

## 总结
本节课中我们一起学习了量化分析趋势的基本框架。我们通过体重变化的例子，逐步讲解了如何定义涨跌、计算连续天数，并最终通过设定阈值来识别趋势。这个过程的核心思想是**将主观的趋势感受转化为客观的、可计算的规则**，这正是量化交易的基石。在接下来的课程中，我们将把这些概念应用到更复杂的金融市场数据中。
