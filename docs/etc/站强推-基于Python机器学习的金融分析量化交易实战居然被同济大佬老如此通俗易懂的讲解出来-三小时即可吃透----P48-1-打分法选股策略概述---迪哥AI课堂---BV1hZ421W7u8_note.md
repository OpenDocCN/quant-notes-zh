# 量化交易入门：P48：1-打分法选股策略概述

在本节课中，我们将学习一种在量化选股中常用的策略——因子打分选股法。我们将了解其核心思想、实施前提，并初步探讨如何将多个因子综合成一个总分来筛选股票。

## 策略核心思想

上一节我们介绍了因子分析，本节中我们来看看如何综合运用多个因子。因子打分选股的核心思想是：**不依赖单一因子，而是综合多个因子对股票进行综合评价和筛选**。

我们可以将这个过程类比为学生考试：
*   每个股票相当于一个学生。
*   每个因子（如市盈率、净资产收益率）相当于一门科目（如语文、数学）。
*   我们需要为每只股票在每个因子上的表现进行“打分”。
*   最后，将所有因子的分数相加，得到一个“总分”。
*   根据总分对所有股票进行排名，选择排名靠前的股票作为投资组合。

这种方法旨在通过多维度评估，选出整体“素质”最优的股票。

## 策略实施前提

在开始打分之前，我们必须明确一个关键的先验知识：**每个因子与股票预期收益率的相关性方向**。

以下是需要预先确定的两类因子：
*   **正向因子**：因子值**越大**，通常预期收益率越高。例如：净资产收益率(ROE)、每股收益(EPS)。
    *   公式表示：`因子值 ↑ → 预期收益 ↑`
*   **负向因子**：因子值**越小**，通常预期收益率越高。例如：市盈率(PE)、市净率(PB)。
    *   公式表示：`因子值 ↓ → 预期收益 ↑`

这个方向的判断至关重要，因为它决定了我们打分的规则——对于正向因子，数值越大得分应越高；对于负向因子，数值越小得分应越高。

如何获取这个先验知识呢？主要有两个途径：
1.  查阅券商、研究机构发布的相关研究报告，其中通常会总结因子的有效性和方向。
2.  通过历史数据进行量化分析，例如计算因子与未来收益的**信息系数(IC)** 来验证其相关性方向。

## 策略流程概述

基于以上思想，一个典型的因子打分选股策略流程可以概括为以下几步：

以下是策略执行的基本步骤：
1.  **确定股票池与因子**：选定待筛选的股票范围（如沪深300），并挑选出若干已知方向的因子（A, B, C, D...）。
2.  **因子数据预处理**：获取所有股票在当期各因子的原始数据。
3.  **因子打分**：根据每个因子的方向（正向/负向），将原始数据转换为标准化分数（例如百分制排名分）。
4.  **计算综合得分**：将每只股票在所有因子上的得分相加（或加权平均），得到该股票的综合总分。
    *   公式表示：`股票综合总分 = 因子A得分 + 因子B得分 + ... + 因子N得分`
5.  **排序与选股**：根据综合总分对所有股票进行降序排序，选取排名前N（如前10或20）的股票。
6.  **定期调仓**：策略通常按固定周期（如每月）重复执行以上步骤，更新投资组合。
