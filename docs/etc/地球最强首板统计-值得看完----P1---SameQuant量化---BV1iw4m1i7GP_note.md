# 量化交易入门：P1：首板涨停统计全解析 📊

在本节课中，我们将学习如何对A股市场的“首板”（即个股的首次涨停）进行量化统计分析。我们将通过数据揭示首板的封板成功率、次日表现等关键特征，并理解为何无脑买入首板的策略长期来看难以盈利。

## 概述与统计结果 📈

首先，我们运行程序得到了A股历史上首板的整体统计数据。以下是核心统计结果：

*   **历史首板总次数**：270,122次
*   **涨停总数**：176,156次
*   **炸板总数**：93,966次
*   **首板非一字板涨停总数**：167,683次

基于以上数据，我们计算出以下关键指标：

*   **整体首板封板率**：65.21%
*   **排除一字板后的首板封板率**：64.09%
*   **首板涨停次日收涨概率**：62.31%
*   **首板一字板次日收涨概率**：85.31%
*   **首板炸板次日收涨概率**：41.71%
*   **首板涨停次日平均涨幅**：1.93%
*   **排除一字板后，首板涨停次日平均涨幅**：1.74%
*   **首板炸板当日平均浮亏比例**：-2.96%
*   **首板炸板次日平均涨幅**：-0.44%

## 首板的定义与数据范围 ⚙️

上一节我们看到了统计结果，本节中我们来看看这些数据是基于怎样的规则计算出来的。我们对“首板”的定义包含以下几个条件：

1.  **包含涨停与炸板**：统计对象既包括成功封住涨停的“涨停板”，也包括触及涨停但未能封住的“炸板”。
2.  **前期涨停次数限制**：该股票在过去10个交易日内的涨停次数小于等于5次。
3.  **昨日非涨停**：该股票在前一个交易日没有涨停。
4.  **上市时间过滤**：只统计上市超过20个交易日后的数据，以排除新股上市初期连续一字板的影响。
5.  **时间范围**：统计数据从1996年开始，因为A股涨跌停板制度于该年实施。

## 首板策略的长期收益模拟 💹

基于上述统计数据，我们可以模拟一个简单的策略：在每只股票出现首板时（无论是否一字板）都无脑买入，并持有至次日收盘卖出。

以下是预估净值计算公式：
`预估净值 = 初始资金 * (封板率 * (1 + 涨停次日平均涨幅) + 炸板率 * (1 + 炸板当日浮亏比例) * (1 + 炸板次日平均涨幅))`

代入我们的统计数字（未扣除交易手续费）：
`预估净值 ≈ 1 * (65.21% * (1 + 1.93%) + 34.79% * (1 - 2.96%) * (1 - 0.44%)) ≈ 0.998`

![](img/a9de264185a7eb2affe62bf75ddf4ffe_1.png)

**计算结果小于1**，这意味着长期执行此策略将导致亏损。这解释了为何单纯模仿“北京炒家”等成功打板选手的无脑首板策略很难复制，他们的成功离不开深厚的主观交易经验。

**第一步：生成历史首板数据**
遍历所有A股股票的历史日线数据，根据“首板的定义与数据范围”一节中的条件进行筛选，将符合条件的首板记录（包括日期、是否涨停、是否炸板、次日涨跌幅等）保存到一个CSV文件中。这个文件可能包含超过27万行记录。

**第二步：进行统计分析**
读取第一步生成的CSV文件，使用`pandas`等数据分析库进行快速计算。

以下是核心统计的计算方法示例（以Python伪代码表示）：

```python
import pandas as pd

stock_stats = df.groupby(‘code’).apply(lambda x: pd.Series({

    ‘封板率’: len(x[x[‘is_zt’] == True]) / len(x) if len(x) > 0 else 0
}))
# 按封板率降序排列得到Top榜
top_10 = stock_stats.sort_values(by=‘封板率’, ascending=False).head(10)
```

其他指标，如炸板相关数据、排除一字板后的数据等，均可通过类似的筛选与分组聚合操作完成。
