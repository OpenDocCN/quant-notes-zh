# 量化交易入门：P1：必须掌握的基础知识 📚

在本节课中，我们将要学习进入量化交易领域所必须掌握的基础编程知识。量化交易的核心是使用计算机程序来分析和执行交易策略，因此扎实的编程基础是第一步。我们将从Python语言基础开始，逐步介绍几个关键的库，为你后续的策略开发打下坚实基础。

## Python基础 🐍

上一节我们概述了量化交易对编程的要求，本节中我们来看看具体的Python基础知识。掌握Python是量化开发的基石，你需要理解其核心的数据结构和编程概念。

以下是Python基础中需要掌握的核心数据结构：

*   **字符串**：用于处理文本信息。
*   **列表**：一种有序、可变的集合，例如 `[1, 2, 3]`。
*   **元组**：一种有序、不可变的集合，例如 `(1, 2, 3)`。
*   **字典**：一种键值对映射的集合，例如 `{‘name’: ‘Alice’, ‘age’: 25}`。
*   **布尔值**：逻辑判断的基础，取值为 `True` 或 `False`。

除了数据结构，函数是组织代码的核心。你需要理解函数的基本构成。

以下是关于函数需要了解的几个关键概念：

*   **函数名**：函数的标识符。
*   **传参**：向函数传递数据的方式。
*   **返回值**：函数执行后返回的结果。
*   **递归函数**：函数调用自身的一种编程技巧。
*   **闭包**：一个函数与其相关引用环境组合而成的实体。
*   **装饰器**：用于修改或增强函数功能的高级语法。

面向对象编程是理解和使用许多第三方库的关键。在量化中，我们经常需要调用封装好的类来获取数据或进行计算。

以下是关于类和对象的核心概念：

*   **类**：创建对象的蓝图或模板，定义了属性和方法。
*   **对象**：类的实例，是具体存在的数据实体。
*   **实例化**：根据类创建对象的过程，例如 `obj = MyClass()`。
*   **方法调用**：通过对象调用类中定义的函数，例如 `obj.calculate()`。

最后，为了编写健壮的程序，你需要掌握基本的错误处理机制。

*   **try-except**：用于捕获和处理代码运行时的异常。

## 核心库：NumPy与Pandas 📊

掌握了Python基础后，我们就可以开始学习量化数据处理的核心工具了。NumPy和Pandas是处理数值和表格数据的利器，几乎所有的量化分析都离不开它们。

NumPy提供了高性能的多维数组对象。它的核心数据结构是`ndarray`（数组），与Python列表不同，它更高效且支持向量化运算。

```python
import numpy as np
# NumPy数组
arr = np.array([1, 2, 3, 4, 5])
```

Pandas构建在NumPy之上，提供了更高级的数据结构，主要用于处理表格型或异质型数据。它是量化数据分析的绝对核心。

Pandas有两种主要数据结构：

*   **Series**：带标签的一维数组，可以看作一个字典或Excel中的一列。
*   **DataFrame**：二维的、表格型数据结构，包含行索引和列标签，是数据分析的主要工具。

以下是Pandas中必须熟练掌握的几个操作：

*   **数据选取**：使用 `iloc`（基于整数位置）和 `loc`（基于标签）进行数据切片和选择。
*   **应用函数**：使用 `apply` 函数对`Series`或`DataFrame`的行/列应用自定义函数。
*   **分组聚合**：使用 `groupby` 进行数据分组，然后进行求和、求平均等聚合操作。
*   **排序**：使用 `sort_values` 函数对数据进行排序。

## 其他工具与学习方法 🔧

在熟练运用Python、NumPy和Pandas之后，你就可以开始接触具体的量化工具了。这些工具通常以库的形式提供，你需要学会查阅其文档并使用它们。

以下是开始实践时需要接触的内容：

*   **数据获取库**：例如 `tushare`、`akshare` 等用于获取股票、财经数据的库。
*   **回测框架**：各大量化平台（如JoinQuant、RiceQuant）提供的策略编写和回测API文档。

如何快速掌握这些新工具呢？关键在于你之前打下的基础。理解了面向对象编程，你就能看懂这些库的文档，知道如何创建对象和调用方法。结合Pandas的数据处理能力，你就能获取数据、清洗数据，并最终实现自己的交易策略进行回测。
