## 求平均值函数

![](img/3e47467548fc3fadeaa9709f4ca90c54_3.png)

上一节我们介绍了缺失值处理，本节中我们来看看如何计算数据的平均值。pandas中的`mean()`方法用于计算平均值。

对于DataFrame对象，`mean()`方法默认按列计算平均值。因为DataFrame可能包含多列数据，所以该方法会返回一个Series对象，其中包含每一列的平均值。

**代码示例**：
```python
df.mean()
```
执行上述代码，会分别计算DataFrame中每一列（忽略缺失值）的平均值，并返回结果。

如果想按行计算平均值，需要指定参数`axis=1`。

**代码示例**：
```python
df.mean(axis=1)
```
这样就会计算每一行的平均值。

## 求和函数

与求平均值类似，pandas也提供了求和函数`sum()`。

`sum()`方法默认也是按列求和。如果需要按行求和，同样需要指定参数`axis=1`。

**代码示例**：
```python
df.sum()        # 按列求和
df.sum(axis=1)  # 按行求和
```

## 排序函数

排序是数据分析中常见的操作。pandas提供了两种主要的排序方式：按值排序和按索引排序。

### 按值排序

按值排序使用`sort_values()`方法。你需要通过`by`参数指定按照哪一列的值进行排序。

**代码示例**：
```python
df.sort_values(by=‘two‘)
```
默认是升序排列。如果需要降序排列，可以设置参数`ascending=False`。

**代码示例**：
```python
df.sort_values(by=‘two‘, ascending=False)
```
当排序的列中存在缺失值（NaN）时，所有包含NaN的行不会参与排序，它们会被统一放置在结果的最后。

### 按索引排序

按索引排序使用`sort_index()`方法。如果不传递参数，默认会按行索引进行排序。

**代码示例**：
```python
df.sort_index()                     # 按行索引升序排序
df.sort_index(ascending=False)      # 按行索引降序排序
```
如果想对列索引（即列标签）进行排序，需要指定参数`axis=1`。

**代码示例**：
```python
df.sort_index(axis=1)               # 按列索引升序排序
df.sort_index(axis=1, ascending=False) # 按列索引降序排序
```

## 其他通用函数

除了上述函数，之前在NumPy库中学习过的许多通用函数同样适用于pandas。

以下是部分同样可以在pandas中使用的函数：
*   `std()`: 计算标准差。
*   `var()`: 计算方差。
*   `max()`: 查找最大值。
*   `min()`: 查找最小值。

这些函数的使用方式与`mean()`和`sum()`类似，默认按列计算，也可以通过`axis`参数指定按行计算。
