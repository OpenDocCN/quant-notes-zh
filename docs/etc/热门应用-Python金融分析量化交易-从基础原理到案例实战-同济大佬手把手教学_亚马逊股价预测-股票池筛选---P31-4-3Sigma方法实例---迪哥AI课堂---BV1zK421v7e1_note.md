## 概述

上一节我们介绍了去极值的概念和几种常见方法。本节中，我们来看看一种基于统计学原理的、非常直观的去极值方法：3-Sigma方法。它的核心思想是利用数据的正态分布特性，将偏离均值过远的数据视为异常值并进行处理。

## 3-Sigma方法原理

3-Sigma方法的原理非常简单。大家先来看下面这张正态分布图。

![](img/8c1915bc5f3704dc6b032a87430b1e58_1.png)

在正态分布中，数据大多集中在均值附近。图中，我们画出了以均值为中心，分别加减1倍、2倍和3倍标准差（Sigma）的区间。

![](img/8c1915bc5f3704dc6b032a87430b1e58_1.png)

*   离均值越近的区间（例如±1σ），覆盖的数据越多，代表这些事件发生的概率越高。
*   离均值越远的区间（例如±3σ），覆盖的数据越少，代表这些事件发生的概率越低。

我们要处理的极值（离群点），通常就落在这些远离均值的、概率很低的区域。所谓“3-Sigma”，就是指以均值为中心，正负3倍标准差的区间。

![](img/8c1915bc5f3704dc6b032a87430b1e58_3.png)

根据统计学知识，在标准正态分布中：
*   **均值 ± 1σ** 包含了约 **68.3%** 的数据。
*   **均值 ± 2σ** 包含了约 **95.4%** 的数据。
*   **均值 ± 3σ** 包含了约 **99.7%** 的数据。

因此，3-Sigma方法的思想是：假设数据服从或近似服从正态分布，我们计算出均值（μ）和标准差（σ），然后定义数据的正常范围下限为 **μ - 3σ**，上限为 **μ + 3σ**。我们将落在这个范围之外的数据（约0.3%）视为极值，并将其规范到边界上。

## Python代码实现

接下来，我们用Python代码来实现3-Sigma方法。我们将编写一个函数，输入一列数据和一个倍数`n`（例如3），返回处理后的数据序列。

![](img/8c1915bc5f3704dc6b032a87430b1e58_3.png)

以下是实现步骤：

![](img/8c1915bc5f3704dc6b032a87430b1e58_5.png)

首先，我们需要计算数据的均值（mean）和标准差（std）。

![](img/8c1915bc5f3704dc6b032a87430b1e58_7.png)

```python
def sigma_method(data, n=3):
    # 计算均值和标准差
    mean = data.mean()
    std = data.std()
```

然后，根据公式计算上限和下限：
*   **上限 (upper_limit) = mean + n * std**
*   **下限 (lower_limit) = mean - n * std**

![](img/8c1915bc5f3704dc6b032a87430b1e58_9.png)

```python
    # 计算上下限
    upper_limit = mean + n * std
    lower_limit = mean - n * std
```

最后，将超出范围的数据规范到边界上，并返回新的序列。

![](img/8c1915bc5f3704dc6b032a87430b1e58_11.png)

![](img/8c1915bc5f3704dc6b032a87430b1e58_13.png)

```python
    # 将超出上限的值设为上限，低于下限的值设为下限
    data_clipped = data.clip(lower=lower_limit, upper=upper_limit)
    return data_clipped
```
