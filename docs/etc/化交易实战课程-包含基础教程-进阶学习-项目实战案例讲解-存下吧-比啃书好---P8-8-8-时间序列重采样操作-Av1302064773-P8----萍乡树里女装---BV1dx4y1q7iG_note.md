# 量化交易教程：P8：时间序列重采样与窗口操作 📊

在本节课中，我们将要学习时间序列分析中的两个核心操作：**重采样**与**时间窗口操作**。重采样用于改变数据的时间频率，而窗口操作则用于在滑动的时间段内进行统计分析。掌握这些技能对于金融数据分析和量化交易策略构建至关重要。

## 重采样操作 🔄

![](img/267613afcdeeae9c921063e986468a4a_3.png)

![](img/267613afcdeeae9c921063e986468a4a_4.png)

上一节我们介绍了时间序列的基础概念，本节中我们来看看如何改变数据的时间频率，即重采样。

重采样是指将时间序列数据从一个频率转换到另一个频率的过程。例如，原始数据可能是按天统计的，我们可以将其转换为按周、按月或按年统计。

![](img/267613afcdeeae9c921063e986468a4a_6.png)

![](img/267613afcdeeae9c921063e986468a4a_7.png)

![](img/267613afcdeeae9c921063e986468a4a_8.png)

在Pandas中，使用`resample()`方法进行重采样。你需要至少指定一个参数，即目标频率（如`‘W’`代表周，`‘M’`代表月）。同时，你需要指定一个聚合函数（如均值、第一个值、中位数等）来计算新频率下的数据点。

![](img/267613afcdeeae9c921063e986468a4a_10.png)

以下是重采样的基本代码结构：
```python
# 假设df是一个以日期为索引的DataFrame
# 按周重采样，并计算每周的均值
weekly_data = df.resample(‘W’).mean()
# 按月重采样，并取每月的第一个值
monthly_data = df.resample(‘M’).first()
```

执行重采样后，你会得到一份按新频率聚合后的结果。例如，原始数据从1月1日开始，按周重采样后，第一周的数据点可能被标记为1月7日（该周的最后一天）。

### 重采样的标签设置

![](img/267613afcdeeae9c921063e986468a4a_12.png)

![](img/267613afcdeeae9c921063e986468a4a_13.png)

![](img/267613afcdeeae9c921063e986468a4a_14.png)

在重采样时，我们还可以指定如何标记聚合后的数据点。这通过`label`参数控制。

![](img/267613afcdeeae9c921063e986468a4a_16.png)

![](img/267613afcdeeae9c921063e986468a4a_17.png)

以下是`label`参数的可选值：
*   **`label=‘right’`**：默认值。使用时间窗口的**结束日期**作为该窗口数据的标签。例如，汇总1月1日至1月7日的数据，标签显示为1月7日。
*   **`label=‘left’`**：使用时间窗口的**开始日期**作为标签。例如，汇总1月1日至1月7日的数据，标签显示为1月1日。

![](img/267613afcdeeae9c921063e986468a4a_18.png)

选择哪种标签取决于你的分析逻辑。通常，使用`‘right’`（结束日期）更符合观察习惯，因为它表示该时间段内数据的最终汇总结果。例如，标记为1月31日的数据，实际上汇总的是整个1月份的情况。

## 时间窗口（滑动窗口）操作 ⏱️

接下来，我们学习时间窗口操作，也称为滑动窗口操作。这与重采样不同，它不改变数据频率，而是在一个固定长度的窗口内进行滑动计算。

![](img/267613afcdeeae9c921063e986468a4a_20.png)

![](img/267613afcdeeae9c921063e986468a4a_21.png)

假设我们拥有100天的数据。时间窗口操作是这样的：
1.  第一个窗口：截取第1天到第10天的数据。
2.  第二个窗口：截取第2天到第11天的数据。
3.  第三个窗口：截取第3天到第12天的数据。
以此类推，窗口每次向后滑动一天。我们可以在每个窗口内计算各种统计指标（如最小值、最大值、均值、标准差）。

### 如何实现滑动窗口

![](img/267613afcdeeae9c921063e986468a4a_23.png)

![](img/267613afcdeeae9c921063e986468a4a_24.png)

在Pandas中，可以使用`rolling()`方法来实现滑动窗口计算。

![](img/267613afcdeeae9c921063e986468a4a_25.png)

以下是实现滑动窗口操作的步骤和代码：
```python
import pandas as pd
import numpy as np

# 1. 准备数据（假设df是原始DataFrame）
# 2. 处理缺失值，这是重要的一步
data2 = df.dropna()

![](img/267613afcdeeae9c921063e986468a4a_27.png)

![](img/267613afcdeeae9c921063e986468a4a_28.png)

![](img/267613afcdeeae9c921063e986468a4a_29.png)

# 3. 定义窗口大小
window_size = 10

![](img/267613afcdeeae9c921063e986468a4a_30.png)

![](img/267613afcdeeae9c921063e986468a4a_31.png)

![](img/267613afcdeeae9c921063e986468a4a_32.png)

# 4. 对某一列数据应用滚动窗口，并计算每个窗口的最小值
min_values = data2[‘column_name’].rolling(window=window_size).min()

![](img/267613afcdeeae9c921063e986468a4a_34.png)

# 5. 可以同时计算多个统计指标
data2[‘min’] = data2[‘column_name’].rolling(window=window_size).min()
data2[‘max’] = data2[‘column_name’].rolling(window=window_size).max()
data2[‘mean’] = data2[‘column_name’].rolling(window=window_size).mean()
data2[‘std’] = data2[‘column_name’].rolling(window=window_size).std()

![](img/267613afcdeeae9c921063e986468a4a_36.png)

![](img/267613afcdeeae9c921063e986468a4a_38.png)

# 查看结果
print(data2[[‘min‘， ‘max‘， ‘mean‘， ‘std‘]].tail())
```

![](img/267613afcdeeae9c921063e986468a4a_39.png)

**关键注意事项**：在应用`rolling()`操作前，务必处理数据中的缺失值（NaN）。如果窗口内存在NaN，许多统计计算的结果也会是NaN。因此，使用`dropna()`或其他方法清理数据是窗口操作前的标准预处理步骤。

## 总结 📝

![](img/267613afcdeeae9c921063e986468a4a_43.png)

![](img/267613afcdeeae9c921063e986468a4a_44.png)

本节课中我们一起学习了时间序列分析的两个重要操作：
1.  **重采样**：通过`resample()`方法改变数据的时间频率（如日变周、月），并使用聚合函数（如`mean()`， `first()`）生成新数据。可以通过`label`参数控制聚合后数据点的标签日期。
2.  **时间窗口操作**：通过`rolling()`方法实现滑动窗口计算。它可以在一个固定长度并不断滑动的窗口内，计算序列的各种统计特征（最小值、最大值、均值、标准差等）。**切记，在执行窗口操作前，需要先处理数据中的缺失值。**
