上一节我们概述了学习重点，本节中我们来看看具体的学习优先级和方法。学习时应遵循特定的顺序。

![](img/9af421549f0fc3becc710709de6812b8_36.png)

![](img/9af421549f0fc3becc710709de6812b8_38.png)

1.  **Python与数学**：点到为止。无需系统性地从头学到尾，了解基本概念后，在实际应用中边用边查。
    *   **学习方法**：在热情最高时，不要学习最基础的内容。应结合具体任务场景来学习Python。

![](img/9af421549f0fc3becc710709de6812b8_40.png)

![](img/9af421549f0fc3becc710709de6812b8_42.png)

![](img/9af421549f0fc3becc710709de6812b8_44.png)

2.  **机器学习**：学习深度取决于目标方向。
    *   如果目标是**数据挖掘**或**数据分析**，则需要详细学习机器学习，包括经典算法原理、公式推导和代码实践。
    *   如果目标是**计算机视觉**，则机器学习作为铺垫，熟悉基础概念即可，为后续深度学习做准备。

![](img/9af421549f0fc3becc710709de6812b8_46.png)

![](img/9af421549f0fc3becc710709de6812b8_48.png)

![](img/9af421549f0fc3becc710709de6812b8_50.png)

3.  **数据挖掘**：建议有一定了解。它代表处理实际任务的完整流程，能帮助你综合运用技能处理图像、文本、结构化等各类数据。
    *   **学习方法**：结合具体案例和项目来学习Python，有目的地编写和运行代码。

以下是计算机视觉的主要项目方向与学习要点：

![](img/9af421549f0fc3becc710709de6812b8_106.png)

![](img/9af421549f0fc3becc710709de6812b8_108.png)

![](img/9af421549f0fc3becc710709de6812b8_110.png)

![](img/9af421549f0fc3becc710709de6812b8_112.png)

*   **物体检测**：需要广泛学习主流算法，而不仅仅是YOLO系列。要掌握各种检测算法（如DETR、Deformable DETR等）的源码级细节。
*   **图像分割**：与检测同等重要。需要学习分割领域经典和最新的算法。
*   **项目模板**：学习使用现成的项目模板（涵盖分类、分割、检测等任务）进行增删改查，能高效地开展自己的项目。
*   **拓展任务**：学习行为识别、姿态估计、目标追踪等经典视觉项目的源码实现。
*   **Transformer应用**：学习Transformer在各类下游视觉任务中是如何实现和应用的。
*   **图神经网络**：学习如何处理具有关联关系的数据，掌握图神经网络的应用。
*   **无人驾驶相关**：学习点云处理、深度估计、车道线检测、三维重建等无人驾驶场景中的技术。
*   **前沿技术**：熟悉对比学习、多模态项目等最新热门技术。
*   **其他专题**：包括缺陷检测、重识别、生成对抗网络、强化学习、大模型、医学影像分析等。
*   **模型部署**：学习如何将训练好的模型部署到服务器或嵌入式设备上。
*   **论文创新**：学习如何设计论文中常用的创新模块。
