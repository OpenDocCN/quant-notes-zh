## 为什么要使用NumPy？⚡

![](img/3f85d4d0af8196453c60cb453235b300_5.png)

![](img/3f85d4d0af8196453c60cb453235b300_7.png)

上一节我们介绍了NumPy是科学计算的基础包。本节中我们来看看为什么需要它。

![](img/3f85d4d0af8196453c60cb453235b300_9.png)

NumPy是一个高性能的科学计算和数据分析基础包。它是后续课程中Pandas库的基础。其主要功能是提供了一个名为`ndarray`的多维数组对象，它能高效地处理批量数据运算，无需使用循环。此外，它还提供了线性代数、傅里叶变换等功能。

安装NumPy非常简单，使用`pip install numpy`命令即可。

引用NumPy的通用方式是：
```python
import numpy as np
```
这是一种约定俗成的缩写，因此在其他代码中看到`np.`开头的函数，通常就代表使用了NumPy。

![](img/3f85d4d0af8196453c60cb453235b300_11.png)

为了直观感受NumPy的便利性，我们来看两个例子。

**示例一：货币转换**
假设已知50家公司的市值（单位：万美元），需要将所有市值转换为人民币（假设汇率为6.8）。

![](img/3f85d4d0af8196453c60cb453235b300_19.png)

![](img/3f85d4d0af8196453c60cb453235b300_21.png)

使用Python原生列表的方法可能需要循环或列表推导式：
```python
# 生成模拟数据
import random
a = [random.uniform(100, 200) for _ in range(50)] # 50家公司的市值（万美元）
# 转换为人民币
b = [value * 6.8 for value in a]
```

![](img/3f85d4d0af8196453c60cb453235b300_23.png)

而使用NumPy则非常简单：
```python
import numpy as np
a_array = np.array(a) # 将列表转换为NumPy数组
result = a_array * 6.8 # 直接进行批量运算
```
NumPy允许我们直接对数组进行标量乘法，代码简洁且执行效率更高。

![](img/3f85d4d0af8196453c60cb453235b300_25.png)

![](img/3f85d4d0af8196453c60cb453235b300_27.png)

**示例二：计算购物车总金额**
已知购物车中每件商品的价格和对应件数，求总金额。

![](img/3f85d4d0af8196453c60cb453235b300_29.png)

![](img/3f85d4d0af8196453c60cb453235b300_31.png)

使用NumPy的方法：
```python
# 模拟价格和件数列表
prices = np.array([random.uniform(10, 20) for _ in range(50)])
quantities = np.array([random.randint(1, 10) for _ in range(50)])
# 计算总金额
total_cost = (prices * quantities).sum()
```
`prices * quantities`会将两个数组的对应元素相乘，生成一个新数组，然后`.sum()`方法可以快速计算总和。

![](img/3f85d4d0af8196453c60cb453235b300_33.png)

通过以上例子，我们可以直观感受到NumPy在批量数据运算上的简洁与高效。接下来，我们将深入探讨其核心对象——`ndarray`。

`ndarray`是NumPy提供的一个N维数组对象。创建数组最基本的方法是使用`np.array()`函数，参数可以是一个列表或其他可迭代对象。

![](img/3f85d4d0af8196453c60cb453235b300_45.png)

![](img/3f85d4d0af8196453c60cb453235b300_47.png)

```python
import numpy as np
arr1 = np.array([1, 2, 3, 4, 5]) # 从列表创建一维数组
arr2 = np.array(range(10)) # 从range对象创建
```

![](img/3f85d4d0af8196453c60cb453235b300_49.png)

**数组与Python列表的主要区别：**
以下是数组与列表的核心区别：
1.  **元素类型必须相同**：Python列表可以存储不同类型的元素（如字符串、整数混合）。而NumPy数组要求所有元素必须是相同的数据类型。
2.  **大小固定**：Python列表可以动态地使用`append()`方法增加元素。NumPy数组在创建后大小是固定的，不能直接添加元素（但可以创建新数组或调整形状）。这种设计与其底层连续的内存存储结构有关，也是其高效的原因之一。

## 数组的常用属性 📊

首先创建一个数组作为示例：
```python
arr = np.array([1, 2, 3, 4, 5])
```

以下是`ndarray`的一些常用属性：

![](img/3f85d4d0af8196453c60cb453235b300_59.png)

**1. dtype - 数据类型**
`dtype`属性返回数组中元素的数据类型。
```python
print(arr.dtype) # 输出：int64
```
NumPy有丰富的数据类型，主要分为以下几类：
*   `bool_`：布尔型
*   `int8/int16/int32/int64`：有符号整数
*   `uint8/uint16/uint32/uint64`：无符号整数
*   `float16/float32/float64`：浮点数
*   `complex64/complex128`：复数
类型名称后的数字（如64）代表该类型在内存中占用的位数。例如，`int64`表示64位（8字节）的整数。在金融量化中，我们主要处理数字类型。

**2. size - 元素总数**
`size`属性返回数组中元素的总个数。
```python
print(arr.size) # 输出：5
```

**3. shape - 数组形状**
`shape`属性返回一个元组，表示数组在每个维度上的大小（长度）。这对于多维数组尤其有用。
```python
# 创建一个二维数组（矩阵）
arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
print(arr_2d.shape) # 输出：(2, 3)，表示2行3列
print(arr_2d.size) # 输出：6，总元素个数
```
*   一维数组的`shape`形如`(n,)`。
*   二维数组的`shape`形如`(行数, 列数)`。
*   三维数组的`shape`形如`(页数, 行数, 列数)`，可以想象成一个有多页的表格。

**4. ndim - 数组维度**
`ndim`属性返回数组的维数。
```python
print(arr.ndim) # 输出：1 (一维)
print(arr_2d.ndim) # 输出：2 (二维)
```

**5. T - 数组的转置**
对于二维数组，`.T`属性可以返回其转置矩阵（行变列，列变行）。
```python
matrix = np.array([[1, 2, 3], [4, 5, 6]])
print(matrix)
# 输出：
# [[1 2 3]
#  [4 5 6]]
print(matrix.T)
# 输出（转置后）：
# [[1 4]
#  [2 5]
#  [3 6]]
```
转置操作在金融数据分析和线性代数中非常常用。
