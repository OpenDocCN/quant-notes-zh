## 交易函数类别与参数详解

上一节我们介绍了定时函数，本节中我们来看看如何执行具体的交易操作。交易函数是实现买卖指令的核心工具。

以下是按照不同交易场景分类的主要交易函数：

### 1. 按交易数量下单
`order` 函数用于按指定数量买卖股票。
*   **`security`**: 股票代码。
*   **`amount`**: 交易数量。在多数量化平台中，**正数表示买入，负数表示卖出**。
*   **`style`**: 下单类型（如市价单、限价单）。
*   **`side`**: 交易方向（`long`为做多，`short`为空）。A股市场主要使用`long`。
*   **`pindex`**: 仓位号，默认为0。

**示例**：按市价买入100股平安银行。
```python
order('000001.XSHE', 100, style=LimitOrderStyle(30))
```

### 2. 按股票价值下单
`order_value` 函数用于按指定的总金额买卖股票。
*   **`value`**: 交易的总价值。**正数表示买入，负数表示卖出**。

**示例**：卖出价值5000元的平安银行股票。
```python
order_value('000001.XSHE', -5000)
```

### 3. 按目标数量调整持仓
`order_target` 函数用于将股票持仓调整至目标数量。
*   **`amount`**: 目标持仓数量。函数会自动计算需要买入或卖出的数量以达到该目标。

**示例**：将平安银行的持仓调整至正好100股。
```python
order_target('000001.XSHE', 100)
```

## 其他常用交易函数

除了核心的下单函数，还有一些辅助函数用于管理订单和账户。

![](img/40aa783355a433d912d037648a3af3fa_1.png)

以下是几个常用的辅助交易函数：

*   **查询订单**：`get_orders`。可按订单号、证券代码或状态查询历史订单。
*   **查询未完成订单**：`get_open_orders`。获取当天所有未成交的订单。
*   **撤单**：`cancel_order`。撤销指定的未成交订单。
*   **账户出入金**：`inout_cash`。向策略账户存入或取出资金。**正数表示入金，负数表示出金**。

**示例**：向账户增加1万元资金。
```python
inout_cash(10000)
```

## 代码实战演示

接下来，我们通过一个简单的策略示例来综合运用上述函数。

### 实战一：基础买卖策略
这个策略演示了如何根据持仓情况，使用`order_value`和`order_target`进行全仓买入或卖出。

```python
# 获取账户可用现金
cash = context.portfolio.available_cash

# 检查是否持有平安银行股票
if ‘000001.XSHE’ not in context.portfolio.positions:
    # 若未持仓，则用全部现金买入
    order_value(‘000001.XSHE’, cash)
else:
    # 若已持仓，则卖出500股
    order(‘000001.XSHE’, -500)
```

### 实战二：盘后撤单与账户管理
这个函数演示了如何在每天交易结束后，撤销所有未成交订单，并记录账户信息。

```python
def after_market_close(context):
    # 获取所有未完成订单
    open_orders = get_open_orders()
    for order_id in open_orders:
        # 遍历并撤销每一个未完成订单
        cancel_order(order_id)
        log.info(f‘已撤销订单：{order_id}’)

    # 账户入金1万元
    inout_cash(10000)

    # 查询并打印当前可用资金
    current_cash = context.portfolio.available_cash
    log.info(f‘当前账户可用资金为：{current_cash}’)
```
