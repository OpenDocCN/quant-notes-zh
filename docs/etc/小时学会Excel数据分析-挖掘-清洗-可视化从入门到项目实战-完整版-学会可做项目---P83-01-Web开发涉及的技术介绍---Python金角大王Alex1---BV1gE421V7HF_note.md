## 什么是Web开发？ 🌐

![](img/cdec9ba63796739a89f11504c953dc25_5.png)

![](img/cdec9ba63796739a89f11504c953dc25_7.png)

Web开发，简单来说就是网站开发，即编写网页。你在浏览器中看到的所有精美页面，其背后都是由代码实现的。浏览器的作用类似于Python解释器：Python解释器负责解释并执行我们编写的Python代码；而浏览器则负责解释并渲染网页代码，将其转化为可视化的页面。

## 网页开发三剑客 ⚔️

![](img/cdec9ba63796739a89f11504c953dc25_13.png)

网页开发依赖于三个核心组件，它们通常被称为“网页开发三剑客”。它们分别是HTML、CSS和JavaScript。这三者共同协作，构建出功能完整、样式美观且具有交互性的网页。

以下是三剑客的详细介绍：

*   **HTML (HyperText Markup Language)**
    HTML是超文本标记语言，它负责构建网页的**骨架和结构**。你可以将其理解为建造毛坯房的过程，它定义了哪里是标题、哪里是段落、哪里是图片、哪里是链接。没有HTML，网页就没有基本的结构。其核心作用是定义内容的结构。

*   **CSS (Cascading Style Sheets)**
    CSS是层叠样式表，它负责为HTML骨架添加**样式和外观**。这相当于对毛坯房进行装修。CSS决定了文字的颜色、字体、大小，元素的背景、布局、间距等。没有CSS，网页将只是一个非常简陋、只有结构的“毛坯房”。

*   **JavaScript**
    JavaScript是一种脚本语言，它负责让网页**动起来**，实现交互功能。这相当于让房子里的灯可以开关、窗帘可以自动拉合。例如，图片轮播、表单验证、按钮点击效果、动态加载内容等，都是由JavaScript实现的。它让网页从静态文档变为动态应用。

为了更直观地理解，我们可以用一个比喻：HTML是人的**骨骼**，CSS是人的**皮肤和衣服**，而JavaScript则是让**人能够说话、走路和做动作的神经与肌肉**。

## 前后端开发 🔄

![](img/cdec9ba63796739a89f11504c953dc25_25.png)

![](img/cdec9ba63796739a89f11504c953dc25_27.png)

![](img/cdec9ba63796739a89f11504c953dc25_29.png)

当你访问一个网站（如知乎）时，你在浏览器中看到的页面和数据，并非存储在你的电脑上。其工作流程大致如下：

1.  你在浏览器输入网址（发送请求）。
2.  请求到达知乎的服务器。
3.  服务器处理请求，从数据库中查询所需的数据和页面逻辑（后端负责）。
4.  服务器将生成好的HTML、CSS、JavaScript代码（即前端代码）和数据，打包返回给你的浏览器。
5.  你的浏览器接收这些代码，并进行解释和渲染，最终呈现出你看到的完整页面。

在这个过程中，涉及两种不同的开发角色：

*   **后端开发**：负责服务器、数据库和应用程序的业务逻辑。比如处理用户登录、存储文章数据、在接收到请求时决定返回什么数据。后端开发常用Python（Django/Flask）、Java、Go等语言。
*   **前端开发**：负责将后端提供的数据，通过HTML、CSS、JavaScript技术，构建成用户直接交互的界面。关注的是页面的布局、样式和交互体验。

现代开发中常采用“前后端分离”模式：后端专注于提供数据接口（API），前端专注于调用接口获取数据并展示。两者通过明确的协议进行通信，使得开发更加高效和清晰。
