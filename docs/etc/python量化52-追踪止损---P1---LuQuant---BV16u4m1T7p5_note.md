## 📊 数据准备与指标计算

首先，我们需要准备数据和计算必要的技术指标。我们将导入必要的库，加载历史价格数据，并计算平均真实波幅（ATR）指标。

```python
import pandas as pd
import ta

# 加载数据
df = pd.read_csv('eurusd_daily.csv')
# 数据清洗
df = df.dropna().reset_index(drop=True)

# 计算ATR指标
df['ATR'] = ta.volatility.average_true_range(df['High'], df['Low'], df['Close'], window=14)
```

## 🛡️ 方法一：固定距离止损

第一种方法是为所有交易设置一个固定的止损和止盈距离。这意味着无论市场波动如何，每次交易的止损点数都相同。

以下是回测该策略的关键参数设置：
*   **止损距离**：`stop_loss_distance = 550` (点子)
*   **止盈距离**：`take_profit_distance = 600` (点子)

使用此方法回测，我们获得了约121%的总回报率和70%的胜率。

## 📊 方法二：基于ATR的动态止损

上一节我们使用了固定的止损距离。本节中我们来看看一种更动态的方法：根据市场的波动性（通过ATR衡量）来设置止损。

这种方法的核心公式是：
**止损距离 = ATR值 / 比率因子**

以下是关键参数：
*   **ATR因子**：`atr_factor = 0.2`
*   **盈亏比**：`ratio_f = 1` (止盈距离等于止损距离)

使用此方法回测，我们获得了约125%的总回报率和71%的胜率。这种方法能更好地适应不同市场环境下的波动。

## 🚀 方法三：固定距离的追踪止损

接下来，我们探讨追踪止损。这种方法不设置固定的止盈目标，而是让利润奔跑，同时用一个移动的止损位来保护利润。

其逻辑是：当价格向有利方向移动时，止损位随之移动，始终保持一个固定距离。只有当价格回撤触及该移动止损位时，交易才会平仓。

关键参数：
*   **追踪距离**：`trail_distance = 500` (点子)

使用此方法回测，我们获得了约131%的总回报率，但胜率降至55%。这显示了追踪止损的力量：即使胜率不高，也能通过抓住大趋势交易来获得良好回报。

## 🔄 方法四：基于ATR的追踪止损

最后一种方法结合了动态性和追踪止损的优点。它使用开仓时的ATR值来确定初始的追踪止损距离。

其核心逻辑与固定距离追踪止损类似，但初始距离由公式决定：
**初始追踪距离 = ATR值 / ATR因子**

关键参数：
*   **ATR因子**：`atr_factor = 1`

使用此方法回测，我们获得了约69%的总回报率和48%的胜率。虽然在此策略中表现不如前几种，但它可能是某些特定策略或市场环境下更稳健的选择。
