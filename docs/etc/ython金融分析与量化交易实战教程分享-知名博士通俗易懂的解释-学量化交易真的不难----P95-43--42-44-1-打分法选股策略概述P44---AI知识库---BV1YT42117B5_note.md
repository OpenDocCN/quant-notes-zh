# 量化交易完全可自学教程：P95：因子打分法选股策略概述 🧮

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分法。我们将了解其核心思想、实施前提以及如何通过综合多个因子来筛选股票。

## 策略核心思想

上一节我们介绍了如何通过单一因子（如市盈率）来评估股票。本节中我们来看看如何综合多个因子进行更全面的评估。

因子打分选股策略的核心思想是：为每只股票的多个因子分别打分，然后将这些分数汇总成一个总分，最后根据总分排名来选出最优的股票组合。这类似于学生通过各科成绩的总分来进行排名。

## 策略实施前提

在开始为因子打分之前，我们必须明确一个关键的先验知识：每个因子与预期收益率的相关性是正向还是负向的。

*   **正向因子**：因子数值越大，通常预期收益越好。例如：`每股收益(EPS)`、`净资产收益率(ROE)`。
*   **负向因子**：因子数值越小，通常预期收益越好。例如：`市盈率(PE)`、`市净率(PB)`。

这个判断可以来源于：
1.  金融领域的经验知识与研究报告。
2.  我们之前课程中学习的因子分析，通过计算**IC值（信息系数）**等指标来验证因子与收益的相关性。

在后续的实际策略编写中，我们会预先选择一组已知特性的因子。

## 策略流程示例

假设我们有300只股票（Stock1, Stock2, … Stock300）和四个选股因子（A, B, C, D）。

以下是实施打分法的简要步骤：

1.  **数据准备**：获取所有股票在调仓日的因子数据。
2.  **因子打分**：根据每个因子与收益的关系（正/负向），对每只股票的每个因子进行评分。例如，对于正向因子，数值越大得分越高。
3.  **计算总分**：将每只股票在所有因子上的得分相加，得到该股票的综合总分。
    *   公式表示：`股票总分 = 因子A得分 + 因子B得分 + 因子C得分 + 因子D得分`
4.  **排名与筛选**：对所有股票的总分进行降序排名，选取排名最高的N只（例如前10或20只）股票作为投资组合。
5.  **定期调仓**：在下一个调仓周期（例如每月），重复以上步骤，更新投资组合。

通过这个过程，我们不再依赖单一因子，而是通过多因子的综合表现来做出更稳健的选股决策。
