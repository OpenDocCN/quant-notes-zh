## 数据准备与初步观察

![](img/c6b4de69408c2c6d4c9d734ffb03e196_4.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_6.png)

上一节我们介绍了如何构建特征。本节中，我们来看看如何对这些特征组合下的结果进行统计分析。

![](img/c6b4de69408c2c6d4c9d734ffb03e196_8.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_10.png)

首先，我们需要对数据进行一点变换。当前数据中，前两列是特征（例如“前一天涨跌”和“前两天涨跌”），最后一列是结果（例如“当天涨跌”）。我们需要将特征组合起来进行分组观察。

![](img/c6b4de69408c2c6d4c9d734ffb03e196_12.png)

以下是数据变换的代码示例：
```python
# 假设 df 是包含特征和结果的DataFrame
# 使用 groupby 对前两列特征进行分组，并统计每组中结果的数量
grouped_result = df.groupby([‘feature1‘, ‘feature2‘])[‘result‘].size()
```

![](img/c6b4de69408c2c6d4c9d734ffb03e196_14.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_16.png)

执行上述代码后，我们得到一个统计表，它清晰地展示了不同特征组合下，对应结果（涨或跌）出现的次数。对于没有数据的组合，其统计值会显示为0。

## 策略制定：基于统计概率

![](img/c6b4de69408c2c6d4c9d734ffb03e196_22.png)

以下是基于统计结果制定策略的步骤。

首先，我们需要观察统计结果，找出规律。例如，我们可能发现当“前一天涨”和“前两天涨”同时发生时，第三天“跌”的概率较大。

![](img/c6b4de69408c2c6d4c9d734ffb03e196_24.png)

接着，我们根据这个观察来制定具体的交易规则。我们可以使用 `numpy.where` 函数来将统计规律转化为具体的交易信号（例如，用 1 表示“买入”，用 -1 表示“卖出”）。

以下是生成交易信号的代码：
```python
import numpy as np

![](img/c6b4de69408c2c6d4c9d734ffb03e196_26.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_28.png)

# 假设 data 是包含特征列的DataFrame
# 计算一个条件：当两个特征值之和等于2时（即前两天都上涨）
condition = (data[‘feature1‘] + data[‘feature2‘]) == 2

![](img/c6b4de69408c2c6d4c9d734ffb03e196_30.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_32.png)

# 根据条件生成仓位信号：条件成立时看跌(-1)，否则看涨(1)
position = np.where(condition, -1, 1)
```
这个 `position` 数组就是我们基于简单统计得出的交易策略信号。

## 策略效果评估

![](img/c6b4de69408c2c6d4c9d734ffb03e196_36.png)

制定策略后，我们需要评估其效果。我们将策略预测的走势与实际走势进行比较。

![](img/c6b4de69408c2c6d4c9d734ffb03e196_38.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_40.png)

我们可以计算策略预测方向与实际市场方向一致的天数比例，这可以作为策略准确率的一个简单衡量。

![](img/c6b4de69408c2c6d4c9d734ffb03e196_42.png)

以下是计算预测准确率的代码：
```python
# 假设 data[‘direction‘] 是实际的市场走势
# position 是我们预测的走势
comparison = (data[‘direction‘] == position)
correct_predictions = comparison.value_counts()[True]
wrong_predictions = comparison.value_counts()[False]
```
通过比较，我们可能发现基于统计的策略比随机猜测或简单跟随大盘的准确率略高。

![](img/c6b4de69408c2c6d4c9d734ffb03e196_44.png)

最后，我们可以计算按照此策略进行交易的理论收益，并与大盘基准收益进行对比，以评估策略是否能减少亏损或创造超额收益。

## 总结

![](img/c6b4de69408c2c6d4c9d734ffb03e196_50.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_52.png)

本节课中我们一起学习了量化策略中的统计分析方法。

![](img/c6b4de69408c2c6d4c9d734ffb03e196_54.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_56.png)

我们首先使用 `groupby` 对特征组合进行分组统计，观察不同情况下市场结果的分布概率。然后，我们基于出现概率最大的结果来制定简单的交易规则，并使用 `np.where` 将其转化为可执行的交易信号。最后，我们通过比较预测准确率和计算理论收益，对策略效果进行了初步评估。

![](img/c6b4de69408c2c6d4c9d734ffb03e196_58.png)

![](img/c6b4de69408c2c6d4c9d734ffb03e196_60.png)

这种方法的核心思想是 **“比谁的可能性大”**，它为我们提供了一种基于历史数据统计的、直观的策略构建思路。虽然这种简单统计方法的稳健性有限，但它清晰地展示了从数据分析到策略生成的完整流程。
