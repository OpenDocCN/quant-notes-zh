## 概述

上一节我们介绍了如何计算时间序列的差异值和简单增长率。本节中我们来看看如何分析连续时间内的累积变化。例如，我们想知道一只股票在2010年的1元钱，到2011年、2012年……分别相当于多少钱。这需要一种特殊的计算方法。

## 为什么增长率不能直接累加？

首先，我们需要理解一个关键概念：连续时间点的增长率不能直接相加来得到整体的总增长率。

![](img/57f9670e153c6986db5e5db34ff24d27_4.png)

举个例子来说明：

假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

如果尝试将每日增长率累加：
*   第二天增长率 = (50 - 100) / 100 = -0.5
*   第三天增长率 = (75 - 50) / 50 = 0.5
*   累加“总增长率” = -0.5 + 0.5 = 0

![](img/57f9670e153c6986db5e5db34ff24d27_6.png)

![](img/57f9670e153c6986db5e5db34ff24d27_7.png)

累加结果为0，似乎表示股票既没涨也没跌。但观察实际价格，从第一天的100元到第三天的75元，实际上是下跌了。这说明直接累加增长率会得到错误结论。

## 解决方案：引入对数计算连续累积收益

![](img/57f9670e153c6986db5e5db34ff24d27_9.png)

为了正确计算连续时间内的累积变化，我们需要借助数学中的对数。

核心思想是：将对数收益率进行累加，等价于计算首尾价格的比例。

![](img/57f9670e153c6986db5e5db34ff24d27_11.png)

**公式推导：**
计算每日的对数收益率：`log(当日价格 / 前日价格)`
例如：
`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/57f9670e153c6986db5e5db34ff24d27_13.png)

![](img/57f9670e153c6986db5e5db34ff24d27_15.png)

最终，对数收益率的累加和 `log(最终价格 / 初始价格)`，恰好反映了从第一天到最后一天的整体变化比例。

![](img/57f9670e153c6986db5e5db34ff24d27_17.png)

![](img/57f9670e153c6986db5e5db34ff24d27_18.png)

**形象理解：** 这相当于计算“初始的1元钱，在经历一系列涨跌后，最终变成了多少钱”。

![](img/57f9670e153c6986db5e5db34ff24d27_19.png)

## 实战操作：计算与可视化

![](img/57f9670e153c6986db5e5db34ff24d27_23.png)

以下是使用Python进行计算的步骤：

![](img/57f9670e153c6986db5e5db34ff24d27_24.png)

![](img/57f9670e153c6986db5e5db34ff24d27_25.png)

### 1. 计算对数收益率

![](img/57f9670e153c6986db5e5db34ff24d27_27.png)

首先，我们需要获取“前一天”的数据。Pandas的`.shift()`函数可以将数据向上或向下移动。

![](img/57f9670e153c6986db5e5db34ff24d27_28.png)

![](img/57f9670e153c6986db5e5db34ff24d27_29.png)

```python
# 计算对数收益率
# data 是包含股价的Pandas Series或DataFrame
returns = np.log(data / data.shift(1))
print(returns.head())
```

### 2. 计算累积对数收益率

![](img/57f9670e153c6986db5e5db34ff24d27_31.png)

![](img/57f9670e153c6986db5e5db34ff24d27_33.png)

接下来，我们对计算出的对数收益率序列进行累加。这可以使用NumPy的`.cumsum()`函数（累计求和）来完成。

![](img/57f9670e153c6986db5e5db34ff24d27_35.png)

```python
# 计算累积对数收益率
cumulative_log_returns = returns.cumsum()
```

![](img/57f9670e153c6986db5e5db34ff24d27_37.png)

![](img/57f9670e153c6986db5e5db34ff24d27_38.png)

### 3. 还原为实际价值比例

由于我们之前取了对数，现在需要将其还原为实际的价值倍数关系。使用`np.exp()`函数（指数函数）进行还原。

![](img/57f9670e153c6986db5e5db34ff24d27_40.png)

![](img/57f9670e153c6986db5e5db34ff24d27_42.png)

```python
# 将累积对数收益率还原为实际的价值倍数
# 这个结果表示：初始的1单位价值，到每个时间点相当于多少单位。
cumulative_value_relative = np.exp(cumulative_log_returns)
```

### 4. 可视化结果

![](img/57f9670e153c6986db5e5db34ff24d27_44.png)

![](img/57f9670e153c6986db5e5db34ff24d27_45.png)

最后，我们可以将结果绘制成图，直观地看到各公司股票初始价值随时间增长的情况。

![](img/57f9670e153c6986db5e5db34ff24d27_46.png)

```python
# 绘制图表
cumulative_value_relative.plot(figsize=(10, 6))
plt.title('初始1单位价值的累积增长情况')
plt.ylabel('价值倍数')
plt.xlabel('日期')
plt.show()
```

![](img/57f9670e153c6986db5e5db34ff24d27_48.png)

![](img/57f9670e153c6986db5e5db34ff24d27_49.png)

执行以上代码后，得到的图表会显示：以起始点为基准（值为1），每条线代表一家公司，其纵坐标值表示“最初的1元钱在该日期相当于多少钱”。例如，亚马逊(AMZN)的线最高，说明其价值增长倍数最大。

4.  **结果还原**：利用**指数函数** `np.exp()` 将累积对数收益率还原为易于理解的**价值倍数**。
5.  **实战工具**：我们使用了 `shift()` 获取滞后数据，`np.log` 和 `np.exp` 进行对数与指数运算，`cumsum()` 进行累加，最终通过绘图进行可视化分析。
