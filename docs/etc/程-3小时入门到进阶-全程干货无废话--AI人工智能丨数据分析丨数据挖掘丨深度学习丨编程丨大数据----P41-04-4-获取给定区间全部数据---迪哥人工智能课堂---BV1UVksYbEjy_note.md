# Python金融分析与量化交易实战教程：P41：04-4-获取给定区间全部数据 📊

在本节课中，我们将学习如何通过循环遍历，获取指定时间区间内每一天的股票因子数据，并将这些数据整合到一个结构化的DataFrame中。这是构建量化分析数据集的关键步骤。

![](img/2d11f917178d429d1d42260c35ac7e3d_1.png)

![](img/2d11f917178d429d1d42260c35ac7e3d_3.png)

上一节我们介绍了如何获取单日的因子数据，本节中我们来看看如何获取一个连续时间区间内的全部数据。

![](img/2d11f917178d429d1d42260c35ac7e3d_5.png)

核心思路是遍历区间内的每一个交易日，分别获取当天的数据，然后将所有日期的数据拼接成一个总表。以下是实现这一过程的具体步骤。

![](img/2d11f917178d429d1d42260c35ac7e3d_7.png)

## 遍历日期序列

![](img/2d11f917178d429d1d42260c35ac7e3d_9.png)

首先，我们需要遍历给定的日期序列。假设你有一个日期序列，要查询其中每一天的数据，可以通过`for`循环来实现。

```python
for i in range(len(date_sequence)):
    # 处理每一天的数据
```

在循环中，变量`i`代表当前是序列中的第几天。

## 获取每日数据

![](img/2d11f917178d429d1d42260c35ac7e3d_11.png)

在循环体内，对于每一天`i`，我们需要获取其对应的因子数据值。这需要调用数据获取函数，并传入索引`i`。

```python
daily_value = get_factor_data(i)
```

![](img/2d11f917178d429d1d42260c35ac7e3d_13.png)

该函数返回的是一个三维数据。根据之前分析，我们只需要第三个维度的全部数据，即实际的指标值。

```python
useful_value = daily_value[0, 2, :]  # 假设有效数据在第三个维度
```

![](img/2d11f917178d429d1d42260c35ac7e3d_15.png)

这样，我们就能取到每一天对应的因子数据值了。

![](img/2d11f917178d429d1d42260c35ac7e3d_17.png)

## 构建每日数据表

![](img/2d11f917178d429d1d42260c35ac7e3d_19.png)

获取到每日数据后，不能仅仅打印出来，我们需要将其汇总。我们希望构建的DataFrame包含三列信息：日期、股票名称、以及当天的因子指标值。

因此，在每次循环中，我们需要创建一个临时的DataFrame来存放当天的数据。

```python
# 创建一个新的DataFrame来存放当天数据
daily_df = pd.DataFrame()
# 将因子值作为一列加入，并指定列名
daily_df[‘factor_value‘] = useful_value
# 新增一列，记录当前日期
daily_df[‘date‘] = date_sequence[i]
```

![](img/2d11f917178d429d1d42260c35ac7e3d_21.png)

现在，`daily_df`这个DataFrame就包含了两列：`date`（日期）和`factor_value`（因子值）。

## 拼接成总表

![](img/2d11f917178d429d1d42260c35ac7e3d_23.png)

每次循环都会生成一个新的`daily_df`。我们需要把这些分散的每日数据表，拼接成一个总的数据表。

![](img/2d11f917178d429d1d42260c35ac7e3d_25.png)

在循环开始前，我们先初始化一个空的DataFrame作为总表。

```python
total_df = pd.DataFrame()
```

![](img/2d11f917178d429d1d42260c35ac7e3d_27.png)

在每次循环生成`daily_df`后，我们使用`pandas.concat`函数将其拼接到总表中。

![](img/2d11f917178d429d1d42260c35ac7e3d_29.png)

```python
total_df = pd.concat([total_df, daily_df], ignore_index=True)
```

**注意**：必须将拼接后的结果重新赋值给`total_df`，否则总表的内容不会更新。

![](img/2d11f917178d429d1d42260c35ac7e3d_31.png)

## 总结

![](img/2d11f917178d429d1d42260c35ac7e3d_41.png)

本节课中我们一起学习了如何获取给定时间区间内的全部因子数据。关键步骤包括：遍历日期序列、在循环中获取每日数据、将每日数据构建成临时DataFrame、以及最终将所有临时DataFrame拼接成一个总表。这个过程虽然需要手动编写循环，但它是灵活处理时间序列数据的基础。
