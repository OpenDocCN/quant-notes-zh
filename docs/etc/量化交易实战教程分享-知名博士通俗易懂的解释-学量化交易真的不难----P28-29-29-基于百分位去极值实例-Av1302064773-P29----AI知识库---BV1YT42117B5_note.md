## 概述
在本节课中，我们将学习一种在量化交易中常用的数据处理技术：**基于百分位去极值**。我们将通过编写一个Python函数，将数据中超出指定百分位范围（例如，小于25%分位数或大于75%分位数）的极端值，替换为对应的边界值，从而将数据规范到一个合理的范围内。

![](img/a6de4426d077a531cbea3eadddc0f804_3.png)

## 核心思路与函数定义
上一节我们介绍了去极值的概念，本节中我们来看看如何通过百分位法具体实现。其核心思路是：计算数据的**第一四分位数（Q1）** 和**第三四分位数（Q3）**，然后将所有小于Q1的值设置为Q1，将所有大于Q3的值设置为Q3。

![](img/a6de4426d077a531cbea3eadddc0f804_5.png)

![](img/a6de4426d077a531cbea3eadddc0f804_7.png)

以下是实现此功能的核心函数定义：

![](img/a6de4426d077a531cbea3eadddc0f804_9.png)

```python
import pandas as pd
import numpy as np

![](img/a6de4426d077a531cbea3eadddc0f804_11.png)

![](img/a6de4426d077a531cbea3eadddc0f804_13.png)

def winsorize_by_percentile(series, low_percentile=0.25, high_percentile=0.75):
    """
    基于百分位对序列进行去极值处理。
    参数:
        series: 输入的Pandas Series数据。
        low_percentile: 下界百分位，默认0.25（Q1）。
        high_percentile: 上界百分位，默认0.75（Q3）。
    返回:
        处理后的Pandas Series。
    """
    # 1. 对数据进行排序
    sorted_series = series.sort_values()
    
    # 2. 计算指定百分位的值
    q = sorted_series.quantile([low_percentile, high_percentile])
    low_bound = q.iloc[0]  # 下边界值（如Q1）
    high_bound = q.iloc[1] # 上边界值（如Q3）
    
    # 3. 使用np.clip将数据规范到[low_bound, high_bound]区间
    # 小于low_bound的值变为low_bound，大于high_bound的值变为high_bound
    result = np.clip(series, low_bound, high_bound)
    
    return result
```

## 函数步骤详解
接下来，我们详细拆解函数中的每一步操作。

![](img/a6de4426d077a531cbea3eadddc0f804_19.png)

### 步骤一：数据排序
为了准确找到指定百分位对应的数值，我们首先需要将数据按从小到大的顺序排列。这通过Pandas的 `sort_values()` 方法实现。

![](img/a6de4426d077a531cbea3eadddc0f804_21.png)

```python
sorted_series = series.sort_values()
```

### 步骤二：计算百分位边界值
排序后，我们使用Pandas Series的 `quantile()` 方法计算分位数。该方法接收一个列表，返回列表中每个百分位对应的数值。

![](img/a6de4426d077a531cbea3eadddc0f804_23.png)

```python
q = sorted_series.quantile([low_percentile, high_percentile])
low_bound = q.iloc[0]  # 对应low_percentile的值
high_bound = q.iloc[1] # 对应high_percentile的值
```

![](img/a6de4426d077a531cbea3eadddc0f804_25.png)

### 步骤三：应用边界限制
最后，我们使用NumPy的 `clip()` 函数将原始数据限制在计算出的边界内。这个函数非常高效，它会直接替换所有超出范围的值。

![](img/a6de4426d077a531cbea3eadddc0f804_27.png)

![](img/a6de4426d077a531cbea3eadddc0f804_29.png)

```python
result = np.clip(series, low_bound, high_bound)
```

## 参数调整与实际应用
在实际的量化因子分析中，我们通常不会使用默认的25%和75%这样宽泛的边界，因为这会将过多数据视为异常值。

![](img/a6de4426d077a531cbea3eadddc0f804_53.png)

![](img/a6de4426d077a531cbea3eadddc0f804_55.png)

以下是更常用的参数设置：
*   **更严格的边界**：例如使用 `low_percentile=0.025` 和 `high_percentile=0.975`，即只处理分布两端各2.5%的极端值。
*   **非对称边界**：根据数据特性，有时可能需要设置不对称的边界，例如 `low_percentile=0.01` 和 `high_percentile=0.99`。

![](img/a6de4426d077a531cbea3eadddc0f804_57.png)

![](img/a6de4426d077a531cbea3eadddc0f804_59.png)

你可以通过修改函数调用时的参数来灵活控制去极值的严格程度。
