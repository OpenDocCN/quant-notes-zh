# 量化交易基础：P20：阿尔法与贝塔概述 📊

在本节课中，我们将要学习量化交易中两个核心概念：阿尔法（α）和贝塔（β）。理解这两个指标对于评估投资策略的收益来源至关重要。

## 收益的构成

上一节我们介绍了策略评估的多种指标，本节中我们来看看如何区分收益的来源。投资所获得的收益可以分解为两个部分。

第一部分收益与整体市场环境相关。当市场大环境向好时，大部分投资都能获得收益。这部分收益可以理解为“随大流”赚到的钱。

第二部分收益则与市场波动关系不大，主要来源于投资者自身的努力。例如，通过深入分析公司财务、运营状况，或凭借独特的策略和敏锐的观察力所获得的收益。

阿尔法和贝塔就是分别用来衡量这两部分收益的指标。

## 阿尔法（α）与贝塔（β）的定义

以下是阿尔法和贝塔的核心定义：

*   **阿尔法（α）**：衡量**超额收益**。它描述的是投资策略带来的、与市场波动无关的收益部分。阿尔法值越高，通常意味着策略本身越有效，能带来超越市场平均水平的回报。
*   **贝塔（β）**：衡量**系统性风险**或市场相关性。它反映了投资策略对大盘波动的敏感程度。例如，贝塔值为1.2，意味着当市场上涨10%时，该策略可能上涨12%；市场下跌10%时，可能下跌12%。

简单来说：
*   **市场收益** 与 **贝塔（β）** 挂钩。
*   **超额收益** 与 **阿尔法（α）** 挂钩。

在策略回测结果中，我们常看到几条曲线：
*   **策略收益**：代表你的策略实际获得的总收益。
*   **基准收益**：代表市场大环境（如沪深300指数）的收益走势。这相当于“什么都不做，单纯跟随市场”所能获得的收益。
*   **超额收益**：即 **策略收益** 减去 **基准收益** 的部分。这部分收益完全由你的策略贡献，是阿尔法（α）的直观体现。

因此，总收益可以看作由两部分组成：**市场收益（贝塔收益）** 和 **超额收益（阿尔法收益）**。

## 阿尔法与贝塔的计算原理

阿尔法和贝塔的数值可以通过统计模型计算得出。一种常见的方法是使用线性回归。

我们可以将策略收益与基准收益的关系表示为以下回归方程：
`策略收益 = α + β * 基准收益 + ε`
其中：
*   `α` 就是阿尔法，代表回归线的截距，即超额收益。
*   `β` 就是贝塔，代表回归线的斜率，即策略对市场的敏感度。
*   `ε` 是误差项。

通过历史数据拟合这个方程，就可以求解出具体的阿尔法（α）和贝塔（β）值。在后续的因子策略分析中，我们会更详细地探讨具体的计算方法。

## 量化交易的核心目标

理解阿尔法和贝塔后，我们需要明确量化交易的核心关注点。

市场整体的涨跌（贝塔部分）是个人投资者难以控制和预测的。因此，我们的核心目标应该是追求**稳定的超额收益（阿尔法）**。量化策略的研究和优化，本质上就是在寻找能够持续产生正阿尔法的方法。

所以，在策略评估中，我们通常更关注**阿尔法（α）** 的大小和稳定性。

## 本章小结

本节课中我们一起学习了量化交易中两个基础且重要的概念：

1.  **收益分解**：投资总收益可分解为**市场收益（贝塔收益）** 和**超额收益（阿尔法收益）**。
2.  **阿尔法（α）**：衡量策略带来的、独立于市场的超额收益，是策略有效性的核心体现。
3.  **贝塔（β）**：衡量策略相对于市场波动的敏感程度，即系统性风险。
4.  **计算原理**：两者可通过 `策略收益 = α + β * 基准收益 + ε` 这一回归模型进行估算。
5.  **交易目标**：量化策略的核心在于追求持续、稳定的正阿尔法，以获取超越市场平均水平的回报。

记住，对于初学者，重点在于理解阿尔法和贝塔所代表的**经济含义**，而不必死记硬背复杂的公式。在实际应用中，已有成熟的工具包可以方便地计算这些指标。
