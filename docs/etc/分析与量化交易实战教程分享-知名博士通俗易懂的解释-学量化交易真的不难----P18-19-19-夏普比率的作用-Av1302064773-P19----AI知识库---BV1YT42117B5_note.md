在本节课中，我们将要学习一个在量化投资中至关重要的风险调整后收益指标——夏普比率。我们将理解它的含义、计算方法，并通过一个简单的例子来掌握其应用。

## 概述

夏普比率用于衡量一项投资在承担单位风险时，所能获得的超额回报。它帮助投资者判断，为了获取更高的收益，所承担的额外风险是否值得。

## 夏普比率的含义

上一节我们介绍了风险的概念，本节中我们来看看如何量化风险与收益的关系。

![](img/0f2b0388e705d028a1791fa986be24e6_1.png)

![](img/0f2b0388e705d028a1791fa986be24e6_3.png)

夏普比率描述的是：对于每承担一单位的风险，我们能获得多少超额收益。这个指标越高，意味着在相同的风险水平下，投资组合能带来更高的回报，因此投资效率也越高。

![](img/0f2b0388e705d028a1791fa986be24e6_5.png)

在选股或构建投资组合时，如果仅从夏普比率这一指标出发，我们应选择比率更高的选项，因为它代表了更好的风险收益比。

## 夏普比率的计算逻辑

为了理解夏普比率的计算，我们可以对比两种投资方式。

*   一种是无风险投资，例如国债或保本理财，其收益率是固定的（例如年化5%）。
*   另一种是风险投资，例如一个股票投资组合，其预期收益率可能更高（例如年化15%），但收益是不确定的，存在波动风险。

夏普比率的核心思想是计算风险投资超越无风险收益的部分，再与投资组合自身的风险（波动性）进行比较。

以下是计算夏普比率的步骤：

1.  **计算超额收益**：用投资组合的平均收益率减去无风险收益率。
2.  **衡量风险**：使用投资组合收益率的标准差来代表其风险。
3.  **计算比率**：将超额收益除以风险（标准差）。

其核心公式可以表示为：

```
夏普比率 = (投资组合平均收益率 - 无风险收益率) / 投资组合收益率的标准差
```

## 代码示例与计算

在Python中，我们可以使用`pandas`和`numpy`库来计算夏普比率。假设我们已经有了一个股票的历史日收益率数据`returns`。

![](img/0f2b0388e705d028a1791fa986be24e6_9.png)

以下是计算夏普比率的关键代码步骤：

```python
import numpy as np
import pandas as pd

# 假设 returns 是一个包含每日收益率的 pandas Series
# 设置年化无风险利率，例如 5%
risk_free_rate = 0.05

# 计算年化超额收益率
# 先计算日均超额收益，再乘以一年交易天数（假设250天）
excess_return_daily = returns - risk_free_rate/250
annualized_excess_return = excess_return_daily.mean() * 250

# 计算年化标准差（风险）
annualized_std = returns.std() * np.sqrt(250)

![](img/0f2b0388e705d028a1791fa986be24e6_11.png)

# 计算夏普比率
sharpe_ratio = annualized_excess_return / annualized_std

![](img/0f2b0388e705d028a1791fa986be24e6_13.png)

print(f"夏普比率为: {sharpe_ratio}")
```

![](img/0f2b0388e705d028a1791fa986be24e6_15.png)

运行上述代码后，我们会得到每个投资标的的夏普比率。数值越大，代表其风险调整后的收益表现越好。通常，我们会选择夏普比率最高的标的或组合。
