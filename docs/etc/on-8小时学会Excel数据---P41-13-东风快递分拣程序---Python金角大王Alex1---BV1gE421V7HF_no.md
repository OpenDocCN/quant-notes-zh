## 概述

![](img/c3a24d63423127ba8170e037ff3f3e83_1.png)

上一节我们学习了数据处理的基本方法，本节中我们来看看如何将这些知识应用于一个具体的场景——快递信息分拣。我们的目标是：给定一个包含多条快递信息的原始列表，我们需要按照收件人地址的省份信息，对这些数据进行归类整理。

## 项目需求与数据源

首先，我们来看一下项目的具体需求和原始数据格式。

我们的数据源是一个大的列表。这个列表中的每一个元素也是一条列表，每条子列表包含两个元素：第一个是收件人姓名，第二个是收件人的详细地址。

例如，原始数据格式可能如下所示（敏感信息已做处理）：
```python
data = [
    ['张三', '北京市海淀区xx路xx号'],
    ['李四', '上海市浦东新区yy路yy号'],
    ['王五', '河北省石家庄市zz区zz路']
    # ... 更多数据
]
```
快递公司收到这样的列表后，需要将不同省份的快递分发给对应区域的快递员。因此，我们的任务就是对这份数据进行分拣。

## 任务目标

我们的分拣目标是：将上述格式的原始数据，按照地址中的“省份”（或直辖市）信息进行归类。最终生成一个结构化的字典，其中键（Key）是省份名称，值（Value）是属于该省份的所有快递信息列表。

更复杂的分级（例如分到市或区）目前不要求实现。我们当前只完成到省级别的分拣即可。

以下是期望的输出格式示例：
```python
result = {
    '北京市': [
        ['张三', '北京市海淀区xx路xx号'],
        # ... 其他北京地址的快递
    ],
    '上海市': [
        ['李四', '上海市浦东新区yy路yy号']
    ],
    '河北省': [
        ['王五', '河北省石家庄市zz区zz路']
    ]
}
```
