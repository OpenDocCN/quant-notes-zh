在本节课中，我们将学习Excel中几个核心的日期与时间函数。这些函数能帮助我们获取当前日期时间、组合日期以及计算日期之间的差值，是数据处理和分析中的基础工具。

上一节我们介绍了其他类型的函数，本节中我们来看看日期时间函数。Excel系统自带了一些实用的日期时间函数，掌握它们能极大提升工作效率。

## 1. TODAY 与 NOW 函数

这两个函数用于获取当前的日期和时间。

*   **`TODAY()`** 函数返回当前的日期。例如，输入 `=TODAY()`，单元格会显示类似 `2023-10-27` 的日期。
*   **`NOW()`** 函数返回当前的日期和时间。例如，输入 `=NOW()`，单元格会显示类似 `2023-10-27 14:30:15` 的日期时间戳。

**公式示例：**
```
=TODAY()
=NOW()
```

## 2. DATE 函数

`DATE` 函数用于将单独的年、月、日数值组合成一个标准的日期。这在将拆分后的日期数据重新合并时非常有用。

它的语法很简单，包含三个参数：年、月、日。

**公式结构：**
```
=DATE(年, 月, 日)
```
例如，`=DATE(2023, 10, 27)` 将返回日期 `2023-10-27`。

![](img/02b7d621607ff7703a70b591911a881d_6.png)

## 3. DATEDIF 函数

![](img/02b7d621607ff7703a70b591911a881d_8.png)

`DATEDIF` 函数用于计算两个日期之间的差值，可以按天、月、年等单位返回结果。这是一个隐藏函数，输入时不会有提示，但功能非常强大。

它的语法包含三个参数：开始日期、结束日期、计算单位。

**公式结构：**
```
=DATEDIF(开始日期, 结束日期, 单位)
```
单位参数使用特定字母指定：
*   `"D"`：计算两个日期相差的天数。
*   `"M"`：计算两个日期相差的整月数。
*   `"Y"`：计算两个日期相差的整年数。

![](img/02b7d621607ff7703a70b591911a881d_10.png)

**公式示例：**
假设A1是开始日期 `2022-01-01`，B1是结束日期 `2023-10-27`。
```
=DATEDIF(A1, B1, "D") // 返回相差的天数
=DATEDIF(A1, B1, "M") // 返回相差的月数
=DATEDIF(A1, B1, "Y") // 返回相差的年数
```
**请注意**：日期的本质在Excel中是数字，因此直接相减（如 `=B1-A1`）也能得到天数差，但无法直接得到精确的月数或年数差。`DATEDIF` 函数在这方面更精确。

---

本节课中我们一起学习了四个关键的日期时间函数：
1.  **`TODAY()`**：获取当前日期。
2.  **`NOW()`**：获取当前日期和时间。
3.  **`DATE(年,月,日)`**：将独立的年、月、日数值组合成日期。
4.  **`DATEDIF(开始日期,结束日期,单位)`**：精确计算两个日期之间的天数、月数或年数差值。

![](img/02b7d621607ff7703a70b591911a881d_12.png)

掌握这些函数，你就能轻松处理与分析中的大部分日期时间计算任务了。
