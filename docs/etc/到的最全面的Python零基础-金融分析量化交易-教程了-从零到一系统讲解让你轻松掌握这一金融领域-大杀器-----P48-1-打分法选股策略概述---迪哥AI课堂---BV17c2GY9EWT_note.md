# Python金融分析量化交易：P48：1-打分法选股策略概述 📊

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分选股法。我们将从概念入手，系统地讲解该策略的核心思想、实施前提以及基本流程，帮助你理解如何综合多个因子来筛选出更具潜力的股票组合。

## 策略核心思想 🎯

上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看如何将多个因子结合起来，形成一个更综合的选股策略。

因子打分选股的核心思想是：**为每只股票的各个因子表现进行评分，然后计算总分，最后根据总分排名来选出最优的股票**。这类似于学生的总成绩由各科成绩加总决定。

假设我们有300只股票（`stock_1`, `stock_2`, ..., `stock_300`）和四个因子（`factor_A`, `factor_B`, `factor_C`, `factor_D`）。我们的目标是每月从这300只股票中选出排名前10的股票进行调仓。

以下是该策略的数据表示思路：
```python
# 示例数据结构
股票数据 = {
    ‘stock_1‘: {‘factor_A‘: 值, ‘factor_B‘: 值, ‘factor_C‘: 值, ‘factor_D‘: 值, ‘总分‘: 值},
    ‘stock_2‘: {‘factor_A‘: 值, ‘factor_B‘: 值, ‘factor_C‘: 值, ‘factor_D‘: 值, ‘总分‘: 值},
    # ... 其他股票
}
```
策略的关键步骤就是计算出每只股票的“总分”，然后进行排序筛选。

## 策略实施前提 📝

在开始为因子打分之前，我们必须明确一个重要的先验知识：**每个因子与股票预期收益率之间的相关性方向**。

![](img/2fb7117c27b675c31045e534e217a82a_1.png)

这是因为，打分需要依据因子的数值大小，但并非所有因子都是数值越大越好。例如：
*   **正向因子**：因子值越大，预期收益可能越好（如`每股收益(EPS)`、`净资产收益率(ROE)`）。
*   **负向因子**：因子值越小，预期收益可能越好（如`市盈率(PE)`、`市净率(PB)`）。

以下是确定因子方向的两种主要途径：
1.  **查阅研究报告**：各大券商的研究报告会提供经验性的结论。
2.  **自行计算分析**：通过历史数据计算因子与收益率的IC值等指标进行验证。

在后续的实战中，我们将选择已知相关性方向的因子，这是构建打分模型的基础。

## 策略流程总结 ✨

本节课我们一起学习了因子打分选股策略的框架。
*   **核心**：通过综合多个因子的评分（`总分 = factor_A_score + factor_B_score + ...`）来评估股票，替代单一的因子分析。
*   **关键前提**：必须预先知道每个因子与收益率的相关性方向（正向或负向），这是正确打分的基础。
*   **最终目标**：计算出每只股票的综合总分，并依据总分进行排名，选取排名靠前的股票构建投资组合。
