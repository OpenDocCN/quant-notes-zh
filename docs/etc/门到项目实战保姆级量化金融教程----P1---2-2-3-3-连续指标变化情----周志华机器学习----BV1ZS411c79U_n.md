## 概述：为什么增长率不能简单累加？ 🤔

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_6.png)

上一节我们介绍了如何计算简单的差异值和增长率。本节中我们来看看，当我们想了解一个连续时间序列（如股票价格）的整体累积变化时，会遇到什么问题。

例如，有一只股票，第一天价格是100元，第二天跌至50元，第三天涨至75元。
*   第一天的增长率是 `(50 - 100) / 100 = -0.5`
*   第二天的增长率是 `(75 - 50) / 50 = 0.5`

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_8.png)

如果简单地将增长率累加：`-0.5 + 0.5 = 0`。这个结果为0，似乎意味着股票从第一天到第三天“没有涨跌”。但观察原始价格：从100元到75元，实际上是下跌的。这说明**增长率不能直接进行加法运算**来反映整体累积效果。

## 解决方案：引入对数变换 📊

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_10.png)

对于连续时间的累积计算，我们需要借助数学中的对数公式。核心思想是计算连续复利或连续增长率。

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_12.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_14.png)

**核心公式**如下，我们计算每日价格比的对数，再进行累加：
```python
log_return = np.log(data / data.shift(1))
```
这个公式的数学原理在于：对数的加法等价于其内部数值的乘法。
`log(a/b) + log(b/c) = log(a/c)`
这意味着，将对数增长率累加后，我们得到的是**最终价格与初始价格之比的对数**，完美地描述了从起点到终点的整体累积变化。

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_16.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_18.png)

## 实践步骤：计算累积价值变化 💻

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_22.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_24.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_26.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_28.png)

以下是计算“初始1单位价值相当于后续每一天价值”的完整步骤：

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_30.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_32.png)

首先，我们计算连续的对数收益率。
```python
# 计算前一天的数据（使用shift方法将数据向上移动一格）
data_shifted = data.shift(1)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_34.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_36.png)

# 计算当日与前一日的比值，并取自然对数，得到对数收益率
log_returns = np.log(data / data_shifted)
```

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_38.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_40.png)

接着，我们需要将对数收益率进行累积求和，以得到从起始点到每一个时间点的总累积变化。
```python
# 使用cumsum函数进行累积求和
cum_log_returns = log_returns.cumsum()
```

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_42.png)

最后，由于我们之前取了对数，现在需要将其还原为实际的价值倍数。
```python
# 通过指数函数exp将累积对数收益率还原为实际的价值倍数
cumulative_value = np.exp(cum_log_returns)
```

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_44.png)

现在，`cumulative_value`这个序列就直观地展示了：假设在最初（例如2010年）投资该资产1元钱，那么到序列中的每一个后续日期，这1元钱相当于变成了多少元。我们可以将其可视化。

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_46.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_48.png)

```python
# 绘制累积价值变化图
cumulative_value.plot(figsize=(16, 8))
```
从生成的图中，你可以清晰地看到不同公司股票在多年间的累积增长表现。例如，可能发现亚马逊的1元投资在期末变成了约10元，而苹果的变成了约4元。

## 本章要点总结 📝

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_56.png)

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_58.png)

本节课中我们一起学习了金融时间序列中连续变化情况的分析方法：

![](img/40482d035bcfb616fa1f9e5a0b68ffb1_60.png)

1.  **增长率累加的局限性**：直接累加各期增长率无法正确反映资产价值的整体累积变化。
2.  **对数变换的应用**：通过计算价格比的对数（`np.log(data / data.shift(1))`），我们将乘法关系转化为加法关系，使得累积计算变得合理有效。
3.  **累积计算与还原**：使用`cumsum()`函数对对数收益率进行累积，再通过`np.exp()`函数还原，最终得到“初始单位价值”随时间变化的序列。
4.  **结果解读**：最终得到的图表直观展示了资产长期投资的累积回报，是量化分析中评估长期趋势的重要工具。
