## 获取历史行情数据
`tushare` 提供了多个数据接口，其中最常用的是获取历史行情数据的接口 `get_k_data`。

该接口的核心参数如下：
*   **`code`**：股票代码，例如 `‘601318’`。
*   **`ktype`**：K线类型。`‘D’` 代表日K线，`‘W’` 代表周K线，`‘M’` 代表月K线。默认为 `‘D’`。
*   **`autype`**：复权类型。`‘qfq’` 代表前复权。对于初学者，默认使用 `‘qfq’` 即可。
*   **`index`**：是否为指数。设置为 `True` 时，`code` 参数应传入指数代码（如 `‘000016’` 代表上证50）。
*   **`start`**：数据开始日期，格式为 `‘YYYY-MM-DD’`。
*   **`end`**：数据结束日期，格式为 `‘YYYY-MM-DD’`。不设置则默认为当前日期。

![](img/98b18448f51d50751d572ae0ebb37534_27.png)

### 基本使用示例
以下是一个获取单只股票历史数据的基本示例。返回的数据是Pandas的DataFrame格式，包含日期、开盘价、收盘价、最高价、最低价、成交量等信息。

```python
import tushare as ts

![](img/98b18448f51d50751d572ae0ebb37534_29.png)

![](img/98b18448f51d50751d572ae0ebb37534_31.png)

![](img/98b18448f51d50751d572ae0ebb37534_33.png)

# 获取股票‘601318’的历史日K线数据
df = ts.get_k_data(‘601318’)
print(df.head())

![](img/98b18448f51d50751d572ae0ebb37534_35.png)

# 将数据保存到CSV文件
df.to_csv(‘601318.csv’, index=False)
```

![](img/98b18448f51d50751d572ae0ebb37534_37.png)

![](img/98b18448f51d50751d572ae0ebb37534_39.png)

### 设置时间范围
如果不指定 `start` 参数，默认会获取最近几年的数据。如果想获取该股票自上市以来的全部历史数据，可以将 `start` 设为一个足够早的日期。

```python
# 获取‘601318’从2000年至今的全部历史数据
df_all = ts.get_k_data(‘601318’, start=‘2000-01-01’)
```

### 获取指数数据
获取大盘指数数据时，需要将 `index` 参数设置为 `True`。

![](img/98b18448f51d50751d572ae0ebb37534_45.png)

![](img/98b18448f51d50751d572ae0ebb37534_47.png)

```python
# 获取上证50指数（代码000016）的历史数据
df_index = ts.get_k_data(‘000016’, index=True)
```

## 其他数据接口
除了历史行情，`tushare` 还提供了其他类型的数据接口，例如：
*   **实时行情**：`get_today_all()`
*   **实时分笔数据**：`get_today_ticks()`
*   **大盘指数列表**：`get_index()`
