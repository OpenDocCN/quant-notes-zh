## 为什么增长率不能直接累加？🤔

我们观察的是一个平均结果，它反映了整体情况。但如果我们想计算一个连续时间序列的增长率，直接累加会存在问题。

举个例子：
*   假设一只股票第一天价格是100元。
*   第二天价格跌至50元。
*   第三天价格涨至75元。

![](img/7a2f900b7089c0b2095621fd05c6ed4b_6.png)

计算每日增长率：
*   第二天增长率 = (50 - 100) / 100 = -0.5
*   第三天增长率 = (75 - 50) / 50 = 0.5

![](img/7a2f900b7089c0b2095621fd05c6ed4b_8.png)

如果我们将增长率累加：-0.5 + 0.5 = 0。结果为0，似乎股票既没跌也没涨。

![](img/7a2f900b7089c0b2095621fd05c6ed4b_10.png)

但观察原始数据：从100元变为75元，实际上是下跌了。这说明对于连续时间的增长率，**不能直接做加法操作**。因为加法结果反映的是第一天和最后一天之间的净变化，而忽略了中间的波动过程。

## 解决方案：引入对数计算 📊

![](img/7a2f900b7089c0b2095621fd05c6ed4b_12.png)

对于连续时间的计算，我们需要借助数学公式——对数。

![](img/7a2f900b7089c0b2095621fd05c6ed4b_14.png)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_16.png)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_18.png)

**核心思想**：将对数形式的每日变化率累加，等价于计算首日与末日价格比值的对数。

![](img/7a2f900b7089c0b2095621fd05c6ed4b_20.png)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_22.png)

**公式推导**：
假设我们计算每日的对数收益率：`log(后一天价格 / 前一天价格)`。
那么累加两天的对数收益率：
`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/7a2f900b7089c0b2095621fd05c6ed4b_24.png)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_26.png)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_28.png)

最终结果 `log(75/100)` 直接反映了第一天价格（100）与第三天价格（75）之间的整体关系。**对数的加法，相当于内部数值的乘法**。

![](img/7a2f900b7089c0b2095621fd05c6ed4b_30.png)

形象地说，这相当于计算：如果初始股价为1个单位（例如1元），经过一系列涨跌后，最终相当于多少钱。

![](img/7a2f900b7089c0b2095621fd05c6ed4b_32.png)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_34.png)

## 代码实现：计算连续对数收益率 💻

![](img/7a2f900b7089c0b2095621fd05c6ed4b_36.png)

以下是计算连续对数收益率（即连续时间增长率）的步骤：

![](img/7a2f900b7089c0b2095621fd05c6ed4b_38.png)

1.  **获取前一天数据**：使用Pandas的 `shift()` 函数将数据整体向上移动一格。
2.  **计算对数收益率**：使用 `np.log(当天数据 / 前一天数据)`。
3.  **累加对数收益率**：使用 `np.cumsum()` 函数进行累积求和，得到每个时间点相对于起始点的总对数收益率。
4.  **还原为实际价值倍数**：使用 `np.exp()` 函数将对数收益率还原，得到“初始1单位相当于当前多少单位”的实际倍数。

![](img/7a2f900b7089c0b2095621fd05c6ed4b_40.png)

以下是关键代码示例：
```python
import numpy as np
import pandas as pd

![](img/7a2f900b7089c0b2095621fd05c6ed4b_42.png)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_44.png)

# 假设 `data` 是一个包含股价的Pandas Series
# 1. 计算对数收益率
log_returns = np.log(data / data.shift(1))

![](img/7a2f900b7089c0b2095621fd05c6ed4b_46.png)

# 2. 累积对数收益率
cumulative_log_returns = log_returns.cumsum()

![](img/7a2f900b7089c0b2095621fd05c6ed4b_48.png)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_50.png)

# 3. 还原为实际价值倍数（初始1单位对应的价值）
value_relative_to_start = np.exp(cumulative_log_returns)

![](img/7a2f900b7089c0b2095621fd05c6ed4b_52.png)

# 4. 绘图展示
value_relative_to_start.plot(figsize=(16, 8))
```

![](img/7a2f900b7089c0b2095621fd05c6ed4b_54.png)

执行上述代码后，我们将得到一张图表。图表显示，假设在起始点（如2010年）对各公司投资1个单位，那么到终点（如2018年），这1个单位价值相当于多少。例如，亚马逊的1元可能变成了约10元，而苹果的1元可能变成了约4元。这直观地展示了不同股票在时间序列上的累积增长表现。

2.  **引入解决方案**：通过引入**对数计算**，将对数收益率进行累加，其数学本质是累乘每日的价格变化比率。

    *   使用 `shift(1)` 获取前一天的数据。
    *   使用 `np.log()` 计算对数收益率。
    *   使用 `np.cumsum()` 实现累积计算。
    *   使用 `np.exp()` 将对数结果还原为实际价值倍数。
