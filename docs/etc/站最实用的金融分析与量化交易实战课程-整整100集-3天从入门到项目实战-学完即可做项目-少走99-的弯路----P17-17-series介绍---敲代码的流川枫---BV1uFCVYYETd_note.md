## 什么是Series？ 🤔

![](img/3a718a8973c12af151c35f76470960cc_5.png)

Series是Pandas中的一种核心数据对象，它是一种类似于一维数组的对象。你可以把它理解为一个**数组和字典的结合体**。

![](img/3a718a8973c12af151c35f76470960cc_7.png)

以下是创建Series的基本方法：

```python
import pandas as pd

![](img/3a718a8973c12af151c35f76470960cc_9.png)

![](img/3a718a8973c12af151c35f76470960cc_11.png)

# 从列表创建Series
s1 = pd.Series([2, 3, 4, 5])
print(s1)
# 输出：
# 0    2
# 1    3
# 2    4
# 3    5
# dtype: int64

# 从列表创建，并指定自定义索引（标签）
s2 = pd.Series([2, 3, 4, 5], index=[‘A‘, ‘B‘, ‘C‘, ‘D‘])
print(s2)
# 输出：
# A    2
# B    3
# C    4
# D    5
# dtype: int64
```

![](img/3a718a8973c12af151c35f76470960cc_13.png)

如你所见，第一种写法像一个带默认下标（0,1,2,3）的列表，第二种写法则像字典，左边是自定义的键（A,B,C,D），右边是对应的值。这正是Series“结合体”特性的直观体现。

![](img/3a718a8973c12af151c35f76470960cc_15.png)

## 继承自数组/列表的特性 🧮

Series继承了NumPy数组或Python列表的许多有用特性，使其能进行高效的数值计算和操作。

以下是Series支持的数组特性列表：

1.  **从列表或数组创建**：可以从Python列表或NumPy数组创建Series对象。
2.  **通过下标索引访问**：即使指定了自定义标签，依然可以通过从0开始的整数下标进行访问。
    ```python
    s = pd.Series([10, 20, 30, 40], index=[‘a‘, ‘b‘, ‘c‘, ‘d‘])
    print(s[0])  # 输出：10
    print(s[‘a‘]) # 输出：10
    ```
3.  **向量化运算**：可以与标量（单个数字）进行运算，也可以在两个相同大小的Series之间进行逐元素运算（加、减、乘、除、比较等）。
    ```python
    s = pd.Series([1, 2, 3, 4])
    print(s + 10)      # 每个元素加10
    print(s * 2)       # 每个元素乘2
    print(s > 2)       # 返回布尔型Series
    ```
4.  **切片**：可以使用整数下标进行切片操作。
    ```python
    s = pd.Series([10, 20, 30, 40, 50])
    print(s[1:4])  # 获取下标1到3的元素
    ```
5.  **支持通用函数**：支持NumPy的通用函数（ufunc），如`np.abs`（取绝对值）、`np.log`（取对数）等。
6.  **布尔型索引**：可以通过布尔条件来筛选数据。
    ```python
    s = pd.Series([5, 10, 15, 20])
    print(s[s > 12])  # 输出大于12的元素
    ```

## 继承自字典的特性 📖

![](img/3a718a8973c12af151c35f76470960cc_19.png)

除了数组特性，Series还具备类似字典的功能，这使得它可以通过有意义的标签来组织和访问数据。

以下是Series支持的字典特性列表：

1.  **从字典创建**：可以直接用一个Python字典来创建Series，字典的键（key）会自动成为Series的索引（标签）。
    ```python
    data = {‘苹果‘: 3.5, ‘香蕉‘: 2.0, ‘橙子‘: 4.0}
    s = pd.Series(data)
    print(s)
    ```
2.  **通过标签索引访问**：这是Series最强大的特性之一，可以通过自定义的标签来获取值。
    ```python
    s = pd.Series([10, 20, 30], index=[‘X‘, ‘Y‘, ‘Z‘])
    print(s[‘Y‘])  # 输出：20
    ```
3.  **`in`操作**：可以检查某个标签是否存在于Series的索引中。
    ```python
    s = pd.Series([10, 20], index=[‘A‘, ‘B‘])
    print(‘A‘ in s)   # 输出：True
    print(‘C‘ in s)   # 输出：False
    ```
4.  **花式索引与标签切片**：可以通过传入一个标签列表来获取多个值（花式索引），也可以使用标签进行切片。**注意：使用标签切片时，结果包含起始和结束标签。**
    ```python
    s = pd.Series([1, 2, 3, 4, 5], index=[‘a‘, ‘b‘, ‘c‘, ‘d‘, ‘e‘])
    # 花式索引
    print(s[[‘a‘, ‘c‘, ‘e‘]])
    # 标签切片 (包含‘b‘和‘d‘)
    print(s[‘b‘:‘d‘])
    ```
5.  **获取索引和值**：可以分别获取Series的索引部分和值部分。
    ```python
    s = pd.Series([100, 200, 300], index=[‘一‘, ‘二‘, ‘三‘])
    print(s.index)   # 输出索引对象
    print(s.values)  # 输出值数组
    ```
    **注意**：对Series进行`for`循环时，遍历的是它的**值**，而不是索引（这与字典遍历键不同）。如果需要遍历索引和值，可以使用`s.items()`方法。

## Series的实际应用场景 💡

Series结合了有序数组和键值对字典的优点，使其在多种场景下非常实用：

![](img/3a718a8973c12af151c35f76470960cc_21.png)

*   **时间序列数据**：例如记录一支股票每日的收盘价。日期可以作为标签，价格是值。这样既可以通过日期（标签）快速查询某天的价格，也可以通过位置（下标）进行切片分析，如前5天的数据。
*   **带标签的测量数据**：在科学实验或业务监控中，测量点（如传感器ID、产品型号）作为标签，测量值作为数据。Series保证了数据的有序性，同时支持通过有意义的标签进行快速检索。
*   **替代复杂的嵌套结构**：正如引言中提到的，如果你之前需要用一个列表存储元组`(时间戳， 值)`来保证顺序，再用循环查找特定时间戳，那么使用Series（以时间戳为索引）会让代码更简洁、效率更高。

本节课中我们一起学习了Pandas库的核心数据结构之一——**Series**。我们了解到Series是一个一维的、带标签的数组，它巧妙地融合了列表（数组）的运算能力和字典的键值访问特性。我们掌握了如何创建Series，以及如何利用其下标和标签进行索引、切片和运算。理解Series是后续学习更复杂的`DataFrame`结构的基础，也是高效进行数据分析的第一步。
