接下来，我们修改股票选择逻辑，观察排名靠后股票的表现。

![](img/db1fad7880ab4742368864b0fc5798c3_50.png)

![](img/db1fad7880ab4742368864b0fc5798c3_52.png)

以下是修改股票选择范围的代码：
```python
# 原代码：选择排名前10的股票（得分最高）
# selected_stocks = ranked_stocks.iloc[:10]

![](img/db1fad7880ab4742368864b0fc5798c3_54.png)

![](img/db1fad7880ab4742368864b0fc5798c3_56.png)

# 修改后：选择排名第10名之后的股票（得分较低）
selected_stocks = ranked_stocks.iloc[10:]
```
执行回测后，基准收益约为-10%。而选择了得分较低（即我们认为基本面较差）的股票后，策略收益比基准收益更差。这从反面验证了我们的打分逻辑在一定程度上是有效的：得分高的股票组合表现优于得分低的组合。

![](img/db1fad7880ab4742368864b0fc5798c3_58.png)

![](img/db1fad7880ab4742368864b0fc5798c3_60.png)

回测报告还包含了详细的交易记录。我们可以查看每月是否按时调仓，以及持仓和账户资金的变动情况。例如，初始资金为100万，在投资了得分最低的股票组合后，最终账户价值约为77万。

![](img/db1fad7880ab4742368864b0fc5798c3_62.png)

现在，我们将股票选择范围改回排名前十的组合。后续大家实验时，默认使用这个设置即可，其效果相对更好。我们的策略是基于研究报告，筛选出“越高越好”和“越低越好”的财务指标，对股票进行综合打分。

![](img/db1fad7880ab4742368864b0fc5798c3_64.png)

综上所述，打分法是一种在量化投资中应用广泛且较为直观的策略构建方法。它能相对直接地获得一个还不错的结果。当然，策略表现会受到市场大环境（如熊市）的显著影响，导致在某些时期收益一般，这是由市场本身决定的常见现象。
