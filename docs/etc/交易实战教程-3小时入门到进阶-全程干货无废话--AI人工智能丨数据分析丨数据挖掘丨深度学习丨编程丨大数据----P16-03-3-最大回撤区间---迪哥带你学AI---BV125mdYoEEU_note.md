在本节课中，我们将要学习量化交易中的一个重要风险指标——最大回撤。我们将理解它的概念、计算方式及其在策略评估中的意义。

## 概述

最大回撤是衡量投资策略风险的关键指标。它描述的是在特定时间段内，资产净值从最高点到之后最低点的最大跌幅。这个指标不评估策略的盈利能力，而是评估其可能面临的最大风险。

![](img/80eece3a0dad583d2c3c50cc039e12e5_3.png)

## 什么是最大回撤？

上一节我们介绍了策略分析中的基本概念，本节中我们来看看最大回撤。

在策略分析中，最大回撤表示的是一个特定的区间。

![](img/80eece3a0dad583d2c3c50cc039e12e5_1.png)

这个区间描述了从某个高点持续下跌到某个低点的过程。最大回撤不能仅从字面意义理解，它描述的是投资过程中可能遭遇的最糟糕情况。

具体来说，在投资过程中，资产价格必然会有下跌的时候。最惨烈、跌幅最大的那个持续下跌区间，就称为最大回撤区间。

![](img/80eece3a0dad583d2c3c50cc039e12e5_3.png)

## 最大回撤的数学定义

为了更精确地理解，我们通过公式来看“惨劲儿”是如何量化的。

最大回撤的计算公式可以表示为：

**最大回撤 = max( (Pi - Pj) / Pi )**，其中 **i < j**

在这个公式中：
*   **Pi** 代表第 i 天的资产价格（或净值）。
*   **Pj** 代表第 i 天之后的某一天 j 的资产价格。

这个公式的含义是：遍历所有可能的时间点 i 和它之后的点 j，计算从 i 点到 j 点的相对跌幅（缩水比例），然后找出其中最大的那个值。

它描述的是在整个数据周期中，哪两点之间的下跌最为惨烈。这个下跌过程可能中间有短暂反弹，但整体趋势是向下的。例如，可能连续30天经历了最惨的一次下跌，从某个价格跌到另一个价格。

## 为什么最大回撤很重要？

![](img/80eece3a0dad583d2c3c50cc039e12e5_5.png)

最大回撤是一个风险指标。在评估策略时，我们不仅要看盈利能力，还要看风险。

风险意味着什么？假设你将全部资金投入一只股票，然后遭遇了持续十天的最大回撤区间，价格从15元一路暴跌，这非常可怕。虽然之后价格可能回升，但这段最大跌幅及其持续时间，是衡量策略风险承受能力的关键。

![](img/80eece3a0dad583d2c3c50cc039e12e5_7.png)

这个指标描述了在这段时间内，你的投资组合“惨”到了什么程度。因此，当你看到“回撤”时，应理解它是一个风险指标，描述的是策略在执行过程中可能经历的最糟糕阶段。

以2008年股市为例，大盘从6000多点跌至1600多点，期间的最大回撤极大，导致了严重的亏损。这个指标能帮助投资者理解极端市场情况下的潜在损失。

![](img/80eece3a0dad583d2c3c50cc039e12e5_5.png)

![](img/80eece3a0dad583d2c3c50cc039e12e5_9.png)

在我们的回测策略中，系统会自动统计并标识出回测期间的最大回撤区间。

![](img/80eece3a0dad583d2c3c50cc039e12e5_7.png)

## 如何计算最大回撤？

![](img/80eece3a0dad583d2c3c50cc039e12e5_13.png)

接下来，我们动手实现最大回撤的计算。首先需要明确计算逻辑：我们需要计算所有可能的下跌区间，然后从中选出跌幅最大的那个。

我们将使用Pandas库来辅助计算。首先介绍一个关键函数：`cummax()`。

![](img/80eece3a0dad583d2c3c50cc039e12e5_15.png)

`cummax()` 是累积最大值函数。对于序列中的每一个位置，它返回从序列开始到该位置为止的最大值。

以下是 `cummax()` 函数的一个示例：

![](img/80eece3a0dad583d2c3c50cc039e12e5_17.png)

```python
import pandas as pd
import numpy as np

![](img/80eece3a0dad583d2c3c50cc039e12e5_19.png)

# 创建一个示例序列
s = pd.Series([2, np.nan, 5, 3, 8, 1])
print("原始序列:")
print(s)

![](img/80eece3a0dad583d2c3c50cc039e12e5_21.png)

# 计算累积最大值
cum_max = s.cummax()
print("\n累积最大值序列:")
print(cum_max)
```

![](img/80eece3a0dad583d2c3c50cc039e12e5_23.png)

运行结果类似于：
```
原始序列:
0    2.0
1    NaN
2    5.0
3    3.0
4    8.0
5    1.0

![](img/80eece3a0dad583d2c3c50cc039e12e5_25.png)

![](img/80eece3a0dad583d2c3c50cc039e12e5_27.png)

累积最大值序列:
0    2.0
1    NaN
2    5.0
3    5.0
4    8.0
5    8.0
```

可以看到，对于第一个数，最大值是它自己（2.0）。对于前三个数，最大值是5.0。NaN值不参与计算。这个操作与之前学过的 `cumsum()`（累积和）类似。

![](img/80eece3a0dad583d2c3c50cc039e12e5_15.png)

## 计算最大回撤的步骤

![](img/80eece3a0dad583d2c3c50cc039e12e5_31.png)

为了找到“最惨”的下跌区间，我们可以先找到每个时间点之前的资产最高点（制高点）。

1.  **获取历史价格序列**：假设我们有一个DataFrame `df`，其中包含资产价格，我们选取其中一列，例如 `‘close’`。
    ```python
    price_series = df[‘close’]
    ```

![](img/80eece3a0dad583d2c3c50cc039e12e5_33.png)

2.  **计算累积最大值序列**：这个序列表示到每一天为止的历史最高价。
    ```python
    cumulative_max = price_series.cummax()
    ```

![](img/80eece3a0dad583d2c3c50cc039e12e5_35.png)

3.  **计算每日潜在回撤**：用历史最高价减去当日的实际价格，得到每日相对于历史高点的下跌幅度。
    ```python
    drawdown = cumulative_max - price_series
    ```
    如果某天价格就是历史最高点，那么该日的 `drawdown` 为0。

![](img/80eece3a0dad583d2c3c50cc039e12e5_37.png)

4.  **计算每日回撤率**：将下跌幅度除以对应的历史最高点，得到回撤比例（即缩水比例）。
    ```python
    drawdown_pct = drawdown / cumulative_max
    ```

5.  **找出最大回撤**：在所有每日回撤率中，找出最大值，这就是整个时间段内的最大回撤。
    ```python
    max_drawdown = drawdown_pct.max()
    ```

将以上步骤整合，计算最大回撤的完整代码如下：

![](img/80eece3a0dad583d2c3c50cc039e12e5_39.png)

```python
# 假设 price_series 是资产的价格序列
cumulative_max = price_series.cummax()
drawdown = cumulative_max - price_series
drawdown_pct = drawdown / cumulative_max
max_drawdown = drawdown_pct.max()

![](img/80eece3a0dad583d2c3c50cc039e12e5_41.png)

print(f"最大回撤为：{max_drawdown:.2%}")
```

执行后，可能得到类似 `3.07%` 的结果。这表示在该策略回测期内，资产净值从某个高点下跌到低点的最大相对跌幅约为3%。

![](img/80eece3a0dad583d2c3c50cc039e12e5_45.png)
