## 核心技能概览

![](img/025042f2afad0fa8eee8627ab388c84e_1.png)

进行量化交易需要掌握一系列跨领域的技能。以下是几个关键方面。

### 机器学习算法 🧠

机器学习是量化交易的核心工具之一。你需要掌握最常规的算法，包括：
*   **回归**：用于预测连续值，例如预测股价。
*   **分类**：用于预测离散类别，例如判断股票涨跌。
*   **聚类**：用于发现数据中的内在分组结构。

### 特征工程 🔧

上一节我们介绍了机器学习算法，本节中我们来看看数据处理的重要性。在数据挖掘中，**特征工程**是一个至关重要的概念。它指的是如何处理原始数据，并从中提取出最有价值的信息。

数据决定了模型性能的上限，而算法只是用来逼近这个上限的工具。在量化交易中，我们面对的是庞大而复杂的金融数据。例如，分析股票不仅涉及开盘价、收盘价，还涉及公司财务数据、市场宏观数据等多个层面。

以下是特征工程的核心任务：
*   整合来自市场、公司、财务报表、股市走势等多层面的数据。
*   设计算法，将这些数据有效地融合到模型中。
*   从海量数据中选择最有价值的信息。

### 统计学与数学方法 📐

量化交易与数学紧密相关。无论是算法还是交易策略，本质上都是将数学公式应用到数据中。许多量化交易岗位都要求应聘者具备数学、统计学、计算机或金融背景。

需要掌握的数学知识点包括但不限于：
*   概率论与数理统计
*   线性代数
*   微积分
*   时间序列分析

## 聚焦领域：股票 vs. 期货

有同学问，在量化交易中，股票和期货哪个是重点？本课程将更侧重于股票分析。

![](img/025042f2afad0fa8eee8627ab388c84e_10.png)

选择股票作为重点的原因如下：
*   **数据丰富性**：股票相关的数据指标非常全面（如公司财务数据、市场数据等），非常适合进行数据挖掘。
*   **期货的特点**：期货交易与市场即时关联度极高，更依赖从业人员对市场的主观判断和行业经验，这不太适合作为标准化数据挖掘教学的重点。

因此，课程会以股票案例分析为主，期货仅作为少数示例提及。

## 量化交易的本质：数据挖掘

既然提到了数据挖掘，我们来看看它具体做什么。简单来说，数据挖掘流程如下：
1.  **数据处理**：对获取的原始数据进行清洗、转换和特征提取。
2.  **策略设计**：基于处理后的数据设计交易逻辑。
3.  **回测验证**：在历史数据上测试策略的表现。其公式可简化为：`策略表现 = 回测(策略， 历史数据)`。
4.  **指导决策**：根据回测结果，为实际投资决策提供依据。

量化交易可以被视为将数据挖掘算法应用于金融数据的一个特定领域。它的目的不仅仅是预测股票的涨跌（一个分类问题），更是为了实现**收益最大化**。例如，从300只股票中，如何分配有限的本金以选择风险收益比最优的组合，同样是一个数据挖掘问题。
