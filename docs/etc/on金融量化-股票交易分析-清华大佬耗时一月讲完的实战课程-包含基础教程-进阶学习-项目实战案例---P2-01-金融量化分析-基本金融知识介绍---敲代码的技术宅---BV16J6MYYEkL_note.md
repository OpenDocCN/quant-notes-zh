## 什么是金融？

金融的定义是：对现有资源进行重新整合之后，实现价值和利润的等效流通。这个概念可能听起来有些抽象，但简单来说，金融就是关于资金流动和增值的行业。它并非完全是不劳而获的投机行为，而是能够促进经济发展、实现双赢的活动。

例如，一位拥有闲置资金的投资者，可以将资金投入一个有潜力的创业公司。创业者获得了发展所需的资金，公司得以壮大；而投资者则获得了公司成长带来的回报。这个过程刺激了经济发展，让资金流向了需要它的地方。这就是金融活动的一个典型例子。

## 常见的金融工具

![](img/61786f3db7c30550405e7827816c1ece_3.png)

在金融市场中，可交易的金融资产被称为金融工具。对于个人投资者而言，最常见的几种金融工具包括：股票、期货、黄金、外汇和基金。以下是它们的简要介绍：

*   **期货**：这是一种高风险、高收益的金融衍生品。交易双方约定在未来某一时间，以约定的价格买卖某种标的物（如大宗商品）。其核心在于交易双方对资产未来价格走势的判断不同。例如，发电厂预计煤炭价格未来会上涨，而煤矿主预计价格会下跌，双方就可以签订一份期货合约来锁定未来的交易价格，以规避各自的风险。
*   **黄金**：作为一种传统的硬通货，黄金价格相对稳定，常被视为保值避险的投资选择。其价格波动主要受全球黄金储量、开采量以及货币供应量等因素影响。公式上可以简单理解为：`黄金价格 ∝ 货币总量 / 黄金总量`。
*   **外汇**：指不同国家货币之间的兑换交易，其价格表现为汇率（如1美元=6.5人民币）。大型投资机构会利用各国经济状况、利率政策等差异导致的汇率波动进行套利。由于汇率日常波动较小，通常不适合个人小额投资。
*   **基金**：这是一种集合投资方式。投资者将资金交给专业的基金管理公司，由基金经理负责投资于股票、债券等多种资产。其优势在于利用专业管理分散风险，但收益和风险通常都低于直接投资股票。公式可表示为：`基金净值 = (总资产 - 总负债) / 基金总份额`。

上一节我们介绍了金融的基本概念和几种常见的金融工具，本节中我们来看看本课程最核心的金融工具——股票。

## 重点：股票

股票是股份有限公司为筹集资金而发行给股东的所有权凭证。购买股票即成为公司的股东，享有公司成长带来的收益（如分红）和资产增值，同时也需承担相应的风险。

简单来说：
*   **你买入股票** = 成为公司部分所有者。
*   **公司盈利** = 你可能获得分红，股票价值可能上涨。
*   **公司亏损** = 你可能没有分红，股票价值可能下跌。

股票市场为投资者提供了分享企业成长红利的渠道，也是本课程后续进行量化分析的主要对象。
