![](img/8d8d291ada94a81b5f5d8b846a9a1762_1.png)

在本节课中，我们将学习Excel中两个极其重要的查找与引用函数：`VLOOKUP`和`MATCH`。它们能高效地跨表匹配数据，是数据处理与分析中的核心工具。

## 概述

查找与匹配函数主要用于从不同来源的数据表中，根据特定条件（如姓名、学号）查找并提取相关信息。手动查找在数据量大时效率极低，而这两个函数可以自动化完成此过程。

## VLOOKUP函数详解

![](img/8d8d291ada94a81b5f5d8b846a9a1762_3.png)

上一节我们介绍了函数的基本概念，本节中我们来看看查找与匹配的明星函数——`VLOOKUP`。

![](img/8d8d291ada94a81b5f5d8b846a9a1762_5.png)

![](img/8d8d291ada94a81b5f5d8b846a9a1762_7.png)

### 应用场景

![](img/8d8d291ada94a81b5f5d8b846a9a1762_9.png)

当需要将不同来源但格式类似的数据（例如，不同科目的成绩表）整合到一个总表中时，如果各表的人员顺序不一致，手动复制粘贴或查找会非常繁琐且容易出错。`VLOOKUP`函数可以完美解决这个问题。

![](img/8d8d291ada94a81b5f5d8b846a9a1762_11.png)

![](img/8d8d291ada94a81b5f5d8b846a9a1762_13.png)

### 函数结构与参数

![](img/8d8d291ada94a81b5f5d8b846a9a1762_15.png)

`VLOOKUP`函数由四个参数组成，其基本语法为：
`=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])`

以下是每个参数的具体解释：

![](img/8d8d291ada94a81b5f5d8b846a9a1762_17.png)

1.  **lookup_value（查找值）**：要查找的对象，例如姓名或学号。通常选择具有唯一性的标识。
2.  **table_array（查找区域）**：要在其中进行查找的数据区域。**关键点**：查找值必须位于该区域的第一列。
3.  **col_index_num（列序数）**：你希望返回的值在`table_array`中位于第几列。计数从查找区域的第一列（即查找值所在列）开始为1。
4.  **range_lookup（匹配条件）**：指定查找方式。
    *   `0` 或 `FALSE`：精确匹配。
    *   `1` 或 `TRUE`：近似匹配。在数据匹配中，通常使用精确匹配。

![](img/8d8d291ada94a81b5f5d8b846a9a1762_19.png)

### 操作演示

假设我们有一个总表（表1）和一个数据库成绩单（表2），需要将表2中的成绩匹配到表1中。

1.  在表1的目标单元格输入公式：`=VLOOKUP(查找的姓名, 表2的查找区域, 成绩在区域中的列数, 0)`
    *   例如：`=VLOOKUP(A2, Sheet2!$B$2:$C$21, 2, 0)`
2.  公式解释：在`Sheet2`的B2到C21区域中，精确查找A2单元格的姓名，并返回该区域第2列（即C列）对应的值。
3.  由于需要向下填充公式且查找区域应固定不变，必须对区域引用使用绝对引用（按`F4`键添加`$`符号）。
4.  双击填充柄，即可为所有姓名匹配到对应的成绩。

### 注意事项与反向匹配

使用`VLOOKUP`时，可能会出现`#N/A`错误，通常是因为：
*   查找值在源表中不存在。
*   源表中的查找值存在不可见字符（如空格）。
*   需要进行数据核对。

一个常见的进阶需求是“反向匹配”：检查表2中的人员是否都存在于表1中。这引出了我们下一个函数。

## MATCH函数详解

学会了`VLOOKUP`进行值匹配后，我们来看看`MATCH`函数，它用于定位数据的位置。

### 函数作用

`MATCH`函数用于在指定区域中搜索特定项，然后返回该项在区域中的相对位置（第几行或第几列），而不是返回值本身。它常用来判断某个值是否存在于另一个列表中。

### 函数结构与参数

`MATCH`函数由三个参数组成，其基本语法为：
`=MATCH(lookup_value, lookup_array, [match_type])`

以下是每个参数的具体解释：

1.  **lookup_value（查找值）**：需要查找的对象。
2.  **lookup_array（查找区域）**：可能包含查找值的单元格区域。
3.  **match_type（匹配类型）**：指定匹配方式。
    *   `0`：精确匹配。
    *   `1` 或 `-1`：近似匹配（用于数字区间查找）。通常使用精确匹配`0`。

![](img/8d8d291ada94a81b5f5d8b846a9a1762_21.png)

### 操作演示

继续之前的例子，我们想知道表2（数据库成绩单）中的姓名，在表1（总表）的姓名列中是否存在。

![](img/8d8d291ada94a81b5f5d8b846a9a1762_23.png)

![](img/8d8d291ada94a81b5f5d8b846a9a1762_25.png)

1.  在表2旁输入公式：`=MATCH(查找的姓名, 表1的姓名区域, 0)`
    *   例如：`=MATCH(F2, Sheet1!$A$2:$A$21, 0)`
2.  公式解释：在`Sheet1`的A2到A21区域中，精确查找F2单元格的姓名。
3.  如果找到，函数返回该姓名在`Sheet1!$A$2:$A$21`区域中的行位置（例如，第4个）。如果未找到，则返回错误值`#N/A`。
4.  通过判断公式结果是否为数字，即可快速识别出哪些人员在表1中缺失。

![](img/8d8d291ada94a81b5f5d8b846a9a1762_27.png)

## 函数对比与总结

本节课中我们一起学习了`VLOOKUP`和`MATCH`这两个查找与匹配函数。

*   **`VLOOKUP`**：用于**查找并返回值**。需要指定查找值、查找区域、返回值的列序数和匹配方式。其核心是“根据A，找到对应的B”。
*   **`MATCH`**：用于**查找并返回位置**。只需指定查找值和查找区域。其核心是“判断A是否存在，并告诉你在哪”。

![](img/8d8d291ada94a81b5f5d8b846a9a1762_29.png)

它们通常结合使用，以完成更复杂的数据核对与整合任务。掌握这两个函数，将极大地提升你在数据处理工作中的效率和准确性。请务必通过练习巩固理解。
