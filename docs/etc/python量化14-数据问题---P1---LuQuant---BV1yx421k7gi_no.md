## 方法一：使用Python和yfinance库

上一节我们概述了课程目标，本节中我们来看看第一种方法——使用Python代码下载数据。这种方法的核心是`yfinance`库，它是一个非官方的库，通过网页抓取的方式从雅虎财经获取数据。

### 安装与导入库

在运行代码之前，需要确保已安装`yfinance`库。以下是安装方法。

**安装命令：**
```bash
pip install yfinance
```
您可以在终端、VS Code或Jupyter Notebook中执行此命令。在Jupyter Notebook中，可以在代码单元前加上感叹号`!`来执行。

**导入库：**
```python
import yfinance as yf
```

### 下载股票历史数据

安装并导入库后，就可以指定股票代码并下载历史数据了。我们以苹果公司（AAPL）的股票为例。

![](img/f1688b05d728f305a8598da9d94def09_6.png)

**代码示例：**
```python
# 指定股票代码
ticker_symbol = "AAPL"

# 创建Ticker对象
ticker = yf.Ticker(ticker_symbol)

# 方法1：指定开始和结束日期，下载日线数据
hist_data = ticker.history(start="1990-01-01", end="2022-12-31", interval="1d")
print(f"下载了 {len(hist_data)} 行日线数据。")
print(hist_data.head())
```

这段代码会下载AAPL从1990年到2022年的每日价格数据，并打印数据框的前几行和数据总行数。

### 下载指定时间段的数据

除了指定起止日期，还可以使用`period`参数下载最近一段时间的数据，这对于获取近期数据非常方便。

**代码示例：**
```python
# 方法2：下载过去20天的日线数据
hist_data_recent = ticker.history(period="20d", interval="1d")
print(f"下载了最近 {len(hist_data_recent)} 天的日线数据。")
```

### 时间范围与数据限制

使用`yfinance`时，需要注意其数据获取的限制。以下是主要的时间范围（`interval`）及其对应的历史数据可获取期限。

以下是`yfinance`支持的主要时间范围及其数据获取限制：

*   **1分钟、2分钟、5分钟、15分钟、30分钟、90分钟数据**：最多只能获取过去**7天**的数据。
*   **1小时数据**：最多可获取过去**730天**（约2年）的数据。
*   **1天、5天、1周、1个月、3个月数据**：通常可以获取很长的历史数据。

如果请求的数据超出了允许的范围，库会返回错误信息。例如，尝试获取8天前的1分钟数据或60天前的15分钟数据都会失败。

**代码示例（触发限制）：**
```python
# 尝试获取8天前的1分钟数据（会失败）
try:
    data = ticker.history(period="8d", interval="1m")
except Exception as e:
    print(f"错误信息: {e}")
```

### 注意事项

使用`yfinance`库时，有几点需要牢记：

1.  **非官方支持**：`yfinance`并非雅虎官方产品，它依赖网页抓取，数据的准确性和更新速度可能无法保证，且未来维护情况不确定。
2.  **查询限制**：由于采用网页抓取方式，过于频繁的查询（例如，构建交易机器人每分钟连续查询）可能会触发反爬机制，导致IP被暂时限制。但对于一般的回测（每天数十次查询）通常没有问题。
3.  **数据清洗**：库会自动过滤掉交易日中的休市时段和周末数据。

## 方法二：使用Dukascopy平台

![](img/f1688b05d728f305a8598da9d94def09_8.png)

上一节我们介绍了使用Python代码获取数据的方法，本节中我们来看看第二种图形化界面方法——使用Dukascopy平台。Dukascopy是瑞银集团提供的交易平台，其“历史数据源”功能允许用户免费下载多种金融工具的CSV格式数据。

### 访问与选择数据

![](img/f1688b05d728f305a8598da9d94def09_10.png)

![](img/f1688b05d728f305a8598da9d94def09_11.png)

首先，您需要访问Dukascopy网站并找到“历史数据源”页面。该页面提供了丰富的数据类别。

以下是数据选择步骤：

1.  在数据类别中选择您需要的类型，例如“外汇主要货币”。
2.  选择具体的交易品种，例如“欧元/美元 (EUR/USD)”。
3.  选择所需的时间范围（如“4小时”）。
4.  选择数据类型（如“卖出价”或“买入价”）。
5.  设置数据的开始日期和结束日期。
6.  您可以选择是否过滤掉周末和非交易日的数据。
7.  时区建议选择“UTC”或“GMT”以获得统一的世界时间。

### 下载数据文件

设置好所有参数后，点击“下载”按钮。系统会提示您登录或注册一个免费账户。完成账户流程后，即可下载CSV格式的数据文件。请确保浏览器将文件保存为`.csv`格式。

这种方法简单直接，特别适合不希望编写代码、只需要一次性导出数据文件的用户。平台提供的数据种类非常全面，包括外汇、股票、商品、指数等，足以满足大多数回测需求。
