现在，我们来执行具体的调仓操作。以下是调仓步骤：

![](img/d610ce80c372929203e6ab410219633f_50.png)

首先，循环遍历所有需要移除的股票，并使用 `order_target_percent` 函数将其持仓比例调整为0，即全部卖出。

```python
    order_target_percent(stock, 0)  # 卖出该股票，使持仓比例变为0
```

关键绩效指标包括：
*   **年化收益率**：策略在一年内的平均收益率。
*   **基准年化收益率**：比较基准（如沪深300指数）的年化收益率。
*   **夏普比率**：衡量每承受一单位风险，所获得的超额回报。
*   **最大回撤**：策略净值从峰值到谷底的最大下跌幅度。

## 总结
本节课中我们一起学习了如何构建并评估一个因子选股策略。我们实现了以下核心步骤：
1.  **设置定时器**：确定调仓频率（如每月）。
2.  **数据查询与预处理**：获取股票财务指标（如市值、市盈率）并进行必要的数据清洗与中性化处理。
3.  **因子筛选**：根据设定规则（如选择因子值最低的30%股票）构建股票池。
4.  **执行调仓**：
    *   卖出当前持仓中不在新股票池的股票（`order_target_percent(stock, 0)`）。
    *   将资金平均买入新股票池中的所有股票（`order_target_percent(stock, 1/N)`）。
5.  **效果评估**：通过回测报告分析策略的收益、风险指标，并与基准进行比较。

这个过程展示了量化策略从构思、编码到回测评估的完整流程。需要注意的是，回测结果基于历史数据，不代表未来表现，实际应用中需考虑更多复杂因素。
