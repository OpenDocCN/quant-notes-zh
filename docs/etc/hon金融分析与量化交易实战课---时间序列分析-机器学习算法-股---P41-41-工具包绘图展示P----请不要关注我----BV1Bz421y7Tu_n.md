# 量化交易实战课程：P41：工具包绘图展示

## 概述
在本节课中，我们将学习如何使用量化工具包中的绘图功能，对之前计算出的因子IC值进行可视化分析。通过图表，我们可以更直观地评估因子的有效性和稳定性。

---

![](img/6c827c058f95341c070c58221b4a48d7_1.png)

上一节我们介绍了如何计算因子与收益率之间的相关系数（IC值）。本节中，我们来看看如何将这些计算结果通过图表展示出来，以便进行更深入的分析。

![](img/6c827c058f95341c070c58221b4a48d7_3.png)

我们通常偏好IC值较大的因子，因为这意味着因子与收益率的相关性更强，可能更具研究价值。然而，IC值每天都会变化，仅从数字结果观察其趋势并不直观。因此，我们需要借助可视化工具来绘制其时间序列图。

以下是使用`alphalens`工具包中的绘图函数进行展示的步骤。

![](img/6c827c058f95341c070c58221b4a48d7_5.png)

1.  **导入绘图函数**：我们从`alphalens`中导入用于绘制IC值时间序列图的函数。
    ```python
    from alphalens import plotting
    ```

2.  **调用绘图函数**：使用`plotting.plot_information_table(ic_data)`函数，其中`ic_data`是我们之前计算好的IC值数据。
    ```python
    plotting.plot_information_table(ic_data)
    ```

3.  **解读图表**：执行代码后，系统会生成一张图表。
    *   图中**蓝色曲线**展示了每日IC值的实际走势，波动范围通常较大。
    *   图中**绿色曲线**是IC值的月度移动平均线（默认以一个月为窗口计算平均值），它有助于平滑每日波动，揭示更长期的趋势。
    *   图表还会标注出IC值的**整体均值（Mean）** 和**整体标准差（Std）**，并计算一个**信息比率（IR）**，其公式为：
        **IR = Mean / Std**
        信息比率用于衡量IC值的稳定性。比值越大，说明IC值相对于其波动（标准差）更为显著和稳定。

由于蓝色曲线波动剧烈，难以直接判断趋势，我们主要观察**绿色移动平均线**。通过该线，我们可以评估因子在较长时间段内是否持续有效。例如，我们希望看到绿色曲线呈现上升或稳定在高位的趋势。如果曲线过于平稳或数值较小，则可能意味着该因子的预测能力不强。

![](img/6c827c058f95341c070c58221b4a48d7_7.png)

## 总结
本节课中我们一起学习了如何利用`alphalens`工具包对因子IC值进行可视化。我们重点掌握了绘制IC值时间序列图的方法，并学会了如何通过观察其移动平均线来评估因子的有效性与稳定性。可视化是量化分析中不可或缺的一环，它能帮助我们从纷繁的数据中快速捕捉到关键信息。
