## 概述
在本节课中，我们将学习量化交易的核心概念。我们将探讨量化交易的本质、它与传统交易的区别，以及入门量化交易所需要掌握的核心技能。

## 什么是量化交易？
上一节我们介绍了课程的整体框架，本节中我们来看看量化交易到底做了一件什么事。

传统炒股需要投资者长时间守在电脑前，紧盯市场动态和指标变化。即便如此，投资者也很难保证每次都盈利。原因主要有两点：第一，人的主观判断可能导致决策不准确；第二，个人的精力、时间和计算能力有限，无法全面分析海量的历史数据和众多股票。

量化交易的目标同样是赚钱，但方法不同。它不依赖人类自己思考具体的赚钱策略，而是将这个任务交给计算机。例如，我们可以让计算机基于历史数据，找出在特定时间段内（如两年）能实现收益最大化的买卖策略。

究其本质，量化交易是基于历史数据进行分析，这个过程常被称为“回测”。回测是指在历史数据中分析价值信息，并设计策略以最大化收益。简而言之，量化交易就是让计算机在历史数据中执行数据挖掘任务，找出能赚钱的有效策略。

## 量化交易的核心技能
了解了量化交易的目标后，我们来看看实现它需要哪些核心技能。量化交易是一个交叉学科，需要综合运用多种知识。

以下是入门量化交易需要关注的核心技能点：

![](img/1ec23849aec60632050ab0ed224da236_7.png)

*   **机器学习算法**：用于基于数据预测未来走势或结果。例如，可以使用线性回归模型 `y = β₀ + β₁x₁ + ... + βₙxₙ + ε` 来预测股价。
*   **统计方法**：用于计算和分析数据中的各项指标，理解数据背后的信息。
*   **编程实践（如Python）**：用于获取数据、实现策略和进行回测。这是将想法付诸实践的必要工具。

在量化交易中，金融学知识只需了解即可，重点应放在与计算机相关的数据处理和分析能力上。量化交易的核心可以看作是数据挖掘在金融领域的一个热门应用，关键在于如何**处理数据、操纵数据，并让数据产生价值**。

## 总结
本节课中我们一起学习了量化交易的基本概念。我们明确了量化交易是利用计算机和数据分析方法，基于历史数据（回测）来制定投资策略，以追求收益最大化的过程。其核心技能不在于深奥的金融理论，而在于机器学习、统计学和编程等数据处理能力。掌握这些技能，是步入量化交易领域的关键一步。
