## 正态分布与3-Sigma原理 📈

上一节我们介绍了去极值的概念，本节中我们来看看基于统计学的3-Sigma方法。它的核心原理基于正态分布。

正态分布，也称为高斯分布，是一种常见的概率分布。如果理解起来有困难，可以将其想象为一种“正常”的分布。例如，在银行贷款场景中，银行多借或少借你少量金额是正常情况，但多借或少借你巨额款项则属于异常情况。

在标准正态分布图中，中心点代表均值（通常为0）。数据点离均值越近，其出现的概率越高；离均值越远，出现的概率则越低。我们的目标是识别并处理那些远离均值的“极值”或“离群点”。

**3-Sigma** 正是基于这一思想。它通过计算数据的均值（μ）和标准差（σ），设定一个以均值为中心的区间。区间的上下限由以下公式决定：
*   **上限** = μ + n * σ
*   **下限** = μ - n * σ

![](img/1376ada0568649ffb95c85e54850519b_1.png)

当 `n=3` 时，即“3-Sigma”，理论上该区间将包含约99.7%的数据，而落在区间外的0.3%数据则被视为极值进行处理。

![](img/1376ada0568649ffb95c85e54850519b_1.png)

## 3-Sigma方法的代码实现 💻

理解了原理后，我们开始动手实现3-Sigma方法。我们将编写一个函数，输入一列数据和Sigma的倍数n，输出处理后的数据上下限。

以下是实现步骤：

![](img/1376ada0568649ffb95c85e54850519b_3.png)

首先，我们需要计算数据的两个核心统计量：均值和标准差。

![](img/1376ada0568649ffb95c85e54850519b_5.png)

```python
# 计算数据的均值 (mean) 和标准差 (std)
mean = data.mean()
std = data.std()
```

![](img/1376ada0568649ffb95c85e54850519b_7.png)

接着，根据公式计算上限和下限。

![](img/1376ada0568649ffb95c85e54850519b_9.png)

```python
# 根据公式计算上限和下限
upper_limit = mean + n * std
lower_limit = mean - n * std
```

![](img/1376ada0568649ffb95c85e54850519b_11.png)

最后，我们将计算逻辑封装成一个完整的函数。

![](img/1376ada0568649ffb95c85e54850519b_13.png)

```python
def three_sigma(data, n=3):
    """
    使用3-Sigma方法计算数据的上下限。
    参数:
        data: 输入数据，Pandas Series格式。
        n: Sigma的倍数，默认为3。
    返回:
        lower_limit: 计算出的下限。
        upper_limit: 计算出的上限。
    """
    mean = data.mean()
    std = data.std()
    upper_limit = mean + n * std
    lower_limit = mean - n * std
    return lower_limit, upper_limit
```

## 核心要点总结 ✅

本节课中我们一起学习了3-Sigma去极值方法。

![](img/1376ada0568649ffb95c85e54850519b_39.png)

![](img/1376ada0568649ffb95c85e54850519b_41.png)

我们来回顾一下核心要点：**去极值**是数据预处理的关键一步，其目的并非简单丢弃异常数据，而是通过定义合理的**下界**和**上界**，将所有数值规范到我们设定的边界之内。

![](img/1376ada0568649ffb95c85e54850519b_43.png)

3-Sigma方法是实现这一目标的经典统计方法之一。它基于数据服从正态分布的假设，通过均值μ和标准差σ来动态确定边界，公式为 **上限 = μ + nσ** 与 **下限 = μ - nσ**。当n=3时，可保留绝大多数（99.7%）的“正常”数据。

![](img/1376ada0568649ffb95c85e54850519b_45.png)

结合之前学习的方法，我们已经掌握了多种去极值的策略，这是构建稳健量化模型的第一步。
