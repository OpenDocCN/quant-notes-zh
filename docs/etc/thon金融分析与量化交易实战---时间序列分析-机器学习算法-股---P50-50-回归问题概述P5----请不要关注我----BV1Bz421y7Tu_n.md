# 人生苦短，我学量化！：P50：回归问题概述

在本节课中，我们将要学习机器学习中的核心问题之一：回归问题。我们将了解什么是回归问题，它与分类问题的区别，并通过一个具体的银行贷款额度预测例子，来理解回归问题的数据构成、目标以及核心的建模思想。

## 什么是回归问题？

在机器学习的有监督学习中，主要存在两类问题：分类问题和回归问题。

分类问题旨在预测样本所属的类别。例如，你去银行申请贷款，银行最终会给出一个“批准”或“拒绝”的明确结果。这通常被编码为1（批准）或0（拒绝），是一个典型的二分类问题。

那么，什么是回归问题呢？回归问题旨在预测一个具体的**连续数值**。同样是去银行贷款，这次我们不问“能不能贷”，而是问“能贷多少钱”。这个答案是一个具体的数字，例如3万、5万或8万。这个数字通常在一个连续的取值范围内。预测这样一个具体数值的问题，就称为回归问题。本节课我们重点讨论的就是回归问题。

## 回归问题的数据构成

为了进行回归建模，我们需要有结构化的数据。以下是一个预测贷款额度的数据示例：

| 工资 (X1) | 年龄 (X2) | 额度 (Y) |
| :--- | :--- | :--- |
| ... | ... | ... |

在这个表格中：
*   **X1（工资）** 和 **X2（年龄）** 被称为**特征**。它们是我们在建模时需要参考的指标。
*   **Y（额度）** 被称为**标签**或**目标值**。它是我们模型最终想要预测的结果。

作为有监督学习算法，回归模型在训练时必须要有标签Y作为参考目标。我们的核心任务就是：利用已知的特征X1和X2，去预测未知的标签Y。

## 建立回归模型

既然Y的值与X1和X2都相关，我们很自然地会想到用一个方程来描述它们之间的关系。一个最简单的想法是直接将它们相加：

`Y = X1 + X2`

![](img/4f5ba2be7e75867e0218c4a51007016f_1.png)

但这个方程隐含了一个假设：工资和年龄对贷款额度的影响程度完全相同（系数都是1）。这显然不符合常识，通常工资对额度的影响会比年龄更大。

因此，我们需要为每个特征引入一个**系数**（或称为**权重**），来量化它们对结果的影响程度。于是，我们的方程变为：

`Y = C1 * X1 + C2 * X2`

在这个方程中：
*   **C1** 是工资特征X1的系数。
*   **C2** 是年龄特征X2的系数。

![](img/4f5ba2be7e75867e0218c4a51007016f_3.png)

例如，如果 `C1 = 3`，`C2 = 1`，就意味着工资的影响是年龄影响的3倍。

现在，在建模时，我们的数据中X1、X2和对应的真实Y值都是已知的。那么，回归建模的核心目标就变成了：**寻找一组最合适的系数C1和C2**，使得根据这个方程计算出的预测值，能够最好地反映真实情况。

## 线性模型的局限与目标

上一节我们介绍了回归方程的基本形式，本节中我们来看看如何寻找“最合适”的系数。

我们面临一个现实：用一个线性的平面（方程 `Y = C1*X1 + C2*X2` 在三维空间中是一个平面）去拟合实际的数据点，几乎不可能让平面穿过所有的点。



既然无法完美地穿过每一个数据点，那么我们的目标就应该调整为：**找到这样一个平面，使得它到所有数据点的“总体距离”最小**。换句话说，这个平面应该尽可能贴近大部分的数据点，从而实现对数据整体趋势的最佳拟合。



## 总结

![](img/4f5ba2be7e75867e0218c4a51007016f_5.png)

本节课中我们一起学习了回归问题的基本概念。我们首先区分了回归问题（预测连续值）与分类问题（预测离散类别）。然后，通过银行贷款额度的例子，我们了解了回归问题中特征(X)和标签(Y)的构成。最后，我们引出了线性回归模型的基本形式 `Y = C1*X1 + C2*X2`，并明确了建模的核心目标是求解最优的系数C1和C2，使得模型能够最佳地拟合数据点。在接下来的课程中，我们将深入探讨如何具体求解这些系数。
