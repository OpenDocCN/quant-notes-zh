# 量化交易完全可自学教程：P78：26. 3-Sigma方法实例

在本节课中，我们将要学习一种在量化交易中常用的数据预处理方法——3-Sigma（三西格玛）方法。该方法基于统计学原理，用于识别和处理数据中的异常值（极值），确保后续分析的稳健性。

## 原理介绍 📊

上一节我们介绍了去极值的概念，本节中我们来看看基于正态分布理论的3-Sigma方法。

其核心思想基于正态分布（也称高斯分布）。我们可以将数据的分布想象成一个“钟形曲线”。在正态分布中，大多数数据点会聚集在平均值（均值）附近，离均值越远的数据点出现的概率越低。

![](img/6c17858e57643d0fdd52ff5350a19d20_1.png)

为了便于理解，可以举一个例子：你去银行贷款，银行批准给你的额度比预期多或少一点（比如几百元）是常见情况，这可以看作是分布在均值附近的“正常”数据。但如果批准额度比预期多或少了几十万元，这就属于罕见情况，很可能是一个“异常值”或“极值”。

在正态分布中，数据偏离均值的程度可以用标准差（Sigma, σ）来衡量。3-Sigma方法规定了一个范围：**均值 ± 3倍标准差**。根据统计学，约有99.7%的数据会落在这个范围内。因此，我们可以认为落在此范围外的数据属于概率极低的异常点，需要对它们进行处理（规范到边界上），而不是直接删除。

以下是3-Sigma范围的示意图与数据覆盖率：
*   **均值 ± 1σ**：包含约68.3%的数据。
*   **均值 ± 2σ**：包含约95.4%的数据。
*   **均值 ± 3σ**：包含约99.7%的数据。

我们的目标是保留绝大多数（如99.7%）的“正常”数据，而对剩余的0.3%的“极值”进行规范化处理。

![](img/6c17858e57643d0fdd52ff5350a19d20_3.png)

## 代码实现 💻

![](img/6c17858e57643d0fdd52ff5350a19d20_5.png)

理解了原理后，我们来看看如何用Python代码实现3-Sigma方法。我们将构建一个函数，它接收一列数据和一个倍数`n`（例如n=3），然后返回处理后的数据以及计算出的上下边界。

![](img/6c17858e57643d0fdd52ff5350a19d20_7.png)

以下是实现步骤：

![](img/6c17858e57643d0fdd52ff5350a19d20_9.png)

1.  **计算均值和标准差**：这是确定边界的两个核心统计量。
    *   均值公式：`mean = data.mean()`
    *   标准差公式：`std = data.std()`
2.  **计算上下边界**：
    *   上界公式：`upper = mean + n * std`
    *   下界公式：`lower = mean - n * std`
3.  **规范化数据**：将所有大于上界的值设置为上界，所有小于下界的值设置为下界，介于之间的值保持不变。

![](img/6c17858e57643d0fdd52ff5350a19d20_11.png)

对应的Python函数代码如下：

![](img/6c17858e57643d0fdd52ff5350a19d20_13.png)

![](img/6c17858e57643d0fdd52ff5350a19d20_15.png)

```python
def three_sigma(data_series, n=3):
    """
    使用3-Sigma方法处理数据极值。
    参数:
        data_series: pandas Series, 待处理的数据列。
        n: int, 西格玛倍数，默认为3。
    返回:
        series: 处理后的数据。
        lower: 计算出的下界。
        upper: 计算出的上界。
    """
    mean = data_series.mean()  # 计算均值
    std = data_series.std()   # 计算标准差
    
    upper = mean + n * std    # 计算上界
    lower = mean - n * std    # 计算下界
    
    # 将超出边界的数据规范到边界上
    series = data_series.clip(lower=lower, upper=upper)
    
    return series, lower, upper
```

现在，我们可以使用这个函数来处理实际数据。通过调整参数`n`，我们可以控制被视为“正常”数据的范围大小。例如，设置`n=1`，我们将保留约68.3%的核心数据，对更外围的数据进行规范。

![](img/6c17858e57643d0fdd52ff5350a19d20_25.png)

以下是不同`n`值可能产生的效果：
*   当`n=3`时，边界非常宽，几乎保留了所有原始数据（99.7%），只对极端异常值进行处理。
*   当`n=2`或`n=1`时，边界收窄，会对更多偏离中心的数据进行规范，使得处理后的数据更加“收紧”。

## 核心要点总结 ✅

![](img/6c17858e57643d0fdd52ff5350a19d20_35.png)

本节课中我们一起学习了3-Sigma去极值方法。我们来回顾一下关键点：

![](img/6c17858e57643d0fdd52ff5350a19d20_37.png)

![](img/6c17858e57643d0fdd52ff5350a19d20_39.png)

1.  **原理基础**：该方法基于数据服从或近似服从**正态分布**的假设，利用**均值（mean）**和**标准差（std）**来确定数据的正常范围。
2.  **核心公式**：上下边界由 **`均值 ± n * 标准差`** 计算得出，其中`n`常取3。
3.  **处理逻辑**：去极值**不是删除数据**，而是通过设定上下界，将超出界限的异常值**规范（Clip）**到边界上，保证所有数据都落在预设的合理区间内。
4.  **代码实现**：使用Pandas的`.mean()`、`.std()`和`.clip()`方法可以简洁高效地实现该功能。

![](img/6c17858e57643d0fdd52ff5350a19d20_41.png)

这是数据预处理“三步走”中第一步“去极值”的又一种实用策略。掌握它，能帮助你在量化分析中更好地清洗数据，为后续的建模打下坚实基础。
