# 金融数据分析：P5：1-打分法选股策略概述

在本节课中，我们将学习一种在量化选股中常用的策略——因子打分选股法。我们将了解其核心思想、实施前提，并最终明确如何通过综合多个因子的评分来筛选出表现优异的股票。

## 策略核心思想

上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看如何综合多个因子进行更全面的评估。

因子打分选股的核心思想是模拟学生考试的总分排名。假设我们有一个股票池（例如300只股票），每只股票都有多个因子指标（例如A、B、C、D四个因子）。我们的目标不是单独看某个因子，而是为每只股票计算一个基于所有因子的“综合得分”。

这个过程可以概括为：
1.  为每只股票的每个因子进行评分。
2.  将所有因子的分数相加，得到该股票的总分。
3.  根据总分对所有股票进行排名。
4.  选择排名最高的前N只股票（如前10名）作为投资组合。

在下一次调仓时，我们重复此过程，重新计算并选择新的排名靠前的股票。这种方法旨在通过多维度综合评估，选出整体“素质”最优的股票组合。

## 策略实施前提

![](img/397e287d1d066cdb01676557a94f3472_1.png)

在开始为因子打分之前，我们必须明确一个关键的先验知识：每个因子与股票未来收益率的相关性方向。

以下是理解这一前提的关键点：

*   **相关性方向**：并非所有因子都是数值越大越好。有些因子（如每股收益、净资产收益率）通常与收益率呈**正相关**，即因子值越大，预期收益可能越高。有些因子（如市盈率、市净率）则可能呈**负相关**，即因子值越小，预期收益可能越高。
*   **知识来源**：这个判断可以来源于：
    *   **金融研究报告**：各大券商的研究报告中常包含对因子有效性的分析和经验总结。
    *   **历史数据分析**：我们之前学习的因子分析（如计算IC值）方法，可以帮助我们通过历史数据验证因子的有效性方向。

这个先验知识是打分的基础，因为它决定了我们打分的规则：对于正相关因子，数值越大，得分应越高；对于负相关因子，数值越小，得分应越高。
