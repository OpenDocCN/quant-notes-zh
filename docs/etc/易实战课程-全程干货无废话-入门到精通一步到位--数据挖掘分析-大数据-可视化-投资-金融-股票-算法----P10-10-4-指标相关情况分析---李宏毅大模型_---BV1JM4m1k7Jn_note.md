## 第三步：绘制散点图矩阵深入分析

除了折线图，散点图是观察两个变量关系的另一种有效方式。在绘制散点图前，我们通常使用收益率数据（而非原始价格）进行分析，以消除趋势影响并专注于波动关系。

以下是计算每日简单收益率并绘制散点图矩阵的步骤：

![](img/f80b7d47952a718c584ba90486e88287_63.png)

![](img/f80b7d47952a718c584ba90486e88287_65.png)

首先，计算收益率：
```python
# 计算每日简单收益率 (今日价格/昨日价格 - 1)
returns = data / data.shift(1) - 1
# 删除包含NaN值的第一行
returns = returns.dropna()
```

接下来，我们使用Pandas的`scatter_matrix`函数来绘制散点图矩阵。这个矩阵图能同时展示单变量的分布（对角线）和两两变量之间的散点关系。

以下是绘制散点图矩阵的代码：

![](img/f80b7d47952a718c584ba90486e88287_67.png)

![](img/f80b7d47952a718c584ba90486e88287_69.png)

```python
from pandas.plotting import scatter_matrix

![](img/f80b7d47952a718c584ba90486e88287_71.png)

# 绘制散点图矩阵，对角线显示核密度估计图
scatter_matrix(returns,
               alpha=0.2,        # 设置点的透明度，避免重叠
               diagonal='kde',   # 对角线图形类型：'kde'为核密度估计，'hist'为直方图
               figsize=(10, 10))
```

![](img/f80b7d47952a718c584ba90486e88287_73.png)

![](img/f80b7d47952a718c584ba90486e88287_75.png)

![](img/f80b7d47952a718c584ba90486e88287_77.png)

执行代码后，我们会得到一个2x2的矩阵图：
*   **对角线**：展示了每个指标（SPX收益率和VIX收益率）自身的分布情况。我们这里使用了核密度估计（KDE），它通过平滑的曲线来近似数据的概率分布，比直方图更连续美观。
*   **非对角线**：展示了两个指标收益率之间的散点图。从散点图中，我们可以更直观地看到点云呈现向下倾斜的趋势，这再次印证了SPX收益率与VIX收益率之间存在负相关关系。

通过散点图矩阵，我们可以同时观察变量的分布和变量间的关系。至于选择直方图还是核密度估计图作为对角线图形，取决于个人偏好，选择看起来更清晰易懂的即可。
