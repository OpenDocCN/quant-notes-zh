## 打分方法概述

在获得每个股票的因子数值后，我们需要根据每个因子的好坏方向（例如，越大越好或越小越好）为其打分。以下是两种常见的打分思路。

### 方法一：区间划分法

此方法将每个因子的数值范围划分为若干区间，并为每个区间赋予一个分数。

首先，我们需要明确每个因子的“好坏”方向。假设我们有四个因子 A、B、C、D：
*   **A因子**：越大越好。
*   **B因子**：越大越好。
*   **C因子**：越小越好。
*   **D因子**：越小越好。

![](img/917de9b92879e23fe8497a40df0f82d5_5.png)

接下来，为每个因子划分数值区间并设定分数。以数值范围在 0 到 1 之间为例（实际中可使用百分位数）：

![](img/917de9b92879e23fe8497a40df0f82d5_7.png)

以下是针对不同方向的打分规则：

![](img/917de9b92879e23fe8497a40df0f82d5_9.png)

**对于“越大越好”的因子（如A、B）：**
| 数值区间 | 得分 |
| :--- | :--- |
| [0.8, 1.0] | 5分 |
| [0.6, 0.8) | 4分 |
| [0.4, 0.6) | 3分 |
| [0.2, 0.4) | 2分 |
| [0.0, 0.2) | 1分 |

![](img/917de9b92879e23fe8497a40df0f82d5_11.png)

**对于“越小越好”的因子（如C、D）：**
| 数值区间 | 得分 |
| :--- | :--- |
| [0.0, 0.2) | 5分 |
| [0.2, 0.4) | 4分 |
| [0.4, 0.6) | 3分 |
| [0.6, 0.8) | 2分 |
| [0.8, 1.0] | 1分 |

### 方法二：直接排序法

![](img/917de9b92879e23fe8497a40df0f82d5_13.png)

![](img/917de9b92879e23fe8497a40df0f82d5_15.png)

此方法更直接，无需划分区间。以300只股票为例：
*   对于“越大越好”的因子：将所有股票按该因子值**从大到小**排序，排名第1的得300分，排名第2的得299分，以此类推，排名第300的得1分。
*   对于“越小越好”的因子：将所有股票按该因子值**从小到大**排序，排名第1的得300分，排名第2的得299分，以此类推。

![](img/917de9b92879e23fe8497a40df0f82d5_17.png)

两种方法的目标一致：将不同量纲和方向的因子，转化为统一的、可加总的分数。

![](img/917de9b92879e23fe8497a40df0f82d5_19.png)

## 打分计算示例

我们通过一个具体例子来演示区间划分法的计算过程。假设有两只股票 `id1` 和 `id2`，它们的因子归一化数值如下：

| 股票ID | A因子 | B因子 | C因子 | D因子 |
| :--- | :--- | :--- | :--- | :--- |
| id1 | 0.71 | 0.28 | 0.39 | 0.11 |
| id2 | 0.45 | 0.17 | 0.81 | 0.02 |

![](img/917de9b92879e23fe8497a40df0f82d5_21.png)

根据上述打分规则，我们为每只股票的每个因子计算得分：

以下是 `id1` 的得分计算：
*   **A因子 (0.71)**：越大越好，落在区间 [0.6, 0.8)，得 **4分**。
*   **B因子 (0.28)**：越大越好，落在区间 [0.2, 0.4)，得 **2分**。
*   **C因子 (0.39)**：越小越好，落在区间 [0.2, 0.4)，得 **4分**。
*   **D因子 (0.11)**：越小越好，落在区间 [0.0, 0.2)，得 **5分**。
*   **总分**：4 + 2 + 4 + 5 = **15分**。

![](img/917de9b92879e23fe8497a40df0f82d5_23.png)

以下是 `id2` 的得分计算：
*   **A因子 (0.45)**：越大越好，落在区间 [0.4, 0.6)，得 **3分**。
*   **B因子 (0.17)**：越大越好，落在区间 [0.0, 0.2)，得 **1分**。
*   **C因子 (0.81)**：越小越好，落在区间 [0.8, 1.0]，得 **1分**。
*   **D因子 (0.02)**：越小越好，落在区间 [0.0, 0.2)，得 **5分**。
*   **总分**：3 + 1 + 1 + 5 = **10分**。

![](img/917de9b92879e23fe8497a40df0f82d5_25.png)

计算完成后，`id1` 得15分，`id2` 得10分。若策略选择总分最高的前N只股票，则 `id1` 会被优先选中。

![](img/917de9b92879e23fe8497a40df0f82d5_27.png)

## 整体策略流程梳理 🔄

基于打分法构建一个完整的月度调仓策略，其流程可以概括为以下几个步骤：

![](img/917de9b92879e23fe8497a40df0f82d5_37.png)

1.  **确定股票池**：首先设定选股范围，例如沪深300指数成分股。
    ```python
    # 示例：设定初始股票池为沪深300成分股
    universe = ‘沪深300’
    ```

2.  **设置调仓周期**：定义策略再平衡（调仓）的频率，例如每月第一个交易日执行。
    ```python
    # 示例：每月调仓一次
    rebalance_frequency = ‘monthly’
    ```

3.  **实现再平衡函数**：这是策略的核心，在每次调仓日被调用。其内部逻辑如下：
    *   **数据获取**：获取当前股票池内所有股票的所需因子数据。
    *   **因子打分**：应用上述打分方法（区间法或排序法），为每只股票的每个因子计算得分。
    *   **汇总总分**：将每只股票的所有因子得分相加，得到该股票的综合总分。
        ```python
        # 示例：计算总分
        total_score = score_A + score_B + score_C + score_D
        ```
    *   **排序与选股**：将所有股票按综合总分**从高到低**排序，选取排名前10的股票作为新的投资组合。

4.  **执行交易**：卖出不在新组合中的股票，买入新加入组合的股票，完成仓位调整。
