在本节课中，我们将要学习如何计算和解读投资策略中的年化收益率。上一节我们介绍了回测收益率，它反映了策略在整个回测周期内的总盈亏情况。本节中我们来看看如何将总收益率转化为更标准化的年化收益率，以便于在不同时间跨度的策略之间进行比较。

## 概述

![](img/e03588f4da21c189928697b0f93f7474_4.png)

回测收益率是评估策略表现的基础指标，它显示了从开始到结束的总收益。然而，当策略运行时间不同时，直接比较总收益率并不公平。年化收益率通过将总收益折算到一年期的标准时间框架下，解决了这个问题，是金融分析中更常用和关键的评估指标。

![](img/e03588f4da21c189928697b0f93f7474_6.png)

![](img/e03588f4da21c189928697b0f93f7474_8.png)

## 从回测收益率到年化收益率

![](img/e03588f4da21c189928697b0f93f7474_10.png)

回测收益率计算的是策略在整个运行期间的总收益。如下图所示，不同策略的回测收益率有正有负，这直观地体现了“投资有风险”的原则。

![](img/e03588f4da21c189928697b0f93f7474_12.png)

![](img/e03588f4da21c189928697b0f93f7474_0.png)

![](img/e03588f4da21c189928697b0f93f7474_14.png)

理解回测收益率后，我们需要一个更通用的指标来进行跨周期比较。这就是年化收益率。它回答了“如果这个策略运行一年，平均能获得多少收益？”的问题。

![](img/e03588f4da21c189928697b0f93f7474_16.png)

![](img/e03588f4da21c189928697b0f93f7474_18.png)

年化收益率的计算基于回测收益率，并考虑了策略运行的总天数。其核心公式如下：

**年化收益率 = (1 + 总收益率) ^ (250 / 总交易日数) - 1**

在这个公式中：
*   **总收益率** 即我们之前计算的回测收益率。
*   **250** 代表一年中大约的平均交易日数量（也可使用252天）。
*   **总交易日数** 是你的回测数据所覆盖的总天数。

![](img/e03588f4da21c189928697b0f93f7474_20.png)

![](img/e03588f4da21c189928697b0f93f7474_22.png)

## 计算过程演示

![](img/e03588f4da21c189928697b0f93f7474_24.png)

以下我们通过代码来演示如何根据回测数据计算年化收益率。

![](img/e03588f4da21c189928697b0f93f7474_26.png)

首先，我们需要获取策略的回测总收益率和总交易日数。假设我们已经有了一个包含回测结果的DataFrame。

![](img/e03588f4da21c189928697b0f93f7474_28.png)

```python
# 假设 df_results 是一个DataFrame，其中‘总收益率’列是计算好的回测收益率
# 假设总交易日数 N 为 730 天
N = 730
M = 250  # 年化交易日基数

![](img/e03588f4da21c189928697b0f93f7474_30.png)

# 计算年化收益率
df_results[‘年化收益率’] = (1 + df_results[‘总收益率’]) ** (M / N) - 1
```

![](img/e03588f4da21c189928697b0f93f7474_32.png)

执行上述计算后，我们可以得到每个策略对应的年化收益率。与回测收益率相比，年化收益率的值通常会有所不同，因为它将收益压缩到年度尺度上进行衡量。

![](img/e03588f4da21c189928697b0f93f7474_34.png)

![](img/e03588f4da21c189928697b0f93f7474_40.png)

![](img/e03588f4da21c189928697b0f93f7474_36.png)

![](img/e03588f4da21c189928697b0f93f7474_38.png)

如图所示，一个在730天（约3年）内获得1.13倍总收益的策略，其年化收益率约为0.2几（即20%+）。原本亏损的策略，其年化收益率也是一个较小的负值。这使得不同运行时间的策略表现具备了可比性。

## 核心指标总结

![](img/e03588f4da21c189928697b0f93f7474_42.png)

在评估一个量化策略时，有两个至关重要的收益指标：

![](img/e03588f4da21c189928697b0f93f7474_44.png)

以下是需要重点观察的两个收益指标：

![](img/e03588f4da21c189928697b0f93f7474_46.png)

![](img/e03588f4da21c189928697b0f93f7474_48.png)

1.  **回测收益率**：反映策略在**整个测试周期内**的绝对总收益。
2.  **年化收益率**：将总收益标准化到**一年期**的收益率，用于**横向比较**不同时间跨度策略的盈利能力。

![](img/e03588f4da21c189928697b0f93f7474_50.png)

## 本节课总结

![](img/e03588f4da21c189928697b0f93f7474_52.png)

本节课中我们一起学习了年化收益率的概念与计算方法。我们明白了为何在回测收益率之外还需要年化收益率——它为评估不同周期长度的策略提供了统一的尺度。通过一个简单的幂运算公式，我们可以将总收益转化为年化收益，这是量化金融分析中评估策略绩效的核心步骤之一。
