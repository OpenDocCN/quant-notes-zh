## 定位功能 🎯

![](img/337376f0456f7cda8093521645995d3d_7.png)

上一节我们介绍了Excel的基础界面，本节中我们来看看如何高效地定位和处理特定数据。

![](img/337376f0456f7cda8093521645995d3d_9.png)

定位功能（快捷键 `Ctrl + G`）能帮助我们快速找到并操作特定类型的单元格，例如空值或可见单元格。

### 案例一：批量填充空值为零

![](img/337376f0456f7cda8093521645995d3d_17.png)

当表格中存在大量空单元格需要统一填充为“0”时，手动操作效率低下。

![](img/337376f0456f7cda8093521645995d3d_19.png)

![](img/337376f0456f7cda8093521645995d3d_21.png)

以下是操作步骤：
1.  在一个单元格中输入“0”并复制（`Ctrl + C`）。
2.  选中需要处理的数据区域。
3.  按下 `Ctrl + G` 打开“定位”对话框，点击“定位条件”。
4.  在“定位条件”中选择“空值”，点击“确定”。此时所有空单元格会被选中。
5.  **注意**：选中后鼠标不要点击其他位置，直接按 `Ctrl + V` 粘贴，即可将所有空值填充为“0”。

### 案例二：对分组数据快速求和

![](img/337376f0456f7cda8093521645995d3d_31.png)

![](img/337376f0456f7cda8093521645995d3d_32.png)

当需要对不连续的分组数据进行求和时，定位功能也能派上用场。

以下是操作步骤：
1.  选中包含空值（用于放置求和结果）的数据区域。
2.  按下 `Ctrl + G`，选择“定位条件”中的“空值”。
3.  点击“确定”后，所有作为求和结果位的空单元格被选中。
4.  此时，可以直接点击工具栏的“自动求和”按钮（或按快捷键 `Alt + =`），Excel会自动为每个分组计算总和。

### 案例三：安全删除筛选后的数据

![](img/337376f0456f7cda8093521645995d3d_34.png)

直接删除筛选后的行可能导致隐藏的数据被误删。使用定位功能可以避免这个问题。

以下是操作步骤：
1.  对数据进行筛选。
2.  选中需要删除的可见行。
3.  按下 `Ctrl + G`，在“定位条件”中选择“可见单元格”。
4.  点击“确定”后，再进行删除操作，这样就只会删除选中的可见行，隐藏数据不受影响。

![](img/337376f0456f7cda8093521645995d3d_36.png)

## 选择性粘贴 🔄

![](img/337376f0456f7cda8093521645995d3d_38.png)

学会了精准定位，接下来我们看看如何更智能地进行粘贴操作。选择性粘贴允许我们在粘贴时选择特定的格式或运算方式。

### 案例一：为所有工资统一增加金额

![](img/337376f0456f7cda8093521645995d3d_40.png)

![](img/337376f0456f7cda8093521645995d3d_42.png)

假设需要为所有员工的“实发工资”增加2000元。

以下是操作步骤：
1.  在一个单元格中输入“2000”并复制（`Ctrl + C`）。
2.  选中“实发工资”列的数据区域。
3.  右键点击，选择“选择性粘贴”。
4.  在对话框的“运算”区域选择“加”，点击“确定”。所有选中的数值将自动加上2000。

### 案例二：将公式结果转换为静态值

当表格中使用了公式，但需要将最终结果发送给他人时，最好将公式转换为静态值，避免因引用单元格变化或删除导致错误。

![](img/337376f0456f7cda8093521645995d3d_44.png)

![](img/337376f0456f7cda8093521645995d3d_45.png)

![](img/337376f0456f7cda8093521645995d3d_47.png)

以下是操作步骤：
1.  选中包含公式的单元格区域并复制（`Ctrl + C`）。
2.  右键点击目标位置，在“选择性粘贴”选项中选择“值”（或快捷键 `Ctrl + Alt + V`，然后选择 `V`）。
3.  这样粘贴的内容就是计算结果本身，不再是公式。

![](img/337376f0456f7cda8093521645995d3d_49.png)

### 案例三：数据转置

![](img/337376f0456f7cda8093521645995d3d_51.png)

选择性粘贴还可以快速实现行与列数据的转换。

![](img/337376f0456f7cda8093521645995d3d_53.png)

![](img/337376f0456f7cda8093521645995d3d_54.png)

以下是操作步骤：
1.  复制需要转换的数据区域。
2.  右键点击目标起始单元格，选择“选择性粘贴”。
3.  勾选对话框底部的“转置”选项，点击“确定”。数据便会从行排列转为列排列，或反之。

![](img/337376f0456f7cda8093521645995d3d_56.png)

## 快速填充 ⚡

![](img/337376f0456f7cda8093521645995d3d_58.png)

处理序列数据时，快速填充能节省大量时间。Excel可以识别模式并自动填充序列。

![](img/337376f0456f7cda8093521645995d3d_60.png)

### 自动填充序列

![](img/337376f0456f7cda8093521645995d3d_62.png)

对于数字、日期或自定义列表，Excel可以自动填充。

![](img/337376f0456f7cda8093521645995d3d_64.png)

*   **连续数字**：输入“1”，拖动单元格右下角的填充柄，可填充1,2,3...
*   **等差数列**：输入“2”和“4”两个单元格，同时选中它们再拖动填充柄，可填充2,4,6,8...
*   **日期**：输入一个日期后拖动填充柄，日期会自动递增。

### 自定义填充列表

Excel允许用户定义自己的填充序列，例如“张、王、李、赵”。

以下是设置自定义列表的步骤：
1.  点击“文件” -> “选项”。
2.  在“高级”选项卡中，滚动到最下方，点击“编辑自定义列表”。
3.  在“输入序列”框中，逐行输入你的序列内容（如：张、王、李、赵），点击“添加”。
4.  之后，在单元格中输入“张”，向下拖动填充柄，即可自动填充“王、李、赵...”。

## 查找与替换 🔍

在大型数据表中，查找与替换是必不可少的工具。

![](img/337376f0456f7cda8093521645995d3d_70.png)

*   **查找**：快捷键 `Ctrl + F`。可以查找特定值，也可以点击“选项”进行更精确的查找，如按格式查找或使用通配符。
    *   `?` 代表一个任意字符。例如查找“李??”可以找到所有姓李的三字姓名。
    *   `*` 代表零个或多个任意字符。例如查找“*师”可以找到所有以“师”结尾的文本。
*   **替换**：快捷键 `Ctrl + H`。可以将查找到的内容替换为新内容。**重要**：在进行批量替换前，建议先“查找全部”确认目标，或先选中特定区域以避免误操作。

![](img/337376f0456f7cda8093521645995d3d_72.png)

## 分列与合并 ✂️

![](img/337376f0456f7cda8093521645995d3d_74.png)

数据整理中，经常需要将一列数据拆分成多列，或将多列数据合并为一列。

![](img/337376f0456f7cda8093521645995d3d_76.png)

### 分列

将“甲,喜欢,Excel”这样的单元格按逗号分隔成三列。

以下是操作步骤：
1.  选中该列数据。
2.  点击“数据”选项卡中的“分列”。
3.  选择“分隔符号”，点击“下一步”。
4.  勾选“逗号”作为分隔符，在数据预览中查看效果，点击“下一步”。
5.  选择每列的数据格式，并设置目标区域，点击“完成”。

### 合并

![](img/337376f0456f7cda8093521645995d3d_78.png)

将分开的三列“甲”、“喜欢”、“Excel”合并成一列。

![](img/337376f0456f7cda8093521645995d3d_80.png)

![](img/337376f0456f7cda8093521645995d3d_82.png)

可以使用连字符 `&` 进行连接：
```excel
=A1 & B1 & C1
```
或者更简洁地使用函数：
```excel
=CONCATENATE(A1, B1, C1)
```
在Excel 2013及以上版本，有一个更智能的功能：**快速填充（`Ctrl + E`）**。只需在第一行手动输入合并后的目标效果（如“甲喜欢Excel”），然后选中该列，按下 `Ctrl + E`，Excel会自动识别模式并填充整列。

## 去除重复项 🗑️

![](img/337376f0456f7cda8093521645995d3d_84.png)

![](img/337376f0456f7cda8093521645995d3d_85.png)

![](img/337376f0456f7cda8093521645995d3d_86.png)

数据分析中，经常需要删除重复的数据记录。

![](img/337376f0456f7cda8093521645995d3d_88.png)

以下是操作步骤：
1.  选中需要去重的数据区域。
2.  点击“数据”选项卡中的“删除重复项”。
3.  在弹出的对话框中，选择依据哪些列来判断重复。例如，如果“姓名”和“日期”两列都相同才算重复，就只勾选这两列。
4.  点击“确定”，Excel会删除重复行并保留唯一值。

![](img/337376f0456f7cda8093521645995d3d_90.png)

## 日期时间规范 📅

![](img/337376f0456f7cda8093521645995d3d_92.png)

从不同系统导出的日期格式往往不统一（如“2023.1.1”、“20230101”），需要规范为标准日期格式（如“2023/1/1”）。

![](img/337376f0456f7cda8093521645995d3d_94.png)

### 方法一：查找替换
对于“2023.1.1”这类格式，可以使用查找替换（`Ctrl + H`），将所有的英文句点 `.` 替换为斜杠 `/`。

![](img/337376f0456f7cda8093521645995d3d_96.png)

![](img/337376f0456f7cda8093521645995d3d_97.png)

![](img/337376f0456f7cda8093521645995d3d_98.png)

![](img/337376f0456f7cda8093521645995d3d_99.png)

![](img/337376f0456f7cda8093521645995d3d_100.png)

### 方法二：分列功能
分列功能是统一日期格式的强大工具。

![](img/337376f0456f7cda8093521645995d3d_102.png)

![](img/337376f0456f7cda8093521645995d3d_103.png)

以下是操作步骤：
1.  选中日期列。
2.  点击“数据”->“分列”。
3.  前两步直接点击“下一步”。
4.  在第三步“列数据格式”中，选择“日期”，并指定原数据的格式（如YMD），点击“完成”。Excel会自动将各种文本日期转换为标准日期格式。

![](img/337376f0456f7cda8093521645995d3d_105.png)

### 方法三：公式与格式
对于“980102”这种缺少前两位年份的文本，可以结合分列和公式处理。先用分列按固定宽度拆分成“98”、“01”、“02”三列，然后用公式 `="19"&A1&"/"&B1&"/"&C1` 拼接，最后再将结果通过分列转换为日期格式。
