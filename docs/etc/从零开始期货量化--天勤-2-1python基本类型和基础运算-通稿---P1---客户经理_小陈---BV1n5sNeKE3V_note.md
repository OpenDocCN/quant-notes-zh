## 代码注释

注释是Python中不会被运行的文字，其作用是对代码进行解释或备注。

以下是注释的几种方式：
*   **单行注释**：在行首使用 `#` 符号。
*   **行尾注释**：在代码行末尾使用 `#` 进行说明。
*   **多行注释（文档字符串）**：使用三个双引号 `"""` 或三个单引号 `'''` 将多行文字包裹起来，通常用于模块、函数或类的开头说明。

在PyCharm中，可以使用快捷键 `Ctrl + /` 快速添加或取消（单行/选中的多行）注释。

注释的格式规范建议：`#` 与代码之间保留两个空格，`#` 与注释文字之间保留一个空格。这能使代码更整洁美观。格式不规范时，编辑器可能会显示浅色波浪线提示，但不影响代码运行。

## print函数

`print()` 是Python中重要的输出函数，用于在控制台显示信息。

其基本用法是向括号内传入要输出的内容，多个内容用逗号分隔。例如：
```python
print(1)          # 输出数字 1
print(1, 2)       # 输出 1 2
print(1, 2, 3)    # 输出 1 2 3
print(‘佟掌柜很帅’) # 输出字符串
print()           # 输出一个空行
```
代码执行顺序是逐行进行的。调整 `print` 语句的顺序，输出结果的顺序也会相应改变。

## 代码格式：空行与空格

在Python代码中：
*   **空行**：没有语法含义，仅用于分隔代码块，使结构更清晰美观。
*   **空格**：在某些位置是语法要求（如运算符两侧），在某些位置则影响代码可读性。关键语法元素（如 `print`）不能无故被空格断开，否则会导致错误。字符串内部的前导空格会被保留并输出。

## 基本数据类型

Python为不同类型的数据定义了不同的“类型”，以便高效地处理和运算。

### 数字类型
数字类型主要分为两种：
*   **整数（int）**：例如 `79`。
*   **浮点数（float，即小数）**：例如 `926.5`。书写时，小数点前的 `0` 可以省略（如 `.0256` 等同于 `0.0256`）。负数在数字前加 `-` 号。

使用 `type()` 函数可以查看数据的类型。
```python
future_number = 79          # int 类型
i_price = 926.5            # float 类型
change_rate = .0256        # float 类型，等同于 0.0256
print(type(future_number)) # 输出：<class ‘int’>
```

### 字符串类型
字符串（str）是由引号包裹的一串字符，用于表示文本。
*   可以使用单引号 `‘ ‘` 或双引号 `“ “` 定义，两者在Python中没有区别。
*   字符串会“所见即所得”地输出引号内的所有内容。

```python
future_name = ‘RB2401‘      # 字符串 ‘RB2401‘
symbol_name = “螺纹钢“       # 字符串 “螺纹钢“
print(type(future_name))    # 输出：<class ‘str’>
```

### 布尔类型
布尔值（bool）只有两个：`True`（真）和 `False`（假）。**首字母必须大写**。它主要用于逻辑判断。
```python
is_rising = True
print(type(is_rising)) # 输出：<class ‘bool’>
```

### 空值类型
空值（NoneType）只有一个值：`None`，表示“没有值”。**首字母必须大写**。
```python
empty_value = None
print(type(empty_value)) # 输出：<class ‘NoneType’>
```

## 变量

变量用于存储数据，是程序中数据的名字。通过给变量赋值，我们可以在程序各处方便地引用和修改这个数据，而无需改动每一处使用它的代码。

变量命名需简明扼要，方便识别。规则是：**只能以字母或下划线 `_` 开头**，后面可以跟字母、数字或下划线。不能以数字开头。
```python
future_name = ‘RB2401‘  # 正确：字母开头
_price = 100           # 正确：下划线开头
2name = ‘test‘         # 错误：数字开头，会报错
```

## 基础运算

上一节我们介绍了如何用变量存储数据，本节我们来看看如何对这些数据进行运算。

### 算术运算
Python支持基本的算术运算。
```python
# 加法
old_futures = 79
new_futures = 5
total_futures = old_futures + new_futures # 结果为 84

# 取余数
remainder = 7 % 3  # 结果为 1

# 乘方
result = 2 ** 10   # 2的10次方，结果为 1024

# 复利计算示例：每日收益1%，365天后的总收益
daily_return = 1.01
total_return = daily_return ** 365  # 结果约为 37.78
```

### 简便写法
Python提供一些运算符的简便写法。
```python
future_number = 79
future_number += 1  # 等同于 future_number = future_number + 1，结果变为 80
```

### 类型间运算
*   `int` 和 `float` 可以直接运算，结果为 `float`。
*   `str` 和 `int` 使用 `*` 运算，表示重复字符串。
*   `str` 和 `int` 使用 `+` 运算会报错，因为类型不匹配。

```python
print(66 + 8.8)        # 输出：74.8
print(‘刘关张‘ * 3)     # 输出：刘关张刘关张刘关张
# print(‘刘关张‘ + 3)   # 错误！会报错
```

### 比较运算与逻辑运算
这些运算主要用于条件判断，在后续策略逻辑中至关重要。

以下是常见的比较运算符：
*   `>`：大于
*   `>=`：大于等于
*   `==`：等于
*   `!=`：不等于

比较运算的结果是布尔值（`True` 或 `False`）。
```python
num1 = 1
num2 = 2
print(num1 > num2)   # False
print(num1 != num2)  # True
```

逻辑运算用于组合多个条件：
*   `and`：与，两者都为真时结果为真。
*   `or`：或，两者中有一个为真时结果为真。

```python
condition1 = (2 > 1)      # True
condition2 = (2 == 1)     # False
print(condition1 and condition2)  # False
print(condition1 or condition2)   # True
```
