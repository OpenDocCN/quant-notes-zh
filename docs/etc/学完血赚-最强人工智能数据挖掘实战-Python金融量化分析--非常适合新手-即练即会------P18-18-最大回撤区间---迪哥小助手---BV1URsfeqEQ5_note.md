## 什么是最大回撤？🤔

上一节我们介绍了策略的整体收益，本节中我们来看看如何衡量策略的风险。最大回撤描述的是在投资过程中，资产净值从最高点到之后最低点的**最大跌幅**。它衡量的是投资者可能经历的“最惨”时期，即资产缩水最严重的连续区间。

其核心公式如下：

**最大回撤 = max( (P_i - P_j) / P_i )**

其中：
*   **P_i** 代表第 i 天的资产价格（或净值）。
*   **P_j** 代表第 i 天之后某一天（第 j 天）的资产价格。
*   我们需要遍历所有可能的 i 和 j（j > i），找出比值最大的那个，即为最大回撤率。

这个指标不能仅从字面理解，它本质上是一个**风险指标**。即使一个策略最终是盈利的，但如果在过程中出现了巨大的回撤，也可能因为心理压力或资金链问题而无法坚持到盈利的时刻。例如，在2008年全球金融危机中，股市从高点大幅下跌，许多投资者承受了巨大的账面亏损，这正是最大回撤所刻画的“惨劲儿”。

## 如何计算最大回撤？🧮

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_6.png)

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_7.png)

理解了概念后，我们来看看如何用Python进行计算。计算最大回撤需要找出净值曲线中“高点”和“后续低点”的最大落差。我们可以借助Pandas库中的 `cummax()` 函数来高效完成。

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_9.png)

`cummax()` 函数用于计算序列的**累积最大值**。即对于序列中的每一个位置，它都返回从序列开始到该位置为止的最大值。

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_11.png)

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_13.png)

以下是计算最大回撤的核心步骤：

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_15.png)

1.  **计算累积最大值序列**：对每日的净值序列应用 `cummax()`，得到每个交易日及其之前的历史最高净值。
2.  **计算每日回撤**：用“历史最高净值”减去“当日实际净值”，得到每日的潜在亏损额。
3.  **计算每日回撤率**：将每日的潜在亏损额除以对应的“历史最高净值”，得到每日的回撤率。
4.  **找出最大值**：在所有每日回撤率中，最大值就是整个考察期内的**最大回撤率**。

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_17.png)

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_18.png)

以下是具体的代码示例：

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_20.png)

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_21.png)

```python
import pandas as pd

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_23.png)

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_24.png)

# 假设 df[‘net_value’] 是包含每日资产净值的序列
# 1. 计算累积最大值
historical_max = df[‘net_value’].cummax()

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_26.png)

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_27.png)

# 2. 计算每日回撤（下跌幅度）
drawdown = historical_max - df[‘net_value’]

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_29.png)

# 3. 计算每日回撤率
drawdown_ratio = drawdown / historical_max

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_30.png)

# 4. 找出最大回撤率
max_drawdown_ratio = drawdown_ratio.max()
print(f“最大回撤率为：{max_drawdown_ratio:.2%}”)
```

![](img/bc4b3f9fb373c73180cb44f1e3a34a14_32.png)

通过以上代码，我们就可以量化策略的历史最大风险。在实际的策略回测报告中，最大回撤通常会与收益指标一同展示，如下图所示，图表中会明确标出最大回撤发生的区间。
