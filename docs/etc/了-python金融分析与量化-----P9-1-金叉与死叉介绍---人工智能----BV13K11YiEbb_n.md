## 双均线策略概述

![](img/b3cfc27e969a2a5116905e3c3d292d55_5.png)

上一节我们介绍了交易数据分析的基本背景，本节中我们来看看如何利用均线来捕捉市场趋势。双均线策略的核心在于使用两条不同周期的移动平均线来判断买卖时机。

![](img/b3cfc27e969a2a5116905e3c3d292d55_7.png)

### 什么是双均线

![](img/b3cfc27e969a2a5116905e3c3d292d55_9.png)

![](img/b3cfc27e969a2a5116905e3c3d292d55_11.png)

在股票数据中，我们每天都能获得一个收盘价。如果仅观察每日价格的波动，很难把握整体的趋势变化。为了更清晰地描述价格趋势，我们引入移动平均线的概念。

![](img/b3cfc27e969a2a5116905e3c3d292d55_13.png)

双均线指的是两条移动平均线：一条是短期均线，另一条是长期均线。

![](img/b3cfc27e969a2a5116905e3c3d292d55_15.png)

### 短期均线与长期均线

![](img/b3cfc27e969a2a5116905e3c3d292d55_17.png)

![](img/b3cfc27e969a2a5116905e3c3d292d55_19.png)

以下是短期均线与长期均线的定义与区别：

![](img/b3cfc27e969a2a5116905e3c3d292d55_21.png)

*   **短期均线**：计算周期较短，例如5天或10天的收盘价平均值。它反映了价格的近期波动趋势，对价格变化反应灵敏。
    *   **公式**：`SMA_short = (P1 + P2 + ... + Pn) / n`，其中 `n` 为短周期（如5）。
*   **长期均线**：计算周期较长，例如20天、40天或60天的收盘价平均值。它反映了价格的中长期趋势，走势更为平缓。
    *   **公式**：`SMA_long = (P1 + P2 + ... + Pm) / m`，其中 `m` 为长周期（如20）。

![](img/b3cfc27e969a2a5116905e3c3d292d55_23.png)

两者的唯一区别在于计算平均值时所选取的“时间窗口”大小。短期均线窗口小，更能紧跟价格；长期均线窗口大，趋势更平滑。

![](img/b3cfc27e969a2a5116905e3c3d292d55_25.png)

![](img/b3cfc27e969a2a5116905e3c3d292d55_27.png)

## 金叉与死叉：核心交易信号

![](img/b3cfc27e969a2a5116905e3c3d292d55_29.png)

在绘制出短期均线和长期均线后，它们的相对位置和交叉关系就构成了关键的交易信号。

![](img/b3cfc27e969a2a5116905e3c3d292d55_31.png)

### 黄金交叉（金叉）—— 买入信号

![](img/b3cfc27e969a2a5116905e3c3d292d55_33.png)

![](img/b3cfc27e969a2a5116905e3c3d292d55_35.png)

当短期均线从下方向上穿越长期均线，并且整体趋势向上时，这个交叉点被称为“黄金交叉”（金叉）。

![](img/b3cfc27e969a2a5116905e3c3d292d55_37.png)

*   **含义**：短期趋势开始强于长期趋势，预示着价格可能进入上涨通道。
*   **操作建议**：被视为一个潜在的**买点**，可以考虑买入或加仓。

![](img/b3cfc27e969a2a5116905e3c3d292d55_39.png)

### 死亡交叉（死叉）—— 卖出信号

当短期均线从上方向下穿越长期均线，并且整体趋势向下时，这个交叉点被称为“死亡交叉”（死叉）。

*   **含义**：短期趋势开始弱于长期趋势，预示着价格可能进入下跌通道。
*   **操作建议**：被视为一个潜在的**卖点**，可以考虑卖出或减仓。

## 策略逻辑与应用

![](img/b3cfc27e969a2a5116905e3c3d292d55_41.png)

现在我们已经理解了金叉（买点）和死叉（卖点）的含义。对于一个金融时间序列数据，如果我们能准确识别出这些交叉点，就可以设计一个简单的交易策略：

![](img/b3cfc27e969a2a5116905e3c3d292d55_43.png)

**策略逻辑**：在出现黄金交叉时买入，在出现死亡交叉时卖出。

理论上，如果每次都能正确预测趋势转折点，并严格执行“低买高卖”，就能累积收益。这正是双均线策略的核心思想。在后续的课程中，我们将使用Python来实际计算均线、识别金叉与死叉，并构建这个策略进行回测。
