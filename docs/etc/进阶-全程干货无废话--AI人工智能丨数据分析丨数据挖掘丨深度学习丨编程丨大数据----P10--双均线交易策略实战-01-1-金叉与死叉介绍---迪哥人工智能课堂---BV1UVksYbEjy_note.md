## 双均线策略简介
上一节我们介绍了量化交易的基本概念，本节中我们来看看一个具体的策略应用。在交易数据分析中，双均线策略是一种常用的技术分析方法。

![](img/bf4582973939f95a5347b30605928bd6_9.png)

![](img/bf4582973939f95a5347b30605928bd6_11.png)

所谓“双均线”，指的是在股价走势图上同时绘制两条移动平均线。一条是短期移动平均线，另一条是长期移动平均线。通过观察这两条线的相对位置和交叉情况，我们可以对市场趋势做出判断。

![](img/bf4582973939f95a5347b30605928bd6_13.png)

![](img/bf4582973939f95a5347b30605928bd6_15.png)

![](img/bf4582973939f95a5347b30605928bd6_17.png)

## 为何需要移动平均线？
直接观察股票每日的收盘价，我们只能看到价格在单个时间点上的波动。这种个体数据的变化虽然直观，但难以从整体上清晰地描述价格的中长期趋势。

![](img/bf4582973939f95a5347b30605928bd6_19.png)

为了解决这个问题，我们引入移动平均线的概念。它的核心思想是：**通过计算一段时间内价格的平均值，来平滑短期波动，从而更清晰地展现趋势**。

![](img/bf4582973939f95a5347b30605928bd6_21.png)

![](img/bf4582973939f95a5347b30605928bd6_23.png)

## 短均线与长均线
以下是双均线策略中两个核心组成部分的定义：

![](img/bf4582973939f95a5347b30605928bd6_25.png)

![](img/bf4582973939f95a5347b30605928bd6_27.png)

![](img/bf4582973939f95a5347b30605928bd6_29.png)

*   **短均线**：计算周期较短的移动平均线。例如，我们可以计算过去5个交易日（代表一周）收盘价的平均值。其公式可以表示为：
    `短均线值 = (P1 + P2 + P3 + P4 + P5) / 5`
    其中 `P1` 到 `P5` 代表最近5个交易日的收盘价。短均线对价格变化反应灵敏，能较好地反映短期趋势。

![](img/bf4582973939f95a5347b30605928bd6_31.png)

![](img/bf4582973939f95a5347b30605928bd6_33.png)

![](img/bf4582973939f95a5347b30605928bd6_35.png)

*   **长均线**：计算周期较长的移动平均线。常见的周期有20日（约一个月）、60日（约一个季度）等。其计算方式与短均线类似，只是取平均的窗口更大。例如20日均线：
    `长均线值 = (P1 + P2 + ... + P20) / 20`
    长均线变化平缓，能更好地过滤市场噪音，反映长期趋势。

短均线与长均线唯一的区别在于计算平均值时所选取的**时间窗口大小**。将这两条线绘制在同一张图上，它们的相对运动就能为我们提供交易信号。

![](img/bf4582973939f95a5347b30605928bd6_37.png)

![](img/bf4582973939f95a5347b30605928bd6_39.png)

## 核心信号：金叉与死叉
当短均线和长均线在图表上发生交叉时，就产生了重要的交易信号。主要有以下两种情况：

![](img/bf4582973939f95a5347b30605928bd6_41.png)

![](img/bf4582973939f95a5347b30605928bd6_43.png)

1.  **黄金交叉（金叉） - 买入信号**
    当短均线从下方**向上**穿越长均线时，形成黄金交叉。这通常意味着短期市场动能开始转强，股价可能开启一波上涨趋势。因此，金叉被视为一个潜在的**买入点**。

![](img/bf4582973939f95a5347b30605928bd6_45.png)

2.  **死亡交叉（死叉） - 卖出信号**
    当短均线从上方**向下**穿越长均线时，形成死亡交叉。这通常意味着短期趋势转弱，股价可能进入下跌阶段。因此，死叉被视为一个潜在的**卖出点**。

![](img/bf4582973939f95a5347b30605928bd6_47.png)

![](img/bf4582973939f95a5347b30605928bd6_49.png)

## 策略逻辑与应用
现在我们已经理解了金叉（买点）和死叉（卖点）。对于一个股票价格序列，如果我们能通过程序自动识别出这些交叉点，就可以设计一个简单的交易策略：

![](img/bf4582973939f95a5347b30605928bd6_51.png)

*   在出现**金叉**时买入股票。
*   在出现**死叉**时卖出股票。

![](img/bf4582973939f95a5347b30605928bd6_53.png)

![](img/bf4582973939f95a5347b30605928bd6_55.png)

理想情况下，如果每次信号都预测准确，通过“低买高卖”的循环，理论上可以实现持续盈利。这个基于双均线交叉进行买卖决策的策略，就是我们所说的**双均线交易策略**。

![](img/bf4582973939f95a5347b30605928bd6_57.png)

在接下来的课程中，我们将使用Python来实际演示如何从数据中计算移动平均线，并自动识别金叉和死叉，从而将这个策略付诸实践。

## 总结
本节课中我们一起学习了双均线策略的基础知识。我们了解了移动平均线如何帮助平滑价格、揭示趋势，并重点掌握了**金叉**和**死叉**这两个核心交易信号的含义。金叉代表短线上穿长线的买入时机，而死叉代表短线下穿长线的卖出时机。理解这一原理是构建我们第一个量化交易策略的基石。
