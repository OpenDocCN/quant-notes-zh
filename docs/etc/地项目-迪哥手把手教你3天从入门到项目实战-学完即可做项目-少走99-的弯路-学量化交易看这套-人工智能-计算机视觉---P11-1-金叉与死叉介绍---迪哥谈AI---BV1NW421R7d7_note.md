## 双均线策略的基本概念

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_1.png)

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_2.png)

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_3.png)

上一节我们介绍了量化交易的基础，本节中我们来看看如何利用均线来分析市场趋势。首先，我们需要理解什么是双均线。

在股票交易数据中，我们每天都能获得一个收盘价。如果只看每一天的单个价格，我们很难把握其整体的变化趋势。为了解决这个问题，我们引入“均线”的概念。均线是对过去一段时间内收盘价进行平均计算后得到的一条平滑曲线，它能更好地反映价格的整体趋势。

双均线策略，顾名思义，就是使用两条不同周期的均线。通常，一条是周期较短的“短期均线”，另一条是周期较长的“长期均线”。

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_5.png)

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_6.png)

## 短期均线与长期均线

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_7.png)

以下是短期均线与长期均线的定义与区别：

*   **短期均线**：计算周期较短，例如5天或10天。它反应了价格的近期波动趋势，对价格变化更为敏感。
    *   **公式**：`短期均线值 = 过去N个交易日收盘价之和 / N` （例如 N=5）
*   **长期均线**：计算周期较长，例如20天、40天或60天。它反应了价格的中长期趋势，走势更为平缓。
    *   **公式**：`长期均线值 = 过去M个交易日收盘价之和 / M` （例如 M=20）

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_9.png)

两者的核心区别在于计算时所选取的“时间窗口”大小。短期窗口小，对变化反应快；长期窗口大，趋势更稳定。

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_11.png)

## 金叉与死叉：核心买卖信号

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_13.png)

在绘制出短期和长期两条均线后，它们的交叉点就构成了关键的交易信号。主要有两种交叉形态：

1.  **黄金交叉（金叉）**：当短期均线从下方向上穿越长期均线时，形成金叉。这通常被视为**买入信号**。
    *   **含义**：短期趋势开始走强并超越长期趋势，预示着价格可能开启一波上涨行情。
    *   **操作**：此时可以考虑买入或加仓。

2.  **死亡交叉（死叉）**：当短期均线从上方向下穿越长期均线时，形成死叉。这通常被视为**卖出信号**。
    *   **含义**：短期趋势开始走弱并跌破了长期趋势，预示着价格可能进入下跌阶段。
    *   **操作**：此时可以考虑卖出或减仓。

## 策略逻辑与应用

![](img/0e83ccf4f3c6258d9a3f3537d3cbc5aa_15.png)

理解了买点（金叉）和卖点（死叉）后，我们就可以构建一个简单的交易策略逻辑：

**策略逻辑**：当出现黄金交叉时，执行买入操作；当出现死亡交叉时，执行卖出操作。

如果每次信号都能准确预测后续走势，理论上可以通过“低买高卖”持续获得收益。这就是双均线交叉策略的核心思想。在后续的实战中，我们将使用Python来具体实现这一策略，计算均线、识别交叉点并模拟交易。
