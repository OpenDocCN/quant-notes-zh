# 量化交易教程：P31：3Sigma方法实例 📊

在本节课中，我们将要学习一种在量化交易中常用的数据预处理方法——3Sigma方法。该方法基于统计学原理，用于识别和处理数据中的极端值（离群点），以确保后续分析的稳定性和可靠性。

上一节我们介绍了数据预处理的重要性，本节中我们来看看如何利用正态分布的特性来定义数据的正常范围。

## 3Sigma方法原理 📈

3Sigma方法的原理非常简单。大家先来看下面这张正态分布图。

![](img/7ae60c9e7266d30f44a260cfb6666b6e3_1.png)

图中展示了一个标准的正态分布（也称为高斯分布）。如果理解起来比较麻烦，可以将其想象为一种“正常”的分布模式。例如，你去银行贷款，银行多借或少借你几块钱是正常的，但如果多借或少借你十万八万，那就不正常了。

![](img/7ae60c9e7266d30f44a260cfb666b6e3_1.png)

在正态分布中，均值（μ）是数据的中心点。数据点距离均值越近，其出现的概率越高；距离均值越远，出现的概率则越低。我们的目标是去除“极值”，而极值通常不会落在均值附近的高概率区域，而是落在远离均值的低概率区域。

**3Sigma** 正是基于这一思想。它通过计算均值（μ）和标准差（σ），来确定一个数据的“正常”范围。

![](img/7ae60c9e7266d30f44a260cfb6666b6e3_3.png)

在上图中，`X` 代表均值。我们定义了以均值为中心的多个区间：
*   **μ ± 1σ**
*   **μ ± 2σ**
*   **μ ± 3σ**

区间范围越大，数据点落在这个区间之外的可能性就越小。**3Sigma方法** 就是指，我们假设数据经过某种变换后服从正态分布，然后取 **μ ± 3σ** 作为正常数据的边界。

![](img/7ae60c9e7266d30f44a260cfb666b6e3_3.png)

根据统计学知识，在正态分布中：
*   **μ ± 1σ** 包含了约 **68.3%** 的数据。
*   **μ ± 2σ** 包含了约 **95.4%** 的数据。
*   **μ ± 3σ** 包含了约 **99.7%** 的数据。

![](img/7ae60c9e7266d30f44a260cfb666b6e3_5.png)

因此，如果我们认为数据中 **99.7%** 的部分是正常的，剩下的 **0.3%** 是极值并对其进行规范化处理，这种方法就叫做 **3Sigma方法**。我们只需要计算出数据的**均值**和**标准差**，就能确定这个范围的上限和下限。

![](img/7ae60c9e7266d30f44a260cfb666b6e3_7.png)

## 代码实现 💻

![](img/7ae60c9e7266d30f44a260cfb666b6e3_9.png)

以下是3Sigma方法的代码实现步骤，非常简单。

首先，我们定义一个函数，其逻辑与之前的方法类似，但计算边界的方式不同。

![](img/7ae60c9e7266d30f44a260cfb666b6e3_11.png)

![](img/7ae60c9e7266d30f44a260cfb666b6e3_13.png)

```python
def three_sigma(data_series, n=3):
    """
    使用3Sigma方法处理极值。
    参数:
        data_series: 待处理的数据序列 (pandas Series)。
        n: 西格玛倍数，默认为3。
    返回:
        series: 处理后的数据序列。
        lower_limit: 计算出的下限。
        upper_limit: 计算出的上限。
    """
    # 计算均值和标准差
    mean = data_series.mean()
    std = data_series.std()
    
    # 根据均值和标准差计算上下限
    upper_limit = mean + n * std
    lower_limit = mean - n * std
    
    # 将超出范围的值规范到边界上
    series_capped = data_series.clip(lower=lower_limit, upper=upper_limit)
    
    return series_capped, lower_limit, upper_limit
```

代码解释如下：
1.  `mean = data_series.mean()`: 计算输入数据序列的**均值（μ）**。
2.  `std = data_series.std()`: 计算输入数据序列的**标准差（σ）**。
3.  `upper_limit = mean + n * std`: 根据公式 **μ + nσ** 计算上限。
4.  `lower_limit = mean - n * std`: 根据公式 **μ - nσ** 计算下限。
5.  `series_capped = data_series.clip(...)`: 使用Pandas的`clip`函数，将所有小于下限的值设置为下限，所有大于上限的值设置为上限，从而实现“去极值”而非“丢弃”。

# 假设 df[‘column_name‘] 是我们要处理的数据列

## 核心要点总结 ✅

![](img/7ae60c9e7266d30f44a260cfb666b6e3_39.png)

![](img/7ae60c9e7266d30f44a260cfb666b6e3_41.png)

本节课中我们一起学习了3Sigma去极值方法。

![](img/7ae60c9e7266d30f44a260cfb666b6e3_43.png)

主要需要记住的是，数据预处理的第一步“去极值”，**其核心不是简单地将异常值丢弃，而是对其进行规范化**。具体做法是：
1.  根据选定的方法（如3Sigma）定义一个合理的**下界**和**上界**。
2.  让所有的数据值都规范在这个定义好的边界之内。

![](img/7ae60c9e7266d30f44a260cfb666b6e3_45.png)

这就是我们“三步走”数据预处理策略中的第一步。掌握它，能为后续的金融分析与建模打下坚实的基础。
