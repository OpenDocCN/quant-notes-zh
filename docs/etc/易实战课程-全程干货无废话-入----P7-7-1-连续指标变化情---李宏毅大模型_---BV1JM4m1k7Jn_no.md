## 概述：增长率累加的问题

上一节我们介绍了如何计算简单的差异值和增长率。本节中我们来看看，当我们希望对一个连续时间序列计算其累积增长率时，会遇到什么问题。

直接对增长率进行累加操作是不可行的。以下是一个例子来说明这个问题：

![](img/d3f2d262e7cb876a43041d7c4f9458b8_4.png)

*   假设一只股票第一天的价格是100元。
*   第二天价格跌至50元。第二天的增长率为：`(50 - 100) / 100 = -0.5`。
*   第三天价格涨至75元。第三天的增长率为：`(75 - 50) / 50 = 0.5`。
*   如果将这两天的增长率累加：`-0.5 + 0.5 = 0`。结果为零，似乎股票没有涨跌。
*   然而，观察原始数据：从第一天的100元到第三天的75元，股票实际上是下跌的。

因此，增长率不能直接相加来反映从起点到终点的整体变化。

## 解决方案：引入对数计算

对于连续时间的计算，我们需要借助数学中的对数公式。核心思想是计算每日价格比的对数，然后进行累加。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_6.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_8.png)

其原理基于对数的性质：对数的加法等价于其内部数值的乘法。公式如下：

```
ln(P2/P1) + ln(P3/P2) = ln( (P2/P1) * (P3/P2) ) = ln(P3/P1)
```

![](img/d3f2d262e7cb876a43041d7c4f9458b8_10.png)

其中，`ln` 表示以自然常数e为底的对数，`P1`、`P2`、`P3` 分别代表第1、2、3天的价格。

这个公式的意义在于，将各期对数收益率累加后，得到的结果直接反映了最终价格与初始价格的比例关系。形象地说，如果你初始投入1元，经过一系列涨跌，这个累加值经过指数还原后，就能告诉你最终这1元变成了多少钱。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_12.png)

## 实践操作：计算连续变化

![](img/d3f2d262e7cb876a43041d7c4f9458b8_14.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_16.png)

以下是计算连续对数收益率及累积价值的具体步骤。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_20.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_22.png)

### 1. 计算每日价格比

![](img/d3f2d262e7cb876a43041d7c4f9458b8_24.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_26.png)

首先，我们需要计算后一天价格与前一天价格的比值。在Pandas中，可以使用 `.shift(1)` 方法将数据整体向上移动一格，从而方便地获取“前一天”的数据。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_28.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_30.png)

```python
# 假设 data 是包含股价的Pandas Series或DataFrame
price_ratio = data / data.shift(1)
```

![](img/d3f2d262e7cb876a43041d7c4f9458b8_32.png)

### 2. 计算对数收益率

![](img/d3f2d262e7cb876a43041d7c4f9458b8_34.png)

接着，我们对价格比取自然对数，得到每日的对数收益率。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_36.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_38.png)

```python
log_return = np.log(price_ratio)
```

### 3. 计算累积对数收益率

![](img/d3f2d262e7cb876a43041d7c4f9458b8_40.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_42.png)

然后，我们对对数收益率进行累积求和。这可以使用NumPy的 `cumsum()` 函数实现。这个累积值表示从起始日到当前日的总对数收益率。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_44.png)

```python
cumulative_log_return = log_return.cumsum()
```

![](img/d3f2d262e7cb876a43041d7c4f9458b8_46.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_48.png)

### 4. 还原为实际价值

![](img/d3f2d262e7cb876a43041d7c4f9458b8_50.png)

最后，为了得到“初始1元相当于当前多少钱”这个直观指标，我们需要将对数累积值通过指数函数还原。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_52.png)

```python
# 假设初始价值为1单位
cumulative_value = np.exp(cumulative_log_return)
```

![](img/d3f2d262e7cb876a43041d7c4f9458b8_54.png)

### 5. 可视化结果

![](img/d3f2d262e7cb876a43041d7c4f9458b8_56.png)

我们可以将计算出的累积价值绘制成图，直观展示各股票从起始点（标准化为1）到结束点的价值增长轨迹。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_58.png)

```python
cumulative_value.plot(figsize=(16, 8))
```

## 常用操作总结

本节课中我们一起学习了金融时间序列分析中几种核心的计算操作：

![](img/d3f2d262e7cb876a43041d7c4f9458b8_64.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_66.png)

以下是关键操作及其实现方法的总结：

![](img/d3f2d262e7cb876a43041d7c4f9458b8_68.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_70.png)

*   **计算差异值**：使用 `.diff()` 方法，直接得到相邻数据的差值。
*   **计算简单增长率**：公式为 `(后值 - 前值) / 前值`，反映了相邻两期的变化比例。
*   **计算连续（对数）增长率**：这是本节的重点。通过 `np.log(data / data.shift(1))` 计算每日对数收益率，它解决了增长率不能直接累加的问题。
*   **计算累积价值**：通过对数收益率的累积和 `np.cumsum()` 再经指数还原 `np.exp()`，得到初始投资随时间变化的实际价值。这个指标非常直观，例如可以清晰看到2010年投入的1元钱，到2018年相当于多少元。

![](img/d3f2d262e7cb876a43041d7c4f9458b8_72.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_74.png)

![](img/d3f2d262e7cb876a43041d7c4f9458b8_76.png)

通过这些操作，我们能够从不同维度深入分析金融时间序列的变化特征，为量化分析和投资决策打下基础。
