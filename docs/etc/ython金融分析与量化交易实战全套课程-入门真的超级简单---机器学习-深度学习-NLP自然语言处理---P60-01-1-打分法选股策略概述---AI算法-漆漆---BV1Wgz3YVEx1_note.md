# 金融量化分析：P60：01-1-打分法选股策略概述

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分选股法。我们将了解其核心思想、实施前提以及基本流程，为后续的实战操作打下基础。

## 策略核心思想

上一节我们介绍了因子分析，本节中我们来看看如何将多个因子综合起来进行选股。

因子打分选股的核心思想是：**不依赖单一因子，而是综合多个因子对股票进行整体评价**。这类似于学生考试，单科成绩优秀不代表总成绩好，我们需要计算各科成绩的总分来排名。

具体而言，假设我们有300支股票（`stock_1`, `stock_2`, ..., `stock_300`）和四个选股因子（`factor_A`, `factor_B`, `factor_C`, `factor_D`）。我们的目标不是单独看某个因子，而是为每支股票的每个因子打分，然后计算一个**总分**。最后，根据总分对所有股票进行排名，选取排名靠前的股票（如前10名）作为投资组合。

**核心公式**可以表示为：
`股票总分 = f(因子A得分， 因子B得分， 因子C得分， 因子D得分)`
其中，`f`通常为加权求和函数。

## 策略实施前提

在开始为因子打分之前，我们必须明确一个关键的先验知识：**每个因子与股票未来收益率的相关性是正向还是负向**。

![](img/8d894bc9f0c68daf22fa38b68035ef6d_1.png)

这是因为，打分需要依据因子值的大小来判断好坏。例如：
*   **正向因子**：因子值越大，预期收益越高。例如`每股收益(EPS)`、`净资产收益率(ROE)`。对于这类因子，数值越大，打分应越高。
*   **负向因子**：因子值越小，预期收益越高。例如`市盈率(PE)`（在某些估值模型中）。对于这类因子，数值越小，打分应越高。

以下是确定因子方向的两种主要方法：
1.  **查阅研究报告**：各大券商的研究报告会提供经验性的结论。
2.  **自行计算分析**：通过历史数据计算因子与收益率的**信息系数(IC)**等指标进行验证。

在后续的Python策略编写中，我们所选取的因子都将是方向已知的。

## 策略流程总结

本节课中我们一起学习了因子打分选股策略的框架。

其流程可概括为：
1.  **确定因子池**：选择一组有效的选股因子（如估值、成长、动量等因子）。
2.  **明确因子方向**：确定每个因子与预期收益的相关性方向（正向或负向）。
3.  **因子数据标准化**：将不同量纲和范围的因子值处理到同一尺度，以便比较和打分。
4.  **为每个因子打分**：根据标准化后的数值，按照其方向对每支股票的各因子进行评分（例如，百分制或十分制）。
5.  **计算综合得分**：将每支股票在所有因子上的得分进行**加权求和**，得到该股票的综合总分。
    `综合总分 = 权重_A * 得分_A + 权重_B * 得分_B + ...`
6.  **排序与选股**：根据综合总分对所有股票进行降序排名，选取排名最高的N支股票构成投资组合。
7.  **定期调仓**：按固定周期（如每月）重复上述过程，更新投资组合。
