# Python量化交易：P1：使用Python与TA-Lib库计算RSI指标 📈

在本节课中，我们将要学习如何使用Python计算一个在交易中非常常用的技术指标——相对强弱指数（RSI）。我们将从理解RSI的基本概念开始，然后学习如何手动计算它，最后介绍一个强大的第三方库`ta`，它能让我们更简便地获取RSI及其他多种技术指标。

## 什么是RSI指标？🤔

RSI指标是衡量资产买卖意向和实力的工具。它通过分析一段时期内收盘价的涨幅与总波动幅度之间的比值，来判断市场的多空双方力道。通常，RSI值大于80被视为**超买**信号，而小于20则被视为**超卖**信号。

![](img/a48996d9e55e91e6f4345e77293d53c7_1.png)

## RSI的计算方法 🧮

RSI的计算公式基于价格的变化。其核心是计算特定周期内（通常为14天）价格上涨和下跌的平均幅度。

**公式**如下：
`RSI = 100 - 100 / (1 + RS)`
其中，`RS`（相对强度）= 周期内上涨幅度的平均值 / 周期内下跌幅度的平均值。

![](img/a48996d9e55e91e6f4345e77293d53c7_3.png)

上一节我们介绍了RSI的基本概念，本节中我们来看看如何用Python代码实现这个计算。

## 手动计算RSI指标 💻

![](img/a48996d9e55e91e6f4345e77293d53c7_5.png)

以下是手动计算RSI指标的步骤：

1.  获取一系列收盘价数据。
2.  计算每日价格的变化（今日收盘价 - 昨日收盘价）。
3.  将价格变化分离为上涨幅度（正变化）和下跌幅度（负变化的绝对值）。
4.  计算指定周期内（如14天）上涨幅度的平均值和下跌幅度的平均值。
5.  计算相对强度`RS`。
6.  代入公式计算`RSI`值。

让我们通过代码来具体实现这个过程。

![](img/a48996d9e55e91e6f4345e77293d53c7_7.png)

```python
# 假设 close_prices 是一个包含收盘价的列表
close_prices = [ ... ] # 你的收盘价数据

# 计算价格变化
deltas = [close_prices[i] - close_prices[i-1] for i in range(1, len(close_prices))]

![](img/a48996d9e55e91e6f4345e77293d53c7_9.png)

# 分离上涨和下跌
gains = [delta if delta > 0 else 0 for delta in deltas]
losses = [-delta if delta < 0 else 0 for delta in deltas]

# 计算14周期的平均上涨和平均下跌（这里使用简单移动平均）
period = 14
avg_gain = sum(gains[-period:]) / period
avg_loss = sum(losses[-period:]) / period

# 计算RS和RSI
if avg_loss == 0:
    rsi = 100
else:
    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))

![](img/a48996d9e55e91e6f4345e77293d53c7_11.png)

print(f"最新的RSI值为: {rsi:.2f}")
```

![](img/a48996d9e55e91e6f4345e77293d53c7_12.png)

运行这段代码，我们可以得到最新的RSI数值。例如，计算出的最新值可能是41.6，前一根K线的值可能是42.16。通过对比交易所提供的数据，可以验证我们计算结果的准确性。

手动计算虽然有助于理解原理，但在实际量化交易中，我们更倾向于使用成熟的库来高效、准确地获取指标。接下来，我们将介绍一个非常实用的库——`ta`库。

## 使用TA-Lib库简化计算 🚀

`ta`库是一个专门为技术分析设计的Python库，它提供了大量常见技术指标的计算方法，包括RSI。使用它可以极大地简化我们的代码。

以下是使用`ta`库计算RSI的步骤：

1.  确保已安装`ta`库。
2.  导入库并准备收盘价数据。
3.  直接调用`ta.momentum.RSIIndicator`函数。

![](img/a48996d9e55e91e6f4345e77293d53c7_14.png)

让我们看看具体的代码实现。

```python
import pandas as pd
import ta

![](img/a48996d9e55e91e6f4345e77293d53c7_16.png)

# 假设我们有一个包含收盘价的Pandas Series，名为‘close’
# close = pd.Series([ ... ])

# 使用ta库计算14周期RSI
rsi_indicator = ta.momentum.RSIIndicator(close=close, window=14)
rsi_values = rsi_indicator.rsi()

![](img/a48996d9e55e91e6f4345e77293d53c7_18.png)

print("最新的几个RSI值：")
print(rsi_values.tail())
```

运行这段代码，`ta`库会输出一个包含所有计算出的RSI值的序列。例如，最新的值可能在41左右，前一个值在42.05，再前一个在45.69。这个结果与我们手动计算的结果是一致的。

通过对比可以发现，使用`ta`库只需一两行代码就能完成计算，比手动实现要简便和可靠得多。

![](img/a48996d9e55e91e6f4345e77293d53c7_20.png)
