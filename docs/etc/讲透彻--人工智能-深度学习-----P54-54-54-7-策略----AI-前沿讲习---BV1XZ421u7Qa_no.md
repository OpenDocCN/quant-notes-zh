此前，我们的策略是选择综合评分**排名前十**的股票。一个自然的疑问是：如果选择排名靠后的股票，结果会怎样？

![](img/33ccd6aaf3eb9d9c581d3fb223a4f4a7_48.png)

我们将选股逻辑从选择前10名，改为选择第10名以后的所有股票（即排名靠后的股票）。

![](img/33ccd6aaf3eb9d9c581d3fb223a4f4a7_50.png)

```python
# 原逻辑：选择排名前10的股票
selected_stocks = ranked_stocks[:10]

![](img/33ccd6aaf3eb9d9c581d3fb223a4f4a7_52.png)

![](img/33ccd6aaf3eb9d9c581d3fb223a4f4a7_54.png)

# 新逻辑：选择排名第10名以后（即较差）的股票

```

## 总结

本节课中我们一起学习了如何对量化策略进行回测分析与总结。

![](img/33ccd6aaf3eb9d9c581d3fb223a4f4a7_74.png)

1.  **策略验证**：我们首先在短周期内验证了策略代码的正确性，并观察到策略能产生一定的超额收益。
2.  **稳健性检验**：通过延长回测时间，我们发现策略在不同市场阶段（如熊市）表现会有所差异，这是正常现象。
3.  **反向验证**：通过选择评分靠后的股票进行回测，反向证明了我们选股指标的有效性。
4.  **报告解读**：我们学习了如何阅读回测报告中的交易详情、持仓变化和账户信息等关键内容。

![](img/33ccd6aaf3eb9d9c581d3fb223a4f4a7_76.png)

**打分法**是一种在量化交易中常用且直观的策略构建方法。它通过综合多个财务或技术指标对股票进行排序，从而选择出理论上更具潜力的投资组合。虽然无法保证在所有市场环境下都盈利，但它为系统化、纪律化的投资决策提供了一个坚实的框架。大家可以在提供的代码基础上，尝试调整指标权重、选股数量或调仓频率，进一步优化策略。
