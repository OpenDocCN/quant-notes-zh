## 为什么需要转换日期格式？ ⏰

![](img/d014f5f97d2be6805b7ec5bce45084bc_10.png)

在开始分析前，需要明确一点：为什么需要将`date`列转换为时间序列并设置为行索引？

![](img/d014f5f97d2be6805b7ec5bce45084bc_12.png)

因为后续的分析需求都与日期相关。将原始数据中的日期字符串转换为时间序列后，我们就可以利用时间序列特有的属性和方法，更高效地完成基于日期的查询和计算。

## 分析需求：找出上涨超过3%的日期 📊

![](img/d014f5f97d2be6805b7ec5bce45084bc_14.png)

我们的目标是找出该股票（例如茅台）所有收盘价比开盘价上涨超过3%的日期。分析股票的涨幅分布情况，有助于我们了解其价格波动规律。

![](img/d014f5f97d2be6805b7ec5bce45084bc_16.png)

以下是实现此需求的核心步骤：

![](img/d014f5f97d2be6805b7ec5bce45084bc_18.png)

1.  **理解计算逻辑**：收盘价比开盘价上涨3%的数学表达式为：
    `(收盘价 - 开盘价) / 开盘价 > 0.03`
2.  **转换为代码**：在Pandas DataFrame中，`开盘价`对应`df[‘open’]`列，`收盘价`对应`df[‘close’]`列。因此，上述逻辑可以写为：
    `(df[‘close’] - df[‘open’]) / df[‘open’] > 0.03`
3.  **获取布尔序列**：执行上述代码会得到一个布尔值序列（Series）。序列中的`True`表示该行数据（即该日期）满足上涨超过3%的条件，`False`则表示不满足。

![](img/d014f5f97d2be6805b7ec5bce45084bc_20.png)

## 核心技巧：利用布尔值筛选数据 🎯

![](img/d014f5f97d2be6805b7ec5bce45084bc_22.png)

在数据分析中，一个非常实用的经验是：**如果得到了一个布尔值序列，下一步可以立即将其作为原始数据的行索引进行筛选。**

![](img/d014f5f97d2be6805b7ec5bce45084bc_24.png)

这是因为，当将一个布尔值序列放入DataFrame的`.loc[]`索引器中时，Pandas会返回所有对应`True`的行，并自动忽略对应`False`的行。

![](img/d014f5f97d2be6805b7ec5bce45084bc_26.png)

以下是具体操作：

![](img/d014f5f97d2be6805b7ec5bce45084bc_28.png)

![](img/d014f5f97d2be6805b7ec5bce45084bc_30.png)

1.  **应用布尔索引**：将上一步得到的布尔序列`bool_series`，通过`df.loc[bool_series]`来筛选数据。
2.  **获取目标行**：上述操作会返回一个只包含满足条件（即上涨超3%）的日期的行数据的新DataFrame。
3.  **提取日期索引**：由于日期已被设置为行索引，我们只需取出这个新DataFrame的索引，即可得到所有目标日期。代码为：`df.loc[bool_series].index`

![](img/d014f5f97d2be6805b7ec5bce45084bc_32.png)

![](img/d014f5f97d2be6805b7ec5bce45084bc_34.png)

## 完整代码示例 💻

![](img/d014f5f97d2be6805b7ec5bce45084bc_36.png)

将以上步骤整合，实现需求的完整代码如下：

![](img/d014f5f97d2be6805b7ec5bce45084bc_38.png)

```python
# 假设df是已经预处理好的DataFrame，日期已作为索引
# 计算布尔序列：标记收盘价比开盘价上涨超过3%的日期
bool_series = (df[‘close’] - df[‘open’]) / df[‘open’] > 0.03

![](img/d014f5f97d2be6805b7ec5bce45084bc_40.png)

![](img/d014f5f97d2be6805b7ec5bce45084bc_42.png)

# 利用布尔序列筛选出满足条件的行
up_days_df = df.loc[bool_series]

![](img/d014f5f97d2be6805b7ec5bce45084bc_44.png)

# 提取这些行的索引，即我们需要的日期
up_days = up_days_df.index

print(“收盘价较开盘价上涨超过3%的日期有：”)
print(up_days)
```
