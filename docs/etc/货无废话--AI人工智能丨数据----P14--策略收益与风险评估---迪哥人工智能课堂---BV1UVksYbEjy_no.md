回测收益率是指，当你完成一个策略后，最终需要知道策略的盈利情况。例如，你选择了从2015年到2020年的某只股票数据，应用策略决定买卖时机，最终得到一个结果。这个结果表示你赚了多少钱。但需要注意的是，它衡量的不是绝对金额，而是相对于初始本金赚取的百分比。例如，图中显示收益率约为4%，这意味着经过这些年的操作，赚取的资金相对于本金增加了大约4%。

![](img/e34b5f4d5ba27c2435e638970fd2a509_23.png)

下面看一下计算公式。回测收益率的计算公式非常简单。

**公式**：
`回测收益率 = (卖出价格 - 初始价格) / 初始价格`

![](img/e34b5f4d5ba27c2435e638970fd2a509_23.png)

公式中的 `Pt` 代表卖出价格。大体意思是，你最初有一笔资金用于购买股票，持有一段时间后卖出。分子部分表示你赚了多少钱，但注意这个“赚了”是相对于你最初投入的本金而言的。计算出的就是回测收益率。

![](img/e34b5f4d5ba27c2435e638970fd2a509_25.png)

回测收益率是我们在计算和评估策略时最常用的指标，主要用来表示当前策略的整体表现。

![](img/e34b5f4d5ba27c2435e638970fd2a509_27.png)

![](img/e34b5f4d5ba27c2435e638970fd2a509_25.png)

接下来，我们演示如何计算回测收益率。我们可以绘制一张图，展示从2015年到2018年（举例）回测收益率随时间的变化情况。如何计算这个变化呢？其实就是用每个时间点的价格 `Pt` 除以最开始的价格。我们来看具体做法。

![](img/e34b5f4d5ba27c2435e638970fd2a509_27.png)

这里我们不直接做减法，因为公式 `(Pt / P0) - 1` 与 `(Pt - P0) / P0` 是等价的。我们采用前一种方式，最后统一减去1，相当于对公式进行了化简。

第一步是计算每个时间点价格与初始价格的比值。

**代码**：
```python
# 构建一个DataFrame，计算每个时间点价格与初始价格的比值
price_ratio = data['close'] / data['close'].iloc[0]
```

构建一个DataFrame，传入每个时间点的收盘价数据，然后除以初始（索引为0）的收盘价。完成这一步后，我们可以先绘图看一下结果。

**代码**：
```python
import matplotlib.pyplot as plt
plt.figure(figsize=(12, 6))

plt.title('资产相对价值变化 (相对于初始值=1)')
plt.xlabel('日期')
plt.ylabel('相对价值')
plt.grid(True)
plt.show()
```

计算尚未完成，我们还需要计算最终的回测收益率。

**代码**：
```python
# 计算最终的回测收益率：最后一个比值减去1
final_return = price_ratio.iloc[-1] - 1
```

最终的回测收益率等于DataFrame中最后一个比值，再根据公式减去1。
