## 数据清洗概述

![](img/181b4528271b20dd92f0114cdeb67c73_3.png)

数据清洗是使用Pandas进行数据分析时常用的操作。进行数据分析的一个必要前提是，我们需要对原始数据进行某种形式的清洗。

为什么需要清洗呢？因为在我们的原始数据中，可能会存在缺失值。这里的缺失值指的就是空值。在某些常规的指定条件或需求下，原始数据中存在的空值是没有意义的，并且这些空值如果存在，反而会干扰我们整个分析结果的产生。

![](img/181b4528271b20dd92f0114cdeb67c73_5.png)

因此，我们需要将原始数据中的这些缺失值（即空值）进行某种形式的清洗。清洗意味着我们可以将其删除。除了删除，还有其他对应的操作，我们将逐步学习。

除了空值之外，原始数据中还可能产生重复数据（重复值）。原始数据中如果产生重复值，那么重复值是没有必要进行多次分析或处理的。我们可能需要将相关的重复值进行过滤和清洗。

![](img/181b4528271b20dd92f0114cdeb67c73_7.png)

![](img/181b4528271b20dd92f0114cdeb67c73_9.png)

最后一种形式是异常值。由于数据采集的手段不同，采集到的数据中可能会产生某种形式的异常值。异常值也一定会干扰我们最终分析结论的产生，因此我们也需要把异常值进行过滤。

在Pandas数据清洗的过程中，我们需要清洗如下三种形式的值：
1.  缺失值
2.  重复值
3.  异常值

![](img/181b4528271b20dd92f0114cdeb67c73_11.png)

接下来，我们首先来看一下缺失值（即空值）的处理方式。

## 缺失值（空值）的类型与区别

![](img/181b4528271b20dd92f0114cdeb67c73_17.png)

如果产生了空值，那么空值会有两种形式。我们之前在讲解Series算术运算时提到，Series进行算术运算时，只能让索引一致的元素进行运算，否则会补空。那时我们补的空是`NaN`形式的空，而不是之前所见到的`None`形式的空。

在整个数据分析中，我们所见到的、所使用的、所产生的空一定是`NaN`形式的空，而不会是`None`的空。为什么呢？接下来我们就来看一下这两种空的区别。

首先，我们通过`type`函数查看`None`的类型。

![](img/181b4528271b20dd92f0114cdeb67c73_19.png)

```python
type(None)
```

![](img/181b4528271b20dd92f0114cdeb67c73_21.png)

你会发现，`None`的类型是`NoneType`，这意味着它是一个对象类型。

而我们在数据分析中能够用到的`NaN`是怎样的类型呢？首先，我们通过`numpy`模块中的`nan`属性来产生一个`NaN`形式的空。

```python
import numpy as np
np.nan
```

通过`type`函数查看这个`NaN`形式的空是什么类型。

![](img/181b4528271b20dd92f0114cdeb67c73_23.png)

```python
type(np.nan)
```

![](img/181b4528271b20dd92f0114cdeb67c73_25.png)

你会发现，`NaN`形式的空是浮点类型。

![](img/181b4528271b20dd92f0114cdeb67c73_27.png)

![](img/181b4528271b20dd92f0114cdeb67c73_29.png)

从这我们就能看到这两种形式的空的主要区别：`None`是对象类型，而`NaN`是浮点类型。

那么，为什么在数据分析中需要用到浮点类型的空呢？我们来看一个例子。

![](img/181b4528271b20dd92f0114cdeb67c73_31.png)

让对象类型的空`None`加上一个整数1。

![](img/181b4528271b20dd92f0114cdeb67c73_33.png)

![](img/181b4528271b20dd92f0114cdeb67c73_35.png)

```python
None + 1
```

![](img/181b4528271b20dd92f0114cdeb67c73_37.png)

这个算术运算执行后没有结果，会报错，因为类型不一样，对象类型不能加上一个整数类型。

![](img/181b4528271b20dd92f0114cdeb67c73_39.png)

现在，我们让浮点类型的空`NaN`加上一个整数1。

![](img/181b4528271b20dd92f0114cdeb67c73_41.png)

```python
np.nan + 1
```

这个运算可以执行，但返回的结果是`nan`。浮点类型的空加上任意的值，返回的都是`nan`，而不是一个具体的数值（如0）。零和空是不一样的。

![](img/181b4528271b20dd92f0114cdeb67c73_43.png)

因此，在数据分析中，常常会使用某些形式的运算来处理原始数据。如果原始数据中的空值为`NaN`的形式，则不会干扰或中断整个运算流程，因为`NaN`可以参与运算。而`None`是不可以参与运算的。

这是这两种空在本质意义上的区别。也就是说，在数据分析中，如果原始数据中的空是`NaN`的形式，那么在整个运算或处理过程中，即使不进行空值的清洗，也不会中断运算流程。而如果你的空是`None`的话，在运算过程中是会中断整个运算流程的。

![](img/181b4528271b20dd92f0114cdeb67c73_45.png)

所以，在数据分析中，一定要用`NaN`形式的空，而不要用`None`形式的空。

![](img/181b4528271b20dd92f0114cdeb67c73_47.png)

这里有一个小的特性需要知道：在Pandas中，如果遇到了`None`形式的空值，Pandas会将其强制转换成`NaN`的形式。也就是说，如果在Pandas的操作过程中遇到了`None`形式的空也没关系，因为Pandas会自动帮我们做强转。但是，这两种空本质意义上的区别，你需要知道是怎么回事。

## 缺失值的清洗方法

接下来，我们来看一下，如果在使用Pandas进行数据分析的过程中产生了空值，这些空值该如何进行清洗。

![](img/181b4528271b20dd92f0114cdeb67c73_49.png)

首先，我们导入必要的模块，并伪造一组包含空值的数据。

![](img/181b4528271b20dd92f0114cdeb67c73_51.png)

```python
import pandas as pd
from pandas import DataFrame, Series
import numpy as np

# 伪造一组数据，保证其中存在空值
df = DataFrame(np.random.randint(0, 100, size=(8, 6)))
# 将其中某些元素赋值为空值
df.iloc[2, 3] = None
df.iloc[4, 4] = np.nan
df.iloc[5, 2] = None
print(df)
```

![](img/181b4528271b20dd92f0114cdeb67c73_53.png)

现在，我们已经有了一组数据，这组数据中有三个空值。接下来，我们需要将这三个空值进行过滤（清洗）。

![](img/181b4528271b20dd92f0114cdeb67c73_55.png)

### 方法一：删除缺失值所在的行

![](img/181b4528271b20dd92f0114cdeb67c73_57.png)

![](img/181b4528271b20dd92f0114cdeb67c73_59.png)

对空值进行过滤，这里的过滤指的是删除空值所在的行数据。不能把空值单独删掉，因为单独删除一个位置的空值，剩下的还是一个不完整的结构。同样，通常也不能随意删除列，因为列是数据结构的一部分（类似于数据库表的字段）。在DataFrame中，通常只能删除行，不能删除列，除非明确指定删除一个无用的列。

![](img/181b4528271b20dd92f0114cdeb67c73_61.png)

因此，对缺失值进行过滤，我们选择的方式是将空值所对应的行删掉。使用的技术是`isnull()`、`notnull()`，结合`any()`和`all()`函数，来对我们控制所对应的行进行过滤。

首先，我们使用`df.isnull()`来查看数据中的空值。

![](img/181b4528271b20dd92f0114cdeb67c73_63.png)

```python
df.isnull()
```

![](img/181b4528271b20dd92f0114cdeb67c73_65.png)

`isnull()`返回一个布尔值DataFrame，其中`True`表示该位置是空值。可以看到，返回的数据中有三个`True`，它们对应的位置刚好是空值。

![](img/181b4528271b20dd92f0114cdeb67c73_67.png)

`notnull()`则与`isnull()`相反。

```python
df.notnull()
```

`notnull()`返回的布尔值DataFrame中，`False`对应的位置是空值。

我们最终是想把空值所对应的行删掉。这意味着我们需要知道哪些行中存在`True`（对于`isnull`）或`False`（对于`notnull`）。

![](img/181b4528271b20dd92f0114cdeb67c73_69.png)

我们可以结合`any()`函数来使用。`any()`函数用于检测行或列中是否存在`True`，如果存在就返回`True`，否则返回`False`。

![](img/181b4528271b20dd92f0114cdeb67c73_71.png)

```python
# 结合 isnull() 和 any(axis=1) 来找出包含空值的行
has_null_rows = df.isnull().any(axis=1)
print(has_null_rows)
```

![](img/181b4528271b20dd92f0114cdeb67c73_73.png)

返回的布尔值Series中，`True`对应的行索引（第2、4、5行）表示这些行中存在空值。

![](img/181b4528271b20dd92f0114cdeb67c73_75.png)

现在，我们可以将这组布尔值作为原数据的行索引，来筛选出包含空值的行。

![](img/181b4528271b20dd92f0114cdeb67c73_77.png)

```python
# 筛选出包含空值的行
rows_with_null = df.loc[has_null_rows]
print(rows_with_null)
```

![](img/181b4528271b20dd92f0114cdeb67c73_79.png)

这样，我们就得到了存在缺失值的行数据。我们可以获取这些行的索引，然后使用`drop`方法删除它们。

![](img/181b4528271b20dd92f0114cdeb67c73_81.png)

```python
# 获取要删除的行索引
index_to_drop = rows_with_null.index
print(index_to_drop)

![](img/181b4528271b20dd92f0114cdeb67c73_83.png)

![](img/181b4528271b20dd92f0114cdeb67c73_85.png)

# 删除这些行
df_cleaned = df.drop(labels=index_to_drop, axis=0)
print(df_cleaned)
```

![](img/181b4528271b20dd92f0114cdeb67c73_87.png)

现在，原数据中空值所对应的行已经被删除了。

![](img/181b4528271b20dd92f0114cdeb67c73_89.png)

我们也可以使用`notnull()`结合`all()`函数来实现。`all()`函数用于检测行或列中是否全部为`True`，全部为`True`才返回`True`，否则返回`False`。

```python
# 结合 notnull() 和 all(axis=1) 来找出不包含空值的行（即所有值都不为空的行）
rows_without_null_mask = df.notnull().all(axis=1)
print(rows_without_null_mask)

![](img/181b4528271b20dd92f0114cdeb67c73_91.png)

# 直接使用这个布尔掩码来筛选出不包含空值的行，效果等同于删除包含空值的行
df_cleaned_v2 = df.loc[rows_without_null_mask]
print(df_cleaned_v2)
```

![](img/181b4528271b20dd92f0114cdeb67c73_93.png)

![](img/181b4528271b20dd92f0114cdeb67c73_95.png)

通过`isnull()`结合`any()`，或者`notnull()`结合`all()`，就可以删除我们空值所对应的行数据。这是一个固定搭配：`isnull()`通常结合`any()`，`notnull()`通常结合`all()`。

![](img/181b4528271b20dd92f0114cdeb67c73_97.png)

### 方法二：使用 `dropna()` 函数直接删除

![](img/181b4528271b20dd92f0114cdeb67c73_99.png)

Pandas已经为我们集成了一个更简单的函数`dropna()`，可以直接将缺失值所在的行或列进行删除。

![](img/181b4528271b20dd92f0114cdeb67c73_101.png)

```python
# 直接删除包含空值的行
df_dropped = df.dropna(axis=0)
print(df_dropped)

![](img/181b4528271b20dd92f0114cdeb67c73_103.png)

![](img/181b4528271b20dd92f0114cdeb67c73_105.png)

# 如果想删除包含空值的列（通常不推荐，会改变数据结构）
# df_dropped_cols = df.dropna(axis=1)
# print(df_dropped_cols)
```

`axis=0`表示删除行，`axis=1`表示删除列。使用`dropna()`一步到位，无需再使用`isnull`、`any`等组合。但是，前面的`isnull`、`any`、`all`等方法最好也需要掌握，因为它们更基础，能帮助你理解原理。

![](img/181b4528271b20dd92f0114cdeb67c73_107.png)

### 方法三：对缺失值进行覆盖（填充）

![](img/181b4528271b20dd92f0114cdeb67c73_109.png)

![](img/181b4528271b20dd92f0114cdeb67c73_111.png)

除了删除，我们还可以对缺失值进行覆盖（填充）。将缺失值覆盖之后，数据中就没有空值了。

![](img/181b4528271b20dd92f0114cdeb67c73_113.png)

![](img/181b4528271b20dd92f0114cdeb67c73_115.png)

例如，原数据有空值，我把空值覆盖了，覆盖完之后就没有空值了。那么，怎么覆盖可以更有合理性呢？

![](img/181b4528271b20dd92f0114cdeb67c73_117.png)

我们可以使用`fillna()`函数进行缺失值的覆盖。

![](img/181b4528271b20dd92f0114cdeb67c73_119.png)

![](img/181b4528271b20dd92f0114cdeb67c73_121.png)

最简单的覆盖方式是手动指定一个值。

![](img/181b4528271b20dd92f0114cdeb67c73_123.png)

![](img/181b4528271b20dd92f0114cdeb67c73_125.png)

```python
# 用固定值（如666）填充所有空值
df_filled = df.fillna(value=666)
print(df_filled)
```

![](img/181b4528271b20dd92f0114cdeb67c73_127.png)

但是，这样用固定值覆盖的意义可能不大。更好的方式是，让空值周围近邻的值去覆盖它。例如，我们采集的是每分钟的温度数据，如果某一分钟的数据缺失，我们可以用前一分钟或后一分钟（近邻）的值去填充它。这样可以在最大合理范围内覆盖我们的空值。

![](img/181b4528271b20dd92f0114cdeb67c73_129.png)

![](img/181b4528271b20dd92f0114cdeb67c73_131.png)

`fillna()`函数提供了向前填充和向后填充的选项。

![](img/181b4528271b20dd92f0114cdeb67c73_133.png)

![](img/181b4528271b20dd92f0114cdeb67c73_135.png)

```python
# 向前填充 (ffill)：用前面的值填充空值
df_ffill = df.fillna(method='ffill', axis=0) # 沿行方向（垂直方向）向前填充
print(df_ffill)

# 向后填充 (bfill)：用后面的值填充空值
df_bfill = df.fillna(method='bfill', axis=0) # 沿行方向（垂直方向）向后填充
print(df_bfill)
```

`method='ffill'`表示向前填充，`method='bfill'`表示向后填充。`axis`参数指定填充的方向：`axis=0`（默认）表示沿着行的方向（垂直方向）填充，`axis=1`表示沿着列的方向（水平方向）填充。

例如，水平方向上的填充：

```python
# 水平方向向前填充
df_ffill_h = df.fillna(method='ffill', axis=1)
print(df_ffill_h)

# 水平方向向后填充
df_bfill_h = df.fillna(method='bfill', axis=1)
print(df_bfill_h)
```

什么时候选择删除，什么时候选择填充呢？一般来说，如果删除的成本太高（例如，10行数据中有6行包含空值，删除后只剩4行，数据损失太大），就选择填充。如果删除成本不高，我们尽可能选择删除，因为填充的值无论如何也不是原始的真实值。

## 总结

本节课中，我们一起学习了Pandas数据清洗中关于缺失值处理的核心内容。

我们首先了解了数据清洗的必要性，以及需要处理的三种数据问题：缺失值、重复值和异常值。

![](img/181b4528271b20dd92f0114cdeb67c73_137.png)

![](img/181b4528271b20dd92f0114cdeb67c73_139.png)

接着，我们重点探讨了缺失值。我们区分了`None`和`NaN`两种空值类型，明白了在数据分析中应使用`NaN`（浮点型）的原因——它可以参与运算而不会导致流程中断。

对于缺失值的清洗，我们掌握了三种主要方法：
1.  **删除缺失行**：通过`isnull().any()`或`notnull().all()`组合定位并删除包含空值的行。
2.  **使用`dropna()`函数**：这是Pandas提供的快捷方法，可直接删除含空值的行或列。
3.  **填充缺失值**：使用`fillna()`函数，可以用固定值填充，也可以用向前(`ffill`)或向后(`bfill`)填充的方式，用邻近的值进行更合理的填充。

![](img/181b4528271b20dd92f0114cdeb67c73_141.png)

![](img/181b4528271b20dd92f0114cdeb67c73_143.png)

选择删除还是填充，通常取决于数据情况：删除成本高时选择填充，反之则选择删除。理解并掌握这些方法，是进行高质量数据分析的基础。在下一节中，我们将继续学习数据清洗的另一个方面：重复值的处理。
