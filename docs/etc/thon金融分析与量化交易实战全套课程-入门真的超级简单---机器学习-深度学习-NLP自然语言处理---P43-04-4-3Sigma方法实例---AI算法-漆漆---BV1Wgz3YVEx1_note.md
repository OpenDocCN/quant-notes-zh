# 金融分析与量化交易：04-4-3：3-Sigma方法实例 📊

在本节课中，我们将要学习一种基于统计学原理的数据预处理方法——3-Sigma方法。该方法用于识别和处理数据集中的异常值（极值），其核心思想是利用正态分布的特性来定义数据的正常范围。

上一节我们介绍了其他去极值的方法，本节中我们来看看基于统计学原理的3-Sigma方法。

## 3-Sigma方法原理 📈

3-Sigma方法的原理非常简单。首先，我们需要理解正态分布（也称为高斯分布）。正态分布描述了一种常见的数据分布形态，大部分数据集中在平均值附近，越偏离平均值的数据出现的概率越低。

为了便于理解，可以想象一个例子：你去银行贷款，银行批准给你的额度在预期值上下有小幅波动是正常的。但如果波动幅度异常巨大（例如多给或少给10万元），这就属于不正常的情况，可以被视为“异常值”。

在标准正态分布中，平均值（μ）为0，标准差（σ）为1。数据分布如下图所示：

![](img/1013f4f4e9de096b8652af923c5ecdcb_1.png)

![](img/1013f4f4e9de096b8652af923c5ecdcb_1.png)

图中，离平均值（中心）越近的区间（如红色阴影区域），其面积越大，代表事件发生的概率越高。区间越往两端扩展，事件发生的概率就越低。

我们的目标是去除极值（异常值）。极值通常不会落在均值附近的高概率区域，而是落在远离均值的低概率区域。离均值越远、偏差越大的数据点，越有可能是离群点。

**3-Sigma** 正是基于这一思想。它通过计算数据的均值（μ）和标准差（σ），然后定义一个区间：`[μ - 3σ, μ + 3σ]`。根据统计学经验法则，对于服从正态分布的数据，约有 **99.7%** 的数据会落在这个区间内。

![](img/1013f4f4e9de096b8652af923c5ecdcb_3.png)

因此，3-Sigma方法认为，落在这个区间内的数据是“正常的”，而落在区间外的 **0.3%** 的数据则被视为“极值”，需要进行处理（如缩放到边界值），而不是直接丢弃。

![](img/1013f4f4e9de096b8652af923c5ecdcb_3.png)

## 代码实现 💻

接下来，我们将用Python代码实现3-Sigma方法。其步骤非常清晰：计算数据的均值和标准差，然后根据指定的倍数N（例如3）计算上下界。

![](img/1013f4f4e9de096b8652af923c5ecdcb_5.png)

以下是实现3-Sigma去极值函数的关键步骤：

首先，我们需要计算选定数据列的均值和标准差。

![](img/1013f4f4e9de096b8652af923c5ecdcb_7.png)

```python
# 计算均值和标准差
mean = data_series.mean()
std = data_series.std()
```

![](img/1013f4f4e9de096b8652af923c5ecdcb_9.png)

然后，根据公式计算上限和下限：

```python
# 计算上下限，其中 n 通常为3
upper_limit = mean + n * std
lower_limit = mean - n * std
```

![](img/1013f4f4e9de096b8652af923c5ecdcb_11.png)

![](img/1013f4f4e9de096b8652af923c5ecdcb_13.png)

最后，将原始数据中超出 `[lower_limit, upper_limit]` 范围的值规范到边界上。

![](img/1013f4f4e9de096b8652af923c5ecdcb_15.png)

```python
# 将超出范围的值规范到边界
data_series_clipped = data_series.clip(lower=lower_limit, upper=upper_limit)
```

![](img/1013f4f4e9de096b8652af923c5ecdcb_17.png)

我们将上述步骤封装成一个函数，以便重复使用。函数接收一个数据序列（Series）和倍数n作为输入，返回处理后的序列以及计算出的上下限。

![](img/1013f4f4e9de096b8652af923c5ecdcb_19.png)

![](img/1013f4f4e9de096b8652af923c5ecdcb_11.png)

![](img/1013f4f4e9de096b8652af923c5ecdcb_21.png)

代码执行后，我们可以通过绘图来直观地查看处理效果。图中绿色线通常代表原始数据的某个基准（如中位数），蓝色线则代表根据3-Sigma（或调整n倍后）计算出的边界。

![](img/1013f4f4e9de096b8652af923c5ecdcb_23.png)

![](img/1013f4f4e9de096b8652af923c5ecdcb_27.png)

![](img/1013f4f4e9de096b8652af923c5ecdcb_25.png)

![](img/1013f4f4e9de096b8652af923c5ecdcb_27.png)

不同的n值（如n=1, 2, 3）会对应不同的置信区间（例如68.3%，95.5%，99.7%），从而得到不同的边界位置。在实际应用中，可以根据具体任务的需求选择合适的n值。

## 方法总结与选择 🤔

![](img/1013f4f4e9de096b8652af923c5ecdcb_33.png)

至此，我们已经介绍了数据预处理中“去极值”环节的几种主要策略。3-Sigma方法是其中之一，它基于数据的统计分布特性来工作。

![](img/1013f4f4e9de096b8652af923c5ecdcb_35.png)

需要特别注意：**“去极值”不是简单地将异常数据删除**，而是通过定义合理的下界和上界，将所有数据规范到我们指定的边界之内。

![](img/1013f4f4e9de096b8652af923c5ecdcb_41.png)
