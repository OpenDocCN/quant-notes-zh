## 概述
在本节课中，我们将学习因子中性化（或称因子提纯）的核心概念与操作方法。我们将以市净率（PB）因子为例，讲解如何剔除其中与市值相关的成分，从而得到更“纯净”的因子。最后，我们会将处理后的因子应用于一个简单的选股策略。

## 因子中性化的核心思想
上一节我们介绍了因子处理的一般流程。本节中我们来看看如何对因子进行“提纯”或“中性化”。

因子中性化的目标是：从一个原始因子中，剔除掉我们不希望它包含的、来自其他因素（如市值、行业）的影响，从而得到该因子更本质的信息。

例如，我们发现市净率（PB）因子与市值（Size）存在较强的相关性。这意味着PB因子中包含了一部分市值的信息。如果我们只想研究PB本身对股价的影响，就需要将市值的影响从PB因子中剔除。

这个过程可以类比为：原始因子是一个整体，我们想从中“减去”能被另一个因素（如市值）解释的部分，剩下的就是“提纯”后的因子。

## 如何实现因子中性化：回归法
那么，具体如何“减去”市值的影响呢？答案是使用线性回归。

我们可以建立以下回归方程：
`因子 = W * 市值 + B + ε`
在这个方程中：
*   **因子** 是我们要处理的原始因子（如PB），作为因变量（Y）。
*   **市值** 是我们想要剔除的影响因素，作为自变量（X）。
*   **W** 和 **B** 是回归系数。
*   **ε** 是误差项。

建立这个回归方程的目的，是找出**市值能够解释原始因子中的多大一部分**。拟合得到的 `W * 市值 + B` 部分，就是市值对原始因子的“预测值”或“解释部分”。

然而，市值通常无法完全解释原始因子，预测值与真实值之间必然存在差异，这个差异就是回归的**残差（ε）**。

![](img/cbf0de0fb64124ddf2dcc1bb04adfe69_2.png)

**残差** 的含义至关重要：它代表了原始因子中**无法被市值所解释**的那部分信息。而这部分信息，正是我们进行中性化后想要保留的“纯净”因子。

![](img/cbf0de0fb64124ddf2dcc1bb04adfe69_4.png)

因此，因子中性化的操作就是：
`中性化后的因子 = 原始因子 - (W * 市值 + B) = 残差 ε`

![](img/cbf0de0fb64124ddf2dcc1bb04adfe69_6.png)

![](img/cbf0de0fb64124ddf2dcc1bb04adfe69_8.png)

以下是实现因子中性化的步骤：
1.  **建立回归方程**：以需要中性化的因子为Y，以需要剔除的影响因素（如市值）为X，进行线性回归，求解系数W和B。
2.  **计算残差**：用原始因子的真实值减去回归方程的预测值，得到的残差即为中性化后的因子。

![](img/cbf0de0fb64124ddf2dcc1bb04adfe69_10.png)

## 策略任务流程
理解了中性化的原理后，我们来看如何将其融入一个完整的因子选股策略。

![](img/cbf0de0fb64124ddf2dcc1bb04adfe69_12.png)

以下是构建一个基于处理后的PB因子进行选股的策略流程：
1.  **准备数据**：获取所有股票的市净率（PB）和市值（Size）数据。
2.  **数据预处理**：
    *   **股票池过滤**：剔除停牌股、ST股、上市时间不足半年的新股等。
    *   **因子处理**：对PB和市值数据依次进行去极值、标准化处理。
    *   **因子中性化**：对处理后的PB因子，进行针对市值的中性化处理，得到“纯净”的PB因子。
3.  **制定选股规则**：基于中性化后的PB因子制定选股标准。例如，选择`中性化PB值 < 0.2`的股票。
4.  **执行策略**：根据选股规则，确定买入和卖出的股票标的。

## 总结
本节课中我们一起学习了因子中性化的核心概念与实现方法。我们了解到，因子中性化本质是通过线性回归，剔除原始因子中与其他变量（如市值）相关的部分，保留其独立信息。具体步骤是建立回归方程并计算残差。最后，我们将中性化处理作为因子数据处理流程的一环，整合进了一个简单的因子选股策略框架中。后续课程中，我们将动手实现这一过程。
