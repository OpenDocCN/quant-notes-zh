## 股票的定义与作用

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

我们可以通过一个例子来形象地理解股票。假设一位创业者需要资金，而投资者（如亿万富翁）愿意提供资金。如果投资者只是借钱给创业者，这属于借贷行为，投资者只能获得利息。但投资者更希望分享公司成长带来的更大收益。于是，投资者将资金投入公司，作为交换，公司向投资者发行股票。股票就是投资者出资并拥有公司部分所有权的证明。

例如，一家初创公司估值5亿，由五位出资人（包括创始人）各出资1亿建立。那么每位出资人将获得公司20%的股票。这代表了他们对公司资产和未来收益的相应所有权。

股票的核心作用主要有两点：

**1. 股东身份与权利凭证**
持有股票意味着成为公司股东。这不仅是出资证明，也赋予了股东参与公司重大决策（如在股东大会上投票）的权利。

**2. 获取收益的途径**
这是普通股民最关心的一点。通过持有股票获利主要有两种方式：
*   **公司分红**：当公司盈利时，可能会将部分利润按持股比例分配给股东。
*   **交易获利**：投资者可以在二级市场（如证券交易所）买卖股票，通过“低买高卖”赚取差价。对于早期投资者，在公司价值增长后出售部分股票（套现），是另一种重要的获利方式。

无论是大股东还是小股民，其通过股票获利的原理是相同的，区别主要在于持股比例和交易规模。

## 公司上市与IPO

接下来我们了解一下“上市”的概念。

所谓上市，就是企业通过证券交易所，首次公开向全社会投资者增发股票，以募集资金的过程。

创业者不能随意向公众募集资金，这属于非法集资。公司需要达到一定体量，并向证监会提交申请，经过严格的审核（如审查财务报表、经营状况等），确认公司资质良好、风险可控后，才会被允许上市。

上市后，公司的股票就可以在证券交易所公开挂牌交易，所有符合条件的投资者都可以自由买卖。这为公司打开了面向广大公众的融资渠道。首次公开募股的行为就称为 **IPO**。

## 股票的分类

股票可以按照不同的标准进行分类，以下是两种常见的分类方式。

### 按公司业绩分类
根据公司的经营表现，股票通常被分为以下三类：

*   **蓝筹股**：指资本雄厚、信誉优良的巨型公司发行的股票。例如中国的“中石油”、“中石化”。这类公司体量大、经营稳定，如同赌场中最值钱的蓝色筹码。
*   **绩优股**：指业绩优良公司的股票。这类公司可能规模不是最大，但盈利能力持续强劲。例如“贵州茅台”，长期保持优异的盈利表现。
*   **ST股**：中文称为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记，以警示投资者该公司存在较高风险。若后续经营改善，可以申请撤销。

### 按上市地区分类
根据股票上市交易的地点和结算货币，可以分为：

*   **A股**：在中国大陆（上海或深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。
*   **H股**：在中国香港上市的股票。
*   **N股**：在美国纽约上市的股票。
*   **S股**：在新加坡上市的股票。

不同市场的交易规则有所不同，以下是几个主要市场的特点：

**A股市场的主要规则：**
*   **涨跌幅限制**：为防止股价剧烈波动，保护投资者，A股设有每日涨跌幅限制，通常为**10%**。即一只股票当日的价格波动幅度不能超过前一日收盘价的±10%。
*   **T+1交割制度**： “T”代表交易当日。T+1意味着**当日买入的股票，下一个交易日才能卖出**。这旨在减少市场中的过度投机行为。

**其他市场对比：**
*   **港股（H股）、美股（N股）** 等市场通常实行 **T+0** 交易（当日可买卖），且**没有涨跌幅限制**，股价波动可能更为剧烈。
*   **B股** 规则稍复杂，实行 **T+1** 交易，但资金交收为 **T+3**，即卖出股票后，资金需3个交易日才能到账提现。

了解不同市场的规则，对于进行跨市场分析或投资至关重要。
