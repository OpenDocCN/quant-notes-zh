## 计算每日差异值

![](img/562431b44f67951a9c08f03979fb57d8_6.png)

首先，我们需要计算每一天指标相对于前一天的差异值。例如，我们希望知道股价每天是上涨了还是下跌了，以及具体的变动数值。

![](img/562431b44f67951a9c08f03979fb57d8_8.png)

在Pandas中，这可以通过一个内置函数轻松实现。以下是计算每日差异值的代码：

```python
# 计算每日差异值，即后一天数据减去前一天数据
diff_data = data.diff()
print(diff_data.head())
```

![](img/562431b44f67951a9c08f03979fb57d8_10.png)

执行这段代码后，你会看到一个新的DataFrame。其中，1月5日的数据等于1月5日的原始值减去1月4日的原始值。对于数据中的第一天（如1月4日），由于没有前一天的数据可减，结果会显示为`NaN`（空值）。

![](img/562431b44f67951a9c08f03979fb57d8_12.png)

这个差异值指标在时间序列分析中非常常用。默认情况下，`diff()`函数计算的是相邻两天的差异。我们还可以通过指定参数来计算间隔多天的差异，例如计算每周或每月的差异。

为了养成良好习惯，当你对函数参数不熟悉时，建议查阅官方文档或相关资料。自己动手实践和查阅是最高效的学习方法。

## 计算变化率（增长率）

![](img/562431b44f67951a9c08f03979fb57d8_14.png)

![](img/562431b44f67951a9c08f03979fb57d8_16.png)

仅仅知道差异值有时还不够。例如，从1元涨到2元和从10000元涨到10001元，虽然都涨了1元，但意义完全不同。前者翻了一倍，后者几乎没变。因此，我们常常需要计算变化率或增长率。

![](img/562431b44f67951a9c08f03979fb57d8_18.png)

变化率的计算公式是：`(后一个值 - 前一个值) / 前一个值`。

Pandas同样提供了现成的函数来计算这个指标。以下是计算每日增长率的代码：

![](img/562431b44f67951a9c08f03979fb57d8_20.png)

```python
# 计算每日变化率（增长率）
pct_change_data = data.pct_change()
print(pct_change_data.head())
```

计算完成后，数据将显示为百分比形式的增长率。通常，单日股价的增长率数值较小，例如0.1%或0.2%，这属于正常现象。

![](img/562431b44f67951a9c08f03979fb57d8_22.png)

## 统计与可视化增长率

![](img/562431b44f67951a9c08f03979fb57d8_24.png)

得到每日的增长率序列后，我们可能希望了解其整体统计特性，例如平均增长率。

![](img/562431b44f67951a9c08f03979fb57d8_26.png)

![](img/562431b44f67951a9c08f03979fb57d8_28.png)

以下是计算平均增长率并将其可视化的步骤：

![](img/562431b44f67951a9c08f03979fb57d8_30.png)

首先，计算各列的平均增长率：

![](img/562431b44f67951a9c08f03979fb57d8_32.png)

```python
# 计算各指标的平均增长率
mean_pct_change = pct_change_data.mean()
print(mean_pct_change)
```

![](img/562431b44f67951a9c08f03979fb57d8_34.png)

接着，我们可以将平均增长率用条形图直观地展示出来。由于平均增长率是每个指标的一个单独数值，条形图比折线图更合适。

以下是绘制条形图的代码：

![](img/562431b44f67951a9c08f03979fb57d8_36.png)

```python
# 绘制平均增长率的条形图
mean_pct_change.plot(kind='bar', figsize=(16, 8))
```

![](img/562431b44f67951a9c08f03979fb57d8_38.png)

通过这张图，各个指标的平均增长情况一目了然。Y轴的值通常较小，这符合股价日常波动的特点。

![](img/562431b44f67951a9c08f03979fb57d8_40.png)
