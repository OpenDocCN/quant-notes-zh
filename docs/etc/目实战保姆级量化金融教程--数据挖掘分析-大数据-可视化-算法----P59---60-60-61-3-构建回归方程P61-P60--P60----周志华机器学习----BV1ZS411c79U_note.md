# Python量化交易：P59-P61：构建回归方程 📈

在本节课中，我们将学习如何利用历史数据构建一个回归方程，用于预测股票每日的涨跌走势。我们将使用Python的`scikit-learn`库来实现线性回归模型，并基于模型的预测结果制定简单的交易策略。

---

## 导入必要的工具包

首先，我们需要借助一些工具包。这里选择的是`scikit-learn`，它是Python中一个非常常见的机器学习库，包含了几乎所有常规的机器学习算法。

```python
from sklearn.linear_model import LinearRegression
```

我们将使用`LinearRegression`，它位于`linear_model`模块中。导入这个包后，我们基本上可以用一行代码来完成核心的建模任务。

---

## 实例化与训练模型

在`scikit-learn`中，构建任何模型的第一步都是对导入的类进行实例化。

```python
model = LinearRegression()
```

实例化完成后，我们需要用数据来训练这个模型。训练操作在`scikit-learn`中通过`.fit()`方法完成。该方法需要传入两个参数：特征数据`X`和目标变量`Y`。

```python
# 假设我们已将特征列名存储在全局变量 `feature_columns` 中
X = data[feature_columns]
Y_returns = data['returns']  # 第一种目标变量：实际报酬率
Y_direction = data['direction']  # 第二种目标变量：涨跌方向（1涨，0跌）

# 训练第一个模型，预测报酬率
model_returns = LinearRegression()
model_returns.fit(X, Y_returns)

# 训练第二个模型，预测涨跌方向
model_direction = LinearRegression()
model_direction.fit(X, Y_direction)
```

我们进行了两个对比实验：一个以实际报酬率`returns`为目标，另一个以涨跌方向`direction`为目标。通过对比实验结果，我们可以判断哪个目标变量更适合我们的预测任务。

---

## 进行预测并解读结果

模型训练好后，就可以用它来对数据进行预测了。预测使用`.predict()`方法。

```python
# 使用训练好的模型进行预测
pred_returns = model_returns.predict(X)
pred_direction = model_direction.predict(X)

# 将预测结果添加到原始数据中
data['pred_returns'] = pred_returns
data['pred_direction'] = pred_direction
```

查看预测结果，我们可能会发现预测值并不完美。例如，实际值为正时，预测值可能为负。这是因为线性回归是一个相对简单的模型，无法100%捕捉复杂的市场波动。但这正是我们实验的意义所在：比较有策略的操作是否优于无作为。

---

## 将预测值转换为交易信号

对于交易策略而言，我们通常不关心预测的具体数值，而只关心其指示的“买”或“卖”信号。因此，我们需要将连续的预测值转换为离散的交易标志位。

以下是处理逻辑：预测值大于0时，视为买入信号（标记为1）；小于0时，视为卖出或观望信号（标记为-1或0）。这里我们统一转换为1（买）和-1（卖）。

```python
# 将回归预测值转换为交易信号（1表示买，-1表示卖）
data['signal_returns'] = (data['pred_returns'] > 0).astype(int) * 2 - 1  # 将True/False映射为1/-1
data['signal_direction'] = (data['pred_direction'] > 0).astype(int) * 2 - 1
```

执行上述代码后，数据中会新增两列`signal_returns`和`signal_direction`，它们直接指明了模型建议的每日操作。

---

## 基于策略计算模拟收益

有了交易信号，我们就可以模拟策略的收益情况了。核心思想是：当信号为1（买入）时，我们就获得当日的实际报酬率`returns`；当信号为-1（卖出）时，我们则获得`-returns`（即做空或避免亏损）。

```python
# 计算策略一的模拟收益：信号 * 当日实际报酬率
data['strategy_returns_1'] = data['signal_returns'] * data['returns']

# 计算策略二的模拟收益
data['strategy_returns_2'] = data['signal_direction'] * data['returns']
```

计算完成后，我们可以查看`data.head()`来观察策略的每日收益情况。通过累计这些收益，并与单纯持有股票（即始终为1）的收益进行对比，就能评估策略的有效性。
