# HTML基础教程：04：div区块元素 📦

在本节课中，我们将要学习HTML中一个非常重要的概念——区块元素，特别是`<div>`标签。我们将了解什么是区块元素，它与内联元素的区别，以及`<div>`和`<span>`标签在网页布局和样式控制中的核心作用。

## 什么是区块元素？

上一节我们介绍了HTML的基本标签，本节中我们来看看区块元素。在HTML中，大多数标签被定义为块级元素或内联元素。

![](img/627a6e240121b8a596d5bccace1acbae_1.png)

*   **块级元素**：以新行开始和结束。例如，`<h1>`标签会独占一行，下一个`<h1>`标签会自动换到新的一行。
*   **内联元素**：不会以新行开始。例如，多个`<img>`（图片）标签会并排显示，直到屏幕宽度被占满才会换行。

以下是常见的元素类型示例：
*   块级元素：`<h1>`, `<p>`, `<div>`
*   内联元素：`<a>`, `<img>`, `<b>`, `<span>`

![](img/627a6e240121b8a596d5bccace1acbae_3.png)

![](img/627a6e240121b8a596d5bccace1acbae_5.png)

## `<div>` 区块元素的作用

理解了区块元素的概念后，我们重点来看`<div>`标签。`<div>`本身没有特定的语义含义（不像`<h1>`代表标题），它的核心作用是作为一个**容器**。

`<div>`可以将其他HTML元素组合起来，形成一个逻辑上的“块”。这样做的主要目的是为了**网页布局**。

![](img/627a6e240121b8a596d5bccace1acbae_7.png)

![](img/627a6e240121b8a596d5bccace1acbae_9.png)

![](img/627a6e240121b8a596d5bccace1acbae_11.png)

现代网页（如B站、知乎）都是由多个内容块组成的。`<div>`就像一个个箱子，可以把相关的标题、段落、图片等元素装在一起。然后，我们可以通过CSS样式统一控制这个“箱子”的位置、大小、颜色等，从而实现复杂的页面布局。

例如，你可以将一个`<div>`及其内部所有元素统一设置为靠右排列，或者设置统一的背景色。

## `<span>` 内联元素的作用

![](img/627a6e240121b8a596d5bccace1acbae_13.png)

与用于布局的`<div>`相对应，`<span>`是一个内联元素。它通常用于**为小段文字或内联内容添加样式**。

![](img/627a6e240121b8a596d5bccace1acbae_15.png)

例如，在一段文字中，你只想将其中几个字变为红色，但又不想让它们换行。这时，就可以用`<span>`标签包裹这几个字，然后为其添加样式。

**代码示例**：将“金角大王”几个字变为红色。
```html
<p>大家好，我是<span style="color: red;">金角大王</span>。</p>
```

## 实践：使用`<div>`进行简单布局

虽然完整的样式控制需要CSS知识，但我们可以先通过一个简单例子直观感受`<div>`作为容器的作用。

假设我们想将页面上的一组图片用一个带边框的盒子包裹起来，并控制这个盒子的宽度和位置。

**HTML结构示例**：
```html
<body>
    <!-- 第一个div容器，包裹一组图片 -->
    <div style="border: 1px dashed yellow; width: 500px; margin-bottom: 20px;">
        <img src="image1.jpg" alt="图片1">
        <img src="image2.jpg" alt="图片2">
        <img src="image3.jpg" alt="图片3">
    </div>

    <!-- 页面的其他内容，比如一个表格 -->
    <table>
        <!-- 表格内容 -->
    </table>
</body>
```

**样式属性解释（初步了解）**：
*   `border: 1px dashed yellow;`：为`<div>`添加一个1像素宽、虚线、黄色的边框。
*   `width: 500px;`：设置`<div>`容器的宽度为500像素。
*   `margin-bottom: 20px;`：设置`<div>`容器的下边距为20像素，使其与下方的表格产生间隔。

通过调整这些样式值，你可以改变这个“盒子”的外观和位置，从而理解`<div>`如何作为布局的基础单元。

## 总结

本节课中我们一起学习了HTML中区块元素的核心概念。
1.  我们区分了**块级元素**（独占一行）和**内联元素**（并排显示）。
2.  我们重点掌握了 **`<div>`标签**，它是一个块级容器，主要用于组合其他元素以实现**网页布局**。
3.  我们了解了 **`<span>`标签**，它是一个内联容器，主要用于为**小段内联内容添加特定样式**。

![](img/627a6e240121b8a596d5bccace1acbae_21.png)

记住：`<div>`用于布局（大块区域），`<span>`用于修饰（小块文字）。它们是后续学习CSS实现精美网页的基石。
