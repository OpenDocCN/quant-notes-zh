## 夏普比率的定义与意义

上一节我们介绍了投资回报率，本节中我们来看看如何结合风险来评估收益。夏普比率描述的是：在承担单位风险的情况下，我们能获得多少超额收益。

为了便于理解，可以想象一个场景：一份在叙利亚日薪数万的雇佣兵工作，其高薪对应的是极高的生命风险。在投资中，我们同样需要衡量，为了获得更高的收益，我们所承担的额外风险是否“值得”。夏普比率就是这样一个衡量“性价比”的指标。**指标越高，意味着在承担相同风险的情况下，获得的超额收益越高**。因此，在比较多个投资组合或股票时，在其他条件相似的情况下，我们倾向于选择夏普比率更高的选项。

## 夏普比率的计算公式

![](img/3961e9278c79220b9ccaaabf5216bbbc_1.png)

理解了夏普比率的意义后，我们来看看如何计算它。其核心思想是比较投资组合的收益与无风险收益的差异，并将这个差异与投资组合自身的波动性（风险）进行比较。

![](img/3961e9278c79220b9ccaaabf5216bbbc_3.png)

计算公式如下：

![](img/3961e9278c79220b9ccaaabf5216bbbc_5.png)

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率。
*   **Rf**：无风险收益率（例如国债利率、银行定期存款利率）。
*   **σp**：投资组合收益率的标准差，代表风险。

这个公式计算的是**每承担一单位的总风险（标准差），所能获得的超过无风险收益的回报**。我们希望这个值越大越好。

## 使用Python计算夏普比率

接下来，我们通过Python代码，实际计算几只股票的夏普比率。假设我们已经有了一系列股票的历史日收益率数据。

以下是计算夏普比率的关键步骤：

1.  **准备数据**：确保收益率数据是完整的，对缺失值进行填充（例如用前一天的收益率填充）。
2.  **设定参数**：确定无风险收益率（本例中假设年化无风险收益率为5%）和年化交易日数（通常为250天）。
3.  **执行计算**：按照公式，计算超额收益的年化均值，再除以收益率的年化标准差。

```python
import numpy as np
import pandas as pd

# 假设 `returns` 是一个Pandas DataFrame，每一列代表一只股票的日收益率序列
# 步骤1: 处理缺失值，用前向填充方法
returns_filled = returns.fillna(method='ffill')

# 步骤2: 设定参数
risk_free_rate = 0.05  # 年化无风险收益率
trading_days_per_year = 250  # 年化交易日数

# 步骤3: 计算夏普比率
# 计算超额收益（日收益率 - 日化无风险收益率）
excess_returns = returns_filled - risk_free_rate / trading_days_per_year

![](img/3961e9278c79220b9ccaaabf5216bbbc_7.png)

# 计算夏普比率公式的分子：超额收益的年化均值
annualized_excess_return = excess_returns.mean() * trading_days_per_year

# 计算夏普比率公式的分母：收益率年化标准差
annualized_volatility = returns_filled.std() * np.sqrt(trading_days_per_year)

# 计算最终的夏普比率
sharpe_ratios = annualized_excess_return / annualized_volatility

# 查看结果
print(sharpe_ratios)
```

运行以上代码后，我们将得到每只股票的夏普比率。例如，输出结果可能如下：

```
股票A   -0.12
股票B    0.85
中兴通讯  1.32
股票D    0.41
dtype: float64
```
