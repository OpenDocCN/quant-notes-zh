## 概述
在本节课中，我们将学习如何利用Pandas创建和操作时间序列。时间序列是以时间对象作为索引的Series或DataFrame，它在金融数据分析中至关重要。我们将学习如何创建时间序列，并使用其强大的索引和重采样功能进行数据切片与聚合分析。

---

## 时间序列的创建
上一节我们介绍了Pandas中生成时间对象的多种函数。本节中，我们来看看如何用这些时间对象构成一个时间序列。

时间序列是以`DatetimeIndex`作为索引的Series或DataFrame。我们可以使用`pd.date_range()`函数生成一个时间索引。

![](img/7953d8ef0cdca7f0d08e1cc2afcddb00_3.png)

![](img/7953d8ef0cdca7f0d08e1cc2afcddb00_5.png)

例如，创建一个带有时间索引的Series对象：
```python
import pandas as pd
import numpy as np

![](img/7953d8ef0cdca7f0d08e1cc2afcddb00_7.png)

# 创建一个长度为1000的Series，值为0到999
sr = pd.Series(np.arange(1000))
# 指定索引为从2017-01-01开始，长度为1000天的时间序列
sr.index = pd.date_range('2017-01-01', periods=1000)
```
执行`sr.index`可以看到，其类型是`DatetimeIndex`，而非普通的字符串索引。这就构成了一个基础的时间序列。

![](img/7953d8ef0cdca7f0d08e1cc2afcddb00_9.png)

## 时间序列的索引与切片
成为时间序列后，一个直观的好处是我们可以方便地按时间范围选取数据。

以下是几种常用的切片方式：

*   **按年切片**：`sr['2017']` 会选取2017年全年的数据。
*   **按年月切片**：`sr['2017-03']` 会选取2017年3月份的所有数据。
*   **按日期范围切片**：`sr['2017-12-25':'2018-02-01']` 会选取从2017年12月25日到2018年2月1日（包含首尾）的数据。

即使传入的是字符串格式的日期，Pandas也能自动识别并完成切片。这种语法使得按时间筛选数据变得非常简洁高效。

---

## 时间序列的重采样
除了切片，时间序列还支持强大的重采样功能。`resample()`函数可以将数据按照新的时间频率（如天、周、月）进行聚合。

以下是`resample()`的常见用法：

*   **按周求和**：`sr.resample('W').sum()` 将数据按周分组，并计算每周值的总和。
*   **按月求平均值**：`sr.resample('M').mean()` 将数据按月分组，并计算每月的平均值。

在金融分析中，这常用于将高频的日数据转换为低频的周数据或月数据，以便观察长期趋势。

## 其他辅助函数
时间序列还有一些其他辅助函数，例如`truncate()`。

`truncate()`函数可以截取指定日期之前或之后的数据：
*   `sr.truncate(before='2018-02-03')` 保留2018年2月3日及之后的数据。
*   `sr.truncate(after='2018-02-03')` 保留2018年2月3日及之前的数据。

不过，由于切片操作`sr['2018-02-03':]`或`sr[:'2018-02-03']`通常能实现相同的效果，因此`truncate()`函数的使用频率相对较低。
