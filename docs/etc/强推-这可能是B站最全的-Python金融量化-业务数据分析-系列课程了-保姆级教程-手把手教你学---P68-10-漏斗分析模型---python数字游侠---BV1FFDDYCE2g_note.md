## 概述
在本节课中，我们将要学习漏斗分析模型。这是一种在业务流程转化分析中广泛使用的模型，尤其在电商和用户行为分析领域。我们将从基础概念入手，理解其原理，并学习如何在Excel中绘制漏斗图，最后深入探讨其在企业运营（如AARRR模型）中的高级应用。

## 漏斗分析模型基础概念

![](img/21948654d5e424a1bfbaafbd3093a7c6_12.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_14.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_16.png)

上一节我们介绍了课程概述，本节中我们来看看漏斗分析模型的基础概念。

![](img/21948654d5e424a1bfbaafbd3093a7c6_18.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_20.png)

漏斗分析模型，在营销领域，指的是将非潜在客户逐步转化为客户的一个量化模型。其核心思想是模拟一个漏斗形状，展现用户在完成最终目标（如下单支付）前所经历的各个关键环节，以及每个环节的用户数量或转化率。

![](img/21948654d5e424a1bfbaafbd3093a7c6_22.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_24.png)

从用户路径分析的角度来简单理解：
1.  用户首先访问并浏览网站。
2.  其中一部分用户开始选购商品。
3.  选购商品的用户中，又有一部分将商品添加至购物车。
4.  添加购物车的用户里，部分会进行结算、提交订单。
5.  最终，只有一部分提交订单的用户完成支付。

![](img/21948654d5e424a1bfbaafbd3093a7c6_26.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_28.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_30.png)

在整个过程中，用户数量像经过漏斗一样逐层减少。例如，初始有100个浏览用户，最终可能只有5-10人完成支付。我们期望最终留存（完成支付）的用户越多越好。如果最终用户数量远低于预期，我们就需要逐层分析，定位流失率异常高的环节，并探究其原因。

![](img/21948654d5e424a1bfbaafbd3093a7c6_32.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_34.png)

## 在Excel中绘制漏斗图

![](img/21948654d5e424a1bfbaafbd3093a7c6_36.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_38.png)

理解了漏斗模型的理论后，本节中我们来看看如何在工具中将其可视化。由于部分Excel版本没有内置的漏斗图，我们可以通过辅助数据来模拟绘制。

以下是使用Excel 2016绘制漏斗图的步骤：
1.  准备数据：除了“环节”和“客户数”两列，需要增加一列“辅助数据”。辅助数据的计算公式为：`(最大客户数 - 当前环节客户数) / 2`。目的是让条形图左右对称，形成漏斗形状。
2.  插入图表：选中“环节”、“客户数”和“辅助数据”三列，点击【插入】->【推荐的图表】->【条形图】，选择第二个“堆积条形图”。
3.  调整图表：
    *   选中纵坐标轴（分类轴），右键选择【设置坐标轴格式】。
    *   在坐标轴选项中找到并勾选【逆序类别】。这将使环节顺序从上到下与数据源顺序一致。
    *   在图表数据系列中，将“辅助数据”系列调整到“客户数”系列的上方。
    *   选中“辅助数据”系列的条形，将其填充色设置为“无填充”。此时，“客户数”的条形就会居中显示，形成漏斗状。

此外，我们通常需要计算两类转化率：
*   **环节转化率**：指相邻两个环节之间的转化效率。公式为：`本环节用户数 / 上一环节用户数`。
*   **整体转化率**：指每个环节相对于最初环节（如浏览）的转化效率。公式为：`本环节用户数 / 第一环节用户数`。

## AARRR模型：企业运营中的漏斗分析

![](img/21948654d5e424a1bfbaafbd3093a7c6_48.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_50.png)

掌握了基础的漏斗模型绘制后，本节中我们将其提升到企业运营的战略层面，看看一个完整的用户生命周期是如何被分析和优化的。这里我们引入**AARRR模型**，它描述了用户增长的五个关键阶段。

![](img/21948654d5e424a1bfbaafbd3093a7c6_52.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_54.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_56.png)

AARRR模型，又称海盗指标模型，包含以下五个环节：
1.  **获取（Acquisition）**：用户如何发现并来到你的产品。
2.  **激活（Activation）**：用户的第一次使用体验是否良好。
3.  **留存（Retention）**：用户是否会重复使用产品，成为回头客。
4.  **收入（Revenue）**：产品如何通过用户赚钱。
5.  **传播（Referral）**：用户是否愿意将产品推荐给他人。

![](img/21948654d5e424a1bfbaafbd3093a7c6_58.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_60.png)

接下来，我们详细拆解每个环节的关注点和关键指标。

![](img/21948654d5e424a1bfbaafbd3093a7c6_62.png)

### 1. 获取用户
这是漏斗的起点。企业需要通过各种渠道让潜在用户知晓并接触产品。

![](img/21948654d5e424a1bfbaafbd3093a7c6_64.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_66.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_68.png)

常见的获客渠道包括：
*   **展示广告**：在网站或APP的广告位展示，通常按展示次数（CPM）收费。
*   **搜索广告**：如百度、谷歌竞价排名，通常按点击次数（CPC）收费。
*   **信息流广告**：如朋友圈、知乎推送的广告，可按点击（CPC）或实际效果（CPA/CPS）付费。

![](img/21948654d5e424a1bfbaafbd3093a7c6_70.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_72.png)

高效的获客策略往往目标明确。例如，Keep在初期并未大规模投放广告，而是先在有健身、减肥需求的社群中分享优质内容，积累目标用户，待时机成熟再推出APP，转化率非常高。

![](img/21948654d5e424a1bfbaafbd3093a7c6_74.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_76.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_78.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_80.png)

在这个阶段，我们需要关注以下指标：
*   各渠道的曝光量、下载量
*   渠道转化率
*   日新增用户数
*   **获客成本（CAC）**：获取一个用户所花费的平均成本。

![](img/21948654d5e424a1bfbaafbd3093a7c6_82.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_84.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_86.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_88.png)

### 2. 激活用户
用户下载后，关键在于提供出色的首次体验，让其感受到产品价值并留下来。

![](img/21948654d5e424a1bfbaafbd3093a7c6_90.png)

提升激活的关键在于满足用户核心需求。例如，Keep在注册时会收集用户的健身目标、水平等信息，从而为其推荐个性化的内容，提升初体验。

这个阶段需要关注的指标包括：
*   **日活跃用户数（DAU）/月活跃用户数（MAU）**
*   活跃率
*   页面浏览量（PV）与独立访客数（UV）
*   注册流程的每一步转化率（用于分析流失环节）

![](img/21948654d5e424a1bfbaafbd3093a7c6_92.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_94.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_96.png)

### 3. 提高留存
用户被激活后，目标是让他们养成使用习惯，成为忠实用户。

![](img/21948654d5e424a1bfbaafbd3093a7c6_98.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_100.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_102.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_104.png)

提高留存需要创造用户粘性。例如，支付宝的“蚂蚁森林”功能，通过收集能量、偷能量等游戏化社交互动，显著增加了用户每日打开APP的次数，从而提高了留存率。

![](img/21948654d5e424a1bfbaafbd3093a7c6_106.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_108.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_110.png)

对于留存阶段，核心指标是**留存率**：
*   次日留存率
*   第7日留存率
*   第30日留存率
*   对于电商，还需关注**复购率**、用户购买频次等。

![](img/21948654d5e424a1bfbaafbd3093a7c6_112.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_114.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_116.png)

### 4. 增加收入
在拥有稳定留存用户的基础上，企业需要实现商业变现。

![](img/21948654d5e424a1bfbaafbd3093a7c6_118.png)

增加收入的方式多样，如直接销售商品、提供增值服务、投放广告等。关键在于在提供价值的同时，让用户愿意付费。

这个阶段的核心指标围绕用户价值和付费行为：
*   **客单价**：每个付费订单的平均金额。
*   **付费用户占比（PUR）**
*   **平均每用户收入（ARPU）**
*   **平均每付费用户收入（ARPPU）**
*   总销售额/营收

![](img/21948654d5e424a1bfbaafbd3093a7c6_120.png)

![](img/21948654d5e424a1bfbaafbd3093a7c6_122.png)

### 5. 病毒传播
这是增长的最高境界，即让老用户自发推荐新用户，形成裂变增长，从而极低成本地获取高质量用户。

病毒传播的典型案例是“砍一刀”和“拼团”模式。用户为了获得优惠，会主动将链接分享给好友，好友在参与过程中就可能成为新用户。

衡量传播效果的关键指标是：
*   **转发率/邀请率**：用户进行分享或邀请的行为比例。
*   **病毒系数（K因子）**：衡量每个用户能带来多少新用户。计算公式为：`K = 每个用户发出的邀请数 * 邀请的转化率`。若K>1，传播将像病毒一样快速扩张。
*   通过邀请带来的新用户数及其成本。

**透过现象看本质**：许多商业活动都暗含AARRR逻辑。例如，支付宝在春节推广“亲情账号”，表面是家庭互动，本质是借助年轻用户向三四线城市及中老年群体（新市场）渗透，低成本获取新用户。

## 总结
本节课中我们一起学习了漏斗分析模型。我们从其基础定义和业务流程中的应用入手，学会了如何在Excel中通过技巧绘制漏斗图。进而，我们深入探讨了更系统的AARRR增长模型，它从获取、激活、留存、收入到传播，完整刻画了用户生命周期和价值提升路径。理解这些模型，能帮助我们在数据分析工作中，不仅看到数据表象，更能洞察业务本质，定位关键问题，并提出有效的优化建议。
