## 课程核心模块介绍

上一节我们介绍了课程的整体目标，本节中我们来看看具体的三大核心模块。

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_4.png)

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_6.png)

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_8.png)

课程内容主要涉及三大模块：Python数据分析与处理、金融数据分析以及量化交易策略。

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_10.png)

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_12.png)

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_14.png)

以下是三大模块的详细说明：

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_16.png)

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_18.png)

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_20.png)

1.  **Python数据分析与处理**：本模块将讲解如何使用Python处理和分析数据。核心是掌握如何用Python“玩转”数据，包括数据的读取、清洗、转换和分析等操作。
2.  **金融数据分析**：本模块将聚焦于股票等金融数据。我们会学习拿到股票数据后能进行哪些操作和分析，特别是针对**时间序列数据**的处理与分析方法。
3.  **量化交易策略**：本模块将进入策略设计与评估阶段。我们将学习如何编写一个交易策略，并利用历史数据进行**回测**，以分析该策略的盈利能力和优劣。回测的核心是验证策略在历史数据上的表现，公式可简化为：`策略表现 = 回测(策略， 历史数据)`。

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_22.png)

我们的课程是纯实战导向，没有传统PPT，重点在于如何在代码中实现金融分析与量化交易任务。所有讲解都将围绕实际代码和操作展开。

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_30.png)

在讲解过程中，我们将使用以下主要工具和环境：

*   **Jupyter Notebook**：这是我们编写和运行Python代码的主要界面，非常适合进行数据分析和交互式学习。
*   **Python数据分析库**：我们将重点依赖两个核心库：
    *   **NumPy**：用于高效的数值计算。
    *   **Pandas**：用于数据分析和处理。
*   **量化回测平台**：用于将我们编写的策略应用到历史数据中，模拟交易并生成详细的绩效报告。在平台上，我们可以运行策略并查看结果，例如：
    ```python
    # 伪代码示例：在回测平台上运行策略
    回测结果 = 回测平台.运行(我的策略， 起始日期， 结束日期， 初始资金)
    print(回测结果.累计收益)
    print(回测结果.超额收益)
    ```

## 详细课程大纲

以下是课程三个模块的详细安排，每个模块都旨在构建你的实战能力。

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_46.png)

1.  **模块一：Python必备工具包实战**
    *   **目标**：为零基础学员打好Python基础，熟悉课程核心工具。
    *   **内容**：Python基础与环境配置；NumPy数值计算库详解；Pandas数据分析库详解。本模块是后续学习的基石。

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_48.png)

2.  **模块二：Python金融数据分析实战**
    *   **目标**：将Python技能应用于真实的金融数据分析场景。
    *   **内容**：股票数据的获取与处理；金融时间序列分析与统计建模；量化回测平台的初步使用；完成第一个简单的回测项目，分析策略的初步表现。

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_50.png)

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_52.png)

3.  **模块三：量化交易策略深入实战**
    *   **目标**：深入量化交易领域，学习经典策略并能自行设计与改进。
    *   **内容**：讲解量化交易中的经典策略（如动量策略、均值回归策略）；策略在实际应用中的数据预处理与因子分析；策略的优化与改进方法；在回测中实现并验证策略，以获得更优的收益结果。

![](img/4ef94d2e66b1044ca7e66a53e5ec11c5_54.png)
