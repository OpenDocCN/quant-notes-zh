## 替换函数补充：`replace`

在讲解统计函数之前，我们先补充一个重要的文本处理函数：`replace`。`replace`函数用于替换文本中的指定内容。

![](img/ad8093229973bcab45ec659c1949c76c_3.png)

以下是`replace`函数的基本语法和操作步骤：

```python
replace(原始文本, 起始位置, 替换长度, 新文本)
```

*   **原始文本**：需要进行替换操作的原始字符串。
*   **起始位置**：开始替换的字符索引位置（从0开始计数）。
*   **替换长度**：需要被替换掉的字符数量。
*   **新文本**：用来替换旧内容的新字符串。

![](img/ad8093229973bcab45ec659c1949c76c_5.png)

例如，若有一个手机号码字符串，我们想隐藏其第8到第10位（索引7到9），可以这样操作：

```python
# 假设原始手机号码为 ‘13800138000’
new_number = replace(‘13800138000’, 7, 3, ‘***’)
# 结果为 ‘1380013***0’
```

![](img/ad8093229973bcab45ec659c1949c76c_7.png)

![](img/ad8093229973bcab45ec659c1949c76c_9.png)

执行后，指定的部分内容就被替换成了新的内容。

![](img/ad8093229973bcab45ec659c1949c76c_11.png)

对于每一种函数而言，只要掌握其语法结构，应用起来就能得心应手。`replace`函数的核心思想就是定位并替换文本中的特定片段。

## 基础取整与求余函数

接下来，我们进入统计函数的学习。首先介绍几个基础的数学函数：向下取整、四舍五入和求余数。

![](img/ad8093229973bcab45ec659c1949c76c_13.png)

### `int` 函数：向下取整

![](img/ad8093229973bcab45ec659c1949c76c_15.png)

`int`函数用于对一个数字进行向下舍入取整。它只有一个参数，即需要处理的数字。

```python
int(数字)
```

向下取整的含义是：取不大于该数字的最大整数。
*   对于正数，例如 `int(5.7)`，结果是 `5`。
*   对于负数，需要特别注意。例如 `int(-7.1)`，结果是 `-8`。因为 `-8` 是比 `-7.1` 更小的相邻整数。

理解负数向下取整的关键在于：在数轴上，数值越小，其位置越靠左。`-7.1` 在 `-8` 和 `-7` 之间，其中更小（更靠左）的整数是 `-8`。

### `round` 函数：四舍五入

![](img/ad8093229973bcab45ec659c1949c76c_17.png)

`round`函数用于对数字进行四舍五入。它包含两个参数：要处理的数字和要保留的小数位数。

```python
round(数字, 小数位数)
```

例如，`round(3.14159, 2)` 会将数字四舍五入到小数点后两位，结果为 `3.14`。

### `mod` 函数：求余数

![](img/ad8093229973bcab45ec659c1949c76c_19.png)

`mod`函数用于计算两数相除后的余数。它包含两个参数：除数和被除数。

```python
mod(除数, 被除数)
```

例如，`mod(5, 3)` 计算 `5` 除以 `3` 的余数，结果为 `2`。

## 描述性统计函数

在数据处理中，我们经常需要快速了解数据集的整体情况。以下是一组最常用的描述性统计函数，用于计算一组数据中的最大值、最小值、平均值、总和以及数据个数。

以下是这组函数的基本用法，它们通常接受一个数据区域作为参数：

```python
max(数据区域)   # 计算最大值
min(数据区域)   # 计算最小值
average(数据区域) # 计算算术平均值
sum(数据区域)    # 计算总和
count(数据区域)  # 计算数值个数（计数）
```

![](img/ad8093229973bcab45ec659c1949c76c_21.png)

假设我们有一个包含笔试成绩的数据区域 `B2:B63`：
*   `max(B2:B63)` 返回该区域中的最高分。
*   `min(B2:B63)` 返回该区域中的最低分。
*   `average(B2:B63)` 返回所有成绩的平均分。
*   `sum(B2:B63)` 返回所有成绩的总和。
*   `count(B2:B63)` 返回成绩的个数（即有多少个数据点）。

这些函数是数据分析中最基础且使用频率最高的工具，能够帮助我们快速把握数据的集中趋势和离散程度。
