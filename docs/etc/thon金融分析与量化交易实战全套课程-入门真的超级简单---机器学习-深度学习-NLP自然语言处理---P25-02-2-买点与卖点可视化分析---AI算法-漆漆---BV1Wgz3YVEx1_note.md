## 计算移动平均线

上一节我们完成了数据清洗，本节中我们来看看如何计算关键的指标——移动平均线。移动平均线是技术分析中常用的工具，用于平滑股价波动并识别趋势。

![](img/b70ed43e76648982383fdc830225e3f9_36.png)

我们首先定义短期和长期移动平均线的窗口大小。考虑到我们的数据集跨度较长（约7-8年），为了在图表中更清晰地观察交叉信号，我们将窗口值设置得比常规（如5日和20日）稍大一些。

```python
# 定义短期和长期移动平均线的窗口周期
short_window = 42
long_window = 252
```

![](img/b70ed43e76648982383fdc830225e3f9_38.png)

![](img/b70ed43e76648982383fdc830225e3f9_40.png)

定义好周期后，即可计算移动平均线。我们使用 pandas 的 `rolling()` 方法结合 `mean()` 函数来计算指定窗口内的平均值。

![](img/b70ed43e76648982383fdc830225e3f9_42.png)

![](img/b70ed43e76648982383fdc830225e3f9_44.png)

以下是计算移动平均线的步骤：

![](img/b70ed43e76648982383fdc830225e3f9_46.png)

![](img/b70ed43e76648982383fdc830225e3f9_48.png)

1.  **计算短期移动平均线 (SMA1)**：对苹果股价列应用 `rolling(short_window).mean()`。
2.  **计算长期移动平均线 (SMA2)**：对苹果股价列应用 `rolling(long_window).mean()`。
3.  **将结果添加到数据框中**：将计算出的均线作为新列存储，方便后续分析和绘图。

![](img/b70ed43e76648982383fdc830225e3f9_50.png)

![](img/b70ed43e76648982383fdc830225e3f9_52.png)

```python
# 计算短期移动平均线
data[‘SMA1‘] = data[‘AAPL‘].rolling(window=short_window).mean()
# 计算长期移动平均线
data[‘SMA2‘] = data[‘AAPL‘].rolling(window=long_window).mean()
# 查看计算后的数据尾部，确保计算成功
print(data.tail())
```

## 标记趋势与策略回测准备

![](img/b70ed43e76648982383fdc830225e3f9_72.png)

仅仅观察图表还不够，我们需要量化这些信号。本节我们将通过添加一个标志位来明确记录每一天的趋势状态，这是进行策略回测的基础。

![](img/b70ed43e76648982383fdc830225e3f9_74.png)

首先，由于移动平均线计算初期会产生大量 `NaN`（非数字）值，我们需要再次清理数据。

![](img/b70ed43e76648982383fdc830225e3f9_76.png)

```python
# 删除因计算移动平均线而产生的缺失值
data.dropna(inplace=True)
```

![](img/b70ed43e76648982383fdc830225e3f9_78.png)

接着，我们添加一个名为 `position` 的列。该列的值由短期均线（SMA1）和长期均线（SMA2）的关系决定：当短期均线大于长期均线时，标记为1，代表“持有”或“看涨”状态；否则标记为0。

```python
# 根据均线关系添加趋势标志位
data[‘position‘] = np.where(data[‘SMA1‘] > data[‘SMA2‘], 1, 0)
# 查看添加标志位后的数据
print(data.head())
```

![](img/b70ed43e76648982383fdc830225e3f9_80.png)

![](img/b70ed43e76648982383fdc830225e3f9_82.png)

这个 `position` 列从数值上清晰地标明了每个交易日所处的趋势阶段（1为上升趋势，0为下降趋势），为我们下一步模拟交易决策提供了直接依据。

## 策略逻辑与收益对比

现在，我们拥有了包含买卖信号标记的完整数据。本节我们来探讨双移动平均线策略的核心逻辑，并理解其如何试图超越简单的“买入并持有”策略。

策略的核心思想是：
*   当 `position` 从 0 变为 1 时（黄金交叉），执行**买入**操作。
*   当 `position` 从 1 变为 0 时（死亡交叉），执行**卖出**操作。

这样做的目的是在上涨趋势开始时入场，在下跌趋势开始时离场，从而规避部分下跌损失，锁定上涨利润。这与“买入并持有”（无论涨跌始终持有）的被动策略形成对比。

![](img/b70ed43e76648982383fdc830225e3f9_86.png)

例如，假设股价从100元跌至80元：
*   **买入并持有策略**：最终亏损20元。
*   **双均线策略**：在死亡交叉信号出现时（假设在95元）卖出，在后续的黄金交叉信号出现时（假设在75元）再买入。虽然股价下跌，但通过这次“高卖低买”的操作，反而实现了盈利。

![](img/b70ed43e76648982383fdc830225e3f9_88.png)

本节课中我们一起学习了金融数据分析中双移动平均线策略的完整实现流程。我们从数据预处理开始，计算了短期和长期移动平均线，并通过可视化图表直观识别了“黄金交叉”和“死亡交叉”信号。接着，我们通过添加趋势标志位量化了这些信号，为基于此策略的自动化交易或收益回测奠定了基础。这个过程展示了如何将理论上的技术指标转化为可执行、可验证的量化分析步骤。
