## 平均结果的局限性

上一节我们介绍了如何计算平均结果。那么，对于这个平均结果，我们能否进行加法操作呢？

例如，我们观察的是一个平均结果，它反映了整体情况。但我们能否计算一个连续时间序列的增长率呢？

## 增长率累加的问题

![](img/51f4d1836e6c2166ce05f44c7de70378_5.png)

什么是连续时间的增长率？让我们举个例子。

假设有一只股票：
*   第一天价格是100元。
*   第二天价格跌至50元。
*   第三天价格涨至75元。

对于第二天，增长率计算为 `(50 - 100) / 100 = -0.5`。
对于第三天，增长率计算为 `(75 - 50) / 50 = 0.5`。

如果我们简单地将这两个增长率相加：`-0.5 + 0.5 = 0`。结果为零，似乎意味着股票从第一天到第三天既没跌也没涨。

但观察原始数据：股价从100元变为75元，实际上是下跌的。这说明，增长率不能直接进行加法运算。

![](img/51f4d1836e6c2166ce05f44c7de70378_7.png)

## 引入对数计算连续变化

对于连续时间序列的累加操作，我们需要借助数学公式——对数。

![](img/51f4d1836e6c2166ce05f44c7de70378_9.png)

实际计算方法是：计算后一天数据除以前一天数据的对数，然后进行累加。

以之前的例子为例：
*   计算 `log(50/100) + log(75/50)`。

在以e为底的对数运算中，加法相当于内部数值的乘法：`log(50/100) + log(75/50) = log((50/100) * (75/50)) = log(75/100)`。

![](img/51f4d1836e6c2166ce05f44c7de70378_11.png)

这个结果正好反映了最后一天（75）与第一天（100）之间的整体差异比例。

形象地说，这相当于计算：股票初始投入的一块钱，在经过一系列涨跌后，在最终时点相当于多少钱。这反映了资产的连续复合增长情况。

![](img/51f4d1836e6c2166ce05f44c7de70378_13.png)

## Python实现连续变化分析

![](img/51f4d1836e6c2166ce05f44c7de70378_15.png)

![](img/51f4d1836e6c2166ce05f44c7de70378_17.png)

以下是计算连续时间增长率并还原为实际价值的步骤。

### 第一步：计算每日相对变化

![](img/51f4d1836e6c2166ce05f44c7de70378_19.png)

我们需要计算每一天数据与其前一天数据的比值。可以使用Pandas的`shift()`函数将数据向前移动一格，来获取“前一天”的数据。

```python
# 假设data是一个包含股价的Pandas Series或DataFrame
shifted_data = data.shift(1)  # 数据向前移动一格，得到前一天的数据
# 计算每日的相对变化（后一天/前一天）
daily_returns = data / shifted_data
```

### 第二步：应用对数并累加

![](img/51f4d1836e6c2166ce05f44c7de70378_21.png)

对每日相对变化取自然对数，然后使用`cumsum()`函数进行累加。`cumsum()`计算的是累积和，例如`[1,2,3,4]`的累积和是`[1,3,6,10]`。

![](img/51f4d1836e6c2166ce05f44c7de70378_23.png)

```python
import numpy as np
# 取对数并计算累积和
log_returns = np.log(daily_returns).cumsum()
```

### 第三步：还原为实际价值指数

将对数累加结果通过指数函数`np.exp()`还原回去，得到以起始点为基准（例如，起始点=1代表1块钱）的价值指数序列。

![](img/51f4d1836e6c2166ce05f44c7de70378_25.png)

![](img/51f4d1836e6c2166ce05f44c7de70378_27.png)

```python
# 还原为实际价值指数
cumulative_value = np.exp(log_returns)
```

### 第四步：可视化结果

![](img/51f4d1836e6c2166ce05f44c7de70378_29.png)

最后，我们可以将这个价值指数序列绘制成图，直观地看到最初的一单位资金随时间推移的变化情况。

```python
import matplotlib.pyplot as plt
cumulative_value.plot(figsize=(16, 8))
plt.title('Cumulative Returns Over Time (Starting from 1)')
plt.ylabel('Cumulative Value')
plt.xlabel('Date')
plt.show()
```

![](img/51f4d1836e6c2166ce05f44c7de70378_31.png)

![](img/51f4d1836e6c2166ce05f44c7de70378_33.png)

生成的图表会显示，例如，2010年初始的“1块钱”，在2011年、2012年……直至2018年，分别相当于多少“钱”。这直接反映了投资该资产在这些年间的整体增长倍数。

## 本节内容总结

![](img/51f4d1836e6c2166ce05f44c7de70378_35.png)

本节课中我们一起学习了金融时间序列的连续变化分析。

![](img/51f4d1836e6c2166ce05f44c7de70378_37.png)

1.  **差异值**：使用`.diff()`可以轻松计算相邻数据的差值。
2.  **简单增长率**：计算方式为`(后值 - 前值) / 前值`，但这种方法计算出的增长率不能直接累加以反映长期整体增长。
3.  **连续复合增长率**：为了解决累加问题，我们引入了对数计算方法。核心公式是计算`log(后一天/前一天)`的累积和，这等价于计算起始点与终止点的整体比例关系。
4.  **价值指数还原**：将对数累积结果通过`np.exp()`还原，得到以起始点为基准的价值指数，直观展示“最初的一块钱相当于未来某日多少钱”。
