## 概述：什么是策略评估指标？

当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以判断策略的盈亏情况。然而，仅仅知道“赚了”或“赔了”是不够的。我们需要更精确的度量标准来回答以下问题：
*   策略具体赚了多少？
*   与市场基准（如大盘指数）相比，策略表现如何？
*   策略承担了哪些风险？这些风险是否值得？
*   与其他策略相比，本策略有何优势？

这些问题的答案，就来自于一系列的策略评估指标。在后续课程中，我们将使用专业的回测平台自动生成这些指标。本节课的目标是理解这些指标背后的逻辑，以便我们能看懂任何分析报告中的结果。

## 回测收益率详解

上一节我们介绍了策略评估指标的重要性，本节中我们来看看最基础、最直观的指标——**回测收益率**。

![](img/ea6d841ebdea42472a0c8b09a5327873_11.png)

![](img/ea6d841ebdea42472a0c8b09a5327873_13.png)

回测收益率衡量的是，在整个回测周期结束后，策略最终获得的收益相对于初始本金的增长比例。它直接回答了“这个策略让我赚了多少钱”这个问题。例如，一个4.5%的回测收益率意味着，相对于初始投入，策略最终带来了4.5%的财富增长。

![](img/ea6d841ebdea42472a0c8b09a5327873_15.png)

其计算公式如下：

![](img/ea6d841ebdea42472a0c8b09a5327873_17.png)

![](img/ea6d841ebdea42472a0c8b09a5327873_19.png)

**回测收益率 = (期末资产总值 - 期初资产总值) / 期初资产总值**

![](img/ea6d841ebdea42472a0c8b09a5327873_21.png)

也可以表示为：

**回测收益率 = (P_t / P_0) - 1**

![](img/ea6d841ebdea42472a0c8b09a5327873_23.png)

其中：
*   `P_t` 代表回测结束时的资产价值（卖出所有头寸后的总资金）。
*   `P_0` 代表回测开始时的初始本金。

## 使用Python计算与可视化

![](img/ea6d841ebdea42472a0c8b09a5327873_25.png)

![](img/ea6d841ebdea42472a0c8b09a5327873_27.png)

理解了概念后，我们通过Python代码来实际计算并可视化回测收益率。我们将使用一份示例股票数据来模拟策略回测后的资产净值走势。

首先，导入必要的工具包并加载数据。

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便图表显示中文
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设‘date’列是日期
data = pd.read_csv('your_stock_data.csv', index_col=0, parse_dates=True)
# 假设‘close’列代表我们策略每日的资产净值
portfolio_value = data['close']
print(portfolio_value.head())
```

假设`portfolio_value`序列代表了我们的策略从期初到期末每日的资产净值。接下来，我们计算相对于期初的净值变化比率。

以下是计算步骤：
1.  计算每日净值相对于期初净值（`P_0`）的比值。
2.  将此比值序列减1，即可得到每日的累计收益率序列。
3.  序列的最后一个值，即为整个回测周期的总收益率。

![](img/ea6d841ebdea42472a0c8b09a5327873_29.png)

```python
# 1. 计算每日净值相对于期初的比值
relative_value = portfolio_value / portfolio_value.iloc[0]

![](img/ea6d841ebdea42472a0c8b09a5327873_31.png)

# 2. 可视化净值增长曲线
plt.figure(figsize=(10, 6))
relative_value.plot()
plt.title('策略资产净值相对增长曲线')
plt.xlabel('日期')
plt.ylabel('相对净值 (P_t / P_0)')
plt.grid(True)
plt.show()

![](img/ea6d841ebdea42472a0c8b09a5327873_33.png)

![](img/ea6d841ebdea42472a0c8b09a5327873_35.png)

# 3. 计算最终的回测收益率
backtest_return = relative_value.iloc[-1] - 1

![](img/ea6d841ebdea42472a0c8b09a5327873_37.png)

![](img/ea6d841ebdea42472a0c8b09a5327873_39.png)

# 4. 以表格形式展示最终结果
result_df = pd.DataFrame({
    '回测收益率': [backtest_return]
}, index=['最终结果'])
print(result_df)
```

执行以上代码后，你会看到一条曲线，展示了资产净值随时间相对于期初的增长情况。同时，在结果输出中，会显示本次回测的最终收益率。如果该值为正，代表策略盈利；为负则代表亏损。

![](img/ea6d841ebdea42472a0c8b09a5327873_41.png)
