## 课程概述

![](img/e77bf9a4455c54ef4b708296c75c7521_1.png)

本课程是一门纯实战课程，不依赖PPT，不教授金融理论或炒股方法。课程的核心重点在于如何使用Python代码实现金融数据分析与量化交易任务。我们将通过实际数据和代码操作，学习如何处理数据、分析数据以及进行策略回测。

### 模块一：Python数据分析与处理

![](img/e77bf9a4455c54ef4b708296c75c7521_11.png)

本模块是课程的基础部分，旨在让零基础学员掌握必要的Python工具。我们将学习如何使用Python处理和分析数据。

![](img/e77bf9a4455c54ef4b708296c75c7521_13.png)

![](img/e77bf9a4455c54ef4b708296c75c7521_15.png)

以下是本模块的核心内容：

![](img/e77bf9a4455c54ef4b708296c75c7521_17.png)

![](img/e77bf9a4455c54ef4b708296c75c7521_19.png)

*   **Python基础与环境配置**：讲解Python最核心的基础知识点，并指导如何配置和安装课程所需的工具。
*   **NumPy工具包**：学习用于数值计算的核心库。其核心功能通常通过数组操作实现，例如 `np.array([1, 2, 3])`。

### 模块二：金融数据分析

![](img/e77bf9a4455c54ef4b708296c75c7521_25.png)

本模块将进入实战环节，使用真实的股票数据作为案例。我们将学习如何对金融时间序列数据进行操作、分析和统计。

以下是本模块将涉及的操作：

*   **数据获取与探索**：学习如何获取股票数据，并进行初步的查看和了解。
*   **常用分析方法**：讲解在金融数据分析中常用的统计与分析方法。
*   **回测平台初探**：介绍量化回测平台的基本使用方法，并完成第一个简单的回测项目，验证策略在历史数据中的表现。

在对金融数据有了基本分析能力之后，我们就可以深入设计并测试更复杂的交易策略了。

### 模块三：量化交易策略与回测

本模块是课程的高级部分，将深入量化交易的核心。我们将学习经典的交易策略，并掌握策略开发、回测与改进的全流程。

![](img/e77bf9a4455c54ef4b708296c75c7521_27.png)

以下是本模块的重点：

*   **经典策略实现**：讲解并实现量化交易中一些常用的经典策略。
*   **策略回测与评估**：深入学习如何在回测平台中运行策略，并获取各项评估指标，如收益率、回撤、超额收益等。回测的核心是模拟历史交易，公式可简化为：`最终资产 = 初始资产 * (1 + 策略收益率)`。
*   **策略分析与优化**：学习如何根据回测结果分析策略的优缺点，并进行针对性的改进。
