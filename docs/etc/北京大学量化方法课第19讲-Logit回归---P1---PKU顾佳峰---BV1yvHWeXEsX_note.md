# 北京大学量化方法课：第19讲：Logit回归 🧮

在本节课中，我们将学习如何使用Python进行Logit回归分析。Logit回归适用于因变量为二元分类变量（例如0和1）的情况，例如“是否上学”、“是否拥有驾照”等。我们将从数据准备开始，逐步讲解模型的建立、结果解读以及模型检验。

---

## 数据准备与变量生成

上一节我们介绍了Logit回归的应用场景。本节中，我们来看看如何准备数据并生成分析所需的变量。

首先，我们有一个包含“姓名”、“性别”、“专业”、“满意度”、“考试成绩”等字段的数据集。为了进行分析，我们需要将“评价”这类分类变量转换为由0和1构成的二元变量。例如，可以将“好”评价编码为1，“差”评价编码为0。

![](img/9bae42176624c3769cb4cc0ac01ce315_1.png)

以下是生成二元评价变量的核心代码示例：

```python
# 假设df是包含‘评价’列的DataFrame，其中‘好’和‘差’为原始值
df['评价_二元'] = df['评价'].apply(lambda x: 1 if x == '好' else 0)
```

完成数据转换后，我们就可以开始建立回归模型了。

---

## 一元Logit回归模型

在准备好数据之后，本节我们将建立一个简单的一元Logit回归模型，即以“考试成绩”作为自变量，预测二元“评价”因变量。

![](img/9bae42176624c3769cb4cc0ac01ce315_3.png)

我们使用`statsmodels`库中的`GLM`（广义线性模型）方法进行拟合。模型的基本形式可以表示为以下公式：

**公式：** `logit(p) = ln(p/(1-p)) = β₀ + β₁ * 考试成绩`

其中，`p` 表示“评价为好”的概率。

![](img/9bae42176624c3769cb4cc0ac01ce315_5.png)

以下是建立和拟合模型的代码：

```python
import statsmodels.api as sm

# 定义因变量和自变量
y = df['评价_二元']
X = sm.add_constant(df['考试成绩'])  # 添加常数项（截距）

# 建立Logit模型（使用二项式族和Logit连接函数）
model = sm.GLM(y, X, family=sm.families.Binomial(link=sm.families.links.Logit()))
result = model.fit()

# 查看模型摘要
print(result.summary())
```

模型摘要会提供丰富的信息，帮助我们判断模型效果。

---

## 解读回归结果

运行模型后，我们将获得详细的统计结果。理解这些结果是分析的关键。

模型摘要主要包含以下部分：
*   **模型信息：** 显示使用的模型类型（GLM）、因变量名称、观测值数量（No. Observations）和自由度（Df Residuals）。
*   **拟合优度：** 例如伪R平方（Pseudo R-sq.），值越接近1，说明模型对数据的解释力越强。
*   **系数表：** 这是核心部分，列出了每个自变量的回归系数（coef）、标准误（std err）、z统计量（z）和P值（P>|z|）。

以下是解读系数表的要点：
*   **截距项（const）：** 表示当所有自变量为0时，结果的对数几率（log-odds）。其显著性通常不是关注重点。
*   **自变量系数：** 例如“考试成绩”的系数。如果其P值（P>|z|）小于0.05，通常认为该变量影响显著。
    *   系数为正，意味着该自变量取值增加会提高因变量为1（好评）的**对数几率**。
    *   系数为负，则意味着会降低该几率。

在我们的示例中，“考试成绩”的系数显著为正，说明考试分数越高，获得好评的概率越大。

---

## 计算优势比（OR值）

为了更直观地解释自变量对结果概率的影响，我们通常会将回归系数转换为优势比。

**优势比（Odds Ratio, OR）** 表示自变量每增加一个单位，因变量为1的“优势”（odds）变为原来的多少倍。其计算公式为：

**公式：** `OR = exp(系数)`

以下是计算OR值的代码：

```python
# 计算OR值及其置信区间
import numpy as np
odds_ratios = np.exp(result.params)  # 对系数取指数
conf = result.conf_int()  # 获取系数的置信区间
conf_odds = np.exp(conf)  # 对置信区间取指数

![](img/9bae42176624c3769cb4cc0ac01ce315_7.png)

# 创建一个包含OR值和置信区间的DataFrame
or_summary = pd.DataFrame({
    'OR': odds_ratios,
    '2.5%': conf_odds[0],
    '97.5%': conf_odds[1]
})
print(or_summary)
```

解读OR值：
*   **OR > 1**：自变量对结果有正向影响（如示例中考试成绩的OR值为1.01）。
*   **OR < 1**：自变量对结果有负向影响。
*   **OR = 1**：自变量没有影响。
如果OR值的置信区间不包含1，则说明该影响在统计上是显著的。

## 多元Logit回归与模型检验

在实际分析中，我们常常需要纳入多个自变量。本节我们来看看如何进行多元Logit回归并检验模型整体效果。

我们可以轻松地将更多自变量加入模型。例如，同时用“考试成绩”和“专业”（如管理学、金融学）来预测“评价”。

以下是多元Logit回归的代码：

```python
# 为分类变量‘专业’创建虚拟变量（哑变量）
df_with_dummies = pd.get_dummies(df, columns=['专业'], drop_first=True)

# 定义新的自变量集，包含考试成绩和专业虚拟变量
X_multi = sm.add_constant(df_with_dummies[['考试成绩', '专业_管理学', '专业_金融学']])

# 建立并拟合多元Logit模型
model_multi = sm.GLM(y, X_multi, family=sm.families.Binomial(link=sm.families.links.Logit()))
result_multi = model_multi.fit()
print(result_multi.summary())
```

在结果中，我们可以分别查看每个自变量的显著性和影响方向。

**模型整体检验：**
我们可以使用似然比检验（Likelihood Ratio Test）来比较当前模型与一个只有截距项的“空模型”，从而判断加入的自变量是否显著改善了模型拟合效果。这通常通过查看模型摘要中的**似然比统计量（LLR p-value）** 来实现。如果该P值显著（如<0.05），则说明模型整体是有效的。

## 总结

本节课中，我们一起学习了使用Python进行Logit回归分析的完整流程。

1.  **数据准备：** 将分类因变量转换为0/1二元变量。
2.  **模型建立：** 使用`statsmodels`库的`GLM`函数，指定二项分布族和Logit连接函数来拟合模型。
3.  **结果解读：** 关注回归系数的符号、大小和统计显著性（P值）。
4.  **优势比转换：** 通过计算`exp(系数)`得到OR值，以便更直观地解释自变量影响。
5.  **多元回归与检验：** 可以纳入多个自变量，并通过似然比检验评估模型整体有效性。

Logit回归是处理二元分类因变量的强大工具，在社会科学、医学、金融等领域有广泛应用。通过本讲的学习，你已经掌握了其基本原理和Python实现方法。
