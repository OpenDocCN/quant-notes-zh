![](img/cc7ae9cb28f450b823e2a0ca1eb8c884_0.png)

## 概述
在本节课中，我们将学习线性回归模型中的核心组成部分：误差项。我们将探讨如何将回归方程转化为矩阵形式，并理解误差项在机器学习模型训练中的意义，它是定义模型“好坏”的关键。

## 回归方程的矩阵化表示
上一节我们介绍了线性回归的基本思想，即用一个平面（或直线）来拟合数据点。本节中我们来看看如何用数学公式精确地描述这个拟合平面。

我们构建的线性回归方程通常写作：
`h_θ(x) = θ₀ + θ₁x₁ + θ₂x₂`
其中：
*   `x₁, x₂` 是特征，例如工资和年龄。
*   `θ₁, θ₂` 被称为**权重项**，它们决定了每个特征对最终结果的影响程度。
*   `θ₀` 被称为**偏置项**。它的作用是**对模型的整体输出进行一个微调**，使拟合的平面能够整体向上或向下平移，以更好地匹配数据。它并非核心影响因素，而是一个调整项。

为了便于后续的矩阵运算，我们需要对这个方程进行改写。观察上式，`θ₁x₁` 和 `θ₂x₂` 都是“权重×特征”的形式，但 `θ₀` 缺少与之相乘的特征变量。

因此，我们引入一个新的特征 `x₀`，并**人为地将其值全部设为1**。这样，原方程就可以改写为：
`h_θ(x) = θ₀x₀ + θ₁x₁ + θ₂x₂`，其中 `x₀ ≡ 1`。

现在，我们可以用简洁的矩阵形式来表示整个方程：
`h_θ(x) = θᵀX`
这里，`θ` 是包含所有权重和偏置的参数向量 `[θ₀, θ₁, θ₂]`，`X` 是添加了常数列 `x₀` 后的特征矩阵。这种形式极大地方便了后续的计算。

## 误差项的定义与意义
有了拟合平面（即预测方程 `h_θ(x)`）后，我们如何衡量它对单个数据点的拟合程度呢？我们来看一个具体的数据点。

![](img/cc7ae9cb28f450b823e2a0ca1eb8c884_4.png)

![](img/cc7ae9cb28f450b823e2a0ca1eb8c884_6.png)

对于一个给定的样本 `i`，我们知道它的特征 `(x₁⁽ⁱ⁾, x₂⁽ⁱ⁾)` 和对应的**真实值** `y⁽ⁱ⁾`（例如，银行历史数据中实际批准的贷款额度）。同时，我们可以用我们的回归方程计算出一个**预测值** `ŷ⁽ⁱ⁾ = h_θ(x⁽ⁱ⁾)`。

![](img/cc7ae9cb28f450b823e2a0ca1eb8c884_8.png)

![](img/cc7ae9cb28f450b823e2a0ca1eb8c884_10.png)

在二维平面上，真实值 `y⁽ⁱ⁾` 和预测值 `ŷ⁽ⁱ⁾` 之间通常存在一个垂直距离。这个距离就是**误差项**，记作 `ε⁽ⁱ⁾` 或 `error⁽ⁱ⁾`。它们的关系可以表示为：
`y⁽ⁱ⁾ = ŷ⁽ⁱ⁾ + ε⁽ⁱ⁾`
或者
`ε⁽ⁱ⁾ = y⁽ⁱ⁾ - ŷ⁽ⁱ⁾`

以下是关于误差项的关键点：
*   每个样本都有自己的误差项。
*   误差可正可负，代表了预测值是高估还是低估了真实值。
*   我们的目标是让所有样本的误差尽可能小。

## 从误差到损失函数
机器学习可以理解为：给机器提供数据（Data），并指定一个学习目标（Goal），机器通过调整模型参数来逼近这个目标。这个目标通常通过**损失函数**（或称目标函数、代价函数）来定义。

误差项直接引出了最自然的损失函数思想：既然我们希望预测准确，那么就应该**让所有样本的误差总和尽可能小**。一个最直观的想法是让所有误差项的和最小化，即 Minimize(Σ|ε⁽ⁱ⁾|)。但在实际中，为了数学处理的便利（例如便于求导），我们更常使用**误差的平方和**作为损失函数 `J(θ)`：
`J(θ) = (1/2m) * Σ (y⁽ⁱ⁾ - ŷ⁽ⁱ⁾)² = (1/2m) * Σ (ε⁽ⁱ⁾)²`
其中 `m` 是样本数量。乘以 `1/2` 是为了后续求导时简化计算。

**损失函数的核心意义在于**：它量化了模型在整个数据集上的预测“糟糕”程度。`J(θ)` 的值越小，说明所有预测值 `ŷ` 与真实值 `y` 的整体差距越小，模型拟合得越好。当 `J(θ)` 接近于0时，意味着模型近乎完美地拟合了所有数据。

因此，机器学习（对于线性回归）的任务，就转化为了一个优化问题：**找到一组参数 `θ`，使得损失函数 `J(θ)` 的值达到最小。**

## 总结
本节课中我们一起学习了线性回归模型数学化的关键步骤：
1.  我们将回归方程 `h_θ(x) = θ₀ + θ₁x₁ + θ₂x₂` 通过添加常数值特征 `x₀=1`，转化为矩阵形式 `h_θ(x) = θᵀX`，这便于计算机进行高效运算。
2.  我们定义了**误差项** `ε⁽ⁱ⁾ = y⁽ⁱ⁾ - ŷ⁽ⁱ⁾`，它衡量了模型对单个样本的预测偏差。
3.  我们引入了**损失函数**（以误差平方和为例）`J(θ)` 的概念。它将所有样本的误差汇总为一个标量值，作为评估模型整体性能和学习目标的准则。模型训练的本质就是寻找最小化损失函数的参数 `θ`。
