# 缠论Python量化框架：P1：框架介绍与核心概念 🚀

在本节课中，我们将学习一个基于缠论的Python量化框架。该框架能够将缠论中的买卖点、笔、中枢、线段等概念进行算法化处理，并应用于币圈的实盘交易。我们将介绍其基本构成和核心概念。

## 框架概述

我开发了一个缠论量化框架。该框架能够逐个K线地计算缠论中的买卖点。框架中的笔、中枢、线段等核心算法均可由用户自定义。本视频展示的仅是一个小型演示案例。

## 核心概念与算法

上一节我们介绍了框架的整体目标，本节中我们来看看其实现所依赖的几个核心概念。这些概念是缠论分析的基础，均通过代码逻辑进行定义。

以下是框架中涉及的几个关键算法对象：

*   **买卖点**：系统通过分析K线序列，自动识别缠论定义的各类买卖点。其判断逻辑可封装为函数，例如 `identify_trading_points(kline_data)`。
*   **笔**：笔是缠论中连接相邻顶分型和底分型的基本结构。算法需要识别顶底分型并确认笔的成立，其流程可简述为：`find_fractals() -> validate_pen()`。
*   **中枢**：由至少三个连续次级别走势类型重叠部分所构成。在框架中，中枢可通过识别特定区间内价格的重叠部分来算法化定义，核心公式为 `ZG = min(highs_of_swings)`， `ZD = max(lows_of_swings)`。
*   **线段**：由至少三笔组成，且前三笔必须有重叠。线段的识别建立在笔的序列之上，算法需要判断笔的破坏与延续。

## 应用场景：币圈实盘

这个框架的设计目标之一是应用于数字货币市场的实盘交易。当前，比特币可能正迎来新一轮牛市行情，为趋势交易策略提供了潜在的机会。通过将缠论的分析逻辑量化并自动化，交易者可以尝试更系统性地捕捉市场中的结构型买卖点。
