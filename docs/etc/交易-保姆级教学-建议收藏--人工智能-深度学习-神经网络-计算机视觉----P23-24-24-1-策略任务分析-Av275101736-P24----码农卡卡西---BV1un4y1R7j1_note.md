## 策略概述

![](img/6e09290a57e46914685ab67deb9262bb_5.png)

我们的目标是创建一个策略，该策略始终持有沪深300指数中表现最好的十只股票。具体来说，我们将每天（或每个交易周期）根据某个财务指标（例如盈利能力）对沪深300的所有成分股进行排序，然后买入排名前十的股票，并卖出手中不再属于前十的股票。这样，我们的投资组合将动态地保持为当前“最好”的十只股票。

## 策略实现步骤分解

上一节我们介绍了策略的整体目标，本节中我们来看看实现这个策略需要哪些具体的步骤。

### 第一步：初始化策略

在策略的初始化函数中，我们需要完成基础设置。以下是初始化阶段需要完成的工作：

*   **设置股票池**：获取沪深300指数的所有成分股，作为我们后续筛选的基础股票池。
*   **其他初始化工作**：例如，可以在这里设置策略参数、初始化变量等。

### 第二步：每日开盘前处理

在`before_trading`函数中，我们需要在每个交易日开始前，执行数据获取和计算工作。这些操作与实际交易下单无关，主要是为当天的交易决策做准备。以下是需要在此环节完成的任务：

*   **获取财务数据**：从数据接口获取股票池中所有股票我们关心的财务指标（例如，每股收益）。
*   **排序与筛选**：根据获取的财务指标对股票进行降序排序，并选取排名前十的股票代码，作为当日的“目标持仓列表”。

### 第三步：盘中交易处理

在`handle_bar`函数中，我们将执行实际的交易逻辑。这里会根据`before_trading`阶段计算出的“目标持仓列表”来调整我们实际的投资组合。以下是核心的交易逻辑：

*   **获取当前持仓**：查询交易账户，获取当前实际持有的所有股票及其数量。
*   **对比目标持仓**：将当前实际持仓与当日的“目标持仓列表”进行对比。
*   **执行调仓操作**：
    *   卖出当前持有但不在“目标持仓列表”中的股票。
    *   买入在“目标持仓列表”中但当前并未持有的股票（使用可用资金进行购买）。

## 核心概念与代码示意

为了让思路更清晰，我们可以用伪代码来描述上述逻辑框架：

```python
def initialize(context):
    # 初始化：设置股票池为沪深300成分股
    context.stock_pool = get_index_stocks('000300.SH')

def before_trading(context):
    # 每日开盘前：获取数据并计算目标股票
    # 1. 获取股票池中所有股票的财务数据（例如：eps）
    financial_data = get_financial_data(context.stock_pool, indicator='eps')
    # 2. 根据财务数据排序，取前十名
    sorted_stocks = sorted(financial_data.items(), key=lambda x: x[1], reverse=True)
    context.target_stocks = [stock for stock, value in sorted_stocks[:10]]

def handle_bar(context, bar_dict):
    # 盘中处理：执行交易
    # 1. 获取当前持仓
    current_positions = get_current_positions()
    # 2. 卖出不在目标列表中的股票
    for stock in current_positions:
        if stock not in context.target_stocks:
            order_target_value(stock, 0) # 卖出该股票至0股
    # 3. 买入目标列表中的股票（等权重分配资金）
    available_cash = context.portfolio.available_cash
    cash_per_stock = available_cash / len(context.target_stocks)
    for stock in context.target_stocks:
        order_target_value(stock, cash_per_stock) # 买入该股票至目标金额
```
