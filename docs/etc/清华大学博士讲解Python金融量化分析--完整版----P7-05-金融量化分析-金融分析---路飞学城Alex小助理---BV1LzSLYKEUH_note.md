# Python金融量化分析：P7：05 金融量化分析-金融分析 📈

在本节课中，我们将要学习金融分析的核心方法。金融分析是判断股票买卖时机、预测价格走势的关键手段，避免盲目投资。我们将介绍两种主要的分析方法：基本面分析和技术面分析。

上一节我们介绍了金融和股票的基础知识，本节中我们来看看如何进行具体的金融分析。

## 基本面分析

基本面分析的核心是评估公司的内在价值，通过研究影响股价的公司自身因素、行业状况和宏观经济环境来判断投资价值。它主要包含以下三个方面：

以下是基本面分析的三个层面：

1.  **宏观经济面分析**：分析国家的财政政策、货币政策等宏观因素，判断整体经济环境对股市的影响。例如，判断政策是鼓励资金流入股市还是存入银行。但需注意，宏观经济规律并非总是与股市表现一致。
2.  **行业分析**：评估特定行业的发展前景和整体状况。例如，分析教育、IT、钢铁等行业的发展趋势。可以通过观察该行业内几只代表性股票的走势来辅助判断。
3.  **公司分析**：这是最具体的一环，直接分析目标上市公司的运营和财务状况。上市公司需定期公开经审计的财务报告（财报）。投资者通过分析财报数据（如利润、每股收益）、关注公司新闻甚至实地考察，来判断公司经营是否良好、是否具有投资价值。一种策略是，如果提前预判公司业绩向好，可在财报发布前买入，待利好财报公布后股价上涨时获利。

## 技术面分析

![](img/4f0937960690e77785b7933f19169b7b_1.png)

技术面分析的核心观点是：**所有信息都已反映在市场价格和交易数据中**。它不关注公司内在价值，而是通过研究历史市场数据（如价格、成交量）的图表和指标，来预测未来价格走势。

以下是技术面分析中两个基础且重要的指标：

### K线图

![](img/4f0937960690e77785b7933f19169b7b_3.png)

![](img/4f0937960690e77785b7933f19169b7b_5.png)

K线图是展示股票每日价格变动的图表。一根K线包含四个关键价格：**开盘价(Open)**、**收盘价(Close)**、**最高价(High)**、**最低价(Low)**。

K线分为阳线和阴线：
*   **阳线**（通常为红色或空心）：表示当日股价上涨，即收盘价高于开盘价。
*   **阴线**（通常为绿色或实心）：表示当日股价下跌，即收盘价低于开盘价。

一根标准K线的构成如下：
*   **实体**：矩形部分，表示开盘价与收盘价之间的价格区间。
*   **上影线**：实体上方的细线，顶端表示当日最高价。
*   **下影线**：实体下方的细线，底端表示当日最低价。

**公式/规则**：
*   对于**阳线**：实体**下沿**为开盘价，实体**上沿**为收盘价。
*   对于**阴线**：实体**上沿**为开盘价，实体**下沿**为收盘价。

特殊形态的K线（如十字星、光头光脚线等）也具有特定的市场含义。

### 移动平均线 (MA)

移动平均线是通过计算过去一段时间内股价的平均值，并将这些平均值连接起来形成的曲线。它用于平滑价格波动，反映股价的趋势。

常见的均线有5日均线(MA5)、60日均线(MA60)等。
*   **MA5**：过去5个交易日收盘价的平均值。
*   **MA60**：过去60个交易日收盘价的平均值。

![](img/4f0937960690e77785b7933f19169b7b_7.png)

![](img/4f0937960690e77785b7933f19169b7b_9.png)

**公式**：
`MA(N) = (P1 + P2 + ... + PN) / N`
其中，`P1` 代表今日收盘价，`P2` 代表昨日收盘价，以此类推，`N` 为周期数。

在图表中，均线（如白色MA5线和黄色MA60线）能帮助判断短期和长期趋势。例如，短期均线上穿长期均线可能被视为买入信号（金叉），反之则为卖出信号（死叉）。这构成了后续会讲到的“双均线策略”基础。

本节课中我们一起学习了金融分析的两大支柱。**基本面分析**帮助我们从公司价值、行业和宏观经济的角度判断股票的长期投资价值；**技术面分析**则教会我们通过解读**K线图**和**移动平均线(MA)** 等市场图表与指标，来捕捉交易时机和趋势。理解这两种方法，是进行理性投资和后续量化策略开发的重要基础。
