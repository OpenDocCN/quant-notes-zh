## 调试利器：%pdb命令 🐛

上一节我们介绍了IPython的魔术命令，本节中我们来看看一个用于调试的强大工具：`%pdb`命令。在编写代码时，我们经常会遇到报错，但难以定位具体是哪一行出了问题。`%pdb`命令可以自动在报错前暂停代码执行，进入交互式调试模式。

![](img/29ae2daa4a51c672bd4567cd117666fe_5.png)

`%pdb`是一个开关性质的命令：
*   `%pdb on`：开启自动调试模式。
*   `%pdb off`：关闭自动调试模式。

开启后，当运行代码遇到错误时，IPython会在出错的那一行之前自动暂停，并进入调试器（pdb）。此时，你可以使用pdb命令查看当前变量的状态，帮助定位问题。

![](img/29ae2daa4a51c672bd4567cd117666fe_7.png)

例如，我们有一段会引发除零错误的测试代码：
```python
def test(a, b):
    c = a / b  # 这一行会报错
    return c

![](img/29ae2daa4a51c672bd4567cd117666fe_9.png)

![](img/29ae2daa4a51c672bd4567cd117666fe_11.png)

test(3, 0)
```
在`%pdb on`状态下运行此代码，调试器会在执行`c = a / b`之前暂停。此时，可以使用`p`命令打印变量值进行排查：
```
p a  # 输出: 3
p b  # 输出: 0
```
通过检查，可以立刻发现`b`的值为0，导致了除零错误。

![](img/29ae2daa4a51c672bd4567cd117666fe_13.png)

以下是pdb调试器中一些常用的命令：
*   `h`：查看帮助文档。
*   `q`：退出调试器。
*   `n`：执行下一行代码。
*   `break`：设置断点。

![](img/29ae2daa4a51c672bd4567cd117666fe_15.png)

对于简单的错误排查，通常使用`p`命令查看变量值就足够了。不需要调试时，记得使用`%pdb off`关闭此功能。

## 便捷操作：命令历史与输入输出 🔄

IPython提供了类似Linux命令行的历史功能，并能方便地获取之前的输入和输出结果，这能节省大量重复输入的时间。

**使用命令历史**
在IPython中，按`上箭头`键可以获取上一条命令。更强大的是，你可以输入命令的开头几个字符，再按`上箭头`，IPython会搜索并显示以这些字符开头的历史命令。例如，输入`a*`再按`上箭头`，可能会找到之前输入过的`a * b`这条命令。

**获取输出结果**
在IPython中，如果你运行了一个表达式但没有将结果赋值给变量，结果会被显示并临时保存。你可以使用下划线`_`来引用上一个输出结果。
```python
a = 1
b = 2
a + b  # 输出: 3
_ * 10 # 输出: 30， 这里的 _ 代表了上一行的结果 3
```
你甚至可以用`_N`（N为行号）来引用特定行的输出，例如`_69`引用第69行的输出。

**获取输入内容**
除了输出，你还可以获取之前的输入内容。使用`_iN`（N为行号）可以获取对应行的输入字符串。例如，`_i72`会返回第72行输入的代码字符串。这个功能在需要重新查看或修改之前输入的代码时很有用。

## 高效导航：目录标签系统 (bookmark) 📂

当需要在多个项目目录间频繁切换时，反复输入冗长的`cd`路径非常麻烦。IPython的`%bookmark`魔术命令可以创建目录的快捷方式（书签）。

以下是`%bookmark`命令的用法：
*   **添加书签**：`%bookmark 书签名 目录路径`
    *   例如：`%bookmark proj /home/user/my_project`
*   **查看所有书签**：`%bookmark -l`
*   **使用书签切换目录**：`cd 书签名`
    *   例如：`cd proj` 即可快速跳转到 `/home/user/my_project`
*   **删除单个书签**：`%bookmark -d 书签名`
*   **删除所有书签**：`%bookmark -r`

![](img/29ae2daa4a51c672bd4567cd117666fe_21.png)

这个功能让项目管理变得非常高效。

## 强大的Web界面：Jupyter Notebook 🌐

![](img/29ae2daa4a51c672bd4567cd117666fe_27.png)

最后，我们介绍一个由IPython衍生出的强大工具：Jupyter Notebook。它是一个基于Web的交互式计算环境，特别适合进行数据清洗、统计建模、可视化以及机器学习等探索性工作。

要使用Jupyter Notebook，需要先安装`jupyter`模块：
```
pip install jupyter
```
安装完成后，在系统命令行中输入`jupyter notebook`，它会自动在浏览器中打开一个本地服务器页面。

在Jupyter Notebook中，你可以创建“笔记本”（Notebook）文件。它允许你混合编写：
1.  **可执行的代码单元**：编写并运行Python代码，结果（包括图表）会直接显示在代码下方。
2.  **富文本单元**：使用Markdown语法编写带格式的文本、数学公式等。

这使得Jupyter Notebook非常适合用于：
*   **数据分析与可视化**：逐步执行代码并即时查看图表结果。
*   **教学与演示**：将代码、运行结果和说明文字完美结合。
*   **创作技术博客**：可以轻松地将笔记本导出为HTML、PDF等多种格式。

它本质上是一个Web版的、功能增强的IPython，是数据科学领域的标准工具之一。
