## 概述

期权是一种赋予持有者在未来特定时间以约定价格买卖资产权利的金融工具。它像一把双刃剑，用得好可以增强收益、对冲风险，用不好则可能带来巨大亏损。

本期教程将分享一个由期权数据构建的市场情绪指标——认沽认购比率（PCR），并演示如何将其转化为一个可操作的量化择时策略。

## 期权基础知识

![](img/3efaea55da7c734aa3928fe4d7d91ff1_5.png)

上一节我们提到了期权的基本概念，本节中我们来具体看看它的定义和分类。

期权（Option）的本质是选择权。它赋予持有者在未来特定时间或之前，以预先约定的价格（行权价）购买或出售某种资产的权利。既然是权利，持有者到期时可以选择执行，也可以选择不执行。

**公式**：期权 = 在未来以约定价格买卖资产的权利

![](img/3efaea55da7c734aa3928fe4d7d91ff1_7.png)

例如，交易者A支付100元给交易者B，约定在2024年底以每股70万元的价格购买100股伯克希尔·哈撒韦的股票。到了2024年底：
*   如果市价低于70万元，交易者A可以选择不执行权利，转而从市场直接购买更便宜的股票。交易者B则白赚100元权利金。
*   如果市价高于70万元（例如80万元），交易者A可以执行权利，以70万元的约定价格购买股票，从而获利。

在上述例子中，交易者A**买入**了一个**看涨期权**（认购期权），交易者B**卖出**了一个看涨期权。同理，如果预期资产价格下跌，则可以交易**看跌期权**（认沽期权）。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_9.png)

因此，期权交易的四个基本方向是：**买入看涨、卖出看涨、买入看跌、卖出看跌**。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_11.png)

中国A股的场内期权起步较晚。2015年2月9日，上证50ETF期权在上海证券交易所上市，实现了场内期权零的突破。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_13.png)

以下是期权合约名称的一个示例：
`50ETF购5月3300`
*   `50ETF`：标的资产是上证50ETF。
*   `购`：代表认购（看涨）期权。
*   `5月`：代表到期月份是5月。
*   `3300`：代表行权价为3.300元。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_15.png)

交易所一般会将期权按到期月份分类，其中当月期权的成交量通常最大，市场影响力也较强。我们后续构建的策略也将基于当月期权数据。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_17.png)

## 市场情绪指标：认沽认购比率（PCR）

![](img/3efaea55da7c734aa3928fe4d7d91ff1_23.png)

![](img/3efaea55da7c734aa3928fe4d7d91ff1_25.png)

了解期权基础后，我们来看看如何利用它来洞察市场情绪。市场上有许多衡量情绪的指标，如融资融券余额、新增投资者数量等。

在众多指标中，我们将重点介绍**期权认沽认购比率**，即 **PCR指标**。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_27.png)

**公式**：`PCR = 认沽期权成交量 / 认购期权成交量`

我们通常使用最近5个交易日的期权成交量来计算PCR。从公式可以看出：
*   **PCR值越大**，说明看跌（认沽）的成交量相对看涨（认购）越多，市场情绪偏向悲观。
*   **PCR值越小**，说明看涨的成交量相对看跌越多，市场情绪偏向乐观。

然而，金融市场中常常存在“物极必反”的现象。当市场情绪极度乐观时，可能意味着行情接近顶部；当情绪极度悲观时，可能意味着底部临近。因此，PCR指标或许可以作为一个逆向操作的参考。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_29.png)

当然，这仅是经验之谈。我们需要借助历史数据和Python代码进行量化验证。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_31.png)

## 构建量化择时策略

![](img/3efaea55da7c734aa3928fe4d7d91ff1_43.png)

仅仅观察指标有些浪费，一个情绪指标最好的归宿是用来构建可执行的量化择时策略。本节我们将基于PCR指标排名构建一个简单的策略。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_45.png)

![](img/3efaea55da7c734aa3928fe4d7d91ff1_47.png)

策略逻辑如下：
1.  **信号生成**：在每个交易日收盘后，计算最近5个交易日的PCR值。然后将最近60个交易日的PCR值从大到小排序。如果当日的PCR排名处于前30%（即排名<=18），则生成买入信号。
2.  **交易执行**：在产生买入信号的下一个交易日开盘买入上证50ETF。一直持有，直到某一天的PCR排名跌出前30%，则在下一个交易日开盘全部卖出。
3.  **循环操作**：卖出后，继续监控PCR排名，等待其再次进入前30%时，重复买入操作。

这个策略的核心是 **“市场悲观时买入，情绪修复后持有，情绪过热前卖出”**。

我们可以使用Python（或Excel）对这个策略进行回测，计算其历史收益、年化收益率、最大回撤等关键绩效指标。

回测结果显示，该策略能显著跑赢上证50ETF指数本身，获得更高的年化收益和更小的最大回撤。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_49.png)

## 策略进阶：加入做空机制

![](img/3efaea55da7c734aa3928fe4d7d91ff1_53.png)

基础策略只涉及做多，对于熟悉股指期货或期权做空机制的投资者，可以尝试改进策略以增厚收益。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_55.png)

改进策略逻辑如下：
在原有条件不变的基础上，增加一个做空条件：
*   当PCR值排名处于**后30%**（即排名>=43）时，在下一交易日开盘**做空**上证50ETF（可通过股指期货、期权等工具实现）。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_57.png)

这样，策略将在市场极度乐观时做空，在市场极度悲观时做多，试图实现双向获利。

![](img/3efaea55da7c734aa3928fe4d7d91ff1_59.png)

回测结果显示，加入做空机制后，策略的总收益和年化收益率得到进一步提升，但同时波动和最大回撤也会相应放大。

> **警告**：做空交易需要相应的知识、经验和资金门槛，不建议新手直接尝试。

## 总结

本节课中我们一起学习了：
1.  **期权基础**：期权是一种未来买卖资产的权利，分为认购（看涨）和认沽（看跌）两大类。
2.  **情绪指标**：认沽认购比率（PCR）是衡量期权市场情绪的有效指标，其公式为 `PCR = 认沽成交量 / 认购成交量`。
3.  **逆向思维**：市场情绪极度悲观（PCR高位）时可能是买入时机，极度乐观（PCR低位）时可能是卖出或做空时机。
4.  **策略构建**：我们基于PCR指标在历史数据中的排名，构建了一个“买入-持有-卖出”的择时策略，并展示了其回测效果。
5.  **策略进阶**：对于有经验的投资者，可以在策略中加入做空机制，以在牛市和熊市中均寻找机会。

期权是一个充满研究价值的工具，而市场情绪是一面镜子。用得好，它们可以帮助我们构建系统化的交易策略，辅助投资决策；用不好，则可能陷入追涨杀跌的陷阱。希望本教程的指标和策略思路能为你提供有益的启发。
