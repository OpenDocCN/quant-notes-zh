## 概述

量化回测是验证交易策略有效性的关键步骤。它通过历史数据模拟交易，来评估策略的潜在盈利能力和风险。本节将解析一个用Python编写的简单回测框架，该框架基于“头七战法”策略逻辑，帮助初学者理解回测的基本构成。

## 策略逻辑详解：“头七战法”

“头七战法”是本回测框架所使用的策略。其核心思想是寻找短期内经历大幅回调，但中期趋势曾表现强劲的股票，试图在回调低位买入。

以下是该策略的两个核心筛选条件：

1.  **近期深幅回调**：在最近的7个交易日内，股价从期间最高点下跌幅度**≥30%**。公式表示为：`(Max(Close, 7) - Current_Close) / Max(Close, 7) >= 0.3`。
2.  **中期趋势强劲**：在触发条件1的日期再往前推20个交易日（即第8到第27个交易日），股价涨幅**≥30%**。公式表示为：`(Close[-27] - Close[-20]) / Close[-20] >= 0.3`。

当一只股票同时满足以上两个条件时，则产生买入信号。

## 交易与风控规则

在产生买入信号后，需要明确的交易规则来执行模拟。A股市场实行T+1制度，即当日买入的股票需到下一个交易日才能卖出。

以下是具体的交易执行与风险控制规则：

*   **买入**：在信号出现日的**下一个交易日**，以该日开盘价买入。
*   **持有期**：买入后持有**10个交易日**。
*   **止盈**：持有期内，若股价较买入价上涨**10%**，则立即卖出。
*   **止损**：持有期内，若股价较买入价下跌**10%**，则立即卖出。
*   **强制平仓**：若持有满10个交易日，仍未触发止盈或止损条件，则在第10个交易日的**14:30**卖出。

## 回测结果分析

我们通过代码对历史数据进行回测，并统计交易结果。绩效统计主要包括胜率（盈利交易笔数占总交易笔数的比例）。

例如，选择2023年10月15日作为信号筛选日，并在10月16日执行买入。回测结果显示，在19笔模拟交易中，胜率达到90%。这表明在该市场阶段（大盘横盘或下跌后），此策略有较高的盈利概率。

然而，策略表现与大盘环境高度相关。例如，在2023年12月26日（市场单边下跌阶段）执行同样的策略，胜率则下降至20%左右。这清晰地说明了**没有放之四海而皆准的策略**，市场环境对策略效果有决定性影响。

## 策略优化方向

![](img/0a5021733c8617b137085a02766e7069_7.png)

认识到基础策略的局限性后，我们可以探讨一些可能的优化方向，以提升策略的稳健性和适应性。

以下是一些潜在的优化思路：

1.  **加入大盘风控**：仅在市场整体处于特定趋势（如非单边下跌）时启用该策略。
2.  **考虑板块热度**：优先选择当前市场热点板块内的股票，利用资金聚集效应。
3.  **分析成交量**：在买入点观察成交量是否萎缩至低位，这可能是抛压减轻、即将变盘的信号，可提高买入成功率。
