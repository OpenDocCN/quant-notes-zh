## 概述

![](img/7fdf5247a1e73717856ae6adf06504e4_7.png)

九九乘法表是一个九行九列的表格，其规律是：第 `i` 行打印 `i` 个乘法算式。例如，第三行打印 `3×1=3`、`3×2=6`、`3×3=9`。要实现这个效果，我们需要使用一个**外层循环**来控制行数，并在每一行内部使用一个**内层循环**来控制该行打印的算式数量。

![](img/7fdf5247a1e73717856ae6adf06504e4_9.png)

上一节我们介绍了循环的基本概念，本节中我们来看看如何将循环嵌套起来解决实际问题。

## 核心思路分析

![](img/7fdf5247a1e73717856ae6adf06504e4_15.png)

![](img/7fdf5247a1e73717856ae6adf06504e4_17.png)

观察九九乘法表的结构，我们可以总结出两个关键点：
1.  总共需要打印 **9 行**。
2.  对于第 `i` 行，需要打印 **i 个** 算式，算式内容为 `i × j = i*j`，其中 `j` 从 1 递增到 `i`。

![](img/7fdf5247a1e73717856ae6adf06504e4_19.png)

这清晰地指向了**嵌套循环**的解决方案：外层循环变量 `i` 控制行数，内层循环变量 `j` 控制当前行打印的列数。

## 分步实现

以下是实现九九乘法表的具体步骤。

![](img/7fdf5247a1e73717856ae6adf06504e4_25.png)

![](img/7fdf5247a1e73717856ae6adf06504e4_27.png)

### 第一步：打印基础列

![](img/7fdf5247a1e73717856ae6adf06504e4_29.png)

![](img/7fdf5247a1e73717856ae6adf06504e4_31.png)

![](img/7fdf5247a1e73717856ae6adf06504e4_33.png)

首先，我们使用一个循环打印出乘法表的第一列（即所有 `i × 1` 的算式）。这帮助我们确认外层循环的逻辑。

![](img/7fdf5247a1e73717856ae6adf06504e4_35.png)

![](img/7fdf5247a1e73717856ae6adf06504e4_37.png)

```python
for i in range(1, 10):
    print(f"{i} × 1 = {i*1}")
```
**代码解释**：`range(1, 10)` 生成从 1 到 9 的数字。每次循环，打印当前数字 `i` 乘以 1 的结果。

![](img/7fdf5247a1e73717856ae6adf06504e4_39.png)

![](img/7fdf5247a1e73717856ae6adf06504e4_41.png)

### 第二步：引入内层循环

![](img/7fdf5247a1e73717856ae6adf06504e4_43.png)

接下来，我们需要在每一行（外层循环的每一次迭代）内部，根据行号 `i` 进行多次打印。这需要引入内层循环。

![](img/7fdf5247a1e73717856ae6adf06504e4_45.png)

```python
for i in range(1, 10):          # 外层循环，控制行数 i
    for j in range(1, i+1):     # 内层循环，控制每行的列数 j
        print(j)                # 暂时打印 j，观察循环次数
```
**代码解释**：内层循环 `for j in range(1, i+1)` 表示在第 `i` 行，`j` 的值会从 1 循环到 `i`。例如，当 `i=3` 时，`j` 会依次取值 1, 2, 3，正好对应第三行的三个算式。

![](img/7fdf5247a1e73717856ae6adf06504e4_47.png)

### 第三步：组合成完整算式

![](img/7fdf5247a1e73717856ae6adf06504e4_49.png)

![](img/7fdf5247a1e73717856ae6adf06504e4_51.png)

现在，我们将内外层循环的变量 `i` 和 `j` 组合起来，打印出完整的乘法算式 `i × j = i*j`。

```python
for i in range(1, 10):
    for j in range(1, i+1):
        print(f"{i} × {j} = {i*j}")
```
运行这段代码，你会发现算式已经全部正确，但每个算式都单独占一行，没有形成九行的表格结构。

### 第四步：控制打印格式

我们的目标是让同一行的所有算式打印在一行内，不同行之间换行。这需要控制 `print` 函数的行为。
*   `print` 函数默认以换行符 `\n` 结尾，所以每次调用都会换行。
*   通过设置参数 `end=' '`，可以让 `print` 以空格结尾而不换行。
*   内层循环结束后，我们需要手动打印一个空行（即调用一个无内容的 `print()`）来实现换行。

![](img/7fdf5247a1e73717856ae6adf06504e4_53.png)

以下是最终代码：

![](img/7fdf5247a1e73717856ae6adf06504e4_55.png)

```python
for i in range(1, 10):
    for j in range(1, i+1):
        print(f"{i}×{j}={i*j}", end=' ')  # 以空格结尾，不换行
    print()  # 内层循环结束，打印空行以实现换行
```
**代码解释**：
1.  内层循环的 `print` 语句设置了 `end=' '`，使得该行所有算式输出在同一行，并用空格分隔。
2.  外层循环中，在内层循环结束后执行的 `print()` 会产生一个换行，从而开始新的一行。

运行最终代码，你将看到整齐打印的九九乘法表。

本节课中我们一起学习了如何利用**嵌套循环**来构建九九乘法表。关键点在于：
1.  **分析规律**：第 `i` 行有 `i` 个算式。
2.  **嵌套循环**：外层循环变量 `i` 控制行，内层循环变量 `j` 控制当前行的算式。
3.  **控制输出**：使用 `print` 函数的 `end` 参数控制不换行输出，并在合适位置换行。
