## 每日开盘前的数据预处理

![](img/02ce6539788ac2c50982a423b2fbb6f0_17.png)

在每日交易开始前，我们需要获取并处理财务数据，以筛选出目标股票。这个步骤在 `before_trading` 函数中完成。

以下是实现股票筛选的核心步骤：

1.  **构建查询（Query）**：首先，我们需要明确要查询的财务指标。这里我们选择查询公司的“营业总收入”指标。所有可用的指标都可以在平台的帮助文档中找到。
2.  **应用过滤器（Filter）**：查询需要限定在特定的股票范围内。我们使用 `filter` 条件，确保只查询属于我们股票池（即沪深300指数）的股票。
3.  **结果排序（Order By）**：查询到所有符合条件的股票后，我们需要根据“营业总收入”指标进行排序。为了选出“最好”的股票，我们按降序排列。
4.  **限制数量（Limit）**：排序后，我们只选取排名前10的股票作为当日的目标股票池。

核心的查询与筛选逻辑可以用以下代码公式表示：

```python
# 公式：筛选逻辑 = 查询(指标) & 过滤(属于股票池) & 排序(指标降序) & 限制(前N名)
query = query(指标.营业总收入)
    .filter(指标.股票代码.in_(context.stock_pool))
    .order_by(指标.营业总收入.desc())
    .limit(10)
```

def before_trading(context):
    # 1. 构建查询，获取营业总收入指标
    q = query(指标.营业总收入)

    # 2. 过滤：只选择沪深300指数中的股票
    # 注意：需要从‘指数’类中获取‘沪深300’的正确代码

    # 3. 排序：按营业总收入降序排列
