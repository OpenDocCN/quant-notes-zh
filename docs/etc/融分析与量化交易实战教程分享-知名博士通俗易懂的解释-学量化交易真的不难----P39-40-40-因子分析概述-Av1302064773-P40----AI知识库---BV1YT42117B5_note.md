# 量化交易完全可自学教程：P39：因子分析概述 📊

## 概述
在本节课中，我们将要学习量化交易中的一个核心环节——因子分析。我们将解释什么是因子，以及如何从众多因子中筛选出对股票收益率有显著影响的优质因子。核心在于理解并计算因子与收益率之间的相关性，即IC值。

## 什么是因子分析
上一节我们介绍了量化交易的基本概念，本节中我们来看看如何从海量数据中筛选有效信息。在股票分析中，我们可以获取到大量指标，例如基本面信息（如市盈率A）和技术指标。这些指标都可以被视为“因子”。

这些因子可以分成多个大类，每个大类下又能细分为许多具体指标。假设我们手中有300个这样的因子。在进行策略回测或设计时，我们面临一个选择：是使用全部因子，还是只使用其中一部分？

![](img/dc67625d91c206699d50218f9389fdb1_1.png)

因此，我们需要做一件事：判断哪些因子是我们想要的，哪些不是。换句话说，我们需要对这300个因子进行排序，依据是它们对最终投资收益的影响程度进行“打分”。影响大的因子得分高（如98分），影响小的得分低，甚至得零分。这样，我们就能得到一个从优到劣的因子排序列表。

## 如何评估因子好坏：收益率
既然要评估因子对结果的影响，我们首先需要明确“结果”是什么。在金融领域，最核心的结果指标之一就是**收益率**。

收益率计算很简单，我们以日收益率为例进行说明。公式如下：
`日收益率 = (当日收盘价 - 上一交易日收盘价) / 上一交易日收盘价`
这个公式表示的是当前交易日相对于上一个交易日的价格变化率。

股票投资希望积少成多，因此我们关注每天的收益。此时，我们发现因子数值每天在变化，收益率每天也在变化。我们需要探究的是：因子的变化与收益率的变化之间有什么关系？

例如，我们连续取一年的数据（约250个交易日）。对于每一个因子（记为F）和每日收益率（记为R），我们都有250对对应的数值。我们需要分析因子F的走势与收益率R的走势之间存在何种关系：是线性相关，还是非线性相关？如果相关，是正相关还是负相关？相关程度有多强？因子分析就是要回答这些问题。

![](img/dc67625d91c206699d50218f9389fdb1_3.png)

## 因子分析的核心：IC值
其实说白了，因子分析就是做上述相关性分析。在分析中，我们主要做两件事，第一件就是计算因子的**IC值**。

IC是“Information Coefficient”的缩写，它是一个衡量相关性的指标。具体来说，IC值是通过计算某个因子与收益率之间的**斯皮尔曼相关系数**来得到的。

斯皮尔曼相关系数的取值范围在-1到+1之间：
*   **越接近+1**，表示因子与收益率之间**正相关性越强**（因子值越大，预期收益率越高）。
*   **越接近-1**，表示因子与收益率之间**负相关性越强**（因子值越大，预期收益率越低）。
*   **越接近0**，表示因子与收益率之间**几乎没有线性关系**。

![](img/dc67625d91c206699d50218f9389fdb1_5.png)

因此，IC值直观地反映了单一因子与收益率之间的相关性强度与方向。这种针对单个因子的分析，也称为**单因子分析**。

![](img/dc67625d91c206699d50218f9389fdb1_7.png)

## 如何进行单因子分析
为什么要进行单因子分析？假设我们有300个备选因子，而收益率序列是相对固定的。在收益率固定的前提下，我们需要逐一考察每个因子与收益率的关系。

![](img/dc67625d91c206699d50218f9389fdb1_9.png)

![](img/dc67625d91c206699d50218f9389fdb1_11.png)

以下是进行单因子分析的基本思路：
1.  我们会编写一个循环过程。
2.  在循环中，依次遍历每一个因子。
3.  针对每一个因子，计算它与收益率序列之间的斯皮尔曼相关系数。
4.  这个计算出的相关系数，就是该因子的**IC值**。

![](img/dc67625d91c206699d50218f9389fdb1_13.png)

计算出所有因子的IC值后，我们就可以根据IC值的大小和正负对它们进行排序和筛选。IC值绝对值越大（无论正负），表明该因子对收益率的预测能力越强，越值得关注。IC值接近0的因子，则表明其与收益率关系不大，在构建模型时可以考虑剔除。

## 分析结果展示
计算完成后，我们需要将结果可视化以便分析。常见的展示方式包括：

![](img/dc67625d91c206699d50218f9389fdb1_15.png)

![](img/dc67625d91c206699d50218f9389fdb1_17.png)

*   **IC值序列图**：展示某个因子在历史上一段时间内（例如每天）的IC值变化情况。
*   **IC值均线图**：在IC值序列图上，可以叠加一条移动平均线（例如10日移动均线）。这有助于观察IC值的长期趋势，平滑短期波动。均线图上最初部分可能出现空值，这是因为移动平均计算需要一定窗口期的数据。

图中，IC值会有正有负，对应着我们之前提到的正相关与负相关关系。我们的目标就是找出那些IC值绝对值持续较高、且方向稳定的因子。

![](img/dc67625d91c206699d50218f9389fdb1_19.png)

## 总结
本节课中我们一起学习了量化交易中因子分析的基础知识。我们首先明确了因子分析的目标是从大量指标中筛选有效因子。然后，我们引入了**收益率**作为评估基准，并指出分析的核心是计算因子与收益率之间的相关性，即**IC值**。IC值通过斯皮尔曼相关系数计算，其绝对值大小代表预测能力强弱，正负号代表影响方向。最后，我们了解了如何通过循环进行单因子分析，以及如何通过图表（IC序列图与均线图）来直观评估因子的有效性。掌握这些是构建量化选股模型的第一步。
