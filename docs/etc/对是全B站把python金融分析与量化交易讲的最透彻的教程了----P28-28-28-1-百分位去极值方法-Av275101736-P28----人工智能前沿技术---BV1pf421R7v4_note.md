## 因子数据简介
上一节我们介绍了量化分析的基本概念，本节中我们来看看什么是因子数据。因子可以理解为选择股票的标准或指标。例如，在购买股票时，投资者可能会依据某些标准进行筛选。

![](img/b0e4043dd0cdbd4c373348038f46b891_1.png)

例如，市净率较低的股票可能被认为具有较大的上涨空间。再比如，营收增长率较高的公司可能被视为优质公司。简单来说，对最终投资结果有影响的指标，就是我们今天要讨论的因子。

![](img/b0e4043dd0cdbd4c373348038f46b891_3.png)

当我们获得因子数据后，需要对其进行处理。这类似于一个数据挖掘问题：我们最终的投资收益是目标变量Y，而各种因子（如X1, X2, X3...）是输入变量。我们需要分析每个因子对结果的影响，并从中做出选择。本节课结束后，我们将举例说明如何在多因子策略中选择合适的股票以获得更高收益。

## 因子数据预处理三步走
在直接使用数据进行建模之前，通常需要对数据进行预处理。以下是因子数据预处理的三个主要步骤：

![](img/b0e4043dd0cdbd4c373348038f46b891_5.png)

1.  **去极值**：处理数据中的离群点或异常值。
2.  **标准化**：将不同取值范围的数据调整到同一尺度，例如0到1或-1到1之间。
3.  **中性化**：在因子策略中特有的一步，用于消除因子与某些系统性风险（如市值、行业）的相关性。

![](img/b0e4043dd0cdbd4c373348038f46b891_7.png)

![](img/b0e4043dd0cdbd4c373348038f46b891_9.png)

前两步在机器学习和数据挖掘中很常见，而第三步“中性化”则是金融量化领域的特殊需求。本节课我们将按照这个顺序，首先详细讲解如何使用百分位方法进行去极值处理。

## 第一步：去极值处理
去极值处理的目标不是简单地删除异常数据点，而是将它们“拉回”到合理的边界内。例如，有一个数据点值为10，而我们设定的上界是5。我们不丢弃这个点，而是将其值改为5，使其不再“越界”。

![](img/b0e4043dd0cdbd4c373348038f46b891_17.png)

以下是几种常见的去极值方法，本节课我们重点介绍第一种：**百分位（分位数）法**。

![](img/b0e4043dd0cdbd4c373348038f46b891_19.png)

### 分位数（百分位）法
在介绍百分位法之前，我们需要理解分位数的概念。与容易受极端值影响的均值不同，中位数等分位数能更好地反映数据的中心趋势。

*   **中位数**：将数据从小到大排列后，处于正中间位置的值。例如，7个数据的中位数是第4个；8个数据的中位数是第4和第5个数据的平均值。
*   **四分位数**：将数据四等分。
    *   **Q1（第一四分位数）**：25%的数据小于等于该值。
    *   **Q2（第二四分位数）**：即中位数，50%的数据小于等于该值。
    *   **Q3（第三四分位数）**：75%的数据小于等于该值。

![](img/b0e4043dd0cdbd4c373348038f46b891_21.png)

**百分位去极值法的操作步骤如下：**

1.  首先计算因子数据的两个关键分位数，例如**1%分位数**（`pct_1`）和**99%分位数**（`pct_99`）。
2.  设定边界：将小于 `pct_1` 的值都调整为 `pct_1`；将大于 `pct_99` 的值都调整为 `pct_99`。
3.  这样，处于两端1%的极端值就被“拉回”到边界上，实现了去极值的目的。

![](img/b0e4043dd0cdbd4c373348038f46b891_23.png)

**代码描述：**
```python
# 假设 factor_data 是一个包含因子值的Pandas Series或DataFrame列
pct_1 = factor_data.quantile(0.01)  # 计算1%分位数
pct_99 = factor_data.quantile(0.99) # 计算99%分位数

![](img/b0e4043dd0cdbd4c373348038f46b891_25.png)

![](img/b0e4043dd0cdbd4c373348038f46b891_27.png)

# 进行去极值处理：将小于下界和大于上界的值分别设为边界值
factor_data_winsorized = factor_data.clip(lower=pct_1, upper=pct_99)
```
这段代码中，`.quantile()` 方法用于计算分位数，`.clip()` 方法将超出指定范围的值裁剪到边界。
