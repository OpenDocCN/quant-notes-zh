## 调试利器：PDB命令 🐛

在编写代码时，经常会遇到报错。传统调试方法是手动添加断点，过程繁琐。IPython提供了一个开关性质的魔术命令 `%pdb`，可以自动进入调试模式。

![](img/c487f6a58cf7d37672842115184813a4_5.png)

*   **用法**：`%pdb on` 开启自动调试；`%pdb off` 关闭。
*   **效果**：当代码段中的某一行即将报错时，执行会自动停在该行之前，并进入交互式调试器。

以下是示例代码和调试过程：

```python
def test(a, b):
    c = a / b  # 如果b为0，此行将引发除零错误
    return c

![](img/c487f6a58cf7d37672842115184813a4_7.png)

test(3, 0)
```

![](img/c487f6a58cf7d37672842115184813a4_9.png)

开启 `%pdb on` 后运行上述代码，执行会在 `c = a / b` 这一行之前暂停，并进入调试模式。此时，可以使用 `p` 命令打印变量的当前值来排查问题。

![](img/c487f6a58cf7d37672842115184813a4_11.png)

![](img/c487f6a58cf7d37672842115184813a4_13.png)

```python
p a  # 输出: 3
p b  # 输出: 0
```

![](img/c487f6a58cf7d37672842115184813a4_15.png)

通过检查变量，可以快速定位到 `b` 的值为0，导致了除零错误。

以下是PDB调试器的常用命令：
*   `h`：查看帮助文档。
*   `q`：退出调试器。
*   `n`：执行下一行代码。
*   `break`：设置断点。

> **注意**：对于复杂的调试，可能更推荐使用专业的IDE（如PyCharm）。但 `%pdb` 命令对于快速定位简单错误非常方便。调试完成后，记得使用 `%pdb off` 关闭自动调试功能。

## 便捷交互：历史与输入输出 🔄

IPython提供了便捷的方式来管理命令历史和获取之前的输入输出结果，这能节省大量重复输入的时间。

### 使用命令历史

类似于Linux命令行，IPython支持使用上下箭头键浏览历史命令。此外，它还支持前缀搜索。

*   **操作**：输入命令的开头几个字符，然后按上箭头键，IPython会筛选出以这些字符开头的历史命令。

![](img/c487f6a58cf7d37672842115184813a4_17.png)

![](img/c487f6a58cf7d37672842115184813a4_19.png)

### 获取输入输出结果

在交互式环境中，有时我们运行了代码查看结果，但忘记将结果保存到变量中。IPython提供了特殊变量来获取这些历史值。

*   `_` （单下划线）：获取上一行代码的输出结果。
*   `__` （双下划线）：获取上上一行代码的输出结果。
*   `_iX` （其中X是行号）：获取指定行号的输入代码（字符串形式）。

例如：
```python
3 + 5          # 输出: 8
_ * 2          # 使用上一个输出(8)进行计算，结果: 16
_i1            # 获取第1行的输入，返回字符串: ‘3 + 5‘
```

## 高效导航：目录标签系统 📂

![](img/c487f6a58cf7d37672842115184813a4_23.png)

当需要在多个项目目录间频繁切换时，反复输入冗长的 `cd` 路径非常低效。IPython的 `%bookmark` 魔术命令可以创建目录的快捷方式（书签）。

![](img/c487f6a58cf7d37672842115184813a4_25.png)

![](img/c487f6a58cf7d37672842115184813a4_27.png)

以下是书签的常用操作：

*   `%bookmark 书签名 目录路径`：创建书签。
*   `%bookmark -l`：列出所有书签。
*   `cd 书签名`：快速跳转到书签对应的目录。
*   `%bookmark -d 书签名`：删除指定书签。
*   `%bookmark -r`：删除所有书签。

例如：
```python
%bookmark proj /home/user/my_project
%bookmark -l      # 查看书签列表
cd proj           # 快速切换到 /home/user/my_project 目录
```

## 强大工具：Jupyter Notebook 📓

Jupyter Notebook 是基于Web的交互式计算环境，它完美继承了IPython的交互特性，并允许我们将代码、运行结果、文本说明、数学公式和图表整合在一个文档中。

要使用Notebook，需要先安装 `jupyter` 模块：
```bash
pip install jupyter
```

安装后，在系统命令行中输入 `jupyter notebook`，它会在浏览器中打开一个文件管理界面。你可以创建新的Notebook文件（后缀为 `.ipynb`）。

![](img/c487f6a58cf7d37672842115184813a4_29.png)

![](img/c487f6a58cf7d37672842115184813a4_31.png)

在Notebook中：
*   可以像在IPython中一样执行代码块，并立即看到输出。
*   支持使用Markdown单元格编写富文本说明，非常适合制作技术博客、教程或数据分析报告。
*   可以方便地展示 `pandas` 的 `DataFrame`、`matplotlib` 绘制的图表等。
*   支持将Notebook导出为多种格式，如Python脚本(.py)、HTML网页、PDF等。

![](img/c487f6a58cf7d37672842115184813a4_33.png)

![](img/c487f6a58cf7d37672842115184813a4_34.png)
