## 数学运算与三角函数

![](img/4f824885cdc0b6c17ec7794cd49b8a9a_3.png)

上一节我们介绍了课程的整体结构，本节中我们来看看数学运算和三角函数。在进行数值运算前，需要先确定浮点数运算的精度。计算机在处理浮点数时存在通用误差，无法做到完全精确，但可以通过设置来无限接近。

设置精度的方法是使用 `sfp_compare` 函数，其参数 `OCCURRENCY` 代表精确度。默认精度为 `mid`（值为2），精度约为 `2.2e-12`。如需更高精度，可设置为 `high`（值为3）或 `very high`（值为4）。

以下是常用的数学运算函数：

*   **`INTEGERPORTION`**：返回数值的整数部分（包含正负号）。例如，`INTEGERPORTION(3.7)` 返回 `3`，`INTEGERPORTION(-2.9)` 返回 `-2`。
*   **`FRACTIONPORTION`**：返回数值的小数部分（包含正负号）。例如，`FRACTIONPORTION(3.7)` 返回 `0.7`。
*   **`ROUND`**：将数值四舍五入到指定的小数位数。公式为 `ROUND(value, decimal_places)`。例如，`ROUND(1.23456, 2)` 返回 `1.23`。
*   **`CEILING`**：返回大于或等于数值的最小整数。例如，`CEILING(9.1)` 返回 `10`，`CEILING(-2.85)` 返回 `-2`。
*   **`FLOOR`**：返回小于或等于数值的最大整数。例如，`FLOOR(9.1)` 返回 `9`，`FLOOR(-2.85)` 返回 `-3`。
*   **`SGN`**：返回数值的符号。正数返回 `1`，负数返回 `-1`，零返回 `0`。
*   **`ABSVALUE`**：返回数值的绝对值。例如，`ABSVALUE(-7)` 返回 `7`。
*   **`NEG`**：返回数值的负绝对值。例如，`NEG(7)` 返回 `-7`，`NEG(-7)` 返回 `-7`。
*   **`MOD`**：取余（取模）运算。公式为 `MOD(dividend, divisor)`。余数的符号与被除数相同。例如，`MOD(10, 3.3)` 返回 `0.1`，`MOD(-10, 3.3)` 返回 `-0.1`。
*   **`EXP`**：返回 e 的乘方。例如，`EXP(1)` 返回 e 的近似值 `2.71828`。
*   **`LOG`**：返回数值的自然对数。
*   **`POWER`**：返回数值的乘幂结果。公式为 `POWER(base, exponent)`。例如，`POWER(5, 3)` 返回 `125`。
*   **`SQUARE`**：返回数值的平方。例如，`SQUARE(5)` 返回 `25`。
*   **`SQUAREROOT`**：返回数值的平方根。
*   **`RANDOM`**：返回介于 0 到指定数之间的随机数值（非整数）。例如，`RANDOM(10)` 可能返回 `7.40`。
*   **`SUMLIST`**：返回一系列数值的总和。
*   **`AVERAGELIST`**：返回一系列数值的平均值。
*   **`MAXLIST`**：返回一系列数值中的最大值。
*   **`MAXLIST2`**：返回一系列数值中的第二大值。
*   **`NMAXLIST`**：返回一系列数值中的第 N 大值。公式为 `NMAXLIST(N, value1, value2, ...)`。
*   **`MINLIST`**, **`MINLIST2`**, **`NMINLIST`**：功能与上述最大值函数对应，用于求最小值。

以下是三角函数，参数为角度值：

*   **`SIN`**：正弦函数。例如，`SIN(30)` 返回 `0.5`。
*   **`COS`**：余弦函数。
*   **`TAN`**：正切函数。
*   **`COTAN`**：余切函数。
*   **`ARCTAN`**：反正切函数。

上一节我们介绍了基础的数学运算，本节中我们来看看日期函数。日期处理在量化策略中非常重要，例如计算止盈后的等待天数。

![](img/4f824885cdc0b6c17ec7794cd49b8a9a_15.png)

![](img/4f824885cdc0b6c17ec7794cd49b8a9a_17.png)

![](img/4f824885cdc0b6c17ec7794cd49b8a9a_19.png)

在EasyLanguage中，日期有两种表示方式：
1.  **EL Date**：如 `1221112` 表示2022年11月12日。可以直接进行大小比较，但不适合直接进行加减运算（例如，11月30日加2天会变成11月32日，这是无效的）。
2.  **儒略日 (Julian Day)**：从1900年1月1日（儒略日为2）开始，每过一天数值加一。它类似于时间戳，便于进行日期运算。

![](img/4f824885cdc0b6c17ec7794cd49b8a9a_21.png)

![](img/4f824885cdc0b6c17ec7794cd49b8a9a_23.png)

进行日期运算时，通常需要先将EL Date转换为儒略日，运算完成后再转换回EL Date。

以下是关键的日期函数：

*   **`CURRENTDATE`**：返回计算机当前的本地日期（EL Date格式）。
*   **`LASTCALCDATE`**：返回图表上最后一根K线的日期（EL Date格式）。
*   **`ENCODEDATE`**：根据指定的年、月、日返回对应的儒略日。参数格式为 `ENCODEDATE(year, month, day)`。例如，`ENCODEDATE(2022, 10, 1)` 返回一个儒略日数值。
*   **`ELDATETOJULIAN`** / **`ELDATETODATETIME`**：将EL Date格式的日期转换为儒略日格式。
*   **`JULIANTODATE`** / **`DATETIMETOELDATE`**：将儒略日格式的日期转换回EL Date格式。
*   **`DATETOSTRING`**：将儒略日格式的日期（整数部分）转换为字符串格式。**注意**：需要一个字符串类型的变量来接收结果。代码示例如下：
    ```easylanguage
    vars: myString("");
    value1 = ELDATETOJULIAN(date); // date是当前K线EL日期
    myString = DATETOSTRING(value1); // value1是儒略日
    Print(myString); // 输出如 "2022-11-10"
    ```
*   **`STRINGTODATE`**：将特定格式的日期字符串转换为儒略日。字符串格式适应性较强，例如 `"2022-11-12"`、`"11/12/2022"` 通常都可正确转换。
