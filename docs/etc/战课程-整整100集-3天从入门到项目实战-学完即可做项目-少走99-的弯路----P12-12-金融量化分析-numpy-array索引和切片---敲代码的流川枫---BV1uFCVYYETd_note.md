## 一维数组的索引与切片

索引用于访问数组中的单个元素，切片用于获取数组的一个子集。其基本语法与Python列表类似。

![](img/af51d7bf43b17d71d7a74be9d3e0c84c_35.png)

![](img/af51d7bf43b17d71d7a74be9d3e0c84c_37.png)

以下是索引与切片的基本规则：

*   **索引**：通过位置下标访问元素，下标从0开始。例如，`A[0]` 获取数组A的第一个元素。
*   **切片**：使用 `start:stop:step` 语法获取一个片段，遵循“前闭后开”原则。例如，`A[0:4]` 获取索引0到3的元素。

**重要区别**：NumPy数组的切片返回的是原始数组数据的**视图（view）**，而非副本。这意味着修改切片数据会直接影响原数组。这是为了在处理大型数据时节省内存。

若需要一份独立的副本，需使用 `copy()` 方法。
代码示例：
```python
C = A[0:4].copy() # C是A切片的一个独立副本
C[0] = 20 # 修改C不会影响A
```

## 二维数组的索引与切片

对于二维数组（可视为矩阵），我们需要通过行和列两个维度来定位元素。

以下是二维数组操作的要点：

*   **索引**：可以通过连续的中括号 `A[0][2]` 或更简洁的逗号分隔格式 `A[0, 2]` 来访问元素。后者是更推荐的做法。
    *   公式：`数组名[行索引, 列索引]`
*   **切片**：在逗号分隔的格式中，逗号前面对行进行切片，逗号后面对列进行切片。
    *   代码示例：`A[0:2, 1:3]` 获取第0、1行，第1、2列交叉区域的元素。
    *   代码示例：`A[:, 0]` 获取所有行的第0列（第一列）。
