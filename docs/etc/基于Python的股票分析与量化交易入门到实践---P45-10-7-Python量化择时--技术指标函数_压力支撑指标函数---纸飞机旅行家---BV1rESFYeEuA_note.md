## 布林带指标

布林带，也有人称为布林线，由约翰·布林创造。其核心原理是计算股价的标准差及其信赖区间，从而确定股价的波动范围及未来走势。它是一个带状指标，用于显示股价的安全高低价位。

布林带由三根线组成：中轨线、上轨线和下轨线。中轨线通常是20日均线。上轨线是20日均线加上20日收盘价的**2倍标准差**，下轨线是20日均线减去20日收盘价的**2倍标准差**。因此，布林带的上下限范围不固定，会随股价波动而变化。

以下是布林带的计算公式：
*   **中轨线 (MB) = MA(CLOSE, N)**
*   **上轨线 (UP) = MB + k * STD(CLOSE, N)**
*   **下轨线 (DN) = MB - k * STD(CLOSE, N)**
其中，`N`通常为20，`k`通常为2。

以下是布林带的应用方式：
*   当股价上升穿越布林带上轨时，回档几率较大。
*   当股价下跌穿越布林带下轨时，反弹几率较大。
*   当整个布林带变窄时，表示变盘在即。

### 布林带的代码实现

布林带的实现需要调用技术分析库。以下是使用示例：

```python
import talib
import numpy as np

# 假设 close_prices 是收盘价序列
close_prices = np.array([...])

# 计算布林带
upper, middle, lower = talib.BBANDS(close_prices,
                                     timeperiod=20,
                                     nbdevup=2,
                                     nbdevdn=2)
```
函数返回一个包含上轨、中轨、下轨值的元组。

## 麦克指标

麦克指标全称是麦克支撑压力指标。它是一种根据股价波动幅度大小而变动的压力支撑指标，设有初级、中级和强力三种不同级别的支撑和压力线，共计六条线。

麦克指标的计算分为三步：首先计算真实波幅，然后分别计算出三条压力线和三条支撑线。压力线分为：初级压力`WR`、中级压力`MR`、强力压力`SR`；支撑线分为：初级支撑`WS`、中级支撑`MS`、强力支撑`SS`。

以下是麦克指标的应用方式：
*   麦克指标共有六条曲线，上方三条为压力线，下方三条为支撑线。
*   当股价脱离盘整（波峰波谷价差约15%以内）开始下跌时，下方的三条支撑线具有参考价值。
*   当股价脱离盘整开始上涨时，上方的三条压力线具有参考价值。
*   当股价单边上涨时，下方支撑线无参考价值；当股价单边下跌时，上方压力线无参考价值。

## 薛斯通道

薛斯通道是判断股价运行区间及相应压力、支撑的趋势性指标，也是一个短线指标。它由美国薛斯先生创造，包含短期小通道和长期大通道。

其基本买卖策略是：当短期小通道接近长期大通道时，可能预示着趋势的近期反转。
*   当股价在通道上沿，且趋势向下反转时，可视为短期卖点。
*   当股价在通道下沿，且趋势向上反转时，可视为短期买点。

## 本章小结

本节课我们一起学习了三种压力支撑指标。

首先介绍了**布林带**。它是一个根据股价标准差动态变化的带状指标，包含上、中、下三轨。其应用要点在于：股价突破上轨可能回档，突破下轨可能反弹；布林带收窄预示可能变盘。

接着介绍了**麦克指标**。这是一种包含六条线的复杂指标，将压力和支撑各分为初、中、强三个级别。其核心应用逻辑是：在股价脱离盘整趋势时，使用对应方向的线（上涨看压力，下跌看支撑）作为参考，单边行情中反向的线参考价值降低。

最后介绍了**薛斯通道**。这是一个用于短线交易的通道指标，包含短期和长期两组通道。其主要策略是观察短期通道与长期通道的接近程度来判断趋势反转，并利用股价在通道上下沿的反转来捕捉买卖点。

在代码实战部分，我们演示了如何使用技术分析库计算这些指标。需要注意的是，不同库的函数名和参数可能略有差异，实际使用时请查阅相关文档。
