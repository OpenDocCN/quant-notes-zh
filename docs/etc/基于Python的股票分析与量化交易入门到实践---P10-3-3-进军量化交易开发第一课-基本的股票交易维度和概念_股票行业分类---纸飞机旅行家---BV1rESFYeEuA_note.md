## 股票行业分类体系

股票行业分类主要有两种权威体系：**中证行业分类**和**申万行业分类**。它们服务于不同目的，是投资者分析市场的重要工具。

以下是两种分类体系的详细介绍：

### 中证行业分类 🏛️

![](img/4553f7be7cc3d47e08089f709d19c07e_3.png)

中证行业分类由中国证监会直属的**中证指数有限公司**发布。它主要服务于监管目的，是官方的行业分类标准。该体系自2007年开始发布，并持续迭代更新。

![](img/4553f7be7cc3d47e08089f709d19c07e_5.png)

中证行业分类采用**四级**结构，层级之间具有明确的从属关系。

![](img/4553f7be7cc3d47e08089f709d19c07e_7.png)

*   **一级行业**：共11个。
*   **二级行业**：共35个。
*   **三级行业**：共90个。
*   **四级行业**：共200余个。

![](img/4553f7be7cc3d47e08089f709d19c07e_9.png)

例如，其一级行业包括工业、商业地产等。投资者可以在中证指数官网查询详细的分类列表和对应的指数信息。

![](img/4553f7be7cc3d47e08089f709d19c07e_11.png)

### 申万行业分类 📈

申万行业分类由国内顶尖的券商研究所之一——**申万宏源研究所**编制。它更侧重于投资分析的实际应用，在金融和量化分析领域被广泛使用。最新版本于2021年8月修订。

![](img/4553f7be7cc3d47e08089f709d19c07e_13.png)

申万行业分类采用**三级**结构，相对简化，更贴近中国市场的实际情况。

![](img/4553f7be7cc3d47e08089f709d19c07e_15.png)

*   **一级行业**：共31个。
*   **二级行业**：共134个。
*   **三级行业**：共346个。

![](img/4553f7be7cc3d47e08089f709d19c07e_16.png)

![](img/4553f7be7cc3d47e08089f709d19c07e_18.png)

其分类中包含了如“美容护理”等具有中国特色的行业类别。以“通信”行业为例，在其分类下可以清晰地找到所有归属于该行业的上市公司列表。

## 行业分类的作用与行业特性分析 🔍

![](img/4553f7be7cc3d47e08089f709d19c07e_20.png)

了解分类体系后，你可能会问：这与我的量化投资有什么关系？研究行业，实际上是从宏观层面把握股票群体的共同特征，是股票分析中极为有效的一环。

行业研究的作用在于，我们可以根据历史表现和未来前景，将行业大致分为四类，并采取不同的投资策略。

以下是基于行业特性的四类划分：

1.  **持续盈利型行业**：这类行业以约10年为一个周期，长期保持稳定盈利。主要集中在**食品饮料**和**医药生物**行业。无论在中国还是美国市场，这两个行业在过去20年里都展现了极强的稳定性。
2.  **传统（周期）型行业**：这类行业具有明显的经济波动周期。其表现通常是前一个10年出色，后一个10年不佳。例如，**电子设备、采掘（如煤炭）、有色金属、传统汽车、机械设备、钢铁**等行业。它们在21世纪初的十年很风光，但近十年则面临挑战。
3.  **新兴（成长）型行业**：这类行业在前10年可能尚未出现或规模很小，但在最近10年，特别是最近5年发展迅猛。例如，**计算机、电子（半导体等）、新能源、家电（智能家电）** 等。它们是当前市场的“朝阳行业”。
4.  **表现不佳型行业**：这类行业长期来看表现疲弱，在金融投资领域普遍不被看好。主要包括**公用事业（如环卫、公共服务）和纺织服装**。对于量化投资者而言，通常建议避开这些行业。

研究行业的最终目的，是为了更好地理解其下属的上市公司。行业特性清楚了，个股的基本面轮廓和投资逻辑也就更容易把握。

![](img/4553f7be7cc3d47e08089f709d19c07e_22.png)

## 总结 📝

本节课中我们一起学习了股票行业分类的核心知识。

首先，我们介绍了**中证**和**申万**两大主流分类体系。**中证分类**是官方监管导向的分类，而**申万分类**则更侧重于投资分析，更为务实和接地气。

其次，我们探讨了研究行业分类的核心目的：**理解行业特性**。我们将其归纳为四类：
*   **持续盈利型**：如食品饮料、医药生物。
*   **传统周期型**：如有色金属、采掘、机械制造等，可视为“夕阳行业”。
*   **新兴成长型**：如计算机、新能源等，是当前的“朝阳行业”。
*   **表现不佳型**：如公用事业、纺织服装，在量化投资中通常建议规避。

![](img/4553f7be7cc3d47e08089f709d19c07e_24.png)

掌握行业分类及其特性，是构建有效股票分析框架和量化交易策略的重要一步。
