## 概述

在策略分析中，我们不仅关注最终的收益，还需要评估策略的风险。最大回撤就是这样一个衡量策略在特定时间段内“最糟糕情况”的风险指标。它描述了投资者可能经历的最大账面亏损幅度。

## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看一个具体的风险指标。

最大回撤描述的是在投资过程中，资产净值从任一历史高点下跌到随后低点的最大幅度。它衡量的是策略的“惨劲儿”，即投资者可能承受的最大心理压力或潜在亏损。

从数学上定义，对于一系列资产净值数据，最大回撤是寻找两个时间点 `i` 和 `j`（`i < j`），使得 `(Pi - Pj) / Pi` 的值最大。其中：
*   **`Pi`** 代表时间点 `i` 的资产净值（一个历史高点）。
*   **`Pj`** 代表时间点 `j` 的资产净值（`i` 之后的一个低点）。

![](img/412c2e29d919badd7f725a3e6152ee82_5.png)

这个公式计算的是从高点 `Pi` 到低点 `Pj` 的“缩水”比例。最大回撤就是遍历所有可能的 `i` 和 `j` 组合后，找到的这个比例的最大值。

![](img/412c2e29d919badd7f725a3e6152ee82_7.png)

## 为什么最大回撤很重要？

最大回撤是一个关键的风险指标。一个策略可能最终盈利，但如果其最大回撤过大，意味着投资者在过程中可能经历非常痛苦的阶段，甚至可能因为无法承受亏损而在低点止损离场，从而错失后期的反弹。例如，2008年股市从高点大幅下跌，就体现了巨大的回撤风险。

![](img/412c2e29d919badd7f725a3e6152ee82_9.png)

![](img/412c2e29d919badd7f725a3e6152ee82_11.png)

在我们的策略回测报告中，最大回撤区间会被明确标识出来，帮助我们直观地看到策略历史上最糟糕的表现阶段。

![](img/412c2e29d919badd7f725a3e6152ee82_13.png)

## 如何计算最大回撤？

![](img/412c2e29d919badd7f725a3e6152ee82_15.png)

![](img/412c2e29d919badd7f725a3e6152ee82_17.png)

理解了概念后，我们来看看如何用Python代码进行计算。核心思路是：对于每一天，计算当日价格与之前历史最高价的差值比例，然后找出所有比例中的最大值。

![](img/412c2e29d919badd7f725a3e6152ee82_19.png)

![](img/412c2e29d919badd7f725a3e6152ee82_21.png)

以下是计算最大回撤的步骤：

![](img/412c2e29d919badd7f725a3e6152ee82_23.png)

![](img/412c2e29d919badd7f725a3e6152ee82_25.png)

1.  **计算历史最高价序列**：我们需要知道到每一天为止的历史最高价是多少。这可以通过Pandas的 `.cummax()` 方法实现。
2.  **计算每日回撤**：用历史最高价序列减去当日的实际价格，得到每日的“潜在亏损”值。
3.  **计算每日回撤率**：将每日的“潜在亏损”值除以对应的历史最高价，得到每日的回撤率。
4.  **找出最大值**：在所有每日回撤率中，最大值就是我们要找的**最大回撤**。

![](img/412c2e29d919badd7f725a3e6152ee82_27.png)

![](img/412c2e29d919badd7f725a3e6152ee82_29.png)

让我们通过代码示例来演示这个过程。假设我们有一个包含股价的Pandas Series `price_series`。

![](img/412c2e29d919badd7f725a3e6152ee82_31.png)

```python
import pandas as pd

![](img/412c2e29d919badd7f725a3e6152ee82_33.png)

![](img/412c2e29d919badd7f725a3e6152ee82_35.png)

# 假设 price_series 是包含每日净值的序列
# 步骤1: 计算到当日为止的历史最高净值序列
historical_max = price_series.cummax()

![](img/412c2e29d919badd7f725a3e6152ee82_37.png)

# 步骤2 & 3: 计算每日的回撤率
# (历史最高 - 当前净值) / 历史最高
daily_drawdown = (historical_max - price_series) / historical_max

![](img/412c2e29d919badd7f725a3e6152ee82_39.png)

# 步骤4: 找出最大回撤
max_drawdown = daily_drawdown.max()
print(f"最大回撤为: {max_drawdown:.2%}")
```

![](img/412c2e29d919badd7f725a3e6152ee82_41.png)

这段代码中：
*   `price_series.cummax()` 生成了一个新序列，其中每个元素都是原序列从开始到当前位置的最大值。
*   `(historical_max - price_series)` 计算了当前净值距离前期高点的下跌绝对值。
*   除以 `historical_max` 后，将其转化为百分比形式的回撤率。
*   最终，`.max()` 方法找到了整个回测周期内的最大回撤值。
