## 数据准备与预处理

首先，我们需要导入必要的工具包并加载数据。做法与之前的课程一致。

![](img/ce082fd220179a67f26351f455b6bd9f_5.png)

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
# 其他必要的库...
```

![](img/ce082fd220179a67f26351f455b6bd9f_7.png)

数据文件仍然是 `data.csv`。本次我们使用欧元兑美元（EUR/USD）的汇率历史数据作为示例。

![](img/ce082fd220179a67f26351f455b6bd9f_9.png)

```python
data = pd.read_csv('data.csv')
# 指定分析EUR/USD汇率列
eur_usd_data = data['EUR/USD']
```

为了进行有效的分析，我们需要对原始价格数据计算对数收益率。这能使数据更平稳，并具有累加性。

![](img/ce082fd220179a67f26351f455b6bd9f_11.png)

```python
# 计算对数收益率
returns = np.log(eur_usd_data / eur_usd_data.shift(1))
# 处理缺失值
returns = returns.dropna()
```

![](img/ce082fd220179a67f26351f455b6bd9f_13.png)

![](img/ce082fd220179a67f26351f455b6bd9f_15.png)

除了具体的收益率数值，我们有时更关心价格的变动方向（涨或跌）。因此，我们新增一个“方向”列。

```python
# 创建方向列：上涨为1，下跌为-1
data['direction'] = np.sign(returns).astype(int)
```

以下是数据基本分布的可视化，可以看到收益率大致以0为中心分布。

```python
plt.hist(returns, bins=50)
plt.title('Returns Distribution')
plt.show()
```

## 构建回归任务

![](img/ce082fd220179a67f26351f455b6bd9f_21.png)

机器学习中的有监督回归任务需要定义特征（X）和目标变量（Y）。我们的目标是利用历史特征预测未来。

![](img/ce082fd220179a67f26351f455b6bd9f_23.png)

对于当前任务，我们可以设定两种不同的目标变量Y进行实验对比：
1.  **Y1**: 预测具体的收益率值（`returns`）。
2.  **Y2**: 预测涨跌方向（`direction`）。

![](img/ce082fd220179a67f26351f455b6bd9f_25.png)

特征X则可以利用过去多个时间点的数据来构建。例如，使用当天及前N天的收益率作为特征，来预测下一天的Y值。

![](img/ce082fd220179a67f26351f455b6bd9f_27.png)

```python
# 示例：构建特征 (使用滞后数据)
lags = 5  # 使用过去5天的数据作为特征
for lag in range(1, lags + 1):
    data[f'lag_{lag}'] = returns.shift(lag)

# 定义特征X和目标Y1, Y2
X = data[['lag_1', 'lag_2', 'lag_3', 'lag_4', 'lag_5']].dropna()
Y1 = returns.reindex(X.index)  # 目标：具体收益率
Y2 = data['direction'].reindex(X.index)  # 目标：涨跌方向
```
