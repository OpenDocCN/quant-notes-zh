在本节课程中，我们将学习梯度下降法中参数更新的具体方法。我们将探讨如何分别优化模型中的不同参数，并详细拆解梯度下降的数学计算步骤。

上一节我们介绍了梯度下降的基本思想，本节中我们来看看如何将下山的过程用数学公式精确地描述出来。

## 🎯 参数是分别优化还是一起优化？

首先需要明确一个核心问题：在优化目标函数时，模型中的参数（例如 θ₀ 和 θ₁）是应该分别进行优化，还是作为一个整体一起优化？

答案是**分别优化**。原因在于，每个参数对应着输入数据中一个独立的特征。例如，θ₀ 对应常数项（或特征 X₀），θ₁ 对应特征 X₁。在数据样本相互独立的假设下，特征 X₀ 和 X₁ 之间没有关系，因此它们对应的参数 θ₀ 和 θ₁ 在优化过程中也应当独立处理。

我们可以将优化过程想象成在一个三维空间中寻找最低点。θ₀ 和 θ₁ 各自代表一个坐标轴。在每一步更新中，我们需要为 θ₀ 寻找最合适的下降方向，同时也为 θ₁ 寻找最合适的下降方向。两者各自沿着自己的方向前进一小步，最终共同作用，使目标函数值下降。

具体来说，我们需要：
1.  计算目标函数 J 对参数 θ₀ 的偏导数，得到 θ₀ 的更新方向。
2.  计算目标函数 J 对参数 θ₁ 的偏导数，得到 θ₁ 的更新方向。

这两个参数的更新是独立进行的，互不影响。

## 📉 梯度下降的核心三步

寻找山谷最低点（即最小化损失函数）的过程可以概括为以下三个步骤，并且需要反复迭代：

![](img/25ad9312206dfbb36a000c3ebc56b393_2.png)

以下是梯度下降的核心步骤：
1.  **求偏导**：计算损失函数 J(θ) 对每个待优化参数 θⱼ 的偏导数 ∂J/∂θⱼ。这给出了该参数在当前点的“最陡上升”方向。
2.  **沿反方向走一小步**：沿着偏导数的**反方向**（即梯度下降方向）移动一小步。步长由学习率 α 控制。**步长必须足够小**，否则可能导致更新幅度过大，错过最优点，甚至造成算法不收敛（“跌倒”）。
3.  **更新参数**：使用计算出的梯度信息更新所有参数。更新完成后，得到一组新的参数值，然后回到步骤1，重新计算在新参数点处的梯度，如此循环，直至收敛。

## 🧮 梯度下降的数学推导

接下来，我们用数学公式来描述“下山”任务。我们从损失函数开始。

假设我们的模型假设函数为 **hθ(x) = θ₀ + θ₁x₁ + θ₂x₂ + ... + θⱼxⱼ**。
常用的损失函数是均方误差（MSE），它衡量模型在所有训练样本上的平均误差：

**J(θ) = (1/2m) * Σᵢ₌₁ᵐ (y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾))²**

公式解释：
*   **m**：训练样本的总数。
*   **Σᵢ₌₁ᵐ**：对所有样本（i 从 1 到 m）的损失进行求和。
*   **(y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾))²**：计算第 i 个样本的预测值与真实值之差的平方。平方项可以放大较大误差的影响，使优化目标更明确。
*   **1/(2m)**：求平均值。分母中的 2 是为了后续求导时与平方项的导数 2 相抵消，简化计算。

现在，我们需要更新某个特定参数 θⱼ。这需要计算损失函数 J(θ) 对 θⱼ 的偏导数。

**∂J(θ)/∂θⱼ = (1/m) * Σᵢ₌₁ᵐ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾**

推导过程解析：
1.  常数项 **1/(2m)** 暂时不变。
2.  对平方项 **(y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾))²** 求关于 θⱼ 的偏导。根据链式法则，先对整体求导得 **2*(y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾))**，再乘以内层函数 **-hθ(x⁽ⁱ⁾)** 对 θⱼ 的偏导数 **-xⱼ⁽ⁱ⁾**。
3.  相乘得到：**2*(y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾)) * (-xⱼ⁽ⁱ⁾) = -2*(y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾)) * xⱼ⁽ⁱ⁾**。
4.  与常数项 **1/(2m)** 相乘，2 被约掉，得到 **-(1/m) * (y⁽ⁱ⁾ - hθ(x⁽ⁱ⁾)) * xⱼ⁽ⁱ⁾**。
5.  将括号内项重新排列，得到最终形式：**(1/m) * Σᵢ₌₁ᵐ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾**。

**符号说明**：
*   **xⱼ⁽ⁱ⁾**：第 i 个训练样本的第 j 个特征值。
*   **偏导数的意义**：它指出了 θⱼ 在当前位置使损失函数 J 上升最快的方向。

## ➡️ 参数更新规则

由于我们的目标是**最小化**损失函数，因此需要沿着梯度（偏导数）的**反方向**前进。这就是“梯度下降”中“下降”的含义。

因此，参数 θⱼ 的更新规则为：

**θⱼ := θⱼ - α * (∂J(θ)/∂θⱼ)**

将上面求得的偏导数代入，得到具体更新公式：

**θⱼ := θⱼ - α * [ (1/m) * Σᵢ₌₁ᵐ (hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾) * xⱼ⁽ⁱ⁾ ]**

公式解释：
*   **:=** 表示赋值更新。
*   **α**：学习率，控制每一步更新的幅度。
*   **- α * (∂J/∂θⱼ)**：从当前参数值 θⱼ 中，减去梯度方向的一小步（α 乘以梯度）。减去梯度相当于沿着梯度的反方向（下降方向）移动。

## 📝 本节总结

本节课中我们一起学习了梯度下降法中参数更新的核心细节。

我们首先明确了模型参数需要**分别独立进行优化**，因为每个参数对应独立的特征。接着，我们将梯度下降过程归纳为**求偏导、沿反方向走一小步、更新参数**这三个核心步骤的循环迭代。

![](img/25ad9312206dfbb36a000c3ebc56b393_4.png)

最后，我们进行了详细的数学推导，从均方误差损失函数出发，推导出对任意参数 θⱼ 的偏导数计算公式，并最终得到了参数更新的通用规则 **θⱼ := θⱼ - α * ∂J/∂θⱼ**。理解这个更新规则是掌握梯度下降法的基础。
