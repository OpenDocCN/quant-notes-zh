# 量化交易教程：P3：03 金融量化分析-股票市场构成

在本节课中，我们将要学习股票市场的构成。了解市场中有哪些参与者以及他们各自扮演的角色，是进行金融量化分析的基础。

上一节我们介绍了股票的分类，本节中我们来看看股票市场是由哪些部分构成的。

## 公司与投资者

公司和投资者是市场中最基础的两方。公司是融资方，需要资金。投资者是出资方，提供资金进行投资。

## 监管与服务机构

但是，股票交易不能让公司和投资者直接进行，这可能导致暗箱操作。因此，市场需要监管和服务机构。

以下是主要的监管与服务机构：

*   **证监会**：这是证券行业的监管机构。公司想要上市，必须向证监会提交各种材料。证监会负责判断公司是否有欺诈、洗钱等违法行为，并决定公司能否上市或是否应退市。
*   **证券业协会**：这是一个行业自律组织，作用相对较弱。例如，证券从业资格考试由其主办。
*   **交易所**：交易所提供股票集中交易的场所。在中国，主要有上海证券交易所和深圳证券交易所。很久以前，投资者需要亲自去交易所排队交易。现在，交易都通过互联网连接到交易所的系统完成。交易所的核心功能是处理所有买卖股票的请求。

## 证券中介机构（券商）

个人投资者通常不能直接去交易所买卖股票。这主要是因为直接交易的成本太高。

在电子化交易之前，买卖股票会获得纸质凭证，核实和清算手续繁琐，成本高。因此，交易所设立了交易“席位”，只有购买席位的会员才能入场交易。一个席位价格昂贵（例如，一百多年前的美国需数万美金），普通投资者无力承担。

于是，拥有席位的会员（即早期的券商）发展了一项业务：他们派人在交易所外接收众多小投资者的买卖委托，汇总后利用自己的席位进入交易所代为执行，以此赚取佣金。

发展到现在，情况演变为：

*   只有券商（如中信证券、中金公司等）在交易所有席位。
*   券商开发了交易软件（如各券商APP或同花顺）。
*   投资者通过券商的软件提交交易指令，券商再通过其在交易所的席位，将指令传达给交易所执行。

## 中国交易所与板块

中国有两个主要的证券交易所：上海证券交易所和深圳证券交易所。

每个交易所内部又划分为不同的板块，以适应不同规模和发展阶段的企业。

以下是各交易所的板块划分：

*   **上海证券交易所**：主要设有**主板**。
*   **深圳证券交易所**：设有三个板块：
    *   **主板**：面向大型成熟企业。
    *   **中小板**：面向中小企业。
    *   **创业板**：面向成长型创新创业企业，上市条件相对主板更灵活。例如，公司可能只需连续几年达到一定净利润即可上市。

## 股票指数（大盘）

对于每个板块，我们常用一个“指数”来代表其整体表现，俗称“大盘”。

**指数**反映了该板块内所有股票价格变动的综合趋势。例如，上海证券交易所主板的指数是**上证指数**（沪指）。深圳证券交易所则有**深证成指**（主板）、**中小板指**和**创业板指**。

如果指数上涨，通常意味着该板块大部分股票在上涨，市场整体向好；反之则意味着市场整体表现不佳。因此，指数是判断市场整体行情的重要工具。
