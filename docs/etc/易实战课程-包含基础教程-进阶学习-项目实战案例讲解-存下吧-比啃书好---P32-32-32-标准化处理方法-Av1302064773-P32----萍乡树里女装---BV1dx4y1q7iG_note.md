# 量化交易教程：P32：标准化处理方法 📊

在本节课中，我们将要学习数据预处理中的一个核心概念——标准化。标准化是数据分析和机器学习中非常基础且重要的一步，它能让不同尺度的数据变得可比，从而提升后续模型的性能。

## 标准化的目的与原理

![](img/8d2042485627d2ac3364faa6ccb0b516_1.png)

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体做法。

![](img/8d2042485627d2ac3364faa6ccb0b516_2.png)

假设我们有一组数据，包含多个指标（或称为特征）。例如，我们有两个指标 X1 和 X2。观察下图，可以发现 X1 和 X2 的取值范围是不同的。X1 的取值范围较大，而 X2 的取值范围较小。

![](img/8d2042485627d2ac3364faa6ccb0b516_1.png)

这种取值范围（量纲）的差异会带来一个问题：当我们直接使用这些原始数据进行计算时，取值范围大的指标可能会对结果产生不成比例的巨大影响，从而掩盖了其他指标的作用。

因此，标准化的核心目的是：**使不同指标的数据处于同一数量级，消除量纲带来的影响，让我们能用“同样的尺度”去观察和比较它们。**

## 标准化的公式与步骤

标准化通常通过一个公式来完成，这个公式分为两个核心步骤。

标准化的公式如下：
`标准化值 = (原始值 - 均值) / 标准差`
用数学符号表示为：
`z = (x - μ) / σ`

这个公式具体做了两件事：

**第一步：去均值（中心化）**
`x - μ`
这一步的目的是将数据的中心移动到零点。原始数据通常不是以原点（0点）为中心对称分布的。减去均值（μ）后，数据会以0为中心对称分布。这解决了数据分布位置不同的问题。

**第二步：除以标准差（缩放）**
`(x - μ) / σ`
这一步的目的是调整数据的尺度。标准差（σ）衡量了数据的离散程度。取值范围大、个体差异大的数据组，其标准差也大；反之则小。

通过除以各自维度的标准差，可以将所有数据的尺度“缩放”到大致相同的范围。原本取值范围大的数据，除以一个较大的标准差后，其范围会缩小；原本取值范围小的数据，除以一个较小的标准差后，其范围得以保留。最终，所有数据的取值都大致分布在一个相近的区间内（例如[-1, 1]或[-3, 3]）。

## 标准化的代码实现

理解了原理后，我们可以用代码来实现标准化操作。以下是使用Python和NumPy库实现标准化的一个示例：

```python
import numpy as np

![](img/8d2042485627d2ac3364faa6ccb0b516_4.png)

def standardize(data):
    """
    对输入的数据进行标准化处理。
    参数:
        data: 一个NumPy数组，形状为 (n_samples, n_features)
    返回:
        标准化后的数据
    """
    # 第一步：计算均值和标准差
    # axis=0 表示沿着列（特征）的方向计算
    mean = np.mean(data, axis=0)
    std = np.std(data, axis=0)

    # 第二步：执行标准化公式： (数据 - 均值) / 标准差
    standardized_data = (data - mean) / std

    return standardized_data

![](img/8d2042485627d2ac3364faa6ccb0b516_5.png)

# 示例数据
original_data = np.array([[30, 200],
                           [25, 180],
                           [35, 220],
                           [100, 50]])
print("原始数据：")
print(original_data)

# 进行标准化
standardized_data = standardize(original_data)
print("\n标准化后的数据：")
print(standardized_data)
```

![](img/8d2042485627d2ac3364faa6ccb0b516_8.png)

运行上述代码，可以看到原始数据中量纲差异很大的数值（如30和200），经过标准化后，都被转换到了一个数值范围相对较小的区间内。

## 使用现成工具库

在实际项目中，我们通常不需要自己编写标准化函数，可以直接使用成熟的机器学习库。

以下是使用 `scikit-learn` 库中的 `StandardScaler` 进行标准化的方法：

![](img/8d2042485627d2ac3364faa6ccb0b516_11.png)

```python
from sklearn.preprocessing import StandardScaler
import numpy as np

![](img/8d2042485627d2ac3364faa6ccb0b516_12.png)

![](img/8d2042485627d2ac3364faa6ccb0b516_13.png)

# 示例数据
data = np.array([[30, 200],
                 [25, 180],
                 [35, 220],
                 [100, 50]])

![](img/8d2042485627d2ac3364faa6ccb0b516_14.png)

![](img/8d2042485627d2ac3364faa6ccb0b516_15.png)

# 创建StandardScaler对象
scaler = StandardScaler()

# 拟合数据并转换
standardized_data = scaler.fit_transform(data)

![](img/8d2042485627d2ac3364faa6ccb0b516_17.png)

![](img/8d2042485627d2ac3364faa6ccb0b516_18.png)

![](img/8d2042485627d2ac3364faa6ccb0b516_19.png)

print(standardized_data)
```

`scikit-learn` 的 `StandardScaler` 功能更强大且稳健，推荐在实战中使用。

## 总结

![](img/8d2042485627d2ac3364faa6ccb0b516_22.png)

![](img/8d2042485627d2ac3364faa6ccb0b516_23.png)

本节课中我们一起学习了数据标准化处理。
*   **目的**：消除不同特征之间因量纲（取值范围）不同造成的偏差，使数据具有可比性。
*   **原理**：通过“去均值”使数据以0为中心，通过“除以标准差”将数据缩放至相似范围。
*   **公式**：`z = (x - μ) / σ`
*   **实现**：可以手动使用NumPy计算，也可以便捷地调用`scikit-learn`中的`StandardScaler`。

![](img/8d2042485627d2ac3364faa6ccb0b516_24.png)

![](img/8d2042485627d2ac3364faa6ccb0b516_25.png)

标准化是数据预处理中几乎必不可少的一步，能显著提高许多机器学习模型（如SVM、KNN、PCA等）的准确性和稳定性。
