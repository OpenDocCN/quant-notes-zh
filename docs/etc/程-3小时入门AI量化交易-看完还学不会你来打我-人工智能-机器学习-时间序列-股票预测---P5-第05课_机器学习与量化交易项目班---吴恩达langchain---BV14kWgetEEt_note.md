## 构建监督学习训练集

监督学习的框架要求我们拥有特征（X）和标签（Y）。我们的任务是将原始的金融时间序列数据转化为这种格式。

假设我们有一个按时间 `T` 排列的原始数据表，包含开盘价、收盘价等属性。对于每一个时刻 `T0`，我们需要：
*   将 `T0` 时刻及之前的历史信息，转化为一组新的特征 `[X1, X2, ..., Xn]`。
*   对应的标签 `Y` 是我们想要预测的目标，例如 `T0+1` 时刻的涨跌（分类）或价格/收益率（回归）。

**公式表示**：
对于每个时间点 `t`，构建：
`X_t = f(原始数据[t-window : t])`
`Y_t = g(原始数据[t+1])` 或 `Y_t = sign(原始数据[t+1] - 原始数据[t])`
其中，`f` 是特征提取函数，`g` 是目标计算函数，`window` 是回顾窗口。

## 特征（因子）选择：方法论

特征（或称因子、预测变量）的选择是量化建模的核心，它直接决定了模型的上限。选择哪些特征，完全取决于你对所预测金融产品的理解。

**核心原则**：特征选择基于你对问题的理解。例如，如果你认为沪深300指数与人民币汇率、黄金价格、标普500指数有关，就应该将这些数据都作为候选特征纳入。

初始阶段应尽可能多地纳入你认为可能相关的特征（“越多越好”），后续再通过特征选择方法进行筛选和优化。这是一个需要大量实验和迭代的过程。

## 常见特征（因子）类型

以下是几种常见的特征计算方法，均已在提供的代码附件中实现。

### 1. 时间滞后特征
最简单的特征就是过去若干个时间点的价格。
`X_i(t) = Price(t-i)`, 其中 `i = 1, 2, ..., k`，`k` 为滞后窗口大小。

### 2. 价格变化率特征
计算当前时刻与之前时刻的价格变化率。
`X(t) = (Price(t) - Price(t-1)) / Price(t-1)`

### 3. 滚动窗口统计特征
计算过去 `λ` 个时间窗口内的最高价、最低价等，并构造特征。
`X1(t) = (Price(t) - Low(t-λ:t)) / (High(t-λ:t) - Low(t-λ:t))`
`X2(t) = (Price(t) - Price(t-1)) / (High(t-λ:t) - Low(t-λ:t))`

### 4. 技术指标特征
许多经典的技术分析指标可以作为特征，例如：
*   **顺势指标**：`CCI = (典型价格 - 典型价格的N期移动平均) / (0.015 * 典型价格N期平均绝对偏差)`
*   **移动平均线**：`MA(λ, t) = (1/λ) * Σ Price(t-i)`, `i=0 to λ-1`
*   **变动率指标**：`ROC(t) = (Price(t) - Price(t-n)) / Price(t-n)`
*   **布林带**：利用移动平均和标准差构造通道。
*   **能量潮指标**：`EVM` 等。

**作业建议**：建立你自己的特征提取工具库，能够方便地计算多种技术指标，并将一个时间点转化为一个多维特征向量（常见维度为50或300维）。

## 特征与模型评估初步

如何验证一个特征是否有效？一个简单的方法是计算该特征（或特征组）与标签 `Y` 之间的相关性或解释度。对于回归问题，可以使用 `sklearn.metrics.r2_score` 来计算R平方值。

**注意**：在计算特征时，必须严格避免使用未来数据（即 `t+1` 时刻及之后的信息），否则会导致模型在回测中表现虚高，而在实际应用中失效。

## 机器学习模型介绍

当训练集 `(X_train, y_train)` 准备就绪后，我们就可以构建预测模型 `F`，使得 `y_pred = F(X)`。以下介绍几个在量化领域常用且有效的模型。

### 1. 随机森林
随机森林是一种集成学习算法，它构建多棵决策树，并通过投票或平均来提升预测准确率和稳定性。

**核心思想**：
*   **随机性1（数据）**：通过有放回抽样为每棵树生成不同的训练子集。
*   **随机性2（特征）**：在每棵树分裂节点时，随机选择一个特征子集进行最优划分。
*   **集成**：综合所有树的预测结果。

**代码示例**（分类与回归）：
```python
from sklearn.ensemble import RandomForestClassifier, RandomForestRegressor
# 分类
clf = RandomForestClassifier(n_estimators=100, max_depth=5, random_state=42)
clf.fit(X_train, y_train)
# 回归
reg = RandomForestRegressor(n_estimators=200, random_state=42)
reg.fit(X_train, y_train)
```

### 2. 梯度提升树
以 `XGBoost` 为代表的梯度提升树是另一种强大的集成模型。它通过串行地训练多棵树，每棵树学习之前所有树预测结果的残差。

**特点**：通常比随机森林有更高的预测精度，但训练是串行的。`XGBoost` 库提供了高效的实现。

**代码示例**：
```python
import xgboost as xgb
# 分类
model = xgb.XGBClassifier(n_estimators=100, learning_rate=0.1, max_depth=3)
model.fit(X_train, y_train)
# 回归
model = xgb.XGBRegressor(n_estimators=200, learning_rate=0.05)
model.fit(X_train, y_train)
```

### 3. 广义线性模型
*   **逻辑回归**：用于二分类问题的基础模型，常作为基准模型。
    `P(y=1|X) = 1 / (1 + exp(-W^T * X))`
*   **LASSO/岭回归**：用于回归问题，引入正则化防止过拟合。

**代码示例**：
```python
from sklearn.linear_model import LogisticRegression, Lasso, Ridge
# 逻辑回归
lr = LogisticRegression()
lr.fit(X_train, y_train)
# LASSO回归
lasso = Lasso(alpha=0.01)
lasso.fit(X_train, y_train)
```

### 4. 支持向量机
用于分类和回归，寻找最大化间隔的超平面。

**代码示例**：
```python
from sklearn.svm import SVC, SVR
# 分类
svc = SVC(kernel='rbf', C=1.0)
svc.fit(X_train, y_train)
# 回归
svr = SVR(kernel='linear')
svr.fit(X_train, y_train)
```

### 5. 人工神经网络
使用 `Keras` 可以快速构建多层感知机模型。深度学习适用于特征与目标之间存在复杂非线性关系的情况。

**代码示例**：
```python
from keras.models import Sequential
from keras.layers import Dense, Dropout
model = Sequential()
model.add(Dense(units=512, activation='relu', input_dim=X_train.shape[1]))
model.add(Dropout(0.2)) # 随机丢弃20%神经元，防止过拟合
model.add(Dense(units=512, activation='relu'))
model.add(Dropout(0.2))
model.add(Dense(units=1, activation='sigmoid')) # 二分类输出
model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])
model.fit(X_train, y_train, epochs=10, batch_size=128, validation_split=0.2)
```

## 模型调优与验证

**重要**：切勿直接使用模型的默认参数进行交易。必须对超参数进行调优。

**交叉验证**：是评估模型性能和选择超参数的黄金标准。将训练集分为多份，轮流将其中一份作为验证集，其余作为训练集，多次训练和验证以获取稳定的性能评估。

**超参数优化**：对于训练成本高的模型（如深度学习），可以使用贝叶斯优化等更智能的方法替代暴力网格搜索，以在有限的尝试中找到较优的参数组合。
