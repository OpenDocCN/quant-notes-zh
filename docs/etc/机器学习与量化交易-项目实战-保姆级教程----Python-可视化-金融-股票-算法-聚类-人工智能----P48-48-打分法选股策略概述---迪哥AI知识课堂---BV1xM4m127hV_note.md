# 机器学习与量化交易：P48：因子打分选股策略概述 📊

## 概述
在本节课中，我们将学习量化交易中一个常用的选股策略——因子打分法。我们将了解该策略的核心思想、实施步骤以及背后的逻辑，为后续的代码实战打下基础。

## 策略核心思想
上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看一个更综合的方法：因子打分选股。

该策略的核心思想是：**不再依赖单一因子，而是综合多个因子对股票进行整体评估**。其过程类似于学生考试，每门科目（因子）都有一个分数，最终根据总分进行排名，选择总分最高的股票。

我们可以用以下公式来描述其核心逻辑：
**股票总分 = 因子A得分 + 因子B得分 + 因子C得分 + ...**

## 策略实施步骤
以下是实施因子打分选股策略的关键步骤。

### 1. 确定因子与方向
在开始打分前，我们必须明确每个因子与股票收益率的相关性方向。这是策略的先验知识。

*   **正相关因子**：因子值越大，预期收益越高。例如：每股收益、净资产收益率。
*   **负相关因子**：因子值越小，预期收益越高。例如：市盈率、市净率。

这个方向性判断可以来源于金融研究报告，或通过我们之前学习的因子分析（如计算IC值）来验证。

![](img/14702328dbdc011ce5a3ac628d16b73f_1.png)

### 2. 为每个因子打分
对于每只股票，我们需要根据其每个因子的具体数值进行打分。打分规则取决于第一步确定的方向。

*   **对于正相关因子**：因子值越大，得分越高。
*   **对于负相关因子**：因子值越小，得分越高。

一种常见的打分方法是**分位数排名法**，即将所有股票在该因子上的数值进行排序，根据排名位置赋予分数。

### 3. 计算综合得分并选股
当所有因子都完成打分后，即可计算每只股票的综合得分。

综合得分计算公式为：
`综合得分 = 因子1得分 + 因子2得分 + ... + 因子N得分`

计算出所有股票的综合得分后，进行从高到低排序。最终，选择排名最靠前的一批股票（如前10或前20名）作为投资组合。
