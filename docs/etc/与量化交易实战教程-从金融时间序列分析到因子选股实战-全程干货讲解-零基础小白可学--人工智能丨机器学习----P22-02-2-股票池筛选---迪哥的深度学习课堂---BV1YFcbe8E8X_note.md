首先，在策略的构造函数中，我们需要定义我们的初始股票池。这里我们选择沪深300指数作为我们的基础股票池。

```python
def initialize(context):
    # 设定初始股票池为沪深300指数成分股
    context.stock_pool = ‘沪深300’
```

接下来，在每日交易开始前，我们需要对股票池进行预处理。这个步骤将在 `before_trading` 函数中完成。我们的目标是获取每只股票的财务数据，并根据特定指标进行筛选。

以下是实现股票筛选的核心步骤：

![](img/e828095b0c91d244347df50cc27710d0_15.png)

1.  **数据查询**：使用平台提供的 `query` 函数，指定我们要查询的财务指标字段。
2.  **条件过滤**：使用 `filter` 方法，确保我们只查询属于沪深300股票池的股票。
3.  **结果排序**：使用 `order_by` 方法，根据选定的财务指标（如营业收入）进行降序排列。
4.  **限制数量**：使用 `limit` 方法，只保留排名前N的股票作为最终的投资标的。

现在，让我们将这些步骤转化为具体的代码。首先，我们从查询财务数据开始。

```python
def before_trading(context):
    # 步骤1：构建查询，获取‘operating_revenue’（营业收入）指标
    q = query(
        fundamentals.financial_indicator.operating_revenue
    )
```

查询构建好后，我们需要添加过滤条件，确保只处理我们股票池里的股票。

```python
    # 步骤2：添加过滤条件，筛选出在沪深300指数中的股票
    q = q.filter(
        fundamentals.financial_indicator.stockcode.in_(context.stock_pool)
    )
```

过滤出目标股票后，我们需要根据营业收入对其进行排序，并选取最好的几只。

![](img/e828095b0c91d244347df50cc27710d0_17.png)

![](img/e828095b0c91d244347df50cc27710d0_19.png)

```python
    # 步骤3：按营业收入降序排列
    q = q.order_by(fundamentals.financial_indicator.operating_revenue.desc())

    # 步骤4：限制只取前10只股票
    q = q.limit(10)
```

![](img/e828095b0c91d244347df50cc27710d0_23.png)

最后，我们执行查询，并将结果存储在 `context` 中，以便在后续的交易逻辑中使用。为了验证筛选结果，我们可以先将其打印出来。

```python
    # 执行查询，获取DataFrame格式的结果
```
