# Python金融分析与量化交易实战：04-4-3：3-Sigma方法实例 📊

在本节课中，我们将要学习一种在数据预处理中常用的去极值方法——3-Sigma方法。我们将从正态分布的基本概念入手，理解3-Sigma方法的原理，并最终通过Python代码实现它，用于处理金融数据中的异常值。

## 概述

上一节我们介绍了其他几种去极值的方法，本节中我们来看看基于统计学原理的3-Sigma方法。它的核心思想是利用数据的均值和标准差来定义一个“正常”数据的范围，并将超出此范围的极值调整到边界上。

## 3-Sigma方法原理

![](img/e0db3612d14453c24f20c8f495824ed7_1.png)

3-Sigma方法的原理基于正态分布（也称高斯分布）。我们可以将正态分布理解为一个描述“正常”情况的概率模型。

例如，在银行贷款场景中，银行批准给你的贷款额度围绕一个均值上下波动。小幅度的波动（如多借或少借几块钱）是常见的、正常的情况。而大幅度的波动（如多借或少借十万八万）则是罕见的、不正常的情况，可以被视为“极值”或“离群点”。

在正态分布曲线中，越靠近均值中心，数据点出现的概率越高；越远离均值，出现的概率则越低。我们的目标是识别并处理那些落在偏远区域、概率极低的异常数据点。

**3-Sigma** 的具体含义是：以均值为中心，向左右各扩展3倍的标准差（σ），形成一个区间。根据统计学知识，在正态分布下，约有 **99.7%** 的数据会落在这个区间内。

![](img/e0db3612d14453c24f20c8f495824ed7_3.png)

因此，3-Sigma方法就是将这个区间（均值 ± 3σ）定义为数据的正常范围。我们将区间内的数据（约99.7%）视为正常数据，而将区间外的那0.3%的数据视为极值，并将其数值调整到区间的边界上。

![](img/e0db3612d14453c24f20c8f495824ed7_5.png)

以下是正态分布中不同Sigma区间所覆盖的数据比例：
*   **均值 ± 1σ**：覆盖约 **68.3%** 的数据。
*   **均值 ± 2σ**：覆盖约 **95.4%** 的数据。
*   **均值 ± 3σ**：覆盖约 **99.7%** 的数据。

## 代码实现

![](img/e0db3612d14453c24f20c8f495824ed7_7.png)

理解了原理后，我们开始用Python代码实现3-Sigma方法。实现步骤非常清晰：

![](img/e0db3612d14453c24f20c8f495824ed7_9.png)

1.  计算指定数据列的 **均值（mean）** 和 **标准差（std）**。
2.  根据公式计算上限和下限：
    *   **上限 = 均值 + N × 标准差**
    *   **下限 = 均值 - N × 标准差**
    （其中N通常取3，代表3-Sigma）
3.  将原始数据中所有大于上限的值设置为上限，所有小于下限的值设置为下限。

![](img/e0db3612d14453c24f20c8f495824ed7_11.png)

![](img/e0db3612d14453c24f20c8f495824ed7_13.png)

以下是具体的函数实现：

![](img/e0db3612d14453c24f20c8f495824ed7_15.png)

```python
def filter_extreme_3sigma(series, n=3):
    """
    使用3-Sigma方法去除极值。
    参数:
        series: 待处理的数据序列（Pandas Series）。
        n: Sigma的倍数，默认为3。
    返回:
        处理后的数据序列。
    """
    # 1. 计算均值和标准差
    mean = series.mean()
    std = series.std()

    # 2. 计算上限和下限
    high = mean + n * std
    low = mean - n * std

    # 3. 将超出范围的值调整到边界
    series = series.clip(lower=low, upper=high)

    return series, low, high
```

## 核心要点总结

![](img/e0db3612d14453c24f20c8f495824ed7_35.png)

本节课中我们一起学习了3-Sigma去极值法。我们来总结一下核心要点：

![](img/e0db3612d14453c24f20c8f495824ed7_37.png)

![](img/e0db3612d14453c24f20c8f495824ed7_39.png)

*   **目的**：处理数据中的异常值（极值），使其不影响后续分析。
*   **原理**：基于正态分布假设，认为距离均值超过3倍标准差的数据点属于小概率事件，应予以处理。
*   **操作**：并非直接删除极值，而是定义一个合理的上下界（**上限 = 均值 + 3σ， 下限 = 均值 - 3σ**），并将所有超出界限的数据“拉回”到边界上。
*   **在流程中的位置**：这是我们数据预处理“三步走”策略（去极值 -> 标准化 -> 中性化）中的第一步。

![](img/e0db3612d14453c24f20c8f495824ed7_41.png)

掌握3-Sigma方法，能为你的金融数据分析和量化交易策略构建提供一个稳健的数据基础。
