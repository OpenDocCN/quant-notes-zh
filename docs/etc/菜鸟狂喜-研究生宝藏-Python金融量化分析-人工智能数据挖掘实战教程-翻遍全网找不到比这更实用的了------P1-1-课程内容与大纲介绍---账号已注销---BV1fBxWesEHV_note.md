## 课程核心模块概述

上一节我们介绍了课程的整体框架，本节中我们来看看具体的三大模块内容。

![](img/648a33e5806f78699a2027a07a99d4ac_5.png)

![](img/648a33e5806f78699a2027a07a99d4ac_7.png)

![](img/648a33e5806f78699a2027a07a99d4ac_9.png)

课程内容主要涉及三大模块：Python数据分析与处理、金融数据分析以及量化交易策略。

![](img/648a33e5806f78699a2027a07a99d4ac_11.png)

![](img/648a33e5806f78699a2027a07a99d4ac_13.png)

以下是三大模块的详细介绍：

![](img/648a33e5806f78699a2027a07a99d4ac_15.png)

1.  **Python数据分析与处理**：本模块将讲解如何使用Python处理和分析数据。核心内容包括数据操作、数据清洗和数据分析方法。
2.  **金融数据分析**：本模块将聚焦于股票等金融数据。我们将学习如何对金融时间序列数据进行操作、分析和建模。
3.  **量化交易策略**：本模块将介绍如何设计和测试交易策略。核心内容是**策略回测**，即在历史数据中运行策略，以分析其表现优劣。

![](img/648a33e5806f78699a2027a07a99d4ac_17.png)

## 课程风格与工具介绍

![](img/648a33e5806f78699a2027a07a99d4ac_19.png)

了解了核心模块后，我们来看看课程的教学风格和将使用的主要工具。

![](img/648a33e5806f78699a2027a07a99d4ac_21.png)

![](img/648a33e5806f78699a2027a07a99d4ac_23.png)

本课程是纯实战导向，没有PPT，不讲授抽象的金融理论或炒股方法。重点在于如何使用Python代码实现具体任务，包括数据分析、回测和策略开发。所有讲解均基于实际数据案例进行。

![](img/648a33e5806f78699a2027a07a99d4ac_25.png)

在讲解过程中，我们将主要使用以下两类工具：

*   **Jupyter Notebook**：用于演示Python代码、进行数据分析和案例教学。我们将学习如何使用关键的Python工具包，如用于数值计算的 **`NumPy`** 和用于数据分析的 **`Pandas`**。
    ```python
    # 示例：导入核心工具包
    import numpy as np
    import pandas as pd
    ```
*   **量化回测平台**：用于将编写好的交易策略在历史数据中运行，以评估其表现。我们将学习如何在此平台上编写策略、执行回测，并分析结果指标，如**回测收益**、**超额收益**（相对于基准的收益）以及详细的交易记录和持仓情况。

![](img/648a33e5806f78699a2027a07a99d4ac_27.png)

## 详细课程大纲

![](img/648a33e5806f78699a2027a07a99d4ac_29.png)

最后，我们来详细看一下整个课程的学习路径。课程大纲同样分为三个循序渐进的模块。

![](img/648a33e5806f78699a2027a07a99d4ac_31.png)

以下是课程的具体大纲安排：

![](img/648a33e5806f78699a2027a07a99d4ac_33.png)

![](img/648a33e5806f78699a2027a07a99d4ac_35.png)

1.  **第一模块：Python必备工具包实战**：为零基础学员打下基础。内容包括Python核心知识点、环境配置与工具安装，并重点讲解`NumPy`和`Pandas`这两个核心工具包的使用。
2.  **第二模块：Python金融数据分析实战**：过渡到金融领域。我们将使用真实的股票数据，学习金融数据分析、建模与统计方法，并首次在量化平台上完成一个完整的回测项目，评估策略的历史表现。
3.  **第三模块：量化交易策略深入实战**：深入探讨量化交易。本模块将讲解经典的量化交易策略，并学习在实际操作中如何处理数据、分析因子、改进策略，最终目标是构建能在历史回测中获得良好收益的策略。

![](img/648a33e5806f78699a2027a07a99d4ac_37.png)

![](img/648a33e5806f78699a2027a07a99d4ac_39.png)
