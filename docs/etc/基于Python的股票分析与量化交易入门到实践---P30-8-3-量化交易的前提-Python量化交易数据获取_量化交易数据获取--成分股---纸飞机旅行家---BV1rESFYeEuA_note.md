在本节课中，我们将要学习如何通过聚宽量化交易平台获取股票市场中的“成分股”数据。成分股是构成指数、行业或概念板块的股票集合，是量化分析中的重要基础数据。

上一节我们介绍了如何获取股票的财务数据，本节中我们来看看如何获取这些构成市场不同维度的股票列表。

## 指数成分股函数

首先，我们向大家介绍指数成分股函数。该函数用于查询指定指数在指定日期下的可交易成分股列表。

其核心功能是：给定一个指数代码和一个日期，返回该指数所包含的股票代码列表。在聚宽平台中，对应的查询函数是 `get_index_stocks`。

该函数有两个主要参数：
*   **index_symbol**：指数代码。例如，沪深300指数代码为 `'000300.XSHG'`。
*   **date**：查询日期。这是一个重要参数，因为指数的成分股会定期调整，并非一成不变。

函数的返回值是一个股票代码的列表 `list`。

## 行业成分股函数

接下来，我们看看行业成分股函数。该函数用于获取指定行业分类下的所有股票。

其核心功能是：根据行业分类代码，返回属于该行业的所有股票列表。在聚宽平台中，对应的查询函数是 `get_industry_stocks`。

以下是该函数的关键参数：
*   **industry_code**：行业编码。聚宽平台支持多种行业分类标准，如证监会行业分类、申万行业分类以及聚宽自有的行业分类。每种分类都有其对应的编码体系。
*   **date**：查询日期。

与指数成分股函数类似，该函数也返回一个股票代码的列表 `list`。

## 概念成分股函数

最后，我们通过概念成分股函数来获取股票的概念板块数据。概念板块是根据市场热点、政策主题等形成的股票分组，如“5G”、“风电”、“元宇宙”等。

其核心功能是：根据概念板块代码，返回属于该概念的所有股票列表。在聚宽平台中，对应的查询函数是 `get_concept_stocks`。

![](img/64f19995851c63ea02629fa58debc10b_1.png)

以下是该函数的关键参数：
*   **concept_code**：概念板块编码。聚宽平台提供了详细的概念板块列表及其编码。
*   **date**：查询日期。

该函数同样返回一个股票代码的列表 `list`。

## 代码实战

接下来我们进入本节的代码实战，让大家直观地了解这三个查询函数的使用方法。

首先，我们进行指数成分股的查询。以下代码演示如何获取沪深300指数的成分股（为简洁起见，只显示前10只）。

```python
# 导入聚宽函数库
from jqdata import *

# 查询沪深300指数成分股（默认使用当前日期）
index_stocks_list = get_index_stocks('000300.XSHG')
print("沪深300指数成分股（前10只）:", index_stocks_list[:10])
```

接下来，我们查询行业成分股。以下以“互联网和相关服务”行业为例。

```python
# 查询行业成分股，例如：互联网和相关服务 (聚宽行业编码示例，实际编码需查询文档)
# 假设行业编码为 'I64'
industry_stocks_list = get_industry_stocks('I64')
print("互联网和相关服务行业成分股（前10只）:", industry_stocks_list[:10])
```

![](img/64f19995851c63ea02629fa58debc10b_3.png)

最后，我们来展示概念成分股的获取。以下以“风电”概念板块为例。

```python
# 查询概念成分股，例如：风电概念 (概念编码需查询聚宽数据字典)
# 假设概念编码为 'GN001'，日期为 '2023-06-01'
concept_stocks_list = get_concept_stocks('GN001', date='2023-06-01')
print("风电概念成分股（前10只）:", concept_stocks_list[:10])
```

## 本章小结

本节课中我们一起学习了在聚宽量化交易平台中获取成分股数据的三种常用函数。

首先是指数成分股函数 `get_index_stocks`，它方便我们获取特定指数（如沪深300）的构成股票列表。接着是行业成分股函数 `get_industry_stocks`，它帮助我们根据证监会、申万或聚宽等分类标准，筛选出属于某个行业的股票。最后是概念成分股函数 `get_concept_stocks`，它让我们能够追踪市场热点主题，获取相关概念板块的股票集合。
