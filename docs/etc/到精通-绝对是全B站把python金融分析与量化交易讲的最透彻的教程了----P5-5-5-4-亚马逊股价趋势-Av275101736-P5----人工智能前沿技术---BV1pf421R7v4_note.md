## 构建基础预测模型

基于观察到的数据，我们接下来构建一个基础的Prophet预测模型。这个模型将帮助我们理解数据的潜在趋势。

![](img/7c7b8bf6dfceee6ed1958c23858979d3_35.png)

以下是构建模型的代码框架：

![](img/7c7b8bf6dfceee6ed1958c23858979d3_37.png)

![](img/7c7b8bf6dfceee6ed1958c23858979d3_39.png)

```python
# 1. 准备数据：将日期列命名为‘ds’，股价列命名为‘y’
df = pd.read_csv('amzn_stock.csv')
df['ds'] = pd.to_datetime(df['Date'])
df['y'] = df['Close']

# 2. 初始化并拟合Prophet模型
model = Prophet()
model.fit(df)

# 3. 创建未来90天的日期数据框并进行预测
future = model.make_future_dataframe(periods=90)
forecast = model.predict(future)

# 4. 绘制预测结果
fig = model.plot(forecast)
plt.show()
```

运行模型后，我们会得到一张图表。其中，黑色点代表历史真实股价，绿色线代表模型对未来90天的预测趋势。初步观察发现，模型预测的趋势线通常比真实数据的波动更加平缓。这是为了防止模型过度拟合（overfitting）历史数据中的每一个突变点，从而保持预测的稳健性。

## 模型性能的初步评估

构建模型后，我们需要评估其预测性能。评估主要从两个维度进行：预测数值的准确性和预测趋势方向的正确性。

![](img/7c7b8bf6dfceee6ed1958c23858979d3_47.png)

以下是评估模型的具体方法：

1.  **划分数据集**：将2014年至2017年的数据作为训练集，将2017年至2018年的数据作为测试集。
2.  **计算预测误差**：在测试集上，计算模型预测值（$\hat{y}$）与真实值（$y$）之间的平均绝对误差。公式为：$\text{error} = |y - \hat{y}|$。分别计算训练集和测试集的平均误差，以判断模型是否过拟合或欠拟合。
3.  **评估趋势方向**：计算模型预测的股价每日涨跌方向（上升或下降）与真实涨跌方向一致的百分比。这通过比较连续两天的差值符号来实现。若预测趋势与真实趋势一致，则计为正确。

![](img/7c7b8bf6dfceee6ed1958c23858979d3_49.png)

通过运行评估代码，我们可能得到类似以下的结果：
*   训练集平均误差：18美元
*   测试集平均误差：160美元
*   上升趋势预测正确率：56%
*   下降趋势预测正确率：43%

![](img/7c7b8bf6dfceee6ed1958c23858979d3_51.png)

这些结果表明，当前的基础模型在测试集上的预测误差较大，且趋势预测准确率接近随机猜测（50%），说明模型有很大的改进空间。可视化图表也清晰显示，预测的蓝色曲线较为平缓，而真实的黑色曲线波动更大，两者存在明显差距。
