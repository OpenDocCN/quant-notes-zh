在本节课中，我们将要学习如何分析金融时间序列的连续变化情况。我们将探讨为什么简单的增长率累加会失效，并引入对数方法来正确计算连续时间的复合增长率，最终计算出初始投资（例如1元）随时间推移的价值变化。

## 概述

上一节我们介绍了如何计算时间序列的简单差异和增长率。本节中我们来看看，当我们面对一个连续的时间序列（如股票每日价格），并希望计算其在一段时间内的整体复合增长时，应该如何操作。我们将发现，直接累加每日增长率是无效的，需要借助数学中的对数变换。

## 为什么增长率不能直接累加？🤔

![](img/c872d8005e3ccfcaf24db85594e59c29_4.png)

首先，我们通过一个例子来理解问题所在。

假设有一只股票：
*   第一天价格：100元
*   第二天价格：50元
*   第三天价格：75元

如果分别计算每日增长率：
*   第二天增长率 = (50 - 100) / 100 = **-0.5**
*   第三天增长率 = (75 - 50) / 50 = **+0.5**

![](img/c872d8005e3ccfcaf24db85594e59c29_6.png)

若将这两个增长率直接相加：-0.5 + 0.5 = **0**。这个结果暗示股票从第一天到第三天既没涨也没跌。

![](img/c872d8005e3ccfcaf24db85594e59c29_8.png)

但观察实际价格：从100元变为75元，显然是下跌了。这说明**简单的增长率累加无法反映资产价值的真实连续变化**。

## 解决方案：引入对数收益率 📊

![](img/c872d8005e3ccfcaf24db85594e59c29_10.png)

为了正确计算连续时间内的复合变化，我们需要借助数学中的对数。核心思想是计算**对数收益率**，其累加等价于计算期初与期末的总比率。

对于连续的两天，对数收益率的计算公式为：
`log_return = log(后一天价格 / 前一天价格)`

![](img/c872d8005e3ccfcaf24db85594e59c29_12.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_14.png)

以上述股票为例：
*   第一到二天的对数收益率 = log(50 / 100)
*   第二到三天的对数收益率 = log(75 / 50)

![](img/c872d8005e3ccfcaf24db85594e59c29_16.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_18.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_20.png)

将对数收益率累加：
`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/c872d8005e3ccfcaf24db85594e59c29_22.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_24.png)

可以看到，**对数收益率的累加，最终等于最后一天价格与第一天价格比值的对数**。这完美地刻画了从期初到期末的整体变化。

![](img/c872d8005e3ccfcaf24db85594e59c29_26.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_28.png)

形象地说，如果你在第一天投入1元，经过一系列涨跌，将对数收益率累加后再进行指数运算还原，就能得到这1元在最后一天相当于多少钱。

![](img/c872d8005e3ccfcaf24db85594e59c29_30.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_32.png)

## 在Python中实现连续变化分析 💻

接下来，我们看看如何在Pandas中实现上述计算。

![](img/c872d8005e3ccfcaf24db85594e59c29_34.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_36.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_38.png)

### 1. 计算每日对数收益率

我们需要计算`后一天数据 / 前一天数据`。这里使用`.shift(1)`方法将数据整体向上移动一行，从而方便地获取“前一天”的数据。

![](img/c872d8005e3ccfcaf24db85594e59c29_40.png)

```python
# 假设 data 是一个包含股价的Pandas Series或DataFrame列
# 计算每日价格比（后一天/前一天）
price_ratio = data / data.shift(1)

![](img/c872d8005e3ccfcaf24db85594e59c29_42.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_44.png)

# 计算对数收益率
log_returns = np.log(price_ratio)
```

![](img/c872d8005e3ccfcaf24db85594e59c29_46.png)

### 2. 计算累积收益并还原为价值

![](img/c872d8005e3ccfcaf24db85594e59c29_48.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_50.png)

我们的目标是计算“第一天投入的1元，在后续每一天相当于多少钱”。这需要对对数收益率进行累积求和（`cumsum`），然后通过指数函数`exp`还原为实际价值倍数。

```python
# 对对数收益率进行累积求和
cumulative_log_returns = log_returns.cumsum()

![](img/c872d8005e3ccfcaf24db85594e59c29_52.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_54.png)

# 通过指数运算还原为实际价值（以第一天的1元为基准）
cumulative_value = np.exp(cumulative_log_returns)

![](img/c872d8005e3ccfcaf24db85594e59c29_56.png)

# 绘制结果图
cumulative_value.plot(figsize=(16, 8))
```

![](img/c872d8005e3ccfcaf24db85594e59c29_58.png)

执行以上代码后，我们将得到一张图表。图表中每条曲线都从1开始，展示了不同股票（或资产）的初始1元投资，随时间推移所增长到的价值。例如，曲线终点值为10，就意味着最初的1元变成了10元。

![](img/c872d8005e3ccfcaf24db85594e59c29_60.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_62.png)

## 核心操作总结

以下是本节课涉及的金融时间序列核心分析操作：

![](img/c872d8005e3ccfcaf24db85594e59c29_64.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_66.png)

*   **计算差异值**：使用`.diff()`方法，计算当前值与前一值的绝对差。
*   **计算简单增长率**：公式为`(当前值 - 前值) / 前值`。
*   **计算对数收益率（用于连续分析）**：公式为`np.log(当前值 / 前值)`。这是分析复合增长的关键。
*   **计算累积价值**：通过对数收益率的累积和（`cumsum`）与指数还原（`np.exp`），得到以期初为基准的连续价值变化。

![](img/c872d8005e3ccfcaf24db85594e59c29_68.png)

![](img/c872d8005e3ccfcaf24db85594e59c29_70.png)
