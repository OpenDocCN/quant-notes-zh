# 量化交易教程：31：3Sigma方法实例 📊

在本节课中，我们将要学习一种在数据预处理中常用的去极值方法——3Sigma方法。该方法基于统计学中的正态分布原理，用于识别和处理数据中的异常值。

## 概述

上一节我们介绍了去极值的概念和目的，本节中我们来看看一种基于统计学原理的具体方法：3Sigma方法。其核心思想是利用数据的均值和标准差来定义一个“正常”数据的范围，并将超出此范围的值视为极值进行处理。

## 3Sigma方法的原理

首先，我们需要理解正态分布（也称高斯分布）。正态分布是一种常见的概率分布，其形状呈对称的钟形曲线。

为了便于理解，可以想象一个日常例子：你去银行贷款，银行批准给你的额度围绕一个平均金额上下波动。小幅度的多贷或少贷是常见的（正常情况），但突然多贷或少贷一个非常大的数额则是不常见的（异常情况）。

![](img/9b0e54b15fd40134a72327295e5bf8d2_1.png)

在正态分布曲线中，曲线的中心是数据的**均值**。数据点离均值越远，出现的概率就越低。下图展示了一个标准正态分布：

![](img/9b0e54b15fd40134a72327295e5bf8d2_1.png)

图中，红色阴影区域靠近均值，面积较大，代表事件发生的概率较高，属于正常数据范围。而越向两侧（远离均值），阴影面积越小，代表事件发生的概率越低，这些区域就更可能包含离群的极值点。

**3Sigma方法**正是基于这一原理。它通过计算均值（μ）和标准差（σ），构建一个以均值为中心的区间。对于正态分布的数据，有这样一个经验法则：
*   数据落在 **μ ± 1σ** 范围内的概率约为68.2%。
*   数据落在 **μ ± 2σ** 范围内的概率约为95.4%。
*   数据落在 **μ ± 3σ** 范围内的概率约为99.7%。

![](img/9b0e54b15fd40134a72327295e5bf8d2_3.png)

因此，3Sigma方法将 **μ - 3σ** 定义为下限，**μ + 3σ** 定义为上限。它认为大约99.7%的数据应落在此区间内，而剩余约0.3%的数据则被视为极值，并对其进行调整（如缩放到边界值），而不是直接删除。

## 3Sigma方法的代码实现

理解了原理后，我们来看看如何用代码实现它。实现过程非常简单，主要分为三步：计算数据的均值和标准差，然后根据指定的倍数N（例如3）计算上下限。

![](img/9b0e54b15fd40134a72327295e5bf8d2_7.png)

以下是具体的代码实现步骤：

![](img/9b0e54b15fd40134a72327295e5bf8d2_9.png)

首先，我们定义一个函数，它接收一列数据和倍数N作为参数。

![](img/9b0e54b15fd40134a72327295e5bf8d2_10.png)

```python
def three_sigma(data_series, n=3):
    # 1. 计算均值和标准差
    mean = data_series.mean()
    std = data_series.std()
    
    # 2. 根据公式计算上限和下限
    # 上限公式：μ + n * σ
    upper_limit = mean + n * std
    # 下限公式：μ - n * σ
    lower_limit = mean - n * std
    
    # 3. 返回计算出的边界
    return lower_limit, upper_limit
```

代码说明：
1.  使用 `.mean()` 方法计算传入数据序列的均值。
2.  使用 `.std()` 方法计算传入数据序列的标准差。
3.  根据公式 **下限 = 均值 - N * 标准差** 和 **上限 = 均值 + N * 标准差** 进行计算。
4.  函数返回计算得到的下限和上限。

在实际应用中，绿色参考线（如中位数）通常不变，而蓝色线（3Sigma方法计算的边界）的位置会根据数据分布和选择的N值而变化。例如，N=1时边界可能在均值±50的位置，N=3时边界则会更宽。

不同的去极值方法（如之前介绍的百分位法）可能会产生略有差异的结果。选择哪种方法取决于具体的任务需求和数据特性。3Sigma方法假设数据大致服从正态分布，这是使用时需要注意的前提。

## 总结

本节课中我们一起学习了3Sigma去极值方法。我们来回顾一下核心要点：

![](img/9b0e54b15fd40134a72327295e5bf8d2_21.png)

1.  **原理基础**：该方法基于**正态分布**的统计学特性。其核心公式为：
    *   下限：**lower_limit = μ - n * σ**
    *   上限：**upper_limit = μ + n * σ**
    *   其中，`μ` 是均值，`σ` 是标准差，`n` 是倍数（常取3）。

![](img/9b0e54b15fd40134a72327295e5bf8d2_23.png)

2.  **方法本质**：去极值不是删除数据，而是通过定义合理的**上界**与**下界**，将超出范围的数据规范到边界之内，从而减少异常值对整体分析的影响。
