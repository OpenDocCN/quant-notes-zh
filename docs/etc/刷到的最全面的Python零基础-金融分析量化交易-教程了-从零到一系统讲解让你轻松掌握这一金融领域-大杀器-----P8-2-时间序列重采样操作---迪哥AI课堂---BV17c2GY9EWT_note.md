## 重采样操作

上一节我们介绍了时间序列的基本概念，本节中我们来看看如何改变数据的时间频率，即重采样。

![](img/f0dc773b3d78b5c3985590566c340f95_5.png)

![](img/f0dc773b3d78b5c3985590566c340f95_7.png)

![](img/f0dc773b3d78b5c3985590566c340f95_9.png)

重采样是指将数据从一个时间频率转换到另一个时间频率。例如，原始数据可能是按天统计的，我们可以将其转换为按周、按月或按年统计。在Pandas中，这通过`resample`方法实现。

![](img/f0dc773b3d78b5c3985590566c340f95_11.png)

以下是重采样的基本步骤：
1.  指定新的时间频率（如‘W’代表周，‘M’代表月）。
2.  指定聚合方法（如均值、第一个值、中位数等）。

![](img/f0dc773b3d78b5c3985590566c340f95_13.png)

```python
# 假设df是一个按天索引的DataFrame
# 按周重采样，并计算每周的均值
weekly_data = df.resample(‘W’).mean()
```

执行重采样后，我们便得到了转换频率后的结果。在指定频率时，可以是一周、两周或任何其他周期。

![](img/f0dc773b3d78b5c3985590566c340f95_15.png)

### 标签位置参数

![](img/f0dc773b3d78b5c3985590566c340f95_17.png)

在重采样过程中，还有一个重要参数是`label`，它决定了新时间点的标签是使用时间窗口的起始点（‘left’）还是结束点（‘right’）。

![](img/f0dc773b3d78b5c3985590566c340f95_19.png)

![](img/f0dc773b3d78b5c3985590566c340f95_21.png)

```python
# 使用时间窗口的结束点作为标签（默认）
monthly_data_right = df.resample(‘M’, label=‘right’).mean()
# 使用时间窗口的起始点作为标签
monthly_data_left = df.resample(‘M’, label=‘left’).mean()
```

![](img/f0dc773b3d78b5c3985590566c340f95_23.png)

![](img/f0dc773b3d78b5c3985590566c340f95_25.png)

`label`参数控制着聚合后数据的索引标签。例如，按月重采样时，`label=‘right’`会用该月的最后一天（如1月31日）作为标签，而`label=‘left’`则会用该月的第一天（如1月1日）。在金融分析中，通常使用`‘right’`，因为它更符合我们观察周期末数据的逻辑。当然，具体选择需根据分析任务的需求而定。

## 时间窗口（滑动窗口）操作

了解了如何改变数据频率后，我们再来看看如何在时间序列上应用滑动窗口进行分析。

![](img/f0dc773b3d78b5c3985590566c340f95_27.png)

时间窗口操作，也称为滑动窗口，是指在一个序列上，以一个固定长度的窗口（如10天）依次滑动，并对每个窗口内的数据进行计算。例如，第一个窗口包含第1到第10天的数据，第二个窗口包含第2到第11天的数据，依此类推。

![](img/f0dc773b3d78b5c3985590566c340f95_29.png)

以下是执行滑动窗口操作的步骤：
1.  首先，确保数据没有缺失值，否则会影响窗口计算。
2.  使用`rolling`方法创建窗口对象。
3.  对窗口对象应用聚合函数（如`min`, `max`, `mean`, `std`）。

```python
# 1. 处理缺失值
df_clean = df.dropna()

![](img/f0dc773b3d78b5c3985590566c340f95_31.png)

![](img/f0dc773b3d78b5c3985590566c340f95_33.png)

# 2. 定义窗口大小，例如10
window_size = 10

![](img/f0dc773b3d78b5c3985590566c340f95_35.png)

# 3. 对某一列数据应用滑动窗口，并计算每个窗口的最小值
rolling_min = df_clean[‘column_name’].rolling(window=window_size).min()
```

![](img/f0dc773b3d78b5c3985590566c340f95_37.png)

初始的几个窗口结果可能是`NaN`，因为数据不足以填满整个窗口长度，这是正常现象。

![](img/f0dc773b3d78b5c3985590566c340f95_39.png)

![](img/f0dc773b3d78b5c3985590566c340f95_41.png)

![](img/f0dc773b3d78b5c3985590566c340f95_43.png)

### 计算多个窗口统计量

![](img/f0dc773b3d78b5c3985590566c340f95_45.png)

![](img/f0dc773b3d78b5c3985590566c340f95_47.png)

我们可以同时对每个窗口计算多个统计指标，并将结果添加到原始数据中。

![](img/f0dc773b3d78b5c3985590566c340f95_49.png)

```python
# 在清理后的数据上计算多个窗口统计量
df_clean[‘rolling_min’] = df_clean[‘column_name’].rolling(window=window_size).min()
df_clean[‘rolling_max’] = df_clean[‘column_name’].rolling(window=window_size).max()
df_clean[‘rolling_mean’] = df_clean[‘column_name’].rolling(window=window_size).mean()
df_clean[‘rolling_std’] = df_clean[‘column_name’].rolling(window=window_size).std()

![](img/f0dc773b3d78b5c3985590566c340f95_51.png)

![](img/f0dc773b3d78b5c3985590566c340f95_53.png)

# 查看结果
print(df_clean.tail())
```

![](img/f0dc773b3d78b5c3985590566c340f95_55.png)

![](img/f0dc773b3d78b5c3985590566c340f95_57.png)

这样，我们就得到了每个滑动窗口内的最小值、最大值、均值和标准差。这有助于我们分析数据在短期内的波动和趋势。

![](img/f0dc773b3d78b5c3985590566c340f95_59.png)

**重要提示**：在进行窗口操作前，务必先处理缺失值（`dropna()`）。如果数据中存在`NaN`，会导致整个窗口的计算结果变为`NaN`，影响分析结果。数据预处理是许多数据分析任务的第一步。

![](img/f0dc773b3d78b5c3985590566c340f95_61.png)

本节课中我们一起学习了时间序列的两个关键操作：
1.  **重采样**：通过`resample()`方法改变数据的时间频率（如日变周、月），并使用`label`参数控制时间标签的显示方式。
2.  **时间窗口操作**：通过`rolling()`方法创建滑动窗口，可以计算窗口内数据的各种统计特征（如最小值、最大值、均值），这是分析短期趋势和波动的有力工具。
