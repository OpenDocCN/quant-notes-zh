![](img/70c2477c92f4ba46b04b7db472d606ee_4.png)

## 什么是最大回撤？

上一节我们介绍了多种策略评估指标，本节中我们来看看最大回撤。它描述的是在投资过程中，资产净值从最高点到之后最低点的最大跌幅。

最大回撤不能仅从字面理解。它描述的是投资过程中“最惨”的阶段。在炒股过程中，价格有涨有跌。从某个高点开始，持续下跌到最低点的那个区间，就是最大回撤区间。

![](img/70c2477c92f4ba46b04b7db472d606ee_4.png)

## 最大回撤的公式与含义

为了更清晰地理解，我们通过公式来看。假设 `Pi` 代表某一天的股价，`Pj` 代表 `i` 天之后的某一天股价（`j > i`）。

最大回撤的计算目标是：在整个数据中，找出哪两个点之间的下跌幅度最大。这个下跌幅度用百分比表示，公式为：

**`最大回撤 = max( (Pi - Pj) / Pi )`**，其中 `i < j`。

这个指标评估的不是策略的绝对好坏，而是策略的风险。风险意味着，即使一个策略整体是盈利的，中间也可能存在一段让你资产大幅缩水的痛苦时期。这段时期就是最大回撤区间。

![](img/70c2477c92f4ba46b04b7db472d606ee_6.png)

例如，2008年股市从6000多点跌至1600多点，就是一次巨大的回撤，导致许多人承受了巨额亏损。最大回撤区间就是描述在某一段时间内，你的亏损有多严重。

![](img/70c2477c92f4ba46b04b7db472d606ee_8.png)

![](img/70c2477c92f4ba46b04b7db472d606ee_6.png)

在我们的回测策略中，系统会自动统计并标识出回测期间内最大回撤发生的区间。

## 如何用Python计算最大回撤？

![](img/70c2477c92f4ba46b04b7db472d606ee_14.png)

接下来，我们看看如何用代码计算最大回撤。核心思路是：遍历所有可能的时间点组合，找出最大的跌幅百分比。

![](img/70c2477c92f4ba46b04b7db472d606ee_16.png)

![](img/70c2477c92f4ba46b04b7db472d606ee_18.png)

以下是计算最大回撤的关键步骤和示例代码：

![](img/70c2477c92f4ba46b04b7db472d606ee_20.png)

首先，我们需要一个计算“滚动最大值”的函数，即计算到当前日期为止的历史最高价。Pandas库中的 `cummax()` 函数可以完成这个任务。

![](img/70c2477c92f4ba46b04b7db472d606ee_22.png)

```python
import pandas as pd

![](img/70c2477c92f4ba46b04b7db472d606ee_24.png)

![](img/70c2477c92f4ba46b04b7db472d606ee_26.png)

# 创建一个示例的股价序列
price_series = pd.Series([100, 105, 102, 98, 110, 108, 95])
print("原始价格序列:")
print(price_series)

![](img/70c2477c92f4ba46b04b7db472d606ee_28.png)

![](img/70c2477c92f4ba46b04b7db472d606ee_30.png)

# 计算滚动历史最大值
rolling_max = price_series.cummax()
print("\n滚动历史最大值序列:")
print(rolling_max)
```

![](img/70c2477c92f4ba46b04b7db472d606ee_32.png)

执行上述代码，`rolling_max` 序列会显示每个时间点对应的历史最高价。

![](img/70c2477c92f4ba46b04b7db472d606ee_34.png)

![](img/70c2477c92f4ba46b04b7db472d606ee_36.png)

计算最大回撤的完整过程如下：
1.  计算每日对应的历史最高价序列。
2.  用历史最高价减去当日的实际价格，得到每日的“潜在亏损额”。
3.  将这个“潜在亏损额”除以对应的历史最高价，得到每日的“回撤率”。
4.  在所有“回撤率”中找出最大值，即为最大回撤。

```python
# 接续上面的代码
# 计算每日回撤：(历史最高价 - 当日价) / 历史最高价
drawdown = (rolling_max - price_series) / rolling_max

![](img/70c2477c92f4ba46b04b7db472d606ee_38.png)

![](img/70c2477c92f4ba46b04b7db472d606ee_40.png)

print("\n每日回撤率序列:")
print(drawdown)

![](img/70c2477c92f4ba46b04b7db472d606ee_42.png)

# 计算最大回撤
max_drawdown = drawdown.max()
print(f"\n最大回撤为: {max_drawdown:.2%}")
```

在实际的股票数据中，计算过程是相同的。我们以一段真实的收盘价数据为例：

![](img/70c2477c92f4ba46b04b7db472d606ee_44.png)

```python
# 假设 df 是一个包含收盘价‘close’列的DataFrame
# 计算收盘价的滚动最大值
df['rolling_max'] = df['close'].cummax()
# 计算每日回撤
df['daily_drawdown'] = (df['rolling_max'] - df['close']) / df['rolling_max']
# 找出最大回撤
max_drawdown_value = df['daily_drawdown'].max()
print(f"最大回撤是: {max_drawdown_value:.2%}")
```

*   **最大回撤**衡量的是资产从历史最高点跌到之后最低点的最大百分比跌幅。
*   它是一个**风险指标**，反映了投资策略可能经历的最大连续亏损程度，帮助投资者了解策略的潜在波动性和心理承受压力。
*   在Python中，我们可以通过计算价格序列的**滚动最大值（`cummax()`）**，然后套用公式轻松计算出最大回撤。
