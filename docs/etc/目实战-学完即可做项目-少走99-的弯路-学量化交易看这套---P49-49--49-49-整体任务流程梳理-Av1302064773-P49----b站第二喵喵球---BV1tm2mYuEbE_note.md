## 打分法的基本思路 🧮

上一节我们介绍了如何获取和预处理因子数据。本节中我们来看看如何利用这些数据对股票进行综合评价，即“打分”。

打分法的核心思想是：为每个选定的因子（指标）设定评分规则，根据每只股票在该因子上的表现给予分数，最后将所有因子的分数加总，得到该股票的综合得分。得分最高的股票将被选入投资组合。

以下是打分法的两个关键步骤：

1.  **确定因子方向**：明确每个因子是“越大越好”（例如收益率、增长率）还是“越小越好”（例如负债率、波动率）。
2.  **设计评分规则**：根据因子值的分布情况，为不同的数值区间分配相应的分数。

## 打分方法详解 📝

接下来，我们通过一个具体的例子来演示两种常见的打分方法。

假设我们有300只股票（ID1 到 ID300），并关注四个因子：A、B、C、D。其中：
*   A因子：越大越好
*   B因子：越大越好
*   C因子：越小越好
*   D因子：越小越好

我们已获得每只股票各因子的归一化数值（范围在0到1之间）。以下是两只示例股票的数据：

![](img/2383f12323e5ca9629e1b204cef20ad4_5.png)

![](img/2383f12323e5ca9629e1b204cef20ad4_7.png)

| 股票ID | 因子A | 因子B | 因子C | 因子D |
| :----- | :---- | :---- | :---- | :---- |
| ID1    | 0.71  | 0.28  | 0.39  | 0.11  |
| ID2    | 0.45  | 0.17  | 0.81  | 0.02  |

![](img/2383f12323e5ca9629e1b204cef20ad4_9.png)

### 方法一：区间划分法

此方法将因子值划分为若干个区间，并为每个区间赋予固定分数。

![](img/2383f12323e5ca9629e1b204cef20ad4_11.png)

首先，我们需要定义评分区间和对应分值。对于“越大越好”和“越小越好”的因子，高分区间是相反的。

![](img/2383f12323e5ca9629e1b204cef20ad4_13.png)

以下是评分规则表示例：

![](img/2383f12323e5ca9629e1b204cef20ad4_15.png)

**对于“越大越好”的因子（如A、B）：**
*   数值在 [0.8, 1.0] 区间：得 **5分**
*   数值在 [0.6, 0.8) 区间：得 **4分**
*   数值在 [0.4, 0.6) 区间：得 **3分**
*   数值在 [0.2, 0.4) 区间：得 **2分**
*   数值在 [0.0, 0.2) 区间：得 **1分**

![](img/2383f12323e5ca9629e1b204cef20ad4_17.png)

**对于“越小越好”的因子（如C、D）：**
*   数值在 [0.0, 0.2) 区间：得 **5分**
*   数值在 [0.2, 0.4) 区间：得 **4分**
*   数值在 [0.4, 0.6) 区间：得 **3分**
*   数值在 [0.6, 0.8) 区间：得 **2分**
*   数值在 [0.8, 1.0] 区间：得 **1分**

现在，我们来为示例股票打分：

*   **ID1 打分**：
    *   因子A = 0.71 → 落在 [0.6, 0.8) → **4分**
    *   因子B = 0.28 → 落在 [0.2, 0.4) → **2分**
    *   因子C = 0.39 → 落在 [0.2, 0.4) → **4分** (越小越好，因此该区间得高分)
    *   因子D = 0.11 → 落在 [0.0, 0.2) → **5分** (越小越好，因此该区间得高分)
    *   **总分** = 4 + 2 + 4 + 5 = **15分**

*   **ID2 打分**：
    *   因子A = 0.45 → 落在 [0.4, 0.6) → **3分**
    *   因子B = 0.17 → 落在 [0.0, 0.2) → **1分**
    *   因子C = 0.81 → 落在 [0.8, 1.0] → **1分** (越小越好，因此该区间得低分)
    *   因子D = 0.02 → 落在 [0.0, 0.2) → **5分** (越小越好，因此该区间得高分)
    *   **总分** = 3 + 1 + 1 + 5 = **10分**

### 方法二：直接排序法

此方法更简单直接，无需划分区间。对于每只股票的某个因子，直接在全部股票中根据该因子值进行排序，然后按排名赋分。

![](img/2383f12323e5ca9629e1b204cef20ad4_19.png)

例如，对于300只股票的A因子（越大越好）：
*   排名第1（A值最大）的股票得 **300分**
*   排名第2的股票得 **299分**
*   ...
*   排名第300（A值最小）的股票得 **1分**

对于C因子（越小越好），则反过来：
*   排名第1（C值最小）的股票得 **300分**
*   排名第300（C值最大）的股票得 **1分**

![](img/2383f12323e5ca9629e1b204cef20ad4_21.png)

![](img/2383f12323e5ca9629e1b204cef20ad4_23.png)

最后，同样将各因子得分加总，得到每只股票的综合总分。

**核心公式**：
对于股票 *i*，其综合得分 *S_i* 可以表示为：
`S_i = Σ (w_j * score_{i,j})`
其中，*score_{i,j}* 是股票 *i* 在因子 *j* 上获得的分数，*w_j* 是因子 *j* 的权重（在简单模型中常设为1）。

![](img/2383f12323e5ca9629e1b204cef20ad4_25.png)

## 策略整体流程 🔄

![](img/2383f12323e5ca9629e1b204cef20ad4_29.png)

理解了打分方法后，我们来看一个完整的、基于月度调仓的量化策略流程。这个流程将指导我们后续的代码实现。

![](img/2383f12323e5ca9629e1b204cef20ad4_31.png)

![](img/2383f12323e5ca9629e1b204cef20ad4_33.png)

以下是策略执行的步骤：

1.  **确定股票池**：首先，我们需要确定选股的范围。在本例中，我们使用**沪深300指数**成分股作为初选股票池。这通过设置 `contest` 或初始配置来完成。
2.  **设置调仓周期**：策略需要定期运行以调整持仓。通常使用月度或季度调仓。我们需要设置一个**定时器**，例如每月第一个交易日触发调仓函数。
3.  **实现调仓函数**：核心是一个名为 `rebalance` 的函数。每次调仓时，系统会自动调用此函数。该函数内部包含以下子步骤：
    *   **数据获取与预处理**：获取当前时刻股票池中所有股票的因子数据（A, B, C, D）。确保数据是清洗和归一化后的。
    *   **因子打分**：应用上述打分法（区间划分法或排序法），为每只股票的每个因子计算得分。
    *   **计算综合得分**：将每只股票在所有因子上的得分相加，得到该股票的综合总分。
    *   **排序与选股**：对所有股票按综合总分进行**降序排序**。选择排名最高的前N只（例如前10只）股票作为本期要调入或持有的标的。
    *   **生成交易指令**：根据新的股票列表，计算需要买入、卖出或调整的仓位，并提交交易指令。

**流程总结图示**：
`设定股票池（沪深300）` → `定时触发（每月）` → `执行rebalance函数` → `获取因子数据` → `计算因子得分` → `汇总综合得分` → `排序并选取Top N` → `调仓交易`
