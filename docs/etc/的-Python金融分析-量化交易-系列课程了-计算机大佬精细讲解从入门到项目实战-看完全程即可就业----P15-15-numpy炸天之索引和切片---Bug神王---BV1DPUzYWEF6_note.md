在本节课中，我们将要学习NumPy库中索引与切片的核心操作。索引与切片是获取和操作数组数据的基础，理解它们对于后续的数据处理至关重要。我们将从基础概念开始，逐步深入到灵活的应用场景，包括数组的倒置和图片处理等实际案例。

## 索引操作

上一节我们介绍了NumPy数组的创建，本节中我们来看看如何获取数组中的数据。索引操作与Python列表的索引方式相同，用于获取数组中的特定元素或子集。

以下是索引操作的基本用法：

*   创建一个5行6列的随机整数数组：`arr = np.random.randint(1, 100, size=(5, 6))`
*   获取数组中的第一行数据：`arr[0]`
*   获取数组中下标为1、3、4的多行数据：`arr[[1, 3, 4]]`

## 切片操作

索引用于获取特定位置的数据，而切片则用于获取一个连续的数据范围。NumPy的切片操作比列表更为灵活，因为它可以同时指定行和列的切片范围。

以下是切片操作的核心机制：

*   切出数组的前两行：`arr[0:2, :]` 或简写为 `arr[:2]`
*   切出数组的前两列：`arr[:, 0:2]`
*   切出前两行的前两列：`arr[0:2, 0:2]`

在NumPy中，`arr[行切片, 列切片]` 的逗号分隔机制是关键。逗号左边控制行的选取，右边控制列的选取。

## 数组反转

基于切片操作，我们可以轻松实现数组的行、列乃至全部元素的反转（倒序排列）。这在数据重组和图像处理中非常有用。

以下是实现数组反转的几种方式：

*   将数组的行倒置：`arr[::-1, :]`
*   将数组的列倒置：`arr[:, ::-1]`
*   将数组所有元素倒置（行列均倒置）：`arr[::-1, ::-1]`

## 应用：图片处理

通过数组反转，我们可以对图片进行上下或左右翻转。一张图片在NumPy中本质上是一个三维数组，其形状通常表示为 `(高度, 宽度, 颜色通道)`。

以下是图片翻转的操作方法：

1.  读取图片：`img_arr = plt.imread(‘1.jpg’)`
2.  左右翻转图片（列倒置）：`plt.imshow(img_arr[:, ::-1, :])`
3.  上下翻转图片（行倒置）：`plt.imshow(img_arr[::-1, :, :])`
