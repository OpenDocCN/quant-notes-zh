# Python金融分析与量化交易实战课程：P28：4-3 Sigma方法实例 📊

在本节课中，我们将要学习一种名为“3-Sigma”的去极值方法。该方法基于统计学中的正态分布原理，通过计算数据的均值和标准差来识别并处理异常值。我们将理解其核心思想，并通过代码实现来掌握其应用。

## 原理介绍：正态分布与极值

上一节我们介绍了去极值的概念，本节中我们来看看基于统计学原理的3-Sigma方法。

首先，我们需要理解正态分布。正态分布，也称为高斯分布，是一种常见的概率分布。在正态分布中，大多数数据点会聚集在均值附近，而远离均值的数据点出现的概率则较低。

![](img/559ba4dd3558a7cee09a1d1160f445e0_1.png)

为了便于理解，可以想象一个去银行贷款的例子。银行多借或少借你少量金额（例如几块钱）是正常情况。但如果多借或少借你一大笔钱（例如十万八万），这就属于异常情况了。

在正态分布图中，均值（通常记为μ）是分布的中心点。数据偏离均值的程度用标准差（记为σ）来衡量。距离均值越远的数据点，其出现的可能性就越低，这些点就越有可能是我们需要处理的“极值”或“离群点”。

## 3-Sigma法则详解

那么，什么是3-Sigma法则呢？其核心思想是：在正态分布中，绝大部分数据（约99.7%）会落在均值正负三个标准差的范围内。

![](img/559ba4dd3558a7cee09a1d1160f445e0_3.png)

以下是不同倍数标准差所覆盖的数据比例：
*   **均值 ± 1σ**：覆盖约68.3%的数据。
*   **均值 ± 2σ**：覆盖约95.4%的数据。
*   **均值 ± 3σ**：覆盖约99.7%的数据。

![](img/559ba4dd3558a7cee09a1d1160f445e0_5.png)

3-Sigma方法就是利用这个特性。我们假设数据经过某种变换后近似服从正态分布。然后，我们计算出数据的均值（μ）和标准差（σ）。接着，我们定义：
*   **下限（Lower Bound）** = μ - 3σ
*   **上限（Upper Bound）** = μ + 3σ

![](img/559ba4dd3558a7cee09a1d1160f445e0_7.png)

我们认为，落在上下限之间的数据（占99.7%）是“正常”数据。而落在界限之外的数据（占0.3%）则被视为“极值”，需要进行规范化处理（例如，将其截断至界限处），而不是直接丢弃。

![](img/559ba4dd3558a7cee09a1d1160f445e0_9.png)

## 代码实现

![](img/559ba4dd3558a7cee09a1d1160f445e0_11.png)

理解了原理后，我们通过代码来实现3-Sigma方法。我们将编写一个函数，它接收一列数据和一个倍数`n`（例如n=3），并返回处理后的数据以及计算出的上下限。

![](img/559ba4dd3558a7cee09a1d1160f445e0_15.png)

以下是实现步骤的代码：

![](img/559ba4dd3558a7cee09a1d1160f445e0_17.png)

![](img/559ba4dd3558a7cee09a1d1160f445e0_19.png)

```python
def sigma_method(data, n=3):
    """
    使用3-Sigma方法处理极值。
    参数:
    data: 输入的数据序列（pandas Series）。
    n: 标准差的倍数，默认为3。
    返回:
    series: 处理后的数据序列。
    lower_limit: 计算出的下限。
    upper_limit: 计算出的上限。
    """
    # 1. 计算数据的均值(mean)和标准差(std)
    mean = data.mean()
    std = data.std()

    # 2. 根据公式计算上限和下限
    upper_limit = mean + n * std
    lower_limit = mean - n * std

    # 3. 对数据进行处理：将超出界限的值截断到界限处
    series = data.copy()
    series[series > upper_limit] = upper_limit
    series[series < lower_limit] = lower_limit

    return series, lower_limit, upper_limit
```

![](img/559ba4dd3558a7cee09a1d1160f445e0_21.png)

代码说明：
1.  首先计算输入数据的均值（`mean`）和标准差（`std`）。
2.  然后根据公式 **上限 = 均值 + n * 标准差** 和 **下限 = 均值 - n * 标准差** 计算界限。
3.  最后，创建数据的副本，将所有大于上限的值设置为上限值，将所有小于下限的值设置为下限值，从而完成极值的规范化处理。

在实际任务中，3-Sigma方法与前几节介绍的百分位法、中位数绝对偏差法得到的结果可能略有差异。例如，上下限的具体数值可能分别在60、50或50多一点。选择哪种方法取决于具体的任务需求和数据特性，它们都是数据预处理中有效的极值处理策略。

## 本节总结

![](img/559ba4dd3558a7cee09a1d1160f445e0_37.png)

![](img/559ba4dd3558a7cee09a1d1160f445e0_39.png)

本节课中我们一起学习了去极值的第三步——3-Sigma方法。
*   其核心是**基于正态分布假设**，利用**均值（μ）**和**标准差（σ）**来定义正常数据的范围。
*   关键公式为：**下限 = μ - nσ**， **上限 = μ + nσ**。通常`n=3`，可覆盖99.7%的数据。
*   该方法**不是丢弃极值**，而是通过**设定上下界**，将所有数据规范到界限之内，从而减少异常值对整体分析的影响。

![](img/559ba4dd3558a7cee09a1d1160f445e0_41.png)

至此，我们已经介绍了三种主要的去极值方法（百分位法、MAD法、3-Sigma法），它们都是数据预处理中用于规范异常值的有效工具。
