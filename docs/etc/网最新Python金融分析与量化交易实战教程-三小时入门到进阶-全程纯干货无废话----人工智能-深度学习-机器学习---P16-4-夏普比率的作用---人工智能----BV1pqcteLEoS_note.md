## 夏普比率的含义
上一节我们介绍了投资回报率，本节中我们来看看如何衡量“性价比”更高的投资。夏普比率描述的是：在承担单位风险的情况下，我们能获得多少超额收益。

为了便于理解，可以举一个生活中的例子。例如，一份在战乱地区日薪数万的工作，其高薪对应的是极高的生命风险。夏普比率就是用来量化这种“风险与收益是否匹配”的指标。**指标越高，意味着承担单位风险所获得的超额收益越高**，这笔投资（或工作）的“性价比”就越高。

在选股或构建投资组合时，如果单从夏普比率这一指标来看，我们应选择**夏普比率更高**的选项，因为它代表了更优的风险收益比。

## 夏普比率的计算逻辑
理解了夏普比率的作用后，我们来看看它的计算原理。计算夏普比率，核心是比较你的投资组合收益与一个“无风险”基准收益的差异。

![](img/7405a7887504106c051abc5bc0d161f6_3.png)

例如，银行存款或国债通常被视为几乎无风险的投资，其收益率（比如年化3%）可作为“无风险收益率”。而一个股票投资组合可能提供更高的预期收益（比如年化15%），但同时伴随着价格波动的风险。

![](img/7405a7887504106c051abc5bc0d161f6_5.png)

夏普比率就是计算这个**超额收益**（投资组合收益 - 无风险收益）与投资组合**风险**（通常用收益率的标准差衡量）的比值。我们希望这个比值越大越好。

以下是夏普比率的计算公式：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率
*   **Rf**：无风险收益率
*   **σp**：投资组合收益率的标准差（代表风险）

![](img/7405a7887504106c051abc5bc0d161f6_5.png)

## 使用Python计算夏普比率
接下来，我们通过Python代码来实际计算一组股票的夏普比率。计算前，我们需要准备好股票的历史收益率数据。

> 注意：在计算收益率时，我们通常需要对原始数据中的缺失值（如停牌日）进行处理，常见的做法是用前一天的收盘价进行填充，以保证数据连续性。

以下是计算夏普比率的关键步骤代码：

```python
# 假设 returns 是包含多只股票每日收益率的DataFrame
# 假设无风险年化收益率 annual_risk_free_rate = 0.05
# 假设一年有250个交易日

import numpy as np

![](img/7405a7887504106c051abc5bc0d161f6_7.png)

# 计算年化无风险日收益率
daily_risk_free_rate = annual_risk_free_rate / 250

# 计算超额收益率（每日收益率 - 无风险日收益率）
excess_returns = returns - daily_risk_free_rate

# 计算夏普比率
# 公式： (平均超额收益率 * 年化因子) / (收益率的标准差 * 年化因子开方)
# 简化后： 平均超额收益率的年化值 / 收益率标准差的年化值
sharpe_ratios = (excess_returns.mean() * 250) / (returns.std() * np.sqrt(250))

print(sharpe_ratios)
```

![](img/7405a7887504106c051abc5bc0d161f6_11.png)

执行以上代码后，我们会得到每只股票的夏普比率。结果分析如下：

![](img/7405a7887504106c051abc5bc0d161f6_9.png)

*   夏普比率为正且值越大，说明该股票在承担单位风险时获得的超额回报越高，投资价值可能更优。
*   夏普比率为负，则意味着投资该资产的收益甚至未能覆盖无风险收益，在考虑风险后是不划算的。

![](img/7405a7887504106c051abc5bc0d161f6_11.png)

在我们的示例结果中，“中兴”的夏普比率最高，表明在这组股票里，它的风险收益“性价比”最好。
