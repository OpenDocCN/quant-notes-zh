# Python机器学习与量化交易：P30：31.31.4-3Sigma方法实例 📊

在本节课中，我们将要学习一种名为“3Sigma”的去极值方法。它的原理基于统计学中的正态分布，通过计算数据的均值和标准差，来识别并处理那些偏离正常范围过远的异常值。

上一节我们介绍了其他去极值方法，本节中我们来看看基于正态分布原理的3Sigma方法。

## 3Sigma方法原理 📈

3Sigma方法的原理非常简单。大家先来看这张图，图中画了一个正态分布。

![](img/230052a633943df1225b7411850fff4b_1.png)

如果大家理解起来比较麻烦，可以把它当作一个正常的分布。什么叫正常分布？以前讲机器学习时经常举一个例子：现在去银行贷款，银行多借给你三块五块，是挺正常的事；但多借或少借你十万八万，就不正常了。

我们来看这样一个走势。这里先画一条线，就拿标准正态分布来说。标准正态分布中有一个均值，均值是零。

比如现在要干一件事，这是一个误差。就像去银行贷款，银行可能多借也可能少借钱。多借钱就往右边点，少借钱就往左边点。但是大家来看，我们现在这个区间来说，比如画的第一个红色区间，阴影画出的区间面积相对来说会比较大一些，代表这些事发生的可能性会比较高。

当把这个区间越往外放的时候，这个可能性发生的概率就越低了。来想一想，我们现在要干什么？是不是要做一个去极值的问题？那极值它可能落到哪？用红色来画，落到这可能吗？这块落下的能是极值吗？它在均值附近，是大概率发生的事，那就是正常的数据。极值一般落在比较偏远的地方，这一块、这一块、这一块，这些地方离均值越远，偏差越大，越有可能是一些离群点。

这样一件事，就叫做一个三倍的西格玛。

## 3Sigma区间定义 📏

你看这里，我们画的这个东西当中有什么？有一个X的均值，加上一倍的、减去一倍的、加上两倍的、减去两倍的、加上三倍的、减去三倍的。

![](img/230052a633943df1225b7411850fff4b_3.png)

![](img/230052a633943df1225b7411850fff4b_1.png)

从这个区间当中能发现一件事：一旦区间选得越大，比如加减三倍，得到的数据越往左右偏的可能性就越小。

这个三西格玛意思是：现在有这样一份数据，原数据来了。然后假设可以对数据做一些变换，使得它理论上服从高斯分布或正态分布。先不知道怎么变的，就假设它就是正态分布的一个规则。

当指定一个正负的三倍西格玛区间时，这个当中包含了百分之多少的数据？这个值已经给大家列出来了，这是统计当中经常会定的指标：**99.7%**。

现在发现一件事：如果认为数据当中，想取最正常的99.7%当作数据，剩下0.3%当作一个极值给它规范一下，是不是可以？这个就叫做一个三西格玛。

相当于你的一个下限和上限谁来定的？在高斯分布当中，要算两个值：**均值**和**标准差**。就能把这个上限和下限全部计算出来了。

## 3Sigma方法实现 💻

这个叫做一个3Sigma方法。写一下，其实也很简单。在这里名字跟之前一样，还是做一个复制。在这里做个复制，然后把最后面那个值改成一个三西格玛就行了。

![](img/230052a633943df1225b7411850fff4b_5.png)

然后这一块输入的值，`n_std`等于三。既然你说三，就写个三。

![](img/230052a633943df1225b7411850fff4b_7.png)

![](img/230052a633943df1225b7411850fff4b_3.png)

然后这块来写。需要算什么？在整个数据当中，这一列数据，得知道当前它的均值是多少，当前这个标准差是多少。

所以对数据可以算两个值：一个是`.mean()`，第二个就是`.std()`。

以下是计算均值和标准差的代码：

![](img/230052a633943df1225b7411850fff4b_5.png)

```python
# 计算均值和标准差
mean_value = data.mean()
std_value = data.std()
```

![](img/230052a633943df1225b7411850fff4b_7.png)

前两个指标算出来了。有两个指标之后，接下来都复制就行了。

![](img/230052a633943df1225b7411850fff4b_13.png)

这也不用自己去写了，把这个东西复制过来。复制过来了，那你看中间等于什么？一个均值，加上`n`倍的一个`std`。那下面呢？就是均值减去`n`倍的一个`std`。

![](img/230052a633943df1225b7411850fff4b_9.png)

返回值是一个`Series`，然后指定下限和上限就完事了。

以下是计算上下限的代码：

![](img/230052a633943df1225b7411850fff4b_11.png)

```python
# 计算上下限
upper_limit = mean_value + n_std * std_value
lower_limit = mean_value - n_std * std_value
```

![](img/230052a633943df1225b7411850fff4b_15.png)

![](img/230052a633943df1225b7411850fff4b_13.png)

非常简单，又完成了这样一个3Sigma方法。

## 总结 📝

本节课中我们一起学习了3Sigma去极值方法。该方法基于数据服从正态分布的假设，通过计算数据的均值（`mean`）和标准差（`std`），并设定`n`倍标准差作为边界（例如`n=3`对应99.7%的数据），来识别和处理异常值。其核心是将超出上下限的极值规范到边界内，而不是直接删除，从而完成数据预处理中的去极值步骤。
