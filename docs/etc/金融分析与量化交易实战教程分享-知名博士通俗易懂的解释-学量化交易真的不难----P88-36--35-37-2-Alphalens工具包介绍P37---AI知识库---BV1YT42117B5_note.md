# 量化交易完全可自学教程：P88：36. 35.37.2-Alphalens工具包介绍

在本节课中，我们将要学习一个在量化交易中非常实用的工具包——Alphalens。这个工具包能极大地简化因子分析和绩效评估的流程，帮助我们高效地计算关键指标并生成可视化图表。

## 工具包简介与获取

上一节我们介绍了因子分析的基本概念，本节中我们来看看如何利用现成的工具来执行这些分析。Alphalens是一个专门用于因子分析的Python工具包，它封装了复杂的计算和绘图逻辑。

*   **核心功能**：Alphalens可以帮助我们自动完成因子收益、信息系数（IC值）等指标的计算，并生成一系列标准化的分析图表。
*   **获取方式**：安装Alphalens非常简单，只需使用pip命令即可。
    ```bash
    pip install alphalens
    ```
    不过，在本课程后续的实战环节中，我们将在提供的量化交易平台中直接使用，该平台已预装好此工具包。如果你想在本地环境进行实验，则可以自行安装。

## 研究环境准备

由于因子分析需要获取大量的金融市场数据，在个人电脑上操作会非常麻烦。因此，我们将在一个在线的量化研究平台中编写和运行代码。

接下来，我们需要进入该平台的研究环境：

![](img/8939cded88383b091cc8178aca04e005_11.png)

1.  在平台界面左侧，找到并点击 **“投资研究”** 功能模块。
2.  点击后，系统会启动一个类似于Jupyter Notebook的交互式编程环境。
3.  在这个环境中，新建一个Python 3笔记文件，我们可以将其命名为“因子分析”。

![](img/8939cded88383b091cc8178aca04e005_13.png)

之后，我们所有的示例代码都将在该研究环境中执行。课程会提供完整的代码，你可以直接复制到平台中运行，或者跟随视频讲解一步步操作。

![](img/8939cded88383b091cc8178aca04e005_15.png)
