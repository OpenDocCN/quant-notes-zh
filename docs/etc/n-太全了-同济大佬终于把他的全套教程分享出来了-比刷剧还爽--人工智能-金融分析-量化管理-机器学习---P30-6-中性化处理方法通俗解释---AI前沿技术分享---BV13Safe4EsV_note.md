# 量化投资基础：P30：6：中性化处理方法通俗解释 🧪

在本节课中，我们将要学习量化投资中的一个重要概念——**因子中性化**。我们将通过一个简单的例子理解其核心目的，并介绍其基本计算方法。

## 概述：什么是中性化？

中性化的目的是进行**提纯**。

为了理解“提纯”，我们先看一个例子。假设我们设计一个选股策略，使用了四个因子：A、B、C、D。这四个因子都会对最终的选股结果产生影响。

但无论我们如何调整这四个因子的组合，每次选出的股票池总是高度相似。为什么会这样？

原因可能在于，这四个因子虽然表面不同，但其内部绝大部分成分是相同的。例如：
*   因子A（如市净率）绝大部分与**市值**相关。
*   因子B、C、D也绝大部分与**市值**相关。

这样一来，无论使用哪个因子，最终起决定性作用的似乎都是“市值”。我们无法体现出每个因子自身独特的信息和价值。所有因子都“说”着同一件事，策略就缺乏多样性。

因此，**中性化**要做的事，就是从每个因子中，将其独特、有价值的部分提取出来，剥离掉与其他因子（如市值）共有的、普遍的部分。这个过程就像“提纯”，目的是获得因子的“个性”。

## 量化中的中性化应用

在量化交易中，我们常使用多个指标（因子）对股票池进行筛选和选股。

在使用这些因子选股的过程中，有时会因为某些共同因素的影响（例如，多个因子都与市值高度相关），导致选出的股票具有我们不希望看到的倾向性，例如股票过于集中在某个风格（如大盘股）上，无法体现策略的差异性。

如果不进行中性化处理，选股结果可能会比较集中，因为因子中起主导作用的可能是共有的影响因素（如市值），而非因子本身的特异信号。

## 核心概念：市净率

在本例中，我们将使用**市净率**作为演示因子。市净率的计算公式为：
`市净率(PB) = 每股股价 / 每股净资产`
其中，**每股净资产** = （公司总资产 - 公司总负债）/ 总股数，它代表了每股股票所代表的公司净价值。

![](img/7974d631ada255ca5a8be8ac0d77aa09_1.png)

在投资指导中，通常认为较低的市净率可能意味着股票被低估，其未来上涨的潜力空间相对更大，投资风险可能更低。

## 中性化的计算方法

上一节我们介绍了中性化的目的和应用场景，本节中我们来看看如何进行计算。

中性化处理通常通过回归分析来实现。其核心思想是：将一个因子对另一个或多个因子（通常是需要剥离的共性因素，如市值、行业）进行回归，然后取回归后的残差作为中性化后的新因子。这个残差就代表了原因子中无法被共性因素解释的、独特的部分。

以下是基本的计算步骤：

1.  **建立回归模型**：例如，将**市净率因子**对**市值因子**进行线性回归。
    `市净率 = α + β * 市值 + ε`
    其中，`ε` 即为残差。

2.  **提取残差**：上述回归模型中的残差 `ε`，就是中性化后的市净率因子。它已经剥离了市值的影响。

在接下来的实践中，我们需要从金融数据平台获取原始数据（如市净率、市值），然后在Python中利用统计库（如`statsmodels`）执行上述回归步骤，得到中性化后的因子值。
