在本节课中，我们将要学习如何分析时间序列数据的变化情况。具体来说，我们将计算两种核心指标：**每日差异值**和**每日变化率**，并通过统计和可视化来理解数据的整体趋势。

上一节我们介绍了时间序列数据的基本结构，本节中我们来看看如何量化其变化。

## 计算每日差异值

![](img/948df752c4920a7098ddeff93bbc2576_6.png)

首先，我们需要计算每一天与前一天的指标差异值。这个指标反映了数据在相邻两天之间的绝对变化量。

![](img/948df752c4920a7098ddeff93bbc2576_8.png)

在Pandas中，我们可以使用 `diff()` 函数轻松实现这个计算。该函数会计算当前行与前一行的差值。

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

![](img/948df752c4920a7098ddeff93bbc2576_10.png)

![](img/948df752c4920a7098ddeff93bbc2576_12.png)

执行上述代码后，你会看到一个新的DataFrame。例如，1月5日的数据等于1月5日的原始值减去1月4日的原始值。由于1月4日没有前一天的数据，因此其差异值会显示为 `NaN`（非数字）。

`diff()` 函数默认计算与前一天的差异。我们还可以通过指定 `periods` 参数来计算间隔多天的差异，例如计算与两天前或一周前的差异。

以下是使用 `diff()` 函数时需要注意的几点：
*   第一行数据由于没有前一行，其差异值始终为 `NaN`。
*   该函数计算的是绝对差值。
*   可以通过 `periods` 参数调整计算间隔。

![](img/948df752c4920a7098ddeff93bbc2576_14.png)

![](img/948df752c4920a7098ddeff93bbc2576_16.png)

## 计算每日变化率

![](img/948df752c4920a7098ddeff93bbc2576_18.png)

差异值是一个绝对数值，有时我们更关心相对变化，即增长率或变化率。例如，股价从1元涨到2元是100%的增长，而从10000元涨到10001元则几乎可以忽略不计。

变化率的计算公式为：**（后一天的值 - 前一天的值）/ 前一天的值**。

![](img/948df752c4920a7098ddeff93bbc2576_20.png)

Pandas提供了 `pct_change()` 函数来直接计算这个百分比变化。

```python
# 计算每日变化率（增长率）
pct_data = data.pct_change()
print(pct_data.head())
```

![](img/948df752c4920a7098ddeff93bbc2576_22.png)

计算完成后，数据将显示为小数形式（例如0.05代表5%的增长）。我们可以使用 `round()` 函数来控制显示的小数位数，使其更易读。

![](img/948df752c4920a7098ddeff93bbc2576_24.png)

![](img/948df752c4920a7098ddeff93bbc2576_26.png)

```python
# 保留三位小数
pct_data_rounded = pct_data.round(3)
print(pct_data_rounded.head())
```

![](img/948df752c4920a7098ddeff93bbc2576_28.png)

![](img/948df752c4920a7098ddeff93bbc2576_30.png)

## 分析整体变化趋势

![](img/948df752c4920a7098ddeff93bbc2576_32.png)

计算出每日的变化率后，我们可以进一步分析其整体统计特征，例如平均增长率。

![](img/948df752c4920a7098ddeff93bbc2576_34.png)

```python
# 计算各列的平均增长率
mean_pct = pct_data.mean()
print(mean_pct)
```

为了更直观地展示各个指标的平均增长情况，我们可以将其可视化。由于平均增长率是单个数值，适合使用条形图。

![](img/948df752c4920a7098ddeff93bbc2576_36.png)

```python
# 绘制平均增长率的条形图
mean_pct.plot(kind='bar', figsize=(16, 8))
```

![](img/948df752c4920a7098ddeff93bbc2576_38.png)

通过图表，我们可以一目了然地看出哪些指标在观察期内的平均增长较高，哪些较低。

![](img/948df752c4920a7098ddeff93bbc2576_40.png)

![](img/948df752c4920a7098ddeff93bbc2576_42.png)

本节课中我们一起学习了时间序列变化分析的两个核心操作。我们首先使用 `diff()` 函数计算了每日的绝对差异值，然后使用 `pct_change()` 函数计算了更能反映相对变化的每日增长率。最后，我们通过对增长率求平均值并进行可视化，掌握了分析数据整体变化趋势的方法。这些是金融时间序列分析中非常基础和重要的技能。
