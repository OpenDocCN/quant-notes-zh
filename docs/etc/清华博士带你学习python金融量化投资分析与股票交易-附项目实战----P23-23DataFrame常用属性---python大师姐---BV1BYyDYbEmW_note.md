在本节课中，我们将学习Pandas中`DataFrame`对象的几个常用属性。这些属性可以帮助我们快速查看和理解数据的基本结构、内容以及统计信息。

上一节我们介绍了`DataFrame`对象的创建方式，本节中我们来看看它有哪些实用的属性。

## 常用属性详解

### 1. 行索引与列索引

和`Series`对象类似，`DataFrame`也有`index`属性，用于获取其行索引。行索引指的是数据最左侧竖排的标签。

`DataFrame`是一个二维结构，因此它还有一个`columns`属性，用于获取其列索引。列索引指的是数据最上方横排的标签。

以下是获取索引的示例：
```python
# 假设 df 是一个已创建的 DataFrame
行索引 = df.index
列索引 = df.columns
```

### 2. 数据值

`values`属性用于获取`DataFrame`中的数据值。与`Series`返回一维数组不同，`DataFrame`的`values`属性返回的是一个二维数组，其中每一行是一个子数组。

以下是获取数据值的示例：
```python
数据数组 = df.values
```

### 3. 转置

`T`属性用于获取`DataFrame`的转置。转置是一个数学概念，意味着将数据的行和列进行交换，即原来的行变成列，原来的列变成行。

以下是获取转置的示例：
```python
转置后的数据 = df.T
```
**注意**：在转置过程中，如果某一列包含浮点数（例如`NaN`），Pandas可能会将整列的数据类型统一转换为浮点数，以确保数据一致性。

### 4. 数据描述

`describe()`是一个方法，而非属性。它可以快速生成关于`DataFrame`中每一列数值型数据的描述性统计信息。

![](img/16195158afa42f27d7560b1c52b59f62_1.png)

以下是使用`describe()`方法的示例：
```python
统计描述 = df.describe()
```
该方法返回的统计信息通常包括：
*   **count**：非缺失值的数量。
*   **mean**：平均值。
*   **std**：标准差。
*   **min**：最小值。
*   **25%**：第一四分位数。
*   **50%**：中位数。
*   **75%**：第三四分位数。
*   **max**：最大值。

## 总结

![](img/16195158afa42f27d7560b1c52b59f62_3.png)

本节课中我们一起学习了`DataFrame`的几个核心属性：
1.  **`index`**：获取行索引。
2.  **`columns`**：获取列索引。
3.  **`values`**：以二维数组形式获取数据值。
4.  **`T`**：获取数据的转置。
5.  **`describe()`**：获取数据各列的描述性统计信息。
