以下是运行策略时可能遇到的几个典型错误及解决方法：

![](img/a16e1312238b1c89c39ab8b17e15327a_25.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_27.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_29.png)

1.  **中性化函数参数缺失**：在调用中性化函数时，需要传入正确的参数，包括数据框和指标列名。
    ```python
    # 错误示例：缺少指标列名参数
    result = neutralize(df, factor)
    # 正确示例
    result = neutralize(df, factor, ‘target_column_name’)
    ```

![](img/a16e1312238b1c89c39ab8b17e15327a_33.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_35.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_37.png)

2.  **模型未训练**：在使用线性回归模型进行中性化处理前，必须先调用 `fit` 方法训练模型。
    ```python
    model = sm.OLS(y, X)
    result = model.fit()  # 必须先进行拟合
    residuals = result.resid
    ```

![](img/a16e1312238b1c89c39ab8b17e15327a_39.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_41.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_43.png)

3.  **函数名拼写错误**：例如，分位数函数 `quantile` 被错误拼写。
    ```python
    # 错误拼写
    df[‘factor_rank’] = df[‘factor’].qant()
    # 正确拼写
    df[‘factor_rank’] = df[‘factor’].quantile()
    ```

![](img/a16e1312238b1c89c39ab8b17e15327a_45.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_47.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_49.png)

4.  **变量名引用错误**：在循环或函数调用中，使用了未定义或拼写错误的变量名。
    ```python
    # 错误示例：变量名不一致
    for stock in stock_list:
        order_target_percent(stok, 0)  # ‘stok’ 未定义
    # 正确示例
    for stock in stock_list:
        order_target_percent(stock, 0)  # 使用正确的变量名 ‘stock’
    ```

以下是评估策略效果时需要关注的核心指标：

![](img/a16e1312238b1c89c39ab8b17e15327a_111.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_113.png)

1.  **累计收益率**：策略在整个回测期间的总收益。对比策略收益率与基准收益率，判断策略是否创造了超额收益。
2.  **年化收益率**：将累计收益率折算为年化标准，便于比较不同时间长度策略的收益能力。
3.  **夏普比率**：衡量每承受一单位总风险，所产生的超额回报。公式为 `(策略年化收益率 - 无风险利率) / 策略收益波动率`。该值越高，说明风险调整后收益越好。
4.  **最大回撤**：在选定周期内，资产净值从最高点到最低点的下跌幅度。它描述了投资者可能面临的最大亏损。回撤越小，策略控制下行风险的能力越强。
5.  **收益波动率**：策略收益率的波动程度，即标准差。波动率越低，代表收益越稳定。

## 总结策略流程

本节课中我们一起学习了量化策略的回测评估全流程。一个完整的因子选股策略通常包含以下步骤：

![](img/a16e1312238b1c89c39ab8b17e15327a_135.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_137.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_139.png)

![](img/a16e1312238b1c89c39ab8b17e15327a_141.png)

1.  **初始化与定时**：在 `initialize` 函数中设置策略参数和定时器，决定调仓频率（如每月）。
2.  **股票池过滤**：在定时触发的函数中，首先过滤掉不符合基本条件的股票（如ST股、停牌股）。
3.  **因子计算与处理**：对剩余股票计算选股因子（如市值、市净率），并进行必要的预处理（如中性化处理）。
4.  **因子筛选**：根据因子逻辑（如认为市净率越低越好）对股票进行排序和筛选，得到当期欲持有的股票池。
5.  **调仓执行**：
    *   **卖出**：遍历当前持仓，如果某只股票不在新的股票池中，则通过 `order_target_percent(stock, 0)` 将其清仓。
    *   **买入**：遍历新的股票池，通过 `order_target_percent(stock, 1/len(pool))` 等比例买入所有股票，完成仓位再平衡。
