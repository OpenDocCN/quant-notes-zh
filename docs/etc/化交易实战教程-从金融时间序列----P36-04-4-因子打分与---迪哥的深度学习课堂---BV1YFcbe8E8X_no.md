## 遍历因子并排序

上一节我们介绍了如何筛选出“越高越好”和“越低越好”的因子数据框。本节中，我们来看看如何遍历这些数据框，并对每个因子下的股票进行排序。

以下是遍历“越高越好”因子数据框并进行排序的步骤：

1.  首先，我们需要导入必要的库。
    ```python
    import numpy as np
    import pandas as pd
    ```

2.  获取数据框的列名（即因子名称），以便遍历每一个因子。
    ```python
    for factor in up.columns:
    ```

![](img/e81d2a06b76d73b598969802b8ce31cd_6.png)

3.  对当前因子下的股票数据进行排序。我们使用 `sort_values` 方法，并指定 `by` 参数为当前因子名。`inplace=True` 参数确保排序结果直接更新到原数据框中，无需重新赋值。
    ```python
    up.sort_values(by=factor, inplace=True)
    ```

## 为排序结果分配分数

![](img/e81d2a06b76d73b598969802b8ce31cd_8.png)

![](img/e81d2a06b76d73b598969802b8ce31cd_10.png)

排序完成后，我们需要为每个股票分配一个分数。为了简单起见，我们采用线性打分法：排名第一的股票获得最高分，排名最后的股票获得最低分。

![](img/e81d2a06b76d73b598969802b8ce31cd_12.png)

![](img/e81d2a06b76d73b598969802b8ce31cd_14.png)

我们将使用NumPy的 `linspace` 函数来生成一个等差数列作为分数。`linspace` 函数的公式如下：
`np.linspace(start, stop, num)`
它会生成从 `start` 到 `stop`（包含两端）的等间隔的 `num` 个数字。

![](img/e81d2a06b76d73b598969802b8ce31cd_16.png)

![](img/e81d2a06b76d73b598969802b8ce31cd_18.png)

以下是具体的打分逻辑：

![](img/e81d2a06b76d73b598969802b8ce31cd_20.png)

*   **对于“越高越好”的因子**：因子值越大，我们希望其得分越高。因此，排序后（从小到大）排名最后的股票（因子值最大）应得最高分。我们可以使用 `np.linspace(1, 300, 300)` 生成1到300的分数，然后按排序顺序分配。
*   **对于“越低越好”的因子**：因子值越小，我们希望其得分越高。因此，排序后（从小到大）排名第一的股票（因子值最小）应得最高分。我们可以使用 `np.linspace(300, 1, 300)` 生成300到1的分数，然后按排序顺序分配。

![](img/e81d2a06b76d73b598969802b8ce31cd_22.png)

![](img/e81d2a06b76d73b598969802b8ce31cd_24.png)

以下是实现代码：

![](img/e81d2a06b76d73b598969802b8ce31cd_26.png)

![](img/e81d2a06b76d73b598969802b8ce31cd_28.png)

```python
# 对“越高越好”的因子进行打分
for factor in up.columns:
    up.sort_values(by=factor, inplace=True)
    # 生成从1到300的分数，分配给排序后的股票
    up[factor] = np.linspace(1, 300, 300)

![](img/e81d2a06b76d73b598969802b8ce31cd_30.png)

![](img/e81d2a06b76d73b598969802b8ce31cd_32.png)

# 对“越低越好”的因子进行打分
for factor in down.columns:
    down.sort_values(by=factor, inplace=True)
    # 生成从300到1的分数，分配给排序后的股票
    down[factor] = np.linspace(300, 1, 300)
```

![](img/e81d2a06b76d73b598969802b8ce31cd_34.png)

## 合并数据框并计算总分

现在，我们得到了两个包含因子得分的数据框。为了计算每个股票的综合得分，我们需要将这两个数据框合并。

一个简单的方法是使用Pandas的加法操作。但在此之前，需要确保两个数据框的索引（股票代码）是对齐的。通常，在排序和打分后，索引顺序会发生变化，因此直接相加可能导致错误。

![](img/e81d2a06b76d73b598969802b8ce31cd_36.png)

更稳妥的做法是，在打分前或合并前，确保我们操作的是数据的副本，或者使用更严谨的按索引对齐的合并方法（如 `pd.concat` 或 `.add` 方法并指定轴）。为了教程清晰，我们假设数据框的索引就是股票代码，并且顺序一致，可以进行直接相加：

![](img/e81d2a06b76d73b598969802b8ce31cd_38.png)

```python
# 将两个得分数据框相加，得到每个股票的总分
total_score = up + down
```

在实际应用中，你可能需要先重置索引或确保索引一致性后再进行合并操作。

![](img/e81d2a06b76d73b598969802b8ce31cd_40.png)

本节课中我们一起学习了因子打分与排序的完整流程。我们首先遍历了“越高越好”和“越低越好”两类因子，对股票进行排序；然后利用 `np.linspace` 函数根据排序名次为股票分配分数；最后，通过合并两个得分数据框来计算每个股票的最终综合得分。这是构建多因子选股模型的核心步骤之一。
