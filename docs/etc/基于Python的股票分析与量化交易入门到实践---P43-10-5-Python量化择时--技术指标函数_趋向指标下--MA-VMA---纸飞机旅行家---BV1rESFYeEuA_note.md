## MA：移动平均线 📊

移动平均线（MA）是我们多次提及的老朋友。它是一种趋势追踪工具，核心特性是**趋势性**。越长期的移动平均线，其走势越**稳定**。它便于识别趋势的终结、反转或新趋势的形成。

国内常用的移动平均线周期包括：
*   **短期**：5日、10日
*   **中期**：20日、30日
*   **长期**：60日、120日、250日

其中，250日均线在国内A股市场有特殊意义，常被用来判断市场整体处于**牛市**还是**熊市**。

下图展示了几条不同周期的移动平均线。可以看到，蓝色的5日均线波动最大，而棕色的30日均线最为平缓，这印证了“周期越长，均线越稳定”的特点。

### 格兰比买卖法则

以下是基于MA判断买卖时机的经典经验法则——格兰比法则。它包含四个买入信号和四个卖出信号。

**买入信号：**
1.  移动平均线从下降逐渐走平或略向上，同时股价自下而上突破平均线。
2.  股价跌破移动平均线，但很快（例如3个交易日内）又回升至平均线上方，且平均线仍保持上升趋势。
3.  股价位于移动平均线上方，回调时未跌破平均线便再度上升。
4.  股价突然暴跌，跌破并远离移动平均线，则有可能产生反弹。

**卖出信号：**
1.  移动平均线从上升逐渐走平或转跌，同时股价向下跌破平均线。
2.  股价向上突破移动平均线，但很快又跌回其下方，且平均线仍保持下降趋势。
3.  股价位于移动平均线下方，反弹时未能突破平均线便再度下跌。
4.  股价突然暴涨，突破并远离移动平均线，则有可能产生回调。

> **注意**：前两条信号相对明确，易于转化为量化策略。后两条信号（涉及“抄底”和“逃顶”）更依赖经验判断，若想量化，需结合**阈值**（如涨跌幅、最大回撤）等参数进行约束。

### MA的Python实现

MA的实现同样依赖于`technical analysis`库。其核心函数是`MA`，一个关键参数是`timeperiod`，用于指定计算均线的周期（如5、10、20）。

函数调用示例如下：
```python
# 计算平安银行（000001.SZ）在指定日期的5日移动平均线
ma_value = MA('000001.SZ', check_date, timeperiod=5)
```
该函数返回一个字典（`dict`），包含计算出的MA值。

介绍完MA，我们再来看看它的一个变体——VMA。

## VMA：变异平均线 🔄

VMA（变异平均线）与我们之前提到的成交量均线不同。之所以称为“变异”，是因为它改进了MA的计算基准。

*   **MA**：仅使用**收盘价**计算。
*   **VMA**：使用每个交易日的**开盘价、收盘价、最高价、最低价之和除以4**得到的均价作为计算基准。

在股市中，股价高于VMA通常意味着处于**强势**，反之则为**弱势**。与MA相比，VMA**敏感度更高**，因为它综合了每日的价格波动范围（最高/最低价），部分消除了MA的滞后性，更能反映实际市场情况。

下图展示了VMA的常见形态。VMA的周期通常是6的倍数，如6、12、24、72、144等。



![](img/af2a4e6ba4f07f970757e7c4b6236819_1.png)

![](img/af2a4e6ba4f07f970757e7c4b6236819_3.png)

VMA的买卖信号判断相对直观：
*   **买入信号**：两条及以上VMA均线**向上交叉**（金叉）。
*   **卖出信号**：两条及以上VMA均线**向下交叉**（死叉）。

其算法是：先计算每日的`(开盘价 + 收盘价 + 最高价 + 最低价) / 4`，再对结果进行移动平均计算。

### VMA的Python实现

![](img/af2a4e6ba4f07f970757e7c4b6236819_3.png)

VMA的实现方式与MA高度相似，主要区别在于函数名。

函数调用示例如下：
```python
# 计算平安银行（000001.SZ）在指定日期的6日变异平均线
vma_value = VMA('000001.SZ', check_date, timeperiod=6)
```
> **注意**：VMA策略通常需要计算**多条**不同周期的均线（如6、12、24、72），通过观察它们之间的交叉情况来产生交易信号。

## 代码实战 💻

### MA实战示例

我们将计算平安银行股票在连续三个交易日内的5日、10日和20日移动平均线。



![](img/af2a4e6ba4f07f970757e7c4b6236819_5.png)

```python
# 导入库
import technical_analysis as ta

# 选定股票和时间段
stock_code = '000001.SZ'
dates = ['2023-10-25', '2023-10-26', '2023-10-27']

![](img/af2a4e6ba4f07f970757e7c4b6236819_7.png)

# 计算并打印不同周期的MA
for date in dates:
    ma5 = ta.MA(stock_code, date, timeperiod=5)
    ma10 = ta.MA(stock_code, date, timeperiod=10)
    ma20 = ta.MA(stock_code, date, timeperiod=20)
    
    print(f"日期 {date}: MA5={ma5:.4f}, MA10={ma10:.4f}, MA20={ma20:.4f}")
```
运行结果将显示，短期均线（如MA5）数值变动更频繁，而长期均线（如MA20）变动更平缓，体现了长期均线的稳定性。

### VMA实战示例

我们基于类似的框架，计算平安银行股票的6日、12日、24日和72日变异平均线。



![](img/af2a4e6ba4f07f970757e7c4b6236819_7.png)

```python
# 计算并打印不同周期的VMA
for date in dates:
    vma6 = ta.VMA(stock_code, date, timeperiod=6)
    vma12 = ta.VMA(stock_code, date, timeperiod=12)
    vma24 = ta.VMA(stock_code, date, timeperiod=24)
    vma72 = ta.VMA(stock_code, date, timeperiod=72)
    
    print(f"日期 {date}: VMA6={vma6:.4f}, VMA12={vma12:.4f}, VMA24={vma24:.4f}, VMA72={vma72:.4f}")
```
运行代码后，我们可以得到多条VMA均线的数值。在实际策略中，需要进一步编写逻辑来判断这些均线之间是否发生交叉，从而生成买卖信号。

## 本章小结 📝

本节课中我们一起学习了趋向指标的两种重要均线：MA和VMA。

*   **MA（移动平均线）**：核心是**趋势追踪**。周期越长越**稳定**。常用周期有5、10、20、30、60、120、250日。**格兰比法则**提供了基于MA的八个买卖经验点，其中前两条较易量化。在Python中，使用`MA(security, check_date, timeperiod=n)`函数计算。
*   **VMA（变异平均线）**：对MA的改进，使用`(开盘+收盘+最高+最低)/4`作为计算基准，**敏感度更高**。常用周期为6的倍数，如6、12、24、72日。其买卖信号依赖于**多条均线的交叉**判断。在Python中，使用`VMA(security, check_date, timeperiod=n)`函数计算。

两者的主要区别在于计算基准和常用周期。MA通常观察单条或少数几条均线即可，而VMA策略则需要同时计算并观察多条均线的互动关系。
