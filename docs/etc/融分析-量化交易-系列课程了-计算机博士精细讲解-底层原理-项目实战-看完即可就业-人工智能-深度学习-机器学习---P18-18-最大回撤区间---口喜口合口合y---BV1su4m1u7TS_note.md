## 概述

最大回撤是衡量投资策略风险的关键指标。它描述的是在特定时间段内，资产净值从最高点到后续最低点的最大跌幅。这个指标不衡量策略的最终盈利能力，而是评估投资者在过程中可能承受的最大心理压力和资金损失风险。

## 什么是最大回撤？

上一节我们介绍了策略回测的基本框架，本节中我们来看看如何评估策略的风险。最大回撤描述的是投资过程中“最惨”的阶段。

具体来说，它衡量的是资产价格在下跌过程中，从某个高点开始到后续某个低点结束，所经历的最大跌幅区间。这个区间可能持续数天甚至数月，期间资产价值持续缩水。

**公式定义**：
假设 `P_i` 代表第 `i` 天的资产价格，`P_j` 代表第 `i` 天之后某一天 `j` 的资产价格（`j > i`）。最大回撤 `MDD` 的计算公式为：

```
MDD = max_{i, j (j>i)} [ (P_i - P_j) / P_i ]
```

这个公式的含义是：遍历所有可能的时间点 `i` 和其后的时间点 `j`，计算从 `i` 到 `j` 的跌幅百分比，然后取其中的最大值。

## 为什么最大回撤很重要？

最大回撤是一个风险指标。一个策略可能最终盈利，但如果其最大回撤过大，意味着投资者在过程中可能经历难以承受的资金缩水和心理压力。例如，2008年股市从高点大幅下跌，期间的最大回撤就导致了巨大的市场恐慌和损失。因此，评估策略时，不仅要看收益，也必须关注其最大回撤所揭示的风险。

## 如何计算最大回撤？

![](img/88ee8346477ff6e392e1905dcd48aec3_11.png)

![](img/88ee8346477ff6e392e1905dcd48aec3_13.png)

理解了最大回撤的概念后，我们来看看如何用Python进行计算。核心思路是：我们需要找到历史上资产净值曲线的“高水位线”（即到当前日期为止的最高点），然后计算当前净值相对于这个高点的回撤，最后找出所有回撤中的最大值。

![](img/88ee8346477ff6e392e1905dcd48aec3_15.png)

以下是计算最大回撤的关键步骤：

![](img/88ee8346477ff6e392e1905dcd48aec3_17.png)

1.  **计算累积最大值**：对于时间序列中的每一个点，计算从起始点到该点的最大值。这代表了到该日期为止的“历史最高点”。
2.  **计算每日回撤**：用当日的“历史最高点”减去当日的实际净值，得到当日的“下跌深度”。
3.  **计算回撤比率**：将“下跌深度”除以对应的“历史最高点”，得到当日的回撤百分比。
4.  **找出最大值**：在所有日期的回撤百分比中，找出最大值，即为最大回撤。

![](img/88ee8346477ff6e392e1905dcd48aec3_19.png)

![](img/88ee8346477ff6e392e1905dcd48aec3_21.png)

让我们通过代码示例来具体实现：

![](img/88ee8346477ff6e392e1905dcd48aec3_23.png)

![](img/88ee8346477ff6e392e1905dcd48aec3_25.png)

![](img/88ee8346477ff6e392e1905dcd48aec3_27.png)

```python
import pandas as pd

![](img/88ee8346477ff6e392e1905dcd48aec3_29.png)

# 假设 df[‘net_value’] 是资产净值的时间序列
# 1. 计算累积最大值（高水位线）
running_max = df[‘net_value’].cummax()

![](img/88ee8346477ff6e392e1905dcd48aec3_31.png)

# 2. 计算每日回撤深度
drawdown_depth = running_max - df[‘net_value’]

![](img/88ee8346477ff6e392e1905dcd48aec3_33.png)

# 3. 计算每日回撤比率
drawdown_ratio = drawdown_depth / running_max

![](img/88ee8346477ff6e392e1905dcd48aec3_35.png)

![](img/88ee8346477ff6e392e1905dcd48aec3_37.png)

# 4. 找出最大回撤
max_drawdown = drawdown_ratio.max()
print(f“最大回撤为：{max_drawdown:.2%}”)
```

这段代码中，`cummax()` 函数是关键，它返回序列到当前位置为止的最大值。通过简单的减法和除法，我们就能得到回撤序列，并最终找到最大回撤值。

## 在回测框架中查看最大回撤

在实际的量化回测框架（如Backtrader、Zipline等）中，最大回撤通常会被自动计算并作为关键统计指标输出。在回测结果图中，最大回撤区间也常被用阴影高亮标出，直观地展示了策略历史上最困难的时期。
