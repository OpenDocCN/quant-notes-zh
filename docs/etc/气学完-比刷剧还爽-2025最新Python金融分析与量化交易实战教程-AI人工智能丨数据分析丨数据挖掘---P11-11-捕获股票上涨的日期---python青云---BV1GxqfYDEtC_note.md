在本节课中，我们将学习如何从股票历史数据中，找出所有收盘价比开盘价上涨超过3%的交易日。这是量化分析中一个基础但重要的步骤，可以帮助我们识别股票的强势上涨日期。

## 数据预处理回顾

![](img/455ea351537c90fe3f74d188454ef4ea_8.png)

上一节我们介绍了如何对股票历史交易数据进行预处理，特别是将`date`列转换为时间序列并设置为行索引。这样做是因为后续的分析需求（例如查找特定日期）会频繁用到日期。将日期作为时间序列索引后，我们可以利用Pandas时间序列的特性来高效地完成这些操作。

![](img/455ea351537c90fe3f74d188454ef4ea_10.png)

## 核心需求：找出上涨超过3%的日期

![](img/455ea351537c90fe3f74d188454ef4ea_12.png)

我们的目标是找出贵州茅台这支股票所有“收盘价比开盘价上涨3%”的交易日。分析这些日期有助于我们总结股票的上涨规律，例如某些月份上涨频率更高，这可能为未来的投资时机提供参考。

以下是实现此需求的核心逻辑：
1.  **计算日涨幅**：通过公式 `(收盘价 - 开盘价) / 开盘价` 计算每日的涨跌幅。
2.  **筛选条件**：找出涨幅大于3%（即 `> 0.03`）的日期。
3.  **提取日期**：从满足条件的数据中提取出对应的日期。

![](img/455ea351537c90fe3f74d188454ef4ea_14.png)

## 分步实现教程

### 步骤一：构建布尔条件序列

首先，我们需要创建一个布尔序列（Boolean Series），其中`True`表示该交易日满足上涨超过3%的条件，`False`则表示不满足。

![](img/455ea351537c90fe3f74d188454ef4ea_16.png)

在Pandas中，我们可以直接对`DataFrame`的列进行向量化运算。假设我们的数据框名为`df`，包含`open`（开盘价）和`close`（收盘价）两列。

![](img/455ea351537c90fe3f74d188454ef4ea_18.png)

![](img/455ea351537c90fe3f74d188454ef4ea_20.png)

```python
# 计算每日涨跌幅，并判断是否大于3%
condition = (df[‘close‘] - df[‘open‘]) / df[‘open‘] > 0.03
```
执行这行代码后，`condition`变量将成为一个与`df`行数相同的布尔序列。序列中的每一个`True`值都对应一个满足我们上涨条件的交易日。

![](img/455ea351537c90fe3f74d188454ef4ea_22.png)

### 步骤二：利用布尔索引提取数据

在Pandas中有一个非常实用的技巧：**可以将布尔序列直接作为`DataFrame`的行索引，从而快速筛选出所有`True`值对应的行数据。**

![](img/455ea351537c90fe3f74d188454ef4ea_24.png)

以下是其工作原理：
*   如果布尔序列作为行索引，Pandas会返回所有对应位置为`True`的行。
*   所有对应位置为`False`的行将被自动忽略。

![](img/455ea351537c90fe3f74d188454ef4ea_26.png)

因此，我们可以直接使用上一步得到的`condition`来筛选数据：

![](img/455ea351537c90fe3f74d188454ef4ea_28.png)

```python
# 使用布尔索引筛选出满足条件的行
filtered_df = df.loc[condition]
```
现在，`filtered_df`包含了所有收盘价比开盘价上涨超过3%的交易日完整数据（包括开盘价、收盘价、最高价、最低价等）。

![](img/455ea351537c90fe3f74d188454ef4ea_30.png)

### 步骤三：提取目标日期

![](img/455ea351537c90fe3f74d188454ef4ea_32.png)

![](img/455ea351537c90fe3f74d188454ef4ea_34.png)

我们最终需要的是日期，而不是整行数据。由于在预处理时我们已经将日期设置为行索引（`index`），因此满足条件的日期就是`filtered_df`的行索引。

![](img/455ea351537c90fe3f74d188454ef4ea_36.png)

```python
# 提取满足条件的日期索引
up_dates = filtered_df.index
```
至此，`up_dates`就是我们需要找的所有上涨日期。

![](img/455ea351537c90fe3f74d188454ef4ea_38.png)

## 完整代码示例

![](img/455ea351537c90fe3f74d188454ef4ea_40.png)

将以上步骤整合，实现需求的完整代码如下：

![](img/455ea351537c90fe3f74d188454ef4ea_42.png)

![](img/455ea351537c90fe3f74d188454ef4ea_44.png)

```python
# 假设 df 是已经预处理好的DataFrame，date列已设为索引
# 计算条件
condition = (df[‘close‘] - df[‘open‘]) / df[‘open‘] > 0.03

# 使用布尔索引筛选数据
filtered_df = df.loc[condition]

![](img/455ea351537c90fe3f74d188454ef4ea_46.png)

# 获取日期
rise_dates = filtered_df.index
print(rise_dates)
```

本节课中我们一起学习了如何捕获股票上涨的日期。我们首先回顾了将日期设为索引的重要性。然后，通过构建`(close - open) / open > 0.03`的布尔条件序列，找出了所有上涨超过3%的交易日。最关键的一步是掌握了**将布尔序列作为`DataFrame`的行索引来筛选数据**的Pandas技巧，这能高效地提取出目标行。最后，我们从筛选结果中提取出行索引，即为我们所需的日期列表。这个方法在量化分析中会经常使用，请务必熟练掌握并勤加练习。
