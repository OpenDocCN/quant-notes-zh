## 计算短期移动平均线

![](img/9153d849ce091070cdf3542a2d6538fd_7.png)

![](img/9153d849ce091070cdf3542a2d6538fd_9.png)

首先，我们需要计算短期移动平均线。移动平均线是过去一段时间内股价的平均值，用于平滑价格波动，反映趋势。

通常情况下，短期移动平均线使用5天（即一周的交易日）作为窗口。但由于我们的示例数据时间跨度较长，为了更清晰地展示，我们将短期窗口设置为10天。

以下是计算短期移动平均线的代码步骤。我们以苹果公司（AAPL）的股票数据为例。

```python
# 假设 df 是包含苹果股价的DataFrame，其中‘Close’是收盘价列
# 计算10日短期移动平均线
short_window = 10
df['M1'] = df['Close'].rolling(window=short_window).mean()
```

![](img/9153d849ce091070cdf3542a2d6538fd_11.png)

在这段代码中，`df[‘Close’].rolling(window=short_window).mean()` 计算了收盘价在10天窗口内的滚动平均值，并将结果存储在新列 `M1` 中。

![](img/9153d849ce091070cdf3542a2d6538fd_13.png)

![](img/9153d849ce091070cdf3542a2d6538fd_15.png)

## 计算长期移动平均线

![](img/9153d849ce091070cdf3542a2d6538fd_17.png)

接下来，我们计算长期移动平均线。长期均线通常使用20天或30天作为窗口，代表一个月左右的趋势。这里我们选择30天作为长期窗口。

![](img/9153d849ce091070cdf3542a2d6538fd_19.png)

以下是计算长期移动平均线的代码。

![](img/9153d849ce091070cdf3542a2d6538fd_21.png)

```python
# 计算30日长期移动平均线
long_window = 30
df['M2'] = df['Close'].rolling(window=long_window).mean()
```

![](img/9153d849ce091070cdf3542a2d6538fd_23.png)

这段代码与短期均线类似，只是窗口大小变为30天，结果存储在新列 `M2` 中。

![](img/9153d849ce091070cdf3542a2d6538fd_25.png)

![](img/9153d849ce091070cdf3542a2d6538fd_27.png)

## 可视化股价与移动平均线

计算完短期（M1）和长期（M2）移动平均线后，我们可以将它们与原始股价一起绘制在图表上，以便直观观察。

以下是绘制折线图的代码。为了图像更清晰，我们只选取了最近250个交易日（大约一年）的数据进行展示。

import matplotlib.pyplot as plt

## 识别黄金交叉与死亡交叉

现在，我们来观察图表，识别“黄金交叉”和“死亡交叉”。

*   **死亡交叉**：当短期移动平均线从上方向下穿越长期移动平均线时，形成死亡交叉。这通常被视为下跌趋势开始的信号，提示投资者可能应考虑卖出或避免买入。在图中，可以看到绿色线（短期）下穿红色线（长期）的区域。
*   **黄金交叉**：当短期移动平均线从下方向上穿越长期移动平均线时，形成黄金交叉。这通常被视为上涨趋势开始的信号，提示投资者可能可以考虑买入。在图中，可以看到绿色线（短期）上穿红色线（长期）的区域。

![](img/9153d849ce091070cdf3542a2d6538fd_41.png)

![](img/9153d849ce091070cdf3542a2d6538fd_43.png)

通过观察这些交叉点，我们可以对市场趋势做出初步判断。

## 使用数值信号增强分析

![](img/9153d849ce091070cdf3542a2d6538fd_45.png)

![](img/9153d849ce091070cdf3542a2d6538fd_47.png)

为了让交叉信号更加明确，我们可以创建一个数值信号列。当短期均线高于长期均线时，标记为1；当低于时，标记为-1。这样，信号的变化点（从1到-1或从-1到1）就直接对应了交叉点。

![](img/9153d849ce091070cdf3542a2d6538fd_49.png)

以下是生成信号列的代码。

![](img/9153d849ce091070cdf3542a2d6538fd_51.png)

```python
import numpy as np

![](img/9153d849ce091070cdf3542a2d6538fd_53.png)

# 生成交易信号：短期均线大于长期均线为1，小于为-1
df['position'] = np.where(df['M1'] > df['M2'], 1, -1)
```

![](img/9153d849ce091070cdf3542a2d6538fd_55.png)

为了更清晰地展示这个信号，我们可以将其绘制在股价图的右侧Y轴上。

```python
fig, ax1 = plt.subplots(figsize=(12, 6))

![](img/9153d849ce091070cdf3542a2d6538fd_57.png)

# 左侧Y轴：绘制股价和均线
ax1.plot(plot_data['Close'], label='Stock Price', color='blue')
ax1.plot(plot_data['M1'], label='Short MA', color='green')
ax1.plot(plot_data['M2'], label='Long MA', color='red')
ax1.set_ylabel('Price')
ax1.legend(loc='upper left')

![](img/9153d849ce091070cdf3542a2d6538fd_59.png)

# 右侧Y轴：绘制交易信号
ax2 = ax1.twinx()
ax2.plot(plot_data['position'], label='Trading Signal', color='orange', linestyle='--')
ax2.set_ylabel('Signal (1/-1)')
ax2.set_yticks([-1, 1])
ax2.legend(loc='upper right')

![](img/9153d849ce091070cdf3542a2d6538fd_61.png)

plt.show()
```

在这张双轴图中，橙色虚线代表交易信号。当信号从-1变为1时，对应**黄金交叉**（买入信号）；当信号从1变为-1时，对应**死亡交叉**（卖出信号）。这使得交叉点的识别变得一目了然。
