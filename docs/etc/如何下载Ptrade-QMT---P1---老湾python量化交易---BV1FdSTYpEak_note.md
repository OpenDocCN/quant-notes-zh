在本节课中，我们将学习如何获取Ptrade和QMT这两个程序化交易平台。许多初学者希望在自己的电脑上运行程序化交易，但常常遇到下载和登录的问题。本节将解释国内程序化交易的环境特点，并说明为何无法直接提供通用的下载链接。

## 国内程序化交易环境解析 🔍

上一节我们提到了下载需求，本节中我们来看看其背后的原因。2019年，中国证券业协会发布通知，要求券商交易系统严防外挂接入。这可以从两方面理解：

1.  券商不能直接向个人投资者提供API交易接口。
2.  投资者的所有交易操作必须在券商提供的软件环境中进行，无论是手机App还是电脑客户端。

Ptrade和QMT正是在此规定下诞生的。它们类似于手机操作系统（如安卓），但投资者无法直接从谷歌下载纯净版安卓系统，必须通过手机厂商（如小米、OPPO）预装并定制后获取。同理，券商从恒生电子或迅投等公司购买Ptrade或QMT的“纯净版”后，会将其部署在自己的服务器上，并接入行情数据接口和交易柜台，最后才将定制好的客户端提供给符合条件的投资者。

因此，投资者获得的只是一个远程控制端，真正的策略运行环境在券商内部，这确保了合规性，防止了外挂接入。

## 平台下载与安装步骤 💻

理解了环境背景后，以下是获取和尝试Ptrade与QMT的步骤。请注意，没有对应的券商账户和密码，即使安装了软件也无法登录使用。

![](img/b80246782e91ab9bf24755e13f7d92b6_5.png)

以下是两个提供客户端下载的券商示例：

*   **国信证券**：提供名为“**iQuant**”的QMT版本。
*   **中泰证券**：提供名为“**XTP**”的实盘QMT版本。

![](img/b80246782e91ab9bf24755e13f7d92b6_7.png)

软件安装过程与常规软件无异。但安装完成后，登录时会要求输入资金账号和交易密码，这些信息需要您向开户券商申请获取。

![](img/b80246782e91ab9bf24755e13f7d92b6_9.png)

## 如何获取实盘交易账号 🔑

上一节我们介绍了如何下载软件，本节中我们来看看如何获得登录权限。由于开通Ptrade或QMT实盘权限通常有资金门槛等要求，建议您通过以下途径：

![](img/b80246782e91ab9bf24755e13f7d92b6_11.png)

1.  联系自己所在券商的客户经理，咨询开通程序化交易权限的具体条件和流程。
2.  在相关视频或社区下方，常有券商客户经理留言提供开户服务，您可以与之联系。
3.  若无法满足资金门槛，可以私信我，我能协助您寻找免门槛开通这些交易平台的渠道。

![](img/b80246782e91ab9bf24755e13f7d92b6_13.png)

请注意，登录Ptrade或QMT的账号密码通常与您的普通股票交易账户一致。

## 总结 📝

本节课中我们一起学习了Ptrade和QMT的获取方式。核心要点在于：**这些平台是券商定制并提供给客户的，并非独立的公开软件**。因此，直接下载的客户端无法脱离券商账户使用。整个流程可以概括为：

![](img/b80246782e91ab9bf24755e13f7d92b6_15.png)

**`投资者 -> 向券商申请权限 -> 获取定制客户端及账号 -> 在券商合规环境下运行策略`**

希望本教程能帮助您清晰了解国内程序化交易平台的获取路径。如果您已准备好，下一步就可以联系券商，开启您的量化交易实践了。
