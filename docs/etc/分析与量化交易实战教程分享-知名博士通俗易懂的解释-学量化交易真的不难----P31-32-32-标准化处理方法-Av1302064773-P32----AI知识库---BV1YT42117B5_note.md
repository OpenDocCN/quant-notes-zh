在本节课中，我们将要学习数据预处理中一个至关重要的技术——标准化。标准化是量化分析和机器学习中常用的数据转换方法，其核心目的是消除不同特征（或指标）之间由于量纲和取值范围不同带来的影响，使数据处于同一尺度，便于后续的模型分析和比较。

## 标准化的目的与原理 🎯

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体作用。

标准化操作旨在解决一个常见问题：数据集中不同特征（例如X1和X2）的原始取值范围可能差异巨大。这种差异会导致在后续分析（如构建模型）时，取值范围大的特征会主导结果，而取值范围小的特征其影响则被削弱，这并非我们期望的。

标准化的战略出发点是为了让各个因子或指标，在不同维度上的取值范围尽可能相同。这样我们可以用同等的“眼光”去观察和比较它们，避免单纯因数值大小差异而对分析结果造成不必要的影响。

## 标准化的数学公式与步骤 ➗

理解了“为什么”要做标准化之后，我们来看看“如何”做。标准化的核心公式如下：

**`z = (x - μ) / σ`**

其中：
*   **`x`** 是原始数据。
*   **`μ`** 是该特征所有数据的均值。
*   **`σ`** 是该特征所有数据的标准差。
*   **`z`** 是标准化后的数据。

这个公式可以分解为两个清晰的步骤：

1.  **去均值（中心化）**：`x - μ`
    *   **目的**：将数据分布的中心移动到零点。原始数据通常不以原点为中心对称，执行去均值操作后，数据在各个维度上都将以零为中心对称。

2.  **除标准差（缩放）**：`(x - μ) / σ`
    *   **目的**：调整数据的尺度，使不同特征的数据分布范围近似。去均值后，各维度的取值范围可能仍然不同。通过除以各自的标准差，可以解决这个问题。
    *   **原理**：标准差反映了数据的离散程度。取值范围大、数据差异大的特征，其标准差也大；反之则小。用原始值除以标准差后，原本取值范围大的数据会被一个较大的数压缩，原本取值范围小的数据则被一个较小的数相对放大。最终，所有特征的数值都被缩放至一个相近的、较小的区间内（通常围绕0分布）。

理论清晰后，实现起来就非常简单了。我们可以根据公式手动实现一个标准化函数。

以下是手动实现标准化操作的Python代码示例：

![](img/3103438a4eaf3188f2fc6d135c6437ae_9.png)

![](img/3103438a4eaf3188f2fc6d135c6437ae_11.png)

```python
def standardize_series(series):
    """
    对输入的Pandas Series进行标准化处理。
    参数:
        series: 待标准化的Pandas Series数据。
    返回:
        标准化后的Pandas Series数据。
    """
    # 计算该序列的均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    # 应用标准化公式： (x - mean) / std
    standardized_series = (series - mean_val) / std_val
    return standardized_series
```

![](img/3103438a4eaf3188f2fc6d135c6437ae_13.png)

代码逻辑非常直接：先计算数据的均值和标准差，然后对序列中的每一个值应用标准化公式。

## 总结 ✨

![](img/3103438a4eaf3188f2fc6d135c6437ae_35.png)

![](img/3103438a4eaf3188f2fc6d135c6437ae_37.png)

本节课中我们一起学习了数据标准化处理。
*   **核心目的**：消除不同特征间量纲和取值范围差异的影响，使数据处于同一尺度。
*   **核心公式**：**`z = (x - μ) / σ`**。
*   **两个关键步骤**：第一步是**去均值**，使数据以零为中心对称；第二步是**除标准差**，将不同特征的数据范围调整至相近区间。
*   **实现方式**：可以手动根据公式编码实现，也可以直接调用如`scikit-learn`中的`StandardScaler`等成熟工具。
