## 第二步：初步可视化观察关系

![](img/27a1ccd2f65cb458840eb941cccff789_35.png)

![](img/27a1ccd2f65cb458840eb941cccff789_37.png)

数据准备好后，最直接的方法是将其可视化。以下是绘制两个指标折线图的方法：

![](img/27a1ccd2f65cb458840eb941cccff789_39.png)

![](img/27a1ccd2f65cb458840eb941cccff789_41.png)

1.  使用Pandas的绘图功能，将两个序列画在同一张图上。
2.  由于两个指标的量纲和数值范围不同，直接绘制可能难以观察趋势。我们可以使用双Y轴的方法，让它们共享同一个X轴（时间），但拥有各自的Y轴刻度。

![](img/27a1ccd2f65cb458840eb941cccff789_43.png)

![](img/27a1ccd2f65cb458840eb941cccff789_45.png)

![](img/27a1ccd2f65cb458840eb941cccff789_47.png)

```python
# 使用双Y轴绘制折线图
ax = data.plot(secondary_y=['VIX'], figsize=(10, 6))
```

通过观察图形，我们可以初步判断：当SPX指数下降时，VIX指数往往上升，显示出一种负相关关系。

![](img/27a1ccd2f65cb458840eb941cccff789_49.png)

## 第三步：使用散点图矩阵深入分析

![](img/27a1ccd2f65cb458840eb941cccff789_63.png)

除了折线图，散点图矩阵是观察多个变量间两两关系的强大工具。以下是绘制步骤：

1.  首先，我们计算指标的日收益率（或对数收益率），以分析变化率之间的关系。
2.  使用Pandas的 `scatter_matrix` 函数绘制散点图矩阵。矩阵的对角线可以显示每个变量自身的分布（直方图或核密度估计图）。

![](img/27a1ccd2f65cb458840eb941cccff789_65.png)

```python
import pandas as pd
import numpy as np

![](img/27a1ccd2f65cb458840eb941cccff789_67.png)

# 计算对数收益率
returns = np.log(data / data.shift(1))

# 绘制散点图矩阵，对角线显示核密度估计图
pd.plotting.scatter_matrix(returns, alpha=0.5, figsize=(10, 10), diagonal='kde')
```

![](img/27a1ccd2f65cb458840eb941cccff789_69.png)

![](img/27a1ccd2f65cb458840eb941cccff789_71.png)

通过散点图，我们可以更直观地看到SPX收益率与VIX收益率之间的负相关关系（点云呈向下倾斜的趋势）。对角线上的核密度估计图则展示了每个收益率序列自身的分布情况。

![](img/27a1ccd2f65cb458840eb941cccff789_73.png)

## 第四步：进行回归分析

![](img/27a1ccd2f65cb458840eb941cccff789_75.png)

![](img/27a1ccd2f65cb458840eb941cccff789_77.png)

![](img/27a1ccd2f65cb458840eb941cccff789_79.png)

散点图展示了关系，但我们还需要量化这种关系。这就是回归分析的作用。我们可以为散点图添加一条回归线，以更清晰地展示趋势。

![](img/27a1ccd2f65cb458840eb941cccff789_81.png)

![](img/27a1ccd2f65cb458840eb941cccff789_83.png)

虽然本段内容未给出完整的回归代码，但其思路是：使用统计库（如`statsmodels`或`scikit-learn`）对两个收益率序列进行线性回归拟合，得到回归方程（形如 `y = a*x + b`），并将这条直线画在散点图上。回归系数 `a` 的大小和符号（正或负）就量化了变量间的影响程度和方向。

![](img/27a1ccd2f65cb458840eb941cccff789_85.png)
