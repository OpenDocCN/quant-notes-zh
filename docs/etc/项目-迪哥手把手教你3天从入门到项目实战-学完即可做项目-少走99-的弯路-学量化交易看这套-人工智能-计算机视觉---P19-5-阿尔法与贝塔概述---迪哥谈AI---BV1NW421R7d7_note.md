# 金融量化分析：P19：5-阿尔法与贝塔概述 📊

在本节课中，我们将要学习量化策略评估中的两个核心概念：阿尔法（α）和贝塔（β）。理解这两个指标，能帮助我们分清投资收益的来源，明确量化策略的优化方向。

## 收益的构成

上一节我们介绍了多种策略评估指标，本节中我们来看看收益的深层构成。任何投资的收益都可以被分解为两个部分。

一部分收益与整体市场环境相关。当市场大环境向好时，大部分投资都能获得收益。这部分收益可以理解为“随大流”赚的钱。

另一部分收益则与市场整体波动关系不大，它主要来源于投资者或策略自身的努力。例如，通过深入分析公司财务、运营状况，凭借独特的眼光和操作手法获得的收益。

## 阿尔法（α）与贝塔（β）的定义

基于上述收益构成，我们引入阿尔法和贝塔这两个指标来分别衡量它们。

*   **贝塔（β）**：衡量投资组合收益对**市场波动**的敏感性，即“系统性风险”。它代表了“随大流”获得的那部分收益。公式上，它常出现在以下回归模型中：
    `投资组合收益 = α + β * 市场收益 + ε`
*   **阿尔法（α）**：衡量**超越市场基准**的超额收益，即“非系统性风险”带来的回报。它代表了通过自身策略和能力获得的那部分收益。在上述公式中，α 即为截距项。

## 通过图表理解概念

以下是理解这两个概念的直观方式：



![](img/18cfb58d81ea186c3b6362474c9adcf5_1.png)

图中展示了三种收益：
1.  **策略收益**：你的投资策略实际获得的总收益。
2.  **基准收益**：市场整体（如沪深300指数）的收益，代表“大环境”。
3.  **超额收益**：策略收益减去基准收益后的部分，这直接对应了我们追求的**阿尔法（α）**。

因此，总收益可拆解为：**总收益 = 市场收益（贝塔部分） + 超额收益（阿尔法部分）**。

## 量化分析中的关注点

在量化策略开发中，我们的核心目标是获取稳定的**阿尔法**。因为市场（贝塔）是个人无法控制的宏观因素，而阿尔法才真正体现了策略的优劣和我们的努力价值。

后续在进行因子分析等深入策略研究时，我们会具体讲解如何通过回归等方法，从历史数据中求解出策略的阿尔法和贝塔值。

## 策略评估指标回顾

最后，我们回顾一下策略评估中的其他常见指标。对于初学者，理解其含义远比死记公式更重要，实际应用中通常有现成的工具包进行计算。

以下是几个关键指标的含义：
*   **最大回撤**：衡量策略在历史上可能面临的最大亏损幅度。
*   **夏普比率**：衡量每承受一单位风险，能获得多少超额回报。
*   **基准收益**：代表“什么都不做，单纯跟随市场”所能获得的收益。我们的目标就是通过策略战胜这个基准，获取正的阿尔法。

## 总结

![](img/18cfb58d81ea186c3b6362474c9adcf5_5.png)

本节课中我们一起学习了阿尔法（α）和贝塔（β）的核心概念。贝塔衡量了收益中与市场共舞的部分，而阿尔法则代表了通过自身策略战胜市场所获得的超额收益。在量化交易中，我们持续努力的方向就是寻找并放大能带来稳定阿尔法的策略因子。理解这一点，是构建有效量化策略的重要基础。
