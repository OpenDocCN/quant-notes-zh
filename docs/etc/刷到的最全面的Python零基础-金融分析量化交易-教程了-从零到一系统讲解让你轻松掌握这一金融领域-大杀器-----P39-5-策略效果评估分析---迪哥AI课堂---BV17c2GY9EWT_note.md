![](img/f09f1bb03b98954ca6300640fde7baf7_37.png)

![](img/f09f1bb03b98954ca6300640fde7baf7_39.png)

*   **参数缺失**：在调用中性化处理函数时，缺少必要的参数，例如指标名称。
*   **方法调用错误**：在使用统计模型（如OLS）后，忘记调用 `fit()` 方法进行训练。
*   **拼写错误**：变量名或函数名拼写错误，例如将 `quantile` 误写为 `qant`。
*   **属性引用错误**：在访问数据结构（如列表）时，使用了错误的属性或方法名。

回测完成后，平台会生成一系列分析报告和图表。我们需要关注几个核心的绩效评估指标：

![](img/f09f1bb03b98954ca6300640fde7baf7_75.png)

![](img/f09f1bb03b98954ca6300640fde7baf7_77.png)

![](img/f09f1bb03b98954ca6300640fde7baf7_79.png)

*   **策略收益率 vs 基准收益率**：比较策略创造的收益与市场基准（如沪深300指数）的收益。公式可以简化为：
    `策略收益 = (期末总资产 - 期初总资产) / 期初总资产`
*   **年化收益率**：将策略的总收益率折算成年利率形式，便于比较。
*   **最大回撤**：策略净值从最高点到最低点的最大跌幅，衡量策略可能面临的最大风险。
*   **夏普比率**：衡量每承受一单位总风险，能产生多少超额回报。公式为：
    `夏普比率 = (策略年化收益率 - 无风险利率) / 策略收益率的年化标准差`

![](img/f09f1bb03b98954ca6300640fde7baf7_81.png)

![](img/f09f1bb03b98954ca6300640fde7baf7_83.png)

![](img/f09f1bb03b98954ca6300640fde7baf7_85.png)

![](img/f09f1bb03b98954ca6300640fde7baf7_87.png)

在我们的示例回测中，策略在选定年份的表现可能优于基准指数（基准收益为负时，策略亏损更少或略有盈利）。然而，其夏普比率可能较低，表明风险调整后的收益能力有待提升。

本节课中我们一起学习了量化策略效果评估的全过程。我们首先运行了基于因子选股的策略回测，期间修正了代码中的常见错误。随后，我们分析了回测输出的关键绩效指标，如收益率、最大回撤和夏普比率，以判断策略的历史表现和风险特征。最后，我们了解到还可以通过交易详情和资产变化等详细数据，对策略行为进行更深入的验证和分析。这个过程是量化交易中检验和优化策略想法的重要环节。
