# 量化投资基础：P37：因子分析概述

## 概述
在本节课中，我们将要学习量化投资中的一个核心环节——因子分析。我们将了解什么是因子，以及如何从众多因子中筛选出对股票收益率有显著影响的优质因子。核心在于计算并分析因子与收益率之间的相关性，即IC值分析。

## 因子分析的目标
在量化投资中，我们通常会收集大量与股票相关的指标数据，这些指标被称为“因子”。例如，基本面因子（如市盈率、市净率）和技术指标因子（如移动平均线、RSI）。一个股票可能对应成百上千个因子。

问题随之而来：当我们进行策略回测或设计时，是否需要使用所有因子？答案是否定的。我们需要判断哪些因子是有效的，哪些是无效的。换句话说，我们需要给这些因子“打分”和“排序”，依据是它们对最终投资收益的影响程度。影响大的因子得分高，影响小甚至起反作用的因子得分低。

## 如何衡量因子的好坏
衡量因子好坏最直接的方法是观察它与“收益率”的关系。收益率是衡量投资回报的核心指标。

**收益率的计算公式**如下：
`日收益率 = (当日收盘价 - 前一日收盘价) / 前一日收盘价`

我们的目标是实现长期盈利，因此关注每日收益率的变化。同时，每个因子的数值也在每日变化。我们需要分析的是：因子的变化趋势与收益率的变化趋势之间存在何种关系？是正相关、负相关，还是无关？相关性的强弱又如何？

## 因子分析的核心：IC值
因子分析主要包含两部分工作，第一项是计算因子的**IC值**。

![](img/23fe8d96008766ab7c23eefed4aa3851_1.png)

**IC** 是 Information Coefficient 的缩写，中文可称为信息系数。它本质上是一个相关性指标，用于衡量**单个因子**与**股票收益率**之间的相关性。我们通常使用**斯皮尔曼秩相关系数**来计算IC值。

**斯皮尔曼相关系数公式**（概念）：
它衡量的是两个变量排序之间的单调关系，取值在 **-1 到 1** 之间。
*   **值接近 1**：表示强正相关（因子值越大，收益率倾向于越高）。
*   **值接近 -1**：表示强负相关（因子值越大，收益率倾向于越低）。
*   **值接近 0**：表示两者基本没有单调关系。

因此，**单因子分析**就是：在收益率相对固定的前提下，遍历每一个因子，分别计算它们与收益率之间的斯皮尔曼相关系数。这个计算出的相关系数，就是该因子的**IC值**。

## IC值的可视化分析
计算出IC值后，我们需要对其进行分析和解读。通常我们会通过图表来直观展示。

以下是IC值分析的两种常见可视化图表：



![](img/23fe8d96008766ab7c23eefed4aa3851_3.png)

*   **IC值序列图**：这张图展示了每一天计算出的IC值（蓝色折线）。通过它，我们可以观察IC值随时间的变化情况，看其是否稳定。
*   **IC值移动平均线**：为了平滑每日IC值的波动，更好地观察其趋势，我们会计算IC值的移动平均值（例如10日移动平均），并在图中用绿色线条表示。图中前期的空值是因为移动平均计算需要一定窗口期的数据。

通过观察这些图表，我们可以判断：
1.  IC值是持续为正还是为负？这决定了因子与收益率是正相关还是负相关。
2.  IC值的绝对值大小如何？绝对值越大，说明因子对收益率的预测能力越强。
3.  IC值的趋势是否稳定？稳定的IC值意味着因子在不同市场环境下都有效。

我们的目标就是找出那些IC值**绝对值较大、方向稳定（正或负）且持续性较好**的因子。对于IC值接近0或非常不稳定的因子，则可以认为其有效性不足，在构建策略时可以考虑剔除。

## 总结
本节课我们一起学习了量化投资中因子分析的基础概念。我们首先明确了分析目标是筛选有效因子，然后引入了**收益率**作为衡量基准。核心内容是**IC值分析**，即通过计算**斯皮尔曼相关系数**来衡量单个因子与收益率的相关性。最后，我们了解了如何通过可视化图表（IC值序列图和移动平均线）来分析和解读IC值，从而完成对因子的初步评估和排序。在接下来的实践中，我们将用代码具体实现这一分析过程。
