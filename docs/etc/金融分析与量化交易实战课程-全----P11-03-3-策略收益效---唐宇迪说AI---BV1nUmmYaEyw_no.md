## 计算原始报酬率

上一节我们介绍了如何生成交易信号，本节中我们来看看如何基于这些信号计算策略收益。首先，我们需要计算股票原始的每日报酬率。

报酬率（Return）的计算公式是：`当日股价 / 前一日股价`。在代码中，我们使用Pandas的`shift`函数来获取前一日的数据。

```python
# 计算每日报酬率
data['return'] = data['close'] / data['close'].shift(1)
```

执行这段代码后，`data`数据框中会新增一列`return`，它代表了每一天相对于前一天的股价增长率。

## 根据策略调整收益

![](img/02db3c607e42217f81f84d88a3d95309_9.png)

接下来，我们需要根据之前生成的交易信号（`position`列）来调整收益计算。我们的策略逻辑是：当`position`为1时，代表我们预测市场将下跌，因此采取与市场走势相反的操作（例如做空或卖出）；当`position`为0时，代表我们跟随市场原始走势。

以下是策略收益的计算逻辑：
*   当 `position` 为 0 时：策略收益 = 原始报酬率 (`return`)
*   当 `position` 为 1 时：策略收益 = -原始报酬率 (`return`)

![](img/02db3c607e42217f81f84d88a3d95309_11.png)

![](img/02db3c607e42217f81f84d88a3d95309_13.png)

这个逻辑可以简化为一个公式：**策略收益 = (1 - 2 * position) * 原始报酬率**。但在代码中，我们更直观地使用`shift`确保信号与收益日期对齐后进行计算。

```python
# 根据策略信号计算策略收益
# 注意：position也需要进行shift(1)操作，以确保使用前一日生成的信号来决定今日的操作
data['strategy'] = (1 - 2 * data['position'].shift(1)) * data['return']
```

![](img/02db3c607e42217f81f84d88a3d95309_15.png)

执行后，`data`数据框中会新增`strategy`列，它代表了应用我们的双均线策略后，每日的理论收益率。

## 计算并比较累计收益

![](img/02db3c607e42217f81f84d88a3d95309_19.png)

为了比较长期效果，我们需要计算从期初到期末的总累计收益。由于我们之前计算的是每日的增长率（乘性关系），总收益是这些每日增长率的连乘积。

![](img/02db3c607e42217f81f84d88a3d95309_21.png)

然而，在计算过程中，我们通常先对收益率取对数（将连乘变为累加），计算完后再通过指数函数还原。以下是计算步骤：

1.  **处理缺失值**：由于使用了`shift`操作，数据第一行会产生缺失值（NaN），需要先将其剔除。
2.  **计算对数收益率总和**：对`return`和`strategy`列取对数（`np.log`），然后求和（`.sum()`）。这相当于计算了总的对数收益。
3.  **还原为总收益**：将对数收益总和通过指数函数（`np.exp`）还原，得到最终的总资产倍数（即1元本金最终变为多少元）。

![](img/02db3c607e42217f81f84d88a3d95309_23.png)

![](img/02db3c607e42217f81f84d88a3d95309_25.png)

以下是核心代码：

![](img/02db3c607e42217f81f84d88a3d95309_27.png)

```python
import numpy as np

![](img/02db3c607e42217f81f84d88a3d95309_29.png)

# 删除含有缺失值的行
data_clean = data.dropna()

# 计算并比较总收益
total_return_buy_hold = np.exp(np.log(data_clean['return']).sum())
total_return_strategy = np.exp(np.log(data_clean['strategy']).sum())

print(f"买入持有策略总收益： {total_return_buy_hold:.2f}")
print(f"双均线策略总收益： {total_return_strategy:.2f}")
```

执行上述代码后，我们可以得到两个数值。例如：
*   买入持有总收益：4.XX
*   双均线策略总收益：5.XX

![](img/02db3c607e42217f81f84d88a3d95309_37.png)

这个结果表明，在回测的时间段内：
*   如果期初投入1元并一直持有该股票（“什么都不做”），期末资产会变为约4元。
*   如果期初投入1元并严格执行我们的双均线策略，期末资产会变为约5.8元。

因此，在这个特定的回测案例中，我们构建的双均线策略获得了超越简单买入持有策略的收益。
