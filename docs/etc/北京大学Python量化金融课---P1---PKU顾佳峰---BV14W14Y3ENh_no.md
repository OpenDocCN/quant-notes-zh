# Python量化金融课程：P1：课程介绍与招生信息

在本节课中，我们将介绍北京大学Python量化金融研修班第二期的课程概况、核心内容、师资力量以及报名方式。

---

北京大学Python量化金融研修班第二期现已开始招生。课程面向对Python量化金融感兴趣的学习者。

课程将于2004年12月13日至15日在北京大学举办。由于第一期课程反响热烈，且近期市场行情波动显著，许多同学表达了强烈的学习意愿。因此，本年度最后一期课程定于年底举办，名额有限，建议有意者尽快报名。

以下是报名方式的具体说明。

*   报名信息与招生简章均发布在指定的微信公众号上。
*   请关注该公众号以获取全部课程详情与报名通道。

上一节我们介绍了课程的基本信息，本节中我们来看看课程的具体内容安排。

课程为期三天，共包含六讲，内容安排紧凑。前三讲主要介绍Python编程基础及其相关工具。后三讲则深入探讨量化金融领域，涵盖从大数据分析到大模型应用的具体案例。

以下是详细的课程内容结构。

![](img/31e57d7f368d7d8031cb13c573ea38d8_3.png)

*   **前三讲**：Python编程基础与工具介绍。
*   **后三讲**：量化金融专题，聚焦大数据与大模型的应用实践。
*   **最后两讲**：由顾佳峰老师讲授，主题分别为“用Python做量化金融分析”与“用机器学习做量化金融分析”。

了解课程内容后，强大的师资团队是保证学习效果的关键。本课程邀请了多位经验丰富的老师。

姚佳慧老师与吕萍老师均来自北京大学中国社会科学调查中心。她们在Python数据挖掘与统计分析方面拥有深厚的学术背景与丰富的实践经验。

![](img/31e57d7f368d7d8031cb13c573ea38d8_5.png)

王冉冉老师来自北京大学大数据分析技术国家工程实验室，并担任北京大学重庆大数据研究院产业金融研究中心的执行主任。她在量化金融、大数据及大语言模型领域有着深入的研究和丰富的项目经验。
