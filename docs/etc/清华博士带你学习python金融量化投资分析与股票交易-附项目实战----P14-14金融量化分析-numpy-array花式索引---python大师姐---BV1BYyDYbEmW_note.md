上一节我们介绍了布尔型索引，它允许我们根据条件筛选数组元素。本节中我们来看看花式索引，它用于根据一个整数数组（即索引列表）来选取数据。

## 花式索引的基本概念

花式索引的核心是使用一个整数数组来指定要选取的元素位置。这与切片不同，选取的位置可以没有规律。

例如，对于一个一维数组，我们想选取第1、3、4、6、7号元素。使用花式索引的代码如下：

```python
import numpy as np
a = np.arange(20)  # 创建一个0到19的数组
indices = [1, 3, 4, 6, 7]  # 指定要选取的索引
result = a[indices]
print(result)  # 输出：[1 3 4 6 7]
```

**注意**：在索引中直接使用逗号（如 `a[1, 3, 4, 6, 7]`）是错误的，因为逗号在NumPy中用于分隔不同维度的索引。必须将索引列表放在一个方括号内传递给数组。

## 二维数组的花式索引

![](img/c9613ea391fe539a6147bf33c31cd8e6_7.png)

对于二维数组，花式索引的规则同样适用。我们可以通过逗号分隔行和列的索引方式。

以下是几种索引方式在二维数组中的组合应用示例：

```python
arr = np.arange(20).reshape(4, 5)  # 创建一个4行5列的二维数组
print(arr)
```

### 组合索引示例

1.  **常规索引与切片组合**：选取第0行，第2到第4列。
    ```python
    print(arr[0, 2:4])
    ```

2.  **常规索引与布尔型索引组合**：选取第0行中所有大于2的元素。
    ```python
    print(arr[0, arr[0] > 2])
    ```

## 花式索引的注意事项与高级用法

一个重要的限制是：**逗号两边不能同时使用花式索引列表**，否则会产生意想不到的结果。这种写法会被解释为选取坐标 `(i, j)` 和 `(m, n)` 位置上的值，而不是分别按行和列选取。

例如，想选取第1行和第3行，以及第1列和第3列交叉的四个值（即 `[6, 8, 16, 18]`）。

**错误写法**：
```python
print(arr[[1, 3], [1, 3]])  # 输出：[6, 18]，它选取的是(1,1)和(3,3)两个点
```

为了实现我们的目标，需要分步操作：

以下是分步选取特定行和列的方法：

1.  首先，使用花式索引选取指定的行（第1行和第3行），并保留所有列。
    ```python
    step1 = arr[[1, 3], :]
    print(step1)
    ```
2.  然后，在上一步的结果上，选取指定的列（第1列和第3列）。
    ```python
    final_result = step1[:, [1, 3]]
    print(final_result)
    ```
    也可以合并为一行：
    ```python
    final_result = arr[[1, 3], :][:, [1, 3]]
    print(final_result)
    ```

通过这种分步索引的方式，我们可以灵活地组合各种索引方法，实现对数组元素的复杂选取。
