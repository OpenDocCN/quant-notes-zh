## 什么是金叉与死叉？🤔

![](img/11f277c72fefa0401719ecce4462e177_15.png)

![](img/11f277c72fefa0401719ecce4462e177_17.png)

在股票技术分析中，金叉和死叉是分析指标中两根线交叉的术语。在我们的图表中，这两根线分别是短期均线（MA5）和长期均线（MA30）。

![](img/11f277c72fefa0401719ecce4462e177_19.png)

以下是金叉和死叉的简单定义：
*   **金叉**：当短期均线（MA5）从下方**上穿**长期均线（MA30）时，其交点附近的位置称为金叉。
*   **死叉**：当短期均线（MA5）从上方**下穿**长期均线（MA30）时，其交点附近的位置称为死叉。

为了更直观地理解，请看下图示意：
```
价格(Y轴)
  ^
  |    MA5 (短期)  ---
  |                 /\
  | MA30 (长期) ---/  \----
  |               /    \    \
  |              /      \    \
  |             /        \    \
  |            /          \    \
  |           / (金叉)     \ (死叉)
  |          /              \
  |         /                \
  +-------------------------------> 时间(X轴)
```
在上图中，短期均线（MA5）与长期均线（MA30）有三个交点。根据定义，第一个和第三个交点是**金叉**，第二个交点是**死叉**。

## 金叉与死叉的交易意义 💰

![](img/11f277c72fefa0401719ecce4462e177_21.png)

![](img/11f277c72fefa0401719ecce4462e177_23.png)

![](img/11f277c72fefa0401719ecce4462e177_25.png)

![](img/11f277c72fefa0401719ecce4462e177_27.png)

金叉和死叉是重要的买卖信号：
*   **金叉**通常被视为**买入信号**，预示着股价可能开始上涨趋势。
*   **死叉**通常被视为**卖出信号**，预示着股价可能开始下跌趋势。

![](img/11f277c72fefa0401719ecce4462e177_29.png)

因此，在双均线策略中，我们会在出现金叉时买入股票，在出现死叉时卖出股票，以期获得收益。

## 如何通过代码找出金叉与死叉日期？🔍

我们的目标是找出图表中所有金叉和死叉点对应的X轴坐标，即它们发生的日期。

观察金叉和死叉点附近均线值（Y轴）的大小关系，我们可以发现一个规律：
*   在**金叉点左侧**，`MA5 < MA30`。
*   在**金叉点右侧**，`MA5 > MA30`。
*   在**死叉点左侧**，`MA5 > MA30`。
*   在**死叉点右侧**，`MA5 < MA30`。

![](img/11f277c72fefa0401719ecce4462e177_31.png)

因此，在交叉点本身，可以近似认为 `MA5 == MA30`。我们可以通过分析 `MA5` 和 `MA30` 序列的大小关系变化来定位这些交叉点。

![](img/11f277c72fefa0401719ecce4462e177_33.png)

![](img/11f277c72fefa0401719ecce4462e177_35.png)

以下是具体的实现步骤：

首先，我们需要确保用于比较的 `MA5` 和 `MA30` 数据长度一致且没有空值。由于 `MA30` 的前29个值为空，我们统一从第30个数据开始切片。

```python
# 统一切片，从第30个数据开始，保证索引一致
ma5 = df[‘ma5’][30:]
ma30 = df[‘ma30’][30:]
df = df[30:] # 原始数据也同步切片
```

接着，我们创建两个布尔序列，分别代表 `MA5 < MA30` 和 `MA5 > MA30` 的条件。

![](img/11f277c72fefa0401719ecce4462e177_37.png)

![](img/11f277c72fefa0401719ecce4462e177_39.png)

```python
# 条件序列：MA5 是否小于 MA30
s1 = ma5 < ma30
# 条件序列：MA5 是否大于 MA30
s2 = ma5 > ma30
```

`s1` 序列中的 `True` 表示 `MA5 < MA30`，`False` 表示 `MA5 > MA30`。`s2` 序列则恰好相反。

通过观察和推导（详细逻辑见课程讲解），我们可以利用 `s1`、`s2` 及其移位来构造判定条件：

**1. 找出死叉日期**

![](img/11f277c72fefa0401719ecce4462e177_41.png)

死叉的特征是 `s1` 序列的值从 `False` 过渡到 `True`（即 `MA5` 从大于 `MA30` 变为小于 `MA30`）。我们可以通过以下条件定位：

```python
# 死叉判定条件：s1 为 True，且 s2 的前一个值为 True (即 s1 的前一个值为 False)
death_cross_condition = s1 & s2.shift(1)
# 获取死叉日期
death_dates = df.loc[death_cross_condition].index
```

**2. 找出金叉日期**

![](img/11f277c72fefa0401719ecce4462e177_43.png)

金叉的特征是 `s1` 序列的值从 `True` 过渡到 `False`（即 `MA5` 从小于 `MA30` 变为大于 `MA30`）。我们可以通过以下条件定位：

![](img/11f277c72fefa0401719ecce4462e177_45.png)

```python
# 金叉判定条件：s1 为 False，且 s2 的前一个值为 False (即 s1 的前一个值为 True)
# 等价于对 (s1 | s2.shift(1)) 的结果取反
golden_cross_condition = ~(s1 | s2.shift(1))
# 获取金叉日期
golden_dates = df.loc[golden_cross_condition].index
```
