# Python金融分析与量化交易实战课程：P57：参数更新方法

在本节课中，我们将要学习梯度下降算法中核心的参数更新方法。我们将详细解释如何分别对模型中的不同参数进行优化，并推导出具体的参数更新公式。

## 参数更新的核心思想

上一节我们介绍了梯度下降的基本概念，本节中我们来看看如何具体更新参数。

在优化目标函数以找到最优参数（如 θ₀ 和 θ₁）的过程中，需要明确一点：参数 θ₀ 和 θ₁ 是分别进行优化的，而不是一起优化。这是因为 θ₀ 对应特征 X₀，θ₁ 对应特征 X₁。数据样本之间是独立的，特征 X₀ 和 X₁ 之间没有关系，因此它们对应的参数也互不影响。

在优化过程中，需要分别寻找 θ₀ 和 θ₁ 各自合适的前进方向。可以想象在一个三维空间中，θ₀ 和 θ₁ 各代表一个坐标轴。优化时，先为 θ₀ 找到一个方向，再为 θ₁ 找到一个方向，两者共同前进，最终达到综合最优的结果。

具体操作是：首先用损失函数 J 对 θ₀ 求偏导，得到 θ₀ 的更新方向；再用 J 对 θ₁ 求偏导，得到 θ₁ 的更新方向。两个参数独立更新，互不干扰。

## 梯度下降的步骤

寻找山谷最低点（即最小化损失函数）的过程可以概括为以下三步，这是一个反复迭代的过程：

1.  **求偏导**：计算损失函数对各个参数的偏导数，以确定下降方向。
2.  **走一小步**：沿着偏导确定的**反方向**（即梯度下降方向）前进一小步。步长必须小，步长过大可能导致不收敛或收敛效果差，这被称为“跌倒”。
3.  **更新参数**：根据计算出的方向和步长，更新参数值，得到新的参数点。然后回到第一步，用新的参数重新计算，直到满足停止条件。

## 数学描述与公式推导

接下来，我们看看如何在数学上描述这个“下山”任务。

首先，回顾之前定义的损失函数。对于单个样本，其损失为：
`loss_i = 1/2 * (y_i - h_θ(x_i))^2`
其中 `h_θ(x_i)` 是模型的预测值。

然而，评估一个模型的好坏不能只看一个样本，而应看其在所有训练样本上的平均表现。因此，整体的损失函数（成本函数）J(θ) 定义为所有样本损失的平均值，并加上平方项以放大误差：
`J(θ) = 1/(2m) * Σ_{i=1}^{m} (y_i - h_θ(x_i))^2`
这里，`m` 是样本总数，`1/2` 是为了后续求导方便而引入的系数。

假设我们的模型是线性模型：`h_θ(x_i) = θ₀ + θ₁x_i1 + θ₂x_i2 + ... + θ_j x_ij + ...`
现在，我们需要对某个特定参数 θ_j 求偏导以进行更新。

以下是求偏导的过程：
`∂J(θ)/∂θ_j = ∂/∂θ_j [ 1/(2m) * Σ_{i=1}^{m} (y_i - h_θ(x_i))^2 ]`

由于求和与求导是线性操作，可以交换顺序：
`= 1/(2m) * Σ_{i=1}^{m} ∂/∂θ_j [ (y_i - h_θ(x_i))^2 ]`

应用链式法则，令 `u = y_i - h_θ(x_i)`，则 `∂u^2/∂θ_j = 2u * ∂u/∂θ_j`。
其中，`∂u/∂θ_j = ∂/∂θ_j (y_i - h_θ(x_i)) = - ∂h_θ(x_i)/∂θ_j`。

因为 `h_θ(x_i)` 中只有包含 θ_j 的项 `θ_j * x_ij` 对 θ_j 求导不为零，所以 `∂h_θ(x_i)/∂θ_j = x_ij`。
因此，`∂u/∂θ_j = -x_ij`。

将各部分组合：
`∂/∂θ_j [ (y_i - h_θ(x_i))^2 ] = 2 * (y_i - h_θ(x_i)) * (-x_ij) = -2 * (y_i - h_θ(x_i)) * x_ij`

代回求和式：
`∂J(θ)/∂θ_j = 1/(2m) * Σ_{i=1}^{m} [ -2 * (y_i - h_θ(x_i)) * x_ij ]`
化简后得到：
`∂J(θ)/∂θ_j = -1/m * Σ_{i=1}^{m} [ (y_i - h_θ(x_i)) * x_ij ]`

这个结果 `∂J(θ)/∂θ_j` 就是损失函数 J 在 θ_j 方向上的**梯度**。梯度方向是函数值上升最快的方向。为了最小化损失函数，我们需要沿着梯度的**反方向**前进。因此，在更新参数时，需要取负号。

## 参数更新公式

最终，参数 θ_j 的更新公式如下：
`θ_j := θ_j - α * ∂J(θ)/∂θ_j`

将上面求得的偏导数代入：
`θ_j := θ_j - α * [ -1/m * Σ_{i=1}^{m} (y_i - h_θ(x_i)) * x_ij ]`
`θ_j := θ_j + α * 1/m * Σ_{i=1}^{m} (y_i - h_θ(x_i)) * x_ij`

其中：
*   `:=` 表示赋值（更新）。
*   `α` 是**学习率**，即“步长”，控制每次参数更新的幅度。
*   `1/m * Σ_{i=1}^{m} (y_i - h_θ(x_i)) * x_ij` 是梯度的负方向（负负得正后）。
*   `x_ij` 表示第 `i` 个样本的第 `j` 个特征值。

这个公式就是梯度下降算法中**核心的参数更新规则**。对于每一个参数 θ_j，都按照此规则，利用所有样本计算出的平均梯度进行更新。

## 总结

本节课中我们一起学习了梯度下降算法的参数更新方法。核心要点包括：
1.  模型参数（如 θ₀, θ₁）需要**分别独立进行优化**。
2.  梯度下降的步骤是：求偏导（得方向）、沿反方向走一小步、更新参数，并迭代进行。
3.  通过对整体损失函数求偏导，我们推导出了关键的参数更新公式：**`θ_j := θ_j + α * 1/m * Σ (预测误差 * 特征值)`**。
4.  公式中的学习率 `α` 至关重要，它决定了每次更新的步长。
