## 增长率累加的问题

上一节我们介绍了如何计算简单的差异值和增长率。本节中我们来看看，当我们想观察一个连续时间段内的整体增长时，能否将每天的增长率简单相加。

![](img/6ef49eaa65122dec4a7802297da32eb4_4.png)

例如，假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

分别计算增长率：
*   第二天增长率 = (50 - 100) / 100 = -0.5
*   第三天增长率 = (75 - 50) / 50 = 0.5

如果将这两个增长率相加：-0.5 + 0.5 = 0。这个结果为0，似乎意味着股票从第一天到第三天既没涨也没跌。

![](img/6ef49eaa65122dec4a7802297da32eb4_6.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_8.png)

但观察原始数据，股价从100元变为75元，实际上是下跌的。这说明，**增长率不能直接进行加法运算**，因为它无法正确反映从起点到终点的整体变化。

![](img/6ef49eaa65122dec4a7802297da32eb4_10.png)

## 解决方案：引入对数变换

对于连续时间的计算，我们需要借助数学中的对数公式。核心思想是：对数的加法等价于其内部数值的乘法。

![](img/6ef49eaa65122dec4a7802297da32eb4_12.png)

具体做法是，计算连续的对数收益率：
`log_return = log(今日价格 / 昨日价格)`

![](img/6ef49eaa65122dec4a7802297da32eb4_14.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_16.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_18.png)

为什么这样做有效？因为当我们累加这些对数收益率时：
`log(P1/P0) + log(P2/P1) + ... + log(Pn/Pn-1) = log(Pn/P0)`

![](img/6ef49eaa65122dec4a7802297da32eb4_20.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_22.png)

最终，累加和等价于**最后一天价格与第一天价格比值的对数**。这完美地描述了从起点到终点的整体差异。

![](img/6ef49eaa65122dec4a7802297da32eb4_24.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_26.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_28.png)

形象地说，这解决了“如果初始投资1元钱，经过一系列涨跌后，最终相当于多少钱”的问题。

![](img/6ef49eaa65122dec4a7802297da32eb4_30.png)

## 代码实现：计算对数收益率

![](img/6ef49eaa65122dec4a7802297da32eb4_32.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_34.png)

以下是计算连续对数收益率的步骤：

![](img/6ef49eaa65122dec4a7802297da32eb4_36.png)

首先，我们需要获取“前一天”的数据。Pandas的`.shift()`函数可以将数据整体向上或向下移动。
```python
# 假设 data 是一个包含股价的Pandas Series或DataFrame
# 将数据向上移动一格，得到“前一天”的价格
data_shifted = data.shift(1)
```

![](img/6ef49eaa65122dec4a7802297da32eb4_38.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_40.png)

接着，计算每日的对数收益率：
```python
# 计算对数收益率：log(今日价格 / 昨日价格)
log_returns = np.log(data / data_shifted)
# 查看结果（保留几位小数）
print(log_returns.head().round(4))
```

![](img/6ef49eaa65122dec4a7802297da32eb4_42.png)

## 还原为累积收益

![](img/6ef49eaa65122dec4a7802297da32eb4_44.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_46.png)

我们计算了对数收益率，但最终想看到的是直观的“1元变多少元”的结果。这就需要将对数收益率累加后，再通过指数函数还原。

![](img/6ef49eaa65122dec4a7802297da32eb4_48.png)

使用`np.cumsum()`函数进行累加：
```python
# 对对数收益率进行累积求和
cumulative_log_returns = log_returns.cumsum()
```

![](img/6ef49eaa65122dec4a7802297da32eb4_50.png)

然后，使用指数函数`np.exp()`将累积对数收益率还原为实际的累积收益倍数：
```python
# 还原为累积收益倍数：初始1单位对应的价值
cumulative_returns = np.exp(cumulative_log_returns)
```

![](img/6ef49eaa65122dec4a7802297da32eb4_52.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_54.png)

最后，我们可以绘制图表，直观展示每只股票从起始点到每个时间点的累积收益情况：
```python
# 绘制累积收益曲线
cumulative_returns.plot(figsize=(16, 8))
plt.show()
```
图表会显示，例如亚马逊(AMZN)的股票，2010年的1个单位价值，到2018年可能增长到了约10个单位，反映了其巨大的增长。

![](img/6ef49eaa65122dec4a7802297da32eb4_56.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_58.png)

## 本章内容总结

本节课中我们一起学习了金融时间序列中连续指标的分析方法：

![](img/6ef49eaa65122dec4a7802297da32eb4_60.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_62.png)

![](img/6ef49eaa65122dec4a7802297da32eb4_64.png)

1.  **差异值与简单增长率**：使用`.diff()`计算差值，以及`(后值-前值)/前值`计算增长率。
2.  **增长率累加的局限性**：通过实例证明了简单累加增长率无法反映资产价值的真实整体变化。
3.  **对数收益率的引入**：使用公式 **`log_return = log(P_t / P_{t-1})`** 来计算连续收益率。其累加和 **`sum(log_return) = log(P_end / P_start)`** 能准确反映起点到终点的总变化。
4.  **累积收益计算**：通过对数收益率累加(`np.cumsum`)后指数还原(`np.exp`)，得到“初始1单位对应的价值随时间变化”的曲线，这是一个在金融分析中非常实用的指标。
