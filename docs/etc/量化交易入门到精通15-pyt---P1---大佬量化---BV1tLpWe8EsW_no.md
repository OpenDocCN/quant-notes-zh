# 量化交易入门到精通：15：Python基础语法之多行语句 📝

在本节课中，我们将要学习Python中的多行语句。当一行代码过长，导致阅读和维护困难时，我们可以使用特定的方法将其拆分成多行，使代码结构更清晰、更易读。

![](img/06ddb7363581cc42184e4d19e3d2c01d_1.png)

## 多行语句的必要性

上一节我们介绍了Python的基本语法结构。本节中我们来看看如何处理过长的代码行。在编写代码时，有时会遇到一行代码特别长的情况，例如定义一个包含很多元素的列表。这会导致代码在编辑器或记事本中超出屏幕显示范围，必须拖动滚动条才能查看完整内容，使得代码难以阅读和理解。

![](img/06ddb7363581cc42184e4d19e3d2c01d_3.png)

## 使用反斜杠实现多行语句

![](img/06ddb7363581cc42184e4d19e3d2c01d_5.png)

为了解决长代码行的问题，Python允许使用反斜杠（`\`）将一行语句拆分成多行。反斜杠位于键盘上回车键附近，通常与竖线（`|`）或反斜杠键共享。

以下是使用反斜杠拆分长语句的步骤：
1.  在需要换行的位置输入反斜杠（`\`）。
2.  在反斜杠后直接回车，将后续内容写到新的一行。
3.  Python解释器会将使用反斜杠连接的多行视为同一行代码。

![](img/06ddb7363581cc42184e4d19e3d2c01d_7.png)

让我们通过一个例子来演示。假设我们有一个非常长的股票代码列表。

![](img/06ddb7363581cc42184e4d19e3d2c01d_9.png)

```python
# 未使用多行语句的冗长代码
stock_codes = ['AAPL', 'GOOGL', 'MSFT', 'AMZN', 'TSLA', 'FB', 'NVDA', 'PYPL', 'ADBE', 'NFLX', 'INTC', 'CSCO', 'PEP', 'CMCSA', 'AVGO']

# 使用反斜杠拆分为多行的清晰代码
stock_codes = ['AAPL', 'GOOGL', 'MSFT', 'AMZN', 'TSLA', \
               'FB', 'NVDA', 'PYPL', 'ADBE', 'NFLX', \
               'INTC', 'CSCO', 'PEP', 'CMCSA', 'AVGO']
```

![](img/06ddb7363581cc42184e4d19e3d2c01d_11.png)

运行上述两段代码，输出的 `stock_codes` 列表内容完全相同。然而，从视觉上看，第二段代码的结构更加清晰，每个元素都更容易定位，大大提升了代码的可读性。

![](img/06ddb7363581cc42184e4d19e3d2c01d_13.png)

## 在括号内直接换行

![](img/06ddb7363581cc42184e4d19e3d2c01d_15.png)

除了使用反斜杠，Python还有一个更简洁的特性：在括号（包括圆括号`()`、方括号`[]`、花括号`{}`）内部可以直接换行，而无需使用反斜杠。这是因为Python能够自动识别未闭合的括号，并将括号内的多行内容视为一个整体。

![](img/06ddb7363581cc42184e4d19e3d2c01d_17.png)

以下是利用括号特性实现多行语句的方法：

```python
# 在列表、元组或字典的括号内直接换行
stock_codes = [
    'AAPL', 'GOOGL', 'MSFT', 'AMZN', 'TSLA',
    'FB', 'NVDA', 'PYPL', 'ADBE', 'NFLX',
    'INTC', 'CSCO', 'PEP', 'CMCSA', 'AVGO'
]

![](img/06ddb7363581cc42184e4d19e3d2c01d_19.png)

# 函数调用参数过多时也可直接换行
result = some_long_function_name(argument_one, argument_two,
                                 argument_three, argument_four)
```

![](img/06ddb7363581cc42184e4d19e3d2c01d_21.png)

这种方法比使用反斜杠更常见，也更为推荐，因为它更清晰，且不容易因遗漏反斜杠而导致语法错误。

![](img/06ddb7363581cc42184e4d19e3d2c01d_23.png)

## 总结

本节课中我们一起学习了Python中处理多行语句的两种主要方法。第一种是使用反斜杠（`\`）进行显式换行连接；第二种是利用括号（`()`, `[]`, `{}`）的隐式换行规则，直接在括号内换行。掌握多行语句的编写技巧，能够帮助我们保持代码的整洁与优雅，是编写可维护性高代码的重要基础。在后续编写复杂的量化交易策略时，这一技能将非常有用。
