## 时间序列的构成
上一节我们介绍了Pandas中各种时间对象的创建方法。本节中我们来看看如何利用这些时间对象构成时间序列。

![](img/1cad4c6aa50dc70b40d2aa48d68f682a_3.png)

所谓时间序列，就是以时间对象作为索引的Series或DataFrame。Pandas的`date_range`函数生成的索引类型是`DatetimeIndex`，我们可以用它来作为Series或DataFrame的索引。

![](img/1cad4c6aa50dc70b40d2aa48d68f682a_5.png)

例如，我们可以创建一个以日期为索引的Series对象：
```python
import pandas as pd
import numpy as np

![](img/1cad4c6aa50dc70b40d2aa48d68f682a_7.png)

![](img/1cad4c6aa50dc70b40d2aa48d68f682a_9.png)

# 创建一个长度为100的数值数组作为数据
data = np.arange(100)
# 创建一个从2017-01-01开始，长度为100天的日期索引
date_index = pd.date_range('2017-01-01', periods=100)
# 用数据和日期索引创建Series
sr = pd.Series(data, index=date_index)
```
执行`sr.index`可以看到，索引的类型是`DatetimeIndex`，而不仅仅是字符串。这就构成了一个基础的时间序列。

![](img/1cad4c6aa50dc70b40d2aa48d68f682a_11.png)

## 时间序列的切片操作
成为时间序列后，一个直观的好处是我们可以方便地按时间范围选取数据。

以下是几种常见的切片方式：

*   **按年月切片**：我们可以使用不完整的日期字符串进行切片。例如，`sr[‘2017-03’]`会选取2017年3月份的所有数据。
*   **按年切片**：只写年份，如`sr[‘2017’]`，会选取该年份的所有数据。
*   **按日期范围切片**：可以传入一个精确的日期范围字符串。例如，`sr[‘2017-12-25’:‘2018-02-01’]`会选取从2017年12月25日到2018年2月1日（包含首尾）的数据。

这些操作使得按时间维度筛选数据变得异常简单和直观。

---

## 时间序列的重采样
除了切片，时间序列还支持强大的重采样功能，这通过`resample`函数实现。

`resample`意为重新采样或重新取样。例如，我们有一个按天记录的数据序列，但想查看每周或每月的汇总情况。

![](img/1cad4c6aa50dc70b40d2aa48d68f682a_13.png)

以下是`resample`的用法示例：
```python
# 按周进行重采样，并计算每周的和
weekly_sum = sr.resample(‘W’).sum()
# 按月进行重采样，并计算每月的平均值
monthly_mean = sr.resample(‘M’).mean()
```
在第一个例子中，参数`’W’`表示按周频率重采样，`.sum()`表示对每周内的数据求和。结果会生成以每周起始日期为索引的新序列，值为该周数据的汇总。同理，`’M’`表示按月，`.mean()`计算月度平均值。这个功能对于时间序列的降采样分析非常方便。

## 其他相关函数
Pandas时间序列还有其他一些函数，例如`truncate`。

`truncate`函数可以根据日期截断序列。它接受`before`和`after`参数，用于保留指定日期之后或之前的数据。
```python
# 保留2018年2月3日之后的数据
sr_truncated_after = sr.truncate(after=‘2018-02-03’)
# 保留2018年2月3日之前的数据
sr_truncated_before = sr.truncate(before=‘2018-02-03’)
```
不过，在实际应用中，直接使用之前介绍的切片操作通常更直观和灵活，例如`sr[:’2018-02-03’]`可以实现类似`truncate(after=…)`的效果。
