## 执行交易逻辑

![](img/f1e109a46299e47b5c9a623cbcbc0db1_50.png)

![](img/f1e109a46299e47b5c9a623cbcbc0db1_52.png)

接下来，我们进入 `handle_bar` 函数，在这里执行实际的交易判断与操作。

![](img/f1e109a46299e47b5c9a623cbcbc0db1_54.png)

首先，我们需要判断当前账户中是否持有股票。这可以通过查询 `context` 对象中的 `portfolio.positions` 属性来实现，它是一个包含所有持仓信息的字典。

![](img/f1e109a46299e47b5c9a623cbcbc0db1_56.png)

![](img/f1e109a46299e47b5c9a623cbcbc0db1_58.png)

以下是判断和交易的核心步骤：

![](img/f1e109a46299e47b5c9a623cbcbc0db1_60.png)

![](img/f1e109a46299e47b5c9a623cbcbc0db1_62.png)

![](img/f1e109a46299e47b5c9a623cbcbc0db1_64.png)

1.  **判断持仓状态**：检查 `context.portfolio.positions` 字典是否为空。
2.  **执行卖出操作**：如果持仓不为空，则遍历当前持仓。对于持仓中不在最新股票池里的股票，执行卖出操作（即将其仓位比例调整为0%）。
3.  **执行买入操作**：遍历最新的股票池。对于股票池中当前未持有的股票，执行买入操作。为了简化，我们采用平均买入的策略，即每只股票分配相同的资金比例。

![](img/f1e109a46299e47b5c9a623cbcbc0db1_66.png)

![](img/f1e109a46299e47b5c9a623cbcbc0db1_68.png)

以下是实现上述逻辑的代码框架：

![](img/f1e109a46299e47b5c9a623cbcbc0db1_70.png)

```python
def handle_bar(context):
    # 获取当前持仓和最新股票池
    current_positions = context.portfolio.positions
    target_stocks = context.selected_stocks  # 假设已在before_trading中存入

    # 如果当前有持仓，则检查是否需要卖出
    if len(current_positions) > 0:
        for stock in current_positions:
            if stock not in target_stocks:
                # 卖出不在目标池中的股票
                order_target_percent(stock, 0)

    # 买入目标股票池中的股票（平均分配资金）
    for stock in target_stocks:
        # 计算每只股票应占的资金比例
        weight = 1.0 / len(target_stocks)
        order_target_percent(stock, weight)
```

## 策略回测结果分析

![](img/f1e109a46299e47b5c9a623cbcbc0db1_74.png)

![](img/f1e109a46299e47b5c9a623cbcbc0db1_76.png)

运行策略后，我们会得到一份回测报告。报告主要包含两部分：收益概况和交易详情。

![](img/f1e109a46299e47b5c9a623cbcbc0db1_78.png)

![](img/f1e109a46299e47b5c9a623cbcbc0db1_80.png)

**收益概况**中的关键指标包括：
*   **回测年化收益**：你的策略在整个回测期间的年化收益率。
*   **基准年化收益**：同期市场指数（如沪深300）的年化收益率。通过对比两者，可以判断策略是否跑赢大盘。
*   **最大回撤**：策略净值从最高点到最低点的最大跌幅，用于衡量策略的风险。
*   **夏普比率**：衡量策略承担单位风险所获得的超额回报。在收益为负时，该指标通常也为负。

![](img/f1e109a46299e47b5c9a623cbcbc0db1_82.png)

![](img/f1e109a46299e47b5c9a623cbcbc0db1_84.png)

**交易详情**则记录了每一次买卖操作的具体信息，如时间、股票、价格、数量等，用于进行更细致的分析。

![](img/f1e109a46299e47b5c9a623cbcbc0db1_86.png)

在本示例的回测中，策略和基准都出现了亏损，但策略的亏损幅度小于基准，且最大回撤控制在17%左右，这表明策略在一定程度上起到了防御作用。当然，策略的表现与回测的时间段选择密切相关。

![](img/f1e109a46299e47b5c9a623cbcbc0db1_88.png)
