# 量化交易入门：1：什么是量化交易策略？🤔

![](img/5fdcf3b240d5710dbbbd6a77d0584706_1.png)

在本节课中，我们将要学习量化交易策略的基本概念，并揭示一个你可能早已接触过，却不自知的量化策略实例。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_3.png)

## 概述

量化投资策略并非由人脑主观决定交易的品种、方向、时机和数量，而是依据一套明确的规则或模型来执行交易。这样形成的策略就是一个量化投资策略。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_5.png)

很多人认为量化投资非常复杂，难以学会。但事实上，超过90%的交易者可能早已交易过量化策略。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_7.png)

## 指数：一个隐藏的量化策略 📈

![](img/5fdcf3b240d5710dbbbd6a77d0584706_9.png)

你相信吗？绝大部分市场指数，其本质上就可以看作一个量化策略。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_11.png)

### 以中证红利指数为例

让我们来看一个简单的例子：今年备受关注的红利指数。我们以其中具有代表性的中证红利指数为例。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_13.png)

以下是其选股规则的量化解读：

![](img/5fdcf3b240d5710dbbbd6a77d0584706_15.png)

1.  **筛选条件**：首先，筛选出过去3年连续进行现金分红的股票。
2.  **财务指标**：同时，这些股票过去3年的平均股利支付率，以及过去一年的股利支付率，必须大于0且小于1。
3.  **排序与选择**：在满足上述条件的股票池中，选取过去3年平均现金股息率最高的100只股票。
4.  **持有与调仓**：买入这些股票并持有。在固定的时间点（如每半年），卖出不再符合条件的股票，并重新买入此时符合条件的股票。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_17.png)

这个过程完全由清晰的规则驱动，没有主观判断的介入。

## 量化策略的普遍逻辑 🔄

![](img/5fdcf3b240d5710dbbbd6a77d0584706_19.png)

上一节我们剖析了红利指数的构建，可以看到其核心是一套选股与调仓规则。实际上，我们量化投资中的很多选股策略，本质上和它是同一个逻辑。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_21.png)

例如，经典的小市值策略也是如此。它使用“市值”这个指标来筛选股票，并按照模型设定的规则（如定期调仓、轮动）进行买入和卖出，仅此而已。

再看其他的红利指数，虽然具体的选股要求可能不同，但本质上都是“选出高分红股票”这一量化策略的变形。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_23.png)

## 更广泛的例子：主流指数 🌐

![](img/5fdcf3b240d5710dbbbd6a77d0584706_25.png)

甚至我们最常用的上证指数、沪深300指数，也都是按照固定规则选股（如上交所全部上市股票、沪深两市市值和流动性排名前300的股票），并定期调整成分股。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_27.png)

这其实就是由国家或指数公司制定好的量化选股策略。只不过，某些策略（如上证指数）的表现可能长期在某个点位附近震荡，表现不尽如人意。

当我们交易指数基金或ETF时，在本质上其实就是在交易一个已经封装好的量化策略。这个策略的股票组合，已经由指数编制方“帮我们买好了”。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_29.png)

## 总结

本节课中，我们一起学习了量化交易策略的核心定义：**依据明确规则而非主观判断进行交易**。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_31.png)

我们通过深入分析中证红利、沪深300等常见指数，揭示了它们本质上就是量化策略。这说明量化交易并不神秘或高深莫测，你可能早已在不知不觉中接触并交易过它。

![](img/5fdcf3b240d5710dbbbd6a77d0584706_33.png)

理解这一点，是打开量化交易大门的第一步。它让我们明白，构建自己的量化策略，其实就是将有效的投资逻辑转化为清晰、可执行的规则。
