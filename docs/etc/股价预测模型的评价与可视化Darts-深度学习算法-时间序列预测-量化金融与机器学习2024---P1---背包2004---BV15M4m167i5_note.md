## 回顾上节课内容

![](img/1614a27eeab6a4f18e5814cdbdd91628_1.png)

上一节我们介绍了使用Darts库进行时间序列预测的基本流程。这个流程的核心步骤如下：

1.  **获取数据**：数据格式为Pandas DataFrame。
2.  **提取序列**：从数据中提取目标股票的价格序列。
3.  **构建时间序列**：这是最关键的一步，需要将DataFrame转换为Darts可用的`TimeSeries`对象。有两个重要要求：
    *   时间列的数据类型必须是`datetime`或时间戳。
    *   时间索引必须是连续的，不能有缺失。由于股票市场存在非交易日，原始数据中会有空缺，因此需要先进行填充处理，确保时间连续性。
4.  **标准化**：对序列进行缩放，例如将所有值除以序列的最大值，将其转换到0-1的范围。公式可以表示为：`scaled_value = original_value / max(series)`。
5.  **分割数据集**：将序列划分为训练集和测试集。
6.  **实例化与训练模型**：选择模型（如深度学习模型），实例化并拟合训练数据。

## 模型评价：平均绝对百分比误差

![](img/1614a27eeab6a4f18e5814cdbdd91628_11.png)

当我们训练好模型后，需要评估其预测性能。本节中我们来看看最常用的评价指标之一：平均绝对百分比误差。

![](img/1614a27eeab6a4f18e5814cdbdd91628_13.png)

![](img/1614a27eeab6a4f18e5814cdbdd91628_15.png)

![](img/1614a27eeab6a4f18e5814cdbdd91628_17.png)

![](img/1614a27eeab6a4f18e5814cdbdd91628_19.png)

MAPE衡量的是预测值与真实值之间绝对误差的平均百分比。其计算公式为：
`MAPE = (1/n) * Σ(|(实际值 - 预测值)| / |实际值|) * 100%`
其中，`n`是预测点的数量。

![](img/1614a27eeab6a4f18e5814cdbdd91628_21.png)

![](img/1614a27eeab6a4f18e5814cdbdd91628_23.png)

以下是使用Darts计算MAPE的步骤：

![](img/1614a27eeab6a4f18e5814cdbdd91628_25.png)

首先，使用训练好的模型进行预测。预测需要指定一个起点（通常是训练集的末尾）和要预测的未来步数（`n`）。
```python
# 假设 model 是已训练好的模型，series 是完整的时间序列
prediction = model.predict(n=12)
```
然后，将预测值与测试集的真实值进行比较，计算MAPE。
```python
from darts.metrics import mape
mape_value = mape(series_test, prediction)
print(f"平均绝对百分比误差为: {mape_value:.2%}")
```
**注意**：模型在训练时定义了输入长度（如24）和输出长度（如12）。当预测步数`n`小于或等于输出长度时，预测相对可靠。如果`n`大于输出长度，模型将使用自己之前的预测值进行自回归预测，误差可能会累积并逐渐增大。

### 1. 简单预测可视化

![](img/1614a27eeab6a4f18e5814cdbdd91628_45.png)

这是最直接的绘图方式，展示从某个起点开始的多步预测。
```python
# 绘制预测结果与真实序列
series.plot(label="真实值")
prediction.plot(label="预测值")
```
这种图显示的是模型基于起点数据进行的一次性多步预测。如果预测步数较长，后半部分基于自回归的结果可能会逐渐偏离真实趋势。

### 2. 历史模拟与滚动预测可视化

上一节我们介绍了简单预测，本节中我们来看看一种更可靠的评估方式：历史模拟。

这种方法在历史数据上进行“滚动预测”，每一步都使用截至该点的真实数据来预测未来几步，从而更真实地模拟模型在历史中的表现。
```python
from darts.models import SomeModel
# 进行历史模拟预测，预测未来3步，从序列60%的位置开始，并且不重新训练模型以节省时间
historical_fcast = model.historical_forecasts(
    series=series,
    forecast_horizon=3,
    start=0.6,
    retrain=False,
    verbose=True
)
# 计算这种滚动预测的MAPE
mape_historical = mape(series, historical_fcast)
```
**关键参数解释**：
*   `forecast_horizon`：每次预测的未来步数。
*   `start`：开始进行滚动预测的起始位置（占序列长度的比例）。
*   `retrain`：是否在每一步都重新训练模型。在真实应用中，为了获得最新信息，应设置为`True`。但考虑到深度学习模型训练耗时，演示时可设为`False`以提高效率。

![](img/1614a27eeab6a4f18e5814cdbdd91628_51.png)

![](img/1614a27eeab6a4f18e5814cdbdd91628_53.png)

这种可视化得到的预测曲线与真实曲线贴合度通常更高，其对应的MAPE值也更能反映模型在“模拟实战”中的性能，通常比一次性多步预测的误差要小。

![](img/1614a27eeab6a4f18e5814cdbdd91628_55.png)

### 3. 概率预测可视化

![](img/1614a27eeab6a4f18e5814cdbdd91628_57.png)

某些模型（如指数平滑模型`ExponentialSmoothing`）支持概率预测，它不仅能给出一个预测值点，还能给出未来值可能的分布区间（例如，90%的置信区间）。
```python
from darts.models import ExponentialSmoothing
prob_model = ExponentialSmoothing()
prob_model.fit(train_series)
# 进行概率预测
prob_forecast = prob_model.predict(n=10, num_samples=500)
# 绘制预测结果，包括置信区间
series.plot(label="真实值")
prob_forecast.plot(label="预测值", low_quantile=0.05, high_quantile=0.95)
```
对于其他模型（如之前使用的深度学习模型），可能需要通过设置`likelihood`参数来启用概率预测功能。概率预测图通过阴影区域展示了预测的不确定性，对于风险评估和制定策略更有帮助。

本节课中我们一起学习了如何评价和可视化时间序列预测模型。
*   我们回顾了使用Darts进行预测的标准流程，重点是构建连续的时间序列。
*   我们学习了使用**平均绝对百分比误差**来量化模型预测精度，并理解了预测步长对误差的影响。
*   我们掌握了三种可视化方法：
    1.  **简单预测图**：展示一次性多步预测结果。
    2.  **历史模拟滚动预测图**：更真实地评估模型在历史数据上的表现，通常伴随更低的误差值。
    3.  **概率预测图**：展示预测值的可能分布区间，适用于支持该功能的模型。
理解这些评估和可视化方法，有助于我们更全面地判断模型性能，并为实际应用（如量化交易）提供更可靠的依据。
