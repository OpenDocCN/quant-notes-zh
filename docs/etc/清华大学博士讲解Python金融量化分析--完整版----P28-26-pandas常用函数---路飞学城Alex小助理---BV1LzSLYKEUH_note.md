## 求平均值与求和

上一节我们介绍了如何处理缺失值，本节中我们来看看如何计算数据的统计量。pandas的DataFrame对象可以方便地对整列或整行数据进行计算。

`mean`方法用于计算平均值。对于一个DataFrame，默认按列计算每列的平均值，返回一个Series对象。

```python
df.mean()
```

对于一个有两列的DataFrame，上述代码会返回一个长度为2的Series，分别对应每一列的平均值。计算时会自动忽略缺失值（NaN）。

如果想按行计算平均值，需要指定`axis`参数。

```python
df.mean(axis=1)
```

`sum`方法用于求和，其用法与`mean`方法类似。默认按列求和，设置`axis=1`则按行求和。

```python
df.sum()        # 按列求和
df.sum(axis=1)  # 按行求和
```

## 数据排序

数据排序是整理和分析数据的重要步骤。pandas提供了两种主要的排序方式：按值排序和按索引排序。

### 按值排序

`sort_values`方法用于根据某一列或某一行中的值进行排序。使用`by`参数指定依据哪一列（或行）排序。

以下是按值排序的操作步骤：
1.  按指定列升序排序：`df.sort_values(by=‘column_name’)`
2.  按指定列降序排序：`df.sort_values(by=‘column_name’, ascending=False)`
3.  当排序键存在缺失值时，所有包含NaN的行不参与排序，统一被放置在结果的最后。

### 按索引排序

`sort_index`方法用于根据行索引或列索引的标签进行排序。

以下是按索引排序的操作步骤：
1.  按行索引标签排序：`df.sort_index()`
2.  按行索引标签降序排序：`df.sort_index(ascending=False)`
3.  按列索引标签排序：`df.sort_index(axis=1)`

## 其他通用函数

除了本节课重点介绍的函数，NumPy库中的许多通用函数同样适用于pandas。这些函数能帮助我们进行更全面的数据分析。

以下是一些同样可用的通用统计函数：
*   `std()`: 计算标准差
*   `var()`: 计算方差
*   `max()`: 查找最大值
*   `min()`: 查找最小值
