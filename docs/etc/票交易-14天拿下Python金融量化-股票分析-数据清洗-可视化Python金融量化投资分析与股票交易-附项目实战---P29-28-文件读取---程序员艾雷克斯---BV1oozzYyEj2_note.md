## CSV文件格式

CSV文件是一种纯文本文件，常用于存储表格数据。其基本规则是：每一行代表一条记录，每一列的数据由逗号分隔。这类似于Excel表格中的一个单元格。

## 使用 `read_csv` 函数读取文件

`read_csv` 函数是Pandas中用于读取CSV文件的核心函数。

以下是读取一个股票行情数据CSV文件的示例代码：

```python
import pandas as pd
df = pd.read_csv(‘stock_data.csv’)
```

执行上述代码后，数据会被读取进来。但初始读取的结果可能存在几个问题，需要我们通过参数进行调整。

## 调整读取参数

### 1. 指定行索引 (`index_col`)

默认情况下，Pandas会为数据自动生成一个从0开始的整数索引。如果我们希望使用文件中的某一列作为行索引，可以使用 `index_col` 参数。

*   **传入列编号**：`index_col=0` 表示使用第一列作为索引。
*   **传入列名**：`index_col=‘date’` 表示使用名为‘date’的列作为索引。这通常对时间序列数据更为合适。

### 2. 解析日期列 (`parse_dates`)

即使将日期列设置为索引，Pandas默认仍会将其视为字符串。为了进行时间序列分析，我们需要将其解析为真正的日期时间对象。

*   **解析所有可识别为日期的列**：`parse_dates=True`
*   **解析指定列**：`parse_dates=[‘date’]` 或 `parse_dates=[0]`（0代表第一列）

解析后，DataFrame的索引类型会变为 `DatetimeIndex`。

### 3. 处理无表头文件 (`header`, `names`)

如果CSV文件的第一行不是列名，而是数据，Pandas会误将第一行数据当作列名。此时需要使用 `header` 和 `names` 参数。

*   **`header=None`**：告知函数文件没有表头，Pandas会自动生成整数列名（0, 1, 2…）。
*   **`names=[列表]`**：在 `header=None` 的基础上，可以传入一个自定义的列名列表。

### 4. 指定分隔符 (`sep`)

`read_csv` 默认分隔符是逗号。Pandas还提供了 `read_table` 函数，其默认分隔符是制表符(`\t`)。两者功能相似，核心区别在于默认分隔符。

通过 `sep` 参数，可以指定任何字符作为分隔符，甚至可以使用正则表达式，例如 `sep=‘\s+’` 可以匹配任意长度的空白字符（如空格、制表符）。

### 5. 识别缺失值 (`na_values`)

数据中的缺失值可能以各种形式存在（如“N/A”, “null”, “-”）。Pandas默认只将空字符串和“NaN”识别为缺失值。

如果数据中使用其他标记（如“N”）代表缺失值，该列可能会被错误地识别为对象（字符串）类型。使用 `na_values` 参数可以指定哪些字符串应被解释为缺失值。

### 6. 其他参数

*   **`skiprows`**：跳过文件开头的指定行数（传入列表）或满足条件的行。
*   **`usecols`**：指定需要读取的列（传入列编号或列名列表）。

## 总结

本节课中我们一起学习了如何使用Pandas的 `read_csv` 和 `read_table` 函数从文件中读取数据。我们掌握了多个关键参数的作用：
*   `index_col` 用于指定行索引。
*   `parse_dates` 用于解析日期时间。
*   `header` 和 `names` 用于处理无表头文件。
*   `sep` 用于指定分隔符。
*   `na_values` 用于自定义缺失值标识。
