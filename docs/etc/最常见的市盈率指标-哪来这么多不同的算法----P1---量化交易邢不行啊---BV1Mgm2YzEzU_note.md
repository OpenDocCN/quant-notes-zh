# 量化投资基础：P1：市盈率指标详解 📊

![](img/88f48eea3f68147746052c333d93be11_1.png)

在本节课中，我们将要学习股票分析中最常见也最核心的财务指标之一——市盈率。我们将详细拆解其四种不同的计算方法，理解它们各自的含义、计算公式、优缺点以及应用场景。

## 概述

市盈率是衡量股票估值水平的关键指标，由市值与净利润构成。其理论计算公式为：

![](img/88f48eea3f68147746052c333d93be11_7.png)

**市盈率 = 市值 / 净利润**

![](img/88f48eea3f68147746052c333d93be11_9.png)

这个指标简洁明了，被绝大多数价值投资者用来判断股票是否被高估或低估。然而，您可能会发现，同一只股票在不同行情软件上显示的市盈率数值存在细微差别。这是因为市盈率有多种计算口径。

![](img/88f48eea3f68147746052c333d93be11_11.png)

接下来，我们将逐一解析这四种最常见的市盈率：静态市盈率、动态市盈率、滚动市盈率和预期市盈率。

---

![](img/88f48eea3f68147746052c333d93be11_13.png)

## 静态市盈率

![](img/88f48eea3f68147746052c333d93be11_15.png)

静态市盈率使用静态的、已公布的年度净利润数据进行计算。

其公式为：
**静态市盈率 = 当前总市值 / 上一年度全年净利润**

![](img/88f48eea3f68147746052c333d93be11_17.png)

例如，在2020年8月20日，贵州茅台的市值为20,903亿元。我们需要使用其2019年全年的净利润数据来计算当天的静态市盈率，计算结果约为47.5倍。

![](img/88f48eea3f68147746052c333d93be11_19.png)

静态市盈率的计算相对简单，但它没有考虑公司的业绩增长，因此越接近年末，该数据与当前实际经营情况的误差可能越大。

![](img/88f48eea3f68147746052c333d93be11_21.png)

---

## 动态市盈率

![](img/88f48eea3f68147746052c333d93be11_23.png)

动态市盈率使用折算的全年净利润数据，通常基于当年已公布的最新季度净利润进行估算。

![](img/88f48eea3f68147746052c333d93be11_25.png)

其公式为：
**动态市盈率 = 当前总市值 / (最新一期单季度净利润 × 4)**

同样以2020年8月20日的茅台为例，我们取用其2020年上半年的净利润，乘以2来估算全年净利润。以此计算出的动态市盈率约为43.7倍。

![](img/88f48eea3f68147746052c333d93be11_27.png)

这种方法在年初时误差较大，因为仅用单季度数据外推全年不够准确。但随着时间推移，可用季度数据增多，其估算会越来越接近实际情况。

![](img/88f48eea3f68147746052c333d93be11_29.png)

---

![](img/88f48eea3f68147746052c333d93be11_31.png)

## 滚动市盈率

滚动市盈率，也称为TTM市盈率，是我们量化交易实盘中最常用、也最准确的计算方法。它使用公司过去四个季度（即最近12个月）的净利润总和进行计算。

![](img/88f48eea3f68147746052c333d93be11_33.png)

其公式为：
**滚动市盈率 = 当前总市值 / 过去四个季度净利润之和**

![](img/88f48eea3f68147746052c333d93be11_35.png)

它能有效反映公司过去一年的真实盈利水平。在2020年8月20日，我们取用茅台2019年第三、四季度和2020年第一、二季度的净利润加总，计算出的滚动市盈率约为44.7倍。这个数据在所有主流行情软件上都会展示。

---

![](img/88f48eea3f68147746052c333d93be11_37.png)

## 预期市盈率

![](img/88f48eea3f68147746052c333d93be11_39.png)

预期市盈率的计算基于各大券商分析师对公司未来净利润的预测，主观性较强，可作为投资时的参考依据。

其公式为：
**预期市盈率 = 当前总市值 / 分析师预测的未来年度净利润平均值**

![](img/88f48eea3f68147746052c333d93be11_41.png)

例如，我们可以收集各家券商对贵州茅台2020年净利润的预测值，求其平均数作为“预期净利润”，进而计算出当时的预期市盈率，例如45.9倍。

![](img/88f48eea3f68147746052c333d93be11_43.png)

---

## 总结

![](img/88f48eea3f68147746052c333d93be11_45.png)

本节课中，我们一起学习了市盈率指标的四种不同计算方法。为了帮助您清晰地对比和记忆，以下是各方法的核心要点总结：

![](img/88f48eea3f68147746052c333d93be11_47.png)

**市盈率计算方法对比**

![](img/88f48eea3f68147746052c333d93be11_49.png)

| 市盈率类型 | 净利润数据来源 | 核心特点 |
| :--- | :--- | :--- |
| **静态市盈率** | 上一个完整财年的净利润 | 计算简单，但数据滞后，未反映增长。 |
| **动态市盈率** | 当年最新季度净利润折算全年 | 试图反映当年业绩，但年初估算误差大。 |
| **滚动市盈率** | 过去四个季度净利润之和 | **最准确**，反映过去12个月真实盈利，量化常用。 |
| **预期市盈率** | 分析师预测的未来年度净利润 | 基于市场预期，主观性强，可作为参考。 |

理解这些差异，能帮助您更准确地解读行情软件上的数据，并选择适合自己分析框架的市盈率指标进行投资决策。
