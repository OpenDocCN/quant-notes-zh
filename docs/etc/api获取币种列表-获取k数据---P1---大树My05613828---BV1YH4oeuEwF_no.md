## 获取指定币种K线数据 📊

用户可以在指定文本框中输入一个或多个币种（例如BTC或BTC,ETH），点击按钮后，程序将获取这些币种在过去一段时间的K线数据。

上一节我们实现了获取所有币种，本节中我们来看看如何获取特定币种的详细数据。

### 实现步骤

以下是获取K线数据的具体步骤：

1.  **解析用户输入**：处理文本框中的内容，可能是一个或多个由逗号分隔的币种名称。
2.  **构造K线API请求**：对于每个币种，向OKX的`/api/v5/market/candles`端点发送请求。需要参数包括币种对（如`BTC-USDT`）、时间周期（如`1H`表示1小时）和数量。
    ```python
    def fetch_kline_data(coin):
        url = "https://www.okx.com/api/v5/market/candles"
        params = {
            'instId': f'{coin}-USDT',
            'bar': '1Hutc',  # 1小时K线
            'limit': 100     # 获取100根
        }
        response = requests.get(url, params=params)
        return response.json()
    ```
3.  **显示数据**：将获取到的K线数据（包含时间、开盘价、最高价、最低价、收盘价等信息）整理并显示在程序界面中。
