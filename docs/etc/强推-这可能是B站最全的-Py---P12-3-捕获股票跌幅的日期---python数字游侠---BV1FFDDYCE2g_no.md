在本节课中，我们将学习如何利用Python的Pandas库，从股票数据中筛选出开盘价较前一日收盘价跌幅超过2%的日期。这是一个在量化分析中常见的需求，有助于我们识别市场中的异常波动。

![](img/0cfa290c0d51ebe8bf734063e1216573_5.png)

上一节我们介绍了如何计算股票的涨跌幅，本节中我们来看看如何根据特定的跌幅条件来筛选日期。

## 需求分析与伪代码

我们的目标是：找到某支股票所有“开盘价”比“前一日收盘价”跌幅超过2%的日期。

![](img/0cfa290c0d51ebe8bf734063e1216573_7.png)

首先，我们需要将需求转化为逻辑步骤。以下是实现此需求的核心思路：

![](img/0cfa290c0d51ebe8bf734063e1216573_9.png)

1.  **计算跌幅**：跌幅的计算公式为 `(当日开盘价 - 前一日收盘价) / 前一日收盘价`。
2.  **设定条件**：我们需要筛选出计算结果小于 `-0.02` 的日期（因为跌幅3%即 `-0.03`，它小于 `-0.02`）。
3.  **获取日期**：将满足上述布尔条件的行从数据中筛选出来，并提取其对应的日期索引。

用伪代码可以表示为：
```
跌幅 = (开盘价 - 前一日收盘价) / 前一日收盘价
满足条件的日期 = 数据[跌幅 < -0.02].index
```

## 代码实现与分步讲解

接下来，我们将伪代码转化为实际的Python代码。关键在于如何正确获取“前一日收盘价”。

![](img/0cfa290c0d51ebe8bf734063e1216573_11.png)

在Pandas的DataFrame中，我们可以使用 `.shift(1)` 方法将`close`列整体向下移动一行。这样，每一行的`close`值就变成了前一行的收盘价，即我们需要的“前一日收盘价”。

![](img/0cfa290c0d51ebe8bf734063e1216573_13.png)

以下是实现该需求的具体代码步骤：

![](img/0cfa290c0d51ebe8bf734063e1216573_15.png)

```python
# 计算跌幅序列
# df['open'] 是当日开盘价
# df['close'].shift(1) 是前一日收盘价
pct_change = (df['open'] - df['close'].shift(1)) / df['close'].shift(1)

![](img/0cfa290c0d51ebe8bf734063e1216573_17.png)

# 创建布尔序列，标识跌幅超过2%的行
condition = pct_change < -0.02

![](img/0cfa290c0d51ebe8bf734063e1216573_19.png)

![](img/0cfa290c0d51ebe8bf734063e1216573_21.png)

# 使用布尔索引筛选出满足条件的行，并获取其日期索引
result_dates = df.loc[condition].index
```

为了更清晰地理解，我们可以将以上步骤合并为一行代码：

![](img/0cfa290c0d51ebe8bf734063e1216573_23.png)

```python
result_dates = df.loc[(df['open'] - df['close'].shift(1)) / df['close'].shift(1) < -0.02].index
```

![](img/0cfa290c0d51ebe8bf734063e1216573_25.png)

![](img/0cfa290c0d51ebe8bf734063e1216573_27.png)

这行代码直接完成了计算、比较和筛选的全过程，`result_dates` 就是我们要找的所有跌幅超过2%的日期。
