## OBV能量潮指标

上一节我们介绍了压力支撑指标，本节中我们来看看量价指标。首先介绍的是OBV能量潮指标。该指标由格兰维尔于上世纪60年代提出，通过统计成交量的变化来绘制OBV线。OBV线是研判股市短期波动的重要方法。通常，OBV线需要配合股价趋势一起分析。

### OBV算法

OBV的计算基于比较当日与前一日的收盘价。其核心公式如下：

**OBV_t = OBV_{t-1} + sign(Close_t - Close_{t-1}) * Volume_t**

其中，`sign`是符号函数：
*   当今日收盘价 **大于等于** 昨日收盘价时，`sign`函数结果为 **+1**。
*   当今日收盘价 **小于** 昨日收盘价时，`sign`函数结果为 **-1**。

公式中的成交量`Volume_t`通常以“手”为单位。计算第一日的OBV时，其初始值可直接使用该日的成交量。

### OBV的应用

以下是OBV指标的几种典型应用场景：

1.  **顶背离与底背离**：当股价一顶比一顶高，但OBV一顶比一顶低时，暗示头部形成，是卖出信号。反之，当股价一底比一底低，但OBV一底比一底高时，暗示底部形成，是买入信号。
2.  **N字形突破**：当OBV线向上突破其N字形波动的高点达到五次时，为短线卖点。当OBV线向下跌破其N字形波动的低点达到五次时，为短线买点。
3.  **结合使用原则**：**OBV指标不能单独使用**，必须与股价走势图结合分析，否则其信号意义有限。

### OBV的Python函数

在Python的技术分析库中，可以直接调用`OBV`函数进行计算。

```python
# 示例：计算OBV值
obv_data = OBV(close_prices, volume, timeperiod=30)
```
函数`OBV`返回一个字典类型的数据，包含计算出的OBV值序列。参数`timeperiod`通常设为30，表示统计周期。

## VOL成交量指标

接下来，我们介绍另一个基础且重要的量价指标——成交量。成交量指个股或市场的成交总手数。在图表上，成交量通常用垂直的柱状图表示。

### 成交量图解读

成交量柱的颜色具有特定含义：
*   若当日**收盘价 >= 开盘价**，成交量柱显示为**红色**（代表上涨）。
*   若当日**收盘价 < 开盘价**，成交量柱显示为**绿色**（代表下跌）。
图中常伴随的均线（如5日、10日、30日均线）是成交量的移动平均线，用于观察成交量的趋势。

### 成交量的应用

以下是成交量常见的几种研判方法：

![](img/592b4189aa0ddcfeb76b9d1cb090acd7_1.png)

1.  **突破信号**：当股价处于盘整行情时，若成交量突然显著放大，则股价在短期内向上或向下突破的可能性较高。
2.  **放量上涨**：当成交量连续出现三条或更多的放量柱时，表明市场交易活跃，股价继续下跌的可能性较小，上涨概率较大。
3.  **量价背离**：股价经过一段时间上涨后，成交量虽然依然较大，但K线收出阴线或实体很小的阳线（上涨乏力）。这通常意味着推动上涨的主力资金可能在撤退，仅靠散户支撑，股价短期内可能反转下跌。

### VOL的Python函数

Python技术分析库提供了计算成交量及其均线的函数。

```python
# 示例：计算成交量及其均线
vol_data = VOL(high_prices, low_prices, close_prices, volume, m1=5, m2=10)
```
函数`VOL`返回一个字典，通常包含三个值：当日的成交量、`M1`日成交量均线、`M2`日成交量均线。参数`m1`和`m2`默认常取5和10。

## 代码实战环节

前面我们介绍了OBV和VOL的理论与应用，现在进入代码实战部分。我们将演示如何使用Python计算这两类指标。

首先，导入必要的技术分析库并准备股票数据。

```python
# 示例代码结构
import talib  # 假设使用TA-Lib库
# 假设已有股票数据：close_prices, volume, high_prices, low_prices

# 1. 计算OBV
obv_values = talib.OBV(close_prices, volume, timeperiod=30)
print("OBV值示例：", obv_values[-5:])  # 打印最近5个OBV值

# 2. 计算VOL（成交量及均线）
# 注意：TA-Lib中计算移动平均成交量的函数可能是MA，此处用MA模拟VOL均线
volume_ma5 = talib.MA(volume, timeperiod=5)
volume_ma10 = talib.MA(volume, timeperiod=10)
print("成交量5日均线示例：", volume_ma5[-5:])
print("成交量10日均线示例：", volume_ma10[-5:])
```
运行代码后，可以得到指定股票在最近交易日的OBV值以及成交量的移动平均值，用于后续分析。
