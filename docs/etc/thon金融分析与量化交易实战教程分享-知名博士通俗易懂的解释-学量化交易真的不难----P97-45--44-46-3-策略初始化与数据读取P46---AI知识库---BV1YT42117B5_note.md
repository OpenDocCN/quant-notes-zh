![](img/ec23b69210652abf777400b317e613f7_22.png)

![](img/ec23b69210652abf777400b317e613f7_24.png)

1.  使用 `run_monthly` 函数。
2.  第一个参数指定调仓时执行的函数，我们将其命名为 `rebalance`。
3.  第二个参数指定调仓日期。

## 数据获取与处理
上一节我们定义了调仓函数，本节中我们来看看如何实现核心的 `get_stocks` 函数，以获取并处理股票数据。

![](img/ec23b69210652abf777400b317e613f7_32.png)

![](img/ec23b69210652abf777400b317e613f7_34.png)

`get_stocks` 函数的第一步是查询所需的股票基本面数据。我们将查询两类指标：越高越好的指标和越低越好的指标。

![](img/ec23b69210652abf777400b317e613f7_36.png)

![](img/ec23b69210652abf777400b317e613f7_38.png)

以下是数据查询的步骤：
1.  构建查询语句 `query`，指定需要获取的指标。
2.  使用 `filter` 条件将查询范围限定在沪深300股票池内。
3.  调用 `get_fundamentals` 函数执行查询。
4.  对返回的 `DataFrame` 进行转置，方便后续按股票代码索引数据。

![](img/ec23b69210652abf777400b317e613f7_40.png)

![](img/ec23b69210652abf777400b317e613f7_42.png)

![](img/ec23b69210652abf777400b317e613f7_44.png)

首先，查询“越高越好”的指标，例如每股收益、净资产收益率等。

```python
def get_stocks(context):
    # 1. 构建查询语句：查询越高越好的指标
    q_high = query(
        fundamentals.eod_derivative_indicator.eps, # 每股收益 (EPS)
        fundamentals.financial_indicator.roe,       # 净资产收益率 (ROE)
        fundamentals.financial_indicator.roa        # 总资产收益率 (ROA)
        # ... 可以添加更多指标
    ).filter(
        fundamentals.stockcode.in_(context.hs300) # 过滤条件：仅限沪深300成分股
    )
    # 执行查询并转置DataFrame
    df_high = get_fundamentals(q_high).T
```

接着，用类似的方法查询“越低越好”的指标，例如资产负债率、市净率等。

```python
    # 2. 构建查询语句：查询越低越好的指标
    q_low = query(
        fundamentals.financial_indicator.debt_to_asset_ratio, # 资产负债率
        fundamentals.eod_derivative_indicator.pb_ratio,       # 市净率 (PB)
        fundamentals.eod_derivative_indicator.market_cap      # 总市值
        # ... 可以添加更多指标
    ).filter(
        fundamentals.stockcode.in_(context.hs300) # 过滤条件：仅限沪深300成分股
    )
    # 执行查询并转置DataFrame
    df_low = get_fundamentals(q_low).T
```

至此，我们得到了两个 `DataFrame`：`df_high` 包含越高越好的指标数据，`df_low` 包含越低越好的指标数据。这些数据将为下一步的因子打分做好准备。
