## 影响股价的六大因素

上一节我们介绍了股票的基本概念，本节中我们来看看哪些因素会影响股票价格的波动。影响股价的因素可以归纳为以下六点。

以下是影响股价的六大因素：

1.  **公司自身因素**：这是影响股价最根本的因素。公司的经营状况、盈利能力、发展前景等基本面直接决定了其长期价值。如果公司发展良好，市值增长，股价通常会上涨；反之，若出现重大负面事件或经营不善，股价则会下跌。
2.  **市场因素**：这是影响股价最直接的因素。股价的短期波动由市场的供求关系决定。当买盘大于卖盘（供不应求）时，股价上涨；当卖盘大于买盘（供过于求）时，股价下跌。
3.  **行业因素**：公司所属行业的整体景气度会影响行业内所有公司的股价。例如，当某个行业（如人工智能）成为市场热点时，相关公司的股票可能普遍上涨；反之，若某个行业前景黯淡，相关股票则可能下跌。
4.  **心理因素**：投资者的情绪和非理性行为会影响股价。例如，从众心理可能导致恐慌性抛售或盲目追涨，即使公司基本面并未发生重大变化。历史上因交易错误或恐慌情绪引发的市场暴跌（如“黑色星期一”）便是例证。
5.  **经济因素**：国家宏观经济政策、利率、存款准备金率、通货膨胀率等都会影响市场整体资金面和投资者信心。例如，央行加息可能导致市场资金流向银行存款，从而减少股市资金供给，对股价产生下行压力。
6.  **政治因素**：国际关系、地区局势、政府政策变动等政治事件会显著影响市场稳定性。例如，地缘政治紧张局势可能引发市场恐慌，导致股市下跌；而相关政策利好的行业（如军工）则可能上涨。

## 股票买卖的基本流程与规则

了解了影响股价的因素后，我们来看看在实际操作中，股票买卖是如何进行的，以及有哪些重要的交易规则。

### 1. 开户与委托

个人投资者不能直接在证券交易所买卖股票，必须通过证券公司（券商）进行。首先需要在券商处开设证券账户和资金账户。开户后，投资者通过券商的交易系统（如软件或电话）提交买卖指令，这个过程称为“委托”。

### 2. 交易日与交易时间

股票交易并非随时可以进行，它有固定的交易日和交易时间。

*   **交易日**：通常为每周一至周五（非法定节假日）。
*   **交易时间**：以中国A股市场为例，每个交易日的交易时段划分如下：
    *   **开盘集合竞价**：**9:15 - 9:25**。在此期间提交的委托单不会立即成交，交易所会在9:25一次性对所有委托进行集中撮合，以产生当日的**开盘价**。撮合原则是**最大化成交量**。
    *   **连续竞价（早市）**：**9:30 - 11:30**。在此期间，交易所系统对买卖委托进行**连续、实时的撮合**，通常每几秒进行一次，成交价格由实时供求决定。
    *   **连续竞价（午市）**：**13:00 - 14:57**（对于深圳证券交易所）；**13:00 - 15:00**（对于上海证券交易所）。此阶段同样是连续竞价。
    *   **收盘集合竞价**（仅深交所）：**14:57 - 15:00**。与开盘集合竞价类似，此期间委托单不实时撮合，在15:00集中撮合产生当日的**收盘价**。上交所的收盘价为当日最后一笔连续竞价的成交价。

### 3. 重要交易制度

以下是两个关键的交易制度：

*   **T+1交易制度**：指当日（T日）买入的股票，必须到下一个交易日（T+1日）才能卖出。但当日卖出股票获得的资金，当日可以用于购买其他股票，提现则需要等到T+1日。
*   **涨跌停板制度**：为抑制过度投机，防止股价剧烈波动，A股市场对普通股票设有每日价格涨跌幅限制。通常为**±10%**（ST等特殊股票为±5%）。这意味着，一只股票当日的交易价格被限制在前一交易日收盘价上下10%的范围内。
