## 累计收益计算 💰

![](img/4da84868f8efafd76938cc5b471e78bd_8.png)

首先，我们需要计算三种不同策略的累计收益。以下是计算步骤：

![](img/4da84868f8efafd76938cc5b471e78bd_10.png)

1.  **提取关键指标**：从数据中提取实际收益率（`returns`）以及两个回归模型的预测结果（`pred_1`, `pred_2`）。
2.  **进行累加操作**：由于我们之前对收益率进行了对数处理（`log(1+收益率)`），这使得收益率具有可加性。对序列进行累加，相当于计算累计对数收益。
3.  **还原为实际收益**：将对数收益通过指数函数 `np.exp()` 还原为实际的累计收益率。

以下是实现代码：

![](img/4da84868f8efafd76938cc5b471e78bd_12.png)

```python
# 提取指标
returns = data['returns']
pred_1 = data['pred_1']
pred_2 = data['pred_2']

# 计算累计收益：先累加，再通过指数函数还原
cumulative_returns = np.exp(returns.cumsum())
cumulative_strategy_1 = np.exp(pred_1.cumsum())
cumulative_strategy_2 = np.exp(pred_2.cumsum())

# 查看最终结果（例如最后一天的累计收益）
print(cumulative_returns.iloc[-1])
print(cumulative_strategy_1.iloc[-1])
print(cumulative_strategy_2.iloc[-1])
```

执行结果分析：
*   **实际收益率 (`returns`)**：最终值小于1，表示初始的1元钱亏损了，变成了约0.8元。
*   **策略一 (`pred_1`)**：最终值约为0.94，虽然也亏损，但比什么都不做的策略亏损得少。
*   **策略二 (`pred_2`)**：最终值大于1（例如1.3），表示初始的1元钱通过该策略增值了，赚取了收益。

## 预测准确率分析 ✅

为了理解策略表现差异的原因，我们可以分析模型的预测准确率。我们将模型预测的涨跌方向与实际涨跌方向进行比较。

![](img/4da84868f8efafd76938cc5b471e78bd_14.png)

以下是分析步骤：

![](img/4da84868f8efafd76938cc5b471e78bd_16.png)

1.  **获取实际方向**：数据中 `direction` 列表示实际的涨跌（例如，1为涨，0为跌）。
2.  **比较预测方向**：将回归模型的预测值（`pred_1`, `pred_2`）转换为涨跌方向（例如，预测值大于0为看涨）。
3.  **计算正确率**：统计预测方向与实际方向一致的次数。

![](img/4da84868f8efafd76938cc5b471e78bd_18.png)

以下是实现代码：

![](img/4da84868f8efafd76938cc5b471e78bd_20.png)

```python
# 假设实际涨跌方向列名为 ‘direction‘
actual_direction = data[‘direction‘]

![](img/4da84868f8efafd76938cc5b471e78bd_22.png)

# 将回归预测值转换为涨跌方向（例如，大于0为涨）
pred_direction_1 = (pred_1 > 0)
pred_direction_2 = (pred_2 > 0)

![](img/4da84868f8efafd76938cc5b471e78bd_24.png)

![](img/4da84868f8efafd76938cc5b471e78bd_26.png)

# 计算策略一的预测准确情况
correct_1 = (pred_direction_1 == actual_direction)
value_counts_1 = correct_1.value_counts()
print(“策略一预测准确情况：“, value_counts_1)

![](img/4da84868f8efafd76938cc5b471e78bd_28.png)

# 计算策略二的预测准确情况
correct_2 = (pred_direction_2 == actual_direction)
value_counts_2 = correct_2.value_counts()
print(“策略二预测准确情况：“, value_counts_2)
```

分析结果可能显示，策略二的预测正确次数略高于策略一，这解释了为什么策略二能获得正向收益。这说明了数据和标签（回归目标）的选择对最终策略效果有重要影响。

![](img/4da84868f8efafd76938cc5b471e78bd_30.png)

## 策略走势可视化 📊

![](img/4da84868f8efafd76938cc5b471e78bd_32.png)

![](img/4da84868f8efafd76938cc5b471e78bd_34.png)

最后，我们通过可视化来观察三种策略在整个时间序列上的动态走势，而不仅仅是最终结果。这需要使用累积求和函数 `cumsum()`。

![](img/4da84868f8efafd76938cc5b471e78bd_36.png)

`cumsum()` 函数的作用是生成一个序列，其中每个位置的值是原序列从开始到当前位置所有值的和。这完美地展示了资产随时间的累计变化过程。

以下是绘制策略走势图的代码：

![](img/4da84868f8efafd76938cc5b471e78bd_38.png)

```python
import matplotlib.pyplot as plt

![](img/4da84868f8efafd76938cc5b471e78bd_40.png)

# 计算累计收益序列（已还原）
cum_returns_series = np.exp(returns.cumsum())
cum_strategy1_series = np.exp(pred_1.cumsum())
cum_strategy2_series = np.exp(pred_2.cumsum())

# 绘制走势图
plt.figure(figsize=(16, 8))
plt.plot(cum_returns_series, label=‘实际走势 (基准)‘, color=‘blue‘)
plt.plot(cum_strategy1_series, label=‘策略一‘, color=‘green‘)
plt.plot(cum_strategy2_series, label=‘策略二‘, color=‘red‘)

![](img/4da84868f8efafd76938cc5b471e78bd_42.png)

plt.title(‘不同交易策略累计收益走势对比‘)
plt.xlabel(‘时间‘)
plt.ylabel(‘累计收益 (起始为1)‘)
plt.legend()
plt.grid(True)
plt.show()
```

![](img/4da84868f8efafd76938cc5b471e78bd_44.png)

![](img/4da84868f8efafd76938cc5b471e78bd_46.png)

图表说明：
*   **蓝色线**：代表“买入并持有”的基准策略走势，可见其总体呈下跌趋势。
*   **绿色线**：代表基于第一个回归方程的策略走势，虽然最终也下跌，但其路径与基准线不同。
*   **红色线**：代表基于第二个回归方程的策略走势，在大部分时间内跑赢了基准线，并实现了最终盈利。
