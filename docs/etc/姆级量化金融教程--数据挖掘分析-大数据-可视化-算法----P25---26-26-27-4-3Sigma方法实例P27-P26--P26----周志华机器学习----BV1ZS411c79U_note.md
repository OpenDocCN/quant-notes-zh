# Python量化交易：P25-P27：3-Sigma方法实例

在本节课中，我们将要学习一种名为“3-Sigma”的数据去极值方法。该方法基于统计学中的正态分布原理，通过计算数据的均值和标准差，来识别并处理数据中的异常值。

## 原理概述

上一节我们介绍了去极值的概念，本节中我们来看看基于正态分布的3-Sigma方法。其核心思想是：在正态分布中，绝大多数数据（约99.7%）会落在均值加减三倍标准差的范围内。因此，我们可以将此范围之外的数据视为异常值并进行处理。

为了便于理解，我们可以将正态分布想象为一种“正常”的数据分布模式。例如，在银行贷款场景中，银行批准额度在均值附近小幅波动是正常的，但若出现极大或极小的偏差，则可能属于异常情况。

下图展示了一个标准正态分布（均值为0）。图中红色阴影区域面积较大，代表事件发生的概率较高，即数据落在均值附近的概率大。而越远离均值的区域（如图中最两侧），其面积越小，代表事件发生的概率越低，这些区域的数据点就更可能是我们需要处理的极值或离群点。

## 3-Sigma规则详解

3-Sigma规则具体定义了数据的正常范围。如下图所示，我们以均值（μ）为中心，分别向左右扩展1倍、2倍和3倍的标准差（σ）。

![](img/0b394fd724301baacbdaeacb888c477b_3.png)

![](img/0b394fd724301baacbdaeacb888c477b_3.png)

可以观察到，区间范围越大（例如μ ± 3σ），数据落在该区间外的可能性就越小。3-Sigma方法就是利用这一特性：

1.  假设数据经过某种变换后，近似服从正态分布。
2.  设定正常数据的范围为 **μ - 3σ** 到 **μ + 3σ**。
3.  根据统计学知识，这个范围理论上包含了约99.7%的数据。
4.  我们将落在此范围之外的约0.3%的数据视为极值，并进行规范化处理。

因此，实施该方法的关键在于计算两个核心指标：数据的**均值（μ）**和**标准差（σ）**。有了这两个值，我们就可以确定处理数据的上下限。

![](img/0b394fd724301baacbdaeacb888c477b_5.png)

上限计算公式为：**上限 = μ + N * σ**
下限计算公式为：**下限 = μ - N * σ**
其中，N通常取3。

![](img/0b394fd724301baacbdaeacb888c477b_7.png)

## Python代码实现

![](img/0b394fd724301baacbdaeacb888c477b_5.png)

接下来，我们将用Python代码实现3-Sigma方法。我们将创建一个函数，它接收一列数据和一个倍数N，返回处理后的数据以及计算出的上下限。

![](img/0b394fd724301baacbdaeacb888c477b_7.png)

首先，我们定义函数并计算数据的均值和标准差。

```python
def three_sigma(data_series, n=3):
    # 计算均值 (μ)
    mean = data_series.mean()
    # 计算标准差 (σ)
    std = data_series.std()
```

![](img/0b394fd724301baacbdaeacb888c477b_9.png)

![](img/0b394fd724301baacbdaeacb888c477b_11.png)
![](img/0b394fd724301baacbdaeacb888c477b_13.png)

接着，根据公式计算上下限。

![](img/0b394fd724301baacbdaeacb888c477b_11.png)

```python
    # 计算上限
    upper_limit = mean + n * std
    # 计算下限
    lower_limit = mean - n * std
```

![](img/0b394fd724301baacbdaeacb888c477b_15.png)

![](img/0b394fd724301baacbdaeacb888c477b_13.png)

最后，对原始数据进行处理：将所有大于上限的值设置为上限，所有小于下限的值设置为下限，并返回结果。

![](img/0b394fd724301baacbdaeacb888c477b_15.png)

```python
    # 处理数据：限制在上下限之间
    processed_series = data_series.clip(lower=lower_limit, upper=upper_limit)
    # 返回处理后的序列及上下限
    return processed_series, lower_limit, upper_limit
```
