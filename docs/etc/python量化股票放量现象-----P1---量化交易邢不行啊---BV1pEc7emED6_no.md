## 从经验到规则：量化思维的必要性

在股票分析中，我们经常听到“高位放量”或“低位放量”这样的说法。人脑通过观察图表可以直观地理解这些概念，但计算机无法理解这种模糊的描述。

![](img/56c725604afbc85d440c799707ec7b66_9.png)

计算机只能执行一行行明确的代码和精确的规则。因此，我们需要将“股价在什么位置算低位？”、“成交量多大算放量？”这些不精确的自然语言描述，转化为数学或程序化的定义。

![](img/56c725604afbc85d440c799707ec7b66_11.png)

## 如何量化定义“高位”与“低位”

![](img/56c725604afbc85d440c799707ec7b66_13.png)

![](img/56c725604afbc85d440c799707ec7b66_15.png)

上一节我们介绍了将经验转化为规则的必要性，本节中我们来看看如何具体定义“高位”和“低位”。

![](img/56c725604afbc85d440c799707ec7b66_17.png)

一个常见的方法是：将当日的股价与过去一段时间的股价进行比较，通过其相对位置来判断高低。

![](img/56c725604afbc85d440c799707ec7b66_19.png)

我们采用以下定义：
*   **高位**：当日收盘价在过去120个交易日的收盘价中，排名位于前10%。
*   **低位**：当日收盘价在过去120个交易日的收盘价中，排名位于后10%。

![](img/56c725604afbc85d440c799707ec7b66_21.png)

![](img/56c725604afbc85d440c799707ec7b66_23.png)

这个定义的核心是比较。当然，你也可以根据自己的理解修改定义，例如要求当日股价必须是过去120天中的最高价才算高位。

## 如何量化定义“放量”

![](img/56c725604afbc85d440c799707ec7b66_25.png)

定义了股价的高低位置后，我们再来看看如何定义“放量”。成交量是另一个需要精确化的维度。

![](img/56c725604afbc85d440c799707ec7b66_27.png)

我们首先将成交量转换为**换手率**，这能更好地反映交易的活跃程度。换手率的计算公式为：
`换手率 = 成交量 / 流通股数`

![](img/56c725604afbc85d440c799707ec7b66_29.png)

接下来，我们借鉴**布林带（Bollinger Bands）** 指标的思想来定义放量。布林带的计算公式涉及均值与标准差。

![](img/56c725604afbc85d440c799707ec7b66_31.png)

我们将此思想应用于换手率，定义“放量”如下：
*   计算过去100个交易日的换手率均值（`mean`）和标准差（`std`）。
*   设定一条“上轨”：`上轨 = mean + 2 * std`
*   如果当日的换手率高于这条上轨，则判定为“放量”。

![](img/56c725604afbc85d440c799707ec7b66_33.png)

以下是该定义的伪代码描述：
```python
# 假设 turnover_rate 是包含每日换手率的序列
mean_100 = rolling_mean(turnover_rate, window=100)
std_100 = rolling_std(turnover_rate, window=100)
upper_band = mean_100 + 2 * std_100
is_volume_spike = turnover_rate > upper_band
```

![](img/56c725604afbc85d440c799707ec7b66_35.png)

需要强调的是，放量的定义方法多种多样。例如，你也可以简单地判断当日成交量是否在过去120天中排名前十。不同的定义可能产生不同的效果，值得尝试和比较。

![](img/56c725604afbc85d440c799707ec7b66_37.png)

## 总结

![](img/56c725604afbc85d440c799707ec7b66_49.png)

本节课中我们一起学习了量化交易的基础步骤：将投资经验转化为精确规则。
1.  我们定义了**高位**和**低位**为股价在过去120日内的相对排名位置。
2.  我们定义了**放量**为当日换手率突破基于过去100日数据计算的“布林带上轨”。
3.  这些定义将模糊的语言描述变成了计算机可执行的明确逻辑，为后续的历史回测和策略构建打下了坚实基础。
