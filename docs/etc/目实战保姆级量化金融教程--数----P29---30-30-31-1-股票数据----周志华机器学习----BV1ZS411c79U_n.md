## 策略初始化与定时器设置

![](img/f2f8fafde8ce9fb7722ae92af4b49002_26.png)

## 股票数据获取与初步筛选
设置好调仓频率后，下一步是获取股票数据。我们将获取市场上所有的股票作为初始的“股票池”，但通常我们不会交易所有股票，因此需要进行初步筛选。

![](img/f2f8fafde8ce9fb7722ae92af4b49002_48.png)

![](img/f2f8fafde8ce9fb7722ae92af4b49002_50.png)

以下是获取和筛选股票数据的基本步骤：

![](img/f2f8fafde8ce9fb7722ae92af4b49002_52.png)

![](img/f2f8fafde8ce9fb7722ae92af4b49002_54.png)

![](img/f2f8fafde8ce9fb7722ae92af4b49002_56.png)

![](img/f2f8fafde8ce9fb7722ae92af4b49002_58.png)

1.  **获取全市场股票**：使用平台提供的API（例如`get_all_securities`）获取当前市场上所有股票的基本信息列表。
2.  **定义筛选条件**：根据策略需求，设定初步的筛选规则。例如，只选择特定板块、排除ST股票、或选择市值大于一定规模的股票。
3.  **应用筛选**：根据设定的条件，从全市场股票列表中筛选出符合条件的股票，构成我们的初始股票池。

![](img/f2f8fafde8ce9fb7722ae92af4b49002_60.png)

![](img/f2f8fafde8ce9fb7722ae92af4b49002_62.png)

![](img/f2f8fafde8ce9fb7722ae92af4b49002_64.png)

```python
# 伪代码示例：获取并筛选股票
all_stocks = get_all_securities(types=['stock']) # 获取所有股票
# 定义筛选条件，例如：非ST、主板上市、市值大于100亿
filtered_stocks = all_stocks[(all_stocks['is_st'] == False) &
                             (all_stocks['board'] == 'main') &
                             (all_stocks['market_cap'] > 100e8)]
stock_pool = filtered_stocks.index.tolist() # 获取股票代码列表
```
