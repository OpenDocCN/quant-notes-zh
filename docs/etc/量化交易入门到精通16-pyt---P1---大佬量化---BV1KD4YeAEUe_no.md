## 数字类型概述

![](img/98d3332a0388b12463710b33d5257eae_5.png)

Python支持多种数字类型，以满足不同的计算需求。在量化交易场景中，我们主要会用到以下几种类型：

![](img/98d3332a0388b12463710b33d5257eae_7.png)

![](img/98d3332a0388b12463710b33d5257eae_9.png)

*   **整数类型 (`int`)**：用于表示没有小数部分的数字，例如股票成交量。
*   **浮点数类型 (`float`)**：用于表示带有小数点的数字，例如股票价格。
*   **布尔类型 (`bool`)**：用于表示逻辑值，即`True`（真）或`False`（假），例如判断股票是否涨停。

![](img/98d3332a0388b12463710b33d5257eae_11.png)

上一节我们介绍了变量的概念，本节中我们来看看如何用这些数字类型来表示具体的交易数据。

![](img/98d3332a0388b12463710b33d5257eae_13.png)

## 数字类型在量化中的应用

![](img/98d3332a0388b12463710b33d5257eae_15.png)

在A股交易中，股价（如19.98元、12.34元）通常使用浮点数类型来表示。涨跌幅的计算结果也通常是浮点数。而成交量（如10000手）则使用整数类型。判断股票是否达到涨跌停板状态，其结果是一个布尔值。

![](img/98d3332a0388b12463710b33d5257eae_17.png)

以下是几个具体的代码示例，展示了如何定义这些变量并查看其类型：

![](img/98d3332a0388b12463710b33d5257eae_19.png)

```python
# 定义股价，使用浮点数
stock_price = 5.98
# 定义是否涨停，使用布尔值
is_limit_up = True
# 定义成交量，使用整数
volume = 10000

![](img/98d3332a0388b12463710b33d5257eae_21.png)

# 打印变量的值
print(“股价：”, stock_price)
print(“是否涨停：”, is_limit_up)
print(“成交量：”, volume)

![](img/98d3332a0388b12463710b33d5257eae_23.png)

# 打印变量的类型
print(“股价的类型是：”, type(stock_price))
print(“是否涨停的类型是：”, type(is_limit_up))
print(“成交量的类型是：”, type(volume))
```

![](img/98d3332a0388b12463710b33d5257eae_25.png)

运行以上代码，输出结果将会是：
```
股价： 5.98
是否涨停： True
成交量： 10000
股价的类型是： <class ‘float’>
是否涨停的类型是： <class ‘bool’>
成交量的类型是： <class ‘int’>
```

![](img/98d3332a0388b12463710b33d5257eae_27.png)

从输出中可以看到，`stock_price`是`float`类型，`is_limit_up`是`bool`类型，`volume`是`int`类型。`type()`是Python的内置函数，用于返回任何对象的类型，这有助于我们确认变量的数据类型。

![](img/98d3332a0388b12463710b33d5257eae_29.png)

## 核心概念与操作

![](img/98d3332a0388b12463710b33d5257eae_31.png)

掌握了基本类型后，了解一些核心概念能让你的代码更严谨。

![](img/98d3332a0388b12463710b33d5257eae_33.png)

**1. 类型转换**
有时我们需要在不同数字类型间进行转换。例如，将浮点数股价转换为整数（会丢失小数部分），或者将整数成交量转换为浮点数。

![](img/98d3332a0388b12463710b33d5257eae_35.png)

```python
# 类型转换示例
price_float = 10.5
price_int = int(price_float)  # 结果为 10
volume_int = 5000
volume_float = float(volume_int)  # 结果为 5000.0
```

![](img/98d3332a0388b12463710b33d5257eae_37.png)

**2. 算术运算**
Python支持基本的加(`+`)、减(`-`)、乘(`*`)、除(`/`)等算术运算。需要注意的是，整数相除的结果是浮点数。

![](img/98d3332a0388b12463710b33d5257eae_39.png)

```python
# 算术运算示例
a = 7
b = 3
print(a + b)  # 加法，输出 10
print(a - b)  # 减法，输出 4
print(a * b)  # 乘法，输出 21
print(a / b)  # 除法，输出 2.3333333333333335
```
