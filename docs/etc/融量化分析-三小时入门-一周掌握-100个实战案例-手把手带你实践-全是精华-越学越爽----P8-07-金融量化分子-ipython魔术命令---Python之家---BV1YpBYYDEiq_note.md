# 金融量化分析：07：IPython魔术命令 🪄

![](img/7da4033f76e59594367f81c7b23c6fd9_1.png)

在本节课中，我们将学习IPython中一系列强大且便捷的“魔术命令”。这些命令以百分号（`%`）开头，能让我们在交互式环境中更高效地执行文件、测试代码片段和测量性能，无需频繁切换窗口或编写额外代码。

上一节我们介绍了IPython的基本交互功能，本节中我们来看看如何利用魔术命令提升我们的开发效率。

## 运行外部Python文件

![](img/7da4033f76e59594367f81c7b23c6fd9_3.png)

在标准的Python命令行中，若要运行一个已写好的Python文件，通常需要退出命令行，再通过`python 文件名.py`的方式执行。在IPython中，我们可以使用`%run`命令直接运行，无需退出当前交互环境。

![](img/7da4033f76e59594367f81c7b23c6fd9_5.png)

例如，假设我们在当前目录下有一个名为`hello.py`的文件，其内容为：
```python
print("Hello, World!")
```
在IPython中，只需输入以下命令即可执行：
```python
%run hello.py
```
该命令会执行`hello.py`文件中的代码，并将结果输出在当前的IPython交互器中。

## 执行剪贴板中的代码

有时，我们可能从编辑器或其他地方复制了一段代码，希望快速在IPython中测试。除了直接粘贴，IPython提供了`%paste`魔术命令来专门处理此场景。

![](img/7da4033f76e59594367f81c7b23c6fd9_7.png)

![](img/7da4033f76e59594367f81c7b23c6fd9_9.png)

`%paste`命令会读取系统剪贴板中的内容，将其作为代码块在IPython中执行。这对于测试长代码中的特定片段非常有用，无需单独创建文件。

![](img/7da4033f76e59594367f81c7b23c6fd9_11.png)

![](img/7da4033f76e59594367f81c7b23c6fd9_13.png)

以下是操作步骤：
1.  在任何地方（如文本编辑器）复制你想测试的代码。
2.  在IPython中输入`%paste`并回车。
3.  IPython会先打印出剪贴板中的代码，然后执行它。

## 测量代码执行时间

在性能分析和优化时，我们经常需要测量某段代码或函数的执行时间。虽然可以使用Python的`time`模块，但对于执行时间极短的代码，单次测量可能不准确（结果可能为0秒）。

IPython的`%timeit`魔术命令为此提供了优雅的解决方案。它会自动多次运行指定的代码，并计算平均执行时间，从而得到更精确的结果，尤其适用于微操作的性能评估。

其基本语法如下：
```python
%timeit [要测试的代码]
```
例如，我们想测量对一个包含1万个随机数的列表进行排序所需的时间：
```python
import random
lst = [random.random() for _ in range(10000)]

%timeit sorted(lst)
```
`%timeit`会输出类似以下的结果：
```
14.6 ms ± 742 µs per loop (mean ± std. dev. of 7 runs, 100 loops each)
```
这表示：它运行了7轮，每轮循环执行`sorted(lst)` 100次，最终计算出平均每次执行时间为14.6毫秒，并给出了标准差。

对于单行表达式，可以直接使用`%timeit`。对于多行代码块（例如函数体），可以使用`%%timeit`（双百分号）来测量整个单元格代码的执行时间。

![](img/7da4033f76e59594367f81c7b23c6fd9_15.png)

本节课中我们一起学习了IPython中三个实用的魔术命令：`%run`用于直接运行外部Python脚本；`%paste`用于便捷地执行剪贴板中的代码；`%timeit`用于精确测量代码片段的执行时间。掌握这些命令能显著提升在交互式环境中的开发和调试效率。
