# Python金融量化+业务数据分析：P46：10 数据透视表简介 📊

在本节课中，我们将要学习数据透视表。数据透视表是Excel中一个非常强大的数据分析工具，它能够快速地对数据进行分类汇总和计算。我们将通过一个具体的业务需求，来理解数据透视表的作用、创建方法以及使用时的注意事项。

## 业务需求场景

假设领导给了一张销售数据表，记录了某地区不同品牌、不同车型的销量和收入数据。

领导希望你将这张明细数据表，整理成按品牌汇总销量和收入总和的报表。

接到这个需求后，我们思考如何实现。将明细数据按类别汇总，本质上是一种分类汇总计算。

## 实现汇总的几种方法

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_5.png)

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_7.png)

以下是几种实现该需求的不同思路，我们将逐一分析其优缺点。

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_9.png)

### 方法一：筛选与手动求和

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_11.png)

这种方法最直接。例如，先筛选出“奥迪”品牌的所有行，然后手动选中“销量”列查看Excel状态栏的求和结果（例如96），再将这个结果填写到汇总表的“奥迪-销量”单元格中。对“收入”列和所有其他品牌重复此操作。

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_13.png)

**缺点**：操作繁琐，效率低下，在数据量大或品牌多时尤其耗时，且手动填写容易出错。

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_15.png)

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_17.png)

### 方法二：使用SUMIF函数

我们可以利用刚学过的`SUMIF`函数进行条件求和。这是一个更高效的方法。

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_19.png)

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_21.png)

**操作步骤**：
1.  在汇总表的“销量”单元格（例如`B2`）输入公式：`=SUMIF(原数据表!$C$2:$C$67, A2, 原数据表!$G$2:$G$67)`
    *   `原数据表!$C$2:$C$67`：条件区域，即品牌列。
    *   `A2`：条件，即当前行的品牌（如“奥迪”）。
    *   `原数据表!$G$2:$G$67`：求和区域，即销量列。
    *   `$`符号用于绝对引用，确保公式下拉时区域固定。
2.  将`B2`单元格的公式向下拖动，即可计算出所有品牌的销量总和。
3.  计算收入总和时，只需将求和区域改为收入列（`H`列），公式为：`=SUMIF(原数据表!$C$2:$C$67, A2, 原数据表!$H$2:$H$67)`。

**优点**：相比手动筛选，效率更高，且数据源变化时，只需刷新或重新计算公式即可。
**缺点**：需要编写和修改公式，对多条件或多维度汇总时，公式会变得复杂。

上一节我们介绍了两种传统的汇总方法，本节中我们来看看更强大、更直观的工具——数据透视表。

## 数据透视表：快速分类汇总

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_23.png)

数据透视表可以通过简单的“拖拉拽”操作，快速完成复杂的分类汇总计算。

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_25.png)

### 创建数据透视表

以下是创建数据透视表的核心步骤。

1.  **定位与插入**：将鼠标光标置于源数据区域的任意单元格内。点击【插入】选项卡，选择【数据透视表】。
2.  **设置创建选项**：
    *   **表/区域**：Excel会自动识别并选中连续的数据区域（如`A1:H67`），通常无需修改。
    *   **选择放置数据透视表的位置**：可以选择“新工作表”或“现有工作表”。为方便演示，我们选择“现有工作表”，并点击一个空白单元格（如`J1`）作为透视表的起始位置。
3.  **点击【确定】**，一个空的数据透视表框架和“数据透视表字段”窗格就会出现。

### 配置数据透视表

“数据透视表字段”窗格是配置透视表的核心，下半部分列出了数据源的所有列标题（字段），上半部分有四个区域：

*   **行**：拖入到此区域的字段，将作为表格的行标签（分类依据）。
*   **列**：拖入到此区域的字段，将作为表格的列标签。
*   **值**：拖入到此区域的字段，将进行汇总计算（如求和、计数等）。
*   **筛选器**：拖入到此区域的字段，可对整个透视表进行筛选。

**实现需求的操作**：
1.  将“品牌”字段拖拽到【行】区域。透视表行方向会立即列出所有不重复的品牌。
2.  将“销量”字段拖拽到【值】区域。Excel默认对数值型字段进行“求和”，因此透视表会计算出每个品牌的总销量。
3.  将“收入”字段也拖拽到【值】区域。透视表会同时显示每个品牌的总销量和总收入。

至此，我们通过三次拖拽，就完成了之前需要用函数或多次手动操作才能实现的任务。若要移除某个字段，只需将其从相应区域拖出即可。

### 值字段的计算类型

双击值区域中的字段（如“求和项:销量”），或点击其下拉箭头选择“值字段设置”，可以更改计算类型。

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_27.png)

**可选类型包括**：求和、计数、平均值、最大值、最小值、乘积等。这为数据分析提供了极大的灵活性。例如，你可以轻松将“求和”改为“平均值”，来查看各品牌的平均销量。

## 使用数据透视表的注意事项

并非所有表格都适合创建数据透视表。为了确保数据透视表能正确工作，源数据应满足以下条件：

*   **列名完整且唯一**：每一列都必须有标题，且不能有重复的列名。
*   **避免空行/空列**：数据区域中不要存在完整的空行或空列，这会影响Excel对数据范围的自动识别。
*   **数据格式统一**：尤其是数值型数据，应确保其为数字格式，而非文本格式。文本格式的数字不参与数值计算。
*   **禁用合并单元格**：数据区域中不应使用合并单元格。合并单元格会导致部分数据缺失（被视为空值），严重影响分类汇总的准确性。

## 数据透视表的应用场景

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_29.png)

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_31.png)

了解其优势后，数据透视表适用于以下场景：

![](img/22adcfb71ed37fbd368a8dc4ef6307bb_33.png)

*   **快速分析大量数据**：当数据量庞大，使用公式或手动处理效率低下时。
*   **制作动态报表**：需要频繁根据最新数据生成或更新汇总报表时。
*   **多维度数据分析**：需要从不同角度（如按品牌、按车型、按月份）对数据进行切片、钻取和分析时。
*   **数据探索与洞察**：快速对数据进行分组、汇总、计算，以发现数据背后的模式和趋势。
