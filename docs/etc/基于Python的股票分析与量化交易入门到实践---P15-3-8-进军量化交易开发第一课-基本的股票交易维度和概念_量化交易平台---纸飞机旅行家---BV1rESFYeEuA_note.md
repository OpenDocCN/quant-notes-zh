## 什么是量化交易平台？🤔

上一节我们介绍了股票交易的基础知识，本节中我们来看看量化交易的核心工具——量化交易平台。量化交易平台主要为量化交易研发人员提供必需的功能，包括量化数据、策略框架、回测以及交易接口。如果这些功能全部从头开发，将耗费大量时间，因此成熟的量化交易平台应运而生。

## 国内常见的量化交易平台

以下是国内最早或第一批出现的量化交易平台：

*   **聚宽量化**：成立较早的云平台。
*   **掘金量化**：同样是较早的平台。
*   **BigQuant**：提供AI驱动的量化平台。
*   **米筐**：国内知名的量化平台之一。
*   **TradeBlazer（交易开拓者）**：主要专注于期货交易。

这些平台目前均已部分商业化，是初学者和开发者常用的选择。

## 聚宽量化平台简介 🚀

在众多平台中，我们选择聚宽量化平台作为主要介绍对象。聚宽是一个面向量化爱好者和初学者的云平台，旨在帮助用户快速便捷地使用量化交易策略。

以下是聚宽平台的主要特点：

![](img/23613b3b4aeb7848f05d54465ff04da8_1.png)

*   **数据历史跨度长**：提供从2005年至今的股票数据。
*   **支持Tick级数据**：Tick数据是比秒更细的毫秒级交易记录，是最高精度的交易数据，通常需要付费获取。
*   **支持多交易品种**：涵盖A股、港股、美股、日股、期货、债券等。
*   **提供健全的Python API**：兼容Python 2和Python 3。
*   **支持实盘交易**：提供高速实盘交易接口。
*   **社区活跃**：拥有活跃的线上社区和丰富的学习资源，便于交流和学习。

## 聚宽平台在线使用指南 🖥️

接下来，我们来看看如何在聚宽平台上进行在线操作。使用聚宽平台通常遵循以下步骤：注册账号、试用申请、新建策略、策略回测，最后进行模拟交易。

![](img/23613b3b4aeb7848f05d54465ff04da8_3.png)

以下是平台核心功能区域的简要说明：

1.  **研究环境**：用于数据分析和策略研究的交互式环境。
2.  **策略库**：平台提供了一些编写好的策略示例，从简单到复杂，供用户学习和运行。
3.  **策略编写与回测**：用户可以创建自己的策略，并进行历史数据回测，评估策略表现。
4.  **模拟交易**：在回测通过后，可以将策略投入模拟交易，观察其在模拟市场环境中的表现。

![](img/23613b3b4aeb7848f05d54465ff04da8_5.png)
