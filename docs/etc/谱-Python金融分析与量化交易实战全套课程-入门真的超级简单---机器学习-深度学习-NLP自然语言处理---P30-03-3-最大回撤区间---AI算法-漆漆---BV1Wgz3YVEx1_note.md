在本节课中，我们将要学习一个重要的风险指标——最大回撤。它用于衡量投资策略在特定时间段内可能面临的最糟糕情况，即资产净值从峰值到谷底的最大跌幅。理解最大回撤对于评估策略的风险至关重要。

![](img/291219e2b7adcde8c080aeec33571d71_3.png)

## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何量化策略的风险。最大回撤描述的是在投资过程中，资产价值从历史最高点到之后最低点的最大百分比跌幅。它衡量的是投资者可能经历的“最惨”时期。

具体来说，它寻找的是在整个投资周期内，哪一段连续的下跌区间造成的损失比例最大。这个指标不直接评价策略的盈利能力，而是评估其风险水平。例如，即使一个策略最终盈利，但过程中可能经历一段非常剧烈的下跌，这段时期就是最大回撤区间。

## 最大回撤的计算原理

理解了最大回撤的概念后，我们来看看如何用数学公式来描述它。最大回撤的计算公式如下：

**最大回撤 = max( (P_i - P_j) / P_i )**

其中：
*   `P_i` 代表第 `i` 天的资产价格（或净值）。
*   `P_j` 代表第 `i` 天之后（`j > i`）的某一天资产价格。
*   公式的含义是：遍历所有可能的 `i` 和 `j`（`j` 在 `i` 之后），计算从 `i` 日到 `j` 日的跌幅比例，然后取其中的最大值。

![](img/291219e2b7adcde8c080aeec33571d71_5.png)

这个计算过程旨在找出资产价值“缩水”最严重的那一段连续时期。

![](img/291219e2b7adcde8c080aeec33571d71_7.png)

## 使用Python计算最大回撤

![](img/291219e2b7adcde8c080aeec33571d71_9.png)

理论部分已经清晰，接下来我们通过Python代码来实践如何计算最大回撤。我们将使用Pandas库来高效地处理数据。

![](img/291219e2b7adcde8c080aeec33571d71_11.png)

首先，我们需要一个代表资产每日净值的时间序列数据。这里假设我们已经有了一个名为 `df[‘net_value’]` 的Pandas Series。

![](img/291219e2b7adcde8c080aeec33571d71_13.png)

计算最大回撤的核心思路是：
1.  计算截至每一天的累计最大值（历史峰值）。
2.  用这个历史峰值序列减去当日的实际净值序列，得到每日的潜在回撤值。
3.  将这个潜在回撤值除以对应的历史峰值，得到每日的回撤比例。
4.  在所有回撤比例中找出最大值，即为最大回撤。

![](img/291219e2b7adcde8c080aeec33571d71_15.png)

![](img/291219e2b7adcde8c080aeec33571d71_17.png)

以下是计算步骤的代码实现：

![](img/291219e2b7adcde8c080aeec33571d71_19.png)

```python
import pandas as pd

![](img/291219e2b7adcde8c080aeec33571d71_21.png)

![](img/291219e2b7adcde8c080aeec33571d71_23.png)

![](img/291219e2b7adcde8c080aeec33571d71_25.png)

# 假设 df['net_value'] 是资产的每日净值序列
# 1. 计算截至每一天的累计最大值（历史峰值）
cumulative_max = df['net_value'].cummax()

![](img/291219e2b7adcde8c080aeec33571d71_27.png)

# 2. 计算每日的回撤值（历史峰值 - 当日净值）
drawdown_value = cumulative_max - df['net_value']

![](img/291219e2b7adcde8c080aeec33571d71_29.png)

![](img/291219e2b7adcde8c080aeec33571d71_31.png)

# 3. 计算每日的回撤比例
drawdown_ratio = drawdown_value / cumulative_max

# 4. 找出最大的回撤比例，即最大回撤
max_drawdown = drawdown_ratio.max()

![](img/291219e2b7adcde8c080aeec33571d71_33.png)

![](img/291219e2b7adcde8c080aeec33571d71_35.png)

![](img/291219e2b7adcde8c080aeec33571d71_37.png)

print(f"最大回撤为：{max_drawdown:.2%}")
```

**代码解释**：
*   `.cummax()`: 这是一个Pandas函数，返回序列截至当前行的最大值。它生成了我们公式中的 `P_i`（对于每一天，`P_i` 是到该天为止的最高价）。
*   `drawdown_value`: 对应公式中的 `(P_i - P_j)`，这里 `P_j` 就是 `df[‘net_value’]`。
*   `drawdown_ratio`: 对应完整的公式 `(P_i - P_j) / P_i`。
*   `.max()`: 最终找出所有回撤比例中的最大值。

![](img/291219e2b7adcde8c080aeec33571d71_39.png)

![](img/291219e2b7adcde8c080aeec33571d71_41.png)

## 结果解读与应用

计算完成后，我们得到了一个百分比数值，例如 `0.307`（即30.7%）。这表示在该策略的回测周期内，投资者可能面临的最大资产跌幅约为30.7%。

在策略回测报告中，最大回撤通常与收益率、夏普比率等指标一同展示。一个优秀的策略不仅要有可观的收益，其最大回撤也应在投资者可承受的风险范围之内。通过分析最大回撤区间发生的具体时间，投资者还可以结合当时的市场环境，进一步理解策略的风险来源。
