## 聚类是什么？🤔

上一节我们介绍了有监督学习，其特点是数据带有标签。本节中，我们来看看无监督学习中的聚类问题。

聚类要解决的问题与分类类似，但数据本身没有标签。我们的目标是将杂乱的数据分成若干组（簇），使得**相似的数据点被分到同一组**。

例如，观察右侧的散点图。原始数据并没有颜色标记来区分类别。聚类算法会根据数据点之间的相似度（如距离），自动将相似的点归为一类，最终可能得到绿色、红色和蓝色三个不同的簇。

虽然聚类算法的原理看似简单，但它面临一些独特的挑战：
*   **评估困难**：由于没有真实标签，很难像有监督学习那样通过预测值与真实值的对比来评估聚类结果的好坏。
*   **参数调节困难**：例如，改变聚类数量（K值）会得到不同的结果，但缺乏“标准答案”来判断哪个结果更优。

接下来，我们将深入探讨第一个经典聚类算法。

## K-Means算法核心概念 🧠

K-Means算法是聚类算法中最简单、最实用的方法之一。理解它需要掌握几个核心概念。

![](img/9814c190c582adfe3dcddfdd38221bef_4.png)

### 1. K值（聚类数量）

![](img/9814c190c582adfe3dcddfdd38221bef_6.png)

K-Means算法需要我们预先指定一个参数：**K值**。K值代表我们希望算法将数据聚合成多少个簇。

*   如果设定 `K=3`，算法会将所有数据点分成3个簇。
*   如果设定 `K=4`，则会分成4个簇，以此类推。

![](img/9814c190c582adfe3dcddfdd38221bef_8.png)

**公式表示**：`K` 是一个需要用户指定的正整数。

![](img/9814c190c582adfe3dcddfdd38221bef_10.png)

### 2. 质心

![](img/9814c190c582adfe3dcddfdd38221bef_12.png)

**质心**是K-Means算法中一个至关重要的概念。它代表一个簇的中心位置，通常由该簇内所有数据点在各个维度上的均值计算得出。

例如，在二维空间中，一个簇的质心坐标 `(C_x, C_y)` 可以通过以下方式计算：
*   `C_x = (x1 + x2 + ... + xn) / n`
*   `C_y = (y1 + y2 + ... + yn) / n`

![](img/9814c190c582adfe3dcddfdd38221bef_14.png)

其中，`(x_i, y_i)` 是属于该簇的第 `i` 个数据点，`n` 是该簇中数据点的总数。质心在算法的迭代优化过程中起到关键作用。

### 3. 距离度量

如何判断两个数据点是否“相似”？我们通过计算它们之间的**距离**来衡量。距离越小，通常认为相似度越高。

对于K-Means算法，最常用的距离度量是**欧氏距离**。对于二维空间中的两点 `P1(x1, y1)` 和 `P2(x2, y2)`，其欧氏距离计算公式为：

**公式**：`distance = sqrt((x2 - x1)^2 + (y2 - y1)^2)`

**重要提示**：在使用距离度量前，通常需要对数据进行**标准化**（如归一化）。这是因为如果不同特征（维度）的数值范围差异巨大（例如一个特征值在0~1，另一个在100~1000），数值范围大的特征会主导距离计算，导致结果失真。标准化能将所有特征缩放到相近的数值范围（如0~1或均值为0、方差为1），确保每个特征对距离计算的贡献是均衡的。

![](img/9814c190c582adfe3dcddfdd38221bef_16.png)

![](img/9814c190c582adfe3dcddfdd38221bef_18.png)

### 4. 优化目标

与大多数机器学习算法一样，K-Means通过优化一个目标函数来求解。它的目标被称为**簇内误差平方和**。

![](img/9814c190c582adfe3dcddfdd38221bef_20.png)

**优化目标**：最小化所有数据点到其所属簇质心的距离平方和。

![](img/9814c190c582adfe3dcddfdd38221bef_22.png)

**公式表示**：
`J = Σ（i=1 到 K） Σ（x 属于 C_i） || x - μ_i ||^2`

**代码描述（伪代码）**：
```python
# 假设有K个簇，centroids是质心列表，clusters是每个簇的数据点集合
total_distance = 0
for i in range(K):
    for point in clusters[i]:
        # 计算点到其质心的欧氏距离平方
        distance = euclidean_distance(point, centroids[i]) ** 2
        total_distance += distance
# 目标是最小化 total_distance
```

**通俗解释**：算法希望每个簇都“紧密”地围绕其质心。如果一个点离当前簇的质心很远，但离另一个簇的质心更近，那么在迭代过程中，算法会倾向于将这个点重新分配到更近的那个簇中，从而降低总体的距离平方和。

本节课中我们一起学习了K-Means聚类算法的核心概念。我们了解到聚类是一种无监督学习方法，目标是将相似数据分组。K-Means算法通过指定K值、计算质心、利用距离度量相似性，并不断优化簇内紧密程度来完成聚类任务。理解这些基础概念是掌握和运用K-Means算法的关键。
