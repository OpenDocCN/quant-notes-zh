## 如何使用Matplotlib？

![](img/c56b9a503bf1091ae6d26d7d7b2181db_3.png)

我们主要使用Jupyter Notebook进行演示。首先需要导入Matplotlib库中用于绘图的子模块`pyplot`。

```python
import matplotlib.pyplot as plt
```

![](img/c56b9a503bf1091ae6d26d7d7b2181db_5.png)

![](img/c56b9a503bf1091ae6d26d7d7b2181db_7.png)

`plt.plot()`函数用于绘制图形，而`plt.show()`函数则用于将绘制的图形展示出来。

![](img/c56b9a503bf1091ae6d26d7d7b2181db_9.png)

```python
plt.plot()
plt.show()
```

运行上述代码会显示一个空的坐标轴图像，因为`plot()`函数没有传入任何数据。

---

## 绘制折线图

`plot()`函数最基本的功能是绘制折线图。它需要传入两个主要参数：X轴坐标列表和Y轴坐标列表。

```python
# 绘制一条直线
plt.plot([1, 2, 3, 4], [2, 4, 6, 8])
plt.show()
```

这段代码会将点(1,2)、(2,4)、(3,6)、(4,8)连接起来，形成一条直线。如果修改Y轴数据，就会得到一条折线。

```python
# 绘制一条折线
plt.plot([1, 2, 3, 4], [2, 3, 5, 7])
plt.show()
```

![](img/c56b9a503bf1091ae6d26d7d7b2181db_11.png)

除了列表，也可以传入NumPy数组作为参数。

![](img/c56b9a503bf1091ae6d26d7d7b2181db_13.png)

## 自定义图表样式

`plot()`函数还有第三个可选参数，它是一个格式字符串，用于控制折线图的线条样式、标记点形状和颜色。

![](img/c56b9a503bf1091ae6d26d7d7b2181db_15.png)

以下是格式字符串的组成部分：
*   **线条样式**：控制线条是实线、虚线还是点线。
*   **标记点**：控制数据点的显示形状。
*   **颜色**：控制线条和标记点的颜色。

![](img/c56b9a503bf1091ae6d26d7d7b2181db_17.png)

例如，`‘o-’`表示用实线连接，并在每个数据点处显示圆形标记。`‘r--’`表示用红色虚线绘制，不显示标记点。

```python
# 绘制带圆形标记的实线
plt.plot([1, 2, 3, 4], [2, 4, 6, 8], ‘o-’)
plt.show()
```

![](img/c56b9a503bf1091ae6d26d7d7b2181db_19.png)

### 常见样式参数

![](img/c56b9a503bf1091ae6d26d7d7b2181db_21.png)

以下是部分常用的样式字符：

**线条样式**
*   `‘-’`：实线
*   `‘--’`：虚线
*   `‘:’`：点线

**标记点**
*   `‘o’`：圆形
*   `‘v’`：倒三角形
*   `‘^’`：正三角形
*   `‘s’`：正方形
*   `‘*’`：星形
*   `‘+’`：加号
*   `‘x’`：叉号
*   `‘d’`：菱形

![](img/c56b9a503bf1091ae6d26d7d7b2181db_23.png)

**颜色**
*   `‘b’`：蓝色
*   `‘g’`：绿色
*   `‘r’`：红色
*   `‘c’`：青色
*   `‘m’`：洋红色
*   `‘y’`：黄色
*   `‘k’`：黑色
*   `‘w’`：白色

![](img/c56b9a503bf1091ae6d26d7d7b2181db_25.png)

你也可以使用关键字参数来分别指定这些属性，这样代码可读性更高。

```python
plt.plot([1, 2, 3, 4], [2, 4, 6, 8], marker=‘o’, linestyle=‘-’, color=‘red’)
plt.show()
```

![](img/c56b9a503bf1091ae6d26d7d7b2181db_27.png)

### 在同一张图中绘制多条线

只需多次调用`plt.plot()`函数，即可在同一坐标轴内叠加绘制多条折线。

![](img/c56b9a503bf1091ae6d26d7d7b2181db_35.png)

```python
# 绘制两条线
plt.plot([1, 2, 3, 4], [1, 4, 9, 16], ‘ro-’) # 红色圆点实线
plt.plot([1, 2, 3, 4], [2, 4, 6, 8], ‘b^--’) # 蓝色三角虚线
plt.show()
```

Matplotlib还提供了许多其他高级功能，例如设置坐标轴标签、标题、图例、绘制散点图、柱状图等，这些内容将在后续章节中详细介绍。
