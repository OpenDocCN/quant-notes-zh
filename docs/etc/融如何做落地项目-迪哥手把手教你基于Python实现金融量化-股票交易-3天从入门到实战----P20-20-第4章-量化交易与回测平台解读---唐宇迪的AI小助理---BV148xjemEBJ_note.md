## 什么是量化交易？

上一节我们介绍了课程概述，本节中我们来看看量化交易到底做了一件什么事。

传统炒股需要投资者长时间盯盘，分析各种指标。这种做法存在两个主要问题：第一，人的主观性可能导致判断不准确；第二，个人的精力、时间和计算能力有限，无法分析海量的历史数据和全局市场。

量化交易的目标同样是赚钱，但方法不同。它将具体的交易策略交给计算机来制定。其本质是：基于股票的历史数据，设计交易策略，并通过回测来验证这些策略在历史数据上的表现，目标是找到能使收益最大化的方法。

**量化交易的核心公式**可以概括为：
`最大化收益 = 策略(历史数据)`

简而言之，量化交易就是让计算机在历史数据中执行数据挖掘任务，找出能赚钱的有效策略。

## 量化交易需要哪些核心技能？

理解了量化交易的本质后，我们来看看实现它需要掌握哪些核心技能。

量化交易是一个交叉学科，但金融学知识只需了解即可，重点应放在与计算机和数据处理相关的技能上。以下是实现量化交易所需的核心技能：

![](img/4aaf511219f2b3e28c77a10cfca1b17a_7.png)

*   **机器学习算法**：用于基于数据预测未来走势或结果。
*   **统计方法**：用于计算和分析数据中的各项指标，挖掘有价值的信息。
*   **编程实践 (如Python)**：用于将策略和方法论转化为可执行的代码，进行数据处理和回测。

在量化交易中，没有绝对的必备工具。任何能帮助你处理数据、分析数据并最终实现收益最大化的工具都是好工具。你可以将量化交易理解为数据挖掘在金融领域的一个热门应用。
