## NumPy模块简介 🧠

首先大致介绍一下NumPy模块。

你可以将NumPy理解为一个**一维或多维的数组**。如果你有其他编程语言（如C、Java）的经验，一定知道数组的概念。如果只接触过Python，Python中没有传统意义上的数组，但我们学习过列表。如果你不清楚数组是什么，可以暂时将数组视为一种特殊的列表，或者将列表视为一种特殊的数组。无论是列表还是数组，它们都表示一种**数据结构**，即一种可以装载数据的**容器**。

因此，NumPy模块对应的容器就叫做**数组**，这个数组可以装载相关数据。暂时，我们先把NumPy模块当成一个一维或多维的数组。

那么，如何理解维度的概念呢？例如，我们定义一个列表 `a_list = [1, 2, 3]`，这个列表可以暂且称为一维列表。什么是二维列表呢？例如 `a_list = [[1, 2, 3], [4, 5, 6]]`，这就是列表套列表，称为二维列表。三维及更高维度的列表也是类似的原理（列表套列表再套列表）。NumPy数组同理，也有一维数组和多维数组。

![](img/f741484e0732f257adbdf084945baa2f_15.png)

数组与列表肯定存在区别，这个区别稍后会介绍。目前，你只需将NumPy模块视为一个一维或多维的数组即可。

![](img/f741484e0732f257adbdf084945baa2f_17.png)

这个一维或多维的数组主要装载**数值型数据**。因此，NumPy模块的重点操作在于对数值进行运算。在后期数据分析中，如果需要对一组数值进行某种形式的计算，我们可以先将待计算的数值装载到NumPy数组中，再基于NumPy数组的相关特性进行运算。

所以，NumPy模块的第一个重点是**数值计算**。这意味着，如果我们想使用NumPy数组作为容器，那么数组中存放的数据必须是数值型数据。

其次，NumPy模块也是Python**科学计算库的基础**。这意味着NumPy模块中已经为我们封装好了各种各样基于科学计算的操作和工具类。例如，后期如果需要对一组数据进行反三角函数的运算，我们不需要手动实现，直接调用NumPy模块中封装好的工具类即可。

**NumPy模块介绍总结：**
1.  它是一个容器，装载的数据是**数值型数据**，主要用于对数值型数据进行运算。
2.  它是Python**科学计算库的基础**，封装了各种科学计算工具类，可直接对数组中的数据进行计算。

## 创建NumPy数组的三种方式 🛠️

介绍完NumPy模块后，接下来我们看看如何使用NumPy数组。首先，你需要创建NumPy数组。

![](img/f741484e0732f257adbdf084945baa2f_19.png)

### 方式一：使用 `np.array()` 从列表创建

![](img/f741484e0732f257adbdf084945baa2f_21.png)

这是最基础、最直接的创建方式。

首先，需要导入NumPy模块。业界约定俗成地为其起别名为 `np`。

![](img/f741484e0732f257adbdf084945baa2f_23.png)

![](img/f741484e0732f257adbdf084945baa2f_25.png)

```python
import numpy as np
```

![](img/f741484e0732f257adbdf084945baa2f_27.png)

模块导入后，即可创建数组。使用 `np.array()` 函数可以创建一维或多维数组。

![](img/f741484e0732f257adbdf084945baa2f_29.png)

**创建一维数组：**
```python
arr1 = np.array([1, 2, 3])
print(arr1)
# 输出：array([1, 2, 3])
```

**创建二维数组：**
```python
arr2 = np.array([[1, 2, 3], [4, 5, 6]])
print(arr2)
# 输出：
# array([[1, 2, 3],
#        [4, 5, 6]])
```

`np.array()` 函数的第一个参数 `object` 是数据源（如列表），第二个参数 `dtype` 可以指定数组元素的数据类型。如果不指定，NumPy会自动推断。

![](img/f741484e0732f257adbdf084945baa2f_31.png)

![](img/f741484e0732f257adbdf084945baa2f_33.png)

通过这种方式，我们可以创建任意维度的数组，但最常用的是**一维、二维和三维数组**，因为维度再高就不便于人类直观理解了。

![](img/f741484e0732f257adbdf084945baa2f_35.png)

#### 数组与列表的核心区别

现在，我们掌握了通过列表创建数组的基本形式。接下来，探讨一个关键问题：数组与列表到底有什么区别？

我们通过一个例子来观察：
```python
arr_mixed = np.array([1, 2.2, ‘three’])
print(arr_mixed)
# 输出：array([‘1’, ‘2.2’, ‘three’], dtype=‘<U32’)
```

![](img/f741484e0732f257adbdf084945baa2f_37.png)

![](img/f741484e0732f257adbdf084945baa2f_39.png)

你会发现，原始数据包含整型、浮点型和字符串型，但最终数组中的所有元素都被统一转换成了**字符串类型**。这在Python列表中是不会发生的，因为列表允许元素类型不同。

这说明：**NumPy数组中存储的数组元素类型必须是统一的**。如果初始元素类型不同，NumPy会尝试将它们转换为同一类型。转换遵循一定的类型优先级（例如：字符串 > 浮点型 > 整型）。了解这一点对后续数据处理非常重要。

### 方式二：使用 `plt.imread()` 从图像文件创建

接下来，我们看看NumPy数组的另一种创建方式。这里我们引入一个需求：将外部的一张图片读取并加载到NumPy数组中，然后尝试改变数组元素的数值，观察对原始图片的影响。

![](img/f741484e0732f257adbdf084945baa2f_41.png)

![](img/f741484e0732f257adbdf084945baa2f_43.png)

![](img/f741484e0732f257adbdf084945baa2f_45.png)

NumPy数组是一个容器，数据装载到不同的容器中，会受到容器特性的影响。例如，列表和字典对数据的组织形式不同，提供的操作方法也不同。NumPy数组的特性使得我们可以方便地对装载的数值数据进行批量运算，包括图像数据（图像本质上是二进制数值数据）。

以下是操作步骤：

![](img/f741484e0732f257adbdf084945baa2f_47.png)

![](img/f741484e0732f257adbdf084945baa2f_49.png)

1.  **导入必要的模块**：这里需要用到三剑客中的 `matplotlib` 模块来读取和显示图像。
2.  **读取图像到数组**：使用 `plt.imread()` 函数。
3.  **显示和修改图像数据**：通过修改数组值来改变图像。

![](img/f741484e0732f257adbdf084945baa2f_51.png)

```python
import matplotlib.pyplot as plt

![](img/f741484e0732f257adbdf084945baa2f_53.png)

# 读取图片，返回一个NumPy数组
image_array = plt.imread(‘1.jpg’)
print(image_array.shape) # 查看数组形状，例如 (高度, 宽度, 通道数)
# 显示原始图片
plt.imshow(image_array)
plt.show()

![](img/f741484e0732f257adbdf084945baa2f_55.png)

![](img/f741484e0732f257adbdf084945baa2f_57.png)

![](img/f741484e0732f257adbdf084945baa2f_59.png)

# 修改数组中的数值（例如，将所有像素值减去100，注意数据类型限制）
modified_array = image_array - 100
# 必须确保值在合理范围内，例如对于uint8类型，可使用np.clip
# modified_array = np.clip(image_array - 100, 0, 255).astype(‘uint8’)

![](img/f741484e0732f257adbdf084945baa2f_61.png)

![](img/f741484e0732f257adbdf084945baa2f_63.png)

# 显示修改后的图片
plt.imshow(modified_array)
plt.show()
```

`plt.imread()` 读取图片后，会返回一个**多维NumPy数组**（通常是三维，代表图片的高度、宽度和颜色通道）。数组中存储的就是图片的像素数据。通过直接对这个数组进行数学运算（如加减乘除），就能实现像调整亮度、对比度等简单的图像处理效果。这揭示了图像处理软件（如美图秀秀）的部分原理：它们本质上也是对图像像素数据进行特定的数学运算。

![](img/f741484e0732f257adbdf084945baa2f_65.png)

![](img/f741484e0732f257adbdf084945baa2f_67.png)

因此，`plt.imread()` 是创建NumPy数组的第二种方式，它创建的是一个包含图像数据的多维数组。

### 方式三：使用NumPy内置函数快速创建

最后，我们介绍第三种创建数组的方式：使用NumPy提供的一系列内置函数，快速生成具有特定规律或形状的数组。

![](img/f741484e0732f257adbdf084945baa2f_75.png)

以下是几个常用函数：

![](img/f741484e0732f257adbdf084945baa2f_77.png)

**1. `np.zeros()` / `np.ones()`：创建全0或全1数组**
可以指定数组的形状 `shape`。
```python
# 创建一个3行4列的全0二维数组
arr_zeros = np.zeros((3, 4))
print(arr_zeros)

![](img/f741484e0732f257adbdf084945baa2f_79.png)

# 创建一个形状为 (2, 3, 4) 的全1三维数组
arr_ones = np.ones((2, 3, 4))
print(arr_ones)
```

![](img/f741484e0732f257adbdf084945baa2f_81.png)

**2. `np.linspace()`：创建等差数列数组**
返回**一维数组**。参数：起始值 `start`，结束值 `stop`，元素个数 `num`。
```python
# 在0到100之间生成20个等间隔的数
arr_lin = np.linspace(0, 100, 20)
print(arr_lin)
```

![](img/f741484e0732f257adbdf084945baa2f_83.png)

**3. `np.arange()`：创建等差数组（类似Python的range）**
返回**一维数组**。参数：起始值 `start`，结束值 `stop`，步长 `step`。
```python
# 从10开始，到50结束（不包含50），步长为2
arr_arange = np.arange(10, 50, 2)
print(arr_arange)
```

![](img/f741484e0732f257adbdf084945baa2f_85.png)

**4. `np.random.randint()`：创建随机整数数组**
可以指定随机数范围和数组形状 `size`。
```python
# 创建一个5行3列的二维数组，元素为0到100之间的随机整数
arr_rand = np.random.randint(0, 100, size=(5, 3))
print(arr_rand)
```

**这些函数的特点：**
*   `np.zeros()` 和 `np.ones()` 可以通过 `shape` 参数创建任意维度的数组。
*   `np.linspace()` 和 `np.arange()` 主要用于创建**一维**的等差数组。
*   `np.random` 下的函数（如 `randint`）用于创建随机数数组。
