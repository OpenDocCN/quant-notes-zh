### 资本资产定价模型

![](img/54a381dfcf5b537c5d17b48318f224aa_13.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_15.png)

资本资产定价模型（CAPM）是金融学的基础模型之一。它描述了一个投资组合或单一资产的预期收益。

![](img/54a381dfcf5b537c5d17b48318f224aa_17.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_19.png)

对于一个投资组合，其总权重之和为1。市场组合（如沪深300指数）代表了整个市场的表现。

![](img/54a381dfcf5b537c5d17b48318f224aa_21.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_23.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_25.png)

对于单一股票i，CAPM模型表述为：
`R_i = β_i * R_m + α_i`
其中：
*   `R_i` 是股票i在时刻t的收益率。
*   `R_m` 是市场在时刻t的收益率。
*   `β_i` 是股票i的贝塔系数，衡量其相对于市场的波动性。
*   `α_i` 是阿尔法值，代表超额收益。

![](img/54a381dfcf5b537c5d17b48318f224aa_27.png)

相信市场有效假说的人认为，从长期看，阿尔法值的期望为零，即 `E[α_i] = 0`。

### 机器学习的视角

现在，我们暂时忘记金融理论，纯粹从机器学习的角度来看这个公式。

如果我们想预测招商银行股票每天的收益率`R_i`，我们可以将其视为我们想要预测的目标变量`Y`。那么，CAPM模型实际上做了一个非常强的假设：**影响`Y`的因素只有一个，即市场收益率`R_m`，并且它们之间的关系是线性的**。

用机器学习的通用公式表示就是：
`Y = f(X) + ε`
在CAPM中，`f(X)`被特指为`β * X`（线性函数），`X`就是`R_m`，`ε`就是`α_i`（期望为零的噪声）。

这个视角的转变至关重要。它意味着：
1.  我们可以质疑`X`是否只有市场收益这一个变量。我们可以加入更多变量，如移动平均线、动量、交易量等，此时`X`就从一个标量变成了一个特征向量。
2.  我们可以质疑`f`是否一定是线性函数。它可以是支持向量机（SVM）、随机森林等任何非线性模型。

因此，**技术分析本质上就是寻找和构建这些额外的特征变量`X`，而机器学习就是用来学习特征`X`与目标`Y`之间复杂函数关系`f`的工具**。

### 技术分析与特征构建

![](img/54a381dfcf5b537c5d17b48318f224aa_29.png)

技术分析认为，可以从历史价格和交易量数据中提取出有预测能力的特征（`X`）。特征的选择是一门艺术，可以基于经验，也可以使用遗传算法等自动选择。

技术分析通常在以下情况更有效：
*   **多指标组合**：使用多个特征比单一特征预测能力更强。
*   **短时间尺度**：技术分析更适用于日内或短期的预测，长时间尺度的预测效力会下降。
*   **异类检测**：寻找与大盘走势不一致的股票，这些地方可能存在套利机会。

与技术分析相对的是基本面分析，它更适用于长时间尺度。两者并无优劣，取决于交易的时间范围。

常见的技朮指标（即特征`X`的例子）包括：
*   **动量**：`Momentum_t = (Price_t / Price_{t-n}) - 1`，反映趋势强度。
*   **简单移动平均**：`SMA_t = (Price_t + Price_{t-1} + ... + Price_{t-n+1}) / n`，反映平滑后的价格趋势。
*   **布林带**：基于移动平均和标准差构造的通道，用以识别超买超卖状态。

在将多个特征输入模型前，**必须进行归一化**，以消除不同特征值范围差异带来的影响。常用的方法是Z-score归一化：`(值 - 均值) / 标准差`。

了解了如何构建特征`X`后，我们来看看如何学习函数`f`。

### 基本框架

监督学习的通用框架是：`Y = f(X) + ε`。我们的任务就是从历史数据中学习`f`。

![](img/54a381dfcf5b537c5d17b48318f224aa_43.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_45.png)

### 从简单到复杂

**最近邻**
最简单的方法是最近邻搜索：对于一个新的`X`，在历史数据中找到最相似的`X`，将其对应的`Y`作为预测值。这种方法在特征维度低时有效，但会受“维数灾难”影响。

![](img/54a381dfcf5b537c5d17b48318f224aa_47.png)

**线性模型**
线性模型假设`f`是线性的，即 `Y = β_0 + β_1*X_1 + β_2*X_2 + ...`。其目标是找到一组参数`β`，使得预测值与真实值之间的误差（如均方误差MSE）最小。`scikit-learn`可以轻松实现这一点。

![](img/54a381dfcf5b537c5d17b48318f224aa_49.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_51.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_53.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_55.png)

**非线性模型**
当数据关系并非线性时，需要使用非线性模型，如多项式回归、SVM、神经网络等。模型越复杂，拟合能力越强。

### 过拟合与偏差-方差权衡

这里引出一个核心概念：**偏差-方差权衡**。
*   **偏差**：模型在训练数据上的误差。简单模型（如线性模型）偏差高，复杂模型偏差低。
*   **方差**：模型对于不同训练数据的敏感度。简单模型方差低，复杂模型方差高。

**过拟合**是指模型在训练数据上表现极好（偏差很低），但在未见过的测试数据上表现很差（方差很高）。我们追求的是在训练误差和测试误差之间取得平衡，即模型的**泛化能力**好。

![](img/54a381dfcf5b537c5d17b48318f224aa_57.png)

![](img/54a381dfcf5b537c5d17b48318f224aa_59.png)

机器学习的关键之一就是通过交叉验证等方法，选择一个复杂度合适的模型，以达到最佳的泛化性能。
