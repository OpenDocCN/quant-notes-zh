在本节课中，我们将要学习统计学中一个至关重要的概念——抽样分布。我们将了解几种重要的抽样分布（卡方分布、T分布、F分布）以及核心的“中心极限定理”。理解这些分布是进行统计推断和数据分析的基础。

## 抽样分布概述

推断统计是根据样本数据去推测总体数据。在抽样过程中，样本统计量（如均值）的分布规律被称为抽样分布。我们上节课介绍了中心极限定理，本节将详细探讨几种具体的抽样分布。

## 卡方分布

![](img/d327b740d9bc530f9c919974a963f3ca_1.png)

卡方分布主要用于解决与方差相关的问题，在后续的方差分析中会用到。

### 卡方分布的定义

若随机变量 `X1, X2, ..., Xn` 相互独立，且每个 `Xi` 都服从标准正态分布 `N(0,1)`，则它们的平方和服从自由度为 `n` 的卡方分布。

![](img/d327b740d9bc530f9c919974a963f3ca_3.png)

**公式表示为：**
```
χ² = X1² + X2² + ... + Xn²
```
其中，`χ² ~ χ²(n)`，`n` 为自由度。

### 如何理解卡方分布

我们可以从正态分布推导出卡方分布。假设总体 `X` 服从正态分布 `N(μ, σ²)`，我们可以将其转化为标准正态分布：
```
Z = (X - μ) / σ
```
这里的 `Z` 就服从标准正态分布 `N(0,1)`。如果我们将多个这样的 `Z` 值取平方并求和，得到的结果就服从卡方分布。

**核心概念：卡方分布是多个独立标准正态分布变量的平方和。**

### 卡方分布的特点

以下是卡方分布的几个关键特性：

*   **变量值始终为正**：因为它是平方和，所以取值不可能为负。
*   **分布形状取决于自由度**：通常为不对称的正偏态分布。但随着自由度 `n` 不断增大，其分布形态会逐渐趋近于正态分布。
*   **期望与方差**：卡方分布的期望值 `E(χ²) = n`，方差 `Var(χ²) = 2n`。

### 卡方分布的图形

随着自由度 `n` 的增加（例如从 `n=1` 到 `n=20`），卡方分布的图形会从高度偏态逐渐变得对称，并向正态分布靠拢。在统计学中，通常认为样本量大于等于30时，分布可近似视为正态。

## T分布

T分布，又称学生T分布，其形态类似于正态分布，是一种对称分布。它主要用于处理小样本数据。

### T分布的定义

若随机变量 `X` 服从标准正态分布 `N(0,1)`，随机变量 `Y` 服从自由度为 `n` 的卡方分布 `χ²(n)`，且 `X` 与 `Y` 相互独立，则随机变量 `t` 服从自由度为 `n` 的T分布。

![](img/d327b740d9bc530f9c919974a963f3ca_5.png)

**公式表示为：**
```
t = X / √(Y/n)
```

### 如何理解T分布

我们可以将公式中的 `Y`（卡方分布）理解为多个标准正态分布平方和的平均值（即均方）。因此，T分布可以看作是**标准正态分布除以均方的根**。

**核心应用：** 当样本量较小（通常 `n < 30`）且总体服从正态分布时，样本均值的标准化统计量服从T分布。当样本量足够大（`n ≥ 30`）时，T分布非常接近标准正态分布，此时可直接使用正态分布。

![](img/d327b740d9bc530f9c919974a963f3ca_7.png)

### T分布的图形

T分布的图形与标准正态分布类似，但尾部更厚（即极端值出现的概率更高）。随着自由度 `n` 的增大，T分布曲线逐渐变瘦，无限趋近于标准正态分布曲线。

## F分布

![](img/d327b740d9bc530f9c919974a963f3ca_9.png)

F分布主要用于比较两个方差，在方差分析和线性回归分析中有着重要应用。

![](img/d327b740d9bc530f9c919974a963f3ca_11.png)

### F分布的定义

设随机变量 `Y` 与 `Z` 相互独立，且 `Y` 服从自由度为 `m` 的卡方分布 `χ²(m)`，`Z` 服从自由度为 `n` 的卡方分布 `χ²(n)`，则随机变量 `X` 服从第一自由度为 `m`、第二自由度为 `n` 的F分布。

**公式表示为：**
```
F = (Y/m) / (Z/n)
```
其中，`F ~ F(m, n)`。

### 如何理解F分布

F分布本质上是**两个独立的卡方分布除以各自自由度后的比值**。它刻画了两个方差估计量之间的相对大小关系。

### F分布的图形

F分布通常为右偏态分布。其图形形态由两个自由度 `m` 和 `n` 共同决定。随着自由度的增大，分布也逐渐趋向于对称。

## 自由度的概念

![](img/d327b740d9bc530f9c919974a963f3ca_13.png)

在理解了上述分布后，我们需要明确“自由度”这个概念。自由度是指可以自由取值的变量个数。

**核心区别在于总体与样本：**

*   **总体数据**：计算方差等统计量时，自由度为 `N`（总体容量）。
*   **样本数据**：当我们用样本方差 `s²` 去估计总体方差 `σ²` 时，自由度为 `n-1`（样本容量减1）。这是因为在计算样本方差时，样本均值 `x̄` 已经确定，限制了一个变量的自由变化，因此可自由取值的个数减少一个。

![](img/d327b740d9bc530f9c919974a963f3ca_15.png)

简单记忆：**用样本推断总体时，自由度常为 `n-1`；仅对样本本身进行描述时，自由度可为 `n`。**

## 中心极限定理

中心极限定理是统计学中最重要的定理之一，它解释了为什么许多统计量最终都服从或近似服从正态分布。

![](img/d327b740d9bc530f9c919974a963f3ca_17.png)

### 定理内容

设从均值为 `μ`、方差为 `σ²` 的任意一个总体中，抽取样本量为 `n` 的样本。当 `n` 足够大时，**样本均值 `x̄` 的抽样分布**近似服从均值为 `μ`、方差为 `σ²/n` 的正态分布。

![](img/d327b740d9bc530f9c919974a963f3ca_19.png)

**公式表示为：**
```
x̄ ~ N(μ, σ²/n)  (近似)
```
其标准差 `σ/√n` 被称为**均值的标准误**，它衡量了样本均值与总体均值之间的典型误差（即抽样误差）。

![](img/d327b740d9bc530f9c919974a963f3ca_21.png)

### 定理的直观解释

1.  **单次抽样**：从总体中随机抽取一个样本量为 `n` 的样本，计算其均值 `x̄₁`。这个 `x̄₁` 是总体均值 `μ` 的一个估计，但存在误差。
2.  **多次抽样**：我们重复上述过程很多次，得到许多个样本均值：`x̄₁, x̄₂, x̄₃, ...`。
3.  **形成分布**：这些样本均值本身会构成一个新的分布，即“样本均值的抽样分布”。
4.  **定理结论**：中心极限定理告诉我们，只要样本量 `n` 足够大，无论原始总体是什么分布，这个“样本均值的抽样分布”的形状都近似于正态分布，且其均值就是总体均值 `μ`，其波动范围（方差）是 `σ²/n`。

![](img/d327b740d9bc530f9c919974a963f3ca_23.png)

### 中心极限定理的意义

该定理使我们能够利用正态分布的性质对样本均值进行推断。例如，我们可以计算样本均值落在某个区间的概率，或者构建总体均值的置信区间。在实际应用中，即使总体分布未知，只要样本量较大，我们就能基于正态分布进行可靠的统计推断。

## 总结

本节课中我们一起学习了抽样分布的核心知识。

*   我们首先介绍了三种重要的抽样分布：**卡方分布**（标准正态分布平方和）、**T分布**（适用于小样本）和**F分布**（用于方差比）。
*   然后，我们明确了**自由度**的概念，它描述了估计过程中独立信息的数量。
*   最后，我们深入探讨了**中心极限定理**。该定理指出，无论总体分布如何，大样本的均值分布都近似正态，这为几乎所有参数统计推断奠定了理论基础。

![](img/d327b740d9bc530f9c919974a963f3ca_25.png)
