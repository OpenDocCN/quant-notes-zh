# 金融量化分析：4-3：Sigma方法实例 📊

在本节课中，我们将要学习一种在金融数据处理中常用的去极值方法——3-Sigma（三西格玛）方法。我们将从原理入手，理解其背后的统计学思想，并通过代码实例演示如何将其应用于实际数据预处理中。

## 原理概述 📈

上一节我们介绍了去极值的概念，本节中我们来看看基于统计学原理的3-Sigma方法。

其核心思想非常简单。大家先来看下面这张图。

![](img/7012287e44e041b13c7097ec05cdff5a_1.png)

这张图中，我们画了一个正态分布（也叫高斯分布）。如果大家理解起来比较麻烦，可以把它当作一个“正常”的分布。我以前讲机器学习时举过一个例子：你现在去银行贷款，银行多借给你3块5块是挺正常的，但多借或少借你10万8万就不正常了。

我们来看这样一个走势。我先画一条线，以标准正态分布来说。这里有一个均值。均值代表什么意思？比如我去银行贷款，它可能多借一点，也可能少借一点。多借就往右边偏一点，少借就往左边偏一点。

但大家来看，我们现在这个区间。比如我现在画第一个红色阴影区间，它的面积相对来说比较大，代表这些事发生的可能性比较高。当我把这个区间越往外放的时候，可能性发生的概率就越低了。

我们来想一想，我们现在要干什么？我们要做一个去极值的问题。那极值可能落到哪儿？用红色画，落到均值附近可能吗？在均值附近、大概率发生的事，那就是正常的数据。极值一般落在哪儿？落在比较偏远的地方。离均值越远、偏差越大的点，越有可能是离群点。

## 什么是3-Sigma？ 🤔

![](img/7012287e44e041b13c7097ec05cdff5a_1.png)

这样一件事，那什么叫做一个三倍的西格玛？它是这样一件事。

你看这里我们画的东西中有什么？有一个X的均值，加上一倍标准差、减去一倍标准差；加上两倍、减去两倍；加上三倍、减去三倍。

![](img/7012287e44e041b13c7097ec05cdff5a_3.png)

从这个区间当中，我们能发现一件事：一旦你选的区间越大（比如加减3倍标准差），数据点越往左右偏的可能性就越小。

3-Sigma的意思是：我有一份原始数据。我假设可以对数据做一个变换，使得它理论上服从高斯分布（我们先不管怎么变）。当我指定一个正负3倍标准差的范围时，这个区间包含了多少数据？这个值在统计学中是一个常用指标，大约是99.7%。

我们现在发现一件事：如果我认为数据中，我想取最正常的99.7%当作有效数据，剩下的0.3%我当作极值给它规范一下，是不是可以？这个就叫做3-Sigma法。

相当于你的一个下限和上限由谁决定？在高斯分布中，你需要计算两个值：均值是多少，标准差是多少。我们就能把这个上限和下限全部计算出来。

![](img/7012287e44e041b13c7097ec05cdff5a_3.png)

## 代码实现 💻

这个叫做3-Sigma方法。写一下代码，其实也很简单。

在这里，函数名跟之前类似，还是做一个复制。我们把最后面那个值改成一个“sigma”就行了。然后我们输入一个参数n，这里n等于3，既然叫3-Sigma，我们就写3。

![](img/7012287e44e041b13c7097ec05cdff5a_5.png)

我们需要计算什么？在整个数据的这一列中，我需要知道当前它的均值是多少，当前的标准差是多少。

所以对我的数据，我可以计算两个值：一个是`.mean()`，另一个是`.std()`。

以下是计算均值和标准差的代码：

![](img/7012287e44e041b13c7097ec05cdff5a_7.png)

```python
mean = data.mean()
std = data.std()
```

![](img/7012287e44e041b13c7097ec05cdff5a_8.png)

指定两个变量，名字是`mean`和`std`。两个指标算出来之后，接下来直接复制公式即可。

以下是计算上下限的代码：

```python
upper_limit = mean + n * std
lower_limit = mean - n * std
```

![](img/7012287e44e041b13c7097ec05cdff5a_10.png)

![](img/7012287e44e041b13c7097ec05cdff5a_11.png)

把这个公式复制过来。你看，上限等于均值加上n倍的标准差。下限就是均值减去n倍的标准差。

返回值是原始数据`s`，以及我们计算出的下限和上限。非常简单，我们就完成了这个3-Sigma函数。

这个函数执行后，sigma等于什么呢？等于我的这个数据中，传入当前这一列数据`data`和`n`值。
