## 股东户数：理解市场情绪的指标

![](img/79beb789ce48a7e9953b4f6f6ad7044e_13.png)

在开始分析之前，我们首先需要理解核心数据——股东户数。在A股市场，上市公司的定期报告（如季报、年报）中会披露一个名为“股东户数”的字段。

![](img/79beb789ce48a7e9953b4f6f6ad7044e_15.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_17.png)

**公式定义**：
对于一个公司，其`股东户数`可以近似表示为：
`股东户数 ≈ 散户股东人数`
这是因为A股市场中散户投资者的数量远远超过机构投资者。因此，一家公司的股东户数越多，通常意味着持有其股票的散户投资者越多。

![](img/79beb789ce48a7e9953b4f6f6ad7044e_19.png)

这个数据为我们提供了一个观察市场散户资金流向和情绪聚集程度的窗口。

## 数据来源：从龙虎榜到股东户数

上一节我们介绍了股东户数的概念，本节中我们来看看用于分析的两类关键数据来源。

![](img/79beb789ce48a7e9953b4f6f6ad7044e_29.png)

首先，我们可以观察被称为“散户大本营”的东方财富证券部分营业部（如拉萨团结路、东环路营业部）在龙虎榜上的表现。龙虎榜是交易所公布的每日交易异动股票及其背后活跃营业部的名单。

![](img/79beb789ce48a7e9953b4f6f6ad7044e_31.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_33.png)

以下是基于公开数据的观察结果：
*   当这些以散户为主的营业部出现在某只股票的龙虎榜买入席位上时，该股票在后续短期内（如1天、3天、10天）的平均收益率多为负数。
*   这表明短期内大量散户集中买入的行为，可能不是一个有效的正向信号。

![](img/79beb789ce48a7e9953b4f6f6ad7044e_35.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_37.png)

然而，龙虎榜数据仅覆盖少数交易异常活跃的股票，样本有限。为了进行更全面、系统的分析，我们需要所有上市公司的**股东户数历史数据**。这份数据记录了每家公司在不同报告期的股东总数量，是研究散户持股分布的理想基础。

## 策略构建：买入散户最多的股票能赚钱吗？

![](img/79beb789ce48a7e9953b4f6f6ad7044e_45.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_47.png)

基于股东户数数据，我们可以构建一个简单的量化策略来验证假设。

**策略逻辑（Python伪代码描述）**：
```python
# 每月末执行以下操作
def strategy(end_of_month_date):
    # 1. 获取当前所有股票的股东户数
    shareholder_data = get_shareholder_data(end_of_month_date)
    # 2. 剔除ST、退市、停牌股票
    valid_stocks = filter_stocks(shareholder_data)
    # 3. 按股东户数从大到小排序
    sorted_stocks = valid_stocks.sort_values(by=‘股东户数‘, ascending=False)
    # 4. 选择股东户数最多的前N只股票（例如前10只）
    selected_stocks = sorted_stocks.head(10)
    return selected_stocks

![](img/79beb789ce48a7e9953b4f6f6ad7044e_49.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_51.png)

# 交易规则：每月第一个交易日买入上月末选出的股票组合，持有至当月最后一个交易日卖出，然后换仓。
```

**策略核心思想**：如果散户的人数优势能转化为股价上涨动力，那么持续买入股东户数最多的股票组合，应该能获得超越市场平均的收益。

## 回测结果：数据给出的答案

![](img/79beb789ce48a7e9953b4f6f6ad7044e_61.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_63.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_65.png)

我们使用Python对上述策略进行了历史回测（测试周期覆盖多年历史数据）。回测结果以净值曲线形式展示，并与市场基准（如沪深300指数）进行对比。

![](img/79beb789ce48a7e9953b4f6f6ad7044e_67.png)

以下是关键结果：
*   **买入股东户数最多的前10只股票**：策略年化收益率约为**2.29%**，与沪深300指数表现接近，但最大回撤较大。该策略未能显著跑赢大盘。
*   **买入股东户数最多的前10%股票**：为了扩大样本，我们测试了排名前10%的股票组合。其最终收益甚至略低于只买前10只的策略。

结论很明确：**在历史数据中，买入散户高度聚集的股票，并不能获得超额收益。**

## 深入探索：股东户数分组的秘密

![](img/79beb789ce48a7e9953b4f6f6ad7044e_75.png)

既然抱团最紧的股票不赚钱，那么股东户数处于其他水平的股票表现如何呢？我们进一步将全市场股票按股东户数从多到少均分为10组，分别回测每组的收益。

![](img/79beb789ce48a7e9953b4f6f6ad7044e_77.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_79.png)

以下是分组回测的核心发现：
*   收益表现与股东户数排名呈现清晰的**单调关系**。
*   **股东户数最少**的组（即散户持股最分散或机构持股比例可能较高的股票），其长期收益**最高**。
*   随着股东户数增多（散户持股集中度上升），该组股票的平均收益**逐渐降低**。

这个发现颠覆了最初的直觉，并印证了金融市场的一条经验法则：**人多的地方不一定好赚钱，机会可能藏在关注度较低的地方。**

## 总结与展望

![](img/79beb789ce48a7e9953b4f6f6ad7044e_89.png)

![](img/79beb789ce48a7e9953b4f6f6ad7044e_91.png)

本节课中我们一起学习了如何利用股东户数数据来量化分析A股市场的散户行为。我们通过构建并回测策略，得出了两个核心结论：
1.  **散户的人数优势（股东户数多）并不能转化为稳定的超额收益**，盲目跟随散户抱团股并非有效策略。
2.  相反，**股东户数较少**的股票群体在历史上表现出了更好的收益特征。这背后可能涉及信息不对称、机构调研深度、流动性溢价等多重金融学原理。
