## 统计方法

NumPy提供了一系列用于计算数组统计量的方法。

以下是常用的统计函数及其说明：

*   **求和**：`A.sum()` 方法对数组 `A` 中的所有元素进行求和。其功能与Python内置的 `sum()` 函数类似。
*   **平均值**：`A.mean()` 方法计算数组 `A` 中所有元素的算术平均值。
*   **方差**：`A.var()` 方法计算数组的方差。方差衡量一组数据的离散程度。其计算公式为：**方差 = Σ(每个元素 - 平均值)² / 元素个数**。方差越大，数据波动越大。
*   **标准差**：`A.std()` 方法计算数组的标准差。标准差是方差的平方根，同样用于衡量数据的离散程度，并且其单位与原始数据一致。计算公式为：**标准差 = sqrt(方差)**。
*   **最大值与最小值**：`A.max()` 和 `A.min()` 分别返回数组中的最大值和最小值。
*   **最大值与最小值的索引**：`A.argmax()` 和 `A.argmin()` 分别返回数组中最大值和最小值所在位置的索引（下标）。

例如，对于数组 `A = np.array([1, 2, 3, 4, 5])`：
*   平均值 `A.mean()` 为 `3.0`。
*   方差 `A.var()` 为 `2.0`。计算过程：`[(1-3)² + (2-3)² + (3-3)² + (4-3)² + (5-3)²] / 5 = 2.0`。
*   标准差 `A.std()` 为 `sqrt(2.0) ≈ 1.414`。

在金融分析中，标准差常被用来估计资产价格的波动率（风险）。通过“均值 ± 倍数×标准差”可以大致估计数据的主要分布范围。

## 随机数生成

NumPy的 `random` 子模块提供了强大的随机数生成功能，它扩展了Python内置 `random` 模块的功能，支持直接生成整个数组的随机数。

以下是 `np.random` 中常用函数的介绍：

*   **生成随机浮点数**：`np.random.rand(d0, d1, ..., dn)` 生成一个给定形状的数组，其中的元素是 `[0, 1)` 区间内均匀分布的随机浮点数。
    ```python
    # 生成一个包含10个随机数的数组
    arr = np.random.rand(10)
    # 生成一个3行5列的二维随机数组
    arr_2d = np.random.rand(3, 5)
    ```
*   **生成随机整数**：`np.random.randint(low, high=None, size=None)` 生成指定范围内的随机整数。`size` 参数可以指定输出数组的形状。
    ```python
    # 生成一个0到10之间的随机整数
    num = np.random.randint(0, 10)
    # 生成一个形状为(3,5)，元素在0到10之间的整数数组
    arr_int = np.random.randint(0, 10, size=(3, 5))
    ```
*   **均匀分布随机数**：`np.random.uniform(low=0.0, high=1.0, size=None)` 生成在指定区间 `[low, high)` 内均匀分布的随机浮点数。
    ```python
    # 生成10个在2.0到4.0之间的随机浮点数
    arr_uniform = np.random.uniform(2.0, 4.0, 10)
    ```
*   **从给定序列中随机选择**：`np.random.choice(a, size=None)` 从数组 `a` 中随机抽取元素生成新数组。
    ```python
    # 从[1,3,4,5]中随机抽取一个数
    elem = np.random.choice([1,3,4,5])
    # 从[1,3,4,5]中有放回地随机抽取10次，形成数组
    arr_choice = np.random.choice([1,3,4,5], size=10)
    ```
*   **打乱数组顺序**：`np.random.shuffle(x)` 将数组 `x` 的元素顺序随机打乱。此操作直接修改原数组。

`np.random` 还支持其他分布，如正态分布 (`np.random.normal`)，在模拟股票收益率等场景中非常常用。
