## 遍历因子并排序 🔄

首先，我们需要遍历每个因子数据框（DataFrame），并对每个因子列进行排序。以下是具体步骤：

1.  **遍历因子列名**：我们需要获取每个因子的列名，并遍历它们。
2.  **对因子值进行排序**：对每个因子的数值进行排序。对于“越高越好”的因子，数值大的排在前面；对于“越低越好”的因子，数值小的排在前面。
3.  **原地修改数据**：使用 `inplace=True` 参数，使排序结果直接保存在原始DataFrame中，无需重新赋值。

以下是实现排序的代码示例：
```python
# 假设 df_up 是“越高越好”因子的DataFrame
for factor in df_up.columns:
    df_up.sort_values(by=factor, inplace=True)
```

## 为排序后的因子打分 🏆

![](img/9b7d429236ea454aa38bbc5ae741581b_10.png)

排序完成后，我们需要为每个股票的每个因子打分。为了简单直观，我们采用顺序打分法：排名第一的股票得最高分，排名最后的股票得最低分。

我们将使用NumPy的 `linspace` 函数来生成一个线性的分数序列。这个函数可以方便地创建指定范围和数量的等差数列。

![](img/9b7d429236ea454aa38bbc5ae741581b_12.png)

以下是 `numpy.linspace` 函数的基本用法：
```python
import numpy as np
# 生成从1到300的300个等间隔数字
scores = np.linspace(1, 300, 300)
```

![](img/9b7d429236ea454aa38bbc5ae741581b_14.png)

![](img/9b7d429236ea454aa38bbc5ae741581b_16.png)

### 为“越高越好”的因子打分

![](img/9b7d429236ea454aa38bbc5ae741581b_18.png)

![](img/9b7d429236ea454aa38bbc5ae741581b_20.png)

对于“越高越好”的因子，排序后数值最大的股票排名第一，应获得最高分。因此，我们使用 `np.linspace(1, 股票总数, 股票总数)` 来生成分数。

![](img/9b7d429236ea454aa38bbc5ae741581b_22.png)

![](img/9b7d429236ea454aa38bbc5ae741581b_24.png)

以下是实现代码：
```python
import pandas as pd
import numpy as np

![](img/9b7d429236ea454aa38bbc5ae741581b_26.png)

![](img/9b7d429236ea454aa38bbc5ae741581b_28.png)

# 假设股票总数为 total_stocks
total_stocks = len(df_up)

![](img/9b7d429236ea454aa38bbc5ae741581b_30.png)

for factor in df_up.columns:
    df_up.sort_values(by=factor, inplace=True)
    # 生成从1到总股票数的分数序列，赋值给当前因子列
    df_up[factor] = np.linspace(1, total_stocks, total_stocks)
```

![](img/9b7d429236ea454aa38bbc5ae741581b_32.png)

![](img/9b7d429236ea454aa38bbc5ae741581b_34.png)

### 为“越低越好”的因子打分

![](img/9b7d429236ea454aa38bbc5ae741581b_36.png)

对于“越低越好”的因子，排序后数值最小的股票排名第一，应获得最高分。因此，我们需要生成一个从高到低的分数序列，即使用 `np.linspace(股票总数, 1, 股票总数)`。

![](img/9b7d429236ea454aa38bbc5ae741581b_38.png)

以下是实现代码：
```python
# 假设 df_down 是“越低越好”因子的DataFrame
total_stocks = len(df_down)

for factor in df_down.columns:
    df_down.sort_values(by=factor, inplace=True)
    # 生成从总股票数到1的分数序列（降序），赋值给当前因子列
    df_down[factor] = np.linspace(total_stocks, 1, total_stocks)
```

## 合并打分结果 ➕

![](img/9b7d429236ea454aa38bbc5ae741581b_40.png)

![](img/9b7d429236ea454aa38bbc5ae741581b_42.png)

完成对所有因子的打分后，我们得到了两个新的DataFrame（`df_up` 和 `df_down`），它们包含了所有股票在各个因子上的得分。为了计算每只股票的综合总分，我们需要将这两个DataFrame合并。

通常，我们可以使用pandas的 `concat` 函数按列进行合并，因为两个DataFrame的行索引（股票代码）是一致的。

以下是合并代码示例：
```python
# 沿列方向合并两个DataFrame
df_all_scores = pd.concat([df_up, df_down], axis=1)
```
合并后的 `df_all_scores` 包含了所有因子（包括“越高越好”和“越低越好”两类）的得分，接下来就可以对每只股票的所有因子得分进行加总，得到最终的综合评分。

![](img/9b7d429236ea454aa38bbc5ae741581b_44.png)

![](img/9b7d429236ea454aa38bbc5ae741581b_46.png)

本节课中我们一起学习了因子打分与排序的核心流程。我们首先遍历因子并进行排序，然后利用 `numpy.linspace` 函数根据因子特性（越高越好或越低越好）生成对应的分数序列，最后将两类因子的得分结果合并，为下一步计算综合评分做好了数据准备。这个过程是将抽象的因子数据转化为可比较、可加总的分数关键步骤。
