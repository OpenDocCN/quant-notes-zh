# 量化交易：08：IPython魔术命令 🪄

![](img/5d9e17d2483f605ea7779df804b94067_1.png)

在本节课中，我们将学习IPython中一个非常实用且有趣的高级功能——魔术命令。魔术命令以百分号（`%`）开头，能够直接在交互式环境中执行一些特殊操作，例如运行外部脚本、粘贴代码块或测量代码执行时间，从而极大提升我们的开发与调试效率。

## 魔术命令简介

上一节我们介绍了IPython的基本使用，本节中我们来看看它的“魔术”功能。魔术命令是IPython提供的一系列以百分号（`%`）开头的特殊指令。它们允许我们在不退出交互式环境的情况下，完成一些通常在外部命令行或编辑器中才能进行的操作。

## 运行外部Python脚本

![](img/5d9e17d2483f605ea7779df804b94067_3.png)

在标准的Python命令行中，如果想运行一个已写好的Python文件（`.py`），通常需要先退出命令行，再使用 `python filename.py` 这样的命令。在IPython中，我们可以使用 `%run` 魔术命令直接运行，无需退出。

![](img/5d9e17d2483f605ea7779df804b94067_5.png)

**命令格式**：
```python
%run 文件名.py
```

例如，假设我们在当前目录下有一个名为 `hello.py` 的文件，其内容为 `print(“Hello World”)`。我们只需在IPython中输入 `%run hello.py`，即可看到输出结果。

## 粘贴并执行代码

![](img/5d9e17d2483f605ea7779df804b94067_7.png)

有时，我们可能从编辑器或其他地方复制了一段代码，希望直接在IPython中执行。虽然可以直接粘贴，但可能会遇到缩进或格式问题。IPython提供了 `%paste` 魔术命令来解决这个问题。

![](img/5d9e17d2483f605ea7779df804b94067_9.png)

`%paste` 命令会执行剪贴板中的代码内容，并自动处理好格式。这对于测试一段较长的代码片段非常方便，无需创建临时文件。

![](img/5d9e17d2483f605ea7779df804b94067_11.png)

![](img/5d9e17d2483f605ea7779df804b94067_13.png)

以下是使用 `%paste` 的步骤：
1.  在任何地方复制你想要执行的代码。
2.  在IPython中输入 `%paste` 并回车。
3.  IPython会先打印出即将执行的代码，然后执行它。

## 测量代码执行时间

在性能分析和优化时，我们经常需要知道某段代码或某个函数运行了多长时间。虽然可以使用Python的 `time` 模块，但对于执行时间极短的代码，测量可能不够精确。IPython的 `%timeit` 魔术命令可以解决这个问题。

`%timeit` 会自动多次运行指定的代码，并计算其平均执行时间，从而得到更精确的结果，尤其适用于微秒或纳秒级别的性能测试。

**命令格式**：
```python
%timeit 要测试的代码
```

例如，我们想测试对一个包含一万个随机数的列表进行排序所需的时间：
```python
import random
lst = [random.random() for _ in range(10000)]
%timeit sorted(lst)
```
`%timeit` 会输出类似 `14.6 ms ± 742 µs per loop` 的结果，表示平均每次循环耗时14.6毫秒，并给出了误差范围。

## 总结

本节课中我们一起学习了IPython中三个核心的魔术命令：
1.  **`%run`**：用于在交互式环境中直接运行外部的Python脚本文件。
2.  **`%paste`**：用于安全地粘贴并执行剪贴板中的代码，避免格式错误。
3.  **`%timeit`**：用于精确测量短小代码片段的执行时间，是性能优化的利器。

掌握这些魔术命令，能让你在数据分析和量化交易的代码编写、测试与优化过程中更加得心应手，显著提升工作效率。
