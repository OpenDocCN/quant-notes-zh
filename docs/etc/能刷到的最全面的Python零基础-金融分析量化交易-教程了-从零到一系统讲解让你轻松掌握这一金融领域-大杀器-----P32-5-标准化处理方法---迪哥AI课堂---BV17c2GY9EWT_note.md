在本节课中，我们将要学习数据预处理中一个至关重要的技术——标准化。标准化是数据挖掘和机器学习中常用的方法，其核心目的是消除不同数据特征（或指标）在量纲和取值范围上的差异，使数据处于同一尺度，便于后续分析和建模。

上一节我们介绍了数据预处理的重要性，本节中我们来看看如何通过标准化操作来统一数据的尺度。

## 标准化的目的与原理 🎯

为了理解标准化的作用，我们先来看一个场景。假设我们有一组数据，包含两个不同的指标，例如X1和X2。这两个指标的取值范围可能完全不同。X1的取值范围可能较大，而X2的取值范围可能较小。

如果直接使用这些原始数据进行分析，取值范围大的指标可能会对模型结果产生不成比例的巨大影响，从而掩盖了其他指标的作用。标准化的出发点，就是为了使不同的指标在各个维度上的取值范围尽可能相同，让我们能用同样的尺度去观察和比较它们，避免数值层面的差异对最终结果造成不当影响。

## 标准化的公式分解 📝

标准化的操作遵循一个明确的数学公式：

**`标准化值 = (原始值 - 平均值) / 标准差`**

这个公式可以分解为两个步骤来理解：

1.  **去均值（中心化）**：`原始值 - 平均值`
    *   **目的**：使数据在各个维度上以零为中心对称。原始数据通常不是以原点（0点）为中心分布的，减去均值后，数据的分布中心就被移动到了0点。
2.  **除标准差（缩放）**：`(去均值后的值) / 标准差`
    *   **目的**：使不同维度的数据取值范围近似相同。去均值后，虽然数据以0为中心，但不同特征的数据波动范围（即取值范围）依然可能差异很大。除以各自的标准差，可以对其进行缩放。

为了更直观地理解“除标准差”的作用，可以想象两组人：
*   **第一组**：几个工资相近的普通人。他们的工资差异小，计算出的**标准差也小**。
*   **第二组**：一个普通人和几位亿万富翁。他们的财富差异巨大，计算出的**标准差非常大**。

对于取值范围小（标准差小）的数据，除以一个小的标准差；对于取值范围大（标准差大）的数据，除以一个大的标准差。这样操作后，所有数据的尺度就被调整到了近似的范围内。

综上所述，标准化通过“去均值”和“除标准差”两个步骤，最终实现两个效果：
*   使数据以零为中心对称。
*   使各维度数据的取值范围处于一个较小且相近的区间内。

以下是手动实现标准化的函数代码：

![](img/0c30cf0a5fa2776b9fe64d75bb496e3a_15.png)

```python
def standardize_series(series):
    """
    对输入的Pandas Series进行标准化处理。
    参数:
        series: 待标准化的数据序列。
    返回:
        标准化后的数据序列。
    """
    # 计算序列的均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    # 应用标准化公式： (值 - 均值) / 标准差
    return (series - mean_val) / std_val
```

![](img/0c30cf0a5fa2776b9fe64d75bb496e3a_17.png)

![](img/0c30cf0a5fa2776b9fe64d75bb496e3a_19.png)

代码解析：
1.  首先计算输入数据序列的均值（`mean_val`）和标准差（`std_val`）。
2.  然后，将序列中的每个值减去均值，再除以标准差，并返回结果。

此外，在机器学习领域，`scikit-learn`库提供了更便捷的标准化工具。如果你想使用，可以在`sklearn.preprocessing`模块中找到`StandardScaler`类，它提供了高效的标准化功能。
