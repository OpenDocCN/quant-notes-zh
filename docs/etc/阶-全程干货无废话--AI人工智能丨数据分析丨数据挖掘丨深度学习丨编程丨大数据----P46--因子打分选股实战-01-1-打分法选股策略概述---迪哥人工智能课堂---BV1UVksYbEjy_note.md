## 概述
在本节课中，我们将要学习量化选股中一个常用且重要的策略——因子打分选股法。我们将了解该策略的核心思想、实施步骤以及背后的逻辑，为后续的实战操作打下基础。

## 策略核心思想
上一节我们介绍了因子分析的基本概念，本节中我们来看看如何综合运用多个因子进行选股。

因子打分选股策略的核心思想是：**综合评估股票在多个因子上的表现，通过加权或汇总打分的方式，得到一个综合得分，并依据此得分进行排序和选股**。这类似于学生考试，总成绩由各科成绩汇总决定，我们最终选择“总成绩”最优的股票。

## 策略实施步骤
以下是实施因子打分选股策略的关键步骤。

### 1. 确定候选因子与方向
首先，我们需要选择用于评估股票的因子，并明确每个因子与预期收益率的相关性方向。

*   **选择因子**：例如估值类因子（市盈率PE、市净率PB）、成长类因子、价量类因子等。
*   **确定方向**：必须预先知道每个因子是“越大越好”还是“越小越好”。这可以通过历史数据分析（如计算**IC值**）或参考专业研究报告获得。
    *   **正相关（越大越好）**：例如每股收益(EPS)、净资产收益率(ROE)。因子值越大，理论上股票未来收益可能越好。
    *   **负相关（越小越好）**：例如市盈率(PE)。因子值越小，可能代表股票估值越低，未来上涨空间越大。

这个步骤是策略的基础，相当于为每个评分科目设定了“加分”规则。

### 2. 为单因子打分
接下来，我们需要为股票池中的每一只股票，在每个因子上进行独立评分。

假设我们有300只股票和4个因子（A, B, C, D）。对于因子A，我们将所有300只股票按照其因子A的数值进行排序。
*   若因子A是“越大越好”，则因子值最大的股票在该因子上的得分最高。
*   若因子A是“越小越好”，则因子值最小的股票在该因子上的得分最高。

常用的打分方法包括**排名归一化**（将排名转化为0-1或1-100的分数）或**分位数分组打分**（如前20%得5分，中间60%得3分，后20%得1分）。

![](img/b0d799e14074dfad1ae1c2e12311c62b_1.png)

```python
# 示例：对因子‘pe_ratio’进行排名打分（假设越小越好）
df['pe_rank'] = df['pe_ratio'].rank(ascending=True)  # 值越小排名越靠前（名次数字越小）
df['pe_score'] = (df['pe_rank'] / df['pe_rank'].max()) * 100  # 归一化为0-100分
```

### 3. 计算综合得分
在得到每只股票在所有因子上的单项得分后，我们需要将这些分数汇总成一个综合得分。

最直接的方法是**等权加权求和**，即将所有因子的得分简单相加。也可以根据因子的预测能力（如历史IC值）进行**加权求和**，给予更有效的因子更高的权重。

```python
# 示例：等权计算综合得分（假设已有 factor_A_score, factor_B_score 等列）
df['total_score'] = df[['factor_A_score', 'factor_B_score', 'factor_C_score', 'factor_D_score']].sum(axis=1)
```

### 4. 依据得分选股与调仓
最后，我们根据综合得分进行决策。

*   **排序选股**：对股票池中所有股票按`total_score`从高到低进行排序。
*   **选择头部股票**：选取排名前N（例如前10或前20名）的股票作为投资组合。
*   **定期调仓**：该策略通常按固定周期（如每月或每季度）重复执行以上步骤，卖出不在新名单中的股票，买入新进入名单的股票，完成组合的更新。

## 总结
本节课中我们一起学习了因子打分选股策略。该策略通过**对多个已知方向的因子进行独立评分、加权汇总得到综合得分，并定期选取得分最高的股票构成投资组合**，实现了从单一因子分析到多因子综合决策的跨越。它逻辑清晰，操作性强，是量化选股中的经典方法。在接下来的实战环节，我们将在一个具体的平台上实现这一策略。
