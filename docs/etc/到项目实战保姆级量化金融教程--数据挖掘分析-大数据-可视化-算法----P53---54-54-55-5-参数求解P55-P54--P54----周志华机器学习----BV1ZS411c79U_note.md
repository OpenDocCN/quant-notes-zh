# 机器学习：P54-P55：参数求解

![](img/d651af7e7b7ababb018977b67e7d4558_0.png)

![](img/d651af7e7b7ababb018977b67e7d4558_0.png)

在本节中，我们将学习如何求解线性回归模型中的参数。我们将从最大似然估计推导出目标函数，并最终得到参数求解的公式。理解这个过程有助于我们掌握机器学习模型求解的基本思想。

上一节我们介绍了如何通过最大似然估计将概率问题转化为优化问题。本节中我们来看看如何对这个优化问题进行求解。

首先，回顾我们得到的对数似然函数。它是一个累加形式，而不是累乘。右边的式子保持不变。我们来观察这个式子。

为了方便分析，我们将式子分成几个部分。首先，框出常数项部分，记为A。另一部分记为B。在当前任务中，我们可以先将常数项提出来。累加项中，常数项A被加了M次，等于M倍的常数项。右边这一项，在对数运算中，A乘B的对数等于对数A加上对数B。由于A是常数，这部分可以单独处理。

对于右边涉及样本的部分，我们假设对数以e为底。这样，指数部分可以直接落下来。这一项不能拆开，因为它需要从第一个样本迭代到第M个样本。每个样本的Y和X都是不同的。

完成展开后，我们需要思考目标。我们希望模型预测Y的可能性越大越好。因此，我们需要找到使该式子取得极大值的参数。观察展开后的式子，它由一个常数项A减去一个值B组成。A是不变的常数。B是一个恒正的项，因为其中包含平方项。要使整体值越大，由于是常数减去一个正数，只能让B越小越好。

![](img/d651af7e7b7ababb018977b67e7d4558_2.png)

因此，我们将B单独列出来进行分析。让B越小越好，就成为了我们的新目标。我们把这个B拿出来，并保留前面的1/2系数，因为它可能在后续计算中用到。

以下是当前目标函数的表达式：
`J(θ) = 1/2 * Σ (y_i - θ^T x_i)^2`，其中求和从i=1到M。

![](img/d651af7e7b7ababb018977b67e7d4558_4.png)

这个`J(θ)`通常被称为目标函数、损失函数或代价函数。现在的目标是让这个函数的值越小越好。这个式子有一个更通俗的名称：**最小二乘法**。

![](img/d651af7e7b7ababb018977b67e7d4558_6.png)

![](img/d651af7e7b7ababb018977b67e7d4558_4.png)

![](img/d651af7e7b7ababb018977b67e7d4558_8.png)

在许多线性回归的介绍中，可能会直接给出最小二乘法，即希望预测值与真实值之间的差距平方和越小越好。这种直接的方式没有问题，但希望大家能将前面的推导过程串联起来。

![](img/d651af7e7b7ababb018977b67e7d4558_10.png)

因为最大似然估计并非线性回归特有，它在很多机器学习问题中都会出现。不需要背诵所有公式，但需要理解每一步的原因：为什么是累乘变累加，为什么目标从最大化变为最小化。同时，理解误差项的定义是这一切推导的基础。没有这个前提，后续的推导都无法成立。

![](img/d651af7e7b7ababb018977b67e7d4558_12.png)

![](img/d651af7e7b7ababb018977b67e7d4558_12.png)

接下来，我们讨论如何求解这个目标函数。求解过程需要将式子展开。这里需要注意：X不是一个单一的数字（如3、5、7），而是代表数据，可能包含多列特征（如X1, X2, ..., Xn）。因此，X应被视为一个矩阵。我们所有的求解都基于矩阵运算。

![](img/d651af7e7b7ababb018977b67e7d4558_14.png)

首先，对于矩阵，其平方项可以表示为转置乘以自身。再次强调，X和θ在这里都是矩阵或向量的概念。

![](img/d651af7e7b7ababb018977b67e7d4558_16.png)

![](img/d651af7e7b7ababb018977b67e7d4558_16.png)

我们的目标是找到使`J(θ)`最小的θ。这可以通过对`J(θ)`关于θ求偏导，并令偏导数等于零来实现。偏导数为零的点就是极值点。

我们来看展开后的求导过程。在矩阵求导中，有这样一个公式：如果A是一个对称矩阵，那么对`θ^T A θ`关于θ求偏导，结果为`2Aθ`。这里不详细推导每个步骤，我们直接给出关键结果。

![](img/d651af7e7b7ababb018977b67e7d4558_18.png)

求导并令结果等于零后，我们得到：
`X^T X θ = X^T y`

![](img/d651af7e7b7ababb018977b67e7d4558_18.png)

在这个等式中，X是已知的数据矩阵，y是已知的目标向量，θ是要求解的未知参数向量。我们需要解出θ。这类似于解方程`Aθ = B`。为了消去左边的`X^T X`，我们可以在等式两边同时乘以`(X^T X)`的逆矩阵。

于是，我们得到参数的解析解公式：
`θ = (X^T X)^{-1} X^T y`

通过这个公式，我们似乎可以直接用已知的X和y计算出未知的θ。但是，这里我们需要思考两个问题。

首先，机器学习强调“学习”的过程。但在我们当前的求解方法中，有学习的过程吗？看起来并没有。它更像是直接代入公式得到一个答案，而没有像人类学习那样，通过多次尝试、根据错误进行调整的渐进过程。

其次，这个求解方法涉及矩阵求逆。矩阵求逆并非总是可行的，存在不可逆的矩阵。一旦`X^T X`不可逆，我们就无法通过这个公式求解θ。

![](img/d651af7e7b7ababb018977b67e7d4558_22.png)

![](img/d651af7e7b7ababb018977b67e7d4558_20.png)

因此，我们需要意识到这种直接求解法（也称为“正规方程法”）的局限性。它虽然直接，但缺乏迭代学习机制，且对数据矩阵有要求。

![](img/d651af7e7b7ababb018977b67e7d4558_24.png)

![](img/d651af7e7b7ababb018977b67e7d4558_22.png)

![](img/d651af7e7b7ababb018977b67e7d4558_24.png)

本节课中我们一起学习了线性回归参数的求解。我们从最大似然估计出发，推导出最小二乘目标函数，并最终得到了参数θ的解析解公式`θ = (X^T X)^{-1} X^T y`。我们同时讨论了该方法的优点（直接求解）和缺点（缺乏学习过程、依赖矩阵可逆性）。这为我们后续理解更复杂的迭代优化算法（如梯度下降）奠定了基础。
