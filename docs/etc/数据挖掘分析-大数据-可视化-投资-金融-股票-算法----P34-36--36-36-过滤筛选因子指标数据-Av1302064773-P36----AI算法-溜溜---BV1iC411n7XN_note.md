在构建策略时，我们通常需要剔除一些不符合条件的股票，例如停牌股、ST股和刚上市的新股。以下是实现这一功能的三个核心函数。

### 1. 过滤停牌股
我们首先定义一个函数，用于过滤掉当前处于停牌状态的股票。其核心逻辑是遍历股票池，并使用API判断每只股票是否全天停牌。

![](img/b262836d9b0c3f8e9e1d404eee869193_32.png)

![](img/b262836d9b0c3f8e9e1d404eee869193_34.png)

![](img/b262836d9b0c3f8e9e1d404eee869193_36.png)

```python
def filter_paused_stocks(stock_list):
    """
    过滤停牌股票
    :param stock_list: 股票代码列表
    :return: 过滤后的股票代码列表
    """
    filtered_stocks = []
    for stock in stock_list:
        # 使用API判断股票是否全天停牌
        if not is_suspended(stock):
            filtered_stocks.append(stock)
    return filtered_stocks
```

### 2. 过滤ST股
接下来，我们过滤掉被标记为ST（特别处理）的股票。这类股票通常存在财务或其他异常状况，风险较高。

![](img/b262836d9b0c3f8e9e1d404eee869193_44.png)

![](img/b262836d9b0c3f8e9e1d404eee869193_46.png)

![](img/b262836d9b0c3f8e9e1d404eee869193_48.png)

```python
def filter_st_stocks(stock_list):
    """
    过滤ST股票
    :param stock_list: 股票代码列表
    :return: 过滤后的股票代码列表
    """
    filtered_stocks = []
    for stock in stock_list:
        # 使用API判断股票是否为ST股
        if not is_st_stock(stock):
            filtered_stocks.append(stock)
    return filtered_stocks
```

### 3. 过滤新股
最后，我们过滤掉上市时间过短的新股。新股通常价格波动较大，且缺乏足够的历史数据进行分析。这里我们设定一个阈值（例如180天）。

![](img/b262836d9b0c3f8e9e1d404eee869193_56.png)

![](img/b262836d9b0c3f8e9e1d404eee869193_58.png)

![](img/b262836d9b0c3f8e9e1d404eee869193_60.png)

```python
def filter_new_stocks(stock_list, days_threshold=180):
    """
    过滤上市时间过短的新股
    :param stock_list: 股票代码列表
    :param days_threshold: 上市天数阈值，默认180天
    :return: 过滤后的股票代码列表
    """
    filtered_stocks = []
    for stock in stock_list:
        # 获取股票上市天数
        days_listed = get_days_listed(stock)
        # 如果上市天数大于阈值，则保留
        if days_listed > days_threshold:
            filtered_stocks.append(stock)
    return filtered_stocks
```

我们使用 `get_fundamentals` 函数来查询数据。该函数需要指定查询的指标和过滤条件。

# 查询市净率(PB Ratio)和市值(Market Cap)指标
query_statement = query(

**核心概念解释**：
*   **`query(...)`**：用于构建查询语句，指定需要获取哪些财务指标。
*   **`.filter(...)`**：用于对查询结果进行筛选，这里我们限定只获取 `final_stocks` 列表中的股票数据。
*   **`.T`**：是Pandas DataFrame的转置操作，将行和列互换。
*   **`.dropna()`**：删除包含空值（NaN）的行，确保数据的完整性。
