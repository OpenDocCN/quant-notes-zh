## 昨日回顾 📝

上一节我们介绍了Excel数据分析中的两大核心模块：公式与函数，以及数据透视表。

对于公式与函数而言，地址引用是一个很重要的知识点。我们按照不同类型将函数分为以下几类，并介绍了相关的常用操作：

*   **逻辑函数**
*   **文本函数**
*   **统计函数**
*   **查找与引用函数**
*   **日期函数**

第二块内容是数据透视表。数据透视表是一种快速对数据进行分类汇总计算的操作方式，它在Excel中属于比较灵活高效的使用方法，也是工作中非常常见和常用的一种方式。

数据透视表有其使用前的注意事项和常用功能，我们昨天介绍了以下几项操作：

*   **更新操作**
*   **排序操作**
*   **切片器操作**
*   **添加计算字段与计算项**
*   **多重合并**
*   **数据透视图**

## 今日内容 🚀

在回顾了公式、函数和数据透视表之后，本节中我们来看看Excel数据分析的最后一项核心内容：图表。

![](img/4b231196ca471815600fb36c8fe015c0_7.png)

图表是数据可视化的关键。我们今天将首先介绍图表的基本结构，然后讲解不同类型的图表及其应用场景。

![](img/4b231196ca471815600fb36c8fe015c0_9.png)

以下是今天将要介绍的主要图表类型：

*   **柱状图**
*   **条形图**
*   **折线图**
*   **散点图**
*   **气泡图**
*   **饼图**
*   **雷达图**
*   **面积图**
*   **组合图**
