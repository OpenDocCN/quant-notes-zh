# 量化编程入门：第12课：字符串处理与数组基础

![](img/4e180212544de1580f7440e6e7a81bdd_0.png)

在本节课中，我们将学习量化编程中两个重要的基础概念：字符串处理和数组。字符串处理可以帮助我们在图表上清晰地展示关键信息，而数组则是存储和操作一系列数据（如价格序列）的核心工具。我们将从字符串的常用函数开始，然后介绍数组的基本定义和使用方法。

## 字符串处理函数 🧵

上一节我们提到了字符串在量化中的用途。本节中，我们来看看MC中一些常用的字符串处理函数和方法。

### Text函数：类型转换

`Text`函数用于将指定的参数（通常是数值）转换为字符串类型并返回。在计算机底层，数值和字符串的存储方式是不同的。数值直接存储，而字符串则根据编码表（如ASCII码）进行存储和表示。`Text`函数的作用就是进行这种类型的转换。

**公式/代码示例：**
```pascal
Value1 = 100;
Value2 = Text(Value1); // 将数值100转换为字符串"100.00"
Print(Value2); // 输出字符串
```
理解这一点很重要，因为许多字符串操作函数（如计算长度）只能作用于字符串类型的数据。

### Space函数：插入空格

`Space`函数用于在字符串中插入指定数量的空格。这在格式化图表输出时非常有用，可以使不同长度的文本对齐显示。

**公式/代码示例：**
```pascal
Value1 = 1;
OutputString = “A” + Space(Value1) + “B”;
// 当Value1变化时，A和B之间的空格数量会变化
```

### 字符串大小写转换与长度

以下是几个基础的字符串操作函数：
*   `UpString`：将字符串中的所有字母转换为大写。
*   `LowString`：将字符串中的所有字母转换为小写。
*   `StringLen`：返回指定字符串的字符长度。

### 字符串与数值的转换

`StringToNumber`和`NumberToString`用于字符串和数值之间的转换。

**需要注意：** `StringToNumber`会尝试提取字符串中的数字部分。如果字符串以非数字字符开头，则可能返回0。
**公式/代码示例：**
```pascal
SymbolName = “SHFE.rb2301”;
Value1 = StringToNumber(SymbolName); // 可能返回0
Value2 = StringToNumber(“2301”); // 返回数值2301
```

### 提取子字符串

为了从字符串中获取特定部分（例如从合约代码中提取年份和月份），我们需要使用子字符串提取函数。

以下是相关函数：
*   `LeftString`：从字符串左侧开始提取指定数量的字符。
*   `RightString`：从字符串右侧开始提取指定数量的字符。
*   `MidString`：从字符串指定位置开始，提取指定数量的字符。

**公式/代码示例：**
```pascal
Contract = “SHFE.rb2301”;
Part1 = LeftString(Contract, 4); // 提取“SHFE”
Part2 = MidString(Contract, 9, 4); // 从第9个字符开始，提取4位，得到“2301”
```

### 查找字符串位置

`InString`函数用于在字符串中查找特定子串，并返回其第一次出现的位置（索引）。搜索是区分大小写的。如果未找到，则返回0。

**公式/代码示例：**
```pascal
Text = “Hello World”;
Position = InString(Text, “W”); // 返回 7
Position2 = InString(Text, “z”); // 返回 0
```

## 数组基础介绍 📊

掌握了字符串操作后，我们进入另一个核心主题：数组。数组是存储多个同类型数据的集合，在量化中常用于存储历史价格、指标值等序列数据。

### For循环：数组的伙伴

在深入数组之前，必须先了解`For`循环，因为遍历数组元素几乎总是离不开它。

`For`循环的基本结构是让一个变量在指定的起始值和结束值之间变化，并重复执行一段代码。
**公式/代码示例：**
```pascal
For Value1 = 1 To 10 Begin
    Print(“Time is: “, Time, “ Value is: “, Value1);
End;
```
上述代码会在每根K线上，将`Value1`从1到10的值依次打印出来。`DownTo`关键字则用于从大到小遍历。

### 数组的定义与赋值

数组使用`Array`关键字定义，并需要指定其大小（索引范围）。

**公式/代码示例：**
```pascal
Array: ClosePrice[10](0); // 定义一个长度为11（索引0到10）的数组，初始值均为0
```
可以直接通过索引为数组赋值：
```pascal
ClosePrice[1] = 100;
ClosePrice[2] = 200;
```

### 数组的存储与更新逻辑

在实际应用中，我们通常需要将最新的数据（如收盘价）不断存入数组。这涉及到一个“数据平移”的过程：将旧数据向后移动，为新数据腾出位置。

**核心更新逻辑代码如下：**
```pascal
For i = 10 DownTo 1 Begin
    ClosePrice[i] = ClosePrice[i-1]; // 将前一个位置的值赋给当前位置
End;
ClosePrice[0] = Close; // 将最新的收盘价放入数组首位
```
这个循环实现了将数组所有元素向后移动一位，并将最新的`Close`价格存入`ClosePrice[0]`。这样就模拟了一个固定长度的历史价格序列。

### 动态数组与最大索引设置

数组也可以定义为动态大小（不指定初始长度）。但对于动态数组，**必须**在使用前使用`SetMaxIndex`函数设置其最大索引，否则可能导致数组溢出错误。

**公式/代码示例：**
```pascal
Input: MaxBars(10); // 定义一个参数，用于设置数组大小
Array: DynamicArray[](0); // 定义动态数组

SetMaxIndex(DynamicArray, MaxBars); // 必须：设置动态数组的最大索引

// 之后就可以像普通数组一样使用DynamicArray了
For i = MaxBars DownTo 1 Begin
    DynamicArray[i] = DynamicArray[i-1];
End;
DynamicArray[0] = Close;
```
这种方法允许我们通过参数灵活调整数组的大小。
