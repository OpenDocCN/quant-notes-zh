## 计算每日差异值

上一节我们介绍了如何加载和查看时间序列数据。本节中，我们来看看如何计算数据中每一天与前一天的差异值。

![](img/2dd54e54c3a06447381a1ed766d00b4e_6.png)

在金融数据分析中，我们不仅关心当天的绝对数值（如股价），更关心其相对于前一天的变化量（即涨跌了多少）。这个指标被称为“差异值”。

在Pandas中，计算差异值非常简单，我们可以使用 `.diff()` 方法。该方法会计算当前行数据与前一行的差值。

![](img/2dd54e54c3a06447381a1ed766d00b4e_8.png)

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

执行上述代码后，你会看到一个新的DataFrame。例如，1月5日的数据等于1月5日的原始值减去1月4日的原始值。由于1月4日没有前一天的数据，所以其差异值会显示为 `NaN`（Not a Number）。

![](img/2dd54e54c3a06447381a1ed766d00b4e_10.png)

`.diff()` 方法默认计算与前一天的差异。我们还可以通过指定 `periods` 参数来计算间隔多天的差异，例如计算与两天前或一周前的差异。养成查阅API文档的习惯是学习编程的重要方法。

![](img/2dd54e54c3a06447381a1ed766d00b4e_12.png)

## 计算每日变化率

了解了差异值后，我们进一步思考：从1元涨到2元（涨了1元）和从10000元涨到10001元（也涨了1元），意义相同吗？显然不同。前者翻了一倍，后者几乎没变。因此，我们常常更关心“变化率”或“增长率”。

![](img/2dd54e54c3a06447381a1ed766d00b4e_14.png)

变化率的计算公式为：
**变化率 = (后一天的值 - 前一天的值) / 前一天的值**

![](img/2dd54e54c3a06447381a1ed766d00b4e_16.png)

在Pandas中，我们可以使用 `.pct_change()` 方法来直接计算这个百分比变化。

![](img/2dd54e54c3a06447381a1ed766d00b4e_18.png)

```python
# 计算每日变化率（增长率）
pct_change_data = data.pct_change()
print(pct_change_data.head())
```

计算后，数据会从具体的涨跌金额转换为百分比。通常股价的日变化率很小，可能只有百分之零点几。我们可以使用 `.round(3)` 来保留三位小数，让结果更清晰。

![](img/2dd54e54c3a06447381a1ed766d00b4e_20.png)

## 对变化率进行统计分析

得到了每日的变化率序列后，我们可以对其进行统计分析，以了解整体趋势。

例如，我们可以计算所有交易日各指标的平均变化率。

![](img/2dd54e54c3a06447381a1ed766d00b4e_22.png)

![](img/2dd54e54c3a06447381a1ed766d00b4e_24.png)

```python
# 计算各指标的平均变化率
mean_pct_change = pct_change_data.mean()
print(mean_pct_change)
```

![](img/2dd54e54c3a06447381a1ed766d00b4e_26.png)

## 可视化平均变化率

![](img/2dd54e54c3a06447381a1ed766d00b4e_28.png)

![](img/2dd54e54c3a06447381a1ed766d00b4e_30.png)

为了更直观地展示各指标的平均变化率，我们可以将其可视化。由于平均变化率是针对每个指标的一个单一数值，使用条形图（Bar Chart）最为合适。

![](img/2dd54e54c3a06447381a1ed766d00b4e_32.png)

以下是使用条形图进行可视化的步骤：

![](img/2dd54e54c3a06447381a1ed766d00b4e_34.png)

1.  使用 `.plot()` 方法。
2.  通过 `kind=‘bar’` 参数指定图表类型为条形图。
3.  通过 `figsize` 参数设置图表大小。

```python
# 绘制平均变化率的条形图
mean_pct_change.plot(kind='bar', figsize=(16, 8))
```

生成的图表会清晰地展示出各个指标（如开盘价、收盘价、成交量等）在观察期内的平均日变化率。Y轴的值通常较小，这符合金融市场的日常波动特征。

![](img/2dd54e54c3a06447381a1ed766d00b4e_36.png)

通过这张图，我们可以一目了然地看出哪些指标在整体上呈现增长趋势（正的平均变化率），哪些呈现下降趋势。

![](img/2dd54e54c3a06447381a1ed766d00b4e_38.png)
