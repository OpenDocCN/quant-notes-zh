## 数据转换与聚合

上一节我们介绍了如何合并数据以创建影子价格。本节中我们来看看对DataFrame进行数据转换和聚合的方法。

数据转换类似于数据库操作。进行聚合时，可以将多列数据取出并进行汇总计算。Pandas内置了多种聚合函数，同时也支持用户使用自定义函数，例如通过`lambda`表达式来定义。

以下是常见的聚合操作示例代码：
```python
# 使用内置函数进行聚合，如求和、求均值
df.groupby('column').sum()
df.groupby('column').mean()

# 使用lambda自定义聚合函数
df.groupby('column').agg(lambda x: x.max() - x.min())
```

## 数据提取与描述

在获得数据后，我们需要从中提取有价值的信息。我们可以提取数据的前几行、后几行，或者使用`describe`方法快速获取数据的统计描述。Pandas内置了这些便捷的数据概览功能。

以下是数据提取与描述的常用方法：
*   `df.head(n)`：提取数据框的前n行。
*   `df.tail(n)`：提取数据框的后n行。
*   `df.describe()`：生成数据的描述性统计摘要，包括计数、均值、标准差、最小值、四分位数和最大值。

## 数据定位方法

Pandas提供了两种高效的数据定位方法：`loc`和`iloc`。它们可以快速、精确地定位到DataFrame中的特定数据。

以下是两种定位方法的区别：
*   **`df.loc[]`**：基于行和列的**标签**进行索引。
*   **`df.iloc[]`**：基于行和列的**整数位置**进行索引。

## 金融回报率计算初步

掌握了基本的数据操作后，我们就可以进入金融数据分析的领域。下一节的核心内容是计算资产回报率。

在金融中，资产回报率是一个关键指标。我们将学习如何使用Pandas高效地计算它。其基本公式为：
**回报率 = (当期价格 - 上期价格) / 上期价格**

本节课中我们一起学习了Pandas的数据转换、聚合、提取与定位方法，并引出了金融数据分析中计算资产回报率的核心概念。这些基础操作是进行更复杂金融建模和策略回测的基石。下一节课，我们将从“计算资产回报率”这一章开始，深入实践如何使用Pandas进行具体的金融计算。
