## 概述
在本节课中，我们将要学习NumPy的基础知识。NumPy是Python数据分析的核心库，它提供了高性能的多维数组对象和用于处理这些数组的工具。理解NumPy是进行数据清洗、金融量化分析等高级操作的基石。

## NumPy是什么
NumPy为我们提供了数据处理的基础，这个基础主要是运算的基础。运算不仅包括加减乘除，还包括创建、访问、类型转换、连接、合并、聚合、排序等操作，以及其运算机制，例如广播机制。这些都属于NumPy提供的基础功能。如果我们想进行业务分析，必须依赖于NumPy，所以NumPy是基础中的基础。

## 数组与列表的区别
NumPy主要通过一种类型来进行运算，这个类型叫**数组**，注意措辞，它叫数组，而不叫列表。数组一般用`array`表示，列表一般用`list`表示。

我们需要明确几个概念。Python列表对数据类型没有要求，列表中可以存放任意类型，例如字符串、整数、小数。列表是一个有序的集合，集合内的元素可以是任意类型的数据组合。

对于数组则不同。数组对数据类型有要求，数据类型必须统一，并且也必须是有序的集合。因此，包含不同类型数据的集合不能称之为数组，因为数组必须将数据进行统一存储。

不同的数据结构，如顺序表和链表，在查找、插入、删除等操作上的效率不同。顺序表在查找时效率高，链表在插入时效率高。数组一般采用顺序表结构，而列表可能采用链表结构。最明显的区别是它们对数据类型的要求不同。

NumPy提供的是数组类型，因此必须确保数组内的数据类型是统一的。

## 构造NumPy数组
### 从Python列表构造
第一种构造NumPy数组的方式是使用Python列表。

```python
L1 = [1, 2, 3, 4, 5]
N1 = np.array(L1)
print(N1)
```

如果列表中的元素类型不一致，NumPy会强制进行类型统一。

```python
L2 = [1, 2, 3.14, 2.68]
N2 = np.array(L2)
print(N2)
```

类型统一的优先级是：对象类型 > 浮点类型 > 整数类型。如果列表中有字符串，所有元素会被统一为字符串类型。

```python
L3 = [‘hello‘, 1, 2, 3.14]
N3 = np.array(L3)
print(N3)
```

这种方式主要用来强调NumPy数组的数据类型必须一致。我们一般不会用这种方式来构造数组，而是使用NumPy提供的快捷函数。

### 使用快捷函数构造数组
NumPy主要提供数组来进行矩阵运算。矩阵就是数组，只是可能是高维数组。NumPy可以构造任意维度的数组。

![](img/36e195615343b99475207dde75d0172a_1.png)

![](img/36e195615343b99475207dde75d0172a_3.png)

#### `np.ones` 函数
`np.ones` 函数用1来填充一个矩阵（数组）。

```python
# 创建一个2行2列的二维数组，元素全为1（浮点型）
arr1 = np.ones((2, 2))
print(arr1)

# 创建一个3行2列的二维数组
arr2 = np.ones((3, 2))
print(arr2)

![](img/36e195615343b99475207dde75d0172a_5.png)

![](img/36e195615343b99475207dde75d0172a_7.png)

# 指定数据类型为整数
arr3 = np.ones((3, 2), dtype=np.int)
print(arr3)

![](img/36e195615343b99475207dde75d0172a_9.png)

# 创建一个三维数组：由3个5行2列的二维数组构成
arr4 = np.ones((3, 5, 2))
print(arr4)
```

参数说明：
*   **`shape`**: 数组的形状，用元组描述每个维度的数据量。
*   **`dtype`**: 数据类型，建议使用`np.int`、`np.float`等NumPy类型以保证兼容性。
*   **`order`**: 与内存存储方式有关，通常使用默认值即可。

#### 理解高维数组
理解高维数组可以将其映射到现实场景。例如，一个人是一个点，一排人是一维，一个班级是二维，多个班级是三维，多个楼层是四维，以此类推。对于数据分析，业务表通常是二维的（行和列），但处理图像（三维：高、宽、颜色通道）或视频（四维：帧序列）时可能会遇到更高维度的数据。

#### `np.zeros` 和 `np.full` 函数
`np.zeros` 函数用0填充数组，`np.full` 函数用指定的值填充数组。

```python
# 创建一个3行3列的全0数组
arr_zeros = np.zeros((3, 3))
print(arr_zeros)

# 创建一个3行3列的全6数组
arr_full = np.full((3, 3), fill_value=6)
print(arr_full)
```
