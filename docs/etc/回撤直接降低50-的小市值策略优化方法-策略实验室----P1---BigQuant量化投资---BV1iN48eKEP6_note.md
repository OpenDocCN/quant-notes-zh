## 什么是小市值策略？📈

![](img/27e2dee9cca11d7bbcf5b1767abac2ce_5.png)

上一节我们介绍了课程概述，本节中我们来看看小市值策略的具体定义。

![](img/27e2dee9cca11d7bbcf5b1767abac2ce_7.png)

小市值策略是量化金融与AI投资领域中最基础的策略之一。这个策略的核心是投资市值较小的公司股票。在巴菲特的早年投资生涯中，他也曾通过购买美国市场的“垃圾股”赚取第一桶金。因此，小市值策略可以理解为投资于市场认为质量较低、市值较小的公司。

在量化领域，市值是一个公认的强风格因子。著名的法玛三因子模型中就包含市值因子。在A股市场，尤其是在过去十年间，小市值风格是表现最强的风格之一。所谓“风格”，指的是市场在特定时期内的整体表现倾向。在过去一段时间，A股市场中小市值公司的增长幅度普遍优于大市值公司。

这个策略不仅受到量化爱好者的关注，许多专业投资机构也会在产品的风险暴露控制中使用市值因子，这证明了该因子的有效性。市值因子虽不能算作长期绝对稳定的策略，但它是一个强有力的风格因子。

## 小市值策略的收益来源 💡

理解了策略的定义后，本节我们来深入探讨其收益来源。这有助于我们理解策略为何有效。

小市值策略的收益主要来源于A股市场以散户为主导的特性。大多数收益并非来自公司基本面的根本性改善，而是来自市场情绪的推动和“公允价值”的变动。

以下是两种主要的收益类型：

1.  **基本面增长收益**：公司因业务真实改善而价值提升。例如，在电动车行业发展初期，相关电池供应商（如天齐锂业、宁德时代）因需求暴增而实现收入和股价的实质性增长。
2.  **公允价值变动收益**：公司基本面未变，但市场情绪（如炒作热点）推动股价短期波动。例如，某公司与热门概念（如自动驾驶）产生关联传闻，引发股价连续涨停，尽管公司随后澄清并无实际关系。这种股价变动与公司实际经营状况脱钩，是按天甚至按分钟发生的。

在小市值公司中，公允价值变动收益往往超过大市值公司。原因在于小市值公司流通盘小、成交额低，少量资金即可显著影响股价（例如，用1000万拉动日成交额仅1000万的股票涨停）。而大市值公司所需资金量巨大，难以操纵。

![](img/27e2dee9cca11d7bbcf5b1767abac2ce_9.png)

本质上，小市值策略赚取的是市场情绪和资金推动带来的“公允价值变动”的钱。参与者主要有三类：
*   **坐庄者**：主动操纵股价（高风险且违法）。
*   **联合坐庄者**：如直播间号召散户共同炒作（游走于灰色地带）。
*   **埋伏跟随者**：通过策略提前买入可能被炒作的小市值股票，等待资金拉升后卖出。**小市值策略通常指的是第三种方式**。

该策略的走强有其背景。在2018年后，以“白马股”、“绩优股”为主的风格走软，相关公司估值过高。市场资金开始寻找新的方向，小盘股因其体量小、易炒作的特点受到关注。随着量化资管规模的扩大，更多资金涌入小市值风格，进一步推高了其收益，形成了风格与资金规模相互强化的效应。

## 小市值策略的传统实现方法 💻

了解了收益逻辑，本节我们来看看如何在技术上实现一个传统的小市值策略。

传统小市值策略的实现方法非常直接：在选股时，按照市值因子进行升序排序（即从小到大），选取市值最小的一批股票，然后等权重买入并定期调仓。

以下是一个基于SQL和Python的简化策略代码框架，展示了核心的选股逻辑：

![](img/27e2dee9cca11d7bbcf5b1767abac2ce_13.png)

![](img/27e2dee9cca11d7bbcf5b1767abac2ce_15.png)

```python
# 假设的配置部分，用于设置策略参数
configuration = {
    'start_date': '2019-01-01',
    'end_date': '2023-12-31',
    'universe': '全A股', # 股票池
    'max_stock_num': 50, # 最大持仓数量
    'rebalance_freq': 'monthly' # 调仓频率
}

# SQL查询语句，用于获取股票数据和市值因子
sql_query = """
SELECT
    stock_code,
    market_cap -- 市值因子字段
FROM
    stock_daily_data
WHERE
    trade_date = '{date}' -- 动态传入交易日
    AND is_st = 0 -- 非ST股票
    AND list_days > 180 -- 上市超过180天
ORDER BY
    market_cap ASC -- 按市值升序排序
LIMIT {limit} -- 限制选取前N只
"""
```
**代码解释**：这段伪代码演示了策略的核心。通过SQL从数据库中选择特定日期、满足基本条件的股票，并按其`market_cap`（市值）从小到大排序，选取前N只作为当期持仓。

使用这种简单方法构建的策略，历史回测可能显示出较高的年化收益率（例如17%），但同时也会伴随较大的波动和回撤（例如最大回撤可达46%）。这反映了该策略高风险、高波动的特性。许多市场中性的指数增强产品，其超额收益部分也常与市值风格暴露相关。

## 如何避免策略过拟合？ 🚫

上一节我们看到了传统方法的简单与高风险，本节中我们来讨论一个在策略开发中至关重要的问题：过拟合。

过拟合是指策略在历史数据上表现优异，但在未来实盘中失效的现象。在量化领域，这通常表现为在回测中加入了过多、过于复杂的过滤条件（例如：必须连续上涨X天、必须在特定技术指标发出信号时买入等），这些条件恰好完美匹配了历史行情，但缺乏逻辑支撑，无法在未来重复。

**核心问题在于“未来函数”或“数据窥探”**。我们站在现在回顾历史，知道所有数据，但实盘操作时面对的是未知的未来。如果一个策略的回测曲线异常完美（例如年化收益远超市场合理水平），就需要警惕。一个简单的思考方式是：如果真有一个稳定年化30%的策略，开发者为何要公开分享而不是自己融资加杠杆操作？

对于小市值策略，避免过拟合的关键在于：
1.  **理解收益来源**：明确策略赚的是风格和情绪的钱，而非通过复杂条件“预测”股价。
2.  **保持策略简洁**：像上一节展示的单纯按市值排序，虽然回撤大，但反而是相对“干净”、未过度优化的策略。
3.  **考虑时间序列特性**：评估策略时，要问自己“在回测起始的那个时点，我是否有可能相信并开始运行这个策略？”如果策略在起始点之前表现很差，那么从起点开始运行就是不现实的假设。
4.  **警惕超高收益**：对远超市场平均水平的回测结果保持怀疑，用常识判断其合理性。

## 小市值策略的高级优化方法 ⚙️

最后，我们介绍一种能有效降低风险、提升策略稳健性的高级做法——构建高频量价市值因子。

传统方法直接使用财报或日频市值数据。而高级方法则尝试仅利用盘中的高频交易数据（如分钟级K线）来间接捕捉“市值”风格。其核心思想是：在A股市值分布极度偏斜（大量小市值公司）的背景下，任何与市场平均值的比较或计算，都可能隐含着市值风格。

以下简述其构建逻辑（概念性描述）：
1.  **使用分钟级数据**：分析每只股票分钟线的成交量变化。
2.  **识别异常波动**：通过计算成交量差分的均值和标准差，定位日内成交量突然放大的时段（可能代表资金介入）。
3.  **提取价量特征**：统计放量时段之后几分钟的收益率表现，计算其均值和波动率。
4.  **合成因子**：将这些高频特征在时间序列上进行平滑、标准化处理，并与全市场股票的平均表现进行比较，最终合成一个**高频量价市值因子**。

**公式/逻辑表示**：
`高频市值因子 ≈ F( 个股分钟成交量特征， 个股放量后价格行为， 全市场同期平均表现 )`
其中 `F` 代表一系列标准化、平滑和对比的计算过程。

![](img/27e2dee9cca11d7bbcf5b1767abac2ce_17.png)

![](img/27e2dee9cca11d7bbcf5b1767abac2ce_19.png)

这种方法的优势在于：
*   **降低回撤**：通过更细腻的量价数据捕捉资金行为，可能更早察觉风格转换或风险，从而将最大回撤从传统方法的46%显著降低（例如降至29%）。
*   **改善持有体验**：策略的收益波动率（如从28%降至23%）降低，曲线更加平滑。
*   **逻辑深化**：因子来源于市场微观交易行为，而非简单的静态市值数据，可能更具信息含量。

![](img/27e2dee9cca11d7bbcf5b1767abac2ce_21.png)

当然，这种方法的计算复杂度高，需要处理高频数据，对计算资源要求更大。在实际应用中，开发者可以直接在提供此类因子的平台上调用，或自行研发。
