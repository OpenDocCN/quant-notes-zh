# 量化交易完全可自学教程：P30：31.3Sigma方法实例 📊

在本节课中，我们将要学习一种基于统计学原理的数据预处理方法——3Sigma方法。该方法用于识别并处理数据中的异常值（极值），是量化交易数据清洗中常用且有效的手段。

上一节我们介绍了其他去极值方法，本节中我们来看看基于正态分布假设的3Sigma方法。

## 原理介绍 📈

3Sigma方法的原理基于正态分布（也称高斯分布）。正态分布是一种常见的概率分布，其形状呈对称的钟形曲线。

如果大家理解起来比较麻烦，可以将其想象为一个“正常”的分布。例如，你去银行贷款，银行多借或少借你少量金额（如三块五块）是正常情况；但如果多借或少借你巨额资金（如十万八万），就属于不正常的情况。

在正态分布中，数据大多集中在均值附近。距离均值越远的数据点，出现的概率就越低，也就越有可能是我们需要处理的异常值或“极值”。

下图展示了一个标准正态分布（均值为0，标准差为1）。图中阴影部分面积代表事件发生的概率。

![](img/56cca560d16ceb8572b33233c328cc27_1.png)

![](img/56cca560d16ceb8572b73233c328cc27_1.png)

可以看到，在均值（μ）附近（例如 μ ± 1σ）区间内，数据出现的概率很高。随着区间向两端扩展（例如 μ ± 2σ， μ ± 3σ），所覆盖的数据点出现的概率逐渐降低。

**核心思想**：极值通常落在远离均值的区域。3Sigma方法就是通过设定一个以均值为中心、上下浮动若干倍标准差的区间，将落在此区间外的数据视为异常值并进行处理。

## 3Sigma区间定义 📏

“3Sigma”指的是均值加减三倍标准差所构成的区间。具体公式如下：

*   **上限**：`upper_bound = mean + n * std`
*   **下限**：`lower_bound = mean - n * std`

其中：
*   `mean` 是数据的平均值。
*   `std` 是数据的标准差。
*   `n` 是系数，通常取3。当 `n=3` 时，在理想的正态分布下，该区间将包含约99.7%的数据。

![](img/56cca560d16ceb8572b73233c328cc27_3.png)

![](img/56cca560d16ceb8572b73233c328cc27_3.png)

这意味着，如果我们认为数据服从或近似服从正态分布，那么可以将最正常的99.7%的数据保留在区间内，而将剩余约0.3%的数据视为极值进行规范化处理。

![](img/56cca560d16ceb8572b73233c328cc27_5.png)

## 代码实现 💻

![](img/56cca560d16ceb8572b73233c328cc27_7.png)

理解了原理后，我们来实现3Sigma去极值函数。这个函数将计算指定数据列的上下限，并对超出范围的值进行截断处理。

以下是函数实现步骤：

![](img/56cca560d16ceb8572b73233c328cc27_9.png)

首先，我们需要计算数据列的均值和标准差。

```python
mean = data.mean()   # 计算均值
std = data.std()     # 计算标准差
```

![](img/56cca560d16ceb8572b73233c328cc27_11.png)

接着，根据传入的倍数 `n` 计算上下限。

![](img/56cca560d16ceb8572b73233c328cc27_13.png)

```python
upper_bound = mean + n * std  # 计算上限
lower_bound = mean - n * std  # 计算下限
```

最后，将数据中所有大于上限的值设置为上限，所有小于下限的值设置为下限。

![](img/56cca560d16ceb8572b73233c328cc27_15.png)

```python
data_clipped = data.clip(lower_bound, upper_bound)  # 截断处理
return data_clipped, lower_bound, upper_bound
```

## 核心要点总结 ✅

![](img/56cca560d16ceb8572b73233c328cc27_39.png)

本节课中我们一起学习了3Sigma去极值方法。

![](img/56cca560d16ceb8572b73233c328cc27_41.png)

![](img/56cca560d16ceb8572b73233c328cc27_43.png)

*   **主要目的**：识别并规范数据中的异常值。
*   **核心原理**：基于数据服从正态分布的假设，认为距离均值超过 `n` 倍标准差的数据点概率极低，属于异常。
*   **关键公式**：
    *   上限：`mean + n * std`
    *   下限：`mean - n * std`
*   **操作实质**：**去极值不是删除数据**，而是通过设定下界和上界，将所有数值规范到指定区间内。
