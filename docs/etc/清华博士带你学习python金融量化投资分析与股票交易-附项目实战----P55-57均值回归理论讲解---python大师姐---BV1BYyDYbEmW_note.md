# 量化投资策略：P55：均值回归理论讲解 📈

在本节课中，我们将要学习一个重要的量化策略概念——均值回归理论。我们将探讨其核心思想、如何将其转化为可计算的指标，并了解如何利用该理论构建一个简单的选股策略。

## 概述

![](img/ed1da0191510816351a60b5096ff5b83_1.png)

均值回归理论的核心思想可以概括为一句话：价格偏离其均值后，最终会向均值回归。这好比说，跌下去的股票迟早会涨回来。本节我们将深入理解这一理论，并学习如何将其应用于量化选股。

## 理论核心

上一节我们介绍了量化策略的基本框架，本节中我们来看看均值回归理论的具体含义。

均值回归理论基于以下观测：价格的波动通常会以其移动平均线为中心。也就是说，当标的价格由于波动而偏离其移动均线时，它将进行调整并重新回归到均线附近。

这里的“均线”指的是移动平均线，例如10日均线或60日均线，它代表了过去一段时间内价格的平均值。因为均线是一个平均值，所以价格的波动通常会围绕均线进行，不会长期、大幅度地偏离。

## 偏离度的量化

理解了理论核心后，我们需要一个方法来量化“偏离”的程度。简单地用价格与均线的差值（例如：均线10元，现价8元，差值为2元）并不理想，因为它无法在不同价格的股票间进行比较。

因此，我们引入一个百分比形式的偏离度公式，以更合理地衡量偏离程度：

**偏离度 = (当前价格 - 移动均线) / 移动均线**

![](img/ed1da0191510816351a60b5096ff5b83_3.png)

用代码表示可以是：
```python
deviation = (current_price - moving_average) / moving_average
```
*   **当前价格 (P)**: 股票在计算时的市场价格。
*   **移动均线 (MA)**: 所选周期（如N日）的移动平均价格。

![](img/ed1da0191510816351a60b5096ff5b83_5.png)

计算结果为负值表示价格低于均线（向下偏离），为正值则表示价格高于均线（向上偏离）。这个比率使得不同价位的股票之间的偏离程度具有可比性。

## 策略应用：择时与选股

定义了偏离度之后，我们可以探讨如何应用这一理论。主要的应用方向有两个：择时交易和选股策略。

### 关于择时交易

择时交易试图找到最佳的买卖点。一个自然的想法是：在向下偏离度最大（即价格远低于均线）时买入，在向上偏离度最大时卖出。

![](img/ed1da0191510816351a60b5096ff5b83_7.png)

然而，这面临一个关键挑战：我们无法预知当前的偏离度是否已达到“最大”。价格可能在今天触底反弹，也可能在明天继续下跌。因此，单纯依靠均值回归理论进行精确择时较为困难。当然，可以通过历史数据分析，设定一个经验性的偏离度阈值（例如，向下偏离超过20%时买入）来构建策略，但这需要大量的参数测试与优化。

### 均值回归选股策略

![](img/ed1da0191510816351a60b5096ff5b83_9.png)

![](img/ed1da0191510816351a60b5096ff5b83_11.png)

相较于择时，均值回归理论更常用于构建选股策略。这个策略不预测单只股票的具体拐点，而是在一组股票中寻找相对机会。

策略思路如下：在每个调仓日（例如每月或每季度），计算股票池中所有股票相对于其N日均线的偏离度。然后，买入偏离度最低的M只股票（即价格相对其均线被低估最严重的股票）。其背后的逻辑是，相信这些被“过度”低估的股票，向均值回归的潜力更大。

以下是该策略的简要步骤：
1.  确定股票池、均线周期（如20日）和调仓周期。
2.  在调仓日，为每只股票计算移动均线（MA）。
3.  根据公式计算每只股票的偏离度。
4.  按照偏离度从低到高排序，选出偏离度最低的M只股票。
5.  买入这些股票，并持有至下一个调仓日。

## 总结

本节课中我们一起学习了均值回归理论。我们从其通俗的理解“跌下去会涨回来”出发，逐步深入到其数学定义，即通过 **`(P - MA) / MA`** 来计算价格相对于均线的偏离度。我们分析了该理论应用于择时交易的难点，并重点介绍了如何将其作为一个有效的因子，用于构建“买入相对低估最严重股票”的选股策略。记住，任何策略都不是绝对的，在实际应用中都需要结合历史数据进行严谨的回测和验证。
