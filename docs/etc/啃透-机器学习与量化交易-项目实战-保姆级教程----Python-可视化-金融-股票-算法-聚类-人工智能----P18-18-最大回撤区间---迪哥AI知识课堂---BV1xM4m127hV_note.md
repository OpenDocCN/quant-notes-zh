在策略分析中，除了关注收益，评估风险同样至关重要。最大回撤就是这样一个描述投资过程中可能遭遇的最糟糕情况的指标。

## 什么是最大回撤？

最大回撤描述的是在投资过程中，资产净值从某个高点下降到后续最低点的最大幅度。它衡量的是投资者可能面临的最大账面亏损。

![](img/6f2901319f619e52970833d79c805d63_1.png)

具体来说，它考察的是投资历程中，从任一峰值到之后任一谷值下跌的最严重情况。这个“最惨”的下跌区间及其幅度，就是最大回撤。

例如，在2008年股市大跌期间，上证指数从约6000点跌至约1600点，这个过程就构成了一个巨大的回撤区间，给投资者带来了严重亏损。最大回撤指标能帮助我们量化此类极端风险。

## 最大回撤的计算公式

最大回撤的数学定义如下：对于一段时间序列中的价格（或净值）`P_i`（i 表示第 i 天），最大回撤 `MaxDrawdown` 是遍历所有时间点对 (i, j) （其中 j > i）后，找到的最大跌幅。

其计算公式为：

**`MaxDrawdown = max_{i, j (j>i)} [ (P_i - P_j) / P_i ]`**

![](img/6f2901319f619e52970833d79c805d63_5.png)

在这个公式中：
*   `P_i` 代表第 i 天的价格（前期高点）。
*   `P_j` 代表第 i 天之后第 j 天的价格（后期低点）。
*   公式计算的是从 `P_i` 到 `P_j` 的相对回撤率，并取所有可能 (i, j) 组合中的最大值。

![](img/6f2901319f619e52970833d79c805d63_3.png)

![](img/6f2901319f619e52970833d79c805d63_7.png)

![](img/6f2901319f619e52970833d79c805d63_9.png)

这个值是一个百分比，直观反映了“缩水”的程度。值越大，表明历史上资产下跌的风险越高。

## 如何用Python计算最大回撤？

理解了概念后，我们来看看如何用Python的Pandas库进行计算。核心思路是：先计算历史价格序列的累积最大值，然后用这个累积最大值序列减去当前价格序列，最后找出差值相对于累积最大值的最大比例。

![](img/6f2901319f619e52970833d79c805d63_11.png)

以下是计算步骤的分解：

![](img/6f2901319f619e52970833d79c805d63_13.png)

![](img/6f2901319f619e52970833d79c805d63_15.png)

首先，我们需要一个价格序列。这里假设我们有一个名为`df`的DataFrame，其中`close`列是每日收盘价。

```python
import pandas as pd

![](img/6f2901319f619e52970833d79c805d63_17.png)

# 假设 df[‘close’] 是我们的价格序列
price_series = df[‘close’]
```

![](img/6f2901319f619e52970833d79c805d63_19.png)

第一步，计算价格序列的累积最大值（`cummax`）。这个操作会生成一个新序列，其中每个位置的值都是截至当前日期为止的历史最高价。

![](img/6f2901319f619e52970833d79c805d63_21.png)

```python
# 计算累积最大值
cumulative_max = price_series.cummax()
```

![](img/6f2901319f619e52970833d79c805d63_23.png)

第二步，用累积最大值序列减去当前的实际价格序列，得到每日的“潜在最大亏损额”。

![](img/6f2901319f619e52970833d79c805d63_25.png)

![](img/6f2901319f619e52970833d79c805d63_27.png)

```python
# 计算每日回撤（高点与当前点的差值）
drawdown_value = cumulative_max - price_series
```

![](img/6f2901319f619e52970833d79c805d63_29.png)

第三步，将每日回撤值除以对应的累积最大值（即前期高点 `P_i`），得到每日的回撤率。

```python
# 计算每日回撤率
drawdown_ratio = drawdown_value / cumulative_max
```

![](img/6f2901319f619e52970833d79c805d63_31.png)

![](img/6f2901319f619e52970833d79c805d63_33.png)

第四步，在所有每日回撤率中找出最大值，这就是整个考察期内的**最大回撤**。

```python
# 计算最大回撤
max_drawdown = drawdown_ratio.max()
```

![](img/6f2901319f619e52970833d79c805d63_35.png)

为了清晰，我们可以将这个过程整合成一个函数：

![](img/6f2901319f619e52970833d79c805d63_37.png)

```python
def calculate_max_drawdown(price_series):
    """
    计算价格序列的最大回撤。
    参数:
        price_series: pandas Series，价格序列。
    返回:
        最大回撤率（浮点数）。
    """
    cumulative_max = price_series.cummax()
    drawdown = (cumulative_max - price_series) / cumulative_max
    max_drawdown = drawdown.max()
    return max_drawdown

![](img/6f2901319f619e52970833d79c805d63_39.png)

# 使用函数
max_dd = calculate_max_drawdown(df[‘close’])
print(f”最大回撤为：{max_dd:.2%}”)
```

通过以上代码，我们就可以计算出任意价格序列在历史中经历的最大回撤率。

![](img/6f2901319f619e52970833d79c805d63_41.png)

## 总结

![](img/6f2901319f619e52970833d79c805d63_51.png)

本节课中我们一起学习了最大回撤这一核心风险指标。
*   **定义**：最大回撤描述了投资过程中可能出现的最大峰值到谷值的跌幅，是衡量策略下行风险的关键指标。
*   **意义**：它反映了投资者需要承受的最大心理压力和市场极端风险。一个策略即使最终盈利，但如果最大回撤过大，也可能因中途止损或心理崩溃而无法坚持。
*   **计算**：其核心公式为 **`MaxDrawdown = max( (P_i - P_j) / P_i )`**，我们可以通过计算价格序列的累积最大值来高效实现。
*   **应用**：在评估量化交易策略时，必须将最大回撤与收益率、夏普比率等指标结合分析，才能全面衡量策略的风险收益特征。

![](img/6f2901319f619e52970833d79c805d63_53.png)

记住，关注最大回撤就是关注投资旅程中的“最坏情况”，做好风险管理是长期稳定盈利的基石。
