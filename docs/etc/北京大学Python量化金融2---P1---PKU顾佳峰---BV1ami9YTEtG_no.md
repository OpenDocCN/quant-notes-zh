## 概述

![](img/bb03589ecd9c83c5c47f9ff73829a26e_1.png)

在投资过程中，资产价格并非总是上涨，也会出现下跌。**回撤**指的是从资产价格的历史最高点到之后某个低点的下跌幅度。而**最大回撤**，则是在选定时间段内，资产价格从峰值到谷底的最大可能跌幅。它是衡量投资策略下行风险和投资者可能承受的最大账面亏损的关键指标。

上一节我们介绍了波动率等风险概念，本节中我们来看看如何具体量化一段投资期内可能面临的最糟糕情况。

## 最大回撤的定义与计算

最大回撤的计算目标是：在给定的历史数据序列中，找到这样一个价格高点（`P_peak`）和其后的一个价格低点（`P_trough`），使得其相对跌幅最大。

其计算公式可以表示为：
**最大回撤 = (P_peak - P_trough) / P_peak**

计算过程通常通过遍历价格序列来实现。以下是核心的计算逻辑，我们可以用伪代码来描述：

```python
# 假设 prices 是一个包含历史价格（如收盘价）的列表
max_drawdown = 0
peak = prices[0]

for price in prices:
    if price > peak:
        peak = price  # 更新峰值
    # 计算当前价格相对于当前峰值的回撤
    drawdown = (peak - price) / peak
    if drawdown > max_drawdown:
        max_drawdown = drawdown  # 更新最大回撤值

return max_drawdown
```

## 实战：计算茅台股价的最大回撤

接下来，我们使用贵州茅台（股票代码：`600519.SH`）的历史日线数据来进行演示。我们首先获取其收盘价序列。

以下是数据处理和计算的关键步骤：

1.  **数据准备**：获取茅台的历史交易日和收盘价数据。
2.  **计算回撤**：遍历收盘价序列，动态更新遇到的历史最高价，并计算当前价格相对于该最高价的回撤率。
3.  **定位区间**：记录最大回撤值及其发生的起止日期（峰值日和谷底日）。

运行计算代码后，我们得到如下结果（数据为示例）：
-   **回撤开始日期（峰值）**：2月18日，收盘价为 `2601.00` 元。
-   **最大回撤发生日期（谷底）**：8月23日。
-   **最大回撤幅度**：**41%**。这意味着，如果投资者在2月18日的峰值买入并持有至8月23日的谷底，期间将承受41%的账面亏损。
