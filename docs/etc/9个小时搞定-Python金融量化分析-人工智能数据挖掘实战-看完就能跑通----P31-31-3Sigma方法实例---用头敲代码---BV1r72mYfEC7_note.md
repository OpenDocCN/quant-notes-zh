## 概述
在本节课程中，我们将学习一种在数据预处理中常用的去极值方法——3-Sigma方法。我们将理解其基于正态分布（高斯分布）的理论原理，并通过Python代码实现它，用于识别和处理金融数据中的异常值。

## 3-Sigma方法原理 📈

上一节我们介绍了分位数和中位数绝对偏差法，本节中我们来看看基于统计分布的3-Sigma方法。

它的原理基于正态分布。正态分布是一种常见的概率分布，形状呈钟形。在正态分布中，大多数数据点会聚集在平均值（均值）附近，而远离均值的数据点出现的概率则较低。

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_1.png)

为了便于理解，可以想象一个例子：你去银行贷款，银行批准给你的额度围绕某个平均金额上下波动。小幅度的波动（如多借或少借几块钱）是常见的、正常的；但大幅度的波动（如多借或少借十万八万）则是不常见的、异常的。

在正态分布中，数据偏离均值的程度可以用标准差（σ，Sigma）来衡量。3-Sigma规则指出：在正态分布下，约有99.7%的数据会落在均值加减3倍标准差的区间内。反之，只有约0.3%的数据会落在这个区间之外，这些数据通常被视为极值或异常值。

**核心公式**：
*   **上限** = 均值 + n × 标准差
*   **下限** = 均值 - n × 标准差

其中，`n`通常取3，代表3倍标准差。我们可以通过调整`n`来控制被视为“正常”的数据范围大小。

## 代码实现 💻

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_5.png)

理解了原理后，我们来看看如何用Python代码实现3-Sigma方法。其步骤非常清晰：计算数据的均值和标准差，然后根据公式计算上下限。

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_7.png)

以下是实现3-Sigma去极值函数的关键步骤：

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_9.png)

1.  **计算均值与标准差**：首先，我们需要计算待处理数据列的均值和标准差。
2.  **计算上下限**：根据公式，使用均值、标准差和指定的倍数`n`计算上限和下限。
3.  **应用限制**：将数据列中所有超过上限的值设置为上限值，所有低于下限的值设置为下限值。

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_11.png)

```python
def sigma_filter(data_series, n=3):
    """
    使用3-Sigma方法进行去极值处理。
    参数:
        data_series: pandas Series，待处理的数据列。
        n: int，标准差的倍数，默认为3。
    返回:
        Series: 处理后的数据。
        tuple: (下限, 上限)
    """
    # 1. 计算均值和标准差
    mean = data_series.mean()
    std = data_series.std()

    # 2. 计算上下限
    upper_limit = mean + n * std
    lower_limit = mean - n * std

    # 3. 应用限制：将超出范围的值截断到边界
    filtered_series = data_series.clip(lower=lower_limit, upper=upper_limit)

    return filtered_series, (lower_limit, upper_limit)
```

## 方法对比与选择 🤔

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_27.png)

至此，我们已经学习了三种主要的去极值方法：分位数法、中位数绝对偏差法（MAD）以及本节介绍的3-Sigma法。

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_29.png)

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_31.png)

不同的方法会计算出不同的上下限。例如，在同一个数据集上，分位数法设定的边界可能在60左右，MAD法可能在50以下，而3-Sigma法可能在50出头。这些差异是正常的，因为它们基于不同的统计假设和鲁棒性。

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_33.png)

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_35.png)

以下是选择方法时的一些考虑：
*   **3-Sigma法**：假设数据服从或近似服从正态分布时效果较好。对异常值本身比较敏感（因为均值和标准差都受异常值影响）。
*   **分位数法**：不依赖分布假设，直接根据数据的位置划分。简单直接，但完全依赖于分位点的选择。
*   **MAD法**：使用中位数代替均值，对异常值的鲁棒性更强，适用于可能包含大量异常值的数据。

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_37.png)

具体选择哪种方法，需要根据实际数据的特性和分析任务的需求来决定。在实践中，可以尝试多种方法并对比结果。

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_39.png)

我们首先从正态分布的概念入手，理解了3-Sigma规则的理论基础：即大部分数据（约99.7%）会落在均值附近3个标准差的范围内。然后，我们一步步实现了该方法的代码，核心是计算`均值 ± n * 标准差`来确定数据的正常范围边界。最后，我们对比了不同去极值方法的特点。

![](img/1c70e89c95c1728f473b5c5f9b69d9e2_45.png)

**关键点回顾**：去极值的目的不是简单地删除数据，而是通过定义合理的**下界**和**上界**，将所有的数据规范到我们预设的合理区间内，从而减少极端值对后续分析模型造成的干扰。这是构建稳健金融量化模型重要的第一步。
