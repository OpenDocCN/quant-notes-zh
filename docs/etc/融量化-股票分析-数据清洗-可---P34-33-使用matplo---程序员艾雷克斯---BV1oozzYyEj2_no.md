## 概述

计算机绘制曲线或图形，本质上是通过连接一系列密集的点来实现的。例如，一个圆在放大后是由许多点构成的。绘制数学函数图像也是同样的原理：我们需要生成一个密集的X值列表，然后根据函数公式计算出对应的Y值列表，最后将这些点连接起来形成图像。

![](img/ff490c6e95552f4226fc8e93e620a3a4_1.png)

## 绘制函数图像的原理

上一节我们介绍了数据可视化的基础，本节中我们来看看如何绘制精确的数学函数曲线。

计算机绘图基于点。所谓的线是由无数个密集的点连接而成。例如，在绘图软件中画一个圆，放大后可以看到它是由许多像素点组成的。绘制函数曲线也是如此。

![](img/ff490c6e95552f4226fc8e93e620a3a4_3.png)

## 生成密集的数据点

以下是生成密集X值数据点的方法。我们将使用NumPy库中的`linspace`函数。

```python
import numpy as np

![](img/ff490c6e95552f4226fc8e93e620a3a4_5.png)

# 使用linspace生成从-100到100的10000个等间距点
x = np.linspace(-100, 100, 10000)
```
`np.linspace`函数的第一个参数是起点，第二个参数是终点，第三个参数是要生成的点的总数。这会产生一个非常密集的数值数组，例如`[-100, -99.979998, ... , 100]`。点数越多，绘制的曲线越平滑。

![](img/ff490c6e95552f4226fc8e93e620a3a4_7.png)

## 计算Y值并绘图

有了X值数组后，我们可以根据数学函数计算对应的Y值。NumPy支持对整个数组进行数学运算，非常方便。

以下是定义三个函数并计算Y值的示例：
1.  Y = X
2.  Y = X²
3.  Y = 3X³ + 5X² + 2X + 1

```python
import matplotlib.pyplot as plt

# 计算三个函数的Y值
y1 = x                     # Y = X
y2 = x ** 2                # Y = X²
y3 = 3*(x**3) + 5*(x**2) + 2*x + 1  # Y = 3X³ + 5X² + 2X + 1

# 绘制三条曲线，并设置颜色和图例标签
plt.plot(x, y1, color='red', label='Y = X')
plt.plot(x, y2, color='green', label='Y = X²')
plt.plot(x, y3, color='blue', label='Y = 3X³ + 5X² + 2X + 1')

# 添加图例
plt.legend()
plt.show()
```

## 调整坐标轴范围

有时不同函数的数值范围差异很大，可能导致某些曲线在图中显示不明显。例如，立方函数`Y = X³`的增长速度远快于线性函数`Y = X`。

这时，我们可以使用`plt.ylim()`函数来手动设置Y轴的范围，以便更清晰地观察所有曲线。

```python
# 设置Y轴的显示范围
plt.ylim(-1000, 1000)
```
通过调整坐标轴范围，可以确保所有曲线都在合适的视野内得到展示。
