## 什么是Series？🤔
Series是Pandas中的一种核心数据对象，它是一种类似于一维数组的对象。你可以把它理解为一个**数组和字典的结合体**。

### 创建Series
创建Series的基本方法是`pd.Series()`。

以下是创建Series的几种方式：
*   **从列表创建**：此时，左侧会自动生成整数索引（0, 1, 2...）。
    ```python
    sr1 = pd.Series([2, 3, 4, 5])
    ```
*   **从列表创建并指定索引**：通过`index`参数，可以为数据指定自定义的标签（类似字典的键）。
    ```python
    sr2 = pd.Series([2, 3, 4, 5], index=[‘A‘, ‘B‘, ‘C‘, ‘D‘])
    ```
*   **从字典创建**：字典的键（key）会自动成为Series的索引。
    ```python
    sr3 = pd.Series({‘A‘: 2, ‘B‘: 3, ‘C‘: 4, ‘D‘: 5})
    ```

## Series的数组（列表）特性
Series继承了许多NumPy数组或Python列表的特性。

![](img/fd873f6324ccc2e5eafb252a92ebdd8e_15.png)

以下是Series支持的数组类操作：
*   **通过整数下标访问**：即使指定了自定义索引（如‘A‘, ‘B‘），依然可以通过原始的整数位置进行访问。
    ```python
    sr2[0]  # 访问第一个元素，返回 2
    ```
*   **与标量运算**：Series中的每个元素都会与标量进行运算。
    ```python
    sr2 * 2  # 返回一个每个元素都乘以2的新Series
    ```
*   **相同大小的Series间运算**：两个长度相同的Series可以进行逐元素的加减乘除等运算。
    ```python
    sr2 + sr2  # 对应位置元素相加
    ```
*   **切片操作**：与列表切片语法一致。
    ```python
    sr2[0:2]  # 切片，返回索引0和1对应的元素
    ```
*   **支持通用函数**：可以使用NumPy的通用函数，如取绝对值、最大值等。
*   **布尔型索引**：通过条件表达式筛选数据。
    ```python
    sr2[sr2 > 3]  # 筛选出值大于3的元素
    ```

## Series的字典特性
Series也具备一些类似字典的操作方式。

以下是Series支持的字典类操作：
*   **通过标签（索引）访问**：使用自定义的索引标签来获取值。
    ```python
    sr2[‘A‘]  # 返回 2
    ```
*   **`in`操作**：判断一个标签是否存在于Series的索引中。
    ```python
    ‘A‘ in sr2  # 返回 True
    ‘Z‘ in sr2  # 返回 False
    ```
*   **花式索引**：传入一个标签列表，可以获取多个对应的值。这是对字典功能的扩展。
    ```python
    sr2[[‘A‘, ‘C‘]]  # 返回标签‘A‘和‘C‘对应的值
    ```
*   **按标签切片**：使用标签进行切片时，切片范围是**前闭后闭**的（包含结束标签）。
    ```python
    sr2[‘A‘:‘C‘]  # 返回标签‘A‘, ‘B‘, ‘C‘对应的值
    ```

**注意**：对一个Series进行`for`循环迭代时，输出的是它的**值**，而不是索引（键）。这与字典的默认行为不同，因为数据分析中我们更常关注数据值本身。

## 获取索引与值
我们可以通过Series的属性来分别获取其索引和值。

以下是获取索引和值的方法：
*   **获取索引**：使用`.index`属性。
    ```python
    sr2.index  # 返回一个Index对象，内容为 [‘A‘, ‘B‘, ‘C‘, ‘D‘]
    ```
*   **获取值**：使用`.values`属性。
    ```python
    sr2.values  # 返回一个NumPy数组，内容为 [2, 3, 4, 5]
    ```

## Series的应用场景
Series结合了有序列表和键值对字典的优点，在实际工作中非常有用。

想象一下这些场景：
*   **股票历史数据**：记录一支股票每天的收盘价。日期可以作为索引（标签），收盘价是值。这样既可以通过日期（标签）快速查询某天的价格，也可以通过整数位置（下标）获取前N天的数据切片。
*   **有序的键值对存储**：当你需要存储一组有序的、且能通过键快速访问的数据时（例如，带有时间戳的传感器读数），使用Series比在列表里存储元组更加方便和高效。

## 总结
本节课中我们一起学习了Pandas的Series对象。我们了解到：
1.  Series是一个一维的、带标签的数组，是数组和字典的结合体。
2.  它继承了数组的特性，如下标访问、切片、运算和布尔索引。
3.  它也具备了字典的特性，如通过标签访问、`in`操作和花式索引。
4.  可以通过`.index`和`.values`属性分别获取其索引和值。
5.  Series非常适合用于表示和处理像时间序列、有序键值对这类具有“标签”的一维数据。
