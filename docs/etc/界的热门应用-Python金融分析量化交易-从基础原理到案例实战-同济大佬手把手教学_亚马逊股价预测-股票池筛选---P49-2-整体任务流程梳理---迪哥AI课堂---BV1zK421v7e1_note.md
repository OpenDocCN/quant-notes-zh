## 打分法的基本原理
打分法的核心思想是为每只股票的每个因子赋予一个分数，然后汇总所有因子的分数得到总分，最后根据总分进行排名和筛选。

以下是打分法的基本计算步骤：

1.  **确定因子方向**：明确每个因子是“越大越好”还是“越小越好”。
2.  **划分区间或排序**：为每个因子的数值划分等级区间，或直接根据数值大小排序。
3.  **分配分数**：根据因子数值所在的区间或排序位置，分配相应的分数。
4.  **汇总分数**：将一只股票在所有因子上的得分相加，得到该股票的总分。
5.  **排序筛选**：对所有股票的总分进行排序，选择排名最高的股票。

## 打分方法详解
以下是两种常见的打分方法：

### 方法一：区间划分法
此方法预先为每个因子的数值划分几个固定的区间，并为每个区间设定分数。

假设因子数值已归一化到 `[0, 1]` 区间，我们可以将其划分为5个等级。

对于 **“越大越好”** 的因子（如因子A、B）：
```python
# 分数映射规则
if value >= 0.8: score = 5
elif value >= 0.6: score = 4
elif value >= 0.4: score = 3
elif value >= 0.2: score = 2
else: score = 1
```

对于 **“越小越好”** 的因子（如因子C、D）：
```python
# 分数映射规则 (与“越大越好”相反)
if value <= 0.2: score = 5
elif value <= 0.4: score = 4
elif value <= 0.6: score = 3
elif value <= 0.8: score = 2
else: score = 1
```

**计算示例**：
假设有股票ID1，其因子值为：A=0.71， B=0.28， C=0.39， D=0.11。
已知：A和B越大越好，C和D越小越好。
*   因子A (0.71): 落入0.6-0.8区间，得4分。
*   因子B (0.28): 落入0.2-0.4区间，得2分。
*   因子C (0.39): 对于越小越好因子，落入0.2-0.4区间，得4分。
*   因子D (0.11): 对于越小越好因子，落入0.0-0.2区间，得5分。
股票ID1的总分为：4 + 2 + 4 + 5 = **15分**。

### 方法二：直接排序法
此方法更简单直接，无需划分固定区间。

对于有N只股票的池子（例如沪深300，N=300）：
*   对于 **“越大越好”** 的因子：将该因子下所有股票的值从大到小排序，排名第1的得300分，排名第2的得299分，...，排名第300的得1分。
*   对于 **“越小越好”** 的因子：将该因子下所有股票的值从小到大排序，同样赋予300到1的分数。

最后，将每只股票在所有因子上的排序得分相加，得到总分，再进行最终排名。

两种方法的目标一致：**通过一个汇总指标（总分）对所有股票进行排名，以便筛选。**

## 整体任务流程梳理
了解打分原理后，我们来看整个策略的实现流程。这个流程也是我们接下来要编码实现的核心。

第一步，我们需要确定股票池。本例中，我们将在 `contest` 中指定 **沪深300** 指数成分股作为我们的初始股票池。

第二步，设置调仓周期。在实际操作中，每月或每季度调仓较为常见。我们将设置一个定时器函数，例如每月触发一次调仓操作。

第三步，也是最关键的一步，实现调仓函数 `rebalance`。在这个函数中，我们将完成以下工作：

![](img/69234dc51d00f391583a1f9a8c138c65_11.png)

1.  **数据获取与预处理**：从数据源查询当前股票池中所有股票的选定因子数据（例如，3个“越大越好”因子，3个“越小越好”因子）。
2.  **因子打分**：使用上述区间划分法或排序法，为每只股票的每个因子计算得分。
3.  **汇总与排名**：对每只股票，将其在所有因子上的得分相加，得到总分。然后根据总分对所有股票进行降序排序。
4.  **生成交易清单**：选取总分排名前10的股票，作为本次调仓的目标买入组合。
