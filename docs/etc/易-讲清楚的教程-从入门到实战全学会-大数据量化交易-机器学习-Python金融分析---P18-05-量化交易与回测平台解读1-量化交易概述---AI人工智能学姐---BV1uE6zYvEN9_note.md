## 什么是量化交易？

![](img/a6971b1f23ea3136462feac151e22814_5.png)

上一节我们介绍了课程目标，本节中我们来看看量化交易的本质。

传统的人工炒股存在一些局限性。首先，人的主观判断可能导致决策不准确。其次，个人的精力、时间和计算能力有限，无法同时分析大量股票或跨越数年的历史数据。

量化交易的核心目标同样是**赚钱**，但方法不同。它将具体的交易策略交给计算机来制定和执行。其本质是：**基于历史数据，通过计算机算法设计交易策略，以期在未来获得最大收益**。

这个过程通常被称为**回测**。即：在已知的、不会改变的历史数据上，测试我们设计的策略是否有效，并通过一系列指标（后续课程会介绍）来评估策略的优劣。

**公式化描述**：
量化交易 ≈ 基于历史数据的数据挖掘任务，目标是找到最优策略 `S*`，使得在历史数据集 `D_historical` 上的评估指标 `M`（如总收益、夏普比率）最大化。
`S* = argmax_S M(S, D_historical)`

简而言之，量化交易是利用大数据和智能算法（如机器学习），让计算机帮助我们从数据中挖掘赚钱规律，替代或辅助人工进行投资决策。

## 量化交易需要哪些核心技能？

理解了量化交易的目标后，我们来看看实现它需要掌握哪些知识和工具。

量化交易是一个交叉学科，但重点并非深奥的金融理论。金融知识只需达到了解程度即可，核心在于处理和分析数据的能力。

![](img/a6971b1f23ea3136462feac151e22814_7.png)

以下是量化交易涉及的核心技能领域：

*   **机器学习算法**：用于预测未来股价走势、识别模式或优化策略。
*   **统计方法**：用于计算和分析各种金融指标（如均值、方差、相关性），从数据中提取有价值的信息。
*   **编程（如Python）**：这是将想法付诸实践的必备工具。用于数据获取、清洗、回测系统搭建和策略实现。

**核心思想**：在量化交易中，没有绝对的“必备工具”。任何能帮助你更有效地处理数据、验证策略并最大化收益的方法或工具，都是好工具。你可以将量化交易理解为**数据挖掘**在金融投资领域的一个热门应用。
