## 策略核心概念

该策略基于两个关键的技术指标：
*   **短期移动平均线 (MA_short)**：通常取**5天**（一周交易日）或**10天**的收盘价平均值。
*   **长期移动平均线 (MA_long)**：通常取**20天**或**30天**（约一个月交易日）的收盘价平均值。

![](img/b0a86135f73e2a69c418bd852f8a8d0f_8.png)

策略的交易信号基于两条均线的交叉：
*   **黄金交叉**：当短期均线**向上突破**长期均线时，被视为**买入信号**。
*   **死亡交叉**：当短期均线**向下跌破**长期均线时，被视为**卖出信号**。

![](img/b0a86135f73e2a69c418bd852f8a8d0f_10.png)

![](img/b0a86135f73e2a69c418bd852f8a8d0f_11.png)

## 计算移动平均线

![](img/b0a86135f73e2a69c418bd852f8a8d0f_13.png)

首先，我们需要从数据中计算出短期和长期的移动平均值。我们将以苹果公司（AAPL）的股票数据为例。

![](img/b0a86135f73e2a69c418bd852f8a8d0f_15.png)

![](img/b0a86135f73e2a69c418bd852f8a8d0f_16.png)

![](img/b0a86135f73e2a69c418bd852f8a8d0f_17.png)

以下是计算步骤：

![](img/b0a86135f73e2a69c418bd852f8a8d0f_18.png)

![](img/b0a86135f73e2a69c418bd852f8a8d0f_19.png)

1.  **计算短期均线 (M1)**：我们使用一个窗口为10天的滚动平均值。
    ```python
    data[‘M1’] = data[‘AAPL’].rolling(window=10).mean()
    ```

2.  **计算长期均线 (M2)**：我们使用一个窗口为30天的滚动平均值。
    ```python
    data[‘M2’] = data[‘AAPL’].rolling(window=30).mean()
    ```

3.  **查看计算结果**：我们可以查看包含股价、短期均线和长期均线的数据框。
    ```python
    print(data[[‘AAPL‘， ’M1‘， ’M2‘]].head())
    ```

## 生成明确的交易信号

我们可以使用`np.where`函数来创建一个信号列，明确标注短期均线是高于还是低于长期均线。

![](img/b0a86135f73e2a69c418bd852f8a8d0f_41.png)

![](img/b0a86135f73e2a69c418bd852f8a8d0f_42.png)

![](img/b0a86135f73e2a69c418bd852f8a8d0f_44.png)

以下是创建信号指标的步骤：

1.  **计算位置指标**：当短期均线大于长期均线时，标记为1；小于时，标记为-1。
    ```python
    data[‘position’] = np.where(data[‘M1’] > data[‘M2’]， 1， -1)
    ```

2.  **可视化信号**：为了清晰展示，我们将股价和均线绘制在主坐标轴（左轴），将“position”信号绘制在次坐标轴（右轴）。
    ```python
    ax = data[[‘AAPL‘， ’M1‘， ’M2‘]].plot(figsize=(12，6))
    data[‘position’].plot(ax=ax， secondary_y=True， style=‘g’)
    ```
    *   右轴上的**绿色线**即为“position”指标。
    *   当绿线值为**1**时，表示短期均线在长期均线之上（潜在看涨）。
    *   当绿线值为**-1**时，表示短期均线在长期均线之下（潜在看跌）。
    *   **由1变为-1的转折点**对应**死亡交叉**。
    *   **由-1变为1的转折点**对应**黄金交叉**。
