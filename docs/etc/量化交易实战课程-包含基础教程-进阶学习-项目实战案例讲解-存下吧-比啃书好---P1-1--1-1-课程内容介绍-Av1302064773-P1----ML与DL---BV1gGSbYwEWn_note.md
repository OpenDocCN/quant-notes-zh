# 量化交易实战：1.1：课程内容介绍 📚

在本节课中，我们将要学习本系列课程的整体内容安排、核心模块以及学习风格。课程将从零基础开始，通过纯实战的方式，带领大家掌握使用Python进行金融数据分析与量化交易策略开发的完整流程。

## 课程核心模块概述

以下是三大模块的详细说明：

![](img/4edd3d2430a17b3ecdbea01afa34f255_3.png)

### 第二模块：Python金融数据分析实战

本模块将进入金融领域，学习如何处理和分析实际的股票数据。

上一节我们掌握了基础工具，本节中我们将应用这些工具处理真实的金融数据。

以下是第二模块的核心学习内容：

![](img/4edd3d2430a17b3ecdbea01afa34f255_39.png)

![](img/4edd3d2430a17b3ecdbea01afa34f255_41.png)

1.  **金融数据处理**：学习如何获取、清洗和分析股票数据。
2.  **初步回测**：学习在量化平台中完成第一个回测项目，评估策略的历史表现。关键评估指标包括**回测收益**和**超额收益**。
    *   **回测收益公式**：`策略最终价值 / 策略初始价值 - 1`
    *   **超额收益公式**：`策略收益 - 基准收益（如大盘指数收益）`
