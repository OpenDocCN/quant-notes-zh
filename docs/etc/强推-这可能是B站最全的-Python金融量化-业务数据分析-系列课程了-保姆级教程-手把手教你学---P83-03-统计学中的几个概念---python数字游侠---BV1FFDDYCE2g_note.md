# Python金融量化+业务数据分析：P83：03 统计学中的几个概念 📊

在本节课中，我们将学习统计学中的几个核心基础概念。理解这些概念是后续进行数据分析、概率计算和统计推断的基石。我们将从变量的分类开始，逐步介绍概率、随机变量、总体与样本等关键知识。

## 变量

![](img/321af4cd5ed00a30fa8a203a46620b0e_1.png)

变量是统计学中描述事物特征或属性的量。根据其性质和取值方式，变量主要分为两大类：分类变量和数值型变量。

![](img/321af4cd5ed00a30fa8a203a46620b0e_3.png)

![](img/321af4cd5ed00a30fa8a203a46620b0e_5.png)

上一节我们介绍了变量的基本定义，本节中我们来看看这两种变量的具体分类。

![](img/321af4cd5ed00a30fa8a203a46620b0e_7.png)

### 分类变量

![](img/321af4cd5ed00a30fa8a203a46620b0e_9.png)

分类变量用于描述事物的类别或属性，其取值是离散的标签。它又可以细分为两种：

以下是分类变量的两种类型：
*   **无序分类变量**：类别之间没有顺序或等级之分。例如：性别（男/女）、血型（A/B/O/AB）、民族。
*   **有序分类变量**：类别之间存在顺序或等级关系，但无法精确衡量等级间的差距。例如：满意度（满意/一般/不满意）、教育程度（小学/初中/高中/大学）。

### 数值型变量

数值型变量用于描述事物的数量特征，其取值是数字。它也可以分为两种：

以下是数值型变量的两种类型：
*   **连续型变量**：在某个区间内可以取无限多个值，取值是连续的。例如：身高、体重。在足够精确的测量下，每个人的身高都可能是一个独特的数值。
*   **离散型变量**：取值是有限个或可数的。例如：一家公司员工的人数、抛硬币正面朝上的次数。

**核心概念总结**：
*   无序分类变量：无大小、无顺序。
*   有序分类变量：无大小、有顺序。
*   连续型变量：有大小，在区间内取值无限。
*   离散型变量：有大小，取值有限可数。

理解变量类型至关重要，因为它决定了后续应选用何种统计方法和数据分析模型。

## 概率

在了解了如何描述数据（变量）后，我们来看看如何描述不确定性。概率是度量随机事件发生可能性大小的工具，其值介于0和1之间。

随机现象某种可能观察的结果称为**随机事件**。概率则是描述这些事件发生可能性的数值。例如，抛一枚均匀硬币，正面朝上这个随机事件的概率是0.5。

### 小概率事件

在统计学中，有一个特别重要的概念——小概率事件。

![](img/321af4cd5ed00a30fa8a203a46620b0e_11.png)

以下是小概率事件的定义与理解：
*   **定义**：通常认为，发生概率小于或等于0.05（即5%）的事件称为小概率事件。
*   **原理**：这意味着该事件在大多数情况下不会发生。统计学中有一个基本共识：**小概率事件在一次试验或抽样中几乎不可能发生**。如果这样的事件竟然发生了，我们就有理由怀疑原有的假设或条件可能不成立。

例如，购买彩票中头奖、出门被陨石砸中都属于典型的小概率事件。

![](img/321af4cd5ed00a30fa8a203a46620b0e_13.png)

**核心概念公式**：
`P(事件) ≤ 0.05` → 该事件可被视为小概率事件。

## 随机变量

概率描述了事件发生的可能性，而随机变量则将这些事件的结果“数量化”，以便于进行数学处理和统计分析。

**随机变量**是将随机事件的结果用数值来表示的变量。它本质上是随机事件数量化的结果。

例如，抛硬币的结果不是“正面”或“反面”，而是将其量化为：`出现正面记为1，出现反面记为0`。这里的1和0就是一个随机变量的取值。

根据取值特点，随机变量也分为两类：

以下是随机变量的两种类型：
*   **离散型随机变量**：取值是可数的、有限的。例如：抛10次硬币正面朝上的次数（0,1,2,...,10）。
*   **连续型随机变量**：取值充满某个区间，不可一一列举。例如：北京成年男性的身高，在精确测量下，理论上在区间内有无限多种可能取值。

随机变量是连接概率论与统计学的桥梁，也是理解概率分布的基础。

## 总体与样本

在实际研究中，我们很少能收集到研究对象的全部数据。这时，就需要引入总体和样本的概念。

*   **总体**：根据研究目的确定的、所有个体观察值的集合。例如，检验一批灯泡的寿命，这批所有灯泡的寿命数据就是总体。
*   **样本**：从总体中随机抽取的一部分个体的观察值集合。例如，从那批灯泡中随机抽取50个进行测试，这50个灯泡的数据就构成一个样本。

统计学的主要工作之一，就是通过分析样本数据来推断总体的特征。

### 随机抽样

从总体中获取样本的过程称为抽样。其中，**随机抽样**是最基本、最重要的抽样方法。

随机抽样是指：总体中的每一个个体被抽中的概率是相同的。这保证了样本对总体具有代表性。
*   **随机 ≠ 随便**：街头“随机”采访路人，往往不是严格的随机抽样，因为并非每个路人都有均等的机会被选中。
*   **实现方法**：要实现随机抽样，通常需要给总体中每个个体编号，然后使用随机数生成器或抽签等方式进行抽取。

只有通过随机抽样得到的样本，其统计结论推广到总体才具有可信度。

## 总体参数与统计量

由于总体和样本的区别，描述它们特征的指标也有不同的名称。

*   **总体参数**：描述总体特征的指标，通常是固定但未知的值。
    *   例如：总体均值（μ）、总体标准差（σ）、总体比例（π）。
*   **统计量**：描述样本特征的指标，根据抽取样本的不同而变化，是已知的。
    *   例如：样本均值（`x̄`）、样本标准差（s）、样本比例（p）。

**核心关系**：我们通过计算样本统计量（如 `x̄`）来估计未知的总体参数（如 μ）。这个过程称为**统计推断**。

## 抽样误差

当我们用样本统计量去估计总体参数时，不可避免地会存在误差，这种误差就叫做抽样误差。

**抽样误差**是指由随机抽样造成的样本统计量与总体参数之间的差异。
*   **必然存在**：只要不是普查（调查全部总体），抽样误差就必然存在。
*   **可以控制**：虽然某一次抽样的误差大小是随机的，但大量重复抽样下的误差分布是有规律的（服从抽样分布）。我们可以根据这个规律来量化误差的范围，从而对总体参数进行区间估计。

![](img/321af4cd5ed00a30fa8a203a46620b0e_15.png)

例如，样本平均身高是175cm，总体平均身高是176cm，这1cm的差异就是抽样误差的一种体现。

![](img/321af4cd5ed00a30fa8a203a46620b0e_17.png)
