在本节课中，我们将学习如何分析时间序列数据的变化情况。具体来说，我们将计算股票数据中每日的差异值和变化率，并学习如何对这些变化进行统计和可视化。这些是金融数据分析中非常基础和重要的操作。

上一节我们介绍了如何加载和查看股票数据，本节中我们来看看如何分析这些数据随时间的变化。

## 计算每日差异值

![](img/8b55a4f027e2c5ae581fec06b31e3c62_6.png)

首先，我们需要计算每一天与前一天的指标差异值。例如，对于股价，我们关心的是它每天上涨或下跌了多少，而不是具体的价格数值。

![](img/8b55a4f027e2c5ae581fec06b31e3c62_8.png)

在Pandas中，计算这种差异非常简单。我们可以使用`.diff()`方法，它会自动用当前行的值减去前一行的值。

以下是计算差异值的代码：

![](img/8b55a4f027e2c5ae581fec06b31e3c62_10.png)

```python
# 假设df是我们的股票数据DataFrame
diff_data = df.diff()
print(diff_data.head())
```

![](img/8b55a4f027e2c5ae581fec06b31e3c62_12.png)

执行这段代码后，你会看到一个新的DataFrame。例如，1月5日的差异值就是用1月5日的数据减去1月4日的数据得到的。对于数据的第一天（如1月4日），由于没有前一天的数据可减，结果会是`NaN`（空值）。

`.diff()`方法默认计算的是与前一行的差异。我们也可以计算间隔多天的差异。

为了养成查API的好习惯，当你遇到不熟悉的函数或参数时，最直接的方法是复制函数名，在搜索引擎或官方文档中查找。自己动手实践和查阅文档是最高效的学习方式。

## 计算变化率（增长率）

![](img/8b55a4f027e2c5ae581fec06b31e3c62_18.png)

仅仅知道差异值有时还不够。例如，股价从1元涨到2元是涨了1元，从10000元涨到10001元也是涨了1元，但前者翻了一倍，后者几乎没变。因此，我们常常更关心变化率或增长率。

变化率的计算公式是：`(后一天的值 - 前一天的值) / 前一天的值`。

![](img/8b55a4f027e2c5ae581fec06b31e3c62_20.png)

Pandas同样提供了现成的方法`.pct_change()`来计算这个百分比变化。

以下是计算增长率的代码：

![](img/8b55a4f027e2c5ae581fec06b31e3c62_22.png)

```python
# 计算每日增长率
pct_change_data = df.pct_change()
print(pct_change_data.head())
```

![](img/8b55a4f027e2c5ae581fec06b31e3c62_24.png)

计算后，数据会从具体的差异值变为百分比增长率。股票日增长率通常很小，例如0.1%或0.2%，这属于正常范围。

![](img/8b55a4f027e2c5ae581fec06b31e3c62_26.png)

![](img/8b55a4f027e2c5ae581fec06b31e3c62_28.png)

## 统计与可视化

![](img/8b55a4f027e2c5ae581fec06b31e3c62_30.png)

计算出每日的变化率后，我们可能希望了解整体的平均变化情况。

![](img/8b55a4f027e2c5ae581fec06b31e3c62_32.png)

我们可以对增长率数据计算平均值，得到一个反映各指标平均日增长率的序列。

![](img/8b55a4f027e2c5ae581fec06b31e3c62_34.png)

以下是计算平均增长率并可视化的代码：

```python
# 计算各列的平均增长率
mean_pct_change = pct_change_data.mean()
print(mean_pct_change)

![](img/8b55a4f027e2c5ae581fec06b31e3c62_36.png)

# 使用条形图可视化平均增长率
mean_pct_change.plot(kind='bar', figsize=(16, 8))
```

![](img/8b55a4f027e2c5ae581fec06b31e3c62_38.png)

由于平均增长率是每个指标的一个单独数值，使用条形图比折线图更合适。通过调整`plot`函数的参数，我们可以轻松改变图表类型和大小。从生成的图表中，我们可以一目了然地看到各个指标的平均增长情况。

![](img/8b55a4f027e2c5ae581fec06b31e3c62_40.png)

![](img/8b55a4f027e2c5ae581fec06b31e3c62_42.png)
