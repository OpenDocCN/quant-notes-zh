在本节课中，我们将要学习Pandas中DataFrame的核心访问方法。DataFrame是二维表格，其访问方式比一维的Series更为复杂，需要从行和列两个维度去理解。我们将系统地介绍显示访问、隐式访问、切片、布尔索引以及元素访问等多种操作，并通过实例帮助初学者掌握这些技巧。

## 理解DataFrame的结构

上一节我们介绍了Series的访问，本节中我们来看看DataFrame。首先，我们需要理解DataFrame的结构。

DataFrame可以看作是一个由Series组成的容器。更具体地说，它像一个字典，其中：
*   **键** 是列标签。
*   **值** 是每一列对应的Series数据。

基于这种理解，访问DataFrame的列就变得非常简单，就像通过字典的键来获取值一样。

## 访问列数据

以下是访问DataFrame列数据的几种主要方法。

### 字典式访问与属性访问

既然DataFrame像字典，我们就可以通过列标签（键）来访问对应的列（值）。

```python
# 假设 df 是一个DataFrame对象
df[‘name’] # 访问 ‘name’ 列
df[‘java’] # 访问 ‘java’ 列
```

此外，DataFrame也支持将列作为属性来访问。

```python
df.name # 通过属性访问 ‘name’ 列
```

### 访问多列数据

![](img/edb420b5061603fd3108506bb56bb7b1_3.png)

如果需要同时访问多列，可以将列标签放入一个列表中作为索引。

```python
df[[‘java‘, ‘python‘]] # 访问 ‘java‘ 和 ‘python‘ 两列，返回一个新的DataFrame
```

以上方法都属于**显示访问**，即直接使用列标签名。

## 访问行数据

与访问列不同，访问行数据有更专门的方法。

### 使用 `.loc` 访问行

`.loc` 是基于标签的访问器，是访问行的标准形式。它通过行索引来定位数据。

```python
df.loc[2] # 访问行索引为 2 的那一行
df.loc[[1, 2]] # 访问行索引为 1 和 2 的两行
```

### 使用 `.iloc` 访问行

`.iloc` 是基于整数位置的访问器，属于**隐式访问**。它把DataFrame看作一个二维数组，按照位置索引进行访问。

```python
df.iloc[0] # 访问第一行（位置0）
df.iloc[[0, 1]] # 访问第一行和第二行（位置0和1）
```

**核心原则**：访问行使用 `.loc` 或 `.iloc`；访问列使用中括号 `[]`。

## 访问单个元素

访问DataFrame中的特定元素（单元格）时，必须遵循**先行后列**的原则，这与二维数组的访问逻辑一致。

### 使用 `.loc`

```python
df.loc[2, ‘name‘] # 访问行索引为2，列标签为’name‘的单元格
```

### 使用 `.iloc`

```python
df.iloc[2, 1] # 访问第3行（位置2），第2列（位置1）的单元格
```

**注意**：顺序不能颠倒，必须先指定行，再指定列。

## 数据切片

切片操作允许我们获取数据的子集，也分为显示切片（用标签）和隐式切片（用整数位置）。

### 行切片

以下是进行行切片的方法。

```python
# 显示切片（闭区间）
df.loc[0:2] # 切片包含行索引 0, 1, 2

# 隐式切片（左闭右开区间）
df.iloc[0:3] # 切片包含位置 0, 1, 2（不包含3）
```

### 列切片

进行列切片需要结合行、列两个维度，通常先行后列。

```python
# 显示切片
df.loc[:, ‘name‘:‘python‘] # 所有行，列从’name‘到’python‘（闭区间）

# 隐式切片
df.iloc[:, 1:3] # 所有行，列从位置1到位置2（左闭右开）
```

**重要区别**：显示索引切片是**闭区间**，隐式索引切片是**左闭右开**区间。

## 使用布尔索引进行条件筛选

布尔索引是一种强大的数据筛选方式，可以根据条件选择行或列。

以下是使用布尔索引筛选数据的步骤。

1.  **创建布尔序列**：通过对列进行条件判断，得到一个布尔值Series。
    ```python
    bool_series = df[‘python‘] > 10 # 筛选出’python‘列大于10的行
    ```
2.  **应用布尔索引**：将布尔序列传入 `.loc` 来筛选行。
    ```python
    df.loc[bool_series] # 获取’python‘成绩大于10的所有行
    ```

**注意**：使用 `.iloc` 进行布尔索引时，需要将布尔Series转换为列表或NumPy数组，以避免索引对齐问题。
```python
df.iloc[bool_series.values] # 使用 .values 获取纯布尔数组
```

## 综合练习与注意事项

让我们通过几个练习来巩固所学知识。

*   **查找Java成绩大于0.5的所有信息**：
    ```python
    df.loc[df[‘java‘] > 0.5]
    ```
*   **查找Python成绩大于平均值的所有信息**：
    ```python
    mean_val = df[‘python‘].mean()
    df.loc[df[‘python‘] > mean_val]
    ```
*   **查找’杰克’的Java成绩并加0.5分**：
    ```python
    df.loc[‘杰克‘, ‘java‘] += 0.5
    ```
*   **获取只包含’name‘和’python‘两列的子表**：
    ```python
    df[[‘name‘, ‘python‘]]
    # 或
    df.iloc[:, [1, 2]] # 假设’name‘和’python‘在第2、3列
    ```

**重要注意事项**：
1.  优先使用 `.loc` 和 `.iloc` 进行**直接访问**（如 `df.loc[行, 列]`），避免通过连续链式操作进行间接访问，后者可能在赋值时产生警告。
2.  当读取DataFrame的某一行时，返回的Series数据类型可能会被统一为`object`类型，因为一行中可能包含多种原始数据类型。这不会影响原始DataFrame，只影响返回的Series对象。

## DataFrame的运算

DataFrame的运算涉及与标量、Series以及其他DataFrame的交互。

### 与标量或数组运算

DataFrame与单个数字或NumPy数组进行运算时，应用**广播机制**，将该值扩展到每个元素进行计算。
```python
df + 5 # 所有元素加5
df + np.array([[1, 2, 3]]) # 与数组广播运算
```

### 与Series运算

DataFrame与Series运算时，默认依据**列索引**进行对齐。如果索引无法匹配，则结果中相应位置会填充空值（NaN）。
```python
s = pd.Series([1, 2, 3], index=[‘java‘, ‘name‘, ‘python‘])
df + s # 按列标签对齐相加
```
可以通过指定轴（`axis`）参数来改变对齐方向。
```python
df.add(s, axis=0) # 尝试按行索引对齐
```

### 与另一个DataFrame运算

两个DataFrame之间运算时，要求**行索引和列索引都对齐**。对不齐的索引，结果位置同样填充NaN。
```python
df1 + df2
```
可以使用运算方法（如 `.add`）的 `fill_value` 参数来填充NaN。
```python
df1.add(df2, fill_value=0) # 将对不齐的位置视为0进行运算
```
