## 平均结果的局限性

上一节我们介绍了如何计算平均结果。平均结果反映了数据的整体情况。那么，我们能否对连续时间序列进行加法操作，例如计算其累加增长率呢？

![](img/15c274ad129fc2d1da2f0d273c328c60_4.png)

让我们通过一个例子来探讨。假设有一只股票：
*   第一天价格：`100`
*   第二天价格：`50`
*   第三天价格：`75`

以下是计算过程：
1.  第二天的增长率：`(50 - 100) / 100 = -0.5`
2.  第三天的增长率：`(75 - 50) / 50 = 0.5`

如果我们将这两个增长率简单相加：`-0.5 + 0.5 = 0`。这个结果为零，似乎意味着股票从第一天到第三天既没涨也没跌。

然而，观察原始数据：第一天`100`，第三天`75`，股价实际上是下跌的。这说明，对于增长率，**不能直接进行加法操作**，因为简单累加无法正确反映从起点到终点的整体变化。

## 引入对数计算连续变化

![](img/15c274ad129fc2d1da2f0d273c328c60_10.png)

对于连续时间序列的累加计算，我们需要借助数学中的对数公式。

我们重新计算之前的例子，但使用对数方法：
*   计算第二天相对于第一天的对数变化：`log(50 / 100)`
*   计算第三天相对于第二天的对数变化：`log(75 / 50)`

当我们以自然常数`e`为底时，对数的加法等于其内部数值的乘法：
`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/15c274ad129fc2d1da2f0d273c328c60_12.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_14.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_16.png)

最终结果 `log(75/100)` 直接反映了**第三天与第一天价格之间的整体差异比例**。

![](img/15c274ad129fc2d1da2f0d273c328c60_18.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_20.png)

这个方法的形象解释是：假设股票初始价格为1单位（如1元），经过一系列涨跌后，将每个时期的对数收益率累加起来，其结果就等价于**最终价格相对于初始价格的对数变化**。这能准确告诉我们，初始的1元钱在经历整个时间序列后，相当于最终值多少钱。

## 使用Python计算连续收益率

![](img/15c274ad129fc2d1da2f0d273c328c60_28.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_30.png)

以下是使用`pandas`计算连续收益率（对数收益率）的步骤。

首先，我们需要获取“前一天”的数据。可以使用`.shift()`函数将整个数据序列向上移动一格。

![](img/15c274ad129fc2d1da2f0d273c328c60_32.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_34.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_36.png)

```python
# 假设data是一个包含股价的pandas Series或DataFrame
# 计算当天数据与前一天数据的比值
price_ratio = data / data.shift(1)

# 计算对数收益率（连续收益率）
returns = np.log(price_ratio)

![](img/15c274ad129fc2d1da2f0d273c328c60_38.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_40.png)

# 查看结果
print(returns.head())
```
执行以上代码，我们便得到了每一天相对于前一天的对数收益率序列。这个序列描述了价格的连续变化情况。

![](img/15c274ad129fc2d1da2f0d273c328c60_42.png)

## 计算累积收益并还原为实际价值

![](img/15c274ad129fc2d1da2f0d273c328c60_44.png)

我们的目标是计算：初始的1单位投资，在后续每一个时间点相当于多少价值。这需要我们对对数收益率进行累积求和，然后将结果还原为实际数值。

![](img/15c274ad129fc2d1da2f0d273c328c60_46.png)

以下是关键步骤：
1.  **累积求和**：使用`np.cumsum()`函数对对数收益率序列进行累加。这个操作相当于连续做对数加法。
2.  **还原数值**：将对数累积和通过指数函数`np.exp()`还原为实际的累积收益倍数。

![](img/15c274ad129fc2d1da2f0d273c328c60_48.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_50.png)

```python
# 对对数收益率进行累积求和
cumulative_returns_log = returns.cumsum()

![](img/15c274ad129fc2d1da2f0d273c328c60_52.png)

# 通过指数函数还原为实际的累积收益
cumulative_returns = np.exp(cumulative_returns_log)

![](img/15c274ad129fc2d1da2f0d273c328c60_54.png)

# 绘制结果图，显示初始1单位投资随时间的变化
cumulative_returns.plot(figsize=(16, 8))
plt.title(‘初始1单位投资的累积价值变化’)
plt.ylabel(‘累积价值倍数’)
plt.xlabel(‘时间’)
plt.show()
```
运行这段代码后，我们将得到一张图表。图表中，每条曲线都从1开始，展示了不同股票（或资产）的初始1元投资，在2010年至2018年间增长为多少元。例如，曲线最高的可能增长到了10元以上，这直观地反映了该资产在此期间强大的增值能力。

## 常用金融序列操作总结

![](img/15c274ad129fc2d1da2f0d273c328c60_60.png)

本节课中我们一起学习了金融时间序列分析中几种核心的计算操作：

![](img/15c274ad129fc2d1da2f0d273c328c60_62.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_64.png)

以下是关键操作及其实现方法：
*   **差值计算**：使用`.diff()`方法可以轻松计算序列中相邻元素的差值。
*   **简单增长率**：公式为 `(后值 - 前值) / 前值`，反映了相邻两期之间的变化比例。
*   **连续收益率（对数收益率）**：这是本节课的重点。通过公式 `log(当期值 / 前一期值)` 计算，其**累加和具有实际意义**，能准确反映从期初到期末的整体增长倍数。
*   **累积收益计算**：通过对数收益率序列进行`np.cumsum()`累加，再使用`np.exp()`还原，即可得到“期初1单位投资在每期的价值”这一重要指标。

![](img/15c274ad129fc2d1da2f0d273c328c60_66.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_68.png)

![](img/15c274ad129fc2d1da2f0d273c328c60_70.png)

通过掌握这些操作，我们能够更科学、更准确地分析资产价格的连续变化趋势，为量化分析和投资决策打下坚实基础。
