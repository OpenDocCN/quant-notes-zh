# 量化交易入门到精通：38：Python循环语句之break 🛑

![](img/b7b9670bb347af2e3dc6795d1db2e501_1.png)

在本节课中，我们将要学习Python循环控制语句中的一个重要成员：`break`。它就像循环中的“紧急刹车”，能够在特定条件下立即终止整个循环的执行。

## 概述

`break`语句用于在循环（如`for`循环或`while`循环）执行过程中，当满足某个条件时，立即跳出并终止当前循环。理解`break`的用法，对于编写灵活、高效的程序逻辑至关重要。

![](img/b7b9670bb347af2e3dc6795d1db2e501_3.png)

## break语句详解

![](img/b7b9670bb347af2e3dc6795d1db2e501_5.png)

上一节我们介绍了循环的基本结构，本节中我们来看看如何使用`break`来主动控制循环的退出。

`break`的核心作用是：**立即退出并终止当前所在的循环**。其行为可以用一个简单的公式来描述：

![](img/b7b9670bb347af2e3dc6795d1db2e501_7.png)

```python
if 特定条件成立:
    break
```

![](img/b7b9670bb347af2e3dc6795d1db2e501_9.png)

当程序执行到`break`语句时，无论循环原本计划执行多少次，都会立刻停止，并继续执行循环之后的代码。

## 代码演示与解析

![](img/b7b9670bb347af2e3dc6795d1db2e501_11.png)

为了更好地理解，我们结合一个具体的`for`循环程序来演示。

![](img/b7b9670bb347af2e3dc6795d1db2e501_13.png)

以下是一个计划循环4次（0, 1, 2, 3）的程序：

```python
for i in range(4):
    print(f"当前循环次数为：{i}")
    print("*** 这是循环体内的一条语句 ***")
```

运行这段代码，它会完整地打印4次信息。

现在，我们希望在循环到第2次（即`i`等于1）时，使用`break`语句让循环提前终止。修改后的代码如下：

![](img/b7b9670bb347af2e3dc6795d1db2e501_15.png)

```python
for i in range(4):
    print(f"当前循环次数为：{i}")
    if i == 1:          # 当循环次数为1时
        break           # 立即终止循环
    print("*** 这是循环体内的一条语句 ***")
```

以下是代码执行过程的详细解析：

1.  第一次循环 (`i = 0`)：
    *   打印“当前循环次数为：0”。
    *   检查`if`条件 (`i == 1`)，结果为`False`，不执行`break`。
    *   继续执行，打印“*** 这是循环体内的一条语句 ***”。

![](img/b7b9670bb347af2e3dc6795d1db2e501_17.png)

2.  第二次循环 (`i = 1`)：
    *   打印“当前循环次数为：1”。
    *   检查`if`条件 (`i == 1`)，结果为`True`，**立即执行`break`语句**。
    *   循环被强制终止，`break`之后的`print`语句**不会被执行**。

因此，程序的最终输出只有：
```
当前循环次数为：0
*** 这是循环体内的一条语句 ***
当前循环次数为：1
```

![](img/b7b9670bb347af2e3dc6795d1db2e501_19.png)

可以看到，当`i`等于1时触发`break`，循环被立即打断，后续的迭代（`i`为2和3）以及本次循环中`break`之后的代码都未执行。

## 关键要点总结

本节课中我们一起学习了`break`语句的核心用法：

*   **作用**：`break`用于在循环中立即退出并终止整个循环。
*   **使用场景**：通常与`if`条件判断结合使用，在满足特定条件时提前结束循环。
*   **执行效果**：一旦执行`break`，循环会立刻停止，循环体内`break`之后的所有语句以及后续的迭代都不会再执行。

理解`break`能帮助你处理诸如“搜索到目标后停止”、“遇到错误条件时退出”等需要提前结束循环的场景。
