## 重复数据的清洗 🔄

![](img/d60a469670cd3c6568af234c1887eb94_11.png)

![](img/d60a469670cd3c6568af234c1887eb94_13.png)

![](img/d60a469670cd3c6568af234c1887eb94_15.png)

重复数据指的是数据集中完全相同的行。在金融数据分析中，重复数据可能由数据采集错误或合并数据源时产生，需要被清理以保证分析的准确性。

![](img/d60a469670cd3c6568af234c1887eb94_17.png)

以下是生成并处理重复数据的步骤：

![](img/d60a469670cd3c6568af234c1887eb94_19.png)

1.  首先，我们创建一个包含重复行的示例数据集。
    ```python
    import pandas as pd
    import numpy as np

    # 生成一个8行4列的随机数据框
    df = pd.DataFrame(np.random.randint(0, 100, size=(8, 4)))
    # 人为制造重复行：将第3、4、5行的数据设置为与第2行相同
    df.iloc[3] = df.iloc[2]
    df.iloc[4] = df.iloc[2]
    df.iloc[5] = df.iloc[2]
    ```

![](img/d60a469670cd3c6568af234c1887eb94_21.png)

2.  使用 `drop_duplicates()` 方法清洗重复数据。该方法的关键参数是 `keep`，用于指定保留哪一行。
    *   `keep='first'`：保留第一次出现的重复行（默认值）。
    *   `keep='last'`：保留最后一次出现的重复行。
    *   `keep=False`：删除所有重复的行。
    ```python
    # 删除重复行，默认保留首次出现的行
    df_cleaned = df.drop_duplicates()
    ```

![](img/d60a469670cd3c6568af234c1887eb94_23.png)

![](img/d60a469670cd3c6568af234c1887eb94_25.png)

## 异常数据的清洗 🚨

异常数据是指那些明显偏离正常范围或分布的值。例如，在分析股票日收益率时，一个远高于历史平均值的极端值可能就是异常值。处理异常值通常需要先定义判定规则。

![](img/d60a469670cd3c6568af234c1887eb94_27.png)

本节我们通过一个案例来学习如何清洗异常值：假设我们判定C列中，**大于该列两倍标准差**的值为异常值，并需要将其所在行删除。

![](img/d60a469670cd3c6568af234c1887eb94_29.png)

以下是清洗异常值的完整流程：

![](img/d60a469670cd3c6568af234c1887eb94_31.png)

1.  生成一个1000行3列的模拟数据集。
    ```python
    # 生成0到1之间的随机数，创建1000行3列的数据框
    df = pd.DataFrame(np.random.random(size=(1000, 3)), columns=['A', 'B', 'C'])
    ```

![](img/d60a469670cd3c6568af234c1887eb94_33.png)

2.  定义异常值的判定条件，并计算阈值。
    ```python
    # 计算C列的两倍标准差
    twice_std = df['C'].std() * 2
    ```

![](img/d60a469670cd3c6568af234c1887eb94_35.png)

3.  根据条件创建布尔掩码，识别异常值。
    ```python
    # 判断C列中哪些值大于两倍标准差，True表示异常值
    abnormal_mask = df['C'] > twice_std
    ```

![](img/d60a469670cd3c6568af234c1887eb94_37.png)

4.  利用布尔索引，**取反后**筛选出正常数据（即删除异常行）。
    ```python
    # 使用 ~ 对布尔掩码取反，然后筛选数据，保留正常值所在的行
    df_cleaned = df[~abnormal_mask]
    ```

本节课中我们一起学习了数据清洗中处理重复值和异常值的方法。
*   对于**重复值**，我们使用 `df.drop_duplicates()` 方法可以方便地删除重复行，并通过 `keep` 参数控制保留策略。
*   对于**异常值**，清洗的关键在于**定义明确的判定条件**（如基于标准差、分位数或业务规则）。之后通过布尔索引筛选数据，即可移除异常值所在的行。
