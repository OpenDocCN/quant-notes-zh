## 变量的概念

所谓变量，顾名思义就是可变的量，即可变的值。在数学中，我们学过类似 `X = Y + 6` 的方程，其中的 `X` 和 `Y` 就是变量。在计算机中，变量的定义有所不同。

**变量**是用于在内存中存放程序运行数据的一个容器。程序运行时，CPU需要数据进行计算，这些数据源就存储在内存中。变量就是我们在内存中存储数据的形式。

## 内存与变量的类比

内存就像一个巨大的仓库，里面可以存放很多东西。要在仓库中存取物品，需要三样东西：
1.  **货架编号**：用于标记和查找物品的位置。
2.  **货物本身**：实实在在要存储的物品。
3.  **实实在在的货架**：物品存放的物理位置。

![](img/464a5efc50170557c8b2e86eb9dc2507_3.png)

在计算机内存中存储数据，同样需要对应的三样东西：
*   **变量名**：对应货架编号，是我们给内存位置起的名字。
*   **变量值**：对应货物本身，是存储在内存中的实际数据。
*   **内存地址**：对应实实在在的货架，是数据在内存芯片中的物理位置。

## 变量的定义与调用

理解了概念后，我们来看如何在Python中操作变量。

### 定义变量
定义变量就是给变量名赋值的过程，使用等号 `=`。
```python
name = “ALEX”
```
*   `name` 是变量名（货架编号）。
*   `“ALEX”` 是变量值（货物本身）。
*   `=` 是赋值操作（把货物放到货架上）。

### 调用变量
调用变量就是使用变量名来获取其存储的值。
```python
name
```
直接输入变量名并回车，即可看到存储的值 `“ALEX”`。

### 查看内存地址
我们可以使用 `id()` 函数查看变量值在内存中的实际地址（货架位置）。
```python
id(name)
```
这会返回一串数字，代表变量 `name` 的值在内存中的地址。

以下是定义和调用多个变量的示例：
```python
# 定义变量
name = “ALEX”
age = “23”  # 注意：文字类型的值需要加引号

# 调用变量
print(name)
print(age)

# 查看它们的内存地址（通常不同）
print(id(name))
print(id(age))
```

**重要规则：先定义，后调用**
你必须先给变量赋值（定义），才能使用它（调用）。尝试调用一个未定义的变量会导致错误。
```python
# 错误示例：未定义就调用
print(hometown)  # 会报错：NameError: name ‘hometown’ is not defined

# 正确做法：先定义
hometown = “Beijing”
print(hometown)  # 正常输出
```

## 变量的定义规则

变量名不能随意起，需要遵守一些基本规范。

1.  **全局变量尽量定义在文件开头**：如果你的变量会在程序后面多处使用，建议在代码文件的开头部分定义，避免因调用时未定义而报错。
2.  **变量名只能是字母、数字或下划线的组合**：
    *   合法示例：`name`, `user_age`, `_id`, `value2`
    *   非法示例：`my-name`, `user age`, `2name`（数字不能开头）
3.  **变量名区分大小写**：`Name` 和 `name` 会被视为两个不同的变量。
4.  **不能使用Python关键字**：Python语言自身已经占用的单词（如 `print`, `if`, `for` 等）不能用作变量名。无需死记，随着学习会自然熟悉。

以下是变量命名规范的示例：
```python
# 合法的变量名
username = “Alex”
user_age = 30
_count = 1
value2 = 100

# 非法的变量名（会报错）
2name = “test”    # 数字不能开头
my-name = “test”  # 含有非法字符“-”
user name = “test” # 含有空格
```

## 总结

本节课中我们一起学习了Python变量的核心知识：
*   **变量的本质**：是内存中存储程序运行数据的容器，包含变量名、变量值和内存地址三个要素。
*   **变量的操作**：使用 `=` 进行赋值（定义），直接使用变量名进行调用，使用 `id()` 函数查看内存地址。
*   **核心规则**：必须遵守“先定义，后调用”的原则。
*   **命名规范**：变量名由字母、数字和下划线组成，且不能以数字开头，不能使用Python关键字。
