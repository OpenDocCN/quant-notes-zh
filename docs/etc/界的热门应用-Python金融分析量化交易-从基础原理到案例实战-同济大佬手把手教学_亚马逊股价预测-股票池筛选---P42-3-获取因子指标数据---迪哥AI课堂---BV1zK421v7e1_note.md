上一节我们介绍了工具包的导入，本节中我们来看看如何具体获取因子数据。

首先，我们需要选择一个日期范围。因为后续查询数据时，需要传入具体的日期参数。

![](img/c433426376cb225d96578adaa16bef62_7.png)

以下是获取交易日期的步骤：
1.  调用 `get_trading_dates` 函数。
2.  传入起始日期参数，例如 `start_date='2019-01-01'`。
3.  传入结束日期参数，例如 `end_date='2020-01-01'`。
4.  函数将返回该时间段内所有的交易日列表。

![](img/c433426376cb225d96578adaa16bef62_9.png)

```python
trading_dates = get_trading_dates(start_date='2019-01-01', end_date='2020-01-01')
```

![](img/c433426376cb225d96578adaa16bef62_11.png)

这样，我们就获得了2019年全年的交易日列表。

---

![](img/c433426376cb225d96578adaa16bef62_13.png)

接下来，我们需要获取因子指标数据。我们以市盈率（PE）因子为例。

![](img/c433426376cb225d96578adaa16bef62_15.png)

首先，我们需要确定要分析的股票范围。这里我们不做限制，直接获取所有A股股票。

以下是构建股票池的步骤：
1.  调用 `all_instruments` 函数。
2.  指定市场类型参数，例如 `market='cn'` 代表A股市场。
3.  函数将返回包含所有A股股票信息的DataFrame。

![](img/c433426376cb225d96578adaa16bef62_17.png)

```python
stock_pool = all_instruments('cn')
```

![](img/c433426376cb225d96578adaa16bef62_19.png)

![](img/c433426376cb225d96578adaa16bef62_21.png)

现在，我们有了一个包含所有A股股票的股票池。

![](img/c433426376cb225d96578adaa16bef62_23.png)

---

![](img/c433426376cb225d96578adaa16bef62_25.png)

有了日期和股票池，接下来就可以执行数据查询了。我们需要查询指定日期、指定股票池内所有股票的PE指标。

![](img/c433426376cb225d96578adaa16bef62_27.png)

![](img/c433426376cb225d96578adaa16bef62_29.png)

以下是构建并执行查询的步骤：
1.  使用 `query` 函数构建查询语句，指定要查询的因子，例如 `valuation.pe_ratio`。
2.  使用 `filter` 方法，筛选出股票代码在 `stock_pool` 列表中的股票。
3.  调用 `get_fundamentals` 函数执行查询，传入查询语句和具体的查询日期（例如交易日列表中的第一天）。
4.  函数将返回一个包含查询结果的多维数据结构。

![](img/c433426376cb225d96578adaa16bef62_31.png)

```python
# 构建查询语句
q = query(valuation.pe_ratio).filter(valuation.code.in_(stock_pool['code']))

![](img/c433426376cb225d96578adaa16bef62_33.png)

# 执行查询（以第一个交易日为例）
factor_data = get_fundamentals(q, trading_dates[0])
```

![](img/c433426376cb225d96578adaa16bef62_35.png)

执行后，`factor_data` 变量中存储了查询结果。这个结果是一个三维结构，但有效数据通常集中在最后一个维度。我们可以使用 `.iloc` 方法查看具体的数据内容。

![](img/c433426376cb225d96578adaa16bef62_37.png)

```python
# 查看前5条数据
print(factor_data.iloc[0, 0, :5])
```

![](img/c433426376cb225d96578adaa16bef62_39.png)

![](img/c433426376cb225d96578adaa16bef62_41.png)

输出将显示股票代码及其对应的PE比率值。需要注意的是，查询结果中不直接包含日期列，因为日期信息在查询时已经作为参数传入。

![](img/c433426376cb225d96578adaa16bef62_43.png)
