# 机器学习：P50-P52：误差项定义

![](img/df03348225c81b6347803e9c7c8f0575_0.png)

在本节课中，我们将学习线性回归模型中的核心组成部分：误差项。我们将探讨如何将回归方程转化为矩阵形式，并理解误差项在模型训练中的意义。

## 从回归方程到矩阵形式

上一节我们介绍了线性回归的基本思想，即用一个平面（或直线）来拟合数据点。本节中我们来看看如何用数学公式精确地描述这个拟合平面。

首先，我们给出线性回归方程的一般形式。对于一个有两个特征（例如工资`X1`和年龄`X2`）的问题，我们的假设平面可以表示为：

**公式：** `Hθ(X) = θ0 + θ1*X1 + θ2*X2`

在这个公式中：
*   `θ1` 和 `θ2` 被称为**权重项**。它们决定了每个特征（`X1`, `X2`）对最终结果的影响程度。
*   `θ0` 被称为**偏置项**。它的作用是**对模型结果进行微调**，使整个拟合平面能够向上或向下平移，以更好地匹配数据。需要强调的是，偏置项的作用是微调，而不是大范围的移动。

然而，这个加法形式不便于进行矩阵运算。为了将计算统一到矩阵层面，我们需要对公式进行转换。观察发现，`θ1*X1`和`θ2*X2`都是参数与特征的乘积，唯独`θ0`缺少对应的特征项。

为了解决这个问题，我们引入一个新的特征`X0`，并规定其值恒为1。这样，原方程就可以改写为：

![](img/df03348225c81b6347803e9c7c8f0575_2.png)

**公式：** `Hθ(X) = θ0*X0 + θ1*X1 + θ2*X2`，其中 `X0 ≡ 1`

现在，假设我们有`m`个样本，每个样本有`n`个特征（加上我们添加的`X0`，实际是`n+1`个）。我们可以将数据组织成矩阵`X`（`m x (n+1)`），参数组织成向量`θ`（`(n+1) x 1`）。那么，对所有样本的预测就可以用一个简洁的矩阵乘法表示：

**公式（矩阵形式）：** `Hθ(X) = X · θ`

![](img/df03348225c81b6347803e9c7c8f0575_4.png)

这个转换至关重要，它使得后续的所有数学推导和计算都能够在高效的矩阵框架下进行。在数据预处理中，为原始数据添加一列全为1的特征，正是为了完成这个转换。

## 理解误差项

![](img/df03348225c81b6347803e9c7c8f0575_6.png)

![](img/df03348225c81b6347803e9c7c8f0575_8.png)

定义了预测方程后，我们来看看模型预测与真实情况之间的差距。

对于任意一个样本`i`，我们已知其特征`Xi`和真实的标签值`yi`（例如银行历史数据中实际的贷款额度）。通过我们的模型，我们可以计算出一个**预测值** `ŷi`。

![](img/df03348225c81b6347803e9c7c8f0575_10.png)

![](img/df03348225c81b6347803e9c7c8f0575_12.png)

**预测值 `ŷi`** 是模型根据特征`Xi`和当前参数`θ`计算出的结果。

**真实值 `yi`** 是数据集中已知的、正确的目标值。

预测值与真实值之间的差异，我们称之为**误差项**，记作 `εi`。

**公式（误差项）：** `εi = yi - ŷi`

对于数据集中的每一个样本，都存在这样一个误差项。显然，我们希望模型的预测尽可能准确，即误差项越小越好。最理想的情况是误差为零，这意味着预测值与真实值完全吻合。

## 目标函数（损失函数）的引入

这引出了机器学习的核心思想：为模型定义一个学习目标。

我们给机器提供数据（`X`和`y`），同时还需要告诉它一个**目标函数**（也称为损失函数）。这个函数量化了模型预测的“不好”程度。机器的任务就是通过调整参数`θ`，来**最小化**这个目标函数。

对于线性回归，最自然的目标就是让所有样本的预测误差尽可能小。一个最直接的想法是让所有误差项的和最小。但是，误差可能有正有负，直接求和会相互抵消。因此，我们通常采用误差的平方和作为目标函数，这被称为**最小二乘法**。

**公式（目标函数-均方误差MSE）：** `J(θ) = (1/(2m)) * Σ (yi - ŷi)^2`

其中，`m`是样本数量。乘以`1/(2m)`是为了方便后续求导计算，不影响优化方向。我们的目标就是找到一组参数`θ`，使得 `J(θ)` 的值达到最小。
