# 量化交易教程：P4：03：股票市场构成 🏛️

在本节课中，我们将要学习股票市场的构成。了解市场中有哪些参与者以及他们各自扮演的角色，是理解股票交易如何运作的基础。我们将从公司和投资者开始，逐步介绍监管机构、交易所和中介机构，最后解释不同板块和指数的含义。

## 公司与投资者

上一节我们介绍了股票的分类，本节中我们来看看股票市场的参与者。首先，市场中最核心的两方是公司和投资者。

*   **公司**：是需要融资的一方。
*   **投资者**：是提供资金的一方。

公司通过发行股票向投资者融资，投资者则通过购买股票进行投资。

## 监管与服务机构

公司和投资者不能直接进行交易，这中间需要监管和服务机构来确保市场的公平与秩序。

以下是主要的监管与服务机构：

*   **证监会**：这是证券行业的监管机构。公司若想上市，必须向证监会提交各种材料。证监会负责审查公司是否存在欺诈、洗钱等违法行为，并有权决定公司能否上市或是否应退市。
*   **证券业协会**：这是一个行业自律组织，作用相对较弱。例如，证券从业资格考试通常由其主办。
*   **交易所**：交易所提供股票集中交易的场所。在中国，主要有上海证券交易所和深圳证券交易所。过去，交易者需要亲自到交易所排队交易；现在，所有交易都通过网络连接到交易所的系统进行。交易所的核心功能是处理所有买卖股票的请求。

## 证券中介机构（券商）

个人投资者通常不能直接在交易所买卖股票，这需要通过证券中介机构，也就是我们常说的券商。

这背后的历史原因是交易成本。在电子化交易之前，买卖股票需要实体凭证和核实手续，成本很高。交易所通过出售价格高昂的“交易席位”来设定门槛，只有购买席位的机构（如早期的富商）才能入场交易。这些机构为了赚回席位费，便代理众多小投资者进行交易，从而演变为今天的券商。

以下是券商的作用：

*   券商（如中信证券、中金公司等）在交易所拥有交易席位。
*   投资者通过券商提供的软件（如各券商APP或同花顺）下达买卖指令。
*   券商将投资者的指令通过自己的席位传达到交易所，完成交易。

## 交易所板块

中国的两个交易所内部还划分了不同的板块，以适应不同规模和发展阶段的企业。

以下是各交易所的板块划分：

*   **上海证券交易所**：主要设有**主板**。
*   **深圳证券交易所**：分为**主板**、**中小板**和**创业板**。中小板和创业板是国家为鼓励中小型或创业型公司发展而设立的，上市门槛相对主板更低。例如，创业板要求公司连续几年达到一定净利润即可，而主板的要求则更为严格。

## 市场指数（大盘）

对于每个板块，我们常用一个“大盘指数”来反映其整体表现。

指数代表了该板块内所有股票的综合走势。例如，上海主板的大盘指数是**上证指数（沪指）**。如果指数上涨，通常意味着板块内大多数股票在上涨，市场整体向好；反之则意味着市场走弱。它提供了一个快速判断市场整体状况的指标。

以下是主要的市场指数：

*   **上海主板**：上证指数（沪指）
*   **深圳主板**：深证成指（深成指）
*   **深圳中小板**：中小板指
*   **深圳创业板**：创业板指

## 总结

本节课中我们一起学习了股票市场的构成。我们了解了市场的核心参与者——公司与投资者，认识了确保市场规范的证监会、证券业协会和提供交易场所的交易所。我们明白了个人投资者需要通过券商进行交易的原因和历史渊源。最后，我们区分了上海和深圳交易所的不同板块，并学会了用大盘指数来快速把握市场的整体动向。理解这些基本构成，是进一步学习量化交易的重要基石。
