## 什么是AKShare？

上一节我们提到了数据分析的重要性，本节中我们来看看一个强大的工具——AKShare。AKShare是一个免费的Python工具包，它本质上是一个金融数据爬虫。它能够从多个金融网站（如新浪财经、东方财富等）爬取数据，并将返回的数据自动处理成易于分析的`DataFrame`格式。这省去了我们手动解析网页数据的麻烦。

## 什么是隐含波动率？

在了解了数据获取工具后，我们需要理解核心的分析指标——隐含波动率。隐含波动率反映了市场对未来资产价格波动的预期。当你购买期权时，你实际上是在购买这种对未来波动的预期。

*   **认购期权**：如果你买入认购期权，意味着你预期未来资产价格**会上涨**。此时，你支付的**权利金**越高，对应的**隐含波动率**通常也越高。
*   **认沽期权**：同理，交易认沽期权也涉及对隐含波动率的买卖。

![](img/f9c2a92be1bb97c046def5606afdc5d8_2.png)

例如，对于行权价为2.450的上证50ETF期权，其隐含波动率数据可以直接从新浪财经等网站获取。



![](img/f9c2a92be1bb97c046def5606afdc5d8_2.png)

AKShare可以帮助我们爬取包括50ETF、300ETF等多种ETF期权的隐含波动率数据。这些数据是动态的，AKShare在查询时实时爬取，本身并不存储数据。

![](img/f9c2a92be1bb97c046def5606afdc5d8_4.png)

## 隐含波动率曲线与市场情绪

隐含波动率并非一成不变，它与行权价的关系形成了著名的“波动率微笑曲线”。通常，行权价越接近标的资产当前价格，隐含波动率越低；行权价偏离越远，隐含波动率越高。

然而，当市场出现单边剧烈波动时，这个曲线会变得不对称。例如，在市场大跌时，投资者对下跌的恐慌情绪加剧，会导致**认沽期权的隐含波动率**显著高于认购期权。

![](img/f9c2a92be1bb97c046def5606afdc5d8_6.png)

上周市场调整，尤其是小盘股跌幅较大。此时，反映市场情绪的认沽期权隐含波动率就会升高。



![](img/f9c2a92be1bb97c046def5606afdc5d8_4.png)

## 如何进行动态分析？

![](img/f9c2a92be1bb97c046def5606afdc5d8_10.png)

静态分析很重要，但市场是瞬息万变的。因此，我们的分析也必须是动态的。真正的市场预测需要观察隐含波动率曲面在交易时间内的实时变化。

例如，在农历新年前，市场曾出现认购和认沽期权隐含波动率同时被拉升的情况。这种同步异动往往是重大市场变盘的前兆，因为大资金进场布局会在期权市场上留下痕迹。**市场通常是正确的**，这种异动信号值得经验丰富的交易者密切关注。



![](img/f9c2a92be1bb97c046def5606afdc5d8_12.png)

![](img/f9c2a92be1bb97c046def5606afdc5d8_8.png)

初学者需要注意：不要简单地因为当前隐含波动率高就去做空波动率。在市场出现单边行情时，波动率可能会持续攀升，盲目做空风险极大。
