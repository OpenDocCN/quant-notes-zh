# 机器学习与量化交易：6：突变点调参 🎯

在本节课中，我们将学习如何调整Prophet模型中的突变点参数，并理解其对预测结果的影响。我们将通过实验对比不同参数值下的模型表现，并学习如何选择最优参数。

## 概述

上一节我们介绍了Prophet模型的基本使用。本节中，我们来看看一个关键参数——突变点权重，它如何影响模型的拟合效果和预测性能。

突变点权重参数 `changepoint_prior_scale` 用于指定模型对数据中突变趋势的重视程度。其默认值为 `0.05`。

## 突变点权重的影响

该参数直接影响模型的拟合行为：
*   **权重值较大**：模型会更重视训练数据中的突变点，拟合程度更高，但可能导致**过拟合**。
*   **权重值较小**：模型对突变点不敏感，趋势预测更为保守和平稳，但可能导致**欠拟合**。

以下是参数影响的直观对比：
```python
# 设置不同的changepoint_prior_scale值进行实验
param_grid = {'changepoint_prior_scale': [0.001, 0.05, 0.1, 0.2]}
```

## 参数调优实验

我们通过对比不同参数下的训练误差和测试误差来选择最优值。

以下是实验步骤：
1.  定义一组候选参数值。
2.  为每个参数值训练一个Prophet模型。
3.  分别计算每个模型在训练集和测试集上的误差。
4.  选择测试误差最小的参数作为最优参数。

在本次实验中，我们对比了 `[0.001, 0.05, 0.1, 0.2]` 这几个参数值。

## 结果分析

根据实验结果，我们得到以下结论：
*   当 `changepoint_prior_scale=0.001` 时，模型欠拟合，无法捕捉数据中的突变趋势。
*   当 `changepoint_prior_scale=0.2` 时，模型对训练数据拟合得很好，但测试误差表明存在过拟合风险。
*   默认值 `0.05` 在欠拟合和过拟合之间取得了较好的平衡。

为了找到更优的参数，我们扩大了搜索范围。

## 扩大参数搜索

我们测试了更大的参数值 `[0.25, 0.4, 0.5, 0.6, 0.7, 0.8]`。

实验结果显示，随着参数增大，训练误差持续下降，而测试误差先下降后趋于平稳。在本案例中，当 `changepoint_prior_scale=0.7` 时，测试误差达到最低值 **66**。

因此，我们选择 **0.7** 作为本项目的最优突变点权重参数。
