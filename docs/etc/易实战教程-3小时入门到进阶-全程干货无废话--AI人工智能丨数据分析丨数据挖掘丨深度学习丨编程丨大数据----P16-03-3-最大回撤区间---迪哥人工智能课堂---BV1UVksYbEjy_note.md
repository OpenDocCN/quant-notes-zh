## 概述

上一节我们介绍了策略分析中的一些基本概念。本节中我们来看看一个衡量投资风险的指标：最大回撤。它描述的是投资过程中可能经历的最糟糕的下跌情况。

## 什么是最大回撤？

最大回撤不能仅从字面意义理解。它描述的是在投资过程中，资产净值从最高点到后续最低点的最大跌幅。简单来说，它衡量的是你投资时可能经历的最“惨”的阶段。

**核心公式**如下：

`最大回撤 = max[(Pi - Pj) / Pi]`

其中：
*   `Pi` 代表某一时刻的资产净值（高点）。
*   `Pj` 代表 `Pi` 之后某一时刻的资产净值（低点）。

这个公式计算的是从某个高点 `Pi` 到之后某个低点 `Pj` 的缩水比例。我们需要遍历所有可能的 `i` 和 `j`（`j > i`），找出其中缩水比例最大的那个值。

![](img/61dac29247ddc8cab8845891d8248b73_1.png)

## 最大回撤的意义

最大回撤是一个**风险指标**。它不直接评估策略的盈利能力，而是评估策略可能带来的最大痛苦程度。

![](img/61dac29247ddc8cab8845891d8248b73_5.png)

即使一个策略长期来看是盈利的，但在过程中可能会经历一段非常难熬的下跌期。最大回撤就是量化这段“最惨”时期的风险。例如，2008年股市从高点大幅下跌，许多投资者承受了巨大损失，这种极端情况就可以用最大回撤来度量。

![](img/61dac29247ddc8cab8845891d8248b73_7.png)

在策略回测报告中，最大回撤区间会被明确标识出来，帮助你了解策略在最坏情况下的表现。

## 如何计算最大回撤？

![](img/61dac29247ddc8cab8845891d8248b73_11.png)

接下来，我们看看如何用Python代码计算最大回撤。核心思路是：对于每一天，计算当日之前的历史最高净值与当日净值的差值比例，然后找出所有比例中的最大值。

![](img/61dac29247ddc8cab8845891d8248b73_13.png)

以下是计算步骤的分解：

![](img/61dac29247ddc8cab8845891d8248b73_15.png)

首先，我们需要一个代表资产净值的时间序列数据。这里使用Pandas库进行操作。

```python
import pandas as pd

![](img/61dac29247ddc8cab8845891d8248b73_17.png)

![](img/61dac29247ddc8cab8845891d8248b73_19.png)

# 假设 df[‘net_value’] 是资产净值序列
net_value = df[‘net_value’]
```

![](img/61dac29247ddc8cab8845891d8248b73_21.png)

第一步，计算历史累计最大值序列。即，对于每一天，计算从开始到当天为止的最高净值。

![](img/61dac29247ddc8cab8845891d8248b73_23.png)

![](img/61dac29247ddc8cab8845891d8248b73_25.png)

```python
# 使用 cummax() 函数计算历史累计最大值
historical_max = net_value.cummax()
```

![](img/61dac29247ddc8cab8845891d8248b73_27.png)

`historical_max` 序列的每个值，都代表到该日为止出现过的最高的净值。

![](img/61dac29247ddc8cab8845891d8248b73_29.png)

第二步，计算每日的回撤。即用历史最高净值减去当日净值，得到下跌的绝对值。

![](img/61dac29247ddc8cab8845891d8248b73_31.png)

```python
# 计算每日回撤（下跌的绝对值）
drawdown_abs = historical_max - net_value
```

第三步，计算每日的回撤率。即用下跌的绝对值除以对应的历史最高净值，得到缩水比例。

![](img/61dac29247ddc8cab8845891d8248b73_33.png)

![](img/61dac29247ddc8cab8845891d8248b73_35.png)

```python
# 计算每日回撤率（缩水比例）
drawdown_rate = drawdown_abs / historical_max
```

![](img/61dac29247ddc8cab8845891d8248b73_37.png)

最后，从所有每日回撤率中找出最大值，这就是整个考察期内的**最大回撤**。

```python
# 计算最大回撤
max_drawdown = drawdown_rate.max()
print(f“最大回撤为：{max_drawdown:.2%}”)
```

![](img/61dac29247ddc8cab8845891d8248b73_39.png)

将以上步骤整合，计算最大回撤的核心代码可以简洁地写为：

![](img/61dac29247ddc8cab8845891d8248b73_41.png)

```python
# 计算最大回撤的完整代码
historical_max = df[‘net_value’].cummax()
max_drawdown = ((historical_max - df[‘net_value’]) / historical_max).max()
```

这个计算结果对应回测报告图中阴影部分所表示的下跌幅度。
