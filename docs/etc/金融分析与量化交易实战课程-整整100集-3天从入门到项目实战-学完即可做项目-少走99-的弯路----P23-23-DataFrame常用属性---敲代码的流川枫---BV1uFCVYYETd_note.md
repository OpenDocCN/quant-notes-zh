# 量化交易：P23：DataFrame常用属性 📊

在本节课中，我们将学习Pandas中`DataFrame`对象的几个核心属性。这些属性可以帮助我们快速查看和理解数据的基本结构、内容以及统计信息。

上一节我们介绍了`DataFrame`对象的几种创建方式，本节中我们来看看它有哪些常用的属性。

## 常用属性详解

### 1. 行索引与值

和`Series`对象类似，`DataFrame`也有`index`和`values`属性。

*   **`index`**：用于获取`DataFrame`的行索引。行索引是数据最左侧竖排的标签。
*   **`values`**：用于获取`DataFrame`中的值。与`Series`返回一维数组不同，`DataFrame`的`values`属性返回的是一个二维数组，其中每一行是一个一维数组。

### 2. 转置

`DataFrame`的`T`属性表示转置。转置操作会将数据的行和列进行交换，即原来的行索引变为列索引，原来的列索引变为行索引。

**公式/概念**：`DataFrame.T` 等价于 `DataFrame.transpose()`。

> **注意**：在转置过程中，如果某一列同时包含整数和浮点数（例如，因为存在`NaN`这个特殊的浮点值），Pandas可能会将整数值统一转换为浮点数，以保持该列数据类型的一致性。

### 3. 列索引

`DataFrame`是二维数据结构，因此它比`Series`多了一个列索引。列索引是数据最上方横排的标签。

*   **`columns`**：用于获取`DataFrame`的列索引。

### 4. 描述性统计

`describe()`是一个方法，而非属性。它可以快速生成数据集中数值型列的描述性统计摘要。

以下是`describe()`方法返回的主要统计指标：
*   **`count`**：非空值的数量。
*   **`mean`**：平均值。
*   **`std`**：标准差。
*   **`min`**：最小值。
*   **`25%`**：第一四分位数（下四分位数）。
*   **`50%`**：中位数。
*   **`75%`**：第三四分位数（上四分位数）。
*   **`max`**：最大值。

**代码示例**：
```python
df.describe()
```

## 总结

![](img/a83693d1a89f873c6e33a82de0a4d435_3.png)

本节课中我们一起学习了`DataFrame`的几个核心属性与方法：
1.  **`df.index`**：获取行索引。
2.  **`df.columns`**：获取列索引。
3.  **`df.values`**：获取数据的二维数组形式。
4.  **`df.T`**：获取数据的转置。
5.  **`df.describe()`**：获取数值列的描述性统计信息。
