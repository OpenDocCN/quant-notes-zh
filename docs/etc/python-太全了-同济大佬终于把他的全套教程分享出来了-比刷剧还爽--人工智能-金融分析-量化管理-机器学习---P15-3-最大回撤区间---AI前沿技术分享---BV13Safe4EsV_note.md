## 概述

![](img/e1115eeb0bb6b328680d45ae7fd8313e_3.png)

最大回撤是衡量投资策略风险的关键指标之一。它描述的是在特定时间段内，资产净值从最高点到后续最低点的最大跌幅。这个指标不评估策略的盈利能力，而是专注于评估策略可能带来的最大痛苦程度，即“惨劲儿”。

## 什么是最大回撤？

在策略分析中，我们经常看到类似下图的区间标注。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_1.png)

这个区间代表了策略净值曲线中，从某个峰值到之后谷底的最大跌幅所持续的时间段。它不能仅从字面理解，其核心是描述投资过程中可能经历的最糟糕阶段。

具体来说，在投资过程中，资产价格必然有涨有跌。最大回撤区间指的就是下跌最严重、持续时间最长的那段时期。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_3.png)

## 最大回撤的公式与含义

我们可以通过公式更精确地理解这个概念。假设 `P_i` 代表第 `i` 天的资产价格，`P_j` 代表 `i` 天之后的某一天（`j > i`）的价格。

最大回撤的计算，本质上是遍历所有可能的 `(i, j)` 组合，找出价格下降幅度最大的那个区间。这个下降幅度不是简单的价差，而是相对于峰值 `P_i` 的缩水比例。

其数学表达式可以理解为寻找以下公式的最大值：

```
最大回撤 = max_{i, j (j>i)} [ (P_i - P_j) / P_i ]
```

![](img/e1115eeb0bb6b328680d45ae7fd8313e_5.png)

这个值描述的是你的资产在最糟糕的情况下缩水了多少比例。它是一个风险指标。评估一个策略时，我们不仅要看它最终能赚多少钱，还要看在这个过程中你需要承受多大的波动和风险。

例如，如果你将全部资金投入一只股票，然后它经历了连续十天从15元跌到1元的暴跌，即使它后来涨回来了，这个过程中的心理压力和实际风险也是巨大的。2008年全球金融危机时，股市从高点大幅下挫，就是最大回撤极大的现实案例，给许多投资者带来了沉重打击。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_7.png)

因此，最大回撤描述的就是在某一段时间内，你的投资组合表现“有多惨”。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_5.png)

## 如何计算最大回撤？

上一节我们介绍了最大回撤的概念，本节中我们来看看如何用Python代码进行计算。计算最大回撤需要比较多个区间，并选出下降比例最大的那个。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_15.png)

以下是计算最大回撤的核心思路和步骤：

![](img/e1115eeb0bb6b328680d45ae7fd8313e_17.png)

首先，我们需要理解一个Pandas库中的函数：`.cummax()`。我们先通过一个简单例子来认识它。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_19.png)

```python
import pandas as pd
import numpy as np

![](img/e1115eeb0bb6b328680d45ae7fd8313e_21.png)

![](img/e1115eeb0bb6b328680d45ae7fd8313e_23.png)

# 创建一个示例序列
s = pd.Series([2, 5, 3, 8, 4, np.nan, 9])
print("原始序列：")
print(s)

![](img/e1115eeb0bb6b328680d45ae7fd8313e_25.png)

# 计算累积最大值
cum_max = s.cummax()
print("\n累积最大值序列：")
print(cum_max)
```

![](img/e1115eeb0bb6b328680d45ae7fd8313e_27.png)

运行上述代码，`.cummax()` 函数会返回一个新序列，其中每个位置的值都是原始序列从开始到当前位置的最大值。例如，第一个值就是它自己（2），第二个值是前两个的最大值（max(2,5)=5），第三个值是前三个的最大值（max(2,5,3)=5），依此类推。`NaN`值会被忽略，不参与计算。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_29.png)

这个操作类似于之前学过的 `.cumsum()`（累积和），只不过这里计算的是累积最大值。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_31.png)

那么，这和最大回撤有什么关系呢？我们要找的是从某个高点 `P_i` 到之后低点 `P_j` 的最大跌幅。我们可以利用 `.cummax()` 先找出到每一天为止的历史最高价。

假设我们有一个资产价格序列 `price_series`。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_33.png)

计算步骤如下：
1.  计算该序列的累积最大值序列 `historical_max`。
2.  用每一天的 `historical_max` 减去当天的实际价格 `price_series`，得到每一天相对于其之前最高点的下跌幅度序列 `drawdown`。
3.  将 `drawdown` 序列除以对应的 `historical_max` 序列，得到每一天相对于历史高点的回撤比例序列 `drawdown_ratio`。
4.  在 `drawdown_ratio` 序列中找出最大值，即为整个时间段内的最大回撤。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_35.png)

以下是代码实现：

![](img/e1115eeb0bb6b328680d45ae7fd8313e_37.png)

```python
# 假设 df[‘close’] 是我们的收盘价序列
price_series = df['close']

# 1. 计算累积最大值
historical_max = price_series.cummax()

![](img/e1115eeb0bb6b328680d45ae7fd8313e_39.png)

# 2. 计算每日回撤（下跌幅度）
drawdown = historical_max - price_series

![](img/e1115eeb0bb6b328680d45ae7fd8313e_41.png)

# 3. 计算每日回撤比例
drawdown_ratio = drawdown / historical_max

# 4. 找出最大回撤比例
max_drawdown = drawdown_ratio.max()
print(f"最大回撤为：{max_drawdown:.2%}")
```

这段代码计算出的 `max_drawdown` 就是我们要找的最大回撤值，例如可能是 `0.307`，即30.7%。它代表了在整个观察期内，资产价值从某个高点最大缩水了30.7%。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_49.png)

## 总结

![](img/e1115eeb0bb6b328680d45ae7fd8313e_47.png)

本节课中我们一起学习了最大回撤这个重要的风险指标。
*   **定义**：最大回撤描述了投资组合在特定时期内从最高点到之后最低点的最大跌幅比例。
*   **意义**：它是一个“风险”或“痛苦”指标，帮助投资者了解策略可能面临的最大亏损幅度，对于资金管理和风险控制至关重要。
*   **计算**：其核心计算思路是借助 `.cummax()` 函数找出历史峰值，然后计算当前值相对于历史峰值的回撤比例，最后在所有比例中取最大值。

![](img/e1115eeb0bb6b328680d45ae7fd8313e_49.png)

在实际的量化分析平台中，这些计算通常已被封装好，我们可以直接调用相关函数或查看回测报告来获取最大回撤值及其发生的区间。理解其原理，能帮助我们更好地解读策略报告和评估策略风险。
