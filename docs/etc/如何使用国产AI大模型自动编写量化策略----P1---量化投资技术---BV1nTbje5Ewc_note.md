### DeepSeek：专注代码生成的利器

![](img/16885479d1ad62f21e9fb3ee1ee83e00_11.png)

DeepSeek是一款专注于代码编写和逻辑处理的大模型。它可能宣传较少，但在代码生成方面能力很强。该模型主要分为“通用对话”和“代码助手”两种模式，对于量化开发者而言，“代码助手”模式尤为实用。

![](img/16885479d1ad62f21e9fb3ee1ee83e00_13.png)

以下是使用DeepSeek代码助手生成一个简单双均线策略的示例：

![](img/16885479d1ad62f21e9fb3ee1ee83e00_15.png)

```python
# 使用Backtrader框架的双均线策略示例代码
import backtrader as bt

![](img/16885479d1ad62f21e9fb3ee1ee83e00_17.png)

![](img/16885479d1ad62f21e9fb3ee1ee83e00_19.png)

class DoubleMovingAverageStrategy(bt.Strategy):
    params = (
        ('short_period', 10),
        ('long_period', 30),
    )

    def __init__(self):
        self.sma_short = bt.indicators.SimpleMovingAverage(
            self.data.close, period=self.params.short_period
        )
        self.sma_long = bt.indicators.SimpleMovingAverage(
            self.data.close, period=self.params.long_period
        )

    def next(self):
        if not self.position:
            if self.sma_short > self.sma_long:
                self.buy()
        else:
            if self.sma_short < self.sma_long:
                self.sell()
```

![](img/16885479d1ad62f21e9fb3ee1ee83e00_23.png)

该模型理解Backtrader的语法和API，生成的策略代码通常可以直接运行。对于更复杂的策略，可能需要提供更详细的提示词。

### Kimi（月之暗面）：擅长文本处理的专家

Kimi是另一款知名的国产大模型，其最擅长的领域是文本处理和内容生成。

Kimi具备联网搜索功能，可以基于网络信息生成或润色文章。例如，可以指令它搜索“如何用MiniQMT获取历史行情”并整理成文，它能有效整合网络上的相关信息（包括个人博客文章），生成逻辑通顺的新内容。这对于撰写研究报告、策略说明或内容润色非常有帮助。

Kimi也提供了功能丰富的开放平台API。

以下是其API能力的部分特点：
*   **支持工具调用（Function Calling）**：允许模型根据描述调用外部工具或函数，构建更复杂的应用。
*   **角色扮演、JSON模式输出**：提供多样化的输出控制功能。
*   **价格**：每百万tokens价格从12元到60元不等，高于DeepSeek，但功能也更全面。

总结来说，Kimi是一个通用的文本处理专家，适合用于量化工作中的文档撰写、信息整理和内容扩写等任务。

### 智谱AI（GLM）：功能全面的通用模型

智谱AI是国内领先的通用大模型，其发展路线对标GPT，并且选择了开源策略。

智谱AI的API功能目前最为全面。

![](img/16885479d1ad62f21e9fb3ee1ee83e00_38.png)

![](img/16885479d1ad62f21e9fb3ee1ee83e00_40.png)

以下是其核心能力介绍：
*   **通用对话与代码生成**：具备与GPT类似的综合能力。
*   **Embedding模型**：提供将文本转化为向量的能力，这对于构建本地知识库、实现语义搜索等高级应用至关重要。此项能力是DeepSeek目前不具备的。
*   **价格**：其API价格是三者中最高的，例如其GLM-4模型每百万tokens价格约为100元人民币。

![](img/16885479d1ad62f21e9fb3ee1ee83e00_42.png)

因此，智谱AI更适合用于功能要求全面、尤其是需要结合本地知识库进行开发的企业级或复杂应用场景。

## 实践应用：本地集成与策略编写

在了解了各个模型的特点后，我们来看看如何将它们集成到本地研究体系中，辅助量化策略开发。

我们可以通过调用大模型的API，构建一个本地的策略编写助手。以下是一个基于DeepSeek API的简单应用思路：

1.  **设计提示词（Prompt）**：明确告诉模型你的需求、使用的框架（如Backtrader）和期望的输出格式（例如JSON）。
2.  **调用API**：将组合好的提示词发送给模型API。
3.  **解析与执行**：接收模型返回的代码，进行解析，并可在本地环境中尝试运行。

例如，我们可以设计如下提示词：
`请用Backtrader编写一个Python量化策略。策略逻辑：首先判断标的处于多头趋势（例如，价格在年线之上），当价格回踩20日均线时买入；当价格上涨至建仓价的110%时止盈卖出；当价格跌破半年线时止损卖出。请以JSON格式返回，包含策略名称和完整代码。`

模型会根据提示生成相应的策略代码框架。开发者可能需要根据返回结果进行多轮交互或细化提示词，以完善策略逻辑（例如，明确定义“多头趋势”的判断标准）。

通过这种方式，AI可以承担策略代码中模式化、基础性的编写工作，让开发者更专注于策略逻辑的构思与优化。
