## 第三步：绘制散点图矩阵
折线图展示了时间序列上的关系。为了更直接地观察两个变量数值点之间的分布关系，我们可以绘制散点图矩阵。这能帮助我们同时看到单个变量的分布以及两两变量间的散点关系。

![](img/4543978d87063b5011ec578867753527_47.png)

![](img/4543978d87063b5011ec578867753527_49.png)

![](img/4543978d87063b5011ec578867753527_51.png)

以下是绘制散点图矩阵的步骤：
```python
# 首先计算指标的日收益率（或日变化），以获得更平稳的数据
returns = np.log(data / data.shift(1)).dropna()

![](img/4543978d87063b5011ec578867753527_53.png)

# 使用pandas的scatter_matrix函数绘制散点图矩阵
from pandas.plotting import scatter_matrix

![](img/4543978d87063b5011ec578867753527_55.png)

![](img/4543978d87063b5011ec578867753527_57.png)

scatter_matrix(returns,
               alpha=0.5,        # 设置点的透明度，避免重叠
               figsize=(10, 10),
               diagonal='kde',   # 对角线绘制核密度估计图，展示单变量分布
               hist_kwds={'bins': 50}) # 如果对角线用直方图，可设置分箱数
plt.suptitle('Scatter Matrix of SPX and VIX Returns')
plt.show()
```

![](img/4543978d87063b5011ec578867753527_59.png)

生成的图形中，对角线展示了每个指标自身的分布（这里使用了更平滑的核密度估计图）。非对角线的散点图清晰地展示了SPX收益率与VIX收益率之间的负相关关系，点云呈现从左上方到右下方的趋势。

## 第四步：进行回归分析
散点图直观地展示了关系，但我们还需要一个量化的描述。接下来，我们通过线性回归来量化这种关系。回归分析可以为我们提供一个具体的公式，描述一个变量如何随另一个变量变化。

![](img/4543978d87063b5011ec578867753527_61.png)

以下是进行简单线性回归分析的步骤：
```python
import statsmodels.api as sm

![](img/4543978d87063b5011ec578867753527_63.png)

# 准备数据：将VIX作为自变量(X)，SPX作为因变量(y)
X = returns['VIX']
y = returns['SPX']

# 为模型添加常数项（截距）
X = sm.add_constant(X)

![](img/4543978d87063b5011ec578867753527_65.png)

![](img/4543978d87063b5011ec578867753527_67.png)

# 构建并拟合普通最小二乘(OLS)回归模型
model = sm.OLS(y, X).fit()

![](img/4543978d87063b5011ec578867753527_69.png)

![](img/4543978d87063b5011ec578867753527_71.png)

# 打印回归结果摘要
print(model.summary())

![](img/4543978d87063b5011ec578867753527_73.png)

![](img/4543978d87063b5011ec578867753527_75.png)

# 提取关键参数：回归系数和R-squared
print(f"回归方程: SPX_Return = {model.params['const']:.6f} + {model.params['VIX']:.6f} * VIX_Return")
print(f"R-squared: {model.rsquared:.4f}")
```

![](img/4543978d87063b5011ec578867753527_77.png)

![](img/4543978d87063b5011ec578867753527_79.png)

回归结果中的系数（特别是VIX的系数）应为负值，这从数值上证实了负相关关系。R-squared值则说明了VIX的变化能在多大程度上解释SPX的变化。

![](img/4543978d87063b5011ec578867753527_81.png)

![](img/4543978d87063b5011ec578867753527_83.png)

## 总结
本节课中我们一起学习了如何分析金融指标间的相关性。

![](img/4543978d87063b5011ec578867753527_85.png)

![](img/4543978d87063b5011ec578867753527_87.png)

1.  **数据选取**：我们首先从数据集中筛选出需要分析的两个特定指标列。
2.  **可视化探索**：通过绘制**双轴折线图**，我们在同一时间维度上观察了两个指标的反向运动趋势。接着，通过**散点图矩阵**，我们更清晰地看到了变量间的分布与负相关关系。
3.  **量化分析**：最后，我们使用**线性回归模型**对关系进行了量化，得到了描述两者关系的具体**公式**（如 `SPX_Return = a + b * VIX_Return`），并通过R-squared等统计量评估了模型的解释力。

![](img/4543978d87063b5011ec578867753527_91.png)
