# 量化投资入门：第1课：构建稳健的杠铃策略 🏋️‍♂️

![](img/d7076d9b19e84a6ebc9296982546c6ee_1.png)

在本节课中，我们将学习一个名为“杠铃策略”的稳健型投资方法。该策略旨在通过平衡高风险与低风险资产，在控制整体风险的同时，追求更高的收益潜力。我们将从策略的基本概念讲起，逐步深入到其在A股市场的具体构建、回测与优化方法。

![](img/d7076d9b19e84a6ebc9296982546c6ee_3.png)

## 杠铃策略的基本概念

近年来，市场波动加剧，许多投资者的风格从追求高收益转向寻求稳健。杠铃策略便是一种符合这种需求的思路。

所谓杠铃策略，是指在投资组合中，**同时配置短期低风险资产和长期高风险资产**，而避免持有中期风险收益特征不明确的资产。这种持仓结构形似杠铃，因此得名。

![](img/d7076d9b19e84a6ebc9296982546c6ee_5.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_7.png)

该策略的优势在于，结合了低风险资产的稳定性和高风险资产的高增长潜力，使整体策略能做到“进可攻，退可守”。

![](img/d7076d9b19e84a6ebc9296982546c6ee_9.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_11.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_13.png)

为了更直观地理解，我们可以看一个简单的例子。

![](img/d7076d9b19e84a6ebc9296982546c6ee_15.png)

以下是杠铃策略收益计算的一个示例：
假设初始资金为100万元。
*   **全部投资低风险资产**：年化收益5%，即年收益5万元。
*   **加入高风险机会A**：该机会可能带来100%收益，也可能造成50%亏损。
    *   若用10%资金（10万元）投资A，剩余90万投资低风险资产。
        *   年收益可能为：`10万 * 100% + 90万 * 5% = 14.5万元`（盈利）
        *   年收益可能为：`10万 * (-50%) + 90万 * 5% = -0.5万元`（亏损）
    *   若用20%资金（20万元）投资A，剩余80万投资低风险资产。
        *   年收益可能为：`20万 * 100% + 80万 * 5% = 24万元`（盈利）
        *   年收益可能为：`20万 * (-50%) + 80万 * 5% = -6万元`（亏损）

![](img/d7076d9b19e84a6ebc9296982546c6ee_17.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_19.png)

在这个例子中，杠铃的一端是低风险低收益资产，另一端是高风险高收益资产。随着高风险资产仓位的增加，策略的整体收益上限和风险都会同步放大。但与纯稳健策略相比，它收益更高；与纯激进策略相比，其风险又显著降低。

![](img/d7076d9b19e84a6ebc9296982546c6ee_21.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_23.png)

## 策略构建：选择“杠铃”的两端

理解了基本概念后，本节我们来看看如何为杠铃策略选择具体的投资标的。

要构建一个实用的杠铃策略，首先需要选好构成“杠铃”两端的核心策略。

![](img/d7076d9b19e84a6ebc9296982546c6ee_25.png)

**1. 低风险端：国债ETF**
我们选择国债ETF作为低风险端。它跟踪国债指数，买入即相当于持有国债。历史数据显示，其年化收益约为2.91%，最大回撤为4.88%。这符合我们对于“低风险且收益稳定”资产的要求。

**2. 高风险端：小市值股票策略**
我们选择小市值策略作为高风险端。该策略买入A股市值最小的一批股票。历史回测显示（2014年至今），该策略年化收益可达37.13%，但最大回撤也高达56.74%。这是一个典型的“高风险、高收益”策略，契合杠铃另一端的需求。

![](img/d7076d9b19e84a6ebc9296982546c6ee_27.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_29.png)

## 策略实施：具体的交易规则

![](img/d7076d9b19e84a6ebc9296982546c6ee_31.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_33.png)

选好了“杠铃”的两端，接下来我们定义具体的交易规则，将策略落到实处。

![](img/d7076d9b19e84a6ebc9296982546c6ee_35.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_37.png)

以下是该杠铃策略的月度执行步骤：
1.  **选股**：在每个月的最后一个交易日，将所有股票按总市值从小到大排序。剔除ST、*ST、退市或上市未满一年的股票。选出此时市值最小的10只股票。
2.  **调仓**：在下个月的第一个交易日开盘时，进行调仓。
    *   使用总资金的 **10%**，平均买入上一步选出的10只小市值股票（即每只股票占1%总资金）。
    *   剩余 **90%** 的资金，全部买入国债ETF。
3.  **持有**：将上述投资组合持有整整一个月。
4.  **循环**：在下一个月的最后一个交易日，卖出所有股票和ETF，然后重复步骤1-3，循环往复。

![](img/d7076d9b19e84a6ebc9296982546c6ee_39.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_41.png)

例如，在2024年5月31日选出10只最小市值股票。在6月3日（6月第一个交易日），若有100万资金，则用10万均仓买这10只股票，用90万买入国债ETF。持有至6月28日（6月最后一个交易日）全部卖出，随后在7月1日根据新的选股名单重新按1:9比例配置。

## 策略优化：调整比例与更换组件

![](img/d7076d9b19e84a6ebc9296982546c6ee_61.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_63.png)

基础策略表现稳健，但仍有优化空间。本节我们探讨两种主要的优化思路：调整仓位比例和更换策略组件。

![](img/d7076d9b19e84a6ebc9296982546c6ee_65.png)

**优化思路一：调整仓位比例**
如果投资者愿意承担更多风险以博取更高收益，可以增加高风险端（小市值策略）的仓位。
*   **将小市值策略仓位提升至20%**（国债ETF降至80%）：回测显示，年化收益提升至10.35%，最大回撤升至12.86%。
*   **将小市值策略仓位提升至30%**：收益和风险会进一步同步提升。
投资者可以通过修改代码中的仓位参数，自行测试不同比例，直到找到最适合自身风险偏好的平衡点。

![](img/d7076d9b19e84a6ebc9296982546c6ee_67.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_69.png)

**优化思路二：更换策略组件**
我们也可以尝试用其他策略替换原有的“杠铃两端”，以改变整体的风险收益特征。
例如，用另一个“低换手率股票策略”替换原有的低风险端（国债ETF）。该策略年化收益约15%，最大回撤约30%，其风险与收益均高于国债ETF。
*   **构建新杠铃策略**：用90%资金配置“低换手率策略”，10%资金配置“小市值策略”。
*   **回测结果**：新策略年化收益约为18.1%，最大回撤为26%。其收益超过了之前用国债ETF构建的杠铃策略，但风险也相应增加。

## 总结与建议

本节课我们一起学习了杠铃策略从理论到实践的全过程。

![](img/d7076d9b19e84a6ebc9296982546c6ee_75.png)

![](img/d7076d9b19e84a6ebc9296982546c6ee_77.png)

我们首先介绍了杠铃策略的核心思想：通过同时配置极低风险和极高风险的资产，放弃中庸的中等风险资产，来实现风险与收益的优化平衡。接着，我们以**国债ETF（低风险端）** 和**小市值股票策略（高风险端）** 为例，详细讲解了策略的具体构建步骤和月度调仓规则。通过历史回测，我们验证了该基础策略在提升收益的同时，能有效控制整体波动。最后，我们探讨了两种优化方法：**调整两端资产的仓位比例**和**更换收益风险特征不同的策略组件**，并展示了优化后的效果。
