## 概述
在本节课中，我们将要学习NumPy数组的“花式索引”。这是一种非常灵活且强大的数据选取方式，它允许我们根据一个整数数组（即索引列表）来选取数组中任意位置的元素，即使这些位置没有规律。我们将通过简单的例子，帮助你理解并掌握这种索引方法。

![](img/b69ba247759fffdf1ad0d0535ca8c6ce_3.png)

上一节我们介绍了布尔型索引，本节中我们来看看如何使用花式索引来选取数据。

## 花式索引的基本概念
花式索引的核心思想是：通过传递一个整数列表（或数组）来选取数组中对应位置的元素。这与我们之前学习的切片和布尔索引不同，它不要求索引是连续的或基于条件判断的。

例如，对于一个一维数组，如果我们想选取第1、3、4、6、7号元素，就可以使用花式索引。

![](img/b69ba247759fffdf1ad0d0535ca8c6ce_5.png)

以下是实现花式索引的基本代码格式：
```python
import numpy as np
arr = np.arange(20)  # 创建一个0到19的数组
indices = [1, 3, 4, 6, 7]  # 定义要选取的索引列表
selected_elements = arr[indices]  # 使用花式索引
print(selected_elements)
```
执行这段代码，`selected_elements` 将包含数组 `arr` 中索引为1、3、4、6、7的元素。

![](img/b69ba247759fffdf1ad0d0535ca8c6ce_7.png)

## 二维数组的花式索引
对于二维数组，花式索引的规则同样适用，并且可以与逗号（`,`）结合使用来选择特定的行和列。逗号左边用于选择行，右边用于选择列。

以下是几种索引方式在二维数组中的混合使用示例：
```python
arr_2d = np.arange(20).reshape(4, 5)  # 创建一个4行5列的二维数组
print(arr_2d)
```
假设我们想选取第0行，第2到第4列（不包含第4列）的数据，可以这样写：
```python
# 左边是常规索引（行），右边是切片（列）
result = arr_2d[0, 2:4]
print(result)
```
如果我们想选取第0行中所有大于2的元素，可以结合常规索引和布尔型索引：
```python
# 左边是常规索引（行），右边是布尔型索引（列）
result = arr_2d[0, arr_2d[0] > 2]
print(result)
```

## 花式索引的注意事项
一个重要的注意事项是：**逗号两边不能同时使用花式索引列表**。这种写法不会得到我们期望的行列组合（即“叉乘”效果），而是会选取索引对对应的单个元素。

例如，如果我们想选取第1行和第3行，与第1列和第3列交叉位置的四个值（即(1,1), (1,3), (3,1), (3,3)），不能直接写成 `arr_2d[[1,3], [1,3]]`。

以下是错误的写法及其结果：
```python
# 这种写法会选取(1,1)和(3,3)位置的两个值，而不是四个值
wrong_result = arr_2d[[1, 3], [1, 3]]
print(wrong_result)  # 输出: [6, 18]
```
可以看到，它只选取了索引对 (1,1) 和 (3,3) 的值。

## 如何正确实现行列组合选取
那么，如何正确选取我们想要的多个行和多个列交叉区域的所有值呢？我们需要分步操作。

以下是分步选取的正确方法：
1.  首先，使用花式索引选取指定的行，并选取所有列。
2.  然后，在上一步的结果上，再选取指定的列。

具体代码如下：
```python
# 第一步：选取第1行和第3行，所有列（用冒号`:`表示）
step1 = arr_2d[[1, 3], :]
print(“第一步结果（行已选定）:”)
print(step1)

# 第二步：在第一步结果的基础上，选取所有行，第1列和第3列
final_result = step1[:, [1, 3]]
print(“\n最终结果（行列组合）:”)
print(final_result)
```
通过这两步，我们就成功选取了第1、3行与第1、3列交叉的四个元素。

## 总结
本节课中我们一起学习了NumPy数组的“花式索引”。我们了解到：
*   **花式索引**允许我们使用一个整数列表来无规律地选取数组元素。
*   在**二维数组**中，可以通过逗号分隔行和列的索引方式，并混合使用常规索引、切片、布尔索引和花式索引。
*   一个关键的注意事项是避免在逗号**两边同时使用花式索引列表**，因为这会导致选取索引对对应的单个元素，而非行列区域。
*   要选取特定的行和列组合区域，应采用**分步索引**的方法：先选取行，再在结果上选取列。
