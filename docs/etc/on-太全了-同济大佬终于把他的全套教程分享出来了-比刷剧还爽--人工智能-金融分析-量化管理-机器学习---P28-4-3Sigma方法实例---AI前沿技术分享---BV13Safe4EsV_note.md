# 量化投资与数据处理：P28：4-3：3-Sigma方法实例 📊

在本节课中，我们将要学习一种基于统计学原理的数据去极值方法——3-Sigma方法。我们将理解其核心思想，并通过Python代码实现它，以处理数据中的异常值。

## 概述

上一节我们介绍了分位数和中位数绝对偏差（MAD）去极值法。本节中，我们来看看另一种基于正态分布假设的经典方法：3-Sigma方法。它的原理非常直观，核心是利用数据的均值和标准差来界定“正常”数据的范围。

## 3-Sigma方法原理 📈

3-Sigma方法的理论基础是正态分布（也称高斯分布）。我们可以将数据的分布想象成一个“钟形曲线”。

![](img/15f248818cf82d96e41082d8183406cd_1.png)

![](img/15f248818cf82d96e41082d8183406cd_1.png)

在这个分布中，大部分数据会聚集在均值附近。距离均值越远，数据出现的概率就越低。那些落在远离均值区域的数据点，就更可能是我们需要处理的“极值”或“离群点”。

3-Sigma具体是指：以数据的均值为中心，向左右各扩展3倍的标准差（σ），形成一个区间。根据统计学知识，在正态分布中，约有99.7%的数据会落在这个区间内。

因此，3-Sigma法的思路是：**将落在（均值 - 3σ， 均值 + 3σ）区间内的数据视为正常数据，而将落在此区间外的数据视为极值并进行规范处理**。这样，我们就保留了绝大多数（99.7%）的数据特征，同时抑制了极端值的影响。

## 代码实现 💻

![](img/15f248818cf82d96e41082d8183406cd_3.png)

理解了原理后，我们来实现3-Sigma方法。其步骤非常清晰：计算数据的均值和标准差，然后根据指定的倍数N（例如3）计算上下限。

![](img/15f248818cf82d96e41082d8183406cd_5.png)

以下是实现该功能的核心代码步骤：

首先，我们需要计算数据的两个关键统计量：均值（mean）和标准差（std）。

![](img/15f248818cf82d96e41082d8183406cd_7.png)

```python
# 计算均值和标准差
mean = data.mean()
std = data.std()
```

![](img/15f248818cf82d96e41082d8183406cd_9.png)

接着，根据公式计算数据的正常值范围上下限：
*   **上限（upper_limit）** = 均值 + N × 标准差
*   **下限（lower_limit）** = 均值 - N × 标准差

其中，N通常取3，代表3-Sigma。我们可以将其参数化，以便灵活调整。

![](img/15f248818cf82d96e41082d8183406cd_11.png)

![](img/15f248818cf82d96e41082d8183406cd_13.png)

```python
def sigma_method(data, n=3):
    """
    使用3-Sigma方法处理数据极值。
    参数:
        data: 输入的数据序列（如Pandas Series）。
        n: Sigma的倍数，默认为3。
    返回:
        series: 处理后的数据序列。
        lower_limit: 计算出的下限。
        upper_limit: 计算出的上限。
    """
    mean = data.mean()
    std = data.std()

    upper_limit = mean + n * std
    lower_limit = mean - n * std

    # 将超出范围的值规范到边界上
    series = data.clip(lower=lower_limit, upper=upper_limit)

    return series, lower_limit, upper_limit
```

![](img/15f248818cf82d96e41082d8183406cd_15.png)

最后，我们使用`clip`函数，将所有小于下限的值设置为下限，所有大于上限的值设置为上限，从而完成去极值操作。

![](img/15f248818cf82d96e41082d8183406cd_17.png)

## 总结

![](img/15f248818cf82d96e41082d8183406cd_35.png)

![](img/15f248818cf82d96e41082d8183406cd_37.png)

本节课中我们一起学习了3-Sigma去极值法。我们来回顾一下核心要点：

![](img/15f248818cf82d96e41082d8183406cd_39.png)

*   **原理**：基于数据服从正态分布的假设，认为距离均值3个标准差以外的数据是极值。
*   **核心公式**：上下限由 **均值 ± N × 标准差** 确定。
*   **操作**：不是删除极值，而是将其规范到计算出的上下限边界上。
*   **定位**：这是数据预处理“三步走”（去极值、标准化、缺失值处理）中第一步“去极值”的第三种常用方法。

![](img/15f248818cf82d96e41082d8183406cd_41.png)

至此，我们已掌握了分位数法、MAD法和3-Sigma法这三种主要的去极值技术，它们都能有效地将数据规范在一个合理的范围内，为后续分析建模打下坚实基础。
