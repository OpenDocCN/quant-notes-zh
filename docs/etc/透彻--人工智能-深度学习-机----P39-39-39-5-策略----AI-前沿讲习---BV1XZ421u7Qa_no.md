**1. 收益对比**
*   **基准收益**：这是市场基准（如沪深300指数）在同一时期的收益率。在我们的回测中，基准年化收益率为负值。
*   **策略收益**：这是我们构建的因子选股策略所获得的收益率。结果显示，策略收益虽然在某些时段也为负，但整体上优于基准收益，甚至在后期转为正值。

**2. 风险调整后收益**
*   **夏普比率 (Sharpe Ratio)**：它衡量的是每承受一单位总风险，会产生多少超额报酬。公式为：
    `Sharpe Ratio = (策略收益率 - 无风险利率) / 策略收益波动率`
    我们的策略夏普比率较低，说明其风险调整后的收益表现一般。
*   **最大回撤 (Max Drawdown)**：这表示策略在任一历史时点从峰值到谷底的最大亏损幅度。我们的策略最大回撤约为9%，这个数值可以接受，表明策略在下跌行情中的风险控制尚可。

**第一步：初始化与定时设置**
在策略的 `initialize` 函数中，我们设置了按月调仓的定时器。

```python
schedule_function(rebalance, date_rule=date_rules.month_start())
```

**第二步：股票池筛选与因子计算**
在 `rebalance` 函数中：
1.  过滤掉ST、停牌等不满足基本交易条件的股票。
2.  查询剩余股票的**市值**和**市净率**因子。
3.  对因子进行**中性化处理**（例如，消除行业影响），这是我们自定义的数据预处理步骤。
    ```python
    # 示例：使用线性回归进行市值中性化
    model = sm.OLS(y, X).fit()
    factor_neutral = model.resid
    ```

**第三步：生成交易信号**
对处理后的因子进行排序和筛选。例如，我们选择市值和市净率都处于最低30%分位数的股票，构建目标股票池。
```python
# 选择市值最小的30%股票
small_cap = market_cap.quantile(0.3)
selected = market_cap[market_cap <= small_cap]
```

**第四步：执行调仓交易**
1.  卖出当前持有但不在新股票池中的股票。
    ```python
    order_target_percent(stock, 0)  # 将持仓比例调整为0
    ```
2.  将资金平均分配到新股票池中的所有股票上。
    ```python
    for stock in target_list:
        order_target_percent(stock, 1.0 / len(target_list))
    ```
