## 概述
在本节课中，我们将学习一种在金融数据分析中常用的数据预处理技术：**基于百分位去极值**。我们将通过编写一个Python函数，将数据中超出指定百分位范围（例如，小于25%分位数或大于75%分位数）的极端值，替换为该分位数的边界值，从而将数据规范到一个更合理的范围内。

![](img/02b1b7c23cbcd4504f311af39f16aca8_6.png)

## 核心思路
我们的目标是：给定一个数据序列，计算其**第一四分位数（Q1）** 和**第三四分位数（Q3）**。将所有小于Q1的值替换为Q1，将所有大于Q3的值替换为Q3。这样，数据就被“修剪”并规范到了Q1到Q3的范围内。

![](img/02b1b7c23cbcd4504f311af39f16aca8_8.png)

![](img/02b1b7c23cbcd4504f311af39f16aca8_10.png)

### 1. 定义函数框架
首先，我们需要定义一个函数，它接收三个参数：待处理的数据序列（Pandas Series）、希望保留的最小百分位和最大百分位。

![](img/02b1b7c23cbcd4504f311af39f16aca8_20.png)

```python
def winsorize_by_percentile(series, low_percentile, high_percentile):
    """
    基于百分位对数据进行去极值处理。

    返回:
        处理后的Pandas Series。
    """
```

### 2. 数据排序
为了计算分位数，我们需要先将数据按从小到大的顺序进行排序。

```python
    sorted_series = series.sort_values()
```

### 3. 计算分位数值
使用Pandas的`quantile`函数，可以方便地计算出指定百分位对应的数值。

![](img/02b1b7c23cbcd4504f311af39f16aca8_28.png)

![](img/02b1b7c23cbcd4504f311af39f16aca8_30.png)

```python
    # 计算指定百分位对应的值
    quantiles = sorted_series.quantile([low_percentile, high_percentile])
    low_bound = quantiles.iloc[0]  # 例如，0.25分位数的值
    high_bound = quantiles.iloc[1] # 例如，0.75分位数的值
```

![](img/02b1b7c23cbcd4504f311af39f16aca8_32.png)

### 4. 应用边界限制
我们将使用NumPy的`clip`函数，将序列中所有小于下限的值设置为下限，所有大于上限的值设置为上限。

![](img/02b1b7c23cbcd4504f311af39f16aca8_34.png)

```python
    import numpy as np
    # 使用np.clip进行边界限制
    winsorized_series = np.clip(series, low_bound, high_bound)
    return winsorized_series
```

![](img/02b1b7c23cbcd4504f311af39f16aca8_36.png)

## 实际应用说明
上一节我们介绍了基于百分位去极值的基本原理和代码实现。本节中我们来看看在实际应用中需要注意的细节。

![](img/02b1b7c23cbcd4504f311af39f16aca8_60.png)

![](img/02b1b7c23cbcd4504f311af39f16aca8_62.png)

在实际的量化分析中，我们通常不会使用像0.25和0.75这样宽泛的范围，因为这会过滤掉过多数据。更常见的做法是使用更严格的边界，例如**0.025和0.975**，即只处理分布两端各2.5%的极端值。本节课为了在图表上更清晰地展示效果，特意选择了较大的范围。大家在实际使用时，应根据具体数据和因子特性调整百分位参数。

## 总结
本节课我们一起学习了如何实现**基于百分位的去极值方法**。我们掌握了其核心思想：通过计算数据的特定分位数（如Q1和Q3）作为边界，并使用`np.clip`函数将超出边界的数据“修剪”到边界上。这种方法能有效减少极端异常值对后续数据分析（如计算均值、方差或构建模型）的干扰，是金融数据预处理中的一个重要步骤。记住，关键在于根据实际情况选择合适的百分位阈值。
