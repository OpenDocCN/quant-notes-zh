## 夏普比率的定义与作用
上一节我们介绍了投资回报率，本节中我们来看看如何衡量风险与收益的关系。夏普比率描述的是：在承担单位风险的情况下，我们能获得多少超额收益。

我们可以用一个生活中的例子来理解：一份在叙利亚日薪数万的雇佣兵工作，其高薪对应的是极高的生命风险。夏普比率就是用来评估，为了获得一定的收益，我们所承担的风险是否“值得”。

![](img/e51581ca40585aebf829b9f2f3619858_1.png)

**核心公式**：
`夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差`

![](img/e51581ca40585aebf829b9f2f3619858_3.png)

指标越高，意味着在承担相同风险的情况下，获得的超额收益越高，因此该投资组合越有吸引力。在选股时，在其他条件相似的情况下，我们倾向于选择夏普比率更高的股票。

![](img/e51581ca40585aebf829b9f2f3619858_5.png)

## 夏普比率的计算逻辑
理解了夏普比率的意义后，我们来看看它的具体计算逻辑。它主要比较两个部分：

1.  **超额收益**：你的投资组合收益率与一个“无风险收益率”（如国债利率、银行固定理财收益）的差值。这代表了你的投资比“躺平”赚取无风险收益多赚了多少钱。
2.  **风险（波动）**：用投资组合收益率的标准差来衡量。标准差越大，说明收益波动越大，风险越高。

夏普比率就是将你多赚的这部分收益（超额收益），除以你所承担的风险（波动）。结果就是“每单位风险所带来的超额收益”。

## 使用Python计算夏普比率
接下来，我们通过Python代码来实际计算一下夏普比率。假设我们已经有了股票的历史日收益率数据。

以下是计算夏普比率的关键步骤和代码：

```python
import numpy as np
import pandas as pd

# 假设 returns 是一个Pandas Series，包含了投资组合的日收益率序列
# 例如：returns = 股票每日的 (今日收盘价 - 昨日收盘价) / 昨日收盘价

![](img/e51581ca40585aebf829b9f2f3619858_7.png)

# 步骤1: 设定年化无风险利率，例如3%，并转换为日利率
annual_risk_free_rate = 0.03
# 假设一年有250个交易日
trading_days_per_year = 250
daily_risk_free_rate = annual_risk_free_rate / trading_days_per_year

![](img/e51581ca40585aebf829b9f2f3619858_9.png)

# 步骤2: 计算超额收益（日收益率 - 日无风险利率）
excess_returns = returns - daily_risk_free_rate

# 步骤3: 计算夏普比率
# 公式: (超额收益的均值 * 年化因子) / (收益率的标准差 * 年化因子)
# 年化因子是 sqrt(年交易天数)
sharpe_ratio = (excess_returns.mean() * trading_days_per_year) / (returns.std() * np.sqrt(trading_days_per_year))

print(f"夏普比率为: {sharpe_ratio}")
```

**代码解释**：
*   `excess_returns.mean()`：计算日平均超额收益。
*   `returns.std()`：计算日收益率的标准差，代表日风险。
*   乘以 `trading_days_per_year` 和 `np.sqrt(trading_days_per_year)` 是为了将日均值和日标准差**年化**，使得不同周期的夏普比率可以进行比较。

![](img/e51581ca40585aebf829b9f2f3619858_11.png)

## 结果解读与应用
运行上述代码后，我们可以得到具体股票的夏普比率数值。

![](img/e51581ca40585aebf829b9f2f3619858_13.png)

*   **正值**：表示收益跑赢了无风险利率。**数值越大越好**，说明单位风险带来的超额回报越高。
*   **负值**：表示收益还不如无风险利率，通常不予考虑。
*   **比较**：当我们计算多只股票的夏普比率后，可以直接比较其数值。例如，股票A的夏普比率为1.2，股票B为0.8，那么在风险调整后收益的角度，股票A是更优的选择。

![](img/e51581ca40585aebf829b9f2f3619858_15.png)

> **注意**：夏普比率是一个重要的参考指标，但并非唯一标准。在实际投资决策中，还需要结合最大回撤、索提诺比率等其他指标，以及基本面分析进行综合判断。
