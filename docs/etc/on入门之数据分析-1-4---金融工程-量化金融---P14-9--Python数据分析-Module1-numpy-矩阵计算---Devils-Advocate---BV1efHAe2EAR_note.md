## 矩阵的基本运算

![](img/cf5a345de45d72546ed8152ec4149bcc_7.png)

首先，我们需要理解Python原生列表（list）与NumPy数组（array）在运算上的区别。列表不支持直接的数学运算，而NumPy数组支持。

![](img/cf5a345de45d72546ed8152ec4149bcc_9.png)

例如，定义两个列表：
```python
x = [1, 2, 3]
y = [4, 5, 6]
```
如果尝试 `x + y`，结果会是列表的拼接 `[1, 2, 3, 4, 5, 6]`，而非数学上的逐元素相加。同样，`x - y` 会报错。

![](img/cf5a345de45d72546ed8152ec4149bcc_11.png)

![](img/cf5a345de45d72546ed8152ec4149bcc_13.png)

为了进行数学运算，我们需要将它们转换为NumPy数组：
```python
import numpy as np
x = np.array([1, 2, 3])
y = np.array([4, 5, 6])
```

![](img/cf5a345de45d72546ed8152ec4149bcc_15.png)

现在，我们可以进行各种运算了。

![](img/cf5a345de45d72546ed8152ec4149bcc_17.png)

以下是NumPy数组支持的基本运算：

![](img/cf5a345de45d72546ed8152ec4149bcc_18.png)

*   **加法**：`x + y` 的结果是 `[5, 7, 9]`，即对应位置元素相加。
*   **减法**：`x - y` 的结果是 `[-3, -3, -3]`。
*   **乘法**：`x * y` 的结果是 `[4, 10, 18]`，这是**逐元素相乘**。
*   **除法**：`x / y` 的结果是 `[0.25, 0.4, 0.5]`。
*   **指数运算**：`x ** 2` 的结果是 `[1, 4, 9]`。

![](img/cf5a345de45d72546ed8152ec4149bcc_20.png)

![](img/cf5a345de45d72546ed8152ec4149bcc_22.png)

## 点积运算

除了逐元素运算，线性代数中更重要的运算是**点积**（Dot Product）。

![](img/cf5a345de45d72546ed8152ec4149bcc_24.png)

![](img/cf5a345de45d72546ed8152ec4149bcc_26.png)

在NumPy中，可以使用 `np.dot()` 函数或数组的 `.dot()` 方法进行计算：
```python
result = np.dot(x, y)  # 或者 result = x.dot(y)
```
点积的计算公式为：`1*4 + 2*5 + 3*6 = 32`。这在线性代数中对应着行向量与列向量的乘法。

## 数组的形状与维度

![](img/cf5a345de45d72546ed8152ec4149bcc_28.png)

理解数组的**形状**（shape）和**维度**（dimension）至关重要。

![](img/cf5a345de45d72546ed8152ec4149bcc_30.png)

我们可以创建一个二维数组：
```python
z = np.array([[1, 2, 3], [4, 5, 6]])
```
这个数组 `z` 看起来有6个数字，但实际上它包含2个元素（两个子数组），每个子数组有3个元素。

![](img/cf5a345de45d72546ed8152ec4149bcc_32.png)

以下是查看数组属性的方法：

![](img/cf5a345de45d72546ed8152ec4149bcc_34.png)

![](img/cf5a345de45d72546ed8152ec4149bcc_36.png)

*   **形状**：`z.shape` 返回 `(2, 3)`，表示2行3列。
*   **转置**：`z.T` 将数组的行列互换，形状变为 `(3, 2)`。
*   **数据类型**：`z.dtype` 返回数组元素的数据类型，例如 `int32`（32位整数）。

![](img/cf5a345de45d72546ed8152ec4149bcc_38.png)

## 数据类型转换

有时我们需要改变数组中数据的类型。例如，将整数数组转换为浮点数数组，以便进行包含小数的运算。

![](img/cf5a345de45d72546ed8152ec4149bcc_40.png)

![](img/cf5a345de45d72546ed8152ec4149bcc_41.png)

转换方法如下：
```python
z_float = z.astype(‘float32’)
print(z_float.dtype)  # 输出：float32
```
`astype(‘float32’)` 方法创建了一个新数组，其元素被转换为32位浮点数。
