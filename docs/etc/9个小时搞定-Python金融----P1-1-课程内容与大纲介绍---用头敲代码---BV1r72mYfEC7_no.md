## 课程风格与工具

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_21.png)

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_23.png)

上一节我们了解了课程的知识框架，本节中我们来看看课程的教学风格和将使用的工具。本课程是纯实战导向，重点在于通过代码实现金融分析与量化交易任务，不涉及PPT理论讲解或炒股教学。

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_25.png)

课程将全程在代码编辑界面中讲解，所有知识点都将通过实际数据和代码案例进行引入和演示。

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_27.png)

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_29.png)

在数据分析与金融数据处理部分，我们将使用Jupyter Notebook等工具，并重点讲解`NumPy`（用于数值计算）和`Pandas`（用于数据分析）这两个核心Python库的使用方法。例如，读取数据可能使用如下代码：
```python
import pandas as pd
data = pd.read_csv(‘stock_data.csv’)
```

在量化交易策略部分，我们将使用专业的回测平台。当你构思出一个交易策略后，可以在该平台中编写策略代码，并将其应用于历史数据（例如2019年至2020年），以检验策略的盈利能力。

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_31.png)

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_33.png)

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_35.png)

回测平台能够提供详细的交易报告，包括：

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_37.png)

*   **每日交易详情**：记录每天买卖了哪些股票。
*   **持仓情况**：展示投资组合的构成。
*   **账户信息**：例如，从初始本金10万元开始，经过一段周期（如3年或5年）后，账户的最终金额和盈亏状况。
*   **关键指标**：如策略的总收益、超额收益（与基准收益如大盘指数的比较）等，用于评估策略表现。

## 详细课程大纲

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_39.png)

上一节我们介绍了课程使用的工具，本节中我们来看看详细的课程大纲安排。课程大纲同样分为三个循序渐进的模块，确保零基础的学习者也能无压力跟上。

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_41.png)

以下是三个模块的具体内容：

![](img/e99ca0aa21a3c25f734c77df1ff24e0d_43.png)

1.  **Python必备工具包实战**：作为基础模块，将讲解Python核心知识点、环境配置与工具安装。重点学习`NumPy`和`Pandas`库，为后续的金融数据分析打下坚实基础。
2.  **Python金融数据分析实战**：本模块进入实战，使用真实的股票数据。你将学习如何对金融数据进行建模、统计与分析，并首次在量化平台上完成一个完整的回测项目，验证策略的历史表现。
3.  **量化交易策略深入实战**：本模块深入量化交易核心，讲解经典的交易策略。你将学习在实际操作中如何处理数据、分析因子、改进策略，并最终实现能在历史回测中获得较好收益的策略。
