## 第一节：风险与收益的本质

上一节我们介绍了多种策略框架和因子挖掘方法。本节中，我们来看看决定策略最终表现的底层逻辑：风险。

![](img/198ebe15bfc82c1a56673f4dea24fb14_3.png)

![](img/198ebe15bfc82c1a56673f4dea24fb14_4.png)

![](img/198ebe15bfc82c1a56673f4dea24fb14_6.png)

![](img/198ebe15bfc82c1a56673f4dea24fb14_8.png)

一个策略的潜在收益，本质上是由其承担的风险决定的。风险与收益同源。策略所承担的潜在风险，决定了其潜在的收益。

一个简单的类比是购买保险。购买保险需要支付保费。按照平均出险率计算，保险公司可以获得盈利。作为保险买家，我们付出保费，其回报是在重大突发事件时获得相应的损失弥补。

回到金融市场，我们有两种类似的收益风格：
*   **保险买家风格**：希望在市场剧烈波动或下跌时获得额外收益。这类投资者需要接受的事实是，其长期回报率可能会低于市场平均回报。
*   **保险卖家风格**：愿意出售保险，通过赚取额外的风险溢价来获得更高的回报。这并非意味着其投资水平更高，而是其赚取了承担额外风险所对应的溢价。

因此，风险敞口决定了我们的收益。投资者投资股票而非更安全的债券，是期待从股票中获得额外回报。这些回报来自于一系列风险因素。例如，小市值公司曾被认为能稳定跑赢大市值公司，后来被证实这是一个风险因子（规模因子）。在债券中，长期债券收益率通常高于短期债券，因为更长期限的借贷承担了更大的风险，因此需要获得更多的风险补偿。

持续的**风险溢价**会导致不同资产间的收益率产生区别，例如股票与债券、长期债券与短期债券。

## 第二节：风险偏好的时变性与收益分布

上一节我们介绍了风险与收益的基本关系。本节中，我们来看看风险如何随时间变化，以及相同风险指标下可能隐藏的不同风险形态。

![](img/198ebe15bfc82c1a56673f4dea24fb14_10.png)

### 风险偏好的时变性

![](img/198ebe15bfc82c1a56673f4dea24fb14_12.png)

人们的风险偏好并非保持不变，而是随时间变化的量。如果能够捕捉人们风险偏好的变化，就存在低买高卖的交易机会。我们赌的是风险偏好会出现均值回归。例如，当前资产存在过高溢价时，我们可以卖出，等待风险偏好回归正常时资产价格回落。

然而，投资者的交易心理和市场状态不断变化，不存在完全正确的估值或市场状态，这需要每个人有自己的判断。

### 夏普比率与收益分布

作为理性投资者，我们关心资产的**夏普比率**。但一个理性的假设是：对于夏普比率相同的两个资产，我们在选择时没有偏好。然而，这个假设可能并不完全正确。

具有相同夏普比率的资产，可能存在完全不同的收益分布：
*   **保险买家模式**：大部分时间稳定赚取小额收益，但在极端市场事件（黑天鹅）时可能产生大额亏损。这种分布被称为**负偏态**。
*   **保险卖家模式**：大部分时间承受小额亏损（如支付期权时间价值），但在市场出现大幅波动时获得巨额收益。这种分布被称为**正偏态**。

这两种形态体现了**偏度**的问题。以下代码展示了如何生成具有相同夏普比率但不同偏度的收益分布：

![](img/198ebe15bfc82c1a56673f4dea24fb14_14.png)

```python
import numpy as np
import matplotlib.pyplot as plt

![](img/198ebe15bfc82c1a56673f4dea24fb14_16.png)

def generate_skewed_returns(target_sharpe=1.0, target_skew=1.0, target_vol=0.2, size=2500, days_in_year=252):
    """
    生成具有目标夏普比率和偏度的日收益率序列。
    
    参数:
    target_sharpe: 目标年化夏普比率
    target_skew: 目标偏度
    target_vol: 目标年化波动率
    size: 样本数量
    days_in_year: 年化交易日数
    """
    # 计算年化收益率和日度参数
    annual_return = target_sharpe * target_vol
    daily_return = annual_return / days_in_year
    daily_vol = target_vol / np.sqrt(days_in_year)
    
    # 使用伽马分布生成具有特定偏度的基础样本
    # 伽马分布参数: shape (k), scale (theta)
    # 偏度 = 2 / sqrt(k)
    shape_param = (2.0 / (abs(target_skew) + 1e-6)) ** 2  # 避免除零
    scale_param = daily_vol / np.sqrt(shape_param)
    
    # 生成样本
    samples = np.random.gamma(shape=shape_param, scale=scale_param, size=size)
    
    # 调整符号以匹配目标偏度的正负
    adjusted_signal = -1.0 if target_skew < 0 else 1.0
    samples = samples * adjusted_signal
    
    # 调整均值至目标日收益率
    sample_mean = np.mean(samples)
    adjustment = daily_return - sample_mean
    adjusted_samples = samples + adjustment
    
    return adjusted_samples

# 生成正偏态和负偏态的收益率序列
positive_skew_returns = generate_skewed_returns(target_sharpe=1.0, target_skew=1.0, target_vol=0.2)
negative_skew_returns = generate_skewed_returns(target_sharpe=1.0, target_skew=-1.0, target_vol=0.2)

# 绘制分布图
fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].hist(positive_skew_returns, bins=100, alpha=0.7, color='blue', edgecolor='black')
axes[0].set_title('正偏态收益分布 (大部分小亏，偶尔大赚)')
axes[0].set_xlabel('日收益率')
axes[0].set_ylabel('频率')

axes[1].hist(negative_skew_returns, bins=100, alpha=0.7, color='red', edgecolor='black')
axes[1].set_title('负偏态收益分布 (大部分小赚，偶尔大亏)')
axes[1].set_xlabel('日收益率')
axes[1].set_ylabel('频率')

plt.tight_layout()
plt.show()
```

**策略启示**：
*   **正偏态策略**（如趋势跟踪、买入期权）：大部分时间可能小亏或持平，但能抓住大趋势获得巨额收益。这类策略对黑天鹅事件有较强抗冲击力，但需要反人性的纪律（如严格止损，让利润奔跑）。
*   **负偏态策略**（如做空波动率、做市、套利）：大部分时间稳定赚取小额收益，但在市场极端波动时可能产生巨大回撤。

机构投资者通常更偏好正偏态策略，因为需要控制最大回撤，避免单日出现巨大亏损。个人投资者需要根据自身风险承受能力进行选择。

## 第三节：其他关键风险维度

上一节我们探讨了收益分布形态的风险。本节中，我们来看看其他几个在构建和管理策略时必须考虑的关键风险维度。

以下是量化交易中需要系统化考虑的其他风险：

1.  **流动性风险**：当资金规模达到一定程度时，必须考虑资产的流动性。在小盘股中投入大量资金可能导致难以出货。机构投资者偏好股票、期货等流动性好的资产，以应对大额申购赎回而不对价格造成过大冲击。低流动性资产（如房地产、私募股权）通常要求更高的收益率作为补偿。

2.  **交易壁垒与成本**：看到市场机会不等于能有效参与。这包括：
    *   **市场准入壁垒**：如外国投资者投资A股的限制。
    *   **技术成本**：如高频交易所需的交易所服务器、专线租赁。
    *   **尽调成本**：如私募股权投资前的法律、会计尽调费用。
    必须权衡潜在收益与所有相关成本。

3.  **行为金融学风险**：传统金融学的理性假设常被现实打破。行为金融学提供了许多解释，例如：
    *   **处置效应**：投资者倾向于过早卖出盈利资产（落袋为安带来成就感），而过久持有亏损资产（承认错误带来痛苦）。
    基于行为偏误可以设计相应的策略。

![](img/198ebe15bfc82c1a56673f4dea24fb14_18.png)

![](img/198ebe15bfc82c1a56673f4dea24fb14_20.png)

4.  **运营风险**：指由内部流程、人员、系统或外部事件导致损失的风险。经典案例是骑士资本因高频交易服务器程序错误，在45分钟内亏损4.6亿美元。稳健的IT系统和风控流程至关重要。

**核心总结**：必须树立“风险与收益同源”的观念，并清楚识别策略中蕴含的各类风险（市场风险、信用风险、流动性风险、操作风险等）。量化投资是严肃的系统工程，不鼓励初学者未经充分模拟和风控测试就投入真金白银，尤其是高杠杆领域。

## 第四节：最小扭转因子 - 一种改进的因子正交化方法

上一节我们全面探讨了各类风险。本节中，我们转向一个提升策略稳健性的技术：因子处理。我们将介绍一种比主成分分析更具可解释性的因子正交化方法。

在因子投资中，我们常使用**主成分分析**将高度相关的原始因子转化为一组互不相关（正交）的主成分。但PCA得到的因子经济含义模糊，且统计稳定性可能不足（尤其对于小特征值对应的成分）。

**最小扭转因子**方法旨在解决这个问题。它的目标是找到一组新的正交因子，使得这组新因子与原始因子之间的“扭转”最小。换言之，它在保持因子正交性的同时，最大程度地保留原始因子的可解释性。

**核心思想**：
*   **输入**：一组相关的原始因子收益率序列。
*   **目标**：找到一个线性变换（扭转矩阵 **T**），将原始因子转化为一组新的正交因子。
*   **优化问题**：最小化新因子与原始因子之间的跟踪误差，同时约束新因子的相关系数矩阵为单位矩阵（即完全正交）。

数学上，这可以表述为一个带约束的优化问题。其解法涉及数值优化（如迭代法），具体实现可参考相关论文提供的代码。

**与PCA的对比优势**：
1.  **更强的可解释性**：新因子与原始因子偏差最小，更容易理解其经济含义。
2.  **更好的稳定性**：对数据缩放等变换具有更良好的性质。
3.  **更直观的风险贡献度**：由于因子正交，可以更清晰地计算每个因子对投资组合总风险的贡献，便于实现真正的**风险平价**配置。

**应用建议**：在因子降维和组合构建阶段，可以尝试用MTB方法替代PCA，可能会得到更稳定、更易解释的因子组合，从而提升策略表现。

## 第五节：基于贝叶斯引援模型的交易策略

上一节我们介绍了一种改进的因子处理方法。本节中，我们来看一个创新的策略框架：基于贝叶斯引援模型的预测与交易。

传统的线性回归模型在金融高维（因子多）、低样本（时间序列短）场景下容易过拟合，即使使用LASSO等正则化方法，参数空间依然复杂且难以解释。

**贝叶斯引援模型** 提供了一种新思路：
1.  **核心假设**：市场收益率由 **K** 个潜在的“引援”生成。每个引援代表一种特定的市场模式或状态。
2.  **模型形式**：将线性回归问题转化为一个贝叶斯推断问题。给定观测数据 **X**，预测收益率 **Y** 的条件期望，正比于所有可能引援下预测的加权平均，权重为该引援的后验概率。
3.  **引援发现**：我们并不知道这K个引援具体是什么。解决方案是使用**聚类算法**（如K-Means）从历史数据中自动发现这些重复出现的模式。

**策略流程（以比特币高频交易为例）**：
1.  **数据**：使用限价订单簿数据（如最优60档买卖价量）。
2.  **特征**：构建特征，如价格短期平均变化、订单不平衡度等。
3.  **分段**：将数据分为三个时间段。
    *   **阶段一（模式发现）**：在第一个时间段，使用聚类算法找出最具代表性的K个市场模式（引援）。
    *   **阶段二（参数学习）**：在第二个时间段，基于发现的引援，学习贝叶斯回归模型的参数。
    *   **阶段三（策略测试）**：在第三个时间段，评估策略性能。
4.  **交易信号**：预测未来一段时间（如10秒）的价格平均变化。当预测值超过正向阈值时买入，低于负向阈值时卖出。

**策略特点**：
*   **自动化模式识别**：让机器从数据中自动发现类似“头肩顶”、“突破”等形态，而非人工定义。
*   **贝叶斯框架**：天然处理不确定性，并将先验知识（发现的模式）与当前观测结合。
*   **参数敏感性**：交易阈值 **T** 影响交易频率和持仓时间。通常，T越大，交易次数越少，但每笔交易的平均利润越高。

**对股票/期货市场的启示**：
可以将此框架迁移到其他市场。需要注意：
*   市场模式可能更复杂，需要调整聚类数量 **K** 和特征工程。
*   需要对聚类结果进行可视化，确保发现的模式具有区分度和经济意义。
*   这是一种“让数据说话”的框架，适合在尚不清楚市场具体有效因子时进行探索性分析。

## 总结

本节课中，我们一起学习了量化交易中关于风险与策略优化的核心内容：

1.  **风险是收益的根源**：深入理解了风险敞口、风险溢价、风险偏好时变性等概念，并学会了如何通过代码分析收益分布的偏度风险。
2.  **全面风险管理**：认识到除了市场风险，还需系统性管理流动性风险、交易成本、行为偏差和运营风险。
3.  **因子处理新方法**：了解了“最小扭转因子”这种比PCA更具可解释性和稳定性的因子正交化工具，有助于更好地进行风险配置。
4.  **创新策略框架**：学习了基于贝叶斯引援模型的交易策略，该框架通过聚类自动发现市场模式，并利用贝叶斯推断进行预测，为策略开发提供了新思路。

量化交易是一个不断迭代和优化的系统工程。从扎实的数据处理、因子挖掘，到深入的风险管理、策略组合优化，每一步都至关重要。鼓励大家持续学习最新研究成果，理解其核心思想，并结合市场实践进行创新和改进。
