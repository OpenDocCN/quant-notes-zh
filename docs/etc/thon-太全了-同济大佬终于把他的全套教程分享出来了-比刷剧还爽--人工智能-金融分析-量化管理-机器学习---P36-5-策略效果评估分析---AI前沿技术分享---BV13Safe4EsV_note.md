1.  **卖出不再持有的股票**：遍历需要删除的股票列表，对每只股票执行卖出操作，将其持仓比例调整为0。
    ```python
    for stock in stocks_to_delete:
        order_target_percent(stock, 0)
    ```

![](img/9d44be8114008b0204b3c81f416b1dce_29.png)

![](img/9d44be8114008b0204b3c81f416b1dce_31.png)

![](img/9d44be8114008b0204b3c81f416b1dce_33.png)

2.  **买入新入选的股票**：遍历当前满足条件的股票池，对每只股票执行买入操作。为了简化，我们采用等权重分配资金的方式。
    ```python
    for stock in current_pool:
        order_target_percent(stock, 1.0 / len(current_pool))
    ```

3.  **变量名拼写错误**：例如，将 `quantile` 误写为 `qant`，将 `list` 属性遗漏等。需要仔细检查并修正拼写。

## 回测执行与结果分析

![](img/9d44be8114008b0204b3c81f416b1dce_77.png)

![](img/9d44be8114008b0204b3c81f416b1dce_79.png)

代码调试通过后，我们启动回测。回测会模拟策略在历史数据上的表现，并生成一系列分析指标。

![](img/9d44be8114008b0204b3c81f416b1dce_81.png)

在回测执行过程中，我们观察到：
*   策略按照设定的每月频率进行调仓，并在日志中打印调仓记录。
*   回测运行需要一定时间，其速度受数据量、计算复杂度及平台负载影响。

![](img/9d44be8114008b0204b3c81f416b1dce_83.png)

回测结束后，我们可以分析关键绩效指标：

![](img/9d44be8114008b0204b3c81f416b1dce_85.png)

![](img/9d44be8114008b0204b3c81f416b1dce_87.png)

*   **累计收益率**：策略相对于初始本金的总收益。
*   **年化收益率**：将累计收益率折算为年化标准，便于比较。
*   **基准收益率**：与市场基准（如沪深300指数）的收益对比。
*   **夏普比率**：衡量每承受一单位风险所产生的超额回报。公式为 `(策略年化收益率 - 无风险利率) / 策略收益波动率`。
*   **最大回撤**：策略净值从峰值到谷底的最大跌幅，反映极端风险。

![](img/9d44be8114008b0204b3c81f416b1dce_89.png)

![](img/9d44be8114008b0204b3c81f416b1dce_91.png)

![](img/9d44be8114008b0204b3c81f416b1dce_93.png)

在本例的回测结果中，策略的年化收益率虽然不高，但在回测期内表现优于基准指数（基准收益为负），且最大回撤控制在较低水平。这表明基于市值和市净率因子的简单筛选与定期调仓逻辑，在一定条件下可能产生相对稳健的效果。

本节课中我们一起学习了如何构建并评估一个完整的因子选股策略。

整个策略流程可以概括为以下几步：
1.  **初始化与定时设置**：在策略初始化函数中设定调仓频率（如每月）。
2.  **股票池筛选**：过滤掉不符合基本条件的股票（如ST股、停牌股）。
3.  **因子计算与处理**：获取所需因子数据（如市值、市净率），并进行必要的预处理（如中性化）。
4.  **条件筛选**：根据因子逻辑（如选择市净率较低的股票）进一步筛选出目标股票池。
5.  **调仓执行**：
    *   卖出持仓中不在新股票池的股票。
    *   买入新股票池中的股票，可按等权重或其他方式分配资金。
6.  **回测评估**：运行回测，通过收益率、夏普比率、最大回撤等指标综合评价策略效果。

![](img/9d44be8114008b0204b3c81f416b1dce_117.png)

![](img/9d44be8114008b0204b3c81f416b1dce_119.png)

![](img/9d44be8114008b0204b3c81f416b1dce_121.png)

需要注意的是，本例是一个简化演示。实际策略开发中需要考虑更多因素，如交易成本、滑点、更复杂的因子模型和风险控制。回测结果也不代表未来表现，主要用于验证逻辑和优化思路。
