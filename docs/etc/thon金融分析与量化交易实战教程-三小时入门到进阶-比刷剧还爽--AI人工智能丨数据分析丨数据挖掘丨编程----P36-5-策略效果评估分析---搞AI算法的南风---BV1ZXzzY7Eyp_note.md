首先，我们需要在策略的`rebalance`函数中实现月度调仓逻辑。核心思路是：比较当前持仓的股票列表与根据因子筛选出的最新股票池，卖出不在新池子中的股票，并平均买入新池子中的所有股票。

![](img/d1dac03979d885a31d6f580a4820f88a_23.png)

以下是实现该逻辑的关键步骤：

![](img/d1dac03979d885a31d6f580a4820f88a_27.png)

![](img/d1dac03979d885a31d6f580a4820f88a_29.png)

![](img/d1dac03979d885a31d6f580a4820f88a_31.png)

```python
# 假设这是rebalance函数的一部分
# 1. 获取需要从持仓中删除的股票（即当前持有但不在新股票池中的股票）
stocks_to_delete = [s for s in current_holdings if s not in new_stock_pool]

![](img/d1dac03979d885a31d6f580a4820f88a_33.png)

![](img/d1dac03979d885a31d6f580a4820f88a_35.png)

![](img/d1dac03979d885a31d6f580a4820f88a_37.png)

![](img/d1dac03979d885a31d6f580a4820f88a_39.png)

# 2. 判断是否需要执行调仓操作
if len(stocks_to_delete) > 0:
    print(f"执行调仓操作，时间: {context.current_dt}")
    # 3. 卖出不在新池子中的股票
    for stock in stocks_to_delete:
        # 使用order_target_percent函数将持仓比例调整为0，即全数卖出
        order_target_percent(stock, 0)
    # 4. 平均买入新股票池中的所有股票
    for stock in new_stock_pool:
        # 计算平均分配的比例：1 / 股票池数量
        order_target_percent(stock, 1.0 / len(new_stock_pool))
```

1.  **函数调用参数缺失**：在调用中性化（预处理）函数时，漏传了必要的参数（如指标列名）。修正方法是将正确的参数传入函数。
2.  **模型未训练**：在使用线性回归模型进行中性化处理前，必须先调用`fit()`方法进行训练，然后才能使用`resid`属性获取残差。
    ```python
    # 错误示例：直接使用未训练的模型
    # result = sm.OLS(y, X).resid

    # 正确示例：先训练模型
    model = sm.OLS(y, X).fit()
    result = model.resid
    ```
3.  **变量名拼写错误**：例如，将`quantile`误写为`qant`，将`list`属性误写为`lst`。这类错误需要通过仔细检查变量名来修正。

## 回测结果分析

![](img/d1dac03979d885a31d6f580a4820f88a_83.png)

回测完成后，我们获得了策略的一系列绩效指标：

![](img/d1dac03979d885a31d6f580a4820f88a_85.png)

*   **累计收益率**：策略在整个回测期间的总收益。
*   **年化收益率**：将累计收益率折算为年化标准，便于比较。
*   **夏普比率**：衡量策略承担单位风险所获得的超额回报。本例中该值较低，说明风险调整后收益不够理想。
*   **最大回撤**：策略净值从高点下降到低点的最大幅度。本例中约为9%，属于可接受范围。
*   **阿尔法（α）与贝塔（β）**：阿尔法代表超额收益，贝塔代表相对于市场的波动性。本例中这两个指标均优于基准。

## 策略流程总结

![](img/d1dac03979d885a31d6f580a4820f88a_97.png)

本节课中我们一起学习了构建一个完整因子选股策略的流程：

![](img/d1dac03979d885a31d6f580a4820f88a_103.png)

![](img/d1dac03979d885a31d6f580a4820f88a_105.png)

![](img/d1dac03979d885a31d6f580a4820f88a_107.png)

![](img/d1dac03979d885a31d6f580a4820f88a_109.png)

1.  **初始化与定时**：在策略的`initialize`函数中设置调仓频率（如每月）。
2.  **股票池筛选**：在`rebalance`函数中，首先过滤掉不符合基本条件的股票（如ST股、停牌股）。
3.  **因子计算与处理**：对剩余股票计算所需因子（如市值、市净率），并进行必要的预处理（如中性化处理）。
4.  **因子筛选**：根据因子逻辑（如选择市净率最低的20%股票）生成最终的目标股票池。
5.  **调仓执行**：
    *   卖出当前持有但不在目标股票池中的股票。使用`order_target_percent(stock, 0)`实现。
    *   对目标股票池中的股票进行平均买入。使用`order_target_percent(stock, 1.0 / len(pool))`实现。
6.  **效果评估**：通过回测系统提供的图表和指标（累计收益、夏普比率、最大回撤等）全面评估策略表现。
