## 问题引入与思路

![](img/20b7518b090a70617b79d00021516564_6.png)

双均线策略的表现与短期和长期均线的窗口大小密切相关。一个核心问题是：短期窗口设为5天还是10天更合适？长期窗口设为20天还是40天更合适？

![](img/20b7518b090a70617b79d00021516564_8.png)

为了解决这个问题，我们可以采用**参数遍历**的方法。即，预设一组短期窗口值和一组长期窗口值，计算所有可能组合下的策略表现，最终选出表现最佳的一组参数。

# 定义短期均线(SMA1)和长期均线(SMA2)的参数空间

sma1_range = range(20, 61, 4)
# 长期均线窗口从180到280，步长为10
sma2_range = range(180, 281, 10)

### 3. 构建评估循环

![](img/20b7518b090a70617b79d00021516564_40.png)

接下来，我们遍历所有参数组合，对每一组参数计算策略收益。

![](img/20b7518b090a70617b79d00021516564_42.png)

![](img/20b7518b090a70617b79d00021516564_43.png)

![](img/20b7518b090a70617b79d00021516564_44.png)

```python
# 创建一个空的DataFrame来存储所有结果
results = pd.DataFrame()

![](img/20b7518b090a70617b79d00021516564_46.png)

![](img/20b7518b090a70617b79d00021516564_47.png)

# 遍历所有参数组合
for sma1, sma2 in product(sma1_range, sma2_range):
    # 复制原始数据，避免修改原数据
    df = data.copy()
    
    # 计算收益率
    df['returns'] = np.log(df['AAPL'] / df['AAPL'].shift(1))
    
    # 计算短期和长期简单移动平均线
    df['SMA1'] = df['AAPL'].rolling(window=sma1).mean()
    df['SMA2'] = df['AAPL'].rolling(window=sma2).mean()
    
    # 去除因计算移动平均产生的缺失值
    df = df.dropna()
    
    # 生成交易信号：当短期均线上穿长期均线时，为1（买入）；否则为0（持有现金）
    df['position'] = np.where(df['SMA1'] > df['SMA2'], 1, 0)
    
    # 根据交易信号计算策略收益率
    df['strategy'] = df['position'].shift(1) * df['returns']
    
    # 计算策略相对于简单买入持有的超额收益
    df['outperformance'] = df['strategy'].cumsum() - df['returns'].cumsum()
    
    # 将本次参数组合的最终结果保存
    result = pd.DataFrame({
        'SMA1': [sma1],
        'SMA2': [sma2],
        'returns': [df['returns'].cumsum().iloc[-1]],
        'strategy': [df['strategy'].cumsum().iloc[-1]],
        'outperformance': [df['outperformance'].iloc[-1]]
    })
    
    # 将本次结果追加到总结果表中
    results = pd.concat([results, result], ignore_index=True)
```

## 核心要点总结

![](img/20b7518b090a70617b79d00021516564_63.png)

![](img/20b7518b090a70617b79d00021516564_64.png)

本节课中我们一起学习了双均线策略的参数调优方法。

![](img/20b7518b090a70617b79d00021516564_66.png)

1.  **思路**：通过系统性地遍历不同的短期和长期窗口参数组合，来寻找最优策略配置。
2.  **工具**：使用 `itertools.product` 函数可以方便地生成参数组合。
3.  **流程**：对于每一组参数，重新计算均线、生成交易信号、计算策略收益，并与基准进行比较。
4.  **输出**：最终得到一个结果表格，其中包含了每一组参数的绩效表现，我们可以据此选择最佳的参数。

![](img/20b7518b090a70617b79d00021516564_67.png)

![](img/20b7518b090a70617b79d00021516564_68.png)

![](img/20b7518b090a70617b79d00021516564_69.png)

![](img/20b7518b090a70617b79d00021516564_70.png)

通过这种方法，我们可以让数据说话，客观地评估不同参数对策略效果的影响，从而避免主观猜测，为量化交易决策提供可靠依据。
