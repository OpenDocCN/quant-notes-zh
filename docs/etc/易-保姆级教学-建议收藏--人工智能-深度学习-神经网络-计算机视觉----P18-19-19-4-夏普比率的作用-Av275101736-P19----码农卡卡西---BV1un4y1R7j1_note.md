# Python机器学习与量化交易：P18：19.19.4-夏普比率的作用 📈

在本节课中，我们将要学习夏普比率这一核心概念。夏普比率是量化投资中用于衡量投资组合风险调整后收益的重要指标。我们将解释其含义、计算公式，并通过代码演示如何计算。

## 概述

上一节我们介绍了投资回报率的计算。本节中，我们来看看如何评估这些回报所承担的风险是否值得。夏普比率正是用来衡量“单位风险能带来多少超额收益”的指标。

## 夏普比率的含义

夏普比率描述的是，在承担一定风险的情况下，所能获得的超额收益。指标越高，意味着对于单位风险，投资者获得的收益补偿越高。

![](img/b962a9d351049a1873e7a3435d50c219_1.png)

举例来说，假设有两份工作：一份是稳定的办公室工作，月薪1万元；另一份是去战乱地区当雇佣兵，日薪3万元。后者的高薪对应着极高的生命风险。夏普比率就像是在计算：为了这多出来的收入，承担如此巨大的风险是否“划算”。

![](img/b962a9d351049a1873e7a3435d50c219_3.png)

在选股或构建投资组合时，我们也会面临类似选择。我们希望找到那些在承担相同风险时，能带来更高收益的资产。夏普比率可以帮助我们做出比较。

![](img/b962a9d351049a1873e7a3435d50c219_5.png)

## 夏普比率的计算公式

夏普比率的计算基于两个核心概念：投资组合的超额收益和其收益的波动性（风险）。

其公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率
*   **Rf**：无风险收益率（例如国债利率）
*   **σp**：投资组合收益率的标准差（代表风险）

这个公式的意思是：用投资组合的收益减去“无风险”的基准收益，得到“超额收益”，再除以投资组合自身的波动风险。结果数值越大，说明该投资组合的风险调整后表现越好。

## 代码实现计算

理解了公式后，我们通过Python代码来计算夏普比率。假设我们已经有了投资组合的历史日收益率数据 `returns`。

以下是计算夏普比率的步骤：

1.  设定年化无风险收益率（例如5%）。
2.  计算投资组合的平均日超额收益率（日收益率 - 日化无风险利率）。
3.  计算投资组合日收益率的标准差。
4.  将日标准差年化（乘以交易天数的平方根，假设一年250个交易日）。
5.  计算夏普比率（年化超额收益均值 / 年化标准差）。

![](img/b962a9d351049a1873e7a3435d50c219_7.png)

```python
import numpy as np

# 假设 returns 是一个包含投资组合日收益率的NumPy数组
# 例如：returns = np.array([0.01, -0.02, 0.015, ...])

![](img/b962a9d351049a1873e7a3435d50c219_9.png)

# 步骤1: 设定年化无风险利率为5%
risk_free_rate_annual = 0.05
# 转换为日利率，假设一年250个交易日
risk_free_rate_daily = risk_free_rate_annual / 250

# 步骤2: 计算平均日超额收益率
excess_returns_daily = returns - risk_free_rate_daily
avg_excess_return_daily = np.mean(excess_returns_daily)

# 步骤3 & 4: 计算年化标准差
std_dev_daily = np.std(returns)
# 年化标准差 = 日标准差 * sqrt(交易天数)
std_dev_annual = std_dev_daily * np.sqrt(250)

# 步骤5: 计算年化夏普比率
# 年化超额收益 = 平均日超额收益 * 交易天数
sharpe_ratio = (avg_excess_return_daily * 250) / std_dev_annual

print(f"夏普比率为: {sharpe_ratio:.4f}")
```

运行以上代码，我们可以得到一个具体的夏普比率数值。例如，对于不同的股票或投资组合策略，我们可以分别计算它们的夏普比率。

## 结果解读与应用

![](img/b962a9d351049a1873e7a3435d50c219_11.png)

计算完成后，我们会得到每个投资标的的夏普比率。数值越高越好。

![](img/b962a9d351049a1873e7a3435d50c219_13.png)

*   **正夏普比率**：表明收益超过了无风险收益，数值越大越好。
*   **负夏普比率**：表明收益还不如无风险国债，通常不予考虑。
*   **比较**：在多个投资选项中，应优先选择夏普比率更高的那个，因为它意味着在同等风险下获得了更高的回报，或者说在获得同等回报时承担了更低的风险。

![](img/b962a9d351049a1873e7a3435d50c219_15.png)

在量化策略回测中，夏普比率是评估策略性能是否优异的关键指标之一。
