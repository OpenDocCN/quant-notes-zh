# 策略之光第17期：机器学习量化交易的艺术：目标函数的设计

在本节课中，我们将学习机器学习在量化交易中一个至关重要但常被忽视的环节：目标函数（标签）的设计。我们将探讨如何根据交易场景的特殊性，定义有效的预测目标，并比较监督学习与强化学习在此应用上的异同。

## 1. 预测目标定义的理解与问题背景

上一节我们介绍了课程概述，本节中我们来看看目标设计的核心背景。在量化交易中，机器学习的目标定义与通用AI场景有显著不同。通常，大家更关注算法和超参数调优，但特征工程和目标定义同样关键，它们直接决定了策略的有效性。

![](img/d16aeb3a00c977d78878b7463ea7eaec_5.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_6.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_8.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_9.png)

在交易场景中，良好的目标定义旨在追求收益最大化，并在一定风险控制条件下实现。这需要我们结合时间和空间两个维度进行考量。

![](img/d16aeb3a00c977d78878b7463ea7eaec_11.png)

*   **时间维度**：预测未来特定时间段（如一天、一小时）的涨跌。时间长度决定了收益累积的周期。
*   **空间维度**：预测价格未来可能达到的幅度（如涨跌百分比）。空间大小决定了单次交易的潜在收益。

![](img/d16aeb3a00c977d78878b7463ea7eaec_13.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_15.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_17.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_19.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_21.png)

单纯预测涨跌不足以形成完整策略，还需结合入场、出场、止盈止损等执行逻辑。因此，设计目标时需将时空结合，并考虑多空对称性（尤其在可做空的市场中），以构建更稳健的策略基础。

![](img/d16aeb3a00c977d78878b7463ea7eaec_23.png)

## 2. 监督学习中的标签生成方式

![](img/d16aeb3a00c977d78878b7463ea7eaec_25.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_27.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_29.png)

上一节我们讨论了目标设计的核心思想，本节中我们来看看具体的标签生成类型。在监督学习框架下，预测目标即是我们需要生成的标签。主要有以下几种方式：

![](img/d16aeb3a00c977d78878b7463ea7eaec_31.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_33.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_35.png)

以下是几种常见的标签定义类型：

![](img/d16aeb3a00c977d78878b7463ea7eaec_37.png)

1.  **回归类目标**：预测未来一段时间内的具体涨跌幅数值。公式可表示为：`标签 = 未来N周期收益率`。这对应连续的数值分布。
2.  **分类类目标**：将未来走势简化为离散类别，如上涨/下跌两类。这强化了不同状态的信息区分度。
3.  **基于价格区间**：预测未来价格是否达到某个特定区间，结合了时空概念。
4.  **基于技术指标**：以技术指标（如RSI、MACD）的未来值或变化作为预测目标。
5.  **基于金融事件**：将突发的金融新闻或事件作为标签生成的依据，但这类数据非结构化，应用难度较大。

![](img/d16aeb3a00c977d78878b7463ea7eaec_39.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_41.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_43.png)

## 3. 机器学习在标签预测中的应用流程

![](img/d16aeb3a00c977d78878b7463ea7eaec_45.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_47.png)

上一节我们介绍了标签的类型，本节中我们来看看如何应用机器学习进行预测。标准的监督学习应用流程包含以下步骤：

以下是机器学习应用于标签预测的标准流程：

![](img/d16aeb3a00c977d78878b7463ea7eaec_49.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_51.png)

1.  **数据准备**：收集和清洗市场数据。
2.  **特征工程**：构建和筛选有效的预测因子（特征）。这是与目标设计并列的关键环节。
3.  **模型训练**：使用标注好的数据（特征+标签）训练机器学习模型。
4.  **预测生成**：利用训练好的模型对新的特征数据进行预测，得到未来的标签（涨跌、幅度等）。

## 4. 应对数据不平衡与多样性

![](img/d16aeb3a00c977d78878b7463ea7eaec_53.png)

上一节我们了解了标准流程，本节中我们来看看实践中常遇到的挑战：数据问题。根据目标设计的不同（如对称多空 vs. 非对称止盈止损），数据可能呈现不平衡或多样性问题，需要特别处理。

![](img/d16aeb3a00c977d78878b7463ea7eaec_55.png)

以下是处理数据不平衡的常用技术：

*   **欠采样**：减少多数类样本的数量，使其与少数类对齐。
*   **过采样**：增加少数类样本的数量，例如使用SMOTE算法生成合成样本。

![](img/d16aeb3a00c977d78878b7463ea7eaec_57.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_59.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_61.png)

对于数据多样性（如回归问题中数值分布广泛），除了使用回归模型直接拟合，也可考虑以下方法：
*   **阈值调整**：在模型输出后，根据样本在不同时间段的表现动态调整分类阈值（如不固定使用0.5作为涨跌分界线），以优化样本外表现。
*   **日内标准化**：关注每日内标的物的相对排名顺序，而非绝对涨跌幅值。

## 5. 监督学习与强化学习的融合

![](img/d16aeb3a00c977d78878b7463ea7eaec_63.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_65.png)

![](img/d16aeb3a00c977d78878b7463ea7eaec_67.png)

前面我们主要讨论了监督学习，本节中我们来看看另一种重要范式：强化学习。在量化交易中，强化学习与监督学习有深刻关联。

![](img/d16aeb3a00c977d78878b7463ea7eaec_69.png)

监督学习（如预测涨跌）只是完整交易链条中的一环。预测结果需要加上“策略执行逻辑”（如何开平仓、止盈止损）才能构成完整策略。而强化学习则将**交易动作**直接作为学习目标，其目标是寻找最优的动作序列以最大化累积收益（同时控制风险）。

可以这样理解：**监督学习是强化学习中的一个子模块**。强化学习通过智能体与环境的不断交互（试错），利用奖励和惩罚机制来优化决策。其中，状态空间类似于监督学习中的特征集合，动作空间则对应交易操作。

深度强化学习是将深度学习模型（如CNN、LSTM）嵌入到强化学习框架中，以处理复杂的状态信息。当前，强化学习是量化交易领域一个重要的研究方向。

## 6. 案例分享与模型选择参考

最后，我们通过一些案例来加深理解，并了解前沿的模型选择。

*   **多空对称性案例**：设计使多空收益曲线互补的目标，可以在不同市场环境下获得更平滑的综合收益曲线，降低单一方向的风险。
*   **数据不平衡处理案例**：对于止盈止损策略产生的非对称数据分布，可以使用过采样（如SMOTE）等技术来改善模型训练。

在模型选择上，当前时间序列预测的公开测试显示了一些有效模型：
*   **长周期预测**：Transformer的变体（如iTransformer）表现突出。
*   **短周期预测**：TimesNet等基于频率分析的模型效果较好。

这些模型可以作为技术选型的参考。
