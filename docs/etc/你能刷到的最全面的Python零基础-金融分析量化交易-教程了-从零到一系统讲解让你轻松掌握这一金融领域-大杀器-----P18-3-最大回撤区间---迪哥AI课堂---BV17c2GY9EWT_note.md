在本节课中，我们将要学习金融量化分析中的一个重要风险指标——**最大回撤**。最大回撤用于衡量投资策略在特定时间段内可能面临的最糟糕情况，即资产净值从峰值到谷底的最大跌幅。理解并计算最大回撤对于评估策略的风险至关重要。

## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何量化策略的风险。最大回撤描述的并非策略的盈利能力，而是其风险程度。它衡量的是在投资过程中，资产价值从历史最高点到之后最低点的最大百分比跌幅。

用通俗的话说，它描述的是你炒股过程中“最惨”的那段经历。例如，你的股票可能从15元跌到1元，这段持续下跌的区间及其缩水比例，就是最大回撤所关注的。

**核心公式**如下：

```
最大回撤 = max( (Pi - Pj) / Pi )
```
其中：
*   `Pi` 代表第 `i` 天的资产价格（峰值）。
*   `Pj` 代表第 `i` 天之后第 `j` 天的资产价格（谷底）。
*   我们需要遍历所有可能的 `i` 和 `j`（`j > i`），找出比值最大的那个，即最大回撤。

![](img/9ff48d6dda82921302fab638c2748b64_5.png)

这个指标之所以重要，是因为即使一个策略长期盈利，但过程中如果出现巨大的回撤，投资者也可能因为无法承受短期的巨额亏损而被迫止损退出。历史上，如2008年股市从6000多点暴跌至1600多点，就是一次巨大的回撤，给许多投资者带来了沉重打击。

## 如何计算最大回撤？

![](img/9ff48d6dda82921302fab638c2748b64_13.png)

理解了概念后，我们来看看如何用Python代码进行计算。虽然后续我们会使用现成的库函数，但了解其计算原理很有帮助。

![](img/9ff48d6dda82921302fab638c2748b64_15.png)

![](img/9ff48d6dda82921302fab638c2748b64_17.png)

计算最大回撤的核心思路是：遍历所有时间点，找出“前期最高点”与“当前点”落差最大的那个比值。

![](img/9ff48d6dda82921302fab638c2748b64_19.png)

以下是计算步骤的分解：

![](img/9ff48d6dda82921302fab638c2748b64_21.png)

![](img/9ff48d6dda82921302fab638c2748b64_23.png)

1.  **计算累积最大值**：首先，我们需要得到每个时间点之前的资产最高值。这可以通过Pandas的 `cummax()` 函数实现。
2.  **计算回撤值**：然后，用每个时间点之前的最高值减去当前的实际资产值，得到潜在的下跌幅度。
3.  **计算回撤比率**：将上述下跌幅度除以前期的最高值（即 `Pi`），得到回撤比率。
4.  **找出最大值**：在所有时间点的回撤比率中，找出最大值，这就是最大回撤。

![](img/9ff48d6dda82921302fab638c2748b64_25.png)

![](img/9ff48d6dda82921302fab638c2748b64_27.png)

以下是使用Pandas进行演示的代码片段：

![](img/9ff48d6dda82921302fab638c2748b64_29.png)

![](img/9ff48d6dda82921302fab638c2748b64_31.png)

![](img/9ff48d6dda82921302fab638c2748b64_33.png)

```python
import pandas as pd

![](img/9ff48d6dda82921302fab638c2748b64_35.png)

![](img/9ff48d6dda82921302fab638c2748b64_37.png)

# 假设 df[‘net_value’] 是代表资产净值的时间序列数据
# 1. 计算累积最大值
cumulative_max = df[‘net_value’].cummax()

![](img/9ff48d6dda82921302fab638c2748b64_39.png)

# 2. 计算每日的回撤值（前期最高 - 当前值）
drawdown_value = cumulative_max - df[‘net_value’]

![](img/9ff48d6dda82921302fab638c2748b64_41.png)

![](img/9ff48d6dda82921302fab638c2748b64_43.png)

# 3. 计算每日的回撤比率
drawdown_ratio = drawdown_value / cumulative_max

![](img/9ff48d6dda82921302fab638c2748b64_45.png)

# 4. 找出最大回撤比率
max_drawdown = drawdown_ratio.max()
print(f”最大回撤为：{max_drawdown:.2%}”)
```

![](img/9ff48d6dda82921302fab638c2748b64_47.png)

通过这段代码，我们可以计算出策略在整个回测期间内经历的最大回撤比率。在实际的策略回测图表中，这个最大回撤区间通常会用阴影高亮标出，直观地展示策略最脆弱的阶段。
