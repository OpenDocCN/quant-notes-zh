## 布尔型索引的原理

![](img/8d759a6264b0352b33f56d690371265a_10.png)

![](img/8d759a6264b0352b33f56d690371265a_12.png)

布尔型索引的原理分为两步：
1.  首先，通过数组与标量的运算，得到一个与原数组形状相同的布尔数组。
2.  然后，将这个布尔数组作为索引传入原数组，即可取出所有对应位置为`True`的元素。

![](img/8d759a6264b0352b33f56d690371265a_14.png)

![](img/8d759a6264b0352b33f56d690371265a_16.png)

以下是其核心公式：
```python
# 筛选出数组arr中所有大于5的元素
result = arr[arr > 5]
```

![](img/8d759a6264b0352b33f56d690371265a_18.png)

## 基础用法示例

![](img/8d759a6264b0352b33f56d690371265a_20.png)

![](img/8d759a6264b0352b33f56d690371265a_22.png)

让我们通过一个例子来理解。首先创建一个数组。

```python
import numpy as np

![](img/8d759a6264b0352b33f56d690371265a_24.png)

# 创建一个示例数组
a = np.array([8, 3, 6, 2, 9, 1])
print(‘原始数组 a：‘, a)
```

![](img/8d759a6264b0352b33f56d690371265a_26.png)

在Python基础中，我们可能会使用`filter`函数来筛选。以下是使用`filter`的方法：

![](img/8d759a6264b0352b33f56d690371265a_28.png)

```python
# 使用filter和lambda函数筛选大于5的数
result_filter = list(filter(lambda x: x > 5, a))
print(‘使用filter筛选结果：‘, result_filter)
```

![](img/8d759a6264b0352b33f56d690371265a_30.png)

![](img/8d759a6264b0352b33f56d690371265a_31.png)

![](img/8d759a6264b0352b33f56d690371265a_33.png)

然而，NumPy的布尔型索引更加简洁高效：

![](img/8d759a6264b0352b33f56d690371265a_37.png)

```python
# 使用NumPy布尔型索引筛选大于5的数

```

可以看到，`a[a > 5]`一行代码就完成了筛选，其原理正是我们之前提到的两步。

## 理解底层机制

![](img/8d759a6264b0352b33f56d690371265a_43.png)

![](img/8d759a6264b0352b33f56d690371265a_45.png)

为了更清晰地理解，我们可以手动分解这个过程：

![](img/8d759a6264b0352b33f56d690371265a_47.png)

```python
# 第一步：数组与标量运算，得到布尔数组
bool_mask = a > 5
print(‘布尔数组 (a > 5)：‘, bool_mask)

![](img/8d759a6264b0352b33f56d690371265a_49.png)

# 第二步：将布尔数组作为索引传入原数组
result = a[bool_mask]
print(‘使用布尔数组索引的结果：‘, result)
```
当我们将一个布尔数组`[True, False, True, False, True, False]`传入`a[]`时，NumPy会返回所有`True`位置对应的值，即`8, 6, 9`。

![](img/8d759a6264b0352b33f56d690371265a_51.png)

## 组合条件筛选

![](img/8d759a6264b0352b33f56d690371265a_53.png)

![](img/8d759a6264b0352b33f56d690371265a_55.png)

![](img/8d759a6264b0352b33f56d690371265a_56.png)

布尔型索引的强大之处在于可以方便地组合多个条件。

![](img/8d759a6264b0352b33f56d690371265a_58.png)

**1. 筛选“与”关系（同时满足多个条件）**

![](img/8d759a6264b0352b33f56d690371265a_60.png)

例如，筛选出数组中所有**大于5的偶数**。这需要满足两个条件：`(元素 > 5) 与 (元素 % 2 == 0)`。

![](img/8d759a6264b0352b33f56d690371265a_62.png)

![](img/8d759a6264b0352b33f56d690371265a_64.png)

在NumPy中，我们不能使用Python的关键字`and`，而需要使用位运算符`&`，并且每个条件需要用括号括起来。

```python
# 筛选大于5的偶数
result_and = a[(a > 5) & (a % 2 == 0)]
print(‘大于5的偶数：‘, result_and)
```

**2. 筛选“或”关系（满足任一条件）**

例如，筛选出数组中**大于5的数或所有的偶数**。这需要满足条件：`(元素 > 5) 或 (元素 % 2 == 0)`。

![](img/8d759a6264b0352b33f56d690371265a_66.png)

这里我们使用位运算符`|`来表示“或”。

```python
# 筛选大于5的数或偶数
result_or = a[(a > 5) | (a % 2 == 0)]
print(‘大于5的数或偶数：‘, result_or)
```

![](img/8d759a6264b0352b33f56d690371265a_68.png)

**重要提示**：组合条件时务必使用`&`（与）、`|`（或）、`~`（非）这些位运算符，而不是`and`、`or`、`not`。同时，每个独立的条件必须用括号`()`包裹，以确保运算顺序正确。

![](img/8d759a6264b0352b33f56d690371265a_70.png)

本节课中我们一起学习了NumPy的布尔型索引。我们掌握了其两步工作原理：先通过条件运算生成布尔掩码，再利用该掩码提取数据。我们重点练习了如何使用简洁的语法`array[condition]`进行单条件筛选，以及如何使用`&`和`|`运算符结合括号`()`进行多条件组合筛选。这个功能在数据清洗和筛选场景下极其高效，是后续进行金融量化分析的重要工具。
