## 计算移动平均线

上一节我们完成了数据清洗，本节中我们来看看如何计算关键的短期和长期移动平均线。

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_36.png)

移动平均线是技术分析中常用的指标，用于平滑价格数据并识别趋势。我们需要定义短期和长期的窗口大小。

```python
# 定义短期和长期窗口大小
short_window = 40
long_window = 100
```

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_38.png)

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_40.png)

以下是计算移动平均线的步骤：

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_42.png)

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_44.png)

1.  使用 `rolling` 方法创建指定大小的数据窗口。
2.  对每个窗口内的价格数据计算平均值。
3.  将计算结果作为新列添加到数据框中。

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_46.png)

```python
# 计算短期移动平均线 (SMA)
apple_data['SMA_short'] = apple_data['AAPL'].rolling(window=short_window).mean()

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_48.png)

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_50.png)

# 计算长期移动平均线 (SMA)
apple_data['SMA_long'] = apple_data['AAPL'].rolling(window=long_window).mean()

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_52.png)

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_54.png)

# 查看数据尾部，确认计算成功
print(apple_data.tail())
```

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_56.png)

## 标记买卖信号位置

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_74.png)

为了进行更量化的分析，我们需要在数据中标记出这些信号点。首先，需要去除计算均线时产生的缺失值。

```python
# 去除因计算移动平均线而产生的缺失值
apple_data.dropna(inplace=True)
```

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_76.png)

接下来，我们创建一个标志位 `Position`。该标志位表示短期均线是否高于长期均线（1为是，0为否）。这个标志位的变化点（从0变1或从1变0）就对应着潜在的买卖点。

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_78.png)

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_80.png)

```python
# 创建标志位：短期均线 > 长期均线 则为1，否则为0
apple_data['Position'] = np.where(apple_data['SMA_short'] > apple_data['SMA_long'], 1, 0)
print(apple_data.head())
```

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_82.png)

我们也可以将这个标志位绘制在图表上，以便更清晰地观察。

```python
# 绘制股价与买卖信号标志位
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 10), sharex=True)

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_84.png)
```

## 策略思路与总结

本节课中我们一起学习了如何通过双移动平均线策略进行买点与卖点的可视化分析。

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_90.png)

通过对比短期和长期均线的位置关系，我们可以制定一个简单的交易策略：
*   当 `Position` 标志位从 **0 变为 1**（黄金交叉）时，考虑**买入**。
*   当 `Position` 标志位从 **1 变为 0**（死亡交叉）时，考虑**卖出**。

![](img/34fac8a8b3d4dbaad943a01d99fc0d48_92.png)

这种策略的核心思想是“跟随趋势”。理论上，通过在这种交叉点进行交易，投资者可以避免在长期下跌趋势中持有股票，并在上升趋势开始时入场，从而可能获得比单纯买入并持有（Buy and Hold）策略更好的风险调整后收益。在下一节课中，我们可以基于这个信号列，进一步计算策略的模拟收益，并与基准收益进行对比。
