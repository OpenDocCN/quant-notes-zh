## 夏普比率的定义与意义
上一节我们介绍了投资回报率，本节中我们来看看如何衡量风险与收益的关系。夏普比率描述的是：在承担单位风险的情况下，所能获得的超额收益是多少。

一个生动的类比是：高薪的叙利亚雇佣兵工作，其高报酬对应的是极高的生命风险。在投资中，我们同样需要评估，为了获得一定的收益，我们所承担的风险是否“值得”。夏普比率就是这样一个衡量“性价比”的指标。

![](img/85a6f48d22e89c98d051fda8c626742d_1.png)

**公式**：`夏普比率 = (投资组合收益率 - 无风险收益率) / 投资组合收益率的标准差`

![](img/85a6f48d22e89c98d051fda8c626742d_3.png)

指标值越高，意味着**单位风险**带来的收益越高，该投资组合的绩效也就越好。在多个投资选择中，我们通常会倾向于夏普比率更高的那个。

![](img/85a6f48d22e89c98d051fda8c626742d_5.png)

## 夏普比率的计算逻辑
理解了夏普比率的意义后，我们来看看它的计算逻辑。计算主要涉及三个部分：

1.  **投资组合收益率**：你的投资（或单只股票）在一段时间内的回报率。
2.  **无风险收益率**：通常指没有任何违约风险的资产的收益率，例如国债、银行存款利率。这是一个基准。
3.  **投资组合收益率的标准差**：用于衡量投资组合收益的波动性，即风险。波动越大，风险越高。

计算步骤是：首先用投资组合收益率减去无风险收益率，得到超额收益；然后将这个超额收益除以收益的标准差，最终得到夏普比率。

## 使用Python计算夏普比率
理论部分我们已经清晰，现在进入实践环节，看看如何用Python代码计算多只股票的夏普比率。

在计算前，我们需要准备好股票的历史收益率数据，并处理可能存在的缺失值。常见的做法是用前一天的收益率填充缺失值。

以下是计算夏普比率的核心代码步骤：

![](img/85a6f48d22e89c98d051fda8c626742d_7.png)

```python
import numpy as np
import pandas as pd

![](img/85a6f48d22e89c98d051fda8c626742d_9.png)

# 假设 returns 是一个Pandas DataFrame，每一列代表一只股票的历史日收益率序列
# 假设无风险年化收益率为 5%
risk_free_rate = 0.05
# 年化因子：假设一年有250个交易日
trading_days_per_year = 250

# 计算超额收益（日度）
excess_returns = returns - risk_free_rate / trading_days_per_year

# 计算夏普比率（年化）
# 公式： (超额收益的均值 * 年化因子) / (超额收益的标准差 * sqrt(年化因子))
sharpe_ratios = (excess_returns.mean() * trading_days_per_year) / (excess_returns.std() * np.sqrt(trading_days_per_year))

print(sharpe_ratios)
```

![](img/85a6f48d22e89c98d051fda8c626742d_11.png)

运行以上代码后，我们会得到一个夏普比率的序列。结果分析非常简单：

![](img/85a6f48d22e89c98d051fda8c626742d_13.png)

*   **数值越大越好**：表示单位风险带来的收益越高。
*   **正值为佳**：正值表示收益跑赢了无风险利率。
*   **负值需谨慎**：负值意味着收益尚不及无风险利率，投资可能不划算。

![](img/85a6f48d22e89c98d051fda8c626742d_15.png)

在我们的示例中，“中兴”股票的夏普比率最高，表明在这组股票里，其风险收益的“性价比”最优。
