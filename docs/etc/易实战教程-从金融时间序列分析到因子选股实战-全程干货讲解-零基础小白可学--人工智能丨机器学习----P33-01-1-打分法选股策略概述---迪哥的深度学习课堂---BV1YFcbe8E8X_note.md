# Python金融分析与量化交易实战教程：P33：01-1-打分法选股策略概述 📊

## 概述
在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分选股。我们将从策略的基本概念入手，解释其核心思想，并了解实施该策略前需要具备的先验知识。

## 策略核心思想
上一节我们介绍了因子分析的基本概念，本节中我们来看看如何将多个因子综合起来进行选股。

因子打分选股的核心思想是：不再单一地依赖某个因子来判断股票的好坏，而是综合多个因子，为每只股票计算一个“总分”，然后根据总分进行排名和筛选。

这个过程类似于学生的综合成绩评定。假设我们有300只股票（学生），每只股票有A、B、C、D四个因子指标（科目）。我们需要为每个因子打分，然后计算一个总分。最后，我们根据总分对300只股票进行排名，选取排名靠前的股票（例如前10名）作为投资组合。在下一次调仓时，我们重复这个过程。

我们的最终目标就是计算出这个“总分”。

## 实施策略的先决条件
在开始实施打分策略之前，我们需要明确一些关键的已知条件或先验知识。

![](img/41ef167a4f7cfed654e456c51c87d3f8_1.png)

以下是实施因子打分法必须预先了解的核心信息：

1.  **因子的方向性**：我们必须知道每个因子与股票未来收益率的相关性是正向还是负向的。换句话说，我们需要判断对于某个因子，是数值越大越好，还是数值越小越好。
    *   **正向因子**：因子值越大，预期收益越高。例如：`每股收益(EPS)`、`净资产收益率(ROE)`。
    *   **负向因子**：因子值越小，预期收益越高。例如：`市盈率(PE)`、`市净率(PB)`（通常认为估值越低越好）。

2.  **方向性的获取途径**：如何确定因子的方向性？
    *   **研究报告**：可以参考各大券商发布的研究报告，其中通常会总结因子的有效性和方向。
    *   **自行计算**：我们之前在因子分析章节中学习的方法，例如计算**IC值**（信息系数），可以帮助我们验证因子与收益的相关性方向。通过历史数据回测，我们可以统计出因子值与未来收益率的相关系数。

这个关于因子方向性的知识，是我们进行有效打分的基础。在后续的实际编程中，我们所选择的因子都将是方向性已知的。

## 总结
本节课中我们一起学习了因子打分选股策略的概述。我们了解到，该策略的核心在于**综合多个因子为股票计算总分并排序**，而非依赖单一因子。同时，我们明确了实施该策略的一个关键前提：**必须预先知道每个因子与收益率的相关方向**（越大越好或越小越好），这是后续进行正确打分的基础。在接下来的课程中，我们将进入实战环节，学习如何在代码中实现这一策略。
