## 布林带量化策略介绍

首先，我们来整体介绍一下布林量化策略。这个策略是基于技术分析中非常重要的一个工具——布林带而设计并实现的。

### 布林指标回顾

布林指标由四条线组成，分别是上轨线、中轨线、下轨线和价格线。其中，上轨线和下轨线又分别称为压力线和支撑线。

![](img/c626f00ddf07346ddc251845e936459f_1.png)

以下是布林带核心线的计算公式：

*   **中轨线（MID）**：N日移动平均线（MA）。
    `MID = MA(CLOSE, N)`
*   **上轨线（UP）**：中轨线加上N倍的标准差。
    `UP = MID + K * STD(CLOSE, N)`
*   **下轨线（DOWN）**：中轨线减去N倍的标准差。
    `DOWN = MID - K * STD(CLOSE, N)`

> 注意：在技术分析实战中，无论是投资者还是量化策略开发者，通常只需了解布林指标的原理和运用方法，无需过度深究其算法细节。

### 策略原理与买卖信号

接下来，我们介绍一下布林量化策略的策略原理和买卖信号是如何识别的。

首先是策略原理。当股价穿越最外面的压力线或支撑线时，表示卖点或买点出现。具体来说：
*   如果股价向上穿越压力线，则是**卖点**出现。
*   如果股价向下穿越支撑线，则是**买点**出现。

布林带还有另一种运用场景：股价沿着压力线或支撑线运行，虽未穿越，但回头突破了中轨线，这也可能构成买卖点。但在常见的量化策略中，我们通常优先实现第一种穿越信号，因为其实现简单且胜率较高。

以下是本策略的买卖信号定义：
*   **买入信号**：当收盘价向下穿越布林带下轨线（支撑线）时。
*   **卖出信号**：当收盘价向上穿越布林带上轨线（压力线）时。

### 2. 盘中策略逻辑实现

接下来，我们在每个交易日运行的主函数中实现布林带计算和交易逻辑。

```python
def handle_data(context, data):
    # 获取股票代码
    stock = g.security
    # 获取最近20日的收盘价
    prices = attribute_history(stock, 20, '1d', ['close'])
    # 计算20日移动平均线（中轨）
    ma = prices['close'].mean()
    # 计算20日收盘价的标准差
    std = prices['close'].std()
    
    # 计算布林带上轨（压力线）和下轨（支撑线）
    up = ma + g.N * std  # 压力线
    down = ma - g.N * std # 支撑线
    
    # 获取当前开盘价和账户可用现金
    current_price = data[stock].open
    cash = context.portfolio.available_cash
    
    # 判断交易信号并执行
    # 买入信号：当前价格跌破支撑线，且当前没有持仓
    if current_price < down and stock not in context.portfolio.positions:
        # 计算可买数量（全仓买入）
        amount = int(cash / current_price)
        if amount > 0:
            order(stock, amount)
            log.info(f"买入 {stock}, 价格{current_price}, 数量{amount}")
    
    # 卖出信号：当前价格突破压力线，且当前持有该股票
    elif current_price > up and stock in context.portfolio.positions:
        # 获取当前持仓数量，全部卖出
        amount = context.portfolio.positions[stock].amount
        if amount > 0:
            order_target(stock, 0)
            log.info(f"卖出 {stock}, 价格{current_price}, 数量{amount}")
```
