### 函数的定义与调用

![](img/ce265b1d5304783a9df94ca3e51f2466_8.png)

上一节我们介绍了代码的基本结构，本节中我们来看看如何将代码封装成可重复使用的函数。

![](img/ce265b1d5304783a9df94ca3e51f2466_10.png)

函数通过`def`关键字定义，其基本结构包含函数名、参数和函数体。定义函数后，可以通过函数名加括号的方式进行调用。

以下是定义一个简单函数的语法：

```python
def 函数名(参数):
    # 函数体：处理数据的代码段
    return 返回值
```

例如，我们定义一个进行数学运算的函数：

```python
def ds(x):
    a = 2 * x + 1
    return a

![](img/ce265b1d5304783a9df94ca3e51f2466_12.png)

# 调用函数
result = ds(5)
print(result)  # 输出：11
```

![](img/ce265b1d5304783a9df94ca3e51f2466_14.png)

调用函数时，程序会跳转到函数内部执行代码，并通过`return`语句将结果返回给调用者。我们可以使用调试（Debug）模式设置断点，来观察程序执行流程和变量的变化。

### 函数参数详解

我们已经学会了如何定义和调用带有参数的函数。接下来，我们深入了解一下函数参数的不同形式。

![](img/ce265b1d5304783a9df94ca3e51f2466_16.png)

![](img/ce265b1d5304783a9df94ca3e51f2466_18.png)

#### 默认参数

在定义函数时，可以为参数指定默认值。这样在调用函数时，如果没有提供该参数的值，函数将使用默认值。

```python
def ds(x=1):  # 设置参数x的默认值为1
    a = 2 * x + 1
    return a

![](img/ce265b1d5304783a9df94ca3e51f2466_20.png)

print(ds())    # 输出：3 (使用默认值x=1)
print(ds(8))   # 输出：17 (使用传入的值x=8)
```

![](img/ce265b1d5304783a9df94ca3e51f2466_22.png)

#### 可变参数

有时我们无法预知函数会接收多少个参数。这时可以使用可变参数，在参数前加一个星号`*`，这样传入的所有参数会被打包成一个元组（tuple）。

```python
def count_args(*args):
    num = len(args)
    print(f"输入了{num}个参数")
    return 0

result = count_args(1, 2, 3, 4, 5, 6, 7, 8)
# 输出：输入了8个参数
```

在函数内部，`args`变量就是一个包含所有传入参数的元组。

### 变量的作用域

在讨论了参数传递之后，我们需要理解函数内部变量的可见范围，即作用域。

![](img/ce265b1d5304783a9df94ca3e51f2466_24.png)

![](img/ce265b1d5304783a9df94ca3e51f2466_26.png)

函数内部定义的变量（局部变量）其作用域仅限于函数内部。在函数外部无法访问它们。同样，在函数内部直接修改函数外部的变量（全局变量）也是无效的，除非使用`global`关键字声明。

```python
a = 10000  # 全局变量

def fun3():
    a = 0.1  # 局部变量，与全局变量a无关
    b = 50   # 局部变量

fun3()
print(a)  # 输出：10000，全局变量a未被修改
# print(b) # 这行会报错：NameError，因为b是局部变量
```

这个例子清晰地展示了函数作用域的边界。

### Lambda表达式

对于简单的函数，每次都使用`def`来定义可能会显得繁琐。Python提供了lambda表达式，允许我们快速创建匿名函数。

![](img/ce265b1d5304783a9df94ca3e51f2466_34.png)

lambda表达式是一种单行函数定义方式，语法为：`lambda 参数: 表达式`。它通常用于需要一个简单函数作为参数的场合。

```python
# 使用def定义函数
def ds_def(x):
    return 2 * x + 1

# 使用lambda表达式定义相同功能的函数
ds_lambda = lambda x: 2 * x + 1

print(ds_def(5))   # 输出：11
print(ds_lambda(5)) # 输出：11
```

![](img/ce265b1d5304783a9df94ca3e51f2466_36.png)

lambda表达式适用于功能简单、无需复用的临时函数场景。
