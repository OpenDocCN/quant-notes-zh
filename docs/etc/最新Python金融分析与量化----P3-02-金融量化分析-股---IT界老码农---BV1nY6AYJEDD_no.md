## 股票的定义与作用

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

形象地解释，假设一位创业者需要资金，而投资者（如亿万富翁）愿意提供资金。投资者将钱给创业者，创业者则向投资者发行公司的股票作为凭证。例如，一个公司初始市值为5亿，由五位出资人（包括创始人）各出资1亿建立，那么每位出资人将获得该公司20%的股票。

股票的核心作用主要有两点。

第一点是证明股东身份与权利。持有股票意味着你是公司的出资人，是公司的老板之一。作为股东，你拥有在股东大会投票等权利。

第二点是获取收益。这是普通股民最关心的作用。获利方式主要有两种。

以下是两种主要的获利方式：
1.  **公司分红**：公司盈利后，将部分利润按持股比例分配给股东。例如，公司年净利润5000万，持有20%股份的股东可获得1000万的分红。
2.  **交易获利（套现）**：股东可以在市场上将股票转让给他人，通过买卖价差获利。例如，初始投资1亿获得20%股份，当公司市值从5亿增长到50亿时，这20%股份的价值就变成了10亿。此时卖出部分股份（如10%），即可套现5亿现金。

对于资金量较小的普通股民，原理相同，只是持股比例和分红金额相对较小。他们需要通过证券交易所来买卖股票。

## 什么是上市？

上一节我们介绍了股票的基本概念，本节中我们来看看公司如何向公众发行股票，即“上市”。

所谓上市，就是企业通过证券交易所，首次公开向社会增发股票以募集资金的过程。

创业者不能随意向公众募集资金，这属于非法集资。公司需要达到一定体量，并向证监会提交申请，经过严格的审核（如审查财务报表、公司前景等），确认公司资质良好、风险可控后，才被允许上市。

![](img/5a79310b036de1131003f6c4d57c0c9e_1.png)

上市后，公司的股票就能在证券交易所挂牌交易，所有股民都可以公开买卖这只股票。这相当于公司获得了在一个由国家监管的大市场上向广大公众募集资金的资格。首次公开募股的行为就称为 **IPO**。

## 股票的分类

了解了上市的概念后，我们来看看股票有哪些常见的分类方式。

### 按业绩分类
根据公司的经营业绩，股票通常分为以下三类：

以下是三种常见的按业绩分类的股票：
*   **蓝筹股**：指资本雄厚、信誉优良的大型公司股票。例如中石油、中石化。名称源于赌场中价值最高的蓝色筹码。
*   **绩优股**：指业绩优良公司的股票。它们可能规模不如蓝筹股庞大，但盈利增长稳定。例如贵州茅台。
*   **ST股**：中文称“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记，以警示投资者该公司存在较高风险。

### 按上市地区分类
根据公司上市交易的地点和所用货币，股票可分为：

以下是几种按上市地区分类的股票：
*   **A股**：在中国大陆（上海、深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。
*   **H股**：在中国香港上市的股票。
*   **N股**：在美国纽约上市的股票。
*   **S股**：在新加坡上市的股票。

### A股市场的特殊规则
中国A股市场有一些特有的交易规则，旨在保护投资者和维护市场稳定，主要包括：

以下是A股市场的两项核心规则：
1.  **涨跌幅限制**：规定股票每日交易价格相对于上一个交易日收盘价的涨跌幅度不得超过10%。即公式为：
    `今日价格 ∈ [昨日收盘价 × 0.9， 昨日收盘价 × 1.1]`
    此规则是为了防止股价短期剧烈波动，给予投资者缓冲空间。
2.  **T+1交割制度**：“T”代表交易当天。该制度规定，当日买入的股票，要到下一个交易日（T+1）才能卖出。代码逻辑上可以理解为：
    ```python
    if 操作 == “买入”:
        可卖出日期 = 交易日期 + 1天 # T+1日后方可卖出
    ```
    此举旨在减少市场中的过度投机行为。

相比之下，港股、美股等市场普遍实行 **T+0** 交易（当日可买卖）且无涨跌幅限制，交易更为灵活，但波动风险也更大。
