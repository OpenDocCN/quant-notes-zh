## 标准数据类型概述

![](img/453dd2768b7b2b7ad681fa118235f004_7.png)

Python内置了多种标准数据类型，用于存储和操作不同种类的数据。这些类型就像量化交易中的不同工具，各有其特定的用途。

![](img/453dd2768b7b2b7ad681fa118235f004_9.png)

以下是Python中主要的标准数据类型：

![](img/453dd2768b7b2b7ad681fa118235f004_11.png)

*   **数字类型**：用于存储数值，如股价、指数点位。
*   **字符串类型**：用于存储文本信息，如股票名称、新闻标题。
*   **列表类型**：用于存储有序的元素集合。
*   **元组类型**：类似于列表，但内容不可更改。
*   **字典类型**：用于存储键值对，如`{‘股票代码’: ‘价格’}`。
*   **集合类型**：用于存储无序且不重复的元素。

![](img/453dd2768b7b2b7ad681fa118235f004_13.png)

## 数字类型详解

![](img/453dd2768b7b2b7ad681fa118235f004_19.png)

![](img/453dd2768b7b2b7ad681fa118235f004_21.png)

数字类型主要用于处理各种数值。在量化交易中，股价、成交量、技术指标值等都属于数字类型。

![](img/453dd2768b7b2b7ad681fa118235f004_23.png)

以下是Python中主要的数字类型：

![](img/453dd2768b7b2b7ad681fa118235f004_25.png)

*   **整数**：表示为`int`类型。例如，沪深300指数的点位`3000`就是一个整数。
    *   **代码示例**：`hs300 = 3000`
*   **浮点数**：表示为`float`类型。例如，创业板指数的点位`2000.45`就是一个浮点数。
    *   **代码示例**：`cyb_index = 2000.45`
*   **布尔值**：表示为`bool`类型。它只有两个值：`True`（真）和`False`（假），常用于条件判断。
    *   **代码示例**：`is_rising = True`
*   **复数**：表示为`complex`类型，由实部和虚部组成，在工程和科学计算中常用。
    *   **代码示例**：`c = 3 + 4j`

## 如何查看和判断类型

![](img/453dd2768b7b2b7ad681fa118235f004_31.png)

在编程时，我们经常需要知道一个变量是什么类型，或者判断它是否属于某种特定类型。Python提供了两个内置函数来完成这些任务。

![](img/453dd2768b7b2b7ad681fa118235f004_33.png)

以下是用于类型操作的两个关键函数：

![](img/453dd2768b7b2b7ad681fa118235f004_35.png)

1.  **`type()`函数**：返回一个对象的类型。
    *   **公式/代码**：`type(对象)`
    *   **作用**：告诉你“这个变量是什么类型”。
2.  **`isinstance()`函数**：判断一个对象是否是指定类型（或其子类）的实例。
    *   **公式/代码**：`isinstance(对象, 类型)`
    *   **作用**：判断“这个变量是否是某个类型”，返回`True`或`False`。

![](img/453dd2768b7b2b7ad681fa118235f004_37.png)

![](img/453dd2768b7b2b7ad681fa118235f004_39.png)

让我们通过一个量化相关的例子来演示它们的用法。假设我们有两个变量，分别代表沪深300指数和创业板指数。

![](img/453dd2768b7b2b7ad681fa118235f004_41.png)

```python
# 定义变量
hs300 = 3000          # 假设沪深300点位为3000点
cyb_index = 2000.45   # 假设创业板指数点位为2000.45点

![](img/453dd2768b7b2b7ad681fa118235f004_43.png)

# 使用type()函数查看类型
print(type(hs300))      # 输出：<class 'int'>
print(type(cyb_index))  # 输出：<class 'float'>

![](img/453dd2768b7b2b7ad681fa118235f004_45.png)

print("-" * 20)  # 分隔线

![](img/453dd2768b7b2b7ad681fa118235f004_47.png)

# 使用isinstance()函数判断类型
print(isinstance(hs300, int))       # 输出：True
print(isinstance(cyb_index, float)) # 输出：True

![](img/453dd2768b7b2b7ad681fa118235f004_49.png)

![](img/453dd2768b7b2b7ad681fa118235f004_51.png)

print(isinstance(hs300, float))     # 输出：False
print(isinstance(cyb_index, int))   # 输出：False
```

**代码解释与输出分析**：
*   前两行`type()`的输出确认了`hs300`是`int`（整数）类型，`cyb_index`是`float`（浮点数）类型。
*   中间两行`isinstance()`的判断是正确的：`hs300`确实是`int`类型，`cyb_index`确实是`float`类型，因此都返回`True`。
*   最后两行`isinstance()`的判断是错误的：`hs300`不是`float`类型，`cyb_index`也不是`int`类型，因此都返回`False`。
