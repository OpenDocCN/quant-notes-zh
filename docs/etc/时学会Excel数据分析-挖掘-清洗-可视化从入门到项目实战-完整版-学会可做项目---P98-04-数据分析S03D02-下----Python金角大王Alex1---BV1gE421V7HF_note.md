## 函数功能详解

上一节我们介绍了数据可视化的进阶工具Power BI，本节中我们来看看Excel中用于数据处理和统计的核心工具——函数。

### 常用数学函数

以下是几个基础的数学函数，用于处理单个或少数几个数值。

*   **`INT(number)`**：将数值向下取整到最接近的整数。例如，`=INT(3.141)` 的结果是 `3`。
*   **`MOD(number, divisor)`**：返回两数相除的余数。例如，`=MOD(10, 3)` 的结果是 `1`。
*   **`ROUND(number, num_digits)`**：将数字四舍五入到指定的小数位数。例如，`=ROUND(3.14159, 2)` 的结果是 `3.14`。
*   **`ABS(number)`**：返回数字的绝对值。例如，`=ABS(-5)` 的结果是 `5`。
*   **`SQRT(number)`**：返回数字的算术平方根。例如，`=SQRT(4)` 的结果是 `2`。
*   **`RAND()`**：返回一个大于等于0且小于1的随机小数。例如，`=RAND()`。
*   **`RANDBETWEEN(bottom, top)`**：返回位于指定两个数之间的一个随机整数。例如，`=RANDBETWEEN(1, 100)`。

### 统计函数

统计函数通常对一组数据进行聚合计算。

*   **`MIN(range)`** / **`MAX(range)`**：返回一组数据中的最小值/最大值。
*   **`AVERAGE(range)`**：返回一组数据的算术平均值。
*   **`SUM(range)`**：返回一组数据的总和。
*   **`COUNT(range)`**：计算包含数字的单元格个数，忽略文本和空值。
*   **`COUNTA(range)`**：计算非空单元格的个数，包括文本和数字。
*   **`COUNTIF(range, criteria)`**：计算满足给定条件的单元格个数。例如，`=COUNTIF(A1:A10, “>20”)`。
*   **`SUMIF(range, criteria, [sum_range])`**：对满足给定条件的单元格求和。例如，`=SUMIF(B1:B10, “A”, C1:C10)` 计算B列为“A”时，对应C列值的总和。
*   **`COUNTIFS(criteria_range1, criteria1, [criteria_range2, criteria2]…)`**：计算满足多个条件的单元格个数。
*   **`SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2]…)`**：对满足多个条件的单元格求和。
*   **`RANK.EQ(number, ref, [order])`**：返回一个数字在数字列表中的排位。使用 `$` 符号（如 `$A$1:$A$10`）或按 `F4` 键锁定引用区域，防止公式下拉时区域变化。

### 逻辑函数

逻辑函数用于构建判断条件，是复杂公式的基础。

*   **`IF(logical_test, value_if_true, value_if_false)`**：根据条件测试的真假返回不同的值。例如，`=IF(A1>60, “及格”, “不及格”)`。
*   **`AND(logical1, [logical2], …)`**：所有参数为真时返回 `TRUE`，否则返回 `FALSE`。
*   **`OR(logical1, [logical2], …)`**：任一参数为真时返回 `TRUE`，否则返回 `FALSE`。
*   **`NOT(logical)`**：对逻辑值取反，`TRUE` 变 `FALSE`，`FALSE` 变 `TRUE`。

### 查找与引用函数

`VLOOKUP` 函数是实现表格间数据关联的关键工具。

*   **`VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])`**：
    *   `lookup_value`：要查找的值。
    *   `table_array`：查找的表格区域，**查找值必须位于该区域的第一列**。
    *   `col_index_num`：返回数据在查找区域的第几列。
    *   `range_lookup`：`FALSE` 为精确匹配，`TRUE` 为近似匹配（要求首列升序排列）。使用 `$` 符号锁定查找区域。

### 文本与日期函数

以下是处理文本和日期数据的常用函数。

*   **`YEAR(serial_number)`** / **`MONTH(serial_number)`** / **`DAY(serial_number)`**：从日期中提取年、月、日。
*   **`TODAY()`** / **`NOW()`**：返回当前日期/当前日期和时间。
*   **`MID(text, start_num, num_chars)`**：从文本字符串的指定位置开始返回特定数量的字符。
*   **`LEFT(text, [num_chars])`** / **`RIGHT(text, [num_chars])`**：从文本左侧/右侧返回指定数量的字符。
*   **`LEN(text)`**：返回文本字符串中的字符个数。
*   **`REPT(text, number_times)`**：将文本重复一定次数。
*   **`REPLACE(old_text, start_num, num_chars, new_text)`**：替换文本中指定位置和长度的字符。
*   **`SUBSTITUTE(text, old_text, new_text, [instance_num])`**：替换文本中指定的旧文本。

## 数据透视表实战

掌握了函数的基础后，我们来看一个更强大的数据聚合工具——数据透视表。它能快速对大量数据进行多维度分组和汇总。

### 创建与基本操作

1.  选中数据区域任意单元格，点击【插入】-【数据透视表】。
2.  确认数据源后，点击确定，即可创建一个空白透视表。
3.  在右侧的“数据透视表字段”窗格中，将字段拖拽到“行”、“列”、“值”区域。
    *   **行/列**：通常放置分类字段（如部门、项目）。
    *   **值**：放置需要计算的数值字段（如金额）。

### 值字段设置

这是透视表的核心，决定了如何计算“值”区域的数据。

*   **值汇总方式**：双击“值”区域字段，或点击其下拉箭头选择“值字段设置”。可以更改计算类型，如求和、计数、平均值、最大值、最小值等。
*   **值显示方式**：在“值字段设置”中，切换到“值显示方式”选项卡。可以计算百分比、差异、累计等，例如选择“列汇总的百分比”可以查看每项占该列总计的百分比。

![](img/5eccb3a766c96828c2f67c712c6a0a96_9.png)

### 布局与切片器

*   **布局调整**：右键点击行标签字段，选择“字段设置”。在“布局和打印”选项卡中，可以将“以大纲形式显示”改为“以表格形式显示”，使分类字段分列展示，结构更清晰。
*   **插入切片器**：选中透视表，点击【数据透视表分析】-【插入切片器】。选择需要的字段（如“部门”），即可生成一个交互式筛选器。点击切片器中的项目，透视表将动态筛选数据。

## 实战案例与作业

### 案例：多条件判断

**需求**：根据学生的数学、计算机、英语三科成绩，判断是否“全部及格”或“至少一科及格”。
**思路**：使用 `IF` 函数嵌套 `AND` 或 `OR` 函数。
**公式示例**：
*   全部及格：`=IF(AND(数学>60, 计算机>60, 英语>60), “全部及格”, “否”)`
*   至少一科及格：`=IF(OR(数学>60, 计算机>60, 英语>60), “是”, “全部不及格”)`

### 案例：条件求和

**需求**：在账目表中，分别计算“借方”和“贷方”的总金额。
**思路**：使用 `SUMIF` 函数。
**公式示例**：
*   借方总额：`=SUMIF(类型列, “借”, 金额列)`
*   贷方总额：`=SUMIF(类型列, “贷”, 金额列)`

### 课后作业

**目标**：从混乱的网页粘贴数据中，提取出整洁的课程大纲信息。
**初始数据**：从课程官网复制的大纲文本，直接粘贴到Excel后，包含多余图标、空行、无关文本等。
**你的任务**：
1.  清理无关内容（如使用 `DELETE` 键删除图片，手动删除首尾无关行）。
2.  **核心挑战**：使用函数（例如 `ISNUMBER` 配合筛选或排序功能）识别并删除所有与课程大纲条目（通常是文本行）无关的数值行或其他干扰行。
3.  整理格式，使大纲清晰易读。
**要求**：过程需使用函数公式实现，确保方案可复用（即使数据量增加也能快速处理）。
