## 概述

在策略分析中，我们不仅需要评估策略的盈利能力，还需要评估其风险。最大回撤就是这样一个风险指标，它描述了在投资过程中，资产价值从峰值到谷底的最大跌幅。

## 什么是最大回撤？

上一节我们介绍了策略分析的基本概念，本节中我们来看看最大回撤的具体含义。

最大回撤描述的是投资过程中“最惨”的阶段。它衡量的是从任意历史高点开始，到之后某个低点为止，资产价值下跌的最大幅度。这个指标反映了投资者在策略执行期间可能承受的最大账面损失。

其核心公式如下：

![](img/de1ec728b7b6613d1b45075c19a3f731_5.png)

**最大回撤 = max( (Pi - Pj) / Pi )**

![](img/de1ec728b7b6613d1b45075c19a3f731_7.png)

其中：
*   **Pi** 代表某个时间点 `i` 的资产价格（峰值）。
*   **Pj** 代表时间点 `i` 之后某个时间点 `j` 的资产价格（谷底）。

这个公式计算的是所有可能时间段内，从高点 `Pi` 到低点 `Pj` 的相对跌幅的最大值。它描述的不是一个绝对值，而是资产“缩水”的比例。

![](img/de1ec728b7b6613d1b45075c19a3f731_9.png)

![](img/de1ec728b7b6613d1b45075c19a3f731_11.png)

## 如何计算最大回撤？

![](img/de1ec728b7b6613d1b45075c19a3f731_13.png)

理解了最大回撤的概念后，我们来看看如何通过编程计算它。我们将使用Python的Pandas库来实现。

![](img/de1ec728b7b6613d1b45075c19a3f731_15.png)

![](img/de1ec728b7b6613d1b45075c19a3f731_17.png)

计算最大回撤的核心思路是：遍历所有可能的时间点组合，找出其中价值下跌最严重的那个区间。以下是计算步骤的分解：

![](img/de1ec728b7b6613d1b45075c19a3f731_19.png)

![](img/de1ec728b7b6613d1b45075c19a3f731_21.png)

以下是计算最大回撤的关键步骤：

![](img/de1ec728b7b6613d1b45075c19a3f731_23.png)

![](img/de1ec728b7b6613d1b45075c19a3f731_25.png)

1.  **计算历史峰值序列**：对于数据集中的每一天，计算截至到该天的历史最高价格。这可以通过Pandas的 `.cummax()` 方法实现。
2.  **计算每日回撤**：用当天的历史峰值减去当天的实际价格，得到每日的潜在回撤值。
3.  **计算每日回撤率**：将每日回撤值除以对应的历史峰值，得到每日的回撤率。
4.  **找出最大值**：在所有每日回撤率中，找出最大值，即为最大回撤。

![](img/de1ec728b7b6613d1b45075c19a3f731_27.png)

![](img/de1ec728b7b6613d1b45075c19a3f731_29.png)

以下是实现该逻辑的示例代码：

![](img/de1ec728b7b6613d1b45075c19a3f731_31.png)

```python
import pandas as pd

![](img/de1ec728b7b6613d1b45075c19a3f731_33.png)

![](img/de1ec728b7b6613d1b45075c19a3f731_35.png)

# 假设 df[‘close’] 是包含每日收盘价的时间序列数据
price_series = df[‘close’]

![](img/de1ec728b7b6613d1b45075c19a3f731_37.png)

# 步骤1: 计算历史峰值序列
historical_max = price_series.cummax()

![](img/de1ec728b7b6613d1b45075c19a3f731_39.png)

# 步骤2 & 3: 计算每日回撤率
daily_drawdown = (historical_max - price_series) / historical_max

![](img/de1ec728b7b6613d1b45075c19a3f731_41.png)

# 步骤4: 找出最大回撤
max_drawdown = daily_drawdown.max()

print(f“最大回撤为：{max_drawdown:.2%}”)
```
