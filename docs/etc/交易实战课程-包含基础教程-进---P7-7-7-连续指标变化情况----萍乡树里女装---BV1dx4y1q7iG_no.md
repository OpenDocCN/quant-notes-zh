在本节课中，我们将要学习如何分析金融时间序列的连续变化情况。我们将探讨如何计算增长率，并引入对数方法来解决连续增长率累加的问题，最终计算出初始投资相对于后续各时间点的价值变化。

## 概述

上一节我们介绍了如何计算时间序列的差异和简单增长率。本节中我们来看看如何处理连续时间序列的增长率累加问题。直接对增长率进行累加无法准确反映从起始点到终点的整体变化，因此我们需要借助数学工具——对数——来进行计算。

## 增长率累加的问题

![](img/3c662c3dc9af239946b674ff315ff189_4.png)

首先，我们通过一个例子来说明直接累加增长率的问题。

假设有一只股票：
*   第一天价格：`100`
*   第二天价格：`50`
*   第三天价格：`75`

计算每日增长率：
*   第二天增长率 = `(50 - 100) / 100 = -0.5`
*   第三天增长率 = `(75 - 50) / 50 = 0.5`

![](img/3c662c3dc9af239946b674ff315ff189_6.png)

![](img/3c662c3dc9af239946b674ff315ff189_7.png)

如果将这两个增长率直接相加：`-0.5 + 0.5 = 0`。这个结果为0，似乎表示股票没有涨跌。然而，观察实际价格，从第一天的`100`到第三天的`75`，股票实际上是下跌的。这说明**增长率不能直接进行累加**来反映整体变化。

![](img/3c662c3dc9af239946b674ff315ff189_9.png)

## 使用对数计算连续变化

为了解决这个问题，我们需要使用对数来计算连续时间序列的累积变化。其核心思想是：对数的加法等价于其内部数值的乘法。

![](img/3c662c3dc9af239946b674ff315ff189_11.png)

我们使用以下公式计算连续的对数收益率：
`log(后一天价格 / 前一天价格)`

![](img/3c662c3dc9af239946b674ff315ff189_13.png)

![](img/3c662c3dc9af239946b674ff315ff189_15.png)

对于上面的例子：
`log(50/100) + log(75/50) = log( (50/100) * (75/50) ) = log(75/100)`

![](img/3c662c3dc9af239946b674ff315ff189_17.png)

![](img/3c662c3dc9af239946b674ff315ff189_18.png)

![](img/3c662c3dc9af239946b674ff315ff189_19.png)

最终结果 `log(75/100)` 直接反映了**第三天价格与第一天价格的比例关系**。通过对数收益率的累加，我们就能得到任意一天相对于起始点的整体变化倍数。

![](img/3c662c3dc9af239946b674ff315ff189_20.png)

![](img/3c662c3dc9af239946b674ff315ff189_21.png)

以下是计算连续对数收益率并还原为实际倍数的步骤：

![](img/3c662c3dc9af239946b674ff315ff189_23.png)

![](img/3c662c3dc9af239946b674ff315ff189_24.png)

![](img/3c662c3dc9af239946b674ff315ff189_25.png)

1.  **计算每日对数收益率**：使用 `shift()` 函数获取前一日数据，然后计算 `log(当日价格 / 前日价格)`。
2.  **计算累积对数收益率**：使用 `cumsum()` 函数对每日对数收益率进行累加。
3.  **还原为实际倍数**：对累积对数收益率取指数 `exp()`，得到从起始点1个单位（如1元）增长到当前时点的实际倍数。

![](img/3c662c3dc9af239946b674ff315ff189_27.png)

以下是关键步骤的代码实现：

![](img/3c662c3dc9af239946b674ff315ff189_28.png)

![](img/3c662c3dc9af239946b674ff315ff189_29.png)

```python
# 1. 计算每日对数收益率
# shift(1) 将数据整体向上移动一格，得到前一日的数据
daily_log_return = np.log(data / data.shift(1))

![](img/3c662c3dc9af239946b674ff315ff189_31.png)

# 2. 计算累积对数收益率
cumulative_log_return = daily_log_return.cumsum()

![](img/3c662c3dc9af239946b674ff315ff189_33.png)

![](img/3c662c3dc9af239946b674ff315ff189_35.png)

# 3. 还原为实际倍数（初始1元相当于当前多少元）
cumulative_return = np.exp(cumulative_log_return)
```

![](img/3c662c3dc9af239946b674ff315ff189_37.png)

执行上述代码后，`cumulative_return` 序列中的每个值就代表了：以起始日（如2010年）的股价为1个单位，到该日期时，这个单位价值相当于多少。例如，2018年的值为10，就意味着2010年投资该股票的1元钱，到2018年价值变成了10元。

![](img/3c662c3dc9af239946b674ff315ff189_38.png)

## 总结

![](img/3c662c3dc9af239946b674ff315ff189_49.png)

本节课中我们一起学习了金融时间序列中连续指标的分析方法。

![](img/3c662c3dc9af239946b674ff315ff189_51.png)

![](img/3c662c3dc9af239946b674ff315ff189_52.png)

![](img/3c662c3dc9af239946b674ff315ff189_53.png)

我们首先指出了直接累加**增长率** (`(P_t - P_{t-1}) / P_{t-1}`) 来评估整体变化是不准确的。接着，我们引入了**对数收益率** (`log(P_t / P_{t-1})`) 的概念，利用对数相加等于内部相乘的性质 (`log(a)+log(b)=log(a*b)`)，完美解决了连续计算的问题。

![](img/3c662c3dc9af239946b674ff315ff189_54.png)

![](img/3c662c3dc9af239946b674ff315ff189_55.png)

核心操作流程总结如下：
1.  使用 `data.shift(1)` 获取滞后一期数据。
2.  使用 `np.log(data / data.shift(1))` 计算每日对数收益率。
3.  使用 `.cumsum()` 计算累积对数收益率。
4.  使用 `np.exp()` 将累积对数收益率还原为实际的累积收益倍数。

![](img/3c662c3dc9af239946b674ff315ff189_58.png)

![](img/3c662c3dc9af239946b674ff315ff189_59.png)

![](img/3c662c3dc9af239946b674ff315ff189_60.png)

通过这套方法，我们可以准确计算出时间序列中任意一点相对于起始点的整体变化情况，这对于量化分析中的收益评估、策略回测等环节至关重要。
