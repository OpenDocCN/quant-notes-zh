## 什么是双均线策略？🤔

首先，我们来简单介绍一下双均线策略的定义。

双均线策略指的是使用两条不同周期的移动平均线（Moving Average, MA）。通常，一条是短周期移动均线，另一条是长周期移动均线。该策略的核心思想是通过判别这两条移动均线的相对位置和交叉关系，来研判买入和卖出的时机。

![](img/83cbf9a12faceb29cc932b688e32f2b7_1.png)

为了更直观地理解，请看下图。我们使用 `matplotlib` 绘制了某只股票的5日均线（MA5）和10日均线（MA10）。

![](img/83cbf9a12faceb29cc932b688e32f2b7_1.png)

图中有两个关键点，即5日均线和10日均线的交叉部分：
1.  在第一个交叉点，5日均线**向上**穿越了10日均线，这被称为 **“金叉”**，是**买入**信号。
2.  在第二个交叉点，5日均线**向下**穿越了10日均线，这被称为 **“死叉”**，是**卖出**信号。

这种判断方式与MACD指标类似。简单总结如下：
*   **金叉买入**：当短期均线（如MA5）向上突破长期均线（如MA10）时。
*   **死叉卖出**：当短期均线（如MA5）向下跌破长期均线（如MA10）时。

以上就是双均线策略的基本介绍。

## 双均线策略的实现思路 💡

上一节我们介绍了双均线策略的原理，本节我们来看看如何用代码实现它。其实现思路与MACD等策略类似，遵循一个清晰的流程。

以下是实现双均线策略的主要步骤：
1.  **导入必要的函数库**：引入回测平台或数据分析所需的库。
2.  **设定初始化函数**：定义策略的全局变量、股票池、基准、回测周期、手续费等参数。
3.  **计算每日数据**：在每天的交易逻辑中，计算现金余额以及短期和长期的移动平均线（MA）。
4.  **识别交易信号**：根据计算出的两条均线，判断是否出现金叉或死叉。
5.  **执行交易**：根据识别出的买卖信号，执行买入或卖出操作。

接下来，我们将进入代码实战环节，把上述思路转化为具体的Python代码。

接着，我们定义每日的交易逻辑函数。在这个函数里，我们将计算均线、判断信号并执行交易。

```python
# 每日交易函数
def trade(context):
    # 获取当前持有的股票
    stock = g.security
    # 计算短期和长期移动平均线
    short_ma = get_ma(stock, g.short_window)
    long_ma = get_ma(stock, g.long_window)
    # 获取当前持仓
    cur_position = context.portfolio.positions[stock].amount
    # 判断金叉（买入信号）
    if get_golden_cross(short_ma, long_ma) and cur_position == 0:
        log.info(‘出现金叉，买入信号。短期MA: %.2f, 长期MA: %.2f‘ % (short_ma, long_ma))
        # 下单买入，例如买入100股（一手）
        order_target_value(stock, 100 * context.portfolio.cash)
    # 判断死叉（卖出信号）
    elif get_dead_cross(short_ma, long_ma) and cur_position > 0:
        log.info(‘出现死叉，卖出信号。短期MA: %.2f, 长期MA: %.2f‘ % (short_ma, long_ma))
        # 下单卖出全部持仓
        order_target(stock, 0)
```

然后，我们需要实现计算移动平均线和判断交叉信号的辅助函数。

![](img/83cbf9a12faceb29cc932b688e32f2b7_7.png)

```python
# 计算移动平均线的函数
def get_ma(security, window):
    # 获取历史收盘价，这里假设使用平台提供的history函数
    # 例如：hist = attribute_history(security, window, ‘1d‘, [‘close‘])
    hist = attribute_history(security, window, ‘1d‘, [‘close‘])
    # 计算移动平均线
    ma = hist[‘close‘].mean()
    return ma

# 判断金叉的函数
def get_golden_cross(short_ma, long_ma):
    # 金叉条件：短期均线上穿长期均线
    # 注意：在实际策略中，可能需要记录前一天的均线值来精确判断“上穿”
    # 此处简化为当日短期MA大于长期MA即视为信号（实际应用需优化）
    return short_ma > long_ma

# 判断死叉的函数
def get_dead_cross(short_ma, long_ma):
    # 死叉条件：短期均线下穿长期均线
    # 此处简化为当日短期MA小于长期MA即视为信号（实际应用需优化）
    return short_ma < long_ma
```
