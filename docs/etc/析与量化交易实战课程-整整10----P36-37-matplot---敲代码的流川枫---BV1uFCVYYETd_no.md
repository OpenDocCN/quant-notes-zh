## 画布与子图的概念
有时我们需要在一个窗口中展示多个图表。例如，分析股票时，我们可能希望上方显示股票的K线图，下方显示大盘的走势图。这就需要用到“画布”和“子图”的概念。

![](img/2d01a5ae39a23b0e9224893eb238f8e6_5.png)

![](img/2d01a5ae39a23b0e9224893eb238f8e6_7.png)

*   **画布**：可以理解为一个窗口或一张画纸，是所有图表的容器。
*   **子图**：是画布上划分出的独立区域，每个区域可以绘制一个独立的图表。

![](img/2d01a5ae39a23b0e9224893eb238f8e6_9.png)

## 创建画布与子图
创建画布的方法是使用 `plt.figure()` 函数。创建子图则需要在画布对象上调用 `add_subplot()` 方法。

以下是创建画布和子图的基本步骤：

1.  首先，创建一个画布对象。
    ```python
    fig = plt.figure()
    ```
2.  然后，在画布上添加子图。`add_subplot()` 的参数格式为 `行数`、`列数`、`位置索引`。
    ```python
    ax1 = fig.add_subplot(2, 2, 1)
    ```
    这行代码表示将画布划分为 2 行 2 列（共4个区域），并在第1个位置（左上角）创建一个子图，并将这个子图对象赋值给变量 `ax1`。
3.  接下来，我们在这个子图 `ax1` 上绘图，而不是使用 `plt.plot()`。
    ```python
    ax1.plot([1, 2, 3, 4, 5, 6, 7, 8])
    ```
4.  最后，显示整个画布。
    ```python
    fig.show()
    # 或者在某些环境下也可以使用 plt.show()
    ```

![](img/2d01a5ae39a23b0e9224893eb238f8e6_11.png)

![](img/2d01a5ae39a23b0e9224893eb238f8e6_13.png)

## 子图排列示例
参数 `行数, 列数, 位置索引` 决定了子图的排列方式。索引从1开始，从左到右、从上到下计数。

例如，要创建上下排列的两个子图（两行一列），可以这样操作：

![](img/2d01a5ae39a23b0e9224893eb238f8e6_15.png)

```python
import matplotlib.pyplot as plt

# 创建画布
fig = plt.figure()

# 在上方（第1行第1列）创建第一个子图
ax1 = fig.add_subplot(2, 1, 1)
ax1.plot([1, 2, 3, 4])  # 在第一个子图上绘图

# 在下方（第2行第1列）创建第二个子图
ax2 = fig.add_subplot(2, 1, 2)
ax2.plot([4, 3, 2, 1])  # 在第二个子图上绘图

# 显示画布
plt.show()
```

![](img/2d01a5ae39a23b0e9224893eb238f8e6_17.png)

执行这段代码，会得到一个包含上下两个独立图表的窗口。

![](img/2d01a5ae39a23b0e9224893eb238f8e6_19.png)

## 调整子图间距
当创建多个子图时，它们之间默认会有一定的间距。如果需要调整，可以使用 `subplots_adjust()` 函数。

![](img/2d01a5ae39a23b0e9224893eb238f8e6_21.png)

以下是该函数的主要参数：
*   `left`：图表区域左侧的空白比例。
*   `right`：图表区域右侧的空白比例。
*   `bottom`：图表区域底部的空白比例。
*   `top`：图表区域顶部的空白比例。
*   `wspace`：子图之间宽度方向的间距。
*   `hspace`：子图之间高度方向的间距。

![](img/2d01a5ae39a23b0e9224893eb238f8e6_23.png)

调用方法如下：
```python
fig.subplots_adjust(left=0.1, bottom=0.1, right=0.9, top=0.9, wspace=0.4, hspace=0.4)
```
通常情况下，使用默认间距即可满足需求。
