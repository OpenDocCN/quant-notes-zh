# 量化交易入门：P22：因子数据预处理

在本节课中，我们将学习如何对量化交易中的因子数据进行预处理。因子是影响我们投资决策的指标，例如市盈率或营收增长率。处理这些数据是构建有效交易策略的关键步骤。我们将按照三个步骤进行：去极值、标准化和中性化。

## 因子数据预处理：P22.1：理解因子与预处理步骤

![](img/6d6f37987acc6cb4ad489be61d8d30aa_1.png)

上一节我们介绍了课程目标，本节中我们来看看什么是因子以及预处理的整体流程。

![](img/6d6f37987acc6cb4ad489be61d8d30aa_3.png)

因子是影响最终投资结果的指标。例如，在购买股票时，低市净率可能意味着较大的上涨空间，高营收增长率可能代表公司前景良好。这些标准就是因子。

当我们获得因子数据后，需要对其进行处理，以便用于建模分析。这类似于数据挖掘问题，我们探究每个因子（X1, X2, X3...）如何影响最终收益（Y），并从中选择合适的因子。

![](img/6d6f37987acc6cb4ad489be61d8d30aa_5.png)

![](img/6d6f37987acc6cb4ad489be61d8d30aa_7.png)

以下是因子数据预处理的三个核心步骤：
1.  去极值
2.  标准化
3.  中性化

![](img/6d6f37987acc6cb4ad489be61d8d30aa_9.png)

![](img/6d6f37987acc6cb4ad489be61d8d30aa_11.png)

![](img/6d6f37987acc6cb4ad489be61d8d30aa_13.png)

前两个步骤在机器学习和数据挖掘中常见。去极值处理异常数据点。标准化将不同量纲的指标转换到相同尺度。中性化在因子策略中尤为重要，用于剥离某些系统性影响。

## 因子数据预处理：P22.2：去极值方法之百分位法

上一节我们介绍了预处理的三步走策略，本节中我们来看看第一步——去极值。

![](img/6d6f37987acc6cb4ad489be61d8d30aa_21.png)

处理极值的一种常见思路不是直接删除异常点，而是将其“拉回”到合理的边界内。例如，设定一个界限为5，如果一个值是10，超过了上界，我们将其修正为5。

以下是几种去极值的方法，首先介绍分位数（百分位）法。

![](img/6d6f37987acc6cb4ad489be61d8d30aa_23.png)

分位数是常用的统计概念。中位数是将数据一分为二的值。相比均值，中位数对极端值不敏感，更能代表数据的中心趋势。例如，计算平均工资时，少数极高工资会大幅拉高均值，使其失去代表性，而中位数则更稳定。

![](img/6d6f37987acc6cb4ad489be61d8d30aa_25.png)

分位数可以进一步细分。除了中位数（Q2，50%分位），常用的还有下四分位数（Q1，25%分位）和上四分位数（Q3，75%分位）。

![](img/6d6f37987acc6cb4ad489be61d8d30aa_27.png)

![](img/6d6f37987acc6cb4ad489be61d8d30aa_29.png)

在去极值操作中，我们可以利用这些分位数来设定数据的合理范围边界。
