### 描述统计核心概念

描述统计主要包含三个部分：集中趋势、离散趋势和分布形态。

以下是描述统计的核心概念与度量方式：

**1. 集中趋势**
集中趋势用于描述数据向中心值聚集的程度。
*   **均值**：所有数据之和除以数据个数。公式为：`均值 = 总和 / 个数`。均值易受极端值影响。
*   **中位数**：将数据从小到大排序后，位于中间位置的值。它不易受极端值影响。
*   **众数**：一组数据中出现次数最多的值。

**2. 离散趋势**
离散趋势用于描述数据的分散或变异程度。
*   **方差与标准差**：衡量各数据点偏离均值的平均程度。方差公式为：`方差 = Σ(每个值 - 均值)² / (个数 - 1)`。标准差是方差的平方根。
*   **四分位差**：上四分位数与下四分位数之差，反映了中间50%数据的离散程度。
*   **变异系数**：标准差与均值的比值，公式为：`变异系数 = 标准差 / 均值`。适用于比较不同尺度或类型数据的离散程度。

**3. 分布形态**
分布形态描述了数据分布的对称性和尖峭程度。
*   **偏态**：衡量分布不对称性。分为左偏（负偏）、右偏（正偏）和对称（正态）。
*   **峰态**：衡量分布曲线的尖峭程度。分为尖峭峰、平阔峰和正态峰。
