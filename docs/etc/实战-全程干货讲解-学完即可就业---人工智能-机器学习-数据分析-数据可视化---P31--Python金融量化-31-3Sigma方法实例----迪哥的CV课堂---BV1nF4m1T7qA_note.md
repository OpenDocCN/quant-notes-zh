# Python金融量化：31：3-Sigma方法实例 📊

在本节课中，我们将要学习一种在金融数据预处理中常用的去极值方法——3-Sigma方法。我们将从正态分布的基本概念入手，理解3-Sigma方法的原理，并通过代码实例演示如何实现它，用于识别和处理数据中的异常值。

## 正态分布与异常值原理 📈

上一节我们介绍了去极值的概念，本节中我们来看看基于统计理论的3-Sigma方法。它的核心原理基于正态分布。

如果大家理解正态分布比较麻烦，可以将其视为一种“正常”的分布。以前在讲解机器学习时经常举一个例子：你去银行贷款，银行多借给你三块五块是正常的，但多借或少借你十万八万就不正常了。

我们来看这样一个走势。在标准正态分布中，均值是0。这就像一个误差：去银行贷款，可能多借（向右偏移），也可能少借（向左偏移）。

现在，我们画第一个区间（用红色阴影表示）。这个区间的面积相对较大，代表这些事件发生的可能性较高。当我把这个区间越往外扩展时，事件发生的概率就越低。

我们现在要解决的是去极值问题。极值可能落在哪里？它可能落在均值附近吗？在均值附近是大概率事件，属于正常数据。极值一般落在比较偏远的地方，离均值越远、偏差越大的点，越有可能是离群点。

## 什么是3-Sigma方法？🔍

![](img/04a97103e5b2354dfaeed844e90002cc_1.png)

3-Sigma方法正是基于上述思想。请看下图：

![](img/04a97103e5b2354dfaeed844e90002cc_1.png)

图中，`X` 代表均值。我们定义了以均值为中心，加减一倍、两倍、三倍标准差的区间。可以发现，区间选得越大（例如加减三倍标准差），数据落在左右两端的可能性就越小。

3-Sigma方法的意思是：假设我们有一份数据，并且可以通过某种变换使其理论上服从高斯分布（正态分布）。当我们指定正负三倍标准差的范围时，这个区间包含了多少数据呢？

这个值在统计学中是确定的：**约99.7%**的数据会落在均值±3倍标准差的区间内。

因此，如果我们将数据中最正常的99.7%当作有效数据，剩下的0.3%当作极值进行处理或规范，这就是3-Sigma方法。它通过计算一个下限和一个上限来界定数据的正常范围。

![](img/04a97103e5b2354dfaeed844e90002cc_3.png)

在高斯分布中，只需要知道两个值：**均值（mean）** 和 **标准差（std）**，就能计算出这个上限和下限。

![](img/04a97103e5b2354dfaeed844e90002cc_5.png)

计算公式如下：
*   上限：`upper = mean + n * std`
*   下限：`lower = mean - n * std`
其中，`n`通常取3。

![](img/04a97103e5b2354dfaeed844e90002cc_7.png)

## 3-Sigma方法代码实现 💻

理解了原理后，我们来看看如何用代码实现3-Sigma方法。我们将编写一个函数来完成计算。

![](img/04a97103e5b2354dfaeed844e90002cc_9.png)

以下是实现3-Sigma去极值函数的步骤：

![](img/04a97103e5b2354dfaeed844e90002cc_11.png)

1.  **计算均值和标准差**：首先，我们需要计算传入数据序列的均值和标准差。
2.  **计算上下限**：根据公式，使用均值、标准差和指定的倍数`n`计算上限和下限。
3.  **返回结果**：函数返回计算出的下限和上限。

![](img/04a97103e5b2354dfaeed844e90002cc_13.png)

具体代码如下：

```python
def three_sigma(data, n=3):
    """
    3-Sigma去极值方法
    :param data: 输入的数据序列，Pandas Series类型
    :param n: 标准差的倍数，默认为3
    :return: 下限(lower), 上限(upper)
    """
    # 1. 计算均值
    mean = data.mean()
    # 2. 计算标准差
    std = data.std()
    
    # 3. 计算上下限
    upper = mean + n * std
    lower = mean - n * std
    
    return lower, upper
```

![](img/04a97103e5b2354dfaeed844e90002cc_15.png)

这个函数非常简单。我们传入一个数据序列`data`和倍数`n`，它就能返回基于3-Sigma原则计算出的正常值范围边界。

![](img/04a97103e5b2354dfaeed844e90002cc_17.png)

## 方法应用与结果可视化 📉

![](img/04a97103e5b2354dfaeed844e90002cc_21.png)
