## 趋势线的设置方法

趋势线绘制完成后，我们常常需要根据策略逻辑动态调整它的外观和位置。MC提供了一系列以 `TL_Set` 开头的函数来实现这些功能。

### 设置趋势线端点

`TL_SetBegin` 和 `TL_SetEnd` 函数用于调整一条已存在趋势线的起点和终点。它们的第一个参数都是趋势线的编号（`TL_ID`），这表明操作对象必须是已经绘制在图表上的趋势线。

![](img/966c982c0c66e45d372b587c49573119_5.png)

![](img/966c982c0c66e45d372b587c49573119_7.png)

![](img/966c982c0c66e45d372b587c49573119_9.png)

以下是设置端点的基本语法：
```easylanguage
TL_SetBegin(TL_ID, Date1, Time1, Price1);
TL_SetEnd(TL_ID, Date2, Time2, Price2);
```
请注意，函数名带 `_S` 后缀的版本（如 `TL_SetBegin_S`）要求时间参数包含秒数（`Time_S`），这通常用于更高精度的周期图表。

![](img/966c982c0c66e45d372b587c49573119_11.png)

### 设置趋势线延伸

![](img/966c982c0c66e45d372b587c49573119_13.png)

`TL_SetExtLeft` 和 `TL_SetExtRight` 函数控制趋势线是否向左或向右无限延伸。
```easylanguage
TL_SetExtLeft(TL_ID, True); // 向左延伸
TL_SetExtRight(TL_ID, False); // 取消向右延伸
```
第二个参数为 `True` 表示添加延伸，`False` 表示删除延伸。

### 设置趋势线视觉属性

![](img/966c982c0c66e45d372b587c49573119_15.png)

![](img/966c982c0c66e45d372b587c49573119_17.png)

我们可以轻松地修改趋势线的颜色、粗细和线型，让图表更具可读性。

*   **设置颜色 (`TL_SetColor`)**: 可以使用内置的17种颜色名称（如 `Yellow`），RGB值，或传统色码（需配合 `LegacyColorValue` 设置）。
*   **设置粗细 (`TL_SetSize`)**: 宽度范围是0到6。
*   **设置线型 (`TL_SetStyle`)**: 可以设置为虚线、点线等不同样式。

![](img/966c982c0c66e45d372b587c49573119_19.png)

![](img/966c982c0c66e45d372b587c49573119_21.png)

以下是一个更改趋势线样式的示例：
```easylanguage
TL_SetStyle(688, 4); // 将编号为688的趋势线设置为“点划线”样式
```
编译并应用该代码后，对应趋势线的外观就会改变。

## 趋势线的警报功能

趋势线可以设置警报，当价格触及或突破趋势线时，平台可以发出声音、弹窗或邮件通知。`TL_SetAlert` 函数用于管理此功能。

```easylanguage
TL_SetAlert(TL_ID, AlertMode);
```
其中 `AlertMode` 参数：
*   `0`: 停用警报。
*   `1`: 棒内突破时警报（当最高价向上突破或最低价向下突破趋势线时）。
*   `2`: 棒收盘突破时警报（当收盘价穿越趋势线时）。

警报的详细设置（如检查频率、通知方式）可以在指标的属性窗口中配置。

## 获取趋势线信息

除了设置，MC也提供了许多 `TL_Get` 开头的函数来获取趋势线的信息，这在一些复杂的绘图管理策略中会用到。

### 获取特定趋势线

`TL_GetFirst` 和 `TL_GetNext` 函数用于遍历图表上的趋势线。
*   `TL_GetFirst(Type)`: 返回图表上符合条件的第一条趋势线的编号。
*   `TL_GetNext(TL_ID, Type)`: 返回在指定编号趋势线之后绘制的、符合条件的一条趋势线的编号。

其中的 `Type` 参数用于筛选趋势线的来源：
*   `1`: 当前指标绘制的趋势线。
*   `2`: 非当前指标绘制或手动绘制的趋势线。
*   `7`: 使用者手动绘制的趋势线。

### 获取趋势线属性

我们可以获取趋势线的几乎所有属性，例如：
*   `TL_GetValue(TL_ID)`: 获取当前K线位置所对应的趋势线上的价格值。
*   `TL_GetBegin(TL_ID)`: 获取趋势线起点的日期时间。
*   `TL_GetColor(TL_ID)`: 获取趋势线的颜色。
*   `TL_GetActive()`: 获取图表上最新创建的趋势线编号。

![](img/966c982c0c66e45d372b587c49573119_23.png)

![](img/966c982c0c66e45d372b587c49573119_25.png)

**重要提示**：在使用 `TL_GetValue` 等函数时，务必注意逻辑的滞后性。趋势线是基于已形成的K线高点/低点绘制的，若在策略中用它来触发实时交易信号，需谨慎避免未来函数问题。

## 在图表上添加文字

文字（Text）是另一个重要的绘图工具，它可以在图表特定位置显示信息，并且最大的优点是**可以在信号（Signal）中直接使用**，而 `Plot` 语句则不能。

### 创建文字

使用 `Text_New` 函数在图表上创建一个文字标签。你需要指定其位置和内容。
```easylanguage
Text_New(Date, Time, Price, “显示内容”);
```
前三个参数 `Date`, `Time`, `Price` 共同决定了文字在图表上的位置（一个点）。同样，有 `Text_New_S` 函数对应包含秒数的时间参数。

### 管理与设置文字

![](img/966c982c0c66e45d372b587c49573119_29.png)

![](img/966c982c0c66e45d372b587c49573119_31.png)

与趋势线类似，文字也可以进行动态管理和样式设置。

![](img/966c982c0c66e45d372b587c49573119_33.png)

![](img/966c982c0c66e45d372b587c49573119_35.png)

*   **删除文字 (`Text_Delete`)**: 及时清理不再需要的文字，保持图表整洁。
*   **移动文字 (`Text_SetLocation`)**: 调整文字的位置。
    ```easylanguage
    Text_SetLocation(Text_ID, NewDate, NewTime, NewPrice);
    ```
*   **修改内容 (`Text_SetString`)**: 更改文字显示的内容。
    ```easylanguage
    Text_SetString(Text_ID, “新内容”);
    ```
*   **设置样式**: 可以分别设置文字颜色、背景色和字体。
    ```easylanguage
    Text_SetColor(Text_ID, Red); // 设置文字为红色
    Text_SetBGColor(Text_ID, White); // 设置背景为白色
    Text_SetFont(Text_ID, “Arial”); // 设置字体为Arial
    ```
    字体的名称可以从你电脑系统的字体库中查找。
