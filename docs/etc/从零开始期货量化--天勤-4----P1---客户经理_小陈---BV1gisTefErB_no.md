## 自动化交易的核心三要素

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_5.png)

量化交易的关键在于自动化。实现自动化交易主要需要进行以下三方面的工作。

以下是实现自动化交易的三个核心步骤：

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_7.png)

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_9.png)

1.  **数据获取**：获取历史数据进行策略回测，获取实时数据用于在实盘中计算交易信号。
2.  **计算分析**：基于获取的数据进行计算，生成交易信号（Signal）。
3.  **自动交易**：根据交易信号自动下单，并接收成交反馈信息，最终实现对账户的查看和管理。

把这三个部分做好，就可以实现完整的量化交易。

## 期货交易的底层：CTP系统

目前国内期货软件的底层大多基于CTP系统。CTP是由上海期货信息技术有限公司（上期所全资子公司）开发的综合交易平台。它是一整套期货经纪业务管理系统，主要由交易、风险控制和结算三部分组成。我们现在使用的所有期货软件或行情软件都是基于这套CTP系统进行开发的。

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_11.png)

基于CTP系统的整体规则，官方提供了客户端开发指南和相应的API（应用程序编程接口）。API主要分为两大块：行情接口和交易接口。目前所有的交易软件以及Python量化交易框架，都是在这套基础API之上进行的二次开发和完善。所有的行情交互和下单交易都基于这套规则进行。

## 为何需要现成的API？

CTP提供的API非常基础和底层，使用C++编写。对于量化交易者而言，核心应专注于策略研究，而非从零开始构建复杂的底层通信程序。学习成本过高且耗时。

解决这个问题的方法是使用“拿来主义”，借助市场上成熟的解决方案——即现成的、封装好的API。这可以让我们避免重复造轮子，直接站在前人的肩膀上，高效地开始策略开发。

## 直观感受API的魅力 🎯

上一节我们介绍了API的概念，本节中我们来看看如何用一行代码快速获取数据。我们将通过天勤（TqSdk）这个Python库来直观感受API的便捷性。

以下是使用天勤API的几个核心功能演示：

*   **获取账户信息**：通过 `api.get_account()` 可以轻松获取账户的权益、可用资金等信息。
*   **订阅实时行情**：通过 `api.get_quote(“合约代码”)` 可以获取指定合约的最新价、买卖盘口等实时数据。
*   **获取历史K线数据**：通过 `api.get_kline_serial(“合约代码”, 周期秒数, 数据长度)` 可以获取历史K线数据。例如，获取日线数据（86400秒）最近3000根。

运行演示代码后，我们可以立即看到打印出的账户信息字典、实时变化的行情字典以及格式化的历史K线数据列表。这些原始数据后续可以方便地进行清洗、转换和分析，用于策略回测。

API的作用就是通过一行或几行代码，轻松获取所需数据，极大提升开发效率。

## 实时数据获取结构

如果想持续获取最新的行情价格，可以使用一个循环结构。在天勤的框架中，`api.wait_update()` 函数是核心，它有两个作用：一是当数据更新时推动整个程序运行一次；二是执行挂起的任务（如报单）。

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_15.png)

以下是一个实时打印最新价格的例子：

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_17.png)

```python
while True:
    api.wait_update()
    quote = api.get_quote(“RB888”) # 获取螺纹钢主连行情
    print(quote.datetime, quote.last_price) # 打印时间和最新价
```

这段代码会以约0.5秒（交易所快照频率）的间隔，持续打印出合约的最新成交价和时间，实现了行情的实时监控。

## 理解SDK与API的关系

*   **SDK（软件开发工具包）**：好比一个**超市**，里面提供了做量化交易所需的各种工具和货品。
*   **API（应用程序编程接口）**：好比一个**购物车**，是我们从SDK这个超市中获取特定工具（函数）和数据的方式。

在我们的课程中，主要使用的SDK是**天勤量化（TqSdk）**。

## 如何开始使用天勤SDK

上一节我们了解了天勤SDK的优势，本节中我们来看看如何安装和配置它。

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_27.png)

以下是开始使用天勤SDK的步骤：

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_29.png)

![](img/1a8ed4b16ae5494ea84c84b134dd7a2a_30.png)

1.  **安装库**：在Anaconda Prompt中使用命令 `pip install tqsdk -i https://pypi.tuna.tsinghua.edu.cn/simple` 进行安装或更新。
2.  **注册账户**：访问天勤官网注册一个快期账户，获取登录凭证。
3.  **导入使用**：在Python代码中，通过 `from tqsdk import TqApi, TqAuth` 导入库，然后使用 `api = TqApi(auth=TqAuth(“用户名”, “密码”))` 创建API实例，即可开始调用各种功能函数。

关于账户绑定、实盘交易框架以及每个函数的详细用法，我们会在后续章节，特别是第五章《实盘交易框架》中深入讲解。第四章我们的重点将放在利用这些API函数建立数据库和搭建回测系统。
