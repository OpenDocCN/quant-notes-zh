## 数据准备与变换

![](img/6d0a41825b9758c64c077c07fc1da89b_2.png)

上一节我们介绍了如何构建特征，本节中我们来看看如何对数据进行变换，以便进行统计分析。

![](img/6d0a41825b9758c64c077c07fc1da89b_4.png)

![](img/6d0a41825b9758c64c077c07fc1da89b_6.png)

我们当前的数据中，最后两列是特征（例如“前一天涨跌”和“前两天涨跌”），最后一列是结果（例如“当天涨跌”）。为了进行统计，我们需要将特征组合起来，观察不同特征组合对应的结果分布。

![](img/6d0a41825b9758c64c077c07fc1da89b_8.png)

![](img/6d0a41825b9758c64c077c07fc1da89b_10.png)

以下是具体的数据变换步骤：

![](img/6d0a41825b9758c64c077c07fc1da89b_12.png)

1.  使用 `groupby` 方法，按照两个特征列进行分组。
2.  对分组后的“结果”列进行计数统计。
3.  使用 `unstack` 方法将结果转换为更易读的透视表格式，并使用 `fillna(0)` 将缺失值填充为0，确保所有组合都显示出来。

![](img/6d0a41825b9758c64c077c07fc1da89b_14.png)

```python
# 假设 df 是包含特征和结果的数据框
# feature1, feature2 是两个特征列名，result 是结果列名
grouped_result = df.groupby([‘feature1‘, ‘feature2‘])[‘result‘].size().unstack().fillna(0)
```

![](img/6d0a41825b9758c64c077c07fc1da89b_16.png)

![](img/6d0a41825b9758c64c077c07fc1da89b_18.png)

执行上述操作后，我们将得到一个表格，行和列分别代表两个特征的不同取值，表格内的数值则代表了在该特征组合下，结果为某个特定值（例如“涨”或“跌”）出现的次数。这使我们能够直观地看到不同特征组合对结果的影响。

## 策略制定与统计

基于上一步得到的统计表格，我们可以分析规律并制定交易策略。

我们的目标是找出一种模式：当特征满足特定条件时，预测下一个交易日的股价走势。例如，我们观察统计表后发现，当“前一天涨”和“前两天涨”同时发生时（即两个特征值都为1），下一个交易日“跌”的可能性似乎更大。

以下是制定策略的步骤：

![](img/6d0a41825b9758c64c077c07fc1da89b_20.png)

1.  根据统计观察，定义策略规则。例如：如果两个特征值之和等于2（即前两天都上涨），则预测下一个交易日下跌（标记为-1）；否则预测上涨（标记为1）。
2.  使用NumPy的 `where` 函数，根据规则为数据中的每一天生成预测信号。

![](img/6d0a41825b9758c64c077c07fc1da89b_22.png)

![](img/6d0a41825b9758c64c077c07fc1da89b_24.png)

```python
import numpy as np
# 假设 df[‘feature1‘] 和 df[‘feature2‘] 是特征列
df[‘position‘] = np.where((df[‘feature1‘] + df[‘feature2‘]) == 2, -1, 1)
```

![](img/6d0a41825b9758c64c077c07fc1da89b_26.png)

![](img/6d0a41825b9758c64c077c07fc1da89b_28.png)

这段代码创建了一个名为 `position` 的新列，其中 `-1` 代表“看跌/卖出”信号，`1` 代表“看涨/买入”信号。这个规则完全基于我们对前面统计结果的观察。

![](img/6d0a41825b9758c64c077c07fc1da89b_30.png)

## 策略效果评估

![](img/6d0a41825b9758c64c077c07fc1da89b_32.png)

制定策略后，我们需要评估其预测的准确性以及与市场真实走势的对比。

![](img/6d0a41825b9758c64c077c07fc1da89b_34.png)

首先，我们计算策略预测结果与实际涨跌方向一致的天数比例。

![](img/6d0a41825b9758c64c077c07fc1da89b_36.png)

![](img/6d0a41825b9758c64c077c07fc1da89b_38.png)

```python
# 假设 df[‘direction‘] 是实际的涨跌方向（1为涨，-1为跌）
accuracy_series = (df[‘position‘] == df[‘direction‘])
correct_count = accuracy_series.value_counts()[True]
wrong_count = accuracy_series.value_counts()[False]
```

![](img/6d0a41825b9758c64c077c07fc1da89b_40.png)

接下来，我们模拟按照此策略进行交易的投资回报，并与单纯跟随大盘走势（即买入并持有）的回报进行对比。计算累计收益的公式如下：

![](img/6d0a41825b9758c64c077c07fc1da89b_42.png)

**累计收益 = (1 + 策略信号 * 当日收益率).cumprod()**

![](img/6d0a41825b9758c64c077c07fc1da89b_44.png)

![](img/6d0a41825b9758c64c077c07fc1da89b_46.png)

```python
# 假设 df[‘returns‘] 是股票每日的收益率
strategy_returns = (1 + df[‘position‘].shift(1) * df[‘returns‘]).cumprod()
buy_hold_returns = (1 + df[‘returns‘]).cumprod()
```

![](img/6d0a41825b9758c64c077c07fc1da89b_48.png)

通过绘制 `strategy_returns` 和 `buy_hold_returns` 的曲线，我们可以直观地比较两种方式的最终收益。在本例的演示中，基于统计的策略可能仍然亏损，但亏损幅度小于简单的买入持有策略，这说明了策略具有一定的效果。

![](img/6d0a41825b9758c64c077c07fc1da89b_50.png)

![](img/6d0a41825b9758c64c077c07fc1da89b_52.png)
