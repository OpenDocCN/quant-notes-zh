# 量化组合优化：P1：使用Portfolioopt库进行投资组合优化 🧮

在本节课中，我们将学习量化投资中的一个核心应用——投资组合优化。我们将重点介绍一个名为`portfolioopt`的Python库，它可以帮助我们以简洁的代码实现复杂的投资组合优化问题。我们将从理论基础讲起，然后通过一个具体的实例，对比等权重组合与基于夏普比率优化的组合之间的表现差异。

## 理论基础：均值-方差模型 📊

上一节我们介绍了量化组合优化的概念，本节中我们来看看其经典的理论基础。

投资组合优化的经典理论是哈里·马科维茨于1952年提出的均值-方差模型。该模型将投资问题数学化。其核心思想是：在投资一个包含N只股票的组合时，我们需要计算每只股票的预期收益率（均值）和股票之间的风险关联（协方差矩阵）。

该优化问题可以概括为以下数学公式：

**目标**：最小化投资组合的风险（方差）。
**约束条件**：
1.  所有股票的权重之和为1：`∑ w_i = 1`
2.  每只股票的权重非负（假设不允许卖空）：`w_i ≥ 0`
3.  组合的预期收益率不低于某个目标值。

![](img/3e4ee5a8df14e84efe2143fcae829041_1.png)

通过求解此优化问题，可以得到一条“有效前沿”。有效前沿代表了在给定风险水平下能获得最高预期收益的投资组合集合。通常，我们会选择夏普比率（Sharpe Ratio）最高的组合，即单位风险所获得的超额回报最高的组合。

## 实践准备：环境与数据 📥

理解了理论基础后，我们进入实践环节。首先需要准备编程环境和数据。

以下是使用`portfolioopt`库前需要完成的准备工作：

1.  **安装库**：通过`pip install portfolioopt`命令安装。若安装缓慢，可使用清华镜像源加速。
2.  **引入必要库**：除了`portfolioopt`，我们还需要`pandas`处理数据，以及`quantstats`进行业绩分析。
3.  **准备数据**：需要两种时间序列数据：股票的每日收益率和每日收盘价（通常使用前复权价格）。

**注意数据格式**：原始数据导入后，索引和日期格式通常需要调整。
*   索引应设置为日期列。
*   日期列的数据类型应转换为`datetime`格式。
*   许多函数要求输入的数据格式为`pandas Series`，而非`DataFrame`。

处理后的收益率数据和收盘价数据将用于后续计算。

## 核心步骤：计算与优化 ⚙️

数据准备就绪后，我们就可以开始计算最优投资组合了。这个过程主要分为三步。

以下是构建最优投资组合的关键步骤：

1.  **计算预期收益与协方差矩阵**：使用`portfolioopt`中的函数，传入处理好的收盘价`DataFrame`，计算出投资组合的预期收益率向量（mu）和样本协方差矩阵（S）。
2.  **构建有效前沿并优化**：将计算出的mu和S传入`efficient_frontier`函数。本例中，我们采用最大化夏普比率的目标来求解最优权重，使用`ef.max_sharpe()`方法。
3.  **清理与提取权重**：计算出的权重可能包含极小的数值（如`0.000001`），这些数值可能干扰结果，通常将其清理为零。清理后，我们就得到了一个最优的资产配置权重字典。

## 策略对比：等权重 vs 优化组合 ⚖️

得到优化组合的权重后，我们需要一个基准来评估其效果。最常用的基准之一是等权重组合。

等权重组合为每只股票分配相同的资金比例（例如，4只股票各占25%）。我们分别计算优化组合和等权重组合的历史每日收益率序列。

以下是使用`quantstats`库进行业绩对比的关键指标：

*   **年化收益率**：优化策略的年化收益通常高于等权重基准。
*   **夏普比率**：以最大化夏普比率为目标构建的组合，其夏普比率应显著高于等权重组合。
*   **最大回撤**：优化组合的最大回撤通常小于等权重组合，表明其风险控制更佳。
*   **累计收益曲线**：可视化图表可以清晰显示，优化策略的净值曲线在大部分时间都跑赢了等权重基准，创造了稳定的超额收益。

通过全面的指标对比和图表分析，可以直观地验证基于`portfolioopt`库进行组合优化的有效性。
