# 量化交易教程：P14：策略收益效果分析 📈

在本节课中，我们将学习如何对一个量化交易策略的收益效果进行分析。我们将通过计算原始股价的累计收益，并与应用了双均线策略后的策略收益进行对比，来评估策略的有效性。

![](img/109131841b23b890ed8b05141f4c79c9_1.png)

![](img/109131841b23b890ed8b05141f4c79c9_3.png)

上一节我们介绍了如何通过双均线策略生成交易信号（`position`）。本节中，我们来看看如何基于这些信号计算策略的最终收益，并与“买入并持有”的原始收益进行比较。

## 计算原始收益率

首先，我们需要计算股价的日收益率。在量化分析中，我们通常使用对数收益率，因为它具有更好的数学性质（如可加性）。计算公式如下：

`log_return = log(price_t / price_{t-1})`

![](img/109131841b23b890ed8b05141f4c79c9_5.png)

![](img/109131841b23b890ed8b05141f4c79c9_7.png)

在代码中，我们使用Pandas的`shift`函数来获取前一天的股价，并进行计算。

![](img/109131841b23b890ed8b05141f4c79c9_9.png)

```python
# 假设 data[‘Close’] 是股价序列
returns = np.log(data[‘Close’] / data[‘Close’].shift(1))
```

![](img/109131841b23b890ed8b05141f4c79c9_11.png)

执行以上代码后，我们得到了一个表示每日增长率的`returns`序列。

![](img/109131841b23b890ed8b05141f4c79c9_13.png)

## 计算策略收益率

![](img/109131841b23b890ed8b05141f4c79c9_15.png)

接下来，我们根据策略信号来计算策略收益率。我们的策略核心是：当`position`为1时，表示我们遵循原始走势；当`position`为-1时，表示我们对当日的收益进行反转（即做空或反向操作）。

![](img/109131841b23b890ed8b05141f4c79c9_17.png)

策略收益的计算公式为：
`strategy_returns = position.shift(1) * returns`

![](img/109131841b23b890ed8b05141f4c79c9_19.png)

这里需要对`position`进行`shift(1)`操作，以确保我们使用的是前一天结束时确定的仓位来影响今天的收益，这符合实际交易中“T+1”的执行逻辑。

![](img/109131841b23b890ed8b05141f4c79c9_21.png)

```python
# 假设 position 是之前生成的策略信号序列（值为1或-1）
strategy = position.shift(1) * returns
```

![](img/109131841b23b890ed8b05141f4c79c9_23.png)

![](img/109131841b23b890ed8b05141f4c79c9_25.png)

计算完成后，我们得到了`strategy`序列，它代表了应用策略后每天的收益率。

![](img/109131841b23b890ed8b05141f4c79c9_27.png)

![](img/109131841b23b890ed8b05141f4c79c9_29.png)

## 累计收益与结果还原

![](img/109131841b23b890ed8b05141f4c79c9_31.png)

![](img/109131841b23b890ed8b05141f4c79c9_33.png)

为了比较最终收益，我们需要将每日的收益率累加起来，并还原为最终的价值比（即1元本金最终变为多少元）。累计对数收益即为各日收益之和，再通过指数函数还原。

累计收益计算公式为：
`total_return = exp(sum(returns))`

我们将分别对原始收益率`returns`和策略收益率`strategy`进行此操作。

![](img/109131841b23b890ed8b05141f4c79c9_35.png)

![](img/109131841b23b890ed8b05141f4c79c9_37.png)

```python
# 删除可能存在的缺失值
returns = returns.dropna()
strategy = strategy.dropna()

![](img/109131841b23b890ed8b05141f4c79c9_39.png)

# 计算累计收益并还原
original_final_value = np.exp(returns.sum())
strategy_final_value = np.exp(strategy.sum())
```

![](img/109131841b23b890ed8b05141f4c79c9_41.png)

以下是计算结果对比：
*   **原始收益**：如果什么都不做，只是买入并持有，1元本金最终价值约为 **4元**。
*   **策略收益**：应用双均线策略后，1元本金最终价值约为 **5.8元**。

![](img/109131841b23b890ed8b05141f4c79c9_43.png)

![](img/109131841b23b890ed8b05141f4c79c9_45.png)

## 核心步骤总结

![](img/109131841b23b890ed8b05141f4c79c9_47.png)

本节课中我们一起学习了策略收益分析的全过程：
1.  **计算基础收益**：使用对数公式计算股价的日收益率。
2.  **计算策略收益**：将策略信号（`position`）与日收益率结合，其中需要对信号进行滞后处理以符合实际交易。
3.  **评估效果**：通过累加收益并指数还原，得到最终本金增长倍数。对比显示，在本案例中，双均线策略获得了超越简单持有策略的收益。

![](img/109131841b23b890ed8b05141f4c79c9_49.png)

![](img/109131841b23b890ed8b05141f4c79c9_51.png)

通过以上分析，我们可以定量地评估一个交易策略是否为我们带来了附加价值。
