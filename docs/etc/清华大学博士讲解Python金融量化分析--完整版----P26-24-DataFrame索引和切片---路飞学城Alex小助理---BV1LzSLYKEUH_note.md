# Python金融量化分析：P26：DataFrame索引和切片 📊

在本节课中，我们将要学习如何对Pandas的DataFrame对象进行索引和切片操作，以获取其中的数据。DataFrame是一个二维表格型数据结构，理解如何精确地定位和提取数据是进行数据分析的基础。

---

上一节我们介绍了DataFrame的一些常用属性。本节中我们来看看如何获取DataFrame对象中的具体数值。

与Series类似，我们可以使用中括号 `[]` 来获取DataFrame的值。例如，如果想获取标签为 `A` 的行和标签为 `one` 的列交叉处的值，可以这样操作：
```python
df['one']['A']
```
这种方式是先指定列，再指定行，这与数据库查询中先选择列（`SELECT`）的思路相似。

然而，如果行索引是整数，这种方式可能会产生混淆。因此，不建议连续使用两个中括号进行取值。

以下是更推荐且不易出错的方法：

我们建议根据访问需求（按标签或按下标），使用 `.loc` 或 `.iloc` 属性，并且避免连续使用两个中括号。

![](img/0c5d25145c3cd266af86b5689d0bbcf6_1.png)

![](img/0c5d25145c3cd266af86b5689d0bbcf6_3.png)

**1. 使用 `.loc` 按标签访问**
`.loc` 属性用于通过行和列的**标签**进行访问。其基本语法是逗号分隔，前面是行索引，后面是列索引。
```python
df.loc['A', 'one']
```
如果想获取 `A` 这一行的所有数据，列的部分可以使用切片 `:` 来表示选择所有列。
```python
df.loc['A', :]
```

![](img/0c5d25145c3cd266af86b5689d0bbcf6_5.png)

**2. 使用 `.iloc` 按位置访问**
`.iloc` 属性用于通过行和列的**整数位置**（下标）进行访问。其语法与 `.loc` 类似。
```python
df.iloc[0, 0]  # 获取第一行，第一列的数据
```
如果想获取第一行的所有数据：
```python
df.iloc[0, :]
```

DataFrame是一个二维数据类型，拥有行索引和列索引。我们可以灵活地组合使用标签、位置、切片、布尔索引和花式索引。

以下是索引和切片可以搭配的方式：

![](img/0c5d25145c3cd266af86b5689d0bbcf6_10.png)

![](img/0c5d25145c3cd266af86b5689d0bbcf6_12.png)

这意味着在 `.loc` 或 `.iloc` 中，行和列的部分可以自由组合。例如，使用 `.loc` 进行花式索引（传递一个标签列表）与切片搭配：
```python
df.loc[['A', 'C'], 'two':'three']
```
上述代码选择了行标签为 `A` 和 `C` 的行，以及从列标签 `two` 到 `three` 的列。

同样，也可以使用布尔值索引进行条件筛选，与其他索引方式灵活组合，从而非常方便地获取各种所需的数据子集。
