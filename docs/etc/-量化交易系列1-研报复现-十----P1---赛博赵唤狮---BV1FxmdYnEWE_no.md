## 十字孕线形态简介

为了帮助那些可能不太了解十字孕线的朋友，我简要介绍一下十字孕线。顾名思义，它的第一根K线较长，能够完全包裹住第二根K线，就像第二根线是由第一根线孕育出来的一样。第二根线呈十字形状，因此得名十字孕线。

![](img/cdec36807cb4a4e003b0fff5b781280e_14.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_16.png)

十字孕线分为正面和负面两种形态。通常认为，正面十字孕线代表下降行情中的底部反转信号，而负面十字孕线则相反，代表上升行情中的顶部反转信号。

![](img/cdec36807cb4a4e003b0fff5b781280e_18.png)

但事实是否真的如此呢？我们可以借助计算机，使用量化的手段来验证其有效性。

![](img/cdec36807cb4a4e003b0fff5b781280e_20.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_22.png)

## 十字孕线的量化定义

![](img/cdec36807cb4a4e003b0fff5b781280e_24.png)

首先我们需要对十字孕线进行准确的定义。以下是将十字孕线的条件准确地描述出来的量化定义：

![](img/cdec36807cb4a4e003b0fff5b781280e_26.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_28.png)

*   **正面十字孕线**：
    1.  第一根K线为阴线：昨开 > 昨收
    2.  第二根K线为十字线：今开 = 今收，且今高 != 今低
    3.  第二根K线被第一根完全吞没：今高 < 昨高，且今低 > 昨低

*   **负面十字孕线**：
    1.  第一根K线为阳线：昨开 < 昨收
    2.  第二根K线为十字线：今开 = 今收，且今高 != 今低
    3.  第二根K线被第一根完全吞没：今高 < 昨高，且今低 > 昨低

![](img/cdec36807cb4a4e003b0fff5b781280e_30.png)

现在我们依据量化描述来编写代码。核心的筛选条件只有几行：

![](img/cdec36807cb4a4e003b0fff5b781280e_32.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_34.png)

```python
# 假设 df 为包含‘open’, ‘close’, ‘high’, ‘low’列的DataFrame，并已计算好前一日数据（如‘open_prev’）
# 定义十字线条件
is_doji = (df[‘open’] == df[‘close’]) & (df[‘high’] != df[‘low’])
# 定义被前一日K线吞没条件
is_engulfed = (df[‘high’] < df[‘high_prev’]) & (df[‘low’] > df[‘low_prev’])

![](img/cdec36807cb4a4e003b0fff5b781280e_36.png)

# 判断第一根K线阴阳并赋值信号：1为正面，-1为负面
df[‘signal’] = 0
df.loc[is_doji & is_engulfed & (df[‘open_prev’] > df[‘close_prev’]), ‘signal’] = 1  # 正面
df.loc[is_doji & is_engulfed & (df[‘open_prev’] < df[‘close_prev’]), ‘signal’] = -1 # 负面
```

![](img/cdec36807cb4a4e003b0fff5b781280e_38.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_40.png)

这段代码中，`signal`字段为1则代表出现正面十字孕线，为-1则代表出现负面十字孕线。执行这段代码，我们就能筛选出历史上所有出现十字孕线形态的时间点。

![](img/cdec36807cb4a4e003b0fff5b781280e_42.png)

## 历史数据回测与胜率计算

![](img/cdec36807cb4a4e003b0fff5b781280e_44.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_46.png)

现在我们已经可以把十字孕线形态筛选出来了。那么现在就可以引入大量的历史数据，来分析和统计十字孕线的历史胜率。

![](img/cdec36807cb4a4e003b0fff5b781280e_48.png)

这里获取了从近3年间所有沪深A股（主板和创业板）个股的前复权历史日线数据。核心的筛选代码依然是上面那四行。此外，我们分别取信号出现后未来5日、10日、20日、60日的收盘价，并与信号发生时的收盘价进行对比，计算未来的股价变化率。

![](img/cdec36807cb4a4e003b0fff5b781280e_50.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_52.png)

以下是计算未来收益率和胜率的逻辑：

![](img/cdec36807cb4a4e003b0fff5b781280e_54.png)

```python
# 计算未来N日的收益率
periods = [5, 10, 20, 60]
for period in periods:
    df[f‘return_{period}d’] = df[‘close’].shift(-period) / df[‘close’] - 1

![](img/cdec36807cb4a4e003b0fff5b781280e_56.png)

# 分别统计正面和负面信号的胜率
positive_df = df[df[‘signal’] == 1]
negative_df = df[df[‘signal’] == -1]

![](img/cdec36807cb4a4e003b0fff5b781280e_58.png)

positive_win_rate = {}
negative_win_rate = {}
for period in periods:
    # 对于正面信号，未来价格上涨视为胜
    positive_win_rate[period] = (positive_df[f‘return_{period}d’] > 0).mean()
    # 对于负面信号，未来价格下跌视为胜
    negative_win_rate[period] = (negative_df[f‘return_{period}d’] < 0).mean()
```

![](img/cdec36807cb4a4e003b0fff5b781280e_60.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_62.png)

运行代码后，我们得到结果。对于正面十字孕线，历史上一共发生了8000多次；对于负面的十字孕线，历史上发生了1万1000多次。统计不同持有周期下的平均胜率，并通过图表展示，可以发现：

![](img/cdec36807cb4a4e003b0fff5b781280e_64.png)

对于大多数周期来说，胜率都非常接近50%。只有正面信号的60日周期，胜率明显低于50%。也就是说，正面的十字孕线在长期（60日）来看，可能反而是一个有效的看空信号。而十字孕线在其他周期没有明显的预测效果。

![](img/cdec36807cb4a4e003b0fff5b781280e_66.png)

![](img/cdec36807cb4a4e003b0fff5b781280e_68.png)

这个结果可能和许多交易者的固有印象大相径庭。
