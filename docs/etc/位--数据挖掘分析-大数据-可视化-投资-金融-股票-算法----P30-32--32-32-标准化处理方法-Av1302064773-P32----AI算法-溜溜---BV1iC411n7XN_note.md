在本节课中，我们将要学习数据预处理中的一个核心步骤——标准化。标准化是数据挖掘和机器学习中非常常用的技术，其目的是消除不同数据特征（或指标）在量纲和取值范围上的差异，使数据更易于分析和建模。

![](img/2935bd28fd73741b6abf2aea0e2d9f40_4.png)

## 标准化的目的与原理

上一节我们介绍了数据的基本处理，本节中我们来看看如何通过标准化来统一数据的尺度。

标准化的核心出发点在于：数据集中不同的指标（或因子）往往具有不同的取值范围。例如，一个指标的值可能在0到100之间，而另一个指标的值可能在0到10000之间。这种数值层面的巨大差异，如果不加处理，可能会在后续的算法计算中，让取值范围大的指标主导结果，从而影响分析的公平性和准确性。

因此，标准化的目的就是为了让所有指标在各个维度上具有可比性，即“用同样的眼光去观察它们”。

## 标准化的数学公式

标准化的具体操作通过一个数学公式完成。该公式分为两个步骤：

**标准化公式**为：
`z = (x - μ) / σ`

其中：
*   `x` 是原始数据。
*   `μ` 是该数据所在维度（特征）的**均值**。
*   `σ` 是该数据所在维度（特征）的**标准差**。

这个公式可以分解为两个连续的操作：

1.  **去均值**：`x - μ`
    *   **目的**：将数据集的中心移动到零点。经过此操作，数据在各个维度上将以零为中心对称分布。
2.  **除标准差**：`(x - μ) / σ`
    *   **目的**：消除不同维度间取值范围的差异。标准差衡量了数据的离散程度。对于取值范围大、离散程度高的数据，其标准差`σ`也大，相除后能将其数值范围压缩。反之，对于取值范围小的数据，其标准差`σ`小，相除后数值范围变化相对温和。最终结果是所有维度的数据都被缩放至一个相近的数值区间（通常围绕0分布）。

通过这两个步骤，数据不仅以零为中心，而且各维度的尺度也被统一了。

## 标准化的代码实现

![](img/2935bd28fd73741b6abf2aea0e2d9f40_6.png)

理解了原理后，我们可以用Python轻松实现标准化。以下是手动实现标准化函数的方法：

![](img/2935bd28fd73741b6abf2aea0e2d9f40_8.png)

```python
def standardize_series(series):
    """
    对输入的pandas Series进行标准化处理。
    参数:
        series: 待标准化的数据序列。
    返回:
        标准化后的数据序列。
    """
    # 计算该序列的均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    # 应用标准化公式： (x - 均值) / 标准差
    standardized_series = (series - mean_val) / std_val
    return standardized_series
```

![](img/2935bd28fd73741b6abf2aea0e2d9f40_10.png)

使用这个函数非常简单。假设我们有一个名为`data`的Pandas DataFrame，其中包含需要处理的列（例如‘price’）：

![](img/2935bd28fd73741b6abf2aea0e2d9f40_12.png)

![](img/2935bd28fd73741b6abf2aea0e2d9f40_14.png)

```python
# 查看原始数据，其取值范围可能差异很大
print(data[‘price‘].head())
# 应用标准化函数
standardized_price = standardize_series(data[‘price‘])
# 查看标准化后的数据，其值将集中在0附近
print(standardized_price.head())
```

## 使用现成工具库

![](img/2935bd28fd73741b6abf2aea0e2d9f40_16.png)

![](img/2935bd28fd73741b6abf2aea0e2d9f40_18.png)

除了手动实现，我们也可以利用成熟的机器学习库来快速完成标准化，这通常只需一行代码。

![](img/2935bd28fd73741b6abf2aea0e2d9f40_20.png)

以下是使用`scikit-learn`库中的`StandardScaler`进行标准化的方法：

![](img/2935bd28fd73741b6abf2aea0e2d9f40_22.png)

```python
from sklearn.preprocessing import StandardScaler
# 初始化标准化器
scaler = StandardScaler()
# 假设data是一个二维数组或DataFrame
# fit_transform方法会计算均值和标准差并立即进行转换
standardized_data = scaler.fit_transform(data)
```
