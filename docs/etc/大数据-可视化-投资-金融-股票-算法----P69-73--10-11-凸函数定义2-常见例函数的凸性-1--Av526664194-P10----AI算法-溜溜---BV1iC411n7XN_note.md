## 概述

## 凸函数定义回顾

上一节我们介绍了凸函数的三个定义，本节中我们来看看第四个定义。首先，我们简单回顾一下之前的内容。

第一个定义是最通用的。一个函数 **f** 是凸函数，等价于满足以下两个条件：
1.  函数的定义域 **dom f** 是一个凸集。
2.  对于定义域内任意两点 **x** 和 **y**，以及任意满足 **0 ≤ θ ≤ 1** 的 **θ**，都有：
    `f(θx + (1-θ)y) ≤ θf(x) + (1-θ)f(y)`
这个定义的几何意义是：函数图像上任意两点的连线（弦）位于函数图像之上。

第二个定义适用于高维函数，通过“降维”来简化问题。对于定义域内任意一点 **x** 和任意方向向量 **v**，构造一个一维函数 **g(t) = f(x + tv)**。如果对于所有这样的 **x** 和 **v**，**g(t)** 都是凸函数，那么原函数 **f** 就是凸函数。这可以形象地理解为在任意方向上“切一刀”，得到的截面都是凸的。

第三个定义是一阶条件，适用于可微函数。函数 **f** 是凸函数，当且仅当其定义域是凸集，且对于定义域内任意两点 **x** 和 **y**，满足：
`f(y) ≥ f(x) + ∇f(x)^T (y - x)`
其中 **∇f(x)** 是 **f** 在点 **x** 处的梯度。几何意义是：函数图像在其任意一点的切线（或切平面）之下。

## 一阶条件的证明（续）

上一节我们开始证明一阶条件，本节我们来看看如何利用定义二来完成高维情况的证明。证明分为两个方向：若 **f** 凸，则一阶条件成立；若一阶条件成立，则 **f** 凸。

**方向一：若 f 凸，则一阶条件成立**
1.  已知 **f** 是凸函数，选择定义域内任意两点 **x** 和 **y**。
2.  根据定义二，构造一维函数 **g(t) = f(x + t(y - x))**，其中 **t ∈ [0, 1]**。由于 **f** 凸，**g(t)** 也是凸函数。
3.  对于一维凸函数 **g(t)**，我们已证明它满足一阶条件：`g(1) ≥ g(0) + g'(0)(1 - 0)`。
4.  计算可得：`g(1)=f(y)`, `g(0)=f(x)`, `g'(0)=∇f(x)^T (y - x)`。
5.  代入即得：`f(y) ≥ f(x) + ∇f(x)^T (y - x)`。证明完毕。

**方向二：若一阶条件成立，则 f 凸**
1.  假设一阶条件对 **f** 成立。在定义域内任取两点 **x**, **y** 和参数 **θ ∈ [0,1]**，构造点 **z = θx + (1-θ)y**。
2.  对点对 **(x, z)** 和 **(y, z)** 分别应用一阶条件：
    *   `f(x) ≥ f(z) + ∇f(z)^T (x - z)`
    *   `f(y) ≥ f(z) + ∇f(z)^T (y - z)`
3.  将第一个不等式乘以 **θ**，第二个乘以 **(1-θ)**，然后相加：
    `θf(x) + (1-θ)f(y) ≥ f(z) + ∇f(z)^T [θ(x - z) + (1-θ)(y - z)]`
4.  注意到 `θ(x - z) + (1-θ)(y - z) = θx + (1-θ)y - z = 0`。
5.  因此，`θf(x) + (1-θ)f(y) ≥ f(z) = f(θx + (1-θ)y)`。这正好是凸函数的第一个定义。证明完毕。

一阶条件是凸函数最重要的性质之一，因为它直接给出了寻找凸函数最小值的方法：如果找到一点 **x*** 使得梯度为零 **∇f(x*) = 0**，那么 **x*** 就是全局最小点。

## 凸函数的二阶条件（第四个定义）

上一节我们介绍了一阶条件，本节中我们来看看最实用的判定方法——二阶条件。这个定义使用频率最高，因为它通常更容易验证。

如果函数 **f** 是二阶可微的（即海森矩阵存在），那么 **f** 是凸函数的充要条件是：
1.  其定义域 **dom f** 是一个凸集。
2.  对于定义域内的任意一点 **x**，其海森矩阵 **∇²f(x)** 是半正定的。记作：
    `∇²f(x) ⪰ 0` （对于所有 `x ∈ dom f`）

**理解二阶条件：**
*   **实用性**：与一阶条件需要比较两点不同，二阶条件只需检查每一点自身的性质（海森矩阵），操作上更简单。
*   **几何意义（一维情况）**：在一维时，海森矩阵退化为二阶导数 **f''(x)**。条件 `f''(x) ≥ 0` 意味着函数的一阶导数 `f'(x)` 单调不减，即函数的“弯曲方向”始终向上（像一个碗），这是凸函数的直观体现。
*   **严格凸与凹函数**：
    *   若海森矩阵正定 (`∇²f(x) ≻ 0`)，则函数是严格凸的。
    *   若海森矩阵半负定 (`∇²f(x) ⪯ 0`)，则函数是凹函数。
    *   **注意**：严格凸函数不一定要求海森矩阵处处正定（例如 `f(x)=x^4` 在 `x=0` 处二阶导为0），但反之，若海森矩阵处处正定，则函数一定是严格凸的。

**一个关键警告**：在使用所有凸函数定义（包括二阶条件）时，**定义域必须是凸集** 这一前提至关重要。例如函数 `f(x) = 1/x²`，在 `x>0` 或 `x<0` 的区间上，其二阶导数 `6/x^4 > 0`，满足二阶条件。但由于其定义域 `(-∞,0) ∪ (0, +∞)` 不是凸集（0点处的连线不包含在定义域内），所以整个函数并不是凸函数。

## 常见函数的凸性分析

掌握了凸函数的定义后，本节我们利用二阶条件来分析几类常见函数的凸性。

以下是几种典型函数的凸性判断：

![](img/f25b766b93b7e8bffe28c1e5bd361c14_3.png)

1.  **仿射函数**
    *   形式：`f(x) = Ax + b`，其中 **A** 是矩阵，**b** 是向量。
    *   凸性分析：其海森矩阵 **∇²f(x) = 0**。零矩阵既是半正定也是半负定的。因此，**仿射函数既是凸函数，也是凹函数**。

2.  **二次函数**
    *   形式：`f(x) = (1/2)x^T P x + q^T x + r`，其中 **P** 为对称矩阵，**q** 为向量，**r** 为标量。
    *   凸性分析：其海森矩阵 **∇²f(x) = P**（常数矩阵）。
        *   若 **P** 半正定，则 **f(x)** 是凸函数。
        *   若 **P** 正定，则 **f(x)** 是严格凸函数。
        *   若 **P** 半负定，则 **f(x)** 是凹函数。
    *   **二次函数的特殊性**：对于二次函数，`严格凸 ⇔ P 正定` 是成立的，这是一般函数不具备的性质。

3.  **指数函数（负指数）**
    *   形式：`f(x) = e^(a^T x + b)`，其中 **a** 是向量，**b** 是标量。
    *   凸性分析：计算其海森矩阵为 `∇²f(x) = a a^T e^(a^T x + b)`。矩阵 `a a^T` 是半正定的（对任意非零向量 **v**，有 `v^T (a a^T) v = (a^T v)^2 ≥ 0`），且指数项恒大于0。因此海森矩阵半正定，**该指数函数是凸函数**。

## 总结
本节课中，我们一起完成了凸函数理论的核心部分学习。我们首先回顾并完整证明了一阶条件，然后重点学习了最实用的判定工具——二阶条件（海森矩阵半正定）。最后，我们应用这些工具分析了仿射函数、二次函数和指数函数等常见函数的凸性。牢记“定义域为凸集”的前提，并熟练运用二阶条件进行判断，是后续进行优化建模和算法理解的关键步骤。
