在本节课中，我们将要学习网页布局中的一个核心概念——盒子模型。理解盒子模型是掌握网页元素定位和间距控制的关键。

## 概述

上一节我们介绍了如何使用`div`进行网页布局。本节中我们来看看构成这些布局元素的基础框架——盒子模型。盒子模型描述了每个HTML元素（如`div`）在页面上所占用的空间结构。



## 什么是盒子模型？

盒子模型是指，在用`div`进行网页布局时，每个`div`元素都可以被视作一个矩形的“盒子”。一个网页通常由多个这样的盒子组合而成，例如顶部导航栏、侧边栏、主内容区和页脚。



![](img/71924e4b9b3b1f7cc4c0102bd8af11af_1.png)

除了我们肉眼可见的**内容**和**边框**，每个盒子还包含两个重要的不可见属性：**内边距**和**外边距**。

*   **内边距**：指盒子边框与内部内容之间的空白区域。它规定了内容不能紧贴边框放置。
*   **外边距**：指盒子边框与外部其他元素之间的空白区域。它规定了其他元素不能紧贴当前盒子的边框。



![](img/71924e4b9b3b1f7cc4c0102bd8af11af_3.png)

因此，一个完整的盒子模型从内到外依次由四部分组成：**内容**、**内边距**、**边框**和**外边距**。



![](img/71924e4b9b3b1f7cc4c0102bd8af11af_3.png)

![](img/71924e4b9b3b1f7cc4c0102bd8af11af_5.png)

![](img/71924e4b9b3b1f7cc4c0102bd8af11af_5.png)

## 盒子模型的属性设置

了解了盒子模型的构成后，我们来看看如何通过CSS代码来设置这些属性。

以下是盒子模型的核心CSS属性：

![](img/71924e4b9b3b1f7cc4c0102bd8af11af_7.png)

*   **内边距**：属性名为 `padding`。
*   **外边距**：属性名为 `margin`。
*   **边框**：属性名为 `border`。
*   **内容尺寸**：属性名为 `width` 和 `height`。



![](img/71924e4b9b3b1f7cc4c0102bd8af11af_9.png)

## 实践演示

接下来，我们通过一个实例来直观感受盒子模型各属性的效果。



![](img/71924e4b9b3b1f7cc4c0102bd8af11af_7.png)

首先，我们创建一个基础的`div`盒子。

```html
<!DOCTYPE html>
<html>
<head>
    <style>
        #box1 {
            width: 500px;        /* 内容宽度 */
            height: 500px;       /* 内容高度 */
            border: 10px solid blue; /* 边框：10像素、实线、蓝色 */
            background-color: lightgray; /* 背景色便于观察 */
        }
    </style>
</head>
<body>
    <div id="box1">Hello</div>
</body>
</html>
```

此时，我们得到一个蓝色边框的正方形，文字“Hello”紧贴左上角。



### 添加内边距

现在，我们为盒子添加内边距。

```css
#box1 {
    padding: 50px; /* 为四个方向添加50像素内边距 */
}
```

添加后你会发现两个现象：
1.  文字“Hello”向内移动了。
2.  **整个盒子的尺寸变大了**。

这是因为`width`和`height`属性定义的仅仅是**内容区域**的尺寸（500x500像素）。当我们添加`padding`时，这部分空间会**额外**增加到盒子的整体尺寸中。



![](img/71924e4b9b3b1f7cc4c0102bd8f11af_11.png)

![](img/71924e4b9b3b1f7cc4c0102bd8af11af_11.png)

使用浏览器的开发者工具（按F12）可以清晰地看到这个变化。在“Elements”标签页的“Styles”面板中，通常会有一个盒子模型的图示，明确标出了`content`、`padding`、`border`和`margin`的尺寸。

此时，盒子的总宽度计算方式为：
**总宽度 = 左边框宽度 + 左内边距 + 内容宽度 + 右内边距 + 右边框宽度**

![](img/71924e4b9b3b1f7cc4c0102bd8af11af_13.png)

代入我们的数值：
**总宽度 = 10px + 50px + 500px + 50px + 10px = 620px**



![](img/71924e4b9b3b1f7cc4c0102bd8af11af_15.png)

![](img/71924e4b9b3b1f7cc4c0102bd8af11af_15.png)

![](img/71924e4b9b3b1f7cc4c0102bd8af11af_17.png)

开发者工具中鼠标悬停在元素上时，显示的尺寸正是620px，验证了我们的计算。



![](img/71924e4b9b3b1f7cc4c0102bd8af11af_17.png)

### 添加外边距

接下来，我们添加外边距，并创建第二个盒子来观察效果。

```html
<style>
    #box1 {
        margin: 50px; /* 为四个方向添加50像素外边距 */
    }
    #box2 {
        width: 200px;
        height: 200px;
        background-color: pink;
    }
</style>
<body>
    <div id="box1">Hello</div>
    <div id="box2"></div>
</body>
```

添加`margin: 50px;`后，`box1`的四周会与其他元素（如浏览器窗口边缘或`box2`）产生50像素的间隔。`box2`会被`box1`的下外边距“推开”。



![](img/71924e4b9b3b1f7cc4c0102bd8af11af_13.png)

在开发者工具中，外边距通常用橙色或棕色区域表示。理解这一点很重要：**想让两个盒子之间产生间隔，应该使用`margin`（外边距），而不是`padding`（内边距）**。



### 单独设置各边距

我们也可以单独为盒子的某一个方向设置边距，而不是同时设置四个方向。

以下是相关的属性：

*   **单独设置内边距**：
    *   `padding-top`
    *   `padding-right`
    *   `padding-bottom`
    *   `padding-left`
*   **单独设置外边距**：
    *   `margin-top`
    *   `margin-right`
    *   `margin-bottom`
    *   `margin-left`

例如，如果只想让盒子的内容距离左边框有空白，可以这样写：
```css
#box1 {
    padding-left: 50px;
}
```
如果只想让`box2`与上方的`box1`拉开距离，可以这样写：
```css
#box2 {
    margin-top: 50px; /* 或者 #box1 { margin-bottom: 50px; } */
}
```

## 总结

本节课中我们一起学习了网页布局的基石——**盒子模型**。

我们了解到，每个HTML元素都可以看作一个盒子，它由内至外包括**内容**、**内边距**、**边框**和**外边距**四部分。其中：
*   `width`和`height`属性定义的是**内容区域**的大小。
*   `padding`（内边距）是内容与边框之间的空间，会增加盒子的整体尺寸。
*   `margin`（外边距）是盒子与其他元素之间的空间，用于控制元素间隔。
*   计算一个元素在页面上占据的总空间时，必须将内容、内边距、边框和外边距全部考虑在内。
