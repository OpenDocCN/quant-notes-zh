## 夏普比率的定义与意义
上一节我们介绍了投资中的风险概念，本节中我们来看看如何量化地衡量风险与收益的关系。

夏普比率描述的是，在承担单位风险的情况下，所能获得的超额收益是多少。这个指标越高，意味着对于每单位风险，投资者获得的补偿（收益）就越高。因此，在比较不同的投资标的或策略时，夏普比率是一个重要的参考依据。

为了便于理解，我们可以将其类比为一个“性价比”指标。就像一份高风险、高报酬的工作（例如战地雇佣兵）需要评估其风险是否值得一样，投资也需要评估其风险与收益是否匹配。

## 夏普比率的计算公式
理解了夏普比率的意义后，我们来看看如何具体计算它。

夏普比率的计算基于两个核心概念：投资组合的超额收益和其收益的波动性（风险）。其标准公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp** 代表投资组合的平均收益率。
*   **Rf** 代表无风险收益率（例如国债利率）。
*   **σp** 代表投资组合收益率的标准差，用于衡量风险。

公式的含义是：用投资组合超越无风险收益的那部分收益（超额收益），除以组合自身的波动风险。结果数值越大，说明该投资组合的“风险收益性价比”越高。

## 夏普比率的计算实践
理论需要结合实践，接下来我们通过Python代码演示如何计算一组股票数据的夏普比率。

![](img/b1ac38f9065546f55477e4a576a41e55_7.png)

以下是计算夏普比率的关键步骤：
1.  获取投资标的的历史收益率数据。
2.  设定一个年化无风险收益率（例如5%）。
3.  计算投资标的的平均日收益率。
4.  计算日收益率的标准差。
5.  将日度数据年化，并代入公式进行计算。

![](img/b1ac38f9065546f55477e4a576a41e55_9.png)

```python
import numpy as np
import pandas as pd

# 假设 `returns` 是一个Pandas Series，包含了投资组合的日收益率序列
# 设定年化无风险收益率，例如5%
risk_free_rate = 0.05

# 计算平均日超额收益率 (投资组合日收益率 - 无风险日收益率)
excess_daily_returns = returns - risk_free_rate / 250  # 假设一年有250个交易日

# 计算夏普比率 (年化)
sharpe_ratio = np.sqrt(250) * (excess_daily_returns.mean() / excess_daily_returns.std())

![](img/b1ac38f9065546f55477e4a576a41e55_11.png)

print(f"计算出的夏普比率为: {sharpe_ratio:.4f}")
```

![](img/b1ac38f9065546f55477e4a576a41e55_13.png)

在执行类似计算后，我们可以得到不同股票或策略的夏普比率。例如，在一次模拟计算中，可能得到如下结果：
*   股票A夏普比率：0.85
*   股票B夏普比率：1.32
*   股票C夏普比率：-0.15

![](img/b1ac38f9065546f55477e4a576a41e55_15.png)

根据“越高越好”的原则，在这三者中，**股票B** 的单位风险收益最高，是最优选择。而结果为负的股票C，意味着其收益尚不及无风险收益，通常不是好的投资选择。
