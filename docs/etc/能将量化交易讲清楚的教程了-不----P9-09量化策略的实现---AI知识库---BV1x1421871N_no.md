## 矩阵的特征值与特征向量

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_11.png)

上一节我们回顾了课程的整体安排。本节中，我们来看看量化策略背后的一个重要数学基础：矩阵的特征值与特征向量。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_13.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_15.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_17.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_18.png)

一个矩阵可以看作一个线性变换。对于一个N×N的方阵A，它把一个N维向量x变换为另一个N维向量。特征值和特征向量是刻画这个矩阵本质性质的工具。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_20.png)

**定义**：对于一个给定的方阵A，如果存在一个非零向量x和一个标量λ，满足以下等式：
`A * x = λ * x`
那么，λ称为矩阵A的**特征值**，x称为对应的**特征向量**。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_22.png)

这个等式的几何意义是：矩阵A作用于特征向量x后，结果向量仍然保持在x所在的直线上，只是长度缩放为原来的λ倍。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_24.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_25.png)

**计算方法**：
根据定义 `A*x = λ*x`，可以推导出：
`(A - λI) * x = 0`
其中I是单位矩阵。要使非零向量x满足此方程，矩阵`(A - λI)`必须是不可逆的，即其行列式必须为零：
`det(A - λI) = 0`
解这个关于λ的N次方程，可以得到N个特征值（可能是实数或复数）。对于每一个求得的特征值λ_i，代入方程`(A - λ_i I) * x = 0`，即可解出对应的特征向量x_i。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_27.png)

## 对称矩阵与奇异值分解（SVD）

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_32.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_34.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_35.png)

理解了特征值的基本概念后，我们来看一类特殊的矩阵——对称矩阵，并引出更强大的工具SVD。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_37.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_39.png)

对称矩阵满足 `A = A^T`（即矩阵等于其转置）。对称矩阵有两个非常好的性质：
1.  其特征值全部是**实数**。
2.  其不同特征值对应的特征向量是**互相垂直（正交）**的。

对于对称矩阵A，我们可以将其进行“谱分解”：
`A = Q * Σ * Q^T`
其中，Q的列是A的正交特征向量，Σ是一个对角矩阵，对角线上的元素是A的特征值。Q是正交矩阵，满足 `Q^T = Q^{-1}`。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_41.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_43.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_45.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_46.png)

然而，实际应用中遇到的矩阵往往不是对称的。对于任意矩阵A（甚至是非方阵），我们可以使用**奇异值分解（SVD）**。SVD是线性代数中一个极其重要的工具，应用广泛。

**SVD定义**：
对于任意一个M×N的实数矩阵A，都可以分解为三个矩阵的乘积：
`A = U * Σ * V^T`
其中：
*   U是一个M×M的正交矩阵，其列向量称为A的**左奇异向量**。
*   Σ是一个M×N的矩形对角矩阵，对角线上的元素称为**奇异值**，通常按从大到小排列。奇异值是A^T * A（或A * A^T）特征值的非负平方根。
*   V是一个N×N的正交矩阵，其列向量称为A的**右奇异向量**。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_48.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_50.png)

**SVD的直观意义与计算**：
SVD可以理解为：我们寻找一组正交基（U的列），使得矩阵A在这组基上的变换结果（A*U）仍然是正交的（即V的列）。这些变换后的向量长度就是奇异值。
一个简单的计算思路是：
1.  计算 `A^T * A`，这是一个对称矩阵。
2.  对这个对称矩阵进行谱分解，得到特征值和特征向量。这些特征向量就是SVD中的V，特征值的平方根就是奇异值。
3.  类似地，可以通过 `A * A^T` 得到U。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_52.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_54.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_55.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_56.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_57.png)

SVD在数据降维（如主成分分析PCA）、推荐系统、图像压缩等领域都有核心应用。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_59.png)

## 在量化投资中的应用：资产组合权重分配

掌握了SVD这一工具后，我们来看看它在量化投资中的一个具体应用：如何分配资金以最小化投资组合的风险。

假设我们选择了N只股票进行投资。在每一个时刻t，我们可以计算每只股票的收益率，形成一个N维的收益率向量。通过回溯历史数据（例如过去M天），我们可以得到M个这样的观测样本。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_68.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_69.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_70.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_72.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_74.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_75.png)

基于这些样本，我们可以计算两样东西：
1.  **平均收益率向量**：一个N维向量，表示每只股票的平均收益。
2.  **协方差矩阵Σ**：一个N×N的对称矩阵。矩阵元素Σ_ij表示股票i和股票j收益率之间的协方差，对角线元素Σ_ii则是股票i自身的方差。**协方差矩阵衡量了投资组合的整体风险结构**。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_77.png)

我们的目标是决定一个投资权重向量 `w = [w1, w2, ..., wN]^T`，其中wi表示分配给第i只股票的资金比例，且满足 `w1 + w2 + ... + wN = 1`。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_79.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_80.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_82.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_84.png)

现代投资组合理论指出，该投资组合的整体风险（方差）可以表示为：
`风险 = w^T * Σ * w`
我们的目标就是找到权重w，在权重之和为1的约束下，最小化这个风险。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_86.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_88.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_90.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_92.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_93.png)

**方法一：基于特征值的风险最小化**
一个美妙的结论是：**使组合风险最小化的权重w，恰好是协方差矩阵Σ的最小特征值所对应的特征向量**（需经过归一化处理，使权重之和为1）。最小特征值本身则对应了该权重下的最小风险值。
`最小风险 = λ_min`
`最优权重 w* = 对应 λ_min 的特征向量 (归一化后)`
在实际操作中，我们通常不选择最小特征值对应的方向，因为它往往代表了整个市场的系统性风险。而是选择第二小或更靠后的特征值对应的特征向量，这些方向代表了与市场关联度较低的风险因子，可能获得更好的风险调整后收益。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_95.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_97.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_98.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_99.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_101.png)

**方法二：基于优化问题的风险最小化**
我们也可以将问题形式化为一个带约束的优化问题：
```
最小化：w^T * Σ * w
约束条件：sum(w_i) = 1
```
使用拉格朗日乘子法可以直接求解，得到解析解：
`w* = (Σ^{-1} * 1) / (1^T * Σ^{-1} * 1)`
其中1是元素全为1的列向量。这种方法称为**最小方差组合**。

## 进阶策略：基于随机矩阵理论的滤波方法

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_119.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_121.png)

上述两种方法都直接使用了历史数据计算出的原始协方差矩阵Σ。然而，金融时间序列中存在大量噪声，原始的Σ可能包含许多随机性成分。我们可以利用**随机矩阵理论**对Σ进行滤波，提升其稳定性。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_123.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_124.png)

随机矩阵理论研究矩阵元素为随机变量时，矩阵特征值的分布规律。对于一个纯粹的随机矩阵（元素来自特定分布），其特征值分布服从已知的规律（如Marchenko-Pastur分布）。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_126.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_128.png)

**滤波算法步骤**：
1.  对历史收益率数据计算原始协方差矩阵Σ。
2.  对Σ进行谱分解，得到其特征值 {λ_i} 和特征向量。
3.  根据随机矩阵理论，将落在“典型随机噪声区间”（如Marchenko-Pastur分布的主要支撑集）内的特征值λ_i**置为零**。这些特征值对应的方向被认为是主要由噪声驱动。
4.  使用修改后的特征值（噪声特征值置零）和原来的特征向量，**重构**一个新的协方差矩阵Σ_filtered。
    `Σ_filtered = Q * diag(λ'_1, λ'_2, ...) * Q^T`，其中λ'_i是滤波后的特征值。
5.  将滤波后的协方差矩阵Σ_filtered代入上述**方法一**或**方法二**，计算最优投资权重w*。

这种方法的核心思想是：剔除协方差矩阵中由随机噪声贡献的成分，保留其真实的、稳定的结构信息，从而得到更稳健的资产配置权重。回测结果表明，经过滤波的协方差矩阵往往能带来更好的投资组合表现。

**论文一：基于深度神经网络的高频交易预测**
*   **核心内容**：该研究使用5分钟级别的期货数据，构建了一个高达9895维的特征向量，并利用深度神经网络进行预测。
*   **可借鉴点**：采用了**小批量梯度下降（Mini-batch SGD）** 来处理海量高维数据，这是训练大规模模型的常用技巧。
*   **批判性思考**：论文中使用多核CPU训练了10小时，但从效率角度看，使用GPU（如NVIDIA Tesla）进行并行加速会是更优选择。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_160.png)

**论文二：结合市场情绪分析的预测模型**
*   **核心内容**：研究探索了市场情绪（通过Twitter和Google搜索中“牛市”、“熊市”等关键词的频率来衡量）对股市未来走势的预测能力。
*   **研究结论**：1）市场情绪指标具有预测能力；2）Twitter数据比Google搜索数据更准确；3）Twitter情绪能影响Google搜索趋势；4）该模型不适用于中国股市（因Twitter使用受限）。
*   **学习要点**：这篇论文展示了如何将**非结构化数据（文本、搜索）** 转化为量化因子，并严谨地验证其有效性。这是量化研究的一个前沿方向。

## 总结

本节课我们一起学习了量化策略实现的关键数学工具与核心金融应用。

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_178.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_180.png)

![](img/d7fd2d88c8c539f9d54f12c63bc6a1cb_184.png)

1.  **数学基础**：我们回顾了矩阵特征值/特征向量的定义与意义，重点介绍了对称矩阵的谱分解和任意矩阵的奇异值分解（SVD）。SVD是理解数据结构和降维的基石。
2.  **金融应用**：我们将协方差矩阵作为风险度量工具，学习了两种最小化投资组合风险的方法：一是利用协方差矩阵**最小特征值对应的特征向量**确定权重；二是通过求解带约束的**最优化问题**得到最小方差组合。
3.  **进阶策略**：引入了**随机矩阵理论**，通过滤波方法剔除协方差矩阵中的噪声成分，从而获得更稳健的资产配置权重，这是一种提升模型稳定性的有效技术。
