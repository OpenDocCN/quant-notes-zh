# 机器学习实战：P22：判别分析 📊

在本节课中，我们将学习线性判别分析（LDA）和二次判别分析（QDA）。我们将从判别函数的概念入手，逐步推导LDA和QDA的数学原理，并比较它们与逻辑回归的异同，帮助你理解不同分类器的适用场景。

## 判别函数与决策面

上一节我们介绍了逻辑回归，本节我们来看看另一种线性分类器——线性判别分析。在介绍LDA之前，需要先理解判别函数的概念。

对于一个分类器，它会为每个类别分配一个判别函数。判别函数用于判定新样本是否属于该类别。假设有K个类别，就有K个判别函数。对于一个新样本点x，我们将其代入所有K个判别函数中。如果第i个判别函数的值最大，则说明该样本属于第i类。

判别函数δ_k(x)与对应的后验概率P(Y=k|X=x)通常呈单调递增关系。因此，判别函数值最大，意味着该类的后验概率最高，我们就有最大的置信度将该样本归为此类。

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_1.png)

当两个类别（如第k类和第l类）的判别函数值相等时，即δ_k(x) = δ_l(x)，满足此条件的点的集合会形成一个分类边界，我们称之为决策面。在决策面上，样本属于两类别的概率相同。

## 从线性回归到线性判别分析

## 后验概率与决策面

判别函数与后验概率满足单调函数关系。在LDA中，我们假设经过单调函数变换后，结果是x的线性函数，因此得到的决策面也是线性的，所以LDA是一个线性分类器。

以之前提到的逻辑回归为例。对于一个二分类问题（类别0和类别1），其后验概率分别为h(x)和1-h(x)，其中h是假设函数。逻辑回归使用Sigmoid函数：
`h(x) = 1 / (1 + exp(-wᵀx))`
我们对两类的概率比取对数：
`log(P/(1-P)) = wᵀx`
可以发现，`log(P/(1-P))`是x的线性函数。如果样本属于第一类，则P/(1-P) > 1，对数大于0；反之则小于0。因此，逻辑回归的决策面就是`wᵀx = 0`。在二维空间这是一条直线，在三维空间是一个平面，更高维则称为超平面。

## 线性判别分析（LDA）的原理

LDA与逻辑回归的最大区别在于其后验概率的计算方式。逻辑回归直接假设后验分布为Sigmoid函数。而LDA则使用贝叶斯定理计算后验概率`P(Y=k|X=x)`。

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_9.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_9.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_11.png)

根据贝叶斯公式：
`P(Y=k|X=x) = (π_k * f_k(x)) / (∑_j π_j * f_j(x))`
其中：
*   `π_k`是第k类的先验概率。
*   `f_k(x)`是特征x在第k类下的条件概率密度。

LDA假设`f_k(x)`服从多元高斯分布（正态分布）。这是LDA与逻辑回归的核心区别：**LDA假设了特征x的分布形态（高斯分布），而逻辑回归没有**。

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_11.png)

LDA进一步假设：**不同类别的高斯分布具有不同的均值μ_k，但拥有相同的协方差矩阵Σ**。
而二次判别分析（QDA）则放松了这个假设，允许不同类别有自己的协方差矩阵Σ_k。

为什么协方差相同就是线性，不同就是二次呢？下面的数学推导将揭示原因。

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_13.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_13.png)

### LDA的数学推导

首先写出第k类下x的高斯分布概率密度函数（因假设协方差相同，故省略下标k）：
`f_k(x) = (1/((2π)^(p/2)|Σ|^(1/2))) * exp(-1/2 (x-μ_k)ᵀ Σ⁻¹ (x-μ_k))`

与逻辑回归类似，我们通过比较两类后验概率的比值来做决策。对比值取对数（log）可以简化计算，消去指数项。

计算第k类与第l类的对数概率比：
`log(P(Y=k|X=x) / P(Y=l|X=x)) = log(π_k/π_l) + log(f_k(x)/f_l(x))`

将高斯密度函数代入并化简后，得到：
`δ_k(x) = log(π_k) - 1/2 μ_kᵀ Σ⁻¹ μ_k + xᵀ Σ⁻¹ μ_k`
（注：推导中，由于假设Σ_k = Σ_l = Σ，关于x的二次项`xᵀΣ⁻¹x`在相减时被消去。）

**判别函数δ_k(x)关于x是线性的**，这正是“线性”判别分析名称的由来。对于新样本x，我们将其代入所有K个判别函数，取值最大的那个类别就是预测结果。

要计算δ_k(x)，需要估计三个参数：
1.  **π_k**：第k类的先验概率。
2.  **μ_k**：第k类特征分布的均值向量。
3.  **Σ**：共同的协方差矩阵。

以下是它们的估计方法：
*   `π_k`：用样本中第k类的比例估计。`π_k ≈ N_k / N`，其中N_k是第k类的样本数，N是总样本数。
*   `μ_k`：用第k类样本的均值向量估计。`μ_k ≈ (1/N_k) ∑_(i:y_i=k) x_i`
*   `Σ`：用所有类的样本的加权协方差矩阵估计。`Σ ≈ (1/(N-K)) ∑_k ∑_(i:y_i=k) (x_i - μ_k)(x_i - μ_k)ᵀ`

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_15.png)

### 二次判别分析（QDA）的推导

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_15.png)

QDA与LDA的唯一区别在于，它假设不同类别的高斯分布具有不同的协方差矩阵Σ_k。

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_17.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_17.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_19.png)

在推导判别函数时，由于Σ_k不同，关于x的二次项`xᵀ Σ_k⁻¹ x`无法在类间比较时消去。

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_19.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_21.png)

最终得到的判别函数形式为：
`δ_k(x) = log(π_k) - 1/2 log|Σ_k| - 1/2 (x-μ_k)ᵀ Σ_k⁻¹ (x-μ_k)`
此时，**判别函数δ_k(x)关于x是一个二次函数**，因此得名“二次”判别分析。

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_23.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_21.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_25.png)

## 逻辑回归与判别分析的比较

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_27.png)

通过以上推导，我们可以总结逻辑回归与判别分析（LDA/QDA）的主要区别：

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_29.png)

1.  **模型假设不同**
    *   **判别分析**：利用贝叶斯定理得到后验概率，并**假设特征X的条件分布f_k(x)是高斯分布**。
        ![](img/46d4db67bc7e32f3f4721d7e9750d7e5_23.png)
        ![](img/46d4db67bc7e32f3f4721d7e9750d7e5_25.png)
    *   **逻辑回归**：**直接假设后验分布P(Y|X)为Sigmoid函数**，没有对X的分布做任何假设。
        ![](img/46d4db67bc7e32f3f4721d7e9750d7e5_27.png)
        ![](img/46d4db67bc7e32f3f4721d7e9750d7e5_29.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_31.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_33.png)

2.  **参数估计方法不同**
    *   **判别分析**：基于联合分布`P(X, Y)`进行**全局最大似然估计**。
        ![](img/46d4db67bc7e32f3f4721d7e9750d7e5_31.png)
        ![](img/46d4db67bc7e32f3f4721d7e9750d7e5_33.png)
    *   **逻辑回归**：基于条件分布`P(Y|X)`进行**条件最大似然估计**。
        ![](img/46d4db67bc7e32f3f4721d7e9750d7e5_35.png)
        ![](img/46d4db67bc7e32f3f4721d7e9750d7e5_37.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_35.png)

3.  **偏差-方差权衡**
    *   **逻辑回归**：**偏差更低（更无偏）**。因为它对数据分布假设更少，模型更灵活，结果更依赖于数据本身。
    *   **判别分析**：**方差更低（更稳健）**。因为它引入了“X服从高斯分布”的先验知识。当训练数据质量不高、噪声较大或样本量较小时，这个先验知识可以起到正则化作用，使模型表现更稳定。

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_39.png)

![](img/46d4db67bc7e32f3f4721d7e9750d7e5_37.png)

**如何选择？**
*   如果数据质量较好，样本量充足，且追求更无偏的估计，逻辑回归是很好的选择。
*   如果数据噪声较大、样本量有限，或者有理由相信特征近似服从高斯分布，那么判别分析（LDA/QDA）可能表现更稳健。

## 总结

本节课我们一起学习了判别分析的核心内容。

我们首先引入了判别函数和决策面的概念，理解了分类器如何做出决策。然后，我们深入探讨了线性判别分析（LDA）的数学原理，从贝叶斯定理出发，通过假设特征服从同方差的高斯分布，推导出其判别函数是线性的。接着，我们放松方差假设，得到了二次判别函数，即二次判别分析（QDA）。最后，我们系统地比较了判别分析与逻辑回归在模型假设、估计方法和统计特性上的关键区别。

理解这些分类器背后的假设和原理，对于在实际问题中正确选择和使用它们至关重要。
