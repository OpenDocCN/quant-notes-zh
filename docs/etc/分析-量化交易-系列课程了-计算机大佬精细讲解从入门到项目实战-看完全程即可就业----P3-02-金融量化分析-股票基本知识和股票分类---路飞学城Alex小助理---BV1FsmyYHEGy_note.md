在本节课中，我们将要学习股票的基础知识，包括股票的定义、作用、上市过程以及分类方式。理解这些概念是进行金融量化分析的第一步。

## 股票的定义

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

我们可以形象地解释一下股票是什么。例如，一位创业者需要资金来运营公司，但他自己可能没有足够的钱。这时，他可以向投资者募集资金。投资者将钱投入公司，作为交换，公司会给予投资者相应比例的股票。这张股票就是投资者出资的证明，也代表了投资者在公司中的所有权份额。

假设一家公司由五位出资人共同创立，每人出资1亿元，那么公司总市值就是5亿元。每位出资人出资1亿元，因此每人可以获得公司20%的股票。计算公式如下：
```
个人持股比例 = 个人出资额 / 公司总股本
```
在这个例子中，`1亿 / 5亿 = 20%`。

## 股票的作用

上一节我们介绍了股票是股东身份的凭证，本节中我们来看看股票具体有哪些作用。

股票主要有两个核心作用。

**第一，证明股东身份与权利。** 持有股票意味着你是公司的所有者之一。作为股东，你拥有参与公司重大决策的权利，例如在股东大会上投票。

**第二，获取收益。** 这是大多数投资者最关心的部分。通过持有股票获利主要有两种方式：

以下是两种主要的获利方式：
1.  **公司分红**：当公司盈利时，可能会将部分利润以现金形式分配给股东。如果你持有公司20%的股份，公司今年净利润为5000万元并决定全部分红，那么你将获得 `5000万 * 20% = 1000万元` 的分红。
2.  **交易获利（资本利得）**：你可以将股票在证券交易所卖给其他投资者。如果公司的价值增长，你买入股票的价格低于卖出的价格，中间的差价就是你的利润。例如，你以1亿元购入公司20%的股份，后来公司市值涨到50亿元，你持有的股份价值相应变为 `50亿 * 20% = 10亿元`。此时你若卖出全部股份，即可获利9亿元。

对于资金量较小的普通股民而言，逻辑完全相同。只是因为持股比例极小，分红收入相对较少，其主要获利方式通常是通过买卖股票赚取差价。

## 什么是上市

了解了股票的基本作用后，一个自然的问题是：普通投资者如何买卖股票？这就引出了“上市”的概念。

所谓上市，就是企业通过证券交易所，首次公开向全社会投资者增发股票以募集资金的过程。

![](img/63b99c1a15550c876b1957e5ac67b970_1.png)

公司不能随意向公众募集资金，那属于非法集资。公司需要达到一定标准，向证监会提出申请，并经过严格的审核（包括财务状况、经营风险等）。证监会确认公司资质良好、风险可控后，才会批准其上市。

上市后，公司的股票就可以在证券交易所挂牌交易。所有开通了股票账户的投资者都可以自由买卖这只股票。IPO即首次公开募股，指的就是公司第一次上市并发行股票的行为。

## 股票的分类

现在我们已经知道公司通过上市进入公开市场，接下来我们看看市场上的股票有哪些常见的分类方式。

股票主要有两种分类维度。

**第一种，按业绩分类。** 这是根据公司的经营表现来划分的。

以下是三种常见的按业绩分类的股票：
*   **蓝筹股**：指资本雄厚、信誉优良的大型公司股票。例如中国的石油、石化、银行等巨头企业。它们像赌场中最值钱的蓝色筹码，体量大，经营稳定。
*   **绩优股**：指业绩优良公司的股票。它们可能规模不如蓝筹股庞大，但盈利能力持续强劲。例如一些长期保持高增长的消费或科技公司。
*   **ST股**：中文称“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记，以警示投资者该公司存在较高风险。

**第二种，按上市地区分类。** 这是根据股票上市交易的地点来划分的。

以下是几种主要的按上市地区分类的股票：
*   **A股**：在中国大陆（上海、深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。
*   **H股**：在中国香港上市的公司股票。
*   **N股**：在美国纽约上市的公司股票。
*   **S股**：在新加坡上市的公司股票。

不同市场的交易规则也不同。这里重点介绍A股的两条核心规则：
1.  **涨跌幅限制**：A股股票每日价格涨跌幅度不得超过前一日收盘价的10%（科创板、创业板等部分板块除外）。例如，昨日收盘价100元的股票，今日最高价不得超过110元，最低价不得低于90元。此规则旨在平抑市场波动，保护投资者。
2.  **T+1交割制度**：“T”代表交易当天。T+1意味着**当天买入的股票，最快要到下一个交易日才能卖出**。与之相对，港股、美股等多实行T+0制度，即当天可多次买卖。A股的T+1制度是为了降低市场的投机性。
