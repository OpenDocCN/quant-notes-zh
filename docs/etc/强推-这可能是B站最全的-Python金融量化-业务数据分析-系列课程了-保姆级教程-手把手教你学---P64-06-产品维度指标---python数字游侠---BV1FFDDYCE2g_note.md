# Python金融量化与业务数据分析：P64：06 产品维度指标 📊

在本节课中，我们将要学习产品数据维度的核心指标。上一节我们介绍了用户和行为数据，本节中我们来看看如何衡量产品的表现，特别是与销售和投资回报相关的关键指标。

## GMV：成交总额 💰

首先，我们介绍一个非常常用的业内指标：**GMV**，即成交总额。在零售业中，它也被称为“流水”。

GMV的计算**不仅仅包含销售额**，它还包含了取消订单的金额、拒收订单的金额以及退款订单的金额。这意味着，只要用户产生了下单行为，无论最终是否成功支付或收货，其订单金额都会被计入GMV。

![](img/64e0ce6fa9e5e922322505e9d77f5577_1.png)

**公式**：`GMV = 销售额 + 取消订单金额 + 拒收订单金额 + 退款订单金额`

![](img/64e0ce6fa9e5e922322505e9d77f5577_3.png)

这个指标反映了用户与产品之间产生的所有关系。例如，用户下单后退货，虽然未产生实际价值，但依然算作一次用户与产品的互动。

![](img/64e0ce6fa9e5e922322505e9d77f5577_5.png)

## ARPU与ARPPU：用户价值指标 👥

接下来，我们需要记住两个重要的用户价值指标：**ARPU**和**ARPPU**。

*   **ARPU**：平均每用户收入。衡量在特定时期内，从每个用户（包括活跃和非付费用户）身上获得的平均收入。
*   **ARPPU**：平均每付费用户收入。衡量在特定时期内，从每个**实际付费用户**身上获得的平均收入。

这两个指标有助于我们更精细地分析用户群体的价值贡献。

## SKU与SPU：商品管理标识 🏷️

![](img/64e0ce6fa9e5e922322505e9d77f5577_7.png)

在零售行业，商品管理常用到**SKU**和**SPU**这两个概念。

以下是它们的核心区别：
*   **SPU**：标准化产品单元。是商品信息聚合的最小单位，通常指一款产品。例如，“iPhone 15”就是一个SPU。
*   **SKU**：库存保有单位。是识别产品的唯一标识符，基于SPU的不同属性（如颜色、尺寸、配置）产生。例如，“iPhone 15 256GB 黑色”就是一个具体的SKU。

简单理解：**一个SPU（如一款衣服）可以对应多个SKU（如该款衣服的红色S码、蓝色M码等）**。

## ROI：投资回报率 📈

**ROI**是一个对公司至关重要的指标，全称为投资回报率。它衡量的是从投入成本到最终产生收益的综合效率。

![](img/64e0ce6fa9e5e922322505e9d77f5577_9.png)

以在线教育行业为例，公司前期在广告上投入大量成本进行“拉新”，目标是吸引用户并最终转化为付费客户。ROI就是衡量这部分投入能带来多少回报的关键指标。

![](img/64e0ce6fa9e5e922322505e9d77f5577_11.png)

**公式**：`ROI = (产生交易的金额 / 活动投放成本) * 100%`

例如，投入100元广告费，最终带来了10位客户，共产生500元销售额，那么ROI就是500%。这个指标直接反映了营销或投资活动的有效性。

## 核心指标总结 📋

在本系列课程中，我们需要掌握的核心指标如下：

**用户维度指标**
*   日活跃用户数
*   日活跃用户率
*   新增用户数
*   留存率
*   客单价

**行为维度指标**
*   页面浏览量
*   独立访客数
*   跳出率
*   平均访问时长

**产品与商业维度指标**
*   成交总额
*   平均每用户收入 / 平均每付费用户收入
*   单位访客成本
*   投资回报率

![](img/64e0ce6fa9e5e922322505e9d77f5577_13.png)

掌握这些指标，是理解业务数据分析、进行金融量化研究的基础。本节课中我们一起学习了产品维度的关键指标，特别是GMV和ROI的计算与含义，它们是评估业务健康度和商业效率的核心工具。
