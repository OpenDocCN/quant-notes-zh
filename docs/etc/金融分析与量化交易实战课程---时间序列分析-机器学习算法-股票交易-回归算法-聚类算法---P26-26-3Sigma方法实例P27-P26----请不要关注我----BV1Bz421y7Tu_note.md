## 概述
在本节课中，我们将要学习一种名为“3-Sigma”的去极值方法。这种方法基于统计学中的正态分布原理，通过计算数据的均值和标准差，来识别并处理那些偏离正常范围的数据点。我们将通过一个简单的实例，一步步理解其原理并实现它。

## 3-Sigma方法原理

![](img/84359cf60db81f417aafdde21cc6d022_2.png)

上一节我们介绍了去极值的概念，本节中我们来看看基于统计学的3-Sigma方法。

其核心思想源于正态分布。在正态分布中，数据大多集中在均值附近，离均值越远的数据点出现的概率越低。我们可以利用这一特性来定义数据的正常范围。

![](img/84359cf60db81f417aafdde21cc6d022_4.png)

具体来说，对于一个服从正态分布的数据集，我们可以计算其均值（μ）和标准差（σ）。根据“经验法则”：
*   数据落在 **μ ± 1σ** 范围内的概率约为 **68.3%**。
*   数据落在 **μ ± 2σ** 范围内的概率约为 **95.4%**。
*   数据落在 **μ ± 3σ** 范围内的概率约为 **99.7%**。

3-Sigma方法就是设定数据的上下限为 **μ ± 3σ**。这意味着我们认为99.7%的数据是“正常”的，而落在该范围之外的0.3%的数据则被视为“极值”或“离群点”。对于这些极值，我们不是直接删除，而是将其规范到边界值上。


![](img/84359cf60db81f417aafdde21cc6d022_12.png)

在函数内部，我们首先计算该列数据的均值和标准差。

![](img/84359cf60db81f417aafdde21cc6d022_16.png)

```python
    mean = series.mean()   # 计算均值 μ
    std = series.std()     # 计算标准差 σ
```

![](img/84359cf60db81f417aafdde21cc6d022_18.png)

![](img/84359cf60db81f417aafdde21cc6d022_20.png)

接着，根据均值和标准差计算数据的上下限。

![](img/84359cf60db81f417aafdde21cc6d022_22.png)

```python
    upper_limit = mean + n * std  # 上限 = μ + nσ
    lower_limit = mean - n * std  # 下限 = μ - nσ
```

![](img/84359cf60db81f417aafdde21cc6d022_24.png)

![](img/84359cf60db81f417aafdde21cc6d022_26.png)

最后，我们对原始数据进行处理：将所有大于上限的值设置为上限值，将所有小于下限的值设置为下限值，并返回处理后的数据。

![](img/84359cf60db81f417aafdde21cc6d022_28.png)

![](img/84359cf60db81f417aafdde21cc6d022_30.png)

```python
    series_clipped = series.clip(lower=lower_limit, upper=upper_limit)
```

## 总结
本节课中我们一起学习了3-Sigma去极值方法。我们了解到，去极值不是简单地删除数据，而是通过定义合理的上下限（如下限=μ - nσ，上限=μ + nσ），将超出范围的数据规范到边界内。这是数据预处理中非常重要的一步，能够有效减少极端值对后续分析模型造成的干扰。结合之前学习的方法，我们已经掌握了三种常用的去极值策略。
