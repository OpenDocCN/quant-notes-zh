在本节课中，我们将学习NumPy库中的通用函数。这些函数可以对数组进行快速的逐元素运算，是进行高效数值计算的基础。我们将从简单的数学运算开始，逐步介绍取整、特殊值处理以及二元运算等核心功能。

## 数组的通用数学运算

上一节我们介绍了数组的索引功能，本节中我们来看看NumPy提供的通用函数。首先，数组对象支持加减乘除等基本运算，这些运算可以批量进行。对于更复杂的数学运算，NumPy也提供了相应的通用函数。

例如，标准库中有一个取绝对值的函数 `abs`。我们希望对数组中的所有值取绝对值。

```python
import numpy as np
A = np.array([-5, -2, 0, 2, 5])
# 使用NumPy的abs函数
result = np.abs(A)
```

虽然直接使用Python内置的 `abs(A)` 在某些情况下也可能有效，但为了确保与NumPy数组的兼容性，建议使用 `np.abs`。

除了绝对值，NumPy还提供了平方根函数 `np.sqrt`。需要注意的是，对负数开方在数学上是不允许的，NumPy会返回一个特殊值 `nan`。

![](img/b5e2b235f3aff7b40a07bc621e4383e3_3.png)

```python
A = np.array([-5, -2, 0, 2, 5])
result = np.sqrt(A)  # 负数位置会得到 nan
```

*   `np.exp`: 计算指数。
*   `np.log`: 计算自然对数。
*   `np.log10`: 计算以10为底的对数。
*   `np.sin`, `np.cos`, `np.tan`: 三角函数。

## 四种取整方式

![](img/b5e2b235f3aff7b40a07bc621e4383e3_9.png)

在数据处理中，我们经常需要对小数进行取整。Python提供了多种取整方式，理解它们的区别很重要。

![](img/b5e2b235f3aff7b40a07bc621e4383e3_11.png)

假设有一个小数 `a = 1.6`，将其变为整数有几种方法：
*   `int(a)`: 结果是 `1`。这种方式是**向零取整**，即直接去掉小数部分。
*   `round(a)`: 结果是 `2`。这是**四舍五入**。

对于负数，例如 `a = -1.6`：
*   `int(a)`: 结果是 `-1`（向零取整）。
*   `round(-1.6)`: 结果是 `-2`（四舍五入）。

除了上述两种，还有两种取整方式：
*   **向上取整** (`np.ceil`): 总是朝着正无穷方向取整。
    *   `np.ceil(1.6)` -> `2`
    *   `np.ceil(-1.6)` -> `-1`
*   **向下取整** (`np.floor`): 总是朝着负无穷方向取整。
    *   `np.floor(1.6)` -> `1`
    *   `np.floor(-1.6)` -> `-2`

为了方便记忆，可以想象一个数轴，零在中间。`int()` 是朝着零的方向取整；`ceil()` 是朝着右边（正方向）取整；`floor()` 是朝着左边（负方向）取整；`round()` 是根据小数部分与0.5的比较决定方向。

NumPy提供了对应的函数，可以对整个数组进行批量取整操作：

```python
A = np.array([-2.5, -1.6, 0.0, 1.6, 2.5])
print(np.floor(A))   # 向下取整
print(np.ceil(A))    # 向上取整
print(np.round(A))   # 四舍五入
print(np.trunc(A))   # 向零取整 (等价于 int 的数组版本)
```

## 分离整数与小数部分

有时我们需要分别获取一个数字的整数部分和小数部分，可以使用 `np.modf` 函数。

```python
A = np.array([-2.7, -1.2, 0.0, 3.8])
integer_part, fractional_part = np.modf(A)
```
`np.modf` 函数返回一个元组，第一个数组是小数部分，第二个数组是整数部分。

## 特殊值：nan与inf

在进行数值计算时，可能会遇到一些未定义或无穷大的情况。NumPy用两个特殊的浮点数来表示它们：
*   **`nan`**: 表示“不是一个数字”（Not a Number），例如 `0.0 / 0.0` 或负数的平方根。
*   **`inf`**: 表示“无穷大”（infinity），例如 `5.0 / 0.0`。

使用它们可以避免因个别非法计算导致整个程序中断，这在处理大规模数据时非常有用。

```python
# 生成包含 nan 和 inf 的数组
A = np.array([0, 1, 2, 3, 4])
B = np.array([0, 2, 0, 4, 5])
C = A / B  # 会出现 0/0 (nan) 和 非零数/0 (inf)
```

需要注意的是，`nan` 有一个特殊属性：它不等于任何值，甚至不等于它自己。因此，不能使用 `==` 来判断一个值是否为 `nan`。

```python
value = np.nan
print(value == np.nan)  # 输出 False，这是错误的判断方式
print(value is np.nan)   # 可行，但不适用于数组
```

对于数组，必须使用NumPy提供的专用函数进行判断：
*   `np.isnan(array)`: 判断数组中的每个元素是否为 `nan`。
*   `np.isinf(array)`: 判断数组中的每个元素是否为 `inf`。

我们可以利用布尔索引来过滤掉这些特殊值：

```python
C = np.array([np.nan, 1.0, np.inf, 2.0, np.nan])
# 过滤掉 nan
valid_data = C[~np.isnan(C)]
# 过滤掉 inf
finite_data = C[~np.isinf(C)]
# 过滤掉 nan 和 inf
clean_data = C[np.isfinite(C)]  # np.isfinite 检查是否为有限数（既非nan也非inf）
```

## 二元通用函数

除了处理单个数组的一元函数，NumPy还提供了处理两个数组的二元通用函数。除了基本的加(`+`)、减(`-`)、乘(`*`)、除(`/`)、乘方(`**`)、取模(`%`)外，有两个函数特别有用：

*   `np.maximum(x1, x2)`: 逐元素比较两个数组，返回最大值。
*   `np.minimum(x1, x2)`: 逐元素比较两个数组，返回最小值。

```python
A = np.array([3, 7, 2, 8])
B = np.array([4, 2, 6, 8])
print(np.maximum(A, B))  # 输出：[4 7 6 8]
print(np.minimum(A, B))  # 输出：[3 2 2 8]
```
这与 `A.max()`（求数组A自身的最大值）是不同的。`np.maximum` 是在两个数组的对应位置之间进行比较。
