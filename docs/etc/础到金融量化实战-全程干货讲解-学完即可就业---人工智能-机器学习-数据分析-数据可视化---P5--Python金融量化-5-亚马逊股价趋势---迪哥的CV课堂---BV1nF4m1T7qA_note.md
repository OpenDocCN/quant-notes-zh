## 数据获取与基础建模

上一节我们介绍了如何使用Prophet模型进行股价预测。本节中，我们来看看如何将同样的方法应用于亚马逊的股票数据。

首先，我们导入必要的库并获取亚马逊（股票代码：AMZN）的历史股价数据。

```python
import pandas as pd
from prophet import Prophet
import yfinance as yf

# 下载亚马逊股票数据
stock = yf.download('AMZN')
```

亚马逊的股价数据跨度很长，最低价出现在1997年5月22日。在我们执行代码的日期（示例中为1月23日），其收盘价为1362.54美元。股价整体呈现上升趋势，但过程中存在明显的波动。

接下来，我们构建一个基础的Prophet预测模型。

```python
# 准备数据格式
df = stock[['Close']].reset_index()
df.columns = ['ds', 'y']

# 初始化并拟合模型
model = Prophet()
model.fit(df)

# 创建未来90天的预测数据框
future = model.make_future_dataframe(periods=90)
forecast = model.predict(future)
```

基础模型预测未来90天的股价走势相对平缓。例如，它预测在4月24日的股价约为1366美元。与真实股价可能存在的剧烈波动相比，这个模型显得更为“保守”。这种保守性有时是好事，因为它可能避免对数据中的噪声（离群点或突变点）过度拟合。

## 模型评估框架

仅仅做出预测是不够的，我们需要知道模型的好坏。因此，我们建立一个评估框架。评估主要关注两个方面：预测数值的误差，以及预测趋势方向的准确性。

以下是评估模型的具体步骤：

1.  **划分数据集**：我们将2014年至2017年的数据作为训练集，将2017年至2018年的数据作为测试集。测试集必须包含真实的股价数据以供对比。
2.  **训练与预测**：用训练集数据拟合模型，并预测测试集时间段（365天）的股价。
3.  **计算误差**：计算预测值与真实值之间的平均绝对误差（MAE）。这能告诉我们预测的数值平均偏差有多大。
    *   **公式**：`误差 = |真实值 - 预测值|`
    *   我们会分别计算训练集误差（`train_error`）和测试集误差（`test_error`）。
4.  **评估趋势方向**：我们不仅关心具体价格，还关心股价是涨是跌。通过比较相邻两天的价格差，我们可以判断趋势。
    *   如果 `今日价格 - 昨日价格 > 0`，则为上升趋势，记为1。
    *   如果 `今日价格 - 昨日价格 < 0`，则为下降趋势，记为0。
    *   然后，我们统计模型预测的趋势方向与真实趋势方向一致的百分比。

以下是实现评估的关键代码片段：

```python
# 计算趋势方向
df['真实趋势'] = np.sign(df['y'].diff()).map({1:1, -1:0})  # 上升为1，下降为0
forecast['预测趋势'] = np.sign(forecast['yhat'].diff()).map({1:1, -1:0})

# 计算趋势预测准确率
# 只选取测试集时间段的数据进行比较
test_mask = (df['ds'] >= '2017-01-24') & (df['ds'] <= '2018-01-23')
trend_accuracy = (df.loc[test_mask, '真实趋势'] == forecast.loc[test_mask, '预测趋势']).mean()

# 计算平均绝对误差
from sklearn.metrics import mean_absolute_error
train_error = mean_absolute_error(df[~test_mask]['y'], forecast[~test_mask]['yhat'])
test_error = mean_absolute_error(df[test_mask]['y'], forecast[test_mask]['yhat'])
```

运行评估后，我们得到了以下结果（数值为示例）：
*   训练集平均误差：约18美元
*   测试集平均误差：约160美元
*   上升趋势预测准确率：56%
*   下降趋势预测准确率：43.81%

这些指标说明，当前的基础模型在测试集上的预测误差较大，且对趋势方向的判断准确率接近随机猜测（50%），模型性能有巨大的提升空间。

## 结果可视化与分析

最后，我们将预测结果与真实数据可视化，以便直观对比。

```python
# 绘制预测结果
fig = model.plot(forecast)
# 在图上叠加真实测试集数据点
ax = fig.gca()
ax.plot(df.loc[test_mask, 'ds'], df.loc[test_mask, 'y'], 'ko', markersize=3, label='真实股价')
ax.legend()
```

在生成的图表中：
*   **黑色点**代表测试集时间段内的真实股价。
*   **深蓝色线**代表模型的预测值。
*   **红色竖线**标识了预测开始的起点（2017年1月）。

从图中可以清晰看到，模型的预测线（深蓝色）相对平缓，而真实股价（黑色点）波动剧烈且整体涨幅更高，两者存在显著差距。这直观地印证了评估指标反映的问题：模型未能有效捕捉股价的波动和强劲的上升趋势。
