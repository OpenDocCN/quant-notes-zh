## 因子打分法详解

上一节我们介绍了如何获取和预处理股票数据。本节中，我们来看看如何利用这些数据对股票进行综合评估，即“打分”。

当获取了股票的多个指标数据后，我们需要一个方法来综合这些指标，从而判断哪些股票更值得投资。打分法就是其中一种直观且常用的方法。

以下是打分法的基本步骤：

1.  **确定指标方向**：首先明确每个指标是“越大越好”还是“越小越好”。例如，收益率是越大越好，而负债率是越小越好。
2.  **划分数值区间**：为每个指标的数值划分区间。例如，将归一化后的值（0到1之间）划分为5个区间。
3.  **设计打分规则**：为每个区间赋予一个分数。对于“越大越好”的指标，数值大的区间得分高；对于“越小越好”的指标，数值小的区间得分高。
4.  **计算单项得分**：根据每只股票在每个指标上的实际值，确定其落入的区间，并得到该指标的得分。
5.  **计算综合总分**：将一只股票在所有指标上的得分相加，得到其综合总分。
6.  **排序与选股**：对所有股票的综合总分进行排序，选择排名最高的若干只股票作为投资组合。

## 打分法实例演示

![](img/3d32fc21586c9c5fc06abaaee30b57f6_5.png)

为了更清晰地理解，我们用一个简化的例子来演示这个过程。

假设我们有一个股票池（例如沪深300），我们关注四个指标：A、B、C、D。其中：
*   A和B是**越大越好**的指标。
*   C和D是**越小越好**的指标。

![](img/3d32fc21586c9c5fc06abaaee30b57f6_7.png)

![](img/3d32fc21586c9c5fc06abaaee30b57f6_9.png)

我们已经获取了两只股票（id1, id2）的指标数据，这些数据已经过归一化处理，范围在0到1之间。

| 股票ID | 指标A | 指标B | 指标C | 指标D |
| :--- | :--- | :--- | :--- | :--- |
| id1 | 0.71 | 0.28 | 0.39 | 0.11 |
| id2 | 0.45 | 0.17 | 0.81 | 0.02 |

### 第一步：划分区间与制定规则

![](img/3d32fc21586c9c5fc06abaaee30b57f6_11.png)

我们将0到1的数值范围平均划分为5个区间，并为“越大越好”和“越小越好”两类指标制定不同的打分规则。

以下是区间划分和对应的打分表：

| 数值区间 | “越大越好”指标得分 | “越小越好”指标得分 |
| :--- | :--- | :--- |
| [0.8, 1.0] | 5分 | 1分 |
| [0.6, 0.8) | 4分 | 2分 |
| [0.4, 0.6) | 3分 | 3分 |
| [0.2, 0.4) | 2分 | 4分 |
| [0.0, 0.2) | 1分 | 5分 |

**公式解释**：
*   对于指标 `X`，若其方向为 `越大越好`，则分数 `Score(X) = f(value)`，其中 `f` 是一个值越大、分数越高的映射函数（如上表左列）。
*   若方向为 `越小越好`，则 `Score(X) = g(value)`，其中 `g` 是一个值越小、分数越高的映射函数（如上表右列）。

### 第二步：计算每只股票的得分

现在，我们根据上表为每只股票的每个指标打分。

**对于 id1：**
*   指标A = 0.71 → 落入 [0.6, 0.8) → **4分** (越大越好)
*   指标B = 0.28 → 落入 [0.2, 0.4) → **2分** (越大越好)
*   指标C = 0.39 → 落入 [0.2, 0.4) → **4分** (越小越好)
*   指标D = 0.11 → 落入 [0.0, 0.2) → **5分** (越小越好)
*   **总分** = 4 + 2 + 4 + 5 = **15分**

**对于 id2：**
*   指标A = 0.45 → 落入 [0.4, 0.6) → **3分**
*   指标B = 0.17 → 落入 [0.0, 0.2) → **1分**
*   指标C = 0.81 → 落入 [0.8, 1.0] → **1分** (越小越好)
*   指标D = 0.02 → 落入 [0.0, 0.2) → **5分** (越小越好)
*   **总分** = 3 + 1 + 1 + 5 = **10分**

![](img/3d32fc21586c9c5fc06abaaee30b57f6_13.png)

### 第三步：排序与选股

计算完所有股票（本例中为id1和id2）的总分后，我们进行排序：
1.  id1: 15分
2.  id2: 10分

![](img/3d32fc21586c9c5fc06abaaee30b57f6_15.png)

如果我们的策略是选择排名前1的股票，那么本次调仓就会选择 **id1**。

## 打分法的其他形式与策略整体流程

![](img/3d32fc21586c9c5fc06abaaee30b57f6_21.png)

![](img/3d32fc21586c9c5fc06abaaee30b57f6_23.png)

除了上述按固定区间打分的方法，还有其他实现方式。例如，可以直接根据指标值在全体股票中的排名（百分位）来打分。如果某指标“越大越好”，那么排名第一的股票直接得最高分（如300分），排名最后的得最低分（如1分）。核心目标是一致的：**得到一个可比较的综合评分，用于排序选股**。

接下来，我们梳理一下将打分法嵌入一个完整量化策略的流程。

以下是基于月度调仓的打分法策略核心步骤：

1.  **确定股票池**：首先定义策略操作的股票范围，例如沪深300指数成分股。
2.  **设置调仓周期**：设定一个定期调仓的触发器，例如每月第一个交易日执行调仓函数。
3.  **实现调仓函数**：这是策略的核心，在该函数中执行以下操作：
    *   **数据获取与预处理**：获取当前股票池内所有股票的所需指标数据，并进行清洗、归一化等处理。
    *   **因子打分**：应用本节课讲解的打分法，为每只股票计算综合总分。
        ```python
        # 伪代码示例
        def calculate_score(stock_data, indicators):
            total_scores = {}
            for stock_id, values in stock_data.items():
                score = 0
                for indicator, value in values.items():
                    # 根据指标方向和值计算单项得分
                    single_score = get_single_score(indicator, value)
                    score += single_score
                total_scores[stock_id] = score
            return total_scores
        ```
    *   **生成交易清单**：根据总分排序，选择排名前N（例如前10）的股票作为本期拟持有的股票。
    *   **执行调仓**：卖出不在新清单中的股票，买入清单中新出现的股票，使投资组合与清单一致。

这个流程相对清晰，接下来我们就可以在实战中尝试这种打分法，看看仅凭几个简单的指标，能否构建出具有超额收益的策略。
