## 矩阵的特征值与奇异值分解（SVD）

上一节我们介绍了课程的整体安排和行业背景，本节中我们来看看支撑量化风控模型的核心数学工具。

### 矩阵作为线性变换
一个矩阵可以看作一个线性变换函数。例如，一个 `100x100` 的矩阵 `A` 乘以一个 `100x1` 的向量 `x`，会得到另一个 `100x1` 的向量。这相当于将一个100维向量映射到另一个100维向量。

### 特征值与特征向量的定义
对于一个给定的方阵 `A`，特征值和特征向量是满足以下等式的标量 `λ` 和非零向量 `x`：
**公式：** `Ax = λx`
其含义是，矩阵 `A` 作用在特征向量 `x` 上，仅使其长度伸缩 `λ` 倍，方向保持不变。求解特征值即解方程 `det(A - λI) = 0`。

### 对称矩阵的特殊性质
在实际应用中，我们特别关注**对称矩阵**（`A = A^T`）。对称矩阵有两个重要性质：
1.  其特征值均为实数。
2.  其不同特征值对应的特征向量相互正交。

### 矩阵的对角化与谱定理
对于可对角化的矩阵 `A`，可以分解为：
**公式：** `A = SΛS^(-1)`
其中，`S` 的列是特征向量，`Λ` 是对角矩阵，其对角线元素为特征值。
对于对称矩阵，由于其特征向量正交，可进一步简化为更优美的形式（谱定理）：
**公式：** `A = QΛQ^T`
其中，`Q` 是正交矩阵（`Q^T = Q^(-1)`）。

### 奇异值分解（SVD）
对于任意矩阵 `A`（不一定是方阵），都可以进行奇异值分解，这是更通用的工具。
**公式：** `A = UΣV^T`
其中，`U` 和 `V` 是正交矩阵，`Σ` 是对角矩阵（对角线元素称为奇异值）。SVD的求解可通过计算 `A^T A` 和 `A A^T` 的特征分解来实现。SVD在降维（如主成分分析PCA）和信号去噪等领域有广泛应用。

## 资产组合理论与权重优化

理解了矩阵分解工具后，我们将其应用于量化交易的核心问题：如何分配资金以控制风险。

### 问题定义
假设我们选取了 N 只股票，在时刻 t，每只股票的收益率为 `R_i(t)`。通过历史数据，我们可以计算出这 N 个收益率的：
*   **均值向量** `μ` （N维）
*   **协方差矩阵** `Σ` （N x N 维），用于衡量资产间的风险和联动关系。

我们的目标是确定一个投资权重向量 `w = [w1, w2, ..., wN]^T`，其中 `∑ wi = 1`，代表将资金按比例分配到各资产。

### 投资组合的风险度量
根据现代投资组合理论，一个投资组合的风险（方差）可以用以下公式计算：
**公式：** `Risk = w^T Σ w`
我们的目标是在权重和为1的约束下，最小化这个风险。

### 基于特征值的最小风险组合
一个重要的结论是：**使组合风险最小化的权重 `w`，恰好是协方差矩阵 `Σ` 的最小特征值所对应的特征向量**（需经过归一化处理，使权重和为1）。
**核心思路：** 协方差矩阵 `Σ` 是对称半正定矩阵。其最小的特征值 `λ_min` 代表了系统可达到的最小风险方向，对应的特征向量 `v_min` 指明了在该风险水平下的资产配置比例。

**注意：** 实践中，最小特征值对应的组合往往接近市场指数组合（即大盘方向）。为了获得超越市场的收益，通常会关注第二小或其他次小特征值对应的特征组合，这些组合可能代表了特定的风险因子暴露。

### 基于优化问题的最小方差组合
另一种直接的方法是将其构建为一个带约束的优化问题：
**目标：** `minimize w^T Σ w`
**约束条件：** `∑ wi = 1`
这是一个凸优化问题，可以通过拉格朗日乘子法求解，得到解析解：
**公式：** `w* = (Σ^(-1) 1) / (1^T Σ^(-1) 1)`，其中 `1` 是元素全为1的向量。

## 基于随机矩阵理论的协方差矩阵滤波

上述方法直接使用历史样本协方差矩阵 `Σ`。然而，金融数据噪声很大，样本协方差矩阵包含大量随机噪声，会影响权重估计的稳定性。

### 随机矩阵理论（RMT）
RMT研究由随机变量构成的矩阵的性质。一个核心结论是：纯粹由随机噪声构成的协方差矩阵，其特征值分布服从特定的理论分布（如Marchenko-Pastur分布）。

### 滤波算法步骤
我们可以利用RMT来“过滤”样本协方差矩阵中的随机噪声：
1.  **计算样本协方差矩阵 `Σ` 的特征分解**：得到特征值 `{λ_i}` 和特征向量 `{v_i}`。
2.  **识别并过滤噪声特征值**：将特征值与RMT理论分布进行比较。落在理论分布所描述的“典型噪声区间”（例如，基于分布均值和标准差定义的区间）内的特征值，被认为是主要由随机噪声贡献的。
3.  **重构“清洁”的协方差矩阵**：将上一步中被标记为噪声的特征值置零（或进行收缩），然后利用特征分解的逆过程重构一个新的协方差矩阵 `Σ_filtered`。
    **公式：** `Σ_filtered = V * Λ_filtered * V^T`
    其中 `Λ_filtered` 是过滤后的对角特征值矩阵。
4.  **使用滤波后矩阵进行优化**：将 `Σ_filtered` 代入前述的最小风险或最小方差模型，计算新的投资权重 `w_filtered`。

### 方法优势
这种方法可以保留协方差矩阵中由真实市场结构驱动的信号（对应较大或较小的特征值），而滤除随机噪声（对应中间区域的特征值），从而得到更稳健的资产权重估计。回测结果通常显示，使用滤波后矩阵构建的组合比使用原始样本矩阵的表现更优。

## 相关论文思路解析

以下是几篇将机器学习应用于量化交易的论文核心思路，供大家了解业界方法。

### 论文一：基于深度神经网络的高频交易
*   **核心方法**：使用全连接深度神经网络，输入为过去一段时间的高频价格数据（如5分钟窗口）。
*   **特点**：处理超高维特征（近千维），采用小批量梯度下降进行训练。作者使用了多核协处理器进行长时间训练。
*   **可借鉴点**：高频数据处理的框架、大规模神经网络的训练技巧。
*   **批判性思考**：使用CPU集群训练深度网络效率较低，现代实践中GPU已成为标准配置。

### 论文二：结合深度信念网络（DBN）的股票预测
*   **核心方法**：首先使用深度信念网络进行无监督预训练，提取特征；然后在顶层添加逻辑回归分类器进行有监督微调，预测股价涨跌。
*   **特点**：特征工程相对简单，主要使用历史收益率。模型在验证集上比传统逻辑回归有约4%的错误率提升。
*   **可借鉴点**：深度生成模型在金融特征学习中的应用流程。
*   **批判性思考**：特征设计较为基础，性能提升有限，可能还有很大优化空间。

### 论文三：基于社交媒体情绪的市场预测
*   **核心方法**：利用Twitter和Google搜索中“牛市”、“熊市”等关键词的出现频率，作为市场情绪的代理指标，研究其对未来市场收益的预测能力。
*   **结论**：社交媒体情绪对市场走势有一定预测能力；Twitter数据比Google搜索数据更领先、更有效；但此效应在非西方市场（如中国）可能不显著。
*   **可借鉴点**：将另类数据（新闻、社交文本）引入量化模型的思路，以及严谨的因果分析框架。
*   **批判性思考**：情绪指标是嘈杂的，如何有效清洗、量化并与价格数据融合是关键挑战。

**阅读论文的建议**：重点阅读引言（了解问题）和实验部分（看如何验证），对技术细节保持批判态度，思考其改进空间。
