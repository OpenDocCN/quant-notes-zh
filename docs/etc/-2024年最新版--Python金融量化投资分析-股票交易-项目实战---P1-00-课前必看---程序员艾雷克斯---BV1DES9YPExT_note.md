# Python金融量化投资：P1：00-课前必看

在本节课中，我们将要学习量化投资的核心概念，并了解如何利用Python编程将投资想法转化为可测试的策略。通过历史数据回测，我们可以科学地评估策略的有效性，从而在实战前提高成功的概率。

## 量化投资的核心优势

量化投资最大的一个特点是能用代码来实现你的投资逻辑和想法。然后在历史数据上面看看你的想法到底是好还是差。

如果策略在历史上表现都很差，那么这个策略我们就直接放弃，就不用浪费真钱和时间去做实盘的测试。

如果策略在历史上表现好，我们就会尝试去改进优化，然后投入实战使用。有历史数据的支撑，你赚钱的概率会高很多。

## 初学者面临的常见挑战

很多人会遇到这样的问题，他有一些策略和交易的想法，并想知道它们在历史回测的表现。可是他苦于自己不会写代码或者找不到相关的数据，以至于一直都没有办法知道答案。

所以既然会有人来问我相关的问题，问我按照某某逻辑学的买卖点效果如何，类似种种问题很多。

我们往期的视频评论区也会有很多同学基于我该期视频提供的策略思路，阐述他进一步衍生的策略想法。很多想法其实是很好的，并且很值得尝试，但是他们却无法自己实现。这个时候你就知道会编程的好处了。

正是因为有很多同学有这样的需求，所以我想着干脆做一期视频。
