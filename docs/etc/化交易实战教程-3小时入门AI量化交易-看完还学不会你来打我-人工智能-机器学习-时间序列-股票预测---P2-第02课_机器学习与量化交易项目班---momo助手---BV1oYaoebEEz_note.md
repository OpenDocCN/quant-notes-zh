## 从金融模型到机器学习

掌握了工具之后，我们来看看如何用机器学习的思维理解金融问题。本节将以资本资产定价模型为例进行阐述。

### 资本资产定价模型

资本资产定价模型是金融学中一个非常基础和重要的模型。一个投资组合是资金在不同资产上的分配。市场组合（如沪深300指数）则代表了整个市场的一个健康状况指标。

该模型对单一股票的描述如下：

**公式**：`R_i(t) = β_i * R_m(t) + α_i(t)`

其中：
*   `R_i(t)`：股票`i`在`t`时刻的收益率。
*   `R_m(t)`：市场组合在`t`时刻的收益率。
*   `β_i`：股票`i`的贝塔系数，衡量其相对于市场波动的敏感度。
*   `α_i(t)`：`t`时刻的阿尔法值，代表超额收益。

相信有效市场假说的人认为，从长期看，阿尔法值的期望为零，即 `E[α_i(t)] = 0`。这意味着无法持续获得超越市场的收益。而主动管理者则认为市场并非完全有效，`α` 值不为零且可预测，这正是量化交易试图捕捉的机会。

### 机器学习的视角

现在，我们暂时忘掉金融理论，纯粹从机器学习的角度来看这个公式。

我们感兴趣的目标变量`Y`是某只股票（如招商银行）在每天的收益率 `R_i(t)`。我们想知道是什么因素（特征`X`）影响了`Y`。任何模型都可以表述为 `Y = f(X) + ε`，其中`ε`是均值为零的噪声项。

**资本资产定价模型本质上是一个特例**：它假设影响股票收益的因素`X`只有一个，即市场收益率 `R_m(t)`，并且它们之间的关系`f`是线性的，即 `f(X) = β * X`。这里的`β`和`α`就是线性模型的斜率和截距。

通过历史数据，我们可以画出股票收益率与市场收益率的散点图，然后用线性回归（如`scikit-learn`）拟合出一条直线，这条直线的参数就是该股票的`β`和`α`。

**关键跃迁**：机器学习的思路可以立刻对此进行扩展。我们不一定认为关系是线性的（可以用SVM、神经网络等），也不认为特征只有一个（可以加入更多变量）。这样，我们就从特定的资本资产定价模型，走向了更通用的机器学习预测框架。

### 投资组合模型

上述思想可以扩展到投资组合。一个投资组合的收益率是其成分股票收益率的加权和。代入资本资产定价模型公式并推导，可以发现投资组合的收益率与市场收益率之间**同样满足一个线性关系**，其贝塔系数是各股票贝塔的加权平均。这意味着我们也可以对投资组合进行类似的建模和分析。

**课后练习建议**：任选一只股票，绘制其收益率与沪深300指数收益率的散点图，并用线性回归拟合，直观感受一下两者之间的关系是否真的像模型假设的那样是线性的。

## 技术分析与特征工程

既然量化交易的核心是学习 `Y = f(X)`，那么如何定义和寻找特征`X`就变得至关重要。技术分析为此提供了丰富的思路。

### 技术分析的核心

技术分析者相信，特征`X`可以从历史价格和交易量数据中提取出来。这些提取出来的指标，在机器学习中被称为“特征”。选择哪些特征、如何组合，是量化策略成败的关键之一。这可以依靠经验，也可以使用机器学习方法（如遗传算法）自动筛选。

技术分析通常在以下情况更有效：
1.  **多指标组合**：使用多个特征比单一特征预测能力更强。
2.  **短时间尺度**：技术分析更适用于日内或短期的预测，长期预测能力较弱。
3.  **异类检测**：识别出与大盘走势差异显著的股票，这些地方可能存在套利机会。

与技术分析相对的是基本面分析，它更适用于长期投资。两者在时间尺度上各有侧重。

### 常见技术指标

以下是一些最基本、最常用的技术指标，它们可以作为构造特征`X`的原材料：

1.  **动量**：衡量价格变动的趋势。计算公式为 `Momentum = (Price_t / Price_{t-n}) - 1`。其中`n`是回溯期，是一个超参数，选择不同的`n`可以生成不同的特征。

2.  **简单移动平均线**：对过去`n`个周期的价格取平均值，用于平滑价格曲线。公式为 `SMA_t = (Price_t + Price_{t-1} + ... + Price_{t-n+1}) / n`。一些策略通过比较当前价格与SMA的偏离来产生交易信号。

3.  **布林带**：在移动平均线上下各画出两条标准差带。公式为：
    *   中轨 = `SMA`
    *   上轨 = `SMA + k * StdDev`
    *   下轨 = `SMA - k * StdDev`
    其中`StdDev`是同期价格的标准差，`k`常取2。它引入了波动性的概念，价格在布林带之外运行时可能预示着交易机会。

### 特征标准化

在将多个技术指标作为特征输入模型前，必须进行**标准化**。因为不同指标的取值范围可能差异巨大（如动量在[-0.5, 0.5]，布林带偏移可能在[-10, 35]）。如果不处理，取值大的特征会主导模型，使其他特征失效。

标准化的常用方法是**Z-score标准化**：
**公式**：`X_standardized = (X - mean(X)) / std(X)`
即对每个特征列，减去其均值，再除以其标准差，使得处理后的数据均值为0，标准差为1。

## 机器学习模型初步

有了特征`X`和目标`Y`，接下来就是选择和学习函数`f`。本节介绍机器学习建模的基本概念。

### 问题定义

监督学习的通用框架是：`Y = f(X) + ε`。我们的目标是基于历史数据（训练集）学习一个`f`，使得对于新的`X`，能准确预测`Y`。

### 从简单到复杂的模型

1.  **最近邻**：最简单的非参数方法。预测新样本`X_new`时，在历史数据中寻找最相似的`X`，将其对应的`Y`（或邻近几个`Y`的平均值）作为预测值。这种方法在特征维度低时有效，在高维时会出现“维数灾难”，效果变差。

2.  **线性模型**：假设`f`是线性的，即 `Y = β_0 + β_1*X_1 + ... + β_p*X_p`。学习过程就是找到一组`β`，使得预测值与真实值之间的误差（如均方误差MSE）最小。线性模型简单有效，是很多复杂模型的基础。

3.  **非线性模型**：当`X`和`Y`的关系明显非线性时，需要使用更复杂的模型，如支持向量机、决策树、神经网络等。这些模型可以拟合更复杂的曲线。

### 过拟合与偏差-方差权衡

在选择模型复杂度时，一个核心问题是**过拟合**。如果一个模型在训练集上误差极小（甚至为0），但在没见过的测试集上误差很大，就发生了过拟合。这好比学生死记硬背了所有习题，但遇到新题就不会做。

与之相关的核心概念是**偏差-方差权衡**：
*   **偏差**：模型在训练集上的错误率。模型越简单，偏差通常越高（欠拟合）。
*   **方差**：模型对不同训练集的敏感度。模型越复杂，方差通常越高（过拟合）。

**机器学习的目标是找到最佳的平衡点**，使得模型在训练集上偏差不过高，同时方差也不过高，从而在未知数据（测试集）上表现良好。评估时，务必使用模型未训练过的测试集来计算最终性能。
