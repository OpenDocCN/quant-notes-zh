## 第三步：散点图与分布分析

![](img/242037812ac103a76366a2728ebec5f5_41.png)

除了折线图，散点图能更直观地展示两个变量数据点之间的分布关系。我们可以使用Pandas的绘图功能来创建一个包含散点图和对角线分布图的矩阵。

![](img/242037812ac103a76366a2728ebec5f5_43.png)

![](img/242037812ac103a76366a2728ebec5f5_45.png)

以下是绘制散点图矩阵的步骤：

```python
# 计算每日收益率（增长率）
returns = np.log(data / data.shift(1))

![](img/242037812ac103a76366a2728ebec5f5_47.png)

![](img/242037812ac103a76366a2728ebec5f5_48.png)

# 绘制散点图矩阵，对角线显示核密度估计图
pd.plotting.scatter_matrix(returns,
                           alpha=0.2,        # 设置点的透明度
                           diagonal=‘kde’,   # 对角线绘制核密度估计图
                           figsize=(10, 10))
```

![](img/242037812ac103a76366a2728ebec5f5_50.png)

在这个矩阵图中：
*   对角线上的图显示了每个指标自身的分布情况（核密度估计）。
*   非对角线上的散点图展示了两两指标之间的关系。从散点图的分布形态（例如呈左下到右上的趋势或反之），我们可以更精确地判断它们之间是正相关还是负相关。

## 第四步：量化相关性与回归分析

![](img/242037812ac103a76366a2728ebec5f5_55.png)

![](img/242037812ac103a76366a2728ebec5f5_57.png)

可视化提供了直观感受，但我们还需要一个量化的指标来精确描述相关性。这就是相关系数。同时，我们可以通过回归分析拟合出一条直线，来量化一个变量如何随另一个变量变化。

![](img/242037812ac103a76366a2728ebec5f5_58.png)

![](img/242037812ac103a76366a2728ebec5f5_60.png)

以下是计算相关系数和进行线性回归的步骤：

1.  **计算相关系数**：使用`.corr()`方法可以轻松计算出两个序列的皮尔逊相关系数。
    ```python
    correlation = returns[‘SPX’].corr(returns[‘VIX’])
    print(f”SPX与VIX收益率的相关系数为：{correlation}“)
    ```
    结果通常介于-1到1之间。负值表示负相关，正值表示正相关，绝对值越接近1，相关性越强。

![](img/242037812ac103a76366a2728ebec5f5_62.png)

![](img/242037812ac103a76366a2728ebec5f5_63.png)

![](img/242037812ac103a76366a2728ebec5f5_64.png)

![](img/242037812ac103a76366a2728ebec5f5_65.png)

![](img/242037812ac103a76366a2728ebec5f5_66.png)

![](img/242037812ac103a76366a2728ebec5f5_67.png)

2.  **进行线性回归**：我们可以使用`statsmodels`或`scikit-learn`库进行回归分析，得到回归方程（`y = a * x + b`），从而量化变化关系。
    ```python
    import statsmodels.api as sm
    X = returns[‘SPX’].dropna()
    y = returns[‘VIX’].dropna()
    X = sm.add_constant(X) # 添加常数项
    model = sm.OLS(y, X).fit()
    print(model.summary())
    ```
    回归结果中的系数告诉我们，当SPX变动一个单位时，VIX平均会变动多少。
