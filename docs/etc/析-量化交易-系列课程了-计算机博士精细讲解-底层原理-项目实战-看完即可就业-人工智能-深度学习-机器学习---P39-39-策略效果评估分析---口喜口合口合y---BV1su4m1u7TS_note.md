在编写和运行策略代码时，常常会遇到各种错误。以下是我们在调试过程中遇到并解决的一些典型问题。

![](img/1c3c069572d80d0771e4c69e579e51a4_63.png)

![](img/1c3c069572d80d0771e4c69e579e51a4_64.png)

![](img/1c3c069572d80d0771e4c69e579e51a4_66.png)

1.  **中性化处理参数缺失**：在中性化函数调用时，缺少了必要的参数。我们需要传入因子名称。
    ```python
    # 错误示例
    factor_neutralized = neutralize(factor_data)
    # 正确示例
    factor_neutralized = neutralize(factor_data, factor_name='your_factor')
    ```

![](img/1c3c069572d80d0771e4c69e579e51a4_68.png)

![](img/1c3c069572d80d0771e4c69e579e51a4_70.png)

2.  **模型未训练**：在使用线性回归模型计算残差前，必须先对模型进行训练。
    ```python
    model.fit(X_train, y_train) # 必须先执行训练
    residuals = y_test - model.predict(X_test)
    ```

![](img/1c3c069572d80d0771e4c69e579e51a4_72.png)

![](img/1c3c069572d80d0771e4c69e579e51a4_74.png)

3.  **函数名拼写错误**：例如，分位数函数 `quantile` 被错误拼写。
    ```python
    # 错误拼写
    threshold = df['factor'].qtile(0.3)
    # 正确拼写
    threshold = df['factor'].quantile(0.3)
    ```

![](img/1c3c069572d80d0771e4c69e579e51a4_76.png)

![](img/1c3c069572d80d0771e4c69e579e51a4_78.png)

4.  **变量名错误**：在循环中使用了未定义的变量名。
    ```python
    # 错误示例，`stock_list` 未定义
    for stock in stock_list:
    # 正确示例，使用已定义的 `delete_stock_list`
    for stock in delete_stock_list:
    ```

![](img/1c3c069572d80d0771e4c69e579e51a4_82.png)

![](img/1c3c069572d80d0771e4c69e579e51a4_84.png)

回顾整个策略流程，其核心步骤可以总结如下：

![](img/1c3c069572d80d0771e4c69e579e51a4_116.png)

![](img/1c3c069572d80d0771e4c69e579e51a4_118.png)

![](img/1c3c069572d80d0771e4c69e579e51a4_120.png)

1.  **设定调仓周期**：在 `rebalance` 函数中设定定时器（如每月初），触发调仓逻辑。
2.  **筛选股票池**：从全市场股票中，根据市值、市净率等因子进行筛选和排序，得到当期想要持有的股票列表。
3.  **执行调仓**：
    *   对比当前持仓与最新股票池，卖出持仓中不在新股票池的股票。
    *   买入新股票池中的所有股票，通常采用等权重配置。
4.  **效果评估**：回测结束后，分析累计收益率、年化收益率、夏普比率、最大回撤等指标，并与基准进行比较。
