# 金融量化分析：02：股票基本知识与分类 📈

在本节课中，我们将要学习股票的基本概念、核心作用以及常见的分类方式。理解这些基础知识是进行后续金融量化分析的前提。

## 股票的定义

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

我们可以用一个例子来形象地解释股票。假设有一位创业者需要资金，而投资者（例如一位亿万富翁）愿意提供资金。如果投资者只是借钱给创业者，这属于借贷行为。但投资者可能不满足于只收取利息。

另一种方式是，投资者提供资金，创业者则给予投资者公司的股票。这张股票代表了投资者对公司出资的证明。例如，如果一家公司由创始人和四位投资者各出资1亿元成立，那么公司总市值就是5亿元。每位出资人将获得公司20%的股票，成为公司的股东。

作为股东身份的证明，除了可能签署合同，还需要在工商部门变更营业执照，将出资人登记为股东。

## 股票的作用

股票主要有两个核心作用。

**第一，证明股东身份与权利。** 持有股票意味着你是公司的出资人，是公司的老板之一。作为股东，你拥有在股东大会投票等权利。

**第二，获取经济收益。** 这是普通股民最关心的作用。通过持有股票获利主要有两种方式：

以下是两种主要的获利方式：
1.  **公司分红**：当公司盈利时，可能会将部分利润按持股比例分配给股东。例如，若公司年净利润为5000万元，持有20%股份的股东可获得1000万元的分红。
2.  **交易获利（套现）**：股东可以在市场上将股票转让给他人。如果公司价值增长，股票价格上升，卖出股票就能赚取差价。例如，初始投资1亿元获得20%股份，当公司市值从5亿增长到50亿时，这20%股份的价值就变成了10亿元。此时卖出部分股份即可实现收益。

对于资金量较小的普通股民而言，逻辑完全相同：通过购买股票，成为公司的小股东，从而享受分红或通过买卖股票赚取差价。区别在于，小股民需要通过**证券交易所**这个公开市场来买卖股票。

## 什么是上市

上市是指企业通过证券交易所，首次公开向全社会投资者增发股票，以募集资金的过程。

创业者不能随意向公众募集资金，那属于非法集资。公司需要达到一定体量，并向证监会提交申请，经过严格的财务和合规审查。证监会评估认为公司经营状况良好、风险可控后，才会批准其上市。

上市后，公司的股票就能在证券交易所挂牌交易。所有符合条件的股民都可以公开买卖这只股票。上市的本质是获得了向广大社会公众募集资金的资格。

**IPO（首次公开募股）** 就是指公司第一次向社会公众公开发行股票的行为。

## 股票的分类

股票主要有两种分类方式。

### 按业绩分类

根据公司的业绩表现，股票可以分为以下三类：

以下是三种常见的按业绩分类的股票：
*   **蓝筹股**：指资本雄厚、信誉优良的大型公司股票。例如中国的**中石油**、**中石化**。名称源于赌场中价值最高的蓝色筹码。
*   **绩优股**：指业绩优良公司的股票。它们可能规模不是最大，但盈利增长稳定。例如**贵州茅台**。
*   **ST股**：中文为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记，以警示投资者该公司存在风险。

### 按上市地区分类

根据公司上市交易的地点和所用货币，股票可分为：

以下是几种按上市地区分类的股票：
*   **A股**：在中国大陆（上海或深圳）证券交易所上市，以**人民币**认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以**外币**（如美元、港币）认购和交易的股票。
*   **H股**：在中国**香港**上市的公司股票。
*   **N股**：在美国**纽约**上市的公司股票。
*   **S股**：在新加坡上市的公司股票。

### 不同市场的交易规则

不同股市有不同的交易规则，主要体现在两点：

1.  **涨跌幅限制**：中国A股设有每日**10%**的涨跌幅限制。即股票当日价格相对于前一日收盘价的波动幅度不得超过10%。此规则旨在平抑市场剧烈波动，保护投资者。美股等市场大多无此限制。
2.  **交割制度**：
    *   **A股实行T+1制度**：当日（T日）买入的股票，需到下一个交易日（T+1日）才能卖出。这旨在减少市场投机行为。
    *   **港股、美股等实行T+0制度**：当日买入的股票可以当日卖出，允许日内多次交易。
    *   **B股涉及T+1与T+3**：T+1指股票交割日（买入后下一交易日才能卖）；T+3指资金交收日（卖出股票后，资金需3个交易日才能提现）。
