## 统计方法

NumPy提供了一系列用于计算数组统计量的方法。以下是常用的几个函数：

以下是核心统计函数及其说明：

*   **求和**：`A.sum()` 方法对数组 `A` 中的所有值进行求和。其功能与Python内置的 `sum()` 函数类似。
*   **平均值**：`A.mean()` 方法用于计算数组 `A` 中所有元素的平均值。
*   **方差**：`A.var()` 方法用于计算方差。方差衡量一组数据的离散程度。其计算公式为：`方差 = Σ(每个数据值 - 平均值)² / 数据个数`。
*   **标准差**：`A.std()` 方法用于计算标准差。标准差是方差的平方根，同样用于衡量数据的离散程度。其计算公式为：`标准差 = sqrt(方差)`。
*   **最大值与最小值**：`A.max()` 和 `A.min()` 分别用于找出数组中的最大值和最小值。
*   **最大值/最小值索引**：`A.argmax()` 和 `A.argmin()` 分别返回数组中最大值和最小值所在位置的索引（下标）。

## 随机数生成

NumPy的 `random` 子模块提供了强大的随机数生成功能，它扩展了Python内置 `random` 模块的功能，支持直接生成指定形状的随机数组。

以下是NumPy `random` 模块中常用的函数：

*   **生成随机浮点数**：`np.random.rand(10)` 生成一个包含10个在 `[0, 1)` 区间内均匀分布的随机浮点数的数组。可以通过传入元组来指定多维数组的形状，例如 `np.random.rand(3, 5)` 生成一个3行5列的数组。
*   **生成随机整数**：`np.random.randint(0, 10, size=10)` 生成10个在 `[0, 10)` 区间内的随机整数。`size` 参数同样可以接收元组来定义输出数组的形状。
*   **从给定序列中随机选择**：`np.random.choice([1, 3, 4, 5], size=10)` 从序列 `[1, 3, 4, 5]` 中随机抽取10次（有放回），返回结果数组。
*   **打乱数组顺序**：`np.random.shuffle(A)` 将数组 `A` 的元素顺序随机打乱。此操作直接修改原数组。
*   **生成均匀分布随机数**：`np.random.uniform(2.0, 4.0, size=10)` 生成10个在 `[2.0, 4.0)` 区间内均匀分布的随机浮点数。
