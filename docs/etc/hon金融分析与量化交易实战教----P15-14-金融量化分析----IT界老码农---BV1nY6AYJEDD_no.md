上一节我们介绍了布尔型索引，本节中我们来看看另一种强大的索引方式——花式索引。

## 花式索引的概念

花式索引允许我们使用一个整数数组来索引目标数组，从而选取任意位置的元素组合成一个新数组。这与切片不同，切片选取的是连续区域，而花式索引可以选取任意位置。

例如，对于一个一维数组，我们想选取第1、3、4、6、7号元素（索引从0开始）。这些位置没有规律，无法用简单的切片完成。

```python
import numpy as np
a = np.arange(20)  # 创建一个0到19的数组
indices = [1, 3, 4, 6, 7]  # 想要选取的索引位置
result = a[indices]  # 使用花式索引
print(result)  # 输出：[1 3 4 6 7]
```

**核心操作**：`array[[index1, index2, ...]]`

## 二维数组的花式索引

![](img/8726f336c798439050b3287ab6d4907d_7.png)

对于二维数组，索引规则同样适用。NumPy数组的索引通用模式是：在方括号内用逗号分隔行和列的索引方式。

以下是几种索引方式的组合示例：

*   **行用常规索引，列用切片**
    ```python
    arr = np.arange(20).reshape(4, 5)
    # 选取第0行，第2到第4列（不包含第4列）
    print(arr[0, 2:4])
    ```

*   **行用常规索引，列用布尔型索引**
    ```python
    arr = np.arange(20).reshape(4, 5)
    # 选取第0行中所有大于2的元素
    print(arr[0, arr[0] > 2])
    ```

*   **行用花式索引，列用全切片**
    ```python
    arr = np.arange(20).reshape(4, 5)
    # 选取第1行和第3行的所有列
    print(arr[[1, 3], :])
    ```

## 一个重要限制与解决方案

需要注意的是，**逗号两边不能同时使用花式索引列表**，否则会产生意想不到的结果。这种写法会被解释为选取坐标对 `(行1, 列1)` 和 `(行2, 列2)` 位置上的值，而不是分别选取行和列。

例如，想选取第1行&第3行 与 第1列&第3列 交叉的四个点（即元素6, 8, 16, 18），不能直接写 `arr[[1,3], [1,3]]`。

正确的做法是分步进行：

1.  先使用花式索引选取目标行。
2.  再在结果上选取目标列。

```python
arr = np.arange(20).reshape(4, 5)
# 第一步：选取第1行和第3行的所有列
step1 = arr[[1, 3], :]
# 第二步：在step1的结果上，选取所有行的第1列和第3列
result = step1[:, [1, 3]]
print(result)
# 输出：
# [[ 6  8]
#  [16 18]]
```

**核心技巧**：当需要行列组合进行复杂选取时，考虑分步操作。
