## 注释

注释是用于解释代码的文本，对于提升代码的可读性和可维护性非常重要。

在Python中，使用井号 `#` 来标记单行注释。`#` 后面的所有内容都会被Python解释器忽略，不会被执行。

![](img/03738ecc9a3a791e9464b6a9c7976b48_1.png)

以下是注释的用法示例：

![](img/03738ecc9a3a791e9464b6a9c7976b48_3.png)

```python
# 这是一行注释，不会被运行
print("Hello World")  # 这行代码会打印“Hello World”，但“#”后面的文字是注释
```

我们可以通过注释来临时禁用某些代码行。例如，在集成开发环境（IDE）中，通常可以使用快捷键（如 `Ctrl + /`）来快速注释或取消注释选中的代码行。被注释掉的代码将不会影响程序的运行结果。

![](img/03738ecc9a3a791e9464b6a9c7976b48_5.png)

我们通常使用注释来解释代码的功能和逻辑，这有助于自己或他人在未来阅读和维护代码。

## 变量

![](img/03738ecc9a3a791e9464b6a9c7976b48_7.png)

变量是Python中用于存储数据的容器。你可以将变量理解为一个贴有标签的盒子，标签是变量名，盒子里装的就是数据。

![](img/03738ecc9a3a791e9464b6a9c7976b48_9.png)

在Python中，使用等号 `=` 进行赋值操作，将等号右边的值赋予左边的变量名。

以下是变量的定义与使用：

![](img/03738ecc9a3a791e9464b6a9c7976b48_11.png)

```python
message = "Hello World"  # 将字符串“Hello World”赋值给变量 message
print(message)           # 打印变量 message 的值，输出：Hello World
```

![](img/03738ecc9a3a791e9464b6a9c7976b48_13.png)

## 数据类型

![](img/03738ecc9a3a791e9464b6a9c7976b48_15.png)

Python中的数据有多种类型，每种类型都有其特定的用途。了解数据类型是进行有效编程的关键。

![](img/03738ecc9a3a791e9464b6a9c7976b48_17.png)

以下是Python中几种基础的数据类型：

![](img/03738ecc9a3a791e9464b6a9c7976b48_19.png)

### 1. 整数
整数用于表示没有小数部分的数字，例如 `0`、`1`、`-100`。在Python中，整数的大小没有限制。在量化投资中，常用来表示交易数量或持仓量。

![](img/03738ecc9a3a791e9464b6a9c7976b48_21.png)

```python
quantity = 10  # 定义一个整数变量
print(type(quantity))  # 输出：<class 'int'>
```

![](img/03738ecc9a3a791e9464b6a9c7976b48_22.png)

### 2. 浮点数
浮点数用于表示带有小数点的数字，例如 `3.14`、`-0.5`。在金融计算中，常用于表示价格、收益率等。

```python
price = 3.14  # 定义一个浮点数变量
print(type(price))  # 输出：<class 'float'>
```

![](img/03738ecc9a3a791e9464b6a9c7976b48_24.png)

![](img/03738ecc9a3a791e9464b6a9c7976b48_26.png)

### 3. 布尔值
布尔值只有两个：`True`（真）和 `False`（假）。它主要用于条件判断和逻辑运算。

```python
is_active = True  # 定义一个布尔值变量
print(type(is_active))  # 输出：<class 'bool'>
```

![](img/03738ecc9a3a791e9464b6a9c7976b48_28.png)

### 4. 字符串
字符串用于表示文本数据，由一系列字符组成，需要用单引号 `‘’` 或双引号 `“”` 括起来。

![](img/03738ecc9a3a791e9464b6a9c7976b48_30.png)

```python
trade_pair = "ETH/USDT"  # 定义一个字符串变量，表示交易对
print(type(trade_pair))  # 输出：<class 'str'>
```

![](img/03738ecc9a3a791e9464b6a9c7976b48_32.png)

### 5. 列表
列表用于存储多个元素的有序集合。列表中的元素可以是不同类型的数据，并且列表本身是可变的（可以修改）。列表使用方括号 `[]` 定义。

```python
price_history = [100, 101.5, 99, 102]  # 定义一个列表，存储价格历史
print(type(price_history))  # 输出：<class 'list'>

![](img/03738ecc9a3a791e9464b6a9c7976b48_34.png)

# 列表可以包含混合类型
mixed_list = [1, "hello", True, 3.14, [1, 2]]
```

### 6. 元组
元组与列表类似，也是有序的元素集合。但元组是不可变的，一旦创建就不能修改。元组使用圆括号 `()` 定义。

```python
exchange_info = ("Binance", "USDT")  # 定义一个元组
print(type(exchange_info))  # 输出：<class 'tuple'>
```

![](img/03738ecc9a3a791e9464b6a9c7976b48_36.png)

### 7. 字典
字典用于存储“键-值”对映射关系的数据。它类似于现实生活中的字典，通过“键”来查找对应的“值”。字典使用花括号 `{}` 定义，键值对之间用冒号 `:` 分隔。

![](img/03738ecc9a3a791e9464b6a9c7976b48_38.png)

```python
# 定义一个字典，存储交易所配置信息
exchange_config = {
    "name": "Binance",
    "api_key": "your_api_key_here",
    "base_currency": "USDT"
}
print(type(exchange_config))  # 输出：<class 'dict'>

![](img/03738ecc9a3a791e9464b6a9c7976b48_40.png)

# 通过“键”来访问对应的“值”
print(exchange_config["name"])  # 输出：Binance
```

![](img/03738ecc9a3a791e9464b6a9c7976b48_42.png)

## 总结

![](img/03738ecc9a3a791e9464b6a9c7976b48_44.png)

本节课我们一起学习了Python编程的三个基础概念：注释、变量和数据类型。

![](img/03738ecc9a3a791e9464b6a9c7976b48_46.png)

![](img/03738ecc9a3a791e9464b6a9c7976b48_48.png)

*   **注释** (`#`) 用于解释代码，提高可读性。
*   **变量** 是存储数据的容器，通过赋值操作符 (`=`) 来关联数据。
*   **数据类型** 定义了数据的种类和可执行的操作，我们介绍了七种基础类型：
    *   **整数** (`int`)： 表示整数。
    *   **浮点数** (`float`)： 表示小数。
    *   **布尔值** (`bool`)： 表示真 (`True`) 或假 (`False`)。
    *   **字符串** (`str`)： 表示文本。
    *   **列表** (`list`)： 表示有序、可变的元素集合。
    *   **元组** (`tuple`)： 表示有序、不可变的元素集合。
    *   **字典** (`dict`)： 表示“键-值”对映射。
