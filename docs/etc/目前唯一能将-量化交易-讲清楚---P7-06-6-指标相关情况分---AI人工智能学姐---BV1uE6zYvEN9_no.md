### 绘制双轴折线图

![](img/fc36da642f49dcf28baf595199291a63_39.png)

![](img/fc36da642f49dcf28baf595199291a63_41.png)

![](img/fc36da642f49dcf28baf595199291a63_43.png)

最初，我们可以尝试将两个指标画在同一张图上。但由于它们的数值范围可能差异很大，直接绘制会导致趋势难以观察。

![](img/fc36da642f49dcf28baf595199291a63_45.png)

以下是改进的绘图方法：
1.  使用 `plot` 方法，并指定 `secondary_y` 参数为第二个指标（如 `VIX`），这样可以创建双Y轴图表。
2.  为了更清晰地观察短期关系，可以截取特定时间段（例如2010年至2012年）的数据进行绘图。
3.  调整图表尺寸以获得更好的视觉效果。

![](img/fc36da642f49dcf28baf595199291a63_47.png)

![](img/fc36da642f49dcf28baf595199291a63_49.png)

![](img/fc36da642f49dcf28baf595199291a63_51.png)

![](img/fc36da642f49dcf28baf595199291a63_53.png)

```python
# 选取2010年至2012年的数据
data_subset = data['2010-01-01':'2012-12-31']
# 绘制双Y轴折线图，VIX使用右侧坐标轴
ax = data_subset.plot(secondary_y=['VIX'], figsize=(12, 6))
```

![](img/fc36da642f49dcf28baf595199291a63_55.png)

![](img/fc36da642f49dcf28baf595199291a63_57.png)

通过观察生成的图表，我们可以发现一个初步规律：当 `SPX` 指数下降时，`VIX` 指数往往上升，反之亦然。这暗示着两者可能存在负相关关系。可视化是一个迭代过程，通常从基础图表开始，根据观察结果不断调整参数，以使数据呈现更清晰。

### 绘制散点图与分布图

![](img/fc36da642f49dcf28baf595199291a63_61.png)

除了折线图，散点图矩阵是观察多变量关系及各自分布的强大工具。它能同时展示变量两两之间的散点关系以及对角线上的单变量分布。

以下是绘制散点图矩阵的步骤：
1.  计算指标的日收益率（或对数收益率）作为分析基础，这能消除价格序列的非平稳性。
2.  使用Pandas的 `scatter_matrix` 函数绘制图表。
3.  可以指定对角线上的图形为直方图或核密度估计图以展示数据分布。
4.  设置点透明度以避免数据点过度重叠。

![](img/fc36da642f49dcf28baf595199291a63_63.png)

![](img/fc36da642f49dcf28baf595199291a63_65.png)

```python
import pandas as pd
import numpy as np

# 计算对数收益率
returns = np.log(data / data.shift(1))
# 绘制散点图矩阵，对角线显示核密度估计图
pd.plotting.scatter_matrix(returns, alpha=0.5, diagonal='kde', figsize=(12, 12))
```

![](img/fc36da642f49dcf28baf595199291a63_67.png)

![](img/fc36da642f49dcf28baf595199291a63_69.png)

在生成的矩阵图中，非对角线部分展示了 `SPX` 收益率与 `VIX` 收益率之间的散点关系，可以更明确地看到负相关趋势。对角线部分则展示了每个变量自身的分布情况。核密度估计图提供了比直方图更平滑的分布曲线，本质上是当分箱数量趋近于无穷时的极限情况。

![](img/fc36da642f49dcf28baf595199291a63_71.png)

![](img/fc36da642f49dcf28baf595199291a63_73.png)

## 引入回归分析

![](img/fc36da642f49dcf28baf595199291a63_75.png)

![](img/fc36da642f49dcf28baf595199291a63_77.png)

通过散点图，我们已经可以直观地看到变量间的负相关趋势。为了量化这种关系，我们可以引入回归分析。回归分析能在散点图上拟合出一条最优直线（或曲线），其方程和系数能精确描述一个变量如何随另一个变量变化。
