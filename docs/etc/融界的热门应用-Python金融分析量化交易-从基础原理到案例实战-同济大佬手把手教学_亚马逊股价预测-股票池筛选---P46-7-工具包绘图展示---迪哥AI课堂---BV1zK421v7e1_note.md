## 图表结果解读

生成的图表包含以下关键元素：

*   **蓝色折线**：代表每日计算的实际IC值。其波动范围通常较大，直接观察趋势可能不够清晰。
*   **绿色折线**：代表以一个月为窗口计算的IC值移动平均线。它平滑了日度数据的噪声，能更清晰地反映IC值的长期趋势。
*   **统计信息**：图表通常会标注IC值的均值（mean）和标准差（std）等统计量。

通过观察绿色移动平均线，我们可以评估因子与收益率的相关性是否稳定以及是否存在趋势。理想情况下，我们希望IC值保持较高且稳定的正值。

## 信息比率简介

图表中还会展示一个名为“信息比率”（Information Ratio, IR）的指标，其计算公式为：

**IR = mean(IC) / std(IC)**

信息比率用于衡量因子IC值的稳定性。比值越大，说明因子的预测能力越稳定（均值相对较高，或波动相对较小）。反之，则说明因子的有效性不够稳定。
