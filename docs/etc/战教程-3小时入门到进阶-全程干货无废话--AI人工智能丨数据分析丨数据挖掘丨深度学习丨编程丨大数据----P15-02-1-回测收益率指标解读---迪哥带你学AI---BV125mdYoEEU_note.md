## 回测收益率详解

![](img/0ad220ef39431f81a8669d24d5cf6ece_21.png)

回测收益率是评估策略盈利能力最直接的指标。它衡量的是，从策略开始到结束，你的总资产相对于初始本金增长（或亏损）的百分比。

**公式**如下：
`回测收益率 = (期末资产总值 - 期初本金) / 期初本金`
也可以表示为：
`回测收益率 = (P_t / P_0) - 1`
其中，`P_t` 代表策略结束时的资产价值（卖出价格），`P_0` 代表策略开始时的初始本金（买入成本）。

![](img/0ad220ef39431f81a8669d24d5cf6ece_23.png)

这个指标直观地告诉你，折腾了一番之后，你的本金最终变成了原来的多少倍。例如，4.2%的收益率意味着你的资产相对于本金增长了4.2%。

### 计算与可视化

![](img/0ad220ef39431f81a8669d24d5cf6ece_25.png)

![](img/0ad220ef39431f81a8669d24d5cf6ece_27.png)

下面，我们通过代码逐步计算并可视化整个回测期间收益率的动态变化。

首先，我们计算每个时间点资产相对于初始本金的比值：

```python
# 计算每个时间点的资产与初始资产的比值
# 即公式中的 P_t / P_0
price_relative = data / data.iloc[0]
```

接下来，我们可以绘制这个比值随时间变化的曲线，直观地看到资产的增长过程：

```python
# 绘制资产相对价值变化图
plt.figure(figsize=(10, 6))
price_relative.plot()
plt.title('资产相对价值变化曲线')
plt.xlabel('日期')
plt.ylabel('相对价值 (P_t / P_0)')
plt.grid(True)
plt.show()
```

![](img/0ad220ef39431f81a8669d24d5cf6ece_29.png)

这张图展示了从期初（价值为1）开始，资产价值随时间是如何波动的。最后，我们计算整个回测周期的总收益率：

![](img/0ad220ef39431f81a8669d24d5cf6ece_31.png)

![](img/0ad220ef39431f81a8669d24d5cf6ece_33.png)

```python
# 计算最终的回测收益率
# 取最后一个时间点的比值，然后减去1
final_return = price_relative.iloc[-1] - 1
```
