## 第三步：计算短期与长期移动平均线 📉

上一节我们完成了数据清洗，本节中我们来看看如何计算关键的移动平均线指标。我们将分别计算短期（例如42天）和长期（例如252天）的移动平均线。

![](img/1a87138c248120f918ab4c77c73b7bdc_42.png)

![](img/1a87138c248120f918ab4c77c73b7bdc_44.png)

以下是计算移动平均线的步骤：

1.  首先，定义短期和长期的窗口大小。
2.  然后，使用Pandas的`rolling`和`mean`方法计算移动平均值。
3.  最后，将计算结果作为新列添加到数据框中。

![](img/1a87138c248120f918ab4c77c73b7bdc_46.png)

![](img/1a87138c248120f918ab4c77c73b7bdc_48.png)

```python
# 定义窗口大小
short_window = 42
long_window = 252

![](img/1a87138c248120f918ab4c77c73b7bdc_50.png)

![](img/1a87138c248120f918ab4c77c73b7bdc_52.png)

# 计算短期移动平均线
stock_data['short_mavg'] = stock_data['AAPL'].rolling(window=short_window, min_periods=1).mean()

![](img/1a87138c248120f918ab4c77c73b7bdc_54.png)

# 计算长期移动平均线
stock_data['long_mavg'] = stock_data['AAPL'].rolling(window=long_window, min_periods=1).mean()
```

![](img/1a87138c248120f918ab4c77c73b7bdc_56.png)

1.  首先，确保数据中已无缺失值。
2.  然后，使用`np.where`函数创建标志列：当短期均线大于长期均线时标记为1（持有/买入信号），否则标记为0（卖出信号）。

stock_data['position'] = np.where(stock_data['short_mavg'] > stock_data['long_mavg'], 1, 0)

## 第六步：理解策略逻辑 💡

最后，我们来探讨这个双均线策略背后的逻辑。如果不采取任何策略，只是简单地买入并持有，最终的收益完全取决于股价的最终走势。

然而，双均线策略试图通过捕捉趋势变化来优化收益。其核心思想是：
*   当出现 **黄金交叉**（短期均线上穿长期均线）时，表明可能开启上涨趋势，策略发出**买入**信号。
*   当出现 **死亡交叉**（短期均线下穿长期均线）时，表明可能开启下跌趋势，策略发出**卖出**信号。

![](img/1a87138c248120f918ab4c77c73b7bdc_104.png)

理论上，通过在这种信号下进行交易，投资者可以在上涨趋势中持有资产，在下跌趋势中避开损失或持有现金，从而有望获得超越简单买入持有策略的回报。下一节中，我们将基于这些信号计算具体的策略回报率。
