## 📊 基准波动率

上一节我们介绍了策略自身的波动率，本节中我们来看看基准波动率。基准波动率用于衡量市场基准（如沪深300指数）的风险水平。

基准波动率的定义是：基准波动率（Benchmark Volatility）用来测量基准的风险。波动越大，代表基准风险越高。

基准波动率的计算公式与策略波动率类似，但计算对象是基准的收益率序列。其核心思想是计算基准收益率相对于其均值的离散程度。

以下是基准波动率的计算公式：

![](img/39f8ff28af82ecb1e1c164b409f5179f_3.png)

```
基准波动率 = 基准年化收益率的标准差
```

具体计算时，通常使用基准每日收益率的序列，计算其年化标准差。



![](img/39f8ff28af82ecb1e1c164b409f5179f_5.png)

为什么需要关注基准波动率呢？因为不同的量化策略适应不同的市场环境。有的策略适合单边上涨行情，有的适合单边下跌行情，而有的则在市场震荡时表现更好。基准波动率的高低，反映了整个市场是处于平稳、单边还是剧烈震荡的状态，这有助于我们判断当前策略是否适应当前市场。



![](img/39f8ff28af82ecb1e1c164b409f5179f_7.png)

接下来，我们看看如何在回测平台中查看基准波动率。



在量化回测平台中，查看基准波动率的方法与查看其他指标类似。通常，在回测结果的风险指标面板中，点击“基准波动率”选项即可看到具体的数值。



例如，在我们的示例策略回测中，基准波动率的值可能显示为0.2左右。如果我们的策略波动率是0.3几，这说明我们的策略波动比市场基准更大，风险相对更高。



## 📉 最大回撤

介绍完基准波动率，我们进入最后一个，也是投资者非常关心的风险指标——最大回撤。它描述了投资过程中可能面临的最糟糕情况。

最大回撤的定义是：在选定周期内，从任一历史时点往后推，资产净值（或股价）走到最低点时，收益率回撤幅度的最大值。简单来说，就是在一段时间内，资产从最高点到最低点的最大跌幅。

最大回撤的公式如下：

![](img/39f8ff28af82ecb1e1c164b409f5179f_9.png)

```
最大回撤 = (Px - Py) / Px
```
其中：
*   `Px` 代表周期内某一历史高点的资产价值。
*   `Py` 代表从该高点之后出现的资产最低价值。

![](img/39f8ff28af82ecb1e1c164b409f5179f_11.png)

该公式计算的是从高点`Px`到后续低点`Py`的亏损比例。最大回撤就是遍历所有可能的`Px`和`Py`组合后，计算出的最大亏损比例。



最大回撤是一个极其重要的风险指标，因为它直观地展示了投资者可能承受的最大账面亏损。例如，如果某个策略在一年内的最高收益为30%，但最低跌至-30%，那么其最大回撤就接近60%。这意味着投资者如果在高点附近买入，在最坏情况下可能面临60%的亏损。

一个理想的情况是最大回撤尽可能小，甚至为零。最大回撤为零意味着资产净值从未从历史高点回落，一直处于横盘或单边上涨状态（例如，过去二十年中国的楼市就近似这种状态）。



![](img/39f8ff28af82ecb1e1c164b409f5179f_13.png)

最后，我们来看看最大回撤在回测平台中的体现。



在量化回测平台的风险指标面板中，点击“最大回撤”即可查看该策略在回测周期内的最大回撤值。



例如，我们的示例策略最大回撤可能显示为0.3（即30%）。这表明，在该策略运行期间，从收益最高点到后续最低点的最大跌幅达到了30%。如果最高点收益仅为10%，那么在最坏情况下，投资者可能面临20%的亏损。这侧面印证了我们之前观察到的结论：该策略的波动性较高。



## 🎯 本章小结

本节课中我们一起学习了量化策略回测的最后两个风险指标。

首先，我们介绍了**基准波动率**。它的算法与策略波动率类似，但计算对象是市场基准的收益率。基准波动率的意义在于反映整个市场的波动情况，帮助我们判断策略是否适应当前的市场环境（如单边市或震荡市）。

接着，我们学习了**最大回撤**。它的作用是表征买入股票或运行投资策略后，可能出现的**最糟糕的账面亏损情况**。其算法是 `(周期内最高点价值 - 后续最低点价值) / 周期内最高点价值`。最大回撤为零通常意味着资产处于持续上涨或横盘状态。
