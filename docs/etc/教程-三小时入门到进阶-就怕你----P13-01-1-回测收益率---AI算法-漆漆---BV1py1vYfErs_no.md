## 回测收益率详解

![](img/3778eca3002909450fa91874f2438cd9_17.png)

回测收益率是最基本、最直观的策略评估指标。它衡量的是，在回测周期内，策略最终获得的收益相对于初始本金的增长比例。

![](img/3778eca3002909450fa91874f2438cd9_19.png)

![](img/3778eca3002909450fa91874f2438cd9_21.png)

例如，你基于2015年至2020年的股票数据设计了一个买卖策略。回测收益率告诉你，在策略结束时，你的总资产相比最初投入的本金增加了多少百分比。下图结果中的4%多，就表示本金增值了约4%。

![](img/3778eca3002909450fa91874f2438cd9_23.png)

它的计算公式如下：

![](img/3778eca3002909450fa91874f2438cd9_23.png)

**回测收益率 = (期末资产总值 - 期初本金) / 期初本金**

用符号表示，若期初本金对应价格为 **P0**，期末资产总值对应价格为 **Pt**，则公式为：

![](img/3778eca3002909450fa91874f2438cd9_25.png)

**收益率 = (Pt - P0) / P0**

![](img/3778eca3002909450fa91874f2438cd9_27.png)

这个公式本质上计算的是资产价值的相对增长。

![](img/3778eca3002909450fa91874f2438cd9_25.png)

为了更直观地观察整个回测周期内收益的变化过程，我们可以绘制资产净值曲线。这条曲线展示了从期初到任意时刻，资产相对于初始本金的累积增长情况。

计算过程是：将每个时间点的资产价格 **Pt** 除以期初价格 **P0**，得到该时刻的累积净值。最后一天的净值减1，即为整个周期的回测收益率。

以下是具体的代码实现：

```python
# 计算累积净值：每个时间点的价格除以期初价格
cumulative_value = data / data.iloc[0]

# 绘制累积净值曲线
plt.figure(figsize=(10, 6))
cumulative_value.plot()
plt.title('策略累积净值曲线')
plt.xlabel('日期')
plt.ylabel('累积净值 (相对于期初)')
plt.grid(True)
plt.show()
```

# 计算最终的回测收益率：最后一天的净值减1
