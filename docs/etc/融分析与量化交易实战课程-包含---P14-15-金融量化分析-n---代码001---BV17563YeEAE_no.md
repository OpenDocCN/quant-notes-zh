## 概述
在本节课中，我们将要学习NumPy库中的通用函数。这些函数可以对数组进行快速的、元素级的数学运算，是进行金融数据批量处理和分析的基础工具。

---

上一节我们介绍了NumPy数组对象的索引功能，本节中我们来看看NumPy还提供了哪些强大的功能。首先，一个常用的功能是通用函数。

我们之前已经知道，数组对象可以进行加减乘除等批量运算。那么，对于一些更复杂的数学运算，是否也能批量执行呢？答案是肯定的。

以下是NumPy提供的一些常用一元通用函数（即只对一个数组进行运算的函数）：

![](img/fb3624564bae8ec90bdc9ffc99826345_3.png)

![](img/fb3624564bae8ec90bdc9ffc99826345_5.png)

*   **绝对值**：`np.abs(array)` 或 `abs(array)`。该函数对数组中的每个元素取绝对值。
*   **平方根**：`np.sqrt(array)`。该函数对数组中的每个元素进行开平方运算。注意，如果数组中有负数，结果会返回`nan`（非数字）。
*   **指数与对数**：例如 `np.exp(array)`（指数运算）和 `np.log(array)`（自然对数运算）等数学函数。

![](img/fb3624564bae8ec90bdc9ffc99826345_7.png)

接下来，我们重点讲解几种不同的取整函数。在Python中，将小数转换为整数有多种方式。

![](img/fb3624564bae8ec90bdc9ffc99826345_9.png)

以下是四种主要的取整方法及其对应的NumPy函数：

![](img/fb3624564bae8ec90bdc9ffc99826345_11.png)

*   **四舍五入**：`np.round(array)`。遵循“四舍六入五成双”的规则。
*   **向下取整**：`np.floor(array)`。总是取不大于原数的最大整数。
*   **向上取整**：`np.ceil(array)`。总是取不小于原数的最小整数。
*   **向零取整**：`np.trunc(array)`。直接舍弃小数部分，向零的方向取整。

为了更直观地理解，我们可以看下面的数轴示意图。假设我们有两个数：1.6 和 -1.6。

```
数轴： ... -2  -1.6  -1  0  1  1.6  2 ...
向下取整(floor): -2, 1
向上取整(ceil):  -1, 2
向零取整(trunc): -1, 1
四舍五入(round): -2, 2 (取决于具体规则)
```

记住这四种取整方式在特定数学或金融计算中非常有用。

除了取整，还有一个实用的函数 `np.modf(array)`。该函数将一个数组的整数部分和小数部分分离，并作为两个独立的数组返回。这在需要分别处理整数和小数部分时很有用。

```python
import numpy as np
arr = np.array([1.2, 3.7, -2.8])
integer_part, fractional_part = np.modf(arr)
# integer_part: [ 1.  3. -2.]
# fractional_part: [ 0.2  0.7 -0.8]
```

接下来，我们讨论两个在数值计算中常见的特殊浮点数值：`nan` 和 `inf`。

*   **`nan` (Not a Number)**：表示一个“不是数字”的值。它不等于任何值，甚至不等于它自己。在进行非法运算时会产生，例如 `0.0 / 0.0` 或对负数开平方。
*   **`inf` (Infinity)**：表示无穷大。当一个非零数除以 `0.0` 时会产生，例如 `5.0 / 0.0`。

由于 `nan` 的特性，我们不能用 `==` 来判断一个值是否为 `nan`。NumPy提供了专门的判断函数：

*   判断是否为 `nan`：`np.isnan(array)`
*   判断是否为 `inf`：`np.isinf(array)`

在数据分析中，我们经常需要过滤掉这些无效或无穷大的值。

```python
arr_with_nan = np.array([1, 2, np.nan, 4, np.inf])
# 过滤掉 nan 和 inf
valid_arr = arr_with_nan[~np.isnan(arr_with_nan) & ~np.isinf(arr_with_nan)]
# valid_arr: [1. 2. 4.]
```

最后，我们介绍两个常用的二元通用函数（即对两个数组进行运算的函数）：`np.maximum` 和 `np.minimum`。

这两个函数会逐元素地比较两个数组，并返回一个由每个位置上较大（或较小）值组成的新数组。

```python
a = np.array([1, 5, 3])
b = np.array([4, 2, 6])
np.maximum(a, b) # 结果: [4 5 6]
np.minimum(a, b) # 结果: [1 2 3]
```

这与求整个数组最大值/最小值的 `array.max()` 或 `array.min()` 方法完全不同。
