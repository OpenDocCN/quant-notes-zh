# 量化交易实战课程：P15：阿尔法与贝塔概述

![](img/2f40a668ac7186327f48ca801ee6d0d8_0.png)

在本节课中，我们将学习量化策略评估中的两个核心概念：阿尔法（α）和贝塔（β）。理解这两个指标对于分析投资策略的收益来源至关重要。

## 收益的构成

投资策略的总收益可以分解为两个部分。

一部分收益与整体市场环境相关。当市场整体向好时，大部分投资都能获得收益。这部分收益被称为**市场收益**。

另一部分收益则与市场整体波动无关，它源于投资者独特的策略、敏锐的观察力或选股能力。这部分收益被称为**超额收益**。

## 阿尔法（α）与贝塔（β）的定义

阿尔法和贝塔正是用来分别衡量上述两部分收益的指标。

*   **贝塔（β）**：衡量策略收益对市场波动的敏感性。它反映了策略跟随市场“随波逐流”的程度。**β值越高，说明策略收益受市场整体走势的影响越大**。
*   **阿尔法（α）**：衡量策略超越市场基准的超额收益。它代表了与市场波动无关的、由策略自身能力带来的回报。**α值越高，说明策略的独立盈利能力越强**。

我们可以用一个简单的线性回归模型来理解它们的关系：

**策略收益 = α + β × 市场收益 + 误差**

在这个公式中：
*   `α` 是截距，代表超额收益。
*   `β` 是斜率，代表策略对市场收益的敏感度。

通过历史数据拟合这个方程，就可以估算出特定策略的α和β值。

## 为何更关注阿尔法（α）？

![](img/2f40a668ac7186327f48ca801ee6d0d8_2.png)

上一节我们介绍了收益的构成，本节我们来看看投资者的关注重点。

市场整体走势是个人投资者无法控制的外部因素。因此，量化策略的核心目标不是预测市场（β），而是通过独特的模型和方法，持续地获取超越市场的**超额收益（α）**。

![](img/2f40a668ac7186327f48ca801ee6d0d8_4.png)

所以，在策略评估中，我们通常更关注**阿尔法（α）** 的大小和稳定性，它直接体现了策略的附加价值。

![](img/2f40a668ac7186327f48ca801ee6d0d8_6.png)

## 其他常见评估指标简介

![](img/2f40a668ac7186327f48ca801ee6d0d8_8.png)

![](img/2f40a668ac7186327f48ca801ee6d0d8_10.png)

除了α和β，策略评估中还有其他重要指标。以下是几个关键概念的简要说明，大家了解其含义即可，无需死记公式。

![](img/2f40a668ac7186327f48ca801ee6d0d8_12.png)

*   **最大回撤**：衡量策略在历史上从任一峰值点到后续最低点的最大亏损幅度，用于评估策略的风险。
*   **夏普比率**：衡量策略每承受一单位总风险，能获得多少超额回报。公式为 `(策略收益 - 无风险收益) / 策略收益波动率`。比率越高，说明风险调整后的收益越好。
*   **基准收益**：指单纯跟踪市场指数（如沪深300）所能获得的收益。它是计算超额收益的基准线。

![](img/2f40a668ac7186327f48ca801ee6d0d8_14.png)

![](img/2f40a668ac7186327f48ca801ee6d0d8_16.png)

在实际操作中，这些指标均有成熟的Python工具包（如`empyrical`、`quantstats`）或量化平台可以直接计算，无需手动实现。

## 课程总结

![](img/2f40a668ac7186327f48ca801ee6d0d8_20.png)

本节课我们一起学习了量化策略评估的核心概念。
*   我们明白了总收益由**市场收益（β）** 和**超额收益（α）** 两部分构成。
*   我们掌握了**贝塔（β）** 衡量市场相关性，**阿尔法（α）** 衡量独立于市场的盈利能力。
*   我们了解到策略研究的核心目标是寻找并获取稳定的**阿尔法（α）**。
*   我们还简要认识了最大回撤、夏普比率等辅助评估指标。
