## 什么是量化交易？🤔

上一节我们介绍了课程目标，本节中我们来看看量化交易究竟做了一件什么事。

传统炒股需要投资者长时间盯盘，分析各种指标。即便如此，投资者也难以保证每次都赚钱。原因主要有两点：第一，人类具有主观性，可能导致判断不准确；第二，个人的精力、时间和计算能力有限，无法分析海量的股票数据或跨越十年八年的历史数据。

量化交易的目标同样是赚钱，但方法不同。它不是由人类自己来思考具体策略，而是将策略设计交给计算机。例如，我们可以让计算机基于历史数据，找出在特定时间段内（如两年）能实现收益最大化的买卖时机。这就是一种策略。

量化交易的本质是：基于已有的、不会改变的历史数据进行分析，设计出交易策略，并通过多项指标来评估该策略的好坏。这个过程常被称为“回测”。简而言之，量化交易就是让计算机在历史数据中执行数据挖掘任务，找出能赚钱的策略。

## 量化交易的核心技能 🛠️

了解了量化交易的目标后，我们来看看实现它需要掌握哪些核心技能。

量化交易是一个交叉学科，但金融学知识并非重点，只需了解即可。真正的重点在于与计算机和数据相关的技能。以下是需要掌握的核心领域：

![](img/3dab311ff3012fd2057585aa6be78ee2_7.png)

*   **机器学习算法**：用于基于数据预测未来走势或结果。
*   **统计方法**：用于计算和分析数据中的各项指标，挖掘其蕴含的信息。
*   **编程实践（如Python）**：用于将方法论转化为可执行的程序，进行数据处理和策略实现。

在量化交易中，没有绝对的必备工具。任何能帮助你处理数据、操纵数据并最终实现收益最大化的工具都是好工具。你可以将量化交易理解为数据挖掘在金融领域的具体应用。

## 总结 📝

![](img/3dab311ff3012fd2057585aa6be78ee2_9.png)

本节课中我们一起学习了量化交易的核心。量化交易是利用计算机和数据分析方法（如机器学习和统计学），基于历史数据设计并验证交易策略，以追求收益最大化的过程。它与依赖人为主观判断的传统交易方式有本质区别。掌握数据处理和编程能力，是进入量化交易领域的关键。
