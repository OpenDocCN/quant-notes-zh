# CSS选择器教程：09：组合选择器 🧩

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_1.png)

在本节课中，我们将学习CSS中的组合选择器。组合选择器允许我们基于元素之间的特定关系（如父子、兄弟关系）或同时选择多个元素来应用样式，从而实现更精确和高效的样式控制。

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_3.png)

## 概述与布局准备

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_5.png)

上一节我们介绍了基础选择器，本节中我们来看看如何组合它们。为了演示组合选择器的效果，我们需要先创建一个包含嵌套结构的HTML布局。

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_7.png)

以下是我们将使用的HTML结构代码，它包含三层嵌套的`div`元素：

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_9.png)

```html
<div id="layer1" class="layer">
    <p>第一层段落</p>
    <div id="layer2" class="layer">
        <p>第二层段落</p>
        <hr>
        <div id="layer3" class="layer">
            <p>第三层段落</p>
            <h3>第三层标题</h3>
        </div>
    </div>
</div>
<p>外部段落</p>
```

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_11.png)

为了清晰地区分每一层，我们为每个层级的`div`添加了边框和内边距样式。内边距（`padding`）是指元素内容与边框之间的空间。

以下是应用的CSS样式：

```css
.layer {
    border: 2px solid;
    padding: 30px;
    margin-bottom: 20px;
}
#layer1 {
    width: 500px;
    border-color: blue;
}
#layer2 {
    border-color: red;
}
#layer3 {
    border-color: green;
}
```

应用样式后，页面将显示三个颜色不同、带有内边距的嵌套方框，便于我们观察选择器的效果。

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_13.png)

## 后代选择器

后代选择器用于选择某个元素内部的所有特定后代元素，不论它们嵌套得多深。

其语法格式为：**`祖先元素 后代元素 { 样式声明 }`**

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_15.png)

例如，如果我们想为`#layer1`这个`div`内部的所有`p`标签设置背景色，可以使用后代选择器。

```css
#layer1 p {
    background-color: yellow;
}
```

执行上述代码后，`#layer1`内部的三个`p`标签（分别位于第一、二、三层）的背景都会变成黄色。这就是后代选择器的作用：选择所有子孙后代。

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_17.png)

## 子元素选择器

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_19.png)

子元素选择器（也称为儿子选择器）比后代选择器更精确，它只选择指定元素的直接子元素，而不会选择更深层的后代。

其语法格式为：**`父元素 > 子元素 { 样式声明 }`**

假设我们只想为`#layer1`的直接子`p`标签（即第一层的段落）设置样式，而不影响第二层和第三层的段落。

```css
#layer1 > p {
    background-color: lightblue;
}
```

此时，只有第一层的`p`标签背景会变为浅蓝色。第二层和第三层的`p`标签不受影响，因为它们不是`#layer1`的直接子元素。

## 相邻兄弟选择器

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_21.png)

相邻兄弟选择器用于选择紧接在另一个元素后的兄弟元素，且二者有相同的父元素。

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_23.png)

其语法格式为：**`元素A + 元素B { 样式声明 }`**

它选择的是**元素A后面紧邻的第一个兄弟元素B**。

例如，在第二层`div`中，有一个`p`标签和一个`hr`标签。如果我们选中了`p`标签，并想给紧随其后的`hr`标签添加样式：

```css
#layer2 > p + hr {
    border: 3px dashed orange;
}
```

这段代码会将第二层中紧跟在`p`标签后面的那个`hr`元素的边框设置为橙色虚线。它不会选择`p`标签之前的元素，也不会选择更远的兄弟元素。

## 多元素组合选择器

多元素组合选择器用于同时为多个不同的元素应用相同的样式规则，这些元素之间用逗号分隔。

其语法格式为：**`元素A, 元素B, 元素C { 样式声明 }`**

如果我们想同时为页面中所有的`h3`标题和`p`段落设置相同的字体颜色：

```css
h3, p {
    color: darkred;
}
```

执行后，所有的`h3`和`p`元素的文字颜色都会变为深红色。这是一种批量设置样式的便捷方式。

## 总结

本节课中我们一起学习了CSS中四种重要的组合选择器：
1.  **后代选择器** (` `)：选择所有嵌套层级的后代元素。
2.  **子元素选择器** (`>`): 仅选择直接子元素。
3.  **相邻兄弟选择器** (`+`): 选择紧邻的下一个兄弟元素。
4.  **多元素组合选择器** (`,`): 同时选择多个元素应用相同样式。

![](img/901ef9a05d2cb56a1fa6f365ef51f8c6_25.png)

理解并熟练运用这些选择器，能够帮助你更精准、更高效地控制网页元素的样式。请务必通过修改示例代码来亲自尝试每种选择器的效果，以加深理解。
