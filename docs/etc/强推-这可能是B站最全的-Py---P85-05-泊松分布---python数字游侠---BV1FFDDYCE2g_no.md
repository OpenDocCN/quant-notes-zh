在本节课中，我们将要学习泊松分布。泊松分布描述了在特定时间范围、面积或体积内，某个事件发生次数的概率分布。我们将通过一个实际的例子来理解其概念和应用。

## 泊松分布的定义

![](img/da6231eb48e34b1570163b5a9858a7d9_3.png)

泊松分布是指在特定时间范围内，或指定的面积或体积内，某一事件出现次数的分布。通常情况下，我们是在一定的时间范围内来研究这个事件。它所对应的随机变量的概率分布，就称为泊松分布。

在实际工作中，有许多场景符合泊松分布。例如：
*   某个企业中，每月某设备出现故障的次数。
*   单位时间内（如早上8:30到中午12:00），到达某服务台需要服务的顾客人数。

![](img/da6231eb48e34b1570163b5a9858a7d9_5.png)

## 一个实际案例：包子铺的难题 🥟

假设某学生的婶婶开了一家包子铺，营业时间为早上6点到10点。她遇到了一个经营难题：有时包子蒸少了，不到8点就卖完，影响了生意；有时包子蒸多了，卖不完造成浪费。她想知道，每天到底蒸多少个包子最合适。

以下是过去一周的销售数据：
*   周一：30个
*   周二：70个
*   周三：40个
*   周四：60个
*   周五：50个

面对这个问题，我们首先想到的是计算平均值。平均值为 `(30+70+40+60+50)/5 = 50` 个。但如果每天只准备50个包子，周二和周四的需求就无法满足，五天中有两天生意受影响，这个方案并不理想。

## 泊松分布的推导思路

![](img/da6231eb48e34b1570163b5a9858a7d9_11.png)

![](img/da6231eb48e34b1570163b5a9858a7d9_13.png)

上一节我们介绍了实际问题，本节中我们来看看如何运用泊松分布的思想来解决它。

我们可以把6点到10点这个营业时间段，想象成一条有长度的线。假设把这条线分成N个非常小的时间段（即N等份），在每个微小时间段内，一个包子要么被卖出，要么没被卖出。这就像进行了N次独立的伯努利试验（类似抛N次硬币，每次只有“卖出”或“未卖出”两种结果）。

因此，在N个时间段内卖出K个包子的概率，最初可以用二项分布来描述：

**公式：** `P(X=K) = C(N, K) * p^K * (1-p)^(N-K)`

![](img/da6231eb48e34b1570163b5a9858a7d9_15.png)

其中，`p` 是每个微小时间段内包子被卖出的概率。

接下来是关键的一步：为了使模型更精确，我们将时间段划分得无限细，即让 `N` 趋向于无穷大。同时，为了保证在有限时间内卖出包子的期望数（均值）是一个固定值（记为 `λ`），我们需要让 `p` 相应地趋向于0。通过数学上的极限推导（此处省略详细过程），上述二项分布的概率公式会演变成泊松分布的概率公式：

**公式：** `P(X=K) = (λ^K * e^(-λ)) / K!`

其中：
*   `λ` 是单位时间（本例中即一个上午）内事件发生次数（卖出包子数）的期望值（均值）。
*   `K` 是我们想要计算概率的特定发生次数。
*   `e` 是自然常数。
*   `K!` 是K的阶乘。

![](img/da6231eb48e34b1570163b5a9858a7d9_17.png)

通过计算，如果我们设定每天准备65个包子（即 `K=65`），并已知过去一周平均每天卖出50个包子（即 `λ=50`），那么通过泊松分布公式计算，能满足当天需求的概率高达约98.2%。这是一个非常可靠的水平。

## 在Excel中应用泊松分布

![](img/da6231eb48e34b1570163b5a9858a7d9_23.png)

![](img/da6231eb48e34b1570163b5a9858a7d9_25.png)

在Excel中，我们可以直接使用泊松分布函数进行计算，无需手动进行复杂的极限推导。

Excel函数为：`POISSON.DIST(x, mean, cumulative)`
*   **x**：事件发生的次数（例如，我们假设准备的包子数65）。
*   **mean**：期望值或平均值（λ，本例中为50）。
*   **cumulative**：逻辑值，决定函数形式。
    *   如果为 `TRUE`，则返回**累积分布函数**，即事件发生次数小于等于x的概率。
    *   如果为 `FALSE`，则返回**概率密度函数**，即事件发生次数恰好等于x的概率。

![](img/da6231eb48e34b1570163b5a9858a7d9_27.png)

在我们的案例中，我们想知道“准备65个包子能满足需求（即卖出数≤65）”的概率，因此使用累积分布函数。在Excel中输入：
`=POISSON.DIST(65, 50, TRUE)`
计算结果约为0.982，与之前的分析一致。

![](img/da6231eb48e34b1570163b5a9858a7d9_29.png)

## 核心总结

![](img/da6231eb48e34b1570163b5a9858a7d9_31.png)

本节课中我们一起学习了泊松分布。

![](img/da6231eb48e34b1570163b5a9858a7d9_33.png)

泊松分布本质上是二项分布的一种极限情况。在N重伯努利试验中，当单次试验成功的概率p很小，而试验次数N很大时，二项分布可以近似为泊松分布，其中参数 `λ = N * p`。

![](img/da6231eb48e34b1570163b5a9858a7d9_35.png)

![](img/da6231eb48e34b1570163b5a9858a7d9_37.png)

泊松分布适用于描述在固定时间或空间内，稀有事件发生次数的概率分布。虽然在实际业务数据分析中直接使用泊松分布的频率可能低于正态分布，但理解其原理对于处理类似“单位时间内的客流量”、“系统故障次数”等问题非常有帮助。它为我们提供了一种将离散事件模型化的强大工具。
