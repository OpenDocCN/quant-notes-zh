## 策略实现：指标计算与信号生成

上一节我们介绍了大小盘轮动策略的核心思想与指标构建。本节中，我们来看看如何将这些数学概念转化为具体的Python代码。

### 一阶与二阶导数的计算

在策略中，我们需要对滤波后的RS指标进行一阶和二阶求导。手动实现求导算法较为繁琐，我们可以借助现有的科学计算库来提高开发效率。

以下是使用`statsmodels`库进行HP滤波和最小二乘法拟合求导的关键步骤：

```python
# 假设 hp_cycle 是经过HP滤波处理后的RS指标序列
# 使用最小二乘法拟合求一阶导数
from statsmodels.regression.linear_model import OLS
import numpy as np

# 构建时间序列
t = np.arange(len(hp_cycle)).reshape(-1, 1)
# 拟合线性模型（一阶）
model_first = OLS(hp_cycle, t).fit()
first_derivative = model_first.params[0]  # 一阶导数

# 拟合二次模型（二阶）
t_squared = np.column_stack([t, t**2])
model_second = OLS(hp_cycle, t_squared).fit()
second_derivative = 2 * model_second.params[1]  # 二阶导数
```

通过上述方法，我们高效地计算出了一阶导数（`first_derivative`）和二阶导数（`second_derivative`），为后续判断做好了准备。

### 交易信号的判断逻辑

在得到一阶和二阶导数后，下一步就是根据它们判断市场状态，并生成具体的交易信号。判断逻辑基于四象限分析法：

![](img/9306ed8d9fe7f8d7ece50b165a1430c5_1.png)

以下是具体的判断条件：
*   **大盘占优**：当一阶导数大于零且二阶导数大于零时，市场处于大盘强势且加速上涨状态。此时应全仓买入大盘ETF（例如沪深300ETF）。
*   **小盘占优**：当一阶导数小于零且二阶导数小于零时，市场处于小盘强势且加速下跌（或大盘加速下跌）状态。此时应全仓买入小盘ETF（例如创业板ETF）。
*   **均衡状态**：除上述两种情况外，市场处于均衡或风格不明确状态。此时应在大盘ETF和小盘ETF上各配置50%的仓位。

```python
def get_signal(first_deriv, second_deriv):
    """
    根据一阶和二阶导数生成交易信号。
    返回: (大盘ETF权重, 小盘ETF权重)
    """
    if first_deriv > 0 and second_deriv > 0:
        # 大盘占优
        return 1.0, 0.0
    elif first_deriv < 0 and second_deriv < 0:
        # 小盘占优
        return 0.0, 1.0
    else:
        # 均衡状态
        return 0.5, 0.5
```

交易信号判断完成后，函数返回一个包含大盘和小盘ETF目标权重的元组。

## 策略实现：交易执行与回测

有了交易信号，接下来我们需要在回测框架中执行具体的买卖操作。

### 交易函数的实现

交易函数的核心是根据计算出的目标持仓比例，调整当前的实际持仓。

以下是交易函数的关键步骤：
1.  **计算信号**：调用`get_signal`函数，传入当前计算出的导数，得到目标持仓比例。
2.  **判断调仓**：比较当前持仓比例与目标持仓比例。如果不一致，则需要进行调仓。
3.  **执行交易**：调仓通常分为两步。首先，清空所有相关持仓。然后，按照目标比例，用账户可用资金重新买入相应的ETF。
4.  **更新状态**：交易完成后，更新记录当前持仓比例的全局变量，以便后续分析和下一次判断。

```python
def trade(context):
    # 计算当前交易信号
    target_large_cap_weight, target_small_cap_weight = get_signal(current_first_deriv, current_second_deriv)

    # 获取当前持仓
    current_holdings = context.portfolio.positions

    # 判断是否需要调仓
    if (current_holdings[‘沪深300ETF‘] != target_large_cap_weight) or (current_holdings[‘创业板ETF‘] != target_small_cap_weight):
        # 执行清仓
        order_target_percent(‘沪深300ETF‘, 0)
        order_target_percent(‘创业板ETF‘, 0)

        # 按目标比例买入
        cash = context.portfolio.cash
        order_value(‘沪深300ETF‘, cash * target_large_cap_weight)
        order_value(‘创业板ETF‘, cash * target_small_cap_weight)

        global current_result

```
