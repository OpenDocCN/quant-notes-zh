# 量化交易系列25研报精读：1：基于CCK模型的羊群效应研究

![](img/ec170f4502d77e8010713d9d6ed7561a_1.png)

在本节课中，我们将要学习国泰君安证券发表的一篇研究报告，该报告基于CCK模型研究股票市场的羊群效应，并构建了一个在大市值股票中表现优异的识别模型。

## 羊群效应概念

羊群效应是指个别股票的暴涨或暴跌，会引起其他相关股票收益率的联动，导致某类股票集体暴涨或暴跌。具体表现为龙头股上涨后，投资者对板块内其他未上涨的股票形成盈利预期，并开始布局，最终形成“龙头领涨，群股跟涨”的现象。如果能识别出羊群效应的发生，便有机会抓住该强趋势中蕴含的获利机会。

## 识别羊群效应的思路

![](img/ec170f4502d77e8010713d9d6ed7561a_3.png)

![](img/ec170f4502d77e8010713d9d6ed7561a_5.png)

传统的择时策略通过大盘的量价数据、宏观经济指标来预测大盘未来走势。而本报告从板块内部的微观结构，即板块成分股的表现这一角度入手，来识别羊群效应。

其核心思路是：当“群羊跟涨”现象发生时，板块内许多股票跟随上涨，这会导致板块成分股收益率的相关性增强。此时，所有股票呈现一致上涨或下跌的趋势，其收益率的离散程度会减弱。因此，我们可以通过监测这种离散程度的降低，来识别羊群效应的发生。

## CCK模型的核心思想

![](img/ec170f4502d77e8010713d9d6ed7561a_7.png)

CCK模型于2000年提出，它通过观察成分股收益率相对于市场收益率的离散程度变化，来识别羊群效应。

该模型首先证明，在理性市场（即无羊群效应）中，离散程度与市场收益率之间存在线性正相关关系。如果这种线性关系被破坏，转变为非线性关系（特别是二次关系），则表明羊群效应可能正在发生。

以下是离散程度（CSAD）的定义公式：

`CSAD_t = (1/N) * Σ |R_i,t - R_m,t|`

其中：
*   `N` 是股票数量。
*   `R_i,t` 是股票 `i` 在时间 `t` 的收益率。
*   `R_m,t` 是市场（或板块）在时间 `t` 的收益率。

![](img/ec170f4502d77e8010713d9d6ed7561a_9.png)

该公式计算了每只股票收益率与市场收益率之差的绝对值的平均值。如果所有股票与市场同步涨跌，则离散度为零。

## 模型构建与羊群效应判定

![](img/ec170f4502d77e8010713d9d6ed7561a_11.png)

在理性市场假设下，离散度（CSAD）与市场收益率（`R_m`）的关系是线性的。我们可以通过回归分析来检验这种关系。

以下是用于检验的回归方程：

![](img/ec170f4502d77e8010713d9d6ed7561a_13.png)

`CSAD_t = α + β1 * |R_m,t| + β2 * (R_m,t)^2 + ε_t`

![](img/ec170f4502d77e8010713d9d6ed7561a_15.png)

模型的核心在于二次项系数 `β2`：
*   如果 `β2` 不显著（即接近于0），则表明 `CSAD` 与 `R_m` 是线性正相关，市场理性，无显著羊群效应。
*   如果 `β2` **显著为负**，则表明 `CSAD` 与 `R_m` 的关系变为开口向下的抛物线型。这意味着在市场大幅上涨或下跌时，离散度反而下降，即股票涨跌趋同，从而判定存在**羊群效应**。

![](img/ec170f4502d77e8010713d9d6ed7561a_17.png)

上一节我们介绍了CCK模型的原理，本节中我们来看看如何将其应用于实际策略。

![](img/ec170f4502d77e8010713d9d6ed7561a_19.png)

## 策略构建步骤

以下是构建基于CCK模型的羊群效应交易策略的主要步骤：

1.  **选择候选池**：首先确定要监控的股票集合。这可以是宽基指数（如上证50、沪深300）的成分股，也可以是某个行业板块的所有股票。在实盘中，可以同时计算多个行业板块的羊群效应。

2.  **计算与拟合**：对于选定的候选池，计算其过去一段时间（例如22个交易日）的日度离散度（CSAD）和市场收益率（`R_m`）。然后，使用上述回归方程进行拟合，得到系数 `β2`。

![](img/ec170f4502d77e8010713d9d6ed7561a_21.png)

3.  **信号生成**：
    *   若 `β2` 显著为负，则判定当前存在羊群效应。
    *   进一步判断市场趋势：如果市场收益率 `R_m` 呈现上涨趋势，则生成**买入**信号；如果呈现下跌趋势，则生成**卖出或清仓**信号。

![](img/ec170f4502d77e8010713d9d6ed7561a_23.png)

4.  **执行交易**：根据信号，选择买入该板块的龙头股或相关股票。

![](img/ec170f4502d77e8010713d9d6ed7561a_25.png)

## 模型改进与回测结果

原报告对模型进行了一项重要改进：将市场收益率（`R_m`）替换为**市值因子收益率**。作者认为，市值综合反映了多种风格因子，使用市值因子收益率能更全面地捕捉市场行为。

![](img/ec170f4502d77e8010713d9d6ed7561a_27.png)

报告对2007年至2017年的数据进行了回测，并设置了严谨的交易规则（如次日均价建仓）。以下是回测的核心发现：

![](img/ec170f4502d77e8010713d9d6ed7561a_29.png)

*   **市值效应**：该策略在高市值指数（如上证50、沪深300）上的表现，显著优于低市值指数（如中证500）。作者解释，高市值股票信息透明度高，特质性信息少，价格更容易受到集体交易行为（羊群效应）的影响。
*   **趋势依赖**：策略在**市场上涨趋势**中表现更为显著，胜率较高（报告中示例胜率达88%）。
*   **收益表现**：无论是宽基指数还是行业指数，策略均能获得显著的超额收益，其中上证50指数的累计收益率表现尤为突出。

![](img/ec170f4502d77e8010713d9d6ed7561a_31.png)

## 总结

![](img/ec170f4502d77e8010713d9d6ed7561a_35.png)

本节课中我们一起学习了基于CCK模型的羊群效应识别方法。我们首先了解了羊群效应的概念，即个股联动导致板块集体变动的现象。然后，我们深入探讨了CCK模型的核心思想——通过监测股票收益率离散度的异常降低（即与市场收益率的线性关系被破坏）来识别羊群效应。接着，我们学习了如何构建回归方程 `CSAD_t = α + β1 * |R_m,t| + β2 * (R_m,t)^2 + ε_t`，并通过判断系数 `β2` 是否显著为负来生成交易信号。最后，我们回顾了该策略的构建步骤、模型改进以及回测结果，其在高市值股票和上涨市场中的优异表现为我们提供了一种有效的市场微观结构分析工具。
