## 核心概念：卷积神经网络（CNN）🧠

在传统的股票预测中，我们常使用时间序列模型。本项目引入卷积神经网络，为预测提供了新的视角。

![](img/eb98358bc91ee7fca0ab05082a0bdf66_7.png)

CNN的核心思想是通过**卷积核（滤波器）**在数据上滑动，提取局部特征。在图像处理中，它提取的是空间特征；而在股票时间序列数据中，我们可以将一维的股价序列视为“图像”，CNN用来提取不同时间尺度下的局部模式特征。

一个简单的1D CNN层操作可以用以下公式表示：

![](img/eb98358bc91ee7fca0ab05082a0bdf66_9.png)

**S(t) = (X * W)(t) = Σ X(a) · W(t - a)**

其中，`X` 是输入序列（如股价），`W` 是卷积核，`*` 代表卷积操作，`S(t)` 是在时间 `t` 的输出特征。

![](img/eb98358bc91ee7fca0ab05082a0bdf66_11.png)

在代码中，使用Keras（TensorFlow的高级API）可以这样定义一个1D CNN层：

```python
from tensorflow.keras.layers import Conv1D

![](img/eb98358bc91ee7fca0ab05082a0bdf66_13.png)

# 定义一个卷积层：64个滤波器，每个滤波器宽度为3，使用ReLU激活函数
conv_layer = Conv1D(filters=64, kernel_size=3, activation='relu', input_shape=(time_steps, features))
```

通过堆叠多个这样的卷积层和池化层，模型可以自动学习从股价历史数据中识别出对预测未来走势有用的复杂模式。

![](img/eb98358bc91ee7fca0ab05082a0bdf66_15.png)

## 系统主要功能模块 🧩

我们已经了解了核心的预测技术，接下来看看整个系统由哪些功能模块组成。

以下是本系统计划实现的主要功能：

1.  **股票数据爬虫模块**：自动从指定数据源（如新浪财经、雅虎财经）抓取股票的历史价格、成交量、财务数据等。
2.  **数据可视化与分析模块**：绘制交互式K线图，计算并展示常见技术指标（如移动平均线MA、相对强弱指数RSI、布林带Bollinger Bands）。
3.  **股票预测模块**：使用预处理后的历史数据，训练卷积神经网络模型，预测未来一段时间（如下一个交易日）的股价走势或涨跌概率。
4.  **股票推荐系统模块**：结合预测结果、基本面分析（市盈率、市净率）和市场情绪，生成个性化的股票投资建议列表。
5.  **量化交易回测模块**：根据设定的交易策略（例如，基于CNN预测信号的简单策略），在历史数据上模拟交易，计算策略的收益、夏普比率、最大回撤等关键指标。
