数据中，每个词条名称包含语言标识符（如`zh`代表中文，`en`代表英文）。我们可以使用正则表达式提取这些标识符并进行统计。

![](img/627674b040a84d319077ce0f2c4f4bc9_56.png)

![](img/627674b040a84d319077ce0f2c4f4bc9_58.png)

![](img/627674b040a84d319077ce0f2c4f4bc9_60.png)

![](img/627674b040a84d319077ce0f2c4f4bc9_62.png)

```python
import re
from collections import Counter

![](img/627674b040a84d319077ce0f2c4f4bc9_64.png)

![](img/627674b040a84d319077ce0f2c4f4bc9_66.png)

# 假设词条名称在 `page` 列中
def get_language(page):
    # 使用正则匹配语言代码，例如 ‘zh.wikipedia.org’
    res = re.search(r'[a-z][a-z].wikipedia.org', page)
    if res:
        return res.group(0)[0:2] # 返回前两个字符，即语言代码
    return ‘na’ # 无法匹配的返回 ‘na’

![](img/627674b040a84d319077ce0f2c4f4bc9_68.png)

# 应用函数并统计
languages = train_df[‘page’].apply(get_language)
language_counts = Counter(languages)
print(language_counts)
```

![](img/627674b040a84d319077ce0f2c4f4bc9_70.png)

统计结果显示，英文（en）词条数量最多，约2万多个；日文（ja）约2万多个；中文（zh）约1万多个，还有其他语言如俄文（ru）等。

![](img/627674b040a84d319077ce0f2c4f4bc9_72.png)

![](img/627674b040a84d319077ce0f2c4f4bc9_74.png)

从可视化结果可以明显看出：
1.  英文词条的总点击量远高于其他语言。
2.  俄文（ru）词条在约第400天（从2015年7月1日起算）点击量出现异常激增，可能是由于当时发生了重大事件。
3.  中文词条的总点击量相对较低，这可能与国内用户更常使用百度百科有关。

这个分析告诉我们，不同语言的数据模式差异巨大。因此，在建模时，更合理的策略是**按语言分别建立预测模型**，而不是使用一个统一的模型。这种“分而治之”的思路在许多时间序列分析（如网络流量异常检测）中都很常见。

从单个词条的曲线中，我们可以观察到一种常见模式：许多词条在大部分时间点击量平稳，但在某个特定时间点会出现爆发式增长（成为“网红”词条）。例如，一个词条可能在前500天默默无闻，然后在第500天因某个事件关注度飙升。

![](img/627674b040a84d319077ce0f2c4f4bc9_134.png)

这种模式提示我们：
1.  预测模型需要能捕捉这种突然的“峰值”行为。
2.  我们可以通过检测点击量的剧烈变化（方差突变），来识别历史上哪些词条在何时成为了热点，进而进行事件分析或舆情研究。

## 总结与核心要点

本节课中我们一起学习了如何对维基百科点击量时间序列数据进行探索性数据分析。

![](img/627674b040a84d319077ce0f2c4f4bc9_162.png)

![](img/627674b040a84d319077ce0f2c4f4bc9_164.png)

我们首先对数据进行了初步探查和内存优化，将`float`类型转换为`int`类型。接着，我们按语言对数据进行了分类，并通过可视化发现不同语言的总点击量趋势存在显著差异，这提示我们需要**分语言建立模型**。然后，我们观察了单个词条的点击模式，发现了“平稳期”与“爆发期”并存的特点。最后，我们探讨了如何识别热点词条以及处理缺失值。

![](img/627674b040a84d319077ce0f2c4f4bc9_166.png)

**核心概念总结**：
*   **探索性数据分析（EDA）**：在正式建模前，通过统计和可视化手段理解数据分布、结构和规律的过程。
*   **数据分箱策略**：对于内部差异大的数据（如不同语言），采用`分箱（Binning）`或分组策略分别建模，通常能提升模型效果。公式表示为：`模型_总 = {模型_语言1, 模型_语言2, ...}`。
*   **缺失值处理**：对于时间序列中的缺失点，根据领域知识进行填充（如用`0`填充表示无点击）。代码表示为：`df.fillna(0)`。

![](img/627674b040a84d319077ce0f2c4f4bc9_168.png)

![](img/627674b040a84d319077ce0f2c4f4bc9_170.png)

通过本课的EDA过程，我们为后续建立更精准的点击量预测模型奠定了坚实的基础。记住，好的模型始于对数据的深刻理解。
