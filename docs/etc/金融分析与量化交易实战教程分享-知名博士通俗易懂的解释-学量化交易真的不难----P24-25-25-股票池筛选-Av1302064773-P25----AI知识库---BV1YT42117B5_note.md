## 数据预处理与查询
初始化股票池后，我们需要在每日交易开始前进行数据预处理。核心任务是获取股票的财务数据并进行筛选。

以下是实现数据预处理的关键步骤：

1.  **使用查询函数**：量化平台通常提供`query`函数来查询财务数据。你需要仔细阅读官方文档，了解可用的指标和函数用法。
2.  **构建查询语句**：在`query`中指定要查询的财务指标，例如营业收入。
3.  **应用过滤器**：使用`filter`条件将查询范围限制在我们的初始股票池（即沪深300成分股）内。
4.  **排序与限制**：对查询结果按营业收入进行降序排序，并使用`limit`只保留排名前10的股票。

![](img/4f3d7be8519bf843e3b6aa4dbcc4dcc7_21.png)

```python
def before_trading_start(context):
    # 构建查询：获取基础财务数据中的营业收入指标
    q = query(
        fundamentals.financial_indicator.operating_revenue
    ).filter(
        fundamentals.financial_indicator.code.in_(get_index_stocks('沪深300'))
    ).order_by(
        fundamentals.financial_indicator.operating_revenue.desc()
    ).limit(10)

    # 执行查询，并将结果（DataFrame格式）存入context中供后续使用
    context.fundamental_df = get_fundamentals(q)
    # 打印查看结果
    print(context.fundamental_df)
```

## 代码调试与常见问题
运行上述代码后，如果查询结果为空，需要检查问题所在。一个常见原因是在`filter`中错误地使用了股票列表。

![](img/4f3d7be8519bf843e3b6aa4dbcc4dcc7_31.png)

**问题排查**：确保用于过滤的股票代码列表是通过正确的函数（如`get_index_stocks(‘沪深300’)`）从指数中动态获取的，而不是一个静态的列表或错误的变量。

![](img/4f3d7be8519bf843e3b6aa4dbcc4dcc7_33.png)

![](img/4f3d7be8519bf843e3b6aa4dbcc4dcc7_35.png)

![](img/4f3d7be8519bf843e3b6aa4dbcc4dcc7_37.png)

修正后的`filter`部分应如下所示：
```python
.filter(
    fundamentals.financial_indicator.code.in_(get_index_stocks('沪深300'))
)
```
修正后重新运行回测，即可在日志中看到每天筛选出的营业收入排名前10的股票列表。
