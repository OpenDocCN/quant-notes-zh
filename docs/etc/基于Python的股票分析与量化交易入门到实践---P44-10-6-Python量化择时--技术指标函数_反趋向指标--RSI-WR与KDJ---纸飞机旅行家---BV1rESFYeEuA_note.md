## 📈 反趋向指标概述

上一节我们介绍了趋向指标的MA和VMA，本节中我们来看看反趋向指标。反趋向指标主要用于判断市场是否处于“超买”或“超卖”状态，从而预测价格可能出现的反转。它们通常用于分析市场的短期波动。

以下是本节课将要介绍的三个核心指标：
1.  **RSI（相对强弱指标）**
2.  **WR（威廉指标）**
3.  **KDJ（随机指标）**

## 1. RSI（相对强弱指标）

RSI是期货和股票市场中非常著名的摆动震荡指标。它表征的是股价向上波动的幅度占总波动幅度的百分比。如果RSI数值大，表示市场处于强势；如果数值小，表示市场处于弱势。

### 核心公式

RSI的计算公式如下：
`RSI(N) = A / (A + B) * 100`
其中：
*   **A** = N日内收盘价涨幅之和
*   **B** = N日内收盘价跌幅之和

RSI的本质是N日内收盘价涨幅与跌幅的比值。向上力量大，比值就高，市场处于强势；向下力量大，指标下降，市场处于弱势。

### RSI的应用

以下是RSI常见的几种应用场景：

*   **超买超卖**：RSI大于80时为超买，是卖出信号；RSI小于20时为超卖，是买入信号。
*   **多空分界线**：RSI以50为中介线。大于50为多头行情，小于50为空头行情。
*   **形态反转**：RSI在80以上形成M头或头肩顶形态时，视为向下反转信号；在20以下形成W底或头肩底形态时，视为向上反转信号。
*   **趋势线突破**：RSI向上突破其高点连线时买进，向下跌破其低点连线时卖出。

### Python代码实现

RSI的实现需要借助技术分析库。以下是使用示例：

```python
# 导入技术分析库
from jqdatasdk import technical_analysis as ta

# 计算单只股票某一天的RSI值（参数N1通常取6）
stock_code = ‘000001.XSHE‘
date = ‘2022-09-01‘
rsi_value = ta.RSI(stock_code, check_date=date, N1=6)
print(f“股票 {stock_code} 在 {date} 的RSI值为：{rsi_value}”)
```
RSI函数返回一个字典，其中包含计算出的RSI值。

## 2. WR（威廉指标）

威廉指标主要用于研究股价的短期波动，通过分析股价波动中的峰和谷来决定买卖时机。它是一个短线技术指标，在中长线分析中不常用。

### 核心公式

威廉指标的计算公式如下：
`WR(N) = (HIGH(N) - CLOSE) / (HIGH(N) - LOW(N)) * 100`
其中：
*   **HIGH(N)** = N日内的最高价
*   **LOW(N)** = N日内的最低价
*   **CLOSE** = 当日收盘价

### WR的应用

威廉指标的应用规则与RSI相反，需要特别注意：

*   **超卖买入**：当威廉指数高于80时，市场处于超卖状态，行情可能见底，是买入信号。
*   **超买卖出**：当威廉指数低于20时，市场处于超买状态，行情可能见顶，是卖出信号。

简而言之，威廉指标记住“高于80买进，低于20卖出”即可。

### Python代码实现

威廉指标的计算同样需要技术分析库，它有两个常用参数N和N1。

```python
from jqdatasdk import technical_analysis as ta

![](img/701b94bc1a411954f6db3d530638a5bd_1.png)

stock_code = ‘600031.XSHG‘
date_list = [‘2022-10-28‘, ‘2022-10-31‘, ‘2022-11-01‘, ‘2022-11-02‘]

for date in date_list:
    wr_result = ta.WR(stock_code, check_date=date, N=14, N1=28)
    # 返回结果包含WR值和其均线MAWR
    print(f“日期 {date}: WR={wr_result[‘WR‘]}, MAWR={wr_result[‘MAWR‘]}”)
```
函数返回一个字典，通常包含`WR`（威廉指标值）和`MAWR`（威廉指标均线）两个键值。

## 3. KDJ（随机指标）

KDJ指标充分考虑了价格波动的随机振幅与中短期波动，在预测市场短期超买超卖方面比RSI更加敏感，是目前非常常用的技术指标。

### 核心算法

KDJ的计算稍复杂，分为几步：
1.  计算未成熟随机值RSV：
    `RSV(N) = (CLOSE - LOW(N)) / (HIGH(N) - LOW(N)) * 100`
2.  计算K值（快速随机值）：
    `K = 2/3 * 前一日K值 + 1/3 * 当日RSV`
3.  计算D值（慢速随机值）：
    `D = 2/3 * 前一日D值 + 1/3 * 当日K值`
4.  计算J值：
    `J = 3 * K - 2 * D`
    （若无前一日K/D值，可用50代替）

### KDJ的应用

以下是KDJ的几种常见应用：

*   **超买超卖**：K、D值大于80时，回档（下跌）几率大；小于20时，反弹（上涨）几率大。
*   **交叉信号**：K在20左右向上交叉D，视为买进信号；K在80左右向下交叉D，视为卖出信号。
*   **J值极端**：J值大于100时，股价易反转下跌；J值小于0时，股价易反转上涨。
*   **无效区间**：KDJ在50左右波动时，其指示意义不大。

### Python代码实现

![](img/701b94bc1a411954f6db3d530638a5bd_3.png)

KDJ函数有三个重要参数：N（计算RSV的天数）、M1（K值的平均天数）、M2（D值的平均天数）。

```python
from jqdatasdk import technical_analysis as ta

stock_code = ‘000001.XSHE‘
date_list = [‘2022-09-05‘, ‘2022-09-06‘, ‘2022-09-07‘]

for date in date_list:
    kdj_result = ta.KDJ(stock_code, check_date=date, N=9, M1=3, M2=3)
    print(f“日期 {date}: K={kdj_result[‘K‘]}, D={kdj_result[‘D‘]}, J={kdj_result[‘J‘]}”)
```
函数返回一个包含`K`、`D`、`J`三个值的字典。

## 💻 代码实战小结

在代码实战环节，我们分别演示了RSI、WR和KDJ指标的计算：
*   **RSI**：使用`ta.RSI()`函数，主要参数为`N1`。
*   **WR**：使用`ta.WR()`函数，主要参数为`N`和`N1`。

## 📚 本章总结

本节课中我们一起学习了三种经典的反趋向技术指标：

1.  **RSI（相对强弱指标）**：通过计算N日内涨跌幅度的比率来判断市场强弱。大于80超买，小于20超卖，50为多空分界线。
2.  **WR（威廉指标）**：通过分析股价在N日波动区间中的位置来判断超买超卖。其规则与RSI相反，**高于80买进，低于20卖出**。
3.  **KDJ（随机指标）**：通过计算未成熟随机值（RSV）并对其进行平滑处理得到K、D、J值，对短期波动非常敏感。同样关注80以上的超买区和20以下的超卖区，并观察K、D线的交叉情况。
