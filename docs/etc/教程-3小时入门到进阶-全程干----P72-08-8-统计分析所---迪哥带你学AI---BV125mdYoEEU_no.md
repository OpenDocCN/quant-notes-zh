# Python金融分析与量化交易实战：P72：08-8-统计分析所需数据准备 📊

在本节课中，我们将学习如何为统计分析准备数据。我们将基于历史股票数据，统计“前两天”和“前一天”的涨跌情况，并观察这些情况与“当天”实际涨跌之间的关系，从而为后续基于概率的交易策略打下基础。

上一节我们介绍了特征工程，本节中我们来看看如何将这些特征转化为可用于统计分析的形式。

## 核心思路与数据准备

我们的目标是：基于“前两天”和“前一天”的股价变化（涨或跌），统计“当天”出现涨、跌或平盘的概率。这听起来像是一种简单的历史统计方法，虽然它过于传统，并非现代量化交易的主流，但能帮助我们熟悉在Python中处理此类问题的流程。

![](img/58ba3d0ad8091045d362144546ce5d58_1.png)

![](img/58ba3d0ad8091045d362144546ce5d58_3.png)

![](img/58ba3d0ad8091045d362144546ce5d58_5.png)

以下是实现此目标的核心步骤：
1.  将连续的价格变化值（如涨跌幅）转换为离散的类别（涨=1， 跌=0）。
2.  基于转换后的“前两天”和“前一天”数据，结合“当天”的实际方向，进行分组统计。

![](img/58ba3d0ad8091045d362144546ce5d58_7.png)

## 第一步：数据转换（连续值转离散值）

![](img/58ba3d0ad8091045d362144546ce5d58_9.png)

首先，我们需要一个函数，将我们之前计算好的特征（如`L1`代表前一天涨跌幅，`L2`代表前两天涨跌幅）从连续数值转换为代表涨跌的0和1。

![](img/58ba3d0ad8091045d362144546ce5d58_11.png)

![](img/58ba3d0ad8091045d362144546ce5d58_13.png)

我们定义一个名为 `create_bins` 的函数来完成这个任务：

![](img/58ba3d0ad8091045d362144546ce5d58_15.png)

![](img/58ba3d0ad8091045d362144546ce5d58_17.png)

```python
def create_bins(data, bins=[0]):
    global col_names
    col_names = list(data.columns)
    for col in col_names:
        if ‘L’ in col: # 假设我们的特征列名都包含‘L’，如L1, L2
            bin_name = col + ‘_b’
            data[bin_name] = np.digitize(data[col], bins=bins)
```

![](img/58ba3d0ad8091045d362144546ce5d58_19.png)

![](img/58ba3d0ad8091045d362144546ce5d58_21.png)

**代码解释**：
*   `np.digitize(data[col], bins=[0])`：这个函数将数据根据给定的边界`bins`进行分箱。这里`bins=[0]`表示以0为分界点。
*   如果`data[col]`的值大于0，则`np.digitize`返回1（表示“涨”）。
*   如果`data[col]`的值小于或等于0，则返回0（表示“跌”或“平”）。
*   我们将转换后的结果以新列（如`L1_b`, `L2_b`）的形式存入原始数据框`data`中。

![](img/58ba3d0ad8091045d362144546ce5d58_23.png)

执行这个函数：

```python
create_bins(data)
```

![](img/58ba3d0ad8091045d362144546ce5d58_25.png)

然后，我们可以查看转换后的部分数据：

![](img/58ba3d0ad8091045d362144546ce5d58_27.png)

```python
print(data[[‘L1_b’, ‘L2_b’]].head())
```

![](img/58ba3d0ad8091045d362144546ce5d58_29.png)

输出结果中，`1`代表对应时期上涨，`0`代表下跌或持平。例如，第一行`L1_b=1, L2_b=0`表示“前一天涨了，前两天跌了”。

![](img/58ba3d0ad8091045d362144546ce5d58_31.png)

![](img/58ba3d0ad8091045d362144546ce5d58_33.png)

## 第二步：结合实际方向进行分组统计

![](img/58ba3d0ad8091045d362144546ce5d58_35.png)

为了进行统计分析，我们还需要“当天”的实际涨跌情况。原始数据中通常有一列名为 `direction`，它记录了当天价格的实际变动方向（例如，1=涨， -1=跌， 0=平）。

![](img/58ba3d0ad8091045d362144546ce5d58_37.png)

现在，我们有了：
*   自变量：`L1_b`（前一天状态）， `L2_b`（前两天状态）。
*   因变量：`direction`（当天实际状态）。

![](img/58ba3d0ad8091045d362144546ce5d58_39.png)

接下来，我们需要统计在各种`(L1_b, L2_b)`组合下，`direction`分别取不同值（涨、跌、平）的次数。

![](img/58ba3d0ad8091045d362144546ce5d58_41.png)

Pandas库中的`groupby`方法非常适合完成这个任务。以下是具体操作：

![](img/58ba3d0ad8091045d362144546ce5d58_43.png)

![](img/58ba3d0ad8091045d362144546ce5d58_45.png)

```python
grouped = data.groupby([‘L1_b’, ‘L2_b’, ‘direction’])
```

![](img/58ba3d0ad8091045d362144546ce5d58_47.png)

![](img/58ba3d0ad8091045d362144546ce5d58_49.png)

这行代码的意思是：按照`L1_b`、`L2_b`和`direction`这三列的唯一组合对数据进行分组。

![](img/58ba3d0ad8091045d362144546ce5d58_51.png)

![](img/58ba3d0ad8091045d362144546ce5d58_53.png)

为了查看每个分组有多少条数据（即每种情况出现的频次），我们使用`.size()`方法：

![](img/58ba3d0ad8091045d362144546ce5d58_55.png)

```python
print(grouped.size())
```

![](img/58ba3d0ad8091045d362144546ce5d58_57.png)

![](img/58ba3d0ad8091045d362144546ce5d58_59.png)

**输出结果解读**：
输出是一个Series，索引是`(L1_b, L2_b, direction)`的组合，值是该组合在历史数据中出现的次数。
例如，`(0, 0, 1)` 表示“前两天跌(`L2_b=0`)，前一天跌(`L1_b=0`)，但当天实际涨了(`direction=1`)”这种情况在历史上发生了多少次。
如果某个组合（如`(0, 0, 0)`）没有出现在输出中，意味着在历史数据中从未出现过“前两天跌，前一天跌，当天实际平盘”的情况。

![](img/58ba3d0ad8091045d362144546ce5d58_61.png)

## 总结

![](img/58ba3d0ad8091045d362144546ce5d58_63.png)

本节课中我们一起学习了为简单的统计分析准备数据的过程。
1.  我们首先通过`np.digitize`函数，将连续的涨跌幅特征（`L1`, `L2`）离散化为二分类变量（`L1_b`, `L2_b`），用1和0分别代表“涨”和“跌”。
2.  接着，我们利用Pandas的`data.groupby`方法，按照“前两天状态(`L2_b`)”、“前一天状态(`L1_b`)”和“当天实际状态(`direction`)”进行分组，并通过`.size()`得到了每种历史情况发生的频次。

![](img/58ba3d0ad8091045d362144546ce5d58_65.png)

这些频次数据是计算条件概率的基础。例如，我们可以计算当`(L1_b=0, L2_b=0)`时，`direction=1`（上涨）的概率，即用`(0,0,1)`的频次除以所有`(0,0,*)`的频次总和。这就构成了一个最基础的、基于历史频率的统计预测模型。虽然模型简单，但完整展示了从数据处理到统计分析的量化分析流程。
