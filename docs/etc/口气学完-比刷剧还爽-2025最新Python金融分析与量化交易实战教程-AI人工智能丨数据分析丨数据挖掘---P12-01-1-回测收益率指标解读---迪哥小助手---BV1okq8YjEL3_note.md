# Python金融量化：P12：01-1：回测收益率指标解读 📈

在本节课中，我们将要学习策略评估的核心指标之一——回测收益率。理解这个指标是判断一个交易策略是否有效的基础。

![](img/a15f85f3b6c1d38957452839026d6c9b_1.png)

当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以判断该策略是盈利还是亏损。然而，仅用“赚”或“赔”来描述是不够的。我们需要更精确的指标来回答一系列问题：具体盈利了多少？与大盘相比有何优势？执行该策略会面临哪些风险？这些风险是否值得承担？以及，我们的策略与其他策略相比有何优势？这些问题的答案，就构成了策略评估指标。

上一节我们介绍了策略评估的必要性，本节中我们来看看最基础的评估指标——回测收益率。

![](img/a15f85f3b6c1d38957452839026d6c9b_3.png)

## 理解回测收益率

![](img/a15f85f3b6c1d38957452839026d6c9b_5.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_7.png)

回测收益率是最基本、最常用的策略评估指标。它衡量的是，在回测周期内，策略最终获得的收益相对于初始本金的增长比例。

![](img/a15f85f3b6c1d38957452839026d6c9b_9.png)

例如，你基于2015年至2020年的某只股票数据，设计了一个买卖策略。回测收益率告诉你，经过这些年的操作，你的最终资产相对于初始本金增加了多少百分比。如果结果是4.32%，就意味着你的策略让你的本金增长了约4.32%。

![](img/a15f85f3b6c1d38957452839026d6c9b_11.png)

其计算公式如下：

![](img/a15f85f3b6c1d38957452839026d6c9b_13.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_15.png)

**回测收益率 = (期末资产 - 期初资产) / 期初资产**

![](img/a15f85f3b6c1d38957452839026d6c9b_17.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_19.png)

或者等价于：

![](img/a15f85f3b6c1d38957452839026d6c9b_21.png)

**回测收益率 = (期末资产 / 期初资产) - 1**

![](img/a15f85f3b6c1d38957452839026d6c9b_23.png)

其中，`期末资产` 可以理解为策略结束时的卖出总价值，`期初资产` 则是策略开始时的买入总成本。

## 动手计算回测收益率

![](img/a15f85f3b6c1d38957452839026d6c9b_25.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_27.png)

为了让大家更直观地理解，我们将使用Python代码，基于一份示例股票数据来计算并可视化回测收益率的变化过程。

首先，我们需要导入必要的工具包并加载数据。

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便后续图表显示
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，并指定日期列为索引
data = pd.read_csv('your_stock_data.csv', index_col=0)
print(data.head())
```

假设我们的数据包含了股票每日的收盘价。接下来，我们将计算每日资产相对于期初的比值，并最终得出总的回测收益率。

![](img/a15f85f3b6c1d38957452839026d6c9b_29.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_31.png)

以下是计算步骤：
1.  计算每日收盘价与第一期收盘价的比值。
2.  将此比值序列减1，即可得到每日相对于期初的收益率序列。
3.  序列的最后一个值，即为整个回测周期的总收益率。

![](img/a15f85f3b6c1d38957452839026d6c9b_33.png)

```python
# 1. 计算每日价格与期初价格的比值
price_ratio = data['close'] / data['close'].iloc[0]

![](img/a15f85f3b6c1d38957452839026d6c9b_35.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_37.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_39.png)

# 2. 可视化资产相对变化曲线
plt.figure(figsize=(12, 6))
price_ratio.plot()
plt.title('资产相对期初价值变化曲线')
plt.xlabel('日期')
plt.ylabel('相对价值 (期初=1)')
plt.grid(True)
plt.show()

# 3. 计算总回测收益率
total_return = price_ratio.iloc[-1] - 1

![](img/a15f85f3b6c1d38957452839026d6c9b_41.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_43.png)

# 4. 将结果整理成表格展示
result_df = pd.DataFrame({
    '回测收益率': [total_return]
}, index=['策略结果'])
print(result_df)
```

![](img/a15f85f3b6c1d38957452839026d6c9b_45.png)

![](img/a15f85f3b6c1d38957452839026d6c9b_47.png)

通过运行以上代码，你可以看到一条曲线，它展示了你的“资产”在整个回测周期内相对于初始本金（设为1）的波动情况。最终，`result_df` 会输出一个具体的百分比数值，这就是你的策略在该历史数据上的回测收益率。

![](img/a15f85f3b6c1d38957452839026d6c9b_49.png)

本节课中我们一起学习了回测收益率这一核心概念。我们了解到，回测收益率是评估策略盈利能力的直接指标，其计算公式简单明了。通过Python实战，我们掌握了如何从价格数据中计算并可视化这一指标的过程。理解它是读懂更复杂评估报告的第一步。在后续课程中，我们将基于专业的回测平台，自动获得包括回测收益率在内的一系列评估指标，而无需再手动计算。
