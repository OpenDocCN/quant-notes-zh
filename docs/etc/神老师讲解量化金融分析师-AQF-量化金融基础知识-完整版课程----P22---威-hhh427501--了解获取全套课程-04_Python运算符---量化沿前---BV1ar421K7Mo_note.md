在本节课中，我们将要学习Python中的各种运算符。运算符是编程语言中用于执行运算的符号，它们是构建逻辑和进行计算的基础。我们将从最基础的算术运算符开始，逐步深入到比较、赋值、逻辑等更复杂的运算符，并通过实际例子帮助大家理解它们的用法。

## 算术运算符 ➕➖✖️➗

算术运算符用于执行基本的数学运算，如加、减、乘、除等。

以下是Python中主要的算术运算符及其功能：

*   **`+`**：加法。
*   **`-`**：减法。在数字前使用表示负数。
*   **`*`**：乘法。
*   **`/`**：除法。
*   **`%`**：取模，返回除法的余数。例如，`10 % 3` 的结果是 `1`。
*   **`**`**：幂运算。例如，`2 ** 3` 表示2的3次方，结果是 `8`。
*   **`//`**：取整除，返回商的整数部分。例如，`4 // 3` 的结果是 `1`。

对于序列（如字符串、列表、元组），加法和乘法有特殊含义：
*   **序列加法**：实现序列的拼接。例如，`[1, 2, 3] + [4, 5, 6]` 的结果是 `[1, 2, 3, 4, 5, 6]`。
*   **序列乘法**：实现序列的重复。例如，`[1, 2, 3] * 3` 的结果是 `[1, 2, 3, 1, 2, 3, 1, 2, 3]`。

除了内置运算符，我们还可以导入 `math` 模块进行更高级的数学运算，例如计算对数：
```python
import math
result = math.log(10) # 计算10的自然对数
```

## 比较运算符 ⚖️

比较运算符用于比较两个值，并返回一个布尔值（`True` 或 `False`）。

![](img/fbed1605c6d88ca6278d142aeccd7618_5.png)

以下是Python中的比较运算符：

*   **`==`**：判断两个对象是否相等。**注意**：判断相等使用两个等号。
*   **`!=`**：判断两个对象是否不相等。
*   **`>`**：判断左侧对象是否大于右侧对象。
*   **`<`**：判断左侧对象是否小于右侧对象。
*   **`>=`**：判断左侧对象是否大于或等于右侧对象。
*   **`<=`**：判断左侧对象是否小于或等于右侧对象。

**示例**：
假设 `close` 表示今日收盘价（值为10），`pre_close` 表示昨日收盘价（值为9）。
```python
close == pre_close # 返回 False，因为10不等于9
close > pre_close  # 返回 True，因为10大于9
```

## 赋值运算符 🏷️

赋值运算符用于为变量赋值。最基本的赋值运算符是 `=`。

赋值运算符还可以与算术运算符结合，形成复合赋值运算符，使代码更简洁。

以下是复合赋值运算符的示例：

*   **`+=`**：`c += a` 等价于 `c = c + a`
*   **`-=`**：`c -= a` 等价于 `c = c - a`
*   **`*=`**：`c *= a` 等价于 `c = c * a`
*   **`/=`**：`c /= a` 等价于 `c = c / a`
*   **`%=`**：`c %= a` 等价于 `c = c % a`
*   **`**=`**：`c **= a` 等价于 `c = c ** a`
*   **`//=`**：`c //= a` 等价于 `c = c // a`

**实际应用示例（计算持仓收益）**：
假设第一天收益率 `r1 = 0.05`，第二天收益率 `r2 = 0.1`，初始净值 `nav = 1`。
```python
nav *= (1 + r1) # 第一天后净值变为 1.05
nav *= (1 + r2) # 第二天后净值变为 1.155
```
使用 `*=` 运算符可以简洁地更新净值。

## 逻辑运算符 🔗

逻辑运算符用于组合多个布尔条件，主要包括与（and）、或（or）、非（not）。

以下是逻辑运算符的规则：

*   **`and`**：逻辑“与”。只有当 `x` 和 `y` 都为 `True` 时，`x and y` 才返回 `True`。
*   **`or`**：逻辑“或”。只要 `x` 或 `y` 中有一个为 `True`，`x or y` 就返回 `True`。
*   **`not`**：逻辑“非”。取反操作，`not True` 返回 `False`，`not False` 返回 `True`。

**示例（多条件选股）**：
假设某股票数据：`roe = 0.12`， `pe = 20`， `pb = 2`。
*   选出 `roe > 0.1` **或** `pe < 15` 的股票：`(roe > 0.1) or (pe < 15)` 返回 `True`。
*   选出 `roe > 0.1` **且** `pe < 15` 的股票：`(roe > 0.1) and (pe < 15)` 返回 `False`。

当有多个复杂条件时，建议使用括号 `()` 明确运算顺序，避免依赖运算符优先级导致混淆。

**进阶技巧：统计满足条件的个数**
有时我们需要判断多个条件中至少满足N个。可以利用布尔值 `True=1`， `False=0` 的特性：
```python
# 判断 roe>0.1, pe<15, pb<3 这三个条件中是否至少满足两个
condition_count = (roe > 0.1) * 1 + (pe < 15) * 1 + (pb < 3) * 1
is_selected = (condition_count >= 2) # 如果满足条件数>=2，则被选中
```

## 成员运算符 🔍

![](img/fbed1605c6d88ca6278d142aeccd7618_9.png)

成员运算符用于测试一个元素是否存在于某个序列（如字符串、列表、元组）中。

![](img/fbed1605c6d88ca6278d142aeccd7618_11.png)

以下是成员运算符：

![](img/fbed1605c6d88ca6278d142aeccd7618_12.png)

*   **`in`**：如果在指定的序列中找到值，则返回 `True`。
*   **`not in`**：如果在指定的序列中没有找到值，则返回 `True`。

**示例**：
```python
my_list = [1, 2, 3]
print(1 in my_list)     # 输出: True
print(4 in my_list)     # 输出: False
```

![](img/fbed1605c6d88ca6278d142aeccd7618_14.png)

![](img/fbed1605c6d88ca6278d142aeccd7618_16.png)

**实际应用示例（验证股票代码）**：
```python
# 假设 get_all_stocks() 返回所有正常上市股票的代码列表
listed_stocks = [‘000001‘, ‘000002‘, ‘688001‘, ...]
code_to_check = ‘688001‘
print(code_to_check in listed_stocks) # 输出: True，说明是正常上市股票
```

## 身份运算符 👥

![](img/fbed1605c6d88ca6278d142aeccd7618_20.png)

身份运算符用于比较两个对象的内存地址（即是否是同一个对象），而不仅仅是比较它们的值。

![](img/fbed1605c6d88ca6278d142aeccd7618_22.png)

以下是身份运算符：

*   **`is`**：判断两个变量是否引用自同一个对象。等价于 `id(a) == id(b)`。
*   **`is not`**：判断两个变量是否引用自不同的对象。

![](img/fbed1605c6d88ca6278d142aeccd7618_24.png)

**重要区别**：`is` 与 `==`
*   **`is`** 判断两个变量是否指向内存中的**同一个对象**（身份相同）。
*   **`==`** 判断两个变量的**值是否相等**。

**示例**：
```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a == b) # True，值相等
print(a is b) # False，是不同的对象（内存地址不同）
print(a is c) # True，c和a指向同一个列表对象
```
对于数字和字符串等不可变类型，Python会进行优化，值相同时 `is` 也可能返回 `True`。但对于列表、字典等可变类型，`is` 和 `==` 的结果通常不同。

## 运算符优先级 📊

不同的运算符具有不同的优先级，这决定了复杂表达式中运算的执行顺序。例如，乘除 (`*`, `/`) 优先于加减 (`+`, `-`)，逻辑非 (`not`) 优先于逻辑与 (`and`)，逻辑与优先于逻辑或 (`or`)。

然而，依赖记忆优先级容易出错。为了保证代码清晰且无误，**最推荐的做法是使用括号 `()` 来明确指定运算顺序**。这样既便于自己阅读，也方便他人理解。

**示例**：
```python
# 使用括号明确优先级，清晰易懂
result = (x > 5) and ((y < 10) or (z == 0))
```

## 总结 📝

本节课中我们一起学习了Python中六类核心运算符：
1.  **算术运算符**：用于数学计算，如 `+`, `-`, `*`, `/`, `%`, `**`, `//`。
2.  **比较运算符**：用于比较值，返回布尔结果，如 `==`, `!=`, `>`, `<`, `>=`, `<=`。
3.  **赋值运算符**：用于给变量赋值，包括基础赋值 `=` 和复合赋值如 `+=`, `*=`。
4.  **逻辑运算符**：用于组合布尔条件，包括 `and`, `or`, `not`。
5.  **成员运算符**：用于判断元素是否在序列中，包括 `in`, `not in`。
6.  **身份运算符**：用于判断两个变量是否引用同一对象，包括 `is`, `is not`。注意与 `==` 的区别。
