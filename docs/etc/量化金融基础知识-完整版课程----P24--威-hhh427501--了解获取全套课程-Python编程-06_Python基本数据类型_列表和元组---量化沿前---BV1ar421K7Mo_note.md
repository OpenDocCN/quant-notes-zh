# Python编程：06：Python基本数据类型 - 列表和元组 🐍

在本节课中，我们将要学习Python中两种重要的基本数据类型：列表（List）和元组（Tuple）。它们是用于存储多个数据项的容器，在量化金融分析中常用于存储价格序列、股票代码等数据。我们将从创建、访问、修改等多个维度来掌握它们。

## 列表（List）的创建与基本概念 📝

列表是Python中最常用的可变序列类型。它使用方括号 `[]` 创建，元素之间用逗号分隔。列表中的元素类型可以不同，支持数字、字符串，甚至其他列表（即嵌套列表）。

以下是列表创建的示例：
```python
my_list = [1, ‘A‘, [1, 2]]
```
在这个例子中，列表 `my_list` 包含三个元素：第一个是数字 `1`，第二个是字符串 `‘A‘`，第三个是另一个列表 `[1, 2]`。注意，嵌套的列表 `[1, 2]` 被视为一个整体，是外层列表的第三个元素。

## 列表的索引与切片 🔍

和字符串一样，列表作为序列类型，同样支持索引和切片操作。索引用于访问单个元素，切片用于获取一个子列表。基本格式为 `列表名[索引]` 或 `列表名[起始索引:结束索引]`，切片遵循左闭右开原则。

以下是索引和切片的操作示例：
```python
my_list = [1, ‘A‘, [1, 2]]
# 索引
print(my_list[0])  # 输出: 1
print(my_list[-1]) # 输出: [1, 2]

# 嵌套列表取值
print(my_list[-1][0]) # 输出: 1

# 切片
print(my_list[:2]) # 输出: [1, ‘A‘]
```
与字符串不同，列表中的元素是可以改变的。我们可以通过赋值直接更新列表中的元素。
```python
my_list[0] = 2
print(my_list) # 输出: [2, ‘A‘, [1, 2]]
```

## 列表的运算符 ➕✖️

列表支持加法（拼接）和乘法（重复）运算，同时也支持成员运算符 `in` 和 `not in` 来判断元素是否存在于列表中。

以下是运算符的使用示例：
```python
list1 = [1, 2, 3]
list2 = [4, 5, 6]

# 加法：拼接
print(list1 + list2) # 输出: [1, 2, 3, 4, 5, 6]

# 乘法：重复
print(list1 * 2) # 输出: [1, 2, 3, 1, 2, 3]

# 成员运算
print(1 in list1) # 输出: True
# 注意：对于嵌套列表，小列表内的元素不属于外层列表的直接元素
print(1 in my_list) # 输出: False，因为1在嵌套列表内
print([1, 2] in my_list) # 输出: True
```

## 修改列表：增加元素 📈

因为列表是可变数据类型，我们可以向其中增加元素。主要方法有 `append()`、`extend()` 和 `insert()`。

以下是三种方法的区别与示例：
*   **`append(object)`**：在列表末尾添加一个对象（整体添加）。
    ```python
    list1 = [1, 2, 3]
    list1.append([4, 5, 6])
    print(list1) # 输出: [1, 2, 3, [4, 5, 6]]
    ```
*   **`extend(iterable)`**：将一个可迭代对象中的每个元素依次添加到列表末尾。
    ```python
    list1 = [1, 2, 3]
    list1.extend([4, 5, 6])
    print(list1) # 输出: [1, 2, 3, 4, 5, 6]
    ```
*   **`insert(index, object)`**：在指定索引位置前插入一个对象。
    ```python
    list1 = [10, 10.2, 9.8]
    list1.insert(1, ‘A‘) # 在索引1（即10.2）之前插入‘A‘
    print(list1) # 输出: [10, ‘A‘, 10.2, 9.8]
    ```
`append` 和 `extend` 是初学者容易混淆的两个方法，关键在于 `append` 添加的是对象本身，而 `extend` 添加的是对象中的元素。

## 修改列表：删除元素 📉

我们可以使用 `remove()`、`pop()` 和 `del` 语句来删除列表中的元素。

以下是三种删除方法的区别与示例：
*   **`remove(value)`**：删除列表中第一个匹配到的指定值，无返回值。
    ```python
    close = [10, 10.2, 9.2, 9.5, 9.2]
    close.remove(9.2)
    print(close) # 输出: [10, 10.2, 9.5, 9.2] (只删除了第一个9.2)
    ```
*   **`pop([index])`**：删除并返回指定索引位置的元素。若不指定索引，则默认删除并返回最后一个元素。
    ```python
    close = [10, 10.2, 9.2, 9.5, 9.2]
    popped_value = close.pop(1) # 删除索引为1的元素
    print(popped_value) # 输出: 10.2
    print(close) # 输出: [10, 9.2, 9.5, 9.2]
    ```
*   **`del` 语句**：可以删除指定索引或切片的元素，也可以删除整个变量，无返回值。
    ```python
    close = [10, 10.2, 9.2, 9.5, 9.2]
    del close[0] # 删除索引0的元素
    print(close) # 输出: [10.2, 9.2, 9.5, 9.2]

    del close[1:3] # 删除切片[1:3]的元素
    print(close) # 输出: [10.2, 9.2]

    del close # 删除变量close本身
    ```

## 列表的常用函数与方法 📚

Python为列表提供了丰富的内置函数和方法。函数是独立的，如 `len()`, `max()`, `min()`；方法是属于列表对象的，如 `list.count()`, `list.sort()`。

以下是一些核心函数与方法的示例：
```python
my_list = [1, 2, 3, 4, 4, 4, 5]

# 内置函数
print(len(my_list)) # 输出: 7 (列表长度)
print(max(my_list)) # 输出: 5 (最大值)
print(min(my_list)) # 输出: 1 (最小值)

# 列表方法
print(my_list.count(4)) # 输出: 3 (统计4出现的次数)
my_list.reverse() # 反转列表顺序
print(my_list) # 输出: [5, 4, 4, 4, 3, 2, 1]
my_list.sort() # 对列表进行排序（默认升序）
print(my_list) # 输出: [1, 2, 3, 4, 4, 4, 5]

# 列表复制
list_copy = my_list.copy() # 创建列表的副本
list_copy.append(‘A‘)
print(my_list) # 输出: [1, 2, 3, 4, 4, 4, 5] (原列表未变)
print(list_copy) # 输出: [1, 2, 3, 4, 4, 4, 5, ‘A‘]
```
**重要提示**：直接使用赋值 `list2 = list1` 并不会创建新列表，而是让两个变量指向同一个列表对象。修改其中一个，另一个也会同步改变。若需独立副本，必须使用 `.copy()` 方法。

## 元组（Tuple）的创建与特性 🧱

元组与列表非常相似，但关键区别在于**元组是不可变数据类型**，创建后其元素不能被修改。元组使用小括号 `()` 创建，元素间用逗号分隔。实际上，定义元组的关键是逗号，括号有时可以省略。

以下是元组创建的示例：
```python
# 标准创建方式
stock = (‘000001‘, ‘平安银行‘, ‘银行‘, 15.6)
print(type(stock)) # 输出: <class ‘tuple‘>

# 省略括号（逗号是关键）
stock2 = ‘000001‘, ‘平安银行‘, ‘银行‘, 15.6
print(type(stock2)) # 输出: <class ‘tuple‘>

# 创建单元素元组，必须在元素后加逗号
single_tuple = (‘只有一个元素‘,)
print(type(single_tuple)) # 输出: <class ‘tuple‘>
# 如果没有逗号，则不是元组
not_a_tuple = (‘只有一个元素‘)
print(type(not_a_tuple)) # 输出: <class ‘str‘>
```

## 元组的索引、切片与运算符 🔧

由于元组也是序列类型，其索引、切片和运算符（`+`, `*`, `in`, `not in`）的用法与列表完全一致。

以下是相关操作示例：
```python
stock = (‘000001‘, ‘平安银行‘, ‘银行‘, 15.6)

# 索引与切片
print(stock[2]) # 输出: ‘银行‘
print(stock[:2]) # 输出: (‘000001‘, ‘平安银行‘)

# 尝试修改元组（会报错）
# stock[2] = ‘地产‘ # TypeError: ‘tuple‘ object does not support item assignment

# 运算符
tuple1 = (‘A‘, ‘B‘)
tuple2 = (‘C‘, ‘D‘)
print(tuple1 + tuple2) # 输出: (‘A‘, ‘B‘, ‘C‘, ‘D‘) (拼接)
print(tuple1 * 2) # 输出: (‘A‘, ‘B‘, ‘A‘, ‘B‘) (重复)
print(‘A‘ in tuple1) # 输出: True
print(‘A‘ not in tuple2) # 输出: True
```

## 元组的函数与方法及转换 🔄

元组的方法较少，常用的有 `count()` 和 `index()`。我们也可以使用 `list()` 和 `tuple()` 函数在列表和元组之间进行转换。

以下是示例：
```python
my_tuple = (‘银行‘, ‘保险‘, ‘证券‘, ‘银行‘)

# 元组方法
print(my_tuple.count(‘银行‘)) # 输出: 2 (计数)
print(my_tuple.index(‘保险‘)) # 输出: 1 (返回首次出现的索引)

# 类型转换
list_from_tuple = list(my_tuple) # 元组转列表
print(list_from_tuple) # 输出: [‘银行‘, ‘保险‘, ‘证券‘, ‘银行‘]
tuple_from_list = tuple([1, 2, 3]) # 列表转元组
print(tuple_from_list) # 输出: (1, 2, 3)
```
元组的一个常见用途是用于序列解包（Unpacking），这在函数返回多个值时非常方便。
```python
# 序列解包示例
a, b = 1, 5 # 本质上右边是一个元组 (1, 5)
print(a) # 输出: 1
print(b) # 输出: 5
```
