## 工具包简介与获取

上一节我们介绍了量化分析的基本概念，本节中我们来看看如何利用现成的工具来提升效率。Alphalens是一个功能丰富的因子分析工具包。

![](img/003004c6a024cb3a2b553eb9c975f109_5.png)

![](img/003004c6a024cb3a2b553eb9c975f109_7.png)

![](img/003004c6a024cb3a2b553eb9c975f109_9.png)

以下是关于Alphalens的基本信息：
*   **来源**：该工具包托管在GitHub上，拥有详细的官方文档和示例。
*   **核心功能**：专门用于金融因子分析，可以自动计算关键指标（如IC值）并生成分析图表。
*   **安装方式**：在本地使用`pip install alphalens`命令即可安装。但在本课程提供的量化平台中，该工具包已经预装完毕，无需手动安装。

## 量化平台的研究环境

由于直接在本地的Notebook环境中获取金融数据并进行计算较为麻烦，我们将在一个专业的量化平台中编写和运行代码。这个平台已经配置好了所需的数据和Alphalens环境。

![](img/003004c6a024cb3a2b553eb9c975f109_21.png)

![](img/003004c6a024cb3a2b553eb9c975f109_23.png)

在平台中，我们需要进入“投资研究”模块，其界面与Jupyter Notebook类似。在该模块中，我们可以新建一个Python 3笔记文件，并将其命名为“因子分析”。后续的所有代码都将在该平台的这个研究环境中编写和执行。

![](img/003004c6a024cb3a2b553eb9c975f109_25.png)

本节课中我们一起学习了Alphalens工具包的用途、获取方式以及如何在量化平台中准备编码环境。Alphalens能帮助我们高效地进行因子分析，而平台的研究环境则为我们提供了便捷的数据和计算支持。接下来，我们就可以开始利用这些工具进行实际的因子分析操作了。
