在本节课中，我们将学习Python编程中两个核心的控制流程结构：**分支**和**循环**。它们是编写任何程序逻辑的基础，能让我们根据条件执行不同的代码块，或者重复执行特定的任务。

## 分支结构：条件判断

上一节我们介绍了变量和基本运算，本节中我们来看看如何让程序根据不同的条件做出决策，这就是分支结构。

### 基本if-else语句

分支的作用是当满足某个条件时，执行一段代码；如果不满足，则执行另一段代码。其基本语法使用 `if` 和 `else` 关键字。

```python
# 获取用户输入并转换为整数
a = int(input("请输入一个数字："))

if a > 10:
    print("输入的数字大于10")
else:
    print("输入的数字小于或等于10")
```

**代码解析**：
1.  `input()` 函数用于获取用户输入，返回一个字符串。
2.  `int()` 函数将字符串转换为整数类型。
3.  `if a > 10:` 判断变量 `a` 是否大于10。如果条件为真（True），则执行下方缩进的代码块。
4.  `else:` 如果 `if` 条件为假（False），则执行 `else` 下方缩进的代码块。
5.  在Python中，**缩进**（通常是4个空格）用于定义代码块，这是语法的一部分。

### 多条件组合判断

条件判断可以结合逻辑运算符（`and`, `or`, `not`）进行更复杂的组合。

```python
a = int(input("请输入第一个数字："))
b = int(input("请输入第二个数字："))
c = int(input("请输入第三个数字："))

if a > b and a > c:
    print("第一个输入的数字是最大的")

print("程序结束")
```

**代码解析**：
- `and` 表示“并且”，要求两边的条件同时为真，整个表达式才为真。
- 无论 `if` 条件是否满足，最后一行 `print(“程序结束”)` 都会执行，因为它没有缩进，不属于 `if` 的代码块。

### 多分支判断：elif

![](img/42734059cb111819c9c425a04bf3cd06_3.png)

![](img/42734059cb111819c9c425a04bf3cd06_5.png)

当存在多种可能情况时，可以使用 `elif`（即 else if）来添加多个分支。

```python
score = int(input("请输入你的分数："))

if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

**代码解析**：
- 程序会按顺序检查每个 `if`/`elif` 条件。
- 一旦某个条件为真，就会执行对应的代码块，然后跳过后续所有 `elif` 和 `else`。
- 如果所有 `if`/`elif` 条件都不满足，则执行 `else` 块。

### 分支的嵌套

分支结构可以嵌套使用，即在一个条件判断的代码块内，再写入另一个条件判断。

```python
score = int(input("请输入分数："))

if score >= 60:
    if score >= 90:
        print("优秀，恭喜及格！")
    else:
        print("及格！")
else:
    print("不及格，需要努力。")
```

**代码解析**：
- 外层 `if` 判断是否及格（>=60）。
- 在内层及格的前提下，再用嵌套的 `if-else` 判断是否优秀（>=90）。
- 注意内层代码块需要额外的缩进（8个空格）。

## 循环结构：重复执行

分支让我们可以选择执行路径，而循环则让我们能够重复执行某段代码，直到满足特定条件。

### for循环

`for` 循环常用于遍历一个序列（如列表、字符串）或执行固定次数的循环。我们结合 `range()` 函数来学习。

以下是 `for` 循环的基本应用，计算从1加到100的和：

![](img/42734059cb111819c9c425a04bf3cd06_7.png)

```python
total = 0
for i in range(1, 101):
    total += i  # 等价于 total = total + i
print(f"从1加到100的和是：{total}")
```

**代码解析**：
- `range(start, stop, step)` 函数生成一个整数序列。
    - `range(101)`：生成 0 到 100（不包含101）的序列。
    - `range(1, 101)`：生成 1 到 100（不包含101）的序列。
    - `range(1, 101, 2)`：生成 1 到 100 的奇数序列，步长为2。
- `for i in range(1, 101):` 会让变量 `i` 依次取 1, 2, 3, ..., 100，并执行循环体内的代码。
- `total += i` 是 `total = total + i` 的简写形式。

![](img/42734059cb111819c9c425a04bf3cd06_9.png)

### while循环

`while` 循环会在条件为真时，持续执行循环体内的代码。

```python
# 这是一个需要谨慎使用的“死循环”模板，必须设置退出条件
while True:
    num = int(input("请输入一个数字（输入50退出）："))
    if num == 50:
        break  # break语句用于立即跳出当前循环
    print(f"你输入的是：{num}")

print("循环结束")
```

![](img/42734059cb111819c9c425a04bf3cd06_11.png)

**代码解析**：
- `while True:` 创建一个条件永远为真的循环，即“死循环”。
- `break` 关键字可以强制终止当前循环，程序继续执行循环之后的代码。
- 务必为 `while True` 循环设置合理的 `break` 条件，否则程序将无法停止。

## 综合应用：列表与循环

现在，我们将分支、循环与上节课学的列表结合起来，完成更实用的操作。

### 遍历列表

以下是使用 `for` 循环遍历列表中所有元素的方法：

```python
students = ["学生A", "学生B", "学生C", "学生D"]
students.append("学生E")  # 在列表末尾添加新元素
students.insert(2, "学生Z")  # 在下标2的位置插入新元素

print("所有学生名单：")
for student in students:
    print(student)
```

![](img/42734059cb111819c9c425a04bf3cd06_13.png)

**重要概念**：**列表下标从0开始**。在列表 `[“A”, “B”, “C”, “D”]` 中，“A”的下标是0，“B”是1，依此类推。因此 `insert(2, “Z”)` 会将“Z”插入到“C”（下标2）之前。

![](img/42734059cb111819c9c425a04bf3cd06_15.png)

### 循环控制：continue

`continue` 关键字用于跳过当前循环的剩余语句，直接进入下一轮循环。

```python
students = ["学生A", "学生B", "学生Z", "学生C", "学生D"]

![](img/42734059cb111819c9c425a04bf3cd06_17.png)

![](img/42734059cb111819c9c425a04bf3cd06_19.png)

print("跳过学生Z后的名单：", end=" ")
for stu in students:
    if stu == "学生Z":
        continue  # 遇到“学生Z”时，跳过本次循环的打印操作
    print(stu, end=" ")  # end=" " 让打印内容不换行，用空格分隔

print()  # 打印一个空行，用于换行
```

![](img/42734059cb111819c9c425a04bf3cd06_21.png)

**代码解析**：
- 当 `stu == “学生Z”` 时，执行 `continue`，循环内后面的 `print` 语句被跳过，直接开始下一次循环。
- `print(stu, end=” “)` 中的 `end=” “` 参数指定了打印结束时的字符，默认为换行符 `\n`，这里替换为空格，使输出在同一行。

### 从列表中删除元素

![](img/42734059cb111819c9c425a04bf3cd06_23.png)

```python
removed_student = students.pop(2)  # 弹出并删除下标为2的元素
print(f"被删除的学生是：{removed_student}")
print(f"删除后的名单：{students}")
```

![](img/42734059cb111819c9c425a04bf3cd06_25.png)

**代码解析**：
- `list.pop(index)` 方法会移除列表中指定下标的元素，并返回该元素的值。
- 删除后，列表的长度会减一，后面元素的下标会自动前移。

## 总结

![](img/42734059cb111819c9c425a04bf3cd06_29.png)

![](img/42734059cb111819c9c425a04bf3cd06_31.png)

本节课中我们一起学习了Python编程的核心控制结构：
1.  **分支结构**：使用 `if`, `elif`, `else` 进行条件判断，让程序具备决策能力。我们学习了基本判断、多条件组合、多分支以及嵌套判断。
2.  **循环结构**：
    - `for` 循环：通常与 `range()` 或列表等可迭代对象结合，用于执行固定次数的循环或遍历所有元素。
    - `while` 循环：在条件满足时持续运行，常用于不确定循环次数的场景，使用时需注意避免死循环。
3.  **循环控制关键字**：
    - `break`：立即终止整个循环。
    - `continue`：跳过当前循环的剩余代码，直接开始下一轮循环。
4.  **综合应用**：我们将分支和循环应用于列表操作，实现了遍历、条件过滤和元素增删等功能。
