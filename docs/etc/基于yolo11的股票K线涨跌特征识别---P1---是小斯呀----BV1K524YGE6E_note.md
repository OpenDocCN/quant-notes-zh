## 项目概述

上一节我们介绍了课程主题，本节中我们来看看项目的核心目标。本项目旨在训练一个计算机视觉模型，使其能够自动识别股票K线图中的关键形态，例如看涨或看跌信号。这有助于量化交易者快速分析大量图表。

## 数据准备

为了训练模型，我们需要准备标注好的K线图数据集。以下是数据准备的核心步骤：

1.  **收集K线图**：首先，需要从股票交易软件或金融数据API中导出大量的历史K线图截图。这些图片将作为我们模型的输入数据。

2.  **标注关键特征**：在收集到的图片上，使用标注工具（如LabelImg、CVAT或Roboflow）框出我们关心的特征区域。例如，一个“早晨之星”形态或一个“乌云盖顶”形态。每个框都需要一个对应的标签（如 `bullish` 或 `bearish`）。

3.  **划分数据集**：将标注好的数据集按比例划分为训练集、验证集和测试集。常见的比例是 **训练集:验证集:测试集 = 70%:20%:10%**。这确保了模型能在未见过的数据上进行有效评估。

4.  **数据格式转换**：YOLO模型需要特定格式的标注文件。通常，每个图像对应一个 `.txt` 文件，其中每一行代表一个标注框，格式为：
    ```
    <class_id> <x_center> <y_center> <width> <height>
    ```
    这里的坐标是相对于图像宽度和高度的归一化值（范围0到1）。
