## 回测收益率详解

![](img/23fbf02ce195c014040bb9b04d9105a1_3.png)

上一节我们介绍了策略评估的必要性，本节中我们来看看最基础的评估指标——回测收益率。

![](img/23fbf02ce195c014040bb9b04d9105a1_5.png)

![](img/23fbf02ce195c014040bb9b04d9105a1_7.png)

回测收益率衡量的是，在回测周期内，策略最终获得的收益相对于初始本金的增长比例。例如，一个策略在几年内的回测收益率为4.23%，这意味着相对于初始投入的资金，策略最终带来了4.23%的增值。

![](img/23fbf02ce195c014040bb9b04d9105a1_9.png)

其计算公式如下：

**回测收益率 = (期末资产总值 - 期初资产总值) / 期初资产总值**

![](img/23fbf02ce195c014040bb9b04d9105a1_11.png)

![](img/23fbf02ce195c014040bb9b04d9105a1_13.png)

或者，如果我们关注的是资产价格序列（例如股票收盘价），公式可以表示为：

![](img/23fbf02ce195c014040bb9b04d9105a1_15.png)

![](img/23fbf02ce195c014040bb9b04d9105a1_17.png)

**回测收益率 = (P_t / P_0) - 1**

![](img/23fbf02ce195c014040bb9b04d9105a1_19.png)

![](img/23fbf02ce195c014040bb9b04d9105a1_21.png)

其中：
*   **P_t** 代表回测结束时的资产价格（或资产总值）。
*   **P_0** 代表回测开始时的资产价格（或初始本金）。

这个指标直观地反映了策略在整个回测周期内的最终盈利表现，是我们评估策略时最先关注的数字。

![](img/23fbf02ce195c014040bb9b04d9105a1_23.png)

## 使用Python计算回测收益率

理解了概念后，我们通过Python代码来实际计算一下。我们将使用一份示例股票数据，模拟策略回测后的资产价格走势，并计算其回测收益率。

![](img/23fbf02ce195c014040bb9b04d9105a1_25.png)

首先，导入必要的工具包并加载数据。

![](img/23fbf02ce195c014040bb9b04d9105a1_27.png)

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便后续显示
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设第一列是日期
data = pd.read_csv('your_stock_data.csv', index_col=0)
data.index = pd.to_datetime(data.index) # 确保索引是日期时间类型
print(data.head())
```

假设我们的 `data['close']` 列代表了应用策略后，每日的资产净值或股票收盘价。我们将基于此计算回测收益率。

计算过程分为两步：
1.  计算相对于期初的资产净值比。
2.  根据净值比计算最终的回测收益率。

以下是计算步骤：

![](img/23fbf02ce195c014040bb9b04d9105a1_29.png)

![](img/23fbf02ce195c014040bb9b04d9105a1_31.png)

```python
# 1. 计算每日资产相对于期初的比值
# 即公式中的 (P_t / P_0)
relative_value = data['close'] / data['close'].iloc[0]

![](img/23fbf02ce195c014040bb9b04d9105a1_33.png)

# 可视化资产变化曲线
plt.figure(figsize=(10, 6))
relative_value.plot()
plt.title('资产相对价值变化曲线')
plt.xlabel('日期')
plt.ylabel('相对期初的价值比')
plt.grid(True)
plt.show()

![](img/23fbf02ce195c014040bb9b04d9105a1_35.png)

![](img/23fbf02ce195c014040bb9b04d9105a1_37.png)

# 2. 计算最终的回测收益率
# 回测收益率 = (期末价值比) - 1
final_return = relative_value.iloc[-1] - 1

![](img/23fbf02ce195c014040bb9b04d9105a1_39.png)

# 将结果整理成DataFrame展示
result_df = pd.DataFrame({
    '回测收益率': [final_return]
}, index=['结果'])
print(result_df)
```

![](img/23fbf02ce195c014040bb9b04d9105a1_41.png)

运行以上代码，你会看到一条展示资产随时间相对增长的曲线，以及一个包含最终回测收益率的表格。这个数值清晰地告诉你策略在回测期间的整体表现。

![](img/23fbf02ce195c014040bb9b04d9105a1_43.png)

![](img/23fbf02ce195c014040bb9b04d9105a1_45.png)
