以下是关于工作表操作的关键点：

*   工作表的移动、复制、添加颜色和重命名等操作。

之后，我们介绍了一些实用的Excel快捷键。

以下是关于Excel操作技巧的核心内容：
*   **定位**：快捷键为 `Ctrl + G` 或 `F5`。
*   **选择性粘贴**：常用于批量更新数据（如集体工资加减）和去除单元格中的公式仅保留值。
*   **查找与替换**：快捷键为 `Ctrl + F`（查找）和 `Ctrl + H`（替换），支持使用通配符（如 `*` 匹配多个字符，`?` 匹配单个字符）。
*   **日期时间格式转换**：可以通过“数据”选项卡下的“分列”功能快速完成。
*   **删除重复值**：用于清理数据中的重复项。
*   **快速填充**：系统提供了一些常用模式，用户也可以根据需求自定义填充规则。
*   **分列与合并**：用于拆分或组合单元格内容。

关于日期格式转换，这里补充一个更简便的方法：选择数据后，点击“数据”->“分列”，在向导中直接选择“日期”格式即可完成转换。

### 1. 公式与函数

![](img/576f76efc1136232ab25f3c2d37de277_10.png)

我们将系统地学习Excel中公式与函数的应用，这是进行数据计算和分析的基础。

![](img/576f76efc1136232ab25f3c2d37de277_12.png)

以下是本部分将涵盖的知识点：
*   **公式与函数的概念**：理解两者之间的区别与联系。
*   **运算符**：学习算术、比较、文本连接和引用运算符。
*   **单元格引用**：掌握相对引用、绝对引用和混合引用的用法。
*   **常用函数分类**：
    *   **逻辑函数**：如 `IF`, `AND`, `OR`。
    *   **文本函数**：如 `LEFT`, `RIGHT`, `MID`, `CONCATENATE`。
    *   **统计函数**：如 `SUM`, `AVERAGE`, `COUNT`, `MAX`, `MIN`。
    *   **查找与引用函数**：如 `VLOOKUP`, `HLOOKUP`, `INDEX`, `MATCH`。
    *   **日期函数**：如 `TODAY`, `NOW`, `YEAR`, `MONTH`, `DAY`。
