## 数组的基本计算

![](img/7598e145718dbff18ad2210aa72941f3_11.png)

![](img/7598e145718dbff18ad2210aa72941f3_13.png)

上一节我们介绍了如何创建NumPy数组，本节中我们来看看如何对数组进行数学计算。

![](img/7598e145718dbff18ad2210aa72941f3_15.png)

![](img/7598e145718dbff18ad2210aa72941f3_17.png)

首先，我们创建一个NumPy数组 `a` 用于演示。

![](img/7598e145718dbff18ad2210aa72941f3_19.png)

![](img/7598e145718dbff18ad2210aa72941f3_21.png)

```python
import numpy as np
a = np.array([1, 2, 3, -1, -2, -3])
```

![](img/7598e145718dbff18ad2210aa72941f3_23.png)

![](img/7598e145718dbff18ad2210aa72941f3_25.png)

以下是NumPy提供的一些常用计算函数：

*   **求和**：`np.sum(a)` 或 `a.sum()`，计算数组中所有元素的总和。
*   **求最小值**：`np.min(a)` 或 `a.min()`，找出数组中的最小值。
*   **求平均值**：`np.mean(a)` 或 `a.mean()`，计算数组元素的平均值。
*   **求标准差**：`np.std(a)` 或 `a.std()`，计算数组元素的标准差。
*   **求最大值索引**：`np.argmax(a)` 或 `a.argmax()`，返回数组中最大值所在的**位置（索引）**。例如，在数组 `[1, 2, 3, -1, -2, -3]` 中，最大值 `3` 的索引是 `2`。
*   **求最小值索引**：`np.argmin(a)` 或 `a.argmin()`，返回数组中最小值所在的位置。例如，最小值 `-3` 的索引是 `5`。

## 一维数组的索引与切片

![](img/7598e145718dbff18ad2210aa72941f3_35.png)

![](img/7598e145718dbff18ad2210aa72941f3_37.png)

掌握了基本计算后，我们来看看如何从数组中提取我们需要的部分，这就是索引和切片。

![](img/7598e145718dbff18ad2210aa72941f3_39.png)

其语法与Python列表的切片非常相似。我们创建一个新的数组 `a` 来演示。

![](img/7598e145718dbff18ad2210aa72941f3_41.png)

![](img/7598e145718dbff18ad2210aa72941f3_43.png)

```python
a = np.arange(13) ** 2  # 生成0到12的平方数组
```

![](img/7598e145718dbff18ad2210aa72941f3_45.png)

![](img/7598e145718dbff18ad2210aa72941f3_47.png)

以下是索引和切片的核心操作：

![](img/7598e145718dbff18ad2210aa72941f3_49.png)

![](img/7598e145718dbff18ad2210aa72941f3_51.png)

*   **访问单个元素**：使用 `a[index]`。例如，`a[0]` 访问第一个元素。
*   **范围切片**：使用 `a[start:stop]`。这表示选取从索引 `start` 到 `stop`（**不包含** `stop`）的元素。例如，`a[1:5]` 选取索引为1, 2, 3, 4的元素。
*   **使用负索引**：`-1` 表示最后一个元素，`-2` 表示倒数第二个，以此类推。例如，`a[-4]` 访问倒数第四个元素。
*   **指定步长**：使用 `a[start:stop:step]`。`step` 表示步长。例如，`a[-5::2]` 表示从倒数第五个元素开始，以步长2取到数组末尾。`a[::-2]` 则表示从末尾开始，以步长2向前取元素。

## 多维数组的索引与切片

在实际数据分析中，我们经常处理表格（矩阵）形式的数据。NumPy可以轻松处理多维数组。

![](img/7598e145718dbff18ad2210aa72941f3_59.png)

![](img/7598e145718dbff18ad2210aa72941f3_61.png)

首先，我们创建一个6x6的二维数组（矩阵）。

![](img/7598e145718dbff18ad2210aa72941f3_63.png)

![](img/7598e145718dbff18ad2210aa72941f3_65.png)

```python
m = np.arange(36).reshape(6, 6)  # 将0到35的数字重塑为6行6列的矩阵
```

![](img/7598e145718dbff18ad2210aa72941f3_67.png)

![](img/7598e145718dbff18ad2210aa72941f3_69.png)

对多维数组进行切片时，使用逗号 `,` 分隔不同维度的索引。

![](img/7598e145718dbff18ad2210aa72941f3_71.png)

![](img/7598e145718dbff18ad2210aa72941f3_73.png)

以下是多维数组切片的方法：

![](img/7598e145718dbff18ad2210aa72941f3_75.png)

*   **访问单个元素**：`m[2, 2]` 访问第3行、第3列的元素（索引从0开始）。
*   **行列范围切片**：`m[3:6, 3:6]` 选取第4到第6行，第4到第6列的子矩阵。
*   **使用冒号表示所有**：冒号 `:` 单独使用表示该维度上的所有元素。
    *   `m[:2, :]` 选取前两行的所有列。
    *   `m[-1, ::2]` 选取最后一行的所有列，但步长为2（即第1, 3, 5列）。
*   **布尔索引（条件筛选）**：这是一种强大的筛选方式。例如，`m[m > 30]` 会返回数组中所有大于30的元素构成的一维数组。我们甚至可以基于条件进行赋值，例如 `m[m > 30] = 99`，会将矩阵中所有大于30的元素替换为99。
