在本节课中，我们将要学习一种在数据预处理中常用的去极值方法——3-Sigma方法。我们将从正态分布的基本概念入手，理解3-Sigma方法的原理，并通过Python代码实现它，最后将其应用于示例数据。

## 概述

上一节我们介绍了去极值的概念和几种基本方法。本节中我们来看看一种基于统计学原理的方法：3-Sigma方法。它的核心思想是利用数据的正态分布特性，将落在均值附近特定范围内的数据视为“正常”，而将落在此范围之外的极端值进行修正。

![](img/48e84413c4fc6d227009566fc0743e99_1.png)

## 3-Sigma方法原理 📈

3-Sigma方法的原理基于正态分布（也称高斯分布）。我们可以通过一个简单的类比来理解：例如去银行贷款，银行多借或少借你少量金额是常见情况（正常分布），但多借或少借你一大笔巨款则属于异常情况（极值）。

在标准正态分布中，均值（μ）为0，标准差（σ）为1。数据点离均值越近，其出现的概率越高；离均值越远，其出现的概率则越低。极值通常就落在这些概率很低的偏远区域。

3-Sigma方法定义了数据正常波动的范围。具体来说，它计算一个区间：`[μ - n*σ, μ + n*σ]`。其中，`n`通常取3。根据统计学经验：
*   `μ ± 1σ` 包含了约68.3%的数据。
*   `μ ± 2σ` 包含了约95.4%的数据。
*   `μ ± 3σ` 包含了约99.7%的数据。

![](img/48e84413c4fc6d227009566fc0743e99_3.png)

因此，当我们设定 `n=3` 时，我们认为99.7%的数据是正常的，而剩下的0.3%的数据则被视为极值，并会被规范到我们定义的边界上。

## 代码实现 💻

![](img/48e84413c4fc6d227009566fc0743e99_9.png)

理解了原理后，我们来看看如何用Python代码实现3-Sigma方法。我们将遵循以下步骤：
1.  计算指定数据列的均值（`mean`）和标准差（`std`）。
2.  根据公式计算上限和下限。
3.  将超出界限的数据值规范到边界上。

![](img/48e84413c4fc6d227009566fc0743e99_11.png)

以下是具体的函数实现：

![](img/48e84413c4fc6d227009566fc0743e99_13.png)

```python
def three_sigma(data_series, n=3):
    """
    使用3-Sigma方法进行去极值处理。
    参数:
        data_series: pandas Series，待处理的数据列。
        n: int，西格玛倍数，默认为3。
    返回:
        series: 处理后的数据。
        lower_limit: 计算出的下限。
        upper_limit: 计算出的上限。
    """
    # 1. 计算均值和标准差
    mean = data_series.mean()
    std = data_series.std()

    # 2. 计算上限和下限
    upper_limit = mean + n * std
    lower_limit = mean - n * std

    # 3. 规范数据：将超出界限的值设置为边界值
    series = data_series.copy()
    series[series > upper_limit] = upper_limit
    series[series < lower_limit] = lower_limit

    return series, lower_limit, upper_limit
```

本节课中我们一起学习了3-Sigma去极值法。我们来回顾一下关键点：

![](img/48e84413c4fc6d227009566fc0743e99_41.png)

![](img/48e84413c4fc6d227009566fc0743e99_43.png)

1.  **原理基础**：该方法基于数据服从正态分布的假设，利用均值（`μ`）和标准差（`σ`）来定义数据的正常波动区间。
2.  **核心公式**：正常区间的上下限由 **`μ ± n*σ`** 决定，通常 `n=3` 以涵盖99.7%的数据。
3.  **操作本质**：去极值不是简单地删除异常数据，而是通过定义上界和下界，将所有数据规范到这个范围内。超出上限的值设为上限值，低于下限的值设为下限值。
4.  **方法比较**：3-Sigma法、分位数法等都是有效的去极值策略，它们共同构成了数据预处理“三步走”中第一步的重要工具。
