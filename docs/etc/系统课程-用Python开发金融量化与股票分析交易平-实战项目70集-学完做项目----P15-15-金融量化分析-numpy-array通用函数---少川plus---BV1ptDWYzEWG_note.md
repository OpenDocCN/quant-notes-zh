## 基本数学运算
NumPy提供了许多基本的数学运算函数，它们可以像加减乘除一样对整个数组进行批量操作。

以下是常见的数学通用函数示例：

*   `np.abs(a)`: 计算数组 `a` 中每个元素的绝对值。
*   `np.sqrt(a)`: 计算数组 `a` 中每个元素的平方根。注意，对负数开方会得到特殊值 `nan`。
*   `np.exp(a)`: 计算数组 `a` 中每个元素的指数（e的a次方）。
*   `np.log(a)`: 计算数组 `a` 中每个元素的自然对数。

```python
import numpy as np
a = np.array([-5, -2, 0, 2, 5])
print(np.abs(a))  # 输出: [5 2 0 2 5]
print(np.sqrt(np.abs(a))) # 先取绝对值再开方，避免出现nan
```

## 取整运算
在处理数据时，我们经常需要对浮点数进行取整。Python和NumPy提供了几种不同的取整方式。

以下是四种主要的取整方法：

*   **向零取整 (`np.trunc`)**: 直接舍弃小数部分。例如，1.6 变成 1，-1.6 变成 -1。
*   **向下取整 (`np.floor`)**: 取不大于原数的最大整数。例如，1.6 变成 1，-1.6 变成 -2。
*   **向上取整 (`np.ceil`)**: 取不小于原数的最小整数。例如，1.6 变成 2，-1.6 变成 -1。
*   **四舍五入 (`np.round`)**: 根据小数部分进行四舍五入。例如，1.6 变成 2，-1.6 变成 -2。

```python
a = np.array([1.6, -1.6, 2.5, -2.5])
print(np.trunc(a))  # 输出: [ 1. -1.  2. -2.]
print(np.floor(a))  # 输出: [ 1. -2.  2. -3.]
print(np.ceil(a))   # 输出: [ 2. -1.  3. -2.]
print(np.round(a))  # 输出: [ 2. -2.  2. -2.]
```

## 分离整数与小数部分
`np.modf` 函数可以将一个数组中的每个元素分解为整数部分和小数部分。

```python
a = np.array([1.6, -1.6, 3.0])
integer_part, fractional_part = np.modf(a)
print(integer_part)   # 输出: [ 0.6 -0.6  0. ]
print(fractional_part) # 输出: [ 1. -1.  3.]
```

## 特殊数值处理
在数值计算中，有时会遇到非法的数学运算，如除以零或对负数开方。NumPy使用两个特殊的浮点数值来标记这些情况，而不是直接报错，以保证其他有效数据的计算能继续进行。

以下是两个关键的特殊值及其判断函数：

*   **`nan` (Not a Number)**: 表示“不是一个数字”。例如，`0.0 / 0.0` 或对负数开方会产生 `nan`。**重要特性：`nan` 不等于任何值，包括它自己**。因此，判断一个值是否为 `nan` 必须使用 `np.isnan()` 函数。
*   **`inf` (Infinity)**: 表示“无穷大”。例如，一个非零数除以 `0.0` 会产生 `inf`。判断一个值是否为 `inf` 可以使用 `np.isinf()` 函数。

```python
# 生成包含特殊值的数组
a = np.array([0, 1, 2, 0])
b = np.array([1, 2, 0, 0])
c = a / b  # 将会产生 inf 和 nan
print(c) # 输出类似: [0.  0.5 inf nan]

# 正确判断 nan 和 inf
print(np.isnan(c)) # 输出: [False False False  True]
print(np.isinf(c)) # 输出: [False False  True False]

# 过滤掉数组中的 nan 或 inf
valid_data = c[~np.isnan(c) & ~np.isinf(c)]
print(valid_data) # 输出: [0.  0.5]
```

## 二元通用函数
除了处理单个数组的一元函数，NumPy也提供了处理两个数组的二元函数，如 `np.maximum` 和 `np.minimum`。

```python
a = np.array([3, 4, 1])
b = np.array([2, 5, 3])
print(np.maximum(a, b)) # 逐元素比较，取较大值。输出: [3 5 3]
print(np.minimum(a, b)) # 逐元素比较，取较小值。输出: [2 4 1]
```
