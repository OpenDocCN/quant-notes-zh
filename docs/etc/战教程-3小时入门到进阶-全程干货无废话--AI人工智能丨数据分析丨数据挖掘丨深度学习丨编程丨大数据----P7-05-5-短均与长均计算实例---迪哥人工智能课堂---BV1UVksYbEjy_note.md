## 计算短期与长期移动平均线

要分析黄金交叉和死亡交叉，首先需要计算两个指标：短期移动平均线和长期移动平均线。移动平均线是过去一段时间内股价的平均值，能平滑价格波动，反映趋势。

以下是计算步骤：

1.  **选择数据**：我们以苹果公司（AAPL）的历史股价数据为例。
2.  **定义窗口**：短期窗口通常较小，例如5天（代表一周的交易日）；长期窗口较大，例如20或30天（代表一个月左右的交易日）。由于我们的数据集时间跨度较长，为让图表更清晰，本例将使用稍大的窗口。
3.  **进行计算**：使用Pandas的`.rolling()`和`.mean()`方法计算移动平均值。

![](img/089a91957fd6041380f44d701184444e_11.png)

![](img/089a91957fd6041380f44d701184444e_13.png)

我们将短期平均线记为`M1`，长期平均线记为`M2`。

![](img/089a91957fd6041380f44d701184444e_15.png)

![](img/089a91957fd6041380f44d701184444e_17.png)

```python
# 假设 df 是包含苹果股价的DataFrame，其中‘Close’为收盘价列
# 计算短期移动平均线（例如10天窗口）
df[‘M1‘] = df[‘Close‘].rolling(window=10).mean()

# 计算长期移动平均线（例如30天窗口）
df[‘M2‘] = df[‘Close‘].rolling(window=30).mean()
```

## 识别黄金交叉与死亡交叉

![](img/089a91957fd6041380f44d701184444e_39.png)

通过观察图表，我们可以手动识别两个重要的交叉信号：

1.  **死亡交叉**：当短期移动平均线（绿色）从上方向下穿越长期移动平均线（红色）时，形成死亡交叉。这通常被视为下跌趋势开始的信号，提示投资者考虑卖出或避免买入。
2.  **黄金交叉**：当短期移动平均线（绿色）从下方向上穿越长期移动平均线（红色）时，形成黄金交叉。这通常被视为上涨趋势开始的信号，提示投资者可以考虑买入。

![](img/089a91957fd6041380f44d701184444e_41.png)

## 使用信号指标进行增强分析

![](img/089a91957fd6041380f44d701184444e_43.png)

手动看图识别交叉点不够精确。我们可以利用NumPy创建一个信号指标，使买卖点一目了然。

![](img/089a91957fd6041380f44d701184444e_45.png)

![](img/089a91957fd6041380f44d701184444e_47.png)

思路是：比较短期均线`M1`和长期均线`M2`。当`M1 > M2`时，标记为`1`，代表“持有”或“看涨”信号；当`M1 < M2`时，标记为`-1`，代表“卖出”或“看跌”信号。

![](img/089a91957fd6041380f44d701184444e_49.png)

以下是实现代码：

![](img/089a91957fd6041380f44d701184444e_51.png)

![](img/089a91957fd6041380f44d701184444e_53.png)

```python
import numpy as np

![](img/089a91957fd6041380f44d701184444e_55.png)

# 生成交易信号，1代表短期均线在长期均线之上，-1代表之下
df[‘position‘] = np.where(df[‘M1‘] > df[‘M2‘], 1, -1)
```

为了在图表中清晰展示这个只取`1`和`-1`的信号线，我们需要使用右侧的Y轴。

```python
fig, ax1 = plt.subplots(figsize=(12, 6))

![](img/089a91957fd6041380f44d701184444e_57.png)

![](img/089a91957fd6041380f44d701184444e_59.png)

# 左侧Y轴：绘制股价和移动平均线
ax1.plot(df[‘Close‘], label=‘Close Price‘, alpha=0.5)
ax1.plot(df[‘M1‘], label=‘Short MA (M1)‘, color=‘green‘)
ax1.plot(df[‘M2‘], label=‘Long MA (M2)‘, color=‘red‘)
ax1.set_ylabel(‘Price‘)
ax1.legend(loc=‘upper left‘)

![](img/089a91957fd6041380f44d701184444e_61.png)

# 右侧Y轴：绘制交易信号
ax2 = ax1.twinx()
ax2.plot(df[‘position‘], label=‘Trading Signal‘, color=‘purple‘, linestyle=‘--‘)
ax2.set_ylabel(‘Signal (1/-1)‘)
ax2.set_ylim([-1.5, 1.5])
ax2.legend(loc=‘upper right‘)

plt.title(‘Stock Price with Moving Averages and Trading Signals‘)
plt.show()
```

![](img/089a91957fd6041380f44d701184444e_63.png)

在最终的图表中，紫色虚线（对应右侧坐标轴）在`1`和`-1`之间切换。当信号从`-1`上穿到`1`时，对应图表中的**黄金交叉**点；当信号从`1`下穿到`-1`时，对应图表中的**死亡交叉**点。这使得买卖时机变得更加清晰和量化。
