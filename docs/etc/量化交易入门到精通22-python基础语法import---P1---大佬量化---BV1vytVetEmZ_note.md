# 量化交易入门到精通：P1：Python基础语法之import 📦

在本节课中，我们将要学习Python中一个极其重要的概念：**导入（import）**。我们将了解如何利用他人编写好的代码包（库）来快速实现复杂功能，这是Python强大生态系统的核心体现。

## 概述：Python的包与导入

![](img/f88a9c8aa5501b1251a410d5f364c4bd_1.png)

Python的强大之处在于其丰富的开源生态。许多优秀的开发者或团队将功能代码打包成“包”（Package）或“库”（Library），供他人直接使用。这些包主要分为两类：开源（Open Source）和闭源（Closed Source）。开源包提供源代码，遵循特定的开源协议（如GPL、MIT协议等），而闭源包则不提供源代码，有些可能是商用收费的。

![](img/f88a9c8aa5501b1251a410d5f364c4bd_3.png)

除了引入第三方库，我们也可以导入自己编写的代码库。接下来，我们将通过一个具体的量化分析示例来学习`import`和`from ... import ...`的用法。

## 导入库的基本语法

以下是导入库的两种基本形式：

1.  **导入整个库**：使用`import`关键字。
    ```python
    import pandas
    ```

![](img/f88a9c8aa5501b1251a410d5f364c4bd_5.png)

2.  **从库中导入特定功能**：使用`from ... import ...`语法。
    ```python
    from talib import SMA
    ```

在导入时，可以使用`as`关键字为库设置一个简短的别名，方便后续调用。

![](img/f88a9c8aa5501b1251a410d5f364c4bd_7.png)

```python
import pandas as pd
```
这里，我们将`pandas`库简称为`pd`。

## 实战演练：计算股票移动平均线

上一节我们介绍了导入的基本语法，本节中我们来看看如何应用这些知识来解决一个实际问题：计算股票的5日移动平均线（MA5）。

我们将使用两个库：
*   **Pandas**：擅长数据处理，我们将用它来创建和操作股票价格数据。
*   **TA-Lib**：一个通用的技术指标计算库，我们将用它来计算移动平均线。

![](img/f88a9c8aa5501b1251a410d5f364c4bd_9.png)

### 步骤一：准备数据

![](img/f88a9c8aa5501b1251a410d5f364c4bd_11.png)

首先，我们使用Pandas创建一个包含股票收盘价（close）的DataFrame。DataFrame可以理解为程序中的“电子表格”。

![](img/f88a9c8aa5501b1251a410d5f364c4bd_13.png)

```python
import pandas as pd

![](img/f88a9c8aa5501b1251a410d5f364c4bd_15.png)

# 创建一个包含收盘价数据的DataFrame
data = {'close': [10, 12, 13, 15, 14, 11]}
df = pd.DataFrame(data)
print(df)
```
执行这段代码，会输出以下结构的数据：
```
   close
0     10
1     12
2     13
3     15
4     14
5     11
```
其中，`close`是列名，下面的数字是每一行的收盘价数据。

![](img/f88a9c8aa5501b1251a410d5f364c4bd_17.png)

### 步骤二：计算移动平均线

![](img/f88a9c8aa5501b1251a410d5f364c4bd_19.png)

接下来，我们从TA-Lib库中导入专门计算简单移动平均线的函数`SMA`，并对我们的收盘价数据计算5日均线。

```python
from talib import SMA

![](img/f88a9c8aa5501b1251a410d5f364c4bd_21.png)

# 计算5日移动平均线
ma5 = SMA(df['close'], timeperiod=5)
print(ma5)
```
执行计算后，程序会输出一个结果序列。由于我们提供了6个数据点，而MA5需要至少5个数据才能开始计算，因此输出的第一个值是前5个价格（10, 12, 13, 15, 14）的平均值。

![](img/f88a9c8aa5501b1251a410d5f364c4bd_23.png)

**计算过程验证**：
第一个MA5值 = (10 + 12 + 13 + 15 + 14) / 5 = 64 / 5 = 12.8
程序输出的结果会与此相符。后续值则是移动窗口计算的结果。

![](img/f88a9c8aa5501b1251a410d5f364c4bd_25.png)

![](img/f88a9c8aa5501b1251a410d5f364c4bd_27.png)

### 步骤三：灵活应用

![](img/f88a9c8aa5501b1251a410d5f364c4bd_29.png)

使用库函数的最大好处是代码简洁且易于修改。例如，如果我们想将计算改为10日移动平均线（MA10），只需修改一个参数即可。

```python
# 计算10日移动平均线
ma10 = SMA(df['close'], timeperiod=10)
print(ma10)
```
系统会自动根据新的参数进行计算。如果数据量不足10个，则前几个结果会是`NaN`（非数字），直到有足够数据为止。

![](img/f88a9c8aa5501b1251a410d5f364c4bd_31.png)
