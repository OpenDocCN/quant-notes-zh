## 什么是移动平均线？ 📈

![](img/264297aaa13d6a76ae359ecd0c10dc56_9.png)

![](img/264297aaa13d6a76ae359ecd0c10dc56_11.png)

上一节我们介绍了课程概述，本节中我们来看看均线的核心定义。

![](img/264297aaa13d6a76ae359ecd0c10dc56_13.png)

移动平均线（Moving Average，简称MA）由美国投资专家葛兰维于20世纪中期提出。其核心思想是：**将一定时期内的证券价格（通常为收盘价）加以平均，并把不同时间的平均值连接起来，形成一条曲线，用以观察价格变动趋势**。

![](img/264297aaa13d6a76ae359ecd0c10dc56_15.png)

说人话版本就是：我们每天计算过去N个交易日的收盘价平均值，然后将这些每日的平均值连成线，就得到了一根均线。

*   **计算公式**：`MA(N) = (P1 + P2 + ... + PN) / N`
    *   其中，`P1` 代表今日收盘价，`P2` 代表昨日收盘价，以此类推，`N` 代表计算周期。

![](img/264297aaa13d6a76ae359ecd0c10dc56_17.png)

![](img/264297aaa13d6a76ae359ecd0c10dc56_19.png)

例如，计算过去5天收盘价的平均值，就得到**MA5**；计算过去10天的平均值，就得到**MA10**。这些就是我们在行情软件上常见的白色、黄色等均线。

![](img/264297aaa13d6a76ae359ecd0c10dc56_21.png)

随着时间发展，均线家族也出现了许多变体，如EMA（指数移动平均线）、WMA（加权移动平均线）等。但根据实际使用经验，这些衍生均线的效果通常不会比最简单的MA好太多。此外，许多复杂的技术指标（如布林带）也内置了均线的理念。可以说，**均线是技术分析的基石之一，也是最常见、最基础的趋势性指标**。

## 均线的常见用法与我们的测试方法 🔍

![](img/264297aaa13d6a76ae359ecd0c10dc56_29.png)

上一节我们了解了均线的定义，本节中我们来看看市场上流传的均线用法，并介绍我们将如何科学地验证它们。

![](img/264297aaa13d6a76ae359ecd0c10dc56_31.png)

因为均线极其常见，很多交易者会自然地使用它来辅助决策，例如将其作为价格的支撑位或压力位。市场上关于均线的用法传说众多，例如“金叉买进，死叉卖出”、“均线粘合行情将变”等。

![](img/264297aaa13d6a76ae359ecd0c10dc56_33.png)

以下是我们在本次研究中整理并测试的25种常见均线用法中的一部分示例：
*   金叉 / 死叉
*   多头排列 / 空头排列
*   蛟龙出海
*   上山爬坡型
*   均线粘合后发散
*   价格回踩均线获得支撑

![](img/264297aaa13d6a76ae359ecd0c10dc56_35.png)

那么，这些用法的实际效果如何？仅凭几张成功案例的图表就下结论是不科学的。作为量化交易者，我们选择用数据说话。

![](img/264297aaa13d6a76ae359ecd0c10dc56_37.png)

![](img/264297aaa13d6a76ae359ecd0c10dc56_39.png)

**我们的测试方法**：
1.  **数据**：使用包含A股历史上所有股票（含已退市）的日级别交易数据（开盘价、最高价、最低价、收盘价、成交量）。
2.  **工具**：使用Python编程，将每一种均线用法转化为明确的、可执行的交易规则（买卖信号）。
3.  **回测**：在全市场历史数据中，找出所有符合该信号的时间点，并统计信号发出后未来不同周期（如1天、5天、20天）的股价平均涨跌幅和上涨概率。

![](img/264297aaa13d6a76ae359ecd0c10dc56_41.png)

![](img/264297aaa13d6a76ae359ecd0c10dc56_43.png)

通过这种方法，我们可以客观地评估一个策略的**胜率**（上涨概率）和**赔率**（平均收益率）。

## 核心策略测试一：金叉与死叉的真相 ⚔️

![](img/264297aaa13d6a76ae359ecd0c10dc56_49.png)

上一节我们介绍了测试方法，本节中我们首先来验证最著名的均线用法：金叉与死叉。

![](img/264297aaa13d6a76ae359ecd0c10dc56_51.png)

假设我们有两根均线：短期均线（如MA5）和长期均线（如MA20）。
*   **金叉（Golden Cross）**：当短期均线从下向上穿越长期均线时形成。传统理论认为这是买入信号，预示后市上涨。
*   **死叉（Death Cross）**：当短期均线从上向下穿越长期均线时形成。传统理论认为这是卖出信号，预示后市下跌。

我们使用全市场历史数据和Python代码对这两种信号进行了回测。

![](img/264297aaa13d6a76ae359ecd0c10dc56_53.png)

![](img/264297aaa13d6a76ae359ecd0c10dc56_55.png)

**金叉测试结果**：
*   自2007年以来，A股共发生约37万次金叉。
*   金叉发生后，股价短期（1-5天）上涨概率约50%，与抛硬币无异，平均收益率接近0。
*   但从中期（10-20天）看，平均收益率转为正（10天后0.66%，20天后1.27%），表现尚可。

![](img/264297aaa13d6a76ae359ecd0c10dc56_57.png)

**死叉测试结果**：
*   死叉发生后，股价下跌概率同样在50%附近。
*   **然而**，死叉后未来各周期的平均收益率均为**正数**，这与“死叉预示下跌”的预期完全相反。

![](img/264297aaa13d6a76ae359ecd0c10dc56_59.png)

![](img/264297aaa13d6a76ae359ecd0c10dc56_61.png)

**结论**：
*   **金叉**作为一个买入信号，虽然短期胜率不高，但中期具备一定的盈利潜力，并非完全无效。
*   **死叉**作为一个卖出信号，历史数据证明其完全无效，甚至可能错失后续上涨行情。**不能简单地依据死叉卖出**。

## 核心策略测试二：两种有效的均线形态 🐉

![](img/264297aaa13d6a76ae359ecd0c10dc56_65.png)

上一节我们分析了经典的金叉死叉，本节中我们从25种形态中挑选出两种经测试表现优异的策略进行介绍。

![](img/264297aaa13d6a76ae359ecd0c10dc56_67.png)

### 形态一：蛟龙出海（短线策略）

![](img/264297aaa13d6a76ae359ecd0c10dc56_69.png)

**形态描述**：该形态通常出现在下跌或横盘末期。某日出现一根实体较大的阳线，这根阳线从下方同时向上穿越多根均线（如MA5, MA10, MA20），且收盘价站于这些均线之上，形如蛟龙跃出水面，视为看涨信号。

![](img/264297aaa13d6a76ae359ecd0c10dc56_71.png)

**量化测试结果**：
*   自2007年以来，该形态出现超4万次。
*   形态出现后，股价短期（1-5天）上涨概率略高于50%，平均收益率约1%。
*   中长期收益逐渐衰减。

![](img/264297aaa13d6a76ae359ecd0c10dc56_73.png)

**策略总结**：**“蛟龙出海”形态更适合作为短线交易的参考买入信号**，需注意快进快出。

![](img/264297aaa13d6a76ae359ecd0c10dc56_75.png)

### 形态二：上山爬坡型（中长线策略）

**形态描述**：该形态出现在上升趋势中。短期、中期、长期均线（如MA5, MA10, MA20）沿着近似平行的坡度稳健向上移动，犹如爬山。坡度越缓，持续时间越长，被认为后市上涨力度越强。

![](img/264297aaa13d6a76ae359ecd0c10dc56_77.png)

**量化测试结果**：
*   自2007年以来，该形态出现次数高达61万次。
*   形态出现后，短期胜率未超过50%，但平均收益率随时间稳步上升。
*   **出现20天后，平均收益率可达2.58%**，对于简单均线指标而言非常可观。

![](img/264297aaa13d6a76ae359ecd0c10dc56_79.png)

![](img/264297aaa13d6a76ae359ecd0c10dc56_81.png)

**策略总结**：**“上山爬坡型”形态更适合用于中长期的趋势跟踪交易**，在确认趋势后持有，有望获取稳健收益。

![](img/264297aaa13d6a76ae359ecd0c10dc56_83.png)

![](img/264297aaa13d6a76ae359ecd0c10dc56_85.png)

## 总结与核心心得 💡

本节课中我们一起学习了移动平均线的基础知识，并通过量化回测的方法，验证了多种常见均线策略的有效性。

![](img/264297aaa13d6a76ae359ecd0c10dc56_87.png)

我们得到的关键结论如下：
1.  **并非所有传说都有效**：最经典的“死叉”卖出信号，历史证明其无效。投资决策需建立在数据验证之上，而非主观经验。
2.  **策略有效性需分场景**：“金叉”中期有效；“蛟龙出海”适合短线；“上山爬坡型”适合中长线。没有放之四海而皆准的单一指标。
3.  **科学投资，用数据说话**：量化回测是检验策略成色的最佳工具。它可以帮你排除大量无效的“噪声”，聚焦于真正有统计优势的机会。
4.  **寻找“更弱的牌局”**：一个策略的盈利不仅取决于其本身，更取决于所应用的市场和参与者。有时，将简单的有效策略（如均线）应用到竞争更少、趋势更明显的市场或周期（如某些高波动性资产的特定时间框架），效果可能远超在高效复杂的A股日线图中使用。

![](img/264297aaa13d6a76ae359ecd0c10dc56_89.png)

归根结底，在投资中保持理性与科学的态度，持续学习并用数据验证想法，才是长期致胜的关键。希望本教程提供的思路和工具有助于你开启自己的量化研究之旅。
