## 夏普比率的含义
上一节我们介绍了投资回报率，本节中我们来看看如何衡量风险与收益的关系。夏普比率描述的是：在承担单位风险的情况下，我们能获得多少超额收益。

一个通俗的比喻是：一份在叙利亚日薪3万的工作，其高薪源于极高的风险。夏普比率就是用来评估，为了获得一定的收益，我们所承担的风险是否“值得”。指标越高，意味着**单位风险带来的收益越高**，投资效率也就越好。

## 夏普比率的计算公式
理解了其含义后，我们来看看如何量化计算夏普比率。它的核心思想是比较投资组合的收益与无风险收益（如国债、银行存款）的差异。

![](img/e217d1ed7faab819fd981d0894f52b60_5.png)

以下是夏普比率的计算公式：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率
*   **Rf**：无风险收益率（例如年化5%）
*   **σp**：投资组合收益率的标准差（代表风险）

这个公式计算的是：投资组合每承担一单位的总风险（标准差），能获得多少超出无风险收益的回报。

## 使用Python计算夏普比率
理论清晰后，我们动手用Python代码实现夏普比率的计算。假设我们已经有了投资组合的历史日收益率数据。

以下是计算夏普比率的关键步骤代码：

```python
import numpy as np
import pandas as pd

![](img/e217d1ed7faab819fd981d0894f52b60_7.png)

# 假设 returns 是一个Pandas Series，包含了投资组合的日收益率
# 步骤1: 设定年化无风险利率，例如5%
risk_free_rate = 0.05

# 步骤2: 计算年化夏普比率
# 首先计算超额收益的平均值（年化）
excess_return = returns.mean() * 250 - risk_free_rate  # 250个交易日
# 然后计算收益率的标准差（年化）
volatility = returns.std() * np.sqrt(250)
# 最后计算夏普比率
sharpe_ratio = excess_return / volatility

print(f"计算出的夏普比率为: {sharpe_ratio}")
```

代码说明：
1.  我们首先计算投资组合日收益率的**年化平均超额收益**（乘以250个交易日，并减去无风险收益）。
2.  接着计算收益率的**年化波动率（标准差）**（日标准差乘以根号250）。
3.  最后，用年化超额收益除以年化波动率，就得到了**年化夏普比率**。

## 结果解读与应用
运行上述代码后，我们会得到一个具体的数值。

![](img/e217d1ed7faab819fd981d0894f52b60_11.png)

例如，我们计算了多只股票的夏普比率，结果可能如下：
*   股票A夏普比率：1.5
*   股票B夏普比率：0.8
*   股票C夏普比率：-0.2

![](img/e217d1ed7faab819fd981d0894f52b60_13.png)

**解读与应用原则**：
*   **夏普比率越高越好**。股票A（1.5）意味着其单位风险带来的补偿最高，是更优的选择。
*   **正 vs 负**：正数表示收益跑赢了无风险利率；负数（如股票C）则意味着投资该资产还不如存银行，通常应避免。
*   **比较工具**：在多个投资选项（如不同的股票、基金或策略）之间，应优先选择夏普比率更高的那个。
