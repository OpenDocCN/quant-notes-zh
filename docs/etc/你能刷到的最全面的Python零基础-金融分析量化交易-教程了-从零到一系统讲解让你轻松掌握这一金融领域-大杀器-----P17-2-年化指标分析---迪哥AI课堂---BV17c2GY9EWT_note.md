## 回测收益率回顾

![](img/d84446391417072f33ef1019c741c15d_8.png)

回测收益率是评估策略表现的基础指标。它计算的是策略在整个回测周期结束后，与初始本金相比的总收益比率。这个值可能是正的（盈利），也可能是负的（亏损），直观地反映了策略的盈亏结果。

![](img/d84446391417072f33ef1019c741c15d_10.png)

## 年化收益率的概念

![](img/d84446391417072f33ef1019c741c15d_12.png)

在理解了总收益之后，我们面临一个新的问题：如何比较不同时间长度的策略表现？这时就需要引入年化收益率。

![](img/d84446391417072f33ef1019c741c15d_14.png)

![](img/d84446391417072f33ef1019c741c15d_16.png)

年化收益率是一个标准化的指标，它将不同时间周期的总收益率换算成相当于一年的收益率，使得不同时间长度的策略之间具有可比性。当向他人展示一个长期策略时，年化收益是更常用、更易懂的衡量标准。

![](img/d84446391417072f33ef1019c741c15d_18.png)

## 年化收益率的计算公式

年化收益率的计算基于总回测收益率，并通过一个公式进行转换。其核心思想是计算收益的“年化”复合增长率。

![](img/d84446391417072f33ef1019c741c15d_20.png)

以下是年化收益率的计算公式：

![](img/d84446391417072f33ef1019c741c15d_22.png)

**公式：**
`年化收益率 = (1 + 总回测收益率) ^ (年交易日数 / 回测总交易日数) - 1`

![](img/d84446391417072f33ef1019c741c15d_24.png)

**公式解析：**
*   **总回测收益率 (R)**：即上一节计算出的策略总收益率。
*   **年交易日数 (M)**：通常取250或252天，代表一年中平均的交易日数量。
*   **回测总交易日数 (N)**：策略回测所使用的数据总天数。

![](img/d84446391417072f33ef1019c741c15d_26.png)

## 代码实现年化收益率计算

![](img/d84446391417072f33ef1019c741c15d_28.png)

![](img/d84446391417072f33ef1019c741c15d_30.png)

接下来，我们通过Python代码来实际计算年化收益率。假设我们已经有了包含各股票回测收益率的数据。

以下是计算年化收益率的关键步骤：

![](img/d84446391417072f33ef1019c741c15d_32.png)

```python
# 假设 df 是一个DataFrame，其中‘回测收益率’列已经计算好
# 定义年交易日数和回测总天数
M = 250  # 年交易日数
N = len(df)  # 回测数据的总天数，这里假设df行数即交易日数

![](img/d84446391417072f33ef1019c741c15d_34.png)

![](img/d84446391417072f33ef1019c741c15d_36.png)

# 计算年化收益率
df[‘年化收益率’] = (1 + df[‘回测收益率’]) ** (M / N) - 1

![](img/d84446391417072f33ef1019c741c15d_38.png)

# 展示结果
print(df[[‘回测收益率‘， ’年化收益率‘]])
```

![](img/d84446391417072f33ef1019c741c15d_40.png)

执行以上代码后，我们可以得到一个新的“年化收益率”列。与“回测收益率”对比，你会发现数值发生了变化。例如，一个持续数年的策略，其总收益率可能看起来很高，但折算成年化收益率后，数值会更为温和，这能更真实地反映其每年的平均盈利能力。

## 核心指标总结

![](img/d84446391417072f33ef1019c741c15d_46.png)

本节课中我们一起学习了评估量化策略时最需要关注的两个收益指标：

![](img/d84446391417072f33ef1019c741c15d_48.png)

1.  **回测收益率**：反映策略在整个测试周期内的**总盈亏**结果，是评估策略的起点。
2.  **年化收益率**：将总收益标准化为**年度收益**，用于公平比较不同时间长度策略的盈利能力，是业界沟通和评估的**核心指标**。

![](img/d84446391417072f33ef1019c741c15d_50.png)

![](img/d84446391417072f33ef1019c741c15d_52.png)

理解并正确计算这两个指标，是分析任何交易策略表现的基础。在后续的课程中，我们将继续学习其他重要的风险与收益指标。
