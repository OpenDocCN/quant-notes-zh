## 概述
在本节课中，我们将学习如何构建一个**多因子选股策略**。上一节我们介绍了基于单一因子的选股策略，本节我们将探讨如何同时考虑多个财务指标（因子）来筛选股票，并介绍数据处理中的关键步骤——**数据标准化**，以确保不同量级的因子能够公平地参与评分。

## 从单因子到多因子
单因子选股策略基于一个标准（如市值最小）来筛选股票。其核心操作是定期（如每月）调仓，卖出不再符合标准的股票，买入新的符合标准的股票。

![](img/f7c70bfdcc098e577df3e9dbebcc6ef0_1.png)

然而，仅依赖一个因子可能不够全面。多因子选股策略旨在综合考虑多个指标（例如市值、净资产收益率、增长率等），以期获得更稳健的投资组合。

那么，如何同时考虑多个因子呢？一种直观的思路是给每个股票基于多个因子进行**综合评分**。

## 构建综合评分模型
一个经典且简单的方法是**评分模型**。其核心思想是：将每个因子转化为可比较的数值，然后根据各因子的重要性（权重）进行加权求和，得到每个股票的总分，最后选择总分最高的若干只股票。

### 核心挑战：因子量纲统一
直接对原始因子值进行加减会遇到问题。例如，若我们想选择“市值小”且“净资产收益率(ROE)高”的股票。ROE是百分比（如0%-100%），而市值单位是亿或万。若简单计算 `ROE - 市值`，市值的数值巨大，会完全主导结果，使ROE的作用被忽略。

为了解决这个问题，我们需要在进行综合计算前，将所有因子转换到**相同的尺度**上。这个过程称为**数据标准化**或**归一化**。

以下是两种常用的标准化方法：

### 1. Min-Max 标准化
此方法将原始数据线性转换到[0, 1]区间内。

**公式**：
`X_scaled = (X - X_min) / (X_max - X_min)`

*   `X`: 原始数据
*   `X_min`: 该因子所有数据中的最小值
*   `X_max`: 该因子所有数据中的最大值
*   `X_scaled`: 标准化后的数据

**代码示例**：
```python
import numpy as np
# 假设 A 是一个包含原始因子值（如市值）的数组
A = np.array([...])
A_scaled = (A - A.min()) / (A.max() - A.min())
```
转换后，该因子所有值都落在0到1之间，且数据间的相对大小关系保持不变。

### 2. Z-Score 标准化
此方法将原始数据转换为均值为0、标准差为1的分布。

**公式**：
`X_scaled = (X - μ) / σ`

*   `μ`: 该因子所有数据的平均值
*   `σ`: 该因子所有数据的标准差

**代码示例**：
```python
A_scaled_z = (A - A.mean()) / A.std()
```
在多因子选股中，**Min-Max标准化**因其结果固定在[0,1]区间，更便于后续的加权评分计算，因此常被采用。

## 多因子评分流程
现在，我们可以将标准化后的因子用于评分。假设我们要根据“市值”（越小越好）和“ROE”（越大越好）选股：

1.  **数据获取与标准化**：获取所有候选股票的市值和ROE数据，并分别进行Min-Max标准化。
2.  **方向调整**：对于“越小越好”的因子（如市值），我们用 `1 - 标准化值` 将其转换为“越大越好”。因为标准化后，市值小的股票其值接近0，`1-0=1`，分值变高。
3.  **计算综合评分**：为每个股票计算综合评分。例如，若我们认为市值和ROE同等重要，则：
    `综合评分 = (1 - 市值_标准化) + ROE_标准化`
4.  **排序与选股**：将所有股票按综合评分从高到低排序，选取排名靠前的N只股票作为投资组合。

通过以上步骤，我们便实现了一个同时考虑多个因子的选股策略。相较于单因子策略，它能更全面地评估股票，潜在提升策略的稳定性和表现。
