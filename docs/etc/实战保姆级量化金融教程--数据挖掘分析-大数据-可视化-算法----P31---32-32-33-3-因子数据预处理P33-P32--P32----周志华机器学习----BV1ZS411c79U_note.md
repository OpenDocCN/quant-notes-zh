## 第一步：去极值操作

![](img/dc6eb3b957bc24db068106a04085c872_30.png)

![](img/dc6eb3b957bc24db068106a04085c872_32.png)

![](img/dc6eb3b957bc24db068106a04085c872_34.png)

去极值的目的是处理数据中的异常值或离群点，防止它们对整体分析产生过大影响。我们将采用“三西格玛”法则来实现。

![](img/dc6eb3b957bc24db068106a04085c872_36.png)

![](img/dc6eb3b957bc24db068106a04085c872_38.png)

以下是去极值函数 `filter_three_sigma` 的实现代码：

![](img/dc6eb3b957bc24db068106a04085c872_40.png)

![](img/dc6eb3b957bc24db068106a04085c872_42.png)

```python
def filter_three_sigma(series, n=3):
    """
    使用三西格玛法则去除极值。
    :param series: 输入的数据序列（pandas Series）
    :param n: 西格玛倍数，默认为3
    :return: 处理后的数据序列
    """
    # 计算均值和标准差
    mean = series.mean()
    std = series.std()
    
    # 计算上限和下限
    upper_limit = mean + n * std
    lower_limit = mean - n * std
    
    # 将超出界限的值替换为界限值，而不是直接删除
    return series.clip(lower=lower_limit, upper=upper_limit)
```

![](img/dc6eb3b957bc24db068106a04085c872_44.png)

![](img/dc6eb3b957bc24db068106a04085c872_46.png)

![](img/dc6eb3b957bc24db068106a04085c872_48.png)

![](img/dc6eb3b957bc24db068106a04085c872_50.png)

该函数计算数据的均值（`mean`）和标准差（`std`），然后根据公式 **上限 = 均值 + n × 标准差** 和 **下限 = 均值 - n × 标准差** 确定正常值的范围，最后将超出此范围的值调整到边界上。

标准化（或Z-Score标准化）的目的是将数据转换为均值为0、标准差为1的分布，使得不同量纲或量级的指标之间具有可比性。

![](img/dc6eb3b957bc24db068106a04085c872_64.png)

![](img/dc6eb3b957bc24db068106a04085c872_66.png)

以下是标准化函数 `standardize` 的实现代码：

![](img/dc6eb3b957bc24db068106a04085c872_68.png)

![](img/dc6eb3b957bc24db068106a04085c872_70.png)

![](img/dc6eb3b957bc24db068106a04085c872_72.png)

```python
def standardize(series):
    """
    对数据序列进行标准化（Z-Score标准化）。
    :param series: 输入的数据序列（pandas Series）
    :return: 标准化后的数据序列
    """
    # 计算均值和标准差
    mean = series.mean()
    std = series.std()
    
    # 应用标准化公式：(值 - 均值) / 标准差
    return (series - mean) / std
```

![](img/dc6eb3b957bc24db068106a04085c872_74.png)

![](img/dc6eb3b957bc24db068106a04085c872_76.png)

其核心公式为：**标准化值 = (原始值 - 均值) / 标准差**。经过处理的数据将符合标准正态分布。

## 第三步：中性化操作

![](img/dc6eb3b957bc24db068106a04085c872_84.png)

![](img/dc6eb3b957bc24db068106a04085c872_86.png)

中性化的目的是消除其他常见风险因子（如市值）对当前分析因子的影响，从而提取出因子“纯粹”的收益预测能力。我们通过线性回归来实现。

![](img/dc6eb3b957bc24db068106a04085c872_88.png)

![](img/dc6eb3b957bc24db068106a04085c872_90.png)

![](img/dc6eb3b957bc24db068106a04085c872_92.png)

以下是中性化函数 `neutralize` 的实现代码：

![](img/dc6eb3b957bc24db068106a04085c872_94.png)

```python
import statsmodels.api as sm

![](img/dc6eb3b957bc24db068106a04085c872_96.png)

![](img/dc6eb3b957bc24db068106a04085c872_98.png)

![](img/dc6eb3b957bc24db068106a04085c872_100.png)

def neutralize(factor_series, market_cap_series):
    """
    对因子进行市值中性化处理。
    :param factor_series: 待中性化的因子数据序列（如市净率）
    :param market_cap_series: 作为控制变量的数据序列（如市值）
    :return: 中性化后的因子残差序列
    """
    # 准备数据：Y是因变量（因子），X是自变量（市值），需要添加常数项
    Y = factor_series.astype(float)
    X = market_cap_series.astype(float)
    X = sm.add_constant(X)  # 添加常数项，用于拟合截距
    
    # 使用普通最小二乘法（OLS）进行线性回归
    model = sm.OLS(Y, X).fit()
    
    # 返回残差，即去除市值影响后的“纯净”因子值
    return model.resid
```

![](img/dc6eb3b957bc24db068106a04085c872_102.png)

![](img/dc6eb3b957bc24db068106a04085c872_104.png)

![](img/dc6eb3b957bc24db068106a04085c872_106.png)

在这个函数中：
1.  我们将待处理的因子（如市净率）作为因变量 `Y`。
2.  将需要控制的风险因子（如市值）作为自变量 `X`。
3.  使用 `statsmodels` 库的 `OLS`（普通最小二乘法）进行线性回归拟合。
4.  回归模型的残差（`model.resid`）即为中性化后的结果，它代表了剔除市值影响后，因子本身的价值。

本节课中我们一起学习了因子数据预处理的三个核心步骤。我们首先使用**三西格玛法**实现了去极值，以抑制异常值的影响。接着，通过**Z-Score标准化**将数据转换到统一量纲。最后，利用**线性回归**技术完成了因子的市值中性化，提取出其独立于市值的预测信息。这三个步骤是构建稳健量化模型的重要数据基础。
