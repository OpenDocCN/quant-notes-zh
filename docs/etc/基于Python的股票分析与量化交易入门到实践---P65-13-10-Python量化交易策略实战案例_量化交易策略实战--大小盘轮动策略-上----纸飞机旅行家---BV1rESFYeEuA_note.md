## 策略原理介绍 🔍

上一节我们介绍了低估值量化交易策略，本节中我们来看看大小盘轮动策略。该策略源自国泰君安于2011年11月10日发布的一份著名研报《风格轮动下的基金组合策略》。与之前介绍的技术指标类或宏观政策类策略不同，此策略展示了如何从专业研报中汲取思想并应用于A股股票交易。

该研报的核心思想是研究市场整体风格的周期性变化。市场有时是大盘股表现强势（大盘风格），有时则是小盘股表现更佳（小盘风格），还有时处于震荡或均衡状态。为了量化这种风格轮动，研报提出了一个核心指标：相对强弱指标（RS）。

## 核心指标：RS指标 📊

### 指标构建基础

RS指标的构建依赖于两个关键的市场风格代表指数：
*   **大盘风格代表**：**沪深300指数**。
*   **小盘风格代表**：**创业板指数**。（注：原研报中使用中证500，但因市值变化，本策略调整为更具小盘代表性的创业板指数。）

### 指标算法

RS指标的计算公式旨在对比沪深300与创业板的超额收益。其数学表达式如下：

`RS = ln(沪深300指数收益率) - ln(创业板指数收益率)`

其中，`ln`代表自然对数。该公式的直观含义是：
*   当 `RS > 0` 时，表示大盘风格相对强势。
*   当 `RS < 0` 时，表示小盘风格相对强势。

### 指标应用与趋势研判

仅凭RS值的正负判断风格过于简单。研报进一步将RS值的时间序列曲线进行**正弦滤波**处理，以平滑噪音、凸显趋势。

通过对滤波后的曲线求**一阶导数**和**二阶导数**，我们可以更精细地研判市场风格的趋势与加速度：
*   **一阶导数 > 0**：RS值处于上升趋势，大盘风格在强化。
*   **一阶导数 < 0**：RS值处于下降趋势，小盘风格在强化。
*   **二阶导数**：反映趋势变化的速度（加速或减速）。

结合一阶和二阶导数的正负，可以将市场状态划分到四个象限中，从而更准确地判断风格轮动的阶段。

## 投资标的与策略具象化 🎯

要将对市场风格的预测转化为实际投资，需要找到合适的投资标的。以下是本策略选择的标的：

*   **大盘投资标的**：**沪深300ETF**。
*   **小盘投资标的**：**创业板ETF**。

选择ETF的原因在于，它们像股票一样可以在交易时段内实时买卖，并且以较低的门槛（低至一手约百元）让投资者便捷地投资整个指数，避免了直接购买一篮子成分股的高资金门槛问题。

根据RS指标及其导数的状态，我们可以将市场划分为三种主要的投资状态，并据此制定交易规则：

以下是基于四象限分析得出的交易信号识别规则：

1.  **大盘占优状态**：当市场处于大盘趋势且加速上涨时，应**全仓持有沪深300ETF**。
2.  **小盘占优状态**：当市场处于小盘趋势且加速下跌时，应**全仓持有创业板ETF**。
3.  **均衡状态**：当市场处于其他状态（如大盘趋势但减速、小盘趋势但上涨等）时，应采取均衡配置，即**沪深300ETF与创业板ETF各持仓一半**。

这样，我们就将复杂的市场风格研判，简化为清晰的三状态交易规则。

## 策略实现思路 💻

实现大小盘轮动策略的总体思路与我们之前介绍的其他策略类似，主要包含以下几个步骤：

1.  **导入必要的函数库**：由于涉及数学计算（求导、滤波），需要导入`math`等库；同时`pandas`、`numpy`等数据处理库必不可少。
2.  **设定策略基础参数**：包括选择的指数代码（沪深300、创业板）、ETF代码、计算RS和滤波所需的时间窗口参数等。
3.  **编写初始化函数**：在策略回测或实盘开始时，进行必要的变量初始化工作。
4.  **实现策略信号识别函数**：这是策略的核心，根据每日计算出的RS值及其导数，按照上述规则判断当前应处的状态（大盘/小盘/均衡）。
5.  **实现策略交易函数**：根据信号识别函数产生的信号，执行具体的调仓操作（全仓大盘ETF、全仓小盘ETF或均衡持仓）。
