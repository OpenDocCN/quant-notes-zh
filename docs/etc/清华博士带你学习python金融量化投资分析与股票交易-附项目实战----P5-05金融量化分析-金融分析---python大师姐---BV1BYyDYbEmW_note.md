# Python金融量化投资分析：P5：金融分析 📈

在本节课中，我们将学习金融分析的核心方法。金融分析是判断股票买卖时机、预测价格走势的关键手段，避免盲目投资。我们将介绍基本面分析和技术面分析两大主流方法，并重点讲解技术分析中的K线与均线这两个基础且重要的概念。

上一节我们介绍了金融和股票的基础知识，本节中我们来看看如何进行具体的金融分析。

## 基本面分析

基本面分析的核心是评估公司的内在价值，通过研究影响股价的公司自身因素、行业状况及宏观经济环境来判断投资价值。它主要包含以下三个方面：

以下是基本面分析的三个层面：

1.  **宏观经济面分析**：分析国家的财政政策、货币政策等宏观因素，判断整体经济环境对股市资金流向的影响。但需注意，宏观经济规律与股市表现并非总是完全一致。
2.  **行业分析**：评估特定行业（如IT、教育、能源）的整体发展前景和景气度。可以通过观察该行业内几只代表性股票的走势来辅助判断。
3.  **公司分析**：这是最具体的层面，针对目标上市公司进行研究。上市公司需定期公开经审计的财务报表，投资者可通过分析财报数据（如盈利能力、每股收益）、关注公司新闻、甚至实地考察等方式，判断公司运营状况的好坏。如果公司运营良好、盈利能力强，则其股票可能具备投资价值。

## 技术面分析

![](img/afc91e2c2a7f407db3ce68e02ef8418d_1.png)

技术面分析与基本面分析不同，其核心理念是“市场行为包容消化一切信息”。它认为所有影响价格的因素都已反映在历史交易数据中，因此主要通过分析历史价格走势和交易量等市场信息来预测未来价格变动。技术分析依赖于各种技术指标。

以下是两个最基础且常见的技术指标：

![](img/afc91e2c2a7f407db3ce68e02ef8418d_3.png)

### K线图

![](img/afc91e2c2a7f407db3ce68e02ef8418d_5.png)

K线图是记录股票每日价格变动的图表。一根K线包含了四个关键价格：**开盘价(Open)**、**收盘价(Close)**、**最高价(High)**、**最低价(Low)**。

K线分为阳线和阴线：
*   **阳线**（通常显示为红色或空心）：表示当日股价上涨，即收盘价高于开盘价。实体部分的下沿是开盘价，上沿是收盘价。
*   **阴线**（通常显示为绿色或蓝色）：表示当日股价下跌，即收盘价低于开盘价。实体部分的上沿是开盘价，下沿是收盘价。

无论是阳线还是阴线，从实体部分向上延伸的细线叫“上影线”，其顶端代表当日**最高价**；向下延伸的细线叫“下影线”，其底端代表当日**最低价**。

K线可能存在一些特殊形态，例如“十字星”（开盘价等于收盘价）或“光头光脚线”（没有上下影线），这些形态都具有特定的市场含义。

### 移动平均线

移动平均线是通过计算过去一段时间内股价的平均值，并将这些平均值连接起来形成的曲线。它用于平滑价格波动，反映股价的趋势。

常见的移动平均线包括：
*   **MA5**：5日均线，计算过去5个交易日收盘价的平均值。
*   **MA60**：60日均线，计算过去60个交易日收盘价的平均值。

![](img/afc91e2c2a7f407db3ce68e02ef8418d_7.png)

![](img/afc91e2c2a7f407db3ce68e02ef8418d_9.png)

其计算公式可以简化为：
`MA(N) = (P1 + P2 + ... + PN) / N`
其中，`P1` 到 `PN` 代表过去N个交易日的收盘价。

在图表中，短期均线（如MA5）对价格变化更敏感，波动较大；长期均线（如MA60）则更平滑，能更好地显示长期趋势。观察不同周期均线之间的位置关系（例如金叉、死叉）是常用的趋势判断方法。

本节课中我们一起学习了金融分析的两大支柱：基本面分析和技术面分析。基本面分析从公司、行业、宏观经济层面评估价值；技术面分析则专注于市场价格走势本身，并初步认识了K线和移动平均线这两个基础技术指标。理解这些概念是后续进行量化策略设计和编程实现的基础。
