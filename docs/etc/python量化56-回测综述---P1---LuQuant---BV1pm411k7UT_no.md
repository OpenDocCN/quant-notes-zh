# Python量化交易教程：P1：回测基础 📊

在本节课中，我们将学习交易策略开发中一个至关重要的环节——**回测**。我们将了解回测的基本概念，并使用一个名为 `backtesting.py` 的Python库，对一个结合了支撑阻力位与蜡烛图形态的策略进行实际回测，评估其历史表现。

## 概述
到目前为止，我们提出了不同的交易策略，并主要依赖统计技术来检验模型的可靠性。回测为我们带来了额外的关键信息：**策略在特定历史时期内可能赚取的利润金额或百分比**。这个信息对于决定是否值得为某个策略投入资金至关重要。

## 数据准备与策略信号生成
上一节我们介绍了回测的重要性，本节中我们来看看如何准备数据并生成交易信号。

首先，我们加载并清理历史价格数据。接着，我们应用之前课程中介绍过的两个技术指标：
1.  **支撑与阻力位自动检测**。
2.  **蜡烛图形态检测**（如吞没形态、流星线、上吊线等长尾拒绝模式）。

交易信号的生成逻辑基于上述指标的组合：
*   **买入信号**：当价格接近**支撑位**，且支撑位旁出现**看涨拒绝形态**的蜡烛时。
*   **卖出信号**：当价格接近**阻力位**，且阻力位旁出现**看跌拒绝形态**的蜡烛时。

我们将这些信号计算出来，并填充到数据框的一个新列中。为了与后续的 `backtesting.py` 库兼容，需要确保数据框的列名符合其要求（区分大小写）。一个典型的数据框结构如下：

```python
# 数据框列结构示例
df.columns = ['Open', 'High', 'Low', 'Close', 'Volume', 'Signal']
# Signal 列包含我们计算好的交易信号（例如，1代表买入，-1代表卖出，0代表无信号）
```

## 使用 Backtesting.py 构建策略类
在准备好数据和信号之后，我们需要定义一个策略类供回测引擎使用。

`backtesting.py` 库要求我们创建一个继承自 `Strategy` 基类的新类，并重写两个核心方法：`init()` 和 `next()`。

*   **`init()` 方法**：在策略回测开始时**仅运行一次**。通常用于预计算策略所依赖的指标和信号。我们将在这里通过 `I()` 函数将数据框中的信号列注入到策略中，确保每个蜡烛都能访问其对应的信号值。
    ```python
    def init(self):
        self.signal = self.I(self.get_signal) # 注入信号指标
    ```
*   **`next()` 方法**：在回测过程中，对**每一根蜡烛**迭代调用。这里是执行交易决策（开仓、平仓）的地方。需要注意的是，订单会在下一根蜡烛的开盘价执行（如果使用`order.close`则是当前收盘价）。

以下是我们的策略类 `MyCandleStrategy` 的核心框架：

```python
from backtesting import Strategy

class MyCandleStrategy(Strategy):
    def init(self):
        # 注入信号指标
        self.signal = self.I(self.get_signal)

    def next(self):
        super().next()
        current_signal = self.signal[-1] # 获取当前蜡烛的信号

        # 买入逻辑
        if current_signal == 1:
            stop_loss = self.data.Close[-1] - 0.0750 # 止损价
            take_profit = self.data.Close[-1] + 0.0600 # 止盈价
            self.buy(sl=stop_loss, tp=take_profit)

        # 卖出逻辑
        elif current_signal == -1:
            stop_loss = self.data.Close[-1] + 0.0750
            take_profit = self.data.Close[-1] - 0.0600
            self.sell(sl=stop_loss, tp=take_profit)
```
> **注意**：示例中的止损（750点）和止盈（600点）值是随意设定的，在实际应用中需要通过参数优化来确定。

## 执行回测与结果分析
定义好策略类后，我们就可以执行回测并分析其性能指标了。

我们使用 `Backtest` 类，传入数据、策略类、初始资金和交易佣金等参数，然后运行回测。

```python
from backtesting import Backtest

bt = Backtest(df, MyCandleStrategy, cash=10000, commission=.002)
stats = bt.run()
print(stats)
```

运行后，我们会得到一份详细的绩效报告。以下是对关键指标的分析：

*   **回报与基准**：策略在18年（2003-2021）测试期内获得了约 **107%** 的总回报，年化回报约 **6%**。作为对比，同期的“买入并持有”回报约为 **59%**。这表明策略跑赢了市场。
*   **风险指标**：最大回撤为 **-20.5%**，平均回撤为 **-2.18%**。最长的回撤持续了927天，这意味着需要有足够的耐心和资金管理来渡过艰难时期。
*   **交易统计**：共进行了184笔交易，胜率 **55.43%**。平均每笔交易盈利 **0.37%**，平均持仓时间21天。最佳单笔交易盈利 **5.4%**，最差亏损 **-6.9%**。

这些数据表明，该策略是一个具有正期望、但收益率并非惊人的稳健型策略。它证明了方法的有效性，但不会带来暴利。

本节课中我们一起学习了量化交易策略回测的全过程。

1.  **回测的意义**：它通过历史数据模拟交易，提供了策略的盈利潜力、风险水平和关键绩效指标，是策略评估不可或缺的步骤。

3.  **实战演练**：我们对一个结合了技术分析（支撑阻力与蜡烛形态）的策略进行了回测，并解读了其产生的夏普比率、最大回撤、胜率等重要数据。
