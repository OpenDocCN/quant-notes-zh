# Python金融量化：48：打分法选股策略概述 🧮

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分选股法。我们将从概念入手，解释其核心思想，并明确实施该策略所需的关键前提条件。

## 策略核心思想

上一节我们介绍了如何通过单一因子（如市盈率）来评估股票。本节中我们来看看一种更综合的方法。

因子打分选股的核心思想是：**不依赖单一指标，而是综合多个因子对股票进行整体评估**。这类似于学生的总成绩由各科分数加总而成。

假设我们有300只股票（股票1， 股票2， …， 股票300），每只股票都有A、B、C、D四个因子指标。我们的目标是从中选出表现最好的10只。

**具体步骤如下：**
1.  为每只股票的每个因子进行打分。
2.  将所有因子的分数加总，得到该股票的“总分”。
3.  根据总分对所有股票进行排名。
4.  选取排名最高的前10只股票作为投资组合。

**公式表示如下：**
`股票总分 = 因子A得分 + 因子B得分 + 因子C得分 + 因子D得分`

通过这种方式，我们能够基于一个综合的、多维度的评价体系来筛选股票，而不仅仅是看某一个方面。

## 关键前提条件

![](img/0ab4e05aad4a01529656d41327e60c19_1.png)

在实施打分之前，我们必须明确一个关键的先验知识：**每个因子与预期收益率的相关性方向**。

这是因为，打分需要依据因子数值的大小，但并非所有因子都是“数值越大越好”。例如：
*   **正向因子**：因子值越大，预期收益可能越高（如每股收益、净资产收益率）。
*   **负向因子**：因子值越小，预期收益可能越高（如市盈率、市净率）。

以下是确定因子相关性方向的两种主要途径：
*   **查阅研究报告**：各大券商的研究报告会提供相关的经验结论。
*   **自行计算分析**：通过历史数据计算因子与收益率的相关系数进行验证。

在后续的实际编程中，我们将选取几组已知特性的因子（一组越高越好，一组越低越好）作为示例。掌握因子的属性是构建有效打分模型的基础。
