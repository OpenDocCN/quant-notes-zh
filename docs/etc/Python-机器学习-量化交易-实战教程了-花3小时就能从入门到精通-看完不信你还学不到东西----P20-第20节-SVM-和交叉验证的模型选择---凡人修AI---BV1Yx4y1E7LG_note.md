在本节课中，我们将学习支持向量机（SVM）的核心原理及其从线性到非线性的演进过程，并探讨如何使用交叉验证进行模型选择。SVM是一种强大的分类与回归算法，其核心思想是寻找一个最优的决策边界来划分数据。

## 支持向量机（SVM）简介

![](img/dc02c9251e583c5184814d6dd3dc537a_3.png)

支持向量机最初是一个二元分类算法，它支持线性分类和非线性分类。非线性分类问题可以通过对线性分类的扩展来解决。经过后续演进，SVM现在也支持多元分类，并且经过进一步扩展，可以用于回归问题。因此，支持向量机可以应用于机器学习中的各类问题。

![](img/dc02c9251e583c5184814d6dd3dc537a_5.png)

## 线性分类与超平面

首先，让我们简单地感受一下分类超平面的概念。线性感知机模型的目标是找到一条直线（在二维空间中）或一个超平面（在更高维度中），能够将二元数据分隔开。

![](img/dc02c9251e583c5184814d6dd3dc537a_7.png)

对于线性感知机，我们将这个超平面定义为：
`W^T * X + b = 0`
如果X是二维的，这就是一个平面；如果是一维的，就是一条线；三维或更高维则称为超平面。

![](img/dc02c9251e583c5184814d6dd3dc537a_9.png)

如下图所示，在超平面上方，我们定义Y=1；在超平面下方，定义Y=-1。这是一个二元分类问题。通过这个定义，我们找到了这样一条线，线上方是Y=1类，下方是Y=-1类。

实际上，满足这样条件的线非常多，例如平移或适当旋转都可以。这导致我们无法得到一个唯一解。因此，我们需要思考：在众多的分类超平面中，哪一个是最好的？或者说，哪一个的鲁棒性最强？为此，我们引入了最大化间隔（margin）的思想。

![](img/dc02c9251e583c5184814d6dd3dc537a_11.png)

## 几何间隔与最大化间隔

![](img/dc02c9251e583c5184814d6dd3dc537a_13.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_15.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_17.png)

在介绍SVM的损失函数之前，我们先来介绍什么是间隔，或者说几何距离。

![](img/dc02c9251e583c5184814d6dd3dc537a_19.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_21.png)

假设我们已经得到了超平面 `W^T * X + b = 0`。对于超平面外的一个点，我们计算该点到超平面的距离。这个距离的公式是：
`y * (W^T * X + b) / ||W||`
这个公式在任何解析几何教材中都有介绍，用于计算点到直线的距离。这个距离是带方向的：如果y与括号内的值同号，则距离为正；异号则为负。这个几何间隔就是点到超平面的带符号的真实距离。

![](img/dc02c9251e583c5184814d6dd3dc537a_23.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_25.png)

在支持向量机中，我们利用的就是这个几何距离。我们引入最大化间隔的想法，即最大化所有点到最终定义的超平面（或决策边界）的距离。

![](img/dc02c9251e583c5184814d6dd3dc537a_27.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_29.png)

这样做的好处是，在优化的同时，我们希望所有点都离超平面尽可能远。这样，即使数据存在一些扰动，这些点也不太可能越过分类边界。那些真正离超平面较远的点我们并不关心，我们关心的是离超平面最近的点。最大化这些最近点到超平面的距离，就是我们求得的最大间隔分类超平面。

![](img/dc02c9251e583c5184814d6dd3dc537a_31.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_33.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_35.png)

如下图所示，落在两条虚线上点，是离超平面最近的点，它们刚好定义了最小的间隔。这些点被称为**支持向量**。

![](img/dc02c9251e583c5184814d6dd3dc537a_37.png)

## SVM的损失函数与优化

![](img/dc02c9251e583c5184814d6dd3dc537a_39.png)

有了几何距离，我们就可以定义损失函数了。损失函数的思想是：让所有被误分类的点到超平面的距离之和最小化。

通过上面介绍的几何间隔（Geometric Margin），其表达式为 `y_i * (W^T * x_i + b) / ||W||`。这些距离的累加和，我们希望最小化。如果我们假设距离已经被归一化（即分母||W||=1），那么损失函数如下所示。

![](img/dc02c9251e583c5184814d6dd3dc537a_41.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_43.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_45.png)

我们将SVM写成一个带约束的最优化表达式：我们要最大化最小的间隔（margin）。这个margin定义为 `min_i (y_i * (W^T * x_i + b))`。

同时，我们要满足所有点的距离都要大于等于这个最小的margin。为了简化表达式，我们可以令函数间隔取1，那么优化函数如下所示。最大化其倒数等价于最小化其本身，因此SVM的优化函数可以写成如下形式。

![](img/dc02c9251e583c5184814d6dd3dc537a_47.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_49.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_51.png)

这个表达式非常像带不等式约束的线性回归。线性回归是最小化均方误差的二范数，而这里是最小化被归一化的几何距离。具体的每一步推导在此不细说，但核心要记住：SVM的想法是最大化最小的几何间隔距离，这是一个对偶优化过程。

我们使用拉格朗日乘子法来解上述最优化算式。目标函数减去约束条件（α乘以约束式），得到拉格朗日函数。我们的优化目标就变成了优化这个拉格朗日函数。

![](img/dc02c9251e583c5184814d6dd3dc537a_53.png)

根据KKT条件中的对偶互补条件，我们可以得到 `α_i * [y_i * (W^T * x_i + b) - 1] = 0`。我们来分析这个式子：
*   如果 `α_i > 0`，那么括号内的式子必须等于0，即 `y_i * (W^T * x_i + b) = 1`。这表示点 `x_i` 刚好落在之前图片上的两条虚线上。
*   如果括号内的式子大于等于1，则说明这些点不是支持向量，它们位于两条虚线围成的间隔带之外。

支持向量机一个有趣的特点是：那些被正确分类的“优秀”点本身并不参与支持向量机参数的拟合。真正参与参数拟合的点，是那些刚好落在间隔带上、容易被错分的点。这些点被称为**支持向量**，这也是“支持向量机”名字的由来。

## 线性SVM面临的问题与软间隔

![](img/dc02c9251e583c5184814d6dd3dc537a_55.png)

现在，我们介绍一下线性SVM面临的问题。最直观的问题是数据的可分性。
*   有些数据是完美线性可分的，可以被很好地划分。
*   但有些数据则存在一些异常点，使得我们无法找到一条直线完美地将两组点分开。我们仍然希望进行分类，毕竟只是少数异常点。
*   还有一种情况是，数据并非完全不可分，但一个点的微小扰动会导致分类平面发生很大改变。我们不希望模型如此不鲁棒。

![](img/dc02c9251e583c5184814d6dd3dc537a_57.png)

为了解决这些问题，我们引入**软间隔最大化**的想法。

![](img/dc02c9251e583c5184814d6dd3dc537a_59.png)

回顾最基础的SVM（硬间隔），我们要求 `y_i * (W^T * x_i + b)` 必须完美地大于等于1。软间隔则引入一个松弛变量 `ξ_i (xi_i >= 0)`，约束条件变为：
`y_i * (W^T * x_i + b) >= 1 - ξ_i`
减去一个小的扰动 `ξ_i`，意味着我们允许一些点进入间隔带，甚至允许一些点被错误分类。但我们必须对这个错分率有所控制，否则学习就失去了意义。

![](img/dc02c9251e583c5184814d6dd3dc537a_61.png)

因此，软间隔的优化目标变为：
`min (1/2 * ||W||^2) + C * Σ(ξ_i)`
约束条件为：`y_i * (W^T * x_i + b) >= 1 - ξ_i` 且 `ξ_i >= 0`。

![](img/dc02c9251e583c5184814d6dd3dc537a_63.png)

其中，`C > 0` 称为惩罚参数。`C` 越大，对分类错误的惩罚越大，意味着我们只允许非常少的错分，间隔带会变窄。`C` 越小，对错误的惩罚越小，间隔带就可以越宽。软间隔在最大化间隔和减少误分类点之间进行权衡，`C` 的调整在实际应用中非常重要，通常需要通过交叉验证和网格搜索来优化。

![](img/dc02c9251e583c5184814d6dd3dc537a_65.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_67.png)

## 从线性到非线性：核函数

![](img/dc02c9251e583c5184814d6dd3dc537a_69.png)

接下来，我们讨论非线性情况。对于线性情况，我们最多只能做到线性的划分。但在很多情况下，非线性的划分（例如画一个圆）能更完美、更合理地分隔数据。这时，我们需要引入非线性SVM分类器。

![](img/dc02c9251e583c5184814d6dd3dc537a_71.png)

对于非线性问题，一个简单的想法是进行特征映射。例如，一个二次多项式可以通过变量替换，映射到一个更高维的线性空间，从而应用线性SVM。但问题是，当特征交叉项很多时，映射后的维度会指数级爆炸，带来极高的计算复杂度。

![](img/dc02c9251e583c5184814d6dd3dc537a_73.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_75.png)

因此，我们需要另辟蹊径，引入**核函数**的想法。

在SVM的最优化问题中，最终表达式涉及样本 `x_i` 和 `x_j` 的内积 `(x_i · x_j)`。对于非线性情况，这个内积可以替换为映射后特征的内积 `(φ(x_i) · φ(x_j))`。核函数 `K(x_i, x_j)` 的精妙之处在于，它直接计算低维空间中的某个函数值，其结果等于在高维特征空间中进行内积计算。这样，我们就避免了显式进行高维映射的恐怖计算量，却获得了高维特征空间线性可分的好处。

![](img/dc02c9251e583c5184814d6dd3dc537a_77.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_79.png)

以下是常见的核函数：
*   **线性核函数**：`K(x, z) = x · z`
*   **多项式核函数**：`K(x, z) = (γ * x · z + r)^d`，其中 `d` 是最高次数。
*   **高斯核函数（RBF）**：`K(x, z) = exp(-γ * ||x - z||^2)`，其中 `γ` 控制分布的集中程度。
*   **Sigmoid核函数**：`K(x, z) = tanh(γ * x · z + r)`

![](img/dc02c9251e583c5184814d6dd3dc537a_81.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_83.png)

![](img/dc02c9251e583c5184814d6dd3dc537a_85.png)

最常用的是高斯核函数，通常只需要调整惩罚参数 `C` 和核参数 `γ` 即可。

![](img/dc02c9251e583c5184814d6dd3dc537a_87.png)

## 从二分类到多分类

我们之前介绍的SVM都是二分类器。将其推广到多分类，主要有两种策略：
1.  **一对一（One-vs-One）**：如果有 `K` 个类别，则构建所有两两组合的分类器，共 `C(K, 2)` 个。对新样本，让所有分类器投票，得票最多的类别即为最终类别。
2.  **一对多（One-vs-Rest）**：构建 `K` 个分类器，第 `i` 个分类器将第 `i` 类作为正类，其余所有类作为负类。对新样本，同样通过投票决定其类别。

## 模型选择与交叉验证

在讲解交叉验证之前，让我们先回顾一下偏差-方差权衡。

![](img/dc02c9251e583c5184814d6dd3dc537a_89.png)

我们构建模型的目标是最小化均方误差。误差可以分解为三部分：偏差的平方、方差和不可约误差。不可约误差无法改变，我们只能通过最小化前两项来最小化总误差。
*   **偏差**：度量了算法本身的拟合能力，即模型预测的准确性。
*   **方差**：度量了模型对数据扰动的敏感度，即模型的稳定性（鲁棒性）。

我们希望偏差和方差都小，但在实际操作中，它们往往不可兼得。当模型复杂度增加（偏差减小），方差往往会增大。因此，我们需要找到一个最优的平衡点，使总误差最小。交叉验证就是帮助我们进行模型选择的重要方法。

![](img/dc02c9251e583c5184814d6dd3dc537a_91.png)

交叉验证的主要想法是：我们不能仅仅通过训练集上的误差来评估模型是否过拟合，而需要通过测试集上的误差来评估。拿到一组数据后，我们不知道哪部分该做训练集，哪部分该做测试集。交叉验证将整个数据集人为地划分为训练集和验证集。

以下是两种主要方法：
*   **留一法交叉验证（LOOCV）**：假设有N个数据点，每次取一个点作为测试集，其余N-1个点作为训练集，重复N次，最后将N次测试误差取平均。
*   **K折交叉验证（K-Fold CV）**：将数据随机分为K份（例如5份）。每次取其中一份作为测试集，其余K-1份作为训练集，重复K次，最后将K次测试误差取平均。

```python

```
*   `C`：惩罚参数，值越大表示对错分的容忍度越低。
*   `kernel`：核函数类型，`‘rbf’` 表示高斯核。
*   `gamma`：高斯核的参数，控制单个样本的影响范围。值越小，影响范围越广，模型越平滑。
