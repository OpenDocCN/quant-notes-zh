## 从资本资产定价模型到机器学习

上一节我们明确了所需的编程工具，本节中我们来看看如何将金融领域的经典模型与机器学习联系起来。我们以资本资产定价模型作为桥梁。

### CAPM模型的金融学表述

资本资产定价模型描述的是单个资产（或资产组合）的收益与市场整体收益之间的关系。

对于单一股票 *i*，在时刻 *t* 的模型公式为：
`R_i(t) = β_i * R_m(t) + α_i(t)`
其中：
*   `R_i(t)` 是股票 *i* 在时刻 *t* 的收益率。
*   `R_m(t)` 是市场（如沪深300指数）在时刻 *t* 的收益率。
*   `β_i` 是该股票相对于市场收益的敏感系数。
*   `α_i(t)` 是残差项，代表无法被市场解释的超额收益。

根据有效市场假说，从长期来看，残差项 `α_i(t)` 的期望值为零，即 `E[α_i(t)] = 0`。

### 机器学习视角的重新解读

现在，让我们换一个角度。如果我们不预先知道CAPM，而只是想预测股票 *i* 的收益率 `R_i(t)`，我们会怎么做？

1.  **定义问题**：我们的目标变量（标签）`Y` 就是 `R_i(t)`。
2.  **寻找特征**：我们需要找到影响 `Y` 的特征变量 `X`。
3.  **建立模型**：我们假设存在一个函数 `f`，使得 `Y ≈ f(X)`，并附带一个期望为零的随机噪声 `ε`，即 `Y = f(X) + ε`。

**那么，CAPM模型实际上做了一个非常强的假设**：
*   它认为特征 `X` 只有一个，就是市场收益率 `R_m(t)`。
*   它认为函数 `f` 是线性的，即 `f(X) = β * X + α`。

因此，从机器学习的角度看，**CAPM就是一个使用单一特征（市场收益）的线性回归模型**。拟合这条回归线的过程，就是利用历史数据求解 `β`（斜率）和 `α`（截距）的过程。

### 模型的图形化与扩展

我们可以将股票收益率与市场收益率的历史数据画成散点图，然后用线性回归拟合出一条直线。这条直线的斜率就是 `β`，截距就是 `α`。

这个框架的强大之处在于它的可扩展性：
*   **特征`X`**：我们可以不局限于市场收益，可以加入更多特征，例如移动平均线、动量、交易量等，将 `X` 从一维扩展到多维。
*   **函数`f`**：我们可以不局限于线性函数，可以尝试支持向量机、随机森林等非线性模型。

**技术分析的本质，就是利用历史价格和交易量数据，构造出一系列特征`X`，然后寻找一个函数`f`来预测未来的价格变化。**

## 技术分析基础与特征工程

理解了模型框架后，我们来看看如何构造特征`X`。技术分析提供了丰富的思路来从历史数据中提取特征。

以下是几种经典的技术指标，它们都可以作为机器学习模型的输入特征：

*   **动量**：衡量价格变化的速度和方向。公式为 `Momentum(t) = Price(t) / Price(t-n) - 1`。其中 `n` 是回溯周期，不同的 `n` 可以产生不同的动量特征。
*   **简单移动平均线**：对过去 `n` 个周期的价格取平均值，用于平滑价格序列，反映趋势。`SMA(t) = (Price(t) + Price(t-1) + ... + Price(t-n+1)) / n`。
*   **布林带**：在移动平均线上下各画出两条标准差带，用以衡量价格的波动性。当价格触及或突破布林带边界时，可能预示着趋势的转折。

**特征标准化**：由于不同特征（如SMA和动量）的数值范围可能差异很大，在将它们输入模型前，必须进行标准化处理。通常使用 **Z-score标准化**：`(特征值 - 特征均值) / 特征标准差`。这能确保每个特征在模型中具有可比的重要性。

## 机器学习模型初步：从K近邻到线性回归

有了特征之后，我们需要选择具体的函数 `f`，即机器学习模型。我们从最简单直观的模型开始。

### K近邻算法

KNN是一种非参数模型。对于一个新的数据点，它的预测值是基于历史数据中与其最相似的K个点的标签的平均值。

*   **优点**：直观，无需假设数据分布。
*   **缺点**：在特征维度很高时，由于“维数灾难”，寻找“最近邻”会变得低效且不准确；模型也缺乏解释性。

### 线性回归模型

线性回归假设目标变量 `Y` 是特征 `X` 的线性组合。模型形式为：`Y = β_0 + β_1*X_1 + β_2*X_2 + ... + β_p*X_p + ε`。

拟合线性回归的目标是找到一组系数 `β`，使得模型在所有训练样本上的预测误差（如均方误差）最小。`scikit-learn` 可以轻松完成这项工作。

### 模型复杂度与过拟合

模型复杂度描述了函数 `f` 的灵活程度。
*   **低复杂度模型**（如线性回归）：假设简单，可能无法捕捉数据中的复杂关系，导致 **欠拟合**（高偏差）。
*   **高复杂度模型**（如高阶多项式回归、深度神经网络）：非常灵活，可以紧密拟合训练数据，但可能过度关注噪声，导致 **过拟合**（高方差），即在训练集上表现好，在新数据上表现差。

机器学习的一个核心挑战就是 **偏差-方差权衡**。我们的目标是找到一个平衡点，使模型在训练集和未见过的测试集上都能有良好的表现。通常，我们会将数据分为训练集和测试集，用训练集拟合模型，用测试集评估其泛化能力。
