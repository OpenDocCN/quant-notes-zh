## 第三步：改进可视化——使用双Y轴

![](img/636a62314e99c79718b3ad0bc9de6670_47.png)

为了使两个指标的趋势对比更清晰，我们可以将它们绘制在同一张图上，但使用左右两个不同的Y轴。

![](img/636a62314e99c79718b3ad0bc9de6670_49.png)

![](img/636a62314e99c79718b3ad0bc9de6670_51.png)

以下是使用双Y轴绘制折线图的代码：

![](img/636a62314e99c79718b3ad0bc9de6670_53.png)

![](img/636a62314e99c79718b3ad0bc9de6670_55.png)

```python
# 使用双Y轴绘图，左侧为SPX，右侧为VIX
ax = data['SPX'].plot(legend=True, figsize=(10,6))
data['VIX'].plot(ax=ax, secondary_y=True, legend=True)
ax.set_ylabel('SPX')
ax.right_ax.set_ylabel('VIX')
```

![](img/636a62314e99c79718b3ad0bc9de6670_57.png)

![](img/636a62314e99c79718b3ad0bc9de6670_59.png)

为了更清晰地观察短期关系，我们可以只选取一段时间的数据（例如2010年至2012年）进行绘图。

![](img/636a62314e99c79718b3ad0bc9de6670_61.png)

```python
# 选取2010-01-01到2012-12-31期间的数据
data_subset = data['2010-01-01':'2012-12-31']
ax = data_subset['SPX'].plot(legend=True, figsize=(12,6))
data_subset['VIX'].plot(ax=ax, secondary_y=True, legend=True)
ax.set_ylabel('SPX')
ax.right_ax.set_ylabel('VIX')
```

通过观察改进后的图表，我们可以初步发现：当`SPX`指数下降时，`VIX`指数往往上升，呈现出一种负相关关系。这是通过可视化得到的最直接的洞察。

## 第四步：深入分析——绘制散点图与分布图

![](img/636a62314e99c79718b3ad0bc9de6670_63.png)

![](img/636a62314e99c79718b3ad0bc9de6670_65.png)

除了观察时间序列上的走势，我们还可以通过散点图来观察两个变量在每一个时间点上的关系。同时，我们可以通过分布图了解每个变量自身的分布情况。

![](img/636a62314e99c79718b3ad0bc9de6670_67.png)

Pandas库提供了便捷的函数来绘制矩阵散点图，它可以同时展示变量间的散点关系和各变量的分布。

![](img/636a62314e99c79718b3ad0bc9de6670_69.png)

以下是绘制矩阵散点图的代码示例：

![](img/636a62314e99c79718b3ad0bc9de6670_71.png)

```python
# 导入必要的绘图库
from pandas.plotting import scatter_matrix

![](img/636a62314e99c79718b3ad0bc9de6670_73.png)

![](img/636a62314e99c79718b3ad0bc9de6670_75.png)

# 绘制散点图矩阵，对角线显示核密度估计图
scatter_matrix(data, alpha=0.2, figsize=(10, 10), diagonal='kde')
```

![](img/636a62314e99c79718b3ad0bc9de6670_77.png)

![](img/636a62314e99c79718b3ad0bc9de6670_79.png)

在这张图中：
*   **对角线**：展示了每个变量（`SPX`和`VIX`）自身的**核密度估计（Kernel Density Estimate， KDE）**图，它比直方图更能平滑地反映数据的分布形态。
*   **非对角线**：展示了两个变量之间的散点图。从`SPX`和`VIX`的散点图中，可以更清晰地看到数据点从左上到右下分布的负相关趋势。

![](img/636a62314e99c79718b3ad0bc9de6670_81.png)

![](img/636a62314e99c79718b3ad0bc9de6670_83.png)

核密度估计的公式可以简化为：
`f̂(x) = (1/nh) * Σ K((x - Xi)/h)`
其中，`K`是核函数（如高斯核），`h`是带宽，`n`是样本数。它通过平滑的曲线来估计数据的概率密度。

![](img/636a62314e99c79718b3ad0bc9de6670_85.png)

![](img/636a62314e99c79718b3ad0bc9de6670_87.png)

## 总结

![](img/636a62314e99c79718b3ad0bc9de6670_91.png)

本节课中我们一起学习了如何对金融指标进行关系分析。
1.  **数据提取**：我们首先学会了从数据集中筛选出目标指标列。
2.  **可视化探索**：我们通过绘制**双Y轴折线图**，直观地观察了两个指标在时间序列上的互动关系，初步判断出`SPX`与`VIX`存在负相关。
3.  **深入分析**：我们利用**散点图矩阵**进行了更深入的分析。该图不仅通过散点图强化了变量间负相关的结论，还通过**核密度估计图**展示了每个变量自身的分布情况。
