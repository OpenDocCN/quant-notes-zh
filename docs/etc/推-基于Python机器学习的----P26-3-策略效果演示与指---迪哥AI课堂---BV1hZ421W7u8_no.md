## 数据结构调整与股票筛选

![](img/5b9274315b216a867bdd07f9a17c6915_3.png)

![](img/5b9274315b216a867bdd07f9a17c6915_5.png)

![](img/5b9274315b216a867bdd07f9a17c6915_7.png)

上一节我们介绍了如何获取股票数据，本节中我们来看看如何整理这些数据以便于后续处理。

![](img/5b9274315b216a867bdd07f9a17c6915_9.png)

![](img/5b9274315b216a867bdd07f9a17c6915_11.png)

我们最初得到的数据是横向排列的，即每一列代表一只股票，每一行代表一个财务指标。这种格式不便于我们按股票进行分析。通常，我们更喜欢纵向排列的数据，即每一行代表一只股票，每一列代表其各项指标。

为了实现这个转换，我们只需要对原始的 `DataFrame` 执行一个转置操作。在Python的pandas库中，这可以通过 `.T` 属性轻松完成。

![](img/5b9274315b216a867bdd07f9a17c6915_13.png)

![](img/5b9274315b216a867bdd07f9a17c6915_15.png)

```python
# 假设 df 是原始的横向DataFrame
df_transposed = df.T
```

![](img/5b9274315b216a867bdd07f9a17c6915_17.png)

![](img/5b9274315b216a867bdd07f9a17c6915_19.png)

执行转置后，数据结构将变为：索引（`index`）是股票代码，每一列是该股票对应的财务指标值。我们的目标是从转置后的数据中，根据特定财务指标（如营收）筛选出排名靠前的股票。

![](img/5b9274315b216a867bdd07f9a17c6915_21.png)

![](img/5b9274315b216a867bdd07f9a17c6915_23.png)

以下是实现筛选和存储的核心步骤：
1.  对原始财务数据 `DataFrame` 进行转置。
2.  根据选定的指标（例如“营收”）对转置后的数据进行排序。
3.  选取排名前N（例如10）的股票代码。
4.  将这些股票代码存储到策略的上下文（`context`）中，供后续交易函数使用。

![](img/5b9274315b216a867bdd07f9a17c6915_25.png)

![](img/5b9274315b216a867bdd07f9a17c6915_27.png)

![](img/5b9274315b216a867bdd07f9a17c6915_28.png)

```python
# 示例代码片段
transposed_df = original_df.T
# 按‘revenue’列降序排序，并取前10名
selected_stocks = transposed_df.sort_values(by='revenue', ascending=False).head(10).index.tolist()
# 存储到上下文
context.selected_stocks = selected_stocks
```

![](img/5b9274315b216a867bdd07f9a17c6915_30.png)

![](img/5b9274315b216a867bdd07f9a17c6915_32.png)

这样，在策略开始交易前（`before_trading` 函数中），我们就完成了股票的初步筛选工作。

![](img/5b9274315b216a867bdd07f9a17c6915_34.png)

![](img/5b9274315b216a867bdd07f9a17c6915_36.png)

## 交易逻辑实现：持仓判断与买卖

![](img/5b9274315b216a867bdd07f9a17c6915_38.png)

![](img/5b9274315b216a867bdd07f9a17c6915_40.png)

![](img/5b9274315b216a867bdd07f9a17c6915_42.png)

在 `handle_bar` 函数中，我们将实现每日的交易逻辑。核心思路是：每日检查当前持仓，并与最新筛选出的股票池对比，卖出不再池中的股票，买入新增的股票。

![](img/5b9274315b216a867bdd07f9a17c6915_44.png)

首先，我们需要判断当前是否有持仓。这可以通过访问策略上下文 `context` 中的 `portfolio.positions` 属性来实现。它是一个字典，包含了所有当前持仓的股票及其信息。

![](img/5b9274315b216a867bdd07f9a17c6915_46.png)

![](img/5b9274315b216a867bdd07f9a17c6915_48.png)

![](img/5b9274315b216a867bdd07f9a17c6915_50.png)

```python
# 获取当前持仓字典
current_positions = context.portfolio.positions
```

![](img/5b9274315b216a867bdd07f9a17c6915_52.png)

![](img/5b9274315b216a867bdd07f9a17c6915_54.png)

以下是具体的交易逻辑步骤：

1.  **判断初始状态**：如果 `current_positions` 为空，说明是首个交易日，我们需要执行首次建仓操作。
2.  **调仓操作（非首日）**：如果持仓不为空，则进入日常调仓逻辑。
    *   **卖出逻辑**：遍历当前持仓的股票。如果某只股票不在最新筛选出的 `context.selected_stocks` 列表中，则将其全部卖出。
    *   **买入逻辑**：遍历最新筛选出的 `context.selected_stocks` 列表。如果某只股票不在当前持仓中，则买入它。

![](img/5b9274315b216a867bdd07f9a17c6915_56.png)

![](img/5b9274315b216a867bdd07f9a17c6915_58.png)

为了实现买卖，我们使用 `order_target_percent` 函数。该函数可以方便地将某只股票的仓位调整到目标百分比。
*   `order_target_percent(security, 0)`：卖出该股票的全部持仓。
*   `order_target_percent(security, 0.1)`：将该股票的仓位调整到总资产的10%。

![](img/5b9274315b216a867bdd07f9a17c6915_60.png)

在买入时，为了简化，我们采用等权分配的方式，即每只股票买入相同的金额。假设我们要买入N只股票，那么每只股票的目标仓位就是 `1 / N`。

![](img/5b9274315b216a867bdd07f9a17c6915_62.png)

![](img/5b9274315b216a867bdd07f9a17c6915_64.png)

```python
# 示例交易逻辑代码片段
if len(current_positions) == 0:
    # 首次建仓：等权买入股票池中的所有股票
    for stock in context.selected_stocks:
        order_target_percent(stock, 1.0 / len(context.selected_stocks))
else:
    # 调仓日
    # 卖出不在新股票池中的股票
    for stock in current_positions:
        if stock not in context.selected_stocks:
            order_target_percent(stock, 0)
    # 买入新股票池中未持有的股票
    for stock in context.selected_stocks:
        if stock not in current_positions:
            order_target_percent(stock, 1.0 / len(context.selected_stocks))
```

## 回测结果分析与指标解读

策略编写完成后，我们运行回测来评估其表现。回测引擎会生成一份详细的报告，其中包含多个关键绩效指标。

![](img/5b9274315b216a867bdd07f9a17c6915_70.png)

首先从整体上看，报告会对比**策略收益曲线**与**基准收益曲线**（例如沪深300指数）。即使策略最终亏损，我们也希望它能**跑赢基准**，即亏损幅度小于大盘。

以下是几个需要重点关注的核心指标：

![](img/5b9274315b216a867bdd07f9a17c6915_72.png)

![](img/5b9274315b216a867bdd07f9a17c6915_74.png)

*   **累计收益率**：策略在整个回测期间的总收益。公式为 `(最终净值 - 初始净值) / 初始净值`。
*   **年化收益率**：将累计收益率折算成年化的形式，便于与其他策略或投资产品比较。
*   **最大回撤**：策略净值从最高点到最低点的最大跌幅。这是衡量策略风险的重要指标，回撤越小，说明策略在下跌期间控制风险的能力越强。公式为 `(峰值 - 谷值) / 峰值`。
*   **夏普比率**：衡量策略承担单位风险所获得的超额回报。夏普比率为正且越大越好，为负则说明风险调整后的收益不佳。其简化公式为 `(策略年化收益率 - 无风险利率) / 策略收益的年化标准差`。

![](img/5b9274315b216a867bdd07f9a17c6915_76.png)

此外，报告中还包含**交易详情**，可以查看每一天的具体买卖操作、持仓变化和费用情况，帮助我们对策略的交易行为进行微观分析。

![](img/5b9274315b216a867bdd07f9a17c6915_78.png)

![](img/5b9274315b216a867bdd07f9a17c6915_80.png)

**注意**：回测结果严重依赖于所选的回测时间段。在不同市场阶段（如牛市、熊市、震荡市），同一策略的表现可能差异巨大。因此，需要进行多周期、多场景的回测来更全面地评估策略的稳健性。
