## 重采样操作 🔄

![](img/61520142f8a194309d00a55e79331a7f_5.png)

上一节我们介绍了时间序列的基本处理，本节中我们来看看如何改变数据的时间频率，即重采样。

![](img/61520142f8a194309d00a55e79331a7f_7.png)

![](img/61520142f8a194309d00a55e79331a7f_9.png)

重采样是指将时间序列数据从一个时间频率转换到另一个时间频率的过程。例如，原始数据可能是按天记录的，但我们可以将其重新聚合为按周、按月或按年统计的数据。

![](img/61520142f8a194309d00a55e79331a7f_11.png)

进行重采样操作时，需要使用 `resample()` 方法，并至少指定一个新的时间频率规则。同时，你需要决定如何聚合每个新时间区间内的数据，例如是计算平均值、第一个值、中位数还是其他统计指标。

以下是重采样的基本步骤：

![](img/61520142f8a194309d00a55e79331a7f_13.png)

![](img/61520142f8a194309d00a55e79331a7f_15.png)

1.  使用 `DataFrame.resample(规则)` 方法指定新的频率。
2.  链式调用一个聚合函数（如 `.mean()`, `.first()`, `.sum()`）来计算新区间内的值。

![](img/61520142f8a194309d00a55e79331a7f_17.png)

例如，将日数据重采样为周数据并计算每周的均值：
```python
# 假设 df 是一个以日期为索引的 DataFrame
weekly_data = df.resample('W').mean()
```
执行后，数据索引将变为每周的日期（默认是每周的最后一天）。

![](img/61520142f8a194309d00a55e79331a7f_19.png)

![](img/61520142f8a194309d00a55e79331a7f_21.png)

![](img/61520142f8a194309d00a55e79331a7f_23.png)

在重采样时，可以通过 `label` 参数控制新时间区间的标签是使用区间的起始点（`'left'`）还是结束点（`'right'`）。默认是 `'right'`，即使用结束点作为标签。例如：
```python
# 使用每月起始日作为标签
monthly_data_left = df.resample('M', label='left').mean()
# 使用每月结束日作为标签（默认）
monthly_data_right = df.resample('M').mean() # 等价于 label='right'
```
选择 `'left'` 或 `'right'` 取决于具体分析需求，通常使用结束点（`'right'`）更符合观察逻辑，因为它代表该时间段内数据的统计结果。

## 时间窗口（滑动窗口）操作 ⏱️

了解了如何改变数据频率后，我们来看看另一种常见操作：时间窗口分析，也称为滑动窗口分析。

![](img/61520142f8a194309d00a55e79331a7f_25.png)

时间窗口操作不是改变数据频率，而是在一个固定长度的“窗口”内沿时间轴滑动，并计算每个窗口内的统计量。例如，有一个100天的序列，我们可以计算第一个窗口（第1-10天）的均值，然后窗口向后滑动一天，计算第二个窗口（第2-11天）的均值，依此类推。

![](img/61520142f8a194309d00a55e79331a7f_27.png)

以下是执行时间窗口操作的基本步骤：

![](img/61520142f8a194309d00a55e79331a7f_29.png)

1.  首先，对数据进行预处理，**处理缺失值**，因为窗口计算遇到缺失值（NaN）会导致结果也为NaN。
2.  使用 `DataFrame.rolling(window=窗口大小)` 方法创建滚动窗口对象。
3.  对滚动窗口对象应用聚合函数，如 `.min()`, `.max()`, `.mean()`, `.std()`。

![](img/61520142f8a194309d00a55e79331a7f_31.png)

以下是具体操作示例。首先预处理数据并定义一个窗口大小：
```python
# 1. 处理缺失值
df_clean = df.dropna()

![](img/61520142f8a194309d00a55e79331a7f_33.png)

![](img/61520142f8a194309d00a55e79331a7f_35.png)

# 2. 定义窗口大小
window_size = 10

![](img/61520142f8a194309d00a55e79331a7f_37.png)

# 3. 对某一列计算滚动窗口的最小值
rolling_min = df_clean['某列'].rolling(window=window_size).min()
```
最初几个窗口（索引小于 `window_size` 的位置）的结果会是NaN，因为数据不足以填满一个完整的窗口。

![](img/61520142f8a194309d00a55e79331a7f_39.png)

![](img/61520142f8a194309d00a55e79331a7f_41.png)

![](img/61520142f8a194309d00a55e79331a7f_43.png)

我们可以同时计算多个窗口统计指标，并将结果添加到原DataFrame中：
```python
# 计算并添加多个滚动统计量
df_clean['rolling_min'] = df_clean['某列'].rolling(window=window_size).min()
df_clean['rolling_max'] = df_clean['某列'].rolling(window=window_size).max()
df_clean['rolling_mean'] = df_clean['某列'].rolling(window=window_size).mean()
df_clean['rolling_std'] = df_clean['某列'].rolling(window=window_size).std()

# 查看结果（例如尾部数据）
print(df_clean.tail())
```
这样，我们就得到了每个滑动窗口内的最小值、最大值、均值和标准差。

![](img/61520142f8a194309d00a55e79331a7f_45.png)

![](img/61520142f8a194309d00a55e79331a7f_47.png)

![](img/61520142f8a194309d00a55e79331a7f_49.png)

**关键点**：在进行时间窗口操作前，务必先处理数据中的缺失值，否则计算结果会出现大量NaN，影响分析。

![](img/61520142f8a194309d00a55e79331a7f_51.png)
