在本节课中，我们将学习如何对时间序列数据进行变化情况分析，包括计算每日差异值、增长率以及进行统计汇总和可视化。这些是金融时间序列分析中的基础且重要的操作。

上一节我们介绍了时间序列数据的基本处理，本节中我们来看看如何分析序列的变化情况。

## 计算每日差异值

![](img/50844c79f59cfc385cd47e167142855c_6.png)

![](img/50844c79f59cfc385cd47e167142855c_8.png)

首先，我们需要计算每一天指标相对于前一天的差异值。这个指标反映了数据的绝对变化量。

在Pandas中，我们可以使用 `diff()` 函数轻松实现这一计算。该函数会计算当前行与前一行的差值。

![](img/50844c79f59cfc385cd47e167142855c_10.png)

```python
# 计算每日差异值
diff_data = data.diff()
print(diff_data.head())
```

![](img/50844c79f59cfc385cd47e167142855c_12.png)

执行上述代码后，你会看到从第二天开始，每个值都等于当天数据减去前一天的数据。第一天的值会是 `NaN`，因为其前面没有数据可供相减。

这个差异值指标在时间序列分析中非常常用。

![](img/50844c79f59cfc385cd47e167142855c_14.png)

## 计算增长率

![](img/50844c79f59cfc385cd47e167142855c_16.png)

差异值反映的是绝对变化，但有时我们更关心相对变化，即增长率。例如，股价从1元涨到2元是100%的增长，而从10000元涨到10001元则增长甚微。

![](img/50844c79f59cfc385cd47e167142855c_18.png)

增长率的计算公式为：`(后一天的值 - 前一天的值) / 前一天的值`。

![](img/50844c79f59cfc385cd47e167142855c_20.png)

Pandas同样提供了现成的函数 `pct_change()` 来计算这个百分比变化。

```python
# 计算每日增长率
pct_data = data.pct_change()
print(pct_data.head())
```

![](img/50844c79f59cfc385cd47e167142855c_22.png)

计算完成后，数据将显示为百分比形式的增长率。通常金融资产的日增长率数值较小。

![](img/50844c79f59cfc385cd47e167142855c_24.png)

## 统计分析与可视化

![](img/50844c79f59cfc385cd47e167142855c_26.png)

![](img/50844c79f59cfc385cd47e167142855c_28.png)

在得到增长率序列后，我们可能希望了解其整体统计特性，例如平均增长率。

![](img/50844c79f59cfc385cd47e167142855c_30.png)

以下是计算平均增长率并绘制条形图的步骤：

![](img/50844c79f59cfc385cd47e167142855c_32.png)

![](img/50844c79f59cfc385cd47e167142855c_34.png)

1.  首先计算增长率的平均值。
2.  然后使用条形图进行可视化，因为平均值是单个汇总值，不适合用折线图表示。

```python
# 计算平均增长率
mean_pct = pct_data.mean()
print(mean_pct)

# 绘制平均增长率的条形图
mean_pct.plot(kind='bar', figsize=(16, 8))
```

![](img/50844c79f59cfc385cd47e167142855c_36.png)

![](img/50844c79f59cfc385cd47e167142855c_38.png)

通过图表，我们可以直观地比较不同指标在观察期内的平均增长表现。

![](img/50844c79f59cfc385cd47e167142855c_40.png)

![](img/50844c79f59cfc385cd47e167142855c_42.png)

本节课中我们一起学习了时间序列变化分析的核心操作：使用 `diff()` 计算差异值，使用 `pct_change()` 计算增长率，以及对结果进行统计汇总和可视化。这些技能是进行更深入金融数据分析的基础。
