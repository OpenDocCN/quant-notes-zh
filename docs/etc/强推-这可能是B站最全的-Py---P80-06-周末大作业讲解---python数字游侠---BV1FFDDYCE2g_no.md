## 概述
在本节课中，我们将详细讲解一个综合性的周末大作业。这个作业模拟了公司人力资源管理中常见的员工信息统计与分析任务，涵盖了从员工档案管理到薪资计算的多个环节。我们将学习如何使用Excel函数（特别是`VLOOKUP`）和数据分析工具来处理这些数据，并理解数据表之间的关联关系。

## 员工档案表分析

![](img/cdddcddcc7cacba20ed97a197117ba5c_11.png)

上一节我们介绍了课程概述，本节中我们来看看大作业的第一个核心部分：员工档案表。

员工档案表记录了员工的基本信息。在实际工作中，员工档案通常包含姓名、身份证号、性别、入职日期、职务和最高学历等。

以下是实现关键信息提取的步骤：

1.  **从身份证号提取出生日期**
    身份证号的第7位开始是出生年月日，共8位。我们可以使用`MID`函数提取。
    `=MID(身份证号单元格, 7, 8)`
    提取结果为文本格式的日期（如“19900209”），需要使用“数据”选项卡下的“分列”功能将其转换为标准日期格式。

2.  **计算年龄**
    年龄是当前日期减去出生日期的年份差。可以使用`DATEDIF`函数。
    `=DATEDIF(出生日期单元格, TODAY(), “Y”)`
    此函数返回两个日期之间的整年数。

3.  **计算工龄**
    工龄的计算逻辑与年龄相同，只是开始日期换成了入职日期。
    `=DATEDIF(入职日期单元格, TODAY(), “Y”)`
    若要计算不足一年的月份，可以先计算总月数差，再求除以12的余数。
    `=MOD(DATEDIF(入职日期单元格, TODAY(), “M”), 12)`

## 员工信息卡片制作

了解了如何计算基本信息后，我们常常需要快速查询某个员工的完整信息。这时，制作一个动态的信息卡片就非常有用。

![](img/cdddcddcc7cacba20ed97a197117ba5c_13.png)

![](img/cdddcddcc7cacba20ed97a197117ba5c_15.png)

![](img/cdddcddcc7cacba20ed97a197117ba5c_17.png)

卡片的核心功能是：根据输入的员工编号，自动匹配并显示该员工的所有档案信息。这主要通过`VLOOKUP`函数实现。

![](img/cdddcddcc7cacba20ed97a197117ba5c_19.png)

![](img/cdddcddcc7cacba20ed97a197117ba5c_21.png)

以下是创建动态信息卡片的步骤：

![](img/cdddcddcc7cacba20ed97a197117ba5c_23.png)

1.  **基础`VLOOKUP`匹配**
    例如，匹配姓名：
    `=VLOOKUP(员工编号单元格, 档案数据区域, 2, 0)`
    其中，“2”代表姓名在数据区域的第2列。

![](img/cdddcddcc7cacba20ed97a197117ba5c_25.png)

![](img/cdddcddcc7cacba20ed97a197117ba5c_27.png)

![](img/cdddcddcc7cacba20ed97a197117ba5c_29.png)

2.  **实现公式横向拖动**
    当我们需要向右拖动公式以匹配性别、部门等其他信息时，“2”需要依次变为3、4、5…。我们可以用`COLUMN`函数动态生成这个数字。
    `=VLOOKUP(员工编号单元格, 档案数据区域, COLUMN(B1), 0)`
    向右拖动时，`COLUMN(B1)`会依次变为`COLUMN(C1)`(3)、`COLUMN(D1)`(4)…。注意，查找区域需要用`$`锁定（如`$A$2:$K$25`），防止拖动时区域变化。

![](img/cdddcddcc7cacba20ed97a197117ba5c_31.png)

3.  **使用名称管理器简化公式**
    如果数据区域经常被引用，可以为其定义一个名称。
    *   选中数据区域。
    *   在左上角的名称框中输入一个名字（如“Info”），按回车。
    *   之后在公式中可以直接使用“Info”来代表这个区域，如`=VLOOKUP(员工编号单元格, Info, COLUMN(B1), 0)`。这使公式更简洁，且区域引用被固定。

![](img/cdddcddcc7cacba20ed97a197117ba5c_33.png)

![](img/cdddcddcc7cacba20ed97a197117ba5c_35.png)

## 员工培训成绩分析

除了静态信息，员工的动态表现如培训成绩也需要记录和分析。培训成绩表通常包含各科成绩、平均分和排名。

以下是进行成绩分析的关键计算：

![](img/cdddcddcc7cacba20ed97a197117ba5c_61.png)

1.  **计算平均分**
    使用`AVERAGE`函数计算所有科目的平均分。
    `=AVERAGE(成绩单元格区域)`

![](img/cdddcddcc7cacba20ed97a197117ba5c_63.png)

2.  **计算排名**
    使用`RANK`函数进行排名。
    `=RANK(当前员工平均分单元格, 所有员工平均分区域, 0)`
    参数“0”表示降序排列（分数越高排名越靠前）。`RANK`函数处理并列排名时为“跳跃”式（如1,2,2,4）。

3.  **统计各科成绩指标**
    可以对各科成绩进行统计分析，例如：
    *   **最高分**：`=MAX(英语成绩列)`
    *   **最低分**：`=MIN(英语成绩列)`
    *   **及格人数**：`=COUNTIF(英语成绩列, “>=60”)`
    *   **及格率**：`=及格人数单元格 / 总人数单元格`
    *   **平均分**：`=AVERAGE(英语成绩列)`
    将这些公式横向拖动，即可快速得到各科目的统计数据。

## 员工考勤与薪资计算

![](img/cdddcddcc7cacba20ed97a197117ba5c_67.png)

员工的考勤记录直接关系到薪资计算。考勤表通常记录事假、病假、迟到/早退等情况，并根据公司规则计算应扣款项。

![](img/cdddcddcc7cacba20ed97a197117ba5c_69.png)

![](img/cdddcddcc7cacba20ed97a197117ba5c_71.png)

以下是考勤扣款的计算示例：

![](img/cdddcddcc7cacba20ed97a197117ba5c_73.png)

假设规则：事假扣全天工资，病假扣当天工资的30%，迟到/早退每小时扣0.2元。日工资按（月基本工资/30）估算。

```
=IF(请假类型单元格=“事假”, (基本工资/30)*请假天数, IF(请假类型单元格=“病假”, (基本工资/30)*请假天数*0.3, 0)) + 迟到小时数*0.2
```
这个公式使用了`IF`函数的嵌套，先判断请假类型，再分别计算事假、病假的扣款，最后加上迟到扣款。

## 福利补贴与五险一金匹配

员工的福利补贴和五险一金缴纳通常与职务挂钩，可以通过查询表进行匹配计算。

1.  **福利补贴匹配**
    建立一个补贴标准表（行是职务，列是补贴类型）。在员工工资表中，使用`VLOOKUP`匹配。
    `=VLOOKUP(员工职务单元格, 补贴标准表区域, COLUMN(B$1), 0)`
    使用`COLUMN`函数实现横向拖动自动匹配不同补贴项。注意锁定行号（`B$1`），使拖动时行不变。

![](img/cdddcddcc7cacba20ed97a197117ba5c_75.png)

2.  **五险一金计算**
    建立一个比例表（保险类型 vs 个人/公司比例）。计算时分为两步：
    *   匹配比例：`=VLOOKUP(保险类型单元格, 比例表区域, 3, 0)` （假设个人比例在第3列）
    *   计算扣款：`=匹配的比例 * 员工基本工资单元格`
    公式组合为：`=VLOOKUP(保险类型单元格, 比例表区域, 3, 0) * 基本工资单元格`
    同样，使用`COLUMN`函数和`$`锁定技巧来实现公式的高效拖动填充。

## 薪资汇总与查询

![](img/cdddcddcc7cacba20ed97a197117ba5c_77.png)

最后，我们将所有数据汇总到员工工资总表，并制作一个薪资查询卡片。

1.  **薪资总表计算**
    应发工资 = 基本工资 - 考勤扣款 + 加班费 + 福利补贴 - 五险一金个人部分。
    其中，考勤扣款、加班费、福利补贴、五险一金等数据均通过`VLOOKUP`函数从前面各分表中根据员工编号匹配过来。

2.  **薪资查询卡片**
    这与员工信息卡片原理相同，但数据源是工资总表。
    `=VLOOKUP(查询的员工编号单元格, 工资总表数据区域, COLUMN(B$1), 0)`
    当需要纵向拖动以显示不同项目（如基本工资、应发工资）时，`COLUMN`函数不再适用，因为拖动方向是纵向，列号不变而行号变。这时应使用`ROW`函数。
    `=VLOOKUP(查询的员工编号单元格, 工资总表数据区域, ROW(A2), 0)`
    假设第一个匹配项（如姓名）在数据区域的第2列，公式中`ROW(A2)`返回2。向下拖动时，`ROW(A3)`返回3，即可匹配下一项（如基本工资）。注意锁定查询区域和员工编号单元格。

## 总结
本节课我们一起完成了一个综合性的人力资源数据分析大作业。我们系统性地练习了以下核心技能：
*   使用`MID`、`DATEDIF`、`TODAY`等函数处理日期信息。
*   深入掌握`VLOOKUP`函数，并结合`COLUMN`和`ROW`函数实现动态数据匹配，这是本作业最重要的知识点。
*   使用`IF`函数进行条件判断和嵌套计算。
*   使用`RANK`、`MAX`、`MIN`、`COUNTIF`、`AVERAGE`等函数进行统计分析。
*   利用数据透视表进行快速分组汇总。
*   通过名称管理器优化公式。
*   理解了多张数据表之间通过关键字段（如员工编号）进行关联、匹配和汇总的逻辑，这是构建复杂数据管理系统的基础。
