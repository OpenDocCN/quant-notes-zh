# Python量化交易入门：19：等待用户输入 📥

在本节课中，我们将学习如何在Python程序中等待并获取用户在控制台的输入。这个功能在需要与用户交互、获取动态参数（如股票代码或名称）的场景中非常有用。

## 核心概念：`input()`函数

上一节我们介绍了Python的基础语法，本节中我们来看看如何实现用户交互。Python提供了一个内置函数 `input()`，用于从标准输入（通常是控制台）读取一行文本。其基本语法是一个简单的函数调用。

**核心代码：**
```python
user_input = input("提示信息：")
```
当程序执行到 `input()` 函数时，它会暂停，在屏幕上显示括号内的提示信息，并等待用户输入内容。用户输入内容并按回车键后，输入的内容会以**字符串**的形式被赋值给等号左边的变量（例如 `user_input`）。

## 应用示例：获取股票信息

理解了 `input()` 函数的基本用法后，我们通过一个具体的例子来加深印象。以下是一个模拟获取股票代码或名称的程序。

**示例代码：**
```python
# 提示用户输入股票代码，并将输入值赋给变量 stock_code
stock_code = input("请输入股票代码：")
# 打印出用户输入的内容
print("您输入的股票代码是：", stock_code)

# 同样，可以提示输入股票名称
stock_name = input("请输入股票名称：")
print("您输入的股票名称是：", stock_name)
```

**运行过程说明：**
1.  程序首先执行 `input(“请输入股票代码：”)`，在控制台显示“请输入股票代码：”并等待。
2.  用户输入内容（例如 `600519`）后按回车。
3.  输入的内容 `“600519”` 被存入变量 `stock_code`。
4.  接着，`print` 函数将变量 `stock_code` 的值输出到屏幕。
5.  程序继续执行下一行，再次使用 `input()` 函数获取股票名称，流程同上。

例如，运行后控制台的交互可能如下：
```
请输入股票代码：600519
您输入的股票代码是： 600519
请输入股票名称：贵州茅台
您输入的股票名称是： 贵州茅台
```

## 重要注意事项

![](img/6ee9b3247dda8be9bf09a9006b2de43e_1.png)

以下是使用 `input()` 函数时需要了解的几个关键点：

*   **返回值类型**：`input()` 函数**总是返回一个字符串（str）**，即使用户输入的是数字。如果后续需要将其作为数字进行计算，需要使用 `int()` 或 `float()` 等函数进行类型转换。
*   **程序暂停**：调用 `input()` 会使程序暂停执行，直到用户完成输入。这是实现交互的基础。
*   **提示信息**：括号内的提示字符串是可选的，但为了用户体验，通常应该提供清晰的提示。

## 总结

本节课中我们一起学习了Python中用于接收用户输入的内置函数 `input()`。我们掌握了它的基本语法 `variable = input(“提示信息”)`，并通过一个获取股票信息的例子实践了其用法。记住，`input()` 的返回值是字符串类型，这在处理不同类型的数据时至关重要。这个简单的函数是构建交互式程序（如命令行工具、数据查询接口等）的重要基石。
