## 第一步：选择日期数据 📅

首先，我们需要确定分析的时间范围，并获取该时间段内所有的交易日列表。因为后续查询数据时，需要传入具体的日期参数。

![](img/1890a0c4f03f94f18365e45846880ed2_6.png)

以下是获取交易日列表的代码：

```python
def get_trading_days(start_date='2019-01-01', end_date='2020-01-01'):
    # 此函数用于获取指定起止日期之间的所有交易日
    trading_days = get_trading_dates(start_date, end_date)
    return trading_days
```

## 第二步：获取因子指标 🔍

![](img/1890a0c4f03f94f18365e45846880ed2_11.png)

![](img/1890a0c4f03f94f18365e45846880ed2_12.png)

有了日期列表后，下一步是获取我们感兴趣的因子指标数据。这里我们以市盈率（PE）为例，查询它在所有A股股票中的数值。

以下是获取因子指标的步骤：

![](img/1890a0c4f03f94f18365e45846880ed2_14.png)

![](img/1890a0c4f03f94f18365e45846880ed2_15.png)

![](img/1890a0c4f03f94f18365e45846880ed2_16.png)

1.  **定义股票池**：首先，我们需要确定要分析哪些股票。这里我们选择获取所有A股股票。

    ```python
    # 获取所有A股股票代码，作为我们的股票池
    stock_pool = get_all_securities(['stock']).index.tolist()
    ```

2.  **构建查询语句**：接下来，我们需要构建一个查询语句（Query），来指定我们要获取的因子（PE）和过滤条件（股票池）。

    ```python
    # 构建查询语句，查询市盈率(pe_ratio)指标
    q = query(
        fundamentals.eod_derivative_indicator.pe_ratio
    ).filter(
        fundamentals.eod_derivative_indicator.code.in_(stock_pool)
    )
    ```

    这个查询语句的意思是：从基本面数据中查询市盈率（`pe_ratio`），并且只筛选出股票代码在我们定义的`stock_pool`中的股票。

![](img/1890a0c4f03f94f18365e45846880ed2_18.png)

![](img/1890a0c4f03f94f18365e45846880ed2_20.png)

3.  **执行查询获取数据**：最后，我们使用`get_fundamentals`函数，传入查询语句和具体的查询日期，来执行查询并获取数据。

    ```python
    # 执行查询，获取指定日期（例如交易日列表中的第0天）的数据
    specified_date = trading_days[0]
    factor_data = get_fundamentals(q, date=specified_date)
    ```

    执行后，`factor_data`就是一个包含指定日期所有股票PE值的数据表。
