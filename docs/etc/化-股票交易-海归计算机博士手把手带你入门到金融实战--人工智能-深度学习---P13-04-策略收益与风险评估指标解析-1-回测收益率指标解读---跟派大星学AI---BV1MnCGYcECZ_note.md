![](img/b104716a8440097d10abecdba2d6b8a2_1.png)

当我们完成一个交易策略的构建后，必须在历史数据中进行测试，以判断该策略是盈利还是亏损。然而，仅用“盈利”或“亏损”来描述是不够的。我们需要更精确的指标来回答一系列问题：具体赚了多少？与市场整体表现（如大盘指数）相比有何优势？执行该策略会面临哪些风险？这些风险是否值得承担？以及我们的策略与其他策略相比有何优劣？这些问题的答案，就构成了策略评估指标。

以下是本节课将要介绍的第一个，也是最重要的评估指标。

## 回测收益率

![](img/b104716a8440097d10abecdba2d6b8a2_5.png)

![](img/b104716a8440097d10abecdba2d6b8a2_7.png)

回测收益率是最基本、最常用的策略评估指标。它衡量的是，在回测周期结束后，你的策略相对于初始本金获得了多少百分比的总收益。

![](img/b104716a8440097d10abecdba2d6b8a2_9.png)

例如，你使用2015年至2020年的某只股票数据，应用一个“何时买入、何时卖出”的策略，最终会得到一个结果。这个结果不是赚取的绝对金额，而是相对于初始投入本金的收益率。如果报告显示收益率为4.23%，就意味着经过这些年的操作，你的总资产比最初投入的本金增长了约4.23%。

![](img/b104716a8440097d10abecdba2d6b8a2_11.png)

其计算公式如下：

![](img/b104716a8440097d10abecdba2d6b8a2_13.png)

**回测收益率 = (期末资产总值 - 期初资产总值) / 期初资产总值**

![](img/b104716a8440097d10abecdba2d6b8a2_15.png)

![](img/b104716a8440097d10abecdba2d6b8a2_17.png)

![](img/b104716a8440097d10abecdba2d6b8a2_19.png)

也可以写作：

![](img/b104716a8440097d10abecdba2d6b8a2_21.png)

**回测收益率 = (P_t / P_0) - 1**

其中：
*   **P_t** 代表回测结束时的资产总值（或卖出价格）。
*   **P_0** 代表回测开始时的初始本金（或买入价格）。

![](img/b104716a8440097d10abecdba2d6b8a2_23.png)

这个公式的本质是计算资产增长的相对比例。

## 用Python计算与可视化回测收益率

![](img/b104716a8440097d10abecdba2d6b8a2_25.png)

![](img/b104716a8440097d10abecdba2d6b8a2_27.png)

上一节我们介绍了回测收益率的概念和公式，本节中我们来看看如何用Python代码实际计算并可视化它。我们将使用一份示例股票数据来模拟策略回测后的每日资产变化。

首先，导入必要的工具包并加载数据。

```python
import pandas as pd
import matplotlib.pyplot as plt
# 设置中文字体，方便后续显示
plt.rcParams['font.sans-serif'] = ['SimHei']
plt.rcParams['axes.unicode_minus'] = False

# 读取数据，假设第一列是日期
data = pd.read_csv('your_stock_data.csv', index_col=0, parse_dates=True)
# 查看数据前几行
print(data.head())
```

假设`data[‘Close’]`列代表我们策略回测后每天的资产净值。接下来，我们计算相对于第一天（期初）的每日资产比值。

![](img/b104716a8440097d10abecdba2d6b8a2_29.png)

```python
# 计算每日资产相对于期初的比值
cumulative_return = data['Close'] / data['Close'].iloc[0]

![](img/b104716a8440097d10abecdba2d6b8a2_31.png)

![](img/b104716a8440097d10abecdba2d6b8a2_33.png)

# 绘制资产变化曲线
plt.figure(figsize=(10, 6))
cumulative_return.plot()
plt.title('策略资产相对变化曲线')
plt.ylabel('资产比值 (相对于期初=1)')
plt.xlabel('日期')
plt.grid(True)
plt.show()
```

![](img/b104716a8440097d10abecdba2d6b8a2_35.png)

这张图展示了从一开始（资产比值为1）到结束，你的策略资产是如何变化的。比值大于1表示盈利，小于1表示亏损。

![](img/b104716a8440097d10abecdba2d6b8a2_37.png)

![](img/b104716a8440097d10abecdba2d6b8a2_39.png)

最后，根据公式计算整个回测期间的总收益率。

![](img/b104716a8440097d10abecdba2d6b8a2_41.png)

```python
# 计算总回测收益率：(期末值/期初值) - 1
total_return = cumulative_return.iloc[-1] - 1

![](img/b104716a8440097d10abecdba2d6b8a2_43.png)

![](img/b104716a8440097d10abecdba2d6b8a2_45.png)

# 将结果整理成DataFrame以便清晰展示
result_df = pd.DataFrame({
    ‘回测收益率’: [total_return]
}, index=[‘策略总收益’])
print(result_df)
```

![](img/b104716a8440097d10abecdba2d6b8a2_47.png)

执行以上代码，我们就能得到该策略在回测期间的总收益率。这个值可能是正数（盈利），也可能是负数（亏损）。在本例中，我们只是用股价模拟了资产走势，你可以将其想象为应用了某个交易策略后的每日资产结果。

![](img/b104716a8440097d10abecdba2d6b8a2_49.png)

本节课中我们一起学习了量化策略评估的基石——回测收益率。我们理解了它的定义是衡量策略相对于初始本金的总收益百分比，掌握了其核心计算公式 **`(P_t / P_0) - 1`**，并亲手使用Python计算和可视化了这一指标。理解回测收益率是读懂任何策略报告、评判策略盈利能力的首要步骤。在接下来的课程中，我们将继续深入其他重要的风险评估与收益指标。
