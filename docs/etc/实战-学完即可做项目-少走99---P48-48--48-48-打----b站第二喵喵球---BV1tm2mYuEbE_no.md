# 量化交易入门：P48：因子打分法选股策略概述

在本节课中，我们将要学习一种在量化选股中常用的策略——因子打分法。我们将了解其核心思想、实施前提，并简要介绍后续的实践步骤。

## 策略核心思想

上一节我们介绍了如何通过单一因子（如IC值）来评估股票。本节中我们来看看如何综合多个因子进行更全面的评估。

因子打分选股策略的核心思想是：**为每只股票的多个因子分别打分，然后计算总分，最后根据总分排名来筛选股票**。

![](img/1608480a556ff96636bdd5fa624a32e7_1.png)

这类似于学生的总成绩由各科成绩汇总决定。假设我们有300只股票（`stock_1`, `stock_2`, ..., `stock_300`）和四个因子（`factor_A`, `factor_B`, `factor_C`, `factor_D`）。我们的目标是计算每只股票的综合得分 `total_score`：

```
total_score_i = score(factor_A_i) + score(factor_B_i) + score(factor_C_i) + score(factor_D_i)
```

计算出所有股票的 `total_score` 后，进行排序，选取排名靠前的股票（如前10名）作为投资组合。在后续的调仓周期（如每月一次）中，重复此过程以更新投资组合。

![](img/1608480a556ff96636bdd5fa624a32e7_3.png)

## 策略实施前提

![](img/1608480a556ff96636bdd5fa624a32e7_5.png)

在开始实施打分策略前，我们必须明确一个关键的先验知识。

![](img/1608480a556ff96636bdd5fa624a32e7_7.png)

以下是实施因子打分法必须预先确定的条件：

*   **因子方向性**：我们必须事先知道每个因子与预期收益率的相关性是正向还是负向。即，对于每个因子，我们需要明确是因子值**越大越好**，还是**越小越好**。
*   **知识来源**：这个方向性判断可以来源于券商的研究报告、金融理论，或者我们自己通过历史数据计算因子与收益率的相关系数（如我们在因子分析课中所做的那样）来验证。
*   **策略基础**：这个方向性是打分的基础。例如，对于“每股收益”因子，我们通常认为越大越好，因此打分时数值大的股票应获得更高分数；而对于“市盈率”因子，在某些策略中可能认为越小越好，打分规则则相反。
