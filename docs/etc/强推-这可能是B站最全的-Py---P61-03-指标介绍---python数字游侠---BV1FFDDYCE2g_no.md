在本节课中，我们将要学习业务分析中至关重要的概念——指标。我们将通过一个具体的工作场景，理解什么是指标，如何选择指标，并以电商数据为例介绍常用指标。

## 什么是业务指标？

上一节我们通过一个离职率分析的例子，初步接触了“指标”的概念。本节中，我们来看看指标的精确定义。简单来说，业务指标是用于衡量、评估和追踪业务表现的可量化数据点。它是将复杂的业务状况转化为具体数字的工具。

## 电商常用指标示例

理解了指标的定义后，我们以电商行业为例，看看在实际业务中会用到哪些常见指标。以下是电商数据分析中几个核心的衡量指标：

*   **销售额**：指在一定时间段内，通过销售商品或服务所获得的总收入。其基本公式为：
    `销售额 = 销售单价 × 销售数量`
*   **转化率**：衡量访客转化为购买顾客的效率。例如，网站转化率的计算方式为：
    `转化率 = (完成购买的用户数 / 网站总访客数) × 100%`
*   **客单价**：指平均每个顾客在一次交易中支付的金额。计算公式为：
    `客单价 = 总销售额 / 总订单数`
*   **复购率**：反映顾客忠诚度，指在一定时期内，有两次或以上购买行为的顾客占总顾客数的比例。
*   **用户留存率**：衡量产品留住用户的能力。例如，七日留存率指新用户在首次使用产品后，第七天仍活跃的比例。

## 如何选择业务指标？

我们已经学习了多种指标，但在实际工作中，面对众多指标，我们该如何选择呢？选择业务指标需要遵循以下原则：

![](img/cf970c8fbbf1a00dcc1ddc43ad1def07_17.png)

1.  **与业务目标对齐**：选择的指标必须直接服务于当前的核心业务目标。例如，若目标是提升市场份额，则应重点关注市场占有率、新客获取成本等指标。
2.  **可衡量与可获取**：指标必须是能够被准确量化的，并且其相关数据是可以通过现有系统或渠道稳定获取的。
3.  **可行动化**：好的指标应能指引具体的改进行动。例如，“用户流失率”过高这个指标，可以引导我们去分析流失原因并制定挽留策略。
4.  **避免虚荣指标**：警惕那些看起来很美但对决策没有实际指导意义的指标，如单纯的“总注册用户数”（可能包含大量僵尸用户），不如“月活跃用户数”有价值。
