## NumPy模块简介 📦

NumPy模块是我们学习的重点之一，但需要明确的是，在整个课程中，**pandas模块才是核心操作的重点**。NumPy和matplotlib虽然也会用到，但其重要性相对低于pandas。NumPy模块的主要作用之一是为pandas模块提供数据源。

![](img/d56fdbe3124586950c826a690b2299f5_13.png)

首先对NumPy模块进行一个大致的介绍。

你可以将NumPy理解为一个**一维或多维的数组**。如果你有其他编程语言（如C、Java）的经验，应该对数组不陌生。如果只接触过Python，可以将数组暂时理解为一个特殊的列表，或者将列表理解为一个特殊的数组。无论是列表还是数组，它们都是一种**数据结构**，即一种可以装载数据的**容器**。

因此，NumPy模块对应的容器就是**数组**，它可以装载相关数据。暂时我们将NumPy模块视为一个一维或多维的数组。

关于维度：
*   **一维数组**：类似于一个简单的列表，例如 `[1, 2, 3]`。
*   **二维数组**：类似于“列表套列表”，例如 `[[1, 2, 3], [4, 5, 6]]`，可以想象成一个表格，有行和列。
*   更高维度的数组原理类似，但实际应用中，一维、二维和三维数组最为常见。

数组与列表是有区别的，具体区别稍后会介绍。目前，只需知道NumPy数组主要装载**数值型数据**。

NumPy模块的重点在于**数值计算**。如果后期需要对一组数值进行某种运算，可以先将待计算的数值装载到NumPy数组中，再利用NumPy数组的特性进行运算。

![](img/d56fdbe3124586950c826a690b2299f5_15.png)

此外，NumPy模块是**Python科学计算库的基础**。这意味着NumPy内部已经封装好了各种科学计算所需的工具类。例如，若需要对一组数据进行反三角函数运算，无需手动实现，直接调用NumPy中封装好的工具类即可。

![](img/d56fdbe3124586950c826a690b2299f5_17.png)

总结NumPy模块的两大重点：
1.  它是一个容器，主要装载**数值型数据**并进行运算。
2.  它是Python科学计算的基础，提供了丰富的科学计算工具类。

## 创建NumPy数组（第一种方式）

介绍完NumPy模块后，接下来我们看看如何使用它。首先需要创建NumPy数组。

使用NumPy前需要先导入模块。业界约定俗成地为其起别名 `np`。

```python
import numpy as np
```

NumPy模块通常无需单独安装，安装好Anaconda集成环境后即已包含。

导入模块后，即可创建数组。以下是创建数组的第一种方式。

![](img/d56fdbe3124586950c826a690b2299f5_19.png)

**使用 `np.array()` 函数创建数组**

![](img/d56fdbe3124586950c826a690b2299f5_21.png)

`np.array()` 函数用于创建数组。其核心参数如下：
*   `object`：数据源，即要装入数组的数据。
*   `dtype`：数组元素的数据类型。如果不指定，则根据输入数据自动推断。

**创建一维数组：**
```python
arr1 = np.array([1, 2, 3])
print(arr1)
```
输出类似于：`array([1, 2, 3])`，表示一个一维数组。

![](img/d56fdbe3124586950c826a690b2299f5_23.png)

![](img/d56fdbe3124586950c826a690b2299f5_25.png)

**创建二维数组：**
```python
arr2 = np.array([[1, 2, 3], [4, 5, 6]])
print(arr2)
```
输出类似于：
```
array([[1, 2, 3],
       [4, 5, 6]])
```
这表示一个2行3列的二维数组（一个平面）。

![](img/d56fdbe3124586950c826a690b2299f5_27.png)

同样，可以创建更高维度的数组，但最常用的是至三维数组。

![](img/d56fdbe3124586950c826a690b2299f5_29.png)

## 数组与列表的核心区别

我们已经掌握了一种创建数组的方式。现在来看一个关键问题：NumPy数组与Python列表的核心区别是什么？

我们通过一个例子来揭示其区别：
```python
arr_mixed = np.array([1, 2.2, ‘three‘])
print(arr_mixed)
print(arr_mixed.dtype) # 查看数组元素的数据类型
```
你会发现，输出结果中所有元素都被转换成了**字符串类型**（如 `[‘1‘ ‘2.2‘ ‘three‘]`）。

![](img/d56fdbe3124586950c826a690b2299f5_31.png)

而在Python列表中，元素类型可以各不相同：
```python
list_mixed = [1, 2.2, ‘three‘] # 列表允许不同类型
```

![](img/d56fdbe3124586950c826a690b2299f5_33.png)

![](img/d56fdbe3124586950c826a690b2299f5_35.png)

**区别在于：NumPy数组要求所有元素的`数据类型必须一致`。** 如果创建时传入的数据类型不一致，NumPy会尝试将它们统一转换为同一个类型。转换的“优先级”通常是：字符串 > 浮点数 > 整数。

这个特性是NumPy能够高效进行数值计算的基础之一。

## 创建NumPy数组（第二种方式）

接下来我们看看NumPy数组的第二种创建方式，并通过一个实例理解“容器”的意义。

![](img/d56fdbe3124586950c826a690b2299f5_37.png)

![](img/d56fdbe3124586950c826a690b2299f5_39.png)

数据装载在不同的容器（如列表、字典、数组）中，就会拥有该容器的特性，从而能进行特定形式的操作。NumPy数组的特性使其能方便地对承载的数值数据进行批量运算。

**需求：将外部图片读入NumPy数组，通过修改数组值来改变图片。**

图片本质上是由二进制数据（数值型数据）构成的，因此可以装入NumPy数组。这里需要用到“三剑客”中的另一个模块 `matplotlib.pyplot` 进行图像读取和显示。

```python
import matplotlib.pyplot as plt

![](img/d56fdbe3124586950c826a690b2299f5_41.png)

![](img/d56fdbe3124586950c826a690b2299f5_43.png)

# 读取图片，数据将存储在NumPy数组中
image_array = plt.imread(‘1.jpg‘)
print(image_array) # 查看数组，通常是一个三维数组（高度，宽度，颜色通道）
print(image_array.shape) # 查看数组形状

![](img/d56fdbe3124586950c826a690b2299f5_45.png)

# 显示原始图片
plt.imshow(image_array)
plt.show()
```
`plt.imread()` 读取图片后返回一个NumPy数组（通常是三维数组）。`plt.imshow()` 可以将这个包含图像数据的数组可视化显示出来。

![](img/d56fdbe3124586950c826a690b2299f5_47.png)

现在，我们利用NumPy数组的特性，对图片数据进行运算：
```python
# 对数组中的每个元素减去一个值（相当于调整亮度）
modified_array = image_array - 50

![](img/d56fdbe3124586950c826a690b2299f5_49.png)

![](img/d56fdbe3124586950c826a690b2299f5_51.png)

# 显示处理后的图片
plt.imshow(modified_array)
plt.show()
```
你会发现，显示出来的图片变暗了。这是因为我们减少了代表像素亮度的数值。

这个例子揭示了图像处理软件的底层原理：**对图像数据（NumPy数组）进行特定的数学运算**，即可实现如调色、滤镜、瘦脸等效果。

![](img/d56fdbe3124586950c826a690b2299f5_53.png)

![](img/d56fdbe3124586950c826a690b2299f5_55.png)

第二种创建数组的方式总结为：**通过 `plt.imread()` 等函数从外部文件（如图片）读取数据，并返回包含该数据的NumPy数组。**

## 创建NumPy数组（第三种方式）

![](img/d56fdbe3124586950c826a690b2299f5_61.png)

![](img/d56fdbe3124586950c826a690b2299f5_63.png)

最后，我们学习第三种创建数组的方式：使用NumPy内置的多种便捷函数来生成具有特定规律或内容的数组。

![](img/d56fdbe3124586950c826a690b2299f5_65.png)

以下是几个常用函数：

![](img/d56fdbe3124586950c826a690b2299f5_67.png)

![](img/d56fdbe3124586950c826a690b2299f5_69.png)

**1. `np.zeros()` 和 `np.ones()`**
*   创建指定形状、元素全为0或全为1的数组。
*   `shape` 参数指定数组形状。

![](img/d56fdbe3124586950c826a690b2299f5_71.png)

![](img/d56fdbe3124586950c826a690b2299f5_73.png)

```python
# 创建3行4列，元素全为0的二维数组
arr_zeros = np.zeros((3, 4))
print(arr_zeros)

# 创建2行3列，元素全为1的二维数组
arr_ones = np.ones((2, 3))
print(arr_ones)
```

![](img/d56fdbe3124586950c826a690b2299f5_75.png)

**2. `np.linspace()`**
*   创建**一维**的等差数列数组。
*   `start`：序列起始值。
*   `stop`：序列结束值。
*   `num`：要生成的样本数。

![](img/d56fdbe3124586950c826a690b2299f5_77.png)

```python
# 在0到100之间生成20个等间隔的数
arr_lin = np.linspace(0, 100, 20)
print(arr_lin)
```

![](img/d56fdbe3124586950c826a690b2299f5_79.png)

**3. `np.arange()`**
*   类似于Python的 `range()`，但返回的是NumPy数组。
*   创建**一维**的等差数列数组。
*   `start`：起始值。
*   `stop`：结束值（不包含）。
*   `step`：步长。

![](img/d56fdbe3124586950c826a690b2299f5_81.png)

```python
# 从10开始，到50结束（不包含50），步长为2
arr_arange = np.arange(10, 50, 2)
print(arr_arange)
```

![](img/d56fdbe3124586950c826a690b2299f5_83.png)

**4. `np.random.randint()`**
*   创建指定形状的随机整数数组。
*   `low`：随机整数下限。
*   `high`：随机整数上限。
*   `size`：数组形状。

![](img/d56fdbe3124586950c826a690b2299f5_85.png)

```python
# 创建一个5行3列，元素为0到100之间随机整数的数组
arr_rand = np.random.randint(0, 100, size=(5, 3))
print(arr_rand)
```

**这些函数的特点：**
*   `np.zeros()` 和 `np.ones()` 可以通过 `shape` 参数创建任意维度的数组。
*   `np.linspace()` 和 `np.arange()` 主要用于创建**一维**数组。
*   `np.random` 模块下的函数用于创建随机数数组。

![](img/d56fdbe3124586950c826a690b2299f5_87.png)
