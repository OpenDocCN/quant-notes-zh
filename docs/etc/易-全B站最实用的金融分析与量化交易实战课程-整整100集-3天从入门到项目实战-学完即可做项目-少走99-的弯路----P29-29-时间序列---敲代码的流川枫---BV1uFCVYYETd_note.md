# 量化交易：P29：时间序列 📅

在本节课中，我们将要学习如何利用Pandas创建和使用时间序列。时间序列是以时间对象作为索引的Series或DataFrame，它在金融数据分析中至关重要。

![](img/8daf5e1c568ec517b7a8821c13740841_1.png)

## 时间序列的创建

上一节我们介绍了Pandas中生成时间对象的函数。生成了这些时间对象之后，我们可以用它来构成一个时间序列。

我们可以使用`pd.date_range`函数生成一个`DatetimeIndex`，并将其作为Series或DataFrame的索引。

以下是创建一个时间序列Series的示例代码：

```python
import pandas as pd
import numpy as np

![](img/8daf5e1c568ec517b7a8821c13740841_3.png)

# 创建一个长度为1000的数值数组作为数据
data = np.arange(1000)

![](img/8daf5e1c568ec517b7a8821c13740841_5.png)

# 创建一个从2017-01-01开始，长度为1000天的DatetimeIndex作为索引
index = pd.date_range('2017-01-01', periods=1000)

![](img/8daf5e1c568ec517b7a8821c13740841_7.png)

# 创建时间序列Series
sr = pd.Series(data, index=index)
```

![](img/8daf5e1c568ec517b7a8821c13740841_9.png)

现在，`sr`就是一个时间序列。它的索引类型是`DatetimeIndex`，而不是普通的字符串。

![](img/8daf5e1c568ec517b7a8821c13740841_11.png)

## 时间序列的切片操作

成为了一个时间序列之后，我们可以非常方便地按时间范围选取数据。

以下是几种切片方式：

*   **按年月切片**：例如，`sr[‘2017-03’]` 会选取2017年3月的所有数据。
*   **按年切片**：例如，`sr[‘2017’]` 会选取2017年的所有数据。
*   **按日期范围切片**：例如，`sr[‘2017-12-25’:’2018-02-01’]` 会选取从2017年12月25日到2018年2月1日（包含）的数据。

这些操作允许我们使用不完整的日期字符串（如年、年月）或完整的日期范围来高效地筛选数据。

## 重采样函数

时间序列还有一个强大的功能是重采样（Resample）。`resample`函数可以按照新的时间频率（如每周、每月）对数据进行聚合。

以下是`resample`函数的使用示例：

```python
# 按周（‘W’）对数据进行求和
weekly_sum = sr.resample('W').sum()

# 按月（‘M’）对数据进行求和
monthly_sum = sr.resample('M').sum()

# 按月（‘M’）对数据求平均值
monthly_mean = sr.resample('M').mean()
```

例如，按周求和后，结果中的每个日期（如1月8日）代表从1月1日到1月7日这一周的数据总和。这个功能对于分析周期性的数据趋势（如周销售额、月均流量）非常方便。

## 其他函数

![](img/8daf5e1c568ec517b7a8821c13740841_13.png)

除了切片和重采样，时间序列还有其他函数，例如`truncate`。

`truncate`函数可以截断指定日期之前或之后的数据。

![](img/8daf5e1c568ec517b7a8821c13740841_15.png)

以下是`truncate`函数的示例：

```python
# 截取2018年2月3日之后的数据
sr_after = sr.truncate(before='2018-02-03')

# 截取2018年2月3日之前的数据
sr_before = sr.truncate(after='2018-02-03')
```

![](img/8daf5e1c568ec517b7a8821c13740841_17.png)

不过，这个功能通常可以直接用更直观的切片操作来实现，例如`sr[‘2018-02-03’:]`。
