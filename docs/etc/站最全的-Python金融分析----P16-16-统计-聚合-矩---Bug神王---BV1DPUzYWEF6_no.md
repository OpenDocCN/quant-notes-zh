## 数组形状变换（Reshape）

上一节我们介绍了数组的索引与切片，本节中我们来看看如何改变数组的形状。变形操作指的是在不改变数组数据的前提下，重新排列数组的维度。

例如，我们有一个原始的二维数组 `arr`，其形状为五行六列。

```python
import numpy as np
arr = np.random.rand(5, 6)  # 创建一个5行6列的二维数组
print(arr.shape)  # 输出: (5, 6)
```

我们可以使用 `reshape` 方法将其变形。变形后的新数组必须能容纳原数组的所有元素。

以下是变形操作的要点：
*   将二维数组变形为一维数组：新的一维数组必须有30个位置（5*6=30）。
*   将一维数组变形为多维数组：只要新形状的元素总数与原数组一致即可，例如可以变形为2行15列或6行5列。

```python
# 二维变一维
arr_1d = arr.reshape(30)
print(arr_1d.shape)  # 输出: (30,)

# 一维变多维
arr_new = arr_1d.reshape(2, 15)
print(arr_new.shape)  # 输出: (2, 15)
```

## 数组拼接（Concatenate）

接下来，我们学习如何将多个数组拼接在一起。拼接操作可以将数组在横向（按行）或纵向（按列）进行连接。

我们使用 `np.concatenate` 函数进行拼接。参数 `axis` 用于指定拼接的方向。

以下是拼接操作的说明：
*   `axis=0`：沿第一个轴（纵向，列方向）拼接。
*   `axis=1`：沿第二个轴（横向，行方向）拼接。
*   只能对**维度相同**的数组进行拼接。
*   如果两个数组维度相同但行列数不一致，拼接可能会失败，需要自行尝试验证。

```python
# 创建两个相同的二维数组进行拼接
arr1 = np.array([[1, 2], [3, 4]])
arr2 = np.array([[5, 6], [7, 8]])

# 纵向拼接（按列）
result_v = np.concatenate((arr1, arr2), axis=0)
print(result_v)
# 输出:
# [[1 2]
#  [3 4]
#  [5 6]
#  [7 8]]

# 横向拼接（按行）
result_h = np.concatenate((arr1, arr2), axis=1)
print(result_h)
# 输出:
# [[1 2 5 6]
#  [3 4 7 8]]
```

拼接操作的一个典型应用是图像处理，例如将多张图片拼接成三宫格或九宫格。

## 聚合操作（Aggregation）

聚合操作是指对数组中的一组值执行计算，返回一个单一的汇总值。这是数据分析中最常用的操作之一。

NumPy提供了多种聚合函数，最基础的是 `sum`（求和）、`max`（最大值）、`min`（最小值）和 `mean`（平均值）。通过指定 `axis` 参数，可以计算特定维度上的聚合值。

以下是聚合函数的用法：
*   `arr.sum()`：计算所有元素的和。
*   `arr.sum(axis=0)`：计算每一列的和。
*   `arr.sum(axis=1)`：计算每一行的和。
*   `arr.max()`, `arr.min()`, `arr.mean()` 用法类似。

```python
arr = np.array([[1, 2, 3], [4, 5, 6]])
print(arr.sum())          # 输出: 21
print(arr.sum(axis=0))    # 输出: [5 7 9] (列和)
print(arr.max(axis=1))    # 输出: [3 6] (每行最大值)
```

## 常用数学与统计函数

除了基础聚合，NumPy还提供了丰富的数学和统计函数。

**数学函数**，如 `np.sin`（正弦）、`np.cos`（余弦）、`np.tan`（正切），会对数组中的每个元素进行计算。`np.around` 函数用于四舍五入，可以通过 `decimals` 参数指定保留的小数位数。

```python
angles = np.array([0, 30, 45, 60, 90]) * np.pi / 180
print(np.sin(angles))  # 计算正弦值

print(np.around(3.14159, decimals=2))  # 输出: 3.14
print(np.around(314.159, decimals=-2)) # 输出: 300.0 (整数位四舍五入)
```

**统计函数**中，**方差（Variance）** 和 **标准差（Standard Deviation）** 尤为重要。它们用于衡量一组数据的离散程度。
*   **方差**：各个数据与平均数之差的平方值的平均数。公式为：$\sigma^2 = \frac{1}{N}\sum_{i=1}^{N}(x_i - \mu)^2$
*   **标准差**：方差的算术平方根。公式为：$\sigma = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(x_i - \mu)^2}$

```python
data = np.array([1, 2, 3, 4, 5, 100])
print(np.var(data))   # 计算方差
print(np.std(data))   # 计算标准差
```

标准差大，说明数据波动大，分布分散；标准差小，说明数据都集中在平均值附近。

## 矩阵运算

![](img/73085620b69f6f0bf24e16162f4351da_1.png)

最后，我们学习矩阵运算。在NumPy中，二维数组可以视作矩阵。需要重点掌握的是**矩阵的乘法**，这在机器学习和线性代数中应用广泛。

首先了解两个概念：
*   **单位矩阵**：主对角线元素为1，其余元素为0的方阵。可用 `np.eye(n)` 创建。
*   **转置**：将矩阵的行列互换。可通过数组的 `.T` 属性实现。

```python
# 创建单位矩阵
I = np.eye(3)
print(I)
# 输出:
# [[1. 0. 0.]
#  [0. 1. 0.]
#  [0. 0. 1.]]

![](img/73085620b69f6f0bf24e16162f4351da_3.png)

# 矩阵转置
A = np.array([[1, 2], [3, 4]])
print(A.T)
# 输出:
# [[1 3]
#  [2 4]]
```

**矩阵乘法**不是简单的元素对应相乘，而是行与列的点积。规则是：第一个矩阵的**行**与第二个矩阵的**列**对应元素相乘后求和。
计算 $C = A \times B$，其中 $C_{ij} = \sum_{k} A_{ik} \times B_{kj}$。

在NumPy中，使用 `np.dot(A, B)` 或 `A @ B` 进行矩阵乘法。

![](img/73085620b69f6f0bf24e16162f4351da_5.png)

```python
A = np.array([[2, 1], [4, 3]])
B = np.array([[1, 2], [1, 0]])

C = np.dot(A, B)
# 或 C = A @ B
print(C)
# 输出:
# [[3 4]
#  [7 8]]
```
计算过程解析：
*   C[0,0] = 2*1 + 1*1 = 3 (A第一行 点乘 B第一列)
*   C[0,1] = 2*2 + 1*0 = 4 (A第一行 点乘 B第二列)
*   C[1,0] = 4*1 + 3*1 = 7 (A第二行 点乘 B第一列)
*   C[1,1] = 4*2 + 3*0 = 8 (A第二行 点乘 B第二列)
