**1. 取绝对值**
使用 `np.abs()` 函数可以对数组中的所有元素取绝对值。
```python
import numpy as np
a = np.array([-5, -3, 0, 3, 5])
result = np.abs(a)  # 结果为 [5 3 0 3 5]
```
注意：虽然Python内置的 `abs()` 函数有时也能对NumPy数组生效，但为了代码清晰和兼容性，建议使用 `np.abs()`。

**2. 开方运算**
使用 `np.sqrt()` 函数可以对数组中的每个元素进行开方运算。
```python
a = np.array([4, 9, 16])
result = np.sqrt(a)  # 结果为 [2. 3. 4.]
```
需要注意的是，对负数开方会产生 `NaN`（非数字）值。

### 四种取整方式
在处理小数时，有四种不同的取整方式。理解它们对后续的数据处理很重要。

以下是四种取整方式的对比：
*   **向零取整 (`np.trunc`)**: 直接去掉小数部分。例如，1.6 -> 1， -1.6 -> -1。
*   **向下取整 (`np.floor`)**: 取不大于原数的最大整数。例如，1.6 -> 1， -1.6 -> -2。
*   **向上取整 (`np.ceil`)**: 取不小于原数的最小整数。例如，1.6 -> 2， -1.6 -> -1。
*   **四舍五入 (`np.round`)**: 遵循四舍五入规则。例如，1.6 -> 2， -1.6 -> -2。

```python
a = np.array([1.6, -1.6])
print(np.trunc(a))   # [ 1. -1.]
print(np.floor(a))   # [ 1. -2.]
print(np.ceil(a))    # [ 2. -1.]
print(np.round(a))   # [ 2. -2.]
```

**3. 分离整数与小数部分**
`np.modf()` 函数可以将数组中每个元素的整数部分和小数部分分开，并返回两个数组。
```python
a = np.array([1.2, 3.7, 5.0])
integer_part, fractional_part = np.modf(a)
# integer_part: [0.2 0.7 0. ]
# fractional_part: [1. 3. 5.]
```

### 特殊数值：NaN与Inf
在数值计算中，会遇到两种特殊的浮点数值。

**1. NaN (Not a Number)**
`NaN` 表示“不是一个数字”，通常由未定义的数学运算产生，例如 `0/0` 或对负数开方。
```python
a = np.array([0, 1, 2])
b = np.array([0, 1, 2])
result = a / b  # 结果为 [nan  1.  1.]
```
关键特性：`NaN` 不等于任何值，**甚至不等于它自己**。因此，判断一个值是否为 `NaN` 必须使用 `np.isnan()` 函数。
```python
value = np.nan
print(value == np.nan)   # False
print(np.isnan(value))   # True
```

**2. Inf (Infinity)**
`Inf` 表示“无穷大”，通常由一个非零数除以零产生。
```python
a = np.array([5, 6])
b = np.array([0, 0])
result = a / b  # 结果为 [inf inf]
```
`Inf` 比任何有限的数都大。判断一个值是否为无穷大，可以使用 `np.isinf()` 函数。

### 二元通用函数
二元函数是指接受两个数组作为参数的函数。除了基本的加减乘除，还有两个实用的函数。

**1. 逐元素最大值/最小值**
`np.maximum()` 和 `np.minimum()` 函数会逐元素比较两个数组，并返回对应位置的最大值或最小值。
```python
a = np.array([3, 4, 1])
b = np.array([2, 5, 0])
print(np.maximum(a, b))  # 结果为 [3 5 1]
print(np.minimum(a, b))  # 结果为 [2 4 0]
```
这与求整个数组最大值的 `a.max()` 方法不同。
