在本节课中，我们将深入探讨梯度下降算法中的关键参数设置，特别是批量大小（Batch Size）和学习率（Learning Rate）的选择。理解这些参数如何影响模型的训练效率和效果，是掌握机器学习优化过程的核心。

## 批量梯度下降的问题

上一节我们介绍了梯度下降的基本原理，本节中我们来看看其具体实现中的挑战。批量梯度下降（Batch Gradient Descent）使用全部样本计算梯度来更新参数。其更新公式为：

**θ = θ - α * (1/m) * Σ( hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾ ) * x⁽ⁱ⁾**

![](img/b3f4a1b97e9d5d5d0f2a9d2b439a5f23_1.png)

其中，`m` 代表样本总数。

![](img/b3f4a1b97e9d5d5d0f2a9d2b439a5f23_3.png)

这种方法虽然能求得所有样本平均下的最优更新方向，但存在一个明显问题：当样本数量 `m` 极大时（例如达到100万），每次参数更新都需要计算100万次，导致训练速度极其缓慢。

## 随机梯度下降的引入

![](img/b3f4a1b97e9d5d5d0f2a9d2b439a5f23_5.png)

为了解决批量梯度下降速度慢的问题，我们引入随机梯度下降（Stochastic Gradient Descent, SGD）。其核心思想是：每次更新参数时，只随机使用**一个**样本计算梯度。更新公式简化为：

![](img/b3f4a1b97e9d5d5d0f2a9d2b439a5f23_7.png)

**θ = θ - α * ( hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾ ) * x⁽ⁱ⁾**

这种方法的速度相比批量梯度下降提升了约 `m` 倍（即样本总数倍）。然而，它带来了新的问题：由于每次更新只依赖单个样本，更新方向受该样本的噪音或离群点影响很大，导致优化路径不稳定，收敛过程波动剧烈。

以下是两种方法收敛过程的对比示意图描述：
*   **批量梯度下降**：损失函数随迭代次数平滑、稳定地下降。
*   **随机梯度下降**：损失函数下降路径波动大，呈现锯齿状。

## 小批量梯度下降：一种折中方案

既然批量梯度下降（精确但慢）和随机梯度下降（快但不稳定）各有优缺点，我们自然想到一种折中方案：**小批量梯度下降（Mini-batch Gradient Descent）**。

这种方法每次更新时，既不使用全部样本，也不使用单个样本，而是随机抽取一小批（Mini-batch）样本进行计算。其更新公式为：

**θ = θ - α * (1/b) * Σ( hθ(x⁽ⁱ⁾) - y⁽ⁱ⁾ ) * x⁽ⁱ⁾**  （其中，`i` 从1到 `b`）

这里的 `b` 代表批量大小（Batch Size），即每次迭代使用的样本数量。

以下是关于批量大小 `b` 选择的要点：
*   **常见取值**：程序员通常偏好使用2的幂次方作为批量大小，例如64、128、256。
*   **大批量的影响**：`b` 值越大，每次梯度计算越接近真实分布（更精确），但训练速度越慢，且占用更多内存/显存。
*   **小批量的影响**：`b` 值越小，训练速度越快，内存占用越少，但梯度估计的噪声越大，可能影响收敛稳定性。
*   **选择原则**：在硬件（内存/显存）和可接受训练时间的限制下，通常选择尽可能大的批量大小。64、128、256是实践中常见且有效的选择。

![](img/b3f4a1b97e9d5d5d0f2a9d2b439a5f23_9.png)

## 学习率的作用与设置

在梯度下降的更新公式中，参数 `α` 被称为**学习率（Learning Rate）** 或步长。它控制着每次参数更新沿着梯度方向前进的幅度。

**θ_new = θ_old - α * Gradient**

![](img/b3f4a1b97e9d5d5d0f2a9d2b439a5f23_11.png)

以下是关于学习率的关键点：
*   **定义**：学习率 `α` 是一个需要手动设定的超参数。
*   **大小的影响**：
    *   学习率过大：可能导致更新步伐太大，在最优解附近震荡甚至发散。
    *   学习率过小：导致收敛速度非常慢，需要更多迭代次数。
*   **常见取值**：学习率通常设置为一个较小的正数，例如 `0.01`、`0.001` 或 `0.0001`。
*   **调参策略**：初始可以设置为一个常见值（如 `0.01`），如果模型训练不稳定或收敛慢，可以尝试逐步调小学习率。一种策略是直到模型能够稳定收敛的最小学习率为止。

## 总结与展望

本节课中我们一起学习了梯度下降算法的关键参数设置。我们分析了批量梯度下降和随机梯度下降的优缺点，并引出了实践中最常用的**小批量梯度下降**。我们明确了**批量大小（Batch Size）** 和**学习率（Learning Rate）** 这两个超参数的含义、影响以及设置的一般性原则。

![](img/b3f4a1b97e9d5d5d0f2a9d2b439a5f23_23.png)

需要牢记的是，在绝大多数机器学习问题中，我们均使用优化算法（如梯度下降）进行迭代求解，而非直接求解解析解。梯度下降是最基础、最经典的优化方法。后续课程中，我们还将接触到在其基础上改进的、更高效的优化算法。
