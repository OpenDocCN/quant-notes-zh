## 什么是matplotlib？🤔
matplotlib是一个强大的Python绘图和数据可视化工具包。简而言之，它是用来画图的。在金融分析或数据分析中，我们通常需要将数据（如股票价格）以图表的形式展示，这比查看一堆表格数据更为直观。

安装matplotlib的方法依然是使用pip命令：
```bash
pip install matplotlib
```

![](img/d14dea0ee95d891051cd19a0d27055f7_3.png)

## 基础使用方法 📈
要使用matplotlib，我们首先需要导入它。通常，我们导入其子模块`pyplot`，并习惯性地将其简写为`plt`。

![](img/d14dea0ee95d891051cd19a0d27055f7_5.png)

![](img/d14dea0ee95d891051cd19a0d27055f7_7.png)

![](img/d14dea0ee95d891051cd19a0d27055f7_9.png)

```python
import matplotlib.pyplot as plt
```

`plt.plot()`函数用于绘制图形，而`plt.show()`函数则用于将绘制的图形展示出来。

```python
plt.plot()
plt.show()
```
运行上述代码，会显示一个空的坐标轴图像，因为`plot`函数没有传入任何数据。

## 绘制折线图 📉
`plot`函数的核心功能是绘制折线图。它需要传入两个主要参数：X轴数据列表和Y轴数据列表。

![](img/d14dea0ee95d891051cd19a0d27055f7_11.png)

例如，我们传入两个列表：
```python
plt.plot([1, 2, 3, 4], [2, 4, 6, 8])
plt.show()
```
这段代码会绘制一条直线，它将点(1,2)、(2,4)、(3,6)、(4,8)依次连接起来。

![](img/d14dea0ee95d891051cd19a0d27055f7_13.png)

如果我们修改Y轴的数据，折线图就会发生变化：
```python
plt.plot([1, 2, 3, 4], [2, 3, 5, 7])
plt.show()
```
此时，图形将根据新的坐标点(1,2)、(2,3)、(3,5)、(4,7)绘制出一条折线。

除了列表，也可以传入NumPy数组作为参数。

![](img/d14dea0ee95d891051cd19a0d27055f7_15.png)

![](img/d14dea0ee95d891051cd19a0d27055f7_17.png)

## 自定义图表样式 🎨
`plot`函数还有第三个可选参数，它是一个格式字符串，用于控制折线图的线条样式、标记点形状和颜色。

![](img/d14dea0ee95d891051cd19a0d27055f7_19.png)

以下是自定义样式的几个关键部分：
*   **线条样式**：控制线条是实线、虚线还是点线。
*   **标记点**：控制数据点的显示形状。
*   **颜色**：控制线条和标记点的颜色。

![](img/d14dea0ee95d891051cd19a0d27055f7_21.png)

格式字符串的简写规则如下：
*   **线条样式**：`‘-‘`（实线），`‘--‘`（虚线），`‘:‘`（点线）。
*   **标记点**：`‘o‘`（圆点），`‘v‘`（下三角），`‘^‘`（上三角），`‘*‘`（星号），`‘+‘`（加号），`‘x‘`（叉号），`‘d‘`（菱形），`‘h‘`（六边形）。
*   **颜色**：`‘b‘`（蓝色），`‘g‘`（绿色），`‘r‘`（红色），`‘y‘`（黄色），`‘k‘`（黑色），`‘w‘`（白色）。

例如，`‘ro--‘`表示绘制红色的圆点，并用虚线连接它们。
```python
plt.plot([1, 2, 3, 4], [2, 4, 6, 8], ‘ro--‘)
plt.show()
```

![](img/d14dea0ee95d891051cd19a0d27055f7_23.png)

![](img/d14dea0ee95d891051cd19a0d27055f7_25.png)

你也可以使用关键字参数来分别指定样式，这样代码可读性更高：
```python
plt.plot([1, 2, 3, 4], [2, 4, 6, 8], color=‘red‘, marker=‘o‘, linestyle=‘--‘)
plt.show()
```

![](img/d14dea0ee95d891051cd19a0d27055f7_27.png)

## 高级功能与多图绘制 🔧
上一节我们介绍了如何绘制单条折线并自定义其样式。本节中我们来看看更复杂的使用场景，例如在一个图形中绘制多条折线。

![](img/d14dea0ee95d891051cd19a0d27055f7_29.png)

![](img/d14dea0ee95d891051cd19a0d27055f7_31.png)

绘制多条线非常简单，只需多次调用`plt.plot()`函数即可。每次调用都会在当前的图形（Figure）和坐标轴（Axes）上添加一条新的折线。

![](img/d14dea0ee95d891051cd19a0d27055f7_33.png)

```python
# 绘制第一条线
plt.plot([1, 2, 3, 4], [1, 4, 9, 16], ‘b-‘, label=‘Line 1‘)
# 绘制第二条线
plt.plot([1, 2, 3, 4], [2, 5, 10, 17], ‘r--o‘, label=‘Line 2‘)
# 添加图例
plt.legend()
plt.show()
```
通过`label`参数为每条线设置标签，并使用`plt.legend()`函数可以显示图例，帮助我们区分不同的数据系列。
