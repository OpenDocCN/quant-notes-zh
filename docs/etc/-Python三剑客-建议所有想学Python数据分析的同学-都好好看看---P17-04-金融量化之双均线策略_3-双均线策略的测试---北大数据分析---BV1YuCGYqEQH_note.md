以下是我们的核心交易规则与假设：
*   **初始资金**：`first_money = 100000` 元。
*   **买卖时机**：金叉时买入，死叉时卖出。
*   **交易价格**：使用信号出现当日的**开盘价**作为买卖单价。
*   **交易单位**：股票按“手”交易，1手为100股。因此，买入时需计算最多能买入多少“整手”股票。
*   **特殊情况**：如果数据最后一天是金叉（买入），则手中会持有未卖出的股票。这部分剩余股票的价值，将使用数据**最后一天的收盘价**进行估算，并计入总收益。

## 数据准备：整合金叉与死叉信号 🔗

首先，我们需要将金叉和死叉的时间点整合到一个序列中，并筛选出2010年之后的信号。

![](img/500cf6c2a50703e8328673daa2f71215_41.png)

```python
# 假设 golden_date 是金叉日期列表，death_date 是死叉日期列表
# 创建序列，用1标记金叉，用0标记死叉
s1 = pd.Series(1, index=golden_date)   # 金叉标识为1
s2 = pd.Series(0, index=death_date)    # 死叉标识为0

![](img/500cf6c2a50703e8328673daa2f71215_43.png)

![](img/500cf6c2a50703e8328673daa2f71215_45.png)

# 合并两个序列并按时间索引排序
s = pd.concat([s1, s2]).sort_index()

# 筛选出2010年之后的数据
s = s[‘2010’:‘2020’]
```
现在，序列 `s` 的索引是时间，值1代表金叉（买入），值0代表死叉（卖出）。

以下是实现策略的核心变量和逻辑步骤：
*   `first_money`：固定本金，用于最终计算收益。
*   `money`：动态资金，初始等于本金，买卖股票时从中增减。
*   `hold`：持有股票的数量（单位：股）。
*   **遍历信号序列**：对每个信号（金叉或死叉），执行相应操作。

![](img/500cf6c2a50703e8328673daa2f71215_85.png)

![](img/500cf6c2a50703e8328673daa2f71215_87.png)

```python
# 初始化变量
first_money = 100000
money = first_money  # 可变资金
hold = 0             # 持有股数

![](img/500cf6c2a50703e8328673daa2f71215_89.png)

![](img/500cf6c2a50703e8328673daa2f71215_91.png)

# 遍历所有交易信号
for i in range(len(s)):
    # 获取当前信号的日期
    current_time = s.index[i]
    
    if s.iloc[i] == 1:  # 金叉，买入信号
        # 1. 获取买入单价（当日开盘价）
        buy_price = df.loc[current_time, ‘open’]
        
        # 2. 计算最多能买多少手（1手=100股）
        # hand_count: 最多能买的手数
        hand_count = money // (buy_price * 100)
        
        # 3. 如果可以买（hand_count > 0），则更新持股和资金
        if hand_count > 0:
            buy_count = hand_count * 100  # 买入的股数
            hold += buy_count
            # 花费资金 = 买入股数 * 单价
            cost = buy_count * buy_price
            money -= cost
            
    else:  # 死叉，卖出信号
        if hold > 0:  # 如果持有股票，则卖出
            # 1. 获取卖出单价（当日开盘价）
            sell_price = df.loc[current_time, ‘open’]
            
            # 2. 卖出全部持有股票，收入资金
            revenue = hold * sell_price
            money += revenue
            
            # 3. 清空持股数量
            hold = 0
```

## 计算最终总收益 📊

循环结束后，我们需要计算策略的总收益。这包括两部分：动态资金 `money` 的变化，以及可能持有的剩余股票的价值。

![](img/500cf6c2a50703e8328673daa2f71215_99.png)

![](img/500cf6c2a50703e8328673daa2f71215_101.png)

![](img/500cf6c2a50703e8328673daa2f71215_102.png)

```python
# 计算剩余股票的价值（如果最后一天是金叉，hold > 0）
# 使用最后一天的收盘价估算剩余股票价值
last_price = df.iloc[-1][‘close’]  # 最后一天的收盘价
last_money = hold * last_price

![](img/500cf6c2a50703e8328673daa2f71215_104.png)

![](img/500cf6c2a50703e8328673daa2f71215_106.png)

![](img/500cf6c2a50703e8328673daa2f71215_108.png)

![](img/500cf6c2a50703e8328673daa2f71215_110.png)

![](img/500cf6c2a50703e8328673daa2f71215_112.png)

# 总收益 = (当前资金 + 剩余股票价值) - 初始本金
total_profit = (money + last_money) - first_money
print(f“策略总收益为：{total_profit:.2f} 元”)
```

核心步骤总结如下：
1.  **整合信号**：将金叉（1）和死叉（0）标记在统一的时间序列中。
2.  **初始化参数**：设定本金、动态资金和持股数。
3.  **模拟交易**：遍历信号，金叉时按开盘价尽可能多地买入，死叉时按开盘价全部卖出，并实时更新资金和持股。
4.  **计算收益**：结算时，将动态资金与剩余股票估值（按最后一天收盘价计算）相加，减去初始本金，得到总收益。
