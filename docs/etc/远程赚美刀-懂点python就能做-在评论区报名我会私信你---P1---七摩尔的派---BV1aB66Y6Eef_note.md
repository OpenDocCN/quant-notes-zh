# 量化交易入门：第1章：什么是量化交易？🚀

![](img/08fb7de0ed3304f5a13c07b7dc9c2658_1.png)

在本节课中，我们将学习量化交易的核心概念。我们将通过一个生动的赛马比喻来解释量化交易的基本思想，并了解一个名为“挖矿”的量化公司提供的兼职顾问机会。

![](img/08fb7de0ed3304f5a13c07b7dc9c2658_3.png)

## 概述

量化交易是一种利用数学模型和计算机程序来执行投资策略的方法。它旨在通过系统性的规则来消除人为情绪的影响，从而更稳定地获取收益。

![](img/08fb7de0ed3304f5a13c07b7dc9c2658_5.png)

## 什么是量化交易？

到底什么是量化交易？我们可以通过一个简单的比喻来理解。

![](img/08fb7de0ed3304f5a13c07b7dc9c2658_7.png)

假设我的马场里面有100匹马。市面上有两种赛马的游戏：一种比谁跑得快，另一种比谁跑得慢。

![](img/08fb7de0ed3304f5a13c07b7dc9c2658_9.png)

我该如何挑选参赛的马，让我获奖的几率最高呢？与其让我细心地去了解每一匹马的性能，不如让我来写一个规则。例如，我认为腿长的马跑得快。

那么我就按腿长给它们排一个序，前50名去跑比快的赛道，后50名去跑比慢的赛道。整个过程全部由代码自动操作。

## 从赛马到股市

![](img/08fb7de0ed3304f5a13c07b7dc9c2658_11.png)

把这个思维放到股市里面呢，就一共是3000只股票。然后我灵机一动，按某个规则给它排序，前多少名拿去做多（买入），后多少名拿去做空（卖出）。

![](img/08fb7de0ed3304f5a13c07b7dc9c2658_13.png)

这个思维就是量化。而我想的这个规则呢，就叫**阿尔法**。在金融领域，**阿尔法**通常指能够带来超额收益的投资策略或规则。

## 一个量化兼职机会

“挖矿”是一个跨国的量化公司，每年在全世界有几次招顾问的机会。这些顾问就替他们公司寻找有效的**阿尔法**策略。

他们现在给推荐人以及被推荐人一个激励，大约1400元人民币。

那么成为顾问的收益怎么样呢？

以下是收益构成：

*   **每日收益**：每天会根据你提交的阿尔法的质量，以及你账号的权重，给到每个阿尔法0~400元不等的回报。
*   **季度奖金**：每个季度，“挖矿”会把全球的量化顾问按照规则排名，发放季度奖金，金额大约在1400元到14万元之间，分为三个梯度。
*   **比赛奖金**：公司会定期在全球举办比赛，比赛过程与在家寻找阿尔法类似，前几名会获得额外奖金。

实际操作下来，前期一两个月，由于阿尔法质量不高以及账号权重不大，月收益可能在1000-3000元。但度过前期后，随着阿尔法质量提升，结合季度奖金，收益可能达到每月上万元甚至几万元。下图就是一位资深顾问的收益增长图，呈现出一条指数曲线。

总结下来就是：入门很简单，但获得高收益要靠个人能力。

## 兼职耗时与操作方式

这个兼职耗时间吗？其实我每天都只花20分钟。真正在持续工作的是网上的一台云电脑。我把设置发到云电脑里，它24小时不间断地运行代码，产出阿尔法供我提交。

这段代码还是“挖矿”公司通过培训课提供的，所以入门门槛很低。之所以目前操作简单，是因为参与的人还不多，竞争不激烈。

## 新人激励与参与流程

当然，一开始提到的1400元新人激励，并不是报名就能拿到。

以下是获得激励的条件：

1.  **填写推荐码**：报名时必须填写推荐码，否则无法获得激励。
2.  **完成学习**：需要参加四节公司的直播课。
3.  **坚持提交**：需要有累计十来天提交阿尔法的记录。

这些条件并不难达成。如果按照课程学习，最终你会有很多阿尔法策略可以提交。每天最多只能提交四个，所以完全够用。

## 报名须知与前景

本次报名比我参加的那次要求稍高，需要做一道题并提交一份简历。这道题对于新手可能看起来复杂，但了解之后会发现非常简单。你可以加入社群，我们一起探讨。

“挖矿”的操作很正规。成为顾问需要签署合同，每日收益在网页上透明可见，并且公司会在发薪前代为缴税。

如果你本身就想进入金融行业，那更需要注意这个机会。成为一名优秀的顾问，有可能收到“挖矿”公司的全职工作邀请。

最重要的是，报名将在1月8日中午左右截止。如果你有兴趣，需要抓紧时间行动。

## 总结

本节课中，我们一起学习了量化交易的核心思想——通过制定规则（阿尔法）让程序自动执行交易。我们还了解了一个具体的量化兼职机会，包括其收益模式、时间投入和参与流程。这是一个将编程知识应用于金融实践的入门途径。
