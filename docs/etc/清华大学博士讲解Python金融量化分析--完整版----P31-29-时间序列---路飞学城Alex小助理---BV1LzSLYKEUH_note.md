## 创建时间序列

上一节我们介绍了Pandas中生成时间对象的函数。本节中我们来看看如何用它们来构成一个时间序列。

我们可以使用`pd.date_range`函数生成一个`DatetimeIndex`，并将其作为Series或DataFrame的索引。以下是一个创建时间序列Series的示例：

```python
import pandas as pd
import numpy as np

![](img/241b9393f81b97ef010f2227cdb3082b_3.png)

# 创建一个长度为1000的Series，值为0到999
sr = pd.Series(np.arange(1000))

![](img/241b9393f81b97ef010f2227cdb3082b_5.png)

![](img/241b9393f81b97ef010f2227cdb3082b_7.png)

# 指定索引为从2017-01-01开始，长度为1000天的DatetimeIndex
sr.index = pd.date_range('2017-01-01', periods=1000)

print(sr.head())
print(type(sr.index))
```

![](img/241b9393f81b97ef010f2227cdb3082b_9.png)

运行上述代码，可以看到`sr.index`的类型是`DatetimeIndex`，而不仅仅是字符串。这标志着`sr`已经成为一个时间序列。

![](img/241b9393f81b97ef010f2227cdb3082b_11.png)

## 时间序列的切片操作

成为时间序列后，一个直观的好处是我们可以方便地进行基于时间的切片选取。

以下是几种常见的切片方式：

*   **按年或年月切片**：可以使用不完整的日期字符串进行切片。
    *   `sr['2017']`：选取2017年全年的数据。
    *   `sr['2017-03']`：选取2017年3月份的数据。
*   **按精确日期范围切片**：可以指定一个起止日期范围。
    *   `sr['2017-12-25':'2018-02-01']`：选取从2017年12月25日到2018年2月1日（包含首尾）的数据。

这种切片方式非常灵活，即使传入的是字符串格式的日期，Pandas也能正确识别并处理。

---

## 重采样（Resample）操作

除了切片，时间序列还支持许多强大的函数，例如`resample`函数。`resample`意为重新采样，它可以将数据按照新的时间频率进行聚合。

例如，我们有一个按天记录的数据序列，如果想查看每周的销售总额，可以使用`resample`。

以下是`resample`的用法示例：

```python
# 按周（‘W’）进行重采样，并计算每周的和
weekly_sum = sr.resample('W').sum()
print(weekly_sum.head())

# 按月（‘M’）进行重采样，并计算每月的平均值
monthly_mean = sr.resample('M').mean()
print(monthly_mean.head())
```

![](img/241b9393f81b97ef010f2227cdb3082b_13.png)

在代码中，`‘W’`代表周频率，`‘M’`代表月频率。调用`.sum()`或`.mean()`等聚合函数后，就会得到按新频率聚合后的结果，例如每周的总和或每月的平均值。

## 其他函数：truncate

时间序列还有一个`truncate`函数，用于截断某个日期之前或之后的数据。

其用法如下：
*   `sr.truncate(before='2018-02-03')`：保留2018年2月3日之后（含）的数据。
*   `sr.truncate(after='2018-02-03')`：保留2018年2月3日之前（含）的数据。

![](img/241b9393f81b97ef010f2227cdb3082b_17.png)

不过，这个函数的功能完全可以通过更直观的切片操作来实现（例如`sr['2018-02-03':]`），因此在实际中使用频率较低。
