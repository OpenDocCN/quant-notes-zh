在本节课中，我们将学习量化策略中的一个核心概念：因子中性化。我们将通过一个具体的例子——从市净率因子中剔除市值的影响——来详细解释其原理和实现步骤。理解这个过程对于构建稳健的量化策略至关重要。

## 因子中性化的核心思想 🎯

上一节我们介绍了因子处理的基本流程。本节中，我们来看看如何通过“中性化”来提纯因子。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_2.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_4.png)

因子中性化的目标是：从一个因子中，剔除我们不希望它包含的、来自其他变量（如市值、行业）的影响，从而得到更“纯净”的信号。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_6.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_8.png)

例如，我们有一个市净率因子。我们发现市净率与市值存在较强的相关性。这意味着市净率因子中，混杂了一部分市值的信息。如果我们想纯粹地使用市净率来选股，就需要剔除这部分市值的影响。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_10.png)

这个过程可以形象地理解为：将原始因子视为一个整体，找出其中能被市值解释的部分，然后将这部分减去，剩下的就是“提纯”后的因子。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_12.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_14.png)

## 如何实现中性化：回归分析法 📈

![](img/c030ec4d2765593c3af95ce6e3bdb19a_16.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_18.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_20.png)

那么，如何找到并剔除市值的影响部分呢？答案是使用线性回归。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_22.png)

我们可以建立以下回归方程：
`因子 = W * 市值 + B + ε`
在这个方程中：
*   **因子** 是因变量。
*   **市值** 是自变量。
*   **W** 和 **B** 是回归系数。
*   **ε** 是误差项。

建立这个回归方程的目的，是量化“市值”能解释“因子”中的多大一部分。通过求解，我们可以得到系数 **W** 和 **B**。

利用 **W** 和 **B**，我们可以计算出一个**预测值**：
`预测值 = W * 市值 + B`
这个预测值代表了因子中能被市值所解释的那部分。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_24.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_26.png)

然而，预测值几乎不可能完全等于真实值。**真实值**与**预测值**之间的差异，就是**误差项 ε**。
`误差项 ε = 真实因子值 - 预测值`
这个误差项 ε 的物理意义非常关键：它代表了市值**无法解释**的那部分因子信息。而这，正是我们通过中性化想要保留的“纯净”因子。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_28.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_30.png)

因此，中性化的操作步骤可以总结为：
1.  **建立回归方程**：以需要提纯的因子为因变量，以需要剔除的影响因子为自变量。
2.  **求解回归系数**：得到 **W** 和 **B**。
3.  **计算纯净因子**：用因子的真实值减去回归预测值。
`纯净因子 = 因子 - (W * 市值 + B)`

## 本节策略任务流程概述 🔄

![](img/c030ec4d2765593c3af95ce6e3bdb19a_34.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_36.png)

理解了中性化的原理后，我们来看一个完整的策略任务流程。在接下来的实践中，我们将以市净率和市值这两个指标为例。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_38.png)

以下是构建策略的基本步骤：
1.  **数据准备**：获取市净率和市值数据。
2.  **股票池初筛**：剔除停牌股、ST股、上市时间过短（如半年内）的股票，得到一个干净的初始股票池。
3.  **因子处理**：按照标准流程对因子进行处理，包括去极值、标准化，以及本节重点学习的中性化（从市净率中剔除市值影响）。
4.  **选股逻辑**：基于处理后的“纯净”因子制定选股规则。例如，筛选出“市净率小于0.2”的股票。
5.  **构建策略**：根据选股结果，决定买入和卖出的标的，形成一个完整的交易策略。

![](img/c030ec4d2765593c3af95ce6e3bdb19a_40.png)

![](img/c030ec4d2765593c3af95ce6e3bdb19a_42.png)
