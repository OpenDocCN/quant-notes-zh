# Python数据分析：P17：12 数据透视表常用功能 📊

在本节课中，我们将学习数据透视表的一系列常用功能。这些功能能帮助你高效地处理和分析数据，包括数据更新、排序、添加计算字段、动态筛选以及合并多个工作表等。掌握这些技巧，将使你的数据分析工作更加得心应手。

## 数据刷新与源数据更改 🔄

上一节我们介绍了数据透视表的基础创建。本节中我们来看看当原始数据发生变化时，如何更新数据透视表。

### 刷新数据透视表

![](img/827a4a10f27276292281b91da0da9db4_1.png)

如果原始表格中的数据值被修改（例如，某个销量数据记错了），数据透视表不会自动更新。此时，你需要使用“刷新”功能。

**操作步骤：**
1.  将鼠标光标定位在数据透视表区域内的任意单元格。
2.  此时，Excel功能区会出现“数据透视表工具”上下文选项卡。
3.  点击“分析”选项卡。
4.  在“数据”组中，点击“刷新”按钮，或选择“全部刷新”。

执行上述操作后，数据透视表中的汇总结果将根据修改后的源数据重新计算。

### 更改数据源范围

如果新增的数据行或列位于原始数据区域的**外部**（例如，在表格底部新增了一行），仅使用“刷新”功能无效。因为数据透视表创建时锁定了初始的数据源范围。此时，需要更改数据源。

**操作步骤：**
1.  同样，将光标定位在数据透视表内。
2.  在“数据透视表工具-分析”选项卡下，点击“数据”组中的“更改数据源”按钮。
3.  在弹出的对话框中，重新选择包含新增数据的整个区域。
4.  点击“确定”。

![](img/827a4a10f27276292281b91da0da9db4_8.png)

完成此操作后，数据透视表将包含新增的数据，再使用“刷新”功能即可更新计算结果。

## 对汇总结果进行排序 📈

有时我们需要查看销量最高或最低的品牌，这就需要对数据透视表的汇总结果进行排序。不能使用Excel常规的排序按钮，数据透视表有其专用的排序方式。

**操作步骤：**
1.  点击行标签（如“品牌”）字段右侧的下拉箭头。
2.  在下拉菜单中，选择“其他排序选项...”。
3.  在弹出的对话框中，选择“降序排序（Z到A）依据”。
4.  在下方的下拉列表中，选择你需要依据其排序的字段，例如“求和项：销量”。
5.  点击“确定”。

数据透视表将按照你指定的字段进行降序或升序排列。

## 添加计算字段与计算项 ➕

在数据分析中，我们经常需要在现有数据基础上计算新的指标，例如计算平均单价（收入/销量）。数据透视表提供了“计算字段”和“计算项”功能来实现这一点。

![](img/827a4a10f27276292281b91da0da9db4_10.png)

### 添加计算字段

计算字段是基于现有字段（列）进行计算，并在数据透视表中添加一个新列。

**操作步骤：**
1.  将光标定位在数据透视表的列字段区域。
2.  在“数据透视表工具-分析”选项卡下，点击“计算”组中的“字段、项目和集”。
3.  在下拉菜单中选择“计算字段”。
4.  在弹出的对话框中：
    *   在“名称”框输入新字段名，如“均价”。
    *   在“公式”框中，通过选择“字段”列表中的项目并插入来构建公式，例如：`=收入/销量`。
5.  点击“确定”后，数据透视表中将新增一列“均价”。

### 添加计算项

计算项是基于现有项（行）进行计算，并在数据透视表中添加一个新行。例如，计算两个品牌销量的差值。

**操作步骤：**
1.  将光标定位在数据透视表的行字段区域（例如“品牌”字段下的某个单元格）。
2.  在“数据透视表工具-分析”选项卡下，点击“计算”组中的“字段、项目和集”。
3.  在下拉菜单中选择“计算项”。
4.  在弹出的对话框中：
    *   在“名称”框输入新项名，如“差值”。
    *   在“公式”框中，通过选择“项”列表中的项目来构建公式，例如：`=宝马 - 丰田`。
5.  点击“确定”后，数据透视表中将新增一行“差值”，显示宝马与丰田在各指标上的差额。

## 优化数据透视表外观与动态筛选 🎨

默认生成的数据透视表格式可能不符合汇报要求，我们可以对其进行美化，并使用切片器实现动态筛选。

### 优化外观

以下是优化数据透视表外观的常用步骤：

**操作列表：**
*   **以表格形式显示**：在“设计”选项卡的“布局”组中，点击“报表布局”，选择“以表格形式显示”。
*   **取消分类汇总**：在“设计”选项卡的“布局”组中，点击“分类汇总”，选择“不显示分类汇总”。
*   **合并居中标签单元格**：右键点击数据透视表，选择“数据透视表选项”。在“布局和格式”选项卡下，勾选“合并且居中排列带标签的单元格”。
*   **显示/隐藏字段列表**：在“数据透视表工具-分析”选项卡的“显示”组中，点击“字段列表”按钮可以控制其显示或隐藏。

### 使用切片器动态筛选

切片器提供了一个直观的按钮式界面，用于筛选数据透视表的内容，效果是动态且交互式的。

**操作步骤：**
1.  点击数据透视表。
2.  在“数据透视表工具-分析”选项卡下，点击“筛选”组中的“插入切片器”。
3.  在弹出的对话框中，勾选你希望用于筛选的字段，例如“地区”和“品牌”，然后点击“确定”。
4.  界面上会出现切片器。点击切片器中的项目（如“东北”），数据透视表将只显示与该筛选条件匹配的数据。按住 `Ctrl` 键可以多选。
5.  要清除筛选，点击切片器右上角的“清除筛选器”按钮。
6.  要删除切片器，选中它并按 `Delete` 键。

## 拆分与合并工作表 📂

![](img/827a4a10f27276292281b91da0da9db4_14.png)

![](img/827a4a10f27276292281b91da0da9db4_15.png)

数据透视表还能帮助我们将一个总表按类别拆分成多个工作表，或者将多个结构相同的工作表合并分析。

### 根据类别拆分成多个工作表

![](img/827a4a10f27276292281b91da0da9db4_17.png)

![](img/827a4a10f27276292281b91da0da9db4_19.png)

如果你需要将数据按“地区”分别发送给不同负责人，可以使用此功能。

![](img/827a4a10f27276292281b91da0da9db4_21.png)

![](img/827a4a10f27276292281b91da0da9db4_23.png)

**操作步骤：**
1.  将需要拆分的字段（如“地区”）拖拽到数据透视表的“筛选器”区域。
2.  点击数据透视表。
3.  在“数据透视表工具-分析”选项卡下，点击“数据透视表”组中的“选项”下拉箭头，选择“显示报表筛选页”。
4.  在弹出的对话框中，确认要拆分的字段（如“地区”），点击“确定”。
5.  Excel会自动创建以每个筛选项（如“东北”、“华北”）命名的新工作表，每个工作表包含对应类别的数据透视表。

### 合并多个工作表（多重合并计算）

当你有多个结构相同的工作表（如各季度销售表），并希望快速合并分析时，可以使用“多重合并计算数据区域”功能。此功能默认不在功能区，需要先调出。

![](img/827a4a10f27276292281b91da0da9db4_25.png)

![](img/827a4a10f27276292281b91da0da9db4_26.png)

**调出“数据透视表和数据透视图向导”方法一（自定义功能区）：**
1.  点击“文件”->“选项”->“自定义功能区”。
2.  在“从下列位置选择命令”下拉列表中选择“不在功能区中的命令”。
3.  在下方列表中找到并选中“数据透视表和数据透视图向导”。
4.  在右侧主选项卡列表中，选择目标位置（如“数据”选项卡），点击“新建组”，然后点击“添加”。
5.  点击“确定”。

![](img/827a4a10f27276292281b91da0da9db4_28.png)

**方法二（使用快捷键）：**
直接按键盘快捷键 `Alt + D + P`（依次按下）即可调出向导。

**合并操作步骤：**
1.  使用上述任一方法打开“数据透视表和数据透视图向导”。
2.  选择“多重合并计算数据区域”，点击“下一步”。
3.  选择“创建单页字段”，点击“下一步”。
4.  依次选择每个要合并的工作表的数据区域，并点击“添加”按钮。
5.  添加完所有区域后，点击“下一步”。
6.  选择将数据透视表放置在新工作表，点击“完成”。
7.  Excel会生成一个合并了所有源数据的数据透视表。

## 创建数据透视图 📉

数据透视表可以快速转换为图表，即数据透视图。它继承了数据透视表的交互性，可以随筛选动态变化。

**操作步骤：**
1.  点击数据透视表。
2.  在“数据透视表工具-分析”选项卡下，点击“工具”组中的“数据透视图”。
3.  在弹出的“插入图表”对话框中选择一种图表类型（如柱形图），点击“确定”。
4.  生成的数据透视图会带有筛选按钮，点击可以像切片器一样动态筛选图表显示的数据。
