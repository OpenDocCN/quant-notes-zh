## 数组变形操作 🔄

上一节我们介绍了数组的索引与切片，本节中我们来看看如何改变数组的形状。变形操作指的是在不改变数组数据的前提下，重新排列其维度结构。

**核心概念**：使用 `reshape` 方法可以改变数组的形状。其基本公式为：
```python
new_array = old_array.reshape(new_shape)
```
其中，`new_shape` 是一个元组，指定了新数组的维度。新形状的元素总数必须与原数组一致。

例如，我们有一个五行六列的二维数组 `arr`，可以将其变形为一维数组：
```python
arr_1d = arr.reshape(30)  # 因为 5 * 6 = 30
```
反之，一维数组也可以变形为多维数组：
```python
arr_2d = arr_1d.reshape(2, 15)  # 或 (6, 5), (3, 10) 等
```

## 数组拼接操作 🔗

接下来，我们学习如何将多个数组组合在一起。拼接操作指的是将两个或多个数组沿指定轴连接起来，形成一个更大的数组。

**核心概念**：使用 `np.concatenate()` 函数进行拼接。其基本公式为：
```python
result = np.concatenate((array1, array2, ...), axis=axis)
```
参数 `axis` 决定了拼接的方向：
*   `axis=0`：沿行方向（纵向）拼接，即上下堆叠。
*   `axis=1`：沿列方向（横向）拼接，即左右并排。

以下是拼接操作的两个关键点：
1.  只能对**维度相同**的数组进行拼接。
2.  沿指定轴拼接时，其他轴的长度必须一致。例如，两个二维数组进行横向拼接（`axis=1`）时，它们的行数必须相同。

拼接操作在图像处理中非常有用，例如将多张图片拼接成九宫格。

## 聚合与统计操作 📈

现在，我们进入数据分析的核心部分：聚合与统计操作。这些函数能帮助我们快速计算数据的整体特征。

**核心概念**：NumPy数组对象内置了多种聚合方法，可以对整个数组或沿特定轴进行计算。

以下是常用的聚合方法：
*   `sum()`: 计算所有元素之和。
*   `min()`: 找出最小值。
*   `max()`: 找出最大值。
*   `mean()`: 计算平均值。

通过在方法中指定 `axis` 参数（0为列，1为行），可以计算每行或每列的统计值：
```python
arr.sum(axis=0)  # 计算每一列的和
arr.max(axis=1)  # 找出每一行的最大值
```

## 常用数学与统计函数 🧮

除了基本的聚合操作，NumPy还提供了丰富的数学函数和更专业的统计函数。

**核心概念**：这些函数通常位于 `np` 命名空间下，可以对数组中的每个元素或整个数组进行计算。

以下是部分重要的函数：
*   **数学函数**：如 `np.sin()`, `np.cos()`, `np.tan()` 计算三角函数；`np.round()` 进行四舍五入。
*   **统计函数**：
    *   `np.std()`: 计算**标准差**，衡量数据的离散程度。公式为：
        $$\sigma = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(x_i - \mu)^2}$$
        其中，$\sigma$ 是标准差，$N$ 是数据个数，$x_i$ 是每个数据点，$\mu$ 是平均值。
    *   `np.var()`: 计算**方差**，是标准差的平方。公式为：
        $$\sigma^2 = \frac{1}{N}\sum_{i=1}^{N}(x_i - \mu)^2$$
    *   `np.ptp()`: 计算极差（最大值与最小值之差）。

## 矩阵操作 ⬛

![](img/6d0c1d11ee38f73c791f70bb71c9d27b_3.png)

最后，我们来学习NumPy中的矩阵操作，这在机器学习和线性代数中应用广泛。

**核心概念**：在NumPy中，二维数组可以视作矩阵。我们重点关注矩阵的转置和乘法。

![](img/6d0c1d11ee38f73c791f70bb71c9d27b_5.png)

以下是矩阵的核心操作：
1.  **转置**：使用 `.T` 属性，将矩阵的行和列互换。
    ```python
    arr.T  # 获取数组arr的转置矩阵
    ```
2.  **矩阵乘法**：使用 `np.dot()` 函数或 `@` 运算符。它与普通元素乘法（`*`）不同，遵循线性代数中的乘法规则。
    **核心公式**：
    ```python
    C = np.dot(A, B)  # 或 C = A @ B
    ```
    对于矩阵 $A$（m行n列）和矩阵 $B$（n行p列），结果矩阵 $C$ 中的元素 $C_{ij}$ 计算方式为：
    $$C_{ij} = \sum_{k=1}^{n} A_{ik} \times B_{kj}$$
    即，$C$ 的第 $i$ 行第 $j$ 列元素，等于 $A$ 的第 $i$ 行与 $B$ 的第 $j$ 列对应元素乘积之和。

    例如：
    $$A = \begin{bmatrix} 2 & 1 \\ 4 & 3 \end{bmatrix}, B = \begin{bmatrix} 1 & 2 \\ 1 & 0 \end{bmatrix}$$
    $$C = A \cdot B = \begin{bmatrix} (2*1+1*1) & (2*2+1*0) \\ (4*1+3*1) & (4*2+3*0) \end{bmatrix} = \begin{bmatrix} 3 & 4 \\ 7 & 8 \end{bmatrix}$$
