## 聚类与无监督学习

上一节我们介绍了有监督学习，本节中我们来看看无监督学习中的聚类问题。

聚类与分类的效果类似，都是将数据分组。但关键区别在于，聚类处理的是无标签数据。我们无法预先知道每个数据点属于哪个类别，只能根据数据本身的相似性进行分组。例如，在右侧的图中，原始数据没有颜色标记，算法需要根据相似度度量，将数据自动归为绿色、红色和蓝色三组。

聚类算法的难点主要在于评估和调参。由于没有真实标签作为标准，我们难以直接判断聚类结果的好坏，也难以比较不同参数下的聚类效果。

## KMEANS算法核心概念

![](img/a175465ce305f78221cabd0a99e57cde_4.png)

接下来，我们将深入探讨KMEANS算法，这是聚类算法中最简单且最实用的一种。

![](img/a175465ce305f78221cabd0a99e57cde_6.png)

### 1. K值（聚类数量）

KMEANS算法需要我们预先指定一个参数：**K值**。K值代表我们希望将数据聚合成多少个簇（组）。

![](img/a175465ce305f78221cabd0a99e57cde_8.png)

*   **K=3**：数据将被聚成3堆。
*   **K=4**：数据将被聚成4堆。
*   **K=5**：数据将被聚成5堆。

![](img/a175465ce305f78221cabd0a99e57cde_10.png)

### 2. 质心（Centroid）

![](img/a175465ce305f78221cabd0a99e57cde_12.png)

质心是KMEANS算法中另一个核心概念。**质心**代表一个簇中所有数据点的平均值位置。

![](img/a175465ce305f78221cabd0a99e57cde_14.png)

对于一个二维数据点（X, Y），其所在簇的质心计算方式为：
*   **质心X坐标** = 簇内所有数据点X坐标的平均值。
*   **质心Y坐标** = 簇内所有数据点Y坐标的平均值。

每个簇都有自己的质心，它在算法迭代过程中起到关键作用。

### 3. 距离度量

聚类依据“物以类聚”的原则，将相似的数据点分到一组。如何衡量“相似”呢？我们通过计算数据点之间的**距离**来判断。

最常见的距离计算方式是**欧式距离**。对于两个二维点 (x1, y1) 和 (x2, y2)，其欧式距离公式为：
`距离 = sqrt((x1 - x2)^2 + (y1 - y2)^2)`

![](img/a175465ce305f78221cabd0a99e57cde_16.png)

**重要提示**：在使用距离度量前，通常需要对数据进行**标准化**（如归一化）。这是因为如果不同特征（如X轴和Y轴）的数值范围差异巨大，数值范围大的特征会主导距离计算，从而影响聚类效果。标准化能确保所有特征在相近的尺度上贡献相似。

![](img/a175465ce305f78221cabd0a99e57cde_18.png)

### 4. 优化目标

![](img/a175465ce305f78221cabd0a99e57cde_20.png)

与大多数机器学习算法一样，KMEANS通过优化一个目标函数来求解。它的优化目标是：**最小化每个簇中所有样本点到其质心距离的总和**。

用公式可以表示为：
`最小化：Σ（i=1 到 K） Σ（x 属于 簇Ci） || x - μi ||^2`
其中：
*   **K** 是簇的数量。
*   **Ci** 代表第i个簇。
*   **x** 是簇Ci中的一个样本点。
*   **μi** 是簇Ci的质心。
*   **|| x - μi ||^2** 是样本点x到其质心μi的欧式距离的平方。

![](img/a175465ce305f78221cabd0a99e57cde_22.png)

![](img/a175465ce305f78221cabd0a99e57cde_24.png)

这个目标函数的直观理解是：我们希望每个簇内的点都尽可能紧密地围绕在质心周围。如果一个点离当前簇的质心很远，但离另一个簇的质心更近，算法在迭代中就会将它重新分配到更近的那个簇里去。

## 总结

![](img/a175465ce305f78221cabd0a99e57cde_26.png)

本节课中我们一起学习了KMEANS聚类算法的基本框架。我们了解到：
1.  聚类是一种无监督学习方法，用于将未标记的数据按相似性分组。
2.  KMEANS算法需要预先指定聚类数量K。
3.  算法的核心是**质心**，即每个簇的中心点。
4.  使用**距离**（通常是欧式距离）来衡量数据点之间的相似性，计算前需对数据进行标准化。
5.  算法的优化目标是**最小化簇内样本点到其质心的距离总和**，通过迭代不断更新质心和样本点所属的簇来实现。

![](img/a175465ce305f78221cabd0a99e57cde_28.png)

KMEANS算法原理直观，是实现聚类任务的一个强大而实用的起点。
