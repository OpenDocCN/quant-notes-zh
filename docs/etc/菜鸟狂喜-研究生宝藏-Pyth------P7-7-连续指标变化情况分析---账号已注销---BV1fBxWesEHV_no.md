## 平均结果的加法操作

上一节我们介绍了如何计算平均值。本节中我们来看看，对于平均结果，我们是否可以进行加法操作。

![](img/403a0759261f585da46ddf21ddc40b05_4.png)

例如，我们观察一个平均结果，它反映了整体情况。那么，我们能否对一个连续时间序列计算其增长率呢？

**增长率计算公式**：
`增长率 = (后一天价格 - 前一天价格) / 前一天价格`

举个例子：
*   第一天价格：100元
*   第二天价格：50元
*   第三天价格：75元

以下是计算过程：
1.  第二天增长率：(50 - 100) / 100 = -0.5
2.  第三天增长率：(75 - 50) / 50 = 0.5

![](img/403a0759261f585da46ddf21ddc40b05_6.png)

如果我们将这两个增长率相加：-0.5 + 0.5 = 0。这个结果为零，似乎意味着股票从第一天到第三天既没涨也没跌。

![](img/403a0759261f585da46ddf21ddc40b05_8.png)

然而，观察原始数据：第一天100元，第三天75元，股票实际上是下跌的。这说明，对于增长率，**不能直接进行加法操作**，因为累加的结果无法反映第一天与最后一天的真实差异。

![](img/403a0759261f585da46ddf21ddc40b05_10.png)

## 连续增长率的计算方法

既然增长率不能直接累加，那么我们该如何处理连续时间序列的累加操作呢？

![](img/403a0759261f585da46ddf21ddc40b05_12.png)

![](img/403a0759261f585da46ddf21ddc40b05_14.png)

对于连续的计算，我们需要借助一个数学公式：**对数**。

![](img/403a0759261f585da46ddf21ddc40b05_16.png)

![](img/403a0759261f585da46ddf21ddc40b05_18.png)

实际计算中，我们使用对数来处理连续增长率。具体方法是计算后一天与前一天比值的对数，然后进行累加。

![](img/403a0759261f585da46ddf21ddc40b05_20.png)

![](img/403a0759261f585da46ddf21ddc40b05_22.png)

继续使用之前的例子：
*   计算对数差值：`log(50/100) + log(75/50)`
*   根据对数运算法则，加法可以转换为乘法：`log( (50/100) * (75/50) ) = log(75/100)`

![](img/403a0759261f585da46ddf21ddc40b05_24.png)

![](img/403a0759261f585da46ddf21ddc40b05_26.png)

这个结果`log(75/100)`，恰恰反映了**最后一天（75）与第一天（100）之间的整体差异**。

![](img/403a0759261f585da46ddf21ddc40b05_28.png)

![](img/403a0759261f585da46ddf21ddc40b05_30.png)

这个方法的形象理解是：假设股票初始为1元钱，经过一段时间序列的涨跌，将每个时间点的增长率用对数表示并累加，最终结果就等价于最后一天股价相对于第一天股价的倍数关系。它回答了“第一天投入的一块钱，相当于最后一天是多少钱”这个问题。

![](img/403a0759261f585da46ddf21ddc40b05_32.png)

**核心公式**：
连续累计收益率 = `log(第N天价格 / 第1天价格) = Σ log(第t天价格 / 第t-1天价格)`

![](img/403a0759261f585da46ddf21ddc40b05_34.png)

## 使用Python计算连续增长率

接下来，我们看看如何在Python中实现上述计算。

![](img/403a0759261f585da46ddf21ddc40b05_40.png)

首先，我们需要得到“前一天”的数据。在Pandas中，可以使用`.shift()`函数将数据向上移动一格。

![](img/403a0759261f585da46ddf21ddc40b05_42.png)

![](img/403a0759261f585da46ddf21ddc40b05_44.png)

```python
# 计算每日的简单收益率（后一天/前一天）
returns = data / data.shift(1)
# 计算对数收益率（连续增长率）
log_returns = np.log(returns)
```

![](img/403a0759261f585da46ddf21ddc40b05_46.png)

执行以上代码，我们就得到了一个表示连续增长率的序列。它描述了每一天相对于前一天的连续变化情况。

## 还原为累积价值

![](img/403a0759261f585da46ddf21ddc40b05_54.png)

目前我们只计算了中间的对数收益率。我们还需要将其还原，以查看“第一天的一块钱相当于后续每一天多少钱”。

![](img/403a0759261f585da46ddf21ddc40b05_56.png)

我们需要将对数收益率进行累加，然后通过指数函数还原为实际的价值倍数。这可以通过`np.cumsum()`（累积和）函数和`np.exp()`（指数）函数实现。

![](img/403a0759261f585da46ddf21ddc40b05_58.png)

`np.cumsum()`函数示例：
输入 `[1, 2, 3, 4]`，输出 `[1, 3, 6, 10]`。即每个位置的值是原序列从开始到当前位置所有值的和。

![](img/403a0759261f585da46ddf21ddc40b05_60.png)

![](img/403a0759261f585da46ddf21ddc40b05_62.png)

以下是还原步骤：
```python
# 1. 将对数收益率进行累积求和
cumulative_log_returns = log_returns.cumsum()
# 2. 通过指数函数还原为实际的价值倍数（以第一天的1元为基准）
cumulative_value = np.exp(cumulative_log_returns)
# 3. 绘制图表，观察各股票从基准日开始的累积价值走势
cumulative_value.plot(figsize=(16, 8))
```

![](img/403a0759261f585da46ddf21ddc40b05_64.png)

生成的图表会显示，假设在第一天为每只股票投入1元钱（或1个单位），到后续每一天，这1元钱相当于多少价值。例如，图表可能显示亚马逊的1元在期末变成了10元，而苹果的1元变成了约4元。这直观地反映了不同股票在时间序列上的整体增长表现。
