## 重采样操作

![](img/6240225849535b3e25e39834839dd9c5_11.png)

上一节我们介绍了时间序列的基本概念，本节中我们来看看如何改变其观测频率。

重采样是指将时间序列数据从一个频率转换到另一个频率的过程。例如，将每日数据聚合为每周或每月数据。

### 基本重采样方法

![](img/6240225849535b3e25e39834839dd9c5_13.png)

以下是重采样的基本步骤：

![](img/6240225849535b3e25e39834839dd9c5_15.png)

1.  **指定目标频率**：你需要告诉`resample`方法新的时间单位，例如`‘W’`代表周，`‘M’`代表月。
2.  **指定聚合方法**：对于新频率下的每个时间段，你需要决定如何聚合原始数据。常见方法包括取均值(`mean`)、第一个值(`first`)、最后一个值(`last`)或中位数(`median`)等。

![](img/6240225849535b3e25e39834839dd9c5_17.png)

核心操作代码如下：
```python
# 假设df是一个带有日期时间索引的DataFrame
# 将日数据重采样为周数据，并计算每周的均值
weekly_data = df.resample(‘W’).mean()
# 将日数据重采样为月数据，并取每月的第一个值
monthly_data = df.resample(‘M’).first()
```

![](img/6240225849535b3e25e39834839dd9c5_19.png)

![](img/6240225849535b3e25e39834839dd9c5_21.png)

### 标签位置参数

![](img/6240225849535b3e25e39834839dd9c5_23.png)

在重采样时，还有一个重要参数是`label`，它决定了新时间段的标签是使用该时间段的起始点(`left`)还是结束点(`right`)。

例如，将1月份的日数据聚合为月度数据：
*   `label=‘right’`（默认）：聚合结果的标签会显示为`‘2023-01-31’`，表示这是1月份数据的汇总。
*   `label=‘left’`：聚合结果的标签会显示为`‘2022-12-31’`，表示从这一天开始汇总至1月末的数据。

选择哪种标签取决于分析逻辑。通常，使用结束点(`right`)作为标签更符合“报告期”的直观理解。

## 滑动窗口操作

了解了如何改变数据频率后，我们来看看如何在连续的时间段上进行滑动计算。

![](img/6240225849535b3e25e39834839dd9c5_29.png)

滑动窗口操作不是改变数据频率，而是在一个固定大小的窗口上（例如过去10天）沿着时间轴滑动，并对窗口内的数据进行计算，从而生成一个新的序列。

![](img/6240225849535b3e25e39834839dd9c5_31.png)

### 滑动窗口的实现

![](img/6240225849535b3e25e39834839dd9c5_33.png)

以下是实现滑动窗口计算的基本步骤：

![](img/6240225849535b3e25e39834839dd9c5_35.png)

1.  **定义窗口大小**：例如，`window=10`表示窗口包含10个时间单位的数据。
2.  **应用滚动方法**：使用`.rolling(window)`方法创建滚动窗口对象。
3.  **应用聚合函数**：对每个窗口应用统计函数，如`.min()`, `.max()`, `.mean()`, `.std()`等。

![](img/6240225849535b3e25e39834839dd9c5_37.png)

![](img/6240225849535b3e25e39834839dd9c5_39.png)

核心操作代码如下：
```python
# 计算‘close’列过去10个交易日的移动最小值
data[‘min_10’] = data[‘close’].rolling(window=10).min()
# 计算过去10个交易日的移动标准差
data[‘std_10’] = data[‘close’].rolling(window=10).std()
```

![](img/6240225849535b3e25e39834839dd9c5_41.png)

### 重要注意事项：处理缺失值

![](img/6240225849535b3e25e39834839dd9c5_43.png)

![](img/6240225849535b3e25e39834839dd9c5_45.png)

在应用滑动窗口前，**必须处理原始数据中的缺失值**。如果窗口内存在`NaN`值，会导致整个窗口的计算结果也为`NaN`。

![](img/6240225849535b3e25e39834839dd9c5_47.png)

建议先进行数据清洗：
```python
# 删除含有缺失值的行
data_clean = data.dropna()
# 在清洗后的数据上执行窗口计算
data_clean[‘rolling_mean’] = data_clean[‘close’].rolling(window=10).mean()
```

![](img/6240225849535b3e25e39834839dd9c5_49.png)

### 多指标窗口计算示例

![](img/6240225849535b3e25e39834839dd9c5_51.png)

![](img/6240225849535b3e25e39834839dd9c5_53.png)

你可以同时计算一个窗口上的多个统计指标，为分析提供更丰富的特征。

![](img/6240225849535b3e25e39834839dd9c5_55.png)

以下是同时计算窗口内最小值、最大值、均值和标准差的示例：
```python
# 假设data_clean是已处理缺失值的DataFrame
window_size = 10
data_clean[‘min_10’] = data_clean[‘close’].rolling(window=window_size).min()
data_clean[‘max_10’] = data_clean[‘close’].rolling(window=window_size).max()
data_clean[‘mean_10’] = data_clean[‘close’].rolling(window=window_size).mean()
data_clean[‘std_10’] = data_clean[‘close’].rolling(window=window_size).std()
# 查看计算结果
print(data_clean[[‘min_10‘， ’max_10‘， ’mean_10‘， ’std_10‘]].tail())
```

## 总结

![](img/6240225849535b3e25e39834839dd9c5_61.png)

![](img/6240225849535b3e25e39834839dd9c5_63.png)

本节课中我们一起学习了时间序列分析的两个关键操作：
1.  **重采样**：通过`df.resample().agg()`方法，可以将数据聚合到不同的时间频率（如日变周、月），并需注意`label`参数对时间标签的影响。
2.  **滑动窗口**：通过`df.rolling().agg()`方法，可以在一个固定长度的连续窗口上进行滑动计算，得到如移动平均、移动标准差等序列。**切记在操作前清洗数据，处理缺失值**。

![](img/6240225849535b3e25e39834839dd9c5_67.png)
