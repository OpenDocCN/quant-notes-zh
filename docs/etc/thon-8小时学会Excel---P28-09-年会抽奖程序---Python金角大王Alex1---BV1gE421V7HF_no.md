## 项目需求分析

以下是具体的抽奖规则与需求：

1.  **员工总数**：公司共有300名员工。
2.  **奖项设置**：
    *   一等奖：3名，奖品为“泰国5日游”。
    *   二等奖：6名，奖品为“iPhone手机”。
    *   三等奖：30名，奖品为“避孕套”。
3.  **抽奖顺序与规则**：
    *   抽奖共进行三次。
    *   第一次抽取三等奖，一次性抽出30名获奖者。
    *   第二次抽取二等奖，一次性抽出6名获奖者。
    *   第三次抽取一等奖，一次性抽出3名获奖者。
4.  **核心限制**：每位员工只能中奖一次。一旦中奖，将从后续的抽奖池中移除，不能重复获奖。

![](img/df337aff9e0757cbc8dd5b0dc663b5a3_7.png)

## 核心思路与步骤

要完成这个程序，我们可以遵循以下逻辑步骤。理解思路比直接看代码更重要。

以下是实现抽奖程序的关键步骤：

1.  **初始化数据**：首先，我们需要创建包含所有300名员工的列表。可以用数字编号（如1-300）或生成包含“员工+编号”的字符串来代表每位员工。
    ```python
    # 示例：使用列表推导式创建员工列表
    all_employees = [f‘员工{i}’ for i in range(1, 301)]
    ```

2.  **准备抽奖池**：由于获奖员工需要从后续抽奖中排除，我们需要一个变量来动态代表当前有资格参与抽奖的员工池。最初，这个池子就是全体员工的列表。

3.  **执行多轮抽奖**：使用循环来控制三轮抽奖。每一轮中：
    *   确定本轮要抽取的**奖项名称**和**获奖人数**。
    *   从当前的**抽奖池**中，随机抽取相应数量的员工，且**不能重复**。
    *   使用 `random.sample()` 函数可以完美实现一次性无重复随机抽取。
        ```python
        import random
        # 从列表pool中随机抽取k个不重复的元素
        winners = random.sample(pool, k)
        ```
    *   打印本轮的中奖结果。
    *   将本轮的中奖者从**抽奖池**中移除，确保他们不会参与后续轮次的抽奖。可以使用列表推导式或集合操作来实现。

4.  **输出最终结果**：循环结束后，输出所有奖项的完整中奖名单。
