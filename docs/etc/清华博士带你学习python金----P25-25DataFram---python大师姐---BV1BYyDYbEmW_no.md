## 数据对齐

和`Series`对象一样，`DataFrame`也涉及数据对齐。因为它拥有行索引和列索引两个维度，所以需要按照行索引和列索引分别进行对齐。

例如，我们有两个`DataFrame`对象`df1`和`df2`，它们的行索引顺序不同。当我们执行`df1 + df2`这样的运算时，Pandas会自动根据索引进行对齐计算。如果某个位置在其中一个`DataFrame`中是缺失值（NaN），那么运算结果在该位置也会是NaN。

以下是数据对齐的核心概念：
*   **行索引对齐**：运算时，系统会匹配两个`DataFrame`中相同的行标签。
*   **列索引对齐**：运算时，系统会匹配两个`DataFrame`中相同的列标签。

![](img/d48fbe09693660912127c77ffed82f4f_3.png)

即使两个`DataFrame`的列顺序相反，Pandas在计算时也会根据列名正确对齐。

## 缺失数据处理

![](img/d48fbe09693660912127c77ffed82f4f_7.png)

`DataFrame`处理缺失数据的方法与`Series`对象大部分相似，但也存在一些区别。

### 填充缺失值

使用`fillna()`方法可以将缺失值填充为指定的值。这与`Series`的用法一致。

```python
df2.fillna(0)  # 将所有NaN值填充为0
```

### 删除缺失值

使用`dropna()`方法可以删除包含缺失值的行或列。这是与`Series`处理方式的主要区别点。

默认情况下，`dropna()`会删除**任何**包含缺失值的行。

```python
df2.dropna()  # 默认删除任何包含NaN的行
```

有时我们可能希望只删除**全部**值都为缺失值的行，而保留仅部分列有缺失值的行。这时可以使用`how`参数。

```python
# 创建一个示例DataFrame
df2 = pd.DataFrame({
    ‘one‘: [1, 2, 3, np.nan],
    ‘two‘: [5, np.nan, np.nan, np.nan]
}, index=[‘a‘, ‘b‘, ‘c‘, ‘d‘])

# 只删除所有值都为NaN的行（即‘d‘行）
df2.dropna(how=‘all‘)
```

`how`参数说明：
*   **`how=‘any‘`**：默认值。如果某行（或列）中**有任何**一个值是NaN，则删除。
*   **`how=‘all‘`**：只有当某行（或列）中**所有**值都是NaN时，才删除。

### 按列删除缺失值

在数据分析中，通常将每一列视为一个属性（特征），每一行视为一条记录。但有时我们可能需要按列来处理缺失值，例如删除包含缺失值的整列。

这可以通过设置`axis`参数来实现。`DataFrame`可以看作一个二维对象，`axis`参数用于指定操作的轴方向。

```python
# 删除任何包含NaN的列
df2.dropna(axis=1)
```

![](img/d48fbe09693660912127c77ffed82f4f_9.png)

`axis`参数说明：
*   **`axis=0` 或 `axis=‘index‘`**：默认值。对行进行操作（删除行）。
*   **`axis=1` 或 `axis=‘columns‘`**：对列进行操作（删除列）。

![](img/d48fbe09693660912127c77ffed82f4f_11.png)

此外，检查缺失值的方法`isna()`和`notna()`，与之前讲解的`Series`对象的用法完全一致。
