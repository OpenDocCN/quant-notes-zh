# Python金融时间序列分析与量化交易实战教程：P52：51.回归问题概述 🎯

在本节课中，我们将要学习回归问题的基本概念，这是机器学习中监督学习的两大核心任务之一。我们将通过一个银行贷款额度的例子，清晰地解释什么是回归问题，并引入线性回归的基本思想。

上一节我们介绍了机器学习中的分类问题，本节中我们来看看什么是回归问题。

## 回归问题定义

在监督学习中，主要存在两类问题：分类与回归。
分类问题有明确的类别，例如银行决定“贷款”（1）或“不贷款”（0）。
回归问题则预测一个具体的数值。例如，询问银行“能贷给我多少钱”，答案可能是一个在一定范围内的具体数字，如3万、4.8万或9.76万。我们的目标就是预测这个连续值。

为了进行回归分析，我们需要有标签的数据。以下是一个示例数据集的结构：

以下是数据表格的示例：

*   **工资 (X1)**：数据的第一列，一个特征。
*   **年龄 (X2)**：数据的第二列，另一个特征。
*   **额度 (Y)**：数据的第三列，这是我们想要预测的目标值（标签）。

在这个数据集中，**X1** 和 **X2** 是我们建模时需要参考的特征（自变量），而 **Y** 是我们希望通过模型得到的结果（因变量）。进行回归任务时，必须同时具备特征（X）和标签（Y）。

## 建立回归方程

既然Y的值与X1和X2都相关，我们可以尝试建立一个方程来描述这种关系。简单的相加（Y = X1 + X2）并不合理，因为这假设工资和年龄对结果的影响程度完全相同。

![](img/e0407abc802dc7fbbbbe4a93163daf7a_1.png)

显然，工资对贷款额度的影响通常比年龄更大。因此，我们需要为每个特征引入一个系数，来表示其影响权重。

于是，我们可以构建如下线性方程：
`Y = θ1 * X1 + θ2 * X2`
其中，**θ1** 和 **θ2** 就是我们需要求解的系数。例如，若 θ1=3，θ2=1，则表示工资的影响力是年龄的3倍。

在回归分析中，X1、X2和Y都是已知数据。核心目标就是找出最合适的 **θ1** 和 **θ2** 的值。

![](img/e0407abc802dc7fbbbbe4a93163daf7a_3.png)

## 线性方程的拟合问题

然而，一个线性方程通常无法完美穿过所有的数据点。如下图所示，数据点（红色）是离散分布的，而任何线性方程（平面）都只能是一条直线。

我们面临的问题是：单一的线性平面无法拟合所有数据点。

既然无法满足所有点，那么合理的解决方案是：**找到一个能尽可能拟合更多数据点的平面**。也就是说，我们需要在众多可能的线性方程中，选择那个与所有数据点整体误差最小的一个。

这个“整体误差”通常用损失函数（如均方误差）来衡量，而寻找最佳θ1和θ2的过程就是模型训练的核心。
