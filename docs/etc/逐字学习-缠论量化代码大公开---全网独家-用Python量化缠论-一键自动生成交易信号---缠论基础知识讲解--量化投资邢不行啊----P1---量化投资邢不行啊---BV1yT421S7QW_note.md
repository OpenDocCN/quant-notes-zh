## 什么是缠论？🤔

![](img/a01d970543e7052e1cecb7c828ecfa2e_11.png)

缠论是由网络名人“缠中说禅”在其“教你炒股”系列文章中整理而来的理论体系，于2006年6月7日正式面世。据说该理论适合一切具有波动走势的投资市场，例如股票、权证和期货等。

![](img/a01d970543e7052e1cecb7c828ecfa2e_13.png)

![](img/a01d970543e7052e1cecb7c828ecfa2e_15.png)

根据百科介绍，缠论依据市场最基本的确定性——“任何走势都会结束”为基础，从几何的角度对走势的结构逐步推导，对市场的各种走势做出科学的完全分类，从而指导实际操作。它被认为是目前唯一能用数学手段进行证明的交易理论。

![](img/a01d970543e7052e1cecb7c828ecfa2e_17.png)

看到“数学手段”这几个字，我们自然联想到量化分析。缠论的作者在2008年停更后便再未出现，其真实身份众说纷纭，但这并不影响其理论在投资者心中的地位。在2006年，缠论一经问世就受到了无数人的追捧，被认为是极其有用的技术形态分析体系。

![](img/a01d970543e7052e1cecb7c828ecfa2e_19.png)

了解了缠论的来历，我们就可以尝试通过其用法去量化它。缠论体系大致可分为分型、笔、线段、中枢与走势四层结构，这些要素一起构成了最终的买卖判断。但要全部讲解会非常复杂，因此我们今天只聚焦于最基础的部分：**缠论中的K线与分型**。

## 缠论中的K线 📈

在了解分型前，首先需要明确：缠论中所说的K线与寻常的K线并不相同。

![](img/a01d970543e7052e1cecb7c828ecfa2e_27.png)

传统的K线通过上下影线和实体来表现股票当天的开盘价、最高价、最低价、收盘价，并区分阴线和阳线。但在缠论中，K线**只看最高价和最低价**，认为其他都是无效信息。因此，缠论中的K线被简化成只保留最高价和最低价的柱体。

![](img/a01d970543e7052e1cecb7c828ecfa2e_29.png)

观察两根相邻K线之间的关系，缠论将其分为两种：
1.  **有趋势关系**：称为“不存在包含关系”。
2.  **无趋势关系**：称为“存在包含关系”。

缠论作者认为，包含关系的K线太乱，影响分析价格趋势，因此需要对其进行简化处理，直到所有K线两两之间都不再是包含关系。

![](img/a01d970543e7052e1cecb7c828ecfa2e_31.png)

---

![](img/a01d970543e7052e1cecb7c828ecfa2e_33.png)

### 如何处理包含关系的K线？

![](img/a01d970543e7052e1cecb7c828ecfa2e_35.png)

处理包含关系的方法其实很简单。我们首先找三根K线，例如以下两种情况（趋势虽不同，但本质一致）：
*   **情况A（上升趋势）**：K线1和K线3呈上升趋势，同时K线1和K线2相互包含。
*   **情况B（下降趋势）**：K线1和K线3呈下降趋势，同时K线1和K线2相互包含。

遇到这种情况，缠论的处理方式比较直接：用一根新的K线4来代替K线1和K线2。

*   **对于上升趋势（情况A）**：K线4的最高价和最低价，分别取K线1和K线二中**相对较高的最高价**和**相对较高的最低价**。总结为：**上升取高**。
*   **对于下降趋势（情况B）**：K线4的最高价和最低价，分别取K线1和K线二中**相对较低的最高价**和**相对较低的最低价**。总结为：**下降取低**。

![](img/a01d970543e7052e1cecb7c828ecfa2e_37.png)

通过上述“上升取高，下降取低”的原则，对K线进行向后迭代处理，我们最终能得到一组新的K线组合，其中任意相邻两根K线之间，都只存在清晰的上升或下降关系，而不再有包含关系。

![](img/a01d970543e7052e1cecb7c828ecfa2e_39.png)

理清K线处理方法是量化缠论的关键所在。当然，以上所讲都是基于个人对缠论的理解，欢迎有不同的看法进行指正和探讨。

![](img/a01d970543e7052e1cecb7c828ecfa2e_41.png)

![](img/a01d970543e7052e1cecb7c828ecfa2e_45.png)

## 缠论中的分型应用 ⚡️

在缠论中，分型是由相邻的三根K线构成的一种结构。这三根K线并非连续的上升或下降趋势，而是呈现走势反转的形态。

![](img/a01d970543e7052e1cecb7c828ecfa2e_59.png)

分型主要分为两种：
*   **底分型**：意味着股价跌至低位即将反弹，被视为抄底或建仓的潜在信号。
*   **顶分型**：意味着股价涨至高位即将见顶，被视为卖出的潜在信号。

![](img/a01d970543e7052e1cecb7c828ecfa2e_61.png)

![](img/a01d970543e7052e1cecb7c828ecfa2e_63.png)

它们的图形定义比较直观，易于辨认。需要注意的是，单一的分型结构还比较基础，不能完全作为买卖点的唯一判断依据。但缠论中真正的买卖点，正是以分型为基础一步步构建起来的。

既然我们知道了顶分型和底分型的定义，就可以利用之前预处理好的K线数据，并借助Python代码，自动找出任意股票在全部历史数据中出现的所有顶分型和底分型。

具体的识别代码已经准备好。运行程序后，结果如图所示：红色方块标记了所有顶分型的位置，绿色三角标记了所有底分型的位置。通过这张图，我们可以非常直观地看到指定时间内该股票的所有分型位置。
