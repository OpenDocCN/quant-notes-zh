## 构建数据获取函数

![](img/92de670335fc9015fb49b0bf2b0f4f8f_62.png)

![](img/92de670335fc9015fb49b0bf2b0f4f8f_64.png)

调仓逻辑依赖于股票的评分，因此我们需要一个函数 `get_stocks` 来获取数据并进行初步处理。

![](img/92de670335fc9015fb49b0bf2b0f4f8f_66.png)

![](img/92de670335fc9015fb49b0bf2b0f4f8f_68.png)

![](img/92de670335fc9015fb49b0bf2b0f4f8f_70.png)

![](img/92de670335fc9015fb49b0bf2b0f4f8f_72.png)

以下是 `get_stocks` 函数的数据获取部分：

![](img/92de670335fc9015fb49b0bf2b0f4f8f_74.png)

1.  **构建查询语句 (Query)**：我们需要查询多个基本面指标。根据因子模型，我们选取两类指标：
    *   **越高越好型指标**：例如每股收益、净资产收益率、投入资本回报率。
    *   **越低越好型指标**：例如资产负债率、市净率(PB)、市值。

    我们使用 `query` 函数来构建查询。
    ```python
    # 查询“越高越好”的指标
    q_up = query(
        fundamentals.eod_derivative_indicator.eps, # 每股收益
        fundamentals.financial_indicator.return_on_equity, # 净资产收益率
        fundamentals.financial_indicator.return_on_invested_capital # 投入资本回报率
    )

    # 查询“越低越好”的指标
    q_down = query(
        fundamentals.financial_indicator.debt_to_asset_ratio, # 资产负债率
        fundamentals.eod_derivative_indicator.pb_ratio, # 市净率(PB)
        fundamentals.eod_derivative_indicator.market_cap # 市值
    )
    ```

![](img/92de670335fc9015fb49b0bf2b0f4f8f_76.png)

![](img/92de670335fc9015fb49b0bf2b0f4f8f_78.png)

![](img/92de670335fc9015fb49b0bf2b0f4f8f_80.png)

2.  **添加过滤条件**：我们只关心沪深300成分股的数据。通过 `filter` 方法将股票代码限制在预设的池子中。
    ```python
    q_up = q_up.filter(fundamentals.stockcode.in_(context.hs300))
    q_down = q_down.filter(fundamentals.stockcode.in_(context.hs300))
    ```

![](img/92de670335fc9015fb49b0bf2b0f4f8f_82.png)

![](img/92de670335fc9015fb49b0bf2b0f4f8f_84.png)

3.  **获取数据并转换格式**：使用 `get_fundamentals` 执行查询，获取数据。返回的数据格式通常需要转置（`.T`），以便股票代码作为行索引，方便后续处理。
    ```python
    df_up = get_fundamentals(q_up).T # “越高越好”指标 DataFrame
    df_down = get_fundamentals(q_down).T # “越低越好”指标 DataFrame
    ```
    此时，`df_up` 和 `df_down` 是两个 `DataFrame`，其形状约为 `(300, 3)`，分别包含了300支股票的3个正向指标和3个负向指标。
