![](img/dc9439d44f3572f66a192e55d02c76de_3.png)

上一节我们介绍了布尔型索引，本节中我们来看看另一种强大的索引方法——花式索引。

## 什么是花式索引？

花式索引是指使用一个整数数组（或列表）作为索引，来选取目标数组中对应位置的元素。这与我们之前学习的普通索引（单个数字）和切片（连续范围）不同，它允许我们选取任意、非连续位置的数据。

![](img/dc9439d44f3572f66a192e55d02c76de_5.png)

例如，对于一个一维数组，我们想选取第1、3、4、6、7号元素（注意：索引从0开始）。使用普通索引或切片无法直接实现，但花式索引可以轻松做到。

![](img/dc9439d44f3572f66a192e55d02c76de_7.png)

以下是花式索引的基本语法：
```python
import numpy as np
a = np.arange(20)  # 创建一个0到19的数组
indices = [1, 3, 4, 6, 7]  # 指定要选取的索引位置
result = a[indices]  # 使用花式索引
print(result)  # 输出：[1 3 4 6 7]
```
代码中，`indices`列表指定了我们需要的索引位置，`a[indices]`则返回这些位置上的元素组成的新数组。

## 二维数组的花式索引

对于二维数组，花式索引的规则同样适用，并且可以与逗号分隔符结合，实现对行和列的不同选取策略。

NumPy数组的索引可以概括为四种方式：普通索引、切片、布尔型索引和花式索引。在二维数组中，我们可以通过逗号分隔行和列的索引方式，灵活组合使用它们。

例如，我们有一个4行5列的二维数组`arr`：
```python
arr = np.arange(20).reshape(4, 5)
```
我们可以进行如下混合索引操作：
*   **行用普通索引，列用切片**：选取第0行，第2到第4列（不包含第4列）。
    ```python
    arr[0, 2:4]
    ```
*   **行用普通索引，列用布尔型索引**：选取第0行中所有大于2的元素。
    ```python
    arr[0, arr[0] > 2]
    ```

## 花式索引的注意事项与高级用法

虽然花式索引功能强大，但在二维数组中使用时有一个重要的限制：**逗号两侧不能同时使用花式索引列表**，否则会产生意想不到的结果，它会被解释为选取坐标对。

假设我们想选取第1行&第1列、第1行&第3列、第3行&第1列、第3行&第3列这四个位置的元素。直觉上可能会写成：
```python
# 错误的写法：这不会得到四个值，而是两个值
arr[[1, 3], [1, 3]]
```
上述代码并不会返回四个值，而是返回`(1,1)`和`(3,3)`这两个坐标位置的值，即`[6, 18]`。

那么，如何正确选取我们想要的四个值呢？需要分步操作：

![](img/dc9439d44f3572f66a192e55d02c76de_9.png)

以下是分步选取特定行和列元素的正确方法：
1.  首先，使用花式索引选取指定的行（第1行和第3行），并保留所有列。
    ```python
    rows = arr[[1, 3], :]
    ```
2.  然后，在第一步结果的基础上，再选取指定的列（第1列和第3列）。
    ```python
    final_result = rows[:, [1, 3]]
    ```
当然，这两步可以合并为一行代码：
```python
final_result = arr[[1, 3], :][:, [1, 3]]
```
通过这种“先选行，再选列”的分步策略，我们就可以灵活地组合各种索引方式，实现复杂的数据选取需求。
