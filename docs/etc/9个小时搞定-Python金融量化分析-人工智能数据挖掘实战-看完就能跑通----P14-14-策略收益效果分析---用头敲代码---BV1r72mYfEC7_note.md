## 计算基础收益率

上一节我们介绍了如何生成交易信号，本节中我们来看看如何计算收益。计算收益时，我们需要知道初始投资（例如1元）最终变成了多少钱。这是一个累积的过程。

在计算累积收益时，不能直接相加，通常需要引入对数收益率进行计算。在我们的数据中，需要对股价进行计算。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_5.png)

以下是计算每日对数收益率的步骤：

![](img/fb02a573c78fcd2390b35480fa1dd6e6_7.png)

1.  获取股价数据。
2.  计算当前日股价除以前一日股价的结果。
3.  对这个比值取自然对数，得到对数收益率。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_9.png)

对应的代码公式如下：
```python
# 假设 `data[‘Close’]` 是收盘价序列
data[‘returns’] = np.log(data[‘Close’] / data[‘Close’].shift(1))
```
执行这段代码后，我们得到了一个名为 `returns` 的列，它代表了股票每日的增长（或下跌）率。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_11.png)

## 结合策略计算调整后收益

![](img/fb02a573c78fcd2390b35480fa1dd6e6_13.png)

接下来，我们将按照我们的交易策略来计算收益。我们的策略核心是：在特定的市场区间内，进行反向操作。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_15.png)

这里需要用到之前计算的 `position` 列。`position` 为 1 代表我们持有股票（跟随市场），为 -1 代表我们进行反向操作。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_17.png)

我们的策略收益计算逻辑是：用 `position` 值乘以基础的 `returns` 值。这样，当 `position` 为 -1 时，我们就实现了与市场走势相反的操作。

以下是计算策略收益的步骤：

![](img/fb02a573c78fcd2390b35480fa1dd6e6_19.png)

![](img/fb02a573c78fcd2390b35480fa1dd6e6_21.png)

1.  确保 `position` 列已经通过 `.shift(1)` 进行了滞后处理，以匹配收益率的时间点（即用前一天的信号决定今天的操作）。
2.  将滞后后的 `position` 与 `returns` 相乘。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_23.png)

对应的代码公式如下：
```python
# 计算策略收益
data[‘strategy’] = data[‘position’].shift(1) * data[‘returns’]
```
计算完成后，我们可以查看 `strategy` 这一列。数据的前面部分可能看起来与 `returns` 相同，这是因为策略信号在前期大部分时间为 1。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_25.png)

![](img/fb02a573c78fcd2390b35480fa1dd6e6_27.png)

## 比较策略收益与基础收益

![](img/fb02a573c78fcd2390b35480fa1dd6e6_29.png)

我们最终想看到的是策略带来的总收益效果。这需要对收益率进行累积求和。

由于我们之前使用的是对数收益率，现在需要将其还原为实际的累积收益。还原的方法是：对累积的对数收益率取指数。

以下是计算并比较总收益的步骤：

![](img/fb02a573c78fcd2390b35480fa1dd6e6_31.png)

![](img/fb02a573c78fcd2390b35480fa1dd6e6_33.png)

1.  分别对 `returns` 列和 `strategy` 列进行求和，得到总的对数收益率。
2.  对这个总和取自然指数 `np.exp()`，将其还原为实际的资产倍数（例如，1元变成了多少元）。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_35.png)

对应的代码公式如下：
```python
# 计算总收益（还原为资产倍数）
total_return = np.exp(data[‘returns’].sum())
total_strategy_return = np.exp(data[‘strategy’].sum())
```
执行后，我们可以比较两个结果。例如，在本例中：
*   **什么都不做（基础收益）**：1元钱最终变成了约4元钱。
*   **使用双均线策略**：1元钱最终变成了约5.8元钱。

![](img/fb02a573c78fcd2390b35480fa1dd6e6_37.png)

结果表明，应用双均线策略后，收益得到了提升。
