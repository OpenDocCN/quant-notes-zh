在本节课中，我们将要学习数据预处理中一个非常核心的技术——标准化。标准化是数据挖掘和机器学习中常用的方法，其目的是消除不同特征之间由于量纲和取值范围不同带来的影响，使数据处于同一尺度，便于后续的分析和建模。

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_3.png)

## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体原理。

标准化的核心目标是让不同指标或因子在数值上具有可比性。假设我们有一组数据，包含两个特征X1和X2。通常，X1和X2的取值范围是不同的。例如，X1的取值范围可能较大，从-10到10；而X2的取值范围可能较小，从-1到1。如果我们直接用这些原始数据进行分析，取值范围大的特征可能会主导模型，从而对结果产生不必要的影响。

为了解决这个问题，我们需要进行标准化处理。标准化的公式如下：

**`z = (x - μ) / σ`**

其中：
*   **`x`** 是原始数据。
*   **`μ`** 是该特征所有数据的均值。
*   **`σ`** 是该特征所有数据的标准差。

这个公式可以分解为两个步骤来理解。

## 标准化的两个步骤

以下是标准化过程的两个核心步骤：

**第一步：去均值**

`x - μ` 这一步称为“去均值”或“中心化”。它的目的是将数据集的中心移动到坐标原点（0点）。原始数据通常不是以0为中心对称分布的。减去均值后，数据在各个维度上都会以0为中心对称。这解决了数据分布中心不一致的问题。

**第二步：除以标准差**

`(x - μ) / σ` 中的除以标准差是第二步。去均值后，虽然数据以0为中心，但不同特征的数据分布范围（即离散程度）仍然不同。除以标准差就是为了解决这个问题。

标准差衡量了一组数据的离散程度。取值范围大、数据分散的特征，其标准差也大；取值范围小、数据集中的特征，其标准差也小。

通过除以各自的标准差，我们实现了：
*   对于取值范围大的特征，除以一个较大的标准差，可以将其数值范围缩小。
*   对于取值范围小的特征，除以一个较小的标准差，可以适当保留其数值变化。

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_5.png)

最终，所有特征的数据都将被转换到均值为0、标准差为1的分布上，它们的数值范围变得近似，具备了可比性。

## 标准化的代码实现

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_7.png)

理解了原理后，我们来看看如何用代码实现标准化。我们可以根据公式手动计算，也可以使用现成的工具库。

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_9.png)

**方法一：手动实现**

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_11.png)

我们可以根据公式 `z = (x - μ) / σ` 编写一个标准化函数。

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_13.png)

```python
def standardize(series):
    """
    对输入的pandas Series进行标准化处理。
    参数:
        series: pandas Series，待处理的数据。
    返回:
        标准化后的pandas Series。
    """
    # 计算均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    # 应用标准化公式
    return (series - mean_val) / std_val
```

**方法二：使用Scikit-learn库**

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_15.png)

在Python的机器学习库Scikit-learn中，提供了更便捷的标准化工具。

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_17.png)

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_19.png)

```python
from sklearn.preprocessing import StandardScaler

# 创建StandardScaler对象
scaler = StandardScaler()
# 注意：fit_transform方法通常接受二维数组，对于单个特征需要reshape
data_reshaped = your_data.values.reshape(-1, 1)
data_standardized = scaler.fit_transform(data_reshaped)
```

![](img/f40b0fa27e60316a2fc9b12134d4bc8e_21.png)

使用Scikit-learn的`StandardScaler`可以高效地处理多维度数据集，并且可以方便地保存拟合后的参数（均值和标准差），用于后续对新数据的转换。
