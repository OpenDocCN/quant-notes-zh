## 选股的定义与作用

选股是通过某种技术手段或量化指标，为投资者提供判断个股的依据，从而帮助投资者选定目标个股的方法。在A股市场超过4000只股票中，选股是进行股票投资或量化投资的第一步。

选股的好坏直接决定了投资者能否盈利。如果选择的股票持续亏损，无论后续如何操作都难以赚钱。



以下是选股的常用方法介绍。

## 基本面选股

基本面选股是分析上市公司发展前景的核心方法。其定义是：分析上市公司在发展过程中面临的重要外部因素和自身内部因素，目标是预测公司未来的发展前景，进而判断其股票是否值得购买。



接下来，我们分析构成基本面的外部因素和内部因素。



![](img/4d90931218a9244b64fd104659f0a001_7.png)

### 外部因素

外部因素主要指宏观环境对公司的影响，主要包括：

![](img/4d90931218a9244b64fd104659f0a001_9.png)

*   **经济增长**：整体的经济环境是影响股价的基础因素。
*   **财政政策**：宏观的财政政策会深刻影响公司的经营环境。
*   **利率变化**：市场利率以及公司所在行业的平均贷款利率等。



### 内部因素

内部因素主要指公司自身的状况，主要包括：

*   **经营状况**：公司的业绩好坏，是否处于长期高速发展阶段。
*   **行业地位**：公司在行业中是龙头、第二名还是其他位置。短时间内排名快速上升的公司（成长股）也值得关注。
*   **财务状况**：包括公司的利润率、净利润、成长性等。



![](img/4d90931218a9244b64fd104659f0a001_11.png)

以上外部和内部因素共同构成了基本面分析的基础。那么，如何科学地综合分析一只股票的基本面呢？



## 股票估值：基本面选股的核心方法

分析股票基本面最重要的方法是**估值**。股票估值是基本面选股的核心，它能帮助投资者发现股票当前价值是否被低估，从而实现“低买高卖”获利。

每个人对股票的估值可能不同，但存在一些公认的常用指标。



以下是股票估值的六个常用指标。



*   **每股收益 (EPS)**：**公式：`每股收益 = 净利润 / 总股本`**。该指标越高越好，代表公司盈利能力强，能为股东带来更多分红等收益。
*   **市盈率 (P/E Ratio)**：**公式：`市盈率 = 股价 / 每股收益`**。反映估值水平，需与同行业公司比较。通常14-30倍属正常，大于30可能高估，超过50倍可能存在泡沫。
*   **毛利率 (Gross Margin)**：**公式：`毛利率 = (营业收入 - 营业成本) / 营业收入`**。该指标越高越好，代表公司初步盈利能力。毛利率长期大于50%的公司非常优质。
*   **净资产收益率 (ROE)**：**公式：`净资产收益率 = 净利润 / 净资产`**。代表公司利用自有资本的盈利效率。长期保持在20%以上的公司通常是优质白马股。
*   **资产负债率 (Debt-to-Asset Ratio)**：**公式：`资产负债率 = 总负债 / 总资产`**。该指标应维持适中水平，一般在10%到40%之间。过高易引发风险，过低则表明公司发展可能过于保守。
*   **净利润增速 (Net Profit Growth Rate)**：**公式：`净利润增速 = (本期净利润 - 上期净利润) / 上期净利润`**。表征公司成长性。通常考察近三年平均增速，超过20%为优质企业，超过50%可能属于高成长股。
