## 字典的基本概念与创建

![](img/165dc248923d0ca5cb826a40dcb35d65_7.png)

上一节我们介绍了列表和元组，本节中我们来看看字典。字典是一种存储“键值对”的数据结构。在量化交易中，我们可以用它来存储股票代码与股票名称的映射关系。

![](img/165dc248923d0ca5cb826a40dcb35d65_9.png)

创建一个字典非常简单，使用大括号 `{}`，内部用冒号 `:` 分隔键和值，不同的键值对之间用逗号 `,` 分隔。

![](img/165dc248923d0ca5cb826a40dcb35d65_11.png)

以下是创建一个简单股票池字典的示例：

![](img/165dc248923d0ca5cb826a40dcb35d65_13.png)

```python
# 创建一个股票代码与名称对应的字典
stock_pool = {
    "600000.SH": "浦发银行",
    "000001.SZ": "平安银行"
}
```

![](img/165dc248923d0ca5cb826a40dcb35d65_15.png)

我们可以直接打印这个字典，查看其内容。

![](img/165dc248923d0ca5cb826a40dcb35d65_17.png)

```python
print("股票池内容：")
print(stock_pool)
```

运行上述代码，输出结果会原样显示我们定义的键值对。

![](img/165dc248923d0ca5cb826a40dcb35d65_19.png)

## 字典的特性与访问元素

![](img/165dc248923d0ca5cb826a40dcb35d65_23.png)

字典的核心特性是“键值对”结构。每个“键”必须是**不可变类型**（如字符串、数字、元组），并且在同一个字典中**不能重复**，这就像股票代码具有唯一性一样。

![](img/165dc248923d0ca5cb826a40dcb35d65_25.png)

访问字典中的值，就像查字典一样，我们通过“键”来获取对应的“值”。

![](img/165dc248923d0ca5cb826a40dcb35d65_27.png)

以下是访问字典中元素的两种主要方法：

![](img/165dc248923d0ca5cb826a40dcb35d65_29.png)

1.  **使用中括号 `[]`**：这是最常用的方法，将键放在中括号内。
    ```python
    # 通过键“600000.SH”获取其值
    stock_name = stock_pool["600000.SH"]
    print(f"股票代码 600000.SH 对应的名称是：{stock_name}")
    ```
    如果键不存在，这种方法会引发 `KeyError` 错误。

2.  **使用 `.get()` 方法**：这种方法更安全。如果键不存在，它会返回 `None` 或者你指定的默认值，而不会报错。
    ```python
    # 安全地获取一个可能不存在的键
    stock_name = stock_pool.get("000002.SZ")
    print(f"尝试获取000002.SZ：{stock_name}")  # 输出：None

    # 指定默认返回值
    stock_name = stock_pool.get("000002.SZ", "未找到该股票")
    print(f"尝试获取000002.SZ（带默认值）：{stock_name}")  # 输出：未找到该股票
    ```

## 字典的常用操作

![](img/165dc248923d0ca5cb826a40dcb35d65_35.png)

了解了如何创建和访问字典后，我们来看看如何修改和更新它。字典的内容是可以动态变化的。

![](img/165dc248923d0ca5cb826a40dcb35d65_37.png)

以下是字典的一些基本操作：

![](img/165dc248923d0ca5cb826a40dcb35d65_39.png)

*   **添加或修改元素**：直接给一个键赋值即可。如果键已存在，则更新其值；如果不存在，则添加新的键值对。
    ```python
    # 添加一个新股票
    stock_pool["000002.SZ"] = "万科A"
    print("添加万科A后的股票池：", stock_pool)

    # 修改已有股票的名称
    stock_pool["600000.SH"] = "浦发银行(更新)"
    print("修改浦发银行后的股票池：", stock_pool)
    ```

![](img/165dc248923d0ca5cb826a40dcb35d65_41.png)

*   **删除元素**：使用 `del` 语句或 `.pop()` 方法。
    ```python
    # 使用del删除
    del stock_pool["000001.SZ"]
    print("删除平安银行后的股票池：", stock_pool)

    # 使用.pop()删除并返回被删除的值
    removed_stock = stock_pool.pop("600000.SH", "键不存在")
    print(f"被删除的股票是：{removed_stock}")
    print("删除后的股票池：", stock_pool)
    ```

![](img/165dc248923d0ca5cb826a40dcb35d65_43.png)

*   **获取所有键、值或键值对**：
    ```python
    # 获取所有键
    all_codes = stock_pool.keys()
    print("所有股票代码：", list(all_codes))

    # 获取所有值
    all_names = stock_pool.values()
    print("所有股票名称：", list(all_names))

    # 获取所有键值对（以元组形式）
    all_items = stock_pool.items()
    print("所有键值对：", list(all_items))
    ```

## 总结

![](img/165dc248923d0ca5cb826a40dcb35d65_47.png)

本节课中我们一起学习了Python的字典数据类型。我们了解到：

![](img/165dc248923d0ca5cb826a40dcb35d65_49.png)

1.  字典是一种**键值对**结构，用大括号 `{}` 定义，键值之间用冒号 `:` 分隔。
2.  字典的**键必须是不可变且唯一的**，值可以是任意数据类型。
3.  我们可以通过键来**访问、添加、修改和删除**字典中的值。
4.  字典提供了 `.keys()`、`.values()` 和 `.items()` 等方法来方便地获取其内容。

![](img/165dc248923d0ca5cb826a40dcb35d65_51.png)

字典是Python中高效管理关联数据的强大工具，在量化交易中，无论是存储股票信息、配置参数还是缓存计算结果，它都将是你不可或缺的帮手。
