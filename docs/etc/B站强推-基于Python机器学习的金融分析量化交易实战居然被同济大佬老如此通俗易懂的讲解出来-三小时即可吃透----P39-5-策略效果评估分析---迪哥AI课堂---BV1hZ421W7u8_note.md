以下是运行策略时可能遇到的几个典型错误及其解决方法：

![](img/6adc180bc6a3008b0f9722da79e2679e_25.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_27.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_29.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_31.png)

1.  **参数缺失错误**：在调用中性化处理函数时，缺少了必要的参数。需要检查函数定义，确保传入所有必需的参数，例如数据框和指标名称。
    ```python
    # 错误示例：缺少指标名称参数
    result = neutralize(factor_data)
    # 正确示例：传入指标名称
    result = neutralize(factor_data, ‘target_factor‘)
    ```

![](img/6adc180bc6a3008b0f9722da79e2679e_33.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_35.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_37.png)

2.  **函数调用错误**：在使用模型（如线性回归）前，没有先进行训练（`fit`操作）。必须先用数据训练模型，才能进行预测或计算残差。
    ```python
    # 错误示例：未训练模型就直接使用
    model = LinearRegression()
    residuals = model.predict(X)
    # 正确示例：先训练，再使用
    model = LinearRegression()
    model.fit(X, y) # 训练模型
    residuals = y - model.predict(X) # 计算残差
    ```

![](img/6adc180bc6a3008b0f9722da79e2679e_39.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_41.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_43.png)

3.  **拼写错误**：变量名或函数名拼写错误，例如将 `quantile` 误写为 `qantile`。需要仔细检查代码中的拼写。
    ```python
    # 错误示例
    selected_stocks = factor_data[factor_data[‘value‘] <= factor_data[‘value‘].qantile(0.3)]
    # 正确示例
    selected_stocks = factor_data[factor_data[‘value‘] <= factor_data[‘value‘].quantile(0.3)]
    ```

![](img/6adc180bc6a3008b0f9722da79e2679e_45.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_47.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_49.png)

4.  **属性错误**：尝试访问对象不存在的属性，例如对一个列表（`list`）使用点号（`.`）访问。通常需要使用方括号（`[]`）进行索引或调用列表方法。
    ```python
    # 错误示例：对列表使用点号语法
    stock_list.delete(stock)
    # 正确示例（假设delete是自定义函数）
    delete_stock(stock_list, stock)
    # 或使用列表的remove方法
    stock_list.remove(stock)
    ```

## 分析最终回测结果 📈

![](img/6adc180bc6a3008b0f9722da79e2679e_89.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_91.png)

回测运行完毕后，系统会提供一份详细的报告。我们需要关注以下几个核心指标来评估策略效果：

![](img/6adc180bc6a3008b0f9722da79e2679e_93.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_95.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_97.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_99.png)

*   **累计收益率**：策略在整个回测期间的总收益。图中显示，基准收益率为负，而我们的策略收益率为正或负值更小，表明策略可能跑赢了市场。
*   **年化收益率**：将累计收益率折算成年化的指标，便于比较不同时间跨度的策略。
*   **夏普比率**：衡量每承受一单位风险，所获得的超额回报。公式为：
    `夏普比率 = (策略年化收益率 - 无风险利率) / 策略年化波动率`
    该值越高，说明风险调整后的收益越好。报告中显示该值较低，说明策略可能收益不高或波动较大。
*   **最大回撤**：策略净值从最高点到最低点的最大跌幅，是衡量策略风险的重要指标。报告中显示最大回撤约为9%，属于需要关注但并非极高的水平。
*   **阿尔法**与**贝塔**：
    *   **贝塔**：衡量策略相对于市场的波动性。贝塔为1表示与市场波动一致。
    *   **阿尔法**：衡量策略超越市场基准的超额收益。正的阿尔法是追求的目标。

![](img/6adc180bc6a3008b0f9722da79e2679e_101.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_103.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_105.png)

在本例的回测结果中，策略在选定年份的表现虽然绝对收益不高，但关键指标（如累计收益、最大回撤）均优于同期市场基准，证明了基于市值和市净率因子进行筛选和定期调仓的基本逻辑是有效的。

## 策略流程回顾 🔄

![](img/6adc180bc6a3008b0f9722da79e2679e_111.png)

本节课实现的因子选股策略，其完整流程可以总结为以下几步：

![](img/6adc180bc6a3008b0f9722da79e2679e_117.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_119.png)

![](img/6adc180bc6a3008b0f9722da79e2679e_121.png)

1.  **初始化与定时设置**：在策略的 `__init__` 构造函数中，设置定时调仓的频率（例如每月第一个交易日）。
2.  **股票池过滤**：在 `before_trading_start` 或类似函数中，过滤掉ST股、停牌股等不符合条件的股票，得到初始股票池。
3.  **因子获取与处理**：查询股票池中每只股票的所需因子（如市值、市净率）。对因子数据进行必要的预处理，如中性化处理（消除行业、市值等风格的影响）。
4.  **选股逻辑执行**：根据处理后的因子值，应用选股规则（例如，选择市净率最低的30%的股票），形成最终的待持仓股票池。
5.  **调仓操作**：
    *   **卖出**：遍历当前持仓，如果某只股票不在新的股票池中，则使用 `order_target_percent(stock, 0)` 函数将其持仓比例调整为0（即清仓）。
    *   **买入**：遍历新的股票池，使用 `order_target_percent(stock, 1/len(pool))` 函数，将资金平均分配到每只股票上，使其持仓比例达到目标值。
6.  **效果评估**：运行回测后，通过分析收益率曲线、夏普比率、最大回撤等指标，全面评估策略的历史表现。
