## 从原始数据到监督学习框架

### 核心转换思路
我们目前拥有的数据是一个按时间序列排列的表格，包含每个时间点（T）的开盘价、收盘价、最高价、最低价等属性。

![](img/95f4446f386b1fd4d88ffdd5df938bf3_2.png)

为了建立机器学习模型，我们需要将每个时刻 T 的数据，转化为一组能够预测未来时刻（如 T+1）的特征（X）和对应的目标值（Y）。

**转换公式可以抽象为：**
对于时间点 T，我们需要：
*   **特征 X_T** = `transform(原始数据[T及T之前的历史窗口])`
*   **目标 Y_T** = `target_function(原始数据[T+1])` （例如：涨跌分类或具体价格）

这个过程的核心在于 `transform` 函数，也就是特征工程。

### 特征（因子）选择：艺术与科学的结合
特征（或称因子、预测变量）的选择是量化建模中最关键、最体现个人见解的环节。它没有绝对正确的答案，完全取决于你对所研究金融产品的理解。

**基本原则是：基于你的经验和假设，将所有你认为可能相关的因素都作为候选特征加入。** 例如，预测沪深300指数，你可能会考虑人民币汇率、黄金价格、标普500指数等。初始阶段应“广撒网”，后续再通过科学方法进行筛选和优化。

下一节课，我们将重点学习如何从大量候选特征中自动筛选出有效的子集，或对其进行变换以提升质量。记住，**输入特征（X）的质量往往比模型本身对预测结果的影响更大**。

## 常见特征（因子）计算方法

以下是构建特征库时一些常见的技术指标和计算方法。**注意：** 本课程提供的代码附件中已包含这些函数的实现，但强烈建议你先尝试自己实现，再与参考答案对比，以加深理解。

### 1. 时间滞后特征
这是最直接的特征，即使用过去若干个时间点的价格作为当前时刻的预测因子。
```python
# 伪代码示例：获取时间序列 data 在时间点 t 之前 window_size 个时间点的价格作为特征
def time_lag_features(data, t, window_size=5):
    features = data['close'][t-window_size:t]  # 获取过去5个收盘价
    return features
```

### 2. 价格变化率特征
计算当前时刻价格相对于前一时刻的变化率，这是一种标准化处理。
```python
# 伪代码示例：计算收益率
def price_change_features(data, t):
    change = (data['open'][t] - data['open'][t-1]) / data['open'][t-1]
    return change
```

### 3. 滚动窗口统计特征
计算过去一段时间窗口内的统计量（如最高价、最低价），并基于此构造新的特征。
```python
# 伪代码示例：基于滚动窗口高/低价构造特征
def rolling_stat_features(data, t, window_lambda=20):
    high_max = data['high'][t-window_lambda:t].max()
    low_min = data['low'][t-window_lambda:t].min()
    # 构造特征，例如：(当前价 - 前期均价) / (滚动区间振幅)
    feature1 = (data['close'][t] - data['close'][t-1]) / (high_max - low_min)
    # ... 可构造更多特征
    return feature1
```

### 4. 技术指标特征
金融市场中有大量成熟的技术分析指标，它们都可以作为特征来源。

*   **顺势指标（CCI）**：用于判断资产是否处于超买或超卖状态。
    *   **计算公式**：`CCI = (典型价格 - 典型价格的N期简单移动平均) / (0.015 * 典型价格的平均绝对偏差)`
    *   典型价格 = (最高价 + 最低价 + 收盘价) / 3
*   **能量潮指标（EVM）**：结合价格和成交量判断市场趋势。
*   **移动平均线（MA）**：最常用的趋势指标之一，用于平滑价格序列。
    *   **计算公式**：`MA(N) = SUM(收盘价, N) / N`
*   **变动率指标（ROC）**：衡量价格变化的速率。
*   **布林带（Bollinger Bands）**：由移动平均线和标准差通道构成，用于衡量波动性。

**作业建议（作业0）**：建立你自己的特征提取工具库（例如 `feature_utils.py`），将上述及更多技术指标实现为可复用的函数。一个全面的特征库是后续建模工作的基础。

### 5. 外部关联特征
除了标的资产自身的历史数据，任何你认为有逻辑关联的外部数据都可以作为特征。例如，其他相关指数、汇率、大宗商品价格，甚至是通过自然语言处理技术从新闻文本中提取的情感向量。

## 构建训练集与测试集

当我们有了特征计算函数后，就可以将整个时间序列数据转换为模型所需的格式。

以下是构建过程：
1.  遍历每个有效时间点 T。
2.  在时间点 T，使用历史数据计算出一组特征向量 `X_T`。
3.  将时间点 T+1 的目标值（如涨跌标签或收益率）作为 `Y_T`。
4.  将所有 `(X_T, Y_T)` 对组合起来，形成完整的训练矩阵 `X` 和目标向量 `Y`。
5.  按时间顺序划分训练集和测试集（**严禁打乱顺序**，以避免未来信息泄露）。

最终，我们得到四个核心数据集：
*   `X_train`: 训练特征矩阵，形状为 `(样本数, 特征维度)`
*   `y_train`: 训练目标向量
*   `X_test`: 测试特征矩阵
*   `y_test`: 测试目标向量

**目标值Y的编码**：
*   **回归问题**：Y 是一个实数（如下一时刻的价格或收益率）。
*   **分类问题**：Y 可以是 `{1, -1}`（涨/跌），或 `{0, 1, 2, ...}`（多分类），也可以使用 one-hot 编码。

## 核心机器学习模型介绍

有了训练集，我们就可以应用机器学习模型来学习从 X 到 Y 的映射关系。以下介绍几种在量化领域常用且有效的模型。

### 1. 线性模型与逻辑回归
这是最基本的基线模型，必须作为性能比较的基准。
*   **线性回归**：用于回归问题，试图找到一组权重，使预测值与真实值的均方误差最小。
*   **逻辑回归**：用于二分类问题，通过 sigmoid 函数将线性组合的结果映射到 [0,1] 区间，解释为概率。
    *   **激活函数公式**：`σ(z) = 1 / (1 + e^{-z})`，其中 `z = w^T * x + b`

### 2. 支持向量机（SVM）
一种强大的分类器，目标是找到能将不同类别样本分开的最大间隔超平面。也有用于回归的版本（SVR）。

### 3. 决策树与随机森林
*   **决策树**：通过一系列 if-else 规则对特征空间进行划分。优点是可解释性强（白盒模型），但单棵树容易过拟合，性能不稳定。
*   **随机森林**：集成学习方法的代表。通过构建大量决策树并让它们投票（分类）或平均（回归）来做出最终预测。它通过两种随机性来确保树之间的差异性：
    1.  **数据随机**：对训练集进行有放回抽样（Bootstrap）。
    2.  **特征随机**：在每棵树分裂节点时，只考虑特征的一个随机子集。
    *   **优点**：性能强劲，不易过拟合，能天然评估特征重要性，并行效率高。

### 4. 梯度提升树（如 XGBoost）
另一种集成学习方法，但与随机森林的并行构建不同，GBDT是串行构建的。每一棵新树都旨在拟合前一棵树预测的残差，通过不断减少残差来提升模型。
*   **特点**：在许多数据集上表现优于随机森林，是数据科学竞赛中的常胜将军。
*   **常用库**：`XGBoost`、`LightGBM`。

### 5. 神经网络（深度学习）
这里主要指多层感知机。它通过多层非线性变换来学习复杂的特征表示。
*   **结构**：输入层 -> 多个隐藏层 -> 输出层。
*   **激活函数**：常用 ReLU (`f(x)=max(0,x)`) 替代传统的 sigmoid。
*   **正则化技巧**：Dropout（随机丢弃一部分神经元）可被视为一种在神经网络内部进行集成的有效方法，能防止过拟合。
*   **训练算法**：反向传播（Backpropagation）通过链式法则高效计算梯度，是训练神经网络的核心。
*   **常用库**：`Keras`（高阶API，易于使用），底层可基于 `TensorFlow` 或 `PyTorch`。

**模型选择与调参建议**：
*   对于金融数据，随机森林和梯度提升树通常是非常可靠的选择。
*   **必须进行超参数调优**（如树的深度、数量等）。使用交叉验证来寻找最优参数组合，这是提升模型性能的关键步骤，且通常“只赚不赔”。
*   对于耗时较长的模型调参，可以考虑贝叶斯优化等更智能的搜索策略。
