在本节课中，我们将学习IPython中一个非常实用且有趣的高级功能——魔术命令。魔术命令以百分号开头，能够让我们在交互式环境中更高效地执行文件、测试代码片段以及测量性能。

## 运行外部Python文件 📂

在标准的Python命令行中，要运行一个外部Python文件，通常需要退出交互环境，然后在终端执行命令。但在IPython中，我们可以直接使用魔术命令 `%run` 来执行，无需退出。

![](img/50d5a3fcd2cd376653df7885386023f8_3.png)

以下是具体操作步骤：
1.  首先，在任意目录下创建一个简单的Python文件，例如 `hello.py`。
2.  在文件中写入代码，例如 `print(“Hello, World!”)`。
3.  在IPython交互环境中，输入命令 `%run hello.py` 即可直接运行该脚本。

![](img/50d5a3fcd2cd376653df7885386023f8_5.png)

```python
# 在IPython中执行
%run hello.py
```

## 粘贴并执行代码 📋

有时，我们可能需要从编辑器或其他地方复制一段代码到IPython中进行测试。直接粘贴可能会因为缩进等问题导致错误。为此，IPython提供了 `%paste` 魔术命令。

![](img/50d5a3fcd2cd376653df7885386023f8_7.png)

`%paste` 命令会智能地处理剪贴板中的代码，先将其内容打印出来，然后自动执行。这尤其适用于测试长代码中的特定片段。

![](img/50d5a3fcd2cd376653df7885386023f8_9.png)

```python
# 复制一段代码到剪贴板后，在IPython中执行
%paste
```

## 测量代码执行时间 ⏱️

在性能分析和优化时，我们需要精确测量代码的执行时间。虽然可以使用Python的 `time` 模块，但对于执行时间极短的代码，测量结果可能不准确（例如显示为0秒）。IPython的 `%timeit` 命令可以解决这个问题。

`%timeit` 通过多次运行目标代码并计算平均时间来获得更精确的结果，特别适合测量微小操作或函数的执行时间。

```python
import random

# 创建一个包含1万个随机数的列表
my_list = [random.random() for _ in range(10000)]

# 测量排序这个列表所需的时间
%timeit sorted(my_list)
```

执行上述命令后，`%timeit` 会输出类似 `146 µs ± 742 ns per loop` 的结果，表示平均每次循环耗时146微秒，并给出了误差范围。它会自动决定运行次数，以确保结果的可靠性。

本节课中我们一起学习了IPython的三个核心魔术命令：`%run` 用于直接运行外部脚本，`%paste` 用于粘贴并执行剪贴板中的代码，`%timeit` 用于精确测量代码段的执行时间。这些工具能显著提升我们在交互式环境中的开发和调试效率。
