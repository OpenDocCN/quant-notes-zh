# 量化面试解题的艺术：001：面条一共几个圈 🍜

在本节课中，我们将学习一道经典的量化面试题。题目涉及**期望、概率、随机过程和动态规划**的核心思想。我们将通过分析“连接面条端点形成环”的过程，一步步推导出最终形成环的期望数量。

---

## 问题描述

![](img/b9a0231f0972751901a872733e33e8a1_1.png)

碗里有100根面条，每根面条有两个端点。你被蒙上眼睛，每次从碗中**随机选择两个端点**并将它们连接起来。每一根面条的任意一端被选中的概率相同。你不断重复这个过程，直到所有端点都被连接完毕。最终形成的环的数量是随机的。我们的目标是**计算最终形成环的数量的期望值**。

---

## 问题抽象与建模

![](img/b9a0231f0972751901a872733e33e8a1_3.png)

首先，我们需要将问题抽象化，以便进行数学分析。

![](img/b9a0231f0972751901a872733e33e8a1_5.png)

### 定义关键变量

![](img/b9a0231f0972751901a872733e33e8a1_7.png)

我们定义随机变量 **`X_n`**，它表示：当碗里有 `n` 根面条时，经过上述随机连接过程后，最终**形成环的个数**。

![](img/b9a0231f0972751901a872733e33e8a1_9.png)

因此，我们的最终目标是计算 **`E[X_100]`**，即当 `n = 100` 时的期望值。

![](img/b9a0231f0972751901a872733e33e8a1_11.png)

---

![](img/b9a0231f0972751901a872733e33e8a1_13.png)

## 思路分析：状态转移

面对 `n=100` 这样的大数字，直接枚举所有可能性是不现实的。一个关键的解题思路是**状态转移**。

上一节我们定义了核心变量 `X_n`。本节中我们来看看，如何从 `n` 根面条的状态，推导到 `n-1` 根面条的状态。

![](img/b9a0231f0972751901a872733e33e8a1_15.png)

### 状态转移的核心洞察

我们从 `n` 根面条（共 `2n` 个端点）的初始状态开始。随机选择两个端点后，只有两种可能的结果：

1.  **情况A**：选中的两个端点**属于同一根面条**。将这两个端点连接后，这根面条会形成一个**独立的环**。碗里剩下的面条结构变成了 `n-1` 根独立的“面条”（可能是直的，也可能是之前连接形成的链）。因此，总环数等于 **`1 + X_{n-1}`**。
2.  **情况B**：选中的两个端点**属于不同的面条（或链）**。将这两个端点连接后，这两根面条（或链）会合并成一根更长的“面条”（或链）。碗里剩下的独立“面条”数量变为 `n-1`。因此，总环数等于 **`X_{n-1}`**。

**重要理解**：在情况B中，连接两个不同面条的端点，并不会立即产生环，只是改变了面条的拓扑结构（将两条链合并为一条），环的数量并未增加。

![](img/b9a0231f0972751901a872733e33e8a1_17.png)

## 计算转移概率

![](img/b9a0231f0972751901a872733e33e8a1_19.png)

接下来，我们需要计算上述两种情况发生的概率。

*   总共有 `2n` 个端点。
*   随机选择两个端点，总的选择方式有 **`C(2n, 2)`** 种。

**情况A（形成环）的概率**：
*   要选中同一根面条的两个端点。对于任何一根面条，只有1种方式选中它的两个端点。
*   因为有 `n` 根面条，所以总共有 `n` 种选择方式能直接形成一个环。
*   因此，概率 `P_A = n / C(2n, 2)`。

![](img/b9a0231f0972751901a872733e33e8a1_21.png)

**情况B（合并面条）的概率**：
*   剩下的所有选择都属于情况B。
*   因此，概率 `P_B = 1 - P_A = [C(2n, 2) - n] / C(2n, 2)`。

---

## 建立期望方程

根据**全期望公式**，`E[X_n]` 可以基于第一次操作后的状态来计算：

![](img/b9a0231f0972751901a872733e33e8a1_23.png)

`E[X_n] = P_A * E[1 + X_{n-1}] + P_B * E[X_{n-1}]`

将概率代入：

`E[X_n] = [n / C(2n, 2)] * (1 + E[X_{n-1}]) + [(C(2n, 2) - n) / C(2n, 2)] * E[X_{n-1}]`

化简这个方程：

`E[X_n] = n / C(2n, 2) + E[X_{n-1}]`

因为 `C(2n, 2) = n(2n-1)`，所以：

![](img/b9a0231f0972751901a872733e33e8a1_25.png)

`n / C(2n, 2) = n / [n(2n-1)] = 1 / (2n-1)`

于是我们得到了简洁的**递推关系**：

`E[X_n] = E[X_{n-1}] + 1/(2n-1)`

---

## 求解递推方程

![](img/b9a0231f0972751901a872733e33e8a1_27.png)

我们有了递推关系，现在需要一个初始条件来求解。

**初始条件**：当只有一根面条时 (`n=1`)，只能将其两端连接，必然形成一个环。所以 `E[X_1] = 1`。

利用递推关系逐步展开：

![](img/b9a0231f0972751901a872733e33e8a1_29.png)

*   `E[X_1] = 1`
*   `E[X_2] = E[X_1] + 1/(2*2-1) = 1 + 1/3`
*   `E[X_3] = E[X_2] + 1/(2*3-1) = 1 + 1/3 + 1/5`
*   ...

因此，我们可以得到通项公式：

![](img/b9a0231f0972751901a872733e33e8a1_31.png)

![](img/b9a0231f0972751901a872733e33e8a1_33.png)

`E[X_n] = 1 + 1/3 + 1/5 + ... + 1/(2n-1)`

![](img/b9a0231f0972751901a872733e33e8a1_35.png)

---

![](img/b9a0231f0972751901a872733e33e8a1_37.png)

## 答案与验证

![](img/b9a0231f0972751901a872733e33e8a1_39.png)

对于原题 `n = 100`，最终的期望值为：

![](img/b9a0231f0972751901a872733e33e8a1_41.png)

`E[X_100] = 1 + 1/3 + 1/5 + ... + 1/199`

![](img/b9a0231f0972751901a872733e33e8a1_43.png)

这是一个调和级数的奇数项求和，没有更简单的闭合形式，但可以计算其近似值。

![](img/b9a0231f0972751901a872733e33e8a1_45.png)

**简单验证**：当 `n=2` 时，公式给出 `E[X_2] = 1 + 1/3 = 4/3`。我们可以手动验证：两根面条（4个端点），共有 C(4,2)=6 种等可能的首次连接。其中，有2种情况（连接同一根面条的两端）会最终产生2个环；其余4种情况最终产生1个环。期望值 = (2*2 + 4*1)/6 = 8/6 = 4/3，与公式结果一致。

![](img/b9a0231f0972751901a872733e33e8a1_47.png)

## 核心概念总结

本节课中我们一起学习了如何解决“面条成环”的期望问题。我们运用了以下核心方法：

1.  **定义随机变量**：`X_n` 表示 `n` 根面条时的最终环数。
2.  **状态转移思想**：将复杂的 `n` 问题，分解为第一次操作后更简单的 `n-1` 问题。这是**动态规划**和**随机过程**分析的典型思路。
3.  **分类讨论**：根据第一次操作的结果（是否直接形成环），将样本空间分为两类。
4.  **全期望公式**：利用 `E[X] = Σ P(状态) * E[X|状态]` 建立方程。
5.  **递推求解**：建立并求解 `E[X_n]` 关于 `E[X_{n-1}]` 的递推关系，最终得到通项公式 `E[X_n] = Σ_{k=1}^{n} 1/(2k-1)`。

![](img/b9a0231f0972751901a872733e33e8a1_53.png)

通过这道题，我们掌握了如何将看似复杂的随机过程问题，通过抽象建模和状态转移，转化为可计算的数学表达式。
