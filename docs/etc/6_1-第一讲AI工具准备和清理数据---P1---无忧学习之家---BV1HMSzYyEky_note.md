# AI辅助量化研究：1：AI工具准备与数据清理

## 概述

在本节课中，我们将学习如何利用AI工具辅助量化研究的第一步：准备和清理数据。我们将介绍AI工具的基本概念，学习如何将数据导入AI工具，并重点掌握使用AI辅助识别和处理数据中的缺失值与极端值的方法。

## AI工具简介

上一节我们介绍了课程的整体框架，本节中我们来看看我们将要使用的主要辅助工具——AI。

生成式AI，例如百度的文心一言，是使用大量文本数据训练的深度学习模型。其核心功能是理解和生成自然语言，并能结合自身知识库，针对用户提出的问题生成相关且有意义的回答。在科研领域，AI可以辅助完成从选题、文献综述到数据分析、结果解释等一系列任务。

## Stata软件简介

在进行量化研究时，我们离不开专业的统计分析软件。Stata是一款功能强大的统计分析软件，广泛应用于社会科学、经济学等领域。与SPSS的菜单操作不同，Stata主要通过命令行进行操作，其命令大多基于英文单词的原意，学习门槛相对较低。Stata可以处理横截面、面板、时间序列等多种类型的数据，并拥有丰富的统计分析模块。

将数据准备妥当后，我们首先需要对变量有一个基本的了解。你可以直接向AI工具提问，让它生成相应的描述性统计分析代码。

例如，你可以提问：“在以上数据中，我需要`age`（年龄）和`income`（收入）两个变量进行描述性统计分析，请生成相应的Stata代码。”

![](img/6b26e4f0bd5e9808bc4196e77b393dee_32.png)

![](img/6b26e4f0bd5e9808bc4196e77b393dee_34.png)

AI通常会建议使用 `summarize` 或 `describe` 命令。例如：
```stata
summarize age income
```
或者获取更详细的信息：
```stata
summarize age income, detail
```
运行代码后，Stata会输出变量的观测值数量、均值、标准差、最小值、最大值等信息。如果你对输出结果不理解，可以将整个结果复制并再次询问AI：“以上Stata返回的结果如何解释？”AI会为你详细解读每个统计量的含义。

## 识别与处理缺失值

![](img/6b26e4f0bd5e9808bc4196e77b393dee_40.png)

数据清理的核心任务之一是处理缺失值。缺失值如果处理不当，会严重影响分析结果的准确性和可靠性。

### 缺失值的后果
*   **影响数据完整性**：导致有效样本量减少。
*   **造成选择性偏误**：如果数据缺失与研究变量相关，会导致估计偏差。
*   **降低模型精度**：导致参数估计方差增大。

### 常用处理方法
1.  **删除法**：直接删除含有缺失值的观测。适用于缺失比例极低的情况。
2.  **插补法**：用某个值（如均值、中位数）或通过模型预测来填补缺失值。
3.  **多重插补**：目前处理缺失值的主流方法。通过多次模拟生成多个完整数据集，分别进行分析后再汇总结果，能更好地反映缺失值的不确定性。

![](img/6b26e4f0bd5e9808bc4196e77b393dee_42.png)

![](img/6b26e4f0bd5e9808bc4196e77b393dee_44.png)

### 使用AI辅助处理
你可以向AI提问：“如何在Stata中识别和处理缺失值？”
AI会给出系列命令，例如：
*   **识别缺失**：`misstable summarize`
*   **均值插补**（示例）：
    ```stata
    summarize income
    replace income = r(mean) if missing(income)
    ```
*   **进行多重插补**（以`income`变量为例，使用`age`和`sex`作为预测变量）：
    ```stata
    mi set mlong
    mi register imputed income
    mi impute regress income = age sex, add(5)
    ```
    **注意**：AI生成的代码有时可能不完整或出错。例如，多重插补前需用 `mi register` 命令指定哪些变量需要插补。如果运行出错，可以将错误信息反馈给AI，请求其修正。

## 识别与处理极端值

![](img/6b26e4f0bd5e9808bc4196e77b393dee_51.png)

![](img/6b26e4f0bd5e9808bc4196e77b393dee_53.png)

极端值（异常值）是指与数据集中大多数观测值相比异常高或低的数值，它们会扭曲统计结果（如均值和回归系数）。

### 常用处理方法
1.  **删除**：直接删除被识别为极端值的观测。
2.  **替换**：用均值、中位数等替代极端值。
3.  **转换**：对变量进行数学转换（如取对数），以减小极端值的影响。
4.  **使用稳健方法**：在分析中使用稳健标准误等方法。

### 使用AI辅助处理
你可以提问：“在Stata中，如何识别并处理变量`income`中的极端值？”
AI可能提供的思路包括：
*   **基于分位数识别**（例如，将超出1.5倍四分位距的值视为极端值）：
    ```stata
    sum income, detail
    gen extreme_income = (income < r(p25) - 1.5*(r(p75)-r(p25))) | (income > r(p75) + 1.5*(r(p75)-r(p25)))
    list income if extreme_income == 1
    ```
*   **进行替换或删除**：
    ```stata
    replace income = 52597.62 if extreme_income == 1 // 用均值替换
    drop if extreme_income == 1 // 删除极端值观测
    ```
    **重要提示**：对数据进行任何修改前，建议先使用 `gen newvar = oldvar` 命令创建变量的副本，在副本上进行操作，以防原始数据被意外更改。
