## 蒙特卡罗模拟的误差与效率 ⚙️

在上一节课中，我们了解到蒙特卡罗模拟通过计算样本均值来估计期望值。该估计值的标准差为 **σ/√N**，其中 σ 是单个样本的标准差，N 是样本数量。因此，蒙特卡罗的收敛速率是 **O(1/√N)**，相对较慢。

![](img/dac766c98f16c58236d5593a25a02571_3.png)

![](img/dac766c98f16c58236d5593a25a02571_5.png)

例如，为了将欧式期权定价的相对误差降至1%以下，可能需要模拟上千万条路径，在计算上耗时较长。为了提高算法效率、缩短模拟时间，我们引入了多种**方差降低技术**。通过降低模拟方差，我们可以在相同样本量下获得更精确的估计，或者以更少的样本达到相同的精度，从而显著提升计算速度。

## 方差降低方法一：反向取路径 🔄

![](img/dac766c98f16c58236d5593a25a02571_7.png)

首先，我们来看一种直观的方差降低方法——反向取路径。

其核心思想是：对于每个生成的标准正态随机数 **W**，我们同时使用 **W** 和 **-W** 来驱动资产价格路径的模拟。具体步骤如下：
1.  生成正态随机数 **W**。
2.  计算由 **W** 驱动的资产价格路径 **X(W)**。
3.  计算由 **-W** 驱动的资产价格路径 **X(-W)**。
4.  取两者的平均值作为新的样本 **Y = (X(W) + X(-W))/2**。

![](img/dac766c98f16c58236d5593a25a02571_9.png)

![](img/dac766c98f16c58236d5593a25a02571_11.png)

通过数学推导，新样本 **Y** 的方差为：**Var(Y) = (σ²/2) * (1 + ρ)**，其中 **ρ** 是 **X(W)** 和 **X(-W)** 的相关系数。由于我们同时计算了两条路径，实际计算量翻倍。因此，有效的方差降低要求 **Var(Y) < σ²/2**，即 **ρ < 0**。这意味着 **X(W)** 和 **X(-W)** 最好呈负相关关系。

![](img/dac766c98f16c58236d5593a25a02571_13.png)

以下是几种相关关系下的效果比较：
*   **理想情况**：当 **X(W)** 是 **W** 的单调函数（例如几何布朗运动），**W** 与 **-W** 完全负相关，此时方差降低效果最好。
*   **一般情况**：当函数在某些区间内不单调，相关系数为零，此时方法效果一般。
*   **最差情况**：当 **X(W) = X(-W)**，即函数关于原点对称，此时该方法无效。

反向取路径方法简单直观，但在函数非单调时效果有限。接下来，我们探讨更通用和强大的方法。

## 方差降低方法二：控制变量法 🎯

![](img/dac766c98f16c58236d5593a25a02571_15.png)

当反向取路径效果不理想时，控制变量法是一种更强大的选择。其核心理念是：**利用一个已知精确解或低方差估计量的相关变量，来修正目标变量的估计误差。**

我们可以用一个比喻来理解：假设有两人射击，一人弹着点紧密但偏离靶心（高精度、有偏差），另一人弹着点围绕靶心但很分散（无偏、低精度）。控制变量法类似于将第一人的所有弹着点整体平移至靶心，从而获得既精确又准确的结果。

![](img/dac766c98f16c58236d5593a25a02571_17.png)

在期权定价中的应用如下：假设我们要为复杂的亚式期权定价（方差大），同时我们可以用相同的模拟路径为简单的欧式期权定价（有精确的Black-Scholes公式）。步骤如下：
1.  用蒙特卡罗模拟同一组资产价格路径，分别计算亚式期权价格估计值 **X̂** 和欧式期权价格估计值 **V̂**。
2.  用Black-Scholes公式计算欧式期权的精确价格 **V**。
3.  计算欧式期权的模拟误差 **ε = V - V̂**。
4.  假设亚式期权的真实价格 **X** 与其模拟值 **X̂** 的误差，与 **ε** 存在线性关系：**X - X̂ ≈ β * ε**。
5.  通过优化选择系数 **β**，将修正项加回：**X_cv = X̂ + β * (V - V̂)**，得到控制变量法修正后的亚式期权价格估计。

**数学推导与最优β**
我们的目标是最小化修正后估计量 **Y = X̂ + β(V - V̂)** 的方差。通过求导可得最优系数：
**β* = Cov(X̂, V̂) / Var(V̂)**
此时，最小方差为：
**Var(Y) = Var(X̂) * (1 - ρ²)**
其中 **ρ** 是 **X̂** 和 **V̂** 的相关系数。

**关键点**：
*   控制变量 **V**（此处为欧式期权）必须能与目标变量 **X**（亚式期权）使用**相同的模拟路径**。
*   控制变量需要有已知的精确解或非常准确的估计值。
*   目标变量与控制变量的相关系数 **|ρ|** 越高，方差降低效果越显著。

![](img/dac766c98f16c58236d5593a25a02571_19.png)

**Python实例：亚式期权定价**
我们使用欧式看涨期权作为控制变量，为亚式看涨期权定价。两者参数一致（行权价K=100，到期日T=1年等）。

```python
# 伪代码示例：控制变量法核心步骤
# 1. 模拟N条资产价格路径 S_paths
S_paths = simulate_gbm_paths(N, ...)

![](img/dac766c98f16c58236d5593a25a02571_21.png)

![](img/dac766c98f16c58236d5593a25a02571_23.png)

# 2. 计算亚式期权和欧式期权的蒙特卡罗估计值
asian_estimates = [asian_payoff(path) for path in S_paths]
european_estimates = [european_payoff(path) for path in S_paths]
X_hat = np.mean(asian_estimates)
V_hat = np.mean(european_estimates)

![](img/dac766c98f16c58236d5593a25a02571_25.png)

# 3. 计算欧式期权的精确解（Black-Scholes）
V_exact = black_scholes_call(...)

![](img/dac766c98f16c58236d5593a25a02571_27.png)

# 4. 计算最优beta
cov_matrix = np.cov(asian_estimates, european_estimates)
beta_star = cov_matrix[0, 1] / cov_matrix[1, 1]

![](img/dac766c98f16c58236d5593a25a02571_29.png)

# 5. 得到控制变量法修正后的价格
X_cv = X_hat + beta_star * (V_exact - V_hat)
```

![](img/dac766c98f16c58236d5593a25a02571_31.png)

![](img/dac766c98f16c58236d5593a25a02571_33.png)

在实际模拟中（10万条路径），亚式与欧式期权价格估计值的相关系数高达0.85。使用控制变量法后，蒙特卡罗误差从约0.032降至0.017，加速比（方差降低倍数）接近理论值4，效果显著。

**应用于美式期权**
将上述方法中的亚式期权替换为美式期权（使用上节课的最小二乘蒙特卡罗方法定价）。由于美式期权与欧式期权的价格相关性相对较低且不稳定，因此控制变量法的效果会打折扣，加速比可能降至1.4倍左右。

**小结**
控制变量法是效果最好的方差降低方法之一，实现相对简单，其最优系数 **β*** 有理论解。但它对“产品对”的选择非常敏感，要求控制变量与目标变量高度相关，且控制变量本身有高精度解。反向取路径可以看作是控制变量法的一个特例（固定 **β = 1/2**），虽不最优但实现更简便。

## 方差降低方法三：重点采样 🎲

![](img/dac766c98f16c58236d5593a25a02571_55.png)

![](img/dac766c98f16c58236d5593a25a02571_57.png)

接下来，我们重点介绍一个在风险管理中尤为重要的方法——重点采样。它涉及更深入的数学概念。

**核心思想**：改变概率测度，使模拟更频繁地采样到对最终结果影响重大的区域（例如，期权深度实值或虚值的区域），从而减少方差。

![](img/dac766c98f16c58236d5593a25a02571_59.png)

![](img/dac766c98f16c58236d5593a25a02571_61.png)

![](img/dac766c98f16c58236d5593a25a02571_63.png)

**数学基础：测度变换**
我们想计算期望 **E^P[f(X)]**，其中 **P** 是原始概率测度。通过引入一个新的概率测度 **Q**，并满足Radon-Nikodym导数 **dP/dQ = L(X)**，我们可以将期望改写为：
**E^P[f(X)] = E^Q[f(X) * L(X)]**
重点采样的关键在于，选择一个合适的 **Q**，使得在 **Q** 测度下，**f(X)L(X)** 的方差比在 **P** 测度下 **f(X)** 的方差更小。

**在期权定价中的应用**
以深度虚值看涨期权为例，在原始测度下，股价到期日超过行权价的概率很低，大部分模拟路径的收益为0，导致方差大、效率低。我们可以通过改变漂移率（如增加股价的期望增长率），在新测度 **Q** 下，股价更可能超过行权价，然后通过乘以似然比 **L** 进行修正，从而获得更有效的估计。

![](img/dac766c98f16c58236d5593a25a02571_65.png)

![](img/dac766c98f16c58236d5593a25a02571_67.png)

**在信用风险模型中的应用**
重点采样在计算信用组合的损失分布尾部分位数（如VaR）时极为有效。标准的蒙特卡罗模拟难以捕捉极端违约事件。通过重点采样，我们可以“扭曲”违约概率，使模拟更倾向于产生大额损失的场景，再通过似然比调整回真实的概率分布，从而用更少的模拟次数准确估计尾部分位数。

## 其他方法与总结 🏁

除了上述方法，还有：
*   **低差异序列**：如Sobol序列，用确定性低差异点集代替伪随机数，能更均匀地填充空间，加速收敛。
*   **矩匹配**：对模拟样本进行变换，使其前几阶矩（如均值、方差）与理论矩匹配。
