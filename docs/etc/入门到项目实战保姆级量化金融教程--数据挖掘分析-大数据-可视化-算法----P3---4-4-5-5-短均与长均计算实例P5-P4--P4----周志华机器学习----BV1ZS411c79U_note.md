## 计算短期与长期移动平均线
上一节我们介绍了移动平均线的概念，本节中我们来看看如何为股票数据计算短期和长期移动平均线。

![](img/ec695581ae4f63284e5c2934208c32d5_8.png)

移动平均线是技术分析中常用的指标，用于平滑价格数据，识别趋势。短期移动平均线对价格变化更敏感，而长期移动平均线则反映更长期的趋势。

![](img/ec695581ae4f63284e5c2934208c32d5_10.png)

以下是计算步骤：
1.  选择目标股票数据。我们将使用苹果公司的股票数据。
2.  计算短期移动平均线。通常使用5天或10天的窗口。
3.  计算长期移动平均线。通常使用20天或30天的窗口。

由于我们的数据集时间跨度较长（2010年至2018年），为了更清晰地展示交叉点，我们将适当增大窗口期。

![](img/ec695581ae4f63284e5c2934208c32d5_12.png)

```python
# 假设 `data` 是一个包含苹果股价的DataFrame，列名为 ‘AAPL’
# 计算10日短期移动平均线
data[‘M1’] = data[‘AAPL’].rolling(window=10).mean()

![](img/ec695581ae4f63284e5c2934208c32d5_14.png)

![](img/ec695581ae4f63284e5c2934208c32d5_16.png)

![](img/ec695581ae4f63284e5c2934208c32d5_18.png)

# 计算30日长期移动平均线
data[‘M2’] = data[‘AAPL’].rolling(window=30).mean()
```

## 识别黄金交叉与死亡交叉
图表绘制完成后，我们可以开始寻找关键的交叉信号。

![](img/ec695581ae4f63284e5c2934208c32d5_44.png)

以下是两种交叉信号的定义：
*   **黄金交叉**：短期移动平均线从下方向上穿越长期移动平均线。这通常被视为买入信号，预示着趋势可能转为上涨。
*   **死亡交叉**：短期移动平均线从上方向下穿越长期移动平均线。这通常被视为卖出信号，预示着趋势可能转为下跌。

![](img/ec695581ae4f63284e5c2934208c32d5_46.png)

观察图表：
1.  当绿色线（短期均线）向下穿越红色线（长期均线）时，形成了一个**死亡交叉**。此时应考虑卖出或避免买入。
2.  当绿色线（短期均线）向上穿越红色线（长期均线）时，形成了一个**黄金交叉**。此时可以考虑买入。

![](img/ec695581ae4f63284e5c2934208c32d5_48.png)

## 使用数值信号增强可视化
为了让交叉信号更加一目了然，我们可以创建一个数值信号列来明确标识短期均线是高于还是低于长期均线。

![](img/ec695581ae4f63284e5c2934208c32d5_50.png)

![](img/ec695581ae4f63284e5c2934208c32d5_52.png)

我们将使用 `np.where` 函数来创建这个信号列。当短期均线大于长期均线时，标记为1；小于时，标记为-1。

![](img/ec695581ae4f63284e5c2934208c32d5_54.png)

![](img/ec695581ae4f63284e5c2934208c32d5_56.png)

![](img/ec695581ae4f63284e5c2934208c32d5_58.png)

```python
import numpy as np

![](img/ec695581ae4f63284e5c2934208c32d5_60.png)

# 创建信号列
data[‘position’] = np.where(data[‘M1’] > data[‘M2’], 1, -1)
```

![](img/ec695581ae4f63284e5c2934208c32d5_62.png)

由于信号值（1和-1）与股价数值相差巨大，直接绘制在同一坐标轴上会看不清楚。因此，我们使用双Y轴来绘图。

```python
fig, ax1 = plt.subplots(figsize=(12, 6))

![](img/ec695581ae4f63284e5c2934208c32d5_64.png)

![](img/ec695581ae4f63284e5c2934208c32d5_66.png)

# 第一个Y轴：绘制股价和移动平均线
data[[‘AAPL’, ‘M1’, ‘M2’]].tail(250).plot(ax=ax1)

# 创建第二个Y轴
ax2 = ax1.twinx()
# 第二个Y轴：绘制信号线
data[‘position’].tail(250).plot(ax=ax2, color=‘grey’, linestyle=‘--’)

![](img/ec695581ae4f63284e5c2934208c32d5_68.png)

![](img/ec695581ae4f63284e5c2934208c32d5_70.png)

plt.show()
```

![](img/ec695581ae4f63284e5c2934208c32d5_72.png)

![](img/ec695581ae4f63284e5c2934208c32d5_74.png)

在新的图表中，灰色的虚线代表信号线。当信号线从-1跃升到1时，对应着**黄金交叉**（买入信号）。当信号线从1下降到-1时，对应着**死亡交叉**（卖出信号）。这使得交叉点的识别变得非常直观。

![](img/ec695581ae4f63284e5c2934208c32d5_76.png)
