# Python量化交易：P28：因子数据预处理

在本节课中，我们将学习如何对量化交易中的因子数据进行预处理。因子是影响股票收益的指标，例如市净率或营收增长率。处理这些数据是构建有效交易策略的关键步骤。我们将介绍三个核心预处理步骤：去极值、标准化和中性化，并解释每个步骤的目的与实现方法。

## 数据预处理三步走

上一节我们介绍了因子的基本概念。本节中，我们来看看处理因子数据的具体流程。以下是数据预处理的三个核心步骤：

1.  **去极值**：处理数据中的异常值或离群点。
2.  **标准化**：将不同量纲的因子数据转换到同一尺度。
3.  **中性化**：消除因子数据中与市场整体趋势相关的部分。

前两个步骤在机器学习中很常见，而中性化则是量化因子分析中的特殊步骤。接下来，我们将按照这个顺序逐一讲解。

## 第一步：去极值方法

![](img/ab85f8b102afc6a246e624c7f3153ee6_1.png)

去极值的目的是处理那些偏离主体数据分布过远的异常值。直接删除这些数据可能造成信息损失，因此我们通常采用“拉回”到边界的方法，而不是直接丢弃。

![](img/ab85f8b102afc6a246e624c7f3153ee6_3.png)

以下是几种常见的去极值方法，我们将重点介绍**百分位去极值法**。

### 分位数概念

在介绍百分位法之前，需要理解分位数的概念。分位数是将数据按比例划分的点。最常用的是四分位数。

![](img/ab85f8b102afc6a246e624c7f3153ee6_5.png)

*   **中位数 (Q2)**：将数据从小到大排列后，处于中间位置的值。公式上，对于有序数据集 `data`，若长度为奇数 `n`，中位数为 `data[(n-1)/2]`；若为偶数，则为 `(data[n/2 - 1] + data[n/2]) / 2`。
*   **第一四分位数 (Q1)**：数据中25%位置的值。
*   **第三四分位数 (Q3)**：数据中75%位置的值。

![](img/ab85f8b102afc6a246e624c7f3153ee6_7.png)

### 百分位去极值法

![](img/ab85f8b102afc6a246e624c7f3153ee6_9.png)

该方法利用分位数设定数据的正常范围边界，并将超出边界的极值“拉回”到边界上。

![](img/ab85f8b102afc6a246e624c7f3153ee6_11.png)

![](img/ab85f8b102afc6a246e624c7f3153ee6_13.png)

具体步骤如下：
1.  计算因子数据的 **Q1** 和 **Q3**。
2.  计算四分位距 **IQR = Q3 - Q1**。
3.  设定正常值范围的下界 **lower_bound = Q1 - k * IQR** 和上界 **upper_bound = Q3 + k * IQR**。`k` 是一个常数，通常取 1.5。
4.  遍历所有数据点，将任何小于 `lower_bound` 的值设置为 `lower_bound`，将任何大于 `upper_bound` 的值设置为 `upper_bound`。

![](img/ab85f8b102afc6a246e624c7f3153ee6_15.png)

```python
import numpy as np

![](img/ab85f8b102afc6a246e624c7f3153ee6_17.png)

def percentile_winsorize(data, k=1.5):
    """
    使用百分位法对数据进行去极值处理。
    :param data: 因子数据，numpy数组或列表。
    :param k: 范围系数，默认为1.5。
    :return: 处理后的数据。
    """
    q1 = np.percentile(data, 25)
    q3 = np.percentile(data, 75)
    iqr = q3 - q1
    lower_bound = q1 - k * iqr
    upper_bound = q3 + k * iqr
    # 将极值拉回边界
    data_winsorized = np.clip(data, lower_bound, upper_bound)
    return data_winsorized
```

### 代码示例

![](img/ab85f8b102afc6a246e624c7f3153ee6_19.png)

我们以苹果公司股价数据为例，演示如何应用百分位去极值法。

```python
# 导入必要的库
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 读取示例数据（这里用股价代替因子数据以便演示）
# 假设‘price’列是我们需要处理的因子
data = pd.read_csv(‘apple_stock.csv’)
factor_data = data[‘Close’].values

# 应用百分位去极值
k = 1.5
q1 = np.percentile(factor_data, 25)
q3 = np.percentile(factor_data, 75)
iqr = q3 - q1
lower_bound = q1 - k * iqr
upper_bound = q3 + k * iqr

![](img/ab85f8b102afc6a246e624c7f3153ee6_21.png)

print(f“Q1: {q1:.2f}, Q3: {q3:.2f}“)
print(f“IQR: {iqr:.2f}“)
print(f“正常值范围: [{lower_bound:.2f}, {upper_bound:.2f}]“)

# 处理数据
processed_data = np.clip(factor_data, lower_bound, upper_bound)

# 可视化处理前后对比
plt.figure(figsize=(12, 5))
plt.subplot(1, 2, 1)
plt.hist(factor_data, bins=50, edgecolor=‘black’)
plt.title(‘原始因子数据分布’)
plt.xlabel(‘数值’)
plt.ylabel(‘频次’)

plt.subplot(1, 2, 2)
plt.hist(processed_data, bins=50, edgecolor=‘black’, color=‘orange’)
plt.title(‘去极值后数据分布’)
plt.xlabel(‘数值’)
plt.ylabel(‘频次’)
plt.tight_layout()
plt.show()
```

![](img/ab85f8b102afc6a246e624c7f3153ee6_23.png)

![](img/ab85f8b102afc6a246e624c7f3153ee6_25.png)

运行以上代码，可以观察到极端值被限制在由 `[lower_bound, upper_bound]` 定义的区间内，数据的整体分布更加集中，这有助于后续建模的稳定性。

## 总结

![](img/ab85f8b102afc6a246e624c7f3153ee6_27.png)

本节课中，我们一起学习了量化交易中因子数据预处理的第一步：**去极值**。我们重点讲解了**百分位去极值法**，该方法通过计算数据的四分位数来确定正常值范围，并将异常值调整至边界。这种方法简单有效，是因子预处理中的常用技术。处理后的数据减少了极端值对模型造成的干扰，为后续的标准化和中性化步骤奠定了良好的基础。
