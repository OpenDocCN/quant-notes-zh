## 投资股票的益处 💰

![](img/654705fea9377b14e4e6a47fb47e0980_2.png)

上一节我们介绍了股票的基本定义和类型，本节中我们来看看投资股票能带来哪些具体的好处。

投资股票可以获得多种形式的收益，主要包括以下几点：

*   **分红**：投资者可以获得公司派发的现金红利。
*   **配股与送股**：公司可能向股东增发新股或赠送股票。
*   **交易获利**：通过买卖股票，可以直接从股价波动中获利了结。
*   **投资门槛低**：股票所需的投资本金相对较少。
*   **流动性好**：股票的交易周期短，易于变现。
*   **抗通胀**：股票资产通常不会随着通货膨胀而贬值，具有一定的保值增值属性。

## 常见金融投资标的对比 📊

![](img/654705fea9377b14e4e6a47fb47e0980_4.png)

了解了股票的好处后，我们将其与其他常见的金融投资工具进行对比，以便更全面地认识其特点。

以下是八种常见金融投资标的在风险、收益、本金需求和变现性方面的简要对比：

*   **定期存款**：风险低，收益低，所需本金少，变现性中等。
*   **基金**：风险中到高，收益较高，所需本金少，变现性高。
*   **房地产**：风险中等，收益中等，所需本金最大，变现时间最长。
*   **股票**：风险高，收益高，所需本金不定（取决于股价），变现性高。

此外，还有期货、债券等投资标的，此处不详细展开，感兴趣的同学可以自行复习。

## 基于风险与收益的股票分类 🏷️

最后，我们根据风险与收益的不同，对股票进行更细致的分类。这有助于我们在投资时根据自身风险偏好进行选择。

根据风险与收益特征，股票通常可分为以下几类：

*   **蓝筹股**：指业绩优良、在行业内占据龙头地位的大型公司股票。通常市值巨大（例如5000亿人民币以上）。示例：贵州茅台。
*   **白马股**：指长期业绩稳定、信息透明、回报率高的公司股票。市值通常低于蓝筹股（例如1000亿至3000亿人民币），多属于消费行业。示例：海尔智家。
*   **成长股**：指在特定时期内，营业收入和利润增长速度远超行业平均水平的公司股票，多出现在高科技行业。示例：东方财富（在2020-2021年期间表现突出）。
*   **周期股**：指公司业绩和股价随经济周期波动而显著变化的股票，常见于房地产、制造业、运输业等。示例：万科A。
*   **题材/概念股**：指因某一市场热点、政策或概念（如元宇宙、人工智能）而被炒作，股价短期波动剧烈的股票，其股价与公司当前业绩关联度可能不高。示例：宋城演艺（曾因元宇宙概念被炒作）。
