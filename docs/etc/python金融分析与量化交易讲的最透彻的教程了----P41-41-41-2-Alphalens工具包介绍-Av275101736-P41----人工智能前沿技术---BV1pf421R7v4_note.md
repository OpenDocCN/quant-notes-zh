## 工具包简介与获取

上一节我们介绍了量化分析的基本流程，本节中我们来看看如何利用现成工具提升效率。

**Alphalens** 是一个专门用于因子分析的Python工具包。它集成了因子计算、绩效评估和可视化功能。

以下是关于Alphalens的基本信息：
*   **GitHub链接**：可以通过其GitHub页面获取源代码和详细说明。
*   **官方文档**：提供了完整的使用指南和API参考。

![](img/03db897c30b6d407b0a73fa735eacf43_5.png)

安装Alphalens非常简单，只需在命令行中执行以下命令：
```bash
pip install alphalens
```
如果你在本地环境进行学习，可以执行上述命令进行安装。不过，在本课程提供的在线量化平台中，该工具包已经预先安装好了，我们可以直接使用。

## 在线研究环境介绍

之前我们介绍了在平台上进行策略回测，本节我们来看看如何进行因子研究。

由于直接获取和处理金融数据较为复杂，我们将在课程提供的在线量化平台中编写代码。这个平台已经集成了所需的数据和Alphalens工具包。

在平台界面左侧，找到并点击 **“投资研究”** 选项。这个环境类似于Jupyter Notebook，允许我们在平台的服务器上直接编写和运行Python代码。

![](img/03db897c30b6d407b0a73fa735eacf43_21.png)

以下是进入研究环境的步骤：
1.  点击“投资研究”。
2.  新建一个Python 3笔记本。
3.  将笔记本重命名为“因子分析”或其他你喜欢的名称。

![](img/03db897c30b6d407b0a73fa735eacf43_23.png)

后续的所有代码都将在该研究笔记本中编写和运行。课程会提供完整的代码，你可以直接上传复制到你的平台中，也可以跟随视频一步步编写。
