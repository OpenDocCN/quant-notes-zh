## 线段的定义与构成 📈

上一节我们介绍了笔的构成，本节中我们来看看由笔构成的更高级结构——线段。

缠论中的线段由相邻的笔构成。例如，一个上升线段可以由上升笔AB、下降笔BC和上升笔CD构成。这三笔之间必须有重合的部分，即A点低于D点。此时，这三笔构成一个缠论中的上升线段AD。

同理，一个下降线段也由三笔构成，但其中有两笔是下降笔，且A点需要高于D点。

和分型不一定能构成笔一样，笔也不一定能构成线段。例如，如果两笔之间没有重合的区域，则无法构成线段。

以下是构成线段的两个核心条件：
*   必须由**三笔**构成。
*   这三笔必须**相邻**且**互相之间有重合部分**。

## 线段的延续与终结 🔄

![](img/97db3675c58381e127be88640d9094d4_3.png)

线段一旦形成，并非一成不变，它会根据后续的走势进行延续或终结。

在上升趋势中，如果后续出现新的上升笔，且其终点高于原线段的终点，则线段会向上延伸。例如，上升线段AD之后，若出现上升笔EF且F点高于D点，则线段延伸为AF。

![](img/97db3675c58381e127be88640d9094d4_5.png)

线段不会无限延续。当出现一个**反方向的线段**时，原线段即被终结。例如，上升线段AH之后，如果出现一个由三笔构成的、有效的下降线段HK，那么上升线段AH就被终结。

做个简单的总结：三个连续且互相有重合部分的笔构成线段。线段延续原有趋势，直到出现反方向的线段作为终结。最终，所有线段两两相邻，并呈现出“上升线段 - 下降线段 - 上升线段”的循环。

![](img/97db3675c58381e127be88640d9094d4_7.png)

这与笔的构建过程非常相似。需要说明的是，缠论中处理线段的方法有多种，这里选取了其中一种较易理解的方式进行讲解。
