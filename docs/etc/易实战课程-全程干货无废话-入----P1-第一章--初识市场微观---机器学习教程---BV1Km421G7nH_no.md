## 什么是交易？
交易的本质是买卖双方在一个提供交易的场所进行交换。卖家提供商品，买家支付金钱。在现代金融市场中，交易场所就是交易所。虽然我们常看到交易所现场有交易员在报价，但实际上，当今市场80%以上的交易都是由计算机程序自动完成的。

## 什么是量化投资？
量化投资的历史并不长。真正的量化交易大约始于1972年。在此之前，交易大多是主观的，例如著名的股票大作手利弗莫尔使用的技术指标。量化投资进入中国大约是近十年的事情，随着2012年左右期货交易所API的开放，量化投资才开始在中国逐渐发展起来。

一个相对完善的现代量化基金通常由以下几个部门组成：
*   **数据处理部门**：负责处理市场行情、基本面数据和另类数据。
*   **特征分析部门**：负责将原始数据加工成可用于策略研究的特征，类似于机器学习中的特征工程。
*   **策略研究部门**：负责利用特征构建能够产生超额收益的交易策略。
*   **回测部门**：对研究出的策略进行严格测试，验证其在不同参数和市场环境下的稳定性。
*   **开发部署部门**：将策略代码转化为高效的生产环境代码（如C++），并部署到托管服务器上。
*   **投资组合管理部门**：负责管理整个策略组合，评估团队表现，确保稳定盈利。

## 深入了解期货合约：以螺纹钢为例
在交易任何一个品种前，必须了解其基本规则。以上海期货交易所的螺纹钢期货为例：

![](img/a22354a1706a6c4809559552bdf9e3b2_14.png)

*   **交易品种**：螺纹钢
*   **交易单位**：10吨/手
*   **报价单位**：元（人民币）/吨
*   **最小变动价位**：1元/吨
*   **涨跌停板幅度**：上一交易日结算价的±3%
*   **合约月份**：1-12月
*   **交易时间**：上午9:00-11:30，下午1:30-3:00（无夜盘特殊时期）
*   **最后交易日**：合约月份的15日（遇法定假日顺延）
*   **最低交易保证金**：合约价值的5%

![](img/a22354a1706a6c4809559552bdf9e3b2_16.png)

![](img/a22354a1706a6c4809559552bdf9e3b2_18.png)

**核心概念：保证金交易**
期货交易采用保证金制度，这是一种杠杆交易。例如，螺纹钢现价3500元/吨，1手（10吨）价值35000元。若保证金比例为5%，则只需支付1750元即可持有1手合约。杠杆放大了收益，同时也放大了风险。盈亏计算公式为：
`盈亏 = (平仓价格 - 开仓价格) × 合约方向 × 合约乘数`
其中，合约方向：买入为1，卖出为-1；合约乘数：螺纹钢为10。

## 市场微观结构核心：订单簿与订单类型
**订单簿** 反映了当前市场上所有买卖订单在不同价位上的分布情况。一个完整的订单簿包含从“买一”到“买五”、“卖一”到“卖五”等多个档位的价格和挂单量。

![](img/a22354a1706a6c4809559552bdf9e3b2_28.png)

![](img/a22354a1706a6c4809559552bdf9e3b2_30.png)

**买一**：当前市场最高的买入报价。
**卖一**：当前市场最低的卖出报价。
**最新价**：最近一笔成交的价格。

国内交易所提供的行情数据通常是每500毫秒一次的切片数据，并非实时连续的订单簿。

**订单类型主要有两种**：
*   **限价单**：指定一个价格进行买卖。例如，当前卖一为3331，你挂出3330的买入限价单，则不会立即成交，直到市场价格跌至3330或以下。
*   **市价单**：以当前市场上最好的价格立即成交。例如，以涨停价发出买入市价单，会立即与当前卖一档的订单成交。市价单能保证成交，但可能产生滑点。

## 从Tick数据到K线：理解市场行情
交易所推送的原始数据称为Tick数据。每一条Tick数据包含时间、最新价、成交量、买卖盘口（买一价量、卖一价量）等信息。

![](img/a22354a1706a6c4809559552bdf9e3b2_44.png)

**持仓量** 是一个重要概念，它代表市场上未平仓的合约总量。多仓总量永远等于空仓总量。持仓量的变化反映了市场参与者是开立新仓还是平掉旧仓。

以下是Tick数据包含的主要字段示例：
```python
# 示例字段
‘TradingDay‘, # 交易日
‘InstrumentID‘, # 合约代码
‘LastPrice‘, # 最新价
‘Volume‘, # 成交量
‘BidPrice1‘, ‘BidVolume1‘, # 买一价，买一量
‘AskPrice1‘, ‘AskVolume1‘, # 卖一价，卖一量
‘OpenInterest‘, # 持仓量
‘UpdateTime‘, ‘UpdateMillisec‘ # 时间戳（含毫秒）
```

我们常见的K线图（如1分钟K线、日K线）是由Tick数据聚合而成的。一根K线包含四个关键价格：开盘价、最高价、最低价、收盘价。

## 本节课作业与实践
**核心作业**：编写一个函数，将原始的Tick数据合成指定周期的K线数据（如1分钟K线）。

![](img/a22354a1706a6c4809559552bdf9e3b2_60.png)

![](img/a22354a1706a6c4809559552bdf9e3b2_62.png)

![](img/a22354a1706a6c4809559552bdf9e3b2_64.png)

![](img/a22354a1706a6c4809559552bdf9e3b2_66.png)

你需要考虑以下几点：
1.  **时间处理**：正确处理交易所时间戳（含毫秒），并按照严格的交易时间段进行切片（注意盘中休市时间）。
2.  **数据聚合**：对每个时间切片内的Tick数据，计算开盘价、最高价、最低价、收盘价、成交量、成交额等。
3.  **稳健性**：处理无交易数据的时段（产生“空K线”或向前填充）。
4.  **扩展性**：函数应能适应不同品种（股票、期货）和不同周期（1分钟、5分钟）。

**工具建议**：使用Pandas库的`resample`功能可以高效完成此任务。
