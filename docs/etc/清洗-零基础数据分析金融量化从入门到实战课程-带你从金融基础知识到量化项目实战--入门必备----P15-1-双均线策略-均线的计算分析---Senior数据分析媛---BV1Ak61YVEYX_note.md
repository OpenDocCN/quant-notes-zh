## 均线概念与计算

![](img/c834bd9cf6b0aed9983fcad22cddc179_7.png)

数据准备完成后，我们进入核心环节：计算移动平均线。首先，我们来理解什么是均线。

在股票分析中，对于每一个交易日，都可以计算出前N天收盘价的移动平均值。将这些移动平均值连接起来形成的线，就称为N日移动平均线，简称均线。常用的均线有5日、10日、30日、60日等。

其计算公式为：
**MA = (C1 + C2 + C3 + … + CN) / N**
其中，MA代表移动平均值，C1到CN代表连续N个交易日的收盘价。

例如，要计算5日均线，我们需要：
1.  计算第一个5天（第1天到第5天）收盘价的平均值，得到点A。
2.  计算第二个5天（第2天到第6天）收盘价的平均值，得到点B。
3.  以此类推，计算出所有可能的5日平均值点。
4.  将这些点（A, B, C…）连接起来，形成的线就是5日均线。

理解了概念后，我们使用pandas来计算贵州茅台股票的5日均线和30日均线。计算均线主要使用`.rolling()`方法。

![](img/c834bd9cf6b0aed9983fcad22cddc179_9.png)

以下是计算步骤：

```python
# 计算5日移动平均线 (MA5)
ma5 = df[‘close’].rolling(5).mean()

# 计算30日移动平均线 (MA30)
ma30 = df[‘close’].rolling(30).mean()
```

这里，`df[‘close’].rolling(5)`会依次选取第一个5天、第二个5天……的收盘价序列，接着`.mean()`方法计算这些序列的平均值。由于前4天无法构成一个完整的5日周期，所以`ma5`序列的前4个值为NaN（非数字）。同理，`ma30`序列的前29个值为NaN。

## 均线可视化

计算出均线数据后，我们可以将其可视化，以便直观观察两条均线的走势和交叉情况。这里我们使用matplotlib库进行简单绘图。

```python
import matplotlib.pyplot as plt
# 确保图表在Notebook内显示
%matplotlib inline

# 绘制5日均线
plt.plot(ma5[50:80], label=‘MA5’)
# 绘制30日均线
plt.plot(ma30[50:80], label=‘MA30’)

plt.legend() # 显示图例
plt.show()
```

为了更清晰地展示均线之间的交叉关系，我们选取了第50到第80个交易日的数据进行绘图。图中两条线分别代表5日均线和30日均线，它们的交叉点将是后续制定交易信号的关键。
