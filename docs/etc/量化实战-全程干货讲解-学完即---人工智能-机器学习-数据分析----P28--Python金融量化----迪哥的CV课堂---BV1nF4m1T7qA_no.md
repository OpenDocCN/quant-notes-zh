## 因子数据简介

上一节我们介绍了预处理的重要性，本节中我们来看看什么是因子数据。因子可以理解为影响最终投资决策的标准或指标。

例如，在选股时，投资者可能会依据某些标准进行筛选。这个标准就是因子。常见的因子包括市净率，通常认为较低的市净率股票更具投资价值。另一个例子是营收增长率，高增长率的公司可能被视为优质投资标的。

简单来说，对最终结果有影响的指标，就是我们要讨论的因子。在量化模型中，我们通常有多个因子（X1, X2, X3...）来预测收益（Y）。本节课的目标是学习如何处理这些因子数据，以便在后续的建模中更有效地选择股票，提升收益。

## 预处理三步走策略

拿到因子数据后，不能直接用于建模，需要进行预处理。以下是三个核心的预处理步骤：

1.  **去极值**：处理数据中的离群点或异常值。
2.  **标准化**：统一不同因子数据的数值范围，使其具有可比性。
3.  **中性化**：消除因子中与某些系统性风险（如行业、市值）相关的偏差。

前两个步骤在机器学习和数据挖掘中很常见，而中性化在因子策略分析中尤为重要。接下来，我们将按照这个顺序，详细讲解每一步的具体做法。

## 第一步：去极值处理

![](img/9bb4cc20c880b397ddade306cc86e247_9.png)

去极值的目标不是简单地删除异常数据点，而是将它们“拉回”到合理的边界内。例如，如果一个数值超过了我们设定的上界，就将其修正为上界值。

### 分位数去极值法

分位数是描述数据分布位置的关键统计量。最熟悉的分位数是中位数，它代表数据集的中间值。与均值相比，中位数对极端值不敏感，更能反映数据的中心趋势。

![](img/9bb4cc20c880b397ddade306cc86e247_19.png)

在分位数去极值法中，我们主要关注三个分位点：
*   **Q1 (第一四分位数)**：代表数据中25%位置的值。
*   **Q2 (中位数)**：代表数据中50%位置的值。
*   **Q3 (第三四分位数)**：代表数据中75%位置的值。

其核心思想是设定一个基于分位数的边界，将超出边界的数据替换为边界值。常用的边界计算公式为：
**上界 = Q3 + k * (Q3 - Q1)**
**下界 = Q1 - k * (Q3 - Q1)**
其中，`k` 是一个常数，通常取 1.5 或 3。`(Q3 - Q1)` 称为**四分位距**。
