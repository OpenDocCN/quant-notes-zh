## 数组形状变换 🔄

上一节我们介绍了数组的索引与切片，本节中我们来看看如何改变数组的形状。数组的`shape`属性描述了其维度结构，例如一个五行六列的二维数组。`reshape`方法可以改变数组的形状，但新形状必须能容纳原数组的所有元素。

**核心操作：**
```python
import numpy as np
arr = np.random.rand(5, 6)  # 创建一个5行6列的二维数组
print(arr.shape)  # 输出: (5, 6)

# 将二维数组变形为一维数组
arr_1d = arr.reshape(30)  # 5*6=30，必须匹配总元素数
print(arr_1d.shape)  # 输出: (30,)

# 将一维数组变形为多维数组
arr_new = arr_1d.reshape(2, 15)  # 变形为2行15列
print(arr_new.shape)  # 输出: (2, 15)
```
变形操作在基础数据分析中不常用，但在机器学习的数据预处理中可能会用到。

## 数组拼接操作 🔗

接下来，我们学习如何将多个数组拼接在一起。拼接操作可以将数组在横向（按行）或纵向（按列）进行连接。

**核心函数：** `np.concatenate()`
以下是拼接操作的关键点：
*   **方向参数`axis`**：`axis=0`表示纵向拼接（按列），`axis=1`表示横向拼接（按行）。
*   **维度要求**：只能对**相同维度**的数组进行拼接。
*   **形状问题**：如果两个数组维度相同但行或列数不一致，拼接可能会失败，需要自行尝试理解错误信息。

```python
arr = np.random.rand(5, 6)
# 纵向拼接（按列）
arr_vstack = np.concatenate((arr, arr), axis=0)
print(arr_vstack.shape)  # 输出: (10, 6)

# 横向拼接（按行）
arr_hstack = np.concatenate((arr, arr), axis=1)
print(arr_hstack.shape)  # 输出: (5, 12)
```
拼接的一个典型应用是图像处理，例如将多张图片拼合成宫格图。

## 数组聚合操作 📉

聚合操作是指对数组中的元素进行统计计算，如求和、求极值、求平均值等。这些操作可以针对整个数组，也可以沿着特定的轴（行或列）进行。

以下是常用的聚合方法：
*   **`.sum()`**: 计算所有元素之和。指定`axis`参数可计算每行(`axis=1`)或每列(`axis=0`)的和。
*   **`.min()` / `.max()`**: 查找最小值或最大值。
*   **`.mean()`**: 计算平均值。

```python
arr = np.random.rand(5, 6)
print(arr.sum())          # 所有元素之和
print(arr.max(axis=0))    # 每一列的最大值
print(arr.mean(axis=1))   # 每一行的平均值
```

## 常用数学与统计函数 🧮

NumPy提供了丰富的数学函数，可以对数组进行逐元素计算。

**常用数学函数：**
*   `np.sin(arr)`, `np.cos(arr)`, `np.tan(arr)`: 计算正弦、余弦、正切。
*   `np.round(arr, decimals=n)`: 四舍五入到指定小数位`n`。

**常用统计函数：**
*   `np.amin(arr, axis)`, `np.amax(arr, axis)`: 沿轴求最小/最大值（功能同上聚合操作）。
*   `np.ptp(arr, axis)`: 沿轴计算最大值与最小值的差（极差）。
*   **`np.std(arr, axis)`**: 计算**标准差**，衡量数据分散程度。
*   **`np.var(arr, axis)`**: 计算**方差**，标准差的平方。

**标准差公式：**
标准差 σ 的计算公式为：
`σ = sqrt( Σ (xi - μ)² / N )`
其中，`xi` 是每个数据点，`μ` 是数据集的均值，`N` 是数据点总数。

```python
arr = np.array([85, 90, 78, 92, 88])  # 假设是某次考试成绩
print(np.std(arr))  # 计算成绩的标准差
print(np.var(arr))  # 计算成绩的方差
# 方差值约等于标准差的平方
```

## 矩阵操作 ⬛

最后，我们学习矩阵的基本操作。在NumPy中，二维数组可以视作矩阵。

**矩阵创建与转置：**
```python
# 创建单位矩阵
identity_matrix = np.eye(3)
print(identity_matrix)

![](img/83537f9373ac99bb1288227fb53ea311_3.png)

# 矩阵转置（行变列，列变行）
arr = np.array([[1, 2], [3, 4]])
print(arr.T)  # 输出: [[1, 3], [2, 4]]
```

**矩阵乘法：**
矩阵乘法是线性代数的核心，规则是“行乘列”。使用`np.dot(A, B)`或`A @ B`进行计算。
给定两个矩阵：
A = [[2, 1], [4, 3]]
B = [[1, 2], [1, 0]]
它们的乘积 C = A · B 计算如下：
C[0,0] = 2*1 + 1*1 = 3
C[0,1] = 2*2 + 1*0 = 4
C[1,0] = 4*1 + 3*1 = 7
C[1,1] = 4*2 + 3*0 = 8
因此，C = [[3, 4], [7, 8]]

![](img/83537f9373ac99bb1288227fb53ea311_5.png)

```python
A = np.array([[2, 1], [4, 3]])
B = np.array([[1, 2], [1, 0]])
C = np.dot(A, B)  # 或使用 C = A @ B
print(C)  # 输出: [[3 4], [7 8]]
```
