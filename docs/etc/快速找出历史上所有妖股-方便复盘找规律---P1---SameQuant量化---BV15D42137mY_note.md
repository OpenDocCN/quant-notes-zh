## 课程概述

“妖股”通常指在短期内出现连续、大幅上涨的股票。为了系统性地研究它们，我们需要一个明确的定义和一套可执行的筛选方法。本节课程将介绍一个基于涨停板次数的量化定义，并展示如何用Python代码批量筛选出所有符合条件的历史妖股，最终生成可供复盘分析的列表。

## 妖股的量化定义

![](img/2f593b51d1c66dee5afa424f24ffb84f_5.png)

![](img/2f593b51d1c66dee5afa424f24ffb84f_7.png)

我们对“妖股”的操作性定义基于两个核心条件：
1.  在最近10个交易日内，涨停次数大于或等于8次。
2.  在最近3个交易日内，涨停次数大于或等于2次。

只有同时满足以上两个条件的股票，我们才将其标记为“妖股”。这个定义抓住了妖股“短期内爆发力极强”的核心特征。

![](img/2f593b51d1c66dee5afa424f24ffb84f_9.png)
