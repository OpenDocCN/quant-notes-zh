## 作业一：清除指定单元格的条件格式

![](img/e5da462152f202e39ea2d82cddb81ba5_3.png)

上一节我们介绍了条件格式的基本应用，本节中我们来看看如何精确地清除部分单元格的条件格式。

![](img/e5da462152f202e39ea2d82cddb81ba5_5.png)

![](img/e5da462152f202e39ea2d82cddb81ba5_7.png)

题目要求是：仅清除选中区域中已设置条件格式的单元格，保留其他单元格的原有格式。

**核心思路**：使用“定位条件”功能，精确选中设置了条件格式的单元格，然后仅清除这些单元格的格式。

以下是具体操作步骤：

1.  选中需要操作的数据区域。
2.  按下 `Ctrl + G` 快捷键，打开“定位”对话框。
3.  点击左下角的“定位条件”按钮。
4.  在弹出的“定位条件”对话框中，选择“条件格式”。
5.  点击“确定”。此时，所有设置了条件格式的单元格会被选中。
6.  在“开始”选项卡的“编辑”组中，点击“清除”，然后选择“清除格式”。

![](img/e5da462152f202e39ea2d82cddb81ba5_9.png)

这样操作后，只有被选中的、原本有条件格式的单元格被清除了格式，其他单元格的原有格式（如黄色背景）得以保留。

![](img/e5da462152f202e39ea2d82cddb81ba5_11.png)

## 作业二：对比两列数据的差异

在数据处理中，经常需要对比两列数据是否一致。本节我们学习两种高亮显示差异单元格的方法。

![](img/e5da462152f202e39ea2d82cddb81ba5_15.png)

### 方法一：使用定位条件

这种方法快速直接，适用于简单的数据对比。

以下是具体操作步骤：

1.  选中需要对比的两列数据（例如A列和B列）。
2.  按下 `Ctrl + G`，打开“定位”对话框。
3.  点击“定位条件”，选择“行内容差异单元格”。
4.  点击“确定”。此时，与活动单元格（通常为每行第一个单元格）内容不同的单元格会被选中。
5.  为选中的单元格设置一个醒目的填充色（如黄色）。

![](img/e5da462152f202e39ea2d82cddb81ba5_17.png)

### 方法二：使用条件格式公式

这种方法更灵活，可以进行更精确的匹配（例如区分大小写或精确到小数位）。

以下是具体操作步骤：

1.  选中需要对比的数据区域（例如A1:B10）。
2.  点击“开始”选项卡下的“条件格式”。
3.  选择“新建规则”。
4.  在规则类型中，选择“使用公式确定要设置格式的单元格”。
5.  在公式框中输入以下代码：
    `=NOT(EXACT($A1, $B1))`
    **公式解释**：
    *   `EXACT($A1, $B1)`：精确比较A1和B1单元格的内容是否完全相同（区分大小写和格式）。
    *   `NOT(...)`：对结果取反。当两单元格内容**不同**时，公式返回 `TRUE`，触发条件格式。
    *   `$A1, $B1`：列绝对引用（`$`），行相对引用。这样公式在向下应用时，会逐行比较A列和B列。
6.  点击“格式”按钮，设置填充颜色（如黄色）。
7.  点击“确定”应用规则。

![](img/e5da462152f202e39ea2d82cddb81ba5_19.png)

**两种方法对比**：方法二（公式法）的对比更为精确，尤其适用于包含数字小数的数据。方法一（定位法）速度更快，但可能忽略细微差异。

![](img/e5da462152f202e39ea2d82cddb81ba5_21.png)

## 作业三：合并两列数据并跳过空白单元格

当我们需要将两列数据合并到一列，且两列数据在相同位置没有重复值时，可以使用“选择性粘贴”功能。

以下是具体操作步骤：

1.  复制第一列数据（`Ctrl + C`）。
2.  将鼠标定位到目标列的首个单元格。
3.  右键点击，选择“选择性粘贴”。
4.  在弹出的对话框中，勾选“跳过空单元”选项。
5.  点击“确定”。

![](img/e5da462152f202e39ea2d82cddb81ba5_23.png)

这样，第二列的非空单元格数据就会“跳过”第一列已有的数据，填充到下方的空白单元格中，实现两列数据的无缝合并。

## 作业四：合并上下半年数据表

本节我们学习如何将结构完全相同的两个表格（如上半年和下半年数据）进行合并计算。

### 方法一：选择性粘贴（加法）

如果只需要数值相加，这是最快捷的方法。

![](img/e5da462152f202e39ea2d82cddb81ba5_25.png)

以下是具体操作步骤：

![](img/e5da462152f202e39ea2d82cddb81ba5_27.png)

1.  复制下半年数据表（`Ctrl + C`）。
2.  选中上半年数据表的起始单元格。
3.  右键点击，选择“选择性粘贴”。
4.  在“运算”区域，选择“加”。
5.  点击“确定”。下半年数据就会与上半年对应单元格的数据相加。

### 方法二：使用公式

![](img/e5da462152f202e39ea2d82cddb81ba5_29.png)

如果表格结构完全一致，使用公式可以建立动态链接，便于后续数据更新。

![](img/e5da462152f202e39ea2d82cddb81ba5_31.png)

以下是具体操作步骤：

1.  在合并结果表的第一个单元格（例如C5）输入公式：
    `=M5 + M12`
    **公式解释**：将上半年表（M5）和下半年表（M12）的对应单元格相加。
2.  由于两个源表格结构一致，直接向右、向下拖动填充柄，即可完成整个表格的合并计算。

## 作业五：快速为小计行求和

这是一个非常实用的技巧，可以快速为分类汇总中的小计行填充求和公式。

以下是具体操作步骤：

![](img/e5da462152f202e39ea2d82cddb81ba5_35.png)

1.  选中包含数据和小计行的整个区域。
2.  按下 `Ctrl + G`，打开“定位”对话框。
3.  点击“定位条件”，选择“空值”。
4.  点击“确定”。所有空白的小计行单元格会被选中。
5.  按下 `Alt + =` 快捷键（或点击“开始”选项卡下的“自动求和”按钮 `Σ`）。
6.  Excel会自动为每个选中的空白单元格插入 `SUM` 函数，并智能选择其上方的连续数据区域作为求和范围。

![](img/e5da462152f202e39ea2d82cddb81ba5_37.png)

## 作业六：快速生成大量连续序号

![](img/e5da462152f202e39ea2d82cddb81ba5_39.png)

当需要生成成千上万个连续序号时，手动下拉填充效率很低。我们可以使用“序列”填充功能。

以下是具体操作步骤：

![](img/e5da462152f202e39ea2d82cddb81ba5_41.png)

1.  在起始单元格（如A1）输入序列的起始数字 `1`。
2.  选中该单元格，在“开始”选项卡的“编辑”组中，点击“填充”。
3.  选择“序列”。
4.  在弹出的对话框中：
    *   **序列产生在**：选择“列”。
    *   **类型**：选择“等差序列”。
    *   **步长值**：输入 `1`。
    *   **终止值**：输入你需要的最大序号，例如 `10000`。
5.  点击“确定”。Excel会自动在A列生成从1到10000的连续序号。

## 作业七：拆分数据并提取不重复值

这道题综合运用了“查找替换”、“分列”和“删除重复项”功能，目标是处理如“中国vs阿尔及利亚”格式的数据。

![](img/e5da462152f202e39ea2d82cddb81ba5_45.png)

以下是具体操作步骤：

![](img/e5da462152f202e39ea2d82cddb81ba5_47.png)

1.  **统一分隔符**：选中数据列，按下 `Ctrl + H` 打开“查找和替换”对话框。将“vs”全部替换为逗号“,”。
2.  **拆分数据**：保持数据列选中状态，点击“数据”选项卡下的“分列”。选择“分隔符号”，点击“下一步”。勾选“逗号”作为分隔符，点击“完成”。数据会被拆分成两列。
3.  **提取不重复值**：
    *   将拆分出的两列数据分别复制到两个单独的新列中。
    *   选中第一列新数据，点击“数据”选项卡下的“删除重复项”，点击“确定”。
    *   对第二列新数据重复此操作。
    *   最后将两列不重复的结果合并，即可得到最终的国家列表。

![](img/e5da462152f202e39ea2d82cddb81ba5_49.png)

## 作业八：从文本中提取特定信息

![](img/e5da462152f202e39ea2d82cddb81ba5_51.png)

本题要求从“工作能力 88.5”这类文本中，分别提取出评价项目（如“工作能力”）和分数（如“88.5”）。

### 方法一：使用“分列”功能

![](img/e5da462152f202e39ea2d82cddb81ba5_53.png)

这是最直观的方法，利用空格作为分隔符。

![](img/e5da462152f202e39ea2d82cddb81ba5_55.png)

以下是具体操作步骤：

![](img/e5da462152f202e39ea2d82cddb81ba5_57.png)

1.  选中文本数据列。
2.  点击“数据”选项卡下的“分列”。
3.  选择“分隔符号”，点击“下一步”。
4.  勾选“空格”作为分隔符，点击“完成”。文本会按空格被拆分成“工作能力”和“88.5”两列。

### 方法二：使用文本函数（更灵活）

如果文本结构规律（如分数总是最后4位），可以使用函数一次性批量提取。

以下是具体操作步骤：

![](img/e5da462152f202e39ea2d82cddb81ba5_59.png)

1.  **提取分数**：在目标单元格输入公式 `=RIGHT(A1, 4)`。此公式会提取A1单元格文本最右侧的4个字符（即分数）。向下拖动填充即可提取所有分数。
2.  **提取项目**：可以使用 `LEFT` 或 `MID` 函数配合 `FIND` 函数来定位空格位置，从而提取项目名称。例如：`=LEFT(A1, FIND(" ", A1)-1)`。

**方法对比**：方法一（分列）操作简单。方法二（函数）更具通用性和可扩展性，尤其当数据量很大或结构有微小变化时，只需调整公式即可。

![](img/e5da462152f202e39ea2d82cddb81ba5_61.png)
