## 选择交易日数据 📅

上一节我们导入了所有必要的工具包。本节中，我们来看看如何获取用于分析的时间段数据。

与回测框架不同，当前模块没有自动按日执行的机制。因此，我们需要手动获取多天的数据。第一步是选择一个日期范围。

以下是获取交易日列表的步骤：

1.  定义一个函数 `get_trading_date`。
2.  函数接收两个参数：起始日期和结束日期。
3.  函数返回指定时间段内的所有交易日列表。

```python
def get_trading_date(start_date, end_date):
    trading_dates = get_trading_dates(start_date, end_date)
    return trading_dates
```

执行这段代码后，我们就获得了从`2019-01-01`到`2020-01-01`的所有交易日列表。

## 获取股票池与构建查询 🏢

![](img/75f7501047a7765f1c0688dc5d79c18f_3.png)

现在我们已经有了交易日数据，接下来需要获取要分析的股票列表，并构建因子查询。

![](img/75f7501047a7765f1c0688dc5d79c18f_5.png)

我们选择分析沪深A股的所有股票，并以市盈率（`pe_ratio`）作为示例因子。

![](img/75f7501047a7765f1c0688dc5d79c18f_7.png)

以下是获取股票池和构建查询的步骤：

1.  使用 `get_all_securities` 函数获取所有股票代码，并存储在变量 `stock_pool` 中。
2.  构建一个查询（`query`），指定要获取的因子字段（`valuation.pe_ratio`）。
3.  在查询中添加过滤器（`filter`），只选择股票代码在 `stock_pool` 中的股票。

![](img/75f7501047a7765f1c0688dc5d79c18f_9.png)

```python
# 获取所有股票代码作为股票池
stock_pool = get_all_securities(['stock']).index.tolist()

![](img/75f7501047a7765f1c0688dc5d79c18f_11.png)

![](img/75f7501047a7765f1c0688dc5d79c18f_13.png)

# 构建查询语句
q = query(
    valuation.pe_ratio
).filter(
    valuation.pe_ratio.code.in_(stock_pool)
)
```

这段代码定义了一个查询 `q`，它旨在获取股票池中所有股票的市盈率数据。

## 执行查询并获取数据 🔍

![](img/75f7501047a7765f1c0688dc5d79c18f_19.png)

构建好查询语句后，我们需要执行它以获取实际数据。由于我们有多个交易日，这里先演示获取单日数据的方法。

![](img/75f7501047a7765f1c0688dc5d79c18f_21.png)

以下是执行查询并获取数据的步骤：

![](img/75f7501047a7765f1c0688dc5d79c18f_23.png)

![](img/75f7501047a7765f1c0688dc5d79c18f_25.png)

1.  使用 `get_fundamentals` 函数执行查询。
2.  需要传入两个关键参数：构建好的查询语句 `q` 和指定的查询日期 `date`。
3.  我们将查询日期设为交易日列表中的第一天（`trading_dates[0]`）进行测试。

```python
# 执行查询，获取指定日期的因子数据
date_to_query = trading_dates[0]
factor_data = get_fundamentals(q, date=date_to_query)
```

![](img/75f7501047a7765f1c0688dc5d79c18f_27.png)

执行后，`factor_data` 变量中存储了查询结果。数据是一个三维结构，但通常我们只关心最后一个维度的具体数值。可以使用 `.iloc` 方法查看前几条数据。

![](img/75f7501047a7765f1c0688dc5d79c18f_29.png)

```python
# 查看获取到的数据形状和前几条记录
print(factor_data.shape)
print(factor_data.iloc[0, 0, :5])  # 查看前5只股票的市盈率数据
```

输出结果将显示股票代码及其对应的市盈率指标值。至此，我们已经成功获取了指定日期（例如2019年1月2日）的因子数据。
