## 概述

K-Means是一种经典的无监督学习算法，用于将数据点划分为K个簇。其核心思想是迭代地寻找簇的中心点（质心），并将每个点分配到距离最近的质心所在的簇。整个过程不依赖于任何预先定义的标签。

## K-Means工作流程详解

上一节我们介绍了K-Means的基本概念，本节中我们来看看它的具体工作步骤。我们将通过一个K=2的例子，分步解析其流程。

### 第一步：初始化质心

由于是无监督问题，我们不知道每个点属于哪个簇。运行K-Means算法需要指定一个参数K，即期望的簇数量。假设我们设定K=2。

算法开始时，会随机初始化两个点作为初始质心。在图中，一个用红色表示，一个用蓝色表示。

### 第二步：分配样本点到最近的质心

初始化后，算法需要计算每个样本点属于红色簇还是蓝色簇。判断依据是距离。

对于一个样本点（例如一个绿色点），计算它到红色质心的距离`D1`，以及到蓝色质心的距离`D2`。如果`D1 < D2`，则认为该点属于红色簇，因为距离越小，相似度越高。

以下是该步骤的核心操作：
*   **遍历所有样本点**。
*   对每个点，计算其到所有K个质心的距离。
*   将该点分配到**距离最小的质心**所在的簇。

完成这一步后，所有点都被临时标记为红色或蓝色，形成了初步的簇划分。

### 第三步：重新计算质心

初步划分后，我们得到的簇可能并不理想（例如，肉眼可见应该分成左右两堆，但算法初步分成了上下两堆）。因此，需要更新衡量依据——即质心。

更新方法如下：
*   对于所有被标记为红色的点，计算它们的平均位置，作为新的红色质心。
*   对于所有被标记为蓝色的点，计算它们的平均位置，作为新的蓝色质心。

这样，质心从随机初始化的位置，移动到了当前簇中所有点的中心位置。

### 第四步：迭代优化

更新质心后，流程并未结束。我们需要重复第二步和第三步：

1.  **重新分配**：基于新的质心位置，再次遍历所有样本点，根据距离重新分配它们到红色或蓝色簇。一些点的归属可能会发生变化（例如，原先的红色点可能因为离新蓝色质心更近而变为蓝色）。
2.  **重新计算**：根据新的分配结果，再次更新红色和蓝色质心的位置。

![](img/331c7d28f36c7aeaa1fc48e09ecc5fb3_2.png)

这个过程（分配 -> 更新 -> 再分配 -> 再更新）会不断重复。

### 第五步：收敛

![](img/331c7d28f36c7aeaa1fc48e09ecc5fb3_4.png)

当迭代进行到某一步时，会发现所有样本点的簇归属不再发生变化，或者质心的位置移动非常小。此时，算法已经收敛，迭代停止。

![](img/331c7d28f36c7aeaa1fc48e09ecc5fb3_6.png)

最终，我们得到了稳定的簇划分结果。对于K=3或更多簇的情况，原理完全相同，只是会初始化更多质心，并相应地进行分配和更新。

综上所述，K-Means的工作流程就是：**随机初始化质心 -> 根据距离分配样本点 -> 根据分配结果更新质心 -> 迭代直至收敛**。

## K-Means算法的优缺点

![](img/331c7d28f36c7aeaa1fc48e09ecc5fb3_8.png)

了解了工作流程后，我们来看看这个算法的优点和缺点。

![](img/331c7d28f36c7aeaa1fc48e09ecc5fb3_10.png)

### 优点

K-Means算法主要有以下优点：
*   **原理简单，易于理解**：其核心思想（找中心、算距离）非常直观。
*   **实现高效，速度快**：对于常规数据集，计算速度较快。
*   **对于结构紧凑、分离度好的常规数据集效果良好**：例如我们示例中能够明显区分为两堆的数据。

### 缺点

同时，K-Means也存在一些明显的缺点：
*   **需要预先指定K值**：在无标签的情况下，很难确定数据集中到底存在多少个自然的簇。通常需要尝试不同的K值并评估效果。
*   **对初始质心敏感**：不同的随机初始化可能导致不同的最终结果。
*   **计算复杂度与样本量线性相关**：每次迭代都需要计算所有样本点到所有质心的距离。当样本量极大（如几千万）时，计算开销很高。
*   **难以处理非球形簇或大小差异大的簇**：这是K-Means最主要的局限性。它基于距离度量，倾向于发现球状的、大小相似的簇。

以下是其局限性的一个典型例子：
```python
# K-Means假设簇是凸形的、各向同性的，对于环形、流形或交叉状的数据分布效果不佳。
# 例如，它无法有效区分一个“同心圆”结构的内圈和外圈，
# 而更可能将其按上下或左右切分成两个簇。
```
*   **对噪声和离群点敏感**：由于质心是所有点的平均值，离群点会显著拉偏质心的位置。
