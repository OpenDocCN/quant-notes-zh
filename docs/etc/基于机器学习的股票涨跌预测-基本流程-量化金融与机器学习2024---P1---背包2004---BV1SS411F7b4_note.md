# 量化金融与机器学习：1：基于机器学习的股票涨跌预测基本流程 📈

在本节课中，我们将学习如何构建一个简单的机器学习模型，用于预测股票在未来30个交易日内的涨跌。我们将使用市盈率和总市值作为特征，并遵循数据准备、模型训练与评估的标准流程。

## 数据准备与特征工程

上一节我们介绍了如何构建目标向量。本节中，我们来看看如何准备特征数据并构建数据集。

我们使用两个最简单的特征：市盈率和总市值。目标向量是之前构建好的，代表股票未来30个交易日的涨跌情况。

以下是准备数据的具体步骤：

1.  **导入数据**：从已构建好的数据集中，提取市盈率、总市值以及目标向量。
2.  **处理缺失值**：删除数据中包含空值的行，确保数据质量。
3.  **构建特征与目标**：将市盈率和总市值作为特征矩阵 `X`，将目标向量作为 `y`。
4.  **数据抽样**：由于原始数据量较大（约100万行），为了演示和快速运行，我们进行抽样。例如，使用 `sample(frac=0.03)` 抽取3%的数据。
5.  **特征标准化**：特征间的数值差异可能很大，需要进行标准化处理，使其符合均值为0、标准差为1的分布。标准化公式如下：
    ```
    X_standardized = (X - mean(X)) / std(X)
    ```

完成上述步骤后，我们就得到了可用于模型训练的标准化特征 `X` 和目标 `y`。

## 模型训练与评估

有了准备好的数据，接下来我们进入机器学习的核心环节：划分数据集、训练模型并进行评估。

首先，我们需要将数据集划分为训练集和测试集。我们使用 `train_test_split` 函数，按7:3的比例进行划分。

```python
from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)
```

![](img/d1f2a41775a320f7be3d93c66bee426a_7.png)

![](img/d1f2a41775a320f7be3d93c66bee426a_9.png)

接着，我们选择一个机器学习算法进行训练。这里我们使用梯度提升决策树（Gradient Boosting Decision Tree），它是集成学习的一种，在 `sklearn` 中通常能取得较好的效果。我们使用其默认参数进行实例化和训练。

```python
from sklearn.ensemble import GradientBoostingClassifier
model = GradientBoostingClassifier()
model.fit(X_train, y_train)
```

模型训练完成后，我们需要评估其性能。以下是评估模型的关键步骤：

1.  **在测试集上进行预测**：使用训练好的模型对测试集特征 `X_test` 进行预测，得到预测结果 `y_pred`。
2.  **计算准确度**：将预测结果 `y_pred` 与真实标签 `y_test` 进行比较，计算模型预测正确的比例。
3.  **绘制混淆矩阵**：混淆矩阵能更细致地展示模型的分类情况。对于二分类问题，矩阵对角线上的值代表预测正确的样本数（包括正类和负类），非对角线上的值代表预测错误的样本数。
4.  **查看特征重要性**：分析模型认为哪些特征对预测结果更重要。这有助于我们理解模型决策的依据。

在我们的示例中，模型取得了约56%的准确率。虽然仅比随机猜测（50%）略好，但它证明了使用当前市盈率和市值来预测未来涨跌存在一定的可行性。此外，特征重要性显示，总市值这一特征比市盈率更为重要。

## 模型输出的深入理解

除了最终的分类结果，模型还能提供更多信息，例如样本属于各个类别的概率。

模型输出的概率值表示某个样本被分为“涨”（类别1）或“跌”（类别0）的置信度。例如，对于第一个样本，模型可能输出 `[0.54, 0.46]`，这意味着模型认为该样本有54%的概率属于“跌”，46%的概率属于“涨”，因此最终将其判定为“跌”。这些概率值在后续构建更复杂的策略（如基于概率的仓位管理）时非常有用。
