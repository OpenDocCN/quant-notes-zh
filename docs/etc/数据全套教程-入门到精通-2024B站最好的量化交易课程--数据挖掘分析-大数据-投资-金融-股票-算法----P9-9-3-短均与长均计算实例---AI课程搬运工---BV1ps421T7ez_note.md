在本节课中，我们将学习一个基础的股票序列分析策略。我们将通过计算短期和长期移动平均线，来识别市场中的“黄金交叉”和“死亡交叉”信号，从而辅助判断买入和卖出的时机。

## 策略概述与核心概念

上一节我们介绍了数据处理的基础，本节中我们来看看如何应用这些数据计算技术指标。这个策略的核心是计算两条移动平均线：一条短期均线（如5日或10日均线）和一条长期均线（如20日或30日均线）。当短期均线从下方上穿长期均线时，形成“黄金交叉”，通常被视为买入信号；反之，当短期均线从上方下穿长期均线时，形成“死亡交叉”，通常被视为卖出信号。

![](img/38e4dfeb042b26a0566445feec26d109_8.png)

## 计算短期与长期移动平均线

![](img/38e4dfeb042b26a0566445feec26d109_10.png)

![](img/38e4dfeb042b26a0566445feec26d109_11.png)

首先，我们需要从数据中计算出这两条均线。我们以苹果公司（AAPL）的股票数据为例。

![](img/38e4dfeb042b26a0566445feec26d109_13.png)

以下是计算步骤：

![](img/38e4dfeb042b26a0566445feec26d109_15.png)

![](img/38e4dfeb042b26a0566445feec26d109_16.png)

![](img/38e4dfeb042b26a0566445feec26d109_17.png)

![](img/38e4dfeb042b26a0566445feec26d109_18.png)

1.  **选择目标股票**：我们从数据集中提取苹果公司（AAPL）的股价数据。
2.  **计算短期均线（M1）**：我们使用一个较小的窗口（例如10天）来计算股价的移动平均值。
    ```python
    data[‘M1’] = data[‘AAPL’].rolling(window=10).mean()
    ```
3.  **计算长期均线（M2）**：我们使用一个较大的窗口（例如30天）来计算另一个移动平均值。
    ```python
    data[‘M2’] = data[‘AAPL’].rolling(window=30).mean()
    ```

![](img/38e4dfeb042b26a0566445feec26d109_19.png)

执行完这些计算后，我们的数据框中将新增两列：`M1`（10日均线）和`M2`（30日均线）。

## 量化交叉信号

![](img/38e4dfeb042b26a0566445feec26d109_36.png)

![](img/38e4dfeb042b26a0566445feec26d109_37.png)

![](img/38e4dfeb042b26a0566445feec26d109_39.png)

为了更精确地定位交叉点，我们可以创建一个信号列。这个信号列明确标记出短期均线是高于还是低于长期均线。

以下是创建信号列的步骤：

![](img/38e4dfeb042b26a0566445feec26d109_41.png)

![](img/38e4dfeb042b26a0566445feec26d109_42.png)

1.  **使用`np.where`函数进行比较**：我们比较短期均线（M1）和长期均线（M2）。
2.  **分配信号值**：当M1 > M2时，我们标记为`1`；当M1 < M2时，我们标记为`-1`。
    ```python
    data[‘position’] = np.where(data[‘M1’] > data[‘M2’], 1, -1)
    ```
3.  **可视化信号**：由于信号值（1和-1）的尺度与股价相差巨大，直接绘制在同一坐标轴上会看不清楚。因此，我们使用双Y轴来绘图：左Y轴显示股价和均线，右Y轴专门显示我们的`position`信号线。
    ```python
    ax = data[[‘AAPL’, ‘M1’, ‘M2’]].plot(figsize=(12,6))
    data[‘position’].plot(ax=ax, secondary_y=True, style=‘g’)
    ```

![](img/38e4dfeb042b26a0566445feec26d109_44.png)

在最终的图表中，右侧Y轴的范围在-1到1之间。`position`信号线在`1`和`-1`之间切换。信号线从`-1`变为`1`的点，对应图表中的“黄金交叉”；从`1`变为`-1`的点，则对应“死亡交叉”。这使得买卖信号的识别变得一目了然。
