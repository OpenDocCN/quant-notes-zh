## 课程特色与教学方式

![](img/e238f1424e5d040b48f3514b770e84ba_27.png)

我们的课程重点在于实战，而非理论讲解。不会教授金融知识或炒股技巧，核心是学习如何在Python中实现具体任务。

![](img/e238f1424e5d040b48f3514b770e84ba_29.png)

![](img/e238f1424e5d040b48f3514b770e84ba_31.png)

![](img/e238f1424e5d040b48f3514b770e84ba_33.png)

以下是本课程的主要特色：

![](img/e238f1424e5d040b48f3514b770e84ba_35.png)

*   **纯实战导向**：每节课均围绕代码展开，重点讲解如何使用Python处理数据、进行分析和完成回测。
*   **工具与平台教学**：课程将详细演示代码编写环境（如Jupyter Notebook）和量化回测平台的使用方法。
*   **基于真实数据**：所有案例均使用实际金融数据引入，讲解数据分析、金融统计的常用方法。
*   **零基础入门**：课程从Python基础开始，逐步过渡到金融数据与量化交易项目。
*   **配套资料齐全**：课程提供所有涉及到的数据和代码，方便学员练习。

接下来，我们详细了解一下课程的具体章节安排。

上一节我们了解了课程的整体特色，本节中我们来看看详细的学习路径。

![](img/e238f1424e5d040b48f3514b770e84ba_47.png)

以下是课程三大模块的详细大纲：

![](img/e238f1424e5d040b48f3514b770e84ba_49.png)

1.  **模块一：Python必备工具包实战**
    *   **目标**：为零基础学员打好Python基础。
    *   **内容**：讲解Python核心知识点、环境配置与工具安装。重点学习两个核心库：
        *   **NumPy**：用于数值计算。例如，计算数组的平均值：`np.mean(array)`。
        *   **Pandas**：用于数据分析和处理。例如，读取CSV文件：`pd.read_csv(‘data.csv’)`。

![](img/e238f1424e5d040b48f3514b770e84ba_51.png)

![](img/e238f1424e5d040b48f3514b770e84ba_53.png)

2.  **模块二：Python金融数据分析**
    *   **目标**：将Python技能应用于金融领域。
    *   **内容**：以真实股票数据为例，讲解金融数据的分析、建模与统计方法。同时，介绍如何在量化回测平台中完成第一个回测项目，评估策略的历史表现（如收益、回撤等指标）。

![](img/e238f1424e5d040b48f3514b770e84ba_55.png)

3.  **模块三：量化交易策略深入**
    *   **目标**：深入量化交易实战。
    *   **内容**：讲解量化交易中的经典策略。涉及实际操作中的数据处理、因子分析、策略制定与改进，最终目标是在历史回测中获得理想的收益结果。
