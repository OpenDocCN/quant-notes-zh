# Python金融分析量化交易：4-3：3-Sigma方法实例 📊

在本节课中，我们将要学习一种基于统计学原理的数据去极值方法——3-Sigma方法。我们将理解其核心思想，并通过代码实现来掌握如何应用此方法处理金融数据中的异常值。

## 概述

上一节我们介绍了分位数和中位数绝对偏差去极值法，本节中我们来看看另一种基于正态分布假设的方法：3-Sigma方法。其原理是利用数据的均值和标准差来定义一个“正常”数据的范围，并将超出此范围的值视为极值进行处理。

## 3-Sigma方法原理 📈

3-Sigma方法的原理基于正态分布（也称高斯分布）。正态分布是一种常见的概率分布，其形状呈钟形曲线。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_1.png)

如果理解正态分布有困难，可以将其想象为一种“正常”的分布。例如，在银行贷款场景中，银行多借或少借你少量金额（如三块五块）是正常情况；但多借或少借你大量金额（如十万八万）则属于不正常情况。

在标准正态分布中，均值（μ）为0。数据点围绕均值分布，距离均值越近的区域，事件发生的概率越高（图中面积越大）；距离均值越远的区域，事件发生的概率则越低。

我们进行去极值的目标，就是要找出那些概率极低、偏离正常范围很远的点，即离群点。这些点通常落在距离均值很远的区域。

所谓“3-Sigma”，是指以均值为中心，向左右各扩展3倍标准差（σ）所构成的区间。标准差是衡量数据离散程度的指标。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_1.png)

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_3.png)

如上图所示，`X`代表均值。区间 `[μ-σ, μ+σ]`、`[μ-2σ, μ+2σ]`、`[μ-3σ, μ+3σ]` 所包含的数据比例是固定的。区间越大，数据点落在左右两端的可能性就越小。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_5.png)

3-Sigma方法的思想是：假设数据经过某种变换后，理论上服从正态分布。当我们指定正负3倍标准差的区间时，这个区间包含了约99.7%的数据。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_7.png)

因此，我们可以认为数据中最正常的99.7%应落在此区间内，而剩下的0.3%则被视为极值并进行规范化处理。这种方法就叫做3-Sigma法。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_9.png)

确定上下限只需要两个参数：数据的**均值（μ）**和**标准差（σ）**。
*   **上限**计算公式为：`μ + n * σ`
*   **下限**计算公式为：`μ - n * σ`
其中，`n`通常取3。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_11.png)

## 代码实现 💻

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_13.png)

接下来，我们将通过代码实现3-Sigma去极值函数。

首先，我们创建一个新的函数，其结构与之前的去极值函数类似，但内部逻辑使用3-Sigma规则。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_15.png)

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_17.png)

```python
def filter_extreme_3sigma(series, n=3):
    """
    使用3-Sigma方法进行去极值处理。
    参数:
        series (pd.Series): 待处理的数据序列。
        n (int): 标准差的倍数，默认为3。
    返回:
        pd.Series: 处理后的数据序列。
    """
    # 计算序列的均值
    mean = series.mean()
    # 计算序列的标准差
    std = series.std()
    
    # 根据均值和标准差计算上限和下限
    up_limit = mean + n * std
    low_limit = mean - n * std
    
    # 将超出上限的值截断为上限，低于下限的值截断为下限
    new_series = series.copy()
    new_series[new_series > up_limit] = up_limit
    new_series[new_series < low_limit] = low_limit
    
    return new_series, low_limit, up_limit
```

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_19.png)

以上代码完成了3-Sigma去极值函数。它接收一个数据序列`series`和倍数`n`，计算均值和标准差后，得到上下限，最后对原数据进行截断处理。

## 核心要点总结 ✅

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_39.png)

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_41.png)

本节课中我们一起学习了3-Sigma去极值方法。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_43.png)

*   **原理**：基于数据服从正态分布的假设，利用**均值（μ）**和**标准差（σ）**计算出正常数据的范围（`[μ - nσ, μ + nσ]`），通常`n=3`。
*   **操作**：将超出上限的值设置为上限，低于下限的值设置为下限，而不是直接删除。
*   **目标**：定义好数据的**下界**和**上界**，使所有数值都落在这个合理的范围内，从而消除极端值的影响。

![](img/b5905bd4cbf1c8d8e3ff2967aef9a49f_45.png)

这是数据预处理“三步走”策略（去极值、标准化、中性化）中的第一步——去极值。掌握多种去极值方法，能让你在处理金融数据时更加得心应手。
