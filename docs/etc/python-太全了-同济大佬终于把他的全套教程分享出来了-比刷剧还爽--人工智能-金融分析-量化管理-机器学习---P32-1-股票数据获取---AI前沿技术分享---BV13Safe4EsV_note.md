## 第四步：股票池的初步筛选

获取全市场股票后，我们通常不会直接在所有股票上进行操作，因为其中可能包含ST股、刚上市的新股、停牌股等不适合交易的股票。因此，我们需要对初始股票池进行过滤。

![](img/9848bd44a592524725236f1e1781f29d_31.png)

![](img/9848bd44a592524725236f1e1781f29d_33.png)

在 `rebalance` 函数中，在获取股票列表后，我们需要添加过滤条件。例如，可以过滤掉上市时间过短的股票，或者当前处于停牌状态的股票。这一步能确保我们的策略在更稳定、流动性更好的股票池中运行。
