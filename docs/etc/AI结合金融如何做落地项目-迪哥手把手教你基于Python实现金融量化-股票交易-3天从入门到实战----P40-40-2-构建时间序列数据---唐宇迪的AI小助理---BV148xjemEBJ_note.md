## 核心函数参数详解

在构建序列之前，我们先理解函数中会用到的几个关键参数。

以下是该函数涉及的参数及其含义：

*   **`dataset`**：表示输入的特征数据集。它决定了在构建时间窗口时使用哪些特征。我们可以选择使用单个特征（如温度）或多个特征组合进行实验。
*   **`start_index`**：起始索引。它指定从数据集的哪个位置开始构建序列。通常起始索引设置为0，表示从数据开头开始。
*   **`history_size`**：历史窗口大小。它定义了每个输入序列 `X` 的长度。例如，若设置为20，则一个 `X` 样本由连续的20个时间步数据组成。
*   **`index`**：索引序列。这个参数用于记录在构建每个序列窗口时，所选取的具体数据索引位置。例如，若窗口大小为10，`index` 会记录这10个数据点在原始数据中的位置。
*   **`target`**：目标标签。它定义了我们要预测的内容。如果 `target` 是一个值（如1），则预测下一个时间点的值；如果 `target` 是一个序列（如 `[1, 2, 3]`），则预测未来多个时间点的序列值。

理解了这些参数，我们接下来看看函数内部的具体构建方法。

## 序列构建方法解析

![](img/28ce1f701ad5ce10100fe9acbc3e9e66_16.png)

函数的核心逻辑是生成配对的 `(X, Y)` 数据。起始索引 `i` 从 `start_index` 开始。

![](img/28ce1f701ad5ce10100fe9acbc3e9e66_18.png)

![](img/28ce1f701ad5ce10100fe9acbc3e9e66_20.png)

为了更直观地理解，我们结合一个具体例子。假设 `start_index = 0`，`history_size = 20`。

![](img/28ce1f701ad5ce10100fe9acbc3e9e66_22.png)

函数内部通过一个循环来生成数据。关键步骤在于索引的计算：

```python
# 索引范围的计算
range_start = i - history_size
range_end = i
```

当 `i = 20` 时，`range_start = 20 - 20 = 0`，`range_end = 20`。这意味着第一个 `X` 样本取索引 `[0, 1, ..., 19]` 的数据，对应的 `Y` 则是索引 `[20]` 的数据（假设 `target=0`，即预测下一个点）。

循环继续，`i = 21` 时，`X` 取索引 `[1, 2, ..., 20]`，`Y` 取索引 `[21]`，依此类推。这样就生成了从 `[0:20]` 预测 `[20]`，`[1:21]` 预测 `[21]` 等一系列序列样本。

获取索引后，函数会对数据进行 `reshape` 操作，将其转换为模型可接受的二维数据格式（样本数, 序列长度, 特征数）。标签 `Y` 的构建方法类似，根据 `target` 参数从 `dataset` 中提取未来对应位置的值。
