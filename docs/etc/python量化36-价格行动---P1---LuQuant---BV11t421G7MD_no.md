## 识别霍夫曼回撤柱

在准备好数据后，我们需要识别策略的核心形态——霍夫曼回撤柱。

![](img/b2fb7c7176c29d8dec2faf4d71ae2bbe_4.png)

霍夫曼回撤柱的定义是：在上升趋势中，蜡烛实体较小且具有较长的下影线（影线长度占整根蜡烛高度的45%以上）；在下降趋势中，则具有较长的上影线。我们通过以下公式判断影线比例：

**上升趋势影线比例 = (Close - Low) / (High - Low)**
**下降趋势影线比例 = (High - Close) / (High - Low)**

![](img/b2fb7c7176c29d8dec2faf4d71ae2bbe_6.png)

当比例大于0.45时，我们认为它符合霍夫曼回撤柱的形态。

以下是识别霍夫曼回撤柱的条件代码框架：

```python
def is_hofmann_bar(row, trend):
    if trend == ‘uptrend‘:
        # 检查是否为上升趋势中的回撤柱（长下影线）
        lower_shadow_ratio = (row[‘Close‘] - row[‘Low‘]) / (row[‘High‘] - row[‘Low‘])
        return lower_shadow_ratio > 0.45
    elif trend == ‘downtrend‘:
        # 检查是否为下降趋势中的回撤柱（长上影线）
        upper_shadow_ratio = (row[‘High‘] - row[‘Close‘]) / (row[‘High‘] - row[‘Low‘])
        return upper_shadow_ratio > 0.45
    return False
```

## 添加枢轴点条件

![](img/b2fb7c7176c29d8dec2faf4d71ae2bbe_10.png)

仅仅识别回撤柱还不够。为了提升信号质量，我们额外要求该回撤柱同时是一个“枢轴点”。

*   **高枢轴点（上升趋势）**：蜡烛的最高价，高于其左边和右边若干根蜡烛的最高价。
*   **低枢轴点（下降趋势）**：蜡烛的最低价，低于其左边和右边若干根蜡烛的最低价。

这样做的逻辑是，寻找试图逆转趋势但失败的最强烈反应（即形成枢轴点），当价格突破该枢轴点时，原趋势延续的可能性更高。

以下是检查枢轴点的逻辑描述：

对于一根疑似霍夫曼回撤柱的蜡烛，我们检查其索引`i`。
*   若在上升趋势中，则检查：`df[‘High‘].iloc[i]` 是否大于 `df[‘High‘].iloc[i-L : i+R]` 这个窗口内（不包括`i`本身）所有蜡烛的最高价。其中`L`和`R`是左右窗口的大小。
*   若在下降趋势中，则检查：`df[‘Low‘].iloc[i]` 是否小于 `df[‘Low‘].iloc[i-L : i+R]` 这个窗口内所有蜡烛的最低价。

## 生成交易信号

结合趋势判断、霍夫曼回撤柱识别和枢轴点条件，我们可以生成具体的交易信号。

以下是生成信号的步骤列表：

1.  **判断趋势方向**：使用移动平均线斜率。例如，斜率大于0为上升趋势，小于0为下降趋势。
2.  **识别霍夫曼回撤柱**：在确定的趋势中，寻找符合影线比例条件的蜡烛。
3.  **验证枢轴点**：确认该回撤柱同时也是相应的高点或低点枢轴。
4.  **确定入场点**：当价格突破该霍夫曼回撤柱的高点（上升趋势）或低点（下降趋势）时，产生入场信号。

我们可以将信号可视化在图表上，以验证算法是否正确运行。例如，用紫色圆点标记识别出的霍夫曼枢轴点。

## 策略回测与风险管理

在信号生成后，我们需要通过历史数据回测来评估策略表现。风险管理是回测中至关重要的一环。

我们主要关注两个参数：
*   **止损**：当交易不利于我们时，平仓退出以限制损失。
*   **止盈**：当交易达到预期利润时，平仓退出以锁定收益。

设置止损止盈的方法有很多种，本课程展示两种常见方式：

以下是两种止损止盈设置方法：

1.  **基于ATR**：止损设置为入场价加减若干倍（如4倍）的ATR值。止盈则根据设定的风险回报比（如1.5）计算。
    ```python
    stop_loss = entry_price - (4 * atr_value) # 做多示例
    take_profit = entry_price + (1.5 * (entry_price - stop_loss)) # 做多示例
    ```
2.  **基于近期价格极值**：止损设置在近期若干根蜡烛的最高价（做空时）或最低价（做多时）附近。

回测结果显示，**不同的止损止盈设置对策略最终收益影响巨大**。例如，仅改变止损止盈方法，就可能将策略从亏损70%改善到盈利7.5%。这强调了资金管理和参数优化在量化交易中的重要性。
