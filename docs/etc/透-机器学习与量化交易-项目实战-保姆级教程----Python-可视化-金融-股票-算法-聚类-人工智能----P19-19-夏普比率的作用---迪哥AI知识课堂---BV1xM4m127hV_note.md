## 夏普比率的定义与意义

夏普比率描述的是：在承担单位风险的情况下，所能获得的超额收益是多少。它帮助我们判断一项投资的风险与回报是否匹配。

![](img/3e49618f82708a2da7a47d40d07b193f_1.png)

我们可以用一个生活中的例子来理解：假设有两份工作，一份是稳定的文员工作，月薪1万元；另一份是去战乱地区当雇佣兵，日薪3万元。后者的高薪对应着极高的生命风险。夏普比率就像是在衡量，为了多赚的这些钱，所承担的额外风险是否值得。

![](img/3e49618f82708a2da7a47d40d07b193f_3.png)

在选股或选择投资组合时，我们不仅看收益，更要看收益背后的风险。夏普比率越高，意味着**单位风险所获得的收益越高**，该投资也就越有吸引力。

![](img/3e49618f82708a2da7a47d40d07b193f_5.png)

## 夏普比率的计算公式

理解了其意义后，我们来看看如何计算夏普比率。它的核心思想是比较投资组合的收益与无风险收益的差异，并将其与投资组合的波动性（风险）相关联。

计算公式如下：

**Sharpe Ratio = (Rp - Rf) / σp**

其中：
*   **Rp**：投资组合的平均收益率
*   **Rf**：无风险收益率（例如国债利率、银行定期存款利率）
*   **σp**：投资组合收益率的标准差（代表风险）

这个公式可以理解为：投资组合每承担一单位的风险（标准差），能够获得多少超出无风险收益的回报。

## 使用Python计算夏普比率

接下来，我们通过Python代码来实际计算一下夏普比率。假设我们已经有了一个包含多只股票历史收益率的数据。

![](img/3e49618f82708a2da7a47d40d07b193f_7.png)

以下是计算夏普比率的关键步骤：

1.  准备数据：获取投资组合的历史收益率序列。
2.  设定参数：确定无风险收益率（通常按年化计算）。
3.  执行计算：应用上述公式进行计算，注意对标准差进行年化处理。

![](img/3e49618f82708a2da7a47d40d07b193f_9.png)

```python
import numpy as np
import pandas as pd

# 假设 returns 是一个Pandas DataFrame，每一列代表一只股票的历史日收益率
# 例如：returns = pd.DataFrame(...)

# 步骤1: 设定年化无风险收益率，例如5%
annual_risk_free_rate = 0.05
# 转换为日度无风险收益率（假设一年有250个交易日）
daily_risk_free_rate = annual_risk_free_rate / 250

# 步骤2: 计算每只股票的平均日收益率和日收益率的标准差
mean_daily_returns = returns.mean()
std_daily_returns = returns.std()

# 步骤3: 计算夏普比率（年化）
# 公式: (平均日收益率 - 日无风险收益率) / 日标准差 * sqrt(年交易天数)
sharpe_ratios = (mean_daily_returns - daily_risk_free_rate) / std_daily_returns * np.sqrt(250)

![](img/3e49618f82708a2da7a47d40d07b193f_11.png)

print(“各股票的夏普比率（年化）:”)
print(sharpe_ratios)

![](img/3e49618f82708a2da7a47d40d07b193f_13.png)

# 找出夏普比率最高的股票
best_stock = sharpe_ratios.idxmax()
print(f”\n夏普比率最高的股票是：{best_stock}，比率为：{sharpe_ratios[best_stock]:.4f}”)
```

运行以上代码后，我们会得到每只股票的夏普比率。一个正值且较大的夏普比率通常是更优的选择。负值则意味着该投资的收益尚不及无风险收益，承担风险并未带来正回报。
