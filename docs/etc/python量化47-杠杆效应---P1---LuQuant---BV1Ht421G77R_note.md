## 杠杆交易基础

上一节我们介绍了课程概述，本节中我们来看看杠杆交易的基本概念。

杠杆允许交易者以少量自有资金控制更大价值的头寸。其运作方式通常通过一个比率来描述，例如1:10或1:50。

**公式**：`可控制头寸价值 = 自有资金 × 杠杆比率`

例如，如果您的账户有1000美元，使用1:10的杠杆，您最多可以交易价值10000美元的头寸。使用1:50的杠杆，则可交易价值50000美元的头寸。

可用的杠杆比率取决于经纪商、所在国家法规和账户类型。常见的比率在1:10到1:50之间，但也有高达1:1000的情况。

![](img/376ee4f90894a0fdeb1e6a947d9347bd_3.png)

杠杆是一把双刃剑。它既能放大利润，也能放大亏损。初学者需要牢记以下两条核心规则：

以下是使用杠杆时需要记住的两条核心规则：
*   如果您的策略**胜率高**，杠杆可以**放大利润**。
*   如果您的策略**胜率低**，杠杆会**放大亏损**。

“高”或“低”胜率的界定，取决于策略类型、止损止盈设置和资金管理计划。请始终记住，杠杆是有代价的，它意味着您的资金仅用于覆盖损失，而由于交易金额被放大，损失可能迅速发生。

## 回测策略与代码框架

上一节我们了解了杠杆的基本原理，本节中我们将介绍用于回测的交易策略和代码框架。

我们将使用一个基于价格行为（如吞没形态、拒绝形态）在支撑位和阻力位附近寻找反转信号的策略。此策略在之前的视频中已介绍过。

以下是策略的核心逻辑代码框架：

```python
# 伪代码示例
if 价格接近阻力位 and 出现看跌拒绝形态:
    产生卖出信号
elif 价格接近支撑位 and 出现看涨拒绝形态:
    产生买入信号
```

初始测试时，我们使用账户资金的10%（即无杠杆的全额资金）进行交易。回测结果显示，该策略能产生约10%的正回报，权益曲线呈稳定上升趋势，表明这是一个有效的策略。

当我们调整参数，使用账户全部资金（例如9990美元）进行交易时，回报率提升至116%，胜率达到71%。这个较高的胜率为我们后续测试杠杆效应提供了良好的基础。

## 杠杆效应回测分析

上一节我们验证了基础策略的有效性，本节中我们来看看引入杠杆后的回测结果。

我们将在回测代码中引入`保证金`参数来模拟杠杆账户。首先测试1:10的杠杆（即用1000美元本金控制10000美元头寸）。

以下是不同杠杆比率下的回测结果对比：
*   **无杠杆（本金100%）**：回报率 **116%**，胜率 71%。
*   **杠杆 1:10**：回报率飙升至 **7140%**，最终净值约为724,000美元（基于18年数据）。
*   **杠杆 1:20**：回报率依然很高，胜率约为66%。
*   **杠杆 1:40**：回报率下降至约1325%。
*   **杠杆 1:50**：回报率降至约-100%，**账户被清零**。

从结果可以看出，在胜率较高的情况下，适度杠杆（如1:10）能带来巨大的回报放大效应。然而，随着杠杆比率增加，抵御亏损的能力急剧下降，过高的杠杆（如1:50）可能导致账户迅速爆仓。

此外，交易成本（如佣金）也会显著影响最终收益。例如，在1:10杠杆下引入小额佣金后，回报率从7140%下降至10800%，虽然仍是极好的结果，但成本的影响不容忽视。
