# 股票摸鱼神器V3.02讲解：P1：软件核心功能与使用指南 🚀

在本节课中，我们将学习一款名为“股票摸鱼神器V3.02”的软件。这款软件专为上班族设计，能在不引起注意的情况下，便捷地查看和监控股票信息。我们将详细介绍其核心功能与基本操作方法。

## 软件简介与核心概念

这款软件将股票信息以**3D圆球**的形式悬浮在电脑屏幕上。每个圆球代表一只股票，当鼠标**悬浮**在圆球上时，会显示该股票的详细信息。其核心交互逻辑可以用以下伪代码描述：

```pseudocode
if mouse_hover_over(ball):
    display_stock_info(ball.stock_code)
else:
    hide_stock_info()
```

这种设计既隐蔽又直观，非常适合在工作环境中使用。

## 股票管理功能

上一节我们了解了软件的显示形式，本节中我们来看看如何管理你关注的股票。软件支持通过右键菜单轻松添加和管理股票。

以下是添加股票的具体步骤列表：

1.  在圆球区域点击鼠标右键。
2.  在弹出的菜单中选择“添加股票”选项。
3.  在输入框中填写股票名称或代码。
4.  点击“添加”按钮，完成操作。

添加成功后，对应的圆球会立即显示在屏幕上。将鼠标悬浮上去，即可验证是否成功添加了你关注的股票。

## 显示控制与快捷键

为了方便控制信息的显示与隐藏，软件提供了灵活的显示模式。默认情况下，信息仅在鼠标悬浮时显示。若需要固定查看，可以使用快捷键。

按下 **`Ctrl + Tab`** 组合键，可以切换显示模式。第一次按下时，股票信息会**持续显示**在屏幕上；再次按下，信息则会**隐藏**，恢复为仅在鼠标悬浮时显示的状态。这个功能在需要短暂专注查看多只股票时非常有用。

## 3D网格与复盘联动功能

除了实时监控，软件还有一个强大的复盘辅助功能，称为“3D网格联动”。这个功能能极大提升你在阅读复盘文章或分析时的效率。

其工作原理是：当你在其他软件（如文档、网页）中**复制**了股票名称或代码时，软件会自动识别并将对应的3D圆球**高亮跳动**，同时在你设置的股票软件（如通达信、同花顺）中自动跳转到该股票的页面。

以下是该功能的操作演示：

1.  确保软件和你的股票交易软件（如通达信）同时运行。
2.  在任意地方（如一篇复盘文章）选中一个股票名称或代码，按 **`Ctrl + C`** 复制。
3.  观察“摸鱼神器”界面，对应的股票圆球会立即产生跳动效果。
4.  同时，你的股票交易软件会自动跳转到该股票的详情页或分时图页面。

例如，复制“新星科技”后，软件会立刻定位到这只股票，实现多平台间的快速联动，让复盘工作更加流畅高效。
