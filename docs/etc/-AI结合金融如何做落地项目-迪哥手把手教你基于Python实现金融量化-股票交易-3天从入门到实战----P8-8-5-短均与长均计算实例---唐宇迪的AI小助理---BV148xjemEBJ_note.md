## 计算短期与长期移动平均线

上一节我们介绍了移动平均线的概念，本节中我们来看看如何用Python具体计算。首先，我们需要选定分析的股票数据，这里我们使用苹果公司的股价。

以下是计算移动平均线的步骤：

1.  **导入数据并选择股票**：我们使用之前处理好的股票数据，并筛选出苹果公司的股价序列。
2.  **计算短期移动平均线**：我们定义一个时间窗口（例如10天），计算该窗口内股价的平均值，并随着时间滚动计算。
3.  **计算长期移动平均线**：同样，我们定义一个更长的时间窗口（例如30天），计算其移动平均值。

![](img/c898a261a084911b5a7d139da1a784d9_11.png)

![](img/c898a261a084911b5a7d139da1a784d9_13.png)

核心的计算公式是移动平均，在Pandas中可以通过 `rolling` 和 `mean` 方法轻松实现：

![](img/c898a261a084911b5a7d139da1a784d9_15.png)

![](img/c898a261a084911b5a7d139da1a784d9_17.png)

```python
# 假设 df 是包含苹果股价的DataFrame，列名为 ‘AAPL’
# 计算10日短期移动平均
short_window = 10
df[‘M1’] = df[‘AAPL’].rolling(window=short_window).mean()

# 计算30日长期移动平均
long_window = 30
df[‘M2’] = df[‘AAPL’].rolling(window=long_window).mean()
```

## 识别黄金交叉与死亡交叉

图表绘制好后，我们就可以识别关键的买卖信号了。这涉及到两个重要的技术分析术语：

![](img/c898a261a084911b5a7d139da1a784d9_35.png)

![](img/c898a261a084911b5a7d139da1a784d9_37.png)

![](img/c898a261a084911b5a7d139da1a784d9_39.png)

以下是黄金交叉与死亡交叉的定义：

*   **黄金交叉**：指短期移动平均线从下方向上穿越长期移动平均线。这通常被视为一个**看涨信号**，暗示股价可能开始上涨趋势，是潜在的买入时机。
*   **死亡交叉**：指短期移动平均线从上方向下穿越长期移动平均线。这通常被视为一个**看跌信号**，暗示股价可能开始下跌趋势，是潜在的卖出或观望时机。

在图表上，当绿色的短期均线（M1）上穿红色的长期均线（M2）时，就形成了黄金交叉；反之，当短期均线下穿长期均线时，则形成死亡交叉。

## 进阶：量化交叉信号

![](img/c898a261a084911b5a7d139da1a784d9_45.png)

为了更精确地定位交叉点，我们可以用代码量化这个信号。思路是创建一个新的列，用来标记短期均线是高于还是低于长期均线。

![](img/c898a261a084911b5a7d139da1a784d9_47.png)

以下是使用NumPy的 `where` 函数创建信号列的代码：

![](img/c898a261a084911b5a7d139da1a784d9_49.png)

```python
import numpy as np

![](img/c898a261a084911b5a7d139da1a784d9_51.png)

![](img/c898a261a084911b5a7d139da1a784d9_53.png)

# 生成交易信号：短期均线大于长期均线时为1，小于时为-1
df[‘position’] = np.where(df[‘M1’] > df[‘M2’], 1, -1)

![](img/c898a261a084911b5a7d139da1a784d9_55.png)

# 为了清晰展示，使用双Y轴绘图：左轴显示股价，右轴显示信号（1或-1）
fig, ax1 = plt.subplots(figsize=(12,6))

ax1.plot(df[‘AAPL’], label=‘Price’, color=‘blue’, alpha=0.5)
ax1.plot(df[‘M1’], label=‘Short MA’, color=‘green’)
ax1.plot(df[‘M2’], label=‘Long MA’, color=‘red’)
ax1.set_ylabel(‘Price’)

![](img/c898a261a084911b5a7d139da1a784d9_57.png)

ax2 = ax1.twinx() # 创建共享X轴的第二个Y轴
ax2.plot(df[‘position’], label=‘Signal’, color=‘purple’, linestyle=‘--’)
ax2.set_ylabel(‘Signal (1/-1)’)
ax2.set_yticks([-1, 1])

![](img/c898a261a084911b5a7d139da1a784d9_59.png)

fig.legend(loc=‘upper left’)
plt.show()
```

![](img/c898a261a084911b5a7d139da1a784d9_61.png)

在这张图上，紫色虚线在 **+1** 和 **-1** 之间切换。当信号从 **-1** 切换到 **+1** 时，对应着黄金交叉（买入信号）；当信号从 **+1** 切换到 **-1** 时，则对应着死亡交叉（卖出信号）。这使得交叉点的识别变得非常机械和明确。
