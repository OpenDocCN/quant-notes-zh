## 中性化（提纯）原理 🔍

上一节我们介绍了因子处理的基本流程。本节中，我们来看看如何将一个因子中我们不希望包含的成分剔除出去，这个过程称为“中性化”或“提纯”。

例如，我们有一个市净率（PB）因子。我们发现市净率与市值（Size）存在较强的相关性。如果我们希望得到一个“纯净”的、不受市值影响的市净率因子，就需要从市净率中剔除掉市值所能解释的那部分信息。

这个过程可以借助线性回归模型来实现。我们将因子（如市净率）视为因变量 `Y`，将希望剔除的影响因素（如市值）视为自变量 `X`。通过建立回归方程，我们可以量化 `X` 对 `Y` 的解释程度。

**核心公式如下：**
```
Y ≈ W * X + B
```
其中：
*   `Y` 代表原始因子值（真实值）。
*   `X` 代表需要剔除的影响因素（如市值）。
*   `W` 和 `B` 是通过回归分析得到的权重和截距项。
*   `W * X + B` 代表由 `X` 所解释的 `Y` 的部分（预测值）。

真实值 `Y` 与预测值 `(W * X + B)` 之间的差值，即回归的残差，就是剔除掉 `X` 影响后“提纯”的因子值。

**提纯后的因子 = Y - (W * X + B)**

这个残差代表了原始因子中无法被 `X`（如市值）解释的部分，也就是我们想要的、相对“纯净”的因子信息。

## 中性化操作步骤 📝

![](img/890a252153d6881724ac5f85ff35683b_3.png)

理解了原理后，我们来看看具体的操作步骤。以下是执行因子中性化的两个关键步骤：

第一步，建立回归方程。以需要提纯的因子（如市净率）作为因变量 `Y`，以需要剔除的影响因素（如市值）作为自变量 `X`，进行线性回归，求解出权重 `W` 和截距 `B`。

第二步，计算提纯后的因子。用因子的真实值 `Y` 减去回归方程得到的预测值 `(W * X + B)`，所得的残差即为中性化处理后的新因子值。

## 策略任务流程 🚀

![](img/890a252153d6881724ac5f85ff35683b_7.png)

在完成因子的预处理（包括去极值、标准化、中性化）后，我们就可以基于处理好的因子来构建选股策略了。本节我们将概述一个简单的策略流程。

整个策略任务将遵循以下步骤进行：

![](img/890a252153d6881724ac5f85ff35683b_9.png)

![](img/890a252153d6881724ac5f85ff35683b_10.png)

![](img/890a252153d6881724ac5f85ff35683b_11.png)

1.  **数据准备**：获取市净率（PB）和市值（Size）等基础因子数据。
2.  **股票池过滤**：从初始股票池中剔除不符合条件的股票，例如ST股、停牌股、上市时间过短（如少于半年）的股票等。
3.  **因子处理**：对保留股票的相关因子依次进行去极值、标准化和中性化处理。
4.  **制定选股规则**：基于处理后的因子设定具体的选股条件。例如，一个简单的规则可以是：**选择“提纯后市净率”小于 0.2 的所有股票**。
5.  **执行策略**：根据选股规则，生成最终的买入股票列表，完成一次选股操作。

在这个过程中，中性化处理是确保因子选股逻辑有效性的关键一环，它能帮助我们发现更独立、更有效的选股信号。

## 总结 ✨

![](img/890a252153d6881724ac5f85ff35683b_15.png)

![](img/890a252153d6881724ac5f85ff35683b_16.png)

本节课中我们一起学习了因子中性化（提纯）的核心概念与操作方法。我们了解到，中性化本质上是通过线性回归剥离因子中其他影响因素（如市值）的过程，其结果是得到更“纯净”的因子信号。随后，我们概述了如何将处理后的因子融入一个完整的选股策略流程中，从数据准备、清洗、处理到最终制定并执行选股规则。掌握这一流程是构建量化选股模型的重要基础。
