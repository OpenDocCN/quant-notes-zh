## 策略需求与核心逻辑分析

![](img/ab9979a7f38c63b89d55039515207deb_11.png)

![](img/ab9979a7f38c63b89d55039515207deb_13.png)

上一节我们识别出了股票历史数据中的金叉和死叉信号。本节中，我们将利用这些信号来执行买卖操作。

我们的目标是：假设一位投资者从2010年1月1日开始，初始资金为10万元。每当出现金叉信号时，他使用可用资金尽可能多地买入股票；每当出现死叉信号时，他将手中持有的所有股票卖出。我们需要计算到数据截止日（例如2020年），他的总收益是多少。

以下是策略实现前需要明确的几个核心点：

![](img/ab9979a7f38c63b89d55039515207deb_29.png)

![](img/ab9979a7f38c63b89d55039515207deb_31.png)

![](img/ab9979a7f38c63b89d55039515207deb_33.png)

![](img/ab9979a7f38c63b89d55039515207deb_35.png)

![](img/ab9979a7f38c63b89d55039515207deb_37.png)

![](img/ab9979a7f38c63b89d55039515207deb_39.png)

1.  **买卖价格**：我们使用交易信号发生当日的**开盘价**作为买卖股票的单价。
2.  **买卖时机**：**金叉**时间点买入，**死叉**时间点卖出。
3.  **特殊情况处理**：如果数据最后一天是金叉（即买入后没有后续的死叉卖出），那么手中会持有未卖出的股票。这部分剩余股票的价值需要计入总收益。我们使用**最后一天的收盘价**来计算剩余股票的价值。
4.  **交易单位**：A股市场买卖股票的最小单位是“手”，1手等于100股。因此，“尽可能买入”意味着用可用资金除以（单价 × 100），得到能买入的最大整数手数。

## 数据准备：整合金叉与死叉信号

![](img/ab9979a7f38c63b89d55039515207deb_41.png)

首先，我们需要将之前分别存储的金叉时间 (`golden_date`) 和死叉时间 (`death_date`) 整合并筛选到2010年之后。

```python
# 创建两个Series，用1标记金叉时间，用0标记死叉时间
s1 = pd.Series(1, index=golden_date)  # 金叉标识
s2 = pd.Series(0, index=death_date)   # 死叉标识

![](img/ab9979a7f38c63b89d55039515207deb_43.png)

# 合并两个Series并按时间索引排序
s = s1.append(s2).sort_index()

# 筛选出2010年及之后的数据
s = s[‘2010’:’2020’]
```

```python
first_money = 100000  # 初始本金，固定不变，用于最终计算收益
money = first_money   # 可用资金，买卖股票时从此变量中增减
hold = 0              # 持有股票的数量（单位：股）
```

我们将遍历`s`中的每一个信号（1或0），并根据信号执行买卖操作。

以下是实现买卖逻辑的代码框架：

![](img/ab9979a7f38c63b89d55039515207deb_89.png)

![](img/ab9979a7f38c63b89d55039515207deb_91.png)

![](img/ab9979a7f38c63b89d55039515207deb_93.png)

```python
# 获取股票历史数据DataFrame，假设为 df
# df 应包含 ‘open’（开盘价）和 ‘close’（收盘价）等列

![](img/ab9979a7f38c63b89d55039515207deb_95.png)

![](img/ab9979a7f38c63b89d55039515207deb_97.png)

for i in range(len(s)):
    # 获取当前信号（1或0）及其对应的时间
    signal_value = s.iloc[i]
    current_time = s.index[i]

    if signal_value == 1:  # 金叉，买入
        # 1. 获取买入单价（当天开盘价）
        buy_price = df.loc[current_time, ‘open’]

        # 2. 计算最多能买多少手
        hand_count = money // (buy_price * 100)

        # 3. 如果可以买（hand_count > 0），则更新持仓和资金
        if hand_count > 0:
            # 买入股数 = 手数 * 100
            buy_count = hand_count * 100
            hold += buy_count
            # 花费资金 = 买入股数 * 单价
            cost = buy_count * buy_price
            money -= cost

    else:  # signal_value == 0，死叉，卖出
        if hold > 0:  # 如果有持仓才执行卖出
            # 1. 获取卖出单价（当天开盘价）
            sell_price = df.loc[current_time, ‘open’]

            # 2. 卖出全部持仓，更新资金
            revenue = hold * sell_price
            money += revenue

            # 3. 清空持仓
            hold = 0
```

## 计算最终收益

循环结束后，我们需要计算总收益。这里要处理两种可能的情况：
1.  最后一天是死叉：`hold` 已经为0，所有股票已卖出。
2.  最后一天是金叉：`hold` 大于0，还有股票未卖出。

![](img/ab9979a7f38c63b89d55039515207deb_99.png)

![](img/ab9979a7f38c63b89d55039515207deb_101.png)

![](img/ab9979a7f38c63b89d55039515207deb_102.png)

![](img/ab9979a7f38c63b89d55039515207deb_104.png)

![](img/ab9979a7f38c63b89d55039515207deb_106.png)

```python
# 计算剩余股票的价值（使用最后一天的收盘价）
last_price = df.iloc[-1][‘close’]  # 获取最后一天的收盘价
last_money = hold * last_price

![](img/ab9979a7f38c63b89d55039515207deb_108.png)

![](img/ab9979a7f38c63b89d55039515207deb_110.png)

![](img/ab9979a7f38c63b89d55039515207deb_112.png)

# 计算总收益
# 总收益 = 当前资金 + 剩余股票价值 - 初始本金
total_profit = money + last_money - first_money
print(f“最终总收益为：{total_profit}元”)
```

如果 `total_profit` 为正数，则表示盈利；为负数则表示亏损。
