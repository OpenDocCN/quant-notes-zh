## 🧠 能量型指标简介

上一节我们介绍了如何综合运用MA和RSI指标。本节中，我们来看看另一类重要的技术指标——能量型指标。能量型指标主要用于衡量市场买卖双方的力量对比和交易活跃度，从而判断市场的趋势和潜在转折点。

以下是本节策略将用到的三大类能量型指标：

1.  **情绪指标 (BRAR)**：通过反映市场的买卖人气来判断整体市场能量。BR代表买卖意愿指标，AR代表人气指标。
2.  **中间意愿指标 (CR)**：该指标认为股票的中间价最具代表性，并以此为基础设计，用于反映市场的多空意愿。
3.  **成交量变异率 (VR)**：从成交量的角度测量股价的热度，反映市场交易的活跃程度和冷热情况。

## 📈 买卖信号判断

![](img/b08b34b7aede71274712be34ed404061_1.png)

了解了指标的基本概念后，我们来看看如何结合这些指标来制定具体的买卖规则。

以下是基于多个能量型指标综合判断的买卖信号逻辑：

*   **买入信号**：当同时满足以下所有条件时，视为买入信号。
    *   `AR < 100`
    *   `BR < 100`
    *   `BR < AR`
    *   `CR < 100`
    *   `VR < 100`
*   **卖出信号**：当同时满足以下所有条件时，视为卖出信号。
    *   `AR > 150`
    *   `BR > 150`
    *   `CR > 150`
    *   `VR > 150`

> 注意：这些阈值（100和150）是经过市场研究和实践总结得出的经验值。在实际应用中，投资者可以根据不同的市场和股票特性进行优化调整。
