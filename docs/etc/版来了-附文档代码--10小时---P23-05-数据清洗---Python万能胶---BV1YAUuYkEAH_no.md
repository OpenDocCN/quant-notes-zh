## 数据清洗步骤详解
数据清洗通常遵循一套标准化的操作步骤。以下是针对我们案例数据的具体清洗过程。

![](img/f4f5d33a38fdd6788984064ad110cb5c_7.png)

### 1. 删除重复值
在分析中，重复的数据记录会导致结果失真。我们的目标是确保每条招聘信息都是唯一的。

**核心思路**：通常，每条招聘职位都对应一个唯一的ID。我们可以将“职位ID”列作为判断重复的依据。

![](img/f4f5d33a38fdd6788984064ad110cb5c_9.png)

**操作流程如下**：
1.  **定位重复值**：选中“职位ID”列，点击【开始】->【条件格式】->【突出显示单元格规则】->【重复值】。系统会将所有重复的ID标记出来。
2.  **筛选并检查**：对“职位ID”列启用筛选，按颜色筛选出标记为重复的行。仔细检查这些行，确认它们是完全相同的冗余数据。
3.  **删除重复值**：确认无误后，选中数据区域，点击【数据】->【删除重复值】。在弹出的对话框中，仅勾选“职位ID”列，点击确定。系统会提示删除了多少个重复值，保留了多少个唯一值。

![](img/f4f5d33a38fdd6788984064ad110cb5c_11.png)

> **提示**：在删除前先筛选检查是一个好习惯，可以避免误删。

![](img/f4f5d33a38fdd6788984064ad110cb5c_13.png)

### 2. 处理缺失值
缺失值（空值）是数据集中常见的问题。我们需要根据缺失情况采取不同的策略。

![](img/f4f5d33a38fdd6788984064ad110cb5c_15.png)

**首先，我们需要找出缺失值**。逐列筛选查看效率低下，这里我们使用 `COUNTIF` 函数进行批量统计。
```excel
=COUNTIF(区域, “”)
```
例如，统计A列的空值数量：`=COUNTIF(A:A, “”)`。结果为2，表示A列有2个空值。

**针对缺失值，一般有三种处理方案**：
1.  **直接删除**：当缺失的数据量很少，且删除后对整体分析影响不大时采用。
2.  **人工判断填充**：适用于文本型数据，且可以根据其他列信息进行合理推断时采用。例如，我们的“城市”列缺失了2条，但根据“公司名称”或“商区”可以推断出是“上海”，则可手动填写。
3.  **不予分析**：当某一列缺失值数量巨大，填充或删除都会严重影响数据完整性时，可以考虑在后续分析中忽略这一列。例如，我们的“商区”列缺失了近1500条数据，占比很高，后续分析将不包含此维度。

![](img/f4f5d33a38fdd6788984064ad110cb5c_17.png)

### 3. 数据一致化处理
数据一致化是指将数据转换为统一、规范的格式，以便于计算和分析。在我们的数据中，“薪水”列是一个文本区间（如“20K-30K”），不便于直接计算平均薪资等指标。

**操作目标**：将“薪水”拆分为“最低薪资”、“最高薪资”，并计算“平均薪资”。

**操作流程如下**：
1.  **分列**：选中“薪水”列，点击【数据】->【分列】。选择“分隔符号”，点击下一步，在“其他”框中输入分隔符“-”，点击完成。这样就将一个单元格拆分成了两列。
2.  **清理单位**：拆分后的数据带有“K”或“以上/以下”等文本。使用查找替换功能（`Ctrl+H`），将“K”、“以上”、“以下”全部替换为空，使其变为纯数字。
3.  **计算平均值**：新增一列“平均薪资”，使用公式计算（最低薪资 + 最高薪资）/ 2。
    ```excel
    =AVERAGE(最低薪资单元格, 最高薪资单元格)
    ```
    或者
    ```excel
    = (最低薪资单元格 + 最高薪资单元格) / 2
    ```

### 4. 处理异常值/筛选目标数据
有时数据中包含与分析目标不符的记录，需要过滤。本例中，我们需要聚焦于“数据分析师”岗位，但原始数据包含“数据工程师”、“架构师”等其他职位。

![](img/f4f5d33a38fdd6788984064ad110cb5c_19.png)

![](img/f4f5d33a38fdd6788984064ad110cb5c_21.png)

**核心思路**：判断“职位名称”是否包含关键词（如“数据”、“分析”、“分析师”），从而筛选出目标岗位。

**我们可以使用 `FIND` 函数结合 `COUNT` 函数来实现**：
```excel
=IF(COUNT(FIND({“数据”,“分析”,“分析师”}, 职位名称单元格))>0, “是”, “否”)
```
*   `FIND(查找文本, 源文本)`：在源文本中查找特定文本，返回其起始位置（数字），如果找不到则返回错误值。
*   `COUNT(值)`：计算参数列表中数字的个数。
*   `IF(条件, 真值, 假值)`：根据条件判断返回不同结果。

**公式解释**：该公式会依次查找“职位名称”中是否出现“数据”、“分析”、“分析师”这些词。只要找到任何一个，`COUNT` 结果就大于0，`IF` 函数即返回“是”，否则返回“否”。

最后，筛选出标记为“是”的行，将其复制到新的工作表中，即可得到纯净的、专注于数据分析师岗位的数据集。

## 总结
本节课我们一起学习了数据清洗的核心方法与实战操作。我们以一个招聘数据分析项目为例，完整演练了：
1.  **删除重复值**：确保数据的唯一性。
2.  **处理缺失值**：根据数据情况选择删除、填充或忽略。
3.  **数据一致化**：将数据转换为统一格式（如拆分薪资区间并计算平均值）。
4.  **筛选目标数据**：使用函数过滤出与分析目标相关的记录。

![](img/f4f5d33a38fdd6788984064ad110cb5c_25.png)

数据清洗是数据分析的基石，虽然过程可能繁琐，但干净的数据是产出准确、可靠分析结论的前提。掌握这些技能，你就能为任何数据分析项目打下坚实的基础。接下来，你就可以利用这份清洗好的数据，开始进行各种维度的分析了。
