## 简单增长率累加的问题

上一节我们介绍了如何计算增长率。本节中我们来看看，能否将一段时间内每天的增长率简单累加，来代表整体的增长情况。

例如，假设一只股票的价格变化如下：
*   第一天：100元
*   第二天：50元
*   第三天：75元

![](img/461fb92720f7eeed9698eef83f7484c3_4.png)

我们分别计算每日的增长率：
*   第二天增长率 = (50 - 100) / 100 = **-0.5**
*   第三天增长率 = (75 - 50) / 50 = **0.5**

如果将这两个增长率累加：-0.5 + 0.5 = **0**。这个结果意味着“整体增长率为0”，即股票价格没有变化。但观察原始数据，股价从100元变成了75元，实际上是下跌的。

**结论**：对于连续时间的序列，简单的增长率加法操作是无效的，因为它不能正确反映第一天与最后一天的整体差异。

## 使用对数计算连续变化

![](img/461fb92720f7eeed9698eef83f7484c3_6.png)

![](img/461fb92720f7eeed9698eef83f7484c3_8.png)

为了解决上述问题，我们需要借助数学中的对数公式。核心思想是：**对数的加法等价于原数的乘法**。

我们计算连续的对数收益率：
`log(第二天价格 / 第一天价格) + log(第三天价格 / 第二天价格)`

![](img/461fb92720f7eeed9698eef83f7484c3_10.png)

根据对数运算法则，上式等于：
`log( (第二天价格 / 第一天价格) * (第三天价格 / 第二天价格) )`

简化后得到：
`log(第三天价格 / 第一天价格)`

这正好反映了**第一天与第三天之间的整体价格变化关系**。推广开来，将一段时间内所有的对数收益率累加，结果就等于最后一天与第一天的价格比值的对数。

![](img/461fb92720f7eeed9698eef83f7484c3_12.png)

![](img/461fb92720f7eeed9698eef83f7484c3_14.png)

![](img/461fb92720f7eeed9698eef83f7484c3_16.png)

形象地说，这解决了“如果初始投资1元钱，经过一系列涨跌后，最终相当于多少钱”的问题。

### 步骤1：计算对数收益率

![](img/461fb92720f7eeed9698eef83f7484c3_30.png)

首先，我们需要计算每天相对于前一天的比率，然后取自然对数。这里使用Pandas的`.shift()`函数来获取前一天的数据。

![](img/461fb92720f7eeed9698eef83f7484c3_32.png)

```python
# 假设`data`是一个包含股价的Pandas Series或DataFrame
# 计算当天价格与前一天价格的比率，并取自然对数
returns = np.log(data / data.shift(1))
print(returns.head())
```

![](img/461fb92720f7eeed9698eef83f7484c3_34.png)

![](img/461fb92720f7eeed9698eef83f7484c3_36.png)

### 步骤2：计算累积对数收益率

接下来，我们需要对对数收益率进行累加，这可以通过`np.cumsum()`函数实现。这个函数会返回一个序列，其中每个位置的值是原始序列从开始到当前位置的累加和。

![](img/461fb92720f7eeed9698eef83f7484c3_38.png)

例如，对于序列 `[1, 2, 3, 4]`，`np.cumsum()`的结果是 `[1, 3, 6, 10]`。

![](img/461fb92720f7eeed9698eef83f7484c3_40.png)

```python
# 计算累积对数收益率
cumulative_log_returns = returns.cumsum()
```

![](img/461fb92720f7eeed9698eef83f7484c3_42.png)

![](img/461fb92720f7eeed9698eef83f7484c3_44.png)

### 步骤3：还原为实际价值变化

![](img/461fb92720f7eeed9698eef83f7484c3_46.png)

由于我们之前取了对数，现在需要将其还原为实际的价值倍数。这通过指数函数 `np.exp()` 来实现。

![](img/461fb92720f7eeed9698eef83f7484c3_48.png)

```python
# 将累积对数收益率还原为实际的价值倍数（以初始1元为基准）
normalized_price = np.exp(cumulative_log_returns)
```

![](img/461fb92720f7eeed9698eef83f7484c3_50.png)

### 步骤4：可视化结果

![](img/461fb92720f7eeed9698eef83f7484c3_52.png)

最后，我们可以将结果绘制成图，直观地看到每只股票（或每个指标）从起始点开始的价值增长轨迹。

![](img/461fb92720f7eeed9698eef83f7484c3_54.png)

```python
# 绘制归一化后的价格走势图
normalized_price.plot(figsize=(16, 8))
plt.title(‘初始1元投资的增长情况’)
```

## 核心方法总结

![](img/461fb92720f7eeed9698eef83f7484c3_60.png)

![](img/461fb92720f7eeed9698eef83f7484c3_62.png)

本节课中我们一起学习了金融时间序列中连续变化情况的分析方法。

![](img/461fb92720f7eeed9698eef83f7484c3_64.png)

1.  **差异值计算**：使用`.diff()`函数可以方便地计算相邻数据的差值。
2.  **简单增长率**：公式为 `(后值 - 前值) / 前值`，反映了单期的变化比例。
3.  **连续变化分析（对数收益率）**：
    *   **问题**：简单增长率无法正确累加以反映长期整体变化。
    *   **解决方案**：引入自然对数，计算对数收益率 `log(后值 / 前值)`。
    *   **累加**：对数收益率可以安全地进行累加 (`np.cumsum`)，其累加和等价于 `log(最终值 / 初始值)`。
    *   **还原**：通过对累加的对数收益率取指数 (`np.exp`)，可以得到“初始1单位对应的最终价值”这一直观指标。

通过这套方法，我们能够准确地评估资产在一段连续时间内的整体表现，这是金融分析和量化交易中一项基础且重要的技能。
