## 什么是聚类？🤔

上一节我们介绍了有监督学习，本节中我们来看看无监督学习中的典型代表——聚类。

聚类要解决的问题与分类类似，都是将数据分组。但关键区别在于，聚类是一种无监督学习，这意味着我们处理的数据**没有预先给定的标签**。我们的目标是从看似杂乱的数据中，根据数据点之间的相似性，自动将它们划分成不同的组或“簇”。

例如，观察右侧的散点图，原始数据并没有颜色标记。聚类算法会根据一种相似度度量方式，将彼此靠近（相似）的点归为一组，最终可能得到绿色、红色和蓝色三个不同的簇。

---

## 聚类的挑战 ⚠️

虽然聚类算法的原理可能看起来简单，但它面临一些独特的挑战，主要源于“无标签”这一特性。

![](img/aa354bc2dce64489600f4a62bae01255_2.png)

以下是聚类任务中的主要难点：

*   **评估困难**：在有监督学习中，我们可以通过比较预测标签和真实标签来计算准确率、F1分数等指标。但在无监督聚类中，由于没有“标准答案”，评估聚类结果的好坏变得非常困难。
*   **参数调节困难**：例如，使用不同的参数进行聚类会得到不同的结果。但由于缺乏评估标准，很难判断哪一组参数得到的结果更优。

## K-Means算法核心概念 🧠

接下来，我们将深入探讨K-Means算法，这是最经典、应用最广的聚类算法之一。

![](img/aa354bc2dce64489600f4a62bae01255_8.png)

### 1. K值（簇的数量）

K-Means算法需要我们预先指定一个关键参数：**K值**。K值代表你希望算法将数据聚合成多少个簇。

![](img/aa354bc2dce64489600f4a62bae01255_10.png)

*   如果设定 `K=3`，算法会将数据聚成3堆。
*   如果设定 `K=4`，则会聚成4堆。

![](img/aa354bc2dce64489600f4a62bae01255_12.png)

**公式表示**：`K` 是一个需要用户定义的超参数。

![](img/aa354bc2dce64489600f4a62bae01255_14.png)

### 2. 质心

质心是K-Means算法中另一个核心概念。**质心**代表一个簇的中心点，通常计算为该簇内所有数据点在各个维度上的平均值。

例如，在二维空间中，一个簇的质心由其所有点的X坐标平均值和Y坐标平均值共同确定。

**代码描述**：对于一个包含 `m` 个样本点的簇，其质心 `centroid` 的计算方式为：
```python
centroid_x = (x1 + x2 + ... + xm) / m
centroid_y = (y1 + y2 + ... + ym) / m
```

### 3. 距离度量

如何判断两个数据点是否“相似”？我们通过计算它们之间的**距离**来衡量。距离越小，通常认为相似度越高。

*   **欧氏距离**：最常用的距离度量方式。在二维空间中，点A(x1, y1)和点B(x2, y2)之间的欧氏距离计算公式为：
    **`distance = sqrt((x1-x2)^2 + (y1-y2)^2)`**
*   **数据标准化**：在使用距离度量（尤其是欧氏距离）前，**必须**对数据进行标准化（如归一化）。这是因为如果不同特征（维度）的数值范围差异巨大（例如一个特征值在0~1，另一个在100~1000），数值范围大的特征会主导距离计算，导致结果失真。标准化的目的是使所有特征处于相近的数值尺度（如0~1或均值为0、方差为1）。

![](img/aa354bc2dce64489600f4a62bae01255_16.png)

### 4. 优化目标

![](img/aa354bc2dce64489600f4a62bae01255_18.png)

与大多数机器学习算法一样，K-Means通过优化一个目标函数来工作。它的目标是：**最小化每个簇中所有样本点到该簇质心的距离之和**。

**公式表示**：
**`Minimize J = Σ（i=1 to K） Σ（x in Cluster_i） distance(x, centroid_i)^2`**

![](img/aa354bc2dce64489600f4a62bae01255_20.png)

这个公式的意思是：对于每一个簇（`i`从1到`K`），计算簇内每一个样本点 `x` 到该簇质心 `centroid_i` 的距离（通常用平方欧氏距离），并将所有距离求和。算法迭代的目标就是让这个总和 `J` 尽可能小。

![](img/aa354bc2dce64489600f4a62bae01255_22.png)

**通俗理解**：算法试图让每一个簇内部的点都尽可能紧密地围绕在质心周围。如果一个点离当前簇的质心很远，但离另一个簇的质心更近，那么在迭代过程中，它就会被重新分配到更近的那个簇中去。

## 总结 📝

本节课我们一起学习了聚类的基本概念以及K-Means算法的核心原理。

![](img/aa354bc2dce64489600f4a62bae01255_24.png)

我们了解到：
1.  聚类是一种无监督学习方法，用于将无标签的数据按相似性分组。
2.  K-Means算法需要预先指定簇的数量 `K`。
3.  算法的核心是**质心**和**距离度量**，通过迭代优化，使每个样本点到其所属簇质心的距离之和最小。
4.  使用距离度量前，对数据进行标准化是至关重要的步骤。

![](img/aa354bc2dce64489600f4a62bae01255_26.png)

K-Means算法因其简单、高效而广泛应用，是理解聚类任务的绝佳起点。
