上一节我们介绍了如何计算简单的差异值和增长率。本节中我们来看看如何处理连续时间序列的增长率累加问题。

## 为什么增长率不能直接累加？ 🤔

![](img/2e8081327d78ef671595e785ba4227a0_4.png)

当我们观察股票等金融产品的价格变化时，常常会计算每日的增长率。然而，将这些每日增长率简单累加，并不能准确反映从起始日到最终日的整体增长情况。

以下是一个例子：
*   第一天股价：100元
*   第二天股价：50元
*   第三天股价：75元

如果计算每日增长率并累加：
*   第二天增长率：(50 - 100) / 100 = -0.5
*   第三天增长率：(75 - 50) / 50 = 0.5
*   累加结果：-0.5 + 0.5 = 0

![](img/2e8081327d78ef671595e785ba4227a0_6.png)

累加结果为零，似乎表示股价没有变化。但实际从第一天（100元）到第三天（75元），股价是下跌的。这说明**增长率不能直接进行加法运算**。

![](img/2e8081327d78ef671595e785ba4227a0_8.png)

## 如何计算连续时间的累积变化？ ➕

![](img/2e8081327d78ef671595e785ba4227a0_10.png)

为了准确计算连续时间的累积变化，我们需要借助数学中的对数。

**核心思想**：将对数形式的每日变化率进行累加，其本质是计算每日价格比值的连乘，最终得到首日与末日的总比值关系。

**计算公式**：
`总累积变化 = exp( sum( log(当日价格 / 前日价格) ) )`

![](img/2e8081327d78ef671595e785ba4227a0_12.png)

![](img/2e8081327d78ef671595e785ba4227a0_14.png)

![](img/2e8081327d78ef671595e785ba4227a0_16.png)

这个公式等价于：
`总累积变化 = (第N日价格 / 第1日价格)`

![](img/2e8081327d78ef671595e785ba4227a0_18.png)

![](img/2e8081327d78ef671595e785ba4227a0_20.png)

![](img/2e8081327d78ef671595e785ba4227a0_22.png)

这意味着，通过对数变化率进行累加，我们可以直接得到**初始的1单位投资，在经历一系列涨跌后，最终相当于多少单位**。

![](img/2e8081327d78ef671595e785ba4227a0_24.png)

## 在Python中实现连续变化分析 💻

![](img/2e8081327d78ef671595e785ba4227a0_26.png)

![](img/2e8081327d78ef671595e785ba4227a0_28.png)

以下是实现步骤：

![](img/2e8081327d78ef671595e785ba4227a0_30.png)

![](img/2e8081327d78ef671595e785ba4227a0_32.png)

### 1. 计算每日的对数变化率
首先，我们需要计算每日价格相对于前一日价格的对数比值。

![](img/2e8081327d78ef671595e785ba4227a0_34.png)

```python
import pandas as pd
import numpy as np

![](img/2e8081327d78ef671595e785ba4227a0_36.png)

![](img/2e8081327d78ef671595e785ba4227a0_38.png)

# 假设 `data` 是一个包含多只股票收盘价的DataFrame
# 计算每日的对数变化率：log(今日收盘价 / 昨日收盘价)
returns = np.log(data / data.shift(1))
```

*   `data.shift(1)` 将数据整体向上移动一行，这样每一行就变成了前一交易日的数据。
*   `np.log()` 计算以e为底的自然对数。

![](img/2e8081327d78ef671595e785ba4227a0_40.png)

![](img/2e8081327d78ef671595e785ba4227a0_42.png)

### 2. 计算累积变化
接下来，我们对这些每日的对数变化率进行累加，然后通过指数函数还原为实际的倍数关系。

![](img/2e8081327d78ef671595e785ba4227a0_44.png)

```python
# 计算累积对数收益率（即对数变化率的累计和）
cumulative_log_returns = returns.cumsum()

![](img/2e8081327d78ef671595e785ba4227a0_46.png)

![](img/2e8081327d78ef671595e785ba4227a0_48.png)

# 将累积对数收益率还原为实际的资产倍数变化
# 即：初始的1单位，最终变成了多少单位
cumulative_change = np.exp(cumulative_log_returns)
```

![](img/2e8081327d78ef671595e785ba4227a0_50.png)

*   `.cumsum()` 方法实现累加操作，例如序列[1, 2, 3]经过`.cumsum()`后变为[1, 3, 6]。
*   `np.exp()` 是指数函数，用于将对数结果还原回原始倍数。

![](img/2e8081327d78ef671595e785ba4227a0_52.png)

### 3. 可视化结果
最后，我们可以将计算结果绘制成图，直观展示每只股票从起始日开始的累积增长轨迹。

![](img/2e8081327d78ef671595e785ba4227a0_54.png)

```python
import matplotlib.pyplot as plt

![](img/2e8081327d78ef671595e785ba4227a0_56.png)

![](img/2e8081327d78ef671595e785ba4227a0_58.png)

# 绘制累积变化曲线
cumulative_change.plot(figsize=(16, 8))
plt.title(‘初始1单位投资的累积增长情况’)
plt.ylabel(‘相当于初始单位的倍数’)
plt.xlabel(‘日期’)
plt.show()
```

![](img/2e8081327d78ef671595e785ba4227a0_60.png)

在这张图中，纵轴表示“初始的1单位投资相当于当前多少单位”。例如，某条曲线在2018年的值约为10，就意味着如果在2010年投资该股票1元，到2018年其价值大约增长为10元（未考虑股息等因素）。

本节课中我们一起学习了金融时间序列中连续变化情况的分析方法：

![](img/2e8081327d78ef671595e785ba4227a0_72.png)

![](img/2e8081327d78ef671595e785ba4227a0_74.png)

1.  **增长率累加的局限性**：直接累加每日增长率无法反映资产的整体变化，因为乘法效应被错误地处理为加法。
2.  **引入对数方法**：通过计算每日价格比值的对数 (`np.log(今日/昨日)`)，可以将乘法关系转化为加法关系，从而允许进行有意义的累加。
3.  **计算累积变化**：
    *   步骤一：计算每日对数变化率 `returns = np.log(data / data.shift(1))`。
    *   步骤二：对对数变化率进行累加 `cumulative_log_returns = returns.cumsum()`。
    *   步骤三：通过指数函数还原为实际倍数 `cumulative_change = np.exp(cumulative_log_returns)`。
4.  **结果解读**：最终得到的 `cumulative_change` 序列，直观展示了**从时间序列起点开始，初始的1单位投资随时间推移所对应的价值倍数**，这是一个在量化分析中非常常用的指标。
