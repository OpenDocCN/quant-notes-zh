# 量化交易实战：P31：3Sigma方法实例 📊

在本节课中，我们将要学习一种在量化交易中常用的数据预处理方法——3Sigma方法。该方法基于统计学原理，用于识别和处理数据中的异常值（极值），确保后续分析的稳定性和可靠性。

## 正态分布与异常值原理

上一节我们介绍了去极值的概念，本节中我们来看看基于统计学原理的3Sigma方法。理解该方法，首先需要了解正态分布。

正态分布，也称为高斯分布，是一种常见的概率分布。如果理解起来有困难，可以将其视为一种“正常”的分布模式。例如，在银行贷款场景中，银行批准给你的额度在期望值（均值）附近有小幅波动是正常的，但如果偏差过大（例如多或少借10万元），则可能被视为异常情况。

![](img/119628b6cbf7b914144c0e7299ec9747_1.png)

我们来看一张标准正态分布的示意图。图中有一条中心线，代表均值（μ），通常设为0。数据点围绕均值分布，距离均值越近的区域，数据点出现的概率（即图中阴影面积）越大；距离均值越远的区域，数据点出现的概率则越小。

![](img/119628b6cbf7b914144c0e7299ec9747_1.png)

![](img/119628b6cbf7b914144c0e7299ec9747_3.png)

我们的目标是去除数据中的极值（异常值）。极值通常不会落在均值附近的高概率区域，而更可能落在远离均值的低概率区域。离均值越远、偏差越大的数据点，越有可能是离群点。

## 3Sigma方法的核心思想

那么，什么是3Sigma（三倍标准差）方法呢？其核心思想基于正态分布的特性。

在正态分布中，我们可以根据均值（μ）和标准差（σ）来划定一个区间。标准差衡量了数据的离散程度。
*   **μ ± 1σ** 的区间包含了约68.3%的数据。
*   **μ ± 2σ** 的区间包含了约95.4%的数据。
*   **μ ± 3σ** 的区间包含了约99.7%的数据。

![](img/119628b6cbf7b914144c0e7299ec9747_5.png)

![](img/119628b6cbf7b914144c0e7299ec9747_3.png)

可以看到，区间范围（即加减的标准差倍数）越大，数据点落在这个区间外的可能性就越小。

![](img/119628b6cbf7b914144c0e7299ec9747_7.png)

![](img/119628b6cbf7b914144c0e7299ec9747_9.png)

3Sigma方法就是：假设数据经过某种变换后，理论上服从正态分布。我们设定上下限为 **μ ± 3σ**，认为落在这个区间内（约99.7%）的数据是“正常”的，而落在这个区间外（约0.3%）的数据则被视为“极值”并进行处理。

![](img/119628b6cbf7b914144c0e7299ec9747_11.png)

因此，我们只需要计算数据的**均值（μ）**和**标准差（σ）**，就能确定处理极值的上限和下限。

**上限公式：** `upper = μ + n * σ`
**下限公式：** `lower = μ - n * σ`
其中，`n` 通常取3。

![](img/119628b6cbf7b914144c0e7299ec9747_13.png)

![](img/119628b6cbf7b914144c0e7299ec9747_15.png)

## 3Sigma方法的代码实现

理解了原理后，我们来看看如何用Python代码实现3Sigma方法。我们将编写一个函数，其逻辑与之前的方法类似。

![](img/119628b6cbf7b914144c0e7299ec9747_17.png)

![](img/119628b6cbf7b914144c0e7299ec9747_19.png)

以下是实现步骤：

![](img/119628b6cbf7b914144c0e7299ec9747_21.png)

1.  **计算均值和标准差**：对于输入的数据列，计算其均值（`mean`）和标准差（`std`）。
2.  **计算上下限**：根据公式 `均值 ± n * 标准差` 计算上限和下限。
3.  **限制数据范围**：将所有数据规范到计算出的上下限之间。

![](img/119628b6cbf7b914144c0e7299ec9747_23.png)

对应的Python函数代码如下：

![](img/119628b6cbf7b914144c0e7299ec9747_25.png)

![](img/119628b6cbf7b914144c0e7299ec9747_27.png)

```python
def sigma_cut(data, n=3):
    """
    使用3Sigma方法处理极值。
    参数:
        data: pandas Series，待处理的数据列。
        n: int，标准差的倍数，默认为3。
    返回:
        series: 处理后的数据。
        lower: 计算出的下限。
        upper: 计算出的上限。
    """
    # 1. 计算均值和标准差
    mean = data.mean()
    std = data.std()

    # 2. 计算上下限
    upper = mean + n * std
    lower = mean - n * std

    # 3. 将数据限制在上下限之间
    series = data.clip(lower=lower, upper=upper)

    return series, lower, upper
```

## 本节总结

![](img/119628b6cbf7b914144c0e7299ec9747_47.png)

本节课中我们一起学习了3Sigma去极值方法。
*   其核心是**基于数据服从正态分布的假设**，利用**均值（μ）**和**标准差（σ）** 来计算正常数据的范围（**μ ± nσ**）。
*   我们将落在该范围外的数据视为极值，并将其规范到边界上，**而不是直接删除**。
*   这是量化交易数据预处理“三步走”中，**第一步“去极值”** 的常用方法之一，目的是使数据更加稳健，为后续分析建模打下基础。
