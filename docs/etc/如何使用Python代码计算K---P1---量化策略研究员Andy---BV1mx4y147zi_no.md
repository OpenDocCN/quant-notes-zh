在本节课中，我们将学习一个经典的技术分析指标——KDJ指标。我们将从基础概念入手，逐步拆解其计算公式，并使用Python代码完整实现计算过程。通过本课，你将掌握如何从原始价格数据计算出KDJ值。

## 数据准备与参数设定

![](img/b75dee980b0c1edc5f8230d6bf007251_0.png)

首先，计算KDJ指标需要获取每日的价格数据，包括最高价、最低价和收盘价。KDJ指标有三个常用参数，我们采用默认设置（9， 3， 3）。

## 计算RSV值

在计算K值之前，我们需要先计算未成熟随机值（RSV）。RSV反映了当前收盘价在最近N天价格区间中的相对位置。

以下是计算RSV的公式：

`RSV = (今日收盘价 - 最近N日最低价) / (最近N日最高价 - 最近N日最低价) * 100`

其中，N通常为9。

## 计算K值与D值

上一节我们介绍了RSV的计算，本节中我们来看看如何由RSV推导出K值和D值。

K值是RSV值的指数移动平均（EMA）。在代码中，我们使用`pandas`库的`ewm`函数，并设置参数`span=2*M1-1`，这里的`M1`值为3。

因此，`span = 2*3 - 1 = 5`，即对RSV进行5日的指数移动平均。

你可能会问：K值的公式不是`今日K值 = 前一日K值 * 2/3 + 今日RSV * 1/3`吗？为什么这里用5日EMA？让我们展开EMA的计算公式来看。

当`N=5`时，EMA的递推公式为：
`Y = (N-1)/(N+1) * Y’ + 2/(N+1) * X`
代入N=5：
`Y = 4/6 * Y’ + 2/6 * X = 2/3 * Y’ + 1/3 * X`

这里的`Y’`就是前一天的K值，`X`就是今天的RSV。这与KDJ中K值的计算公式完全一致。关于指数移动平均更详细的讲解，可以参考相关技术分析专题视频。

D值是K值的指数移动平均，其计算方法与计算K值的过程完全相同，即对K值序列再次进行3日（`M2=3`）的指数移动平均。

以下是计算K值和D值的核心代码步骤：

1.  初始化K值序列，首日K值通常设为50。
2.  使用循环或向量化方法，根据递推公式`K_t = 2/3 * K_{t-1} + 1/3 * RSV_t`计算每日K值。
3.  同样方法，初始化D值后，根据`D_t = 2/3 * D_{t-1} + 1/3 * K_t`计算每日D值。

## 计算J值

最后，J值是一个辅助指标，由K值和D值推导得出，其计算公式非常简单：

`J = 3 * K - 2 * D`

![](img/b75dee980b0c1edc5f8230d6bf007251_2.png)

J值反映了K值与D值的乖离程度，波动通常比K线和D线更为敏感。

## 总结

本节课中我们一起学习了KDJ技术指标的核心计算流程。我们从获取高、低、收盘价数据开始，先计算了9日RSV；然后深入分析了K值实为RSV的5日指数移动平均，并理解了其递推公式的原理；接着，我们知道了D值是K值的3日指数移动平均；最后计算了J值。掌握这一计算过程是后续进行指标分析和策略构建的基础。
