## 概述

![](img/7a697bd48e9a88991beb33886e288f5f_3.png)

最大回撤是评估投资策略风险的关键指标。它描述的是在特定时间段内，资产净值从最高点到后续最低点的最大跌幅。这个指标不衡量策略的盈利能力，而是专注于衡量投资者可能经历的最大“痛苦”或资金缩水程度。

## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何量化策略的风险。最大回撤描述的是投资过程中可能遭遇的最惨烈下跌区间。

它不能仅从字面理解。最大回撤描述的是一件事：在投资过程中，资产价值持续下跌最严重的那段时期。例如，股价从某个高点开始，经历一段下跌，直到某个低点，这段区间就是一次回撤。在所有这样的回撤中，跌幅最大的那个区间，就是最大回撤区间。

![](img/7a697bd48e9a88991beb33886e288f5f_1.png)

## 最大回撤的意义

最大回撤是一个风险指标。它评估的是，即使一个策略整体盈利，但在过程中可能让你非常“惨”的那段时间。这段最惨的时间就是最大回撤区间。

风险控制至关重要。例如，2008年股市从高点暴跌，许多投资者承受了巨大损失。最大回撤区间就是用来量化这种“惨劲儿”有多大的。

![](img/7a697bd48e9a88991beb33886e288f5f_3.png)

在策略回测报告中，通常会统计并标出最大回撤区间，帮助投资者了解策略在最坏情况下的表现。

## 如何计算最大回撤？

![](img/7a697bd48e9a88991beb33886e288f5f_5.png)

理解了概念后，我们来看看如何通过计算来找到这个区间。计算最大回撤的核心是遍历所有可能的时间点对，找到跌幅最大的那个。

其数学公式可以表示为：

![](img/7a697bd48e9a88991beb33886e288f5f_7.png)

**最大回撤 = max( (P_i - P_j) / P_i )**， 其中 **i < j**

*   **P_i** 代表第 i 天的价格（或资产净值）。
*   **P_j** 代表第 i 天之后某一天 j 的价格。
*   我们需要遍历所有满足 **i < j** 的日期组合，计算跌幅，并找出最大值。

## 使用Python计算最大回撤

下面，我们通过Python代码来演示计算过程。我们将使用Pandas库来高效处理时间序列数据。

![](img/7a697bd48e9a88991beb33886e288f5f_15.png)

首先，我们需要一个关键的Pandas函数：`cummax()`。它与我们之前可能用过的`cumsum()`（累积和）类似，但计算的是累积最大值。

以下是`cummax()`函数的一个简单示例：

![](img/7a697bd48e9a88991beb33886e288f5f_17.png)

![](img/7a697bd48e9a88991beb33886e288f5f_19.png)

```python
import pandas as pd
import numpy as np

![](img/7a697bd48e9a88991beb33886e288f5f_21.png)

# 创建一个示例序列
s = pd.Series([2, np.nan, 5, 3, 8, 1])
print("原始序列:")
print(s)

![](img/7a697bd48e9a88991beb33886e288f5f_23.png)

print("\n累积最大值序列:")
print(s.cummax())
```
执行上述代码，`s.cummax()`会输出：对于第一个元素，最大值是2；对于前两个元素（忽略NaN），最大值是2；对于前三个元素，最大值是5，依此类推。它始终记录到当前位置为止所遇到的最大值。

![](img/7a697bd48e9a88991beb33886e288f5f_25.png)

![](img/7a697bd48e9a88991beb33886e288f5f_27.png)

![](img/7a697bd48e9a88991beb33886e288f5f_15.png)

现在，我们将这个函数应用到实际的股价数据上。假设我们有一个名为`df`的DataFrame，其中`close`列是每日收盘价。

![](img/7a697bd48e9a88991beb33886e288f5f_29.png)

计算步骤如下：
1.  计算到每一天为止的历史最高价（累积最大值）。
2.  用这个历史最高价减去当天的实际价格，得到每日的“潜在下跌幅度”。
3.  将这个下跌幅度除以对应的历史最高价（即公式中的P_i），得到每日的回撤率。
4.  在所有每日回撤率中，最大值就是整个时间段的最大回撤。

![](img/7a697bd48e9a88991beb33886e288f5f_31.png)

对应的Python代码如下：

![](img/7a697bd48e9a88991beb33886e288f5f_33.png)

```python
# 假设 df[‘close’] 是我们的价格序列
# 1. 计算累积最大值序列
cumulative_max = df[‘close’].cummax()

![](img/7a697bd48e9a88991beb33886e288f5f_35.png)

# 2. 计算每日回撤：(历史最高 - 当前价) / 历史最高
daily_drawdown = (cumulative_max - df[‘close’]) / cumulative_max

![](img/7a697bd48e9a88991beb33886e288f5f_37.png)

# 3. 找出所有每日回撤中的最大值，即为最大回撤
max_drawdown = daily_drawdown.max()
print(f“最大回撤为：{max_drawdown:.2%}”)
```

![](img/7a697bd48e9a88991beb33886e288f5f_39.png)

![](img/7a697bd48e9a88991beb33886e288f5f_39.png)

这段代码计算出的`max_drawdown`，例如0.037（即3.7%），就代表了在这段数据期间，投资者可能经历的最大资金缩水比例。

![](img/7a697bd48e9a88991beb33886e288f5f_41.png)
