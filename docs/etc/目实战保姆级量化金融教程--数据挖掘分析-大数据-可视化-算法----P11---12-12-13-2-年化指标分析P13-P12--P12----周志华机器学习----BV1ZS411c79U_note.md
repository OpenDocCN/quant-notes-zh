## 理解年化收益率

![](img/7a40ccb8efa399ea006270cd5b1809b2_8.png)

回测收益率计算的是策略在整个回测期间的总收益。例如，一个策略运行了3年，总收益率为50%。然而，这个数字难以直接与其他运行时间不同的策略进行比较。

![](img/7a40ccb8efa399ea006270cd5b1809b2_10.png)

**年化收益率**的作用正在于此。它将不同时间长度的总收益，标准化为“假设投资期为一年”所能获得的收益率。这使得不同策略之间的横向对比成为可能。通常，在评估策略时，年化收益率是比总收益率更受关注的指标。

![](img/7a40ccb8efa399ea006270cd5b1809b2_12.png)

## 年化收益率的计算公式

年化收益率的计算基于复利思想。其**公式**如下：

**年化收益率 = (1 + 总收益率) ^ (年交易日数 / 回测总交易日数) - 1**

![](img/7a40ccb8efa399ea006270cd5b1809b2_14.png)

![](img/7a40ccb8efa399ea006270cd5b1809b2_16.png)

其中：
*   **总收益率 (R)**：即我们上一节计算出的回测收益率。
*   **年交易日数 (M)**：通常取250或252天，代表一年平均的交易日数量。
*   **回测总交易日数 (N)**：你的回测数据所覆盖的总交易日数量。

![](img/7a40ccb8efa399ea006270cd5b1809b2_18.png)

## 使用Python计算年化收益率

![](img/7a40ccb8efa399ea006270cd5b1809b2_20.png)

接下来，我们将使用Python代码，基于上一节计算出的回测收益率，来计算每只股票策略的年化收益率。

![](img/7a40ccb8efa399ea006270cd5b1809b2_22.png)

![](img/7a40ccb8efa399ea006270cd5b1809b2_24.png)

以下是计算年化收益率的关键步骤代码：

```python
# 假设 total_return 是包含各股票回测收益率的Series或DataFrame列
# 假设 N 是回测数据的总交易日数
M = 250  # 设定一年有250个交易日
N = len(data)  # 获取回测数据的总行数，即总交易日数

![](img/7a40ccb8efa399ea006270cd5b1809b2_26.png)

![](img/7a40ccb8efa399ea006270cd5b1809b2_28.png)

# 应用年化收益率公式进行计算
annualized_return = (1 + total_return) ** (M / N) - 1
```

![](img/7a40ccb8efa399ea006270cd5b1809b2_30.png)

执行上述代码后，我们将得到一个新的`annualized_return`序列，它包含了每只股票策略对应的年化收益率。

![](img/7a40ccb8efa399ea006270cd5b1809b2_32.png)

通过对比可以发现，年化收益率数值通常比总收益率小，因为它将总收益“摊薄”到每一年。这个指标更能反映策略持续、稳定的盈利能力，是策略评价的核心。
