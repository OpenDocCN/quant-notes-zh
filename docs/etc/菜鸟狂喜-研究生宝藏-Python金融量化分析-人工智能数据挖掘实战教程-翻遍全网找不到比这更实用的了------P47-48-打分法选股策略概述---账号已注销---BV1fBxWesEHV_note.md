# Python金融量化分析：P47：打分法选股策略概述

在本节课中，我们将学习一种在量化选股中常用的策略——因子打分选股法。我们将了解其核心思想、实施步骤以及背后的逻辑。

## 策略核心思想

上一节我们介绍了如何通过单一因子评估股票。本节中我们来看看如何综合多个因子进行更全面的选股。

因子打分选股的核心思想是：**为每只股票的多个因子分别打分，然后计算总分，最后根据总分排名选出最优的股票组合**。

这个过程类似于学生考试：
*   每个因子（如市盈率、市净率）相当于一门科目（如语文、数学）。
*   每只股票在每个因子上的表现会获得一个分数。
*   将所有因子的分数相加，得到该股票的“总分”。
*   对所有股票的总分进行排名，选取排名靠前的股票构成投资组合。

## 策略实施前提

在开始实施打分策略前，我们需要明确一个关键的先验知识：**每个因子与预期收益率的相关性方向**。

以下是需要预先了解的信息：
*   **正相关因子**：因子数值**越大**，通常预期收益率**越高**。例如：每股收益、净资产收益率。
*   **负相关因子**：因子数值**越小**，通常预期收益率**越高**。例如：市盈率、市净率。

![](img/74841d531a567cc3e1bde63e66febc16_1.png)

这个方向性判断至关重要，因为它决定了我们打分的标准。我们可以通过查阅券商研究报告或自行进行因子分析（如计算IC值）来获得这些知识。

## 策略流程概述

接下来，我们梳理一下该策略的具体操作流程。

假设我们有300只股票（股票1， 股票2， …， 股票300），并选取了A、B、C、D四个因子。策略流程如下：

1.  **数据准备**：获取所有股票在四个因子上的数值。
2.  **因子打分**：对每个因子，根据其与收益率的正/负相关性，为每只股票在该因子上的表现打分。例如，对于“越大越好”的因子，数值最大的股票得最高分。
3.  **计算总分**：对每只股票，将其在所有因子上的得分相加，得到该股票的综合总分。
    *   公式表示为：`股票总分 = 因子A得分 + 因子B得分 + 因子C得分 + 因子D得分`
4.  **排名与筛选**：根据总分对所有股票进行降序排名，选取排名前N（如10或20）的股票。
5.  **定期调仓**：在设定的调仓周期（如每月）重复上述步骤，更新投资组合。
