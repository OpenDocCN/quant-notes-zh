## 获取交易日列表

![](img/423dc2ec53a3cbe74eef9df2e5f05cfd_3.png)

与回测框架不同，当前的分析模块没有内置的每日数据获取机制。因此，我们需要手动获取指定时间段内的所有交易日数据。

以下是获取交易日列表的步骤：

![](img/423dc2ec53a3cbe74eef9df2e5f05cfd_5.png)

1.  定义一个函数，用于获取指定起始日期和结束日期之间的所有交易日。
2.  调用该函数，生成我们需要的日期序列。

```python
# 第一步：获取交易日列表
def get_trading_dates(start_date, end_date):
    """
    获取指定时间段内的所有交易日。
    参数:
        start_date (str): 起始日期，格式为'YYYY-MM-DD'
        end_date (str): 结束日期，格式为'YYYY-MM-DD'
    返回:
        pandas.DatetimeIndex: 交易日列表
    """
    # 这里使用平台提供的API获取交易日历，示例中使用伪代码
    trading_dates = get_trading_calendar(start_date, end_date)
    return trading_dates

# 指定分析的时间段：2019年1月1日至2020年1月1日
start_date = '2019-01-01'
end_date = '2020-01-01'
all_trading_dates = get_trading_dates(start_date, end_date)
```

通过以上代码，我们成功获取了`2019-01-01`到`2020-01-01`之间的所有交易日列表，为后续查询数据做好了准备。

## 理解与检查数据

![](img/423dc2ec53a3cbe74eef9df2e5f05cfd_29.png)

数据查询完成后，我们需要检查数据的结构和内容，确保其符合分析要求。

![](img/423dc2ec53a3cbe74eef9df2e5f05cfd_31.png)

以下是检查数据的步骤：

1.  查看数据的维度（shape）。
2.  查看前几行数据，了解其格式。

![](img/423dc2ec53a3cbe74eef9df2e5f05cfd_33.png)

```python
# 检查数据的形状
print(f"查询数据的形状: {factor_data.shape}")

![](img/423dc2ec53a3cbe74eef9df2e5f05cfd_35.png)

# 查看前5条数据
print(factor_data.iloc[:5])
```
**输出示例**:
```
查询数据的形状: (1, 1, 3500)
                    pe_ratio
symbol
000001.XSHE         12.34
000002.XSHE         8.90
000004.XSHE         25.67
...                 ...
```
**公式解释**:
数据形状`(1, 1, 3500)`表示这是一个三维数据结构。通常，前两个维度（这里是1和1）可能与数据表或面板数据的结构有关，而最后一个维度`3500`代表了我们查询到的股票数量。对于分析，我们主要关注包含股票代码和因子值的二维部分。

检查结果显示，我们成功获取了约3500只股票在指定日期的市盈率数据。数据中包含了股票代码和对应的因子值。
