## 理解回测收益率

![](img/b0be81dc9f96a7945ae057258607cb98_11.png)

在众多评估指标中，**回测收益率**是最基本的一个。它衡量的是，在整个回测周期结束后，你的策略带来的总收益相对于初始本金的增长比例。

![](img/b0be81dc9f96a7945ae057258607cb98_13.png)

例如，你从2015年到2020年对某只股票应用了一个买卖策略，最终你的资产总值相比初始本金增长了4.23%。这个4.23%就是你的回测收益率。

![](img/b0be81dc9f96a7945ae057258607cb98_15.png)

![](img/b0be81dc9f96a7945ae057258607cb98_17.png)

它的计算公式非常直观：

**回测收益率 = (期末资产总值 / 期初资产总值) - 1**

用符号表示即为：
**Return = (P_T / P_0) - 1**

其中：
*   **P_T** 代表回测结束时的资产总值（或最终卖出价格）。
*   **P_0** 代表回测开始时的初始本金（或初始买入价格）。

这个指标直接反映了策略的绝对盈利能力，是我们评估策略表现的首要参考。

![](img/b0be81dc9f96a7945ae057258607cb98_19.png)

## 计算与可视化回测收益率

![](img/b0be81dc9f96a7945ae057258607cb98_21.png)

上一节我们介绍了回测收益率的定义，本节中我们来看看如何用Python代码进行计算，并观察其随时间的变化趋势。

计算思路是：先计算每个时间点的资产相对于期初的比值，然后根据最后一个比值计算总收益率。

以下是计算步骤：

```python
# 1. 计算每个时间点的资产与期初资产的比值
# 假设我们使用‘close’列作为资产价格序列
price_series = data['close']
cumulative_return = price_series / price_series.iloc[0]

# 2. 可视化资产净值变化曲线
plt.figure(figsize=(10, 6))
cumulative_return.plot()
plt.title('策略累计收益曲线 (相对于期初)')
plt.xlabel('日期')
plt.ylabel('累计收益倍数')
plt.grid(True)
plt.show()
```

执行代码后，你将看到一条曲线。该曲线展示了从期初开始，你的1元本金是如何随时间变化的。如果曲线向上，代表盈利；向下则代表亏损。

![](img/b0be81dc9f96a7945ae057258607cb98_27.png)

![](img/b0be81dc9f96a7945ae057258607cb98_23.png)

接下来，我们计算最终的回测收益率：

![](img/b0be81dc9f96a7945ae057258607cb98_25.png)

```python
# 3. 计算最终的回测收益率
# 取最后一个时间点的比值，然后减去1
final_return = cumulative_return.iloc[-1] - 1

![](img/b0be81dc9f96a7945ae057258607cb98_27.png)

# 4. 将结果以表格形式美观地展示
result_df = pd.DataFrame({
    '回测收益率': [final_return]
}, index=['结果'])

![](img/b0be81dc9f96a7945ae057258607cb98_29.png)

![](img/b0be81dc9f96a7945ae057258607cb98_31.png)

print(result_df)
```

![](img/b0be81dc9f96a7945ae057258607cb98_33.png)

代码执行后，将输出一个包含回测收益率数值的表格。这个数值就是你的策略在整个回测期间的总回报率。

![](img/b0be81dc9f96a7945ae057258607cb98_35.png)

![](img/b0be81dc9f96a7945ae057258607cb98_39.png)

**请注意**：本例中，我们直接使用了股票的原始收盘价序列进行计算，这相当于假设了一个“买入并持有”的简单策略。在实际回测中，`price_series` 应替换为你根据策略信号实际计算出的**投资组合每日净值**序列。
