## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何量化投资过程中的“惨劲儿”，即最大回撤。

最大回撤描述的是在投资过程中，资产价值从某一高点下跌到随后低点的最大幅度。它不能仅从字面理解，其核心是衡量投资者可能经历的最大痛苦区间，即资产缩水最严重的连续时段。

**公式定义**：
对于一个价格序列，最大回撤（Maximum Drawdown, MDD）的计算公式可以表示为：
`MDD = max_{i, j (i<j)} [ (P_i - P_j) / P_i ]`
其中，`P_i` 代表时间点 `i` 的资产价值（峰值），`P_j` 代表 `i` 之后时间点 `j` 的资产价值（谷底）。该公式计算的是所有可能峰值到谷底组合中，相对跌幅最大的一个。

这个指标不直接评估策略的盈利能力，而是评估其风险。例如，即使一个策略长期盈利，但过程中可能经历一段剧烈的、持续的价值下跌，这段时期就是最大回撤区间。了解这个区间有助于投资者管理心理预期和资金风险。

## 如何计算最大回撤？

![](img/fb53ad84f8f0ede1548d4f1767835a89_5.png)

理解了最大回撤的概念后，我们来看看如何通过Python代码进行计算。核心思路是遍历价格序列，找出所有可能的峰值和谷底组合，并计算其中最大的相对跌幅。

![](img/fb53ad84f8f0ede1548d4f1767835a89_7.png)

以下是计算最大回撤的关键步骤：

1.  **计算累积最大值序列**：首先，我们需要找出到每一个时间点为止的历史最高价（峰值）。这可以通过Pandas的 `cummax()` 函数轻松实现。
2.  **计算回撤值序列**：然后，用这个历史最高价序列减去当前的实际价格序列，得到每一个时间点的“潜在下跌值”。
3.  **计算相对回撤序列**：将上一步得到的“潜在下跌值”除以对应的历史最高价，得到每一个时间点的相对回撤比例。
4.  **找出最大值**：最后，从这个相对回撤比例序列中找出最大值，即为最大回撤。

![](img/fb53ad84f8f0ede1548d4f1767835a89_9.png)

```python
import pandas as pd

![](img/fb53ad84f8f0ede1548d4f1767835a89_17.png)

# 假设‘close’是包含每日收盘价的Pandas Series
close_prices = df['close']

![](img/fb53ad84f8f0ede1548d4f1767835a89_19.png)

![](img/fb53ad84f8f0ede1548d4f1767835a89_21.png)

# 1. 计算累积最大值序列（到当前日期为止的历史最高价）
historical_max = close_prices.cummax()

![](img/fb53ad84f8f0ede1548d4f1767835a89_23.png)

![](img/fb53ad84f8f0ede1548d4f1767835a89_25.png)

![](img/fb53ad84f8f0ede1548d4f1767835a89_27.png)

# 2. 计算每日的回撤值（历史最高 - 当前价）
drawdown_value = historical_max - close_prices

![](img/fb53ad84f8f0ede1548d4f1767835a89_29.png)

# 3. 计算每日的相对回撤比例
drawdown_ratio = drawdown_value / historical_max

![](img/fb53ad84f8f0ede1548d4f1767835a89_31.png)

# 4. 找出最大回撤
max_drawdown = drawdown_ratio.max()
print(f"最大回撤为：{max_drawdown:.2%}")
```
