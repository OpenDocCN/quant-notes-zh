# Python金融量化投资分析与股票交易：P4：03 金融量化分析-股票市场构成 🏛️

在本节课中，我们将要学习股票市场的构成。了解市场中有哪些参与者以及他们各自扮演的角色，是理解股票交易和进行量化分析的基础。

上一节我们介绍了股票的分类，本节中我们来看看股票市场的构成，即市场中有哪些参与者。

## 市场主要参与者

以下是股票市场中的主要参与者：

*   **公司与投资者**：公司是需要融资的一方，投资者是提供资金的一方。公司通过发行股票向投资者融资。
*   **证监会**：这是证券行业的监管机构。公司若想上市，必须向证监会提交各种材料。证监会负责审查公司是否存在违法行为（如欺诈、洗钱等），并有权决定公司能否上市或是否应退市。
*   **证券业协会**：这是一个行业自律组织，作用相对较弱。例如，证券从业资格考试由其主办。
*   **交易所**：交易所提供股票集中交易的场所。在中国，主要有上海证券交易所和深圳证券交易所。早期交易需要到交易所现场进行，现在则通过网络连接。
*   **证券中介机构（券商）**：个人投资者通常不能直接进入交易所买卖股票。他们需要在券商（如中信证券、中金公司等）开户，通过券商开发的软件（如同花顺）提交交易指令。券商利用其在交易所拥有的席位，将投资者的指令传达至交易所执行。

## 中国的交易所与板块

中国有两个主要的证券交易所：上海证券交易所和深圳证券交易所。每个交易所内部又划分为不同的板块。

以下是各交易所的板块构成：

*   **上海证券交易所**：主要包含**主板**。
*   **深圳证券交易所**：包含**主板**、**中小板**和**创业板**。中小板和创业板是国家为鼓励中小型或创业型公司发展而设立的板块，上市门槛相对主板较低。

## 股票指数（大盘）

对于每个板块，我们常用一个“大盘”指数来反映其整体表现。指数是该板块内众多股票价格综合计算的结果，用以衡量市场整体的涨跌趋势。

以下是各主要板块对应的指数：

*   **上海主板**：**沪指**（上证指数）
*   **深圳主板**：**深成指**（深证成份指数）
*   **深圳中小板**：**中小板指**
*   **深圳创业板**：**创业板指**

指数可以理解为一个“平均值”或“趋势图”。例如，如果沪指上涨，通常意味着上海证券交易所主板的大多数股票整体向好。
