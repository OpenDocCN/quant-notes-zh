在本节课中，我们将学习财务分析的另外两个核心原理：**复式记账法**与**权责发生制**。理解这两个原理是看懂财务报表并进行量化分析的基础。

## 复式记账法 📝

上一节我们介绍了会计恒等式，本节中我们来看看如何通过复式记账法来记录交易，并始终保持等式的平衡。

**复式记账法**的定义是：每一笔交易，都至少要影响两个或两个以上的账户。其核心目的是确保会计恒等式 `资产 = 负债 + 所有者权益` 始终保持平衡。

### 理解复式记账法

![](img/8af9757b92bfd1caee185f7c96afdfa1_4.png)

以下是理解复式记账法的关键逻辑：

*   **影响等式同一边**：如果一笔交易只影响资产、负债或所有者权益中的一类，那么必须在同类账户中进行“一增一减”，等式才能平衡。
*   **影响等式两边**：如果一笔交易同时影响等式的两边，那么两边必须“同增”或“同减”，等式才能平衡。

### 应用示例

我们通过两个例子来具体说明。

**示例一：购买资产**
假设一家公司花费2000元现金购买一台机器。
*   **交易分析**：现金（资产）减少2000元，同时机器（资产）增加2000元。
*   **记账影响**：这笔交易只影响等式的左边（资产）。现金减少，机器增加，属于“一增一减”，等式 `资产 = 负债 + 所有者权益` 保持平衡。

![](img/8af9757b92bfd1caee185f7c96afdfa1_6.png)

**示例二：实现利润**
假设公司本月获得收入10000元，发生费用4000元，净利润为6000元。
*   **交易分析**：公司赚取了6000元利润，这导致现金（资产）增加6000元。同时，这6000元利润属于所有者，因此所有者权益下的“留存收益”科目也增加6000元。
*   **记账影响**：这笔交易同时影响等式的左边（资产）和右边（所有者权益）。两边“同增”6000元，等式 `资产 = 负债 + 所有者权益` 保持平衡。

通过复式记账法，我们可以清晰地追踪每一笔经济业务对公司财务状况的影响，并确保财务报表的内在逻辑一致。

## 权责发生制 ⚖️

理解了如何记账后，我们需要明确记账的时点标准。这就是财务分析的第三个核心——权责发生制。

**权责发生制**是指，会计记账不是根据现金的实际收付来确认，而是根据**权利和义务是否已经发生转移**来确认收入和费用。

与之相对的是**收付实现制**，即只有在实际收到或支付现金时才进行确认。在实务中，**资产负债表**和**利润表**采用权责发生制编制，而**现金流量表**采用收付实现制编制，两者结合能更全面地反映公司状况。

### 确认收入的条件

在权责发生制下，确认收入必须同时满足以下两个条件：
1.  货物已交付或服务已提供，相关的权利义务已发生转移。
2.  收回款项的可能性很高。

### 四大权责发生制账户

权责发生制产生了四类重要的过渡性账户，它们连接了现金流与实际业务。

以下是这四大账户的介绍：

1.  **应收账款**
    *   **场景**：货已发，款未收。
    *   **性质**：资产。代表企业拥有的收款权利。
    *   **记账**：确认收入，同时增加“应收账款”。收款时，现金增加，应收账款减少。
    *   **公式/代码表示**：`资产 += 应收账款`

2.  **预收账款 / 合同负债**
    *   **场景**：款已收，货未发（如订阅杂志）。
    *   **性质**：负债。代表企业承担的供货义务。
    *   **记账**：收到现金时，不确认收入，而是增加“预收账款”。发货时，确认收入，同时减少预收账款。
    *   **公式/代码表示**：`负债 += 预收账款`

3.  **预付账款**
    *   **场景**：款已付，服务未享受（如预付宽带费）。
    *   **性质**：资产。代表企业享有的未来接受服务的权利。
    *   **记账**：支付现金时，不确认费用，而是增加“预付账款”。享受服务时，确认费用，同时减少预付账款。
    *   **公式/代码表示**：`资产 += 预付账款`

4.  **应付账款**
    *   **场景**：货已到，款未付。
    *   **性质**：负债。代表企业承担的付款义务。
    *   **记账**：收到货物时，增加存货（资产），同时增加“应付账款”。付款时，现金减少，应付账款减少。
    *   **公式/代码表示**：`负债 += 应付账款`

## 三大报表的勾稽关系 🔗

现在，我们可以将利润表、资产负债表和现金流量表完整地联系起来。

![](img/8af9757b92bfd1caee185f7c96afdfa1_10.png)

1.  **利润表与资产负债表**：通过“留存收益”科目勾稽。利润表中的**净利润**会转入资产负债表的“留存收益”中，增加所有者权益。
    *   `期末留存收益 = 期初留存收益 + 本期净利润 - 本期股利`

![](img/8af9757b92bfd1caee185f7c96afdfa1_12.png)

![](img/8af9757b92bfd1caee185f7c96afdfa1_14.png)

![](img/8af9757b92bfd1caee185f7c96afdfa1_16.png)

2.  **现金流量表与资产负债表**：通过“现金”科目勾稽。现金流量表展示了期间内现金的变动详情，其**净现金流量**等于资产负债表上“现金”科目的期末与期初差额。
    *   `期末现金 = 期初现金 + 经营活动现金流 + 投资活动现金流 + 融资活动现金流`

因此，**利润表**是对**留存收益**变动的详细说明，而**现金流量表**是对**现金**变动的详细说明。
