上一节我们介绍了如何通过双均线交叉来生成交易信号（`position`）。本节中，我们来看看如何利用这个信号来计算策略的实际收益，并与“买入并持有”的原始收益进行对比。

## 计算原始日收益率

首先，我们需要计算股价的原始日收益率。收益率是衡量资产价格变化的核心指标，计算公式为：**当日收盘价 / 前一日收盘价 - 1**。但在量化分析中，我们常使用对数收益率，因为它具有更好的数学性质（如可加性）。

以下是计算对数收益率的代码：

![](img/57724eb1a6c761603df3c8024c0ddaa1_5.png)

```python
# 计算股价的对数收益率
data['returns'] = np.log(data['close'] / data['close'].shift(1))
```

![](img/57724eb1a6c761603df3c8024c0ddaa1_7.png)

在这段代码中：
*   `data[‘close’]` 代表股价序列。
*   `.shift(1)` 函数将股价序列向下移动一行，从而得到前一天的收盘价。
*   `np.log()` 用于计算自然对数，得到对数收益率。

![](img/57724eb1a6c761603df3c8024c0ddaa1_9.png)

执行后，`data` 数据框中会新增一列 `returns`，代表每日的股价增长率。

## 根据策略信号调整收益率

![](img/57724eb1a6c761603df3c8024c0ddaa1_11.png)

接下来，我们需要根据策略信号来调整收益率。我们的 `position` 列信号含义如下：
*   `position = 1`：代表策略判断为“持有”或“做多”，我们遵循原始的市场走势。
*   `position = -1`：代表策略判断为“反转”或“做空”（在本例中，我们将其视为与原始走势相反的操作）。

![](img/57724eb1a6c761603df3c8024c0ddaa1_13.png)

因此，策略的日收益率应为：**原始收益率 × 策略信号**。同样，我们需要对信号进行移位操作，以确保使用前一天的信号来指导今天的交易。

![](img/57724eb1a6c761603df3c8024c0ddaa1_15.png)

以下是计算策略收益率的代码：

```python
# 根据策略信号调整收益率
data['strategy'] = data['position'].shift(1) * data['returns']
```

![](img/57724eb1a6c761603df3c8024c0ddaa1_17.png)

在这段代码中：
*   `data[‘position’].shift(1)` 使用了前一天的交易信号。
*   将信号与原始收益率相乘，得到策略调整后的每日收益率 `strategy`。

![](img/57724eb1a6c761603df3c8024c0ddaa1_19.png)

## 计算并比较累计收益

![](img/57724eb1a6c761603df3c8024c0ddaa1_21.png)

有了每日的收益率序列后，我们需要计算从期初到期末的总收益。这个过程分为两步：
1.  **累加收益率**：将对数收益率逐日相加。
2.  **还原为资金倍数**：通过对累加和取指数，将对数收益率还原为最终的资金增长倍数。公式为：**最终资金 = 初始资金 × exp(累计对数收益率)**。

![](img/57724eb1a6c761603df3c8024c0ddaa1_23.png)

我们将同时计算原始走势和策略的累计收益。

![](img/57724eb1a6c761603df3c8024c0ddaa1_25.png)

以下是计算并比较累计收益的代码：

![](img/57724eb1a6c761603df3c8024c0ddaa1_27.png)

```python
# 计算累计对数收益率
cumulative_log_returns = data[['returns', 'strategy']].sum()

![](img/57724eb1a6c761603df3c8024c0ddaa1_29.png)

# 将缺失值填充为0（通常由.shift()操作产生）
cumulative_log_returns = cumulative_log_returns.fillna(0)

# 将对数收益率还原为资金增长倍数
final_value = np.exp(cumulative_log_returns)

print(final_value)
```

![](img/57724eb1a6c761603df3c8024c0ddaa1_31.png)

在这段代码中：
*   `.sum()` 对 `returns` 和 `strategy` 两列分别求和，得到累计对数收益率。
*   `.fillna(0)` 处理因数据移位产生的缺失值。
*   `np.exp()` 是 `np.log()` 的逆运算，将累加的对数收益率还原为最终的资金倍数。

![](img/57724eb1a6c761603df3c8024c0ddaa1_33.png)

假设初始资金为1元，打印出的 `final_value` 结果可能类似于：
*   `returns`: 4.2
*   `strategy`: 5.8

![](img/57724eb1a6c761603df3c8024c0ddaa1_35.png)

![](img/57724eb1a6c761603df3c8024c0ddaa1_37.png)

这表示：
*   **不做任何操作（买入并持有）**：1元本金最终变为约4.2元。
*   **使用双均线策略**：1元本金最终变为约5.8元。

通过对比可以看出，在该段历史数据上，我们的双均线策略获得了超越市场原始走势的收益。

本节课中我们一起学习了量化策略收益分析的全过程。我们从计算基础的日收益率开始，接着根据交易信号调整得到策略收益率，最后通过累加和指数还原计算出策略的最终收益，并与基准收益进行对比。这个过程是评估任何一个量化策略是否有效的核心步骤。虽然本例中的策略在历史数据上表现优异，但请记住，实际市场交易中需要考虑手续费、滑点等因素，并且过去的表现并不能完全代表未来。
