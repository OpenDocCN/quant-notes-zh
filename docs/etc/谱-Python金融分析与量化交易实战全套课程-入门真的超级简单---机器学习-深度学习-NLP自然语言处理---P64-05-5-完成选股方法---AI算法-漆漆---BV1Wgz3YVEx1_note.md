## 概述
在本节课中，我们将学习如何完成一个基于多因子打分的选股方法。我们将把多个因子的评分数据表进行拼接，计算每只股票的综合得分，并根据得分进行排序，最终筛选出排名前十的股票。

## 数据表拼接操作

上一节我们介绍了如何为单个因子数据打分。本节中，我们来看看如何将多个因子的打分结果合并起来。

拼接操作使用 `pd.concat` 函数，将一个 `DataFrame` 与另一个 `DataFrame` 进行合并。

以下是具体操作步骤：

```python
# 假设 df1 和 df2 是两个已经打好分的因子数据表
df_total = pd.concat([df1, df2], axis=1)
```

现在，两个 `DataFrame` 已经拼接成一个名为 `df_total` 的大表。但此表中还没有综合排名的信息。

![](img/f301f03634ace5d19f7183bad50391f7_4.png)

---

## 计算综合得分并排序

在得到合并后的大表后，我们需要计算每只股票的综合得分并进行排序。

首先，我们新建一个名为“得分值”的列，用于存放最终的综合得分。

```python
# 初始化一个全为零的列，行数与样本数据数量一致（例如300行）
df_total[‘得分值’] = np.zeros((300, 1))
```

接下来，我们需要计算总分。假设我们有六个指标，总分就是将这六个指标的分数横向相加。

以下是计算总分并排序的步骤：

```python
# 计算总分：对六个指标列进行横向求和
df_total[‘得分值’] = df_total[[‘指标1’, ‘指标2’, ‘指标3’, ‘指标4’, ‘指标5’, ‘指标6’]].sum(axis=1)

# 根据“得分值”列进行降序排序，以找到分数最高的股票
df_sorted = df_total.sort_values(by=‘得分值’, ascending=False)
```

排序完成后，我们只需要“得分值”这一列以及对应的股票代码（索引）。

---

## 获取最终选股结果

![](img/f301f03634ace5d19f7183bad50391f7_6.png)

排序之后，我们的目标是获取排名前十的股票列表。

![](img/f301f03634ace5d19f7183bad50391f7_8.png)

我们从排序后的 `DataFrame` 中提取股票代码索引，并转换为列表，然后取前十个。

![](img/f301f03634ace5d19f7183bad50391f7_10.png)

```python
# 获取排序后DataFrame的索引（即股票代码），并转换为列表
stock_list = df_sorted.index.tolist()

![](img/f301f03634ace5d19f7183bad50391f7_12.png)

# 选取排名前十的股票
top_10_stocks = stock_list[:10]
```

至此，`get_stocks` 函数的核心逻辑就完成了。执行一次该函数，它就会根据当前所有因子的数据，返回当天综合排名前十的股票列表。
