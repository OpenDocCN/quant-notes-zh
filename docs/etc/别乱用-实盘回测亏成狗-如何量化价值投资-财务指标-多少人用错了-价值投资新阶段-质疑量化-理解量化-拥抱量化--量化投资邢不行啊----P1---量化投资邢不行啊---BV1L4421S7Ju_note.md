## 市盈率指标回测

![](img/ed1317dbaec3c04c6eb0b57743f80624_27.png)

首先，我们来测试最重要的财务指标之一：市盈率。

### 市盈率简介

![](img/ed1317dbaec3c04c6eb0b57743f80624_29.png)

市盈率的计算公式为：
**市盈率 = 市值 / 净利润**

这个指标非常简洁明了，默认显示在所有行情软件中。绝大多数价值投资者用它来判断股票是否被高估或低估。传统观点认为，市盈率越低的股票估值越低、越便宜，未来上涨的概率也越大。

### 构建低市盈率选股策略

![](img/ed1317dbaec3c04c6eb0b57743f80624_31.png)

我们可以尝试构建一个买入低市盈率股票的量化选股策略，检验其有效性。

![](img/ed1317dbaec3c04c6eb0b57743f80624_33.png)

策略构建步骤如下：

![](img/ed1317dbaec3c04c6eb0b57743f80624_35.png)

1.  **计算指标**：在每个月的最后一个交易日收盘后，计算每只股票的市盈率倒数（使用倒数是为了避免净利润为零导致无法计算市盈率），并从大到小排序。
2.  **筛选股票**：删除当时ST、退市或停牌的股票，同时删除上市不满一年的股票。
3.  **选择股票**：挑选出此时市盈率倒数排名最大的十只股票。
4.  **执行交易**：在下个月的第一个交易日开盘时，等权重买入这十只股票。
5.  **持有与换仓**：持有该组合整整一个月，直到下个月的最后一个交易日收盘时全部卖出。然后，重新选择下个月市盈率最低的十只股票，并在下下个月的第一个交易日买入。

![](img/ed1317dbaec3c04c6eb0b57743f80624_37.png)

如此循环往复。

![](img/ed1317dbaec3c04c6eb0b57743f80624_39.png)

### 回测结果分析

运行准备好的Python代码和财务数据后，可以得到策略的回测结果。代表策略的橙色曲线从最初的1元变为约1.9元，年化收益不足5%，仅略微跑赢代表大盘的沪深300指数。并且，策略在最近几年表现明显不佳。

![](img/ed1317dbaec3c04c6eb0b57743f80624_41.png)

由此可见，仅通过低市盈率原则炒股，收益与存款利息差不多，但风险却大得多，非常不划算。

![](img/ed1317dbaec3c04c6eb0b57743f80624_43.png)

### 深入探究市盈率

![](img/ed1317dbaec3c04c6eb0b57743f80624_45.png)

为了更深入研究，我们还测试了以下两种策略：

![](img/ed1317dbaec3c04c6eb0b57743f80624_47.png)

1.  **分行业策略**：买入每个行业中市盈率最低的一只股票。结果显示，该策略表现同样不佳。
2.  **分组对比策略**：按市盈率倒数将股票分为十组，分别构建策略。结果显示，高市盈率组合的表现确实更差，但市盈率最低的组合表现也不是最好的。

![](img/ed1317dbaec3c04c6eb0b57743f80624_49.png)

这表明，一味追求低估值并不可取，只要股票的估值在合理区间内即可。这与市场上流传的一些经验有所不同。

![](img/ed1317dbaec3c04c6eb0b57743f80624_51.png)

## 净利润增长率指标回测

![](img/ed1317dbaec3c04c6eb0b57743f80624_59.png)

### 净利润增长率简介

![](img/ed1317dbaec3c04c6eb0b57743f80624_61.png)

净利润可以简单理解为公司扣除成本、交完税后赚到的所有钱。我们所说的增长率是同比数据，即用今年净利润与去年同期净利润进行对比。

![](img/ed1317dbaec3c04c6eb0b57743f80624_63.png)
