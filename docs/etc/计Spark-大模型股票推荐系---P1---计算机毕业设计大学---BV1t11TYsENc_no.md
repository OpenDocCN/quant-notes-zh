# 计算机毕业设计：P1：项目概述与核心技术栈

在本节课中，我们将要学习一个结合了大数据处理与人工智能技术的股票分析系统。该系统旨在通过数据爬取、分析、预测和可视化，为量化交易提供决策支持。我们将了解项目的整体架构和所涉及的核心技术。

## 项目概述

本项目是一个综合性的股票分析系统。它从多个来源获取股票数据，利用大数据框架进行高效处理，并应用大语言模型进行趋势预测。最终，系统通过直观的可视化界面展示分析结果，辅助用户进行投资决策。

## 核心技术栈

上一节我们介绍了项目的整体目标，本节中我们来看看实现这一目标所依赖的关键技术。以下是构成该系统的核心组件：

![](img/c5e3455933ea1a786040b479b47c8d9d_1.png)

*   **数据处理引擎**：**Apache Spark**。这是一个用于大规模数据处理的统一分析引擎，能够快速处理海量股票数据。
*   **智能预测核心**：**大语言模型**。利用先进的AI模型分析市场情绪、新闻文本和历史数据，生成股票走势预测。
*   **数据获取层**：**股票爬虫**。编写自动化程序从财经网站或API接口实时抓取股票行情、公司财报、新闻资讯等数据。
*   **可视化呈现**：**股票K线图与数据分析图表**。使用前端图表库（如ECharts）将复杂的股票数据以K线图、趋势线、成交量图等形式直观展示。
*   **应用框架**：可能包含**Spring Boot**（后端服务）、**Vue.js/React**（前端界面）等，用于构建完整的Web应用系统。

## 核心概念与公式

为了深入理解系统原理，我们需要掌握几个基础概念。

![](img/c5e3455933ea1a786040b479b47c8d9d_3.png)

**1. 简单移动平均线**
SMA用于平滑股价数据，识别趋势。其计算公式为：
`SMA = (P1 + P2 + ... + Pn) / n`
其中，`P_n` 是第n天的收盘价，`n` 是时间周期（例如20天）。

**2. 数据批处理代码示例**
以下是使用Spark读取股票交易数据的简化代码片段：
```scala
val spark = SparkSession.builder().appName("StockAnalysis").getOrCreate()
val stockDF = spark.read.option("header", "true").csv("hdfs://path/to/stock_data.csv")
stockDF.show()
```

## 系统工作流程

了解了核心技术后，我们来看一下数据在系统中是如何流动和处理的。

![](img/c5e3455933ea1a786040b479b47c8d9d_5.png)

1.  **数据采集**：爬虫程序定期运行，获取最新的股票市场数据。
2.  **数据存储与处理**：原始数据被存入分布式文件系统（如HDFS），由Spark进行数据清洗、转换和特征工程。
3.  **模型预测**：处理后的数据送入大语言模型或机器学习模型，进行涨跌预测和推荐评分。
4.  **结果可视化**：预测结果与分析指标通过Web界面渲染成交互式图表，如K线图。
5.  **策略回测**：系统可能包含量化交易模块，基于历史数据对交易策略进行回测验证。
