## Anaconda介绍与安装

首先，我们来看第一个核心组件：Anaconda。Anaconda是一个**集成环境**。这意味着它已经为我们预先打包好了进行数据分析和机器学习开发所需的所有工具和库。

**核心概念**：集成环境 = 预先配置好的软件包集合。

因此，要进行数据分析和机器学习开发，第一步就是在你的电脑上安装Anaconda。安装好Anaconda后，就意味着我们可以使用这个集成的环境进行开发工作。

以下是安装Anaconda的步骤：
1.  访问Anaconda官网下载安装包。
2.  根据你的操作系统（Windows、macOS或Linux）选择对应的版本。
3.  下载完成后，双击安装包进行安装。

安装过程中的注意事项：
*   **安装路径**：安装目录不能包含中文或特殊符号。建议安装在某个磁盘的根目录下（例如 `C:\` 或 `D:\`）。

## Jupyter介绍与使用

![](img/92fbb46ccf105297a3bd2c0a4accb340_1.png)

接下来是Jupyter。Jupyter是Anaconda提供的一个**基于浏览器的可视化开发工具**。我们将在Jupyter中编写和执行数据分析相关的代码。

![](img/92fbb46ccf105297a3bd2c0a4accb340_3.png)

**核心概念**：Jupyter = 基于网页的交互式代码笔记本。

Jupyter不需要单独安装。只要成功安装了Anaconda，Jupyter就已经包含在其中，可以直接使用。

### 如何启动Jupyter

启动Jupyter的步骤如下：
1.  打开你的系统终端（或命令提示符/Anaconda Prompt）。
2.  在终端中输入指令 `jupyter notebook` 并按下回车。

![](img/92fbb46ccf105297a3bd2c0a4accb340_5.png)

![](img/92fbb46ccf105297a3bd2c0a4accb340_7.png)

```bash
jupyter notebook
```

执行该指令后，它会自动启动一个本地服务并打开你的默认浏览器，显示Jupyter的主界面。这个界面展示的是你当前终端所在目录的文件结构。

### Jupyter的基本操作

启动Jupyter后，我们需要新建一个文件来编写代码。

以下是新建文件的操作：
1.  在Jupyter主界面，点击右上角的 **“New”** 按钮。
2.  在下拉菜单中选择 **“Python 3”**。这将创建一个新的源代码文件，文件后缀为 `.ipynb`。

创建文件后，你可以重命名它。文件内部默认有一个输入框，我们称之为 **“Cell”**。

**核心概念**：Cell = Jupyter笔记本中的基本编辑单元，可以输入代码或文本。

在Cell中，我们可以编写内容并执行。例如，输入 `print(“Hello World”)`，然后点击工具栏的 **“Run”** 按钮或使用快捷键，即可看到执行结果。

### Cell的两种模式

Cell有两种工作模式，用于不同目的：

1.  **Code模式**：用于编写和运行程序代码。
2.  **Markdown模式**：用于编写格式化的文本笔记（如教程、注释）。

你可以在Cell上方的下拉菜单中切换这两种模式。两种模式下的Cell都需要执行（Run）才能显示最终效果。

### 常用快捷键

熟练使用快捷键可以极大提升在Jupyter中的工作效率。以下是几个最常用的快捷键：

*   **添加Cell**：
    *   `A`：在当前选中的Cell上方插入一个新的Cell。
    *   `B`：在当前选中的Cell下方插入一个新的Cell。
*   **删除Cell**：`X`（删除当前选中的Cell）。
*   **切换Cell模式**：
    *   `M`：将当前Cell切换到Markdown模式。
    *   `Y`：将当前Cell切换到Code模式。
*   **执行Cell**：`Shift + Enter`（执行当前Cell，并跳转到下一个Cell）。
*   **代码自动补全**：`Tab`键。
*   **查看帮助文档**：选中函数名（如 `print`），按下 `Shift + Tab`，可以查看该函数的帮助信息。
