## 深入分析季节性趋势

![](img/d63aeb72f685058557484a0f90e849e5_37.png)

构造完模型后，我们还可以利用Prophet框架自带的函数进行更深入的分析。

Prophet框架内置了`plot_components`函数，它可以帮助我们可视化模型发现的各种规律。

![](img/d63aeb72f685058557484a0f90e849e5_3.png)

该函数会生成多张子图，展示数据中不同时间维度的趋势：
1.  **整体趋势**：展示数据在整个时间范围内的变化情况。
2.  **年度趋势**：展示数据在不同年份间的变化规律。
3.  **月度趋势**：展示数据在一个月内不同日期的变化规律。
4.  **周度趋势**（如果启用）：展示数据在一周内不同日期的变化规律。

## 模型构建代码详解

现在，让我们深入看一下建模的具体代码是如何实现的。

![](img/d63aeb72f685058557484a0f90e849e5_51.png)

核心建模步骤如下：

![](img/d63aeb72f685058557484a0f90e849e5_53.png)

![](img/d63aeb72f685058557484a0f90e849e5_55.png)

1.  **初始化模型**：首先实例化一个Prophet模型对象，并传入相关参数。
    ```python
    model = cf.create_model(...)
    ```
    跳转到`create_model`函数内部，可以看到模型实例化的过程。关键参数包括：
    *   `seasonality_mode`: 是否绘制季节性变化图。
    *   `changepoint_prior_scale`: 这是一个关键参数，默认值为`0.05`，它控制着模型对趋势变化的灵活度。值越大，模型对趋势变化越敏感；值越小，趋势线越平滑。

![](img/d63aeb72f685058557484a0f90e849e5_57.png)

![](img/d63aeb72f685058557484a0f90e849e5_59.png)

2.  **配置季节性**：在实例化时，可以指定不同周期的季节性分析，例如添加一个以30.5天为周期的月度趋势。
    ```python
    model.add_seasonality(name='monthly', period=30.5, fourier_order=5)
    ```

![](img/d63aeb72f685058557484a0f90e849e5_61.png)

![](img/d63aeb72f685058557484a0f90e849e5_63.png)

3.  **训练模型**：将历史数据传入模型进行拟合（训练）。
    ```python
    model.fit(history_data)
    ```
    这里可以通过`training_years`参数指定使用最近多少年的数据进行训练，例如只用近3年的数据。

![](img/d63aeb72f685058557484a0f90e849e5_65.png)

![](img/d63aeb72f685058557484a0f90e849e5_67.png)

4.  **生成预测框架**：使用`make_future_dataframe`方法创建用于预测的时间序列。
    ```python
    future = model.make_future_dataframe(periods=0)
    ```
    当`periods=0`时，表示不预测未来新的时间点，只对历史数据进行“预测”（相当于在历史数据上做拟合，用于检验模型）。当`periods=90`时，则表示预测未来90天。

![](img/d63aeb72f685058557484a0f90e849e5_69.png)

![](img/d63aeb72f685058557484a0f90e849e5_71.png)

5.  **进行预测**：对生成的时间框架进行预测。
    ```python
    forecast = model.predict(future)
    ```
    预测结果`forecast`是一个DataFrame，其中包含几个重要的列：
    *   `yhat`: 预测值。
    *   `yhat_lower`: 预测区间的下限。
    *   `yhat_upper`: 预测区间的上限。

![](img/d63aeb72f685058557484a0f90e849e5_73.png)

![](img/d63aeb72f685058557484a0f90e849e5_75.png)

![](img/d63aeb72f685058557484a0f90e849e5_77.png)

6.  **绘制结果**：最后，将观测值、预测值和置信区间绘制在同一张图上。
    ```python
    fig1 = model.plot(forecast)
    ```
    执行后，便得到了我们最初看到的那张包含黑点（观测值）、绿线（预测值）和浅色区域（置信区间）的图表。

![](img/d63aeb72f685058557484a0f90e849e5_85.png)

*   **整体趋势**：在观察的三年内，股价整体呈上升趋势。

## 识别突变点

![](img/d63aeb72f685058557484a0f90e849e5_99.png)

![](img/d63aeb72f685058557484a0f90e849e5_101.png)

除了趋势和季节性，Prophet还能帮助我们识别时间序列中的**突变点**。

![](img/d63aeb72f685058557484a0f90e849e5_103.png)

![](img/d63aeb72f685058557484a0f90e849e5_105.png)

突变点是指时间序列趋势发生显著变化的点，例如从上升转为下降，或变化速率突然加快/减慢。

![](img/d63aeb72f685058557484a0f90e849e5_107.png)

![](img/d63aeb72f685058557484a0f90e849e5_109.png)

以下是识别和可视化突变点的步骤：

1.  在训练模型后，Prophet会自动计算潜在的突变点，可以通过`model.changepoints`获取。
2.  我们可以计算这些突变点处趋势变化的方向（通过二阶导数的正负来判断是加速增长还是减速增长）。
3.  最后，在趋势图上用不同的颜色（如红色和绿色）标记出正向和负向的突变点。

![](img/d63aeb72f685058557484a0f90e849e5_111.png)

![](img/d63aeb72f685058557484a0f90e849e5_103.png)

![](img/d63aeb72f685058557484a0f90e849e5_113.png)

![](img/d63aeb72f685058557484a0f90e849e5_115.png)

![](img/d63aeb72f685058557484a0f90e849e5_117.png)

上图展示了识别出的突变点，这有助于我们理解趋势在何时发生了关键转变。

## 进行未来预测

![](img/d63aeb72f685058557484a0f90e849e5_119.png)

现在，我们来完成最核心的一步：预测未来。

![](img/d63aeb72f685058557484a0f90e849e5_121.png)

![](img/d63aeb72f685058557484a0f90e849e5_123.png)

预测未来的步骤与历史拟合类似，关键区别在于`make_future_dataframe`函数中的`periods`参数。

```python
# 预测未来180天
future = model.make_future_dataframe(periods=180)
forecast = model.predict(future)
fig = model.plot(forecast)
```

![](img/d63aeb72f685058557484a0f90e849e5_125.png)

![](img/d63aeb72f685058557484a0f90e849e5_127.png)

![](img/d63aeb72f685058557484a0f90e849e5_129.png)

当`periods=180`时，模型会生成一个包含未来180天日期的新数据框架，并基于历史规律对这些未来日期进行预测。

![](img/d63aeb72f685058557484a0f90e849e5_135.png)

![](img/d63aeb72f685058557484a0f90e849e5_131.png)

在结果图中：
*   黑色点（观测值）只到当前日期（例如2018年7月21日）。
*   绿色线（预测值）会向后延伸180天，展示未来的预测趋势。
*   预测部分的置信区间通常会变宽，这反映了对未来不确定性的增加。

![](img/d63aeb72f685058557484a0f90e849e5_133.png)

![](img/d63aeb72f685058557484a0f90e849e5_135.png)

从示例图看，模型预测未来趋势会先有一个小幅下降，然后可能再上升。这种相对“保守”的预测风格，可以通过调整`changepoint_prior_scale`参数来改变。**增大该参数值会使模型更贴合历史数据的波动，预测可能更“奔放”；减小该值则会使趋势线更平滑，预测更“保守”**。
