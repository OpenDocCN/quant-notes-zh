在本节课中，我们将学习量化投资中四个核心的收益与风险指标：年化收益率、最大回撤、波动率和夏普比率。我们将逐一解释它们的含义，并展示如何使用Python代码进行计算，帮助初学者理解如何评估一个投资策略的表现。

## 年化收益率 📈

上一节我们介绍了本课程的目标，本节中我们首先来看第一个指标：年化收益率。年化收益率是将投资的总收益率换算成以一年为标准的收益率，便于比较不同时间长度的投资表现。

总收益率的计算公式如下：
`总收益率 = (期末总资产 - 期初总资产) / 期初总资产 * 100%`

年化收益率的计算相对复杂，其公式为：
`年化收益率 R = ((1 + P) ^ (M / N) - 1) * 100%`
其中：
*   **R** 代表年化收益率。
*   **P** 代表策略的总收益率。
*   **N** 代表策略执行的总天数。
*   **M** 代表每年的交易天数。

关于参数 **M**（年交易天数）的选择：
*   当 **M = 365** 时，适用于以自然日连续计算的投资，如货币基金、外汇。
*   在股票市场中，通常使用 **M = 250**，因为扣除周末和节假日后，每年大约有250个交易日。

以下是使用Python计算沪深300、纳斯达克100和标普500指数近10年年化收益率的代码示例：

```python
import pandas as pd
import numpy as np

# 获取并筛选数据（以沪深300为例）
# data = pd.read_excel(‘your_data.xlsx‘) # 假设数据已加载
# hs300_data = data[data[‘date‘] >= ‘2014-01-01‘] # 筛选近10年数据

# 计算总收益率
total_return = hs300_data[‘close‘].iloc[-1] / hs300_data[‘close‘].iloc[0] - 1

# 计算年化收益率 (假设M=250)
annual_return = (1 + total_return) ** (250 / len(hs300_data)) - 1
print(f“总收益率: {total_return:.2%}“)
print(f“年化收益率: {annual_return:.2%}“)
```

运行代码后，我们可以得到各指数的表现对比。例如，标普500在过去10年展现了强劲的增长，而沪深300表现相对平稳。

## 最大回撤 📉

了解了衡量收益的指标后，我们接下来看一个重要的风险指标：最大回撤。它反映了投资过程中可能出现的最大亏损幅度，是评估策略下行风险的关键。

最大回撤的计算公式如下：
`最大回撤 = max((Px - Py) / Px)`, 其中 `y > x`
*   **Px** 和 **Py** 代表策略某日的净值，且 **Py** 的日期在 **Px** 之后。
*   该算法计算从每一个高点开始，到后续所有低点的回撤幅度，并取其中的最大值。

以下是计算最大回撤的Python代码核心部分：

```python
# 计算每日的历史最高点（累计最大值）
hs300_data[‘cummax‘] = hs300_data[‘close‘].cummax()

# 计算每日的回撤
hs300_data[‘drawdown‘] = (hs300_data[‘cummax‘] - hs300_data[‘close‘]) / hs300_data[‘cummax‘]

# 计算最大回撤
max_drawdown = hs300_data[‘drawdown‘].max()
print(f“最大回撤: {max_drawdown:.2%}“)
```

计算结果与年化收益率表现一致：通常表现越强的指数，其最大回撤可能相对较小（或可承受），反之亦然。

## 波动率与夏普比率 ⚖️

在评估了潜在的最大亏损后，我们进一步考察收益的稳定性。波动率衡量的是收益率围绕其平均值波动的程度，即风险的大小。

年化波动率通常由日收益率的标准差进行年化处理得到，公式为：
`年化波动率 = 日收益率的标准差 * sqrt(年交易天数)`

高波动率策略收益起伏大，低波动率策略则更为平稳。

在了解了波动率之后，我们引入夏普比率。它综合了收益和风险，衡量的是每承担一单位风险所获得的超额收益，计算公式为：
`夏普比率 = (年化收益率 - 无风险利率) / 年化波动率`
*   **无风险利率** 通常参考国债收益率。
*   夏普比率 > 1，表明风险收益性价比高；夏普比率 < 1，则性价比偏低。

以下是计算波动率和夏普比率的Python代码：

```python
# 计算日收益率
hs300_data[‘daily_return‘] = hs300_data[‘close‘].pct_change()

# 计算年化波动率 (M=250)
annual_volatility = hs300_data[‘daily_return‘].std() * np.sqrt(250)

# 设置无风险利率 (例如2%)
risk_free_rate = 0.02

# 计算夏普比率 (假设已计算出年化收益率 annual_return)
sharpe_ratio = (annual_return - risk_free_rate) / annual_volatility

print(f“年化波动率: {annual_volatility:.2%}“)
print(f“夏普比率: {sharpe_ratio:.2f}“)
```

根据计算结果，如果单纯投资宽基指数，其夏普比率可能较低，说明这是一种投资性价比相对较低的行为。关于如何利用夏普比率构建更优的投资组合，可以参考马科维茨投资组合理论。

## 总结

本节课中我们一起学习了量化投资中四个核心指标：
1.  **年化收益率**：标准化后的收益指标，用于跨周期比较。
2.  **最大回撤**：衡量投资过程中可能面临的最大亏损，反映下行风险。
3.  **波动率**：衡量收益的波动程度，即风险大小。
4.  **夏普比率**：综合收益与风险，衡量每单位风险带来的超额回报，是评估策略性价比的关键指标。

通过Python代码，我们可以实际计算这些指标，从而对不同的投资策略或产品进行量化的评估和比较。
