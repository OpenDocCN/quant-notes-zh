首先，我们需要在策略的 `context` 对象中定义一些关键变量。`context` 是贯穿整个策略运行周期的上下文对象，用于存储和传递数据。

    ```python
    # 在 context 中定义股票池
    ```

    ```python
    # 设置按月调仓，并指定调仓函数为 rebalance
    ```

```python
def rebalance(context):
```

## 数据读取与处理

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_51.png)

现在，让我们深入 `get_stocks` 函数，看看如何读取和处理财务数据。

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_53.png)

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_55.png)

该函数的目标是查询我们所需的财务指标，并将其整理成便于计算的数据格式。我们计划使用六个财务指标，分为两类：“越高越好”的指标和“越低越好”的指标。

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_57.png)

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_59.png)

以下是数据读取的具体步骤：

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_61.png)

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_63.png)

1.  **构建查询语句 (Query)**：我们需要指定要查询的财务指标。首先构建“越高越好”指标的查询。
    ```python
    # 构建查询语句，获取“越高越好”的指标
    query_up = query(
        fundamentals.eod_derivative_indicator.eps, # 每股收益 (EPS)，越高越好
        fundamentals.financial_indicator.roe,       # 净资产收益率 (ROE)，越高越好
        fundamentals.financial_indicator.roa        # 总资产收益率 (ROA)，越高越好
    )
    ```

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_65.png)

2.  **添加过滤条件**：确保我们只查询股票池（沪深300）内的股票。
    ```python
    # 添加过滤条件，仅查询股票池内的股票
    query_up = query_up.filter(
        fundamentals.stockcode.in_(context.hs300)
    )
    ```

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_67.png)

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_69.png)

3.  **执行查询并转换数据**：使用 `get_fundamentals` 执行查询，并对返回的数据进行转置，使股票代码成为行索引，指标成为列，方便后续处理。
    ```python
    # 执行查询，并对结果进行转置
    df_up = get_fundamentals(query_up).T
    ```

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_71.png)

![](img/a62d7953f7ba1f9bc2301ec2c28bb1fd_73.png)

4.  **获取“越低越好”的指标**：用同样的方法获取另一类指标。
    ```python
    # 构建查询语句，获取“越低越好”的指标
    query_down = query(
        fundamentals.financial_indicator.debt_to_asset_ratio, # 资产负债率，越低越好
        fundamentals.eod_derivative_indicator.pb_ratio,       # 市净率 (PB)，越低越好
        fundamentals.eod_derivative_indicator.market_cap      # 总市值，越低越好
    ).filter(
        fundamentals.stockcode.in_(context.hs300)
    )
    df_down = get_fundamentals(query_down).T
    ```

执行完以上步骤后，我们将得到两个 `DataFrame`：`df_up` 和 `df_down`。它们的形状预计是 `(300, 3)`，分别包含了300只股票在3个“越高越好”和3个“越低越好”指标上的数据。这为我们下一步的因子打分奠定了基础。
