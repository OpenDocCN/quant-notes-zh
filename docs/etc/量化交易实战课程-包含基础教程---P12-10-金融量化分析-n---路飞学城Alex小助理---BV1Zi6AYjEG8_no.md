## 创建特定数值的数组

![](img/2356764ab9d6c200df4e649ef467ec71_7.png)

除了使用`np.array()`将列表转换为数组外，NumPy还提供了多种便捷函数来创建具有特定初始值的数组。

![](img/2356764ab9d6c200df4e649ef467ec71_9.png)

以下是几种创建特定数值数组的方法：

![](img/2356764ab9d6c200df4e649ef467ec71_11.png)

![](img/2356764ab9d6c200df4e649ef467ec71_13.png)

*   **创建全零数组**：使用`np.zeros(shape, dtype)`函数。`shape`参数指定数组的形状（如长度），`dtype`参数可选，用于指定数据类型。默认创建浮点型（`float64`）数组。
    ```python
    import numpy as np
    a = np.zeros(10) # 创建长度为10的全零浮点数组
    b = np.zeros(10, dtype=int) # 创建长度为10的全零整数数组
    ```

![](img/2356764ab9d6c200df4e649ef467ec71_15.png)

![](img/2356764ab9d6c200df4e649ef467ec71_17.png)

*   **创建全一数组**：使用`np.ones(shape, dtype)`函数。其参数和默认行为与`np.zeros`类似。
    ```python
    c = np.ones(5) # 创建长度为5的全一浮点数组
    ```

![](img/2356764ab9d6c200df4e649ef467ec71_19.png)

![](img/2356764ab9d6c200df4e649ef467ec71_21.png)

![](img/2356764ab9d6c200df4e649ef467ec71_23.png)

*   **创建“空”数组**：使用`np.empty(shape, dtype)`函数。此函数只分配内存空间，**不会初始化数组元素的值**，因此数组内容将是内存中的随机残留值。其速度比`np.zeros`略快，但仅在你确定后续会覆盖所有数据时使用。
    ```python
    d = np.empty(100) # 创建长度为100的“空”数组，内容随机
    ```

![](img/2356764ab9d6c200df4e649ef467ec71_25.png)

![](img/2356764ab9d6c200df4e649ef467ec71_27.png)

**注意**：`np.empty`创建的不是值为`None`或`NaN`的数组，而是未初始化的内存值。这解释了为什么输出看起来是“随机”的小数。在性能要求高且后续会填充所有数据的场景下，可以使用此方法。

![](img/2356764ab9d6c200df4e649ef467ec71_29.png)

![](img/2356764ab9d6c200df4e649ef467ec71_31.png)

## 创建序列数组

![](img/2356764ab9d6c200df4e649ef467ec71_33.png)

![](img/2356764ab9d6c200df4e649ef467ec71_35.png)

在数据处理中，经常需要生成有规律的数值序列。NumPy为此提供了强大的工具。

![](img/2356764ab9d6c200df4e649ef467ec71_37.png)

![](img/2356764ab9d6c200df4e649ef467ec71_39.png)

以下是创建序列数组的两种主要方法：

![](img/2356764ab9d6c200df4e649ef467ec71_41.png)

![](img/2356764ab9d6c200df4e649ef467ec71_43.png)

![](img/2356764ab9d6c200df4e649ef467ec71_45.png)

*   **使用`np.arange`**：类似于Python内置的`range`函数，但功能更强大，可以生成数组。它接受起始值、结束值（不包含）和步长参数。**关键区别**在于`np.arange`的步长可以是小数。
    ```python
    e = np.arange(100) # 生成0到99的整数数组
    f = np.arange(0, 100, 0.5) # 生成0到99.5，步长为0.5的数组
    ```

![](img/2356764ab9d6c200df4e649ef467ec71_47.png)

![](img/2356764ab9d6c200df4e649ef467ec71_49.png)

*   **使用`np.linspace`**：用于在指定的区间内生成**等间隔**的数值。它接受起始值、结束值（默认包含）和**要生成的元素数量**（份数）。这在需要固定采样点数量的场景（如绘图）中非常有用。
    ```python
    g = np.linspace(0, 10, 5) # 在0到10之间生成5个等间隔的数：[0., 2.5, 5., 7.5, 10.]
    ```

![](img/2356764ab9d6c200df4e649ef467ec71_51.png)

![](img/2356764ab9d6c200df4e649ef467ec71_53.png)

![](img/2356764ab9d6c200df4e649ef467ec71_55.png)

**应用示例**：绘制函数图像。例如，要绘制函数`y = x²`在区间[-10, 10]上的图像，可以先用`np.linspace`生成足够密集的X坐标点，再计算对应的Y值，最后用绘图库（如Matplotlib）连接这些点，形成平滑的曲线。
```python
x = np.linspace(-10, 10, 10000) # 生成10000个点
y = x ** 2 # 向量化计算y值
# 之后可将x, y传入matplotlib进行绘图
```

*   **创建单位矩阵**：使用`np.eye(N)`函数。单位矩阵是一个方阵，其主对角线上的元素为1，其余元素为0。
    ```python
    I = np.eye(3) # 创建3x3的单位矩阵
    # 输出：[[1., 0., 0.],
    #        [0., 1., 0.],
    #        [0., 0., 1.]]
    ```
