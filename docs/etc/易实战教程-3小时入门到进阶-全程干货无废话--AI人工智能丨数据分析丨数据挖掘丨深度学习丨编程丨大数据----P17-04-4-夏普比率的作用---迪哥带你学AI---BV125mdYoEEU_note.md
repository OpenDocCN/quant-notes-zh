## 夏普比率的定义与意义

上一节我们介绍了投资回报率，本节中我们来看看如何结合风险来评估收益。夏普比率描述的是：在承担单位风险的情况下，我们能获得多少超额收益。

我们可以用一个生活中的例子来理解：一份在叙利亚日薪数万的雇佣兵工作，其高薪对应的是极高的生命风险。在投资中，高收益往往也伴随着高风险。夏普比率就是用来衡量，为了获得这些超额收益，我们所承担的风险是否“值得”。

![](img/842d9bd2236abe2216f284d830e35c22_1.png)

**核心公式**：夏普比率 = (投资组合预期收益率 - 无风险收益率) / 投资组合收益率的标准差

![](img/842d9bd2236abe2216f284d830e35c22_3.png)

指标越高，意味着在承担相同风险的情况下，获得的超额收益越高，该投资也就越有吸引力。

![](img/842d9bd2236abe2216f284d830e35c22_5.png)

## 夏普比率的计算逻辑

理解了夏普比率的意义后，我们来看看它的具体计算逻辑。它主要比较两个部分：一是你的投资组合带来的超额收益（即总收益减去无风险收益），二是你的投资组合所承担的风险（用收益率的标准差来衡量）。

例如，银行的无风险理财产品年化收益可能是5%，而一个股票投资组合的年化收益可能是15%。两者的差值10%就是你可能获得的超额收益。夏普比率就是将这个超额收益，与你投资组合收益的波动性（风险）进行比较。

## 使用Python计算夏普比率

接下来，我们通过Python代码来实际计算夏普比率。在计算前，我们需要准备好投资组合的历史收益率数据，并处理好可能存在的缺失值。

以下是计算夏普比率的步骤和关键代码：

1.  **准备数据**：获取投资组合的历史日收益率序列，并填充缺失值（例如，用前一天的收益率填充）。
2.  **设定参数**：确定年化无风险收益率（例如5%）和一年的交易天数（例如250天）。
3.  **执行计算**：按照夏普比率公式进行计算。注意，由于我们使用的是日收益率数据，需要将其年化。

```python
import numpy as np
import pandas as pd

![](img/842d9bd2236abe2216f284d830e35c22_7.png)

# 假设 `returns` 是一个Pandas Series，包含了投资组合的日收益率序列
# 步骤1: 填充缺失值（前向填充）
returns_filled = returns.fillna(method='ffill')

# 步骤2: 设定参数
risk_free_rate = 0.05  # 年化无风险收益率，例如5%
trading_days_per_year = 250  # 年交易天数

# 步骤3: 计算夏普比率
# 计算超额收益的均值（日度）
excess_return_daily = returns_filled - risk_free_rate / trading_days_per_year
# 计算日收益率的标准差
return_std_daily = returns_filled.std()
# 计算年化夏普比率
sharpe_ratio = np.sqrt(trading_days_per_year) * (excess_return_daily.mean() / return_std_daily)

print(f"该投资组合的年化夏普比率为: {sharpe_ratio:.4f}")
```

运行以上代码后，我们可以得到一个具体的夏普比率数值。例如，在比较多个股票时，我们可以分别计算它们的夏普比率。

计算完成后，我们需要解读结果。夏普比率的值可以是正数、负数或零。

*   **正值**：表示投资组合获得了超越无风险收益的回报，值越大越好。
*   **零**：表示投资组合的收益仅与无风险收益相当。
*   **负值**：表示投资组合的收益还不如无风险收益，承担风险没有带来正向回报。
