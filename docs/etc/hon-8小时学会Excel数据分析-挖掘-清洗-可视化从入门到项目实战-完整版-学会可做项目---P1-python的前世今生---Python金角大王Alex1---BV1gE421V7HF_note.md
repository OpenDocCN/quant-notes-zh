世界上存在众多编程语言。在茫茫的编程大海中，我们为什么要选择Python呢？

首先，请听一个故事。编程语言的发明，大多源于人们对现状的不满意。荷兰程序员吉多·范罗苏姆认为当时的各类编程语言十分复杂、麻烦，便发明了一门功能强大、语法简洁的新语言。Python的设计理念就是优雅、明确、简单。对于初学者来说，Python是最合适不过的语言。

在1989年的圣诞节，吉多开始着手编写新的脚本解释程序。到1991年，第一个用C语言编写的Python编译器问世。吉多慷慨大方地把Python开放给所有人，让Python在众多程序员的呵护之下逐步强化。1994年1月，Python发布新版本。2000年，Python 2诞生。2008年，Python 3.0成功面世。Python逐步成为一门现代化的编程语言。

吉多的T恤上印有“人生苦短，我用Python”的字样。这句话不止表达出他本人对Python的喜爱，也说明了Python的高效。

它为什么叫Python呢？一个编程语言能和蟒蛇有什么关系？实际上，之所以取这个名字，是因为吉多当时在追一个英国喜剧《蒙提·派森的飞行马戏团》，于是便用喜剧团体的名字来命名新程序语言。之后由于这个名字的缘故，Python的logo被设计成两条缠绕的蟒蛇。

随着大数据、人工智能的兴起，Python风头无两，在数据科学和机器学习中得到广泛应用。全球范围内如同海啸一般的数据，通过Python进行收集、整理、分析。应用将会不断造福整个社会。许多知名网站如豆瓣、知乎、谷歌等都使用Python开发。曾经风靡一时的《阴阳师》和《魔兽世界》也是利用Python开发的。

上一节我们介绍了Python的诞生背景和发展历程，本节中我们来看看Python的几个核心特点。

![](img/33735e9ab3a08788f67e1c9dc640939f_6.png)

Python的代码非常简洁，易于读写。你会发现读起来就像在读英文一样。我们将它与编程语言C进行对比，一目了然。

Python还是一门解释型语言。Python代码不需要编译成二进制代码，而是由Python解释器把源代码转换成字节码的中间形式直接运行。这个过程可以用一个简单的公式表示：

![](img/33735e9ab3a08788f67e1c9dc640939f_8.png)

**源代码 -> Python解释器 -> 字节码 -> 执行**

![](img/33735e9ab3a08788f67e1c9dc640939f_10.png)

并且，Python具有丰富的库。当想要使用某个功能时，他人说不定已经帮你写好了。因此，它大大提高了工作效率。在TIOBE编程语言指数走势上，我们也可以发现，Python是过去一年来最受欢迎的编程语言。
