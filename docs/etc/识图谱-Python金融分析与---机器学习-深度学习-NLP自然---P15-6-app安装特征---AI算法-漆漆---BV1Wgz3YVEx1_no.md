## 图统计特征回顾
上一节我们介绍了图结构本身的特征。本节中，我们来看看如何从图中提取节点的统计特征。这些特征包括节点的出度、入度以及与边权重相关的统计量。

![](img/6507167aa5c96d3456b411a79790fa3c_12.png)

![](img/6507167aa5c96d3456b411a79790fa3c_14.png)

![](img/6507167aa5c96d3456b411a79790fa3c_16.png)

![](img/6507167aa5c96d3456b411a79790fa3c_18.png)

以下是提取图统计特征的核心步骤：

![](img/6507167aa5c96d3456b411a79790fa3c_20.png)

![](img/6507167aa5c96d3456b411a79790fa3c_22.png)

![](img/6507167aa5c96d3456b411a79790fa3c_24.png)

1.  **加载边数据**：首先，从数据源加载表示节点间连接的边数据，通常包含 `from_id`、`to_id` 和 `weight` 等字段。
2.  **计算基础统计量**：使用 `groupby` 操作分别按 `from_id` 和 `to_id` 进行分组统计。
    *   **出度**：统计每个 `from_id` 连接到的不同 `to_id` 的数量。
    *   **入度**：统计每个 `to_id` 被多少个不同的 `from_id` 所连接。
3.  **计算加权统计量**：除了计数，还可以计算与边权重（如 `number` 字段）相关的统计值，例如权重的总和、平均值等。
    *   例如：`sum_weight = df.groupby('from_id')['weight'].sum()`
4.  **合并特征**：将计算出的各种统计特征（出度、入度、权重和等）通过 `merge` 操作合并到原始节点特征表中。
5.  **数据过滤与保存**：最后，根据训练集、测试集和验证集的ID列表对特征表进行过滤，确保特征与目标数据集对应，然后将结果保存。

这个过程为图数据增加了节点连接模式的量化描述。

## APP安装特征提取
接下来，我们处理用户的APP安装数据。这份数据记录了每个用户安装了哪些APP（以APP ID列表表示）。其核心假设是：用户的APP安装模式可能与信用行为相关。例如，频繁安装多个借贷类APP的用户可能风险更高。

1.  **数据加载与过滤**：加载 `data.app.txt` 文件。该文件每行包含一个用户ID及其安装的APP ID列表（以逗号分隔）。首先，根据目标用户ID列表（如训练集ID）进行过滤，只保留相关用户的数据。
    ```python
    # 伪代码示例
    target_ids = train_set['id'].tolist()
    app_data = app_data[app_data['id'].isin(target_ids)]
    ```

2.  **数据预处理**：将逗号分隔的APP ID字符串拆分为列表格式，便于后续处理。
    ```python

    ```

3.  **计算统计特征**：基于APP ID列表，计算一系列统计量。这些是基础但有效的特征。
    *   **安装APP总数**：`app_count = len(app_list)`
    *   **安装APP种类数**：`unique_app_count = len(set(app_list))` （如果列表有重复）
    *   **统计值**：还可以计算列表长度（即安装数量）的最大值、最小值、平均值、中位数等。这些统计可以针对整个用户群体计算后作为该用户的特征，或者直接使用该用户自身的列表长度。

4.  **特征衍生（可选）**：除了基础统计，还可以进行更复杂的特征工程。
    *   **与时间特征融合**：如果边数据或其它数据包含时间戳，可以分析用户安装APP的时间密度（例如，短时间内安装大量借贷APP可能是一个风险信号）。本节示例中暂未实现此部分。
    *   **降维处理**：如果APP ID数量非常多（高维稀疏），可以考虑使用如 **PCA（主成分分析）** 或 **SVD（奇异值分解）** 进行降维。但需注意，降维主要用于减少计算复杂度和缓解维度灾难，对模型效果的提升不一定显著，可根据实际情况选择是否采用。
    ```python
    # 伪代码示例：使用PCA降维
    from sklearn.decomposition import PCA
    pca = PCA(n_components=50) # 降至50维
    app_features_reduced = pca.fit_transform(app_features)
    ```
