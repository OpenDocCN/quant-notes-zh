# Python数据分析：P8：03 逻辑函数

在本节课中，我们将要学习Excel中的逻辑函数。逻辑函数用于进行条件判断，返回“是”或“否”的结果，即`TRUE`或`FALSE`。我们将从最基础的`IF`函数开始，逐步学习如何与`AND`、`OR`函数嵌套使用，以及如何进行多层次的`IF`函数嵌套，以应对不同的业务场景。

## 逻辑函数概述

![](img/321b501b71630d0c7d318d7de0e277b1_1.png)

逻辑函数的核心是进行条件判断。在Python中，我们使用`if...elif...else`结构来实现逻辑判断。Excel中的逻辑函数思想与之类似，只是在语法和写法上有所不同。学习函数时，需要掌握其应用场景和参数构成，参数决定了函数在不同场景下的具体行为。

## IF函数：基础条件判断

我们首先介绍最基础的逻辑函数——`IF`函数。它根据给定条件的真假，返回两个不同的值。

### 应用场景与语法

假设有一个计算销售人员提成比例的场景：销售额大于等于1500时，提成比例为10%；否则为8%。

`IF`函数的语法结构如下：
```
=IF(条件, 条件成立时返回的值, 条件不成立时返回的值)
```

### 操作步骤

以下是使用`IF`函数计算提成的具体步骤：

1.  在目标单元格输入等号`=`。
2.  输入函数名`IF`，可以手动输入后按`Tab`键自动补全并添加左括号。
3.  输入第一个参数（条件）：选择销售额单元格，并输入条件`>=1500`。
4.  输入英文逗号`,`分隔参数。
5.  输入第二个参数（条件成立返回值）：`10%`。
6.  输入英文逗号`,`。
7.  输入第三个参数（条件不成立返回值）：`8%`。
8.  输入右括号`)`并按`Enter`键完成公式。

为了将公式正确填充至下方单元格，需要注意单元格引用方式。销售额的引用应随行变化（相对引用），而提成比例`10%`和`8%`是固定值，不应变化。因此，在公式中选中提成比例的值（如`10%`），按`F4`键将其转换为绝对引用（如`$C$2`），然后向下拖动填充公式。

最后，计算提成金额的公式为：`=提成比例 * 销售额`。

### 函数总结

![](img/321b501b71630d0c7d318d7de0e277b1_3.png)

`IF`函数包含三个核心部分：
1.  **条件**：需要进行判断的逻辑表达式。
2.  **条件成立返回值**：当逻辑表达式为`TRUE`时返回的值。
3.  **条件不成立返回值**：当逻辑表达式为`FALSE`时返回的值。

记住函数的用途和参数构成，是熟练使用它的关键。

## 与AND、OR函数嵌套使用

上一节我们学习了单一条件的判断。在实际工作中，我们经常需要判断多个条件是否同时成立或至少一个成立。这时，就需要`AND`和`OR`函数与`IF`函数配合使用。

### AND函数：多条件同时成立

![](img/321b501b71630d0c7d318d7de0e277b1_5.png)

![](img/321b501b71630d0c7d318d7de0e277b1_6.png)

`AND`函数用于判断所有参数是否均为`TRUE`，只有全部为`TRUE`时才返回`TRUE`。

![](img/321b501b71630d0c7d318d7de0e277b1_8.png)

#### 应用场景

![](img/321b501b71630d0c7d318d7de0e277b1_10.png)

假设规则是：员工级别为“A类”**并且**工作年限大于等于10，奖金为5000，否则为0。

#### 操作步骤

以下是使用`IF`和`AND`函数嵌套的步骤：

![](img/321b501b71630d0c7d318d7de0e277b1_12.png)

1.  输入公式：`=IF(AND(条件1, 条件2), 成立返回值, 不成立返回值)`。
2.  第一个条件（`条件1`）：员工级别等于“A类”，即 `B2="A类"`。注意文本需用双引号括起。
3.  第二个条件（`条件2`）：工作年限大于等于10，即 `D2>=10`。
4.  条件成立返回`5000`，不成立返回`0`。
5.  完整公式示例：`=IF(AND(B2="A类", D2>=10), 5000, 0)`。

### OR函数：多条件至少一个成立

`OR`函数用于判断参数中是否至少有一个为`TRUE`，只要有一个为`TRUE`就返回`TRUE`。

#### 应用场景

假设规则是：员工级别为“A类”**或者**客户评价等于5，奖金为5000，否则为0。

#### 操作步骤

1.  输入公式：`=IF(OR(条件1, 条件2), 成立返回值, 不成立返回值)`。
2.  第一个条件（`条件1`）：员工级别等于“A类”，即 `B2="A类"`。
3.  第二个条件（`条件2`）：客户评价等于5，即 `E2=5`。
4.  条件成立返回`5000`，不成立返回`0`。
5.  完整公式示例：`=IF(OR(B2="A类", E2=5), 5000, 0)`。

### 函数总结

*   **`AND(条件1, 条件2, ...)`**：所有条件同时成立，结果才为`TRUE`。
*   **`OR(条件1, 条件2, ...)`**：只要有一个条件成立，结果就为`TRUE`。

![](img/321b501b71630d0c7d318d7de0e277b1_14.png)

`AND`和`OR`通常与`IF`嵌套，用于处理多列数据构成的多条件判断。

## IF函数嵌套：多层级条件判断

前面我们学习了处理多列条件的方法。对于同一列数据需要划分多个区间进行判断的情况（例如根据分数段评定等级），则需要使用`IF`函数的嵌套。

### 应用场景

![](img/321b501b71630d0c7d318d7de0e277b1_16.png)

假设根据客户评价发放奖金：
*   评价大于等于4，奖金5000。
*   评价大于等于3但小于4，奖金3000。
*   评价小于3，奖金0。

### 操作步骤与逻辑

这需要两层`IF`函数进行嵌套判断：

1.  第一层`IF`判断是否`>=4`：
    *   如果成立，返回`5000`。
    *   如果不成立，则进入第二层判断。
2.  第二层`IF`判断是否`>=3`：
    *   注意，此时数据已经默认是`<4`了，所以只需判断是否`>=3`。
    *   如果成立，返回`3000`。
    *   如果不成立（即`<3`），返回`0`。

具体公式如下：
```
=IF(E2>=4, 5000, IF(E2>=3, 3000, 0))
```
**要点**：书写时需确保括号配对，每个`IF`函数都有完整的三个参数。

### 函数总结

`IF`函数嵌套用于实现**同一列数据多阶段、多层级**的条件判断。其核心思想是：当第一个条件不满足时，在“条件不成立返回值”的位置嵌入一个新的`IF`函数，进行下一层判断，如此往复。

![](img/321b501b71630d0c7d318d7de0e277b1_18.png)

## 课程总结

本节课我们一起学习了Excel中的逻辑函数，这是进行数据条件判断的基础。

1.  **`IF`函数**：用于最基本的条件判断，结构为`IF(条件, 真值, 假值)`。
2.  **`AND`与`OR`函数**：与`IF`嵌套，处理涉及多列数据的复合条件判断。`AND`要求所有条件同时满足，`OR`要求至少一个条件满足。
3.  **`IF`函数嵌套**：用于对单列数据进行多区间、多层级的复杂条件判断。

![](img/321b501b71630d0c7d318d7de0e277b1_20.png)

掌握这些逻辑函数，能够帮助我们高效地完成数据清洗、分类和计算等任务，是进行数据分析的重要一步。
