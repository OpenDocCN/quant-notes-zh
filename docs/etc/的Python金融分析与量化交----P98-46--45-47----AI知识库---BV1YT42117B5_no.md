## 概述
在本节课中，我们将学习如何对选出的股票因子进行排序和打分。这是构建多因子模型的关键步骤，我们将分别处理“越高越好”和“越低越好”两类因子，并为它们分配合理的分数。

上一节我们介绍了如何筛选和分类因子，本节中我们来看看如何为这些因子进行具体的打分操作。

## 遍历因子与排序
我们得到了两个DataFrame，分别包含“越高越好”和“越低越好”的因子。现在需要分别遍历这两个DataFrame，对其中的每个因子进行排序。

以下是遍历和排序的具体步骤：
1.  首先，从“越高越好”的DataFrame开始。这个DataFrame的结构是300行（股票）× 3列（因子）。
2.  我们需要遍历的是每一个因子，而不是每一行数据。因此，要获取DataFrame的列名列表，然后遍历每一列。
3.  对于遍历到的每一个因子（列），我们需要根据该因子的值对DataFrame进行排序。使用 `sort_values` 方法，并通过 `by` 参数指定按照当前因子列排序。
4.  为了将排序结果直接保存在原始DataFrame中，而不需要额外赋值，我们设置参数 `inplace=True`。

```python
# 示例：对‘越高越好’的DataFrame按某一因子排序
df_up.sort_values(by=factor_name, inplace=True)
```

## 为排序结果打分
排序完成后，我们需要为排序结果打分。为了简化，我们直接根据排名顺序分配分数：第一名分数最高，最后一名分数最低。

![](img/548401da6f0d8d62241febd0340fd964_5.png)

![](img/548401da6f0d8d62241febd0340fd964_7.png)

对于包含300只股票的列表：
*   **“越高越好”的因子**：因子值最大的股票应得最高分（300分），值最小的得最低分（1分）。
*   **“越低越好”的因子**：因子值最小的股票应得最高分（300分），值最大的得最低分（1分）。

![](img/548401da6f0d8d62241febd0340fd964_9.png)

![](img/548401da6f0d8d62241febd0340fd964_11.png)

我们将使用NumPy的 `linspace` 函数来生成一个等差分数序列。这个函数非常方便，可以指定起始值、终止值和元素个数。

![](img/548401da6f0d8d62241febd0340fd964_13.png)

![](img/548401da6f0d8d62241febd0340fd964_15.png)

以下是 `linspace` 函数的使用方法：
```python
import numpy as np
# 生成从1到300的300个等差数字（用于“越高越好”）
scores_up = np.linspace(1, 300, 300)
# 生成从300到1的300个等差数字（用于“越低越好”）
scores_down = np.linspace(300, 1, 300)
```

## 整合代码流程
现在，我们将上述步骤整合到完整的代码流程中。

首先，需要导入必要的工具包。
```python
import numpy as np
import pandas as pd
```

以下是完整的打分流程：
1.  **对“越高越好”的因子进行打分**：遍历因子，排序后，使用 `np.linspace(1, len(df), len(df))` 生成分数并赋值给该因子列。
2.  **对“越低越好”的因子进行打分**：流程相同，但在生成分数时使用 `np.linspace(len(df), 1, len(df))`。
3.  操作完成后，我们得到两个新的DataFrame，它们的结构仍是300×3，但其中的值已从原始因子值替换为对应的排名分数。

## 后续步骤：合并与计算总分
为每个因子打完分后，我们得到了两个分数表。接下来，需要将这两个表合并，并计算每只股票在所有因子上的总分，从而得到最终的股票综合排名。这将是下一节要完成的内容。
