# 量化金融基础知识：P30：量化沿革

## 概述
在本节课中，我们将要学习量化金融的发展历程。我们将从历史背景出发，了解量化方法如何逐步融入金融领域，并探讨其核心驱动因素与关键发展阶段。

## 量化金融的起源与背景

量化金融并非凭空出现，它的兴起与金融市场的发展和科技进步紧密相连。

在20世纪中叶以前，金融分析主要依赖经验和定性判断。随着计算机技术的诞生和理论经济学的发展，人们开始尝试用数学和统计模型来描述市场行为。

## 量化金融发展的核心驱动因素

上一节我们介绍了量化金融的历史背景，本节中我们来看看推动其发展的几个关键力量。以下是三个主要驱动因素：

*   **理论突破**：例如**哈里·马科维茨**在1952年提出的**现代投资组合理论**，其核心公式是寻求在给定风险水平下收益最大化，或在给定收益水平下风险最小化。该理论用数学方法证明了分散投资的好处。
*   **技术进步**：计算机性能的飞跃和电子交易系统的普及，使得处理海量数据、运行复杂模型和实现高速交易成为可能。
*   **数据可得性**：金融市场电子化产生了巨量的交易数据，为模型构建和验证提供了基础。

## 量化金融的主要发展阶段

了解了驱动因素后，我们进一步梳理量化金融演进的脉络。其发展大致经历了以下几个阶段：

1.  **理论奠基期**：以马科维茨投资组合理论、**威廉·夏普**的资本资产定价模型等为代表，为量化分析提供了理论基础。
2.  **模型应用期**：金融机构开始将理论模型应用于实践，如风险管理中的**风险价值模型**。
3.  **高频交易与算法交易期**：借助强大的算力，机构开发出复杂的算法进行高频交易，追求微小的价差收益。
4.  **大数据与人工智能期**：当前阶段，机器学习、自然语言处理等AI技术被用于挖掘非结构化数据，以预测市场走势。

## 总结
本节课中我们一起学习了量化金融的沿革。我们从其起源背景出发，探讨了理论突破、技术进步和数据可得性三大驱动因素，并回顾了从理论奠基到人工智能应用的四个主要发展阶段。理解这段历史有助于我们把握量化金融的本质和未来趋势。
