## 气泡图：展示三维数据关系

![](img/9165fe7bd8be56f2929eea08f591a74a_3.png)

上一节我们介绍了散点图，本节中我们来看看气泡图。气泡图与散点图类似，但能展示三维数据之间的关系。

![](img/9165fe7bd8be56f2929eea08f591a74a_5.png)

散点图展示的是两类数据（二维）之间的相关关系。气泡图则能展示三类数据（三维）之间的相关关系。气泡图通过两个坐标轴（如X轴和Y轴）来体现前两个维度的数据关系，而第三个维度的数据则通过气泡的大小来体现。气泡越大，代表该数据点的第三个维度值越大；气泡越小，则代表值越小。

以下是创建气泡图的步骤：

![](img/9165fe7bd8be56f2929eea08f591a74a_7.png)

1.  选择所有需要绘制的数据。
2.  点击“插入”选项卡，选择“推荐的图表”。
3.  在弹出的窗口中，切换到“所有图表”标签页，选择“散点图”或“气泡图”类别，并选择一种气泡图样式（例如第二种带气泡的样式），点击“确定”。

初始生成的图表可能不符合预期，需要进行数据系列编辑。

1.  右键点击图表，选择“选择数据”。
2.  在弹出的对话框中，选中数据系列（如“锻炼小时时长”），点击“编辑”。
3.  在“编辑数据系列”对话框中：
    *   **X轴系列值**：设置为代表第一个维度的数据区域（例如体重数据 `B2:B9`）。
    *   **Y轴系列值**：设置为代表第二个维度的数据区域（例如减轻体重数据 `D2:D9`）。
    *   **气泡大小**：设置为代表第三个维度的数据区域（例如锻炼小时时长 `C2:C9`）。
4.  点击“确定”完成设置。

![](img/9165fe7bd8be56f2929eea08f591a74a_9.png)

为了使数据趋势更明显，可以调整坐标轴范围。例如，右键点击水平坐标轴，选择“设置坐标轴格式”，在“坐标轴选项”中将最小值从0调整为150。

![](img/9165fe7bd8be56f2929eea08f591a74a_11.png)

通过分析气泡图，我们可以观察不同数据点之间的关系。例如，气泡的位置（X, Y坐标）反映了前两个维度的关系，而气泡的大小则提供了第三个维度的信息，从而帮助我们进行综合判断。

## 饼图：展示部分与整体关系

接下来我们学习饼图。饼图主要用于描述部分与整体之间的关系，展示某一整体中各个组成部分的占比情况。

![](img/9165fe7bd8be56f2929eea08f591a74a_13.png)

例如，分析一家公司各款手机的销量占比，就可以使用饼图。

![](img/9165fe7bd8be56f2929eea08f591a74a_15.png)

以下是创建和设置饼图的步骤：

1.  选择数据区域（如手机型号和销量）。
2.  点击“插入” > “推荐的图表”，选择“饼图”类别中的一种样式，点击“确定”。

创建饼图后，通常需要添加数据标签以显示具体数值或百分比。

1.  点击饼图，然后点击右上角的“+”号（图表元素），勾选“数据标签”。
2.  默认的数据标签可能只显示值。为了同时显示值和百分比，需要进一步设置：右键点击数据标签，选择“设置数据标签格式”。
3.  在右侧窗格的“标签选项”中，勾选“值”和“百分比”。这样，饼图的每个扇区就会同时显示具体数值和所占百分比。
4.  如果只想显示百分比，则只勾选“百分比”即可。

通过饼图，我们可以清晰地看到每个类别（如各款手机）在总量中的占比，从而快速识别出主要贡献部分或畅销产品。

![](img/9165fe7bd8be56f2929eea08f591a74a_17.png)

## 雷达图：进行多维度综合对比

![](img/9165fe7bd8be56f2929eea08f591a74a_19.png)

![](img/9165fe7bd8be56f2929eea08f591a74a_21.png)

下面我们来看雷达图。雷达图常用于在多维度上对多个分类进行综合对比分析，例如员工能力评估。

雷达图将多个维度的数据映射到由中心点辐射出的多条轴上，形成一个多边形。不同分类的数据系列会形成不同的多边形，便于直观比较。

以下是创建雷达图的步骤：

![](img/9165fe7bd8be56f2929eea08f591a74a_23.png)

![](img/9165fe7bd8be56f2929eea08f591a74a_25.png)

1.  选择需要对比的数据（如不同员工在各能力维度上的得分）。
2.  点击“插入” > “推荐的图表”，切换到“所有图表”标签页，选择“雷达图”类别，点击“确定”。

![](img/9165fe7bd8be56f2929eea08f591a74a_27.png)

在生成的雷达图中，每个点离中心越远，代表在该维度上的能力越强。通过比较不同数据系列（如员工A和员工B）形成的多边形，可以快速识别各自的优势与劣势。例如，一个多边形在“沟通能力”轴上延伸更远，说明该员工沟通能力更强。

雷达图能有效呈现个体在多个维度上的综合表现，适用于人才评估、产品特性对比等场景。

![](img/9165fe7bd8be56f2929eea08f591a74a_29.png)

## 面积图：通过面积度量数据大小

现在我们来学习面积图。面积图通过图形覆盖面积的大小来度量数据量，面积越大，代表在该分类或时间点上的数据量越大。

![](img/9165fe7bd8be56f2929eea08f591a74a_31.png)

面积图可以看作是折线图的一种变体，它用颜色填充折线下方与坐标轴之间的区域，形成“面积”感，更强调数据量的累积与对比。

以下是创建面积图的步骤：

![](img/9165fe7bd8be56f2929eea08f591a74a_33.png)

1.  选择数据（如不同城市随时间变化的销量数据）。
2.  点击“插入” > “推荐的图表”，在“所有图表”中选择“面积图”类别（如堆积面积图），点击“确定”。

![](img/9165fe7bd8be56f2929eea08f591a74a_35.png)

在面积图中，不同颜色代表不同分类（如不同城市）。水平坐标轴通常是时间。在某个时间点（如1月），面积最大的颜色块代表该分类（如广州）的数据量最大。随着时间轴变化，各颜色块面积的变化趋势反映了该分类数据量的增减。

![](img/9165fe7bd8be56f2929eea08f591a74a_37.png)

![](img/9165fe7bd8be56f2929eea08f591a74a_39.png)

面积图既能展示趋势（如同折线图），又能通过面积大小直观对比不同分类的数据量。

![](img/9165fe7bd8be56f2929eea08f591a74a_41.png)

## 组合图：对比数值差异大的数据系列

![](img/9165fe7bd8be56f2929eea08f591a74a_43.png)

![](img/9165fe7bd8be56f2929eea08f591a74a_45.png)

最后，我们学习组合图。组合图将两种或以上的图表类型结合在一个坐标系中，常用于对比数值尺度差异很大的数据系列。

![](img/9165fe7bd8be56f2929eea08f591a74a_47.png)

例如，同时展示各城市的销量（数值较大，如几千）和销量占比（数值较小，在0-1之间）。如果使用同一坐标轴，占比数据的折线会因数值太小而几乎贴在坐标轴上，无法观察其变化。此时可以使用组合图，为主数据系列（销量）设置主坐标轴并使用柱状图，为次数据系列（占比）设置次坐标轴并使用折线图。

以下是创建组合图的步骤：

1.  选择所有需要绘制的数据（包括分类、销量、占比等）。
2.  点击“插入” > “推荐的图表”，切换到“所有图表”标签页，选择“组合图”类别。
3.  在组合图设置界面，为每个数据系列指定图表类型和坐标轴：
    *   **系列1（如销量）**：图表类型选择“簇状柱形图”，坐标轴选择“主坐标轴”。
    *   **系列2（如占比）**：图表类型选择“折线图”，坐标轴选择“次坐标轴”。
4.  点击“确定”。

生成的组合图将拥有两个纵坐标轴。主坐标轴对应柱形图，显示销量；次坐标轴对应折线图，显示占比（0-1）。这样，两个数据系列都能清晰地展示其变化和对比关系。

## 图表操作技巧与总结

在操作图表时，有时可能难以准确选中某个图表元素（如图例、数据系列）。此时，可以点击图表，然后在“格式”选项卡下的“当前所选内容”下拉框中，直接选择需要编辑的图表元素名称（如图表标题、绘图区、图例等），即可精确定位。

本节课中我们一起学习了多种高级图表：

*   **气泡图**：用于展示**三维数据**之间的内在联系，通过气泡大小表示第三维度。
*   **饼图**：用于展示**部分与整体**的占比关系。
*   **雷达图**：用于在**多个维度**上对分类进行**综合对比**分析。
*   **面积图**：通过**面积大小**来对比分析数据量，同时展示趋势。
*   **组合图**：将两种图表类型结合，用于对比**数值尺度差异大**的数据系列，通常涉及主、次坐标轴。
