## 循环的必要性
上一节我们介绍了条件判断（`if`语句），它允许程序根据条件执行不同的分支。但在猜年龄的例子中，程序只能猜测一次，无论对错都会结束。如果一次猜不中，就需要反复重启程序，这非常低效。

本节中我们来看看如何让程序能够**重复执行**某段代码，即实现“循环猜测”，直到猜对或达到尝试次数上限。

## `for`循环基础语法
`for`循环是Python中用于重复执行代码块的主要结构之一。它的特点是需要明确指定循环的次数或遍历的范围。

![](img/9fc08ceb04d132d3aacbf45376d3bbb0_7.png)

其基本语法格式如下：
```python
for 临时变量 in 序列:
    # 循环执行的代码块
```

**核心概念解析**：
*   `for` 和 `in` 是固定关键字。
*   **临时变量**：在每次循环中，它会自动被赋值为序列中的当前元素。变量名可以自定义，但习惯上常用 `i`、`j` 等。
*   **序列**：可以是一个列表、字符串，或由 `range()` 函数生成的数字序列。循环会遍历序列中的每一个元素。

![](img/9fc08ceb04d132d3aacbf45376d3bbb0_9.png)

一个简单的例子是使用 `range()` 函数：
```python
for i in range(10):
    print(i)
```
这段代码会打印数字0到9。`range(10)` 生成一个从0开始、到10结束（不包含10）的整数序列，即 `[0, 1, 2, ..., 9]`。每次循环，变量 `i` 会依次取序列中的一个值。

![](img/9fc08ceb04d132d3aacbf45376d3bbb0_11.png)

![](img/9fc08ceb04d132d3aacbf45376d3bbb0_13.png)

> **注意**：`range(10)` 遵循“顾头不顾尾”原则，包含起始值0，不包含结束值10。

## 实践应用：改进猜年龄游戏
现在，我们运用`for`循环来改进之前的猜年龄程序，允许用户最多猜测三次。

![](img/9fc08ceb04d132d3aacbf45376d3bbb0_19.png)

以下是实现步骤：
1.  将黑姑娘的年龄（固定值）定义在循环之外，避免在每次循环中重复定义，以优化程序性能。
2.  使用 `for i in range(3):` 构建一个执行三次的循环。
3.  在循环体内，接收用户输入并进行判断。
4.  如果猜对，使用 `exit()` 函数立即退出程序，避免不必要的后续循环。

![](img/9fc08ceb04d132d3aacbf45376d3bbb0_21.png)

![](img/9fc08ceb04d132d3aacbf45376d3bbb0_23.png)

优化后的核心代码如下：
```python
black_girl_age = 26  # 固定变量放在循环外定义

for i in range(3):
    guess_age = int(input("请输入你猜的年龄："))
    if guess_age == black_girl_age:
        exit("恭喜你，猜对了！")
    elif guess_age > black_girl_age:
        print("猜大了")
    else:
        print("猜小了")
```

## 练习：打印0-100之间的偶数
为了巩固对`for`循环和条件判断的理解，我们来做一个小练习：打印出0到100之间所有的偶数。

**思路分析**：
1.  使用 `for` 循环遍历0到100（`range(101)`）。
2.  在循环体内，判断当前数字是否为偶数。判断方法是：用当前数字对2取模（求余数），如果余数为0，则是偶数。
3.  如果是偶数，则将其打印出来。

实现代码如下：
```python
for num in range(101):
    if num % 2 == 0:  # 判断是否为偶数
        print(f"{num} 是偶数")
```
运行这段代码，控制台将输出0到100之间所有的偶数。

1.  `for`循环用于对序列（如`range()`生成的数字序列、列表等）进行遍历，重复执行代码块。

4.  编写程序时应注意代码优化，例如将不变的变量定义在循环体外。
