## 课程核心模块

我们的课程主要涉及三大核心模块。

![](img/4822d421eee13e22a8a725cb85ebfecf_7.png)

以下是三大模块的简要介绍：

1.  **Python数据分析与处理**：在这个模块中，我们将学习如何使用Python处理和分析数据。
2.  **金融数据分析**：我们将学习如何获取股票数据，并对这些时间序列数据进行操作和分析。
3.  **量化交易策略**：我们将学习如何编写交易策略，并在历史数据中进行回测，以分析策略的优劣。

以上就是我们的三大核心模块。接下来，我们将详细了解一下课程大纲。

## 课程工具与实战风格

在讲解过程中，我们会使用一些特定的工具。现在打开的界面就是我们课程中主要的讲解界面。

本课程没有PPT，不会深入讲解金融理论知识或教授如何炒股。我们的重点在于如何在Python中实现相关任务，即如何分析数据、进行回测并得到结果。因此，课程的核心全部聚焦于代码。整个课程是纯实战内容，每一节课都会讲解如何在代码中实现具体操作。最核心的是学习如何使用Python处理数据、进行回测以及完成量化交易策略。

这就是我们的课程界面。大家现在可能对这个界面还不熟悉，没关系，在正式课程中，我会详细讲解这些工具和代码的使用方法。

在讲解过程中，所有课程内容都将以实际数据为例进行引入。我们会讲解在数据分析、金融数据和时间序列分析中常用的分析方法和统计方法，并一步步演示如何操作。这是我们的第一模块，相当于要掌握这些Python工具包的使用方法，以及学会如何处理金融数据。

这个Notebook主要用于演示我们如何使用Python工具包，并完成一些数据分析案例。

![](img/4822d421eee13e22a8a725cb85ebfecf_9.png)

接下来我们看下一个工具。这里类似，我会讲解我们将要使用的一些回测平台。例如，现在我有了一个量化交易策略的好想法，我想实现它，并看看这个策略在历史数据（比如从2019年到2020年）中能赚多少钱。这就是量化交易回测。

在量化交易部分，我会讲解各种指标（如回测收益、超额收益）如何获取及其含义，并将我们的项目和任务应用到回测平台中。回测平台是实际编写回测策略时会用到的工具，这里先不详细介绍，正式课程中会详细讲解。

这里的含义是：当我们有了一个策略后，可以将其应用到历史数据中，查看回测结果。我们编写了许多策略，包含各种方法和计算逻辑。

![](img/4822d421eee13e22a8a725cb85ebfecf_11.png)

写完策略后，我们运行它。运行完毕后，可以查看在一年、三年或五年中，使用当前策略得到的结果。我们还会与大盘（基准收益）进行比较，看看我们的回测收益是赚是赔，以及当前策略能带来多少超额收益。这就是我们的分析和展示结果。

![](img/4822d421eee13e22a8a725cb85ebfecf_13.png)

相当于在这里我们编写策略，然后执行策略以得到结果。在此基础上，平台会提供详细的交易详情。

例如，每一天买卖了什么，当前的持仓情况，以及账户信息。比如，初始给你10万元，经过一定周期（如3年或5年）后，查看这笔钱还剩多少，最终是赔是赚。

这就是我们在回测平台中可以观察到的指标值。

## 课程特点与大纲

以上简单介绍了几个核心模块。正如之前所说，本课程从零基础开始。如果你觉得以前对Python不熟悉，没关系。课程中会先讲解Python及其工具包的使用方法，以及如何将Python融入金融数据和量化交易的实际项目中。

因此，我们的课程是从零基础开始的纯实战课程，核心重点都放在如何在代码中实现这些功能。在讲解过程中，我的风格会比较通俗，用接地气的方式讲解工具包的使用、策略的设计以及如何在Python中完成项目。所以，这门课程对零基础学员非常友好，学习无压力。

为了方便大家练习和完成小任务，我会提供课程中涉及的所有数据和代码。

最后，我们看一下课程大纲。大纲也分为三个模块。
