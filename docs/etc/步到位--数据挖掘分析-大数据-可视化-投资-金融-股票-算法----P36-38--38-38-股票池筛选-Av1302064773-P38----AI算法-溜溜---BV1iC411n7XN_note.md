## 因子预处理流程回顾
在开始筛选之前，我们需要确保因子数据已经完成了必要的预处理。以下是完整的预处理步骤：

![](img/20b533b2136d4c7068da431dfa2f4904_14.png)

![](img/20b533b2136d4c7068da431dfa2f4904_16.png)

![](img/20b533b2136d4c7068da431dfa2f4904_18.png)

1.  **去除离群值**：使用3σ原则处理因子数据中的极端值。
2.  **标准化**：将因子数据转换为均值为0，标准差为1的标准分布。
3.  **中性化**：消除因子与市值、行业等风格因素的影响。

![](img/20b533b2136d4c7068da431dfa2f4904_20.png)

![](img/20b533b2136d4c7068da431dfa2f4904_22.png)

![](img/20b533b2136d4c7068da431dfa2f4904_24.png)

![](img/20b533b2136d4c7068da431dfa2f4904_26.png)

以下是预处理步骤的代码实现：
```python
# 1. 去除离群值 (3σ)
factor_no_outliers = three_sigma(factor_data)
# 2. 标准化
factor_standardized = standardize(factor_no_outliers)
# 3. 中性化
factor_neutralized = neutralize(factor_standardized)
```
完成以上步骤后，我们得到了一个经过清洗、标准化和中性化处理的因子 `factor_neutralized`，可以用于后续的选股。

![](img/20b533b2136d4c7068da431dfa2f4904_28.png)

![](img/20b533b2136d4c7068da431dfa2f4904_30.png)

## 基于因子进行股票筛选
接下来，我们将使用处理好的因子对股票池进行筛选。核心思想是选择因子值最有利（例如，市净率最低）的一批股票。

![](img/20b533b2136d4c7068da431dfa2f4904_32.png)

以下是筛选逻辑的步骤：

![](img/20b533b2136d4c7068da431dfa2f4904_34.png)

1.  **确定筛选比例**：首先，我们需要决定选择多少比例的股票。例如，选择因子值最小的前20%的股票。
2.  **计算分位点**：根据设定的比例，计算因子对应的分位点数值。
3.  **生成股票列表**：筛选出因子值小于或等于该分位点的所有股票，这些股票构成我们的目标股票池。

![](img/20b533b2136d4c7068da431dfa2f4904_36.png)

![](img/20b533b2136d4c7068da431dfa2f4904_38.png)

![](img/20b533b2136d4c7068da431dfa2f4904_40.png)

以下是实现筛选的代码：
```python
# 设定筛选比例，例如选择前20%
selection_ratio = 0.2
# 计算20%分位点的因子值
threshold = factor_neutralized.quantile(selection_ratio)
# 筛选出因子值小于等于阈值的股票
selected_stocks = factor_neutralized[factor_neutralized <= threshold].index.tolist()
```
这样，我们就得到了一个根据因子策略选出的股票列表 `selected_stocks`。

## 生成调仓操作列表
在量化交易中，我们通常需要根据新的股票列表来调整现有持仓。这意味着我们需要计算需要卖出哪些股票，以及需要买入哪些股票。

![](img/20b533b2136d4c7068da431dfa2f4904_48.png)

![](img/20b533b2136d4c7068da431dfa2f4904_50.png)

![](img/20b533b2136d4c7068da431dfa2f4904_52.png)

以下是计算调仓操作的步骤：

![](img/20b533b2136d4c7068da431dfa2f4904_54.png)

![](img/20b533b2136d4c7068da431dfa2f4904_56.png)

![](img/20b533b2136d4c7068da431dfa2f4904_58.png)

1.  **获取当前持仓**：首先，需要获取当前账户中持有的所有股票。
2.  **计算需卖出的股票**：找出当前持仓中，不在新股票池 `selected_stocks` 里的股票。这些股票需要被卖出。
3.  **计算需买入的股票**：找出新股票池 `selected_stocks` 中，当前未持有的股票。这些股票需要被买入。

![](img/20b533b2136d4c7068da431dfa2f4904_60.png)

![](img/20b533b2136d4c7068da431dfa2f4904_62.png)

以下是计算调仓列表的代码：
```python
# 假设 context.portfolio.positions 可以获取当前持仓的股票代码列表
current_holdings = list(context.portfolio.positions.keys())

![](img/20b533b2136d4c7068da431dfa2f4904_64.png)

![](img/20b533b2136d4c7068da431dfa2f4904_66.png)

![](img/20b533b2136d4c7068da431dfa2f4904_68.png)

# 计算需要卖出的股票：当前持有但不在新选股池中的股票
stocks_to_sell = list(set(current_holdings) - set(selected_stocks))

![](img/20b533b2136d4c7068da431dfa2f4904_70.png)

![](img/20b533b2136d4c7068da431dfa2f4904_72.png)

# 计算需要买入的股票：在新选股池中但当前未持有的股票
stocks_to_buy = list(set(selected_stocks) - set(current_holdings))
```
通过以上计算，我们得到了明确的调仓指令列表 `stocks_to_sell` 和 `stocks_to_buy`，可以在后续的交易函数中执行。
