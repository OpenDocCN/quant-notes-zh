以下是判断是否需要调仓的核心代码逻辑：

```python
if len(stocks_to_delete) > 0:
```

### 问题一：中性化函数参数缺失

![](img/247359c682d9ed5e9548f1bc2a584396_47.png)

在中性化处理数据时，我们调用的函数缺少了必要的参数。

![](img/247359c682d9ed5e9548f1bc2a584396_49.png)

![](img/247359c682d9ed5e9548f1bc2a584396_51.png)

**错误信息**：函数缺少参数。
**解决方案**：检查函数定义，补全所需的参数。在本例中，需要传入数据框和指标名称。

![](img/247359c682d9ed5e9548f1bc2a584396_53.png)

![](img/247359c682d9ed5e9548f1bc2a584396_55.png)

```python
# 修正前（可能缺少参数）
neutralized_data = neutralize_factor(data)
# 修正后
neutralized_data = neutralize_factor(data, factor_name='target_factor')
```

### 问题二：模型未训练

![](img/247359c682d9ed5e9548f1bc2a584396_59.png)

![](img/247359c682d9ed5e9548f1bc2a584396_61.png)

在使用统计模型（如线性回归）进行中性化时，必须先对模型进行训练（拟合数据），然后才能使用其残差。

![](img/247359c682d9ed5e9548f1bc2a584396_63.png)

![](img/247359c682d9ed5e9548f1bc2a584396_65.png)

**错误信息**：尝试使用未训练的模型计算结果。
**解决方案**：在计算残差前，先调用模型的 `fit` 方法。

![](img/247359c682d9ed5e9548f1bc2a584396_67.png)

![](img/247359c682d9ed5e9548f1bc2a584396_69.png)

```python
# 修正前
result = model.resid
# 修正后
model.fit(X, y) # 先训练模型
result = model.resid # 再获取残差
```

![](img/247359c682d9ed5e9548f1bc2a584396_77.png)

**错误信息**：`Series` 对象没有某个属性。
**解决方案**：仔细检查并修正函数名拼写。

![](img/247359c682d9ed5e9548f1bc2a584396_79.png)

```python
# 修正后（正确拼写）
selected = data[data['factor'] <= data['factor'].quantile(0.3)]
```

### 回测结果解读

![](img/247359c682d9ed5e9548f1bc2a584396_97.png)

回测完成后，我们主要关注以下几个核心绩效指标：

![](img/247359c682d9ed5e9548f1bc2a584396_99.png)

![](img/247359c682d9ed5e9548f1bc2a584396_101.png)

*   **累计收益率**：策略在整个回测期间的总收益。
*   **年化收益率**：将累计收益率折算为年化标准，便于比较。
*   **基准收益率**：同期市场基准（如沪深300指数）的收益，用于对比。
*   **夏普比率**：衡量策略承担单位风险所获得的超额回报。公式为：
    `Sharpe Ratio = (Rp - Rf) / σp`
    其中 `Rp` 是策略收益率，`Rf` 是无风险利率，`σp` 是策略收益波动率。
*   **最大回撤**：策略净值从高点回落的最大幅度，反映策略的风险控制能力。

![](img/247359c682d9ed5e9548f1bc2a584396_103.png)

![](img/247359c682d9ed5e9548f1bc2a584396_105.png)

在本例的回测中，我们观察到：
*   策略的累计收益率和年化收益率在部分时段优于基准指数。
*   夏普比率虽然不高，但表明策略获得了一定的风险调整后收益。
*   最大回撤控制在可接受范围内。

![](img/247359c682d9ed5e9548f1bc2a584396_107.png)

**注意**：本次回测结果基于历史数据和特定参数，仅为演示流程，不构成任何投资建议。实际市场表现可能完全不同。

## 策略流程总结

![](img/247359c682d9ed5e9548f1bc2a584396_117.png)

![](img/247359c682d9ed5e9548f1bc2a584396_119.png)

本节课中我们一起学习了如何构建并评估一个完整的因子选股策略。整个流程可以概括为以下几步：

![](img/247359c682d9ed5e9548f1bc2a584396_121.png)

![](img/247359c682d9ed5e9548f1bc2a584396_123.png)

1.  **初始化与定时器设置**：在策略的 `initialize` 函数中，设定调仓频率（如每月）。
2.  **股票池初筛**：在 `before_trading_start` 或调仓函数中，过滤掉不满足基本条件的股票（如ST股、停牌股）。
3.  **因子计算与预处理**：对初筛后的股票计算所需因子（如市值、市净率），并进行标准化、中性化等预处理。
4.  **因子筛选**：根据因子值（例如，选择市净率最低的30%股票）构建最终的目标股票池。
5.  **调仓执行**：
    *   卖出当前持有但不在新目标池中的股票。
    *   将资金平均买入新目标池中的所有股票。
6.  **回测与评估**：运行策略回测，分析收益率、夏普比率、最大回撤等关键指标，评估策略效果。

![](img/247359c682d9ed5e9548f1bc2a584396_125.png)

通过这个实战案例，我们掌握了将量化选股思想转化为可执行代码的基本方法，并对策略的回测评估有了直观的认识。
