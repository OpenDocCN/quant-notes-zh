## 构建回归方程

![](img/a2aa5cdffbd74d46b0d804ea90491047_8.png)

上一节我们介绍了数据的基本处理，本节中我们来看看如何构建回归方程来描述两个变量之间的关系。

![](img/a2aa5cdffbd74d46b0d804ea90491047_10.png)

构建回归方程的方法有很多，可以使用多种工具包。这里我们选择使用NumPy库中的`polyfit`函数来计算回归系数。该函数并非训练一个复杂的模型，而是直接帮我们计算出回归方程的系数。

![](img/a2aa5cdffbd74d46b0d804ea90491047_12.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_14.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_16.png)

以下是使用`polyfit`函数的关键步骤：

![](img/a2aa5cdffbd74d46b0d804ea90491047_18.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_20.png)

1.  **准备数据**：首先需要明确分析哪两个指标之间的关系，并将它们作为参数传入。
2.  **指定方程阶数**：`polyfit`函数需要指定回归方程的阶数（`deg`参数）。
    *   当 `deg=1` 时，拟合的是一次线性方程，形式为 **`y = kx + b`**。
    *   当 `deg=2` 时，拟合的是二次曲线方程，形式为 **`y = ax² + bx + c`**。
    *   在本例中，我们分析的是线性关系，因此指定 `deg=1`。
3.  **处理缺失值**：在计算前，必须确保数据中没有空值（NaN），否则可能导致计算错误。我们可以使用Pandas的`dropna`方法进行清理。

![](img/a2aa5cdffbd74d46b0d804ea90491047_22.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_24.png)

以下是构建回归方程的代码示例：
```python
import numpy as np
import pandas as pd

![](img/a2aa5cdffbd74d46b0d804ea90491047_26.png)

# 假设 df 是包含数据的DataFrame，`column_x` 和 `column_y` 是两个指标列
# 1. 清理缺失值
df = df.dropna(subset=[‘column_x‘, ‘column_y‘])

# 2. 提取数据
x = df[‘column_x‘].values
y = df[‘column_y‘].values

![](img/a2aa5cdffbd74d46b0d804ea90491047_28.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_30.png)

# 3. 使用 polyfit 计算回归系数 (k, b)
coefficients = np.polyfit(x, y, deg=1)
k, b = coefficients
print(f“回归方程为: y = {k:.4f} * x + {b:.4f}“)
```

![](img/a2aa5cdffbd74d46b0d804ea90491047_32.png)

## 计算相关系数

除了回归方程，相关系数是衡量两个变量线性关系强度和方向的另一个重要指标。它的计算和展示非常简单。

![](img/a2aa5cdffbd74d46b0d804ea90491047_74.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_76.png)

在Pandas中，可以直接调用`.corr()`方法计算DataFrame中所有数值列两两之间的相关系数矩阵。

![](img/a2aa5cdffbd74d46b0d804ea90491047_78.png)

以下是计算相关系数的代码：
```python
# 计算两个指定列的相关系数
correlation = df[[‘column_x‘, ‘column_y‘]].corr()
print(correlation)

![](img/a2aa5cdffbd74d46b0d804ea90491047_80.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_82.png)

# 也可以直接计算两个Series的相关系数
corr_value = df[‘column_x‘].corr(df[‘column_y‘])
print(f“指标X与指标Y的相关系数为: {corr_value:.4f}“)
```
相关系数矩阵是一个对称矩阵，对角线上的值均为1（表示自身完全相关）。非对角线上的值就是两个不同指标之间的相关系数，其值介于-1到1之间。

![](img/a2aa5cdffbd74d46b0d804ea90491047_84.png)

## 附加任务：随时间变化的相关系数

![](img/a2aa5cdffbd74d46b0d804ea90491047_88.png)

为了更深入地分析，我们可以研究两个指标间的相关性如何随时间变化。这通常通过计算滚动窗口（滑动窗口）内的相关系数来实现。

![](img/a2aa5cdffbd74d46b0d804ea90491047_90.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_92.png)

以下是实现步骤：

![](img/a2aa5cdffbd74d46b0d804ea90491047_94.png)

1.  **定义窗口**：选择一个合适的时间窗口大小（例如250个交易日）。
2.  **滚动计算**：对其中一个指标序列，在定义的窗口上滚动，并计算每个窗口内该序列与另一个固定指标序列的相关系数。
3.  **可视化结果**：将计算出的滚动相关系数绘制成折线图，观察其随时间变化的趋势。

![](img/a2aa5cdffbd74d46b0d804ea90491047_96.png)

以下是实现代码示例：
```python
# 设置滚动窗口大小
window_size = 250

![](img/a2aa5cdffbd74d46b0d804ea90491047_98.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_100.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_102.png)

# 计算滚动相关系数
rolling_corr = df[‘column_x‘].rolling(window=window_size).corr(df[‘column_y‘])

![](img/a2aa5cdffbd74d46b0d804ea90491047_104.png)

![](img/a2aa5cdffbd74d46b0d804ea90491047_106.png)

# 绘制滚动相关系数变化图
plt.figure(figsize=(12, 6))
rolling_corr.plot(figsize=(12, 6), title=f‘{window_size}日滚动相关系数变化‘)
plt.axhline(y=0, color=‘black‘, linestyle=‘-‘, linewidth=0.5) # 添加y=0参考线
plt.ylabel(‘相关系数‘)
plt.xlabel(‘时间‘)
plt.grid(True, linestyle=‘--‘, alpha=0.5)
plt.show()
```
这张图能告诉我们，两个指标间的相关性并非一成不变，可能会在市场不同阶段呈现增强或减弱的变化，这对于动态策略的构建具有参考意义。

![](img/a2aa5cdffbd74d46b0d804ea90491047_110.png)
