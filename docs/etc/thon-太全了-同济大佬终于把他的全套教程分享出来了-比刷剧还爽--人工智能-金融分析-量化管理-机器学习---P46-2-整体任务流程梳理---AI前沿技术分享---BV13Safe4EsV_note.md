在本节课中，我们将学习量化投资策略中的一个核心环节——如何对股票进行多因子综合打分，并梳理出一个完整的策略执行流程。我们将从获取数据开始，逐步讲解如何根据因子特性进行评分，最终通过汇总分数来筛选出目标股票。

---

上一节我们介绍了如何获取和预处理因子数据。本节中，我们来看看如何利用这些数据对股票进行打分。

有了已知的因子数据后，下一步就是进行打分。

以下是打分的基本思路：我们可以根据每个因子的数值大小，将其划分到不同的区间，并为每个区间赋予相应的分数。打分的方法有很多种，这里我们通过一个例子来详细说明。

假设我们有一个股票池，例如沪深300指数成分股。我们为每只股票计算了几个关键指标：
*   **A因子**：越大越好
*   **B因子**：越大越好
*   **C因子**：越小越好
*   **D因子**：越小越好

每只股票都会得到这四个因子的具体数值。例如，`ID1` 这只股票的因子值为：
*   A = 0.71
*   B = 0.28
*   C = 0.39
*   D = 0.11

`ID2` 这只股票的因子值为：
*   A = 0.45
*   B = 0.17
*   C = 0.81
*   D = 0.02

接下来，我们需要为每个因子设计打分规则。首先，将所有因子的数值范围（假设已归一化到0-1之间）划分为若干个区间。

对于 **“越大越好”** 的因子（如A、B），分数设计如下：
*   数值在 [0.8, 1.0] 区间：得 **5分**
*   数值在 [0.6, 0.8) 区间：得 **4分**
*   数值在 [0.4, 0.6) 区间：得 **3分**
*   数值在 [0.2, 0.4) 区间：得 **2分**
*   数值在 [0.0, 0.2) 区间：得 **1分**

对于 **“越小越好”** 的因子（如C、D），分数设计则相反：
*   数值在 [0.0, 0.2) 区间：得 **5分**
*   数值在 [0.2, 0.4) 区间：得 **4分**
*   数值在 [0.4, 0.6) 区间：得 **3分**
*   数值在 [0.6, 0.8) 区间：得 **2分**
*   数值在 [0.8, 1.0] 区间：得 **1分**

现在，我们可以为示例股票打分了。

对于 `ID1`：
*   A因子值0.71，属于[0.6, 0.8)区间，得 **4分**。
*   B因子值0.28，属于[0.2, 0.4)区间，得 **2分**。
*   C因子值0.39，属于[0.2, 0.4)区间。由于C是“越小越好”，此区间得 **4分**。
*   D因子值0.11，属于[0.0, 0.2)区间。由于D是“越小越好”，此区间得 **5分**。
*   **总分** = 4 + 2 + 4 + 5 = **15分**。

对于 `ID2`：
*   A因子值0.45，属于[0.4, 0.6)区间，得 **3分**。
*   B因子值0.17，属于[0.0, 0.2)区间，得 **1分**。
*   C因子值0.81，属于[0.8, 1.0]区间。由于C是“越小越好”，此区间得 **1分**。
*   D因子值0.02，属于[0.0, 0.2)区间。由于D是“越小越好”，此区间得 **5分**。
*   **总分** = 3 + 1 + 1 + 5 = **10分**。

按照这个方法，我们可以计算出股票池中所有股票的总分。最后，对所有股票按总分从高到低进行排序，选择排名最高的若干只（例如前10名）作为下一次调仓的投资标的。

除了上述按固定区间打分的方法，还有其他方式，例如直接根据数值在所有股票中的排名来赋分。如果因子是“越大越好”，那么排名第一的股票赋予最高分，排名最后的赋予最低分。核心目标是一致的：**通过一个综合指标（总分）对所有股票进行横向比较和排序**。

了解了打分法的核心思想后，我们来看一个完整的策略执行流程。这个流程也是我们后续将要实现的策略蓝图。

![](img/c810c6a9832bbb39c6f23b1581492b88_9.png)

以下是整体策略的执行步骤：

1.  **确定股票池**：首先，需要明确策略操作的股票范围。在本例中，我们选择沪深300指数成分股作为初始股票池。
2.  **设置调仓周期**：确定策略调仓的频率。常见的调仓周期有每月或每季度。我们需要设置一个定时器，在每个调仓日自动触发调仓函数。
3.  **实现调仓函数**：这是策略的核心部分，通常命名为 `rebalance`。在该函数中，我们将执行以下操作：
    *   **数据获取与处理**：读取当前时刻所有股票池内股票的因子数据（A, B, C, D）。
    *   **因子打分**：根据预设的规则（如上述区间打分法），为每只股票的每个因子计算得分。
    *   **汇总总分**：将每只股票的所有因子得分相加，得到该股票的综合总分。
    *   **筛选标的**：根据总分对所有股票进行排序，选出排名靠前的股票（如前10名）作为新的持仓组合。

这个流程结构清晰，相对简单。接下来，我们就可以尝试用这种打分法来构建一个简单的策略，并回测其历史表现，看看仅凭几个基础因子能否获得超越基准的收益。
