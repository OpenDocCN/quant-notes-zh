## 概述：什么是布尔型索引？

![](img/998b76243424c35f6d84c5d9199c6acf_10.png)

![](img/998b76243424c35f6d84c5d9199c6acf_12.png)

布尔型索引的核心思想是：通过一个与原始数组长度相同的布尔值数组（True/False）来选取数据。只有对应位置为True的元素才会被选中并返回。

![](img/998b76243424c35f6d84c5d9199c6acf_14.png)

![](img/998b76243424c35f6d84c5d9199c6acf_16.png)

## 从基础方法到NumPy方法

![](img/998b76243424c35f6d84c5d9199c6acf_18.png)

以下是筛选数组中大于5的元素的两种方法对比。

![](img/998b76243424c35f6d84c5d9199c6acf_20.png)

![](img/998b76243424c35f6d84c5d9199c6acf_22.png)

首先，我们创建一个示例数组。

```python
import numpy as np

![](img/998b76243424c35f6d84c5d9199c6acf_24.png)

# 创建一个示例列表并转换为NumPy数组
list_a = [8, 3, 6, 2, 9, 1]
A = np.array(list_a)
print("原始数组 A:", A)
```

![](img/998b76243424c35f6d84c5d9199c6acf_26.png)

![](img/998b76243424c35f6d84c5d9199c6acf_28.png)

如果你熟悉Python基础，可能会想到使用`filter`函数结合`lambda`表达式。

```python
# 使用filter和lambda筛选大于5的数
result_filter = list(filter(lambda x: x > 5, list_a))
print("使用filter筛选的结果:", result_filter)
```
这种方法可行，但代码相对繁琐。

### 方法二：使用NumPy的布尔型索引

![](img/998b76243424c35f6d84c5d9199c6acf_37.png)

NumPy提供了一种极其简洁的写法。

![](img/998b76243424c35f6d84c5d9199c6acf_39.png)

![](img/998b76243424c35f6d84c5d9199c6acf_41.png)

```python
# 使用NumPy布尔型索引筛选大于5的数
result_numpy = A[A > 5]

```
可以看到，仅用 `A[A > 5]` 一行代码就完成了筛选，非常高效。

## 布尔型索引的工作原理

![](img/998b76243424c35f6d84c5d9199c6acf_45.png)

![](img/998b76243424c35f6d84c5d9199c6acf_47.png)

布尔型索引的实现分为两步，理解这两步至关重要。

**第一步：创建布尔数组**
`A > 5` 是一个数组与标量的比较运算。它会将数组`A`中的每个元素与5进行比较，返回一个由布尔值（True/False）组成的新数组。
```python
bool_array = A > 5
print("布尔数组 (A > 5):", bool_array)
# 输出示例：[ True False  True False  True False]
```

![](img/998b76243424c35f6d84c5d9199c6acf_49.png)

![](img/998b76243424c35f6d84c5d9199c6acf_51.png)

**第二步：应用布尔索引**
将上一步得到的布尔数组放入原始数组的方括号 `A[...]` 中。NumPy会根据布尔数组，只返回对应位置为True的元素。
```python
# 这等价于 A[A > 5]
result = A[bool_array]
print("应用布尔索引后的结果:", result)
```
这就是布尔型索引的底层原理：**传入一个长度匹配的布尔数组，取出所有True位置上的值**。

## 组合条件筛选

![](img/998b76243424c35f6d84c5d9199c6acf_56.png)

![](img/998b76243424c35f6d84c5d9199c6acf_58.png)

在实际应用中，我们经常需要根据多个条件进行筛选。以下是实现方法。

![](img/998b76243424c35f6d84c5d9199c6acf_60.png)

### 问题一：筛选数组中所有“大于5的偶数”

![](img/998b76243424c35f6d84c5d9199c6acf_62.png)

这需要同时满足两个条件（“与”关系）。我们不能直接写成 `A[A > 5][A % 2 == 0]`，因为第二次索引时数组长度已经改变，会导致错误。

**正确写法**：使用位运算符 `&` (与) 来组合条件，并且每个条件必须用括号括起来。
```python
# 筛选大于5的偶数
result = A[(A > 5) & (A % 2 == 0)]
print("大于5的偶数:", result)
```
**注意**：这里必须使用 `&` 而不是Python关键字 `and`，因为 `&` 是用于数组的逐元素位与操作，而 `and` 无法处理数组。

### 问题二：筛选数组中“大于5的数”或“所有的偶数”

这满足任一条件即可（“或”关系）。我们使用位运算符 `|` (或)。

![](img/998b76243424c35f6d84c5d9199c6acf_64.png)

```python
# 筛选大于5的数 或 偶数
result = A[(A > 5) | (A % 2 == 0)]
print("大于5的数 或 偶数:", result)
```

## 核心要点与总结

![](img/998b76243424c35f6d84c5d9199c6acf_66.png)

本节课中我们一起学习了NumPy的布尔型索引。

![](img/998b76243424c35f6d84c5d9199c6acf_68.png)

1.  **核心功能**：`array[boolean_array]` 可以根据布尔数组快速筛选数据。
2.  **工作原理**：
    *   先通过比较运算（如 `A > 5`）生成布尔数组。
    *   再用该布尔数组作为索引，提取数据。
3.  **多条件组合**：
    *   **与** 操作使用 `&`，语法：`(条件1) & (条件2)`
    *   **或** 操作使用 `|`，语法：`(条件1) | (条件2)`
    *   **重要**：每个条件必须用括号括起来。
4.  **优势**：相比传统的循环或`filter`方法，布尔型索引语法简洁、执行效率高，是数据筛选的利器。
