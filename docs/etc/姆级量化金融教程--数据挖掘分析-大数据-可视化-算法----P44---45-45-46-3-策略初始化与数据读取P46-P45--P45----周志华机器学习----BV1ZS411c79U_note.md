## 构建数据获取函数

调仓函数的核心是获取经过评分的股票列表。为此，我们需要一个专门的函数 `get_stocks` 来获取并处理财务数据。

以下是 `get_stocks` 函数需要完成的主要步骤。

![](img/4a99ef6999e34707d2706653a2f07ccf_50.png)

1.  **查询财务数据**：使用 `get_fundamentals` 函数查询所需的财务指标。
2.  **数据转换**：对返回的数据进行转置，以便后续处理。
3.  **区分指标类型**：将指标分为“越高越好”和“越低越好”两类。

![](img/4a99ef6999e34707d2706653a2f07ccf_52.png)

## 获取“越高越好”的指标

![](img/4a99ef6999e34707d2706653a2f07ccf_54.png)

![](img/4a99ef6999e34707d2706653a2f07ccf_56.png)

我们首先获取那些数值越大代表公司基本面越好的指标。

![](img/4a99ef6999e34707d2706653a2f07ccf_58.png)

```python
def get_stocks(context):
    # 构建查询语句，获取“越高越好”的指标
    query_up = query(
        fundamentals.eod_derivative_indicator.eps, # 每股收益 (EPS)
        fundamentals.financial_indicator.return_on_equity, # 净资产收益率 (ROE)
        fundamentals.financial_indicator.return_on_invested_capital # 投入资本回报率 (ROIC)
    ).filter(
        fundamentals.stockcode.in_(context.hs300) # 过滤，只选择沪深300成分股
    )
    # 执行查询并对结果进行转置，方便按股票索引
    df_up = get_fundamentals(query_up).T
```

## 获取“越低越好”的指标

![](img/4a99ef6999e34707d2706653a2f07ccf_62.png)

![](img/4a99ef6999e34707d2706653a2f07ccf_64.png)

接下来，我们获取那些数值越小越好的财务指标。

![](img/4a99ef6999e34707d2706653a2f07ccf_66.png)

![](img/4a99ef6999e34707d2706653a2f07ccf_68.png)

```python
    # 构建查询语句，获取“越低越好”的指标
    query_down = query(
        fundamentals.financial_indicator.debt_to_asset_ratio, # 资产负债率
        fundamentals.eod_derivative_indicator.pb_ratio, # 市净率 (PB)
        fundamentals.eod_derivative_indicator.market_cap # 总市值
    ).filter(
        fundamentals.stockcode.in_(context.hs300) # 过滤，只选择沪深300成分股
    )
    # 执行查询并对结果进行转置
    df_down = get_fundamentals(query_down).T
```

请注意，`get_fundamentals` 返回的是一个 `DataFrame`，我们通过 `.T` 属性对其进行转置操作，使得行索引为股票代码，列索引为财务指标，这更符合后续排序和计算的需求。

![](img/4a99ef6999e34707d2706653a2f07ccf_70.png)

执行完上述代码后，我们将得到两个 `DataFrame`：
*   `df_up`：形状约为 `(300, 3)`，包含300支股票的3个“越高越好”指标。
*   `df_down`：形状约为 `(300, 3)`，包含300支股票的3个“越低越好”指标。
