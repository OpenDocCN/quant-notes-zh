# 量化交易入门：P1：使用AI辅助搭建本地回测环境 🚀

![](img/bc076b2cf6a411465c0469e69a8a9c3a_1.png)

![](img/bc076b2cf6a411465c0469e69a8a9c3a_3.png)

在本节课中，我们将学习如何在不具备系统编程知识的情况下，利用AI工具（如ChatGPT）来搭建一个本地的量化交易回测环境。我们将使用`Backtrader`这个强大的Python库，它支持股票、期货等多种资产的回测，并且所有计算都在本地进行，能有效保护策略隐私。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_5.png)

## 环境与工具准备

![](img/bc076b2cf6a411465c0469e69a8a9c3a_7.png)

上一节我们概述了课程目标，本节中我们来看看搭建环境需要准备哪些工具和步骤。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_9.png)

首先，我们需要在计算机上安装Python。Python环境搭建的教程网上资源丰富，此处不再赘述。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_11.png)

安装好Python后，我们需要安装本课程的核心库——`Backtrader`。安装过程非常简单，使用Python的包管理工具`pip`即可完成。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_13.png)

以下是安装`Backtrader`的命令：
```bash
pip install backtrader
```
运行该命令后，若显示安装成功的信息，则表明环境准备就绪。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_15.png)

![](img/bc076b2cf6a411465c0469e69a8a9c3a_17.png)

## 创建项目结构

![](img/bc076b2cf6a411465c0469e69a8a9c3a_19.png)

安装好必要的库之后，我们需要创建一个清晰的项目文件夹来管理代码和数据。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_21.png)

1.  新建一个主文件夹，可以命名为`QuantTrading`。
2.  在主文件夹内，再新建一个名为`data`的子文件夹，用于存放股票的历史数据文件。
3.  在主文件夹内，创建一个新的文本文件，并将其后缀名改为`.py`，例如`strategy.py`。这就创建了一个Python脚本文件。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_23.png)

至此，项目在文件路径上的基础框架已经搭建完成。我们可以使用VS Code或其他代码编辑器打开这个主文件夹进行后续操作。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_25.png)

## 获取历史数据

![](img/bc076b2cf6a411465c0469e69a8a9c3a_27.png)

项目结构创建好后，我们需要获取回测所需的历史价格数据。一个常用且免费的数据来源是雅虎财经（Yahoo Finance）。

以下是获取数据的步骤：
1.  访问雅虎财经网站。
2.  搜索你感兴趣的股票代码（例如：AMD）。
3.  在股票页面找到“历史数据”选项卡。
4.  选择时间范围（例如“最大”范围），然后点击“下载”按钮，数据会以CSV格式保存。
5.  将下载好的CSV文件复制到之前创建的`data`文件夹内。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_29.png)

![](img/bc076b2cf6a411465c0469e69a8a9c3a_31.png)

## 利用AI生成回测代码

核心的编程部分我们将借助AI来完成。你可以使用ChatGPT（GPT-3.5/4）或类似的AI助手。与AI沟通时，指令必须清晰明确。

首先，向AI发送如下指令：
“你好，请使用Backtrader库帮我搭建一个量化交易回测的基本框架。我的数据文件是`AMD.csv`，存放在项目文件夹下的`data`子文件夹里。请使用一个简单的移动平均线交叉策略，并输出完整的Python代码。”

AI在理解你的需求后，会生成一段包含数据加载、策略定义和回测引擎的完整代码。将生成的代码复制到你之前创建的`strategy.py`文件中。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_33.png)

**注意**：AI生成的代码可能默认使用`yfinance`库在线获取数据。为确保使用我们本地下载的数据，可能需要根据AI的代码提示，调整数据加载部分的路径，指向本地的`AMD.csv`文件。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_35.png)

## 运行与优化回测

![](img/bc076b2cf6a411465c0469e69a8a9c3a_37.png)

![](img/bc076b2cf6a411465c0469e69a8a9c3a_39.png)

![](img/bc076b2cf6a411465c0469e69a8a9c3a_41.png)

代码复制并调整好后，就可以运行第一次回测了。在终端中进入你的项目目录，运行以下命令：
```bash
python strategy.py
```
程序会执行回测，并输出图表。图表上通常会显示资产价格曲线、买卖信号点以及资金曲线。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_43.png)

初次回测的结果可能未考虑交易成本。为了使回测更贴近真实交易，我们需要添加佣金模块。你可以将你的券商佣金规则（例如每股固定费用或按比例收费）告知AI，请求它帮你修改代码，加入自定义的佣金计算逻辑。将AI生成的新代码替换原有部分，再次运行回测，就能得到考虑交易成本后的更准确结果。

## 核心要点总结

![](img/bc076b2cf6a411465c0469e69a8a9c3a_45.png)

本节课中我们一起学习了从零开始搭建本地量化回测环境的完整流程。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_47.png)

![](img/bc076b2cf6a411465c0469e69a8a9c3a_49.png)

![](img/bc076b2cf6a411465c0469e69a8a9c3a_51.png)

我们来总结几个关键要点：
*   **环境隔离**：`Backtrader`在本地运行，保障了交易策略的安全性。
*   **明确指令**：与AI交互时，对文件路径、数据格式、需求目标的描述务必清晰准确。
*   **迭代优化**：回测是一个迭代过程，可以从简单策略开始，逐步加入佣金、滑点等真实交易因素进行优化。

![](img/bc076b2cf6a411465c0469e69a8a9c3a_53.png)

通过本教程，你已经掌握了无需深厚编程基础，利用现代AI工具快速构建和测试交易策略原型的方法。这为后续探索更复杂的量化策略打下了坚实的基础。
