## 模块的安装方法

![](img/a0ac275b77c2044005a39620c3c735d9_11.png)

了解了模块的来源后，我们来看看如何将它们安装到自己的电脑上。安装第三方模块主要有两种方式。

### 方式一：手动下载安装

第一种方式是直接在PYPI网站上找到目标模块，手动下载后进行安装。

![](img/a0ac275b77c2044005a39620c3c735d9_5.png)

![](img/a0ac275b77c2044005a39620c3c735d9_7.png)

在模块页面找到 `Download` 按钮，下载压缩包，然后解压并执行安装命令。这种方式对初学者可能不够友好。

### 方式二：使用pip命令安装（推荐）

更简单通用的方法是使用Python自带的包管理工具 **pip**。`pip install` 命令可以自动从网络下载并安装指定的模块。

以下是基本命令格式：
```bash
pip install 模块名
```
或针对Python 3环境：
```bash
pip3 install 模块名
```

例如，安装一个名为 `requests` 的模块：
```bash
pip3 install requests
```
命令执行后，pip会连接到PYPI服务器下载该模块并自动完成安装。安装成功后，会显示 `Successfully installed` 的提示。

![](img/a0ac275b77c2044005a39620c3c735d9_11.png)

安装完成后，就可以在Python代码中像导入内置模块一样导入并使用它了：
```python
import requests
```

## 在PyCharm中安装模块

如果你使用的是PyCharm集成开发环境，还有更直观的图形化安装方式。

以下是操作步骤：
1.  打开PyCharm，点击界面上的扳手图标或进入 `File -> Settings`。
2.  找到 `Project: [你的项目名] -> Python Interpreter`。
3.  这个页面会列出当前Python解释器已安装的所有第三方包。
4.  点击列表左上角的 `+` 号。
5.  在弹出的搜索框中输入你想安装的模块名（例如 `openpyxl`）。
6.  选择模块和版本，点击 `Install Package` 按钮即可。

![](img/a0ac275b77c2044005a39620c3c735d9_9.png)

安装过程无报错即表示成功。之后，你就可以在项目代码中直接 `import openpyxl` 了。

![](img/a0ac275b77c2044005a39620c3c735d9_13.png)

## 配置国内镜像源加速下载

![](img/a0ac275b77c2044005a39620c3c735d9_15.png)

由于网络原因，从国外的PYPI官方源下载模块可能速度很慢甚至失败。为了解决这个问题，我们可以使用国内的镜像服务器，它们同步了PYPI上的所有包，下载速度更快。

国内常用的镜像源有：
*   清华大学：`https://pypi.tuna.tsinghua.edu.cn/simple`
*   豆瓣：`https://pypi.douban.com/simple`

### 在PyCharm中配置镜像源

在PyCharm的包管理界面配置镜像源非常方便：
1.  在 `Python Interpreter` 页面，点击 `Manage Repositories`。
2.  在弹出的窗口中，你可以修改或添加新的仓库地址。将默认地址替换为上述国内镜像地址之一即可。
3.  点击 `Apply` 和 `OK` 保存。

之后在PyCharm中安装模块，就会从你配置的国内镜像源下载。

![](img/a0ac275b77c2044005a39620c3c735d9_17.png)

### 在命令行中使用镜像源

![](img/a0ac275b77c2044005a39620c3c735d9_19.png)

在终端或命令行中使用pip时，也可以通过附加参数指定镜像源。

![](img/a0ac275b77c2044005a39620c3c735d9_21.png)

![](img/a0ac275b77c2044005a39620c3c735d9_23.png)

以下是使用豆瓣镜像源安装模块的命令格式：
```bash
pip install 模块名 -i https://pypi.douban.com/simple --trusted-host pypi.douban.com
```
其中 `-i` 参数指定镜像地址，`--trusted-host` 参数是为了让pip信任这个主机地址。

![](img/a0ac275b77c2044005a39620c3c735d9_25.png)

![](img/a0ac275b77c2044005a39620c3c735d9_13.png)

![](img/a0ac275b77c2044005a39620c3c735d9_27.png)

使用国内镜像源后，下载和安装速度通常会显著提升。

![](img/a0ac275b77c2044005a39620c3c735d9_29.png)
