在本节课中，我们将要学习NumPy数组的“花式索引”。这是一种非常灵活的数据选取方式，允许我们根据一个整数数组（即索引列表）来选取数组中任意位置、无规律的元素。我们将通过具体例子，学习如何在二维数组中应用花式索引，并了解它与其他索引方式的组合规则。

上一节我们介绍了布尔型索引，本节中我们来看看另一种强大的索引方式——花式索引。

![](img/a52ec6b81c8c9da889cf3f3e2b73fdfd_5.png)

花式索引的核心在于，它允许我们使用一个整数数组来指定要选取的元素位置。例如，对于一个一维数组，如果我们想选取第1、3、4、6、7号元素，使用普通索引或切片都无法直接实现，因为它们的位置没有规律。

![](img/a52ec6b81c8c9da889cf3f3e2b73fdfd_7.png)

以下是花式索引的基本语法：
```python
import numpy as np
a = np.arange(20)
indices = [1, 3, 4, 6, 7]
result = a[indices]
```
在这个例子中，`indices`列表指定了我们想要选取的索引位置，`a[indices]`会返回这些位置上的元素组成的新数组。

对于二维数组，花式索引同样适用，并且可以与其他索引方式（如普通索引、切片、布尔索引）组合使用，但需要遵循特定的规则。

以下是二维数组中花式索引的几种组合方式：

1.  **行使用普通索引，列使用切片**
    例如，选取第0行，第2到第4列（不包含第4列）的数据。
    ```python
    arr = np.arange(20).reshape(4, 5)
    result = arr[0, 2:4]
    ```

2.  **行使用普通索引，列使用布尔索引**
    例如，选取第0行中所有大于2的元素。
    ```python
    arr = np.arange(20).reshape(4, 5)
    result = arr[0, arr[0] > 2]
    ```

3.  **行和列同时使用花式索引（需注意）**
    如果我们想选取第1行和第3行，以及第1列和第3列交叉位置的四个元素（即位置(1,1), (1,3), (3,1), (3,3)），直接写`arr[[1,3], [1,3]]`会得到意想不到的结果。这种写法会被解释为选取(1,1)和(3,3)这两个位置的值，而不是一个矩形区域。

    正确的做法是分步进行：
    ```python
    arr = np.arange(20).reshape(4, 5)
    # 第一步：选取第1行和第3行的所有列
    step1 = arr[[1, 3], :]
    # 第二步：在step1的结果上，选取所有行的第1列和第3列
    result = step1[:, [1, 3]]
    ```
    通过先选取行再选取列，我们可以得到想要的2x2子数组。

本节课中我们一起学习了NumPy的花式索引。我们了解到，花式索引通过传递一个整数索引数组，可以灵活地选取数组中任意位置、无规律的元素。在二维数组中，花式索引可以与其他索引方式组合，但需要注意，行和列不能同时直接使用花式索引来选取矩形区域，而需要分步操作。掌握花式索引能极大地增强我们在金融数据分析中处理复杂数据选取需求的能力。
