# Python金融量化投资：P45：第一个量化策略-1 🚀

在本节课中，我们将学习如何在一个量化交易平台上编写第一个简单的量化策略。我们的主要目标是熟悉平台的基本框架、常用函数和API的使用方法。这个策略虽然简单，但涵盖了设置股票池、制定买卖规则等核心步骤。

## 策略概述

我们的策略逻辑如下：
1.  **设置股票池**：选择沪深300指数的所有成分股作为我们关注和交易的对象。
2.  **买入条件**：当某只股票的当前价格低于10元，并且我们当前没有持有该股票时，则买入。
3.  **卖出条件**：
    *   **止盈**：当持有的股票价格相比买入价上涨了25%时，全部卖出以锁定利润。
    *   **止损**：当持有的股票价格相比买入价下跌了10%时，全部卖出以控制损失。

接下来，我们将一步步实现这个策略。

### 设置股票池

策略的第一步是确定我们要交易的股票范围，即“股票池”。我们选择沪深300指数的成分股。沪深300指数是由上海和深圳证券交易所中选取的300只具有市场代表性的股票组成，其成分股会定期调整。

为了获取这300只股票的代码列表，我们可以使用平台提供的API函数 `get_index_stocks`。获取到的股票代码列表需要存储在一个全局变量中，以便在策略的其他部分随时调用。在这个框架中，预定义了一个全局对象 `g` 来存储用户自定义的全局数据。

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_10.png)

以下是初始化股票池的代码：

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_12.png)

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_14.png)

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_16.png)

```python
# 获取沪深300指数的成分股代码列表，并存储在全局变量g中
g.security = get_index_stocks(‘000300.XSHG‘)
```

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_18.png)

*   `g.security`：我们在全局对象 `g` 上创建了一个名为 `security` 的属性，用于存储股票池。
*   `get_index_stocks(‘000300.XSHG‘)`：这是一个API函数，传入指数代码 `‘000300.XSHG‘`（代表沪深300指数），返回该指数当前的所有成分股代码列表。

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_20.png)

如果你想查看获取到的股票列表，可以使用 `print` 语句打印 `g.security`，输出结果会显示在平台的日志区域。

### 设置交易选项和费用

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_22.png)

除了股票池，在初始化阶段我们通常还需要设置一些交易规则和成本。

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_24.png)

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_26.png)

1.  **使用真实价格**：在模拟交易中使用经过复权处理后的真实价格进行计算，这能更准确地反映实际收益。复权是一个金融数据处理概念，对于初学者，我们只需记住添加这行固定代码即可。

    ```python
    # 设置使用真实价格
    set_option(‘use_real_price‘, True)
    ```

2.  **设置交易成本**：股票交易会产生手续费，主要包括券商佣金和国家印花税。这部分成本会直接影响策略的最终收益，因此必须在回测中设定。以下是A股市场常见的费率设置：

    ```python
    # 设置交易成本：佣金和印花税
    set_order_cost(
        cost_type=‘stock‘,          # 交易类型为股票
        open_tax=0,                 # 买入时印花税为0
        close_tax=0.001,            # 卖出时印花税为千分之一 (0.1%)
        open_commission=0.0003,     # 买入时佣金为万分之三 (0.03%)
        close_commission=0.0003,    # 卖出时佣金为万分之三 (0.03%)
        close_today_commission=0,   # 平今仓佣金（股票为0）
        min_commission=5            # 最低佣金为5元
    )
    ```
    *   **佣金**：买卖双向收取，通常为成交金额的万分之三，但每笔交易有最低5元的限制。
    *   **印花税**：仅在卖出时收取，税率为成交金额的千分之一。

### 初始化函数完整代码

![](img/8cb9572bd4b8f775bd69aaaa8dbc7690_28.png)

将以上部分组合起来，就得到了完整的 `initialize` 函数：

```python
def initialize(context):
    # 设置股票池为沪深300成分股
    g.security = get_index_stocks(‘000300.XSHG‘)
    
    # 设置使用真实价格
    set_option(‘use_real_price‘, True)
    
    # 设置交易成本（佣金和印花税）
    set_order_cost(
        cost_type=‘stock‘,
        open_tax=0,
        close_tax=0.001,
        open_commission=0.0003,
        close_commission=0.0003,
        close_today_commission=0,
        min_commission=5
    )
```

本节课中我们一起学习了如何编写量化策略的初始化部分。我们主要完成了三件事：
1.  使用 `get_index_stocks` API 获取了沪深300指数的成分股作为我们的股票池，并存储在全局变量 `g.security` 中。
2.  通过 `set_option` 设置了使用真实价格进行回测。
3.  通过 `set_order_cost` 设置了股票交易的佣金和印花税成本，使回测环境更贴近现实。
