首先，我们需要将连续的价格变化数据转换为离散的“涨”或“跌”标签。这可以通过与零值比较来实现：大于零为“涨”，小于零为“跌”。

![](img/9e8227ad621a3dd33942c32c94457f19_25.png)

以下是实现该功能的函数代码：

![](img/9e8227ad621a3dd33942c32c94457f19_27.png)

![](img/9e8227ad621a3dd33942c32c94457f19_29.png)

```python
def clip_ins(data, bins):
    global col_names
    col_names = []
    for col in feature_names:
        band = col + ‘_B‘
        data[band] = np.digitize(data[col], bins)
        col_names.append(band)
```

```python
grouped = data.groupby([‘L1_B‘, ‘L2_B‘, ‘direction‘])
result = grouped.size()
print(result)
```

![](img/9e8227ad621a3dd33942c32c94457f19_65.png)

这段代码会输出一个序列，显示在“前一天涨跌情况（L1_B）”、“前两天涨跌情况（L2_B）”以及“当天实际方向（direction）”这三个维度下的数据点数量。例如，`(0, 0, 1.0)` 表示前两天都跌、前一天也跌，但当天实际是上涨的天数。
