## 使用map函数比较数据

![](img/b36d720a51e9c31991b9e92e8726948f_4.png)

![](img/b36d720a51e9c31991b9e92e8726948f_6.png)

上一节我们介绍了如何获取列表中的最小值。本节中我们来看看如何比较两个列表对应元素的大小。

![](img/b36d720a51e9c31991b9e92e8726948f_8.png)

![](img/b36d720a51e9c31991b9e92e8726948f_10.png)

假设我们有两个商店的价格列表：
```python
store1 = [9.99, 11.99, 12.34, 2.01]
store2 = [10.99, 9.99, 11.50, 2.50]
```
如果只想获取单个列表的最小值，可以使用`min()`函数。但要比较两个列表对应位置的价格并取较小值，就需要使用`map`函数。

`map`函数将指定的方法应用到可迭代对象的每个元素上。我们可以用它来比较两个列表：
```python
cheapest = map(min, store1, store2)
```
此时，`cheapest`是一个内存地址（map对象）。要查看结果，需要使用循环将其取出：
```python
for price in cheapest:
    print(price)
```
执行上述代码，将输出两个列表对应位置的较小值：`9.99`、`9.99`、`11.5`、`2.01`。这就是`map`函数的基本用法。

## 理解lambda表达式

![](img/b36d720a51e9c31991b9e92e8726948f_12.png)

接下来，我们探讨`lambda`表达式。`lambda`是一种创建匿名函数的简洁方式。

![](img/b36d720a51e9c31991b9e92e8726948f_14.png)

例如，定义一个计算三个数之和的函数：
```python
def my_function(a, b, c):
    return a + b + c
```
使用`lambda`可以将其简化为一行：
```python
my_lambda = lambda a, b, c: a + b + c
```
调用`my_lambda(1, 2, 3)`将返回`6`。`lambda`表达式适用于定义简单的、一次性的函数。

`if`语句用于根据条件执行不同代码块：
```python
a = 3

```
如果`a`等于`3`，则打印“Good”；否则，不执行任何操作。这是控制程序流程的基础。

## 练习解答与列表推导

![](img/b36d720a51e9c31991b9e92e8726948f_37.png)

以下是练习的一种实现方式：
```python
my_list = []
for number in range(0, 100):
    if number % 2 == 0:
        my_list.append(number)
print(my_list)
```
这段代码创建了一个空列表，遍历0到99，检查每个数字是否为偶数，如果是则添加到列表中。

![](img/b36d720a51e9c31991b9e92e8726948f_39.png)

然而，Python提供了一种更简洁的方法来实现相同功能，称为**列表推导**。

列表推导可以将循环和条件判断压缩成一行代码：
```python
my_list = [number for number in range(0, 100) if number % 2 == 0]
```
这行代码等价于上面多行的`for`循环。它的结构是：`[表达式 for 变量 in 可迭代对象 if 条件]`。在程序员中，这种简洁的“一行式”代码非常流行。

同样，`lambda`表达式也是为了简化代码而生。当你发现代码冗长时，可以思考是否能使用`lambda`或列表推导进行简化。
