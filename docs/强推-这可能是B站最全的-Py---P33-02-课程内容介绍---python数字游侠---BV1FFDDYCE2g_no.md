## 什么是数据分析？

上一节我们介绍了课程的整体框架，本节中我们来看看数据分析究竟是什么。

数据分析是指收集大量数据，运用分析方法和工具，对数据进行分类、汇总，从中提取有价值的信息，并概括总结形成有效结论的过程。

简单来说，数据分析是从大量、看似杂乱无章的数据中提炼信息、研究内在规律、发现问题、提出建议方案，并最终应用于企业决策，以促进企业发展的一项工作。

为了让大家更清晰地理解这个概念，我们来看两个具体的例子。

以下是第一个数据分析的例子：

*   **原始数据**：一张记录某企业不同地区、不同车型的销量与收入的表格。仅看数据表格，难以发现明显规律或问题。
*   **分析过程**：运用对比分析方法，并通过图表进行可视化展示。
*   **发现规律**：图表显示，西北地区的销量最高，而西南地区的销量最低（假设各地区门店数量相同）。
*   **价值体现**：基于此规律，可以分析西北地区的成功经验供其他地区学习，同时探究西南地区销量低的原因并进行优化改进。这能帮助企业整体向好发展。

以下是第二个数据分析的例子：

*   **原始数据**：某车型一整年的月度销量数据表。单看数字，趋势不明显。
*   **分析过程**：通过对比分析与图表可视化。
*   **发现规律**：图表清晰显示，5月份的销量出现了断崖式下滑。
*   **价值体现**：分析销量骤降背后的原因（例如，突发市场变化）。更重要的是，基于此发现，企业可以思考未来如何提高市场敏感度，制定应急预案（A方案、B方案），以减轻类似突发事件对企业的冲击。

从以上例子可以看出，数据分析是运用专业的分析方法和工具，发现数据背后隐藏的规律，总结分析并得出有价值的信息，用于指导商业决策的科学手段。

## 数据分析师需要具备哪些能力？

理解了数据分析是什么之后，我们来看看要成为一名合格的业务数据分析师，需要具备哪些能力。

所需能力主要分为两大类：**硬实力**（工具技能）和**软实力**（思维与业务能力）。因为分析不能凭空想象，必须借助工具呈现；同时，只会使用工具的分析师如同机器人，必须拥有分析思维。

以下是需要掌握的硬实力（分析工具）：

1.  **Excel**：职场必备技能，简单易学，在处理部分数据时高效便捷。本课程将以Excel作为切入点。
2.  **SQL**：用于从数据库中调取数据。随着数据量增长，SQL在面试和日常工作中都至关重要。本课程将重点回顾并讲解多表查询及实战案例。
3.  **SPSS**：专业的统计分析软件，需要一定的统计学基础。掌握SPSS是加分项，但非必须精通。
4.  **Python**：强大的加分项。在实现业务数据分析的自动化操作方面非常高效，目前被越来越多的企业列为优先考虑技能。
5.  **数据可视化工具**：分析结果需要向他人汇报。除了PPT，还需掌握专业工具如 **Tableau**，它能制作出更专业、更具洞察力的图表。

以下是需要培养的软实力：

1.  **业务知识**：了解所从事行业的相关知识，可以通过网络资源提前学习。
2.  **逻辑思维与分析方法**：核心能力。缺乏逻辑思维会导致分析杂乱无章。本课程将融入常用的分析方法。
3.  **沟通表达能力**：在任何岗位都必不可少，用于清晰呈现分析结果并推动决策。
4.  **统计学与机器学习知识**：数据分析的理论基础。本课程将通过接地气的实际案例帮助大家理解这些理论知识。

## 本系列课程大纲设计

基于上述需要掌握的能力，我们的课程设计如下。

首先，我们需要掌握Excel操作，这是数据分析的基础。

以下是Excel数据分析部分的内容：

*   Excel基础操作与技巧
*   公式与函数
*   数据可视化（图表与数据透视表）
*   实战案例：综合练习数据处理与分析能力

![](img/898ab903668d7439da050a7dc559f69e_3.png)

掌握了工具，我们还需要懂业务和分析方法。因此，课程加入了分析思维部分。

以下是业务指标与分析思维部分的内容：

*   常见业务指标与分析方法讲解
*   实战案例：基于电商网站数据的分析，综合运用分析方法、业务指标与Excel技能

以下是SQL部分的内容：

*   SQL语法回顾（重点多表查询）
*   实战案例：基于淘宝天猫用户行为数据的分析，加深对分析方法、业务指标和SQL的理解

以下是统计学基础部分的内容：

*   描述统计与推断统计
*   SPSS软件入门及简单模型应用
*   通过案例理解统计学在分析中的应用

最后，专业的数据可视化能让你的报告脱颖而出。

以下是数据可视化（Tableau）部分的内容：

*   Tableau基础操作与常见分析图表制作
*   对比Excel图表，了解专业可视化工具的优势
