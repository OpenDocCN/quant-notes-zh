## 回归策略分析概述

上一节我们介绍了量化交易的基本概念，本节中我们来看看回归策略分析。

回归策略的目标是预测未来。例如，在股票交易中，我们希望预测明天股价是上涨还是下跌。如果预测上涨，则可以买入；如果预测下跌，则可以提前卖出并在低位买回，从而获利。回归分析就是借助我们拥有的历史数据，去预测接下来走势的过程。

所谓的回归，就是我们要借助于我们现在手里有的一些历史数据。例如，我想预测明天的指标，可以查看今天、前一天、前两天等多个时间点的指标。这里前一天、前两天不是一个选择题，不是说只能选前一天或者前两天。我们可以选择更多的数据。例如，历史数据中，我能利用的信息很多，有前一天的数据，还有前两天的、前三天的、前四天的、前五天的，我都能给它利用上去。相对来说，你用的数据特征越多，效果可能会越好。这就是我们要做的一件事：在历史数据中，找一下之前我们的走势，去预测一下接下来它的走势会长什么样子。这个东西叫做回归策略分析。

## 第二步：数据预处理与特征工程

![](img/7ae9a6ca694aa1e90f557248d6954d28_7.png)

对数据完成基本的清洗和转换，为建模做准备。

第一步把数据当中我需要的列给它拿出来。之前我们是不是说要做一个指标叫做**对数回报率**。在汇率当中，我们之前说了，为了使得我这个结果有一个累加性，所以我们之前做了一个对数转换。接下来我们还是用这种方法，先对我们的数据做一个基本的预处理操作。

![](img/7ae9a6ca694aa1e90f557248d6954d28_9.png)

```python
# 计算对数回报率
eur_usd_data['log_returns'] = np.log(eur_usd_data['EUR/USD'] / eur_usd_data['EUR/USD'].shift(1))
```

然后处理缺失值，这跟之前我们的操作方法都是一模一样的。

```python
# 去除缺失值
eur_usd_data = eur_usd_data.dropna()
```

![](img/7ae9a6ca694aa1e90f557248d6954d28_11.png)

然后这里，我在数据当中加了一个新的列。这个回报率来说，有的时候它是一个负的，有的时候是一个正的。其实这样我对于它的一个正多少、负多少，我都不管了，我只关心它的一个方向。你只需要告诉我它是一个涨还是一个跌就得了。所以我这里做了非常简单的一件事：指定一个`direction`，相当于我的一个走势，它是往上走还是往下走。

这个指标怎么做出来呢？非常简单。在咱们当前的`returns`数据当中，我直接取了一个正负号就行了。看它是一个负的，如果是个负的，我就用-1；正的，我就用个正1数就完事了，再给它`astype`一下，这是一个整形，是不是就得了。

```python
# 创建方向标签：上涨为1，下跌为-1
eur_usd_data['direction'] = np.sign(eur_usd_data['log_returns']).astype(int)
```

这是我们当前在数据当中引入的一个新的列，表示的是我的一个走势，是一个往上还是一个往下的。

## 第四步：定义回归任务与目标变量

明确机器学习任务的具体设置，包括特征(X)和目标(Y)。

![](img/7ae9a6ca694aa1e90f557248d6954d28_19.png)

下面就是我们的重点了：怎么样去做这个回归分析。先来看一下，今天我们要做几个任务。

![](img/7ae9a6ca694aa1e90f557248d6954d28_21.png)

第一个任务，之前咱们说的要做这个回归。我们手里先有什么？我们要做的这个东西，它是一个机器学习任务：基于以前的一些指标去预测未来的一个值。我们管这个任务叫**有监督任务**。所谓有监督任务，就是说手里你得有X还得有Y。X是你的指标，Y是你的一个答案，咱们得都有，这叫做一个回归任务。

![](img/7ae9a6ca694aa1e90f557248d6954d28_23.png)

那在咱们当前的回归任务当中，你说我都需要做什么呢？是不是说我得把X找到，Y也找到。Y对于我们来说其实挺简单的。Y就是谁？我就想预测一下明天它是一个涨还是一个跌。你可以把这个`returns`当做Y可以吧。或者说你换一下，或者说我现在不想去预测它具体的一个涨跌的值，我只想预测一下它到底是涨还是跌，你把这个`direction`这个东西当做Y是不是也行。

一会儿相对来说我们可以做两条分支，做两个对比实验。一个呢我写个`Y1`，一个写`Y2`。就是第一次咱拿这个`returns`当做我的一个目标值，第二次呢我拿这个`Y2`当做我的一个目标值，是不是都行，反正都是预测一个结果。

```python
# 定义特征 (X) 和目标变量 (Y)
# 特征X：可以使用滞后期的数据，例如前1天到前5天的对数回报率
lags = 5
for lag in range(1, lags + 1):
    eur_usd_data[f'lag_{lag}'] = eur_usd_data['log_returns'].shift(lag)

# 目标变量Y1：预测具体的回报率数值
eur_usd_data['Y1'] = eur_usd_data['log_returns'].shift(-1)  # 预测下一天的回报率

![](img/7ae9a6ca694aa1e90f557248d6954d28_25.png)

# 目标变量Y2：预测涨跌方向
eur_usd_data['Y2'] = eur_usd_data['direction'].shift(-1)  # 预测下一天的方向

# 去除因创建滞后特征和未来目标而产生的缺失值
eur_usd_data = eur_usd_data.dropna()

# 分离特征和目标
feature_columns = [f'lag_{i}' for i in range(1, lags+1)]
X = eur_usd_data[feature_columns]
Y1 = eur_usd_data['Y1']
Y2 = eur_usd_data['Y2']
```
