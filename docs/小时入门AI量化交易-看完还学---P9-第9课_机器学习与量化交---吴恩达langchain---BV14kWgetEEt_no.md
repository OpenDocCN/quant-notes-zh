## 矩阵特征值与SVD：量化分析的数学基石

上一节我们回顾了课程安排与行业信息。本节中，我们来看看支撑现代量化风控模型的核心数学概念。

### 线性变换与矩阵
矩阵可以看作一个线性变换函数。一个 `m x n` 的矩阵 **A**，能将一个 `n` 维向量 **x** 映射为一个 `m` 维向量 **Ax**。这正是许多线性模型（如线性回归、分类器）参数化表示的基础。

### 特征值与特征向量的定义
对于一个 `n x n` 的方阵 **A**，特征值 **λ** 和特征向量 **x** 满足以下等式：
**Ax = λx**
其含义是：矩阵 **A** 作用在特征向量 **x** 上，仅改变其长度（缩放 **λ** 倍），而不改变其方向。求解特征值即解方程 **det(A - λI) = 0**，一个 `n` 阶方阵有 `n` 个特征值（可能为实数或复数）。

### 对称矩阵的特殊性质
在量化分析中，我们尤其关注**对称矩阵**（即 **A = Aᵀ**）。对称阵有两个优美性质：
1.  所有特征值都是**实数**。
2.  所有特征向量互相**正交**（垂直）。

对于对称阵 **A**，存在**谱分解**（也称特征分解）：
**A = QΛQᵀ**
其中，**Q** 是由正交特征向量组成的矩阵，**Λ** 是由特征值构成的对角阵。这意味着一个对称矩阵完全由其**特征值和特征向量**刻画。

### 奇异值分解：任意矩阵的利器
对于非对称的任意矩阵 **A**（`m x n`），我们有更通用的**奇异值分解**：
**A = UΣVᵀ**
其中：
*   **U** 是一个 `m x m` 的正交矩阵（左奇异向量）。
*   **Σ** 是一个 `m x n` 的对角矩阵，对角线元素为**奇异值**（均为非负实数）。
*   **V** 是一个 `n x n` 的正交矩阵（右奇异向量）。

SVD的推导核心在于考虑 **AᵀA** 和 **AAᵀ** 这两个对称矩阵的谱分解。SVD是许多降维技术（如主成分分析PCA）和信号处理的基础。

## 资产组合理论：从数学到实践

理解了特征值与SVD后，我们来看看如何将其应用于实际的资产配置问题。

### 问题定义
假设我们精选了 `N` 只股票，拥有它们历史 `M` 天的收益率数据。对于每一天 `t`，我们有一个 `N` 维收益率向量：
**r_t = [r_{1t}, r_{2t}, ..., r_{Nt}]ᵀ**
根据这 `M` 个样本，我们可以计算出：
1.  **平均收益率向量** **μ** （`N` 维）。
2.  **收益率协方差矩阵** **Σ** （`N x N` 维，对称半正定矩阵）。

我们的目标是决定一个投资权重向量 **w = [w₁, w₂, ..., w_N]ᵀ**，其中 `∑w_i = 1`，以构建投资组合。

### 最小方差组合与特征值
根据现代投资组合理论，该投资组合的**风险（方差）** 可表示为：
**Risk = wᵀΣw**
我们的目标是最小化风险。一个重要的结论是：**使风险最小化的权重向量 w，恰好是协方差矩阵 Σ 的最小特征值所对应的特征向量**（需归一化使权重和为1）。
> **核心公式**：`min Risk = min(wᵀΣw)` 的解 `w*` 正比于 `Σ` 的最小特征值对应的特征向量。

然而，最小特征值方向往往代表“市场整体”方向，与指数化投资无异。实践中，我们常选择**第二小或更靠后的特征值**对应的特征向量作为权重，以寻求超越市场的收益。

### 基于优化的权重求解
另一种直接思路是将问题形式化为带约束的优化问题：
```
minimize: wᵀΣw
subject to: ∑w_i = 1
```
这可以通过拉格朗日乘子法求解，得到解析解：
**w* = (Σ⁻¹1) / (1ᵀΣ⁻¹1)**
其中 **1** 是元素全为1的向量。这种方法称为**最小方差组合**。

## 高级技巧：基于随机矩阵理论的协方差矩阵滤波

上述方法直接使用历史样本协方差矩阵 **Σ**。然而，金融数据噪声很大，**Σ** 包含大量随机噪声。我们可以利用**随机矩阵理论**进行滤波。

### 随机矩阵理论简介
随机矩阵理论研究矩阵元素为随机变量（如独立同分布高斯变量）的矩阵性质。对于这样的随机矩阵，其特征值分布服从著名的**Marchenko-Pastur分布**。

### 滤波算法步骤
以下是利用RMT滤波改进协方差矩阵的流程：
1.  **计算样本协方差矩阵** **Σ**，并进行谱分解：**Σ = QΛQᵀ**。
2.  **识别并过滤噪声特征值**：将特征值 `λ_i` 与Marchenko-Pastur分布的典型区间（例如 `[λ_min, λ_max]`）比较。落在此区间内的特征值被视为主要由随机噪声产生。
3.  **构建滤波后矩阵**：将上一步中被判定为噪声的特征值置零（或收缩），得到新的特征值对角阵 **Λ_filtered**。然后用公式 **Σ_filtered = Q Λ_filtered Qᵀ** 重构滤波后的协方差矩阵。
4.  **计算最优权重**：将 **Σ_filtered** 代入前述的最小方差组合公式或特征值方法，求解新的权重向量 **w_filtered**。

### 效果
研究表明，使用滤波后的协方差矩阵 **Σ_filtered** 计算出的投资组合权重，其样本外表现通常优于使用原始 **Σ** 的结果，能更有效地降低风险。

## 量化研究论文导读与批判性阅读

掌握了核心方法后，我们来看看学术界和业界是如何应用这些思想的。高效阅读论文是量化研究员的关键技能。

### 阅读方法论
*   **抓大放小**：重点阅读 `Introduction`（问题定义）和 `Conclusion`（结论与框架），技术细节可速览。
*   **保持批判**：论文并非金科玉律，常存在假设缺陷、数据瑕疵或方法局限。要思考其“傻逼之处”和改进空间。
*   **关注实践**：留意作者如何处理数据、选择特征、评估模型，以及计算资源的使用。

以下是几篇代表性论文的要点分析：

**1. 高频交易与深度学习**
*   **核心**：使用5分钟级高频数据，特征维度极高（~900维），采用全连接神经网络。
*   **亮点**：展示了处理海量高频数据的流程。
*   **批判点**：在CPU集群上训练耗时过长，未利用GPU加速，工程效率有待提升。

**2. 传统特征与深度信念网络**
*   **核心**：使用简单的历史收益率（日度、月度）作为特征，结合深度信念网络进行分类预测。
*   **亮点**：对比了DBN与传统逻辑回归的性能。
*   **批判点**：特征工程较为简单，模型性能提升有限（错误率从50%降至46%），若特征更优，传统模型也可能有类似表现。

**3. 市场情绪与另类数据**
*   **核心**：利用Twitter和Google搜索中“牛市”、“熊市”等关键词的频率作为市场情绪指标，预测市场走势。
*   **结论**：社交媒体情绪对市场有预测力；Twitter数据比Google搜索更有效；该模式不适用于中国股市。
*   **价值**：提供了将另类数据（非价格数据）引入量化模型的范例。

**4. 量化研究员报告**
*   **特点**：篇幅短（约4页），结构清晰（问题->方法->实验->结论），类似内部研究报告。
*   **启示**：这是业界量化研究员日常工作的产出形式，注重可解释性和快速迭代。
