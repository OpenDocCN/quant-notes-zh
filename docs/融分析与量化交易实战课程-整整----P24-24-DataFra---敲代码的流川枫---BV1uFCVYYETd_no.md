## 概述
在本节课中，我们将学习如何从Pandas的DataFrame对象中获取数据。DataFrame是一个二维表格型数据结构，拥有行索引和列索引。我们将重点掌握使用`.loc`和`.iloc`属性进行索引和切片的方法，这是数据操作的核心技能。

## DataFrame取值的基本方式
上一节我们介绍了DataFrame的一些常用属性。本节中我们来看看如何从DataFrame表格对象中获取具体的值。

与Series类似，我们可以使用中括号`[]`来获取值。例如，想获取标签为`1.0`的值。如果按照标签获取，通常的做法是先传递列索引（选择一列），再选择行。这与二维数组（先选行，再选列）的顺序不同，它更类似于数据库表的查询逻辑（先`SELECT`列）。

```python
# 示例：通过中括号先列后行取值（不推荐）
df['列名']['行名']
```

然而，如果行索引是整数，这种方式可能会出现问题。因此，我们不建议连续使用两个中括号来取值。

## 推荐的取值方法：.loc与.iloc
我们推荐根据你是按标签还是按下标来取值，分别使用`.loc`或`.iloc`属性，并且避免连续使用两个中括号。

以下是更安全、更不容易出错的获取`1.0`这个值的方法：

```python
# 使用 .loc 通过标签访问
df.loc['行标签', '列标签']
```

这种写法首先通过`.loc`属性指定按标签访问。括号内的写法与NumPy类似，用逗号隔开，左边是行，右边是列。此时，顺序变回了先行后列。

![](img/a3cdc93c4803682e0853817afb01b4b5_1.png)

如果你想查看某一整行的数据，不应该直接使用`df['行标签']`，因为DataFrame的实现方式是将多个Series拼接在一起，这样访问可能出错。

![](img/a3cdc93c4803682e0853817afb01b4b5_3.png)

![](img/a3cdc93c4803682e0853817afb01b4b5_5.png)

以下是查看一行数据的正确方法：

```python
# 查看‘A’这一行的所有数据
df.loc['A', :]  # 冒号‘:’表示选择所有列
```

![](img/a3cdc93c4803682e0853817afb01b4b5_7.png)

`.iloc`属性则是按照整数位置（下标）进行访问。

![](img/a3cdc93c4803682e0853817afb01b4b5_8.png)

```python
# 使用 .iloc 通过位置访问
df.iloc[行位置, 列位置]
```

## 索引与切片的灵活应用
DataFrame是一个二维数据类型，拥有行索引和列索引。我们可以使用标签和位置两种方法进行切片和索引。

![](img/a3cdc93c4803682e0853817afb01b4b5_10.png)

`.loc`和`.iloc`属性的使用方法如上所示，用逗号隔开行索引和列索引。在行索引和列索引的部分，可以灵活搭配多种索引方式：

![](img/a3cdc93c4803682e0853817afb01b4b5_12.png)

以下是可用的索引方式组合：
*   **一般下标/标签切片**：例如 `df.loc[‘A’:‘C’, ‘one’]`
*   **布尔值索引**：例如 `df.loc[df[‘one’] > 0, :]`
*   **花式索引**：传递一个列表，选择列表中所有索引对应的值。

这些方式可以任意混搭，实现灵活的数据选取。

![](img/a3cdc93c4803682e0853817afb01b4b5_14.png)

例如：
```python
# 混搭示例：选择行索引为‘A’和‘C’的两行，以及列索引为‘two’的列
df.loc[['A', 'C'], 'two']
```

![](img/a3cdc93c4803682e0853817afb01b4b5_16.png)

## 总结
本节课我们一起学习了DataFrame的索引与切片操作。核心要点是区分并使用`.loc`（基于标签）和`.iloc`（基于整数位置）属性来安全、准确地获取数据。我们了解了如何通过逗号分隔行和列，并可以灵活地将切片、布尔索引和花式索引组合使用，从而高效地从DataFrame中提取所需的数据子集。
