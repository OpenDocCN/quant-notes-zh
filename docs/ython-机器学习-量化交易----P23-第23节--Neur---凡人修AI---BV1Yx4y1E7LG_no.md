# 机器学习：23：神经网络入门指南 🧠

在本节课中，我们将学习神经网络的基础知识。神经网络是一种强大的机器学习模型，它通过模拟人脑神经元的工作方式来处理复杂的数据模式。我们将从基本概念入手，逐步了解其结构、工作原理以及常见的激活函数。

## 神经网络的基本概念

神经网络是一种模拟人脑神经信息传递的学习技术。人脑中的神经网络由成千上万的神经元组成，而人工神经网络也遵循类似的结构。

神经网络主要分为三个层面：输入层、隐藏层和输出层。输入层对应我们感官接收的信息，如看到的图像或听到的声音。这些信息被量化后传递到隐藏层。隐藏层可以看作是神经递质，负责处理和转换信息。最后，信息到达输出层，产生特定的输出结果，类似于大脑控制身体做出反应。

## 神经网络的结构

一个神经网络的结构图通常由箭头表示数据的流向。例如，从输入层指向隐藏层的箭头表示数据从输入流向隐藏层。大多数神经网络使用单向箭头，但循环神经网络（RNN）可以包含循环箭头。

每个连接线上都有一个数字，代表权重。权重越接近1，表示该连接对信息传递的影响越大。例如，如果从输入层1到隐藏层1的权重是0.9，而其他连接的权重是0.05，那么输入层1对隐藏层1的影响就更为显著。

## 神经元的信息传递模拟

让我们来看一个神经元信息传递的简单模拟。假设有三个输入，我们有两层隐藏层。第一层对权重进行处理，通过加权求和后，再经过一个非线性函数（称为激活函数）进行过滤或处理，最终产生输出。

激活函数直接影响加权求和值到输出之间的映射关系。例如，从输入X1到XN，权重从W1到WN，经过加权平均后得到值B，再通过激活函数（如Sigmoid）处理，输出A。如果有多层隐藏层，A将作为下一层的输入，重复此过程直到输出层。

**Sigmoid函数**
Sigmoid函数的公式为：`σ(z) = 1 / (1 + e^{-z})`。它的输出值在0到1之间，形状类似于概率的累积分布函数。因此，当输出层需要表示概率时（如逻辑回归），Sigmoid函数非常合适。然而，由于其值域较窄，在中间层中较少使用。

**Tanh函数**
Tanh函数的公式为：`tanh(z) = (e^{z} - e^{-z}) / (e^{z} + e^{-z})`。它的输出值在-1到1之间，比Sigmoid函数能表示更宽泛的值域。但它的两端同样较为平坦，在某些分布均匀的情况下可能存在较大误差。

**ReLU函数**
ReLU函数的公式为：`ReLU(z) = max(0, z)`。它在某个阈值（通常为0）以下输出0，以上则为线性函数。这种特性使其适用于门限模型，即当输入落入特定区间时，神经元“死亡”（不响应）。ReLU函数能够处理较强的刺激，并给出相应的信息。

**Leaky ReLU函数**
Leaky ReLU函数的公式为：`LeakyReLU(z) = max(αz, z)`，其中α是一个小的正数（如0.01）。与ReLU不同，它在负区间也有微弱的输出，因此神经元不会完全“死亡”，权重仍可更新。这使得它比ReLU更为保守，适用于不希望直接杀死神经元的场景。

## 神经网络的训练与结构

神经网络的训练过程复杂，涉及权重更新和损失函数优化。对于单层神经网络，输出通常是概率，因此常用Sigmoid函数。对于两层或多层神经网络，中间层常使用Leaky ReLU或ReLU函数，以避免神经元死亡并保持权重更新。

神经网络虽然能处理复杂模型，但也存在过拟合和方差大的问题。解决方法包括使用正则化技术（如L1/L2正则化）、简化模型结构或利用GPU加速计算。在深度学习中，还会使用如dropout和数据增强等技术来防止过拟合。

## 前馈神经网络与循环神经网络

![](img/3547ad5cd47f45f9cbcda5c857d0793a_11.png)

**前馈神经网络（Feedforward Neural Network）**
前馈神经网络的数据流向是单向的，从输入层经过隐藏层到输出层。例如，计算Z1点的值：`Z1 = 0.1*X1 + 0.2*X2 + 0.01`，然后通过Sigmoid激活函数得到A1。如果有多个隐藏层，此过程会重复直到输出层。

**循环神经网络（Recurrent Neural Network, RNN）**
循环神经网络引入了循环连接，允许隐藏层之间存在信息传递。这使得RNN能够处理时间序列数据或具有序列相关性的数据，如自然语言处理中的句子生成。在RNN中，当前隐藏状态`S_t`依赖于前一个状态`S_{t-1}`和当前输入`X_t`，公式为：`S_t = f(U*X_t + W*S_{t-1})`，其中U和W是权重矩阵，f是激活函数。

## 循环神经网络的应用

![](img/3547ad5cd47f45f9cbcda5c857d0793a_15.png)

循环神经网络在多个领域有重要应用：

*   **语言模型与文本生成**：通过分析单词序列的依赖关系，预测下一个可能出现的单词。
*   **图像描述**：识别图像中的物体和场景，并生成描述性文本。例如，识别照片中的“狗”、“石头”、“爪子”和“球”等元素。
