上一节我们介绍了布尔型索引，本节中我们来看看另一种强大的索引方式——花式索引。

## 什么是花式索引？

花式索引是指通过传递一个整数数组（或列表）来一次性选取多个元素。它适用于需要选取一组没有规律的位置元素的情况。

例如，对于一个一维数组，我们想选取第1、3、4、6、7号元素。这些索引位置没有规律，无法用简单的切片完成。这时就可以使用花式索引。

```python
import numpy as np
A = np.arange(20)  # 创建一个0到19的数组
indices = [1, 3, 4, 6, 7]  # 定义要选取的索引
result = A[indices]  # 使用花式索引
print(result)  # 输出：[1 3 4 6 7]
```

**核心概念**：`array[[index1, index2, ...]]`。将需要选取的索引放在一个列表或数组中，作为索引参数传入。

## 二维数组的花式索引

![](img/edb95a06f772edd44524e53408ab4bf6_7.png)

对于二维数组，花式索引的规则同样适用。我们可以通过逗号分隔行和列的索引方式，灵活地选取数据。

NumPy提供了四种主要的索引方式：普通索引、切片、布尔型索引和花式索引。在二维数组中，逗号左边控制行，右边控制列，两边可以自由组合使用这四种方式。

以下是几种组合示例：

*   **普通索引 + 切片**：选取指定行，并对列进行切片。
    ```python
    A = np.arange(20).reshape(4, 5)  # 创建一个4行5列的二维数组
    # 选取第0行，第2到第4列（不包含第4列）
    print(A[0, 2:4])
    ```

*   **普通索引 + 布尔型索引**：选取指定行，并根据条件筛选列。
    ```python
    # 选取第0行，并筛选出该行中大于2的元素
    print(A[0, A[0] > 2])
    ```

*   **花式索引 + 切片**：选取指定的多行，并选取所有列。
    ```python
    # 选取第1行和第3行，所有列
    print(A[[1, 3], :])
    ```

## 一个重要限制：避免两边同时使用花式索引

需要注意的是，**不能在逗号两边同时使用花式索引**来直接选取一个矩形区域。这种写法会产生与直觉不同的结果。

例如，我们想选取第1行&第3行 与 第1列&第3列 交叉位置的四个元素（即位置(1,1), (1,3), (3,1), (3,3)）。

**错误写法**：
```python
# 这并不会得到四个元素组成的2x2数组
print(A[[1, 3], [1, 3]])
```
这段代码的输出是 `[6, 18]`。它被解释为选取位置(1,1)和(3,3)这两个单独的元素，而不是一个矩形区域。

## 如何正确选取矩形区域？

如果需要选取一个由特定行和列交叉形成的矩形子数组，需要分两步进行。

以下是正确的操作步骤：
1.  首先，使用花式索引选取目标行，并选取所有列（使用切片 `:`）。
2.  然后，在上一步的结果上，再使用花式索引选取目标列。

![](img/edb95a06f772edd44524e53408ab4bf6_9.png)

![](img/edb95a06f772edd44524e53408ab4bf6_11.png)

```python
# 第一步：选取第1行和第3行，所有列
temp_result = A[[1, 3], :]
# 第二步：在临时结果中，选取所有行，第1列和第3列
final_result = temp_result[:, [1, 3]]
print(final_result)
```
这段代码会正确输出一个2行2列的数组，包含我们想要的四个元素。
