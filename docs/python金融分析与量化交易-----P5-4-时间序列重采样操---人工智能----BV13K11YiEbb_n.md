## 重采样操作 🔄

![](img/223d34186407238db95c48edd27b6902_7.png)

![](img/223d34186407238db95c48edd27b6902_9.png)

上一节我们介绍了时间序列的基础概念，本节中我们来看看如何改变数据的时间频率。

![](img/223d34186407238db95c48edd27b6902_11.png)

重采样是指将数据从一个时间频率转换到另一个时间频率的过程。例如，原始数据是按天统计的，我们可以将其转换为按周、按月或按年统计。这通过`resample`方法实现。

**核心公式/代码**：
```python
# 假设df是一个按天索引的DataFrame
# 按周重采样，并计算每周的均值
weekly_data = df.resample('W').mean()
```

![](img/223d34186407238db95c48edd27b6902_13.png)

![](img/223d34186407238db95c48edd27b6902_15.png)

以下是重采样的具体步骤和注意事项：

![](img/223d34186407238db95c48edd27b6902_17.png)

1.  **指定重采样频率**：调用`resample()`方法时，必须指定目标频率，如`'W'`（周）、`'M'`（月）、`'Y'`（年）。
2.  **指定聚合方法**：重采样后，需要指定如何聚合窗口内的数据，例如`.mean()`（均值）、`.first()`（第一个值）、`.median()`（中位数）等。
3.  **处理标签位置**：重采样后，每个时间窗口的标签（索引）默认使用窗口的右边界（结束日期）。可以通过`label`参数进行调整。
    *   `label='right'`（默认）：使用窗口结束日期作为标签。
    *   `label='left'`：使用窗口开始日期作为标签。

![](img/223d34186407238db95c48edd27b6902_19.png)

![](img/223d34186407238db95c48edd27b6902_21.png)

![](img/223d34186407238db95c48edd27b6902_23.png)

例如，按月重采样时，`label='right'`会以每月的最后一天（如1月31日）作为该月数据的标签；而`label='left'`则会以每月的第一天（如1月1日）作为标签。在金融分析中，通常使用`label='right'`，因为它更符合“截至某日”的观察逻辑。

## 时间窗口（滑动窗口）操作 🪟

![](img/223d34186407238db95c48edd27b6902_25.png)

![](img/223d34186407238db95c48edd27b6902_27.png)

了解了如何改变数据频率后，我们来看看如何在固定长度的窗口内进行滑动分析。

时间窗口操作，也称为滑动窗口操作，它不是在改变数据频率，而是在原始时间序列上，以一个固定长度的窗口（如10天）进行滑动，并对每个窗口内的数据独立进行计算。这常用于计算移动平均线、滚动波动率等指标。

![](img/223d34186407238db95c48edd27b6902_29.png)

**核心概念**：假设有100天的数据，窗口长度为10天。
*   第一个窗口：第1天到第10天。
*   第二个窗口：第2天到第11天（滑动1个单位）。
*   第三个窗口：第3天到第12天。
*   以此类推...

![](img/223d34186407238db95c48edd27b6902_31.png)

**核心代码**：
```python
# 计算10天窗口的滚动最小值
rolling_min = df['column_name'].rolling(window=10).min()
```

![](img/223d34186407238db95c48edd27b6902_33.png)

以下是执行时间窗口操作的关键步骤：

![](img/223d34186407238db95c48edd27b6902_35.png)

![](img/223d34186407238db95c48edd27b6902_37.png)

1.  **数据预处理**：**这是至关重要的一步**。如果原始数据存在缺失值（NaN），在窗口计算中会导致结果出现大量NaN。因此，需要先使用`df.dropna()`等方法清理数据。
2.  **创建滚动对象**：使用`.rolling(window=N)`方法创建一个滚动窗口对象，其中`N`是窗口大小。
3.  **应用统计函数**：对滚动窗口对象应用所需的统计函数，如`.min()`（最小值）、`.max()`（最大值）、`.mean()`（均值）、`.std()`（标准差）等。
4.  **整合结果**：可以将多个滚动统计结果作为新列添加到原始DataFrame中，便于综合分析。

![](img/223d34186407238db95c48edd27b6902_39.png)

例如，可以同时计算一个时间序列的滚动最小值、最大值、均值和标准差，从而全面了解该序列在近期窗口内的表现特征。

本节课中我们一起学习了时间序列分析的两个强大工具。

![](img/223d34186407238db95c48edd27b6902_49.png)

![](img/223d34186407238db95c48edd27b6902_51.png)

*   **重采样**：用于改变数据的时间频率（如日→周），通过`df.resample(freq).agg_func()`实现，需注意`label`参数对结果标签的影响。
*   **时间窗口操作**：用于在滑动窗口内计算统计指标（如10日移动平均），通过`df['col'].rolling(window=N).agg_func()`实现。**务必牢记先进行数据清洗，处理缺失值**，否则计算结果将不可靠。
