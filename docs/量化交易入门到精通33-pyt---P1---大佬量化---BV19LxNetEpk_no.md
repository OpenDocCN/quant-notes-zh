# 量化交易入门到精通：33：Python基本数据类型bytes - P1

在本节课中，我们将要学习Python中的二进制序列类型——`bytes`。这是一种不可变的二进制数据序列，常用于处理网络传输、文件读写（如图片、音频）以及金融交易中的底层数据。

## 🧱 什么是bytes？

`bytes`是Python中表示二进制数据的基本数据类型。它由0到255之间的整数构成，是一个不可变的序列。这意味着一旦创建，其内容就不能被修改。

上一节我们介绍了`bytes`的基本概念，本节中我们来看看如何创建和查看`bytes`对象。

## ✍️ 创建与查看bytes

我们可以通过在字符串前加上前缀`b`来创建一个`bytes`对象。

```python
x = b'hello'
print(x)
```

执行上述代码，输出结果为：`b'hello'`。这表明`x`是一个`bytes`对象。

为了查看其底层的整数值，我们可以通过索引来访问其中的元素。

```python
print(x[0])
```

执行这行代码，输出结果为：`104`。这个数字是字符`'h'`对应的ASCII码值。

## 🔢 理解ASCII编码

`bytes`对象中存储的实际上是字符对应的ASCII码（或其他编码）的整数值。例如，小写字母`'a'`对应的ASCII码是97。

以下是`'a'`到`'h'`的ASCII码对应关系示例：

```python
my_bytes = b'abcdefgh'
print(my_bytes[0])  # 输出: 97，对应 'a'
print(my_bytes[1])  # 输出: 98，对应 'b'
print(my_bytes[2])  # 输出: 99，对应 'c'
print(my_bytes[3])  # 输出: 100，对应 'd'
print(my_bytes[4])  # 输出: 101，对应 'e'
print(my_bytes[5])  # 输出: 102，对应 'f'
print(my_bytes[6])  # 输出: 103，对应 'g'
print(my_bytes[7])  # 输出: 104，对应 'h'
```

从以上示例可以看出，`bytes`对象`b'hello'`的第一个元素`x[0]`是104，这正是字母`'h'`的ASCII码。

## 💡 核心要点总结

本节课中我们一起学习了Python的`bytes`类型：

1.  **定义**：`bytes`是一种**不可变的二进制序列**，用于处理原始的字节数据。
2.  **创建**：通过在字符串字面量前加`b`前缀来创建，例如 `b'hello'`。
3.  **内容**：`bytes`对象中的每个元素都是一个**0到255之间的整数**，代表一个字节的值。
4.  **编码**：这些整数通常对应字符的**ASCII码**（或其他字符编码）。例如，`104` 对应小写字母 `'h'`。

`bytes`类型是处理网络数据、文件I/O和许多底层协议的基础，在量化交易中解析原始市场数据时尤为重要。
