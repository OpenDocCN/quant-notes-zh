以下是实现因子打分与排序的核心步骤：

1.  **遍历因子**：分别对两个数据框中的每一个因子列进行操作。
2.  **排序**：对每个因子列下的股票数据进行排序。
3.  **打分**：根据排序结果，为每只股票分配分数（例如，排名第一得最高分）。
4.  **合并**：将两个数据框的得分结果合并，为后续计算总分做准备。

### 实现因子打分

首先，我们需要导入必要的工具包。

```python
import numpy as np
import pandas as pd
```

接下来，我们开始编写打分逻辑。我们将分别处理“越高越好”和“越低越好”的因子。

![](img/13c50261068c86c64eac06f1196c549f_5.png)

#### 对“越高越好”的因子进行打分

对于这类因子，因子值越大，我们认为其质量越好，因此排名最高的股票应获得最高分。

```python
# 假设 df_up 是包含“越高越好”因子的数据框，形状为 300行 x N列（N个因子）
for factor in df_up.columns.tolist():  # 遍历每一个因子列名
    # 1. 按当前因子值进行降序排序（值越大排名越靠前）
    df_up.sort_values(by=factor, ascending=False, inplace=True)
    # 2. 生成分数：第一名得最高分（300分），第二名得299分，依此类推
    # 使用 np.linspace 生成一个从 300 到 1 的等差数列，长度为股票总数
    scores = np.linspace(300, 1, num=len(df_up))
    # 3. 将生成的分数赋值给当前因子列，替换原来的因子值
    df_up[factor] = scores
```

![](img/13c50261068c86c64eac06f1196c549f_7.png)

**代码解释**：
*   `df_up.columns.tolist()` 获取数据框的所有列名（即因子名称）并转换为列表，便于遍历。
*   `df_up.sort_values(by=factor, ascending=False, inplace=True)` 对数据框按指定因子列进行降序排序。`inplace=True` 参数使得排序结果直接修改原数据框，无需重新赋值。
*   `np.linspace(start, stop, num)` 用于生成等差数列。这里 `start=300`, `stop=1`, `num=股票数量(300)`，生成 `[300, 299, ..., 2, 1]` 的数组，正好对应排序后的股票得分。

![](img/13c50261068c86c64eac06f1196c549f_9.png)

![](img/13c50261068c86c64eac06f1196c549f_11.png)

#### 对“越低越好”的因子进行打分

![](img/13c50261068c86c64eac06f1196c549f_13.png)

对于这类因子，因子值越小越好，因此排名第一（值最小）的股票应获得最高分。

![](img/13c50261068c86c64eac06f1196c549f_15.png)

```python
# 假设 df_down 是包含“越低越好”因子的数据框
for factor in df_down.columns.tolist():
    # 1. 按当前因子值进行升序排序（值越小排名越靠前）
    df_down.sort_values(by=factor, ascending=True, inplace=True)
    # 2. 生成分数：第一名（值最小）得最高分（300分）
    # 注意：分数序列依然是 [300, 299, ..., 1]，因为排序顺序已经保证了最小值在第一行
    scores = np.linspace(300, 1, num=len(df_down))
    # 3. 将分数赋值给当前因子列
    df_down[factor] = scores
```

![](img/13c50261068c86c64eac06f1196c549f_17.png)

**代码解释**：
*   逻辑与“越高越好”因子类似，关键区别在于 `ascending=True`，即按因子值升序排列，确保因子值最小的股票排在最前面。
*   打分数组 `scores` 的生成方式完全相同，因为排序操作已经调整了股票的物理顺序。排在第一行的股票（此时是因子值最小的）将获得 `scores` 数组的第一个值，即300分。

### 关于 `np.linspace` 的补充说明

![](img/13c50261068c86c64eac06f1196c549f_23.png)

为了更清晰地理解打分过程，我们单独看一下 `np.linspace` 的用法。

![](img/13c50261068c86c64eac06f1196c549f_25.png)

```python
# 生成从1到300，共300个数的等差数列
print(np.linspace(1, 300, num=300))
# 输出：[  1.   2.   3. ... 298. 299. 300.]

![](img/13c50261068c86c64eac06f1196c549f_27.png)

# 生成从300到1，共300个数的等差数列
print(np.linspace(300, 1, num=300))
# 输出：[300. 299. 298. ...   3.   2.   1.]
```

在代码中，我们通过 `np.linspace(300, 1, num)` 生成了一个从高到低的分数序列。无论股票是按升序还是降序排列，排在第一位的股票总是拿到这个序列的第一个值（300分），从而实现了“排名越高，得分越高”的逻辑。

![](img/13c50261068c86c64eac06f1196c549f_29.png)

### 合并打分结果

完成两个数据框的因子打分后，我们得到了两个新的数据框（`df_up`, `df_down`），其中的原始因子值已被替换为得分。

为了计算每只股票在所有因子上的综合得分，我们需要将这两个数据框合并。

```python
# 沿列方向合并两个数据框
# 假设原始 df_up 有 A, B, C 三个因子，df_down 有 D, E 两个因子
# 合并后，df_combined 将有 A, B, C, D, E 五列因子得分
df_combined = pd.concat([df_up, df_down], axis=1)

# 此时，df_combined 的每一行代表一只股票，每一列代表该股票在某个因子上的得分
# 接下来就可以对每一行求和，得到该股票的总分
df_combined['总分'] = df_combined.sum(axis=1)
```

![](img/13c50261068c86c64eac06f1196c549f_33.png)

**代码解释**：
*   `pd.concat([df1, df2], axis=1)` 将两个数据框按列拼接起来。`axis=1` 表示按列操作。
*   `df_combined.sum(axis=1)` 对每只股票（每一行）的所有因子得分进行求和，`axis=1` 表示按行求和，结果即为该股票的综合得分。

## 总结
本节课中，我们一起学习了量化交易中因子打分与排序的核心操作。

1.  **排序**：根据因子对股票收益的影响方向（正向或负向），使用 `DataFrame.sort_values()` 方法对股票进行升序或降序排列。
2.  **打分**：利用 `np.linspace()` 函数生成与股票数量相等的分数序列，并根据排序结果将分数赋予每只股票，实现了排名到分数的映射。
3.  **合并**：最后，使用 `pd.concat()` 将不同组的因子得分合并到一个数据框中，并通过行求和计算出每只股票的初步综合得分。

这一步的结果为我们后续的因子加权、组合优化以及最终的选股清单构建奠定了坚实的基础。在下一节，我们将探讨如何为不同的因子分配合适的权重。
