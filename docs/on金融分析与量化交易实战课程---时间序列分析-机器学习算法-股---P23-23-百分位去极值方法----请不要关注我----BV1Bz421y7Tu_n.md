# 量化交易实战课程：P23：因子数据预处理之百分位去极值方法

![](img/c0c7274356462b12db49c29833ea6f99_0.png)

在本节课中，我们将学习如何对金融量化分析中的因子数据进行预处理。预处理是构建有效量化模型的关键步骤，能帮助我们处理数据中的异常值、量纲差异等问题，使后续的分析和建模更加稳健。

## 什么是因子？

在开始预处理之前，我们首先需要理解什么是“因子”。在股票投资中，我们通常不会随机选择股票，而是依据某些标准或想法进行筛选。这些用于筛选股票的标准或指标，就是“因子”。

例如，一个常见的投资常识是：市净率较低的股票可能更具投资价值，因为其未来上涨空间相对较大。再比如，一个公司的营收增长率高，可能意味着这是一家好公司，其股票值得投资。简单来说，**因子就是那些可能对你最终投资结果产生影响的指标**。

我们可以将这个问题抽象为一个数据挖掘问题：最终的投资收益是目标变量 `Y`，而各种因子（如市净率、营收增长率等）是输入变量 `X1, X2, X3...`。我们的目标是分析每个因子对结果的影响，并从中选择有效的因子来构建投资策略。

![](img/c0c7274356462b12db49c29833ea6f99_2.png)

## 因子数据预处理三步走

拿到因子数据后，我们不能直接用于建模，通常需要进行预处理。以下是三个核心的预处理步骤：

1.  **去极值**：处理数据中的异常值或离群点。
2.  **标准化**：消除不同因子之间由于量纲和取值范围不同带来的影响。
3.  **中性化**：消除因子数据中与某些特定风格（如行业、市值）相关的系统性偏差。

![](img/c0c7274356462b12db49c29833ea6f99_4.png)

前两步在机器学习和数据挖掘中很常见，而第三步“中性化”在因子投资策略中尤为重要。接下来，我们将按照这个顺序，详细讲解每一步的具体做法。本节课，我们先聚焦于第一步：去极值。

![](img/c0c7274356462b12db49c29833ea6f99_6.png)

## 去极值方法：百分位法

![](img/c0c7274356462b12db49c29833ea6f99_8.png)

![](img/c0c7274356462b12db49c29833ea6f99_10.png)

上一节我们介绍了因子和预处理的基本概念，本节中我们来看看如何处理数据中的极值。极值，或称离群点，是那些明显偏离数据主体分布的数值。直接删除这些数据点可能造成信息损失，因此我们通常采用“调整”而非“删除”的策略。核心思想是：为数据设定合理的上下边界，将超出边界的极值“拉回”到边界上。

![](img/c0c7274356462b12db49c29833ea6f99_12.png)

以下是几种常见的去极值方法，我们先介绍最直观的**百分位法**。

![](img/c0c7274356462b12db49c29833ea6f99_14.png)

![](img/c0c7274356462b12db49c29833ea6f99_16.png)

### 理解分位数

要理解百分位法，首先需要了解分位数的概念。分位数是将数据按大小排序后，分成等份的临界点值。

![](img/c0c7274356462b12db49c29833ea6f99_18.png)

*   **中位数**：最熟悉的分位数，代表数据排序后处于50%位置的值。它将数据分为上下两半。与均值相比，中位数对极值不敏感，更能代表数据的“一般水平”。
*   **四分位数**：将数据四等分。
    *   **第一四分位数**：处于25%位置的值。
    *   **第二四分位数**：即中位数，处于50%位置的值。
    *   **第三四分位数**：处于75%位置的值。

### 百分位去极值操作

百分位去极值法的原理很简单：我们通过分位数来确定数据的正常范围上下界。例如，我们可以将低于1%分位数和高于99%分位数的值视为极值。

![](img/c0c7274356462b12db49c29833ea6f99_20.png)

具体操作步骤如下：
1.  计算因子数据序列的指定下分位数（如1%）和上分位数（如99%）。
2.  将小于下分位数的所有值，都替换为下分位数的值。
3.  将大于上分位数的所有值，都替换为上分位数的值。

这样，所有数据都被限制在由1%和99%分位数定义的区间内，既处理了极端值，又保留了数据的分布形态。

**代码描述**：
```python
import numpy as np

def winsorize_percentile(data, lower_percentile=0.01, upper_percentile=0.99):
    """
    使用百分位法对数据进行去极值处理。
    
    参数:
    data: 输入的数据序列（如一个因子的值）。
    lower_percentile: 下分位数，默认0.01。
    upper_percentile: 上分位数，默认0.99。
    
    返回:
    去极值处理后的数据序列。
    """
    lower_bound = np.percentile(data, lower_percentile * 100)
    upper_bound = np.percentile(data, upper_percentile * 100)
    
    # 将小于下界的值设为下界，大于上界的值设为上界
    data_winsorized = np.clip(data, lower_bound, upper_bound)
    
    return data_winsorized
```
