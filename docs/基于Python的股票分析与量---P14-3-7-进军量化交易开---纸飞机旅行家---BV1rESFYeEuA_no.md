在本节课中，我们将要学习量化交易中至关重要的一环——择时。择时决定了交易的成败与盈利多少，是选股之后的关键步骤。



## 择时的定义与作用

上一节我们介绍了选股，本节中我们来看看择时。择时是指选择股票买入和卖出的时机。选股决定了能否赚钱，而择时的好坏直接决定了能赚多少钱。择时效果不佳可能导致一无所获，而择时精准则可能带来丰厚回报。择时通常依赖于技术分析这一工具。



![](img/157df5a00b6def055a169189eed2222d_3.png)

## 什么是技术分析

技术分析是通过研究市场行为数据（主要是价格和成交量）来预测未来价格走势的方法。其核心是认为市场行为会反映一切信息，并且历史会重演。技术分析重点关注股价变化，并利用各种指标和工具来识别趋势和交易信号。

以下是技术分析最常用的六种工具：

1.  **K线形态**
2.  **成交量**
3.  **均线**
4.  **布林带**
5.  **MACD**
6.  **KDJ**

接下来，我们将逐一详细介绍这些工具。



## 常用技术分析工具详解

### 1. K线形态

K线，又称日本蜡烛图，是技术分析中最基础、最常见的工具。一根K线包含了开盘价、收盘价、最高价和最低价，其形态能直观反映多空（看涨与看跌）双方的力量对比。当看涨力量强时，股价倾向于上涨；反之则下跌。

K线形态有数十种，这里我们看两个简单示例：

*   **红三兵**：连续三根小阳线。这种形态通常被视为持续看涨的信号，可能意味着可以考虑买入。
    *   **量化思路**：在程序中，可以定义为连续三个交易日上涨，且涨幅不超过特定阈值（如5%），则触发买入信号。
*   **黑三兵**：连续三根小阴线。这种形态通常被视为持续看跌的信号，表明空方力量增强。



![](img/157df5a00b6def055a169189eed2222d_5.png)

### 2. 成交量

![](img/157df5a00b6def055a169189eed2222d_7.png)

成交量是指一定时间内股票成交的总数量。它不仅能反映买卖数量的变化，更能揭示多空双方力量的强弱转换，从而辅助预测股价走势。

以下是几个成交量分析的简单示例（仅为说明原理，不构成投资建议）：

![](img/157df5a00b6def055a169189eed2222d_9.png)

*   **价升量增**：股价从底部突破均线，同时成交量显著放大。这常被视为触底反弹、后市看涨的信号。
*   **价创新高，量维持高位**：股价创出新高，同时成交量保持在高位。这表明市场情绪依然乐观，股价可能继续上升。
*   **价跌量增**：股价从高位开始下跌，伴随巨大的卖出成交量。这常预示股价可能已到顶部，后市可能持续下跌。
*   **价跌量缩**：股价持续下跌，但成交量逐步萎缩。这表明抛售力量在减弱，持股者可能在等待触底反弹的时机。随后若出现放量上涨，则可能是反弹信号。



![](img/157df5a00b6def055a169189eed2222d_11.png)

### 3. 均线

均线（Moving Average， MA）是将一定周期内的收盘价之和除以该周期数，得到的一条价格平均线。它平滑了股价波动，能更清晰地显示趋势方向。常见的均线有5日（周线）、10日（半月线）、20日（月线）、30日、60日（季线）等。

![](img/157df5a00b6def055a169189eed2222d_13.png)

均线也能反映多空力量对比：

*   **看涨情况（均线支撑）**：当股价运行在多条均线（如5日、10日、20日）之上时，表明市场普遍看多，均线构成“支撑位”，股价处于上升趋势。
*   **看跌情况（均线压力）**：当股价运行在多条均线之下时，表明市场普遍看空，均线构成“压力位”，股价处于下降趋势，此时不宜买入。



![](img/157df5a00b6def055a169189eed2222d_15.png)

### 4. 布林带

![](img/157df5a00b6def055a169189eed2222d_17.png)

布林带（Bollinger Bands）是一种由三条轨道线组成的技术指标。中间线是股价的移动平均线（通常为20日均线），上轨和下轨分别是中间线加上和减去两倍的标准差。上轨被视为“压力线”，下轨被视为“支撑线”。

布林带主要用于判断股价的波动区间和趋势强度：

*   **上升趋势**：股价沿布林带上轨运行，布林带通道向上开口，表明处于单边上升趋势。
*   **下降趋势**：股价沿布林带下轨运行，布林带通道向下开口，表明处于单边下降趋势。
*   **震荡趋势**：股价在布林带上下轨之间来回波动，通道收窄，表明市场处于震荡整理阶段。



### 5. MACD

MACD（Moving Average Convergence Divergence，异同移动平均线）是反映股价变化**速率**（加速度）的指标。它由快线（DIF）、慢线（DEA）和柱状图（MACD Histogram）组成。

通过MACD可以判断趋势的强度和转折：

*   **MACD > 0**：表示短期趋势强于长期趋势，股价处于上升态势。
*   **MACD < 0**：表示短期趋势弱于长期趋势，股价处于下降态势。
*   **MACD 趋近于0**：表示趋势动能减弱，股价可能进入盘整。
*   **金叉与死叉**：快线（DIF）上穿慢线（DEA）形成“金叉”，是看涨信号；反之，下穿形成“死叉”，是看跌信号。
    *   **量化思路**：程序可以捕捉MACD由负转正（金叉）或由正转负（死叉）的时点，作为交易信号。



### 6. KDJ

KDJ指标，也称为随机指标，主要用于分析**短期**行情。它通过计算特定周期内的最高价、最低价及收盘价，来反映价格波动的真实波幅，并判断市场的超买和超卖状态。

KDJ由三条线组成：K线（快线）、D线（慢线）和J线（方向敏感线）。

*   **超买与超卖**：
    *   通常认为 **KDJ值 > 80** 时，市场处于“超买”状态，股价可能面临回调。
    *   通常认为 **KDJ值 < 20** 时，市场处于“超卖”状态，股价可能酝酿反弹。
*   **金叉与死叉**：K线上穿D线形成金叉，是短期买入信号；K线下穿D线形成死叉，是短期卖出信号。

KDJ指标在股价的震荡行情中尤为有效，能帮助捕捉短期的买卖点。



## 本节总结

本节课中我们一起学习了量化交易的核心环节——择时。

首先，我们明确了择时的定义，即选择股票买卖的时机，它直接决定了交易的盈利水平。

接着，我们介绍了择时的主要工具——技术分析，并详细讲解了六种最常用的技术分析工具：
1.  **K线形态**：通过蜡烛图形态判断多空力量对比。
2.  **成交量**：通过成交数量变化分析资金动向和多空强弱。
3.  **均线**：通过价格平均线识别趋势方向和支撑压力位。
4.  **布林带**：通过上、中、下三条轨道线判断股价波动区间和趋势。
5.  **MACD**：通过异同移动平均线反映股价变化速率和趋势转折。
6.  **KDJ**：通过随机指标分析短期超买超卖状态和买卖点。
