## 模型评估指标

![](img/4a2f2d25f36692b749be236a89df700f_33.png)

现在，我们需要量化地评估模型性能。我们主要关注两个方面的评估：

1.  **预测误差**：计算预测值与真实值之间的差异。
2.  **趋势预测准确率**：判断模型是否能正确预测股价的上升或下降趋势。

我们将数据划分为训练集和测试集。例如，使用2014年至2017年的数据作为训练集，2017年至2018年的数据作为测试集。

以下是评估过程的核心代码逻辑：

```python
# 划分训练集和测试集
train = data[‘2014-01-01‘:‘2017-01-23‘]
test = data[‘2017-01-24‘:‘2018-01-23‘]

# 建立并训练模型
model = Prophet()
model.fit(train)

# 预测未来365天（即测试集区间）
future = model.make_future_dataframe(periods=365)
forecast = model.predict(future)

# 提取预测结果
predictions = forecast[‘yhat‘].tail(len(test)).values
actuals = test[‘y‘].values

![](img/4a2f2d25f36692b749be236a89df700f_35.png)

![](img/4a2f2d25f36692b749be236a89df700f_37.png)

# 计算误差
train_error = np.mean(np.abs(model.history[‘y‘] - model.history[‘yhat‘]))
test_error = np.mean(np.abs(actuals - predictions))

![](img/4a2f2d25f36692b749be236a89df700f_39.png)

# 计算趋势预测准确率
# 1. 计算每日变化
pred_diff = np.diff(predictions)
actual_diff = np.diff(actuals)
# 2. 判断变化方向（上升为1，下降为0）
pred_trend = np.sign(pred_diff)
actual_trend = np.sign(actual_diff)
# 3. 计算准确率
trend_accuracy = np.mean(pred_trend == actual_trend)
```

在评估过程中，我们使用 `np.sign()` 函数来判断变化方向。该函数的作用是：输入大于0返回1，小于0返回-1，等于0返回0。



![](img/4a2f2d25f36692b749be236a89df700f_41.png)

![](img/4a2f2d25f36692b749be236a89df700f_35.png)



运行评估代码后，我们得到以下关键指标：

*   **训练集平均误差**：约18美元。
*   **测试集平均误差**：约160美元。这表明模型在未知数据上的表现显著下降。
*   **上升趋势预测准确率**：约56%。
*   **下降趋势预测准确率**：约43.81%。

这些准确率接近50%，意味着模型在判断涨跌趋势方面几乎等同于随机猜测，效果不理想。
