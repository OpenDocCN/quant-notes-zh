## MAD法原理介绍
MAD法，即中位数绝对偏差法，其核心思想是通过数据的中位数和偏差来设定合理的上下边界，从而识别并处理极端值。

以下是MAD法的计算步骤：

![](img/1a722867815781ae1d2061309280021d_6.png)

1.  **计算中位数**：首先，计算原始数据序列的中位数。
    *   公式：`median = 序列.quantile(0.5)`

![](img/1a722867815781ae1d2061309280021d_8.png)

2.  **计算绝对偏差**：然后，计算序列中每个数据点与中位数之差的绝对值，形成一个新的偏差序列。
    *   公式：`偏差序列 = abs(原始序列 - 中位数)`

![](img/1a722867815781ae1d2061309280021d_10.png)

![](img/1a722867815781ae1d2061309280021d_12.png)

3.  **计算MAD值**：接着，计算这个偏差序列的中位数，这个值就是MAD。
    *   公式：`MAD = 偏差序列.quantile(0.5)`

![](img/1a722867815781ae1d2061309280021d_14.png)

![](img/1a722867815781ae1d2061309280021d_16.png)

4.  **确定边界**：最后，使用中位数加上和减去N倍的MAD值，分别作为数据的上下边界。N通常取一个固定值（如1.4826）。
    *   上界公式：`上界 = 中位数 + N * MAD`
    *   下界公式：`下界 = 中位数 - N * MAD`

![](img/1a722867815781ae1d2061309280021d_18.png)

![](img/1a722867815781ae1d2061309280021d_20.png)

对于原始数据，任何超过上界或低于下界的值，都将被边界值所替代。

![](img/1a722867815781ae1d2061309280021d_22.png)

![](img/1a722867815781ae1d2061309280021d_24.png)

## 代码实现
理解了原理后，我们来看看如何用Python代码实现MAD去极值。

以下是MAD去极值函数的完整代码：

![](img/1a722867815781ae1d2061309280021d_32.png)

```python
def filter_extreme_MAD(series, n):
    """
    使用MAD法对序列进行去极值处理。
    :param series: 待处理的pandas Series
    :param n: 倍数，通常取1.4826
    :return: 处理后的Series
    """
    # 1. 计算中位数
    median = series.quantile(0.5)
    
    # 2. & 3. 计算MAD值
    # 计算每个数据与中位数的绝对偏差，再求该偏差序列的中位数
    mad = ((series - median).abs()).quantile(0.5)
    
    # 4. 确定上下边界
    high = median + n * mad
    low = median - n * mad
    
    # 5. 应用边界，替换极值
    # 将大于上界的值设为上界，小于下界的值设为下界
    return series.clip(lower=low, upper=high)
```
