## 分型的基本概念
在缠论中，分型是由相邻的三根K线构成的结构。这三根K线并非呈现连续的上升或下降趋势，而是预示着走势可能发生反转。分型主要分为底分型和顶分型两种。

![](img/ce87950b1960211d3f6021fac41e5ef4_3.png)

以下是两种分型的定义：
*   **顶分型**：中间K线的最高价是三者中最高的，同时其最低价也是三者中最高的。
*   **底分型**：中间K线的最低价是三者中最低的，同时其最高价也是三者中最低的。

在缠论体系中，底分型的出现通常被视为股价跌至低位、即将反弹的信号，是潜在的买入或建仓时机。相反，顶分型的出现则意味着股价可能即将见顶，是潜在的卖出信号。需要注意的是，单一的分型结构较为基础，不能完全作为买卖点的唯一判断依据。缠论中真正的买卖点是以分型为基础，通过更复杂的结构（如笔、线段）来构建的，这将在后续课程中详细介绍。

## 使用Python识别分型
上一节我们介绍了分型的定义，本节中我们来看看如何用程序自动化地识别它们。

既然我们明确了顶分型和底分型的定义，就可以利用处理好的股票K线数据，并借助Python代码来批量找出所有历史数据中的分型位置。

![](img/ce87950b1960211d3f6021fac41e5ef4_5.png)

以下是识别分型的核心逻辑代码示例：
```python
# 假设df是一个包含‘high’, ‘low’价格的DataFrame
def detect_fractal(df):
    # 识别顶分型：中间K线高点最高且低点最高
    top_condition = (df[‘high’].shift(1) > df[‘high’].shift(2)) & \
                    (df[‘high’].shift(1) > df[‘high’]) & \
                    (df[‘low’].shift(1) > df[‘low’].shift(2)) & \
                    (df[‘low’].shift(1) > df[‘low’])
    # 识别底分型：中间K线低点最低且高点最低
    bottom_condition = (df[‘low’].shift(1) < df[‘low’].shift(2)) & \
                       (df[‘low’].shift(1) < df[‘low’]) & \
                       (df[‘high’].shift(1) < df[‘high’].shift(2)) & \
                       (df[‘high’].shift(1) < df[‘high’])
    df[‘top_fractal’] = top_condition
    df[‘bottom_fractal’] = bottom_condition
    return df
```

运行程序后，我们可以将结果可视化。在生成的图表中，红色方块通常用于标记顶分型的位置，绿色三角则用于标记底分型的位置。通过这张图，我们可以直观地看到指定时间段内某只股票的所有分型位置，一目了然。这项自动化工作能为研习缠论的分析师节省大量手动识别的时间，使其能更专注于策略研究。
