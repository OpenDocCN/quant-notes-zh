# 金融量化分析：02：股票基本知识与分类 📈

在本节课中，我们将要学习股票的基础概念、核心作用以及常见的分类方式。理解这些知识是进行金融量化分析的基石。

## 股票的定义

股票是股份公司发给出资人的一种凭证。股票的持有者就是股份公司的股东。

我们可以通过一个例子来形象地理解。假设一位创业者需要资金，而投资者（如亿万富翁）愿意提供资金。如果只是简单的借贷，投资者只能获得固定利息。但投资者可以选择另一种方式：用资金换取创业公司的股票。股票代表了投资者对公司出资的证明，使其成为公司的所有者（股东）之一。

例如，一个公司初始由创业者本人和四位投资者各出资1亿元，公司总市值即为5亿元。那么每位出资人将获得公司20%的股票。这代表了他们对公司的所有权和相应的权益。

## 股票的作用

上一节我们介绍了股票是股东身份的凭证，本节中我们来看看股票具体有哪些作用。

股票的核心作用主要体现在两个方面：

以下是股票的两个主要作用：

1.  **证明股东身份与权利**：持有股票意味着你是公司的股东。作为出资人，你拥有参与公司重大决策（如在股东大会上投票）的权利。
2.  **获取收益**：这是大多数投资者关注的重点。通过持有股票获利主要有两种途径：
    *   **公司分红**：当公司盈利时，可能会将部分利润按持股比例分配给股东。例如，若公司年净利润为5000万，持有20%股份的股东可获得1000万的分红。
    *   **交易获利（资本利得）**：投资者可以在二级市场（如证券交易所）买卖股票。如果以较低价格买入，再以较高价格卖出，其中的差价就是利润。例如，初始投资1亿元获得20%股份，当公司市值从5亿增长到50亿时，这20%股份的价值就变成了10亿。通过出售部分股份，投资者可以实现“套现”。

对于资金量较小的普通股民而言，逻辑完全相同：通过购买股票成为微小的股东，同样可以享受分红和交易价差带来的收益，只是涉及的金额和比例不同。

## 什么是上市？

我们知道了股票的作用，那么公司的股票如何能被普通大众交易呢？这就涉及到“上市”的概念。

所谓上市，就是企业通过证券交易所，首次公开向全社会投资者发行股票，以募集资金的过程。

公司不能随意向公众募集资金。它需要达到一定体量，并向证监会提交申请，接受严格的审核（包括财务状况、经营风险等）。证监会评估通过后，公司才获准在证券交易所挂牌。上市后，公司的股票就能被所有符合条件的投资者公开买卖。

公司寻求上市的主要动机是融资。相对于向少数富人融资，上市可以让公司面向海量的普通投资者，汇聚社会闲散资金，融资潜力巨大。中国的A股市场就是这样一个由国家监管、旨在保护投资者的大市场。

公司首次公开发行股票的行为，称为 **IPO**。

## 股票的分类

了解了上市的概念后，我们来看看股票有哪些常见的分类方式。

![](img/86a7b24d5f2c6070a26cddea4cb04f2f_1.png)

### 按业绩分类

根据公司的经营业绩，股票通常被分为以下三类：

以下是三种常见的按业绩分类的股票：

1.  **蓝筹股**：指资本雄厚、信誉优良的大型公司股票。通常像“大胖子”，体量巨大，经营稳定。例如中国的石油、石化、银行等巨头公司。名称源于赌场中价值最高的蓝色筹码。
2.  **绩优股**：指业绩优良公司的股票。它们可能规模不及蓝筹股，但盈利能力突出，增长表现良好。例如贵州茅台，就是典型的绩优股。
3.  **ST股**：中文为“特别处理股票”。如果公司连续两年亏损，或每股净资产低于股票面值，其股票名称前会被加上“ST”标记。这是一种风险警示，提醒投资者该公司经营状况不佳。如果公司后续经营改善，可以申请撤销ST。

### 按上市地区分类

根据公司股票上市交易的地理位置和结算货币，可以进行如下分类：

以下是按上市地区分类的主要股票类型：

*   **A股**：在中国大陆（上海或深圳证券交易所）上市，以人民币认购和交易的股票。
*   **B股**：同样在中国大陆上市，但以外币（如美元、港币）认购和交易的股票。最初主要面向境外投资者，现已对境内投资者放宽。
*   **H股**：在中国香港上市的公司股票。
*   **N股**：在美国纽约上市的公司股票。
*   **S股**：在新加坡上市的公司股票。

### 不同市场的交易规则

不同的股票市场有其特定的交易规则，其中A股的规则对投资者影响显著。

以下是A股市场的两个关键交易规则：

1.  **涨跌幅限制**：A股设有每日涨跌幅限制，通常为**10%**。即一只股票当日的价格波动，不得超过前一个交易日收盘价的±10%。此规则旨在平抑市场剧烈波动，保护投资者，防止股价在短时间内暴涨暴跌。
2.  **T+1交割制度**：`T`代表交易当天。`T+1`意味着**当日买入的股票，最快要到下一个交易日才能卖出**。与之对应，港股、美股等多实行`T+0`制度，即当日可随意买卖。`T+1`制度是为了减少市场的过度投机行为。对于B股，资金提现还有`T+3`的交收日限制。

相比之下，港股（H股）、美股（N股）等市场通常没有涨跌幅限制，且实行`T+0`交易，灵活性更高，但波动风险也更大。
