在本节课中，我们将要学习数据预处理中的一个核心步骤——标准化。标准化是数据挖掘和机器学习中非常常用的技术，其目的是消除不同特征（指标）之间由于量纲和取值范围不同带来的影响，使数据更易于分析和建模。

![](img/ee38e828f37f574f76b4abb20f7528dc_4.png)

## 标准化的目的与原理

上一节我们介绍了数据预处理的重要性，本节中我们来看看标准化的具体作用。

标准化的核心目标是让不同特征的数据处于**相同的尺度**。想象一下，我们有两个特征X1和X2。X1的取值范围可能在0到100之间，而X2的取值范围在0到1之间。如果不做处理，在后续的算法计算中，取值范围大的特征（如X1）可能会主导模型的结果，这并不是我们希望看到的。标准化就是为了让所有特征都能被“平等”地看待。

标准化的数学公式如下：

**`z = (x - μ) / σ`**

其中：
*   **`x`** 是原始数据。
*   **`μ`** 是该特征所有数据的**均值**。
*   **`σ`** 是该特征所有数据的**标准差**。

这个公式可以分解为两个步骤：

1.  **去均值**：`x - μ`。这一步的目的是将数据的中心移动到零点。处理后的数据会以零为中心对称分布。
2.  **除标准差**：`(x - μ) / σ`。这一步的目的是调整数据的“离散程度”。通过除以标准差，可以将不同取值范围的数据“缩放”到大致相同的尺度上。

为了更直观地理解，可以想象两组人：
*   第一组：你和你的几位同学，大家的月薪都在1万到2万之间。这组数据的标准差较小。
*   第二组：你、马云、马化腾。这组数据的取值范围从1万到数千亿，其标准差非常大。

对于取值范围小、标准差也小的第一组数据，除以一个小的标准差，其数值范围变化不大。对于取值范围大、标准差也大的第二组数据，除以一个大的标准差，其巨大的数值范围会被显著缩小。最终，两组数据经过标准化后，其数值范围会变得相近。

## 标准化的代码实现

理解了原理后，我们来看看如何用Python代码实现标准化。

![](img/ee38e828f37f574f76b4abb20f7528dc_6.png)

以下是手动实现标准化处理的函数：

![](img/ee38e828f37f574f76b4abb20f7528dc_8.png)

```python
def standardize(series):
    """
    对输入的pandas Series进行标准化处理。
    参数:
        series: 待处理的数据序列。
    返回:
        标准化后的数据序列。
    """
    # 计算序列的均值和标准差
    mean_val = series.mean()
    std_val = series.std()
    
    # 应用标准化公式: (x - 均值) / 标准差
    return (series - mean_val) / std_val
```

代码非常简单：首先计算数据的均值和标准差，然后对每个数据点应用标准化公式。

![](img/ee38e828f37f574f76b4abb20f7528dc_14.png)

## 效果演示

![](img/ee38e828f37f574f76b4abb20f7528dc_22.png)

让我们通过一个简单的例子来看看标准化的效果。假设我们有一组原始数据，其数值范围差异很大。

![](img/ee38e828f37f574f76b4abb20f7528dc_24.png)

```python
# 假设这是原始数据的一列
original_data = pd.Series([30, 25, 120, 80, 15])
print("原始数据：")
print(original_data)

![](img/ee38e828f37f574f76b4abb20f7528dc_28.png)

# 应用我们的标准化函数

```

![](img/ee38e828f37f574f76b4abb20f7528dc_30.png)

运行后，你会发现原本数值在十几到一百多的数据，经过标准化后，其值大多会落在 -2 到 2 这个较小的区间内。这验证了标准化的效果：
1.  数据以零为中心分布。
2.  各特征的数据被缩放到了相近的数值范围。
