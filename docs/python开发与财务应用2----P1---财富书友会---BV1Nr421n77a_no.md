## 数据类型回顾与变量定义

上一节我们介绍了Python数字数据类型的基础理论。本节中，我们来看看如何定义这些类型的变量，并使用`type()`函数来验证它们。

以下是定义三种基本数字类型变量的代码示例：

```python
# 定义整型变量
int_demo = 123
# 定义浮点型变量
dec_demo = 55.78
# 定义布尔型变量
bool_demo = True

# 使用type()函数测试变量的数据类型
print(int_demo, type(int_demo))
print(dec_demo, type(dec_demo))
print(bool_demo, type(bool_demo))
```
运行上述代码，控制台将分别输出：
- `123 <class 'int'>`
- `55.78 <class 'float'>`
- `True <class 'bool'>`

`type()`是一个内置函数，用于返回变量的数据类型。`<class ‘int’>` 表示该变量是整型。

理解了如何定义变量后，接下来我们实践最基本的算术运算。我们通过两个变量来演示加、减、乘、除等操作。

![](img/8693652a13b05365f34f5f1fad3ae72e_12.png)

以下是使用算术运算符的示例：

![](img/8693652a13b05365f34f5f1fad3ae72e_14.png)

```python
# 定义两个变量
a = 6
b = 10

# 加法运算

# 减法运算
print(a - b)  # 输出：-4
```
其他运算如乘法（`*`）、除法（`/`）、求幂（`**`）等，可以按照相同格式自行尝试。编写代码时，建议在运算符前后添加空格，使代码更清晰易读。

## 赋值运算符的使用

![](img/8693652a13b05365f34f5f1fad3ae72e_16.png)

![](img/8693652a13b05365f34f5f1fad3ae72e_18.png)

除了基本的等号赋值，Python还提供了一种简洁的赋值运算符，如 `+=`、`-=`。它们可以在原值基础上进行运算并重新赋值。

![](img/8693652a13b05365f34f5f1fad3ae72e_19.png)

以下是赋值运算符的示例：

![](img/8693652a13b05365f34f5f1fad3ae72e_21.png)

```python
a = 6
b = 10

# 基本赋值运算
c = a + b
print(c)  # 输出：16

![](img/8693652a13b05365f34f5f1fad3ae72e_23.png)

# 加等赋值运算
a += b  # 等价于 a = a + b
print(a)  # 输出：16
```
`a += b` 意味着将 `a` 与 `b` 相加的结果再赋值给 `a`。类似的，`-=`、`*=`、`/=` 等运算符用法相同。

## 关系运算符比较

关系运算符用于比较两个值的大小或相等关系，其结果为布尔值（`True` 或 `False`）。这在程序逻辑判断中非常有用。

![](img/8693652a13b05365f34f5f1fad3ae72e_29.png)

以下是关系运算符的示例：

![](img/8693652a13b05365f34f5f1fad3ae72e_31.png)

```python
a = 6
b = 10

![](img/8693652a13b05365f34f5f1fad3ae72e_33.png)

![](img/8693652a13b05365f34f5f1fad3ae72e_35.png)

# 大于比较
print(a > b)  # 输出：False
# 小于比较
print(a < b)  # 输出：True
# 等于比较
print(a == b) # 输出：False
```
其他关系运算符还包括：大于等于（`>=`）、小于等于（`<=`）、不等于（`!=`）。请自行尝试这些运算。

## 逻辑运算符实践

![](img/8693652a13b05365f34f5f1fad3ae72e_41.png)

最后，我们来实践逻辑运算符。逻辑运算符用于对布尔值进行“与（and）”、“或（or）”、“非（not）”运算，常用于组合多个条件。

以下是逻辑运算符的示例：

```python
# 定义两个布尔变量
a = True
b = False

![](img/8693652a13b05365f34f5f1fad3ae72e_43.png)

![](img/8693652a13b05365f34f5f1fad3ae72e_45.png)

# 与运算：两者都为真，结果才为真
print(a and b)  # 输出：False
# 或运算：至少一个为真，结果即为真
print(a or b)   # 输出：True
# 非运算：取反
print(not a)    # 输出：False
```
请尝试修改 `a` 和 `b` 的值，观察不同组合下逻辑运算的结果。
