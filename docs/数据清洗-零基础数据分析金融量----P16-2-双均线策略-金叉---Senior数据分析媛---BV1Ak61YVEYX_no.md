## 📈 什么是金叉与死叉？

![](img/6fff5110b75d6e03e540270207ba4cf6_5.png)

![](img/6fff5110b75d6e03e540270207ba4cf6_7.png)

上一节我们介绍了如何计算并绘制5日和30日移动平均线。本节中我们来看看如何利用这两条均线判断买卖点，这涉及到两个核心概念：金叉和死叉。

![](img/6fff5110b75d6e03e540270207ba4cf6_11.png)

![](img/6fff5110b75d6e03e540270207ba4cf6_13.png)

![](img/6fff5110b75d6e03e540270207ba4cf6_15.png)

在股票技术分析中，金叉和死叉是分析指标中两根线交叉时的状态。在我们的图表中，这两根线分别是5日均线（短期均线，`ma5`）和30日均线（长期均线，`ma30`）。

![](img/6fff5110b75d6e03e540270207ba4cf6_17.png)

![](img/6fff5110b75d6e03e540270207ba4cf6_19.png)

*   **金叉**：当短期均线（`ma5`）从下方**上穿**长期均线（`ma30`）时，交叉点附近的位置称为金叉。
*   **死叉**：当短期均线（`ma5`）从上方**下穿**长期均线（`ma30`）时，交叉点附近的位置称为死叉。

**核心公式**：
*   金叉：`ma5` 上穿 `ma30`
*   死叉：`ma5` 下穿 `ma30`

一般情况下，金叉出现后操作趋向于**买入**，死叉出现后操作趋向于**卖出**。双均线策略的核心就是根据金叉和死叉的信号来制定股票买卖决策。

## 🔍 如何找出金叉与死叉的日期？

![](img/6fff5110b75d6e03e540270207ba4cf6_29.png)

我们的目标是找出图表中所有金叉和死叉发生的时间点（即X轴对应的日期）。这需要分析`ma5`和`ma30`两条线在交点前后的数值大小关系。

观察交点可以发现一个规律：
*   在**金叉点**左侧，`ma5 < ma30`；在右侧，`ma5 > ma30`。
*   在**死叉点**左侧，`ma5 > ma30`；在右侧，`ma5 < ma30`。

因此，我们可以通过判断`ma5`和`ma30`的大小关系序列，找到从“小于”变为“大于”（金叉）或从“大于”变为“小于”（死叉）的转折点。

![](img/6fff5110b75d6e03e540270207ba4cf6_31.png)

以下是具体步骤：

![](img/6fff5110b75d6e03e540270207ba4cf6_33.png)

![](img/6fff5110b75d6e03e540270207ba4cf6_35.png)

首先，我们需要确保`ma5`和`ma30`两个序列长度一致且没有空值，以便进行后续比较。由于`ma30`的前29个值为空，我们统一从第30个数据开始切片。

```python
# 统一切片，从第30个数据开始，保证索引一致且无空值
ma5 = ma5[30:]
ma30 = ma30[30:]
df = df[30:] # 原数据也需要同步切片
```

接着，我们创建两个布尔序列，分别表示`ma5`小于`ma30`和`ma5`大于`ma30`的条件。

```python
# 创建大小关系判断序列
s1 = ma5 < ma30  # 序列，True表示 ma5 < ma30
s2 = ma5 > ma30  # 序列，True表示 ma5 > ma30
```

![](img/6fff5110b75d6e03e540270207ba4cf6_37.png)

![](img/6fff5110b75d6e03e540270207ba4cf6_39.png)

`s1`和`s2`是互补的。在`s1`序列中，`True`表示`ma5 < ma30`，`False`表示`ma5 > ma30`。我们的目标是找到序列值发生变化的点。

通过分析，我们可以得到以下规律：
*   在`s1`序列中，从`True`过渡到`False`，对应一个**金叉**。
*   在`s1`序列中，从`False`过渡到`True`，对应一个**死叉**。

为了用代码捕捉这种“过渡”，我们可以利用`shift()`函数移动序列，然后进行逻辑运算。

![](img/6fff5110b75d6e03e540270207ba4cf6_41.png)

**找出死叉日期**：
死叉对应`s1`中从`False`过渡到`True`。我们可以通过`s1 & (s2.shift(1))`这个条件来定位。因为`s2`是`s1`的取反，`s2.shift(1)`相当于`s1`序列向前看一位。当`s1`为`True`且前一位`s1`为`False`时，这个逻辑与运算的结果为`True`。

```python
# 判断死叉的条件：当前s1为True，且前一天的s1为False (即s2为True)
death_cross_condition = s1 & s2.shift(1)
# 获取死叉日期：将条件作为布尔索引，筛选原数据，然后取索引（日期）
death_dates = df.loc[death_cross_condition].index
```

**找出金叉日期**：
金叉对应`s1`中从`True`过渡到`False`。我们可以通过`~(s1 | s2.shift(1))`这个条件来定位。`s1 | s2.shift(1)`在过渡点会得到`False`，取反后即为`True`。

![](img/6fff5110b75d6e03e540270207ba4cf6_43.png)

```python
# 判断金叉的条件：当前s1为False，且前一天的s1为True (即s2为False)
golden_cross_condition = ~(s1 | s2.shift(1))
# 获取金叉日期：将条件作为布尔索引，筛选原数据，然后取索引（日期）
golden_dates = df.loc[golden_cross_condition].index
```
