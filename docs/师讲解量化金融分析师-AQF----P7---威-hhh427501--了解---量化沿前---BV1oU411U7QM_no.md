## 梯度下降回顾 📉

在深入逻辑回归之前，有必要回顾一下梯度下降的核心思想。梯度下降是一个最小化损失函数（Cost Function）的过程。

我们可以把损失函数想象成一座山，我们的目标是找到山谷的最低点（即误差最小）。我们从随机一个点开始，计算该点损失函数的梯度（即斜率），然后沿着梯度下降最快的方向移动一小步。重复这个过程，就像一步步下山，最终找到损失函数的最小值点。

这个过程的核心是**求导**。当在某一点的切线斜率为0时，我们就找到了一个极值点（可能是最小值）。公式上，我们通过迭代更新参数 `θ` 来最小化损失函数 `J(θ)`：
`θ := θ - α * ∇J(θ)`
其中 `α` 是学习率，`∇J(θ)` 是损失函数关于参数 `θ` 的梯度。

## 从回归到分类：逻辑回归的引入 🔄

线性回归适用于预测连续值（如股价收益率）。但当我们需要预测离散的类别标签时，例如判断股票下一期是“涨”（标记为1）还是“跌”（标记为0），就需要使用分类算法。逻辑回归就是解决这类二分类问题的经典算法。

为什么不能直接用线性回归做分类？假设我们用肿瘤尺寸预测其是恶性（1）还是良性（0）。如果用线性回归拟合，会得到一条斜线。我们可能会设定一个阈值（如0.5），高于阈值预测为恶性，反之预测为良性。但问题是，如果数据分布发生变化，这条回归线会移动，导致基于固定阈值的分类效果很差。逻辑回归的目标是找到一条“S”形曲线，能更好地在两类数据之间划出分界线。

对于有两个特征的问题（如肿瘤尺寸和患者年龄），逻辑回归的目标是找到一条**决策边界（Decision Boundary）**，最好地将两类数据点分割开来。

## 逻辑回归的核心原理 ⚙️

逻辑回归背后的核心思想是“分割数据”。它通过一个特殊的函数将线性方程的输出映射到0和1之间，从而得到样本属于某一类的概率。

### 激活函数：Sigmoid函数

这个关键函数叫做 **Sigmoid函数**（或称Logistic函数）。它的作用是将任何实数输入映射到(0,1)区间内，完美地代表了概率。

Sigmoid函数的公式如下：
`g(z) = 1 / (1 + e^{-z})`

其中，`z` 通常是一个线性方程，例如 `z = θ₀ + θ₁X₁ + θ₂X₂`。

这个函数的图形是一个平滑的“S”形曲线：
*   当 `z` 趋近于正无穷大时，`g(z)` 趋近于1。
*   当 `z` 趋近于负无穷大时，`g(z)` 趋近于0。
*   当 `z = 0` 时，`g(z) = 0.5`。

![](img/c46cfb6037d8e4add1db80679f483937_9.png)

![](img/c46cfb6037d8e4add1db80679f483937_11.png)

因此，我们可以设定一个规则：如果 `g(z) >= 0.5`，则预测类别为1；如果 `g(z) < 0.5`，则预测类别为0。由于 `g(z) >= 0.5` 等价于 `z >= 0`，所以决策边界实际上就是线性方程 `z = 0` 所构成的线或平面。

### 损失函数与极大似然估计

逻辑回归也需要一个损失函数来衡量预测的误差，并通过梯度下降来最小化它。但这里不能使用线性回归的均方误差（MSE），因为套用Sigmoid函数后，损失函数会变成非凸（non-convex）的，容易陷入局部最优解。

逻辑回归的损失函数基于**极大似然估计**的思想构建。其核心是：寻找一组参数 `θ`，使得在这组参数下，当前观测到的所有数据出现的**联合概率最大**。

以下是构建损失函数的简化思路：

1.  对于单个样本，其预测概率为：
    *   如果真实标签 `y=1`，我们希望预测概率 `h_θ(x)` 尽量接近1。损失可以表示为 `-log(h_θ(x))`。预测越准（`h_θ(x)`越接近1），`-log`值越小。
    *   如果真实标签 `y=0`，我们希望预测概率 `h_θ(x)` 尽量接近0，即 `1 - h_θ(x)` 尽量接近1。损失可以表示为 `-log(1 - h_θ(x))`。

2.  将两种情况合并，得到单个样本的损失：
    `Cost(h_θ(x), y) = -[y * log(h_θ(x)) + (1-y) * log(1 - h_θ(x))]`

3.  对所有训练样本的损失求和或求平均，就得到了逻辑回归的损失函数（对数损失函数）：
    `J(θ) = -1/m * Σ [y⁽ⁱ⁾ * log(h_θ(x⁽ⁱ⁾)) + (1-y⁽ⁱ⁾) * log(1 - h_θ(x⁽ⁱ⁾))]`

这个函数是凸函数，可以通过梯度下降找到全局最优解。最小化这个损失函数，就等价于最大化所有数据出现的联合概率（极大似然估计）。

## 逻辑回归的简单示例 💡

让我们通过一个具体例子理解决策边界。

假设我们有两个特征 `X1` 和 `X2`，并假设通过训练得到的逻辑回归模型参数为 `θ₀ = -3`, `θ₁ = 1`, `θ₂ = 1`。那么我们的线性方程 `z` 为：
`z = -3 + X1 + X2`

决策边界是 `z = 0`，即：
`-3 + X1 + X2 = 0` 或 `X1 + X2 = 3`

这是一条直线。在二维平面上：
*   对于 `X1 + X2 >= 3` 的区域，`z >= 0`，因此 `g(z) >= 0.5`，模型预测类别为1。
*   对于 `X1 + X2 < 3` 的区域，`z < 0`，因此 `g(z) < 0.5`，模型预测类别为0。

这条直线 `X1 + X2 = 3` 就是模型的决策边界。

## 逻辑回归的应用与实践 🛠️

尽管原理可能看起来复杂，但在实际使用Python进行机器学习时，逻辑回归的实现非常简便，这得益于 `scikit-learn` 等强大的库。

![](img/c46cfb6037d8e4add1db80679f483937_13.png)

以下是使用 `scikit-learn` 实现逻辑回归的核心代码框架：

```python
from sklearn.linear_model import LogisticRegression

# 1. 创建模型实例，可以设置参数（如正则化强度C）
model = LogisticRegression(C=1.0)

# 2. 使用训练数据拟合模型
model.fit(X_train, y_train)

# 3. 使用训练好的模型进行预测
y_pred = model.predict(X_test)

# 4. 评估模型性能（例如准确率）
from sklearn.metrics import accuracy_score
accuracy = accuracy_score(y_test, y_pred)
```

在量化金融中，逻辑回归可以用于构建趋势预测模型。例如，我们可以使用市盈率（PE）作为特征，将未来一段时间涨幅排名前50的股票标记为1（涨），后50的标记为0（跌），然后训练逻辑回归模型来预测未来的涨跌方向。
