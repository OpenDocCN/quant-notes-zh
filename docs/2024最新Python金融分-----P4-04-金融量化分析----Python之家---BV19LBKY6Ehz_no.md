# 金融量化分析：P4：影响股价因素与股票买卖知识 📉

在本节课中，我们将学习影响股价波动的主要因素以及股票买卖的基本流程。这些内容对于理解金融市场的运行至关重要，尤其是对于初学者而言。接下来，我们将深入探讨这些因素，并一步步讲解股票交易的步骤。

---

## 1. 影响股价的因素

影响股价的因素有很多，本节将从**公司自身因素**、**市场因素**、**行业因素**、**心理因素**、**经济因素**、**政治因素**等方面进行详细分析。

### 1.1 公司自身因素

公司自身的运营状况是影响股价最根本的因素。公司是否盈利，发展潜力如何，这些都会直接影响投资者对股票的买卖决策。

**例子：** 假设某公司成功上市，市值为50亿人民币。如果公司在市场上公开发行1亿股，那么每股的发行价格为5元。当公司经营状况良好，市值增加至70亿时，股票价格也随之上涨。

**公式：**
- 市值 = 股数 × 每股价格
- 市值变化 → 股价变化

### 1.2 市场因素

市场上的供求关系直接影响股价。当买的人多、卖的人少时，股价会上涨；反之，卖的人多、买的人少时，股价会下跌。

**例子：** 如果某股票买家数量远超过卖家，股价就会因为需求过大而上涨。

**公式：**
- 买入需求 > 卖出供应 → 股价上涨
- 卖出供应 > 买入需求 → 股价下跌

### 1.3 行业因素

行业的整体发展趋势也会对股价产生影响。如果行业前景看好，相应的股票价格可能会上涨。比如，近年来人工智能行业的快速发展吸引了大量投资，相关公司股票普遍上涨。

### 1.4 心理因素

股民的情绪和心理也会影响股价。例如，在股市恐慌中，大股东可能会集中抛售股票，引发小股东跟风抛售，导致股价大幅下跌。

**例子：** 美国股市历史上曾发生“黑色星期五”事件，因程序交易失误导致大规模抛售，市场情绪一度崩溃。

### 1.5 经济因素

国家的经济政策、利率水平等都会影响股市。例如，当存款利率上升时，银行存款的吸引力增大，股市资金流入减少，可能导致股价下跌。

**例子：** 存款利率上升 → 资金流出股市 → 股价下跌

### 1.6 政治因素

政治因素同样会对股价产生重大影响。例如，国际局势紧张时，投资者可能会转移资金，导致股价波动。

**例子：** 2017年，美国与朝鲜关系紧张时，股市大幅下跌，而与军工相关的股票则可能上涨。

## 2. 股票买卖的基本流程

股票交易的过程并不复杂，但涉及到一些重要步骤。接下来我们详细介绍股票买卖的过程。

### 2.1 开户与委托

首先，投资者需要在券商处开设股票账户。之后，可以通过券商提交买卖委托订单。股票的买卖不能直接进行，而是需要通过证券公司进行中介。

### 2.2 交易时间

股票交易并不是全天开放的。一般来说，股票市场的交易时间为工作日的上午9点到下午3点，但具体时间安排有所不同。

**交易时间：**
- 交易日：周一到周五（法定节假日除外）
- 交易时间：上午9:00 - 15:00

### 2.3 开盘前的竞价阶段

在9:00到9:15之间，股市会进行竞价撮合。这个阶段没有实际的交易，但系统会根据买卖需求计算出一个开盘价。

### 2.4 开盘后的连续竞价

从9:15到15:00，股市进入连续竞价阶段。这个阶段，股价会每隔几秒进行一次撮合交易，价格会随着买卖的供需情况实时波动。

### 2.5 收盘集合竞价（仅适用于深圳股市）

深圳股市在3点之前的最后3分钟进行收盘集合竞价，用于确定收盘价。上海股市没有收盘集合竞价，收盘价由3点的最后一笔交易决定。
