## 从传统编程到机器学习

上一节我们介绍了数据获取与处理，本节中我们来看看如何利用这些数据建立模型。

传统的交易系统策略由人工定义。程序员或交易员观察到某些市场规律（例如移动平均线交叉），并将这些规则写成 `if-else` 逻辑程序。计算机根据输入（如MA5和MA10）触发规则，输出看涨或看跌信号。

机器学习与传统方法最大的不同在于，其规则不是由程序员制定的，而是从数据中自动学习出来的。你只需向程序提供大量“输入-输出”对应关系的事实，程序就会自动生成一系列规则。这种方法的好处在于，计算机能从海量、高维数据中挖掘出人类难以直观发现的隐藏关系。

机器学习应用广泛，例如：
*   **垃圾邮件过滤**：输入邮件，输出是否为垃圾邮件。
*   **图像识别**：输入图片，输出图片中的字符。
*   **自动驾驶**
*   **电影推荐**

其美妙之处在于，识别手写数字的模型（如SVM）与进行股票分类的模型，背后的数学原理是相通的，具有很强的鲁棒性和普适性。

## 机器学习的三种类型

![](img/db553f00b041fc3f160ea321c1cc0290_2.png)

以下是机器学习的主要分类：

1.  **监督学习**：提供输入 `X` 和对应的输出 `Y`。`Y` 可以是一个具体数值（回归问题），也可以是一个类别标签（分类问题）。量化中，预测下一时刻价格是回归问题，判断涨跌是分类问题。
2.  **非监督学习**：只提供输入 `X`，没有对应的 `Y`。目标是让模型从 `X` 中自行发现规律，例如对股票进行聚类分组。
3.  **强化学习**：智能体与环境（如股票市场）交互。智能体给出动作（买入/卖出），环境返回奖励。目标是学习一系列动作以最大化累积奖励。其理论基础是马尔可夫决策过程。

## Scikit-learn 使用指南

Scikit-learn（sklearn）是Python中著名且常用的机器学习库，它封装了除深度神经网络外的主流算法，能极大简化建模过程。

![](img/db553f00b041fc3f160ea321c1cc0290_4.png)

一个典型的建模流程包含以下三步：

```python
# 1. 准备数据
training_x, training_y, test_x, test_y = get_data()
print(training_x.shape, training_y.shape) # 调试，确保维度匹配

# 2. 训练模型
model = SomeModel() # 例如 LinearRegression(), Lasso(), SVM()
model.fit(training_x, training_y) # 模型学习参数

# 3. 评估模型
predictions = model.predict(test_x)
score = model.score(test_x, test_y) # 使用评分函数评估
```

**重要提示**：模型在训练集上的评分（`score`）好，不等于回测效果好，更不等于实盘能盈利。评分是必要不充分条件。

## 从线性回归到正则化

现在，我们开始深入探讨线性模型。我们将从最简单的普通最小二乘法开始，一路推导到工业界广泛使用的正则化方法。

### 线性回归模型

线性回归假设输出 `Y` 与输入 `X` 呈线性关系。其模型表示为：

**Y = Xθ**

其中，`Y` 是 `N×1` 的向量（`N` 个样本），`X` 是 `N×D` 的矩阵（`D` 个特征），`θ` 是 `D×1` 的参数向量。

线性回归的目标是找到一组参数 `θ`，使得预测值与真实值之间的误差平方和最小（最小二乘）：

**minimize ||Y - Xθ||²**

这个优化问题有闭式解（解析解）：

**θ = (XᵀX)⁻¹XᵀY**

### 岭回归的诞生

在实际计算中，矩阵 `XᵀX` 可能不可逆或病态。数值计算中的一个技巧是给 `XᵀX` 加上一个小的对角线矩阵（如 `λI`）再求逆。有趣的是，人们发现，使用这个技巧求出的 `θ`，在测试集上的表现往往比标准的线性回归解更好。

这引导数学家重新审视目标函数。岭回归的目标函数是在最小二乘的基础上，增加了一个对参数 `θ` 的惩罚项（L2正则化）：

**minimize ||Y - Xθ||² + δ²||θ||²**

其中，`||θ||² = θ₁² + θ₂² + ... + θ_D²`，`δ²` 是超参数，控制惩罚力度。

从优化角度理解，这个目标等价于在约束条件 `||θ||² ≤ t`（`t` 为某个常数）下，最小化 `||Y - Xθ||²`。在二维情况下，约束区域是一个圆，最小二乘的等高线是椭圆。最优解出现在圆与椭圆的切点处。

**岭回归的作用**：通过惩罚大的参数值，使所有参数向零收缩，防止某些不重要的特征对模型产生过大影响，从而提高模型的泛化能力。

### LASSO回归

岭回归虽然收缩参数，但不会将任何参数精确变为零。然而，在特征选择中，我们常希望模型能自动剔除不重要的特征（即将其系数设为0）。LASSO回归通过修改惩罚项实现了这一点。

LASSO回归的目标函数使用L1正则化：

**minimize ||Y - Xθ||² + δ²||θ||₁**

其中，`||θ||₁ = |θ₁| + |θ₂| + ... + |θ_D|`。

其几何解释是，约束区域从岭回归的“圆”变成了“菱形”。菱形具有尖角，与椭圆等高线相切时，切点更容易落在坐标轴上，使得某些 `θ_i` 恰好为0。这样，LASSO就实现了特征选择。

## 从线性到非线性：核方法

线性模型能力有限。一个简单的非线性化方法是进行特征变换。例如，对于一维输入 `x`，我们可以构造新特征 `[1, x, x², x³, ...]`，然后对新特征做线性回归，这样就得到了一个关于 `x` 的非线性函数。

核方法（如径向基函数网络）是更优雅的非线性化手段。其核心思想是：用一组非线性基函数的线性组合来拟合数据。常用的基函数是高斯核函数：

**φ(x) = exp(-λ||x - μ||²)**

其中，`μ` 是基函数的中心位置，`λ` 控制函数的宽度。

**如何放置这些基函数？**
1.  **简单方法**：在每个训练数据点 `x_i` 处放置一个基函数。但这会导致基函数数量等于样本数，计算量大。
2.  **常用方法**：使用K-Means聚类算法，在数据中找出 `K` 个聚类中心，将这些中心作为基函数的位置 `μ`。`K` 和 `λ` 都是需要调节的超参数。

通过调节 `λ`，可以控制模型的复杂度：`λ` 太小，基函数很“瘦”，容易过拟合；`λ` 太大，基函数很“胖”，模型趋向于直线，容易欠拟合。

## 超参数选择与交叉验证

无论是岭回归的 `δ`、LASSO的 `δ`，还是核方法的 `K` 和 `λ`，这些超参数都需要谨慎选择。直接在测试集上选择超参数会导致“数据窥探”，使评估结果过于乐观。

科学的做法是使用**交叉验证**。典型流程如下：

1.  将数据分为三部分：训练集（70%）、验证集（15%）、测试集（15%）。
2.  在训练集上，用不同的超参数组合训练多个模型。
3.  在验证集上评估这些模型，选择表现最好的超参数组合。
4.  用选出的超参数，在**训练集+验证集**上重新训练最终模型。
5.  在**测试集**上评估最终模型，得到汇报用的性能分数。
6.  如果决定部署模型，则使用**全部数据**和选定的超参数训练最终的生产模型。

对于多个超参数，可以使用网格搜索（Grid Search）暴力尝试所有组合，并通过热力图可视化结果。sklearn提供了方便的 `GridSearchCV` 等工具来实现自动化交叉验证。
