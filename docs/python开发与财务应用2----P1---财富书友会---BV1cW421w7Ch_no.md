## 数据类型概述 📝

上一节我们介绍了Python编程的基本环境。本节中，我们来看看程序如何处理不同的数据。

数据类型是编程中用于区分数据种类的概念。现实世界中的数据多种多样，例如文本、数字、日期等。在编程中，我们需要明确区分这些类型，以便计算机能正确存储和处理它们。

计算机通常以**二维表**的形式存储数据。例如，一个学生成绩表，每一行代表一个学生的数据（如姓名、年龄、分数），每一列则代表一种属性。每个属性（列）都有其特定的数据类型。

以下是Python中几种基础的数据类型：

*   **数值型**：代表数字，包括整数（如 `10`）和带小数点的浮点数（如 `3.14`）。
*   **字符型**：代表文本，由单引号 `‘ ’`、双引号 `“ ”` 或三引号 `‘’‘ ’’’` 包围。单/双引号用于单行文本，三引号用于多行文本。
*   **布尔型**：代表逻辑值，只有 `True`（真）和 `False`（假）两种。在底层，`True` 对应数字 `1`，`False` 对应数字 `0`。
*   **复合型**：如**列表**（类似数组）和**字典**（存储键值对），这些我们将在后续课程中详细介绍。

## 算术运算符 ➕➖✖️➗

了解了数字型数据后，我们来看看如何对它们进行计算。连接两个数字型数据（常量或变量）进行运算的符号，称为**算术运算符**。

以下是基本的算术运算符：

*   **加法**：`+`
    *   示例：`5 + 6` 的结果是 `11`。
*   **减法**：`-`
    *   示例：`10 - 4` 的结果是 `6`。
*   **乘法**：`*`
    *   示例：`3 * 7` 的结果是 `21`。
*   **除法**：`/`
    *   示例：`9 / 2` 的结果是 `4.5`。
*   **求模（取余）**：`%`
    *   这是求除法运算后的余数。
    *   示例：`10 % 6` 的结果是 `4`。因为 `10` 除以 `6` 商 `1` 余 `4`。
*   **求幂（乘方）**：`**`
    *   示例：`2 ** 3` 表示 `2` 的 `3` 次方，结果是 `8`。

运算符之间存在**优先级**。当多个运算符同时出现时，计算顺序为：
1.  先计算括号 `()` 内的表达式。
2.  然后计算乘方 `**`。
3.  接着计算乘法 `*`、除法 `/`、取余 `%`。
4.  最后计算加法 `+` 和减法 `-`。
同级运算符（如多个加减法）则按**从左到右**的顺序计算。

## 赋值运算符 🏷️

在编程中，我们经常需要将运算结果保存起来供后续使用，这时就需要用到**赋值运算符**。

最基本的赋值运算符是单个等号 `=`。它的作用是将等号右边的值“赋予”等号左边的变量。

> **重要提示**：请将 `=` 读作“赋值给”或“设为”，而不是“等于”。“等于”是后面要讲的关系比较概念。

示例：
```python
a = 5  # 将数字5赋值给变量a
b = 3  # 将数字3赋值给变量b
c = a + b  # 先计算a+b（得到8），然后将结果8赋值给变量c
print(c)  # 输出：8
```

此外，还有一类复合赋值运算符，它们将运算和赋值合并为一步：

![](img/611dae947ea285699c5a024deaaff2ca_1.png)

![](img/611dae947ea285699c5a024deaaff2ca_3.png)

*   `+=`：加后赋值。`c += a` 等价于 `c = c + a`。
*   `-=`：减后赋值。`c -= a` 等价于 `c = c - a`。
*   `*=`：乘后赋值。
*   `/=`：除后赋值。
*   `%=`：取余后赋值。
*   `**=`：求幂后赋值。

![](img/611dae947ea285699c5a024deaaff2ca_5.png)

示例：
```python
c = 10
a = 4
c += a  # 相当于 c = 10 + 4
print(c)  # 输出：14
# 注意，变量a的值在这个过程中没有改变，仍然是4。
```

## 比较运算符 ⚖️

![](img/611dae947ea285699c5a024deaaff2ca_7.png)

![](img/611dae947ea285699c5a024deaaff2ca_9.png)

程序不仅需要计算，还需要做出判断。**比较运算符**用于比较两个值之间的关系，其运算结果是一个**布尔值**（`True` 或 `False`）。

![](img/611dae947ea285699c5a024deaaff2ca_11.png)

以下是常用的比较运算符：

![](img/611dae947ea285699c5a024deaaff2ca_13.png)

*   **等于**：`==` （注意是两个等号，用于比较）
*   **不等于**：`!=`
*   **大于**：`>`
*   **小于**：`<`
*   **大于等于**：`>=`
*   **小于等于**：`<=`

![](img/611dae947ea285699c5a024deaaff2ca_15.png)

示例：
```python
a = 4
c = 17
print(c > a)   # 判断c是否大于a，输出：True
print(c == a)  # 判断c是否等于a，输出：False
print(c != a)  # 判断c是否不等于a，输出：True
```

## 逻辑运算符 🔗

![](img/611dae947ea285699c5a024deaaff2ca_17.png)

有时我们需要对多个比较条件进行组合判断，这时就要用到**逻辑运算符**。它们操作的对象是布尔值，结果也是布尔值。

主要的逻辑运算符有三个：

1.  **与** `and`：可以理解为乘法。只有当**两边条件都为真**时，结果才为真。
    *   `True and True` 结果为 `True`
    *   `True and False`、`False and True`、`False and False` 结果均为 `False`

![](img/611dae947ea285699c5a024deaaff2ca_19.png)

2.  **或** `or`：可以理解为加法。只要**两边条件有一个为真**，结果就为真。
    *   `False or False` 结果为 `False`
    *   `True or False`、`False or True`、`True or True` 结果均为 `True`

3.  **非** `not`：取反运算。将真变假，假变真。
    *   `not True` 结果为 `False`
    *   `not False` 结果为 `True`

---

## 运算符优先级总结 🥇

当表达式包含多种运算符时，必须明确计算顺序。请记住以下优先级规则（从高到低）：

1.  **括号** `()`：拥有最高优先级，优先计算括号内的内容。
2.  **算术运算符**：按 `**` → `*` `/` `%` → `+` `-` 的顺序。
3.  **比较运算符**：如 `>`、`<`、`==` 等。
4.  **逻辑运算符**：按 `not` → `and` → `or` 的顺序。

**核心口诀**：**从左往右看，括号最优先，先乘除后加减，再比较后逻辑。**
