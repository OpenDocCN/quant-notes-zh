## 计算移动平均线

![](img/a787a07321c0c597e362a2d0a5f9647e_34.png)

上一节我们准备好了数据，本节中我们来看看如何计算技术指标。我们将计算短期和长期移动平均线（Moving Average, MA），这是识别趋势和交叉信号的基础。

![](img/a787a07321c0c597e362a2d0a5f9647e_36.png)

首先，定义短期和长期的时间窗口。考虑到我们的数据集时间跨度较长（约7-8年），为了在图表中更清晰地观察趋势，我们将窗口设置得比通常的5日和20日更大一些。

```python
short_window = 40
long_window = 100
```

![](img/a787a07321c0c597e362a2d0a5f9647e_38.png)

接下来，我们使用`rolling`和`mean`方法计算移动平均线，并将结果作为新列添加到数据框中。

![](img/a787a07321c0c597e362a2d0a5f9647e_40.png)

```python
# 计算短期移动平均线
apple_data['short_ma'] = apple_data['AAPL'].rolling(window=short_window).mean()
# 计算长期移动平均线
apple_data['long_ma'] = apple_data['AAPL'].rolling(window=long_window).mean()
```

![](img/a787a07321c0c597e362a2d0a5f9647e_42.png)

![](img/a787a07321c0c597e362a2d0a5f9647e_44.png)

在这张图中，我们可以观察到短期均线（`short_ma`）和长期均线（`long_ma`）的交叉点。这些交叉点通常被视为交易信号：
*   **黄金交叉**：短期均线从下向上穿越长期均线，可能预示着上涨趋势的开始，是潜在的**买入信号**。
*   **死亡交叉**：短期均线从上向下穿越长期均线，可能预示着下跌趋势的开始，是潜在的**卖出信号**。

![](img/a787a07321c0c597e362a2d0a5f9647e_64.png)

## 标记交易信号

![](img/a787a07321c0c597e362a2d0a5f9647e_66.png)

![](img/a787a07321c0c597e362a2d0a5f9647e_68.png)

为了更系统地进行策略分析，我们需要用数据明确标记出这些信号点。首先，我们需要删除在计算移动平均线时产生的缺失值。

![](img/a787a07321c0c597e362a2d0a5f9647e_70.png)

![](img/a787a07321c0c597e362a2d0a5f9647e_72.png)

```python
apple_data.dropna(inplace=True)
```

![](img/a787a07321c0c597e362a2d0a5f9647e_74.png)

然后，我们创建一个新的列`position`作为标志位。当短期均线高于长期均线时，标记为1；否则标记为0。这代表了“持有”或“看涨”的信号强度。

![](img/a787a07321c0c597e362a2d0a5f9647e_76.png)

```python
apple_data['position'] = np.where(apple_data['short_ma'] > apple_data['long_ma'], 1, 0)
```

![](img/a787a07321c0c597e362a2d0a5f9647e_78.png)

现在，我们的数据框中就包含了一个清晰的信号列。我们可以再次可视化，将买卖信号（通过`position`的变化来体现）也展示在图上。

![](img/a787a07321c0c597e362a2d0a5f9647e_80.png)

```python
fig, ax1 = plt.subplots(figsize=(14, 7))

![](img/a787a07321c0c597e362a2d0a5f9647e_82.png)

ax1.plot(apple_data['AAPL'], label='Price', alpha=0.5)

ax2 = ax1.twinx()

ax2.set_ylim([-0.5, 1.5])

![](img/a787a07321c0c597e362a2d0a5f9647e_84.png)
```

## 策略思路探讨

通过可视化，我们能够清晰地看到理论上的买卖点。这引出了一个重要的策略思路：与简单的“买入并持有”策略不同，我们可以尝试根据这些交叉信号进行主动交易。

例如：
*   当出现**黄金交叉**（`position`从0变为1）时，买入股票。
*   当出现**死亡交叉**（`position`从1变为0）时，卖出股票。

理论上，这种策略旨在抓住主要的趋势变动，在上涨初期买入，在下跌初期卖出，从而可能获得比被动持有更高的收益，或规避部分下跌风险。本节课我们完成了信号的可视化识别，为后续回测这种双均线交叉策略的盈利能力打下了基础。
