# 量化交易教程：P46：工具包绘图展示

在本节课中，我们将学习如何使用专门的绘图工具包，将计算得到的因子IC值（信息系数）进行可视化展示。通过图表，我们可以更直观地评估因子的有效性和稳定性。

上一节我们介绍了如何计算因子与收益率之间的相关系数（IC值）。本节中，我们来看看如何将这些数值结果转化为直观的图表。

## 绘图工具导入与使用

![](img/404997d68477250de6c613545eaec441_1.png)

为了进行可视化，我们需要导入一个名为 `plotting` 的工具包。这个工具包专门用于展示量化分析中的各类指标。

以下是使用 `plotting` 绘制IC值时间序列图的步骤：

1.  **导入工具包**：首先确保 `plotting` 工具包已正确导入。
2.  **调用绘图函数**：使用 `plotting` 模块下的 `plot_ic_ts` 函数。
3.  **传入数据**：将我们之前计算好的IC值序列（通常是一个 `pandas.Series` 对象）作为参数传入该函数。

核心的绘图代码如下：
```python
plotting.plot_ic_ts(IC)
```
执行这段代码后，系统会自动生成并显示图表。

![](img/404997d68477250de6c613545eaec441_3.png)

## 图表结果解读

生成的图表包含了丰富的信息，主要分为以下几个部分：

*   **蓝色折线**：这条线展示了**每日IC值**的实际走势。由于市场每日波动，该线条的起伏通常较大。
*   **绿色折线**：这条线是**滚动平均线**，默认以一个月（约21个交易日）为窗口计算IC值的均值。它有助于我们观察IC值的长期趋势，过滤掉短期噪音。
*   **统计信息**：图表上方会显示IC值的**均值（Mean）** 和**标准差（Std）**。此外，还会计算并显示一个重要的衍生指标——**信息比率（Information Ratio, IR）**。

信息比率的计算公式为：
**IR = Mean(IC) / Std(IC)**
这个比率用于衡量因子IC值的稳定性。**比值越大，说明因子的预测能力越稳定**；比值小则意味着因子的有效性波动较大。

通过观察图表，我们可以重点分析绿色平均线的趋势。一个理想的有效因子，其IC均值（绿色线）应该保持在一个较高的正值水平，并且呈现一定的稳定性。如果绿色线长期在零值附近徘徊或波动无序，则说明该因子与未来收益率的相关性较弱，可能不是有效的选股因子。

## 工具包的其他功能

`plotting` 工具包的功能远不止绘制时间序列图。在它的API文档中，还提供了多种其他分析图表，例如：

![](img/404997d68477250de6c613545eaec441_5.png)

*   **直方图（Histogram）**：查看IC值的分布情况。
*   **QQ图（Q-Q Plot）**：检验IC值是否服从正态分布。
*   **热力图（Heatmap）**：用颜色深浅直观展示不同维度下的IC值。

![](img/404997d68477250de6c613545eaec441_6.png)

![](img/404997d68477250de6c613545eaec441_7.png)

这些高级功能在我们需要对因子进行深入诊断和全面分析时会非常有用。大家可以在后续深入学习因子策略时，自行探索和实践这些工具。
