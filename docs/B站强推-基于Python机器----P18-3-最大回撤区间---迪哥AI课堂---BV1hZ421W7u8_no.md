在本节课中，我们将要学习一个重要的风险指标——最大回撤。它用于衡量投资策略在特定时间段内可能面临的最糟糕情况，即资产净值从峰值到谷底的最大跌幅。理解最大回撤对于评估投资风险至关重要。

## 什么是最大回撤？

上一节我们介绍了策略分析的基本框架，本节中我们来看看如何量化风险。最大回撤描述的并非策略的盈利能力，而是其风险程度。它衡量的是在投资过程中，资产价值从历史最高点到之后最低点的最大百分比跌幅。

**核心公式**如下：
`最大回撤 = max[(Pi - Pj) / Pi]`，其中 `Pi` 和 `Pj` 是资产在时间 `i` 和 `j` (`j > i`) 的价格。这个公式计算的是从某个高点 `Pi` 到后续某个低点 `Pj` 的“惨烈”程度。

例如，在2008年股市大跌期间，许多投资者经历了巨大的回撤，这正体现了该指标的现实意义。在策略回测报告中，最大回撤区间会被明确标识出来，帮助我们识别策略最脆弱的阶段。

## 如何计算最大回撤？

![](img/9d5a11bfa06d9ce46c533ef616f38c37_5.png)

![](img/9d5a11bfa06d9ce46c533ef616f38c37_7.png)

理解了概念后，我们来看看如何用Python进行计算。计算最大回撤的核心思路是：遍历所有时间点，找出从之前最高点到之后某一点的最大跌幅。

首先，我们需要一个价格序列。这里使用Pandas库进行操作。

![](img/9d5a11bfa06d9ce46c533ef616f38c37_15.png)

```python
import pandas as pd

![](img/9d5a11bfa06d9ce46c533ef616f38c37_17.png)

![](img/9d5a11bfa06d9ce46c533ef616f38c37_19.png)

![](img/9d5a11bfa06d9ce46c533ef616f38c37_21.png)

# 假设 df[‘close’] 是我们的收盘价序列
price_series = df[‘close’]

![](img/9d5a11bfa06d9ce46c533ef616f38c37_23.png)

![](img/9d5a11bfa06d9ce46c533ef616f38c37_25.png)

![](img/9d5a11bfa06d9ce46c533ef616f38c37_27.png)

# 计算截至当前时间的累积最大值
cumulative_max = price_series.cummax()

![](img/9d5a11bfa06d9ce46c533ef616f38c37_29.png)

![](img/9d5a11bfa06d9ce46c533ef616f38c37_31.png)

# 计算每日的回撤值：（历史最高点 - 当日价格）/ 历史最高点
drawdown = (cumulative_max - price_series) / cumulative_max

![](img/9d5a11bfa06d9ce46c533ef616f38c37_33.png)

# 最大回撤就是所有每日回撤值中的最大值
max_drawdown = drawdown.max()
```

![](img/9d5a11bfa06d9ce46c533ef616f38c37_35.png)

![](img/9d5a11bfa06d9ce46c533ef616f38c37_37.png)

这段代码中：
1.  `cummax()` 函数用于计算序列的累积最大值。
2.  用累积最大值减去当日价格，得到下跌的绝对幅度。
3.  将绝对幅度除以累积最大值，转化为百分比回撤。
4.  最后使用 `max()` 函数找出整个序列中的最大回撤值。

![](img/9d5a11bfa06d9ce46c533ef616f38c37_39.png)

计算结果（例如0.307或30.7%）表示在该策略周期内，投资者可能面临的最大资产缩水比例。
