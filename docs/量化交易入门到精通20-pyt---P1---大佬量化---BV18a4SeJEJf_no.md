在本节课中，我们将要学习Python中一个非常重要的概念——**代码组**。代码组通过缩进来组织多条语句，是构建条件判断、循环等复杂逻辑的基础。理解并正确使用代码组，是编写清晰、有效Python代码的关键。

## 什么是代码组？

上一节我们介绍了Python的基本语法结构，本节中我们来看看代码组。代码组是由**缩进相同**的多条语句构成的逻辑单元。它通常与`if`、`else`、`while`、`for`等复合语句结合使用，用于清晰地表达程序的控制流。

例如，在条件语句`if`中，所有需要在该条件成立时执行的语句，都必须具有相同的缩进级别，它们共同组成了一个代码组。

## 代码组的应用与嵌套

代码组不仅用于单一的条件分支，还可以进行嵌套，以实现更复杂的逻辑判断。这让我们能够构建多层次的决策结构。

以下是代码组在条件判断中的一个应用示例：

```python
# 假设我们有一个涨跌幅数据
change = 0.09

if change > 0.05:
    print(“涨”)
    # 嵌套的代码组：进一步判断是大涨还是小涨
    if change > 0.08:
        print(“大涨”)
    else:
        print(“小涨”)
else:
    print(“跌”)
```

在这个例子中：
1.  第一个`if`语句检查涨幅是否大于5%，其下的`print(“涨”)`构成了一个代码组。
2.  在这个代码组内部，又嵌套了一个`if-else`语句，用于判断是大涨还是小涨，这形成了另一个代码组。

运行上述代码，因为`change = 0.09`，程序会依次输出“涨”和“大涨”。

## 缩进的重要性

在Python中，**缩进不是可选的，而是语法的一部分**。它严格定义了代码组的开始和结束。如果缩进不正确，程序将无法正确解析逻辑，从而导致语法错误。

例如，以下代码的缩进是错误的：

```python
if change > 0.05:
print(“涨”)  # 错误：这条语句没有正确缩进
```

执行这段代码会引发`IndentationError`，因为`print`语句没有与`if`语句体保持一致的缩进。

本节课中我们一起学习了Python的**代码组**概念。我们了解到：
