# 量化投资入门：第1章：基于财务报表的量化策略构建 📊

在本节课中，我们将学习如何利用企业的财务报表数据来构建一个简单的量化投资策略。我们将从理解财报的基本构成开始，逐步探讨财报如何影响股价，并最终动手设计一个基于净利润增长的选股策略进行回测。

## 理解财务报表 📑

财务报表是了解企业健康状况的核心工具。它主要由三张表构成。

资产负债表展示了企业在某一特定时间点的财务状况，包括资产、负债和所有者权益。其核心公式为：
**资产 = 负债 + 所有者权益**

利润表则显示了企业在一定时期内的经营成果，包括收入、成本和利润。

而现金流量表记录了企业在该时期内的现金流入和流出情况，反映其真实的现金创造能力。

## 财报如何驱动股价波动 📈

上一节我们介绍了财务报表的基本构成，本节中我们来看看它们如何影响市场。在财报发布前，市场通常会形成对业绩的预期。如果实际财报数据优于预期，股价可能会上涨；反之，如果财报不及预期，股价则可能下跌。这种市场反应机制为量化策略提供了切入点。

## 构建量化策略：从理念到实现 ⚙️

理解了财报与股价的关系后，我们可以尝试将财报数据转化为可量化的投资因子。以下是构建一个简单策略的关键步骤。

首先，我们需要明确策略目标：通过财报数据筛选出优质股票，并自动剔除已退市等无效数据。

接着，我们需要收集和处理财报数据，并选择具体的量化指标。例如，我们可以使用“净利润同比增长率”作为核心选股指标。

然后，根据指标数值对所有股票进行排序，并选择排名靠前的股票构建投资组合。一个简单的规则是：选择净利润同比增长最大的十只股票，并均分仓位。

最后，进行交易设置。例如，设定在交易日开盘时买入，收盘时卖出，初始资金为100万元，持仓股票数为10支，每20天调仓一次。
