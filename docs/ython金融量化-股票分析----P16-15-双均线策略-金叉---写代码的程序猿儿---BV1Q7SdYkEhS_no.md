## 什么是金叉与死叉？🤔

![](img/ed2a4afaa9a96253b272fb5b141181d2_11.png)

![](img/ed2a4afaa9a96253b272fb5b141181d2_13.png)

在股票技术分析中，金叉和死叉是分析指标中两根线交叉的形态。目前我们绘制的图表中有两条线：一条是5日均线（MA5），另一条是30日均线（MA30）。其中，MA5被称为短期均线，MA30被称为长期均线。

![](img/ed2a4afaa9a96253b272fb5b141181d2_15.png)

以下是金叉和死叉的定义：

![](img/ed2a4afaa9a96253b272fb5b141181d2_17.png)

![](img/ed2a4afaa9a96253b272fb5b141181d2_19.png)

*   **金叉**：当短期均线从下方拐头向上，并穿过长期均线时，其交点附近的状态称为金叉。
*   **死叉**：当短期均线从上方拐头向下，并穿过长期均线时，其交点附近的状态称为死叉。

为了更直观地理解，我们可以参考一个简单的图示。假设黑色线代表短期均线（MA5），红色线代表长期均线（MA30）。

```
        金叉点         死叉点         金叉点
           ▲             ▼             ▲
黑线(MA5)  / \           / \           / \
          /   \         /   \         /   \
_________/_____\_______/_____\_______/_____\______> 时间
        /       \     /       \     /
红线(MA30)      \___/         \___/
```

![](img/ed2a4afaa9a96253b272fb5b141181d2_21.png)

在上图中，存在三个交点。第一个和第三个交点是短期均线上穿长期均线，为**金叉**。第二个交点是短期均线下穿长期均线，为**死叉**。

![](img/ed2a4afaa9a96253b272fb5b141181d2_23.png)

一般情况下，金叉出现后，操作趋向于**买入**；死叉出现后，操作趋向于**卖出**。双均线策略的核心就是根据金叉和死叉的信号来制定股票的买卖决策。

![](img/ed2a4afaa9a96253b272fb5b141181d2_25.png)

## 如何在数据中定位金叉与死叉？🔍

我们的目标是找出金叉和死叉发生的具体日期。在图表中，X轴代表时间，Y轴代表计算出的移动平均值（即均线的值）。金叉或死叉的点，代表在某一时刻，短期均线（MA5）的值与长期均线（MA30）的值相等。

观察交叉点前后的数据，我们可以发现一个规律：

*   在金叉点**左侧**，`MA5 < MA30`。
*   在金叉点**右侧**，`MA5 > MA30`。
*   在死叉点**左侧**，`MA5 > MA30`。
*   在死叉点**右侧**，`MA5 < MA30`。

![](img/ed2a4afaa9a96253b272fb5b141181d2_27.png)

因此，我们可以通过分析 `MA5` 和 `MA30` 两列数据的大小关系变化，来定位金叉和死叉。

![](img/ed2a4afaa9a96253b272fb5b141181d2_29.png)

![](img/ed2a4afaa9a96253b272fb5b141181d2_31.png)

## 用代码找出金叉与死叉日期 💻

我们将通过以下步骤，使用Python代码找出金叉和死叉日期。

首先，为了确保数据长度一致且没有空值（NaN），我们对均线数据进行切片。由于MA30的前29个值为空，我们统一从第30个数据开始取。

```python
# 对均线数据进行切片，从第30个数据开始，确保索引一致
ma5 = df[‘ma5’][30:]
ma30 = df[‘ma30’][30:]
# 同时，对原始数据框也进行同样的切片，以便后续对齐
df = df[30:]
```

接下来，我们创建两个布尔序列，分别表示 `MA5 < MA30` 和 `MA5 > MA30` 的条件。

![](img/ed2a4afaa9a96253b272fb5b141181d2_33.png)

![](img/ed2a4afaa9a96253b272fb5b141181d2_35.png)

```python
# 创建布尔序列
s1 = ma5 < ma30   # 序列中True表示 MA5 < MA30
s2 = ma5 > ma30   # 序列中True表示 MA5 > MA30
```

`s1` 和 `s2` 是互补的。我们的目标是找到 `s1` 中由 `True` 变为 `False` 的点（金叉），以及由 `False` 变为 `True` 的点（死叉）。

以下是定位死叉和金叉的核心方法：

**1. 找出死叉日期**

死叉对应 `s1` 序列中 `False` 过渡到 `True` 的位置。我们可以通过 `s1 & (s2.shift(1))` 这个条件来定位。
*   `s2.shift(1)` 表示将 `s2` 序列向下移动一位。
*   当 `s1` 的当前值为 `True`，且 `s2` 的前一个值为 `True`（即移动后的当前值）时，`s1 & (s2.shift(1))` 的结果为 `True`。这正好对应了 `s1` 从 `False` 变为 `True` 的瞬间。

![](img/ed2a4afaa9a96253b272fb5b141181d2_37.png)

```python
# 判定死叉的条件
death_ex = s1 & s2.shift(1)
# 获取死叉对应的行数据，并提取其索引（日期）
death_date = df.loc[death_ex].index
```

**2. 找出金叉日期**

金叉对应 `s1` 序列中 `True` 过渡到 `False` 的位置。我们可以通过 `~(s1 | s2.shift(1))` 这个条件来定位。
*   `s1 | s2.shift(1)` 表示 `s1` 的当前值或 `s2` 的前一个值为 `True`。
*   对这个结果取反（`~`），得到 `True` 的位置，就对应了 `s1` 从 `True` 变为 `False` 的瞬间。

![](img/ed2a4afaa9a96253b272fb5b141181d2_39.png)

```python
# 判定金叉的条件
golden_ex = ~(s1 | s2.shift(1))
# 获取金叉对应的行数据，并提取其索引（日期）
golden_date = df.loc[golden_ex].index
```
