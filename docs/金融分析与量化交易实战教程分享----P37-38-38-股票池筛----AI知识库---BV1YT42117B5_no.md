## 概述
在本节课中，我们将学习如何基于预处理后的因子数据，对股票池进行筛选。我们将完成因子数据的离群值处理、标准化和中性化操作，并基于处理后的因子值，筛选出符合特定条件的股票，为后续的调仓交易做准备。

首先，我们需要对因子数据执行一系列预处理操作。以下是预处理操作的步骤：

![](img/0681f1444e29b78e54e8979b51035e21_18.png)

![](img/0681f1444e29b78e54e8979b51035e21_20.png)

![](img/0681f1444e29b78e54e8979b51035e21_22.png)

1.  **去除离群值**：使用 `winsorize` 方法处理因子数据中的极端值。
2.  **标准化**：使用 `standardize` 方法将因子数据转换为均值为0、标准差为1的标准正态分布。
3.  **中性化**：使用之前编写的 `neutralization` 函数，消除因子在行业和市值上的暴露。

![](img/0681f1444e29b78e54e8979b51035e21_24.png)

![](img/0681f1444e29b78e54e8979b51035e21_26.png)

我们将这些步骤整合到代码中。首先，获取基础因子数据 `fundamental_df`，其中包含我们所需的指标，例如市净率（PB）。

![](img/0681f1444e29b78e54e8979b51035e21_28.png)

![](img/0681f1444e29b78e54e8979b51035e21_30.png)

![](img/0681f1444e29b78e54e8979b51035e21_32.png)

```python
# 获取基础因子数据
fundamental_df = get_fundamentals(...)
# 提取市净率因子
factor_value = fundamental_df['PB']
```

![](img/0681f1444e29b78e54e8979b51035e21_34.png)

![](img/0681f1444e29b78e54e8979b51035e21_36.png)

接着，按顺序执行预处理三步操作：

![](img/0681f1444e29b78e54e8979b51035e21_38.png)

![](img/0681f1444e29b78e54e8979b51035e21_40.png)

![](img/0681f1444e29b78e54e8979b51035e21_42.png)

```python
# 1. 去除离群值
factor_no_outliers = winsorize(factor_value)
# 2. 标准化
factor_standardized = standardize(factor_no_outliers)
# 3. 中性化 (假设已定义neutralization函数和行业、市值数据)
factor_neutralized = neutralization(factor_standardized, industry_data, market_cap_data)
# 将最终处理后的因子命名为‘preprocessed_PB’
preprocessed_PB = factor_neutralized
```

接下来，我们需要基于处理后的因子值对股票池进行筛选。我们的目标是选出因子值较小（例如市净率较低）的股票，因为根据策略假设，这类股票未来可能有更高的期望收益。

![](img/0681f1444e29b78e54e8979b51035e21_52.png)

![](img/0681f1444e29b78e54e8979b51035e21_54.png)

以下是筛选股票的具体步骤：

![](img/0681f1444e29b78e54e8979b51035e21_56.png)

![](img/0681f1444e29b78e54e8979b51035e21_58.png)

1.  **确定筛选阈值**：我们计划选择因子值最小的前20%的股票。可以通过计算因子的分位数来确定阈值。
2.  **执行筛选**：比较每只股票的因子值与阈值，选出符合条件的股票代码。

![](img/0681f1444e29b78e54e8979b51035e21_60.png)

![](img/0681f1444e29b78e54e8979b51035e21_62.png)

![](img/0681f1444e29b78e54e8979b51035e21_64.png)

在代码中实现如下：

![](img/0681f1444e29b78e54e8979b51035e21_66.png)

![](img/0681f1444e29b78e54e8979b51035e21_68.png)

```python
# 计算20%分位数作为阈值
threshold = preprocessed_PB.quantile(0.2)
# 筛选出因子值小于等于阈值的股票
selected_stocks = preprocessed_PB[preprocessed_PB <= threshold].index.tolist()
```

1.  **获取当前持仓**：从上下文 `context` 中获取当前账户持有的所有股票。
2.  **找出需卖出的股票**：即当前持有但不在新股票池 `selected_stocks` 中的股票。
3.  **找出需买入的股票**：即在新股票池 `selected_stocks` 中但当前未持有的股票。

## 总结
本节课中我们一起学习了股票池筛选的完整流程。我们首先对原始因子数据进行了**去除离群值**、**标准化**和**中性化**三步预处理。然后，基于处理后的因子值，通过设定分位数阈值**筛选出目标股票**。最后，通过对比目标股票池与当前持仓，**确定了需要调仓的股票列表**（包括买入和卖出）。这个过程是量化策略中股票选择模块的核心，为下一步执行具体的交易订单奠定了基础。
