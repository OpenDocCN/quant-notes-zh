## 概述

![](img/44dc6bbe80269c4de61781a4144c4395_3.png)

人工智能技术已广泛应用于各个领域。在编程界，AI工具正扮演着越来越重要的角色。上一节我们介绍了其他工具，本节中我们来看看同样备受关注的智能编程神器——Cursor。

Cursor是基于GPT-4模型的编程工具，可用于生成、编辑代码，并与AI讨论分析代码逻辑。它内置了许多编程模板，对于转行量化交易的编程新手而言，能轻松应对基础编程任务。

使用体验上，Cursor更像一个编程助手。你向它描述编程任务，它会帮你生成代码。如果代码出现错误，你可以将错误信息放入聊天框询问，它会帮你分析原因并提供解决方案。

关键在于，Cursor目前所有功能均可免费使用，无需注册。

## 工具安装与界面

Cursor不像ChatGPT那样直接在网页端使用，而是需要下载客户端软件。

下载安装后，打开软件，你会看到一个采用暗黑色风格的开发界面，类似于VS Code。

## 核心使用技巧

Cursor的使用非常简单，主要记住两个快捷键即可。

*   **`Ctrl + K`**：主要用于生成和编辑代码。
*   **`Ctrl + L`**：主要用于开启聊天，进行代码说明、注释和解释。

对于代码中不理解或出错的地方，都可以使用 `Ctrl + L` 让Cursor进行说明和改进。

## 实战操作步骤

打开Cursor后，默认会创建一个Python（`.py`）或JavaScript（`.js`）文件。它也支持C#、HTML等多种编程语言。

以下是几个关键功能的操作指南。

### 1. 为代码添加注释

如果你是编程新手，希望为现有代码添加注释以增强可读性，操作如下：

1.  选中目标代码段。
2.  按下 `Ctrl + K`。
3.  在出现的输入框中输入指令，例如：`请给每一行代码添加注释`。
4.  代码会自动被修改，添加详细的注释。

修改后的代码段左上角会出现绿色（Accept，接受）和红色（Reject，拒绝）按钮。点击绿色按钮即可确认修改，使代码可读性大幅提升。

### 2. 与AI讨论代码

在阅读代码过程中，若有不明白之处，可以使用聊天功能深入询问。

1.  将光标置于相关代码区域或选中代码。
2.  按下 `Ctrl + L`。
3.  在弹出的聊天框中输入你的问题。

例如，对于函数 `kindle_stick_ohl()` 不理解，可以输入：`请解释一下这个函数的功能和参数。`

4.  按下回车键，界面右侧会开启一个聊天小窗，Cursor将在此回复你的问题。
5.  若对回复仍有疑问，可以继续在聊天框中追问，使用过程与ChatGPT非常相似。

## 代码示例与练习

安娜老师提供了一段简单的示例代码，供大家练习使用：

```python
# 示例：计算简单移动平均线 (SMA)
def calculate_sma(prices, window):
    """
    计算简单移动平均线。
    参数:
        prices (list): 价格列表。
        window (int): 移动平均窗口大小。
    返回:
        list: 移动平均值列表。
    """
    if len(prices) < window:
        return []
    sma_values = []
    for i in range(window - 1, len(prices)):
        window_slice = prices[i - window + 1: i + 1]
        sma = sum(window_slice) / window
        sma_values.append(sma)
    return sma_values

# 示例数据
price_data = [100, 102, 101, 105, 107, 106, 110]
sma_result = calculate_sma(price_data, 3)
print("简单移动平均线(SMA):", sma_result)
```

你可以将这段代码复制到Cursor中，尝试使用 `Ctrl + K` 和 `Ctrl + L` 功能来添加注释或询问其工作原理。
